`README.md`:

```md
# com.sipvlib.anticheat

Part of [SiPVLib](https://github.com/phajmvawnsix/SiPVLib). Server-verified game time (`GameTime`, fetched from a fallback chain of public time APIs instead of trusting the device's raw system clock) plus lightweight heuristic runtime integrity checks (`IntegrityChecker`: debugger-attached, root/jailbreak, emulator, and clock-tamper detection).

## Install

Add to your project's `Packages/manifest.json`:

```json
"com.sipvlib.anticheat": "https://github.com/phajmvawnsix/com.sipvlib.anticheat.git",
"com.sipvlib.config": "https://github.com/phajmvawnsix/com.sipvlib.config.git",
"com.sipvlib.debugging": "https://github.com/phajmvawnsix/com.sipvlib.debugging.git",
"com.sipvlib.event": "https://github.com/phajmvawnsix/com.sipvlib.event.git",
"com.sipvlib.utilities": "https://github.com/phajmvawnsix/com.sipvlib.utilities.git",
"com.cysharp.unitask": "https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask"
```

UPM does not automatically resolve nested git dependencies — you must add the `com.sipvlib.*` and UniTask entries above yourself alongside this package.

## `GameTime` — trusted time

Fetches "true" UTC time from a fallback chain of public time APIs instead of trusting
`DateTime.Now`/`DateTime.UtcNow`, which a cheater can freely change. Between fetches,
elapsed time is derived from `Time.deltaTime` accumulation, not by re-reading the system
clock, so adjusting the OS clock mid-session doesn't move `GameTime.UtcNow`.

```csharp
using SiPVLib.AntiCheat;

// Bootstrap once, e.g. from your app's init sequence
await GameTime.Instance.Initialize();

// Read anywhere afterward — static, no Instance needed
var utc = GameTime.UtcNow;
var local = GameTime.Now;

if (GameTime.IsInitialized) { /* first fetch has completed */ }
```

`GameTime.UtcNow` falls back to the raw (untrusted) `DateTime.UtcNow` before `Initialize()`
completes a fetch — it logs a warning once when this happens. Always await `Initialize()`
before relying on `GameTime.UtcNow` for anything cheat-sensitive (rewards, cooldowns,
daily resets).

### Events

| Const | Fired | Payload |
|---|---|---|
| `GameTime.EventTimeTick` | Every `TimeTickInterval` seconds, once first fetch succeeds | none |
| `GameTime.EventDateChanged` | When the local calendar date rolls over | `DateTime` (local) |
| `GameTime.EventDateUtcChanged` | When the UTC calendar date rolls over | `DateTime` (UTC) |

```csharp
EventManager.Add(GameTime.EventDateUtcChanged, OnDailyReset);
void OnDailyReset(DateTime utcNow) { /* reset daily quests, shop, etc. */ }
```

### Re-verification

Re-fetches automatically on an interval (`FetchInterval`, default 300s) and whenever the
app regains focus (`OnApplicationFocus`) — catches a cheater backgrounding the app,
changing the clock, then resuming.

### Simulating a clock cheat (dev/editor only)

```csharp
GameTime.AddCheatTimeOffset(3600);   // pretend the clock jumped forward 1 hour
GameTime.GetCheatTimeOffset();
```

Compiled only under `DEVELOPMENT_BUILD || UNITY_EDITOR`; no-ops in release builds. Use
this to test `IntegrityChecker`'s clock-tamper detection (below) without touching the
actual OS clock.

## `ConfigGameTime` — tuning

A `GameConfig` asset (see [Config module](https://github.com/phajmvawnsix/com.sipvlib.config)) referenced by `GameTime`
via `[ConfigRef]` on its `_configId` field.

| Field | Meaning |
|---|---|
| `TimeAPIs` | Ordered fallback list of time-API endpoints (`url`, response `key`, parse `format`). Tried in order each fetch until one succeeds. |
| `RequireInitFetch` | If true, `Initialize()` blocks until the first fetch succeeds. |
| `TimeTickInterval` | Seconds between `EventTimeTick` dispatches (and date-change checks). |
| `FetchInterval` | Seconds between periodic re-fetches. |
| `FetchTimeout` | Per-request timeout (seconds) for each time-API call. |
| `EnableIntegrityCheck` | Toggles `IntegrityChecker.RunAll` calls from `GameTime`. |
| `ClockTamperThresholdSeconds` | Drift (seconds) between raw system clock and verified `GameTime.UtcNow` before `ClockTampered` is flagged. |

## `IntegrityChecker` — runtime tamper signals

Static, no `MonoBehaviour` — runs cheap heuristic checks and caches the result so callers
can poll without re-running them. Driven automatically by `GameTime` (after first fetch
completes, and on every app refocus) when `ConfigGameTime.EnableIntegrityCheck` is true;
can also be called directly.

```csharp
using SiPVLib.AntiCheat;

if (IntegrityChecker.IsCompromised) { /* block leaderboard submission, etc. */ }

if (IntegrityChecker.HasViolation(IntegrityViolationType.ClockTampered)) { /* ... */ }

EventManager.Add<IntegrityViolationType>(IntegrityChecker.EventIntegrityViolation, OnViolation);
void OnViolation(IntegrityViolationType type) { CustomLog.LogWarning($"Integrity: {type}"); }
```

| Check | Platform | Technique |
|---|---|---|
| `DebuggerAttached` | All | `System.Diagnostics.Debugger.IsAttached` |
| `RootedOrJailbroken` | Android / iOS | File-existence heuristic against known root/jailbreak paths (`su` binaries, Cydia, etc.) |
| `EmulatorDetected` | Android | Substring match on `SystemInfo.deviceModel`/`graphicsDeviceName` against known emulator markers (Bluestacks, Nox, Genymotion, ...) |
| `ClockTampered` | All | Drift between raw `DateTime.UtcNow` and verified `GameTime.UtcNow` exceeding `ClockTamperThresholdSeconds` |

All checks are heuristic, not tamper-proof (there's no native plugin backing this) — treat
a positive as "worth extra scrutiny / soft-block", not cryptographic proof of cheating.
Root/jailbreak and emulator checks are skipped in the Editor and return `false` on
platforms other than Android/iOS.

`Runtime/GameTime.cs`:

```cs
using System;
using System.Collections.Generic;
using System.Globalization;
using Cysharp.Threading.Tasks;
using SiPVLib.Config;
using SiPVLib.Debugging;
using SiPVLib.Event;
using SiPVLib.Utilities;
using UnityEngine;
using UnityEngine.Networking;

namespace SiPVLib.AntiCheat
{
    public class GameTime : MonoSingleton<GameTime>
    {
        public const string EventTimeTick = "Time.Tick";
        public const string EventDateChanged = "Time.DateChanged";
        public const string EventDateUtcChanged = "Time.DateUtcChanged";
        private const string CheatTimeOffset = "CheatTimeOffset";

        [SerializeField, ConfigRef(typeof(ConfigGameTime))] private string _configId;

        private ConfigGameTime _config;

        private bool _isFirstTimeFetched;
        private bool _hasLoggedPreInitFallback;

        private DateTime _lastFetchedTime = DateTime.MinValue;
        private float _timeSpanSinceLastFetch;
        private float _timeSinceLastIntervalFetch;
        private float _timeSinceLastTick;
        private int _addedTimeOffset = -1;

        private readonly Dictionary<TimeFetchType, UniTask> _runningFetchesDict = new();

        private long _frameCountGet;
        private DateTime _frameUtcTime;
        private DateTime _lastDateUtcCheck = DateTime.MinValue;
        private DateTime _lastLocalDateCheck = DateTime.MinValue;

        public static DateTime UtcNow
        {
            get
            {
                if (!Instance._isFirstTimeFetched)
                {
                    if (!Instance._hasLoggedPreInitFallback)
                    {
                        Instance._hasLoggedPreInitFallback = true;
                        CustomLog.LogWarning("GameTime.UtcNow read before Initialize() completed a fetch. Falling back to untrusted system clock.");
                    }
                    return DateTime.UtcNow;
                }

                try
                {
                    if (Instance._frameCountGet != Time.frameCount)
                    {
                        Instance._frameUtcTime = Instance._lastFetchedTime.AddSeconds(Instance._timeSpanSinceLastFetch);

#if DEVELOPMENT_BUILD || UNITY_EDITOR
                        var addedOffset = Instance._addedTimeOffset >= 0 ? Instance._addedTimeOffset : 0;
                        var offsetTimeSpan = TimeSpan.FromSeconds(addedOffset);
                        Instance._frameUtcTime = Instance._frameUtcTime.Add(offsetTimeSpan);
#endif
                        Instance._frameCountGet = Time.frameCount;
                    }
                }
                catch (Exception e)
                {
                    CustomLog.LogWarning($"GameTime.UtcNow called from non-main thread: {e.Message}. Using cached time.");
                }

                return Instance._frameUtcTime;
            }
        }

        public static DateTime Now => UtcNow.ToLocalTime();

        public static bool IsInitialized => Instance != null && Instance._isFirstTimeFetched;

        public async UniTask<bool> Initialize()
        {
            await UniTask.WaitUntil(() =>
            {
                if (ConfigManager.Instance.IsFullInitialized)
                {
                    _config = ConfigManager.Get<ConfigGameTime>(_configId, findAllIfNotFound:true);
                    return _config != null;
                }
                return false;
            });

            Fetch(TimeFetchType.Init);

            if (_config.RequireInitFetch)
            {
                await UniTask.WaitUntil(() => _isFirstTimeFetched);
            }

#if DEVELOPMENT_BUILD || UNITY_EDITOR
            if (_addedTimeOffset < 0)
            {
                _addedTimeOffset = PlayerPrefs.GetInt(CheatTimeOffset);
            }
#endif
            RunIntegrityCheckIfEnabled();
            return true;
        }

        private void Fetch(TimeFetchType fetchType)
        {
            if (_runningFetchesDict.ContainsKey(fetchType))
            {
                CustomLog.LogWarning($"GameTime: {fetchType} fetch is already running. Skipping duplicate request.");
                return;
            }

            _runningFetchesDict[fetchType] = FetchAsync(fetchType);
        }

        private async UniTask FetchAsync(TimeFetchType fetchType)
        {
            var success = false;
            var retryCount = 0;

            while (!success)
            {
                foreach (var apiConfig in _config.TimeAPIs)
                {
                    var url = apiConfig.url;

                    using (var request = UnityWebRequest.Get(url))
                    {
                        request.timeout = (int) _config.FetchTimeout;
                        request.useHttpContinue = false;

                        await request.SendWebRequest();

                        if (request.result == UnityWebRequest.Result.Success)
                        {
                            try
                            {
                                var responseText = request.downloadHandler.text;
                                var fetchedTime = ParseTimeFromResponse(responseText, apiConfig);

                                if (fetchedTime > _lastFetchedTime)
                                {
                                    CustomLog.Log($" Fetched GameTime: {fetchedTime.ToLocalTime()}");

                                    _lastFetchedTime = fetchedTime;
                                    _timeSpanSinceLastFetch = 0f;
                                    success = true;

                                    if (fetchType == TimeFetchType.Init)
                                    {
                                        _isFirstTimeFetched = true;
                                        CustomLog.Log(
                                            $"GameTime: First time fetch successful from {url}. Time: {_lastFetchedTime:yyyy-MM-dd HH:mm:ss}");
                                    }
                                }
                                else if (fetchedTime != DateTime.MinValue)
                                {
                                    CustomLog.LogWarning($"GameTime: Fetched time {fetchedTime} from {url} is not newer than last fetched time {_lastFetchedTime}. Ignoring.");
                                }
                            }
                            catch (Exception e)
                            {
                                CustomLog.LogWarning($"GameTime: Failed to parse time from {url}. Error: {e.Message}");
                            }
                        }
                        else
                        {
                            CustomLog.LogWarning($"GameTime: Failed to fetch time from {url}. Error: {request.error}");
                        }
                    }

                    if (success) break;
                }

                if (!success)
                {
                    CustomLog.LogError($"GameTime: All API endpoints failed for {fetchType} fetch. Retrying in {OtherUtils.GetRetryTime(retryCount)} seconds...");
                    await UniTask.Delay(OtherUtils.GetRetryTime(retryCount++));
                }
            }

            _runningFetchesDict.Remove(fetchType);
        }

        private DateTime ParseTimeFromResponse(string response, TimeApi api)
        {
            try
            {
                if (api.format == "unix")
                {
                    var lines = response.Split('\n');
                    foreach (var line in lines)
                    {
                        if (line.StartsWith(api.key + "="))
                        {
                            var tsValue = line.Substring(api.key.Length + 1);
                            if (double.TryParse(tsValue, NumberStyles.Any, CultureInfo.InvariantCulture, out var unixTimestamp))
                            {
                                return DateTimeOffset.FromUnixTimeSeconds((long)unixTimestamp).UtcDateTime;
                            }
                        }
                    }
                }
                else
                {
                    var searchKey = $"\"{api.key}\":\"";
                    var keyIndex = response.IndexOf(searchKey, StringComparison.Ordinal);
                    if (keyIndex > -1)
                    {
                        var startIndex = keyIndex + searchKey.Length;
                        var endIndex = response.IndexOf("\"", startIndex, StringComparison.Ordinal);
                        if (endIndex > -1)
                        {
                            var timeValue = response.Substring(startIndex, endIndex - startIndex);
                            if (DateTime.TryParseExact(timeValue, api.format, CultureInfo.InvariantCulture, DateTimeStyles.AdjustToUniversal, out var parsedTime))
                            {
                                return parsedTime;
                            }
                        }
                    }
                }
            }
            catch (Exception e)
            {
                CustomLog.LogError($"GameTime: Error parsing time from {api.url}. Response: {response}. Error: {e.Message}");
            }

            return DateTime.MinValue;
        }

        private void Update()
        {
            if (_config == null) return;

            _timeSpanSinceLastFetch += Time.deltaTime;
            _timeSinceLastIntervalFetch += Time.deltaTime;
            _timeSinceLastTick += Time.deltaTime;

            if (_timeSinceLastIntervalFetch >= _config.FetchInterval && _isFirstTimeFetched)
            {
                Fetch(TimeFetchType.Interval);
                _timeSinceLastIntervalFetch = 0f;
            }

            if (_isFirstTimeFetched && _timeSinceLastTick >= _config.TimeTickInterval)
            {
                _timeSinceLastTick = 0f;
                EventManager.Invoke(EventTimeTick);
                CheckDateChanged();
            }
        }

        private void CheckDateChanged()
        {
            var utcNow = UtcNow;
            if (_lastDateUtcCheck == DateTime.MinValue)
            {
                _lastDateUtcCheck = utcNow.Date;
            }
            else if (utcNow.Date != _lastDateUtcCheck)
            {
                _lastDateUtcCheck = utcNow.Date;
                EventManager.Invoke(EventDateUtcChanged, utcNow);
            }

            var localNow = utcNow.ToLocalTime();
            if (_lastLocalDateCheck == DateTime.MinValue)
            {
                _lastLocalDateCheck = localNow.Date;
            }
            else if (localNow.Date != _lastLocalDateCheck)
            {
                _lastLocalDateCheck = localNow.Date;
                EventManager.Invoke(EventDateChanged, localNow);
            }
        }

        private void OnApplicationFocus(bool hasFocus)
        {
            if (hasFocus && _isFirstTimeFetched)
            {
                UniTask.NextFrame().ContinueWith(() => Fetch(TimeFetchType.Refocus));
                RunIntegrityCheckIfEnabled();
            }
        }

        private void RunIntegrityCheckIfEnabled()
        {
            if (_config != null && _config.EnableIntegrityCheck && _isFirstTimeFetched)
            {
                IntegrityChecker.RunAll(UtcNow, _config.ClockTamperThresholdSeconds);
            }
        }

        public static void AddCheatTimeOffset(int offsetSeconds)
        {
#if DEVELOPMENT_BUILD || UNITY_EDITOR
            Instance._addedTimeOffset += offsetSeconds;
            PlayerPrefs.SetInt(CheatTimeOffset, Instance._addedTimeOffset);
#endif
        }

        public static long GetCheatTimeOffset() => Instance._addedTimeOffset;
    }
}

using System;
using System.Collections.Generic;
using System.Diagnostics;
using System.IO;
using SiPVLib.Debugging;
using SiPVLib.Event;
using UnityEngine;

namespace SiPVLib.AntiCheat
{
    public static class IntegrityChecker
    {
        public const string EventIntegrityViolation = "AntiCheat.IntegrityViolation";

        private static readonly string[] AndroidRootPaths =
        {
            "/system/app/Superuser.apk",
            "/sbin/su",
            "/system/bin/su",
            "/system/xbin/su",
            "/data/local/xbin/su",
            "/data/local/bin/su",
            "/system/sd/xbin/su",
            "/system/bin/failsafe/su",
            "/data/local/su",
            "/su/bin/su"
        };

        private static readonly string[] IosJailbreakPaths =
        {
            "/Applications/Cydia.app",
            "/Library/MobileSubstrate/MobileSubstrate.dylib",
            "/bin/bash",
            "/usr/sbin/sshd",
            "/etc/apt",
            "/private/var/lib/apt"
        };

        private static readonly string[] EmulatorMarkers =
        {
            "bluestacks", "nox", "genymotion", "andy", "memu", "ldplayer"
        };

        private static readonly HashSet<IntegrityViolationType> Violations = new();

        public static bool IsCompromised => Violations.Count > 0;

        public static bool HasViolation(IntegrityViolationType type) => Violations.Contains(type);

        // ── Public API ───────────────────────────────────────────────────

        public static void RunAll(DateTime verifiedUtc, float clockTamperThresholdSeconds)
        {
            Report(IntegrityViolationType.DebuggerAttached, CheckDebugger());
            Report(IntegrityViolationType.RootedOrJailbroken, CheckRootOrJailbreak());
            Report(IntegrityViolationType.EmulatorDetected, CheckEmulator());
            Report(IntegrityViolationType.ClockTampered, CheckClockTamper(verifiedUtc, clockTamperThresholdSeconds));
        }

        public static bool CheckDebugger() => Debugger.IsAttached;

        public static bool CheckRootOrJailbreak()
        {
#if UNITY_EDITOR
            return false;
#elif UNITY_ANDROID
            foreach (var path in AndroidRootPaths)
            {
                if (File.Exists(path)) return true;
            }
            return false;
#elif UNITY_IOS
            foreach (var path in IosJailbreakPaths)
            {
                if (File.Exists(path) || Directory.Exists(path)) return true;
            }
            return false;
#else
            return false;
#endif
        }

        public static bool CheckEmulator()
        {
#if UNITY_EDITOR
            return false;
#elif UNITY_ANDROID
            var model = SystemInfo.deviceModel?.ToLowerInvariant() ?? string.Empty;
            var gpu = SystemInfo.graphicsDeviceName?.ToLowerInvariant() ?? string.Empty;

            foreach (var marker in EmulatorMarkers)
            {
                if (model.Contains(marker) || gpu.Contains(marker)) return true;
            }
            return false;
#else
            return false;
#endif
        }

        public static bool CheckClockTamper(DateTime verifiedUtc, float thresholdSeconds)
        {
            var drift = Math.Abs((DateTime.UtcNow - verifiedUtc).TotalSeconds);
            return drift > thresholdSeconds;
        }

        // ── Private ──────────────────────────────────────────────────────

        private static void Report(IntegrityViolationType type, bool violated)
        {
            if (!violated)
            {
                Violations.Remove(type);
                return;
            }

            if (!Violations.Add(type)) return;

            CustomLog.LogWarning($"IntegrityChecker: violation detected - {type}");
            EventManager.Invoke(EventIntegrityViolation, type);
        }
    }
}

`USAGE.md`:

```md
# SiPV.AntiCheat

Server-verified game time plus lightweight runtime integrity checks, so gameplay code
never trusts the device's raw system clock or an untampered-process assumption.

Depends on `SiPV.Config` (`GameTime` fetches its tuning from a `ConfigGameTime` asset,
looked up by Id), `SiPV.Debugging` (`CustomLog`), `SiPV.Utilities` (`MonoSingleton<T>`,
`OtherUtils.GetRetryTime`), `SiPV.Event` (`EventManager` dispatches tick/date-change/
integrity-violation events), and `UniTask` for async fetch orchestration.

---

## `GameTime` — trusted time

Fetches "true" UTC time from a fallback chain of public time APIs instead of trusting
`DateTime.Now`/`DateTime.UtcNow`, which a cheater can freely change. Between fetches,
elapsed time is derived from `Time.deltaTime` accumulation, not by re-reading the system
clock, so adjusting the OS clock mid-session doesn't move `GameTime.UtcNow`.

```csharp
using SiPVLib.AntiCheat;

// Bootstrap once, e.g. from your app's init sequence
await GameTime.Instance.Initialize();

// Read anywhere afterward — static, no Instance needed
var utc = GameTime.UtcNow;
var local = GameTime.Now;

if (GameTime.IsInitialized) { /* first fetch has completed */ }
```

`GameTime.UtcNow` falls back to the raw (untrusted) `DateTime.UtcNow` before `Initialize()`
completes a fetch — it logs a warning once when this happens. Always await `Initialize()`
before relying on `GameTime.UtcNow` for anything cheat-sensitive (rewards, cooldowns,
daily resets).

### Events

| Const | Fired | Payload |
|---|---|---|
| `GameTime.EventTimeTick` | Every `TimeTickInterval` seconds, once first fetch succeeds | none |
| `GameTime.EventDateChanged` | When the local calendar date rolls over | `DateTime` (local) |
| `GameTime.EventDateUtcChanged` | When the UTC calendar date rolls over | `DateTime` (UTC) |

```csharp
EventManager.Add(GameTime.EventDateUtcChanged, OnDailyReset);
void OnDailyReset(DateTime utcNow) { /* reset daily quests, shop, etc. */ }
```

### Re-verification

Re-fetches automatically on an interval (`FetchInterval`, default 300s) and whenever the
app regains focus (`OnApplicationFocus`) — catches a cheater backgrounding the app,
changing the clock, then resuming.

### Simulating a clock cheat (dev/editor only)

```csharp
GameTime.AddCheatTimeOffset(3600);   // pretend the clock jumped forward 1 hour
GameTime.GetCheatTimeOffset();
```

Compiled only under `DEVELOPMENT_BUILD || UNITY_EDITOR`; no-ops in release builds. Use
this to test `IntegrityChecker`'s clock-tamper detection (below) without touching the
actual OS clock.

---

## `ConfigGameTime` — tuning

A `GameConfig` asset (see [Config module](../Config/README.md)) referenced by `GameTime`
via `[ConfigRef]` on its `_configId` field.

| Field | Meaning |
|---|---|
| `TimeAPIs` | Ordered fallback list of time-API endpoints (`url`, response `key`, parse `format`). Tried in order each fetch until one succeeds. |
| `RequireInitFetch` | If true, `Initialize()` blocks until the first fetch succeeds. |
| `TimeTickInterval` | Seconds between `EventTimeTick` dispatches (and date-change checks). |
| `FetchInterval` | Seconds between periodic re-fetches. |
| `FetchTimeout` | Per-request timeout (seconds) for each time-API call. |
| `EnableIntegrityCheck` | Toggles `IntegrityChecker.RunAll` calls from `GameTime`. |
| `ClockTamperThresholdSeconds` | Drift (seconds) between raw system clock and verified `GameTime.UtcNow` before `ClockTampered` is flagged. |

---

## `IntegrityChecker` — runtime tamper signals

Static, no `MonoBehaviour` — runs cheap heuristic checks and caches the result so callers
can poll without re-running them. Driven automatically by `GameTime` (after first fetch
completes, and on every app refocus) when `ConfigGameTime.EnableIntegrityCheck` is true;
can also be called directly.

```csharp
using SiPVLib.AntiCheat;

if (IntegrityChecker.IsCompromised) { /* block leaderboard submission, etc. */ }

if (IntegrityChecker.HasViolation(IntegrityViolationType.ClockTampered)) { /* ... */ }

EventManager.Add<IntegrityViolationType>(IntegrityChecker.EventIntegrityViolation, OnViolation);
void OnViolation(IntegrityViolationType type) { CustomLog.LogWarning($"Integrity: {type}"); }
```

| Check | Platform | Technique |
|---|---|---|
| `DebuggerAttached` | All | `System.Diagnostics.Debugger.IsAttached` |
| `RootedOrJailbroken` | Android / iOS | File-existence heuristic against known root/jailbreak paths (`su` binaries, Cydia, etc.) |
| `EmulatorDetected` | Android | Substring match on `SystemInfo.deviceModel`/`graphicsDeviceName` against known emulator markers (Bluestacks, Nox, Genymotion, ...) |
| `ClockTampered` | All | Drift between raw `DateTime.UtcNow` and verified `GameTime.UtcNow` exceeding `ClockTamperThresholdSeconds` |

All checks are heuristic, not tamper-proof (there's no native plugin backing this) — treat
a positive as "worth extra scrutiny / soft-block", not cryptographic proof of cheating.
Root/jailbreak and emulator checks are skipped in the Editor and return `false` on
platforms other than Android/iOS.
