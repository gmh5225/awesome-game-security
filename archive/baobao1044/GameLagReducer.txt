# Changelog

All notable changes to Game Lag Reducer. Dates are local time.

## [0.2.0] — 2026-07-23

### Added — MaxFPS Flat tier (aggressive visual/poly reduction)

- **`Flat (Max FPS)` preset** (`preset-flat`): flat pixel shaders, compute/tessellation/geometry shaders off, MSAA off, low mips. Keeps the vertex shader so silhouettes stay correct (flat grey scene, game still runnable). Supersedes `Low (Max FPS)`, which uses the visually-breaking simplified-vertex path.
- **4 new tweak knobs** (`DisableGeometryShader`, `MsaaOff`, `SkipSmallDraws`, `SkipSmallDrawThreshold`) wired through the existing JSON IPC (C# model ↔ C++ struct ↔ hand-rolled parser). Exposed in the profile editor under an "Experimental" group.
- **D3D11 — geometry shader off**: `CreateGeometryShader` + `CreateGeometryShaderWithStreamOutput` → `E_FAIL` (engines fall back to the VS path).
- **D3D11 — MSAA off**: `CreateTexture2D` forces `SampleDesc.Count=1, Quality=0` on MSAA render targets.
- **D3D11 — experimental small-draw cull** (off by default): vtable-hooks 7 `ID3D11DeviceContext` draw methods and drops draws whose `IndexCount*InstanceCount` is below a threshold. ⚠️ Can hide UI text/quads — gated behind its own checkbox.
- **OpenGL — stage-aware GLSL substitution**: `glShaderSource` now classifies the uploaded source and substitutes a flat fragment shader (`SimplifyPixelShaders`) or a no-op compute shader (`SkipComputeShaders`). Vertex shaders are never substituted (keep silhouette). Also `glDisable(GL_MULTISAMPLE)` when `MsaaOff`.
- **Vulkan Phase 1**: intercepts `vkCreateGraphicsPipelines` to strip tessellation-control/eval + geometry stages (`disableTessellation`/`disableGeometryShader`) and force `rasterizationSamples=1` (`msaaOff`). Per-call mutable copies, fail-soft try/catch. No SPIR-V editing needed.
- **`glsl_classify`** pure module (`ClassifyGlsl` + `ExtractVersionLine`) with a unit-tested CMake target — the first real automated test in the project.
- **`Launcher.SmokeTests`** console exe: regression test for preset seeding + `Tweaks` JSON round-trip.
- **App icon** (`app.ico`, multi-res 16–256) generated via `tools/gen_icon.py` (Pillow, no network). Embedded in the exe and the window.

### Changed

- README: "How it works" diagram, new "Visual reduction — what it does and cannot" section (honest about the mesh-decimation limit), Phase 9 status row, arch entries for new files, GLSL test build instructions.
- CONTRIBUTING: project layout (+ `Launcher.SmokeTests/`, `HookDll/tests/`, `tools/`), IPC field count (~12 → ~16), warning that `ID3D11DeviceContext` draw-hook vtable indices are non-contiguous and must be re-verified against `d3d11.h`.
- Version bumped to `0.2.0` (csproj + header + footer). Stale "Phase 7" footer text replaced.

### Known limitations

- **Vulkan Phase 2** (flat/no-op SPIR-V substitution) was **not** done — there is no SPIR-V toolchain (glslangValidator/SPIRV-Tools) on the build machine. Phase 1 (tess/GS strip + MSAA) still works without it.
- **Mesh / triangle decimation is not possible at the API-hook layer** — meshes are already on the GPU and constant-buffer layouts are engine-specific. Poly reduction here comes only from disabling tessellation/GS (removing amplification) plus the optional small-draw cull.
- Manual game testing (the real end-to-end hook verification) is user-side; no automated hook harness exists.

## [0.1.0] — 2026-07-21

Initial release. Launcher (WPF .NET 8) + HookDll (C++20) covering Phases 1–8:
launcher scaffold, game library, engine/anti-cheat detection, DLL injection,
IPC named pipe, DXGI/D3D11 vtable hooks (Present/ResizeBuffers/Create*Shader),
DXBC capture/cache/disasm, no-op compute replacement, FPS cap, render scale,
OpenGL/Vulkan IAT hooks (capture-only), profile system + presets, shader cache UI.

# Contributing to Game Lag Reducer

Thanks for your interest in improving Game Lag Reducer! This is a small open-source
project — contributions of all sizes are welcome.

## Project layout

```
Launcher/            — WPF .NET 8 (C#) app: UI, game library, profiles, injection, IPC
Launcher.SmokeTests/ — .NET 8 console: regression test for preset seeding + Tweaks round-trip
HookDll/             — C++20 native DLL: graphics hooks, shader capture, tweaks
HookDll/tests/       — unit tests (no GPU): test_glsl_classify.cpp
docs/superpowers/    — design specs + implementation plans
tools/               — build helpers (gen_icon.py regenerates the app icon)
```

The two halves communicate over a named pipe (`GameLagReducer_IPC`) with a
line-based JSON protocol. See `Launcher/Services/IpcServer.cs` and
`HookDll/src/ipc_client.cpp` for the wire format.

## Build

### Launcher (C#)
```bash
cd Launcher
dotnet build
dotnet run
```
Requires the **.NET 8 SDK**. The app runs as administrator (the manifest
requests it — needed for `CreateRemoteThread` injection).

### HookDll (C++)
```bash
cd HookDll
cmake -B build -A x64
cmake --build build --config Release
```
Requires **CMake 3.20+** and a **C++20 compiler** (MSVC recommended). The
DLL is output to `Launcher/bin/HookDll/Release/HookDll.dll` so the Launcher
finds it automatically.

## Design principles

This project follows a minimalist ("Ponytail") philosophy:

1. **YAGNI first** — don't add features nobody asked for. Defer until needed.
2. **Reuse before writing** — check the existing codebase before adding new code.
3. **Standard library / native platform before dependencies** — use Windows APIs
   (`d3dcompiler.dll`, `QueryPerformanceCounter`, `System.IO.Pipes`,
   `System.Text.Json`) instead of pulling in packages.
4. **No new dependencies without strong justification.** The project currently
   has **zero** third-party dependencies. Keep it that way if possible.
5. **Minimal code** — the best code is the code that never gets written. Cut
   wrappers, boilerplate, and gold-plating. But **never** cut error handling,
   validation, or security.

When proposing a change, explain which rung of this ladder you stopped at and why.

## Hooking techniques (for HookDll contributors)

- **D3D11**: vtable patching. COM vtables are process-global, so patching a
  dummy device's vtable affects all instances. See `src/d3d11_hook.cpp`.
  Vtable indices are verified against `d3d11.h` in the Windows SDK — see the
  comments in `Install()` for the exact layout.
- **OpenGL / Vulkan**: IAT patching. These are free exported functions (not COM),
  so we walk the host EXE's PE import table and overwrite the IAT slot. See
  `src/iat_hook.cpp`. Limitation: only catches statically-imported functions.
- **D3D11 draw-call cull**: `ID3D11DeviceContext` draw methods are vtable-hooked
  for the experimental small-draw cull. ⚠️ These indices are **non-contiguous**:
  `DrawIndexed=12, Draw=13, DrawIndexedInstanced=20, DrawInstanced=21,
  DrawAuto=38, DrawIndexedInstancedIndirect=39, DrawInstancedIndirect=40`
  (verified against `d3d11.h` 10.0.26100.0). If you touch these, **re-verify
  against the installed SDK header** before changing the index numbers — a
  contiguous assumption will silently hook the wrong methods.

## IPC protocol (keep in sync!)

The pipe name and Tweaks struct fields must match between C# and C++:

- **Pipe name**: `GameLagReducer_IPC` (`Launcher/Services/IpcServer.cs` ↔ `HookDll/src/ipc_client.cpp`)
- **Tweaks fields**: `Launcher/Models/Tweaks.cs` ↔ `HookDll/include/ipc_client.h`

If you add a tweak knob, update **both** sides and the hand-rolled JSON parser
in `ipc_client.cpp` (no JSON library — by design, currently ~16 fields). Also
add a C# unit assertion in `Launcher.SmokeTests/Program.cs` if it should be
preset-exercised, and a `glsl_classify`-style pure test in `HookDll/tests/`
where possible.

## Shader cache

Captured shaders live in `%AppData%\GameLagReducer\shaders\`:
- `<hash>.bin` — original bytecode (FNV-1a 64-bit hash of the bytecode)
- `<hash>.asm` — disassembled DXBC text (D3D11 only, via `D3DDisassemble`)
- `<hash>.repl.bin` — user-provided replacement (optional; wins over built-in tweaks)

To test shader replacement: launch a game, find the captured `.bin`, disassemble
it externally, modify, recompile to bytecode, save as `<hash>.repl.bin`, relaunch.

## Anti-cheat

**Injecting into games with anti-cheat can cause permanent bans.** The launcher
warns users, but contributors must not add features that bypass anti-cheat
detection or hide injection. The goal is a transparent, user-consented tool.

## Pull requests

1. Keep PRs focused — one feature or fix per PR.
2. Test on at least one real game (a simple Unity/Unreal title is fine).
3. Don't introduce new third-party dependencies without discussion.
4. Update `README.md` and this file if you change the architecture or protocol.

    const D3D11_TEXTURE2D_DESC*, const D3D11_SUBRESOURCE_DATA*, ID3D11Texture2D**);

// ID3D11DeviceContext draw-call typedefs (for experimental small-draw cull).
// Vtable indices verified against d3d11.h (10.0.26100.0): the DeviceContext
// virtual order is NOT contiguous — indices are 12,13,20,21,38,39,40.
using ID3D11DeviceContext_DrawIndexed_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*, UINT, UINT, INT);
using ID3D11DeviceContext_Draw_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*, UINT, UINT);
using ID3D11DeviceContext_DrawIndexedInstanced_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*, UINT, UINT, UINT, INT, UINT);
using ID3D11DeviceContext_DrawInstanced_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*, UINT, UINT, UINT, UINT);
using ID3D11DeviceContext_DrawAuto_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*);
using ID3D11DeviceContext_DrawIndexedInstancedIndirect_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*, ID3D11Buffer*, UINT);
using ID3D11DeviceContext_DrawInstancedIndirect_fn = void(STDMETHODCALLTYPE*)(ID3D11DeviceContext*, ID3D11Buffer*, UINT);

/// D3D11/DXGI hook via vtable patching. No external dependency (no MinHook).
///
/// Technique: create a dummy device+swapchain, read the vtable pointers, and
/// overwrite the vtable entries we care about with our own trampolines. Because
/// D3D11 COM interfaces share a single vtable per interface type across all
/// instances, patching the dummy's vtable affects every device the game creates
/// afterwards.
///
/// Phase 3 hooks:
///   - IDXGISwapChain::Present        → log frame, apply FPS cap (Phase 5)
///   - IDXGISwapChain::ResizeBuffers   → apply render scale (Phase 5)
///   - ID3D11Device::CreateVertexShader / CreatePixelShader / CreateComputeShader
///     → capture shader bytecode (Phase 4 will modify + replace)
///
/// All hooks log via IPC and forward to the original (no behavior change yet).
class D3D11Hook {
public:
    /// Installs all D3D11/DXGI hooks. Returns true on success.
    /// Must be called after the IPC client has connected (so logs reach the Launcher).
    static bool Install();

    /// Removes all hooks. Called on DLL detach.
    static void Uninstall();

    /// True if hooks are currently installed.
    static bool IsInstalled();
};

}  // namespace hookdll

#pragma once

#include <string>
#include <string_view>

namespace hookdll {

/// GLSL shader stage classification (pure, testable).
/// Used by opengl_hook to decide whether a glShaderSource upload is a
/// fragment / vertex / compute shader so it can substitute a flat or no-op
/// replacement. Unknown sources are never modified.
enum class GlslStage {
    Vertex,
    Fragment,
    Compute,
    Unknown,
};

/// Classifies a GLSL source string into a shader stage by keyword.
/// Order matters: compute is detected first (layout(local_size...), then
/// vertex (gl_Position), then fragment (gl_FragColor or "out vec4").
/// Returns Unknown if no marker matches.
GlslStage ClassifyGlsl(std::string_view src);

/// Extracts the leading "#version ..." line (with trailing newline) from a
/// GLSL source string. Returns "" if there is no #version directive. The
/// returned line is prepended to a replacement shader so the GLSL version
/// matches the original (avoids redefinition / feature-level mismatches).
std::string ExtractVersionLine(std::string_view src);

}  // namespace hookdll

`HookDll/include/iat_hook.h`:

```h
#pragma once

#include <windows.h>

namespace hookdll {

/// IAT (Import Address Table) hooking utility.
///
/// Unlike COM vtable hooking (used for D3D11 in Phase 3), OpenGL and Vulkan
/// expose free functions via DLL exports. The cleanest dependency-free way to
/// intercept them is to patch the host module's IAT: each imported function
/// has a slot in the IAT that the loader fills with the resolved address. We
/// overwrite that slot with our trampoline.
///
/// Limitations (honest scope, Ponytail):
///   - Only catches calls that go through the import table. If the game calls
///     GetProcAddress("wglSwapBuffers") at runtime and invokes the result
///     directly, the hook is bypassed. This is rare for wglSwapBuffers (games
///     usually import it statically) but possible.
///   - Patches only the host EXE's IAT by default. DLL-loaded imports are not
///     patched. Sufficient for the common case.
///
/// For the common case (statically imported wglSwapBuffers / vulkan-1.dll),
/// IAT hooking is ~30 lines, no dependency, and safe (no code bytes modified).
class IatHook {
public:
    /// Patches the IAT of `module` so that imports of `importDll` (e.g.
    /// "gdi32.dll") for function `funcName` (e.g. "wglSwapBuffers") are
    /// redirected to `newFn`. Saves the original to *originalOut.
    /// Returns true on success.
    static bool Install(HMODULE module, const char* importDll,
        const char* funcName, void* newFn, void** originalOut);

    /// Restores the original IAT entry. Safe to call even if Install failed.
    static void Remove(HMODULE module, const char* importDll,
        const char* funcName, void* original);
};

}  // namespace hookdll

`HookDll/include/ipc_client.h`:

```h
#pragma once

#include <string>

namespace hookdll {

/// Tweak profile received from the Launcher via IPC. Mirrors Launcher/Models/Tweaks.cs.
/// Fields are kept simple (POD) so they can be parsed from JSON by hand without a
/// JSON library dependency (Phase 2 — YAGNI: only the fields we actually use).
struct Tweaks {
    int renderScaleMode = 0;   // 0=Off, 1=Scale, 2=Custom
    double renderScale = 0.75;
    int customWidth = 1280;
    int customHeight = 720;
    int fpsCap = 0;
    bool simplifyVertexShaders = false;
    bool simplifyPixelShaders = false;
    bool skipComputeShaders = false;
    double lodBias = 0.0;
    bool disableTessellation = false;
    bool disablePostProcessing = false;
    int maxTextureMipLevel = 0;
    bool disableGeometryShader = false;
    bool msaaOff = false;
    bool skipSmallDraws = false;
    int skipSmallDrawThreshold = 256;
};

/// IPC client. Connects to the Launcher's named pipe, receives the tweak profile,
/// and sends back log/telemetry lines. Mirrors Launcher/Services/IpcServer.cs.
///
/// Protocol (line-based JSON, see IpcServer.cs):
///   Launcher → DLL:  {"cmd":"profile","profile":{...}}
///   Launcher → DLL:  {"cmd":"ping"}
///   DLL → Launcher:  {"log":"message"}
///   DLL → Launcher:  {"ack":"profile"}
class IpcClient {
public:
    /// Connects to the Launcher pipe. Blocks up to timeoutMs.
    /// Returns true on success. Safe to call from DllMain (separate thread).
    static bool Connect(int timeoutMs = 5000);

    /// Disconnects and closes the pipe.
    static void Disconnect();

    /// Returns true if currently connected.
    static bool IsConnected();

    /// Blocks until a profile message is received, or returns false on timeout/disconnect.
    /// The received tweaks are stored in `out`.
    static bool WaitForProfile(Tweaks& out, int timeoutMs = 10000);

    /// Sends a log line to the Launcher. No-op if not connected.
    static void SendLog(const std::string& message);

    /// Sends an acknowledgment that the profile was applied.
    static void SendAck(const std::string& what);

    /// Sends a telemetry value (e.g. FPS).
    static void SendTelemetry(const std::string& name, double value);
};

}  // namespace hookdll

#define IPC_LOG(msg) hookdll::IpcClient::SendLog(msg)

///   - GLSL capture via glShaderSource: reuses ShaderCache (saves source text).
///   - GLSL modification: flat fragment + no-op compute substitution (MaxFPS

///   - MSAA off: glDisable(GL_MULTISAMPLE) when msaaOff is set.
class OpenGLHook {
public:

    /// opengl32!glShaderSource. Idempotent: calling twice is safe.
    /// Returns true if at least one hook was installed.
    static bool Install();

    static void Uninstall();
};

}  // namespace hookdll

/// 64-bit FNV-1a hash. Fast, no dependency (SHA256 would be overkill for a cache
/// key — collisions are irrelevant here: worst case two shaders alias and we pick
/// the wrong cached replacement, which the user would notice and re-capture).
/// FNV-1a is ~5 lines and good enough for bytecode fingerprinting.
uint64_t HashBytecode(const void* data, size_t size);

/// Shader cache. Stores original bytecode + disassembled text on disk so the user
/// (and future phases) can inspect what a game's shaders look like.
///
/// Layout:
///   %AppData%\GameLagReducer\shaders\<hash>.bin   — original bytecode
///   %AppData%\GameLagReducer\shaders\<hash>.asm  — disassembled DXBC text
///
/// Phase 4 uses this for capture + inspection. Replacement lookup is by hash:
/// if a `<hash>.repl.bin` file exists (dropped there by the user or a future
/// editor), the hook will substitute it for the original bytecode.
class ShaderCache {
public:
    /// Returns the singleton cache root directory, creating it if needed.
    /// e.g. C:\Users\<user>\AppData\Roaming\GameLagReducer\shaders
    static std::string CacheDir();

    /// Saves the original bytecode to <hash>.bin. No-op if already cached.
    /// Returns the hash (also used as the cache key).
    static uint64_t SaveOriginal(const void* bytecode, size_t size);

    /// Saves disassembled text to <hash>.asm. Overwrites if present (disasm is
    /// deterministic for a given bytecode).
    static void SaveDisasm(uint64_t hash, const std::string& text);

    /// If a replacement bytecode file <hash>.repl.bin exists, loads it into out
    /// and returns true. Used by the hook to swap in a modified shader.
    static bool LoadReplacement(uint64_t hash, std::vector<uint8_t>& out);

`HookDll/include/shader_modifier.h`:

```h
#pragma once

#include <cstdint>
#include <vector>
#include <windows.h>
#include "ipc_client.h"  // for Tweaks

namespace hookdll {

/// Shader stage. Matches the three D3D11 Create*Shader entry points we hook.
enum class ShaderStage {
    Vertex,
    Pixel,
    Compute,
};

/// Result of processing a shader through the modifier pipeline.
struct ProcessedShader {
    /// Pointer to the bytecode to actually pass to Create*Shader. Points either
    /// to the original buffer (no modification) or to `replacement` (if modified).
    const void* bytecode = nullptr;
    SIZE_T bytecodeLength = 0;

    /// Owned replacement buffer (only used when we generate a new shader).
    std::vector<uint8_t> replacement;

    /// True if the shader was modified (for logging/telemetry).
    bool modified = false;

    /// Hash of the original bytecode (for cache lookup + logging).
    uint64_t originalHash = 0;
};

/// Shader modifier. The core of Phase 4.
///
/// Pipeline (applied per Create*Shader call):
///   1. Hash the incoming bytecode.
///   2. Save original to cache (if not already present).
///   3. Disassemble via D3DDisassemble and save .asm (first capture only).
///   4. Check for a user-provided <hash>.repl.bin → use it if present.
///   5. Apply built-in tweaks based on the active profile:
///        - SkipComputeShaders → replace compute with a no-op compute shader.
///        - SimplifyVertexShaders → replace vertex with a pass-through shader.
///        - SimplifyPixelShaders / DisablePostProcessing → replace pixel with
///          a flat-color pass-through shader.
///
/// All replacement shaders are pre-compiled once and cached in memory, so the
/// cost per Create*Shader call is a hash + a couple of file existence checks.
class ShaderModifier {
public:
    /// Sets the active tweak profile. Must be called before any Process() call.
    /// Called once from dllmain after the profile is received via IPC.
    static void SetTweaks(const Tweaks& tweaks);

    /// Processes an incoming shader. Returns a ProcessedShader whose `bytecode`
    /// and `bytecodeLength` should be forwarded to the original Create*Shader.
    static ProcessedShader Process(ShaderStage stage,
        const void* bytecode, SIZE_T bytecodeLength);

private:
    /// Lazily compiles the no-op compute shader (cached after first call).
    /// Returns nullptr if compilation fails (D3DCompile error).
    static const std::vector<uint8_t>& GetNoopComputeBytecode();

    /// Lazily compiles the simplified vertex shader (cached after first call).
    static const std::vector<uint8_t>& GetSimpleVertexBytecode();

    /// Lazily compiles the simplified pixel shader (cached after first call).
    static const std::vector<uint8_t>& GetSimplePixelBytecode();
};

}  // namespace hookdll

```h
#pragma once

#include <windows.h>
#include "ipc_client.h"  // for Tweaks

namespace hookdll {

/// Applies runtime tweaks that don't involve shader bytecode editing.
/// Phase 5 implements:
///   - FPS cap: throttle Present() to a target frame time.
///   - Render scale: shrink the backbuffer in ResizeBuffers().
///
/// Both tweaks use only native Win32 timing (QueryPerformanceCounter) and
/// arithmetic — no dependencies. The Tweaks struct is shared with
/// ShaderModifier (set once via SetTweaks in dllmain).
///
/// Thread-safety: SetTweaks copies under a mutex; the per-frame reads in
/// Present/ResizeBuffers take the same mutex. The lock is held for a few
/// nanoseconds (POD copy), so contention with the render thread is negligible.
class TweakApplier {
public:
    /// Sets the active tweak profile. Called once from dllmain after IPC.
    static void SetTweaks(const Tweaks& tweaks);

    /// Returns a snapshot of the active tweaks. Called from hook trampolines.
    static Tweaks GetTweaks();

    /// Called at the top of HookPresent. If fpsCap > 0, sleeps/spins until the
    /// target frame time has elapsed since the last Present, then returns.
    /// SyncInterval and Flags are unchanged — we only throttle timing.
    static void ThrottleForFpsCap();

    /// Called from HookResizeBuffers. If renderScaleMode != Off, rewrites
    /// Width/Height in place before the original ResizeBuffers is called.
    /// The game's own backbuffer is created smaller; the swapchain's
    /// DXGI_SCALING behavior upscales to the window on Present.
    static void ApplyRenderScale(UINT& Width, UINT& Height);
};

}  // namespace hookdll

#pragma once

#include <windows.h>
#include <cstdint>

namespace hookdll {

/// Vulkan hook (Phase 6).
///
/// Vulkan loading is two-level: the loader (vulkan-1.dll) exports
/// vkGetInstanceProcAddr / vkGetDeviceProcAddr, and games resolve every other
/// function through those. So IAT-patching vkCreateShaderModule directly almost
/// never fires (games don't statically import it). Instead we IAT-hook
/// vkGetDeviceProcAddr and intercept the lookup for vkCreateShaderModule and
/// vkQueuePresentKHR, returning our trampolines.
///
/// Ponytail scope:
///   - FPS cap via vkQueuePresentKHR throttle: reuses TweakApplier.
///   - SPIR-V capture via vkCreateShaderModule: reuses ShaderCache (saves the
///     raw SPIR-V bytecode; .asm disasm is deferred — would need SPIRV-Tools,
///     a heavy dependency, YAGNI for Phase 6).
///   - Pipeline patching (Phase 1, MaxFPS Flat): intercept
///     vkCreateGraphicsPipelines to strip tessellation + geometry stages
///     (disableTessellation / disableGeometryShader) and force
///     rasterizationSamples=1 (msaaOff). No SPIR-V editing required.
///   - SPIR-V modification (Phase 2): deferred — SPIRV-Tools is a large dep.
///
/// Vulkan types are declared inline (no Vulkan SDK header on the build
/// machine). The ABI is stable and documented in the Vulkan spec.
class VulkanHook {
public:
    /// Patches the host EXE's IAT for vulkan-1!vkGetDeviceProcAddr. Idempotent.
    /// Returns true if the hook was installed (false if the game doesn't use
    /// Vulkan or imports the loader dynamically).
    static bool Install();

    /// Restores the original IAT entry. Called on DLL unload.
    static void Uninstall();
};

}  // namespace hookdll

// ---- Sampler LOD bias / max mip ----
// Patches D3D11_SAMPLER_DESC before the game's sampler is created. Positive
// MipLODBias makes the GPU pick lower-detail mips sooner (less texture bandwidth);
// MaxLOD clamps the highest mip level used. Both reduce texture fetch cost.
HRESULT STDMETHODCALLTYPE HookCreateSamplerState(ID3D11Device* This,
    const D3D11_SAMPLER_DESC* pDesc, ID3D11SamplerState** ppSamplerState) {
    Tweaks t = TweakApplier::GetTweaks();
    if (pDesc && (t.lodBias != 0.0 || t.maxTextureMipLevel > 0)) {
        D3D11_SAMPLER_DESC patched = *pDesc;
        if (t.lodBias != 0.0) {
            patched.MipLODBias += (FLOAT)t.lodBias;
        }
        if (t.maxTextureMipLevel > 0) {
            // MaxLOD is a float; D3D11_FLT_MAX means "no limit". Clamp to the user value.
            patched.MaxLOD = (FLOAT)t.maxTextureMipLevel;
            if (patched.MinLOD > patched.MaxLOD) patched.MinLOD = patched.MaxLOD;
        }
        static std::atomic<int> logged{0};
        if (logged++ < 3) {
            char buf[160];
            std::snprintf(buf, sizeof(buf),
                "[D3D11] CreateSamplerState: MipLODBias %.2f→%.2f, MaxLOD %.1f→%.1f",
                pDesc->MipLODBias, patched.MipLODBias, pDesc->MaxLOD, patched.MaxLOD);
            IPC_LOG(buf);
        }
        return g_origCreateSamplerState(This, &patched, ppSamplerState);
    }
    return g_origCreateSamplerState(This, pDesc, ppSamplerState);
}

// ---- Vtable patching helpers ----

/// Patches a single vtable slot. Saves the original to *originalOut, then writes
/// the new function pointer. Uses VirtualProtect to make the vtable writable.
/// COM vtables are in read-only memory by default.
static bool PatchVtable(void** vtable, int index, void* newFn, void** originalOut) {
    void* target = vtable + index;
    DWORD oldProtect = 0;
    if (!VirtualProtect(target, sizeof(void*), PAGE_READWRITE, &oldProtect)) {
        HOOK_LOG("PatchVtable: VirtualProtect failed.");
        return false;
    }
    *originalOut = vtable[index];   // save original
    vtable[index] = newFn;          // install hook
    DWORD dummy;
    VirtualProtect(target, sizeof(void*), oldProtect, &dummy);
    return true;
}

/// Creates a dummy D3D11 device + swapchain to obtain vtable pointers.
/// The vtable is shared across all instances of the same COM interface, so
/// patching the dummy's vtable affects every device the game creates later.
static bool CreateDummyDeviceAndSwapChain(
    ID3D11Device** ppDevice, IDXGISwapChain** ppSwapChain) {
    // Minimal window for the dummy swapchain (invisible).
    WNDCLASSEXA wc = {};
    wc.lpfnWndProc = DefWindowProcA;
    wc.hInstance = GetModuleHandleA(nullptr);
    wc.lpszClassName = "GameLagReducerDummy";
    RegisterClassExA(&wc);
    HWND hwnd = CreateWindowExA(0, "GameLagReducerDummy", "", WS_OVERLAPPED,
        0, 0, 1, 1, nullptr, nullptr, wc.hInstance, nullptr);
    if (!hwnd) {
        HOOK_LOG("CreateDummyDevice: CreateWindow failed.");
        return false;
    }

    DXGI_SWAP_CHAIN_DESC scd = {};
    scd.BufferCount = 1;
    scd.BufferDesc.Width = 1;
    scd.BufferDesc.Height = 1;
    scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;
    scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;
    scd.OutputWindow = hwnd;
    scd.SampleDesc.Count = 1;
    scd.Windowed = TRUE;

    D3D_FEATURE_LEVEL fl;
    HRESULT hr = D3D11CreateDeviceAndSwapChain(
        nullptr, D3D_DRIVER_TYPE_HARDWARE, nullptr, 0,
        nullptr, 0, D3D11_SDK_VERSION,
        &scd, ppSwapChain, ppDevice, &fl, nullptr);

    DestroyWindow(hwnd);
    UnregisterClassA("GameLagReducerDummy", wc.hInstance);

    if (FAILED(hr)) {
        HOOK_LOG("CreateDummyDevice: D3D11CreateDeviceAndSwapChain failed.");
        return false;
    }
    return true;
}

bool D3D11Hook::Install() {
    std::lock_guard<std::mutex> lock(g_hookMtx);
    if (g_installed) return true;

    HOOK_LOG("D3D11Hook: installing...");
    IPC_LOG("[D3D11] Installing hooks...");

    // Create dummy device + swapchain to get vtables.
    ID3D11Device* device = nullptr;
    IDXGISwapChain* swapChain = nullptr;
    if (!CreateDummyDeviceAndSwapChain(&device, &swapChain)) {
        IPC_LOG("[D3D11] Failed to create dummy device — D3D11 not available?");
        return false;
    }

    // Helper to release the dummy objects. The vtable stays patched (process-global).
    auto releaseDummy = [&] {
        if (swapChain) swapChain->Release();
        if (device) device->Release();
    };

    // IDXGISwapChain vtable: Present is index 8, ResizeBuffers is index 13.
    // (Standard DXGI vtable order: QueryInterface, AddRef, Release, SetPrivateData,
    //  SetPrivateDataInterface, GetPrivateData, GetParent, GetDevice,
    //  Present, GetBuffer, SetFullscreenState, GetFullscreenState, GetDesc,
    //  ResizeBuffers, ResizeTarget, ...)
    void** swapVtable = *reinterpret_cast<void***>(swapChain);
    if (!PatchVtable(swapVtable, 8, (void*)HookPresent, (void**)&g_origPresent)) {
        IPC_LOG("[D3D11] Failed to hook Present.");
        releaseDummy();
        return false;
    }
    if (!PatchVtable(swapVtable, 13, (void*)HookResizeBuffers, (void**)&g_origResizeBuffers)) {
        IPC_LOG("[D3D11] Failed to hook ResizeBuffers.");
        releaseDummy();
        return false;
    }

    // ID3D11Device vtable (inherits IUnknown: QueryInterface=0, AddRef=1, Release=2):
    //   CreateBuffer(3), CreateTexture1D(4), CreateTexture2D(5), CreateTexture3D(6),
    //   CreateShaderResourceView(7), CreateUnorderedAccessView(8),
    //   CreateRenderTargetView(9), CreateDepthStencilView(10), CreateInputLayout(11),
    //   CreateVertexShader(12), CreateGeometryShader(13),
    //   CreateGeometryShaderWithStreamOutput(14), CreatePixelShader(15),
    //   CreateHullShader(16), CreateDomainShader(17), CreateComputeShader(18).
    // (Verified against d3d11.h — ID3D11Device : public IUnknown.)
    void** devVtable = *reinterpret_cast<void***>(device);
    if (!PatchVtable(devVtable, 12, (void*)HookCreateVertexShader, (void**)&g_origCreateVertexShader) ||
        !PatchVtable(devVtable, 15, (void*)HookCreatePixelShader, (void**)&g_origCreatePixelShader) ||
        !PatchVtable(devVtable, 18, (void*)HookCreateComputeShader, (void**)&g_origCreateComputeShader) ||
        !PatchVtable(devVtable, 16, (void*)HookCreateHullShader, (void**)&g_origCreateHullShader) ||
        !PatchVtable(devVtable, 17, (void*)HookCreateDomainShader, (void**)&g_origCreateDomainShader) ||
        !PatchVtable(devVtable, 23, (void*)HookCreateSamplerState, (void**)&g_origCreateSamplerState) ||
        !PatchVtable(devVtable, 13, (void*)HookCreateGeometryShader, (void**)&g_origCreateGeometryShader) ||
        !PatchVtable(devVtable, 14, (void*)HookCreateGeometryShaderWithStreamOutput, (void**)&g_origCreateGSWithSO) ||
        !PatchVtable(devVtable, 5, (void*)HookCreateTexture2D, (void**)&g_origCreateTexture2D)) {
        IPC_LOG("[D3D11] Failed to hook Create*Shader/Sampler/Texture.");
        releaseDummy();
        return false;
    }

    // Small-draw cull: hook the immediate context's draw calls. Best-effort —
    // partial failure only disables the cull, not the whole install. Indices
    // verified against d3d11.h: DeviceContext DrawIndexed=12, Draw=13,
    // DrawIndexedInstanced=20, DrawInstanced=21, DrawAuto=38,
    // DrawIndexedInstancedIndirect=39, DrawInstancedIndirect=40.
    ID3D11DeviceContext* ctx = nullptr;
    device->GetImmediateContext(&ctx);
    if (ctx) {
        void** ctxVtable = *reinterpret_cast<void***>(ctx);
        bool ctxOk =
            PatchVtable(ctxVtable, 12, (void*)HookDrawIndexed, (void**)&g_origDrawIndexed) &&
            PatchVtable(ctxVtable, 13, (void*)HookDraw, (void**)&g_origDraw) &&
            PatchVtable(ctxVtable, 20, (void*)HookDrawIndexedInstanced, (void**)&g_origDrawIndexedInstanced) &&
            PatchVtable(ctxVtable, 21, (void*)HookDrawInstanced, (void**)&g_origDrawInstanced) &&
            PatchVtable(ctxVtable, 38, (void*)HookDrawAuto, (void**)&g_origDrawAuto) &&
            PatchVtable(ctxVtable, 39, (void*)HookDrawIndexedInstancedIndirect, (void**)&g_origDrawIndexedInstancedIndirect) &&
            PatchVtable(ctxVtable, 40, (void*)HookDrawInstancedIndirect, (void**)&g_origDrawInstancedIndirect);
        if (ctxOk) {
            IPC_LOG("[D3D11] context draw-hooks installed (small-draw cull available).");
        } else {
            IPC_LOG("[D3D11] context draw-hook partial/failed — small-draw cull inactive.");
        }
        ctx->Release();
    } else {
        IPC_LOG("[D3D11] no immediate context — small-draw cull unavailable.");
    }

    g_installed = true;
    IPC_LOG("[D3D11] Hooks installed (Present, ResizeBuffers, Create*Shader, Sampler, Texture2D, GS, Draw cull).");
    HOOK_LOG("D3D11Hook: installed.");
    releaseDummy();
    return g_installed;
}

void D3D11Hook::Uninstall() {
    std::lock_guard<std::mutex> lock(g_hookMtx);
    if (!g_installed) return;

    // Restore original vtable entries. We need a live instance to get the vtable,
    // but since the game may have closed, we just leave the hooks in place —
    // process is exiting anyway. This is a no-op for safety.
    g_installed = false;
    HOOK_LOG("D3D11Hook: uninstalled (process exiting, vtables not restored).");
}

bool D3D11Hook::IsInstalled() {
    return g_installed;
}

/// Background worker thread: connects to the Launcher IPC, waits for the profile,
/// then installs D3D11 hooks. Phase 4+ will apply the profile to installed hooks.
static void IpcWorkerThread() {
    HOOK_LOG("IPC worker: connecting to Launcher...");
    if (!hookdll::IpcClient::Connect(5000)) {
        HOOK_LOG("IPC worker: could not connect — running without profile (no tweaks).");
        return;
    }

    hookdll::Tweaks tweaks;
    if (hookdll::IpcClient::WaitForProfile(tweaks, 10000)) {
        HOOK_LOG("IPC worker: profile received and parsed.");
        IPC_LOG("HookDll attached and profile received.");
        hookdll::IpcClient::SendAck("profile");

        // Phase 4/5: hand the profile to the shader modifier and the tweak
        // applier before installing hooks, so the first hook call already
        // sees the active tweaks.
        hookdll::ShaderModifier::SetTweaks(tweaks);
        hookdll::TweakApplier::SetTweaks(tweaks);

        if (hookdll::D3D11Hook::Install()) {
            HOOK_LOG("D3D11 hooks installed.");
        } else {
            HOOK_LOG("D3D11 hook install failed — game may not use D3D11.");
        }

        // Phase 6: also attempt OpenGL and Vulkan hooks. A game uses exactly
        // one graphics API, so at most one of {D3D11, OpenGL, Vulkan} will
        // install successfully; the others log "no import found" and return
        // false. All three are attempted unconditionally — cheap and safe.
        if (hookdll::OpenGLHook::Install()) {
            HOOK_LOG("OpenGL hooks installed.");
        }
        if (hookdll::VulkanHook::Install()) {
            HOOK_LOG("Vulkan hooks installed.");
        }
    } else {
        HOOK_LOG("IPC worker: no profile received (timeout). Running idle.");
    }

    // Keep the IPC connection alive so the Launcher can detect when we exit.
    // The connection closes automatically when the process exits.
}

/// DLL entry point. Called when the DLL is loaded via LoadLibrary (injection).
/// Phase 2: connect IPC + receive profile in a background thread.
BOOL APIENTRY DllMain(HMODULE hModule, DWORD reason, LPVOID) {
    if (reason == DLL_PROCESS_ATTACH) {
        DisableThreadLibraryCalls(hModule);
        HOOK_LOG("=== HookDll attached to process ===");
        HOOK_LOG_HEX("Module handle", hModule);

        // Spawn the IPC worker thread. We must NOT block in DllMain.
        // The thread connects to the Launcher pipe and waits for the profile.
        std::thread(IpcWorkerThread).detach();
    } else if (reason == DLL_PROCESS_DETACH) {
        HOOK_LOG("=== HookDll detaching ===");
        hookdll::D3D11Hook::Uninstall();
        hookdll::OpenGLHook::Uninstall();
        hookdll::VulkanHook::Uninstall();
        hookdll::IpcClient::Disconnect();
    }
    return TRUE;
}

#include "glsl_classify.h"

namespace hookdll {

// Classify by keyword. Order matters:
//   1. Compute: layout(local_size...). Must be checked first because a compute
//      shader has no gl_Position/gl_FragColor and would otherwise be Unknown.
//   2. Vertex: gl_Position. Checked before fragment so a vertex shader that
//      also declares an `out vec4` (e.g. transform-feedback varying) is
//      correctly identified as vertex, not fragment.
//   3. Fragment: gl_FragColor (legacy) or an `out vec4` output declaration
//      (modern #version >= 130).
GlslStage ClassifyGlsl(std::string_view src) {
    if (src.find("layout(local_size") != std::string_view::npos) return GlslStage::Compute;
    if (src.find("gl_Position") != std::string_view::npos) return GlslStage::Vertex;
    if (src.find("gl_FragColor") != std::string_view::npos) return GlslStage::Fragment;
    if (src.find("out vec4") != std::string_view::npos) return GlslStage::Fragment;
    return GlslStage::Unknown;
}

// Extract the leading "#version ..." directive (the first one, if present).
// Returns the line including its trailing newline, or "" if there is none.
// The replacement shaders prepend this so the GLSL version / profile matches
// the original — avoids feature-level mismatches and #version redefinition.
std::string ExtractVersionLine(std::string_view src) {
    auto p = src.find("#version");
    if (p == std::string_view::npos) return "";
    auto e = src.find('\n', p);
    auto line = src.substr(p, (e == std::string_view::npos ? std::string_view::npos : e - p));
    return std::string(line) + "\n";
}

}  // namespace hookdll

namespace hookdll {

// Walks the PE header of `module` to find the IAT entry for
// `importDll`!`funcName`. Returns a pointer to the IAT slot (the slot that
// holds the resolved function pointer), or nullptr if not found.
static void** FindIatSlot(HMODULE module, const char* importDll, const char* funcName) {
    if (!module) return nullptr;

    auto base = reinterpret_cast<BYTE*>(module);
    auto dos = reinterpret_cast<IMAGE_DOS_HEADER*>(base);
    if (dos->e_magic != IMAGE_DOS_SIGNATURE) return nullptr;

    auto nt = reinterpret_cast<IMAGE_NT_HEADERS*>(base + dos->e_lfanew);
    if (nt->Signature != IMAGE_NT_SIGNATURE) return nullptr;

    auto& importsDir = nt->OptionalHeader.DataDirectory[IMAGE_DIRECTORY_ENTRY_IMPORT];
    if (importsDir.Size == 0) return nullptr;

    auto importDesc = reinterpret_cast<IMAGE_IMPORT_DESCRIPTOR*>(
        base + importsDir.VirtualAddress);

    for (; importDesc->Name != 0; ++importDesc) {
        const char* dllName = reinterpret_cast<const char*>(base + importDesc->Name);
        // Case-insensitive compare (Windows DLL names are case-insensitive).
        if (_stricmp(dllName, importDll) != 0) continue;

        auto thunkRef = reinterpret_cast<IMAGE_THUNK_DATA*>(
            base + (importDesc->OriginalFirstThunk
                ? importDesc->OriginalFirstThunk : importDesc->FirstThunk));
        auto funcRef = reinterpret_cast<IMAGE_THUNK_DATA*>(base + importDesc->FirstThunk);

        for (; thunkRef->u1.AddressOfData != 0; ++thunkRef, ++funcRef) {
            // Skip ordinal imports (high bit set).
            if (IMAGE_SNAP_BY_ORDINAL(thunkRef->u1.Ordinal)) continue;

            auto byName = reinterpret_cast<IMAGE_IMPORT_BY_NAME*>(
                base + thunkRef->u1.AddressOfData);
            if (strcmp(byName->Name, funcName) == 0) {
                return reinterpret_cast<void**>(&funcRef->u1.Function);
            }
        }
    }
    return nullptr;
}

bool IatHook::Install(HMODULE module, const char* importDll,
    const char* funcName, void* newFn, void** originalOut) {
    void** slot = FindIatSlot(module, importDll, funcName);
    if (!slot) {
        HOOK_LOG(std::string("IatHook: ") + importDll + "!" + funcName + " not in IAT.");
        return false;
    }

    DWORD oldProtect = 0;
    if (!VirtualProtect(slot, sizeof(void*), PAGE_READWRITE, &oldProtect)) {
        HOOK_LOG("IatHook: VirtualProtect failed.");
        return false;
    }
    if (originalOut) *originalOut = *slot;
    *slot = newFn;
    DWORD dummy;
    VirtualProtect(slot, sizeof(void*), oldProtect, &dummy);

    HOOK_LOG_HEX(std::string("IatHook: patched ") + importDll + "!" + funcName, newFn);
    return true;
}

void IatHook::Remove(HMODULE module, const char* importDll,
    const char* funcName, void* original) {
    if (!original) return;
    void** slot = FindIatSlot(module, importDll, funcName);
    if (!slot) return;

    DWORD oldProtect = 0;
    if (!VirtualProtect(slot, sizeof(void*), PAGE_READWRITE, &oldProtect)) return;
    *slot = original;
    DWORD dummy;
    VirtualProtect(slot, sizeof(void*), oldProtect, &dummy);
}

#include "ipc_client.h"
#include "logger.h"

#include <windows.h>
#include <string>
#include <thread>
#include <mutex>
#include <atomic>

namespace hookdll {

// Pipe name must match Launcher/Services/IpcServer.cs
static const char* kPipeName = "\\\\.\\pipe\\GameLagReducer_IPC";

static HANDLE g_pipe = nullptr;
static std::mutex g_writeMtx;
static std::atomic<bool> g_connected{false};

// Minimal hand-rolled JSON value extractor. Avoids pulling a JSON library.
// Looks for `"key":value` and returns the value substring. Returns empty if not found.
static std::string ExtractValue(const std::string& json, const std::string& key) {
    std::string needle = "\"" + key + "\":";
    auto pos = json.find(needle);
    if (pos == std::string::npos) return "";
    pos += needle.size();
    // skip whitespace
    while (pos < json.size() && (json[pos] == ' ' || json[pos] == '\t')) pos++;
    if (pos >= json.size()) return "";
    // string value
    if (json[pos] == '"') {
        auto end = json.find('"', pos + 1);
        return end == std::string::npos ? "" : json.substr(pos + 1, end - pos - 1);
    }
    // number/bool/null — read until comma or closing brace
    auto end = pos;
    while (end < json.size() && json[end] != ',' && json[end] != '}' && json[end] != ']') end++;
    return json.substr(pos, end - pos);
}

static bool WriteLine(const std::string& line) {
    if (!g_connected || g_pipe == nullptr) return false;
    std::lock_guard<std::mutex> lock(g_writeMtx);
    std::string buf = line + "\n";
    DWORD written = 0;
    return WriteFile(g_pipe, buf.c_str(), (DWORD)buf.size(), &written, nullptr) && written == buf.size();
}

static bool ReadLine(std::string& out) {
    if (!g_connected || g_pipe == nullptr) return false;
    out.clear();
    char c;
    DWORD read = 0;
    while (ReadFile(g_pipe, &c, 1, &read, nullptr) && read == 1) {
        if (c == '\n') return true;
        if (c != '\r') out.push_back(c);
    }
    return false;
}

bool IpcClient::Connect(int timeoutMs) {
    if (g_connected) return true;
    // Try connecting in a loop — the Launcher creates the pipe server before injecting,
    // but there may be a brief race. Retry for up to timeoutMs.
    int slept = 0;
    while (slept < timeoutMs) {
        g_pipe = CreateFileA(kPipeName, GENERIC_READ | GENERIC_WRITE,
            0, nullptr, OPEN_EXISTING, 0, nullptr);
        if (g_pipe != INVALID_HANDLE_VALUE) {
            g_connected = true;
            HOOK_LOG("IPC: connected to Launcher pipe.");
            return true;
        }
        // ERROR_PIPE_BUSY means server is busy — wait. Other errors mean not ready yet.
        Sleep(50);
        slept += 50;
    }
    HOOK_LOG("IPC: failed to connect to Launcher pipe (timeout).");
    return false;
}

void IpcClient::Disconnect() {
    g_connected = false;
    if (g_pipe && g_pipe != INVALID_HANDLE_VALUE) {
        CloseHandle(g_pipe);
    }
    g_pipe = nullptr;
}

bool IpcClient::IsConnected() {
    return g_connected;
}

bool IpcClient::WaitForProfile(Tweaks& out, int timeoutMs) {
    if (!g_connected) return false;
    // Read lines until we get a profile command or timeout.
    // We do a simple blocking read with a watchdog thread.
    std::atomic<bool> gotProfile{false};
    std::atomic<bool> timedOut{false};
    std::string receivedProfile;

    std::thread watchdog([&] {
        Sleep(timeoutMs);
        timedOut = true;
        // Cancel pending read by disconnecting — harsh but effective for a watchdog.
        // The read thread will get a short read and exit.
        if (!gotProfile) {
            HOOK_LOG("IPC: WaitForProfile timed out.");
            CancelIo(g_pipe);
        }
    });
    watchdog.detach();

    std::string line;
    while (!timedOut && ReadLine(line)) {
        // Look for "cmd":"profile"
        if (line.find("\"cmd\":\"profile\"") != std::string::npos) {
            // Extract the profile object — find "profile":{ ... }
            auto p = line.find("\"profile\":{");
            if (p != std::string::npos) {
                // Find matching closing brace
                int depth = 0;
                size_t start = line.find('{', p + 10);
                size_t end = start;
                for (; end < line.size(); end++) {
                    if (line[end] == '{') depth++;
                    else if (line[end] == '}') { depth--; if (depth == 0) { end++; break; } }
                }
                receivedProfile = line.substr(start, end - start);
            } else {
                receivedProfile = "{}";
            }
            gotProfile = true;
            break;
        }
        // Ignore ping and other commands for now
    }

    if (!gotProfile) return false;

    // Parse the profile fields by hand (minimal — only the ones we use)
    out.renderScaleMode = std::atoi(ExtractValue(receivedProfile, "renderScaleMode").c_str());
    out.renderScale = std::atof(ExtractValue(receivedProfile, "renderScale").c_str());
    out.customWidth = std::atoi(ExtractValue(receivedProfile, "customWidth").c_str());
    out.customHeight = std::atoi(ExtractValue(receivedProfile, "customHeight").c_str());
    out.fpsCap = std::atoi(ExtractValue(receivedProfile, "fpsCap").c_str());
    out.simplifyVertexShaders = ExtractValue(receivedProfile, "simplifyVertexShaders") == "true";
    out.simplifyPixelShaders = ExtractValue(receivedProfile, "simplifyPixelShaders") == "true";
    out.skipComputeShaders = ExtractValue(receivedProfile, "skipComputeShaders") == "true";
    out.lodBias = std::atof(ExtractValue(receivedProfile, "lodBias").c_str());
    out.disableTessellation = ExtractValue(receivedProfile, "disableTessellation") == "true";
    out.disablePostProcessing = ExtractValue(receivedProfile, "disablePostProcessing") == "true";
    out.maxTextureMipLevel = std::atoi(ExtractValue(receivedProfile, "maxTextureMipLevel").c_str());
    out.disableGeometryShader = ExtractValue(receivedProfile, "disableGeometryShader") == "true";
    out.msaaOff = ExtractValue(receivedProfile, "msaaOff") == "true";
    out.skipSmallDraws = ExtractValue(receivedProfile, "skipSmallDraws") == "true";
    out.skipSmallDrawThreshold = std::atoi(ExtractValue(receivedProfile, "skipSmallDrawThreshold").c_str());

    HOOK_LOG("IPC: received profile (fpsCap=" + std::to_string(out.fpsCap) +
             ", renderScale=" + std::to_string(out.renderScale) + ")");
    return true;
}

void IpcClient::SendLog(const std::string& message) {
    // Escape quotes in the message
    std::string escaped;
    escaped.reserve(message.size() + 8);
    for (char c : message) {
        if (c == '"' || c == '\\') escaped.push_back('\\');
        escaped.push_back(c);
    }
    WriteLine("{\"log\":\"" + escaped + "\"}");
}

void IpcClient::SendAck(const std::string& what) {
    WriteLine("{\"ack\":\"" + what + "\"}");
}

void IpcClient::SendTelemetry(const std::string& name, double value) {
    WriteLine("{\"telemetry\":\"" + name + "\",\"value\":" + std::to_string(value) + "}");
}

// void glShaderSource(GLuint shader, GLsizei count,
//                      const GLchar* const* string, const GLint* length);
// GL types are fixed-width in gl.h, but we avoid pulling gl.h here to keep the
// translation unit lean — the ABI is stable (int/unsigned int/char**).
using GlShaderSourceFn = void(APIENTRY*)(unsigned int, int, const char* const*, const int*);

// void glDisable(GLenum cap); resolved lazily from opengl32.dll (GL_MULTISAMPLE=0x809D).
using GlDisableFn = void(APIENTRY*)(unsigned int);

// --- Originals (filled by Install, restored by Uninstall) ---------------------

static WglSwapBuffersFn g_origWglSwapBuffers = nullptr;
static GlShaderSourceFn g_origGlShaderSource = nullptr;
static GlDisableFn g_glDisable = nullptr;  // resolved lazily on first use
static std::atomic<bool> g_msaaLogged{false};
static bool g_installed = false;

// --- Trampolines ---------------------------------------------------------------

static BOOL WINAPI HookWglSwapBuffers(HDC hdc) {
    // MSAA off: re-assert each frame (cheap). glDisable is resolved lazily; the
    // GL_MULTISAMPLE enum (0x809D) is defined locally to avoid pulling gl.h.
    if (TweakApplier::GetTweaks().msaaOff) {
        if (!g_glDisable) {
            HMODULE gl = GetModuleHandleA("opengl32.dll");
            if (gl) g_glDisable = reinterpret_cast<GlDisableFn>(GetProcAddress(gl, "glDisable"));
        }
        if (g_glDisable) {
            g_glDisable(0x809D);  // GL_MULTISAMPLE
            if (!g_msaaLogged.exchange(true)) IPC_LOG("[GL] glDisable(GL_MULTISAMPLE) per-frame");
        }
    }

    // FPS cap: same throttle as D3D11 Present. No-op when fpsCap == 0.
    TweakApplier::ThrottleForFpsCap();
    return g_origWglSwapBuffers(hdc);
}

static void APIENTRY HookGlShaderSource(unsigned int shader, int count,
    const char* const* strings, const int* lengths) {
    // Concatenate the (count) source strings. Lengths may be nullptr (then each
    // string is NUL-terminated). We always capture the ORIGINAL combined source
    // before any substitution so the cache is lossless.
    std::string combined;
    if (strings && count > 0) {
        for (int i = 0; i < count; ++i) {
            if (lengths && lengths[i] >= 0) {
                combined.append(strings[i], static_cast<size_t>(lengths[i]));
            } else if (strings[i]) {
                combined.append(strings[i]);
            }
        }
    }
    if (!combined.empty()) {
        ShaderCache::SaveOriginal(combined.data(), combined.size());
    }

    // Stage-aware substitution. We never replace vertex shaders (keep
    // silhouette). Fragment -> flat color; compute -> no-op.
    Tweaks t = TweakApplier::GetTweaks();
    GlslStage stage = combined.empty() ? GlslStage::Unknown : ClassifyGlsl(combined);
    std::string ver = ExtractVersionLine(combined);
    std::string repl;
    if (stage == GlslStage::Fragment && t.simplifyPixelShaders) {
        repl = (ver.empty() ? "#version 330 core\n" : ver) +
            "out vec4 fragColor;\n"
            "void main(){ fragColor = vec4(0.35, 0.35, 0.35, 1.0); }\n";
    } else if (stage == GlslStage::Compute && t.skipComputeShaders) {
        repl = (ver.empty() ? "#version 430 core\n" : ver) +
            "layout(local_size_x = 1, local_size_y = 1, local_size_z = 1) in;\n"
            "void main(){}\n";
    }

    if (!repl.empty()) {
        const char* ptrs[1] = { repl.c_str() };
        int lens[1] = { static_cast<int>(repl.size()) };
        static std::atomic<int> logged{0};
        if (logged.fetch_add(1) < 5) {
            const char* which = (stage == GlslStage::Fragment) ? "flat" : "no-op compute";
            char buf[96];
            std::snprintf(buf, sizeof(buf), "[GL] glShaderSource substituted (%s)", which);
            IPC_LOG(buf);
        }
        g_origGlShaderSource(shader, 1, ptrs, lens);
        return;
    }

    // No substitution (vertex / unknown / tweak off): forward original strings.
    g_origGlShaderSource(shader, count, strings, lengths);
}

// --- Install / Uninstall -------------------------------------------------------

bool OpenGLHook::Install() {
    if (g_installed) return true;

    HMODULE host = GetModuleHandleW(nullptr);  // host EXE
    if (!host) {
        HOOK_LOG("OpenGLHook: cannot get host EXE handle.");
        return false;
    }

    bool ok = false;
    ok |= IatHook::Install(host, "gdi32.dll", "wglSwapBuffers",
        reinterpret_cast<void*>(&HookWglSwapBuffers),
        reinterpret_cast<void**>(&g_origWglSwapBuffers));
    ok |= IatHook::Install(host, "opengl32.dll", "glShaderSource",
        reinterpret_cast<void*>(&HookGlShaderSource),
        reinterpret_cast<void**>(&g_origGlShaderSource));

    if (ok) {
        g_installed = true;
        HOOK_LOG("OpenGLHook: installed.");
    } else {
        HOOK_LOG("OpenGLHook: no OpenGL imports found (game may not use OpenGL).");
    }
    return ok;
}

void OpenGLHook::Uninstall() {
    if (!g_installed) return;
    HMODULE host = GetModuleHandleW(nullptr);
    if (host) {
        IatHook::Remove(host, "gdi32.dll", "wglSwapBuffers",
            reinterpret_cast<void*>(g_origWglSwapBuffers));
        IatHook::Remove(host, "opengl32.dll", "glShaderSource",
            reinterpret_cast<void*>(g_origGlShaderSource));
    }
    g_origWglSwapBuffers = nullptr;
    g_origGlShaderSource = nullptr;
    g_installed = false;
}

// ---- FNV-1a 64-bit ----
// Public domain hash. Good distribution, ~5 lines, no dependency.
uint64_t HashBytecode(const void* data, size_t size) {
    const uint8_t* p = static_cast<const uint8_t*>(data);
    uint64_t h = 0xcbf29ce484222325ULL;        // FNV offset basis
    for (size_t i = 0; i < size; ++i) {
        h ^= p[i];
        h *= 0x100000001b3ULL;                  // FNV prime
    }
    return h;
}

// ---- Path helpers ----

static std::string AppDataRoot() {
    char path[MAX_PATH] = {};
    if (SUCCEEDED(SHGetFolderPathA(nullptr, CSIDL_APPDATA, nullptr, 0, path))) {
        return std::string(path) + "\\GameLagReducer";
    }
    return "C:\\GameLagReducer";  // fallback (rare)
}

static std::string HashToHex(uint64_t h) {
    char buf[20];
    std::snprintf(buf, sizeof(buf), "%016llx", (unsigned long long)h);
    return buf;
}

static bool DirExists(const std::string& path) {
    struct stat st;
    return stat(path.c_str(), &st) == 0 && (st.st_mode & S_IFDIR);
}

static bool MakeDir(const std::string& path) {
    if (DirExists(path)) return true;
    // CreateDirectory needs the parent to exist; recurse up.
    auto pos = path.find_last_of("\\/");
    if (pos != std::string::npos && pos > 3) {
        MakeDir(path.substr(0, pos));
    }
    return CreateDirectoryA(path.c_str(), nullptr) != 0 || DirExists(path);
}

// ---- ShaderCache ----

std::string ShaderCache::CacheDir() {
    static std::string dir;
    if (!dir.empty()) return dir;
    dir = AppDataRoot() + "\\shaders";
    MakeDir(dir);
    return dir;
}

static std::string FilePath(uint64_t hash, const char* suffix) {
    return ShaderCache::CacheDir() + "\\" + HashToHex(hash) + suffix;
}

uint64_t ShaderCache::SaveOriginal(const void* bytecode, size_t size) {
    uint64_t hash = HashBytecode(bytecode, size);
    std::string path = FilePath(hash, ".bin");
    if (HasOriginal(hash)) return hash;  // already cached — don't rewrite
    std::ofstream f(path, std::ios::binary);
    if (!f) {
        HOOK_LOG("ShaderCache: cannot write " + path);
        return hash;
    }
    f.write(static_cast<const char*>(bytecode), (std::streamsize)size);
    return hash;
}

void ShaderCache::SaveDisasm(uint64_t hash, const std::string& text) {
    std::string path = FilePath(hash, ".asm");
    std::ofstream f(path);
    if (!f) return;
    f << text;
}

bool ShaderCache::LoadReplacement(uint64_t hash, std::vector<uint8_t>& out) {
    std::string path = FilePath(hash, ".repl.bin");
    std::ifstream f(path, std::ios::binary | std::ios::ate);
    if (!f) return false;
    auto size = f.tellg();
    if (size <= 0) return false;
    f.seekg(0);
    out.resize((size_t)size);
    f.read(reinterpret_cast<char*>(out.data()), size);
    return true;
}

bool ShaderCache::HasOriginal(uint64_t hash) {
    struct stat st;
    return stat(FilePath(hash, ".bin").c_str(), &st) == 0;
}

// Pre-compiled no-op compute shader bytecode (lazy init).

// ---- No-op compute shader ----
// A compute shader that does nothing. Compiled once with D3DCompile, reused
// for every compute shader the game tries to create when SkipComputeShaders is on.
// We compile for cs_5_0 (D3D11 feature level 11.0). The thread group size [1,1,1]
// is the minimum; the shader body is empty so it costs ~nothing on the GPU.
static const char* kNoopComputeHlsl =
    "[numthreads(1,1,1)] void cs_main() {}";

const std::vector<uint8_t>& ShaderModifier::GetNoopComputeBytecode() {
    std::call_once(g_noopOnce, [] {
        ID3DBlob* blob = nullptr;
        ID3DBlob* errors = nullptr;
        HRESULT hr = D3DCompile(kNoopComputeHlsl, strlen(kNoopComputeHlsl),
            nullptr, nullptr, nullptr, "cs_main", "cs_5_0",
            0, 0, &blob, &errors);
        if (FAILED(hr) || !blob) {
            HOOK_LOG("ShaderModifier: no-op compute compile failed.");
            if (errors) {
                HOOK_LOG(std::string("  D3DCompile: ") +
                    (const char*)errors->GetBufferPointer());
                errors->Release();
            }
            return;
        }
        g_noopCompute.assign(
            (const uint8_t*)blob->GetBufferPointer(),
            (const uint8_t*)blob->GetBufferPointer() + blob->GetBufferSize());
        blob->Release();
        HOOK_LOG("ShaderModifier: no-op compute shader compiled (" +
            std::to_string(g_noopCompute.size()) + " bytes).");
    });
    return g_noopCompute;
}

// ---- Lightweight replacement shaders ----
// These are intentionally tiny pass-through shaders. Their goal is to reduce
// GPU work in the game by replacing heavy vertex/pixel shader programs with
// very inexpensive variants when the user enables the corresponding tweak.
static const char* kSimpleVertexHlsl =

    "struct VS_OUT { float4 pos : SV_POSITION; };"
    "VS_OUT main(VS_IN input) { VS_OUT o; o.pos = input.pos; return o; }";

    std::vector<uint8_t>& outBytes) {
    ID3DBlob* blob = nullptr;
    ID3DBlob* errors = nullptr;
    HRESULT hr = D3DCompile(hlsl, strlen(hlsl), nullptr, nullptr, nullptr,
        entry, target, 0, 0, &blob, &errors);

        if (errors) {
            HOOK_LOG(std::string("ShaderModifier: compile failed: ") +
                (const char*)errors->GetBufferPointer());
            errors->Release();
        }
        return false;
    }

    outBytes.assign(
        (const uint8_t*)blob->GetBufferPointer(),
        (const uint8_t*)blob->GetBufferPointer() + blob->GetBufferSize());
    blob->Release();
    return true;
}

ProcessedShader ShaderModifier::Process(ShaderStage stage,
    const void* bytecode, SIZE_T bytecodeLength) {

    ProcessedShader result;
    result.bytecode = bytecode;
    result.bytecodeLength = bytecodeLength;

    // 1. Hash + cache original (always — capture is the foundation).
    uint64_t hash = ShaderCache::SaveOriginal(bytecode, bytecodeLength);
    result.originalHash = hash;

    // 2. Disassemble + save .asm on first capture only. D3DDisassemble is not
    // free, so we gate on an in-memory set of already-disassembled hashes.
    // (A game has hundreds of unique shaders, not millions — a vector with a
    // 4096 cap is plenty and avoids a std::unordered_set dependency.)
    static std::mutex disasmMtx;
    static std::vector<uint64_t> disasmDone;
    {
        std::lock_guard<std::mutex> lock(disasmMtx);
        bool found = false;
        for (auto h : disasmDone) if (h == hash) { found = true; break; }
        if (!found) {
            std::string text = Disassemble(bytecode, bytecodeLength);
            if (!text.empty()) {
                ShaderCache::SaveDisasm(hash, text);
            }
            disasmDone.push_back(hash);
            if (disasmDone.size() > 4096) disasmDone.erase(disasmDone.begin());
        }
    }

    // 3. User-provided replacement file wins over built-in tweaks.
    std::vector<uint8_t> userRepl;
    if (ShaderCache::LoadReplacement(hash, userRepl)) {
        result.replacement = std::move(userRepl);
        result.bytecode = result.replacement.data();
        result.bytecodeLength = result.replacement.size();
        result.modified = true;
        char buf[128];
        std::snprintf(buf, sizeof(buf), "[Shader] %016llx: user replacement applied",
            (unsigned long long)hash);
        IPC_LOG(buf);
        return result;
    }

    // 4. Built-in tweaks.
    Tweaks t;
    {
        std::lock_guard<std::mutex> lock(g_mtx);
        t = g_tweaks;
    }

    if (stage == ShaderStage::Compute && t.skipComputeShaders) {
        const auto& noop = GetNoopComputeBytecode();
        if (!noop.empty()) {
            result.replacement = noop;  // copy (cheap: ~200 bytes)
            result.bytecode = result.replacement.data();
            result.bytecodeLength = result.replacement.size();
            result.modified = true;
            char buf[128];
            std::snprintf(buf, sizeof(buf), "[Shader] %016llx: compute → no-op",
                (unsigned long long)hash);
            IPC_LOG(buf);
        }
        return result;
    }

    if (stage == ShaderStage::Vertex && t.simplifyVertexShaders) {
        const auto& simple = GetSimpleVertexBytecode();
        if (!simple.empty()) {
            result.replacement = simple;
            result.bytecode = result.replacement.data();
            result.bytecodeLength = result.replacement.size();
            result.modified = true;
            char buf[128];
            std::snprintf(buf, sizeof(buf), "[Shader] %016llx: vertex → simplified",
                (unsigned long long)hash);
            IPC_LOG(buf);
        }
        return result;
    }

    if (stage == ShaderStage::Pixel && (t.simplifyPixelShaders || t.disablePostProcessing)) {
        const auto& simple = GetSimplePixelBytecode();
        if (!simple.empty()) {
            result.replacement = simple;
            result.bytecode = result.replacement.data();
            result.bytecodeLength = result.replacement.size();
            result.modified = true;
            char buf[128];
            std::snprintf(buf, sizeof(buf), "[Shader] %016llx: pixel → simplified",
                (unsigned long long)hash);
            IPC_LOG(buf);
        }
        return result;
    }

    return result;
}

#include "tweak_applier.h"
#include "logger.h"
#include "ipc_client.h"

#include <windows.h>
#include <emmintrin.h>  // _mm_pause
#include <cstdio>
#include <mutex>
#include <atomic>

namespace hookdll {

// ---- State ----
static std::mutex g_mtx;
static Tweaks g_tweaks{};

// High-resolution timer for FPS cap. Initialized once.
static LARGE_INTEGER g_freq{0};
static LARGE_INTEGER g_lastPresent{0};
static bool g_timerInit = false;

static void EnsureTimer() {
    if (!g_timerInit) {
        QueryPerformanceFrequency(&g_freq);
        QueryPerformanceCounter(&g_lastPresent);
        g_timerInit = true;
    }
}

// ---- Public API ----

void TweakApplier::SetTweaks(const Tweaks& tweaks) {
    std::lock_guard<std::mutex> lock(g_mtx);
    g_tweaks = tweaks;
    char buf[256];
    std::snprintf(buf, sizeof(buf),
        "TweakApplier: tweaks set (fpsCap=%d, renderScaleMode=%d, renderScale=%.3f)",
        tweaks.fpsCap, tweaks.renderScaleMode, tweaks.renderScale);
    HOOK_LOG(buf);
    IPC_LOG(buf);
}

Tweaks TweakApplier::GetTweaks() {
    std::lock_guard<std::mutex> lock(g_mtx);
    return g_tweaks;
}

void TweakApplier::ThrottleForFpsCap() {
    Tweaks t = GetTweaks();
    if (t.fpsCap <= 0) return;  // uncapped

    EnsureTimer();
    if (g_freq.QuadPart == 0) return;

    // Target frame time in counts. e.g. 60fps → 1/60s.
    double targetSeconds = 1.0 / (double)t.fpsCap;
    LONGLONG targetCounts = (LONGLONG)(targetSeconds * (double)g_freq.QuadPart);

    LARGE_INTEGER now;
    QueryPerformanceCounter(&now);

    LONGLONG elapsed = now.QuadPart - g_lastPresent.QuadPart;
    if (elapsed < targetCounts) {
        // We're ahead of schedule — wait the remaining time.
        LONGLONG remaining = targetCounts - elapsed;

        // Coarse sleep first (1ms granularity, leaves the CPU mostly idle),
        // then spin the last ~0.5ms for accuracy. Sleep(0) yields; Sleep(1) is
        // ~15.6ms on Windows by default but we only use it when remaining is
        // large enough to be worth it.
        double remainingMs = (double)remaining * 1000.0 / (double)g_freq.QuadPart;
        if (remainingMs > 2.0) {
            // Sleep in 1ms chunks minus a safety margin for the spin phase.
            DWORD sleepMs = (DWORD)(remainingMs - 1.0);
            if (sleepMs > 0) Sleep(sleepMs);
        }
        // Spin-wait the final fraction for sub-ms accuracy.
        while (true) {
            QueryPerformanceCounter(&now);
            if (now.QuadPart - g_lastPresent.QuadPart >= targetCounts) break;
            _mm_pause();  // hint to CPU that we're in a spin loop
        }
    }

    // Update the timestamp for the next frame. We anchor on the previous
    // timestamp (not "now") so that a long frame doesn't shift the cadence —
    // this keeps the cap stable under variable frame times.
    QueryPerformanceCounter(&now);
    g_lastPresent.QuadPart = now.QuadPart;
}

void TweakApplier::ApplyRenderScale(UINT& Width, UINT& Height) {
    Tweaks t = GetTweaks();
    if (t.renderScaleMode == 0) return;  // Off

    UINT origW = Width;
    UINT origH = Height;

    if (t.renderScaleMode == 1) {
        // Scale factor (e.g. 0.75 → 75% of native).
        if (t.renderScale <= 0.0 || t.renderScale >= 1.0) return;  // no-op / invalid
        Width = (UINT)((double)Width * t.renderScale);
        Height = (UINT)((double)Height * t.renderScale);
    } else if (t.renderScaleMode == 2) {
        // Custom resolution.
        if (t.customWidth <= 0 || t.customHeight <= 0) return;
        Width = (UINT)t.customWidth;
        Height = (UINT)t.customHeight;
    }

    // Clamp to at least 1x1 to avoid degenerate swapchains.
    if (Width < 1) Width = 1;
    if (Height < 1) Height = 1;

    if (Width != origW || Height != origH) {
        static std::atomic<int> logged{0};
        if (logged++ < 5) {
            char buf[160];
            std::snprintf(buf, sizeof(buf),
                "[Tweak] ResizeBuffers: %ux%u → %ux%u", origW, origH, Width, Height);
            IPC_LOG(buf);
        }
    }
}

    const uint32_t* pCode;      // SPIR-V bytecode
};

// --- Pipeline-create structs + shader stage bits (Phase 1: tess/GS strip + MSAA) ---
// Stage bit values per the Vulkan spec (VkShaderStageFlagBits). These were
// verified — note GEOMETRY=0x8, not 0x4 (a correction over an earlier draft).
static constexpr uint32_t VK_SHADER_STAGE_VERTEX_BIT              = 0x00000001;
static constexpr uint32_t VK_SHADER_STAGE_TESSELLATION_CONTROL_BIT   = 0x00000002;
static constexpr uint32_t VK_SHADER_STAGE_TESSELLATION_EVALUATION_BIT = 0x00000004;
static constexpr uint32_t VK_SHADER_STAGE_GEOMETRY_BIT           = 0x00000008;
static constexpr uint32_t VK_SHADER_STAGE_FRAGMENT_BIT           = 0x00000010;
static constexpr uint32_t VK_SAMPLE_COUNT_1_BIT                  = 1;

// VkPipelineShaderStageCreateInfo. We only read `stage`, `module`, `pName`,
// `pSpecializationInfo`; the rest is preserved verbatim when copying.
struct VkPipelineShaderStageCreateInfo {
    int32_t sType; const void* pNext; uint32_t flags; uint32_t stage;
    VkShaderModule module; const char* pName; const void* pSpecializationInfo;
};

struct VkPipelineMultisampleStateCreateInfo {
    int32_t sType; const void* pNext; uint32_t flags; uint32_t rasterizationSamples;
    uint32_t sampleShadingEnable; float minSampleShading; const uint32_t* pSampleMask;
    uint32_t alphaToCoverageEnable; uint32_t alphaToOneEnable;
};

// VkGraphicsPipelineCreateInfo. Only the fields we touch are named; the others
// (vertex input, IA, tess, viewport, raster, D/S, blend, dynamic, layout,
// renderPass, subpass, base pipeline) are carried through as opaque void*/scalars.
struct VkGraphicsPipelineCreateInfo {
    int32_t sType; const void* pNext; uint32_t flags; uint32_t stageCount;
    const VkPipelineShaderStageCreateInfo* pStages; const void* pVertexInputState;
    const void* pInputAssemblyState; const void* pTessellationState; const void* pViewportState;
    const void* pRasterizationState; const VkPipelineMultisampleStateCreateInfo* pMultisampleState;
    const void* pDepthStencilState; const void* pColorBlendState; const void* pDynamicState;
    uint64_t layout; uint64_t renderPass; uint32_t subpass; uint64_t basePipelineHandle; int32_t basePipelineIndex;
};

struct VkComputePipelineCreateInfo {
    int32_t sType; const void* pNext; uint32_t flags; VkPipelineShaderStageCreateInfo stage;
    uint64_t layout; uint64_t basePipelineHandle; int32_t basePipelineIndex;
};

using VkPipelineCache = uint64_t;
using VkPipeline = uint64_t;

// --- Function pointer types ---------------------------------------------------

// VKAPI_ATTR VkResult VKAPI_CALL vkCreateShaderModule(
//     VkDevice device, const VkShaderModuleCreateInfo* pCreateInfo,
//     const VkAllocationCallbacks* pAllocator, VkShaderModule* pShaderModule);
using VkCreateShaderModuleFn = VkResult(VKAPI_PTR*)(VkDevice,
    const VkShaderModuleCreateInfo*, const VkAllocationCallbacks*, VkShaderModule*);

// VKAPI_ATTR VkResult VKAPI_CALL vkCreateGraphicsPipelines(
//     VkDevice device, VkPipelineCache pipelineCache, uint32_t createInfoCount,
//     const VkGraphicsPipelineCreateInfo* pCreateInfos,
//     const VkAllocationCallbacks* pAllocator, VkPipeline* pPipelines);
using VkCreateGraphicsPipelinesFn = VkResult(VKAPI_PTR*)(VkDevice, VkPipelineCache, uint32_t,
    const VkGraphicsPipelineCreateInfo*, const VkAllocationCallbacks*, VkPipeline*);

// VKAPI_ATTR VkResult VKAPI_CALL vkCreateComputePipelines(... same shape ...);
using VkCreateComputePipelinesFn = VkResult(VKAPI_PTR*)(VkDevice, VkPipelineCache, uint32_t,
    const VkComputePipelineCreateInfo*, const VkAllocationCallbacks*, VkPipeline*);

// VKAPI_ATTR VkResult VKAPI_CALL vkQueuePresentKHR(
//     VkQueue queue, const VkPresentInfoKHR* pPresentInfo);
using VkQueuePresentKHRFn = VkResult(VKAPI_PTR*)(VkQueue, const void*);

// PFN_vkVoidFunction vkGetDeviceProcAddr(VkDevice device, const char* pName);
using VkGetDeviceProcAddrFn = void*(VKAPI_PTR*)(VkDevice, const char*);

// --- Originals -----------------------------------------------------------------

static VkGetDeviceProcAddrFn g_origGetDeviceProcAddr = nullptr;
static VkCreateShaderModuleFn g_origCreateShaderModule = nullptr;
static VkQueuePresentKHRFn g_origQueuePresentKHR = nullptr;
static VkCreateGraphicsPipelinesFn g_origCreateGraphicsPipelines = nullptr;
static VkCreateComputePipelinesFn g_origCreateComputePipelines = nullptr;
static bool g_installed = false;

// --- Trampolines ---------------------------------------------------------------

static VkResult VKAPI_PTR HookVkCreateShaderModule(VkDevice device,
    const VkShaderModuleCreateInfo* pCreateInfo,
    const VkAllocationCallbacks* pAllocator, VkShaderModule* pShaderModule) {
    // Capture SPIR-V bytecode. codeSize is in bytes; pCode is uint32_t*.
    if (pCreateInfo && pCreateInfo->pCode && pCreateInfo->codeSize > 0) {
        ShaderCache::SaveOriginal(pCreateInfo->pCode, pCreateInfo->codeSize);
    }
    return g_origCreateShaderModule(device, pCreateInfo, pAllocator, pShaderModule);
}

static VkResult VKAPI_PTR HookVkQueuePresentKHR(VkQueue queue, const void* pPresentInfo) {
    // FPS cap: same throttle as D3D11 Present / wglSwapBuffers.
    TweakApplier::ThrottleForFpsCap();
    return g_origQueuePresentKHR(queue, pPresentInfo);
}

// ---- Pipeline-create interception (Phase 1: strip tess/GS + force MSAA=1) ----
// We never touch SPIR-V here — we just drop tessellation-control/eval and
// geometry stages from the pStages array and set rasterizationSamples=1 on the
// multisample state. Per-call mutable copies keep the game's (likely const)
// input untouched. Fail-soft: any exception -> forward the original unchanged.
static VkResult VKAPI_PTR HookVkCreateGraphicsPipelines(VkDevice device,
    VkPipelineCache pipelineCache, uint32_t createInfoCount,
    const VkGraphicsPipelineCreateInfo* pCreateInfos,
    const VkAllocationCallbacks* pAllocator, VkPipeline* pPipelines) {
    if (!pCreateInfos || createInfoCount == 0) {
        return g_origCreateGraphicsPipelines(device, pipelineCache, createInfoCount,
            pCreateInfos, pAllocator, pPipelines);
    }
    Tweaks t = TweakApplier::GetTweaks();
    if (!t.disableTessellation && !t.disableGeometryShader && !t.msaaOff) {
        return g_origCreateGraphicsPipelines(device, pipelineCache, createInfoCount,
            pCreateInfos, pAllocator, pPipelines);
    }
    try {
        std::vector<VkGraphicsPipelineCreateInfo> cis(createInfoCount);
        // Per-pipeline stage buffer (only allocated if we strip stages) + ms copy.
        std::vector<std::vector<VkPipelineShaderStageCreateInfo>> stageBufs(createInfoCount);
        std::vector<VkPipelineMultisampleStateCreateInfo> msBufs(createInfoCount);
        bool anyStageStripped = false;
        bool anyMsaaForced = false;

        for (uint32_t i = 0; i < createInfoCount; ++i) {
            cis[i] = pCreateInfos[i];
            const auto& src = pCreateInfos[i];

            // Strip tessellation + geometry stages.
            if ((t.disableTessellation || t.disableGeometryShader) &&
                src.pStages && src.stageCount > 0) {
                for (uint32_t s = 0; s < src.stageCount; ++s) {
                    uint32_t st = src.pStages[s].stage;
                    if (t.disableTessellation &&
                        (st == VK_SHADER_STAGE_TESSELLATION_CONTROL_BIT ||
                         st == VK_SHADER_STAGE_TESSELLATION_EVALUATION_BIT)) {
                        continue;
                    }
                    if (t.disableGeometryShader && st == VK_SHADER_STAGE_GEOMETRY_BIT) {
                        continue;
                    }
                    stageBufs[i].push_back(src.pStages[s]);
                }
                // Only repoint if we actually kept a non-empty set; a pipeline
                // with all stages stripped would be invalid, so leave it alone
                // (the driver will reject it — same as the game's intent).
                if (!stageBufs[i].empty() && stageBufs[i].size() != src.stageCount) {
                    cis[i].pStages = stageBufs[i].data();
                    cis[i].stageCount = static_cast<uint32_t>(stageBufs[i].size());
                    anyStageStripped = true;
                }
            }

            // Force MSAA off.
            if (t.msaaOff && src.pMultisampleState) {
                msBufs[i] = *src.pMultisampleState;
                msBufs[i].rasterizationSamples = VK_SAMPLE_COUNT_1_BIT;
                cis[i].pMultisampleState = &msBufs[i];
                anyMsaaForced = true;
            }
        }

        if (anyStageStripped || anyMsaaForced) {
            static std::atomic<int> logged{0};
            if (logged.fetch_add(1) < 3) {
                char buf[128];
                std::snprintf(buf, sizeof(buf),
                    "[VK] vkCreateGraphicsPipelines patched (strip=%d msaa=%d)",
                    anyStageStripped ? 1 : 0, anyMsaaForced ? 1 : 0);
                IPC_LOG(buf);
            }
            return g_origCreateGraphicsPipelines(device, pipelineCache,
                createInfoCount, cis.data(), pAllocator, pPipelines);
        }
        // Nothing to patch (e.g. pipelines with no tess/GS and msaa already 1).
        return g_origCreateGraphicsPipelines(device, pipelineCache, createInfoCount,
            pCreateInfos, pAllocator, pPipelines);
    } catch (...) {
        // Fail soft: forward original, never crash the host.
        return g_origCreateGraphicsPipelines(device, pipelineCache, createInfoCount,
            pCreateInfos, pAllocator, pPipelines);
    }
}

// Compute pipelines: observe-only this round. Skipping compute safely requires
// knowing the engine's dependency on the pipeline (derivatives, etc.) and is
// deferred. We log the first few when skipComputeShaders is set so the user can
// see compute is happening; the actual skip is Phase 2 / future work.
static VkResult VKAPI_PTR HookVkCreateComputePipelines(VkDevice device,
    VkPipelineCache pipelineCache, uint32_t createInfoCount,
    const VkComputePipelineCreateInfo* pCreateInfos,
    const VkAllocationCallbacks* pAllocator, VkPipeline* pPipelines) {
    if (TweakApplier::GetTweaks().skipComputeShaders) {
        static std::atomic<int> logged{0};
        if (logged.fetch_add(1) < 3) {
            char buf[96];
            std::snprintf(buf, sizeof(buf),
                "[VK] vkCreateComputePipelines observed (count=%u)", createInfoCount);
            IPC_LOG(buf);
        }
    }
    return g_origCreateComputePipelines(device, pipelineCache, createInfoCount,
        pCreateInfos, pAllocator, pPipelines);
}

// vkGetDeviceProcAddr trampoline: intercept the lookup for the two functions
// we care about, return our trampoline; delegate everything else to the
// loader's original.
static void* VKAPI_PTR HookVkGetDeviceProcAddr(VkDevice device, const char* pName) {
    if (pName) {
        if (strcmp(pName, "vkCreateShaderModule") == 0) {
            if (!g_origCreateShaderModule) {
                g_origCreateShaderModule = reinterpret_cast<VkCreateShaderModuleFn>(
                    g_origGetDeviceProcAddr(device, pName));
            }
            return reinterpret_cast<void*>(&HookVkCreateShaderModule);
        }
        if (strcmp(pName, "vkQueuePresentKHR") == 0) {
            if (!g_origQueuePresentKHR) {
                g_origQueuePresentKHR = reinterpret_cast<VkQueuePresentKHRFn>(
                    g_origGetDeviceProcAddr(device, pName));
            }
            return reinterpret_cast<void*>(&HookVkQueuePresentKHR);
        }
        if (strcmp(pName, "vkCreateGraphicsPipelines") == 0) {
            if (!g_origCreateGraphicsPipelines) {
                g_origCreateGraphicsPipelines = reinterpret_cast<VkCreateGraphicsPipelinesFn>(
                    g_origGetDeviceProcAddr(device, pName));
            }
            return reinterpret_cast<void*>(&HookVkCreateGraphicsPipelines);
        }
        if (strcmp(pName, "vkCreateComputePipelines") == 0) {
            if (!g_origCreateComputePipelines) {
                g_origCreateComputePipelines = reinterpret_cast<VkCreateComputePipelinesFn>(
                    g_origGetDeviceProcAddr(device, pName));
            }
            return reinterpret_cast<void*>(&HookVkCreateComputePipelines);
        }
    }
    return g_origGetDeviceProcAddr(device, pName);
}

// --- Install / Uninstall -------------------------------------------------------

bool VulkanHook::Install() {
    if (g_installed) return true;

    HMODULE host = GetModuleHandleW(nullptr);
    if (!host) {
        HOOK_LOG("VulkanHook: cannot get host EXE handle.");
        return false;
    }

    bool ok = IatHook::Install(host, "vulkan-1.dll", "vkGetDeviceProcAddr",
        reinterpret_cast<void*>(&HookVkGetDeviceProcAddr),
        reinterpret_cast<void**>(&g_origGetDeviceProcAddr));

    if (ok) {
        g_installed = true;
        HOOK_LOG("VulkanHook: installed (vkGetDeviceProcAddr intercepted).");
    } else {
        HOOK_LOG("VulkanHook: no vulkan-1.dll import found (game may not use Vulkan).");
    }
    return ok;
}

void VulkanHook::Uninstall() {
    if (!g_installed) return;
    HMODULE host = GetModuleHandleW(nullptr);
    if (host) {
        IatHook::Remove(host, "vulkan-1.dll", "vkGetDeviceProcAddr",
            reinterpret_cast<void*>(g_origGetDeviceProcAddr));
    }
    g_origGetDeviceProcAddr = nullptr;
    g_origCreateShaderModule = nullptr;
    g_origQueuePresentKHR = nullptr;
    g_origCreateGraphicsPipelines = nullptr;
    g_origCreateComputePipelines = nullptr;
    g_installed = false;
}

}  // namespace hookdll

// Unit tests for ClassifyGlsl + ExtractVersionLine (pure functions).
// Build as a standalone exe (no GPU / no game needed):
//   cmake --build build --config Release --target glsl_classify_tests
//   ./build/Release/glsl_classify_tests

#include "../include/glsl_classify.h"

#include <cstdio>

static int failures = 0;

#define CK(expr, name) do { if (!(expr)) { printf("FAIL: %s\n", name); ++failures; } } while (0)

int main() {
    using S = hookdll::GlslStage;

    // Vertex: gl_Position present.
    CK(hookdll::ClassifyGlsl("#version 330\nvoid main(){ gl_Position=vec4(1); }") == S::Vertex, "vertex gl_Position");

    // Fragment (modern): out vec4 output declaration.
    CK(hookdll::ClassifyGlsl("#version 330\nout vec4 f; void main(){ f=vec4(1); }") == S::Fragment, "fragment out vec4");

    // Fragment (legacy): gl_FragColor.
    CK(hookdll::ClassifyGlsl("#version 120\nvoid main(){ gl_FragColor=vec4(1); }") == S::Fragment, "fragment gl_FragColor");

    // Compute: layout(local_size_x=...).
    CK(hookdll::ClassifyGlsl("#version 430\nlayout(local_size_x=1) in; void main(){}") == S::Compute, "compute");

    // Vertex wins over fragment marker: a vertex shader that also declares an
    // out vec4 (e.g. for a transform-feedback varying) should classify as Vertex.
    CK(hookdll::ClassifyGlsl("#version 330\nout vec4 v_color; void main(){ gl_Position=vec4(1); }") == S::Vertex, "vertex beats out vec4");

    // Unknown: no markers.
    CK(hookdll::ClassifyGlsl("void main(){}") == S::Unknown, "unknown");

    // ExtractVersionLine: present.
    CK(hookdll::ExtractVersionLine("#version 330 core\nvoid main(){}") == "#version 330 core\n", "version line present");

    // ExtractVersionLine: absent.
    CK(hookdll::ExtractVersionLine("void main(){}") == "", "version line absent");

    if (failures == 0) {
        printf("ALL PASS\n");
        return 0;
    }
    printf("%d FAILURES\n", failures);
    return 1;
}

using System.Globalization;
using System.Windows;
using System.Windows.Data;

namespace GameLagReducer.Converters;

/// <summary>
/// Converts a boolean to Visibility. True → Visible, False → Collapsed.
/// Inverts if Parameter="invert".
/// </summary>
public class BoolToVisibilityConverter : IValueConverter
{
    public object Convert(object value, System.Type targetType, object parameter, CultureInfo culture)
    {
        var b = value is bool b1 && b1;
        if (parameter is string s && s == "invert") b = !b;
        return b ? Visibility.Visible : Visibility.Collapsed;
    }

    public object ConvertBack(object value, System.Type targetType, object parameter, CultureInfo culture)
    {
        var v = value is Visibility vis && vis == Visibility.Visible;
        if (parameter is string s && s == "invert") v = !v;
        return v;
    }
}

/// <summary>
/// Converts a string to Visibility. Non-empty → Visible, empty/null → Collapsed.
/// </summary>
public class StringToVisibilityConverter : IValueConverter
{
    public object Convert(object value, System.Type targetType, object parameter, CultureInfo culture)
    {
        var s = value as string;
        return !string.IsNullOrEmpty(s) ? Visibility.Visible : Visibility.Collapsed;
    }

    public object ConvertBack(object value, System.Type targetType, object parameter, CultureInfo culture)
        => throw new System.NotImplementedException();
}

using Microsoft.Win32;

namespace GameLagReducer;

public partial class MainWindow : Window, INotifyPropertyChanged
{
    private readonly GameLibrary _library = new();
    private readonly LaunchOrchestrator _orchestrator;

    private ObservableCollection<GameViewModel> _games = new();
    private ObservableCollection<ProfileViewModel> _profiles = new();
    private readonly ObservableCollection<string> _logLines = new();

    public ObservableCollection<GameViewModel> Games
    {
        get => _games;
        set { _games = value; OnPropertyChanged(nameof(Games)); }
    }

    public ObservableCollection<ProfileViewModel> Profiles
    {
        get => _profiles;
        set { _profiles = value; OnPropertyChanged(nameof(Profiles)); }
    }

    public ObservableCollection<string> LogLines => _logLines;

    public MainWindow()
    {
        InitializeComponent();
        DataContext = this;
        _orchestrator = new LaunchOrchestrator(_library);
        _orchestrator.Status += OnLaunchStatus;
        _orchestrator.GameExited += OnGameExited;
        LogList.ItemsSource = LogLines;
        RefreshGames();
        RefreshProfiles();
        UpdateCacheInfo();
    }

    private void OnLaunchStatus(string msg)
    {
        // Marshal to UI thread
        Dispatcher.Invoke(() =>
        {
            LogLines.Add($"{DateTime.Now:HH:mm:ss} {msg}");
            // Keep log capped to last 500 lines to avoid unbounded memory.
            while (LogLines.Count > 500) LogLines.RemoveAt(0);
            LogList.ScrollIntoView(LogLines[LogLines.Count - 1]);
            StatusText.Text = msg;
        });
    }

    private void OnGameExited()
    {
        Dispatcher.Invoke(() => StatusText.Text = "Game exited. Ready.");
    }

    // ---- Game Library tab ----

    private void RefreshGames()
    {
        Games = new ObservableCollection<GameViewModel>(
            _library.Games.Select(g => new GameViewModel(g)));
        GameList.ItemsSource = Games;
        GameCountText.Text = $"{Games.Count} game{(Games.Count == 1 ? "" : "s")}";
    }

    private void AddGame_Click(object sender, RoutedEventArgs e)
    {
        var dlg = new OpenFileDialog
        {
            Title = "Select game executable",
            Filter = "Executable files (*.exe)|*.exe|All files (*.*)|*.*",
        };
        if (dlg.ShowDialog() != true) return;

        var exePath = dlg.FileName;
        var engine = EngineDetector.Detect(exePath);
        var antiCheat = AntiCheatDetector.Detect(exePath);

        // If engine not detected, ask user to pick
        if (engine == GameEngine.Unknown)
        {
            var choice = PickEngineDialog();
            if (choice == null) return; // user cancelled
            engine = choice.Value;
        }

        var game = new Game
        {
            Name = Path.GetFileNameWithoutExtension(exePath),
            ExecutablePath = exePath,
            WorkingDirectory = Path.GetDirectoryName(exePath) ?? string.Empty,
            Engine = engine,
            HasAntiCheat = !string.IsNullOrEmpty(antiCheat),
            AntiCheatName = antiCheat,
        };

        _library.AddGame(game);
        RefreshGames();
        StatusText.Text = $"Added: {game.Name} ({game.EngineLabel})";
    }

    /// <summary>Shows a dialog to pick engine manually. Returns null if cancelled.</summary>
    private GameEngine? PickEngineDialog()
    {
        var msg = "Could not auto-detect the game engine.\n\nPlease pick the engine:";
        var result = MessageBox.Show(
            msg + "\n\nYes = Unity\nNo = Unreal\nCancel = Custom (generic)",
            "Pick Engine",
            MessageBoxButton.YesNoCancel,
            MessageBoxImage.Question);
        return result switch
        {
            MessageBoxResult.Yes => GameEngine.Unity,
            MessageBoxResult.No => GameEngine.Unreal,
            MessageBoxResult.Cancel => GameEngine.Custom,
            _ => null,
        };
    }

    private void RemoveGame_Click(object sender, RoutedEventArgs e)
    {
        if (GameList.SelectedItem is not GameViewModel selected) return;
        var confirm = MessageBox.Show(
            $"Remove '{selected.Name}' from the library?\n(This does not delete the game files.)",
            "Confirm Remove", MessageBoxButton.YesNo, MessageBoxImage.Question);
        if (confirm != MessageBoxResult.Yes) return;

        _library.RemoveGame(selected.Id);
        RefreshGames();
        StatusText.Text = $"Removed: {selected.Name}";
    }

    private void GameList_SelectionChanged(object sender, System.Windows.Controls.SelectionChangedEventArgs e)
    {
        RemoveGameBtn.IsEnabled = GameList.SelectedItem is GameViewModel;

    private void GameProfile_Changed(object sender, System.Windows.Controls.SelectionChangedEventArgs e)
    {
        // ComboBox SelectionChanged fires during initial binding (SelectedValue set
        // from the game's ProfileId). Ignore those — only act on real user changes.
        // We detect this by checking e.AddedItems is non-empty AND the combo is loaded.
        if (sender is not System.Windows.Controls.ComboBox combo || !combo.IsLoaded) return;
        if (e.AddedItems.Count == 0) return;
        if (combo.Tag is not string gameId) return;

        var game = _library.GetGame(gameId);
        if (game == null) return;

        // SelectedValue is the profile Id (string). Empty = default preset.
        var newProfileId = combo.SelectedValue as string ?? string.Empty;
        if (game.ProfileId == newProfileId) return;  // no change

        game.ProfileId = newProfileId;
        _library.UpdateGame(game);
        var profileName = string.IsNullOrEmpty(newProfileId)
            ? "default (Medium)"
            : _library.GetProfile(newProfileId)?.Name ?? "unknown";
        StatusText.Text = $"{game.Name} → profile: {profileName}";
    }

    private void LaunchGame_Click(object sender, RoutedEventArgs e)
    {
        if (sender is not Button btn || btn.Tag is not string id) return;
        var game = _library.GetGame(id);
        if (game == null) return;

        // Anti-cheat warning before launch
        if (game.HasAntiCheat)
        {
            var warn = MessageBox.Show(
                $"⚠ WARNING: '{game.Name}' has anti-cheat ({game.AntiCheatName}).\n\n" +
                "Injecting into this game may result in a PERMANENT BAN.\n\n" +
                "Do you want to continue?",
                "Anti-Cheat Warning",
                MessageBoxButton.YesNo,
                MessageBoxImage.Warning);
            if (warn != MessageBoxResult.Yes) return;
        }

        // Resolve HookDll path. Built into Launcher output dir (see HookDll CMakeLists.txt).
        // AppContext.BaseDirectory is reliable for both `dotnet run` and published single-file
        // (Assembly.Location returns "" for single-file apps — see IL3000 warning).
        var exeDir = AppContext.BaseDirectory;
        var dllPath = Path.Combine(exeDir, "HookDll", "Release", "HookDll.dll");
        if (!File.Exists(dllPath))
            dllPath = Path.Combine(exeDir, "HookDll", "Debug", "HookDll.dll");
        if (!File.Exists(dllPath))
        {
            var build = MessageBox.Show(
                $"HookDll.dll not found at:\n{dllPath}\n\n" +
                "Build HookDll first (see README). Launch without injection?",
                "HookDll Missing",
                MessageBoxButton.YesNo,
                MessageBoxImage.Warning);
            if (build != MessageBoxResult.Yes) return;
            // Launch without injection as a fallback
            try
            {
                System.Diagnostics.Process.Start(new System.Diagnostics.ProcessStartInfo
                {
                    FileName = game.ExecutablePath,
                    Arguments = game.Arguments,
                    WorkingDirectory = game.WorkingDirectory,
                    UseShellExecute = false,
                });
                StatusText.Text = $"Launched {game.Name} (no injection — DLL missing).";
            }
            catch (Exception ex)
            {
                MessageBox.Show($"Launch failed: {ex.Message}", "Error", MessageBoxButton.OK, MessageBoxImage.Error);
            }
            return;
        }

        // Launch with injection via the orchestrator.
        _orchestrator.Launch(game, dllPath);
    }

    // ---- Settings tab: shader cache management (Phase 8) ----

    private void RefreshCache_Click(object sender, RoutedEventArgs e) => UpdateCacheInfo();

    private void ClearCache_Click(object sender, RoutedEventArgs e)
    {
        var (size, _) = ShaderCacheManager.GetSizeAndCount();
        if (size == 0)
        {
            StatusText.Text = "Shader cache is already empty.";
            return;
        }
        var confirm = MessageBox.Show(
            $"Delete all cached shaders ({FormatBytes(size)})?\n" +
            "Captured shaders will be re-captured on next launch. " +
            "User replacement files (.repl.bin) will also be deleted.",
            "Clear Shader Cache", MessageBoxButton.YesNo, MessageBoxImage.Question);
        if (confirm != MessageBoxResult.Yes) return;

        if (ShaderCacheManager.Clear())
        {
            UpdateCacheInfo();
            StatusText.Text = "Shader cache cleared.";
        }
        else
        {
            MessageBox.Show("Could not clear the cache (files may be in use by a running game).",
                "Error", MessageBoxButton.OK, MessageBoxImage.Warning);
        }
    }

    private void UpdateCacheInfo()
    {
        var (size, count) = ShaderCacheManager.GetSizeAndCount();
        CacheInfoText.Text = size == 0
            ? "Cache is empty."
            : $"{count} files · {FormatBytes(size)} · {ShaderCacheManager.DirectoryPath}";
    }

    private static string FormatBytes(long bytes) => bytes switch
    {
        < 1024 => $"{bytes} B",
        < 1024 * 1024 => $"{bytes / 1024.0:F1} KB",
        < 1024 * 1024 * 1024 => $"{bytes / (1024.0 * 1024):F1} MB",
        _ => $"{bytes / (1024.0 * 1024 * 1024):F2} GB",
    };

    // ---- INotifyPropertyChanged ----

    public event PropertyChangedEventHandler? PropertyChanged;
    private void OnPropertyChanged(string name) =>
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(name));

using System.Text.Json.Serialization;

namespace GameLagReducer.Models;

/// <summary>
/// Render-scale mode. Controls how the backbuffer is resized relative to the display.
/// </summary>
public enum RenderScaleMode
{
    /// <summary>No scaling. Game renders at native resolution.</summary>
    Off,
    /// <summary>Fixed scale factor (e.g. 0.75 = 75% of native). GPU upscaling handles the rest.</summary>
    Scale,
    /// <summary>Fixed custom resolution (e.g. 1280x720). Bypasses native entirely.</summary>
    Custom,
}

/// <summary>
/// All tweak knobs applied at runtime via GPU hooks. Each maps to a hook behavior.
/// Values are conservative defaults; users override per-game via Profile.
/// </summary>
public class Tweaks
{
    /// <summary>Render-scale mode. Off = no change.</summary>
    public RenderScaleMode RenderScaleMode { get; set; } = RenderScaleMode.Off;

    /// <summary>Scale factor (0.5 - 1.0). Only used when RenderScaleMode = Scale.</summary>
    public double RenderScale { get; set; } = 0.75;

    /// <summary>Custom render width in pixels. Only used when RenderScaleMode = Custom.</summary>
    public int CustomWidth { get; set; } = 1280;

    /// <summary>Custom render height in pixels. Only used when RenderScaleMode = Custom.</summary>
    public int CustomHeight { get; set; } = 720;

    /// <summary>Target FPS cap. 0 = uncapped. Throttles Present() to hit target frame time.</summary>
    public int FpsCap { get; set; } = 0;

    /// <summary>Simplify vertex shaders (skip non-essential lighting/transform). Reduces GPU vertex load.</summary>
    public bool SimplifyVertexShaders { get; set; } = false;

    /// <summary>Simplify pixel/fragment shaders (skip post-processing: bloom, blur, DoF).</summary>
    public bool SimplifyPixelShaders { get; set; } = false;

    /// <summary>Skip compute shaders (no-op). Heavy compute like GI/particle sim gets disabled.</summary>
    public bool SkipComputeShaders { get; set; } = false;

    /// <summary>LOD bias. Positive = use lower-detail mips sooner. 0 = no change.</summary>
    public double LodBias { get; set; } = 0.0;

    /// <summary>Disable tessellation. Saves GPU on high-poly terrain/character meshes.</summary>
    public bool DisableTessellation { get; set; } = false;

    /// <summary>Disable post-processing entirely (replace post-process shaders with pass-through).</summary>
    public bool DisablePostProcessing { get; set; } = false;

    /// <summary>Maximum texture mip level. Higher = lower texture detail. 0 = no limit.</summary>
    public int MaxTextureMipLevel { get; set; } = 0;

    /// <summary>Disable geometry shaders (E_FAIL on D3D11 CreateGeometryShader; strip GS stage from VK pipelines).</summary>
    public bool DisableGeometryShader { get; set; } = false;

    /// <summary>Force MSAA off (D3D11 SampleDesc.Count=1; glDisable(GL_MULTISAMPLE); VK rasterizerSamples=1).</summary>
    public bool MsaaOff { get; set; } = false;

    /// <summary>EXPERIMENTAL: drop draw calls below the threshold (D3D11 only). Off by default — can hide UI.</summary>
    public bool SkipSmallDraws { get; set; } = false;

    /// <summary>Draw-call IndexCount*InstanceCount threshold for SkipSmallDraws.</summary>
    public int SkipSmallDrawThreshold { get; set; } = 256;

    /// <summary>Returns true if any tweak is non-default (i.e. would actually do something).</summary>
    [JsonIgnore]
    public bool HasAnyTweak =>
        RenderScaleMode != RenderScaleMode.Off ||
        FpsCap > 0 ||
        SimplifyVertexShaders ||
        SimplifyPixelShaders ||
        SkipComputeShaders ||
        LodBias != 0.0 ||
        DisableTessellation ||
        DisablePostProcessing ||
        MaxTextureMipLevel > 0 ||
        DisableGeometryShader ||
        MsaaOff ||
        SkipSmallDraws;
}

using System;
using System.Globalization;

using GameLagReducer.Models;

namespace GameLagReducer;

/// <summary>
/// Modal editor for a Profile's name, description, and Tweaks.
/// Phase 7: replaces the Phase 1 stub in EditProfile_Click.
///
/// The editor edits a working copy (clone) of the profile. On Save, the clone
/// is committed back to the library. On Cancel, the clone is discarded.
/// Presets are read-only (caller checks before opening).
/// </summary>
public partial class ProfileEditorWindow : Window
{
    private readonly Profile _profile;
    private readonly Action<Profile> _onSave;

    /// <summary>Opens the editor for `profile`. `onSave` is invoked on Save.</summary>
    public ProfileEditorWindow(Profile profile, Action<Profile> onSave)
    {
        InitializeComponent();
        _profile = profile;
        _onSave = onSave;
        LoadFromProfile();
    }

    private void LoadFromProfile()
    {
        NameBox.Text = _profile.Name;
        DescBox.Text = _profile.Description;
        var t = _profile.Tweaks;

        // Render scale mode combo (select by Tag string)
        for (int i = 0; i < RenderScaleModeBox.Items.Count; i++)
        {
            if (RenderScaleModeBox.Items[i] is ComboBoxItem item
                && item.Tag is string tag && tag == t.RenderScaleMode.ToString())
            {
                RenderScaleModeBox.SelectedIndex = i;
                break;
            }
        }

        RenderScaleSlider.Value = t.RenderScale;
        UpdateRenderScaleLabel();
        CustomWidthBox.Text = t.CustomWidth.ToString(CultureInfo.InvariantCulture);
        CustomHeightBox.Text = t.CustomHeight.ToString(CultureInfo.InvariantCulture);
        FpsCapBox.Text = t.FpsCap.ToString(CultureInfo.InvariantCulture);

        SimplifyVsBox.IsChecked = t.SimplifyVertexShaders;
        SimplifyPsBox.IsChecked = t.SimplifyPixelShaders;
        SkipComputeBox.IsChecked = t.SkipComputeShaders;
        DisableTessBox.IsChecked = t.DisableTessellation;
        DisablePostBox.IsChecked = t.DisablePostProcessing;

        LodBiasBox.Text = t.LodBias.ToString(CultureInfo.InvariantCulture);
        MaxMipBox.Text = t.MaxTextureMipLevel.ToString(CultureInfo.InvariantCulture);

        DisableGsBox.IsChecked = t.DisableGeometryShader;
        MsaaOffBox.IsChecked = t.MsaaOff;
        SkipSmallDrawsBox.IsChecked = t.SkipSmallDraws;
        SmallDrawThresholdBox.Text = t.SkipSmallDrawThreshold.ToString(CultureInfo.InvariantCulture);

        UpdatePanelVisibility();
        UpdateSmallDrawVisibility();
    }

    private bool SaveToProfile()
    {
        _profile.Name = string.IsNullOrWhiteSpace(NameBox.Text) ? "Untitled" : NameBox.Text.Trim();
        _profile.Description = DescBox.Text ?? string.Empty;

        var t = _profile.Tweaks;

        // Render scale mode
        if (RenderScaleModeBox.SelectedItem is ComboBoxItem modeItem && modeItem.Tag is string modeTag)
        {
            t.RenderScaleMode = Enum.Parse<RenderScaleMode>(modeTag);
        }

        t.RenderScale = RenderScaleSlider.Value;
        if (!int.TryParse(CustomWidthBox.Text, NumberStyles.Integer, CultureInfo.InvariantCulture, out int cw) || cw < 1)
        {
            MessageBox.Show("Custom width must be a positive integer.", "Invalid", MessageBoxButton.OK, MessageBoxImage.Warning);
            return false;
        }
        t.CustomWidth = cw;
        if (!int.TryParse(CustomHeightBox.Text, NumberStyles.Integer, CultureInfo.InvariantCulture, out int ch) || ch < 1)
        {
            MessageBox.Show("Custom height must be a positive integer.", "Invalid", MessageBoxButton.OK, MessageBoxImage.Warning);
            return false;
        }
        t.CustomHeight = ch;

        if (!int.TryParse(FpsCapBox.Text, NumberStyles.Integer, CultureInfo.InvariantCulture, out int fps) || fps < 0)
        {
            MessageBox.Show("FPS cap must be 0 or a positive integer.", "Invalid", MessageBoxButton.OK, MessageBoxImage.Warning);
            return false;
        }
        t.FpsCap = fps;

        t.SimplifyVertexShaders = SimplifyVsBox.IsChecked == true;
        t.SimplifyPixelShaders = SimplifyPsBox.IsChecked == true;
        t.SkipComputeShaders = SkipComputeBox.IsChecked == true;
        t.DisableTessellation = DisableTessBox.IsChecked == true;
        t.DisablePostProcessing = DisablePostBox.IsChecked == true;

        if (!double.TryParse(LodBiasBox.Text, NumberStyles.Float, CultureInfo.InvariantCulture, out double lod))
        {
            MessageBox.Show("LOD bias must be a number.", "Invalid", MessageBoxButton.OK, MessageBoxImage.Warning);
            return false;
        }
        t.LodBias = lod;

        if (!int.TryParse(MaxMipBox.Text, NumberStyles.Integer, CultureInfo.InvariantCulture, out int mip) || mip < 0)
        {
            MessageBox.Show("Max texture mip level must be 0 or a positive integer.", "Invalid", MessageBoxButton.OK, MessageBoxImage.Warning);
            return false;
        }
        t.MaxTextureMipLevel = mip;

        t.DisableGeometryShader = DisableGsBox.IsChecked == true;
        t.MsaaOff = MsaaOffBox.IsChecked == true;
        t.SkipSmallDraws = SkipSmallDrawsBox.IsChecked == true;
        if (!int.TryParse(SmallDrawThresholdBox.Text, NumberStyles.Integer, CultureInfo.InvariantCulture, out int threshold) || threshold < 1)
        {
            MessageBox.Show("Small-draw threshold must be a positive integer.", "Invalid", MessageBoxButton.OK, MessageBoxImage.Warning);
            return false;
        }
        t.SkipSmallDrawThreshold = threshold;

        return true;
    }

    private void UpdatePanelVisibility()
    {
        if (RenderScaleModeBox.SelectedItem is ComboBoxItem item && item.Tag is string tag)
        {
            ScalePanel.Visibility = tag == "Scale" ? Visibility.Visible : Visibility.Collapsed;
            CustomPanel.Visibility = tag == "Custom" ? Visibility.Visible : Visibility.Collapsed;
        }

    /// Known anti-cheat signatures. Key = display name, Value = list of file/folder names
    /// that indicate the anti-cheat is present in the game folder.
    /// </summary>

        ("EasyAntiCheat", new[] { "EasyAntiCheat", "EasyAntiCheat_x64.dll", "EasyAntiCheat_x86.dll" }),
        ("BattlEye", new[] { "BEService", "BattlEye", "BEClient.dll" }),

        ("nProtect GameGuard", new[] { "GameGuard", "nProtect" }),
        ("XignCode", new[] { "xigncode", "XignCode" }),
        ("Denuvo Anti-Cheat", new[] { "DenuvoAntiCheat" }),
        ("ACE Anti-Cheat (Tencent)", new[] { "ACE-BASE", "ACE-Guard", "AntiCheatExpert" }),
    };

    /// <summary>
    /// Scans the game folder for known anti-cheat signatures.
    /// </summary>
    /// <param name="executablePath">Absolute path to the game .exe.</param>
    /// <returns>Anti-cheat name if detected, empty string if none found.</returns>
    public static string Detect(string executablePath)
    {
        if (string.IsNullOrWhiteSpace(executablePath))
            return string.Empty;

        var exeFile = new FileInfo(executablePath);
        if (!exeFile.Exists)
            return string.Empty;

        var gameDir = exeFile.DirectoryName ?? string.Empty;
        if (string.IsNullOrEmpty(gameDir))
            return string.Empty;

        // Collect entry names from top-level + one level of subfolders.
        // Anti-cheat binaries live next to the exe or in a known subfolder (e.g. EasyAntiCheat/),
        // never buried deep — so AllDirectories would waste time scanning asset trees.
        // try/catch: game folders often contain access-denied subdirs.
        var names = new HashSet<string>(StringComparer.OrdinalIgnoreCase);
        try
        {
            foreach (var e in Directory.EnumerateFileSystemEntries(gameDir))
                names.Add(Path.GetFileName(e));
            foreach (var sub in Directory.EnumerateDirectories(gameDir))
            {
                foreach (var e in Directory.EnumerateFileSystemEntries(sub))
                    names.Add(Path.GetFileName(e));
            }
        }
        catch (UnauthorizedAccessException) { /* partial scan is fine */ }

        foreach (var (name, signatures) in KnownAntiCheats)
        {
            if (signatures.Any(sig => names.Contains(sig)))
                return name;
        }

        return string.Empty;
    }
}

using System.Collections.Generic;
using System.IO;
using System.Linq;
using GameLagReducer.Models;

namespace GameLagReducer.Services;

/// <summary>
/// Detects a game's engine by scanning its folder for signature files.
/// Returns Unknown if no signature matches; caller should then ask the user.
/// </summary>
public static class EngineDetector
{
    /// <summary>
    /// Scans the game folder (parent of the executable) for engine signatures.
    /// </summary>
    /// <param name="executablePath">Absolute path to the game .exe.</param>
    /// <returns>Detected engine, or Unknown if no signature matched.</returns>
    public static GameEngine Detect(string executablePath)
    {
        if (string.IsNullOrWhiteSpace(executablePath))
            return GameEngine.Unknown;

        var exeFile = new FileInfo(executablePath);
        if (!exeFile.Exists)
            return GameEngine.Unknown;

        var gameDir = exeFile.DirectoryName ?? string.Empty;
        if (string.IsNullOrEmpty(gameDir))
            return GameEngine.Unknown;

        // Enumerate top-level dirs ONCE — reused for both Unity checks.
        // TopLevelOnly: engine signatures live next to the exe, not deep in data trees.
        // try/catch: game folders often contain access-denied subdirs that would
        // otherwise throw UnauthorizedAccessException and abort detection.
        List<string> topDirs;
        try { topDirs = Directory.EnumerateDirectories(gameDir).ToList(); }
        catch (UnauthorizedAccessException) { return GameEngine.Unknown; }

        // Unity: UnityPlayer.dll next to exe, or *_Data folder sibling
        var dataDirs = topDirs.Where(d => d.EndsWith("_Data", StringComparison.OrdinalIgnoreCase)).ToList();
        if (File.Exists(Path.Combine(gameDir, "UnityPlayer.dll")) || dataDirs.Count > 0)
            return GameEngine.Unity;

        // Older Unity: boot.config inside *_Data
        if (dataDirs.Any(d => File.Exists(Path.Combine(d, "boot.config"))))
            return GameEngine.Unity;

        // Unreal: shipped games have no .uproject — rely on directory layout.
        // (Source builds keep .uproject at the project root, so TopLevelOnly suffices.)
        if (topDirs.Any(d => string.Equals(Path.GetFileName(d), "Saved", StringComparison.OrdinalIgnoreCase) &&
                             Directory.Exists(Path.Combine(d, "Config"))) ||
            Directory.Exists(Path.Combine(gameDir, "Engine", "Binaries", "Win64")) ||
            Directory.Exists(Path.Combine(gameDir, "Binaries", "Win64")) ||
            Directory.EnumerateFiles(gameDir, "*.uproject").Any())
        {
            return GameEngine.Unreal;
        }

        return GameEngine.Unknown;
    }
}

using System;
using System.Collections.Generic;

namespace GameLagReducer.Services;

/// <summary>
/// Persists the game library and profiles as JSON files in %AppData%\GameLagReducer.
/// Uses System.Text.Json (built-in) — no SQLite dependency needed for a small library.
/// </summary>
public class GameLibrary
{
    private static readonly string AppDataDir = Path.Combine(
        Environment.GetFolderPath(Environment.SpecialFolder.ApplicationData),
        "GameLagReducer");

    private static readonly string GamesPath = Path.Combine(AppDataDir, "games.json");
    private static readonly string ProfilesPath = Path.Combine(AppDataDir, "profiles.json");

    private static readonly JsonSerializerOptions JsonOpts = new()
    {
        WriteIndented = true,
        PropertyNamingPolicy = JsonNamingPolicy.CamelCase,
    };

    private readonly List<Game> _games = new();
    private readonly List<Profile> _profiles = new();

    public IReadOnlyList<Game> Games => _games;
    public IReadOnlyList<Profile> Profiles => _profiles;

    public GameLibrary()
    {
        Load();
    }

    /// <summary>Loads games and profiles from disk. Creates defaults on first run.</summary>
    private void Load()
    {
        Directory.CreateDirectory(AppDataDir);

        if (File.Exists(GamesPath))
        {
            try
            {
                var json = File.ReadAllText(GamesPath);
                var games = JsonSerializer.Deserialize<List<Game>>(json, JsonOpts);
                if (games != null) _games.AddRange(games);
            }
            catch { /* corrupt file — start fresh */ }
        }

        if (File.Exists(ProfilesPath))
        {
            try
            {
                var json = File.ReadAllText(ProfilesPath);
                var profiles = JsonSerializer.Deserialize<List<Profile>>(json, JsonOpts);
                if (profiles != null) _profiles.AddRange(profiles);
            }
            catch { /* corrupt file — start fresh */ }
        }

        // Always ensure built-in presets exist (in case user deleted profiles.json)
        EnsureBuiltInPresets();
    }

    /// <summary>Ensures the built-in presets are present.</summary>
    private void EnsureBuiltInPresets()
    {
        foreach (var preset in Profile.CreateBuiltInPresets())
        {
            if (_profiles.All(p => p.Id != preset.Id))
                _profiles.Add(preset);
        }
    }

    /// <summary>Persists games and profiles to disk.</summary>
    public void Save()
    {
        Directory.CreateDirectory(AppDataDir);
        File.WriteAllText(GamesPath, JsonSerializer.Serialize(_games, JsonOpts));
        File.WriteAllText(ProfilesPath, JsonSerializer.Serialize(_profiles, JsonOpts));
    }

    // ---- Game CRUD ----

    /// <summary>Adds a game to the library. Returns the added game.</summary>
    public Game AddGame(Game game)
    {
        _games.Add(game);
        Save();
        return game;
    }

    /// <summary>Removes a game by id. Returns true if removed.</summary>
    public bool RemoveGame(string id)
    {
        var removed = _games.RemoveAll(g => g.Id == id) > 0;
        if (removed) Save();
        return removed;
    }

    /// <summary>Updates an existing game by id.</summary>
    public void UpdateGame(Game game)
    {
        var idx = _games.FindIndex(g => g.Id == game.Id);
        if (idx >= 0)
        {
            _games[idx] = game;
            Save();
        }
    }

    /// <summary>Gets a game by id, or null.</summary>
    public Game? GetGame(string id) => _games.FirstOrDefault(g => g.Id == id);

    // ---- Profile CRUD ----

    /// <summary>Adds a custom profile. Presets cannot be added via this method.</summary>
    public Profile AddProfile(Profile profile)
    {
        if (profile.Kind == ProfileKind.Preset)
            throw new ArgumentException("Cannot add a preset profile. Presets are built-in.", nameof(profile));
        _profiles.Add(profile);
        Save();
        return profile;
    }

    /// <summary>Removes a custom profile by id. Presets cannot be removed.</summary>
    public bool RemoveProfile(string id)
    {
        var profile = _profiles.FirstOrDefault(p => p.Id == id);
        if (profile == null || profile.IsPreset) return false;
        var removed = _profiles.RemoveAll(p => p.Id == id) > 0;
        if (removed) Save();
        return removed;
    }

    /// <summary>Updates an existing custom profile. Presets are read-only.</summary>
    public void UpdateProfile(Profile profile)
    {
        var idx = _profiles.FindIndex(p => p.Id == profile.Id);
        if (idx < 0) return;
        if (_profiles[idx].IsPreset)
            throw new InvalidOperationException("Cannot modify a built-in preset. Create a custom profile instead.");
        _profiles[idx] = profile;
        Save();
    }

    /// <summary>Gets a profile by id, or null.</summary>
    public Profile? GetProfile(string id) => _profiles.FirstOrDefault(p => p.Id == id);

    /// <summary>Exports a profile to a JSON file for sharing.</summary>
    public void ExportProfile(string id, string outputPath)
    {
        var profile = GetProfile(id) ?? throw new KeyNotFoundException($"Profile '{id}' not found.");
        File.WriteAllText(outputPath, JsonSerializer.Serialize(profile, JsonOpts));
    }

    /// <summary>Imports a profile from a JSON file. Returns the imported profile.</summary>
    public Profile ImportProfile(string inputPath)
    {
        var json = File.ReadAllText(inputPath);
        var profile = JsonSerializer.Deserialize<Profile>(json, JsonOpts)
            ?? throw new InvalidDataException("Invalid profile file.");
        // Force custom kind + new id to avoid colliding with presets
        profile.Kind = ProfileKind.Custom;
        profile.Id = Guid.NewGuid().ToString("N");
        _profiles.Add(profile);
        Save();
        return profile;
    }

```cs
using System;
using System.IO;
using System.IO.Pipes;
using System.Text;
using System.Text.Json;
using System.Threading;
using System.Threading.Tasks;
using GameLagReducer.Models;

namespace GameLagReducer.Services;

/// <summary>
/// Named-pipe IPC server. The Launcher creates the server before injecting the DLL;
/// the injected HookDll connects as a client to receive the tweak profile and send
/// back log/telemetry lines.
///
/// Protocol (line-based, newline-delimited JSON):
///   Launcher → DLL:  {"cmd":"profile","profile":{...Tweaks JSON...}}
///   Launcher → DLL:  {"cmd":"ping"}
///   DLL → Launcher:  {"log":"message"}
///   DLL → Launcher:  {"telemetry":"fps","value":60}
///   DLL → Launcher:  {"ack":"profile"}
///
/// Uses System.IO.Pipes (built-in) — no external dependency.
/// </summary>
public class IpcServer : IDisposable
{
    // Pipe name must match HookDll's client. Keep in sync with HookDll/src/ipc_client.cpp.
    public const string PipeName = "GameLagReducer_IPC";

    private NamedPipeServerStream? _server;
    private StreamReader? _reader;
    private StreamWriter? _writer;
    private CancellationTokenSource? _cts;
    private Task? _listenTask;

    /// <summary>Raised when a log line is received from the DLL.</summary>
    public event Action<string>? LogReceived;

    /// <summary>Raised when telemetry is received from the DLL.</summary>
    public event Action<string, double>? TelemetryReceived;

    /// <summary>Raised when the DLL acknowledges the profile was applied.</summary>
    public event Action? ProfileAcknowledged;

    /// <summary>Raised when the DLL disconnects (game closed or hook lost).</summary>
    public event Action? Disconnected;

    private static readonly JsonSerializerOptions JsonOpts = new()
    {
        PropertyNamingPolicy = JsonNamingPolicy.CamelCase,
    };

    /// <summary>
    /// Starts the pipe server and begins listening for the DLL to connect.
    /// Non-blocking — runs in a background task.
    /// </summary>
    public void Start()
    {
        _cts = new CancellationTokenSource();
        _listenTask = Task.Run(() => ListenLoop(_cts.Token));
    }

    /// <summary>Stops the server and closes the connection.</summary>
    public void Stop()
    {
        _cts?.Cancel();
        try { _server?.Dispose(); } catch { }
        try { _reader?.Dispose(); } catch { }
        try { _writer?.Dispose(); } catch { }
    }

    /// <summary>
    /// Sends the tweak profile to the connected DLL. No-op if not connected yet.
    /// </summary>
    public void SendProfile(Tweaks tweaks)
    {
        if (_writer == null) return;
        var msg = new { cmd = "profile", profile = tweaks };
        var json = JsonSerializer.Serialize(msg, JsonOpts);
        try
        {
            _writer.WriteLine(json);
            _writer.Flush();
        }
        catch { /* connection lost — Disconnected event will fire from listener */ }
    }

    /// <summary>Sends a ping to keep the connection alive / check liveness.</summary>
    public void SendPing()
    {
        if (_writer == null) return;
        try
        {
            _writer.WriteLine("{\"cmd\":\"ping\"}");
            _writer.Flush();
        }
        catch { }
    }

    private async void ListenLoop(CancellationToken ct)
    {
        while (!ct.IsCancellationRequested)
        {
            // Create a fresh server stream for each connection (supports reconnect).
            _server = new NamedPipeServerStream(PipeName,
                PipeDirection.InOut, 1,
                PipeTransmissionMode.Byte,
                PipeOptions.Asynchronous);

            try
            {
                await _server.WaitForConnectionAsync(ct);
            }
            catch (OperationCanceledException) { return; }
            catch (Exception ex)
            {
                LogReceived?.Invoke($"[IPC] Listen error: {ex.Message}");
                _server.Dispose();
                return;
            }

            _reader = new StreamReader(_server, new UTF8Encoding(false));
            _writer = new StreamWriter(_server, new UTF8Encoding(false)) { AutoFlush = true };
            LogReceived?.Invoke("[IPC] HookDll connected.");

            // Read loop for this connection
            try
            {
                string? line;
                while ((line = await _reader.ReadLineAsync(ct)) != null)
                {
                    HandleMessage(line);
                }
            }
            catch (OperationCanceledException) { return; }
            catch { /* connection dropped — fall through to disconnect */ }

            LogReceived?.Invoke("[IPC] HookDll disconnected.");
            Disconnected?.Invoke();
            try { _reader.Dispose(); _reader = null; } catch { }
            try { _writer.Dispose(); _writer = null; } catch { }
            try { _server.Dispose(); _server = null; } catch { }
        }
    }

    private void HandleMessage(string line)
    {
        try
        {
            using var doc = JsonDocument.Parse(line);
            if (!doc.RootElement.TryGetProperty("log", out var logVal))
            {
                if (doc.RootElement.TryGetProperty("telemetry", out var telVal) &&
                    doc.RootElement.TryGetProperty("value", out var valEl) &&
                    telVal.GetString() == "fps" && valEl.TryGetDouble(out var fps))
                {
                    TelemetryReceived?.Invoke("fps", fps);
                }
                else if (doc.RootElement.TryGetProperty("ack", out var ackVal) &&
                         ackVal.GetString() == "profile")
                {
                    ProfileAcknowledged?.Invoke();
                }
                return;
            }
            LogReceived?.Invoke(logVal.GetString() ?? string.Empty);
        }
        catch (JsonException)
        {
            // Non-JSON line — treat as plain log text
            LogReceived?.Invoke(line);
        }
    }

    public void Dispose()
    {
        Stop();
        _cts?.Dispose();
    }
}

using System;
using System.IO;
using System.Threading;
using System.Threading.Tasks;
using GameLagReducer.Models;

namespace GameLagReducer.Services;

/// <summary>
/// Orchestrates a single game launch: starts the IPC server, injects the HookDll,
/// sends the tweak profile, and monitors the game process until exit.
///
/// Decoupled from the UI so it can be unit-tested and reused. The UI subscribes to
/// events for status updates.
/// </summary>
public class LaunchOrchestrator : IDisposable
{
    private readonly GameLibrary _library;
    private readonly IpcServer _ipc = new();
    private IntPtr _processHandle = IntPtr.Zero;
    private Task? _monitorTask;
    private CancellationTokenSource? _cts;

    /// <summary>Raised with a status/log message (UI shows in log panel).</summary>
    public event Action<string>? Status;

    /// <summary>Raised when the launched game exits.</summary>
    public event Action? GameExited;

    /// <summary>True if a game is currently launched and being monitored.</summary>
    public bool IsRunning => _processHandle != IntPtr.Zero;

    public LaunchOrchestrator(GameLibrary library)
    {
        _library = library;
        _ipc.LogReceived += msg => Status?.Invoke($"[hook] {msg}");
        _ipc.TelemetryReceived += (name, value) => Status?.Invoke($"[telemetry] {name}={value:F1}");
        _ipc.ProfileAcknowledged += () => Status?.Invoke("[hook] Profile applied.");
        _ipc.Disconnected += () => Status?.Invoke("[hook] HookDll disconnected.");
    }

    /// <summary>
    /// Launches a game with the HookDll injected and the resolved profile applied.
    /// </summary>
    /// <param name="game">The game to launch.</param>
    /// <param name="dllPath">Absolute path to HookDll.dll.</param>
    /// <returns>True if the launch started successfully.</returns>
    public bool Launch(Game game, string dllPath)
    {
        if (IsRunning)
        {
            Status?.Invoke("A game is already running. Wait for it to exit first.");
            return false;
        }

        // Resolve the profile: game's profile, or fall back to the Medium preset.
        var profile = !string.IsNullOrEmpty(game.ProfileId)
            ? _library.GetProfile(game.ProfileId)
            : _library.GetProfile("preset-medium");
        profile ??= _library.GetProfile("preset-medium");
        if (profile == null)
        {
            Status?.Invoke("No profile available (not even built-in presets).");
            return false;
        }

        Status?.Invoke($"Launching '{game.Name}' with profile '{profile.Name}'...");

        // Start the IPC server BEFORE injecting so the DLL can connect immediately.
        _cts = new CancellationTokenSource();
        _ipc.Start();
        Status?.Invoke("[ipc] Server started, waiting for HookDll to connect...");

        // Inject + launch. The game starts suspended, DLL loads, then resumes.
        var result = ProcessInjector.LaunchWithInjection(
            game.ExecutablePath, game.WorkingDirectory, game.Arguments, dllPath);

        if (!result.Success)
        {
            Status?.Invoke($"[error] Launch failed: {result.Error}");
            _ipc.Stop();
            _cts?.Dispose();
            _cts = null;
            return false;
        }

        _processHandle = result.ProcessHandle;
        Status?.Invoke($"[ok] Game launched (PID {result.ProcessId}). Sending profile...");

        // Send the profile. The DLL's IPC worker will receive it.
        // Small delay to let the DLL connect — it spawns a thread in DllMain.
        _ = SendProfileWhenConnected(profile.Tweaks);

        // Monitor the process for exit (background task, non-blocking).
        _monitorTask = Task.Run(() => MonitorProcess(result.ProcessId));

        // Update last-launched timestamp.
        game.LastLaunchedAt = DateTime.UtcNow.ToString("o");
        _library.UpdateGame(game);

        return true;
    }

    /// <summary>Sends the profile once the DLL has connected (retries for up to 5s).</summary>
    private async Task SendProfileWhenConnected(Tweaks tweaks)
    {
        // The DLL connects in a background thread; give it a moment.
        // We retry SendProfile a few times — the IPC server queues nothing, so if the
        // DLL isn't connected yet the write is silently dropped. Polling IsConnected
        // isn't exposed, so we just retry with small delays.
        for (int i = 0; i < 20; i++)
        {
            _ipc.SendProfile(tweaks);
            await Task.Delay(250);
            if (_cts?.IsCancellationRequested == true) return;
        }
        Status?.Invoke("[ipc] Profile sent (will be applied when DLL connects).");
    }

    /// <summary>Waits for the game process to exit, then cleans up.</summary>
    private void MonitorProcess(uint pid)
    {
        try
        {
            // Wait on the process handle (efficient — no polling).
            ProcessInjector.WaitForExit(_processHandle);
        }
        catch { /* process may have already exited */ }

        Status?.Invoke($"[ok] Game (PID {pid}) exited.");
        ProcessInjector.CloseProcessHandle(_processHandle);
        _processHandle = IntPtr.Zero;
        _ipc.Stop();
        _cts?.Cancel();
        _cts?.Dispose();
        _cts = null;
        GameExited?.Invoke();
    }

    /// <summary>Stops monitoring and disconnects IPC. Does NOT kill the game.</summary>
    public void Dispose()
    {
        _cts?.Cancel();
        if (_processHandle != IntPtr.Zero)
            ProcessInjector.CloseProcessHandle(_processHandle);
        _ipc.Dispose();
    }
}

using System;
using System.ComponentModel;
using System.Diagnostics;
using System.IO;
using System.Runtime.InteropServices;

namespace GameLagReducer.Services;

/// <summary>
/// Injects a DLL into a target process by:
/// 1. CreateProcess with CREATE_SUSPENDED (game starts paused)
/// 2. VirtualAllocEx a buffer in the target for the DLL path string
/// 3. WriteProcessMemory the DLL path into the buffer
/// 4. CreateRemoteThread calling LoadLibraryW with the buffer as argument
/// 5. Wait for the thread (LoadLibrary returns) — DLL is now loaded
/// 6. ResumeThread to let the game run with the hook active
///
/// This is the classic "LoadLibrary injection" technique. No external dependency.
/// </summary>
public static class ProcessInjector
{
    // ---- Win32 constants ----
    private const uint CREATE_SUSPENDED = 0x00000004;
    private const uint CREATE_UNICODE_ENVIRONMENT = 0x00000400;
    private const uint MEM_COMMIT = 0x00001000;
    private const uint MEM_RESERVE = 0x00002000;
    private const uint PAGE_READWRITE = 0x04;
    private const uint INFINITE = 0xFFFFFFFF;

    // ---- Win32 structs ----
    [StructLayout(LayoutKind.Sequential, CharSet = CharSet.Unicode)]
    private struct STARTUPINFO
    {
        public int cb;
        public string lpReserved;
        public string lpDesktop;
        public string lpTitle;
        public uint dwX, dwY, dwXSize, dwYSize;
        public uint dwXCountChars, dwYCountChars;
        public uint dwFillAttribute;
        public uint dwFlags;
        public short wShowWindow;
        public short cbReserved2;
        public IntPtr lpReserved2;
        public IntPtr hStdInput, hStdOutput, hStdError;
    }

    [StructLayout(LayoutKind.Sequential)]
    private struct PROCESS_INFORMATION
    {
        public IntPtr hProcess;
        public IntPtr hThread;
        public uint dwProcessId;
        public uint dwThreadId;
    }

    [StructLayout(LayoutKind.Sequential)]
    private struct SECURITY_ATTRIBUTES
    {
        public int nLength;
        public IntPtr lpSecurityDescriptor;
        public bool bInheritHandle;
    }

    // ---- Win32 imports ----
    [DllImport("kernel32.dll", SetLastError = true, CharSet = CharSet.Unicode)]
    private static extern bool CreateProcessW(
        string lpApplicationName, string lpCommandLine, IntPtr lpProcessAttributes,
        IntPtr lpThreadAttributes, bool bInheritHandles, uint dwCreationFlags,
        IntPtr lpEnvironment, string lpCurrentDirectory,
        ref STARTUPINFO lpStartupInfo, out PROCESS_INFORMATION lpProcessInformation);

    [DllImport("kernel32.dll", SetLastError = true)]
    private static extern IntPtr VirtualAllocEx(IntPtr hProcess, IntPtr lpAddress,

    [DllImport("kernel32.dll", SetLastError = true)]

    /// <summary>
    /// Result of an injection attempt. Carries the process id (for monitoring)
    /// and an error message if injection failed.
    /// </summary>
    public class InjectionResult
    {
        public bool Success { get; init; }
        public uint ProcessId { get; init; }
        public IntPtr ProcessHandle { get; init; }
        public string? Error { get; init; }

        public static InjectionResult Ok(uint pid, IntPtr hProcess) => new()
        {
            Success = true, ProcessId = pid, ProcessHandle = hProcess,
        };

        public static InjectionResult Fail(string error) => new() { Success = false, Error = error };
    }

    /// <summary>
    /// Launches a game with a DLL injected. The game starts suspended, the DLL is
    /// loaded via LoadLibrary in a remote thread, then the main thread is resumed.
    /// </summary>
    /// <param name="executablePath">Absolute path to the game .exe.</param>
    /// <param name="workingDirectory">Working directory for the game.</param>
    /// <param name="arguments">Extra command-line arguments (may be empty).</param>
    /// <param name="dllPath">Absolute path to the DLL to inject.</param>
    /// <returns>Injection result with process id + handle (for monitoring), or error.</returns>
    public static InjectionResult LaunchWithInjection(
        string executablePath, string workingDirectory, string arguments, string dllPath)
    {
        if (!File.Exists(executablePath))
            return InjectionResult.Fail($"Game executable not found: {executablePath}");
        if (!File.Exists(dllPath))
            return InjectionResult.Fail($"HookDll not found: {dllPath}. Build HookDll first (see README).");

        // Quote paths in the command line so spaces don't break parsing.
        var commandLine = $"\"{executablePath}\"";
        if (!string.IsNullOrWhiteSpace(arguments))
            commandLine += " " + arguments;

        var si = new STARTUPINFO { cb = Marshal.SizeOf<STARTUPINFO>() };

        // 1. Create the game process suspended so we can inject before it runs.
        if (!CreateProcessW(
            executablePath, commandLine, IntPtr.Zero, IntPtr.Zero, false,
            CREATE_SUSPENDED | CREATE_UNICODE_ENVIRONMENT,
            IntPtr.Zero, workingDirectory, ref si, out var pi))
        {
            var err = new Win32Exception(Marshal.GetLastWin32Error()).Message;
            return InjectionResult.Fail($"CreateProcess failed: {err}");
        }

        try
        {
            // 2. Allocate memory in the target process for the DLL path (UTF-16).
            var dllPathW = dllPath + "\0";
            var bytes = System.Text.Encoding.Unicode.GetBytes(dllPathW);
            var remoteBuf = VirtualAllocEx(pi.hProcess, IntPtr.Zero, (uint)bytes.Length,
                MEM_COMMIT | MEM_RESERVE, PAGE_READWRITE);
            if (remoteBuf == IntPtr.Zero)
            {
                var err = new Win32Exception(Marshal.GetLastWin32Error()).Message;
                return InjectionResult.Fail($"VirtualAllocEx failed: {err}");
            }

            try
            {
                // 3. Write the DLL path into the target process memory.
                if (!WriteProcessMemory(pi.hProcess, remoteBuf, bytes, (uint)bytes.Length, out _))
                {
                    var err = new Win32Exception(Marshal.GetLastWin32Error()).Message;
                    return InjectionResult.Fail($"WriteProcessMemory failed: {err}");
                }

                // 4. Get LoadLibraryW address from kernel32 (same address in any process on modern Windows).
                var kernel32 = GetModuleHandleW("kernel32.dll");
                var loadLibraryAddr = GetProcAddress(kernel32, "LoadLibraryW");
                if (loadLibraryAddr == IntPtr.Zero)
                    return InjectionResult.Fail("GetProcAddress(LoadLibraryW) failed.");

                // 5. Create a remote thread that calls LoadLibraryW(dllPath).
                var hThread = CreateRemoteThread(pi.hProcess, IntPtr.Zero, 0,
                    loadLibraryAddr, remoteBuf, 0, out _);
                if (hThread == IntPtr.Zero)
                {
                    var err = new Win32Exception(Marshal.GetLastWin32Error()).Message;
                    return InjectionResult.Fail($"CreateRemoteThread failed: {err}");
                }

                try
                {
                    // 6. Wait for LoadLibrary to return (DLL is now loaded + DllMain called).
                    WaitForSingleObject(hThread, INFINITE);

                    // Check the exit code — LoadLibrary returns the module handle (non-zero on success).
                    if (!GetExitCodeThread(hThread, out var exitCode) || exitCode == 0)
                        return InjectionResult.Fail("LoadLibraryW returned 0 — DLL failed to load. Check hook.log.");
                }
                finally
                {
                    CloseHandle(hThread);
                }
            }
            finally
            {
                // Free the remote buffer regardless of success.
                VirtualFreeEx(pi.hProcess, remoteBuf, 0, 0x8000 /* MEM_RELEASE */);
            }

            // 7. Resume the main thread — game now runs with the hook active.
            ResumeThread(pi.hThread);

            // We keep the process handle open so the caller can wait for exit.
            // The caller is responsible for closing it.
            return InjectionResult.Ok(pi.dwProcessId, pi.hProcess);
        }
        catch (Exception ex)
        {
            // On any unexpected error, kill the suspended process to avoid a zombie.
            try { Process.GetProcessById((int)pi.dwProcessId)?.Kill(); } catch { }
            CloseHandle(pi.hThread);
            CloseHandle(pi.hProcess);
            return InjectionResult.Fail($"Injection exception: {ex.Message}");
        }
    }

    /// <summary>
    /// Waits for the injected process to exit. Returns true if it exited normally.
    /// </summary>
    public static bool WaitForExit(IntPtr processHandle, uint timeoutMs = INFINITE)
    {
        return WaitForSingleObject(processHandle, timeoutMs) == 0; // WAIT_OBJECT_0
    }

    /// <summary>
    /// Closes a process handle returned by LaunchWithInjection.
    /// </summary>
    public static void CloseProcessHandle(IntPtr processHandle)
    {
        if (processHandle != IntPtr.Zero) CloseHandle(processHandle);
    }
}

using System;
using System.IO;
using System.Linq;

namespace GameLagReducer.Services;

/// <summary>
/// Manages the shader cache directory (%AppData%\GameLagReducer\shaders\).
///
/// The cache is written by the HookDll (C++ side, Phase 4) as:
///   <hash>.bin      — original bytecode
///   <hash>.asm      — disassembled DXBC text
///   <hash>.repl.bin — user-provided replacement (optional)
///
/// This service (Phase 8) lets the Launcher inspect and clear the cache from
/// the Settings tab. It uses System.IO directly — no IPC needed, since the
/// cache is plain files on disk shared between the Launcher and HookDll via
/// the same %AppData% path.
/// </summary>
public static class ShaderCacheManager
{
    private static readonly string CacheDir = Path.Combine(
        Environment.GetFolderPath(Environment.SpecialFolder.ApplicationData),
        "GameLagReducer", "shaders");

    /// <summary>Cache directory path. Created on first cache write by HookDll.</summary>
    public static string DirectoryPath => CacheDir;

    /// <summary>Total size of all files in the cache, in bytes. 0 if dir missing.</summary>
    public static long GetSize()
    {
        if (!Directory.Exists(CacheDir)) return 0;
        try
        {
            long total = 0;
            foreach (var f in Directory.EnumerateFiles(CacheDir, "*", SearchOption.AllDirectories))
            {
                try { total += new FileInfo(f).Length; } catch { /* file deleted mid-scan */ }
            }
            return total;
        }
        catch { return 0; }
    }

    /// <summary>Number of cached shader files (each shader = up to 3 files).</summary>
    public static int GetFileCount()
    {
        if (!Directory.Exists(CacheDir)) return 0;
        try
        {
            int count = 0;
            foreach (var _ in Directory.EnumerateFiles(CacheDir, "*", SearchOption.AllDirectories))
                count++;
            return count;
        }
        catch { return 0; }
    }

    /// <summary>
    /// Returns both size and file count in a single enumeration (avoids scanning
    /// the cache directory twice when the Settings tab refreshes).
    /// </summary>
    public static (long size, int count) GetSizeAndCount()
    {
        if (!Directory.Exists(CacheDir)) return (0, 0);
        try
        {
            long total = 0;
            int count = 0;
            foreach (var f in Directory.EnumerateFiles(CacheDir, "*", SearchOption.AllDirectories))
            {
                count++;
                try { total += new FileInfo(f).Length; } catch { }
            }
            return (total, count);
        }
        catch { return (0, 0); }
    }

    /// <summary>Deletes all files in the cache directory. Returns true on success.</summary>
    public static bool Clear()
    {
        if (!Directory.Exists(CacheDir)) return true;
        try
        {
            Directory.Delete(CacheDir, recursive: true);
            return true;
        }
        catch { return false; }
    }
}

```md
# Game Lag Reducer

> **GitHub:** https://github.com/baobao1044/GameLagReducer

> Open-source launcher that reduces GPU render load by hooking into a game's graphics pipeline and simplifying shaders at runtime.

⚠️ **WARNING**: Injecting into games with anti-cheat (EasyAntiCheat, BattlEye, Vanguard, etc.) may result in a **permanent ban**. The launcher warns you before launching such games, but you use it at your own risk.

## How it works

```
Launcher (WPF .NET 8)  ──inject──▶  HookDll (C++ native)
- Game library (JSON)              - DXGI/D3D11 vtable hook (Present/ResizeBuffers/Create*Shader/CreateTexture2D/Draw*)
- Profile (preset + per-game)       - OpenGL IAT hook (wglSwapBuffers/glShaderSource)
- Engine auto-detect                - Vulkan IAT hook (vkGetDeviceProcAddr -> vkCreate*Pipelines)
- Anti-cheat detect                - Shader capture → cache → user replacement
- IPC named pipe                   - Flat pixel shader (GLSL/DXBC), no-op compute (GLSL)
- Shader cache management          - Tessellation/GS disable, MSAA off (3 APIs)
                                    - FPS cap (Present throttle), render scale (ResizeBuffers)
                                    - Experimental small-draw cull (D3D11, off by default)
```

Instead of editing config files, the launcher injects a native DLL into the game process that hooks the graphics API (DXGI/D3D11/OpenGL/Vulkan), captures shaders at runtime, and applies runtime tweaks (skip compute shaders, render scale, FPS cap, flat pixel shaders, tessellation/GS off, MSAA off). Users can also drop replacement shader bytecode (`.repl.bin`) into the cache to swap shaders on the fly. This reduces GPU workload and increases FPS.

## Visual reduction — what it does and what it cannot

The `Flat (Max FPS)` preset (and the underlying tweak knobs) cut GPU load by:

- **Flat pixel shaders** — replaces pixel/fragment shaders with a flat-grey
  pass-through (D3D11 PS, GLSL fragment, Vulkan fragment via Stage-2 only).
  The **vertex shader is intentionally kept**, so geometry silhouettes remain
  correct (you still see shapes, just flat grey).
- **Compute shaders off** — replaces compute with a no-op (D3D11, GLSL).
- **Tessellation off** — `E_FAIL` on hull/domain shaders (D3D11); strips
  tess-control/eval stages from Vulkan graphics pipelines. Removes per-vertex
  amplification on high-poly terrain/characters.
- **Geometry shaders off** — `E_FAIL` on `CreateGeometryShader` (D3D11); strips
  the geometry stage from Vulkan graphics pipelines. Removes another
  amplification stage.
- **MSAA off** — forces `SampleDesc.Count=1` on D3D11 render targets;
  `glDisable(GL_MULTISAMPLE)` for OpenGL; `rasterizationSamples=1` on Vulkan
  pipelines.
- **Low mips** — LOD bias + max-mip clamp on D3D11 samplers.
- **Experimental small-draw cull** (D3D11 only, off by default) — drops draw
  calls whose `IndexCount*InstanceCount` is below a threshold. ⚠️ Can hide UI
  text/quads (which often have low index counts), so it ships behind its own
  checkbox.

### What it **cannot** do (honest limit)

**Mesh / triangle decimation is not possible at the API-hook layer.** By the
time vertex/index buffers are visible to the hook, the mesh is already on the
GPU, and constant-buffer layouts (where the transform matrices live) are
engine-specific. So poly reduction here comes only from disabling tessellation
and geometry shaders (which remove *amplification*) plus the optional
small-draw cull — not from re-simplifying meshes. On scenes that lean on
tessellation/GS this is a large win; on scenes that don't, it's modest. For
mesh simplification, use the engine's own low-quality settings.

## Usage

1. Launch `GameLagReducer.exe` (runs as administrator — required for DLL injection).
2. **Game Library** tab → click **+ Add Game** → browse to the game's `.exe`.
3. The launcher auto-detects the engine (Unity/Unreal/Custom) and flags anti-cheat.
4. **Profiles** tab → click **+ New Profile** → name it, tweak the knobs, Save.
   - Four built-in presets (Low/Medium/Balanced/Flat) are read-only starting points.
   - `Flat (Max FPS)` is the recommended max-FPS tier: flat pixel shaders,
     compute/tessellation/geometry shaders off, MSAA off, low mips. It keeps the
     vertex shader so silhouettes stay correct (flat grey scene). It supersedes
     `Low (Max FPS)`, which uses the visually-breaking simplified-vertex path.
   - Export/Import profiles as JSON to share with the community.
5. Back in **Game Library**, pick a profile from the dropdown next to each game.
6. Click **▶ Launch**. If the game has anti-cheat, confirm the warning dialog.
7. Watch the **Launch Log** tab for live hook status and telemetry.
8. **Settings** tab → inspect/clear the shader cache as needed.

break (flat grey scene is fine). This is the "tối đa ép FPS" / "flat tối thiểu,
game chạy" tier.

This closes three gaps in the current project, surfaced during review:

1. **Shader removal is shallow/wrong.** `SimplifyVertexShaders` replaces the VS
   with one that only copies `pos` (no MVP multiply) — geometry renders wrong,
   it is not "simplification". Pixel shader "simplification" = nuke *every* PS to
   flat grey, with no post-process detection. OpenGL `glShaderSource` and Vulkan
   `vkCreateShaderModule` only *capture* shaders, they never modify them — so on
   GL/VK games the tweaks do nothing.
2. **Visual removal is nearly absent.** No MSAA disabling (swapchain creation is
   not hooked, only `ResizeBuffers`). No post-process pass detection (no
   fullscreen-triangle/quad heuristic). No draw-call skipping.
3. **Poly reduction is one knob.** Only `DisableTessellation`. No geometry
   shader disabling, no draw-call cull, no LOD forcing.

## 2. Non-goals

- True mesh decimation / re-tessellation. By the time vertex/index buffers are
  visible at the API layer the mesh is already on the GPU and constant-buffer
  layouts are engine-specific. Out of scope; documented as a hard limit.
- D3D12. Not currently hooked; lifting to D3D12 (command-list/PSO model) is a
  separate, larger project.
- Anti-cheat compatibility. Existing README warning still applies.
- Keeping visuals pretty. Flat grey is the intended result.

## 3. Design principles

- **Default-safe, opt-in aggressive.** Built-in tweak flags default off; the new
  `MaxFPS Flat` preset turns the safe aggressive set on. Risky knobs (small-draw
  cull) ship **off** even inside the preset, so a user who just selects the
  preset still gets "game runs".
- **Never replace the vertex shader.** Flat is achieved by replacing only the
  **pixel/fragment** stage. Keeping the original VS preserves silhouette (MVP
  still applied by the game's VS). The existing `SimplifyVertexShaders` path is
  left in code but is **not** used by the preset and is documented as
  visually-breaking.
- **Per-API, stage-aware.** D3D11 gets the full set. GL and Vulkan get the
  shader-stage flattening + tessellation/GS/MSAA disabling ported up from
  capture-only. Small-draw cull is D3D11-only this round (VK draw hooks are the
  highest-risk surface; GL draw cull is low value).
- **Fail soft.** Any hook that cannot do its job logs and forwards the original
  call unchanged. The host game must never crash because of a hook.

## 4. Changes by component

### 4.1 Shared data — `Tweaks` (C# `Launcher/Models/Tweaks.cs` + C++ struct in
`HookDll/include/ipc_client.h`)

Add four fields, mirrored across both sides and carried over the existing IPC
JSON:

| Field | C# type | C++ type | Default | Meaning |
|-------|---------|----------|---------|---------|
| `DisableGeometryShader` | `bool` | `bool` | `false` | `E_FAIL` `CreateGeometryShader` / strip GS stage from VK pipelines |
| `MsaaOff` | `bool` | `bool` | `false` | Force `SampleDesc.Count=1` on D3D11 textures; `glDisable(GL_MULTISAMPLE)`; VK `rasterizerSamples=1` |
| `SkipSmallDraws` | `bool` | `bool` | `false` | (D3D11 only) Drop draw calls below threshold — **experimental, off by default even in preset** |
| `SkipSmallDrawThreshold` | `int` | `int` | `256` | `IndexCount*InstanceCount < threshold` → drop the draw |

`Tweaks.HasAnyTweak` (C#) extended to include the new bools. IPC struct byte
layout stays JSON (already hand-rolled), so no ABI concern.

### 4.2 Launcher

- **New built-in preset** `Flat (Max FPS)` (id `preset-flat`), added in
  `Launcher/Models/Profile.cs::CreateBuiltInPresets()` alongside the existing
  Low/Medium/Balanced. Sets:
  `SimplifyPixelShaders=true`, `SkipComputeShaders=true`,
  `DisableTessellation=true`, `DisablePostProcessing=true`,
  `DisableGeometryShader=true`, `MsaaOff=true`, `MaxTextureMipLevel=8`,
  `LodBias=+2.0`, `RenderScaleMode=Scale`, `RenderScale=0.5`,
  `SimplifyVertexShaders=false`, `SkipSmallDraws=false`.
- **Relationship to the existing `Low (Max FPS)` preset:** that preset sets
  `SimplifyVertexShaders=true`, which uses the visually-breaking VS path (no MVP
  → warped geometry). It is left unchanged to avoid surprising games already
  assigned to it, but `Flat (Max FPS)` supersedes it as the recommended
  max-FPS tier because it keeps the VS (correct silhouette) while flattening via
  the pixel stage + the new poly/visual knobs. The README notes this.
- **Profile editor** (`ProfileEditorWindow.xaml`): add checkboxes + a numeric
  spinner for `SkipSmallDrawThreshold` under a clearly-marked "Experimental"
  group. The threshold control is disabled unless `SkipSmallDraws` is checked.
- **README**: update the "How it works" + status table; add a "Visual reduction
  — what it does and what it cannot" section explaining the mesh-decimation hard
  limit honestly.

### 4.3 D3D11 (`HookDll/src/d3d11_hook.cpp` + `shader_modifier.cpp`)

1. **Geometry shader off.** Hook `CreateGeometryShader` (vtable 13) and
   `CreateGeometryShaderWithStreamOutput` (vtable 14). When
   `DisableGeometryShader`, return `E_FAIL` and null the out-pointer (same
   pattern as the existing hull/domain hook). Engines fall back to the VS path.
2. **MSAA off.** Hook `CreateTexture2D` (vtable 5). When `MsaaOff` and the
   desc's `SampleDesc.Count > 1`, mutate a copy of the desc to `Count=1,
   Quality=0` before forwarding. Affects render-target / depth-stencil textures
   (where MSAA lives); shader-resource textures with `Count=1` are untouched.
3. **Small-draw cull (experimental).** Obtain the immediate context from the
   dummy device (`device->GetImmediateContext(&ctx)`), vtable-hook
   `DrawIndexed` (12), `DrawIndexedInstanced` (13), `Draw` (14),
   `DrawInstanced` (15), `DrawAuto` (16), `DrawIndexedInstancedIndirect` (18),
   `DrawInstancedIndirect` (19) on `ID3D11DeviceContext`. Per function, the
   count used for the threshold test is:
   - `DrawIndexed(IndexCount, StartIndex, BaseVertex)` → `IndexCount`
   - `DrawIndexedInstanced(IndexCountPerInstance, InstanceCount, …)` → `IndexCountPerInstance * InstanceCount`
   - `Draw(VertexCount, StartVertex)` → `VertexCount`
   - `DrawInstanced(VertexCountPerInstance, InstanceCount, …)` → `VertexCountPerInstance * InstanceCount`
   - `DrawAuto()` → never culled (count not known on CPU); pass through.
   - `DrawIndexedInstancedIndirect` / `DrawInstancedIndirect` → never culled
     (count lives in a GPU buffer, not visible on CPU); pass through.
   If `SkipSmallDraws` and `count < SkipSmallDrawThreshold`, return without
   calling the original. Guard with a logged cap (first 5 dropped draws) to
   avoid log spam.
   - **Known risk, documented:** small UI quads / text draws often have low
     index counts and can be dropped, causing missing HUD. This is why the flag
     is off by default and lives behind a separate checkbox.
4. **Flat pixel shader** — keep the existing `GetSimplePixelBytecode` flat-grey
   shader; it already does the right thing for the "flat" tier. No change to the
   shader body. The `SimplifyPixelShaders || DisablePostProcessing` condition
   stays (both still route to flat).
5. **Vertex shader** — do **not** replace. Leave the existing
   `SimplifyVertexShaders` code path (so power users can still flip it), but the
   preset leaves it `false`. Add a code comment that this path is
   visually-breaking and intentionally not preset-default.
6. Sampler LOD / max-mip (`CreateSamplerState`) — unchanged, already correct.

### 4.4 OpenGL (`HookDll/src/opengl_hook.cpp`)

Upgrade `glShaderSource` from capture-only to stage-aware rewrite:

- After concatenating the source string, classify it (`ClassifyGlsl`):
  - **Fragment** if it contains `gl_FragColor` (legacy) OR `layout(location =` followed by `out` on the same line OR a non-void `out vec4` declaration (modern). These are fragment-only markers.
  - **Vertex** if it contains `gl_Position` and not `gl_FragColor`.
  - **Compute** if it contains `layout(local_size`.
  - **Unknown** otherwise → never substitute (pass through).
- When `SimplifyPixelShaders` and stage == Fragment → substitute a flat fragment
  source. Extract the original's `#version ...` line (if present) and prepend it
  to the flat body; if no `#version` line is found, prepend `#version 330 core`.
  Flat body:
  `out vec4 fragColor; void main(){ fragColor = vec4(0.35, 0.35, 0.35, 1.0); }`
- When `SkipComputeShaders` and stage == Compute → substitute
  `layout(local_size_x = 1, local_size_y = 1, local_size_z = 1) in;\nvoid main(){}`
  (with the original `#version` line prepended, or `#version 430 core`).
- **Vertex source is never substituted** (same principle as D3D11: keep
  silhouette).
- Still call `ShaderCache::SaveOriginal` with the *original* combined source
  before substituting (capture stays lossless).
- In `HookWglSwapBuffers`: when `MsaaOff`, call `glDisable(GL_MULTISAMPLE)`. (GL
  enum constant 0x809D — define locally to avoid pulling GL headers.)

Unit-testable helper: `static ShaderStage ClassifyGlsl(std::string_view src)`
— pure function, easy to test with sample strings. (No test harness exists in
the project; this is the one function worth a tiny standalone test if a harness
is added later — noted as optional.)

### 4.5 Vulkan (`HookDll/src/vulkan_hook.cpp`) — two phases

**Phase 1 (no SPIR-V needed — primary deliverable):**

Extend `vkGetDeviceProcAddr` interception to also return trampolines for
`vkCreateGraphicsPipelines` and `vkCreateComputePipelines`.

In `HookVkCreateGraphicsPipelines`:
- The pipeline-create struct stores `pStages` as a pointer to a
  `const VkPipelineShaderStageCreateInfo` array of length `stageCount`. We must
  not mutate the game's array (it may be `const`/read-only), so allocate a
  per-call mutable copy: a `std::vector<VkPipelineShaderStageCreateInfo>`
  containing *copies of the stage structs we keep* (we never mutate the contents
  of an individual stage struct, we only drop entries), and repoint
  `pStages`/`stageCount` at our vector. Walk the stages; drop any stage whose
  `stage` field is `VK_SHADER_STAGE_TESSELLATION_CONTROL` (0x10),
  `TESSELLATION_EVALUATION` (0x20), or `GEOMETRY` (0x04) when the matching flag
  is set (`DisableTessellation` / `DisableGeometryShader`). Keep vertex +
  fragment stages.
- When `MsaaOff` and `pMultisampleState` is non-null, allocate a per-call copy
  of the `VkPipelineMultisampleStateCreateInfo` struct, set
  `rasterizationSamples = 1` (`VK_SAMPLE_COUNT_1_BIT`), repoint
  `pMultisampleState` at it.
- Forward to the original with the patched struct(s). The vectors are local
  stack temporaries; valid for the duration of the forwarded call.
- **Validation:** before building the copies, null-check `pCreateInfo`,
  `pCreateInfos[i].pStages`, and (if mutating) `pMultisampleState`. If anything
  looks off or `stageCount` is 0, forward the *original* unchanged and log.

In `HookVkCreateComputePipelines`:
- When `SkipComputeShaders`, return `VK_ERROR_FEATURE_NOT_PRESENT` for each
  pipeline (engines treat compute as optional) **only if** we judge it safe —
  i.e. only when `pCreateInfos[i].basePipelineHandle == 0` (not a derivative we
  must produce). Otherwise forward unchanged. Conservative: if unsure, forward.

**Phase 2 (SPIR-V bytecode substitution — highest risk, time-boxed):**

Replace the fragment `VkShaderModule` with a flat SPIR-V blob and compute with a
no-op SPIR-V blob. Two static byte arrays embedded in the .cpp.

- **Risk:** the blob must be a valid SPIR-V module matching the original's
  entry-point name and execution model, or `vkCreateShaderModule` /
  `vkCreateGraphicsPipelines` validation layers reject it. We have no SPIR-V
  toolchain on the build machine.
- **Mitigation:** hand-author minimal SPIR-V (a fragment shader returning
  `vec4(0.35)` and an empty `main` compute) using the known binary layout, or
  precompile with `glslangValidator` if available at build time. Validate by
  loading with validation layers enabled on one test game.
- **Fallback if Phase 2 cannot validate:** ship Phase 1 only (tess/GS off +
  MSAA off, no flat). Phase 2 is explicitly **optional** in the implementation
  plan — it does not block Phase 1 value.

### 4.6 Shader cache & IPC

No structural changes. New tweak fields ride the existing JSON IPC. The flat
GLSL / flat SPIR-V blobs are static constants in their respective hook files,
not cached to disk.

## 5. Error handling

- Every new hook: on dummy-device/vtable fetch failure → log via `HOOK_LOG` /
  `IPC_LOG`, return `false` from `Install`, do not patch. Game proceeds without
  that hook.
- `E_FAIL` (D3D11 GS) / `VK_ERROR_FEATURE_NOT_PRESENT` (VK compute) return:
  engines have fallback paths; if an engine *requires* the stage and crashes,
  the user turns off that flag. Flags are deliberately separate (not one "kill
  all") so a problematic one can be disabled independently.
- VK pipeline patching: any malformed input → forward original, log once.
- Small-draw cull: if the immediate context vtable hook fails to install, set
  an internal flag so the cull path is a no-op even if `SkipSmallDraws` is later
  turned on.

## 6. Testing / verification

- **Build:** `cd HookDll && cmake --build build --config Release` (expect zero
  new warnings under MSVC `/W4`) and `cd Launcher && dotnet build`. Both must
  succeed.
- **Manual D3D11:** pick a known D3D11 title, apply `MaxFPS Flat` preset,
  launch via the launcher. Expect: flat grey scene, silhouette correct (no
  warped geometry), FPS up vs. baseline, no crash, HUD visible (because
  `SkipSmallDraws` is off). Then toggle `SkipSmallDraws` and confirm some
  draws vanish (HUD may degrade — expected).
- **Manual OpenGL:** if a GL title is available, confirm fragment flattening
  fires (IPC log shows "fragment → flat") and `glDisable(GL_MULTISAMPLE)` runs.
- **Manual Vulkan:** on a Vulkan title, confirm Phase 1 (IPC log shows stage
  strip / MSAA=1). Phase 2 attempted only if a SPIR-V blob validates.
- **IPC log** is the primary evidence channel; capture relevant lines into the
  PR/commit message.
- **No automated hook test suite** exists in this project and is out of scope
  to add; verification is build + manual + IPC-log-driven.

## 7. Honest limits (to be written into the README)

- Mesh / triangle **decimation** is not possible at the API-hook layer (meshes
  are already on the GPU; CB layouts are engine-specific). Poly reduction comes
  only from: disabling tessellation (removes amplification), disabling geometry
  shaders (removes amplification), and (experimental, D3D11) culling small
  draws. On scenes that lean on tessellation/GS this is a large win; on scenes
  that don't, it is modest.
- This tool competes with engine-side low-quality settings only where the engine
  exposes no such setting or the user wants a floor lower than the engine
  offers.

## 8. Out of scope for this spec (future)

- D3D12 hooks.
- VK small-draw cull (`vkCmdDraw*` interception).
- GL small-draw cull.
- Mesh decimation / LOD forcing at buffer-creation time (requires engine
  cooperation).
- Automatic post-process pass detection beyond "flatten all fragment shaders"
  (e.g. fullscreen-triangle heuristic) — the flat tier intentionally flattens
  everything, so detection is unnecessary for this tier.

"""Generate the Game Lag Reducer app icon as a multi-resolution .ico.

Design: a dark rounded-square tile (#1E1E2E) with a bold purple (#7B5CFF)
lightning bolt — "lag reduction / speed". Clean enough to read at 16px.
Drawn at 512 then exported at 16/32/48/64/128/256 (Pillow ICO multi-res).
No external assets, no AI, no network.
"""
import math
from PIL import Image, ImageDraw, ImageFilter

# Theme palette (matches App.xaml)
BG = (30, 30, 46, 255)        # #1E1E2E
BG2 = (42, 42, 60, 255)      # #2A2A3C (subtle gradient end)
BOLT = (123, 92, 255, 255)    # #7B5CFF accent
BOLT_HI = (155, 124, 255, 255)  # #9B7CFF accent hover (highlight edge)
EDGE = (54, 54, 80, 255)      # #363650 border

S = 512  # working size
R = 96   # corner radius for the rounded square


def rounded_tile(img, radius, fill, border=None):
    d = ImageDraw.Draw(img)
    d.rounded_rectangle([0, 0, S - 1, S - 1], radius=radius, fill=fill)
    if border is not None:
        d.rounded_rectangle([0, 0, S - 1, S - 1], radius=radius, outline=border, width=6)


def bolt_polygon(cx, cy, w, h):
    """A classic lightning bolt centered at (cx,cy), width w, height h.
    Points go clockwise from the top tip."""
    # proportions tuned to look like a fat, readable bolt
    top = cy - h / 2
    bot = cy + h / 2
    left = cx - w / 2
    right = cx + w / 2
    midL = cx - w * 0.10
    midR = cx + w * 0.42
    # zig-zag bolt outline (clockwise)
    return [
        (cx, top),            # top tip
        (right, cy - h * 0.02),
        (midR, cy + h * 0.02),  # right notch in
        (right, cy + h * 0.18),
        (cx + w * 0.04, bot),   # bottom tip
        (left, cy + h * 0.06),
        (midL, cy + h * 0.02),  # left notch out
        (left, cy - h * 0.10),
        (cx - w * 0.04, top + h * 0.06),
    ]


def render(size):
    img = Image.new("RGBA", (size, size), (0, 0, 0, 0))
    # sub-pixel accurate draw via supersample: draw at S then downscale
    base = Image.new("RGBA", (S, S), (0, 0, 0, 0))
    rounded_tile(base, R, BG, EDGE)

    d = ImageDraw.Draw(base)
    # subtle vertical gradient: darken bottom slightly for depth
    grad = Image.new("RGBA", (S, S), (0, 0, 0, 0))
    gd = ImageDraw.Draw(grad)
    for y in range(S):
        t = y / S
        a = int(40 * t)
        gd.line([(0, y), (S, y)], fill=(0, 0, 0, a))
    base = Image.alpha_composite(base, grad)
    d = ImageDraw.Draw(base)

    # lightning bolt — bold, centered, slightly above center
    pts = bolt_polygon(cx=S / 2, cy=S / 2 - 8, w=S * 0.46, h=S * 0.66)
    d.polygon(pts, fill=BOLT)
    # highlight edge on the left for a touch of dimension
    d.line(pts[:4], fill=BOLT_HI, width=10, joint="curve")
    d.line([pts[7], pts[8], pts[0]], fill=BOLT_HI, width=10, joint="curve")

    # soft glow under the bolt for pop
    glow = Image.new("RGBA", (S, S), (0, 0, 0, 0))
    gd = ImageDraw.Draw(glow)
    gd.polygon(pts, fill=(123, 92, 255, 90))
    glow = glow.filter(ImageFilter.GaussianBlur(28))
    base = Image.alpha_composite(glow, base)

    # downscale to target size with LANCZOS for crisp edges
    return base.resize((size, size), Image.LANCZOS)


def main():
    sizes = [16, 32, 48, 64, 128, 256]
    images = [render(s) for s in sizes]
    # Pillow multi-res ICO: pass the largest plus a sizes list, or pass a list.
    images[-1].save(
        r"C:/Users/Admin/Downloads/New folder (5)/Launcher/app.ico",
        format="ICO",
        sizes=[(s, s) for s in sizes],
        append_images=images[:-1],
    )
    # also keep a 512 PNG for reference / GitHub og
    render(512).save(r"C:/Users/Admin/Downloads/New folder (5)/Launcher/app.png")
    print("wrote app.ico (16/32/48/64/128/256) + app.png (512)")


if __name__ == "__main__":
    main()
