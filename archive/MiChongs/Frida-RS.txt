## 项目简介

Frida Server for KernelSU 将官方 `frida-server` 封装为可直接安装的 KernelSU 模块，并由一个小型 Rust Supervisor 负责生命周期管理。

模块默认只监听 `127.0.0.1:27042`，适合通过 USB/ADB 使用。非回环监听默认要求 Frida Token；仅 `0.0.0.0` 可在明确确认风险后启用无认证模式，避免误操作直接暴露调试端点。

> 本项目面向安全研究、应用调试和已获授权的测试场景。请只分析你拥有或明确获准测试的设备与应用。

## 功能

| 能力 | 说明 |
| --- | --- |
| KernelSU 原生模块 | 可直接在 KernelSU Manager 中从本地 ZIP 安装 |
| 四 ABI 支持 | `arm64-v8a`、`armeabi-v7a`、`x86_64`、`x86` |
| Rust 守护进程 | 启动、停止、重启、PID 管理、异常恢复与安全退出 |
| 官方 Frida Server | 构建时从 Frida GitHub Releases 下载并校验 SHA-256 |
| Material 3 WebUI | 响应式布局、系统动态配色、深色模式和安全区域适配 |
| 安全配置 | 远程监听默认强制 Token；无认证公开监听需要显式开关和 WebUI 风险确认 |
| 日志管理 | Supervisor 与 Frida Server 独立日志、限量读取及一键清空 |
| 可验证构建 | Rust/Node 版本锁定，依赖锁文件和固定 Release 摘要校验 |
| 多架构注入 | 官方 64 位 Server 内嵌 32/64 位 Helper，按目标进程自动选择 |

## 工作原理

```mermaid
flowchart LR
    K[KernelSU boot service] --> R[frida-ksud]
    W[KernelSU WebUI] -->|spawn argv| R
    R --> C[Validated config.conf]
    R --> F[Official frida-server]
    F --> H32[frida-helper-32]
    F --> H64[frida-helper-64]
    A[Frida client] -->|ADB / TCP 27042| F
    R --> L[PID files and logs]
```

安装阶段会根据设备 ABI 从模块 Payload 中选择对应的 Rust Supervisor 和 Frida Server，复制到模块私有 `bin/` 目录，然后删除其余架构 Payload，减少安装后的磁盘占用。

运行阶段只保留一个与设备主 ABI 匹配的 `frida-server`。对于原生支持 32 位进程的 64 位 Android，Frida 会根据目标 PID 自动选择 32 位或 64 位 Helper，无需切换端口或重启服务。

## 兼容性

### 系统要求

- Android 7.0 / API 24 或更高版本
- KernelSU 及 KernelSU Manager
- 已授予模块运行所需的 Root 环境
- PC 端 Frida 版本应与模块内 Frida Server 版本一致

### 支持的架构

| KernelSU 架构 | Android ABI | Rust Target | Frida Release |
| --- | --- | --- | --- |
| `arm64` | `arm64-v8a` | `aarch64-linux-android` | `android-arm64` |
| `arm` | `armeabi-v7a` | `armv7-linux-androideabi` | `android-arm` |
| `x64` | `x86_64` | `x86_64-linux-android` | `android-x86_64` |
| `x86` | `x86` | `i686-linux-android` | `android-x86` |

### 纯 64 位设备与 32 位翻译层

Xiaomi 14 Ultra 等纯 AArch64 设备可能通过 HyperOS Native Bridge/Tango 一类翻译层运行 32 位 APK。这不等同于原生 ARM32 Linux 进程：

- 模块仍应安装并运行 ARM64 Frida Server；
- Frida 已内嵌 `frida-helper-32` 与 `frida-helper-64`；
- 如果 ARM32 代码由翻译器作为 Guest Code 执行，普通 32 位 Server 切换无法保证 Hook 翻译器内部代码；
- 此类问题需要根据进程 Maps、Native Bridge 属性和具体 Frida 错误诊断。

参见 [Frida 的架构 Helper 选择实现](https://github.com/frida/frida-core/blob/main/src/linux/frida-helper-process.vala) 和 [AOSP Native Bridge](https://source.android.com/docs/core/architecture/vndk/native-bridge)。

## 安装

1. 从 [GitHub Releases](https://github.com/MiChongs/Frida-RS/releases/latest) 下载最新 ZIP：

   ```text
   frida-kernelsu-<module-version>-frida-<frida-version>.zip
   ```

2. 打开 **KernelSU Manager → 模块 → 从本地安装**。
3. 选择下载的 ZIP，等待安装完成。
4. 重启设备。
5. 在模块详情页打开 WebUI，确认服务状态为“运行中”。

> KernelSU 模块 ZIP 不应在第三方 Recovery 中刷入。

## 使用

### USB / ADB

默认配置只监听回环地址。连接设备后可以直接使用 Frida USB 设备模式：

> `frida-ps`、`frida` 和 `frida-trace` 是 **PC 端 Frida Tools 命令**，不要在 `adb shell` 或设备终端中运行。此模块在设备端提供 `frida-server`，两端通过 USB/ADB 或 TCP 通信。

```sh
frida-ps -U
frida -U -f com.example.app
frida-trace -U -i open -N com.example.app
```

如需显式使用 TCP 转发：

```sh
adb forward tcp:27042 tcp:27042
frida-ps -H 127.0.0.1:27042
```

检查两端版本：

```sh
frida --version
adb shell su -c '/data/adb/modules/frida-kernelsu/bin/frida-server --version'
```

版本不一致时，应升级 PC 端 Frida 或安装对应版本的模块。

### KernelSU Action

点击 KernelSU Manager 模块页面中的 Action 按钮会执行：

```text
frida-ksud status
```

用于快速查看 Supervisor 与 Frida Server PID。

### 设备端命令行控制

模块会把 `frida-rs`、`frida-ksud` 和 `frida-server` 注册到 KernelSU Root Shell 的 `PATH`。重启后可从 PC 直接执行：

```sh
adb shell su -c 'frida-rs status'
adb shell su -c 'frida-rs start'
adb shell su -c 'frida-rs stop'
adb shell su -c 'frida-rs restart'
adb shell su -c 'frida-rs inspect'
adb shell su -c 'frida-rs logs --source server --lines 200'
adb shell su -c 'frida-rs logs --source supervisor --lines 200'
adb shell su -c 'frida-rs clear-logs'
adb shell su -c 'frida-server --version'
```

也可以先执行 `adb shell`、再执行 `su`，随后直接使用同一组命令。`frida-ksud` 是 `frida-rs` 的等价技术入口；不要手动启动 `frida-server`，其生命周期由 Supervisor 管理。

`daemon` 是 Supervisor 内部前台模式，通常由 `start` 自动拉起，无需手动执行。

## WebUI

WebUI 使用 Lit 与 Material Web 构建，并通过 KernelSU 官方 JavaScript Bridge 直接调用模块内 Rust 二进制。

| 页面 | 功能 |
| --- | --- |
| 概览 | 服务状态、Frida 版本、PID、监听端点、访问安全和启动控制 |
| 配置 | 开机启动、异常恢复、监听地址、Token、TLS 证书和延迟设置 |
| 日志 | Server/Supervisor 日志切换、自动刷新、复制和清空 |
| 诊断 | Android、内核、目标架构、模块版本、进程及日志占用 |

安全设计：

- WebUI 使用参数数组调用后端，不拼接 Shell 命令；
- KernelSU 返回的模块目录必须通过白名单格式验证；
- Rust 后端重新验证全部配置，不信任浏览器输入；
- 状态接口只返回 `tokenConfigured`，不会把现有 Token 传回 WebUI；
- 外部 JSON 在 TypeScript 边界进行运行时字段校验。

## 配置

配置文件位于：

```text
/data/adb/modules/frida-kernelsu/config.conf
```

默认配置：

```ini
enabled=true
listen=127.0.0.1:27042
token=
allow_unauthenticated_remote=false
certificate=
restart=true
restart_delay_seconds=2
startup_delay_seconds=0
```

| 字段 | 默认值 | 说明 |
| --- | --- | --- |
| `enabled` | `true` | 是否允许 Supervisor 启动 Frida Server |
| `listen` | `127.0.0.1:27042` | Frida Server 监听地址与端口 |
| `token` | 空 | Frida 远程认证 Token；所有远程监听的推荐保护方式 |
| `allow_unauthenticated_remote` | `false` | 仅允许 `0.0.0.0:PORT` 在空 Token 下启动；属于高风险显式开关 |
| `certificate` | 空 | 可选 PEM Bundle；相对路径从模块目录解析 |
| `restart` | `true` | Frida Server 异常退出后是否自动重启 |
| `restart_delay_seconds` | `2` | 异常退出后的重试等待时间，范围 `0–3600` |
| `startup_delay_seconds` | `0` | 开机后的启动等待时间，范围 `0–300` |

配置可通过 WebUI 安全更新。手动编辑后执行：

```sh
adb shell su -c '/data/adb/modules/frida-kernelsu/bin/frida-ksud restart --module-dir /data/adb/modules/frida-kernelsu'
```

### 远程监听

推荐使用 Token 保护所有远程端点：

```ini
listen=0.0.0.0:27042
token=请替换为高强度随机令牌
allow_unauthenticated_remote=false
```

如确需在可信隔离网络中开放无认证访问，可使用：

```ini
listen=0.0.0.0:27042
token=
allow_unauthenticated_remote=true
```

WebUI 在首次保存该组合前会显示风险确认对话框；取消后不会写入配置。手动编辑配置时也必须显式设置开关。无认证模式只适用于 IPv4 通配地址 `0.0.0.0`，其他非回环地址仍需要 Token。

无认证 Frida 端点允许任何可达客户端控制设备进程、读取内存并执行注入。只能在可信且有防火墙隔离的网络中短时使用，绝不要暴露到公网。

dist/frida-kernelsu-<module-version>-frida-<frida-version>.zip

## 常见问题

### 在设备中执行 `frida-ps` 提示 `command not found`

这是预期行为：`frida-ps` 属于 PC 端的 [`frida-tools`](https://frida.re/docs/installation/)，不是 Android `frida-server` 命令。在 PC 安装匹配版本后运行：

```sh
python -m pip install --upgrade frida-tools
frida --version
frida-ps -U
```

设备端使用 `su -c 'frida-rs status'` 检查服务，或使用 KernelSU WebUI。

### `frida-ps -U` 找不到设备

确认 ADB 能看到设备并重新执行：

```sh
adb devices -l
frida-ls-devices
```

### 能看到设备但无法连接 Server

确认模块状态、端口和版本：

```sh
adb shell su -c '/data/adb/modules/frida-kernelsu/bin/frida-ksud status --module-dir /data/adb/modules/frida-kernelsu'
frida --version
```

### 服务反复重启

读取 Supervisor 与 Server 日志：

```sh
adb shell su -c '/data/adb/modules/frida-kernelsu/bin/frida-ksud logs --source supervisor --lines 200 --module-dir /data/adb/modules/frida-kernelsu'
adb shell su -c '/data/adb/modules/frida-kernelsu/bin/frida-ksud logs --source server --lines 200 --module-dir /data/adb/modules/frida-kernelsu'
```

### 非回环监听配置无法保存

普通远程地址仍必须配置非空 Token。对于 `0.0.0.0:PORT`，可在 WebUI 保持 Token 为空并保存，阅读风险提示后点击“我了解风险，继续”；或手动同时设置：

```ini
listen=0.0.0.0:27042
token=
allow_unauthenticated_remote=true
```

### 纯 64 位 Xiaomi 可以运行 32 位 App，但无法 Hook

先检查 Native Bridge：

```sh
adb shell getprop ro.product.cpu.abilist32
adb shell getprop ro.dalvik.vm.native.bridge
adb shell getprop ro.enable.native.bridge.exec
```

如果目标通过 Tango/Native Bridge 翻译运行，问题不是简单缺少 ARM32 Server。请同时提供目标包名、HyperOS 版本、Frida 错误、`/proc/<PID>/maps` 和 Server 日志。

`module/commands.sh`:

```sh
#!/system/bin/sh

FRIDA_COMMAND_DIR=/data/adb/ksu/bin
FRIDA_MODULE_ID=frida-kernelsu

frida_link_is_ours() {
    target=$(readlink "$1" 2>/dev/null) || return 1
    case "$target" in
        /data/adb/modules/$FRIDA_MODULE_ID/* | /data/adb/modules_update/$FRIDA_MODULE_ID/*) return 0 ;;
        *) return 1 ;;
    esac
}

frida_register_command() {
    name=$1
    target=$2
    link="$FRIDA_COMMAND_DIR/$name"

    if [ -e "$link" ] || [ -L "$link" ]; then
        if ! frida_link_is_ours "$link"; then
            echo "Frida-RS: keeping existing command: $link" >&2
            return 0
        fi
        rm -f "$link"
    fi

    ln -s "$target" "$link"
}

register_frida_commands() {
    module_dir=$1
    mkdir -p "$FRIDA_COMMAND_DIR"
    frida_register_command frida-rs "$module_dir/bin/frida-ksud"
    frida_register_command frida-ksud "$module_dir/bin/frida-ksud"
    frida_register_command frida-server "$module_dir/bin/frida-server"
}

unregister_frida_commands() {
    for name in frida-rs frida-ksud frida-server; do
        link="$FRIDA_COMMAND_DIR/$name"
        if [ -L "$link" ] && frida_link_is_ours "$link"; then

def cargo_metadata() -> tuple[str, int]:
    with (ROOT / "Cargo.toml").open("rb") as file:
        version = tomllib.load(file)["package"]["version"]
    numbers = version.split("-", 1)[0].split(".")
    if len(numbers) != 3 or any(not part.isdigit() for part in numbers):
        raise RuntimeError(f"package version must be semantic, got {version!r}")
    major, minor, patch = map(int, numbers)
    if minor > 999 or patch > 999:
        raise RuntimeError("minor and patch versions must not exceed 999")
    return version, major * 1_000_000 + minor * 1_000 + patch


def run_cargo_ndk() -> None:
    command = ["cargo", "ndk"]
    for abi in ABIS:
        command.extend(("-t", abi.ndk))
    command.extend(("-P", str(ANDROID_PLATFORM), "build", "--release", "--locked"))
    print("+", " ".join(command), flush=True)
    subprocess.run(command, cwd=ROOT, check=True)


def build_webui() -> Path:

def release_digests(version: str) -> dict[str, str]:
    if version == DEFAULT_FRIDA_VERSION:
        return {abi.frida: abi.sha256 for abi in ABIS}

    tag = urllib.parse.quote(version, safe="")
    request = urllib.request.Request(
        f"https://api.github.com/repos/frida/frida/releases/tags/{tag}",
        headers={"Accept": "application/vnd.github+json", "User-Agent": "Frida-RS-builder"},
    )
    try:
        with urllib.request.urlopen(request, timeout=30) as response:
            release = json.load(response)
    except (urllib.error.URLError, json.JSONDecodeError) as error:
        raise RuntimeError(f"cannot load Frida {version} release metadata: {error}") from error

    digests: dict[str, str] = {}
    for abi in ABIS:
        name = frida_asset_name(version, abi)
        asset = next((item for item in release.get("assets", []) if item.get("name") == name), None)
        if asset is None:
            raise RuntimeError(f"Frida release does not contain {name}")
        digest = asset.get("digest", "")
        if not digest.startswith("sha256:"):
            raise RuntimeError(f"GitHub did not publish a SHA-256 digest for {name}")
        digests[abi.frida] = digest.removeprefix("sha256:")
    return digests


def frida_asset_name(version: str, abi: Abi) -> str:
    return f"frida-server-{version}-android-{abi.frida}.xz"


def download_frida(version: str, cache_dir: Path) -> dict[str, Path]:
    cache_dir.mkdir(parents=True, exist_ok=True)
    digests = release_digests(version)
    servers: dict[str, Path] = {}

    for abi in ABIS:
        name = frida_asset_name(version, abi)
        archive = cache_dir / name
        expected = digests[abi.frida]
        if not archive.is_file() or sha256(archive) != expected:
            url = f"https://github.com/frida/frida/releases/download/{version}/{name}"
            temporary = archive.with_suffix(".download")
            print(f"Downloading {url}", flush=True)
            request = urllib.request.Request(url, headers={"User-Agent": "Frida-RS-builder"})
            try:
                with urllib.request.urlopen(request, timeout=120) as response, temporary.open("wb") as output:
                    shutil.copyfileobj(response, output)
            except urllib.error.URLError as error:
                temporary.unlink(missing_ok=True)
                raise RuntimeError(f"cannot download {name}: {error}") from error
            actual = sha256(temporary)
            if actual != expected:
                temporary.unlink(missing_ok=True)
                raise RuntimeError(
                    f"SHA-256 mismatch for {name}: expected {expected}, got {actual}"
                )
            temporary.replace(archive)

        output = cache_dir / name.removesuffix(".xz")
        if not output.is_file() or output.stat().st_mtime_ns < archive.stat().st_mtime_ns:
            temporary = output.with_suffix(".decompressing")
            with lzma.open(archive, "rb") as source, temporary.open("wb") as target:
                shutil.copyfileobj(source, target)
            temporary.replace(output)
        validate_elf(output, f"Frida Server for {abi.ksu}")
        servers[abi.ksu] = output
    return servers


def sha256(path: Path) -> str:
    digest = hashlib.sha256()
    with path.open("rb") as file:
        for chunk in iter(lambda: file.read(1024 * 1024), b""):
            digest.update(chunk)
    return digest.hexdigest()


def validate_elf(path: Path, description: str) -> None:
    try:
        with path.open("rb") as file:
            magic = file.read(4)
    except OSError as error:
        raise RuntimeError(f"cannot read {description} at {path}: {error}") from error
    if magic != b"\x7fELF":
        raise RuntimeError(f"{description} is not an ELF executable: {path}")


def stage_module(
    module_version: str,
    version_code: int,
    frida_version: str,
    launchers: dict[str, Path],
    servers: dict[str, Path],
    webui: Path,
    output_dir: Path,
) -> Path:
    stage = output_dir / "staging"
    if stage.exists():
        shutil.rmtree(stage)
    shutil.copytree(ROOT / "module", stage)
    shutil.copytree(webui, stage / "webroot")

    module_prop = stage / "module.prop"
    metadata = module_prop.read_text(encoding="utf-8")
    metadata = metadata.replace("@MODULE_VERSION@", module_version)
    metadata = metadata.replace("@VERSION_CODE@", str(version_code))
    metadata = metadata.replace("@FRIDA_VERSION@", frida_version)
    module_prop.write_text(metadata, encoding="utf-8", newline="\n")
    (stage / "frida.version").write_text(frida_version + "\n", encoding="utf-8", newline="\n")

    for abi in ABIS:
        payload = stage / "payload" / abi.ksu
        payload.mkdir(parents=True)
        shutil.copyfile(launchers[abi.ksu], payload / "frida-ksud")
        shutil.copyfile(servers[abi.ksu], payload / "frida-server")
    return stage


def write_zip(stage: Path, destination: Path) -> None:
    destination.parent.mkdir(parents=True, exist_ok=True)
    temporary = destination.with_suffix(".zip.tmp")
    timestamp = (2020, 1, 1, 0, 0, 0)

    with zipfile.ZipFile(
        temporary, "w", compression=zipfile.ZIP_DEFLATED, compresslevel=9
    ) as archive:
        for path in sorted(item for item in stage.rglob("*") if item.is_file()):
            relative = path.relative_to(stage).as_posix()
            executable = relative.endswith(".sh") or relative.startswith("payload/")
            mode = stat.S_IFREG | (0o755 if executable else 0o644)
            info = zipfile.ZipInfo(relative, timestamp)
            info.create_system = 3
            info.compress_type = zipfile.ZIP_DEFLATED
            info.external_attr = mode << 16
            archive.writestr(info, path.read_bytes())
    temporary.replace(destination)


def verify_zip(path: Path) -> None:
    required = {
        "module.prop",
        "customize.sh",
        "commands.sh",
        "service.sh",
        "action.sh",
        "uninstall.sh",
        "skip_mount",
        "config.conf.example",
        "frida.version",
        "webroot/index.html",
        "webroot/icon.svg",
    }
    required.update(
        f"payload/{abi.ksu}/{binary}"
        for abi in ABIS
        for binary in ("frida-ksud", "frida-server")
    )
    with zipfile.ZipFile(path) as archive:
        names = set(archive.namelist())
        missing = sorted(required - names)
        if missing:
            raise RuntimeError(f"module archive is missing: {', '.join(missing)}")
        if any("\\" in name or name.startswith("/") for name in names):
            raise RuntimeError("module archive contains a non-portable path")
        if b"@" in archive.read("module.prop"):
            raise RuntimeError("module.prop still contains a build placeholder")
        for name in names:
            data = archive.read(name)
            if name.endswith((".sh", ".prop", ".conf")) and b"\r\n" in data:
                raise RuntimeError(f"module text file uses CRLF: {name}")
        if not any(name.startswith("webroot/assets/") and name.endswith(".js") for name in names):
            raise RuntimeError("module archive does not contain the WebUI JavaScript bundle")
        if not any(name.startswith("webroot/assets/") and name.endswith(".css") for name in names):
            raise RuntimeError("module archive does not contain the WebUI stylesheet")


def main() -> int:
    args = parse_args()
    module_version, version_code = cargo_metadata()
    launchers = launcher_paths(args.launcher_dir)
    servers = download_frida(
        args.frida_version, ROOT / ".cache" / "frida" / args.frida_version
    )
    webui = build_webui()
    stage = stage_module(
        module_version,
        version_code,
        args.frida_version,
        launchers,
        servers,
        webui,
        args.output_dir,
    )
    destination = args.output_dir / (
        f"frida-kernelsu-{module_version}-frida-{args.frida_version}.zip"
    )
    write_zip(stage, destination)
    verify_zip(destination)
    print(destination)
    return 0


if __name__ == "__main__":

use std::env;
use std::ffi::OsString;
use std::fmt;
use std::path::PathBuf;

#[derive(Debug, Clone, Copy, Eq, PartialEq)]
pub enum CommandKind {
    Start,
    Daemon,
    Stop,
    Restart,
    Status,
    Inspect,
    Configure,
    Logs,
    ClearLogs,
    Help,
    Version,
}

#[derive(Debug, Eq, PartialEq)]
pub struct Cli {
    pub command: CommandKind,
    pub module_dir: Option<PathBuf>,
    pub command_args: Vec<OsString>,
}

#[derive(Debug, Eq, PartialEq)]
pub struct ParseError(pub String);

impl fmt::Display for ParseError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        f.write_str(&self.0)
    }
}

impl std::error::Error for ParseError {}

impl Cli {
    pub fn parse() -> Result<Self, ParseError> {
        Self::parse_from(env::args_os().skip(1))
    }

    fn parse_from<I>(args: I) -> Result<Self, ParseError>
    where
        I: IntoIterator<Item = OsString>,
    {
        let mut args = args.into_iter();
        let Some(command) = args.next() else {
            return Err(ParseError("missing command; use --help for usage".into()));
        };

        let command = match command.to_str() {
            Some("start") => CommandKind::Start,
            Some("daemon") => CommandKind::Daemon,
            Some("stop") => CommandKind::Stop,
            Some("restart") => CommandKind::Restart,
            Some("status") => CommandKind::Status,
            Some("inspect") => CommandKind::Inspect,
            Some("configure") => CommandKind::Configure,
            Some("logs") => CommandKind::Logs,
            Some("clear-logs") => CommandKind::ClearLogs,
            Some("help" | "--help" | "-h") => CommandKind::Help,
            Some("--version" | "-V") => CommandKind::Version,
            Some(other) => return Err(ParseError(format!("unknown command: {other}"))),
            None => return Err(ParseError("command is not valid UTF-8".into())),
        };

        let mut module_dir = None;
        let mut command_args = Vec::new();
        while let Some(argument) = args.next() {
            match argument.to_str() {
                Some("--module-dir") => {
                    if module_dir.is_some() {
                        return Err(ParseError("--module-dir may only be specified once".into()));
                    }
                    let value = args
                        .next()
                        .ok_or_else(|| ParseError("--module-dir requires a path".into()))?;
                    module_dir = Some(PathBuf::from(value));
                }
                Some(_) if matches!(command, CommandKind::Configure | CommandKind::Logs) => {
                    command_args.push(argument);
                }
                Some(other) => return Err(ParseError(format!("unknown argument: {other}"))),
                None => return Err(ParseError("argument is not valid UTF-8".into())),
            }
        }

        Ok(Self {
            command,
            module_dir,
            command_args,
        })
    }

    pub fn resolve_module_dir(self) -> Result<(CommandKind, PathBuf, Vec<OsString>), ParseError> {
        if let Some(path) = self.module_dir {
            return Ok((self.command, path, self.command_args));
        }

        let executable = env::current_exe()
            .map_err(|error| ParseError(format!("cannot locate current executable: {error}")))?;
        let module_dir = executable
            .parent()
            .and_then(|bin_dir| bin_dir.parent())
            .ok_or_else(|| {
                ParseError("cannot infer module directory from executable path".into())
            })?;
        Ok((self.command, module_dir.to_path_buf(), self.command_args))
    }
}

pub const CONFIG_FILE: &str = "config.conf";

#[derive(Clone, Debug, Eq, PartialEq)]
pub struct Config {

    pub listen: String,
    pub token: Option<String>,
    pub allow_unauthenticated_remote: bool,
    pub certificate: Option<PathBuf>,
    pub restart: bool,
    pub restart_delay: Duration,
    pub startup_delay: Duration,
}

impl Default for Config {

            enabled: true,
            listen: "127.0.0.1:27042".into(),
            token: None,
            allow_unauthenticated_remote: false,
            certificate: None,
            restart: true,
            restart_delay: Duration::from_secs(2),
            startup_delay: Duration::ZERO,
        }
    }

#[derive(Debug)]
pub enum ConfigError {
    Read { path: PathBuf, source: io::Error },
    Invalid { line: usize, message: String },
}

impl fmt::Display for ConfigError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        match self {
            Self::Read { path, source } => {
                write!(f, "cannot read {}: {source}", path.display())
            }
            Self::Invalid { line, message } => {
                write!(f, "invalid configuration at line {line}: {message}")
            }
        }
    }
}

impl std::error::Error for ConfigError {
    fn source(&self) -> Option<&(dyn std::error::Error + 'static)> {
        match self {
            Self::Read { source, .. } => Some(source),
            Self::Invalid { .. } => None,
        }
    }
}

impl Config {
    pub fn load(module_dir: &Path) -> Result<Self, ConfigError> {
        let path = module_dir.join(CONFIG_FILE);
        match fs::read_to_string(&path) {
            Ok(contents) => Self::parse(&contents),
            Err(source) if source.kind() == io::ErrorKind::NotFound => Ok(Self::default()),
            Err(source) => Err(ConfigError::Read { path, source }),
        }
    }

    pub fn apply_hex_payload(&mut self, payload: &str) -> Result<(), ConfigError> {
        let document = decode_hex(payload)?;
        let mut fields = BTreeMap::new();
        for (index, line) in document.lines().enumerate() {
            let (key, value) = line.split_once('=').ok_or_else(|| ConfigError::Invalid {
                line: index + 1,
                message: "expected key=value in configuration payload".into(),
            })?;
            if !matches!(
                key,
                "enabled"
                    | "listen"
                    | "token_action"
                    | "token"
                    | "allow_unauthenticated_remote"
                    | "certificate"
                    | "restart"
                    | "restart_delay_seconds"
                    | "startup_delay_seconds"
            ) {
                return Err(ConfigError::Invalid {
                    line: index + 1,
                    message: format!("unknown payload key: {key}"),
                });
            }
            if fields.insert(key, value).is_some() {
                return Err(ConfigError::Invalid {
                    line: index + 1,
                    message: format!("duplicate payload key: {key}"),
                });
            }
        }

        let mut updated = self.clone();
        updated.enabled = parse_bool(required_field(&fields, "enabled")?, 0)?;
        updated.listen = required_field(&fields, "listen")?.to_owned();
        updated.restart = parse_bool(required_field(&fields, "restart")?, 0)?;
        updated.restart_delay =
            parse_duration(required_field(&fields, "restart_delay_seconds")?, 0, 3_600)?;
        updated.startup_delay =
            parse_duration(required_field(&fields, "startup_delay_seconds")?, 0, 300)?;

        let token = required_field(&fields, "token")?;
        updated.token = match required_field(&fields, "token_action")? {
            "keep" => updated.token,
            "clear" => None,
            "set" if !token.is_empty() => Some(token.to_owned()),
            "set" => {
                return Err(ConfigError::Invalid {
                    line: 0,
                    message: "token must not be empty when token_action=set".into(),
                });
            }
            value => {
                return Err(ConfigError::Invalid {
                    line: 0,
                    message: format!("unknown token action: {value}"),
                });
            }
        };
        updated.allow_unauthenticated_remote =
            parse_bool(required_field(&fields, "allow_unauthenticated_remote")?, 0)?;

        let certificate = required_field(&fields, "certificate")?;
        updated.certificate = (!certificate.is_empty()).then(|| PathBuf::from(certificate));
        updated.validate()?;
        *self = updated;
        Ok(())
    }

    pub fn save(&self, module_dir: &Path) -> io::Result<()> {
        let path = module_dir.join(CONFIG_FILE);
        let temporary = module_dir.join(format!(".{CONFIG_FILE}.{}.tmp", std::process::id()));
        let result = (|| {
            let mut file = fs::OpenOptions::new()
                .create_new(true)
                .write(true)
                .open(&temporary)?;
            #[cfg(unix)]
            {
                use std::os::unix::fs::PermissionsExt;
                file.set_permissions(fs::Permissions::from_mode(0o600))?;
            }
            file.write_all(self.document().as_bytes())?;
            file.sync_all()?;
            #[cfg(windows)]
            if path.exists() {
                fs::remove_file(&path)?;
            }
            fs::rename(&temporary, &path)
        })();
        if result.is_err() {
            let _ = fs::remove_file(temporary);
        }
        result
    }

    fn document(&self) -> String {
        format!(
            "# Managed by Frida KernelSU. Changes are validated before startup.\n\
             enabled={}\n\
             listen={}\n\
             token={}\n\
             allow_unauthenticated_remote={}\n\
             certificate={}\n\
             restart={}\n\
             restart_delay_seconds={}\n\
             startup_delay_seconds={}\n",
            self.enabled,
            self.listen,
            self.token.as_deref().unwrap_or_default(),
            self.allow_unauthenticated_remote,
            self.certificate
                .as_deref()
                .map(Path::to_string_lossy)
                .unwrap_or_default(),
            self.restart,
            self.restart_delay.as_secs(),
            self.startup_delay.as_secs(),
        )
    }

    fn parse(contents: &str) -> Result<Self, ConfigError> {
        let mut config = Self::default();

        for (index, raw_line) in contents.lines().enumerate() {
            let line_number = index + 1;
            let line = raw_line.trim();
            if line.is_empty() || line.starts_with('#') {
                continue;
            }

            let (key, value) = line.split_once('=').ok_or_else(|| ConfigError::Invalid {
                line: line_number,
                message: "expected key=value".into(),
            })?;
            let key = key.trim();
            let value = value.trim();

            match key {
                "enabled" => config.enabled = parse_bool(value, line_number)?,
                "listen" => config.listen = value.to_owned(),
                "token" => {
                    config.token = (!value.is_empty()).then(|| value.to_owned());
                }
                "allow_unauthenticated_remote" => {
                    config.allow_unauthenticated_remote = parse_bool(value, line_number)?;
                }
                "certificate" => {
                    config.certificate = (!value.is_empty()).then(|| PathBuf::from(value));
                }
                "restart" => config.restart = parse_bool(value, line_number)?,
                "restart_delay_seconds" => {
                    config.restart_delay = parse_duration(value, line_number, 3_600)?;
                }
                "startup_delay_seconds" => {
                    config.startup_delay = parse_duration(value, line_number, 300)?;
                }
                _ => {
                    return Err(ConfigError::Invalid {
                        line: line_number,
                        message: format!("unknown key: {key}"),
                    });
                }
            }
        }

        config.validate()?;
        Ok(config)
    }

    fn validate(&self) -> Result<(), ConfigError> {
        let invalid = |message: String| ConfigError::Invalid { line: 0, message };
        let host = parse_listen_host(&self.listen).map_err(invalid)?;

        if self.allow_unauthenticated_remote
            && (host != IpAddr::from([0, 0, 0, 0]) || self.token.is_some())
        {
            return Err(invalid(
                "allow_unauthenticated_remote requires listen=0.0.0.0:PORT and an empty token"
                    .into(),
            ));
        }

        if self.enabled
            && !host.is_loopback()
            && self.token.is_none()
            && !self.allow_unauthenticated_remote
        {
            return Err(invalid(
                "a non-loopback listen address requires a non-empty token or explicit allow_unauthenticated_remote=true for 0.0.0.0"
                    .into(),
            ));
        }

        if let Some(token) = &self.token {
            if token.len() > 1_024 || token.chars().any(char::is_control) {
                return Err(invalid(
                    "token must be at most 1024 bytes and contain no control characters".into(),
                ));
            }
        }

        if let Some(certificate) = &self.certificate {
            let value = certificate.to_string_lossy();
            if value.chars().any(char::is_control) {
                return Err(invalid(
                    "certificate path must contain no control characters".into(),
                ));
            }
        }

        Ok(())
    }

    pub fn server_args(&self, module_dir: &Path) -> Vec<OsString> {
        let mut args = vec![OsString::from("--listen"), OsString::from(&self.listen)];
        if let Some(token) = &self.token {
            args.push(OsString::from("--token"));
            args.push(OsString::from(token));
        }
        if let Some(certificate) = &self.certificate {
            let path = if certificate.is_absolute() {
                certificate.clone()
            } else {
                module_dir.join(certificate)
            };
            args.push(OsString::from("--certificate"));
            args.push(path.into_os_string());
        }
        args
    }
}

fn decode_hex(value: &str) -> Result<String, ConfigError> {
    if value.len() > 16_384 || value.len() % 2 != 0 {
        return Err(ConfigError::Invalid {
            line: 0,
            message: "configuration payload has an invalid size".into(),
        });
    }
    let mut bytes = Vec::with_capacity(value.len() / 2);
    for pair in value.as_bytes().chunks_exact(2) {
        let high = hex_nibble(pair[0]).ok_or_else(|| ConfigError::Invalid {
            line: 0,
            message: "configuration payload is not hexadecimal".into(),
        })?;
        let low = hex_nibble(pair[1]).ok_or_else(|| ConfigError::Invalid {
            line: 0,
            message: "configuration payload is not hexadecimal".into(),
        })?;
        bytes.push((high << 4) | low);
    }
    String::from_utf8(bytes).map_err(|_| ConfigError::Invalid {
        line: 0,
        message: "configuration payload is not UTF-8".into(),
    })
}

fn hex_nibble(value: u8) -> Option<u8> {
    match value {
        b'0'..=b'9' => Some(value - b'0'),
        b'a'..=b'f' => Some(value - b'a' + 10),
        b'A'..=b'F' => Some(value - b'A' + 10),
        _ => None,
    }
}

fn required_field<'a>(
    fields: &BTreeMap<&'a str, &'a str>,
    key: &str,
) -> Result<&'a str, ConfigError> {
    fields
        .get(key)
        .copied()
        .ok_or_else(|| ConfigError::Invalid {
            line: 0,
            message: format!("missing payload key: {key}"),
        })
}

fn parse_bool(value: &str, line: usize) -> Result<bool, ConfigError> {
    match value {
        "true" => Ok(true),
        "false" => Ok(false),
        _ => Err(ConfigError::Invalid {
            line,
            message: format!("expected true or false, got {value:?}"),
        }),
    }
}

fn parse_duration(value: &str, line: usize, maximum: u64) -> Result<Duration, ConfigError> {
    let seconds = value.parse::<u64>().map_err(|_| ConfigError::Invalid {
        line,
        message: format!("expected a non-negative integer, got {value:?}"),
    })?;
    if seconds > maximum {
        return Err(ConfigError::Invalid {
            line,
            message: format!("value must not exceed {maximum} seconds"),
        });
    }
    Ok(Duration::from_secs(seconds))
}

fn parse_listen_host(value: &str) -> Result<IpAddr, String> {
    if value.is_empty() || value.chars().any(char::is_whitespace) {
        return Err("listen must be an IP address and port without whitespace".into());
    }

    let (host, port) = if let Some(rest) = value.strip_prefix('[') {
        let (host, port) = rest
            .split_once("]:")
            .ok_or_else(|| "invalid bracketed IPv6 listen address".to_string())?;
        (host, port)
    } else {
        value
            .rsplit_once(':')
            .ok_or_else(|| "listen must use IP:PORT syntax".to_string())?
    };

    let port = port
        .parse::<u16>()
        .map_err(|_| "listen port must be between 1 and 65535".to_string())?;
    if port == 0 {
        return Err("listen port must be between 1 and 65535".into());
    }

    if host.eq_ignore_ascii_case("localhost") {
        return Ok(IpAddr::from([127, 0, 0, 1]));
    }
    host.parse::<IpAddr>()
        .map_err(|_| "listen host must be an IP address or localhost".into())
}

fn main() -> ExitCode {
    match run() {
        Ok(code) => ExitCode::from(code),
        Err(error) => {
            eprintln!("frida-ksud: {error}");
            ExitCode::FAILURE
        }
    }

fn run() -> Result<u8, Box<dyn Error>> {
    let cli = Cli::parse()?;
    match cli.command {
        CommandKind::Help => {
            print!("{USAGE}");
            return Ok(0);
        }
        CommandKind::Version => {
            println!("frida-ksud {}", env!("CARGO_PKG_VERSION"));
            return Ok(0);
        }
        _ => {}
    }

    let (command, module_dir, command_args) = cli.resolve_module_dir()?;
    let layout = Layout::new(module_dir)?;
    match command {
        CommandKind::Start => print_start(supervisor::start(&layout)?),
        CommandKind::Daemon => supervisor::daemon(&layout)?,
        CommandKind::Stop => match supervisor::stop(&layout)? {
            StopOutcome::Stopped => println!("Frida supervisor stopped"),
            StopOutcome::NotRunning => println!("Frida supervisor is not running"),
        },
        CommandKind::Restart => print_start(supervisor::restart(&layout)?),
        CommandKind::Status => {
            let status = supervisor::status(&layout);
            match status.supervisor_pid {
                Some(pid) => println!("supervisor: running (PID {pid})"),
                None => println!("supervisor: stopped"),
            }
            match status.server_pid {
                Some(pid) => println!("frida-server: running (PID {pid})"),
                None => println!("frida-server: stopped"),
            }
            if !status.is_running() {
                return Ok(3);
            }
        }
        CommandKind::Inspect => println!("{}", supervisor::inspect(&layout)?),
        CommandKind::Configure => {
            let payload = configure_payload(&command_args)?;
            let mut config = Config::load(&layout.module_dir)?;
            config.apply_hex_payload(payload)?;
            config.save(&layout.module_dir)?;
            println!("configuration updated");
        }
        CommandKind::Logs => {
            let (source, lines) = log_options(&command_args)?;
            print!("{}", supervisor::log_tail(&layout, source, lines)?);
        }
        CommandKind::ClearLogs => {
            supervisor::clear_logs(&layout)?;
            println!("logs cleared");
        }
        CommandKind::Help | CommandKind::Version => unreachable!(),
    }
    Ok(0)
}

fn print_start(outcome: StartOutcome) {
    match outcome {
        StartOutcome::Started(pid) => println!("Frida supervisor started (PID {pid})"),
        StartOutcome::AlreadyRunning(pid) => {
            println!("Frida supervisor is already running (PID {pid})")
        }
        StartOutcome::Disabled => println!("Frida supervisor is disabled in config.conf"),
    }
}

fn configure_payload(args: &[OsString]) -> Result<&str, io::Error> {
    if args.len() != 2 || args[0] != "--payload-hex" {
        return Err(io::Error::new(
            io::ErrorKind::InvalidInput,
            "configure requires --payload-hex HEX",
        ));
    }
    args[1]
        .to_str()
        .ok_or_else(|| io::Error::new(io::ErrorKind::InvalidInput, "payload is not UTF-8"))
}

fn log_options(args: &[OsString]) -> Result<(LogSource, usize), io::Error> {
    let mut source = LogSource::Server;
    let mut lines = 400;
    let mut index = 0;
    while index < args.len() {
        let option = args[index]
            .to_str()
            .ok_or_else(|| io::Error::new(io::ErrorKind::InvalidInput, "option is not UTF-8"))?;
        let value = args
            .get(index + 1)
            .and_then(|value| value.to_str())
            .ok_or_else(|| {
                io::Error::new(
                    io::ErrorKind::InvalidInput,
                    format!("{option} requires a value"),
                )
            })?;
        match option {
            "--source" => {
                source = match value {
                    "supervisor" => LogSource::Supervisor,
                    "server" => LogSource::Server,
                    _ => {
                        return Err(io::Error::new(
                            io::ErrorKind::InvalidInput,
                            "log source must be supervisor or server",
                        ));
                    }
                };
            }
            "--lines" => {
                lines = value.parse::<usize>().map_err(|_| {
                    io::Error::new(
                        io::ErrorKind::InvalidInput,
                        "log line count must be an integer",
                    )
                })?;
            }
            _ => {
                return Err(io::Error::new(
                    io::ErrorKind::InvalidInput,
                    format!("unknown logs option: {option}"),
                ));
            }
        }
        index += 2;
    }
    Ok((source, lines))
}

use std::io;
use std::sync::atomic::{AtomicBool, Ordering};

static SHUTDOWN: AtomicBool = AtomicBool::new(false);

#[cfg(unix)]
const SIGINT: i32 = 2;
#[cfg(unix)]
const SIGKILL: i32 = 9;
#[cfg(unix)]
const SIGTERM: i32 = 15;
#[cfg(unix)]
const EPERM: i32 = 1;
#[cfg(unix)]
const ESRCH: i32 = 3;

#[cfg(unix)]
unsafe extern "C" {
    #[link_name = "kill"]
    fn c_kill(pid: i32, signal: i32) -> i32;
    #[link_name = "signal"]
    fn c_signal(signal: i32, handler: extern "C" fn(i32)) -> usize;
}

#[cfg(unix)]
extern "C" fn handle_shutdown(_: i32) {
    SHUTDOWN.store(true, Ordering::Relaxed);
}

#[cfg(unix)]
pub fn process_exists(pid: u32) -> bool {
    let Ok(pid) = i32::try_from(pid) else {
        return false;
    };
    let result = unsafe { c_kill(pid, 0) };
    result == 0 || io::Error::last_os_error().raw_os_error() == Some(EPERM)
}

#[cfg(unix)]
pub fn terminate(pid: u32) -> io::Result<()> {
    send_signal(pid, SIGTERM)
}

#[cfg(unix)]
pub fn kill(pid: u32) -> io::Result<()> {
    send_signal(pid, SIGKILL)
}

#[cfg(unix)]
fn send_signal(pid: u32, signal: i32) -> io::Result<()> {
    let pid = i32::try_from(pid)
        .map_err(|_| io::Error::new(io::ErrorKind::InvalidInput, "PID exceeds i32 range"))?;
    if unsafe { c_kill(pid, signal) } == 0 {
        return Ok(());
    }

    let error = io::Error::last_os_error();
    if error.raw_os_error() == Some(ESRCH) {
        Ok(())
    } else {
        Err(error)
    }
}

#[cfg(unix)]
pub fn shutdown_flag() -> io::Result<&'static AtomicBool> {
    SHUTDOWN.store(false, Ordering::Relaxed);
    if unsafe { c_signal(SIGTERM, handle_shutdown) } == usize::MAX {
        return Err(io::Error::last_os_error());
    }
    if unsafe { c_signal(SIGINT, handle_shutdown) } == usize::MAX {
        return Err(io::Error::last_os_error());
    }
    Ok(&SHUTDOWN)
}

#[cfg(windows)]
mod windows {
    use super::*;
    use std::ffi::c_void;

    type Handle = *mut c_void;

    const PROCESS_QUERY_LIMITED_INFORMATION: u32 = 0x1000;
    const PROCESS_TERMINATE: u32 = 0x0001;
    const STILL_ACTIVE: u32 = 259;

    #[link(name = "kernel32")]
    unsafe extern "system" {
        fn OpenProcess(access: u32, inherit_handle: i32, process_id: u32) -> Handle;
        fn GetExitCodeProcess(process: Handle, exit_code: *mut u32) -> i32;
        fn TerminateProcess(process: Handle, exit_code: u32) -> i32;
        fn CloseHandle(object: Handle) -> i32;
    }

    pub fn process_exists(pid: u32) -> bool {
        let handle = unsafe { OpenProcess(PROCESS_QUERY_LIMITED_INFORMATION, 0, pid) };
        if handle.is_null() {
            return false;
        }

        let mut exit_code = 0;
        let success = unsafe { GetExitCodeProcess(handle, &mut exit_code) } != 0;
        unsafe { CloseHandle(handle) };
        success && exit_code == STILL_ACTIVE
    }

    pub fn terminate(pid: u32) -> io::Result<()> {
        if !process_exists(pid) {
            return Ok(());
        }

        let handle = unsafe { OpenProcess(PROCESS_TERMINATE, 0, pid) };
        if handle.is_null() {
            return Err(io::Error::last_os_error());
        }

        let result = unsafe { TerminateProcess(handle, 1) };
        let error = (result == 0).then(io::Error::last_os_error);
        unsafe { CloseHandle(handle) };
        match error {
            Some(error) => Err(error),
            None => Ok(()),
        }
    }

    pub fn shutdown_flag() -> io::Result<&'static AtomicBool> {
        SHUTDOWN.store(false, Ordering::Relaxed);
        Ok(&SHUTDOWN)
    }
}

#[cfg(windows)]

    terminate(pid)

use std::io::{self, Read, Seek, SeekFrom, Write};
use std::path::{Path, PathBuf};
use std::process::{Child, Command, Stdio};
use std::sync::atomic::Ordering;
use std::thread;
use std::time::{Duration, SystemTime};

const SUPERVISOR_PID_FILE: &str = "supervisor.pid";
const SERVER_PID_FILE: &str = "frida-server.pid";
const POLL_INTERVAL: Duration = Duration::from_millis(100);
const STOP_TIMEOUT: Duration = Duration::from_secs(5);

type Result<T, E = Box<dyn Error>> = std::result::Result<T, E>;

#[derive(Clone, Debug)]
pub struct Layout {
    pub module_dir: PathBuf,
    bin_dir: PathBuf,
    run_dir: PathBuf,
    log_dir: PathBuf,
}

impl Layout {
    pub fn new(module_dir: PathBuf) -> io::Result<Self> {
        let module_dir = fs::canonicalize(module_dir)?;
        Ok(Self {
            bin_dir: module_dir.join("bin"),
            run_dir: module_dir.join("run"),
            log_dir: module_dir.join("logs"),
            module_dir,
        })
    }

    fn supervisor_pid_file(&self) -> PathBuf {
        self.run_dir.join(SUPERVISOR_PID_FILE)
    }

    fn server_pid_file(&self) -> PathBuf {
        self.run_dir.join(SERVER_PID_FILE)
    }

    fn server(&self) -> PathBuf {
        self.bin_dir.join("frida-server")
    }

    fn supervisor_log(&self) -> PathBuf {
        self.log_dir.join("supervisor.log")
    }

    fn server_log(&self) -> PathBuf {
        self.log_dir.join("frida-server.log")
    }

    fn prepare(&self) -> io::Result<()> {
        fs::create_dir_all(&self.run_dir)?;
        fs::create_dir_all(&self.log_dir)
    }
}

#[derive(Debug, Eq, PartialEq)]
pub struct ProcessStatus {
    pub supervisor_pid: Option<u32>,
    pub server_pid: Option<u32>,
}

impl ProcessStatus {
    pub fn is_running(&self) -> bool {
        self.supervisor_pid.is_some()
    }
}

#[derive(Debug, Eq, PartialEq)]
pub enum StartOutcome {
    Started(u32),
    AlreadyRunning(u32),
    Disabled,
}

#[derive(Debug, Eq, PartialEq)]
pub enum StopOutcome {
    Stopped,
    NotRunning,
}

#[derive(Clone, Copy, Debug, Eq, PartialEq)]
pub enum LogSource {
    Supervisor,
    Server,
}

pub fn status(layout: &Layout) -> ProcessStatus {
    ProcessStatus {
        supervisor_pid: read_live_pid(&layout.supervisor_pid_file()),
        server_pid: read_live_pid(&layout.server_pid_file()),
    }
}

pub fn inspect(layout: &Layout) -> Result<String> {
    let processes = status(layout);
    let config = Config::load(&layout.module_dir)?;
    let service_state = if processes.server_pid.is_some() {
        "running"
    } else if processes.supervisor_pid.is_some() {
        "starting"
    } else {
        "stopped"
    };
    let frida_version = fs::read_to_string(layout.module_dir.join("frida.version"))
        .unwrap_or_else(|_| "unknown".into());
    let certificate = config
        .certificate
        .as_deref()
        .map(|path| json_string(&path.to_string_lossy()))
        .unwrap_or_else(|| "null".into());
    let supervisor_log_bytes = file_size(&layout.supervisor_log());
    let server_log_bytes = file_size(&layout.server_log());

    Ok(format!(
        concat!(
            "{{",
            "\"schemaVersion\":2,",
            "\"service\":{{",
            "\"state\":{},",
            "\"supervisorPid\":{},",
            "\"serverPid\":{}",
            "}},",
            "\"config\":{{",
            "\"enabled\":{},",
            "\"listen\":{},",
            "\"tokenConfigured\":{},",
            "\"allowUnauthenticatedRemote\":{},",
            "\"certificate\":{},",
            "\"restart\":{},",
            "\"restartDelaySeconds\":{},",
            "\"startupDelaySeconds\":{}",
            "}},",
            "\"runtime\":{{",
            "\"moduleVersion\":{},",
            "\"fridaVersion\":{},",
            "\"architecture\":{},",
            "\"moduleDir\":{},",
            "\"supervisorLogBytes\":{},",
            "\"serverLogBytes\":{}",
            "}}",
            "}}"
        ),
        json_string(service_state),
        json_pid(processes.supervisor_pid),
        json_pid(processes.server_pid),
        config.enabled,
        json_string(&config.listen),
        config.token.is_some(),
        config.allow_unauthenticated_remote,
        certificate,
        config.restart,
        config.restart_delay.as_secs(),
        config.startup_delay.as_secs(),
        json_string(env!("CARGO_PKG_VERSION")),
        json_string(frida_version.trim()),
        json_string(std::env::consts::ARCH),
        json_string(&layout.module_dir.to_string_lossy()),
        supervisor_log_bytes,
        server_log_bytes,
    ))
}

pub fn log_tail(layout: &Layout, source: LogSource, lines: usize) -> io::Result<String> {
    if !(1..=2_000).contains(&lines) {
        return Err(io::Error::new(
            io::ErrorKind::InvalidInput,
            "log line count must be between 1 and 2000",
        ));
    }
    let path = match source {
        LogSource::Supervisor => layout.supervisor_log(),
        LogSource::Server => layout.server_log(),
    };
    let mut file = match File::open(path) {
        Ok(file) => file,
        Err(error) if error.kind() == io::ErrorKind::NotFound => return Ok(String::new()),
        Err(error) => return Err(error),
    };
    let length = file.metadata()?.len();
    let start = length.saturating_sub(512 * 1_024);
    file.seek(SeekFrom::Start(start))?;
    let mut data = Vec::with_capacity((length - start) as usize);
    file.read_to_end(&mut data)?;
    if start > 0
        && let Some(newline) = data.iter().position(|byte| *byte == b'\n')
    {
        data.drain(..=newline);
    }

    let mut line_breaks = 0;
    let mut tail_start = 0;
    for index in (0..data.len()).rev() {
        if data[index] == b'\n' && index + 1 < data.len() {
            line_breaks += 1;
            if line_breaks == lines {
                tail_start = index + 1;
                break;
            }
        }
    }
    Ok(String::from_utf8_lossy(&data[tail_start..]).into_owned())
}

pub fn clear_logs(layout: &Layout) -> io::Result<()> {
    layout.prepare()?;
    for path in [layout.supervisor_log(), layout.server_log()] {
        OpenOptions::new()
            .create(true)
            .write(true)
            .truncate(true)
            .open(path)?;
    }
    Ok(())
}

pub fn start(layout: &Layout) -> Result<StartOutcome> {
    layout.prepare()?;
    let config = Config::load(&layout.module_dir)?;
    if !config.enabled {
        return Ok(StartOutcome::Disabled);
    }
    validate_payload(layout, &config)?;

    if let Some(pid) = status(layout).supervisor_pid {
        return Ok(StartOutcome::AlreadyRunning(pid));
    }

    let executable = std::env::current_exe()?;
    let log = open_log(&layout.log_dir.join("supervisor.log"))?;
    let error_log = log.try_clone()?;
    let mut child = Command::new(executable)
        .arg("daemon")
        .arg("--module-dir")
        .arg(&layout.module_dir)
        .stdin(Stdio::null())
        .stdout(Stdio::from(log))
        .stderr(Stdio::from(error_log))
        .spawn()?;

    for _ in 0..50 {
        if let Some(pid) = status(layout).supervisor_pid {
            return Ok(StartOutcome::Started(pid));
        }
        if let Some(exit_status) = child.try_wait()? {
            return Err(io::Error::other(format!(
                "supervisor exited during startup with {exit_status}"
            ))
            .into());
        }
        thread::sleep(POLL_INTERVAL);
    }

    platform::terminate(child.id())?;
    Err(io::Error::new(
        io::ErrorKind::TimedOut,
        "supervisor did not report ready within five seconds",
    )
    .into())
}

pub fn daemon(layout: &Layout) -> Result<()> {
    layout.prepare()?;
    let config = Config::load(&layout.module_dir)?;
    if !config.enabled {
        return Ok(());
    }
    validate_payload(layout, &config)?;

    let _lock = SupervisorLock::acquire(layout.supervisor_pid_file())?;
    let shutdown = platform::shutdown_flag()?;
    if sleep_interruptibly(config.startup_delay, shutdown) {
        return Ok(());
    }

    loop {
        let server_log = open_log(&layout.log_dir.join("frida-server.log"))?;
        let server_error_log = server_log.try_clone()?;
        let mut child = Command::new(layout.server())
            .args(config.server_args(&layout.module_dir))
            .current_dir(&layout.module_dir)
            .stdin(Stdio::null())
            .stdout(Stdio::from(server_log))
            .stderr(Stdio::from(server_error_log))
            .spawn()?;
        let child_pid = child.id();
        let child_pid_guard = PidFileGuard::write(layout.server_pid_file(), child_pid)?;
        eprintln!("frida-server started with PID {child_pid}");

        let exit_status = loop {
            if shutdown.load(Ordering::Relaxed) {
                stop_child(&mut child)?;
                drop(child_pid_guard);
                return Ok(());
            }
            if let Some(exit_status) = child.try_wait()? {
                break exit_status;
            }
            thread::sleep(POLL_INTERVAL);
        };
        drop(child_pid_guard);
        eprintln!("frida-server PID {child_pid} exited with {exit_status}");

        if !config.restart || sleep_interruptibly(config.restart_delay, shutdown) {
            return Ok(());
        }
    }
}

pub fn stop(layout: &Layout) -> Result<StopOutcome> {
    let initial = status(layout);
    if initial.supervisor_pid.is_none() && initial.server_pid.is_none() {
        remove_if_exists(&layout.supervisor_pid_file())?;
        remove_if_exists(&layout.server_pid_file())?;
        return Ok(StopOutcome::NotRunning);
    }

    if let Some(pid) = initial.supervisor_pid {
        platform::terminate(pid)?;
    }
    if let Some(pid) = initial.server_pid {
        platform::terminate(pid)?;
    }

    let deadline = std::time::Instant::now() + STOP_TIMEOUT;
    while std::time::Instant::now() < deadline {
        let current = status(layout);
        if current.supervisor_pid.is_none() && current.server_pid.is_none() {
            remove_if_exists(&layout.supervisor_pid_file())?;
            remove_if_exists(&layout.server_pid_file())?;
            return Ok(StopOutcome::Stopped);
        }
        thread::sleep(POLL_INTERVAL);
    }

    let remaining = status(layout);
    if let Some(pid) = remaining.supervisor_pid {
        platform::kill(pid)?;
    }
    if let Some(pid) = remaining.server_pid {
        platform::kill(pid)?;
    }
    remove_if_exists(&layout.supervisor_pid_file())?;
    remove_if_exists(&layout.server_pid_file())?;
    Ok(StopOutcome::Stopped)
}

pub fn restart(layout: &Layout) -> Result<StartOutcome> {
    stop(layout)?;
    start(layout)
}

fn file_size(path: &Path) -> u64 {
    fs::metadata(path)
        .map(|metadata| metadata.len())
        .unwrap_or(0)
}

fn json_pid(pid: Option<u32>) -> String {
    pid.map(|value| value.to_string())
        .unwrap_or_else(|| "null".into())
}

fn json_string(value: &str) -> String {
    const HEX: &[u8; 16] = b"0123456789abcdef";
    let mut output = String::with_capacity(value.len() + 2);
    output.push('"');
    for character in value.chars() {
        match character {
            '"' => output.push_str("\\\""),
            '\\' => output.push_str("\\\\"),
            '\n' => output.push_str("\\n"),
            '\r' => output.push_str("\\r"),
            '\t' => output.push_str("\\t"),
            value if value <= '\u{1f}' => {
                let code = value as usize;
                output.push_str("\\u00");
                output.push(HEX[(code >> 4) & 0xf] as char);
                output.push(HEX[code & 0xf] as char);
            }
            value => output.push(value),
        }
    }
    output.push('"');
    output
}

fn validate_payload(layout: &Layout, config: &Config) -> io::Result<()> {
    let server = layout.server();
    if !server.is_file() {
        return Err(io::Error::new(
            io::ErrorKind::NotFound,
            format!("Frida Server is missing: {}", server.display()),
        ));
    }

    if let Some(certificate) = &config.certificate {
        let certificate = if certificate.is_absolute() {
            certificate.clone()
        } else {
            layout.module_dir.join(certificate)
        };
        if !certificate.is_file() {
            return Err(io::Error::new(
                io::ErrorKind::NotFound,
                format!(
                    "configured certificate is missing: {}",
                    certificate.display()
                ),
            ));
        }
    }
    Ok(())
}

fn open_log(path: &Path) -> io::Result<File> {
    OpenOptions::new().create(true).append(true).open(path)
}

fn stop_child(child: &mut Child) -> io::Result<()> {
    platform::terminate(child.id())?;
    let deadline = std::time::Instant::now() + STOP_TIMEOUT;
    while std::time::Instant::now() < deadline {
        if child.try_wait()?.is_some() {
            return Ok(());
        }
        thread::sleep(POLL_INTERVAL);
    }
    platform::kill(child.id())?;
    child.wait()?;
    Ok(())
}

fn sleep_interruptibly(duration: Duration, shutdown: &std::sync::atomic::AtomicBool) -> bool {
    let deadline = std::time::Instant::now() + duration;
    while std::time::Instant::now() < deadline {
        if shutdown.load(Ordering::Relaxed) {
            return true;
        }
        thread::sleep(
            POLL_INTERVAL.min(deadline.saturating_duration_since(std::time::Instant::now())),
        );
    }
    shutdown.load(Ordering::Relaxed)
}

fn read_live_pid(path: &Path) -> Option<u32> {
    let pid = fs::read_to_string(path).ok()?.trim().parse::<u32>().ok()?;
    (pid > 0 && platform::process_exists(pid)).then_some(pid)
}

fn remove_if_exists(path: &Path) -> io::Result<()> {
    match fs::remove_file(path) {
        Ok(()) => Ok(()),
        Err(error) if error.kind() == io::ErrorKind::NotFound => Ok(()),
        Err(error) => Err(error),
    }
}

struct SupervisorLock {
    path: PathBuf,
    pid: u32,
}

impl SupervisorLock {
    fn acquire(path: PathBuf) -> io::Result<Self> {
        let pid = std::process::id();
        for _ in 0..2 {
            match OpenOptions::new().write(true).create_new(true).open(&path) {
                Ok(mut file) => {
                    writeln!(file, "{pid}")?;
                    file.sync_all()?;
                    return Ok(Self { path, pid });
                }
                Err(error) if error.kind() == io::ErrorKind::AlreadyExists => {
                    if let Some(existing_pid) = read_live_pid(&path) {
                        return Err(io::Error::new(
                            io::ErrorKind::AlreadyExists,
                            format!("supervisor is already running with PID {existing_pid}"),
                        ));
                    }
                    let recently_created = fs::metadata(&path)
                        .and_then(|metadata| metadata.modified())
                        .ok()
                        .and_then(|modified| SystemTime::now().duration_since(modified).ok())
                        .is_some_and(|age| age < Duration::from_secs(5));
                    if recently_created {
                        return Err(io::Error::new(
                            io::ErrorKind::AlreadyExists,
                            "supervisor lock is being initialized",
                        ));
                    }
                    remove_if_exists(&path)?;
                }
                Err(error) => return Err(error),
            }
        }
        Err(io::Error::other("failed to acquire supervisor lock"))
    }
}

impl Drop for SupervisorLock {
    fn drop(&mut self) {
        if fs::read_to_string(&self.path)
            .ok()
            .and_then(|value| value.trim().parse::<u32>().ok())
            == Some(self.pid)
        {
            let _ = fs::remove_file(&self.path);
        }
    }
}

struct PidFileGuard {
    path: PathBuf,
    pid: u32,
}

impl PidFileGuard {
    fn write(path: PathBuf, pid: u32) -> io::Result<Self> {
        fs::write(&path, format!("{pid}\n"))?;
        Ok(Self { path, pid })
    }
}

impl Drop for PidFileGuard {
    fn drop(&mut self) {
        if fs::read_to_string(&self.path)
            .ok()
            .and_then(|value| value.trim().parse::<u32>().ok())
            == Some(self.pid)
        {
            let _ = fs::remove_file(&self.path);
        }
    }
}

```ts
import '@material/web/button/filled-button.js';
import '@material/web/button/filled-tonal-button.js';
import '@material/web/button/outlined-button.js';
import '@material/web/button/text-button.js';
import '@material/web/dialog/dialog.js';
import '@material/web/divider/divider.js';
import '@material/web/iconbutton/icon-button.js';
import '@material/web/progress/circular-progress.js';
import '@material/web/progress/linear-progress.js';
import '@material/web/select/outlined-select.js';
import '@material/web/select/select-option.js';
import '@material/web/switch/switch.js';
import '@material/web/textfield/outlined-text-field.js';
import { LitElement, css, html, nothing, type TemplateResult } from 'lit';
import { customElement, state } from 'lit/decorators.js';
import './theme.css';
import { icon } from './icons';
import { KernelSuService } from './ksu-service';
import type {
  DeviceInfo,
  EditableConfig,
  InspectSnapshot,
  LogSource,
  ModuleInfo,
  Page,
  TokenAction,
} from './types';

@customElement('frida-app')
export class FridaApp extends LitElement {
  private readonly service = new KernelSuService();
  private statusTimer: number | undefined;
  private logTimer: number | undefined;
  private snackTimer: number | undefined;
  private pendingUnsafeRestart = false;

  @state() private page: Page = 'dashboard';
  @state() private snapshot: InspectSnapshot | null = null;
  @state() private module: ModuleInfo | null = null;
  @state() private device: DeviceInfo | null = null;
  @state() private form: EditableConfig = {
    enabled: true,
    listen: '127.0.0.1:27042',
    tokenAction: 'keep',
    token: '',
    allowUnauthenticatedRemote: false,
    certificate: '',
    restart: true,
    restartDelaySeconds: '2',
    startupDelaySeconds: '0',
  };
  @state() private formDirty = false;
  @state() private loading = true;
  @state() private refreshing = false;
  @state() private busyAction = '';
  @state() private errorMessage = '';
  @state() private logSource: LogSource = 'server';
  @state() private logText = '';
  @state() private logLoading = false;
  @state() private autoRefreshLogs = false;
  @state() private clearDialogOpen = false;
  @state() private unsafeDialogOpen = false;
  @state() private snackMessage = '';

  override firstUpdated(): void {
    void this.bootstrap();
  }

  override disconnectedCallback(): void {
    super.disconnectedCallback();
    window.clearInterval(this.statusTimer);
    window.clearInterval(this.logTimer);
    window.clearTimeout(this.snackTimer);
  }

  private async bootstrap(): Promise<void> {
    this.loading = true;
    this.errorMessage = '';
    try {
      this.module = await this.service.initialize();
      const [snapshot, device] = await Promise.all([
        this.service.inspect(),
        this.service.deviceInfo(),
      ]);
      this.snapshot = snapshot;
      this.device = device;
      this.syncForm(snapshot);
      this.statusTimer = window.setInterval(() => void this.refreshSnapshot(), 5_000);
    } catch (error) {
      this.errorMessage = this.errorText(error);
    } finally {
      this.loading = false;
    }
  }

  private async refreshSnapshot(showProgress = false): Promise<void> {
    if (!this.service.available || this.refreshing) return;
    this.refreshing = true;
    if (showProgress) this.busyAction = 'refresh';
    try {
      const snapshot = await this.service.inspect();
      this.snapshot = snapshot;
      if (!this.formDirty) this.syncForm(snapshot);
      this.errorMessage = '';
    } catch (error) {
      this.errorMessage = this.errorText(error);
    } finally {
      this.refreshing = false;
      if (showProgress) this.busyAction = '';
    }
  }

  private syncForm(snapshot: InspectSnapshot): void {
    this.form = {
      enabled: snapshot.config.enabled,
      listen: snapshot.config.listen,
      tokenAction: 'keep',
      token: '',
      allowUnauthenticatedRemote: snapshot.config.allowUnauthenticatedRemote,
      certificate: snapshot.config.certificate ?? '',
      restart: snapshot.config.restart,
      restartDelaySeconds: String(snapshot.config.restartDelaySeconds),
      startupDelaySeconds: String(snapshot.config.startupDelaySeconds),
    };
    this.formDirty = false;
  }

  private updateForm(patch: Partial<EditableConfig>): void {
    this.form = { ...this.form, ...patch };
    this.formDirty = true;
  }

  private tokenWillBeConfigured(): boolean {
    return this.form.tokenAction === 'set'
      || (this.form.tokenAction === 'keep' && Boolean(this.snapshot?.config.tokenConfigured));
  }

  private willUseUnauthenticatedWildcard(listen: string): boolean {
    return /^0\.0\.0\.0:\d{1,5}$/.test(listen) && !this.tokenWillBeConfigured();
  }

  private async runServiceAction(action: 'start' | 'stop' | 'restart'): Promise<void> {
    this.busyAction = action;
    try {
      await this.service.control(action);
      await this.refreshSnapshot();
      this.showSnack(
        action === 'start' ? 'Frida 服务已启动' : action === 'stop' ? 'Frida 服务已停止' : 'Frida 服务已重启',
      );
    } catch (error) {
      this.showSnack(this.errorText(error));
    } finally {
      this.busyAction = '';
    }
  }

  private async saveConfiguration(
    restartAfterSave: boolean,
    unsafeAccessConfirmed = false,
  ): Promise<void> {
    const listen = this.form.listen.trim();
    const restartDelay = Number(this.form.restartDelaySeconds);
    const startupDelay = Number(this.form.startupDelaySeconds);
    if (!listen) {
      this.showSnack('监听地址不能为空');
      return;
    }
    if (!Number.isInteger(restartDelay) || restartDelay < 0 || restartDelay > 3_600) {
      this.showSnack('重启延迟必须是 0–3600 秒的整数');
      return;
    }
    if (!Number.isInteger(startupDelay) || startupDelay < 0 || startupDelay > 300) {
      this.showSnack('启动延迟必须是 0–300 秒的整数');
      return;
    }
    if (this.form.tokenAction === 'set' && !this.form.token) {
      this.showSnack('设置新令牌时，令牌不能为空');
      return;
    }

    const allowUnauthenticatedRemote = this.willUseUnauthenticatedWildcard(listen);
    const alreadyApproved = Boolean(
      this.snapshot?.config.allowUnauthenticatedRemote
      && !this.snapshot.config.tokenConfigured
      && this.snapshot.config.listen === listen,
    );
    if (allowUnauthenticatedRemote && !alreadyApproved && !unsafeAccessConfirmed) {
      this.pendingUnsafeRestart = restartAfterSave;
      this.unsafeDialogOpen = true;
      return;
    }

    this.busyAction = restartAfterSave ? 'save-restart' : 'save';
    try {
      await this.service.configure({
        ...this.form,
        listen,
        allowUnauthenticatedRemote,
        certificate: this.form.certificate.trim(),
      });
      if (restartAfterSave) await this.service.control('restart');
      this.formDirty = false;
      await this.refreshSnapshot();
      this.showSnack(
        allowUnauthenticatedRemote
          ? restartAfterSave
            ? '无认证公开监听已保存并应用'
            : '无认证公开监听已保存'
          : restartAfterSave
            ? '配置已保存并应用'
            : '配置已安全保存',
      );
    } catch (error) {
      this.showSnack(this.errorText(error));
    } finally {
      this.busyAction = '';
    }
  }

  private confirmUnauthenticatedRemote(): void {
    const restartAfterSave = this.pendingUnsafeRestart;
    this.unsafeDialogOpen = false;
    void this.saveConfiguration(restartAfterSave, true);
  }

  private async refreshLogs(): Promise<void> {
    if (this.logLoading || !this.service.available) return;
    this.logLoading = true;
    try {
      this.logText = await this.service.logs(this.logSource, 600);
    } catch (error) {
      this.showSnack(this.errorText(error));
    } finally {
      this.logLoading = false;
    }
  }

  private setLogSource(source: LogSource): void {
    this.logSource = source;
    void this.refreshLogs();
  }

  private setAutoRefreshLogs(enabled: boolean): void {
    this.autoRefreshLogs = enabled;
    window.clearInterval(this.logTimer);
    this.logTimer = undefined;
    if (enabled) {
      this.logTimer = window.setInterval(() => void this.refreshLogs(), 3_000);
    }
  }

  private async copyLogs(): Promise<void> {
    try {
      await navigator.clipboard.writeText(this.logText);
      this.showSnack('日志已复制到剪贴板');
    } catch (error) {
      this.showSnack(`无法复制日志：${this.errorText(error)}`);
    }
  }

  private async clearLogs(): Promise<void> {
    this.clearDialogOpen = false;
    this.busyAction = 'clear-logs';
    try {
      await this.service.clearLogs();
      await Promise.all([this.refreshLogs(), this.refreshSnapshot()]);
      this.showSnack('所有日志已清空');
    } catch (error) {
      this.showSnack(this.errorText(error));
    } finally {
      this.busyAction = '';
    }
  }

  private navigate(page: Page): void {
    this.page = page;
    if (page === 'logs') void this.refreshLogs();
  }

  private showSnack(message: string): void {
    this.snackMessage = message;
    this.service.notify(message);
    window.clearTimeout(this.snackTimer);
    this.snackTimer = window.setTimeout(() => {
      this.snackMessage = '';
    }, 4_000);
  }

  private errorText(error: unknown): string {
    if (error instanceof Error) return error.message;
    return String(error);
  }

  override render(): TemplateResult {
    return html`
      <div class="app-shell">
        ${this.renderTopBar()} ${this.renderNavigation()}
        ${this.loading || this.busyAction ? html`<md-linear-progress indeterminate></md-linear-progress>` : nothing}
        <main>
          ${this.errorMessage ? this.renderError() : nothing}
          ${this.loading
            ? this.renderLoading()
            : this.page === 'dashboard'
              ? this.renderDashboard()
              : this.page === 'settings'
                ? this.renderSettings()
                : this.page === 'logs'
                  ? this.renderLogs()
                  : this.renderDiagnostics()}
        </main>
        ${this.renderClearDialog()}
        ${this.renderUnsafeDialog()}
        ${this.snackMessage
          ? html`<div class="snackbar" role="status" aria-live="polite">${this.snackMessage}</div>`
          : nothing}
      </div>
    `;
  }

  private renderTopBar(): TemplateResult {
    const state = this.snapshot?.service.state ?? 'stopped';
    return html`
      <header class="top-bar">
        <div class="brand-mark">F</div>
        <div class="brand-copy">
          <span class="title">Frida 控制中心</span>
          <span class="subtitle">KernelSU · ${this.snapshot?.runtime.fridaVersion ?? '正在连接'}</span>
        </div>
        <span class="top-status ${state}">
          <span class="status-dot"></span>${this.stateLabel(state)}
        </span>
        <md-icon-button
          aria-label="刷新状态"
          title="刷新状态"
          ?disabled=${this.refreshing}
          @click=${() => void this.refreshSnapshot(true)}
        >${icon('refresh')}</md-icon-button>
      </header>
    `;
  }

  private renderNavigation(): TemplateResult {
    const pages: Array<{ page: Page; label: string; icon: Parameters<typeof icon>[0] }> = [
      { page: 'dashboard', label: '概览', icon: 'dashboard' },
      { page: 'settings', label: '配置', icon: 'settings' },
      { page: 'logs', label: '日志', icon: 'terminal' },
      { page: 'diagnostics', label: '诊断', icon: 'info' },
    ];
    return html`
      <nav class="nav-tabs" aria-label="主要页面">
        ${pages.map(
          (item) => html`
            <button
              class="nav-item ${this.page === item.page ? 'active' : ''}"
              type="button"
              aria-current=${this.page === item.page ? 'page' : nothing}
              @click=${() => this.navigate(item.page)}
            >
              ${icon(item.icon)}<span>${item.label}</span>
            </button>
          `,
        )}
      </nav>
    `;
  }

  private renderError(): TemplateResult {
    return html`
      <section class="error-banner" role="alert">
        <div class="banner-icon">${icon('warning')}</div>
        <div>
          <strong>控制通道不可用</strong>
          <p>${this.errorMessage}</p>
        </div>
        <md-filled-tonal-button @click=${() => void this.bootstrap()}>重试</md-filled-tonal-button>
      </section>
    `;
  }

  private renderLoading(): TemplateResult {
    return html`
      <section class="loading-state" aria-label="正在加载">
        <md-circular-progress indeterminate></md-circular-progress>
        <h2>正在连接 KernelSU</h2>
        <p>读取守护进程、Frida Server 和设备状态…</p>
      </section>
    `;
  }

  private renderDashboard(): TemplateResult {
    const snapshot = this.snapshot;
    if (!snapshot) return html`<div></div>`;
    const running = snapshot.service.state === 'running';

      <section class="hero-card ${snapshot.service.state}">

          <md-filled-tonal-button
            ?disabled=${stopped || Boolean(this.busyAction)}

          <md-outlined-button

  private formatBytes(bytes: number): string {
    if (bytes < 1_024) return `${bytes} B`;
    if (bytes < 1_048_576) return `${(bytes / 1_024).toFixed(1)} KiB`;
    return `${(bytes / 1_048_576).toFixed(1)} MiB`;
  }

`webui/src/ksu-service.ts`:

```ts
import {
  enableEdgeToEdge,
  exec,
  moduleInfo as readModuleInfo,
  spawn,
  toast,
} from 'kernelsu';
import type {
  DeviceInfo,
  EditableConfig,
  InspectSnapshot,
  LogSource,
  ModuleInfo,
} from './types';

const SAFE_MODULE_DIR = /^\/data\/adb\/modules\/[A-Za-z][A-Za-z0-9._-]+$/;
const COMMAND_TIMEOUT_MS = 20_000;

export class KernelSuService {
  private module: ModuleInfo | null = null;

  get available(): boolean {
    return typeof (globalThis as typeof globalThis & { ksu?: unknown }).ksu !== 'undefined';
  }

  get moduleInfo(): ModuleInfo | null {
    return this.module;
  }

  async initialize(): Promise<ModuleInfo> {
    if (!this.available) {
      throw new Error('KernelSU WebUI 桥接不可用，请从 KernelSU Manager 打开此页面。');
    }
    enableEdgeToEdge(true);
    const parsed = parseModuleInfo(readModuleInfo());
    if (!SAFE_MODULE_DIR.test(parsed.moduleDir)) {
      throw new Error('KernelSU 返回了无效的模块目录。');
    }
    this.module = parsed;
    return parsed;
  }

  async inspect(): Promise<InspectSnapshot> {
    return parseInspectSnapshot(await this.runBackend(['inspect']));
  }

  async control(action: 'start' | 'stop' | 'restart'): Promise<string> {
    return this.runBackend([action]);
  }

  async configure(config: EditableConfig): Promise<string> {
    const payload = [
      `enabled=${config.enabled}`,
      `listen=${config.listen}`,
      `token_action=${config.tokenAction}`,
      `token=${config.token}`,
      `allow_unauthenticated_remote=${config.allowUnauthenticatedRemote}`,
      `certificate=${config.certificate}`,
      `restart=${config.restart}`,
      `restart_delay_seconds=${config.restartDelaySeconds}`,
      `startup_delay_seconds=${config.startupDelaySeconds}`,
    ].join('\n');
    const encoded = Array.from(
      new TextEncoder().encode(payload),
      (byte) => byte.toString(16).padStart(2, '0'),
    ).join('');
    return this.runBackend(['configure', '--payload-hex', encoded]);
  }

  async logs(source: LogSource, lines = 400): Promise<string> {
    return this.runBackend(['logs', '--source', source, '--lines', String(lines)]);
  }

  async clearLogs(): Promise<string> {
    return this.runBackend(['clear-logs']);
  }

  async deviceInfo(): Promise<DeviceInfo> {
    const values = await Promise.all([
      this.runSystem('getprop ro.product.manufacturer'),
      this.runSystem('getprop ro.product.model'),
      this.runSystem('getprop ro.build.version.release'),
      this.runSystem('getprop ro.build.version.sdk'),
      this.runSystem('uname -r'),
    ]);
    return {
      manufacturer: values[0] || '未知',
      model: values[1] || '未知',
      android: values[2] || '未知',
      sdk: values[3] || '未知',
      kernel: values[4] || '未知',
    };
  }

  notify(message: string): void {
    if (this.available) {
      toast(message);
    }
  }

  private async runSystem(command: string): Promise<string> {
    const result = await exec(command);
    if (result.errno !== 0) {
      throw new Error(result.stderr || `${command} 执行失败 (${result.errno})`);
    }
    return result.stdout.trim();
  }

  private runBackend(args: string[]): Promise<string> {
    if (!this.module) {
      return Promise.reject(new Error('WebUI 尚未初始化。'));
    }
    const binary = `${this.module.moduleDir}/bin/frida-ksud`;
    const fullArgs = [...args, '--module-dir', this.module.moduleDir];

    const { promise, resolve, reject } = Promise.withResolvers<string>();
    const stdout: string[] = [];
    const stderr: string[] = [];
    let settled = false;
    const timeout = window.setTimeout(() => {
      if (!settled) {
        settled = true;
        reject(new Error('控制命令执行超时。'));
      }
    }, COMMAND_TIMEOUT_MS);
    const child = spawn(binary, fullArgs);
    child.stdout.on('data', (data) => stdout.push(data));
    child.stderr.on('data', (data) => stderr.push(data));
    child.on('error', (error) => {
      if (!settled) {
        settled = true;
        window.clearTimeout(timeout);
        reject(error instanceof Error ? error : new Error(String(error)));
      }
    });
    child.on('exit', (code) => {
      if (settled) return;
      settled = true;
      window.clearTimeout(timeout);
      const output = stdout.join('').trim();
      const errors = stderr.join('').trim();
      if (code === 0) {
        resolve(output);
      } else {
        reject(new Error(errors || output || `控制命令失败 (${code})`));
      }
    });
    return promise;
  }
}


function parseModuleInfo(raw: string): ModuleInfo {
  const data = parseJsonObject(raw, 'KernelSU 模块信息');
  return {
    moduleDir: readString(data, 'moduleDir'),
    id: readOptionalString(data, 'id'),
    name: readOptionalString(data, 'name'),
    version: readOptionalString(data, 'version'),
    versionCode: readOptionalVersionCode(data),
    author: readOptionalString(data, 'author'),
    description: readOptionalString(data, 'description'),
  };
}

function parseInspectSnapshot(raw: string): InspectSnapshot {
  const data = parseJsonObject(raw, '守护进程状态');
  if (data.schemaVersion !== 2) {
    throw new Error('守护进程返回了不兼容的状态数据。');
  }
  const service = requireRecord(data.service, 'service');
  const config = requireRecord(data.config, 'config');
  const runtime = requireRecord(data.runtime, 'runtime');
  const state = readString(service, 'state');
  if (state !== 'running' && state !== 'starting' && state !== 'stopped') {
    throw new Error('守护进程返回了无效的服务状态。');
  }

  return {
    schemaVersion: 2,
    service: {
      state,
      supervisorPid: readNullableNumber(service, 'supervisorPid'),
      serverPid: readNullableNumber(service, 'serverPid'),
    },
    config: {
      enabled: readBoolean(config, 'enabled'),
      listen: readString(config, 'listen'),
      tokenConfigured: readBoolean(config, 'tokenConfigured'),
      allowUnauthenticatedRemote: readBoolean(config, 'allowUnauthenticatedRemote'),
      certificate: readNullableString(config, 'certificate'),
      restart: readBoolean(config, 'restart'),
      restartDelaySeconds: readNumber(config, 'restartDelaySeconds'),
      startupDelaySeconds: readNumber(config, 'startupDelaySeconds'),
    },
    runtime: {
      moduleVersion: readString(runtime, 'moduleVersion'),
      fridaVersion: readString(runtime, 'fridaVersion'),
      architecture: readString(runtime, 'architecture'),
      moduleDir: readString(runtime, 'moduleDir'),
      supervisorLogBytes: readNumber(runtime, 'supervisorLogBytes'),
      serverLogBytes: readNumber(runtime, 'serverLogBytes'),
    },
  };
}

function parseJsonObject(raw: string, label: string): Record<string, unknown> {
  let value: unknown;
  try {
    value = JSON.parse(raw);
  } catch {
    throw new Error(`${label}不是有效 JSON。`);
  }
  return requireRecord(value, label);
}

function isRecord(value: unknown): value is Record<string, unknown> {
  return typeof value === 'object' && value !== null && !Array.isArray(value);
}

function requireRecord(value: unknown, label: string): Record<string, unknown> {
  if (!isRecord(value)) {
    throw new Error(`${label}格式无效。`);
  }
  return value;
}

function readString(record: Record<string, unknown>, key: string): string {
  const value = record[key];
  if (typeof value !== 'string') {
    throw new Error(`状态字段 ${key} 必须是字符串。`);
  }
  return value;
}

function readOptionalString(record: Record<string, unknown>, key: string): string | undefined {
  const value = record[key];
  if (value === undefined) return undefined;
  if (typeof value !== 'string') {
    throw new Error(`状态字段 ${key} 必须是字符串。`);
  }
  return value;
}

function readNullableString(record: Record<string, unknown>, key: string): string | null {
  const value = record[key];
  if (value === null) return null;
  if (typeof value !== 'string') {
    throw new Error(`状态字段 ${key} 必须是字符串或 null。`);
  }
  return value;
}

function readBoolean(record: Record<string, unknown>, key: string): boolean {
  const value = record[key];
  if (typeof value !== 'boolean') {
    throw new Error(`状态字段 ${key} 必须是布尔值。`);
  }
  return value;
}

function readNumber(record: Record<string, unknown>, key: string): number {
  const value = record[key];
  if (typeof value !== 'number' || !Number.isFinite(value) || value < 0) {
    throw new Error(`状态字段 ${key} 必须是非负数。`);
  }
  return value;
}

function readOptionalVersionCode(record: Record<string, unknown>): number | undefined {
  const value = record.versionCode;
  if (value === undefined) return undefined;
  if (typeof value === 'number' && Number.isSafeInteger(value) && value >= 0) {
    return value;
  }
  if (typeof value === 'string' && /^\d+$/.test(value)) {
    const parsed = Number(value);
    if (Number.isSafeInteger(parsed)) return parsed;
  }
  throw new Error('KernelSU 字段 versionCode 必须是非负整数。');
}

function readNullableNumber(record: Record<string, unknown>, key: string): number | null {
  const value = record[key];
  if (value === null) return null;
  if (typeof value !== 'number' || !Number.isInteger(value) || value <= 0) {
    throw new Error(`状态字段 ${key} 必须是正整数或 null。`);
  }
  return value;
}
