## 🚀 Overview
This repository contains a technical proposal and a Python-based proof-of-concept for a hybrid anti-cheat system designed for Counter-Strike 2. The goal is to close the "arms race" gap where cheat developers adapt faster than machine learning (VAC Live) can retrain.

## 🧠 Key Features

### 1. Glicko-2 Judge Rating
Unlike the old Overwatch, this system uses the **Glicko-2 algorithm** to rate judges based on their accuracy. 
* **Bot-Farm Protection:** High entry barriers (wins/hours) and weighted voting make bribery or botting economically impossible.
* **Vote Weight:** A Senior Judge's verdict carries more weight than a Trainee's.

### 2. Invisible Honeypot Entities (Deterministic Detection)
A method to provide mathematical proof of cheating. 
* Invisible "honeypot" entities are placed inside geometry or behind walls. 
* If a player's telemetry shows consistent snapping or tracking of these invisible targets, the system records a **100% deterministic violation**.

### 3. Shadow Monitoring (2-3 Match Verification)
To eliminate False Positives:
* If found "Guilty" by consensus, the account is shadow-flagged.
* It remains under surveillance for the next 2-3 matches.
* The ban is only finalized if cheating patterns (AIM/WH anomalies) persist.

---

## 🛠 Movement Bug Fix (Bonus)
This proposal also includes a logic fix for the "Fall Damage" bug in Source 2:
* **The Problem:** Engine calculates damage based on "air-time," leading to deaths while sliding on boxes.
* **The Fix:** Switch to a **Vertical Height Delta (Z-delta)** method. Reset the fall interval on *any* solid contact (ledge, slope, or box edge).

---

## 💻 How to use this repository
1. Read the documentation above for the logic.
2. Run `prototype.py` (to be added) to see the Python simulation of judge ratings and case processing.

@dataclass
class EvidenceFragment:
    fragment_id: str
    timestamp: float
    duration: int  # seconds
    telemetry_data: Dict
    is_honeypot: bool = False
    honeypot_type: Optional[str] = None  # "aimlock", "wallhack"

    def analyze_honeypot_trigger(self) -> bool:
        """Deterministic detection: did crosshair snap to invisible entity?"""
        if not self.is_honeypot:
            return False

        aim_data = self.telemetry_data.get("aim_snap_events", [])
        wall_data = self.telemetry_data.get("wall_trace_events", [])

        if self.honeypot_type == "aimlock":
            # Mathematical proof: crosshair snapped to invisible hitbox
            return len(aim_data) > 0 and any(e["target_invisible"] for e in aim_data)

        elif self.honeypot_type == "wallhack":
            # Mathematical proof: player tracked entity through solid geometry
            return len(wall_data) > 0 and any(e["through_wall"] for e in wall_data)

        return False

            raise ValueError("Judge is temporarily banned")

            judge.apply_lazy_penalty()
            if not judge.is_active:
                return

        self.verdicts.append((judge.steam_id, verdict, weight, time_spent))

    def calculate_consensus(self, min_judges: int = 20, max_judges: int = 30) -> Optional[Verdict]:
        if len(self.verdicts) < min_judges:
            return None

        weighted_guilty = sum(v[2] for v in self.verdicts if v[1] == Verdict.GUILTY)
        weighted_not_guilty = sum(v[2] for v in self.verdicts if v[1] == Verdict.NOT_GUILTY)
        weighted_too_complex = sum(v[2] for v in self.verdicts if v[1] == Verdict.TOO_COMPLEX)
        total_weight = weighted_guilty + weighted_not_guilty + weighted_too_complex

        if total_weight == 0:
            return None

        guilty_ratio = weighted_guilty / total_weight
        not_guilty_ratio = weighted_not_guilty / total_weight

        # Require 75% weighted consensus
        if guilty_ratio >= 0.75:
            return Verdict.GUILTY
        elif not_guilty_ratio >= 0.75:
            return Verdict.NOT_GUILTY
        elif len(self.verdicts) >= max_judges:
            # Max judges reached, case too complex
            return Verdict.TOO_COMPLEX

        return None

    def finalize_verdict(self, verdict: Verdict):
        self.final_verdict = verdict
        self.resolved_at = datetime.now()
        self.data_deletion_scheduled = self.resolved_at + timedelta(days=7)

        if verdict == Verdict.GUILTY:
            self.status = CaseStatus.SHADOW_FLAGGED
            self.shadow_matches_remaining = random.randint(2, 3)
        else:

class Glicko2System:
    """Implementation of Glicko-2 rating system for judge reliability."""

    TAU = 0.5  # System constant

    @staticmethod
    def scale_down(rating: float, rd: float) -> Tuple[float, float]:
        mu = (rating - 1500) / 173.7178
        phi = rd / 173.7178
        return mu, phi

    @staticmethod
    def scale_up(mu: float, phi: float) -> Tuple[float, float]:
        rating = 173.7178 * mu + 1500
        rd = 173.7178 * phi
        return rating, rd

    @classmethod
    def update_rating(cls, judge: Judge, outcomes: List[Tuple[float, float, float]]):
        """
        Update judge rating based on case outcomes.
        outcomes: List of (opponent_rating, opponent_rd, score) where score is 1 for correct, 0 for incorrect
        """
        mu, phi = cls.scale_down(judge.rating, judge.rd)
        sigma = judge.sigma

        if not outcomes:
            # Rating decay for inactivity
            phi = min(350 / 173.7178, (phi**2 + sigma**2)**0.5)
            judge.rating, judge.rd = cls.scale_up(mu, phi)
            return

        v = sum(
            (cls._g(phi_j)**2 * cls._E(mu, mu_j, phi_j) * (1 - cls._E(mu, mu_j, phi_j)))
            for mu_j, phi_j, _ in outcomes
        ) ** -1

        delta = v * sum(
            cls._g(phi_j) * (score - cls._E(mu, mu_j, phi_j))
            for mu_j, phi_j, score in outcomes
        )

        # Update volatility
        sigma_prime = cls._update_volatility(mu, phi, sigma, v, delta)

        phi_star = (phi**2 + sigma_prime**2)**0.5
        phi_prime = 1 / ((1 / phi_star**2) + (1 / v))
        mu_prime = mu + phi_prime * sum(
            cls._g(phi_j) * (score - cls._E(mu, mu_j, phi_j))
            for mu_j, phi_j, score in outcomes
        )

        judge.rating, judge.rd = cls.scale_up(mu_prime, phi_prime)
        judge.sigma = sigma_prime

    @staticmethod
    def _g(phi: float) -> float:
        return 1 / (1 + 3 * phi**2 / (3.14159265359**2))**0.5

    @staticmethod
    def _E(mu: float, mu_j: float, phi_j: float) -> float:
        return 1 / (1 + 10**(-Glicko2System._g(phi_j) * (mu - mu_j) / 2))

    @classmethod
    def _update_volatility(cls, mu: float, phi: float, sigma: float, v: float, delta: float) -> float:
        a = (sigma**2)
        epsilon = 0.000001

        def f(x):
            return ((3.14159265359**2 * x**2) / (2 * (phi**2 + v + x**2)**2)) - ((delta**2 - phi**2 - v - x**2) / (2 * (phi**2 + v + x**2)**2))

        # Simplified iteration
        return max(0.01, sigma * 0.95)  # Simplified for prototype

        verdict = judge.evaluate_case(player)
        weight = judge.vote_weight
        total_weight += weight
        if verdict == "Guilty":
            guilty_weight += weight

    guilty_ratio = (guilty_weight / total_weight) if total_weight > 0 else 0
    if guilty_ratio >= 0.65:
        player.status = "Shadow_Flagged"
        player.shadow_matches_left = 3
