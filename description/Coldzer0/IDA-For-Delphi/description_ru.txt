Этот проект — скрипт IDAPython, который помогает восстанавливать имена функций Delphi по шаблонам конструкторов событий при реверс-инжиниринге.
Он написан на Python для IDA Pro и поддерживает 64-битные цели.
Рабочий процесс ориентирован на загрузку скрипта в сеансе живой отладки, чтобы символы можно было разрешать из контекста выполнения.
Его в основном используют аналитики, работающие с бинарниками Delphi при анализе вредоносного ПО или реверсе программного обеспечения.
