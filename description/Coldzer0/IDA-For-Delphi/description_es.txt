Este proyecto es un script de IDAPython que ayuda a recuperar nombres de funciones de Delphi a partir de patrones de constructores de eventos durante la ingeniería inversa.
Está escrito en Python para IDA Pro e incluye soporte para objetivos de 64 bits.
El flujo de trabajo se centra en cargar el script en una sesión de depuración en vivo para que los símbolos se puedan resolver desde el contexto de ejecución.
Lo usan principalmente analistas que trabajan con binarios de Delphi en análisis de malware o tareas de ingeniería inversa de software.
