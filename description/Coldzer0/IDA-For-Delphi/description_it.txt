Questo progetto è uno script IDAPython che aiuta a recuperare i nomi delle funzioni Delphi dai pattern dei costruttori di eventi durante il reverse engineering.
È scritto in Python per IDA Pro e include il supporto per target a 64 bit.
Il flusso di lavoro è incentrato sul caricamento dello script in una sessione di debugging live, così da risolvere i simboli dal contesto di runtime.
È usato principalmente da analisti che lavorano su binari Delphi nell'analisi di malware o nel reverse engineering del software.
