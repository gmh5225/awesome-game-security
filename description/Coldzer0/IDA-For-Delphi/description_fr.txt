Ce projet est un script IDAPython qui aide à récupérer les noms de fonctions Delphi à partir des motifs de constructeurs d'événements lors du reverse engineering.
Il est écrit en Python pour IDA Pro et prend en charge les cibles 64 bits.
Le flux de travail consiste à charger le script dans une session de débogage en direct afin de résoudre les symboles à partir du contexte d'exécution.
Il est principalement utilisé par les analystes travaillant sur des binaires Delphi dans l'analyse de malwares ou le reverse engineering logiciel.
