Este proyecto añade soporte de scripting Lua a entornos C# como Unity, .NET y Mono.
Proporciona interoperabilidad bidireccional entre Lua y C#, capacidades de hotfix en tiempo de ejecución y opciones orientadas al rendimiento como código de binding generado y optimizaciones de GC.
El repositorio incluye componentes C#, Lua y runtime nativo para múltiples plataformas usadas por clientes de juegos.
Se usa principalmente por equipos de juegos que quieren scripting flexible y actualizaciones de lógica en vivo sin builds completos frecuentes.
