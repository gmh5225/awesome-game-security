Este proyecto es un framework de hotfix lógico para Unity que permite a los equipos parchear código de gameplay en C# sin reconstruir todo el cliente.
Apunta a una amplia cobertura de versiones y plataformas de Unity, y está diseñado para funcionar incluso con proyectos antiguos que no pueden refactorizarse en profundidad.
El código base es principalmente C# y usa herramientas IL para inyectar y enrutar la lógica parcheada en tiempo de ejecución, con documentación y proyectos Unity de ejemplo incluidos.
Su caso de uso principal es la corrección rápida de bugs y las actualizaciones en caliente controladas para juegos Unity en producción.
