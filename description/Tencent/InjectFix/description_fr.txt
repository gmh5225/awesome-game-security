Ce projet est un framework de hotfix logique pour Unity qui permet aux équipes de corriger le code de gameplay C# sans reconstruire tout le client.
Il vise une large couverture des versions et plateformes Unity, et est conçu pour fonctionner même avec d'anciens projets qui ne peuvent pas être fortement refactorisés.
Le code est principalement en C# et s'appuie sur des outils IL pour injecter et router la logique corrigée à l'exécution, avec documentation et projets Unity d'exemple inclus.
Son cas d'usage principal est la correction rapide de bugs et les mises à jour à chaud contrôlées pour les jeux Unity en production.
