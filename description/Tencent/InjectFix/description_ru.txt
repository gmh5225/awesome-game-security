Этот проект — фреймворк логических хотфиксов для Unity, позволяющий командам патчить C#-код геймплея без полной пересборки клиента.
Он рассчитан на широкую поддержку версий и платформ Unity и спроектирован так, чтобы работать даже со старыми проектами, которые нельзя сильно рефакторить.
Кодовая база в основном на C# и использует IL-инструменты для внедрения и маршрутизации исправленной логики во время выполнения; в комплект входят документация и примеры проектов Unity.
Основной сценарий использования — быстрое исправление ошибок и контролируемые горячие обновления для живых Unity-игр.
