Un server Model Context Protocol (MCP) che espone le operazioni sui Jupyter notebook come strumenti MCP, consentendo agli assistenti IA di creare, leggere, modificare ed eseguire Jupyter notebook in modo programmatico.
Supporta più modalità di trasporto (stdio, streamable HTTP) e può connettersi a server Jupyter locali, JupyterHub o ambienti Google Colab.
È soprattutto utile per i flussi di lavoro di analisi dei dati assistiti da IA e per integrare il calcolo basato su notebook nelle pipeline di tool-use degli LLM.
