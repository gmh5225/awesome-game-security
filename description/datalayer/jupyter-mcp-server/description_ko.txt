Jupyter notebook 작업을 MCP 도구로 노출하는 Model Context Protocol(MCP) 서버로, AI 어시스턴트가 Jupyter notebook을 프로그래밍 방식으로 생성, 읽기, 편집, 실행할 수 있게 합니다.
stdio, streamable HTTP 등 여러 전송 모드를 지원하며, 로컬 Jupyter 서버, JupyterHub 또는 Google Colab 환경에 연결할 수 있습니다.
주로 AI 지원 데이터 분석 워크플로와 notebook 기반 연산을 LLM 도구 사용 파이프라인에 통합하는 데 유용합니다.
