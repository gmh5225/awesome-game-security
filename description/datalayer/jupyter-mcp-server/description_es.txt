Un servidor Model Context Protocol (MCP) que expone operaciones de Jupyter notebook como herramientas MCP, permitiendo a los asistentes de IA crear, leer, editar y ejecutar Jupyter notebooks de forma programática.
Admite varios modos de transporte (stdio, streamable HTTP) y puede conectarse a servidores Jupyter locales, JupyterHub o entornos de Google Colab.
Es especialmente útil para flujos de trabajo de análisis de datos asistidos por IA y para integrar el cómputo basado en notebooks en pipelines de uso de herramientas de LLM.
