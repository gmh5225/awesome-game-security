Сервер Model Context Protocol (MCP), который предоставляет операции Jupyter notebook в виде MCP-инструментов, позволяя ИИ-ассистентам программно создавать, читать, редактировать и выполнять Jupyter notebooks.
Поддерживает несколько режимов транспорта (stdio, streamable HTTP) и может подключаться к локальным серверам Jupyter, JupyterHub или средам Google Colab.
В основном полезен для рабочих процессов анализа данных с помощью ИИ и для интеграции вычислений на основе notebook в конвейеры использования инструментов LLM.
