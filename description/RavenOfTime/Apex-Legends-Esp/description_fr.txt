Ce projet est un modèle de source de cheat ESP externe pour les jeux de tir, illustré avec Apex Legends.
Il contient un pilote noyau, une superposition en mode utilisateur et des composants de chargement basés sur kdmapper, avec un rendu via Windows GDI.
Le code est écrit en C++ et organisé en solution Visual Studio pour Windows.
Il s’adresse surtout aux apprenants du game hacking qui étudient les pipelines de lecture mémoire et la visualisation d’entités à l’écran.
