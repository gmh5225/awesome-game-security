Этот проект — шаблон исходников внешнего ESP-чита для шутеров, продемонстрированный на примере Apex Legends.
В нём есть отдельный драйвер ядра, пользовательский оверлей и компоненты загрузки на базе kdmapper; отрисовка выполняется через Windows GDI.
Код написан на C++ и организован как решение Visual Studio для Windows.
В основном его используют изучающие game hacking, которым нужны пайплайны чтения памяти и визуализация сущностей на экране.
