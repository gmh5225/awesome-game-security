Screenshot-Detection-Bypass — это proof of concept на C++, который перехватывает API BitBlt в gdi32, чтобы возвращать чистые снимки вместо изменённого кадра игры.
Проект показывает, как конвейеры скриншотов античита можно перехватить на уровне графического API Windows, с простым классовым хуком и структурой настроек.
Реализация сделана с упором на читаемость, чтобы было понятно, как вызов идёт от исходной функции к обработчику хука.
Основной сценарий — образовательное исследование античита вокруг обхода детекта скриншотов и поведения хуков захвата.
