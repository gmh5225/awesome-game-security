Este proyecto es un framework de protección PE de Windows diseñado para dificultar el crackeo y el análisis de ejecutables. Está implementado principalmente en C++ y ofrece mutación de instrucciones, una canalización de ensamblador x86 integrada, lógica de stub configurable y compatibilidad con compresión binaria. El repositorio también incluye componentes de compilador, pruebas y herramientas de compilación basadas en CMake para un desarrollo reproducible. Su caso de uso principal es la investigación en protección de software, la experimentación anti-manipulación y la comprensión de técnicas de defensa al estilo packer.
