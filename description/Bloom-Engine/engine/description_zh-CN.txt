Bloom 是一款 TypeScript 游戏引擎，通过基于 LLVM 的 TypeScript 提前编译器 Perry，将游戏编译为原生二进制与 Web 目标。它提供简洁、受 raylib 启发的函数式 API，面向 2D 与 3D 游戏；引擎核心用 Rust 实现，并通过 wgpu 在 Metal、DirectX 12、Vulkan、OpenGL 与 WebGPU 上渲染。技术栈集成了 Jolt Physics、面向 glTF/GLB 模型的 GPU 骨骼动画、WGSL 着色器，以及以 npm 包形式提供的 TypeScript 接口，使开发者可用同一套代码面向桌面、移动端与 WASM 发布。它面向希望获得原生性能与跨平台部署能力、同时用 TypeScript 而非 C++ 或重度编辑器驱动引擎编写玩法的游戏开发者。
