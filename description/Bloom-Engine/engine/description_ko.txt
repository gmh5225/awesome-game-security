Bloom은 TypeScript 게임 엔진으로, LLVM 기반 TypeScript 선행 컴파일러 Perry를 통해 게임을 네이티브 바이너리와 웹으로 컴파일합니다. raylib에서 영감을 받은 간단한 함수 기반 API를 2D 및 3D 게임에 제공하며, 엔진 코어는 Rust로 구현되고 wgpu를 통해 Metal, DirectX 12, Vulkan, OpenGL, WebGPU에서 렌더링됩니다. 스택에는 Jolt Physics, glTF/GLB 모델용 GPU 스켈레탈 애니메이션, WGSL 셰이더, npm 패키지로 제공되는 TypeScript 인터페이스가 포함되어 개발자가 하나의 코드베이스로 데스크톱, 모바일, WASM 타깃에 배포할 수 있습니다. C++나 무거운 에디터 중심 엔진 대신 TypeScript로 게임플레이를 작성하면서 네이티브 성능과 크로스 플랫폼 배포를 원하는 게임 개발자를 대상으로 합니다.
