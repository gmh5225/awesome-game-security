Bloom ist eine TypeScript-Spiele-Engine, die Spiele über Perry — einen LLVM-basierten Ahead-of-Time-TypeScript-Compiler — zu nativen Binärdateien und für das Web kompiliert. Sie stellt eine einfache, von raylib inspirierte, funktionsbasierte API für 2D- und 3D-Spiele bereit; der Engine-Kern ist in Rust umgesetzt und rendert über wgpu auf Metal, DirectX 12, Vulkan, OpenGL und WebGPU. Der Stack bündelt Jolt Physics, GPU-Skelettanimation für glTF/GLB-Modelle, WGSL-Shader und eine als npm-Paket bereitgestellte TypeScript-Oberfläche, sodass Entwickler eine Codebasis schreiben und für Desktop, Mobilgeräte und WASM-Ziele ausliefern können. Sie richtet sich an Spieleentwickler, die native Performance und plattformübergreifendes Deployment wollen und Gameplay in TypeScript statt in C++ oder mit einer schweren, editorzentrierten Engine erstellen möchten.
