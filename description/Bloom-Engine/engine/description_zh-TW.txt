Bloom 是一款 TypeScript 遊戲引擎，透過基於 LLVM 的 TypeScript 提前編譯器 Perry，將遊戲編譯為原生二進位與 Web 目標。它提供簡潔、受 raylib 啟發的函數式 API，面向 2D 與 3D 遊戲；引擎核心以 Rust 實作，並透過 wgpu 在 Metal、DirectX 12、Vulkan、OpenGL 與 WebGPU 上渲染。技術堆疊整合了 Jolt Physics、面向 glTF/GLB 模型的 GPU 骨骼動畫、WGSL 著色器，以及以 npm 套件形式提供的 TypeScript 介面，讓開發者能以同一套程式碼面向桌面、行動裝置與 WASM 發布。它面向希望獲得原生效能與跨平台部署能力、同時以 TypeScript 而非 C++ 或重度編輯器導向引擎撰寫玩法的遊戲開發者。
