Bloom è un motore di gioco in TypeScript che compila i giochi in binari nativi e per il web tramite Perry, un compilatore TypeScript ahead-of-time basato su LLVM. Espone una API semplice, ispirata a raylib e basata su funzioni, per giochi 2D e 3D; il core del motore è implementato in Rust e il rendering avviene tramite wgpu su Metal, DirectX 12, Vulkan, OpenGL e WebGPU. Lo stack integra Jolt Physics, animazione scheletrica GPU per modelli glTF/GLB, shader WGSL e una superficie TypeScript distribuita come pacchetto npm, così gli sviluppatori possono scrivere un’unica codebase e pubblicare su desktop, mobile e target WASM. È rivolto agli sviluppatori di giochi che vogliono prestazioni native e distribuzione multipiattaforma autorando il gameplay in TypeScript anziché in C++ o con un motore pesante incentrato sull’editor.
