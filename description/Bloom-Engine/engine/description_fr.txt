Bloom est un moteur de jeu TypeScript qui compile les jeux en binaires natifs et pour le Web via Perry, un compilateur TypeScript anticipé (AOT) basé sur LLVM. Il expose une API simple, inspirée de raylib et fondée sur des fonctions, pour les jeux 2D et 3D ; le cœur du moteur est implémenté en Rust et le rendu passe par wgpu sur Metal, DirectX 12, Vulkan, OpenGL et WebGPU. La pile intègre Jolt Physics, l’animation squelettique GPU pour les modèles glTF/GLB, des shaders WGSL et une surface TypeScript empaquetée via npm, afin que les développeurs écrivent une seule base de code et déploient sur bureau, mobile et cibles WASM. Il s’adresse aux développeurs de jeux qui veulent des performances natives et un déploiement multiplateforme tout en écrivant le gameplay en TypeScript plutôt qu’en C++ ou avec un moteur centré sur un éditeur lourd.
