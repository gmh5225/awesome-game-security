Bloom は TypeScript 製のゲームエンジンで、LLVM ベースの TypeScript 先行コンパイル（AOT）コンパイラ Perry を通じて、ゲームをネイティブバイナリと Web 向けにコンパイルします。raylib に着想を得たシンプルな関数ベース API を 2D／3D ゲーム向けに提供し、エンジンコアは Rust で実装され、wgpu 経由で Metal、DirectX 12、Vulkan、OpenGL、WebGPU 上に描画します。スタックには Jolt Physics、glTF／GLB モデル向けの GPU スケルタルアニメーション、WGSL シェーダー、npm パッケージ化された TypeScript 表面が含まれ、開発者は一つのコードベースでデスクトップ、モバイル、WASM ターゲットへ出荷できます。C++ や重量級のエディタ中心エンジンではなく TypeScript でゲームプレイを書きつつ、ネイティブ性能とクロスプラットフォーム展開を求めるゲーム開発者向けです。
