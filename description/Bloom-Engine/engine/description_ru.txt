Bloom — игровой движок на TypeScript, который компилирует игры в нативные бинарники и для веба через Perry, LLVM-компилятор TypeScript с опережающей (AOT) компиляцией. Он предоставляет простой функциональный API в духе raylib для 2D и 3D игр; ядро движка реализовано на Rust, а отрисовка идёт через wgpu на Metal, DirectX 12, Vulkan, OpenGL и WebGPU. Стек включает Jolt Physics, GPU-скелетную анимацию для моделей glTF/GLB, шейдеры WGSL и TypeScript-интерфейс в виде npm-пакета, чтобы разработчики писали одну кодовую базу и выпускали сборки для десктопа, мобильных платформ и WASM. Он рассчитан на разработчиков игр, которым нужна нативная производительность и кроссплатформенная поставка при написании геймплея на TypeScript, а не на C++ или в тяжёлом редактороцентричном движке.
