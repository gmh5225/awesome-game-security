Ce projet est une étude par rétro-ingénierie d’une implémentation d’anti-cheat pour jeu mobile et de sa logique de protection native. Il inclut du code C/C++ reconstruit pour des composants tels que les contrôles CPU et émulateur, le traitement lié à Dex, des routines utilitaires et le comportement de chargement chiffré. Le code et les notes se concentrent sur la façon dont les chemins de détection sont déclenchés et sur la structure concrète des routines anti-altération internes. Il est surtout utile aux chercheurs en sécurité des jeux qui souhaitent comprendre les internals de l’anti-cheat Android et évaluer d’éventuelles surfaces de contournement dans un cadre de recherche contrôlé.
