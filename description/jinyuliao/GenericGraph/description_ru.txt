Этот проект посвящён плагину универсальной графовой структуры данных для UE4.
Он в основном написан на C/C++ и C++ и сосредоточен на конвейерах ассетов, инструментах редактора и разработке плагинов.
Он особенно полезен разработчикам игр, программистам движков и исследователям в области графики, работающим в сфере game engine / game engine plugins:unreal.
