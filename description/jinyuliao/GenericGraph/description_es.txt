Este proyecto se centra en un plugin de estructura de datos de grafo genérico para UE4.
Está escrito principalmente en C/C++ y C++, y se enfoca en pipelines de assets, herramientas del editor y desarrollo de plugins.
Resulta especialmente útil para desarrolladores de juegos, programadores de motor e investigadores gráficos que trabajan en el área game engine / game engine plugins:unreal.
