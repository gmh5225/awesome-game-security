ida-sigmaker is an IDA Pro 9.0+ signature maker plugin that creates and searches unique byte patterns for reverse engineering. Written mainly in Python with optional Cython/SIMD speedups (AVX2, NEON, SSE2), it runs cross-platform on Windows, macOS, and Linux with no required third-party dependencies. It can generate the shortest unique signature for a function or selection, wildcard address-bearing operands, fall back to XREF-based signatures, and batch-search multiple patterns with text, CSV, or JSON results. A seed-then-refine uniqueness algorithm plus an optional native SIMD scanner keep large-database searches practical. The primary audience is reverse engineers and game-security practitioners who need durable code signatures inside IDA for analysis, tooling, or anti-cheat related workflows.
