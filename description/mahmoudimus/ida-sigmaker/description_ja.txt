ida-sigmaker は、リバースエンジニアリング向けの一意なバイトパターンを作成・検索する IDA Pro 9.0+ 用シグネチャ作成プラグインです。主に Python で書かれ、任意の Cython/SIMD 高速化（AVX2、NEON、SSE2）に対応し、Windows、macOS、Linux で動作し、必須のサードパーティ依存はありません。関数や選択範囲に対する最短の一意シグネチャ生成、アドレスを含むオペランドのワイルドカード化、XREF ベースへのフォールバック、テキスト／CSV／JSON での複数パターン一括検索が可能です。seed-then-refine による一意性アルゴリズムと任意のネイティブ SIMD スキャナにより、大規模データベースでも実用的な検索を保てます。主な対象は、解析、ツール開発、チート対策関連のワークフローのために IDA 内で耐久性のあるコードシグネチャを必要とするリバースエンジニアとゲームセキュリティ実務者です。
