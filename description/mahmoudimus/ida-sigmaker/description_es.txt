ida-sigmaker es un complemento generador de firmas para IDA Pro 9.0+ que crea y busca patrones de bytes únicos para ingeniería inversa. Escrito principalmente en Python con aceleraciones opcionales Cython/SIMD (AVX2, NEON, SSE2), funciona de forma multiplataforma en Windows, macOS y Linux sin dependencias de terceros obligatorias. Puede generar la firma única más corta para una función o selección, aplicar comodines a operandos con direcciones, recurrir a firmas basadas en XREF y buscar por lotes varios patrones con resultados en texto, CSV o JSON. Un algoritmo de unicidad seed-then-refine y un escáner SIMD nativo opcional mantienen prácticas las búsquedas en bases de datos grandes. El público principal son reverse engineers y profesionales de seguridad en juegos que necesitan firmas de código duraderas dentro de IDA para análisis, herramientas o flujos relacionados con anti-cheat.
