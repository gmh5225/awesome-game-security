ida-sigmaker 是一款面向 IDA Pro 9.0+ 的签名生成插件，可创建并搜索用于逆向工程的唯一字节特征码。主体以 Python 编写，可选 Cython/SIMD 加速（AVX2、NEON、SSE2），跨平台支持 Windows、macOS 与 Linux，且无必需的第三方依赖。它能为函数或选区生成最短唯一签名，对含地址操作数进行通配，在必要时回退到基于 XREF 的签名，并支持以文本、CSV 或 JSON 批量搜索多组特征。种子再精炼的唯一性算法，配合可选的原生 SIMD 扫描器，使大型数据库搜索仍保持实用。主要面向需要在 IDA 中构建稳定代码签名以用于分析、工具开发或反作弊相关流程的逆向工程师与游戏安全从业者。
