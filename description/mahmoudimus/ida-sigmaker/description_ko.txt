ida-sigmaker는 리버스 엔지니어링을 위한 고유 바이트 패턴을 생성하고 검색하는 IDA Pro 9.0+ 시그니처 메이커 플러그인입니다. 주로 Python으로 작성되었으며 선택적 Cython/SIMD 가속(AVX2, NEON, SSE2)을 지원하고, Windows, macOS, Linux에서 동작하며 필수 서드파티 의존성이 없습니다. 함수나 선택 영역에 대해 最短 고유 시그니처를 생성하고, 주소를 담은 피연산자를 와일드카드 처리하며, 필요 시 XREF 기반 시그니처로 대체하고, 텍스트·CSV·JSON 결과로 여러 패턴을 일괄 검색할 수 있습니다. seed-then-refine 고유성 알고리즘과 선택적 네이티브 SIMD 스캐너로 대규모 데이터베이스 검색도 실용적으로 유지합니다. 주요 대상은 분석, 툴링, 안티치트 관련 워크플로를 위해 IDA 안에서 견고한 코드 시그니처가 필요한 리버스 엔지니어와 게임 보안 실무자입니다.
