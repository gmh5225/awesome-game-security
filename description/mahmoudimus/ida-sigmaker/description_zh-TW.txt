ida-sigmaker 是一款適用於 IDA Pro 9.0+ 的簽章產生外掛，可建立並搜尋用於逆向工程的唯一位元組特徵碼。主體以 Python 撰寫，可選 Cython/SIMD 加速（AVX2、NEON、SSE2），跨平台支援 Windows、macOS 與 Linux，且無需必要的第三方相依套件。它能為函式或選取範圍產生最短唯一簽章，對帶位址的運算元進行萬用字元處理，必要時回退至以 XREF 為基礎的簽章，並支援以文字、CSV 或 JSON 批次搜尋多組特徵。種子再精煉的唯一性演算法，搭配可選的原生 SIMD 掃描器，使大型資料庫搜尋仍具實用性。主要對象是需要在 IDA 內建立穩定程式碼簽章，以用於分析、工具開發或反作弊相關流程的逆向工程師與遊戲資安從業人員。
