ida-sigmaker — плагин для создания сигнатур в IDA Pro 9.0+, который формирует и ищет уникальные байтовые шаблоны для реверс-инжиниринга. Написан в основном на Python с опциональными ускорениями Cython/SIMD (AVX2, NEON, SSE2), работает кроссплатформенно на Windows, macOS и Linux без обязательных сторонних зависимостей. Он может генерировать кратчайшую уникальную сигнатуру для функции или выделения, подставлять маски в операнды с адресами, при необходимости переходить к сигнатурам на основе XREF и пакетно искать несколько шаблонов с результатами в виде текста, CSV или JSON. Алгоритм уникальности seed-then-refine и опциональный нативный SIMD-сканер делают поиск по большим базам практичным. Основная аудитория — реверс-инженеры и специалисты по игровой безопасности, которым нужны устойчивые сигнатуры кода в IDA для анализа, инструментов или рабочих процессов, связанных с античитом.
