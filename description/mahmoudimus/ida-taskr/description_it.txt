Questo progetto è una libreria Python pura per il calcolo parallelo legato a IDA Pro.
Consente di usare la potenza di Qt (integrato in IDA!) e del multiprocessing di Python per scaricare le attività computazionalmente intensive su processi worker senza bloccare l'interfaccia di IDA Pro.
È soprattutto utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin IDA.
