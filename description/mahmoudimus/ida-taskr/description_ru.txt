Этот проект — чистая Python-библиотека для параллельных вычислений в контексте IDA Pro.
Она позволяет использовать возможности Qt (встроен в IDA!) и multiprocessing в Python, чтобы выносить ресурсоёмкие задачи в рабочие процессы без зависания интерфейса IDA Pro.
Библиотека особенно полезна исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / плагинов IDA.
