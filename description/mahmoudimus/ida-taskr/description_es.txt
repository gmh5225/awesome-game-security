Este proyecto es una biblioteca de Python puro para computación paralela relacionada con IDA Pro.
Permite usar la potencia de Qt (¡integrado en IDA!) y el multiprocessing de Python para descargar tareas intensivas en cálculo a procesos trabajadores sin congelar la interfaz de IDA Pro.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
