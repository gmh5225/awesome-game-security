Este proyecto es una herramienta de omisión de Windows Driver Signature Enforcement que cambia temporalmente la fecha del sistema a un momento anterior al periodo de revocación de certificados, firma un controlador con certificados filtrados y luego restaura la fecha. Incluye un componente de controlador en modo kernel que usa BlackBone para la carga y reubicación PE, un controlador en modo de usuario y archivos de certificados VeriSign. Resulta especialmente útil para investigadores de seguridad del kernel que estudian técnicas de omisión de DSE y el abuso de certificados de firma de controladores.
