Este proyecto es una prueba de concepto bring-your-own-vulnerable-driver para la terminación de procesos. Usa un ejecutable de Windows y un controlador de terceros vulnerable para realizar acciones privilegiadas contra un PID objetivo. El repositorio enmarca la técnica en el contexto de patrones de abuso reales y hace referencia a su entrada CVE asignada. Es relevante principalmente para la investigación de seguridad del kernel de Windows, el análisis de herramientas ofensivas y el endurecimiento de anti-cheat o defensa de endpoints.
