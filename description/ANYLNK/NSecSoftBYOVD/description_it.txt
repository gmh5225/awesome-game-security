Questo progetto è una proof of concept bring-your-own-vulnerable-driver per la terminazione di processi. Usa un eseguibile Windows e un driver di terze parti vulnerabile per eseguire azioni privilegiate contro un PID di destinazione. Il repository colloca la tecnica nel contesto di pattern di abuso reali e fa riferimento alla voce CVE assegnata. È rilevante soprattutto per la ricerca sulla sicurezza del kernel Windows, l’analisi di tooling offensivo e l’hardening di anti-cheat o difesa endpoint.
