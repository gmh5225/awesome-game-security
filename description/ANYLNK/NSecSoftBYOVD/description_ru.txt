Этот проект — proof of concept техники bring-your-own-vulnerable-driver для завершения процессов. Он использует исполняемый файл Windows и уязвимый сторонний драйвер для выполнения привилегированных действий в отношении целевого PID. Репозиторий рассматривает технику в контексте реальных сценариев злоупотребления и ссылается на присвоенную запись CVE. Материал в первую очередь полезен для исследований безопасности ядра Windows, анализа атакующих инструментов и усиления античита или защиты конечных точек.
