rootkit-detection-ebpf-time-trace는 커널 실행 경로의 타이밍 이상을 분석하여 루트킷을 탐지하는 Linux 연구 프레임워크입니다. eBPF 프로브를 사용해 getdents 흐름의 함수에서 세분화된 타이밍 데이터를 수집하며, 여기에는 파일 은닉형 루트킷이 조작하는 경로도 포함됩니다. 저장소에는 실험 오케스트레이션, 데이터셋 처리, 준지도 통계 이상 탐지 및 평가 출력을 위한 Python 도구가 포함되어 있습니다. 주요 대상은 행위 기반 루트킷 탐지와 안티스텔스 텔레메트리 기법을 연구하는 커널 보안 연구자입니다.
