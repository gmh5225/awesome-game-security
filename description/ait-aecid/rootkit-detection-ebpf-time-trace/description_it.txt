rootkit-detection-ebpf-time-trace è un framework di ricerca per Linux per rilevare rootkit analizzando anomalie temporali nei percorsi di esecuzione del kernel. Usa sonde eBPF per raccogliere dati di timing a grana fine dalle funzioni nel flusso getdents, inclusi i percorsi manipolati dai rootkit che nascondono file. Il repository include strumenti Python per l’orchestrazione degli esperimenti, la gestione dei dataset e il rilevamento statistico di anomalie semi-supervisionato con output di valutazione. Il pubblico principale sono i ricercatori di sicurezza del kernel che studiano il rilevamento di rootkit basato sul comportamento e le tecniche di telemetria anti-stealth.
