rootkit-detection-ebpf-time-trace は、カーネル実行パスにおけるタイミング異常を分析してルートキットを検出するための Linux 研究フレームワークです。eBPF プローブを用い、getdents フロー内の関数から細粒度のタイミングデータを収集します。これにはファイル隠蔽型ルートキットが操作するパスも含まれます。リポジトリには、実験のオーケストレーション、データセット処理、半教師あり統計的異常検知と評価出力向けの Python ツールが含まれます。主な想定利用者は、振る舞いベースのルートキット検出やアンチステルス・テレメトリ技術を研究するカーネルセキュリティ研究者です。
