rootkit-detection-ebpf-time-trace — исследовательский фреймворк для Linux, предназначенный для обнаружения руткитов путём анализа временных аномалий в путях выполнения ядра. Он использует зонды eBPF для сбора мелкозернистых данных о времени выполнения функций в потоке getdents, включая пути, которыми манипулируют руткиты, скрывающие файлы. Репозиторий включает инструменты на Python для оркестрации экспериментов, работы с наборами данных и полуконтролируемого статистического обнаружения аномалий с результатами оценки. Основная аудитория — исследователи безопасности ядра, изучающие поведенческое обнаружение руткитов и методы антистелс-телеметрии.
