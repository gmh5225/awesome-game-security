rootkit-detection-ebpf-time-trace es un marco de investigación para Linux orientado a detectar rootkits analizando anomalías temporales en las rutas de ejecución del kernel. Usa sondas eBPF para recopilar datos de temporización de grano fino de funciones en el flujo getdents, incluidas las rutas manipuladas por rootkits que ocultan archivos. El repositorio incluye herramientas en Python para la orquestación de experimentos, el manejo de conjuntos de datos y la detección estadística de anomalías semisupervisada con salidas de evaluación. Su audiencia principal son investigadores de seguridad del kernel que estudian la detección de rootkits basada en comportamiento y técnicas de telemetría antiocultamiento.
