rootkit-detection-ebpf-time-trace ist ein Linux-Forschungsframework zur Erkennung von Rootkits durch die Analyse von Timing-Anomalien in Kernel-Ausführungspfaden. Es nutzt eBPF-Sonden, um fein granulare Timing-Daten von Funktionen im getdents-Ablauf zu erfassen, einschließlich Pfaden, die von dateiverbergenden Rootkits manipuliert werden. Das Repository enthält Python-Werkzeuge für die Orchestrierung von Experimenten, die Handhabung von Datensätzen sowie die halbüberwachte statistische Anomalieerkennung mit Evaluierungsausgaben. Die primäre Zielgruppe sind Kernel-Sicherheitsforscher, die verhaltensbasierte Rootkit-Erkennung und Anti-Stealth-Telemetrietechniken untersuchen.
