Este proyecto es una referencia compacta de offsets y firmas de Fortnite con fragmentos de C++ de apoyo. Contiene código de ejemplo de pattern scanning, notas de offsets para globals clave y lógica auxiliar para la derivación de direcciones y flujos de descifrado de cadenas. El enfoque está en el mantenimiento rápido de offsets y la reubicación basada en firmas tras las actualizaciones del juego. Está pensado para ingenieros inversos y desarrolladores de cheats que mantienen herramientas externas o internas.
