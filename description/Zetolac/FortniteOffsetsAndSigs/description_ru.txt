Этот проект — компактный справочник по оффсетам и сигнатурам Fortnite с вспомогательными фрагментами кода на C++. В нём есть примеры кода pattern scanning, заметки по оффсетам ключевых глобальных переменных и вспомогательная логика для вычисления адресов и сценариев расшифровки строк. Акцент сделан на быстром сопровождении оффсетов и переносе по сигнатурам после обновлений игры. Материал предназначен для реверс-инженеров и разработчиков читов, поддерживающих внешние или внутренние инструменты.
