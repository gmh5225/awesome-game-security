Este proyecto es una prueba de concepto en C++ para cálculos externos world-to-screen en Fortnite. Incluye código para leer y descifrar datos de cámara/punto de vista desde la memoria del proceso y convertir posiciones 3D del mundo en coordenadas de pantalla 2D. La implementación demuestra la construcción de matrices, la transformación de ejes y la matemática de proyección en perspectiva usada por los overlays ESP. Resulta útil principalmente como base para herramientas de visualización externas en la investigación de game hacking.
