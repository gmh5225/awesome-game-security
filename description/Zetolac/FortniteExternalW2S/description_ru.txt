Этот проект — proof of concept на C++ для внешних расчётов world-to-screen в Fortnite. Он включает код для чтения и расшифровки данных камеры/точки обзора из памяти процесса и преобразования 3D-позиций мира в 2D-координаты экрана. Реализация демонстрирует построение матриц, преобразование осей и математику перспективной проекции, используемую ESP-оверлеями. В основном полезен как основа для внешних инструментов визуализации в исследованиях по game hacking.
