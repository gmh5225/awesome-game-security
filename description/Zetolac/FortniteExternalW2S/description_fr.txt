Ce projet est une preuve de concept en C++ pour les calculs externes world-to-screen dans Fortnite. Il comprend du code pour lire et déchiffrer les données de caméra/point de vue depuis la mémoire du processus, puis convertir les positions 3D du monde en coordonnées d'écran 2D. L'implémentation démontre la construction de matrices, la transformation des axes et les mathématiques de projection perspective utilisées par les overlays ESP. Il sert principalement de base pour des outils de visualisation externes dans la recherche en game hacking.
