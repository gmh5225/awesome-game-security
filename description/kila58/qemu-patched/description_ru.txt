Этот проект — пропатченный форк QEMU с антидетект-модификациями, скрывающими виртуальную машину от техник обнаружения гостевой ОС. Он изменяет ответы CPUID, данные SMBIOS/DMI, таблицы ACPI, имена устройств и другие отпечатки ВМ, чтобы ВМ QEMU выглядели как физическое оборудование для анти-ВМ проверок вредоносных программ и античитов. Предназначен для аналитиков malware и исследователей игровой безопасности, которым нужно запускать в QEMU ПО с обнаружением ВМ без срабатывания детекта.
