Este proyecto es un fork de QEMU parcheado con modificaciones anti-detección para ocultar la máquina virtual de las técnicas de detección del sistema operativo invitado. Modifica las respuestas CPUID, los datos SMBIOS/DMI, las tablas ACPI, los nombres de dispositivos y otras huellas de VM para que las VM de QEMU parezcan hardware físico ante comprobaciones anti-VM usadas por malware y sistemas anti-trampas. Está dirigido a analistas de malware e investigadores de seguridad en juegos que necesitan ejecutar en QEMU software que detecta VMs sin activar la detección.
