Questo progetto è un fork di QEMU patchato con modifiche anti-rilevamento per nascondere la macchina virtuale alle tecniche di rilevamento del sistema operativo guest. Modifica le risposte CPUID, i dati SMBIOS/DMI, le tabelle ACPI, i nomi dei dispositivi e altre impronte della VM affinché le VM QEMU appaiano come hardware fisico ai controlli anti-VM usati da malware e sistemi anti-cheat. È rivolto ad analisti di malware e ricercatori di game security che devono eseguire in QEMU software in grado di rilevare le VM senza attivare il rilevamento.
