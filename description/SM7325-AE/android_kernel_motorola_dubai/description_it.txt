Questo progetto è il codice sorgente del kernel Android per i dispositivi Motorola Dubai basati sulla piattaforma SM7325 (Snapdragon 778G). Include definizioni ABI GKI, driver specifici Qualcomm e configurazioni del device tree Motorola. È principalmente utile per sviluppatori del kernel Android e ricercatori di sicurezza mobile che lavorano alla personalizzazione e all'analisi del kernel di dispositivi Qualcomm Snapdragon 778G.
