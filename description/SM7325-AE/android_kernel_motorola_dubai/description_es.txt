Este proyecto es el código fuente del kernel de Android para dispositivos Motorola Dubai basados en la plataforma SM7325 (Snapdragon 778G). Incluye definiciones de ABI GKI, controladores específicos de Qualcomm y configuraciones del device tree de Motorola. Resulta especialmente útil para desarrolladores del kernel de Android e investigadores de seguridad móvil que trabajan en la personalización y el análisis del kernel de dispositivos Qualcomm Snapdragon 778G.
