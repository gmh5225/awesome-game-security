Этот проект — исходный код ядра Android для устройств Motorola Dubai на платформе SM7325 (Snapdragon 778G). Он включает определения GKI ABI, драйверы Qualcomm и конфигурации device tree Motorola. В первую очередь полезен разработчикам ядра Android и исследователям мобильной безопасности, работающим с кастомизацией и анализом ядра устройств Qualcomm Snapdragon 778G.
