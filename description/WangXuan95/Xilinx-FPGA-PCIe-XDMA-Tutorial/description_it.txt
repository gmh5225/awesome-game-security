Questo progetto è un tutorial pratico per costruire sistemi di comunicazione PCIe su FPGA Xilinx usando il core IP XDMA.
Fornisce esempi passo passo per progetti di lettura e scrittura BRAM, integrazione AXI, software C lato Linux e un flusso di lavoro più ampio di accelerazione MPEG2.
Il materiale combina progetti Vivado, logica Verilog e AXI e indicazioni sugli strumenti host, incluso il caricamento dei driver e le procedure di test.
È rivolto a professionisti di FPGA e sicurezza hardware, inclusi scenari di ricerca sulla game security orientati al DMA.
