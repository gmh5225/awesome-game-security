Este proyecto es un tutorial práctico para construir sistemas de comunicación PCIe en FPGAs Xilinx mediante el núcleo IP XDMA.
Ofrece ejemplos paso a paso para diseños de lectura y escritura BRAM, integración AXI, software en C del lado de Linux y un flujo de trabajo de aceleración MPEG2 de mayor alcance.
El material combina proyectos Vivado, lógica Verilog y AXI, y orientación sobre herramientas del host, incluida la carga de controladores y procedimientos de prueba.
Está dirigido a profesionales de FPGA y seguridad de hardware, incluidos escenarios de investigación en seguridad de juegos orientados a DMA.
