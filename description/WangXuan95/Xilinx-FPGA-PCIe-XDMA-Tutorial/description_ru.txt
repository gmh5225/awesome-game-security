Этот проект — практическое руководство по построению систем PCIe-связи на FPGA Xilinx с использованием IP-ядра XDMA.
В нём даны пошаговые примеры конструкций чтения и записи BRAM, интеграции AXI, ПО на C для Linux, а также более крупного рабочего процесса ускорения MPEG2.
Материалы сочетают проекты Vivado, логику Verilog и AXI, а также рекомендации по инструментам на стороне хоста, включая загрузку драйверов и процедуры тестирования.
Оно рассчитано на специалистов по FPGA и аппаратной безопасности, в том числе на сценарии исследований игровой безопасности, ориентированные на DMA.
