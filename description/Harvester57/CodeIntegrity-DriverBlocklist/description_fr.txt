CodeIntegrity-DriverBlocklist est un dépôt de données de listes de blocage de stratégies Windows Code Integrity pour des pilotes vulnérables ou susceptibles d’abus. Il fournit de grands fichiers de stratégie XML avec des règles de refus basées sur des hachages et des identités de pilotes, y compris des modules noyau pertinents pour l’anti-cheat. Le contenu est des données de configuration plutôt que du code source exécutable, et il est destiné à être consommé par WDAC ou des outils de stratégie associés. Son cas d’usage principal est le durcissement défensif, la réduction de la surface d’attaque noyau et la protection des environnements anti-cheat.
