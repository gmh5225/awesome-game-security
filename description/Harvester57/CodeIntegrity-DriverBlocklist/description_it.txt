CodeIntegrity-DriverBlocklist è un repository di dati con blocklist di criteri Windows Code Integrity per driver vulnerabili o soggetti ad abuso. Fornisce grandi file di criteri XML con regole di deny basate su hash e identità dei driver, inclusi moduli kernel rilevanti per l’anti-cheat. Il contenuto è costituito da dati di configurazione piuttosto che da codice sorgente eseguibile, ed è pensato per essere consumato da WDAC o da strumenti di policy correlati. Il caso d’uso principale è l’hardening difensivo, la riduzione della superficie di attacco del kernel e la protezione degli ambienti anti-cheat.
