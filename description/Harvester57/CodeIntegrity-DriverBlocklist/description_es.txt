CodeIntegrity-DriverBlocklist es un repositorio de datos de listas de bloqueo de políticas de Windows Code Integrity para controladores vulnerables o propensos al abuso. Proporciona grandes archivos de política XML con reglas de denegación basadas en hashes e identidades de controladores, incluidos módulos de kernel relevantes para el anti-cheat. El contenido es datos de configuración más que código fuente ejecutable, y está pensado para ser consumido por WDAC o herramientas de política relacionadas. Su caso de uso principal es el endurecimiento defensivo, la reducción de la superficie de ataque del kernel y la protección de entornos anti-cheat.
