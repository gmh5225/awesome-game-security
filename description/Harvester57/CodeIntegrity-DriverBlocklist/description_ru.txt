CodeIntegrity-DriverBlocklist — это репозиторий данных со списками блокировки политик Windows Code Integrity для уязвимых или подверженных злоупотреблению драйверов. Он предоставляет крупные XML-файлы политик с правилами запрета на основе хешей и идентификаторов драйверов, включая модули ядра, релевантные для античита. Содержимое представляет собой конфигурационные данные, а не исполняемый исходный код, и предназначено для использования WDAC или связанными средствами политик. Основной сценарий применения — оборонительное усиление защиты, сокращение поверхности атаки ядра и защита сред античита.
