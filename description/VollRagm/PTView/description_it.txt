Questo progetto è un'utilità Windows per l'ispezione in tempo reale delle tabelle delle pagine di processo tramite un driver e un client grafico. Combina componenti lato kernel in C++ con un'interfaccia C# per scorrere le voci, eseguire il dump delle pagine fisiche e ispezionare i metadati PTE. Lo strumento supporta anche i flussi di lavoro di traduzione da virtuale a fisico e la consapevolezza delle large page per un'analisi più approfondita del mapping della memoria. Il caso d'uso principale è la ricerca sulla memoria OS a basso livello, il debugging e le indagini kernel orientate all'anti-cheat.
