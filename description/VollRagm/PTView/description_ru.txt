Этот проект — утилита для Windows, предназначенная для живой инспекции таблиц страниц процессов через драйвер и графический клиент. Она сочетает компоненты на стороне ядра на C++ с интерфейсом на C# для просмотра записей, дампа физических страниц и анализа метаданных PTE. Инструмент также поддерживает сценарии преобразования виртуальных адресов в физические и учёт больших страниц для более глубокого анализа отображения памяти. Основное применение — низкоуровневое исследование памяти ОС, отладка и ориентированные на античит исследования ядра.
