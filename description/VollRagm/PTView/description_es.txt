Este proyecto es una utilidad de Windows para la inspección en vivo de las tablas de páginas de procesos mediante un controlador y un cliente gráfico. Combina componentes del lado del kernel en C++ con una interfaz en C# para explorar entradas, volcar páginas físicas e inspeccionar metadatos de PTE. La herramienta también admite flujos de trabajo de traducción de virtual a físico y reconocimiento de páginas grandes para un análisis más profundo del mapeo de memoria. Su caso de uso principal es la investigación de memoria del SO a bajo nivel, la depuración y las investigaciones de kernel orientadas al anti-cheat.
