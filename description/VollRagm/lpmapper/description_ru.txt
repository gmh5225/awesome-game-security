Этот проект — Windows kernel mapper, который размещает shellcode в уже загруженных large-page драйверах без нового выделения памяти. Он реализован на C++ и следует рабочему процессу, вдохновлённому известными manual mapper'ами, при этом требуя специальной настройки реестра для large-page драйверов. Техника представлена как способ снизить видимость для распространённых путей обнаружения kernel anti-cheat. Основной сценарий использования — углублённые исследования безопасности ядра и эксперименты по обходу anti-cheat.
