Этот проект — внутренний чит для Rainbow Six Siege на C++, который внедряется в процесс игры и предоставляет ESP, аимбот и информацию об игроках. Он читает данные игровых сущностей через прямой доступ к памяти с использованием структур SDK, полученных реверс-инжинирингом, и рисует оверлеи через собственный конвейер рендеринга игры. Кодовая база демонстрирует типичные для R6S паттерны FPS-читов на базе Unreal Engine. Предназначен для исследователей игровой безопасности, изучающих реализации читов для Rainbow Six Siege и векторы обнаружения античита.
