Ce projet est un cheat interne pour Rainbow Six Siege écrit en C++ qui s'accroche au processus du jeu pour fournir des fonctionnalités d'ESP, d'aimbot et d'informations sur les joueurs. Il lit les données des entités du jeu par accès mémoire direct à l'aide de structures SDK reverse-engineerées, et affiche des overlays via le pipeline de rendu du jeu lui-même. La base de code illustre des schémas de cheats FPS basés sur Unreal Engine spécifiques à R6S. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les implémentations de cheats pour Rainbow Six Siege et les vecteurs de détection anti-cheat.
