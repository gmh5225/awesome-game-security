Этот проект — расширение Ghidra, которое генерирует скрипты хуков Frida непосредственно из контекста дизассемблирования. Оно реализовано на Java и добавляет действия по правому клику для полной генерации скриптов, повторно используемых фрагментов и расширенной настройки хуков. Плагин поддерживает как хуки на уровне функций, так и по произвольным адресам, ускоряя подготовку динамического анализа. Он полезен реверс-инженерам и исследователям безопасности, связывающим статический анализ в Ghidra с рантайм-инструментированием во Frida.
