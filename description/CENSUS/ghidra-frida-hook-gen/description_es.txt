Este proyecto es una extensión de Ghidra que genera scripts de hook de Frida directamente desde el contexto de desensamblado. Está implementado en Java y añade acciones de clic derecho para la generación completa de scripts, fragmentos reutilizables y configuración avanzada de hooks. El complemento admite hooks a nivel de función y en direcciones arbitrarias para acelerar la configuración del análisis dinámico. Es útil para ingenieros inversos e investigadores de seguridad que conectan el análisis estático en Ghidra con la instrumentación en tiempo de ejecución en Frida.
