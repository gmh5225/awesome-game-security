Questo progetto è un esempio minimo di cattura screenshot su Windows con DirectX 9. Crea un dispositivo Direct3D9, acquisisce il front buffer con GetFrontBufferData e copia i pixel del frame in buffer di memoria di sistema. Usa le API Windows Imaging Component per codificare e salvare i frame catturati come file PNG. È utile per prototipi di strumenti grafici e per comprendere le tecniche di cattura frame basate su DirectX nella ricerca sui giochi.
