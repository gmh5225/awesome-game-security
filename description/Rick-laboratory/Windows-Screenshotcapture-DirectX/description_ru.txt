Этот проект — минимальный пример захвата скриншотов в Windows на DirectX 9. Он создаёт устройство Direct3D9, получает передний буфер через GetFrontBufferData и копирует пиксели кадра в буферы системной памяти. Для кодирования и сохранения захваченных кадров в PNG используются API Windows Imaging Component. Пример полезен для прототипов графических инструментов и для понимания техник захвата кадров на базе DirectX в исследованиях игр.
