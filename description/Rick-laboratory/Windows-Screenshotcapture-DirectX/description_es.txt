Este proyecto es un ejemplo mínimo de captura de pantallas en Windows con DirectX 9. Crea un dispositivo Direct3D9, obtiene el búfer frontal con GetFrontBufferData y copia los píxeles del fotograma a búferes de memoria del sistema. Usa las API de Windows Imaging Component para codificar y guardar los fotogramas capturados como archivos PNG. Es útil para prototipos de herramientas gráficas y para comprender técnicas de captura de fotogramas basadas en DirectX en la investigación de juegos.
