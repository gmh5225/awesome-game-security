Ce projet est un exemple minimal de capture d'écran Windows avec DirectX 9. Il crée un périphérique Direct3D9, récupère le tampon frontal avec GetFrontBufferData, puis copie les pixels de la trame dans des tampons en mémoire système. Il utilise les API Windows Imaging Component pour encoder et enregistrer les trames capturées en fichiers PNG. Il est utile pour prototyper des outils graphiques et pour comprendre les techniques de capture de trames basées sur DirectX dans la recherche sur les jeux.
