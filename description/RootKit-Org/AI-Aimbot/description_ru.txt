Этот проект — ИИ-aimbot, который с помощью моделей машинного обучения обнаруживает и наводится на цели в играх через анализ захвата экрана. Он поддерживает пользовательские YOLO-модели для разных игр, включая Fortnite и Rust, использует Conda для управления окружением и включает настройку чувствительности обнаружения и параметров прицеливания. В основном полезен исследователям игровой безопасности, изучающим задачи обнаружения ИИ-читов и подходы машинного обучения к автоматизированному взаимодействию с играми.
