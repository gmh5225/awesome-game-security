Destiny es un emulador de servidor C# de código abierto para MapleStory v0.83. Ofrece una arquitectura multiseridor con componentes center, login y game, además de soporte de scripting en Lua para el comportamiento de NPCs y portales. El código está organizado como una solución de Visual Studio y se basa en datos de juego respaldados por SQL para gestionar cuentas y el estado del mundo. Su uso principal es el desarrollo de servidores privados y la investigación de protocolos o jugabilidad de MMORPG.
