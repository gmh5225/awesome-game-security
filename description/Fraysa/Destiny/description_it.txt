Destiny è un emulatore di server C# open source per MapleStory v0.83. Fornisce un’architettura multi-server con componenti center, login e game, oltre al supporto di scripting Lua per il comportamento di NPC e portali. Il codice è organizzato come soluzione Visual Studio e si basa su dati di gioco gestiti tramite SQL per account e stato del mondo. Il caso d’uso principale è lo sviluppo di server privati e la ricerca su protocolli o gameplay degli MMORPG.
