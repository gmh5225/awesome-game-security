Destiny est un émulateur de serveur C# open source pour MapleStory v0.83. Il propose une architecture multi-serveurs avec des composants center, login et game, ainsi qu’un support de scripts Lua pour le comportement des PNJ et des portails. La base de code est organisée en solution Visual Studio et s’appuie sur des données de jeu stockées en SQL pour la gestion des comptes et de l’état du monde. Son usage principal est le développement de serveurs privés et la recherche sur les protocoles ou le gameplay des MMORPG.
