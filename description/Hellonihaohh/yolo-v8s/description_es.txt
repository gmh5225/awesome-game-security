Este repositorio es un paquete de distribución ligero para un artefacto de modelo YOLOv8s, no una base de código de implementación completa. El contenido archivado incluye solo información de licencia y un archivo de modelo comprimido, sin README ni scripts de entrenamiento o inferencia. Su función principal es proporcionar recursos de detección preentrenados para otros proyectos, incluidos experimentos de visión por computadora orientados a juegos.
