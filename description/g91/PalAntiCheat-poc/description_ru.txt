Концептуальная (proof-of-concept) система античита для Palworld (Unreal Engine 5) с полным дампом SDK, включающим определения классов, сигнатуры функций и раскладки структур для модулей NPC, игрока, анимации, аудио и ИИ. Реализует процедуры исправления свойств (PropertyFixup.hpp) для проверки согласованности UObject/UProperty, закладывая основу для обнаружения манипуляций с памятью состояния игры через проверки целостности на уровне SDK.
