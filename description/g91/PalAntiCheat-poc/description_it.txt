Sistema anti-cheat proof-of-concept per Palworld (Unreal Engine 5) che include un SDK completo dumpato con definizioni di classi, firme di funzioni e layout delle strutture per i moduli NPC, giocatore, animazione, audio e IA del gioco. Implementa routine di property fixup (PropertyFixup.hpp) per validare la coerenza di UObject/UProperty, fornendo una base per rilevare la manipolazione in memoria dello stato di gioco tramite controlli di integrità a livello SDK.
