Sistema anticheat de prueba de concepto para Palworld (Unreal Engine 5) que incluye un SDK completo volcado con definiciones de clases, firmas de funciones y diseños de estructuras para los módulos de NPC, jugador, animación, audio e IA del juego. Implementa rutinas de corrección de propiedades (PropertyFixup.hpp) para validar la consistencia de UObject/UProperty, proporcionando una base para detectar manipulación de memoria del estado del juego mediante comprobaciones de integridad a nivel de SDK.
