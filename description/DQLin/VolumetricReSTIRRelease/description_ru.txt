Этот проект — исследовательская реализация пространственно-временно́го резервуарного ресемплинга для быстрого объёмного рендеринга. Он интегрирует пользовательский объёмный проход рендеринга в рендерер на C++ и Slang на базе Falcor и включает скрипты и конвейеры сцен для воспроизведения результатов статьи. Код ориентирован на качество и производительность в реальном времени на современном оборудовании класса RTX. Предназначен в основном для исследователей компьютерной графики и разработчиков движков, изучающих продвинутые методы рендеринга с практическими применениями в реальном времени.
