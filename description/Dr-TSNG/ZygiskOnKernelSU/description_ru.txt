Этот проект представляет собой автономную реализацию среды выполнения Zygisk для рутированных Android-окружений. Он обеспечивает совместимость с Zygisk API для KernelSU и может заменить встроенный режим Zygisk в Magisk. В README описаны требования к платформе для KernelSU, Magisk и APatch, а также примечания по поведению и совместимости. В основном его используют опытные пользователи, занимающиеся моддингом и безопасностью Android, которым нужна поддержка модулей на базе Zygisk вне стандартного стека Magisk.
