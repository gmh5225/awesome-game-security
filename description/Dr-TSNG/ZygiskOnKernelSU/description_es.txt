Este proyecto es una implementación independiente del runtime Zygisk para entornos Android con root. Proporciona compatibilidad con la API Zygisk para KernelSU y puede reemplazar el modo Zygisk integrado de Magisk. El README documenta los requisitos de plataforma en KernelSU, Magisk y APatch, junto con notas sobre el comportamiento y la compatibilidad. Lo utilizan principalmente usuarios avanzados de modding y seguridad en Android que necesitan soporte de módulos basados en Zygisk fuera de la pila Magisk predeterminada.
