Este proyecto es una biblioteca de detección para Android y una aplicación de demostración para identificar entornos de software sospechosos como Magisk y Xposed. Está construido con Kotlin y componentes nativos en C++, combinando inspección de paquetes, comprobaciones de artefactos del sistema de archivos, sondeo de archivos basado en syscalls y detección del estado de Xposed. El código también incluye comprobaciones de anomalías en perfiles duales o de trabajo y rastros conocidos relacionados con root. Está destinado principalmente a la investigación sobre anti-trampas móviles, detección de root e integridad de aplicaciones.
