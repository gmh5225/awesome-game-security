Questo progetto è un proxy DLL che intercetta le funzioni standard dell’API di memoria di Windows e le reindirizza a un dispositivo remoto via DMA usando il framework PCILeech/MemProcFS. Include un livello di hooking basato su MinHook per operazioni su processi, moduli, thread e memoria, una libreria di memoria DMA con accesso a input e registro, e un’applicazione loader del proxy. È soprattutto utile per i ricercatori di sicurezza DMA che studiano l’accesso remoto alla memoria trasparente a livello API e le tecniche di interazione con i giochi tramite proxy DMA.
