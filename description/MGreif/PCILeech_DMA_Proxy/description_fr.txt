Ce projet est un proxy DLL qui intercepte les fonctions standard de l’API mémoire Windows et les redirige vers un appareil distant via DMA à l’aide du framework PCILeech/MemProcFS. Il comprend une couche de hooking basée sur MinHook pour les opérations sur les processus, modules, threads et la mémoire, une bibliothèque mémoire DMA avec accès aux entrées et au registre, ainsi qu’une application chargeur de proxy. Il s’adresse surtout aux chercheurs en sécurité DMA qui étudient l’accès mémoire distant transparent au niveau de l’API et les techniques d’interaction de jeu via un proxy DMA.
