Este proyecto es un proxy DLL que intercepta las funciones estándar de la API de memoria de Windows y las redirige a un dispositivo remoto mediante DMA usando el framework PCILeech/MemProcFS. Incluye una capa de hooking basada en MinHook para operaciones de proceso, módulo, hilo y memoria, una biblioteca de memoria DMA con acceso a entrada y al registro, y una aplicación cargadora del proxy. Es especialmente útil para investigadores de seguridad DMA que estudian el acceso remoto a memoria transparente a la API y técnicas de interacción con juegos mediante proxy DMA.
