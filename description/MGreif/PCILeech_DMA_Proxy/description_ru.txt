Этот проект — DLL-прокси, который перехватывает стандартные функции Windows API для работы с памятью и перенаправляет их на удалённое устройство через DMA с помощью фреймворка PCILeech/MemProcFS. Включает слой хуков на базе MinHook для операций с процессами, модулями, потоками и памятью, библиотеку DMA-памяти с доступом к вводу и реестру, а также приложение-загрузчик прокси. В основном полезен исследователям безопасности DMA, изучающим API-прозрачный удалённый доступ к памяти и техники взаимодействия с играми через DMA-прокси.
