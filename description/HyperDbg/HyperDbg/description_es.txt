Este proyecto es un depurador de código abierto asistido por hipervisor para el análisis en modo usuario y modo kernel de Windows.
Está implementado principalmente en C y C++ y se apoya en funciones de virtualización por hardware como Intel VT-x y EPT.
Sus capacidades clave incluyen puntos de interrupción sigilosos, hooks ocultos, supervisión de acceso a memoria y flujos de depuración extensibles.
Está diseñado para ingeniería inversa, fuzzing, análisis de malware e investigación anti-cheat o anti-debug.
