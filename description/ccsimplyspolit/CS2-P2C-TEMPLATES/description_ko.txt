이 프로젝트는 Counter-Strike 2용 보안 연구 목적의 유료 치트(P2C) 템플릿을 제공하며, VMProtect로 보호된 anti-VAC 헬퍼와 관련 인젝터 도구를 일대일로 리버스 엔지니어링하여 이식한 것입니다. 핵심 산출물은 C/C++로 작성된 주입 DLL(VacLiveBypass)로, MinHook 디투어를 CreateMove, LevelInit 및 protobuf 직렬화 경로에 적용하여 네트워크로 전송되기 전에 입력 기록과 시야 각도 데이터를 변조합니다. 저장소에는 서버 플래그 및 랭크 스푸핑용 Windows 커널 드라이버, 다중 방식의 사용자 모드·커널 인젝터, 런타임 GitHub 오프셋 가져오기를 지원하는 depot 인식 오프셋 매니페스트, 그리고 VMProtect 메커니즘과 데모 분석 문서도 포함됩니다. 주요 언어는 CMake로 빌드하는 C와 C++이며, 데모 파싱용 Python 스크립트와 Lua 리버스 엔지니어링 노트도 제공합니다. VAC Live 및 CS2 클라이언트 내부를 대상으로 한 안티치트 연구, 리버스 엔지니어링 교육, 그리고 비보안 로컬 또는 CTF 환경에서의 버그 바운티 스타일 학습을 위해 고안되었습니다.
