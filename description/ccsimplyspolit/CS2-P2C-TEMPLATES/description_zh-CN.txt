本项目提供用于 Counter-Strike 2 安全研究的付费作弊（P2C）模板，作为对受 VMProtect 保护的 anti-VAC 辅助程序及相关注入工具的一对一逆向移植。核心产物是以 C/C++ 编写的注入式 DLL（VacLiveBypass），通过 MinHook 对 CreateMove、LevelInit 以及 protobuf 序列化路径进行 detour，在数据发送到网络之前修改输入历史与视角数据。仓库还包含用于服务器标志与段位伪造的 Windows 内核驱动、多方式用户态与内核注入器、带运行时 GitHub 偏移拉取的 depot 感知偏移清单，以及关于 VMProtect 机制与 demo 分析的文档。主要语言为 C 与 C++（使用 CMake 构建），并附有用于 demo 解析的 Python 脚本与 Lua 逆向工程笔记。项目面向反作弊研究、逆向工程教育，以及对 VAC Live 与 CS2 客户端内部机制在本地不安全环境或 CTF 场景下的漏洞赏金式研究。
