Questo progetto fornisce template pay-to-cheat (P2C) per la ricerca di sicurezza su Counter-Strike 2, realizzati come port reverse-engineered uno a uno di un helper anti-VAC protetto con VMProtect e relativi strumenti di injection. L'artefatto centrale è una DLL iniettata in C/C++ (VacLiveBypass) che usa detour MinHook su CreateMove, LevelInit e sui percorsi di serializzazione protobuf per modificare la cronologia degli input e i dati degli angoli di visuale prima dell'invio sulla rete. Il repository include anche driver kernel Windows per lo spoofing di flag server e rank, un injector multimodo in user-mode e kernel, manifest di offset depot-aware con recupero runtime degli offset da GitHub, e documentazione sui meccanismi di VMProtect e sull'analisi delle demo. I linguaggi principali sono C e C++ con build CMake, più script Python per il parsing delle demo e note di reverse engineering in Lua. È pensato per la ricerca anti-cheat, l'educazione al reverse engineering e lo studio in stile bug bounty di VAC Live e degli interni del client CS2 su setup locali non sicuri o CTF.
