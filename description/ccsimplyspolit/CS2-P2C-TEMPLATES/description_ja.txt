本プロジェクトは、Counter-Strike 2 向けのセキュリティ研究用ペイ・トゥ・チート（P2C）テンプレートを提供し、VMProtect で保護された anti-VAC ヘルパーおよび関連インジェクターツールを一対一でリバースエンジニアリング移植したものです。中核となる成果物は C/C++ 製の注入 DLL（VacLiveBypass）で、MinHook による CreateMove、LevelInit、および protobuf シリアライズ経路へのデツアーを用い、ワイヤ送信前に入力履歴と視点角度データを改変します。リポジトリにはサーバーフラグとランクスプーフィング用の Windows カーネルドライバ、複数方式のユーザーモード／カーネルインジェクター、実行時に GitHub からオフセットを取得する depot 対応オフセットマニフェスト、および VMProtect の仕組みとデモ解析に関するドキュメントも含まれます。主な言語は CMake ビルドの C／C++ で、デモ解析用の Python スクリプトと Lua リバースエンジニアリング用ノートも付属します。VAC Live および CS2 クライアント内部のアンチチート研究、リバースエンジニアリング教育、ならびにローカルの非セキュア環境や CTF セットアップにおけるバグバウンティ風の調査を想定しています。
