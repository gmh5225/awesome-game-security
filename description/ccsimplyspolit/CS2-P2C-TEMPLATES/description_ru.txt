Этот проект предоставляет шаблоны pay-to-cheat (P2C) для исследований безопасности Counter-Strike 2, выполненные как портирования один-к-одному с реверс-инжинирингом защищённого VMProtect anti-VAC хелпера и связанного инструментария инъекции. Основной артефакт — внедряемая DLL на C/C++ (VacLiveBypass), которая с помощью MinHook-обходов CreateMove, LevelInit и путей сериализации protobuf изменяет историю ввода и данные углов обзора до отправки по сети. Репозиторий также включает драйверы ядра Windows для спуфинга серверных флагов и ранга, многометодный инжектор в user-mode и ядре, depot-aware манифесты смещений с загрузкой смещений с GitHub во время выполнения, а также документацию по механизмам VMProtect и анализу демо. Основные языки — C и C++ со сборкой через CMake, плюс скрипты Python для разбора демо и заметки по реверс-инжинирингу на Lua. Предназначен для исследований античита, обучения реверс-инжинирингу и изучения VAC Live и внутренностей клиента CS2 в стиле bug bounty на незащищённых локальных или CTF-стендах.
