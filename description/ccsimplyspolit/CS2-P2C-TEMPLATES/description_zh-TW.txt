本專案提供用於 Counter-Strike 2 安全研究的付費作弊（P2C）模板，作為對受 VMProtect 保護的 anti-VAC 輔助程式及相關注入工具的一對一逆向移植。核心產物是以 C/C++ 撰寫的注入式 DLL（VacLiveBypass），透過 MinHook 對 CreateMove、LevelInit 以及 protobuf 序列化路徑進行 detour，在資料送上網路之前修改輸入歷史與視角資料。儲存庫還包含用於伺服器旗標與段位偽造的 Windows 核心驅動程式、多方式使用者模式與核心注入器、具備執行時期 GitHub 偏移擷取的 depot 感知偏移清單，以及關於 VMProtect 機制與 demo 分析的文件。主要語言為 C 與 C++（使用 CMake 建置），並附有用於 demo 解析的 Python 腳本與 Lua 逆向工程筆記。專案面向反作弊研究、逆向工程教育，以及對 VAC Live 與 CS2 用戶端內部機制在本地不安全環境或 CTF 場景下的漏洞賞金式研究。
