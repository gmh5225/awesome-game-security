Little Anti-Cheat è un plugin anti-cheat SourceMod open source per server multiplayer del motore Source.
Scritto in SourcePawn, include rilevamenti di comportamenti aimbot e aimlock, angoli anomali, automazione del bunnyhop, exploit di duck rapido e stati ConVar sospetti.
Fornisce anche funzionalità di mitigazione come contromisure di interpolazione e backtrack, applicazione opzionale di ping elevato e un ampio supporto per logging e traduzioni.
L'uso principale è l'amministrazione di server community per giochi come Team Fortress 2 e i titoli Counter-Strike.
