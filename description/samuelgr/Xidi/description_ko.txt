이 프로젝트는 Xidi로, XInput 및 DirectInput API 호출을 가로채 가상 컨트롤러 매핑과 입력 변환을 제공하는 Windows DLL입니다. 호환성 계층으로 동작하여 최신 XInput 컨트롤러를 레거시 DirectInput 게임에서 사용할 수 있게 하고, 그 반대도 가능하며, 구성 가능한 버튼 매핑, 데드존 조정, 포스 피드백 변환을 지원합니다. 이 C++ 라이브러리는 프록시 DLL로 로드됩니다. DirectInput과 XInput API 간의 컨트롤러 호환성 문제를 해결하려는 게임 플레이어와 개발자를 대상으로 합니다.
