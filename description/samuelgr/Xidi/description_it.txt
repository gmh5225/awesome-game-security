Questo progetto è Xidi, una DLL Windows che intercetta le chiamate alle API XInput e DirectInput per fornire mappatura di controller virtuali e traduzione degli input. Funziona come livello di compatibilità che consente ai controller XInput moderni di funzionare con i giochi DirectInput legacy e viceversa, con mappatura dei pulsanti configurabile, regolazione della zona morta e traduzione del force feedback. La libreria C++ si carica come DLL proxy. È rivolta a giocatori e sviluppatori che risolvono problemi di compatibilità dei controller tra le API DirectInput e XInput.
