Этот проект — Xidi, Windows DLL, которая перехватывает вызовы API XInput и DirectInput для виртуального сопоставления контроллеров и преобразования ввода. Она работает как слой совместимости: современные контроллеры XInput можно использовать в устаревших играх DirectInput и наоборот, с настраиваемым сопоставлением кнопок, регулировкой мёртвой зоны и преобразованием отдачи (force feedback). Библиотека на C++ загружается как прокси-DLL. Предназначена для игроков и разработчиков, решающих проблемы совместимости контроллеров между API DirectInput и XInput.
