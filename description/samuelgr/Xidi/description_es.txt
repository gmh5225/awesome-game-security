Este proyecto es Xidi, una DLL de Windows que intercepta las llamadas a las API XInput y DirectInput para ofrecer mapeo de mandos virtuales y traducción de entradas. Actúa como capa de compatibilidad que permite usar mandos XInput modernos con juegos DirectInput antiguos y viceversa, con mapeo de botones configurable, ajuste de zona muerta y traducción de force feedback. La biblioteca C++ se carga como una DLL proxy. Está dirigida a jugadores y desarrolladores que resuelven problemas de compatibilidad de mandos entre las API DirectInput y XInput.
