Реализация на Rust средства чтения/записи PDB (Program Database) с полной поддержкой контейнерного формата MSF, записей символов и типов CodeView, разбора потока DBI, потоков TPI/IPI и метаданных образа COFF. Включает crate codeview для кодирования/декодирования видов символов (S_PUB, S_PROC, S_LOCAL) и записей типов (LF_CLASS, LF_POINTER, LF_PROCEDURE) с архитектурно-зависимыми соответствиями регистров для x86, AMD64 и ARM64.
