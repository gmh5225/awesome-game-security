Dieses Projekt ist die Direct3D9-On-12-Mapping-Schicht.
D3D9On12 ist keine Implementierung der D3D9-API, sondern eine Implementierung der D3D9-Usermode-DDI (Device Driver Interface).
Es ist vor allem für Grafikprogrammierer und Entwickler von Windows-Spiel-Tooling im Bereich DirectX / Kompatibilität nützlich.
