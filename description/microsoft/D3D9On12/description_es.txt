Este proyecto es la capa de mapeo Direct3D9-On-12.
D3D9On12 no es una implementación de la API de D3D9, sino una implementación del DDI (device driver interface) en modo usuario de D3D9.
Es principalmente útil para programadores gráficos y desarrolladores de herramientas para juegos de Windows que trabajan en el área de DirectX / compatibilidad.
