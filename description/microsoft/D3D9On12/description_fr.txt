Ce projet est la couche de mapping Direct3D9-On-12.
D3D9On12 n'est pas une implémentation de l'API D3D9, mais une implémentation du DDI (device driver interface) usermode de D3D9.
Il est principalement utile aux programmeurs graphiques et aux développeurs d'outils pour jeux Windows travaillant dans le domaine DirectX / compatibilité.
