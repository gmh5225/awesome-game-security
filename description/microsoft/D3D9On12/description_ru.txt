Этот проект — слой сопоставления Direct3D9-On-12.
D3D9On12 — это не реализация API D3D9, а реализация пользовательского DDI (device driver interface) D3D9.
Он в основном полезен графическим программистам и разработчикам игровых инструментов для Windows, работающим в области DirectX / совместимости.
