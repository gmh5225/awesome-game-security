Composants open source de Windows Subsystem for Linux, notamment le frontal wsl.exe/wslconfig.exe, le service Lxss Manager, l’intégration du système de fichiers Plan 9 (DrvFS), la pile réseau GNS, l’orchestration init/systemd, et l’infrastructure de machine virtuelle légère permettant d’exécuter des binaires Linux ELF non modifiés sous Windows.
