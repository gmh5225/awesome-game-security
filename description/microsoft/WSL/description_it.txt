Componenti open source di Windows Subsystem for Linux, tra cui il front-end wsl.exe/wslconfig.exe, il servizio Lxss Manager, l’integrazione del filesystem Plan 9 (DrvFS), lo stack di rete GNS, l’orchestrazione init/systemd e l’infrastruttura di VM leggera per eseguire binari Linux ELF non modificati su Windows.
