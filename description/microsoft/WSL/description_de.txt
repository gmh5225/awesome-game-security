Open-Source-Komponenten von Windows Subsystem for Linux, darunter das wsl.exe/wslconfig.exe-Frontend, der Lxss-Manager-Dienst, die Plan-9-Dateisystemintegration (DrvFS), der GNS-Netzwerkstack, die init/systemd-Orchestrierung sowie die leichtgewichtige VM-Infrastruktur zum Ausführen unveränderter Linux-ELF-Binaries unter Windows.
