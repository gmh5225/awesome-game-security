Componentes de código abierto de Windows Subsystem for Linux, que incluyen el front-end wsl.exe/wslconfig.exe, el servicio Lxss Manager, la integración del sistema de archivos Plan 9 (DrvFS), la pila de red GNS, la orquestación init/systemd y la infraestructura de máquina virtual ligera para ejecutar binarios Linux ELF sin modificar en Windows.
