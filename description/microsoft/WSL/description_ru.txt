Открытые компоненты Windows Subsystem for Linux, включая фронтенд wsl.exe/wslconfig.exe, службу Lxss Manager, интеграцию файловой системы Plan 9 (DrvFS), сетевой стек GNS, оркестрацию init/systemd и инфраструктуру легковесной виртуальной машины для запуска неизменённых Linux ELF-бинарников в Windows.
