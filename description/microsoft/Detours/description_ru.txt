Этот проект — программный пакет для мониторинга и инструментирования вызовов API в Windows.
Detours используется многими независимыми поставщиками ПО (ISV), а также продуктовыми командами Microsoft.
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов и хуков.
