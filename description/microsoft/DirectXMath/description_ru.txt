Полностью inline SIMD-библиотека линейной алгебры на C++ от Microsoft с оптимизированными под SSE/SSE2/AVX/AVX2/ARM-NEON векторами, матрицами, кватернионами и примитивами обнаружения столкновений (BoundingBox, BoundingSphere, BoundingFrustum). Включает расширения для преобразования half-float F16C, FMA3/FMA4, сферических гармоник (SHMath) и DSP-функций (XDSP).
