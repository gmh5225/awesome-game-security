Biblioteca de álgebra lineal SIMD en C++ completamente inline de Microsoft que ofrece vectores, matrices, cuaterniones y primitivas de detección de colisiones (BoundingBox, BoundingSphere, BoundingFrustum) optimizadas para SSE/SSE2/AVX/AVX2/ARM-NEON. Incluye extensiones para conversión half-float F16C, FMA3/FMA4, armónicos esféricos (SHMath) y funciones DSP (XDSP).
