Libreria di algebra lineare SIMD C++ interamente inline di Microsoft che fornisce vettori, matrici, quaternioni e primitive di rilevamento collisioni (BoundingBox, BoundingSphere, BoundingFrustum) ottimizzate per SSE/SSE2/AVX/AVX2/ARM-NEON. Include estensioni per la conversione half-float F16C, FMA3/FMA4, armoniche sferiche (SHMath) e funzioni DSP (XDSP).
