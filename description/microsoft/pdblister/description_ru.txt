Консольная утилита на Rust, которая формирует манифесты загрузки PDB, сканируя каталоги на наличие PE-файлов и извлекая записи из их debug directory (CodeView GUID и age). Повторяет поведение symchk /om, но работает значительно быстрее, строя URL Microsoft Symbol Server для пакетной загрузки PDB в блокирующем и асинхронном режимах.
