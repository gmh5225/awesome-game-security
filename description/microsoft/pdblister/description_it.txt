Strumento a riga di comando in Rust che genera manifesti di download PDB scansionando directory alla ricerca di file PE ed estraendo le voci della directory di debug (GUID CodeView e age). Replica il comportamento di symchk /om ma è significativamente più veloce, costruendo URL del Microsoft Symbol Server per il recupero batch di PDB in modalità di download bloccante e asincrona.
