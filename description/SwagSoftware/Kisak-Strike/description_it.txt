Questo progetto è un port Linux open source di Counter-Strike: Global Offensive basato sul motore Source di Valve. Include l'intera codebase del gioco con supporto di build CMake, app framework, material system, sound system, networking, file di progetto VPC e adattamenti specifici per Linux, inclusi i percorsi di rendering SDL e OpenGL. È principalmente utile per ricercatori del motore Source e sviluppatori di giochi che studiano l'architettura del motore Source e il porting di giochi su Linux.
