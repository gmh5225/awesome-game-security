SubZero는 에임봇 및 관련 게임플레이 보조 기능을 갖춘 Fortnite 클라이언트 측 치트를 구현한 Windows C++ Visual Studio 프로젝트입니다. 커널 스타일 드라이버 인터페이스로 원격 메모리 읽기, 메시 기반 가시성 검사, 게임 오프셋을 사용해 화면상의 대상으로 마우스를 부드럽게 조준합니다. 코드베이스에는 ImGui DirectX 9 오버레이와 메뉴, NtUserSendInput 기반 마우스 주입 및 라이브러리 스푸핑, 콜 스택 스푸핑, XOR 문자열 난독화, 선택적 인증 헬퍼가 포함됩니다. 주로 외부 오버레이 치트, 커널 메모리 접근 패턴, Windows상의 회피 기법에 대한 게임 보안 및 안티치트 연구에 유용합니다.
