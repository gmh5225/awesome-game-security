SubZero は、エイムボットおよび関連するゲームプレイ支援を備えた Fortnite クライアント側チートを実装した Windows C++ Visual Studio プロジェクトです。カーネル風のドライバインタフェースによるリモートメモリ読み取り、メッシュベースの可視性チェック、ゲームオフセットを用いて、画面上のターゲットへマウス照準を滑らかに動かします。コードベースには ImGui DirectX 9 オーバーレイとメニュー、NtUserSendInput ベースのマウス注入とライブラリスプーフィング、コールスタックスプーフィング、XOR 文字列難読化、および任意の認証ヘルパーが含まれます。主に外部オーバーレイチート、カーネルメモリアクセスパターン、Windows 上の回避手法に関するゲームセキュリティおよびアンチチート研究向けです。
