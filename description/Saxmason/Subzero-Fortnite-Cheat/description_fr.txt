SubZero est un projet Windows C++ Visual Studio qui implémente un cheat côté client Fortnite avec aimbot et aides de gameplay associées. Il utilise une interface de pilote de type noyau pour les lectures mémoire distantes, des contrôles de visibilité basés sur les mesh, et des offsets de jeu pour orienter en douceur la souris vers les cibles à l'écran. Le code comprend une overlay et un menu ImGui DirectX 9, une injection souris basée sur NtUserSendInput avec spoofing de bibliothèque, du spoofing de call stack, une obfuscation de chaînes XOR, et des helpers d'authentification optionnels. Il sert surtout à la recherche en sécurité des jeux et anti-cheat sur les cheats à overlay externe, les motifs d'accès mémoire noyau et les techniques d'évasion sous Windows.
