SubZero — проект Windows C++ Visual Studio, реализующий клиентский чит для Fortnite с аимботом и связанными игровыми помощниками. Он использует интерфейс драйвера в стиле ядра для удалённого чтения памяти, проверки видимости на основе mesh и игровые оффсеты, чтобы плавно наводить мышь на цели на экране. В кодовой базе есть оверлей и меню ImGui DirectX 9, инъекция мыши через NtUserSendInput со спуфингом библиотек, спуфинг стека вызовов, XOR-обфускация строк и опциональные помощники аутентификации. Проект в первую очередь полезен для исследований игровой безопасности и античита: внешние оверлей-читы, паттерны доступа к памяти ядра и техники обхода на Windows.
