SubZero es un proyecto Windows C++ de Visual Studio que implementa un cheat del lado del cliente de Fortnite con aimbot y ayudas de jugabilidad relacionadas. Usa una interfaz de controlador de estilo kernel para lecturas remotas de memoria, comprobaciones de visibilidad basadas en mesh y offsets del juego para dirigir el ratón suavemente hacia objetivos en pantalla. El código incluye una overlay y menú ImGui DirectX 9, inyección de ratón basada en NtUserSendInput con spoofing de bibliotecas, spoofing de call stack, ofuscación de cadenas XOR y helpers de autenticación opcionales. Resulta útil principalmente para investigación de seguridad de juegos y anti-cheat sobre cheats de overlay externos, patrones de acceso a memoria del kernel y técnicas de evasión en Windows.
