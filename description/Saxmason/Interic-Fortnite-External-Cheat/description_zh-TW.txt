本專案是一款面向 Fortnite 的 Windows C++ 外部作弊工具，在遊戲進程外執行並透過疊加層提供遊戲輔助。它以 Visual Studio 解決方案形式建置，使用 Dear ImGui 與 DirectX 9 後端實作選單與繪製，並搭配 Unreal 風格數學輔助及遊戲偏移量定位玩家與武器。核心能力包括可設定的自瞄按鍵與命中框瞄準、基於可見性的 ESP 風格著色，以及與 FOV 相關的視覺效果，並由驅動輔助的記憶體存取提供支援。程式碼庫還整合了 KeyAuth 授權許可，以及反偵錯、反傾印與字串混淆等保護措施。此專案主要可作為遊戲安全與逆向工程研究外部 Fortnite 作弊技術的參考。
