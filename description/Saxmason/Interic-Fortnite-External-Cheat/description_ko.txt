이 프로젝트는 게임 프로세스 외부에서 실행되며 오버레이로 플레이 보조를 제공하는 Fortnite용 Windows C++ 외부 치트입니다. Visual Studio 솔루션으로 구축되며, 메뉴와 그리기를 위해 Dear ImGui와 DirectX 9 백엔드를 사용하고, Unreal 스타일 수학 헬퍼와 게임 오프셋으로 플레이어와 무기를 찾습니다. 핵심 기능에는 구성 가능한 에임봇 키바인드와 히트박스 타겟팅, 가시성 기반 ESP 스타일 색상 구분, FOV 관련 시각 효과가 포함되며, 드라이버 지원 메모리 접근에 기반합니다. 코드베이스에는 KeyAuth 라이선싱과 함께 안티디버그, 안티덤프, 문자열 난독화 보호도 통합되어 있습니다. 주로 게임 보안 및 외부 Fortnite 치트 기법의 리버스 엔지니어링 연구 참고 자료로 유용합니다.
