本プロジェクトは、ゲームプロセス外で動作し、オーバーレイでプレイ支援を行う Fortnite 向けの Windows C++ 外部チートです。Visual Studio ソリューションとして構築され、メニューと描画には Dear ImGui と DirectX 9 バックエンドを用い、Unreal 風の数学ヘルパーとゲームオフセットでプレイヤーや武器を特定します。主な機能として、設定可能なエイムボットのキーバインドとヒットボックス照準、可視性を考慮した ESP 風の着色、FOV 関連のビジュアルがあり、ドライバ支援によるメモリアクセスに支えられています。コードベースには KeyAuth によるライセンス認証に加え、アンチデバッグ、アンチダンプ、文字列難読化などの保護も組み込まれています。主にゲームセキュリティや、外部 Fortnite チート手法のリバースエンジニアリング研究の参考として有用です。
