This project is a Windows C++ external cheat for Fortnite that runs outside the game process and overlays gameplay assistance. It is built as a Visual Studio solution using Dear ImGui with a DirectX 9 backend for the menu and drawing, Unreal-style math helpers, and game offsets to locate players and weapons. Core capabilities include configurable aimbot keybinds and hitbox targeting, visibility-aware ESP-style coloring, and FOV-related visuals, backed by driver-assisted memory access. The codebase also integrates KeyAuth licensing plus anti-debug, anti-dump, and string obfuscation protections. It is primarily useful as a reference for game security and reverse-engineering study of external Fortnite cheat techniques.
