本项目是一款面向 Fortnite 的 Windows C++ 外部作弊工具，在游戏进程外运行并通过叠加层提供游戏辅助。它以 Visual Studio 解决方案形式构建，使用 Dear ImGui 与 DirectX 9 后端实现菜单与绘制，并配合 Unreal 风格数学辅助及游戏偏移量定位玩家与武器。核心能力包括可配置的自瞄按键与命中框瞄准、基于可见性的 ESP 风格着色，以及与 FOV 相关的视觉效果，并由驱动辅助的内存访问提供支持。代码库还集成了 KeyAuth 授权许可，以及反调试、反转储和字符串混淆等保护措施。该项目主要可作为游戏安全与逆向工程研究外部 Fortnite 作弊技术的参考。
