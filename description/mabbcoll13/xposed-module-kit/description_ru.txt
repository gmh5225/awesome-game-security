Xposed Module Kit — это готовый Android-каркас для создания модулей Xposed и LSPosed с переиспользуемыми шаблонами хуков и небольшими CLI-помощниками для поиска целей хуков. Написан в основном на Java (Android-приложение на Gradle под Xposed API 82–93) и включает утилиты на Python, в том числе сканер классов, который разбирает скомпилированные .class или JAR и выдаёт готовые заготовки MethodHook. В комплекте — помощники HookTemplate, MethodHook и PackageHook, удобный для logcat логгер и пример, нейтрализующий типичные проверки обнаружения root. Предназначен для исследований безопасности, реверс-инжиниринга и законного тестирования Android-приложений — включая анализ игр и античитов — на устройствах, которыми вы владеете или которые уполномочены тестировать.
