Xposed Module Kit is a batteries-included Android scaffold for building Xposed and LSPosed modules, with reusable hook templates and small CLI helpers for finding hook targets. It is written mainly in Java (Gradle Android app targeting Xposed API 82–93) and includes Python utilities such as a class scanner that parses compiled .class or JAR files and emits ready-to-use MethodHook stubs. The kit ships HookTemplate, MethodHook, and PackageHook helpers, a logcat-friendly logger, and an example that neutralizes common root-detection checks. It is intended for security research, reverse engineering, and lawful testing of Android apps—including game and anti-cheat related analysis—on devices you own or are authorized to test.
