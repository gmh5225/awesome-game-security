CiDllDemo — демонстрационный драйвер ядра, показывающий использование API ci.dll для проверки подписей исполняемых файлов. Он перехватывает уведомления о создании процессов, вызывает CiValidateFileObject и CiCheckSignedFile и анализирует возвращаемую политику для извлечения сведений о сертификате. Проект реализован на C/C++ для разработки драйверов Windows и поддерживает сборки x86 и x64. Предназначен для исследователей безопасности, изучающих поведение Windows Code Integrity в режиме ядра.
