CiDllDemo es una demostración de controlador de kernel que ilustra el uso de las API de ci.dll para validar firmas de ejecutables. Engancha notificaciones de creación de procesos, invoca CiValidateFileObject y CiCheckSignedFile, e inspecciona la información de directiva devuelta para extraer detalles del certificado. El proyecto está implementado en C/C++ para el desarrollo de controladores de Windows y admite compilaciones x86 y x64. Está dirigido a investigadores de seguridad que exploran el comportamiento de Windows Code Integrity en modo kernel.
