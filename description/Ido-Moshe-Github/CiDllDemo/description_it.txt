CiDllDemo è una dimostrazione di driver kernel sull'uso delle API di ci.dll per convalidare le firme degli eseguibili. Aggancia le notifiche di creazione dei processi, invoca CiValidateFileObject e CiCheckSignedFile e ispeziona le informazioni di policy restituite per estrarre i dettagli del certificato. Il progetto è implementato in C/C++ per lo sviluppo di driver Windows e supporta build x86 e x64. È rivolto ai ricercatori di sicurezza che esplorano il comportamento di Windows Code Integrity in modalità kernel.
