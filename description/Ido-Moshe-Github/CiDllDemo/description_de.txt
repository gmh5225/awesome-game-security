CiDllDemo ist eine Kernel-Treiber-Demonstration zur Verwendung der ci.dll-APIs zur Validierung von Signaturen ausführbarer Dateien. Es hookt Benachrichtigungen zur Prozesserstellung, ruft CiValidateFileObject und CiCheckSignedFile auf und prüft die zurückgegebenen Richtlinieninformationen, um Zertifikatdetails zu extrahieren. Das Projekt ist in C/C++ für die Windows-Treiberentwicklung umgesetzt und unterstützt sowohl x86- als auch x64-Builds. Es richtet sich an Sicherheitsforscher, die das Verhalten von Windows Code Integrity im Kernelmodus untersuchen.
