CiDllDemo est une démonstration de pilote noyau illustrant l'utilisation des API de ci.dll pour valider les signatures d'exécutables. Il intercepte les notifications de création de processus, appelle CiValidateFileObject et CiCheckSignedFile, puis examine les informations de stratégie renvoyées pour en extraire les détails de certificat. Le projet est implémenté en C/C++ pour le développement de pilotes Windows et prend en charge les builds x86 et x64. Il s'adresse aux chercheurs en sécurité qui explorent le comportement de Windows Code Integrity en mode noyau.
