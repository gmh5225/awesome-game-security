C-библиотека для вычисления Authenticode-дайджестов PE-файлов, реализующая алгоритм исключения при хешировании, который пропускает поле контрольной суммы PE и каталог безопасности. Использует API BCrypt (SHA-1/SHA-256/SHA-384/SHA-512), разбирает встроенные структуры WIN_CERTIFICATE и рассчитана как на режим ядра (ksecdd.lib), так и на пользовательский режим (bcrypt.lib).
