Biblioteca C para calcular digests Authenticode de archivos PE, que implementa el algoritmo de exclusión de hash que omite el campo de checksum PE y el directorio de seguridad durante el hashing. Usa las API BCrypt (SHA-1/SHA-256/SHA-384/SHA-512), analiza estructuras WIN_CERTIFICATE incrustadas y fue diseñada para entornos en modo kernel (ksecdd.lib) y en modo usuario (bcrypt.lib).
