Libreria C per calcolare i digest Authenticode dei file PE, che implementa l’algoritmo di esclusione dell’hash che salta il campo checksum PE e la directory di sicurezza durante l’hashing. Usa le API BCrypt (SHA-1/SHA-256/SHA-384/SHA-512), analizza le strutture WIN_CERTIFICATE incorporate ed è stata progettata per ambienti in modalità kernel (ksecdd.lib) e in modalità utente (bcrypt.lib).
