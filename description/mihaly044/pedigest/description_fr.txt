Bibliothèque C pour calculer les digests Authenticode des fichiers PE, implémentant l’algorithme d’exclusion de hachage qui ignore le champ checksum PE et le répertoire de sécurité lors du hachage. Utilise les API BCrypt (SHA-1/SHA-256/SHA-384/SHA-512), analyse les structures WIN_CERTIFICATE intégrées, et a été conçue pour les environnements en mode noyau (ksecdd.lib) et en mode utilisateur (bcrypt.lib).
