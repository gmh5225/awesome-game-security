Este proyecto demuestra el enganche silencioso de syscalls en ARM64 Linux parcheando la ruta de manejo SVC del kernel.
En lugar de modificar directamente las entradas de sys_call_table, redirige números de syscall seleccionados mediante lógica alternativa para reducir indicadores de detección habituales.
La implementación es C de bajo nivel, con empalme manual de funciones y parcheo estilo trampolín en torno a las rutas del manejador de excepciones.
Está orientado a investigadores de seguridad del kernel que analizan técnicas de rootkits sigilosos y mejoran métodos de detección de hooks de syscall.
