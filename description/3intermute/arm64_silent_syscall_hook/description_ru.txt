Этот проект демонстрирует скрытый перехват системных вызовов на ARM64 Linux путём патчинга пути обработки SVC в ядре.
Вместо прямого изменения записей sys_call_table он перенаправляет выбранные номера syscall через альтернативную логику, снижая типичные признаки обнаружения.
Реализация написана на низкоуровневом C с ручным splicing функций и trampoline-подобным патчингом вокруг путей обработчика исключений.
Предназначен для исследователей безопасности ядра, анализирующих техники скрытных rootkit и улучшающих методы обнаружения перехвата syscall.
