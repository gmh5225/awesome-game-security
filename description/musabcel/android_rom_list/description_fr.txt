Ce projet fournit une liste.
Il est organisé sous forme de documentation et de matériel de référence pour le domaine des cheats / ROM Android, plutôt que comme une base de code d'exécution autonome.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / ROM Android.
