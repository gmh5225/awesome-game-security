Este proyecto es una aplicación Android para descubrir, descargar, instalar y gestionar módulos Magisk desde repositorios en línea. Está construida con herramientas de Android y código de aplicación basado en Kotlin, con funciones como búsqueda en repositorios, comprobación de actualizaciones de módulos, compatibilidad con repositorios personalizados y vistas separadas de módulos locales y remotos. La aplicación también admite metadatos de módulos más completos y gestión de compatibilidad para desarrolladores de módulos y usuarios avanzados. Su caso de uso principal es la gestión de módulos en Android con root y los flujos de personalización de seguridad móvil.
