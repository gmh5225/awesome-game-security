Ce projet est une implémentation C++ simple d’un speed hack de type Cheat Engine. Il montre comment modifier le temps perçu par le jeu en accrochant les API concernées avec Microsoft Detours. Le code est compact et se concentre sur la logique de hooking de base plutôt que sur un framework de triche complet. Il s’adresse surtout aux apprenants qui étudient la manipulation du temps et les techniques de base du game hacking en mode utilisateur.
