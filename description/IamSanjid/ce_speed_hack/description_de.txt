Dieses Projekt ist eine einfache C++-Implementierung eines Speed Hacks im Stil von Cheat Engine. Es zeigt, wie sich die wahrgenommene Spielzeit ändern lässt, indem relevante APIs mit Microsoft Detours gehookt werden. Der Code ist kompakt und konzentriert sich auf die zentrale Hook-Logik statt auf ein vollständiges Cheat-Framework. Er eignet sich vor allem für Lernende, die Zeitmanipulation und grundlegende User-Mode-Techniken des Game Hackings studieren.
