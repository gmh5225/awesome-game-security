Questo progetto è una semplice implementazione in C++ di uno speed hack in stile Cheat Engine. Mostra come modificare il tempo percepito dal gioco agganciando le API rilevanti con Microsoft Detours. Il codice è compatto e si concentra sulla logica di hooking essenziale, non su un framework completo di cheat. È soprattutto utile per chi studia la manipolazione del tempo e le tecniche di base del game hacking in user mode.
