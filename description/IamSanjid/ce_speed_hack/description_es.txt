Este proyecto es una implementación sencilla en C++ de un speed hack al estilo de Cheat Engine. Demuestra cómo alterar el tiempo percibido del juego enganchando las API relevantes con Microsoft Detours. El código es compacto y se centra en la lógica esencial de hooking, no en un framework completo de trampas. Resulta útil sobre todo para quienes estudian la manipulación del tiempo y técnicas básicas de game hacking en modo usuario.
