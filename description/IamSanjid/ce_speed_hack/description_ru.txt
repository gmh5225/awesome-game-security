Этот проект — простая реализация на C++ speed hack в стиле Cheat Engine. Он показывает, как изменить воспринимаемое игровое время, перехватывая соответствующие API с помощью Microsoft Detours. Код компактный и сосредоточен на базовой логике хуков, а не на полноценном чит-фреймворке. В первую очередь полезен изучающим манипуляцию временем и базовые техники user-mode game hacking.
