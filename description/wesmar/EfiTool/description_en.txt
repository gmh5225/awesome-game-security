This project is a UEFI EFI application that patches the Windows SYSTEM registry hive in RAM at ExitBootServices, before the NT kernel takes over.
The demo sets SetupType and CmdLine so cmd.exe launches as NT AUTHORITY\SYSTEM on the next boot, with no disk writes and no kernel driver.
It documents ExitBootServices hooking, regf hive cell patching, anti-analysis string obfuscation, and BitLocker or PCR boundary implications for Windows 11 research.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / efi driver area.
