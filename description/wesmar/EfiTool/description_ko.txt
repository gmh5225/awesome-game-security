이 프로젝트는 ExitBootServices 시점에 RAM의 Windows SYSTEM 레지스트리 하이브를 패치하여 NT 커널이 제어권을 갖기 전에 동작하는 UEFI EFI 애플리케이션입니다.
데모는 SetupType과 CmdLine을 설정해 다음 부팅 시 cmd.exe가 NT AUTHORITY\SYSTEM으로 실행되도록 하며, 디스크 쓰기와 커널 드라이버가 필요 없습니다.
ExitBootServices 후킹, regf 하이브 셀 패치, 분석 방해용 문자열 난독화, 그리고 Windows 11 연구에서의 BitLocker 또는 PCR 경계 영향을 문서화합니다.
주로 치트 / EFI 드라이버 영역의 공격 기법을 연구하는 게임 보안 연구자와 리버스 엔지니어에게 유용합니다.
