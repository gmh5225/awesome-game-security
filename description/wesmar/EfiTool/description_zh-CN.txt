本项目是一个 UEFI EFI 应用程序，在 ExitBootServices 时于内存中修补 Windows SYSTEM 注册表配置单元，抢在 NT 内核接管之前完成。
演示通过设置 SetupType 和 CmdLine，使下次启动时以 NT AUTHORITY\SYSTEM 身份启动 cmd.exe，且不写磁盘、不加载内核驱动。
文档涵盖 ExitBootServices 挂钩、regf 配置单元 cell 修补、反分析字符串混淆，以及面向 Windows 11 研究的 BitLocker 或 PCR 边界影响。
主要面向游戏安全研究人员与逆向工程师，用于学习作弊 / EFI 驱动领域相关的进攻性技术。
