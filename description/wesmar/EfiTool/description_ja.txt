本プロジェクトは、ExitBootServices 時に RAM 上の Windows SYSTEM レジストリハイブをパッチし、NT カーネルが制御を握る前に処理する UEFI EFI アプリケーションです。
デモでは SetupType と CmdLine を設定し、次回ブート時に cmd.exe を NT AUTHORITY\SYSTEM として起動させます。ディスクへの書き込みやカーネルドライバは不要です。
ExitBootServices のフック、regf ハイブセルのパッチ、解析妨害向け文字列難読化、および Windows 11 研究における BitLocker や PCR 境界の影響について文書化しています。
主に、チート / EFI ドライバ領域の攻撃手法を学ぶゲームセキュリティ研究者やリバースエンジニア向けです。
