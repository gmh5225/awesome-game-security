CMDT 是一款精简的 Windows 实用工具，可在 TrustedInstaller 服务令牌下启动进程，从而在不获取所有权的情况下访问受 TrustedInstaller ACL 保护的资源。它完全用手工编写的原生 x86 与 x64 汇编实现，提供双模式 GUI 与 CLI 二进制，无 C 运行时或第三方依赖，体积分别约在 40 KiB 与 30 KiB 以内。主要能力包括复制 TrustedInstaller 进程令牌、启用大量令牌特权、UAC 自提升、Explorer 右键菜单安装、CLI I/O 重定向中继、可选 MRU 历史，以及相关的系统维护辅助功能。面向管理员、取证与逆向工程工作流，以及仅靠 SYSTEM 仍不足以修改受保护操作系统组件时的深度 Windows 修复或调试场景。
