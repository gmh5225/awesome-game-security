CMDT는 TrustedInstaller 서비스 토큰 아래에서 프로세스를 실행하여, 소유권을 가져오지 않고도 TrustedInstaller ACL로 보호된 리소스에 접근할 수 있게 해주는 컴팩트한 Windows 유틸리티입니다. 전부 수작업으로 작성된 네이티브 x86 및 x64 어셈블리로 구현되며, C 런타임이나 서드파티 의존성 없이 듀얼 모드 GUI 및 CLI 바이너리로 제공되고 크기는 각각 약 40 KiB와 30 KiB 이하입니다. 주요 기능으로는 TrustedInstaller 프로세스 토큰 복제, 광범위한 토큰 권한 활성화, UAC 자체 상승, Explorer 컨텍스트 메뉴 설치, CLI I/O 리다이렉션 중계, 선택적 MRU 기록, 그리고 관련 시스템 유지보수 도우미가 있습니다. 관리자, 포렌식 및 리버스 엔지니어링 워크플로, SYSTEM만으로는 보호된 OS 구성 요소를 수정하기에 부족한 심층 Windows 복구·디버깅 환경을 대상으로 합니다.
