CMDT 是一款精簡的 Windows 公用程式，可在 TrustedInstaller 服務權杖下啟動程序，從而在不取得擁有權的情況下存取受 TrustedInstaller ACL 保護的資源。它完全以手工撰寫的原生 x86 與 x64 組語實作，提供雙模式 GUI 與 CLI 二進位檔，無 C 執行階段或第三方相依，體積分別約在 40 KiB 與 30 KiB 以內。主要能力包括複製 TrustedInstaller 程序權杖、啟用大量權杖特權、UAC 自我提升、Explorer 右鍵功能表安裝、CLI I/O 重新導向中繼、可選 MRU 歷史，以及相關的系統維護輔助功能。面向系統管理員、鑑識與逆向工程工作流程，以及僅靠 SYSTEM 仍不足以修改受保護作業系統元件時的深度 Windows 修復或偵錯場景。
