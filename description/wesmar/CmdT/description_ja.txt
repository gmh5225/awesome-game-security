CMDT は、TrustedInstaller サービストークンの下でプロセスを起動し、所有権を取得せずに TrustedInstaller ACL で保護されたリソースへアクセスできる、コンパクトな Windows ユーティリティです。すべて手書きのネイティブ x86 / x64 アセンブリで実装され、C ランタイムやサードパーティ依存なしのデュアルモード GUI / CLI バイナリとして、それぞれ約 40 KiB と 30 KiB 以下に抑えられています。主な機能には、TrustedInstaller プロセストークンの複製、多数のトークン特権の有効化、UAC 自己昇格、Explorer コンテキストメニューへのインストール、CLI I/O リダイレクト中継、任意の MRU 履歴、および関連するシステム保守ヘルパーがあります。管理者、フォレンジックやリバースエンジニアリングのワークフロー、および SYSTEM だけでは保護された OS コンポーネントを変更できない深い Windows 修復・デバッグ用途を想定しています。
