CMDT — компактная утилита для Windows, которая запускает процессы под токеном службы TrustedInstaller, чтобы они могли обращаться к ресурсам, защищённым ACL TrustedInstaller, без получения владения. Она полностью написана на рукописном нативном ассемблере x86 и x64 в виде dual-mode GUI и CLI бинарников без C runtime и сторонних зависимостей и укладывается примерно в 40 KiB и 30 KiB соответственно. Ключевые возможности включают дублирование токена процесса TrustedInstaller, включение широкого набора привилегий токена, самоповышение через UAC, установку в контекстное меню Explorer, ретрансляцию перенаправления CLI I/O, необязательную историю MRU и связанные помощники по обслуживанию системы. Предназначена для администраторов, рабочих процессов форензики и реверс-инжиниринга, а также глубокого восстановления или отладки Windows, где одного SYSTEM недостаточно для изменения защищённых компонентов ОС.
