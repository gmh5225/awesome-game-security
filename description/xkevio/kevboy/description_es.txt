Este proyecto es un emulador de Game Boy escrito en Rust con interfaz gráfica. Implementa la CPU de Game Boy, la gestión de memoria, el renderizado gráfico y el manejo de entrada, con soporte para varios formatos ROM y recursos de iconos personalizados. Resulta especialmente útil para desarrolladores de emuladores y entusiastas de los juegos retro que estudian la emulación de hardware de Game Boy y la arquitectura de emuladores en Rust.
