Этот проект — эмулятор Game Boy на Rust с графическим интерфейсом. Он реализует CPU Game Boy, управление памятью, графический рендеринг и обработку ввода, а также поддерживает различные форматы ROM и пользовательские иконки. В первую очередь полезен разработчикам эмуляторов и любителям ретро-игр, изучающим аппаратную эмуляцию Game Boy и архитектуру эмуляторов на Rust.
