Este proyecto se centra en simple-EFT-Base: una base OOP sencilla para Escape from Tarkov hecha por el autor; los offsets probablemente estén desactualizados y no se planea actualizarlos pronto.
Está escrito principalmente en C++ y C/C++ y se centra en el desarrollo de drivers, el renderizado y la animación.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito cheat / juego: EFT.
