Ce projet porte sur Simple Rust Base : une base de cheat OOP simple pour le jeu vidéo Rust, réalisée par l’auteur il y a quelque temps ; le code est ancien et pourrait être largement amélioré, et il est également obsolète.
Il est principalement écrit en C++ et C/C++ et s’articule autour du développement de pilotes, du rendu et du modding.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine cheat / jeu Rust.
