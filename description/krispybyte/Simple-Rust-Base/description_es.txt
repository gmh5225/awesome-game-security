Este proyecto se centra en Simple Rust Base: una base de cheat OOP sencilla para el videojuego Rust hecha por el autor hace tiempo; el código es antiguo y podría mejorarse mucho, y además está desactualizado.
Está escrito principalmente en C++ y C/C++ y se centra en el desarrollo de drivers, el rendering y el modding.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de reverse engineering que estudian técnicas ofensivas en el ámbito cheat / juego Rust.
