Этот проект — Olive.c, простая 2D-графическая библиотека на C без зависимостей. Она реализует программную растеризацию: отрисовку линий, заливку треугольников, рендеринг кругов, вывод текста и базовую композицию изображений, используя только функции стандартной библиотеки C. Библиотека из одного заголовочного файла рендерит в сырые пиксельные буферы, которые можно отображать через любую оконную систему. Она предназначена для начинающих программистов и любителей, изучающих основы компьютерной графики на минимальных реализациях на C.
