Questo progetto è Olive.c, una semplice libreria grafica 2D scritta in C senza dipendenze. Implementa la rasterizzazione software, incluso il disegno di linee, il riempimento di triangoli, il rendering di cerchi, la visualizzazione di testo e la composizione di immagini di base, usando solo le funzioni della libreria standard C. La libreria a header singolo esegue il rendering su buffer di pixel grezzi visualizzabili tramite qualsiasi sistema a finestre. È rivolta a programmatori principianti e sviluppatori amatoriali che imparano i fondamenti della computer grafica con implementazioni C minime.
