Dieses Projekt zeigt, dass der Shadow Stack eine interessante Erkennungsmöglichkeit bietet. Das Tool implementiert CaptureStackBackTrace/StackWalk64 mit dem Shadow Stack (auch CET/HSP) zur Erkennung von Thread-Stack-Spoofing. Es ist vor allem für Anti-Cheat-Ingenieure und defensive Security-Forscher nützlich, die im Bereich Anti-Cheat / Detection: Stack-Spoof arbeiten.
