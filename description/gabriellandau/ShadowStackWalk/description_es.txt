Este proyecto muestra que el shadow stack ofrece una interesante oportunidad de detección. Esta herramienta implementa CaptureStackBackTrace/StackWalk64 con el shadow stack (también conocido como CET/HSP) para detectar el spoofing de pila de hilos. Es especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área anti-cheat / detección: spoof de pila.
