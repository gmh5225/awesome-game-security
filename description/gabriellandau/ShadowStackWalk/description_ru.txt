Этот проект показывает, что теневой стек (shadow stack) открывает интересную возможность для детектирования. Инструмент реализует CaptureStackBackTrace/StackWalk64 с использованием теневого стека (также CET/HSP) для выявления подмены стека потока. Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / детектирования: spoof stack.
