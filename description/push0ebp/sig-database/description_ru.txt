Этот проект — коллекция файлов сигнатур FLIRT (Fast Library Identification and Recognition Technology) для различных версий библиотек Linux. Он охватывает OpenSSL и другие системные библиотеки для нескольких выпусков Ubuntu и архитектур (amd64/i386), организованные по версии дистрибутива. В основном полезен инженерам по реверс-инжинирингу, работающим с IDA Pro и нуждающимся в сигнатурах для идентификации библиотечных функций при анализе Linux-бинарников.
