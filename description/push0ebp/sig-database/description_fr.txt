Ce projet est une collection de fichiers de signatures FLIRT (Fast Library Identification and Recognition Technology) pour diverses versions de bibliothèques Linux. Il couvre OpenSSL et d'autres bibliothèques système sur plusieurs versions d'Ubuntu et architectures (amd64/i386), organisées par version de distribution. Il est principalement utile aux reverse engineers utilisant IDA Pro qui ont besoin de signatures d'identification de fonctions de bibliothèques pour l'analyse de binaires Linux.
