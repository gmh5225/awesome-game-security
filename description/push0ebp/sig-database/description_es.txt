Este proyecto es una colección de archivos de firmas FLIRT (Fast Library Identification and Recognition Technology) para diversas versiones de bibliotecas de Linux. Cubre OpenSSL y otras bibliotecas del sistema en múltiples versiones de Ubuntu y arquitecturas (amd64/i386), organizadas por versión de distribución. Es principalmente útil para ingenieros inversos que usan IDA Pro y necesitan firmas de identificación de funciones de biblioteca para el análisis de binarios de Linux.
