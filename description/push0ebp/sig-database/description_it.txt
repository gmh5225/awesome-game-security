Questo progetto è una raccolta di file di firme FLIRT (Fast Library Identification and Recognition Technology) per varie versioni di librerie Linux. Copre OpenSSL e altre librerie di sistema su più release di Ubuntu e architetture (amd64/i386), organizzate per versione della distribuzione. È principalmente utile per i reverse engineer che usano IDA Pro e necessitano di firme per l'identificazione delle funzioni di libreria nell'analisi di binari Linux.
