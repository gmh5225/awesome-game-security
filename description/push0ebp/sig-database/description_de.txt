Dieses Projekt ist eine Sammlung von FLIRT-Signaturdateien (Fast Library Identification and Recognition Technology) für verschiedene Linux-Bibliotheksversionen. Es umfasst OpenSSL und andere Systembibliotheken über mehrere Ubuntu-Releases und Architekturen (amd64/i386), organisiert nach Distributionsversion. Es ist vor allem für Reverse Engineers nützlich, die IDA Pro nutzen und Signaturen zur Identifikation von Bibliotheksfunktionen für die Analyse von Linux-Binärdateien benötigen.
