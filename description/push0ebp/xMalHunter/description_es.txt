Este proyecto es una herramienta de detección de malware que aprovecha pe-sieve para escanear procesos en ejecución en busca de código inyectado, hooks en línea y módulos hollowed. Se enlaza con las bibliotecas pe-sieve y libpeconv para realizar la detección automatizada de artefactos de malware en memoria en procesos Windows de 32 y 64 bits. Es principalmente útil para analistas de malware e investigadores de anti-trampas que estudian la detección de inyección de código en tiempo de ejecución y la verificación de integridad de procesos.
