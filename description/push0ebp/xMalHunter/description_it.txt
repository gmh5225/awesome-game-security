Questo progetto è uno strumento di rilevamento malware che sfrutta pe-sieve per analizzare i processi in esecuzione alla ricerca di codice iniettato, hook inline e moduli hollowed. Si collega alle librerie pe-sieve e libpeconv per eseguire il rilevamento automatico di artefatti malware in memoria nei processi Windows a 32 e 64 bit. È principalmente utile per analisti di malware e ricercatori anti-cheat che studiano il rilevamento dell'iniezione di codice a runtime e la verifica dell'integrità dei processi.
