Этот проект — набор инструментов с открытым исходным кодом для цифровой криминалистики изображений, предназначенный для выявления манипуляций и подделок изображений. GUI на Python включает модули анализа для Error Level Analysis (ELA), проверки метаданных EXIF, анализа в частотной области, градиентных карт, анализа гистограмм, обнаружения клонирования, а также интеграцию с нейросетевым детектором TruFor. Он в первую очередь полезен исследователям цифровой криминалистики и аналитикам античита, изучающим подлинность изображений и манипуляции со скриншотами.
