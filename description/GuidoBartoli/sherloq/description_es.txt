Este proyecto es un conjunto de herramientas de código abierto de forense digital de imágenes para detectar manipulación y adulteración de imágenes. La interfaz gráfica basada en Python incluye módulos de análisis para análisis de nivel de error (ELA), inspección de metadatos EXIF, análisis en el dominio de la frecuencia, mapas de gradiente, análisis de histograma, detección de clonación e integración con el detector de redes neuronales TruFor. Es principalmente útil para investigadores de forense digital y analistas anti-trampas que investigan la autenticidad de imágenes y la manipulación de capturas de pantalla.
