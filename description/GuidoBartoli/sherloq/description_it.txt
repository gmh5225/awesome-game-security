Questo progetto è un set di strumenti open source di forensics digitale delle immagini per rilevare manipolazione e alterazione delle immagini. La GUI basata su Python include moduli di analisi per error level analysis (ELA), ispezione dei metadati EXIF, analisi nel dominio della frequenza, mappe di gradiente, analisi dell'istogramma, rilevamento del cloning e integrazione con il rilevatore a rete neurale TruFor. È principalmente utile per ricercatori di digital forensics e analisti anti-cheat che indagano sull'autenticità delle immagini e sulla manipolazione degli screenshot.
