Этот проект — коллекция техник внедрения в процессы Windows, которые злоупотребляют внутренностями пула потоков для выполнения кода в удалённых процессах.
Он написан на C++ и реализует несколько вариантов, включая перезапись стартовой процедуры worker factory и вставку элементов TP_WORK, TP_WAIT, TP_IO, TP_ALPC, TP_JOB, TP_DIRECT и TP_TIMER.
Кодовая база включает обёртки нативных API, вспомогательные средства для перехвата дескрипторов и структурированные реализации для каждой примитивы пула потоков.
Проект предназначен для исследований red team и для проверки видимости античитов или EDR в отношении слабообнаруживаемых техник внедрения.
