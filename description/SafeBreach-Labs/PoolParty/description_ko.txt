이 프로젝트는 스레드 풀 내부를 악용해 원격 프로세스에서 코드를 실행하는 Windows 프로세스 인젝션 기법 모음입니다.
C++로 작성되었으며, worker factory 시작 루틴 덮어쓰기와 TP_WORK, TP_WAIT, TP_IO, TP_ALPC, TP_JOB, TP_DIRECT, TP_TIMER 항목 삽입을 포함한 여러 변형을 구현합니다.
코드베이스에는 네이티브 API 래퍼, 핸들 하이재킹 헬퍼, 그리고 각 스레드 풀 프리미티브에 대한 구조화된 구현이 포함됩니다.
레드팀 연구와 저탐지 인젝션 기법에 대한 안티치트 또는 EDR 가시성 테스트 용도로 설계되었습니다.
