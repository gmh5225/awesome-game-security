本プロジェクトは、スレッドプールの内部機構を悪用してリモートプロセスでコードを実行する Windows プロセスインジェクション手法のコレクションです。
C++ で書かれており、worker factory の開始ルーチンの上書きや、TP_WORK、TP_WAIT、TP_IO、TP_ALPC、TP_JOB、TP_DIRECT、TP_TIMER アイテムの挿入など、複数の派生手法を実装しています。
コードベースには、ネイティブ API ラッパー、ハンドルハイジャック用ヘルパー、および各スレッドプールプリミティブ向けの構造化された実装が含まれます。
レッドチーム研究、および低検知のインジェクション手法に対するアンチチートや EDR の可視性を検証する用途向けに設計されています。
