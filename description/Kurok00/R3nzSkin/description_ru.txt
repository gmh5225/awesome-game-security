Этот проект — скиньченджер для League of Legends, который изменяет отрисовку скинов на стороне клиента без модификации игровых файлов. Он включает C++ DLL-инжектор, инструменты сканирования паттернов для поиска игровых оффсетов после обновлений, автоматизированные обновлятели паттернов на Python и скрипты загрузки сообщественных паттернов с интеграцией KsDumper. В основном полезен исследователям моддинга и реверс-инженерам, изучающим внутреннее устройство клиента League of Legends и техники патчинга памяти во время выполнения.
