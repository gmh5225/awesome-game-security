Questo progetto è un skin changer per League of Legends che modifica il rendering delle skin lato client senza alterare i file di gioco. Include un injector DLL in C++, strumenti di pattern scanning per trovare gli offset del gioco dopo gli aggiornamenti, updater automatici di pattern basati su Python e script per recuperare pattern della community con integrazione KsDumper. È utile soprattutto a ricercatori di game modding e reverse engineer che studiano gli interni del client di League of Legends e le tecniche di patching della memoria a runtime.
