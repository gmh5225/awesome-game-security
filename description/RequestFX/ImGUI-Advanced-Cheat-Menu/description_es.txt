Este proyecto es un framework de menú de trampas basado en ImGui para Windows.
Ofrece componentes modulares como ESP, asistencia de mira, control de retroceso, módulos HUD y paneles de configuración activados por teclas rápidas.
La implementación es principalmente en C++ y usa SFML como backend, con widgets ImGui personalizados y carga de fuentes en memoria.
Se usa sobre todo como plantilla inicial para construir y organizar interfaces de usuario de trampas para juegos.
