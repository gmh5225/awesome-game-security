Этот проект — фреймворк чит-меню на базе ImGui для Windows.
Он предоставляет модульные компоненты: ESP, помощь в прицеливании, контроль отдачи, модули HUD и панели настроек, управляемые горячими клавишами.
Реализация в основном на C++ и использует SFML в качестве бэкенда, с пользовательскими виджетами ImGui и загрузкой шрифтов в память.
В основном применяется как стартовый шаблон для создания и организации пользовательских интерфейсов игровых читов.
