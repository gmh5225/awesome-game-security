SwiftShield は、iOS プロジェクト内の型やメソッドをランダムで不可逆な暗号化名にリネームし、リバースエンジニアリングやジェイルブレイク向け tweak を困難にする Swift 難読化ツールです。Apple の SourceKit を使って Xcode と同様にプロジェクトをインデックスし、コードベース全体でシンボルを安全にリネームします。ソースがアンロックされている場合はサードパーティライブラリも含めます。クラッシュログを元に戻すための変換マップを出力でき、public や open API の無視、ドライランモードなどのオプションもサポートします。主に Swift で書かれたコマンドラインツールで、アプリの静的解析やリバースエンジニアリングを大幅に難しくしたい iOS 開発者向けです。
