SwiftShield — инструмент обфускации для Swift, который переименовывает типы и методы в iOS-проектах в случайные необратимые зашифрованные имена, чтобы затруднить реверс-инжиниринг и jailbreak-твики. Он использует SourceKit от Apple для индексации проекта подобно Xcode, затем безопасно переименовывает символы по всей кодовой базе, включая сторонние библиотеки при разблокированных исходниках. Может создавать карту преобразований для деобфускации crash-логов и поддерживает опции вроде игнорирования public или open API и режима dry-run. Написан в основном на Swift как утилита командной строки и предназначен для iOS-разработчиков, которые хотят существенно усложнить статический анализ и реверс-инжиниринг своих приложений.
