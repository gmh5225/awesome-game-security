Este proyecto es un ejemplo mínimo de eBPF en Android para monitorear eventos de llamadas al sistema del kernel. Define un programa BPF de tipo tracepoint en raw_syscalls/sys_enter y almacena los PID e identificadores de syscall observados en un mapa BPF. Una herramienta complementaria en C++ en espacio de usuario adjunta el programa y lee el contenido del mapa mediante las bibliotecas bpf de Android. Está pensado para el monitoreo de seguridad Android de bajo nivel y el análisis del comportamiento de las syscalls.
