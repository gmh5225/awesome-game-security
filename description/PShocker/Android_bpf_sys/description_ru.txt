Этот проект — минимальный пример eBPF для Android, предназначенный для мониторинга событий системных вызовов ядра. Он определяет BPF-программу типа tracepoint на raw_syscalls/sys_enter и сохраняет наблюдаемые PID и идентификаторы системных вызовов в BPF map. Сопутствующий пользовательский инструмент на C++ подключает программу и читает содержимое map через библиотеки bpf Android. Предназначен для низкоуровневого мониторинга безопасности Android и анализа поведения системных вызовов.
