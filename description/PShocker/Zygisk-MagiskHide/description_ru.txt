Этот проект — модуль Magisk на базе Zygisk, воссоздающий сокрытие в стиле MagiskHide. Он содержит нативный код для скрытия связанных с Magisk точек монтирования и изменения чувствительных системных свойств Android, которые часто проверяют средства обнаружения root. Скрипты сборки упаковывают бинарные файлы для нескольких ABI в устанавливаемые архивы модуля для развёртывания на устройствах с root. Проект ориентирован на исследования мобильной безопасности и тестирование антидетекта в средах, где приложения принуждают к проверкам root.
