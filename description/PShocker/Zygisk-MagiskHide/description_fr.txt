Ce projet est un module Magisk basé sur Zygisk qui recrée un comportement de masquage de type MagiskHide. Il contient du code natif pour dissimuler les montages liés à Magisk et modifier des propriétés système Android sensibles que les contrôles de détection de root inspectent souvent. Les scripts de build empaquettent des binaires multi-ABI dans des archives de module installables pour le déploiement sur des appareils rootés. Il s'adresse à la recherche en sécurité mobile et aux tests anti-détection dans des environnements où les applications imposent des vérifications de root.
