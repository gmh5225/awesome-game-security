Этот проект предоставляет руководство по CE на китайском языке.
Он организован как документация и справочные материалы для области читов / гайдов, а не как единая автономная runtime-кодовая база.
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / гайдов.
