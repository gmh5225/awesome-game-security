Этот проект сосредоточен на дампе Il2Cpp с помощью Frida.
Он должен работать с любой версией Unity в диапазоне 5.3.0 - 6000.1.x.
В основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования игровых движков: Unity.
