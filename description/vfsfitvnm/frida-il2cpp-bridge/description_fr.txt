Ce projet se concentre sur le dump Il2Cpp avec Frida.
Il devrait fonctionner pour toute version d'Unity dans la plage 5.3.0 - 6000.1.x.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine cheat / explorateur de moteur de jeu : Unity.
