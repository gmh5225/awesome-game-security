Este proyecto es un emulador ligero de un entorno de lanzador de juegos al estilo BattlEye.
Es una base de código C++ compacta, organizada en torno a archivos de proyecto de Visual Studio y cabeceras auxiliares.
La implementación se centra en reproducir el comportamiento del lado del lanzador para experimentos controlados.
Su caso de uso principal es la investigación de anti-trampas y las pruebas de compatibilidad en laboratorios aislados.
