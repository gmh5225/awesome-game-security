Ce projet est un émulateur léger d’un environnement de lanceur de jeu de style BattlEye.
Il s’agit d’une base de code C++ compacte, organisée autour de fichiers de projet Visual Studio et d’en-têtes utilitaires.
L’implémentation vise à reproduire le comportement côté lanceur pour des expériences contrôlées.
Son usage principal est la recherche anti-triche et les tests de compatibilité dans des laboratoires isolés.
