Этот проект — лёгкий эмулятор среды игрового лаунчера в стиле BattlEye.
Это компактная кодовая база на C++, построенная вокруг файлов проекта Visual Studio и вспомогательных заголовков.
Реализация сосредоточена на воспроизведении поведения со стороны лаунчера для контролируемых экспериментов.
Основной сценарий использования — исследование античита и проверка совместимости в изолированных лабораторных условиях.
