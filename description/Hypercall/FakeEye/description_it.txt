Questo progetto è un emulatore leggero di un ambiente di launcher di gioco in stile BattlEye.
È una codebase C++ compatta costruita intorno a file di progetto Visual Studio e header di supporto.
L’implementazione si concentra sulla riproduzione del comportamento lato launcher per esperimenti controllati.
Il caso d’uso principale è la ricerca anti-cheat e i test di compatibilità in laboratori isolati.
