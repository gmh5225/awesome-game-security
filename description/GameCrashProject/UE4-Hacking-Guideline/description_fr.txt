UE4-Hacking-Guideline est un projet documentaire orienté recherche qui présente une méthodologie pour analyser les faiblesses de sécurité d'Unreal Engine 4. Le dépôt est organisé en chapitres Markdown couvrant le contexte du moteur, les vecteurs d'attaque, les outils, la configuration de build et le processus d'analyse. Il se concentre sur l'étude systématique des vulnérabilités au niveau du moteur, plutôt que uniquement sur la détection opérationnelle de triches. Son public principal comprend les étudiants et les chercheurs en sécurité qui apprennent l'évaluation structurée de la sécurité des moteurs de jeu.
