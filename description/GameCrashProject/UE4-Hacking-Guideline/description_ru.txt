UE4-Hacking-Guideline — исследовательский документационный проект, описывающий методологию анализа уязвимостей безопасности Unreal Engine 4. Репозиторий организован в виде глав Markdown, охватывающих фон движка, векторы атак, инструментарий, настройку сборки и процесс анализа. Акцент сделан на систематическом изучении уязвимостей на уровне движка, а не только на операционных темах обнаружения читов. Основная аудитория — студенты и исследователи безопасности, изучающие структурированную оценку безопасности игровых движков.
