Este proyecto es un módulo Magisk que inicia automáticamente un servidor Frida modificado de forma furtiva al arrancar Android. Incluye scripts del módulo para la instalación basada en Magisk y la gestión del ciclo de vida del servicio, además de una pequeña utilidad en C++ para metadatos de actualización y descargas del servidor. Una interfaz web en HTML y JavaScript ofrece control de inicio/parada, visualización del estado y entrada de parámetros personalizados. Está orientado a flujos de ingeniería inversa móvil y pruebas de seguridad donde la instrumentación persistente y la reducción de detección son importantes.
