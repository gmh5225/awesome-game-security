Questo progetto è un modulo Magisk che avvia automaticamente un server Frida modificato in modo stealth all'avvio di Android. Include script del modulo per l'installazione basata su Magisk e la gestione del ciclo di vita del servizio, oltre a una piccola utility C++ per i metadati di aggiornamento e il download del server. Un'interfaccia web in HTML e JavaScript offre controllo avvio/arresto, visualizzazione dello stato e input di parametri personalizzati. È pensato per i flussi di reverse engineering mobile e test di sicurezza in cui sono importanti l'instrumentazione persistente e la riduzione del rilevamento.
