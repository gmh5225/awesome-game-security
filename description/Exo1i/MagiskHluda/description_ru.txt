Этот проект — модуль Magisk, который автоматически запускает скрытно модифицированный сервер Frida при загрузке Android. Он включает скрипты модуля для установки через Magisk и управления жизненным циклом службы, а также небольшую утилиту на C++ для метаданных обновлений и загрузки сервера. Веб-интерфейс на HTML и JavaScript обеспечивает управление запуском/остановкой, отображение статуса и ввод пользовательских параметров. Он ориентирован на рабочие процессы мобильного реверс-инжиниринга и тестирования безопасности, где важны постоянная инструментализация и снижение обнаружения.
