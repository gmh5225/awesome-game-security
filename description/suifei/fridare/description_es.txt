Este proyecto es una herramienta de reempaquetado y anti-detección de Frida para plataformas iOS y Android. Proporciona scripts de shell y una utilidad de reemplazo hexadecimal en Go que parchea cadenas, símbolos y nombres de artefactos Frida reconocibles para evadir la detección anti-instrumentación común, junto con scripts de instalación automatizados y documentación para configuraciones iOS con root y sin root. Es especialmente útil para ingenieros de ingeniería inversa móvil que necesitan despliegues sigilosos de Frida que eviten los mecanismos de detección anti-Frida.
