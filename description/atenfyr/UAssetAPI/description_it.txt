Questo progetto è una libreria .NET di basso livello per leggere e scrivere file asset di Unreal Engine. Supporta un'ampia gamma di versioni cooked e uncooked, molti tipi di property ed export, il round-trip JSON, la gestione del bytecode Kismet e il parsing opzionale dei file .usmap per dati non versionati. L'implementazione è principalmente in C# ed è progettata per preservare la fedeltà binaria quando si modificano gli asset in modo programmatico. Si rivolge a workflow di modding, tooling e reverse engineering per giochi basati su Unreal.
