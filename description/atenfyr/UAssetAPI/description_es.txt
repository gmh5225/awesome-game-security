Este proyecto es una biblioteca .NET de bajo nivel para leer y escribir archivos de assets de Unreal Engine. Admite un amplio rango de versiones cooked y uncooked, muchos tipos de propiedades y exports, round-trip JSON, manejo de bytecode Kismet y el parseo opcional de .usmap para datos sin versión. La implementación es principalmente en C# y está diseñada para preservar la fidelidad binaria al editar assets de forma programática. Está orientada a flujos de trabajo de modding, herramientas y reverse engineering para juegos basados en Unreal.
