Этот проект — низкоуровневая .NET-библиотека для чтения и записи файлов ассетов Unreal Engine. Она поддерживает широкий спектр cooked и uncooked версий, множество типов свойств и экспортов, JSON round-tripping, работу с байткодом Kismet, а также опциональный разбор .usmap для неверсионированных данных. Реализация в основном на C# и рассчитана на сохранение бинарной точности при программном редактировании ассетов. Ориентирована на моддинг, инструментарий и reverse engineering для игр на базе Unreal.
