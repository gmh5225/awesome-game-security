Questo progetto è UAssetGUI, uno strumento GUI per visualizzare e modificare i file .uasset e .umap di Unreal Engine. Analizza il formato binario UAsset e mostra export, import, name map, dati delle proprietà e metadati degli asset in una vista ad albero interattiva con capacità di editing. L'applicazione C# supporta più versioni di Unreal Engine, da UE4 a UE5. È destinata a modder di Unreal Engine e ricercatori di giochi che devono ispezionare o modificare file di asset cooked senza l'Unreal Editor.
