本プロジェクトは UAssetGUI であり、Unreal Engine の .uasset および .umap ファイルを閲覧・編集するための GUI ツールです。UAsset バイナリ形式を解析し、エクスポート、インポート、ネームマップ、プロパティデータ、アセットメタデータを編集可能なインタラクティブなツリービューで表示します。この C# アプリケーションは UE4 から UE5 までの複数の Unreal Engine バージョンに対応しています。Unreal Editor を使わずに cooked アセットファイルを調査・改変する必要がある Unreal Engine のモッダーやゲーム研究者を対象としています。
