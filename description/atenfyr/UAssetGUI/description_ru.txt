Этот проект — UAssetGUI, GUI-инструмент для просмотра и редактирования файлов .uasset и .umap Unreal Engine. Он разбирает бинарный формат UAsset и отображает exports, imports, name maps, данные свойств и метаданные ассетов в интерактивном дереве с возможностью редактирования. C#-приложение поддерживает несколько версий Unreal Engine — от UE4 до UE5. Оно предназначено для моддеров Unreal Engine и исследователей игр, которым нужно просматривать или изменять cooked-ассеты без Unreal Editor.
