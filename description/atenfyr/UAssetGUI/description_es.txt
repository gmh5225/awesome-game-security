Este proyecto es UAssetGUI, una herramienta GUI para ver y editar archivos .uasset y .umap de Unreal Engine. Analiza el formato binario UAsset y muestra exports, imports, name maps, datos de propiedades y metadatos de assets en una vista de árbol interactiva con capacidades de edición. La aplicación en C# admite varias versiones de Unreal Engine, desde UE4 hasta UE5. Está dirigida a modders de Unreal Engine e investigadores de juegos que necesitan inspeccionar o modificar archivos de assets cooked sin el Unreal Editor.
