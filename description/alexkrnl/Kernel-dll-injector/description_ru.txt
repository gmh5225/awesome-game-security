Этот проект — DLL-инжектор в режиме ядра для Windows, который внедряет выбранную DLL в вновь создаваемые процессы при загрузке kernel32. Реализация основана на анализе техники руткита Sirifef (Max++) и включает компоненты драйвера и примерной DLL. Разработан на C/C++ с помощью Visual Studio и инструментов WDK и документирован как ориентированный на x86. Репозиторий полезен для изучения внедрения процессов с поддержкой ядра и для оборонительных исследований по обнаружению или смягчению такого поведения.
