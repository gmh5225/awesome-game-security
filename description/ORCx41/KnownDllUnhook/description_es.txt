Este proyecto es una utilidad de unhooking de API de Windows que restaura las secciones de código de módulos a partir de imágenes limpias en el espacio de nombres \KnownDlls. Itera las DLL cargadas, mapea copias de confianza, reemplaza la sección .text enganchada y restaura las protecciones de memoria originales. La implementación se apoya en syscalls nativos de bajo nivel para operaciones críticas de memoria y mapeo. Está pensado para investigación de seguridad sobre detección y evasión de hooks de EDR o anti-cheat.
