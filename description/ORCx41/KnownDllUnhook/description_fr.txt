Ce projet est un utilitaire d'unhooking d'API Windows qui restaure les sections de code des modules à partir d'images propres dans l'espace de noms \KnownDlls. Il parcourt les DLL chargées, mappe des copies de confiance, remplace la section .text accrochée et rétablit les protections mémoire d'origine. L'implémentation s'appuie sur des syscalls natifs de bas niveau pour les opérations critiques de mémoire et de mapping. Il est destiné à la recherche en sécurité sur la détection et le contournement des hooks EDR ou anti-cheat.
