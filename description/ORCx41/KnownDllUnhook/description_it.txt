Questo progetto è un'utilità di unhooking delle API Windows che ripristina le sezioni di codice dei moduli da immagini pulite nello spazio dei nomi \KnownDlls. Scorre le DLL caricate, mappa copie attendibili, sostituisce la sezione .text agganciata e ripristina le protezioni di memoria originali. L'implementazione si basa su syscall nativi di basso livello per le operazioni critiche di memoria e mapping. È pensato per la ricerca sulla sicurezza relativa a rilevamento ed evasione degli hook di EDR o anti-cheat.
