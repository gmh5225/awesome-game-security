Этот проект — утилита unhooking Windows API, которая восстанавливает секции кода модулей из чистых образов в пространстве имён \KnownDlls. Она перебирает загруженные DLL, отображает доверенные копии, заменяет перехваченную секцию .text и восстанавливает исходные защиты памяти. Реализация опирается на низкоуровневые нативные syscalls для критических операций с памятью и отображением. Предназначена для исследований безопасности по обнаружению и обходу хуков EDR или античита.
