Этот проект предоставляет конфигурации сборки для компиляции инструмента magiskboot из Magisk с помощью Android NDK в Linux. Он патчит исходный код magiskboot для автономной сборки с toolchain NDK, создавая нативный бинарник для распаковки и упаковки Android boot-образов, а также работы с ramdisk. Предназначен для Android-разработчиков и сборщиков ROM, которым нужно компилировать magiskboot через NDK для сценариев кастомного recovery и root.
