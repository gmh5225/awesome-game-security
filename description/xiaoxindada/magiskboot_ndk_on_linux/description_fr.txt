Ce projet fournit des configurations de build pour compiler l'outil magiskboot de Magisk avec l'Android NDK sous Linux. Il applique des correctifs au code source de magiskboot pour permettre une compilation autonome avec les toolchains NDK, produisant un binaire natif pour le dépaquetage, le rempaquetage d'images de boot Android et la manipulation du ramdisk. Il s'adresse aux développeurs Android et aux créateurs de ROM qui ont besoin de compiler magiskboot avec le NDK pour des workflows de recovery personnalisée et de root.
