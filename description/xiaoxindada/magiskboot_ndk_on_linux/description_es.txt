Este proyecto proporciona configuraciones de build para compilar la herramienta magiskboot de Magisk con el Android NDK en Linux. Aplica parches al código fuente de magiskboot para construirlo de forma independiente con las toolchains del NDK, generando un binario nativo para desempaquetar y reempaquetar imágenes de boot de Android y manipular el ramdisk. Está dirigido a desarrolladores de Android y creadores de ROM que necesitan compilar magiskboot con el NDK para flujos de trabajo de recovery personalizada y root.
