Этот проект — Pine, динамический фреймворк для хукинга Java-методов под Android (среда выполнения ART). Он перехватывает Java-методы во время выполнения, изменяя внутренние точки входа методов ART, и поддерживает как inline-хуки, так и замену методов без необходимости root-доступа. Фреймворк работает на Android 7.0+ и предоставляет Xposed-совместимый API для хуков before/after методов. Предназначен для Android reverse-инженеров и модификаторов приложений, которым нужен перехват Java-методов во время выполнения без фреймворка Xposed.
