Dieses Projekt ist Pine, ein dynamisches Java-Methoden-Hooking-Framework für Android (ART-Runtime). Es hookt Java-Methoden zur Laufzeit, indem es die internen Methodeneinstiegspunkte von ART verändert, und unterstützt sowohl Inline-Hooks als auch Methodenersetzung ohne Root-Zugriff. Das Framework läuft unter Android 7.0+ und stellt eine Xposed-kompatible API für Before-/After-Hooks bereit. Es richtet sich an Android-Reverse-Engineers und App-Modifizierer, die zur Laufzeit Java-Methoden ohne das Xposed-Framework abfangen müssen.
