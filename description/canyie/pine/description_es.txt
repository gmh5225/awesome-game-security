Este proyecto es Pine, un framework de hook dinámico de métodos Java para Android (runtime ART). Engancha métodos Java en tiempo de ejecución modificando los puntos de entrada internos de los métodos de ART, y admite tanto hooks inline como reemplazo de métodos sin requerir acceso root. El framework funciona en Android 7.0+ y ofrece una API compatible con Xposed para hooks before/after de métodos. Está dirigido a ingenieros inversos de Android y modificadores de aplicaciones que necesitan interceptar métodos Java en tiempo de ejecución sin el framework Xposed.
