Ce projet est Pine, un framework de hook dynamique de méthodes Java pour Android (runtime ART). Il accroche les méthodes Java à l'exécution en modifiant les points d'entrée internes des méthodes ART, et prend en charge à la fois les hooks inline et le remplacement de méthodes sans nécessiter l'accès root. Le framework fonctionne sur Android 7.0 et versions ultérieures, et fournit une API compatible Xposed pour les hooks before/after des méthodes. Il s'adresse aux reverse engineers Android et aux modificateurs d'applications qui ont besoin d'intercepter des méthodes Java à l'exécution sans le framework Xposed.
