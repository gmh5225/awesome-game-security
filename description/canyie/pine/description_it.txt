Questo progetto è Pine, un framework di hook dinamico di metodi Java per Android (runtime ART). Aggancia i metodi Java a runtime modificando i punti di ingresso interni dei metodi di ART, supportando sia hook inline sia la sostituzione di metodi senza richiedere l'accesso root. Il framework funziona su Android 7.0+ e fornisce un'API compatibile con Xposed per hook before/after sui metodi. È rivolto a reverse engineer Android e a chi modifica app e necessita di intercettare metodi Java a runtime senza il framework Xposed.
