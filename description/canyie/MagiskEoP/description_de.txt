Dieses Projekt ist ein Proof-of-Concept für eine Android-Privilegieneskalation, der eine Schwachstelle im su-Daemon von Magisk demonstriert. Es nutzt eine Race Condition oder einen Logikfehler im Root-Zugriffsmechanismus von Magisk aus, um Privilegien einer unprivilegierten App ohne Benutzerfreigabe auf Root zu erhöhen. Die Java/C-Implementierung dokumentiert die Schwachstelle und den Angriffsvektor. Es richtet sich an Android-Sicherheitsforscher, die Magisk-Internals und die Sicherheit der Root-Verwaltung untersuchen.
