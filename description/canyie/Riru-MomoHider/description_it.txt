Questo progetto è un modulo Riru per nascondere il root Magisk dalla rilevazione da parte delle app. Intercetta le system call e le API Java usate dalle librerie di root detection (come MagiskDetector/RootBeer) per occultare la presenza di Magisk, incluso lo spoofing dei mount point, l'occultamento dei file Magisk e il blocco delle query sulle proprietà che rivelano lo stato di root. Il modulo C/Java opera tramite il meccanismo di iniezione Zygote di Riru. È rivolto a utenti Android e ricercatori di sicurezza che studiano le tecniche di bypass della root detection.
