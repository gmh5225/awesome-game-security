Ce projet est un module Riru destiné à masquer le root Magisk face à la détection par les applications. Il accroche les appels système et les API Java utilisés par les bibliothèques de détection de root (comme MagiskDetector/RootBeer) pour dissimuler la présence de Magisk, notamment en falsifiant les points de montage, en masquant les fichiers Magisk et en bloquant les requêtes de propriétés révélant le statut root. Le module C/Java fonctionne via le mécanisme d'injection Zygote de Riru. Il s'adresse aux utilisateurs Android et aux chercheurs en sécurité étudiant les techniques de contournement de la détection de root.
