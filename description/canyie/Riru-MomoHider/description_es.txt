Este proyecto es un módulo Riru para ocultar el root de Magisk frente a la detección por aplicaciones. Engancha llamadas al sistema y APIs de Java usadas por bibliotecas de detección de root (como MagiskDetector/RootBeer) para ocultar la presencia de Magisk, incluyendo la falsificación de puntos de montaje, el ocultamiento de archivos de Magisk y el bloqueo de consultas de propiedades que revelan el estado root. El módulo C/Java opera mediante el mecanismo de inyección Zygote de Riru. Está orientado a usuarios de Android e investigadores de seguridad que estudian técnicas de evasión de la detección de root.
