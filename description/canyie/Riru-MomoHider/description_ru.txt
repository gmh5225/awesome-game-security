Этот проект — модуль Riru для скрытия Magisk root от обнаружения приложениями. Он перехватывает системные вызовы и Java API, используемые библиотеками детектирования root (например MagiskDetector/RootBeer), чтобы скрыть присутствие Magisk, включая подмену точек монтирования, сокрытие файлов Magisk и блокировку запросов свойств, раскрывающих статус root. Модуль на C/Java работает через механизм инъекции Zygote в Riru. Предназначен для пользователей Android и исследователей безопасности, изучающих техники обхода детектирования root.
