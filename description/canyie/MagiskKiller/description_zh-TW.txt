此專案是一款 Android 應用程式，透過多種偵測向量辨識裝置上的 Magisk root 與 MagiskHide。
它會檢查活躍的追蹤器（MagiskHide）、解鎖的 bootloader 狀態、透過屬性區域檢查發現的被修改系統屬性、經 PTS 偵測發現的活躍 Magisk su 工作階段，並採用基於子程序的方法以避免自我追蹤干擾。
其 Java 與 JNI/C++ 程式碼庫在經 fork 的子程序中執行偵測，並透過基於管線的 IPC 執行 MagiskHide 無法在呼叫程序上攔截的檢查。
它主要適用於研究 Android 上 Magisk 偵測技術與反 root 隱藏方法的行動安全研究人員。
