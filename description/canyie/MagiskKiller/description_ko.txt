이 프로젝트는 여러 탐지 벡터를 통해 기기에서 Magisk root와 MagiskHide를 탐지하는 Android 앱입니다.
활성 트레이서(MagiskHide), 잠금 해제된 부트로더 상태, 속성 영역 검사를 통한 수정된 시스템 속성, PTS 탐지를 통한 활성 Magisk su 세션을 확인하고, 자기 트레이싱 간섭을 피하기 위해 서브프로세스 기반 접근 방식을 사용합니다.
Java 및 JNI/C++ 코드베이스는 fork된 서브프로세스에서 탐지를 실행하며, 파이프 기반 IPC를 통해 MagiskHide가 호출 프로세스에서 가로챌 수 없는 검사를 수행합니다.
주로 Android에서 Magisk 탐지 기법과 안티 루트 은닉 방법을 연구하는 모바일 보안 연구자에게 유용합니다.
