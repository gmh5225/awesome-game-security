Este proyecto es una aplicación Android que detecta Magisk root y MagiskHide en dispositivos mediante múltiples vectores de detección.
Comprueba trazadores activos (MagiskHide), el estado de bootloader desbloqueado, propiedades del sistema modificadas mediante inspección del área de propiedades, sesiones Magisk su activas por detección PTS, y usa un enfoque basado en subprocesos para evitar la interferencia del autotrazado.
El código Java y JNI/C++ ejecuta la detección en un subproceso creado con fork y con IPC basada en pipes para realizar comprobaciones que MagiskHide no puede interceptar en el proceso llamante.
Resulta especialmente útil para investigadores de seguridad móvil que estudian técnicas de detección de Magisk y métodos anti-ocultación de root en Android.
