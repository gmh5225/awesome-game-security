Этот проект — Android-приложение, которое обнаруживает Magisk root и MagiskHide на устройствах с помощью нескольких векторов детектирования.
Оно проверяет активные трассировщики (MagiskHide), разблокированное состояние загрузчика, изменённые системные свойства через инспекцию области свойств, активные сессии Magisk su через обнаружение PTS и использует подход на основе подпроцесса, чтобы избежать помех от самотрассировки.
Кодовая база на Java и JNI/C++ выполняет детектирование в форкнутом подпроцессе с IPC на основе каналов (pipe), чтобы проводить проверки, которые MagiskHide не может перехватить у вызывающего процесса.
Оно в основном полезно исследователям мобильной безопасности, изучающим техники обнаружения Magisk и методы противодействия скрытию root на Android.
