Ce projet est une application Android qui détecte Magisk root et MagiskHide sur les appareils via plusieurs vecteurs de détection.
Il vérifie les traceurs actifs (MagiskHide), l'état de bootloader déverrouillé, les propriétés système modifiées via l'inspection de la zone de propriétés, les sessions Magisk su actives par détection PTS, et utilise une approche basée sur un sous-processus pour éviter l'interférence d'auto-traçage.
Le code Java et JNI/C++ exécute la détection dans un sous-processus forké avec une IPC basée sur des pipes pour réaliser des contrôles que MagiskHide ne peut pas intercepter d'après le processus appelant.
Il est surtout utile aux chercheurs en sécurité mobile qui étudient les techniques de détection Magisk et les méthodes anti-masquage de root sur Android.
