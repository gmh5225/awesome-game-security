该项目是一款 Android 应用，通过多种检测向量识别设备上的 Magisk root 与 MagiskHide。
它会检查活跃的跟踪器（MagiskHide）、解锁的 bootloader 状态、通过属性区域检查发现的被修改系统属性、经 PTS 检测发现的活跃 Magisk su 会话，并采用基于子进程的方法以避免自我跟踪干扰。
其 Java 与 JNI/C++ 代码库在经 fork 的子进程中运行检测，并通过基于管道的 IPC 执行 MagiskHide 无法在调用进程上拦截的检查。
它主要面向研究 Android 上 Magisk 检测技术与反 root 隐藏方法的移动安全研究人员。
