このプロジェクトは、複数の検出手法を用いて端末上の Magisk root および MagiskHide を検出する Android アプリです。
アクティブなトレーサー（MagiskHide）、アンロックされたブートローダー状態、プロパティ領域検査による改変されたシステムプロパティ、PTS 検出によるアクティブな Magisk su セッションを確認し、自己トレーシング干渉を避けるためにサブプロセスベースの手法を用います。
Java および JNI/C++ のコードベースは、fork したサブプロセス内で検出を実行し、パイプベースの IPC により MagiskHide が呼び出し元プロセス上で傍受できない検査を行います。
主に、Android における Magisk 検出技術やアンチルート隠蔽手法を研究するモバイルセキュリティ研究者向けです。
