Плагин-ассистент для реверс-инжиниринга на базе ИИ для x64dbg, который встраивает чат-панель WebView2 прямо в отладчик и поддерживает вызов инструментов LLM через API Anthropic или совместимые с OpenAI.
ИИ-агент может читать память, дизассемблировать код, ставить точки останова, выполнять пошаговую отладку и программно взаимодействовать с отладчиком через диалоговый интерфейс.
Он особенно полезен реверс-инженерам, которым нужна интерактивная отладка с помощью ИИ и автоматизированный анализ внутри x64dbg.
