Un plugin assistente di reverse engineering basato sull'IA per x64dbg che incorpora un pannello chat WebView2 direttamente nel debugger, con capacità di tool-use LLM tramite API Anthropic o compatibili con OpenAI.
L'agente IA può leggere la memoria, disassemblare il codice, impostare breakpoint, eseguire il single-step e interagire con il debugger in modo programmatico tramite un'interfaccia conversazionale.
È soprattutto utile per i reverse engineer che cercano debugging interattivo assistito dall'IA e analisi automatizzata all'interno di x64dbg.
