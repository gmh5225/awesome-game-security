Ein KI-gestütztes Reverse-Engineering-Assistenten-Plugin für x64dbg, das ein WebView2-Chat-Panel direkt in den Debugger einbettet und LLM-Tool-Nutzung über Anthropic- oder OpenAI-kompatible APIs bereitstellt.
Der KI-Agent kann Speicher lesen, Code disassemblieren, Breakpoints setzen, schrittweise ausführen und über eine konversationelle Oberfläche programmatisch mit dem Debugger interagieren.
Es eignet sich vor allem für Reverse Engineers, die KI-unterstütztes interaktives Debugging und automatisierte Analyse in x64dbg suchen.
