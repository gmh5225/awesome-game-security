Un plugin d'assistant de reverse engineering propulsé par l'IA pour x64dbg, qui intègre un panneau de chat WebView2 directement dans le débogueur, avec des capacités d'utilisation d'outils LLM via les API Anthropic ou compatibles OpenAI.
L'agent IA peut lire la mémoire, désassembler le code, poser des points d'arrêt, exécuter pas à pas et interagir avec le débogueur de façon programmatique via une interface conversationnelle.
Il est surtout utile aux reverse engineers qui recherchent un débogage interactif assisté par l'IA et une analyse automatisée au sein de x64dbg.
