Questa è una libreria client Python di riferimento per automatizzare il debugger x64dbg tramite il protocollo RPC di x64dbg Automate. Usa ZeroMQ con serializzazione msgpack per comandi sincroni ed eventi di debug asincroni, avvolgendo le chiamate RPC di basso livello in API di più alto livello per breakpoint, accesso a memoria e registri, assembly e disassembly, controllo di sessione e operazioni GUI. Il pacchetto include anche un server opzionale Model Context Protocol (MCP) così che agenti basati su LLM possano guidare sessioni x64dbg. I casi d'uso principali sono l'analisi di malware, il reverse engineering e la ricerca di vulnerabilità, quando serve un controllo del debugger basato su script o assistito da agenti.
