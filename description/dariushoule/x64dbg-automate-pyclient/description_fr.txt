Il s'agit d'une bibliothèque cliente Python de référence pour automatiser le débogueur x64dbg via le protocole RPC x64dbg Automate. Elle utilise ZeroMQ avec une sérialisation msgpack pour les commandes synchrones et les événements de débogage asynchrones, en encapsulant les appels RPC de bas niveau dans des API de plus haut niveau pour les points d'arrêt, l'accès à la mémoire et aux registres, l'assemblage et le désassemblage, le contrôle de session et les opérations GUI. Le package inclut également un serveur Model Context Protocol (MCP) optionnel permettant aux agents basés sur des LLM de piloter des sessions x64dbg. Les cas d'usage principaux sont l'analyse de malwares, la rétro-ingénierie et la recherche de vulnérabilités, lorsque un contrôle scripté ou assisté par agent du débogueur est nécessaire.
