이는 x64dbg Automate RPC 프로토콜을 통해 x64dbg 디버거를 자동화하기 위한 참고용 Python 클라이언트 라이브러리입니다. ZeroMQ와 msgpack 직렬화를 사용해 동기 명령과 비동기 디버그 이벤트를 처리하며, 하위 수준 RPC 호출을 중단점, 메모리 및 레지스터 접근, 어셈블리와 디스어셈블리, 세션 제어, GUI 작업을 위한 상위 수준 API로 감쌉니다. 패키지에는 LLM 기반 에이전트가 x64dbg 세션을 제어할 수 있도록 선택적 Model Context Protocol(MCP) 서버도 포함됩니다. 주요 사용 사례는 스크립트 또는 에이전트 지원 디버거 제어가 필요한 악성코드 분석, 리버스 엔지니어링, 취약점 탐구입니다.
