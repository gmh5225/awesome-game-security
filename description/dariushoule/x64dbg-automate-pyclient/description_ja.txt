これは、x64dbg Automate RPC プロトコル経由で x64dbg デバッガを自動化するための参考 Python クライアントライブラリです。ZeroMQ と msgpack シリアライズを用いて同期コマンドと非同期デバッグイベントを扱い、ブレークポイント、メモリおよびレジスタアクセス、アセンブリとディスアセンブリ、セッション制御、GUI 操作向けの高レベル API で低レベル RPC 呼び出しをラップします。パッケージには、LLM ベースのエージェントが x64dbg セッションを操作できるオプションの Model Context Protocol（MCP）サーバーも同梱されています。主な用途は、スクリプト化またはエージェント支援によるデバッガ制御が必要なマルウェア解析、リバースエンジニアリング、脆弱性調査です。
