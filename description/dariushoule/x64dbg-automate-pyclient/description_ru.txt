Это эталонная клиентская библиотека на Python для автоматизации отладчика x64dbg через протокол RPC x64dbg Automate. Она использует ZeroMQ с сериализацией msgpack для синхронных команд и асинхронных событий отладки, оборачивая низкоуровневые RPC-вызовы в API более высокого уровня для точек останова, доступа к памяти и регистрам, ассемблирования и дизассемблирования, управления сеансом и операций GUI. Пакет также поставляется с необязательным сервером Model Context Protocol (MCP), чтобы агенты на базе LLM могли управлять сеансами x64dbg. Основные сценарии применения — анализ вредоносного ПО, реверс-инжиниринг и поиск уязвимостей, где нужен скриптовый или агентный контроль отладчика.
