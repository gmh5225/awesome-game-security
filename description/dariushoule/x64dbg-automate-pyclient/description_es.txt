Esta es una biblioteca cliente de referencia en Python para automatizar el depurador x64dbg mediante el protocolo RPC de x64dbg Automate. Usa ZeroMQ con serialización msgpack para comandos síncronos y eventos de depuración asíncronos, encapsulando las llamadas RPC de bajo nivel en APIs de más alto nivel para puntos de interrupción, acceso a memoria y registros, ensamblado y desensamblado, control de sesión y operaciones de GUI. El paquete también incluye un servidor opcional de Model Context Protocol (MCP) para que agentes basados en LLM puedan dirigir sesiones de x64dbg. Los casos de uso principales son el análisis de malware, la ingeniería inversa y la búsqueda de vulnerabilidades, cuando se necesita control del depurador mediante scripts o con asistencia de agentes.
