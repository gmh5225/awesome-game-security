Этот проект представляет собой цепочку эксплойтов для чтения/записи ядра, нацеленную на устройства Apple с поддержкой Neural Engine. Он объединяет несколько уязвимостей в компонентах, связанных с ANE, и документирует практические ограничения, диапазоны протестированных устройств и заметки о надёжности эксплуатации. Репозиторий сочетает код на Objective-C, C и C с каркасом iOS-приложения и низкоуровневыми вспомогательными средствами взаимодействия с IOKit и IOSurface. Он в первую очередь предназначен для исследователей безопасности iOS и macOS, анализирующих многоэтапные техники эксплуатации ядра.
