Этот проект — минимальная база для хукинга DirectX 11 на C++, которая перехватывает функцию IDXGISwapChain::Present для отрисовки пользовательских оверлеев в приложениях D3D11. Он показывает создание фиктивного устройства D3D11 для поиска vtable, хук Present через trampoline и отрисовку меню ImGui в перехваченном кадре. Проект служит стартовым шаблоном для создания внутренних игровых оверлеев. Он рассчитан на game hackers, изучающих отрисовку оверлеев и разработку чит-меню на основе хуков DirectX 11.
