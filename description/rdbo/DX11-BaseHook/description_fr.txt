Ce projet est une base minimale de hooking DirectX 11 en C++ qui intercepte la fonction IDXGISwapChain::Present pour afficher des overlays personnalisés dans les applications D3D11. Il montre comment créer un périphérique D3D11 factice pour localiser la vtable, accrocher Present avec un trampoline, et rendre des menus ImGui dans la frame accrochée. Le projet sert de modèle de départ pour construire des overlays de jeu internes. Il s'adresse aux game hackers qui apprennent le rendu d'overlays et le développement de menus de cheat basés sur le hooking DirectX 11.
