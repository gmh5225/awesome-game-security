Este proyecto es una base mínima de hooking de DirectX 11 en C++ que intercepta la función IDXGISwapChain::Present para renderizar overlays personalizados dentro de aplicaciones D3D11. Demuestra cómo crear un dispositivo D3D11 ficticio para localizar la vtable, enganchar Present con un trampoline y renderizar menús ImGui dentro del frame enganchado. El proyecto sirve como plantilla de partida para construir overlays internos de juegos. Está dirigido a game hackers que aprenden el renderizado de overlays y el desarrollo de menús de cheat basados en hooks de DirectX 11.
