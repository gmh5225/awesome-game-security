libmem es una biblioteca avanzada de game hacking multiplataforma para leer y manipular la memoria de procesos en Windows, Linux y FreeBSD. Escrita principalmente en C y C++, con bindings oficiales para Rust, Python y Lua, admite flujos de trabajo internos y externos. Sus capacidades principales incluyen enumerar procesos, módulos, símbolos, hilos y segmentos de memoria; leer, escribir, asignar y proteger memoria; escaneo de patrones y firmas; y resolución de mapas de punteros. También ofrece hooking de funciones y VMT, ensamblado y desensamblado JIT mediante Capstone y Keystone, y ayudas para cargar y descargar módulos. La biblioteca está orientada a game hackers, ingenieros inversos e investigadores de seguridad que necesitan una API unificada para la introspección de procesos y la manipulación en tiempo de ejecución en las plataformas de escritorio habituales.
