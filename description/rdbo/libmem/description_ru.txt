libmem — кроссплатформенная продвинутая библиотека для game hacking, предназначенная для чтения и манипуляции памятью процессов в Windows, Linux и FreeBSD. Написана в основном на C и C++, с официальными привязками для Rust, Python и Lua, и поддерживает как внутренние, так и внешние сценарии работы. Основные возможности включают перечисление процессов, модулей, символов, потоков и сегментов памяти; чтение, запись, выделение и защиту памяти; поиск по паттернам и сигнатурам; а также разрешение карт указателей. Также предоставляет хукинг функций и VMT, JIT-ассемблирование и дизассемблирование через Capstone и Keystone и вспомогательные средства загрузки и выгрузки модулей. Библиотека ориентирована на game hackers, reverse engineers и исследователей безопасности, которым нужен единый API для интроспекции процессов и манипуляций во время выполнения на распространённых настольных платформах.
