libmem ist eine plattformübergreifende, fortschrittliche Game-Hacking-Bibliothek zum Lesen und Manipulieren von Prozessspeicher unter Windows, Linux und FreeBSD. Sie ist hauptsächlich in C und C++ geschrieben, bietet offizielle Bindings für Rust, Python und Lua und unterstützt sowohl interne als auch externe Workflows. Zu den Kernfunktionen gehören das Auflisten von Prozessen, Modulen, Symbolen, Threads und Speichersegmenten; das Lesen, Schreiben, Allokieren und Schützen von Speicher; Pattern- und Signature-Scanning sowie das Auflösen von Pointer-Maps. Zudem stellt sie Function- und VMT-Hooking, JIT-Assemblierung und -Disassemblierung über Capstone und Keystone sowie Hilfen zum Laden und Entladen von Modulen bereit. Die Bibliothek richtet sich an Game Hacker, Reverse Engineers und Sicherheitsforscher, die eine einheitliche API für Prozessintrospection und Laufzeitmanipulation auf gängigen Desktop-Plattformen benötigen.
