libmem is a cross-platform advanced game hacking library for reading and manipulating process memory on Windows, Linux, and FreeBSD. Written primarily in C and C++ with official bindings for Rust, Python, and Lua, it supports both internal and external workflows. Core capabilities include enumerating processes, modules, symbols, threads, and memory segments; reading, writing, allocating, and protecting memory; pattern and signature scanning; and resolving pointer maps. It also provides function and VMT hooking, JIT assemble and disassemble via Capstone and Keystone, and module load and unload helpers. The library targets game hackers, reverse engineers, and security researchers who need a unified API for process introspection and runtime manipulation across common desktop platforms.
