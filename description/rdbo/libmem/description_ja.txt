libmem は、Windows、Linux、FreeBSD 上でプロセスメモリの読み取りと操作を行うクロスプラットフォームの高度なゲームハッキングライブラリです。主に C と C++ で書かれ、Rust、Python、Lua 向けの公式バインディングを備え、内部・外部の両方のワークフローに対応します。主な機能には、プロセス、モジュール、シンボル、スレッド、メモリセグメントの列挙、メモリの読み書き・割り当て・保護、パターンおよびシグネチャスキャン、ポインタマップの解決が含まれます。また、関数および VMT フック、Capstone と Keystone による JIT アセンブル／ディスアセンブル、モジュールのロード／アンロード補助も提供します。一般的なデスクトップ環境でプロセス調査と実行時操作を統一 API で行いたいゲームハッカー、リバースエンジニア、セキュリティ研究者を対象としています。
