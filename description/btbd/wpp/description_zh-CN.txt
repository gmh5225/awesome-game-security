该项目是一个概念验证，通过劫持 Windows WPP（Windows Software Trace Preprocessor）跟踪基础设施来拦截驱动的 DeviceControl 调用。
它会修改 disk.sys、mountmgr.sys 等驱动中的 WPP 跟踪函数指针与控制标志，再通过返回地址检查以及基于栈回溯或寄存器捕获的 IRP 指针提取，识别 DeviceControl 调用。
该内核驱动展示了如何将系统驱动 .data 段中的 WPP 指针重新用于 DeviceControl 拦截。
它主要面向研究非常规驱动挂钩技术，以及通过磁盘序列号拦截进行 HWID 欺骗的内核安全研究人员。
