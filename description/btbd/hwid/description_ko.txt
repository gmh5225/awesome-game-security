이 프로젝트는 커널 수준에서 디스크, 볼륨, NIC, ARP, SMBIOS, 부트 및 GPU 식별자를 수정하는 Windows용 하드웨어 ID 스푸퍼입니다.
커널 드라이버는 IOCTL 응답을 가로채 변조하여 위장된 하드웨어 일련번호와 식별자를 반환하고, 사용자 모드 구성 요소는 레지스트리 키와 일반적인 추적 파일을 처리합니다.
Windows 10 1507부터 1903까지의 버전에서 테스트되었으며 x64 시스템을 대상으로 하지만, NVME 전용 IOCTL은 처리하지 않습니다.
하드웨어 핑거프린팅 기법과 HWID 기반 제재 우회 방법을 연구하는 게임 보안 연구자에게 주로 유용합니다.
