このプロジェクトは、カーネルレベルでディスク、ボリューム、NIC、ARP、SMBIOS、ブート、GPU の識別子を改変する Windows 向けハードウェア ID スプーファーです。
カーネルドライバが IOCTL 応答を傍受・改変して偽装されたハードウェアシリアル番号と識別子を返し、ユーザーモード側がレジストリキーや一般的な追跡ファイルを処理します。
Windows 10 の 1507 から 1903 までのバージョンでテストされ、x64 システムを対象としますが、NVME 固有の IOCTL には対応していません。
ハードウェア指紋技術や HWID ベースの BAN 回避手法を研究するゲームセキュリティ研究者にとって主に有用です。
