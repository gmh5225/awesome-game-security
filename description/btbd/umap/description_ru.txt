Этот проект — umap, минималистичный маппер драйверов ядра Windows на C, который вручную отображает неподписанный драйвер в память ядра. Он использует уязвимый подписанный драйвер для доступа к физической памяти, чтобы выделить место в пуле ядра, скопировать секции драйвера, обработать релокации, разрешить импорты и вызвать точку входа отображённого драйвера полностью из пользовательского режима. Маппер не оставляет следов в реестре и не загружается через стандартные пути загрузки драйверов. Он предназначен для исследователей ядра, изучающих скрытные техники маппинга драйверов и векторы их обнаружения.
