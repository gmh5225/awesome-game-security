Este proyecto es una prueba de concepto sobre el uso de dispositivos de disco (controladores HBA) para realizar operaciones DMA en Windows, eludiendo protecciones de memoria a nivel de hipervisor.
Demuestra cómo hipervisores nativos como Hyper-V, que permiten acceso no virtualizado a dispositivos, habilitan la elusión de SLAT (Second Level Address Translation) mediante DMA de dispositivos de disco ATA.
El controlador de kernel se demostró modificando Hyper-V en tiempo de ejecución sobre bare metal, aunque solo admite ATA y puede verse limitado por las capacidades de direccionamiento de 64 bits del HBA.
Resulta especialmente útil para investigadores de seguridad de kernel e hipervisores que estudian ataques DMA basados en disco y técnicas de bypass de SLAT.
