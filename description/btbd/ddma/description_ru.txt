Этот проект — proof-of-concept использования дисковых устройств (контроллеров HBA) для выполнения DMA-операций в Windows с обходом защиты памяти на уровне гипервизора.
Он показывает, как нативные гипервизоры вроде Hyper-V, допускающие невиртуализированный доступ к устройствам, позволяют обходить SLAT (Second Level Address Translation) через DMA дисковых устройств ATA.
Драйвер ядра демонстрировался с изменением Hyper-V во время выполнения на bare metal, однако поддерживает только ATA и может быть ограничен возможностями 64-битной адресации HBA.
В основном полезен исследователям безопасности ядра и гипервизоров, изучающим дисковые DMA-атаки и техники обхода SLAT.
