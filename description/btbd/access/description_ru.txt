Этот проект — обёртка системных вызовов в режиме ядра, позволяющая выполнять операции PROCESS_ALL_ACCESS над защищёнными процессами без создания реальных handle.
Она перехватывает системный вызов через изменение секции .data в ядре и предоставляет пользовательскую DLL-обёртку, прозрачно перенаправляющую привилегированные операции через драйвер.
Реализация избегает структурированной обработки исключений (SEH), сохраняя при этом безопасные операции, и протестирована на защищённых игровых процессах вроде Fortnite.
Проект в основном полезен исследователям игровой безопасности, изучающим доступ к процессам в режиме ядра без handle и техники перехвата системных вызовов для обхода защиты процессов.
