이 프로젝트는 실제 핸들을 생성하지 않고도 보호된 프로세스에 대해 PROCESS_ALL_ACCESS 작업을 가능하게 하는 커널 모드 시스템 호출 래퍼입니다.
커널의 .data 섹션을 수정하여 시스템 호출을 후킹하고, 특권 작업을 드라이버를 통해 투명하게 리다이렉트하는 사용자 모드 DLL 래퍼를 제공합니다.
구현은 구조적 예외 처리(SEH)를 피하면서도 안전한 작업을 수행하며, Fortnite와 같은 보호된 게임 프로세스에 대해 테스트되었습니다.
핸들 없는 커널 모드 프로세스 접근과 프로세스 보호를 우회하는 시스템 호출 후킹 기법을 연구하는 게임 보안 연구자에게 주로 유용합니다.
