AutoHotkey es una plataforma de código abierto de automatización y scripts de macros para Windows, con un lenguaje personalizado centrado en hotkeys.
El repositorio contiene principalmente código fuente en C y C++ para construir binarios del intérprete, runtimes de scripts autocontenidos y herramientas relacionadas.
También incluye una interfaz experimental de alojamiento DLL para incrustar el intérprete en otras aplicaciones.
Su caso de uso principal es la automatización de tareas y el scripting de entrada, incluida la experimentación con macros relacionadas con juegos y el análisis defensivo de comportamientos impulsados por scripts.
