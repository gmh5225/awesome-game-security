AutoHotkey est une plateforme open source d'automatisation et de scripts macros pour Windows, munie d'un langage personnalisé centré sur les raccourcis clavier.
Le dépôt contient surtout du code source C et C++ pour construire les binaires de l'interpréteur, des runtimes de scripts autonomes et les outils associés.
Il inclut aussi une interface expérimentale d'hébergement DLL pour intégrer l'interpréteur dans d'autres applications.
Son usage principal est l'automatisation de tâches et le scripting d'entrées, y compris l'expérimentation de macros liées aux jeux et l'analyse défensive de comportements pilotés par scripts.
