AutoHotkey — это открытая платформа автоматизации и макросов для Windows с собственным языком, ориентированным на горячие клавиши.
Репозиторий в основном содержит исходный код на C и C++ для сборки бинарников интерпретатора, автономных сред выполнения скриптов и связанных инструментов.
Также включён экспериментальный DLL-интерфейс хостинга для встраивания интерпретатора в другие приложения.
Основное применение — автоматизация задач и скриптинг ввода, включая эксперименты с игровыми макросами и защитный анализ поведения, управляемого скриптами.
