AutoHotkey è una piattaforma open source di automazione e scripting di macro per Windows, con un linguaggio personalizzato incentrato sugli hotkey.
Il repository contiene soprattutto codice sorgente C e C++ per costruire i binari dell'interprete, runtime di script autonomi e gli strumenti correlati.
Include anche un'interfaccia sperimentale di hosting DLL per incorporare l'interprete in altre applicazioni.
Il caso d'uso principale è l'automazione di attività e lo scripting degli input, inclusi esperimenti di macro legati ai giochi e l'analisi difensiva di comportamenti guidati da script.
