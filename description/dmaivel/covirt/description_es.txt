Un virtualizador de código x86-64 que transforma instrucciones nativas en una arquitectura de máquina virtual basada en pila para ofuscación basada en VM, con soporte para binarios PE (MinGW) y ELF.
Implementa transformaciones de aritmética booleana mixta (MBA) y pases de ofuscación con código auto-modificable, junto con marcadores de código para definir regiones protegidas en el binario objetivo.
Resulta especialmente útil para investigadores de seguridad que estudian técnicas de ofuscación basadas en VM y construyen esquemas personalizados de protección binaria.
