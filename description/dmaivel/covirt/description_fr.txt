Un virtualiseur de code x86-64 qui transforme les instructions natives en une architecture de machine virtuelle à pile pour l'obfuscation basée sur une VM, avec prise en charge des binaires PE (MinGW) et ELF.
Il implémente des transformations d'arithmétique booléenne mixte (MBA) et des passes d'obfuscation par code auto-modifiant, avec des marqueurs de code pour définir les régions protégées dans le binaire cible.
Il est surtout utile aux chercheurs en sécurité qui étudient les techniques d'obfuscation basées sur une VM et conçoivent des schémas personnalisés de protection binaire.
