Отладчик ядра Windows для хостов Linux, предназначенный для отладки гостевых систем Windows 10/11 под KVM/QEMU; предоставляет команды в стиле WinDbg, загрузку/разбор PDB-символов и поддержку точек останова.
Он подключается к гипервизору KVM через GDB stub для доступа к памяти и работы с регистрами, по сути выступая заменой WinDbg в средах виртуализации на базе Linux.
В первую очередь полезен исследователям безопасности, выполняющим отладку и интроспекцию ядра Windows с Linux без необходимости в Windows-хосте или WinDbg.
