Un debugger del kernel Windows per host Linux che esegue il debug di guest Windows 10/11 sotto KVM/QEMU, offrendo comandi in stile WinDbg, recupero/parsing dei simboli PDB e supporto per i breakpoint.
Si connette tramite GDB stub all'hypervisor KVM per l'accesso alla memoria e la manipolazione dei registri, fungendo essenzialmente da sostituto di WinDbg per ambienti di virtualizzazione basati su Linux.
È principalmente utile ai ricercatori di sicurezza che effettuano debug e introspezione del kernel Windows da Linux, senza richiedere un host Windows o WinDbg.
