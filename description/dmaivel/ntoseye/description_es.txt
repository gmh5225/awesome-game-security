Un depurador del kernel de Windows para hosts Linux que depura invitados Windows 10/11 en KVM/QEMU, con comandos al estilo de WinDbg, obtención/análisis de símbolos PDB y soporte de puntos de interrupción.
Se conecta mediante el stub GDB al hipervisor KVM para acceso a memoria y manipulación de registros, y funciona esencialmente como un reemplazo de WinDbg en entornos de virtualización basados en Linux.
Resulta especialmente útil para investigadores de seguridad que realizan depuración e introspección del kernel de Windows desde Linux, sin necesitar un host Windows ni WinDbg.
