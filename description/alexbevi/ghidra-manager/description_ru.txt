Ghidra Manager — кроссплатформенный Python CLI, который устанавливает Ghidra и собирает подобранный набор расширений под точный выпуск. Он отслеживает стабильные релизы на GitHub, проверяет загружаемые файлы по опубликованным SHA-256 дайджестам и хранит активную установку вместе с полной парой для отката. Инструмент управляет плагинами из неизменяемых коммитов релизов, находит и запускает проекты Ghidra, а также предоставляет рабочие процессы MCP-моста, мультиэкземплярного запуска, проверок doctor и сравнения двоичных файлов. Он рассчитан на инженеров по реверс-инжинирингу и аналитиков безопасности, которым нужна воспроизводимая и версионно безопасная цепочка инструментов Ghidra на Windows, Linux и macOS.
