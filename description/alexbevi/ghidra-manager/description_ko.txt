Ghidra Manager는 Ghidra를 설치하고 해당 정확한 릴리스에 맞춘 엄선된 확장 세트를 구축하는 크로스 플랫폼 Python CLI입니다. 안정적인 GitHub 릴리스를 추적하고, 게시된 SHA-256 다이제스트로 다운로드 자산을 검증하며, 활성 설치와 완전한 롤백 쌍 하나를 유지합니다. 불변 릴리스 커밋에서 플러그인을 관리하고, Ghidra 프로젝트를 검색·실행하며, MCP 브리지, 다중 인스턴스 실행, doctor 검사, 바이너리 비교 워크플로를 제공합니다. Windows, Linux, macOS에서 재현 가능하고 버전 안전한 Ghidra 툴체인이 필요한 리버스 엔지니어와 보안 분석가를 대상으로 합니다.
