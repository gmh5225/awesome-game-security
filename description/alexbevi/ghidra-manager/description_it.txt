Ghidra Manager è una CLI Python multipiattaforma che installa Ghidra e crea un set curato di estensioni abbinato a quella specifica release. Tiene traccia delle release stabili su GitHub, verifica gli asset scaricati con i digest SHA-256 pubblicati e conserva l'installazione attiva più una coppia completa di rollback. Lo strumento gestisce i plugin da commit di release immutabili, scopre e avvia progetti Ghidra ed espone workflow di bridge MCP, avvio multi-istanza, controlli doctor e confronto binario. Si rivolge a reverse engineer e analisti di sicurezza che necessitano di una toolchain Ghidra ripetibile e sicura a livello di versione su Windows, Linux e macOS.
