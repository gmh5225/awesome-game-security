Ghidra Manager es una CLI de Python multiplataforma que instala Ghidra y construye un conjunto curado de extensiones adaptadas a esa versión exacta. Realiza un seguimiento de las versiones estables de GitHub, verifica los archivos descargados con los digests SHA-256 publicados y conserva la instalación activa más un par completo de rollback. La herramienta gestiona plugins desde commits de release inmutables, descubre y lanza proyectos de Ghidra, y expone flujos de trabajo de puente MCP, lanzamiento multiinstancia, comprobaciones doctor y comparación binaria. Está orientada a ingenieros de ingeniería inversa y analistas de seguridad que necesitan una cadena de herramientas Ghidra repetible y segura en versión en Windows, Linux y macOS.
