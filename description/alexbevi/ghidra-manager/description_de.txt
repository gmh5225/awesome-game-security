Ghidra Manager ist eine plattformübergreifende Python-CLI, die Ghidra installiert und einen kuratierten Satz von Erweiterungen passend zu genau diesem Release erstellt. Sie verfolgt stabile GitHub-Releases, prüft heruntergeladene Assets anhand veröffentlichter SHA-256-Digests und behält die aktive Installation sowie ein vollständiges Rollback-Paar. Das Tool verwaltet Plugins aus unveränderlichen Release-Commits, findet und startet Ghidra-Projekte und bietet MCP-Bridge-, Multi-Instanz-Start-, Doctor-Prüfungs- und Binärvergleichs-Workflows. Es richtet sich an Reverse Engineers und Sicherheitsanalysten, die eine wiederholbare, versionssichere Ghidra-Toolchain unter Windows, Linux und macOS benötigen.
