Ghidra Manager は、Ghidra をインストールし、その正確なリリースに合わせた厳選拡張セットを構築するクロスプラットフォームの Python CLI です。安定版の GitHub リリースを追跡し、公開された SHA-256 ダイジェストでダウンロード資産を検証し、アクティブなインストールと完全なロールバック用の一組を保持します。不変なリリースコミットからプラグインを管理し、Ghidra プロジェクトの検出と起動を行い、MCP ブリッジ、マルチインスタンス起動、doctor チェック、バイナリ比較のワークフローを提供します。Windows、Linux、macOS で再現可能かつバージョン安全な Ghidra ツールチェーンを必要とするリバースエンジニアとセキュリティアナリストを対象としています。
