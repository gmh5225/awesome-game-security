This project is a GTA IV RTX Remix compatibility mod that adapts The Complete Edition for NVIDIA's RTX Remix path-traced remaster pipeline.
It ships a custom Remix runtime, Ultimate ASI Loader integration, light translation, dynamic emissives, and wetness or rain effects, and complements FusionFix and related PBR asset packs.
It is mainly useful for graphics programmers and Windows game tooling developers working in the directx / compatibility area.
