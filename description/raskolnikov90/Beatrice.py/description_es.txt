Este proyecto es Beatrice.py, una herramienta en Python para modificar código máquina x86-64 sustituyendo instrucciones por codificaciones alternativas semánticamente equivalentes. Reemplaza los opcodes originales por secuencias de instrucciones funcionalmente idénticas pero distintas a nivel de bytes, para evadir la detección basada en firmas de antivirus y anti-trampas. La herramienta opera a nivel binario mediante coincidencia de patrones y reglas de sustitución a nivel de instrucción. Está dirigida a investigadores de seguridad que estudian la mutación binaria a nivel de instrucción para la evasión de AV y la diversidad binaria.
