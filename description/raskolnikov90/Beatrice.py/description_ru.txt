Этот проект — Beatrice.py, инструмент на Python для изменения машинного кода x86-64 путём замены инструкций семантически эквивалентными альтернативными кодировками. Он заменяет исходные опкоды функционально идентичными, но отличающимися по байтам последовательностями инструкций, чтобы обходить сигнатурное обнаружение антивирусами и античитами. Инструмент работает на уровне бинарного кода с использованием сопоставления шаблонов и правил подстановки на уровне инструкций. Он предназначен для исследователей безопасности, изучающих мутацию бинарного кода на уровне инструкций для обхода AV и бинарного разнообразия.
