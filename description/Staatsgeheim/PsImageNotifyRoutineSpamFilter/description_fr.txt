Ce projet est un utilitaire noyau Windows qui filtre les événements de rappel PsImageNotifyRoutine bruyants.
Il utilise le parcours de pile avec RtlWalkFrameChain pour distinguer les notifications de chargement d'image pertinentes des sources de bruit de fond courantes.
L'implémentation est en C pour le développement de pilotes Windows 64 bits et illustre une hygiène pratique des rappels.
Il est utile pour la surveillance noyau, la collecte de télémétrie anti-cheat et une analyse d'événements plus claire côté pilote.
