Этот проект — утилита ядра Windows, которая фильтрует шумные события обратного вызова PsImageNotifyRoutine.
Она использует обход стека через RtlWalkFrameChain, чтобы отличать значимые уведомления о загрузке образов от типичных источников фонового шума.
Реализация написана на C для разработки 64-битных драйверов Windows и демонстрирует практическую гигиену callback'ов.
Полезна для мониторинга ядра, сбора телеметрии античита и более чистого анализа событий на стороне драйвера.
