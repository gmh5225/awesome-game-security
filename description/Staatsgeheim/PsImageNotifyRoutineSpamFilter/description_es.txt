Este proyecto es una utilidad del kernel de Windows que filtra eventos de devolución de llamada PsImageNotifyRoutine ruidosos.
Usa recorrido de pila con RtlWalkFrameChain para distinguir notificaciones significativas de carga de imagen de fuentes habituales de ruido de fondo.
La implementación está en C para el desarrollo de controladores de Windows de 64 bits y demuestra una higiene práctica de callbacks.
Es útil para la supervisión del kernel, la recopilación de telemetría anti-cheat y un análisis de eventos más limpio en el lado del controlador.
