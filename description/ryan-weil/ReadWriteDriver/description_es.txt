Este proyecto se centra en ntUserSetSysColors. Tenga en cuenta que las direcciones de las funciones están actualmente codificadas de forma fija para el kernel de Windows 11 10.0.22000.376. Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de comunicación entre cheats y controladores.
