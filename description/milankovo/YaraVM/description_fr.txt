Module processeur et chargeur IDA Pro pour désassembler les binaires de règles YARA compilées (.yar.bin), permettant l'analyse statique du bytecode YARA et du bytecode d'expressions régulières dans l'environnement IDA. Analyse le format de fichier YARA basé sur une arena (espaces de noms, règles, chaînes, sections de code, tables de transition AC) et fournit une bibliothèque d'informations de types personnalisée (libyara.til) pour l'annotation des structures.
