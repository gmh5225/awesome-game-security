Модуль процессора и загрузчик IDA Pro для дизассемблирования скомпилированных бинарных файлов правил YARA (.yar.bin), обеспечивающий статический анализ байткода YARA и байткода регулярных выражений в среде IDA. Разбирает формат файлов YARA на основе arena (пространства имён, правила, строки, секции кода, таблицы переходов AC) и предоставляет пользовательскую библиотеку типовой информации (libyara.til) для аннотации структур.
