Ce projet est un plugin IDA Pro 9.x qui apporte à IDA une recherche binaire consciente des types dans le style de 010 Editor.
Le manifeste ida-plugin.json indique à IDA de charger plugin.py au démarrage.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
