Dieses Projekt ist ein IDA Pro 9.x-Plugin, das typsensitive Binärsuche im Stil von 010 Editor nach IDA bringt.
Das Manifest ida-plugin.json weist IDA an, plugin.py beim Start zu laden.
Es ist vor allem nützlich für Spielsicherheitsforscher und Reverse Engineers, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
