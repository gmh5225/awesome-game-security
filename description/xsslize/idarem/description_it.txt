idarem è un plugin per IDA Pro che consente di consultare un database IDA live da un browser su un altro dispositivo. Un server IDAPython/Flask all’interno di IDA espone API REST e Server-Sent Events per funzioni, disassembly, grafi, pseudocodice Hex-Rays, riferimenti incrociati, stringhe, import e viste correlate, mentre un client React e TypeScript le visualizza in remoto. Supporta il follow live dell’indirizzo e della finestra correnti di IDA, la navigazione opzionale di ritorno in IDA e la scrittura opzionale di rinomine e commenti quando abilitata esplicitamente. L’accesso è autenticato con token ed è pensato per l’uso tramite tunnel come Tailscale, Cloudflare Tunnel o ngrok. Il caso d’uso principale riguarda i reverse engineer che vogliono esplorare da remoto l’analisi IDA in modo leggero, senza RDP, VNC o copiare il database.
