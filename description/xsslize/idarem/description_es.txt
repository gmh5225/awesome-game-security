idarem es un plugin de IDA Pro que permite revisar una base de datos de IDA en vivo desde un navegador en otro dispositivo. Un servidor IDAPython/Flask dentro de IDA expone APIs REST y Server-Sent Events para funciones, desensamblado, grafos, pseudocódigo de Hex-Rays, referencias cruzadas, cadenas, importaciones y vistas relacionadas, mientras un cliente de React y TypeScript las renderiza de forma remota. Admite el seguimiento en vivo de la dirección y la ventana actuales de IDA, navegación opcional de vuelta hacia IDA y escritura opcional de renombres y comentarios cuando se habilita explícitamente. El acceso se autentica con token y está pensado para usarse a través de túneles como Tailscale, Cloudflare Tunnel o ngrok. El caso de uso principal son los ingenieros inversos que quieren explorar a distancia el análisis de IDA de forma ligera, sin RDP, VNC ni copiar la base de datos.
