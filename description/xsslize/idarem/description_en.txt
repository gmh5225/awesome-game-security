idarem is an IDA Pro plugin that lets you review a live IDA database from a browser on another device. An IDAPython/Flask server inside IDA exposes REST and Server-Sent Events APIs for functions, disassembly, graphs, Hex-Rays pseudocode, cross-references, strings, imports, and related views, while a React and TypeScript client renders them remotely. It supports live follow of IDA’s current address and window, optional drive-back navigation into IDA, and optional write-back for renames and comments when explicitly enabled. Access is token-authenticated and intended for use over tunnels such as Tailscale, Cloudflare Tunnel, or ngrok. The primary use case is reverse engineers who want lightweight remote browsing of IDA analysis without RDP, VNC, or copying the database.
