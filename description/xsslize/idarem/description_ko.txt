idarem은 다른 기기의 브라우저에서 라이브 IDA 데이터베이스를 살펴볼 수 있게 해 주는 IDA Pro 플러그인입니다. IDA 내부의 IDAPython/Flask 서버가 함수, 디스어셈블리, 그래프, Hex-Rays 의사코드, 교차 참조, 문자열, 임포트 및 관련 뷰를 위한 REST와 Server-Sent Events API를 제공하고, React와 TypeScript 클라이언트가 이를 원격으로 렌더링합니다. IDA의 현재 주소와 창을 실시간으로 따라가는 기능, 선택적으로 IDA로 되돌아가는 탐색, 명시적으로 활성화했을 때의 이름 변경·주석 쓰기 되돌리기를 지원합니다. 접근은 토큰 인증이며 Tailscale, Cloudflare Tunnel, ngrok 같은 터널을 통한 사용을 전제로 합니다. 주요 사용 사례는 RDP, VNC나 데이터베이스 복사 없이 IDA 분석을 가볍게 원격으로 보고 싶은 리버스 엔지니어입니다.
