idarem は、別のデバイスのブラウザからライブの IDA データベースを確認できる IDA Pro プラグインです。IDA 内の IDAPython/Flask サーバーが、関数、逆アセンブル、グラフ、Hex-Rays 疑似コード、クロス参照、文字列、インポート、および関連ビュー向けに REST と Server-Sent Events API を公開し、React と TypeScript のクライアントがそれらをリモートで描画します。IDA の現在のアドレスとウィンドウへのライブ追従、任意で IDA 側へ戻すナビゲート、明示的に有効化したときのリネームやコメントの書き戻しをサポートします。アクセスはトークン認証で、Tailscale、Cloudflare Tunnel、ngrok などのトンネル経由での利用を想定しています。主な用途は、RDP、VNC、データベースのコピーなしで IDA 解析を軽量にリモート閲覧したいリバースエンジニア向けです。
