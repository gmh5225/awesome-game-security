idarem — плагин для IDA Pro, позволяющий просматривать живую базу IDA в браузере на другом устройстве. Сервер IDAPython/Flask внутри IDA предоставляет REST и Server-Sent Events API для функций, дизассемблирования, графов, псевдокода Hex-Rays, перекрёстных ссылок, строк, импортов и связанных представлений, а клиент на React и TypeScript отображает их удалённо. Поддерживаются живое следование за текущим адресом и окном IDA, опциональная навигация обратно в IDA и опциональная запись переименований и комментариев при явном включении. Доступ аутентифицируется токеном и рассчитан на использование через туннели вроде Tailscale, Cloudflare Tunnel или ngrok. Основной сценарий — для reverse-инженеров, которым нужен лёгкий удалённый просмотр анализа IDA без RDP, VNC или копирования базы.
