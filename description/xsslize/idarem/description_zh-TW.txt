idarem 是一款 IDA Pro 外掛，可讓你在另一台裝置的瀏覽器中檢視即時 IDA 資料庫。IDA 內的 IDAPython/Flask 伺服器透過 REST 與 Server-Sent Events API 暴露函式、反組譯、圖表、Hex-Rays 偽程式碼、交叉參考、字串、匯入及相關檢視，而基於 React 與 TypeScript 的用戶端則在遠端呈現這些內容。它支援即時跟隨 IDA 目前位址與視窗、可選的回傳導覽以驅動 IDA，以及在明確啟用時對重新命名與註解的可選寫回。存取採用權杖驗證，並設計為透過 Tailscale、Cloudflare Tunnel 或 ngrok 等通道使用。主要使用情境是希望在不依賴 RDP、VNC 或複製資料庫的情況下，輕量遠端瀏覽 IDA 分析的逆向工程師。
