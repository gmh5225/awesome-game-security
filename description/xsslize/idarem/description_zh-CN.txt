idarem 是一款 IDA Pro 插件，可让你在另一台设备的浏览器中查看实时 IDA 数据库。IDA 内的 IDAPython/Flask 服务器通过 REST 与 Server-Sent Events API 暴露函数、反汇编、图表、Hex-Rays 伪代码、交叉引用、字符串、导入及相关视图，而基于 React 与 TypeScript 的客户端则在远程渲染这些内容。它支持实时跟随 IDA 当前地址与窗口，可选的回传导航以驱动 IDA，以及在显式启用时对重命名与注释的可选写回。访问采用令牌认证，并面向通过 Tailscale、Cloudflare Tunnel 或 ngrok 等隧道使用。主要面向希望在不依赖 RDP、VNC 或复制数据库的情况下，轻量远程浏览 IDA 分析的逆向工程师。
