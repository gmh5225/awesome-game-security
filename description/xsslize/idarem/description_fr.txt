idarem est un plugin IDA Pro qui permet de consulter une base de données IDA en direct depuis un navigateur sur un autre appareil. Un serveur IDAPython/Flask intégré à IDA expose des API REST et Server-Sent Events pour les fonctions, le désassemblage, les graphes, le pseudocode Hex-Rays, les références croisées, les chaînes, les imports et les vues associées, tandis qu’un client React et TypeScript les affiche à distance. Il prend en charge le suivi en direct de l’adresse et de la fenêtre courantes d’IDA, une navigation optionnelle de retour vers IDA, et une écriture optionnelle des renommages et commentaires lorsqu’elle est explicitement activée. L’accès est authentifié par jeton et destiné à un usage via des tunnels tels que Tailscale, Cloudflare Tunnel ou ngrok. Le cas d’usage principal concerne les reverse engineers qui souhaitent parcourir à distance l’analyse IDA de façon légère, sans RDP, VNC ni copie de la base.
