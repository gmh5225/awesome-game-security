hardware-breakpoint — проект ядра Linux для ARM64, реализующий настраиваемые аппаратные точки останова через экспортируемые API и интерфейсы proc.
Он поддерживает добавление и удаление точек останова выполнения или наблюдения по символу или адресу, вывод списка активных точек останова и сбор статистики срабатываний.
Проект также предоставляет утилиты для разрешения значений символов и сопоставления физических IO-адресов с соответствующими виртуальными отображениями.
Основной сценарий использования — отладка на уровне ядра и исследования безопасности на Android или встраиваемых системах, требующих детального мониторинга во время выполнения.
