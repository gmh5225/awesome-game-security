hardware-breakpoint は、エクスポートされた API と proc インターフェースを通じて設定可能なハードウェアブレークポイントを実装する、ARM64 向けの Linux カーネルプロジェクトです。
シンボルまたはアドレスによる実行ブレークポイント／ウォッチブレークポイントの追加・削除、有効なブレークポイントの一覧表示、トリガー統計の収集に対応しています。
シンボル値の解決や、物理 IO アドレスを関連する仮想マッピングへ対応付けるユーティリティも提供します。
主な用途は、細粒度のランタイム監視が必要な Android や組み込みシステムにおけるカーネルレベルのデバッグとセキュリティ研究です。
