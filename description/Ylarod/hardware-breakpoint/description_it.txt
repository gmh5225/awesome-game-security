hardware-breakpoint è un progetto del kernel Linux per ARM64 che implementa breakpoint hardware configurabili tramite API esportate e interfacce proc.
Supporta l'aggiunta e la rimozione di breakpoint di esecuzione o di watch per simbolo o indirizzo, l'elenco dei breakpoint attivi e la raccolta di statistiche di attivazione.
Il progetto fornisce anche utilità per risolvere i valori dei simboli e mappare gli indirizzi IO fisici sulle relative mappature virtuali.
Il caso d'uso principale è il debugging a livello kernel e la ricerca sulla sicurezza su Android o sistemi embedded che richiedono un monitoraggio runtime a grana fine.
