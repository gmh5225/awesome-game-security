hardware-breakpoint es un proyecto de kernel de Linux para ARM64 que implementa puntos de interrupción de hardware configurables mediante APIs exportadas e interfaces proc.
Admite añadir y eliminar puntos de interrupción de ejecución o de vigilancia por símbolo o dirección, listar los puntos de interrupción activos y recopilar estadísticas de activación.
El proyecto también ofrece utilidades para resolver valores de símbolos y asociar direcciones IO físicas con sus mapeos virtuales relacionados.
Su caso de uso principal es la depuración a nivel de kernel y la investigación de seguridad en Android o sistemas embebidos que requieren monitorización en tiempo de ejecución de grano fino.
