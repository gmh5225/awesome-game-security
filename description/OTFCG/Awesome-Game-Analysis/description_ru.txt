Этот проект — курируемый индекс в стиле awesome ресурсов по анализу игровых технологий. Репозиторий в основном на Markdown и организует ссылки по названию игры, движку, разработчику, году и технической теме. Он поддерживается как управляемая сообществом коллекция с руководством по вкладу для согласованных обновлений. Полезен программистам игр, инженерам рендеринга и reverse engineers, которым нужны справочные материалы по внутреннему устройству движков и производственным техникам.
