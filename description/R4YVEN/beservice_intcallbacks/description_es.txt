Este proyecto es un experimento de prueba de concepto para eludir BattleEye basado en el comportamiento de los instrumentation callbacks.
Está implementado en C++, con algo de ensamblador y archivos de proyecto de Visual Studio centrados en técnicas de callbacks y manejo de símbolos.
El repositorio parece documentar un enfoque exploratorio más que una herramienta pulida para el usuario final.
Su caso de uso principal es la investigación sobre elusión de anti-cheat y el estudio de los internos de Windows a bajo nivel.
