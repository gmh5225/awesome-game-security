Questo progetto è un esperimento proof-of-concept di bypass di BattleEye basato sul comportamento degli instrumentation callback.
È implementato in C++, con un po' di assembly e file di progetto Visual Studio incentrati su tecniche di callback e gestione dei simboli.
Il repository sembra documentare un approccio esplorativo piuttosto che uno strumento rifinito per l'utente finale.
Il suo caso d'uso principale è la ricerca sul bypass anti-cheat e lo studio degli interni di Windows a basso livello.
