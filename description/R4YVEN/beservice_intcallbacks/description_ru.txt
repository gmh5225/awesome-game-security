Этот проект — экспериментальный proof-of-concept обхода BattleEye, основанный на поведении instrumentation callback.
Он реализован на C++ с небольшим объёмом ассемблера и файлами проекта Visual Studio, с акцентом на техники работы с callback и символами.
Репозиторий, судя по всему, описывает исследовательский подход, а не законченный инструмент для конечных пользователей.
Основное применение — исследование обхода античита и изучение низкоуровневых внутренностей Windows.
