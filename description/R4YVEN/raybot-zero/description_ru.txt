Этот проект — чит для CS:GO в режиме ядра, в котором основная логика работает без традиционного контроллера в пользовательском режиме.
Он сочетает драйвер Windows на C++ с небольшим загрузчиком на C# и реализует такие функции, как триггербот, баннихоп, glow-визуалы и чтение состояния клавиш на уровне ядра.
Код опирается на игровые оффсеты и низкоуровневые процедуры работы с памятью для взаимодействия с сущностями игроков и данными движка.
Проект предназначен для исследований в области разработки читов и экспериментов по обходу античита на уровне ядра.
