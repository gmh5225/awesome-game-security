Этот проект — базовый упаковщик исполняемых файлов Win32 для 32-битных PE-файлов.
Он написан на C++ и включает сопутствующий компонент сжатия, чтобы уменьшить размер полезной нагрузки перед пересборкой упакованного исполняемого файла.
Репозиторий содержит файлы решения Visual Studio, руководство по использованию из командной строки, а также ветку расширения, демонстрирующую идеи самопроверки.
Основной сценарий использования — изучение концепций PE-упаковки для реверс-инжиниринга, экспериментов по защите бинарных файлов и исследований распаковки.
