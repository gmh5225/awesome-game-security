Questo progetto è un packer di base per eseguibili Win32 destinato a file PE a 32 bit.
È scritto in C++ e integra un componente di compressione associato per ridurre la dimensione del payload prima di ricostruire l'eseguibile impacchettato.
Il repository include file di soluzione Visual Studio, indicazioni per l'uso da riga di comando e un branch di estensione che illustra idee di auto-validazione.
Il caso d'uso principale è l'apprendimento dei concetti di packing PE per reverse engineering, esperimenti di protezione binaria e ricerca sull'unpacking.
