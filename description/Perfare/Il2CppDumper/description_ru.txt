Этот проект — утилита для реверс-инжиниринга бинарников и файлов метаданных Unity IL2CPP. Она написана на C# и поддерживает несколько форматов исполняемых файлов, включая ELF, Mach-O, PE, NSO и WASM, для широкого диапазона версий Unity. Инструмент может восстанавливать фиктивные метаданные DLL и генерировать вспомогательные выходные данные для IDA, Ghidra и Binary Ninja, а также заголовки структур для более глубокого анализа. Основное применение — реверс-инжиниринг игр на Unity, статический анализ и исследование средств защиты.
