Este proyecto es una herramienta de escritorio para explorar, extraer y exportar assets de Unity y AssetBundles. Está escrita principalmente en C# y admite una amplia gama de versiones de Unity y tipos de assets, incluidas texturas, audio, fuentes, mallas, shaders y datos relacionados con animación. La herramienta puede descomprimir bundles, inspeccionar jerarquías de escenas y exportar datos a formatos prácticos como imágenes, OBJ, JSON y salidas relacionadas con FBX. La usan habitualmente ingenieros inversos, modders y analistas de seguridad de juegos que necesitan inspeccionar el contenido de juegos Unity.
