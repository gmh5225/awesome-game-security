Este proyecto es un módulo de volcado en tiempo de ejecución para Android que extrae datos IL2CPP mediante Zygisk. Combina componentes nativos C/C++ con las herramientas de compilación de Android para inyectarse en aplicaciones objetivo y recopilar metadatos de runtime que pueden estar protegidos en archivos estáticos. Al volcar tras la carga, puede eludir ciertos cifrados, ofuscaciones y empaquetados usados en títulos Unity móviles. Se emplea principalmente en flujos de ingeniería inversa y de investigación de seguridad de juegos móviles.
