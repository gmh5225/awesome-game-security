Questo progetto è un modulo di dump runtime Android che estrae dati IL2CPP tramite Zygisk. Combina componenti nativi C/C++ con gli strumenti di build Android per iniettarsi nelle app di destinazione e raccogliere metadati runtime che possono essere protetti nei file statici. Eseguendo il dump dopo il caricamento, può aggirare alcune difese di crittografia, offuscamento e packing usate nei titoli Unity mobili. È usato soprattutto nei flussi di reverse engineering e di ricerca sulla sicurezza dei giochi mobile.
