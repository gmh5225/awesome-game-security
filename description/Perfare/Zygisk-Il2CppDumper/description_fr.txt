Ce projet est un module de dump runtime Android qui extrait les données IL2CPP via Zygisk. Il combine des composants natifs C/C++ avec les outils de build Android pour s'injecter dans les applications cibles et collecter des métadonnées runtime qui peuvent être protégées dans les fichiers statiques. En dumpant après le chargement, il peut contourner certains chiffrement, obfuscation et packing utilisés dans les jeux Unity mobiles. Il sert principalement aux workflows de rétro-ingénierie et de recherche en sécurité des jeux mobiles.
