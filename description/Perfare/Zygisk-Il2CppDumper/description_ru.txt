Этот проект — модуль дампа во время выполнения Android, извлекающий данные IL2CPP с помощью Zygisk. Он сочетает нативные компоненты C/C++ с инструментами сборки Android для внедрения в целевые приложения и сбора runtime-метаданных, которые могут быть защищены в статических файлах. Дамп после загрузки позволяет обойти некоторые средства шифрования, обфускации и упаковки, применяемые в мобильных играх на Unity. В основном используется в рабочих процессах реверс-инжиниринга и исследований безопасности мобильных игр.
