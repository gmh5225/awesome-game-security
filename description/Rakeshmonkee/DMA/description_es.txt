Este proyecto es una guía práctica y un conjunto de herramientas para crear y personalizar firmware PCIe DMA para placas basadas en FPGA.
Incluye instrucciones paso a paso para la personalización de Vivado, scripts de generación automática de firmware, clonación del espacio de configuración y flujos de flasheo.
El repositorio combina documentación Markdown con utilidades en Python y Tcl para la generación de proyectos y la preparación del firmware.
Su caso de uso principal es la investigación en seguridad de juegos sobre el comportamiento del hardware DMA y pruebas de evasión de anti-cheat en entornos controlados.
