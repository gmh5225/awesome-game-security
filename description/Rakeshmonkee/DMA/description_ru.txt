Этот проект — практическое руководство и набор инструментов для создания и настройки прошивок PCIe DMA для плат на базе FPGA.
Он включает пошаговые инструкции по кастомизации в Vivado, скрипты автоматической генерации прошивки, клонирование конфигурационного пространства и рабочие процессы прошивки.
Репозиторий сочетает Markdown-документацию с утилитами на Python и Tcl для генерации проектов и подготовки прошивки.
Основной сценарий использования — исследования игровой безопасности, связанные с поведением DMA-оборудования и тестами обхода античита в контролируемых средах.
