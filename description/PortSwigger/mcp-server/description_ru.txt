Этот проект — расширение Burp Suite, которое открывает возможности Burp для ИИ-клиентов через Model Context Protocol. Оно реализовано на Kotlin и Java, включает скрипты сборки Gradle и упакованные артефакты расширения. Ключевые возможности: конечная точка MCP-сервера, встроенный режим stdio-прокси и поддержка конфигурации клиентов для настольных ИИ-инструментов. Предназначено для специалистов по безопасности приложений, которым нужно взаимодействие с рабочими процессами Burp при поддержке ИИ.
