Dieses Projekt ist eine Burp-Suite-Erweiterung, die Burp-Funktionen über das Model Context Protocol für KI-Clients bereitstellt. Es ist in Kotlin und Java implementiert, verfügt über Gradle-Build-Skripte und wird als paketierte Erweiterungsartefakte ausgeliefert. Zu den Kernfunktionen gehören ein MCP-Server-Endpunkt, ein gebündelter stdio-Proxy-Modus sowie Client-Konfigurationsunterstützung für Desktop-KI-Tools. Es richtet sich an Anwendungssicherheitstester, die KI-gestützt mit Burp-Workflows arbeiten möchten.
