Este proyecto es una extensión de Burp Suite que expone las capacidades de Burp a clientes de IA mediante el Model Context Protocol. Está implementada en Kotlin y Java, con scripts de compilación Gradle y artefactos de extensión empaquetados. Las funciones principales incluyen un endpoint de servidor MCP, un modo proxy stdio integrado y soporte de configuración de cliente para herramientas de IA de escritorio. Está pensada para probadores de seguridad de aplicaciones que desean interactuar con flujos de trabajo de Burp asistidos por IA.
