Questo progetto è un'estensione di Burp Suite che espone le funzionalità di Burp ai client AI tramite il Model Context Protocol. È implementata in Kotlin e Java, con script di build Gradle e artefatti di estensione pacchettizzati. Le funzionalità principali includono un endpoint server MCP, una modalità proxy stdio inclusa e il supporto alla configurazione del client per strumenti AI desktop. È pensata per i tester di sicurezza delle applicazioni che desiderano interagire con i flussi di lavoro Burp assistiti dall'AI.
