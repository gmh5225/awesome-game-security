Ce projet est une extension Burp Suite qui expose les capacités de Burp aux clients d'IA via le Model Context Protocol. Il est implémenté en Kotlin et Java, avec des scripts de build Gradle et des artefacts d'extension empaquetés. Les fonctionnalités clés incluent un point de terminaison serveur MCP, un mode proxy stdio intégré, et la prise en charge de la configuration client pour les outils d'IA de bureau. Il s'adresse aux testeurs en sécurité applicative qui souhaitent interagir avec les flux de travail Burp assistés par l'IA.
