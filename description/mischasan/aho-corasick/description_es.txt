Biblioteca de alto rendimiento para coincidencia de cadenas multipatrón Aho-Corasick que usa una representación de matriz de transición de estados intercalada (ISM), logrando a la vez una huella de memoria mínima y la máxima velocidad de búsqueda. Admite serialización con mapeo de memoria para autómatas persistentes y ofrece una API C para construir, consultar y volcar la máquina de estados compilada.
