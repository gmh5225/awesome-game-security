Высокопроизводительная библиотека многошаблонного сопоставления строк Aho-Corasick с представлением на основе Interleaved State-transition Matrix (ISM), обеспечивающая минимальный объём памяти и максимальную скорость поиска. Поддерживает сериализацию с отображением в память для постоянных автоматов и предоставляет C API для построения, запроса и дампа скомпилированного конечного автомата.
