Анализ исходного кода Unreal Engine на основе профайлера, охватывающий инициализацию движка, игровой цикл, управление памятью, управление потоками, скриптинг Blueprint и конвейер рендеринга с подробными схемами.
Он прослеживает пути выполнения через task graph UE4, конвейер отрисовки мешей, списки команд RHI и платформо-зависимые бэкенды рендеринга Metal/D3D11 с помощью стеков вызовов профайлера.
Материал в первую очередь полезен программистам игровых движков и исследователям безопасности, изучающим внутреннее устройство Unreal Engine и его runtime-архитектуру по исходному коду.
