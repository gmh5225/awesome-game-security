Un'analisi del codice sorgente di Unreal Engine guidata dal profiler che copre l'inizializzazione del motore, il game loop, la gestione della memoria, la gestione dei thread, lo scripting Blueprint e la pipeline di rendering con diagrammi dettagliati.
Traccia i percorsi di codice attraverso il task graph di UE4, la pipeline di disegno delle mesh, le liste di comandi RHI e i backend di rendering specifici della piattaforma Metal/D3D11 usando le call stack del profiler.
È soprattutto utile per programmatori di game engine e ricercatori di sicurezza che studiano gli interni di Unreal Engine e ne comprendono l'architettura runtime dal codice sorgente.
