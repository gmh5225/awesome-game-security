Une analyse du code source d'Unreal Engine pilotée par le profileur, couvrant l'initialisation du moteur, la boucle de jeu, la gestion mémoire, la gestion des threads, le scripting Blueprint et le pipeline de rendu avec des diagrammes détaillés.
Elle suit les chemins d'exécution via le task graph d'UE4, le pipeline de dessin de maillages, les listes de commandes RHI et les backends de rendu spécifiques aux plateformes Metal/D3D11 à l'aide des piles d'appels du profileur.
Elle s'adresse surtout aux programmeurs de moteurs de jeu et aux chercheurs en sécurité qui étudient les internals d'Unreal Engine et son architecture d'exécution à partir du code source.
