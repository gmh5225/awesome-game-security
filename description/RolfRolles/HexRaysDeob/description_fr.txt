Ce projet est un plugin microcode Hex-Rays destiné à désobfusquer des binaires protégés. Il est écrit en C++ pour les SDK IDA et Hex-Rays et inclut des passes de simplification basées sur des motifs pour les expressions obfusquées. Il implémente également une logique de dé-aplatissement du flot de contrôle qui reconstruit les régions aplaties pilotées par un dispatcher et nettoie les blocs inaccessibles. L'outil est conçu pour les workflows de reverse engineering, notamment l'analyse de code de jeux et de malwares obfusqués.
