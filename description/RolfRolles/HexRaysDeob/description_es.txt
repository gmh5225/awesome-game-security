Este proyecto es un complemento de microcódigo de Hex-Rays para desofuscar binarios protegidos. Está escrito en C++ contra los SDK de IDA y Hex-Rays e incluye pases de simplificación basados en patrones para expresiones ofuscadas. También implementa lógica de desaplanado del flujo de control que reconstruye regiones aplanadas impulsadas por un dispatcher y limpia bloques inalcanzables. La herramienta está pensada para flujos de trabajo de ingeniería inversa, incluido el análisis de código de juegos y malware ofuscado.
