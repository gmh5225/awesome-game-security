Questo progetto è un plugin microcode di Hex-Rays per deoffuscare binari protetti. È scritto in C++ contro gli SDK di IDA e Hex-Rays e include passate di semplificazione basate su pattern per espressioni offuscate. Implementa anche una logica di unflattening del controllo di flusso che ricostruisce le regioni appiattite guidate da un dispatcher e pulisce i blocchi irraggiungibili. Lo strumento è destinato ai flussi di lavoro di reverse engineering, inclusa l'analisi di codice di giochi e malware offuscato.
