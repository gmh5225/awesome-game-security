Этот проект — плагин микрокода Hex-Rays для деобфускации защищённых бинарных файлов. Он написан на C++ с использованием SDK IDA и Hex-Rays и включает проходы упрощения на основе шаблонов для обфусцированных выражений. Также реализована логика восстановления управляющего потока (control-flow unflattening), которая реконструирует области, сглаженные через диспетчер, и очищает недостижимые блоки. Инструмент предназначен для рабочих процессов реверс-инжиниринга, включая анализ обфусцированного игрового и вредоносного кода.
