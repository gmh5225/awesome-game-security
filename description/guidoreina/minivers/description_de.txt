Dieses Projekt erzeugt Backup Copies (Sicherungskopien).
Wenn eine überwachte Datei geändert, gelöscht oder umbenannt werden soll, wird vor Beginn der Operation eine Sicherungskopie der Datei erstellt.
Es ist vor allem für Anti-Cheat-Ingenieure und defensive Security-Forscher nützlich, die im Bereich Anti-Cheat / Sicherungsdateien arbeiten.
