Este proyecto genera Backup Copies (copias de seguridad).
Si un archivo supervisado está a punto de ser modificado, eliminado o renombrado, se realiza una copia de seguridad del archivo antes de que comience la operación.
Es principalmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / archivos de respaldo.
