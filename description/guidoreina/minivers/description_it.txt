Questo progetto genera Backup Copies (copie di backup).
Se un file monitorato sta per essere modificato, eliminato o rinominato, viene eseguita una copia di backup del file prima che l'operazione abbia inizio.
È principalmente utile per ingegneri anti-cheat e ricercatori di sicurezza difensiva che lavorano nell'ambito anti-cheat / file di backup.
