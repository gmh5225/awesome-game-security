Dieses Projekt ist ein Cheat-Engine-Plugin, das das Lesen und Schreiben von Speicher über DMA-Hardware (Direct Memory Access) wie PCILeech-kompatible FPGA-Boards ermöglicht. Es ersetzt den standardmäßigen Prozessspeicherzugriff von Cheat Engine durch DMA-basierte physische Speicheroperationen und erlaubt so Speicher-Scanning und -Bearbeitung, ohne dass das Betriebssystem des Zielsystems den Zugriff erkennt. Das C/C++-Plugin integriert sich in die pcileech/LeechCore-Bibliothek. Es richtet sich an DMA-basierte Spielsicherheitsforscher und Anti-Cheat-Analysten, die DMA-Angriffsszenarien über die vertraute Cheat-Engine-Oberfläche testen.
