이 프로젝트는 PCILeech 호환 FPGA 보드와 같은 DMA(Direct Memory Access) 하드웨어를 통해 메모리 읽기/쓰기를 가능하게 하는 Cheat Engine 플러그인입니다. Cheat Engine의 표준 프로세스 메모리 접근을 DMA 기반 물리 메모리 작업으로 대체하여, 대상 시스템의 OS가 접근을 감지하지 못한 채 메모리 스캔과 편집을 수행할 수 있습니다. C/C++로 작성된 이 플러그인은 pcileech/LeechCore 라이브러리와 연동되며, 익숙한 Cheat Engine 인터페이스로 DMA 공격 시나리오를 테스트하는 DMA 기반 게임 보안 연구자 및 안티치트 분석가를 대상으로 합니다.
