Этот проект — плагин для Cheat Engine, который обеспечивает чтение и запись памяти через DMA-оборудование (Direct Memory Access), например FPGA-платы, совместимые с PCILeech. Он заменяет стандартный доступ Cheat Engine к памяти процессов DMA-операциями с физической памятью, позволяя сканировать и изменять память без обнаружения доступа ОС целевой системы. Плагин на C/C++ интегрируется с библиотекой pcileech/LeechCore. Он предназначен для исследователей игровой безопасности, работающих с DMA, и аналитиков античита, тестирующих сценарии DMA-атак через привычный интерфейс Cheat Engine.
