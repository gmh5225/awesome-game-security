Este proyecto es un complemento de Cheat Engine que permite leer y escribir memoria mediante hardware DMA (Direct Memory Access), como placas FPGA compatibles con PCILeech. Sustituye el acceso estándar a la memoria de procesos de Cheat Engine por operaciones de memoria física basadas en DMA, lo que permite escanear y editar memoria sin que el sistema operativo de destino detecte el acceso. El complemento en C/C++ se integra con la biblioteca pcileech/LeechCore. Está orientado a investigadores de seguridad en juegos basados en DMA y a analistas de anti-trampas que prueban escenarios de ataque DMA a través de la interfaz familiar de Cheat Engine.
