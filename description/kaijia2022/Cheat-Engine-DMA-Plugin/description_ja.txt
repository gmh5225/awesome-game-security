本プロジェクトは、PCILeech 互換の FPGA ボードなどの DMA（Direct Memory Access）ハードウェアを通じてメモリの読み書きを可能にする Cheat Engine プラグインです。Cheat Engine 標準のプロセスメモリアクセスを DMA ベースの物理メモリ操作に置き換え、ターゲットシステムの OS に検知されずにメモリスキャンや編集を行えます。C/C++ 製のこのプラグインは pcileech/LeechCore ライブラリと連携し、使い慣れた Cheat Engine のインターフェースで DMA 攻撃シナリオを検証する DMA ベースのゲームセキュリティ研究者およびアンチチート分析者を対象としています。
