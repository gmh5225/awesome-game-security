Este proyecto se centra en un toolkit de exploits de iOS filtrado: cadenas WebKit para varias versiones de iOS, parcialmente desofuscadas. Estos scripts se han modificado para poder alojarlos localmente. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / jailbreaks de iOS.
