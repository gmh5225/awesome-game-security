Этот проект посвящён утечшему набору инструментов для эксплуатации iOS — цепочкам WebKit для нескольких версий iOS, частично деобфусцированным. Эти скрипты модифицированы так, чтобы их можно было размещать локально. Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / джейлбрейка iOS.
