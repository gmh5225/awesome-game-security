Dieses Projekt konzentriert sich auf ein geleaktes iOS-Exploit-Toolkit — WebKit-Chains für mehrere iOS-Versionen, teilweise deobfuskiert. Die Skripte wurden so angepasst, dass sie lokal gehostet werden können. Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / iOS-Jailbreak untersuchen.
