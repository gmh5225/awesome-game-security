Este proyecto es un controlador del kernel de Windows que protege frente a ataques DMA (Direct Memory Access) configurando IOMMU/VT-d para restringir qué dispositivos PCIe pueden acceder a la memoria del sistema. Configura tablas de remapeo DMA para bloquear lecturas de memoria no autorizadas desde placas de ataque DMA basadas en FPGA, permitiendo al mismo tiempo que los dispositivos legítimos funcionen con normalidad. El controlador en C demuestra el uso defensivo de la tecnología Intel VT-d. Está orientado a ingenieros de anti-trampas e investigadores de seguridad que construyen mitigaciones contra ataques DMA.
