Этот проект — драйвер ядра Windows, который защищает от атак DMA (Direct Memory Access), настраивая IOMMU/VT-d так, чтобы ограничить доступ PCIe-устройств к системной памяти. Он создаёт таблицы DMA-переадресации, блокируя несанкционированное чтение памяти платами DMA-атак на базе FPGA и при этом позволяя легитимным устройствам работать как обычно. Драйвер на C демонстрирует оборонительное применение технологии Intel VT-d. Он предназначен для инженеров античита и исследователей безопасности, разрабатывающих средства снижения рисков DMA-атак.
