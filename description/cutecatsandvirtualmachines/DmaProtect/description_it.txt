Questo progetto è un driver del kernel Windows che protegge dagli attacchi DMA (Direct Memory Access) configurando IOMMU/VT-d per limitare quali dispositivi PCIe possono accedere alla memoria di sistema. Imposta tabelle di remapping DMA per bloccare letture di memoria non autorizzate da schede di attacco DMA basate su FPGA, consentendo al contempo ai dispositivi legittimi di funzionare normalmente. Il driver in C dimostra l’uso difensivo della tecnologia Intel VT-d. È rivolto a ingegneri anti-cheat e ricercatori di sicurezza che sviluppano mitigazioni contro gli attacchi DMA.
