Questo progetto si concentra su kiero v2 — localizzatore a runtime degli indirizzi dei metodi delle API grafiche D3D9–D12, OpenGL e Vulkan; libreria di hooking da fornire autonomamente (BYO); CMake FetchContent; OpenGL/Vulkan multipiattaforma su Win/Linux/macOS.
È scritto principalmente in C++ e ruota intorno a DirectX, OpenGL e Vulkan.
È soprattutto utile a programmatori grafici e sviluppatori di tool per giochi Windows che lavorano nell'ambito DirectX / hook.
