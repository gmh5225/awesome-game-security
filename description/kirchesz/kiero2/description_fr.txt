Ce projet se concentre sur kiero v2 — localisateur à l'exécution des adresses de méthodes des API graphiques D3D9–D12, OpenGL et Vulkan ; bibliothèque de hooking à fournir soi-même (BYO) ; CMake FetchContent ; OpenGL/Vulkan multiplateforme sur Win/Linux/macOS.
Il est principalement écrit en C++ et s'articule autour de DirectX, OpenGL et Vulkan.
Il est surtout utile aux programmeurs graphiques et aux développeurs d'outils de jeux Windows travaillant dans le domaine DirectX / hook.
