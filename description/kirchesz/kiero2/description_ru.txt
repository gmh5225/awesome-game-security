Этот проект посвящён kiero v2 — рантайм-локатору адресов методов графических API D3D9–D12, OpenGL и Vulkan; библиотеку хуков подключаете сами (BYO); CMake FetchContent; кроссплатформенные OpenGL/Vulkan на Win/Linux/macOS.
В основном написан на C++ и ориентирован на DirectX, OpenGL и Vulkan.
Полезен главным образом графическим программистам и разработчикам игровых инструментов под Windows, работающим в области DirectX / hook.
