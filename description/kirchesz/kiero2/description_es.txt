Este proyecto se centra en kiero v2: localizador en tiempo de ejecución de direcciones de métodos de las API gráficas D3D9–D12, OpenGL y Vulkan; biblioteca de hooking propia (BYO); CMake FetchContent; OpenGL/Vulkan multiplataforma en Win/Linux/macOS.
Está escrito principalmente en C++ y se centra en DirectX, OpenGL y Vulkan.
Es especialmente útil para programadores gráficos y desarrolladores de herramientas de juegos en Windows que trabajan en el área de DirectX / hook.
