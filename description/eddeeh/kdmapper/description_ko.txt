이 프로젝트는 kdmapper로, 서명되었지만 취약한 Intel iqvw64e.sys 드라이버를 악용해 서명되지 않은 드라이버를 커널 메모리에 로드하는 Windows 커널 드라이버 매퍼입니다. 해당 드라이버의 임의 물리 메모리 읽기/쓰기 IOCTL을 사용해 커스텀 드라이버를 수동으로 매핑합니다. 커널 풀 메모리 할당, 섹션 복사, ntoskrnl 대상 임포트 해석, 재배치 처리, 드라이버 진입점 호출을 수행합니다. 이 C++ 도구는 취약 드라이버 배포와 정리를 포함한 전체 매핑 파이프라인을 자동화합니다. 수동 드라이버 매핑과 DSE 우회 기법을 연구하는 커널 연구자 및 치트 개발자를 대상으로 합니다.
