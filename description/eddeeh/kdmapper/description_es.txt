Este proyecto es kdmapper, un mapeador de controladores del kernel de Windows que carga controladores sin firmar en memoria del kernel explotando el controlador firmado vulnerable Intel iqvw64e.sys. Usa el IOCTL de lectura/escritura arbitraria de memoria física del controlador para mapear manualmente un controlador personalizado: asignar memoria de pool del kernel, copiar secciones, resolver importaciones contra ntoskrnl, procesar reubicaciones y llamar al punto de entrada del controlador. La herramienta en C++ automatiza todo el proceso de mapeo, incluido el despliegue y la limpieza del controlador vulnerable. Está dirigida a investigadores del kernel y desarrolladores de cheats que estudian el mapeo manual de controladores y las técnicas de bypass de DSE.
