Этот проект — kdmapper, маппер драйверов ядра Windows, который загружает неподписанные драйверы в память ядра, эксплуатируя уязвимый подписанный драйвер Intel iqvw64e.sys. Он использует IOCTL произвольного чтения/записи физической памяти этого драйвера для ручного маппинга пользовательского драйвера: выделение памяти из kernel pool, копирование секций, разрешение импортов относительно ntoskrnl, обработка релокаций и вызов точки входа драйвера. Инструмент на C++ автоматизирует весь конвейер маппинга, включая развёртывание и очистку уязвимого драйвера. Он предназначен для исследователей ядра и разработчиков читов, изучающих ручной маппинг драйверов и техники обхода DSE.
