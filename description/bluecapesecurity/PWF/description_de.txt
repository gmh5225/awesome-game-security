Dieses Projekt ist das Personal Windows Forensics Toolkit, eine Sammlung von Skripten und Tools zur forensischen Analyse von Windows-Systemen. Es bietet Dienstprogramme für die Artefakt-Erfassung, Timeline-Erstellung, Registry-Analyse und Beweissicherung in Incident-Response-Szenarien. Das Toolkit bündelt mehrere forensische Dienstprogramme in einem schlanken Workflow. Es richtet sich an digitale Forensiker und Incident Responder, die Windows-basierte Untersuchungen durchführen.
