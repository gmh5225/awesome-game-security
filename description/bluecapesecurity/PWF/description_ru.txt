Этот проект — набор Personal Windows Forensics, коллекция скриптов и инструментов для проведения криминалистического анализа систем Windows. Он предоставляет средства для сбора артефактов, построения временных шкал, анализа реестра и сохранения доказательств в сценариях реагирования на инциденты. Набор объединяет несколько криминалистических утилит в упрощённый рабочий процесс. Предназначен для специалистов по цифровой криминалистике и специалистов по реагированию на инциденты, проводящих расследования на базе Windows.
