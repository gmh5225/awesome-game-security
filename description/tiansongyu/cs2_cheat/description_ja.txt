これは教育目的の Counter-Strike 2 用外部 ESP／チート学習プロジェクトで、外部オーバーレイを通じてゲームのリバースエンジニアリングを示します。主に C++ で書かれ、描画と UI に SDL2 と ImGui を用い、Windows のプロセスメモリ読み取りや cs2-dumper などのツールで生成したオフセットを利用します。ボックス／スケルトン ESP、体力・武器表示、エイムボット、トリガーボット、レーダーオーバーレイ、アンチフラッシュ、爆弾タイマーなどの機能があり、ゲームオフセットは毎時 CI で更新されます。ゲームのメモリ構造、プロセス通信、グラフィックスオーバーレイを学ぶ学生・研究者向けで、オンライン対戦ではなくオフラインの insecure モード専用です。
