Образовательный проект внешнего ESP и читов для Counter-Strike 2, демонстрирующий реверс-инжиниринг игр через внешний оверлей. Написан в основном на C++, использует SDL2 и ImGui для отрисовки и интерфейса, чтение памяти процессов Windows и смещения, сгенерированные такими инструментами, как cs2-dumper. Включает box- и skeleton-ESP, отображение здоровья и оружия, аимбот, триггербот, радар-оверлей, анти-флэш, таймер бомбы и связанные дополнения, с почасовым CI-обновлением игровых смещений. Предназначен для студентов и исследователей структур игровой памяти, межпроцессного взаимодействия и графических оверлеев; строго для офлайн-использования в insecure-режиме, а не для онлайн-игры.
