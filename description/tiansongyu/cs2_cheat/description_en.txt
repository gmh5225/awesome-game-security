This is an educational Counter-Strike 2 external ESP and cheat learning project that demonstrates game reverse engineering through an external overlay. Written primarily in C++, it uses SDL2 and ImGui for rendering and UI, with Windows process memory reading and generated offsets from tools such as cs2-dumper. Features include box and skeleton ESP, health and weapon displays, aimbot, triggerbot, radar overlay, anti-flash, bomb timer, and related extras, with hourly CI updates for game offsets. It targets students and researchers studying game memory structures, process communication, and graphics overlays, and is intended strictly for offline insecure-mode use rather than online play.
