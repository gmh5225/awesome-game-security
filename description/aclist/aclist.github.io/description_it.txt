Questo progetto è un sito web statico che pubblica informazioni sulla compatibilità anti-cheat e documentazione correlata sul gaming Linux.
È realizzato con HTML, CSS e JavaScript e include un elenco di compatibilità curato e guide per gli strumenti complementari.
I contenuti si basano su una verifica manuale prudente del comportamento dei giochi protetti da anti-cheat sui sistemi Linux.
Si rivolge principalmente a gamer Linux e ricercatori di sicurezza dei giochi che seguono il supporto delle piattaforme e l'interoperabilità anti-cheat.
