Dieses Projekt ist eine statische Website, die Anti-Cheat-Kompatibilitätsinformationen und zugehörige Dokumentation zum Spielen unter Linux veröffentlicht.
Es ist mit HTML, CSS und JavaScript erstellt und enthält eine kuratierte Kompatibilitätsliste sowie Anleitungen zu begleitenden Tools.
Die Inhalte basieren auf einer konservativen manuellen Prüfung, wie sich anti-cheat-geschützte Spiele auf Linux-Systemen verhalten.
Es richtet sich vor allem an Linux-Gamer und Spiele-Sicherheitsforscher, die Plattformunterstützung und Anti-Cheat-Interoperabilität verfolgen.
