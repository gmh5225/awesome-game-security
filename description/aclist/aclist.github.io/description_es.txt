Este proyecto es un sitio web estático que publica información de compatibilidad de anti-cheat y documentación relacionada con el juego en Linux.
Está construido con HTML, CSS y JavaScript e incluye una lista de compatibilidad seleccionada y guías para herramientas complementarias.
El contenido se basa en una verificación manual conservadora del comportamiento de los juegos protegidos con anti-cheat en sistemas Linux.
Sirve principalmente a jugadores de Linux e investigadores de seguridad en juegos que siguen el soporte de plataformas y la interoperabilidad del anti-cheat.
