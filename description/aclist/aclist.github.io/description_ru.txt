Этот проект — статический сайт, публикующий сведения о совместимости античитов и связанную документацию по играм на Linux.
Он создан на HTML, CSS и JavaScript и включает курируемый список совместимости, а также руководства по сопутствующим инструментам.
Содержание основано на осторожной ручной проверке того, как ведут себя игры с античитом в системах Linux.
Он в первую очередь рассчитан на Linux-геймеров и исследователей игровой безопасности, отслеживающих поддержку платформ и совместимость античитов.
