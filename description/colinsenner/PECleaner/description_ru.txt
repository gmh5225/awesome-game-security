Этот проект — PECleaner, Windows-инструмент для удаления отладочной информации, Rich-заголовков, временных меток и других метаданных из PE-файлов (Portable Executable). Он обнуляет или удаляет артефакты компиляции, которые могут использоваться для атрибуции или фингерпринтинга, включая путь к PDB, версию линкера, временные метки компилятора и данные Rich-заголовка. Инструмент на C# помогает очищать бинарники перед распространением. Он предназначен для операторов red team, исследователей вредоносного ПО и разработчиков, желающих удалить идентифицирующие метаданные из скомпилированных исполняемых файлов.
