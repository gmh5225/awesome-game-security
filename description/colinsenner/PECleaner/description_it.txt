Questo progetto è PECleaner, uno strumento Windows per rimuovere informazioni di debug, Rich header, timestamp e altri metadati dai file PE (Portable Executable). Azzera o elimina artefatti di compilazione utilizzabili per attribuzione o fingerprinting, tra cui il percorso PDB, la versione del linker, i timestamp del compilatore e i dati del Rich header. Lo strumento in C# aiuta a sanificare i binari prima della distribuzione. È rivolto a operatori red team, ricercatori di malware e sviluppatori che vogliono rimuovere metadati identificativi dagli eseguibili compilati.
