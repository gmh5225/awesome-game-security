Dieses Projekt ist ein Framework oder Tool für Narrative- bzw. Story-Management-Systeme in Spielen. Es bietet Strukturen zur Organisation von verzweigten Dialogen, Quest-Fortschritt oder interaktiven Storytelling-Elementen. Es richtet sich an Spieleentwickler, die an narrativ getriebenen Spielinhalten und Story-Skriptsystemen arbeiten.
