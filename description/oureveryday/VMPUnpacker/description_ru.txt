Этот проект — распаковщик исполняемых файлов, упакованных VMProtect, доступный в реализациях на C++ и Python. Он использует распаковку LZMA, чтобы извлечь исходный PE из упакованного payload VMProtect и восстановить исходные секции и таблицы импорта защищённого бинарника. В первую очередь полезен для реверс-инженеров и аналитиков вредоносного ПО, работающих с бинарниками, упакованными VMProtect, которым нужна автоматическая распаковка.
