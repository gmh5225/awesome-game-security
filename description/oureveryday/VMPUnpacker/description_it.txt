Questo progetto è un unpacker per eseguibili packati con VMProtect, disponibile in implementazioni C++ e Python. Usa la decompressione LZMA per estrarre il PE originale dal payload packato di VMProtect, recuperando le sezioni e le tabelle di import originali del binario protetto. È soprattutto utile per reverse engineer e analisti di malware che lavorano con binari packati con VMProtect e necessitano di capacità di unpacking automatizzato.
