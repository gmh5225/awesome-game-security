Este proyecto es un unpacker para ejecutables empaquetados con VMProtect, disponible en implementaciones en C++ y Python. Utiliza descompresión LZMA para extraer el PE original del payload empaquetado de VMProtect, recuperando las secciones y tablas de importación originales del binario protegido. Resulta especialmente útil para ingenieros inversos y analistas de malware que trabajan con binarios empaquetados con VMProtect y necesitan capacidades de desempaquetado automatizado.
