Questo progetto è una codebase tutorial in C++ per costruire un gioco 3D e un motore da zero. Guida passo dopo passo attraverso le finestre Win32, la grafica DirectX 11, la gestione dell’input, i pattern entity-component e l’integrazione di esempi di gioco. Il repository separa i moduli riutilizzabili del motore dal progetto di gioco e traccia le tappe di apprendimento tramite versioni taggate. È pensato per chi impara e per gli sviluppatori che vogliono comprendere in modo pratico l’architettura di basso livello di un motore di gioco.
