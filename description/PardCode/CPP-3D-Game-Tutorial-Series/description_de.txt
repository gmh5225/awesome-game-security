Dieses Projekt ist eine C++-Tutorial-Codebasis zum Aufbau eines 3D-Spiels und einer Engine von Grund auf. Es führt schrittweise durch Win32-Fensterverwaltung, DirectX-11-Grafik, Eingabebehandlung, Entity-Component-Muster und die Integration von Spielbeispielen. Das Repository trennt wiederverwendbare Engine-Module vom Spielprojekt und verfolgt Lernmeilensteine über getaggte Versionen. Es richtet sich an Lernende und Entwickler, die ein praxisnahes Verständnis der Low-Level-Architektur einer Spiele-Engine gewinnen möchten.
