Ce projet est une base de code tutorielle en C++ pour construire un jeu 3D et un moteur à partir de zéro. Il couvre par étapes la fenêtrage Win32, les graphismes DirectX 11, la gestion des entrées, les motifs entité-composant et l’intégration d’exemples de jeu. Le dépôt sépare les modules moteur réutilisables du projet de jeu et suit les jalons d’apprentissage via des versions étiquetées. Il s’adresse aux apprenants et développeurs qui veulent une compréhension concrète de l’architecture bas niveau d’un moteur de jeu.
