Этот проект — учебная кодовая база на C++ для создания 3D-игры и движка с нуля. Он поэтапно разбирает оконную систему Win32, графику DirectX 11, обработку ввода, паттерны entity-component и интеграцию игровых примеров. Репозиторий отделяет переиспользуемые модули движка от игрового проекта и отмечает этапы обучения через помеченные версии. Он предназначен для обучающихся и разработчиков, которым нужно практическое понимание низкоуровневой архитектуры игрового движка.
