Este proyecto es una base de código tutorial en C++ para construir un juego 3D y un motor desde cero. Recorre por etapas el ventaneo Win32, gráficos DirectX 11, manejo de entrada, patrones entidad-componente y la integración de ejemplos de juego. El repositorio separa los módulos reutilizables del motor del proyecto de juego y sigue los hitos de aprendizaje mediante versiones etiquetadas. Está pensado para estudiantes y desarrolladores que quieren comprender de forma práctica la arquitectura de bajo nivel de un motor de juego.
