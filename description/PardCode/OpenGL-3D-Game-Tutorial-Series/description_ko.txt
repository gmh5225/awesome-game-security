이 프로젝트는 OpenGL로 3D 게임과 엔진 시스템을 구축하기 위한 크로스플랫폼 C++ 튜토리얼 코드베이스입니다. Windows, macOS, Linux에서 각각 Win32, Cocoa, X11을 사용한 플랫폼별 윈도우 설정을 보여 주고, VAO, 셰이더, UBO 워크플로 같은 핵심 렌더링 구성 요소를 단계적으로 구축합니다. 저장소에는 단계별 학습 경로의 일환으로 엔티티 관리와 입력 처리 같은 기초 게임 서브시스템도 포함되어 있습니다. 실용적인 게임 프로그래밍과 그래픽스 엔진 아키텍처를 처음부터 배우려는 초급·중급 개발자를 주요 대상으로 합니다.
