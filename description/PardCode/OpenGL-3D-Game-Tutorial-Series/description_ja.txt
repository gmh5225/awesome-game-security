本プロジェクトは、OpenGL を用いて 3D ゲームとエンジンシステムを構築するためのクロスプラットフォームな C++ チュートリアルコードベースです。Windows、macOS、Linux 上でそれぞれ Win32、Cocoa、X11 を使ったプラットフォーム固有のウィンドウ初期化を示し、VAO、シェーダー、UBO ワークフローなどのコアなレンダリング要素を組み立てていきます。リポジトリには、段階的な学習パスの一環として、エンティティ管理や入力処理といった基礎的なゲームサブシステムも含まれます。実用的なゲームプログラミングとグラフィックスエンジンアーキテクチャをゼロから学ぶ、初心者から中級者向けの内容です。
