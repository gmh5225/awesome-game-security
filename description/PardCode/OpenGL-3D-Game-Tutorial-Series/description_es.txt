Este proyecto es una base de código tutorial en C++ multiplataforma para construir un juego 3D y sistemas de motor con OpenGL. Demuestra la configuración de ventanas específica de cada plataforma en Windows, macOS y Linux mediante Win32, Cocoa y X11, y luego construye piezas centrales de renderizado como VAO, shaders y flujos de trabajo con UBO. El repositorio también incluye subsistemas de juego fundamentales como la gestión de entidades y el manejo de entrada, como parte de una ruta de aprendizaje paso a paso. Está dirigido principalmente a desarrolladores principiantes e intermedios que aprenden programación práctica de juegos y arquitectura de motores gráficos desde cero.
