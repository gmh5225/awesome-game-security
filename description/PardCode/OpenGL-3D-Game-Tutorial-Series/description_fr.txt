Ce projet est une base de code tutorielle C++ multiplateforme pour construire un jeu 3D et des systèmes de moteur avec OpenGL. Il présente la configuration de fenêtres spécifique à chaque plateforme sur Windows, macOS et Linux via Win32, Cocoa et X11, puis construit les éléments de rendu essentiels tels que les VAO, les shaders et les flux de travail UBO. Le dépôt inclut également des sous-systèmes de jeu fondamentaux comme la gestion des entités et la gestion des entrées, dans le cadre d’un parcours d’apprentissage pas à pas. Il s’adresse principalement aux développeurs débutants et intermédiaires qui apprennent la programmation de jeux pratique et l’architecture de moteurs graphiques depuis zéro.
