Dieses Projekt ist ManyTypes, eine Cheat-Engine-Erweiterung bzw. ein Begleitwerkzeug, das die Typsuche und Strukturerkennung beim Reverse Engineering von Spielespeicher verbessert. Es bietet erweiterte Datentyperkennung, Identifikation von Strukturmitgliedern und Auflösung von Pointer-Ketten über die Standardfähigkeiten von Cheat Engine hinaus. Es richtet sich an Game-Hacker und Reverse Engineers, die Cheat Engine für Speicheranalyse und Strukturrekonstruktion nutzen.
