Este proyecto ofrece una herramienta de línea de comandos para Windows que automatiza la adición de controladores a la lista de bloqueo personalizada de HVCI (HvciDisallowedImages). Hypervisor-Protected Code Integrity (HVCI), a menudo comercializada como Memory Integrity en la configuración de seguridad de Windows, es una función crítica de seguridad basada en virtualización (VBS). Resulta especialmente útil para investigadores de bajo nivel en Windows, Linux y móvil que trabajan en el área de trucos relacionados / Windows ring3.
