Questo progetto fornisce uno strumento da riga di comando per Windows che automatizza l'aggiunta di driver all'elenco di blocco personalizzato HVCI (HvciDisallowedImages). Hypervisor-Protected Code Integrity (HVCI), spesso presentata come Memory Integrity nelle impostazioni di sicurezza di Windows, è una funzione critica di sicurezza basata sulla virtualizzazione (VBS). È soprattutto utile per ricercatori di basso livello su Windows, Linux e mobile che operano nell'ambito di tecniche correlate / Windows ring3.
