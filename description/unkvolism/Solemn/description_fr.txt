Ce projet fournit un outil en ligne de commande pour Windows qui automatise l'ajout de pilotes à la liste de blocage personnalisée HVCI (HvciDisallowedImages). Hypervisor-Protected Code Integrity (HVCI), souvent présentée sous le nom Memory Integrity dans les paramètres de sécurité Windows, est une fonctionnalité critique de sécurité basée sur la virtualisation (VBS). Il est surtout utile aux chercheurs bas niveau Windows, Linux et mobile travaillant dans le domaine des techniques associées / Windows ring3.
