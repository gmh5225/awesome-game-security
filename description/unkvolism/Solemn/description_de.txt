Dieses Projekt stellt ein Kommandozeilen-Tool für Windows bereit, das das Hinzufügen von Treibern zur benutzerdefinierten HVCI-Blockliste (HvciDisallowedImages) automatisiert. Hypervisor-Protected Code Integrity (HVCI), in den Windows-Sicherheitseinstellungen oft als Memory Integrity vermarktet, ist eine kritische virtualisierungsbasierte Sicherheitsfunktion (VBS). Es ist vor allem für Low-Level-Forscher im Windows-, Linux- und Mobile-Bereich nützlich, die sich mit entsprechenden Tricks / Windows ring3 beschäftigen.
