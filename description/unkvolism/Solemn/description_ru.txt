Этот проект предоставляет инструмент командной строки для Windows, который автоматизирует добавление драйверов в пользовательский список блокировки HVCI (HvciDisallowedImages). Hypervisor-Protected Code Integrity (HVCI), часто предлагаемая как Memory Integrity в параметрах безопасности Windows, — критически важная функция безопасности на основе виртуализации (VBS). В основном полезен исследователям низкоуровневых систем Windows, Linux и мобильных платформ, работающим в области связанных приёмов / Windows ring3.
