이 프로젝트는 Windows 커널 섀도 스택(CET - Control-flow Enforcement Technology) 구현과 보안 영향에 대한 연구입니다. Windows가 Intel CET를 사용해 커널 모드 섀도 스택을 구현하는 방식, PatchGuard와의 상호작용, 그리고 우회 또는 약화 시나리오를 문서화합니다. 분석은 KVAS 섀도 스택 초기화, 예외 처리, 기존 커널 익스플로잇 기법과의 호환성을 다룹니다. CET가 커널 익스플로잇과 Windows 커널 강화에 미치는 영향을 연구하는 커널 보안 연구자를 대상으로 합니다.
