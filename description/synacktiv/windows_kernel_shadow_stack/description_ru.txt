Этот проект посвящён исследованию реализации теневого стека ядра Windows (CET - Control-flow Enforcement Technology) и её последствий для безопасности. В нём описано, как Windows реализует теневые стеки в режиме ядра с помощью Intel CET, взаимодействие с PatchGuard, а также возможные сценарии обхода или ослабления защиты. Анализ охватывает инициализацию теневого стека KVAS, обработку исключений и совместимость с существующими техниками эксплуатации ядра. Материал рассчитан на исследователей безопасности ядра, изучающих влияние CET на эксплуатацию ядра и hardening ядра Windows.
