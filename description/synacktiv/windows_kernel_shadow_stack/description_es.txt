Este proyecto investiga la implementación de la shadow stack del kernel de Windows (CET - Control-flow Enforcement Technology) y sus implicaciones de seguridad. Documenta cómo Windows implementa las shadow stacks en modo kernel mediante Intel CET, la interacción con PatchGuard y posibles escenarios de bypass o debilitamiento. El análisis abarca la inicialización de la shadow stack KVAS, el manejo de excepciones y la compatibilidad con técnicas existentes de explotación del kernel. Está dirigido a investigadores de seguridad del kernel que estudian el impacto de CET en la explotación del kernel y el endurecimiento del kernel de Windows.
