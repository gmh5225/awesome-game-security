本プロジェクトは、Windows カーネルのシャドウスタック（CET - Control-flow Enforcement Technology）の実装とセキュリティ上の影響に関する研究です。Windows が Intel CET を用いてカーネルモードのシャドウスタックをどのように実装するか、PatchGuard との相互作用、および回避や弱体化の可能性を文書化しています。分析は KVAS シャドウスタックの初期化、例外処理、既存のカーネルエクスプロイト手法との互換性を扱います。CET がカーネルエクスプロイトと Windows カーネルの強化に与える影響を研究するカーネルセキュリティ研究者向けです。
