Questo progetto studia l'implementazione dello shadow stack del kernel Windows (CET - Control-flow Enforcement Technology) e le relative implicazioni di sicurezza. Documenta come Windows implementa gli shadow stack in modalità kernel tramite Intel CET, l'interazione con PatchGuard e possibili scenari di bypass o indebolimento. L'analisi riguarda l'inizializzazione dello shadow stack KVAS, la gestione delle eccezioni e la compatibilità con le tecniche di exploitation del kernel esistenti. È rivolto ai ricercatori di sicurezza del kernel che studiano l'impatto di CET sull'exploitation del kernel e sul rafforzamento del kernel Windows.
