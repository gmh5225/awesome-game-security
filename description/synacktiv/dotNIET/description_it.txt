Questo progetto è dotNIET, un plugin per IDA Pro per deoffuscare binari compilati con .NET Native (NativeAOT). Ricostruisce le informazioni sui tipi, i nomi dei metodi e i riferimenti ai metadati persi durante la compilazione .NET Native AOT, mappando il codice nativo ai tipi e metodi .NET originali. Il plugin Python aiuta ad analizzare applicazioni .NET compilate in codice nativo. È rivolto agli ingegneri di reverse engineering che analizzano binari .NET Native dove non è possibile usare gli strumenti standard di decompilazione .NET.
