Este proyecto es dotNIET, un complemento de IDA Pro para desofuscar binarios compilados con .NET Native (NativeAOT). Reconstruye la información de tipos, los nombres de métodos y las referencias de metadatos que se pierden durante la compilación .NET Native AOT, mapeando el código nativo de nuevo a los tipos y métodos .NET originales. El complemento de Python ayuda a analizar aplicaciones .NET que han sido compiladas a código nativo. Está dirigido a ingenieros de ingeniería inversa que analizan binarios .NET Native donde no se pueden usar las herramientas estándar de descompilación de .NET.
