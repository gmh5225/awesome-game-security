Ce projet est dotNIET, un plugin IDA Pro pour désobfusquer les binaires compilés avec .NET Native (NativeAOT). Il reconstruit les informations de types, les noms de méthodes et les références de métadonnées perdues lors de la compilation .NET Native AOT, en faisant correspondre le code natif aux types et méthodes .NET d'origine. Ce plugin Python facilite l'analyse des applications .NET compilées en code natif. Il s'adresse aux ingénieurs en rétro-ingénierie qui analysent des binaires .NET Native où les outils de décompilation .NET standards ne peuvent pas être utilisés.
