Dieses Projekt ist dotNIET, ein IDA-Pro-Plugin zur Deobfuskation von mit .NET Native (NativeAOT) kompilierten Binärdateien. Es rekonstruiert Typinformationen, Methodennamen und Metadatenreferenzen, die bei der .NET-Native-AOT-Kompilierung verloren gehen, und ordnet nativen Code den ursprünglichen .NET-Typen und -Methoden zu. Das Python-Plugin hilft bei der Analyse von .NET-Anwendungen, die zu nativem Code kompiliert wurden. Es richtet sich an Reverse Engineers, die .NET-Native-Binärdateien analysieren, bei denen Standard-.NET-Dekompilierungswerkzeuge nicht verwendet werden können.
