Этот проект — dotNIET, плагин для IDA Pro для деобфускации бинарных файлов, скомпилированных с помощью .NET Native (NativeAOT). Он восстанавливает информацию о типах, имена методов и ссылки на метаданные, утраченные при компиляции .NET Native AOT, сопоставляя нативный код с исходными типами и методами .NET. Плагин на Python помогает анализировать приложения .NET, скомпилированные в нативный код. Он предназначен для специалистов по реверс-инжинирингу, анализирующих бинарные файлы .NET Native, где стандартные инструменты декомпиляции .NET неприменимы.
