Este proyecto es un ejemplo en C++ para Windows que usa el controlador y la API de WinRing0 para leer telemetría de CPU de bajo nivel, especialmente las temperaturas de los núcleos. Accede a datos de CPUID y MSR mediante la biblioteca en modo usuario y los componentes del controlador en modo kernel incluidos, y registra los resultados en un archivo local. El repositorio incluye encabezados, binarios y un programa de consola sencillo para el monitoreo por núcleo con privilegios de administrador. Su principal caso de uso es la monitorización de hardware y la práctica de programación de sistemas Windows de bajo nivel.
