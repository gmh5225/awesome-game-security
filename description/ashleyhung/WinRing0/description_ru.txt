Этот проект — пример на Windows C++, который использует драйвер и API WinRing0 для чтения низкоуровневой телеметрии ЦП, в частности температур ядер. Он обращается к данным CPUID и MSR через прилагаемую пользовательскую библиотеку и компоненты драйвера ядра, затем записывает результаты в локальный файл. Репозиторий содержит заголовки, двоичные файлы и простую консольную программу для мониторинга по ядрам с правами администратора. Основное применение — мониторинг оборудования и практика низкоуровневого системного программирования Windows.
