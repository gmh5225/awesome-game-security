Questo progetto è un esempio Windows in C++ che usa il driver e l'API WinRing0 per leggere telemetria CPU a basso livello, in particolare le temperature dei core. Accede ai dati CPUID e MSR tramite la libreria in user mode e i componenti del driver kernel inclusi, quindi registra i risultati in un file locale. Il repository include header, binari e un semplice programma console per il monitoraggio per core con privilegi di amministratore. Il caso d'uso principale è il monitoraggio hardware e la pratica di programmazione di sistemi Windows a basso livello.
