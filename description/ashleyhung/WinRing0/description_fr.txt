Ce projet est un exemple Windows en C++ qui utilise le pilote et l'API WinRing0 pour lire la télémétrie CPU de bas niveau, notamment les températures des cœurs. Il accède aux données CPUID et MSR via la bibliothèque en mode utilisateur et les composants du pilote noyau fournis, puis enregistre les résultats dans un fichier local. Le dépôt inclut les en-têtes, les binaires et un programme console simple pour la surveillance par cœur sous privilèges administrateur. Son principal cas d'usage est la surveillance matérielle et la pratique de la programmation système Windows de bas niveau.
