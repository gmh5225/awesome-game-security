HydraHook es un framework DLL en C++ para Windows que engancha DirectX y APIs relacionadas para renderizar overlays personalizados dentro de procesos de juegos ajenos, sin necesidad de un conocimiento profundo de Direct3D o del hooking. Expone una API pequeña, detecta en tiempo de ejecución la versión de DirectX del juego y admite DirectX 9 a 12, además de hooks de Core Audio y relacionados con la entrada. Incluye inyección y expulsión a mitad de proceso, un hot path sin bloqueos para callbacks, un apagado seguro frente al loader lock y un runtime autocontenido con dependencias como Microsoft Detours enlazadas estáticamente. Las DLL host de ejemplo demuestran overlays con ImGui, DirectXTK y OpenCV sobre la biblioteca principal. Está dirigido a desarrolladores que exploran overlays de juegos, hooking de APIs y técnicas de ingeniería inversa en títulos DirectX que poseen, con precaución en entornos anti-cheat.
