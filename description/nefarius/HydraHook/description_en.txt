HydraHook is a Windows C++ DLL framework that hooks DirectX and related APIs so you can render custom overlays inside foreign game processes without deep Direct3D or hooking expertise. It exposes a small API, detects the game’s DirectX version at runtime, and supports DirectX 9 through 12, with Core Audio and input-related hooks as well. Features include mid-process inject and eject, a lock-free hot path for callbacks, loader-lock-safe shutdown, and a self-contained runtime with dependencies such as Microsoft Detours statically linked. Sample host DLLs demonstrate ImGui, DirectXTK, and OpenCV overlays on top of the core library. It is aimed at developers exploring game overlays, API hooking, and reverse-engineering techniques on DirectX titles they own, with caution around anti-cheat environments.
