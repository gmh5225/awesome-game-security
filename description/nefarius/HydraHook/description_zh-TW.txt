HydraHook 是一套 Windows C++ DLL 框架，透過掛鉤 DirectX 及相關 API，讓你能在外部遊戲程序中繪製自訂疊加層，而無需深入掌握 Direct3D 或掛鉤技術。它提供精簡 API，在執行期間偵測遊戲的 DirectX 版本，支援 DirectX 9 到 12，並包含 Core Audio 與輸入相關掛鉤。功能包括程序中注入與卸載、回呼的無鎖熱路徑、對載入器鎖安全的關閉，以及自包含執行階段（例如將 Microsoft Detours 靜態連結）。範例主機 DLL 在核心程式庫之上示範了 ImGui、DirectXTK 與 OpenCV 疊加層。面向在自有 DirectX 遊戲上探索遊戲疊加、API 掛鉤與逆向工程技術的開發者，並提醒在反作弊環境下需謹慎使用。
