HydraHook는 DirectX 및 관련 API를 후킹하여 깊은 Direct3D나 후킹 전문 지식 없이도 외부 게임 프로세스 안에 커스텀 오버레이를 렌더링할 수 있게 해주는 Windows C++ DLL 프레임워크입니다. 작은 API를 제공하고 런타임에 게임의 DirectX 버전을 감지하며 DirectX 9부터 12까지 지원하고, Core Audio 및 입력 관련 후킹도 포함합니다. 프로세스 중 inject/eject, 콜백용 락프리 핫 패스, 로더 락에 안전한 종료, Microsoft Detours 등을 정적 링크한 자체 완결 런타임 같은 기능이 있습니다. 샘플 호스트 DLL은 코어 라이브러리 위에서 ImGui, DirectXTK, OpenCV 오버레이를 보여 줍니다. 소유한 DirectX 타이틀에서 게임 오버레이, API 후킹, 리버스 엔지니어링 기법을 탐구하는 개발자를 대상으로 하며, 안티치트 환경에서는 주의가 필요합니다.
