HydraHook è un framework DLL C++ per Windows che effettua l’hook di DirectX e delle API correlate per renderizzare overlay personalizzati nei processi di giochi esterni, senza una conoscenza approfondita di Direct3D o dell’hooking. Espone una piccola API, rileva a runtime la versione DirectX del gioco e supporta DirectX da 9 a 12, con hook di Core Audio e relativi all’input. Tra le funzionalità: inject ed eject a processo in corso, un hot path lock-free per i callback, spegnimento sicuro rispetto al loader lock e un runtime autonomo con dipendenze come Microsoft Detours collegate staticamente. Le DLL host di esempio mostrano overlay ImGui, DirectXTK e OpenCV sopra la libreria core. È rivolto agli sviluppatori che esplorano overlay di gioco, API hooking e tecniche di reverse engineering su titoli DirectX di loro proprietà, con cautela negli ambienti anti-cheat.
