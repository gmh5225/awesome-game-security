HydraHook — это Windows C++ DLL-фреймворк, который перехватывает DirectX и связанные API, чтобы рисовать пользовательские оверлеи во внешних игровых процессах без глубоких знаний Direct3D или хукинга. Он предоставляет небольшой API, во время выполнения определяет версию DirectX игры и поддерживает DirectX 9–12, а также хуки Core Audio и ввода. Среди возможностей: inject и eject в уже запущенный процесс, lock-free горячий путь для колбэков, безопасное завершение с учётом loader lock и самодостаточный runtime со статически слинкованными зависимостями вроде Microsoft Detours. Примеры host-DLL демонстрируют оверлеи на ImGui, DirectXTK и OpenCV поверх основной библиотеки. Предназначен для разработчиков, изучающих игровые оверлеи, API-хукинг и reverse engineering на принадлежащих им DirectX-играх, с осторожностью в средах с античитом.
