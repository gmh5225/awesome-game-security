HydraHook は、DirectX および関連 API をフックし、深い Direct3D やフックの専門知識なしに外部ゲームプロセス内へカスタムオーバーレイを描画できる Windows C++ DLL フレームワークです。小さな API を公開し、実行時にゲームの DirectX バージョンを検出して DirectX 9 から 12 までをサポートし、Core Audio や入力関連のフックも備えます。プロセス途中での inject／eject、コールバック向けのロックフリーなホットパス、ローダーロック安全なシャットダウン、Microsoft Detours などを静的リンクした自己完結ランタイムといった機能があります。サンプルホスト DLL はコアライブラリ上で ImGui、DirectXTK、OpenCV オーバーレイを示します。所有する DirectX タイトルでのゲームオーバーレイ、API フック、リバースエンジニアリング手法の探求向けで、アンチチート環境では注意が必要です。
