Этот проект сосредоточен на usermode-эксплойте для обхода любого античита (AC) с помощью 0day shatter-атаки.
Он в основном написан на C++ и посвящён исследованию античитов, рендерингу и моддингу.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / RPM.
