Этот проект — KACE (Kernel Automated Compatibility Emulator), эмулятор ядра Windows, который запускает драйверы режима ядра в пользовательском режиме для анализа. Он эмулирует основные API ядра, управление памятью, обработку IRQL и соглашения о вызовах точек входа драйверов, позволяя исследователям безопасности загружать и анализировать драйверы ядра без риска для стабильности системы. Эмулятор на C++ предоставляет изолированную среду для изучения поведения драйверов. Он предназначен для реверс-инженеров и аналитиков античитов, которым нужно безопасно анализировать драйверы ядра и компоненты античита.
