本プロジェクトは KACE（Kernel Automated Compatibility Emulator）であり、カーネルモードドライバをユーザーモードで実行して解析できる Windows カーネルエミュレータです。主要なカーネル API、メモリ管理、IRQL 処理、ドライバエントリポイントの呼び出し規約をエミュレートし、セキュリティ研究者がシステムの安定性を損なわずにカーネルドライバをロードして解析できるようにします。この C++ エミュレータは、ドライバの挙動を調査するためのサンドボックス環境を提供し、カーネルドライバやアンチチートコンポーネントを安全に解析する必要があるリバースエンジニアおよびアンチチート分析者を対象としています。
