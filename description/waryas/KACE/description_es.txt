Este proyecto es KACE (Kernel Automated Compatibility Emulator), un emulador de kernel de Windows que ejecuta controladores en modo kernel en modo usuario para su análisis. Emula las API esenciales del kernel, la gestión de memoria, el manejo de IRQL y las convenciones de llamada de los puntos de entrada de los controladores, permitiendo a los investigadores de seguridad cargar y analizar controladores de kernel sin poner en riesgo la estabilidad del sistema. El emulador en C++ proporciona un entorno aislado para estudiar el comportamiento de los controladores. Está dirigido a ingenieros inversos y analistas de anti-trampas que necesitan analizar de forma segura controladores de kernel y componentes anti-trampas.
