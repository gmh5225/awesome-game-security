이 프로젝트는 KACE(Kernel Automated Compatibility Emulator)로, 커널 모드 드라이버를 사용자 모드에서 실행하여 분석할 수 있게 해주는 Windows 커널 에뮬레이터입니다. 필수 커널 API, 메모리 관리, IRQL 처리, 드라이버 진입점 호출 규약을 에뮬레이션하여 보안 연구자가 시스템 안정성을 위협하지 않고 커널 드라이버를 로드하고 분석할 수 있게 합니다. 이 C++ 에뮬레이터는 드라이버 동작을 연구하기 위한 샌드박스 환경을 제공하며, 커널 드라이버와 안티치트 구성 요소를 안전하게 분석해야 하는 리버스 엔지니어와 안티치트 분석가를 대상으로 합니다.
