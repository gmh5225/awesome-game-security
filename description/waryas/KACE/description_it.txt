Questo progetto è KACE (Kernel Automated Compatibility Emulator), un emulatore del kernel Windows che esegue driver in modalità kernel in user-mode per l'analisi. Emula le API essenziali del kernel, la gestione della memoria, la gestione degli IRQL e le convenzioni di chiamata dei punti di ingresso dei driver, consentendo ai ricercatori di sicurezza di caricare e analizzare driver del kernel senza compromettere la stabilità del sistema. L'emulatore in C++ fornisce un ambiente sandbox per studiare il comportamento dei driver. È rivolto a reverse engineer e analisti anti-cheat che devono analizzare in sicurezza driver del kernel e componenti anti-cheat.
