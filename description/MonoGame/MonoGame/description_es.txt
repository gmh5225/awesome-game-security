Este proyecto es un framework de juegos .NET multiplataforma para crear juegos 2D y 3D. La base de código es principalmente C#, con componentes nativos de apoyo, y ofrece amplios módulos de gráficos, audio, procesamiento de contenido, entrada y abstracciones de plataforma. Incluye bibliotecas del framework, plantillas, herramientas y recursos de ejemplo para admitir entornos de escritorio, móviles y otros destinos. Su principal caso de uso es el desarrollo de juegos en producción y la experimentación a nivel de motor en flujos de trabajo con lenguajes administrados.
