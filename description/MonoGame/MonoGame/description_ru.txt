Этот проект — кроссплатформенный игровой фреймворк на .NET для создания 2D- и 3D-игр. Кодовая база в основном на C# с нативными компонентами и предоставляет обширные модули для графики, аудио, обработки контента, ввода и платформенных абстракций. В комплекте — библиотеки фреймворка, шаблоны, инструменты и примеры ассетов для поддержки настольных, мобильных и других целевых сред. Основной сценарий использования — промышленная разработка игр и эксперименты на уровне движка в управляемых языковых рабочих процессах.
