Этот проект — репозиторий из раздела anti cheat / game engine protection:unity, посвящённый il2cpp encrtypt.
Он в основном написан на C++ и C/C++ и сосредоточен на пайплайнах ассетов, разработке плагинов и Unity.
Он прежде всего полезен инженерам античита и исследователям защитной безопасности, работающим в области anti cheat / game engine protection:unity.
