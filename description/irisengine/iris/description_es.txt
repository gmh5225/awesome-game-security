Este proyecto es un motor de juego C++ multiplataforma llamado Iris, con arquitectura basada en componentes, sistema de compilación CMake y soporte para múltiples backends de renderizado. Incluye sistemas de cámara, gestión del color, utilidades de smart pointers e interfaces basadas en concepts del C++ moderno. Resulta especialmente útil para desarrolladores de motores de juego que estudian un diseño ligero y moderno en C++ con patrones de metaprogramación con plantillas.
