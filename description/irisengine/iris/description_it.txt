Questo progetto è un motore di gioco C++ multipiattaforma chiamato Iris, con architettura basata su componenti, sistema di build CMake e supporto per più backend di rendering. Include sistemi di telecamera, gestione del colore, utilità per smart pointer e interfacce basate sui concepts del C++ moderno. È soprattutto utile per gli sviluppatori di motori di gioco che studiano un design di engine C++ leggero e moderno con pattern di metaprogrammazione a template.
