Questo progetto è PCM (Process Context Monitor), uno strumento o una libreria per monitorare a basso livello i context switch dei processi, lo scheduling dei thread e l'utilizzo della CPU. Cattura eventi di scheduling per core e dati di context switch per offrire una visibilità dettagliata sui modelli di esecuzione di processi e thread nel sistema. È rivolto ad analisti delle prestazioni e ricercatori di sicurezza che studiano il comportamento di scheduling dei processi e il consumo di risorse CPU.
