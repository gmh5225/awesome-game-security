Этот проект — PCM (Process Context Monitor), инструмент или библиотека для низкоуровневого мониторинга переключений контекста процессов, планирования потоков и загрузки CPU. Он собирает события планирования по ядрам и данные о переключениях контекста, обеспечивая детальную видимость шаблонов выполнения процессов и потоков в системе. Предназначен для аналитиков производительности и исследователей безопасности, изучающих поведение планирования процессов и потребление ресурсов CPU.
