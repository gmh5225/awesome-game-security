Hotkey-based Keylogger Detector es un controlador del kernel de Windows que detecta keyloggers que abusan del registro de teclas rápidas globales. Escanea los internos de win32kfull para resolver la tabla de hotkeys globales e inspecciona las entradas registradas en busca de patrones de uso sospechosos. La implementación está escrita en C++ como un controlador KMDF, con orientación de instalación y depuración para entornos de prueba. Este proyecto está pensado para investigación defensiva de endpoints y pruebas de seguridad de Windows.
