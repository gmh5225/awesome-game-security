Hotkey-based Keylogger Detector — это драйвер ядра Windows, который обнаруживает кейлоггеры, злоупотребляющие регистрацией глобальных горячих клавиш. Он сканирует внутренности win32kfull, чтобы разрешить глобальную таблицу hotkey, и проверяет зарегистрированные записи на подозрительные шаблоны использования. Реализация написана на C++ как KMDF-драйвер и включает рекомендации по установке и отладке для тестовых сред. Проект предназначен для защитных исследований конечных точек и тестирования безопасности Windows.
