Hotkey-based Keylogger Detector è un driver del kernel Windows che rileva i keylogger che abusano della registrazione degli hotkey globali. Analizza le strutture interne di win32kfull per risolvere la tabella degli hotkey globali e ispeziona le voci registrate in cerca di schemi di utilizzo sospetti. L'implementazione è scritta in C++ come driver KMDF, con indicazioni di installazione e debug per ambienti di test. Questo progetto è destinato alla ricerca difensiva sugli endpoint e ai test di sicurezza Windows.
