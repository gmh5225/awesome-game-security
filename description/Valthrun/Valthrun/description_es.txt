Este proyecto es un framework de código abierto externo, de solo lectura y a nivel de kernel, para mejorar el gameplay de Counter-Strike 2.
Está construido principalmente en Rust y combina un controlador de kernel, un renderizador de overlay y componentes de radar, sin inyectar DLL en el proceso del juego.
Sus capacidades clave incluyen ESP de jugadores configurable, información de bomba y espectadores, funciones de trigger y un overlay a prueba de streaming.
Está orientado principalmente a la investigación de seguridad de juegos y anti-detección de bajo nivel, especialmente para desarrolladores que estudian técnicas del kernel de Windows.
