Этот проект — открытый внешний фреймворк только для чтения на уровне ядра для улучшения геймплея в Counter-Strike 2.
Он в основном написан на Rust и сочетает драйвер ядра, рендерер оверлея и компоненты радара, не внедряя DLL в процесс игры.
Ключевые возможности включают настраиваемый ESP игроков, информацию о бомбе и зрителях, функции триггера и оверлей, защищённый от стрима.
Он в первую очередь предназначен для исследований низкоуровневой безопасности игр и антидетекта, особенно для разработчиков, изучающих техники ядра Windows.
