Этот проект представляет собой маппер на базе UEFI, который загружает игровой драйвер во время загрузки системы.
Он написан на Rust для целевой платформы x86_64 UEFI и включает скрипты для создания загрузочных ISO-образов, предназначенных для развёртывания с USB.
Загрузчик может стартовать раньше обычных драйверов операционной системы, что полезно для экспериментов по ранней инициализации драйверов.
Основной сценарий использования — исследование загрузки драйверов на этапе загрузки системы и скрытных рабочих процессов безопасности Windows.
