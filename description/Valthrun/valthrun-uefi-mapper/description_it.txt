Questo progetto è un mapper basato su UEFI che carica un driver di gioco durante l'avvio.
È scritto in Rust per il target UEFI x86_64 e include script per creare immagini ISO avviabili destinate al deployment su USB.
Il loader può avviarsi prima dei driver normali del sistema operativo, il che è utile per esperimenti di inizializzazione precoce dei driver.
Il caso d'uso principale è la ricerca sul caricamento di driver all'avvio e sui flussi di lavoro di sicurezza Windows orientati alla stealth.
