Questo progetto è un repository mirror e di automazione della build per il kernel Linux di Windows Subsystem for Android, incluse varianti patchate per superuser.
Mantiene varianti di branch per il codice sorgente stock e il codice sorgente con KernelSU abilitato, e fornisce pipeline GitHub Actions per immagini kernel x86_64 e arm64.
Uno script shell di supporto può applicare le voci di configurazione KernelSU necessarie negli alberi del kernel WSA.
È usato principalmente da ricercatori di sicurezza Android e di piattaforma che necessitano di build riproducibili del kernel WSA e di personalizzazione.
