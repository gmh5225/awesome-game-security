이 프로젝트는 Windows Subsystem for Android(WSA) Linux 커널의 미러 및 빌드 자동화 저장소이며, 슈퍼유저 패치가 적용된 변형도 포함합니다.
스톡 소스와 KernelSU가 활성화된 소스의 브랜치 변형을 유지하며, x86_64 및 arm64 커널 이미지용 GitHub Actions 파이프라인을 제공합니다.
도우미 셸 스크립트로 필요한 KernelSU 구성 항목을 WSA 커널 트리에 패치할 수 있습니다.
재현 가능한 WSA 커널 빌드와 커스터마이징이 필요한 Android 및 플랫폼 보안 연구자들이 주로 사용합니다.
