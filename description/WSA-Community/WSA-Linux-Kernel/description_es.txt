Este proyecto es un repositorio espejo y de automatización de compilación del kernel Linux de Windows Subsystem for Android, incluidas variantes parcheadas para superusuario.
Mantiene variantes de ramas para el código fuente stock y el código fuente con KernelSU habilitado, y proporciona pipelines de GitHub Actions para imágenes de kernel x86_64 y arm64.
Un script shell auxiliar puede aplicar las entradas de configuración de KernelSU necesarias en los árboles del kernel de WSA.
Lo utilizan principalmente investigadores de seguridad de Android y de plataformas que necesitan compilaciones reproducibles del kernel de WSA y personalización.
