Этот проект — зеркало и репозиторий автоматизации сборки Linux-ядра Windows Subsystem for Android, включая варианты с патчами для суперпользователя.
Он поддерживает варианты веток для исходного кода stock и исходного кода с включённым KernelSU, а также предоставляет конвейеры GitHub Actions для образов ядра x86_64 и arm64.
Вспомогательный shell-скрипт может вносить необходимые записи конфигурации KernelSU в деревья ядра WSA.
Его в основном используют исследователи безопасности Android и платформ, которым нужны воспроизводимые сборки ядра WSA и его настройка.
