Ce projet est un dépôt miroir et d'automatisation de build pour le noyau Linux de Windows Subsystem for Android, y compris des variantes patchées pour le superutilisateur.
Il maintient des variantes de branches pour le code source stock et le code source avec KernelSU activé, et fournit des pipelines GitHub Actions pour les images noyau x86_64 et arm64.
Un script shell d'aide peut appliquer les entrées de configuration KernelSU requises dans les arbres du noyau WSA.
Il est principalement utiliséé par les chercheurs en sécurité Android et plateforme qui ont besoin de builds WSA reproductibles et de personnalisation.
