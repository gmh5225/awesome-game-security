本プロジェクトは、Windows Subsystem for Android（WSA）用 Linux カーネルのミラーおよびビルド自動化リポジトリであり、スーパーユーザー向けパッチ適用済みのバリアントも含みます。
ストックソースと KernelSU 有効化ソースのブランチバリアントを維持し、x86_64 および arm64 カーネルイメージ向けの GitHub Actions パイプラインを提供します。
補助シェルスクリプトにより、必要な KernelSU 設定エントリを WSA カーネルツリーにパッチ適用できます。
再現可能な WSA カーネルビルドとカスタマイズを必要とする Android およびプラットフォームセキュリティ研究者を主な利用対象としています。
