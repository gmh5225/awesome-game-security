EfiGuard 是一款可移植的 x64 UEFI bootkit，透過修補 Windows 啟動鏈，在啟動過程中停用 PatchGuard 與驅動程式簽章強制（Driver Signature Enforcement）。
程式碼庫以 C 與 C++ 為主，包含 EDK2 UEFI 元件、載入器應用程式，以及 EfiDSEFix 等輔助工具。
它支援廣泛的 Windows x64 版本，利用執行階段反組譯實現穩健修補，並提供包括基於 SetVariable 的控制路徑在內的多種啟動時修補模式。
此專案主要面向研究啟動完整性、核心保護機制，以及反作弊或驅動程式載入繞過行為的底層 Windows 安全性研究人員。
