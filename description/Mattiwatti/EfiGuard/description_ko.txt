EfiGuard는 Windows 부트 체인을 패치하여 시작 시 PatchGuard와 Driver Signature Enforcement를 비활성화하는 휴대용 x64 UEFI bootkit입니다.
코드베이스는 주로 C와 C++로 작성되었으며, EDK2 UEFI 구성 요소, 로더 애플리케이션, EfiDSEFix와 같은 보조 도구를 포함합니다.
다양한 Windows x64 버전을 지원하고, 런타임 디스어셈블리를 사용해 견고한 패치를 적용하며, SetVariable 기반 제어 경로를 포함한 여러 부트 타임 패치 모드를 제공합니다.
이 프로젝트는 부트 무결성, 커널 보호, 그리고 안티치트나 드라이버 로딩 우회 동작을 연구하는 저수준 Windows 보안 연구자를 주요 대상으로 합니다.
