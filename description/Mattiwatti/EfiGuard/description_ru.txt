EfiGuard — это переносимый x64 UEFI bootkit, который патчит цепочку загрузки Windows, чтобы отключить PatchGuard и Driver Signature Enforcement при старте.
Кодовая база в основном на C и C++ и включает компоненты EDK2 UEFI, приложение-загрузчик, а также вспомогательные инструменты вроде EfiDSEFix.
Он поддерживает широкий диапазон версий Windows x64, использует runtime-дизассемблирование для устойчивого патчинга и предоставляет несколько режимов патча на этапе загрузки, включая пути управления на основе SetVariable.
Проект ориентирован в первую очередь на исследователей низкоуровневой безопасности Windows, изучающих целостность загрузки, защиту ядра, а также обход античита или загрузки драйверов.
