EfiGuard は、Windows のブートチェーンをパッチして起動時に PatchGuard と Driver Signature Enforcement を無効化する、ポータブルな x64 UEFI bootkit です。
コードベースは主に C と C++ で構成され、EDK2 UEFI コンポーネント、ローダーアプリケーション、EfiDSEFix などの補助ツールを含みます。
幅広い Windows x64 バージョンをサポートし、ランタイム逆アセンブルによる堅牢なパッチ適用と、SetVariable ベースの制御パスを含む複数のブート時パッチモードを提供します。
本プロジェクトは、ブート整合性、カーネル保護、およびアンチチートやドライバー読み込み回避の挙動を研究する低レイヤー Windows セキュリティ研究者を主な対象としています。
