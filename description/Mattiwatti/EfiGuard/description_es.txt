EfiGuard es un bootkit UEFI x64 portable que parchea la cadena de arranque de Windows para desactivar PatchGuard y Driver Signature Enforcement durante el inicio.
El código está escrito principalmente en C y C++, con componentes UEFI de EDK2, una aplicación cargadora y herramientas auxiliares como EfiDSEFix.
Admite una amplia gama de versiones de Windows x64, usa desensamblado en tiempo de ejecución para un parcheo robusto y ofrece varios modos de parcheo en el arranque, incluidos caminos de control basados en SetVariable.
Este proyecto está dirigido principalmente a investigadores de seguridad de Windows de bajo nivel que estudian la integridad del arranque, las protecciones del núcleo y el comportamiento de bypass relacionado con anti-cheat o la carga de controladores.
