EfiGuard 是一款可移植的 x64 UEFI bootkit，通过修补 Windows 启动链，在启动过程中禁用 PatchGuard 与驱动签名强制（Driver Signature Enforcement）。
代码库以 C 和 C++ 为主，包含 EDK2 UEFI 组件、加载器应用，以及 EfiDSEFix 等辅助工具。
它支持广泛的 Windows x64 版本，利用运行时反汇编实现稳健修补，并提供包括基于 SetVariable 的控制路径在内的多种启动时修补模式。
该项目主要面向研究启动完整性、内核保护机制，以及反作弊或驱动加载绕过行为的底层 Windows 安全研究人员。
