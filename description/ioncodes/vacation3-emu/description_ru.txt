Этот проект — эмулятор модулей VAC3 (Valve Anti-Cheat версии 3), который загружает и выполняет модули сканирования VAC в контролируемой среде вне клиента Steam. Он эмулирует интерфейс загрузки модулей, предоставляет поддельную игровую память для сканирования и журналирует, какие области памяти и шаблоны проверяют модули VAC. Эмулятор на C++ помогает исследователям понять методологию обнаружения VAC. Он предназначен для исследователей античита, занимающихся реверс-инжинирингом поведения сканирования и сигнатур обнаружения VAC3.
