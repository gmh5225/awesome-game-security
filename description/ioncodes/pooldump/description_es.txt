Este proyecto es un dumper de memoria del pool del kernel de Windows que extrae y muestra el contenido de las asignaciones del pool del kernel. Escanea páginas del pool del kernel para enumerar bloques asignados, sus pool tags, tamaños y controladores propietarios, y puede volcar el contenido de asignaciones específicas del pool. La herramienta ayuda a identificar artefactos en modo kernel dejados por controladores y rootkits en la memoria del pool. Está dirigida a investigadores de forense de kernel y analistas anti-cheat que examinan artefactos del pool del kernel en busca de evidencia de controladores mapeados manualmente.
