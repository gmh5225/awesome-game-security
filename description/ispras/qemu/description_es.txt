Este proyecto es un fork modificado de QEMU del ISP RAS (Institute for System Programming of the Russian Academy of Sciences) con capacidades adicionales de análisis e instrumentación. Amplía el motor de traducción binaria dinámica de QEMU con funciones para el análisis de programas, la recolección de cobertura de código, el seguimiento de taint o el soporte de ejecución simbólica. Está dirigido a investigadores de sistemas y analistas de seguridad que usan QEMU como plataforma para el análisis binario y la investigación de vulnerabilidades.
