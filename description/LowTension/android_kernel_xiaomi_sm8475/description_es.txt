Este proyecto es el código fuente del kernel de Android para dispositivos Xiaomi SM8475 (Snapdragon 8+ Gen 1). Incluye el árbol completo del kernel con definiciones de ABI GKI para varios fabricantes, soporte de plataforma específico de Qualcomm y configuraciones de dispositivos. Resulta especialmente útil para desarrolladores de kernel de Android e investigadores de seguridad móvil que trabajan con compilaciones de kernel personalizadas en hardware Qualcomm Snapdragon 8+ Gen 1.
