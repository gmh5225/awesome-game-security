Этот проект представляет собой исходный код ядра Android для устройств Xiaomi SM8475 (Snapdragon 8+ Gen 1). Он включает полное дерево ядра с определениями GKI ABI для нескольких производителей, поддержку платформы Qualcomm и конфигурации устройств. В первую очередь полезен разработчикам ядра Android и исследователям мобильной безопасности, работающим с пользовательскими сборками ядра на оборудовании Qualcomm Snapdragon 8+ Gen 1.
