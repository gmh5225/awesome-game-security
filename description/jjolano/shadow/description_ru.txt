Этот проект — обход обнаружения джейлбрейка для современных джейлбрейков iOS.
Shadow не гарантирует работу со всеми приложениями, но здесь есть несколько идей, которые стоит попробовать.
Он в первую очередь полезен исследователям игровой безопасности и инженерам по реверс-инжинирингу, изучающим атакующие техники в области читов / джейлбрейка iOS.
