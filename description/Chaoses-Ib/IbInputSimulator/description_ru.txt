Этот проект — библиотека симуляции ввода для Windows, которая отправляет события клавиатуры и мыши через методы на уровне драйверов. Поддерживаются несколько бэкендов, включая программные пути Logitech, Razer Synapse, MouClassInputInjection и виртуальные устройства DD, с примерами интеграции AutoHotkey. Репозиторий объединяет компоненты на C и C++ со скриптами AHK, предоставляя единый интерфейс инициализации и отправки для разных драйверов. В основном используется для автоматизации, игровых инструментов и исследований обхода античита, когда стандартных API ввода недостаточно.
