Questo progetto è una libreria di simulazione dell'input per Windows che invia eventi di tastiera e mouse tramite metodi basati su driver. Supporta più backend, tra cui percorsi software Logitech, Razer Synapse, MouClassInputInjection e dispositivi virtuali DD, con esempi di integrazione AutoHotkey. Il repository combina componenti C e C++ con script AHK per fornire un'interfaccia unificata di inizializzazione e invio tra i driver. È usato soprattutto per automazione, tooling di gioco e ricerca sull'evasione degli anti-cheat quando le API di input standard non sono sufficienti.
