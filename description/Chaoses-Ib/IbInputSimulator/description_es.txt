Este proyecto es una biblioteca de simulación de entrada para Windows que envía eventos de teclado y ratón mediante métodos respaldados por controladores. Admite varios backends, incluidos rutas de software Logitech, Razer Synapse, MouClassInputInjection y dispositivos virtuales DD, con ejemplos de integración con AutoHotkey. El repositorio combina componentes en C y C++ con scripts AHK para ofrecer una interfaz unificada de inicialización y envío entre controladores. Se usa principalmente en automatización, herramientas para juegos e investigación de evasión de anti-cheat cuando las API de entrada estándar no bastan.
