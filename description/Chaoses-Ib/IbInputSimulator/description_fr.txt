Ce projet est une bibliothèque de simulation d'entrée Windows qui envoie des événements clavier et souris via des méthodes basées sur des pilotes. Elle prend en charge plusieurs backends, notamment les chemins logiciels Logitech, Razer Synapse, MouClassInputInjection et les périphériques virtuels DD, avec des exemples d'intégration AutoHotkey. Le dépôt combine des composants C et C++ avec des scripts AHK pour offrir une interface unifiée d'initialisation et d'envoi sur différents pilotes. Il est principalement utilisé pour l'automatisation, les outils de jeu et la recherche sur le contournement d'anti-cheat lorsque les API d'entrée standard sont insuffisantes.
