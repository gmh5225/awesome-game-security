Lazenca S est un moteur anti-triche Android conçu pour détecter les comportements courants de falsification de jeux.
Le projet combine des composants Java Android avec des bibliothèques partagées natives C/C++ chargées via JNI.
Ses contrôles exportés couvrent le débogage, le rootage, les speed hacks, la modification de binaires et des indicateurs liés aux machines virtuelles.
Il est destiné aux expériences de protection de jeux mobiles et à l’étude des techniques de détection anti-triche sur Android.
