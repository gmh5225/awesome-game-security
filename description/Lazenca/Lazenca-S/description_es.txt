Lazenca S es un motor anti-trampas para Android diseñado para detectar comportamientos habituales de manipulación de juegos.
El proyecto combina componentes Java de Android con bibliotecas compartidas nativas en C/C++ cargadas mediante JNI.
Sus comprobaciones exportadas cubren depuración, rooting, speed hacks, modificación de binarios e indicadores relacionados con máquinas virtuales.
Está pensado para experimentos de protección de juegos móviles y para estudiar técnicas de detección anti-trampas en Android.
