Lazenca S è un motore anti-cheat per Android progettato per rilevare comportamenti comuni di manomissione dei giochi.
Il progetto combina componenti Java Android con librerie condivise native C/C++ caricate tramite JNI.
I controlli esportati coprono debugging, rooting, speed hack, modifica dei binari e indicatori legati alle macchine virtuali.
È pensato per esperimenti di protezione dei giochi mobile e per studiare tecniche di rilevamento anti-cheat su Android.
