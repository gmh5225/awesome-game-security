Questo progetto è un framework di analisi per scoprire se un file è sospetto e mostrare comodamente i risultati.
Si raccomanda vivamente di usare Ubuntu 24.04, che include una versione più recente di LibreOffice.
È particolarmente utile per gli ingegneri anti-cheat e i ricercatori di sicurezza difensiva che lavorano nell'ambito dei framework anti-cheat / di analisi.
