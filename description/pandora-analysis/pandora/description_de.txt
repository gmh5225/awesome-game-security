Dieses Projekt ist ein Analyse-Framework, um festzustellen, ob eine Datei verdächtig ist, und die Ergebnisse übersichtlich darzustellen.
Es wird dringend empfohlen, Ubuntu 24.04 zu verwenden, das eine neuere Version von LibreOffice mitbringt.
Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat-/Analyse-Frameworks arbeiten.
