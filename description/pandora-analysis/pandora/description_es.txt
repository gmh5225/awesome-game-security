Este proyecto es un framework de análisis para descubrir si un archivo es sospechoso y mostrar los resultados de forma conveniente.
Se recomienda encarecidamente usar Ubuntu 24.04, que incluye una versión más reciente de LibreOffice.
Resulta especialmente útil para ingenieros antitrampas e investigadores de seguridad defensiva que trabajan en el ámbito de frameworks antitrampas / de análisis.
