Этот проект — аналитический фреймворк для определения подозрительности файла и удобного отображения результатов.
Настоятельно рекомендуется использовать Ubuntu 24.04, в котором установлена более новая версия LibreOffice.
Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / аналитических фреймворков.
