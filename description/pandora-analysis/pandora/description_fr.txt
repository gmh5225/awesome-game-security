Ce projet est un framework d'analyse permettant de déterminer si un fichier est suspect et d'afficher commodément les résultats.
Il est fortement recommandé d'utiliser Ubuntu 24.04, qui inclut une version plus récente de LibreOffice.
Il s'adresse principalement aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant dans le domaine des frameworks anti-triche / d'analyse.
