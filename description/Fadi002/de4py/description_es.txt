Este proyecto es un framework de código abierto para desofuscación y análisis de Python, con interfaces GUI y CLI. Admite varios ofuscadores de Python e incluye analizadores para artefactos empaquetados, inspección de hashes y cadenas, supervisión del comportamiento y ejecución de código integrada en el proceso. El núcleo está escrito en Python con una interfaz PySide6, además de componentes nativos de ayuda para inyección y herramientas asistidas por hooks en Windows. Está pensado para analistas de malware e ingenieros inversos que necesitan flujos de trabajo prácticos de desempaquetado e inspección de Python.
