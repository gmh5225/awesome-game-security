이 프로젝트는 GUI와 CLI 인터페이스를 모두 제공하는 오픈소스 Python 난독화 해제 및 분석 프레임워크입니다. 여러 Python obfuscator를 지원하며, 패킹된 아티팩트, 해시 및 문자열 검사, 행위 모니터링, 프로세스 통합 코드 실행을 위한 분석기를 포함합니다. 코어는 Python으로 작성되고 인터페이스는 PySide6이며, Windows에서 인젝션과 훅 기반 도구를 위한 네이티브 헬퍼 컴포넌트도 제공합니다. 실용적인 Python 언패킹 및 검사 워크플로가 필요한 멀웨어 분석가와 리버스 엔지니어를 위해 설계되었습니다.
