Questo progetto è un framework open source di deoffuscazione e analisi Python con interfacce GUI e CLI. Supporta più offuscatori Python e include analyzer per artefatti packed, ispezione di hash e stringhe, monitoraggio del comportamento ed esecuzione di codice integrata nel processo. Il core è scritto in Python con interfaccia PySide6, più componenti nativi di supporto per injection e strumenti assistiti da hook su Windows. È pensato per analisti di malware e reverse engineer che necessitano di workflow pratici di unpacking e ispezione Python.
