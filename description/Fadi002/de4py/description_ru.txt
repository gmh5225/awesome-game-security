Этот проект — открытый фреймворк деобфускации и анализа Python с интерфейсами GUI и CLI. Он поддерживает несколько Python-обфускаторов и включает анализаторы упакованных артефактов, проверку хешей и строк, мониторинг поведения и выполнение кода, интегрированное с процессом. Ядро написано на Python с интерфейсом на PySide6, плюс нативные вспомогательные компоненты для инъекции и hook-инструментов в Windows. Предназначен для аналитиков вредоносного ПО и reverse engineer’ов, которым нужны практичные рабочие процессы распаковки и инспекции Python.
