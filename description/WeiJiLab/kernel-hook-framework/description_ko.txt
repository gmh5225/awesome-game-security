이 프로젝트는 런타임에 커널 함수를 가로채고, 교체하고, 복원하기 위한 Linux 커널 inline-hook 프레임워크입니다.
코어 모듈과 샘플 모듈을 제공하며, x86, x86_64, arm, arm64, riscv64를 포함한 여러 아키텍처를 지원하고, proc 인터페이스를 통해 런타임 제어를 노출합니다.
이 프레임워크는 더 넓은 kallsyms 대상과 동작하도록 trampoline 기반 패칭과 확장된 심볼 해석에 중점을 둡니다.
커널 디버깅, 라이브 실험, 그리고 안티치트 관련 커널 연구를 포함한 저수준 보안 연구를 위해 설계되었습니다.
