本プロジェクトは、実行時にカーネル関数をインターセプト、置換、復元するための Linux カーネル inline-hook フレームワークです。
コアモジュールとサンプルモジュールを提供し、x86、x86_64、arm、arm64、riscv64 を含む複数アーキテクチャをサポートし、proc インターフェース経由で実行時制御を公開します。
本フレームワークは、より幅広い kallsyms ターゲットに対応するため、trampoline ベースのパッチ適用と拡張シンボル解決に重点を置いています。
カーネルデバッグ、ライブ実験、およびアンチチート関連のカーネル研究を含む低レベルセキュリティ研究向けに設計されています。
