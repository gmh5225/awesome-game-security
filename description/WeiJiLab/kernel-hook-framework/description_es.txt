Este proyecto es un framework de inline-hook para el kernel de Linux destinado a interceptar, reemplazar y restaurar funciones del kernel en tiempo de ejecución.
Proporciona un módulo principal y módulos de ejemplo, admite múltiples arquitecturas como x86, x86_64, arm, arm64 y riscv64, y expone control en tiempo de ejecución mediante interfaces proc.
El framework se centra en el parcheo basado en trampolines y en la resolución ampliada de símbolos para trabajar con un conjunto más amplio de objetivos kallsyms.
Está diseñado para depuración del kernel, experimentación en vivo e investigación de seguridad de bajo nivel, incluidas las relacionadas con anti-cheat.
