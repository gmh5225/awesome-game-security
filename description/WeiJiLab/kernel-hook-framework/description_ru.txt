Этот проект — фреймворк inline-hook для ядра Linux, предназначенный для перехвата, замены и восстановления функций ядра во время выполнения.
Он предоставляет основной модуль и примеры модулей, поддерживает несколько архитектур, включая x86, x86_64, arm, arm64 и riscv64, и даёт управление во время выполнения через интерфейсы proc.
Фреймворк ориентирован на патчинг на основе trampoline и расширенное разрешение символов для работы с более широким набором целей kallsyms.
Он предназначен для отладки ядра, живых экспериментов и низкоуровневых исследований безопасности, включая связанные с античитом исследования ядра.
