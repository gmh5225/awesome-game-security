Questo progetto è un framework di inline-hook per il kernel Linux per intercettare, sostituire e ripristinare funzioni del kernel a runtime.
Fornisce un modulo principale e moduli di esempio, supporta più architetture tra cui x86, x86_64, arm, arm64 e riscv64, ed espone il controllo a runtime tramite interfacce proc.
Il framework si concentra sul patching basato su trampoline e sulla risoluzione estesa dei simboli per lavorare con un insieme più ampio di target kallsyms.
È progettato per il debugging del kernel, la sperimentazione live e la ricerca di sicurezza a basso livello, inclusi gli studi sul kernel legati all'anti-cheat.
