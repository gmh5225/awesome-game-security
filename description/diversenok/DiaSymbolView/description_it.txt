Strumento GUI basato su Delphi per ispezionare visivamente i simboli di debug e le loro oltre 200 proprietà archiviati nei file PDB tramite l'API Microsoft MSDIA (Debug Interface Access).
Presenta una gerarchia navigabile di tipi di simboli, funzioni, variabili e unità di compilazione, con enumerazione completa delle proprietà e risoluzione dei nomi dei registri.
È soprattutto utile per reverse engineer e ricercatori di sicurezza che esaminano la struttura delle informazioni di debug PDB e i metadati dei simboli nei binari Windows.
