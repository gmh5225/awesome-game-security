Ein auf Delphi basierendes GUI-Tool zur visuellen Untersuchung von Debug-Symbolen und ihren über 200 Eigenschaften in PDB-Dateien über die Microsoft MSDIA (Debug Interface Access) API.
Es zeigt eine navigierbare Hierarchie von Symboltypen, Funktionen, Variablen und Compilation Units mit vollständiger Eigenschaftsauflistung und Auflösung von Registernamen.
Es ist vor allem für Reverse Engineers und Sicherheitsforscher nützlich, die PDB-Debug-Informationsstrukturen und Symbol-Metadaten von Windows-Binaries untersuchen.
