GUI-инструмент на Delphi для визуального просмотра отладочных символов и их более чем 200 свойств, хранящихся в PDB-файлах, через API Microsoft MSDIA (Debug Interface Access).
Он представляет навигируемую иерархию типов символов, функций, переменных и единиц компиляции с полным перечислением свойств и разрешением имён регистров.
В основном полезен реверс-инженерам и исследователям безопасности, изучающим структуру отладочной информации PDB и метаданные символов Windows-бинарников.
