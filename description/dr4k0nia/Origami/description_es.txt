Un empaquetador de ensamblados .NET que comprime ejecutables administrados y almacena la carga útil comprimida dentro de estructuras del formato PE, abusando del directorio de depuración o de una sección PE personalizada (.origami) para el almacenamiento de datos.
Incluye un cargador de reubicación en tiempo de ejecución (RelocLoader) que descomprime y ejecuta el ensamblado original a partir de los datos PE incrustados.
Resulta especialmente útil para investigadores de seguridad que estudian técnicas de empaquetado .NET, el abuso del formato PE y los mecanismos de protección de código administrado.
