Упаковщик сборок .NET, который сжимает управляемые исполняемые файлы и сохраняет сжатую полезную нагрузку в структурах формата PE, используя для хранения данных либо каталог отладки, либо пользовательскую секцию PE (.origami).
Включает загрузчик с перемещением во время выполнения (RelocLoader), который распаковывает и запускает исходную сборку из встроенных данных PE.
В основном полезен исследователям безопасности, изучающим техники упаковки .NET, злоупотребление форматом PE и механизмы защиты управляемого кода.
