Empaquetador de archivos PE .NET escrito en C# que incrusta un ensamblado administrado de destino como recurso cifrado y comprimido dentro de un ejecutable stub cargador. En tiempo de ejecución, el stub descifra, descomprime e invoca el punto de entrada del ensamblado original por completo dentro del CLR, sin código nativo y con un retraso de desempaquetado casi nulo. Una GUI WinForms asociada (SimplePacker) ofrece una interfaz de arrastrar y soltar para empaquetar ensamblados.
