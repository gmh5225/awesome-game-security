Упаковщик PE-файлов .NET на C#, который внедряет целевую управляемую сборку как зашифрованный и сжатый ресурс в исполняемый файл загрузчика-заглушки. Во время выполнения заглушка расшифровывает, распаковывает и вызывает точку входа исходной сборки полностью внутри CLR, без нативного кода и почти без задержки распаковки. Сопутствующий WinForms GUI (SimplePacker) предоставляет интерфейс drag-and-drop для упаковки сборок.
