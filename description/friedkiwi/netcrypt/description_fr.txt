Packer de fichiers PE .NET écrit en C# qui intègre un assembly managé cible en tant que ressource chiffrée et compressée dans un exécutable stub chargeur. À l'exécution, le stub déchiffre, décompresse et invoque le point d'entrée de l'assembly d'origine entièrement dans le CLR, sans code natif et avec un délai de dépaquetage quasi nul. Une interface WinForms associée (SimplePacker) offre un glisser-déposer pour empaqueter les assemblies.
