In C# geschriebener .NET-PE-Datei-Packer, der eine Ziel-Managed-Assembly als verschlüsselte und komprimierte Ressource in eine Loader-Stub-Executable einbettet. Zur Laufzeit entschlüsselt, dekomprimiert und ruft der Stub den Einstiegspunkt der Original-Assembly vollständig innerhalb der CLR auf — ohne nativen Code und mit nahezu keiner Entpackverzögerung. Eine begleitende WinForms-GUI (SimplePacker) bietet eine Drag-and-Drop-Oberfläche zum Packen von Assemblies.
