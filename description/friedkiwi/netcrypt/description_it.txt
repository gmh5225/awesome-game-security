Packer di file PE .NET scritto in C# che incorpora un assembly gestito di destinazione come risorsa crittografata e compressa all'interno di un eseguibile stub di caricamento. A runtime lo stub decrittografa, decomprime e invoca il punto di ingresso dell'assembly originale interamente all'interno del CLR, senza codice nativo e con un ritardo di unpacking quasi nullo. Una GUI WinForms companion (SimplePacker) offre un'interfaccia drag-and-drop per il packing degli assembly.
