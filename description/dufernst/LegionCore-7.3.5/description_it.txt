Questo progetto è un emulatore di server World of Warcraft per l'espansione Legion (versione client 7.3.5), basato sul framework TrinityCore. Il server C++ implementa la simulazione del mondo, le meccaniche di dungeon e incursioni, le armi artefatto, le sale di classe, le missioni mondiali e i contenuti specifici di Legion. Gestisce l'amministrazione dei giocatori, lo scripting degli PNG, l'elaborazione degli incantesimi e il protocollo di rete di WoW. È rivolto agli sviluppatori di server privati WoW e ai ricercatori di server di gioco che studiano l'implementazione dei server delle espansioni MMO moderne.
