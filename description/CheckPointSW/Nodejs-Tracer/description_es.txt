Este proyecto es un script de rastreo de Node.js que instrumenta las llamadas a módulos centrales en tiempo de ejecución. Puede registrar el uso de API, falsificar comportamientos seleccionados para eludir comprobaciones anti-análisis y conservar artefactos como archivos escritos por el proceso objetivo. La implementación es ligera y se lanza mediante el mecanismo de precarga de Node, lo que facilita adjuntarlo a scripts ofuscados sin una configuración compleja. Está pensado para analistas de malware e investigadores de seguridad que realizan análisis dinámico del comportamiento de JavaScript.
