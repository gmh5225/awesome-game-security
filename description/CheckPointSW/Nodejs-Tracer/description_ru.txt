Этот проект — скрипт трассировки для Node.js, который инструментирует вызовы основных модулей во время выполнения. Он может журналировать использование API, подменять отдельные поведения для обхода антианалитических проверок и сохранять артефакты, например файлы, записанные целевым процессом. Реализация легковесна и запускается через механизм preload в Node, что упрощает подключение к обфусцированным скриптам без сложной настройки. Предназначен для аналитиков вредоносного ПО и исследователей безопасности, выполняющих динамический анализ поведения JavaScript.
