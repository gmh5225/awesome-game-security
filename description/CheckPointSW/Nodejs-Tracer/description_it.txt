Questo progetto è uno script di tracing per Node.js che strumenta le chiamate ai moduli core a runtime. Può registrare l'uso delle API, simulare comportamenti selezionati per aggirare i controlli anti-analisi e conservare artefatti come i file scritti dal processo target. L'implementazione è leggera e si avvia tramite il meccanismo di preload di Node, rendendo semplice l'aggancio a script offuscati senza una configurazione complessa. È pensato per analisti di malware e ricercatori di sicurezza che effettuano analisi dinamica del comportamento JavaScript.
