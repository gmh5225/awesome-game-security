Este proyecto es una herramienta de exploración anti-debug para Windows orientada a probar la resiliencia de los depuradores frente a técnicas anti-debug reales. Implementa una amplia colección de comprobaciones y trucos procedentes de muestras de malware e investigación publicada, y expone las direcciones de funciones relevantes para la experimentación práctica. Los investigadores pueden adjuntar depuradores y comparar el comportamiento entre distintas herramientas, plugins y estrategias de mitigación en versiones modernas de Windows. Está dirigida a ingenieros de ingeniería inversa, analistas de malware y desarrolladores de herramientas que validan capacidades anti-anti-debug.
