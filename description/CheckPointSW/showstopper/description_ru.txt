Этот проект — инструмент исследования антиотладки в Windows для проверки устойчивости отладчиков к реальным антиотладочным техникам. Он реализует большой набор проверок и приёмов из образцов вредоносного ПО и опубликованных исследований и предоставляет адреса соответствующих функций для практических экспериментов. Исследователи могут подключать отладчики и сравнивать поведение разных инструментов, плагинов и стратегий противодействия на современных версиях Windows. Он предназначен для реверс-инженеров, аналитиков вредоносного ПО и разработчиков инструментов, проверяющих возможности anti-anti-debug.
