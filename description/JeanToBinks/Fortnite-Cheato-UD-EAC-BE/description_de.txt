Dieses Projekt ist ein Windows-External-Cheat-Prototyp für Fortnite, der den Spielspeicher liest und ein On-Screen-Overlay rendert. Es ist in C++ geschrieben und enthält Routinen für Prozessspeicherzugriff, Unreal-orientierte Transformationsmathematik sowie DirectX-ähnliche Zeichenhilfen. Der Code konzentriert sich auf das Extrahieren von Spielerdatenstrukturen und die Echtzeitvisualisierung von Entitäten. Der primäre Anwendungsfall ist die Experimentierung in der Cheat-Entwicklung und die Erforschung von Anti-Cheat-Umgehung.
