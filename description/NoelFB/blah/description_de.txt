Dieses Projekt ist ein leichtgewichtiges 2D-Game-Framework für C++.
Es zielt auf einfache Portabilität mit wenigen Abhängigkeiten und einer klaren API für App-Lebenszyklus, Rendering, Eingabe, Audio und Mathematik-Hilfsfunktionen.
Der Build nutzt CMake mit C++17 und unterstützt OpenGL- oder Direct3D11-Renderer-Backends.
Es richtet sich an Entwickler, die eine kleine, leicht anpassbare Engine-Basis für 2D-Spielprototypen wollen.
