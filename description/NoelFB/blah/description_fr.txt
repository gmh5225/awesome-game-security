Ce projet est un framework de jeu 2D léger pour C++.
Il vise une portabilité simple, peu de dépendances et une API claire pour le cycle de vie de l’application, le rendu, les entrées, l’audio et les utilitaires mathématiques.
La configuration de build utilise CMake avec C++17 et prend en charge les backends de rendu OpenGL ou Direct3D11.
Il s’adresse aux développeurs qui veulent une base de moteur petite et facilement modifiable pour des prototypes de jeux 2D.
