Questo progetto è un framework leggero per giochi 2D in C++.
Punta a una portabilità semplice con poche dipendenze e a un’API pulita per il ciclo di vita dell’app, il rendering, l’input, l’audio e le utilità matematiche.
La configurazione di build usa CMake con C++17 e supporta backend di rendering OpenGL o Direct3D11.
È pensato per gli sviluppatori che vogliono una base di motore piccola e facilmente modificabile per prototipi di giochi 2D.
