Este proyecto es un framework ligero de juegos 2D para C++.
Busca una portabilidad sencilla con pocas dependencias y una API limpia para el ciclo de vida de la aplicación, el renderizado, la entrada, el audio y utilidades matemáticas.
La configuración de compilación usa CMake con C++17 y admite backends de renderizado OpenGL o Direct3D11.
Está pensado para desarrolladores que quieren una base de motor pequeña y fácil de modificar para prototipos de juegos 2D.
