Этот проект — лёгкий 2D-игровой фреймворк для C++.
Он ориентирован на простую переносимость с минимальными зависимостями и чистый API для жизненного цикла приложения, рендеринга, ввода, звука и математических утилит.
Сборка выполняется через CMake с C++17 и поддерживает бэкенды рендерера OpenGL или Direct3D11.
Предназначен для разработчиков, которым нужна небольшая и удобная для доработки база движка для прототипов 2D-игр.
