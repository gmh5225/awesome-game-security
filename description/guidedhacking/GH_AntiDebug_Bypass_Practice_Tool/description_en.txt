This project is a Win32 practice application that implements common anti-debugging techniques so users can attach a debugger and practice bypassing detection. It is written in C++ as a Visual Studio solution and presents an ImGui overlay on DirectX 11 so each method can be toggled on and watched for DETECTED status. Covered checks include IsDebuggerPresent, PEB BeingDebugged and NtGlobalFlag, CheckRemoteDebuggerPresent, heap and LFH flags, ThreadHideFromDebugger, trap-flag and SEH tricks, parent-process inspection, and timing checks via QueryPerformanceCounter, GetTickCount, and GetLocalTime. New methods can be added as header callbacks registered in the main loop. It is aimed at reverse engineers and game security learners studying Windows anti-debug defenses and bypass practice.
