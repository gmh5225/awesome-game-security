本项目是一款 Win32 练习应用，实现常见反调试技术，便于用户附加调试器并练习绕过检测。采用 C++ 编写，以 Visual Studio 解决方案形式提供，并通过 DirectX 11 上的 ImGui 叠加界面切换各方法并观察 DETECTED 状态。涵盖检查包括 IsDebuggerPresent、PEB BeingDebugged 与 NtGlobalFlag、CheckRemoteDebuggerPresent、堆与 LFH 标志、ThreadHideFromDebugger、陷阱标志与 SEH 技巧、父进程检查，以及通过 QueryPerformanceCounter、GetTickCount 和 GetLocalTime 的计时检测。新方法可作为头文件回调在主循环中注册添加。面向研究 Windows 反调试防护与绕过练习的逆向工程师与游戏安全学习者。
