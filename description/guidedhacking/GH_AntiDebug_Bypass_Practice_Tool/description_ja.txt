本プロジェクトは、一般的なアンチデバッグ手法を実装した Win32 練習用アプリケーションで、デバッガをアタッチして検知回避の練習ができます。C++ で書かれた Visual Studio ソリューションとして提供され、DirectX 11 上の ImGui オーバーレイで各手法を切り替え、DETECTED 状態を確認できます。対象チェックには IsDebuggerPresent、PEB BeingDebugged と NtGlobalFlag、CheckRemoteDebuggerPresent、ヒープおよび LFH フラグ、ThreadHideFromDebugger、トラップフラグと SEH の技巧、親プロセス検査、ならびに QueryPerformanceCounter、GetTickCount、GetLocalTime によるタイミング検査が含まれます。新しい手法はヘッダのコールバックとしてメインループに登録して追加できます。Windows のアンチデバッグ防御とバイパス練習を学ぶリバースエンジニアやゲームセキュリティ学習者向けです。
