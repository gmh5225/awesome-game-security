本專案是一款 Win32 練習應用程式，實作常見反除錯技術，方便使用者附加除錯器並練習繞過偵測。以 C++ 撰寫，採 Visual Studio 方案形式提供，並透過 DirectX 11 上的 ImGui 疊加介面切換各方法並觀察 DETECTED 狀態。涵蓋檢查包括 IsDebuggerPresent、PEB BeingDebugged 與 NtGlobalFlag、CheckRemoteDebuggerPresent、堆積與 LFH 旗標、ThreadHideFromDebugger、陷阱旗標與 SEH 技巧、父程序檢查，以及透過 QueryPerformanceCounter、GetTickCount 和 GetLocalTime 的計時偵測。新方法可作為標頭回呼在主迴圈中註冊加入。面向研究 Windows 反除錯防護與繞過練習的逆向工程師與遊戲安全學習者。
