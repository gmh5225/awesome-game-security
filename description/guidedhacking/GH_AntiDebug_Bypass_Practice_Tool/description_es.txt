Este proyecto es una aplicación de práctica Win32 que implementa técnicas anti-depuración comunes para que los usuarios puedan adjuntar un depurador y practicar la evasión de la detección. Está escrito en C++ como solución de Visual Studio y presenta una overlay ImGui sobre DirectX 11 para activar cada método y observar el estado DETECTED. Las comprobaciones cubiertas incluyen IsDebuggerPresent, PEB BeingDebugged y NtGlobalFlag, CheckRemoteDebuggerPresent, flags de heap y LFH, ThreadHideFromDebugger, trucos de trap-flag y SEH, inspección del proceso padre y comprobaciones de tiempo mediante QueryPerformanceCounter, GetTickCount y GetLocalTime. Se pueden añadir nuevos métodos como callbacks de cabecera registrados en el bucle principal. Está dirigido a ingenieros inversos y estudiantes de seguridad de juegos que investigan las defensas anti-debug de Windows y la práctica de bypass.
