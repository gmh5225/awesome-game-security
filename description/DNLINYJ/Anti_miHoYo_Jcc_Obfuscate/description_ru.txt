Этот проект — плагин для x64dbg, помогающий снимать обфускацию потока управления на основе JCC и переходов, применяемую в защищённых путях кода игр на Unity. Он отслеживает заданные диапазоны процедур расшифровки, фиксирует динамическое поведение переходов и патчит инструкции, чтобы восстановить более читаемый поток выполнения во время отладочных сессий. Реализация выполнена на C++ с использованием SDK плагинов x64dbg и включает логику, привязанную к известным смещениям сборок игр. Предназначен для реверс-инжиниринга игр и исследований по антиобфускации; автор отмечает, что код больше не поддерживается.
