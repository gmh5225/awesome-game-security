Ce projet est un plugin x64dbg qui aide à défaire l'obfuscation de flux de contrôle basée sur les JCC et les sauts, utilisée dans les chemins de code protégés de jeux Unity. Il surveille des plages spécifiques de routines de déchiffrement, suit le comportement dynamique des sauts et corrige les instructions pour reconstruire un flux d'exécution plus lisible pendant les sessions de débogage. L'implémentation est en C++ avec le SDK de plugins x64dbg et inclut une logique adaptée à des offsets connus de builds de jeux. Il est destiné à la rétro-ingénierie de jeux et à la recherche anti-obfuscation ; le code est indiqué par l'auteur comme n'étant plus maintenu.
