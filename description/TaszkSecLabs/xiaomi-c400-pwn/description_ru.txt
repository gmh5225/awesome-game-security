Этот проект — пакет исследований по безопасности, содержащий эксплойт-код и инструменты джейлбрейка для роутера Xiaomi C400. В него входят Python-эксплойт с предсказанием RNG, модели формальной верификации на Tamarin, документация по последовательности рукопожатия, а также отдельный компонент джейлбрейка для получения постоянного доступа. В первую очередь полезен исследователям встроенной безопасности и разработчикам IoT-эксплойтов, изучающим анализ уязвимостей маршрутизаторов и эксплуатацию прошивок.
