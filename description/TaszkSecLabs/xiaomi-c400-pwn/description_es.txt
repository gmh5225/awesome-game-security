Este proyecto es un paquete de investigación de seguridad con código de exploit y herramientas de jailbreak para el router Xiaomi C400. Incluye un exploit en Python con predicción de RNG, modelos de verificación formal en Tamarin, documentación de secuencias de handshake y un componente de jailbreak independiente para obtener acceso persistente. Es principalmente útil para investigadores de seguridad embebida y desarrolladores de exploits IoT que estudian el análisis de vulnerabilidades en routers y la explotación de firmware.
