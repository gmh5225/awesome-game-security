Dieses Projekt ist eine Sammlung von Cheat-Engine-Tabellen (.CT-Dateien) für verschiedene Spiele. Jede Tabelle enthält vorkonfigurierte Speicheradressen, Zeigerketten, skriptbasierte Cheats und Hotkey-Belegungen zum Ändern von Spielwerten wie Gesundheit, Währung, Erfahrung und Gegenständen. Die Tabellen sind für das direkte Laden in Cheat Engine formatiert. Es richtet sich an Spielmodder und Gelegenheitshacker, die Cheat Engine für die Modifikation von Einzelspieler-Spielen nutzen.
