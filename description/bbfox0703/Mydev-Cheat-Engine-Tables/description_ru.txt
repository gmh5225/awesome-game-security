Этот проект — коллекция таблиц Cheat Engine (файлы .CT) для различных игр. Каждая таблица содержит предварительно настроенные адреса памяти, цепочки указателей, читы на основе скриптов и привязки горячих клавиш для изменения игровых значений, таких как здоровье, валюта, опыт и предметы. Таблицы предназначены для прямой загрузки в Cheat Engine. Проект ориентирован на моддеров и любителей взлома игр, которые используют Cheat Engine для модификации одиночных игр.
