Este proyecto es una colección de tablas de Cheat Engine (archivos .CT) para diversos juegos. Cada tabla contiene direcciones de memoria preconfiguradas, cadenas de punteros, trucos basados en scripts y atajos de teclado para modificar valores del juego como salud, moneda, experiencia y objetos. Las tablas están formateadas para cargarse directamente en Cheat Engine. Está dirigido a modders y hackers ocasionales que usan Cheat Engine para modificar juegos de un solo jugador.
