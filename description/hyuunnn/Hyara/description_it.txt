Questo progetto è un plugin multipiattaforma per la generazione di regole YARA con supporto per IDA Pro, Ghidra, Binary Ninja e Cutter. Lo strumento basato su Python fornisce API integrate per ciascun disassemblatore per creare, verificare e rilevare firme YARA direttamente dagli ambienti di analisi binaria. È soprattutto utile ad analisti di malware e reverse engineer che devono generare regole YARA da pattern binari su diverse piattaforme di disassemblaggio.
