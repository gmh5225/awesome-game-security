Этот проект — многоплатформенный плагин для генерации правил YARA с поддержкой IDA Pro, Ghidra, Binary Ninja и Cutter. Инструмент на Python предоставляет интегрированные API для каждого дизассемблера, позволяя создавать, проверять и обнаруживать сигнатуры YARA непосредственно в средах анализа бинарных файлов. Он в первую очередь полезен аналитикам вредоносного ПО и инженерам по реверс-инжинирингу, которым нужно генерировать правила YARA по бинарным шаблонам на разных платформах дизассемблирования.
