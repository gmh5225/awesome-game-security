Este proyecto es un complemento multipplataforma de generación de reglas YARA compatible con IDA Pro, Ghidra, Binary Ninja y Cutter. La herramienta basada en Python ofrece APIs integradas para cada desensamblador con el fin de crear, comprobar y detectar firmas YARA directamente desde entornos de análisis binario. Resulta especialmente útil para analistas de malware e ingenieros de ingeniería inversa que necesitan generar reglas YARA a partir de patrones binarios en distintas plataformas de desensamblado.
