ida-slides は、ドッキング可能な IDA タブ内で Marp または Slidev のスライドを描画する IDA Pro プラグイン（IDA 9.2+、Python）で、ライブのリバースエンジニアリング発表に使えます。スライドと IDB は双方向に連動し、@name や @0xADDR トークンでディスアセンブリや Hex-Rays 疑似コードへジャンプでき、範囲構文では保存のたびにライブのデコンパイル行をデッキに埋め込めます。デコンパイル抜粋のホバープレビュー、IDA ビューからの右クリックで @reference トークンをコピーする操作、未解決参照向けのデッキ lint、ファイルウォッチャーによるライブリロードに対応します。描画はネイティブ OS の webview（macOS は WKWebView、Windows は WebView2）と Marp または Slidev CLI を使い、外部ブラウザではなくディスアセンブリの横でデッキを表示します。主な対象は、IDA 内で分析の発表やウォークスルーを行いたいリバースエンジニアです。
