Этот проект — автоматический обновлятор смещений (offsets) для инструментальных цепочек читов и реверс-инжиниринга. Он в основном реализован на C++ для Visual Studio и использует данные смещений в формате JSON вместе с компонентами сетевой загрузки. Кодовая база адаптирует существующий шаблон обновлятора, чтобы смещения можно было обновлять без пересборки всего загрузчика после каждого патча игры. Основной сценарий использования — поддержка исследовательских или автоматизационных проектов, связанных с VALORANT и зависящих от актуальных смещений в памяти.
