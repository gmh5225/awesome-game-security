Este proyecto es un actualizador automático de offsets diseñado para cadenas de herramientas de trampas y de ingeniería inversa. Está implementado principalmente en C++ para Visual Studio y usa datos de offsets en JSON junto con componentes de obtención por red. El código adapta un patrón de actualizador existente para refrescar los offsets sin reconstruir todo el cargador tras cada parche del juego. Su caso de uso principal es mantener proyectos de investigación o automatización relacionados con VALORANT que dependen de offsets de memoria actuales.
