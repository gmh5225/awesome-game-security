Questo progetto è un aggiornatore automatico di offset pensato per toolchain di cheat e reverse engineering. È implementato principalmente in C++ per Visual Studio e utilizza dati di offset in JSON con componenti di recupero via rete. Il codice adatta un pattern di updater esistente così da aggiornare gli offset senza ricostruire l'intero loader dopo ogni patch del gioco. Il caso d'uso principale è mantenere progetti di ricerca o di automazione legati a VALORANT che dipendono da offset di memoria aggiornati.
