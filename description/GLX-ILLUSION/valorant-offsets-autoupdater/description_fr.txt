Ce projet est un outil de mise à jour automatique des offsets conçu pour les chaînes d'outils de triche et de rétro-ingénierie. Il est principalement implémenté en C++ pour Visual Studio et utilise des données d'offsets au format JSON avec des composants de récupération réseau. La base de code adapte un modèle d'updater existant afin de rafraîchir les offsets sans reconstruire tout le chargeur après chaque correctif du jeu. Son cas d'usage principal est le maintien de projets de recherche ou d'automatisation liés à VALORANT qui dépendent d'offsets mémoire à jour.
