Этот проект — приложение Windows .NET для дампа нативных и управляемых сборок из работающих процессов. Помимо базового дампа, оно включает средства проверки модулей, обнаружения anti-dump и хуков, просмотра виртуальной памяти, исследования кучи и процессов, а также перечисления AppDomain. Инструмент реализован на C# WinForms и содержит компоненты управляемой инъекции и генерации minidump для рабочих процессов реверс-инжиниринга. В основном используется аналитиками вредоносного ПО и реверсерами ПО, которым нужно извлекать бинарники из памяти и исследовать внутренности процессов.
