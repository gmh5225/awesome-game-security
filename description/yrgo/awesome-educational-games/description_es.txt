Este proyecto ofrece una lista curada de excelentes juegos educativos para aprender editores, lenguajes, programación, etc.
Los juegos educativos están diseñados para enseñar sobre ciertos temas, ampliar conceptos, reforzar el desarrollo, comprender un acontecimiento histórico o una cultura, o ayudar a aprender una habilidad mientras se juega - Wikipedia.
Es útil principalmente para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo / guías de juegos.
