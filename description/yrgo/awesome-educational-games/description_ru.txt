Этот проект предоставляет подборку отличных образовательных игр для изучения редакторов, языков, программирования и т. д.
Образовательные игры предназначены для того, чтобы в процессе игры обучать определённым предметам, расширять понятия, закреплять развитие, помогать понять историческое событие или культуру либо освоить навык - Wikipedia.
Он в основном полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / гайдов.
