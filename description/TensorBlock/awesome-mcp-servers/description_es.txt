Este proyecto es un amplio índice curado de servidores Model Context Protocol e integraciones relacionadas.
Está organizado como documentación Markdown con páginas de categorías que recopilan herramientas de IA, datos, infraestructura, seguridad, juegos y muchos otros dominios.
En lugar de implementar un solo servidor, se centra en el descubrimiento, el catálogo y la navegación rápida de miles de entradas MCP.
El público principal son desarrolladores e investigadores que construyen flujos de trabajo de IA y necesitan backends de herramientas MCP listos para usar.
