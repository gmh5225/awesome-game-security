Ce projet est un vaste index organisé de serveurs Model Context Protocol et d'intégrations associées.
Il se présente sous forme de documentation Markdown avec des pages de catégories qui recensent des outils dans l'IA, les données, l'infrastructure, la sécurité, le jeu et bien d'autres domaines.
Au lieu d'implémenter un serveur unique, il se concentre sur la découverte, le catalogage et la navigation rapide parmi des milliers d'entrées MCP.
Le public principal est constitué de développeurs et de chercheurs qui construisent des flux de travail IA nécessitant des backends d'outils MCP prêts à l'emploi.
