Dieses Projekt konzentriert sich auf eine einfache Header-only-Bibliothek, mit der Sie skalare Werte in Ihrer Anwendung verschlüsseln können. Sie ist hauptsächlich in C++ geschrieben. Sie ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat / Variablenverschlüsselung arbeiten.
