Этот проект посвящён простой библиотеке, состоящей только из заголовочного файла, которая позволяет шифровать скалярные значения в вашем приложении. Она написана преимущественно на C++. Библиотека особенно полезна инженерам по защите от читов и исследователям оборонительной безопасности, работающим в области античита / шифрования переменных.
