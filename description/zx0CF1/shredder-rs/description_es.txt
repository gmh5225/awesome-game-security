Este proyecto es un motor de mutación polimórfica x86_64 de alta fidelidad, centrado en la fragmentación a nivel de instrucción y la preservación del contexto.
Resumen: Shredder-RS implementa shredding a nivel de instrucción para derrotar el análisis estático.
Resulta especialmente útil para ingenieros anti-trampas e investigadores de seguridad defensiva que trabajan en el área de motores anti-trampas / de ofuscación.
