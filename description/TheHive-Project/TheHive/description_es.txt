Este proyecto es el código fuente archivado de una plataforma de respuesta a incidentes de seguridad y gestión de casos usada en flujos de trabajo SOC.
Contiene un backend sustancial en Scala y componentes de frontend web, junto con bibliotecas cliente y módulos orientados a la integración.
El repositorio sirve ahora como una instantánea histórica, y los mantenedores indican que las ediciones más recientes y mantenidas se distribuyen por separado.
Su caso de uso principal es estudiar la arquitectura heredada de gestión de incidentes y los flujos de trabajo de seguridad operativa.
