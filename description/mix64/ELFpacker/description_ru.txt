Упаковщик ELF32-бинарников, который XOR-шифрует секцию .text исполняемого файла и добавляет в начало stub расшифровки, восстанавливающий исходный код во время выполнения перед переходом к исходной точке входа. Изменяет ELF-заголовки, program headers и section headers для внедрения stub-сегмента, сохраняя загружаемую структуру бинарника.
