Packeur de binaires ELF32 qui chiffre en XOR la section .text d'un exécutable et y préfixe un stub de déchiffrement restaurant le code d'origine à l'exécution avant de sauter vers le point d'entrée original. Manipule les en-têtes ELF, les en-têtes de programme et de section pour injecter le segment stub tout en préservant la structure chargeable du binaire.
