Questo progetto è uno strumento Windows per esplorare e decompilare i file di package di Unreal Engine. Offre decompilazione UnrealScript, esportazione di classi e suoni, ispezione delle dipendenze dei package e viste orientate al modding, come un visualizzatore esadecimale strutturato. L'applicazione è realizzata in C# con .NET WinForms e si rivolge ai workflow legacy dei package Unreal. È ampiamente usata da modder e reverse engineer che lavorano con giochi dell'era Unreal Engine da 1 a 3.
