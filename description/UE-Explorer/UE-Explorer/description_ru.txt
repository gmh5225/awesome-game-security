Этот проект — инструмент для Windows, предназначенный для просмотра и декомпиляции файлов пакетов Unreal Engine. Он обеспечивает декомпиляцию UnrealScript, экспорт классов и звуков, проверку зависимостей пакетов, а также ориентированные на моддинг представления, такие как структурированный hex-просмотрщик. Приложение написано на C# с использованием .NET WinForms и ориентировано на устаревшие рабочие процессы с пакетами Unreal. Его широко используют моддеры и специалисты по реверс-инжинирингу, работающие с играми эпохи Unreal Engine 1–3.
