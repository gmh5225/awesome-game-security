Этот проект — плагин для IDA Pro, реализующий сервер Model Context Protocol (MCP) и предоставляющий возможности анализа IDA ИИ-ассистентам и внешним инструментам. Он позволяет MCP-клиентам удалённо запрашивать дизассемблирование, декомпиляцию, перекрёстные ссылки, сведения о функциях и данные о типах из баз данных IDA. Python-плагин связывает IDA Pro с рабочими процессами реверс-инжиниринга на базе ИИ. Он предназначен для реверс-инженеров, интегрирующих IDA Pro с инструментами анализа на основе LLM через протокол MCP.
