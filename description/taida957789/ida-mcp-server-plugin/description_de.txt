Dieses Projekt ist ein IDA-Pro-Plugin, das einen Model Context Protocol (MCP)-Server implementiert und IDAs Analysefähigkeiten für KI-Assistenten und externe Tools bereitstellt. MCP-Clients können Disassembly, Dekompilation, Querverweise, Funktionsinformationen und Typdaten aus IDA-Datenbanken remote abfragen. Das Python-Plugin verbindet IDA Pro mit KI-gestützten Reverse-Engineering-Workflows. Es richtet sich an Reverse Engineers, die IDA Pro über das MCP-Protokoll mit LLM-basierten Analysetools integrieren.
