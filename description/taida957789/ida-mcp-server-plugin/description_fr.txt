Ce projet est un plugin IDA Pro qui implémente un serveur Model Context Protocol (MCP), exposant les capacités d'analyse d'IDA aux assistants IA et aux outils externes. Il permet aux clients MCP d'interroger à distance le désassemblage, la décompilation, les références croisées, les informations de fonctions et les données de types des bases IDA. Le plugin Python fait le lien entre IDA Pro et les workflows de reverse engineering assistés par l'IA. Il s'adresse aux reverse engineers qui intègrent IDA Pro à des outils d'analyse basés sur des LLM via le protocole MCP.
