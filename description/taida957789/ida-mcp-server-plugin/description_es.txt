Este proyecto es un complemento de IDA Pro que implementa un servidor Model Context Protocol (MCP), exponiendo las capacidades de análisis de IDA a asistentes de IA y herramientas externas. Permite a los clientes MCP consultar de forma remota el desensamblado, la descompilación, las referencias cruzadas, la información de funciones y los datos de tipos de las bases de datos de IDA. El complemento en Python conecta IDA Pro con flujos de trabajo de ingeniería inversa impulsados por IA. Está dirigido a ingenieros inversos que integran IDA Pro con herramientas de análisis basadas en LLM mediante el protocolo MCP.
