Questo progetto è un iniettore di DLL a livello kernel per Windows che nasconde l'esecuzione manipolando i permessi delle pagine tramite lo scambio del bit NX e tecniche legate al VAD.
È scritto principalmente in C con header nativi delle strutture interne di Windows e include codice per la gestione dinamica dei dati del kernel, I/O e logica del loader.
L'iniettore alloca pagine scrivibili, modifica sotto il cofano il comportamento delle page table per ottenere capacità di esecuzione e tenta di ridurre gli artefatti evidenti di protezione della memoria.
È rivolto alla ricerca avanzata su sicurezza dei giochi e bypass degli anti-cheat, dove si studiano metodi furtivi di iniezione assistita dal kernel.
