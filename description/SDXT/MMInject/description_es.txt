Este proyecto es un inyector de DLL en modo kernel para Windows que oculta la ejecución manipulando los permisos de página mediante el intercambio del bit NX y técnicas relacionadas con el VAD.
Está escrito principalmente en C con cabeceras nativas de internas de Windows e incluye código para el manejo dinámico de datos del kernel, E/S y lógica del cargador.
El inyector asigna páginas escribibles, modifica bajo el capó el comportamiento de las tablas de páginas para obtener capacidad de ejecución e intenta reducir artefactos evidentes de protección de memoria.
Está orientado a la investigación avanzada en seguridad de juegos y evasión de anti-cheat, donde se estudian métodos furtivos de inyección asistida por el kernel.
