Этот проект — инжектор DLL уровня ядра для Windows, который скрывает выполнение, манипулируя правами страниц с помощью обмена NX-бита и техник, связанных с VAD.
Он в основном написан на C с нативными заголовками внутренних структур Windows и включает код для динамической работы с данными ядра, I/O и логики загрузчика.
Инжектор выделяет записываемые страницы, скрытно изменяет поведение таблиц страниц для получения права на исполнение и пытается снизить заметные артефакты защиты памяти.
Он ориентирован на продвинутые исследования в области игровой безопасности и обхода античита, где изучаются скрытные методы инъекции с поддержкой ядра.
