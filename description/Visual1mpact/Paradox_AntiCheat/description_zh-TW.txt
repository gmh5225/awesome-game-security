Paradox AntiCheat 是一款面向 Minecraft Bedrock Edition 的高效能反作弊與管理外掛，用於在 Realms 與 Bedrock Dedicated Server（BDS）世界中偵測並阻止作弊。它主要以 TypeScript 撰寫，並以行為包形式執行於 Minecraft 的 Script API（包括 Beta server、server-net、server-admin 與 server-ui API）。偵測採模組化設計，涵蓋飛行、殺戮光環、超距攻擊、連點器、搭路、改名欺騙、穿牆、透視以及相關移動或戰鬥濫用等常見作弊手段。專案亦提供面向管理團隊的管理與實用指令（封鎖、踢出、禁言、凍結、白名單、封鎖伺服器及 GUI 工具），方便營運者依世界設定執行策略。它面向需要基於腳本的反作弊與公平遊玩工具、且無需安裝外部用戶端的 Bedrock 伺服器與 Realm 管理員。
