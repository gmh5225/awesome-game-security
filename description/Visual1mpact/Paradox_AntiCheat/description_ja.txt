Paradox AntiCheat は、Minecraft Bedrock Edition 向けの高性能アンチチート兼モデレーションアドオンで、Realms および Bedrock Dedicated Server（BDS）ワールドでのチートを検知・阻止するために作られています。主に TypeScript で書かれ、Minecraft の Script API（Beta server、server-net、server-admin、server-ui API を含む）上でビヘイビアパックとして動作します。検知はモジュール式で、飛行、キラオーラ、リーチ、オートクリッカー、スキャフォールド、ネームスプーフ、ノークリップ、X線、および関連する移動・戦闘の不正など、よくあるエクスプロイトをカバーします。プロジェクトにはスタッフ向けのモデレーション／ユーティリティコマンド（ban、kick、mute、freeze、whitelist、lockdown、GUI ツール）も含まれ、ワールドごとに執行設定ができます。外部クライアントのインストールなしでスクリプトベースのアンチチートとフェアプレイツールを必要とする Bedrock サーバー／Realm 管理者向けです。
