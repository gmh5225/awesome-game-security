Paradox AntiCheat 是一款面向 Minecraft Bedrock Edition 的高性能反作弊与管理插件，用于在 Realms 和 Bedrock Dedicated Server（BDS）世界中检测并阻止作弊。它主要使用 TypeScript 编写，并以行为包形式运行于 Minecraft 的 Script API（包括 Beta server、server-net、server-admin 和 server-ui API）。检测模块化，覆盖飞行、杀戮光环、超距攻击、连点器、搭路、改名欺骗、穿墙、透视以及相关移动或战斗滥用等常见作弊手段。项目还提供面向管理团队的管理与实用命令（封禁、踢出、禁言、冻结、白名单、封锁及 GUI 工具），便于运营者按世界配置执行策略。它面向需要基于脚本的反作弊与公平游戏工具、且无需安装外部客户端的 Bedrock 服务器与 Realm 管理员。
