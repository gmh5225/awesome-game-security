Paradox AntiCheat es un complemento de alto rendimiento de anticheat y moderación para Minecraft Bedrock Edition, creado para detectar y detener trampas en Realms y mundos de Bedrock Dedicated Server (BDS). Está escrito principalmente en TypeScript y se ejecuta como un behavior pack sobre la Script API de Minecraft (incluidas las API Beta server, server-net, server-admin y server-ui). La detección es modular y cubre exploits habituales como fly, killaura, reach, autoclicker, scaffold, namespoof, noclip, xray y abusos relacionados de movimiento o combate. El proyecto también incluye comandos de moderación y utilidad para el staff (ban, kick, mute, freeze, whitelist, lockdown y herramientas GUI) para que los operadores configuren la aplicación por mundo. Está pensado para administradores de servidores Bedrock y Realms que necesitan anticheat y herramientas de juego limpio basadas en scripts, sin instalaciones de cliente externas.
