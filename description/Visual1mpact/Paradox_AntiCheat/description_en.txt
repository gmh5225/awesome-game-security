Paradox AntiCheat is a high-performance anti-cheat and moderation addon for Minecraft Bedrock Edition, built to detect and stop cheating on Realms and Bedrock Dedicated Server (BDS) worlds. It is written mainly in TypeScript and runs as a behavior pack on Minecraft’s Script API (including Beta server, server-net, server-admin, and server-ui APIs). Detection is modular and covers common exploits such as fly, killaura, reach, autoclicker, scaffold, namespoof, noclip, xray, and related movement or combat abuse. The project also ships staff-facing moderation and utility commands (ban, kick, mute, freeze, whitelist, lockdown, and GUI tools) so operators can configure enforcement per world. It is aimed at Bedrock server and Realm administrators who need script-based anti-cheat and fair-play tooling without external client installs.
