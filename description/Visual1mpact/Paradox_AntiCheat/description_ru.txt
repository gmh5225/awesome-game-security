Paradox AntiCheat — высокопроизводительный античит и аддон модерации для Minecraft Bedrock Edition, созданный для обнаружения и пресечения читерства на Realms и мирах Bedrock Dedicated Server (BDS). Он написан в основном на TypeScript и работает как behavior pack поверх Script API Minecraft (включая API Beta server, server-net, server-admin и server-ui). Обнаружение модульное и охватывает распространённые эксплойты: fly, killaura, reach, autoclicker, scaffold, namespoof, noclip, xray, а также связанные злоупотребления движением и боем. Проект также включает команды модерации и утилиты для персонала (ban, kick, mute, freeze, whitelist, lockdown и GUI-инструменты), чтобы операторы могли настраивать применение правил для каждого мира. Он рассчитан на администраторов Bedrock-серверов и Realm, которым нужны скриптовый античит и инструменты честной игры без установки внешних клиентов.
