Garble is a Go obfuscation tool that wraps the Go toolchain so you can build, test, and run programs with as little recoverable source information left in the binary as possible. Written in Go, it replaces identifiers, package paths, and position data with short hashes, strips build, module, and debug metadata, and optionally obfuscates string and other literals. It also offers a tiny mode that further shrinks binaries by removing panic and debug runtime details, plus experimental control-flow obfuscation, while remaining deterministic and coupled with cmd/go modules and build caching. A reverse command can map obfuscated stack traces back to original names when the source is available. It is aimed at developers who need to discourage reverse engineering of distributed Go binaries or private libraries, including use cases relevant to software protection and anti-analysis.
