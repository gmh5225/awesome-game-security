本项目是一款用 C++ 编写的外部 Counter-Strike 2 工具，通过基于 FPGA 的 DMA 硬件在另一台机器上读取游戏内存，并在不注入游戏进程的情况下渲染叠加层。开源版本为只读 DMA，侧重视觉功能，例如方框与骨骼 ESP、武器与弹药显示、可见性着色、炸弹与投掷物辅助，以及支持手雷辅助的局域网 Web 雷达。项目依赖 MemProcFS 与 LeechCore 进行 DMA 访问，使用 ImGui 与 DirectX 11 构建界面，并包含散射批量读取、快照插值与分级 DMA 恢复等低延迟优化。通过 cs2-dumper 自动更新偏移、双源热键以及排障工具进一步完善工具链。面向研究基于 DMA 的外部 CS2 分析与可视化技术的游戏安全研究者与逆向工程师。
