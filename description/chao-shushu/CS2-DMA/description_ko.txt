이 프로젝트는 C++로 작성된 외부 Counter-Strike 2 도구로, FPGA 기반 DMA 하드웨어를 사용해 별도 기기에서 게임 메모리를 읽고 게임 프로세스에 주입하지 않고 오버레이를 렌더링합니다. 오픈 소스 빌드는 읽기 전용 DMA이며, 박스·본 ESP, 무기·탄약 표시, 가시성 색상, 폭탄·투사체 도우미, 수류탄 도우미를 지원하는 LAN 접근 가능 웹 레이더 등 시각 기능에 중점을 둡니다. DMA 접근에는 MemProcFS와 LeechCore를, UI에는 ImGui와 DirectX 11을 사용하며, 스캐터 일괄 읽기, 스냅샷 보간, 단계별 DMA 복구 같은 저지연 최적화를 포함합니다. cs2-dumper를 통한 오프셋 자동 업데이트, 이중 소스 핫키, 문제 해결 유틸리티도 갖추고 있습니다. DMA 기반 외부 CS2 분석 및 시각화 기법을 연구하는 게임 보안 연구원과 리버스 엔지니어를 대상으로 합니다.
