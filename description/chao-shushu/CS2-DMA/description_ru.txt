Этот проект — внешний инструмент для Counter-Strike 2 на C++, который использует DMA-оборудование на базе FPGA для чтения игровой памяти на отдельной машине и отрисовки оверлеев без внедрения в процесс игры. Открытая сборка — DMA только для чтения и ориентирована на визуалы: box и bone ESP, отображение оружия и боеприпасов, окраска видимости, помощники по бомбе и снарядам, а также веб-радар с доступом по LAN и поддержкой помощника гранат. Для DMA используются MemProcFS и LeechCore, для интерфейса — ImGui с DirectX 11; есть низколатентные оптимизации: пакетные scatter-чтения, интерполяция снимков и многоуровневое восстановление DMA. Автообновление оффсетов через cs2-dumper, горячие клавиши из двух источников и утилиты диагностики дополняют набор. Предназначен для исследователей игровой безопасности и реверс-инженеров, изучающих внешний анализ и визуализацию CS2 на основе DMA.
