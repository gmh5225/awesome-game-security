本プロジェクトは C++ で書かれた外部 Counter-Strike 2 ツールで、FPGA ベースの DMA ハードウェアを使い、別マシン上でゲームメモリを読み取り、ゲームプロセスへ注入せずにオーバーレイを描画します。オープンソース版は読み取り専用 DMA で、ボックス／ボーン ESP、武器・弾薬表示、可視性カラーリング、爆弾・投射物ヘルパー、グレネードヘルパー対応の LAN 利用可能な Web レーダーといった視覚機能に焦点を当てています。DMA アクセスには MemProcFS と LeechCore、UI には ImGui と DirectX 11 を用い、スキャッター一括読み取り、スナップショット補間、段階的 DMA リカバリなどの低遅延最適化を備えます。cs2-dumper によるオフセット自動更新、デュアルソースホットキー、トラブルシューティング用ユーティリティも揃っています。DMA ベースの外部 CS2 解析・可視化手法を研究するゲームセキュリティ研究者およびリバースエンジニア向けです。
