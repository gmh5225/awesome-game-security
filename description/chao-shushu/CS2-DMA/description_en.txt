This project is an external Counter-Strike 2 tool written in C++ that uses FPGA-based DMA hardware to read game memory on a separate machine and render overlays without injecting into the game process. The open-source build is read-only DMA and focuses on visuals such as box and bone ESP, weapon and ammo displays, visibility coloring, bomb and projectile helpers, and a LAN-accessible web radar with grenade helper support. It relies on MemProcFS and LeechCore for DMA access, ImGui with DirectX 11 for the UI, and includes low-latency optimizations like scatter batch reads, snapshot interpolation, and tiered DMA recovery. Automatic offset updates via cs2-dumper, dual-source hotkeys, and troubleshooting utilities round out the tooling. It is aimed at game-security researchers and reverse engineers studying DMA-based external CS2 analysis and visualization techniques.
