Este proyecto es una colección de encabezados de runtime y definiciones de metadatos de IL2CPP para múltiples versiones de Unity IL2CPP. Sirve como referencia de seguimiento de versiones sobre las estructuras internas, los cambios de API y la evolución del formato de metadatos del backend de scripting IL2CPP de Unity. Es especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que necesitan definiciones de estructuras IL2CPP específicas por versión al analizar juegos de Unity creados con distintos motores.
