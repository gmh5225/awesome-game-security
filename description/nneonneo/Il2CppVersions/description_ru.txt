Этот проект — коллекция заголовков runtime и определений метаданных IL2CPP для нескольких версий Unity IL2CPP. Он служит справочником для отслеживания версий внутренних структур, изменений API и эволюции формата метаданных скриптового бэкенда IL2CPP в Unity. Особенно полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, которым нужны определения структур IL2CPP для конкретных версий при анализе игр Unity, собранных на разных версиях движка.
