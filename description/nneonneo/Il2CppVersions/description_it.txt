Questo progetto è una raccolta di header runtime e definizioni di metadati IL2CPP per più versioni di Unity IL2CPP. Funge da riferimento per il tracciamento delle versioni sulle strutture interne, le modifiche alle API e l'evoluzione del formato dei metadati del backend di scripting IL2CPP di Unity. È particolarmente utile per ricercatori di sicurezza nei videogiochi e reverse engineer che necessitano di definizioni di strutture IL2CPP specifiche per versione per analizzare giochi Unity realizzati con diverse versioni del motore.
