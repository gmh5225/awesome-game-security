Este proyecto es un inyector de shaders D3D12 para FF7 Rebirth en PC que intercepta llamadas a la API de renderizado para inyectar o reemplazar pixel shaders en tiempo de ejecución.
Usa MinHook e ImGui, admite edición de shaders en vivo y, en teoría, puede adaptarse a otros títulos D3D12 porque es principalmente un interceptor de DX12.
Resulta especialmente útil para programadores gráficos y desarrolladores de herramientas de juegos en Windows que trabajan en el ámbito de DirectX / hooks.
