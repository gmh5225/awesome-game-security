Этот проект — инжектор шейдеров D3D12 для FF7 Rebirth на PC, который перехватывает вызовы API рендеринга, чтобы внедрять или заменять пиксельные шейдеры во время выполнения.
Он использует MinHook и ImGui, поддерживает живое редактирование шейдеров и теоретически может быть адаптирован под другие игры на D3D12, поскольку по сути это в первую очередь перехватчик DX12.
Он в основном полезен графическим программистам и разработчикам игровых инструментов под Windows, работающим в области DirectX / хуков.
