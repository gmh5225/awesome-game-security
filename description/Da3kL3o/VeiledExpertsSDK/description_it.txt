VeiledExpertsSDK è un dump di kit di sviluppo software generato per il gioco Veiled Experts. Contiene principalmente un ampio insieme di header C++ in stile Unreal Engine, insieme a tabelle di oggetti e nomi che descrivono classi, struct e metadati di runtime nel gioco. Questa codebase è utile per comprendere i layout dei dati interni del gioco e le superfici API quando si creano strumenti esterni o si esegue analisi statica. È destinata soprattutto a reverse engineer e ricercatori di sicurezza dei giochi che necessitano di una mappa di riferimento degli interni del gioco target.
