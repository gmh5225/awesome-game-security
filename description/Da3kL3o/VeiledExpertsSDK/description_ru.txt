VeiledExpertsSDK — это сгенерированный дамп комплекта средств разработки (SDK) для игры Veiled Experts. Он в основном содержит большой набор C++-заголовков в стиле Unreal Engine, а также таблицы объектов и имён, описывающие внутриигровые классы, структуры и метаданные времени выполнения. Кодовая база полезна для понимания внутренней раскладки игровых данных и поверхностей API при создании внешних инструментов или статическом анализе. Она предназначена прежде всего для реверс-инженеров и исследователей игровой безопасности, которым нужна справочная карта внутренностей целевой игры.
