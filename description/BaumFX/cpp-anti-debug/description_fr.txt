Ce projet est une bibliothèque C++ d'anti-débogage pour applications Windows. Elle regroupe de nombreuses vérifications : détection de débogueur basée sur le PEB et les API, sondes pilotées par exceptions, mesures de timing et tests CPU/registres de débogage. Le code propose à la fois des fonctions granulaires et un point d'entrée de contrôle de sécurité combiné pour une validation répétée à l'exécution. Le cas d'usage principal est le prototypage anti-altération et l'étude des techniques de détection de débogueurs.
