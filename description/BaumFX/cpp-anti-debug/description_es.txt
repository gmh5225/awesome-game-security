Este proyecto es una biblioteca C++ de anti-depuración para aplicaciones Windows. Agrega numerosas comprobaciones, incluida la detección de depuradores basada en PEB y API, sondas impulsadas por excepciones, mediciones de tiempo y pruebas de CPU/registros de depuración. El código ofrece tanto funciones granulares como un punto de entrada de comprobación de seguridad combinada para validación repetida en tiempo de ejecución. El caso de uso principal es el prototipado anti-manipulación y el estudio de técnicas de detección de depuradores.
