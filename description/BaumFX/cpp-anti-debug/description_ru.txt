Этот проект — библиотека антиотладки на C++ для приложений Windows. Она объединяет множество проверок, включая обнаружение отладчика на основе PEB и API, зонды на исключениях, измерения времени и тесты CPU/отладочных регистров. Код предоставляет как гранулярные функции, так и единую точку входа комплексной проверки безопасности для повторной валидации во время выполнения. Основной сценарий использования — прототипирование защиты от вмешательства и изучение техник обнаружения отладчиков.
