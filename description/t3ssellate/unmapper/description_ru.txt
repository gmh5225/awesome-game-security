Этот проект — автоматический инструмент для исправления дампов PE-файлов перед декомпиляцией.
Он изменяет заголовки PE так, чтобы декомпилятор или другие средства статического анализа могли корректно загрузить файл без ошибок.
В первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / исправления дампов.
