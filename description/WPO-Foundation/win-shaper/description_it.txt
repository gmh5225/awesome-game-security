Questo progetto è un filtro di pacchetti per il traffic shaping su Windows, pensato per emulare condizioni di rete reali.
Combina un driver kernel basato sui callout di Windows Filtering Platform con applicazioni di controllo da riga di comando e con interfaccia grafica.
Lo strumento può iniettare latenza, limitare la banda, applicare perdita di pacchetti e gestire le dimensioni delle code per il traffico in ingresso e in uscita.
È pensato per testare software e giochi sensibili alla rete in condizioni avverse controllate.
