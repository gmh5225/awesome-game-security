Этот проект — фильтр пакетов для traffic shaping в Windows, предназначенный для эмуляции реальных сетевых условий.
Он сочетает драйвер ядра на основе callout Windows Filtering Platform с приложениями управления через командную строку и графический интерфейс.
Инструмент может добавлять задержку, ограничивать пропускную способность, вносить потери пакетов и управлять размером очередей для входящего и исходящего трафика.
Он предназначен для тестирования сетечувствительного ПО и игр в контролируемых неблагоприятных условиях.
