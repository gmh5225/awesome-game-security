Este proyecto es un filtro de paquetes de traffic shaping para Windows destinado a emular condiciones de red del mundo real.
Combina un controlador de kernel basado en callouts de Windows Filtering Platform con aplicaciones de control por línea de comandos y GUI.
La herramienta puede inyectar latencia, limitar el ancho de banda, aplicar pérdida de paquetes y gestionar el tamaño de las colas del tráfico entrante y saliente.
Está pensada para probar software y juegos sensibles a la red bajo condiciones adversas controladas.
