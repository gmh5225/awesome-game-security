Dieses Projekt ist ein Windows-Traffic-Shaping-Paketfilter zur Emulation realer Netzwerkbedingungen.
Es kombiniert einen Kernel-Treiber auf Basis von Windows Filtering Platform Callouts mit Kommandozeilen- und GUI-Steueranwendungen.
Das Tool kann Latenz injizieren, Bandbreite begrenzen, Paketverlust anwenden und Warteschlangengrößen für eingehenden und ausgehenden Verkehr verwalten.
Es ist für das Testen netzwerksensitiver Software und Spiele unter kontrollierten ungünstigen Bedingungen gedacht.
