Этот проект — коллекция открытых библиотек на C/C++ для разработки игр.
Библиотеки написаны в духе data-oriented подхода: предпочтение отдаётся POD-структурам, а не классам, плоским массивам как основной структуре данных и т. д.
Он в первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / гайдов.
