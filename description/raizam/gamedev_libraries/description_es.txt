Este proyecto es una colección de bibliotecas C/C++ de código abierto para el desarrollo de juegos.
Las bibliotecas se han escrito con una filosofía orientada a datos: se prefieren las estructuras POD a las clases, los arrays planos son la estructura de datos preferida, etc.
Es especialmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo de juegos / guías.
