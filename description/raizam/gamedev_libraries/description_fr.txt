Ce projet est une collection de bibliothèques C/C++ open source pour le développement de jeux.
Les bibliothèques ont été écrites selon une philosophie orientée données : les structures POD sont préférées aux classes, les tableaux plats sont la structure de données privilégiée, etc.
Il est principalement utile aux développeurs de jeux, aux ingénieurs en rétro-ingénierie et aux créateurs d'outils travaillant dans le domaine du développement de jeux / guides.
