Questo progetto è una raccolta di librerie C/C++ open source per lo sviluppo di giochi.
Le librerie sono state scritte con una filosofia orientata ai dati: le strutture POD sono preferite alle classi, gli array piatti sono la struttura dati preferita, ecc.
È utile soprattutto per sviluppatori di giochi, reverse engineer e costruttori di tool che operano nell'area dello sviluppo di giochi / guide.
