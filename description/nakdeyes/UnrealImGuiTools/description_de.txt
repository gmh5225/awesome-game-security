Dieses Projekt ist eine Sammlung von Tools und Hilfsprogrammen für Unreal-Engine-Projekte, die ImGui verwenden.
Es bietet außerdem ein kleines Framework, mit dem sich spielspezifische Tool-Fenster schneller und einfacher erstellen lassen.
Es ist vor allem nützlich für Spieleentwickler, Engine-Programmierer und Grafikforscher, die im Bereich game engine / game engine plugins:unreal arbeiten.
