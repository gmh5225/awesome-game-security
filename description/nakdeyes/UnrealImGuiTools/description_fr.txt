Ce projet est un ensemble d'outils et d'utilitaires destinés aux projets Unreal Engine utilisant ImGui.
Il fournit également un petit framework pour écrire plus rapidement et plus facilement vos propres fenêtres d'outils spécifiques à un jeu.
Il est principalement utile pour les développeurs de jeux, les programmeurs moteur et les chercheurs en graphisme travaillant dans le domaine game engine / game engine plugins:unreal.
