Questo progetto è un insieme di strumenti e utilità per progetti Unreal Engine che usano ImGui.
Fornisce anche un piccolo framework per scrivere in modo più rapido e semplice le proprie finestre di strumenti specifiche per il gioco.
È principalmente utile per sviluppatori di giochi, programmatori di engine e ricercatori grafici che lavorano nell'area game engine / game engine plugins:unreal.
