Этот проект — набор инструментов и утилит для проектов Unreal Engine, использующих ImGui.
Он также предоставляет небольшой фреймворк, чтобы быстрее и проще писать собственные игровые окна инструментов.
В основном полезен разработчикам игр, программистам движка и исследователям графики, работающим в области game engine / game engine plugins:unreal.
