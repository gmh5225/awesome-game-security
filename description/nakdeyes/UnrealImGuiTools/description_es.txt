Este proyecto es un conjunto de herramientas y utilidades para proyectos de Unreal Engine que usan ImGui.
También proporciona un pequeño framework para escribir de forma más rápida y sencilla tus propias ventanas de herramientas específicas del juego.
Es principalmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el área game engine / game engine plugins:unreal.
