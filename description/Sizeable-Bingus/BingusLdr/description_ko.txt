BingusLdr는 Crystal Palace로 구축된 Windows x64 DLL 로더로, CET 호환 스택 스푸핑을 사용하면서 페이로드를 로드합니다. C로 작성되고 mingw-w64로 빌드되며, 로딩, 스티칭, 서비스, 메모리 마스킹용 모듈을 포함합니다. 주요 기능은 CET 친화적 스택 스푸핑, EAF 호환 API 해석, 그리고 로드된 코드의 노출을 줄이는 힙 및 이미지 마스킹입니다. .cna 스크립트와 LinkSpec 기반 빌드를 통해 Cobalt Strike와 연동할 수 있으며, 임의의 DLL을 PIC 스타일 바이너리로 감쌀 수도 있습니다. 주된 용도는 리플렉티브 로딩, 콜스택 회피, 그리고 안티치트 및 엔드포인트 탐지 분석과 관련된 기술에 대한 공격적 보안 연구입니다.
