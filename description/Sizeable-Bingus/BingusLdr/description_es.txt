BingusLdr es un cargador de DLL para Windows x64 construido con Crystal Palace que carga cargas útiles usando spoofing de pila compatible con CET. Está escrito en C y se construye con mingw-w64, con módulos para carga, stitching, servicios y enmascaramiento de memoria. Sus capacidades clave incluyen spoofing de pila compatible con CET, resolución de API compatible con EAF, y enmascaramiento de heap e imagen para reducir la exposición del código cargado. Se integra con Cobalt Strike mediante un script .cna y una compilación basada en LinkSpec, y también puede empaquetar DLL arbitrarias en un binario de estilo PIC. El caso de uso principal es la investigación de seguridad ofensiva sobre carga reflectiva, evasión de la pila de llamadas y técnicas relevantes para el análisis de anti-cheat y detección en endpoints.
