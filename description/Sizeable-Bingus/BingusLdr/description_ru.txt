BingusLdr — загрузчик DLL для Windows x64 на базе Crystal Palace, который загружает полезную нагрузку с использованием CET-совместимой подмены стека. Он написан на C и собирается с mingw-w64, с модулями для загрузки, stitching, сервисов и маскирования памяти. Ключевые возможности включают CET-friendly подмену стека, EAF-совместимое разрешение API, а также маскирование кучи и образа для снижения заметности загруженного кода. Интегрируется с Cobalt Strike через скрипт .cna и сборку на основе LinkSpec, а также может упаковывать произвольные DLL в бинарник в стиле PIC. Основное применение — исследование offensive security в области рефлексивной загрузки, обхода call stack и техник, релевантных для античита и анализа endpoint detection.
