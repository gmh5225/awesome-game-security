BingusLdr は Crystal Palace で構築された Windows x64 DLL ローダーで、CET 互換のスタックスプーフィングを用いながらペイロードを読み込みます。C で書かれ、mingw-w64 でビルドされ、ロード、スティッチング、サービス、メモリマスキング用のモジュールを備えます。主な機能は CET 対応のスタックスプーフィング、EAF 互換の API 解決、およびロード済みコードの露出を抑えるヒープ／イメージマスキングです。.cna スクリプトと LinkSpec ベースのビルドで Cobalt Strike と連携でき、任意の DLL を PIC 風バイナリに包むことも可能です。主な用途は、リフレクティブロード、コールスタック回避、およびアンチチートやエンドポイント検知分析に関連する技術の攻撃的セキュリティ研究です。
