BingusLdr is a Windows x64 DLL loader built with Crystal Palace that loads payloads while using CET-compatible stack spoofing. It is written in C and built with mingw-w64, with modules for loading, stitching, services, and memory masking. Key capabilities include CET-friendly stack spoofing, EAF-compatible API resolution, and heap and image masking to reduce exposure of loaded code. It integrates with Cobalt Strike through a .cna script and LinkSpec-based build, and can also wrap arbitrary DLLs into a PIC-style binary. The primary use case is offensive security research into reflective loading, call-stack evasion, and techniques relevant to anti-cheat and endpoint detection analysis.
