BingusLdr è un loader di DLL Windows x64 costruito con Crystal Palace che carica payload usando lo spoofing dello stack compatibile con CET. È scritto in C e compilato con mingw-w64, con moduli per il caricamento, lo stitching, i servizi e il mascheramento della memoria. Le capacità principali includono lo spoofing dello stack CET-friendly, la risoluzione API compatibile con EAF e il mascheramento di heap e image per ridurre l’esposizione del codice caricato. Si integra con Cobalt Strike tramite uno script .cna e una build basata su LinkSpec, e può anche avvolgere DLL arbitrarie in un binario in stile PIC. L’uso principale è la ricerca di sicurezza offensiva su reflective loading, evasione della call stack e tecniche rilevanti per l’analisi di anti-cheat e rilevamento endpoint.
