IDA2LLVM est un outil de lifting qui traduit le microcode IDA en LLVM IR pour une analyse en aval.
Il est principalement implémenté sous forme de script Python utilisant les API IDA avec llvmlite, et inclut la logique de correspondance des types et structures IDA vers les types LLVM.
Le dépôt fournit également des binaires d'exemple et des exemples d'IR généré pour illustrer les résultats de la traduction.
Il est destiné à la recherche en reverse engineering, aux pipelines d'analyse binaire et aux expériences combinant les artefacts de décompilation IDA avec les outils LLVM.
