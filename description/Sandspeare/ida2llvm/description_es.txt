IDA2LLVM es una herramienta de lifting que traduce el microcódigo de IDA a LLVM IR para análisis posteriores.
Se implementa principalmente como un script de Python que usa las API de IDA junto con llvmlite e incluye lógica para mapear tipos y estructuras de IDA a tipos de LLVM.
El repositorio también ofrece binarios de ejemplo e IR generado para demostrar los resultados de la traducción.
Está pensado para investigación de ingeniería inversa, pipelines de análisis binario y experimentos que combinan artefactos de descompilación de IDA con herramientas LLVM.
