IDA2LLVM è uno strumento di lifting che traduce il microcodice di IDA in LLVM IR per analisi successive.
È implementato principalmente come script Python che usa le API di IDA insieme a llvmlite e include la logica per mappare tipi e strutture IDA sui tipi LLVM.
Il repository fornisce anche binari di esempio e IR generato per dimostrare i risultati della traduzione.
È pensato per la ricerca di reverse engineering, pipeline di analisi binaria ed esperimenti che combinano artefatti di decompilazione IDA con gli strumenti LLVM.
