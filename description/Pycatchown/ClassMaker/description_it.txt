Questo progetto è un plugin IDA Python che ricostruisce automaticamente le classi C++ dal pseudocodice dei costruttori.
Traccia le assegnazioni alle vtable, crea o aggiorna le struct di IDA e applica euristiche di naming ai layout delle classi.
L’implementazione supporta workflow a 32 e 64 bit ed è pensata per il reverse engineering pratico, non per l’automazione completa del binario.
È soprattutto utile ai reverse engineer che vogliono accelerare il recupero manuale di classi e vtable in binari di giochi o nativi.
