Este proyecto es un plugin de IDA Python que reconstruye automáticamente clases C++ a partir del pseudocódigo de los constructores.
Rastrea asignaciones de vtable, crea o actualiza structs de IDA y aplica heurísticas de nombres a los layouts de clases.
La implementación cubre flujos de trabajo de 32 y 64 bits, y está pensada para reverse engineering práctico, no para automatización completa del binario.
Resulta especialmente útil para reverse engineers que quieren acelerar la recuperación manual de clases y vtables en binarios de juegos o nativos.
