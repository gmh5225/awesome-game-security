Questo progetto è GoReSym, uno strumento di Mandiant per estrarre metadati delle funzioni e informazioni sui tipi dai binari Go. Analizza le strutture pclntab (tabella delle linee PC) e moduledata del runtime Go per recuperare nomi di funzioni, percorsi dei file sorgente e definizioni di tipi anche da eseguibili Go privati dei simboli. Lo strumento produce i simboli recuperati in un formato compatibile con IDA Pro e altri disassemblatori. È rivolto ad analisti di malware e reverse engineer che devono ripristinare le informazioni sui simboli nei binari Go privati usati dal malware.
