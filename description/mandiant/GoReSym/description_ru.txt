Этот проект — GoReSym, инструмент от Mandiant для извлечения метаданных функций и информации о типах из бинарных файлов Go. Он разбирает структуры pclntab (таблица строк PC) и moduledata среды выполнения Go, чтобы восстановить имена функций, пути к исходным файлам и определения типов даже в stripped-исполняемых файлах Go. Инструмент выводит восстановленные символы в формате, совместимом с IDA Pro и другими дизассемблерами. Он предназначен для аналитиков вредоносного ПО и специалистов по обратной инженерии, которым нужно восстановить символьную информацию в stripped-бинарях Go, используемых вредоносными программами.
