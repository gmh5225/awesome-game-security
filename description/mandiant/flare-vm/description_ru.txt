Этот проект — FLARE VM, полностью настраиваемый дистрибутив безопасности на базе Windows от Mandiant для анализа вредоносного ПО, реагирования на инциденты и реверс-инжиниринга. С помощью Chocolatey и Boxstarter он автоматизирует установку сотен инструментов анализа — дизассемблеров (IDA, Ghidra, x64dbg), отладчиков, декомпиляторов, сетевых анализаторов, форензик-утилит и окружений Python/Ruby — на чистую Windows VM. Предназначен для аналитиков malware и reverse-инженеров, которым нужна воспроизводимая Windows-среда анализа с богатым набором инструментов.
