Dieses Projekt ist FLARE VM, eine vollständig anpassbare Windows-basierte Sicherheitsdistribution von Mandiant für Malware-Analyse, Incident Response und Reverse Engineering. Mit Chocolatey und Boxstarter automatisiert sie die Installation von Hunderten von Analysetools — darunter Disassembler (IDA, Ghidra, x64dbg), Debugger, Decompiler, Netzwerkanalysatoren, forensische Utilities sowie Python-/Ruby-Umgebungen — auf einer sauberen Windows-VM. Sie richtet sich an Malware-Analysten und Reverse Engineers, die eine reproduzierbare, toolreiche Windows-Analyseumgebung benötigen.
