Este proyecto es FLARE VM, una distribución de seguridad basada en Windows totalmente personalizable de Mandiant para análisis de malware, respuesta a incidentes e ingeniería inversa. Usa Chocolatey y Boxstarter para automatizar la instalación de cientos de herramientas de análisis —incluidos desensambladores (IDA, Ghidra, x64dbg), depuradores, decompiladores, analizadores de red, utilidades forenses y entornos Python/Ruby— en una máquina virtual Windows limpia. Está dirigida a analistas de malware e ingenieros inversos que necesitan un entorno de análisis Windows reproducible y rico en herramientas.
