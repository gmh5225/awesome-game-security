本プロジェクトは FLARE VM であり、Mandiant が提供する完全にカスタマイズ可能な Windows ベースのセキュリティディストリビューションで、マルウェア解析、インシデントレスポンス、リバースエンジニアリング向けです。Chocolatey と Boxstarter を使い、クリーンな Windows VM 上に、ディスアセンブラ（IDA、Ghidra、x64dbg）、デバッガ、デコンパイラ、ネットワーク解析ツール、フォレンジックユーティリティ、Python/Ruby 環境など数百の解析ツールの導入を自動化します。再現可能でツールが充実した Windows 解析環境を必要とするマルウェア解析者やリバースエンジニアを対象としています。
