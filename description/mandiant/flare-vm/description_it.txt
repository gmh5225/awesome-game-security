Questo progetto è FLARE VM, una distribuzione di sicurezza basata su Windows completamente personalizzabile di Mandiant per l’analisi di malware, incident response e reverse engineering. Usa Chocolatey e Boxstarter per automatizzare l’installazione di centinaia di strumenti di analisi — tra cui disassemblatori (IDA, Ghidra, x64dbg), debugger, decompilatori, analizzatori di rete, utility forensi e ambienti Python/Ruby — su una VM Windows pulita. È pensata per analisti di malware e reverse engineer che necessitano di un ambiente di analisi Windows riproducibile e ricco di strumenti.
