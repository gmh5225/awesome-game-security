이 프로젝트는 Mandiant의 FLARE VM으로, 멀웨어 분석, 사고 대응, 리버스 엔지니어링을 위한 완전 맞춤형 Windows 기반 보안 배포판입니다. Chocolatey와 Boxstarter를 사용해 깨끗한 Windows VM에 디스어셈블러(IDA, Ghidra, x64dbg), 디버거, 디컴파일러, 네트워크 분석기, 포렌식 유틸리티, Python/Ruby 환경을 포함한 수백 개의 분석 도구 설치를 자동화합니다. 재현 가능하고 도구가 풍부한 Windows 분석 환경이 필요한 멀웨어 분석가와 리버스 엔지니어를 대상으로 합니다.
