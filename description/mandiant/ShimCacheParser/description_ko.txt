이 프로젝트는 Windows 레지스트리 하이브에서 Application Compatibility Shim Cache(AppCompatCache) 항목을 파싱하는 Python 도구입니다. SYSTEM 레지스트리 하이브에 저장된 ShimCache 데이터에서 타임스탬프, 파일 경로, 실행 플래그를 추출하며, XP부터 Windows 10까지 여러 Windows 버전을 지원합니다. 파싱된 결과는 CSV 또는 타임라인 형식으로 내보내 포렌식 분석에 활용할 수 있습니다. Windows 시스템의 프로그램 실행 기록을 조사하는 인시던트 대응 담당자와 디지털 포렌식 분석가를 대상으로 합니다.
