Этот проект — инструмент на Python для разбора записей Application Compatibility Shim Cache (AppCompatCache) из кустов реестра Windows. Он извлекает временные метки, пути к файлам и флаги выполнения из данных ShimCache, хранящихся в кусте SYSTEM, и поддерживает несколько версий Windows от XP до Windows 10. Результаты разбора можно экспортировать в формате CSV или временной шкалы для криминалистического анализа. Инструмент предназначен для специалистов по реагированию на инциденты и аналитиков цифровой криминалистики, исследующих историю запуска программ в системах Windows.
