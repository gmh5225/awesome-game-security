Ce projet est un outil Python permettant d'analyser les entrées Application Compatibility Shim Cache (AppCompatCache) des ruches du Registre Windows. Il extrait les horodatages, chemins de fichiers et indicateurs d'exécution des données ShimCache stockées dans la ruche SYSTEM, et prend en charge plusieurs versions de Windows, de XP à Windows 10. Les résultats analysés peuvent être exportés au format CSV ou chronologique pour l'analyse forensique. Il s'adresse aux équipes de réponse aux incidents et aux analystes en criminalistique numérique qui enquêtent sur l'historique d'exécution des programmes sur les systèmes Windows.
