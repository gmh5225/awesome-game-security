Este proyecto es una herramienta en Python para analizar entradas de Application Compatibility Shim Cache (AppCompatCache) desde las colmenas del Registro de Windows. Extrae marcas de tiempo, rutas de archivos e indicadores de ejecución de los datos ShimCache almacenados en la colmena SYSTEM, y admite varias versiones de Windows desde XP hasta Windows 10. La salida analizada puede exportarse en formato CSV o de línea de tiempo para el análisis forense. Está dirigida a responsables de respuesta a incidentes y analistas de informática forense que investigan el historial de ejecución de programas en sistemas Windows.
