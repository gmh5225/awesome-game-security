本プロジェクトは、Windows レジストリハイブ内の Application Compatibility Shim Cache（AppCompatCache）エントリを解析する Python ツールです。SYSTEM レジストリハイブに保存された ShimCache データからタイムスタンプ、ファイルパス、実行フラグを抽出し、XP から Windows 10 までの複数の Windows バージョンに対応します。解析結果は CSV またはタイムライン形式でエクスポートでき、フォレンジック分析に利用できます。Windows システム上のプログラム実行履歴を調査するインシデント対応担当者やデジタルフォレンジック分析者を対象としています。
