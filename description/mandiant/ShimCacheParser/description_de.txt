Dieses Projekt ist ein Python-Tool zum Parsen von Application Compatibility Shim Cache (AppCompatCache)-Einträgen aus Windows-Registrierungsstrukturen. Es extrahiert Zeitstempel, Dateipfade und Ausführungsflags aus den ShimCache-Daten in der SYSTEM-Registrierungsstruktur und unterstützt mehrere Windows-Versionen von XP bis Windows 10. Die geparsten Ergebnisse können im CSV- oder Timeline-Format für forensische Analysen exportiert werden. Es richtet sich an Incident Responder und digitale Forensik-Analysten, die die Programmausführungshistorie auf Windows-Systemen untersuchen.
