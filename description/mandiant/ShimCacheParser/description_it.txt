Questo progetto è uno strumento Python per l'analisi delle voci Application Compatibility Shim Cache (AppCompatCache) dalle hive del Registro di Windows. Estrae timestamp, percorsi dei file e flag di esecuzione dai dati ShimCache memorizzati nella hive SYSTEM, supportando più versioni di Windows da XP a Windows 10. L'output analizzato può essere esportato in formato CSV o timeline per l'analisi forense. È rivolto a chi risponde agli incidenti e agli analisti di digital forensics che indagano sulla cronologia di esecuzione dei programmi sui sistemi Windows.
