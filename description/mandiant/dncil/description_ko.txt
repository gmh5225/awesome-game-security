이 프로젝트는 dncil로, Python으로 작성된 .NET 바이너리용 Common Intermediate Language(CIL) 디스어셈블리 라이브러리입니다. .NET PE 메타데이터를 파싱하고 메서드 본체를 피연산자 해석이 포함된 개별 CIL 명령으로 디스어셈블하여 .NET 바이너리 분석 도구의 기반을 제공합니다. 모든 CIL 옵코드와 메타데이터 토큰 해석을 지원합니다. 자동화된 .NET 분석 도구를 구축하거나 YARA 규칙 및 탐지 파이프라인에 CIL 디스어셈블리를 통합하는 악성코드 분석가와 리버스 엔지니어를 대상으로 합니다.
