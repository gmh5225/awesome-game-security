Questo progetto è dncil, una libreria di disassemblaggio Common Intermediate Language (CIL) per binari .NET scritta in Python. Analizza i metadati PE .NET e disassembla i corpi dei metodi in singole istruzioni CIL con risoluzione degli operandi, fornendo una base per gli strumenti di analisi dei binari .NET. La libreria supporta tutti gli opcode CIL e la risoluzione dei token di metadati. È pensata per analisti di malware e reverse engineer che creano strumenti automatizzati di analisi .NET o integrano il disassemblaggio CIL nelle regole YARA e nelle pipeline di rilevamento.
