Этот проект — dncil, библиотека дизассемблирования Common Intermediate Language (CIL) для .NET-бинарей, написанная на Python. Она разбирает метаданные .NET PE и дизассемблирует тела методов в отдельные инструкции CIL с разрешением операндов, создавая основу для инструментов анализа .NET-бинарей. Библиотека поддерживает все опкоды CIL и разрешение токенов метаданных. Она предназначена для аналитиков вредоносного ПО и специалистов по реверс-инжинирингу, создающих автоматизированные средства анализа .NET или интегрирующих дизассемблирование CIL в правила YARA и конвейеры обнаружения.
