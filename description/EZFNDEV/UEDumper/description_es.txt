Este proyecto es un dumper de Unreal Engine que genera salida al estilo SDK a partir de juegos en ejecución. Está escrito principalmente en C++ y se centra en la extracción de nombres de objetos, el descubrimiento de offsets y la generación de encabezados para títulos basados en UE. El README describe un flujo de trabajo basado en inyección y la búsqueda automática de offsets, además de señalar limitaciones de versión y algunas partes codificadas de forma fija o inestables. Está pensado principalmente para la ingeniería inversa de Unreal y la investigación en seguridad de juegos.
