Этот проект — дампер Unreal Engine, который формирует вывод в стиле SDK из запущенных игр. Он в основном написан на C++ и ориентирован на извлечение имён объектов, поиск смещений и генерацию заголовков для игр на базе UE. В README описан рабочий процесс на основе инъекции и автоматический поиск смещений, а также отмечены ограничения по версиям и некоторые жёстко заданные или нестабильные части. Предназначен главным образом для реверс-инжиниринга Unreal и исследований в области безопасности игр.
