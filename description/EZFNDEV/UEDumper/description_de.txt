Dieses Projekt ist ein Unreal Engine Dumper, der SDK-ähnliche Ausgaben aus laufenden Spielen erzeugt. Es ist hauptsächlich in C++ geschrieben und zielt auf die Extraktion von Objektnamen, die Ermittlung von Offsets sowie die Header-Generierung für UE-basierte Titel ab. Die README beschreibt einen Injektions-basierten Workflow und die automatische Offset-Suche und weist zugleich auf Versionsbeschränkungen sowie einige hartcodierte oder instabile Teile hin. Es ist vor allem für Unreal-Reverse-Engineering und Game-Security-Forschung gedacht.
