Этот проект — драйвер Windows в режиме ядра для спуфинга HWID. Код в основном написан на C и собран с KMDF; в нём есть процедуры, перехватывающие пути управления диском, монтированием и сетью для изменения значений, идентифицирующих оборудование. Содержит логику изменения идентификаторов дисков, данных NIC, сведений SMBIOS и строк, связанных с GPU. Основной сценарий использования — обход античита и низкоуровневое исследование аппаратной идентичности в контексте безопасности игр.
