Este proyecto es un controlador Windows en modo kernel para spoofing de HWID. El código está principalmente en C y se construye con KMDF, con rutinas que enganchan las rutas de control de disco, montaje y red para alterar valores de identificación de hardware. Contiene lógica para cambiar identificadores relacionados con discos, datos de NIC, información SMBIOS y cadenas relacionadas con la GPU. El caso de uso principal es la evasión de anti-cheat y la investigación de bajo nivel sobre la identidad de hardware en contextos de seguridad de juegos.
