Este proyecto es un lanzador de spoofer HWID para escritorio Windows con una GUI personalizada. Está escrito en C++ y usa Dear ImGui con DirectX 9 para presentar una interfaz sencilla y ejecutar acciones de spoofing. El flujo de la aplicación descarga binarios externos de controlador y utilidades auxiliares, y luego ejecuta un paso de línea de comandos para aplicar los cambios de spoofing. Su caso de uso previsto son flujos de trabajo de evasión de anti-cheat en juegos, especialmente para usuarios que prefieren un front end de un solo clic.
