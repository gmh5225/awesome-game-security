Questo progetto è un launcher di HWID spoofer per desktop Windows con una GUI personalizzata. È scritto in C++ e usa Dear ImGui con DirectX 9 per presentare un'interfaccia semplice ed eseguire azioni di spoofing. Il flusso dell'applicazione scarica binari esterni di driver e helper, quindi esegue un passaggio da riga di comando per applicare le modifiche di spoofing. Il caso d'uso previsto sono i workflow di bypass dell'anti-cheat nei giochi, in particolare per gli utenti che preferiscono un front end one-click.
