Этот проект — лаунчер HWID-спуфера для Windows с пользовательским GUI. Он написан на C++ и использует Dear ImGui вместе с DirectX 9 для отображения простого интерфейса и выполнения операций спуфинга. Рабочий процесс приложения загружает внешние драйверные и вспомогательные бинарники, затем выполняет шаг командной строки для применения изменений спуфинга. Предполагаемый сценарий использования — рабочие процессы обхода игрового античита, особенно для пользователей, предпочитающих фронтенд в один клик.
