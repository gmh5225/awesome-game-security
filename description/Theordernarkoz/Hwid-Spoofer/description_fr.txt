Ce projet est un lanceur de spoofer HWID pour bureau Windows, doté d'une interface graphique personnalisée. Il est écrit en C++ et utilise Dear ImGui avec DirectX 9 pour afficher une interface simple et exécuter les actions de spoofing. Le flux de l'application télécharge des binaires externes de pilote et d'assistance, puis exécute une étape en ligne de commande pour appliquer les modifications de spoofing. Son cas d'usage prévu concerne les workflows de contournement d'anti-cheat de jeux, en particulier pour les utilisateurs qui préfèrent une interface frontale en un clic.
