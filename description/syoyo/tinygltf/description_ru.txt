Этот проект — TinyGLTF, header-only C++ библиотека для загрузки и сохранения файлов 3D-моделей glTF 2.0. Она разбирает форматы glTF JSON и бинарный GLB, обрабатывая меши, материалы, текстуры, анимации, скины, камеры и расширения, без внешних зависимостей, кроме stb_image для загрузки текстур. Библиотека поддерживает чтение из файлов, буферов памяти и пользовательских I/O-колбэков. Предназначена для разработчиков игр и программистов 3D-приложений, которым нужен лёгкий и переносимый загрузчик glTF.
