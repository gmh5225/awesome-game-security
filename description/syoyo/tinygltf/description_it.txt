Questo progetto è TinyGLTF, una libreria C++ header-only per caricare e salvare file di modelli 3D glTF 2.0. Analizza sia il formato glTF JSON sia quello binario GLB, gestendo mesh, materiali, texture, animazioni, skin, telecamere ed estensioni, senza dipendenze esterne oltre a stb_image per il caricamento delle texture. La libreria supporta la lettura da file, buffer di memoria e callback I/O personalizzati. È pensata per sviluppatori di giochi e programmatori di applicazioni 3D che necessitano di un loader glTF leggero e portabile.
