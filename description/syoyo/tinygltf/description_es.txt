Este proyecto es TinyGLTF, una biblioteca C++ de solo cabeceras para cargar y guardar archivos de modelos 3D glTF 2.0. Analiza tanto el formato glTF JSON como el binario GLB, gestionando mallas, materiales, texturas, animaciones, skins, cámaras y extensiones, sin dependencias externas más allá de stb_image para la carga de texturas. La biblioteca admite lectura desde archivos, buffers de memoria y callbacks de E/S personalizados. Está orientada a desarrolladores de juegos y programadores de aplicaciones 3D que necesitan un cargador glTF ligero y portable.
