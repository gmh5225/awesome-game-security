Questo progetto è un emulatore di server privato di MapleStory rivolto alla versione KMS 316. Il server basato su Java include schemi di database SQL, script di eventi e NPC in JavaScript che coprono scontri con i boss, missioni di gruppo e sistemi di progressione, insieme a configurazioni di gestione dei pacchetti e file di proprietà del server. È soprattutto utile per ricercatori di emulazione di server di gioco e comunità di modding di MapleStory che studiano l'architettura dei server MMORPG e la reimplementazione della logica di gioco.
