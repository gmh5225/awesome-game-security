Spirit は、Pokémon Trading Card Game Online（PTCGO）を Python で再実装したプライベートサーバーで、クライアントがローカルまたは自ホスト環境で接続できる完全なゲームバックエンドを提供します。多数のセットにわたるカードルールと効果、アカウントとインベントリのデータベース、経済・ショップシステム、対戦、ライブトーナメント、アバター・スリーブ・パックなどのカスタムコスメティックを実装しています。コードベースには HTTP およびゲームサーバースタック、クライアントプロトコル向けの Protocol Buffer 定義、カードバンドルのリバースエンジニアリング用ヘルパー、nginx または Caddy 向けのデプロイ設定が含まれます。PTCGO のネットワーキング、ゲームロジック、アセットツールを研究するリバースエンジニア、プライベートサーバー運用者、研究者向けです。
