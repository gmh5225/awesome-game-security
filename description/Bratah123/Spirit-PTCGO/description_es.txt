Spirit es una reimplementación en Python de un servidor privado de Pokémon Trading Card Game Online (PTCGO), que ofrece un backend de juego completo al que los clientes pueden conectarse de forma local o en entornos autoalojados. Implementa reglas y efectos de cartas en muchos sets, bases de datos de cuentas e inventario, sistemas de economía y tienda, juego versus, torneos en vivo y cosmética personalizada como avatares, sleeves y packs. El código incluye una pila de servidor HTTP y de juego, definiciones Protocol Buffer para el protocolo del cliente, ayudas de ingeniería inversa para paquetes de cartas y configuraciones de despliegue para nginx o Caddy. Está orientado a ingenieros inversos, operadores de servidores privados e investigadores que estudian la red de PTCGO, la lógica del juego y las herramientas de assets.
