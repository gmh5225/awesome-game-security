Spirit è una reimplementazione in Python di un server privato di Pokémon Trading Card Game Online (PTCGO), che fornisce un backend di gioco completo a cui i client possono connettersi in locale o in ambienti self-hosted. Implementa regole ed effetti delle carte su molti set, database di account e inventario, sistemi di economia e negozio, gioco versus, tornei live e cosmetici personalizzati come avatar, sleeves e pack. Il codice include uno stack di server HTTP e di gioco, definizioni Protocol Buffer per il protocollo client, helper di reverse engineering per i bundle di carte e configurazioni di deployment per nginx o Caddy. È rivolto a reverse engineer, operatori di server privati e ricercatori che studiano il networking PTCGO, la logica di gioco e gli strumenti per gli asset.
