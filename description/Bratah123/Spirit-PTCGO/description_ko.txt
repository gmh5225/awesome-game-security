Spirit는 Pokémon Trading Card Game Online(PTCGO)을 Python으로 재구현한 프라이빗 서버로, 클라이언트가 로컬 또는 셀프호스팅 환경에서 연결할 수 있는 완전한 게임 백엔드를 제공합니다. 여러 세트의 카드 규칙과 효과, 계정·인벤토리 데이터베이스, 경제·상점 시스템, 대전, 라이브 토너먼트, 아바타·슬리브·팩 같은 커스텀 코스메틱을 구현합니다. 코드베이스에는 HTTP 및 게임 서버 스택, 클라이언트 프로토콜용 Protocol Buffer 정의, 카드 번들 리버스 엔지니어링 헬퍼, nginx 또는 Caddy용 배포 설정이 포함됩니다. PTCGO 네트워킹, 게임 로직, 에셋 툴링을 연구하는 리버스 엔지니어, 프라이빗 서버 운영자, 연구자를 대상으로 합니다.
