AtomicShield Client è un client anti-cheat per Windows che collega le macchine dei giocatori a un server AtomicShield, con particolare attenzione alla protezione dei giochi FiveM. Combina un agente C#/.NET (dashboard WinForms e WebView2) che scarica gli aggiornamenti e comunica con il backend tramite API HTTP crittografate, con un motore anti-cheat nativo in C++. Il motore include protezioni su processi, moduli, euristiche e manual mapping, anti-debugging, raccolta di hardware ID, screenshot e networking basato su WebSocket, oltre ad AES e crittografia correlata nelle librerie condivise. I componenti separati EngineLoader e RuntimeLoader usano manual mapping in-process e named pipes per caricare ed eseguire il motore. Il caso d’uso principale è l’anti-cheat di gioco lato client per operatori che necessitano di monitoraggio, controlli di integrità e enforcement supportato dal server sui giocatori Windows.
