AtomicShield Client は、プレイヤーのマシンを AtomicShield サーバーに接続する Windows 向けアンチチートクライアントで、FiveM のゲーム保護に重点を置いています。更新のダウンロードと暗号化 HTTP API によるバックエンド通信を担う C#/.NET エージェント（WinForms と WebView2 ダッシュボード）と、ネイティブ C++ アンチチートエンジンを組み合わせています。エンジンにはプロセス・モジュール・ヒューリスティック・マニュアルマッピングのガード、アンチデバッグ、ハードウェア ID 収集、スクリーンショット、WebSocket ベースのネットワーキングに加え、共有ライブラリ内の AES および関連暗号機能が含まれます。独立した EngineLoader と RuntimeLoader コンポーネントは、プロセス内マニュアルマッピングと名前付きパイプを使ってエンジンをロードし実行します。主な用途は、Windows プレイヤーに対する監視、整合性チェック、サーバー連携の強制執行を必要とする運用者向けのクライアント側ゲームアンチチートです。
