AtomicShield Client is a Windows anti-cheat client that connects player machines to an AtomicShield server, with a focus on FiveM game protection. It combines a C#/.NET agent (WinForms and WebView2 dashboard) that downloads updates and talks to the backend over encrypted HTTP APIs with a native C++ anti-cheat engine. The engine includes process, module, heuristic, and manual-mapping guards, anti-debugging, hardware ID collection, screenshots, and WebSocket-based networking, plus AES and related crypto in shared libraries. Separate EngineLoader and RuntimeLoader components use in-process manual mapping and named pipes to load and run the engine. The primary use case is client-side game anti-cheat for operators who need monitoring, integrity checks, and server-backed enforcement on Windows players.
