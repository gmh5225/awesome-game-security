AtomicShield Client es un cliente anticheat para Windows que conecta las máquinas de los jugadores a un servidor AtomicShield, con énfasis en la protección de juegos FiveM. Combina un agente C#/.NET (panel WinForms y WebView2) que descarga actualizaciones y se comunica con el backend mediante API HTTP cifradas, con un motor anticheat nativo en C++. El motor incluye protecciones de procesos, módulos, heurísticas y manual mapping, anti-depuración, recopilación de identificadores de hardware, capturas de pantalla y red basada en WebSocket, además de AES y criptografía relacionada en bibliotecas compartidas. Los componentes separados EngineLoader y RuntimeLoader usan manual mapping en proceso y named pipes para cargar y ejecutar el motor. El caso de uso principal es el anticheat de juegos del lado del cliente para operadores que necesitan monitorización, comprobaciones de integridad y aplicación respaldada por servidor en jugadores Windows.
