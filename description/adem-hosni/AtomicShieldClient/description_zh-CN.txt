AtomicShield Client 是一款 Windows 反作弊客户端，可将玩家机器连接到 AtomicShield 服务器，重点面向 FiveM 游戏防护。它将 C#/.NET 代理（WinForms 与 WebView2 控制面板，负责下载更新并通过加密 HTTP API 与后端通信）与原生 C++ 反作弊引擎结合使用。该引擎包含进程、模块、启发式与手动映射防护、反调试、硬件 ID 采集、截图以及基于 WebSocket 的网络通信，并在共享库中提供 AES 及相关加密。独立的 EngineLoader 与 RuntimeLoader 组件通过进程内手动映射和命名管道加载并运行引擎。主要用例是面向运营方的客户端游戏反作弊，用于对 Windows 玩家进行监控、完整性检查以及由服务器侧支持的强制执行。
