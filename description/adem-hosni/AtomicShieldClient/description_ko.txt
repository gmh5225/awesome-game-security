AtomicShield Client는 플레이어 PC를 AtomicShield 서버에 연결하는 Windows 안티치트 클라이언트로, FiveM 게임 보호에 중점을 둡니다. 업데이트를 다운로드하고 암호화된 HTTP API로 백엔드와 통신하는 C#/.NET 에이전트(WinForms 및 WebView2 대시보드)와 네이티브 C++ 안티치트 엔진을 결합합니다. 엔진에는 프로세스·모듈·휴리스틱·수동 매핑 가드, 안티디버깅, 하드웨어 ID 수집, 스크린샷, WebSocket 기반 네트워킹이 포함되며, 공유 라이브러리에서 AES 및 관련 암호화를 제공합니다. 별도의 EngineLoader와 RuntimeLoader 컴포넌트는 프로세스 내 수동 매핑과 네임드 파이프를 사용해 엔진을 로드하고 실행합니다. 주요 용도는 Windows 플레이어에 대한 모니터링, 무결성 검사, 서버 연동 강제 조치가 필요한 운영자를 위한 클라이언트 측 게임 안티치트입니다.
