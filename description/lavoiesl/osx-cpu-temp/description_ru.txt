Этот проект — инструмент командной строки для чтения температуры CPU в macOS через интерфейс SMC (System Management Controller). Он запрашивает у Apple SMC показания тепловых датчиков и выводит текущую температуру CPU в градусах Цельсия или Фаренгейта. Реализация на C напрямую обращается к службе AppleSMC в IOKit. Предназначен для разработчиков macOS и авторов средств системного мониторинга, которым нужен программный доступ к данным о температуре оборудования.
