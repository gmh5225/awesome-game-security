Questo progetto è uno strumento a riga di comando per leggere la temperatura della CPU su macOS tramite l'interfaccia SMC (System Management Controller). Interroga l'Apple SMC per le letture dei sensori termici e mostra la temperatura attuale della CPU in Celsius o Fahrenheit. L'implementazione in C accede direttamente al servizio AppleSMC di IOKit. È pensato per sviluppatori macOS e creatori di strumenti di monitoraggio di sistema che necessitano di accesso programmatico ai dati di temperatura hardware.
