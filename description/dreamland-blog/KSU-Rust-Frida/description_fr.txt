Un module KernelSU écrit en Rust qui charge le Frida gadget dans les applications Android cibles au démarrage, combinant la modification d'applications au niveau noyau de KernelSU avec les capacités d'instrumentation dynamique de Frida.
Il injecte la bibliothèque partagée Frida dans le processus issu du fork de Zygote avant l'exécution du code de l'application, permettant le hooking à l'exécution sans devoir contourner la détection de root.
Il est surtout utile aux chercheurs en sécurité mobile qui effectuent l'analyse dynamique et l'instrumentation de jeux et d'applications Android avec KernelSU et Frida.
