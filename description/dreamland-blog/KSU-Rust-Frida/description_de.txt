Ein in Rust geschriebenes KernelSU-Modul, das beim Start Frida Gadget in Ziel-Android-Anwendungen lädt und damit KernelSUs kernellevel-App-Modifikation mit Fridas dynamischer Instrumentierung kombiniert.
Es injiziert die Frida-Shared-Library in den von Zygote geforkten Prozess, bevor der eigene Code der App ausgeführt wird, und ermöglicht so Runtime-Hooking ohne Umgehung der Root-Erkennung.
Es ist vor allem für Mobile-Security-Forscher nützlich, die mit KernelSU und Frida dynamische Analyse und Instrumentierung von Android-Spielen und -Apps durchführen.
