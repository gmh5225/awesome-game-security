Dieses Projekt ist ein Unreal-Engine-Tutorial-Plugin, das zeigt, wie man mit C++ einen eigenen Asset-Typ und einen zugehörigen Editor erstellt. Es umfasst Runtime- und Editor-Module, beispielhafte UObject-Klassen sowie benutzerdefinierte Editor-Werkzeuge für ein NormalDistribution-Asset. Der Code folgt den Unreal-Plugin-Konventionen mit Modul-Build-Skripten und Reflection-fähigen Klassen. Es richtet sich vor allem an Unreal-Entwickler, die eine praxisnahe Referenz zur Erweiterung der Asset-Pipeline und des Editor-Workflows der Engine suchen.
