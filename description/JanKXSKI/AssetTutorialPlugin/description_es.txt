Este proyecto es un plugin tutorial de Unreal Engine que muestra cómo crear un tipo de asset personalizado y un editor dedicado en C++. Incluye módulos de runtime y de editor, clases UObject de ejemplo y herramientas de editor personalizadas para trabajar con un asset NormalDistribution. El código sigue las convenciones de plugins de Unreal, con scripts de compilación de módulos y clases habilitadas para reflexión. Está dirigido principalmente a desarrolladores de Unreal que buscan una referencia práctica para ampliar el pipeline de assets y el flujo de trabajo del editor del motor.
