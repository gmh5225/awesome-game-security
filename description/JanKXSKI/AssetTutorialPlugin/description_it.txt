Questo progetto è un plugin tutorial di Unreal Engine che mostra come creare un tipo di asset personalizzato e un editor dedicato in C++. Include moduli runtime ed editor, classi UObject di esempio e strumenti editor personalizzati per lavorare con un asset NormalDistribution. Il codice segue le convenzioni dei plugin Unreal, con script di build dei moduli e classi abilitate alla reflection. È pensato soprattutto per gli sviluppatori Unreal che cercano un riferimento pratico per estendere la pipeline degli asset e il flusso di lavoro dell'editor del motore.
