Ce projet est un plugin tutoriel Unreal Engine qui montre comment créer un type d'asset personnalisé et un éditeur dédié en C++. Il comprend des modules runtime et éditeur, des classes UObject d'exemple, et des outils d'éditeur personnalisés pour travailler avec un asset NormalDistribution. Le code suit les conventions des plugins Unreal, avec des scripts de build de modules et des classes compatibles avec la réflexion. Il s'adresse surtout aux développeurs Unreal qui cherchent une référence pratique pour étendre le pipeline d'assets et le flux de travail de l'éditeur du moteur.
