Ce projet est un serveur Model Context Protocol (MCP) qui intègre l’analyse documentaire DeepWiki dans les flux de travail de développement assistés par l’IA. Il permet aux assistants IA d’interroger et de récupérer des informations depuis la documentation projet, les wikis et les bases de connaissances via le protocole MCP, offrant un accès contextuel à la documentation pendant l’analyse et le développement du code. Il s’adresse aux développeurs utilisant des outils IA compatibles MCP qui souhaitent une recherche et une référence documentaires automatisées.
