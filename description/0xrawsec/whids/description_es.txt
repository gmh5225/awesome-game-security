Este proyecto es una plataforma EDR de Windows de código abierto centrada en la respuesta impulsada por la detección.
Está construida en gran parte en Go y se basa en telemetría ETW y Sysmon, con evaluación de reglas impulsada por el motor Gene.
Las alertas pueden desencadenar la recolección de artefactos casi en tiempo real, como archivos, datos del registro y memoria de procesos, y la plataforma incluye un servicio manager con una API administrativa.
Su caso de uso principal es la respuesta a incidentes y la monitorización de endpoints empresariales con lógica de detección transparente y personalizable.
