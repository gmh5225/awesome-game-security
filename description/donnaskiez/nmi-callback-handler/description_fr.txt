Un pilote noyau Windows qui montre comment les interruptions non masquables (NMI) peuvent servir au parcours de pile en localisant la structure MACHINE_FRAME depuis le retour ISR iretq afin de déterminer le pointeur d'instruction interrompu.
Il enregistre un gestionnaire de rappel NMI qui capture le RIP interrompu sur tous les processeurs, permettant de détecter du code exécuté depuis des régions mémoire suspectes.
Il est surtout utile aux développeurs d'anti-cheat et aux chercheurs en sécurité noyau qui étudient le parcours de pile de threads basé sur les NMI comme technique de détection de triche.
