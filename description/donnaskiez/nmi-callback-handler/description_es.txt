Un controlador de kernel de Windows que demuestra cómo las interrupciones no enmascarables (NMI) pueden usarse para recorrer la pila localizando la estructura MACHINE_FRAME desde el retorno ISR de iretq a fin de determinar el puntero de instrucción interrumpido.
Registra un controlador de devolución de llamada NMI que captura el RIP interrumpido en todos los procesadores, lo que permite detectar código que se ejecuta desde regiones de memoria sospechosas.
Resulta especialmente útil para desarrolladores de anti-cheat e investigadores de seguridad del kernel que estudian el recorrido de pila de hilos basado en NMI como técnica de detección de trampas.
