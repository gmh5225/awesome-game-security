Ein Windows-Kerneltreiber, der zeigt, wie Non-Maskable Interrupts (NMIs) für Stack Walking genutzt werden können, indem die MACHINE_FRAME-Struktur beim iretq-ISR-Rücksprung lokalisiert wird, um den unterbrochenen Instruction Pointer zu ermitteln.
Er registriert einen NMI-Callback-Handler, der den unterbrochenen RIP über alle Prozessoren erfasst und so die Erkennung von Code ermöglicht, der aus verdächtigen Speicherbereichen ausgeführt wird.
Er ist vor allem für Anti-Cheat-Entwickler und Kernel-Sicherheitsforscher nützlich, die NMI-basiertes Thread-Stack-Walking als Cheat-Erkennungstechnik untersuchen.
