Controlador anti-trampas de código abierto para el kernel de Windows que implementa múltiples vectores de detección, incluidos el recorrido de pila basado en NMI, el análisis de pila APC/DPC, comprobaciones de integridad de .text, el stripping de handles mediante object callbacks y la detección de punteros de datos encadenados.
Incluye detección de hilos adjuntos, detección de hooks de excepción en direcciones de retorno, verificación de device objects de módulos del sistema y enumeración de la tabla de handles de procesos para identificar patrones de acceso sospechosos.
Resulta especialmente útil para ingenieros de anti-trampas e investigadores de seguridad en videojuegos que estudian técnicas de detección de trampas a nivel de kernel y construyen sistemas anti-trampas defensivos.
