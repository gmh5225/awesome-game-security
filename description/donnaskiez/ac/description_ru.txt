Открытый античит-драйвер ядра Windows, реализующий несколько векторов обнаружения: обход стека на базе NMI, анализ стека APC/DPC, проверки целостности .text, снятие handle через object callbacks и обнаружение цепочек указателей данных.
Поддерживает обнаружение прикреплённых потоков, обнаружение exception-хуков адресов возврата, проверку device object системных модулей и перечисление таблицы handle процессов для выявления подозрительных шаблонов доступа.
В основном полезен инженерам античита и исследователям игровой безопасности, изучающим методы обнаружения читов на уровне ядра и создающим защитные античит-системы.
