Driver anti-cheat open source per il kernel di Windows che implementa più vettori di rilevamento, tra cui stack walking basato su NMI, analisi dello stack APC/DPC, controlli di integrità di .text, handle stripping tramite object callback e rilevamento di puntatori a dati concatenati.
Offre rilevamento di thread allegati, rilevamento di hook di eccezione sugli indirizzi di ritorno, verifica dei device object dei moduli di sistema ed enumerazione della tabella degli handle di processo per identificare schemi di accesso sospetti.
È principalmente utile per ingegneri anti-cheat e ricercatori di sicurezza nei videogiochi che studiano tecniche di rilevamento delle cheat a livello kernel e costruiscono sistemi anti-cheat difensivi.
