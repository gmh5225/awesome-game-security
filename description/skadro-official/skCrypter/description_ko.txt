이 프로젝트는 skCrypter로, 컴파일 시 문자열 리터럴을 암호화하고 런타임에 복호화하는 C++용 컴파일 타임 문자열 암호화 라이브러리입니다. constexpr 함수와 템플릿 메타프로그래밍을 사용해 컴파일 중 문자열을 XOR 인코딩하여, 정적 분석 도구가 바이너리에서 읽기 쉬운 문자열을 추출하지 못하도록 합니다. 헤더 전용 라이브러리로, 임의의 문자열 리터럴을 암호화할 수 있는 간단한 매크로 인터페이스를 제공합니다. 시그니처 스캐너와 정적 분석으로부터 문자열을 숨겨야 하는 치트 개발자와 소프트웨어 보호 연구자를 대상으로 합니다.
