Этот проект — skCrypter, библиотека шифрования строк на этапе компиляции для C++, которая шифрует строковые литералы при компиляции и расшифровывает их во время выполнения. Она использует constexpr-функции и шаблонное метапрограммирование для XOR-кодирования строк во время компиляции, не давая средствам статического анализа извлекать читаемые строки из бинарного файла. Header-only библиотека предоставляет простой макрос-интерфейс для шифрования любого строкового литерала. Она предназначена для разработчиков читов и исследователей защиты ПО, которым нужно скрывать строки от сигнатурных сканеров и статического анализа.
