Dieses Projekt ist skCrypter, eine Compile-Time-String-Verschlüsselungsbibliothek für C++, die String-Literale zur Kompilierzeit verschlüsselt und zur Laufzeit entschlüsselt. Sie nutzt constexpr-Funktionen und Template-Metaprogrammierung, um Strings während der Kompilierung mit XOR zu kodieren und so zu verhindern, dass Tools zur statischen Analyse lesbare Strings aus der Binärdatei extrahieren. Die Header-Only-Bibliothek bietet eine einfache Makro-Schnittstelle zum Verschlüsseln beliebiger String-Literale. Sie richtet sich an Cheat-Entwickler und Forscher im Bereich Softwareschutz, die Strings vor Signatur-Scannern und statischer Analyse verbergen müssen.
