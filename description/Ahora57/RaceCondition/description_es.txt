Este proyecto es una pequeña herramienta anti-anti-debug de tipo proof-of-concept que explora el comportamiento de elusión basado en condiciones de carrera. Está implementada en C++ para Visual Studio y usa las API nativas de Windows NT para sondear puertos de depuración, el comportamiento de hilos ocultos y artefactos relacionados del depurador. El ejemplo se centra en demostrar cómo el temporizado y las comprobaciones de estado pueden usarse contra mecanismos habituales de ocultación en escenarios de depuración en userland. Su público principal son investigadores de ingeniería inversa que experimentan con técnicas anti-debug y anti-anti-debug.
