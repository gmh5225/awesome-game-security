Questo progetto è un wrapper C++ dell’API Windows ReadDirectoryChangesW per monitorare in tempo reale le modifiche al file system. Fornisce un’interfaccia chiara per osservare alberi di directory e ricevere notifiche su creazione, modifica, eliminazione e rinomina di file. La libreria gestisce I/O asincrono, gestione dei buffer e monitoraggio ricorsivo delle directory. È pensata per sviluppatori Windows che creano file system watcher, strumenti di sincronizzazione o applicazioni di monitoraggio della sicurezza.
