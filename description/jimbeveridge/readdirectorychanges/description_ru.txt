Этот проект — C++-обёртка над Windows API ReadDirectoryChangesW для мониторинга изменений файловой системы в реальном времени. Он предоставляет понятный интерфейс для наблюдения за деревьями каталогов и получения уведомлений о создании, изменении, удалении и переименовании файлов. Библиотека обрабатывает асинхронный ввод-вывод, управление буферами и рекурсивный мониторинг каталогов. Предназначена для Windows-разработчиков, создающих наблюдатели файловой системы, инструменты синхронизации или приложения для мониторинга безопасности.
