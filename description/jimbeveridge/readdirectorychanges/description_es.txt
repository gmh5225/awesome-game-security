Este proyecto es un envoltorio en C++ sobre la API de Windows ReadDirectoryChangesW para monitorear cambios del sistema de archivos en tiempo real. Ofrece una interfaz clara para vigilar árboles de directorios y recibir notificaciones de creación, modificación, eliminación y renombrado de archivos. La biblioteca gestiona E/S asíncrona, administración de búferes y monitoreo recursivo de directorios. Está pensada para desarrolladores de Windows que crean observadores del sistema de archivos, herramientas de sincronización o aplicaciones de monitoreo de seguridad.
