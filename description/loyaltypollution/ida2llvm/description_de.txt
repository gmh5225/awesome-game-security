Dieses Projekt konzentriert sich auf iDA2LLVM – dynamisches Binary Lifting von IDA-Code nach LLVM IR.
Der Viewer synchronisiert sich stets mit dem Cursor des Benutzers im Disassembly-Fenster und zeigt an, ob der Code geliftet werden kann.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
