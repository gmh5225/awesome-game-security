Este proyecto se centra en iDA2LLVM: el levantamiento binario dinámico de código de IDA a LLVM IR.
El visor siempre se sincroniza con el cursor del usuario en la ventana de desensamblado, indicando si el código puede levantarse.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
