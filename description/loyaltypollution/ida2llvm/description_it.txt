Questo progetto si concentra su iDA2LLVM: il dynamic binary lifting del codice IDA in LLVM IR.
Il visualizzatore si sincronizza sempre con il cursore dell'utente nella finestra di disassemblaggio, indicando se il codice può essere sollevato.
È soprattutto utile per i ricercatori di sicurezza nei giochi e per i reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin IDA.
