Ce projet porte sur iDA2LLVM — le levage binaire dynamique du code IDA vers du LLVM IR.
La visionneuse se synchronise toujours avec le curseur de l'utilisateur dans la fenêtre de désassemblage, indiquant si le code peut être levé.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
