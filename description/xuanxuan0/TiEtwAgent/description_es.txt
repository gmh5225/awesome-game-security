Este proyecto es TiEtwAgent, un agente de detección de inyección de procesos basado en ETW que consume eventos Microsoft-Windows-Threat-Intelligence para obtener telemetría de inyección visible a nivel del kernel.
Utiliza krabsetw para la configuración de sesiones ETW, incluye lógica de detección asistida por Yara y está diseñado para ejecutarse como un servicio protegido con consideraciones ELAM/PPL.
Es especialmente útil para investigadores de anti-trampas y defensa que estudian la detección de inyección sin hooks frágiles en userland.
