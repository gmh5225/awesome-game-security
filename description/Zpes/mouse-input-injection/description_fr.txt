Ce projet démontre l'injection d'événements souris de bas niveau via l'appel système non documenté NtUserInjectMouseInput. Il est implémenté en C++ avec des structures d'entrée personnalisées et une petite couche d'interface pour les mouvements et les clics. Cette approche offre une alternative aux API de simulation d'entrée de plus haut niveau et met en lumière les mécanismes internes Windows liés aux chemins d'entrée. Elle est surtout utile pour la recherche en automatisation, les outils d'émulation d'entrée et l'expérimentation proche du domaine des cheats.
