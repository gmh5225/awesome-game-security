Este proyecto demuestra la inyección de eventos de ratón a bajo nivel mediante la llamada al sistema no documentada NtUserInjectMouseInput. Está implementado en C++ con estructuras de entrada personalizadas y una pequeña capa de interfaz para acciones de movimiento y clic. El enfoque ofrece una alternativa a las API de simulación de entrada de más alto nivel y pone de relieve los internos de Windows relevantes para las rutas de entrada. Es útil principalmente para investigación en automatización, herramientas de emulación de entrada y experimentación cercana a los cheats.
