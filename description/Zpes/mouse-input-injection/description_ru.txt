Этот проект демонстрирует низкоуровневую инъекцию событий мыши через недокументированный системный вызов NtUserInjectMouseInput. Он реализован на C++ с пользовательскими структурами ввода и небольшим интерфейсным слоем для перемещения и кликов. Подход служит альтернативой API симуляции ввода более высокого уровня и раскрывает внутренности Windows, связанные с путями ввода. В основном полезен для исследований автоматизации, инструментов эмуляции ввода и экспериментов, смежных с читерством.
