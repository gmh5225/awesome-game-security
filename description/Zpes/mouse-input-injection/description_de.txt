Dieses Projekt demonstriert die Low-Level-Injektion von Mausereignissen über den undokumentierten Syscall NtUserInjectMouseInput. Es ist in C++ mit benutzerdefinierten Eingabestrukturen und einer kleinen Schnittstellenschicht für Bewegungs- und Klickaktionen umgesetzt. Der Ansatz bietet eine Alternative zu höher gelegenen APIs zur Eingabesimulation und beleuchtet Windows-Internals, die für Eingabepfade relevant sind. Er eignet sich vor allem für Automatisierungsforschung, Tools zur Eingabeemulation und cheat-nahe Experimente.
