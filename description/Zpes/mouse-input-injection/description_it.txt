Questo progetto dimostra l'iniezione di eventi del mouse a basso livello tramite la syscall non documentata NtUserInjectMouseInput. È implementato in C++ con strutture di input personalizzate e un piccolo strato di interfaccia per movimenti e clic. L'approccio offre un'alternativa alle API di simulazione dell'input di livello superiore e mette in luce gli interni di Windows rilevanti per i percorsi di input. È utile soprattutto per la ricerca sull'automazione, gli strumenti di emulazione dell'input e la sperimentazione affine ai cheat.
