Плагин-загрузчик Binary Ninja для прошивочных бинарников AMD Secure Processor (SP) / Platform Security Processor (PSP), корректно задающий адреса загрузки для блобов AGESA Bootloader (ABL) и PSP bootloader.
Опционально аннотирует системные вызовы PSP с помощью прилагаемого словаря и рассчитан на работу с бинарниками, извлечёнными через PSPTool.
В основном полезен исследователям безопасности прошивок, занимающимся реверс-инжинирингом прошивки AMD PSP и анализом цепочки загрузчиков AMD Secure Processor.
