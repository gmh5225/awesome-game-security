AMD Secure Processor（SP）/ Platform Security Processor（PSP）ファームウェアバイナリ向けの Binary Ninja ローダープラグインで、AGESA Bootloader（ABL）および PSP ブートローダーblob のロードアドレスを正しく設定します。
付属の辞書を使って PSP システムコールに任意で注釈を付けられ、PSPTool で抽出したバイナリと連携するよう設計されています。
主に、AMD PSP ファームウェアをリバースエンジニアリングし、AMD Secure Processor のブートローダーチェーンを解析するファームウェアセキュリティ研究者向けです。
