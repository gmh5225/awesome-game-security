Un plugin chargeur Binary Ninja pour les binaires de firmware AMD Secure Processor (SP) / Platform Security Processor (PSP), qui configure correctement les adresses de chargement des blobs AGESA Bootloader (ABL) et du bootloader PSP.
Il peut optionnellement annoter les syscalls PSP à l’aide d’un dictionnaire fourni, et est conçu pour fonctionner avec les binaires extraits via PSPTool.
Il est surtout utile aux chercheurs en sécurité firmware qui reverse-engineerent le firmware AMD PSP et analysent la chaîne de bootloaders du AMD Secure Processor.
