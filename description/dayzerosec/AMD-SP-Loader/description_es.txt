Un complemento cargador de Binary Ninja para binarios de firmware del AMD Secure Processor (SP) / Platform Security Processor (PSP) que configura correctamente las direcciones de carga de los blobs del AGESA Bootloader (ABL) y del bootloader PSP.
Opcionalmente anota syscalls PSP mediante un diccionario incluido y está diseñado para trabajar con binarios extraídos con PSPTool.
Resulta especialmente útil para investigadores de seguridad de firmware que hacen ingeniería inversa del firmware AMD PSP y analizan la cadena de bootloaders del AMD Secure Processor.
