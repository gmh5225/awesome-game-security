Un plugin loader di Binary Ninja per i binari del firmware AMD Secure Processor (SP) / Platform Security Processor (PSP) che imposta correttamente gli indirizzi di caricamento per i blob di AGESA Bootloader (ABL) e del bootloader PSP.
Opzionalmente annota le syscall PSP usando un dizionario incluso ed è progettato per funzionare con binari estratti tramite PSPTool.
È soprattutto utile per i ricercatori di sicurezza del firmware che reverse-engineano il firmware AMD PSP e analizzano la catena di bootloader dell’AMD Secure Processor.
