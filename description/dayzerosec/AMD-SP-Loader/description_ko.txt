AMD Secure Processor(SP) / Platform Security Processor(PSP) 펌웨어 바이너리를 위한 Binary Ninja 로더 플러그인으로, AGESA Bootloader(ABL)와 PSP 부트로더 blob의 로드 주소를 올바르게 설정합니다.
포함된 사전을 사용해 PSP 시스템 콜에 선택적으로 주석을 달 수 있으며, PSPTool로 추출한 바이너리와 함께 쓰도록 설계되었습니다.
주로 AMD PSP 펌웨어를 리버스 엔지니어링하고 AMD Secure Processor 부트로더 체인을 분석하는 펌웨어 보안 연구자에게 유용합니다.
