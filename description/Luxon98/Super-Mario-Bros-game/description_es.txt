Este proyecto es un remake en C++ del clásico Super Mario Bros, con niveles inspirados en el original y niveles personalizados. Está construido con SDL2, SDL_image y SDL_mixer, y organizado como un proyecto de juego de Visual Studio con recursos de arte y audio. El código demuestra sistemas centrales de un plataformas 2D como el renderizado, el flujo de niveles y la lógica de juego en una implementación de escritorio independiente. El uso principal es el aprendizaje del desarrollo de juegos y la práctica con un motor basado en SDL, más que la investigación de seguridad.
