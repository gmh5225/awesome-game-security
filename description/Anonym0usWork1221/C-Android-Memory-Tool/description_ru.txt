Этот проект — набор инструментов на C++ для манипуляции памятью процессов Android на устройствах с root. Он предоставляет поиск, чтение и запись памяти процесса, сканирование смещений, управление результатами и заморозку значений через переиспользуемый классовый интерфейс. Реализация работает напрямую с отображениями процессов и файлами памяти, автоматизируя типичные рабочие процессы в стиле Cheat Engine. Предназначен для исследования памяти игр, отладки и экспериментов в контролируемых средах Android.
