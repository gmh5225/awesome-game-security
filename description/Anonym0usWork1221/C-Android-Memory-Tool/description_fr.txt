Ce projet est une boîte à outils C++ de manipulation mémoire pour les processus Android sur appareils rootés. Il fournit la recherche, la lecture et l’écriture en mémoire, le scan d’offsets, la gestion des résultats et le gel de valeurs via une interface de classes réutilisable. L’implémentation travaille directement avec les mappings de processus et les fichiers mémoire pour automatiser les flux de travail de type Cheat Engine. Il est destiné à la recherche, au débogage et à l’expérimentation sur la mémoire de jeux dans des environnements Android contrôlés.
