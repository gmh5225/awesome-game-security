Este proyecto es un kit de herramientas C++ para manipulación de memoria de procesos Android en dispositivos con root. Ofrece búsqueda, lectura y escritura de memoria de procesos, escaneo de offsets, gestión de resultados y control de congelación de valores mediante una interfaz de clases reutilizable. La implementación trabaja directamente con los mapeos de procesos y los archivos de memoria para automatizar flujos de trabajo al estilo Cheat Engine. Está pensado para investigación de memoria de juegos, depuración y experimentación en entornos Android controlados.
