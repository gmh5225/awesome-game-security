Dieses Projekt ist ein C++-Toolkit zur Speichermanipulation für Android-Prozesse auf gerooteten Geräten. Es bietet Prozessspeicher-Suche, Lese- und Schreiboperationen, Offset-Scanning, Ergebnisverwaltung und Wert-Freeze-Steuerung über eine wiederverwendbare Klassenschnittstelle. Die Implementierung arbeitet direkt mit Prozess-Mappings und Speicherdateien, um typische Cheat-Engine-ähnliche Arbeitsabläufe zu automatisieren. Es ist für Spiele-Speicherforschung, Debugging und Experimente in kontrollierten Android-Umgebungen gedacht.
