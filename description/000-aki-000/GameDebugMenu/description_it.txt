Questo progetto è un plugin di menu di debug per giochi Unreal Engine 5. Combina codice di modulo C++ con asset Blueprint/UI per fornire interfacce di debug in-game. Le capacità principali includono l'esecuzione di comandi console, la modifica di proprietà e funzioni di gameplay, la gestione della localizzazione e il salvataggio dei dati di stato del menu. È rivolto principalmente a sviluppatori di giochi e team QA che necessitano di strumenti di test e regolazione a runtime.
