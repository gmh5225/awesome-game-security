Этот проект — плагин отладочного меню для игр на Unreal Engine 5. Он сочетает код C++-модуля с ассетами Blueprint/UI, чтобы предоставить внутриигровые интерфейсы отладки. Ключевые возможности включают выполнение консольных команд, редактирование свойств и функций геймплея, работу с локализацией и сохранение данных состояния меню. В первую очередь он предназначен для разработчиков игр и QA-команд, которым нужны инструменты тестирования и настройки во время выполнения.
