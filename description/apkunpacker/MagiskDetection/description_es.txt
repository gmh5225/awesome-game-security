Este proyecto es una colección de tipo archivo de aplicaciones Android de prueba de concepto usadas para detectar root, Magisk y entornos de manipulación relacionados. Agrega muchas muestras APK y documenta comprobaciones como detección de Zygisk, detección de hooks e instrumentación, verificaciones de bootloader o ROM, y descubrimiento de aplicaciones root. El repositorio también proporciona hashes y referencias para ayudar a los analistas a comparar el comportamiento de las muestras entre herramientas. Está pensado principalmente para analistas de seguridad móvil e investigadores de anti-trampas que estudian técnicas reales de detección de root.
