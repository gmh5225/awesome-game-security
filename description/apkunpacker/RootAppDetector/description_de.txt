Dieses Projekt ist ein kleines Android-Proof-of-Concept, das bekannte Root-Verwaltungsanwendungen auf einem Gerät erkennt. Die App ist in Java geschrieben und prüft Ziel-Pakete und Activity-Paare, indem sie Starts versucht und Sicherheitsausnahmen auswertet. Sie wird als minimales Gradle-basiertes Android-Projekt mit einer einfachen UI zum erneuten Scannen und Anzeigen der Ergebnisse ausgeliefert. Sie ist nützlich für mobile Anti-Cheat- und Integritätsteams, die eine auf App-Präsenz basierende Root-Erkennung validieren.
