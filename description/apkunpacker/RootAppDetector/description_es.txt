Este proyecto es una pequeña prueba de concepto de Android que detecta aplicaciones conocidas de gestión de root en un dispositivo. La app está escrita en Java y comprueba pares de paquete y Activity de destino intentando lanzarlos e interpretando excepciones de seguridad. Se entrega como un proyecto Android mínimo basado en Gradle, con una UI sencilla para volver a escanear y mostrar los hallazgos. Es útil para equipos de anti-trampas móvil e integridad que validan la detección de root basada en la presencia de apps.
