Ce projet est une petite preuve de concept Android qui détecte les applications de gestion root connues sur un appareil. L'application est écrite en Java et vérifie des paires package/activité cibles en tentant de les lancer et en interprétant les exceptions de sécurité. Elle est livrée comme un projet Android minimal basé sur Gradle, avec une interface simple pour rescanner et afficher les résultats. Elle est utile aux équipes d'anti-triche mobile et d'intégrité qui valident une détection root fondée sur la présence d'applications.
