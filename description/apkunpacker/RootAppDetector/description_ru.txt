Этот проект — небольшое Android-доказательство концепции, которое обнаруживает известные приложения для управления root на устройстве. Приложение написано на Java и проверяет целевые пары package и Activity, пытаясь их запускать и интерпретируя исключения безопасности. Поставляется как минимальный Android-проект на Gradle с простым интерфейсом для повторного сканирования и отображения результатов. Полезно командам мобильного античита и целостности, проверяющим обнаружение root по наличию приложений.
