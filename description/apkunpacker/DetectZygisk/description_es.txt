Este proyecto es un detector de Android de tipo prueba de concepto para identificar comportamientos de inyección de procesos al estilo Zygisk. Implementa la lógica de detección en C++ y JNI creando un proceso hijo con fork, adjuntándose con ptrace y leyendo artefactos de PTRACE_GETEVENTMSG. El repositorio incluye un APK de ejemplo y registros de tiempo de ejecución que muestran cómo se pueden detectar varios forks de Zygisk, aunque las implementaciones base pueden diferir. Es útil principalmente para investigación de anti-trampas móvil y detección de root centrada en comprobaciones de integridad en tiempo de ejecución.
