Этот проект — proof-of-concept детектор для Android, предназначенный для выявления поведения процессной инъекции в стиле Zygisk. Логика обнаружения реализована на C++ и JNI: создаётся дочерний процесс через fork, выполняется attach через ptrace и читаются артефакты PTRACE_GETEVENTMSG. В репозитории есть пример APK и runtime-логи, показывающие, как можно обнаружить несколько форков Zygisk, при этом базовые реализации могут отличаться. В основном полезен для исследований мобильного античита и root-детекции с упором на проверки целостности во время выполнения.
