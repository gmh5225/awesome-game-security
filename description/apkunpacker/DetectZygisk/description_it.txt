Questo progetto è un detector Android di tipo proof-of-concept per identificare comportamenti di process injection in stile Zygisk. Implementa la logica di rilevamento in C++ e JNI creando un processo figlio con fork, collegandosi con ptrace e leggendo gli artefatti PTRACE_GETEVENTMSG. Il repository include un APK di esempio e log di runtime che mostrano come diversi fork Zygisk possano essere rilevati, mentre le implementazioni di baseline possono differire. È utile soprattutto per la ricerca su anti-cheat mobile e root detection incentrata sui controlli di integrità a runtime.
