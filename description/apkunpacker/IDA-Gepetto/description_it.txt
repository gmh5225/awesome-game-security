Questo progetto è un plugin per IDA Pro che usa LLM locali per spiegare le funzioni decompilate e suggerire nomi di variabili migliori. Scritto in Python per IDAPython, si integra nell'interfaccia dello pseudocodice con azioni del menu contestuale e scorciatoie da tastiera. Supporta backend di modelli configurabili e la localizzazione tramite le impostazioni del plugin e i file di traduzione. È pensato per gli ingegneri del reverse engineering che vogliono accelerare l'analisi di malware e i flussi di ricerca sulla sicurezza dei giochi in IDA.
