Этот проект — плагин для IDA Pro, который использует локальные LLM для объяснения декомпилированных функций и предложения лучших имён переменных. Написан на Python для IDAPython и интегрируется в интерфейс псевдокода через действия контекстного меню и горячие клавиши. Поддерживает настраиваемые бэкенды моделей и локализацию через настройки плагина и файлы перевода. Предназначен для инженеров обратной разработки, которые хотят ускорить анализ вредоносного ПО и исследование безопасности игр в IDA.
