Este proyecto es un complemento de IDA Pro que usa LLM locales para explicar funciones descompiladas y sugerir mejores nombres de variables. Escrito en Python para IDAPython, se integra en la interfaz de pseudocódigo con acciones del menú contextual y atajos de teclado. Admite backends de modelos configurables y localización mediante ajustes del complemento y archivos de traducción. Está dirigido a ingenieros inversos que quieren acelerar el análisis de malware y los flujos de investigación en seguridad de juegos dentro de IDA.
