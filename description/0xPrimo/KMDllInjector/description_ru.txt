Этот проект — фреймворк инъекции DLL в режиме ядра для Windows, нацеленный на процессы на ранних стадиях запуска. Он поддерживает срабатывание по колбэкам через уведомления о создании процесса или загрузке образа и демонстрирует хукинг процедур загрузчика ntdll с помощью позиционно-независимого шеллкода. Также включает путь инъекции на основе APC из режима ядра для раннего выполнения в пользовательском режиме. Кодовая база написана на C++ и используется для исследований продвинутой инъекции процессов в контролируемых тестах безопасности.
