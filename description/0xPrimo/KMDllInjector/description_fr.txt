Ce projet est un framework d'injection de DLL en mode noyau pour Windows, ciblant les processus durant les premières étapes de démarrage. Il prend en charge un déclenchement par callbacks via les notifications de création de processus ou de chargement d'image, et démontre le hooking des routines du chargeur ntdll avec du shellcode indépendant de la position. Il inclut également un chemin d'injection basé sur APC depuis le mode noyau pour un timing d'exécution anticipé en mode utilisateur. Le code source est écrit en C++ et sert à la recherche avancée sur l'injection de processus dans des tests de sécurité contrôlés.
