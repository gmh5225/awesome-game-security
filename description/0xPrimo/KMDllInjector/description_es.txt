Este proyecto es un framework de inyección de DLL en modo kernel para Windows que apunta a procesos durante las primeras etapas de arranque. Admite disparo basado en callbacks mediante notificaciones de creación de procesos o carga de imágenes, y demuestra el hooking de rutinas del cargador ntdll con shellcode independiente de la posición. También incluye una ruta de inyección basada en APC desde el modo kernel para un timing temprano de ejecución en modo usuario. El código está escrito en C++ y se usa para investigación avanzada de inyección de procesos en pruebas de seguridad controladas.
