Questo progetto è un framework di iniezione DLL in modalità kernel per Windows che mira ai processi durante le fasi iniziali di avvio. Supporta l'attivazione basata su callback tramite notifiche di creazione processo o di caricamento immagine, e dimostra l'hooking delle routine del loader ntdll con shellcode position-independent. Include anche un percorso di iniezione basato su APC dalla modalità kernel per un timing di esecuzione anticipato in modalità utente. Il codice è scritto in C++ ed è usato per la ricerca avanzata sull'iniezione di processi in test di sicurezza controllati.
