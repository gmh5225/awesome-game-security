Ce projet est une bibliothèque Android destinée à contourner les restrictions sur les API non SDK cachées d’Android 9 à Android 12.
Elle combine une intégration Java et Kotlin avec des composants natifs C et C++ construits via JNI et CMake.
Le code fournit un flux de dépendance empaqueté et un appel d’initialisation simple pouvant être placé au démarrage de l’application.
Elle est surtout utile pour la rétro-ingénierie Android, les tests de compatibilité et les workflows de recherche en sécurité qui nécessitent des interfaces de plateforme restreintes.
