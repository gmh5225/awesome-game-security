Este proyecto es una biblioteca de Android para eludir las restricciones de las API ocultas no SDK en Android 9 hasta Android 12.
Combina integración en Java y Kotlin con componentes nativos en C y C++ construidos mediante JNI y CMake.
El código ofrece un flujo de dependencia empaquetado y una llamada de inicialización sencilla que puede colocarse en el arranque de la aplicación.
Resulta especialmente útil para ingeniería inversa en Android, pruebas de compatibilidad y flujos de investigación de seguridad que necesitan interfaces de plataforma restringidas.
