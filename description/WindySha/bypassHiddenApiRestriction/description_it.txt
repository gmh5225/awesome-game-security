Questo progetto è una libreria Android per aggirare le restrizioni sulle API nascoste non SDK da Android 9 ad Android 12.
Combina l’integrazione con Java e Kotlin e componenti nativi in C e C++ costruiti tramite JNI e CMake.
Il codice fornisce un flusso di dipendenza confezionato e una semplice chiamata di inizializzazione che può essere inserita all’avvio dell’applicazione.
È utile soprattutto per reverse engineering Android, test di compatibilità e flussi di ricerca sulla sicurezza che richiedono interfacce di piattaforma con restrizioni.
