Этот проект — библиотека для Android, предназначенная для обхода ограничений скрытых non-SDK API на Android 9–12.
Она сочетает интеграцию на Java и Kotlin с нативными компонентами на C и C++, собираемыми через JNI и CMake.
Кодовая база предоставляет упакованный способ подключения зависимости и простой вызов инициализации, который можно разместить при запуске приложения.
В основном полезна для реверс-инжиниринга Android, тестов совместимости и рабочих процессов исследований безопасности, которым нужны ограниченные интерфейсы платформы.
