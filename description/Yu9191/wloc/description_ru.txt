Этот проект посвящён подмене сетевой геолокации Apple Wi‑Fi/сотовой сети (gs-loc WLOC) — MITM-патч protobuf через Surge/Quantumult X/Loon/Stash/Shadowrocket; онлайн-выбор точки + Shortcuts; GCJ-02→WGS84; без джейлбрейка; только indoor/Wi‑Fi-позиционирование.
Он в основном написан на JavaScript и сосредоточен на networking и моддинге.
Прежде всего полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / iOS-сети / геолокации.
