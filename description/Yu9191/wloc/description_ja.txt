本プロジェクトは、Apple の Wi‑Fi/セルラーネットワーク位置情報（gs-loc WLOC）の偽装に焦点を当てています——Surge/Quantumult X/Loon/Stash/Shadowrocket 経由の MITM protobuf パッチ、オンライン地点ピッカー + Shortcuts、GCJ-02→WGS84、ジェイルブレイク不要、屋内/Wi‑Fi 測位のみ。
主に JavaScript で書かれ、ネットワーキングとモッディングを中心としています。
チート / iOS ネットワーク / 位置情報分野の攻撃技術を研究するゲームセキュリティ研究者およびリバースエンジニアに特に有用です。
