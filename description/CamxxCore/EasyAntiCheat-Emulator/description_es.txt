Este proyecto es una DLL ligera de Windows que emula la interfaz local de EasyAntiCheat que esperan algunos juegos. Está desarrollado en C++ y ofrece exportaciones falsas del lado del cliente para que los binarios puedan continuar como si el módulo anticheat estuviera presente. El repositorio lo describe como un stub orientado a la depuración y no implementa una falsificación real del protocolo del lado del servidor. Está pensado principalmente para ingeniería inversa y pruebas controladas de compatibilidad con anticheat.
