Dieses Projekt ist eine schlanke Windows-DLL, die die lokale EasyAntiCheat-Schnittstelle emuliert, die einige Spiele erwarten. Es ist in C++ entwickelt und stellt gefälschte clientseitige Exporte bereit, damit Binaries so fortfahren können, als wäre das Anti-Cheat-Modul vorhanden. Das Repository beschreibt es als debug-orientierten Stub und implementiert kein echtes serverseitiges Protokoll-Spoofing. Es zielt vor allem auf Reverse Engineering und kontrollierte Anti-Cheat-Kompatibilitätstests ab.
