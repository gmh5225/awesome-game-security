Этот проект — лёгкая Windows DLL, эмулирующая локальный интерфейс EasyAntiCheat, ожидаемый некоторыми играми. Он написан на C++ и предоставляет поддельные клиентские экспорты, чтобы бинарники могли продолжать работу так, будто модуль античита присутствует. В репозитории он описан как отладочный stub и не реализует настоящее подделывание серверного протокола. В основном предназначен для реверс-инжиниринга и контролируемого тестирования совместимости с античитом.
