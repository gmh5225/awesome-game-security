Ce projet est une DLL Windows légère qui émule l'interface locale EasyAntiCheat attendue par certains jeux. Développé en C++, il fournit de fausses exportations côté client afin que les binaires puissent continuer comme si le module anti-triche était présent. Le dépôt le décrit comme un stub orienté débogage et n'implémente pas de véritable usurpation du protocole côté serveur. Il vise principalement l'ingénierie inverse et les tests de compatibilité anti-triche en environnement contrôlé.
