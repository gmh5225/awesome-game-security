TLAC — лёгкий открытый локальный античит для Linux, защищающий игры пользовательским сканированием с опциональным eBPF и ИИ-анализом. Написан в основном на Rust, с C для eBPF-зондов и модуля ядра, а также скриптами на Python для обучения ONNX-модели аномалий. Ключевые возможности: сканирование сигнатур памяти с шаблонами (wildcards), самопроверка целостности SHA256, баны по hardware ID и локальный IPC-сервер на Tokio, чтобы принуждение оставалось на устройстве без облачной зависимости. eBPF-трейспоинты отслеживают подозрительные open, exec, ptrace и clone, а поведенческое обнаружение аномалий нацелено на неизвестные читы. Ориентирован на сценарии безопасности Linux-игр, включая Steam Deck, для разработчиков и операторов, которым нужен прозрачный античит-стек под лицензией MIT.
