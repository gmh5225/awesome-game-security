TLAC 是一款輕量級、開源的 Linux 本機反作弊方案，透過使用者空間掃描以及可選的 eBPF 與 AI 分析來保護遊戲。主體以 Rust 撰寫，eBPF 探針與核心模組使用 C，並以 Python 腳本訓練 ONNX 異常偵測模型。核心能力包括萬用字元記憶體特徵掃描、SHA256 自我完整性檢查、硬體 ID 封鎖，以及基於 Tokio 的本機 IPC 伺服器，使執法留在裝置端、無需依賴雲端。eBPF tracepoint 監控可疑的 open、exec、ptrace 與 clone 活動，行為異常偵測旨在發現未知作弊。面向 Linux 遊戲安全場景（含 Steam Deck），供希望採用透明 MIT 授權反作弊技術堆疊的開發者與營運方使用。
