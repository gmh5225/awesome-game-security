TLAC 是一款轻量级、开源的 Linux 本地反作弊方案，通过用户空间扫描以及可选的 eBPF 与 AI 分析来保护游戏。主体以 Rust 编写，eBPF 探针与内核模块使用 C，并用 Python 脚本训练 ONNX 异常检测模型。核心能力包括通配符内存特征扫描、SHA256 自完整性校验、硬件 ID 封禁，以及基于 Tokio 的本地 IPC 服务器，使执法留在设备本地、无需依赖云端。eBPF tracepoint 监控可疑的 open、exec、ptrace 与 clone 活动，行为异常检测旨在发现未知作弊。面向 Linux 游戏安全场景（含 Steam Deck），供希望采用透明 MIT 许可反作弊技术栈的开发者与运营方使用。
