TLAC은 사용자 공간 스캔과 선택적 eBPF 및 AI 분석으로 게임을 보호하는 Linux용 경량 오픈소스 로컬 안티치트입니다. 주로 Rust로 작성되었으며, eBPF 프로브와 커널 모듈은 C로, ONNX 이상 탐지 모델 학습에는 Python 스크립트를 사용합니다. 주요 기능으로는 와일드카드 메모리 시그니처 스캔, SHA256 자체 무결성 검사, 하드웨어 ID 차단, 그리고 Tokio 기반 로컬 IPC 서버가 있어 집행이 클라우드 없이 기기에서 이루어집니다. eBPF 트레이스포인트는 의심스러운 open, exec, ptrace, clone 활동을 감시하고, 행동 이상 탐지는 알려지지 않은 치트를 잡는 것을 목표로 합니다. Steam Deck을 포함한 Linux 게임 보안 사례를 대상으로 하며, 투명한 MIT 라이선스 안티치트 스택을 원하는 개발자와 운영자에게 적합합니다.
