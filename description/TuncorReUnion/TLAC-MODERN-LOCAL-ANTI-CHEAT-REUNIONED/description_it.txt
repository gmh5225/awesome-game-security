TLAC è un anti-cheat locale open source leggero per Linux che protegge i giochi con scansione in user space più eBPF e analisi AI opzionali. È scritto principalmente in Rust, con C per le probe eBPF e un modulo kernel, e script Python per addestrare un modello di anomalie ONNX. Le capacità chiave includono scansione di signature di memoria con wildcard, controlli di auto-integrità SHA256, ban basati su hardware ID e un server IPC locale basato su Tokio così che l’enforcement resti sul dispositivo senza dipendenza dal cloud. I tracepoint eBPF osservano attività sospette di open, exec, ptrace e clone, mentre il rilevamento di anomalie comportamentali mira a intercettare cheat sconosciuti. Si rivolge a casi d’uso di sicurezza dei giochi Linux, incluso Steam Deck, per sviluppatori e operatori che vogliono uno stack anti-cheat trasparente con licenza MIT.
