TLAC is a lightweight, open-source local anti-cheat for Linux that protects games with user-space scanning plus optional eBPF and AI analysis. It is written mainly in Rust, with C for eBPF probes and a kernel module, and Python scripts for training an ONNX anomaly model. Key capabilities include wildcard memory signature scanning, SHA256 self-integrity checks, hardware ID bans, and a Tokio-based local IPC server so enforcement stays on-device without cloud dependency. eBPF tracepoints watch suspicious open, exec, ptrace, and clone activity, while behavioral anomaly detection aims to catch unknown cheats. It targets Linux game security use cases, including Steam Deck, for developers and operators who want a transparent MIT-licensed anti-cheat stack.
