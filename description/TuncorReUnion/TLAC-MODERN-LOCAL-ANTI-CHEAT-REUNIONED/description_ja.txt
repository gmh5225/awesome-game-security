TLAC は、ユーザ空間スキャンに加え、任意で eBPF と AI 解析を用いてゲームを保護する、Linux 向けの軽量オープンソース・ローカルアンチチートです。主に Rust で書かれ、eBPF プローブとカーネルモジュールは C、ONNX 異常検知モデルの学習には Python スクリプトを用います。主な機能は、ワイルドカードによるメモリ署名スキャン、SHA256 自己完全性チェック、ハードウェア ID による BAN、および Tokio ベースのローカル IPC サーバーで、執行はクラウドに依存せず端末内で完結します。eBPF のトレースポイントが不審な open、exec、ptrace、clone を監視し、行動異常検知は未知のチートの捕捉を目指します。Steam Deck を含む Linux ゲームセキュリティ用途向けで、透明な MIT ライセンスのアンチチートスタックを求める開発者・運用者に適しています。
