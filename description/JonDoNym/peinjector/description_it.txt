Questo progetto è un framework di patching e infezione PE che inietta payload personalizzati negli eseguibili Windows preservando il comportamento originale. Include un core in C per il parsing e la modifica PE, oltre a componenti di controllo e intercettazione in Python e Java. Le capacità principali comprendono più metodi di infezione, generazione di patch per la modifica al momento del trasferimento e configurazione remota via web. È destinato alla ricerca sugli eseguibili, alle simulazioni red team e all'analisi dei meccanismi difensivi legati alla manipolazione PE.
