Этот проект — фреймворк для патчинга и заражения PE, который внедряет пользовательские полезные нагрузки в исполняемые файлы Windows, сохраняя исходное поведение. Он включает ядро на C для разбора и изменения PE, а также компоненты управления и перехвата на Python и Java. Ключевые возможности: несколько методов заражения, генерация патчей для модификации на этапе передачи и веб-конфигурация удалённо. Предназначен для исследования исполняемых файлов, red-team симуляций и анализа защитных механизмов, связанных с подменой PE.
