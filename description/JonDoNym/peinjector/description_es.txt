Este proyecto es un framework de parcheo e infección PE que inyecta cargas útiles personalizadas en ejecutables de Windows preservando el comportamiento original. Incluye un núcleo en C para el análisis y la modificación de PE, además de componentes de control e interceptación en Python y Java. Sus capacidades clave incluyen múltiples métodos de infección, generación de parches para modificación en el momento de la transferencia y configuración remota basada en web. Está pensado para la investigación de ejecutables, simulaciones de red team y el análisis de mecanismos defensivos frente a la manipulación PE.
