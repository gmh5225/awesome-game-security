Этот проект посвящён гипервизору типа 1 для ARM64.
Открытый проект Gunyah Hypervisor предоставляет эталонную конфигурацию Type-1 гипервизора, подходящую для общего размещения нескольких доверенных и зависимых виртуальных машин.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / QEMU / KVM / PVE / VirtualBox.
