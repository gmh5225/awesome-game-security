Ce projet se concentre sur un hyperviseur de type 1 pour ARM64.
Le projet open source Gunyah Hypervisor fournit une configuration de référence d’hyperviseur Type-1 adaptée à l’hébergement général de plusieurs machines virtuelles de confiance et dépendantes.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / QEMU / KVM / PVE / VirtualBox.
