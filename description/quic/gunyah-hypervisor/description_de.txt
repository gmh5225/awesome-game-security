Dieses Projekt konzentriert sich auf einen Type-1-Hypervisor für ARM64.
Das Open-Source-Projekt Gunyah Hypervisor stellt eine Referenzkonfiguration eines Type-1-Hypervisors bereit, die sich für das allgemeine Hosting mehrerer vertrauenswürdiger und abhängiger VMs eignet.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / QEMU / KVM / PVE / VirtualBox untersuchen.
