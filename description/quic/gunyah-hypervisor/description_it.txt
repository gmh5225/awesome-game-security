Questo progetto si concentra su un hypervisor di tipo 1 per ARM64.
Il progetto open source Gunyah Hypervisor fornisce una configurazione di riferimento di hypervisor Type-1 adatta all’hosting di uso generale di più macchine virtuali fidate e dipendenti.
È soprattutto utile per ricercatori di sicurezza dei giochi e reverse engineer che studiano tecniche offensive nell’ambito di cheat / QEMU / KVM / PVE / VirtualBox.
