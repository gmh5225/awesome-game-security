Este proyecto se centra en un hipervisor de tipo 1 para ARM64.
El proyecto de código abierto Gunyah Hypervisor ofrece una configuración de referencia de hipervisor Type-1 adecuada para el alojamiento de propósito general de múltiples máquinas virtuales de confianza y dependientes.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / QEMU / KVM / PVE / VirtualBox.
