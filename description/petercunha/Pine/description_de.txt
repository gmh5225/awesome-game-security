Dieses Projekt konzentriert sich auf neuronale Netze.
Die Software kann so angepasst werden, dass sie in CS:GO, Fortnite und Overwatch reibungslos funktioniert.
Sie ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Triggerbots und Aimbots untersuchen.
