Ce projet est une architecture de cheat Windows externe pour un titre battle royale, composée d’un pilote noyau, d’un chargeur et d’une DLL cliente.
Il est implémenté en C++ avec des projets distincts pour la logique côté pilote, le chargement en mode utilisateur et les modules de fonctionnalités.
Le code inclut des visuels de type ESP et chams, des utilitaires de visée assistée par entrée, ainsi qu’une approche de contournement centrée sur le hooking des syscalls et l’exécution via des threads noyau.
Son cas d’usage principal est la rétro-ingénierie et la recherche anti-cheat sur les pipelines de cheat médiés par le noyau.
