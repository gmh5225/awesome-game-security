Este proyecto es un framework de cheat externo para Call of Duty: Warzone, construido en torno a un controlador de kernel mapeado manualmente.
Combina el acceso a memoria en modo usuario con un enfoque de hijacking de overlay para renderizar ESP y funciones relacionadas en pantalla.
La base de código es principalmente C++, con proyectos Visual Studio separados para cliente y controlador, más andamiaje de SDK del juego y offsets.
Está orientado a la investigación de game hacking y anti-cheat en herramientas externas asistidas por controlador.
