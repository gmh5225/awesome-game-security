Questo progetto è un framework di cheat esterno per Call of Duty: Warzone, costruito intorno a un driver kernel mappato manualmente.
Combina l’accesso alla memoria in user-mode con un approccio di hijacking dell’overlay per renderizzare ESP e funzionalità correlate a schermo.
Il codice è principalmente in C++, con progetti Visual Studio separati per client e driver, oltre a scaffolding di game SDK e offset.
È pensato per la ricerca su game hacking e anti-cheat relativi a tooling esterno assistito da driver.
