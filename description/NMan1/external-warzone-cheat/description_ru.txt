Этот проект — внешний чит-фреймворк для Call of Duty: Warzone, построенный вокруг вручную отображённого (manual map) драйвера ядра.
Он сочетает доступ к памяти из user-mode с перехватом оверлея для отрисовки ESP и связанных экранных функций.
Кодовая база в основном на C++: отдельные проекты Visual Studio для клиента и драйвера, а также каркас игрового SDK и смещений (offsets).
Предназначен для исследований в области game hacking и античита применительно к внешним инструментам с поддержкой драйвера.
