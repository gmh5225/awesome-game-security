Ce projet est un framework de cheat externe pour Call of Duty: Warzone, construit autour d’un pilote noyau mappé manuellement.
Il combine l’accès mémoire en mode utilisateur et une approche de détournement d’overlay pour afficher l’ESP et les fonctionnalités associées à l’écran.
Le code est principalement en C++, avec des projets Visual Studio distincts pour le client et le pilote, ainsi qu’une base SDK jeu et des offsets.
Il vise la recherche en game hacking et anti-cheat sur des outils externes assistés par pilote.
