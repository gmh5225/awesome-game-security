Este proyecto es un framework de cheat para Windows con código fuente completo que combina un controlador de kernel, un menú externo y comunicación por memoria compartida.
Está implementado en gran parte en C y C++ e incluye lógica de cargador, componentes de renderizado y módulos de manipulación del gameplay.
Las capacidades documentadas cubren visuales ESP y chams, silent-aim y controles de retroceso o dispersión, modificaciones de desbloqueo y movimiento, y ajustes configurables.
Su caso de uso principal es la investigación de game hacking y el análisis de diseños de cheats asistidos por kernel desde una perspectiva anti-cheat.
