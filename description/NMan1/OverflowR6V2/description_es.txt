Este proyecto es un framework de trampas externo de segunda generación para un shooter de Windows, acompañado de un bypass con controlador en modo kernel.
Está escrito en C y C++ y utiliza una estrategia de hook inline de funciones del kernel, con un menú y componentes de renderizado en modo usuario.
El código incluye funciones de combate y visuales como aimbot, chams, disparo rápido, ediciones de retroceso y dispersión, y controles de velocidad o FOV.
Su caso de uso principal es estudiar la arquitectura de trampas asistidas por kernel y las compensaciones de detección anti-trampas.
