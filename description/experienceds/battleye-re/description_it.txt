Questo progetto è un riferimento indipendente di reverse engineering del driver anti-cheat in modalità kernel di BattlEye, BEDaisy.sys, incentrato sulla ricerca di sicurezza a scopo educativo piuttosto che sulla fornitura di codice di bypass o cheat. Documenta il layout delle sezioni PE, la risoluzione dinamica di dozzine di API del kernel, il dispatch degli IOCTL sul dispositivo BattlEye, la verifica della tabella HAL, il comportamento anti-DMA, una regione di offuscamento VM personalizzata e la derivazione del security cookie. I materiali allegati includono risultati JSON strutturati, elenchi di indirizzi, estratti di disassembly e dump esadecimali delle sezioni dati chiave. Il pubblico principale è costituito da ricercatori di sicurezza dei giochi e anti-cheat che studiano come un anti-cheat kernel moderno protegge titoli come Enlisted.
