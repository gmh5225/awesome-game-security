이 프로젝트는 BattlEye의 커널 모드 안티치트 드라이버 BEDaisy.sys에 대한 독립적인 리버스 엔지니어링 참고 자료로, 우회나 치트 코드를 제공하는 것이 아니라 교육적 보안 연구에 중점을 둡니다. PE 섹션 레이아웃, 수십 개의 커널 API 동적 해석, BattlEye 장치의 IOCTL 디스패치, HAL 테이블 검증, 안티 DMA 동작, 커스텀 VM 난독화 영역, 보안 쿠키 도출 등을 문서화합니다. 부속 자료로는 구조화된 JSON 조사 결과, 주소 목록, 디스어셈블리 발췌, 주요 데이터 섹션의 헥스 덤프가 포함됩니다. 주요 독자는 Enlisted와 같은 타이틀을 보호하는 현대 커널 안티치트의 동작 방식을 연구하는 게임 보안 및 안티치트 연구자입니다.
