本プロジェクトは、BattlEye のカーネルモードアンチチートドライバー BEDaisy.sys に関する独立したリバースエンジニアリング参考資料であり、バイパスやチートコードの提供ではなく、教育的なセキュリティ研究を目的としています。PE セクション構成、数十のカーネル API の動的解決、BattlEye デバイス上の IOCTL ディスパッチ、HAL テーブル検証、アンチ DMA 挙動、独自の VM 難読化領域、およびセキュリティ Cookie の導出などを文書化しています。付属資料には、構造化された JSON の調査結果、アドレス一覧、逆アセンブル抜粋、主要データセクションのヘックスダンプが含まれます。主な対象読者は、Enlisted のようなタイトルを保護する現代のカーネルアンチチートの仕組みを研究するゲームセキュリティおよびアンチチート研究者です。
