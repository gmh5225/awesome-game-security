Este proyecto es un código fuente básico de cheat interno de Fortnite en C++ que demuestra la funcionalidad central de cheats para juegos basados en Unreal Engine 4. Incluye ESP de jugadores, aimbot y modificaciones visuales básicas, implementadas mediante interacción con el SDK de UE4, enumeración de GObject/GNames y hooking de funciones del motor. El código muestra la estructura típica de un cheat interno de UE4 con hooks a nivel de proceso. Está dirigido a investigadores de seguridad en juegos que estudian la arquitectura de cheats de Fortnite y los patrones de detección de cheats basados en UE4.
