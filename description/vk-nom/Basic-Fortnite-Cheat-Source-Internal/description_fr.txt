Ce projet est une source basique de cheat interne Fortnite en C++ qui présente les fonctionnalités essentielles de cheat pour les jeux basés sur Unreal Engine 4. Il inclut l’ESP joueur, un aimbot et des modifications visuelles de base, mis en œuvre via l’interaction avec le SDK UE4, l’énumération GObject/GNames et le hooking de fonctions du moteur. Le code illustre la structure typique d’un cheat interne UE4 avec des hooks à l’échelle du processus. Il s’adresse aux chercheurs en sécurité des jeux qui étudient l’architecture des cheats Fortnite et les modèles de détection de cheats basés sur UE4.
