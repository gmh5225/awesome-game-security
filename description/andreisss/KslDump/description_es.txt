Este proyecto extrae credenciales de LSASS protegido por PPL usando únicamente componentes firmados por Microsoft ya presentes en el sistema. Explota una versión vulnerable del controlador KslD.sys de Microsoft Defender que expone el IOCTL 0x222044 con subcomandos para lectura arbitraria de memoria del kernel mediante MmCopyMemory y la revelación de registros de control de la CPU para derrotar KASLR. La cadena de ataque no requiere cargar controladores externos, ya que redirige el servicio al binario antiguo sin parchear del controlador que Microsoft dejó en el disco. Resulta especialmente útil para investigadores de seguridad que estudian ataques de tipo BYOVD con controladores vulnerables suministrados por el fabricante y técnicas de evasión de PPL.
