Этот проект извлекает учётные данные из LSASS, защищённого PPL, используя только уже присутствующие в системе компоненты с подписью Microsoft. Он эксплуатирует уязвимую версию драйвера KslD.sys из Microsoft Defender, который предоставляет IOCTL 0x222044 с подкомандами для произвольного чтения памяти ядра через MmCopyMemory и раскрытия управляющих регистров ЦП для обхода KASLR. Цепочка атаки не требует загрузки внешнего драйвера: служба перенаправляется на старый незапатченный двоичный файл драйвера, оставленный Microsoft на диске. Проект в основном полезен исследователям безопасности, изучающим атаки в стиле BYOVD с уязвимыми драйверами от вендора и техники обхода PPL.
