이 프로젝트는 시스템에 이미 존재하는 Microsoft 서명 구성 요소만으로 PPL로 보호된 LSASS에서 자격 증명을 추출합니다. Microsoft Defender의 취약한 KslD.sys 드라이버 버전을 악용하며, 해당 드라이버는 IOCTL 0x222044와 하위 명령을 통해 MmCopyMemory로 임의 커널 메모리 읽기를 수행하고 KASLR을 무력화하기 위해 CPU 제어 레지스터를 노출합니다. 공격 체인은 외부 드라이버 로드가 필요 없으며, 서비스를 Microsoft가 디스크에 남겨 둔 이전의 패치되지 않은 드라이버 바이너리로 다시 가리킵니다. 주로 벤더가 배포한 취약 드라이버를 이용한 BYOVD 스타일 공격과 PPL 우회 기법을 연구하는 보안 연구자에게 유용합니다.
