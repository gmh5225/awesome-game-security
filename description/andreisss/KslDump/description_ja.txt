本プロジェクトは、システム上に既にある Microsoft 署名コンポーネントのみを用い、PPL 保護下の LSASS から資格情報を抽出します。Microsoft Defender の脆弱なバージョンの KslD.sys ドライバを悪用し、IOCTL 0x222044 とそのサブコマンドにより MmCopyMemory を通じた任意のカーネルメモリ読み取りと、KASLR 突破のための CPU 制御レジスタ開示を行います。攻撃チェーンは外部ドライバの読み込みを必要とせず、サービスを Microsoft がディスク上に残した古い未パッチのドライババイナリへ向け直します。主に、ベンダー同梱の脆弱ドライバを用いた BYOVD 型攻撃や PPL バイパス手法を研究するセキュリティ研究者向けです。
