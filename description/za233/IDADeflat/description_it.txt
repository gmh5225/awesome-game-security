Questo progetto è un plugin di IDA Pro per la rimozione semi-automatica dell'offuscamento mediante control flow flattening. Il plugin basato su Python usa angr come backend di esecuzione simbolica per identificare i basic block reali, recuperare il grafo del flusso di controllo originale e patchare la funzione appiattita nella sua forma deoffuscata. È soprattutto utile per reverse engineer e analisti di malware che affrontano il control flow flattening in stile OLLVM in binari protetti.
