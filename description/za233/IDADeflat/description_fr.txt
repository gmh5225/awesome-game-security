Ce projet est un plugin IDA Pro pour la suppression semi-automatisée de l'obfuscation par aplatissement du flot de contrôle. Le plugin basé sur Python utilise angr comme backend d'exécution symbolique afin d'identifier les vrais blocs de base, de reconstruire le graphe de flot de contrôle d'origine et de corriger la fonction aplatie pour retrouver sa forme désobfusquée. Il s'adresse principalement aux ingénieurs en rétro-ingénierie et aux analystes de logiciels malveillants confrontés à l'aplatissement du flot de contrôle de type OLLVM dans des binaires protégés.
