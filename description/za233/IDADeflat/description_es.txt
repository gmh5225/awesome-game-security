Este proyecto es un complemento de IDA Pro para la eliminación semiautomatizada de la ofuscación por aplanamiento del flujo de control. El complemento basado en Python usa angr como backend de ejecución simbólica para identificar bloques básicos reales, recuperar el grafo de flujo de control original y parchear la función aplanada a su forma desofuscada. Resulta especialmente útil para ingenieros de ingeniería inversa y analistas de malware que enfrentan aplanamiento del flujo de control al estilo OLLVM en binarios protegidos.
