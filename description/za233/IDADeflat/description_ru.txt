Этот проект — плагин для IDA Pro, предназначенный для полуавтоматического снятия обфускации методом выравнивания потока управления (control flow flattening). Плагин на Python использует angr в качестве бэкенда символьного выполнения, чтобы выявлять настоящие базовые блоки, восстанавливать исходный граф потока управления и патчить выровненную функцию до деобфусцированного вида. Он в первую очередь полезен реверс-инженерам и аналитикам вредоносного ПО, работающим с control flow flattening в стиле OLLVM в защищённых бинарных файлах.
