Этот проект — Windows proof of concept, который отслеживает и защищает Export Address Table (EAT) загруженных модулей от изменений во время выполнения. Он использует VEH (Vectored Exception Handling), чтобы установить page guard на страницах памяти EAT, обнаруживая и журналируя любые попытки перезаписи указателей экспортируемых функций. Реализация на C показывает, как защитники могут обнаруживать EAT hooking — технику, широко используемую руткитами и читами. Проект предназначен для исследователей безопасности, изучающих мониторинг целостности EAT и обнаружение хуков.
