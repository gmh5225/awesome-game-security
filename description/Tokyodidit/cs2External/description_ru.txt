Этот проект — внешняя реализация ESP для Counter-Strike 2. Он написан на C++ и использует чтение памяти процесса, настраиваемые JSON-смещения и преобразования world-to-screen для отслеживания внутриигровых сущностей. Отрисовка выполняется через прозрачный оверлей Win32/GDI, который рисует рамки, полоски здоровья и имена игроков. Основное применение — прототипирование читов и практический анализ игровой памяти.
