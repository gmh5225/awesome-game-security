Questo progetto è un'implementazione ESP esterna per Counter-Strike 2. È scritto in C++ e usa letture della memoria di processo, offset JSON configurabili e calcoli world-to-screen per tracciare le entità di gioco. Il rendering avviene tramite un overlay Win32/GDI trasparente che disegna riquadri, barre della vita e nomi dei giocatori. Il caso d'uso principale è la prototipazione di cheat e l'analisi pratica della memoria di gioco.
