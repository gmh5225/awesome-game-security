Este proyecto es una implementación ESP externa para Counter-Strike 2. Está escrito en C++ y utiliza lecturas de memoria de proceso, offsets JSON configurables y matemáticas world-to-screen para rastrear entidades del juego. El renderizado se realiza mediante una superposición transparente Win32/GDI que dibuja cajas, barras de vida y nombres de jugadores. Su caso de uso principal es el prototipado de cheats y el análisis práctico de memoria de juegos.
