Dieses Projekt ist eine externe ESP-Implementierung für Counter-Strike 2. Es ist in C++ geschrieben und nutzt Prozess-Speicherzugriffe, konfigurierbare JSON-Offsets sowie World-to-Screen-Berechnungen, um In-Game-Entitäten zu verfolgen. Das Rendering erfolgt über ein transparentes Win32/GDI-Overlay, das Boxen, Lebensbalken und Spielernamen zeichnet. Der Hauptanwendungsfall ist Cheat-Prototyping und praktische Spielspeicheranalyse.
