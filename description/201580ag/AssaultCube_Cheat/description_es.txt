Este proyecto es una colección de código de práctica de cheats para AssaultCube que cubre varias versiones del juego.
Incluye implementaciones internas y externas con funciones habituales de cheats FPS como aimbot, ESP, overlays, utilidades de memoria y ejemplos de manejo de offsets.
La base de código combina C/C++, C# y scripts de apoyo, y usa herramientas como ImGui, hooks de OpenGL y flujos de trabajo de edición de memoria.
Está dirigido principalmente a quienes quieren experiencia práctica con técnicas de game hacking e ingeniería inversa en un contexto de entrenamiento controlado.
