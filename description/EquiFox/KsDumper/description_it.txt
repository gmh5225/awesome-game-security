Questo progetto è un dumper di processi assistito dal kernel Windows pensato per applicazioni protette con handle in user mode limitati.
Combina un driver personalizzato e un client in user mode per copiare il modulo principale di un processo dalla memoria e ricostruire header e sezioni PE32 o PE64.
L'implementazione è scritta in C++ e documenta dettagli pratici del workflow per il caricamento del driver, la generazione dei dump e la gestione di target protetti da anti-cheat.
Il caso d'uso principale è la ricerca di reverse engineering a basso livello focalizzata su binari di giochi protetti e pattern di interazione kernel-user.
