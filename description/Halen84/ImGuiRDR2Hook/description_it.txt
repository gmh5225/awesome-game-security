ImGuiRDR2Hook è un framework di hook per Red Dead Redemption 2 pensato per renderizzare menu ImGui personalizzati tramite Vulkan e DirectX 12. È implementato in C++ e integra componenti di hooking comuni come MinHook, Kiero e Dear ImGui per intercettare i percorsi di rendering. Il codice fornisce punti di ingresso degli hook, routine di render, gestione della configurazione e note pratiche per un disegno stabile dei menu e un comportamento di input affidabile. Viene usato soprattutto per lo sviluppo di overlay di gioco, la pratica di hooking delle API grafiche e la sperimentazione sulla sicurezza dei giochi.
