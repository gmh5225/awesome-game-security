ImGuiRDR2Hook es un framework de hooks para Red Dead Redemption 2 que permite renderizar menús ImGui personalizados mediante Vulkan y DirectX 12. Está implementado en C++ e integra componentes de hooking habituales como MinHook, Kiero y Dear ImGui para interceptar las rutas de renderizado. El código ofrece puntos de entrada de hooks, rutinas de render, manejo de configuración y notas prácticas para un dibujado estable del menú y un comportamiento de entrada fiable. Se usa principalmente para el desarrollo de overlays de juego, la práctica de hooking de APIs gráficas y la experimentación en seguridad de juegos.
