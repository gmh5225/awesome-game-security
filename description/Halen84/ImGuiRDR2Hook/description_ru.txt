ImGuiRDR2Hook — это hook-фреймворк для Red Dead Redemption 2, предназначенный для отрисовки пользовательских меню ImGui через Vulkan и DirectX 12. Он реализован на C++ и интегрирует распространённые компоненты хукинга — MinHook, Kiero и Dear ImGui — для перехвата путей рендеринга. Кодовая база предоставляет точки входа хуков, процедуры отрисовки, обработку конфигурации и практические заметки по стабильной отрисовке меню и поведению ввода. В основном используется для разработки игровых оверлеев, практики хукинга графических API и экспериментов в области игровой безопасности.
