Этот проект — официальный исходный код Unreal Engine и документация API от Epic Games. Он охватывает полный runtime движка, включая системы сборки, рендеринг, физику, сеть, абстракцию платформ, архитектуру плагинов и API скриптинга на Python. Это основной справочник для разработчиков игр и исследователей движка, работающих с внутренним устройством Unreal Engine, разработкой модулей и платформо-зависимой интеграцией.
