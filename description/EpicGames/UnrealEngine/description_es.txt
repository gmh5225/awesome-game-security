Este proyecto es el código fuente oficial de Unreal Engine y la documentación de la API de Epic Games. Cubre el runtime completo del motor, incluidos los sistemas de compilación, renderizado, física, redes, abstracción de plataforma, arquitectura de plugins y la API de scripting de Python. Es la referencia principal para desarrolladores de juegos e investigadores del motor que trabajan con los internos de Unreal Engine, el desarrollo de módulos y la integración específica de cada plataforma.
