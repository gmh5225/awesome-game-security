Ce projet est le code source officiel d'Unreal Engine et la documentation de l'API fournis par Epic Games. Il couvre l'ensemble du runtime du moteur, notamment les systèmes de build, le rendu, la physique, le réseau, l'abstraction de plateforme, l'architecture des plugins et l'API de scripting Python. C'est la référence principale pour les développeurs de jeux et les chercheurs sur le moteur qui travaillent sur les internals d'Unreal Engine, le développement de modules et l'intégration spécifique aux plateformes.
