Questo progetto è il codice sorgente ufficiale di Unreal Engine e la documentazione delle API di Epic Games. Copre l'intero runtime del motore, inclusi sistemi di build, rendering, fisica, networking, astrazione di piattaforma, architettura dei plugin e l'API di scripting Python. È il riferimento principale per sviluppatori di giochi e ricercatori sul motore che lavorano con gli interni di Unreal Engine, lo sviluppo di moduli e l'integrazione specifica per piattaforma.
