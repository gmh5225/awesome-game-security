Этот проект посвящён runtime-дампу il2Cpp.
Как это работает: метод сильно опирается на систему рефлексии il2cpp и использует вызовы il2cpp для восстановления DLL.
Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования игровых движков: Unity.
