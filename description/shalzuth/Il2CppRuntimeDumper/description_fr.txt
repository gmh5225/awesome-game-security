Ce projet se concentre sur le dump runtime d'il2Cpp.
Fonctionnement : la méthode s'appuie fortement sur le système de réflexion il2cpp et utilise des appels il2cpp pour reconstruire les DLL.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / exploration de moteurs de jeu : Unity.
