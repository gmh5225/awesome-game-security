Este repositorio es un archivo del código fuente del runtime de Unity IL2CPP a lo largo de varios rangos de generaciones de Unity.
Contiene numerosos archivos de implementación en C y C++ de los internos de libil2cpp, incluidos servicios de runtime, manejo de metadatos, recolección de basura y capas de abstracción de plataforma.
Las notas de correspondencia de versiones incluidas ayudan a correlacionar las revisiones de IL2CPP con las versiones correspondientes del motor Unity.
Se usa principalmente por ingenieros inversos e investigadores de seguridad en juegos que necesitan referencias internas para el análisis de código nativo de Unity y el desarrollo de herramientas.
