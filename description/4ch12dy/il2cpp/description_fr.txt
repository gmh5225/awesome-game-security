Ce dépôt est une archive du code source du runtime Unity IL2CPP couvrant plusieurs générations de Unity.
Il contient de nombreux fichiers d’implémentation en C et C++ des internes de libil2cpp, notamment les services runtime, la gestion des métadonnées, le garbage collection et les couches d’abstraction de plateforme.
Les notes de correspondance de versions incluses aident à relier les révisions IL2CPP aux versions correspondantes du moteur Unity.
Il est principalement utilisé par les reverse engineers et les chercheurs en sécurité des jeux qui ont besoin d’une référence interne pour l’analyse du code natif Unity et le développement d’outils.
