Этот репозиторий — архив исходного кода runtime Unity IL2CPP для нескольких поколений Unity.
В нём содержатся обширные файлы реализации на C и C++ внутренностей libil2cpp, включая runtime-сервисы, обработку метаданных, сборку мусора и слои платформенной абстракции.
Приложенные заметки по соответствию версий помогают сопоставить ревизии IL2CPP с соответствующими выпусками движка Unity.
Им в основном пользуются инженеры по реверс-инжинирингу и исследователи безопасности игр, которым нужны внутренние справочные материалы для анализа нативного кода Unity и разработки инструментов.
