Questo repository è un archivio del codice sorgente del runtime Unity IL2CPP su più intervalli di generazioni di Unity.
Contiene numerosi file di implementazione in C e C++ degli interni di libil2cpp, inclusi servizi runtime, gestione dei metadati, garbage collection e livelli di astrazione della piattaforma.
Le note di mappatura delle versioni incluse aiutano a correlare le revisioni di IL2CPP con le corrispondenti release del motore Unity.
È usato principalmente da reverse engineer e ricercatori di sicurezza dei giochi che necessitano di riferimenti interni per l’analisi del codice nativo Unity e lo sviluppo di strumenti.
