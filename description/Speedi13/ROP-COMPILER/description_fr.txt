Ce projet est un compilateur de programmation orientée retour (ROP) utilisé pour construire des payloads de triche pour CSGO, Battlefield 3 et Battlefield 4.
Il prend des scripts personnalisés de type assembleur x86 et les traduit en chaînes ROP, avec le scan de gadgets et la gestion des offsets implémentés en C++.
La base de code inclut des exemples pratiques tels que triggerbot, glow ESP et la logique de spotting sur la minimap.
Son cas d'usage principal est la recherche en sécurité des jeux sur l'exécution de triches de type exploit et les techniques d'évasion anti-cheat.
