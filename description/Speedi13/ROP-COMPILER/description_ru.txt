Этот проект — компилятор return-oriented programming (ROP), используемый для создания игровых чит-пейлоадов для CSGO, Battlefield 3 и Battlefield 4.
Он принимает пользовательские скрипты в стиле x86-ассемблера и переводит их в ROP-цепочки; сканирование гаджетов и обработка смещений реализованы на C++.
В кодовой базе есть практические примеры: triggerbot, glow ESP и логика spotting на миникарте.
Основной сценарий использования — исследование игровой безопасности, связанное с exploit-стилем выполнения читов и техниками обхода античита.
