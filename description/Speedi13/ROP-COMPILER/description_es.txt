Este proyecto es un compilador de programación orientada al retorno (ROP) usado para construir payloads de trampas para CSGO, Battlefield 3 y Battlefield 4.
Toma scripts personalizados similares a ensamblador x86 y los traduce en cadenas ROP, con el escaneo de gadgets y el manejo de offsets implementados en C++.
El código incluye ejemplos prácticos como triggerbot, glow ESP y lógica de spotting en el minimapa.
Su caso de uso principal es la investigación en seguridad de juegos sobre ejecución de trampas tipo exploit y técnicas de evasión anti-cheat.
