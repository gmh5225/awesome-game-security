该项目是一套面向游戏环境的 Windows 扫描套件，用于检测常见的恶意行为。主体以 C++ 编写，并包含部分 C 语言内核驱动，构建为 Visual Studio Win32/x64 控制台应用；它通过已文档化与未文档化的 Native WinAPI 例程，在受限检查下审查进程。功能涵盖 USN 与 DNS 扫描、PE 与 PEB 解析、模块与句柄枚举、线程检查、驱动与系统模块扫描、模式匹配，以及用户态 IAT Hook 检测，还有面向内核的 MSR、IDT、SSDT 与 IRP 处理程序 Hook 检测。面向游戏安全与反作弊研究，用于在 Windows 上原型化客户端完整性校验与 Hook 检测。
