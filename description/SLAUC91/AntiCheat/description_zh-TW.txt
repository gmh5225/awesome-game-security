此專案是一套面向遊戲環境的 Windows 掃描套件，用於偵測常見的惡意行為。主體以 C++ 撰寫，並包含部分 C 語言核心驅動程式，建置為 Visual Studio Win32/x64 主控台應用程式；它透過已文件化與未文件化的 Native WinAPI 常式，在受限檢查下檢視行程。功能涵蓋 USN 與 DNS 掃描、PE 與 PEB 解析、模組與控制代碼列舉、執行緒檢查、驅動程式與系統模組掃描、模式比對，以及使用者模式 IAT Hook 偵測，還有面向核心的 MSR、IDT、SSDT 與 IRP 處理常式 Hook 偵測。面向遊戲安全與反作弊研究，用於在 Windows 上原型化用戶端完整性檢查與 Hook 偵測。
