本プロジェクトは、ゲーム環境でよく見られる悪意のある活動を検出するための Windows スキャニングスイートです。主に C++ で書かれ、一部 C のカーネルドライバを含み、Visual Studio の Win32/x64 コンソールアプリケーションとしてビルドされます。文書化された／未文書化の Native WinAPI ルーチンを用い、制約付きのチェックの下でプロセスを検査します。機能は USN および DNS スキャン、PE／PEB 解析、モジュールとハンドルの列挙、スレッド検査、ドライバとシステムモジュールのスキャン、パターンマッチング、ユーザモードの IAT フック検出、ならびにカーネル向けの MSR、IDT、SSDT、IRP ハンドラフック検出をカバーします。ゲームセキュリティおよびアンチチート研究向けで、Windows 上でのクライアント側整合性チェックとフック検出のプロトタイピングを目的とします。
