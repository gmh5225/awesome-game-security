This project is a Windows scanning suite designed to detect malicious activity commonly seen in gaming environments. Written mainly in C++ with a partial C kernel driver and built as a Visual Studio Win32/x64 console application, it uses documented and undocumented Native WinAPI routines to inspect processes under constrained checks. Its feature set covers USN and DNS scanning, PE and PEB parsing, module and handle enumeration, thread inspection, driver and system-module scanning, pattern matching, and detection of user-mode IAT hooks plus kernel-oriented MSR, IDT, SSDT, and IRP handler hooks. It is aimed at game security and anti-cheat research, prototyping client-side integrity and hook detection on Windows.
