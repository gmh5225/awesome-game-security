Este proyecto es una suite de escaneo para Windows diseñada para detectar actividad maliciosa habitual en entornos de juego. Escrito principalmente en C++ con un controlador de kernel parcial en C y construido como aplicación de consola Visual Studio Win32/x64, usa rutinas Native WinAPI documentadas y no documentadas para inspeccionar procesos bajo comprobaciones restringidas. Su conjunto de funciones cubre el escaneo USN y DNS, el análisis PE y PEB, la enumeración de módulos y handles, la inspección de hilos, el escaneo de controladores y módulos del sistema, el emparejamiento de patrones y la detección de hooks IAT en modo usuario, además de hooks orientados al kernel en MSR, IDT, SSDT y manejadores IRP. Está orientado a la investigación de seguridad en juegos y anti-cheat, para prototipar la integridad del lado del cliente y la detección de hooks en Windows.
