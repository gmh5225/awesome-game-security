Este proyecto se centra en Awesome Game Engine Development.
Lista curada de recursos relacionados con el desarrollo de motores de juegos (herramientas que mejoran y aceleran la creación de juegos).
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores de gráficos que trabajan con DirectX y OpenGL en el ámbito de motores de juego / guías.
