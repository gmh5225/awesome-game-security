Dieses Projekt konzentriert sich auf Awesome Game Engine Development.
Kuratierte Liste von Ressourcen zur Entwicklung von Spiele-Engines (Werkzeuge, die die Spieleerstellung verbessern und beschleunigen).
Es ist vor allem nützlich für Spieleentwickler, Engine-Programmierer und Grafikforscher, die im Bereich Spiele-Engine / Guides mit DirectX und OpenGL arbeiten.
