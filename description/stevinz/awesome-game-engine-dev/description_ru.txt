Этот проект посвящён Awesome Game Engine Development.
Курируемый список ресурсов по разработке игровых движков (инструменты, которые улучшают и ускоряют создание игр).
Он прежде всего полезен разработчикам игр, программистам движков и исследователям графики, работающим с DirectX и OpenGL в области игровых движков / руководств.
