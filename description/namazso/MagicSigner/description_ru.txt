Этот проект сосредоточен на подписи с использованием утёкших сертификатов. Однако для этого нужны права администратора, и есть склонность ломать другие приложения, поскольку на время аннулируются все сертификаты, используемые TLS-соединениями (например, HTTPS). Он в основном полезен инженерам по античиту и исследователям оборонительной безопасности, работающим в области античита / инструментов подписи.
