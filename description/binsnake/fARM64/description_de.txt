Dieses Projekt konzentriert sich auf einen reinen Rust-no_std-AArch64-Disassembler und -Encoder — iced-x86-ähnliche API, heapfreier Decode-Pfad, SVE/SME/SIMD/FP-Abdeckung, semantische Round-Trip-Kodierung; wasm- und Bare-Metal-freundlich.
Die öffentliche API ist bewusst iced-x86-ähnlich gestaltet (ein leihender Decoder, ein wertetypischer Instruction, typisierte OpKind/Operand-Zugriffe, ein Token-emittierender Formatter).
Sie ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat-/RE-Tools untersuchen.
