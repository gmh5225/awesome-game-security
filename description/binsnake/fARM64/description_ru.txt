Этот проект посвящён чистому Rust no_std дизассемблеру и кодировщику AArch64 — API в стиле iced-x86, путь декодирования без кучи, покрытие SVE/SME/SIMD/FP, семантическое round-trip кодирование; удобен для wasm и bare-metal.
Публичный API намеренно сделан в стиле iced-x86 (заимствующий Decoder, Instruction по значению, типизированные аксессоры OpKind/Operand, Formatter с выдачей токенов).
В основном полезен исследователям безопасности игр и reverse-инженерам, изучающим атакующие техники в области cheat / RE-инструментов.
