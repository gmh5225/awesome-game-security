This project focuses on pure-Rust no_std AArch64 disassembler and encoder — iced-x86-shaped API, zero-heap decode path, SVE/SME/SIMD/FP coverage, semantic round-trip encode; wasm and bare-metal friendly.
The public API is deliberately iced-x86-shaped (a borrowing Decoder, a value-type Instruction, typed OpKind/Operand accessors, a token-emitting Formatter).
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / re tools area.
