Este proyecto se centra en un desensamblador y codificador AArch64 en Rust puro no_std — API al estilo iced-x86, ruta de decodificación sin heap, cobertura SVE/SME/SIMD/FP, codificación ida y vuelta semántica; apto para wasm y bare-metal.
La API pública está deliberadamente diseñada al estilo iced-x86 (un Decoder por préstamo, un Instruction de tipo valor, accesores OpKind/Operand tipados, un Formatter que emite tokens).
Es principalmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de herramientas de trampas / RE.
