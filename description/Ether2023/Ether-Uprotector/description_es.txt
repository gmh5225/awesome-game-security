Este proyecto es una herramienta de protección para juegos Unity centrada en IL2CPP y el cifrado de assets. La base de código en C# gestiona el análisis de archivos de assets de Unity, el cifrado de metadatos IL2CPP con XXTEA y criptografía personalizada, la ofuscación de funciones clave y flujos de protección basados en configuración. Admite protección a nivel de assets y de código para juegos Unity creados con el backend IL2CPP. Resulta especialmente útil para desarrolladores de juegos e investigadores de seguridad que estudian la protección de juegos Unity y las técnicas de ofuscación IL2CPP.
