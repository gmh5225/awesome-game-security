Questo progetto è un codice sorgente di kernel Android personalizzato per OnePlus Nord (piattaforma sm7250) con integrazione KernelSU. Fornisce un albero del kernel modificato basato sul BSP Qualcomm SM7250 con patch KernelSU per la gestione dell'accesso root, consentendo il root systemless sui dispositivi OnePlus Nord. È rivolto a sviluppatori di kernel Android e builder di ROM personalizzate che mirano a dispositivi OnePlus con supporto KernelSU.
