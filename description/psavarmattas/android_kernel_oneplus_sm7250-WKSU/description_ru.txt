Этот проект — исходный код кастомного ядра Android для OnePlus Nord (платформа sm7250) с интеграцией KernelSU. Он предоставляет модифицированное дерево ядра на базе Qualcomm SM7250 BSP с патчами KernelSU для управления root-доступом, что позволяет использовать systemless root на устройствах OnePlus Nord. Предназначен для разработчиков ядер Android и сборщиков кастомных ROM, ориентированных на устройства OnePlus с поддержкой KernelSU.
