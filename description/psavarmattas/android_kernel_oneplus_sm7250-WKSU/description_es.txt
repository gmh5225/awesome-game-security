Este proyecto es un código fuente de kernel de Android personalizado para el OnePlus Nord (plataforma sm7250) con integración de KernelSU. Proporciona un árbol de kernel modificado basado en el BSP Qualcomm SM7250 con parches de KernelSU para la gestión del acceso root, lo que permite un root systemless en dispositivos OnePlus Nord. Está dirigido a desarrolladores de kernels de Android y creadores de ROM personalizadas que trabajan con dispositivos OnePlus con soporte de KernelSU.
