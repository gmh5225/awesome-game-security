Ce projet est un code source de noyau Android personnalisé pour le OnePlus Nord (plateforme sm7250) avec intégration KernelSU. Il fournit une arborescence de noyau modifiée basée sur le BSP Qualcomm SM7250 avec des correctifs KernelSU pour la gestion de l'accès root, permettant un root systemless sur les appareils OnePlus Nord. Il s'adresse aux développeurs de noyaux Android et aux créateurs de ROM personnalisées ciblant les appareils OnePlus avec prise en charge de KernelSU.
