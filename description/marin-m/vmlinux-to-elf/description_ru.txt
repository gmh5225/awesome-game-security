Этот проект — vmlinux-to-elf, инструмент на Python, который преобразует сырые образы ядра Linux (vmlinux, bzImage, zImage) в корректные ELF-файлы с таблицами символов. Он извлекает таблицу символов kallsyms ядра, восстанавливает заголовки секций и создаёт ELF-бинарник, который можно загрузить в IDA Pro, Ghidra или другие дизассемблеры с полными именами функций. Предназначен для исследователей ядра Linux, разработчиков эксплойтов и специалистов по реверс-инжинирингу, которым нужно анализировать бинарники ядра с полной информацией о символах.
