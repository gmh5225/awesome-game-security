Pyarmor es una herramienta de línea de comandos de Python que ofusca scripts Python para proteger el código fuente frente a la ingeniería inversa y el uso no autorizado. La salida ofuscada se mantiene como archivos .py normales que pueden reemplazar a los originales, y admite varias estrategias de ofuscación que equilibran seguridad y rendimiento, incluido el renombrado irreversible y la conversión de funciones seleccionadas a C optimizado. Los scripts pueden vincularse a máquinas concretas, tener fechas de caducidad y, en Windows, protegerse aún más con Themida. Funciona con Python 2 y 3 en Windows, Linux (incluido embebido y Raspberry Pi) y macOS en arquitecturas habituales. El caso de uso principal es la protección de software para desarrolladores que necesitan distribuir aplicaciones o bibliotecas Python sin exponer el código fuente legible.
