Pyarmor 是一款 Python 命令行工具，通过对 Python 脚本进行混淆来保护源代码，防止逆向工程与未经授权的使用。混淆后的输出仍为普通的 .py 文件，可直接替换原文件；并支持多种在安全性与性能之间权衡的混淆策略，包括不可逆重命名以及将选定函数转换为优化的 C 代码。脚本可绑定到特定机器、设置过期日期，在 Windows 上还可借助 Themida 进一步加强防护。它支持 Python 2 与 3，可在 Windows、Linux（含嵌入式与 Raspberry Pi）以及常见架构上的 macOS 上运行。主要面向需要交付 Python 应用或库、又不希望暴露可读源代码的开发者，用于软件保护。
