Pyarmor 是一款 Python 命令列工具，透過對 Python 腳本進行混淆來保護原始碼，防止逆向工程與未經授權的使用。混淆後的輸出仍為一般的 .py 檔案，可直接取代原檔；並支援多種在安全性與效能之間取捨的混淆策略，包括不可逆重新命名以及將選定函式轉換為最佳化的 C 程式碼。腳本可綁定到特定機器、設定到期日，在 Windows 上還可借助 Themida 進一步加強防護。它支援 Python 2 與 3，可在 Windows、Linux（含嵌入式與 Raspberry Pi）以及常見架構上的 macOS 上執行。主要面向需要交付 Python 應用程式或函式庫、又不希望暴露可讀原始碼的開發者，用於軟體保護。
