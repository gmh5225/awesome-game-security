Pyarmor is a Python command-line tool that obfuscates Python scripts to protect source code from reverse engineering and unauthorized use. It keeps obfuscated output as normal .py files that can replace originals, and supports multiple obfuscation strategies that trade off security and performance, including irreversible renaming and converting selected functions to optimized C. Scripts can be bound to specific machines, given expiration dates, and on Windows further shielded with Themida. It runs on Python 2 and 3 across Windows, Linux (including embedded and Raspberry Pi), and macOS on common architectures. The primary use case is software protection for developers who need to ship Python applications or libraries without exposing readable source.
