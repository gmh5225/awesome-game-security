Pyarmor — это инструмент командной строки для Python, который обфусцирует Python-скрипты, защищая исходный код от реверс-инжиниринга и несанкционированного использования. Обфусцированный результат остаётся обычными файлами .py, которыми можно заменить оригиналы, и поддерживает несколько стратегий обфускации с компромиссом между безопасностью и производительностью, включая необратимое переименование и преобразование выбранных функций в оптимизированный C. Скрипты можно привязывать к конкретным машинам, задавать сроки действия, а в Windows дополнительно защищать с помощью Themida. Работает с Python 2 и 3 в Windows, Linux (включая встраиваемые системы и Raspberry Pi) и macOS на распространённых архитектурах. Основной сценарий — защита ПО для разработчиков, которым нужно поставлять приложения или библиотеки на Python без раскрытия читаемого исходного кода.
