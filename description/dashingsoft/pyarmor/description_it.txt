Pyarmor è uno strumento da riga di comando Python che offusca gli script Python per proteggere il codice sorgente da reverse engineering e uso non autorizzato. L’output offuscato resta in file .py normali che possono sostituire gli originali e supporta più strategie di offuscamento che bilanciano sicurezza e prestazioni, tra cui la ridenominazione irreversibile e la conversione di funzioni selezionate in C ottimizzato. Gli script possono essere vincolati a macchine specifiche, avere date di scadenza e, su Windows, essere ulteriormente protetti con Themida. Funziona con Python 2 e 3 su Windows, Linux (inclusi embedded e Raspberry Pi) e macOS sulle architetture comuni. Il caso d’uso principale è la protezione del software per sviluppatori che devono distribuire applicazioni o librerie Python senza esporre il codice sorgente leggibile.
