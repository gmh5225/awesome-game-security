Pyarmor는 Python 스크립트를 난독화하여 소스 코드를 역공학과 무단 사용으로부터 보호하는 Python 명령줄 도구입니다. 난독화된 결과는 일반 .py 파일로 유지되어 원본을 대체할 수 있으며, 보안과 성능의 균형을 맞추는 여러 난독화 전략을 지원합니다. 여기에는 되돌릴 수 없는 이름 변경과 선택한 함수를 최적화된 C로 변환하는 방식이 포함됩니다. 스크립트를 특정 머신에 바인딩하거나 만료일을 설정할 수 있고, Windows에서는 Themida로 추가 보호할 수 있습니다. Python 2와 3에서 동작하며 Windows, Linux(임베디드 및 Raspberry Pi 포함), 일반적인 아키텍처의 macOS를 지원합니다. 주요 용도는 읽기 쉬운 소스를 노출하지 않고 Python 애플리케이션이나 라이브러리를 배포해야 하는 개발자를 위한 소프트웨어 보호입니다.
