Pyarmor est un outil en ligne de commande Python qui obscurcit les scripts Python afin de protéger le code source contre la rétro-ingénierie et l’utilisation non autorisée. La sortie obscurcie reste des fichiers .py normaux pouvant remplacer les originaux, et il prend en charge plusieurs stratégies d’obscurcissement qui arbitrent entre sécurité et performances, notamment le renommage irréversible et la conversion de fonctions sélectionnées en C optimisé. Les scripts peuvent être liés à des machines spécifiques, dotés de dates d’expiration, et sous Windows davantage protégés avec Themida. Il fonctionne avec Python 2 et 3 sous Windows, Linux (y compris embarqué et Raspberry Pi) et macOS sur les architectures courantes. Le cas d’usage principal est la protection logicielle pour les développeurs qui doivent livrer des applications ou bibliothèques Python sans exposer un code source lisible.
