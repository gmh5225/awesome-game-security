Pyarmor は、Python スクリプトを難読化してソースコードをリバースエンジニアリングや不正利用から守る Python コマンドラインツールです。難読化後も通常の .py ファイルとして出力され、元のファイルと置き換え可能です。セキュリティとパフォーマンスのバランスを取った複数の難読化戦略をサポートし、不可逆なリネームや選択した関数の最適化された C への変換などを含みます。スクリプトは特定マシンへのバインドや有効期限の設定が可能で、Windows では Themida による追加保護にも対応します。Python 2 および 3 で動作し、Windows、Linux（組み込み環境や Raspberry Pi を含む）、一般的なアーキテクチャの macOS で利用できます。主な用途は、可読なソースを公開せずに Python アプリケーションやライブラリを配布したい開発者向けのソフトウェア保護です。
