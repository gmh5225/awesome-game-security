Этот проект — библиотека хуков DirectX 11 на C++, которая перехватывает IDXGISwapChain::Present и методы ID3D11DeviceContext для отрисовки пользовательских оверлеев в приложениях D3D11. Она находит vtable D3D11 через фиктивное устройство, устанавливает хуки функций и рисует меню ImGui внутри перехваченного цикла рендеринга. Библиотека служит основой для создания внутренних игровых оверлеев и чит-меню. Она предназначена для игровых хакеров и исследователей безопасности, изучающих техники отрисовки оверлеев на основе хуков D3D11.
