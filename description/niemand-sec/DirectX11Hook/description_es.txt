Este proyecto es una biblioteca de hooks de DirectX 11 en C++ que intercepta IDXGISwapChain::Present y métodos de ID3D11DeviceContext para renderizar overlays personalizados en aplicaciones D3D11. Localiza la vtable de D3D11 mediante un dispositivo ficticio, instala hooks de funciones y dibuja menús ImGui dentro del bucle de renderizado interceptado. La biblioteca sirve como base para crear overlays internos de juegos y menús de cheats. Está dirigida a game hackers e investigadores de seguridad que estudian técnicas de renderizado de overlays basadas en hooks de D3D11.
