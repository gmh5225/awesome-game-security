Этот проект — модульный фреймворк для проверки возможностей обнаружения античитов в Windows. Он предоставляет структурированный набор атакующих примитивов: чтение/запись памяти процесса, инъекция DLL, отрисовка оверлеев, симуляция ввода и операции ядра на основе драйверов, которые можно выборочно запускать, чтобы оценить, какие техники конкретный античит обнаруживает, а какие пропускает. Фреймворк на C/C++ помогает оценивать покрытие античитов. Он предназначен для разработчиков античитов и исследователей безопасности, проводящих систематическую оценку и анализ пробелов.
