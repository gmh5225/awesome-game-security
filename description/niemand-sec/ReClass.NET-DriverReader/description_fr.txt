Ce projet est un plugin ReClass.NET qui lit la mémoire d’un processus via un pilote noyau plutôt que par les API standard en mode utilisateur. Il remplace ReadProcessMemory par une lecture mémoire basée sur un pilote, permettant à ReClass.NET d’inspecter la mémoire de processus protégés par des anti-cheats qui bloquent l’accès mémoire classique. Le plugin C#/C++ fait le lien entre ReClass.NET et un lecteur mémoire noyau. Il s’adresse aux chercheurs en sécurité des jeux qui utilisent ReClass.NET pour analyser les structures mémoire de processus protégés par anti-cheat.
