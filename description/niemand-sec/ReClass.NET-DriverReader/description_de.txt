Dieses Projekt ist ein ReClass.NET-Plugin, das Prozessspeicher über einen Kernel-Treiber statt über Standard-User-Mode-APIs liest. Es ersetzt ReadProcessMemory durch treiberbasiertes Speicherlesen und ermöglicht ReClass.NET so die Inspektion von Speicher geschützter Prozesse, deren Anti-Cheat den normalen Speicherzugriff blockiert. Das C#/C++-Plugin verbindet ReClass.NET mit einem Kernel-Speicherleser. Es richtet sich an Game-Security-Forscher, die ReClass.NET zur Analyse von Speicherstrukturen bei Anti-Cheat-geschützten Prozessen nutzen.
