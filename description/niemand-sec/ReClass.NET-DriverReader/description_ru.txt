Этот проект — плагин ReClass.NET, который читает память процесса через драйвер ядра, а не через стандартные API пользовательского режима. Он заменяет ReadProcessMemory чтением памяти через драйвер, позволяя ReClass.NET просматривать память процессов, защищённых античитом, который блокирует обычный доступ к памяти. Плагин на C#/C++ связывает ReClass.NET с модулем чтения памяти в ядре. Предназначен для исследователей безопасности игр, использующих ReClass.NET для анализа структур памяти процессов, защищённых античитом.
