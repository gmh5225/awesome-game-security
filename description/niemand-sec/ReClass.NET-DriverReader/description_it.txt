Questo progetto è un plugin di ReClass.NET che legge la memoria di processo tramite un driver del kernel anziché le API standard in user-mode. Sostituisce ReadProcessMemory con una lettura di memoria basata su driver, consentendo a ReClass.NET di ispezionare la memoria di processi protetti da anti-cheat che bloccano l’accesso standard. Il plugin C#/C++ collega ReClass.NET a un memory reader del kernel. È rivolto ai ricercatori di game security che usano ReClass.NET per analizzare le strutture di memoria di processi protetti da anti-cheat.
