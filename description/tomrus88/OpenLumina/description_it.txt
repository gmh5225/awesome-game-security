Questo progetto si concentra sul consentire la connessione a server Lumina di terze parti.
Compila o scarica una versione precompilata del plugin e mettila nella directory IDA\plugins; copia il/i file di certificato hexrays .crt fornito/i dal proprietario del server Lumina nella directory di installazione di IDA.
È soprattutto utile per i ricercatori di sicurezza dei giochi e i reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin IDA.
