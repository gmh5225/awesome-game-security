Este proyecto es palera1n, un jailbreak para desarrolladores de iOS 15+ en dispositivos compatibles con checkm8 (A8–A11). Explota la vulnerabilidad de bootrom checkm8 a nivel de hardware por USB para arrancar un kernel parcheado con AMFI y la firma de código deshabilitados, instalando el gestor de paquetes Sileo y un entorno bootstrap. La herramienta admite modos de jailbreak rootful y rootless; este último deja el sistema de archivos raíz sellado. Está dirigida a investigadores de seguridad de iOS, desarrolladores de jailbreak e ingenieros de ingeniería inversa que necesitan acceso root en versiones modernas de iOS.
