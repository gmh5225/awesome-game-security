Этот проект — palera1n, developer jailbreak для iOS 15+ на устройствах, совместимых с checkm8 (A8–A11). Он использует аппаратную уязвимость bootrom checkm8 по USB, чтобы загрузить пропатченное ядро с отключёнными AMFI и проверкой подписи кода, устанавливая пакетный менеджер Sileo и bootstrap-окружение. Инструмент поддерживает режимы jailbreak rootful и rootless; во втором корневая файловая система остаётся запечатанной. Предназначен для исследователей безопасности iOS, разработчиков jailbreak и reverse-инженеров, которым нужен root-доступ на современных версиях iOS.
