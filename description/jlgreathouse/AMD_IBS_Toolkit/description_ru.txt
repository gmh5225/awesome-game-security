Этот проект посвящён AMD Sampling. IBS — аппаратный механизм, который выполняет выборку подмножества всех инструкций, проходящих через процессор. Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования ядра Windows.
