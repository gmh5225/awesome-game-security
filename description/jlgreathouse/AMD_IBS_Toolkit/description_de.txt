Dieses Projekt konzentriert sich auf AMD Sampling. IBS ist ein Hardware-Mechanismus, der eine Teilmenge aller Anweisungen abtastet, die einen Prozessor durchlaufen. Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Windows-Kernel-Exploration untersuchen.
