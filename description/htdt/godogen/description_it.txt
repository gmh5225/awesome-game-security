Questo progetto offre sviluppo di giochi autonomo per Godot, Bevy e Babylon.js con Claude Code e Codex. Pianifica le funzionalità, genera asset e usa cicli di auto-riparazione guidati da screenshot, organizzati lungo l'asse del motore. È soprattutto utile per sviluppatori di giochi, reverse engineer e creatori di tooling che lavorano nell'area game develop / AI agents.
