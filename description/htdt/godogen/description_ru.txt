Этот проект обеспечивает автономную разработку игр для Godot, Bevy и Babylon.js с помощью Claude Code и Codex. Он планирует функции, генерирует ассеты и использует циклы самоисправления по скриншотам, организованные вдоль оси движка. В первую очередь полезен разработчикам игр, реверс-инженерам и создателям инструментов в области game develop / AI agents.
