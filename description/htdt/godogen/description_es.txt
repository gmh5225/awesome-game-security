Este proyecto ofrece desarrollo autónomo de juegos para Godot, Bevy y Babylon.js con Claude Code y Codex. Planifica funciones, genera assets y usa bucles de autorreparación guiados por capturas de pantalla, organizados según el eje del motor. Es especialmente útil para desarrolladores de juegos, ingenieros inversos y creadores de herramientas que trabajan en el área de desarrollo de juegos / agentes de IA.
