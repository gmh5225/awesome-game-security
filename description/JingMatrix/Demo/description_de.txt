Dieses Projekt ist eine Android-Demoanwendung zur Laufzeiterkennung von Bibliotheksinjektionen im User Space. Es demonstriert mehrere Erkennungsstrategien, darunter Prüfungen der soinfo-Verkettungsliste, die Untersuchung der virtuellen Memory Map sowie die Überwachung eines Modul-Unload-Zählers. Die Implementierung kombiniert Kotlin-Android-Code mit nativen C++-Komponenten, die über CMake gebaut werden. Der primäre Anwendungsfall ist die mobile Anti-Cheat- und Sicherheitsforschung mit Fokus auf die Identifikation von Injektionstechniken.
