Этот проект — демонстрационное Android-приложение для обнаружения внедрения пользовательских библиотек во время выполнения. В нём показаны несколько стратегий детектирования, включая проверки связанного списка soinfo, анализ карты виртуальной памяти и мониторинг счётчика выгрузки модулей. Реализация сочетает код Android на Kotlin с нативными компонентами на C++, собираемыми через CMake. Основной сценарий использования — мобильные античиты и исследования безопасности с упором на выявление техник инъекции.
