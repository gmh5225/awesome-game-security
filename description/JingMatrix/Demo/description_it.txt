Questo progetto è un'applicazione demo Android per rilevare a runtime l'iniezione di librerie in spazio utente. Illustra diverse strategie di rilevamento, tra cui controlli della linked list soinfo, ispezione della mappa di memoria virtuale e monitoraggio di un contatore di scaricamento dei moduli. L'implementazione combina codice Android in Kotlin con componenti nativi C++ compilati tramite CMake. Il caso d'uso principale è la ricerca su anti-cheat mobile e sicurezza, focalizzata sull'identificazione delle tecniche di iniezione.
