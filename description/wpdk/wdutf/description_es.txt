Este proyecto es el Windows Driver Unit Test Framework (WDUTF) y permite realizar pruebas unitarias de controladores del kernel de Windows mediante el Microsoft Unit Testing Framework for C++, que se ejecuta en espacio de usuario. Ejecutar pruebas unitarias contra código escrito para el entorno Windows Kernel es difícil. Resulta especialmente útil para ingenieros anti-cheat e investigadores de seguridad defensiva que trabajan en el área de frameworks de pruebas unitarias anti-cheat / de controladores.
