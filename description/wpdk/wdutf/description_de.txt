Dieses Projekt ist das Windows Driver Unit Test Framework (WDUTF) und ermöglicht Unit-Tests von Windows-Kernel-Treibern mit dem Microsoft Unit Testing Framework for C++, das im User Space ausgeführt wird. Unit-Tests für Code, der für die Windows-Kernel-Umgebung geschrieben wurde, sind schwierig. Es ist vor allem nützlich für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher, die im Bereich Anti-Cheat- / Treiber-Unit-Test-Frameworks arbeiten.
