Questo progetto è il Windows Driver Unit Test Framework (WDUTF) e consente di eseguire unit test dei driver del kernel Windows usando il Microsoft Unit Testing Framework for C++, che gira in user space. Eseguire unit test sul codice scritto per l'ambiente Windows Kernel è difficile. È soprattutto utile per gli ingegneri anti-cheat e i ricercatori di sicurezza difensiva che lavorano nell'area dei framework di unit test anti-cheat / driver.
