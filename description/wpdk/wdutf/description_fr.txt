Ce projet est le Windows Driver Unit Test Framework (WDUTF), qui permet de réaliser des tests unitaires de pilotes noyau Windows à l'aide du Microsoft Unit Testing Framework for C++, qui s'exécute en espace utilisateur. Exécuter des tests unitaires sur du code écrit pour l'environnement Windows Kernel est difficile. Il est surtout utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine des frameworks de tests unitaires anti-cheat / pilotes.
