Этот проект — Windows Driver Unit Test Framework (WDUTF), он позволяет выполнять модульное тестирование драйверов ядра Windows с помощью Microsoft Unit Testing Framework for C++, работающего в пользовательском пространстве. Запускать модульные тесты для кода, написанного под среду Windows Kernel, сложно. Он в первую очередь полезен инженерам античита и исследователям оборонительной безопасности, работающим в области фреймворков модульного тестирования античита / драйверов.
