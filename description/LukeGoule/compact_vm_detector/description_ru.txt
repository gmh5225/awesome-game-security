Этот проект — компактный компонент обнаружения виртуальных машин, предназначенный для встраивания в приложения Windows на C++. Это более компактная производная, вдохновлённая pafish, и ориентирована на лёгкие проверки антивиртуализации, а не на полноценный исследовательский набор. Реализация уделяет приоритет простой интеграции в среды Visual C++ при минимальном размере. Основной сценарий использования — антианалитические эксперименты и быстрое снятие отпечатка окружения в инструментах, ориентированных на безопасность.
