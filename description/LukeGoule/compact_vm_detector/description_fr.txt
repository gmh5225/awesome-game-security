Ce projet est un composant compact de détection de machines virtuelles destiné à être intégré dans des applications Windows C++. Il s'agit d'un dérivé plus léger inspiré de pafish, axé sur des contrôles anti-virtualisation légers plutôt que sur une suite de recherche complète. L'implémentation privilégie une intégration facile dans les environnements Visual C++ avec une empreinte minimale. Son cas d'usage principal est l'expérimentation anti-analyse et le fingerprinting rapide d'environnement dans des outils orientés sécurité.
