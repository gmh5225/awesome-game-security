Questo progetto è UnrealMCP, un’integrazione server Model Context Protocol per Unreal Engine. Espone le funzionalità dell’Unreal Editor tramite il protocollo MCP, consentendo ad assistenti IA e strumenti di automazione di interagire con l’editor per attività come gestione degli asset, editing dei livelli, manipolazione dei blueprint e configurazione del progetto. È rivolto agli sviluppatori Unreal Engine che integrano workflow assistiti dall’IA e automazione nella propria pipeline di sviluppo di giochi.
