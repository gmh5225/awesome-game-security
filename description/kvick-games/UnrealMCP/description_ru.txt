Этот проект — UnrealMCP, интеграция сервера Model Context Protocol для Unreal Engine. Он предоставляет функции Unreal Editor через протокол MCP, позволяя ИИ-ассистентам и средствам автоматизации взаимодействовать с редактором для управления ассетами, редактирования уровней, работы с blueprint и настройки проекта. Предназначен для разработчиков Unreal Engine, внедряющих рабочие процессы с поддержкой ИИ и автоматизацию в пайплайн разработки игр.
