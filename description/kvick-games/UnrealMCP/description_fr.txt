Ce projet est UnrealMCP, une intégration serveur Model Context Protocol pour Unreal Engine. Il expose les fonctionnalités de l’Unreal Editor via le protocole MCP, permettant aux assistants IA et aux outils d’automatisation d’interagir avec l’éditeur pour la gestion des assets, l’édition de niveaux, la manipulation de blueprints et la configuration de projet. Il s’adresse aux développeurs Unreal Engine qui intègrent des workflows assistés par l’IA et l’automatisation dans leur pipeline de développement de jeux.
