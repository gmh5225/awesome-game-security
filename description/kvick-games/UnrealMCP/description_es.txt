Este proyecto es UnrealMCP, una integración de servidor Model Context Protocol para Unreal Engine. Expone la funcionalidad del Unreal Editor a través del protocolo MCP, permitiendo que asistentes de IA y herramientas de automatización interactúen con el editor en tareas como gestión de assets, edición de niveles, manipulación de blueprints y configuración del proyecto. Está orientado a desarrolladores de Unreal Engine que integran flujos de trabajo asistidos por IA y automatización en su pipeline de desarrollo de juegos.
