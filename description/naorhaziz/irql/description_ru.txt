Этот проект — Rust-крейт, предоставляющий примитивы выделения памяти и структур данных с учётом IRQL для разработки ядра Windows. Рабочая область включает основные абстракции irql, крейт irql_alloc с типами Box и Vec на основе пулов, а также утилиты для написания IRQL-безопасного кода ядра на Rust. Он в первую очередь полезен разработчикам ядра и исследователям безопасности, создающим драйверы ядра Windows на Rust с корректным управлением памятью на уровне IRQL.
