Este proyecto es una crate de Rust que proporciona primitivas de asignación de memoria y estructuras de datos conscientes de IRQL para el desarrollo del kernel de Windows. El espacio de trabajo incluye las abstracciones principales de irql, una crate irql_alloc con tipos Box y Vec respaldados por pools, y utilidades para escribir código de kernel seguro en IRQL en Rust. Resulta especialmente útil para desarrolladores de kernel e investigadores de seguridad que construyen controladores del kernel de Windows en Rust con una gestión de memoria adecuada a nivel de IRQL.
