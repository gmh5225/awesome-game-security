Ce projet est une crate Rust qui fournit des primitives d'allocation mémoire et de structures de données sensibles à l'IRQL pour le développement du noyau Windows. L'espace de travail comprend les abstractions principales irql, une crate irql_alloc avec des types Box et Vec basés sur les pools, ainsi que des utilitaires pour écrire du code noyau IRQL-safe en Rust. Il est surtout utile aux développeurs noyau et aux chercheurs en sécurité qui créent des pilotes noyau Windows en Rust avec une gestion mémoire correcte au niveau IRQL.
