Questo progetto è una crate Rust che fornisce primitive di allocazione di memoria e strutture dati consapevoli dell'IRQL per lo sviluppo del kernel Windows. Il workspace include le astrazioni principali di irql, una crate irql_alloc con tipi Box e Vec basati su pool, e utilità per scrivere codice kernel IRQL-safe in Rust. È particolarmente utile per sviluppatori kernel e ricercatori di sicurezza che realizzano driver del kernel Windows in Rust con una corretta gestione della memoria a livello di IRQL.
