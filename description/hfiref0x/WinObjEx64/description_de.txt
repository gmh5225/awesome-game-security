Dieses Projekt ist ein fortschrittliches Dienstprogramm zum Erkunden des Windows Object Manager-Namespaces.
Für die Anzeige großer Teile des Namespaces sowie zum Bearbeiten objektspezifischer Sicherheitsinformationen sind jedoch Administratorrechte erforderlich.
Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat / Windows-Ring0-Callbacks arbeiten.
