Este proyecto es una utilidad avanzada que permite explorar el espacio de nombres del Windows Object Manager.
Sin embargo, se requieren privilegios de administrador para ver gran parte del espacio de nombres y para editar la información de seguridad relacionada con los objetos.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / callbacks ring0 de Windows.
