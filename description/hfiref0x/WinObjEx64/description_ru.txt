Этот проект — продвинутая утилита для исследования пространства имён Windows Object Manager.
Однако для просмотра большей части пространства имён и редактирования сведений безопасности, связанных с объектами, требуются права администратора.
Она в первую очередь полезна инженерам античита и исследователям защитной безопасности, работающим в области античита / Windows ring0 callback.
