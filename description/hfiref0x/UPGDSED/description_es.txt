Este proyecto es UPGDSED (Universal PatchGuard and DSE Disable), una herramienta de Windows para desactivar PatchGuard (Kernel Patch Protection) y Driver Signature Enforcement (DSE) en tiempo de ejecución. Emplea varias técnicas, entre ellas explotar controladores firmados vulnerables, manipular variables globales de CI.dll y parchear datos de contexto de KPP, para permitir la carga de controladores de núcleo sin firmar y modificar estructuras de núcleo protegidas. El código en C cubre varias versiones de Windows, de la 7 a la 11. Está dirigido a investigadores de núcleo que estudian los internos de PatchGuard, métodos de bypass de DSE y mecanismos de seguridad del núcleo de Windows.
