本プロジェクトは UPGDSED（Universal PatchGuard and DSE Disable）であり、実行時に PatchGuard（カーネルパッチ保護）と Driver Signature Enforcement（DSE）を無効化する Windows ツールです。脆弱な署名済みドライバの悪用、CI.dll のグローバル変数の操作、KPP コンテキストデータのパッチなど複数の手法を用い、未署名のカーネルドライバの読み込みや保護されたカーネル構造の改変を可能にします。C で書かれたコードベースは Windows 7 から 11 までの複数バージョンを対象としています。PatchGuard の内部、DSE バイパス手法、Windows カーネルのセキュリティ機構を研究するカーネル研究者向けです。
