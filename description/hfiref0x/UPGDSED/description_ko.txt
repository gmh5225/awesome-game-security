이 프로젝트는 UPGDSED(Universal PatchGuard and DSE Disable)로, 런타임에 PatchGuard(커널 패치 보호)와 Driver Signature Enforcement(DSE)를 비활성화하는 Windows 도구입니다. 취약한 서명된 드라이버 악용, CI.dll 전역 변수 조작, KPP 컨텍스트 데이터 패치 등 여러 기법을 사용하여 서명되지 않은 커널 드라이버 로드와 보호된 커널 구조 수정을 가능하게 합니다. C로 작성된 코드베이스는 Windows 7부터 11까지 여러 버전을 대상으로 합니다. PatchGuard 내부 구조, DSE 우회 방법, Windows 커널 보안 메커니즘을 연구하는 커널 연구자를 위한 도구입니다.
