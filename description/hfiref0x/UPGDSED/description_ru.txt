Этот проект — UPGDSED (Universal PatchGuard and DSE Disable), инструмент для Windows, отключающий PatchGuard (Kernel Patch Protection) и Driver Signature Enforcement (DSE) во время выполнения. Он использует несколько техник, включая эксплуатацию уязвимых подписанных драйверов, манипуляцию глобальными переменными CI.dll и патчинг контекстных данных KPP, чтобы разрешить загрузку неподписанных драйверов ядра и изменение защищённых структур ядра. Кодовая база на C рассчитана на несколько версий Windows — с 7 по 11. Предназначен для исследователей ядра, изучающих внутренности PatchGuard, методы обхода DSE и механизмы безопасности ядра Windows.
