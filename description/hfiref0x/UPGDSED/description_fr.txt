Ce projet est UPGDSED (Universal PatchGuard and DSE Disable), un outil Windows permettant de désactiver PatchGuard (Kernel Patch Protection) et Driver Signature Enforcement (DSE) à l'exécution. Il emploie plusieurs techniques, notamment l'exploitation de pilotes signés vulnérables, la manipulation des variables globales de CI.dll et le patching des données de contexte KPP, afin d'autoriser le chargement de pilotes noyau non signés et la modification de structures noyau protégées. Le code source en C cible plusieurs versions de Windows, de 7 à 11. Il s'adresse aux chercheurs noyau qui étudient les internals de PatchGuard, les méthodes de contournement de DSE et les mécanismes de sécurité du noyau Windows.
