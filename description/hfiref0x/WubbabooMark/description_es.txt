Este proyecto tiene como objetivo detectar rastros del uso de depuradores de software o de software especial diseñado para ocultar la presencia del depurador ante el programa depurado manipulando diversos aspectos del entorno del programa.
El conjunto típico de depuradores actuales se limita en realidad a unas pocas soluciones muy populares como Ghidra/IDA/OllyDbg/x32+x64dbg/WinDbg y similares.
Resulta especialmente útil para ingenieros de anti-trampas e investigadores de seguridad defensiva que trabajan en el área de anti-trampas / anti-depuración.
