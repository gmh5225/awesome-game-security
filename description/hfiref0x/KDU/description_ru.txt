Этот проект — Kernel Driver Utility (KDU), инструмент для Windows, загружающий неподписанные драйверы ядра путём эксплуатации уязвимых легитимных подписанных драйверов (BYOVD). Он включает расширяемую систему провайдеров с поддержкой десятков известных уязвимых драйверов (Intel, ASUS, MSI, Gigabyte и др.) для получения произвольного чтения/записи в ядре или выполнения кода, а затем использует эти примитивы для отображения пользовательского неподписанного драйвера в память ядра. Кодовая база на C автоматизирует обход DSE, отображение драйвера и очистку. Предназначен для исследователей безопасности, изучающих обход принудительной проверки подписи драйверов и поверхность атаки BYOVD.
