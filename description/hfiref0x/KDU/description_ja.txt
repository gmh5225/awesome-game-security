本プロジェクトは Kernel Driver Utility（KDU）で、脆弱な正規の署名付きドライバを悪用する BYOVD により、未署名のカーネルドライバを読み込む Windows ツールです。拡張可能なプロバイダシステムを備え、Intel、ASUS、MSI、Gigabyte など数十の既知の脆弱なドライバに対応し、任意のカーネル読み書きやコード実行を得たうえで、そのプリミティブを用いてカスタム未署名ドライバをカーネルメモリへマップします。C で書かれたコードベースは DSE バイパス、ドライバマッピング、クリーンアップを自動化します。ドライバ署名強制のバイパスと BYOVD 攻撃面を研究するセキュリティ研究者向けです。
