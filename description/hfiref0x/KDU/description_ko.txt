이 프로젝트는 Kernel Driver Utility(KDU)로, 취약한 정상 서명 드라이버를 악용하는 BYOVD 방식으로 서명되지 않은 커널 드라이버를 로드하는 Windows 도구입니다. Intel, ASUS, MSI, Gigabyte 등 수십 개의 알려진 취약 드라이버를 지원하는 확장 가능한 프로바이더 시스템을 포함하며, 임의의 커널 읽기/쓰기 또는 코드 실행을 확보한 뒤 그 프리미티브를 사용해 사용자 정의 미서명 드라이버를 커널 메모리에 매핑합니다. C 코드베이스는 DSE 우회, 드라이버 매핑, 정리를 자동화합니다. 드라이버 서명 강제(DSE) 우회와 BYOVD 공격면을 연구하는 보안 연구자를 대상으로 합니다.
