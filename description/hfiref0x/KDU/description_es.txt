Este proyecto es el Kernel Driver Utility (KDU), una herramienta de Windows para cargar controladores de kernel sin firmar explotando controladores firmados legítimos vulnerables (BYOVD). Incluye un sistema de proveedores extensible que admite docenas de controladores vulnerables conocidos (Intel, ASUS, MSI, Gigabyte, etc.) para obtener lectura/escritura arbitraria en el kernel o ejecución de código, y luego usa esas primitivas para mapear un controlador personalizado sin firmar en la memoria del kernel. La base de código en C automatiza el bypass de DSE, el mapeo de controladores y la limpieza. Está dirigido a investigadores de seguridad que estudian el bypass de la aplicación de firmas de controladores y la superficie de ataque BYOVD.
