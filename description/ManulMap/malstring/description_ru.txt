Этот проект — header-only библиотека на C++23 для обфускации строк и байтовых массивов на этапе компиляции.
Она использует шаблонное метапрограммирование для генерации XOR-шифрованных стековых строк, call-строк и вызываемых массивов, сохраняя при этом краткость использования в исходном коде.
Реализация поддерживает отдельные ключи для каждой строки и схемы расшифровки при использовании, чтобы уменьшить очевидные следы открытого текста в бинарниках.
Основная аудитория — разработчики низкоуровневой безопасности и исследователи реверс-инжиниринга, экспериментирующие с техниками устойчивости к статическому анализу.
