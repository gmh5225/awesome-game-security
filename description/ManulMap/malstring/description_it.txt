Questo progetto è una libreria C++23 header-only per l'offuscamento a compile-time di stringhe e array di byte.
Usa la metaprogrammazione con template per generare stack string, call string e array richiamabili cifrati con XOR, mantenendo un uso conciso nel codice sorgente.
L'implementazione supporta chiavi per stringa e pattern di decifratura all'uso per ridurre evidenti artefatti in chiaro nei binari.
Il pubblico principale sono sviluppatori di sicurezza a basso livello e ricercatori di reverse engineering che sperimentano tecniche di resistenza all'analisi statica.
