Questo progetto è un motore di gioco 2D open source multipiattaforma. È scritto principalmente in C++ e include un’ampia stack del motore con rendering, fisica Box2D, sistemi audio, supporto alle animazioni e strumenti di progetto modulari. Il codice punta a Windows, macOS, Linux, iOS, Android e flussi di lavoro legati al web, e include moduli di esempio per l’apprendimento e l’iterazione rapida. Il caso d’uso principale è costruire e personalizzare giochi 2D completi con controllo a livello di codice sorgente.
