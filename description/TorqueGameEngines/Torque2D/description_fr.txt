Ce projet est un moteur de jeu 2D open source multiplateforme. Il est principalement écrit en C++ et fournit une large pile moteur incluant le rendu, la physique Box2D, les systèmes audio, le support de l’animation et des outils de projet modulaires. La base de code cible Windows, macOS, Linux, iOS, Android et des flux de travail liés au web, et inclut des modules d’exemple pour l’apprentissage et l’itération rapide. Son cas d’usage principal est la création et la personnalisation de jeux 2D complets avec un contrôle au niveau du code source.
