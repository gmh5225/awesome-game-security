Este proyecto es un motor de juegos 2D multiplataforma de código abierto. Está escrito principalmente en C++ y ofrece una amplia pila de motor que incluye renderizado, física Box2D, sistemas de audio, soporte de animación y herramientas de proyecto modulares. El código base está orientado a Windows, macOS, Linux, iOS, Android y flujos de trabajo relacionados con la web, e incluye módulos de ejemplo para aprender e iterar con rapidez. Su caso de uso principal es crear y personalizar juegos 2D completos con control a nivel de código fuente.
