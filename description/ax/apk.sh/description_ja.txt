本プロジェクトは、APK の取得、デコード、再構築、パッチ適用など、Android APK のリバースエンジニアリング作業を自動化する Bash スクリプトです。
apktool による逆アセンブルと再構築に対応し、ランタイム計測のための Frida gadget 注入をサポートし、app bundle や split APK を単一の APK に結合したうえで、apksigner により署名します。
arm、arm64、x86、x86_64 など複数アーキテクチャに対応し、root 化された Android 端末を必要としません。
主に、Android アプリ解析や Frida ベースの計測を行うモバイルセキュリティ研究者およびリバースエンジニア向けです。
