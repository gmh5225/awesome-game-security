本プロジェクトは Wazuh で、Linux、Windows、macOS のエンドポイント向けに侵入検知、ログ分析、ファイル整合性監視、脆弱性スキャン、コンプライアンス監査を提供するオープンソースのセキュリティ監視プラットフォームです。エージェント／サーバー構成を採用し、中央マネージャーがイベントを関連付け、検知ルールを適用し、可視化のために Elasticsearch と連携します。C と Python のコードベースにはリアルタイムアラート、アクティブレスポンス機能、RESTful API が含まれます。企業向けのホストベース侵入検知とセキュリティ情報管理を導入するセキュリティ運用チームを対象としています。
