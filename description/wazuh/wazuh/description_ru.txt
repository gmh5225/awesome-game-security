Этот проект — Wazuh, открытая платформа мониторинга безопасности, которая обеспечивает обнаружение вторжений, анализ журналов, контроль целостности файлов, сканирование уязвимостей и аудит соответствия на конечных точках Linux, Windows и macOS. Используется архитектура агент–сервер с центральным менеджером, который коррелирует события, применяет правила обнаружения и интегрируется с Elasticsearch для визуализации. Кодовая база на C и Python включает оповещения в реальном времени, возможности активного реагирования и RESTful API. Предназначена для команд Security Operations, разворачивающих корпоративное хост-ориентированное обнаружение вторжений и управление информацией о безопасности.
