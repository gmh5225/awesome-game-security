Ce projet est Wazuh, une plateforme open source de supervision de la sécurité qui offre la détection d'intrusions, l'analyse de journaux, le contrôle d'intégrité des fichiers, le scan de vulnérabilités et l'audit de conformité sur les postes Linux, Windows et macOS. Elle s'appuie sur une architecture agent-serveur avec un gestionnaire central qui corrèle les événements, applique des règles de détection et s'intègre à Elasticsearch pour la visualisation. La base de code en C et Python inclut des alertes en temps réel, des capacités de réponse active et une API RESTful. Elle s'adresse aux équipes SOC déployant une détection d'intrusions basée sur l'hôte et une gestion des informations de sécurité de niveau entreprise.
