이 프로젝트는 Wazuh로, Linux, Windows, macOS 엔드포인트에서 침입 탐지, 로그 분석, 파일 무결성 모니터링, 취약점 스캔, 규정 준수 감사를 제공하는 오픈소스 보안 모니터링 플랫폼입니다. 에이전트-서버 아키텍처를 사용하며, 중앙 매니저가 이벤트를 상관 분석하고 탐지 규칙을 적용하며 시각화를 위해 Elasticsearch와 통합됩니다. C와 Python 코드베이스에는 실시간 알림, 능동 대응 기능, RESTful API가 포함되어 있습니다. 엔터프라이즈급 호스트 기반 침입 탐지 및 보안 정보 관리를 배포하는 보안 운영 팀을 대상으로 합니다.
