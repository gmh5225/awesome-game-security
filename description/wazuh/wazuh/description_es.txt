Este proyecto es Wazuh, una plataforma de monitorización de seguridad de código abierto que ofrece detección de intrusiones, análisis de registros, monitorización de integridad de archivos, escaneo de vulnerabilidades y auditoría de cumplimiento en endpoints Linux, Windows y macOS. Utiliza una arquitectura agente-servidor con un gestor central que correlaciona eventos, aplica reglas de detección e integra con Elasticsearch para la visualización. La base de código en C y Python incluye alertas en tiempo real, capacidades de respuesta activa y una API RESTful. Está pensada para equipos de operaciones de seguridad que despliegan detección de intrusiones basada en host y gestión de información de seguridad de nivel empresarial.
