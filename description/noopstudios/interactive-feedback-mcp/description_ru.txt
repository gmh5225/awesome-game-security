Этот проект — сервер MCP (Model Context Protocol), предоставляющий ИИ-ассистентам для написания кода возможности интерактивной обратной связи с пользователем. Он позволяет ИИ запрашивать ввод, подтверждения и выбор пользователя через структурированные UI-приглашения в ходе автоматизированных рабочих процессов, обеспечивая взаимодействие «человек в контуре» в инструментах, совместимых с MCP. Предназначен для разработчиков, создающих интерактивные ИИ-ассистированные рабочие процессы, требующие обратной связи пользователя во время выполнения.
