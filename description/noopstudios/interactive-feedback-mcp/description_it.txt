Questo progetto è un server MCP (Model Context Protocol) che fornisce capacità di feedback utente interattivo agli assistenti di coding basati su IA. Consente all'IA di richiedere input, conferme e selezioni dell'utente tramite prompt UI strutturati durante i flussi di lavoro automatizzati, abilitando l'interazione human-in-the-loop all'interno di strumenti compatibili con MCP. È rivolto agli sviluppatori che creano flussi di lavoro interattivi assistiti da IA che richiedono feedback dell'utente durante l'esecuzione.
