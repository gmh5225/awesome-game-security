Este proyecto es un servidor MCP (Model Context Protocol) que proporciona capacidades de retroalimentación interactiva del usuario a los asistentes de codificación con IA. Permite a la IA solicitar entradas, confirmaciones y selecciones del usuario mediante mensajes de interfaz estructurados durante flujos de trabajo automatizados, habilitando la interacción humana en el bucle dentro de herramientas compatibles con MCP. Está dirigido a desarrolladores que construyen flujos de trabajo interactivos asistidos por IA que requieren retroalimentación del usuario durante la ejecución.
