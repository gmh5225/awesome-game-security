Dieses Projekt ist ein MCP-Server (Model Context Protocol), der KI-Coding-Assistenten interaktive Benutzer-Feedback-Funktionen bereitstellt. Es ermöglicht der KI, während automatisierter Workflows über strukturierte UI-Prompts Benutzereingaben, Bestätigungen und Auswahlen anzufordern und so eine Human-in-the-Loop-Interaktion in MCP-kompatiblen Tools zu realisieren. Es richtet sich an Entwickler, die interaktive KI-gestützte Workflows erstellen, die während der Ausführung Benutzer-Feedback benötigen.
