Dieses Projekt ist eine Automatisierungs-Einrichtung zum Erstellen einer Anti-Erkennungs-Variante von frida-server für Android. Es verfolgt Upstream-Änderungen von Frida und wendet eine Folge von Kern-Patches an, die erkennbare Strings, Pipes, Symbole und Protokollverhalten verändern. Das Repository ist schlank und auf den Patch-Workflow ausgerichtet, nicht auf eine große eigenständige Codebasis. Es richtet sich an mobile Reverse Engineering und Anti-Instrumentation-Forschung, wenn Standard-Frida-Signaturen blockiert werden.
