Ce projet est une configuration d'automatisation pour construire une variante anti-détection de frida-server pour Android. Il suit les évolutions de Frida en amont et applique une séquence de correctifs principaux qui modifient les chaînes, pipes, symboles et comportements de protocole reconnaissables. Le dépôt est léger et centré sur le flux de travail des correctifs plutôt que sur une vaste base de code autonome. Il vise la rétro-ingénierie mobile et la recherche anti-instrumentation lorsque les signatures Frida standards sont bloquées.
