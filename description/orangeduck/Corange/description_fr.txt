Ce projet se concentre sur un moteur de jeu écrit en C pur.
Avertissement : certains éléments présentés proviennent d'une version précédente et peuvent différer dans cette version.
Il est principalement utile aux développeurs de jeux, aux ingénieurs en rétro-ingénierie et aux créateurs d'outils travaillant dans le domaine du développement de jeux / du code source.
