Questo progetto è un'estensione dell'interfaccia del debugger in stile PEDA per WinDbg, realizzata con il motore di scripting Python pykd.
Visualizza registri, codice disassemblato vicino al program counter e contenuto dello stack con dereference intelligente a ogni step o trace, insieme a comandi in stile PEDA per l'ispezione della memoria e la ricerca di simboli.
L'estensione Python offre un'esperienza di debugging ispirata a GDB-PEDA nell'ambiente WinDbg, per uno sviluppo di exploit e un'analisi binaria più confortevoli.
È soprattutto utile per sviluppatori di exploit Windows e reverse engineer che preferiscono un flusso di debugging in stile PEDA in WinDbg.
