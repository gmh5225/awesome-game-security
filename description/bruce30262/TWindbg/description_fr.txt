Ce projet est une extension d'interface de débogueur de type PEDA pour WinDbg, conçue avec le moteur de scripts Python pykd.
Elle affiche les registres, le code désassemblé près du compteur de programme et le contenu de la pile avec déréférencement intelligent à chaque pas ou trace, ainsi que des commandes de style PEDA pour l'inspection mémoire et la recherche de symboles.
Cette extension Python offre une expérience de débogage inspirée de GDB-PEDA dans l'environnement WinDbg, pour un développement d'exploits et une analyse binaire plus confortables.
Elle est surtout utile aux développeurs d'exploits Windows et aux reverse engineers qui préfèrent un flux de débogage de type PEDA dans WinDbg.
