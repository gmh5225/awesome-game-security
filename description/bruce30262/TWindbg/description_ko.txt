이 프로젝트는 pykd Python 스크립팅 엔진으로 구축된 WinDbg용 PEDA 스타일 디버거 UI 확장입니다.
각 스텝 또는 트레이스마다 레지스터, 프로그램 카운터 근처의 디스어셈블 코드, 스마트 역참조가 적용된 스택 내용을 표시하며, 메모리 검사와 심볼 조회를 위한 PEDA 스타일 명령도 제공합니다.
이 Python 확장은 WinDbg 환경에서 GDB-PEDA에서 영감을 받은 디버깅 경험을 제공하여 익스플로잇 개발과 바이너리 분석을 더 편리하게 만듭니다.
주로 WinDbg에서 PEDA 스타일 디버깅 워크플로를 선호하는 Windows 익스플로잇 개발자와 리버스 엔지니어에게 유용합니다.
