Este proyecto es una extensión de interfaz de depurador al estilo PEDA para WinDbg, creada con el motor de scripting Python pykd.
Muestra registros, código desensamblado cerca del contador de programa y el contenido de la pila con desreferencia inteligente en cada paso o traza, junto con comandos al estilo PEDA para inspección de memoria y búsqueda de símbolos.
La extensión de Python ofrece una experiencia de depuración inspirada en GDB-PEDA dentro del entorno WinDbg, para un desarrollo de exploits y un análisis binario más cómodos.
Resulta especialmente útil para desarrolladores de exploits de Windows e ingenieros inversos que prefieren un flujo de depuración al estilo PEDA en WinDbg.
