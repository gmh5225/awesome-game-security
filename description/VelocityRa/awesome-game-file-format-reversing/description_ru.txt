Awesome Game File Format Reversing — это курируемая подборка документации, кода, инструментов и ресурсов по реверс-инжинирингу и работе с форматами файлов видеоигр. Она охватывает специализированные, часто недокументированные форматы моделей, текстур, анимаций, аудио, архивов, скриптов и данных уровней во многих играх и движках. Список систематизирует общие инструменты — просмотрщики ассетов, распаковщики архивов, hex-редакторы и утилиты анализа форматов, а также разделы по движкам Unity, Unreal, Source, CryEngine и другим. Также приведены ссылки на сообщества, вики, учебные материалы и документацию middleware. Основная аудитория — разработчики, моддеры и реверс-инженеры, которым нужно понимать, извлекать, конвертировать или иным образом работать с проприетарными игровыми ассетами.
