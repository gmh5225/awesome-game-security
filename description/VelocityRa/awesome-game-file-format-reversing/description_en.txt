Awesome Game File Format Reversing is a curated collection of documentation, code, tools, and resources for reverse engineering and working with video game file formats. It covers specialized, often undocumented formats for models, textures, animations, audio, archives, scripts, and level data across many games and engines. The list organizes general tools such as asset viewers, archive extractors, hex editors, and format-analysis utilities, plus engine-specific sections for Unity, Unreal, Source, CryEngine, and others. It also points to communities, wikis, learning resources, and middleware documentation. Its primary audience is developers, modders, and reverse engineers who need to understand, extract, convert, or otherwise work with proprietary game assets.
