DoubleDataPointer — это proof of concept коммуникации с ядром Windows, использующий канал с двойным указателем между пользовательским режимом и вручную отображённым драйвером. Реализация в основном на C++ и предоставляет примитивы для чтения и записи виртуальной и физической памяти из контекста ядра. Также демонстрируются техники, ориентированные на скрытность: очистка номеров кадровых страниц и сокращение артефактов, связанных с пулом, с документированием рисков обнаружения. Проект предназначен для экспериментов по обходу античита и низкоуровневых исследований безопасности игр.
