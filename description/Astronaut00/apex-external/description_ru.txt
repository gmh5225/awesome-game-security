ApexCheat — проект внешнего чита и SDK для Apex Legends. Он в основном написан на C++ и включает меню и overlay-конвейер на ImGui с модулями для glow-эффектов, логики аимбота и обработки сущностей через память. В репозитории также есть компонент дампера оффсетов на Rust для поддержания оффсетов игровых структур. Проект предназначен для прототипирования читов и экспериментов по реверс-инжинирингу; в прилагаемых заметках реализация описана как устаревшая.
