ApexCheat è un progetto di cheat esterno e SDK per Apex Legends. È scritto principalmente in C++ e include un menu e una pipeline di overlay basati su ImGui, con moduli per effetti glow, logica di aimbot e gestione delle entità basata sulla memoria. Il repository include anche un componente dumper di offset in Rust per aiutare a mantenere gli offset delle strutture di gioco. È pensato per il prototyping di cheat e esperimenti di reverse engineering, e le note incluse descrivono l'implementazione come obsoleta.
