ApexCheat es un proyecto de cheat externo y SDK para Apex Legends. Está escrito principalmente en C++ e incluye un menú y un pipeline de overlay basados en ImGui, con módulos para efectos glow, lógica de aimbot y manejo de entidades impulsado por memoria. El repositorio también incluye un componente dumper de offsets en Rust para ayudar a mantener los offsets de las estructuras del juego. Está pensado para prototipado de cheats y experimentos de ingeniería inversa, y las notas incluidas describen la implementación como obsoleta.
