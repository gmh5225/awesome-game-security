Ce projet est un plugin IDA Pro qui s’appuie sur des grands modèles de langage pour assister les workflows d’analyse binaire.
Il peut expliquer le comportement des fonctions, renommer des variables, tenter des reconstructions Python de petites routines, et lancer des contrôles orientés vulnérabilités à partir des vues décompilées.
Le plugin est écrit en Python et s’intègre aux API compatibles OpenAI, tout en proposant un mode automatisé qui parcourt les arbres de fonctions et résume les résultats.
Il s’adresse aux reverse engineers et aux chercheurs en sécurité des jeux qui souhaitent un triage assisté par l’IA dans IDA.
