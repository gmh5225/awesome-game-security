Dieses Projekt ist ein IDA-Pro-Plugin, das große Sprachmodelle zur Unterstützung von Binäranalyse-Workflows einsetzt.
Es kann Funktionsverhalten erklären, Variablen umbenennen, Python-Rekonstruktionen kleiner Routinen versuchen und vulnerabilitätsorientierte Prüfungen aus dekompilierten Ansichten ausführen.
Das Plugin ist in Python geschrieben und integriert sich mit OpenAI-kompatiblen APIs; zudem bietet es einen Automatikmodus, der Funktionsbäume durchläuft und Ergebnisse zusammenfasst.
Es richtet sich an Reverse Engineers und Game-Security-Forscher, die eine KI-gestützte Triage in IDA wünschen.
