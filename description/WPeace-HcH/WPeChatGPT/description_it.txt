Questo progetto è un plugin per IDA Pro che usa grandi modelli linguistici per assistere i flussi di lavoro di analisi binaria.
Può spiegare il comportamento delle funzioni, rinominare variabili, tentare ricostruzioni in Python di piccole routine ed eseguire controlli orientati alle vulnerabilità dalle viste decompilate.
Il plugin è scritto in Python e si integra con API compatibili con OpenAI, offrendo inoltre una modalità automatica che attraversa gli alberi di funzioni e riepiloga i risultati.
È rivolto a reverse engineer e ricercatori di sicurezza nei videogiochi che desiderano un triage assistito dall’IA all’interno di IDA.
