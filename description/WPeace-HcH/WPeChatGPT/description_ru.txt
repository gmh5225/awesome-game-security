Этот проект — плагин для IDA Pro, который использует большие языковые модели для помощи в рабочих процессах анализа бинарного кода.
Он может объяснять поведение функций, переименовывать переменные, пытаться восстанавливать небольшие процедуры на Python и выполнять проверки, ориентированные на уязвимости, по декомпилированным представлениям.
Плагин написан на Python и интегрируется с API, совместимыми с OpenAI, а также предоставляет автоматический режим, который обходит деревья функций и суммирует находки.
Он предназначен для реверс-инженеров и исследователей игровой безопасности, которым нужен AI-ассистированный триаж внутри IDA.
