Questo progetto è GamesoftACS, un'implementazione di sistema anti-cheat per server di gioco. Fornisce rilevamento dei cheat lato server, inclusa la rilevazione di speed hack, teletrasporto, validazione di danni anomali, controlli sulla manipolazione dell'inventario e verifica dell'integrità del client. Il sistema monitora i pattern di gameplay e segnala le attività sospette per la revisione. È rivolto agli sviluppatori di server di gioco che implementano una protezione anti-cheat personalizzata per i giochi multiplayer.
