Este proyecto es GamesoftACS, una implementación de sistema anticheat para servidores de juegos. Ofrece detección de trampas del lado del servidor, incluyendo detección de speed hacks, detección de teletransporte, validación de daño anormal, comprobaciones de manipulación del inventario y verificación de integridad del cliente. El sistema monitoriza los patrones de juego y marca la actividad sospechosa para su revisión. Está dirigido a desarrolladores de servidores de juegos que implementan protección anticheat personalizada para juegos multijugador.
