Этот проект — GamesoftACS, реализация античит-системы для игровых серверов. Он обеспечивает серверное обнаружение читов, включая детекцию спидхаков, телепортов, проверку аномального урона, контроль манипуляций с инвентарём и проверку целостности клиента. Система отслеживает игровые паттерны и помечает подозрительную активность для проверки. Она предназначена для разработчиков игровых серверов, внедряющих собственную античит-защиту для многопользовательских игр.
