本プロジェクトは Termux です。ルート化なしで完全な Linux コマンドライン体験を提供する Android ターミナルエミュレータ兼 Linux 環境アプリです。独自のパッケージマネージャ（pkg/apt）を備え、コンパイラ、インタプリタ、エディタ、ネットワークツール、開発フレームワークなど数千のパッケージを利用できます。Java/C 製の本アプリはカスタム Linux sysroot とファイルシステム分離のための proot を使用し、SSH、Python、Node.js、Rust などの開発環境にも対応しています。Android 端末で Linux ライクな環境を求める開発者、セキュリティ研究者、パワーユーザー向けです。
