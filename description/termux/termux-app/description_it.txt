Questo progetto è Termux, un'app emulatore di terminale Android e ambiente Linux che offre un'esperienza completa di riga di comando Linux senza root. Include un proprio gestore di pacchetti (pkg/apt) con migliaia di pacchetti, tra cui compilatori, interpreti, editor, strumenti di rete e framework di sviluppo. L'applicazione Java/C usa un sysroot Linux personalizzato con proot per l'isolamento del file system e supporta SSH, Python, Node.js, Rust e altri ambienti di sviluppo. È rivolta a sviluppatori, ricercatori di sicurezza e power user che desiderano un ambiente simile a Linux su dispositivi Android.
