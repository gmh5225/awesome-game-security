Este proyecto es Termux, una aplicación de emulador de terminal Android y entorno Linux que ofrece una experiencia completa de línea de comandos Linux sin root. Incluye su propio gestor de paquetes (pkg/apt) con miles de paquetes, entre ellos compiladores, intérpretes, editores, herramientas de red y frameworks de desarrollo. La aplicación Java/C usa un sysroot Linux personalizado con proot para el aislamiento del sistema de archivos y admite SSH, Python, Node.js, Rust y otros entornos de desarrollo. Está dirigida a desarrolladores, investigadores de seguridad y usuarios avanzados que desean un entorno similar a Linux en dispositivos Android.
