이 프로젝트는 Termux로, 루팅 없이 완전한 Linux 명령줄 환경을 제공하는 Android 터미널 에뮬레이터 겸 Linux 환경 앱입니다. 자체 패키지 관리자(pkg/apt)를 포함하며, 컴파일러, 인터프리터, 에디터, 네트워킹 도구, 개발 프레임워크 등 수천 개의 패키지를 제공합니다. Java/C 애플리케이션은 파일 시스템 격리를 위한 proot과 함께 커스텀 Linux sysroot를 사용하며, SSH, Python, Node.js, Rust 및 기타 개발 환경을 지원합니다. Android 기기에서 Linux와 유사한 환경을 원하는 개발자, 보안 연구원, 파워 유저를 대상으로 합니다.
