Dieses Projekt ist Termux, eine Android-Terminalemulator- und Linux-Umgebungs-App, die ohne Root eine vollständige Linux-Kommandozeilen-Erfahrung bietet. Sie verfügt über einen eigenen Paketmanager (pkg/apt) mit Tausenden von Paketen, darunter Compiler, Interpreter, Editoren, Netzwerkwerkzeuge und Entwicklungsframeworks. Die Java/C-Anwendung nutzt ein eigenes Linux-Sysroot mit proot zur Dateisystemisolation und unterstützt SSH, Python, Node.js, Rust sowie weitere Entwicklungsumgebungen. Sie richtet sich an Entwickler, Sicherheitsforscher und Power-User, die eine Linux-ähnliche Umgebung auf Android-Geräten wünschen.
