Этот проект — Termux, приложение-эмулятор терминала Android и среда Linux, обеспечивающая полноценный опыт командной строки Linux без root-прав. Оно включает собственный менеджер пакетов (pkg/apt) с тысячами пакетов: компиляторы, интерпретаторы, редакторы, сетевые инструменты и фреймворки разработки. Приложение на Java/C использует собственный Linux sysroot с proot для изоляции файловой системы и поддерживает SSH, Python, Node.js, Rust и другие среды разработки. Ориентировано на разработчиков, исследователей безопасности и опытных пользователей, которым нужна Linux-подобная среда на устройствах Android.
