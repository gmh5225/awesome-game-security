Dieses Projekt ist eine plattformübergreifende Low-Level-3D-Grafikbibliothek und ein Rendering-Framework. Es bietet eine einheitliche API für Direct3D 11/12, Vulkan, Metal, OpenGL, OpenGLES, WebGL und WebGPU und nutzt HLSL als gemeinsame Shadersprache. Die Engine legt Wert auf modulares C++-Design, multithreaded Command- und Resource-Workflows, erweiterte Rendering-Funktionen sowie Tooling rund um das Render-State-Management. Sie wird vor allem von Spiele-Engine- und Visualisierungsentwicklern genutzt, einschließlich Sicherheitsforschern, die portable Grafikinfrastruktur für Instrumentierung oder Overlays benötigen.
