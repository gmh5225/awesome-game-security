本プロジェクトは、クロスプラットフォームの低レベル 3D グラフィックスライブラリおよびレンダリングフレームワークです。Direct3D 11/12、Vulkan、Metal、OpenGL、OpenGLES、WebGL、WebGPU に対して統一 API を提供し、共通シェーダー言語として HLSL を使用します。エンジンはモジュール化された C++ 設計、マルチスレッドのコマンド／リソースワークフロー、高度なレンダリング機能、およびレンダー状態管理まわりのツールを重視しています。主にゲームエンジンやビジュアライゼーションの開発者に利用され、計測やオーバーレイ向けのポータブルなグラフィックス基盤を必要とするセキュリティ研究者にも用いられます。
