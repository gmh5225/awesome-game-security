Este proyecto es una biblioteca gráfica 3D de bajo nivel y un framework de renderizado multiplataforma. Ofrece una API unificada para Direct3D 11/12, Vulkan, Metal, OpenGL, OpenGLES, WebGL y WebGPU, con HLSL como lenguaje de shaders común. El motor enfatiza un diseño modular en C++, flujos de trabajo multithread de comandos y recursos, funciones avanzadas de renderizado y herramientas para la gestión del estado de render. Lo usan principalmente desarrolladores de motores de juego y visualización, incluidos investigadores de seguridad que necesitan infraestructura gráfica portable para instrumentación o overlays.
