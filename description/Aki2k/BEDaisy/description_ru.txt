Этот проект — proof of concept по реверс-инжинирингу, сфокусированный на драйвере ядра античита BattlEye bedaisy.sys. Он написан на C++ и демонстрирует техники перехвата с помощью image load callbacks и IAT-хукинга вокруг MmGetSystemRoutineAddress. Код рассматривает получение контроля над последующими путями выполнения античита и включает эксперименты, связанные с APC. Предназначен для исследователей, изучающих поведение ядерных античитов, поверхности обхода и возможности защитного hardening.
