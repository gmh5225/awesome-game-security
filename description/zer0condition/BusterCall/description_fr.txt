Ce projet est un outil de recherche sur le noyau Windows qui présente des techniques pour détecter et désactiver les callbacks et routines de notification en mode noyau. Il énumère les callbacks enregistrés pour les processus, les threads, le chargement d'images et le registre, identifie leurs pilotes propriétaires, et peut les supprimer ou les patcher de manière sélective. Le pilote en C offre un aperçu de la façon dont les systèmes anti-triche enregistrent les callbacks noyau et dont les attaquants tentent de les désactiver. Il s'adresse aux chercheurs en sécurité noyau étudiant la détection basée sur les callbacks et son contournement.
