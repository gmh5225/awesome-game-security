Этот проект — минимальный гипервизор Intel VT-x для Windows, написанный на C. Он настраивает режим VMX root, конфигурирует VMCS (Virtual Machine Control Structure), обрабатывает VM exit для CPUID, доступа к MSR и операций с управляющими регистрами, а также виртуализирует работающую операционную систему. Драйвер работает как гипервизор типа 2 под существующей ОС. Он предназначен для исследователей ядра, изучающих разработку гипервизоров Intel VT-x и внутреннее устройство аппаратной виртуализации.
