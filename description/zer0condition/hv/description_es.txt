Este proyecto es un hipervisor Intel VT-x mínimo para Windows escrito en C. Configura el modo VMX root, prepara la VMCS (Virtual Machine Control Structure), gestiona las VM exits de CPUID, acceso a MSR y operaciones sobre registros de control, y virtualiza el sistema operativo en ejecución. El controlador opera como un hipervisor de tipo 2 debajo del sistema operativo existente. Está dirigido a investigadores de kernel que aprenden el desarrollo de hipervisores Intel VT-x y estudian los internos de la virtualización asistida por hardware.
