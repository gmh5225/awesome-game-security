Этот проект — скрытный гипервизор Intel VT-x Type-2, реализованный как драйвер ядра Windows x64 WDM на языке C. Он настраивает работу VMX с управлением EPT, обрабатывает VM exit для событий вроде CPUID и доступа к управляющим регистрам, распространяет виртуализацию процессора на все ядра и включает меры против обнаружения: кэширование результатов CPUID, скрытие CR4.VMXE, компенсацию смещения TSC и приватный host CR3, чтобы не раскрывать таблицы страниц гипервизора. Предназначен для исследователей безопасности ядра, изучающих построение лёгких гипервизоров, обход обнаружения гипервизоров античитами и внутреннее устройство аппаратной виртуализации в Windows.
