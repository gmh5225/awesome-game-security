本プロジェクトは、C 言語による Windows x64 WDM カーネルドライバーとして実装された、ステルス型 Intel VT-x Type-2 ハイパーバイザーです。EPT 管理付きの VMX 動作を構築し、CPUID や制御レジスタアクセスなどのイベントに対する VM Exit を処理し、全コアへのプロセッサ仮想化のブロードキャストを行い、CPUID 結果キャッシュ、CR4.VMXE の隠蔽、TSC オフセット補正、ハイパーバイザーのページテーブル漏洩を防ぐプライベートホスト CR3 などの検知回避策を備えています。軽量ハイパーバイザーの構築、アンチチート向けハイパーバイザー検知の迂回、および Windows 上のハードウェア支援仮想化の内部構造を研究するカーネルセキュリティ研究者を対象としています。
