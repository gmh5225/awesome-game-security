이 프로젝트는 C로 작성된 Windows x64 WDM 커널 드라이버로 구현된 스텔스 Intel VT-x Type-2 하이퍼바이저입니다. EPT 관리가 포함된 VMX 동작을 설정하고, CPUID 및 제어 레지스터 접근 등의 이벤트에 대한 VM Exit을 처리하며, 모든 코어에 프로세서 가상화를 브로드캐스트하고, CPUID 결과 캐싱, CR4.VMXE 숨김, TSC 오프셋 보정, 하이퍼바이저 페이지 테이블 유출을 막기 위한 프라이빗 호스트 CR3 등의 탐지 회피 조치를 포함합니다. 경량 하이퍼바이저 구축, 안티치트 하이퍼바이저 탐지 우회, Windows의 하드웨어 지원 가상화 내부를 연구하는 커널 보안 연구자를 대상으로 합니다.
