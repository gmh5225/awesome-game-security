Этот проект — proof of concept коммуникации в ядре Windows, выполняющий код ядра без создания видимых системных потоков. Он злоупотребляет существующими контекстами потоков ядра или таймерными callback'ами для запуска пользовательского кода, обходя античиты, которые перечисляют системные потоки в поисках вручную отображённых драйверов. Драйвер на C демонстрирует техники выполнения в ядре без создания потоков. Предназначен для исследователей ядра, изучающих скрытое выполнение кода и обход обнаружения потоков античитами.
