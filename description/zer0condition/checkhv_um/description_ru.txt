Этот проект — инструмент обнаружения гипервизоров в пользовательском режиме для Windows, который проверяет наличие гипервизоров Type-1 и Type-2 с помощью нескольких методов. Он тестирует бит hypervisor present в CPUID, обнаружение по таймингу через аномалии RDTSC/RDTSCP, сканирование артефактов VMCS и сопоставление известных сигнатур гипервизоров. Реализация на C полностью работает в пользовательском режиме и не требует драйверов. Предназначен для разработчиков античитов и исследователей безопасности, создающих средства обнаружения гипервизоров.
