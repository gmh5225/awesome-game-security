Questo progetto è una libreria di manipolazione della memoria del kernel Windows scritta in C che fornisce funzioni per leggere, scrivere e allocare memoria oltre i confini di processo dalla modalità kernel. Implementa l'accesso alla memoria dei processi tramite mapping MDL (Memory Descriptor List), traduzione della memoria fisica e attraversamento delle page table basato su CR3, aggirando le protezioni a livello di API standard. È rivolta ai ricercatori di sicurezza del kernel che studiano tecniche alternative di accesso alla memoria usate nei cheat in modalità kernel e nell'evasione degli anti-cheat.
