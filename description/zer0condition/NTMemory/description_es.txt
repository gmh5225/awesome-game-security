Este proyecto es una biblioteca de manipulación de memoria del kernel de Windows escrita en C que proporciona funciones para leer, escribir y asignar memoria a través de los límites de proceso desde el modo kernel. Implementa el acceso a la memoria de procesos mediante el mapeo MDL (Memory Descriptor List), la traducción de memoria física y el recorrido de tablas de páginas basado en CR3, eludiendo las protecciones a nivel de API estándar. Está dirigida a investigadores de seguridad del kernel que estudian técnicas alternativas de acceso a memoria usadas en cheats en modo kernel y en la evasión de anti-cheats.
