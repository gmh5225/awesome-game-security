Этот проект — библиотека манипуляции памятью ядра Windows на языке C, предоставляющая функции чтения, записи и выделения памяти через границы процессов из режима ядра. Она реализует доступ к памяти процессов через отображение MDL (Memory Descriptor List), трансляцию физической памяти и обход таблиц страниц на основе CR3, обходя стандартные защиты на уровне API. Предназначена для исследователей безопасности ядра, изучающих альтернативные техники доступа к памяти, применяемые в читах режима ядра и обходе античитов.
