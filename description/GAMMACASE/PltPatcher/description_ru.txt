PltPatcher — набор плагинов для IDA Pro, который восстанавливает записи Procedure Linkage Table, когда автоматический анализ не справляется. Написан на Python с использованием API IDAPython и в настоящее время ориентирован на бинарники ELF64. В пакет также входит сохранятель типов thunk, который во время декомпиляции удерживает выведенные типы аргументов вокруг extern-thunk. Основной сценарий использования — рабочие процессы реверс-инжиниринга бинарников, где важны точное восстановление PLT и thunk, в том числе исследования в области безопасности игр.
