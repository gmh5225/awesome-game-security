PltPatcher est un ensemble de plugins IDA Pro qui répare les entrées de la Procedure Linkage Table lorsque l'analyse automatique échoue. Il est écrit en Python avec les API IDAPython et cible actuellement les binaires ELF64. Le paquet inclut aussi un préserveur de types de thunks qui conserve les types d'arguments déduits autour des thunks extern pendant la décompilation. Son cas d'usage principal est les workflows de reverse engineering binaire où une récupération précise des PLT et des thunks est importante, y compris la recherche en sécurité des jeux.
