PltPatcher è un set di plugin per IDA Pro che ripara le voci della Procedure Linkage Table quando l'analisi automatica fallisce. È scritto in Python con le API IDAPython e attualmente punta a binari ELF64. Il pacchetto include anche un preservatore di tipi dei thunk che mantiene i tipi degli argomenti inferiti intorno ai thunk extern durante la decompilazione. Il caso d'uso principale sono i flussi di reverse engineering di binari in cui è importante un recupero accurato di PLT e thunk, inclusa la ricerca sulla sicurezza dei giochi.
