PltPatcher ist ein Set von IDA-Pro-Plugins, das Procedure-Linkage-Table-Einträge repariert, wenn die automatische Analyse fehlschlägt. Es ist in Python mit IDAPython-APIs geschrieben und zielt derzeit auf ELF64-Binärdateien ab. Das Paket enthält außerdem einen Thunk-Typ-Preserver, der während der Dekompilierung abgeleitete Argumenttypen um extern-Thunks herum beibehält. Der Hauptanwendungsfall sind Workflows im Binary-Reverse-Engineering, bei denen eine genaue Wiederherstellung von PLT und Thunks wichtig ist, einschließlich Forschung zur Spiele-Sicherheit.
