Este proyecto es una utilidad de Windows para cargar controladores de kernel sin firmar en sistemas modernos.
Es un cargador C/C++ actualizado que mejora la compatibilidad con Windows 10 y 11 y agiliza la carga de bytes de controladores vulnerables.
La herramienta ofrece un flujo de trabajo sencillo por línea de comandos para cargar y descargar controladores objetivo durante las pruebas.
Su caso de uso principal es la investigación de seguridad del kernel y experimentos de laboratorio controlados que requieren la ejecución de controladores sin firmar.
