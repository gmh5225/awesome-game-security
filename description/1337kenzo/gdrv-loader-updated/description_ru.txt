Этот проект — утилита Windows для загрузки неподписанных драйверов ядра на современных системах.
Это обновлённый загрузчик на C/C++, улучшающий совместимость с Windows 10 и 11 и упрощающий загрузку байтов уязвимых драйверов.
Инструмент предоставляет простой интерфейс командной строки для загрузки и выгрузки целевых драйверов во время тестирования.
Основной сценарий использования — исследования безопасности ядра и контролируемые лабораторные эксперименты, требующие выполнения неподписанных драйверов.
