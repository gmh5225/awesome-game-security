Este proyecto es el motor PlayCanvas, un motor de juegos WebGL/WebGPU de código abierto escrito en JavaScript. Proporciona una canalización de renderizado 3D completa con materiales PBR, mapeo de sombras, sistemas de partículas, integración física (Ammo.js), animación esquelética, audio y soporte XR. El motor se ejecuta directamente en navegadores web y se combina con el editor en la nube de PlayCanvas para el desarrollo colaborativo de juegos. Está dirigido a desarrolladores de juegos web y creadores de aplicaciones 3D interactivas orientadas a plataformas de navegador.
