Ce projet est le moteur PlayCanvas, un moteur de jeu WebGL/WebGPU open source écrit en JavaScript. Il fournit un pipeline de rendu 3D complet avec matériaux PBR, shadow mapping, systèmes de particules, intégration physique (Ammo.js), animation squelettique, audio et prise en charge XR. Le moteur s'exécute directement dans les navigateurs web et s'associe à l'éditeur cloud PlayCanvas pour le développement collaboratif de jeux. Il s'adresse aux développeurs de jeux web et aux créateurs d'applications 3D interactives ciblant les plateformes navigateur.
