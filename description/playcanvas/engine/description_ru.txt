Этот проект — движок PlayCanvas, открытый игровой движок WebGL/WebGPU на JavaScript. Он предоставляет полный конвейер 3D-рендеринга с материалами PBR, тенями, системами частиц, интеграцией физики (Ammo.js), скелетной анимацией, аудио и поддержкой XR. Движок работает прямо в веб-браузерах и сочетается с облачным редактором PlayCanvas для совместной разработки игр. Он предназначен для разработчиков веб-игр и создателей интерактивных 3D-приложений под браузерные платформы.
