Questo progetto è il motore PlayCanvas, un motore di gioco WebGL/WebGPU open source scritto in JavaScript. Fornisce una pipeline di rendering 3D completa con materiali PBR, shadow mapping, sistemi di particelle, integrazione fisica (Ammo.js), animazione scheletrica, audio e supporto XR. Il motore viene eseguito direttamente nei browser web e si abbina all'editor cloud PlayCanvas per lo sviluppo collaborativo di giochi. È rivolto agli sviluppatori di giochi web e ai creatori di applicazioni 3D interattive destinate alle piattaforme browser.
