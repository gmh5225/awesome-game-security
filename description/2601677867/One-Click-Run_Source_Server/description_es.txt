Este proyecto es un script shell de un clic para desplegar servidores dedicados del motor Source en Linux.
Automatiza comprobaciones del entorno, la instalación de dependencias, la configuración de SteamCMD y la descarga del servidor del juego para distribuciones como Debian, Ubuntu y CentOS.
El script también ayuda a instalar componentes habituales de servidores Source como Sourcemod y Metamod, y ofrece orientación para solucionar errores típicos de despliegue.
Su caso de uso principal es simplificar la configuración del servidor para principiantes y reducir el trabajo manual repetitivo para operadores experimentados de servidores comunitarios.
