Este proyecto es un editor 2D de código abierto todo en uno para pixel art, pintura, trabajo vectorial, animación y edición de imágenes. Está construido principalmente en C# sobre .NET, con una interfaz de escritorio basada en Avalonia y una amplia base de código modular. Sus funciones principales incluyen múltiples conjuntos de herramientas en un mismo lienzo, herramientas de animación con línea de tiempo y un flujo de renderizado basado en nodos para efectos y ediciones procedurales. Está pensado para artistas y desarrolladores de juegos indie que necesitan una herramienta flexible de creación de contenido 2D.
