Этот проект — универсальный open-source 2D-редактор для пиксель-арта, рисования, векторной графики, анимации и редактирования изображений. Он в основном написан на C# под .NET, с десктопным интерфейсом на Avalonia и большой модульной кодовой базой. Среди ключевых возможностей — несколько наборов инструментов на одном холсте, инструменты анимации на таймлайне и узловой пайплайн рендеринга для эффектов и процедурного редактирования. Редактор рассчитан на художников и инди-разработчиков игр, которым нужен гибкий инструмент для создания 2D-контента.
