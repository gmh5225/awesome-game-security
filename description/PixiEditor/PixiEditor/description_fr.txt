Ce projet est un éditeur 2D open source tout-en-un pour le pixel art, la peinture, le travail vectoriel, l'animation et l'édition d'images. Il est principalement écrit en C# sur .NET, avec une interface de bureau basée sur Avalonia et une vaste base de code modulaire. Ses fonctionnalités majeures incluent plusieurs jeux d'outils sur un même canevas, des outils d'animation sur timeline, et un flux de rendu basé sur des nœuds pour les effets et les modifications procédurales. Il s'adresse aux artistes et aux développeurs de jeux indépendants qui ont besoin d'un outil flexible de création de contenu 2D.
