Этот проект — небольшой Windows proof of concept для обнаружения песочниц вроде Any.Run с помощью проверок артефактов на хосте.
Он написан на C++ и сочетает проверки наличия папок, перечисление процессов, эвристики пользовательского профиля, а также логику поиска служб или драйверов.
Образец выдает результат обнаружения только когда несколько индикаторов совпадают с ожидаемыми признаками песочницы, демонстрируя многоуровневый подход к обходу.
Он предназначен для исследований в области анализа вредоносного ПО и для понимания того, как код, учитывающий песочницу, может уклоняться от автоматизированной проверки.
