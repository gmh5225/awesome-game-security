Este proyecto es una pequeña prueba de concepto para Windows que detecta entornos sandbox similares a Any.Run mediante comprobaciones de artefactos del host.
Está escrito en C++ y combina comprobaciones de presencia de carpetas, enumeración de procesos, heurísticas del perfil de usuario y lógica de búsqueda de servicios o controladores.
La muestra solo dispara un resultado de detección cuando varios indicadores coinciden con rasgos esperados de sandbox, demostrando un enfoque de evasión por capas.
Está pensado para la investigación en análisis de malware y para comprender cómo el código consciente del sandbox puede evadir la inspección automatizada.
