Dieses Projekt ist ein kleiner Windows-Proof-of-Concept zur Erkennung von Any.Run-ähnlichen Sandbox-Umgebungen anhand von Host-Artefaktprüfungen.
Es ist in C++ geschrieben und kombiniert Ordner-Existenzprüfungen, Prozessauflistung, Heuristiken zum Benutzerprofil sowie Logik zur Suche nach Diensten oder Treibern.
Das Beispiel meldet eine Erkennung erst, wenn mehrere Indikatoren mit erwarteten Sandbox-Merkmalen übereinstimmen, und demonstriert damit einen schichtweisen Umgehungsansatz.
Es dient der Malware-Analyseforschung und dem Verständnis, wie sandbox-bewusster Code automatisierte Inspektion umgehen kann.
