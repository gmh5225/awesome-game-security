Este proyecto es una lista curada de recursos del ecosistema Ghidra para ingeniería inversa. Se trata principalmente de una colección en Markdown de scripts, plugins, extensiones y material de referencia que abarca automatización, análisis de malware, diffing y mejoras del flujo de trabajo. Las herramientas listadas incluyen extensiones basadas en Java y Python, con opciones de uso headless e interactivo. Resulta especialmente útil para ingenieros inversos e investigadores de seguridad en juegos que buscan un mapa rápido de herramientas prácticas de Ghidra.
