Ce projet porte sur l'ancien code source Paradise SPRX BO2. Il s'agit en réalité d'une version plus récente, probablement la plus récemment publiée, car elle contient le code lié aux fenêtres. Il est surtout utile aux développeurs de jeux, aux reverse engineers et aux créateurs d'outils travaillant dans le domaine du développement / du code source de jeux.
