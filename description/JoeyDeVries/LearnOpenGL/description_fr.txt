Ce projet est le code source compagnon de la série de tutoriels LearnOpenGL, offrant des exemples C++ progressifs qui couvrent les fondamentaux d'OpenGL jusqu'aux techniques de rendu avancées. Il inclut des exemples pour l'éclairage, le chargement de modèles avec Assimp, le shadow mapping, le deferred shading, le PBR et le post-traitement, ainsi que les dépendances GLFW, GLEW et GLM. Il est surtout utile aux programmeurs graphiques et développeurs de jeux qui apprennent les techniques de rendu OpenGL modernes depuis le début.
