OMath es un framework moderno de plantillas C++ constexpr para cálculo matemático, físico y orientado a juegos de alto rendimiento, pensado para juegos, mods y desarrollo de cheats. Proporciona álgebra lineal, proyección mundo-a-pantalla, predicción de proyectiles y detección de colisiones (incluido GJK/EPA), con aceleración AVX2 opcional y sin dependencias de tiempo de ejecución de terceros obligatorias. La biblioteca incluye soporte predefinido de sistemas de coordenadas y cámaras para motores como Source, Unity, Unreal, Frostbite, IW Engine, CryEngine y OpenGL, además de utilidades como el escaneo de patrones PE/ELF/Mach-O y, de forma opcional, el hooking de DirectX/OpenGL y el scripting con Lua. Está dirigida a desarrolladores que crean overlays, ayudas de puntería, herramientas de ingeniería inversa y otros flujos de trabajo de seguridad o modding de juegos en Windows, macOS y Linux.
