OMath 是一個面向遊戲、模組與作弊開發的現代 C++ constexpr 模板框架，用於高效能數學、物理與遊戲相關計算。它提供線性代數、世界到螢幕投影、彈道預測以及碰撞偵測（含 GJK/EPA），可選 AVX2 加速，且無需第三方執行階段相依套件。該函式庫為 Source、Unity、Unreal、Frostbite、IW Engine、CryEngine 與 OpenGL 等引擎預先提供座標系與攝影機支援，並附帶 PE/ELF/Mach-O 特徵掃描，以及可選的 DirectX/OpenGL Hook 與 Lua 腳本功能。它面向在 Windows、macOS 與 Linux 上建置疊加層、瞄準輔助、逆向工程工具及其他遊戲安全或遊戲模組工作流程的開發者。
