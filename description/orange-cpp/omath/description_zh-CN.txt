OMath 是一个面向游戏、模组和作弊开发的现代 C++ constexpr 模板框架，用于高性能数学、物理与游戏相关计算。它提供线性代数、世界到屏幕投影、弹道预测以及碰撞检测（含 GJK/EPA），可选 AVX2 加速，且无需第三方运行时依赖。该库为 Source、Unity、Unreal、Frostbite、IW Engine、CryEngine 和 OpenGL 等引擎预置了坐标系与摄像机支持，并附带 PE/ELF/Mach-O 特征扫描，以及可选的 DirectX/OpenGL Hook 与 Lua 脚本功能。它面向在 Windows、macOS 和 Linux 上构建叠加层、瞄准辅助、逆向工程工具及其他游戏安全或游戏模组工作流的开发者。
