Este proyecto es una prueba de concepto en modo kernel y usuario que sustituye un puntero a función dentro de NtCompareSigningLevels para establecer comunicación encubierta con un driver. Está implementado en C y C++ como un driver de Windows y un controlador en espacio de usuario emparejados. El repositorio marca explícitamente el enfoque como abandonado porque PatchGuard lo hace inestable para un uso práctico. Resulta más útil como investigación histórica sobre la evasión de anti-cheat en superficies de hook de nivel de firma y sus limitaciones.
