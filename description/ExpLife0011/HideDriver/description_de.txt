Dieses Projekt ist ein Proof of Concept zur Verbergung von Windows-Kernel-Treibern für x64-Systeme. Es nutzt ETW-bezogene Symbolsuche, um MiProcessLoaderEntry zu finden, und entfernt DriverObject->DriverSection auf eine PatchGuard-bewusste Weise. Der Code löscht außerdem treiberkennzeichnende Artefakte nach dem Laden, indem er die Bereinigungslogik in einem separaten Thread ausführt. Der Hauptanwendungsfall ist die Anti-Cheat-Evasionsforschung und die Low-Level-Analyse forensischer Fußabdrücke von Treibern.
