Este proyecto es una prueba de concepto de ocultación de controladores del kernel de Windows para sistemas x64. Usa el descubrimiento de símbolos relacionados con ETW para localizar MiProcessLoaderEntry y elimina DriverObject->DriverSection de forma consciente de PatchGuard. El código también borra artefactos que identifican el controlador tras la carga ejecutando la lógica de limpieza en un hilo separado. Su caso de uso principal es la investigación sobre evasión de anti-cheat y el estudio a bajo nivel de huellas forenses de controladores.
