Ce projet est une preuve de concept de masquage de pilote noyau Windows pour systèmes x64. Il utilise la découverte de symboles liés à ETW pour localiser MiProcessLoaderEntry et retire DriverObject->DriverSection d’une manière compatible avec PatchGuard. Le code efface aussi les artefacts identifiant le pilote après le chargement en exécutant une logique de nettoyage dans un thread séparé. Son usage principal est la recherche sur le contournement d’anti-cheat et l’étude bas niveau des empreintes forensiques des pilotes.
