Questo progetto è LOVE (Love2D), un framework per giochi 2D gratuito e open source scritto in C++ con scripting Lua. Fornisce moduli per il rendering grafico (OpenGL/Metal), la riproduzione audio, la fisica (Box2D), la gestione delle finestre, l'input, il threading e l'accesso al filesystem tramite una semplice API Lua. I giochi vengono distribuiti come file archivio .love contenenti script Lua e asset. È rivolto a sviluppatori indie e appassionati che prototipano e creano giochi 2D con il minimo codice ripetitivo.
