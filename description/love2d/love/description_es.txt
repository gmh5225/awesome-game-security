Este proyecto es LOVE (Love2D), un framework de juegos 2D gratuito y de código abierto escrito en C++ con scripts en Lua. Ofrece módulos para renderizado gráfico (OpenGL/Metal), reproducción de audio, física (Box2D), ventanas, manejo de entrada, hilos y acceso al sistema de archivos mediante una API Lua sencilla. Los juegos se distribuyen como archivos de archivo .love con scripts Lua y recursos. Está pensado para desarrolladores indie y aficionados que prototipan y crean juegos 2D con el mínimo código repetitivo.
