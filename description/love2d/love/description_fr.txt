Ce projet est LOVE (Love2D), un framework de jeux 2D gratuit et open source écrit en C++ avec des scripts Lua. Il fournit des modules pour le rendu graphique (OpenGL/Metal), la lecture audio, la physique (Box2D), la gestion des fenêtres, les entrées, le multithreading et l'accès au système de fichiers via une API Lua simple. Les jeux sont distribués sous forme d'archives .love contenant scripts Lua et ressources. Il s'adresse aux développeurs indépendants et aux amateurs qui prototypent et créent des jeux 2D avec un minimum de code répétitif.
