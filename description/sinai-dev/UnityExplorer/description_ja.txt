本プロジェクトは UnityExplorer で、BepInEx または MelonLoader 経由で任意の Unity ゲームに注入できるランタイム向けのインスペクター兼デバッグツールです。リアルタイムのシーン階層ブラウザ、GameObject および Component インスペクター、C# REPL コンソール、マウス／フリーカム操作、クリップボード機能を備え、実行中にゲーム状態の調査と変更が可能です。C# コードベースはリフレクションと IL2CPP unhollowing を用い、Mono および IL2CPP の Unity ビルドの両方に対応します。Unity モッダー、ゲームハッカー、Unity 製ゲームをリバースエンジニアリングする研究者を対象としています。
