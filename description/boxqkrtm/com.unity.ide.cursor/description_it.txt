Questo progetto si concentra sull'integrazione dell'editor di codice per supportare Cursor come editor di codice per Unity.
È scritto principalmente in C# e C/C++ e ruota intorno a tooling per editor, sviluppo di plugin e Unity.
È soprattutto utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell'area game engine / game engine plugins:unity.
