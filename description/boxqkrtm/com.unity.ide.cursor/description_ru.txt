Этот проект посвящён интеграции редактора кода для поддержки Cursor в качестве редактора кода для Unity.
Он в основном написан на C# и C/C++ и сосредоточен на инструментах редактора, разработке плагинов и Unity.
В первую очередь полезен разработчикам игр, программистам движков и исследователям графики, работающим в области game engine / game engine plugins:unity.
