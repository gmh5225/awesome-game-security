Ce projet se concentre sur l'intégration d'éditeur de code pour prendre en charge Cursor comme éditeur de code pour Unity.
Il est principalement écrit en C# et C/C++ et s'articule autour des outils d'éditeur, du développement de plugins et d'Unity.
Il est surtout utile aux développeurs de jeux, programmeurs moteur et chercheurs en graphisme travaillant dans le domaine game engine / game engine plugins:unity.
