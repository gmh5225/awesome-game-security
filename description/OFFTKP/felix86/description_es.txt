felix86 es un emulador de espacio de usuario de Linux que ejecuta programas x86 y x86-64 en procesadores RISC-V. Utiliza un recompilador JIT, traducción SSE vectorizada y múltiples extensiones RISC-V para mejorar la compatibilidad y el rendimiento. El proyecto está implementado principalmente en C++ moderno e integra bibliotecas externas para decodificación, formato y pruebas. Está pensado para desarrolladores de emuladores e investigadores de sistemas de bajo nivel que exploran la ejecución entre arquitecturas.
