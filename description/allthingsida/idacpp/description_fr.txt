Ce projet est un plugin de scripting C++ pour IDA Pro qui intègre un interpréteur C++ basé sur Cling/Clang 20 directement dans le moteur de scripting d'IDA.
Il offre un accès natif à l'intégralité des API IDA SDK et Hex-Rays dans un REPL, en utilisant les mêmes types (ea_t, func_t*, cfunc_t*) que les plugins compilés, sans FFI ni bindings.
Le plugin inclut un éditeur de snippets, un onglet REPL dans la fenêtre de sortie, ainsi qu'une prise en charge multiplateforme pour Windows et macOS.
Il s'adresse principalement aux reverse engineers qui souhaitent un scripting C++ interactif dans IDA Pro avec un accès direct au niveau du SDK.
