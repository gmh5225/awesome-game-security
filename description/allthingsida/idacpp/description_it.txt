Questo progetto è un plugin di scripting C++ per IDA Pro che incorpora un interprete C++ basato su Cling/Clang 20 direttamente nel motore di scripting di IDA.
Fornisce accesso nativo alle API complete di IDA SDK e Hex-Rays in una REPL, usando gli stessi tipi (ea_t, func_t*, cfunc_t*) dei plugin compilati, senza FFI o binding.
Il plugin include un editor di snippet, una scheda REPL nella finestra di output e il supporto multipiattaforma per Windows e macOS.
È particolarmente utile per i reverse engineer che desiderano scripting C++ interattivo in IDA Pro con accesso diretto a livello SDK.
