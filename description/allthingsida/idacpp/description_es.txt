Este proyecto es un complemento de scripting en C++ para IDA Pro que integra un intérprete C++ basado en Cling/Clang 20 directamente en el motor de scripting de IDA.
Proporciona acceso nativo a las API completas de IDA SDK y Hex-Rays en un REPL, usando los mismos tipos (ea_t, func_t*, cfunc_t*) que los plugins compilados, sin FFI ni bindings.
Incluye un editor de snippets, una pestaña REPL en la ventana de salida y compatibilidad multiplataforma para Windows y macOS.
Resulta especialmente útil para ingenieros inversos que buscan scripting C++ interactivo en IDA Pro con acceso directo a nivel de SDK.
