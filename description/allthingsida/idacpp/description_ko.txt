이 프로젝트는 Cling/Clang 20 기반 C++ 인터프리터를 IDA 스크립팅 엔진에 직접 내장하는 IDA Pro용 C++ 스크립팅 플러그인입니다.
REPL에서 전체 IDA SDK와 Hex-Rays API에 네이티브로 접근할 수 있으며, FFI나 바인딩 없이 컴파일된 플러그인과 동일한 타입(ea_t, func_t*, cfunc_t*)을 사용합니다.
스니펫 편집기, 출력 창 REPL 탭을 포함하며 Windows와 macOS 크로스 플랫폼을 지원합니다.
주로 IDA Pro에서 SDK 수준으로 직접 접근하는 대화형 C++ 스크립팅을 원하는 리버스 엔지니어에게 유용합니다.
