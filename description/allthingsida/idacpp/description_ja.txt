本プロジェクトは、Cling/Clang 20 ベースの C++ インタープリタを IDA のスクリプティングエンジンに直接埋め込む、IDA Pro 向け C++ スクリプティングプラグインです。
REPL から IDA SDK および Hex-Rays API 全体へネイティブにアクセスでき、コンパイル済みプラグインと同じ型（ea_t、func_t*、cfunc_t*）を FFI やバインディングなしで利用できます。
スニペットエディタ、出力ウィンドウの REPL タブを備え、Windows と macOS のクロスプラットフォームに対応しています。
主に、IDA Pro で SDK レベルに直接アクセスできる対話的な C++ スクリプティングを求めるリバースエンジニア向けです。
