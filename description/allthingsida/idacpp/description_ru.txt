Этот проект — плагин C++-скриптов для IDA Pro, который встраивает интерпретатор C++ на базе Cling/Clang 20 непосредственно в движок скриптов IDA.
Он обеспечивает нативный доступ к полным API IDA SDK и Hex-Rays в REPL, используя те же типы (ea_t, func_t*, cfunc_t*), что и скомпилированные плагины, без FFI и привязок.
Плагин включает редактор сниппетов, вкладку REPL в окне вывода и кроссплатформенную поддержку Windows и macOS.
Он в первую очередь полезен инженерам по реверс-инжинирингу, которым нужно интерактивное C++-скриптование в IDA Pro с прямым доступом на уровне SDK.
