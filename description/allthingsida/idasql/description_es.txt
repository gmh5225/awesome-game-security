IDASQL expone las bases de datos de IDA como tablas SQL y añade una interfaz asistida por IA para consultas de ingeniería inversa en lenguaje natural. Puede ejecutarse como CLI independiente sobre archivos .i64 o como plugin dentro de IDA, y permite consultar funciones, cadenas, xrefs, tipos y más sin escribir scripts de IDAPython. El sistema también ofrece capacidades de consulta remota para que herramientas o agentes externos interactúen con una sesión de análisis activa. Está pensado para ingenieros inversos que buscan flujos de análisis binario rápidos y orientados a consultas.
