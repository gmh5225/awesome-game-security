IDASQL espone i database di IDA come tabelle SQL e aggiunge un’interfaccia assistita dall’IA per query di reverse engineering in linguaggio naturale. Può funzionare come CLI autonoma su file .i64 o come plugin in IDA, e consente di interrogare funzioni, stringhe, xref, tipi e altro senza scrivere script IDAPython. Il sistema offre anche query remote, così strumenti o agent esterni possono interagire con una sessione di analisi attiva. È pensato per reverse engineer che vogliono flussi di analisi binaria rapidi e guidati dalle query.
