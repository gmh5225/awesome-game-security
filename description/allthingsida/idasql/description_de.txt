IDASQL stellt IDA-Datenbanken als SQL-Tabellen bereit und ergänzt eine KI-gestützte Oberfläche für Reverse-Engineering-Abfragen in natürlicher Sprache. Es läuft als eigenständige CLI gegen .i64-Dateien oder als Plugin in IDA und unterstützt Abfragen zu Funktionen, Strings, Xrefs, Typen und mehr — ohne IDAPython-Skripte. Das System bietet zudem Remote-Abfragen, damit externe Tools oder Agents mit einer aktiven Analysesitzung interagieren können. Es richtet sich an Reverse Engineers, die schnelle, abfragegesteuerte Workflows zur Binäranalyse wollen.
