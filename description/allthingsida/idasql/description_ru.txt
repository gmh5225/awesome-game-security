IDASQL представляет базы данных IDA в виде SQL-таблиц и добавляет интерфейс с поддержкой ИИ для запросов по реверс-инжинирингу на естественном языке. Его можно запускать как автономный CLI для файлов .i64 или как плагин внутри IDA; поддерживаются запросы к функциям, строкам, xref, типам и другим сущностям без написания скриптов IDAPython. Система также обеспечивает удалённые запросы, чтобы внешние инструменты или агенты могли взаимодействовать с активной сессией анализа. Решение создано для реверс-инженеров, которым нужны быстрые, query-ориентированные рабочие процессы анализа бинарников.
