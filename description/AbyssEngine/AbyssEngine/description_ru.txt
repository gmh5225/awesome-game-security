Этот проект — clean-room-реимплементация классического движка action RPG на языке C. Он предоставляет кроссплатформенные компоненты движка: рендеринг, аудио, управление сценами, работу с MPQ-ассетами и декодирование медиа с помощью библиотек вроде SDL2, FFmpeg, zlib и libarchive. Кодовая база ориентирована на переносимость и расширяемость, а не на повторное использование исходного проприетарного игрового кода. Основная аудитория — разработчики движков и исследователи, связанные с моддингом, которым нужна прозрачная основа для изучения и расширения поведения устаревших игр.
