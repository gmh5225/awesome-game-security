Ce projet est une réimplémentation en clean-room d'un moteur de jeu de rôle d'action classique, écrite en C. Il fournit des composants de moteur multiplateformes tels que le rendu, l'audio, la gestion de scènes, le traitement des assets MPQ et le décodage multimédia à l'aide de bibliothèques comme SDL2, FFmpeg, zlib et libarchive. La base de code privilégie la portabilité et l'extensibilité plutôt que la réutilisation du code propriétaire original du jeu. Son public principal comprend les développeurs de moteurs et les chercheurs axés sur le modding qui souhaitent une fondation transparente pour étudier et étendre le comportement des jeux hérités.
