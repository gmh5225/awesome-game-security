Этот проект — подборка ресурсов и справочных материалов по разработке игр, собранная в README на китайском языке. Он объединяет ссылки на инструменты разработки, ресурсы смешанной реальности, фреймворки движков и связанные учебные материалы. Особенно полезен разработчикам игр, которым нужна комплексная отправная точка для поиска ресурсов и инструментов.
