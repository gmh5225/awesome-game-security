Este proyecto es una plantilla de menú de mods nativo para Android que usa Dear ImGui para el renderizado. Ofrece un framework C++ listo para usar para crear menús de modificación de juegos en Android, con renderizado ImGui mediante OpenGL ES, manejo de entrada táctil e integración JNI para enganchar funciones del juego. La plantilla incluye interruptores, deslizadores y campos de entrada para funciones de trucos. Está dirigida a modders de juegos Android que construyen menús de trucos visuales con inyección de código nativo.
