Ce projet est un modèle de menu de mod natif Android utilisant Dear ImGui pour le rendu. Il fournit un framework C++ prêt à l'emploi pour créer des menus de modification de jeux sur Android, avec un rendu ImGui via OpenGL ES, la gestion des entrées tactiles et une intégration JNI pour accrocher les fonctions du jeu. Le modèle inclut des interrupteurs, des curseurs et des champs de saisie pour les fonctionnalités de triche. Il s'adresse aux moddeurs de jeux Android qui construisent des menus de triche visuels avec injection de code natif.
