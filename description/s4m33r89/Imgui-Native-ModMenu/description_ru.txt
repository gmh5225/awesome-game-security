Этот проект — шаблон нативного мод-меню для Android с отрисовкой через Dear ImGui. Он даёт готовый C++-фреймворк для создания меню модификации игр на Android: отрисовка ImGui через OpenGL ES, обработка сенсорного ввода и интеграция JNI для хука игровых функций. В шаблоне есть переключатели, слайдеры и поля ввода для чит-функций. Он рассчитан на Android-моддеров, создающих визуальные чит-меню с внедрением нативного кода.
