Questo progetto è un template di menu mod nativo per Android che usa Dear ImGui per il rendering. Fornisce un framework C++ pronto all'uso per creare menu di modifica dei giochi su Android, con rendering ImGui tramite OpenGL ES, gestione dell'input touch e integrazione JNI per l'hooking delle funzioni di gioco. Il template include interruttori, cursori e campi di input per le funzioni cheat. È rivolto ai modder di giochi Android che costruiscono menu cheat visuali con iniezione di codice nativo.
