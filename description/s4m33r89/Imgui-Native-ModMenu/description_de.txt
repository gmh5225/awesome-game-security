Dieses Projekt ist eine native Android-Mod-Menü-Vorlage mit Dear ImGui für das Rendering. Es bietet ein einsatzbereites C++-Framework zum Erstellen von Spielmodifikationsmenüs unter Android, mit ImGui-Rendering über OpenGL ES, Touch-Eingabeverarbeitung und JNI-Integration zum Hooken von Spielfunktionen. Die Vorlage enthält Schalter, Schieberegler und Eingabefelder für Cheat-Funktionen. Sie richtet sich an Android-Spielmodder, die visuelle Cheat-Menüs mit nativer Code-Injection erstellen.
