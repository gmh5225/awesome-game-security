Dieses Projekt konzentriert sich auf einen Windows-Kernelmode-Treiber, der DLLs in jeden Prozess injiziert und systemweites Function Hooking durchführt.
Ein Kernel-Treiber stellt sicher, dass jeder Prozess erreichbar ist — einschließlich der allerersten Prozesse, die Windows noch vor dem Anmeldebildschirm startet.
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Injection unter Windows untersuchen.
