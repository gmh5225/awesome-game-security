Этот проект посвящён драйверу Windows в режиме ядра, который внедряет DLL в каждый процесс и выполняет системный function hooking.
Драйвер ядра обеспечивает доступ ко всем процессам, включая самые первые, запускаемые Windows ещё до экрана входа.
Он в основном полезен исследователям игровой безопасности и инженерам-реверсерам, изучающим атакующие техники в области cheat / injection под Windows.
