Babylon.js — это движок реального времени для 3D-рендеринга, предназначенный для создания интерактивной графики, игр и визуализаций в браузере. Кодовая база представляет собой TypeScript-монорепозиторий с основными модулями движка для сцен, анимаций, материалов, физики и постобработки, а также бэкендами WebGL и WebGPU и ресурсами шейдеров. Также включены дополнения: рендеринг атмосферы, поиск путей по navigation mesh, интеграция HTML-мешей и продвинутые конвейеры рендеринга, такие как SSAO, SSR и временное сглаживание. Проект ориентирован на веб- и игровых разработчиков, которым нужен полнофункциональный клиентский 3D-движок для игр, симуляций и иммерсивных впечатлений.
