Babylon.js は、ブラウザ上でインタラクティブなグラフィックス、ゲーム、可視化を構築するためのリアルタイム 3D レンダリングエンジンです。コードベースは TypeScript のモノレポで、シーン、アニメーション、マテリアル、物理、ポストプロセス向けのコアエンジンモジュールに加え、WebGL および WebGPU バックエンドとシェーダーアセットを含みます。さらに、大気レンダリング、ナビゲーションメッシュによる経路探索、HTML メッシュ統合、SSAO、SSR、テンポラルアンチエイリアシングなどの高度なレンダーパイプラインといったアドオンも提供します。本プロジェクトは、ゲーム、シミュレーション、没入型体験向けに、フル機能のクライアントサイド 3D エンジンを必要とする Web およびゲーム開発者を対象としています。
