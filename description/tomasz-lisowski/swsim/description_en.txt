This project focuses on pure-software USIM/UICC SIM card simulator — full APDU/SIM logic without physical hardware; PC/SC via swICC reader; pairs with SIMtrace2/cardem for phone testing; GSM/LTE auth, MILENAGE, private LTE/IoT and cellular security research.
It is primarily written in C and C/C++ and centers on modding.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / cellular / sim area.
