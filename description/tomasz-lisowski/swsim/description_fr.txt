Ce projet se concentre sur un simulateur de carte SIM USIM/UICC purement logiciel — logique APDU/SIM complète sans matériel physique ; PC/SC via le lecteur swICC ; s'associe à SIMtrace2/cardem pour les tests téléphoniques ; authentification GSM/LTE, MILENAGE, LTE/IoT privé et recherche en sécurité cellulaire.
Il est principalement écrit en C et C/C++ et centré sur le modding.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie étudiant les techniques offensives dans le domaine cheat / cellulaire / SIM.
