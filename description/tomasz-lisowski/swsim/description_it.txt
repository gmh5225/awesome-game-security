Questo progetto si concentra su un simulatore di scheda SIM USIM/UICC puramente software — logica APDU/SIM completa senza hardware fisico; PC/SC tramite il reader swICC; si abbina a SIMtrace2/cardem per test su telefono; autenticazione GSM/LTE, MILENAGE, LTE/IoT privato e ricerca sulla sicurezza cellulare.
È scritto principalmente in C e C/C++ ed è incentrato sul modding.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito cheat / cellulare / SIM.
