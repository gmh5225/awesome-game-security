Este proyecto se centra en un simulador de tarjeta SIM USIM/UICC puramente software: lógica APDU/SIM completa sin hardware físico; PC/SC mediante el lector swICC; se combina con SIMtrace2/cardem para pruebas con teléfonos; autenticación GSM/LTE, MILENAGE, LTE/IoT privado e investigación de seguridad celular.
Está escrito principalmente en C y C/C++ y se centra en el modding.
Resulta especialmente útil para investigadores de seguridad de juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / celular / SIM.
