Этот проект посвящён чисто программному симулятору SIM-карты USIM/UICC — полная логика APDU/SIM без физического оборудования; PC/SC через считыватель swICC; сочетается с SIMtrace2/cardem для тестирования на телефоне; аутентификация GSM/LTE, MILENAGE, частный LTE/IoT и исследования безопасности сотовой связи.
В основном написан на C и C/C++ и сосредоточен на modding.
В первую очередь полезен исследователям игровой безопасности и reverse-инженерам, изучающим атакующие техники в области cheat / cellular / SIM.
