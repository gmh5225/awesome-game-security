Este proyecto es un detector al estilo anti-cheat de kernel para encontrar controladores mapeados manualmente e hilos de kernel sospechosos. Aplica comprobaciones heurísticas como el recorrido de pila de hilos, la validación de puntos de entrada de hilos, el escaneo de big pools en busca de referencias anómalas y señales de detección de hooks IOCTL. La implementación está escrita en C++ como una base de código orientada a controladores de Windows, centrada en telemetría de bajo nivel. Está pensado para ingeniería anti-cheat e investigación de seguridad del kernel frente a implantes de trampas sigilosos.
