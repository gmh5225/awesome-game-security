Этот проект — плагин, добавляющий поддержку скриптов на Python 3 в отладчик x64dbg.
Он написан на C++ и предоставляет функции отладчика через удобные для Python обёртки, отражающие распространённые API SDK плагинов.
В репозитории есть примеры скриптов для доступа к памяти, работы с ассемблером, проверки модулей и взаимодействия с GUI, что упрощает автоматизацию повторяющихся задач реверс-инжиниринга.
Основной сценарий использования — реверс-инжиниринг и автоматизация отладчика для аналитиков, которым нужно скриптовать задачи отладки x86 и x64.
