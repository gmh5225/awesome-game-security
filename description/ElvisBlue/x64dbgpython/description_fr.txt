Ce projet est un plugin qui ajoute le support du scripting Python 3 au débogueur x64dbg.
Il est écrit en C++ et expose les fonctionnalités du débogueur via des wrappers adaptés à Python qui reflètent les API courantes du SDK de plugins.
Le dépôt inclut des scripts d'exemple pour l'accès mémoire, les tâches d'assemblage, l'inspection de modules et les interactions GUI, afin de faciliter l'automatisation des flux de reverse répétitifs.
Son principal cas d'usage est la rétro-ingénierie et l'automatisation du débogueur pour les analystes qui souhaitent scripter des tâches de débogage x86 et x64.
