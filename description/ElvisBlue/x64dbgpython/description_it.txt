Questo progetto è un plugin che aggiunge il supporto allo scripting in Python 3 al debugger x64dbg.
È scritto in C++ ed espone le funzionalità del debugger tramite wrapper adatti a Python che rispecchiano le API comuni dell'SDK dei plugin.
Il repository include script di esempio per l'accesso alla memoria, operazioni di assembly, ispezione dei moduli e interazioni con la GUI, semplificando l'automazione di flussi di reverse ripetitivi.
Il caso d'uso principale è il reverse engineering e l'automazione del debugger per analisti che vogliono scrivere script per attività di debugging x86 e x64.
