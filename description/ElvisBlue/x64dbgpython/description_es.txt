Este proyecto es un plugin que añade soporte de scripting en Python 3 al depurador x64dbg.
Está escrito en C++ y expone la funcionalidad del depurador mediante envoltorios amigables con Python que reflejan las API habituales del SDK de plugins.
El repositorio incluye scripts de ejemplo para acceso a memoria, tareas de ensamblado, inspección de módulos e interacciones con la GUI, lo que facilita la automatización de flujos de reverse repetitivos.
Su principal caso de uso es la ingeniería inversa y la automatización del depurador para analistas que quieren scriptear tareas de depuración x86 y x64.
