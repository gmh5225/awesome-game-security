Questo progetto è un plugin di IDA Hex-Rays che deoffusca la logica di controllo del flusso tipica dei campioni Emotet.
È scritto in Python e usa l’API microcode di IDA per identificare i registri del dispatcher, i valori di stato e i pattern di branch appiattiti.
Il plugin riscrive le transizioni tra blocchi, inserisce i target di salto corretti e ripulisce le istruzioni di dispatch residue per produrre pseudocodice più chiaro.
Il caso d’uso principale è il reverse engineering di malware e l’analisi di binari fortemente offuscati.
