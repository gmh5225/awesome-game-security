Ce projet est un plugin IDA Hex-Rays qui désobfusque la logique de flux de contrôle fréquemment rencontrée dans les échantillons Emotet.
Il est écrit en Python et utilise l’API microcode d’IDA pour identifier les registres de dispersion, les valeurs d’état et les motifs de branches aplaties.
Le plugin réécrit les transitions entre blocs, insère les cibles de saut corrigées et nettoie les instructions de dispatch restantes afin de produire un pseudocode plus clair.
Son cas d’usage principal est la rétro-ingénierie de malwares et l’analyse de binaires fortement obfuscés.
