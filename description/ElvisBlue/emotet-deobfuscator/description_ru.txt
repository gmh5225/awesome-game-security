Этот проект — плагин IDA Hex-Rays, который снимает обфускацию управляющей логики, типичную для образцов Emotet.
Он написан на Python и использует API микрокода IDA для выявления регистров диспетчера, значений состояния и сглаженных шаблонов ветвления.
Плагин переписывает переходы между блоками, вставляет исправленные цели переходов и очищает оставшиеся инструкции диспетчеризации, чтобы получить более понятный псевдокод.
Основное применение — реверс-инжиниринг вредоносного ПО и анализ сильно обфусцированных двоичных файлов.
