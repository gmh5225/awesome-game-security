Este proyecto es un complemento de IDA Hex-Rays que desofusca la lógica de flujo de control habitual en muestras de Emotet.
Está escrito en Python y usa la API de microcódigo de IDA para identificar registros del despachador, valores de estado y patrones de ramas aplanadas.
El complemento reescribe las transiciones entre bloques, inserta destinos de salto corregidos y limpia las instrucciones de despacho restantes para producir pseudocódigo más claro.
Su caso de uso principal es la ingeniería inversa de malware y el análisis de binarios muy ofuscados.
