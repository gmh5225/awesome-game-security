Этот проект — Trinity, фреймворк эксплуатации ядра Windows на C/C++, который связывает несколько примитивов для достижения произвольного выполнения кода в ядре. Он объединяет уязвимости драйверов, техники повреждения памяти и методы повышения привилегий в модульный пайплайн эксплуатации. Фреймворк демонстрирует построение надёжных цепочек эксплуатации ядра из отдельных примитивов уязвимостей. Он предназначен для разработчиков эксплойтов ядра и исследователей безопасности, изучающих методологию эксплуатации ядра Windows.
