Este proyecto es Trinity, un framework de explotación del kernel de Windows escrito en C/C++ que encadena múltiples primitivas para lograr ejecución arbitraria de código en el kernel. Combina vulnerabilidades de drivers, técnicas de corrupción de memoria y métodos de elevación de privilegios en un pipeline de explotación modular. El framework demuestra cómo construir cadenas fiables de explotación del kernel a partir de primitivas de vulnerabilidad individuales. Está dirigido a desarrolladores de exploits de kernel e investigadores de seguridad que estudian la metodología de explotación del kernel de Windows.
