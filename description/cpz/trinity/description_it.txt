Questo progetto è Trinity, un framework di exploitation del kernel Windows scritto in C/C++ che concatena più primitive per ottenere l'esecuzione arbitraria di codice nel kernel. Combina vulnerabilità dei driver, tecniche di corruzione della memoria e metodi di escalation dei privilegi in una pipeline di exploitation modulare. Il framework mostra come costruire catene affidabili di exploitation del kernel a partire da primitive di vulnerabilità individuali. È rivolto a sviluppatori di exploit kernel e ricercatori di sicurezza che studiano la metodologia di exploitation del kernel Windows.
