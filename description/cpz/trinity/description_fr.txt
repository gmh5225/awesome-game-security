Ce projet est Trinity, un framework d'exploitation du noyau Windows écrit en C/C++ qui enchaîne plusieurs primitives pour obtenir une exécution arbitraire de code noyau. Il combine vulnérabilités de pilotes, techniques de corruption mémoire et méthodes d'élévation de privilèges dans un pipeline d'exploitation modulaire. Le framework montre comment construire des chaînes d'exploitation noyau fiables à partir de primitives de vulnérabilité individuelles. Il s'adresse aux développeurs d'exploits noyau et aux chercheurs en sécurité qui étudient la méthodologie d'exploitation du noyau Windows.
