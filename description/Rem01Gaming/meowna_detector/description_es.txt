Este proyecto es un detector Android de prueba de concepto para módulos de ocultación de root que interrumpen los servicios de registro.
Comprueba indicadores como sockets logd ausentes y rastros de paquetes para señalar comportamientos sospechosos de manipulación del entorno.
La implementación es un pequeño programa en C compilado con makefiles de Android NDK.
Está pensado para pruebas de seguridad móvil y para estudiar cómo técnicas frágiles de ocultación de root pueden volverse detectables.
