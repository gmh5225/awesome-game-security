Ce projet est un détecteur Android de type preuve de concept pour les modules de masquage de root qui perturbent les services de journalisation.
Il vérifie des indicateurs tels que l'absence de sockets logd et des traces de paquets afin de signaler un comportement suspect de falsification de l'environnement.
L'implémentation est un petit programme C compilé avec les makefiles Android NDK.
Il est destiné aux tests de sécurité mobile et à l'étude de la façon dont des techniques fragiles de masquage de root peuvent devenir détectables.
