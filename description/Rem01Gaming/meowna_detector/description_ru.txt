Этот проект — proof-of-concept Android-детектор модулей скрытия root, нарушающих работу служб логирования.
Он проверяет такие признаки, как отсутствующие сокеты logd и следы пакетов, чтобы отмечать подозрительное вмешательство в окружение.
Реализация представляет собой небольшую программу на C, собираемую с помощью makefile Android NDK.
Предназначен для тестирования мобильной безопасности и изучения того, как хрупкие техники скрытия root могут становиться обнаруживаемыми.
