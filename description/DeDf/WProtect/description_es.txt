WProtect es un proyecto de protección de ejecutables de Windows que virtualiza bloques de código nativo seleccionados a un formato de máquina virtual personalizado. El código desensambla las instrucciones objetivo, construye bytecode de VM, redirige el código original con stubs de salto y añade una nueva sección PE que almacena la lógica transformada y las tablas en tiempo de ejecución. Está implementado principalmente en C++ e integra componentes como AsmJit, udis86 y utilidades personalizadas de análisis y reescritura de PE. El proyecto se orienta sobre todo a la investigación en protección de software, la práctica de ingeniería inversa y el análisis de técnicas anti manipulación basadas en VM.
