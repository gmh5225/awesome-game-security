WProtect est un projet de protection d'exécutables Windows qui virtualise des blocs de code natif sélectionnés dans un format de machine virtuelle personnalisé. Le code désassemble les instructions cibles, construit du bytecode VM, redirige le code d'origine avec des stubs de saut, et ajoute une nouvelle section PE qui stocke la logique transformée et les tables d'exécution. Il est principalement implémenté en C++ et intègre des composants tels qu'AsmJit, udis86, ainsi que des utilitaires personnalisés d'analyse et de réécriture PE. Le projet vise surtout la recherche en protection logicielle, la pratique du reverse engineering et l'analyse des techniques anti-falsification basées sur une VM.
