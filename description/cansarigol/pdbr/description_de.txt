Dieses Projekt ist pdbr, ein Werkzeug zum Extrahieren von Symbolinformationen aus Microsoft-PDB-(Program Database)-Debug-Symboldateien. Es parst PDB-Streams, um Funktionsnamen, Typdefinitionen, globale Variablen und Quelldateireferenzen wiederherzustellen, und gibt sie in einem lesbaren Format aus. Das Werkzeug arbeitet eigenständig und benötigt weder Visual Studio noch das DIA SDK. Es richtet sich an Reverse Engineers und Sicherheitsforscher, die Debug-Symboldaten aus PDB-Dateien für die Binäranalyse extrahieren müssen.
