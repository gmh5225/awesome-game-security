Этот проект — pdbr, инструмент для извлечения информации о символах из файлов отладочных символов Microsoft PDB (Program Database). Он разбирает потоки PDB, восстанавливая имена функций, определения типов, глобальные переменные и ссылки на исходные файлы, и выводит их в удобочитаемом формате. Инструмент работает автономно и не требует Visual Studio или DIA SDK. Он предназначен для специалистов по реверс-инжинирингу и исследователей безопасности, которым нужно извлекать данные отладочных символов из PDB-файлов для анализа бинарных файлов.
