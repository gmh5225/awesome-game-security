Este proyecto es pdbr, una herramienta para extraer información de símbolos de archivos de símbolos de depuración Microsoft PDB (Program Database). Analiza los streams PDB para recuperar nombres de funciones, definiciones de tipos, variables globales y referencias a archivos fuente, y los presenta en un formato legible. La herramienta funciona de forma independiente, sin necesidad de Visual Studio ni del DIA SDK. Está dirigida a ingenieros de ingeniería inversa e investigadores de seguridad que necesitan extraer datos de símbolos de depuración de archivos PDB para el análisis binario.
