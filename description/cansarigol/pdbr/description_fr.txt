Ce projet est pdbr, un outil d'extraction d'informations de symboles à partir des fichiers de symboles de débogage Microsoft PDB (Program Database). Il analyse les flux PDB pour récupérer les noms de fonctions, les définitions de types, les variables globales et les références aux fichiers sources, et les restitue dans un format lisible. L'outil fonctionne de manière autonome, sans Visual Studio ni le DIA SDK. Il s'adresse aux ingénieurs en reverse engineering et aux chercheurs en sécurité qui doivent extraire les données de symboles de débogage des fichiers PDB pour l'analyse binaire.
