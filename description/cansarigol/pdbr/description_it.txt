Questo progetto è pdbr, uno strumento per estrarre informazioni sui simboli dai file di simboli di debug Microsoft PDB (Program Database). Analizza gli stream PDB per recuperare nomi di funzioni, definizioni di tipi, variabili globali e riferimenti ai file sorgente, e li restituisce in un formato leggibile. Lo strumento funziona in modo autonomo, senza richiedere Visual Studio o il DIA SDK. È rivolto a reverse engineer e ricercatori di sicurezza che devono estrarre dati sui simboli di debug dai file PDB per l'analisi binaria.
