Ce projet est la version open source d'Overgrowth par Wolfire Games, un moteur de jeu 3D d'action et le jeu associé. Il comprend le code source complet du jeu avec des composants de moteur personnalisés pour le rendu, la physique, l'animation, l'IA, l'édition de niveaux et le scripting. Il est surtout utile aux chercheurs en moteurs de jeu et aux moddeurs qui étudient les internals des moteurs commerciaux et les bases de code de jeux ouvertes.
