Este proyecto es la versión de código abierto de Overgrowth de Wolfire Games, un motor de juego de acción 3D y el juego asociado. Incluye el código fuente completo del juego con componentes de motor personalizados para renderizado, física, animación, IA, edición de niveles y scripting. Resulta especialmente útil para investigadores de motores de juego y modders que estudian los internos de motores comerciales y bases de código de juegos abiertas.
