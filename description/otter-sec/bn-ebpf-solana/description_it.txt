Questo progetto è un plugin Binary Ninja per disassemblare e analizzare il bytecode Solana eBPF (SBF). Aggiunge a Binary Ninja il supporto per il set di istruzioni BPF personalizzato di Solana, consentendo la decompilazione e l'analisi degli smart contract (programmi) Solana compilati. Il plugin gestisce la codifica delle istruzioni specifica di SBF, il modello di memoria e le convenzioni di chiamata. È rivolto ad auditor e ricercatori di sicurezza blockchain che analizzano i binari dei programmi Solana alla ricerca di vulnerabilità.
