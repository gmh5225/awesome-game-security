Este proyecto es un plugin de Binary Ninja para desensamblar y analizar bytecode Solana eBPF (SBF). Añade a Binary Ninja soporte para el conjunto de instrucciones BPF personalizado de Solana, permitiendo la decompilación y el análisis de smart contracts (programas) Solana compilados. El plugin gestiona la codificación de instrucciones específica de SBF, el modelo de memoria y las convenciones de llamada. Está dirigido a auditores e investigadores de seguridad blockchain que analizan binarios de programas Solana en busca de vulnerabilidades.
