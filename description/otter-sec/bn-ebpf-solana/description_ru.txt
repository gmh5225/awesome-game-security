Этот проект — плагин Binary Ninja для дизассемблирования и анализа байткода Solana eBPF (SBF). Он добавляет в Binary Ninja поддержку пользовательского набора инструкций BPF для Solana, позволяя декомпилировать и анализировать скомпилированные смарт-контракты (программы) Solana. Плагин обрабатывает специфичное для SBF кодирование инструкций, модель памяти и соглашения о вызовах. Предназначен для аудиторов и исследователей безопасности блокчейна, анализирующих бинарные файлы программ Solana на предмет уязвимостей.
