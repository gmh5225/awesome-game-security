Este proyecto es un controlador de seguridad del kernel de Windows diseñado para bloquear técnicas de inyección de procesos. Protege los procesos objetivo filtrando derechos de handle peligrosos, como la escritura remota en memoria y los permisos de operación de los que dependen las cadenas de inyección. El código incluye fuentes del controlador en C++, pruebas y documentación sobre ataques como process hollowing, doppelgänging, ghosting e inyección de DLL. Resulta especialmente útil para investigación de seguridad defensiva y endurecimiento al estilo anti-cheat en Windows.
