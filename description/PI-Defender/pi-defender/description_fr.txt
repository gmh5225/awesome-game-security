Ce projet est un pilote de sécurité du noyau Windows conçu pour bloquer les techniques d'injection de processus. Il protège les processus cibles en filtrant les droits de handle dangereux, comme l'écriture mémoire à distance et les permissions d'opération dont dépendent les chaînes d'injection. La base de code comprend des sources de pilote en C++, des tests et une documentation couvrant des attaques telles que le process hollowing, le doppelgänging, le ghosting et l'injection de DLL. Il est principalement utile pour la recherche en sécurité défensive et le durcissement de type anti-cheat sous Windows.
