Este proyecto es un servidor Model Context Protocol (MCP) para Binary Ninja que expone funcionalidad de análisis binario a asistentes de IA. Permite a los LLM consultar desensamblado, descompilación, referencias cruzadas, listas de funciones e información de tipos desde bases de datos de Binary Ninja a través del protocolo MCP. El servidor en Python habilita flujos de ingeniería inversa impulsados por IA. Está dirigido a ingenieros de reverse engineering que integran Binary Ninja con herramientas de análisis basadas en IA y clientes compatibles con MCP.
