Этот проект — сервер Model Context Protocol (MCP) для Binary Ninja, предоставляющий ИИ-ассистентам функции анализа бинарных файлов. Он позволяет LLM запрашивать дизассемблирование, декомпиляцию, перекрёстные ссылки, списки функций и сведения о типах из баз данных Binary Ninja через протокол MCP. Python-сервер обеспечивает рабочие процессы реверс-инжиниринга на базе ИИ. Он ориентирован на специалистов по реверс-инжинирингу, интегрирующих Binary Ninja с ИИ-инструментами анализа и MCP-совместимыми клиентами.
