Ce projet est un serveur Model Context Protocol (MCP) pour Binary Ninja qui expose des fonctionnalités d'analyse binaire aux assistants IA. Il permet aux LLM d'interroger le désassemblage, la décompilation, les références croisées, les listes de fonctions et les informations de types depuis les bases Binary Ninja via le protocole MCP. Le serveur Python permet des workflows de rétro-ingénierie pilotés par l'IA. Il s'adresse aux ingénieurs en reverse engineering qui intègrent Binary Ninja avec des outils d'analyse basés sur l'IA et des clients compatibles MCP.
