Questo progetto è un server Model Context Protocol (MCP) per Binary Ninja che espone funzionalità di analisi binaria agli assistenti IA. Consente agli LLM di interrogare disassemblaggio, decompilazione, riferimenti incrociati, elenchi di funzioni e informazioni sui tipi dalle banche dati Binary Ninja tramite il protocollo MCP. Il server Python abilita flussi di reverse engineering guidati dall'IA. È rivolto agli ingegneri di reverse engineering che integrano Binary Ninja con strumenti di analisi basati sull'IA e client compatibili con MCP.
