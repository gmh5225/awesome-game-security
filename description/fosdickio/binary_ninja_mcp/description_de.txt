Dieses Projekt ist ein Model Context Protocol (MCP)-Server für Binary Ninja, der Binäranalysefunktionen für KI-Assistenten bereitstellt. Er ermöglicht LLMs, Disassembly, Dekompilierung, Querverweise, Funktionslisten und Typinformationen aus Binary-Ninja-Datenbanken über das MCP-Protokoll abzufragen. Der Python-Server unterstützt KI-gestützte Reverse-Engineering-Workflows. Er richtet sich an Reverse Engineers, die Binary Ninja mit KI-basierten Analysewerkzeugen und MCP-kompatiblen Clients integrieren.
