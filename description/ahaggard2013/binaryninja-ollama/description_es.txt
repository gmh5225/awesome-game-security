binaryninja-ollama es un complemento de Binary Ninja que usa un servidor Ollama alojado localmente para renombrar funciones y variables con ayuda de un LLM. Está escrito en Python y se integra directamente con flujos de trabajo HLIL para renombrado masivo y acciones de renombrado dirigidas. El complemento ofrece ajustes configurables de servidor, puerto y modelo para ejecutar el análisis sin conexión con modelos disponibles en local. Está pensado para ingenieros inversos que quieren un etiquetado semántico más rápido sin enviar binarios ni contexto de código a servicios en la nube de terceros.
