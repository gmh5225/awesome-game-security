binaryninja-ollama est un plugin Binary Ninja qui s'appuie sur un serveur Ollama hébergé en local pour renommer fonctions et variables avec l'aide d'un LLM. Écrit en Python, il s'intègre directement aux workflows HLIL pour le renommage en masse et le renommage ciblé. Le plugin propose des paramètres configurables pour le serveur, le port et le modèle, afin d'exécuter l'analyse hors ligne avec des modèles disponibles localement. Il s'adresse aux reverse engineers qui souhaitent un étiquetage sémantique plus rapide sans envoyer de binaires ni de contexte de code à des services cloud tiers.
