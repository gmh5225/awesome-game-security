binaryninja-ollama ist ein Binary-Ninja-Plugin, das einen lokal gehosteten Ollama-Server nutzt, um Funktionen und Variablen mit LLM-Unterstützung umzubenennen. Es ist in Python geschrieben und integriert sich direkt in HLIL-Workflows für Massen- und gezieltes Umbenennen. Das Plugin bietet konfigurierbare Server-, Port- und Modelleinstellungen, damit die Analyse offline mit lokal verfügbaren Modellen durchgeführt werden kann. Es richtet sich an Reverse Engineers, die schnellere semantische Benennung wollen, ohne Binärdateien oder Codekontext an Drittanbieter-Cloud-Dienste zu senden.
