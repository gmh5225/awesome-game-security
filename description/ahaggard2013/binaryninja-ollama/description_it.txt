binaryninja-ollama è un plugin di Binary Ninja che usa un server Ollama ospitato in locale per rinominare funzioni e variabili con l'assistenza di un LLM. È scritto in Python e si integra direttamente con i workflow HLIL per la rinominazione in blocco e azioni di rinominazione mirate. Il plugin offre impostazioni configurabili per server, porta e modello, così gli utenti possono eseguire l'analisi offline con modelli disponibili in locale. È pensato per reverse engineer che vogliono un'etichettatura semantica più rapida senza inviare binari o contesto di codice a servizi cloud di terze parti.
