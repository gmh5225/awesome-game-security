binaryninja-ollama — плагин для Binary Ninja, который использует локально размещённый сервер Ollama для переименования функций и переменных с помощью LLM. Он написан на Python и напрямую интегрируется в рабочие процессы HLIL для массового и целевого переименования. Плагин предоставляет настраиваемые параметры сервера, порта и модели, чтобы пользователи могли проводить анализ офлайн с локально доступными моделями. Он предназначен для специалистов по реверс-инжинирингу, которым нужна более быстрая семантическая разметка без передачи бинарников или контекста кода сторонним облачным сервисам.
