This project is E3 (Encrypt-Everything-Everywhere), a C++ homomorphic encryption framework from NYU Abu Dhabi's MoMA Lab for computing on encrypted operands.
It wraps FHE backends and exposes programmer-friendly encrypted variable types and operators so privacy-preserving code can be written without deep crypto expertise.
It is mainly useful for researchers and engineers exploring encrypted computation techniques relevant to protected game state and anti-cheat variable protection research.
