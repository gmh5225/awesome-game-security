Este proyecto es E3 (Encrypt-Everything-Everywhere), un framework C++ de cifrado homomórfico del MoMA Lab de NYU Abu Dhabi para computar sobre operandos cifrados.
Envuelve backends FHE y expone tipos de variables cifradas y operadores amigables para programadores, de modo que se pueda escribir código que preserve la privacidad sin un profundo conocimiento criptográfico.
Es especialmente útil para investigadores e ingenieros que exploran técnicas de computación cifrada relacionadas con el estado protegido del juego y la protección de variables anti-trampas.
