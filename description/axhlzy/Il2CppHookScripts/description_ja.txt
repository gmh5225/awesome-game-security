本プロジェクトは、Unity IL2CPP ゲーム向けの Frida ベース実行時解析・フックフレームワークであり、il2cpp-hooker という npm パッケージとして公開されています。
IL2CPP のメソッド、クラス、フィールド、インスタンスの実行時解析、バッチフックユーティリティ、関数戻り値の変更、オブジェクト階層の出力、メソッド情報付き逆アセンブル、および QBDI ベースの関数エミュレーションを提供します。
TypeScript コードベースには、JNI RegisterNatives のフック、Il2CppMethod の検査、ゲームオブジェクト詳細の表示、および MCP 対応版も含まれます。
主に Frida を用いて Unity IL2CPP ゲームの実行時解析と計装を行うモバイルゲームセキュリティ研究者およびリバースエンジニア向けです。
