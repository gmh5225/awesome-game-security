本專案是基於 Frida 的 Unity IL2CPP 遊戲執行階段解析與 Hook 框架，並以 il2cpp-hooker npm 套件形式發布。
它提供 IL2CPP 方法、類別、欄位與實例的執行階段解析，批次 Hook 工具、函式回傳值修改、物件階層列印、含方法資訊的反組譯，以及基於 QBDI 的函式模擬。
TypeScript 程式碼庫還包括 JNI RegisterNatives Hook、Il2CppMethod 檢查、遊戲物件詳情顯示，以及配套的 MCP 版本。
該專案主要面向使用 Frida 對 Unity IL2CPP 遊戲進行執行階段分析與插樁的手遊安全研究員與逆向工程師。
