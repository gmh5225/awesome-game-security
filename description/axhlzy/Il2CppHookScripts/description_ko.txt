이 프로젝트는 Unity IL2CPP 게임을 위한 Frida 기반 런타임 파싱 및 후킹 프레임워크로, il2cpp-hooker npm 패키지로 게시됩니다.
IL2CPP 메서드·클래스·필드·인스턴스의 런타임 파싱, 일괄 후킹 유틸리티, 함수 반환값 수정, 객체 계층 출력, 메서드 정보가 포함된 디스어셈블리, QBDI 기반 함수 에뮬레이션을 제공합니다.
TypeScript 코드베이스에는 JNI RegisterNatives 후킹, Il2CppMethod 검사, 게임 객체 상세 표시, MCP 동반 버전도 포함됩니다.
주로 Frida로 Unity IL2CPP 게임의 런타임 분석과 계측을 수행하는 모바일 게임 보안 연구원 및 리버스 엔지니어에게 유용합니다.
