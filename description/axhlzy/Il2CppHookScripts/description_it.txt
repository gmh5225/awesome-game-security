Questo progetto è un framework basato su Frida per il parsing e l'hooking a runtime di giochi Unity IL2CPP, pubblicato come pacchetto npm il2cpp-hooker.
Fornisce parsing a runtime di metodi, classi, campi e istanze IL2CPP, utility di hooking in batch, modifica del valore di ritorno delle funzioni, stampa della gerarchia degli oggetti, disassemblaggio con informazioni sui metodi ed emulazione di funzioni basata su QBDI.
Il codice TypeScript include l'hooking di JNI RegisterNatives, l'ispezione di Il2CppMethod, la visualizzazione dei dettagli degli oggetti di gioco e una versione companion MCP.
È soprattutto utile a ricercatori di sicurezza di giochi mobile e reverse engineer che eseguono analisi e strumentazione a runtime di giochi Unity IL2CPP con Frida.
