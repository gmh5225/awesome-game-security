本项目是一个基于 Python 的 ARM ELF 内联 Hook 工具，专门面向 Unity IL2CPP 共享库（libil2cpp.so）。
它使用 LIEF 进行 ELF 操作、Keystone 进行汇编、Capstone 进行反汇编，以合并代码段、修补 GOT 表，并注入带寄存器检查与 LDR 指令修复的内联 Hook。
该工具支持 Hook InitArray 函数、为 IDA 调试添加断点，以及调用封装后的 Android 原生函数（如 android_log_print 和 mprotect）。
它主要适用于对 Unity IL2CPP 游戏进行静态二进制修补与内联 Hook 注入的移动游戏安全研究人员。
