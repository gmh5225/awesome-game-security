本專案是一個基於 Python 的 ARM ELF 內聯 Hook 工具，專門針對 Unity IL2CPP 共用程式庫（libil2cpp.so）。
它使用 LIEF 進行 ELF 操作、Keystone 進行組譯、Capstone 進行反組譯，以合併程式碼區段、修補 GOT 表，並注入帶暫存器檢查與 LDR 指令修復的內聯 Hook。
該工具支援 Hook InitArray 函式、為 IDA 除錯新增中斷點，以及呼叫封裝後的 Android 原生函式（如 android_log_print 與 mprotect）。
它主要適用於對 Unity IL2CPP 遊戲進行靜態二進位修補與內聯 Hook 注入的行動遊戲安全研究人員。
