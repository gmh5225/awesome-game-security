이 프로젝트는 ARM ELF 바이너리를 위한 Python 기반 인라인 후킹 도구로, 특히 Unity IL2CPP 공유 라이브러리(libil2cpp.so)를 대상으로 합니다.
LIEF로 ELF를 조작하고, Keystone로 어셈블하며, Capstone으로 디스어셈블하여 코드 섹션을 병합하고 GOT 테이블을 패치하며, 레지스터 검사와 LDR 명령어 수정이 포함된 인라인 훅을 주입합니다.
InitArray 함수 후킹, IDA 디버깅용 브레이크포인트 추가, android_log_print 및 mprotect와 같은 래핑된 Android 네이티브 함수 호출을 지원합니다.
주로 Unity IL2CPP 게임에 대한 정적 바이너리 패칭과 인라인 훅 주입을 수행하는 모바일 게임 보안 연구자에게 유용합니다.
