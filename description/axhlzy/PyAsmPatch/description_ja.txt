本プロジェクトは、ARM ELF バイナリ向けの Python 製インラインフックツールで、特に Unity IL2CPP の共有ライブラリ（libil2cpp.so）を対象としています。
LIEF による ELF 操作、Keystone によるアセンブリ、Capstone によるディスアセンブリを用い、コードセクションの統合、GOT テーブルのパッチ、レジスタ検査と LDR 命令修正を伴うインラインフックの注入を行います。
InitArray 関数のフック、IDA デバッグ用ブレークポイントの追加、android_log_print や mprotect などのラップされた Android ネイティブ関数の呼び出しにも対応しています。
主に、Unity IL2CPP ゲームに対する静的バイナリパッチやインラインフック注入を行うモバイルゲームセキュリティ研究者向けです。
