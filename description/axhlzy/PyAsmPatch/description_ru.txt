Этот проект — инструмент inline-хукинга на Python для ARM ELF-бинарников, ориентированный в первую очередь на разделяемые библиотеки Unity IL2CPP (libil2cpp.so).
Он использует LIEF для работы с ELF, Keystone для ассемблирования и Capstone для дизассемблирования, чтобы объединять секции кода, патчить таблицы GOT и внедрять inline-хуки с проверкой регистров и исправлением инструкций LDR.
Инструмент поддерживает хукинг функций InitArray, добавление точек останова для отладки в IDA, а также вызов обёрнутых нативных функций Android вроде android_log_print и mprotect.
Он в основном полезен исследователям безопасности мобильных игр, которые выполняют статический бинарный патчинг и внедрение inline-хуков в игры на Unity IL2CPP.
