Este proyecto es una herramienta de inline hooking basada en Python para binarios ELF de ARM, orientada específicamente a bibliotecas compartidas de Unity IL2CPP (libil2cpp.so).
Usa LIEF para manipular ELF, Keystone para ensamblar y Capstone para desensamblar, con el fin de fusionar secciones de código, parchear tablas GOT e inyectar hooks inline con inspección de registros y corrección de instrucciones LDR.
La herramienta permite hacer hook de funciones InitArray, añadir puntos de interrupción para depuración con IDA y llamar a funciones nativas de Android encapsuladas como android_log_print y mprotect.
Resulta especialmente útil para investigadores de seguridad en juegos móviles que realizan parcheo binario estático e inyección de hooks inline en juegos Unity IL2CPP.
