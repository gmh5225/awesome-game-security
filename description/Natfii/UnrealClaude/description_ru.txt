Этот проект — плагин Unreal Engine, который интегрирует Claude AI как помощника разработчика через мост MCP. Он обеспечивает контекстную ИИ-помощь при работе с actors, анимациями, ассетами, blueprints, персонажами, materials, Enhanced Input, replication и разработкой Slate UI в Unreal Editor. В первую очередь полезен разработчикам Unreal Engine, которым нужны ИИ-поддерживаемые процессы разработки игр и автоматическая генерация кода для паттернов, специфичных для UE.
