Este proyecto es un plugin de Unreal Engine que integra Claude AI como copiloto de desarrollo mediante un puente MCP. Proporciona asistencia de IA con contexto para el desarrollo de actors, animaciones, assets, blueprints, personajes, materials, Enhanced Input, replication y la interfaz Slate UI dentro del Unreal Editor. Resulta especialmente útil para desarrolladores de Unreal Engine que buscan flujos de trabajo de desarrollo de juegos asistidos por IA y generación automática de código para patrones específicos de UE.
