Ce projet est un plugin Unreal Engine qui intègre Claude AI comme copilote de développement via un pont MCP. Il fournit une assistance IA contextuelle pour le développement d'actors, d'animations, d'assets, de blueprints, de personnages, de materials, d'Enhanced Input, de la replication et de l'interface Slate UI dans l'Unreal Editor. Il est surtout utile aux développeurs Unreal Engine qui recherchent des workflows de développement de jeux assistés par l'IA et une génération de code automatisée pour les motifs spécifiques à UE.
