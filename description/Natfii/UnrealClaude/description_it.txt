Questo progetto è un plugin di Unreal Engine che integra Claude AI come copilota di sviluppo tramite un bridge MCP. Fornisce assistenza IA contestuale per lo sviluppo di actor, animazioni, asset, blueprint, personaggi, material, Enhanced Input, replication e UI Slate all'interno dell'Unreal Editor. È soprattutto utile per gli sviluppatori Unreal Engine che cercano flussi di lavoro di game development assistiti dall'IA e generazione automatica di codice per pattern specifici di UE.
