Ce projet est xemu, un émulateur Xbox original open source basé sur un fork de QEMU. Il émule le GPU NV2A, l’audio MCPX, le chipset NForce et le processeur x86 Pentium III personnalisé de la Xbox pour exécuter des images de disques de jeux Xbox originaux (ISO/XISO). La base de code en C comprend l’émulation bas niveau (LLE) des composants matériels, le rendu GPU via OpenGL, la prise en charge des manettes USB et une interface SDL2 multiplateforme. Il s’adresse aux développeurs d’émulation, aux chercheurs en préservation de jeux et aux passionnés de rétro-gaming qui étudient le matériel Xbox original.
