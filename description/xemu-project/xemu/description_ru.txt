Этот проект — xemu, открытый эмулятор оригинальной Xbox на базе форка QEMU. Он эмулирует GPU NV2A, аудио MCPX, чипсет NForce и кастомный x86-процессор Pentium III Xbox для запуска образов игровых дисков оригинальной Xbox (ISO/XISO). Кодовая база на C включает LLE (низкоуровневую эмуляцию) аппаратных компонентов, рендеринг GPU на OpenGL, поддержку USB-контроллеров и кроссплатформенный интерфейс SDL2. Предназначен для разработчиков эмуляции, исследователей сохранения игр и любителей ретро-гейминга, изучающих внутреннее устройство аппаратного обеспечения оригинальной Xbox.
