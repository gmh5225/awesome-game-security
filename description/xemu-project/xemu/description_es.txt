Este proyecto es xemu, un emulador de Xbox original de código abierto basado en un fork de QEMU. Emula la GPU NV2A, el audio MCPX, el chipset NForce y la CPU x86 Pentium III personalizada de la Xbox para ejecutar imágenes de discos de juegos de Xbox original (ISO/XISO). La base de código en C incluye LLE (emulación de bajo nivel) de componentes de hardware, renderizado GPU basado en OpenGL, soporte de mandos USB y una interfaz SDL2 multiplataforma. Está dirigido a desarrolladores de emulación, investigadores de preservación de juegos y entusiastas del retro gaming que estudian el hardware interno de la Xbox original.
