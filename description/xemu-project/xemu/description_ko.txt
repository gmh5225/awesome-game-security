이 프로젝트는 QEMU 포크를 기반으로 한 오픈소스 오리지널 Xbox 에뮬레이터 xemu입니다. Xbox의 NV2A GPU, MCPX 오디오, NForce 칩셋, 커스텀 x86 Pentium III CPU를 에뮬레이트하여 오리지널 Xbox 게임 디스크 이미지(ISO/XISO)를 실행합니다. C 코드베이스에는 하드웨어 구성 요소의 LLE(저수준 에뮬레이션), OpenGL 기반 GPU 렌더링, USB 컨트롤러 지원, 크로스 플랫폼 SDL2 UI가 포함됩니다. 에뮬레이션 개발자, 게임 보존 연구자, 오리지널 Xbox 하드웨어 내부를 연구하는 레트로 게이밍 애호가를 대상으로 합니다.
