Este proyecto es una implementación de hooking del kernel de Windows que moderniza la compatibilidad de InfinityHook desde Windows 7 hasta builds más recientes de Windows 11. Está escrito en C/C++ para Visual Studio y se centra en mantener un comportamiento de hook fiable ante cambios de offsets del kernel específicos de cada versión. El código fuente enfatiza los internals de bajo nivel e incluye comentarios extensos para comprender los detalles de la implementación. Se usa principalmente por investigadores de seguridad del kernel y por profesionales de ingeniería inversa de anti-cheat o EDR.
