Dieses Projekt ist eine Windows-Kernel-Hooking-Implementierung, die die InfinityHook-Kompatibilität von Windows 7 bis zu neueren Windows-11-Builds modernisiert. Es ist in C/C++ für Visual Studio geschrieben und konzentriert sich darauf, zuverlässiges Hook-Verhalten trotz versionspezifischer Kernel-Offset-Änderungen aufrechtzuerhalten. Der Quellcode betont Low-Level-Internals und enthält ausführliche Kommentare zum Verständnis der Implementierungsdetails. Es wird hauptsächlich von Kernel-Sicherheitsforschern sowie von Anti-Cheat- oder EDR-Reverse-Engineering-Praktikern genutzt.
