Этот проект — реализация хукинга ядра Windows, модернизирующая совместимость InfinityHook от Windows 7 до более новых сборок Windows 11. Написан на C/C++ для Visual Studio и ориентирован на сохранение надёжного поведения хуков при изменении офсетов ядра, зависящих от версии. Исходный код уделяет внимание низкоуровневым внутренностям и содержит обширные комментарии для понимания деталей реализации. В основном используется исследователями безопасности ядра и специалистами по реверс-инжинирингу античитов или EDR.
