bloodhunt_External ist ein externes C++-Cheat-Projekt für Bloodhunt, das als Visual-Studio-Lösung erstellt wurde.
Es kombiniert eine ImGui- und DirectX9-Overlay-Schicht mit Speicherzugriffs-Helfern, Offsets und Konfigurations-Headern für das Runtime-Targeting.
Das Repository enthält außerdem mhyprot-bezogene Treiber-Utility-Komponenten, die Low-Level-Prozessinteraktionsmuster unterstützen, wie sie von externen Tools genutzt werden.
Der primäre Anwendungsfall ist die Game-Security-Forschung zu externen Cheat-Architekturen und Implementierungsdetails mit Fokus auf Anti-Cheat-Evasion.
