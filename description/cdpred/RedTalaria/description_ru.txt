Этот проект — плагин для Unreal Engine, который предоставляет общие deep link на конкретные части проекта: узлы Blueprint, ассеты и позиции в редакторе.
Он опирается на фреймворк обработки URL Hermes и добавляет конечные точки для копирования и открытия ссылок на узлы Blueprint, упрощая командную работу через Slack или другие мессенджеры.
Плагин на C++ совместим с UE 5.3 и UE 4.27 и добавляет пункты контекстного меню для создания URL фокуса и навигации из любого узла Blueprint.
Он в первую очередь полезен разработчикам Unreal Engine и игровым командам, которым нужны сценарии deep linking для обмена ассетами проекта и совместной работы.
