本プロジェクトは、Blueprint ノード、アセット、エディタ上の位置など、プロジェクト内の特定箇所への共有可能なディープリンクを提供する Unreal Engine プラグインです。
Hermes URL 処理フレームワークを基盤とし、Blueprint ノードへのリンクのコピーとオープン用エンドポイントを追加することで、Slack などのメッセージングツールを通じたチーム協業を可能にします。
この C++ プラグインは UE 5.3 および UE 4.27 と互換性があり、任意の Blueprint ノードからフォーカス用・ナビゲーション用 URL を生成するコンテキストメニュー項目を追加します。
プロジェクト資産の共有と協業のためのディープリンクワークフローを求める Unreal Engine 開発者やゲーム開発チームに特に役立ちます。
