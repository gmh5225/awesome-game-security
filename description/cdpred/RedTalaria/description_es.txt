Este proyecto es un plugin de Unreal Engine que proporciona enlaces profundos compartibles a partes concretas de un proyecto, como nodos Blueprint, assets y ubicaciones del editor.
Se basa en el framework de manejo de URL Hermes y añade endpoints para copiar y abrir enlaces a nodos Blueprint, lo que facilita la colaboración en equipo a través de Slack u otras herramientas de mensajería.
El plugin en C++ es compatible con UE 5.3 y UE 4.27, y añade opciones de menú contextual para generar URL de enfoque y navegación desde cualquier nodo Blueprint.
Resulta especialmente útil para desarrolladores de Unreal Engine y equipos de desarrollo de juegos que buscan flujos de deep linking para compartir assets del proyecto y colaborar.
