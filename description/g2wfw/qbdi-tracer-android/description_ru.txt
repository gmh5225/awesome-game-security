Фреймворк трассировки нативного кода Android на базе QBDI (QuarkslaB Dynamic binary Instrumentation) и библиотеки inline-хуков Dobby. Перехватывает загрузку разделяемых библиотек через хуки Android linker, инструментирует целевые функции для покомандной трассировки с захватом стека вызовов и включает утилиты сканирования памяти и поиска по шаблонам. Файлы toolchain CMake поддерживают кросс-компиляцию для Android, iOS и других целей ARM64.
