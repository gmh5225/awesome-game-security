Framework de trazado de código nativo de Android basado en QBDI (QuarkslaB Dynamic binary Instrumentation) y la biblioteca de hooks inline Dobby. Intercepta la carga de bibliotecas compartidas mediante hooks del linker de Android, instrumenta funciones objetivo para trazado por instrucción con captura de backtrace e incluye utilidades de escaneo de memoria y coincidencia de patrones. Los archivos de toolchain de CMake admiten la compilación cruzada para Android, iOS y otros objetivos ARM64.
