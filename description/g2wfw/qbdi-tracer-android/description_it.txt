Framework di tracciamento del codice nativo Android basato su QBDI (QuarkslaB Dynamic binary Instrumentation) e sulla libreria di hooking inline Dobby. Intercetta il caricamento delle librerie condivise tramite hook del linker Android, strumenta le funzioni target per il tracciamento a livello di istruzione con cattura del backtrace e include utility per la scansione della memoria e il pattern matching. I file di toolchain CMake supportano la cross-compilazione per Android, iOS e altri target ARM64.
