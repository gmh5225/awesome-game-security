Win32.Nebula — это proof-of-concept фреймворк упакованного и защищённого загрузчика модулей для Windows. Он написан в основном на C++ и разделяет лёгкую библиотеку загрузчика и утилиту сборки, которая патчит, шифрует и упаковывает бинарные файлы. В дизайне предусмотрены API в стиле SDK, концепция динамического менеджера служб и решения реализации, ориентированные на низкоуровневые внутренности Windows. Основной сценарий использования — исследование защиты ПО, архитектуры загрузчиков и антианалитических техник, применяемых в экосистемах вредоносного ПО и античита.
