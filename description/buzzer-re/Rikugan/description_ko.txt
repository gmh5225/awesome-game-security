이 프로젝트는 IDA Pro와 Binary Ninja를 위한 리버스 엔지니어링 AI 에이전트로, 다중 공급자 LLM 지원을 디스어셈블러 UI에 직접 통합합니다.
제너레이터 기반 에이전틱 루프와 스트리밍, 프로세스 내 도구 오케스트레이션, 자동 오류 복구, 다단계 워크플로용 플랜 모드, RE 환경을 벗어나지 않는 컨텍스트 관리를 제공합니다.
이 Python 플러그인은 클라우드 API와 로컬 Ollama를 포함한 여러 LLM 공급자를 지원하며, Ctrl+Shift+I로 채팅 인터페이스에 접근할 수 있습니다.
IDA Pro 또는 Binary Ninja 내에서 네이티브로 동작하는 바이너리 분석용 임베디드 AI 어시스턴트를 원하는 리버스 엔지니어에게 주로 유용합니다.
