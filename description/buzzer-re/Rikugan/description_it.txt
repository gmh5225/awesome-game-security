Questo progetto è un agente di IA per il reverse engineering su IDA Pro e Binary Ninja che integra il supporto LLM multi-provider direttamente nell'interfaccia del disassemblatore.
Offre un loop agentico basato su generatori con streaming, orchestrazione degli strumenti in-process, ripristino automatico degli errori, modalità plan per workflow multi-step e gestione del contesto senza lasciare l'ambiente di RE.
Il plugin Python supporta più provider LLM, incluse API cloud e installazioni locali di Ollama, con un'interfaccia chat accessibile tramite Ctrl+Shift+I.
È soprattutto utile per i reverse engineer che cercano un assistente IA integrato per l'analisi binaria che operi in modo nativo in IDA Pro o Binary Ninja.
