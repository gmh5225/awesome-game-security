Этот проект — ИИ-агент для реверс-инжиниринга в IDA Pro и Binary Ninja, который интегрирует поддержку LLM от нескольких провайдеров прямо в интерфейс дизассемблера.
Он включает агентный цикл на генераторах со стримингом, внутрипроцессную оркестрацию инструментов, автоматическое восстановление после ошибок, режим плана для многошаговых рабочих процессов и управление контекстом без выхода из среды RE.
Python-плагин поддерживает несколько LLM-провайдеров, включая облачные API и локальные установки Ollama, с чат-интерфейсом, доступным по Ctrl+Shift+I.
Он в основном полезен реверс-инженерам, которым нужен встроенный ИИ-ассистент для анализа бинарников, работающий нативно в IDA Pro или Binary Ninja.
