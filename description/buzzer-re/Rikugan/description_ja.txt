本プロジェクトは、IDA Pro と Binary Ninja 向けのリバースエンジニアリング AI エージェントで、複数プロバイダーの LLM サポートを逆アセンブラ UI に直接統合します。
ジェネレータベースのエージェンティックループとストリーミング、プロセス内ツールオーケストレーション、自動エラー復旧、多段階ワークフロー向けのプランモード、RE 環境を離れないコンテキスト管理を備えています。
この Python プラグインはクラウド API やローカルの Ollama を含む複数の LLM プロバイダーに対応し、Ctrl+Shift+I でチャット画面を開けます。
IDA Pro や Binary Ninja 内でネイティブに動作する、バイナリ解析向け組み込み AI アシスタントを求めるリバースエンジニアに主に有用です。
