Este proyecto es un agente de IA de ingeniería inversa para IDA Pro y Binary Ninja que integra soporte de LLM de múltiples proveedores directamente en la interfaz del desensamblador.
Incluye un bucle agéntico basado en generadores con streaming, orquestación de herramientas en proceso, recuperación automática de errores, modo plan para flujos de trabajo de varios pasos y gestión de contexto sin salir del entorno de RE.
El plugin de Python admite varios proveedores de LLM, incluidas APIs en la nube e instalaciones locales de Ollama, con una interfaz de chat accesible mediante Ctrl+Shift+I.
Resulta especialmente útil para ingenieros inversos que buscan un asistente de IA integrado para el análisis binario que funcione de forma nativa en IDA Pro o Binary Ninja.
