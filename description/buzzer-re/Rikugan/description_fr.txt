Ce projet est un agent d'IA de reverse engineering pour IDA Pro et Binary Ninja qui intègre le support de LLM multi-fournisseurs directement dans l'interface du désassembleur.
Il propose une boucle agentique basée sur des générateurs avec streaming, une orchestration d'outils en processus, une récupération automatique des erreurs, un mode plan pour les workflows multi-étapes, et une gestion du contexte sans quitter l'environnement de RE.
Le plugin Python prend en charge plusieurs fournisseurs de LLM, y compris les API cloud et les installations locales d'Ollama, avec une interface de chat accessible via Ctrl+Shift+I.
Il est surtout utile aux reverse engineers qui cherchent un assistant IA intégré pour l'analyse binaire, fonctionnant nativement dans IDA Pro ou Binary Ninja.
