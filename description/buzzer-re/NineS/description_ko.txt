이 프로젝트는 PlayStation 5용 ELF 인젝터로, ELF 파일을 원격 프로세스에 수동 매핑하고 원격 스레드에서 실행합니다.
포트 9033에서 TCP 서버를 실행하여 대상 프로세스 이름과 ELF 페이로드를 수신하고, 대상 PS5 프로세스 내에서 섹션 로딩, 재배치 처리, 스레드 생성을 포함한 수동 매핑을 수행합니다.
C 코드베이스는 John Törnblom의 PS5 SDK로 빌드되며, 인젝션 페이로드 전송용 Python 헬퍼 스크립트를 포함합니다.
주로 PS5 프로세스 인젝션, ELF 수동 매핑, FreeBSD 기반 게임 콘솔에서의 원격 코드 실행을 연구하는 콘솔 보안 연구자에게 유용합니다.
