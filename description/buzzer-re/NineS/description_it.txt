Questo progetto è un injector ELF per PlayStation 5 che mappa manualmente file ELF in processi remoti e li esegue in thread remoti.
Esegue un server TCP sulla porta 9033 che accetta un nome di processo di destinazione e un payload ELF, eseguendo il mapping manuale con caricamento delle sezioni, elaborazione delle relocation e creazione di thread nel processo PS5 di destinazione.
Il codice in C è compilato con l’SDK PS5 di John Törnblom e include uno script di supporto in Python per inviare i payload di iniezione.
È soprattutto utile per i ricercatori di sicurezza delle console che studiano l’iniezione di processi su PS5, il mapping manuale ELF e l’esecuzione remota di codice su console di gioco basate su FreeBSD.
