Ce projet est un injecteur ELF pour PlayStation 5 qui mappe manuellement des fichiers ELF dans des processus distants et les exécute dans des threads distants.
Il fait tourner un serveur TCP sur le port 9033 qui accepte un nom de processus cible et une charge utile ELF, puis effectue le mapping manuel, y compris le chargement des sections, le traitement des relocations et la création de threads dans le processus PS5 cible.
La base de code C est construite avec le SDK PS5 de John Törnblom et inclut un script d’aide Python pour envoyer les charges d’injection.
Il est surtout utile aux chercheurs en sécurité console qui étudient l’injection de processus PS5, le mapping manuel ELF et l’exécution de code à distance sur des consoles de jeu basées sur FreeBSD.
