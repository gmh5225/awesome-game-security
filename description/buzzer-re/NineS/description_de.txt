Dieses Projekt ist ein ELF-Injector für die PlayStation 5, der ELF-Dateien manuell in entfernte Prozesse mappt und sie in Remote-Threads ausführt.
Es betreibt einen TCP-Server auf Port 9033, der einen Zielprozessnamen und eine ELF-Payload entgegennimmt und manuelles Mapping einschließlich Section-Laden, Relocation-Verarbeitung und Thread-Erstellung im Ziel-PS5-Prozess durchführt.
Die C-Codebasis wird mit John Törnbloms PS5 SDK gebaut und enthält ein Python-Hilfsskript zum Senden von Injection-Payloads.
Es eignet sich vor allem für Console-Security-Forscher, die PS5-Prozessinjektion, manuelles ELF-Mapping und Remote Code Execution auf FreeBSD-basierten Spielkonsolen untersuchen.
