Este proyecto es un inyector ELF para PlayStation 5 que mapea manualmente archivos ELF en procesos remotos y los ejecuta en hilos remotos.
Ejecuta un servidor TCP en el puerto 9033 que acepta un nombre de proceso objetivo y una carga útil ELF, y realiza el mapeo manual, incluida la carga de secciones, el procesamiento de reubicaciones y la creación de hilos dentro del proceso PS5 objetivo.
La base de código en C se construye con el SDK de PS5 de John Törnblom e incluye un script auxiliar en Python para enviar cargas de inyección.
Es especialmente útil para investigadores de seguridad de consolas que estudian la inyección de procesos en PS5, el mapeo manual de ELF y la ejecución remota de código en consolas de juegos basadas en FreeBSD.
