本プロジェクトは PlayStation 5 向けの ELF インジェクタであり、ELF ファイルをリモートプロセスへ手動マップし、リモートスレッド上で実行します。
ポート 9033 で TCP サーバを起動し、対象プロセス名と ELF ペイロードを受け取り、対象 PS5 プロセス内でセクション読み込み、リロケーション処理、スレッド作成を含む手動マッピングを行います。
C コードベースは John Törnblom の PS5 SDK でビルドされ、インジェクション用ペイロード送信のための Python 補助スクリプトも含みます。
主に PS5 のプロセスインジェクション、ELF 手動マッピング、FreeBSD ベースのゲーム機におけるリモートコード実行を研究するコンソールセキュリティ研究者向けです。
