Este proyecto es un framework de hooking del kernel de Windows derivado de variantes anteriores de InfinityHook. Está implementado en C++ como una base de código orientada a controladores e incluye componentes de gestión de hooks de bajo nivel y de desensamblado de instrucciones. El repositorio enfatiza una mayor compatibilidad y estabilidad, con pruebas que abarcan múltiples versiones de Windows en entornos virtualizados. Su principal caso de uso es la investigación de seguridad del kernel, incluida la experimentación con anti-cheat y monitorización del sistema.
