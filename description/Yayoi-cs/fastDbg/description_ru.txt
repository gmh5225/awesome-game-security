fastDbg — высокоскоростной отладчик x64 ELF, реализованный на Go для сред Linux.
Он поддерживает управление точками останова, просмотр памяти, операции с регистрами, дизассемблирование и управление процессами на основе ptrace через командный интерфейс.
В проекте также есть поддержка трассировки eBPF и модули, связанные с QEMU, для продвинутых сценариев низкоуровневого анализа.
Основная аудитория — специалисты по реверс-инжинирингу и исследователи безопасности, которым нужна лёгкая, но функциональная инструментальная цепочка отладки.
