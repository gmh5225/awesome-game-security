BinaryAudit es un benchmark de código abierto para evaluar agentes de IA en la detección de puertas traseras y otras modificaciones maliciosas ocultas en binarios compilados de software open source real. Los agentes reciben ejecutables strippeados sin código fuente dentro de entornos Docker aislados y pueden usar herramientas de ingeniería inversa como Ghidra y Radare2. El conjunto de tareas cubre puertas traseras inyectadas artificialmente, controles negativos limpios y detección de bombas de tiempo en programas escritos en C, Go y Rust, incluidos lighttpd, dnsmasq, Dropbear, Sozu y Caddy. Está construido sobre el framework de evaluación Harbor, con herramientas en Python y configuraciones de trabajos YAML para ejecutar experimentos multimodelo. El caso de uso principal es la investigación en seguridad y la evaluación de ingeniería inversa de agentes de IA en el análisis de malware binario y puertas traseras.
