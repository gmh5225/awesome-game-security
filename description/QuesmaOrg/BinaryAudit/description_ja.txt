BinaryAudit は、実際のオープンソースソフトウェアのコンパイル済みバイナリに隠されたバックドアやその他の悪意ある改変を発見する能力について、AI エージェントを評価するためのオープンソースベンチマークです。エージェントは隔離された Docker 環境内で、ソースコードを含まないストリップ済み実行ファイルを受け取り、Ghidra や Radare2 などのリバースエンジニアリングツールを利用できます。タスクスイートは、人工的に注入されたバックドア、クリーンなネガティブコントロール、および C、Go、Rust で書かれたプログラム（lighttpd、dnsmasq、Dropbear、Sozu、Caddy を含む）におけるタイムボム検出をカバーします。Harbor 評価フレームワークを中心に構築され、Python ツールと YAML ジョブ設定でマルチモデル実験を実行できます。主な用途はセキュリティ研究と、バイナリマルウェアおよびバックドア分析における AI エージェントのリバースエンジニアリング評価です。
