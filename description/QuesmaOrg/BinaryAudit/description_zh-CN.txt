BinaryAudit 是一个开源基准，用于评估 AI 智能体在真实开源软件的编译二进制文件中发现后门及其他恶意修改的能力。智能体在隔离的 Docker 环境中收到不含源代码的剥离可执行文件，并可使用 Ghidra 和 Radare2 等逆向工程工具。任务套件覆盖人工注入的后门、干净的负对照样本，以及针对 C、Go 和 Rust 编写的程序（包括 lighttpd、dnsmasq、Dropbear、Sozu 和 Caddy）的定时炸弹检测。它基于 Harbor 评估框架构建，并配备 Python 工具与 YAML 任务配置，用于运行多模型实验。主要用途是安全研究，以及对 AI 智能体在二进制恶意软件与后门分析方面的逆向工程评估。
