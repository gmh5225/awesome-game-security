BinaryAudit 是一個開源基準，用於評估 AI 代理在真實開源軟體的編譯二進位檔案中找出後門及其他惡意修改的能力。代理在隔離的 Docker 環境中收到不含原始碼的剝離可執行檔，並可使用 Ghidra 與 Radare2 等逆向工程工具。任務套件涵蓋人工注入的後門、乾淨的負對照樣本，以及針對以 C、Go 與 Rust 撰寫的程式（包括 lighttpd、dnsmasq、Dropbear、Sozu 與 Caddy）的定時炸彈偵測。它以 Harbor 評估框架為核心，並搭配 Python 工具與 YAML 工作設定，用於執行多模型實驗。主要用途是安全研究，以及對 AI 代理在二進位惡意軟體與後門分析方面的逆向工程評估。
