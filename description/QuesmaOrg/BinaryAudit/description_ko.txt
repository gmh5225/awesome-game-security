BinaryAudit는 실제 오픈소스 소프트웨어의 컴파일된 바이너리에 숨겨진 백도어 및 기타 악의적 수정을 찾아내는 AI 에이전트를 평가하기 위한 오픈소스 벤치마크입니다. 에이전트는 격리된 Docker 환경에서 소스 코드가 없는 스트립된 실행 파일을 받으며, Ghidra와 Radare2 같은 리버스 엔지니어링 도구를 사용할 수 있습니다. 작업 스위트는 인위적으로 주입된 백도어, 깨끗한 음성 대조군, 그리고 C, Go, Rust로 작성된 프로그램(lighttpd, dnsmasq, Dropbear, Sozu, Caddy 포함)에 대한 타임밤 탐지를 다룹니다. Harbor 평가 프레임워크를 중심으로 Python 도구와 YAML 작업 설정을 사용해 다중 모델 실험을 실행할 수 있습니다. 주요 용도는 보안 연구와 바이너리 멀웨어 및 백도어 분석에서의 AI 에이전트 리버스 엔지니어링 평가입니다.
