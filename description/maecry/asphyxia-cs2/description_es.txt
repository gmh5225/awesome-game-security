Este proyecto es Asphyxia, un cheat de código abierto para Counter-Strike 2 escrito en C++. Proporciona ESP (cajas de jugadores, barras de vida, nombres), aimbot, triggerbot y modificaciones visuales mediante la interacción con el SDK del motor Source 2. El cheat interno se engancha a los sistemas de renderizado y entrada del juego usando punteros de interfaz y pattern scanning. Está dirigido a investigadores de seguridad en juegos que estudian la arquitectura de cheats de CS2 y los internals del motor Source 2 para el desarrollo de anti-cheats.
