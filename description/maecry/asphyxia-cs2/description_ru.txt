Этот проект — Asphyxia, открытый чит для Counter-Strike 2 на C++. Он предоставляет ESP (боксы игроков, полоски здоровья, имена), аимбот, триггербот и визуальные модификации через взаимодействие с SDK движка Source 2. Внутренний чит перехватывает системы отрисовки и ввода игры с помощью указателей интерфейсов и pattern scanning. Он предназначен для исследователей игровой безопасности, изучающих архитектуру читов CS2 и внутреннее устройство движка Source 2 для разработки античитов.
