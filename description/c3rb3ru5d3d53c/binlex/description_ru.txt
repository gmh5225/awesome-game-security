Этот проект — инструмент извлечения бинарных шаблонов и анализа генетических признаков, написанный на C++ и Rust. Он дизассемблирует файлы PE, ELF и сырые бинарные файлы, извлекая байтовые шаблоны на уровне функций и признаки инструкций, которые можно использовать для анализа сходства вредоносного ПО, генерации правил YARA и корреляции данных threat intelligence. Инструмент поддерживает несколько архитектур и выводит шаблоны в формате JSON для интеграции с аналитическими конвейерами. Он предназначен для аналитиков malware, исследователей threat intelligence и инженеров по реверс-инжинирингу, создающих сигнатуры обнаружения на основе бинарных артефактов.
