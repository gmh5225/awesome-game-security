Этот проект сосредоточен на внедрении Frida Gadget с помощью Zygisk.
Zygisk — часть Magisk — позволяет выполнять код в процессе каждого Android-приложения.
Он особенно полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / Magisk.
