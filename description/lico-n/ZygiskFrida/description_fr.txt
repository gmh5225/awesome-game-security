Ce projet se concentre sur l'injection de Frida Gadget via Zygisk.
Zygisk, composant de Magisk, permet d'exécuter du code dans le processus de chaque application Android.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine du cheat / Magisk.
