Dieses Projekt konzentriert sich auf die Injektion von Frida Gadget mit Zygisk.
Zygisk als Teil von Magisk ermöglicht es, Code im Prozess jeder Android-Anwendung auszuführen.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Cheat-/Magisk-Bereich untersuchen.
