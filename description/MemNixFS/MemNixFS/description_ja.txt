本プロジェクトは Linux のメモリダンプをファイルシステムとしてマウントし、調査者が通常のツールでプロセス、ファイル、ソケット、モジュール、フォレンジックタイムラインを閲覧できるようにします。AVML、LiME、raw、kdump イメージに対応し、MemProcFS のメモリ・アズ・ファイルシステムという発想を Linux ダンプへ持ち込み、Windows および Linux ホストの両方で利用できます。主に C++17 で書かれており、カーネルレベルのメモリフォレンジックと脅威ハンティングのワークフローを中心としています。アンチチートエンジニアや、アンチチート／情報システム＆フォレンジック領域で活動する防御側のセキュリティ研究者に特に有用です。
