Este proyecto es un plugin de movimiento de personaje para Unreal Engine que recrea el comportamiento de movimiento FPS clásico al estilo Source y HL2. Está escrito en C++ y ofrece sistemas como bunnyhopping, surfing, strafe boosting, wall strafing y movimiento agachado avanzado. El repositorio incluye el código fuente y binarios precompilados, con instrucciones de configuración para integrarlo en proyectos de Unreal. Está pensado para desarrolladores de juegos que buscan mecánicas de movimiento rápidas y basadas en la habilidad en FPS modernos.
