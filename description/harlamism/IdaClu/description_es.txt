Este proyecto es un complemento de IDA Pro para agrupar y organizar funciones en binarios grandes. Ofrece una interfaz gráfica basada en Qt con soporte de internacionalización, agrupación de funciones según diversos criterios y ayudas de navegación visual para facilitar el análisis de binarios con muchas funciones. Resulta especialmente útil para ingenieros inversos que trabajan con binarios grandes en IDA Pro y necesitan mejores herramientas de organización y navegación de funciones.
