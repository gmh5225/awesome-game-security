Этот проект — плагин для IDA Pro, предназначенный для кластеризации и упорядочивания функций в крупных бинарных файлах. Он предоставляет GUI на Qt с поддержкой интернационализации, группировку функций по различным критериям и средства визуальной навигации, помогающие анализировать бинарники с большим числом функций. В первую очередь полезен реверс-инженерам, работающим с крупными бинарными файлами в IDA Pro и нуждающимся в более удобных инструментах организации и навигации по функциям.
