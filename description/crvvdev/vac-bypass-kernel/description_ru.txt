Этот проект посвящён полностью рабочему обходу VAC в режиме ядра.
По сути античит полностью внешний: он использует системные вызовы вроде NtReadVirtualMemory для чтения памяти игры и выполнения ряда проверок.
Он в первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / исследования античит-систем: VAC.
