Este proyecto es tinyobjloader, una biblioteca C++ ligera de un solo encabezado para analizar archivos de modelos 3D Wavefront OBJ. Carga posiciones de vértices, normales, coordenadas de textura, materiales (archivos MTL) e índices de caras, con soporte para triangulación, grupos de suavizado y materiales por cara. La implementación solo de encabezado no tiene dependencias externas y gestiona modelos grandes de forma eficiente. Está orientada a desarrolladores de juegos y programadores gráficos que necesitan un cargador OBJ simple y portable para aplicaciones de renderizado 3D.
