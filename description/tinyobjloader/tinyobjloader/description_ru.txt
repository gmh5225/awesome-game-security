Этот проект — tinyobjloader, лёгкая однозаголовочная библиотека на C++ для разбора файлов 3D-моделей Wavefront OBJ. Она загружает позиции вершин, нормали, текстурные координаты, материалы (файлы MTL) и индексы граней, с поддержкой триангуляции, групп сглаживания и материалов по граням. Реализация только из заголовка не имеет внешних зависимостей и эффективно обрабатывает крупные модели. Предназначена для разработчиков игр и графических программистов, которым нужен простой и переносимый загрузчик OBJ для приложений 3D-рендеринга.
