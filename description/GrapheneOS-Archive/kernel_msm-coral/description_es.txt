Este proyecto es el código fuente del kernel de GrapheneOS para los dispositivos Pixel 4 (coral) y Pixel 4 XL, basado en el árbol de kernel MSM de Qualcomm. Incluye configuraciones de kernel endurecidas, parches específicos de Android y modificaciones orientadas a la seguridad características del proyecto GrapheneOS. Resulta especialmente útil para investigadores de seguridad del kernel de Android y desarrolladores de plataformas móviles que estudian implementaciones de kernels endurecidos en dispositivos Qualcomm Snapdragon.
