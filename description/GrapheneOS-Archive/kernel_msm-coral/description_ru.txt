Этот проект — исходный код ядра GrapheneOS для устройств Pixel 4 (coral) и Pixel 4 XL на основе дерева ядра Qualcomm MSM. Он включает усиленные конфигурации ядра, патчи для Android и ориентированные на безопасность изменения, характерные для проекта GrapheneOS. В основном полезен исследователям безопасности ядер Android и разработчикам мобильных платформ, изучающим усиленные реализации ядра на устройствах Qualcomm Snapdragon.
