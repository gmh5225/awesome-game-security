Questo progetto è un'implementazione PoC di una tecnica avanzata di evasione in-memory che spoofa il Thread Call Stack.
La tecnica consente di aggirare le regole di esame della memoria basate sui thread e di nascondere meglio gli shellcode nella memoria in-process.
È principalmente utile per ricercatori di game security e reverse engineer che studiano tecniche offensive nell'ambito di cheat e spoof dello stack.
