VMPImportFixer は、VMProtect 3.x Import Protection で保護されたバイナリ内の難読化された API インポート呼び出しを解決する C++ ツールです。実行中のプロセスにアタッチし、VMProtect セクション（通常は .vmp0）へ分岐する near-call stub を特定して、実際のインポートアドレスを復元し、それらの呼び出しを通常のインポートへパッチできるようにします。解決はスタブを中間言語へリフトするのではなく、Unicorn ベースの CPU エミュレーションで行い、命令デコードには Zydis、モジュールとエクスポートの扱いに pepp PE ライブラリを使います。単一ビルドで x86 と x86-64 の両方をサポートし、64 ビット環境から 32 ビットプロセスを修正することもできます。ゲームセキュリティやマルウェア解析の流れで、VMProtect 保護ソフトウェアを解析・アンパックするリバースエンジニア向けです。
