VMPImportFixer 是一款 C++ 工具，用于解析受 VMProtect 3.x Import Protection 保护的二进制文件中被混淆的 API 导入调用。它会附加到正在运行的进程，定位那些转入 VMProtect 区段（通常为 .vmp0）的 near-call stub，并恢复真实的导入地址，以便将这些调用修补回正常导入。解析通过基于 Unicorn 的 CPU 模拟完成，而不是将 stub 提升到中间语言；指令解码使用 Zydis，模块与导出处理使用 pepp PE 库。一次构建同时支持 x86 与 x86-64 目标，包括在 64 位环境下修复 32 位进程。面向在游戏安全与恶意软件分析流程中分析或脱壳 VMProtect 保护软件的逆向工程师。
