VMPImportFixer는 VMProtect 3.x Import Protection으로 보호된 바이너리에서 난독화된 API import 호출을 해석하는 C++ 도구입니다. 실행 중인 프로세스에 연결하여 VMProtect 섹션(보통 .vmp0)으로 전환되는 near-call stub을 찾고, 실제 import 주소를 복구해 해당 호출을 일반 import로 패치할 수 있게 합니다. 해석은 stub을 중간 언어로 올리는 방식이 아니라 Unicorn 기반 CPU 에뮬레이션으로 수행하며, 명령어 디코딩에는 Zydis를, 모듈과 export 처리에는 pepp PE 라이브러리를 사용합니다. 단일 빌드로 x86과 x86-64 대상을 모두 지원하며, 64비트 환경에서 32비트 프로세스를 수정하는 것도 가능합니다. 게임 보안 및 악성코드 분석 워크플로에서 VMProtect로 보호된 소프트웨어를 분석하거나 언패킹하는 리버스 엔지니어를 대상으로 합니다.
