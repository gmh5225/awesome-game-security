VMPImportFixer è uno strumento C++ che risolve le chiamate di import API offuscate nei binari protetti con VMProtect 3.x Import Protection. Si collega a un processo in esecuzione, individua gli stub near-call che dirottano nella sezione VMProtect (tipicamente .vmp0) e recupera gli indirizzi di import reali così da poter ripristinare tali chiamate come import normali. La risoluzione avviene tramite emulazione CPU basata su Unicorn anziché sollevando gli stub a un linguaggio intermedio, con Zydis per la decodifica delle istruzioni e la libreria PE pepp per la gestione di moduli ed export. Una singola build supporta sia target x86 sia x86-64, inclusa la correzione di processi a 32 bit da un contesto a 64 bit. È pensato per reverse engineer che analizzano o unpackano software protetto da VMProtect in flussi di lavoro di game security e analisi malware.
