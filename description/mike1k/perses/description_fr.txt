Ce projet est un moteur d'obfuscation de code X86 qui fonctionne avec les fichiers Portable Executable.
L'obfuscation consiste à remplacer une instruction spécifique par un ensemble plus large et plus sophistiqué, sémantiquement compatible avec l'original.
Il est principalement utile aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant dans le domaine des moteurs anti-triche / d'obfuscation.
