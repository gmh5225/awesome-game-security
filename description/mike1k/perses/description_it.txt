Questo progetto è un motore di offuscamento del codice X86 che lavora con file Portable Executable.
L'offuscamento funziona sostituendo una specifica istruzione con un insieme più ampio e sofisticato, semanticamente compatibile con l'originale.
È principalmente utile per ingegneri anti-cheat e ricercatori di sicurezza difensiva che lavorano nell'ambito dei motori anti-cheat / di offuscamento.
