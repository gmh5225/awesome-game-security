Este proyecto es un fragmento mínimo de cheat item-glow para Apex Legends.
Está escrito en C++ y recorre los slots de entidades para escribir valores de resaltado directamente en memoria.
El código es muy pequeño y se centra en una sola función de estilo wall-highlight, en lugar de un framework de cheat completo.
Resulta útil principalmente como prueba de concepto rápida de hacks visuales basados en escritura de memoria.
