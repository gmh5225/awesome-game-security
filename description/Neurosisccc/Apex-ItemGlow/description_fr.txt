Ce projet est un extrait minimal de cheat item-glow pour Apex Legends.
Il est écrit en C++ et parcourt les slots d'entités pour écrire directement les valeurs de surbrillance en mémoire.
Le code est très court et se concentre sur une seule fonctionnalité de type wall-highlight, plutôt que sur un framework de cheat complet.
Il sert surtout de preuve de concept rapide pour des hacks visuels basés sur l'écriture mémoire.
