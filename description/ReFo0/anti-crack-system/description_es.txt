Este proyecto es un prototipo de framework anti-tamper y anti-crack para Windows escrito en C++.
Incluye protecciones anti-debugging, anti-dump y anti-attach, comprobaciones de integridad, rutinas de finalización de procesos y rutas de código de auto-remapeo.
También contiene ofuscación de cadenas y helpers de cifrado ligero para reforzar constantes sensibles.
Está orientado a experimentos de protección de software y demuestra técnicas defensivas que pueden adaptarse a herramientas de seguridad en juegos.
