CSGO-Cheats — это ориентированный на обучение набор примеров читов для Counter-Strike: Global Offensive на C++. Он показывает основные сценарии внешнего (external) взлома: поиск окна игры, открытие процесса, чтение и запись памяти через вспомогательные обёртки. Проект рассчитан на сборку в Visual Studio и опирается на актуальные оффсеты, чтобы примеры оставались рабочими после обновлений игры. В основном его используют новички для изучения манипуляций с памятью игр, а защитники — чтобы понять типичные паттерны читов.
