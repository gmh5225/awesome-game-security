CSGO-Cheats es una colección orientada a tutoriales de ejemplos de cheats para Counter-Strike: Global Offensive, implementados en C++. Muestra flujos esenciales de hacking externo, como localizar la ventana del juego, abrir el proceso y leer o escribir memoria mediante wrappers auxiliares. El proyecto está pensado para compilaciones en Visual Studio y depende de offsets actualizados para mantener los ejemplos funcionales tras las actualizaciones del juego. Se usa principalmente para que principiantes aprendan manipulación de memoria en juegos y para que los defensores entiendan patrones comunes de cheats.
