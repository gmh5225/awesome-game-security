Este proyecto es un conjunto de analizadores NTFS forenses que vincula artefactos entre las fuentes de datos MFT, LogFile y UsnJrnl.
Ofrece herramientas de línea de comandos para exportar registros, analizar transacciones, extraer datos y realizar un análisis combinado de línea temporal como prueba de concepto.
La implementación está escrita en Python con analizadores modulares, scripts auxiliares y formatos de salida estructurados como texto analizado y CSV.
Está pensado para la informática forense y la investigación de sistemas de archivos, no para trampas en juegos ni para eludir anti-cheat.
