Этот проект — набор форензик-парсеров NTFS, связывающий артефакты из источников данных MFT, LogFile и UsnJrnl.
Он предоставляет инструменты командной строки для экспорта записей, разбора транзакций, извлечения данных и комбинированного proof-of-concept анализа временной шкалы.
Реализация написана на Python с модульными парсерами, вспомогательными скриптами и структурированными форматами вывода, такими как разобранный текст и CSV.
Проект предназначен для цифровой криминалистики и исследования файловых систем, а не для читерства в играх или обхода античита.
