Este proyecto es un índice seleccionado de recursos y referencias para el aprendizaje de la seguridad en Android.
Se mantiene como contenido Markdown y CSV que organiza enlaces teóricos, herramientas, artículos técnicos, material de prueba de concepto e informes de vulnerabilidades.
La colección abarca temas como la explotación del kernel, pruebas de aplicaciones móviles, flujos de trabajo con Frida y estudios de caso prácticos.
Está dirigido a investigadores de seguridad móvil, penetesters e ingenieros que construyen una base de conocimiento sobre seguridad Android.
