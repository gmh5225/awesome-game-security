Этот проект — курируемый индекс учебных ресурсов и справочных материалов по безопасности Android.
Он ведётся в виде Markdown- и CSV-контента, систематизирующего теоретические ссылки, инструменты, разборы, материалы proof-of-concept и отчёты об уязвимостях.
Коллекция охватывает такие темы, как эксплуатация ядра, тестирование мобильных приложений, рабочие процессы Frida и практические кейсы.
Предназначен для исследователей мобильной безопасности, пентестеров и инженеров, формирующих базу знаний по безопасности Android.
