Este proyecto se centra en wxshadow: un breakpoint/hook furtivo KPM mediante división de páginas R^X, para eludir la comprobación de integridad por autolección.
Cualquier obra derivada que use, modifique o distribuya el código de este proyecto debe publicarse como código abierto bajo la misma licencia.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / exploración del kernel de Android.
