Ce projet se concentre sur wxshadow — un point d'arrêt/hook furtif KPM via la séparation de pages R^X, pour contourner le contrôle d'intégrité par auto-lecture.
Toute œuvre dérivée qui utilise, modifie ou distribue le code de ce projet doit être open source sous la même licence.
Il s'adresse surtout aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / de l'exploration du noyau Android.
