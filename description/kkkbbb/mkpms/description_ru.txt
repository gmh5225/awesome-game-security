Этот проект посвящён wxshadow — скрытому breakpoint/hook KPM через разделение страниц R^X для обхода проверки целостности самочтением.
Любые производные работы, которые используют, изменяют или распространяют код этого проекта, должны быть открыты под той же лицензией.
Он в первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / исследования ядра Android.
