Este proyecto es un framework de cheat externo para el juego Rust que combina componentes en modo usuario y modo kernel en Windows. Está escrito principalmente en C++ e incluye un controlador de kernel personalizado para operaciones de lectura/escritura de memoria, además de un módulo DLL externo. El lado en modo usuario integra ImGui y renderizado DirectX para ofrecer funciones como ESP, aimbot y asistencia relacionada con el retroceso. Se usa principalmente para investigación sobre cheats en juegos y para estudiar las superficies de detección del anti-cheat en torno a externos asistidos por drivers.
