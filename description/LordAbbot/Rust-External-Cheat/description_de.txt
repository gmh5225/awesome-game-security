Dieses Projekt ist ein externes Cheat-Framework für das Spiel Rust, das unter Windows User-Mode- und Kernel-Mode-Komponenten kombiniert. Es ist hauptsächlich in C++ geschrieben und enthält einen benutzerdefinierten Kernel-Treiber für Speicher-Lese-/Schreibvorgänge sowie ein externes DLL-Modul. Die User-Mode-Seite integriert ImGui und DirectX-Rendering und bietet Funktionen wie ESP, Aimbot und rückstoßbezogene Unterstützung. Es dient vor allem der Erforschung von Game Cheating und dem Studium von Anti-Cheat-Erkennungsflächen rund um treiberunterstützte Externals.
