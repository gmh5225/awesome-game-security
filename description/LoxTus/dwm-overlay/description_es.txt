Este proyecto es una prueba de concepto de overlay DWM basada en DLL para Windows que engancha el renderizado de composición del escritorio. Está implementado en C++ con MinHook, DirectX 11 e ImGui, y usa pattern scanning para localizar la rutina Present objetivo en dwmcore. Tras el hook, inicializa una ruta de renderizado D3D11 y dibuja UI personalizada a través de la swap chain. El proyecto resulta útil principalmente para investigación de hooks gráficos, experimentación con overlays y comprensión de la interceptación de renderizado a nivel de escritorio.
