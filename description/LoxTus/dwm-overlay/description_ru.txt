Этот проект — DLL-based proof of concept DWM-оверлея для Windows, который перехватывает рендеринг композиции рабочего стола. Он реализован на C++ с MinHook, DirectX 11 и ImGui и использует pattern scanning для поиска целевой процедуры Present в dwmcore. После хука инициализируется путь рендеринга D3D11, а пользовательский UI рисуется через swap chain. Проект в основном полезен для исследования графических хуков, экспериментов с оверлеями и понимания перехвата рендеринга на уровне рабочего стола.
