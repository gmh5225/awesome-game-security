Ursine 3D — универсальный игровой движок, созданный для академических программ разработки игр. Он написан в основном на C++ и предоставляет систему сущностей и компонентов, поддержку рефлексии, инструменты редактора, системы анимации и полноценный конвейер 3D-рендеринга. Движок интегрирует такие технологии, как Bullet Physics, аудио Wwise и Chromium Embedded Framework для интерфейсов на HTML/CSS/JS. Основное применение — обучение игровым движкам, прототипирование и разработка полноценных проектов.
