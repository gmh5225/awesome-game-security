Este proyecto es una base interna de cheats para Apex Legends pensada como marco de partida para el desarrollo de funciones.
Proporciona andamiaje de hooking, abstracciones de juego y entidades, tipos matemáticos y de utilidad, y una capa de menú basada en ImGui.
La implementación es principalmente en C++ en un entorno Visual Studio con integración de Windows y DirectX.
Está orientado a la práctica de desarrollo de cheats y a la experimentación en ingeniería inversa en contextos de juegos FPS.
