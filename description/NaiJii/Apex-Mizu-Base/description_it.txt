Questo progetto è una base interna di cheat per Apex Legends pensata come framework di partenza per lo sviluppo di funzionalità.
Fornisce scaffolding per l'hooking, astrazioni di gioco e delle entità, tipi matematici e di utilità, e un layer di menu basato su ImGui.
L'implementazione è principalmente in C++ in un ambiente Visual Studio con integrazione Windows e DirectX.
È rivolto alla pratica di sviluppo di cheat e alla sperimentazione di reverse engineering in contesti di giochi FPS.
