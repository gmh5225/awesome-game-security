Этот проект — внутренняя база чита для Apex Legends, задуманная как стартовый каркас для разработки функций.
Она предоставляет каркас для хуков, абстракции игры и сущностей, математические и вспомогательные типы, а также слой меню на ImGui.
Реализация в основном на C++ в среде Visual Studio с интеграцией Windows и DirectX.
Она ориентирована на практику разработки читов и эксперименты по реверс-инжинирингу в контексте FPS-игр.
