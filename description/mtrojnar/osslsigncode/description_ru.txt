Кроссплатформенный инструмент Authenticode-подписи на базе OpenSSL и cURL, который подписывает и ставит метки времени на файлы PE (EXE/SYS/DLL), CAB, CAT, MSI, APPX и скрипты PowerShell/JS. Поддерживает сертификаты PKCS#12/PFX, аппаратные токены PKCS#11 (SoftHSM), метки времени RFC 3161, вычисление хешей страниц, вложенные подписи и создание каталожных файлов без необходимости в Windows или Microsoft signtool.
