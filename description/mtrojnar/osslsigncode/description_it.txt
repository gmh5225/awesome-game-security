Strumento di firma Authenticode multipiattaforma basato su OpenSSL e cURL che firma e applica timestamp a file PE (EXE/SYS/DLL), CAB, CAT, MSI, APPX e script PowerShell/JS. Supporta certificati PKCS#12/PFX, token hardware PKCS#11 (SoftHSM), timestamping RFC 3161, calcolo degli hash di pagina, firme annidate e creazione di file catalogo, senza richiedere Windows o il signtool di Microsoft.
