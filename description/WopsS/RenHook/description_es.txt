Este proyecto es una biblioteca de hooking de Windows de código abierto para objetivos x86 y x86-64.
Está escrita en C++ moderno y usa desensamblado basado en Zydis para construir hooks inline y trampolines fiables.
El código enfatiza APIs seguras, soporte de arquitecturas y ejemplos prácticos para flujos de intercepción de funciones.
Es útil para ingeniería inversa, herramientas de instrumentación e investigación de seguridad en juegos que requieren redirección de código en tiempo de ejecución.
