Dieses Projekt ist eine Open-Source-Windows-Hooking-Bibliothek für x86- und x86-64-Ziele.
Sie ist in modernem C++ geschrieben und nutzt Zydis-basiertes Disassembly, um zuverlässige Inline-Hooks und Trampolines zu erzeugen.
Der Code legt Wert auf sichere APIs, Architekturunterstützung und praxisnahe Beispiele für Function-Interception-Workflows.
Sie eignet sich für Reverse Engineering, Instrumentierungs-Tools und Game-Security-Forschung, die Laufzeit-Code-Umleitung benötigt.
