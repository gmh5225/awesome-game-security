Ce projet est une bibliothèque de hooking Windows open source pour les cibles x86 et x86-64.
Elle est écrite en C++ moderne et s'appuie sur le désassemblage basé sur Zydis pour construire des hooks inline et des trampolines fiables.
Le code met l'accent sur des API sûres, le support d'architectures et des exemples pratiques pour les workflows d'interception de fonctions.
Elle est utile pour la rétro-ingénierie, les outils d'instrumentation et la recherche en sécurité des jeux nécessitant une redirection de code à l'exécution.
