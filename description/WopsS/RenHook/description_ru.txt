Этот проект — открытая библиотека хукинга для Windows, рассчитанная на цели x86 и x86-64.
Она написана на современном C++ и использует дизассемблирование на базе Zydis для построения надёжных inline-хуков и трамплинов.
Код делает акцент на безопасных API, поддержке архитектур и практических примерах перехвата функций.
Библиотека полезна для реверс-инжиниринга, инструментов инструментирования и исследований безопасности игр, где нужна переадресация кода во время выполнения.
