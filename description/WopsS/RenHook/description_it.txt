Questo progetto è una libreria open-source di hooking per Windows rivolta a target x86 e x86-64.
È scritta in C++ moderno e usa il disassemblaggio basato su Zydis per costruire inline hook e trampoline affidabili.
Il codice punta su API sicure, supporto delle architetture ed esempi pratici per i flussi di intercettazione delle funzioni.
È utile per reverse engineering, strumenti di strumentazione e ricerca di game security che richiedono ridirezione del codice a runtime.
