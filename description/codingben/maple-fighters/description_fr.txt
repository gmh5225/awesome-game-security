Ce projet est un petit jeu en ligne similaire à MapleStory.
Maple Fighters est un jeu multijoueur en ligne inspiré de MapleStory où vous combattez des monstres avec d'autres joueurs en temps réel.
Il est surtout utile aux développeurs de jeux, aux reverse engineers et aux créateurs d'outils qui travaillent dans le domaine du développement de jeux / du code source.
