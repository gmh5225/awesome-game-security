Этот проект — небольшая онлайн-игра, похожая на MapleStory.
Maple Fighters — это многопользовательская онлайн-игра по мотивам MapleStory, в которой вы сражаетесь с монстрами вместе с другими игроками в реальном времени.
Она особенно полезна разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов, работающим в области разработки игр / исходного кода.
