Этот проект — gta-reversed, совместная работа по реверс-инжинирингу и воссозданию исходного кода Grand Theft Auto: San Andreas на современном C++. Он заменяет исходные игровые функции декомпилированными и переписанными эквивалентами, которые можно компилировать вместе с оригинальным бинарником, постепенно замещая игровой код при сохранении совместимости. Проект охватывает рендеринг, физику, управление транспортом, управление игроком, скриптинг миссий и аудиосистемы. Он предназначен для реверс-инженеров игр, моддеров и исследователей, изучающих архитектуру AAA-игровых движков через систематическую декомпиляцию.
