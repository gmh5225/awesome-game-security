This project is a DirectDraw7/D3D7 to D3D9 wrapper for Hard Truck 2 (King of the Road).
It adds modern resolutions, MSAA, anisotropic filtering, graphics fixes, and widescreen hooks for the legacy title.
It is primarily written in C++ and centers on DirectX compatibility and rendering upgrades.
It is mainly useful for graphics programmers and Windows game tooling developers working in the directx / compatibility area.
