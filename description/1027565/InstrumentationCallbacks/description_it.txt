Questo progetto è una libreria proof-of-concept che estende gli instrumentation callbacks non documentati di Windows dalla modalità utente.
Può intercettare le transizioni da kernel a user mode per system call, consegna APC, eccezioni, callback in user mode ed eventi di inizializzazione di nuovi thread.
L'implementazione usa C++ e assembly con dipendenze minime incentrate su NTDLL, e mira a sistemi x86-64.
Il caso d'uso principale è la ricerca a basso livello sugli internals di Windows, esperimenti di debugging e studi di telemetria legati agli EDR.
