Этот проект — библиотека proof-of-concept, расширяющая недокументированные Windows instrumentation callbacks из пользовательского режима.
Она может перехватывать переходы из ядра в пользовательский режим для системных вызовов, доставки APC, исключений, пользовательских callback и событий инициализации новых потоков.
Реализация использует C++ и ассемблер с минимальными зависимостями, сосредоточенными вокруг NTDLL, и ориентирована на системы x86-64.
Основной сценарий применения — исследование низкоуровневых internals Windows, отладочные эксперименты и исследования телеметрии, связанной с EDR.
