Ce projet est une bibliothèque de preuve de concept qui étend les instrumentation callbacks Windows non documentés depuis le mode utilisateur.
Elle peut intercepter les transitions noyau-vers-utilisateur pour les appels système, la livraison d'APC, les exceptions, les callbacks en mode utilisateur et les événements d'initialisation de nouveaux threads.
L'implémentation utilise C++ et de l'assembleur avec des dépendances minimales centrées sur NTDLL, et cible les systèmes x86-64.
Son cas d'usage principal est la recherche sur les internals Windows de bas niveau, les expériences de débogage et les études de télémétrie liées aux EDR.
