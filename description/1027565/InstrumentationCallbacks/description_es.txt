Este proyecto es una biblioteca de prueba de concepto que amplía los instrumentation callbacks no documentados de Windows desde el modo de usuario.
Puede interceptar las transiciones de núcleo a usuario para llamadas al sistema, entrega de APC, excepciones, callbacks en modo usuario y eventos de inicialización de nuevos hilos.
La implementación usa C++ y ensamblador con dependencias mínimas centradas en NTDLL, y está orientada a sistemas x86-64.
Su caso de uso principal es la investigación de internos de Windows a bajo nivel, experimentos de depuración y estudios de telemetría relacionados con EDR.
