Este proyecto es una colección en C++ de técnicas de inyección de código basadas en el host para Windows. Incluye alrededor de dos docenas de implementaciones de inyección de PE, DLL y shellcode, como variantes de hollowing, métodos basados en callbacks y rutas de abuso del cargador. Los scripts de compilación organizan cada técnica como un ejecutable independiente e incluyen notas de compatibilidad para 32 bits, 64 bits, WoW64 y varias versiones de Windows. El público principal son investigadores de seguridad que estudian el tradecraft ofensivo y la cobertura de detección defensiva.
