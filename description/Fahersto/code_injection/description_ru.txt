Этот проект — коллекция на C++ техник host-based внедрения кода для Windows. В него входят около двух десятков реализаций внедрения PE, DLL и shellcode, включая варианты hollowing, методы на основе callback и пути злоупотребления загрузчиком. Скрипты сборки оформляют каждую технику как отдельный исполняемый файл и содержат заметки о совместимости для 32-бит, 64-бит, WoW64 и нескольких версий Windows. Основная аудитория — исследователи безопасности, изучающие наступательный tradecraft и покрытие средств защитного обнаружения.
