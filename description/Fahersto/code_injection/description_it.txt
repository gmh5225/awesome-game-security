Questo progetto è una raccolta in C++ di tecniche di code injection basate sull'host per Windows. Include circa due dozzine di implementazioni di injection PE, DLL e shellcode, come varianti di hollowing, metodi basati su callback e percorsi di abuso del loader. Gli script di build organizzano ogni tecnica come eseguibile separato e includono note di compatibilità per 32 bit, 64 bit, WoW64 e più versioni di Windows. Il pubblico principale sono i ricercatori di sicurezza che studiano il tradecraft offensivo e la copertura del rilevamento difensivo.
