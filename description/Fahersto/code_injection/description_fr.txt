Ce projet est une collection C++ de techniques d'injection de code basées sur l'hôte pour Windows. Il regroupe environ une vingtaine d'implémentations d'injection PE, DLL et shellcode, telles que des variantes de hollowing, des méthodes basées sur des callbacks et des chemins d'abus du chargeur. Les scripts de build organisent chaque technique en exécutable distinct et incluent des notes de compatibilité pour le 32 bits, le 64 bits, WoW64 et plusieurs versions de Windows. Le public principal est constitué de chercheurs en sécurité qui étudient le tradecraft offensif et la couverture de détection défensive.
