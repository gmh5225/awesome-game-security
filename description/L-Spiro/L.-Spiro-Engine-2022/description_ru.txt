Этот проект — кроссплатформенный игровой движок на C++ с модульными библиотеками для графики, физики, звука, сетей, математики, управления памятью, многопоточности и работы с моделями. Он поддерживает сборки под Windows (Visual Studio) и macOS (Xcode), включает инструменты сжатия текстур DXT и предоставляет полноценный слой абстракции стандартной библиотеки. В первую очередь полезен программистам движков и разработчикам игр, изучающим архитектуру и проектирование низкоуровневых кроссплатформенных игровых движков.
