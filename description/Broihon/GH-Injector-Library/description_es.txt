Este proyecto es una biblioteca de inyección de DLL para Windows con numerosas funciones, orientada a destinos x86, x64 y WOW64. Está escrita en C++ e implementa múltiples estrategias de carga, incluidas varias rutas basadas en Ldr y el mapeo manual (manual mapping). La biblioteca también incluye muchos métodos de ejecución de shellcode, como NtCreateThreadEx, APC, secuestro de hilos, SetWindowsHookEx y técnicas de callback del kernel. Componentes adicionales para cloaking, manejo de hooks y carga de ensamblados .NET la hacen útil para flujos avanzados de game hacking y la investigación del comportamiento de los anti-cheats.
