Этот проект — функционально богатая библиотека инъекции DLL для Windows, рассчитанная на цели x86, x64 и WOW64. Она написана на C++ и реализует несколько стратегий загрузки, включая различные пути на основе Ldr и ручное отображение (manual mapping). Библиотека также включает множество методов выполнения шеллкода, таких как NtCreateThreadEx, APC, захват потока, SetWindowsHookEx и техники kernel callback. Дополнительные компоненты для cloaking, обработки хуков и загрузки сборок .NET делают её полезной для продвинутых игровых хакерских рабочих процессов и исследования поведения античитов.
