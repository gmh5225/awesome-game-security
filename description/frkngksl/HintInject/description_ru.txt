Техника PE-инъекции, скрывающая shellcode в записях Hint/Name Table поддельных элементов Import Directory. Она разбивает исходный payload shellcode на фрагменты, подогнанные под слоты имён экспорта каждой поддельной импортируемой DLL, записывает их в таблицу поиска импортов PE и восстанавливает shellcode при загрузке через обычный путь разрешения IAT загрузчика. Инструмент разбирает заголовки секций целевого PE, манипулирует преобразованиями RVA в смещения и случайным образом выбирает имена экспорта из системных DLL для заполнения поддельных записей.
