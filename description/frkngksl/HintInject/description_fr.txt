Technique d'injection PE qui dissimule du shellcode dans les entrées Hint/Name Table d'entrées Import Directory fabriquées. Elle découpe une charge utile shellcode brute en blocs dimensionnés pour les emplacements de noms d'export de chaque fausse DLL importée, les écrit dans la table de recherche d'imports du PE, et reconstruit le shellcode au chargement via le chemin normal de résolution IAT du chargeur. L'outil analyse les en-têtes de sections du PE cible, manipule les conversions RVA vers offset, et sélectionne aléatoirement des noms d'export dans des DLL système pour peupler les fausses entrées.
