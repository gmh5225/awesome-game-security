Tecnica di injection PE che nasconde shellcode nelle voci Hint/Name Table di voci Import Directory fabbricate. Suddivide un payload shellcode grezzo in chunk dimensionati per adattarsi agli slot dei nomi di export di ciascuna DLL importata falsa, li scrive nella import lookup table del PE e ricostruisce lo shellcode al caricamento attraverso il normale percorso di risoluzione IAT del loader. Lo strumento analizza gli header di sezione del PE target, manipola le conversioni da RVA a offset e seleziona casualmente nomi di export dalle DLL di sistema per popolare le voci false.
