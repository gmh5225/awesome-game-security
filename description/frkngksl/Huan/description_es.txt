Este proyecto es un generador de cargadores PE cifrados que desarrollé para aprender la estructura de archivos PE y los procesos de carga PE.
Cifra el archivo PE a ejecutar con claves diferentes cada vez y lo incrusta en una nueva sección del binario del cargador.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / empaquetadores binarios.
