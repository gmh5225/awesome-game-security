Этот проект — коллекция PoC для выполнения обычных задач нестандартными способами.
Он в основном написан на C и сосредоточен на пайплайнах ассетов и анализе памяти.
Он прежде всего полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / Windows ring3 callbacks.
