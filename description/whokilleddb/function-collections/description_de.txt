Dieses Projekt ist eine Sammlung von PoCs, die gängige Dinge auf unkonventionelle Weise erledigen.
Es ist hauptsächlich in C geschrieben und dreht sich um Asset-Pipelines und Speicheranalyse.
Es eignet sich vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher im Bereich Anti-Cheat / Windows-Ring3-Callbacks.
