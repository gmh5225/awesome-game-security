Этот проект — proof of concept для сбора аппаратных идентификаторов напрямую с устройств PCI и AHCI.
Он перебором выполняет перечисление PCI и выявляет контроллеры хранения без опоры на API операционной системы более высокого уровня.
Цель — снизить подверженность программным хукам, которые могут подменять или перехватывать обычные запросы HWID.
Он в первую очередь полезен для исследований античита и низкоуровневой безопасности платформы, с явной оговоркой об экспериментальном характере.
