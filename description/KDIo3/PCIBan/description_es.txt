Este proyecto es una prueba de concepto para recopilar identificadores de hardware directamente desde dispositivos PCI y AHCI.
Fuerza la enumeración PCI e identifica controladores relacionados con el almacenamiento sin depender de API del sistema operativo de más alto nivel.
El objetivo es reducir la exposición a ganchos de software que puedan falsificar o interceptar consultas HWID convencionales.
Es útil principalmente para investigación de anti-trampas y seguridad de plataforma de bajo nivel, con una advertencia experimental explícita.
