Questo progetto è una proof of concept per raccogliere identificatori hardware direttamente da dispositivi PCI e AHCI.
Forza l'enumerazione PCI e identifica i controller legati allo storage senza affidarsi alle API di sistema di livello superiore.
L'obiettivo è ridurre l'esposizione a hook software che possono falsificare o intercettare le query HWID convenzionali.
È utile soprattutto per la ricerca su anti-cheat e sicurezza di piattaforma a basso livello, con un'esplicita avvertenza sperimentale.
