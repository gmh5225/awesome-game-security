Dieses Projekt stellt MCP-Server (Model Context Protocol) bereit, um KI-Modelle mit Binäranalysetools zu integrieren, beginnend mit IDA Pro. Der Python-basierte IDA-MCP-Server stellt Funktionsanalyse, Querverweise, Variableninspektion und Hilfswerkzeuge über eine strukturierte API bereit und ermöglicht so LLM-gesteuerte Reverse-Engineering-Workflows. Es eignet sich vor allem für Reverse Engineers, die eine KI-unterstützte Binäranalyse über MCP-kompatible Sprachmodelle suchen.
