Ce projet fournit des serveurs MCP (Model Context Protocol) pour intégrer des modèles d'IA aux outils d'analyse binaire, en commençant par IDA Pro. Le serveur MCP IDA basé sur Python expose l'analyse de fonctions, les références croisées, l'inspection de variables et des outils utilitaires via une API structurée, permettant des flux de travail de reverse engineering pilotés par LLM. Il est principalement utile aux reverse engineers qui souhaitent une analyse binaire augmentée par l'IA via des modèles de langage compatibles MCP.
