Este proyecto proporciona servidores MCP (Model Context Protocol) para integrar modelos de IA con herramientas de análisis binario, empezando por IDA Pro. El servidor MCP de IDA basado en Python expone análisis de funciones, referencias cruzadas, inspección de variables y herramientas utilitarias mediante una API estructurada que habilita flujos de trabajo de ingeniería inversa impulsados por LLM. Es especialmente útil para ingenieros inversos que buscan análisis binario aumentado con IA a través de modelos de lenguaje compatibles con MCP.
