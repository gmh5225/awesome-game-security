Questo progetto fornisce server MCP (Model Context Protocol) per integrare modelli di IA con strumenti di analisi binaria, a partire da IDA Pro. Il server MCP di IDA basato su Python espone analisi delle funzioni, riferimenti incrociati, ispezione delle variabili e strumenti di utilità tramite un'API strutturata che abilita flussi di lavoro di reverse engineering guidati da LLM. È principalmente utile per i reverse engineer che cercano un'analisi binaria potenziata dall'IA tramite modelli linguistici compatibili con MCP.
