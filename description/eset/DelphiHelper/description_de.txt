Dieses Projekt konzentriert sich darauf, die Analyse von in Delphi geschriebenen x86/x86_64-Binärdateien zu unterstützen.
_NOTE_ : Für die Ausführung des Setup-Skripts ist die Installation von py7zr erforderlich: pip install py7zr.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
