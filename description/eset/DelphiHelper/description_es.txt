Este proyecto se centra en ayudar al análisis de binarios x86/x86_64 escritos en Delphi.
_NOTE_ : se requiere la instalación de py7zr para ejecutar el script de configuración: pip install py7zr.
Es principalmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
