Ce projet vise à aider l'analyse des binaires x86/x86_64 écrits en Delphi.
_NOTE_ : l'installation de py7zr est requise pour exécuter le script d'installation : pip install py7zr.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
