Этот проект помогает анализировать бинарные файлы x86/x86_64, написанные на Delphi.
_NOTE_ : для запуска скрипта установки требуется установить py7zr: pip install py7zr.
Он в основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / плагинов IDA.
