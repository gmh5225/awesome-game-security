WindMouse — это библиотека на Python, которая генерирует человекоподобные траектории движения мыши для задач автоматизации. Она реализует алгоритм WindMouse и создаёт изогнутые нелинейные траектории с переменной скоростью вместо детерминированного прямолинейного движения. Проект поддерживает несколько бэкендов автоматизации, включая AutoHotkey и PyAutoGUI, и предоставляет настройки физики движения. Основной сценарий использования — снижение риска поведенческого обнаружения ботов при автоматизации UI и скриптинге игр.
