WindMouse es una biblioteca de Python que genera trayectorias de ratón similares a las humanas para tareas de automatización. Implementa el algoritmo WindMouse para producir trayectorias curvas y no lineales con velocidad variable, en lugar de un movimiento rectilíneo determinista. El proyecto admite varios backends de automatización, incluidos AutoHotkey y PyAutoGUI, y expone controles de ajuste para la física del movimiento. Su principal caso de uso es reducir el riesgo de detección conductual de bots en automatización de interfaces y scripts de juegos.
