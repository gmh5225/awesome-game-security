WindMouse è una libreria Python che genera percorsi di movimento del mouse simili a quelli umani per attività di automazione. Implementa l'algoritmo WindMouse per produrre traiettorie curve e non lineari a velocità variabile, invece di un movimento rettilineo deterministico. Il progetto supporta diversi backend di automazione, tra cui AutoHotkey e PyAutoGUI, e espone controlli di regolazione per la fisica del movimento. Il caso d'uso principale è ridurre il rischio di rilevamento comportamentale dei bot nell'automazione delle UI e nello scripting di giochi.
