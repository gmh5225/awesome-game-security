Этот проект — proof-of-concept механизма связи Windows из ядра в пользовательский режим на основе подмены указателя .data в ядре. Он написан на C++ с отдельными компонентами драйвера ядра и пользовательского режима и включает pattern scanning, а также вспомогательные процедуры для операций с памятью и поиска базы процесса. Подход делает акцент на высокоскоростной обмене сообщениями и коммуникации в стиле чтения-записи, но несёт риски стабильности на защищённых системах. Предназначен для исследования ядра и экспериментов, связанных с античитом, а не для промышленного развёртывания.
