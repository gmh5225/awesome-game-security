Этот проект — концептуальное доказательство (proof of concept) для ядра Windows, демонстрирующее обход самопроверок целостности античита путём манипуляции иерархией вызовов и чтениями памяти. Реализован на C++ с фрагментами на ассемблере и перехватывает выбранные процедуры проверки, перенаправляя инспекцию на очищенную копию образа. Код отслеживает события загрузки модулей, границы памяти драйвера EasyAntiCheat и подменяет ссылки на стек и регистры при обращениях, связанных с проверкой целостности. Предназначен для исследования античита методом обратной разработки и для защитного понимания путей верификации целостности.
