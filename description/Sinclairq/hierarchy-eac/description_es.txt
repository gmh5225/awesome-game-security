Este proyecto es un controlador de kernel de Windows de prueba de concepto que demuestra cómo eludir las comprobaciones de autointegridad de EasyAntiCheat.sys abusando del comportamiento de la jerarquía de llamadas (call hierarchy). El código está mayormente en C++ con ensamblador x64 de apoyo e incluye análisis PE, comprobaciones ligadas a secciones y una lógica de control personalizada al estilo de máquina virtual. Incluye archivos de proyecto de Visual Studio y referencia un artículo técnico que explica el enfoque en detalle. El caso de uso principal es la ingeniería inversa de anti-cheat y la investigación de mecanismos de integridad a nivel de kernel.
