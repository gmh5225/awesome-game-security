Código fuente del kernel de Samsung Galaxy para la plataforma SoC Exynos 5433 (usada en dispositivos como el Galaxy Note 4 y el Galaxy Alpha), que proporciona un árbol completo de kernel Linux de la era 3.10 con soporte de placa específico de Samsung, planificación ARM big.LITTLE e integraciones de controladores propietarios para los subsistemas de pantalla, cámara y módem, relevante para la investigación de seguridad a nivel de kernel de Android.
