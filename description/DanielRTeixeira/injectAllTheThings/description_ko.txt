injectAllTheThings는 Windows에서 여러 DLL 인젝션 기법을 보여주는 교육용 Visual Studio 프로젝트입니다. CreateRemoteThread, NtCreateThreadEx, QueueUserAPC, SetWindowsHookEx, RtlCreateUserThread, SetThreadContext, 그리고 리플렉티브 DLL 로딩을 포함한 일곱 가지 기법을 구현하며, x86과 x64 대상을 모두 지원합니다. 각 기법은 별도의 소스 파일로 분리되어 있어 구현 차이와 실행상의 트레이드오프를 비교하기 쉽습니다. 이 프로젝트는 주로 보안 학습자, 리버스 엔지니어, 프로세스 인젝션 메커니즘을 연구하는 연구자를 대상으로 합니다.
