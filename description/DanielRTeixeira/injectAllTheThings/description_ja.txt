injectAllTheThings は、Windows 上で複数の DLL インジェクション手法を示す教育用 Visual Studio プロジェクトです。CreateRemoteThread、NtCreateThreadEx、QueueUserAPC、SetWindowsHookEx、RtlCreateUserThread、SetThreadContext、およびリフレクティブ DLL ローディングを含む 7 つの技法を実装し、x86 と x64 の両方のターゲットに対応しています。各技法はそれぞれ別のソースファイルに分離されており、実装の違いや実行上のトレードオフを学びやすくなっています。このプロジェクトは主に、セキュリティ学習者、リバースエンジニア、プロセスインジェクションの仕組みを研究する研究者向けです。
