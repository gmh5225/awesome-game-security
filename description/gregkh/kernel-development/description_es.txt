Este proyecto es un recurso de documentación y tutoriales sobre el desarrollo del kernel de Linux, elaborado por Greg Kroah-Hartman. Cubre el proceso de desarrollo del kernel, el estilo de codificación, el envío de parches, el desarrollo de controladores de dispositivos y la programación de módulos del kernel. Los materiales orientan sobre cómo contribuir al kernel de Linux upstream. Está dirigido a aspirantes a desarrolladores del kernel de Linux y a programadores de sistemas que aprenden los flujos de trabajo y las convenciones del desarrollo del kernel.
