Этот проект — документация и учебные материалы по разработке ядра Linux от Greg Kroah-Hartman. Он охватывает процесс разработки ядра, стиль кодирования, отправку патчей, разработку драйверов устройств и программирование модулей ядра. Материалы дают рекомендации по участию в разработке upstream-ядра Linux. Они предназначены для начинающих разработчиков ядра Linux и системных программистов, изучающих рабочие процессы и соглашения разработки ядра.
