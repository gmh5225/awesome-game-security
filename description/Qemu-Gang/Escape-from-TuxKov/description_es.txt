Este proyecto es un framework de cheat en C++ para Escape from Tarkov diseñado para ejecutarse mediante flujos de trabajo basados en QEMU.
Incluye módulos de jugabilidad como ESP, aimbot y control de retroceso, con componentes separados para acceso a memoria y recopilación de datos.
El código integra estructuras de juego de Unity y Mono y proporciona scripts para compilación, inyección, extracción y recarga rápida.
Está dirigido a investigadores de game hacking que experimentan con configuraciones externas o híbridas de Linux/QEMU.
