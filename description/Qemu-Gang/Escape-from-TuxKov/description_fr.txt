Ce projet est un framework de cheat C++ pour Escape from Tarkov conçu pour s'exécuter via des workflows basés sur QEMU.
Il comprend des modules de gameplay tels que ESP, aimbot et contrôle du recul, avec des composants séparés pour l'accès mémoire et la collecte de données.
Le code intègre les structures de jeu Unity et Mono et fournit des scripts pour la compilation, l'injection, l'extraction et le rechargement rapide.
Il s'adresse aux chercheurs en game hacking qui expérimentent des configurations externes ou hybrides Linux/QEMU.
