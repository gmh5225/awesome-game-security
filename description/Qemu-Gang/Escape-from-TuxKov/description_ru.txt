Этот проект — C++-фреймворк чита для Escape from Tarkov, рассчитанный на работу через рабочие процессы на базе QEMU.
Он включает игровые модули, такие как ESP, аимбот и контроль отдачи, а также отдельные компоненты для доступа к памяти и сбора данных.
Код интегрирует игровые структуры Unity и Mono и предоставляет скрипты для сборки, инъекции, извлечения и быстрой перезагрузки.
Предназначен для исследователей game hacking, экспериментирующих с внешними или гибридными конфигурациями Linux/QEMU.
