Este proyecto es un port a QEMU y memflow de un dumper de Unreal Engine 4.25 para extraer metadatos en tiempo de ejecución. Está implementado en C++ y analiza procesos objetivo para localizar matrices de objetos, pools de nombres y offsets específicos del motor. La herramienta puede volcar nombres y objetos, y está estructurada para que se puedan añadir offsets por juego en el código de configuración del motor. Se usa principalmente en flujos de ingeniería inversa de Unreal e investigación de seguridad en juegos.
