Этот проект — input-overlay, плагин OBS Studio, который отображает ввод с клавиатуры, мыши и геймпада в виде оверлея на стриме. Он захватывает события ввода в реальном времени и рисует визуальные индикаторы нажатых клавиш и кнопок, с настраиваемыми раскладками, скинами и позиционированием. Плагин на C++ интегрируется в конвейер рендеринга OBS. Предназначен для игровых стримеров и авторов контента, которые хотят показывать ввод контроллера/клавиатуры в прямом эфире.
