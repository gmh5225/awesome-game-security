Questo progetto è input-overlay, un plugin di OBS Studio che mostra gli input di tastiera, mouse e gamepad come overlay sullo stream. Cattura eventi di input in tempo reale e li rende come indicatori visivi che mostrano quali tasti e pulsanti sono premuti, con layout, skin e posizionamento personalizzabili. Il plugin C++ si integra con la pipeline di rendering di OBS. È rivolto a streamer di giochi e content creator che vogliono mostrare gli input di controller/tastiera nei loro livestream.
