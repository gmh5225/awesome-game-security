Este proyecto es una colección en C# de técnicas de ofuscación y protección de ensamblados .NET para dificultar la ingeniería inversa del código administrado. Ofrece muchas protecciones modulares como ofuscación del flujo de control, anti-dump y ocultación de métodos, defensas frente a de4dot/dnSpy/ildasm, renombrado, cifrado de enteros y cadenas, proxies, mutación, código basura, metadatos/opcodes inválidos y trucos relacionados. Las implementaciones están escritas en C# y dependen de la biblioteca dnlib para cargar, reescribir y guardar ensamblados. Está dirigido a desarrolladores e ingenieros inversos que quieran estudiar o aplicar ofuscación .NET para protección de software y casos de uso anti-análisis.
