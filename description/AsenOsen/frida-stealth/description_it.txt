frida-stealth è un set di patch che modifica Frida per ridurre le impronte di rilevamento runtime comuni sui target Android. Le patch alterano tratti identificativi come porte predefinite, nomi dei socket, nomi dei thread, etichette dei loop e relativi marker Frida in frida-core e frida-gum. Il repository fornisce file di patch e istruzioni di build per produrre binari strumentati personalizzati. È usato principalmente nella reverse engineering mobile e nella ricerca sul bypass anti-strumentazione.
