frida-stealth ist ein Patch-Set, das Frida so verändert, dass gängige Laufzeit-Erkennungsmerkmale auf Android-Zielen verringert werden. Die Patches ändern identifizierende Merkmale wie Standardports, Socket-Namen, Thread-Namen, Loop-Labels und verwandte Frida-Marker in frida-core und frida-gum. Das Repository stellt Patch-Dateien und Build-Anleitungen bereit, um benutzerdefinierte instrumentierte Binaries zu erzeugen. Es wird vor allem in der mobilen Reverse Engineering- und Anti-Instrumentierungs-Bypass-Forschung eingesetzt.
