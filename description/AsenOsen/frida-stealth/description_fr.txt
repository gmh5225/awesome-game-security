frida-stealth est un ensemble de correctifs qui modifie Frida afin de réduire les empreintes de détection d'exécution courantes sur les cibles Android. Les correctifs altèrent des traits d'identification tels que les ports par défaut, les noms de sockets, les noms de threads, les libellés de boucle et d'autres marqueurs Frida associés dans frida-core et frida-gum. Le dépôt fournit des fichiers de correctifs et des instructions de build pour produire des binaires instrumentés personnalisés. Il est principalement utilisé en reverse engineering mobile et dans la recherche sur le contournement anti-instrumentation.
