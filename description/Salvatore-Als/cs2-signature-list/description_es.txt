Este proyecto es un conjunto de referencia de notas sobre firmas de funciones de Counter-Strike 2, usadas para localizar rutinas internas del juego.
Está organizado principalmente como documentación Markdown y un script auxiliar IDC, con páginas separadas para funciones como cambio de equipo, entrega de objetos, manejo del chat y rutinas relacionadas con el daño.
En lugar de ofrecer patrones universales exactos, documenta anclas de cadenas y orientación para refinar las búsquedas de firmas entre builds.
Es útil para la ingeniería inversa de juegos, la investigación sobre modding y el mantenimiento de actualizaciones de offsets y firmas en herramientas de seguridad.
