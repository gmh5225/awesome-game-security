Этот проект — инструментарий для GPU-ускоренного упрощения выражений Mixed Boolean-Arithmetic (MBA) на базе CUDA. Он включает несколько реализаций CUDA-ядер для вычисления и упрощения MBA, а также обработку данных в формате JSON и собственные вспомогательные библиотеки для cooperative groups, управления памятью и хеширования. В первую очередь полезен исследователям деобфускации, изучающим высокопроизводительные подходы к упрощению MBA с использованием параллелизма GPU.
