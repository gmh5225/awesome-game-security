Este proyecto es un kit de herramientas acelerado por GPU para la simplificación de expresiones Mixed Boolean-Arithmetic (MBA) mediante CUDA. Incluye varias implementaciones de kernels CUDA para la evaluación y simplificación MBA, junto con manejo de datos basado en JSON y bibliotecas auxiliares personalizadas para grupos cooperativos, gestión de memoria y hashing. Resulta especialmente útil para investigadores de desofuscación que estudian enfoques de alto rendimiento para la simplificación MBA mediante paralelismo en GPU.
