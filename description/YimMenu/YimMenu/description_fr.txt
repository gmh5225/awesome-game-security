YimMenu est désormais un dépôt axé sur la documentation, consacré aux interfaces de scripting Lua et aux fonctionnalités de support liées à l'écosystème du menu.
Le code source d'origine a été retiré, et le contenu restant porte principalement sur les références de l'API Lua, la documentation des commandes et des guides utilitaires.
Il inclut des instructions pratiques de configuration pour des services tels que LibreTranslate, utilisés par la fonctionnalité de traduction du chat.
Le dépôt est surtout utile aux développeurs de scripts et aux utilisateurs qui maintiennent des extensions basées sur Lua, plutôt qu'au développement du menu principal.
