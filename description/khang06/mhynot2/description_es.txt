Este proyecto se centra en la idea de que hacer trampas es malo, pero exigir un controlador de kernel para jugar a un juego (en su mayoría) para un solo jugador es peor.
Aunque es una herramienta para eludir las medidas anti-trampas, está pensada para investigación y experimentación, y no está diseñada específicamente para hacer trampas.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / juego: Genshin Impact.
