Dieses Projekt konzentriert sich auf Il2Cpp Dump für Genshin Impact. Das Programm erzeugt anschließend alle Ausgabedateien im aktuellen Arbeitsverzeichnis. Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Game-Engine-Explorer: Unity untersuchen.
