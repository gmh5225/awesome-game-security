Этот проект посвящён Il2Cpp Dump для Genshin Impact. Программа затем создаёт все выходные файлы в текущем рабочем каталоге. Он в первую очередь полезен исследователям безопасности игр и инженерам по реверс-инжинирингу, изучающим наступательные техники в области читов / исследования игровых движков: Unity.
