Этот проект посвящён CLI для управления Unity Editor из терминала — компиляция, тестирование, сборка, проверка GameObjects; более 80 команд, готовность к ИИ-агентам (плагин Claude Code, Agent Skills).
В основном написан на C# и сосредоточен на шейдерах, рендеринге и графике.
В первую очередь полезен разработчикам игр, программистам движков и исследователям графики, работающим в области game engine / game engine plugins:unity.
