Ce projet se concentre sur une CLI pour contrôler Unity Editor depuis le terminal — compiler, tester, construire, inspecter les GameObjects ; plus de 80 commandes, prêt pour les agents IA (plugin Claude Code, Agent Skills).
Il est principalement écrit en C# et s’articule autour des shaders, du rendu et du graphisme.
Il est surtout utile aux développeurs de jeux, programmeurs moteur et chercheurs en graphisme travaillant dans le domaine game engine / game engine plugins:unity.
