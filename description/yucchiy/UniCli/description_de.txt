Dieses Projekt konzentriert sich auf eine CLI zur Steuerung des Unity Editors vom Terminal aus — Kompilieren, Testen, Bauen, GameObjects inspizieren; über 80 Befehle, KI-Agent-tauglich (Claude-Code-Plugin, Agent Skills).
Es ist hauptsächlich in C# geschrieben und dreht sich um Shader-Arbeit, Rendering und Grafik.
Es ist vor allem nützlich für Spieleentwickler, Engine-Programmierer und Grafikforscher im Bereich game engine / game engine plugins:unity.
