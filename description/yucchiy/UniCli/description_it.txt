Questo progetto si concentra su una CLI per controllare Unity Editor dal terminale — compilare, testare, buildare, ispezionare GameObjects; oltre 80 comandi, pronto per agenti IA (plugin Claude Code, Agent Skills).
È scritto principalmente in C# e ruota intorno a shader, rendering e grafica.
È soprattutto utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell’area game engine / game engine plugins:unity.
