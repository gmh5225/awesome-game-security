Este proyecto se centra en una CLI para controlar Unity Editor desde la terminal — compilar, probar, construir, inspeccionar GameObjects; más de 80 comandos, listo para agentes de IA (plugin de Claude Code, Agent Skills).
Está escrito principalmente en C# y se centra en shaders, renderizado y gráficos.
Es especialmente útil para desarrolladores de juegos, programadores de motor e investigadores de gráficos que trabajan en el área game engine / game engine plugins:unity.
