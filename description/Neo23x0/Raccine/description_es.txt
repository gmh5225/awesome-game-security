Este proyecto es una herramienta anti-ransomware ligera para Windows que bloquea comandos destructivos de eliminación de copias sombra (shadow copies). Se registra como depurador de utilidades como vssadmin y wmic, y luego evalúa el comportamiento de la línea de comandos con reglas YARA. Cuando detecta patrones maliciosos, termina la cadena de procesos padre y registra eventos sin requerir un servicio de agente residente. La base de código combina componentes en C, C++ y C# y está pensada para operaciones de seguridad defensiva y prevención de incidentes.
