Ce projet est un outil anti-ransomware léger pour Windows qui bloque les commandes destructrices de suppression de copies d’ombre (shadow copies). Il s’enregistre comme débogueur pour des utilitaires tels que vssadmin et wmic, puis évalue le comportement en ligne de commande à l’aide de règles YARA. Lorsqu’il détecte des schémas malveillants, il termine la chaîne de processus parents et journalise les événements sans nécessiter de service agent résident. Le code source combine des composants C, C++ et C# et vise les opérations de sécurité défensive et la prévention d’incidents.
