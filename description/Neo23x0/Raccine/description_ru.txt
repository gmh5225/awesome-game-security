Этот проект — лёгкий инструмент защиты от программ-вымогателей для Windows, блокирующий разрушительные команды удаления теневых копий (shadow copies). Он регистрирует себя как отладчик для утилит вроде vssadmin и wmic, а затем оценивает поведение командной строки с помощью правил YARA. При обнаружении вредоносных шаблонов завершает цепочку родительских процессов и журналирует события без необходимости в постоянно работающей службе-агенте. Кодовая база сочетает компоненты на C, C++ и C# и предназначена для оборонительных операций безопасности и предотвращения инцидентов.
