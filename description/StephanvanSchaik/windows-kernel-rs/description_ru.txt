Этот проект — фреймворк на Rust для разработки драйверов ядра Windows. Он сочетает сгенерированные низкоуровневые привязки с более безопасными абстракциями для модулей ядра, ввода-вывода устройств, IRP, примитивов синхронизации и сценариев присоединения к процессам. В репозитории есть поэтапные примеры и каркас сборки, которые ведут разработчиков от базовой настройки драйвера к более структурированным реализациям. Проект рассчитан на исследователей и системных разработчиков, которые хотят изучать программирование ядра Windows на Rust в более безопасной архитектуре.
