Este proyecto es un framework de Rust para desarrollar controladores del kernel de Windows. Combina bindings generados de bajo nivel con abstracciones más seguras para módulos del kernel, E/S de dispositivos, IRP, primitivas de sincronización y flujos de adjunto de procesos. El repositorio incluye ejemplos por etapas y andamiaje de compilación que guían a los desarrolladores desde la configuración básica de un controlador hasta implementaciones más estructuradas. Está dirigido a investigadores y desarrolladores de sistemas que quieran explorar la programación del kernel de Windows con Rust en una arquitectura más segura.
