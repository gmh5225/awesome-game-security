Questo progetto è un framework Rust per lo sviluppo di driver del kernel Windows. Combina binding generati a basso livello con astrazioni più sicure per moduli kernel, I/O dei dispositivi, IRP, primitive di sincronizzazione e flussi di attachment dei processi. Il repository include esempi progressivi e scaffolding di build che guidano gli sviluppatori dalla configurazione di base di un driver fino a implementazioni più strutturate. È rivolto a ricercatori e sviluppatori di sistemi che vogliono esplorare la programmazione del kernel Windows con Rust in un'architettura più sicura.
