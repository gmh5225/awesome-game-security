Ce projet est un framework Rust pour développer des pilotes noyau Windows. Il combine des bindings générés de bas niveau avec des abstractions plus sûres pour les modules noyau, les E/S périphériques, les IRP, les primitives de synchronisation et les flux d'attachement de processus. Le dépôt inclut des exemples progressifs et une structure de build qui guident les développeurs, de la mise en place basique d'un pilote jusqu'à des implémentations plus structurées. Il s'adresse aux chercheurs et développeurs systèmes qui souhaitent explorer la programmation du noyau Windows avec Rust dans une architecture plus sûre.
