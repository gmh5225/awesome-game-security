Этот проект — расширение Ghidra, которое загружает Windows-минидампы как анализируемые программы.
Оно сопоставляет модули дампа с их адресами во время выполнения, импортирует приватную память и стеки потоков, а также сохраняет метаданные потоков для анализа.
В основном написано на Java с Gradle, расширяет внутренности загрузки PE и добавляет рабочий процесс просмотра потоков с поддержкой обхода стека.
Предназначено для исследователей обратной разработки и специалистов по реагированию на инциденты, которым нужно изучать дампы сбоев или посмертные образцы вредоносного ПО в Ghidra.
