Dieses Projekt ist eine Ghidra-Erweiterung, die Windows-Minidumps als analysierbare Programme lädt.
Sie mappt Dump-Module auf ihre Laufzeitadressen, importiert privaten Speicher und Thread-Stacks und speichert Thread-Metadaten für die Analyse.
Hauptsächlich in Java mit Gradle gebaut, erweitert sie die PE-Ladeinterna und ergänzt einen Thread-View-Workflow mit Stack-Walking-Unterstützung.
Sie richtet sich an Reverse Engineers und Incident Responder, die Crash-Dumps oder Post-Mortem-Malware-Samples in Ghidra untersuchen müssen.
