Este proyecto es una extensión de Ghidra que carga minidumps de Windows como programas analizables.
Asigna los módulos del volcado a sus direcciones en tiempo de ejecución, importa la memoria privada y las pilas de hilos, y almacena metadatos de hilos para el análisis.
Desarrollada principalmente en Java con Gradle, amplía la carga interna de PE y añade un flujo de trabajo de vista de hilos con soporte de recorrido de pila.
Está dirigida a ingenieros de ingeniería inversa y respondedores de incidentes que necesitan inspeccionar volcados de fallos o muestras de malware post mortem dentro de Ghidra.
