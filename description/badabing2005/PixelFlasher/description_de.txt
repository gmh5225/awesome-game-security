Plattformübergreifende GUI-Anwendung zum Flashen von Google-Pixel-Smartphones. Sie unterstützt Bootloader-Entsperrung, das Flashen von Factory- und OTA-Images, Rooting mit Magisk/KernelSU/APatch, Boot-Image-Patching sowie Backup-Verwaltung und wird als eigenständiges, mit Python erstelltes Executable ohne Laufzeitabhängigkeiten auf dem Hostsystem bereitgestellt.
