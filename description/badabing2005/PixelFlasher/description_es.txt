Aplicación gráfica multiplataforma para flashear teléfonos Google Pixel. Admite el desbloqueo del bootloader, el flasheo de imágenes factory y OTA, el root con Magisk/KernelSU/APatch, el parcheo de la imagen boot y la gestión de copias de seguridad. Se distribuye como un ejecutable autónomo construido en Python que no requiere dependencias en tiempo de ejecución en el sistema anfitrión.
