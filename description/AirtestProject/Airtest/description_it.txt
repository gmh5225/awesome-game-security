Questo progetto è un framework di automazione UI multipiattaforma per giochi e app mobili. È scritto principalmente in Python e usa il riconoscimento delle immagini per individuare gli elementi UI senza iniettare nel processo di destinazione. Il framework fornisce API di controllo dei dispositivi, esecuzione scalabile su device farm, interfacce a riga di comando e Python, report HTML e integrazione con un IDE e strumenti di gerarchia degli oggetti per interazioni UI avanzate. È usato principalmente da ingegneri QA e team di giochi o app che necessitano di automazione end-to-end ripetibile su Android, iOS e ambienti desktop.
