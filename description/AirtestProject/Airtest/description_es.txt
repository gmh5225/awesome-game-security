Este proyecto es un framework de automatización de UI multiplataforma para juegos y aplicaciones móviles. Está escrito principalmente en Python y usa reconocimiento de imágenes para localizar elementos de UI sin inyectar en el proceso objetivo. El framework ofrece APIs de control de dispositivos, ejecución escalable en granjas de dispositivos, interfaces de línea de comandos y Python, informes HTML, e integración con un IDE y herramientas de jerarquía de objetos para interacción avanzada de UI. Lo usan principalmente ingenieros de QA y equipos de juegos o apps que necesitan automatización de extremo a extremo repetible en Android, iOS y entornos de escritorio.
