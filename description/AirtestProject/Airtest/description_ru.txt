Этот проект — кроссплатформенный фреймворк UI-автоматизации для игр и мобильных приложений. Он в основном написан на Python и использует распознавание изображений для поиска элементов интерфейса без внедрения в целевой процесс. Фреймворк предоставляет API управления устройствами, масштабируемое выполнение на фермах устройств, интерфейсы командной строки и Python, HTML-отчёты, а также интеграцию с IDE и инструментами иерархии объектов для продвинутого взаимодействия с UI. Его в основном используют QA-инженеры и команды игр или приложений, которым нужна воспроизводимая сквозная автоматизация в средах Android, iOS и настольных систем.
