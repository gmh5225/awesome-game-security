Ce projet est un framework d'automatisation d'interface utilisateur multiplateforme pour les jeux et applications mobiles. Il est principalement écrit en Python et utilise la reconnaissance d'images pour localiser les éléments d'interface sans injection dans le processus cible. Le framework fournit des API de contrôle d'appareils, une exécution évolutive sur des fermes d'appareils, des interfaces en ligne de commande et Python, des rapports HTML, ainsi qu'une intégration avec un IDE et des outils de hiérarchie d'objets pour des interactions UI avancées. Il est principalement utilisé par les ingénieurs QA et les équipes jeux ou applications qui ont besoin d'une automatisation de bout en bout reproductible sur Android, iOS et les environnements de bureau.
