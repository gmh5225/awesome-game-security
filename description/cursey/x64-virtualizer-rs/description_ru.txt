Этот проект — экспериментальный виртуализирующий обфускатор x86-64, написанный на Rust. Он использует крейт iced-x86 для дизассемблирования сырых массивов байтов x86-64, переводит нативные инструкции в пользовательский байткод стековой машины с опкодами вроде Const, Load, Store, Add и Mul, а также JIT-ассемблирует мостовые процедуры vmenter и vmexit во время выполнения. Структура Machine хранит полный набор виртуальных регистров и указатель стека для исполнения переведённого байткода. Проект предназначен для исследователей безопасности, изучающих внутреннее устройство движков обфускации кода на основе ВМ и рассматривающих Rust как язык для инструментов бинарной трансформации.
