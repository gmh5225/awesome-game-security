Este proyecto es un ofuscador virtualizador x86-64 experimental escrito en Rust. Usa el crate iced-x86 para desensamblar arrays de bytes x86-64 en bruto, traduce las instrucciones nativas a un bytecode de máquina de pila personalizado con opcodes como Const, Load, Store, Add y Mul, y ensambla en tiempo de ejecución (JIT) las rutinas puente vmenter y vmexit. La estructura Machine mantiene un conjunto completo de registros virtuales y un puntero de pila para ejecutar el bytecode traducido. Está orientado a investigadores de seguridad que estudian los internos de motores de ofuscación de código basados en VM y exploran Rust como lenguaje para herramientas de transformación binaria.
