Questo progetto è un ofuscatore virtualizzante x86-64 sperimentale scritto in Rust. Usa il crate iced-x86 per disassemblare array di byte x86-64 grezzi, traduce le istruzioni native in un bytecode di macchina a stack personalizzato con opcode come Const, Load, Store, Add e Mul, e assembla a runtime (JIT) le routine di ponte vmenter e vmexit. La struttura Machine mantiene un set completo di registri virtuali e un puntatore allo stack per eseguire il bytecode tradotto. È rivolto ai ricercatori di sicurezza che studiano gli interni dei motori di ofuscazione del codice basati su VM e esplorano Rust come linguaggio per tooling di trasformazione binaria.
