本プロジェクトは、Rust で書かれた実験的な x86-64 仮想化難読化ツールです。iced-x86 クレートで生の x86-64 バイト配列を逆アセンブルし、ネイティブ命令を Const、Load、Store、Add、Mul などのオペコードを持つ独自のスタックマシンバイトコードへ変換し、実行時に vmenter と vmexit のブリッジルーチンを JIT アセンブルします。Machine 構造体は、変換後のバイトコード実行用に仮想レジスタ一式とスタックポインタを保持します。VM ベースのコード難読化エンジンの内部や、バイナリ変換ツールとしての Rust を研究するセキュリティ研究者向けです。
