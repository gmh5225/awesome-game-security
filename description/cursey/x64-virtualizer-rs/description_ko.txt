이 프로젝트는 Rust로 작성된 실험적인 x86-64 가상화 난독화기입니다. iced-x86 크레이트로 원시 x86-64 바이트 배열을 디스어셈블하고, 네이티브 명령을 Const, Load, Store, Add, Mul 같은 옵코드를 가진 사용자 정의 스택 머신 바이트코드로 변환하며, 런타임에 vmenter와 vmexit 브리지 루틴을 JIT 어셈블합니다. Machine 구조체는 변환된 바이트코드를 실행하기 위한 전체 가상 레지스터 집합과 스택 포인터를 유지합니다. VM 기반 코드 난독화 엔진의 내부를 연구하고 바이너리 변환 도구용 언어로서 Rust를 탐색하는 보안 연구자를 대상으로 합니다.
