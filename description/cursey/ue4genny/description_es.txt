Este proyecto es UE4Genny, un generador de SDK de Unreal Engine 4 en tiempo de ejecución escrito en C++. Escanea en runtime el sistema de reflexión de UE4 (UObject, UClass, UStruct, UEnum, UFunction) para generar cabeceras SDK C++ completas con jerarquías de clases, offsets de propiedades y firmas de funciones correctas. El SDK generado permite interactuar directamente con los objetos del juego para modding o investigación. Está dirigido a game hackers, modders e ingenieros inversos que necesitan dumps precisos del SDK de UE4 desde instancias de juego en ejecución.
