Ce projet est UE4Genny, un générateur d'SDK Unreal Engine 4 à l'exécution écrit en C++. Il analyse au runtime le système de réflexion d'UE4 (UObject, UClass, UStruct, UEnum, UFunction) pour produire des en-têtes SDK C++ complets avec les hiérarchies de classes, les offsets de propriétés et les signatures de fonctions appropriés. Le SDK généré permet d'interagir directement avec les objets du jeu à des fins de modding ou de recherche. Il s'adresse aux game hackers, moddeurs et reverse engineers qui ont besoin de dumps SDK UE4 précis issus d'instances de jeu en cours d'exécution.
