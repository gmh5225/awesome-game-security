本プロジェクトは UE4Genny で、C++ で書かれたランタイム Unreal Engine 4 SDK ジェネレーターです。実行時に UE4 のリフレクションシステム（UObject、UClass、UStruct、UEnum、UFunction）を走査し、適切なクラス階層、プロパティオフセット、関数シグネチャを含む完全な C++ SDK ヘッダーを生成します。生成された SDK により、MOD 作成や研究目的でゲームオブジェクトと直接やり取りできます。実行中のゲームインスタンスから正確な UE4 SDK ダンプを必要とするゲームハッカー、モッダー、リバースエンジニアを対象としています。
