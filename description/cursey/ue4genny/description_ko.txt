이 프로젝트는 C++로 작성된 런타임 Unreal Engine 4 SDK 생성기 UE4Genny입니다. 실행 시 UE4 리플렉션 시스템(UObject, UClass, UStruct, UEnum, UFunction)을 스캔하여 올바른 클래스 계층, 프로퍼티 오프셋, 함수 시그니처가 포함된 완전한 C++ SDK 헤더를 생성합니다. 생성된 SDK를 통해 모드 제작이나 연구 목적으로 게임 객체와 직접 상호작용할 수 있습니다. 실행 중인 게임 인스턴스에서 정확한 UE4 SDK 덤프가 필요한 게임 해커, 모더, 리버스 엔지니어를 대상으로 합니다.
