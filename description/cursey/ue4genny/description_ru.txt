Этот проект — UE4Genny, генератор SDK Unreal Engine 4 во время выполнения, написанный на C++. Он сканирует систему рефлексии UE4 (UObject, UClass, UStruct, UEnum, UFunction) в runtime и создаёт полные заголовочные файлы C++ SDK с корректными иерархиями классов, смещениями свойств и сигнатурами функций. Сгенерированный SDK позволяет напрямую взаимодействовать с игровыми объектами для моддинга или исследований. Ориентирован на гейм-хакеров, моддеров и специалистов по реверс-инжинирингу, которым нужны точные дампы UE4 SDK из работающих экземпляров игр.
