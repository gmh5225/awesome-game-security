Questo progetto è UE4Genny, un generatore di SDK Unreal Engine 4 a runtime scritto in C++. Analizza a runtime il sistema di riflessione di UE4 (UObject, UClass, UStruct, UEnum, UFunction) per generare header SDK C++ completi con gerarchie di classi, offset delle proprietà e firme delle funzioni corretti. L'SDK generato consente l'interazione diretta con gli oggetti di gioco per modding o ricerca. È rivolto a game hacker, modder e reverse engineer che necessitano di dump SDK UE4 accurati da istanze di gioco in esecuzione.
