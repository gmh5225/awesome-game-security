Этот проект — библиотека для генерации C++-совместимых SDK для сторонних приложений.
Единственная зависимость — PEGTL, и она нужна только при использовании встроенного парсера.
В основном полезна исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / кодогенерации SDK.
