Ce projet est une bibliothèque permettant de générer des SDK compatibles C++ pour des applications tierces.
La seule dépendance est PEGTL, nécessaire uniquement si vous utilisez le parseur inclus.
Elle est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine du cheat / de la génération de SDK.
