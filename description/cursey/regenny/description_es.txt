Este proyecto es ReGenny, una herramienta de ingeniería inversa para reconstruir de forma interactiva estructuras de datos a partir de la memoria. Ofrece un visor de memoria en tiempo real que mapea bytes sin procesar a layouts de structs definidos por el usuario, con soporte para structs anidados, arrays, punteros, enums y bitfields. La aplicación en C++ lee la memoria en vivo de un proceso y actualiza las vistas en tiempo real, permitiendo refinar de forma iterativa las definiciones de structs. Está orientada a game hackers e ingenieros inversos que necesitan reconstruir estructuras de datos desconocidas desde procesos en ejecución, especialmente para la generación de SDK.
