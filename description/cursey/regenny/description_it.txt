Questo progetto è ReGenny, uno strumento di reverse engineering per ricostruire in modo interattivo strutture dati dalla memoria. Fornisce un visualizzatore di memoria in tempo reale che mappa i byte grezzi su layout di struct definiti dall'utente, con supporto per struct annidate, array, puntatori, enum e bitfield. L'applicazione C++ legge la memoria live di un processo e aggiorna le viste in tempo reale, consentendo di affinare iterativamente le definizioni delle struct. È pensata per game hacker e reverse engineer che devono ricostruire strutture dati sconosciute da processi in esecuzione, in particolare per la generazione di SDK.
