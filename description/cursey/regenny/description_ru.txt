Этот проект — ReGenny, инструмент для реверс-инжиниринга, позволяющий интерактивно восстанавливать структуры данных из памяти. Он предоставляет просмотрщик памяти в реальном времени, который сопоставляет сырые байты с пользовательскими раскладками структур и поддерживает вложенные структуры, массивы, указатели, перечисления и битовые поля. C++-приложение читает живую память процесса и обновляет представления в реальном времени, что позволяет итеративно уточнять определения структур. Оно рассчитано на игровых хакеров и реверс-инженеров, которым нужно восстанавливать неизвестные структуры данных из работающих процессов, в частности для генерации SDK.
