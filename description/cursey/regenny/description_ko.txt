이 프로젝트는 ReGenny로, 메모리에서 데이터 구조를 대화형으로 재구성하는 리버스 엔지니어링 도구입니다. 원시 바이트를 사용자 정의 구조체 레이아웃에 매핑하는 실시간 메모리 뷰어를 제공하며, 중첩 구조체, 배열, 포인터, 열거형, 비트필드를 지원합니다. 이 C++ 애플리케이션은 실행 중인 프로세스 메모리를 읽고 뷰를 실시간으로 갱신하여 구조체 정의를 반복적으로 다듬을 수 있게 합니다. 실행 중인 프로세스에서 알 수 없는 데이터 구조를 재구성해야 하는 게임 해커와 리버스 엔지니어를 대상으로 하며, 특히 SDK 생성에 적합합니다.
