本プロジェクトは、IDA Pro 向けの AI 駆動リバースエンジニアリングプラグインに焦点を当てています。LLM による関数説明、セマンティックナレッジグラフ、RAG、MCP 連携を提供します。
Python と PySide6 で構築された IDAssist は、IDA Pro 9.0+ 内のドッキング可能なパネルとして動作し、LLM プロバイダ（OpenAI、Anthropic、Ollama、LiteLLM など）と連携して関数の解析、リネーム提案、コードに関する質問への回答、バイナリ全体の検索可能なナレッジグラフ構築を行います。
主にゲームセキュリティ研究者や、チート / IDA プラグイン分野で攻撃技術を研究するリバースエンジニアに役立ちます。
