Este proyecto se centra en un plugin de ingeniería inversa impulsado por IA para IDA Pro: explicación de funciones con LLM, grafo de conocimiento semántico, RAG e integración MCP.
Creado con Python y PySide6, IDAssist se ejecuta como un panel acoplable dentro de IDA Pro 9.0+ y se comunica con proveedores de LLM (OpenAI, Anthropic, Ollama, LiteLLM y más) para analizar funciones, sugerir renombrados, responder preguntas sobre el código y construir un grafo de conocimiento buscable de un binario completo.
Es especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
