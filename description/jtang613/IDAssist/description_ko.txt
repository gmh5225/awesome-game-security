이 프로젝트는 IDA Pro용 AI 기반 리버스 엔지니어링 플러그인에 초점을 둡니다. LLM 함수 설명, 시맨틱 지식 그래프, RAG, MCP 통합을 제공합니다.
Python과 PySide6로 구축된 IDAssist는 IDA Pro 9.0+ 안에서 도킹 가능한 패널로 실행되며, LLM 제공자(OpenAI, Anthropic, Ollama, LiteLLM 등)와 통신하여 함수를 분석하고, 이름 변경을 제안하며, 코드에 대한 질문에 답하고, 전체 바이너리의 검색 가능한 지식 그래프를 구축합니다.
주로 게임 보안 연구원과, 치트 / IDA 플러그인 영역에서 공격 기법을 연구하는 리버스 엔지니어에게 유용합니다.
