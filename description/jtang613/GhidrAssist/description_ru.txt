Этот проект — расширение LLM для Ghidra, обеспечивающее ИИ-помощь при реверс-инжиниринге.
Оно поддерживает любой API, совместимый с OpenAI v1, включая локальные модели (Ollama, LM-Studio, Open-WebUI) и облачных провайдеров (OpenAI, Anthropic, Azure).
В основном полезно исследователям безопасности игр и реверс-инженерам, изучающим атакующие техники в области читов / плагинов Ghidra.
