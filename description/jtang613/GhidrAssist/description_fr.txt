Ce projet est une extension LLM pour Ghidra qui apporte une assistance IA à la rétro-ingénierie.
Il prend en charge toute API compatible OpenAI v1, y compris les modèles locaux (Ollama, LM-Studio, Open-WebUI) et les fournisseurs cloud (OpenAI, Anthropic, Azure).
Il s'adresse principalement aux chercheurs en sécurité des jeux et aux ingénieurs inverses qui étudient les techniques offensives dans le domaine des cheats / plugins Ghidra.
