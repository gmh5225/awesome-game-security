Questo progetto è un'estensione LLM per Ghidra che abilita l'assistenza IA nel reverse engineering.
Supporta qualsiasi API compatibile con OpenAI v1, inclusi modelli locali (Ollama, LM-Studio, Open-WebUI) e provider cloud (OpenAI, Anthropic, Azure).
È soprattutto utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin Ghidra.
