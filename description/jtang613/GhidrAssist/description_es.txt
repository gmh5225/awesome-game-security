Este proyecto es una extensión LLM para Ghidra que aporta asistencia de IA en ingeniería inversa.
Admite cualquier API compatible con OpenAI v1, incluidos modelos locales (Ollama, LM-Studio, Open-WebUI) y proveedores en la nube (OpenAI, Anthropic, Azure).
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de Ghidra.
