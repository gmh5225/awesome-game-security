Este proyecto se centra en mCP para Ghidra.
Al implementar el Model Context Protocol, esta extensión permite que asistentes de IA externos, herramientas de análisis automatizado y scripts personalizados interactúen de forma fluida con las capacidades de análisis de Ghidra.
Es principalmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo de juegos / servidores MCP.
