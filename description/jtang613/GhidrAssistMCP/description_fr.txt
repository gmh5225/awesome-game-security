Ce projet se concentre sur le mCP pour Ghidra.
En implémentant le Model Context Protocol, cette extension permet aux assistants IA externes, aux outils d'analyse automatisés et aux scripts personnalisés d'interagir de manière transparente avec les capacités d'analyse de Ghidra.
Il est principalement utile aux développeurs de jeux, aux ingénieurs en rétro-ingénierie et aux créateurs d'outils travaillant dans le domaine du développement de jeux / serveurs MCP.
