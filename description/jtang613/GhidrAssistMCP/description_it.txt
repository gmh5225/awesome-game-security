Questo progetto si concentra su mCP per Ghidra.
Implementando il Model Context Protocol, questa estensione consente ad assistenti IA esterni, strumenti di analisi automatizzata e script personalizzati di interagire senza soluzione di continuità con le capacità di analisi di Ghidra.
È principalmente utile per sviluppatori di giochi, reverse engineer e costruttori di strumenti che lavorano nell'ambito dello sviluppo di giochi / server MCP.
