Этот проект посвящён mCP для Ghidra.
Реализуя Model Context Protocol, это расширение позволяет внешним ИИ-ассистентам, средствам автоматического анализа и пользовательским скриптам беспрепятственно взаимодействовать с аналитическими возможностями Ghidra.
Оно в первую очередь полезно разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов, работающим в области разработки игр / MCP-серверов.
