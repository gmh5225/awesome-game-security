Ghidra-nativeaot is a Ghidra analyzer and UI plugin that helps reverse engineers recover structure from .NET Native AOT binaries where symbols and FunctionID databases are unavailable. Written primarily in Java as a Gradle-built Ghidra extension, it reconstructs the full type hierarchy from method tables, annotates frozen objects such as string literals, and provides an interactive metadata browser. It locates ReadyToRun metadata via symbols or heuristic signature scanning and includes a refactoring engine for renaming virtual methods and related symbols across .NET 7/8 Native AOT layouts. The primary use case is reverse engineering stripped Native AOT executables for malware analysis, CTF challenges, and game or application security research.
