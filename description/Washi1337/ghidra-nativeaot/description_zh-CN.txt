这是一款 Ghidra 分析器与 UI 插件，帮助逆向工程师分析 .NET Native AOT 二进制（.NET 8 及更高版本引入），尤其适用于缺少符号、难以构建 FunctionID 数据库的场景。该扩展主要以 Java 编写，可从方法表重建完整类型层次结构，标注字符串字面量等冻结对象，并检测虚表重定向。它还提供交互式元数据浏览器，以及用于重命名虚方法及相关符号的重构引擎。插件可通过符号或启发式特征扫描定位 ReadyToRun 元数据，从而从剥离符号的 Native AOT 可执行文件中恢复结构。主要用例是在 Ghidra 中对 .NET Native AOT 程序进行逆向工程，服务于恶意软件分析、CTF 挑战及其他安全研究。
