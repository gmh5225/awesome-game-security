Это анализатор и UI-плагин для Ghidra, помогающий инженерам по реверс-инжинирингу разбирать бинарники .NET Native AOT (появившиеся в .NET 8 и новее), особенно когда символы отсутствуют и базы FunctionID сложно построить. Написан в основном на Java как расширение Ghidra: восстанавливает полные иерархии типов из таблиц методов, аннотирует замороженные объекты вроде строковых литералов и обнаруживает перенаправления vtable. Также предоставляет интерактивный браузер метаданных и движок рефакторинга для переименования виртуальных методов и связанных символов. Плагин находит метаданные ReadyToRun по символам или эвристическому сканированию сигнатур, чтобы восстанавливать структуру из stripped-исполняемых файлов Native AOT. Основной сценарий — реверс-инжиниринг программ .NET Native AOT в Ghidra для анализа вредоносного ПО, CTF и других исследований безопасности.
