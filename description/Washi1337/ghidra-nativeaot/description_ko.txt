이 프로젝트는 .NET Native AOT 바이너리(.NET 8 이상에서 도입)를 분석하도록 돕는 Ghidra 분석기 및 UI 플러그인으로, 특히 심볼이 없고 FunctionID 데이터베이스를 만들기 어려울 때 유용합니다. 주로 Java로 작성된 Ghidra 확장으로, 메서드 테이블에서 전체 타입 계층을 재구성하고, 문자열 리터럴과 같은 고정(frozen) 객체에 주석을 달며, vtable 리다이렉션을 탐지합니다. 대화형 메타데이터 브라우저와 가상 메서드 및 관련 심볼 이름 변경을 위한 리팩터링 엔진도 제공합니다. 심볼 또는 휴리스틱 시그니처 스캔으로 ReadyToRun 메타데이터를 찾아, 심볼이 제거된 Native AOT 실행 파일에서 구조를 복구할 수 있습니다. 주요 사용 사례는 멀웨어 분석, CTF, 기타 보안 연구를 위한 Ghidra에서의 .NET Native AOT 프로그램 리버스 엔지니어링입니다.
