これは、.NET Native AOT バイナリ（.NET 8 以降で導入）の解析を支援する Ghidra アナライザおよび UI プラグインで、特にシンボルが欠落し FunctionID データベースの構築が難しい場合に有用です。主に Java で書かれた Ghidra 拡張として、メソッドテーブルから完全な型階層を再構築し、文字列リテラルなどの凍結オブジェクトに注釈を付け、vtable のリダイレクトを検出します。対話的なメタデータブラウザと、仮想メソッドおよび関連シンボルのリネーム用リファクタリングエンジンも提供します。シンボルまたはヒューリスティックなシグネチャスキャンにより ReadyToRun メタデータを特定し、シンボルが剥がされた Native AOT 実行ファイルから構造を復元できます。主な用途は、マルウェア解析、CTF、その他のセキュリティ研究における Ghidra 上での .NET Native AOT プログラムのリバースエンジニアリングです。
