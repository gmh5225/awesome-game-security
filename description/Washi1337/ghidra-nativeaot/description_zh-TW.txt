這是一款 Ghidra 分析器與 UI 外掛，協助逆向工程師分析 .NET Native AOT 二進位檔（.NET 8 及更高版本引入），尤其適用於缺少符號、難以建置 FunctionID 資料庫的情況。此擴充主要以 Java 撰寫，可從方法表重建完整型別階層，標註字串常值等凍結物件，並偵測虛表重新導向。它也提供互動式中繼資料瀏覽器，以及用於重新命名虛擬方法與相關符號的重構引擎。外掛可透過符號或啟發式特徵掃描定位 ReadyToRun 中繼資料，從而從已剝除符號的 Native AOT 可執行檔中還原結構。主要用途是在 Ghidra 中對 .NET Native AOT 程式進行逆向工程，服務於惡意軟體分析、CTF 挑戰及其他安全研究。
