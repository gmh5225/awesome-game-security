Il s'agit d'un analyseur Ghidra et d'un plugin d'interface qui aide les reverse engineers à analyser les binaires .NET Native AOT (introduits à partir de .NET 8), surtout lorsque les symboles manquent et que les bases FunctionID sont difficiles à construire. Écrit principalement en Java comme extension Ghidra, il reconstruit les hiérarchies de types complètes à partir des tables de méthodes, annote les objets gelés tels que les littéraux de chaînes, et détecte les redirections de vtable. Il fournit aussi un navigateur de métadonnées interactif et un moteur de refactoring pour renommer les méthodes virtuelles et les symboles associés. Le plugin localise les métadonnées ReadyToRun via les symboles ou un balayage heuristique de signatures, afin de retrouver la structure dans des exécutables Native AOT dépouillés. Son cas d'usage principal est le reverse engineering de programmes .NET Native AOT dans Ghidra pour l'analyse de malwares, les CTF et d'autres recherches en sécurité.
