Se trata de un analizador de Ghidra y un complemento de interfaz que ayuda a los ingenieros inversos a analizar binarios .NET Native AOT (introducidos en .NET 8 y posteriores), especialmente cuando faltan símbolos y las bases de datos FunctionID son difíciles de construir. Escrito principalmente en Java como extensión de Ghidra, reconstruye jerarquías de tipos completas a partir de tablas de métodos, anota objetos congelados como literales de cadena y detecta redirecciones de vtable. También ofrece un navegador interactivo de metadatos y un motor de refactorización para renombrar métodos virtuales y símbolos relacionados. El complemento localiza metadatos ReadyToRun mediante símbolos o un escaneo heurístico de firmas, de modo que los analistas puedan recuperar la estructura de ejecutables Native AOT sin símbolos. Su caso de uso principal es la ingeniería inversa de programas .NET Native AOT en Ghidra para análisis de malware, retos CTF y otras investigaciones de seguridad.
