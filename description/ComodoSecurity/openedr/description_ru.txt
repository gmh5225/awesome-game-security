Этот проект — открытая платформа Endpoint Detection and Response от Comodo Security. Обширная кодовая база на C++ реализует мониторинг системы, сбор событий, обнаружение угроз и интеграцию облачного анализа через подключение AWS SDK. Она предоставляет возможности телеметрии конечных точек и реагирования в реальном времени для выявления вредоносной активности в системах Windows. В основном полезна исследователям безопасности и инженерам античита, изучающим архитектуру EDR, методы мониторинга конечных точек и проектирование оборонительных систем безопасности.
