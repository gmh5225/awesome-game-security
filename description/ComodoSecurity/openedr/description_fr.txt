Ce projet est une plateforme open source d'Endpoint Detection and Response développée par Comodo Security. La vaste base de code C++ implémente la surveillance système, la collecte d'événements, la détection de menaces et l'intégration d'analyses cloud via la connectivité AWS SDK. Elle fournit des capacités de télémétrie et de réponse en temps réel pour identifier les activités malveillantes sur les systèmes Windows. Elle est principalement utile aux chercheurs en sécurité et aux ingénieurs anti-triche qui étudient l'architecture EDR, les techniques de surveillance des endpoints et la conception de systèmes de sécurité défensive.
