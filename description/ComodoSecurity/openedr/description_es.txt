Este proyecto es una plataforma de Endpoint Detection and Response de código abierto de Comodo Security. La amplia base de código en C++ implementa supervisión del sistema, recopilación de eventos, detección de amenazas e integración de análisis en la nube mediante conectividad con el AWS SDK. Proporciona capacidades de telemetría y respuesta en tiempo real para identificar actividad maliciosa en sistemas Windows. Es especialmente útil para investigadores de seguridad e ingenieros de anti-trampas que estudian la arquitectura EDR, las técnicas de supervisión de endpoints y el diseño de sistemas de seguridad defensiva.
