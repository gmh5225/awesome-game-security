Questo progetto è una piattaforma open source di Endpoint Detection and Response di Comodo Security. L'ampia codebase in C++ implementa monitoraggio di sistema, raccolta di eventi, rilevamento delle minacce e integrazione dell'analisi basata su cloud tramite connettività AWS SDK. Fornisce capacità di telemetria e risposta in tempo reale per identificare attività dannose sui sistemi Windows. È particolarmente utile per ricercatori di sicurezza e ingegneri anti-cheat che studiano l'architettura EDR, le tecniche di monitoraggio degli endpoint e la progettazione di sistemi di sicurezza difensiva.
