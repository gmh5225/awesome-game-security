Questo progetto si concentra sul consumo dal provider ETW Threat-Intelligence senza driver né privilegio PPL.
È stato formalmente testato su Windows 11 24H2 e Windows 11 25H2 (insieme all'ultima build Canary Insider Preview al momento del caricamento di questo POC).
È principalmente utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / esplorazione del kernel Windows.
