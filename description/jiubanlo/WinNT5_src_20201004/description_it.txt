Questo progetto ospita materiale sorgente estratto di Windows NT 5.x (era Windows 2000 / XP / Server 2003), organizzato sotto un albero NT/ che copre base, drivers, windows, net, shell e componenti correlati.
È un ampio corpus storico di sorgenti Windows utile per studiare i dettagli di implementazione del kernel, di Win32 e dei sottosistemi di quella generazione.
È principalmente utile per ricercatori di sicurezza del kernel e della piattaforma Windows che analizzano gli interni dell'era NT5.
