Dieses Projekt hostet extrahiertes Windows-NT-5.x-Quellenmaterial (Ära Windows 2000 / XP / Server 2003), organisiert unter einem NT/-Baum mit base, drivers, windows, net, shell und verwandten Komponenten.
Es ist ein großes historisches Windows-Quellenkorpus, nützlich zum Studium von Kernel-, Win32- und Subsystem-Implementierungsdetails jener Generation.
Es eignet sich vor allem für Windows-Kernel- und Plattformsicherheitsforscher, die NT5-Ära-Internals analysieren.
