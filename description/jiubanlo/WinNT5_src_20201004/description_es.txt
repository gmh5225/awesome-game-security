Este proyecto aloja material fuente extraído de Windows NT 5.x (era Windows 2000 / XP / Server 2003), organizado bajo un árbol NT/ que cubre base, drivers, windows, net, shell y componentes relacionados.
Es un amplio corpus histórico de código fuente de Windows útil para estudiar detalles de implementación del kernel, Win32 y subsistemas de esa generación.
Resulta especialmente útil para investigadores de seguridad del kernel y la plataforma Windows que analizan los internos de la era NT5.
