В этом проекте размещён извлечённый исходный материал Windows NT 5.x (эпоха Windows 2000 / XP / Server 2003), организованный в дереве NT/ с разделами base, drivers, windows, net, shell и связанными компонентами.
Это крупный исторический корпус исходников Windows, полезный для изучения деталей реализации ядра, Win32 и подсистем того поколения.
В основном полезен исследователям безопасности ядра и платформы Windows, анализирующим внутренности эпохи NT5.
