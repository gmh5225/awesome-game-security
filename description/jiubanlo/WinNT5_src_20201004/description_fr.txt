Ce projet héberge du matériel source Windows NT 5.x extrait (ère Windows 2000 / XP / Server 2003), organisé sous une arborescence NT/ couvrant base, drivers, windows, net, shell et composants associés.
Il s'agit d'un vaste corpus historique de sources Windows utile pour étudier les détails d'implémentation du noyau, de Win32 et des sous-systèmes de cette génération.
Il est surtout destiné aux chercheurs en sécurité du noyau et de la plateforme Windows analysant les internes de l'ère NT5.
