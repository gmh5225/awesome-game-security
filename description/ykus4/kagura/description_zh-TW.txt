Kagura 是一套基於 LLVM 的程式碼混淆與防竄改工具包，用於保護行動端、桌面端與 WebAssembly 目標上的原生程式碼。它以 New Pass Manager 外掛（LLVM 17+）形式提供，透過 -fpass-plugin 載入，無需修改 LLVM 原始碼樹，主要以 C++ 與 C 實作。Pass 涵蓋控制流平坦化與虛假控制流、字串與資料加密、記憶體值混淆、匯入間接化、基本區塊完整性檢查，以及相關的反分析轉換。配套執行時期提供反偵錯、Hook 與中斷點偵測、完整性檢查，以及越獄或 Root 偵測等平台特定防護。它可與 Android NDK、iOS/Xcode、CMake、Bazel、Unity 和 Unreal 整合，面向遊戲保護、DRM、SDK 強化及類似的反逆向工程場景。
