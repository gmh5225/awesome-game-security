Kagura は、モバイル、デスクトップ、WebAssembly 向けのネイティブコードを保護するための、LLVM ベースのコード難読化および改ざん防止ツールキットです。LLVM 17+ の New Pass Manager プラグインとして -fpass-plugin 経由で読み込まれ、LLVM ソースツリーのパッチは不要で、主に C++ と C で実装されています。パスには制御フロー平坦化と偽制御フロー、文字列・データ暗号化、メモリ値難読化、インポート間接化、基本ブロック整合性チェック、および関連する解析妨害変換が含まれます。付属のランタイムはアンチデバッグ、フック／ブレークポイント検出、整合性チェック、ジェイルブレイク／ルート検出などのプラットフォーム固有防御を提供します。Android NDK、iOS/Xcode、CMake、Bazel、Unity、Unreal と連携でき、ゲーム保護、DRM、SDK 強化、および同様のリバースエンジニアリング耐性用途を想定しています。
