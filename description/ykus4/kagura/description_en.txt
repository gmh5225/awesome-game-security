Kagura is an LLVM-based code obfuscation and anti-tamper toolkit for protecting native code on mobile, desktop, and WebAssembly targets. It ships as a New Pass Manager plugin (LLVM 17+) loaded via -fpass-plugin, with no need to patch the LLVM source tree, and is implemented mainly in C++ and C. Passes cover control-flow flattening and bogus control flow, string and data encryption, memory-value obfuscation, import indirection, basic-block integrity checks, and related anti-analysis transforms. A companion runtime adds anti-debug, hook and breakpoint detection, integrity checks, and platform-specific defenses such as jailbreak or root detection. It integrates with Android NDK, iOS/Xcode, CMake, Bazel, Unity, and Unreal, and is aimed at game protection, DRM, SDK hardening, and similar reverse-engineering resistance use cases.
