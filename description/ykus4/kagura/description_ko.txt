Kagura는 모바일, 데스크톱, WebAssembly 대상의 네이티브 코드를 보호하기 위한 LLVM 기반 코드 난독화 및 변조 방지 툴킷입니다. LLVM 17+ New Pass Manager 플러그인으로 -fpass-plugin을 통해 로드되며 LLVM 소스 트리를 패치할 필요가 없고, 주로 C++와 C로 구현되어 있습니다. 패스는 제어 흐름 평탄화와 가짜 제어 흐름, 문자열·데이터 암호화, 메모리 값 난독화, 임포트 간접화, 기본 블록 무결성 검사 및 관련 분석 방해 변환을 다룹니다. 동반 런타임은 안티디버그, 훅·브레이크포인트 탐지, 무결성 검사, 탈옥·루트 탐지 같은 플랫폼별 방어를 제공합니다. Android NDK, iOS/Xcode, CMake, Bazel, Unity, Unreal과 통합되며, 게임 보호, DRM, SDK 강화 및 유사한 리버스 엔지니어링 저항 용도를 목표로 합니다.
