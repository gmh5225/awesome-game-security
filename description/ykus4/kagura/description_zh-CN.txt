Kagura 是一套基于 LLVM 的代码混淆与防篡改工具包，用于保护移动端、桌面端和 WebAssembly 目标上的原生代码。它以 New Pass Manager 插件（LLVM 17+）形式提供，通过 -fpass-plugin 加载，无需修补 LLVM 源码树，主要用 C++ 和 C 实现。Pass 覆盖控制流平坦化与虚假控制流、字符串与数据加密、内存值混淆、导入间接化、基本块完整性校验，以及相关的反分析变换。配套运行时提供反调试、Hook 与断点检测、完整性检查，以及越狱或 Root 检测等平台特定防护。它可与 Android NDK、iOS/Xcode、CMake、Bazel、Unity 和 Unreal 集成，面向游戏保护、DRM、SDK 加固及类似的反逆向工程场景。
