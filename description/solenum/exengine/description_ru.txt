Этот проект — exengine, минималистичный 3D-игровой движок, написанный на C с использованием OpenGL. Он обеспечивает базовый 3D-рендеринг с освещением, загрузку моделей (формат IQM), управление сценами, отрисовку текста, обработку ввода и простую систему сущностей. Движок делает акцент на простоте и читаемости как учебный ресурс для понимания основ игровых движков. Он предназначен для любителей разработки игр и студентов, изучающих архитектуру 3D-движков с нуля на языке C.
