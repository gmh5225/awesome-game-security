Dieses Projekt ist exengine, eine minimale 3D-Spiel-Engine, geschrieben in C mit OpenGL. Sie bietet grundlegende 3D-Darstellung mit Beleuchtung, Modellladen (IQM-Format), Szenenverwaltung, Textrendering, Eingabebehandlung und ein einfaches Entity-System. Die Engine legt Wert auf Einfachheit und Lesbarkeit und dient als Lernressource zum Verständnis der Grundlagen von Spiel-Engines. Sie richtet sich an Hobby-Spieleentwickler und Studierende, die 3D-Engine-Architektur von Grund auf in C erlernen.
