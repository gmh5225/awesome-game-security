Questo progetto è un toolkit C++ per lo spoofing dello stack di chiamate, pensato per rendere più difficile l'analisi basata sullo stack sia in user mode sia in kernel mode. Fornisce macro e template per spoofare i frame delle funzioni e proxyare le chiamate attraverso percorsi di shellcode generati. L'implementazione mira agli ambienti x64 e discute vincoli pratici come le impostazioni del compilatore e le protezioni del control flow. Il caso d'uso principale è la ricerca su anti-analisi e evasione degli anti-cheat, dove conta la visibilità dello stackwalk.
