Этот проект — концептуальное доказательство (proof of concept) для ядра Windows, предназначенное для перехвата диспетчерской процедуры IRP_MJ_DEVICE_CONTROL драйвера. Он демонстрирует перенаправление указателя диспетчеризации с использованием подхода trampoline и пересылает трафик ioctl через пользовательскую логику обработки. Пример включает обработчики запросов копирования/выделения/защиты памяти процесса и поведение поиска базового адреса модуля. Основной сценарий использования — изучение хуков коммуникации драйверов и методов обнаружения античита, связанных с перехватом ioctl.
