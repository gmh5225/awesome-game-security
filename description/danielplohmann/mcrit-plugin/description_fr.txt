Un plugin IDA Pro qui s'intègre à MCRIT (MinHash-based Code Recognition and Intelligence Toolkit) pour effectuer la correspondance de similarité de fonctions binaires et la reconnaissance de code directement depuis le désassembleur.
Il fournit une interface graphique pour téléverser des échantillons, interroger les correspondances de fonctions, parcourir les scores de similarité et gérer une connexion au serveur MCRIT dans l'interface d'IDA.
Il est principalement utile aux analystes de malwares et aux ingénieurs en rétro-ingénierie qui réalisent une analyse de similarité de code binaire à grande échelle et l'identification de fonctions.
