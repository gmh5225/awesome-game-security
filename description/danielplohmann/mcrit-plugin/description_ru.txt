Плагин для IDA Pro, интегрирующийся с MCRIT (MinHash-based Code Recognition and Intelligence Toolkit) для сопоставления сходства бинарных функций и распознавания кода непосредственно из дизассемблера.
Предоставляет графический интерфейс для загрузки образцов, запроса совпадений функций, просмотра оценок сходства и управления подключением к серверу MCRIT в интерфейсе IDA.
В основном полезен аналитикам вредоносного ПО и специалистам по реверс-инжинирингу, выполняющим крупномасштабный анализ сходства бинарного кода и идентификацию функций.
