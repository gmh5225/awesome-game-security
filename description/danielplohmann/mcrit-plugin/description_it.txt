Un plugin per IDA Pro che si integra con MCRIT (MinHash-based Code Recognition and Intelligence Toolkit) per eseguire il matching di similarità delle funzioni binarie e il riconoscimento del codice direttamente dal disassemblatore.
Fornisce una GUI per caricare campioni, interrogare corrispondenze di funzioni, consultare i punteggi di similarità e gestire una connessione al server MCRIT nell'interfaccia di IDA.
È soprattutto utile per analisti di malware e reverse engineer che effettuano analisi di similarità del codice binario su larga scala e identificazione delle funzioni.
