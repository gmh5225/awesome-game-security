Ein IDA-Pro-Plugin, das sich mit MCRIT (MinHash-based Code Recognition and Intelligence Toolkit) integriert, um binäre Funktionsähnlichkeitsvergleiche und Code-Erkennung direkt im Disassembler durchzuführen.
Es bietet eine GUI zum Hochladen von Samples, Abfragen von Funktionsübereinstimmungen, Durchsuchen von Ähnlichkeitswerten und Verwalten einer MCRIT-Serververbindung innerhalb der IDA-Oberfläche.
Es ist vor allem für Malware-Analysten und Reverse Engineers nützlich, die groß angelegte Binärcode-Ähnlichkeitsanalysen und Funktionsidentifikation durchführen.
