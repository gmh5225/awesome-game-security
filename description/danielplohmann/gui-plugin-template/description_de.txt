Eine Python-Plugin-Vorlage, die ein plattformübergreifend kompatibles GUI-Framework bereitstellt, um Plugins mit PyQt/PySide gleichzeitig für IDA Pro, Ghidra, Binary Ninja und Cutter zu entwickeln.
Sie abstrahiert die API jedes Tools über eine gemeinsame, harmonisierte Schnittstellenschicht, sodass Plugin-Entwickler eine einzige GUI-Codebasis schreiben können, die auf allen vier Plattformen läuft.
Sie ist vor allem für Reverse Engineers und Sicherheitsforscher nützlich, die portable Plugins zur Binäranalyse entwickeln möchten, ohne getrennte Codebasen pro Tool zu pflegen.
