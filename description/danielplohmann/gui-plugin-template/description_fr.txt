Un modèle de plugin Python qui fournit un framework GUI multi-compatible pour créer des plugins fonctionnant simultanément sous IDA Pro, Ghidra, Binary Ninja et Cutter avec PyQt/PySide.
Il abstrait l’API de chaque outil via une couche d’interface harmonisée commune, permettant aux développeurs d’écrire une seule base de code GUI qui s’exécute sur les quatre plateformes.
Il est surtout utile aux reverse engineers et chercheurs en sécurité qui souhaitent développer des plugins d’analyse binaire portables sans maintenir des bases de code distinctes par outil.
