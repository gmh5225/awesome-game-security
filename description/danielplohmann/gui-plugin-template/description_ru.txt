Шаблон плагина на Python, предоставляющий кросс-совместимый GUI-фреймворк для создания плагинов, которые одновременно работают в IDA Pro, Ghidra, Binary Ninja и Cutter с использованием PyQt/PySide.
Он абстрагирует API каждого инструмента через общий унифицированный слой интерфейса, позволяя разработчикам писать единую GUI-кодовую базу, работающую на всех четырёх платформах.
В первую очередь полезен реверс-инженерам и исследователям безопасности, которым нужны переносимые плагины анализа бинарных файлов без поддержки отдельных кодовых баз под каждый инструмент.
