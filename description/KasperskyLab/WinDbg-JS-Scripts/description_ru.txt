Этот проект — набор JavaScript-скриптов для WinDbg, помогающих аналитикам исследовать дампы памяти Windows. Он предоставляет команды для поиска кандидатов exception record, обхода структур STL map, исправления повреждённых noexcept stack traces и проверки x86-стеков в x64 kernel dumps. Кодовая база в основном на JavaScript, с сопутствующими манифестными XML-файлами и небольшим вспомогательным Python-скриптом для связанных отладочных сценариев. Предназначен для reverse engineers и исследователей игровой безопасности, которым нужен более быстрый низкоуровневый triage дампов при анализе античита и вредоносного ПО.
