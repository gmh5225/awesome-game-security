Questo progetto è una raccolta di script JavaScript per WinDbg che aiuta gli analisti a esaminare i dump di memoria Windows. Fornisce comandi per trovare candidati a record di eccezione, attraversare strutture STL map, riparare stack trace noexcept danneggiati e ispezionare stack x86 nei dump del kernel x64. Il codice è principalmente JavaScript, con file manifest XML di supporto e un piccolo script Python di aiuto per flussi di debugging correlati. È rivolto a reverse engineer e ricercatori di sicurezza nei videogiochi che necessitano di un triage più rapido dei dump a basso livello durante l'analisi di anti-cheat e malware.
