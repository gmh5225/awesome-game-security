Ce projet est une collection de scripts JavaScript pour WinDbg qui aide les analystes à examiner les dumps mémoire Windows. Il fournit des commandes pour rechercher des candidats d'enregistrements d'exception, parcourir des structures STL map, réparer des traces de pile noexcept cassées et inspecter des piles x86 dans des dumps noyau x64. Le code est principalement en JavaScript, avec des fichiers manifeste XML associés et un petit script d'aide Python pour des flux de débogage connexes. Il s'adresse aux reverse engineers et aux chercheurs en sécurité des jeux qui ont besoin d'un triage plus rapide des dumps bas niveau lors d'analyses anti-cheat et malware.
