Este proyecto es una colección de scripts JavaScript para WinDbg que ayuda a los analistas a examinar volcados de memoria de Windows. Ofrece comandos para encontrar candidatos a registros de excepción, recorrer estructuras STL map, reparar rastros de pila noexcept rotos e inspeccionar pilas x86 en volcados de kernel x64. El código es principalmente JavaScript, con archivos de manifiesto XML de apoyo y un pequeño script auxiliar en Python para flujos de depuración relacionados. Está orientado a ingenieros inversos e investigadores de seguridad en juegos que necesitan un triaje más rápido de volcados de bajo nivel durante el análisis de anti-cheat y malware.
