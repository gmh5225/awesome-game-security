Vanguard Update Notifier 是一個 Python Discord 機器人，用於監視 Riot Vanguard 反作弊的發布，並在版本或捆綁檔案雜湊值變更時向已設定的頻道發送提醒。它會輪詢 Riot 的公開 clientconfig API 以取得目前 Vanguard 版本與安裝程式 URL，下載安裝程式壓縮檔，並以 py7zr 與 SHA-256 對解壓內容計算雜湊；若解壓失敗，則改為對原始 setup.exe 計算雜湊。透過 discord.py 與 aiohttp，每兩小時執行一次定期檢查，並提供斜線指令用於註冊或移除通知頻道，狀態保存在本機 JSON 設定中。主要面向希望及時在 Discord 收到 Vanguard 反作弊更新提醒的遊戲安全研究者與社群。
