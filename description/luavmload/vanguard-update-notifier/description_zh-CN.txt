Vanguard Update Notifier 是一个 Python Discord 机器人，用于监视 Riot Vanguard 反作弊的发布，并在版本或捆绑文件哈希发生变化时向已配置的频道发送提醒。它会轮询 Riot 的公开 clientconfig API 以获取当前 Vanguard 版本与安装包 URL，下载安装程序压缩包，并用 py7zr 与 SHA-256 对解压内容计算哈希；若解压失败，则回退为对原始 setup.exe 计算哈希。借助 discord.py 与 aiohttp，每两小时执行一次定期检查，并提供斜杠命令用于注册或移除通知频道，状态保存在本地 JSON 配置中。主要面向希望及时在 Discord 收到 Vanguard 反作弊更新提醒的游戏安全研究者与社区。
