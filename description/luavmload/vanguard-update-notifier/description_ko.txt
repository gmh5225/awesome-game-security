Vanguard Update Notifier는 Riot Vanguard 안티치트 릴리스를 감시하고, 버전 또는 번들 파일 해시가 변경되면 설정된 채널에 알림을 보내는 Python Discord 봇입니다. Riot의 공개 clientconfig API를 폴링하여 현재 Vanguard 버전과 설치 URL을 가져오고, 설치 프로그램 아카이브를 다운로드한 뒤 py7zr와 SHA-256으로 추출된 내용의 해시를 계산합니다. 추출에 실패하면 원본 setup.exe 해시로 대체합니다. discord.py와 aiohttp를 통해 2시간마다 주기적으로 확인하며, 알림 채널을 등록하거나 제거하는 슬래시 명령을 제공하고 상태는 로컬 JSON 설정에 저장됩니다. 주요 사용 사례는 Vanguard 안티치트 업데이트에 대한 적시 Discord 알림을 원하는 게임 보안 연구자와 커뮤니티입니다.
