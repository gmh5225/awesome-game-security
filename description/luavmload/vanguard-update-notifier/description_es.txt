Vanguard Update Notifier es un bot de Discord en Python que vigila las versiones del anti-cheat Riot Vanguard y avisa a un canal configurado cuando cambian la versión o los hashes de los archivos incluidos. Consulta la API pública clientconfig de Riot para obtener la versión actual de Vanguard y la URL de instalación, descarga el archivo del instalador y calcula los hashes del contenido extraído con py7zr y SHA-256; si la extracción falla, usa como respaldo el hash en bruto de setup.exe. Las comprobaciones periódicas se ejecutan cada dos horas mediante discord.py y aiohttp, con comandos de barra para registrar o eliminar canales de notificación, y el estado se persiste en una configuración JSON local. El caso de uso principal son investigadores y comunidades de seguridad en juegos que quieren alertas oportunas en Discord sobre actualizaciones del anti-cheat Vanguard.
