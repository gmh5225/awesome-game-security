Vanguard Update Notifier est un bot Discord en Python qui surveille les versions de l'anti-cheat Riot Vanguard et alerte un canal configuré lorsque la version ou les hachages des fichiers inclus changent. Il interroge l'API publique clientconfig de Riot pour obtenir la version Vanguard actuelle et l'URL d'installation, télécharge l'archive de l'installateur, puis calcule les hachages du contenu extrait avec py7zr et SHA-256, en se rabattant sur le hachage brut de setup.exe si l'extraction échoue. Des vérifications périodiques s'exécutent toutes les deux heures via discord.py et aiohttp, avec des commandes slash pour enregistrer ou retirer des canaux de notification, et l'état est persisté dans une configuration JSON locale. Le cas d'usage principal concerne les chercheurs et communautés en sécurité des jeux qui souhaitent des alertes Discord rapides sur les mises à jour de l'anti-cheat Vanguard.
