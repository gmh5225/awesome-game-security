Vanguard Update Notifier is a Python Discord bot that watches Riot Vanguard anti-cheat releases and alerts a configured channel when the version or installer file hashes change. It polls Riot’s public clientconfig API for the current Vanguard version and setup URL, downloads the setup archive, and hashes extracted contents with py7zr (falling back to hashing setup.exe when extraction fails). Periodic checks run about every two hours, slash commands (/setup and /remove) manage per-server alert channels, and state such as last version and file hashes is stored in bot_config.json. Built with discord.py, aiohttp, and python-dotenv, it is aimed at game security researchers, reverse engineers, and anti-cheat watchers who want timely Discord notice of Vanguard updates.
