Vanguard Update Notifier — Discord-бот на Python, который отслеживает релизы античита Riot Vanguard и уведомляет настроенный канал при изменении версии или хэшей входящих файлов. Он опрашивает публичный API clientconfig Riot для получения текущей версии Vanguard и URL установщика, скачивает архив установщика и вычисляет хэши извлечённого содержимого с помощью py7zr и SHA-256; при сбое извлечения используется хэш исходного setup.exe. Периодические проверки выполняются каждые два часа через discord.py и aiohttp, есть slash-команды для регистрации или удаления каналов уведомлений, а состояние сохраняется в локальном JSON-конфиге. Основной сценарий использования — исследователи игровой безопасности и сообщества, которым нужны своевременные Discord-оповещения об обновлениях античита Vanguard.
