Vanguard Update Notifier è un bot Discord in Python che monitora i rilasci dell'anti-cheat Riot Vanguard e avvisa un canale configurato quando cambiano la versione o gli hash dei file inclusi. Interroga l'API pubblica clientconfig di Riot per ottenere la versione corrente di Vanguard e l'URL di setup, scarica l'archivio dell'installer e calcola gli hash del contenuto estratto con py7zr e SHA-256; se l'estrazione fallisce, usa come fallback l'hash grezzo di setup.exe. I controlli periodici vengono eseguiti ogni due ore tramite discord.py e aiohttp, con comandi slash per registrare o rimuovere i canali di notifica e stato persistito in una configurazione JSON locale. Il caso d'uso principale è per ricercatori e community di game security che vogliono avvisi Discord tempestivi sugli aggiornamenti dell'anti-cheat Vanguard.
