Vanguard Update Notifier は、Riot Vanguard アンチチートのリリースを監視し、バージョンまたは同梱ファイルのハッシュが変わったときに設定済みチャンネルへ通知する Python 製 Discord ボットです。Riot の公開 clientconfig API をポーリングして現在の Vanguard バージョンとセットアップ URL を取得し、インストーラーアーカイブをダウンロードしたうえで、py7zr と SHA-256 により展開内容のハッシュを計算します。展開に失敗した場合は生の setup.exe のハッシュにフォールバックします。discord.py と aiohttp により2時間ごとに定期チェックを行い、通知チャンネルの登録・削除用スラッシュコマンドを備え、状態はローカルの JSON 設定に永続化します。主な用途は、Vanguard アンチチートの更新を Discord でいち早く把握したいゲームセキュリティ研究者やコミュニティ向けです。
