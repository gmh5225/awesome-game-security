Este proyecto es un escáner de controladores del kernel que busca hooks sospechosos en los manejadores IRP_MJ_DEVICE_CONTROL. Está escrito en C++ como un controlador KMDF y recorre el directorio de objetos \Driver para enumerar los objetos de controladores cargados. Para cada controlador, comprueba si el puntero de dispatch de control de dispositivo permanece dentro de los límites de la imagen del controlador e intenta resolver el módulo para los punteros fuera de rango. El caso de uso principal es la auditoría de integridad del kernel y la investigación de seguridad de bajo nivel en torno a la detección de hooks en controladores.
