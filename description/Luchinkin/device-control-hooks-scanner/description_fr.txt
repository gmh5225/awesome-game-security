Ce projet est un scanner de pilotes noyau qui recherche des hooks suspects dans les gestionnaires IRP_MJ_DEVICE_CONTROL. Il est écrit en C++ sous forme de pilote KMDF et parcourt le répertoire d'objets \Driver pour énumérer les objets pilotes chargés. Pour chaque pilote, il vérifie si le pointeur de dispatch de contrôle de périphérique reste dans les limites de l'image du pilote et tente une résolution de module pour les pointeurs hors plage. Le cas d'usage principal est l'audit d'intégrité du noyau et la recherche en sécurité bas niveau autour de la détection de hooks de pilotes.
