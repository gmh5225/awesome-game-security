Этот проект — сканер драйверов ядра, который ищет подозрительные хуки в обработчиках IRP_MJ_DEVICE_CONTROL. Он написан на C++ как драйвер KMDF и обходит каталог объектов \Driver для перечисления загруженных объектов драйверов. Для каждого драйвера проверяется, остаётся ли указатель диспетчера device-control в пределах образа драйвера, а для указателей вне диапазона выполняется попытка разрешения модуля. Основное применение — аудит целостности ядра и низкоуровневые исследования безопасности, связанные с обнаружением хуков драйверов.
