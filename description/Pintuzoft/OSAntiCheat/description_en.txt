OSAntiCheat is an experimental server-side heuristic anticheat for Counter-Strike 2, implemented as a CounterStrikeSharp plugin in C# (.NET). It observes only server-visible data such as player positions, view angles, shots, and timing, then uses statistical detectors for spinbot, aimbot snaps, triggerbot, and several wallhack signals including tracking, gaze, and a McNemar null test. Independent detector outputs feed a fusion suspicion engine that grades confidence with decay and corroboration, producing Watch/Review tiers rather than automatic kicks or bans. The project is aimed at CS2 server operators and researchers who want log-only, probability-based cheat flagging during calibration against real demos and live server data.
