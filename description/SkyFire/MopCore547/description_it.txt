Questo progetto è un emulatore di server di World of Warcraft rivolto all'espansione Mists of Pandaria (5.4.7). Realizzato con C++ e CMake, implementa la simulazione del mondo di gioco, la gestione dei giocatori, lo scripting degli NPC e la gestione dei protocolli di rete per eseguire server WoW privati. È soprattutto utile per i ricercatori di emulazione di server di gioco e per i reverse engineer di MMORPG che studiano l'architettura dei server WoW e l'implementazione dei protocolli di gioco.
