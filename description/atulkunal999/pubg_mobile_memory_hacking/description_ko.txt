이 프로젝트는 Gameloop 에뮬레이터용 PUBG Mobile 치트로, C++로 ESP와 에임봇 기능을 구현합니다.
메모리 접근에는 커널 드라이버를 사용하고, ESP 표시에는 DirectX 기반 오버레이 렌더링을 쓰며, 게임 구조 접근을 위한 전체 Unreal Engine SDK 헤더를 포함합니다.
코드베이스는 DSEFix를 이용한 드라이버 로딩, x64 빌드 구성, 프로세스 열거를 통한 창 탐지를 다룹니다.
주로 PC 에뮬레이터에서 커널 수준 메모리 접근으로 구현된 모바일 게임 치트를 연구하는 게임 보안 연구자에게 유용합니다.
