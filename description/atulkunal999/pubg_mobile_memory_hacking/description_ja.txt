本プロジェクトは、Gameloop エミュレータ向けの PUBG Mobile チートで、C++ により ESP とエイムボット機能を実装しています。
メモリへのアクセスにはカーネルドライバを用い、ESP 表示には DirectX ベースのオーバーレイ描画を使い、ゲーム構造へのアクセス用に完全な Unreal Engine SDK ヘッダも含まれます。
コードベースは DSEFix によるドライバ読み込み、x64 ビルド設定、プロセス列挙によるウィンドウ検出を扱います。
PC エミュレータ上でカーネルレベルメモリアクセスを用いたモバイルゲームチート実装を研究するゲームセキュリティ研究者に主に役立ちます。
