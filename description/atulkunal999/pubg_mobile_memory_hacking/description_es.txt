Este proyecto es un cheat de PUBG Mobile para el emulador Gameloop, con funcionalidad ESP y aimbot implementada en C++.
Usa un controlador de kernel para el acceso a memoria, renderizado de overlay basado en DirectX para mostrar el ESP, e incluye los encabezados completos del SDK de Unreal Engine para acceder a las estructuras del juego.
El código cubre la carga del controlador con DSEFix, la configuración de compilación x64 y la detección de ventanas mediante enumeración de procesos.
Resulta especialmente útil para investigadores de seguridad en videojuegos que estudian implementaciones de cheats para juegos móviles en emuladores de PC con acceso a memoria a nivel de kernel.
