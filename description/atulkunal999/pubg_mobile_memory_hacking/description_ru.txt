Этот проект — чит для PUBG Mobile под эмулятор Gameloop с функциями ESP и аимбота, реализованными на C++.
Он использует драйвер ядра для доступа к памяти, оверлейный рендеринг на DirectX для отображения ESP и включает полные заголовки Unreal Engine SDK для доступа к структурам игры.
Кодовая база охватывает загрузку драйвера с DSEFix, конфигурацию сборки x64 и обнаружение окон через перечисление процессов.
Он в основном полезен исследователям игровой безопасности, изучающим реализации читов для мобильных игр на PC-эмуляторах с доступом к памяти на уровне ядра.
