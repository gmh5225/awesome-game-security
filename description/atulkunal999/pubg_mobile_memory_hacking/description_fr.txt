Ce projet est un cheat PUBG Mobile pour l'émulateur Gameloop, avec des fonctionnalités ESP et aimbot implémentées en C++.
Il utilise un pilote noyau pour l'accès mémoire, un rendu d'overlay basé sur DirectX pour l'affichage ESP, et inclut les en-têtes complets du SDK Unreal Engine pour accéder aux structures du jeu.
Le code couvre le chargement du pilote avec DSEFix, la configuration de build x64, et la détection de fenêtres par énumération de processus.
Il est surtout utile aux chercheurs en sécurité des jeux qui étudient les implémentations de cheats pour jeux mobiles sur émulateurs PC avec accès mémoire au niveau noyau.
