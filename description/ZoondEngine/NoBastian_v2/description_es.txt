Este proyecto es un kit de herramientas cliente/servidor en C++ para Windows destinado a la interacción con la memoria de procesos a través de tuberías con nombre (named pipes). El lado del servidor expone operaciones como lectura y escritura de memoria, cambio de protecciones, asignación de memoria remota y consulta de bases de módulos, mientras que el cliente emite solicitudes estructuradas. El código de utilidad inspecciona los handles del sistema y el estado de acceso a los procesos para obtener handles utilizables antes de realizar operaciones de memoria. Se usa principalmente para investigación de game hacking de bajo nivel y experimentos de evasión de anti-cheat basados en handles.
