Этот проект — клиент-серверный инструментарий на Windows C++ для взаимодействия с памятью процессов через именованные каналы (named pipes). Серверная сторона предоставляет операции чтения и записи памяти, изменения защиты, выделения удалённой памяти и запроса баз модулей, а клиент отправляет структурированные запросы. Вспомогательный код проверяет системные дескрипторы (handles) и состояние доступа к процессам, чтобы получить пригодные дескрипторы перед выполнением операций с памятью. Он в основном используется для исследований низкоуровневого game hacking и экспериментов по обходу античитов на основе дескрипторов.
