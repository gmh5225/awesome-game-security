Questo progetto è un toolkit client/server Windows in C++ per l'interazione con la memoria dei processi tramite named pipe. Il lato server espone operazioni come lettura e scrittura della memoria, modifica delle protezioni, allocazione di memoria remota e interrogazione delle basi dei moduli, mentre il client invia richieste strutturate. Il codice di utilità ispeziona gli handle di sistema e lo stato di accesso ai processi per acquisire handle utilizzabili prima di eseguire le operazioni sulla memoria. È usato principalmente per la ricerca di game hacking a basso livello e per esperimenti di evasione degli anti-cheat basati sugli handle.
