Ce projet est une boîte à outils client/serveur Windows en C++ pour l'interaction avec la mémoire des processus via des pipes nommés. Le côté serveur expose des opérations telles que la lecture et l'écriture mémoire, la modification des protections, l'allocation de mémoire distante et l'interrogation des bases de modules, tandis que le client émet des requêtes structurées. Le code utilitaire inspecte les handles système et l'état d'accès aux processus afin d'obtenir des handles utilisables avant d'effectuer les opérations mémoire. Il est principalement utilisé pour la recherche en game hacking bas niveau et les expériences d'évasion d'anti-cheat basées sur les handles.
