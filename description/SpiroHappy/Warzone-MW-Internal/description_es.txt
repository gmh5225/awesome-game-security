Este proyecto es un framework de cheat interno obsoleto para Call of Duty: Modern Warfare y Warzone.
Implementa módulos comunes como ESP, renderizado de huesos, controles de aimbot personalizables, reducción de retroceso y opciones varias como ajustes de FOV y UAV.
La implementación está basada en C++ y usa un menú ImGui para la configuración dentro del juego.
Sirve principalmente como referencia para estudiar la arquitectura de cheats internos y la integración de funciones en títulos FPS.
