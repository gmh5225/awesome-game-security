Этот проект представляет собой конфигурацию Hackintosh OpenCore EFI для Lenovo ThinkPad X260. Он включает полную настройку загрузчика OpenCore с ACPI-патчами, kexts и конфигурацией для запуска macOS на оборудовании ThinkPad X260, с поддержкой графики Intel HD 520, аудио Realtek, Intel WiFi, тачпада и управления питанием. Предназначен для энтузиастов Hackintosh и исследователей безопасности, которым нужна macOS на оборудовании не от Apple для тестирования.
