Este proyecto es una configuración EFI OpenCore de Hackintosh para el Lenovo ThinkPad X260. Incluye una configuración completa del cargador de arranque OpenCore con parches ACPI, kexts y configuración para ejecutar macOS en el hardware ThinkPad X260, con soporte para gráficos Intel HD 520, audio Realtek, WiFi Intel, trackpad y gestión de energía. Está dirigido a entusiastas de Hackintosh e investigadores de seguridad que necesitan macOS en hardware no Apple con fines de prueba.
