Dieses Projekt ist ein TrustDecision Device-Fingerprinting-SDK für Android, das eindeutige Gerätekennungen und Risikosignale zur Betrugserkennung erzeugt. Die auf Kotlin und Java basierende Bibliothek enthält instrumentierte Tests, Dokumentation zur Datenschutzrichtlinie sowie eine ProGuard-Konfiguration für den Produktionseinsatz. Sie ist vor allem für Anti-Cheat-Ingenieure und Mobile-Security-Forscher nützlich, die Android-Gerätefingerprinting und Verfahren zur Überprüfung der Geräteintegrität untersuchen.
