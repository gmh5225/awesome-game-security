Этот проект — SDK TrustDecision для снятия отпечатков устройств на Android, который формирует уникальные идентификаторы устройств и сигналы риска для выявления мошенничества. Библиотека на Kotlin и Java включает инструментальные тесты, документацию политики конфиденциальности и конфигурацию ProGuard для развёртывания в production. Она особенно полезна инженерам античита и исследователям мобильной безопасности, изучающим fingerprinting устройств Android и методы проверки целостности устройства.
