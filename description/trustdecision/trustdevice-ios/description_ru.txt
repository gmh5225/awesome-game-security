Этот проект — SDK отпечатка устройства TrustDecision для iOS, распространяемый как CocoaPod. Он собирает специфические для устройства атрибуты для обнаружения мошенничества и оценки целостности устройства, предоставляя уникальные идентификаторы и сигналы риска через нативный интерфейс Objective-C/Swift. Особенно полезен инженерам античита и исследователям мобильной безопасности, изучающим fingerprinting устройств iOS и методы проверки целостности устройства.
