VAC Bypass 是一個以 Windows DLL 形式建置的 C 語言函式庫，旨在停用 Steam 用戶端內的 Valve Anti-Cheat（VAC）掃描。它會注入 Steam.exe、修補 steamservice.dll，並掛鉤 LoadLibraryExW、GetProcAddress、GetSystemInfo 等 Win32 API，使 VAC 模組中止作弊掃描。此專案是面向 Win32 平台工具組的 Visual Studio 解決方案，並附帶用於匯入掛鉤與特徵碼搜尋的輔助工具。面向遊戲安全與反作弊研究人員，用於研究 VAC 如何載入、如何檢查環境，以及在 CS:GO 等 Steam 遊戲中如何被干預。
