VAC Bypass 是一个以 Windows DLL 形式构建的 C 语言库，旨在禁用 Steam 客户端内的 Valve Anti-Cheat（VAC）扫描。它注入到 Steam.exe，修补 steamservice.dll，并挂钩 LoadLibraryExW、GetProcAddress、GetSystemInfo 等 Win32 API，使 VAC 模块中止作弊扫描。该项目是面向 Win32 平台工具集的 Visual Studio 解决方案，并附带用于导入挂钩与特征码搜索的辅助工具。面向游戏安全与反作弊研究人员，用于研究 VAC 如何加载、如何检查环境，以及在 CS:GO 等 Steam 游戏中如何被干预。
