VAC Bypass は、Steam クライアント内の Valve Anti-Cheat（VAC）スキャンを無効化することを目的とした、Windows DLL としてビルドされる C ライブラリです。Steam.exe に注入し、steamservice.dll をパッチし、LoadLibraryExW、GetProcAddress、GetSystemInfo などの Win32 API をフックすることで、VAC モジュールがチートスキャンを中止するようにします。本プロジェクトは Win32 プラットフォーム ツールセット向けの Visual Studio ソリューションで、インポート フッキングとパターン検索用の補助ユーティリティを含みます。VAC の読み込み方、環境の検査方法、および CS:GO などの Steam タイトル上でどのように干渉され得るかを研究する、ゲームセキュリティおよびアンチチート研究者向けです。
