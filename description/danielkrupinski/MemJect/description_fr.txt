Un injecteur DLL Windows minimal qui embarque une DLL compilée sous forme de tableau d'octets bruts dans le code source, puis la mappe manuellement dans un processus cible (csgo.exe) entièrement en mode utilisateur via VirtualAllocEx, WriteProcessMemory et CreateRemoteThread.
Le mappeur analyse les en-têtes PE pour allouer une mémoire de taille adaptée dans la cible, copie les sections, résout les imports via LoadLibraryA/GetProcAddress, applique les relocations, et peut éventuellement effacer l'en-tête PE et le point d'entrée après l'appel à DllMain afin de réduire les artefacts forensiques.
Il est surtout utile aux chercheurs en sécurité des jeux qui étudient les techniques d'injection DLL en mémoire et le mapping PE manuel depuis le mode utilisateur.
