Questo progetto è GOESP, un cheat ESP per CS:GO multipiattaforma scritto in C++ moderno con overlay Dear ImGui. Legge i dati delle entità di gioco tramite accesso alla memoria e disegna le informazioni sui giocatori (box, nomi, salute, armi) come overlay usando la pipeline di rendering del gioco stesso. Il progetto supporta sia Windows che Linux, dimostrando tecniche di rendering di overlay esterno. È rivolto ai ricercatori di sicurezza nei giochi che studiano i pattern di implementazione ESP e il rilevamento di cheat basati su overlay.
