Этот проект — GOESP, кроссплатформенный ESP-чит для CS:GO на современном C++ с оверлеем Dear ImGui. Он читает данные сущностей игры через доступ к памяти и отображает информацию об игроках (рамки, имена, здоровье, оружие) в виде оверлея, используя собственный конвейер рендеринга игры. Проект поддерживает Windows и Linux и демонстрирует техники внешнего оверлейного рендеринга. Он предназначен для исследователей безопасности игр, изучающих паттерны реализации ESP и обнаружение читов на основе оверлея.
