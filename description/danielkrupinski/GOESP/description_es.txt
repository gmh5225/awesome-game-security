Este proyecto es GOESP, un cheat ESP de CS:GO multiplataforma escrito en C++ moderno con una superposición Dear ImGui. Lee los datos de entidades del juego mediante acceso a memoria y renderiza información de jugadores (cajas, nombres, salud, armas) como superposición usando el propio pipeline de renderizado del juego. El proyecto es compatible con Windows y Linux, y demuestra técnicas de renderizado de superposición externa. Está orientado a investigadores de seguridad en juegos que estudian patrones de implementación ESP y detección de cheats basados en superposición.
