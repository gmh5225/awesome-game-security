Этот проект ориентирован на 16.1.6); для сборки vac-hooks требуются platform toolset v142 и Windows SDK 10.0.
Если у вас их нет, VS можно скачать здесь (Windows SDK устанавливается во время установки Visual Studio).
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования античит-системы VAC.
