Este proyecto se centra en 16.1.6); se requieren platform toolset v142 y Windows SDK 10.0 para compilar vac-hooks.
Si no los tiene, puede descargar VS aquí (Windows SDK se instala durante la configuración de Visual Studio).
Es útil principalmente para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / exploración del sistema anticheat: VAC.
