Dieses Projekt konzentriert sich auf 16.1.6); zum Kompilieren von vac-hooks sind platform toolset v142 und Windows SDK 10.0 erforderlich.
Falls Sie diese nicht haben, können Sie VS hier herunterladen (Windows SDK wird während der Visual Studio-Installation mitinstalliert).
Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Erkundung von Anticheat-Systemen (VAC) untersuchen.
