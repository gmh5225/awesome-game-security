Bibliothèque C++ header-only permettant d'appeler des fonctions Windows x86 (32 bits) avec une adresse de retour falsifiée, en acheminant les appels via un gadget JMP DWORD PTR [EBX] trouvé dans la section de code d'un module cible.
Elle prend en charge les conventions d'appel stdcall, cdecl, fastcall et thiscall, de sorte que la fonction appelée voit l'adresse du gadget comme adresse de retour plutôt que le véritable appelant.
Elle est surtout utile aux chercheurs en sécurité des jeux qui étudient les techniques de spoofing d'adresse de retour utilisées par les cheats pour contourner l'analyse de la pile d'appels.
