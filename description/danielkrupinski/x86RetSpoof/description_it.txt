Libreria C++ header-only per invocare funzioni Windows x86 (32 bit) con un indirizzo di ritorno falsificato, instradando le chiamate attraverso un gadget JMP DWORD PTR [EBX] trovato nella sezione codice di un modulo di destinazione.
Supporta le convenzioni di chiamata stdcall, cdecl, fastcall e thiscall, facendo sì che la funzione invocata veda l'indirizzo del gadget come indirizzo di ritorno invece del vero chiamante.
È soprattutto utile ai ricercatori di sicurezza nei videogiochi che studiano le tecniche di spoofing dell'indirizzo di ritorno usate dai cheat per eludere l'analisi dello stack di chiamate.
