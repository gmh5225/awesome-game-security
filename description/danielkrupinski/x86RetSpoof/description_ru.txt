Библиотека C++ только из заголовочных файлов для вызова x86 (32-битных) функций Windows с подменённым адресом возврата путём маршрутизации вызовов через гаджет JMP DWORD PTR [EBX], найденный в секции кода целевого модуля.
Поддерживает соглашения о вызовах stdcall, cdecl, fastcall и thiscall, поэтому вызываемая функция видит адрес гаджета как адрес возврата вместо настоящего вызывающего кода.
В основном полезна исследователям игровой безопасности, изучающим техники подмены адреса возврата, которые читы используют для обхода анализа стека вызовов.
