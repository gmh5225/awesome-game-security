Этот проект посвящён VAC Bypass Loader — загрузчику для обхода VAC, написанному на C. Для компиляции VAC Bypass Loader требуются Microsoft Visual Studio 2019 (желательно последняя версия), набор инструментов платформы v142 и Windows SDK 10.0. Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов и античитов, в частности VAC.
