Dieses Projekt konzentriert sich auf den VAC Bypass Loader, einen in C geschriebenen Loader für den VAC-Bypass. Zum Kompilieren von VAC Bypass Loader werden Microsoft Visual Studio 2019 (vorzugsweise die neueste Version), das Plattform-Toolset v142 und das Windows SDK 10.0 benötigt. Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat-/Anticheat-Systeme (VAC) untersuchen.
