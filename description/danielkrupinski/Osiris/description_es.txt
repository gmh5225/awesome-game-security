Este proyecto es Osiris, un cheat de CS:GO de código abierto escrito en C++ moderno. Ofrece funciones como ESP (cajas de jugadores, salud, nombres), contornos glow, aimbot, triggerbot, backtrack, skin changer y manipulación del inventario. El código se engancha al motor Source mediante punteros de interfaz, pattern scanning y VMT hooking, mostrando la estructura interna de un cheat de juego completo. Está dirigido a investigadores de seguridad en juegos que estudian la arquitectura de cheats del motor Source y a desarrolladores de anti-cheat que construyen detección para estas técnicas.
