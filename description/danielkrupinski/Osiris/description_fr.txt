Ce projet est Osiris, un cheat CS:GO open source écrit en C++ moderne. Il propose des fonctionnalités telles que l'ESP (boîtes de joueurs, santé, noms), les contours glow, l'aimbot, le triggerbot, le backtrack, le skin changer et la manipulation d'inventaire. Le code s'accroche au moteur Source via des pointeurs d'interfaces, le pattern scanning et le VMT hooking, illustrant la structure interne d'un cheat de jeu complet. Il s'adresse aux chercheurs en sécurité des jeux qui étudient l'architecture des cheats pour le moteur Source, ainsi qu'aux développeurs d'anti-cheat qui mettent en place la détection de ces techniques.
