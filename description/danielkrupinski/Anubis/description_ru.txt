Этот проект — Anubis, чит для CS:GO под Linux на C++, который работает путём внедрения в процесс игры. Он предоставляет ESP, аимбот и другие чит-функции через взаимодействие с SDK движка Source, перехватывая клиентский рендеринг и обработку игровых событий. Нативная реализация под Linux демонстрирует техники game hacking, адаптированные для платформы Linux. Проект предназначен для исследователей безопасности игр, изучающих реализации читов под Linux и внутренности движка Source на платформах, отличных от Windows.
