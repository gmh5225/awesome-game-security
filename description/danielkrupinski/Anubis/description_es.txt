Este proyecto es Anubis, un cheat de CS:GO para Linux escrito en C++ que funciona inyectándose en el proceso del juego. Ofrece ESP, aimbot y otras funciones de cheat mediante la interacción con el SDK del motor Source, enganchándose al renderizado del cliente y al procesamiento de eventos del juego. La implementación nativa para Linux demuestra técnicas de game hacking adaptadas a la plataforma Linux. Está dirigido a investigadores de seguridad en juegos que estudian implementaciones de cheats en Linux y los internos del motor Source en plataformas distintas de Windows.
