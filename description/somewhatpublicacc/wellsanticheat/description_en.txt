Wells Anti Cheat is a host-side BepInEx moderation plugin for Among Us that protects lobbies with automatic detection and optional kick or ban. Written in C# for .NET 6 with Harmony patches, it inspects player RPCs for offensive names, chat spam, crash attempts, RPC floods, illegal vents, lobby teleports, task spoofing, and other common cheat patterns. Only the host can discard RPCs or punish players; the host is always exempt, and non-hosts still get notifications without enforcement. It also offers meeting grace timing, a modded-lobby mode to reduce false positives with role mods, and host tools such as map hotswap and force win. The primary use case is lobby hosts who want lightweight anti-cheat and moderation without affecting their own actions.
