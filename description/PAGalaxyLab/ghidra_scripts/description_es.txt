Este proyecto es una colección de scripts de Python para la suite de ingeniería inversa Ghidra. Incluye utilidades para rastrear parámetros de funciones con P-Code, analizar el despacho de mensajes de Objective-C, renombrar funciones a partir de patrones de impresión de errores y desofuscar el aplanamiento del flujo de control de OLLVM. Los scripts están organizados con módulos auxiliares compartidos y ejemplos de documentación para tareas prácticas de análisis de firmware y binarios. Está pensado para ingenieros inversos y analistas de seguridad de juegos que automatizan flujos de análisis estático repetitivos.
