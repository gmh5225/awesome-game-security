Dieses Projekt ist eine Sammlung von Python-Skripten für die Reverse-Engineering-Suite Ghidra. Es enthält Hilfsmittel zum Nachverfolgen von Funktionsparametern mit P-Code, zur Analyse von Objective-C-Message-Dispatch, zum Umbenennen von Funktionen anhand von Fehlerausgabe-Mustern sowie zur Deobfuskation der OLLVM-Control-Flow-Flattening. Die Skripte sind mit gemeinsamen Hilfsmodulen und Dokumentationsbeispielen für praktische Firmware- und Binäranalyse organisiert. Sie richten sich an Reverse Engineers und Game-Security-Analysten, die wiederkehrende Workflows der statischen Analyse automatisieren.
