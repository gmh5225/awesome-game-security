Questo progetto è una raccolta di script Python per la suite di reverse engineering Ghidra. Include utilità per tracciare i parametri delle funzioni con P-Code, analizzare il dispatch dei messaggi Objective-C, rinominare le funzioni a partire da pattern di stampa degli errori e deoffuscare l’appiattimento del flusso di controllo OLLVM. Gli script sono organizzati con moduli di supporto condivisi ed esempi di documentazione per attività pratiche di analisi di firmware e binari. È pensato per reverse engineer e analisti di sicurezza dei giochi che automatizzano flussi di lavoro di analisi statica ripetitivi.
