Этот проект — набор скриптов на Python для среды реверс-инжиниринга Ghidra. В него входят утилиты для трассировки параметров функций через P-Code, анализа диспетчеризации сообщений Objective-C, переименования функций по шаблонам вывода ошибок и деобфускации выравнивания потока управления OLLVM. Скрипты организованы с общими вспомогательными модулями и примерами документации для практических задач анализа прошивок и бинарников. Предназначен для реверс-инженеров и аналитиков игровой безопасности, автоматизирующих повторяющиеся рабочие процессы статического анализа.
