Questo progetto è un framework di method hooking per Android ART destinato alla sostituzione e all'intercettazione di metodi Java. Combina componenti Java e nativi per fornire API backup-and-hook, supporto alle ABI Android comuni ed esempi di hook su metodi statici, virtuali, JNI e di framework. Il repository include una libreria riutilizzabile oltre a un'app demo e moduli plugin che mostrano flussi reali di deployment degli hook. È usato principalmente per l'instrumentazione del runtime Android, la ricerca sulla sicurezza e la modifica dinamica del comportamento.
