Este proyecto es un framework de hooking de métodos ART de Android para el reemplazo e interceptación de métodos Java. Combina componentes Java y nativos para ofrecer APIs backup-and-hook, compatibilidad con ABI comunes de Android y ejemplos de hooks de métodos estáticos, virtuales, JNI y de framework. El repositorio incluye una biblioteca reutilizable, además de una aplicación de demostración y módulos plugin que muestran flujos reales de despliegue de hooks. Se usa principalmente para instrumentación del runtime de Android, investigación de seguridad y modificación dinámica del comportamiento.
