Этот проект — упрощённый фреймворк для разработки UEFI-приложений с использованием Visual Studio и инструментария EDK-II. Он предоставляет шаблоны проектов MSVC, property sheets и Python-скрипт настройки для быстрой инициализации проектов UEFI-приложений без полной сложности сборки EDK-II. В первую очередь полезен исследователям безопасности UEFI и разработчикам буткитов, которым нужна упрощённая среда разработки UEFI на базе Visual Studio.
