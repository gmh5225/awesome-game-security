Este proyecto es un framework simplificado para el desarrollo de aplicaciones UEFI con Visual Studio y la cadena de herramientas EDK-II. Proporciona plantillas de proyecto MSVC, hojas de propiedades y un script de configuración en Python para arrancar rápidamente proyectos de aplicaciones UEFI sin la complejidad completa de la compilación EDK-II. Es especialmente útil para investigadores de seguridad UEFI y desarrolladores de bootkits que necesitan un entorno de desarrollo UEFI ágil basado en Visual Studio.
