Questo progetto è un framework semplificato per lo sviluppo di applicazioni UEFI con Visual Studio e la toolchain EDK-II. Fornisce modelli di progetto MSVC, property sheet e uno script di setup in Python per avviare rapidamente progetti di applicazioni UEFI senza la piena complessità della build EDK-II. È principalmente utile a ricercatori di sicurezza UEFI e sviluppatori di bootkit che necessitano di un ambiente di sviluppo UEFI snello basato su Visual Studio.
