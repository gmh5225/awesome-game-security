Этот проект публикует собственную реализацию поведения IoCreateDriver для экспериментов в ядре Windows.
В нём рассматриваются техники, направленные на обход стандартных путей видимости загрузки драйверов, включая обход распространённых точек журналирования.
Код написан на C/C++ для рабочих процессов Visual Studio и WDK, с заметками по настройкам manual mapping и правкам точки входа.
Материал в первую очередь рассчитан на исследование низкоуровневых internals Windows и обхода античитов.
