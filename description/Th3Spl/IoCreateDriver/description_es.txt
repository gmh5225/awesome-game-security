Este proyecto comparte una implementación personalizada del comportamiento de IoCreateDriver para experimentación en el kernel de Windows.
Destaca técnicas pensadas para evitar las rutas habituales de visibilidad al cargar controladores, incluido eludir puntos comunes de registro.
El código está escrito en C/C++ para flujos de trabajo de Visual Studio y WDK, con notas sobre configuraciones de mapeo manual y ajustes del punto de entrada.
Está orientado principalmente a la investigación de internals de Windows a bajo nivel y a la evasión de anti-cheat.
