Ce projet est une boîte à outils Python pour lire, convertir et modifier les fichiers de sauvegarde de Palworld. Il analyse le format de sauvegarde GVAS personnalisé de Palworld, convertit entre les représentations binaire et JSON, et permet de modifier les données du joueur, les statistiques des Pal, le contenu de l'inventaire et l'état du monde. L'outil gère la compression, la sérialisation des propriétés personnalisées et les structures de sauvegarde UE5. Il s'adresse aux joueurs, moddeurs et administrateurs de serveurs Palworld qui gèrent la manipulation et la migration des fichiers de sauvegarde.
