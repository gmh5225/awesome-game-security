Этот проект — набор инструментов на Python для чтения, преобразования и редактирования файлов сохранений Palworld. Он разбирает собственный формат сохранений GVAS в Palworld, выполняет преобразование между двоичным и JSON-представлениями и позволяет изменять данные игрока, характеристики палов, содержимое инвентаря и состояние мира. Инструмент обрабатывает сжатие, сериализацию пользовательских свойств и структуры сохранений UE5. Предназначен для игроков Palworld, моддеров и администраторов серверов, которым нужно управлять правкой и миграцией файлов сохранений.
