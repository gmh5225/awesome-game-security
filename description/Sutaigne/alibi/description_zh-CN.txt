Alibi 是一款只读的 Windows 取证扫描套件，通过生成可分享的文本与 HTML 报告，帮助玩家向赛事管理员、Discord 管理员或大厅审核者证明自己没有作弊。它主要用 PowerShell 编写，配合批处理启动器与离线 HTML 配套页面，以调查人员的方式检查本机，无需安装软件、注册账号，也不会向本机外发送遥测数据。它支持两种模式：面向作弊软件、DMA 痕迹与 HWID 伪装工具的 PC 扫描，以及面向采集卡链路、视觉自瞄和 XIM、Cronus、ReaSnow 等输入适配器的主机外设扫描。扫描会给出清晰的判定等级，并生成自包含的可视化报告，覆盖 Call of Duty、CS2、Apex、Tarkov、Rust、Rainbow Six、Marvel Rivals 等游戏。主要用途是为被指控的玩家提供防御性的游戏安全证明，生成可审计的本地证据，而非传统反作弊产品。
