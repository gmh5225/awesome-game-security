Alibi 是一套唯讀的 Windows 鑑識掃描套件，透過產生可分享的文字與 HTML 報告，協助玩家向賽事管理員、Discord 管理員或大廳審核者證明自己沒有作弊。它主要以 PowerShell 撰寫，搭配批次啟動器與離線 HTML 配套頁面，以調查人員的方式檢查本機，無需安裝軟體、建立帳號，也不會將遙測資料傳出電腦。它支援兩種模式：針對作弊軟體、DMA 痕跡與 HWID 偽裝工具的 PC 掃描，以及針對擷取卡架構、視覺自瞄與 XIM、Cronus、ReaSnow 等輸入轉接器的主機外設掃描。掃描會輸出清楚的判定等級，並產生自包含的視覺化報告，涵蓋 Call of Duty、CS2、Apex、Tarkov、Rust、Rainbow Six、Marvel Rivals 等遊戲。主要用途是為遭指控的玩家提供防禦性的遊戲安全證明，產生可稽核的本地證據，而非傳統反作弊產品。
