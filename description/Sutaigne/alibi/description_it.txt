Alibi è un kit di scansione forense Windows in sola lettura che aiuta i giocatori a dimostrare di non barare producendo report di testo e HTML condividibili per admin di tornei, mod Discord o revisori di lobby. Realizzato principalmente in PowerShell con launcher batch e companion HTML offline, ispeziona la macchina come farebbe un investigatore, senza installare software, creare account o inviare telemetria fuori dal PC. Supporta due modalità: una scansione PC per software cheat, artefatti DMA e spoofer HWID, e una scansione console-rig per stack di capture card, aimbot vision e adattatori di input come XIM, Cronus o ReaSnow. Le scansioni emettono livelli di verdetto chiari e report visivi autocontenuti che coprono titoli come Call of Duty, CS2, Apex, Tarkov, Rust, Rainbow Six e Marvel Rivals. Il caso d’uso principale è l’attestazione difensiva di game security per giocatori accusati che necessitano di prove locali verificabili piuttosto che di un prodotto anti-cheat tradizionale.
