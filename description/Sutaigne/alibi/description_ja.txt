Alibi は読み取り専用の Windows フォレンジック・スキャンキットで、共有可能なテキストおよび HTML レポートを生成し、プレイヤーが大会運営者、Discord モデレーター、ロビー審査者に対してチートをしていないことを証明できるよう支援します。主に PowerShell で構築され、バッチ起動とオフライン HTML コンパニオンを備え、ソフトウェアのインストール、アカウント作成、PC 外へのテレメトリ送信なしに、調査官のようにマシンを検査します。モードは二つあり、チートソフト、DMA 痕跡、HWID スプーファー向けの PC スキャンと、キャプチャカード構成、ビジョンエイムボット、XIM、Cronus、ReaSnow などの入力アダプター向けのコンソール構成スキャンです。スキャンは明確な判定ティアと、Call of Duty、CS2、Apex、Tarkov、Rust、Rainbow Six、Marvel Rivals などのタイトルを扱う自己完結型のビジュアルレポートを出力します。主な用途は、従来のアンチチート製品ではなく、監査可能なローカル証拠を必要とする告発されたプレイヤー向けの防御的なゲームセキュリティ証明です。
