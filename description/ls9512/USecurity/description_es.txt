Este proyecto es un componente de seguridad usado en proyectos Unity.
Proporciona cifrado de tipos de datos de uso común en tiempo de ejecución, cifrado del almacenamiento de PlayerPrefs y una interfaz de llamada rápida para el cifrado y descifrado de algoritmos de cifrado habituales.
Es especialmente útil para ingenieros anti-trampas e investigadores de seguridad defensiva que trabajan en el área de anti-trampas / protección de motores de juego: Unity.
