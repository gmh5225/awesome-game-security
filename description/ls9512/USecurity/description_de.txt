Dieses Projekt ist eine Sicherheitskomponente für Unity-Projekte.
Es bietet die Verschlüsselung gängiger Datentypen zur Laufzeit, die Verschlüsselung der PlayerPrefs-Speicherung sowie eine Schnellaufruf-Schnittstelle für Ver- und Entschlüsselung gängiger Verschlüsselungsalgorithmen.
Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher im Bereich Anti-Cheat / Game-Engine-Schutz: Unity nützlich.
