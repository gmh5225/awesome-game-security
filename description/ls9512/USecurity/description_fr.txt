Ce projet est un composant de sécurité utilisé dans les projets Unity.
Il fournit le chiffrement des types de données courants à l'exécution, le chiffrement du stockage PlayerPrefs, ainsi qu'une interface d'appel rapide pour le chiffrement et le déchiffrement des algorithmes de chiffrement courants.
Il est principalement utile aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant dans le domaine de l'anti-triche / protection des moteurs de jeu : Unity.
