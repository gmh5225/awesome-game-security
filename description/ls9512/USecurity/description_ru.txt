Этот проект — компонент безопасности для использования в проектах Unity.
Он обеспечивает шифрование распространённых типов данных во время выполнения, шифрование хранилища PlayerPrefs, а также быстрый интерфейс вызова для шифрования и расшифровки распространённых алгоритмов шифрования.
В первую очередь полезен инженерам по античиту и исследователям защитной безопасности, работающим в области античита / защиты игровых движков: Unity.
