Este proyecto es una herramienta de Windows para crear y reenviar solicitudes DeviceIoControl contra controladores del kernel. Está escrita en C# y ofrece tanto una GUI WinForms como un modo CLI, permitiendo un control arbitrario de los búferes de entrada y salida similar a un flujo de trabajo tipo repeater. La aplicación ayuda a guardar, editar y volver a ejecutar cargas ioctl mientras se inspeccionan las respuestas durante las pruebas. Su caso de uso principal es la investigación de seguridad de controladores, la auditoría de interfaces ioctl y la preparación para fuzzing.
