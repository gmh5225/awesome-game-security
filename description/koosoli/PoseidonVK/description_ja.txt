PoseidonVK は、Bohemia Interactive が公開した Arma: Cold War Assault Remastered の GPL ソースリリースをコミュニティが継続するプロジェクトで、Poseidon（CWR）ゲームエンジンを中心に構築されています。主にモダンな C++20 と CMake で書かれ、専用の Vulkan グラフィックスバックエンドに加え、SDL3 経由の OpenGL および OpenAL Soft オーディオを含みます。PBO、P3D、PAA、SQF といったクラシックな Bohemia のアセット／スクリプト形式やネットワークメッセージ復号向けのフォーマットツールおよび libFuzzer ハーネスに加え、Blender 用 P3D インポーターや開発者向けスタジオアプリなどのユーティリティも同梱されています。一部のサポートサービスや統合ハーネスは Rust で書かれています。主な用途は、Operation Flashpoint / Cold War Assault エンジンのリマスターと研究であり、モデリング、移植性、リバースエンジニアリング、ゲーム形式のセキュリティ研究に役立ちます。
