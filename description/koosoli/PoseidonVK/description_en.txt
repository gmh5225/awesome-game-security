PoseidonVK is a community continuation of Bohemia Interactive's GPL source release for Arma: Cold War Assault Remastered, built around the Poseidon (CWR) game engine. The project is primarily written in modern C++20 with CMake, and includes a dedicated Vulkan graphics backend alongside OpenGL and OpenAL Soft audio via SDL3. It ships format tooling and libFuzzer harnesses for classic Bohemia asset and script formats such as PBO, P3D, PAA, SQF, and network message decoding, plus utilities like a Blender P3D importer and developer studio apps. Some supporting services and integration harnesses are written in Rust. The primary use case is remastering and studying the Operation Flashpoint / Cold War Assault engine for modding, portability, reverse engineering, and game-format security research.
