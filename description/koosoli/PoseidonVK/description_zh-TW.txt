PoseidonVK 是 Bohemia Interactive 為 Arma: Cold War Assault Remastered 釋出的 GPL 原始碼之社群延續專案，以 Poseidon（CWR）遊戲引擎為核心建構。本專案主要以現代 C++20 與 CMake 撰寫，並包含專用的 Vulkan 圖形後端，以及透過 SDL3 整合的 OpenGL 與 OpenAL Soft 音訊。它提供針對經典 Bohemia 資源與腳本格式（如 PBO、P3D、PAA、SQF）以及網路訊息解碼的格式工具與 libFuzzer 測試套件，另有 Blender P3D 匯入器與開發者工作室應用等公用程式。部分支援服務與整合測試框架以 Rust 撰寫。主要用途是對 Operation Flashpoint / Cold War Assault 引擎進行重製與研究，服務於模組開發、可移植性、逆向工程以及遊戲格式安全研究。
