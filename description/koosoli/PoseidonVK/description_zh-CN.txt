PoseidonVK 是 Bohemia Interactive 为 Arma: Cold War Assault Remastered 发布的 GPL 源代码的社区延续项目，基于 Poseidon（CWR）游戏引擎构建。该项目主要使用现代 C++20 与 CMake 编写，包含专用的 Vulkan 图形后端，以及通过 SDL3 集成的 OpenGL 与 OpenAL Soft 音频。它提供面向经典 Bohemia 资源与脚本格式（如 PBO、P3D、PAA、SQF）以及网络消息解码的格式工具与 libFuzzer 测试套件，另有 Blender P3D 导入器与开发者工作室应用等实用工具。部分配套服务与集成测试框架用 Rust 编写。主要用途是对 Operation Flashpoint / Cold War Assault 引擎进行重制与研究，服务于模组开发、可移植性、逆向工程以及游戏格式安全研究。
