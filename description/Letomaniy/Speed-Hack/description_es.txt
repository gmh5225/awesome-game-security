Speed Hack by Letomaniy es un proyecto DLL en C++ para Windows que manipula la velocidad del tiempo en el juego tras inyectarse en un proceso objetivo.
Sigue un enfoque al estilo de Cheat Engine y se construye con Visual Studio, incluyendo componentes de hooking basados en Detours.
El ejemplo expone controles de teclado para ralentizar, acelerar y restaurar la velocidad normal, con valores configurables en el código.
Se usa principalmente para practicar el desarrollo de cheats y para investigar cómo los juegos y los anti-cheat gestionan la manipulación del tiempo.
