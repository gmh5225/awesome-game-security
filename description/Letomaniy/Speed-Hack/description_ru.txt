Speed Hack by Letomaniy — это Windows C++ DLL-проект, который после внедрения в целевой процесс манипулирует скоростью игрового времени.
Он следует подходу в стиле Cheat Engine и собирается в Visual Studio, включая компоненты перехвата на базе Detours.
Пример предоставляет управление с клавиатуры для замедления, ускорения и возврата к нормальной скорости, с настраиваемыми в коде значениями.
В основном используется для практики разработки читов и исследования того, как игры и античиты обрабатывают манипуляцию временем.
