Questo progetto è una piattaforma di sicurezza e osservabilità runtime per Linux basata sulla raccolta di eventi eBPF. È implementata principalmente in Go e combina telemetria del kernel a basso livello con rilevamenti di più alto livello su pattern comportamentali sospetti. Il codice include percorsi di deployment per container e Kubernetes, un ampio contenuto di detection e strumenti per il monitoraggio su scala produttiva. È destinata ai team di sicurezza cloud e host impegnati in threat detection, incident response e forensics runtime.
