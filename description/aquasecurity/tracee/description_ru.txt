Этот проект — платформа безопасности и наблюдаемости runtime для Linux на основе сбора событий eBPF. Реализована преимущественно на Go и сочетает низкоуровневую телеметрию ядра с детекциями более высокого уровня для подозрительных поведенческих паттернов. Кодовая база включает пути развёртывания в контейнерах и Kubernetes, обширный detection-контент и инструменты для мониторинга в производственном масштабе. Предназначена для команд безопасности облака и хостов, занимающихся обнаружением угроз, реагированием на инциденты и runtime-форензикой.
