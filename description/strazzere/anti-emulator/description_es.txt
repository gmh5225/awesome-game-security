Este proyecto es una biblioteca de detección de emuladores Android que comprueba artefactos comunes de emulación, incluidos propiedades del sistema específicas de QEMU, identificadores de hardware genéricos, nombres de operador, huellas de build, disponibilidad de sensores y firmas del sistema de archivos. La biblioteca Java ofrece una API sencilla que devuelve los resultados de detección de cada heurística, ayudando a las aplicaciones a determinar si se ejecutan en un dispositivo físico o en un entorno emulado. Está dirigida a investigadores de seguridad Android y desarrolladores que implementan detección de emuladores con fines anticheat, antifraude o DRM.
