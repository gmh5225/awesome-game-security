Ce projet est une bibliothèque de détection d'émulateurs Android qui recherche des artefacts courants d'émulation, notamment les propriétés système spécifiques à QEMU, les identifiants matériels génériques, les noms d'opérateur, les empreintes de build, la disponibilité des capteurs et les signatures du système de fichiers. La bibliothèque Java fournit une API simple qui renvoie les résultats de détection pour chaque heuristique, aidant les applications à déterminer si elles s'exécutent sur un appareil physique ou dans un environnement émulé. Elle s'adresse aux chercheurs en sécurité Android et aux développeurs qui mettent en œuvre la détection d'émulateurs à des fins d'anti-triche, antifraude ou DRM.
