Questo progetto è una libreria di rilevamento emulatori Android che verifica artefatti comuni degli emulatori, tra cui proprietà di sistema specifiche di QEMU, identificatori hardware generici, nomi degli operatori, fingerprint di build, disponibilità dei sensori e firme del file system. La libreria Java fornisce una API semplice che restituisce i risultati di rilevamento per ogni euristica, aiutando le app a stabilire se sono in esecuzione su un dispositivo fisico o in un ambiente emulato. È rivolta a ricercatori di sicurezza Android e sviluppatori che implementano il rilevamento di emulatori per anti-cheat, antifrode o DRM.
