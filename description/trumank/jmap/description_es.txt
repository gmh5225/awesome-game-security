Este proyecto es jmap, una herramienta para mapear y visualizar dependencias y relaciones entre assets de Unreal Engine. Analiza archivos .uasset y .umap de UE4/UE5 para construir grafos de dependencias que muestran cómo los assets se referencian entre sí, ayudando a identificar cadenas de carga, dependencias circulares y la estructura de assets. Está dirigido a modders de Unreal Engine y analistas de juegos que mapean la estructura del contenido y las relaciones entre assets.
