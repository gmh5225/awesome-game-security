Questo progetto è PatternSleuth, una libreria Rust per lo scanning di pattern ad alte prestazioni (signature scanning) in file binari e nella memoria dei processi. Offre matching multi-pattern veloce con supporto per wildcard, implementazioni SIMD ottimizzate e capacità di scanning in batch per la ricerca in grandi regioni di memoria. La libreria supporta sia lo scanning basato su file sia lo scanning della memoria di processi in tempo reale. È rivolta a game hacker, reverse engineer e sviluppatori di anti-cheat che necessitano di un matching efficiente di pattern binari per la scoperta di offset o il rilevamento di signature.
