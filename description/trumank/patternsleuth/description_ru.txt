Этот проект — PatternSleuth, библиотека на Rust для высокопроизводительного поиска паттернов (сканирования сигнатур) в бинарных файлах и памяти процессов. Она обеспечивает быстрое сопоставление нескольких паттернов с поддержкой подстановочных символов, оптимизированные SIMD-реализации и пакетное сканирование для поиска в больших областях памяти. Библиотека поддерживает как сканирование файлов, так и сканирование памяти работающих процессов. Она предназначена для гейм-хакеров, специалистов по реверс-инжинирингу и разработчиков античита, которым нужно эффективное сопоставление бинарных паттернов для поиска смещений или обнаружения сигнатур.
