Ce projet fournit un guide.
Il est organisé comme documentation et matériel de référence pour le domaine des cheats / de l'exploration du noyau Windows, plutôt que comme une base de code d'exécution autonome.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans ce domaine.
