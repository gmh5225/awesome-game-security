Этот проект — TOG (The Online Gamer), игровой чит или инструмент, связанный с модификацией онлайн-игр. Он предоставляет функции взаимодействия с процессами онлайн-игр, потенциально включая манипуляцию памятью, изменение пакетов или функции автоматизации для конкретных онлайн-проектов. Предназначен для исследователей безопасности игр, изучающих инструменты читов для онлайн-игр и задачи античита в live-service играх.
