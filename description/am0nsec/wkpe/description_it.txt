Questo repository contiene esperimenti di programmazione del kernel Windows implementati come driver proof-of-concept e strumenti companion in user mode. L'attenzione è rivolta agli interni a basso livello come il comportamento del memory manager, inclusi esempi come l'enumerazione delle strutture VAD dei processi. Usa C/C++ con Visual Studio e progetti in stile WDK e documenta un forte accoppiamento di versione a build e simboli Windows specifici. Il progetto è destinato alla ricerca educativa sul kernel e a sperimentazioni di sicurezza controllate, non al deployment in produzione.
