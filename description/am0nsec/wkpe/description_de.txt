Dieses Repository enthält Experimente zur Windows-Kernel-Programmierung, umgesetzt als Proof-of-Concept-Treiber und begleitende User-Mode-Tools. Der Fokus liegt auf Low-Level-Internals wie dem Verhalten des Memory Managers, einschließlich Beispielen zum Auflisten von Prozess-VAD-Strukturen. Es verwendet C/C++ mit Visual Studio und WDK-artigen Projekten und dokumentiert eine starke Versionskopplung an bestimmte Windows-Builds und Symbole. Das Projekt ist für bildende Kernel-Forschung und kontrollierte Sicherheitsexperimente gedacht, nicht für den Produktionseinsatz.
