Этот репозиторий содержит эксперименты по программированию ядра Windows, реализованные в виде proof-of-concept драйверов и сопутствующих пользовательских утилит. Основное внимание уделяется низкоуровневым внутренним механизмам, таким как поведение диспетчера памяти, включая примеры перечисления структур VAD процессов. Используются C/C++ с Visual Studio и проектами в стиле WDK; документирована сильная привязка к конкретным сборкам Windows и символам. Проект предназначен для образовательного исследования ядра и контролируемых экспериментов в области безопасности, а не для промышленного развёртывания.
