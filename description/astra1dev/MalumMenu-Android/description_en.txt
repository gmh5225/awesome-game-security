MalumMenu-Android is an Among Us cheat and mod menu for Android that overlays a floating UI on the game. It is written mainly in TypeScript and uses Frida with frida-il2cpp-bridge to hook the Unity IL2CPP game at runtime, plus frida-java-menu for the on-screen interface. Features include movement cheats such as NoClip and speedhack, ESP options like no shadows, ship controls for meetings, sabotages, and vents, plus cosmetics unlocks and other utilities. The menu is packaged by embedding Frida gadget into the APK with objection so it can run without root on real Android devices. It is aimed at game modding, mobile reverse engineering, and experimenting with Among Us client-side behavior on Android.
