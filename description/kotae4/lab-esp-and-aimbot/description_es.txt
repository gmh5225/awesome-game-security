Este proyecto es un laboratorio educativo de game hacking que ofrece un entorno controlado para aprender la implementación de ESP y aimbot. Incluye una aplicación 3D sencilla de objetivos y código de cheat acompañante que demuestra lectura de memoria externa, transformación de coordenadas world-to-screen, renderizado ESP de jugadores/entidades y matemáticas básicas de aimbot (cálculo de ángulos y movimiento del ratón). Está dirigido a principiantes que aprenden los fundamentos del game hacking en un entorno de práctica seguro.
