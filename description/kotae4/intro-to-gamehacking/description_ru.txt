Этот проект — вводный учебный курс и лабораторная среда для изучения техник game hacking. В нём есть пошаговые руководства по сканированию памяти, цепочкам указателей, внедрению кода, перехвату функций и pattern scanning, а также практические бинарники. Материалы ведут от базового использования Cheat Engine к написанию собственных внешних и внутренних читов на C++. Проект рассчитан на начинающих исследователей безопасности игр и новичков, только начинающих путь в game hacking.
