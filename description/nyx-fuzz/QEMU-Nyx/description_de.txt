Dieses Projekt konzentriert sich auf Intel-PT. Dazu gehören das schnelle Zurücksetzen von Speicher und Geräten, die präzise Disassemblierung des laufenden Codes (auch wenn Code teilweise ausgelagert / nicht verfügbar ist) sowie Intel-PT-Dekodierung, die Instrumentierung von Code in der VM mit breakpoint-basierten Hooks sowie die Kommunikation mit einem Fuzzing-Frontend. Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / QEMU / KVM / PVE / VBox untersuchen.
