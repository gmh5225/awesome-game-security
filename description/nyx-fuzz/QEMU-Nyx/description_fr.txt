Ce projet se concentre sur Intel-PT. Il inclut la capacité de réinitialiser rapidement la mémoire et les périphériques, d'obtenir un désassemblage précis du code en cours d'exécution (même lorsque le code est partiellement swapé / indisponible) et le décodage Intel-PT, d'instrumenter le code s'exécutant dans la VM avec des hooks basés sur des points d'arrêt, ainsi que de communiquer avec un frontend de fuzzing. Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers étudiant les techniques offensives dans le domaine des cheats / QEMU / KVM / PVE / VBox.
