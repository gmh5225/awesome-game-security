Este proyecto se centra en Intel-PT. Incluye la capacidad de restablecer rápidamente la memoria y los dispositivos, obtener un desensamblado preciso del código en ejecución (incluso cuando el código está parcialmente intercambiado / no disponible) y la decodificación de Intel-PT, instrumentar el código que se ejecuta en la VM con hooks basados en puntos de interrupción, así como comunicarse con un frontend de fuzzing. Es especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / QEMU / KVM / PVE / VBox.
