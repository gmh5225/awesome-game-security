Ce projet est un framework de wrapper et de proxy DLL Direct3D9 avec un système d'overlay intégré en jeu. Écrit en C++, il fournit une console à l'écran, le rendu d'un flux de texte et une interface de plugins pour charger des modules de fonctionnalités externes. Le code s'appuie sur les composants DirectX 9 et une interception de type Detours pour s'intégrer aux applications cibles. Il est principalement destiné au modding de jeux solo, à l'instrumentation d'UI et au développement d'overlays graphiques.
