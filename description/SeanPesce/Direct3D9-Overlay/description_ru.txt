Этот проект представляет собой фреймворк-обёртку и proxy DLL для Direct3D9 со встроенной системой внутриигрового оверлея. Он написан на C++ и предоставляет экранную консоль, отрисовку текстовой ленты и интерфейс плагинов для загрузки внешних модулей функций. Кодовая база опирается на компоненты DirectX 9 и перехват в стиле Detours для интеграции в целевые приложения. В основном предназначен для моддинга одиночных игр, инструментации UI и разработки графических оверлеев.
