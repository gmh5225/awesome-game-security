Este proyecto es un framework de wrapper y proxy DLL de Direct3D9 con un sistema de overlay integrado en el juego. Está escrito en C++ y proporciona una consola en pantalla, renderizado de un feed de texto y una interfaz de plugins para cargar módulos de funcionalidad externos. El código se basa en componentes de DirectX 9 y en una intercepción al estilo de Detours para integrarse en las aplicaciones objetivo. Está pensado principalmente para el modding de juegos para un jugador, la instrumentación de UI y el desarrollo de overlays gráficos.
