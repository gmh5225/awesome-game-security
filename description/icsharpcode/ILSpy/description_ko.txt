이 프로젝트는 ILSpy로, 오픈소스 .NET 어셈블리 브라우저이자 디컴파일러입니다. .NET 어셈블리를 C#, VB.NET 또는 IL 코드로 다시 디컴파일하며 .NET Framework, .NET Core, .NET 5+를 포함한 주요 .NET 버전을 모두 지원합니다. 대화형 트리 기반 어셈블리 브라우저, 구문 강조 소스 코드 보기, 검색 기능, 플러그인 확장성을 제공합니다. async/await 패턴, LINQ 식, 제네릭 타입 등 복잡한 디컴파일 시나리오도 처리합니다. 관리 코드 바이너리를 검토하는 .NET 개발자, 리버스 엔지니어, 보안 연구자를 대상으로 합니다.
