Questo progetto è ILSpy, un browser e decompilatore open source di assembly .NET. Decompila gli assembly .NET in codice C#, VB.NET o IL, supportando tutte le principali versioni di .NET, tra cui .NET Framework, .NET Core e .NET 5+. Lo strumento offre un browser interattivo delle assembly basato su albero, una vista del codice sorgente con evidenziazione della sintassi, funzionalità di ricerca ed estensibilità tramite plugin. Gestisce scenari di decompilazione complessi, inclusi i pattern async/await, le espressioni LINQ e i tipi generici. È rivolto a sviluppatori .NET, reverse engineer e ricercatori di sicurezza che esaminano binari di codice gestito.
