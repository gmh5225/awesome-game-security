Ce projet est ILSpy, un navigateur et décompilateur d'assemblies .NET open source. Il décompile les assemblies .NET en code C#, VB.NET ou IL, et prend en charge toutes les versions majeures de .NET, notamment .NET Framework, .NET Core et .NET 5+. L'outil offre un navigateur d'assemblies interactif en arborescence, une vue du code source avec coloration syntaxique, une fonction de recherche et une extensibilité par plugins. Il gère des scénarios de décompilation complexes, notamment les motifs async/await, les expressions LINQ et les types génériques. Il s'adresse aux développeurs .NET, aux ingénieurs en reverse engineering et aux chercheurs en sécurité qui examinent des binaires de code managé.
