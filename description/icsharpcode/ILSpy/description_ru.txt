Этот проект — ILSpy, открытый браузер и декомпилятор сборок .NET. Он декомпилирует сборки .NET обратно в код C#, VB.NET или IL и поддерживает все основные версии .NET, включая .NET Framework, .NET Core и .NET 5+. Инструмент предоставляет интерактивный древовидный браузер сборок, просмотр исходного кода с подсветкой синтаксиса, поиск и расширяемость через плагины. Он обрабатывает сложные сценарии декомпиляции, включая шаблоны async/await, выражения LINQ и обобщённые типы. Предназначен для разработчиков .NET, специалистов по реверс-инжинирингу и исследователей безопасности, изучающих двоичные файлы управляемого кода.
