本プロジェクトは ILSpy で、オープンソースの .NET アセンブリブラウザ兼デコンパイラです。.NET アセンブリを C#、VB.NET、または IL コードに戻してデコンパイルでき、.NET Framework、.NET Core、.NET 5+ を含む主要な .NET バージョンをすべてサポートします。対話型のツリーベースアセンブリブラウザ、シンタックスハイライト付きソース表示、検索機能、プラグインによる拡張性を備えています。async/await パターン、LINQ 式、ジェネリック型など、複雑なデコンパイルにも対応します。.NET 開発者、リバースエンジニア、マネージドコードのバイナリを調査するセキュリティ研究者を対象としています。
