Este proyecto es ILSpy, un explorador y descompilador de ensamblados .NET de código abierto. Descompila ensamblados .NET de nuevo a código C#, VB.NET o IL, y admite todas las versiones principales de .NET, incluidas .NET Framework, .NET Core y .NET 5+. La herramienta ofrece un explorador de ensamblados interactivo basado en árbol, una vista de código fuente con resaltado de sintaxis, funciones de búsqueda y extensibilidad mediante plugins. Gestiona escenarios de descompilación complejos, incluidos patrones async/await, expresiones LINQ y tipos genéricos. Está dirigido a desarrolladores .NET, ingenieros de ingeniería inversa e investigadores de seguridad que examinan binarios de código administrado.
