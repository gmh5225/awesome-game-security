DecBench 是一套 Python 基準測試套件，用於評估反編譯器從已編譯 C 二進位檔還原原始碼的能力。它採用三階段管線——在多種最佳化等級下編譯專案、以可插拔後端反編譯，再對輸出評分——支援 angr、Ghidra、IDA、Binary Ninja、r2dec、dewolf 等傳統工具，以及可選的 LLM 或程式設計代理後端。評估使用三項核心指標：衡量結構正確性的 CFG 圖編輯距離、對照 DWARF 真值的型別匹配，以及衡量組譯相似度的重編譯位元組匹配。基準語料包括 Debian 風格套件、交叉編譯的嵌入式與 RTOS 韌體，以及經仔細沙箱隔離的惡意軟體樣本。面向逆向工程師、二進位分析研究者，以及所有比較或改進反編譯器品質的人。
