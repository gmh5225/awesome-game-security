DecBench 是一套 Python 基准测试套件，用于评估反编译器从已编译 C 二进制中恢复源代码的能力。它采用三阶段流水线——在多种优化级别下编译项目、用可插拔后端反编译，再对输出打分——支持 angr、Ghidra、IDA、Binary Ninja、r2dec、dewolf 等传统工具，以及可选的 LLM 或编程代理后端。评估使用三项核心指标：衡量结构正确性的 CFG 图编辑距离、对照 DWARF 真值的类型匹配，以及衡量汇编相似度的重编译字节匹配。基准语料包括 Debian 风格软件包、交叉编译的嵌入式与 RTOS 固件，以及经仔细沙箱隔离的恶意软件样本。面向逆向工程师、二进制分析研究者，以及所有比较或改进反编译器质量的人。
