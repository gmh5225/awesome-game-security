DecBench는 컴파일된 C 바이너리에서 소스를 얼마나 잘 복원하는지 디컴파일러를 평가하기 위한 Python 벤치마킹 스위트입니다. 여러 최적화 수준에서 프로젝트를 컴파일하고, 플러그형 백엔드로 디컴파일한 뒤 출력을 채점하는 3단계 파이프라인을 실행하며, angr, Ghidra, IDA, Binary Ninja, r2dec, dewolf 같은 전통적 도구와 선택적 LLM 또는 코딩 에이전트 백엔드를 지원합니다. 평가는 구조적 정확성을 위한 CFG 그래프 편집 거리, DWARF 정답에 대한 타입 매칭, 어셈블리 유사도를 위한 재컴파일 바이트 일치라는 세 가지 핵심 지표를 사용합니다. 벤치마크 코퍼스에는 Debian 스타일 패키지, 크로스 컴파일된 임베디드 및 RTOS 펌웨어, 신중히 샌드박스된 악성코드 샘플이 포함됩니다. 리버스 엔지니어, 바이너리 분석 연구자, 디컴파일러 품질을 비교하거나 개선하려는 모든 이를 대상으로 합니다.
