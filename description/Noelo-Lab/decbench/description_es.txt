DecBench es una suite de benchmarking en Python para evaluar qué tan bien los descompiladores recuperan el código fuente a partir de binarios C compilados. Ejecuta un pipeline de tres etapas —compilar proyectos en varios niveles de optimización, descompilar con backends enchufables y puntuar la salida— con soporte para herramientas tradicionales como angr, Ghidra, IDA, Binary Ninja, r2dec y dewolf, además de backends opcionales de LLM o agentes de código. La evaluación usa tres métricas principales: distancia de edición de grafos CFG para la corrección estructural, coincidencia de tipos frente a la verdad de referencia DWARF y coincidencia de bytes tras recompilación para la similitud del ensamblador. Los corpus de benchmark incluyen paquetes al estilo Debian, firmware embebido y RTOS cross-compilado, y muestras de malware cuidadosamente aisladas en sandbox. Está dirigido a ingenieros de ingeniería inversa, investigadores de análisis binario y a cualquiera que compare o mejore la calidad de los descompiladores.
