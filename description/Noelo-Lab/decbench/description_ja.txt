DecBench は、コンパイル済み C バイナリからソースを復元するデコンパイラの性能を評価するための Python ベンチマークスイートです。複数の最適化レベルでプロジェクトをコンパイルし、プラグイン可能なバックエンドでデコンパイルし、出力を採点する三段階パイプラインを実行し、angr、Ghidra、IDA、Binary Ninja、r2dec、dewolf などの従来ツールに加え、任意で LLM やコーディングエージェントのバックエンドにも対応します。評価には三つの主要指標を用います。構造的正しさのための CFG グラフ編集距離、DWARF の正解に対する型照合、アセンブリ類似度のための再コンパイル時バイト一致です。ベンチマークコーパスには Debian 形式のパッケージ、クロスコンパイルされた組み込み／RTOS ファームウェア、慎重にサンドボックス化されたマルウェア試料が含まれます。リバースエンジニア、バイナリ解析研究者、デコンパイラ品質を比較・改善するすべての人を対象としています。
