Этот проект посвящён exrop. Exrop — инструмент автоматической генерации ROP-цепочек, который может автоматически строить цепочку гаджетов по заданному бинарному файлу и ограничениям.
Он в основном написан на Python.
Особенно полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / генерации ROP.
