Dieses Projekt behandelt exrop. Exrop ist ein Tool zur automatischen Generierung von ROP-Chains, das aus einer gegebenen Binary und Constraints automatisch eine Gadget-Chain aufbauen kann.
Es ist hauptsächlich in Python geschrieben.
Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat- / ROP-Generierung untersuchen.
