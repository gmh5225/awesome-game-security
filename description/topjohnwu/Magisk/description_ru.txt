Этот проект — Magisk, наиболее широко используемое решение для root на Android и фреймворк системных изменений без изменения раздела system (systemless). Он обеспечивает управление root-доступом через демон su, модульные изменения системы без изменения системного раздела, MagiskHide для обхода обнаружения root, а также полноценное Android-приложение для управления модулями и правами суперпользователя. В первую очередь полезен исследователям безопасности Android, инженерам античита и мобильным разработчикам, изучающим root-фреймворки, systemless-модификации и техники обхода обнаружения root.
