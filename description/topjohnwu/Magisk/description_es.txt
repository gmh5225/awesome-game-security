Este proyecto es Magisk, la solución de root para Android más utilizada y un framework de modificaciones systemless. Proporciona gestión de acceso root mediante un demonio su, modificaciones del sistema basadas en módulos sin alterar la partición del sistema, MagiskHide para evadir la detección de root y una aplicación Android completa para gestionar módulos y permisos de superusuario. Es especialmente útil para investigadores de seguridad Android, ingenieros anti-cheat y desarrolladores móviles que estudian frameworks de root, modificaciones systemless y técnicas de evasión de la detección de root.
