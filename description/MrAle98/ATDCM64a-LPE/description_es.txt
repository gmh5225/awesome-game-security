Este proyecto es una prueba de concepto de escalada de privilegios local dirigida a un controlador de Windows vulnerable. Está implementado en C++ con archivos de proyecto de Visual Studio y acompañado de scripts de PowerShell y por lotes para apoyar los pasos de despliegue y ejecución. La plantilla de exploit demuestra la interacción con el controlador y la preparación del flujo de trabajo en un entorno orientado al laboratorio. Su caso de uso principal es la investigación de explotación del kernel de Windows, incluidos estudios relevantes para el anti-cheat y el endurecimiento de la seguridad de controladores.
