Questo progetto è una proof of concept di elevazione dei privilegi locali che prende di mira un driver Windows vulnerabile. È implementato in C++ con file di progetto Visual Studio e accompagnato da script PowerShell e batch a supporto delle fasi di distribuzione ed esecuzione. Il modello di exploit illustra l'interazione con il driver e la preparazione del flusso di lavoro in una configurazione orientata al laboratorio. Il caso d'uso principale è la ricerca sull'exploitation del kernel Windows, inclusi studi rilevanti per anti-cheat e hardening della sicurezza dei driver.
