Ce projet est un utilitaire permettant d'extraire et de dumpter les fichiers de pilote et de modules EasyAntiCheat (EAC) depuis les installations de jeux. Il localise les composants EAC dans le répertoire du jeu et dans le magasin de pilotes Windows, puis extrait le pilote anti-triche, les modules en mode utilisateur et les fichiers de configuration pour une analyse de reverse engineering hors ligne. Il s'adresse aux chercheurs en anti-triche qui ont besoin d'obtenir les binaires EAC pour l'analyse statique et le reverse engineering.
