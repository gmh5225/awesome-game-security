Этот проект — утилита для извлечения и дампа файлов драйвера и модулей EasyAntiCheat (EAC) из установок игр. Она находит компоненты EAC в каталоге игры и в хранилище драйверов Windows, извлекает античит-драйвер, модули пользовательского режима и конфигурационные файлы для офлайн-анализа методом реверс-инжиниринга. Предназначена для исследователей античитов, которым нужны бинарные файлы EAC для статического анализа и реверс-инжиниринга.
