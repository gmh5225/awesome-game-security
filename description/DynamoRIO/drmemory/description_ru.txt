Dr. Memory — динамический отладчик памяти, выявляющий типичные ошибки памяти во время выполнения в нативных программах. В основном написан на C и C++ поверх платформы инструментирования DynamoRIO; находит неинициализированные чтения, выходы за границы, use-after-free, двойные free и утечки, а также выполняет проверки, специфичные для Windows, например неправильное использование handle и GDI. Поддерживает неизменённые бинарники на Windows, Linux, macOS и Android для целей IA-32, AMD64 и ARM. Проект широко применяют для упрочнения ПО и поиска уязвимостей, в том числе для проверки надёжности кода игр и античитов.
