Dr. Memory est un débogueur mémoire dynamique qui détecte les erreurs mémoire runtime courantes dans les programmes natifs. Principalement écrit en C et C++ au-dessus de la plateforme d'instrumentation DynamoRIO, il peut trouver les lectures non initialisées, les accès hors limites, les use-after-free, les doubles free et les fuites, avec des contrôles spécifiques à Windows comme les mauvais usages de handles et de GDI. Il prend en charge les binaires non modifiés sous Windows, Linux, macOS et Android, pour les cibles IA-32, AMD64 et ARM. Le projet est largement utilisé pour le durcissement logiciel et la chasse aux bugs de sécurité, y compris les tests de fiabilité du code de jeux et d'anti-cheat.
