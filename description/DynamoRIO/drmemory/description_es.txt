Dr. Memory es un depurador dinámico de memoria que detecta errores de memoria en tiempo de ejecución habituales en programas nativos. Escrito en gran medida en C y C++ sobre la plataforma de instrumentación DynamoRIO, puede encontrar lecturas no inicializadas, accesos fuera de límites, use-after-free, dobles liberaciones y fugas, con comprobaciones específicas de Windows como el uso indebido de handles y GDI. Admite binarios sin modificar en Windows, Linux, macOS y Android, para objetivos IA-32, AMD64 y ARM. El proyecto se usa ampliamente para el endurecimiento de software y la búsqueda de bugs de seguridad, incluida la prueba de fiabilidad de código de juegos y anti-cheat.
