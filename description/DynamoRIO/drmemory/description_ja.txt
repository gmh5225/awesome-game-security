Dr. Memory は、ネイティブプログラムでよくある実行時メモリエラーを検出する動的メモリデバッガです。主に C と C++ で書かれ、DynamoRIO 計測プラットフォーム上に構築されており、未初期化読み取り、範囲外アクセス、use-after-free、二重解放、リークを発見でき、Windows 固有のハンドルや GDI の誤用チェックも備えます。Windows、Linux、macOS、Android 上の未改変バイナリを、IA-32、AMD64、ARM ターゲットでサポートします。ソフトウェアの強化やセキュリティバグ探索に広く使われ、ゲームやアンチチートコードの信頼性テストにも利用されています。
