Dr. Memory는 네이티브 프로그램에서 흔히 발생하는 런타임 메모리 오류를 탐지하는 동적 메모리 디버거입니다. 주로 C와 C++로 작성되어 DynamoRIO 계측 플랫폼 위에 구축되었으며, 초기화되지 않은 읽기, 범위 밖 접근, use-after-free, 이중 해제, 누수를 찾아내고, 핸들 및 GDI 오용 같은 Windows 전용 검사도 제공합니다. Windows, Linux, macOS, Android에서 수정되지 않은 바이너리를 지원하며, 대상은 IA-32, AMD64, ARM입니다. 이 프로젝트는 소프트웨어 강화와 보안 버그 헌팅에 널리 쓰이며, 게임 및 안티치트 코드의 신뢰성 테스트에도 활용됩니다.
