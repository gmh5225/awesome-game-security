Volatility ist das ursprüngliche Python-2-Framework für Memory Forensics zur Analyse von RAM-Dumps von Windows-, Linux- und macOS-Systemen und bietet Plugins für Prozesslisten (pslist/psscan), DLL-Enumeration, Extraktion von Registry-Hives, Wiederherstellung von Netzwerkverbindungen, Erkennung von Kernelmodulen, Rootkit-Identifikation sowie Analyse von Malware-Artefakten.
Das Framework unterstützt mehrere Address-Space-Backends (Raw Dumps, Crash Dumps, Hibernate-Dateien, VMware-Snapshots, FireWire/IEEE 1394), nutzt profilbasierte Typdefinitionen für die OS-versionsspezifische Strukturanalyse und enthält Werkzeuge zum Erstellen von Linux-/Mac-Kernelprofilen aus Debug-Symbolen.
Es ist vor allem für DFIR-Analysten, Malware-Forscher und Game-Security-Forscher nützlich, die post-mortem Memory-Analysen durchführen, um injizierten Code, versteckte Prozesse und Rootkit-Artefakte zu erkennen.
