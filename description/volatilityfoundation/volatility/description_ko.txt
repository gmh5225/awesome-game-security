Volatility는 Windows, Linux, macOS 시스템의 RAM 덤프를 분석하기 위한 최초의 Python 2 메모리 포렌식 프레임워크로, 프로세스 목록(pslist/psscan), DLL 열거, 레지스트리 하이브 추출, 네트워크 연결 복원, 커널 모듈 탐지, 루트킷 식별, 악성코드 아티팩트 분석 등의 플러그인을 제공합니다.
이 프레임워크는 여러 주소 공간 백엔드(raw 덤프, 크래시 덤프, 최대 절전 모드 파일, VMware 스냅샷, FireWire/IEEE 1394)를 지원하며, OS 버전별 구조체 파싱을 위한 프로파일 기반 타입 정의를 사용하고, 디버그 심볼로부터 Linux/Mac 커널 프로파일을 구축하는 도구를 포함합니다.
주로 DFIR 분석가, 악성코드 연구원, 게임 보안 연구원이 주입된 코드, 숨겨진 프로세스, 루트킷 흔적을 탐지하기 위한 사후 메모리 분석에 활용합니다.
