Volatility è il framework originale di memory forensics in Python 2 per analizzare dump della RAM di sistemi Windows, Linux e macOS, con plugin per l'elenco dei processi (pslist/psscan), l'enumerazione delle DLL, l'estrazione degli hive del Registro, il recupero delle connessioni di rete, il rilevamento dei moduli del kernel, l'identificazione di rootkit e l'analisi di artefatti malware.
Il framework supporta più backend di spazio di indirizzi (dump raw, crash dump, file di ibernazione, snapshot VMware, FireWire/IEEE 1394), usa definizioni di tipi basate su profili per l'analisi delle strutture specifiche della versione del SO e include strumenti per creare profili kernel Linux/Mac dai simboli di debug.
È principalmente utile ad analisti DFIR, ricercatori di malware e ricercatori di game security che eseguono analisi della memoria post-mortem per rilevare codice iniettato, processi nascosti e artefatti di rootkit.
