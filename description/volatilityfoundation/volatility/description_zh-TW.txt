Volatility 是最初的 Python 2 記憶體鑑識框架，用於分析來自 Windows、Linux 與 macOS 系統的 RAM 傾印，並提供程序清單（pslist/psscan）、DLL 列舉、登錄檔 Hive 擷取、網路連線還原、核心模組偵測、Rootkit 識別以及惡意軟體痕跡分析等外掛。
該框架支援多種位址空間後端（原始傾印、當機傾印、休眠檔案、VMware 快照、FireWire/IEEE 1394），採用基於設定檔（profile）的型別定義以依作業系統版本解析結構，並包含從除錯符號建置 Linux/Mac 核心設定檔的工具。
它主要適用於 DFIR 分析師、惡意軟體研究人員以及遊戲安全研究人員，用於進行事後記憶體分析，以偵測注入程式碼、隱藏程序與 Rootkit 痕跡。
