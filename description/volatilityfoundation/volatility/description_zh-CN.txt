Volatility 是最初的 Python 2 内存取证框架，用于分析来自 Windows、Linux 和 macOS 系统的 RAM 转储，并提供进程列表（pslist/psscan）、DLL 枚举、注册表配置单元提取、网络连接恢复、内核模块检测、Rootkit 识别以及恶意软件痕迹分析等插件。
该框架支持多种地址空间后端（原始转储、崩溃转储、休眠文件、VMware 快照、FireWire/IEEE 1394），采用基于配置文件（profile）的类型定义以按操作系统版本解析结构，并包含从调试符号构建 Linux/Mac 内核配置文件的工具。
它主要适用于 DFIR 分析师、恶意软件研究人员以及游戏安全研究人员，用于进行事后内存分析，以检测注入代码、隐藏进程和 Rootkit 痕迹。
