Volatility は、Windows、Linux、macOS システムの RAM ダンプを解析するためのオリジナルの Python 2 メモリフォレンジックフレームワークであり、プロセス一覧（pslist/psscan）、DLL 列挙、レジストリハイブ抽出、ネットワーク接続の復元、カーネルモジュール検出、ルートキット識別、マルウェアアーティファクト分析などのプラグインを提供します。
本フレームワークは複数のアドレス空間バックエンド（raw ダンプ、クラッシュダンプ、ハイバネーションファイル、VMware スナップショット、FireWire/IEEE 1394）をサポートし、OS バージョン固有の構造体解析のためのプロファイルベースの型定義を用い、デバッグシンボルから Linux/Mac カーネルプロファイルを構築するツールも含まれます。
主に DFIR アナリスト、マルウェア研究者、ゲームセキュリティ研究者が、注入コード、隠蔽プロセス、ルートキットの痕跡を検出するための事後メモリ分析に利用します。
