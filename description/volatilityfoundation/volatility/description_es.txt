Volatility es el framework original de forense de memoria en Python 2 para analizar volcados de RAM de sistemas Windows, Linux y macOS, con plugins para listado de procesos (pslist/psscan), enumeración de DLL, extracción de hives del Registro, recuperación de conexiones de red, detección de módulos del kernel, identificación de rootkits y análisis de artefactos de malware.
El framework admite múltiples backends de espacio de direcciones (volcados raw, volcados de crash, archivos de hibernación, instantáneas de VMware, FireWire/IEEE 1394), usa definiciones de tipos basadas en perfiles para el análisis de estructuras según la versión del SO e incluye herramientas para crear perfiles de kernel de Linux/Mac a partir de símbolos de depuración.
Es especialmente útil para analistas DFIR, investigadores de malware e investigadores de seguridad en juegos que realizan análisis de memoria post-mortem para detectar código inyectado, procesos ocultos y artefactos de rootkits.
