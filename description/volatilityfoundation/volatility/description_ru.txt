Volatility — оригинальный фреймворк memory forensics на Python 2 для анализа дампов ОЗУ систем Windows, Linux и macOS, предоставляющий плагины для списка процессов (pslist/psscan), перечисления DLL, извлечения кустов реестра, восстановления сетевых соединений, обнаружения модулей ядра, идентификации руткитов и анализа артефактов вредоносного ПО.
Фреймворк поддерживает несколько бэкендов адресного пространства (raw-дампы, crash dump, файлы гибернации, снимки VMware, FireWire/IEEE 1394), использует профильные определения типов для разбора структур под конкретную версию ОС и включает инструменты для построения профилей ядра Linux/Mac по отладочным символам.
Он в первую очередь полезен DFIR-аналитикам, исследователям вредоносного ПО и специалистам по безопасности игр для посмертного анализа памяти с целью обнаружения внедрённого кода, скрытых процессов и артефактов руткитов.
