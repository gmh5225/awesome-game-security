Este proyecto es la tercera versión principal del framework de forense de memoria Volatility, una reescritura completa en Python 3 diseñada para extraer artefactos digitales de muestras de memoria volátil (RAM). Admite imágenes de memoria de Windows, Linux y macOS y ofrece una arquitectura basada en plugins para analizar listas de procesos, conexiones de red, hives del registro, objetos del kernel y otros artefactos en tiempo de ejecución, sin requerir acceso al sistema investigado. El framework incluye generación de tablas de símbolos, traducción de memoria por capas y detección automática de perfiles de SO. Es la herramienta estándar para respondedores de incidentes, analistas forenses e investigadores de seguridad que realizan análisis de memoria e investigación de malware.
