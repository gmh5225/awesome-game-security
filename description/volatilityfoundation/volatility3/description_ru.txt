Этот проект — третий крупный выпуск фреймворка Volatility для анализа оперативной памяти, полная переработка на Python 3, предназначенная для извлечения цифровых артефактов из снимков энергозависимой памяти (RAM). Он поддерживает образы памяти Windows, Linux и macOS и предоставляет архитектуру на основе плагинов для анализа списков процессов, сетевых соединений, кустов реестра, объектов ядра и других артефактов времени выполнения без доступа к исследуемой системе. Фреймворк включает генерацию таблиц символов, послойный перевод памяти и автоматическое определение профилей ОС. Это стандартный инструмент для специалистов по реагированию на инциденты, судебных аналитиков и исследователей безопасности, занимающихся анализом памяти и расследованием вредоносного ПО.
