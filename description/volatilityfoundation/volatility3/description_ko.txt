이 프로젝트는 Volatility 메모리 포렌식 프레임워크의 세 번째 주요 릴리스로, 휘발성 메모리(RAM) 샘플에서 디지털 아티팩트를 추출하도록 설계된 완전한 Python 3 재작성 버전입니다. Windows, Linux, macOS 메모리 이미지를 지원하며, 조사 대상 시스템에 접근하지 않고도 프로세스 목록, 네트워크 연결, 레지스트리 하이브, 커널 객체 및 기타 런타임 아티팩트를 분석할 수 있는 플러그인 기반 아키텍처를 제공합니다. 심볼 테이블 생성, 레이어 기반 메모리 변환, OS 프로파일 자동 탐지 기능을 포함합니다. 인시던트 대응자, 포렌식 분석가, 보안 연구원이 메모리 분석과 악성코드 조사를 수행할 때 사용하는 표준 도구입니다.
