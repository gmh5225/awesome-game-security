Este proyecto es una base de código de estilo tutorial para un cheat externo de Apex Legends con conceptos de acceso a memoria asistido por el kernel. Está escrito principalmente en C++ y documenta componentes para el secuestro del kernel, lectura/escritura de memoria del juego y manejo de entidades basado en SDK. La estructura también incluye elementos de proyecto en modo usuario típicamente usados para visualización externa o lógica de control. El caso de uso principal es la exploración educativa de la arquitectura de cheats externos y los retos de detección relacionados con el anti-cheat.
