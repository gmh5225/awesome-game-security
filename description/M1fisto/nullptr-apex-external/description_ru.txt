Этот проект — учебная по стилю кодовая база внешнего чита для Apex Legends с концепциями доступа к памяти при поддержке ядра. Он в основном написан на C++ и документирует компоненты для захвата ядра, чтения/записи игровой памяти и обработки сущностей на основе SDK. В структуре также есть элементы пользовательского режима, обычно применяемые для внешней визуализации или логики управления. Основное назначение — образовательное изучение архитектуры внешних читов и связанных задач обнаружения античитом.
