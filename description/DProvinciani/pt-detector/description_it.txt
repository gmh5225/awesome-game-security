Questo progetto è un prototipo di ricerca per Windows volto a rilevare exploit basati sul riuso di codice mediante i dati di Intel Processor Trace. Combina componenti in modalità kernel e user-mode per raccogliere i pacchetti di traccia, quindi decodifica i flussi di esecuzione per individuare comportamenti sospetti del flusso di controllo associati a tecniche come ROP o JOP. L'implementazione usa C/C++ per l'infrastruttura di acquisizione e strumenti Python per l'interpretazione e l'analisi. È rivolto alla ricerca sulla rilevazione degli exploit e alla sperimentazione sull'integrità del flusso di controllo.
