Este proyecto es un prototipo de investigación para Windows orientado a detectar exploits de reutilización de código mediante datos de Intel Processor Trace. Combina componentes de modo kernel y de modo usuario para recopilar paquetes de traza y, a continuación, decodifica los flujos de ejecución para identificar comportamientos sospechosos del flujo de control asociados a técnicas como ROP o JOP. La implementación usa C/C++ para la infraestructura de captura y herramientas en Python para la interpretación y el análisis. Está dirigido a la investigación en detección de exploits y a la experimentación con integridad del flujo de control.
