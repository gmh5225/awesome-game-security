Ce projet est un prototype de recherche Windows destiné à détecter les exploits de réutilisation de code à partir des données Intel Processor Trace. Il combine des composants en mode noyau et en mode utilisateur pour collecter les paquets de trace, puis décode les flux d'exécution afin d'identifier les comportements de flux de contrôle suspects associés à des techniques comme le ROP ou le JOP. L'implémentation utilise le C/C++ pour l'infrastructure de capture et des outils Python pour l'interprétation et l'analyse. Il vise la recherche en détection d'exploits et l'expérimentation sur l'intégrité du flux de contrôle.
