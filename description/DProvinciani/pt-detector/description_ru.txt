Этот проект — исследовательский прототип для Windows, предназначенный для обнаружения эксплойтов на основе повторного использования кода по данным Intel Processor Trace. Он сочетает компоненты режима ядра и пользовательского режима для сбора пакетов трассировки, затем декодирует потоки выполнения, чтобы выявлять подозрительное поведение потока управления, связанное с такими техниками, как ROP или JOP. Реализация использует C/C++ для инфраструктуры захвата и инструменты на Python для интерпретации и анализа. Проект ориентирован на исследования в области обнаружения эксплойтов и эксперименты с целостностью потока управления.
