Cheater Detector is a Lua anticheat plugin for the Starfish framework that monitors in-game players and flags behavior patterns associated with cheating. It implements configurable detection checks such as NoSlow, AutoBlock, Eagle, Scaffold, Tower, LagRange, and NoBreakDelay, each with enable toggles, violation thresholds, alert cooldowns, and optional sound alerts. The plugin tracks player movement, equipment, animations, metadata, and block-break timing, then raises alerts and emits events when violation levels are exceeded. It is aimed at Minecraft-style game security and staff monitoring use cases where operators need to identify likely cheaters from client-side or plugin-side observation.
