Questo progetto è una raccolta di programmi C Windows leggeri per raccogliere dati di sistema volatili durante la risposta agli incidenti. Ogni strumento, implementato come utilità autonoma, raccoglie artefatti specifici tra cui handle, elenchi di moduli del kernel, sessioni utente, informazioni sui driver, connessioni ICMP e impostazioni del Registro. È soprattutto utile ad analisti forensi e ricercatori di sicurezza che effettuano raccolta dello stato del sistema Windows e acquisizione di dati volatili.
