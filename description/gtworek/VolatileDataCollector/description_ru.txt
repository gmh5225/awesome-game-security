Этот проект — набор лёгких Windows-программ на C для сбора энергозависимых системных данных при реагировании на инциденты. Отдельные инструменты реализованы как автономные утилиты и собирают конкретные артефакты: дескрипторы (handles), списки модулей ядра, пользовательские сеансы, сведения о драйверах, ICMP-соединения и параметры реестра. Он в первую очередь полезен судебным аналитикам и исследователям безопасности при сборе состояния системы Windows и получении энергозависимых данных.
