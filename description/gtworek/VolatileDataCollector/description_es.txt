Este proyecto es una colección de programas C ligeros para Windows destinados a recopilar datos volátiles del sistema durante la respuesta a incidentes. Cada herramienta, implementada como utilidad independiente, reúne artefactos específicos como handles, listas de módulos del kernel, sesiones de usuario, información de controladores, conexiones ICMP y ajustes del Registro. Resulta especialmente útil para analistas forenses e investigadores de seguridad que realizan recolección del estado del sistema Windows y adquisición de datos volátiles.
