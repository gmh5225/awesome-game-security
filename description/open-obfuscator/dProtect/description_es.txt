Este proyecto es dProtect, una herramienta de ofuscación de código para aplicaciones Android e iOS construida sobre el framework LLVM obfuscator. Proporciona aplanamiento del flujo de control, sustitución de instrucciones, cifrado de cadenas, predicados opacos y transformaciones booleanas-aritméticas mixtas para código nativo compilado a través del pipeline de LLVM. Los pases LLVM en C++ operan a nivel IR, lo que los hace independientes de la arquitectura. Está dirigido a desarrolladores móviles e investigadores en protección de software que aplican ofuscación al código nativo de aplicaciones móviles.
