Этот проект — dProtect, инструмент обфускации кода для приложений Android и iOS на базе фреймворка LLVM obfuscator. Он обеспечивает выравнивание потока управления, подстановку инструкций, шифрование строк, непрозрачные предикаты и смешанные булево-арифметические преобразования для нативного кода, компилируемого через конвейер LLVM. Проходы LLVM на C++ работают на уровне IR, благодаря чему они независимы от архитектуры. Инструмент предназначен для мобильных разработчиков и исследователей защиты ПО, применяющих обфускацию к нативному коду мобильных приложений.
