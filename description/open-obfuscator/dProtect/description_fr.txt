Ce projet est dProtect, un outil d'obfuscation de code pour applications Android et iOS, construit sur le framework LLVM obfuscator. Il fournit l'aplatissement du flot de contrôle, la substitution d'instructions, le chiffrement de chaînes, les prédicats opaques et des transformations booléennes-arithmétiques mixtes pour le code natif compilé via le pipeline LLVM. Les passes LLVM en C++ opèrent au niveau IR, ce qui les rend indépendantes de l'architecture. Il s'adresse aux développeurs mobiles et aux chercheurs en protection logicielle qui appliquent l'obfuscation au code natif d'applications mobiles.
