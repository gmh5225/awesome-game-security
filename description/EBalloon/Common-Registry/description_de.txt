Dieses Projekt ist ein Proof of Concept für die Kommunikation zwischen User Mode und Kernel Mode über die Windows-Registrierung. Es ist in C++ geschrieben und umfasst sowohl eine KMDF-Treiberkomponente als auch eine zugehörige User-Mode-Clientanwendung. Die Implementierung untersucht Techniken wie die benutzerdefinierte Prozess-Attach-Behandlung sowie verwandte Low-Level-Speicherverwaltungsideen, die der Autor anmerkt. Es ist hauptsächlich für die Forschung zu Windows-Internals und Experimente zum Anti-Cheat-Bypass gedacht.
