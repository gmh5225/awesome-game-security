Questo progetto è una proof of concept per la comunicazione tra user mode e kernel mode tramite il Registro di Windows. È scritto in C++ e include sia un componente driver KMDF sia un'applicazione client in user mode abbinata. L'implementazione esplora tecniche come la gestione personalizzata dell'attach dei processi e idee correlate di gestione della memoria a basso livello indicate dall'autore. È pensato principalmente per la ricerca sugli internals di Windows e per sperimentare il bypass degli anti-cheat.
