Este proyecto es una prueba de concepto para la comunicación entre el modo de usuario y el modo kernel a través del Registro de Windows. Está escrito en C++ e incluye tanto un componente de controlador KMDF como una aplicación cliente en modo de usuario asociada. La implementación explora técnicas como el manejo personalizado de attach de procesos e ideas relacionadas de gestión de memoria de bajo nivel señaladas por el autor. Está pensado principalmente para la investigación de los internals de Windows y la experimentación con bypass de anti-cheat.
