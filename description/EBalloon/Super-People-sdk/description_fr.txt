Ce projet est un SDK Unreal Engine extrait pour le jeu Super People, contenant des fichiers d'en-tête C++ générés pour les classes, structures, énumérations et objets Blueprint du jeu. Le SDK couvre les armes, les véhicules, les pions joueurs, les modules d'IA, les blueprints d'animation et les systèmes spécifiques au jeu. Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les structures du SDK Unreal Engine et la hiérarchie des objets de Super People.
