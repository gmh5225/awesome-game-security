Этот проект — дамп Unreal Engine SDK для игры Super People, содержащий сгенерированные C++-заголовочные файлы для классов, структур, перечислений и объектов Blueprint игры. SDK охватывает оружие, транспорт, пешки игроков, модули ИИ, анимационные Blueprint и игровые системы. Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим структуры Unreal Engine SDK и иерархию игровых объектов Super People.
