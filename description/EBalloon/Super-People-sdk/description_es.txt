Este proyecto es un SDK de Unreal Engine volcado del juego Super People, que contiene archivos de cabecera C++ generados para las clases, estructuras, enumeraciones y objetos Blueprint del juego. El SDK cubre armas, vehículos, peones de jugador, módulos de IA, blueprints de animación y sistemas específicos del juego. Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian las estructuras del SDK de Unreal Engine y la jerarquía de objetos de Super People.
