Este proyecto es un SDK de Unreal Engine volcado para el juego Veiled Experts, que incluye encabezados C++ generados, un volcado de nombres, un volcado de objetos y un archivo de offsets. El SDK cubre agentes, armas, vehículos, elementos de interfaz, modos de juego y subsistemas del motor propios de este shooter táctico. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que analizan los internos de Unreal Engine y las técnicas de generación de SDK.
