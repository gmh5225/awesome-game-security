Questo progetto è un SDK Unreal Engine dumpato per il gioco Veiled Experts, che include header C++ generati, un dump dei nomi, un dump degli oggetti e un file di offset. L'SDK copre agenti, armi, veicoli, elementi UI, modalità di gioco e sottosistemi del motore specifici di questo sparatutto tattico. È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che analizzano gli interni di Unreal Engine e le tecniche di generazione di SDK.
