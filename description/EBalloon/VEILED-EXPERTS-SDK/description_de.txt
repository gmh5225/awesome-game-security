Dieses Projekt ist ein gedumptes Unreal Engine SDK für das Spiel Veiled Experts und enthält generierte C++-Header, einen Names-Dump, einen Objects-Dump sowie eine Offsets-Datei. Das SDK deckt Agents, Waffen, Fahrzeuge, UI-Elemente, Spielmodi und enginespezifische Subsysteme dieses Taktik-Shooters ab. Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die Unreal-Engine-Interna und SDK-Generierungstechniken analysieren.
