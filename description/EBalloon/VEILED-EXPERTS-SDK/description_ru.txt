Этот проект — дамп Unreal Engine SDK для игры Veiled Experts: сгенерированные C++-заголовки, дамп имён, дамп объектов и файл смещений (offsets). SDK охватывает агентов, оружие, транспорт, элементы UI, игровые режимы и подсистемы движка, специфичные для этого тактического шутера. Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, анализирующим внутренности игр на Unreal Engine и методы генерации SDK.
