Este proyecto es una prueba de concepto de remapeo del kernel de Windows que copia páginas de un proceso protegido al espacio de direcciones de otro proceso. Está implementado en C++ y demuestra cómo este enfoque puede habilitar flujos de lectura/escritura de memoria y volcado tras la configuración. El repositorio incluye notas sobre los rangos de Windows 10 admitidos, advertencias operativas y riesgos de fallo si la limpieza se gestiona incorrectamente. Se usa principalmente en escenarios de bypass de anti-cheat e investigación de memoria de procesos a bajo nivel.
