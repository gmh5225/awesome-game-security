Ce projet est un instantané de type source décompilé d'un binaire de pilote noyau EasyAntiCheat. L'archive contient une volumineuse sortie C générée par Hex-Rays ainsi que des en-têtes exposant les prototypes de fonctions internes et les structures de données. Son contenu se concentre sur les routines noyau Windows de bas niveau liées aux opérations mémoire de processus, à la validation et aux chemins de contrôle anti-triche. Il est principalement utilisé par les ingénieurs en rétro-ingénierie et les chercheurs en sécurité des jeux pour une analyse éducative des mécanismes internes anti-triche.
