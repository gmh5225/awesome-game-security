Этот проект — декомпилированный снимок в стиле исходного кода бинарного файла драйвера ядра EasyAntiCheat. Архив содержит обширный вывод на C, сгенерированный Hex-Rays, а также заголовки, раскрывающие внутренние прототипы функций и структуры данных. Содержимое сосредоточено на низкоуровневых процедурах ядра Windows, связанных с операциями с памятью процессов, проверкой и путями управления античитом. В основном используется инженерами по реверс-инжинирингу и исследователями игровой безопасности для образовательного анализа внутренней работы античита.
