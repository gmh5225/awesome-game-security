Dieses Projekt ist ein kurzer technischer Proof of Concept zum Verhalten von MmCopyMemory in Windows-Kernel-Anti-Cheat-Kontexten. Es erläutert, wie Speicherscans durchgeführt werden können, und liefert ein minimales C++-Beispiel, das einen bestimmten Prüpfad patcht, um die Scan-Behandlung zu verändern. Das Repository ist klein und konzentriert sich auf einen gezielten Mechanismus statt auf ein vollständiges Framework. Es richtet sich vor allem an Anti-Cheat-Bypass-Forschung und das defensive Verständnis von Kernel-Speicherinspektionstechniken.
