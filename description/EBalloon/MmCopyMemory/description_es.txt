Este proyecto es una breve prueba de concepto técnica sobre el comportamiento de MmCopyMemory en contextos de anti-cheat del kernel de Windows. Explica cómo se pueden realizar escaneos de memoria y ofrece un ejemplo mínimo en C++ que parchea una ruta de comprobación concreta para alterar el manejo del escaneo. El repositorio es pequeño y se centra en un mecanismo concreto, no en un framework completo. Está pensado principalmente para la investigación de bypass de anti-cheat y para una comprensión defensiva de las técnicas de inspección de memoria del kernel.
