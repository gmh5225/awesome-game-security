NoVmp は、x64 PE バイナリ内の VMProtect で仮想化されたコードをリフト・解析し、プロテクターの仮想マシンからより高水準なルーチンを復元する C++ ツールです。実行ファイルから VMENTER 形式の制御転送を走査し、VMProtect のアーキテクチャとオペコードをモデル化し、仮想化されたハンドラを VTIL 中間表現へリフトして難読化解除と最適化を行います。CMake または Visual Studio でビルドでき、依存関係として VTIL-Core（Capstone と Keystone を含む）および PE 解析用の linux-pe を使用します。特定の VM の指定、イメージベースの上書き、定数難読化の除去、実験的な再コンパイルの試行といった CLI オプションも提供します。VMProtect 形式のコード仮想化を理解・解除したいリバースエンジニアやゲームセキュリティ研究者向けです。
