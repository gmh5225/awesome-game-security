NoVmp is a C++ tool that lifts and analyzes VMProtect-virtualized code in x64 PE binaries, recovering higher-level routines from the protector’s virtual machine. It scans executables for VMENTER-style control transfers, models VMProtect architecture and opcodes, and lifts virtualized handlers into the VTIL intermediate representation for deobfuscation and optimization. The project builds with CMake or Visual Studio, depends on VTIL-Core (with Capstone and Keystone) and linux-pe for PE parsing, and exposes CLI options to target specific VMs, override the image base, strip constant obfuscation, and optionally attempt experimental recompilation. It is aimed at reverse engineers and game-security researchers who need to understand or undo VMProtect-style code virtualization.
