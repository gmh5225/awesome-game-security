NoVmp는 x64 PE 바이너리에서 VMProtect로 가상화된 코드를 리프트·분석하고, 보호기의 가상 머신에서 더 높은 수준의 루틴을 복원하는 C++ 도구입니다. 실행 파일에서 VMENTER 방식의 제어 전이를 스캔하고, VMProtect 아키텍처와 opcodes를 모델링한 뒤, 가상화된 핸들러를 VTIL 중간 표현으로 리프트하여 난독화 해제와 최적화를 수행합니다. CMake 또는 Visual Studio로 빌드하며, VTIL-Core(Capstone 및 Keystone 포함)와 PE 파싱용 linux-pe에 의존합니다. 특정 VM 지정, 이미지 베이스 재정의, 상수 난독화 제거, 실험적 재컴파일 시도 등의 CLI 옵션을 제공합니다. VMProtect 스타일 코드 가상화를 이해하거나 해제해야 하는 리버스 엔지니어와 게임 보안 연구자를 대상으로 합니다.
