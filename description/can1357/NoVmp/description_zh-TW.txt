NoVmp 是一款 C++ 工具，可對 x64 PE 二進位檔中經 VMProtect 虛擬化的程式碼進行提升與分析，並從保護器的虛擬機中還原更高層次的常式。它掃描可執行檔中的 VMENTER 風格控制轉移，建模 VMProtect 架構與操作碼，並將虛擬化處理常式提升為 VTIL 中間表示，以便進行去混淆與最佳化。專案可透過 CMake 或 Visual Studio 建置，依賴 VTIL-Core（含 Capstone 與 Keystone）以及用於 PE 解析的 linux-pe，並提供命令列選項以指定目標 VM、覆寫映像基底位址、剝除常數混淆，以及選擇性地嘗試實驗性重新編譯。它面向需要理解或還原 VMProtect 風格程式碼虛擬化的逆向工程師與遊戲安全研究人員。
