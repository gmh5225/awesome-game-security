NoVmp 是一款 C++ 工具，可对 x64 PE 二进制中经 VMProtect 虚拟化的代码进行提升与分析，并从保护器的虚拟机中恢复更高层次的例程。它扫描可执行文件中的 VMENTER 风格控制转移，建模 VMProtect 架构与操作码，并将虚拟化处理程序提升为 VTIL 中间表示，以便进行去混淆与优化。项目可通过 CMake 或 Visual Studio 构建，依赖 VTIL-Core（含 Capstone 与 Keystone）以及用于 PE 解析的 linux-pe，并提供命令行选项以指定目标 VM、覆盖映像基址、剥离常量混淆，以及可选地尝试实验性重编译。它面向需要理解或还原 VMProtect 风格代码虚拟化的逆向工程师与游戏安全研究人员。
