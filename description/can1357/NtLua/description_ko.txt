이 프로젝트는 Windows 커널 안에서 Lua 스크립트를 실행하는 개념 증명입니다. Lua 5.4 인터프리터를 커널 드라이버에 내장하고, ring 0에서 실행되는 Lua 스크립트에 물리·가상 메모리 접근, 프로세스 열거, MSR 읽기/쓰기 같은 NT 커널 API를 노출합니다. C/C++ 드라이버는 사용자 모드 클라이언트와 통신하며, 클라이언트가 보낸 Lua 코드를 커널 측에서 평가합니다. 스크립트 기반 커널 인트로스펙션과 특권 컨텍스트에서 인터프리터 언어를 실행할 때의 함의를 탐구하는 커널 연구자 및 익스플로잇 개발자를 대상으로 합니다.
