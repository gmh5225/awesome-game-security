Этот проект — proof of concept для запуска Lua-скриптов внутри ядра Windows. Он встраивает интерпретатор Lua 5.4 в драйвер ядра и предоставляет Lua-скриптам, выполняющимся в ring 0, API ядра NT: доступ к физической и виртуальной памяти, перечисление процессов, чтение и запись MSR. Драйвер на C/C++ взаимодействует с клиентом в пользовательском режиме, который отправляет код Lua для выполнения на стороне ядра. Предназначен для исследователей ядра и разработчиков эксплойтов, изучающих скриптовую интроспекцию ядра и последствия запуска интерпретируемых языков в привилегированных контекстах.
