本プロジェクトは、x86-64 システム上でハイパーバイザーまたは仮想マシンモニターの存在を検出するための各種手法を示します。
プロセッサ挙動テスト、パフォーマンスモニタリングカウンタ分析、メモリ管理および TLB 異常検出、DRAM 電力利用サイドチャネルを含む複数ソースによるタイミング分析、MSR 挙動チェック、割り込み処理テストを含みます。
この C++ コードベースは Intel VMX と AMD SVM の両方を対象とし、SMBIOS、ACPI テーブル、PCI 列挙を通じて Type 1 ハイパーバイザーも検出します。
主に VM 検出技術や仮想化回避の分析を研究するアンチチートおよびハイパーバイザーセキュリティ研究者向けです。
