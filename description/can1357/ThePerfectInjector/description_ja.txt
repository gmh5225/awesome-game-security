本プロジェクトは C++ で書かれた Windows DLL インジェクターで、NtCreateThreadEx と実行時に LdrLoadDll を解決するスレッドセーフな shellcode stub を組み合わせた新しい注入手法を用います。インジェクターは位置非依存の shellcode を組み立て、ターゲットプロセスに割り当て、固定アドレスの kernel32.LoadLibrary に依存せずに対象 DLL を読み込むリモートスレッドを作成します。WoW64 注入やプロセス作成フラグといったエッジケースにも対応しています。高度な DLL 注入技術とアンチチート回避を研究するセキュリティ研究者を対象としています。
