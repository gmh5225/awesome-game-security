stelf-loader est une chaîne d'outils de recherche qui convertit des exécutables ELF Linux en scripts shell auto-chargeants. Il utilise Python et du shellcode NASM généré pour mapper les segments exécutables, restaurer les protections mémoire et sauter vers le point d'entrée d'origine via un chargeur piloté par script. Le format de sortie prend en charge les charges utiles compressées, le transport en base64, les modes d'exécution d'entrée brute et la génération de one-liners compacts. Il est surtout utile pour la recherche en développement d'exploits, les expériences de livraison de payloads et l'étude du chargement runtime des ELF.
