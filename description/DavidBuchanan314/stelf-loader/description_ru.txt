stelf-loader — исследовательский инструментарий, который преобразует исполняемые файлы Linux ELF в самозагружающиеся shell-скрипты. Он использует Python и сгенерированный NASM shellcode для отображения исполняемых сегментов, восстановления защиты памяти и перехода к исходной точке входа по пути загрузчика, управляемого скриптом. Формат вывода поддерживает сжатые полезные нагрузки, передачу в base64, режимы выполнения с raw entry и генерацию компактных one-liner. В основном полезен для исследований по разработке эксплойтов, экспериментов с доставкой payload и изучения поведения загрузки ELF во время выполнения.
