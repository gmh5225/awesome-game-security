stelf-loader es una cadena de herramientas de investigación que convierte ejecutables ELF de Linux en scripts shell de autocarga. Usa Python y shellcode NASM generado para mapear segmentos ejecutables, restaurar las protecciones de memoria y saltar al punto de entrada original desde una ruta de carga impulsada por script. El formato de salida admite cargas comprimidas, transporte en base64, modos de ejecución de entrada en bruto y generación de one-liners compactos. Resulta útil sobre todo para investigación de desarrollo de exploits, experimentos de entrega de payloads y el estudio del comportamiento de carga en tiempo de ejecución de ELF.
