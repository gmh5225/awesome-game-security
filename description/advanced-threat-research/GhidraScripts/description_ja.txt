GhidraScripts は、Ghidra 内の分析ワークフローを拡張・自動化するための、継続的に維持されているスクリプト集です。主に Java ベースで、AI 支援によるリネーム、複雑度の可視化、Golang サポート、BSim ベースのマッチング、FunctionID データベース生成などのツールを含みます。また、SHAREM や Malpedia のデータなど外部の分析リソースを統合し、リバースエンジニアリングの文脈を豊かにします。スケーラブルな静的解析の自動化を必要とするマルウェアアナリストやリバースエンジニア向けに設計されており、ゲームセキュリティ調査に関連するワークフローにも対応します。
