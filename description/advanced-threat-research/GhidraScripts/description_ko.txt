GhidraScripts는 Ghidra 내부의 분석 워크플로를 확장하고 자동화하기 위해 지속적으로 유지되는 스크립트 모음입니다. 주로 Java 기반이며, AI 지원 이름 변경, 복잡도 시각화, Golang 지원, BSim 기반 매칭, FunctionID 데이터베이스 생성 도구를 포함합니다. 또한 SHAREM 및 Malpedia 데이터와 같은 외부 분석 리소스를 통합하여 리버스 엔지니어링 맥락을 보강합니다. 확장 가능한 정적 분석 자동화가 필요한 멀웨어 분석가와 리버스 엔지니어를 위해 설계되었으며, 게임 보안 조사와 관련된 워크플로도 포함합니다.
