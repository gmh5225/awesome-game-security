GhidraScripts es una colección mantenida de scripts para ampliar y automatizar flujos de trabajo de análisis dentro de Ghidra. Está basada principalmente en Java e incluye herramientas para renombrado asistido por IA, visualización de complejidad, soporte de Golang, coincidencia basada en BSim y generación de bases de datos FunctionID. El proyecto también integra recursos de análisis externos como SHAREM y datos de Malpedia para enriquecer el contexto de ingeniería inversa. Está pensado para analistas de malware e ingenieros inversos que necesitan automatización escalable del análisis estático, incluidos flujos de trabajo relevantes para investigaciones de seguridad en juegos.
