GhidraScripts — поддерживаемая коллекция скриптов для расширения и автоматизации рабочих процессов анализа в Ghidra. Проект в основном написан на Java и включает инструменты для ИИ-ассистированного переименования, визуализации сложности, поддержки Golang, сопоставления на основе BSim и генерации баз FunctionID. Также интегрируются внешние аналитические ресурсы, такие как SHAREM и данные Malpedia, для обогащения контекста реверс-инжиниринга. Предназначен для аналитиков вредоносного ПО и реверс-инженеров, которым нужна масштабируемая автоматизация статического анализа, включая рабочие процессы, релевантные для исследований безопасности игр.
