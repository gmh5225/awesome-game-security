Этот проект — ядро помощника прицеливания на базе ИИ в реальном времени, сочетающее захват экрана, обнаружение объектов и автоматическое управление мышью. Он написан на C++ и использует DXGI Desktop Duplication для захвата с низкой задержкой, ONNX Runtime с TensorRT для инференса YOLO и OpenCV для обработки изображений. Слой ввода построен вокруг интерфейса симуляции мыши на уровне драйвера, преобразующего обнаружения в относительное перемещение курсора. Основной сценарий использования — технические исследования автоматизации игр на основе компьютерного зрения в реальном времени.
