Este proyecto es un núcleo de asistente de apuntado con IA en tiempo real que combina captura de pantalla, detección de objetos y control automatizado del ratón. Está escrito en C++ y usa DXGI Desktop Duplication para captura de baja latencia, ONNX Runtime con TensorRT para inferencia YOLO y OpenCV para el procesamiento de visión. La capa de entrada se basa en una interfaz de simulación de ratón a nivel de controlador para convertir las detecciones en movimiento relativo del cursor. Su caso de uso principal es la investigación técnica sobre automatización de juegos impulsada por visión por computadora en tiempo real.
