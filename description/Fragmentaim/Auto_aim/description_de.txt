Dieses Projekt ist ein Echtzeit-KI-Zielhilfekern, der Bildschirmaufnahme, Objekterkennung und automatisierte Maussteuerung kombiniert. Es ist in C++ geschrieben und nutzt DXGI Desktop Duplication für latenzarme Aufnahme, ONNX Runtime mit TensorRT für YOLO-Inferenz sowie OpenCV für die Bildverarbeitung. Die Eingabeschicht basiert auf einer maussimulierenden Schnittstelle auf Treiberebene, die Erkennungen in relative Cursorbewegungen umsetzt. Der Hauptanwendungsfall ist die technische Forschung zur echtzeitbasierten, computervisionsgetriebenen Spielautomatisierung.
