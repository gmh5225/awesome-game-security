Herramienta de posprocesado de binarios PE que ofusca los destinos de las llamadas a API reescribiendo la tabla de importaciones para apuntar a funciones distintas (señuelo) y, al cargar, parchea los puntos de llamada para redirigir a los destinos reales mediante un resolvedor basado en shellcode, de modo que el análisis estático muestre referencias de API engañosas.
Se configura con un archivo INI que asocia llamadas reales a API con falsas; la herramienta modifica las entradas del directorio de importaciones PE e inyecta shellcode independiente de la posición que resuelve las direcciones de función reales en tiempo de ejecución recorriendo módulos vía PEB->Ldr y analizando la tabla de exportaciones.
Es útil sobre todo para investigadores de malware, red teamers e investigadores de seguridad en juegos que estudian técnicas de ofuscación de la tabla de importaciones para evadir el análisis estático y la detección basada en firmas.
