Strumento di post-processing per binari PE che offusca le destinazioni delle chiamate API riscrivendo la tabella degli import per puntare a funzioni diverse (esca), poi patcha i siti di chiamata al caricamento per reindirizzare agli obiettivi reali tramite un resolver basato su shellcode, così che l’analisi statica mostri riferimenti API fuorvianti.
Configurato tramite un file INI che mappa le chiamate API reali a quelle false, lo strumento modifica le voci della directory di import PE e inietta shellcode position-independent che risolve gli indirizzi delle funzioni reali a runtime tramite la scansione dei moduli PEB->Ldr e il parsing della tabella degli export.
È soprattutto utile a ricercatori di malware, red teamer e ricercatori di sicurezza nei giochi che studiano tecniche di offuscamento della tabella degli import per eludere l’analisi statica e il rilevamento basato su firme.
