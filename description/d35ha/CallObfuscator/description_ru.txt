Инструмент пост-обработки PE-бинарей, который обфусцирует цели вызовов API, переписывая таблицу импорта так, чтобы она указывала на другие (приманные) функции, а затем патчит места вызовов при загрузке, перенаправляя на реальные цели через резолвер на основе shellcode, из‑за чего статический анализ показывает вводящие в заблуждение ссылки на API.
Настраивается через INI-файл, сопоставляющий реальные вызовы API с поддельными; инструмент изменяет записи каталога импорта PE и внедряет позиционно-независимый shellcode, который во время выполнения разрешает адреса настоящих функций, обходя модули через PEB->Ldr и разбирая таблицу экспорта.
В основном полезен исследователям вредоносного ПО, red team и специалистам по безопасности игр, изучающим техники обфускации таблицы импорта для обхода статического анализа и сигнатурного обнаружения.
