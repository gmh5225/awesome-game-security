PE バイナリのポストプロセシングツールで、インポートテーブルを書き換え、API 呼び出し先を別の（おとり）関数へ向けて難読化し、ロード時に呼び出し箇所をパッチして shellcode ベースのリゾルバ経由で実ターゲットへリダイレクトすることで、静的解析に誤導的な API 参照を見せます。
実 API 呼び出しを偽のものへ対応づける INI ファイルで設定し、PE のインポートディレクトリ項目を変更するとともに、実行時に PEB->Ldr のモジュール走査とエクスポートテーブル解析で実関数アドレスを解決する位置非依存 shellcode を注入します。
主にマルウェア研究者、レッドチーマー、および静的解析やシグネチャベース検出を回避するインポートテーブル難読化手法を研究するゲームセキュリティ研究者向けです。
