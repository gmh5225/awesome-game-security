PE 바이너리 후처리 도구로, 임포트 테이블을 다시 써서 API 호출 대상을 다른(미끼) 함수로 가리키게 난독화한 뒤, 로드 시점에 호출 지점을 패치하여 shellcode 기반 리졸버를 통해 실제 대상으로 리다이렉트함으로써 정적 분석에 오해의 소지가 있는 API 참조를 보이게 합니다.
실제 API 호출을 가짜 호출에 매핑하는 INI 파일로 구성되며, PE 임포트 디렉터리 항목을 수정하고 PEB->Ldr 모듈 탐색과 익스포트 테이블 파싱으로 런타임에 실제 함수 주소를 해석하는 위치 독립적 shellcode를 주입합니다.
주로 정적 분석과 시그니처 기반 탐지를 우회하는 임포트 테이블 난독화 기법을 연구하는 악성코드 연구원, 레드팀, 게임 보안 연구원에게 유용합니다.
