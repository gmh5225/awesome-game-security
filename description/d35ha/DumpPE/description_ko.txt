OpenProcess/ReadProcessMemory로 원격 프로세스 메모리에 매핑된 PE 이미지를 읽고, DOS 및 NT 헤더를 파싱해 SizeOfImage로 전체 이미지 크기를 구한 뒤, 메모리상의 완전한 PE 덤프를 파일로 저장하는 경량 커맨드라인 PE 덤퍼입니다.
32비트와 64비트 프로세스를 모두 지원하며, PID·16진수 베이스 주소·출력 파일명을 인자로 받아 라이브 매핑 이미지에서 모든 섹션을 포함한 전체 PE를 재구성합니다.
주로 런타임에 언팩된 후 패킹되거나 보호된 실행 파일을 메모리에서 덤프하려는 리버스 엔지니어와 게임 보안 연구자에게 유용합니다.
