Questo progetto fornisce un ambiente basato su Docker per compilare il kernel di Windows Subsystem for Android (WSA).
Include gli strumenti di build e le dipendenze di cross-compilation necessari per i target x86_64 e arm64.
Il flusso di lavoro è pensato per build locali riproducibili e può essere usato anche nelle pipeline CI.
Le tecnologie principali sono Docker e gli strumenti di build del kernel Linux.
È rivolto soprattutto a sviluppatori e ricercatori di sicurezza che necessitano di kernel WSA personalizzati per test o strumentazione.
