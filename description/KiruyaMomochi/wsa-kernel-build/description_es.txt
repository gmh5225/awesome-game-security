Este proyecto proporciona un entorno basado en Docker para compilar el kernel de Windows Subsystem for Android (WSA).
Empaqueta las herramientas de compilación y las dependencias de compilación cruzada necesarias para los destinos x86_64 y arm64.
El flujo de trabajo está pensado para compilaciones locales reproducibles y también puede usarse en pipelines de CI.
Las tecnologías principales son Docker y las herramientas de compilación del kernel de Linux.
Está dirigido principalmente a desarrolladores e investigadores de seguridad que necesitan kernels WSA personalizados para pruebas o instrumentación.
