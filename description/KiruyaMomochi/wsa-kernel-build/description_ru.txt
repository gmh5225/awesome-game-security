Этот проект предоставляет среду на базе Docker для сборки ядра Windows Subsystem for Android (WSA).
В нём упакованы необходимые инструменты сборки и зависимости для кросс-компиляции под цели x86_64 и arm64.
Рабочий процесс рассчитан на воспроизводимые локальные сборки и также может использоваться в CI-пайплайнах.
Основные технологии — Docker и средства сборки ядра Linux.
Он в первую очередь предназначен для разработчиков и исследователей безопасности, которым нужны кастомные ядра WSA для тестирования или инструментирования.
