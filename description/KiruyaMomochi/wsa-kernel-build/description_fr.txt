Ce projet fournit un environnement basé sur Docker pour compiler le noyau Windows Subsystem for Android (WSA).
Il regroupe les outils de build et les dépendances de compilation croisée nécessaires pour les cibles x86_64 et arm64.
Le flux de travail est conçu pour des builds locaux reproductibles et peut aussi être utilisé dans des pipelines CI.
Les technologies principales sont Docker et les outils de compilation du noyau Linux.
Il s’adresse surtout aux développeurs et chercheurs en sécurité qui ont besoin de noyaux WSA personnalisés pour les tests ou l’instrumentation.
