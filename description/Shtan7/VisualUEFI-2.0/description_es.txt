Este proyecto ofrece un entorno de desarrollo al estilo VisualUEFI configurado para Clang e información de depuración DWARF. Combina código de ejemplo UEFI en C y C++ con proyectos relacionados con EDK2 y herramientas de Visual Studio para construir aplicaciones de firmware. El flujo de trabajo está pensado para depuración a nivel de código fuente en Windows mediante VMware y frontends GDB remotos como CLion. Resulta útil para desarrolladores de firmware e investigadores de seguridad que estudian los internals de UEFI y el comportamiento en el arranque.
