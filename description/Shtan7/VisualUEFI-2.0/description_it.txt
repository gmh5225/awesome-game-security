Questo progetto fornisce un ambiente di sviluppo in stile VisualUEFI configurato per Clang e le informazioni di debug DWARF. Combina codice di esempio UEFI in C e C++ con progetti correlati a EDK2 e gli strumenti di Visual Studio per compilare applicazioni firmware. Il flusso di lavoro è pensato per il debug a livello di sorgente su Windows tramite VMware e frontend GDB remoti come CLion. È utile per sviluppatori firmware e ricercatori di sicurezza che studiano gli interni di UEFI e il comportamento in fase di boot.
