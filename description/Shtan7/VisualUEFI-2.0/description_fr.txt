Ce projet fournit un environnement de développement de style VisualUEFI configuré pour Clang et les informations de débogage DWARF. Il combine des exemples de code UEFI en C et C++ avec des projets liés à EDK2 et les outils Visual Studio pour compiler des applications firmware. Le flux de travail est conçu pour le débogage au niveau source sous Windows via VMware et des frontends GDB distants tels que CLion. Il est utile aux développeurs firmware et aux chercheurs en sécurité qui étudient les internals UEFI et le comportement au démarrage.
