Этот проект предоставляет среду разработки в стиле VisualUEFI, настроенную для Clang и отладочной информации DWARF. Он объединяет примеры кода UEFI на C и C++ с проектами, связанными с EDK2, и инструментами Visual Studio для сборки прошивочных приложений. Рабочий процесс рассчитан на отладку на уровне исходного кода в Windows через VMware и удалённые фронтенды GDB, такие как CLion. Полезен разработчикам прошивок и исследователям безопасности, изучающим внутреннее устройство UEFI и поведение на этапе загрузки.
