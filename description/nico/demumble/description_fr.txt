Ce projet se concentre sur un demangler de symboles Itanium + MSVC ; D/Rust/Swift ; remplacement multiplateforme de c++filt / undname.exe.
Il peut également demangler les symboles D, Rust et Swift.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des outils de cheat / RE.
