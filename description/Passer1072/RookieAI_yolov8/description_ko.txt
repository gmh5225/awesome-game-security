이 프로젝트는 화면 캡처를 분석하고 자동 조준 동작을 구동하는 FPS 게임용 YOLO 기반 실시간 에임 어시스트 프레임워크입니다. 주로 Python으로 작성되었으며 Ultralytics 추론을 통합하고, PyTorch 가중치, TensorRT 엔진 파일, ONNX 기반 워크플로를 포함한 여러 모델 형식을 지원합니다. 코드베이스에는 구성 가능한 조준·트리거 설정, 멀티프로세스 캡처 및 추론 파이프라인, 그리고 Win32, Logitech DLL 제어, kmNet과 같은 여러 입력 제어 백엔드가 포함됩니다. 주요 사용 사례는 컴퓨터 비전 기반 조준 시스템을 중심으로 한 게임 보안 및 치트 연구입니다.
