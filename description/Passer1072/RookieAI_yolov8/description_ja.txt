本プロジェクトは、画面キャプチャを解析して自動照準動作を駆動する、FPS 向け YOLO ベースのリアルタイムエイムアシストフレームワークです。主に Python で書かれており、Ultralytics 推論を統合し、PyTorch 重み、TensorRT エンジンファイル、ONNX ベースのワークフローなど複数のモデル形式に対応します。コードベースには、設定可能な照準・トリガー設定、マルチプロセスのキャプチャと推論パイプライン、ならびに Win32、Logitech DLL 制御、kmNet などの複数の入力制御バックエンドが含まれます。主な用途は、コンピュータビジョン支援の照準システムをめぐるゲームセキュリティおよびチート研究です。
