Este proyecto es ConBeerLib, una biblioteca para detectar entornos de contenedores y virtualización desde dentro de procesos Linux. Comprueba indicadores de Docker, LXC, Kubernetes, WSL y máquinas virtuales examinando jerarquías cgroup, marcadores del sistema de archivos, variables de entorno y características de hardware. La biblioteca C/Python ayuda a las aplicaciones a determinar su entorno de ejecución. Está dirigida a investigadores de seguridad y desarrolladores de aplicaciones que implementan detección de entorno para anti-análisis o concienciación del despliegue.
