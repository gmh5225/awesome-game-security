Ce projet se concentre sur un pont Python 3 vers le scripting Python de Ghidra.
Or le scripting Python de Ghidra repose sur Jython, qui n'est pas dans un grand état de nos jours.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / plugins Ghidra.
