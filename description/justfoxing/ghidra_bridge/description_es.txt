Este proyecto se centra en un puente de Python 3 hacia el scripting Python de Ghidra.
Pero el scripting Python de Ghidra se basa en Jython, que hoy en día no está en buen estado.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / plugins de Ghidra.
