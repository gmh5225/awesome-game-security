Dieses Projekt konzentriert sich auf eine Python-3-Brücke zur Python-Skriptumgebung von Ghidra.
Doch die Python-Skriptumgebung von Ghidra basiert auf Jython, das heutzutage nicht in einem guten Zustand ist.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich von Cheats / Ghidra-Plugins untersuchen.
