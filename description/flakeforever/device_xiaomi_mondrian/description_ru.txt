Этот проект — Android device tree для Xiaomi Redmi K60 / POCO F5 Pro (mondrian) под ROM Pixel Experience Plus. Он включает конфигурацию платы, настройки аудио и дисплея, mixer paths, определения медиакодеков, а также vendor-специфичные overlay-файлы для платформы Qualcomm Waipio. В основном полезен разработчикам Android ROM и исследователям мобильной безопасности, работающим с кастомными сборками на устройствах Xiaomi на базе SM8475.
