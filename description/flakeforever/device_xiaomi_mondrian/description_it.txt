Questo progetto è un device tree Android per Xiaomi Redmi K60 / POCO F5 Pro (mondrian) destinato alla ROM Pixel Experience Plus. Include la configurazione della board, le config audio e display, i mixer paths, le definizioni dei codec multimediali e i file di overlay specifici del vendor per la piattaforma Qualcomm Waipio. È soprattutto utile per sviluppatori di ROM Android e ricercatori di sicurezza mobile che lavorano con build personalizzate su dispositivi Xiaomi basati su SM8475.
