Este proyecto es un árbol de dispositivo Android para el Xiaomi Redmi K60 / POCO F5 Pro (mondrian) orientado a la ROM Pixel Experience Plus. Incluye la configuración de la placa, configs de audio y pantalla, mixer paths, definiciones de códecs multimedia y archivos de overlay específicos del fabricante para la plataforma Qualcomm Waipio. Es especialmente útil para desarrolladores de ROM Android e investigadores de seguridad móvil que trabajan con builds personalizados en dispositivos Xiaomi basados en SM8475.
