Questo progetto è un’integrazione ufficiale di IDA Pro per un flusso di lavoro di intelligence malware cloud e ricerca del codice. È scritto in Python e consente ricerche di similarità del codice su byte, stringhe e funzioni direttamente dalle viste di disassemblaggio. Include anche un pannello in stile notebook per riepiloghi di funzioni generati dall’IA, note modificabili e artefatti di collaborazione esportabili. Il caso d’uso principale è accelerare il triage di reverse engineering di malware e altri binari sconosciuti.
