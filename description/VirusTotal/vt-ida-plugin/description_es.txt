Este proyecto es una integración oficial de IDA Pro para un flujo de trabajo de inteligencia de malware en la nube y búsqueda de código. Está escrito en Python y permite búsquedas de similitud de código sobre bytes, cadenas y funciones directamente desde las vistas de desensamblado. También incluye un panel estilo cuaderno para resúmenes de funciones generados por IA, notas editables y artefactos de colaboración exportables. El caso de uso principal es acelerar el triaje de ingeniería inversa de malware y otros binarios desconocidos.
