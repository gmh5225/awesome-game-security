Этот проект — официальная интеграция IDA Pro для облачного рабочего процесса разведки вредоносного ПО и поиска кода. Он написан на Python и позволяет выполнять поиск схожести кода по байтам, строкам и функциям прямо из представлений дизассемблирования. Также включает панель в стиле блокнота для сводок функций, сгенерированных ИИ, редактируемых заметок и экспортируемых артефактов для совместной работы. Основной сценарий использования — ускорение triage при реверс-инжиниринге вредоносного ПО и других неизвестных двоичных файлов.
