Этот проект — библиотека обфускации строк и чисел на этапе компиляции для C++20.
Она шифрует литералы с помощью XOR-ключей, полученных из compile-time вычислений, временных меток и счётчиков, а затем расшифровывает их во время выполнения через inline или виртуальные пути.
Код предоставляет макросы для строк и числовых констант и использует вариацию ключа по символам и по значениям, чтобы снизить читаемость статических шаблонов.
Библиотека предназначена для сценариев антиреверса и усиления защиты игр, где константы в открытом виде легко становятся целями извлечения.
