Python-инструмент для reverse engineering целой программы: через ghidra_bridge извлекает вывод декомпиляции Ghidra и графы вызовов, затем рекурсивно снизу вверх суммирует функции с помощью GPT-3 (text-davinci-003). Строит естественно-языковые сводки зависимостей вызываемых функций как контекст для каждой функции, обходя ограничения окна контекста LLM и выдавая читаемый анализ программы.
