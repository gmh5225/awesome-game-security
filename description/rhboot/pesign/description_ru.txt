Этот проект — pesign, инструмент для Linux, предназначенный для подписи и проверки бинарных файлов UEFI Secure Boot в формате подписи Authenticode. Он создаёт и встраивает подписи PKCS#7 в PE-исполняемые файлы, совместимые с проверкой прошивки UEFI, и поддерживает управление сертификатами через базы данных NSS. Инструмент на языке C выполняет создание, удаление и проверку подписей для EFI-загрузчиков и образов ядра. Он рассчитан на сопровождающих дистрибутивов Linux и системных инженеров, управляющих инфраструктурой подписи UEFI Secure Boot.
