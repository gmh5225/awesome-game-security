Este proyecto es pesign, una herramienta de Linux para firmar y verificar binarios de UEFI Secure Boot con el formato de firma Authenticode. Genera e inserta firmas PKCS#7 en ejecutables PE compatibles con la verificación del firmware UEFI, y admite la gestión de certificados mediante bases de datos NSS. La herramienta en C gestiona la generación, eliminación y validación de firmas para cargadores de arranque EFI e imágenes del kernel. Está dirigida a mantenedores de distribuciones Linux e ingenieros de sistemas que administran la infraestructura de firma de UEFI Secure Boot.
