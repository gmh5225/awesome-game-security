Dieses Projekt ist eine Ghidra-Erweiterung, die Ghidra mit der RevEng.AI-API für KI-gestützte Binäranalyse verbindet. Primär in Java als Ghidra-Plugin geschrieben (gebaut mit Gradle, für Ghidra 11.4+ und Java 21), können Analysten die aktuell geöffnete Binärdatei für eine Remote-Analyse hochladen. Zu den Kernfunktionen gehören Binärcode-Ähnlichkeit, Abgleich und Umbenennung von Funktionen (einzeln oder in Batches) anhand ähnlicher bekannter Funktionen sowie KI-gestützte Dekompilierungsansichten. Es richtet sich an Reverse Engineers, die mit gestrippten oder anderweitig schwer zu analysierenden Binärdateien arbeiten und KI-gestützte Funktionsidentifikation sowie Dekompilierung in Ghidra nutzen möchten.
