Questo progetto è un'estensione di Ghidra che collega Ghidra all'API RevEng.AI per l'analisi binaria assistita dall'IA. Scritto principalmente in Java come plugin Ghidra (compilato con Gradle, destinato a Ghidra 11.4+ e Java 21), consente agli analisti di caricare il binario attualmente aperto per un'analisi remota. Le funzionalità principali includono la similarità del codice binario, l'abbinamento e la ridenominazione di funzioni (singolarmente o in batch) rispetto a funzioni note simili, e viste di decompilazione basate sull'IA. È rivolto agli reverse engineer che lavorano su binari senza simboli o altrimenti difficili da analizzare e desiderano identificazione delle funzioni e decompilazione assistite dall'IA all'interno di Ghidra.
