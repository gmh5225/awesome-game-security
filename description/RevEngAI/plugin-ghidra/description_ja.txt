本プロジェクトは、Ghidra を RevEng.AI API に接続し、AI 支援によるバイナリ解析を行う Ghidra 拡張機能です。主に Java で書かれた Ghidra プラグイン（Gradle でビルド、Ghidra 11.4+ および Java 21 対応）で、解析者は現在開いているバイナリをアップロードしてリモート解析できます。主な機能には、バイナリコードの類似性解析、既知の類似関数との照合と関数のリネーム（個別または一括）、AI によるデコンパイルビューが含まれます。シンボルが除去された、あるいは解析が難しいバイナリを扱うリバースエンジニアが、Ghidra 内で AI 支援の関数識別とデコンパイルを活用することを目的としています。
