Ce projet est une extension Ghidra qui connecte Ghidra à l'API RevEng.AI pour une analyse binaire assistée par IA. Écrit principalement en Java sous forme de plugin Ghidra (construit avec Gradle, ciblant Ghidra 11.4+ et Java 21), il permet aux analystes de téléverser le binaire actuellement ouvert pour une analyse à distance. Les capacités clés incluent la similarité de code binaire, la correspondance et le renommage de fonctions (individuellement ou en lot) par rapport à des fonctions connues similaires, ainsi que des vues de décompilation alimentées par l'IA. Il s'adresse aux ingénieurs en reverse engineering qui travaillent sur des binaires dépouillés ou difficiles à analyser et souhaitent une identification de fonctions et une décompilation assistées par IA dans Ghidra.
