Este proyecto es una extensión de Ghidra que conecta Ghidra con la API de RevEng.AI para el análisis binario asistido por IA. Escrito principalmente en Java como plugin de Ghidra (compilado con Gradle, orientado a Ghidra 11.4+ y Java 21), permite a los analistas subir el binario actualmente abierto para un análisis remoto. Las capacidades clave incluyen similitud de código binario, coincidencia y renombrado de funciones (individualmente o por lotes) frente a funciones conocidas similares, y vistas de descompilación impulsadas por IA. Está dirigido a ingenieros de ingeniería inversa que trabajan con binarios sin símbolos u otros difíciles de analizar y desean identificación de funciones y descompilación asistidas por IA dentro de Ghidra.
