Этот проект — расширение Ghidra, которое подключает Ghidra к API RevEng.AI для анализа бинарных файлов с помощью ИИ. Написано в основном на Java как плагин Ghidra (сборка через Gradle, целевые версии Ghidra 11.4+ и Java 21); аналитики могут загружать открытый бинарный файл для удалённого анализа. Ключевые возможности включают сходство бинарного кода, сопоставление и переименование функций (по отдельности или пакетно) по похожим известным функциям, а также представления декомпиляции на основе ИИ. Предназначено для специалистов по реверс-инжинирингу, работающих со stripped или иным образом трудноанализируемыми бинарными файлами и желающих использовать идентификацию функций и декомпиляцию с помощью ИИ внутри Ghidra.
