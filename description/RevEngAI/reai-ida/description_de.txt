Dieses Projekt ist ein IDA-Pro-Plugin, das den Disassembler mit einer KI-gestützten Reverse-Engineering-Plattform verbindet.
Es unterstützt den Upload von Binärdateien, funktionsbasiertes Matching nach Ähnlichkeit, automatisches Umbenennen, Auto-Unstrip-Workflows und KI-Dekompilierungsansichten.
Das Plugin ist in Python implementiert und bietet Qt-basierte Dialoge sowie Service-Schichten für die API-Integration.
Es richtet sich an Reverse Engineers, die die Analyse gestrippter Binärdateien mit maschinellem Lernen beschleunigen möchten.
