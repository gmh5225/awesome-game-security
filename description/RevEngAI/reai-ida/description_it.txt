Questo progetto è un plugin per IDA Pro che collega il disassemblatore a una piattaforma di reverse engineering assistita dall’IA.
Supporta il caricamento di binari, l’abbinamento di funzioni basato sulla similarità, la ridenominazione automatica, i flussi di lavoro auto-unstrip e le viste di decompilazione IA.
Il plugin è implementato in Python con finestre di dialogo basate su Qt e livelli di servizio per l’integrazione API.
È pensato per i reverse engineer che vogliono accelerare l’analisi di binari stripped con strumenti assistiti dal machine learning.
