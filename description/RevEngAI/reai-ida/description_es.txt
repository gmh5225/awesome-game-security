Este proyecto es un complemento de IDA Pro que conecta el desensamblador a una plataforma de ingeniería inversa asistida por IA.
Admite la carga de binarios, coincidencia de funciones por similitud, renombrado automático, flujos de trabajo de auto-unstrip y vistas de descompilación con IA.
El complemento está implementado en Python con diálogos basados en Qt y capas de servicio para la integración con la API.
Está pensado para ingenieros inversos que quieren acelerar el análisis de binarios sin símbolos con herramientas asistidas por aprendizaje automático.
