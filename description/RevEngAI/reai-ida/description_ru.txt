Этот проект — плагин для IDA Pro, который подключает дизассемблер к платформе реверс-инжиниринга с поддержкой ИИ.
Он поддерживает загрузку бинарников, сопоставление функций по сходству, автоматическое переименование, рабочие процессы auto-unstrip и представления декомпиляции с ИИ.
Плагин реализован на Python с диалогами на основе Qt и сервисными слоями для интеграции с API.
Он предназначен для реверс-инженеров, которые хотят ускорить анализ stripped-бинарников с помощью инструментов на базе машинного обучения.
