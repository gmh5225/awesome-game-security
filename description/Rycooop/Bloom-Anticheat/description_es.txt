Este proyecto es un prototipo de anti-cheat Windows x64 de múltiples componentes que combina protecciones en modo kernel y en modo usuario. Su capa de controlador usa mecanismos de callback como ObRegisterCallbacks para proteger el proceso anti-cheat y el proceso objetivo frente a operaciones hostiles sobre handles. El repositorio incluye código DLL y de aplicación de apoyo, además de la configuración de compilación para proyectos de Visual Studio. Está pensado para la experimentación con anti-cheat y para estudiar las compensaciones prácticas entre callbacks de kernel y supervisión en modo usuario.
