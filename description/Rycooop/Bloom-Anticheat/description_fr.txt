Ce projet est un prototype d'anti-cheat Windows x64 multi-composants qui combine des protections en mode noyau et en mode utilisateur. Sa couche pilote utilise des mécanismes de callback tels que ObRegisterCallbacks pour protéger le processus anti-cheat et le processus cible contre les opérations hostiles sur les handles. Le dépôt inclut le code DLL et applicatif associé, ainsi que la configuration de build pour des projets Visual Studio. Il est destiné à l'expérimentation anti-cheat et à l'étude des compromis pratiques entre les callbacks noyau et la surveillance en mode utilisateur.
