Questo progetto è un prototipo anti-cheat Windows x64 multi-componente che combina protezioni in modalità kernel e user-mode. Il relativo layer driver usa meccanismi di callback come ObRegisterCallbacks per proteggere il processo anti-cheat e il processo target da operazioni ostili sugli handle. Il repository include codice DLL e applicativo di supporto, oltre alla configurazione di build per progetti Visual Studio. È pensato per la sperimentazione anti-cheat e per lo studio dei compromessi pratici tra callback del kernel e monitoraggio in user-mode.
