Ce projet est une boîte à outils de trainer MapleStory héritée d'une époque antérieure du jeu. La base de code mélange des composants trainer en C++ avec un lanceur C# et une structure de projet Visual Studio. Elle comprend des modules liés à l'injection et au contrôle à l'exécution, le README décrivant l'efficacité historique pour certaines versions du jeu. Elle sert principalement de référence historique aux chercheurs qui étudient l'architecture des anciens trainers MMO.
