Este proyecto es un framework heredado de bot de MapleStory para tareas de juego automatizadas. Combina componentes en C++ y C# con archivos de recursos que incluyen datos de mapas y portales usados por la lógica del bot. El README indica que el proyecto se discontinuó y que algunas piezas importantes de la lógica no están incluidas en el estado publicado. Es más adecuado para la ingeniería inversa histórica y el estudio de patrones de diseño de bots de juegos antiguos.
