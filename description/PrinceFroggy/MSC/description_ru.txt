Этот проект — устаревший фреймворк бота MapleStory для автоматизированных игровых задач. Он сочетает компоненты на C++ и C# с файлами ресурсов, включающими данные карт и порталов, используемые логикой бота. В README указано, что проект прекращён и что некоторые ключевые части логики не включены в опубликованное состояние. Лучше всего подходит для исторического реверс-инжиниринга и изучения паттернов проектирования старых игровых ботов.
