Questo progetto è un framework legacy di bot per MapleStory destinato a compiti di gioco automatici. Combina componenti C++ e C# con file di risorse che includono dati di mappe e portali usati dalla logica del bot. Il README indica che il progetto è stato interrotto e che alcune parti importanti della logica non sono incluse nello stato pubblicato. È più adatto al reverse engineering storico e allo studio dei pattern di progettazione dei bot di gioco più datati.
