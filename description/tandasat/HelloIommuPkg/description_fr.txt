Ce projet est un pilote d'exécution DXE d'exemple démontrant comment programmer le remappage DMA.
Il vise à fournir un exemple de code simplifié mais fonctionnel à des fins d'apprentissage, et n'est pas conçu pour une utilisation réelle en production.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine anti cheat / detection:dma.
