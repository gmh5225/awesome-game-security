Этот проект — исследовательский гипервизор, написанный в виде драйвера UEFI и Windows в образовательных целях для процессоров Intel. Как драйвер UEFI, этот MiniVisor позволяет инспектировать активность системы ещё до загрузки операционной системы; как драйвер Windows — даёт разработчикам возможность отлаживать его привычными инструментами вроде WinDbg. Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / EFI-драйверов.
