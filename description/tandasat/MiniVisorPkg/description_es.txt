Este proyecto es un hipervisor de investigación escrito como controlador UEFI y Windows con fines educativos para procesadores Intel. Como controlador UEFI, este MiniVisor permite inspeccionar las actividades del sistema incluso antes de que arranque el sistema operativo; como controlador Windows, permite a los desarrolladores depurarlo con herramientas familiares como WinDbg. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheats / controladores EFI.
