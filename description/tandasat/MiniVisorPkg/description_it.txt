Questo progetto è un hypervisor di ricerca scritto come driver UEFI e Windows a scopo didattico per processori Intel. Come driver UEFI, questo MiniVisor consente di ispezionare le attività di sistema ancora prima dell'avvio del sistema operativo; come driver Windows, permette agli sviluppatori di eseguirne il debug con strumenti familiari come WinDbg. È principalmente utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / driver EFI.
