本プロジェクトは、DMA ベースの PCIe デバイスエミュレーション向けに PCILeech を基盤としたオープンソース FPGA ファームウェアを提供し、当初はネットワークカード型ハードウェアを主な対象としていました。実装は主に SystemVerilog と Verilog で行われ、Screamer M2、Enigma X1、Squirrel、および関連する Artix-7 ボード向けの Xilinx Vivado TCL スクリプトと基板制約ファイルも含まれます。設計は PCIe TLP 処理、コンフィギュレーション空間のシャドーイング、BAR 制御、FT601 USB 通信、DMA 構成で用いられる FIFO/mux データパスをカバーします。TLP 割り込み動作は特定のマザーボード向けに調整されているとされ、ACE が関連するネットワークカードファームウェアの検知を始めた後、本リリースは教育目的として位置づけられています。主な用途は、ゲームセキュリティおよびアンチチート研究の文脈で DMA/PCILeech ファームウェア技術を学ぶことです。
