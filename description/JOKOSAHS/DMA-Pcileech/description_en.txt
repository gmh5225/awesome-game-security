This project is open-source FPGA firmware for PCILeech-style DMA hardware that emulates network-card devices over PCIe. It is implemented mainly in SystemVerilog and Verilog, with Xilinx Vivado TCL scripts and board constraint files for platforms such as Screamer M.2, Enigma X1, Squirrel, and Captain/Immortal 75T boards. The design includes PCIe TLP handling, configuration-space shadowing, BAR control, FT601 USB communication, and TLP interrupt behavior tuned for certain motherboards. It targets full emulation of Killer-series network cards and related DMA firmware techniques discussed in the context of anti-cheat detection. The primary use case is educational and research-oriented study of DMA device impersonation, PCIe firmware behavior, and game-security / anti-cheat threat modeling around FPGA-based memory access.
