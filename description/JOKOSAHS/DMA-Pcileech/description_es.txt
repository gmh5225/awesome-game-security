Este proyecto ofrece firmware FPGA de código abierto basado en PCILeech para la emulación de dispositivos PCIe mediante DMA, originalmente centrado en hardware de tipo tarjeta de red. Está implementado principalmente en SystemVerilog y Verilog, con scripts TCL de Xilinx Vivado y archivos de restricciones de placa para destinos como Screamer M2, Enigma X1, Squirrel y placas Artix-7 relacionadas. El diseño cubre el manejo de TLP PCIe, el sombreado del espacio de configuración, el control de BAR, la comunicación USB FT601 y las rutas de datos FIFO/mux usadas en configuraciones DMA. Se indica que el comportamiento de interrupciones TLP está ajustado para determinadas placas base, y la publicación se presenta con fines educativos tras que ACE comenzara a detectar firmware de tarjetas de red relacionado. Su caso de uso principal es estudiar técnicas de firmware DMA/PCILeech en contextos de investigación de seguridad en juegos y anti-trampas.
