rs-ldr는 Windows x86_64에서 해시 기반 동적 WinAPI 해석을 위한 Rust 라이브러리로, 가져오기 테이블·가시적 API 문자열·로딩용 kernel32 의존성 없이 동작합니다. PEB를 순회해 해시된 이름으로 로드된 모듈을 찾고, 내보내기 디렉터리(전달된 내보내기 포함)를 파싱하며, DynApi 인터페이스를 통해 ntdll의 LdrLoadDll로 DLL을 로드할 수 있습니다. 컴파일 타임 XOR 문자열 난독화 매크로는 스택 버퍼로 디코딩되고 drop 시 스스로를 제로화하여 정적 문자열·시그니처 스캔 회피에 도움이 됩니다. 선택 기능으로는 교체 가능한 syscall SSN 해석기(Hell's Gate 스타일용 DirectResolver 참조 구현), 스핀락으로 보호되는 프로세스 전역 DynApi, 빌드별 솔트 djb2 해시, NODEFAULTLIB 빌드용 순수 Rust 메모리 내장 함수가 있습니다. 은밀한 API 해석이 필요한 no_std Windows 레드팀·리버스 엔지니어링·게임 보안 도구를 대상으로 합니다.
