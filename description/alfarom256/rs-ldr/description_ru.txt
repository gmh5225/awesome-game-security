rs-ldr — библиотека на Rust для хеш-ориентированного динамического разрешения WinAPI в Windows x86_64 без таблицы импорта, видимых строк API и зависимости от kernel32 для загрузки. Она обходит PEB, чтобы находить загруженные модули по хешированному имени, разбирает каталоги экспорта (включая форвардированные экспорты) и может загружать DLL через LdrLoadDll из ntdll посредством интерфейса DynApi. Макросы XOR-обфускации строк на этапе компиляции декодируют в стековые буферы, которые обнуляются при drop, помогая обходить статический поиск строк и сигнатур. Опциональные возможности включают подключаемый резолвер SSN для syscall (с эталонным DirectResolver для техник в стиле Hell's Gate), процессный DynApi за spinlock, djb2-хеши с солью на сборку и чисто Rust-интринсики памяти для сборок NODEFAULTLIB. Целевая аудитория — no_std-инструменты Windows для red team, реверс-инжиниринга и игровой безопасности, которым нужно скрытное разрешение API.
