rs-ldr 是一个 Rust 库，用于在 Windows x86_64 上基于哈希进行动态 WinAPI 解析，无需导入表、可见的 API 字符串，也不依赖 kernel32 来加载。它通过遍历 PEB 按哈希名称查找已加载模块，解析导出目录（包括转发导出），并可通过 DynApi 接口借助 ntdll 的 LdrLoadDll 加载 DLL。编译期 XOR 字符串混淆宏会解码到栈缓冲区，并在 drop 时清零，有助于规避静态字符串与特征扫描。可选功能包括可插拔的 syscall SSN 解析器（以及面向 Hell's Gate 风格技术的 DirectResolver 参考实现）、自旋锁保护的进程级 DynApi、按构建加盐的 djb2 哈希，以及面向 NODEFAULTLIB 构建的纯 Rust 内存内建函数。它面向需要隐蔽 API 解析的 no_std Windows 红队、逆向工程与游戏安全工具。
