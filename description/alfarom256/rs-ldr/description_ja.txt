rs-ldr は、Windows x86_64 向けにハッシュベースで動的に WinAPI を解決する Rust ライブラリで、インポートテーブル、可視の API 文字列、および読み込み用の kernel32 依存を必要としません。PEB を走査してハッシュ名でロード済みモジュールを探し、エクスポートディレクトリ（転送エクスポート含む）を解析し、DynApi インターフェース経由で ntdll の LdrLoadDll により DLL をロードできます。コンパイル時 XOR 文字列難読化マクロはスタックバッファへ復号し、drop 時にゼロ化するため、静的文字列やシグネチャスキャンの回避に役立ちます。オプション機能には、差し替え可能な syscall SSN リゾルバ（Hell's Gate 系向け DirectResolver 参照実装）、スピンロック保護のプロセス全体 DynApi、ビルドごとのソルト付き djb2 ハッシュ、NODEFAULTLIB ビルド向け純 Rust メモリ組み込み関数があります。ステルスな API 解決が必要な no_std Windows のレッドチーム、リバースエンジニアリング、ゲームセキュリティ向けツールを対象としています。
