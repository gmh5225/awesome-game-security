rs-ldr 是一個 Rust 函式庫，用於在 Windows x86_64 上以雜湊為基礎進行動態 WinAPI 解析，無需匯入表、可見的 API 字串，亦不依賴 kernel32 來載入。它會走訪 PEB，依雜湊名稱尋找已載入模組，解析匯出目錄（包含轉送匯出），並可透過 DynApi 介面借助 ntdll 的 LdrLoadDll 載入 DLL。編譯期 XOR 字串混淆巨集會解碼到堆疊緩衝區，並在 drop 時清零，有助於規避靜態字串與特徵掃描。可選功能包括可插拔的 syscall SSN 解析器（以及面向 Hell's Gate 風格技術的 DirectResolver 參考實作）、自旋鎖保護的行程級 DynApi、依建置加鹽的 djb2 雜湊，以及面向 NODEFAULTLIB 建置的純 Rust 記憶體內建函式。它面向需要隱蔽 API 解析的 no_std Windows 紅隊、逆向工程與遊戲安全工具。
