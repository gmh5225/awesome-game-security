Этот проект сосредоточен на инъекции через Virtual Space без root-прав.
Ограничения: инжектор несовместим с играми, защищёнными античит-механизмами.
Он в основном полезен исследователям безопасности игр и инженерам по реверс-инжинирингу, изучающим атакующие техники в области cheat / injection:android.
