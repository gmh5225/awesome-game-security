Ce projet se concentre sur l'injection via Virtual Space sans permissions root.
Limitations : l'injecteur n'est pas compatible avec les jeux protégés par des mécanismes anti-triche.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie étudiant les techniques offensives dans le domaine cheat / injection:android.
