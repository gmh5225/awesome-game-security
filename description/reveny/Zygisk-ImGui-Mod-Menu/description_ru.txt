Этот проект посвящён меню ImGui с использованием Zygisk.
Предупреждение: проект всё ещё находится в разработке и может работать не так, как ожидается.
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / рендера / отрисовки.
