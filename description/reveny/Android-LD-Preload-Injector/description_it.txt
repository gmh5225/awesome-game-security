Questo progetto si concentra su android-LD-Preload-Injector e usa LD-Preload per iniettare una shared library su qualsiasi versione e architettura Android. L'injector è morto e non funziona più.
Per build e installazione è richiesto Android Studio.
È principalmente utile per ricercatori di game security e reverse engineer che studiano tecniche offensive nell'ambito cheat / injection:android.
