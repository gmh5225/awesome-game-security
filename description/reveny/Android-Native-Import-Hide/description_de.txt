Dieses Projekt ist eine Bibliothek zum Verbergen und Abrufen von Imports in ELF-Binärdateien.
Sie ist hauptsächlich in C++ geschrieben und konzentriert sich auf Hooking und Debugging.
Sie ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat / Compile-Time arbeiten.
