Questo progetto è osu!framework, un framework di gioco in C# creato per osu! ma utilizzabile come motore di gioco 2D generico. Fornisce un grafo di scena con gerarchie di drawable, gestione degli input, gestione audio, modello di threading, interpolazione fluida, rendering del testo, texture atlasing e astrazione della piattaforma su OpenGL. Il framework gestisce il timing del game loop, la gestione delle risorse e il layout dell'interfaccia. È rivolto agli sviluppatori di giochi C# che cercano un framework 2D collaudato con le moderne funzionalità di .NET.
