Ce projet est osu!framework, un framework de jeu C# conçu pour osu! mais utilisable comme moteur de jeu 2D polyvalent. Il fournit un graphe de scène avec des hiérarchies de drawables, la gestion des entrées, la gestion audio, un modèle de threading, une interpolation fluide, le rendu de texte, l'atlasing de textures et une abstraction de plateforme au-dessus d'OpenGL. Le framework gère le timing de la boucle de jeu, la gestion des ressources et la disposition de l'interface. Il s'adresse aux développeurs de jeux C# souhaitant un framework 2D éprouvé avec les fonctionnalités modernes de .NET.
