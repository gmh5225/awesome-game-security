Este proyecto es osu!framework, un framework de juegos en C# creado para osu! pero usable como motor de juegos 2D de propósito general. Proporciona un grafo de escena con jerarquías de drawables, manejo de entrada, gestión de audio, modelo de hilos, interpolación suave, renderizado de texto, atlasing de texturas y abstracción de plataforma sobre OpenGL. El framework gestiona el temporizado del bucle de juego, la administración de recursos y el diseño de la IU. Está dirigido a desarrolladores de juegos en C# que buscan un framework 2D bien probado con características modernas de .NET.
