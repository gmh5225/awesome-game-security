Этот проект — osu!, открытая ритм-игра на C# с использованием osu!framework. Реализованы несколько режимов (osu!standard, osu!taiko, osu!catch, osu!mania) с разбором битмапов, отрисовкой hit object, подсчётом очков, записью реплеев, онлайн-мультиплеером и поддержкой скинов. Кодовая база включает собственный UI-фреймворк, конвейер обработки аудио, обработку ввода и системы плавной интерполяции. Проект предназначен для разработчиков ритм-игр, C#-геймдев-программистов и сообщества моддинга osu!.
