Questo progetto è osu!, un ritmo game open source scritto in C# con osu!framework. Implementa più modalità di gioco (osu!standard, osu!taiko, osu!catch, osu!mania) con parsing delle beatmap, rendering degli hit object, scoring, registrazione dei replay, multigiocatore online e supporto allo skinning. Il codice include un framework UI personalizzato, una pipeline di elaborazione audio, gestione dell’input e sistemi di interpolazione fluida. È rivolto agli sviluppatori di rhythm game, ai programmatori di giochi in C# e alla community di modding di osu!.
