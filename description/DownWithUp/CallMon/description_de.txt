Dieses Projekt ist ein Windows-Systemaufruf-Monitoring-Tool aus Kernel-Treiber und User-Mode-GUI. Es nutzt PsAltSystemCallHandlers, um Syscalls ausgewählter Prozesse abzufangen und Trap-Frame- sowie Stack-Daten über eine Named Pipe weiterzuleiten. Das Repository enthält eine C-Implementierung und eine optionale Rust-Treiber-Variante zum Experimentieren. Hauptanwendungsfälle sind Kernel-Telemetrie, Analyse von Syscall-Verhalten und Anti-Cheat-Forschung zur prozessbezogenen API-Überwachung.
