Questo progetto è uno strumento di monitoraggio delle system call di Windows composto da un driver kernel e da una GUI in user mode. Usa PsAltSystemCallHandlers per intercettare le syscall di processi selezionati e inoltra i dati di trap frame e stack tramite una named pipe. Il repository include un'implementazione in C e una variante opzionale di driver in Rust per la sperimentazione. Il caso d'uso principale è la telemetria del kernel, l'analisi del comportamento delle syscall e la ricerca anti-cheat sul monitoraggio delle API a livello di processo.
