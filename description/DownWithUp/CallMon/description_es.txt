Este proyecto es una herramienta de supervisión de llamadas al sistema de Windows formada por un controlador de núcleo y una GUI en modo usuario. Usa PsAltSystemCallHandlers para interceptar syscalls de procesos seleccionados y reenvía datos de trap frame y pila a través de una named pipe. El repositorio incluye una implementación en C y una variante opcional de controlador en Rust para experimentación. Su uso principal es telemetría de núcleo, análisis del comportamiento de syscalls e investigación anti-cheat sobre supervisión de API a nivel de proceso.
