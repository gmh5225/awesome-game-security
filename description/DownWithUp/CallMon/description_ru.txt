Этот проект — инструмент мониторинга системных вызовов Windows на основе драйвера ядра и пользовательского GUI. Он использует PsAltSystemCallHandlers для перехвата syscall выбранных процессов и передаёт данные trap frame и стека через именованный канал. Репозиторий включает реализацию на C и опциональный вариант драйвера на Rust для экспериментов. Основное применение — телеметрия ядра, анализ поведения syscall и исследование античита в области мониторинга API на уровне процесса.
