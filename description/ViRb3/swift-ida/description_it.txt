Questo progetto è un plugin per IDA Pro che migliora l'analisi di binari con convenzioni di chiamata non standard e ritorni a più valori. È scritto in Python e aggiunge azioni al menu contestuale per ritaggare le funzioni secondo convenzioni come Swift, Golang e fastcall. Il plugin può anche generare strutture di ritorno simili a tuple per rendere più leggibile l'output decompilato. Il caso d'uso principale è il reverse engineering di runtime di linguaggi moderni e ABI dei compilatori.
