Este proyecto es un plugin de IDA Pro que mejora el análisis de binarios con convenciones de llamada no estándar y retornos de múltiples valores. Está escrito en Python y añade acciones al menú contextual para volver a etiquetar funciones según convenciones como Swift, Golang y fastcall. El plugin también puede generar estructuras de retorno similares a tuplas para facilitar la interpretación de la salida descompilada. Su principal caso de uso es la ingeniería inversa de runtimes de lenguajes modernos y ABIs de compiladores.
