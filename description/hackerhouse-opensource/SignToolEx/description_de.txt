Dieses Projekt konzentriert sich auf die Signierung mit geleakten Zertifikaten. Es ermöglicht die Nutzung abgelaufener (geleakter) Zertifikate für Code Signing, gestattet jedoch kein Spoofing von Authenticode-Zeitstempeln. Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher im Bereich Anti-Cheat und Signaturtools nützlich.
