Этот проект — учебная лаборатория по безопасности с намеренно уязвимыми MCP-серверами для практических упражнений по атаке и защите. Включает сценарии вроде path traversal, косвенной prompt injection, выполнения кода через eval, typosquatting, устаревших зависимостей и утечки секретов. Реализации в основном на Python и JavaScript, с документацией по каждому сценарию и фрагментами конфигурации клиента для воспроизводимых демонстраций. Предназначен для исследователей безопасности и специалистов red team, которые хотят оценивать риски toolchain ИИ-агентов в контролируемых средах.
