Ce projet est un laboratoire de formation en sécurité composé de serveurs MCP volontairement vulnérables pour des exercices pratiques d'attaque et de défense. Il couvre des scénarios tels que le path traversal, l'injection de prompts indirecte, l'exécution de code via eval, le typosquatting, les dépendances obsolètes et la fuite de secrets. Les implémentations sont principalement en Python et JavaScript, avec une documentation par scénario et des extraits de configuration client pour des démonstrations reproductibles. Il s'adresse aux chercheurs en sécurité et aux praticiens red team qui souhaitent évaluer les risques des chaînes d'outils d'agents IA dans des environnements contrôlés.
