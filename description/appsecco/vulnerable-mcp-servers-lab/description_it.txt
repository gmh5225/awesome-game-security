Questo progetto è un laboratorio di formazione sulla sicurezza con server MCP intenzionalmente vulnerabili per esercizi pratici di attacco e difesa. Include scenari come path traversal, injection indiretta di prompt, esecuzione di codice basata su eval, typosquatting, dipendenze obsolete e perdita di secret. Le implementazioni sono principalmente in Python e JavaScript e includono documentazione per scenario e snippet di configurazione del client per demo riproducibili. Si rivolge a ricercatori di sicurezza e professionisti red team che vogliono valutare i rischi delle toolchain degli agenti IA in ambienti controllati.
