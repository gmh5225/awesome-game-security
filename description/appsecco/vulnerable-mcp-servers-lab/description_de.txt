Dieses Projekt ist ein Sicherheitstrainingslabor mit absichtlich verwundbaren MCP-Servern für praxisnahe Angriffs- und Verteidigungsübungen. Es umfasst Szenarien wie Path Traversal, indirekte Prompt Injection, Codeausführung über eval, Typosquatting, veraltete Abhängigkeiten und Secret Leakage. Die Implementierungen sind hauptsächlich in Python und JavaScript und enthalten dokumentierte Szenarien sowie Client-Konfigurationsausschnitte für reproduzierbare Demos. Zielgruppe sind Sicherheitsforscher und Red-Team-Praktiker, die Risiken in Toolchains von KI-Agenten in kontrollierten Umgebungen bewerten möchten.
