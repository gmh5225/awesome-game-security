Este proyecto es un laboratorio de formación en seguridad con servidores MCP intencionadamente vulnerables para ejercicios prácticos de ataque y defensa. Incluye escenarios como path traversal, inyección indirecta de prompts, ejecución de código basada en eval, typosquatting, dependencias obsoletas y filtración de secretos. Las implementaciones están principalmente en Python y JavaScript e incluyen documentación por escenario y fragmentos de configuración del cliente para demos reproducibles. Está dirigido a investigadores de seguridad y profesionales de red team que quieran evaluar riesgos de las toolchains de agentes de IA en entornos controlados.
