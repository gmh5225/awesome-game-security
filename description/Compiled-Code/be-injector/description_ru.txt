Этот проект — proof of concept инъекции для Windows, который патчит код подписанных модулей через отображение физической памяти, чтобы избежать артефактов copy-on-write.
Он реализован на C++ и демонстрирует технику, призванную обойти типичные предположения античита о целостности загруженных модулей.
В дизайне рассматривается обход проверок вроде мониторинга потоков, анализа API-вызовов и сигнатурных сканов путём изменения страниц до того, как обычное поведение отображения модулей разойдётся.
Сценарий использования — низкоуровневые исследования обхода античита и эксперименты по устойчивости к обнаружению.
