Este proyecto es un framework C++ externo para navegar metadatos IL2CPP y estructuras de juego desde otro proceso.
Utiliza el descubrimiento de procesos y módulos mediante WinAPI, con acceso basado en ReadProcessMemory, para enumerar ensamblados, imágenes, clases y campos por nombre.
El código proporciona abstracciones ligeras para resolver direcciones de campos estáticos y de instancia a partir de datos de offset de GameAssembly.
Su caso de uso principal es la ingeniería inversa de Unity IL2CPP y el desarrollo de herramientas externas para la investigación en seguridad de juegos.
