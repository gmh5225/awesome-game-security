Dieses Projekt ist ein externes C++-Framework zum Navigieren von IL2CPP-Metadaten und Spielstrukturen aus einem anderen Prozess.
Es nutzt WinAPI-Prozess- und Modul-Erkennung mit ReadProcessMemory-basiertem Zugriff, um Assemblies, Images, Klassen und Felder nach Namen aufzuzählen.
Der Code bietet leichtgewichtige Abstraktionen zur Auflösung von Adressen statischer und Instanzfelder aus GameAssembly-Offset-Daten.
Der primäre Anwendungsfall ist Reverse Engineering von Unity IL2CPP sowie die Entwicklung externer Tools für die Spielsicherheitsforschung.
