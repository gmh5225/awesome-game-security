Ce projet est un framework C++ externe pour parcourir les métadonnées IL2CPP et les structures de jeu depuis un autre processus.
Il utilise la découverte de processus et de modules via WinAPI, avec un accès basé sur ReadProcessMemory, pour énumérer les assemblies, images, classes et champs par nom.
Le code fournit des abstractions légères pour résoudre les adresses de champs statiques et d'instance à partir des données d'offset de GameAssembly.
Son cas d'usage principal est la rétro-ingénierie Unity IL2CPP et le développement d'outils externes pour la recherche en sécurité des jeux.
