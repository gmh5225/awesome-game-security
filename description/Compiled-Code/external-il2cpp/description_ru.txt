Этот проект — внешний C++ фреймворк для навигации по метаданным IL2CPP и игровым структурам из другого процесса.
Он использует обнаружение процессов и модулей через WinAPI и доступ на основе ReadProcessMemory для перечисления сборок, образов, классов и полей по имени.
Код предоставляет лёгкие абстракции для разрешения адресов статических и экземплярных полей на основе данных смещений GameAssembly.
Основной сценарий использования — реверс-инжиниринг Unity IL2CPP и разработка внешних инструментов для исследований в области безопасности игр.
