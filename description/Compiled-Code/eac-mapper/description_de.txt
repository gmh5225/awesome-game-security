Dieses Projekt ist ein Kernel-Mapping-Proof-of-Concept, das eine Lücke bei der Integritätsprüfung von Session-Treibern in EasyAntiCheat adressiert.
Es ist in C++ geschrieben und erklärt, wie Checks schreibgeschützter Abschnitte umgangen werden können, wenn Treiber im Anti-Cheat-Ausführungskontext nicht global gemappt sind.
Die Implementierung zeigt Strategien für Patching und Hook-Platzierung für geräuscharme User-Kernel-Kommunikation in diesem Szenario.
Es dient der Forschung zu Anti-Cheat-Internals und dem defensiven Verständnis mapper-artiger Angriffspfade.
