Этот проект — proof of concept по kernel mapping, нацеленный на слепую зону проверки целостности session-драйверов в EasyAntiCheat.
Он написан на C++ и объясняет, как проверки секций только для чтения могут быть обойдены, когда драйверы не отображены глобально в контексте выполнения античита.
Реализация демонстрирует стратегии патчинга и размещения хуков для низкошумной связи user-kernel в этом сценарии.
Проект предназначен для исследования internals античита и защитного понимания путей атаки типа mapper.
