Ce projet est une preuve de concept de mapping noyau qui cible un angle mort d’intégrité des pilotes de session dans EasyAntiCheat.
Il est écrit en C++ et explique comment les contrôles de sections en lecture seule peuvent être contournés lorsque les pilotes ne sont pas mappés globalement dans le contexte d’exécution de l’anti-cheat.
L’implémentation présente des stratégies de patch et de placement de hooks pour une communication utilisateur-noyau à faible bruit dans ce scénario.
Il est destiné à la recherche sur les internals de l’anti-cheat et à une compréhension défensive des chemins d’attaque de type mapper.
