Este proyecto es una prueba de concepto de mapeo en kernel que apunta a un punto ciego de integridad de controladores de sesión en EasyAntiCheat.
Está escrito en C++ y explica cómo pueden eludirse las comprobaciones de secciones de solo lectura cuando los controladores no están mapeados globalmente en el contexto de ejecución del anti-cheat.
La implementación muestra estrategias de parcheo y colocación de hooks para una comunicación usuario-kernel de bajo ruido en ese escenario.
Está pensado para la investigación de internals del anti-cheat y para una comprensión defensiva de rutas de ataque tipo mapper.
