DirectInput は、標準の SendInput API に頼らず、クラスサービスルーチンを呼び出すことでキーボードとマウス入力をシミュレートする Windows C++ プロジェクトです。カーネルモードドライバとユーザーモードの付属モジュールの両方を含み、Visual Studio と Windows Driver Kit で構築されています。実装では、キーボードおよびマウスのクラススタックの発見、サービスコールバックの取得、低レベルドライバ経路を通じた入力データの注入方法が示されています。そのため、ゲーム自動化やアンチチート挙動分析などの文脈における低レベル入力パイプライン研究に役立ちます。
