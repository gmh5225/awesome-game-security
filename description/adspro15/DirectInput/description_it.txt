DirectInput è un progetto Windows in C++ che simula l’input da tastiera e mouse richiamando le routine di servizio delle classi invece di affidarsi alle API standard SendInput. Include sia un driver in modalità kernel sia un modulo companion in modalità utente, creati con Visual Studio e il Windows Driver Kit. L’implementazione mostra come individuare gli stack delle classi tastiera e mouse, catturare i callback di servizio e iniettare dati di input tramite percorsi driver a basso livello. Ciò lo rende utile per la ricerca sulle pipeline di input a basso livello in contesti come l’automazione di gioco e l’analisi del comportamento degli anti-cheat.
