DirectInput — это проект на C++ для Windows, который имитирует ввод с клавиатуры и мыши через вызов class service routines, а не через стандартные API SendInput. Он включает драйвер режима ядра и пользовательский модуль-компаньон, собираемые с помощью Visual Studio и Windows Driver Kit. Реализация показывает, как обнаруживать стеки классов клавиатуры и мыши, перехватывать service callbacks и внедрять данные ввода через низкоуровневые пути драйвера. Это делает проект полезным для исследования низкоуровневого конвейера ввода в таких областях, как автоматизация игр и анализ поведения античитов.
