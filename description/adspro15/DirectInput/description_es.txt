DirectInput es un proyecto de Windows en C++ que simula la entrada de teclado y ratón invocando rutinas de servicio de clase en lugar de depender de las API estándar SendInput. Incluye un controlador en modo kernel y un módulo compañero en modo usuario, construidos con Visual Studio y el Windows Driver Kit. La implementación muestra cómo descubrir las pilas de clase de teclado y ratón, capturar callbacks de servicio e inyectar datos de entrada a través de rutas de controlador de bajo nivel. Esto lo hace útil para la investigación de pipelines de entrada de bajo nivel en contextos como la automatización de juegos y el análisis del comportamiento de anti-cheats.
