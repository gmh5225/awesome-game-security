DirectInput est un projet Windows en C++ qui simule les entrées clavier et souris en appelant des routines de service de classe, plutôt qu’en s’appuyant sur les API SendInput standard. Il comprend un pilote en mode noyau et un module compagnon en mode utilisateur, construits avec Visual Studio et le Windows Driver Kit. L’implémentation montre comment découvrir les piles de classes clavier et souris, capturer les callbacks de service et injecter des données d’entrée via des chemins de pilote bas niveau. Cela le rend utile pour la recherche sur les pipelines d’entrée bas niveau, notamment dans des contextes d’automatisation de jeu et d’analyse du comportement des anti-cheats.
