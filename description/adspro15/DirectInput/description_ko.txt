DirectInput는 표준 SendInput API에 의존하지 않고 클래스 서비스 루틴을 호출하여 키보드와 마우스 입력을 시뮬레이션하는 Windows C++ 프로젝트입니다. 커널 모드 드라이버와 사용자 모드 보조 모듈을 모두 포함하며, Visual Studio와 Windows Driver Kit으로 빌드됩니다. 구현은 키보드 및 마우스 클래스 스택을 찾고, 서비스 콜백을 캡처하며, 저수준 드라이버 경로를 통해 입력 데이터를 주입하는 방법을 보여줍니다. 따라서 게임 자동화 및 안티치트 행위 분석과 같은 맥락에서 저수준 입력 파이프라인 연구에 유용합니다.
