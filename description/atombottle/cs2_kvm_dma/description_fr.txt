Ce projet est un cheat CS2 (Counter-Strike 2) qui s'exécute depuis une machine virtuelle KVM et utilise le DMA (Direct Memory Access) pour lire la mémoire. La logique du cheat tourne dans une VM ou un OS hôte distincts, accède à la mémoire physique du jeu via du matériel DMA ou le mappage mémoire de KVM, et affiche radar ou ESP en dehors de l'OS du jeu. Cette architecture rend le cheat invisible à l'anti-cheat du jeu qui s'exécute dans la VM invitée. Il s'adresse aux chercheurs en sécurité qui étudient les architectures de cheat basées sur KVM/DMA et les défis de leur détection.
