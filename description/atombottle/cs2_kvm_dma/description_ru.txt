Этот проект — чит для CS2 (Counter-Strike 2), работающий из виртуальной машины KVM и использующий DMA (Direct Memory Access) для чтения памяти. Логика чита выполняется в отдельной ВМ или на хостовой ОС, обращаясь к физической памяти игры через DMA-оборудование или отображение памяти KVM и выводя радар или ESP вне ОС игры. Такая архитектура делает чит невидимым для античита игры, работающего внутри гостевой ВМ. Проект предназначен для исследователей безопасности, изучающих архитектуры читов на базе KVM/DMA и проблемы их обнаружения.
