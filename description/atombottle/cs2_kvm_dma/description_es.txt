Este proyecto es un cheat de CS2 (Counter-Strike 2) que opera desde una máquina virtual KVM usando DMA (Direct Memory Access) para leer memoria. Ejecuta la lógica del cheat en una VM u OS host separados, accede a la memoria física del juego mediante hardware DMA o el mapeo de memoria de KVM, y renderiza radar o ESP fuera del SO del juego. Esta arquitectura hace que el cheat sea invisible para el anti-cheat del juego que se ejecuta dentro de la VM invitada. Está dirigido a investigadores de seguridad que estudian arquitecturas de cheat basadas en KVM/DMA y los desafíos de su detección.
