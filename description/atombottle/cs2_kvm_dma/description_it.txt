Questo progetto è un cheat per CS2 (Counter-Strike 2) che opera da una macchina virtuale KVM usando DMA (Direct Memory Access) per la lettura della memoria. Esegue la logica del cheat in una VM o in un OS host separati, accede alla memoria fisica del gioco tramite hardware DMA o il memory mapping di KVM e visualizza radar o ESP fuori dall'OS del gioco. Questa architettura rende il cheat invisibile all'anti-cheat del gioco in esecuzione nella VM guest. È rivolto ai ricercatori di sicurezza che studiano le architetture di cheat basate su KVM/DMA e le sfide della loro rilevazione.
