이 프로젝트는 KVM 가상 머신에서 동작하며 DMA(Direct Memory Access)로 메모리를 읽는 CS2(Counter-Strike 2) 치트입니다. 치트 로직은 별도의 VM 또는 호스트 OS에서 실행되고, DMA 하드웨어 또는 KVM의 메모리 매핑을 통해 게임의 물리 메모리에 접근하며, 게임 OS 밖에서 레이더 또는 ESP 정보를 렌더링합니다. 이 구조 덕분에 게스트 VM 내부에서 동작하는 게임의 안티치트에는 치트가 보이지 않습니다. KVM/DMA 기반 치트 아키텍처와 그 탐지 과제를 연구하는 보안 연구자를 대상으로 합니다.
