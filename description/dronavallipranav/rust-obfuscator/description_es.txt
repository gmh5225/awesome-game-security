Un conjunto de herramientas de ofuscación de código fuente en Rust que incluye un ofuscador automático que inserta macros procedimentales para el cifrado de literales de cadena, el aplanamiento del flujo de control y el renombrado de símbolos en proyectos Rust.
Proporciona el crate de macros procedimentales cryptify para un cifrado fino de cadenas en tiempo de compilación, y el crate labyrinth_macros para ofuscar el flujo de control a nivel de código fuente.
Resulta especialmente útil para investigadores de seguridad y desarrolladores Rust que estudian técnicas de ofuscación a nivel de fuente y protegen binarios Rust frente al análisis estático.
