Набор инструментов обфускации исходного кода на Rust, включая автоматический обфускатор, который вставляет процедурные макросы для шифрования строковых литералов, выравнивания потока управления и переименования символов в проектах на Rust.
Предоставляет proc-macro крейт cryptify для детального шифрования строк на этапе компиляции и крейт labyrinth_macros для обфускации потока управления на уровне исходного кода.
В первую очередь полезен исследователям безопасности и разработчикам на Rust, изучающим техники обфускации на уровне исходников и защищающим бинарники Rust от статического анализа.
