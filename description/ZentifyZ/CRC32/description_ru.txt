Этот проект — внутренний C++ фреймворк читов для игры Rust. Он включает игровые модули для визуалов, хуков, математики, логики оружия и доступа к данным Unity/IL2CPP, а также внутриигровое меню на базе ImGui. В кодовой базе также есть утилиты обхода античита, detours, сканирование сигнатур и вспомогательные средства для spoofed-вызовов на C++ и ассемблере. В основном предназначен для разработки читов и экспериментов с обходом античита.
