Этот проект — исходный код внутреннего чита для Rust на C++ с модулями рендеринга, управления хуками и игровыми функциями. Он предоставляет компоненты для визуализации в стиле ESP, внутренних логических хуков, доступа к структурам Unity/IL2CPP и настраиваемых меню на базе ImGui. В репозитории также есть утилиты, ориентированные на обход защиты: detours, вспомогательные средства import/scan и spoofcall-процедуры для низкоуровневой обработки вызовов. В первую очередь он предназначен для прототипирования игровых читов и практики реверс-инжиниринга.
