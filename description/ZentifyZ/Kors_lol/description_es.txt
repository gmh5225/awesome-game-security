Este proyecto es un código fuente de cheat interno de Rust en C++, con módulos de renderizado, gestión de hooks y funciones de gameplay. Proporciona componentes para visuales tipo ESP, hooks de lógica interna, acceso a estructuras Unity/IL2CPP y sistemas de menú configurables construidos con ImGui. El repositorio también incluye utilidades orientadas a bypass, como detours, ayudas de import/scan y rutinas spoofcall para el manejo de llamadas de bajo nivel. Está pensado principalmente para prototipar cheats de juegos y practicar ingeniería inversa.
