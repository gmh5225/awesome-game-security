Dieses Projekt konzentriert sich auf die automatische Erkennung, welche Game-Engine eine .apk- oder .ipa-Datei verwendet.
Es ist hauptsächlich in Python geschrieben.
Es ist vor allem nützlich für Spieleentwickler, Engine-Programmierer und Grafikforscher, die im Bereich Game-Engine / Game-Engine-Erkennung arbeiten.
