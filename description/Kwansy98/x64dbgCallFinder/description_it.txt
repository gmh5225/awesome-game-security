x64dbgCallFinder è un plugin di x64dbg che aiuta a individuare funzioni runtime importanti tracciando quanto spesso vengono chiamate.
Analizza le funzioni utente, imposta breakpoint condizionali e incrementa contatori così che gli analisti possano filtrare i risultati per numero di chiamate dopo aver attivato azioni nell'applicazione.
Il progetto è implementato in C++ per l'ambiente plugin di x64dbg e include documentazione d'uso bilingue.
È utile nei flussi di reverse engineering e sicurezza dei giochi quando serve identificare rapidamente handler come callback dell'interfaccia o logica di gameplay.
