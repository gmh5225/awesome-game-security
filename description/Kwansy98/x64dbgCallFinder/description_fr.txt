x64dbgCallFinder est un plugin x64dbg qui aide à localiser les fonctions runtime importantes en suivant la fréquence à laquelle elles sont appelées.
Il parcourt les fonctions utilisateur, pose des points d'arrêt conditionnels et incrémente des compteurs afin que les analystes puissent filtrer les résultats par nombre d'appels après avoir déclenché des actions dans l'application.
Le projet est implémenté en C++ pour l'environnement de plugins x64dbg et inclut une documentation d'utilisation bilingue.
Il est utile pour l'ingénierie inverse et les workflows de sécurité des jeux lorsqu'il faut identifier rapidement des gestionnaires tels que des callbacks d'interface ou la logique de gameplay.
