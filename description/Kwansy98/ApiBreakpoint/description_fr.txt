ApiBreakpoint est un plugin GUI pour x64dbg qui facilite et accélère la pose de points d'arrêt sur les API lors du reverse engineering sous Windows.
Il est écrit en C++ avec le SDK de plugins x64dbg et compilé sous Visual Studio pour les cibles x86 et x64.
Le plugin propose un flux de travail visuel pour configurer et gérer les points d'arrêt, au lieu de s'appuyer uniquement sur des commandes manuelles du débogueur.
Il s'adresse principalement aux analystes malware, aux reverse engineers et aux chercheurs en sécurité des jeux qui ont besoin d'un suivi rapide des API à l'exécution.
