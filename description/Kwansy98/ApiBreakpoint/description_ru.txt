ApiBreakpoint — GUI-плагин для x64dbg, который упрощает и ускоряет установку точек останова на API при реверс-инжиниринге под Windows.
Он написан на C++ с использованием SDK плагинов x64dbg и собирается в Visual Studio для целей x86 и x64.
Плагин предоставляет визуальный рабочий процесс для настройки и управления точками останова вместо опоры только на ручные команды отладчика.
Он в первую очередь предназначен для аналитиков вредоносного ПО, реверс-инженеров и исследователей игровой безопасности, которым нужно быстрое отслеживание API во время выполнения.
