ApiBreakpoint es un complemento GUI para x64dbg que hace más rápido y sencillo establecer puntos de interrupción de API durante la ingeniería inversa en Windows.
Está escrito en C++ con el SDK de plugins de x64dbg y se compila en Visual Studio para destinos x86 y x64.
El complemento ofrece un flujo de trabajo visual para configurar y gestionar puntos de interrupción, en lugar de depender solo de comandos manuales del depurador.
Está orientado principalmente a analistas de malware, ingenieros inversos e investigadores de seguridad en videojuegos que necesitan un rastreo rápido de API en tiempo de ejecución.
