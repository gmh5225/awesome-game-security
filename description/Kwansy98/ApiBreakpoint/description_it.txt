ApiBreakpoint è un plugin GUI per x64dbg che rende più rapido e semplice impostare breakpoint sulle API durante il reverse engineering su Windows.
È scritto in C++ con l'SDK dei plugin di x64dbg e compilato in Visual Studio per i target x86 e x64.
Il plugin offre un flusso di lavoro visuale per configurare e gestire i breakpoint, invece di affidarsi solo ai comandi manuali del debugger.
È rivolto principalmente ad analisti malware, reverse engineer e ricercatori di sicurezza nei videogiochi che necessitano di un tracciamento rapido delle API a runtime.
