Этот проект — демонстрация user-mode обхода, которая повторно использует дескрипторы процесса LSASS для сценариев доступа к памяти.
Он реализован на C++ и представлен как альтернатива загрузке драйвера ядра в определённых сценариях обхода античита.
Репозиторий подчёркивает практические ограничения, включая риски стабильности при взаимодействии с чувствительными системными процессами.
Основной сценарий использования — изучение компромиссов user-mode в исследованиях обхода игрового античита.
