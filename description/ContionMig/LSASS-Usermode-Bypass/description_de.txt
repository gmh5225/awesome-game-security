Dieses Projekt ist eine Usermode-Bypass-Demonstration, die LSASS-Prozesshandles für Speicherzugriffs-Workflows wiederverwendet.
Es ist in C++ implementiert und wird als Alternative zum Laden von Kernel-Treibern in bestimmten Anti-Cheat-Evasion-Szenarien vorgestellt.
Das Repository betont praktische Einschränkungen, einschließlich Stabilitätsrisiken bei der Interaktion mit sensiblen Systemprozessen.
Der Hauptanwendungsfall ist die Untersuchung von Usermode-Tradeoffs in der Forschung zu Game-Anti-Cheat-Bypässen.
