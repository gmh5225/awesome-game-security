Este proyecto es un gestor de arranque personalizado y un conjunto de herramientas para sistemas Nintendo Switch, con una interfaz gráfica. Está escrito principalmente en C y código de sistema de bajo nivel, e incluye gestión de arranque para múltiples entornos, lanzamiento de payloads y soporte de parches de firmware. El kit también ofrece funciones de mantenimiento como copia de seguridad y restauración de eMMC o emuMMC, herramientas de particiones y diagnósticos de hardware. Está pensado para el modding de consolas, la investigación de firmware y la experimentación avanzada con la plataforma Switch.
