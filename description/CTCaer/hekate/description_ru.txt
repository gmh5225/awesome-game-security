Этот проект — пользовательский загрузчик и набор инструментов для систем Nintendo Switch с графическим интерфейсом. Он в основном написан на C и низкоуровневом системном коде и включает управление загрузкой для нескольких окружений, запуск payload и поддержку патчей прошивки. Набор также предоставляет функции обслуживания: резервное копирование и восстановление eMMC или emuMMC, инструменты для разделов и аппаратную диагностику. Предназначен для моддинга консолей, исследования прошивок и продвинутых экспериментов с платформой Switch.
