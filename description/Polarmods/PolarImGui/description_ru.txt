Этот проект — Android-реализация фреймворка меню на базе ImGui для игр Unity IL2CPP. Он сочетает нативный код на C++ со средствами сборки Android и описывает шаги упаковки в Android Studio. Рабочий процесс сосредоточен на создании нативных библиотек и их интеграции в целевые архитектуры APK для отрисовки внутриигрового UI. В первую очередь предназначен для моддинга мобильных игр и прототипирования чит-интерфейсов в средах IL2CPP.
