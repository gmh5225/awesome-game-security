Questo progetto è un'implementazione Android di un framework di menu basato su ImGui per giochi Unity IL2CPP. Combina codice nativo C++ con gli strumenti di build Android e documenta i passaggi di packaging in Android Studio. Il flusso di lavoro si concentra sulla produzione di librerie native e sulla loro integrazione nelle architetture APK di destinazione per il rendering dell'UI in gioco. È pensato principalmente per il modding di giochi mobile e per la prototipazione di UI cheat in ambienti IL2CPP.
