Ce projet est une implémentation Android d'un framework de menu basé sur ImGui pour les jeux Unity IL2CPP. Il combine du code natif C++ avec les outils de build Android et documente les étapes de packaging dans Android Studio. Le flux de travail se concentre sur la production de bibliothèques natives et leur intégration dans les architectures APK cibles pour le rendu d'UI en jeu. Il est principalement destiné au modding de jeux mobiles et au prototypage d'interfaces de cheat dans des environnements IL2CPP.
