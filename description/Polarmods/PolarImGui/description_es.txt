Este proyecto es una implementación en Android de un framework de menús basado en ImGui para juegos Unity IL2CPP. Combina código nativo en C++ con las herramientas de build de Android y documenta los pasos de empaquetado en Android Studio. El flujo de trabajo se centra en producir bibliotecas nativas e integrarlas en las arquitecturas APK objetivo para el renderizado de UI en el juego. Está pensado principalmente para el modding de juegos móviles y el prototipado de interfaces de cheat en entornos IL2CPP.
