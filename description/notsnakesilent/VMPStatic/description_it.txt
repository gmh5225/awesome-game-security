Questo progetto è un unpacker statico di VMProtect per file PE; supporta VMProtect 1.x–3.x e la ricostruzione delle immagini PE unpackate.
È scritto principalmente in Go e si concentra sulle pipeline di asset.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito cheat / fix VMP.
