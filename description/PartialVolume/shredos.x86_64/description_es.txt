Este proyecto ofrece un entorno Linux arrancable centrado en el borrado seguro de discos. Incluye nwipe con varios métodos de saneamiento, como patrones DoD, pases Gutmann, flujos PRNG y modos de verificación, y permite borrar varios discos. El sistema se construye con Buildroot y se distribuye como imágenes IMG e ISO compatibles con BIOS/UEFI para hardware de 64 y 32 bits, con certificados y registros opcionales de las operaciones de borrado. Lo usan principalmente equipos de seguridad, informática forense y operaciones de TI que necesitan un saneamiento fiable de medios antes de reutilizarlos o desecharlos.
