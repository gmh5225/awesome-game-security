Este proyecto demuestra varias técnicas de inyección de DLL para destinos Windows de 32 y 64 bits.
Implementa métodos como CreateRemoteThread, variantes de creación de hilos nativos, QueueUserAPC, SetWindowsHookEx y RtlCreateUserThread.
La implementación está escrita en C++ y organizada en proyectos de inyector y DLL de carga útil para Visual Studio.
Destaca los compromisos prácticos entre simplicidad de implementación, compatibilidad de procesos y superficie de detección.
Se usa principalmente para investigación de inyección de procesos y experimentación con API de Windows de bajo nivel.
