이 프로젝트는 32비트 및 64비트 Windows 대상에 대한 여러 DLL 인젝션 기법을 보여줍니다.
CreateRemoteThread, 네이티브 스레드 생성 변형, QueueUserAPC, SetWindowsHookEx, RtlCreateUserThread 등의 방법을 구현합니다.
구현은 C++로 작성되었으며 Visual Studio용 인젝터와 페이로드 DLL 프로젝트로 구성되어 있습니다.
구현 단순성, 프로세스 호환성, 탐지 면적 사이의 실질적인 절충을 강조합니다.
주로 프로세스 인젝션 연구와 저수준 Windows API 실험에 사용됩니다.
