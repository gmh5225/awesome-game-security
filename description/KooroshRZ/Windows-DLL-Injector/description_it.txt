Questo progetto dimostra diverse tecniche di iniezione DLL per target Windows a 32 e 64 bit.
Implementa metodi come CreateRemoteThread, varianti di creazione di thread nativi, QueueUserAPC, SetWindowsHookEx e RtlCreateUserThread.
L'implementazione è scritta in C++ ed è organizzata in progetti injector e DLL payload per Visual Studio.
Mette in evidenza i compromessi pratici tra semplicità di implementazione, compatibilità dei processi e superficie di rilevamento.
È usato principalmente per la ricerca sull'iniezione di processi e per esperimenti con le API Windows a basso livello.
