Этот проект демонстрирует несколько техник DLL-инъекции для 32-битных и 64-битных целей Windows.
В нём реализованы методы вроде CreateRemoteThread, вариантов создания нативных потоков, QueueUserAPC, SetWindowsHookEx и RtlCreateUserThread.
Реализация написана на C++ и организована в проекты инжектора и payload DLL для Visual Studio.
Он подчёркивает практические компромиссы между простотой реализации, совместимостью процессов и поверхностью обнаружения.
В основном используется для исследований процессной инъекции и экспериментов с низкоуровневыми Windows API.
