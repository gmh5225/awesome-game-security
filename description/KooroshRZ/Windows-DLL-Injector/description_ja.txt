このプロジェクトは、32ビットおよび64ビットの Windows ターゲット向けに、複数の DLL インジェクション手法を示します。
CreateRemoteThread、ネイティブスレッド作成の派生手法、QueueUserAPC、SetWindowsHookEx、RtlCreateUserThread などの方法を実装しています。
実装は C++ で書かれ、Visual Studio 向けにインジェクターとペイロード DLL のプロジェクトに整理されています。
実装の簡潔さ、プロセス互換性、検知されやすさのあいだの実用的なトレードオフを明確にします。
主にプロセスインジェクション研究と低レベルな Windows API の実験に用いられます。
