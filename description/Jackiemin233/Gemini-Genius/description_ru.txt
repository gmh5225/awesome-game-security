Gemini-Genius — это набор инструментов на Python для анализа схожести бинарных функций с использованием графовых эмбеддингов.
Он объединяет скрипты предобработки наборов данных, код обучения моделей и утилиты инференса, работающие с представлениями CFG и ACFG, извлечёнными с помощью IDA.
Рабочий процесс охватывает построение графов, экспорт эмбеддингов и поиск схожести между функциями-кандидатами.
Предназначен для задач реверс-инжиниринга и исследования уязвимостей, например сопоставления функций между разными бинарными файлами.
