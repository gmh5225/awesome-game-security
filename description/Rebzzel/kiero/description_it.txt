Questo progetto è una libreria universale di hooking delle API grafiche per giochi Windows.
Supporta Direct3D da 9 a 12, OpenGL e Vulkan, e si abbina a MinHook per associare detour di funzioni tramite indice della method table.
Il codice è scritto in C++ e include method table e esempi di integrazione ImGui per il rendering di overlay.
È comunemente usata nella ricerca e negli strumenti di sicurezza dei giochi per intercettare le pipeline di rendering e costruire funzionalità di debug o overlay.
