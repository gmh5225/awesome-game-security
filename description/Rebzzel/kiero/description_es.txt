Este proyecto es una biblioteca universal de hooking de APIs gráficas para juegos de Windows.
Admite Direct3D 9 a 12, OpenGL y Vulkan, y se combina con MinHook para enlazar detours de funciones por índice de tabla de métodos.
El código está escrito en C++ e incluye tablas de métodos y ejemplos de integración de ImGui para el renderizado de overlays.
Se usa habitualmente en investigación y herramientas de seguridad de juegos para interceptar pipelines de renderizado y construir funciones de depuración o overlay.
