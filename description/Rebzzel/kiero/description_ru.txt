Этот проект — универсальная библиотека перехвата графических API для игр под Windows.
Она поддерживает Direct3D 9–12, OpenGL и Vulkan и работает с MinHook, привязывая обходы функций по индексу в таблице методов.
Код написан на C++ и включает таблицы методов, а также примеры интеграции ImGui для отрисовки оверлеев.
Её часто используют в исследованиях и инструментах безопасности игр для перехвата конвейеров рендеринга и создания отладочных или оверлейных функций.
