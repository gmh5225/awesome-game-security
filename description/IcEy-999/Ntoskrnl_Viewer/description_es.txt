Ntoskrnl_Viewer es un visor de memoria del kernel de Windows que usa símbolos para inspeccionar datos de ntoskrnl desde un controlador personalizado y un cliente en modo usuario. Ofrece comandos al estilo de WinDbg como db, dw, dd, dq, d y x para leer memoria por símbolo o dirección, incluidos símbolos del kernel exportados y no exportados. El proyecto está escrito en C/C++ con módulos ring-0 y ring-3, y está orientado a sistemas x64. Es útil para ingeniería inversa del kernel, diagnóstico de problemas y el estudio de los internos de Windows a bajo nivel.
