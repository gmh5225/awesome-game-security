Ntoskrnl_Viewer — просмотрщик памяти ядра Windows, который с помощью символов анализирует данные ntoskrnl через собственный драйвер и клиент в режиме пользователя. Он предоставляет команды в стиле WinDbg (db, dw, dd, dq, d и x) для чтения памяти по символу или адресу, включая экспортируемые и неэкспортируемые символы ядра. Проект написан на C/C++ с модулями ring-0 и ring-3 и ориентирован на системы x64. Полезен для реверс-инжиниринга ядра, диагностики и изучения внутренних механизмов Windows на низком уровне.
