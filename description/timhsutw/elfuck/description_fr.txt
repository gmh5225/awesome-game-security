Ce projet est un packer et protecteur de binaires ELF pour Linux qui implémente la compression d'exécutables, le chiffrement et des techniques d'anti-débogage. La base de code en C inclut la décompression NRV2E, un verrouillage par mot de passe et le chargement ELF à l'exécution avec du code auto-modifiable. Il est surtout utile aux chercheurs en protection binaire qui étudient les techniques de packing ELF et l'obfuscation d'exécutables Linux.
