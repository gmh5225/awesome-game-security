Este proyecto es un packer y protector de binarios ELF para Linux que implementa compresión de ejecutables, cifrado y técnicas antidepuración. La base de código en C incluye descompresión NRV2E, bloqueo basado en contraseña y carga ELF en tiempo de ejecución con código auto-modificable. Resulta especialmente útil para investigadores de protección binaria que estudian técnicas de packing ELF y ofuscación de ejecutables en Linux.
