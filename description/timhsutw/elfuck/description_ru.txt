Этот проект — упаковщик и протектор ELF-бинарников для Linux, реализующий сжатие исполняемых файлов, шифрование и техники антиотладки. Кодовая база на C включает декомпрессию NRV2E, блокировку по паролю и загрузку ELF во время выполнения с самомодифицирующимся кодом. В основном полезен исследователям защиты бинарников, изучающим техники упаковки ELF и обфускацию исполняемых файлов Linux.
