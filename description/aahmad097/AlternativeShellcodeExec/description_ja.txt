Alternative Code Execution は、CreateThread ではなくコールバック API を通じてシェルコードを実行する Windows 向け概念実証プロジェクトのコレクションです。
リポジトリは C++ で実装され、多数の Visual Studio サンプルとして整理されており、それぞれが異なるコールバックベースの実行プリミティブを示します。
位置非依存のシェルコード実行と、一般的なインジェクション経路への依存を減らす API 悪用パターンに焦点を当てています。
主な用途は、ユーザーモードの回避および検知バイパス手法に関する攻撃的セキュリティとゲームセキュリティ研究です。
