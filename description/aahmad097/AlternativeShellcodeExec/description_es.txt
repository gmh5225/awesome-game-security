Alternative Code Execution es una colección de proyectos de prueba de concepto para Windows que ejecutan shellcode mediante APIs de callback en lugar de CreateThread.
El repositorio está implementado en C++ y organizado como múltiples ejemplos de Visual Studio, cada uno demostrando una primitiva de ejecución basada en callbacks distinta.
Se centra en la ejecución de shellcode independiente de la posición y en patrones de abuso de API que pueden reducir la dependencia de rutas de inyección habituales.
El caso de uso principal es la investigación en seguridad ofensiva y seguridad de juegos sobre técnicas de evasión en modo usuario y de omisión de detección.
