Alternative Code Execution — это набор Windows proof-of-concept проектов для запуска shellcode через callback API вместо CreateThread.
Репозиторий реализован на C++ и организован как множество примеров Visual Studio, каждый из которых демонстрирует отдельный callback-примитив выполнения.
Он сосредоточен на position-independent выполнении shellcode и паттернах злоупотребления API, которые могут снизить зависимость от распространённых путей инъекции.
Основной сценарий использования — исследования в offensive security и game security, посвящённые техникам уклонения в user-mode и обхода детектирования.
