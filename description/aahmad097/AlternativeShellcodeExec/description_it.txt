Alternative Code Execution è una raccolta di progetti proof-of-concept per Windows per eseguire shellcode tramite API di callback invece di CreateThread.
Il repository è implementato in C++ e organizzato in numerosi esempi Visual Studio, ciascuno dei quali dimostra una diversa primitiva di esecuzione basata su callback.
Si concentra sull'esecuzione di shellcode position-independent e su pattern di abuso delle API che possono ridurre la dipendenza dai percorsi di injection comuni.
Il caso d'uso principale è la ricerca di sicurezza offensiva e di sicurezza dei giochi sulle tecniche di evasion in user-mode e di bypass del rilevamento.
