Этот проект — маппер драйверов Windows, который очищает следы вручную отображённых драйверов ядра после их выполнения. Он отображает неподписанный драйвер в память ядра, выполняет его точку входа, а затем отменяет отображение, освобождая выделенную память, удаляя теги пула и очищая ссылки, чтобы избежать обнаружения сканерами памяти античита. Инструмент на C++ демонстрирует методы очистки после выполнения для отображённых драйверов. Он предназначен для исследователей безопасности ядра, изучающих антифорензик-маппинг драйверов и обход обнаружения.
