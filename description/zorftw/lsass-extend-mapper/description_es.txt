Este proyecto es un mapper de controladores de Windows que carga controladores de kernel no firmados ampliando el espacio de direcciones del proceso lsass.exe y usando su contexto para operaciones de kernel. Abusa del estado de confianza de lsass para mapear el código del controlador mediante una técnica que evita las huellas estándar de carga de controladores. La implementación en C++ demuestra la ejecución de código de kernel basada en el contexto del proceso. Está dirigido a investigadores de kernel que estudian técnicas avanzadas de mapeo de controladores que aprovechan contextos de procesos de confianza.
