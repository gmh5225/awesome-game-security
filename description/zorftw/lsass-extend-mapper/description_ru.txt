Этот проект — маппер драйверов Windows, который загружает неподписанные драйверы ядра, расширяя адресное пространство процесса lsass.exe и используя его контекст для операций ядра. Он злоупотребляет доверенным статусом lsass, чтобы отображать код драйвера методом, избегающим стандартных следов загрузки драйверов. Реализация на C++ демонстрирует выполнение кода ядра на основе контекста процесса. Предназначен для исследователей ядра, изучающих продвинутые техники маппинга драйверов с использованием доверенных контекстов процессов.
