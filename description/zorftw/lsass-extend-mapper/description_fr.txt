Ce projet est un mappeur de pilotes Windows qui charge des pilotes noyau non signés en étendant l'espace d'adressage du processus lsass.exe et en utilisant son contexte pour les opérations noyau. Il abuse du statut de confiance de lsass pour mapper le code du pilote via une technique qui évite les traces de chargement de pilotes standard. L'implémentation C++ démontre l'exécution de code noyau basée sur le contexte de processus. Il s'adresse aux chercheurs noyau étudiant les techniques avancées de mapping de pilotes qui s'appuient sur des contextes de processus de confiance.
