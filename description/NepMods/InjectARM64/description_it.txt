Questo progetto è una piattaforma di iniezione di cheat senza root per dispositivi ARM su Android.
Combina componenti Android in Java e Kotlin con moduli nativi in C e C++ e codice di hooking per iniettare payload in uno spazio applicativo virtualizzato.
Il framework dichiara il supporto per ARM32 e ARM64, flussi di iniezione configurabili e compatibilità con le versioni più recenti di Android.
È rivolto ai ricercatori di game hacking che vogliono prototipare tecniche di iniezione mobile senza accesso root.
