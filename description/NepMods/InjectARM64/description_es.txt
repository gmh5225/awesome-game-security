Este proyecto es una plataforma de inyección de cheats sin root para dispositivos ARM en Android.
Combina componentes Android en Java y Kotlin con módulos nativos en C y C++ y código de hooking para inyectar payloads en un espacio de aplicación virtualizado.
El framework anuncia soporte para ARM32 y ARM64, flujos de inyección configurables y compatibilidad con versiones recientes de Android.
Está dirigido a investigadores de game hacking que desean prototipar técnicas de inyección móvil sin acceso root.
