Этот проект — платформа внедрения читов без root для ARM-устройств на Android.
Она сочетает Android-компоненты на Java и Kotlin с нативными модулями на C и C++ и кодом хукинга для внедрения payload в виртуализированное пространство приложения.
Фреймворк заявляет поддержку ARM32 и ARM64, настраиваемые сценарии внедрения и совместимость с более новыми версиями Android.
Он рассчитан на исследователей в области game hacking, которым нужно прототипировать мобильные техники внедрения без root-доступа.
