Ce projet est une plateforme d'injection de cheats sans root pour les appareils ARM sous Android.
Il combine des composants Android en Java et Kotlin avec des modules natifs C et C++ et du code de hooking pour injecter des payloads dans un espace d'application virtualisé.
Le framework annonce la prise en charge d'ARM32 et d'ARM64, des flux d'injection configurables et une compatibilité avec les versions récentes d'Android.
Il s'adresse aux chercheurs en game hacking qui souhaitent prototyper des techniques d'injection mobile sans accès root.
