libGDX ist ein plattformübergreifendes Java-Framework für die Entwicklung von 2D- und 3D-Spielen aus einer gemeinsamen Codebasis. Es bietet eine High-Level-API für Grafik, Audio, Eingabe, Datei-I/O, Netzwerk und Asset-Laden, mit OpenGL-ES-ähnlichem Rendering sowie Unterstützung für Sprites, Partikeleffekte, 3D-Modelle und Shader. Plattform-Backends umfassen Android, Desktop über LWJGL, iOS über RoboVM, HTML5/WebGL über GWT und einen Headless-Modus für Server oder Tests, plus native JNI-Helfer und Erweiterungen wie Box2D-Physik. Es richtet sich an Spieleentwickler, die ein Java-Projekt für Mobile, Desktop und Web bereitstellen wollen, und ist auch nützlich beim Analysieren oder Reverse Engineering von Spielen auf diesem Stack.
