libGDX는 공유 코드베이스로 2D 및 3D 게임을 만드는 크로스플랫폼 Java 게임 개발 프레임워크입니다. 그래픽, 오디오, 입력, 파일 I/O, 네트워킹, 에셋 로딩을 아우르는 고수준 API를 제공하며, OpenGL ES 스타일 렌더링과 함께 스프라이트, 파티클 효과, 3D 모델, 셰이더를 지원합니다. 플랫폼 백엔드로는 Android, LWJGL 기반 데스크톱, RoboVM 기반 iOS, GWT 기반 HTML5/WebGL, 서버·테스트용 헤드리스 모드가 있고, 네이티브 JNI 헬퍼와 Box2D 물리 같은 확장도 포함합니다. 하나의 Java 프로젝트로 모바일, 데스크톱, 웹에 배포하려는 게임 개발자를 대상으로 하며, 이 스택으로 만든 게임을 분석하거나 리버스 엔지니어링할 때도 유용합니다.
