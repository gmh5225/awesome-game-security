libGDX è un framework Java multipiattaforma per lo sviluppo di giochi 2D e 3D a partire da una codebase condivisa. Fornisce un’API di alto livello che copre grafica, audio, input, I/O file, networking e caricamento degli asset, con rendering in stile OpenGL ES e supporto per sprite, effetti particellari, modelli 3D e shader. I backend di piattaforma includono Android, desktop tramite LWJGL, iOS tramite RoboVM, HTML5/WebGL tramite GWT e una modalità headless per server o test, oltre a helper JNI nativi ed estensioni come la fisica Box2D. È rivolto agli sviluppatori che vogliono un unico progetto Java distribuibile su mobile, desktop e web, ed è utile anche per analizzare o fare reverse engineering di giochi basati su questo stack.
