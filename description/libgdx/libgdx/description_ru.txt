libGDX — кроссплатформенный Java-фреймворк для разработки 2D- и 3D-игр из общей кодовой базы. Он предоставляет высокоуровневый API для графики, аудио, ввода, файлового I/O, сети и загрузки ресурсов, с рендерингом в стиле OpenGL ES и поддержкой спрайтов, частиц, 3D-моделей и шейдеров. Среди платформенных бэкендов — Android, десктоп через LWJGL, iOS через RoboVM, HTML5/WebGL через GWT и headless-режим для сервера или тестов, а также нативные JNI-помощники и расширения вроде физики Box2D. Фреймворк рассчитан на разработчиков, которым нужен один Java-проект для мобильных, десктопных и веб-платформ, и полезен при анализе или реверс-инжиниринге игр на этом стеке.
