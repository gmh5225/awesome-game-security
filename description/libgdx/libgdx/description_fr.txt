libGDX est un framework de développement de jeux Java multiplateforme permettant de créer des jeux 2D et 3D à partir d’une base de code partagée. Il fournit une API de haut niveau couvrant les graphismes, l’audio, les entrées, les E/S fichiers, le réseau et le chargement d’assets, avec un rendu de type OpenGL ES et la prise en charge des sprites, effets de particules, modèles 3D et shaders. Les backends incluent Android, le bureau via LWJGL, iOS via RoboVM, HTML5/WebGL via GWT, ainsi qu’un mode headless pour serveur ou tests, plus des helpers JNI natifs et des extensions telles que la physique Box2D. Il s’adresse aux développeurs qui veulent un seul projet Java déployable sur mobile, bureau et web, et est aussi utile pour analyser ou rétroconcevoir des jeux basés sur cette stack.
