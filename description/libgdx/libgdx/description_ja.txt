libGDX は、共通のコードベースから 2D / 3D ゲームを構築できるクロスプラットフォームの Java ゲーム開発フレームワークです。グラフィックス、オーディオ、入力、ファイル I/O、ネットワーク、アセット読み込みをカバーする高レベル API を提供し、OpenGL ES スタイルのレンダリングに加え、スプライト、パーティクルエフェクト、3D モデル、シェーダーをサポートします。プラットフォームバックエンドには Android、LWJGL によるデスクトップ、RoboVM による iOS、GWT による HTML5/WebGL、サーバーやテスト向けのヘッドレスモードがあり、ネイティブ JNI ヘルパーや Box2D 物理などの拡張も備えます。モバイル・デスクトップ・Web へ一つの Java プロジェクトでデプロイしたいゲーム開発者向けであり、このスタック上で作られたゲームの解析やリバースエンジニアリングにも役立ちます。
