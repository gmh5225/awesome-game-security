libGDX is a cross-platform Java game development framework for building 2D and 3D games from a shared codebase. It provides a high-level API covering graphics, audio, input, file I/O, networking, and asset loading, with OpenGL ES-style rendering and support for sprites, particle effects, 3D models, and shaders. Platform backends include Android, desktop via LWJGL, iOS via RoboVM, HTML5/WebGL via GWT, and a headless mode for server or test use, plus native JNI helpers and extensions such as Box2D physics. It is aimed at game developers who want one Java project that deploys across mobile, desktop, and web, and is also useful when analyzing or reverse-engineering games built on this stack.
