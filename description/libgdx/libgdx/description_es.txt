libGDX es un framework de desarrollo de juegos en Java multiplataforma para crear juegos 2D y 3D desde una base de código compartida. Ofrece una API de alto nivel que cubre gráficos, audio, entrada, E/S de archivos, red y carga de assets, con renderizado al estilo OpenGL ES y soporte para sprites, efectos de partículas, modelos 3D y shaders. Los backends de plataforma incluyen Android, escritorio mediante LWJGL, iOS mediante RoboVM, HTML5/WebGL mediante GWT y un modo headless para servidor o pruebas, además de helpers JNI nativos y extensiones como la física Box2D. Está pensado para desarrolladores que quieren un solo proyecto Java desplegable en móvil, escritorio y web, y también es útil al analizar o hacer ingeniería inversa de juegos construidos sobre este stack.
