gan-aimbots è un repository di ricerca con codice e dataset per studiare gli aimbot basati sull'apprendimento automatico negli sparatutto in prima persona.
È principalmente basato su Python e include script per la raccolta dati, l'addestramento di modelli, la valutazione di classificatori, la generazione di grafici e l'orchestrazione di esperimenti con scenari ViZDoom.
Il progetto riproduce le pipeline sperimentali di una pubblicazione accademica e fornisce parametri condivisi per diversi gruppi di GAN-aimbot.
È rivolto ai ricercatori di sicurezza nei videogiochi che esplorano sia i metodi di generazione di cheat sia le strategie di rilevamento.
