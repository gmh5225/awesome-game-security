gan-aimbots — исследовательский репозиторий с кодом и наборами данных для изучения аимботов на основе машинного обучения в шутерах от первого лица.
Он в основном написан на Python и включает скрипты для сбора данных, обучения моделей, оценки классификаторов, построения графиков и оркестрации экспериментов со сценариями ViZDoom.
Проект воспроизводит экспериментальные конвейеры из научной публикации и предоставляет общие параметры для разных групп GAN-аимботов.
Он предназначен для исследователей игровой безопасности, изучающих как методы генерации читов, так и стратегии их обнаружения.
