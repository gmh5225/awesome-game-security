gan-aimbots ist ein Forschungsrepository mit Code und Datensätzen zur Untersuchung maschinell lernbasierter Aimbots in Ego-Shootern.
Es basiert hauptsächlich auf Python und enthält Skripte zur Datenerfassung, Modelltrainierung, Klassifikatorbewertung, Visualisierung und Experimentorchestrierung mit ViZDoom-Szenarien.
Das Projekt reproduziert experimentelle Pipelines aus einer akademischen Veröffentlichung und stellt gemeinsame Parameter für verschiedene GAN-Aimbot-Gruppen bereit.
Es richtet sich an Spielsicherheitsforscher, die sowohl Methoden zur Cheat-Erzeugung als auch Erkennungsstrategien untersuchen.
