Questo progetto è uno scanner del layout della memoria del kernel Windows x64 incentrato su driver, processi e oggetti section. Implementato in C per WDK e Visual Studio, analizza le regioni di memoria del kernel per enumerare strutture come DRIVER_OBJECT, voci LDR e oggetti file correlati. Le note discutono le differenze di comportamento da Windows 7 a Windows 10 e includono aggiornamenti con correzioni di stabilità. Il caso d’uso principale è la forensics del kernel e la ricerca sulle strutture di memoria orientata all’anti-cheat.
