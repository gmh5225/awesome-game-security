Este proyecto es un escáner de diseño de memoria del kernel de Windows x64 centrado en controladores, procesos y objetos de sección. Implementado en C para WDK y Visual Studio, analiza regiones de memoria del kernel para enumerar estructuras como DRIVER_OBJECT, entradas LDR y objetos de archivo relacionados. Las notas tratan las diferencias de comportamiento entre Windows 7 y Windows 10 e incluyen actualizaciones con correcciones de estabilidad. Su principal caso de uso es la forense de kernel y la investigación de estructuras de memoria orientada a anti-trampas.
