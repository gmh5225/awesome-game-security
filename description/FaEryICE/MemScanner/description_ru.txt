Этот проект — сканер раскладки памяти ядра Windows x64 с упором на драйверы, процессы и объекты секций. Реализован на C для WDK и Visual Studio; сканирует области памяти ядра, чтобы перечислить структуры вроде DRIVER_OBJECT, записей LDR и связанных файловых объектов. В заметках рассматриваются различия поведения от Windows 7 до Windows 10 и включены обновления со стабилизирующими исправлениями. Основной сценарий использования — форензика ядра и исследование структур памяти в контексте античита.
