Ce projet est un scanner de disposition mémoire du noyau Windows x64 centré sur les pilotes, les processus et les objets de section. Implémenté en C pour le WDK et Visual Studio, il analyse les régions mémoire du noyau afin d’énumérer des structures telles que DRIVER_OBJECT, les entrées LDR et les objets fichier associés. Les notes abordent les différences de comportement de Windows 7 à Windows 10 et incluent des correctifs de stabilité. Son principal cas d’usage est la criminalistique du noyau et la recherche sur les structures mémoire orientée anti-triche.
