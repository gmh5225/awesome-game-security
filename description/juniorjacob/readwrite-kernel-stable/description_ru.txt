Этот проект — репозиторий из раздела cheat / rpm, посвящённый readwrite kernel stable.
Он в основном написан на C и C/C++ и сосредоточен на работе на уровне ядра, разработке драйверов и моддинге.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / rpm.
