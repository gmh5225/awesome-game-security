Этот проект — решение для трассировки и хукинга системных вызовов Android на базе Frida и обработки ловушек seccomp. Оно сочетает JavaScript-инструментацию с оркестрацией процессов на Python для захвата SVC-вызовов, стектрейсов, аргументов в регистрах и возвращаемых значений, включая поддержку многопроцессного логирования. Реализация использует проверку символов линкера и Frida CModule для перенаправления и воспроизведения поведения системных вызовов из контролируемого контекста потока. Предназначено для мобильного реверс-инжиниринга и анализа безопасности игр, где нужна видимость системных вызовов на низком уровне.
