Este proyecto es una solución de trazado y hooking de syscalls de Android construida con Frida y el manejo de traps de seccomp. Combina instrumentación en JavaScript con orquestación de procesos en Python para capturar llamadas SVC, trazas de pila, argumentos de registros y valores de retorno, incluyendo soporte de registro multiproceso. La implementación usa inspección de símbolos del linker y un Frida CModule para redirigir y reproducir el comportamiento de syscalls desde un contexto de hilo controlado. Está orientado a la ingeniería inversa móvil y al análisis de seguridad de juegos cuando se necesita visibilidad de syscalls a bajo nivel.
