本プロジェクトは、Frida と seccomp トラップ処理を用いて構築された Android システムコール追跡・フックソリューションです。JavaScript による計装と Python によるプロセス制御を組み合わせ、SVC 呼び出し、スタックトレース、レジスタ引数、戻り値を取得でき、マルチプロセスのログ記録にも対応します。実装ではリンカのシンボル検査と Frida CModule を用いて、制御されたスレッドコンテキストからシステムコールの振る舞いをリダイレクトおよびリプレイします。低レベルのシステムコール可視化が必要なモバイルリバースエンジニアリングおよびゲームセキュリティ分析を対象としています。
