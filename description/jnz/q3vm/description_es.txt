Este proyecto es un intérprete/máquina virtual (VM) ligero (un solo archivo: vm.c) e integrable para archivos de bytecode compilados (.qvm) basados en la buena y vieja entrada en lenguaje C (.c).
Se incluye un compilador C completo para generar archivos .qvm (LCC).
Resulta especialmente útil para ingenieros anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / scripts dinámicos.
