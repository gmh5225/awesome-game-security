Этот проект — лёгкий (один файл: vm.c) встраиваемый интерпретатор/виртуальная машина (VM) для скомпилированных файлов байткода (.qvm), получаемых из доброго старого ввода на языке C (.c).
В комплект входит полноценный компилятор C для генерации файлов .qvm (LCC).
Он в основном полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / динамических скриптов.
