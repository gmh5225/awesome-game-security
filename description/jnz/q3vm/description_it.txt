Questo progetto è un interprete/macchina virtuale (VM) leggero (singolo file: vm.c) e incorporabile per file bytecode compilati (.qvm) basati sul buon vecchio input in linguaggio C (.c).
È incluso un compilatore C completo per generare file .qvm (LCC).
È principalmente utile per ingegneri anti-cheat e ricercatori di sicurezza difensiva che lavorano nell'area anti-cheat / script dinamici.
