Dieses Projekt ist ein leichtgewichtiger (einzelne Datei: vm.c), einbettbarer Interpreter/eine Virtual Machine (VM) für kompilierte Bytecode-Dateien (.qvm) auf Basis guter alter C-Spracheingaben (.c).
Ein vollständiger C-Compiler zur Erzeugung von .qvm-Dateien ist enthalten (LCC).
Es eignet sich vor allem für Anti-Cheat-Ingenieure und Forscher der defensiven Sicherheit, die im Bereich Anti-Cheat / dynamische Skripte arbeiten.
