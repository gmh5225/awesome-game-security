Optick은 가벼운 인게임 계측 SDK와 WPF 기반 GUI 뷰어를 갖춘 C++ 게임 성능 프로파일러로, 프레임별 CPU 타이밍, GPU 이벤트(D3D12/Vulkan), 스레드 스케줄링, 컨텍스트 스위치, 그리고 Windows에서 ETW를 통한 하드웨어 카운터를 캡처합니다.
SDK는 OPTICK_EVENT/OPTICK_FRAME 같은 매크로로 Unreal Engine 4/5, Unity, 커스텀 엔진을 지원하며, 캡처 데이터를 TCP 소켓으로 뷰어에 전달하고 플레임 그래프, 스레드별 타임라인, 콜스택 샘플링, 프레임 비교 분석과 작업 트래커 연동(GitHub, Jira)을 제공합니다.
주로 게임 개발자와 엔진 프로그래머가 렌더링, 물리, 게임플레이 시스템을 프로파일링하여 CPU/GPU 병목을 찾는 데 유용합니다.
