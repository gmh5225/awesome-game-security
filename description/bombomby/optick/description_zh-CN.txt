Optick 是一款 C++ 游戏性能分析器，提供轻量级游戏内插桩 SDK，以及基于 WPF 的 GUI 查看器，可采集逐帧 CPU 耗时、GPU 事件（D3D12/Vulkan）、线程调度、上下文切换，并在 Windows 上通过 ETW 获取硬件计数器。
该 SDK 通过 OPTICK_EVENT/OPTICK_FRAME 等宏支持 Unreal Engine 4/5、Unity 与自研引擎，经 TCP 套接字将采集数据传至查看器，并提供火焰图、按线程时间线、调用栈采样与帧对比分析，还可对接任务跟踪工具（GitHub、Jira）。
它主要面向游戏开发者与引擎程序员，用于剖析渲染、物理与玩法系统，定位 CPU/GPU 瓶颈。
