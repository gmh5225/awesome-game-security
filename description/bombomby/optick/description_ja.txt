Optick は、軽量なゲーム内インストゥルメンテーション SDK と WPF ベースの GUI ビューアを備えた C++ 向けゲーム性能プロファイラで、フレームごとの CPU タイミング、GPU イベント（D3D12/Vulkan）、スレッドスケジューリング、コンテキストスイッチ、および Windows 上の ETW 経由によるハードウェアカウンタを取得します。
SDK は OPTICK_EVENT/OPTICK_FRAME などのマクロで Unreal Engine 4/5、Unity、独自エンジンをサポートし、キャプチャデータを TCP ソケット経由でビューアに送信し、フレイムグラフ、スレッド別タイムライン、コールスタックサンプリング、フレーム比較分析、さらにタスクトラッカー連携（GitHub、Jira）を提供します。
主にゲーム開発者やエンジンプログラマが、レンダリング、物理、ゲームプレイシステムをプロファイルして CPU/GPU のボトルネックを特定するために役立ちます。
