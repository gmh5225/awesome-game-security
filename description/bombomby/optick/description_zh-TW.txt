Optick 是一款 C++ 遊戲效能分析器，提供輕量級遊戲內插樁 SDK，以及基於 WPF 的 GUI 檢視器，可擷取逐幀 CPU 耗時、GPU 事件（D3D12/Vulkan）、執行緒排程、上下文切換，並在 Windows 上透過 ETW 取得硬體計數器。
該 SDK 透過 OPTICK_EVENT/OPTICK_FRAME 等巨集支援 Unreal Engine 4/5、Unity 與自研引擎，經 TCP 通訊端將擷取資料傳至檢視器，並提供火焰圖、依執行緒時間軸、呼叫堆疊取樣與幀對比分析，還可整合任務追蹤工具（GitHub、Jira）。
它主要面向遊戲開發者與引擎程式設計師，用於剖析算繪、物理與玩法系統，找出 CPU/GPU 瓶頸。
