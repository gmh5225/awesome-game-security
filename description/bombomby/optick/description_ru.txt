Optick — это профилировщик производительности игр на C++ с лёгким SDK инструментирования в игре и GUI-просмотрщиком на WPF, который захватывает пофреймовый тайминг CPU, GPU-события (D3D12/Vulkan), планирование потоков, переключения контекста и аппаратные счётчики через ETW в Windows.
SDK поддерживает Unreal Engine 4/5, Unity и собственные движки через макросы вроде OPTICK_EVENT/OPTICK_FRAME, передаёт данные захвата во viewer по TCP-сокетам и предоставляет flame graph’ы, таймлайны по потокам, сэмплирование стеков вызовов и сравнительный анализ кадров с интеграциями трекеров задач (GitHub, Jira).
Он в первую очередь полезен разработчикам игр и программистам движков для профилирования рендеринга, физики и gameplay-систем с целью выявления узких мест CPU/GPU.
