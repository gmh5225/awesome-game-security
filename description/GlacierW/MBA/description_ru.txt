Этот проект — основанный на QEMU каркас целосистемного динамического анализа бинарного кода под названием MBA (Malware Behavior Analyzer). Он расширяет QEMU модулями для анализа вредоносного ПО, включая memory forensics, трассировку API и мониторинг поведения в гостевых системах Windows и Linux. Репозиторий содержит полный эмулятор QEMU с добавленными возможностями инструментирования и анализа. Проект в первую очередь полезен аналитикам malware и исследователям безопасности, выполняющим целосистемный динамический анализ и интроспекцию.
