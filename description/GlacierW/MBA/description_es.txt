Este proyecto es un framework de análisis binario dinámico a nivel de sistema basado en QEMU llamado MBA (Malware Behavior Analyzer). Extiende QEMU con módulos para el análisis de malware, incluyendo forense de memoria, trazado de API y monitorización conductual en invitados Windows y Linux. El código incluye el emulador QEMU completo con capacidades añadidas de instrumentación y análisis. Resulta especialmente útil para analistas de malware e investigadores de seguridad que realizan análisis dinámico e introspección a nivel de sistema.
