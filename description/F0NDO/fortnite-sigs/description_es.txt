Este proyecto es un volcado compacto de firmas para localizar offsets clave de Unreal Engine y específicos del juego. Proporciona patrones de bytes para objetos y rutinas como punteros del mundo, name pools, despacho de eventos, comprobaciones de visibilidad y funciones relacionadas con la mira. El contenido es texto plano y está pensado para pipelines de actualización manuales o automatizados. Su caso de uso principal es mantener herramientas de análisis de juegos externas o internas que dependen de actualizaciones rápidas de offsets.
