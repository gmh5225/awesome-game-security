LLM Explainer is an IDA Pro plugin that uses a local llama.cpp server to explain functions and propose renames, types, and comments in Hex-Rays or disassembly, without writing to the database until you accept. Written mainly in Python, it streams model output live, keeps a human in the loop with editable checkboxes, and can rename functions, locals, labels, callees, and globals, plus infer structs and recover packed string tables. It is call-graph aware, supports batch and recursive auto-accept workflows across multiple llama-server endpoints, and can export a function as compiler-verified standalone C (with optional Compiler Explorer). It also offers CFG recovery and optional patching for obfuscated x86/x64 and AArch64 code. The primary audience is reverse engineers analyzing binaries—including protected or game-related software—who want private, offline LLM assistance inside IDA.
