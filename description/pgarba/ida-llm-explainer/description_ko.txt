LLM Explainer는 로컬 llama.cpp 서버를 사용해 함수를 설명하고 Hex-Rays 또는 디스어셈블리에서 이름 변경·타입·주석을 제안하는 IDA Pro 플러그인으로, 수락하기 전까지는 데이터베이스에 쓰지 않습니다. 주로 Python으로 작성되었으며, 모델 출력을 실시간 스트리밍하고 편집 가능한 체크박스로 사람이 검토하며, 함수·로컬·레이블·호출 대상·전역 이름을 바꾸고 구조체를 추론하며 패킹된 문자열 테이블을 복구할 수 있습니다. 호출 그래프를 인식하고, 여러 llama-server 엔드포인트에 걸친 배치·재귀 자동 수락 워크플로를 지원하며, 컴파일러로 검증된 독립 C로 함수를 내보낼 수 있습니다(Compiler Explorer 선택 가능). 난독화된 x86/x64 및 AArch64 코드에 대한 CFG 복구와 선택적 패치도 제공합니다. 주요 대상은 IDA 안에서 보호되거나 게임 관련 바이너리를 포함한 바이너리를 분석하며 비공개 오프라인 LLM 지원을 원하는 리버스 엔지니어입니다.
