LLM Explainer 是一款 IDA Pro 插件，通过本地 llama.cpp 服务器解释函数，并在 Hex-Rays 或反汇编视图中提出重命名、类型与注释建议，在您接受前不会写入数据库。主要用 Python 编写，可实时流式输出模型结果，通过可编辑复选框保持人工确认，并能重命名函数、局部变量、标签、被调用函数与全局变量，还可推断结构体、恢复打包字符串表。它感知调用图，支持跨多个 llama-server 端点的批量与递归自动接受工作流，并能将函数导出为经编译器验证的独立 C 代码（可选 Compiler Explorer）。还提供针对混淆的 x86/x64 与 AArch64 代码的 CFG 恢复及可选修补。主要面向在 IDA 中分析二进制（包括受保护或游戏相关软件）、需要私有离线 LLM 辅助的逆向工程师。
