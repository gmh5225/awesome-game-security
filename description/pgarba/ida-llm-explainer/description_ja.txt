LLM Explainer は、ローカルの llama.cpp サーバーを使い、関数の説明や Hex-Rays／逆アセンブリ上での名前変更・型・コメント提案を行う IDA Pro プラグインで、承認するまでデータベースには書き込みません。主に Python で書かれ、モデル出力をライブストリームし、編集可能なチェックボックスで人間が確認しながら、関数・ローカル・ラベル・呼び出し先・グローバルのリネーム、構造体の推論、パックされた文字列テーブルの復元が可能です。コールグラフを意識し、複数の llama-server エンドポイントにまたがるバッチ／再帰的自動承認ワークフローに対応し、コンパイラ検証済みのスタンドアロン C として関数をエクスポートできます（Compiler Explorer は任意）。難読化された x86/x64 および AArch64 向けの CFG 復元と任意のパッチ適用も提供します。主な対象は、保護されたものやゲーム関連を含むバイナリを IDA 上で解析し、プライベートなオフライン LLM 支援を求めるリバースエンジニアです。
