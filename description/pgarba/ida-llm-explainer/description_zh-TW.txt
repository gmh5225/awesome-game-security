LLM Explainer 是一款 IDA Pro 外掛，透過本機 llama.cpp 伺服器解釋函式，並在 Hex-Rays 或反組譯檢視中提出重新命名、型別與註解建議，在您接受前不會寫入資料庫。主要以 Python 撰寫，可即時串流模型輸出，透過可編輯核取方塊保持人工確認，並能重新命名函式、區域變數、標籤、被呼叫函式與全域變數，還可推斷結構體、還原封裝字串表。它具備呼叫圖感知能力，支援跨多個 llama-server 端點的批次與遞迴自動接受工作流程，並能將函式匯出為經編譯器驗證的獨立 C 程式碼（可選 Compiler Explorer）。亦提供針對混淆的 x86/x64 與 AArch64 程式碼的 CFG 還原及可選修補。主要面向在 IDA 中分析二進位檔（包括受保護或遊戲相關軟體）、需要私有離線 LLM 輔助的逆向工程師。
