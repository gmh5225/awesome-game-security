Emulator Wrapper Solution（EWS）と呼ばれる IDA Pro プラグインで、Unicorn Engine による CPU エミュレーションを統合し、ARM、x86、x64 など複数アーキテクチャ向けにデバッガに近いエミュレーション機能を提供します。
ワンクリックでのトレース生成、基本的なコード探索、および IDA のインターフェース内で Keystone と Capstone を用いたアセンブラ／ディスアセンブラ連携を備えています。
主に、組み込みバイナリ、Android ネイティブライブラリ、非ネイティブアーキテクチャ上の自動車ファームウェアを解析するリバースエンジニア向けです。
