Plugin de IDA Pro llamado Emulator Wrapper Solution (EWS) que integra emulación de CPU mediante Unicorn Engine para ofrecer funciones de emulación similares a un depurador en varias arquitecturas, incluidas ARM, x86 y x64.
Ofrece generación de trazas lista para usar con un clic, exploración básica de código e integración de ensamblador/desensamblador con Keystone y Capstone dentro de la interfaz de IDA.
Resulta especialmente útil para ingenieros inversos que analizan binarios embebidos, bibliotecas nativas de Android y firmware automotriz en arquitecturas no nativas.
