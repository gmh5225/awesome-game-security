本プロジェクトは btbd/access の更新フォークであり、xKdEnumerateDebuggingDevices ポインタをフックしてカーネル・ユーザーモード間通信を行い、ハンドル不要のカーネルモードプロセス操作を提供します。
ハードコードされたオフセットを Zydis ベースの動的パターン検索に置き換え、実行時にカーネル関数を発見し、単一バイナリで Windows 10（1607+）から Windows 11（24H2）までをサポートします。
カーネルドライバは実ハンドルを作成せずに PROCESS_ALL_ACCESS 操作を行い、インラインパッチなしのクリーンな .data セクションフックを使用します。
主に、ハンドルレスなプロセスアクセス、システムコールフック、バージョン非依存の Windows カーネル技術を研究するカーネルセキュリティ研究者向けです。
