Этот проект — обновлённый форк btbd/access, предоставляющий операции с процессами в режиме ядра без хендлов за счёт хука указателя xKdEnumerateDebuggingDevices для связи ядра и пользовательского режима.
Он заменяет жёстко заданные смещения динамическим поиском сигнатур на базе Zydis для обнаружения функций ядра во время выполнения и поддерживает Windows 10 (1607+) до Windows 11 (24H2) в одном бинарнике.
Драйвер ядра выполняет операции PROCESS_ALL_ACCESS без создания реальных хендлов, используя чистый хук секции .data без inline-патчей.
В основном полезен исследователям безопасности ядра, изучающим бесхендловый доступ к процессам, хуки syscall и версионно-независимые техники ядра Windows.
