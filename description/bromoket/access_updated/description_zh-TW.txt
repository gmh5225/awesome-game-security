本專案是 btbd/access 的更新分支，透過掛鉤 xKdEnumerateDebuggingDevices 指標實現核心與使用者模式通訊，從而提供無控制代碼的核心模式行程操作。
它以基於 Zydis 的動態特徵碼搜尋取代硬編碼偏移，在執行階段發現核心函式，單一二進位即可支援 Windows 10（1607+）至 Windows 11（24H2）。
核心驅動程式在不建立真實控制代碼的情況下執行 PROCESS_ALL_ACCESS 操作，採用乾淨的 .data 區段掛鉤，無內聯修補。
主要面向研究無控制代碼行程存取、系統呼叫掛鉤以及與版本無關的 Windows 核心技術的核心安全性研究人員。
