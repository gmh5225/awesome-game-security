이 프로젝트는 btbd/access의 업데이트된 포크로, xKdEnumerateDebuggingDevices 포인터를 후킹해 커널-유저모드 통신을 구현하며 핸들 없는 커널 모드 프로세스 작업을 제공합니다.
하드코딩된 오프셋을 Zydis 기반 동적 패턴 검색으로 대체하여 런타임에 커널 함수를 찾으며, 단일 바이너리로 Windows 10(1607+)부터 Windows 11(24H2)까지 지원합니다.
커널 드라이버는 실제 핸들을 만들지 않고 PROCESS_ALL_ACCESS 작업을 수행하며, 인라인 패치 없이 깔끔한 .data 섹션 후킹을 사용합니다.
주로 핸들리스 프로세스 접근, 시스템 콜 후킹, 버전 독립적인 Windows 커널 기법을 연구하는 커널 보안 연구자에게 유용합니다.
