Questo progetto è un server MCP che fornisce agli assistenti IA il controllo completo del debugger x64dbg tramite 23 mega-tools che coprono 151 endpoint REST.
Supporta stepping, breakpoint, operazioni sulla memoria, disassemblaggio, tracing, bypass anti-debug, analisi del flusso di controllo, dumping PE ed è compatibile con Claude, Cursor, Windsurf e altri client MCP.
Il server TypeScript usa union discriminate Zod per una mappatura tipizzata degli endpoint, con un plugin nativo x64dbg (.dp64/.dp32) che collega il debugger all'API REST.
È soprattutto utile per i reverse engineer che cercano workflow di debugging potenziati dall'IA e analisi binaria automatizzata tramite x64dbg.
