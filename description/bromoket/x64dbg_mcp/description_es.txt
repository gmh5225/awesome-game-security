Este proyecto es un servidor MCP que ofrece a los asistentes de IA control total sobre el depurador x64dbg mediante 23 mega-tools que cubren 151 endpoints REST.
Admite ejecución paso a paso, puntos de interrupción, operaciones de memoria, desensamblado, trazado, bypasses anti-debug, análisis de flujo de control, volcado PE, y es compatible con Claude, Cursor, Windsurf y otros clientes MCP.
El servidor TypeScript usa uniones discriminadas de Zod para un mapeo tipado de endpoints, con un plugin nativo de x64dbg (.dp64/.dp32) que conecta el depurador con la API REST.
Resulta especialmente útil para ingenieros inversos que buscan flujos de depuración aumentados por IA y análisis binario automatizado a través de x64dbg.
