이 프로젝트는 23개의 mega-tools로 151개의 REST 엔드포인트를 제공하여 AI 어시스턴트가 x64dbg 디버거를 완전히 제어할 수 있게 하는 MCP 서버입니다.
스텝 실행, 브레이크포인트, 메모리 연산, 디스어셈블리, 트레이싱, 안티디버그 우회, 제어 흐름 분석, PE 덤핑을 지원하며 Claude, Cursor, Windsurf 및 기타 MCP 클라이언트와 호환됩니다.
TypeScript 서버는 Zod 판별 유니온으로 타입 안전한 엔드포인트 매핑을 수행하며, 네이티브 x64dbg 플러그인(.dp64/.dp32)이 디버거와 REST API를 연결합니다.
주로 x64dbg를 통한 AI 강화 디버깅 워크플로와 자동화된 바이너리 분석을 원하는 리버스 엔지니어에게 유용합니다.
