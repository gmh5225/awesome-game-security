Этот проект — MCP-сервер, который даёт ИИ-ассистентам полный контроль над отладчиком x64dbg через 23 mega-tools, охватывающие 151 REST-эндпоинт.
Он поддерживает пошаговое выполнение, точки останова, операции с памятью, дизассемблирование, трассировку, обход антиотладки, анализ потока управления, дамп PE и совместим с Claude, Cursor, Windsurf и другими MCP-клиентами.
TypeScript-сервер использует дискриминированные объединения Zod для типобезопасного сопоставления эндпоинтов, а нативный плагин x64dbg (.dp64/.dp32) связывает отладчик с REST API.
Он в первую очередь полезен reverse-инженерам, которым нужны ИИ-усиленные сценарии отладки и автоматизированный анализ бинарников через x64dbg.
