Ce projet est un serveur MCP qui donne aux assistants IA un contrôle complet du débogueur x64dbg via 23 mega-tools couvrant 151 points de terminaison REST.
Il prend en charge le pas à pas, les points d'arrêt, les opérations mémoire, le désassemblage, le traçage, les contournements anti-debug, l'analyse du flot de contrôle, le dumping PE, et est compatible avec Claude, Cursor, Windsurf et d'autres clients MCP.
Le serveur TypeScript utilise des unions discriminées Zod pour un mappage d'endpoints typé de façon sûre, avec un plugin natif x64dbg (.dp64/.dp32) qui relie le débogueur à l'API REST.
Il est principalement utile aux reverse engineers cherchant des workflows de débogage augmentés par l'IA et une analyse binaire automatisée via x64dbg.
