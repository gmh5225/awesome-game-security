本プロジェクトは、23 の mega-tools で 151 の REST エンドポイントをカバーし、AI アシスタントに x64dbg デバッガの完全な制御を提供する MCP サーバーです。
ステップ実行、ブレークポイント、メモリ操作、逆アセンブル、トレーシング、アンチデバッグ回避、制御フロー解析、PE ダンプに対応し、Claude、Cursor、Windsurf をはじめとする MCP クライアントと互換性があります。
TypeScript サーバーは Zod の判別共用体により型安全なエンドポイント対応を行い、ネイティブ x64dbg プラグイン（.dp64/.dp32）がデバッガと REST API を橋渡しします。
主に、x64dbg を通じた AI 支援デバッグワークフローと自動バイナリ解析を求めるリバースエンジニア向けです。
