Este proyecto indica que el fundador de Urho3D (Lasse Öörni) está trabajando actualmente en Turso3D. Urho3D es un motor de juegos 2D y 3D gratuito, ligero y multiplataforma. Es principalmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el área de motores de juegos / código fuente.
