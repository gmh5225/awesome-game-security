Ce projet indique que le fondateur d'Urho3D (Lasse Öörni) travaille actuellement sur Turso3D. Urho3D est un moteur de jeu 2D et 3D gratuit, léger et multiplateforme. Il est principalement utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
