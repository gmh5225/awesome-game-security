Este proyecto es una instantánea del SDK de Counter-Strike 2 solo con cabeceras que expone amplios conjuntos de declaraciones de clases y enumeraciones de Source 2. Proporciona definiciones C++ estructuradas para entidades, sistemas de gameplay, tipos de animación y muchos diseños de datos del motor con offsets de campos. El SDK también incluye tipos de utilidad del motor, como implementaciones de vectores de bits usadas por estructuras de datos de juego en red. Está pensado principalmente para ingeniería inversa, herramientas e investigación de seguridad en juegos que requieren información actualizada de tipos internos.
