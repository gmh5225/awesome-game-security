Ce projet est un instantané de SDK Counter-Strike 2 constitué uniquement d’en-têtes, exposant de vastes ensembles de déclarations de classes et d’énumérations Source 2. Il fournit des définitions C++ structurées pour les entités, les systèmes de gameplay, les types d’animation et de nombreuses dispositions de données moteur avec offsets de champs. Le SDK inclut également des types utilitaires du moteur, tels que des implémentations de vecteurs de bits utilisées par les structures de données de jeu réseau. Il est principalement destiné à la rétro-ingénierie, aux outils et à la recherche en sécurité des jeux nécessitant des informations de types internes à jour.
