Этот проект — snapshot header-only SDK для Counter-Strike 2, предоставляющий большие наборы объявлений классов и перечислений Source 2. Он даёт структурированные определения C++ для сущностей, игровых систем, типов анимации и множества раскладок данных движка с смещениями полей. SDK также включает служебные типы движка, например реализации битовых векторов, используемые сетевыми игровыми структурами данных. Он в первую очередь предназначен для реверс-инжиниринга, инструментов и исследований безопасности игр, которым нужна актуальная информация о внутренних типах.
