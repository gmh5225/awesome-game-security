Questo progetto è uno snapshot header-only dell’SDK di Counter-Strike 2 che espone ampi set di dichiarazioni di classi ed enumerazioni Source 2. Fornisce definizioni C++ strutturate per entità, sistemi di gameplay, tipi di animazione e numerosi layout di dati del motore con offset dei campi. L’SDK include anche tipi di utilità del motore, come implementazioni di bit vector usate dalle strutture dati di gioco in rete. È pensato principalmente per reverse engineering, tooling e ricerca di sicurezza sui giochi che richiedono informazioni aggiornate sui tipi interni.
