PowerHook es una prueba de concepto en modo kernel de Windows que engancha una devolución de llamada de administración de energía del procesador para interceptar la ejecución en contexto de kernel. Está implementado en C++ como un controlador KMDF y redirige la rutina IdlePreselect del PRCB preservando el controlador original para la limpieza. La rutina de hook demuestra búsquedas de objetos de hilo y proceso desde el espacio de kernel y registra detalles del contexto de ejecución. Este proyecto es útil principalmente para aprender los internos de Windows y para investigación de seguridad de juegos a bajo nivel.
