Этот проект — высокопроизводительная сетевая библиотека на C++11 для построения событийно-управляемых TCP-, UDP- и HTTP-сервисов.
Она предоставляет неблокирующие серверные и клиентские компоненты, потокобезопасные циклы событий, таймеры, асинхронный DNS и утилитарные слои, ориентированные на соединения.
Кодовая база построена вокруг libevent и включает ориентированные на продакшен инструменты, бенчмарки и связанные клиенты для систем вроде Memcached и NSQ.
Она в первую очередь предназначена для разработчиков бэкенда и инфраструктуры, которым нужны масштабируемые сетевые сервисы с низкой задержкой.
