Este proyecto es una biblioteca de red C++11 de alto rendimiento para construir servicios TCP, UDP y HTTP orientados a eventos.
Proporciona componentes de servidor y cliente no bloqueantes, bucles de eventos seguros para hilos, temporizadores, DNS asíncrono y capas de utilidades orientadas a la conexión.
El código se basa en libevent e incluye herramientas orientadas a producción, benchmarks y clientes relacionados para sistemas como Memcached y NSQ.
Está pensada principalmente para desarrolladores de backend e infraestructura que necesitan servicios de red escalables y de baja latencia.
