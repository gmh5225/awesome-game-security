Этот проект — курируемая база знаний по ресурсам сетевого взаимодействия многопользовательских игр. В основном это Markdown-подборка статей, докладов, библиотек и инструментов по темам netcode: компенсация задержки, rollback, предсказание и репликация. Материалы охватывают практические инженерные ссылки для разных движков, языков и архитектурных подходов. Основное назначение — помочь разработчикам и исследователям изучать компромиссы проектирования и реализации игрового networking.
