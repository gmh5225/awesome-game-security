blender-mcp conecta Blender con asistentes de IA mediante el Model Context Protocol para inspeccionar y editar escenas con comandos en lenguaje natural. El proyecto combina un addon de Blender con un servidor MCP en Python que se comunica mediante un protocolo JSON basado en sockets. Admite manipulación de objetos y materiales, consultas de escena, ejecución de código mediante scripts e integraciones opcionales para assets externos y modelos 3D generados. El caso de uso principal es la creación de contenido 3D asistida por IA y la creación rápida de prototipos de escenas para artistas y desarrolladores.
