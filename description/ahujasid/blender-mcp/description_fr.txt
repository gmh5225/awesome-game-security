blender-mcp connecte Blender aux assistants IA via le Model Context Protocol, afin d’inspecter et de modifier des scènes par commandes en langage naturel. Le projet combine un addon Blender et un serveur MCP Python qui communiquent via un protocole JSON basé sur des sockets. Il prend en charge la manipulation d’objets et de matériaux, les requêtes de scène, l’exécution de code scripté, ainsi que des intégrations optionnelles pour des assets externes et des modèles 3D générés. Le cas d’usage principal est la création de contenu 3D assistée par l’IA et le prototypage rapide de scènes pour les artistes et les développeurs.
