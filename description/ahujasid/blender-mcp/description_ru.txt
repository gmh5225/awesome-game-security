blender-mcp соединяет Blender с ИИ-ассистентами через Model Context Protocol, позволяя просматривать и редактировать сцены командами на естественном языке. Проект сочетает аддон Blender и Python MCP-сервер, которые обмениваются данными по сокетному JSON-протоколу. Поддерживаются манипуляции с объектами и материалами, запросы к сцене, выполнение скриптового кода, а также опциональные интеграции внешних ассетов и сгенерированных 3D-моделей. Основной сценарий использования — создание 3D-контента с помощью ИИ и быстрое прототипирование сцен для художников и разработчиков.
