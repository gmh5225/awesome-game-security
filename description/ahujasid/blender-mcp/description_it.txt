blender-mcp collega Blender agli assistenti IA tramite il Model Context Protocol, così da ispezionare e modificare le scene con comandi in linguaggio naturale. Il progetto combina un addon Blender con un server MCP in Python che comunica tramite un protocollo JSON basato su socket. Supporta la manipolazione di oggetti e materiali, le query sulla scena, l’esecuzione di codice tramite script e integrazioni opzionali per asset esterni e modelli 3D generati. Il caso d’uso principale è la creazione di contenuti 3D assistita dall’IA e la prototipazione rapida di scene per artisti e sviluppatori.
