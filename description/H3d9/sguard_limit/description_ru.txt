sguard_limit — это набор инструментов для Windows, предназначенный для ограничения или патчинга поведения в игровом клиенте, защищённом ACE-Guard. Репозиторий объединяет user-mode контроллер на C++ с kernel-mode модулем на C и ассемблере для патчинга памяти и манипуляции процессами. В него входят процедуры для операций с виртуальной памятью, обхода VAD, управления приостановкой и возобновлением, а также хуки на основе detour в решении Visual Studio. Проект ориентирован на реверс-инжиниринг античита и эксперименты с обходом, а не на разработку обычных приложений.
