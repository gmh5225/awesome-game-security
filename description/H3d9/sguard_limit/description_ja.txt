sguard_limit は、ACE-Guard で保護されたゲームクライアントの挙動を制限またはパッチするための Windows ツールキットです。このリポジトリは、ユーザーモードの C++ コントローラと、C およびアセンブリで書かれたカーネルモードモジュールを組み合わせ、メモリパッチやプロセス操作を行います。仮想メモリ操作、VAD 走査、サスペンド／レジューム制御、detour ベースのフックなどのルーチンを含み、Visual Studio ソリューションとして構成されています。本プロジェクトは一般的なアプリケーション開発ではなく、アンチチートのリバースエンジニアリングとバイパス実験を目的としています。
