Этот проект — UEVR, универсальный VR-инъекционный мод для Unreal Engine, добавляющий поддержку 6DOF VR практически в любую игру на Unreal Engine 4/5. Он перехватывает конвейер рендеринга движка, реализуя стереоскопический рендеринг, трекинг головы через OpenXR/OpenVR, ввод с motion-контроллеров и перемещение roomscale без правок под конкретную игру. C++-инжектор использует рефлексию UE и pattern scanning для поиска внутренних структур движка во время выполнения. Предназначен для энтузиастов VR и исследователей UE, добавляющих VR в плоскоэкранные игры на UE.
