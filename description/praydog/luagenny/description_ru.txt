Этот проект — репозиторий в категории cheat / sdk codegen, посвящённый luagenny.
Он в основном написан на C++ и C/C++ и сосредоточен на реверс-инжиниринге и генерации SDK.
Он особенно полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / sdk codegen.
