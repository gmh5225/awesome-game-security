Este proyecto es un repositorio clasificado bajo cheat / sdk codegen y se centra en luagenny.
Está escrito principalmente en C++ y C/C++ y se centra en la ingeniería inversa y la generación de SDK.
Resulta especialmente útil para investigadores de seguridad de juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheat / sdk codegen.
