Una raccolta di tecniche di process injection su Windows con esempi funzionanti e note di ricerca di supporto. Parte dai fondamenti dell’iniezione di shellcode locale e remota, poi tratta la risoluzione dinamica delle funzioni, il walking di PEB e export address table, syscall dirette e indirette, module stomping, injection tramite thread pool e fiber, PPID spoofing e call-stack spoofing. Gran parte del codice è scritta in C e C++ con helper in assembly per le syscall, oltre a utility Python per la ricerca sul module stomping e il profiling dei processi. Il progetto è pensato per reverse engineer e professionisti di offensive security che studiano il tradecraft di injection su Windows e l’evasione degli EDR, inclusi argomenti rilevanti per la game security e la ricerca anti-cheat.
