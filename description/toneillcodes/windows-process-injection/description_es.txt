Una colección de técnicas de inyección de procesos en Windows con ejemplos funcionales y notas de investigación de apoyo. Parte de los fundamentos de la inyección de shellcode local y remota, y luego cubre la resolución dinámica de funciones, el recorrido del PEB y de la tabla de direcciones de exportación, syscalls directos e indirectos, module stomping, inyección mediante thread pool y fibras, spoofing de PPID y spoofing de la pila de llamadas. La mayor parte del código está escrita en C y C++ con ayudas en ensamblador para syscalls, además de utilidades en Python para investigar module stomping y perfilar procesos. El proyecto está orientado a ingenieros inversos y profesionales de seguridad ofensiva que estudian el tradecraft de inyección en Windows y la evasión de EDR, incluidos temas relevantes para la seguridad de juegos y la investigación anti-trampas.
