Коллекция техник внедрения в процессы Windows с рабочими примерами и сопроводительными исследовательскими заметками. Начинается с основ локальной и удалённой инъекции shellcode, затем охватывает динамическое разрешение функций, обход PEB и таблицы адресов экспорта, прямые и косвенные syscalls, module stomping, инъекцию через пул потоков и fiber, спуфинг PPID и спуфинг стека вызовов. Большая часть кода написана на C и C++ с ассемблерными вспомогательными модулями для syscalls, а также есть утилиты на Python для исследований module stomping и профилирования процессов. Проект рассчитан на реверс-инженеров и специалистов по offensive security, изучающих tradecraft инъекций в Windows и обход EDR, включая темы, актуальные для игровой безопасности и исследований античитов.
