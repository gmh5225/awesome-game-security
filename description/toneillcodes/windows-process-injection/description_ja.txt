動作するサンプルと研究ノートを備えた、Windows プロセスインジェクション手法のコレクションです。ローカル／リモートのシェルコード注入の基礎から始まり、動的関数解決、PEB とエクスポートアドレステーブルの走査、直接／間接システムコール、モジュールストンピング、スレッドプールおよびファイバー注入、PPID スプーフィング、コールスタックスプーフィングまでを扱います。コードの多くは C／C++ で書かれ、システムコール用のアセンブリ補助が含まれ、モジュールストンピング研究とプロセスプロファイリング向けの Python ユーティリティもあります。Windows の注入技術と EDR 回避を学ぶリバースエンジニアや攻撃的セキュリティ実務者向けで、ゲームセキュリティやアンチチート研究に関連するトピックも含みます。
