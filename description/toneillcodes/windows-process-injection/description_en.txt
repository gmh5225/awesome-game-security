A collection of Windows process injection techniques with working examples and supporting notes for security research. It covers fundamentals such as local and remote shellcode injection with memory-protection toggling, then advances into dynamic function resolution, PEB and export-address-table walking, direct and indirect syscalls, module stomping, thread-pool injection, fiber injection, PPID spoofing, and call-stack spoofing. Implementations are primarily in C and C++ with assembly helpers for syscalls, plus Python tooling for module-stomping research and profiling. The material is aimed at reverse engineers and offensive security practitioners studying Windows injection tradecraft and EDR evasion, including topics relevant to game-security and anti-cheat research.
