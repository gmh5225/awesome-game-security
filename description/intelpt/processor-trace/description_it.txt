Questo progetto è la libreria di decodifica Intel Processor Trace (PT) (libipt), un'implementazione di riferimento open source per decodificare i pacchetti di traccia Intel PT. Ricostruisce tracce di esecuzione a livello di istruzione dai dati PT grezzi, gestendo timing, flusso di controllo e decodifica dei pacchetti su più formati di traccia. La libreria C fornisce API per la decodifica delle tracce a livello di pacchetto e di istruzione. È rivolta ad autori di debugger, sviluppatori di strumenti di prestazione e ricercatori di sicurezza che realizzano strumenti di tracing e coverage basati su Intel PT.
