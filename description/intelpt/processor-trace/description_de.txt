Dieses Projekt ist die Intel-Processor-Trace-(PT)-Decoder-Bibliothek (libipt), eine Open-Source-Referenzimplementierung zum Dekodieren von Intel-PT-Trace-Paketen. Sie rekonstruiert Ausführungs-Traces auf Instruktionsebene aus rohen PT-Daten und behandelt Timing, Kontrollfluss sowie Paketdekodierung über mehrere Trace-Formate hinweg. Die C-Bibliothek stellt APIs für die Trace-Dekodierung auf Paket- und Instruktionsebene bereit. Sie richtet sich an Debugger-Autoren, Entwickler von Performance-Tools und Sicherheitsforscher, die Tracing- und Coverage-Tools auf Basis von Intel PT erstellen.
