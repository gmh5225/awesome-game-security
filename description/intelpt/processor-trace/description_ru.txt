Этот проект — библиотека декодирования Intel Processor Trace (PT) (libipt), открытая эталонная реализация для декодирования пакетов трассировки Intel PT. Она восстанавливает трассы выполнения на уровне инструкций из сырых данных PT, обрабатывая тайминг, поток управления и декодирование пакетов для нескольких форматов трасс. Библиотека на C предоставляет API для декодирования трасс на уровне пакетов и инструкций. Она предназначена для авторов отладчиков, разработчиков инструментов производительности и исследователей безопасности, создающих средства трассировки и покрытия на базе Intel PT.
