Este proyecto es la biblioteca decodificadora de Intel Processor Trace (PT) (libipt), una implementación de referencia de código abierto para decodificar paquetes de traza Intel PT. Reconstruye trazas de ejecución a nivel de instrucción a partir de datos PT en bruto, gestionando temporización, flujo de control y decodificación de paquetes en múltiples formatos de traza. La biblioteca en C proporciona API para la decodificación de trazas a nivel de paquete y de instrucción. Está dirigida a autores de depuradores, desarrolladores de herramientas de rendimiento e investigadores de seguridad que construyen herramientas de trazado y cobertura basadas en Intel PT.
