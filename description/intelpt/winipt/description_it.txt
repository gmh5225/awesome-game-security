Questo progetto è WinIPT, una libreria e strumenti per controllare Intel Processor Trace (IPT) su Windows. Fornisce API per configurare e raccogliere tracce IPT da codice in user-mode e kernel-mode, supportando modalità di tracing per processo e per CPU. La libreria C avvolge le interfacce del driver IPT di Windows per la gestione e la configurazione dei buffer di traccia. È destinata a ricercatori di sicurezza e analisti delle prestazioni che usano Intel PT per code coverage, fuzzing e tracing dell'esecuzione su Windows.
