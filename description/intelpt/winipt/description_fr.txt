Ce projet est WinIPT, une bibliothèque et des outils pour contrôler Intel Processor Trace (IPT) sous Windows. Il fournit des API pour configurer et collecter des traces IPT depuis le code en mode utilisateur et en mode noyau, avec prise en charge des modes de traçage par processus et par CPU. La bibliothèque C encapsule les interfaces du pilote IPT Windows pour la gestion et la configuration des tampons de trace. Elle s'adresse aux chercheurs en sécurité et aux analystes de performance qui utilisent Intel PT pour la couverture de code, le fuzzing et le traçage d'exécution sous Windows.
