Dieses Projekt ist WinIPT, eine Bibliothek und Werkzeuge zur Steuerung von Intel Processor Trace (IPT) unter Windows. Es stellt APIs zum Konfigurieren und Erfassen von IPT-Traces aus User-Mode- und Kernel-Mode-Code bereit und unterstützt Trace-Modi pro Prozess und pro CPU. Die C-Bibliothek kapselt die Windows-IPT-Treiberschnittstellen für Trace-Pufferverwaltung und -konfiguration. Sie richtet sich an Sicherheitsforscher und Performance-Analysten, die Intel PT für Code Coverage, Fuzzing und Ausführungs-Tracing unter Windows nutzen.
