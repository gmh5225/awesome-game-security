Этот проект — WinIPT, библиотека и инструменты для управления Intel Processor Trace (IPT) в Windows. Он предоставляет API для настройки и сбора IPT-трасс из кода в user-mode и kernel-mode, поддерживая режимы трассировки по процессу и по CPU. Библиотека на C оборачивает интерфейсы драйвера Windows IPT для управления и настройки буферов трассировки. Она предназначена для исследователей безопасности и аналитиков производительности, использующих Intel PT для покрытия кода, фаззинга и трассировки выполнения в Windows.
