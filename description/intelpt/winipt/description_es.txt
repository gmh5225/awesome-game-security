Este proyecto es WinIPT, una biblioteca y herramientas para controlar Intel Processor Trace (IPT) en Windows. Proporciona APIs para configurar y recopilar trazas IPT desde código en modo usuario y modo kernel, con soporte para modos de trazado por proceso y por CPU. La biblioteca C encapsula las interfaces del controlador IPT de Windows para la gestión y configuración de búferes de traza. Está orientada a investigadores de seguridad y analistas de rendimiento que usan Intel PT para cobertura de código, fuzzing y trazado de ejecución en Windows.
