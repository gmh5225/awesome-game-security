本プロジェクトは WinIPT であり、Windows 上で Intel Processor Trace（IPT）を制御するためのライブラリとツール群です。ユーザーモードおよびカーネルモードのコードから IPT トレースを設定・収集する API を提供し、プロセス単位および CPU 単位のトレースモードをサポートします。この C ライブラリは、トレースバッファの管理と設定のために Windows IPT ドライバのインターフェースをラップします。Windows 上でコードカバレッジ、ファジング、実行トレースに Intel PT を活用するセキュリティ研究者やパフォーマンス分析者を対象としています。
