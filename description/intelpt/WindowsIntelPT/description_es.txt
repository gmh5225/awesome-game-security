Este proyecto es un controlador de Windows y una biblioteca en modo usuario para capturar datos de Intel Processor Trace (IPT) en sistemas Windows. Proporciona un controlador de kernel que configura los MSR de IPT y gestiona los búferes de rastreo, junto con una API en modo usuario para iniciar/detener rastros y procesar los datos capturados. La herramienta admite modos de rastreo por proceso y a nivel de sistema. Está dirigida a investigadores de seguridad y desarrolladores de herramientas que usan Intel PT para el rastreo binario en Windows, fuzzing guiado por cobertura y análisis de ejecución.
