Ce projet est un pilote Windows et une bibliothèque en mode utilisateur pour capturer les données Intel Processor Trace (IPT) sur les systèmes Windows. Il fournit un pilote noyau qui configure les MSR IPT et gère les tampons de trace, ainsi qu'une API en mode utilisateur pour démarrer/arrêter les traces et traiter les données capturées. L'outil prend en charge les modes de traçage par processus et à l'échelle du système. Il s'adresse aux chercheurs en sécurité et aux développeurs d'outils utilisant Intel PT pour le traçage binaire Windows, le fuzzing guidé par couverture et l'analyse d'exécution.
