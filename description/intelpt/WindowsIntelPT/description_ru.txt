Этот проект — драйвер Windows и библиотека пользовательского режима для сбора данных Intel Processor Trace (IPT) в системах Windows. Он предоставляет драйвер ядра, который настраивает MSR IPT и управляет буферами трассировки, а также API пользовательского режима для запуска/остановки трассировок и обработки собранных данных. Инструмент поддерживает режимы трассировки как для отдельного процесса, так и в масштабе всей системы. Он предназначен для исследователей безопасности и разработчиков инструментов, использующих Intel PT для трассировки бинарных файлов Windows, фаззинга с покрытием и анализа выполнения.
