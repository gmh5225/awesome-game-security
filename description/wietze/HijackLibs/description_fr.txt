Ce projet est HijackLibs, une collection organisée d'opportunités de détournement de DLL dans les applications Windows. Il fournit une base de données consultable d'exécutables et de leurs dépendances DLL exploitables, incluant les noms de DLL spécifiques, les exports requis et les conditions d'exploitation pour chaque entrée. Le dépôt contient des données de vulnérabilité au format YAML et une interface web. Il s'adresse aux testeurs d'intrusion et aux opérateurs red team à la recherche d'opportunités de persistance et d'élévation de privilèges via le détournement de DLL.
