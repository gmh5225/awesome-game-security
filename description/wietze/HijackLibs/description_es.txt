Este proyecto es HijackLibs, una colección curada de oportunidades de secuestro de DLL en aplicaciones de Windows. Ofrece una base de datos consultable de ejecutables y sus dependencias DLL explotables, incluyendo los nombres específicos de DLL, las exportaciones requeridas y las condiciones de explotación de cada entrada. El repositorio incluye datos de vulnerabilidades en formato YAML y una interfaz web. Está dirigido a testers de penetración y operadores de red team que buscan oportunidades de persistencia y escalada de privilegios mediante secuestro de DLL.
