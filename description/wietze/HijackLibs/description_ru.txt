Этот проект — HijackLibs, курируемая коллекция возможностей DLL hijacking в приложениях Windows. Он предоставляет поисковую базу исполняемых файлов и их эксплуатируемых DLL-зависимостей, включая конкретные имена DLL, необходимые экспорты и условия эксплуатации для каждой записи. Репозиторий содержит данные об уязвимостях в формате YAML и веб-интерфейс. Он предназначен для пентестеров и операторов red team, ищущих возможности персистентности и повышения привилегий через DLL hijacking.
