Questo progetto è HijackLibs, una raccolta curata di opportunità di hijacking di DLL nelle applicazioni Windows. Fornisce un database ricercabile di eseguibili e delle loro dipendenze DLL sfruttabili, inclusi i nomi specifici delle DLL, gli export richiesti e le condizioni di sfruttamento per ciascuna voce. Il repository include dati sulle vulnerabilità in formato YAML e un'interfaccia web. È rivolto a penetration tester e operatori red team alla ricerca di opportunità di persistenza e privilege escalation tramite hijacking di DLL.
