Этот проект — комплексная база данных и исследование уязвимостей DLL hijacking в Windows. В нём каталогизированы сотни исполняемых файлов и уязвимых путей поиска DLL, а также задокументировано, какие системные двоичные файлы можно использовать для DLL sideloading, подмены порядка поиска и загрузки phantom DLL. Репозиторий включает инструменты автоматического сканирования и подробные данные об уязвимостях. Он предназначен для операторов red team, аудиторов безопасности и защитников, которые картируют поверхность атаки DLL hijacking в Windows.
