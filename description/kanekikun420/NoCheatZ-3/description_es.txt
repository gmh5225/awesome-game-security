Este proyecto se centra en un plugin anticheat del lado del servidor para Source Engine.
Está escrito principalmente en C/C++ y C++ y se centra en la investigación anticheat, el desarrollo de plugins y la generación de SDK.
Resulta especialmente útil para ingenieros anticheat e investigadores de seguridad defensiva que trabajan en el área de anticheat / protección de motores de juego: Source.
