Questo progetto si concentra su un plugin anti-cheat lato server per Source Engine.
È scritto principalmente in C/C++ e C++ e ruota intorno alla ricerca anti-cheat, allo sviluppo di plugin e alla generazione di SDK.
È soprattutto utile per ingegneri anti-cheat e ricercatori di sicurezza difensiva che operano nell'ambito anti-cheat / protezione dei motori di gioco: Source.
