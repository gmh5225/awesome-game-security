Этот проект — инструмент на Python, который снимает обфускацию выравнивания потока управления (control flow flattening), применённую OLLVM (Obfuscator-LLVM), с помощью фреймворка символьного выполнения Miasm.
Он восстанавливает исходный поток управления, выявляя и соединяя базовые блоки, поддерживает многоуровневую деобфускацию через отслеживание вызовов на основе BFS и генерирует деобфусцированные бинарники для Windows и Linux на архитектурах x86 и x64.
В отличие от статических подходов, инструмент использует движок символьного выполнения Miasm для динамического восстановления исходного потока программы из плоской структуры диспетчеризации по переменной состояния.
Он в первую очередь полезен реверс-инженерам и исследователям деобфускации, изучающим методы восстановления control flow flattening в OLLVM.
