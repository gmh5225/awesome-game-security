本プロジェクトは、Miasm のシンボリック実行フレームワークを用いて、OLLVM（Obfuscator-LLVM）が適用した制御フロー平坦化を解除する Python ツールです。
基本ブロックを識別して接続することで元の制御フローを再構築し、BFS ベースのコール追跡による多層デオブファケーションに対応し、Windows と Linux、x86 および x64 向けのデオブファケート済みバイナリを生成します。
静的手法とは異なり、Miasm のシンボリック実行エンジンを使い、平坦化された状態変数ディスパッチ構造から元のプログラムフローを動的に復元します。
主にリバースエンジニアや、OLLVM 制御フロー平坦化の復元手法を研究するデオブファケーション研究者向けです。
