Este proyecto es una herramienta en Python que desofusca el aplanamiento del flujo de control aplicado por OLLVM (Obfuscator-LLVM) mediante el framework de ejecución simbólica Miasm.
Reconstruye el flujo de control original identificando y conectando bloques básicos, admite desofuscación multinivel mediante seguimiento de llamadas basado en BFS y genera binarios desofuscados para Windows y Linux en arquitecturas x86 y x64.
A diferencia de los enfoques estáticos, la herramienta usa el motor de ejecución simbólica de Miasm para recuperar dinámicamente el flujo original del programa a partir de la estructura de despacho basada en una variable de estado aplanada.
Es especialmente útil para ingenieros inversos e investigadores de desofuscación que estudian técnicas de recuperación del aplanamiento del flujo de control de OLLVM.
