이 프로젝트는 Miasm 심볼릭 실행 프레임워크를 사용해 OLLVM(Obfuscator-LLVM)이 적용한 제어 흐름 평탄화를 해제하는 Python 도구입니다.
기본 블록을 식별하고 연결하여 원래 제어 흐름을 재구성하며, BFS 기반 호출 추적을 통한 다층 디오브퍼스케이션을 지원하고, Windows와 Linux의 x86 및 x64 아키텍처용 디오브퍼스케이션된 바이너리를 생성합니다.
정적 접근 방식과 달리, Miasm의 심볼릭 실행 엔진을 사용해 평탄화된 상태 변수 디스패치 구조에서 원래 프로그램 흐름을 동적으로 복구합니다.
주로 리버스 엔지니어와 OLLVM 제어 흐름 평탄화 복구 기법을 연구하는 디오브퍼스케이션 연구자에게 유용합니다.
