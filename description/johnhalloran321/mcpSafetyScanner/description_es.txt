Este proyecto se centra en mCPSafetyScanner: auditoría de seguridad MCP automatizada y remediación mediante Agents.
Ejemplo de uso: en examples del repositorio se incluye una configuración MCP de ejemplo.
Resulta especialmente útil para desarrolladores de juegos, ingenieros inversos y creadores de herramientas que trabajan en desarrollo de juegos / seguridad de servidores MCP.
