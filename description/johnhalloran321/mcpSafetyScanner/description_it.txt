Questo progetto si concentra su mCPSafetyScanner: audit di sicurezza MCP automatizzato e remediation tramite Agents.
Esempio d'uso: nel repository, in examples, è fornita una configurazione MCP di esempio.
È soprattutto utile per sviluppatori di giochi, reverse engineer e chi costruisce tooling nell'ambito dello sviluppo di giochi / della sicurezza dei server MCP.
