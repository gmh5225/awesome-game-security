Ce projet se concentre sur mCPSafetyScanner — l'audit de sécurité MCP automatisé et la remédiation à l'aide d'Agents.
Exemple d'utilisation : une configuration MCP d'exemple est fournie dans le dossier examples du dépôt.
Il est surtout utile aux développeurs de jeux, aux reverse engineers et aux créateurs d'outils travaillant dans le domaine du développement de jeux / de la sécurité des serveurs MCP.
