Этот проект посвящён mCPSafetyScanner — автоматизированному аудиту безопасности MCP и устранению проблем с помощью Agents.
Пример использования: в каталоге examples репозитория есть пример конфигурации MCP.
Он в первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / безопасности MCP-серверов.
