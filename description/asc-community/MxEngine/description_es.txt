Este proyecto es un motor de juegos 3D moderno en C++, pensado para la educación y el desarrollo en tiempo real de propósito general. Ofrece una amplia pila de renderizado basada en OpenGL con PBR, efectos en espacio de pantalla, sombras, sistemas de partículas y un flujo de trabajo de editor con iteración de shaders y código en tiempo de ejecución. El motor también integra una arquitectura de estilo ECS, física, audio, pipelines de importación de assets y herramientas de serialización para la creación completa de escenas. Está dirigido a programadores de motores y desarrolladores de juegos que buscan un framework con código fuente disponible para estudiar o crear aplicaciones 3D de escritorio.
