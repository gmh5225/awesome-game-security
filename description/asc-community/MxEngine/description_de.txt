Dieses Projekt ist eine moderne C++-3D-Spiel-Engine für Bildung und allgemeine Echtzeitentwicklung. Sie bietet einen umfangreichen OpenGL-basierten Rendering-Stack mit PBR, Screen-Space-Effekten, Schatten, Partikelsystemen sowie einem Editor-Workflow mit Laufzeit-Iteration von Shadern und Code. Die Engine integriert außerdem eine ECS-artige Architektur, Physik, Audio, Asset-Import-Pipelines und Serialisierungswerkzeuge für die vollständige Szenenerstellung. Sie richtet sich an Engine-Programmierer und Spieleentwickler, die ein Framework mit verfügbarem Quellcode zum Studium oder zur Erstellung von Desktop-3D-Anwendungen suchen.
