Libreria C++14 per analizzare Linux /proc/pid/maps in oggetti MemorySegment strutturati che espongono intervalli di indirizzi, flag di permesso, offset dei file e percorsi dei file di backing, fornendo un'interfaccia programmatica per l'analisi del layout di memoria dei processi in esecuzione, utile in forensics della memoria e negli strumenti di sicurezza dei giochi.
