C++14-Bibliothek zum Einlesen von Linux /proc/pid/maps in strukturierte MemorySegment-Objekte mit Adressbereichen, Berechtigungsflags, Dateioffsets und Pfaden der hinterlegten Dateien; stellt eine programmatische Schnittstelle zur Analyse des Speicherlayouts laufender Prozesse bereit und eignet sich für Memory Forensics sowie Game-Security-Werkzeuge.
