Библиотека на C++14 для разбора Linux /proc/pid/maps в структурированные объекты MemorySegment с диапазонами адресов, флагами прав доступа, смещениями файлов и путями к файлам-источникам; предоставляет программный интерфейс для анализа раскладки памяти работающих процессов, полезный в memory forensics и инструментах игровой безопасности.
