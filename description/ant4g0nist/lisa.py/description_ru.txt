Этот проект — интеграция LLDB, которая предоставляет возможности отладчика через сервер Model Context Protocol.
Он объединяет компоненты на Python для плагина LLDB и моста MCP, чтобы ИИ-ассистенты могли управлять отладкой через структурированные вызовы инструментов.
Кодовая база включает команды для создания целей, управления процессами, точек останова, backtrace, дизассемблирования, чтения памяти и вычисления выражений.
Проект предназначен для рабочих процессов реверс-инжиниринга и исследования уязвимостей, где интерактивную отладку нужно автоматизировать или вести с помощью ассистента.
