Este proyecto es una integración de LLDB que expone las capacidades del depurador a través de un servidor Model Context Protocol.
Combina componentes Python para un complemento LLDB y un puente MCP, de modo que los asistentes de IA puedan dirigir la depuración mediante llamadas estructuradas a herramientas.
El código incluye comandos para creación de destinos, control de procesos, puntos de interrupción, backtraces, desensamblado, lecturas de memoria y evaluación de expresiones.
Está pensado para flujos de trabajo de ingeniería inversa e investigación de vulnerabilidades en los que la depuración interactiva deba automatizarse o asistirse.
