Ce projet est une intégration LLDB qui expose les capacités du débogueur via un serveur Model Context Protocol.
Il combine des composants Python pour un plugin LLDB et un pont MCP afin que les assistants IA puissent piloter le débogage via des appels d'outils structurés.
Le code inclut des commandes pour la création de cibles, le contrôle de processus, les points d'arrêt, les backtraces, le désassemblage, les lectures mémoire et l'évaluation d'expressions.
Il est destiné aux flux de travail de rétro-ingénierie et de recherche de vulnérabilités où le débogage interactif doit être automatisé ou assisté.
