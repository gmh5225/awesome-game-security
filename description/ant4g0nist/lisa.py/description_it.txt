Questo progetto è un'integrazione LLDB che espone le funzionalità del debugger tramite un server Model Context Protocol.
Combina componenti Python per un plugin LLDB e un bridge MCP, così che gli assistenti IA possano guidare il debugging tramite chiamate strutturate agli strumenti.
Il codice include comandi per la creazione di target, il controllo dei processi, i breakpoint, i backtrace, il disassemblaggio, le letture di memoria e la valutazione di espressioni.
È pensato per flussi di lavoro di reverse engineering e ricerca di vulnerabilità in cui il debugging interattivo deve essere automatizzato o assistito.
