KDBG — это набор инструментов для отладки ядра Windows, сочетающий backend в виде драйвера ядра и клиент командной строки. Он обеспечивает низкоуровневые операции чтения/записи памяти для пользовательских процессов и модулей ядра, а также перечисление модулей/потоков и функции, ориентированные на трассировку; дополнительные возможности отладчика находятся в разработке. Код ориентирован на рабочие процессы x64 и требует этапов загрузки драйвера, которые могут временно изменить поведение подписи драйверов. Предназначен для опытных исследователей, занимающихся низкоуровневой отладкой игр, анализом античита и экспериментами по безопасности ядра.
