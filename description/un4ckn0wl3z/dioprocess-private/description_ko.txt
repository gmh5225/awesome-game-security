이 프로젝트는 DioProcess로, 회피, 루트킷, 하이퍼바이저, SMM, 부트킷을 다루는 고급 Windows 내부 연구 프레임워크입니다.
Rust 2021과 Dioxus 기반 데스크톱 앱으로 프로세스, 핸들, 모듈, 네트워크를 실시간 모니터링하며, DSE 또는 KPP 우회 연구를 위한 커널 드라이버와 UEFI 부트킷 경로를 함께 제공합니다.
핵심 기능은 SQLite 영속화가 포함된 실시간 커널 이벤트 캡처와, Windows 10 22H2에서 프로세스 보호 및 권한 상승 실험을 위한 보안 연구용 IOCTL입니다.
주로 치트 / 디버깅 영역의 공격 기법을 연구하는 게임 보안 연구원과 리버스 엔지니어에게 유용합니다.
