Este proyecto es DioProcess, un framework avanzado de investigación de internals de Windows que cubre evasión, rootkits, hipervisores, SMM y bootkits.
Es una aplicación de escritorio en Rust 2021 con Dioxus para monitorización en tiempo real de procesos, handles, módulos y red, con un controlador de kernel acompañante y una ruta de bootkit UEFI para investigación de bypass de DSE o KPP.
Las funciones principales incluyen captura en vivo de eventos del kernel con persistencia SQLite e IOCTL de investigación de seguridad para experimentos de protección de procesos y elevación de privilegios en Windows 10 22H2.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheat / debugging.
