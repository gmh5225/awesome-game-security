本プロジェクトは DioProcess で、回避、Rootkit、Hypervisor、SMM、Bootkit を扱う高度な Windows 内部研究フレームワークです。
Rust 2021 と Dioxus によるデスクトップアプリで、プロセス、ハンドル、モジュール、ネットワークをリアルタイム監視し、DSE または KPP バイパス研究向けのカーネルドライバと UEFI Bootkit パスを伴います。
主な機能は SQLite 永続化付きのライブカーネルイベント取得と、Windows 10 22H2 上でのプロセス保護・特権昇格実験向けセキュリティ研究用 IOCTL です。
主にゲームセキュリティ研究者やリバースエンジニアが、チート / デバッグ領域の攻撃的手法を学ぶ用途に適しています。
