Questo progetto è DioProcess, un framework avanzato di ricerca sugli internals di Windows che copre evasion, rootkit, hypervisor, SMM e bootkit.
È un'app desktop in Rust 2021 con Dioxus per il monitoraggio in tempo reale di processi, handle, moduli e rete, con un driver kernel companion e un percorso bootkit UEFI per la ricerca sul bypass di DSE o KPP.
Le funzionalità principali includono la cattura live di eventi kernel con persistenza SQLite e IOCTL per la ricerca di sicurezza, utili per esperimenti di protezione dei processi e privilege escalation su Windows 10 22H2.
È soprattutto utile per ricercatori di game security e reverse engineer che studiano tecniche offensive nell'area cheat / debugging.
