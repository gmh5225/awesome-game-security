This project is DioProcess, an advanced Windows internals research framework covering evasion, rootkits, hypervisors, SMM, and bootkits.
It is a Rust 2021 plus Dioxus desktop app for real-time process, handle, module, and network monitoring, with a companion kernel driver and UEFI bootkit path for DSE or KPP bypass research.
Core features include live kernel event capture with SQLite persistence and security-research IOCTLs for process protection and privilege-escalation experiments on Windows 10 22H2.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / debugging area.
