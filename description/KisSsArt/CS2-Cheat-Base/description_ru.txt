Этот проект — база читов Windows DirectX для Counter-Strike 2, собранная как внедряемая DLL.
Он предоставляет инфраструктуру хуков и рендеринга с настраиваемыми игровыми модулями, такими как ESP и bunny hop.
Реализация в основном на C++ и интегрирует распространённые инструменты вроде ImGui, kiero и конфигурации на основе JSON.
Описанный процесс сборки ориентирован на вывод Visual Studio x64 Release с внедрением в стиле manual-map.
Предназначен как стартовый фреймворк для разработки читов и практических исследований безопасности игр.
