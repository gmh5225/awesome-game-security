Dieses Projekt ist eine Windows-DirectX-Cheat-Basis für Counter-Strike 2, die als injizierbare DLL umgesetzt ist.
Es stellt Hook- und Rendering-Infrastruktur mit konfigurierbaren Gameplay-Modulen wie ESP und Bunny Hop bereit.
Die Implementierung ist hauptsächlich in C++ und integriert gängige Tools wie ImGui, kiero sowie eine JSON-basierte Konfiguration.
Der dokumentierte Build-Prozess zielt auf Visual Studio x64 Release-Ausgabe mit Injection im Manual-Map-Stil ab.
Es ist als Starter-Framework für Cheat-Entwicklung und praxisnahe Game-Security-Forschung gedacht.
