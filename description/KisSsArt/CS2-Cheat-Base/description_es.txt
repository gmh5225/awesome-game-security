Este proyecto es una base de cheat de Windows DirectX para Counter-Strike 2, construida como una DLL inyectable.
Proporciona infraestructura de hooks y renderizado con módulos de jugabilidad configurables como ESP y bunny hop.
La implementación es principalmente en C++ e integra herramientas comunes como ImGui, kiero y configuración basada en JSON.
El proceso de compilación documentado apunta a una salida Visual Studio x64 Release con inyección de estilo manual-map.
Está pensado como un framework inicial para el desarrollo de cheats y la investigación práctica de seguridad en juegos.
