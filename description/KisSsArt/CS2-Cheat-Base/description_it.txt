Questo progetto è una base cheat Windows DirectX per Counter-Strike 2, realizzata come DLL iniettabile.
Fornisce un'infrastruttura di hook e rendering con moduli di gameplay configurabili come ESP e bunny hop.
L'implementazione è principalmente in C++ e integra strumenti comuni come ImGui, kiero e configurazione basata su JSON.
Il processo di build documentato mira a un output Visual Studio x64 Release con iniezione in stile manual-map.
È pensato come framework di partenza per lo sviluppo di cheat e la ricerca pratica sulla sicurezza dei giochi.
