Ce projet est une base de cheat Windows DirectX pour Counter-Strike 2, conçue comme une DLL injectable.
Il fournit une infrastructure de hooks et de rendu avec des modules de gameplay configurables tels que l'ESP et le bunny hop.
L'implémentation est principalement en C++ et intègre des outils courants comme ImGui, kiero et une configuration basée sur JSON.
Le processus de compilation documenté cible une sortie Visual Studio x64 Release avec une injection de type manual-map.
Il est destiné à servir de framework de démarrage pour le développement de cheats et la recherche pratique en sécurité des jeux.
