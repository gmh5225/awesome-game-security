このプロジェクトは、注入可能な DLL として構築された Counter-Strike 2 向けの Windows DirectX チート基盤です。
ESP やバニーホップなどの設定可能なゲームプレイモジュールとともに、フックと描画のインフラを提供します。
実装は主に C++ で、ImGui、kiero、JSON ベースの設定など一般的なツールを統合しています。
文書化されたビルド手順は Visual Studio の x64 リリース出力を対象とし、manual-map 方式の注入を想定しています。
チート開発および実践的なゲームセキュリティ研究のためのスターターフレームワークとして設計されています。
