Этот проект предоставляет ориентированные на Unity примеры и вспомогательные библиотеки для автоматизации тестирования мобильных игр. Он демонстрирует сочетание Appium и инструментов для устройств с игровыми логами инструментации для проверки поведения без опоры только на хрупкое сопоставление изображений. Репозиторий включает образцы Unity-ассетов, скрипты и тестовые рабочие процессы для пайплайнов в стиле Android и iOS. Он предназначен для игровых QA-инженеров и разработчиков, выстраивающих практики автоматизированного тестирования для проектов Unity.
