Este proyecto ofrece ejemplos y bibliotecas auxiliares centrados en Unity para la automatización de pruebas de juegos móviles. Demuestra cómo combinar Appium y herramientas de dispositivo con registros de instrumentación en el juego para validar el comportamiento sin depender solo de una coincidencia de imágenes frágil. El repositorio incluye assets de Unity de ejemplo, scripts y flujos de trabajo del lado de las pruebas para pipelines al estilo Android e iOS. Está dirigido a ingenieros de QA de juegos y desarrolladores que construyen prácticas de pruebas automatizadas para proyectos Unity.
