Questo progetto è unKover, uno strumento del kernel Windows per rilevare thread in modalità kernel nascosti e artefatti di rootkit. Enumera i thread di sistema con più metodi (liste dello scheduler, PspCidTable, scansione dello stack) e confronta i risultati per identificare i thread nascosti all'enumerazione API standard. Il driver in C rileva le tecniche di occultamento dei thread usate da anti-cheat e rootkit. È rivolto a sviluppatori di anti-cheat e ricercatori di forensics del kernel che individuano thread di sistema stealth.
