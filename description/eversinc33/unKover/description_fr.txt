Ce projet est unKover, un outil noyau Windows destiné à détecter les threads en mode noyau dissimulés et les artefacts de rootkit. Il énumère les threads système par plusieurs méthodes (listes du scheduler, PspCidTable, analyse de pile) et croise les résultats pour identifier les threads masqués à l'énumération API standard. Le pilote en C détecte les techniques de dissimulation de threads utilisées par les anti-cheats et les rootkits. Il s'adresse aux développeurs d'anti-cheat et aux chercheurs en forensics noyau qui traquent les threads système furtifs.
