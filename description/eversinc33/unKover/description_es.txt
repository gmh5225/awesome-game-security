Este proyecto es unKover, una herramienta de kernel de Windows para detectar hilos en modo kernel ocultos y artefactos de rootkit. Enumera los hilos del sistema mediante varios métodos (listas del planificador, PspCidTable, escaneo de pila) y cruza los resultados para identificar hilos ocultos a la enumeración API estándar. El controlador en C detecta técnicas de ocultación de hilos usadas por anti-cheats y rootkits. Está dirigido a desarrolladores de anti-cheat e investigadores de forense de kernel que detectan hilos de sistema furtivos.
