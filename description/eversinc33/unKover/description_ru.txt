Этот проект — unKover, инструмент ядра Windows для обнаружения скрытых потоков в режиме ядра и артефактов руткитов. Он перечисляет системные потоки несколькими методами (списки планировщика, PspCidTable, сканирование стека) и сопоставляет результаты, чтобы выявить потоки, скрытые от стандартного API-перечисления. Драйвер на C обнаруживает техники сокрытия потоков античитов и руткитов. Предназначен для разработчиков античитов и исследователей ядерной форензики, выявляющих скрытые системные потоки.
