Este proyecto es drvtrace, una herramienta de trazado de controladores del kernel de Windows que registra el tráfico IRP (I/O Request Packet) hacia y desde controladores específicos. Se adjunta como controlador filtro para interceptar todas las solicitudes IRP, registrando los códigos de función mayor/menor de IRP, el contenido de los buffers y el estado de finalización. El controlador en C ofrece visibilidad en tiempo de ejecución de los patrones de comunicación entre controladores. Está orientado a ingenieros inversos e investigadores de seguridad que analizan interfaces IOCTL de controladores y protocolos de comunicación de dispositivos.
