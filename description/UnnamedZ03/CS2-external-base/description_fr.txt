Ce projet est un framework externe de base pour les fonctionnalités de visualisation en overlay de Counter-Strike 2. Il inclut des exemples tels que la vérification d'équipe et plusieurs sorties ESP pour boîtes, santé, distance, squelette et affichage de texte. Le code est écrit en C++ et se concentre sur la structure côté overlay, en omettant volontairement les composants de pilote de lecture/écriture mémoire. Il est conçu comme une base pédagogique de démarrage pour étudier l'architecture des outils externes dans le contexte de la sécurité des jeux.
