Este proyecto es un framework externo básico para funciones de visualización en overlay de Counter-Strike 2. Incluye ejemplos como comprobaciones de equipo y múltiples salidas ESP para cajas, salud, distancia, esqueleto y visualización de texto. El código está escrito en C++ y se centra en la estructura del lado del overlay, omitiendo intencionadamente los componentes de controlador de lectura/escritura de memoria. Se presenta como una base educativa de partida para estudiar la arquitectura de herramientas externas en contextos de seguridad en juegos.
