Questo progetto è un framework esterno di base per le funzionalità di visualizzazione overlay di Counter-Strike 2. Include esempi come i controlli di squadra e diverse uscite ESP per box, salute, distanza, scheletro e visualizzazione del testo. Il codice è scritto in C++ e si concentra sulla struttura lato overlay, omettendo intenzionalmente i componenti del driver di lettura/scrittura della memoria. È pensato come base educativa introduttiva per studiare l'architettura degli strumenti esterni nei contesti di sicurezza dei giochi.
