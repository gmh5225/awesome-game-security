Este proyecto es FlatRedBall, un motor de juegos 2D maduro en C# / .NET con un editor, bibliotecas de tiempo de ejecución y complementos para carga de contenido, scripting y depuración.
Está orientado al desarrollo productivo de juegos 2D en .NET moderno, con mantenimiento frecuente y un flujo de trabajo completo impulsado por el editor.
Resulta especialmente útil para desarrolladores de juegos 2D y para investigadores que examinan una base de código completa de un motor de juegos administrado.
