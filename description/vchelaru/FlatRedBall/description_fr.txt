Ce projet est FlatRedBall, un moteur de jeux 2D C# / .NET mature, avec un éditeur, des bibliothèques d'exécution et des modules complémentaires pour le chargement de contenu, le scripting et le débogage.
Il vise un développement 2D productif sur .NET moderne, avec une maintenance régulière et un flux de travail entièrement piloté par l'éditeur.
Il est surtout utile aux développeurs de jeux 2D et aux chercheurs qui étudient une base de code complète de moteur de jeu managé.
