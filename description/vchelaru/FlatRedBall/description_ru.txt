Этот проект — FlatRedBall, зрелый 2D-игровой движок на C# / .NET с редактором, библиотеками времени выполнения и дополнениями для загрузки контента, скриптинга и отладки.
Он ориентирован на продуктивную разработку 2D-игр на современном .NET с регулярной поддержкой и полноценным рабочим процессом на основе редактора.
В первую очередь полезен разработчикам 2D-игр и исследователям, изучающим полную кодовую базу управляемого игрового движка.
