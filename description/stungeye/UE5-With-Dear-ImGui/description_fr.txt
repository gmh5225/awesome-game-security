Ce projet se concentre sur un exemple de projet C++ Unreal Engine 5.0.1 intégrant Dear ImGui.
Il a été réalisé dans le cadre de ce tutoriel YouTube d'environ 25 minutes.
Il est surtout utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / plugins de moteurs de jeu : Unreal.
