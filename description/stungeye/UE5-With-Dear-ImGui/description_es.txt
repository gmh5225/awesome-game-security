Este proyecto se centra en un proyecto de ejemplo en C++ de Unreal Engine 5.0.1 que incorpora Dear ImGui.
Se creó como parte de este tutorial de YouTube de unos 25 minutos.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores en gráficos que trabajan en el área de motores de juego / plugins de motores de juego: Unreal.
