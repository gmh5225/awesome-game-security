BinjaLattice es una integración de Binary Ninja que conecta sesiones de análisis con servidores MCP externos a través de una interfaz HTTP autenticada.
Está implementada principalmente en Python y ofrece autenticación basada en tokens, TLS opcional y una API de estilo REST para la comunicación segura con herramientas.
El complemento puede exportar contexto de desensamblado y pseudocódigo, a la vez que permite ediciones controladas como renombrar funciones o añadir comentarios.
Está dirigido a ingenieros inversos que quieren que pipelines de IA o automatización interactúen de forma segura con bases de datos Binary Ninja en vivo.
