BinjaLattice — интеграция для Binary Ninja, которая связывает сеансы анализа с внешними MCP-серверами через аутентифицированный HTTP-интерфейс.
Она в основном реализована на Python и обеспечивает аутентификацию по токену, опциональный TLS и REST-подобный API для безопасного взаимодействия с инструментами.
Плагин может экспортировать контекст дизассемблирования и псевдокода, а также позволяет контролируемые правки — например, переименование функций или добавление комментариев.
Он предназначен для реверс-инженеров, которым нужно безопасно подключать ИИ или конвейеры автоматизации к живым базам Binary Ninja.
