BinjaLattice est une intégration Binary Ninja qui relie les sessions d'analyse à des serveurs MCP externes via une interface HTTP authentifiée.
Elle est principalement implémentée en Python et fournit une authentification par jeton, un TLS optionnel et une API de style REST pour une communication sécurisée avec les outils.
Le plugin peut exporter le contexte de désassemblage et de pseudocode, tout en permettant des modifications contrôlées telles que le renommage de fonctions ou l'ajout de commentaires.
Il s'adresse aux ingénieurs en rétro-ingénierie qui souhaitent que des pipelines d'IA ou d'automatisation interagissent en toute sécurité avec des bases Binary Ninja actives.
