BinjaLattice ist eine Binary-Ninja-Integration, die Analysesitzungen über eine authentifizierte HTTP-Schnittstelle mit externen MCP-Servern verbindet.
Sie ist hauptsächlich in Python implementiert und bietet tokenbasierte Authentifizierung, optionales TLS sowie eine REST-artige API für sichere Tool-Kommunikation.
Das Plugin kann Disassembly- und Pseudocode-Kontext exportieren und erlaubt zugleich kontrollierte Bearbeitungen wie das Umbenennen von Funktionen oder das Hinzufügen von Kommentaren.
Es richtet sich an Reverse Engineers, die KI- oder Automatisierungspipelines sicher mit lebenden Binary-Ninja-Datenbanken interagieren lassen möchten.
