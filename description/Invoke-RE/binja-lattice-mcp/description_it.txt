BinjaLattice è un'integrazione Binary Ninja che collega le sessioni di analisi a server MCP esterni tramite un'interfaccia HTTP autenticata.
È implementata principalmente in Python e fornisce autenticazione basata su token, TLS opzionale e un'API in stile REST per la comunicazione sicura con gli strumenti.
Il plugin può esportare il contesto di disassemblaggio e pseudocodice, consentendo al contempo modifiche controllate come la ridenominazione di funzioni o l'aggiunta di commenti.
È pensato per reverse engineer che vogliono far interagire in modo sicuro pipeline di IA o di automazione con database Binary Ninja attivi.
