Dieses Projekt hieß ursprünglich Process Hacker.
Nach dem Klonen des Repositories führen Sie build_init.cmd im Verzeichnis build aus; erneut ausführen ist nur nötig, wenn Tools oder Drittanbieterbibliotheken aktualisiert wurden.
Es ist vor allem nützlich für Spielsicherheitsforscher und Reverse Engineers, die offensive Techniken im Bereich Cheats / Windows-Kernel-Explorer untersuchen.
