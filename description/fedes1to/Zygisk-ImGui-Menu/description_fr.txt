Ce projet se concentre sur un menu ImGui utilisant Zygisk.
Ce dépôt utilise également cURL et ImGui (évidemment). Pensez à modifier module.gradle avant de publier ; le code principal se trouve dans hook.cpp.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / rendu / dessin.
