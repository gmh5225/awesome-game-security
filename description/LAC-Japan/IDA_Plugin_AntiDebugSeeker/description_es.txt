AntiDebugSeeker es un complemento de IDA Pro que detecta automáticamente lógica potencial antidepuración en binarios analizados.
Usa Python y PyQt5 para encontrar uso sospechoso de API de Windows y técnicas antidebug basadas en palabras clave a partir de archivos de reglas configurables.
La herramienta resalta coincidencias, anota direcciones, permite navegar rápidamente a las detecciones e incluye un editor de configuración dentro del IDE.
Está orientada a analistas de malware e investigadores de seguridad que necesitan un triaje más rápido de protecciones antidebug.
