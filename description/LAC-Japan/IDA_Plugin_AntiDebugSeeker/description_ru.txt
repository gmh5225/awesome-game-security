AntiDebugSeeker — плагин для IDA Pro, который автоматически обнаруживает потенциальную антиотладочную логику в анализируемых бинарных файлах.
Он использует Python и PyQt5 для поиска подозрительного использования Windows API и антиотладочных приёмов по ключевым словам на основе настраиваемых файлов правил.
Инструмент подсвечивает совпадения, аннотирует адреса, поддерживает быстрый переход к обнаружениям и включает редактор конфигурации внутри IDE.
Предназначен для аналитиков вредоносного ПО и исследователей безопасности, которым нужен более быстрый разбор антиотладочных защит.
