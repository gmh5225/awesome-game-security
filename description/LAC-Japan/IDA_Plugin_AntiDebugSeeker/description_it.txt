AntiDebugSeeker è un plugin per IDA Pro che rileva automaticamente la logica di anti-debugging potenziale nei binari analizzati.
Usa Python e PyQt5 per individuare l’uso sospetto di API Windows e tecniche anti-debug basate su parole chiave da file di regole configurabili.
Lo strumento evidenzia le corrispondenze, annota gli indirizzi, consente una navigazione rapida alle rilevazioni e include un editor di configurazione nell’IDE.
È pensato per analisti di malware e ricercatori di sicurezza che devono triagare più rapidamente le protezioni anti-debug.
