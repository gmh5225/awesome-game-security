AntiDebugSeeker est un plugin IDA Pro qui détecte automatiquement la logique anti-débogage potentielle dans les binaires analysés.
Il s’appuie sur Python et PyQt5 pour repérer l’usage suspect d’API Windows et des techniques anti-debug basées sur des mots-clés, à partir de fichiers de règles configurables.
L’outil met en évidence les correspondances, annote les adresses, permet une navigation rapide vers les détections et inclut un éditeur de configuration intégré à l’IDE.
Il s’adresse aux analystes de malwares et aux chercheurs en sécurité qui ont besoin d’un triage plus rapide des protections anti-debug.
