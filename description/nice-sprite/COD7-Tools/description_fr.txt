Ce projet met l'accent sur les dépendances : vcpkg est utilisé pour toutes les dépendances ; c'est excellent, donc je l'utilise dès que possible.
Notes : certaines fonctionnalités sont encore en cours, mais j'ai beaucoup de projets pour celui-ci et j'espère l'orienter pour que chacun puisse s'amuser avec différents jeux, mieux comprendre leur fonctionnement et étendre facilement les fonctionnalités existantes à ses propres jeux.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / game:cod7.
