Questo progetto è una raccolta di tecniche di esecuzione di shellcode implementate in più linguaggi.
Include esempi in C, C# e MATLAB, con indicazioni pratiche d’uso incorporate nei file sorgente.
Il repository presenta diversi percorsi di esecuzione su processo e memoria spesso discussi nella ricerca sull’evasione.
È rivolto soprattutto a professionisti della sicurezza offensiva e difensiva che vogliono studiare queste tecniche di esecuzione in un contesto controllato.
