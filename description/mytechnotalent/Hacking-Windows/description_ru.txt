Этот проект — репозиторий из раздела cheat / guide, посвящённый взлому Windows.
Он в основном написан на C и сосредоточен на реверс-инжиниринге и отладке.
Он особенно полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / guide.
