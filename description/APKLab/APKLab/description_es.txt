Este proyecto es una extensión de VS Code que convierte la ingeniería inversa de APK de Android en un flujo de trabajo IDE integrado. Está creada con TypeScript y encapsula herramientas como Apktool, JADX, utilidades de firma y ayudas de parcheo HTTPS detrás de comandos del editor. Admite decodificación, desensamblado, descompilación, reconstrucción, firma, instalación e inicialización de dependencias en las principales plataformas de escritorio. El público principal son investigadores de seguridad móvil, analistas de malware y testers de manipulación de aplicaciones.
