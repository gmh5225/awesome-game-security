Questo progetto è un’estensione VS Code che trasforma il reverse engineering di APK Android in un flusso di lavoro IDE integrato. È sviluppata in TypeScript e avvolge strumenti come Apktool, JADX, utility di firma e ausili per il patching HTTPS dietro comandi dell’editor. Supporta decode, disassemblaggio, decompilazione, rebuild, firma, installazione e bootstrap delle dipendenze sulle principali piattaforme desktop. Il pubblico principale comprende ricercatori di sicurezza mobile, analisti di malware e tester di manomissione delle app.
