Ce projet est un générateur et éditeur de configuration basé sur le Web pour les fichiers de paramètres PalWorld. Il est construit avec JavaScript, HTML et des assets front-end associés pour offrir une interface pilotée par formulaires afin de modifier les paramètres de gameplay. Les utilisateurs peuvent personnaliser des options comme la difficulté et les taux de progression, puis télécharger immédiatement le fichier de configuration généré. Il s’adresse principalement aux joueurs et aux opérateurs de serveurs qui souhaitent une personnalisation plus rapide et moins sujette aux erreurs.
