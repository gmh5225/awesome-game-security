Questo progetto è un generatore ed editor di configurazione basato sul Web per i file delle impostazioni di PalWorld. È realizzato con JavaScript, HTML e relative risorse front-end per offrire un’interfaccia basata su moduli con cui modificare i parametri di gameplay. Gli utenti possono personalizzare opzioni come difficoltà e tassi di progressione e scaricare subito il file di configurazione generato. È rivolto principalmente a giocatori e gestori di server che desiderano una personalizzazione più rapida e meno soggetta a errori.
