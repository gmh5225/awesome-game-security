Este proyecto es un controlador de Windows y una utilidad en modo usuario para modificar la apariencia y el comportamiento del BSOD.
Gestiona la carga del kernel mediante un flujo de trabajo de servicio, resuelve los offsets de kernel necesarios y parchea las rutinas relacionadas con bugcheck.
La base de código es principalmente C y C++, con una aplicación de control de escritorio y componentes de kernel de apoyo.
Está pensado para la investigación de mecanismos de fallo de Windows a bajo nivel y para demostrar técnicas de parcheo del kernel.
