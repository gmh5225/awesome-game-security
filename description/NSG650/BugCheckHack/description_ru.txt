Этот проект — драйвер Windows и пользовательская утилита для изменения внешнего вида и поведения BSOD.
Он управляет загрузкой ядра через сервисный рабочий процесс, определяет необходимые смещения ядра и патчит процедуры, связанные с bugcheck.
Кодовая база в основном на C и C++, с настольным приложением-контроллером и сопутствующими компонентами ядра.
Предназначен для исследования низкоуровневых механизмов сбоев Windows и демонстрации техник патчинга ядра.
