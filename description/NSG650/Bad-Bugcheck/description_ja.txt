本プロジェクトは、クラッシュ時のフレームバッファ経路を通じて Bad Apple のフレームを描画する、更新版のカーネル BSOD ビジュアルハックです。
レガシーな Bootvid VGA 出力に頼る代わりに、ディスプレイのフレームバッファをマップして書き込み、KeBugCheckEx をフックしてクラッシュ処理を横取りします。
実装の主体は C および C++ のカーネルコードで、stb_image による解析と直接メモリコピーによるフレーム描画を用います。
bugcheck のフック、ディスプレイ所有権、クラッシュ画面レンダリング動作に焦点を当てた Windows 内部構造の研究を対象としています。
