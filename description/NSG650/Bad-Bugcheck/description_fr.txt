Ce projet est un hack visuel BSOD noyau mis à jour qui affiche les images de Bad Apple via le chemin du framebuffer de crash.
Au lieu de s'appuyer sur la sortie VGA Bootvid héritée, il mappe et écrit dans le framebuffer d'affichage et accroche KeBugCheckEx pour intercepter le flux de crash.
L'implémentation est principalement du code noyau en C et C++ et utilise l'analyse stb_image avec des routines de copie mémoire directe pour le dessin des images.
Il cible la recherche sur les internaux Windows, centrée sur le hooking de bugcheck, la propriété de l'affichage et le comportement de rendu de l'écran de crash.
