Этот проект — драйвер ядра, который подавляет стандартную обработку BSOD в Windows путём патчинга KeBugCheckEx.
Он перезаписывает поведение точки входа при сбое, чтобы фатальные ошибки немедленно не запускали обычный путь bugcheck.
Реализация написана на C и C++ для режима ядра Windows и включает прямое патчинг кода и логику восстановления.
В основном это эксперимент по внутренностям Windows и перехвату в ядре, демонстрирующий риски обхода механизмов безопасности.
