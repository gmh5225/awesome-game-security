Questo progetto è un driver del kernel che sopprime la gestione standard dei BSOD di Windows patchando KeBugCheckEx.
Sovrascrive il comportamento di ingresso in caso di crash affinché gli errori fatali non attivino immediatamente il percorso bugcheck normale.
L'implementazione è scritta in C e C++ per la modalità kernel di Windows, con patching diretto del codice e logica di ripristino.
È principalmente un esperimento sugli internals di Windows e sul kernel hooking che mostra i rischi di aggirare i meccanismi di sicurezza.
