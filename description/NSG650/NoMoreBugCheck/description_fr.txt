Ce projet est un pilote noyau qui supprime la gestion standard des BSOD Windows en patchant KeBugCheckEx.
Il remplace le comportement d'entrée en cas de plantage afin que les erreurs fatales ne déclenchent pas immédiatement le chemin bugcheck habituel.
L'implémentation est écrite en C et C++ pour le mode noyau Windows, avec un patching de code direct et une logique de restauration.
Il s'agit principalement d'une expérience sur les internals Windows et le hooking noyau, illustrant les risques de contourner les mécanismes de sécurité.
