Este proyecto es un controlador del kernel que suprime el manejo estándar de BSOD de Windows parcheando KeBugCheckEx.
Sobrescribe el comportamiento de entrada ante fallos para que los errores fatales no activen de inmediato la ruta normal de bugcheck.
La implementación está escrita en C y C++ para el modo kernel de Windows, con parcheo directo de código y lógica de restauración.
Es principalmente un experimento sobre internals de Windows y hooking del kernel que demuestra los riesgos de eludir los mecanismos de seguridad.
