Este proyecto es un controlador del núcleo de Windows que arranca un emulador de Linux RISC-V dentro de una pantalla de bug check (BSOD). Utiliza un emulador RISC-V mini-rv32ima, BOOTVID para el renderizado del framebuffer durante la fase de bug check, y un árbol de dispositivos e imagen de arranque incrustados para ejecutar un sistema Linux mínimo. Resulta especialmente útil para investigadores del núcleo y entusiastas que exploran usos creativos del entorno de bug check de Windows y la emulación embebida desde el modo núcleo.
