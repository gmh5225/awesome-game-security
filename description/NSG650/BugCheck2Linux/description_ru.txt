Этот проект — драйвер ядра Windows, который запускает эмулятор RISC-V Linux внутри экрана bug check (BSOD). Он использует эмулятор RISC-V mini-rv32ima, BOOTVID для отрисовки framebuffer на этапе bug check, а также встроенные device tree и загрузочный образ для запуска минимальной системы Linux. В первую очередь полезен исследователям ядра и энтузиастам, изучающим творческое применение среды Windows bug check и встроенную эмуляцию из режима ядра.
