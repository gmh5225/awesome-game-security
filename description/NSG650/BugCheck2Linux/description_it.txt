Questo progetto è un driver del kernel Windows che avvia un emulatore Linux RISC-V all'interno di una schermata di bug check (BSOD). Usa un emulatore RISC-V mini-rv32ima, BOOTVID per il rendering del framebuffer durante la fase di bug check, e un device tree e un'immagine di boot incorporati per eseguire un sistema Linux minimale. È principalmente utile per ricercatori del kernel e appassionati che esplorano usi creativi dell'ambiente di bug check di Windows e l'emulazione embedded dal kernel mode.
