Ce projet est un pilote noyau Windows qui démarre un émulateur Linux RISC-V à l'intérieur d'un écran de bug check (BSOD). Il utilise un émulateur RISC-V mini-rv32ima, BOOTVID pour le rendu du framebuffer pendant la phase de bug check, ainsi qu'un arbre de périphériques et une image de démarrage intégrés pour exécuter un système Linux minimal. Il s'adresse principalement aux chercheurs noyau et aux passionnés qui explorent des usages créatifs de l'environnement de bug check Windows et de l'émulation embarquée depuis le mode noyau.
