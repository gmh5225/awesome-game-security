Este proyecto es una prueba de concepto no-BSOD basada en UEFI que parchea el kernel durante el arranque.
Mueve el parcheo desde un controlador Windows en tiempo de ejecución a un cargador en fase EFI y luego altera el comportamiento de gestión de fallos al inicio temprano.
El código combina componentes de kernel y firmware en C y C++, incluyendo búsqueda de exportaciones, búsqueda de patrones y ayudantes de sobrescritura de memoria de bajo nivel.
Está diseñado para investigación avanzada de seguridad de firmware a kernel sobre parcheo en arranque temprano e interceptación de bugchecks.
