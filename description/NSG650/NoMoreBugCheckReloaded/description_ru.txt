Этот проект — UEFI-основанное proof of concept без BSOD, которое патчит ядро во время загрузки.
Оно переносит патчинг из драйвера Windows времени выполнения в загрузчик на стадии EFI, а затем изменяет поведение обработки сбоев на раннем этапе запуска.
Код сочетает компоненты ядра и прошивки на C и C++, включая поиск экспортов, поиск по сигнатурам и вспомогательные средства низкоуровневой перезаписи памяти.
Он предназначен для продвинутых исследований безопасности на пути от прошивки к ядру, связанных с ранним патчингом при загрузке и перехватом bugcheck.
