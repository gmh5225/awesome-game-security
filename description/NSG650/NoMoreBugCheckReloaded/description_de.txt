Dieses Projekt ist ein UEFI-basierter no-BSOD-Proof-of-Concept, der den Kernel während des Bootvorgangs patcht.
Es verlagert das Patchen von einem Windows-Laufzeit-Treiber in einen EFI-Phasen-Loader und ändert anschließend früh beim Start das Absturzverhalten.
Der Code kombiniert Kernel- und Firmware-Komponenten in C und C++, einschließlich Export-Lookup, Mustersuche und Low-Level-Helfern zum Überschreiben von Speicher.
Es ist für fortgeschrittene Firmware-zu-Kernel-Sicherheitsforschung zu Early-Boot-Patching und Bugcheck-Abfangen ausgelegt.
