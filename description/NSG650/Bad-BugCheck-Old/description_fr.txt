Ce projet est un pilote noyau Windows qui lit des images animées pendant un écran de plantage système forcé.
Il utilise des routines Bootvid telles que VidBitBlt pour un rendu de style VGA, puis déclenche un bugcheck après la lecture.
Le code est écrit en C pour une exécution en mode noyau et inclut une logique bas niveau de chargement et d'affichage des images.
Il s'agit d'une preuve de concept pour l'expérimentation graphique noyau Windows autour du comportement BSOD.
