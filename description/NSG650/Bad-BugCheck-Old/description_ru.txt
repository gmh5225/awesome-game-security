Этот проект — драйвер ядра Windows, который воспроизводит анимированные кадры на экране принудительного сбоя системы.
Он использует процедуры Bootvid, такие как VidBitBlt, для отрисовки в стиле VGA, а затем вызывает bugcheck после воспроизведения.
Код написан на C для выполнения в режиме ядра и включает низкоуровневую логику загрузки и отображения кадров.
Это proof-of-concept для экспериментов с графикой ядра Windows вокруг поведения BSOD.
