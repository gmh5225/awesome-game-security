Este proyecto es un controlador del kernel de Windows que reproduce fotogramas animados durante una pantalla de fallo del sistema forzada.
Usa rutinas Bootvid como VidBitBlt para un renderizado de estilo VGA y luego dispara un bugcheck tras la reproducción.
El código está escrito en C para ejecución en modo kernel e incluye lógica de bajo nivel para cargar y mostrar los fotogramas.
Es una prueba de concepto para experimentar con gráficos del kernel de Windows en torno al comportamiento del BSOD.
