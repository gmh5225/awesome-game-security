Questo progetto è un driver del kernel Windows che riproduce fotogrammi animati durante una schermata di crash di sistema forzata.
Usa routine Bootvid come VidBitBlt per un rendering in stile VGA e poi attiva un bugcheck dopo la riproduzione.
Il codice è scritto in C per l'esecuzione in modalità kernel e include logica a basso livello per il caricamento e la visualizzazione dei fotogrammi.
È una proof of concept per sperimentare la grafica del kernel Windows intorno al comportamento del BSOD.
