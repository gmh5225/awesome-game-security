Este proyecto es un port al kernel de Windows que ejecuta DOOM desde el contexto de un controlador en modo kernel.
Utiliza el manejo de syscalls del lado de win32k, la falsificación del contexto de hilo y las interacciones gráficas y de entrada en el kernel para ejecutar la lógica del juego.
La implementación es principalmente código de controlador en C y C++, con un trabajo sustancial sobre los internals de NT y una base PureDOOM adaptada.
Sirve como demostración de investigación para experimentación extrema con GUI y syscalls en el kernel, más que como desarrollo práctico de juegos.
