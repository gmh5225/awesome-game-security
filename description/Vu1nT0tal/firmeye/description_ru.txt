Этот проект — плагин для IDA, предназначенный для поиска уязвимостей в прошивках IoT на основе трассировки аргументов до чувствительных функций. Он в основном написан на Python и сочетает статические проверки с динамическим анализом при поддержке отладчика, чтобы снизить объём ручной сортировки находок. Документированные правила нацелены на такие проблемы, как переполнения буфера, риски выполнения команд и ошибки форматных строк, а автоматизация поддерживается через пакетные рабочие процессы из командной строки. Основной сценарий использования — аудит безопасности прошивок и исследования по реверс-инжинирингу встраиваемых целей.
