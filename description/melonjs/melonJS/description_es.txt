Este proyecto es un motor de juegos HTML5 nuevo, moderno y ligero. Se ha reconstruido por completo con clases, herencia y semántica ES6, y se empaqueta con esbuild para builds muy rápidos y funciones modernas como el tree-shaking. Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el área de motores de juego / código fuente.
