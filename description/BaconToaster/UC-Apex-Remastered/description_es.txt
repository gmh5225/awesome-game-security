Este proyecto es un framework de game hacking en Windows C++ para Apex que combina un controlador de kernel con un cliente en modo usuario. El lado del controlador ofrece operaciones de memoria privilegiadas y rutinas de comunicación, mientras que el lado de usuario implementa módulos de jugabilidad e interacción con procesos. Usa DirectX9 e ImGui para el renderizado de overlay y los controles de la herramienta, junto con objetivos de compilación de Visual Studio/WDK. El caso de uso principal es la investigación en seguridad de juegos centrada en el acceso a memoria asistido por controlador y técnicas de overlay en tiempo real.
