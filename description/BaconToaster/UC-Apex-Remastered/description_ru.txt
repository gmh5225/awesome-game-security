Этот проект — Windows C++ фреймворк для game hacking в Apex, сочетающий драйвер ядра с клиентом в пользовательском режиме. Сторона драйвера обеспечивает привилегированные операции с памятью и процедуры обмена данными, а пользовательская сторона реализует игровые модули и взаимодействие с процессами. Для рендеринга оверлея и элементов управления в инструменте используются DirectX9 и ImGui, а также цели сборки Visual Studio/WDK. Основное применение — исследования в области игровой безопасности с акцентом на доступ к памяти через драйвер и техники оверлея в реальном времени.
