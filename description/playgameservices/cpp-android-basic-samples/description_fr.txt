Ce projet se concentre sur des jeux d'exemple utilisant le Google Play Games C++ SDK. (Facultatif) Sur certaines versions de Windows, il est utile de mapper le dossier samples vers un chemin plus court. Il s'adresse principalement aux développeurs de jeux, aux ingénieurs en rétro-ingénierie et aux créateurs d'outils travaillant dans le domaine du développement de jeux / du code source.
