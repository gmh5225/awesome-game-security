Этот проект посвящён примерам игр с использованием Google Play Games C++ SDK. (Необязательно) В некоторых версиях Windows полезно сопоставить папку samples с укороченным путём. Он в основном полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов, работающим в области разработки игр / исходного кода.
