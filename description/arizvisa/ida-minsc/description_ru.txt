Этот проект — плагин для IDA Pro, который заменяет и расширяет пространство имён скриптинга IDAPython по умолчанию. Он перестраивает API дизассемблера в более простой интерфейс на Python и добавляет возможности вроде тегирования, фильтрации, а также упрощённой автоматизации работы с базой данных и функциями. В репозитории есть руководство по установке, модульные абстракции и документация для быстрого создания скриптов обратной разработки. Им в основном пользуются reverse-инженеры, которым нужны более быстрые сценарии аннотирования, навигации и анализа.
