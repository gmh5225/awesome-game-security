Ce projet se concentre sur un plugin IDA 9.X pour les opérations sur les vtables : dump du squelette d'interface C++ (.hpp), renommage avec préfixe de classe, définition du type du pointeur this, affichage de l'index/offset des slots.
Il est principalement écrit en Python et s'articule autour des pipelines d'assets et du développement de plugins.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers étudiant les techniques offensives dans le domaine des cheats / plugins IDA.
