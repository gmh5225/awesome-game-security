Этот проект посвящён плагину IDA 9.X для операций с vtable: дамп скелета C++-интерфейса (.hpp), переименование с префиксом класса, задание типа указателя this, отображение индекса/смещения слота.
Он в основном написан на Python и ориентирован на конвейеры ассетов и разработку плагинов.
Прежде всего полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / плагинов IDA.
