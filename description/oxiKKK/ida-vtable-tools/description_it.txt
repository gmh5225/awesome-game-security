Questo progetto si concentra su un plugin IDA 9.X per operazioni sulle vtable: dump dello scheletro di interfaccia C++ (.hpp), rinomina con prefisso di classe, impostazione del tipo del puntatore this e visualizzazione di indice/offset degli slot.
È scritto principalmente in Python e si concentra su pipeline di asset e sviluppo di plugin.
È soprattutto utile a ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin IDA.
