Ein engine-agnostischer C#-ASP.NET-Core-Server für das Model Context Protocol (MCP), der als lokaler Proxy-Host für Plugins von Unity, Godot und Unreal dient. Er ist ein schlanker Host über NuGet-Pakete, die MCP-Clients wie Claude, Cursor und Copilot über SignalR mit einem In-Editor- oder In-Game-Plugin verbinden; dieses Projekt enthält keinen enginespezifischen Code. Der Server unterstützt die Client-Transports stdio und streamableHttp, konfigurierbare Authentifizierung (einschließlich OAuth), Origin-Allowlisting und Sitzungsverwaltung und wird als eigenständige Executables, Docker-Image und globales .NET-Tool ausgeliefert. Der Hauptzweck ist, KI-Coding-Agenten die Steuerung und Inspektion von Game Engines während der Entwicklung über einen gemeinsamen lokalen MCP-Endpunkt zu ermöglichen.
