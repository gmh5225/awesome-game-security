エンジン非依存の C# ASP.NET Core Model Context Protocol（MCP）サーバーで、Unity、Godot、Unreal エンジンのプラグインが共有するローカルプロキシホストとして動作します。NuGet パッケージ上の薄いホストであり、Claude、Cursor、Copilot などの MCP クライアントを SignalR 経由でエディタ内またはゲーム内プラグインに橋渡しし、本プロジェクト自体にはエンジン固有のコードは含まれません。stdio および streamableHttp クライアントトランスポート、設定可能な認証（OAuth を含む）、オリジンの許可リスト、セッション処理をサポートし、スタンドアロン実行ファイル、Docker イメージ、グローバル .NET ツールとして提供されます。主な用途は、AI コーディングエージェントが共通のローカル MCP エンドポイントを通じて開発中にゲームエンジンを制御・検査できるようにすることです。
