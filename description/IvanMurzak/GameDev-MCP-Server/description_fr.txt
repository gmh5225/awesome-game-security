Serveur Model Context Protocol (MCP) C# ASP.NET Core indépendant du moteur, qui agit comme hôte proxy local partagé par les plugins Unity, Godot et Unreal. Il s’agit d’un hôte léger au-dessus de packages NuGet qui relient des clients MCP tels que Claude, Cursor et Copilot à un plugin dans l’éditeur ou en jeu via SignalR, sans code spécifique à un moteur dans ce projet. Le serveur prend en charge les transports clients stdio et streamableHttp, une authentification configurable (dont OAuth), une liste blanche d’origines et la gestion de session, et est livré en exécutables autonomes, image Docker et outil .NET global. Son cas d’usage principal est de permettre aux agents de codage IA de contrôler et d’inspecter les moteurs de jeu pendant le développement via un point de terminaison MCP local commun.
