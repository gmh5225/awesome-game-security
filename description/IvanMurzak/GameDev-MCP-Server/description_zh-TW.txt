一個與引擎無關的 C# ASP.NET Core Model Context Protocol（MCP）伺服器，作為 Unity、Godot 與 Unreal 引擎外掛共用的本機代理主機。它是對 NuGet 套件的輕量主機，將 Claude、Cursor、Copilot 等 MCP 用戶端透過 SignalR 橋接至編輯器內或遊戲內外掛，本專案不含任何引擎特定程式碼。伺服器支援 stdio 與 streamableHttp 用戶端傳輸、可設定的身份驗證（含 OAuth）、來源允許清單與工作階段處理，並以獨立執行檔、Docker 映像與全域 .NET 工具形式提供。其主要用途是讓 AI 程式設計代理在開發過程中透過共同的本機 MCP 端點控制並檢視遊戲引擎。
