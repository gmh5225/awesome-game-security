Независимый от движка сервер Model Context Protocol (MCP) на C# ASP.NET Core, который работает как локальный прокси-хост, общий для плагинов Unity, Godot и Unreal. Это тонкий хост над пакетами NuGet, связывающими MCP-клиенты вроде Claude, Cursor и Copilot с плагином в редакторе или в игре через SignalR; в самом проекте нет кода, специфичного для движка. Сервер поддерживает клиентские транспорты stdio и streamableHttp, настраиваемую аутентификацию (включая OAuth), список разрешённых origin и обработку сессий, и поставляется как отдельные исполняемые файлы, Docker-образ и глобальный инструмент .NET. Основной сценарий — дать ИИ-агентам для разработки возможность управлять игровыми движками и инспектировать их через общий локальный MCP-эндпоинт.
