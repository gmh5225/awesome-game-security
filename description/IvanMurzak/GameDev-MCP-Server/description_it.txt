Server Model Context Protocol (MCP) in C# ASP.NET Core indipendente dal motore, che funge da host proxy locale condiviso dai plugin di Unity, Godot e Unreal. È un host leggero sopra pacchetti NuGet che collegano client MCP come Claude, Cursor e Copilot a un plugin in-editor o in-game tramite SignalR, senza codice specifico del motore in questo progetto. Il server supporta i trasporti client stdio e streamableHttp, autenticazione configurabile (incluso OAuth), allow-list delle origini e gestione delle sessioni, e viene distribuito come eseguibili standalone, immagine Docker e strumento .NET globale. Il caso d’uso principale è consentire agli agenti di coding basati su IA di controllare e ispezionare i motori di gioco durante lo sviluppo tramite un endpoint MCP locale comune.
