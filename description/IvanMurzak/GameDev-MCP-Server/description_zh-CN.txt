一个与引擎无关的 C# ASP.NET Core Model Context Protocol（MCP）服务器，作为 Unity、Godot 和 Unreal 引擎插件共享的本地代理宿主。它是对 NuGet 包的轻量宿主，通过这些包将 Claude、Cursor、Copilot 等 MCP 客户端经 SignalR 桥接到编辑器内或游戏内插件，本项目不含任何引擎特定代码。服务器支持 stdio 与 streamableHttp 客户端传输、可配置身份验证（含 OAuth）、来源白名单与会话处理，并以独立可执行文件、Docker 镜像和全局 .NET 工具形式发布。其主要用途是让 AI 编程代理在开发过程中通过统一的本地 MCP 端点控制并检视游戏引擎。
