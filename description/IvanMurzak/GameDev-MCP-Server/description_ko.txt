엔진에 종속되지 않는 C# ASP.NET Core Model Context Protocol(MCP) 서버로, Unity, Godot, Unreal 엔진 플러그인이 공유하는 로컬 프록시 호스트 역할을 합니다. NuGet 패키지 위의 얇은 호스트이며 Claude, Cursor, Copilot 같은 MCP 클라이언트를 SignalR를 통해 에디터 내 또는 게임 내 플러그인에 연결하고, 이 프로젝트에는 엔진별 코드가 없습니다. stdio 및 streamableHttp 클라이언트 전송, 구성 가능한 인증(OAuth 포함), origin 허용 목록, 세션 처리를 지원하며 독립 실행 파일, Docker 이미지, 전역 .NET 도구로 제공됩니다. 주요 용도는 AI 코딩 에이전트가 공통 로컬 MCP 엔드포인트를 통해 개발 중 게임 엔진을 제어하고 검사하도록 하는 것입니다.
