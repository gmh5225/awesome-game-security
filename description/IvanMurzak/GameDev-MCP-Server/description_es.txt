Servidor Model Context Protocol (MCP) en C# ASP.NET Core independiente del motor, que actúa como host proxy local compartido por plugins de Unity, Godot y Unreal. Es un host ligero sobre paquetes NuGet que conectan clientes MCP como Claude, Cursor y Copilot con un plugin en el editor o en el juego mediante SignalR, sin código específico de motor en este proyecto. El servidor admite transportes de cliente stdio y streamableHttp, autenticación configurable (incluido OAuth), lista de orígenes permitidos y gestión de sesiones, y se distribuye como ejecutables independientes, imagen Docker y herramienta .NET global. Su caso de uso principal es permitir que agentes de codificación con IA controlen e inspeccionen motores de juego durante el desarrollo a través de un endpoint MCP local común.
