An engine-agnostic C# ASP.NET Core Model Context Protocol (MCP) server that acts as a local proxy host shared by Unity, Godot, and Unreal engine plugins. It is a thin host over NuGet packages that bridge MCP clients such as Claude, Cursor, and Copilot to an in-editor or in-game plugin over SignalR, with no engine-specific code in this project. The server supports stdio and streamableHttp client transports, configurable authentication (including OAuth), origin allow-listing, and session handling, and ships as standalone executables, a Docker image, and a global .NET tool. Its primary use case is letting AI coding agents control and inspect game engines during development through a common local MCP endpoint.
