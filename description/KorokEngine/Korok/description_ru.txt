Этот проект — лёгкий кроссплатформенный 2D-игровой движок с компонентной архитектурой.
Он предоставляет повторно используемые модули для анимации, рендеринга, ассетов, аудио и игровых систем.
Движок в основном написан на Go и следует компактному пакетно-ориентированному дизайну, вдохновлённому идеями современной архитектуры движков.
Документация и примеры проектов включены, чтобы помочь разработчикам быстро начать работу.
Он предназначен для разработчиков игр и исследователей движков, которым нужен небольшой open-source фреймворк для создания 2D-игр.
