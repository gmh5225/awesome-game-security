Mystic — это header-only библиотека на C++17, которая шифрует строки и целые числа на этапе компиляции и расшифровывает их во время выполнения с помощью SIMD. Она использует AVX, SSE или ARM NEON для векторизованной расшифровки, а ключи и IV для каждого значения генерируются через constexpr-вычисления из compile-time seed. Опциональная инъекция junk-кода и приёмы, ломающие декомпиляторы (раздувание стека, непрозрачные предикаты), засоряют дизассемблирование и могут ломать инструменты вроде IDA Pro, не влияя на скорость выполнения. Также поддерживаются среды без CRT или минимальные окружения через _MYSTIC_MINIMAL. Библиотека предназначена для разработчиков, которым нужна обфускация строк и целых чисел для безопасности игр, античита или устойчивости к реверс-инжинирингу.
