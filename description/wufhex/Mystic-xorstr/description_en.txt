Mystic is a C++17 header-only library that encrypts strings and integers at compile time and decrypts them at runtime with SIMD. It uses AVX, SSE, or ARM NEON for vectorized decryption, with per-value keys and IVs generated via constexpr computations from a compile-time seed. Optional junk-code injection and decompiler-breaking tricks (stack bloating, opaque predicates) clutter disassembly and can break tools like IDA Pro without affecting runtime speed. It also supports CRT-less or minimal environments via _MYSTIC_MINIMAL. The library is aimed at developers who need string and integer obfuscation for game security, anti-cheat, or reverse-engineering resistance.
