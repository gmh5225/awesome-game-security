Этот проект — практическая книга по изучению игровых движков с большим сопроводительным кодом. В центре внимания — C++ и OpenGL; материал шаг за шагом проходит построение движка от основ рендеринга до анимации, GUI, аудио, профилирования и многопоточного рендеринга. Репозиторий содержит множество примеров по главам, а также вспомогательные технологии: интеграцию Lua-скриптов, рабочие процессы экспорта из Blender и шаги по аудио с FMOD. Основное назначение — обучение архитектуре игровых движков тех, кто хочет получить практический опыт реализации.
