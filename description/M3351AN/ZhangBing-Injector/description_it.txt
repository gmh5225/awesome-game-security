Questo progetto è un injector di DLL per Windows che utilizza driver kernel vulnerabili per iniettare in processi protetti. Include più file driver .sys e fornisce un'applicazione C++ in user mode che comunica con i driver caricati per eseguire operazioni di memoria e iniezione di DLL. È pensato soprattutto per i ricercatori di sicurezza nei videogiochi che studiano tecniche di iniezione di DLL assistite dal kernel e lo sfruttamento di driver vulnerabili per manipolare i processi.
