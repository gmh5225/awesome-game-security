Этот проект — Windows DLL-инжектор, который использует уязвимые драйверы ядра для внедрения в защищённые процессы. Он включает несколько файлов драйверов .sys и предоставляет пользовательское C++-приложение, которое взаимодействует с загруженными драйверами для операций с памятью и DLL-инъекции. В первую очередь полезен исследователям игровой безопасности, изучающим техники DLL-инъекции с помощью ядра и эксплуатацию уязвимых драйверов для манипуляции процессами.
