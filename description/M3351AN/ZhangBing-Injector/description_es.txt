Este proyecto es un inyector de DLL para Windows que utiliza controladores de kernel vulnerables para inyectar en procesos protegidos. Incluye varios archivos de controlador .sys y proporciona una aplicación C++ en modo usuario que se comunica con los controladores cargados para realizar operaciones de memoria e inyección de DLL. Resulta especialmente útil para investigadores de seguridad en juegos que estudian técnicas de inyección de DLL asistidas por el kernel y la explotación de controladores vulnerables para manipular procesos.
