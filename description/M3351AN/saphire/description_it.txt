Questo progetto è un framework di cheat per CSGO con un menu overlay DirectX9 basato su ImGui e un editor di codice integrato. La codebase C++ include l'integrazione del rendering dei font FreeType, widget ImGui personalizzati e un'interfaccia cheat modulare con supporto alla configurazione. È principalmente utile per i ricercatori di sicurezza dei giochi che studiano framework UI per cheat e implementazioni di overlay basate su ImGui nei giochi Source engine.
