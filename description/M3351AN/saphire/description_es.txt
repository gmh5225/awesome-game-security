Este proyecto es un framework de cheat para CSGO con un menú overlay DirectX9 basado en ImGui y un editor de código integrado. La base de código en C++ incluye integración de renderizado de fuentes FreeType, widgets ImGui personalizados y una interfaz de cheat modular con soporte de configuración. Es especialmente útil para investigadores de seguridad en videojuegos que estudian frameworks de UI de cheats e implementaciones de overlay basadas en ImGui en juegos del motor Source.
