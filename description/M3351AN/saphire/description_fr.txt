Ce projet est un framework de cheat CSGO avec un menu overlay DirectX9 basé sur ImGui et un éditeur de code intégré. La base de code C++ inclut l'intégration du rendu de polices FreeType, des widgets ImGui personnalisés et une interface de cheat modulaire avec support de configuration. Il est principalement utile aux chercheurs en sécurité des jeux qui étudient les frameworks d'interface de cheat et les implémentations d'overlay basées sur ImGui dans les jeux Source engine.
