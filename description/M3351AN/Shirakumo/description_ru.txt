Этот проект — proof-of-concept RPM/WPM-прокси, пересылающий операции с памятью через named pipes. Он написан на C++ и выносит выполнение чтения/записи в отдельный процесс, с опциональной загрузкой DLL для развёртывания прокси. Реализация явно экспериментальная, с известными ограничениями: только x64 и отсутствие потокобезопасности. В основном используется для изучения паттернов доступа к памяти с разделением по процессам в контексте игровых инструментов и исследований обхода защиты.
