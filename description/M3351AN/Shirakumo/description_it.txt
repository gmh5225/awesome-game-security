Questo progetto è un proxy RPM/WPM proof-of-concept che inoltra le operazioni di memoria tramite named pipes. È scritto in C++ e separa l’esecuzione di lettura/scrittura in un altro processo, con caricamento DLL opzionale per il deployment del proxy. L’implementazione è esplicitamente sperimentale, con limiti noti come supporto solo x64 e assenza di thread safety. Serve soprattutto a studiare schemi di accesso alla memoria separati per processo per tool di gioco e ricerca sull’evasione.
