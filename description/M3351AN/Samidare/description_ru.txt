Этот проект представляет собой внешний чит для CS2 с доступом к памяти через драйвер, включающий DirectX-оверлей, управление смещениями и чтение игровых данных через интерфейс связи с драйвером ядра. Кодовая база на C++ содержит FIFO-коммуникацию с драйвером, отрисовку оверлея и математические утилиты для расчёта игровых координат. В первую очередь полезен исследователям игровой безопасности, изучающим реализации внешних читов с поддержкой драйвера и техники отрисовки оверлеев.
