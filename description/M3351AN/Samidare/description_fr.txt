Ce projet est un cheat externe pour CS2 avec accès mémoire basé sur un pilote, proposant une overlay DirectX, la gestion des offsets et la lecture des données de jeu via une interface de communication avec un pilote noyau. La base de code C++ inclut une communication pilote basée sur FIFO, le rendu d'overlay et des utilitaires mathématiques pour le calcul des coordonnées de jeu. Il est principalement utile aux chercheurs en sécurité des jeux qui étudient les implémentations de cheats externes assistés par pilote et les techniques de rendu d'overlay.
