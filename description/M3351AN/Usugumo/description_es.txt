Este proyecto es un controlador de Windows en modo kernel de tipo prueba de concepto que hace de proxy de operaciones de memoria y entrada para clientes en modo usuario. Usa C/C++ con algo de MASM y gestiona peticiones IRP DIRECT_IO para RPM/WPM, búsqueda de procesos, información de módulos e inyección de ratón/teclado. El repositorio también incluye ejemplos de anti-captura y comunicación, dejando claro que no está listo para producción. Su caso de uso principal es la comunicación con el kernel y la experimentación en seguridad de juegos a bajo nivel en Windows x64.
