Этот проект — концептуальный драйвер Windows в режиме ядра, который проксирует операции с памятью и вводом для клиентов в пользовательском режиме. Он написан на C/C++ с элементами MASM и обрабатывает запросы DIRECT_IO IRP для RPM/WPM, поиска процессов, сведений о модулях и инъекции мыши/клавиатуры. В репозитории также есть примеры anti-capture и коммуникации; при этом явно указано, что решение не готово к промышленному использованию. Основной сценарий — межрежимная коммуникация с ядром и низкоуровневые эксперименты по безопасности игр в x64 Windows.
