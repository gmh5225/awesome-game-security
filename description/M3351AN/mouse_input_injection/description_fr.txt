Ce projet est un utilitaire léger d’injection d’entrées souris pour Windows, fourni en en-tête uniquement. Il s’appuie sur l’appel système non documenté NtUserInjectMouseInput tout en conservant une API proche de mouse_event pour faciliter la migration. L’implémentation est du C/C++ minimal, conçue pour une intégration rapide en incluant un seul en-tête et en appelant des fonctions wrappers. Le cas d’usage principal est l’automatisation des entrées et la recherche sur les chemins d’entrée bas niveau.
