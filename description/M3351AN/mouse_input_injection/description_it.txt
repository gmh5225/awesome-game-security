Questo progetto è un’utilità leggera header-only per l’iniezione di input del mouse su Windows. Si basa sulla syscall non documentata NtUserInjectMouseInput, mantenendo uno stile API simile a mouse_event per facilitare la migrazione. L’implementazione è C/C++ minimale e pensata per un’integrazione rapida includendo un solo header e chiamando funzioni wrapper. Il caso d’uso principale è l’automazione dell’input e la ricerca sui percorsi di input a basso livello.
