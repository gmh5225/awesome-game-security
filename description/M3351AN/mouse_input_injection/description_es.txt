Este proyecto es una utilidad ligera de solo cabeceras para inyección de entrada de ratón en Windows. Se basa en la syscall no documentada NtUserInjectMouseInput y mantiene un estilo de API similar a mouse_event para facilitar la migración. La implementación es C/C++ mínimo y está pensada para una integración rápida incluyendo un solo encabezado y llamando a funciones envoltorio. El caso de uso principal es la automatización de entrada y la investigación de rutas de entrada de bajo nivel.
