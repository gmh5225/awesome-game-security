Dieses Projekt ist ein leichtgewichtiges, header-only Maus-Eingabeinjektions-Utility für Windows. Es stützt sich auf den undokumentierten Syscall NtUserInjectMouseInput und behält dabei einen API-Stil ähnlich zu mouse_event bei, um die Migration zu erleichtern. Die Implementierung ist minimales C/C++ und für schnelle Integration ausgelegt: ein einzelner Header einbinden und Wrapper-Funktionen aufrufen. Der Hauptanwendungsfall ist Eingabeautomatisierung sowie Forschung an niedrigstufigen Eingabepfaden.
