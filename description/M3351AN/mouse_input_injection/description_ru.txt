Этот проект — лёгкая header-only утилита для инъекции мышиного ввода в Windows. Она опирается на недокументированный системный вызов NtUserInjectMouseInput и сохраняет стиль API, близкий к mouse_event, чтобы упростить миграцию. Реализация — минимальный C/C++, рассчитанный на быструю интеграцию: достаточно подключить один заголовок и вызывать функции-обёртки. Основной сценарий использования — автоматизация ввода и исследование низкоуровневых путей ввода.
