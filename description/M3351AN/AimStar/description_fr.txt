Ce projet est un framework de cheat externe pour Counter-Strike 2, écrit en C++. Il comprend une logique d'aimbot basée sur les bones, le suivi d'entités, le rendu ESP, un triggerbot, la gestion des offsets, ainsi qu'un système de configuration de menu avec une documentation multilingue en anglais, russe et chinois. Il est surtout utile aux chercheurs en sécurité des jeux qui étudient l'architecture des cheats externes et les techniques de lecture mémoire de CS2.
