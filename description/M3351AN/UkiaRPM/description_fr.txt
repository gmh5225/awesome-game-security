Ce projet est un cheat CS2 externe qui utilise la lecture de mémoire de processus distant via un pilote noyau pour l'aimbot, l'ESP, le radar et d'autres modifications de jeu. La base de code C++ comprend un menu DirectX9 basé sur ImGui, un système de contrôle du recul, la sauvegarde de configuration et des fonctionnalités d'overlay visuel. Il est principalement utile aux chercheurs en sécurité des jeux qui étudient les schémas de cheats externes basés sur RPM et la communication avec les pilotes noyau pour l'accès à la mémoire du jeu.
