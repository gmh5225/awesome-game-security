Dieses Projekt konzentriert sich auf praktische Vereinfachungen der Mixed-Boolean-Arithmetic-Obfuskation. Ziel ist es, Code schwerer analysierbar zu machen und Software vor Reverse Engineering und Manipulation zu schützen. Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Mixed Boolean-Arithmetic untersuchen.
