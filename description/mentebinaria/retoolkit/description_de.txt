Kuratiertes Windows-Installationsprogramm (mit Inno Setup erstellt), das über 80 Reverse-Engineering-Tools in einer einzigen Bereitstellung bündelt, darunter Debugger (x64dbg, HyperDbg), Decompiler (Ghidra, Cutter), .NET-Analysatoren (dnSpyEx, ILSpy, de4dot), PE-Inspektoren (PE-bear, pestudio, DIE), Hex-Editoren (ImHex, HxD), Netzwerktools (FakeNet, nmap) sowie YARA/YARA-X-Signaturscanner.
