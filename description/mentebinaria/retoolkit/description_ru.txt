Курируемый установщик Windows (собранный с помощью Inno Setup), объединяющий более 80 инструментов реверс-инжиниринга в одно развёртывание: отладчики (x64dbg, HyperDbg), декомпиляторы (Ghidra, Cutter), анализаторы .NET (dnSpyEx, ILSpy, de4dot), инспекторы PE (PE-bear, pestudio, DIE), шестнадцатеричные редакторы (ImHex, HxD), сетевые утилиты (FakeNet, nmap) и сканеры сигнатур YARA/YARA-X.
