Этот проект — плагин для IDA Pro, который встраивает рабочие процессы с помощью GPT непосредственно в сеансы реверс-инжиниринга. Он написан на Python и интегрируется с IDAPython, Hex-Rays и LangChain для анализа псевдокода, проверки символов и применения комментариев или предложений по переименованию. Плагин добавляет пункты меню и команды по горячим клавишам для полуавтоматического исследования на уровне функций. Он предназначен для реверс-инженеров, которым нужна более быстрая первичная оценка и интерактивная поддержка ИИ при анализе бинарников.
