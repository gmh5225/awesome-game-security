Этот проект сосредоточен на перечислении и удалении kernel callbacks с помощью подписанных уязвимых драйверов. Данная программа представляет собой Proof of Concept, позволяющий перечислять и изменять некоторые из этих kernel callbacks с помощью подписанного, но уязвимого драйвера MSI. Она в основном полезна исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования ядра Windows.
