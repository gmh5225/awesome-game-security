Este proyecto se centra en enumerar y eliminar callbacks del kernel mediante controladores vulnerables firmados. Este programa es una prueba de concepto (Proof of Concept) que permite enumerar y modificar algunos de estos callbacks del kernel usando un controlador MSI firmado pero vulnerable. Resulta principalmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / exploración del kernel de Windows.
