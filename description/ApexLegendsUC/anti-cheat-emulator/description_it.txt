Questo progetto è un driver Windows in modalità kernel che simula un anti-cheat ed esegue diverse routine di rilevamento euristico. Scritto in C++, analizza i thread di sistema, le stack trace, le allocazioni BigPool, le voci della cache PiDDB, le tabelle di dispatch dei driver e l'uso sospetto di handle sulla memoria fisica. Include anche controlli su hypervisor e mapping del kernel pensati per emulare la logica pratica di telemetria anti-cheat. Il codice è rivolto ai ricercatori che studiano come implementare e testare le pipeline di rilevamento anti-cheat a livello kernel.
