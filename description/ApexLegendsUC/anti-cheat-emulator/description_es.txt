Este proyecto es un controlador de simulación anti-cheat en el kernel de Windows que ejecuta múltiples rutinas de detección heurística. Escrito en C++, analiza hilos del sistema, rastros de pila, asignaciones BigPool, entradas de la caché PiDDB, tablas de despacho de controladores y el uso sospechoso de handles de memoria física. También incluye comprobaciones de hipervisor y de mapeo del kernel pensadas para emular la lógica práctica de telemetría anti-cheat. El código está orientado a investigadores que estudian cómo implementar y probar pipelines de detección anti-cheat a nivel de kernel.
