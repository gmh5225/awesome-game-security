Dieses Projekt ist ein Windows-Kernel-Treiber zur Anti-Cheat-Simulation, der mehrere heuristische Erkennungsroutinen ausführt. In C++ geschrieben, scannt er Systemthreads, Stack-Traces, BigPool-Allokationen, PiDDB-Cache-Einträge, Treiber-Dispatch-Tabellen sowie verdächtige Nutzung von Handles auf physischen Speicher. Er enthält außerdem Hypervisor- und Kernel-Mapping-Prüfungen, die praktische Anti-Cheat-Telemetrie-Logik nachbilden sollen. Der Code richtet sich an Forscher, die untersuchen, wie Erkennungspipelines von Anti-Cheat-Systemen auf Kernel-Ebene implementiert und getestet werden können.
