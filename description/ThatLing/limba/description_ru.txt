Этот проект — proof-of-concept обфускации потока управления на этапе компиляции с помощью смешанных булево-арифметических преобразований.
Он генерирует шаблонный код, скрывающий реальные цели вызовов путём кодирования адресов переходов случайными правилами переписывания и смещениями для каждой сборки.
Реализация ориентирована на C++20 и предпочитает toolchain Clang/clang-cl, с примерами интеграции на основе premake.
Предназначен для исследований устойчивости к реверс-инжинирингу и экспериментов с обфускацией бинарников.
