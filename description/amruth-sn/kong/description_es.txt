Este proyecto es Kong, una herramienta para desofuscar o analizar código ofuscado, orientada probablemente a expresiones de aritmética booleana mixta (MBA) o patrones de ofuscación similares. Aplica simplificación algebraica, coincidencia de patrones o ejecución simbólica para reducir expresiones ofuscadas complejas a sus formas originales más simples. Está dirigida a ingenieros de ingeniería inversa e investigadores de desofuscación que trabajan con binarios y expresiones ofuscados mediante MBA.
