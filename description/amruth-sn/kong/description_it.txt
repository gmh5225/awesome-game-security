Questo progetto è Kong, uno strumento per deoffuscare o analizzare codice offuscato, mirato probabilmente a espressioni di aritmetica booleana mista (MBA) o a pattern di offuscamento simili. Applica semplificazione algebrica, pattern matching o esecuzione simbolica per ricondurre espressioni offuscate complesse alle loro forme originali più semplici. È rivolto a reverse engineer e ricercatori di deoffuscamento che lavorano con binari ed espressioni offuscati tramite MBA.
