Этот проект — proof of concept ручного маппера (manual mapper) в режиме ядра Windows на языке C. Он перехватывает указатель на функцию ядра для приёма запросов на маппинг, копирует PE-образы в исполняемую память ядра и разрешает импорты и релокации перед вызовом точки входа. Код организован как проект драйвера Visual Studio со вспомогательными утилитами для поиска по сигнатурам и записи в защищённую память. Основное применение — низкоуровневое исследование техник загрузки драйверов, используемых в экспериментах по обходу античита.
