Este proyecto es una prueba de concepto de un manual mapper en modo kernel de Windows, escrito en C. Engancha un puntero a función del kernel para recibir solicitudes de mapeo, copia imágenes PE en memoria de kernel ejecutable y resuelve imports y relocations antes de llamar al punto de entrada. El código está estructurado como un proyecto de controlador de Visual Studio, con utilidades auxiliares para el escaneo de patrones y escrituras en memoria protegida. Su principal caso de uso es la investigación de bajo nivel sobre técnicas de carga de controladores usadas en experimentos de evasión de anti-cheat.
