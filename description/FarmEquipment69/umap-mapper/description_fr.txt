Ce projet est une preuve de concept de manual mapper en mode noyau Windows, écrite en C. Il accroche un pointeur de fonction noyau pour recevoir les requêtes de mapping, copie des images PE dans de la mémoire noyau exécutable, puis résout les imports et les relocations avant d’appeler le point d’entrée. Le code est organisé comme un projet de pilote Visual Studio, avec des utilitaires d’aide pour le pattern scanning et les écritures en mémoire protégée. Son principal cas d’usage est la recherche bas niveau sur les techniques de chargement de pilotes utilisées dans des expériences d’évasion d’anti-cheat.
