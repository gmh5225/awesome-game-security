Anti-Cheat Scanner는 다층 로컬 시스템 분석을 통해 서드파티 안티치트 소프트웨어의 존재, 구성, 실행 흔적을 탐지하는 Windows 포렌식 유틸리티입니다. Python 3.10+로 작성되며, 드라이버, 프로세스, 서비스, 레지스트리 키, 예약된 작업, 파일시스템 아티팩트와 BAM, Prefetch, MUICache 같은 실행 흔적을 외부 시그니처 데이터베이스와 대조하여 스캔합니다. 매칭에는 O(1) 시그니처 인덱스, rapidfuzz 기반 퍼지 이름 매칭, PE 메타데이터, 일괄 Authenticode 검증을 사용하며, 병렬 체커는 공유 BaseChecker 인터페이스로 구현됩니다. ACE, EA Anti-Cheat/Javelin, Easy Anti-Cheat, BattlEye, HoYoProtect 등 제품을 대상으로 하며, 읽기 전용 포렌식 감사, 프라이버시 검토, 사용자 본인 기기에서의 교육적 게임 보안 연구용으로 설계되었습니다.
