Anti-Cheat Scanner 是一款 Windows 鑑識工具，透過多層本機系統分析偵測第三方反作弊軟體的存在、組態與執行痕跡。以 Python 3.10+ 撰寫，可掃描驅動程式、行程、服務、登錄機碼、排程工作、檔案系統產物，以及 BAM、Prefetch、MUICache 等執行痕跡，並對照外部簽章資料庫進行比對。比對採用 O(1) 簽章索引、rapidfuzz 模糊名稱比對、PE 中繼資料與批次 Authenticode 驗證，平行檢查器統一實作共用的 BaseChecker 介面。支援偵測 ACE、EA Anti-Cheat/Javelin、Easy Anti-Cheat、BattlEye、HoYoProtect 等產品，適用於唯讀鑑識稽核、隱私審查，以及使用者在自有機器上進行的遊戲安全教育研究。
