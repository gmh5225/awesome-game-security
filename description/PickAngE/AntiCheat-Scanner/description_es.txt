Anti-Cheat Scanner es una utilidad forense para Windows que detecta la presencia, configuración y rastros de ejecución de software anti-trampas de terceros mediante un análisis local multicapa del sistema. Escrito en Python 3.10+, examina controladores, procesos, servicios, claves del registro, tareas programadas, artefactos del sistema de archivos y rastros de ejecución como BAM, Prefetch y MUICache frente a una base de firmas externa. La coincidencia usa un índice de firmas O(1), coincidencia difusa de nombres con rapidfuzz, metadatos PE y verificación Authenticode por lotes, con comprobadores en paralelo bajo una interfaz BaseChecker compartida. Se orienta a productos como ACE, EA Anti-Cheat/Javelin, Easy Anti-Cheat, BattlEye y HoYoProtect, y está pensado para auditoría forense de solo lectura, revisión de privacidad e investigación educativa sobre seguridad en juegos en la máquina del propio usuario.
