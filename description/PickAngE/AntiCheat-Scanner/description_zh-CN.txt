Anti-Cheat Scanner 是一款 Windows 取证工具，通过多层本地系统分析检测第三方反作弊软件的存在、配置与执行痕迹。基于 Python 3.10+ 编写，可扫描驱动程序、进程、服务、注册表项、计划任务、文件系统产物，以及 BAM、Prefetch、MUICache 等执行痕迹，并对照外部签名数据库进行匹配。匹配采用 O(1) 签名索引、rapidfuzz 模糊名称匹配、PE 元数据与批量 Authenticode 验证，并行检查器统一实现共享的 BaseChecker 接口。支持检测 ACE、EA Anti-Cheat/Javelin、Easy Anti-Cheat、BattlEye、HoYoProtect 等产品，面向只读取证审计、隐私审查以及用户在自有机器上开展的游戏安全教育研究。
