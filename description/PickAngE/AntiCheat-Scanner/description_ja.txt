Anti-Cheat Scanner は、多層的なローカルシステム分析により、サードパーティ製アンチチートソフトの存在・設定・実行痕跡を検出する Windows フォレンジックユーティリティです。Python 3.10+ で記述され、ドライバ、プロセス、サービス、レジストリキー、スケジュールタスク、ファイルシステム成果物、および BAM、Prefetch、MUICache などの実行痕跡を外部シグネチャデータベースと照合してスキャンします。照合には O(1) シグネチャインデックス、rapidfuzz によるあいまい名称マッチング、PE メタデータ、バッチ Authenticode 検証を用い、並列チェッカーは共通の BaseChecker インタフェースで実装されます。ACE、EA Anti-Cheat/Javelin、Easy Anti-Cheat、BattlEye、HoYoProtect などの製品を対象とし、読み取り専用のフォレンジック監査、プライバシー確認、およびユーザー自身のマシン上での教育的なゲームセキュリティ研究を目的としています。
