Anti-Cheat Scanner — утилита для судебно-криминалистического анализа Windows, выявляющая наличие, конфигурацию и следы выполнения стороннего античит-ПО через многоуровневый локальный анализ системы. Написана на Python 3.10+, сканирует драйверы, процессы, службы, ключи реестра, задания планировщика, артефакты файловой системы и следы выполнения, такие как BAM, Prefetch и MUICache, по внешней базе сигнатур. Сопоставление использует индекс сигнатур O(1), нечёткое сопоставление имён через rapidfuzz, метаданные PE и пакетную проверку Authenticode, а параллельные проверки реализованы через общий интерфейс BaseChecker. Нацелена на продукты ACE, EA Anti-Cheat/Javelin, Easy Anti-Cheat, BattlEye и HoYoProtect и предназначена для аудита только для чтения, проверки конфиденциальности и образовательных исследований по безопасности игр на собственной машине пользователя.
