ModFinder è un’utilità Windows in C++ progettata per individuare moduli mappati manualmente (manual map) nella memoria di un processo. Enumera le regioni mappate e verifica i pattern dell’header DOS per identificare iniezioni sospette, anche quando parti degli header opzionali sono state rimosse. L’implementazione è C++ nativo, con una struttura di progetto Visual Studio e un focus pratico sull’analisi di processi x86. Il caso d’uso principale è la forensics della memoria a runtime orientata ad anti-cheat e malware.
