ModFinder es una utilidad de Windows en C++ diseñada para localizar módulos mapeados manualmente (manual map) en la memoria de un proceso. Enumera las regiones mapeadas y comprueba patrones de cabecera DOS para identificar inyecciones sospechosas, incluidos los casos en los que se han eliminado partes de las cabeceras opcionales. La implementación es C++ nativo, con una estructura de proyecto de Visual Studio y un enfoque práctico en el análisis de procesos x86. Su caso de uso principal es la forense de memoria en tiempo de ejecución orientada a anti-cheat y malware.
