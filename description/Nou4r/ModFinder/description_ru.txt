ModFinder — утилита на Windows C++ для поиска вручную отображённых (manual map) модулей в памяти процесса. Она перечисляет отображённые области и проверяет шаблоны DOS-заголовков, чтобы выявлять подозрительные инъекции, в том числе когда части optional headers удалены. Реализация — нативный C++, с компоновкой проекта Visual Studio и практическим акцентом на анализ x86-процессов. Основной сценарий использования — runtime-форензика памяти в контексте античита и вредоносного ПО.
