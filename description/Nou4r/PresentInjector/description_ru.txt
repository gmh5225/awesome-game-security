PresentInjector — проект DLL-инъекции с поддержкой ядра Windows, выполняющий загрузку через подмену графических указателей Present. Он сочетает компоненты пользовательского и ядерного режимов, включая работу с памятью, связанную с PFN, техники цепочек указателей и поведение очистки после инъекции. Код написан на C/C++ и сфокусирован на доставке отображённых (mapped) полезных нагрузок в защищённые игровые процессы. Основной сценарий использования — исследование продвинутой инъекции читов и эксперименты по обходу античита.
