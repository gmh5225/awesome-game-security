Este proyecto es un controlador de kernel de Windows que desactiva PatchGuard en tiempo real en Windows 10 versión 1903 manipulando los atributos NX (no ejecutable) de las páginas de contexto. El código en C y ensamblador x64 incluye rutinas de escaneo de memoria para localizar las estructuras de contexto de PatchGuard y neutralizar sus comprobaciones de integridad. Resulta especialmente útil para investigadores de seguridad del kernel que estudian los internos de PatchGuard, técnicas de omisión de la protección de parches del kernel y los mecanismos de integridad del kernel de Windows.
