Этот проект — драйвер ядра Windows, который в реальном времени отключает PatchGuard в Windows 10 версии 1903, манипулируя NX-атрибутами (запрет исполнения) страниц контекста. Кодовая база на C и x64-ассемблере включает процедуры сканирования памяти для поиска структур контекста PatchGuard и нейтрализации их проверок целостности. Проект в основном полезен исследователям безопасности ядра, изучающим внутреннее устройство PatchGuard, техники обхода защиты патчей ядра и механизмы целостности ядра Windows.
