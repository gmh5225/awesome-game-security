Questo progetto è un header C basato solo su macro per l'offuscamento del codice a compile-time, incentrato sui workflow del compilatore Tiny C su Windows. Offre offuscamento delle chiamate di funzione, mutazione del control-flow, nascondimento delle stringhe, tecniche anti-debug, trucchi anti-decompilazione, inserimento di firme false e logica matematica virtualizzata opzionale. L'implementazione è pensata per integrarsi direttamente includendo un singolo header e attivando il comportamento con flag del preprocessore. È usata principalmente da sviluppatori low-level e ricercatori di sicurezza che esplorano protezione del software e hardening binario.
