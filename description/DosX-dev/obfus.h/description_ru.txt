Этот проект — C-заголовок только на макросах для обфускации кода на этапе компиляции, ориентированный на рабочие процессы Tiny C под Windows. Он предлагает обфускацию вызовов функций, мутацию потока управления, сокрытие строк, антиотладочные техники, приёмы против декомпиляции, вставку поддельных сигнатур и опциональную виртуализированную математическую логику. Реализация рассчитана на прямую интеграцию через подключение одного заголовка с переключением поведения флагами препроцессора. В основном используется низкоуровневыми разработчиками и исследователями безопасности, изучающими защиту ПО и hardening бинарников.
