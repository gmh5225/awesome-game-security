Este proyecto es un encabezado C solo de macros para ofuscación de código en tiempo de compilación, centrado en flujos de trabajo con el compilador Tiny C en Windows. Ofrece ofuscación de llamadas a funciones, mutación del flujo de control, ocultación de cadenas, técnicas anti-debug, trucos anti-descompilación, inserción de firmas falsas y lógica matemática virtualizada opcional. La implementación está pensada para integrarse directamente incluyendo un único encabezado y activando el comportamiento con flags del preprocesador. Se usa principalmente por desarrolladores de bajo nivel e investigadores de seguridad que exploran protección de software y endurecimiento binario.
