Этот проект посвящён Apex-Legends-SDK-Remaster: это упрощённая версия, так как у многих возникают сложности с Python binding.
Открытый чит для Apex Legends, рассчитанный на простоту использования.
Он в первую очередь полезен исследователям безопасности игр и reverse engineers, изучающим атакующие техники в области читов / Apex Legends.
