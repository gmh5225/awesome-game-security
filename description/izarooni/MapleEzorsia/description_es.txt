Este proyecto es MapleEzorsia, un código fuente de servidor privado de MapleStory v83 con contenido personalizado y modificaciones. Implementa la lógica de juego del lado del servidor para la versión clásica v83, incluyendo el manejo de inicio de sesión, gestión de personajes, instancias de mapas, mecánicas de monstruos, sistemas de misiones y funciones de jugabilidad personalizadas. El código base en Java amplía el framework de servidor v83 estándar con mejoras de calidad de uso. Está dirigido a desarrolladores de servidores privados de MapleStory e investigadores de servidores de juegos.
