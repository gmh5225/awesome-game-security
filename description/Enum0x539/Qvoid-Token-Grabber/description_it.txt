Questo progetto è un toolkit .NET per il furto di credenziali e token, incentrato sulla raccolta di dati Discord e del browser.
Implementa l'esfiltrazione tramite webhook e include controlli anti-analisi come anti-debugging, anti-VM, anti-sandbox e anti-emulazione.
Le capacità principali comprendono l'harvesting di token Discord, l'estrazione di cookie e password del browser, l'acquisizione di screenshot, la raccolta di password Wi-Fi e il dirottamento degli appunti.
Il caso d'uso principale è la ricerca di sicurezza offensiva e di analisi malware sulle tecniche degli information stealer e sui flussi di abuso dei token di account.
