このプロジェクトは、Discord およびブラウザデータの収集に焦点を当てた .NET ベースの認証情報・トークン取得ツールキットです。
Webhook によるデータ持ち出しを実装し、アンチデバッグ、アンチ VM、アンチサンドボックス、アンチエミュレーションなどの解析妨害チェックを組み込んでいます。
主な機能には、Discord トークンの収穫、ブラウザの Cookie とパスワード抽出、スクリーンショット取得、Wi-Fi パスワード収集、クリップボードハイジャック動作が含まれます。
主な用途は、情報スティーラー手法およびアカウント・トークン悪用ワークフローに関する攻撃的セキュリティとマルウェア解析研究です。
