Этот проект — набор инструментов на .NET для сбора учётных данных и токенов с упором на данные Discord и браузеров.
Он реализует эксфильтрацию через webhook и включает проверки против анализа: антиотладку, анти-VM, антипесочницу и антиэмуляцию.
Ключевые возможности включают сбор Discord-токенов, извлечение cookie и паролей браузера, захват скриншотов, сбор паролей Wi-Fi и перехват буфера обмена.
Основной сценарий использования — исследования в области offensive security и анализа вредоносного ПО, посвящённые техникам стилеров информации и схемам злоупотребления токенами аккаунтов.
