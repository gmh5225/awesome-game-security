Este proyecto es un asistente de análisis IDA Pro sin interfaz gráfica que extrae cadenas y el pseudocódigo relacionado de binarios. Está implementado en Rust con bindings de idalib y utilidades de descompilación para procesar objetivos compatibles con Hex-Rays. La salida se organiza en directorios donde cada cadena se asocia a las funciones descompiladas que la referencian, lo que facilita el triage a escala. Está pensado para ingenieros inversos e investigadores de vulnerabilidades que necesitan pipelines rápidos de análisis estático.
