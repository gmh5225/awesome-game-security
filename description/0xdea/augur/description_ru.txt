Этот проект — headless-ассистент анализа для IDA Pro, который извлекает строки и связанный псевдокод из бинарных файлов. Он реализован на Rust с привязками idalib и вспомогательными средствами декомпиляции для обработки целей, поддерживаемых Hex-Rays. Вывод организован по каталогам, где каждая строка сопоставлена с декомпилированными функциями, которые на неё ссылаются, что упрощает triage в больших объёмах. Предназначен для реверс-инженеров и исследователей уязвимостей, которым нужны быстрые конвейеры статического анализа.
