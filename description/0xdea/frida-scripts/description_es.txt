Este proyecto es una colección de scripts de instrumentación Frida para el análisis dinámico en plataformas móviles y de escritorio. Incluye ayudas de trazado y enumeración para iOS, Android y Linux, junto con fragmentos reutilizables para escenarios habituales de hooks en tiempo de ejecución. Los scripts están escritos en JavaScript para Frida y cubren tareas como el trazado de métodos, el descubrimiento de clases y la experimentación orientada a bypass. Lo usan principalmente ingenieros de ingeniería inversa y testers de seguridad de aplicaciones durante el análisis en tiempo de ejecución en vivo.
