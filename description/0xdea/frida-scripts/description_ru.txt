Этот проект — набор скриптов инструментирования Frida для динамического анализа на мобильных и десктопных платформах. В него входят вспомогательные средства трассировки и перечисления для iOS, Android и Linux, а также переиспользуемые фрагменты для типичных сценариев runtime-хуков. Скрипты написаны на JavaScript для Frida и охватывают такие задачи, как трассировка методов, обнаружение классов и эксперименты, ориентированные на обход. В основном их используют reverse engineers и специалисты по безопасности приложений при живом анализе во время выполнения.
