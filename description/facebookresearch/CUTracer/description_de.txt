Dieses Projekt ist ein Tool zur dynamischen Binärinstrumentierung zum Trace und zur Analyse von CUDA-Kernel-Instruktionen.
Es trennt die leichte Datenerfassung (Instrumentierung) klar von der hostseitigen Verarbeitung (Analyse).
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats und dynamischer Binärinstrumentierung untersuchen.
