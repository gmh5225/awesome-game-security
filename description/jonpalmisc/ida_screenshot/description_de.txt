Dieses Projekt konzentriert sich auf ein Plugin zur hochauflösenden Screenshot-Aufnahme für IDA Pro.
Führen Sie make install aus (unter der Annahme, dass Sie macOS oder Linux nutzen), oder legen Sie die Datei screenshot.py andernfalls in Ihren IDA-Benutzer-Plugin-Ordner.
Es ist vor allem nützlich für Game-Security-Forscher und Reverse Engineers, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
