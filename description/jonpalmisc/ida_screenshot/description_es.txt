Este proyecto se centra en un plugin de captura de pantalla de alta resolución para IDA Pro.
Ejecuta make install (asumiendo que usas macOS o Linux), o coloca el archivo screenshot.py en la carpeta de plugins de usuario de IDA.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / plugins de IDA.
