Questo progetto si concentra su un plugin di cattura screenshot ad alta risoluzione per IDA Pro.
Esegui make install (assumendo che tu sia su macOS o Linux), oppure inserisci il file screenshot.py nella cartella dei plugin utente di IDA.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin IDA.
