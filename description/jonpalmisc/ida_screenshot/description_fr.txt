Ce projet se concentre sur un plugin de capture d'écran haute résolution pour IDA Pro.
Exécutez make install (en supposant que vous êtes sur macOS ou Linux), ou placez autrement le fichier screenshot.py dans le dossier des plugins utilisateur d'IDA.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
