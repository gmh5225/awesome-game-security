Этот проект посвящён плагину для захвата скриншотов высокого разрешения в IDA Pro.
Запустите make install (предполагается macOS или Linux) или поместите файл screenshot.py в папку пользовательских плагинов IDA.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / плагинов IDA.
