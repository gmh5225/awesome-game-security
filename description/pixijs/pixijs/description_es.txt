Este proyecto es la biblioteca 2D más rápida y ligera disponible para la web, funciona en todos los dispositivos y permite crear gráficos interactivos ricos y aplicaciones multiplataforma mediante WebGL y WebGPU.
Está escrita principalmente en TypeScript y JavaScript y se centra en el trabajo con shaders, el renderizado y los gráficos.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores de gráficos que trabajan en el área de motores de juego / código fuente.
