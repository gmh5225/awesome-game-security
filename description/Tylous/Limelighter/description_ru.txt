Этот проект — инструмент командной строки на Go для создания и использования сертификатов цифровой подписи кода для бинарных файлов Windows. Он может строить поддельный сертификатный материал на основе метаданных доменных сертификатов, упаковывать ключи в файлы PFX и подписывать исполняемые файлы или DLL через внешние утилиты подписи. Инструмент также поддерживает подпись с помощью существующих действительных сертификатов, предоставленных оператором. Основной сценарий использования — эксперименты по подписи в стиле red team и оборонительные исследования доверия и поведения обнаружения EDR.
