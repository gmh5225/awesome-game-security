Este proyecto es un dumper de Unreal Engine para Windows y un generador de SDK orientado a versiones concretas de UE4. Está implementado en C++ y usa acceso a la memoria de procesos, pattern scanning y análisis de estructuras del motor para localizar datos de nombres y objetos en tiempo de ejecución. Genera nombres, objetos y cabeceras SDK para los builds de juego inspeccionados, con manejo de versiones impulsado por configuración. Está pensado para la ingeniería inversa de juegos Unreal y el desarrollo de herramientas en investigación de seguridad de juegos.
