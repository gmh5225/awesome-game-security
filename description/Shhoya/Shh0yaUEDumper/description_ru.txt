Этот проект — дампер Unreal Engine для Windows и генератор SDK для определённых версий UE4. Он реализован на C++ и использует доступ к памяти процесса, сканирование сигнатур и разбор структур движка, чтобы находить данные имён и объектов во время выполнения. Он выводит имена, объекты и сгенерированные заголовки SDK для исследуемых сборок игр, с управлением версиями через конфигурацию. Предназначен для реверс-инжиниринга игр на Unreal и разработки инструментов в исследованиях безопасности игр.
