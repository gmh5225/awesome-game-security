Questo progetto è un dumper Unreal Engine per Windows e un generatore di SDK per versioni specifiche di UE4. È implementato in C++ e usa accesso alla memoria di processo, pattern scanning e analisi delle strutture del motore per individuare dati di nomi e oggetti a runtime. Produce nomi, oggetti e header SDK generati per le build di gioco ispezionate, con gestione delle versioni basata sulla configurazione. È pensato per il reverse engineering di giochi Unreal e lo sviluppo di strumenti nella ricerca sulla sicurezza dei giochi.
