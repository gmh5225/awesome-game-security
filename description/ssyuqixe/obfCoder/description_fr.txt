Ce projet est un outil d'obfuscation de code C++ qui applique diverses techniques de transformation au code source et aux binaires compilés. Il inclut une prise en charge de build basée sur CMake, des exemples d'expériences, une intégration de la bibliothèque cryptographique OpenSSL, ainsi que des programmes de démonstration montrant les résultats avant et après obfuscation. Il est principalement utile aux chercheurs en protection logicielle qui étudient les techniques d'obfuscation de code et les transformations anti-rétro-ingénierie.
