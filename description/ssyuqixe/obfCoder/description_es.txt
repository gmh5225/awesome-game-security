Este proyecto es una herramienta de ofuscación de código C++ que aplica diversas técnicas de transformación al código fuente y a binarios compilados. Incluye soporte de compilación basado en CMake, ejemplos de experimentos, integración con la biblioteca criptográfica OpenSSL y programas de demostración que muestran resultados antes y después de la ofuscación. Es especialmente útil para investigadores de protección de software que estudian técnicas de ofuscación de código y transformaciones anti-ingeniería inversa.
