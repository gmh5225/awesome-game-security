Plugin x64dbg avec une interface Qt qui surveille les exports DLL des processus débogués, construit avec le x64dbg Plugin SDK (bridgemain, TitanEngine, XEDParse). Suit les tables d'export des modules chargés et affiche en temps réel les noms de fonctions, ordinaux et adresses, avec des aides aux internals Windows pour résoudre les noms d'appareils et les informations de processus.
