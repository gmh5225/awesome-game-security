Плагин x64dbg с GUI на Qt для мониторинга экспортов DLL в отлаживаемых процессах, собранный с x64dbg Plugin SDK (bridgemain, TitanEngine, XEDParse). Отслеживает таблицы экспорта загруженных модулей и в реальном времени показывает имена функций, ординалы и адреса; также включает вспомогательные средства Windows internals для разрешения имён устройств и сведений о процессах.
