x64dbg-Plugin mit Qt-basierter GUI zur Überwachung von DLL-Exporten in debuggten Prozessen, gebaut gegen das x64dbg Plugin SDK (bridgemain, TitanEngine, XEDParse). Verfolgt die Exporttabellen geladener Module und zeigt Funktionsnamen, Ordinale und Adressen in Echtzeit an, mit Windows-Internals-Hilfen zur Auflösung von Gerätenamen und Prozessinformationen.
