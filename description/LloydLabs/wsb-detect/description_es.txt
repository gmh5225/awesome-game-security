wsb-detect es una biblioteca C y un programa de ejemplo para detectar si el código se ejecuta dentro de Windows Sandbox.
Implementa múltiples comprobaciones de huella digital, como procesos específicos del sandbox, nombres de usuario, rutas de dispositivos, sufijos DNS, artefactos del registro e indicios de temporización.
El proyecto expone funciones de detección modulares que pueden combinarse según la tolerancia a falsos positivos.
Su principal caso de uso es la investigación anti-análisis y la concienciación del entorno para estudios de malware, simulación de red team y pruebas de contramedidas defensivas.
