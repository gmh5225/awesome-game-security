wsb-detect — это библиотека на C и пример программы для определения, выполняется ли код внутри Windows Sandbox.
Она реализует несколько проверок по отпечаткам окружения: специфичные для песочницы процессы, имена пользователей, пути устройств, DNS-суффиксы, артефакты реестра и временные признаки.
Проект предоставляет модульные функции обнаружения, которые можно комбинировать в зависимости от допустимого уровня ложных срабатываний.
Основной сценарий использования — исследования по антианализу и определение окружения для изучения вредоносного ПО, red-team симуляций и тестирования защитных контрмер.
