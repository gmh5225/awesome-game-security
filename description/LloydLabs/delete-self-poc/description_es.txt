delete-self-poc es una prueba de concepto en C que muestra cómo un ejecutable en ejecución o bloqueado puede eliminarse a sí mismo del disco en Windows.
La técnica renombra el flujo de datos principal del archivo y luego establece flags de disposición mediante las API SetFileInformationByHandle.
Demuestra el orden de los handles, la semántica de eliminación y casos límite prácticos en torno a la eliminación de archivos bloqueados.
Su principal caso de uso es la investigación de internals de Windows a bajo nivel, relevante para antiforense, limpieza segura e ingeniería de detección defensiva.
