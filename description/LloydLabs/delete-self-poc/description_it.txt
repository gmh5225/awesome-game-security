delete-self-poc è una proof of concept in C che mostra come un eseguibile in esecuzione o bloccato possa eliminarsi da solo dal disco su Windows.
La tecnica rinomina lo stream di dati primario del file e poi imposta i flag di disposition tramite le API SetFileInformationByHandle.
Dimostra la sequenza degli handle, la semantica di eliminazione e casi limite pratici relativi alla rimozione di file bloccati.
Il suo caso d'uso principale è la ricerca a basso livello sugli internals di Windows, rilevante per anti-forensics, pulizia sicura e detection engineering difensivo.
