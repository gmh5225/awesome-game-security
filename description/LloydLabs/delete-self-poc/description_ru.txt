delete-self-poc — это доказательство концепции на C, показывающее, как выполняемый или заблокированный исполняемый файл может удалить себя с диска в Windows.
Метод переименовывает основной поток данных файла, а затем устанавливает флаги disposition через API SetFileInformationByHandle.
Он демонстрирует порядок работы с дескрипторами, семантику удаления и практические пограничные случаи при удалении заблокированных файлов.
Основной сценарий применения — исследование низкоуровневых внутренних механизмов Windows, актуальное для антифорензики, безопасной очистки и защитного detection engineering.
