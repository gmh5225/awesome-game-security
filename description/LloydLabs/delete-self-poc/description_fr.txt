delete-self-poc est une preuve de concept en C qui montre comment un exécutable en cours d'exécution ou verrouillé peut se supprimer du disque sous Windows.
La technique renomme le flux de données principal du fichier, puis définit des indicateurs de disposition via les API SetFileInformationByHandle.
Elle illustre l'ordonnancement des handles, la sémantique de suppression et des cas limites concrets autour du retrait de fichiers verrouillés.
Son principal cas d'usage est la recherche sur les internals Windows de bas niveau, pertinente pour l'anti-forensics, le nettoyage sécurisé et l'ingénierie de détection défensive.
