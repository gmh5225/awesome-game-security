Инструмент разработчика от Block, который через JVMTI внедряет код в любой отлаживаемый процесс Android (API 26+) без изменения его APK, обеспечивая живой хукинг методов, инспекцию объектов кучи и вызовы внутренних API через систему плагинов на Kotlin/Java; задержка первого подключения — менее трёх секунд, последующие соединения — менее секунды.
