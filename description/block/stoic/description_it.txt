Strumento per sviluppatori di Block che inietta codice in qualsiasi processo Android debuggabile (API 26+) tramite JVMTI senza modificarne l'APK, consentendo hooking live dei metodi, ispezione degli oggetti nell'heap e chiamate ad API interne tramite un sistema di plugin Kotlin/Java, con latenza di primo attach inferiore a tre secondi e connessioni successive sotto il secondo.
