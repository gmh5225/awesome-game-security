Herramienta de desarrollo de Block que inyecta código en cualquier proceso Android depurable (API 26+) mediante JVMTI sin modificar su APK, permitiendo el hooking de métodos en vivo, la inspección de objetos del heap y llamadas a API internas a través de un sistema de plugins Kotlin/Java, con una latencia de primer adjunto inferior a tres segundos y conexiones posteriores de menos de un segundo.
