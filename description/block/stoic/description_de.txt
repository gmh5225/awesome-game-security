Entwicklerwerkzeug von Block, das über JVMTI Code in jeden debuggbaren Android-Prozess (API 26+) einschleust, ohne dessen APK zu verändern, und über ein Kotlin/Java-Plugin-System Live-Method-Hooking, Heap-Objektinspektion sowie interne API-Aufrufe ermöglicht – mit einer First-Attach-Latenz unter drei Sekunden und nachfolgenden Verbindungen im Subsekundenbereich.
