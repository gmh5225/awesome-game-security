Este proyecto es un importador perezoso C++ solo de cabeceras para resolver módulos y exportaciones de API en tiempo de ejecución.
Está diseñado para evitar entradas en la tabla de importación estática, evitar cadenas en texto plano y mantener el código generado muy pequeño.
La biblioteca admite modos de resolución seguros, en caché y reenviados, y aleatoriza los hashes por compilación.
Se usa habitualmente en herramientas resistentes a la ingeniería inversa y en investigación de seguridad de juegos.
