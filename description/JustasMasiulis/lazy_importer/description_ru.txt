Этот проект — header-only C++ lazy importer для разрешения модулей и экспортов API во время выполнения.
Он спроектирован так, чтобы избегать записей в статической таблице импорта, избегать строк в открытом виде и держать сгенерированный код очень компактным.
Библиотека поддерживает безопасный, кэшируемый и перенаправляемый режимы разрешения и рандомизирует хеши при каждой сборке.
Её часто используют в инструментах, устойчивых к реверс-инжинирингу, и в исследованиях безопасности игр.
