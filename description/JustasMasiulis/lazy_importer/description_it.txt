Questo progetto è un lazy importer C++ header-only per risolvere moduli ed export API a runtime.
È progettato per evitare voci nella tabella di import statica, evitare stringhe in chiaro e mantenere il codice generato molto compatto.
La libreria supporta modalità di risoluzione sicure, con cache e inoltrate, e randomizza gli hash a ogni build.
È comunemente usata in strumenti resistenti al reverse engineering e nella ricerca sulla sicurezza dei giochi.
