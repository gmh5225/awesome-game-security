Этот проект — библиотека C++17 для шифрования строк на этапе компиляции.
Она использует векторизованные операции SSE или AVX и встраиваемые хелперы расшифровки, чтобы использование в runtime-коде оставалось лёгким.
Её дизайн стремится не оставлять данные строк в обычных секциях только для чтения и генерировать ключи во время компиляции.
Она широко применяется для обфускации в инструментах безопасности, задачах по реверс-инжинирингу и экспериментах в области безопасности игр.
