Este proyecto es una prueba de concepto en C++ para Windows orientada a detectar hooking o monitorización de acceso basados en EPT.
Prueba cómo se comporta REP MOVS ante fallos frente a una ejecución ininterrumpida y usa patrones de sobrescritura como señal.
El repositorio incluye una implementación compacta en un solo archivo, con configuración de memoria ejecutable y lógica de manejo de excepciones.
Está pensado principalmente para investigación sobre anti-cheat y detección de hipervisores.
