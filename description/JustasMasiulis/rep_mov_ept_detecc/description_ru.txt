Этот проект — proof of concept на Windows C++ для обнаружения хукинга или мониторинга доступа на основе EPT.
Он проверяет поведение REP MOVS при сбоях (faults) по сравнению с непрерывным выполнением и использует шаблоны перезаписи как сигнал.
Репозиторий содержит компактную однофайловую реализацию с настройкой исполняемой памяти и логикой обработки исключений.
В первую очередь предназначен для исследований в области античита и обнаружения гипервизоров.
