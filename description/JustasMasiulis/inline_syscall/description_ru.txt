Этот проект — header-only библиотека на C++ для inline-генерации прямых системных вызовов Windows.
Она предоставляет инициализацию и макро-обёртки, позволяющие вызывать нативные процедуры без обычной таблицы импорта.
Реализация ориентирована на компактный, встраиваемый машинный код и низкие накладные расходы на x64 Windows.
В основном применяется в низкоуровневом системном программировании, экспериментах по anti-hooking и исследованиях игровой безопасности.
