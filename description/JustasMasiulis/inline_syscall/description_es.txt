Este proyecto es una biblioteca C++ solo de encabezados para generar llamadas al sistema de Windows directas en línea.
Proporciona inicialización y envoltorios basados en macros para invocar rutinas nativas sin usar la tabla de importación habitual.
La implementación se centra en código máquina compacto e inlinable, con bajo overhead en objetivos Windows x64.
Se usa principalmente en programación de sistemas de bajo nivel, experimentos anti-hooking e investigación de seguridad en juegos.
