Questo progetto è una libreria C++ header-only per generare chiamate di sistema Windows dirette in inline.
Fornisce inizializzazione e wrapper basati su macro per invocare routine native senza usare la normale tabella di import.
L’implementazione punta a codice macchina compatto e inlinabile, con basso overhead su target Windows x64.
Si usa soprattutto nella programmazione di sistema a basso livello, negli esperimenti anti-hooking e nella ricerca sulla sicurezza dei giochi.
