Este proyecto se centra en una biblioteca moderna y fácil de usar para Vulkan.
Este framework ligero está escrito en un C++23 limpio y busca, en la medida de lo posible, un lanzamiento continuo modular.
Resulta especialmente útil para programadores de gráficos de bajo nivel y desarrolladores de motores centrados en el rendimiento que trabajan en el ámbito de Vulkan / API.
