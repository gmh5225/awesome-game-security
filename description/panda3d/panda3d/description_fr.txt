Ce projet est un moteur de jeu, un framework de rendu 3D et de développement de jeux pour les programmes Python et C++.
Panda3D est open source et gratuit pour tout usage, y compris commercial, grâce à sa licence libérale.
Il est surtout utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
