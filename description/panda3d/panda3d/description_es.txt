Este proyecto es un motor de juegos, un framework de renderizado 3D y desarrollo de juegos para programas en Python y C++.
Panda3D es de código abierto y gratuito para cualquier propósito, incluidos los comerciales, gracias a su licencia liberal.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores de gráficos que trabajan en el área de motores de juego / código fuente.
