Этот проект — игровой движок, фреймворк для 3D-рендеринга и разработки игр для программ на Python и C++.
Panda3D является открытым и бесплатным для любых целей, включая коммерческие, благодаря либеральной лицензии.
Он в основном полезен разработчикам игр, программистам движков и исследователям графики, работающим в области игровых движков / исходного кода.
