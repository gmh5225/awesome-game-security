Этот проект — чит для Escape from Tarkov, который использует потоковый/внешний подход для отображения игровой информации вне окна игры. Он внешне читает память игры EFT через драйвер и отрисовывает радар, позиции игроков, информацию о луте и другие данные ESP на отдельном дисплее или оверлее. Такой подход позволяет избежать обнаружения внутренней инъекции. Предназначен для исследователей безопасности игр, изучающих архитектуры внешних/потоковых читов для игр на Unity с защитой BattlEye.
