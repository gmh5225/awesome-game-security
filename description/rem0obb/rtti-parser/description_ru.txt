Этот проект посвящён IDA-скрипту для разбора RTTI-информации в исполняемых файлах с поддержкой IDA 9.2.
Не удаётся переименовать байт по адресу 'xxx', потому что это имя уже используется в программе.
Он в основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / плагинов IDA.
