Questo progetto si concentra su uno script IDA per analizzare le informazioni RTTI negli eseguibili, con supporto per IDA 9.2.
Impossibile rinominare il byte all'indirizzo 'xxx' perché il nome è già usato nel programma.
È principalmente utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin IDA.
