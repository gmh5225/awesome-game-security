Ce projet se concentre sur un script IDA destiné à analyser les informations RTTI dans les exécutables, avec prise en charge d'IDA 9.2.
Impossible de renommer l'octet à l'adresse 'xxx' car ce nom est déjà utilisé dans le programme.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
