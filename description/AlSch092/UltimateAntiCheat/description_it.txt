Questo progetto è un framework anti-cheat open source per Windows, realizzato principalmente in C++. Si concentra sul rilevamento e sulla prevenzione in user mode di vettori comuni di manomissione dei giochi, come debugging, memory patching e modifiche sospette a runtime. Il codice include networking heartbeat client-server opzionale e percorsi configurabili per deployment ibridi user mode con assistenza kernel. È pensato principalmente come riferimento didattico per sviluppatori e ricercatori che costruiscono o valutano sistemi anti-cheat.
