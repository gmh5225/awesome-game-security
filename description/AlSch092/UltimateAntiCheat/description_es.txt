Este proyecto es un framework anti-cheat de Windows de código abierto, construido principalmente en C++. Se centra en la detección y prevención en modo usuario de vectores habituales de manipulación de juegos, como depuración, parcheo de memoria y modificaciones sospechosas en tiempo de ejecución. El código incluye redes de heartbeat cliente-servidor opcionales y rutas configurables para despliegues híbridos de modo usuario con asistencia del kernel. Está pensado principalmente como referencia educativa para desarrolladores e investigadores que construyen o evalúan sistemas anti-cheat.
