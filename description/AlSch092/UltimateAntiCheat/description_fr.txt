Ce projet est un framework anti-cheat Windows open source, principalement écrit en C++. Il se concentre sur la détection et la prévention en mode utilisateur des vecteurs courants de falsification de jeux, tels que le débogage, le patch mémoire et les modifications suspectes à l'exécution. Le code source inclut une mise en réseau heartbeat client-serveur optionnelle et des chemins configurables pour des déploiements hybrides mode utilisateur plus assistance noyau. Il est principalement destiné à servir de référence éducative pour les développeurs et chercheurs qui construisent ou évaluent des systèmes anti-cheat.
