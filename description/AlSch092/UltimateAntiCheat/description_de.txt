Dieses Projekt ist ein Open-Source-Windows-Anti-Cheat-Framework, das hauptsächlich in C++ entwickelt wurde. Es konzentriert sich auf die Erkennung und Verhinderung gängiger Spielmanipulationen im User Mode, etwa Debugging, Memory Patching und verdächtige Laufzeitänderungen. Die Codebasis umfasst optionales Client-Server-Heartbeat-Networking sowie konfigurierbare Pfade für hybride Deployments mit User Mode und Kernel-Unterstützung. Es dient vor allem als Bildungsreferenz für Entwickler und Forscher, die Anti-Cheat-Systeme aufbauen oder bewerten.
