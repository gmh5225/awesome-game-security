Этот проект представляет собой технику драйвер плюс DLL для получения дескрипторов процессов с обходом kernel-колбэков на создание handle. Реализован на C и C++ и использует ObOpenPointerToObject в режиме ядра, а пользовательский хук OpenProcess перенаправляет запросы через IOCTL. Это позволяет отладчикам и подобным инструментам подключаться к процессам, защищённым колбэками, где обычные пути создания handle заблокированы, с учётом ограничений вроде целей PPL. Проект в основном предназначен для исследований безопасности Windows, связанных с механизмами защиты handle в античитах и EDR.
