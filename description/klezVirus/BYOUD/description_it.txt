Questo progetto è un framework per lo stack spoofing x64 su Windows.
Adotta un approccio completamente opposto allo stack spoofing classico, manipolando i metadati unwind per nascondere porzioni arbitrarie della call chain in debugger ed EDR.
È principalmente utile per ricercatori di game security e reverse engineer che studiano tecniche offensive nell'ambito di cheat / spoof dello stack.
