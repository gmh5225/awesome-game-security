Этот инструмент запускает процессы Windows с заданными уровнями Protected Process Light (PPL).
Он реализован на C++ и предоставляет режимы командной строки для уровней защиты, таких как WinTCB, Windows, Antimalware и LSA.
Проект сосредоточен на практическом обращении с семантикой защиты процессов и на том, как меняется поведение запуска исполняемых файлов в разных режимах PPL.
Он в основном полезен исследователям внутренних механизмов Windows и безопасности, которые проверяют границы защищённых процессов, совместимость инструментов и оборонительные допущения.
