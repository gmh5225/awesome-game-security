Этот проект — однозаголовочная библиотека графики и утилит на C для прототипирования, демо-сцен и небольших игр. Она предоставляет кроссплатформенные модули для окон, утилит рендеринга, генерации шума, математических помощников, структур данных, сетей и потоков. В репозитории также есть обёртки и примеры на Rust и Python, а также опциональные системы вроде bloom, аудио и редактирования узлов. Библиотека рассчитана на разработчиков, которым нужен лёгкий инструментарий на C/C++ для быстрых графических экспериментов.
