Questo progetto è una libreria C di grafica e utilità a header singolo, pensata per prototipazione, scene demo e piccoli giochi. Fornisce moduli multipiattaforma per la gestione delle finestre, utilità di rendering, generazione di rumore, helper matematici, strutture dati, networking e threading. Il repository include anche wrapper ed esempi in Rust e Python, oltre a sistemi opzionali come bloom, audio e node editing. È rivolta agli sviluppatori che cercano un toolkit C/C++ leggero per sperimentare rapidamente con la grafica.
