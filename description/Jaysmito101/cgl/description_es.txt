Este proyecto es una biblioteca C de gráficos y utilidades de cabecera única, pensada para prototipado, escenas de demostración y juegos pequeños. Ofrece módulos multiplataforma para ventanas, utilidades de renderizado, generación de ruido, ayudas matemáticas, estructuras de datos, redes e hilos. El repositorio también incluye envoltorios y ejemplos en Rust y Python, además de sistemas opcionales como bloom, audio y edición de nodos. Está destinada a desarrolladores que buscan un kit de herramientas C/C++ ligero para experimentar rápidamente con gráficos.
