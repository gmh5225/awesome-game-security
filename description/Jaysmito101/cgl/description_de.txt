Dieses Projekt ist eine Single-Header-C-Grafik- und Hilfsbibliothek für Prototyping, Demo-Szenen und kleine Spiele. Sie bietet plattformübergreifende Module für Fensterverwaltung, Rendering-Hilfen, Rauschgenerierung, Mathematik-Helfer, Datenstrukturen, Netzwerk und Threading. Das Repository enthält außerdem Wrapper und Beispiele in Rust und Python sowie optionale Systeme wie Bloom, Audio und Node-Editing. Es richtet sich an Entwickler, die ein schlankes C/C++-Toolkit für schnelle Grafikexperimente suchen.
