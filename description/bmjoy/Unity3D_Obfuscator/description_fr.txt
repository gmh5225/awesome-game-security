Ce projet est un dépôt classé dans anti cheat / game engine protection:unity et s'articule autour de Unity3D Obfuscator.
Il est principalement écrit en C# et se concentre sur Unity et le débogage.
Il est surtout utile aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant dans le domaine anti cheat / game engine protection:unity.
