Этот проект — репозиторий из раздела anti cheat / game engine protection:unity, посвящённый Unity3D Obfuscator.
Он в основном написан на C# и сосредоточен на Unity и отладке.
Он особенно полезен инженерам античита и исследователям защитной безопасности, работающим в области anti cheat / game engine protection:unity.
