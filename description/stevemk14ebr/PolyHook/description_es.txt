Este proyecto se centra en PolyHook — una biblioteca de hooking x86/x64 que proporciona una interfaz abstracta en C++11 para diversos métodos de hooking.
Ejemplos: el archivo Tests.cpp ofrece ejemplos de cada tipo de método de hooking.
Es principalmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de los cheats y el hooking.
