Questo progetto si concentra su PolyHook — una libreria di hooking x86/x64 che fornisce un'interfaccia astratta C++11 per vari metodi di hooking.
Esempi: il file Tests.cpp fornisce esempi per ogni tipo di metodo di hooking.
È principalmente utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat e hooking.
