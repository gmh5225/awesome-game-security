Ce projet se concentre sur PolyHook — une bibliothèque de hooking x86/x64 qui fournit une interface C++11 abstraite pour diverses méthodes de hooking.
Exemples : le fichier Tests.cpp fournit des exemples pour chaque type de méthode de hooking.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats et du hooking.
