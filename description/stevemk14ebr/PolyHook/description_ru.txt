Этот проект посвящён PolyHook — библиотеке хуков для x86/x64, предоставляющей абстрактный интерфейс C++11 для различных методов хукинга.
Примеры: файл Tests.cpp содержит примеры для каждого типа методов хукинга.
В первую очередь он полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов и хуков.
