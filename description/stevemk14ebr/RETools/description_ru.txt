Этот проект — курируемая подборка инструментов реверс-инжиниринга и безопасности, организованная по категориям. В нём каталогизированы дизассемблеры, отладчики, декомпиляторы, шестнадцатеричные редакторы, сетевые анализаторы, фреймворки анализа бинарных файлов, инструменты песочниц и утилиты анализа вредоносного ПО с описаниями и ссылками. Репозиторий служит справочным руководством для сборки набора инструментов реверс-инжиниринга. Он предназначен для исследователей безопасности и реверс-инженеров, которые ищут и выбирают подходящие инструменты для рабочих процессов анализа бинарных файлов.
