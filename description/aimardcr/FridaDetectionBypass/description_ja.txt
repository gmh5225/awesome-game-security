FridaDetectionBypass は、Android ターゲット上で一般的な Frida 検知手法を回避するための実践的なテクニック集です。デバッガ検査、デフォルトポート検知、プロトコルフィンガープリンティング、メモリアーティファクト走査に対する対策を文書化し、実例で示します。リポジトリは JavaScript フック、コマンドライン設定、ネイティブ Android/JNI コンポーネントを組み合わせ、標準の gadget 構成だけに頼らないスクリプト注入ワークフローを提供します。モバイル逆向エンジニアリングおよび改ざん防止研究において、分析者がインスツルメンテーション耐性を評価または強化するために用いることを想定しています。
