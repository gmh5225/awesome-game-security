Ce projet est un pack de comportements anti-triche pour Minecraft Bedrock, conçu pour les Realms, les mondes et les serveurs.
Il est principalement construit en JavaScript et JSON à l’aide de l’API Bedrock Scripting et de fichiers de commandes/fonctions.
Le pack inclut de nombreux modules de détection pour le combat, les déplacements, l’abus de paquets, le spam de chat, le comportement de scaffold et d’autres triches courantes, ainsi que des commandes de modération et des outils de statistiques joueurs.
Son cas d’usage principal est la détection de triches côté serveur et l’amélioration du flux de modération pour les administrateurs Bedrock.
