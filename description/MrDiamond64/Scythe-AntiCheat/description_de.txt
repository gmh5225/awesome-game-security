Dieses Projekt ist ein Minecraft-Bedrock-Anti-Cheat-Verhaltenpack für Realms, Welten und Server.
Es ist hauptsächlich mit JavaScript und JSON über die Bedrock Scripting API sowie Kommando-/Funktionsdateien aufgebaut.
Das Pack enthält zahlreiche Erkennungsmodule für Kampf, Bewegung, Paketmissbrauch, Chat-Spam, Scaffold-Verhalten und andere gängige Cheats sowie Moderationsbefehle und Spielerstatistik-Tools.
Der primäre Einsatzzweck ist serverseitige Cheat-Erkennung und die Verbesserung des Moderationsworkflows für Bedrock-Administratoren.
