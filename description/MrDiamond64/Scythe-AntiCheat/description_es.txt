Este proyecto es un paquete de comportamientos anti-trampas para Minecraft Bedrock, diseñado para realms, mundos y servidores.
Está construido principalmente con JavaScript y JSON mediante la Bedrock Scripting API y archivos de comandos/funciones.
El paquete incluye muchos módulos de detección para combate, movimiento, abuso de paquetes, spam de chat, comportamiento de scaffold y otras trampas comunes, además de comandos de moderación y herramientas de estadísticas de jugadores.
Su caso de uso principal es la detección de trampas en el servidor y la mejora del flujo de moderación para administradores de Bedrock.
