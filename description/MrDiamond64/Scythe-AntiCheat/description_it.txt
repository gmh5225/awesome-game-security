Questo progetto è un behavior pack anti-cheat per Minecraft Bedrock, pensato per realm, mondi e server.
È realizzato principalmente con JavaScript e JSON usando la Bedrock Scripting API e file di comandi/funzioni.
Il pack include molti moduli di rilevamento per combattimento, movimento, abuso di pacchetti, spam in chat, comportamento scaffold e altri cheat comuni, oltre a comandi di moderazione e strumenti per le statistiche dei giocatori.
Il caso d’uso principale è il rilevamento cheat lato server e il miglioramento del flusso di moderazione per gli amministratori Bedrock.
