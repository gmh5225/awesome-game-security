Этот проект — behavior pack античита для Minecraft Bedrock, предназначенный для Realms, миров и серверов.
Он в основном написан на JavaScript и JSON с использованием Bedrock Scripting API и файлов команд/функций.
Пакет включает множество модулей обнаружения для боя, перемещения, злоупотребления пакетами, спама в чате, поведения scaffold и других распространённых читов, а также команды модерации и инструменты статистики игроков.
Основной сценарий использования — серверное обнаружение читов и улучшение рабочего процесса модерации для администраторов Bedrock.
