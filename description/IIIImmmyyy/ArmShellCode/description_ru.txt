Этот проект — фреймворк ARM64 shellcode для Android, обеспечивающий позиционно-независимое выполнение кода с разрешением символов на базе Dobby. Кодовая база на C включает разбор ELF, разбор proc maps, обёртки syscall и модульный загрузчик shellcode с пользовательскими скриптами линкера для генерации автономных shellcode-пейлоадов. Он в первую очередь полезен исследователям безопасности Android и разработчикам эксплойтов, изучающим построение ARM64 shellcode и техники внедрения кода во время выполнения.
