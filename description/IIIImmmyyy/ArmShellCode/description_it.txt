Questo progetto è un framework di shellcode ARM64 per Android che fornisce esecuzione di codice indipendente dalla posizione con risoluzione dei simboli basata su Dobby. Il codebase in C include parsing ELF, parsing di proc maps, wrapper per syscall e un loader di shellcode modulare con script del linker personalizzati per generare payload shellcode autonomi. È principalmente utile a ricercatori di sicurezza Android e sviluppatori di exploit che studiano la costruzione di shellcode ARM64 e le tecniche di code injection a runtime.
