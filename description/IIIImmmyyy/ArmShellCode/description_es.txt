Este proyecto es un framework de shellcode ARM64 para Android que proporciona ejecución de código independiente de la posición con resolución de símbolos basada en Dobby. La base de código en C incluye análisis ELF, análisis de proc maps, wrappers de syscalls y un cargador modular de shellcode con scripts de linker personalizados para generar payloads de shellcode independientes. Es especialmente útil para investigadores de seguridad de Android y desarrolladores de exploits que estudian la construcción de shellcode ARM64 y técnicas de inyección de código en tiempo de ejecución.
