Ce projet est un framework de shellcode ARM64 pour Android qui offre une exécution de code indépendante de la position avec une résolution de symboles basée sur Dobby. La base de code en C inclut l'analyse ELF, l'analyse des proc maps, des wrappers de syscalls, ainsi qu'un chargeur de shellcode modulaire avec des scripts de linker personnalisés pour générer des payloads shellcode autonomes. Il est principalement utile aux chercheurs en sécurité Android et aux développeurs d'exploits qui étudient la construction de shellcode ARM64 et les techniques d'injection de code à l'exécution.
