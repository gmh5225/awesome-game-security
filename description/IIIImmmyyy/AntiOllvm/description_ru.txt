AntiOllvm — это фреймворк симулированного выполнения Arm64 для снятия flattening потока управления OLLVM. Он выявляет шаблоны диспетчера и восстанавливает полные графы потока управления if-else из обфусцированных функций. Основной инструмент написан на C# и сопровождается Python-скриптами для извлечения CFG в IDA, генерации машинного кода через Keystone и пересборки CFG. Предназначен в первую очередь для реверс-инженеров, анализирующих обфусцированные бинарные файлы в исследованиях безопасности и защиты игр.
