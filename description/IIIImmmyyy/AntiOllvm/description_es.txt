AntiOllvm es un framework de ejecución simulada Arm64 para eliminar el aplanamiento del flujo de control de OLLVM. Identifica patrones de dispatcher y reconstruye grafos de flujo de control if-else completos a partir de funciones ofuscadas. La herramienta principal está escrita en C#, con scripts Python complementarios para la extracción de CFG en IDA, la generación de código máquina mediante Keystone y la reconstrucción del CFG. Está destinado principalmente a ingenieros de ingeniería inversa que analizan binarios ofuscados en investigación de seguridad y protección de juegos.
