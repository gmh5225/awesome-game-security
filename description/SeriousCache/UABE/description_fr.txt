Ce projet est un éditeur de bureau pour les fichiers Unity .assets et AssetBundle, compatible avec plusieurs versions du moteur. Il est principalement implémenté en C++ avec une interface basée sur Win32, une prise en charge de la compilation CMake et une architecture de plugins. Les plugins inclus gèrent les textures, les assets texte, les clips audio, les maillages et les données de ressources brutes, avec import et export vers des formats courants tels que PNG, WAV, OBJ et DAE. Son cas d’usage principal est le modding Unity, l’inspection d’assets et les workflows de recherche sur le contenu de jeux.
