Questo progetto è un editor desktop per file Unity .assets e AssetBundle su più versioni del motore. È implementato principalmente in C++ con un’interfaccia basata su Win32, supporto alla build CMake e un’architettura a plugin. I plugin inclusi gestiscono texture, asset di testo, clip audio, mesh e dati di risorse grezze, con importazione ed esportazione verso formati comuni come PNG, WAV, OBJ e DAE. Il caso d’uso principale è il modding di Unity, l’ispezione degli asset e i flussi di lavoro di ricerca sui contenuti di gioco.
