Кроссплатформенный шестнадцатеричный редактор и рабочая среда анализа двоичных данных на базе Qt, объединяющие в одном GUI дизассемблирование Capstone, графики энтропии, сканирование сигнатур binwalk, сравнение файлов, визуализацию byte-map, извлечение строк и криптографическое хеширование. Поддерживает определения шаблонов форматов файлов на основе YAML (ELF, PE, JPEG, PDF, WAV) для структурированной подсветки наложением и включает виджет QHexEdit с отменой/повтором, поиском и цветовой разметкой регионов.
