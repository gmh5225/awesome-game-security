Este proyecto es un emulador basado en hipervisor para ejecutables Windows x64 en modo usuario que utiliza la API de Windows Hypervisor Platform.
Lleva el concepto más lejos y permite emular ejecutables Windows x64 dentro de un entorno virtualizado.
Resulta especialmente útil para desarrolladores de emuladores e investigadores de la plataforma Windows que trabajan en el área de emulación de Windows.
