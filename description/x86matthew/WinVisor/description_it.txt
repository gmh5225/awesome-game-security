Questo progetto è un emulatore basato su hypervisor per eseguibili Windows x64 in modalità utente che utilizza l'API Windows Hypervisor Platform.
Porta avanti questo concetto e consente di emulare eseguibili Windows x64 all'interno di un ambiente virtualizzato.
È principalmente utile per gli sviluppatori di emulatori e per i ricercatori della piattaforma Windows che operano nell'ambito dell'emulazione Windows.
