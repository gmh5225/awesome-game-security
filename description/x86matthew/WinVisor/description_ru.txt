Этот проект — эмулятор на основе гипервизора для пользовательских исполняемых файлов Windows x64, использующий Windows Hypervisor Platform API.
Он развивает эту идею дальше и позволяет эмулировать исполняемые файлы Windows x64 в виртуализированной среде.
В первую очередь он полезен разработчикам эмуляторов и исследователям платформы Windows, работающим в области эмуляции Windows.
