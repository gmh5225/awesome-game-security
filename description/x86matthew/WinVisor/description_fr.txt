Ce projet est un émulateur basé sur un hyperviseur pour les exécutables Windows x64 en mode utilisateur, utilisant l'API Windows Hypervisor Platform.
Il pousse ce concept plus loin et permet d'émuler des exécutables Windows x64 dans un environnement virtualisé.
Il s'adresse principalement aux développeurs d'émulateurs et aux chercheurs de la plateforme Windows travaillant dans le domaine de l'émulation Windows.
