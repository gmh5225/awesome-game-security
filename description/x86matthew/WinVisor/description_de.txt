Dieses Projekt ist ein Hypervisor-basierter Emulator für Windows-x64-Benutzermodus-Ausführungsdateien unter Verwendung der Windows Hypervisor Platform API.
Es führt das Konzept weiter und ermöglicht die Emulation von Windows-x64-Ausführungsdateien in einer virtualisierten Umgebung.
Es ist vor allem für Emulator-Entwickler und Windows-Plattformforscher nützlich, die im Bereich der Windows-Emulation arbeiten.
