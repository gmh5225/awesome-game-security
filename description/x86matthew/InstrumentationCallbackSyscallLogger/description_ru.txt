Этот проект посвящён Instrumentation Callback.
Этот callback вызывается при каждом возврате из системного вызова ядра, давая коду в пользовательском режиме возможность проверить каждый вызов до возобновления выполнения.
Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / Windows Ring3 callback.
