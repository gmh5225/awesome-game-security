Este proyecto se centra en Instrumentation Callback.
Este callback se invoca en cada retorno de una syscall del kernel, dando al código en modo usuario la oportunidad de inspeccionar cada llamada antes de que se reanude la ejecución.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de callbacks anti-cheat / Windows Ring3.
