Este proyecto es una implementación independiente de la interfaz de tiempo de ejecución Zygisk para entornos Android con root. Busca compatibilidad con configuraciones KernelSU, Apatch y Magisk, ofreciendo una vía alternativa de carga de módulos cuando las implementaciones integradas o cerradas no están disponibles. El código usa herramientas de compilación de Android con Kotlin y componentes nativos para proporcionar el comportamiento a nivel de API que esperan los módulos Zygisk. Lo usan principalmente investigadores de seguridad y modding de Android que necesitan inyección flexible de procesos y experimentación con módulos en dispositivos con root.
