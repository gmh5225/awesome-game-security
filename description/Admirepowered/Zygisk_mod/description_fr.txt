Ce projet est une implémentation autonome de l'interface d'exécution Zygisk pour les environnements Android rootés. Il vise la compatibilité avec KernelSU, Apatch et Magisk, et propose une voie alternative de chargement de modules lorsque les implémentations intégrées ou fermées ne sont pas disponibles. La base de code s'appuie sur les outils de build Android, avec Kotlin et des composants natifs, pour fournir le comportement au niveau API attendu par les modules Zygisk. Il est principalement utilisé par les chercheurs en sécurité Android et en modding qui ont besoin d'une injection de processus flexible et d'expérimentations de modules sur des appareils rootés.
