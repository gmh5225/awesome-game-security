本プロジェクトは、root 化された Android 環境向けの Zygisk ランタイムインターフェースのスタンドアロン実装です。KernelSU、Apatch、Magisk 各環境との互換性を目標とし、組み込みまたはクローズドな実装が利用できない場合の代替モジュール読み込み経路を提供します。コードベースは Android のビルドツールチェーンと Kotlin、ネイティブコンポーネントを用い、Zygisk モジュールが期待する API レベルの振る舞いを実現します。主に、root 端末上で柔軟なプロセス注入やモジュール実験を必要とする Android セキュリティおよび改造研究者に利用されます。
