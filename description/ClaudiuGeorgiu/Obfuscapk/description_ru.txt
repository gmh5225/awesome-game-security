Этот проект — модульный инструмент на Python для чёрноящиковой обфускации Android-приложений. Он декомпилирует APK с помощью apktool, применяет проходы обфускации к smali-коду, ресурсам и манифестам, а затем пересобирает функционально эквивалентные, но более сложные для анализа результаты. Репозиторий включает несколько обфускаторов, документацию и раннюю поддержку Android App Bundles через внешний компонент декомпиляции. Им в основном пользуются исследователи мобильной безопасности и разработчики, оценивающие устойчивость к реверс-инжинирингу и обнаружению по сигнатурам.
