Ce projet est le Egret Engine, un moteur de jeu HTML5.
Il fournit des modules pour gérer des tâches courantes du développement de jeux, telles que le rendu 2D et 3D, les systèmes GUI, ainsi que la gestion de l'audio et des ressources.
Il est principalement utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
