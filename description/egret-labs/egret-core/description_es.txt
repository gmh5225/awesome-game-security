Este proyecto es el Egret Engine, un motor de juegos HTML5.
Proporciona módulos para gestionar tareas habituales del desarrollo de juegos, como el renderizado 2D y 3D, sistemas GUI, y la gestión de audio y recursos.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el ámbito de motores de juego / código fuente.
