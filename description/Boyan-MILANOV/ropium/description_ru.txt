ROPium — библиотека и утилита командной строки для построения цепочек return-oriented programming из бинарных гаджетов. Она автоматически извлекает и анализирует гаджеты, а также поддерживает семантические запросы для генерации сложных цепочек с меньшими ручными усилиями. Проект сочетает ядро на C++ с привязками Python, поэтому рабочие процессы можно автоматизировать скриптами или использовать интерактивно через CLI. Предназначен для исследований в области разработки эксплойтов, обучения бинарной безопасности и экспериментов с продвинутыми наступательными инструментами.
