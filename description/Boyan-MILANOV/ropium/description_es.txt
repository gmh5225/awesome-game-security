ROPium es una biblioteca y una herramienta de línea de comandos para construir cadenas de return-oriented programming a partir de gadgets binarios. Extrae y analiza gadgets de forma automática y admite consultas semánticas para generar cadenas complejas con menos esfuerzo manual. El proyecto combina un núcleo en C++ con bindings de Python para que los flujos de trabajo puedan automatizarse por script o usarse de forma interactiva desde una CLI. Está pensado para investigación en desarrollo de exploits, educación en seguridad binaria y experimentos con herramientas ofensivas avanzadas.
