Este proyecto ofrece una lista de juegos de código abierto. Incluye Commander Keen (1-6, Dreams) y la serie Cosmo's Cosmic, lo que permite jugar los episodios originales y algunos mods creados para ellos. Resulta especialmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo / guías de juegos.
