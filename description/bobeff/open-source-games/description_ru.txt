Этот проект предоставляет список игр с открытым исходным кодом. Он охватывает Commander Keen (1–6, Dreams) и серию Cosmo's Cosmic, позволяя играть в оригинальные эпизоды и некоторые моды, созданные для них. В первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов, работающим в области разработки игр / гайдов.
