Этот проект предоставляет изменённые исходники обработчиков KVM, которые перехватывают и изменяют поведение тайминга RDTSC на путях Intel VMX и AMD SVM.
В нём объясняется, как патчить код виртуализации ядра, настраивать поддельные дельты временных меток и конфигурировать флаги CPU в QEMU, например отключая RDTSCP.
Реализация написана на C для ядра Linux и сосредоточена на низкоуровневом управлении таймингом гипервизора.
Основной сценарий использования — исследования виртуализации и антидетекта, включая эксперименты вокруг проверок античита, основанных на тайминге.
