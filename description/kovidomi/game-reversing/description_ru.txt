Этот проект предоставляет составленный список лично рекомендованных учебных материалов, помогающих изучать реверс-инжиниринг видеоигр.
Большинство собранных ресурсов посвящено реверсу PC-игр под Microsoft Windows с упором на x86-ассемблер, а не x64, поскольку его проще изучать.
Материалы в первую очередь полезны исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / гайдов.
