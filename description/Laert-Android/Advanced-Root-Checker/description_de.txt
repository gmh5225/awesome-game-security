Advanced Root Checker ist eine kostenlose Open-Source-Android-App, die ein Gerät vollständig offline auf Root-Indikatoren und damit verbundene Sicherheitsrisiken prüft. In Java für Android 5.0 und höher geschrieben, führt sie Dutzende Checks durch, darunter su-Binaries, BusyBox, Magisk, KernelSU, APatch, Zygisk, Xposed/LSPosed, Root-Cloaking-Apps, SELinux-Status und gefährliche Systemeigenschaften. Sie enthält außerdem einen Anti-Tamper-Bereich, der nach Frida, Xposed-Hooks, Debuggern, verdächtigen Bibliotheken sowie Änderungen an APK-Signatur oder Paketnamen sucht, plus Geräte-Sicherheitsinfos und eine Risiko-Score-Zusammenfassung. Der Hauptanwendungsfall ist die lokale Android-Sicherheitsbewertung und Root-Erkennung für Nutzer, Entwickler und alle, die Anti-Root- oder Anti-Cheat-ähnliche Checks ohne Netzwerkzugang bewerten möchten.
