Advanced Root Checker 是一款免费开源的 Android 应用，可完全离线扫描设备上的 root 迹象及相关安全风险。该应用用 Java 编写，支持 Android 5.0 及以上系统，会执行数十项检测，涵盖 su 二进制文件、BusyBox、Magisk、KernelSU、APatch、Zygisk、Xposed/LSPosed、root 隐藏应用、SELinux 状态以及危险的系统属性。它还包含反篡改模块，用于检测 Frida、Xposed hooks、调试器、可疑库，以及 APK 签名或包名变更，并提供设备安全信息与风险评分摘要。主要用途是供用户、开发者以及任何需要评估反 root 或反作弊类检测的人，在无网络环境下进行本地 Android 安全评估与 root 检测。
