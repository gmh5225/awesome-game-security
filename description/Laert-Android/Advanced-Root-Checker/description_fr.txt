Advanced Root Checker est une application Android gratuite et open source qui analyse hors ligne les indicateurs de root et les risques de sécurité associés sur un appareil. Écrite en Java pour Android 5.0 et versions ultérieures, elle exécute des dizaines de contrôles couvrant les binaires su, BusyBox, Magisk, KernelSU, APatch, Zygisk, Xposed/LSPosed, les applications de dissimulation de root, l'état SELinux et les propriétés système dangereuses. Elle inclut aussi une section anti-altération qui recherche Frida, les hooks Xposed, les débogueurs, les bibliothèques suspectes et les modifications de signature APK ou de nom de paquet, ainsi que des informations de sécurité de l'appareil et un résumé du score de risque. Le cas d'usage principal est l'évaluation locale de la sécurité Android et la détection de root pour les utilisateurs, les développeurs et quiconque évalue des contrôles de type anti-root ou anti-cheat sans accès réseau.
