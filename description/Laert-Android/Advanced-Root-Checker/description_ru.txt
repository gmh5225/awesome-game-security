Advanced Root Checker — бесплатное открытое Android-приложение, которое полностью офлайн сканирует устройство на признаки root и связанные риски безопасности. Написано на Java для Android 5.0 и новее; выполняет десятки проверок, охватывающих бинарники su, BusyBox, Magisk, KernelSU, APatch, Zygisk, Xposed/LSPosed, приложения для сокрытия root, состояние SELinux и опасные системные свойства. Также включает раздел anti-tamper, который ищет Frida, хуки Xposed, отладчики, подозрительные библиотеки и изменения подписи APK или имени пакета, плюс сведения о безопасности устройства и сводный риск-балл. Основной сценарий — локальная оценка безопасности Android и обнаружение root для пользователей, разработчиков и всех, кто проверяет anti-root или anti-cheat проверки без доступа к сети.
