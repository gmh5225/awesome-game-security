Advanced Root Checker は、端末の root 兆候と関連するセキュリティリスクを完全オフラインでスキャンする無料のオープンソース Android アプリです。Java で書かれ、Android 5.0 以降に対応し、su バイナリ、BusyBox、Magisk、KernelSU、APatch、Zygisk、Xposed/LSPosed、root 隠蔽アプリ、SELinux の状態、危険なシステムプロパティなど数十項目を検査します。また、Frida、Xposed hooks、デバッガ、不審なライブラリ、APK 署名やパッケージ名の変更を調べる改ざん対策セクションに加え、端末のセキュリティ情報とリスクスコアの概要も提供します。主な用途は、ユーザーや開発者、およびネットワークなしで anti-root や anti-cheat 系のチェックを評価したい人向けの、ローカルな Android セキュリティ評価と root 検出です。
