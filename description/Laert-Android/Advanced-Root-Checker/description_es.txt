Advanced Root Checker es una aplicación Android gratuita y de código abierto que analiza un dispositivo en busca de indicadores de root y riesgos de seguridad relacionados, totalmente sin conexión. Escrita en Java para Android 5.0 y superior, ejecuta docenas de comprobaciones que cubren binarios su, BusyBox, Magisk, KernelSU, APatch, Zygisk, Xposed/LSPosed, apps de ocultación de root, el estado de SELinux y propiedades del sistema peligrosas. También incluye una sección anti-manipulación que busca Frida, hooks de Xposed, depuradores, bibliotecas sospechosas y cambios en la firma del APK o el nombre del paquete, además de información de seguridad del dispositivo y un resumen de puntuación de riesgo. El caso de uso principal es la evaluación local de seguridad en Android y la detección de root para usuarios, desarrolladores y cualquiera que evalúe comprobaciones tipo anti-root o anti-cheat sin acceso a la red.
