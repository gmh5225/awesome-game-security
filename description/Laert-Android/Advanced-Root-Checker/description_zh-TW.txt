Advanced Root Checker 是一款免費開源的 Android 應用程式，可完全離線掃描裝置上的 root 跡象及相關資安風險。該應用以 Java 撰寫，支援 Android 5.0 及以上系統，會執行數十項檢測，涵蓋 su 二進位檔、BusyBox、Magisk、KernelSU、APatch、Zygisk、Xposed/LSPosed、root 隱藏應用、SELinux 狀態以及危險的系統屬性。它還包含反竄改模組，用於偵測 Frida、Xposed hooks、除錯器、可疑函式庫，以及 APK 簽名或套件名稱變更，並提供裝置資安資訊與風險評分摘要。主要用途是供使用者、開發者以及任何需要評估反 root 或反作弊類檢測的人，在無網路環境下進行本機 Android 資安評估與 root 偵測。
