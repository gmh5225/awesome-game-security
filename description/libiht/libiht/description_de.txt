Dieses Projekt konzentriert sich auf die Intel Hardware Trace Library.
Es wird stolz von Tencent Security Xuanwu Lab und dem langfristigen Talentförderungsprogramm Tencent Spark Talent Program bereitgestellt.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat-/Windows-Kernel-Exploration untersuchen.
