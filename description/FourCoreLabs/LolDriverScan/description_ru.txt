Этот проект — сканер на Go, который обнаруживает известные уязвимые драйверы на системе. Он получает сведения об уязвимых драйверах из публичного канала угроз, связанных с драйверами, и сравнивает их с хешами и метаданными локальных драйверов. Инструмент поддерживает подробные отчёты и экспорт в JSON для автоматизации или интеграции в другие процессы безопасности и рассчитан на работу без повышенных привилегий. Основной сценарий использования — защитный аудит безопасности, включая усиление защиты и проверки среды античита.
