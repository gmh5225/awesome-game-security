Этот проект — улучшенная открытая версия движка X-Ray, использовавшегося в серии S.T.A.L.K.E.R. Он сохраняет совместимость с оригинальным игровым контентом и при этом добавляет поддержку 64 бит, оптимизации производительности, исправления ошибок и возможности для авторов модов. Движок написан в основном на C++ и ориентирован на несколько настольных платформ, включая Linux и macOS наряду с Windows. Сообщество моддинга и реверс-инжиниринга использует его для изучения и расширения внутренностей промышленного игрового движка.
