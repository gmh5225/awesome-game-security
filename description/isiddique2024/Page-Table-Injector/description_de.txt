Dieses Projekt konzentriert sich auf Page Table Injector (PT-Injector).
Es ist hauptsächlich in C++ und C/C++ geschrieben und widmet sich der Anti-Cheat-Forschung, Kernel-Arbeit und Treiberentwicklung.
Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Injection unter Windows untersuchen.
