Этот проект посвящён Page Table Injector (PT-Injector).
Он в основном написан на C++ и C/C++ и сосредоточен на исследованиях античита, работе на уровне ядра и разработке драйверов.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / injection для Windows.
