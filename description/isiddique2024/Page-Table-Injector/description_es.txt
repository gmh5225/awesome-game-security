Este proyecto se centra en Page Table Injector (PT-Injector).
Está escrito principalmente en C++ y C/C++ y se enfoca en la investigación anti-cheat, el trabajo a nivel de kernel y el desarrollo de controladores.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheat / injection en Windows.
