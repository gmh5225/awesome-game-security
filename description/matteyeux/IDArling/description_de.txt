Dieses Projekt ist ein kollaboratives Reverse-Engineering-Plugin für IDA Pro und Hex-Rays. Es ermöglicht die Echtzeit-Synchronisierung von Änderungen, die mehrere IDA-Benutzer an einer IDA-Datenbank vornehmen, indem verschiedene Instanzen von IDA Pro miteinander verbunden werden. Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
