Этот проект — плагин для совместного реверс-инжиниринга в IDA Pro и Hex-Rays. Он позволяет в реальном времени синхронизировать изменения, вносимые несколькими пользователями IDA в одну базу данных IDA, соединяя между собой разные экземпляры IDA Pro. Плагин в первую очередь полезен исследователям безопасности игр и реверс-инженерам, изучающим атакующие техники в области читов / плагинов для IDA.
