Ce projet est un plugin collaboratif de reverse engineering pour IDA Pro et Hex-Rays. Il permet de synchroniser en temps réel les modifications apportées à une base de données IDA par plusieurs utilisateurs IDA, en connectant entre elles différentes instances d'IDA Pro. Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
