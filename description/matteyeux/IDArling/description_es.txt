Este proyecto es un plugin colaborativo de ingeniería inversa para IDA Pro y Hex-Rays. Permite sincronizar en tiempo real los cambios realizados en una base de datos de IDA por varios usuarios de IDA, conectando entre sí distintas instancias de IDA Pro. Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
