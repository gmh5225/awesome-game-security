Anti-Debug est une petite preuve de concept en C++ montrant la détection de débogueurs via le comportement du compteur de suspension de ResumeThread.
Il démontre comment un thread brièvement suspendu par l'attachement d'un débogueur peut être détecté en inspectant le compteur renvoyé par les appels WinAPI.
L'exemple est minimal et se concentre sur un seul signal anti-debug plutôt que sur un cadre de protection complet.
Il s'adresse aux apprenants en sécurité Windows qui expérimentent les techniques anti-tamper et anti-analyse.
