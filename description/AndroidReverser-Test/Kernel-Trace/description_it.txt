Questo progetto è un modulo kernel Linux e Android che usa le uprobe per agganciare un gran numero di funzioni in user space. Scritto in C e C++, espone header e componenti helper lato utente per configurare librerie target, offset e metadati degli hook. Supporta l’output basato su tracefs e fornisce API per registrare e rimuovere i punti di probe sulle versioni del kernel supportate. Il modulo è pensato per analisi dinamica, reverse engineering e tracciamento a basso livello del comportamento su sistemi Android.
