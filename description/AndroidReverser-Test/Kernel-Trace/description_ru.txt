Этот проект — модуль ядра Linux и Android, который с помощью uprobes перехватывает большое число функций пользовательского пространства. Написан на C и C++; предоставляет пользовательские заголовки и вспомогательные компоненты для настройки целевых библиотек, смещений и метаданных хуков. Поддерживает вывод на основе tracefs и предоставляет API для регистрации и очистки точек зондирования на поддерживаемых версиях ядра. Модуль предназначен для динамического анализа, реверс-инжиниринга и низкоуровневого трассирования поведения на системах Android.
