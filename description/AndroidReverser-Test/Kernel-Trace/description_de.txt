Dieses Projekt ist ein Linux- und Android-Kernelmodul, das mit uprobes eine große Anzahl von Userspace-Funktionen hookt. Es ist in C und C++ geschrieben und stellt benutzerseitige Header sowie Hilfskomponenten bereit, um Zielbibliotheken, Offsets und Hook-Metadaten zu konfigurieren. Es unterstützt eine tracefs-basierte Ausgabe und bietet APIs zum Registrieren und Löschen von Probepunkten auf unterstützten Kernelversionen. Das Modul ist für dynamische Analyse, Reverse Engineering und Low-Level-Verhaltensverfolgung auf Android-Systemen gedacht.
