Este proyecto es un módulo de kernel de Linux y Android que usa uprobes para enganchar un gran número de funciones de espacio de usuario. Escrito en C y C++, expone encabezados y componentes auxiliares del lado del usuario para configurar bibliotecas objetivo, offsets y metadatos de hooks. Admite salida basada en tracefs y proporciona APIs para registrar y borrar puntos de sonda en las versiones de kernel compatibles. El módulo está pensado para análisis dinámico, ingeniería inversa y trazado de comportamiento a bajo nivel en sistemas Android.
