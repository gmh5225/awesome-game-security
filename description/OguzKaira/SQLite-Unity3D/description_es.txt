Este proyecto es un ejemplo en C# para Unity que almacena y restaura las coordenadas del jugador con una base de datos SQLite. Ofrece un flujo sencillo con DatabaseManager para guardar datos de posición y cargarlos durante la partida mediante atajos de teclado. El código muestra cómo integrar un archivo SQLite en StreamingAssets y mapear valores básicos del estado del juego a almacenamiento persistente. Está pensado para quienes quieran una base ligera de guardado y carga en proyectos Unity.
