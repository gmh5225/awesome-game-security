Questo progetto è un esempio Unity in C# che memorizza e ripristina le coordinate del giocatore con un database SQLite. Fornisce un semplice flusso DatabaseManager per salvare i dati di posizione e caricarli durante il gameplay con scorciatoie da tastiera. Il codice mostra come integrare un file SQLite in StreamingAssets e mappare valori di base dello stato di gioco su uno storage persistente. È pensato per chi vuole una base leggera di salvataggio e caricamento nei progetti Unity.
