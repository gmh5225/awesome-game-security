Ce projet est un exemple Unity en C# qui stocke et restaure les coordonnées du joueur à l’aide d’une base SQLite. Il propose un flux DatabaseManager simple pour enregistrer les données de position et les recharger en jeu via des raccourcis clavier. Le code montre l’intégration d’un fichier SQLite dans StreamingAssets et le mapping de valeurs d’état de jeu basiques vers un stockage persistant. Il s’adresse aux apprenants qui souhaitent une base légère de sauvegarde et de chargement dans leurs projets Unity.
