Dieses Projekt ist ein Unity-C#-Beispiel, das Spielerkoordinaten mit einer SQLite-Datenbank speichert und wiederherstellt. Es bietet einen einfachen DatabaseManager-Ablauf, um Positionsdaten zu speichern und während des Spiels per Tastenkürzel zu laden. Der Code zeigt die Einbindung einer SQLite-Datei in StreamingAssets und die Abbildung grundlegender Spielzustandswerte auf persistente Speicherung. Es richtet sich an Lernende, die eine schlanke Speicher- und Ladegrundlage für Unity-Projekte suchen.
