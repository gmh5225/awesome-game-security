Этот проект — пример перемещения от первого лица в Unity 3D с реализацией обзора мышью, ходьбы, прыжка, рывка (dash) и приседания. Он написан на скриптах C#, которые нужно прикрепить к объекту игрока с компонентами Rigidbody и Collider. В README описаны шаги настройки, привязки управления и известные особенности поведения, например скольжение при рывке. В основном это дружелюбный к новичкам игровой прототип для изучения механики FPS-контроллера в Unity.
