Este proyecto es un ejemplo de movimiento en primera persona en Unity 3D que implementa mirada con el ratón, desplazamiento, salto, dash y agacharse. Está escrito en scripts C# pensados para adjuntarse a un objeto de jugador con componentes Rigidbody y Collider. El README documenta los pasos de configuración, los controles y problemas de comportamiento conocidos como el deslizamiento del dash. Es principalmente un prototipo de gameplay apto para principiantes para aprender la mecánica de un controlador FPS en Unity.
