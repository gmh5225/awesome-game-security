Questo progetto è un insieme di librerie .NET per leggere i dati binari degli asset di Unreal Engine 4. Include componenti per aprire file system virtuali basati su PAK e analizzare strutture degli asset e registry. L'implementazione è principalmente in C# ed è organizzata in moduli riutilizzabili per flussi di lavoro di tooling. È pensato per l'ispezione di asset Unreal, utilità di modding e ricerca tecnica sui formati dei contenuti di gioco.
