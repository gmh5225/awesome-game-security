Ce projet est un ensemble de bibliothèques .NET pour lire les données d'assets binaires d'Unreal Engine 4. Il comprend des composants pour ouvrir des systèmes de fichiers virtuels basés sur PAK et analyser les structures d'assets ainsi que les registres. L'implémentation est principalement en C# et organisée en modules réutilisables pour les flux d'outils. Il vise l'inspection d'assets Unreal, les utilitaires de modding et la recherche technique sur les formats de contenu de jeux.
