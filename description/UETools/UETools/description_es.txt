Este proyecto es un conjunto de bibliotecas .NET para leer datos binarios de assets de Unreal Engine 4. Incluye componentes para abrir sistemas de archivos virtuales basados en PAK y analizar estructuras de assets y registros. La implementación está principalmente en C# y se organiza como módulos reutilizables para flujos de trabajo de herramientas. Está pensado para la inspección de assets de Unreal, utilidades de modding e investigación técnica sobre formatos de contenido de juegos.
