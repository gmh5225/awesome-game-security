Этот проект — набор библиотек .NET для чтения бинарных данных ассетов Unreal Engine 4. Он включает компоненты для открытия виртуальных файловых систем на основе PAK и разбора структур ассетов и реестров. Реализация в основном на C# и организована как переиспользуемые модули для инструментальных рабочих процессов. Предназначен для инспекции ассетов Unreal, утилит моддинга и технических исследований форматов игрового контента.
