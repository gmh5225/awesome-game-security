Dieses Projekt ist eine Sammlung von .NET-Bibliotheken zum Lesen binärer Asset-Daten von Unreal Engine 4. Es enthält Komponenten zum Öffnen PAK-basierter virtueller Dateisysteme sowie zum Parsen von Asset-Strukturen und Registries. Die Implementierung ist überwiegend in C# und als wiederverwendbare Module für Tooling-Workflows organisiert. Es richtet sich an die Inspektion von Unreal-Assets, Modding-Hilfsmittel und technische Forschung zu Spielinhaltsformaten.
