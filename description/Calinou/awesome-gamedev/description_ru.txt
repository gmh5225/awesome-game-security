Этот проект — курируемый список бесплатных и открытых ресурсов для разработки игр. Репозиторий ведётся в Markdown и упорядочивает инструменты, движки, библиотеки, учебные материалы и ассеты по многим языкам и областям. Акцент сделан на ПО и контент с открытыми лицензиями, с дополнительными примечаниями о несвободных вариантах. Основное назначение — помочь разработчикам и исследователям быстро находить справочные материалы по инструментам разработки игр.
