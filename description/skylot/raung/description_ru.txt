Этот проект посвящён ассемблеру/дизассемблеру для байткода Java.
Он находится на очень ранней стадии разработки: многие функции ещё не готовы, а синтаксис нестабилен.
В первую очередь он полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области инструментов для читов / реверса.
