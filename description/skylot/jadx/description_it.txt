Questo progetto è JADX, un decompilatore da DEX a Java che converte file Android APK, DEX e AAR in codice sorgente Java leggibile. Include sia uno strumento a riga di comando sia una GUI basata su Swing, con funzionalità di ricerca, navigazione, deoffuscamento ed esportazione. Il decompilatore gestisce flussi di controllo complessi, gestori di eccezioni, classi interne, annotazioni e la ricostruzione degli enum. È rivolto a reverse engineer Android, analisti di malware e ricercatori di sicurezza mobile che devono esaminare gli interni delle applicazioni Android.
