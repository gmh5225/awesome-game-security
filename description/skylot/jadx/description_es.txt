Este proyecto es JADX, un descompilador de DEX a Java que convierte archivos Android APK, DEX y AAR en código fuente Java legible. Incluye tanto una herramienta de línea de comandos como una interfaz gráfica basada en Swing, con capacidades de búsqueda, navegación, desofuscación y exportación. El descompilador maneja flujos de control complejos, manejadores de excepciones, clases internas, anotaciones y la reconstrucción de enumeraciones. Está dirigido a ingenieros de ingeniería inversa de Android, analistas de malware e investigadores de seguridad móvil que necesitan examinar el interior de las aplicaciones Android.
