Ce projet est JADX, un décompilateur DEX vers Java qui convertit les fichiers Android APK, DEX et AAR en code source Java lisible. Il propose à la fois un outil en ligne de commande et une interface graphique basée sur Swing, avec des fonctions de recherche, de navigation, de désobfuscation et d'export. Le décompilateur gère les flux de contrôle complexes, les gestionnaires d'exceptions, les classes internes, les annotations et la reconstruction des énumérations. Il s'adresse aux ingénieurs en reverse engineering Android, aux analystes de malwares et aux chercheurs en sécurité mobile qui doivent examiner le fonctionnement interne des applications Android.
