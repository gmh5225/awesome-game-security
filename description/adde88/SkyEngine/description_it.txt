SkyEngine è un unlocker Lua per World of Warcraft che mira alle restrizioni di esecuzione Lua protetta nel client.
L'implementazione è in C++ con componenti legati a Lua e file di solution Windows per build pratici.
Funziona reimpostando ripetutamente uno stato legato al taint in modo che le funzioni Lua protette possano essere eseguite in modo più affidabile.
Il progetto è rilevante soprattutto per la ricerca sullo scripting WoW, la prototipazione di cheat e gli studi sul rischio di rilevamento anti-cheat.
