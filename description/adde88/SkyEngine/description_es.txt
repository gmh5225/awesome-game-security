SkyEngine es un desbloqueador Lua de World of Warcraft que aborda las restricciones de ejecución Lua protegida en el cliente.
La implementación está en C++ con componentes relacionados con Lua y archivos de solución de Windows para compilaciones prácticas.
Funciona restableciendo repetidamente un estado relacionado con el taint para que las funciones Lua protegidas puedan ejecutarse de forma más fiable.
El proyecto es relevante principalmente para la investigación de scripting en WoW, el prototipado de cheats y los estudios de riesgo de detección anti-cheat.
