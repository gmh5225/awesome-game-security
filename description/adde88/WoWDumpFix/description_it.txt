WoWDumpFix è un plugin per x64dbg che rimuove gli ostacoli anti-dump dai processi di giochi Blizzard protetti.
È scritto in C e C++ ed è pensato per essere usato con Scylla per la ricostruzione degli import e la correzione dei dump.
Il plugin include patch orientate al debugger, come il ripristino del comportamento atteso dei breakpoint durante l'attach al processo.
Il caso d'uso principale è il reverse engineering e l'analisi statica di binari di giochi che impiegano misure anti-tamper.
