Este proyecto es una colección de firmware FPGA de PCILeech que admite varias configuraciones de hardware DMA, incluidas las variantes 100T, Squirrel y Captain. Incluye núcleos IP de Vivado para el espacio de configuración PCIe, BRAM, clock wizards y FIFOs, así como scripts por lotes para generar compilaciones de firmware orientadas a distintas placas FPGA. Resulta especialmente útil para investigadores de seguridad DMA y desarrolladores de firmware que trabajan con hardware FPGA compatible con PCILeech en forense de memoria e investigación de ataques DMA.
