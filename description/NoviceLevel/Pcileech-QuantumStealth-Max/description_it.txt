Questo progetto è una raccolta di firmware FPGA PCILeech che supporta più configurazioni hardware DMA, tra cui le varianti 100T, Squirrel e Captain. Include core IP Vivado per lo spazio di configurazione PCIe, BRAM, clock wizard e FIFO, nonché script batch per generare build del firmware destinate a diverse schede FPGA. È principalmente utile per ricercatori di sicurezza DMA e sviluppatori di firmware che lavorano con hardware FPGA compatibile con PCILeech per forensics della memoria e ricerca sugli attacchi DMA.
