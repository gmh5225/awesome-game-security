This project is a King of the Road / Hard Truck 2 ASI patch covering collision, camera, physics, AI traffic, trailer, and UI fixes.
It complements D2GI and loads via ModUtils with Ultimate ASI Loader.
It is primarily written in C++ and centers on physics, plugin development, and modding for legacy DirectX titles.
It is mainly useful for graphics programmers and Windows game tooling developers working in the directx / compatibility area.
