Ce projet est un correctif ASI pour King of the Road / Hard Truck 2 couvrant les collisions, la caméra, la physique, le trafic IA, la remorque et l'interface.
Il complète D2GI et se charge via ModUtils avec Ultimate ASI Loader.
Il est principalement écrit en C++ et se concentre sur la physique, le développement de plugins et le modding pour les titres DirectX hérités.
Il est surtout utile aux programmeurs graphiques et aux développeurs d'outils de jeux Windows travaillant dans le domaine DirectX / compatibilité.
