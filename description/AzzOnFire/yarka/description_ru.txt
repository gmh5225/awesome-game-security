Этот проект — плагин для IDA Pro, который генерирует YARA-сигнатуры из выбранных инструкций, строк или сырых байтов. Он реализован на Python и включает встроенный редактор с подсветкой синтаксиса и настраиваемые шаблоны правил. Он может строить правила на основе дизассемблирования, вывода декомпилятора, представлений строк и функциональных выборок для hunting со связанными ссылками на данные. Основной сценарий использования — рабочие процессы анализа вредоносного ПО и бинарников, где аналитикам нужно быстро создавать надёжные YARA-правила.
