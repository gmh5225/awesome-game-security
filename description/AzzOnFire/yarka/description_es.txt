Este proyecto es un plugin de IDA Pro que genera firmas YARA a partir de instrucciones, cadenas o bytes en bruto seleccionados. Está implementado en Python e incluye una interfaz de editor integrada con resaltado de sintaxis y plantillas de reglas personalizables. Puede crear reglas a partir del desensamblado, la salida del descompilador, vistas de cadenas y selecciones de hunting a nivel de función con referencias de datos relacionadas. El caso de uso principal son los flujos de trabajo de análisis de malware y binarios donde los analistas necesitan crear rápidamente reglas YARA fiables.
