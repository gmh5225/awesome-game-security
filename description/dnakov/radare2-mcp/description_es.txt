Un servidor Model Context Protocol (MCP) escrito en C que expone las capacidades de análisis binario de radare2 a agentes de IA, con soporte para modos CLI y plugin y comunicación por stdin/stdout.
Proporciona herramientas de desensamblado, descompilación, referencias cruzadas y análisis binario mediante r2pipe, con sandbox configurable, modo de solo lectura y restricciones de herramientas detalladas.
Resulta especialmente útil para investigadores de seguridad que integran el análisis binario basado en radare2 en flujos de ingeniería inversa asistida por IA.
