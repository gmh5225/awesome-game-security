Сервер Model Context Protocol (MCP) на языке C, который открывает возможности бинарного анализа radare2 для ИИ-агентов и поддерживает режимы CLI и плагина с обменом данными через stdin/stdout.
Через r2pipe он предоставляет инструменты дизассемблирования, декомпиляции, перекрёстных ссылок и бинарного анализа, а также настраиваемую песочницу, режим только чтения и детальные ограничения инструментов.
В основном полезен исследователям безопасности, которые встраивают бинарный анализ на базе radare2 в рабочие процессы реверс-инжиниринга с помощью ИИ.
