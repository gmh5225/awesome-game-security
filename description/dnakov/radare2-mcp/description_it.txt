Un server Model Context Protocol (MCP) scritto in C che espone le capacità di analisi binaria di radare2 agli agenti IA, supportando sia la modalità CLI sia quella plugin con comunicazione via stdin/stdout.
Fornisce strumenti per disassemblaggio, decompilazione, riferimenti incrociati e analisi binaria tramite r2pipe, con sandbox configurabile, modalità di sola lettura e restrizioni sugli strumenti a grana fine.
È soprattutto utile per i ricercatori di sicurezza che integrano l'analisi binaria basata su radare2 in flussi di reverse engineering assistiti dall'IA.
