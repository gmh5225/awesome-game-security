Este proyecto es un kit de herramientas de Windows Intel Processor Trace que combina un controlador de kernel, un controlador en modo usuario y un decodificador de trazas en Python. El controlador configura la recolección de Intel PT, mientras que la aplicación cliente envía comandos de control y puede transmitir los paquetes de traza capturados a Kafka. Un decodificador complementario analiza los flujos de paquetes, reconstruye el comportamiento de ejecución y genera análisis visuales del temporizado de las trazas y de los eventos de flujo de control. Está diseñado para la investigación del comportamiento de exploits y malware, especialmente para estudiar patrones de ejecución de bajo nivel y la posible actividad de reutilización de código.
