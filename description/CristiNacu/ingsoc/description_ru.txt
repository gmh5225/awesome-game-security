Этот проект — набор инструментов Windows Intel Processor Trace, объединяющий драйвер ядра, пользовательский контроллер и декодер трасс на Python. Драйвер настраивает сбор Intel PT, а клиентское приложение отправляет управляющие команды и может потоково передавать захваченные пакеты трасс в Kafka. Сопутствующий декодер разбирает потоки пакетов, восстанавливает поведение выполнения и формирует визуальную аналитику по таймингу трасс и событиям потока управления. Он предназначен для исследования поведения эксплойтов и вредоносного ПО, особенно для изучения низкоуровневых шаблонов выполнения и потенциальной активности повторного использования кода.
