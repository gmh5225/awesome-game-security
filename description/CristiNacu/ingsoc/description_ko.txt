이 프로젝트는 커널 드라이버, 사용자 모드 컨트롤러, Python 트레이스 디코더를 결합한 Windows Intel Processor Trace(Intel PT) 툴킷입니다. 드라이버가 Intel PT 수집을 구성하고, 클라이언트 애플리케이션이 제어 명령을 보내며 캡처된 트레이스 패킷을 Kafka로 스트리밍할 수 있습니다. 함께 제공되는 디코더는 패킷 스트림을 파싱하고 실행 동작을 재구성하며, 트레이스 타이밍과 제어 흐름 이벤트에 대한 시각적 분석을 생성합니다. 익스플로잇 및 멀웨어 행위 연구를 위해 설계되었으며, 특히 저수준 실행 패턴과 잠재적인 코드 재사용 활동을 분석하는 데 적합합니다.
