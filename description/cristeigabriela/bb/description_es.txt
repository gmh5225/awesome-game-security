Este proyecto se centra en benowin Blanc: analiza el Windows SDK/PHNT mediante libclang; diseños de estructuras, enums y constantes al estilo dt sin WinDbg; CLI + TUI y exportación JSON.
Por separado, el PHNT (Process Hacker NT headers) mantenido por la comunidad documenta estructuras internas que Microsoft no publica.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / exploración del kernel de Windows.
