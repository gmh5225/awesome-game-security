Dieses Projekt konzentriert sich auf benowin Blanc — parst das Windows SDK/PHNT über libclang; Struct-Layouts, Enums und Konstanten wie mit dt, ohne WinDbg; CLI + TUI, JSON-Export.
Separat dokumentiert das community-gepflegte PHNT (Process Hacker NT headers) interne Strukturen, die Microsoft nicht veröffentlicht.
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Windows-Kernel-Explorer untersuchen.
