Ce projet se concentre sur benowin Blanc — parse le Windows SDK/PHNT via libclang ; dispositions de structures, énums et constantes façon dt sans WinDbg ; CLI + TUI, export JSON.
Par ailleurs, le PHNT (Process Hacker NT headers) maintenu par la communauté documente des structures internes que Microsoft ne publie pas.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / de l'exploration du noyau Windows.
