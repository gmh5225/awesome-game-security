Этот проект сосредоточен на benowin Blanc — парсит Windows SDK/PHNT через libclang; раскладки структур, перечисления и константы в духе dt без WinDbg; CLI + TUI, экспорт в JSON.
Отдельно community-поддерживаемый PHNT (Process Hacker NT headers) документирует внутренние структуры, которые Microsoft не публикует.
Он в основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / исследования ядра Windows.
