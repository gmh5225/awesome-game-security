Blackbone es una biblioteca de hacking de memoria para Windows orientada a flujos de trabajo avanzados de manipulación de procesos e inyección de código. Admite objetivos x86 y x64 y ofrece APIs para asignación de memoria, operaciones de lectura y escritura, cambios de protección, enumeración de módulos, mapeo PE manual y control de hilos a través de los límites WOW64. El proyecto incluye capacidades de modo usuario y relacionadas con el kernel, lo que lo hace adecuado para trabajos de internos de bajo nivel. Se utiliza principalmente en ingeniería inversa, investigación de seguridad en juegos y experimentos con herramientas relacionadas con el anti-cheat.
