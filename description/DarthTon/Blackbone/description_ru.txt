Blackbone — это библиотека для взлома памяти Windows, предназначенная для продвинутых сценариев манипуляции процессами и внедрения кода. Она поддерживает цели x86 и x64 и предоставляет API для выделения памяти, операций чтения и записи, изменения защиты, перечисления модулей, ручного PE-маппинга и управления потоками через границы WOW64. Проект включает возможности пользовательского режима и связанные с ядром, что делает его подходящим для работы с низкоуровневыми внутренностями системы. Он в основном используется в реверс-инжиниринге, исследованиях игровой безопасности и экспериментах с инструментами, связанными с античитами.
