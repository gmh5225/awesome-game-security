Blackbone è una libreria di memory hacking per Windows pensata per flussi di lavoro avanzati di manipolazione dei processi e di iniezione di codice. Supporta target x86 e x64 e fornisce API per allocazione di memoria, operazioni di lettura e scrittura, modifiche della protezione, enumerazione dei moduli, mapping PE manuale e controllo dei thread oltre i confini WOW64. Il progetto include capacità in user mode e correlate al kernel, rendendolo adatto al lavoro sugli interni a basso livello. Viene usato principalmente in reverse engineering, ricerca sulla sicurezza dei giochi e sperimentazioni di tooling legato all'anti-cheat.
