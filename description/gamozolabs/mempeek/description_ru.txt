Инструмент командной строки Linux для живой проверки памяти процессов через /proc/pid/mem с поиском значений в стиле Cheat Engine и фильтрами ограничений (равно, не равно, изменилось, увеличилось, уменьшилось, диапазон). Использует crate libprocmem для разбора /proc/pid/maps и поиска читаемых областей, поддерживает вычисление выражений в нескольких системах счисления (шестнадцатеричная, восьмеричная, десятичная) и предоставляет постоянный REPL на базе rustyline с историей команд между сессиями.
