Ce projet est un mappeur manuel de pilotes noyau Windows appelé Saturn, qui mappe des pilotes non signés en mémoire noyau en chargeant manuellement les sections PE, en résolvant les imports et en gérant les relocations. La base de code C++ comprend un composant pilote noyau WDK ainsi que des binaires de pilotes précompilés avec des capacités de scan de signatures. Il s’adresse principalement aux chercheurs en sécurité noyau qui étudient les techniques de mapping manuel de pilotes et le chargement de code au niveau noyau sans signature de pilote standard.
