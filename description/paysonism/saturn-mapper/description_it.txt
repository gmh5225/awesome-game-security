Questo progetto è un mapper manuale di driver del kernel Windows chiamato Saturn, che mappa driver non firmati nella memoria del kernel caricando manualmente le sezioni PE, risolvendo gli import e gestendo le rilocazioni. La codebase C++ include un componente driver kernel WDK e binari di driver precompilati con capacità di scansione delle signature. È soprattutto utile per i ricercatori di sicurezza del kernel che studiano le tecniche di mapping manuale dei driver e il caricamento di codice a livello kernel senza la firma standard dei driver.
