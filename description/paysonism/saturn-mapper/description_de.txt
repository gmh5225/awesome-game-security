Dieses Projekt ist ein manueller Mapper für Windows-Kernel-Treiber namens Saturn, der unsignierte Treiber durch manuelles Laden von PE-Sektionen, Auflösen von Imports und Behandeln von Relocations in den Kernel-Speicher mappt. Die C++-Codebasis umfasst sowohl eine WDK-Kernel-Treiberkomponente als auch vorgebaute Treiberbinärdateien mit Signatur-Scan-Funktionen. Es ist vor allem für Kernel-Sicherheitsforscher nützlich, die manuelle Treiber-Mapping-Techniken und das Laden von Code auf Kernel-Ebene ohne standardmäßige Treibersignierung untersuchen.
