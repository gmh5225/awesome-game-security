Este proyecto es un framework externo de prueba de concepto en C++ para leer la memoria del juego Valorant mediante un controlador de núcleo emparejado y un cliente en modo usuario. El componente de núcleo engancha una función de win32k para recibir paquetes de solicitud personalizados y expone operaciones como lecturas de memoria de procesos y obtención de datos relacionados con PML4. El componente de usuario localiza el proceso del juego, inicializa el canal del controlador y lee de forma continua punteros esenciales de Unreal como UWorld, ULevel y GameState. Está orientado principalmente a la investigación de hacking de juegos a bajo nivel y anticheat, centrada en la comunicación con el controlador y la extracción externa de datos.
