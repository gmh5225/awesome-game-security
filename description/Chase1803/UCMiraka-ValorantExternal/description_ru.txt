Этот проект — внешний proof-of-concept фреймворк на C++ для чтения памяти игры Valorant с парным драйвером ядра и клиентом в пользовательском режиме. Компонент ядра перехватывает функцию win32k для приёма пользовательских пакетов запросов и предоставляет операции вроде чтения памяти процесса и получения данных, связанных с PML4. Пользовательский компонент находит процесс игры, инициализирует канал связи с драйвером и многократно читает ключевые указатели Unreal, такие как UWorld, ULevel и GameState. Он в первую очередь ориентирован на исследования низкоуровневого game hacking и античита с акцентом на связь с драйвером и внешнее извлечение данных.
