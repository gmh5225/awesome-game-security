이 프로젝트는 짝을 이루는 커널 드라이버와 유저 모드 클라이언트로 Valorant 게임 메모리를 읽는 C++ 개념 증명용 외부 프레임워크입니다. 커널 구성 요소는 win32k 함수를 후킹하여 사용자 정의 요청 패킷을 수신하고, 프로세스 메모리 읽기 및 PML4 관련 데이터 조회 등의 작업을 제공합니다. 유저 구성 요소는 게임 프로세스를 찾고 드라이버 채널을 초기화한 뒤 UWorld, ULevel, GameState와 같은 핵심 Unreal 포인터를 반복적으로 읽습니다. 주로 드라이버 통신과 외부 데이터 추출에 초점을 둔 저수준 게임 해킹 및 안티치트 연구를 목적으로 합니다.
