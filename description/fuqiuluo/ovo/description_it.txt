Modulo driver kernel Android ARM64 che espone lettura/scrittura della memoria di processo, manipolazione delle page table MMU (mmuhack) e simulazione di input touch tramite un server TCP in spazio kernel. Fornisce traversamento delle VMA e mapping indirizzo-verso-PFN per la traduzione virtuale-verso-fisico, un modulo peekaboo per accesso stealth alla memoria e SDK client in C++ e Rust per la comunicazione in userspace o tra dispositivi.
