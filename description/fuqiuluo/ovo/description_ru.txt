Модуль драйвера ядра Android ARM64, предоставляющий чтение/запись памяти процессов, манипуляцию таблицами страниц MMU (mmuhack) и симуляцию сенсорного ввода через TCP-сокет-сервер в пространстве ядра. Обеспечивает обход VMA и отображение адреса в PFN для перевода виртуальных адресов в физические, модуль peekaboo для скрытого доступа к памяти, а также клиентские SDK на C++ и Rust для связи из userspace или между устройствами.
