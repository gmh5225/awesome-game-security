Módulo de controlador del kernel Android ARM64 que expone lectura/escritura de memoria de procesos, manipulación de tablas de páginas MMU (mmuhack) y simulación de entrada táctil mediante un servidor TCP en espacio de kernel. Ofrece recorrido de VMA y mapeo de dirección a PFN para la traducción virtual a física, un módulo peekaboo para acceso sigiloso a memoria, y SDKs cliente en C++ y Rust para comunicación en espacio de usuario o entre dispositivos.
