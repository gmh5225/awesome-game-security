Ce projet est une bibliothèque C++ qui élève des primitives d'écriture MSR (Model Specific Register) arbitraires en exécution complète de code noyau sous Windows. Elle exploite la réécriture du LSTAR MSR pour rediriger le point d'entrée du gestionnaire de syscall, permettant une exécution contrôlée de shellcode noyau depuis le mode utilisateur. La bibliothèque s'intègre aux backends VDM (Voyager) et bluepill pour obtenir la capacité initiale d'écriture MSR. Elle s'adresse aux chercheurs en sécurité bas niveau étudiant l'exploitation noyau basée sur les MSR et les techniques d'élévation de privilèges.
