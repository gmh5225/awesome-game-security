Este proyecto es una biblioteca C++ que eleva primitivas arbitrarias de escritura de MSR (Model Specific Register) a ejecución completa de código en el kernel de Windows. Aprovecha sobrescrituras del LSTAR MSR para redirigir el punto de entrada del manejador de syscall, permitiendo la ejecución controlada de shellcode en el kernel desde modo usuario. La biblioteca se integra con los backends VDM (Voyager) y bluepill para obtener la capacidad inicial de escritura de MSR. Está dirigida a investigadores de seguridad de bajo nivel que estudian la explotación del kernel basada en MSR y técnicas de elevación de privilegios.
