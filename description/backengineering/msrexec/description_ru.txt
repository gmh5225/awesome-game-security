Этот проект — библиотека на C++, которая повышает произвольные примитивы записи MSR (Model Specific Register) до полноценного выполнения кода в ядре Windows. Она использует перезапись LSTAR MSR для перенаправления точки входа обработчика syscall, позволяя контролируемо выполнять kernel shellcode из пользовательского режима. Библиотека интегрируется с бэкендами VDM (Voyager) и bluepill для получения исходной возможности записи MSR. Предназначена для исследователей низкоуровневой безопасности, изучающих эксплуатацию ядра на основе MSR и техники повышения привилегий.
