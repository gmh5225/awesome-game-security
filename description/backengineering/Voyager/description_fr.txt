Ce projet est un framework de hacking Hyper-V pour Windows 10 x64 (AMD et Intel). Il fonctionne sur toutes les versions de Windows 10-x64 (2004–1507). Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats et de l’exploration du noyau Windows.
