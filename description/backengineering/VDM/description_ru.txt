Этот проект — VDM (Voyager Driver Manager), библиотека на C++ для эксплуатации уязвимых подписанных драйверов с целью получения произвольного чтения/записи физической памяти и выполнения кода в ядре Windows. Она предоставляет чистый API поверх нескольких бэкендов уязвимых драйверов (gdrv, cpuz и др.) для чтения/записи физической памяти, трансляции виртуальных адресов и выполнения шеллкода в режиме ядра. Библиотека спроектирована как модульный бэкенд для инструментов вроде kdmapper, msrexec и bluepill. Она предназначена для исследователей kernel exploitation, изучающих примитивы атак BYOVD и создающих инструменты уровня ядра.
