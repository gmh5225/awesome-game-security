本プロジェクトは VDM（Voyager Driver Manager）であり、脆弱な署名済みドライバを悪用して Windows 上で任意の物理メモリ読み書きとカーネルコード実行を得るための C++ ライブラリです。gdrv や cpuz など複数の脆弱ドライババックエンド上にクリーンな API を提供し、物理メモリの読み書き、仮想アドレス変換、カーネルモードでのシェルコード実行を行えます。本ライブラリは kdmapper、msrexec、bluepill といったツール向けのモジュール式バックエンドとして設計されています。BYOVD 攻撃プリミティブの研究やカーネルレベルツール構築に取り組むカーネルエクスプロイト研究者を対象としています。
