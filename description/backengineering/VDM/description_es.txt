Este proyecto es VDM (Voyager Driver Manager), una biblioteca C++ para explotar controladores firmados vulnerables y obtener lectura/escritura arbitraria de memoria física y ejecución de código en el kernel de Windows. Proporciona una API limpia sobre múltiples backends de controladores vulnerables (gdrv, cpuz, etc.) para leer/escribir memoria física, traducir direcciones virtuales y ejecutar shellcode en modo kernel. La biblioteca está diseñada como un backend modular para herramientas como kdmapper, msrexec y bluepill. Está orientada a investigadores de explotación de kernel que estudian primitivas de ataque BYOVD y construyen herramientas a nivel de kernel.
