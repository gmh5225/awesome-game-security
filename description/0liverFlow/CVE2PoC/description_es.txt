Este proyecto agrega PoCs y exploits públicos para un ID de CVE determinado desde GitHub, ExploitDB, Nuclei y Metasploit.
Enriquece los resultados con contexto de CVSS, EPSS y CISA KEV, y puede generar laboratorios Docker, write-ups de bug bounty, mapeo de CVE a CPE, además de informes JSON o HTML.
Está escrito principalmente en Python y resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de herramientas de cheat / reverse engineering.
