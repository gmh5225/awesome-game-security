Questo progetto aggrega PoC ed exploit pubblici per un dato ID CVE da GitHub, ExploitDB, Nuclei e Metasploit.
Arricchisce i risultati con il contesto CVSS, EPSS e CISA KEV e può produrre laboratori Docker, write-up per bug bounty, mappatura CVE-to-CPE e report JSON o HTML.
È scritto principalmente in Python ed è soprattutto utile a ricercatori di game security e reverse engineer che studiano tecniche offensive nell’ambito degli strumenti di cheat / reverse engineering.
