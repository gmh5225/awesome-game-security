Этот проект собирает публичные PoC и эксплойты для заданного идентификатора CVE из GitHub, ExploitDB, Nuclei и Metasploit.
Он дополняет результаты контекстом CVSS, EPSS и CISA KEV и может создавать Docker-лаборатории, материалы для bug bounty, сопоставление CVE с CPE, а также отчёты в форматах JSON или HTML.
Проект в основном написан на Python и особенно полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области инструментов читов / RE.
