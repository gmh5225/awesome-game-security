This project provides a modified PCILeech Artix-7 PCIe configuration SystemVerilog module aimed at VGK (Vanguard) DMA-related research.
The archive centers on pcileech_pcie_cfg_a7.sv with CFG-space / MSIX-oriented changes derived from the PCILeech FPGA stack.
It is mainly useful for game security researchers studying DMA firmware behavior and anti-cheat DMA threat models.
