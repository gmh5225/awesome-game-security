Este proyecto es un pequeño repositorio de dump de offsets que publica offsets de memoria actualizados para un popular juego de disparos.
Almacena constantes en un encabezado C++ y mantiene los datos simples para integrarlos directamente en herramientas externas.
El repositorio es mínimo y se centra en mantener offsets de direcciones y estructuras en lugar de ofrecer un framework completo.
Suele ser usado por profesionales de ingeniería inversa de juegos que siguen las actualizaciones del cliente para análisis o mantenimiento de herramientas.
