Этот проект — небольшой репозиторий dump смещений, публикующий обновлённые memory offsets для популярного шутера.
Он хранит константы в заголовочном файле C++ и держит данные простыми для прямой интеграции во внешние инструменты.
Репозиторий минимален и сосредоточен на поддержке смещений адресов и структур, а не на полноценном фреймворке.
Его обычно используют специалисты по реверс-инжинирингу игр, отслеживающие обновления клиента для анализа или сопровождения инструментов.
