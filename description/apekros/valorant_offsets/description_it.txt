Questo progetto è un piccolo repository di dump di offset che pubblica offset di memoria aggiornati per un popolare gioco sparatutto.
Memorizza le costanti in un header C++ e mantiene i dati semplici per l'integrazione diretta in strumenti esterni.
Il repository è minimale e si concentra sul mantenere offset di indirizzi e strutture piuttosto che fornire un framework completo.
È tipicamente usato da professionisti del reverse engineering di giochi che tracciano gli aggiornamenti del client per analisi o manutenzione di tool.
