Этот проект — плагин Binary Ninja для генерации сигнатур байтовых шаблонов по дизассемблированным функциям.
Он выдаёт сигнатуры с подстановочными символами в стиле IDA, пригодные для поиска по шаблону, и при необходимости может переходить к сигнатурам начала функции.
Реализация написана на Python и упакована для совместимости с менеджером плагинов Binary Ninja.
В основном полезен для реверс-инжиниринга, работы с сигнатурами читов или античитов, а также сравнения обновлений бинарных файлов.
