Questo progetto è un plugin di Binary Ninja per generare firme a pattern di byte a partire da funzioni disassemblate.
Produce firme con wildcard in stile IDA adatte allo scanning di pattern e, se necessario, può usare firme sull'inizio della funzione.
L'implementazione è scritta in Python e pacchettizzata per la compatibilità con il gestore plugin di Binary Ninja.
È utile soprattutto per il reverse engineering, il lavoro su firme per cheat o anti-cheat e il confronto degli aggiornamenti binari.
