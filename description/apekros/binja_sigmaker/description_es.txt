Este proyecto es un complemento de Binary Ninja para generar firmas de patrones de bytes a partir de funciones desensambladas.
Emite firmas con comodines al estilo de IDA aptas para el escaneo de patrones y puede recurrir a firmas de inicio de función cuando sea necesario.
La implementación está escrita en Python y empaquetada para ser compatible con el administrador de complementos de Binary Ninja.
Resulta útil principalmente para ingeniería inversa, trabajo con firmas de cheats o anti-cheats y tareas de comparación de actualizaciones binarias.
