Questo progetto è un driver kernel per Android e Linux che espone operazioni di lettura e scrittura della memoria dei processi tramite un'interfaccia dispositivo. È scritto principalmente in C e implementa comandi ioctl per l'accesso alla memoria, la ricerca della base dei moduli e l'inoltro delle richieste. Il repository include i file di build del modulo kernel e componenti in spazio utente per comunicare con il driver. È usato soprattutto per la ricerca a basso livello sulla memoria di gioco, esperimenti di reverse engineering e lo studio di strumenti lato kernel.
