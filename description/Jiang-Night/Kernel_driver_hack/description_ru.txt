Этот проект — драйвер ядра Android и Linux, предоставляющий операции чтения и записи памяти процессов через интерфейс устройства. Он в основном написан на C и реализует команды ioctl для доступа к памяти, поиска базового адреса модуля и диспетчеризации запросов. Репозиторий включает файлы сборки модуля ядра и пользовательские компоненты для взаимодействия с драйвером. Он в основном используется для низкоуровневых исследований игровой памяти, экспериментов по реверс-инжинирингу и изучения инструментов на стороне ядра.
