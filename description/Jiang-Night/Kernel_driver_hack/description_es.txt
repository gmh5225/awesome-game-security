Este proyecto es un controlador de kernel para Android y Linux que expone operaciones de lectura y escritura de memoria de procesos a través de una interfaz de dispositivo. Está escrito principalmente en C e implementa comandos ioctl para acceso a memoria, búsqueda de la base de módulos y despacho de solicitudes. El repositorio incluye archivos de compilación del módulo de kernel y componentes de espacio de usuario para comunicarse con el controlador. Se usa principalmente para investigación de memoria de juegos a bajo nivel, experimentos de ingeniería inversa y el estudio de herramientas del lado del kernel.
