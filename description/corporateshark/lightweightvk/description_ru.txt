Этот проект — глубоко рефакторенный bindless-only форк IGL, рассчитанный на работу поверх Vulkan 1.3 с опциональной поддержкой mesh shaders и трассировки лучей.
Основные цели LightweightVK.
Он в первую очередь полезен низкоуровневым графическим программистам и разработчикам движков, ориентированным на производительность, работающим в области Vulkan / API.
