Ce projet est un fork profondément refactorisé et exclusivement bindless d’IGL, conçu pour fonctionner au-dessus de Vulkan 1.3 avec un support optionnel des mesh shaders et du ray tracing.
Les principaux objectifs de LightweightVK.
Il est surtout utile aux programmeurs graphiques bas niveau et aux développeurs de moteurs axés sur les performances travaillant dans le domaine Vulkan / API.
