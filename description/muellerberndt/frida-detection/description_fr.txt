Ce projet est un dépôt listé sous anti cheat / detection:frida et se concentre sur la détection de Frida.
Il est principalement écrit en Java et C/C++ et met l'accent sur le hooking.
Il est surtout utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine anti cheat / detection:frida.
