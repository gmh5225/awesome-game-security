Dieses Projekt ist ein IDA-Pro-Plugin zur Deobfuskation zur Dekompilierzeit durch Umschreiben von Microcode. Es ist überwiegend in Python geschrieben und so ausgelegt, dass es sich mit konfigurierbaren Regelsätzen in übliche Reverse-Engineering-Workflows einfügt. Das Framework ist erweiterbar gestaltet, sodass Forschende neue Deobfuskationslogik hinzufügen können, ohne den Kerncode stark zu ändern. Es wird vor allem von Malware- und Spiele-Sicherheitsanalysten genutzt, die klareren Pseudocode aus stark obfuskierten Binärdateien benötigen.
