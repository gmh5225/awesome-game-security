Ce projet est un plugin IDA Pro destiné à la désobfuscation au moment de la décompilation, en réécrivant le microcode. Il est principalement écrit en Python et conçu pour s'intégrer aux flux de reverse engineering habituels grâce à des jeux de règles configurables. Le framework privilégie l'extensibilité, permettant aux chercheurs d'ajouter une nouvelle logique de désobfuscation sans modifier lourdement le code cœur. Il est surtout utilisé par les analystes malware et de sécurité des jeux qui ont besoin d'un pseudocode plus clair à partir de binaires fortement obscurcis.
