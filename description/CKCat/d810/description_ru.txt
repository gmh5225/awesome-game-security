Этот проект — плагин для IDA Pro, выполняющий деобфускацию на этапе декомпиляции путём перезаписи микрокода. Он в основном написан на Python и рассчитан на встраивание в обычные процессы реверс-инжиниринга с настраиваемыми наборами правил. Фреймворк спроектирован с упором на расширяемость: исследователи могут добавлять новую логику деобфускации без серьёзных правок ядра. Им в основном пользуются аналитики вредоносного ПО и игровой безопасности, которым нужен более понятный псевдокод из сильно обфусцированных бинарников.
