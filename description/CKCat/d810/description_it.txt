Questo progetto è un plugin per IDA Pro che esegue la deoffuscazione in fase di decompilazione riscrivendo il microcode. È scritto principalmente in Python e pensato per integrarsi nei flussi di reverse engineering abituali con set di regole configurabili. Il framework è progettato per l'estensibilità, consentendo ai ricercatori di aggiungere nuova logica di deoffuscazione senza modificare pesantemente il codice core. È usato soprattutto da analisti di malware e di sicurezza dei videogiochi che necessitano di pseudocodice più chiaro da binari fortemente offuscati.
