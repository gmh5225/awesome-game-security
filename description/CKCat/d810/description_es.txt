Este proyecto es un plugin de IDA Pro para desofuscar en tiempo de descompilación reescribiendo el microcódigo. Está escrito principalmente en Python y pensado para integrarse en flujos habituales de ingeniería inversa con conjuntos de reglas configurables. El framework está diseñado para ser extensible, de modo que los investigadores puedan añadir nueva lógica de desofuscación sin modificar en exceso el código central. Lo usan sobre todo analistas de malware y de seguridad en videojuegos que necesitan un pseudocódigo más claro a partir de binarios muy ofuscados.
