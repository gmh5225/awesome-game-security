Ce projet est un spoofer d'identité matérielle Windows divulgué, avec une interface graphique basée sur ImGui. Il est principalement écrit en C++ et regroupe des composants loader et mapper pour l'interaction au niveau noyau, ainsi que des routines système basées sur WinAPI. Le code inclut aussi des comportements anti-analyse tels que des vérifications de débogueur et de processus, indiquant une conception pour des environnements adverses. Son usage principal est de comprendre comment les outils de spoofing orientés contournement de ban sont structurés dans les écosystèmes de triche de jeux.
