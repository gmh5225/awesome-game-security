Dieses Projekt ist ein geleakter Windows-Hardware-Identitäts-Spoofer mit einer auf ImGui aufgebauten grafischen Oberfläche. Es ist hauptsächlich in C++ umgesetzt und bündelt Loader- und Mapper-Komponenten für Kernel-Interaktion sowie WinAPI-basierte Systemroutinen. Der Code enthält außerdem Anti-Analyse-Verhalten wie Debugger- und Prozessprüfungen, was auf eine Auslegung für adversarial Umgebungen hindeutet. Der Hauptzweck ist zu verstehen, wie Spoofer-Tools im Stil der Ban-Umgehung in Game-Cheating-Ökosystemen aufgebaut sind.
