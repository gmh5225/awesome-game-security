BepInEx es un framework de plugins y modding para juegos Unity Mono, Unity IL2CPP y otros basados en .NET. Está escrito principalmente en C# e incluye sistemas de carga encadenada, precarga, registro y configuración para la gestión de plugins en tiempo de ejecución. El framework integra herramientas de parcheo y detour para ayudar a los desarrolladores a extender o instrumentar el comportamiento del juego en Windows, Linux y macOS. Lo usan sobre todo autores de mods y comunidades de ingeniería inversa de juegos que necesitan un cargador estable para objetivos Unity administrados y nativos.
