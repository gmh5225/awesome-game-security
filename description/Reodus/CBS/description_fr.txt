Ce projet est un plugin IDA Pro qui définit, active, désactive ou supprime des points d'arrêt selon des motifs d'instructions.
Il utilise Python et des expressions régulières pour parcourir les lignes de désassemblage et appliquer des actions de points d'arrêt sur les fonctions.
Le plugin inclut une interface PyQt pour gérer interactivement les motifs d'opcodes.
Il s'adresse aux ingénieurs en rétro-ingénierie qui ont besoin d'une automatisation rapide et reproductible des points d'arrêt lors de l'analyse binaire.
