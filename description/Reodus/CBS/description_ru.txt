Этот проект — плагин для IDA Pro, который устанавливает, включает, отключает или удаляет точки останова по шаблонам инструкций.
Он использует Python и регулярные выражения для сканирования строк дизассемблирования и применения действий с точками останова в функциях.
Плагин включает интерфейс на PyQt для интерактивного управления шаблонами опкодов.
Предназначен для специалистов по реверс-инжинирингу, которым нужна быстрая и повторяемая автоматизация точек останова при анализе бинарных файлов.
