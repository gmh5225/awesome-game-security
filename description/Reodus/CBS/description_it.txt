Questo progetto è un plugin per IDA Pro che imposta, abilita, disabilita o rimuove breakpoint in base a pattern di istruzioni.
Usa Python con espressioni regolari per esaminare le righe di disassembly e applicare azioni sui breakpoint nelle funzioni.
Il plugin include un'interfaccia PyQt per gestire interattivamente i pattern di opcode.
È pensato per reverse engineer che necessitano di un'automazione rapida e ripetibile dei breakpoint durante l'analisi binaria.
