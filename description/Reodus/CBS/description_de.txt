Dieses Projekt ist ein IDA-Pro-Plugin, das Breakpoints anhand von Befehlsmustern setzt, aktiviert, deaktiviert oder entfernt.
Es nutzt Python mit regulären Ausdrücken, um Disassembly-Zeilen zu scannen und Breakpoint-Aktionen über Funktionen hinweg anzuwenden.
Das Plugin enthält eine PyQt-Oberfläche zur interaktiven Verwaltung von Opcode-Mustern.
Es richtet sich an Reverse Engineers, die bei der Binäranalyse eine schnelle, wiederholbare Breakpoint-Automatisierung benötigen.
