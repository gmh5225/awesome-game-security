Este proyecto es un complemento de IDA Pro que establece, habilita, deshabilita o elimina puntos de interrupción según patrones de instrucciones.
Usa Python con expresiones regulares para examinar líneas de desensamblado y aplicar acciones de puntos de interrupción en las funciones.
El complemento incluye una interfaz PyQt para gestionar de forma interactiva los patrones de opcodes.
Está pensado para ingenieros de ingeniería inversa que necesitan una automatización rápida y repetible de puntos de interrupción durante el análisis binario.
