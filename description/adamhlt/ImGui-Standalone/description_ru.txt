ImGui Standalone — фреймворк на базе Direct3D11 для запуска внешнего интерфейса ImGui в виде EXE или DLL в Windows.
Он реализован на C++ и поставляется с заранее настроенными проектами Visual Studio для целей x86 и x64.
Проект создаёт собственное окно рендеринга, поэтому полезен даже когда целевой процесс не предоставляет удобный внутренний путь рендеринга DirectX.
Его часто используют для игровых инструментов, прототипирования меню и внешних UI-рабочих процессов в исследованиях безопасности игр.
