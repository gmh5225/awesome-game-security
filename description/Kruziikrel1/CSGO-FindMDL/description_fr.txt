Ce projet est un changeur de modèles interne pour Counter-Strike: Global Offensive basé sur un hook FindMDL.
Il est implémenté sous forme de DLL C++ avec logique de hooks, wrappers d’interfaces, gestion des offsets et utilitaires de gestion du VMT.
Le dépôt inclut des fichiers de projet Visual Studio et un flux de travail typique piloté par injecteur pour le chargement dans le processus du jeu.
Le code source est structuré pour permettre de personnaliser le comportement et les chemins de remplacement de modèles.
Il est principalement destiné à la pratique du développement de cheats et à la recherche en rétro-ingénierie de jeux.
