Este proyecto es un cambiador de modelos interno para Counter-Strike: Global Offensive basado en un hook de FindMDL.
Está implementado como una DLL en C++ con lógica de hooks, wrappers de interfaces, manejo de offsets y utilidades de gestión de VMT.
El repositorio incluye archivos de proyecto de Visual Studio y un flujo de trabajo típico impulsado por inyector para cargarlo en el proceso del juego.
El código fuente está estructurado para que los usuarios puedan personalizar el comportamiento y las rutas de reemplazo de modelos.
Está pensado principalmente para practicar el desarrollo de cheats y la investigación de ingeniería inversa de juegos.
