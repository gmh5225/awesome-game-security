이 프로젝트는 FindMDL 훅을 기반으로 한 Counter-Strike: Global Offensive용 내부 모델 체인저입니다.
훅 로직, 인터페이스 래퍼, 오프셋 처리, VMT 관리 유틸리티를 포함한 C++ DLL로 구현됩니다.
저장소에는 Visual Studio 프로젝트 파일이 포함되어 있으며, DLL을 게임 프로세스에 로드하는 일반적인 인젝터 기반 워크플로를 따릅니다.
소스는 사용자가 모델 교체 동작과 경로를 쉽게 커스터마이즈할 수 있도록 구성되어 있습니다.
주로 치트 개발 연습과 게임 리버스 엔지니어링 연구용으로 설계되었습니다.
