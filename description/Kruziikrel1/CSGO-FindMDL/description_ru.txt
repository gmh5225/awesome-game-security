Этот проект — внутренний model changer для Counter-Strike: Global Offensive на основе хука FindMDL.
Он реализован как C++ DLL с логикой хуков, обёртками интерфейсов, обработкой оффсетов и утилитами управления VMT.
Репозиторий включает файлы проекта Visual Studio и типичный сценарий загрузки через инжектор в процесс игры.
Исходный код структурирован так, чтобы можно было настраивать поведение и пути замены моделей.
В основном предназначен для практики разработки читов и исследований по реверс-инжинирингу игр.
