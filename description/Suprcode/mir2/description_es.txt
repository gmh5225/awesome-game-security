Este proyecto es una implementación completa de código abierto de cliente y servidor para The Legend of Mir 2. Está escrito principalmente en C# e incluye en una sola base de código el cliente del juego, el servidor, el lanzador, herramientas administrativas y utilidades de datos de apoyo. El repositorio enfatiza una arquitectura gestionada modular, mejoras de rendimiento y compatibilidad con contenido personalizado y extensiones de la comunidad. Se usa principalmente por desarrolladores de servidores privados y comunidades centradas en la preservación que mantienen y amplían la infraestructura de MMORPG clásicos.
