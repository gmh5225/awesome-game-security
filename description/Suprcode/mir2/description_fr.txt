Ce projet est une implémentation client et serveur open source complète pour The Legend of Mir 2. Il est principalement écrit en C# et regroupe dans une même base de code le client de jeu, le serveur, le lanceur, les outils d’administration et les utilitaires de données associés. Le dépôt met l’accent sur une architecture managée modulaire, des améliorations de performances et la compatibilité avec les contenus personnalisés et les extensions communautaires. Il est surtout utilisé par les développeurs de serveurs privés et les communautés axées sur la préservation qui maintiennent et étendent l’infrastructure des MMORPG classiques.
