Este proyecto es un motor y un conjunto de herramientas de código abierto para The Legend of Mir 3. Está escrito principalmente en C# e incluye proyectos de cliente, servidor, lanzador, editor y bibliotecas compartidas para un flujo de trabajo MMORPG completo. El código incluye capas de red, modelos de contenido/datos y componentes de renderizado basados en pipelines DirectX/SharpDX. Está pensado para operadores de servidores Mir y comunidades de desarrollo de juegos que crean contenido personalizado, lógica de servidor y despliegues privados modernizados.
