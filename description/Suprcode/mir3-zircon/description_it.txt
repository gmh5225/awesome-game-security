Questo progetto è un motore open source e una suite di strumenti per The Legend of Mir 3. È scritto principalmente in C# e include progetti client, server, launcher, editor e librerie condivise per un flusso di lavoro MMORPG completo. Il codice comprende livelli di rete, modelli di contenuti/dati e componenti di rendering basati su pipeline DirectX/SharpDX. È pensato per operatori di server Mir e comunità di sviluppo di giochi che realizzano contenuti personalizzati, logica server e deployment privati modernizzati.
