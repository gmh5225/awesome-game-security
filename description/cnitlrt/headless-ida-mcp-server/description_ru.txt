Этот проект посвящён MCP для IDA Pro (headless).
Он использует headless-режим IDA Pro для анализа бинарных файлов и предоставляет через MCP набор инструментов для управления и изменения функций, переменных и многого другого.
В основном полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / MCP-серверов.
