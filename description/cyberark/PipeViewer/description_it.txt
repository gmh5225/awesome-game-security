Questo progetto è PipeViewer, uno strumento GUI Windows per enumerare e ispezionare i named pipe sul sistema. Elenca tutte le istanze attive di named pipe con i relativi descrittori di sicurezza, client connessi, modalità di accesso e informazioni sul processo proprietario. L’applicazione C# consente di filtrare, cercare e monitorare in tempo reale la creazione e l’eliminazione dei pipe. È pensata per ricercatori di sicurezza, analisti di malware e penetration tester che studiano i meccanismi IPC di Windows e le superfici di attacco basate sui named pipe.
