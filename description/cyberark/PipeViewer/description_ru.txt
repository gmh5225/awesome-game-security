Этот проект — PipeViewer, Windows GUI-инструмент для перечисления и анализа именованных каналов (named pipes) в системе. Он показывает все активные экземпляры именованных каналов вместе с дескрипторами безопасности, подключёнными клиентами, режимами доступа и сведениями о процессе-владельце. Приложение на C# позволяет фильтровать, искать и отслеживать создание и удаление каналов в реальном времени. Предназначено для исследователей безопасности, аналитиков вредоносного ПО и пентестеров, изучающих механизмы IPC в Windows и поверхности атаки на основе named pipes.
