Este proyecto es PipeViewer, una herramienta GUI de Windows para enumerar e inspeccionar named pipes del sistema. Lista todas las instancias activas de named pipes con sus descriptores de seguridad, clientes conectados, modos de acceso e información del proceso propietario. La aplicación en C# permite filtrar, buscar y monitorear en tiempo real la creación y eliminación de pipes. Está dirigida a investigadores de seguridad, analistas de malware y pentesters que estudian los mecanismos IPC de Windows y las superficies de ataque basadas en named pipes.
