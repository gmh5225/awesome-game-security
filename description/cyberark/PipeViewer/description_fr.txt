Ce projet est PipeViewer, un outil GUI Windows permettant d’énumérer et d’inspecter les named pipes du système. Il liste toutes les instances de named pipes actives avec leurs descripteurs de sécurité, clients connectés, modes d’accès et informations sur le processus propriétaire. L’application C# permet de filtrer, de rechercher et de surveiller en temps réel la création et la suppression de pipes. Elle s’adresse aux chercheurs en sécurité, analystes de malwares et testeurs d’intrusion qui étudient les mécanismes IPC Windows et les surfaces d’attaque basées sur les named pipes.
