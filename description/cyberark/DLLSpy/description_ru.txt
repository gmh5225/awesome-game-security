Этот проект — DLLSpy, инструмент CyberArk для обнаружения уязвимостей DLL hijacking в установленных приложениях Windows. Он сканирует запущенные процессы и установленные программы на предмет отсутствующих зависимостей DLL, доступных для записи каталогов DLL и небезопасных конфигураций путей поиска, которые могут быть использованы для выполнения кода. Инструмент формирует отчёты с указанием конкретных возможностей hijacking. Предназначен для аудиторов безопасности и системных администраторов, оценивающих риск DLL hijacking в своих средах Windows.
