Questo progetto è DLLSpy, uno strumento CyberArk per rilevare vulnerabilità di DLL hijacking nelle applicazioni Windows installate. Analizza i processi in esecuzione e i programmi installati alla ricerca di dipendenze DLL mancanti, directory DLL scrivibili e configurazioni non sicure dei percorsi di ricerca che potrebbero essere sfruttate per l'esecuzione di codice. Lo strumento genera report che individuano opportunità specifiche di hijacking. È rivolto ad auditor di sicurezza e amministratori di sistema che valutano il rischio di DLL hijacking nei propri ambienti Windows.
