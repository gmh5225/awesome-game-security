本项目是一份面向《反恐精英2》（Counter-Strike 2）的混合反作弊系统技术提案与 Python 概念验证，将自动化检测与社区 Overwatch 式复核流程相结合。它模拟 VAC Live 风格的机器学习，并配合 Overwatch 2.0，采用 Glicko-2 评分与加权投票，使更准确的评审者拥有更大影响力，并提高滥用机器人农场的难度。核心技术包括用于确定性自瞄与透视取证的隐形蜜罐实体、封禁前两到三场比赛的影子监控以降低误报，以及对电竞职业选手案例的特殊处理。仓库还勾勒了一种基于垂直高度差而非滞空时间的 Source 2 跌落伤害修复思路。面向游戏安全研究人员、反作弊设计师，以及评估可扩展混合检测与人工复核方案的 CS2 开发者。
