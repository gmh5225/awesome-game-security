本プロジェクトは、Counter-Strike 2 向けハイブリッドアンチチートの技術提案と Python による概念実証であり、自動検知とコミュニティの Overwatch 型レビューパイプラインを組み合わせています。VAC Live 風の機械学習を Overwatch 2.0 と併用してシミュレートし、Glicko-2 レーティングと重み付き投票により、精度の高い審査員ほど影響力を持ち、ボットファームによる悪用をより困難にします。主な手法には、決定的なエイム／ウォールハック証拠を得る不可視ハニーポットエンティティ、誤検知削減のためのバン前 2〜3 試合にわたるシャドウ監視、eスポーツプロ案件への特別対応が含まれます。リポジトリでは、滞空時間ではなく垂直方向の高さ差分に基づく Source 2 の落下ダメージ修正案も示しています。ゲームセキュリティ研究者、アンチチート設計者、スケーラブルなハイブリッド検知と人手レビューを検討する CS2 開発者向けです。
