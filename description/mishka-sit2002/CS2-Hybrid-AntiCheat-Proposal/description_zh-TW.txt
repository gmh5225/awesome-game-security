本專案是一份面向《絕對武力2》（Counter-Strike 2）的混合反作弊系統技術提案與 Python 概念驗證，將自動化偵測與社群 Overwatch 式覆核流程相結合。它模擬 VAC Live 風格的機器學習，並搭配 Overwatch 2.0，採用 Glicko-2 評分與加權投票，使更準確的審核者擁有更大影響力，並提高濫用機器人農場的難度。核心技術包括用於確定性瞄準與透視取證的隱形蜜罐實體、封鎖前兩到三場比賽的影子監控以降低誤報，以及對電競職業選手案例的特殊處理。倉庫亦勾勒一種基於垂直高度差而非滯空時間的 Source 2 跌落傷害修復思路。面向遊戲安全研究人員、反作弊設計師，以及評估可擴展混合偵測與人工覆核方案的 CS2 開發者。
