Este proyecto es una base de conocimiento curada sobre técnicas antiforenses, no un paquete de software ejecutable. El contenido está organizado en un README que enumera métodos de ocultación de datos, manipulación de registros (logs), manipulación de artefactos del registro y del sistema de archivos, e ideas de evasión basadas en virtualización. También referencia ejemplos aportados por la comunidad en debates de DFIR para destacar patrones reales de tradecraft de atacantes. El público principal son respondientes de incidentes e investigadores de seguridad que estudian cómo se puede ocultar la evidencia.
