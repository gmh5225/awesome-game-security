Questo progetto è una knowledge base curata di tecniche anti-forensi, non un pacchetto software eseguibile. Il contenuto è organizzato in un README che elenca metodi per l'occultamento dei dati, la manomissione dei log, la manipolazione di artefatti di registro e file system, nonché idee di evasione basate sulla virtualizzazione. Fa anche riferimento a esempi provenienti dalla community nelle discussioni DFIR per evidenziare i pattern reali del tradecraft degli attaccanti. Il pubblico principale è costituito da incident responder e ricercatori di sicurezza che studiano come le prove possano essere oscurate.
