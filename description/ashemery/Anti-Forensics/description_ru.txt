Этот проект представляет собой курируемую базу знаний по антифорензик-техникам, а не исполняемый программный пакет. Содержание организовано в README и перечисляет методы сокрытия данных, подмены журналов, манипуляции артефактами реестра и файловой системы, а также идеи обхода на основе виртуализации. Также приведены примеры из обсуждений DFIR сообщества, иллюстрирующие реальные модели тактик злоумышленников. Основная аудитория — специалисты по реагированию на инциденты и исследователи безопасности, изучающие способы сокрытия улик.
