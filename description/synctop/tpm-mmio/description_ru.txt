Этот проект посвящён чтению открытого Endorsement Key TPM 2.0 через MMIO (Memory-Mapped I/O). Данное доказательство концепции (POC) показывает, как с помощью Memory-Mapped I/O (MMIO) можно напрямую запрашивать состояние TPM и EK с самого чипа, обходя любые OS-хуки. В первую очередь полезно инженерам античита и исследователям защитной безопасности, работающим в области античита / детекции: HWID.
