Este proyecto se centra en usar MMIO (Memory-Mapped I/O) para leer la clave pública de endoso (Endorsement Key) de TPM 2.0. Esta prueba de concepto (POC) demuestra cómo el Memory-Mapped I/O (MMIO) puede usarse para consultar directamente el estado del TPM y la EK desde el propio chip, eludiendo cualquier hook del sistema operativo. Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el ámbito anti-cheat / detección: HWID.
