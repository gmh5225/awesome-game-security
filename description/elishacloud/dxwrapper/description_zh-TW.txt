DxWrapper 是一個 C++ DirectX 相容性包裝 DLL，透過攔截並重對應圖形與輸入 API，協助較舊的 Windows 遊戲在現代系統上正確執行。它為 DirectDraw、Direct3D 8/9、DirectInput、DirectSound 及相關系統函式庫提供代理包裝，行為可透過 INI 設定。專案整合了 DDrawCompat、d3d8to9 轉換與 Dd7to9 轉換，並使用 API Hook（包括 Detours 與基於模式的 Hook）以及 GDI 與視窗程序修復，以處理全螢幕與渲染異常。可選工具如日誌、偵錯疊加層與資源傾印，便於診斷渲染與裝置問題。面向遊戲相容性工作、舊版 DirectX 遊戲逆向，以及修復老遊戲圖形問題的玩家與開發者。
