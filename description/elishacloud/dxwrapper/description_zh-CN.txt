DxWrapper 是一个 C++ DirectX 兼容性包装 DLL，通过拦截并重映射图形与输入 API，帮助较旧的 Windows 游戏在现代系统上正确运行。它为 DirectDraw、Direct3D 8/9、DirectInput、DirectSound 及相关系统库提供代理包装，行为可通过 INI 配置。项目集成了 DDrawCompat、d3d8to9 转换与 Dd7to9 转换，并使用 API Hook（包括 Detours 与基于模式的 Hook）以及 GDI 与窗口过程修复，以处理全屏与渲染异常。可选工具如日志、调试叠加层与资源转储，便于诊断渲染与设备问题。面向游戏兼容性工作、旧版 DirectX 游戏逆向，以及修复老游戏图形问题的玩家与开发者。
