DxWrapper is a C++ DirectX compatibility wrapper DLL that helps older Windows games run correctly on modern systems by intercepting and remapping graphics and input APIs. It provides proxy wrappers for DirectDraw, Direct3D 8/9, DirectInput, DirectSound, and related system libraries, with configurable behavior driven by INI settings. The project integrates DDrawCompat, d3d8to9 translation, and Dd7to9 conversion, and uses API hooking (including Detours and pattern-based hooks) plus GDI and window-procedure fixes for fullscreen and rendering quirks. Optional tooling such as logging, debug overlays, and resource dumping supports diagnosis of rendering and device issues. It is aimed at game compatibility work, reverse engineering of legacy DirectX titles, and players or developers fixing old-game graphics problems.
