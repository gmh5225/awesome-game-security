DxWrapper は、グラフィックスと入力 API を傍受して再マップすることで、古い Windows ゲームが現代のシステムで正しく動作するよう支援する C++ 製 DirectX 互換ラッパー DLL です。DirectDraw、Direct3D 8/9、DirectInput、DirectSound および関連システムライブラリ向けのプロキシラッパーを提供し、INI 設定で動作を構成できます。DDrawCompat、d3d8to9 変換、Dd7to9 変換を統合し、API フッキング（Detours およびパターンベースのフックを含む）に加え、GDI とウィンドウプロシージャの修正でフルスクリーンやレンダリングの不具合に対応します。ログ、デバッグオーバーレイ、リソースダンプなどの任意ツールで、レンダリングやデバイス問題の診断を支援します。ゲーム互換性作業、レガシー DirectX タイトルのリバースエンジニアリング、古いゲームのグラフィックス問題を直すプレイヤーや開発者向けです。
