이 프로젝트는 WinAFL로, AFL(American Fuzzy Lop) 커버리지 유도 퍼저의 Windows 이식판입니다. DynamoRIO 또는 Intel PT로 바이너리를 계측하여 퍼징 중 코드 커버리지를 추적하고, 빠른 인프로세스 퍼징을 위한 지속 모드를 지원하며, 코퍼스 최소화와 크래시 분류 유틸리티를 포함합니다. 이 C 도구는 대상 함수를 후킹하고 입력을 변이시켜 소스 코드 없이 클로즈드소스 Windows 바이너리를 퍼징할 수 있습니다. Windows 애플리케이션, 드라이버, 파서의 보안 취약점을 찾는 취약점 연구자를 대상으로 합니다.
