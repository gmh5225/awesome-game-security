Questo progetto è WinAFL, un port Windows del fuzzer guidato dalla copertura AFL (American Fuzzy Lop). Usa DynamoRIO o Intel PT per l’instrumentazione binaria per tracciare la copertura del codice durante il fuzzing, supporta la modalità persistente per un fuzzing rapido in-process e include utilità per la minimizzazione del corpus e il triage dei crash. Lo strumento in C può fare fuzzing di binari Windows closed-source senza codice sorgente agganciando le funzioni target e mutando l’input. È rivolto ai ricercatori di vulnerabilità che fuzzano applicazioni, driver e parser Windows alla ricerca di bug di sicurezza.
