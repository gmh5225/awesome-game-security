本プロジェクトは WinAFL であり、AFL（American Fuzzy Lop）のカバレッジ誘導ファザーの Windows 移植版です。DynamoRIO または Intel PT によるバイナリ計装でファジング中のコードカバレッジを追跡し、高速なインプロセスファジング向けの永続モードに対応し、コーパス最小化とクラッシュ分類ユーティリティを備えます。この C 製ツールは、対象関数をフックして入力を変異させることで、ソースコードなしにクローズドソースの Windows バイナリをファジングできます。Windows アプリケーション、ドライバ、パーサの脆弱性を探すセキュリティ研究者向けです。
