Этот проект — WinAFL, порт для Windows покрывающего фаззера AFL (American Fuzzy Lop). Он использует DynamoRIO или Intel PT для бинарной инструментации, чтобы отслеживать покрытие кода при фаззинге, поддерживает persistent mode для быстрого фаззинга в процессе и включает утилиты минимизации корпуса и triage сбоев. Инструмент на C может фаззить закрытые Windows-бинарники без исходного кода, перехватывая целевые функции и мутируя входные данные. Он предназначен для исследователей уязвимостей, фаззящих приложения, драйверы и парсеры Windows в поисках ошибок безопасности.
