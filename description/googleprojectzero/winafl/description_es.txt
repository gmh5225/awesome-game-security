Este proyecto es WinAFL, un port a Windows del fuzzer guiado por cobertura AFL (American Fuzzy Lop). Usa DynamoRIO o Intel PT para instrumentación binaria y seguir la cobertura de código durante el fuzzing, admite modo persistente para fuzzing rápido en proceso e incluye utilidades de minimización de corpus y triaje de fallos. La herramienta en C puede hacer fuzzing de binarios Windows de código cerrado sin código fuente enganchando funciones objetivo y mutando la entrada. Está orientada a investigadores de vulnerabilidades que hacen fuzzing de aplicaciones, controladores y parsers de Windows en busca de fallos de seguridad.
