Este proyecto es una guía sobre Unreal Engine que incluye las aplicaciones, bibliotecas y herramientas que te ayudarán a ser un desarrollador de Unreal Engine mejor y más eficiente.
Nota: puedes convertir fácilmente este archivo markdown a PDF en VSCode con la práctica extensión Markdown PDF.
Es útil principalmente para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en investigación anti-trampas y trabajo a nivel de kernel en el área de motores de juego / guías.
