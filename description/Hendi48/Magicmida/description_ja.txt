Magicmida は、Themida で保護された実行ファイル向けの Windows アンパッキングツールで、主に Object Pascal（Delphi/Free Pascal）で書かれています。独自のユーザーモードデバッガで対象を起動またはアタッチし、Themida の保護をトレースしてアンパック済みイメージをダンプし、インポートを再構築・修復してダンプが実行できるようにします。32 ビットと 64 ビットの両方、PE および .NET ダンプ、OEP 周辺のアンチダンプ対策、BeaEngine による逆アセンブルに対応し、GUI とコマンドラインの /unpack モードを備えます。アンパック中のアンチデバッグ干渉を抑えるため、ScyllaHide 関連の設定も含まれます。Themida 保護バイナリを解析・復元するリバースエンジニアやゲームセキュリティ研究者向けです。
