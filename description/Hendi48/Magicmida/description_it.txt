Magicmida è un unpacker Windows per eseguibili protetti con Themida, scritto principalmente in Object Pascal (Delphi/Free Pascal). Guida un debugger personalizzato in user-mode per avviare o collegarsi a un target, tracciare la protezione Themida, dumpare l’immagine unpackata e ricostruire o correggere gli import affinché il dump possa essere eseguito. Il progetto copre percorsi a 32 e 64 bit, dump PE e .NET, mitigazioni anti-dump intorno all’entry point originale e disassemblaggio tramite BeaEngine, con una GUI e una modalità da riga di comando /unpack. Sono incluse impostazioni legate a ScyllaHide per ridurre le interferenze anti-debug durante l’unpacking. È pensato per reverse engineer e ricercatori di game security che analizzano o recuperano binari protetti con Themida.
