Magicmida 是一款面向 Themida 加壳可执行文件的 Windows 脱壳工具，主要使用 Object Pascal（Delphi/Free Pascal）编写。它通过自定义用户态调试器启动或附加目标进程，跟踪 Themida 保护流程，转储脱壳后的映像，并重建或修复导入表，使转储结果可运行。项目同时覆盖 32 位与 64 位路径、PE 与 .NET 转储、围绕原始入口点的反转储缓解，以及基于 BeaEngine 的反汇编，并提供 GUI 与命令行 /unpack 模式。其中包含与 ScyllaHide 相关的设置，以降低脱壳过程中的反调试干扰。该工具面向分析或还原 Themida 保护二进制的逆向工程师与游戏安全研究人员。
