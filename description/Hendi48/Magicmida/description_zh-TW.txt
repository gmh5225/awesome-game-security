Magicmida 是一款針對 Themida 加殼執行檔的 Windows 脫殼工具，主要以 Object Pascal（Delphi/Free Pascal）撰寫。它透過自訂使用者模式偵錯器啟動或附加目標，追蹤 Themida 的保護流程，傾印脫殼後的映像，並重建或修復匯入表，使傾印結果可執行。專案涵蓋 32 位元與 64 位元路徑、PE 與 .NET 傾印、圍繞原始進入點的反傾印緩解，以及透過 BeaEngine 的反組譯，並提供 GUI 與命令列 /unpack 模式。其中包含與 ScyllaHide 相關的設定，以降低脫殼期間的反偵錯干擾。此工具面向分析或還原 Themida 保護二進位的逆向工程師與遊戲安全研究人員。
