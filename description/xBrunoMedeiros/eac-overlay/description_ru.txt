Этот проект — proof-of-concept рендерер оверлеев, работающий вместе с играми, защищёнными EasyAntiCheat. Он показывает, как рисовать ESP и другие визуальные оверлеи методами, которые обходят мониторинг оверлеев EAC, например через альтернативные поверхности рендеринга или манипуляцию окнами. Реализация на C++ демонстрирует, как отрисовывать оверлеи, не вызывая срабатывания античита по скриншотам или обнаружению оверлеев. Предназначен для исследователей безопасности игр, изучающих обход детекта оверлеев и ограничения визуального мониторинга античита.
