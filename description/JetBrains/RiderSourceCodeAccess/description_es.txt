Este proyecto es un plugin de Unreal Engine que integra Rider como editor de código fuente seleccionable dentro de Unreal Editor. Está implementado en C++ con módulos de plugin de Unreal e incluye lógica para descubrir instalaciones de Rider en Windows, macOS y Linux. El plugin admite distintas versiones de Rider y rutas de instalación personalizadas configurables. El caso de uso principal es mejorar los flujos de trabajo de desarrollo de juegos en C++ para equipos de Unreal que usan el ecosistema JetBrains.
