Этот проект — плагин Unreal Engine, который интегрирует Rider как выбираемый редактор исходного кода внутри Unreal Editor. Он реализован на C++ с модулями плагина Unreal и включает логику обнаружения установок Rider в Windows, macOS и Linux. Плагин поддерживает разные версии Rider и настраиваемые пользовательские пути установки. Основной сценарий использования — улучшение рабочих процессов C++-разработки игр для команд Unreal, работающих в экосистеме JetBrains.
