Questo progetto è un plugin per Unreal Engine che integra Rider come editor di codice sorgente selezionabile all'interno di Unreal Editor. È implementato in C++ con moduli plugin Unreal e include la logica per rilevare le installazioni di Rider su Windows, macOS e Linux. Il plugin supporta diverse versioni di Rider e percorsi di installazione personalizzati configurabili. Il caso d'uso principale è migliorare i workflow di sviluppo di giochi in C++ per i team Unreal che usano l'ecosistema JetBrains.
