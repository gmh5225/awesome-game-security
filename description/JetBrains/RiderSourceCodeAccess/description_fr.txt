Ce projet est un plugin Unreal Engine qui intègre Rider comme éditeur de code source sélectionnable dans Unreal Editor. Il est implémenté en C++ avec des modules de plugin Unreal et inclut une logique pour découvrir les installations Rider sous Windows, macOS et Linux. Le plugin prend en charge différentes versions de Rider ainsi que des chemins d'installation personnalisés configurables. Le principal cas d'usage est d'améliorer les workflows de développement de jeux C++ pour les équipes Unreal utilisant l'écosystème JetBrains.
