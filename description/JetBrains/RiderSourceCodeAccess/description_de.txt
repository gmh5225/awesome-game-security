Dieses Projekt ist ein Unreal-Engine-Plugin, das Rider als auswählbaren Quellcode-Editor in den Unreal Editor integriert. Es ist in C++ mit Unreal-Plugin-Modulen implementiert und enthält Logik zum Erkennen von Rider-Installationen unter Windows, macOS und Linux. Das Plugin unterstützt verschiedene Rider-Versionen sowie konfigurierbare benutzerdefinierte Installationspfade. Der Hauptanwendungsfall ist die Verbesserung von C++-Spieleentwicklungs-Workflows für Unreal-Teams, die das JetBrains-Ökosystem nutzen.
