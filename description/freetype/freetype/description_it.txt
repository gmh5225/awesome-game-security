Libreria portabile di rasterizzazione dei font in C che genera immagini di glifi di alta qualità dai formati TrueType, OpenType, CFF, Type 1 e font bitmap. Implementa l'interpretazione del hinting tramite bytecode, il rendering anti-aliasing, il posizionamento subpixel e l'auto-hinting, esponendo un'architettura modulare con driver di font plug-in. FreeType si compila su praticamente ogni piattaforma (incluse DOS, Amiga, OS/2, Symbian) tramite il suo sistema di build a strati che supporta Make, CMake e Meson.
