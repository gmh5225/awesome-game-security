Biblioteca portable de rasterización de fuentes en C que genera imágenes de glifos de alta calidad a partir de formatos TrueType, OpenType, CFF, Type 1 y fuentes bitmap. Implementa interpretación de hinting por bytecode, renderizado antialiased, posicionamiento subpíxel y auto-hinting, y expone una arquitectura modular con controladores de fuentes enchufables. FreeType se compila en prácticamente todas las plataformas (incluidas DOS, Amiga, OS/2 y Symbian) mediante su sistema de build por capas que admite Make, CMake y Meson.
