Este proyecto es un sistema de radar DMA para Counter-Strike 2 que transmite el estado del juego a una vista de mapa táctico basada en el navegador. Utiliza un backend Java con Spring Boot y componentes WebSocket, junto con un frontend Leaflet y JavaScript para el renderizado en vivo. El backend lee memoria a través de interfaces relacionadas con VMM y LeechCore, mientras que los recursos de mapa e iconos admiten varios mapas competitivos y marcadores de jugadores. Se usa principalmente para experimentos de conciencia situacional externa en contextos de seguridad de juegos e investigación anti-trampas.
