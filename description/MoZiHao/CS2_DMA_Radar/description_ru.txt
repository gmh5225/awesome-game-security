Этот проект — система DMA-радара для Counter-Strike 2, которая транслирует состояние игры в тактическую карту в браузере. Используются Java-бэкенд на Spring Boot с компонентами WebSocket и фронтенд на Leaflet и JavaScript для отрисовки в реальном времени. Бэкенд читает память через интерфейсы, связанные с VMM и LeechCore, а ресурсы карт и иконки поддерживают несколько соревновательных карт и маркеры игроков. В основном применяется для внешних экспериментов по situational awareness в исследованиях игровой безопасности и античита.
