本プロジェクトは、ゲーム状態をブラウザベースの戦術マップ表示へストリーミングする Counter-Strike 2 向け DMA レーダーシステムです。バックエンドは Java（Spring Boot および WebSocket コンポーネント）で構成され、フロントエンドは Leaflet と JavaScript によるライブ描画を行います。バックエンドは VMM および LeechCore 関連インターフェースを通じてメモリを読み取り、マップ資産とアイコンは複数の競技マップとプレイヤーマーカーに対応します。主にゲームセキュリティおよびアンチチート研究における外部シチュエーショナル・アウェアネス実験に用いられます。
