Ce projet est une application UEFI permettant de dumper la RAM physique depuis un environnement de démarrage live.
Il est écrit en C avec les outils de build UEFI et conçu pour s'exécuter depuis un support USB avec un shell UEFI.
Le dépôt inclut des scripts, des artefacts de build et de la documentation pour collecter des images mémoire sur des systèmes modernes.
Il sert principalement à l'acquisition forensique et à la recherche en sécurité bas niveau lorsqu'une capture mémoire hors ligne est requise.
