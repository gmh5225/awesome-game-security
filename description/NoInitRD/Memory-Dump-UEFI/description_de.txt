Dieses Projekt ist eine UEFI-Anwendung zum Dumpen des physischen RAM aus einer Live-Boot-Umgebung.
Es ist in C mit UEFI-Build-Tools geschrieben und so konzipiert, dass es von USB-Medien zusammen mit einer UEFI-Shell ausgeführt werden kann.
Das Repository enthält Skripte, Build-Artefakte und Dokumentation zur Erfassung von Speicherabbildern auf modernen Systemen.
Es wird hauptsächlich für forensische Erfassung und Low-Level-Sicherheitsforschung eingesetzt, wenn eine Offline-Speichererfassung erforderlich ist.
