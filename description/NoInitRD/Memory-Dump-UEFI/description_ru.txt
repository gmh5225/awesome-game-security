Этот проект — UEFI-приложение для дампа физической оперативной памяти из среды live-boot.
Оно написано на C с использованием инструментов сборки UEFI и рассчитано на запуск с USB-носителя вместе с UEFI shell.
Репозиторий включает скрипты, артефакты сборки и документацию для сбора образов памяти на современных системах.
В основном применяется для криминалистического изъятия и низкоуровневых исследований безопасности, когда требуется офлайн-захват памяти.
