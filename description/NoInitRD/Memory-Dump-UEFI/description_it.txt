Questo progetto è un'applicazione UEFI per il dump della RAM fisica da un ambiente di avvio live.
È scritta in C con gli strumenti di build UEFI ed è progettata per essere eseguita da supporti USB insieme a una shell UEFI.
Il repository include script, artefatti di build e documentazione per raccogliere immagini di memoria su sistemi moderni.
È usata principalmente per l'acquisizione forense e la ricerca di sicurezza a basso livello quando è richiesta una cattura di memoria offline.
