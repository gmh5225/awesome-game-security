Este proyecto es una aplicación UEFI para volcar la RAM física desde un entorno de arranque en vivo.
Está escrita en C con herramientas de compilación UEFI y diseñada para ejecutarse desde medios USB junto con un shell UEFI.
El repositorio incluye scripts, artefactos de compilación y documentación para recopilar imágenes de memoria en sistemas modernos.
Se usa principalmente para adquisición forense e investigación de seguridad de bajo nivel cuando se requiere una captura de memoria sin conexión.
