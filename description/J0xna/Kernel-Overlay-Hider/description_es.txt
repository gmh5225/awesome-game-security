Este proyecto es una prueba de concepto que oculta el handle de una ventana overlay de la enumeración de ventanas manipulando estructuras de ventana del lado del kernel.
Utiliza un controlador de kernel de Windows junto con programas de prueba en modo usuario para localizar y modificar punteros relacionados con win32k y datos vinculados a TAGWND.
El código se centra en técnicas de estilo DKOM e incluye ejemplos para activar el comportamiento desde un contexto de overlay DirectX.
Está pensado para la investigación de bajo nivel de los internals de Windows en torno a la visibilidad de overlays y los mecanismos de evasión anti-cheat.
