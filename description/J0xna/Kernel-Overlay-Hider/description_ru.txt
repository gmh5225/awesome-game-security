Этот проект — proof of concept, который скрывает handle окна overlay от перечисления окон путём манипуляции оконными структурами на стороне ядра.
Он использует драйвер ядра Windows и пользовательские тестовые программы для поиска и изменения указателей, связанных с win32k, и данных, связанных с TAGWND.
Код сосредоточен на техниках в стиле DKOM и включает примеры запуска этого поведения из контекста DirectX overlay.
Предназначен для низкоуровневого исследования внутренностей Windows в области видимости overlay и механизмов обхода античита.
