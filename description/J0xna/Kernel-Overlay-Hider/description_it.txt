Questo progetto è una proof of concept che nasconde l'handle di una finestra overlay dall'enumerazione delle finestre manipolando le strutture finestra lato kernel.
Usa un driver kernel Windows insieme a programmi di test in user-mode per individuare e modificare puntatori correlati a win32k e dati collegati a TAGWND.
Il codice si concentra su tecniche in stile DKOM e include esempi per attivare il comportamento da un contesto di overlay DirectX.
È destinato alla ricerca a basso livello sugli internals di Windows intorno alla visibilità degli overlay e ai meccanismi di evasione anti-cheat.
