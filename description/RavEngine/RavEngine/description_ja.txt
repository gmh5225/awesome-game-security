本プロジェクトは、データ指向の Entity Component System アーキテクチャを採用したモダンな C++ ゲームエンジンであり、RGL 抽象レイヤー経由の Vulkan レンダリングと、GameNetworkingSockets によるマルチプレイヤーネットワーキングを備えています。クロスプラットフォーム対応、CMake ベースのビルドシステム、リソースパッキングパイプライン、および入力・GUI・オーディオなどの包括的なエンジンサブシステムが特徴です。主に、Vulkan レンダリングを伴う現代的な ECS ベースのエンジン設計を研究するゲームエンジン開発者やグラフィックス研究者に役立ちます。
