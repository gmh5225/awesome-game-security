이 프로젝트는 데이터 지향 Entity Component System 아키텍처로 구축된 현대적인 C++ 게임 엔진으로, RGL 추상화 계층을 통한 Vulkan 기반 렌더링과 GameNetworkingSockets를 이용한 멀티플레이어 네트워킹을 제공합니다. 크로스 플랫폼 지원, CMake 기반 빌드 시스템, 리소스 패킹 파이프라인, 그리고 입력·GUI·오디오를 포함한 포괄적인 엔진 서브시스템을 갖추고 있습니다. 주로 Vulkan 렌더링을 활용한 현대적인 ECS 기반 엔진 설계를 연구하는 게임 엔진 개발자와 그래픽스 연구자에게 유용합니다.
