Questo progetto è una reimplementazione o modifica del motore di gioco di Red Alert 2. Lavora con la logica di Command & Conquer: Red Alert 2 / Yuri's Revenge, implementando o estendendo meccaniche RTS tra cui IA delle unità, costruzione di edifici, gestione delle risorse, networking multiplayer e rendering delle mappe. È rivolto a modder di RTS classici e a ricercatori di motori di gioco che studiano l'architettura dei motori RTS dell'era Westwood.
