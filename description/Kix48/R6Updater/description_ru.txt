Этот проект — утилита для дампа и обновления оффсетов для Rainbow Six Siege.
Она использует pattern scanning на C++ и модули работы с памятью, чтобы находить менеджеры и обновлять чувствительные к патчам оффсеты после обновлений.
Кодовая база оформлена как инструмент Visual Studio x64 для Windows с раздельными компонентами сканирования и работы с памятью.
Сценарий использования сосредоточен на обновлении сигнатур и извлечении рабочих оффсетов во время игровой сессии.
В первую очередь рассчитан на реверс-инженеров и разработчиков читов, поддерживающих игроспецифичные данные.
