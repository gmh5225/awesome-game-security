Este proyecto es una utilidad de volcado y actualización de offsets para Rainbow Six Siege.
Usa pattern scanning en C++ y módulos de memoria para localizar managers y actualizar offsets sensibles tras las actualizaciones.
El código está estructurado como una herramienta Windows x64 de Visual Studio, con componentes separados para el escaneo y el manejo de memoria.
Su flujo de uso se centra en refrescar firmas y extraer offsets utilizables durante una sesión de juego.
Está dirigido principalmente a ingenieros inversos y desarrolladores de cheats que mantienen datos específicos del juego.
