이 프로젝트는 게임과 도구에서 널리 쓰이는 즉시 모드 GUI 라이브러리 Dear ImGui를 위한 공식 유지보수 소형 확장 모음입니다. 키보드 조작, 읽기 전용 모드, ASCII 및 HexII 뷰, 주소 탐색, 범위 강조, 사용자 정의 읽기/쓰기 핸들러를 갖춘 미니 16진수 메모리 편집기가 포함됩니다. 또한 여러 ImGui 컨텍스트 간에 Z 순서, 입력 라우팅, 드래그 앤 드롭을 관리하는 멀티 컨텍스트 컴포지터와, ImDrawData를 스냅샷하여 지연 그리기에 쓰는 스레드 렌더링 헬퍼도 제공합니다. 주로 C++로 작성되었으며, 인게임 디버깅 UI와 메모리 검사가 필요한 게임 개발자, 도구 제작자, 리버스 엔지니어를 대상으로 합니다.
