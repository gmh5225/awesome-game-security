This project is a collection of officially maintained small extensions for Dear ImGui, the immediate-mode GUI library widely used in games and tools. It includes a mini hexadecimal memory editor with keyboard controls, read-only mode, ASCII and HexII views, address navigation, range highlighting, and custom read/write handlers. It also provides a multi-context compositor for managing z-order, input routing, and drag-and-drop across multiple ImGui contexts, plus a threaded rendering helper that snapshots ImDrawData for deferred drawing. Written primarily in C++, these components are aimed at game developers, tool authors, and reverse engineers who need in-game debugging UIs and memory inspection.
