本プロジェクトは、ゲームやツールで広く使われる即時モード GUI ライブラリ Dear ImGui 向けの、公式に保守されている小型拡張のコレクションです。キーボード操作、読み取り専用モード、ASCII および HexII 表示、アドレス移動、範囲ハイライト、カスタム読み書きハンドラを備えたミニ十六進メモリエディタが含まれます。また、複数の ImGui コンテキスト間で Z オーダー、入力ルーティング、ドラッグ＆ドロップを管理するマルチコンテキストコンポジタと、ImDrawData をスナップショットして遅延描画するスレッド化レンダリングヘルパーも提供します。主に C++ で書かれており、ゲーム内デバッグ UI やメモリ検査を必要とするゲーム開発者、ツール作者、リバースエンジニア向けです。
