Этот проект — коллекция официально поддерживаемых небольших расширений для Dear ImGui, библиотеки GUI с немедленным режимом, широко используемой в играх и инструментах. Включает мини-редактор шестнадцатеричной памяти с управлением с клавиатуры, режимом только чтения, представлениями ASCII и HexII, навигацией по адресам, подсветкой диапазонов и пользовательскими обработчиками чтения/записи. Также предоставляет мультиконтекстный композитор для управления Z-порядком, маршрутизацией ввода и drag-and-drop между несколькими контекстами ImGui, а также потоковый помощник отрисовки, который делает снимок ImDrawData для отложенного рисования. Написанные в основном на C++, эти компоненты предназначены для разработчиков игр, авторов инструментов и инженеров по реверс-инжинирингу, которым нужны внутриигровые отладочные UI и инспекция памяти.
