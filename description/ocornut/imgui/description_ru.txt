Этот проект — Dear ImGui, лёгкая библиотека графического пользовательского интерфейса в immediate mode для C++. Она формирует вершинные буферы, которые можно отрисовывать любой графической API (DirectX 9/10/11/12, OpenGL, Vulkan, Metal), и предоставляет окна, виджеты, графики, таблицы, docking и поддержку multi-viewport при минимальной настройке. Библиотека широко используется для внутриигровых отладочных оверлеев, GUI инструментов и интерфейсов чит-меню в контексте game hacking. Она предназначена для разработчиков игр, программистов инструментов и всех, кому нужен быстрый и переносимый GUI-рендеринг внутри приложения.
