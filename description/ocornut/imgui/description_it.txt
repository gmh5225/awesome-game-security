Questo progetto è Dear ImGui, una libreria di interfaccia grafica utente in immediate mode, leggera e senza bloat, per C++. Produce buffer di vertici renderizzabili con qualsiasi API grafica (DirectX 9/10/11/12, OpenGL, Vulkan, Metal) e offre finestre, widget, grafici, tabelle, docking e supporto multi-viewport con una configurazione minima. La libreria è ampiamente usata per overlay di debug in-game, GUI per strumenti e interfacce di menu cheat nei contesti di game hacking. È pensata per sviluppatori di giochi, programmatori di tool e chiunque abbia bisogno di un rendering GUI veloce e portabile all'interno di un'applicazione.
