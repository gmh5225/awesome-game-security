Este proyecto es Dear ImGui, una biblioteca de interfaz gráfica de usuario en modo inmediato, ligera y sin sobrecarga, para C++. Genera buffers de vértices que se pueden renderizar con cualquier API gráfica (DirectX 9/10/11/12, OpenGL, Vulkan, Metal) y ofrece ventanas, widgets, gráficos, tablas, docking y soporte multi-viewport con una configuración mínima. La biblioteca se usa ampliamente para overlays de depuración en el juego, GUIs de herramientas e interfaces de menús de trampas en contextos de game hacking. Está dirigida a desarrolladores de juegos, programadores de herramientas y a cualquiera que necesite un renderizado GUI rápido y portable dentro de una aplicación.
