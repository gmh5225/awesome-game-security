Este proyecto es una solución root de kernel Android modificada, basada en KernelSU, con cambios de comportamiento orientados a la depuración. Combina código C del lado del kernel con una aplicación Android y recursos de compilación para gestionar el acceso root, el control de perfiles y el manejo de políticas personalizadas. Las modificaciones incluyen comprobaciones del gestor relajadas, opciones orientadas a SELinux permisivo, soporte de scripts init y amplias funciones de comodidad root para flujos de prueba. Está pensado principalmente para investigación de seguridad en Android, depuración de sistemas y experimentación en entornos con root.
