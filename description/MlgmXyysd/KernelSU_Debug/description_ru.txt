Этот проект — модифицированное решение root для ядра Android на базе KernelSU с изменениями поведения, ориентированными на отладку. Оно сочетает код на C со стороны ядра с Android-приложением и ресурсами сборки для управления root-доступом, контроля профилей и обработки пользовательских политик. Изменения включают ослабленные проверки менеджера, более разрешающие параметры, связанные с SELinux, поддержку init-скриптов и широкие удобства root для тестовых сценариев. Предназначено в первую очередь для исследований безопасности Android, системной отладки и экспериментов в окружениях с root.
