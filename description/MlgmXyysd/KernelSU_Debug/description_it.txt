Questo progetto è una soluzione root per kernel Android modificata, basata su KernelSU, con cambiamenti di comportamento orientati al debugging. Combina codice C lato kernel con un’app Android e asset di build per gestire l’accesso root, il controllo dei profili e la gestione di policy personalizzate. Le modifiche includono controlli del manager allentati, opzioni orientate a SELinux permissivo, supporto agli script init e ampie funzioni di comodità root per i flussi di test. È pensato soprattutto per la ricerca di sicurezza Android, il debugging di sistema e la sperimentazione in ambienti rootati.
