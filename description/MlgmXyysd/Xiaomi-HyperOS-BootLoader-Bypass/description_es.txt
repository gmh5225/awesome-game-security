Este proyecto es una prueba de concepto para eludir ciertas restricciones de vinculación de cuenta del bootloader de Xiaomi HyperOS. La implementación se centra en herramientas PHP y scripts de automatización, con auxiliares Docker y shell, además de documentación multilingüe. Aprovecha interacciones orientadas a ADB mediante bibliotecas relacionadas para ejecutar el flujo de desbloqueo de forma reproducible. El caso de uso principal es la investigación en seguridad móvil y la validación de debilidades y mitigaciones de las restricciones del bootloader.
