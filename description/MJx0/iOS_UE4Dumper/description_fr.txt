Ce projet est un dumper Unreal Engine pour iOS fourni sous forme de tweak MobileSubstrate. Sa base de code est principalement en C++ et Objective-C++, cible les appareils arm64 et arm64e, et découvre les structures centrales du moteur par motifs. Il peut extraire offsets, classes, structures, énumérations et fonctions, puis générer un JSON de symboles prêt pour l'analyse avec des outils comme IDA et Ghidra. Le cas d'usage principal est la rétro-ingénierie de jeux iOS et la recherche en sécurité Unreal Engine sur titres mobiles.
