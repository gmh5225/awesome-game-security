Este proyecto es un dumper de Unreal Engine para iOS distribuido como tweak de MobileSubstrate. Su código está principalmente en C++ y Objective-C++, apunta a dispositivos arm64 y arm64e, y descubre estructuras centrales del motor mediante patrones. Puede volcar offsets, clases, structs, enums y funciones, y generar JSON de símbolos listo para análisis con herramientas como IDA y Ghidra. El caso de uso principal es la ingeniería inversa de juegos iOS y la investigación de seguridad de Unreal Engine en títulos móviles.
