Этот проект — дампер Unreal Engine для iOS, поставляемый как твик MobileSubstrate. Кодовая база в основном на C++ и Objective-C++, рассчитана на устройства arm64 и arm64e и использует поиск по сигнатурам для обнаружения ключевых структур движка. Он может дампить смещения, классы, структуры, перечисления и функции, а затем генерировать JSON символов, готовый для анализа в таких инструментах, как IDA и Ghidra. Основное применение — реверс-инжиниринг iOS-игр и исследование безопасности Unreal Engine на мобильных проектах.
