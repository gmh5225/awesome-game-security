Этот проект — библиотека патчинга памяти во время выполнения для приложений Android и iOS.
Она реализована на C++ и предоставляет утилиты для сканирования памяти, проверки указателей, работы с файлами и создания патчей кода.
Библиотека интегрируется с ассемблером Keystone для генерации патчей на основе ассемблера и включает предсобранные бинарники и примеры для конкретных платформ.
Её часто используют в реверс-инжиниринге мобильных игр и исследованиях безопасности, когда требуется контролируемое изменение памяти.
