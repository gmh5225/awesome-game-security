Este proyecto es una biblioteca de parcheo de memoria en tiempo de ejecución para aplicaciones Android e iOS.
Está implementada en C++ y ofrece utilidades para el escaneo de memoria, la validación de punteros, ayudantes de archivos y la creación de parches de código.
La biblioteca se integra con el ensamblador Keystone para la generación de parches basados en ensamblador e incluye binarios precompilados y ejemplos específicos de cada plataforma.
Se usa habitualmente en ingeniería inversa de juegos móviles e investigación de seguridad cuando se requiere una modificación controlada en memoria.
