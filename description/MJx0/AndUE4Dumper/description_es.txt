Este proyecto es un kit de herramientas de dumping de Unreal Engine para Android que genera salida al estilo SDK y artefactos de análisis a partir de juegos en ejecución. Está escrito principalmente en C++ para el NDK de Android y puede construirse como ejecutable externo o como biblioteca compartida inyectable. Sus capacidades clave incluyen volcar offsets del motor, clases, estructuras, enumeraciones y funciones, además de producir scripts de símbolos JSON para desensambladores como IDA y Ghidra. Lo usan principalmente ingenieros de ingeniería inversa de juegos móviles e investigadores de seguridad de juegos que necesitan metadatos estructurados de Unreal Engine para un análisis más profundo.
