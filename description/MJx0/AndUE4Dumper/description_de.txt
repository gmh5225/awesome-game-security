Dieses Projekt ist ein Android-Toolkit zum Dump von Unreal Engine, das aus laufenden Spielen SDK-ähnliche Ausgaben und Analyseartefakte erzeugt. Es ist hauptsächlich in C++ für das Android NDK geschrieben und kann entweder als externes Executable oder als injizierbare Shared Library gebaut werden. Zu den Kernfunktionen gehören das Dump von Engine-Offsets, Klassen, Structs, Enums und Funktionen sowie die Erzeugung von JSON-Symbolskripten für Disassembler wie IDA und Ghidra. Es wird vor allem von Reverse Engineers mobiler Spiele und Game-Security-Forschern genutzt, die strukturierte Unreal-Engine-Metadaten für tiefere Analysen benötigen.
