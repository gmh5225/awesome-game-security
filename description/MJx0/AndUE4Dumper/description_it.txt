Questo progetto è un toolkit di dumping Unreal Engine per Android che genera output in stile SDK e artefatti di analisi a partire da giochi in esecuzione. È scritto principalmente in C++ per l'Android NDK e può essere compilato come eseguibile esterno o come libreria condivisa iniettabile. Le capacità principali includono il dump di offset del motore, classi, struct, enum e funzioni, oltre alla produzione di script di simboli JSON per disassemblatori come IDA e Ghidra. È usato soprattutto da reverse engineer di giochi mobile e ricercatori di sicurezza dei giochi che necessitano di metadati Unreal Engine strutturati per analisi più approfondite.
