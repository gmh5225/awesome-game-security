이 프로젝트는 실행 중인 게임에서 SDK 스타일 출력과 분석 아티팩트를 생성하는 Android용 Unreal Engine 덤프 툴킷입니다. 주로 C++로 Android NDK용으로 작성되며, 외부 실행 파일 또는 주입 가능한 공유 라이브러리로 빌드할 수 있습니다. 주요 기능으로는 엔진 오프셋, 클래스, 구조체, 열거형, 함수 덤프와 IDA, Ghidra 같은 디스어셈블러용 JSON 심볼 스크립트 생성이 포함됩니다. 심층 분석을 위해 구조화된 Unreal Engine 메타데이터가 필요한 모바일 게임 리버스 엔지니어와 게임 보안 연구자가 주로 사용합니다.
