Этот проект — набор инструментов для дампа Unreal Engine на Android, который формирует вывод в стиле SDK и артефакты анализа из запущенных игр. Он написан в основном на C++ для Android NDK и может собираться как внешний исполняемый файл или как внедряемая разделяемая библиотека. Ключевые возможности включают дамп смещений движка, классов, структур, перечислений и функций, а также создание JSON-скриптов символов для дизассемблеров вроде IDA и Ghidra. Им в основном пользуются специалисты по реверс-инжинирингу мобильных игр и исследователи игровой безопасности, которым нужны структурированные метаданные Unreal Engine для более глубокого анализа.
