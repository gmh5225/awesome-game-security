本プロジェクトは、実行中のゲームから SDK 形式の出力と解析用アーティファクトを生成する Android 向け Unreal Engine ダンプツールキットです。主に C++ で Android NDK 向けに書かれており、外部実行ファイルまたは注入可能な共有ライブラリとしてビルドできます。主な機能は、エンジンのオフセット、クラス、構造体、列挙型、関数のダンプに加え、IDA や Ghidra などの逆アセンブラ向け JSON シンボルスクリプトの生成です。構造化された Unreal Engine メタデータを必要とするモバイルゲームのリバースエンジニアやゲームセキュリティ研究者に主に利用されます。
