Dieses Projekt ist eine kleine plattformübergreifende Grafikbibliothek, die eine einheitliche API für Windows, macOS, Linux, iOS und Android bereitstellt.
Der Kern von TIGR ist eine einfache Framebuffer-Bibliothek.
Sie ist vor allem für Engine- und Tooling-Entwickler nützlich, die an Asset-Pipelines im Bereich Bildcodecs arbeiten.
