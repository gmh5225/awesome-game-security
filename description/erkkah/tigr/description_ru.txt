Этот проект — небольшая кроссплатформенная графическая библиотека, предоставляющая единый API для Windows, macOS, Linux, iOS и Android.
Ядро TIGR — простая библиотека framebuffer.
Она в основном полезна разработчикам движков и инструментов, работающим над пайплайнами ассетов в области кодеков изображений.
