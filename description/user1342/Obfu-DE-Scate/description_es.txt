Este proyecto es una herramienta en Python para la desofuscación de APK de Android que analiza nombres ofuscados de clases y métodos para generar mapeos legibles. Produce informes HTML visuales con diagramas de jerarquía de clases y clasificación de métodos, para ayudar a los ingenieros inversos a navegar aplicaciones Android muy ofuscadas. Es especialmente útil para ingenieros inversos de Android y analistas de malware que trabajan con APK ofuscados con ProGuard o R8.
