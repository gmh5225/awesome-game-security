Questo progetto è uno strumento Python per la deoffuscazione di APK Android che analizza nomi di classi e metodi offuscati per generare mapping leggibili. Produce report HTML visuali con diagrammi della gerarchia delle classi e classificazione dei metodi, per aiutare i reverse engineer a orientarsi in applicazioni Android fortemente offuscate. È soprattutto utile per reverse engineer Android e analisti di malware che lavorano con APK offuscati con ProGuard o R8.
