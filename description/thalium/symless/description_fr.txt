Ce projet est Symless, un plugin IDA Pro pour la récupération automatisée de structures et de types à partir de binaires strippés. Il s'appuie sur l'analyse de flux de données et les motifs d'accès mémoire pour reconstruire les définitions de structures, identifier les types et tailles des champs, et propager les informations de type entre fonctions sans symboles de débogage. Ce plugin Python améliore la sortie de décompilation d'IDA pour les binaires strippés. Il s'adresse aux reverse engineers travaillant sur des binaires C/C++ strippés qui ont besoin d'une reconstruction automatique des types.
