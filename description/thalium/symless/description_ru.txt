Этот проект — Symless, плагин для IDA Pro для автоматического восстановления структур и типов из бинарников без отладочных символов. Он использует анализ потока данных и шаблоны доступа к памяти, чтобы реконструировать определения структур, определять типы и размеры полей и распространять информацию о типах между функциями без отладочных символов. Плагин на Python улучшает вывод декомпиляции IDA для stripped-бинарников. Он предназначен для реверс-инженеров, работающих со stripped C/C++ бинарниками и нуждающихся в автоматической реконструкции типов.
