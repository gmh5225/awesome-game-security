Este proyecto es Symless, un plugin de IDA Pro para la recuperación automatizada de estructuras y tipos a partir de binarios stripped. Usa análisis de flujo de datos y patrones de acceso a memoria para reconstruir definiciones de struct, identificar tipos y tamaños de campos, y propagar información de tipos entre funciones sin necesidad de símbolos de depuración. El plugin en Python mejora la salida de descompilación de IDA para binarios stripped. Está orientado a ingenieros inversos que trabajan con binarios C/C++ stripped y necesitan reconstrucción automática de tipos.
