Dieses Projekt ist ein IDA-Pro-Plugin zur Analyse von Windows Kernel-Mode Driver Framework (KMDF)-Treibern. Es erkennt und annotiert KMDF-Framework-Strukturen, Callback-Registrierungen, I/O-Warteschlangenkonfigurationen und Geräteinitialisierungsmuster in Treiberbinärdateien. Das Python-Plugin wendet KMDF-Typdefinitionen an und benennt Framework-Funktionsaufrufe, wodurch die Reverse Engineering von Treibern vereinfacht wird. Es richtet sich an Reverse Engineers, die KMDF-basierte Windows-Kernel-Treiber analysieren.
