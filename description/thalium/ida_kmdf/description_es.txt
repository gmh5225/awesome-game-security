Este proyecto es un plugin de IDA Pro para analizar controladores de Windows Kernel-Mode Driver Framework (KMDF). Identifica y anota estructuras del framework KMDF, registros de callbacks, configuraciones de colas de E/S y patrones de inicialización de dispositivos en binarios de controladores. El plugin de Python aplica definiciones de tipos KMDF y nombra las llamadas a funciones del framework, simplificando la ingeniería inversa de controladores. Está dirigido a ingenieros inversos que analizan controladores del kernel de Windows basados en KMDF.
