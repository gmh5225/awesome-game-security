Этот проект — плагин IDA Pro для анализа драйверов Windows Kernel-Mode Driver Framework (KMDF). Он выявляет и аннотирует структуры фреймворка KMDF, регистрации колбэков, конфигурации очередей ввода-вывода и шаблоны инициализации устройств в бинарных файлах драйверов. Плагин на Python применяет определения типов KMDF и назначает имена вызовам функций фреймворка, упрощая реверс-инжиниринг драйверов. Предназначен для специалистов по реверс-инжинирингу, анализирующих ядерные драйверы Windows на базе KMDF.
