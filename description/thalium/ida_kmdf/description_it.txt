Questo progetto è un plugin di IDA Pro per l’analisi dei driver Windows Kernel-Mode Driver Framework (KMDF). Identifica e annota strutture del framework KMDF, registrazioni di callback, configurazioni delle code di I/O e pattern di inizializzazione dei dispositivi nei binari dei driver. Il plugin Python applica le definizioni di tipo KMDF e assegna nomi alle chiamate di funzione del framework, semplificando il reverse engineering dei driver. È destinato ai reverse engineer che analizzano driver del kernel Windows basati su KMDF.
