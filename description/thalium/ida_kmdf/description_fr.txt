Ce projet est un plugin IDA Pro destiné à l’analyse des pilotes Windows Kernel-Mode Driver Framework (KMDF). Il identifie et annote les structures du framework KMDF, les enregistrements de callbacks, les configurations de files d’attente d’E/S et les motifs d’initialisation des périphériques dans les binaires de pilotes. Le plugin Python applique les définitions de types KMDF et nomme les appels de fonctions du framework, simplifiant ainsi la rétro-ingénierie des pilotes. Il s’adresse aux ingénieurs en rétro-ingénierie qui analysent les pilotes noyau Windows basés sur KMDF.
