Este proyecto es Rumba, un framework de Python para el análisis automatizado de binarios protegidos con VMProtect. Utiliza ejecución simbólica y análisis de trazas para reconstruir el flujo de control original y la semántica de las instrucciones a partir de los handlers de código virtualizado de VMProtect. La herramienta procesa trazas de ejecución para identificar handlers de opcodes virtuales, extraer su semántica y recuperar la lógica original del programa. Está orientado a ingenieros de ingeniería inversa y analistas de malware que desofuscan binarios protegidos con VMProtect.
