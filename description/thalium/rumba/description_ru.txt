Этот проект — Rumba, Python-фреймворк для автоматизированного анализа бинарных файлов, защищённых VMProtect. Он использует символьное выполнение и анализ трасс для восстановления исходного потока управления и семантики инструкций из виртуализированных обработчиков кода VMProtect. Инструмент обрабатывает трассы выполнения, чтобы выявить обработчики виртуальных опкодов, извлечь их семантику и восстановить исходную логику программы. Предназначен для реверс-инженеров и аналитиков вредоносного ПО, занимающихся деобфускацией бинарных файлов, защищённых VMProtect.
