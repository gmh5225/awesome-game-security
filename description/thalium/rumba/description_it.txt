Questo progetto è Rumba, un framework Python per l'analisi automatizzata di binari protetti con VMProtect. Usa esecuzione simbolica e analisi delle tracce per ricostruire il flusso di controllo originale e la semantica delle istruzioni a partire dagli handler del codice virtualizzato di VMProtect. Lo strumento elabora le tracce di esecuzione per identificare gli handler degli opcode virtuali, estrarne la semantica e recuperare la logica originale del programma. È rivolto a reverse engineer e analisti di malware che deofuscano binari protetti con VMProtect.
