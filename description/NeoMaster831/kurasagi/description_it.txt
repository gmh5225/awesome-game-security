Questo progetto è una proof-of-concept di bypass di PatchGuard mirata alle build recenti di Windows da 24H2 a 25H2.
Utilizza componenti in modalità kernel in C e C++ ed è progettato per essere caricato con un mapper manuale come kdmapper.
Il repository si concentra sulla ricerca PatchGuard a basso livello, sui casi limite soggetti a crash e sulla sperimentazione pratica intorno al comportamento di CRITICAL_STRUCTURE_CORRUPTION.
È destinato a ricercatori avanzati di sicurezza del kernel Windows che operano in ambienti di laboratorio controllati.
