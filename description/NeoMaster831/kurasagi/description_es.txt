Este proyecto es una prueba de concepto de bypass de PatchGuard dirigida a las builds recientes de Windows 24H2 a 25H2.
Utiliza componentes en modo kernel en C y C++ y está diseñado para cargarse con un mapeador manual como kdmapper.
El repositorio se centra en la investigación de bajo nivel de PatchGuard, casos límite propensos a fallos y la experimentación práctica en torno al comportamiento de CRITICAL_STRUCTURE_CORRUPTION.
Está destinado a investigadores avanzados de seguridad del kernel de Windows que trabajan en entornos de laboratorio controlados.
