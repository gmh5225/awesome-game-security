Этот проект — proof-of-concept обхода PatchGuard, нацеленный на недавние сборки Windows от 24H2 до 25H2.
Он использует компоненты режима ядра на C и C++ и рассчитан на загрузку с помощью manual mapper, например kdmapper.
Репозиторий посвящён низкоуровневому исследованию PatchGuard, склонным к сбоям краевым случаям и практическим экспериментам вокруг поведения CRITICAL_STRUCTURE_CORRUPTION.
Он предназначен для опытных исследователей безопасности ядра Windows, работающих в контролируемых лабораторных условиях.
