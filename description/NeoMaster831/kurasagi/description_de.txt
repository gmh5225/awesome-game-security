Dieses Projekt ist ein Proof-of-Concept für einen PatchGuard-Bypass, der auf aktuelle Windows-Builds von 24H2 bis 25H2 abzielt.
Es verwendet Kernel-Mode-Komponenten in C und C++ und ist für das Laden mit einem Manual Mapper wie kdmapper ausgelegt.
Das Repository konzentriert sich auf Low-Level-PatchGuard-Forschung, absturzanfällige Randfälle und praktische Experimente rund um das Verhalten von CRITICAL_STRUCTURE_CORRUPTION.
Es richtet sich an fortgeschrittene Windows-Kernel-Sicherheitsforscher, die in kontrollierten Laborumgebungen arbeiten.
