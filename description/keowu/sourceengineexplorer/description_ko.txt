이 프로젝트는 탐색기(explorer)에 초점을 둡니다.
CS Source Engine²용으로 치트 디버그용 Valve 코드 작성, 게임 엔진의 숨겨진 기능 탐색, 익스플로잇 개발, 어셈블리 분석기 또는 완전한 네트워크 에뮬레이터, 학습용 커스텀 스크립팅 인터페이스, 테스트(및 바이너리 분석)용 클라우드 기반 디컴파일러 등을 아우르는 기반입니다. 자신만의 도구를 만드는 출발점으로 쓰거나 원하는 대로 활용하세요. 결국 이것은 학습과 연구를 위해 존재합니다.
