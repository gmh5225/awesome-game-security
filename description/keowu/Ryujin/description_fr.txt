Ce projet se concentre sur le BIN2BIN x86 PE.
Ryūjin est un protecteur et obfuscateur de niveau recherche conçu pour explorer les techniques de transformation Bin2Bin.
Il est principalement utile aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant dans le domaine des moteurs anti-triche / d'obfuscation.
