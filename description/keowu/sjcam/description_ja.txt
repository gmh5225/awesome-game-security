本プロジェクトは、SJCAM アクションカメラとその組み込みファームウェアスタックに焦点を当てたリバースエンジニアリングおよびセキュリティ研究用ツールキットです。
カメラの TCP 制御プロトコルでログイン、撮影、ストリーミング、ファイル転送、設定を行う Python AVIOCTRL クライアントに加え、Android ARM ツールチェーンで構築した C++ Lelouch ネイティブコード（端末上の UI、カメラ、Wi-Fi、関連サービス向け）を含みます。
アーカイブにはカメラバイナリの IDA データベース、ファームウェアダンプ、AllWinner EGON/IMAGEWTY および Novatek BCL1 イメージ用の Python パーサに加え、脆弱性開示資料と概念実証（PoC）ペイロードも収録されています。
カメラプロトコルのリバースエンジニアリング、組み込みエクスプロイト、責任ある開示を研究する IoT・ファームウェアセキュリティ研究者を対象としています。
