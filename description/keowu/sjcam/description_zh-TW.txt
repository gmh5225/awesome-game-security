本專案是一套面向 SJCAM 運動相機及其嵌入式韌體堆疊的逆向工程與安全研究工具包。
其中包含一個 Python AVIOCTRL 用戶端，透過相機的 TCP 控制協定實現登入、拍攝、串流、檔案傳輸與設定；另有使用 Android ARM 工具鏈建置的 C++ Lelouch 原生程式碼，用於裝置端 UI、相機、Wi-Fi 及相關服務。
封存中還提供相機二進位檔的 IDA 資料庫、韌體傾印，以及針對 AllWinner EGON/IMAGEWTY 與 Novatek BCL1 映像的 Python 解析器，並附帶漏洞揭露資料與概念驗證（PoC）載荷。
專案面向從事相機協定逆向、嵌入式漏洞利用與負責任揭露的物聯網與韌體安全研究人員。
