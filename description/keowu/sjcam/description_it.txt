Questo progetto è un toolkit di reverse engineering e ricerca sulla sicurezza incentrato sulle action cam SJCAM e sui relativi stack di firmware embedded.
Include un client Python AVIOCTRL che parla il protocollo di controllo TCP della camera per login, acquisizione, streaming, trasferimento file e configurazione, oltre a codice nativo C++ Lelouch costruito con una toolchain Android ARM per UI, camera, Wi-Fi e servizi correlati sul dispositivo.
L'archivio fornisce anche database IDA dei binari della camera, dump del firmware e parser Python per immagini AllWinner EGON/IMAGEWTY e Novatek BCL1, insieme a materiali di disclosure delle vulnerabilità e payload di proof-of-concept.
È rivolto a ricercatori di sicurezza IoT e firmware che studiano il reverse engineering dei protocolli delle camera, lo exploitation embedded e la responsible disclosure.
