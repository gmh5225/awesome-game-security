Этот проект — набор инструментов для реверс-инжиниринга и исследований безопасности, ориентированный на экшн-камеры SJCAM и их стеки встроенного ПО (прошивок).
В него входят Python-клиент AVIOCTRL, работающий по TCP-протоколу управления камерой (вход, съёмка, потоковая передача, передача файлов и настройка), а также нативный код C++ Lelouch, собранный с Android ARM toolchain для UI, камеры, Wi-Fi и связанных сервисов на устройстве.
Архив также содержит базы IDA для бинарников камеры, дампы прошивок и Python-парсеры образов AllWinner EGON/IMAGEWTY и Novatek BCL1, а также материалы по раскрытию уязвимостей и proof-of-concept полезные нагрузки.
Проект предназначен для исследователей безопасности IoT и прошивок, изучающих реверс протоколов камер, эксплуатацию встраиваемых систем и ответственное раскрытие.
