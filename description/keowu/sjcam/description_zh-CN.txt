本项目是一套面向 SJCAM 运动相机及其嵌入式固件栈的逆向工程与安全研究工具包。
其中包含一个 Python AVIOCTRL 客户端，通过相机的 TCP 控制协议实现登录、拍摄、推流、文件传输与配置；另有使用 Android ARM 工具链构建的 C++ Lelouch 原生代码，用于设备端 UI、相机、Wi-Fi 及相关服务。
归档中还提供相机二进制文件的 IDA 数据库、固件转储，以及针对 AllWinner EGON/IMAGEWTY 与 Novatek BCL1 镜像的 Python 解析器，并附带漏洞披露材料与概念验证（PoC）载荷。
项目面向从事相机协议逆向、嵌入式漏洞利用与负责任披露的物联网与固件安全研究人员。
