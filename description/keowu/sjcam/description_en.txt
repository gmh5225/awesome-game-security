This project is a reverse-engineering and security research toolkit focused on SJCAM action cameras and their embedded firmware stacks.
It includes a Python AVIOCTRL client that speaks the camera's TCP control protocol for login, capture, streaming, file transfer, and configuration, plus C++ Lelouch native code built with an Android ARM toolchain for on-device UI, camera, Wi-Fi, and related services.
The archive also provides IDA databases of camera binaries, firmware dumps, and Python parsers for AllWinner EGON/IMAGEWTY and Novatek BCL1 images, along with vulnerability disclosure materials and proof-of-concept payloads.
It is aimed at IoT and firmware security researchers studying camera protocol reverse engineering, embedded exploitation, and responsible disclosure.
