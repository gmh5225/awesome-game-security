Este proyecto es un kit de herramientas de ingeniería inversa e investigación de seguridad centrado en las cámaras de acción SJCAM y sus stacks de firmware embebido.
Incluye un cliente Python AVIOCTRL que habla el protocolo de control TCP de la cámara para inicio de sesión, captura, streaming, transferencia de archivos y configuración, además de código nativo C++ Lelouch compilado con una toolchain Android ARM para la UI, la cámara, el Wi-Fi y servicios relacionados en el dispositivo.
El archivo también aporta bases de datos IDA de los binarios de la cámara, volcados de firmware y parsers Python para imágenes AllWinner EGON/IMAGEWTY y Novatek BCL1, junto con materiales de divulgación de vulnerabilidades y cargas de prueba de concepto.
Está orientado a investigadores de seguridad IoT y de firmware que estudian la ingeniería inversa de protocolos de cámara, la explotación embebida y la divulgación responsable.
