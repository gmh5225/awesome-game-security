Overload è un motore di gioco 3D open source che include sia sistemi runtime sia un editor integrato.
È realizzato principalmente in C++ moderno con supporto allo scripting Lua e offre strumenti per rendering, fisica, audio, profiling e materiali.
Il progetto pone l'accento su usabilità ed estensibilità attraverso documentazione, progetti di esempio e un'architettura guidata dalla community.
Per la ricerca sulla sicurezza dei giochi, fornisce una codebase di motore sostanziale per studiare superfici di attacco, interni degli strumenti e pattern di progettazione difensiva.
