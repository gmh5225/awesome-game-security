Overload es un motor de juegos 3D de código abierto que incluye tanto sistemas de tiempo de ejecución como un editor integrado.
Está construido principalmente en C++ moderno con soporte de scripting en Lua y ofrece herramientas de renderizado, física, audio, profiling y materiales.
El proyecto enfatiza la usabilidad y la extensibilidad mediante documentación, proyectos de ejemplo y una arquitectura impulsada por la comunidad.
Para la investigación en seguridad de juegos, proporciona una base de código de motor sustancial para estudiar superficies de ataque, el funcionamiento interno de las herramientas y patrones de diseño defensivo.
