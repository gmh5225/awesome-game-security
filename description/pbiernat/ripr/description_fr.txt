Ce projet est un plugin IDA Pro qui extrait le code binaire de fonctions désassemblées et le regroupe en scripts Python autonomes destinés à l'émulation. Il effectue une analyse du flux de contrôle, un scan des dépendances et une génération de code pour extraire des fragments binaires réutilisables, avec une intégration r2pipe pour le support de Radare2. Il s'adresse principalement aux ingénieurs en rétro-ingénierie qui doivent extraire et émuler des fonctions individuelles de binaires complexes en dehors d'IDA Pro.
