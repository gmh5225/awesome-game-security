Este proyecto es un complemento de IDA Pro que extrae código binario de funciones desensambladas y lo empaqueta en scripts de Python independientes para emulación. Realiza análisis de flujo de control, escaneo de dependencias y generación de código para extraer fragmentos binarios reutilizables, con integración r2pipe para soporte de Radare2. Resulta especialmente útil para ingenieros de ingeniería inversa que necesitan extraer y emular funciones individuales de binarios complejos fuera de IDA Pro.
