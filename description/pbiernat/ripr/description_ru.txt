Этот проект — плагин для IDA Pro, который извлекает двоичный код из дизассемблированных функций и упаковывает его в автономные Python-скрипты для эмуляции. Он выполняет анализ потока управления, сканирование зависимостей и генерацию кода для извлечения повторно используемых двоичных фрагментов, а также поддерживает Radare2 через интеграцию с r2pipe. В первую очередь полезен для инженеров по реверс-инжинирингу, которым нужно извлекать и эмулировать отдельные функции из сложных бинарных файлов вне IDA Pro.
