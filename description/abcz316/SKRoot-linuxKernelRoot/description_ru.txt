SKRoot — набор инструментов для root-патчинга ядра Linux с ориентацией на Android, предназначенный для предоставления глубоко скрытых root-возможностей.
Он сочетает компоненты патчинга на C и C++ с прикладными средствами на Java и JNI для управления правами и процессов развёртывания.
Проект заявляет о таких возможностях, как выполнение root-команд, установка и внедрение su, а также совместимость со многими версиями ядра без пересборки ядра из исходников.
В основном применяется в исследованиях безопасности мобильных игр и приложений для изучения детектирования root, обхода защиты и управления привилегиями на уровне ядра.
