SKRoot es un kit de herramientas de parcheo root del kernel Linux orientado a Android, diseñado para proporcionar capacidades root profundamente ocultas.
Combina componentes de parcheo en C y C++ con herramientas de aplicación basadas en Java y JNI para la gestión de permisos y los flujos de despliegue.
El proyecto anuncia funciones como la ejecución de comandos root, la instalación e inyección de su, y la compatibilidad con muchas versiones del kernel sin reconstruirlo desde el código fuente.
Se usa principalmente en la investigación de seguridad de juegos y aplicaciones móviles para estudiar la detección de root, el comportamiento de bypass y el control de privilegios a nivel de kernel.
