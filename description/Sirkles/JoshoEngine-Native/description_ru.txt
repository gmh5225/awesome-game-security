Этот проект — нативный 2D-игровой движок на C++ с рендерингом через OpenGL. Он предоставляет относительно высокоуровневый кроссплатформенный API с геометрическими примитивами (векторы, прямоугольники, окружности) и намечает пункты дорожной карты: шейдеры, физика и поддержка мобильных платформ. Репозиторий содержит исходники движка и артефакты сборки, ориентированные на Visual Studio, для разработки под Windows. Основное назначение — изучение архитектуры движка и использование в качестве основы для экспериментов по созданию собственных 2D-игр.
