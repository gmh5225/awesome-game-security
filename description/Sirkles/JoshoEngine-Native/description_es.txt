Este proyecto es un motor de juegos 2D nativo implementado en C++ con renderizado OpenGL. Ofrece una API multiplataforma de nivel relativamente alto con primitivas geométricas como vectores, rectángulos y círculos, y esboza elementos de la hoja de ruta como shaders, física y soporte móvil. El repositorio incluye los archivos fuente del motor y artefactos de compilación orientados a Visual Studio para flujos de trabajo en Windows. Su caso de uso principal es aprender la arquitectura de un motor y servir como base para experimentos personalizados de desarrollo de juegos 2D.
