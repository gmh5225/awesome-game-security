Questo progetto è un motore di gioco 2D nativo implementato in C++ con rendering OpenGL. Fornisce un'API multipiattaforma di livello relativamente alto con primitive geometriche come vettori, rettangoli e cerchi, e delinea elementi della roadmap come shader, fisica e supporto mobile. Il repository include i file sorgente del motore e artefatti di build orientati a Visual Studio per i flussi di lavoro di sviluppo su Windows. Il caso d'uso principale è lo studio dell'architettura di un motore e l'uso come base per esperimenti di sviluppo di giochi 2D personalizzati.
