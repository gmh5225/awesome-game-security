Ce projet est un moteur de jeu 2D natif implémenté en C++ avec un rendu OpenGL. Il fournit une API multiplateforme de relativement haut niveau, avec des primitives géométriques telles que des vecteurs, des rectangles et des cercles, et présente une feuille de route incluant les shaders, la physique et le support mobile. Le dépôt contient les fichiers sources du moteur ainsi que des artefacts de build orientés Visual Studio pour les flux de travail Windows. Son principal cas d'usage est l'apprentissage de l'architecture d'un moteur et servir de base à des expérimentations de développement de jeux 2D personnalisés.
