Этот проект посвящён самостоятельно размещаемому обратному геокодированию на основе OpenStreetMap; стек Traccar; сервер на Rust + индексатор PBF на C++, Docker.
Он в основном написан на Rust и C++ и сосредоточен на анализе памяти.
Он особенно полезен разработчикам бэкенда, мультиплеера и онлайн-игр, работающим в области игровых сетей / геолокации / геокодирования.
