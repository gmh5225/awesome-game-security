ZeroImport — это библиотека C++ для драйверов ядра Windows, которая разрешает импорты во время выполнения вместо статических импортов.
Она хеширует имена экспортов и обходит экспорты ntoskrnl, чтобы находить функции и переменные, такие как MmIsAddressValid и PsInitialSystemProcess.
Проектирование сводит к минимуму артефакты таблицы импорта и не встраивает строки API в открытом виде в скомпилированный драйвер; для производительности доступно опциональное кэширование.
В основном применяется в низкоуровневой разработке драйверов и исследованиях безопасности, где важны скрытность и устойчивость к статическому анализу.
