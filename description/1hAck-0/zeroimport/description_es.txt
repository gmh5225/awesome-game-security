ZeroImport es una biblioteca C++ para controladores del kernel de Windows que resuelve las importaciones en tiempo de ejecución en lugar de depender de importaciones estáticas.
Hashea nombres de exportaciones y recorre las exportaciones de ntoskrnl para localizar funciones y variables como MmIsAddressValid y PsInitialSystemProcess.
El diseño minimiza los artefactos de la tabla de importaciones y evita incrustar cadenas de API en texto claro en el controlador compilado, con almacenamiento en caché opcional para el rendimiento.
Se usa principalmente en el desarrollo de controladores de bajo nivel y en investigación de seguridad, donde importan el sigilo y la resistencia al análisis estático.
