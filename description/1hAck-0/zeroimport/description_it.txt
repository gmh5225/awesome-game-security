ZeroImport è una libreria C++ per driver del kernel Windows che risolve gli import a runtime invece di affidarsi a import statici.
Esegue l'hash dei nomi degli export e attraversa gli export di ntoskrnl per individuare funzioni e variabili come MmIsAddressValid e PsInitialSystemProcess.
Il design riduce gli artefatti della tabella degli import ed evita di incorporare stringhe API in chiaro nel driver compilato, con caching opzionale per le prestazioni.
È usata soprattutto nello sviluppo di driver a basso livello e nella ricerca di sicurezza, dove contano stealth e resistenza all'analisi statica.
