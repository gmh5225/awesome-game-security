Este proyecto es una base de código y una guía educativa de cheats para Unreal Engine 4, orientada a aprender flujos habituales de game hacking.
Su código fuente en C++ demuestra funciones como overlays ESP, lógica de aimbot, comportamiento de trigger y varias técnicas de manipulación del gameplay.
El repositorio también documenta técnicas centrales como el function hooking y el pattern scanning para explicar cómo se construyen las herramientas centradas en UE4.
Su uso principal es la formación práctica y la experimentación para quienes aprenden seguridad de juegos e ingeniería inversa en entornos controlados.
