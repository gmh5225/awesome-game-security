This project is a large curated repository of Cheat Engine table files (.ct) for many PC games. The archive is organized mainly under a tables directory containing thousands of game-specific tables named for titles, versions, and authors. Entries cover common Cheat Engine techniques such as pointer chains, AOB (array-of-byte) scans, trainers, Lua-style scripts, and tutorial or test tables. Content spans single-player and multiplayer titles and is aimed at reverse engineers, Cheat Engine users, and game-security researchers studying memory editing and in-game value manipulation.
