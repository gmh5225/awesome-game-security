Questo progetto è un ampio repository curato di file tabella di Cheat Engine (.ct) per molti giochi PC. L’archivio è organizzato principalmente sotto una directory tables che contiene migliaia di tabelle specifiche per gioco, nominate per titoli, versioni e autori. Le voci coprono tecniche comuni di Cheat Engine come catene di puntatori, scansioni AOB (array di byte), trainer, script in stile Lua e tabelle tutorial o di test. I contenuti spaziano tra titoli single-player e multiplayer e si rivolgono a reverse engineer, utenti di Cheat Engine e ricercatori di sicurezza nei giochi che studiano l’editing della memoria e la manipolazione dei valori in gioco.
