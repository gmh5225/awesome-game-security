Этот проект — декомпилятор на основе символьного выполнения, написанный на Rust для восстановления высокоуровневой структуры программ из бинарных файлов.
Его архитектура делает акцент на отслеживании ограничений, переписывании выражений и структурированной реконструкции потока управления за пределами классических подходов, основанных только на SSA.
Кодовая база включает ресурсы спецификаций процессоров и современный фронтенд-стек для интерактивного анализа.
Проект предназначен для исследователей и разработчиков в области реверс-инжиниринга, изучающих продвинутые методы декомпиляции.
