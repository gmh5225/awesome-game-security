Este proyecto es un descompilador de ejecución simbólica escrito en Rust para recuperar la estructura de alto nivel de programas a partir de binarios.
Su diseño enfatiza el seguimiento de restricciones, la reescritura de expresiones y la reconstrucción estructurada del flujo de control más allá de los enfoques clásicos basados solo en SSA.
El código incluye activos de especificación de procesador y una pila frontend moderna para el análisis interactivo.
Está dirigido a investigadores y desarrolladores de ingeniería inversa que exploran técnicas avanzadas de descompilación.
