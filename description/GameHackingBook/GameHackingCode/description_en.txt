This repository is the companion example code for the Game Hacking book, delivered as a Visual Studio Win32 solution of chapter-by-chapter demos. The projects are primarily written in C++ and cover foundational topics such as process memory layout, pointers, basic debugging, mutex and file discovery, memory scanning, and remote memory access. Later chapters demonstrate code injection (including DLL injection), control-flow manipulation with NOP patches, IAT hooks, virtual-function and call hooks, Adobe AIR hooking, and Direct3D overlay or Present hooking. Supporting Lua scripts illustrate assembly and string searches plus linked-list and map verification for memory forensics workflows. It is aimed at students and researchers learning game hacking, reverse engineering, and the defensive implications of common cheat techniques on Windows.
