Ce dépôt contient le code d'exemple accompagnant le livre Game Hacking, sous forme d'une solution Visual Studio Win32 avec des démos chapitre par chapitre. Les projets sont principalement écrits en C++ et couvrent des sujets fondamentaux tels que la disposition mémoire des processus, les pointeurs, le débogage de base, la découverte de mutex et de fichiers, le scan mémoire et l'accès mémoire distant. Les chapitres suivants démontrent l'injection de code (y compris l'injection de DLL), la manipulation du flux de contrôle par patches NOP, les hooks IAT, les hooks de fonctions virtuelles et d'appels, le hooking Adobe AIR, ainsi que les overlays Direct3D ou le hooking de Present. Des scripts Lua associés illustrent les recherches d'assembly et de chaînes, ainsi que la vérification de listes chaînées et de maps pour les workflows de forensic mémoire. Il s'adresse aux étudiants et chercheurs qui étudient le game hacking, la rétro-ingénierie et les implications défensives des techniques de triche courantes sous Windows.
