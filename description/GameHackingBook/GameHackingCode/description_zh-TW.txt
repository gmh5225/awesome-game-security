本儲存庫是《Game Hacking》一書的配套範例程式碼，以 Visual Studio Win32 方案形式依章節提供示範專案。專案主要以 C++ 撰寫，涵蓋程序記憶體配置、指標、基礎除錯、互斥鎖與檔案探索、記憶體掃描以及遠端記憶體存取等入門主題。後續章節示範程式碼注入（含 DLL 注入）、以 NOP 修補操控控制流程、IAT Hook、虛擬函式與呼叫 Hook、Adobe AIR Hook，以及 Direct3D 疊加層或 Present Hook。配套 Lua 腳本展示組譯與字串搜尋，以及用於記憶體鑑識流程的鏈結串列與 map 驗證。面向學習遊戲破解、逆向工程，以及 Windows 上常見作弊技術防禦意涵的學生與研究人員。
