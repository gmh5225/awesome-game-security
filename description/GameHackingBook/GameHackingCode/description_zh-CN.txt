本仓库是《Game Hacking》一书的配套示例代码，以 Visual Studio Win32 解决方案形式按章节提供演示项目。项目主要使用 C++ 编写，涵盖进程内存布局、指针、基础调试、互斥量与文件发现、内存扫描以及远程内存访问等入门主题。后续章节演示代码注入（含 DLL 注入）、通过 NOP 补丁操控控制流、IAT Hook、虚函数与调用 Hook、Adobe AIR Hook，以及 Direct3D 叠加层或 Present Hook。配套 Lua 脚本展示汇编与字符串搜索，以及用于内存取证流程的链表与 map 校验。面向学习游戏破解、逆向工程，以及 Windows 上常见作弊技术防御含义的学生与研究人员。
