本リポジトリは『Game Hacking』書籍の付属サンプルコードで、章ごとのデモを含む Visual Studio Win32 ソリューションとして提供されます。プロジェクトは主に C++ で書かれており、プロセスメモリレイアウト、ポインタ、基本的なデバッグ、ミューテックスとファイル探索、メモリスキャン、リモートメモリアクセスといった基礎トピックを扱います。後半の章ではコードインジェクション（DLL インジェクションを含む）、NOP パッチによる制御フロー操作、IAT フック、仮想関数および呼び出しフック、Adobe AIR のフッキング、Direct3D オーバーレイや Present フックを示します。付属の Lua スクリプトは、アセンブリと文字列検索、およびメモリフォレンジック向けのリンクリストと map の検証を説明します。ゲームハッキング、リバースエンジニアリング、Windows 上の一般的なチート手法の防御的含意を学ぶ学生や研究者を対象としています。
