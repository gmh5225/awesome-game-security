Этот репозиторий содержит сопроводительный пример кода к книге Game Hacking в виде решения Visual Studio Win32 с демо по главам. Проекты в основном написаны на C++ и охватывают базовые темы: раскладку памяти процесса, указатели, основы отладки, обнаружение мьютексов и файлов, сканирование памяти и удалённый доступ к памяти. Поздние главы демонстрируют внедрение кода (включая DLL-инъекцию), манипуляцию потоком управления с помощью NOP-патчей, IAT-хуки, хуки виртуальных функций и вызовов, хукинг Adobe AIR, а также Direct3D-оверлеи или хукинг Present. Сопроводительные Lua-скрипты иллюстрируют поиск по ассемблеру и строкам, а также проверку связных списков и map для рабочих процессов memory forensics. Материал предназначен для студентов и исследователей, изучающих game hacking, реверс-инжиниринг и защитные аспекты распространённых техник читов в Windows.
