Direct3DHook es una biblioteca C#/.NET que se inyecta en un proceso de Windows objetivo y engancha su ruta de renderizado Direct3D para capturar capturas de pantalla y dibujar superposiciones. Usa EasyHook para la inyección remota de DLL e IPC, y enlaces SharpDX para Direct3D 9, 10, 10.1 y 11, con detección automática opcional de la API que el proceso ha cargado. Las aplicaciones anfitrionas se comunican con el ensamblado Capture inyectado mediante una CaptureInterface de remoting para solicitar fotogramas, mostrar superposiciones de texto o imagen y recibir resultados de captura. El ejemplo TestScreenshot muestra una inyección y captura de extremo a extremo contra una ventana Direct3D activa. El proyecto está orientado a investigadores de seguridad en juegos, ingenieros inversos y autores de herramientas que necesitan captura gráfica en proceso o hooks de superposición para aplicaciones Direct3D.
