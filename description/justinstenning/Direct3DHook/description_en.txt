Direct3DHook is a C#/.NET library that injects into a target Windows process and hooks its Direct3D rendering path so you can capture screenshots and draw overlays. It uses EasyHook for remote DLL injection and IPC, and SharpDX bindings for Direct3D 9, 10, 10.1, and 11, with optional auto-detection of which API the process has loaded. Host applications talk to the injected Capture assembly through a remoting CaptureInterface to request frames, display text or image overlays, and receive capture results. A TestScreenshot sample shows end-to-end injection and capture against a live Direct3D window. The project is aimed at game security researchers, reverse engineers, and tool authors who need in-process graphics capture or overlay hooks for Direct3D applications.
