Direct3DHook は、対象の Windows プロセスに注入し、Direct3D の描画パスをフックしてスクリーンショット取得やオーバーレイ描画を行える C#/.NET ライブラリです。リモート DLL 注入と IPC には EasyHook を用い、Direct3D 9 / 10 / 10.1 / 11 向けには SharpDX バインディングを利用し、プロセスが読み込んだ API の自動検出もオプションで行えます。ホストアプリはリモートの CaptureInterface 経由で注入された Capture アセンブリと通信し、フレーム要求、テキスト／画像オーバーレイの表示、キャプチャ結果の受信を行います。TestScreenshot サンプルは、動作中の Direct3D ウィンドウに対する一連の注入とキャプチャを示します。本プロジェクトは、Direct3D アプリケーション向けのプロセス内グラフィックスキャプチャやオーバーレイフックを必要とするゲームセキュリティ研究者、リバースエンジニア、ツール作者を対象としています。
