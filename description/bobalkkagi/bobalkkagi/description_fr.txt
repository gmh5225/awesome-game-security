Bobalkkagi est un outil Python pour dépacker et dérouler les exécutables Windows protégés par Themida 3.1.3 (Tiger red64), avec un support prévu pour la dévirtualisation. Il émule les binaires protégés via le moteur Unicorn et accroche les API Windows sur un jeu de DLL win10_v1903 fourni, afin que l’anti-debug et la logique de chargeur du packer puissent s’exécuter assez loin pour récupérer le code d’origine. Les utilisateurs peuvent choisir les modes d’émulation fast, hook_block ou hook_code, localiser éventuellement le point d’entrée d’origine (OEP), et activer un flux de travail de type débogueur en mode hook_code lors de l’extension des hooks pour d’autres protecteurs ou versions de Themida. Construit avec Capstone/distorm3 et empaqueté via Poetry/PyInstaller, il s’adresse aux reverse engineers et aux chercheurs en sécurité des jeux qui analysent des binaires packés par Themida.
