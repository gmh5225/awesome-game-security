Bobalkkagi es una herramienta en Python para desempaquetar y desenvolver ejecutables de Windows protegidos con Themida 3.1.3 (Tiger red64), con soporte planificado para desvirtualización. Emula binarios protegidos con el motor Unicorn y engancha APIs de Windows frente a un conjunto de DLL win10_v1903 proporcionado, para que la lógica antidebug y de cargador del packer avance lo suficiente y se pueda recuperar el código original. Los usuarios pueden elegir los modos de emulación fast, hook_block o hook_code, localizar opcionalmente el punto de entrada original (OEP) y activar un flujo de trabajo tipo depurador en el modo hook_code al ampliar hooks para otros protectores o versiones de Themida. Construido con Capstone/distorm3 y empaquetado con Poetry/PyInstaller, está orientado a ingenieros inversos e investigadores de seguridad en juegos que analizan binarios empaquetados con Themida.
