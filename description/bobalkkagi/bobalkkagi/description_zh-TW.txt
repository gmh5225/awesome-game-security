Bobalkkagi 是一款用於脫殼與解包受 Themida 3.1.3 保護的 Windows 可執行檔（Tiger red64）的 Python 工具，並計劃支援去虛擬化。它使用 Unicorn 引擎模擬受保護二進位，並對照提供的 win10_v1903 DLL 集掛鉤 Windows API，使加殼器的反偵錯與載入邏輯能執行到足以還原原始程式碼。使用者可選擇 fast、hook_block 或 hook_code 模擬模式，可選定位原始進入點（OEP），並在 hook_code 模式下啟用類似除錯器的工作流程，以便為其他保護器或 Themida 版本擴充掛鉤。該工具以 Capstone/distorm3 建置，透過 Poetry/PyInstaller 打包，面向分析 Themida 加殼二進位的逆向工程師與遊戲安全研究人員。
