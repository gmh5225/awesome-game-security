Bobalkkagi 是一款用于脱壳与解包受 Themida 3.1.3 保护的 Windows 可执行文件（Tiger red64）的 Python 工具，并计划支持去虚拟化。它使用 Unicorn 引擎模拟受保护二进制，并对照提供的 win10_v1903 DLL 集挂钩 Windows API，使加壳器的反调试与加载逻辑能运行到足以恢复原始代码。用户可选择 fast、hook_block 或 hook_code 模拟模式，可选定位原始入口点（OEP），并在 hook_code 模式下启用类似调试器的工作流，以便为其他保护器或 Themida 版本扩展挂钩。该工具基于 Capstone/distorm3 构建，通过 Poetry/PyInstaller 打包，面向分析 Themida 加壳二进制的逆向工程师与游戏安全研究人员。
