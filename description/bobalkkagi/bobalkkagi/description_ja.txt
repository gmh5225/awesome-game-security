Bobalkkagi は、Themida 3.1.3 で保護された Windows 実行ファイル（Tiger red64）のアンパック／アンラップ用 Python ツールで、デバーチャライゼーション対応も予定されています。Unicorn エンジンで保護バイナリをエミュレートし、提供される win10_v1903 DLL セットに対して Windows API をフックすることで、パッカーのアンチデバッグとローダー処理を十分に進め、元のコードを復元します。ユーザーは fast、hook_block、hook_code の各エミュレーションモードを選べ、必要に応じて OEP（オリジナルエントリポイント）を特定でき、hook_code モードでは他の保護や Themida 版向けにフックを拡張する際のデバッガ風ワークフローも利用できます。Capstone/distorm3 で構築され Poetry/PyInstaller でパッケージ化されており、Themida パック済みバイナリを解析するリバースエンジニアやゲームセキュリティ研究者向けです。
