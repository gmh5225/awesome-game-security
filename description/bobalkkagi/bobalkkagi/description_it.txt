Bobalkkagi è uno strumento Python per unpacking e unwrapping di eseguibili Windows protetti con Themida 3.1.3 (Tiger red64), con supporto pianificato per la devirtualizzazione. Emula i binari protetti con il motore Unicorn e intercetta le API Windows rispetto a un set di DLL win10_v1903 fornito, così che l’anti-debug e la logica del loader del packer possano avanzare abbastanza da recuperare il codice originale. Gli utenti possono scegliere le modalità di emulazione fast, hook_block o hook_code, individuare opzionalmente l’original entry point (OEP) e abilitare un flusso di lavoro in stile debugger in modalità hook_code quando si estendono gli hook per altri protector o versioni di Themida. Realizzato con Capstone/distorm3 e confezionato tramite Poetry/PyInstaller, è rivolto a reverse engineer e ricercatori di game security che analizzano binari packati con Themida.
