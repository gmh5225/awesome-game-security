Bobalkkagi는 Themida 3.1.3으로 보호된 Windows 실행 파일(Tiger red64)을 언패킹·언래핑하는 Python 도구이며, 디버추얼라이제이션 지원도 계획되어 있습니다. Unicorn 엔진으로 보호된 바이너리를 에뮬레이션하고, 제공된 win10_v1903 DLL 세트에 맞춰 Windows API를 후킹해 패커의 안티디버그와 로더 로직이 충분히 진행되도록 하여 원본 코드를 복구합니다. 사용자는 fast, hook_block, hook_code 에뮬레이션 모드를 선택할 수 있고, 선택적으로 OEP(원본 진입점)를 찾은 뒤, hook_code 모드에서 다른 보호기나 Themida 버전용 후크를 확장할 때 디버거 스타일 워크플로를 사용할 수 있습니다. Capstone/distorm3로 구축되고 Poetry/PyInstaller로 패키징되며, Themida로 패킹된 바이너리를 분석하는 리버스 엔지니어와 게임 보안 연구자를 대상으로 합니다.
