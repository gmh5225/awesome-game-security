Bobalkkagi — инструмент на Python для распаковки и развёртки Windows-исполняемых файлов, защищённых Themida 3.1.3 (Tiger red64), с планируемой поддержкой девиртуализации. Он эмулирует защищённые бинарники через движок Unicorn и перехватывает Windows API на предоставленном наборе DLL win10_v1903, чтобы антиотладочная и загрузочная логика пакера успела отработать достаточно далеко для восстановления исходного кода. Пользователи могут выбрать режимы эмуляции fast, hook_block или hook_code, при необходимости найти оригинальную точку входа (OEP) и включить отладочный рабочий процесс в режиме hook_code при расширении хуков для других протекторов или версий Themida. Собран с Capstone/distorm3 и упакован через Poetry/PyInstaller; ориентирован на реверс-инженеров и исследователей игровой безопасности, анализирующих бинарники, упакованные Themida.
