Bobalkkagi is a Python tool for unpacking and unwrapping Themida 3.1.3–protected Windows executables (Tiger red64), with planned support for devirtualization. It emulates protected binaries using the Unicorn engine and hooks Windows APIs against a provided win10_v1903 DLL set so the packer’s anti-debug and loader logic can run far enough to recover the original code. Users can choose fast, hook_block, or hook_code emulation modes, optionally locate the original entry point (OEP), and enable a debugger-style workflow in hook_code mode when extending hooks for other protectors or Themida versions. Built with Capstone/distorm3 and packaged via Poetry/PyInstaller, it is aimed at reverse engineers and game-security researchers analyzing Themida-packed binaries.
