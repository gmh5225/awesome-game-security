Ce projet est une DLL Windows et un outil d'injection permettant d'extraire les fichiers d'applications UWP à l'exécution.
Il fonctionne en s'injectant dans un processus cible et en extrayant les données du package autrement protégées par le modèle de système de fichiers UWP.
L'implémentation est en C++, gère la ligne de commande et est conçue pour être compilée avec le Windows 10 SDK.
Il est utile pour la rétro-ingénierie et l'analyse de sécurité des logiciels UWP, y compris les builds de jeux protégés distribués via l'écosystème Microsoft.
