Este proyecto es una DLL de Windows y una herramienta de inyección para volcar archivos de aplicaciones UWP en tiempo de ejecución.
Funciona inyectándose en un proceso objetivo y extrayendo datos del paquete que de otro modo están protegidos por el modelo de sistema de archivos UWP.
La implementación está basada en C++, incluye manejo de línea de comandos y está diseñada para compilarse con el Windows 10 SDK.
Es útil para la ingeniería inversa y el análisis de seguridad de software UWP, incluidas las builds de juegos protegidas distribuidas a través del ecosistema de Microsoft.
