Этот проект — Windows DLL и инструмент внедрения для дампа файлов UWP-приложений во время выполнения.
Он работает путём внедрения в целевой процесс и извлечения данных пакета, которые иначе защищены моделью файловой системы UWP.
Реализация на C++, включает обработку командной строки и рассчитана на сборку с Windows 10 SDK.
Полезен для реверс-инжиниринга и анализа безопасности UWP-ПО, включая защищённые игровые сборки, распространяемые через экосистему Microsoft.
