Questo progetto è una DLL Windows e uno strumento di injection per il dumping dei file delle applicazioni UWP a runtime.
Funziona iniettandosi in un processo di destinazione ed estraendo i dati del pacchetto altrimenti protetti dal modello di file system UWP.
L'implementazione è basata su C++, include la gestione della riga di comando ed è progettata per essere compilata con il Windows 10 SDK.
È utile per il reverse engineering e l'analisi di sicurezza del software UWP, inclusi i build di giochi protetti distribuiti tramite l'ecosistema Microsoft.
