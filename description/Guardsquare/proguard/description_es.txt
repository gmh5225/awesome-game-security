ProGuard es un reductor, optimizador, ofuscador y preverificador de bytecode Java para aplicaciones y bibliotecas. Elimina clases, campos, métodos y atributos no utilizados para reducir el tamaño del paquete y mejorar la eficiencia en tiempo de ejecución. La herramienta también aplica optimizaciones de bytecode y renombra símbolos para dificultar la ingeniería inversa, preservando el comportamiento mediante reglas configurables. Está implementado principalmente en Java y se integra habitualmente mediante flujos de trabajo de línea de comandos y builds de Gradle para la protección de software y el endurecimiento del despliegue.
