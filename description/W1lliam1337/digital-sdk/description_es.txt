Este proyecto es una base interna de cheat en C++ para Counter-Strike: Global Offensive. Incluye funciones habituales como ESP, bunnyhop, predicción del motor, autowall y chams, junto con puntos de hook como CreateMove y Direct3D reset. El código usa ImGui y MinHook y se organiza en torno a interfaces, netvars, renderizado y módulos de utilidades para facilitar la extensión. Está pensado principalmente para practicar el desarrollo de cheats y la investigación de ingeniería inversa en títulos del motor Source.
