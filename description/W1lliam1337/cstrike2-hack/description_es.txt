Este proyecto es una base de cheat interno para Counter-Strike 2 escrita en Rust, diseñada como un framework modular para construir funciones dentro del juego.
Organiza interfaces, hooks, ajustes y renderizado en crates y módulos separados, con manejo de interfaces asistido por macros y pattern scanning para offsets dinámicos.
El código integra DirectX 11 y egui para un menú overlay y utiliza la interceptación de funciones basada en MinHook.
Está orientado principalmente a la investigación en game hacking y a desarrolladores que buscan un punto de partida para herramientas internas de CS2.
