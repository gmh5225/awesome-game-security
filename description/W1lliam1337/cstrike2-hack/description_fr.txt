Ce projet est une base de cheat interne pour Counter-Strike 2 écrite en Rust, conçue comme un framework modulaire pour développer des fonctionnalités en jeu.
Il organise les interfaces, hooks, paramètres et le rendu dans des crates et modules distincts, avec une gestion d'interfaces assistée par macros et un pattern scanning pour les offsets dynamiques.
Le code intègre DirectX 11 et egui pour un menu overlay et utilise l'interception de fonctions basée sur MinHook.
Il s'adresse principalement à la recherche en game hacking et aux développeurs cherchant un point de départ pour des outils internes CS2.
