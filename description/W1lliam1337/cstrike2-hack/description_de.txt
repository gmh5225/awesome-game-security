Dieses Projekt ist eine auf Rust basierende interne Cheat-Basis für Counter-Strike 2 und als modulares Framework zum Aufbau von Ingame-Funktionen konzipiert.
Es organisiert Interfaces, Hooks, Einstellungen und Rendering in separaten Crates und Modulen, mit makrogestützter Interface-Verarbeitung und Pattern-Scanning für dynamische Offsets.
Der Code integriert DirectX 11 und egui für ein Overlay-Menü und verwendet MinHook-basiertes Abfangen von Funktionen.
Es richtet sich vor allem an Game-Hacking-Forschung und Entwickler, die einen Einstiegspunkt für interne CS2-Tools suchen.
