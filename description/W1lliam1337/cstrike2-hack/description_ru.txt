Этот проект — внутренняя база читов для Counter-Strike 2 на Rust, спроектированная как модульный фреймворк для создания внутриигровых функций.
Он разделяет интерфейсы, хуки, настройки и рендеринг по отдельным crate и модулям, с макросной обработкой интерфейсов и сканированием сигнатур для динамических смещений.
Код интегрирует DirectX 11 и egui для оверлей-меню и использует перехват функций на базе MinHook.
Он в первую очередь предназначен для исследований в области game hacking и разработчиков, которым нужна отправная точка для внутренних инструментов CS2.
