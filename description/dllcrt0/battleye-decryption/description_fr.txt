Un outil qui déchiffre les paquets de communication chiffrés multicouches de BattlEye échangés entre le composant usermode BEService et le pilote noyau BEDaisy via des named pipes. Il implémente les algorithmes de déchiffrement basés sur XOR, notamment le déchiffrement générique des paquets, le chiffrement/déchiffrement des informations matérielles, ainsi que les routines de déchiffrement de second stade dérivées d'une clé. Il est principalement utile aux chercheurs anti-triche qui reverse-engineerent le protocole de communication client-pilote de BattlEye et ses schémas de chiffrement des paquets.
