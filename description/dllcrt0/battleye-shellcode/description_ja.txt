リバースエンジニアリングおよび逆コンパイルされた BattlEye shellcode モジュールのコレクションで、AutoHotKey 検出、Present hook スキャン、およびアンチチートのランタイム整合性チェックで使用されるスタックウォーキング処理を含みます。
逆コンパイルされたコードから、BattlEye が自動化ツールを検出し、スワップチェーン上のグラフィックス API フックを走査し、疑わしい戻りアドレスについてコールスタックを検証する手法が明らかになります。
BattlEye の shellcode ベースの検出メカニズムとランタイムスキャン戦略を研究するアンチチート研究者にとって、特に有用です。
