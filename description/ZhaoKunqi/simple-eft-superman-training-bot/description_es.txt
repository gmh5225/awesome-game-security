Este proyecto es un bot de automatización basado en Arduino para rutinas de entrenamiento de Escape from Tarkov. Usa placas compatibles con HID para emular teclado y ratón y ejecutar automáticamente ciclos de movimiento repetitivos en el juego. El repositorio incluye varios sketches .ino y un ayudante Python de conversión de coordenadas para adaptar posiciones de botones y actualizaciones del flujo. Está orientado principalmente a experimentos de automatización asistida por hardware y scripts de grind de habilidades.
