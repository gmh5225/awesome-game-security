Этот проект — экспериментальный рендерер глобального освещения в реальном времени, рассчитанный на приближение к качеству path tracing в динамических сценах. Код движка написан на Rust, графический бэкенд использует Vulkan, а большой стек HLSL-шейдеров обслуживает гибридные проходы растеризации, compute и трассировки лучей. Ключевые возможности: динамический GI без предзапечённых проб, временная реконструкция, тени и отражения с трассировкой лучей, а также эталонный режим path tracing для проверки. Основное применение — исследования и прототипирование продвинутого рендеринга для разработчиков, изучающих современные техники освещения в реальном времени.
