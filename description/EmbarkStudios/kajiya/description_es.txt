Este proyecto es un renderizador experimental de iluminación global en tiempo real, diseñado para aproximarse a la calidad del path tracing en escenas dinámicas. Usa Rust para el código del motor, Vulkan como backend gráfico y un amplio conjunto de shaders HLSL para pases híbridos de rasterización, compute y ray tracing. Sus características clave incluyen GI dinámica sin sondas prehorneadas, reconstrucción temporal, sombras y reflejos por ray tracing, y un modo de path tracing de referencia para validación. Su caso de uso principal es la investigación y el prototipado de renderizado avanzado para desarrolladores que estudian técnicas modernas de iluminación en tiempo real.
