Ce projet est un moteur de rendu expérimental d’illumination globale en temps réel, conçu pour approcher la qualité du path tracing dans des scènes dynamiques. Il utilise Rust pour le code moteur, Vulkan pour le backend graphique, et une large pile de shaders HLSL pour des passes hybrides de rasterisation, de compute et de ray tracing. Les fonctionnalités clés incluent une GI dynamique sans sondes précalculées, la reconstruction temporelle, les ombres et réflexions en ray tracing, ainsi qu’un mode path tracing de référence pour la validation. Son usage principal est la recherche et le prototypage de rendu avancé pour les développeurs qui étudient les techniques d’éclairage temps réel modernes.
