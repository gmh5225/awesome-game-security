Questo progetto è un renderer sperimentale di illuminazione globale in tempo reale, progettato per avvicinarsi alla qualità del path tracing in scene dinamiche. Usa Rust per il codice del motore, Vulkan come backend grafico e un ampio stack di shader HLSL per pass ibridi di rasterizzazione, compute e ray tracing. Le funzionalità principali includono GI dinamica senza probe precalcolati, ricostruzione temporale, ombre e riflessi ray-traced e una modalità di path tracing di riferimento per la validazione. Il caso d’uso principale è la ricerca e la prototipazione di rendering avanzato per sviluppatori che studiano le tecniche moderne di illuminazione in tempo reale.
