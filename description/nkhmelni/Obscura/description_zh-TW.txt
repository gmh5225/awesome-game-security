Obscura 是一款樹外（out-of-tree）LLVM pass 外掛，可在編譯期對二進位進行混淆，無需修改原始碼。它提供十三種 pass，涵蓋 Objective-C 中繼資料保護、反除錯與反 Hook 檢查、字串與常數加密、動態符號解析，以及虛假控制流、控制流平坦化、指令替換與間接分支等經典程式碼混淆技術。這些 pass 透過 LLVM 新版 Pass Manager 注入，可用編譯器旗標或小型設定標頭控制，並支援可選的依函式註解。它面向 Clang、AppleClang 與 Swift 工具鏈，支援 C、C++、Objective-C 與 Swift，在 Darwin 上支援最為完善。主要用途是強化應用與遊戲用戶端，抵禦逆向工程、類別傾印、除錯以及執行階段 Hook。
