Obscura is an out-of-tree LLVM pass plugin that obfuscates binaries at compile time without requiring source code changes. It ships thirteen passes covering Objective-C metadata protection, anti-debug and anti-hook checks, string and constant encryption, dynamic symbol resolution, and classic code obfuscation such as bogus control flow, control flow flattening, instruction substitution, and indirect branching. Passes are injected through LLVM’s new Pass Manager and controlled with compiler flags or a small config header, with optional per-function annotations. It targets Clang, AppleClang, and Swift toolchains for C, C++, Objective-C, and Swift, with strongest support on Darwin. The primary use case is hardening apps and game clients against reverse engineering, class dumping, debugging, and runtime hooking.
