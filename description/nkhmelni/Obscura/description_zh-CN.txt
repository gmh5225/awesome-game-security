Obscura 是一款树外（out-of-tree）LLVM pass 插件，可在编译期对二进制进行混淆，无需修改源代码。它提供十三种 pass，涵盖 Objective-C 元数据保护、反调试与反 Hook 检测、字符串与常量加密、动态符号解析，以及虚假控制流、控制流平坦化、指令替换和间接跳转等经典代码混淆技术。这些 pass 通过 LLVM 新版 Pass Manager 注入，可用编译器标志或小型配置头文件控制，并支持可选的按函数注解。它面向 Clang、AppleClang 和 Swift 工具链，支持 C、C++、Objective-C 与 Swift，在 Darwin 上支持最为完善。主要用途是加固应用与游戏客户端，抵御逆向工程、类转储、调试以及运行时 Hook。
