Obscura は、ソースコードの変更を必要とせず、コンパイル時にバイナリを難読化するアウトオブツリーの LLVM パスプラグインです。Objective-C メタデータ保護、アンチデバッグおよびアンチフック検査、文字列・定数の暗号化、動的シンボル解決に加え、偽制御フロー、制御フロー平坦化、命令置換、間接分岐といった古典的なコード難読化まで、十三種類のパスを提供します。パスは LLVM の新しい Pass Manager 経由で注入され、コンパイラフラグや小さな設定ヘッダで制御でき、関数単位のアノテーションも任意で利用できます。Clang、AppleClang、Swift ツールチェーン向けで、C、C++、Objective-C、Swift を対象とし、Darwin でのサポートが最も充実しています。主な用途は、リバースエンジニアリング、クラスダンプ、デバッグ、ランタイムフックに対するアプリおよびゲームクライアントの強化です。
