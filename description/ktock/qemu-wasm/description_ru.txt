Этот проект — порт QEMU на WebAssembly, позволяющий запускать виртуальные машины прямо в веб-браузерах. Он компилирует эмуляторы x86 и других архитектур QEMU в WASM с помощью Emscripten, давая возможность загружать Linux, Windows и другие ОС во вкладке браузера без серверной обработки. Проект обеспечивает эмуляцию устройств VirtIO, сеть и хранилище через API браузера. Он предназначен для исследователей эмуляции, преподавателей и разработчиков, которым нужен доступ к виртуальным машинам в браузере.
