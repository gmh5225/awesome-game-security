Questo progetto è un port WebAssembly di QEMU che esegue macchine virtuali direttamente nei browser web. Compila gli emulatori x86 e di altre architetture di QEMU in WASM con Emscripten, consentendo di avviare Linux, Windows e altri sistemi operativi in una scheda del browser senza elaborazione lato server. Il progetto gestisce l'emulazione dei dispositivi VirtIO, la rete e lo storage tramite le API del browser. È rivolto a ricercatori di emulazione, educatori e sviluppatori che desiderano accedere a macchine virtuali nel browser.
