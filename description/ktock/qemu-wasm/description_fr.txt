Ce projet est un portage WebAssembly de QEMU qui exécute des machines virtuelles directement dans les navigateurs web. Il compile les émulateurs x86 et d'autres architectures de QEMU en WASM avec Emscripten, permettant de démarrer Linux, Windows et d'autres systèmes d'exploitation dans un onglet de navigateur sans traitement côté serveur. Le projet gère l'émulation des périphériques VirtIO, le réseau et le stockage via les API du navigateur. Il s'adresse aux chercheurs en émulation, aux enseignants et aux développeurs souhaitant un accès aux machines virtuelles dans le navigateur.
