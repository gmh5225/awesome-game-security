GuardAC は、Minecraft 向けの AI 駆動・無料オープンソースのサーバーサイドアンチチートプラグインで、ゲームプレイを監視し、クラウド API にエイムチェック判定を依頼し、アラート・違反追跡・処罰をローカルで適用します。Kotlin で書かれ、Gradle でビルドされる JavaPlugin として、Spigot、Paper、または Folia 上の Minecraft 1.21.x に対応します（実行は Java 17+、ビルドは JDK 21+）。主な機能には、設定可能なアラートのみモード、サーバー間レピュテーション共有、任意のバンアニメーション付き処罰ラダー、ライブ容疑者監視とホログラム、オンデマンドのディープスキャン、Geyser Bedrock プレイヤーおよび選択した WorldGuard 領域の自動免除があります。公開リポジトリはプラグインクライアントのみを提供し、リモート推論バックエンドは非公開です。AI 支援の戦闘チート検出を、運営者が慎重に制御できる形で求める Minecraft サーバー運営者向けです。
