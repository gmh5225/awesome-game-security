GuardAC is an AI-powered, free and open-source server-side anti-cheat plugin for Minecraft that watches gameplay, requests aim-check verdicts from its cloud API, and applies alerts, violation tracking, and punishments locally. It is written in Kotlin as a Gradle-built JavaPlugin for Spigot, Paper, or Folia on Minecraft 1.21.x (Java 17+ to run, JDK 21+ to build). Key capabilities include configurable alert-only mode, cross-server reputation sharing, punishment ladders with optional ban animations, live suspect monitoring and holograms, on-demand deep scans, and automatic exemptions for Geyser Bedrock players and selected WorldGuard regions. The public repository ships only the plugin client; the remote inference backend stays private. It is aimed at Minecraft server operators who want AI-assisted combat cheat detection with conservative, operator-controlled enforcement.
