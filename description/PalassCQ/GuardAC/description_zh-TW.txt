GuardAC 是一款以 AI 驅動的免費開源 Minecraft 伺服器端反作弊外掛，可監控遊戲行為，透過雲端 API 請求瞄準檢測判定，並在本機執行警示、違規追蹤與懲處。它以 Kotlin 撰寫，作為以 Gradle 建置的 JavaPlugin，支援 Spigot、Paper 或 Folia，適用於 Minecraft 1.21.x（執行需 Java 17+，建置需 JDK 21+）。主要功能包括可設定的僅警示模式、跨伺服器信譽共享、含可選封鎖動畫的懲處階梯、即時嫌疑監控與全像顯示、隨需深度掃描，以及對 Geyser Bedrock 玩家與指定 WorldGuard 區域的自動豁免。公開儲存庫僅提供外掛用戶端，遠端推論後端維持私有。面向希望獲得 AI 輔助戰鬥作弊偵測、並由營運者保守可控執行懲處的 Minecraft 伺服器營運者。
