GuardAC 是一款基于 AI 的免费开源 Minecraft 服务端反作弊插件，可监控游戏行为，通过云端 API 请求瞄准检测判定，并在本地执行告警、违规追踪与处罚。它使用 Kotlin 编写，作为基于 Gradle 构建的 JavaPlugin，支持 Spigot、Paper 或 Folia，面向 Minecraft 1.21.x（运行需 Java 17+，构建需 JDK 21+）。主要功能包括可配置的仅告警模式、跨服信誉共享、带可选封禁动画的处罚阶梯、实时嫌疑监控与全息显示、按需深度扫描，以及对 Geyser Bedrock 玩家和指定 WorldGuard 区域的自动豁免。公开仓库仅提供插件客户端，远程推理后端保持私有。面向希望获得 AI 辅助战斗作弊检测、并由管理员保守可控执行处罚的 Minecraft 服主。
