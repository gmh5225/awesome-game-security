GuardAC è un plugin anti-cheat lato server per Minecraft, gratuito, open source e basato sull’IA: monitora il gameplay, richiede verdetti di aim-check alla sua API cloud e applica localmente avvisi, tracciamento delle violazioni e sanzioni. È scritto in Kotlin come JavaPlugin costruito con Gradle per Spigot, Paper o Folia su Minecraft 1.21.x (Java 17+ per l’esecuzione, JDK 21+ per la build). Le funzionalità principali includono una modalità solo avvisi configurabile, reputazione condivisa tra server, scale di punizione con animazioni di ban opzionali, monitoraggio live dei sospetti e ologrammi, deep scan su richiesta ed esenzioni automatiche per i giocatori Geyser Bedrock e regioni WorldGuard selezionate. Il repository pubblico fornisce solo il client del plugin; il backend di inferenza remoto resta privato. È rivolto agli operatori di server Minecraft che vogliono un rilevamento dei cheat di combattimento assistito dall’IA con applicazione conservativa e controllata dall’operatore.
