GuardAC — бесплатный open-source серверный античит-плагин для Minecraft на базе ИИ: он отслеживает геймплей, запрашивает вердикты aim-check у своего облачного API и локально применяет оповещения, учёт нарушений и наказания. Написан на Kotlin как JavaPlugin, собираемый Gradle, для Spigot, Paper или Folia на Minecraft 1.21.x (для запуска нужен Java 17+, для сборки — JDK 21+). Ключевые возможности: настраиваемый режим только оповещений, межсерверный обмен репутацией, лестницы наказаний с опциональными анимациями бана, живой мониторинг подозреваемых и голограммы, углубленные проверки по запросу, а также автоматические исключения для игроков Geyser Bedrock и выбранных регионов WorldGuard. Публичный репозиторий содержит только клиент плагина; удалённый backend инференса остаётся закрытым. Плагин рассчитан на операторов серверов Minecraft, которым нужна ИИ-помощная детекция боевых читов с осторожным, контролируемым оператором применением мер.
