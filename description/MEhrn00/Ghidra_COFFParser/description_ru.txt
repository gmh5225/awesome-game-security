Этот проект — скрипт анализа Ghidra, выполняющий полноценный разбор COFF сверх поведения загрузчика по умолчанию. Он написан на Python и сопоставляет заголовки, данные символов, таблицы строк и записи релокаций с добавленной информацией о типах. Скрипт также применяет релокации и перекрёстные ссылки, улучшая контекст анализа в дизассемблере. Основной сценарий использования — реверс-инжиниринг объектных файлов COFF с более богатой структурной видимостью в Ghidra.
