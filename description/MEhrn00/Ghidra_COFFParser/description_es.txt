Este proyecto es un script de análisis de Ghidra que realiza un análisis exhaustivo de COFF más allá del comportamiento del cargador predeterminado. Está escrito en Python y mapea cabeceras, datos de símbolos, tablas de cadenas y entradas de reubicación con información de tipos añadida. El script también aplica reubicaciones y referencias cruzadas para mejorar el contexto de análisis dentro del desensamblador. El caso de uso principal es la ingeniería inversa de archivos objeto COFF con mayor visibilidad estructural en Ghidra.
