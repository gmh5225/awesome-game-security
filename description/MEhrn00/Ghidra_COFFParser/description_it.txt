Questo progetto è uno script di analisi Ghidra che esegue un parsing COFF completo oltre il comportamento del loader predefinito. È scritto in Python e mappa header, dati dei simboli, tabelle delle stringhe e voci di relocation con informazioni di tipo aggiuntive. Lo script applica anche relocation e riferimenti incrociati per migliorare il contesto di analisi nel disassemblatore. Il caso d'uso principale è il reverse engineering di file oggetto COFF con una più ricca visibilità strutturale in Ghidra.
