Ce projet est un script d'analyse Ghidra qui effectue un parsing COFF complet au-delà du comportement du chargeur par défaut. Il est écrit en Python et mappe les en-têtes, les données de symboles, les tables de chaînes et les entrées de relocation avec des informations de type supplémentaires. Le script applique également les relocations et les références croisées pour enrichir le contexte d'analyse dans le désassembleur. Le cas d'usage principal est l'ingénierie inverse de fichiers objets COFF avec une meilleure visibilité structurelle dans Ghidra.
