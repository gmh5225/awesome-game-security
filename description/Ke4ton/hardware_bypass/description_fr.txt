Ce projet est un contournement basé sur une DLL visant une vérification matérielle GPU côté jeu. Il est conçu comme un projet C++ Visual Studio et destiné à être injecté peu après le lancement du jeu afin de patcher ou modifier le comportement de validation à l'exécution. Le dépôt se concentre sur un flux de contournement étroit plutôt que sur un framework complet, avec un usage simple de type build-and-inject. Son public principal est constitué de reverse engineers qui étudient les contrôles d'intégrité client et le filtrage matériel lié à l'anti-cheat.
