Этот проект — DLL-обход, нацеленный на игровую проверку GPU-оборудования. Он оформлен как C++-проект Visual Studio и предназначен для внедрения вскоре после запуска игры, чтобы пропатчить или изменить поведение проверок во время выполнения. Репозиторий сосредоточен на узком сценарии обхода, а не на полноценном фреймворке, с простым использованием по схеме сборка-и-инъекция. Основная аудитория — реверс-инженеры, изучающие проверки целостности клиента и аппаратное гейтирование, связанное с античитом.
