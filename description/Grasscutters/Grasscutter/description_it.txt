Questo progetto è un emulatore open-source di server di gioco per Genshin Impact, che implementa la logica di gioco lato server, la progressione del giocatore, la simulazione del mondo e la gestione del protocollo di rete. Il server basato su Java supporta missioni, sistemi gacha, dungeon, interazioni multiplayer ed eventi di gioco scriptati. È soprattutto utile per ricercatori di server di gioco e reverse engineer che studiano l'architettura dei server MMORPG e il protocollo client-server di Genshin Impact.
