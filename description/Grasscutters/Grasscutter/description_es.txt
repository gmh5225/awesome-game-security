Este proyecto es un emulador de servidor de juego de código abierto para Genshin Impact, que implementa la lógica de juego del lado del servidor, la progresión del jugador, la simulación del mundo y el manejo del protocolo de red. El servidor basado en Java admite misiones, sistemas de gacha, mazmorras, interacciones multijugador y eventos de juego con scripts. Es especialmente útil para investigadores de servidores de juego e ingenieros de ingeniería inversa que estudian la arquitectura de servidores MMORPG y el protocolo cliente-servidor de Genshin Impact.
