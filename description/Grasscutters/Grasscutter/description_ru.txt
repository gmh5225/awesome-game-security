Этот проект — открытый эмулятор игрового сервера для Genshin Impact, реализующий серверную игровую логику, прогресс игрока, симуляцию мира и обработку сетевого протокола. Сервер на Java поддерживает квесты, системы гачи, подземелья, многопользовательское взаимодействие и скриптовые игровые события. Он в первую очередь полезен исследователям игровых серверов и специалистам по реверс-инжинирингу, изучающим архитектуру MMORPG-серверов и клиент-серверный протокол Genshin Impact.
