本プロジェクトは、Xiaomi K70e（コードネーム duchamp）向けのワンクリック Android root ツールであり、CVE-2026-43499（IonStack）カーネル脆弱性を悪用して組み込みの KernelSU デーモンをインストールします。主に aarch64 Android 向けの C で記述され、NDK/clang ツールチェーンでビルドされ、LD_PRELOAD 共有ライブラリ（preload.so）として提供されます。エクスプロイト連鎖は pselect サイドチャネルで KASLR ベースを漏えいし、futex PI 競合でファイル操作テーブルを乗っ取り、pipe buffer ops を偽造して任意のカーネル物理読み書きを実現し、さらにプロセス資格情報をパッチして uid 0 を取得します。また root 後の権限管理用に ksud を埋め込み起動し、duchamp および関連する Android 16 ビルド向けのデバイス別ターゲットオフセットも含みます。想定読者は、Android カーネルの権限昇格、root 永続化、および関連するリバースエンジニアリング手法を研究するセキュリティ研究者です。
