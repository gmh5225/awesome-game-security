Ce projet est un outil de root Android en un clic pour le Xiaomi K70e (nom de code duchamp) qui exploite la vulnérabilité noyau CVE-2026-43499 (IonStack) et installe un démon KernelSU intégré. Il est principalement écrit en C pour Android aarch64, compilé avec la chaîne d'outils NDK/clang, et livré sous forme de bibliothèque partagée LD_PRELOAD (preload.so). La chaîne d'exploitation divulgue la base KASLR via un canal auxiliaire pselect, détourne les tables d'opérations sur fichiers par une condition de course futex PI, forge les pipe buffer ops pour une lecture/écriture physique arbitraire du noyau, puis modifie les credentials du processus pour obtenir l'uid 0. Il embarque et lance également ksud pour la gestion des privilèges après le root, et inclut des offsets cibles par appareil pour duchamp et les builds Android 16 associés. Le public visé est constitué de chercheurs en sécurité étudiant l'élévation de privilèges du noyau Android, la persistance du root et les techniques de rétro-ingénierie associées.
