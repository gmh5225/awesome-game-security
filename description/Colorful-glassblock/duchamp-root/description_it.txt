Questo progetto è uno strumento di root Android one-click per lo Xiaomi K70e (codename duchamp) che sfrutta la vulnerabilità del kernel CVE-2026-43499 (IonStack) e installa un demone KernelSU incorporato. È scritto principalmente in C per Android aarch64, compilato con la toolchain NDK/clang e distribuito come libreria condivisa LD_PRELOAD (preload.so). La catena di exploit rivela la base KASLR tramite un canale laterale pselect, dirotta le tabelle delle operazioni sui file mediante una race condition futex PI, contraffà le pipe buffer ops per lettura/scrittura fisica arbitraria del kernel, quindi applica patch alle credenziali del processo per ottenere uid 0. Incorpora e avvia anche ksud per la gestione dei privilegi dopo il root e include offset target per dispositivo per duchamp e build Android 16 correlate. Il pubblico previsto sono i ricercatori di sicurezza che studiano l'escalation di privilegi del kernel Android, la persistenza del root e le tecniche correlate di reverse engineering.
