Este proyecto es una herramienta de root de Android en un clic para el Xiaomi K70e (nombre en clave duchamp) que explota la vulnerabilidad del kernel CVE-2026-43499 (IonStack) e instala un demonio KernelSU integrado. Está escrito principalmente en C para Android aarch64, se compila con la cadena de herramientas NDK/clang y se entrega como biblioteca compartida LD_PRELOAD (preload.so). La cadena de explotación filtra la base KASLR mediante un canal lateral pselect, secuestra tablas de operaciones de archivos a través de una condición de carrera futex PI, falsifica pipe buffer ops para lectura/escritura física arbitraria del kernel y luego parchea las credenciales del proceso para obtener uid 0. También incorpora y lanza ksud para la gestión de privilegios tras el root e incluye offsets de destino por dispositivo para duchamp y builds relacionados de Android 16. El público previsto son investigadores de seguridad que estudian la escalada de privilegios del kernel de Android, la persistencia del root y técnicas relacionadas de ingeniería inversa.
