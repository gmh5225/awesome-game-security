이 프로젝트는 Xiaomi K70e(코드명 duchamp)용 원클릭 Android root 도구로, CVE-2026-43499(IonStack) 커널 취약점을 악용해 내장 KernelSU 데몬을 설치합니다. 주로 aarch64 Android용 C로 작성되었으며 NDK/clang 툴체인으로 빌드되고, LD_PRELOAD 공유 라이브러리(preload.so) 형태로 제공됩니다. 익스플로잇 체인은 pselect 사이드 채널로 KASLR 베이스를 누설하고, futex PI 레이스로 파일 연산 테이블을 하이재킹하며, pipe buffer ops를 위조해 임의 커널 물리 읽기/쓰기를 구현한 뒤 프로세스 credential을 패치해 uid 0을 얻습니다. 또한 root 이후 권한 관리를 위해 ksud를 내장·실행하며, duchamp 및 관련 Android 16 빌드용 기기별 타깃 오프셋을 포함합니다. 대상 독자는 Android 커널 권한 상승, root 지속성, 관련 리버스 엔지니어링 기법을 연구하는 보안 연구자입니다.
