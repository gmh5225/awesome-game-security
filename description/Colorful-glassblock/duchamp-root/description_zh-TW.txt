本專案是面向小米 K70e（代號 duchamp）的一鍵 Android root 工具，利用 CVE-2026-43499（IonStack）核心漏洞並安裝內嵌的 KernelSU 守護程式。主體以 C 撰寫，面向 aarch64 Android，使用 NDK/clang 工具鏈建置，並以 LD_PRELOAD 共用函式庫（preload.so）形式交付。利用鏈透過 pselect 側通道洩漏 KASLR 基址，藉由 futex PI 競態劫持檔案操作表，偽造 pipe buffer ops 以實現任意核心實體讀寫，再修補行程憑證以取得 uid 0。專案還嵌入並啟動 ksud，用於 root 後的權限管理，並包含 duchamp 及相關 Android 16 建置的依裝置目標偏移。目標讀者為研究 Android 核心提權、root 持久化及相關逆向工程技術的安全研究人員。
