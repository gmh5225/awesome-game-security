本项目是面向小米 K70e（代号 duchamp）的一键 Android root 工具，利用 CVE-2026-43499（IonStack）内核漏洞并安装内嵌的 KernelSU 守护进程。主体以 C 编写，面向 aarch64 Android，使用 NDK/clang 工具链构建，并以 LD_PRELOAD 共享库（preload.so）形式交付。利用链通过 pselect 侧信道泄露 KASLR 基址，借助 futex PI 竞态劫持文件操作表，伪造 pipe buffer ops 以实现任意内核物理读写，再修补进程凭证以获得 uid 0。项目还嵌入并启动 ksud，用于 root 后的权限管理，并包含 duchamp 及相关 Android 16 构建的按设备目标偏移。目标读者为研究 Android 内核提权、root 持久化及相关逆向工程技术的安全研究人员。
