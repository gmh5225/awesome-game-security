This project is a one-click Android root tool for the Xiaomi K70e (codename duchamp) that exploits the CVE-2026-43499 (IonStack) kernel vulnerability and installs an embedded KernelSU daemon. It is written primarily in C for aarch64 Android, built with the NDK/clang toolchain, and delivered as an LD_PRELOAD shared library (preload.so). The exploit chain leaks the KASLR base via a pselect side channel, hijacks file-operation tables through a futex PI race, forges pipe buffer ops for arbitrary kernel physical read/write, then patches process credentials for uid 0. It also embeds and launches ksud for post-root privilege management and includes per-device target offsets for duchamp and related Android 16 builds. The intended audience is security researchers studying Android kernel privilege escalation, root persistence, and related reverse-engineering techniques.
