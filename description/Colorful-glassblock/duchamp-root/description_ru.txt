Этот проект — инструмент однокликового Android root для Xiaomi K70e (кодовое имя duchamp), эксплуатирующий уязвимость ядра CVE-2026-43499 (IonStack) и устанавливающий встроенный демон KernelSU. Он написан в основном на C для aarch64 Android, собирается инструментарием NDK/clang и поставляется как разделяемая библиотека LD_PRELOAD (preload.so). Цепочка эксплойта раскрывает базу KASLR через побочный канал pselect, перехватывает таблицы файловых операций через гонку futex PI, подделывает pipe buffer ops для произвольного физического чтения/записи ядра, затем патчит учётные данные процесса для получения uid 0. Также встраивает и запускает ksud для управления привилегиями после root и включает целевые смещения для устройств duchamp и связанных сборок Android 16. Целевая аудитория — исследователи безопасности, изучающие повышение привилегий ядра Android, сохранение root и связанные методы реверс-инжиниринга.
