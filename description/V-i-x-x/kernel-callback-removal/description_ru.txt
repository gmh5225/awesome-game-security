Этот проект документирует и реализует технику обхода ядра ETW-TI, которая переключает состояние провайдера из памяти ядра. В нём объясняется, как находить соответствующие структуры ядра Windows, определять смещения и изменять флаги включения с помощью существующего примитива чтения/записи. Репозиторий сочетает код реализации на C++ с подробными заметками по реверс-инжинирингу на основе WinDbg и IDA. Он предназначен для продвинутых образовательных исследований пентестеров и защитников, изучающих методы обхода EDR.
