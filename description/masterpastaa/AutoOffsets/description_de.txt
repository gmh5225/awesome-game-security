C++-Dienstprogramm, das aktuelle Fortnite-Spieloffsets von einem Remote-API-Endpunkt über den WinINet-HTTP-Stack (InternetOpenUrlA/InternetReadFile) abruft. Die Offsets stammen aus einer externen Reverse-Engineering-Datenbank und können auch programmatisch über eine REST-API mit schlüsselbasierter Authentifizierung abgefragt werden.
