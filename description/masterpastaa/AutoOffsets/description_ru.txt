C++-утилита, которая получает актуальные игровые offsets Fortnite с удалённой конечной точки API через HTTP-стек WinINet (InternetOpenUrlA/InternetReadFile). Offsets берутся из внешней базы данных для reverse engineering и также могут запрашиваться программно через REST API с аутентификацией по ключу.
