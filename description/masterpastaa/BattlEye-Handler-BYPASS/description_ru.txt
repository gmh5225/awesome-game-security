Драйвер ядра Windows KMDF, обходящий механизм снятия дескрипторов BattlEye за счёт непрерывного повторного создания дескрипторов процессов до того, как их удалит цикл очистки BattlEye длительностью около 5 секунд. Собран с помощью набора инструментов WindowsKernelModeDriver10.0 и содержит процедуры диспетчеризации I/O Control для связи с пользовательским режимом.
