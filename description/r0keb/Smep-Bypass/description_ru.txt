Этот проект — proof of concept обхода SMEP (Supervisor Mode Execution Prevention) в Windows. SMEP запрещает ядру выполнять код из страниц пользовательского режима. Данный эксплойт демонстрирует техники отключения SMEP путём манипуляции регистром CR4 через ROP-цепочки или примитивы уязвимых драйверов, что позволяет выполнять shellcode пользовательского пространства в режиме ядра. Предназначен для исследователей kernel exploitation, изучающих обход аппаратных механизмов безопасности.
