Этот проект — обучающий набор инструментов BYOVD, который через уязвимый драйвер предоставляет примитивы чтения и записи ядра в Windows. Он включает несколько proof-of-concept утилит на C для изменения уровней защиты процессов, редактирования токенов, отключения телеметрии ETW, удаления колбэков ядра и минифильтров, а также изменения состояния DSE. Инструменты разрешают смещения и символы ядра, затем выполняют привилегированные операции с памятью через обмен по IOCTL. Набор предназначен для санкционированного исследования безопасности ядра и изучения пробелов в защите античитов или EDR.
