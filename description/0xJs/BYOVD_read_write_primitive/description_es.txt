Este proyecto es un kit de herramientas BYOVD educativo que expone primitivas de lectura y escritura en el kernel de Windows mediante un controlador vulnerable. Contiene varias herramientas de prueba de concepto en C para cambiar niveles de protección de procesos, editar tokens, desactivar la telemetría ETW, eliminar callbacks del kernel y minifiltros, y modificar el estado de DSE. Las herramientas resuelven offsets y símbolos del kernel y luego realizan operaciones de memoria privilegiadas mediante comunicación basada en IOCTL. Está diseñado para investigación autorizada de seguridad del kernel y para estudiar lagunas de endurecimiento en anti-cheat o EDR.
