Dieses Projekt ist ein didaktisches BYOVD-Toolkit, das über einen verwundbaren Treiber Kernel-Lese- und Schreibprimitive unter Windows bereitstellt. Es enthält mehrere C-Proof-of-Concept-Tools zum Ändern von Prozessschutzstufen, Bearbeiten von Tokens, Deaktivieren von ETW-Telemetrie, Entfernen von Kernel-Callbacks und Minifiltern sowie zum Modifizieren des DSE-Zustands. Die Werkzeuge lösen Kernel-Offsets und -Symbole auf und führen dann privilegierte Speicheroperationen über IOCTL-basierte Kommunikation aus. Es ist für autorisierte Kernel-Sicherheitsforschung und zum Studium von Härtungslücken bei Anti-Cheat- oder EDR-Systemen gedacht.
