Этот проект — образовательный proof of concept Windows BYOVD, который злоупотребляет уязвимым подписанным драйвером для завершения выбранных процессов безопасности. В нём задокументирован реверс-инжиниринг интерфейса драйвера, включая именование устройства, выбор IOCTL и структуру требуемого входного буфера. Реализация на C автоматизирует развёртывание драйвера, перечисление целевых процессов, повторные попытки завершения и очистку при выходе. Предназначен для санкционированных red-team исследований устойчивости защитных продуктов и поверхности атаки ядра.
