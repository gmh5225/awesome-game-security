Ce projet est NoVmpy, un outil Python permettant de désobscurcir le code virtualisé par VMProtect à l'aide de l'exécution symbolique. Il parcourt les chaînes de gestionnaires de la machine virtuelle VMProtect, exécute symboliquement chaque gestionnaire pour en extraire la sémantique, et reconstruit la séquence d'instructions d'origine à partir du bytecode virtualisé. L'outil s'appuie sur Triton ou des frameworks d'exécution symbolique similaires pour l'analyse. Il s'adresse aux ingénieurs en rétro-ingénierie et aux analystes de logiciels malveillants qui cherchent à récupérer le code d'origine depuis des binaires protégés par VMProtect.
