Этот проект — NoVmpy, инструмент на Python для деобфускации кода, виртуализированного VMProtect, с помощью символьного выполнения. Он проходит по цепочкам обработчиков виртуальной машины VMProtect, символически выполняет каждый обработчик для извлечения его семантики и восстанавливает исходную последовательность инструкций из виртуализированного байткода. Инструмент опирается на Triton или аналогичные фреймворки символьного выполнения для анализа. Он предназначен для реверс-инженеров и аналитиков вредоносного ПО, восстанавливающих исходный код из защищённых VMProtect двоичных файлов.
