Dieses Projekt ist NoVmpy, ein Python-Werkzeug zur Entschleierung von durch VMProtect virtualisiertem Code mittels symbolischer Ausführung. Es verfolgt die Handler-Ketten der virtuellen Maschine von VMProtect, führt jeden Handler symbolisch aus, um dessen Semantik zu extrahieren, und rekonstruiert die ursprüngliche Befehlssequenz aus dem virtualisierten Bytecode. Das Werkzeug nutzt Triton oder ähnliche Frameworks für symbolische Ausführung zur Analyse. Es richtet sich an Reverse Engineers und Malware-Analysten, die Originalcode aus mit VMProtect geschützten Binärdateien wiederherstellen.
