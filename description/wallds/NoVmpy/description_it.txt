Questo progetto è NoVmpy, uno strumento Python per deoffuscare il codice virtualizzato da VMProtect mediante esecuzione simbolica. Traccia le catene di handler della macchina virtuale di VMProtect, esegue simbolicamente ciascun handler per estrarne la semantica e ricostruisce la sequenza di istruzioni originale a partire dal bytecode virtualizzato. Lo strumento sfrutta Triton o framework simili di esecuzione simbolica per l'analisi. È rivolto a reverse engineer e analisti di malware che recuperano il codice originale da binari protetti con VMProtect.
