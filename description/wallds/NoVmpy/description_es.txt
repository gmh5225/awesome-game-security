Este proyecto es NoVmpy, una herramienta en Python para desofuscar código virtualizado por VMProtect mediante ejecución simbólica. Recorre las cadenas de manejadores de la máquina virtual de VMProtect, ejecuta simbólicamente cada manejador para extraer su semántica y reconstruye la secuencia original de instrucciones a partir del bytecode virtualizado. La herramienta aprovecha Triton o frameworks similares de ejecución simbólica para el análisis. Está dirigida a ingenieros de ingeniería inversa y analistas de malware que recuperan el código original de binarios protegidos con VMProtect.
