Этот проект предоставляет функции unpack, pack, list, check и mount для архивов .pak Unreal Engine 4.
Обратите внимание, что поддерживается лишь ограниченное число версий pak.
Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game engine explorer:unreal.
