Dieses Projekt bietet Unpack-, Pack-, List-, Check- und Mount-Funktionen für Unreal Engine 4 .pak-Archive.
Beachten Sie, dass nur eine begrenzte Anzahl von Pak-Versionen unterstützt wird.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Game Engine Explorer:Unreal untersuchen.
