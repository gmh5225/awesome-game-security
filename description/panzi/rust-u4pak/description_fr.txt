Ce projet fournit les fonctions unpack, pack, list, check et mount pour les archives .pak d'Unreal Engine 4.
Notez que seul un nombre limité de versions de pak est pris en charge.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine cheat / game engine explorer:unreal.
