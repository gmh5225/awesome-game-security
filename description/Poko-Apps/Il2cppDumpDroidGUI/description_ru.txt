Этот проект — Android GUI-адаптация Il2CppDumper для задач дампа IL2CPP непосредственно на устройстве. Распространяется в основном в виде APK-релизов с документацией, журналами изменений и рекомендациями по настройке, а не как полностью открытая исходная реализация. Инструмент рассчитан на Android 6–14 для устройств armeabi-v7a и arm64-v8a и следует обновлениям upstream Il2CppDumper. Предназначен для специалистов по реверс-инжинирингу мобильных игр, которым нужен упрощённый процесс дампа на Android.
