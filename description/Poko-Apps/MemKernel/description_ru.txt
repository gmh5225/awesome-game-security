Этот проект — набор драйвера ядра Android для низкоуровневого доступа к памяти процессов. Он содержит компоненты ядра и пользовательского пространства на C и C++ для чтения и записи целевой памяти через собственный интерфейс. В репозитории есть рекомендации по интеграции: сборке драйвера в образ ядра и связке с пользовательскими инструментами. Он рассчитан на опытных исследователей безопасности Android и игровой памяти, изучающих операции с памятью при поддержке ядра.
