Este proyecto es una configuración de controlador del kernel de Android para acceso de bajo nivel a la memoria de procesos. Contiene componentes de kernel y de usuario en C y C++ para leer y escribir la memoria objetivo mediante una interfaz personalizada. El repositorio incluye orientación de integración para compilar el controlador en una compilación del kernel y emparejarlo con herramientas de espacio de usuario. Está dirigido a investigadores avanzados de seguridad Android y memoria de juegos que exploran operaciones de memoria asistidas por el kernel.
