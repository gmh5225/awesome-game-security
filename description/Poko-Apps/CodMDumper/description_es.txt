Este proyecto es un paquete dumper centrado en extraer binarios il2cpp y metadatos de Call of Duty Mobile en dispositivos Android. El repositorio es sobre todo una distribución orientada a releases, con documentación de uso en lugar del código fuente completo. Destaca el soporte para entornos armv7 y arm64, Termux o terminales similares, y artefactos de salida como dump.cs, ida.py y datos JSON para IDA. Está pensado para flujos de ingeniería inversa de juegos móviles y el autor lo marca como deprecated.
