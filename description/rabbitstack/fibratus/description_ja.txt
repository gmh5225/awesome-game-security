本プロジェクトは、Go で書かれた Windows カーネル探索・可観測性ツールであり、ETW（Event Tracing for Windows）を通じてカーネルイベントストリームをキャプチャし分析します。プロセス生成、スレッド活動、ファイル I/O、レジストリ操作、ネットワーク接続、ドライバ読み込みをリアルタイムで監視し、フィルタリングルール、アラート、Elasticsearch を含む各種シンクへの出力に対応します。不審な挙動パターンを検出するルールエンジンを備え、脅威ハンティングやエンドポイントテレメトリに利用できます。Windows カーネルレベルの活動に関する検知能力を構築するセキュリティアナリスト、脅威ハンター、防御担当者を対象としています。
