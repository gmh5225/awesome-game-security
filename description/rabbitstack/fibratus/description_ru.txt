Этот проект — инструмент исследования и наблюдаемости ядра Windows на Go, который захватывает и анализирует потоки событий ядра через ETW (Event Tracing for Windows). Он в реальном времени отслеживает создание процессов, активность потоков, файловый ввод-вывод, операции с реестром, сетевые соединения и загрузку драйверов, поддерживая правила фильтрации, оповещения и вывод в различные приёмники, включая Elasticsearch. В инструмент входит движок правил для обнаружения подозрительных шаблонов поведения; его можно использовать для threat hunting и телеметрии конечных точек. Он ориентирован на аналитиков безопасности, threat hunter'ов и защитников, строящих возможности детектирования вокруг активности на уровне ядра Windows.
