이 프로젝트는 Go로 작성된 Windows 커널 탐색 및 관측 도구로, ETW(Event Tracing for Windows)를 통해 커널 이벤트 스트림을 캡처하고 분석합니다. 프로세스 생성, 스레드 활동, 파일 I/O, 레지스트리 작업, 네트워크 연결, 드라이버 로딩을 실시간으로 모니터링하며, 필터링 규칙, 알림, Elasticsearch를 포함한 다양한 싱크로의 출력을 지원합니다. 의심스러운 행위 패턴을 탐지하는 규칙 엔진을 포함하며, 위협 헌팅과 엔드포인트 텔레메트리에 사용할 수 있습니다. Windows 커널 수준 활동을 중심으로 탐지 역량을 구축하는 보안 분석가, 위협 헌터, 방어 담당자를 대상으로 합니다.
