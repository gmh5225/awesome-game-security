Tool-Tree 是一款面向 ARM64 设备的 Android ROM 与 APK 工具应用，可用于解包、打包和编辑固件镜像与应用程序包。它支持 boot、dtbo、ext4、erofs、f2fs、payload、super、zip、APK、APKS、APEX、CAPEX、squashfs 以及 Amlogic 镜像等格式的解包与打包，并内置类似 apktool 的 APK 解码/构建、签名以及 BusyBox 等 shell 实用工具。项目主要使用 Kotlin 和 Java 实现，辅以 Bash 脚本和类 KRScript 的界面来运行工具工作流，可在有 root 或无 root 环境下使用。可扩展的 Addon 与 Apkon 模块允许用户增加 ROM 与 APK 编辑能力。它面向需要在设备上操作固件和软件包的 Android 逆向工程师、ROM 修改者以及 APK 分析人员。
