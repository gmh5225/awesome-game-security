Tool-Tree は、ARM64 デバイス上でファームウェアイメージとアプリケーションパッケージの展開・梱包・編集を行う Android ROM / APK ツールキットアプリです。boot、dtbo、ext4、erofs、f2fs、payload、super、zip、APK、APKS、APEX、CAPEX、squashfs、Amlogic イメージなどの形式の展開と梱包に対応し、apktool 風の APK デコード／ビルド、署名、BusyBox などの組み込みシェルユーティリティも備えます。プロジェクトは主に Kotlin と Java で実装され、Bash スクリプトと KRScript 風 UI でツールワークフローを実行でき、root の有無を問わず動作します。拡張可能な Addon および Apkon モジュールにより、ROM と APK の編集機能を追加できます。デバイス上でファームウェアとパッケージを操作したい Android リバースエンジニア、ROM モッダー、APK 分析者向けです。
