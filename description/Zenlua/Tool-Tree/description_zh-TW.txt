Tool-Tree 是一款面向 ARM64 裝置的 Android ROM 與 APK 工具應用，可用於解包、打包與編輯韌體映像與應用程式套件。它支援 boot、dtbo、ext4、erofs、f2fs、payload、super、zip、APK、APKS、APEX、CAPEX、squashfs 以及 Amlogic 映像等格式的解包與打包，並內建類似 apktool 的 APK 解碼/建置、簽名以及 BusyBox 等 shell 實用工具。專案主要以 Kotlin 與 Java 實作，輔以 Bash 腳本與類 KRScript 的介面來執行工具工作流程，可在有 root 或無 root 環境下使用。可擴充的 Addon 與 Apkon 模組讓使用者能新增 ROM 與 APK 編輯能力。它面向需要在裝置上操作韌體與套件的 Android 逆向工程師、ROM 修改者以及 APK 分析人員。
