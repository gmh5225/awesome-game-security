Questo progetto si concentra su un toolkit di hardening rCE per i giocatori PC.
Rileva, previene e rimedia alle vulnerabilità di Remote Code Execution (RCE) nei launcher di giochi, nei mod loader, nel software overlay, nei client di chat vocale e nelle periferiche di gioco.
È principalmente utile per gli sviluppatori di tooling e i reverse engineer che lavorano nell'ambito degli strumenti di gioco.
