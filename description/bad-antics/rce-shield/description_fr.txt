Ce projet se concentre sur une boîte à outils de durcissement rCE destinée aux joueurs PC.
Elle détecte, prévient et remédie aux vulnérabilités d'exécution de code à distance (RCE) dans les lanceurs de jeux, les chargeurs de mods, les logiciels d'overlay, les clients de chat vocal et les périphériques de jeu.
Elle est principalement utile aux développeurs d'outils et aux ingénieurs en reverse engineering travaillant dans le domaine des outils de jeu.
