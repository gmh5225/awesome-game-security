Este proyecto se centra en archivos rlib de solana sBPF para la generación de firmas de IDA Pro / Ghidra / Binary Ninja.
Está escrito principalmente en Python y se centra en el desarrollo de plugins, el modding y la generación de SDK.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / bases de datos de firmas de IDA.
