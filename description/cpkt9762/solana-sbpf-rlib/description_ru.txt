Этот проект сосредоточен на файлах solana sBPF rlib для генерации сигнатур IDA Pro / Ghidra / Binary Ninja.
Он в основном написан на Python и посвящён разработке плагинов, моддингу и генерации SDK.
Он особенно полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / баз сигнатур IDA.
