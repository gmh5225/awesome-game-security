Questo progetto si concentra sui file rlib solana sBPF per la generazione di firme di IDA Pro / Ghidra / Binary Ninja.
È scritto principalmente in Python e ruota intorno allo sviluppo di plugin, al modding e alla generazione di SDK.
È soprattutto utile per i ricercatori di sicurezza nei giochi e per i reverse engineer che studiano tecniche offensive nell'ambito di cheat / database di firme IDA.
