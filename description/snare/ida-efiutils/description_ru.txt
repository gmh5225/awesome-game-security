Этот проект — ida-efiutils, плагин IDA Pro для анализа бинарных файлов прошивки UEFI. Он выявляет и аннотирует GUID протоколов EFI, ссылки на таблицы Boot Services и Runtime Services, установки и поиск протоколов, а также точки входа модулей PEI/DXE. Плагин на Python обогащает анализ бинарников UEFI автоматической аннотацией символов и применением типов. Он предназначен для исследователей безопасности прошивок, занимающихся реверс-инжинирингом реализаций UEFI BIOS и вредоносного ПО на уровне загрузки.
