Este proyecto es ida-efiutils, un plugin de IDA Pro para analizar binarios de firmware UEFI. Identifica y anota GUID de protocolos EFI, referencias a las tablas Boot Services y Runtime Services, instalaciones y búsquedas de protocolos, y puntos de entrada de módulos PEI/DXE. El plugin de Python enriquece el análisis de binarios UEFI con anotación automática de símbolos y aplicación de tipos. Está dirigido a investigadores de seguridad de firmware que realizan ingeniería inversa de implementaciones de BIOS UEFI y malware a nivel de arranque.
