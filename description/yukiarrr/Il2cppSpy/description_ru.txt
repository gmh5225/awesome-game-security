Этот проект посвящён дизассемблеру Unity IL2CPP (для APK).
Выберите два APK — и будет показана разница между ними.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования игровых движков: Unity.
