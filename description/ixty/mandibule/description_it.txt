Questo progetto è uno strumento di process injection per Linux che usa ptrace per iniettare ed eseguire binari ELF arbitrari all'interno di un processo target in esecuzione. Il codice in C include un runtime C minimale (icrt) con wrapper di syscall raw, caricamento e relocation ELF, costruzione di uno stack fittizio e passaggio di argomenti shellcode per payload di iniezione position-independent. È principalmente utile per i ricercatori di sicurezza Linux che studiano la process injection basata su ptrace e le tecniche di caricamento ELF a runtime.
