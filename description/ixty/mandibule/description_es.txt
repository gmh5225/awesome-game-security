Este proyecto es una herramienta de inyección de procesos en Linux que usa ptrace para inyectar y ejecutar binarios ELF arbitrarios dentro de un proceso objetivo en ejecución. El código en C incluye un runtime C mínimo (icrt) con wrappers de syscalls en bruto, carga y reubicación ELF, construcción de una pila falsa y paso de argumentos de shellcode para cargas de inyección independientes de la posición. Es principalmente útil para investigadores de seguridad en Linux que estudian la inyección de procesos basada en ptrace y las técnicas de carga ELF en tiempo de ejecución.
