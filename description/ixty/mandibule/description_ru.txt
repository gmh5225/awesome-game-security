Этот проект — инструмент внедрения в процессы Linux, который с помощью ptrace внедряет и выполняет произвольные ELF-бинарники внутри работающего целевого процесса. Кодовая база на C включает минимальный C runtime (icrt) с обёртками сырых системных вызовов, загрузкой и обработкой релокаций ELF, построением поддельного стека и передачей аргументов shellcode для позиционно-независимых payload внедрения. В основном полезен исследователям безопасности Linux, изучающим process injection на базе ptrace и техники загрузки ELF во время выполнения.
