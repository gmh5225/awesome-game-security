Dieses Projekt ist eine Sammlung von Repositories zum Modding von Android-Spielen.
Il2CppDumper YuanShen by kagurazakasanae: Modifizierte Version von Il2CppDumper, mit der Methoden aus UserAssembly.dll des Spiels Genshin Impact gedumpt werden können.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Guides untersuchen.
