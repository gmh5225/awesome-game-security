Этот проект — подборка репозиториев, связанных с моддингом Android-игр.
Il2CppDumper YuanShen by kagurazakasanae: модифицированная версия Il2CppDumper, позволяющая дампить методы UserAssembly.dll игры Genshin Impact.
Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / гайдов.
