Game Lag Reducer is an open-source Windows tool that lowers GPU load and improves frame rates by injecting a native hook DLL into games and simplifying their graphics pipeline at runtime. It pairs a WPF .NET 8 C# launcher (game library, profiles, engine/anti-cheat detection, process injection, and named-pipe IPC) with a C++20 HookDll that patches D3D11 via COM vtable hooks and OpenGL/Vulkan via IAT hooks. Key capabilities include shader capture and caching, runtime shader substitution (flat pixel shaders, no-op compute), disabling tessellation/geometry stages and MSAA, FPS caps, render-scale tweaks, and experimental small-draw culling. It is aimed at players and graphics reverse-engineering enthusiasts who want transparent, user-consented GPU visual reduction rather than anti-cheat bypass.
