Game Lag Reducer는 네이티브 hook DLL을 게임에 주입하고 런타임에 그래픽스 파이프라인을 단순화하여 GPU 부하를 낮추고 프레임 레이트를 향상시키는 오픈소스 Windows 도구입니다. WPF .NET 8 C# 런처(게임 라이브러리, 프로필, 엔진/안티치트 감지, 프로세스 인젝션, 네임드 파이프 IPC)와 COM vtable 훅으로 D3D11을, IAT 훅으로 OpenGL/Vulkan을 패치하는 C++20 HookDll을 함께 사용합니다. 주요 기능으로는 셰이더 캡처 및 캐싱, 런타임 셰이더 치환(플랫 픽셀 셰이더, no-op 컴퓨트), 테셀레이션/지오메트리 단계 및 MSAA 비활성화, FPS 상한, 렌더 스케일 조정, 실험적 소규모 드로우 컬링이 있습니다. 안티치트 우회가 아니라 투명하고 사용자 동의에 기반한 GPU 시각 부하 감소를 원하는 플레이어와 그래픽스 리버스 엔지니어링 애호가를 대상으로 합니다.
