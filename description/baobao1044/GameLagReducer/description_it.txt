Game Lag Reducer è uno strumento Windows open source che riduce il carico GPU e migliora i frame rate iniettando una DLL di hook nativa nei giochi e semplificando la loro pipeline grafica a runtime. Abbina un launcher WPF .NET 8 in C# (libreria giochi, profili, rilevamento motore/anti-cheat, iniezione di processi e IPC via named pipe) a un HookDll C++20 che patcha D3D11 tramite hook COM vtable e OpenGL/Vulkan tramite hook IAT. Le funzionalità principali includono cattura e caching degli shader, sostituzione shader a runtime (pixel shader piatti, compute no-op), disabilitazione degli stadi di tessellation/geometry e MSAA, limiti FPS, regolazioni dello scale di rendering e un culling sperimentale dei piccoli draw. È pensato per giocatori e appassionati di reverse engineering grafico che vogliono una riduzione visiva GPU trasparente e con consenso dell’utente, non un bypass anti-cheat.
