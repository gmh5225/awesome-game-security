Game Lag Reducer 是一款开源 Windows 工具，通过向游戏注入原生 hook DLL，并在运行时简化其图形管线，从而降低 GPU 负载并提升帧率。它由 WPF .NET 8 C# 启动器（游戏库、配置文件、引擎/反作弊检测、进程注入与命名管道 IPC）与 C++20 HookDll（通过 COM vtable hook 修补 D3D11，并通过 IAT hook 处理 OpenGL/Vulkan）组成。主要功能包括着色器捕获与缓存、运行时着色器替换（平面像素着色器、空操作计算着色器）、禁用曲面细分/几何阶段与 MSAA、FPS 上限、渲染缩放调整，以及实验性小绘制剔除。面向希望以透明、经用户同意的方式进行 GPU 视觉减负（而非反作弊绕过）的玩家与图形逆向爱好者。
