Game Lag Reducer 是一款開源 Windows 工具，透過向遊戲注入原生 hook DLL，並在執行階段簡化其圖形管線，從而降低 GPU 負載並提升幀率。它由 WPF .NET 8 C# 啟動器（遊戲庫、設定檔、引擎/反作弊偵測、行程注入與命名管道 IPC）與 C++20 HookDll（透過 COM vtable hook 修補 D3D11，並透過 IAT hook 處理 OpenGL/Vulkan）組成。主要功能包括著色器擷取與快取、執行階段著色器替換（平面像素著色器、空操作計算著色器）、停用曲面細分/幾何階段與 MSAA、FPS 上限、渲染縮放調整，以及實驗性小繪製剔除。面向希望以透明、經使用者同意的方式進行 GPU 視覺減負（而非反作弊繞過）的玩家與圖形逆向愛好者。
