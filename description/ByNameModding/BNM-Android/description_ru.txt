Этот проект — библиотека на C++20 для моддинга Android-игр на Unity IL2CPP по именам классов, методов и полей. Она предоставляет вспомогательные абстракции для доступа во время выполнения, патчинга и интеграции с распространёнными Android-фреймворками перехвата. Репозиторий документирует поддерживаемые версии Unity и включает зависимости, ориентированные на работу с текстом и облегчённое поведение RTTI. Он предназначен для исследователей моддинга мобильных игр и реверс-инжиниринга, которым нужна переиспользуемая основа вместо разовых патчей.
