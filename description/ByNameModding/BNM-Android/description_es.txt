Este proyecto es una biblioteca C++20 para modificar juegos Android Unity IL2CPP mediante nombres de clases, métodos y campos. Proporciona abstracciones auxiliares para acceso en tiempo de ejecución, parches e integración con frameworks de hooking de Android habituales. El repositorio documenta las versiones de Unity compatibles e incluye dependencias centradas en el manejo de texto y un comportamiento RTTI ligero. Está dirigido a investigadores de modding de juegos móviles y de ingeniería inversa que buscan una base reutilizable en lugar de parches puntuales.
