Questo progetto è una libreria C++20 per il modding di giochi Android Unity IL2CPP tramite nomi di classi, metodi e campi. Fornisce astrazioni di supporto per l'accesso a runtime, il patching e l'integrazione con i comuni framework di hooking Android. Il repository documenta le versioni di Unity supportate e include dipendenze orientate alla gestione del testo e a un comportamento RTTI leggero. È rivolto a ricercatori di modding di giochi mobili e di reverse engineering che desiderano una base riutilizzabile invece di patch monouso.
