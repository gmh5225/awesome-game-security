Ce projet est un outil en ligne de commande Objective-C de type class-dump pour les binaires Apple et les images du cache partagé. Il peut lister les dylibs chargés, énumérer les classes Objective-C, extraire les interfaces de classes, afficher les signatures démanglées et tenter des opérations de signature en place. L’outil s’appuie sur dlopen avec des stratégies de gestion d’exceptions pour éviter les effets de bord des constructeurs pendant l’analyse. Il est conçu pour les flux de reverse engineering macOS et iOS où les analystes ont besoin d’une introspection Objective-C rapide assistée par l’exécution.
