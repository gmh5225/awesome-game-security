Questo progetto è uno script Ghidra che importa singoli struct C anche quando dipendono da tipi già definiti. Aggira i limiti del flusso Parse C Source di Ghidra analizzando direttamente le strutture mirate e risolvendo le dipendenze in modo più pratico per il reverse engineering iterativo. L'implementazione è scritta in Java come codice di scripting Ghidra ed è pensata per le versioni moderne di Ghidra. È utile per il reverse engineering e l'analisi di sicurezza nei giochi quando si ricostruiscono i layout dei dati da leak di SDK o header decompilati.
