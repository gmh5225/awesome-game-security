Este proyecto es un script de Ghidra que importa structs C individuales aunque dependan de tipos definidos previamente. Elude las limitaciones del flujo Parse C Source de Ghidra al analizar directamente las estructuras objetivo y resolver las dependencias de forma más práctica para el reverse engineering iterativo. La implementación está escrita en Java como código de scripting de Ghidra y está pensada para versiones modernas de Ghidra. Es útil para reverse engineering y análisis de seguridad en juegos al reconstruir layouts de datos a partir de filtraciones de SDK o cabeceras descompiladas.
