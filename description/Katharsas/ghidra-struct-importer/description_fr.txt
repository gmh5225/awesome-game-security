Ce projet est un script Ghidra qui importe des structs C individuels même lorsqu'ils dépendent de types déjà définis. Il contourne les limites du flux Parse C Source de Ghidra en analysant directement les structures ciblées et en résolvant les dépendances de façon plus pratique pour le reverse engineering itératif. L'implémentation est écrite en Java comme code de scripting Ghidra et vise les versions modernes de Ghidra. Il est utile pour le reverse engineering et l'analyse de sécurité des jeux lors de la reconstruction de layouts de données à partir de fuites SDK ou d'en-têtes décompilés.
