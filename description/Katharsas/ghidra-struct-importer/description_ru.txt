Этот проект — скрипт для Ghidra, который импортирует отдельные C-структуры даже при зависимости от ранее определённых типов. Он обходит ограничения потока Parse C Source в Ghidra, напрямую разбирая нужные структуры и более практично разрешая зависимости для итеративного реверса. Реализация написана на Java как код скриптинга Ghidra и рассчитана на современные версии Ghidra. Полезен для реверс-инжиниринга и анализа безопасности игр при восстановлении раскладок данных по утечкам SDK или декомпилированным заголовкам.
