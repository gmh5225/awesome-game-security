Этот проект — расширение WinDbg с поддержкой ИИ, которое отвечает на вопросы по отладке и автоматически выполняет команды отладчика. Оно поддерживает нескольких поставщиков моделей, постоянный контекст диалога, выполнение команд с пояснениями и помощь, ориентированную на декомпиляцию. Кодовая база на C++ также предоставляет интерфейсы HTTP и в стиле MCP, а также сопутствующий CLI для интеграции внешних инструментов. Решение рассчитано на triage сбоев, отладку эксплойтов и продвинутые рабочие процессы анализа безопасности Windows.
