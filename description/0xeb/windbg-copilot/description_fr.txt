Ce projet est une extension WinDbg assistée par l'IA qui peut répondre aux questions de débogage et exécuter automatiquement des commandes du débogueur. Il prend en charge plusieurs fournisseurs de modèles, un contexte de conversation persistant, l'exécution de commandes avec explications et une assistance orientée décompilation. La base de code C++ expose également des interfaces HTTP et de style MCP, avec une CLI compagnon pour l'intégration d'outils externes. Il est conçu pour le triage de plantages, le débogage d'exploits et les workflows avancés d'analyse de sécurité Windows.
