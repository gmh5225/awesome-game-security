Este proyecto es una extensión de WinDbg asistida por IA que puede responder preguntas de depuración y ejecutar comandos del depurador automáticamente. Admite múltiples proveedores de modelos, contexto de conversación persistente, ejecución de comandos con explicaciones y asistencia orientada a la descompilación. La base de código en C++ también expone interfaces HTTP y de estilo MCP, con una CLI complementaria para la integración de herramientas externas. Está diseñado para el triaje de fallos, la depuración de exploits y flujos de trabajo avanzados de análisis de seguridad en Windows.
