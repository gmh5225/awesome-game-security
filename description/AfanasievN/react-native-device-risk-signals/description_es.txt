React Native Device Risk Signals es un TurboModule de React Native de código abierto que recopila señales brutas de inteligencia del dispositivo y de prevención de fraude en Android e iOS. Reúne resultados tipados de sondas sobre indicadores de root y jailbreak, detección de emuladores, rastros de depurador y Frida, estado de VPN y proxy, así como datos de hardware, configuración regional, aplicación y tiempo de ejecución, sin calcular una puntuación de riesgo en el cliente ni subir resultados. Construido con Kotlin, Objective-C/C++ y TypeScript para la New Architecture, aísla los fallos de las sondas y deja que la app anfitriona controle el consentimiento, los tiempos de espera y qué señales recopilar. El caso de uso principal es enriquecer modelos backend de prevención de fraude, riesgo del dispositivo y seguridad móvil para inicio de sesión, checkout, pagos y otras acciones de alto riesgo.
