React Native Device Risk Signals 是一个开源的 React Native TurboModule，用于在 Android 和 iOS 上采集原始设备情报与防欺诈信号。它收集针对 root 与越狱指标、模拟器检测、调试器与 Frida 痕迹、VPN 与代理状态，以及硬件、区域设置、应用与运行时数据的类型化探测结果，不在客户端计算风险评分，也不上传结果。基于 Kotlin、Objective-C/C++ 和 TypeScript 为 New Architecture 构建，可隔离探测失败，并由宿主应用控制同意、超时以及要采集的信号。主要用例是为登录、结账、支付及其他高风险操作丰富后端防欺诈、设备风险与移动安全模型。
