React Native Device Risk Signals 是一個開源的 React Native TurboModule，用於在 Android 與 iOS 上採集原始裝置情報與防詐欺訊號。它收集針對 root 與越獄指標、模擬器偵測、偵錯器與 Frida 痕跡、VPN 與代理狀態，以及硬體、地區設定、應用程式與執行階段資料的型別化探測結果，不在用戶端計算風險評分，也不上傳結果。以 Kotlin、Objective-C/C++ 與 TypeScript 為 New Architecture 建置，可隔離探測失敗，並由宿主應用程式控制同意、逾時以及要採集的訊號。主要用途是為登入、結帳、付款及其他高風險操作豐富後端防詐欺、裝置風險與行動安全模型。
