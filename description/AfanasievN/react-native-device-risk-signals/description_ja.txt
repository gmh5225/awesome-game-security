React Native Device Risk Signals は、Android と iOS 上で生のデバイス情報と不正防止シグナルを収集するオープンソースの React Native TurboModule です。root および脱獄の指標、エミュレータ検出、デバッガと Frida の痕跡、VPN とプロキシの状態、ハードウェア、ロケール、アプリケーション、ランタイムデータに関する型付きプローブ結果を集め、クライアント側でリスクスコアを計算したり結果をアップロードしたりしません。Kotlin、Objective-C/C++、TypeScript で New Architecture 向けに構築されており、プローブ失敗を分離し、同意、タイムアウト、収集するシグナルをホストアプリが制御できます。主な用途は、ログイン、チェックアウト、決済などの高リスク操作向けに、バックエンドの不正防止・デバイスリスク・モバイルセキュリティモデルを強化することです。
