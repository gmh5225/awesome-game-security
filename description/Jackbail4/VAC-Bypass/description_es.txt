Este proyecto es una prueba de concepto basada en DLL que intenta interferir con las comprobaciones de Valve Anti-Cheat en el contexto del servicio de Steam.
La implementación en C++ usa escaneo de firmas y hooks de Detours para parchear rutinas internas y falsificar respuestas seleccionadas de la API de Windows.
Engancha funciones como VirtualQuery, la enumeración de procesos y módulos, las comprobaciones de depurador y las rutas de lectura de memoria para reducir la visibilidad ante el escáner.
El repositorio está archivado como no funcional y sirve principalmente como material histórico de investigación sobre el bypass de anti-cheat.
