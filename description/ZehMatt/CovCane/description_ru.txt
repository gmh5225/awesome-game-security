Этот проект — экспериментальный фреймворк динамической бинарной инструментации для x86-64-бинарников в Windows. Он написан на C++ и организован из загрузчика, runtime инструментации и отдельной тестовой цели для проверки. В кодовой базе есть компоненты перевода, переписывания, работы с памятью и обработки исключений; используются зависимости Zydis и AsmJIT. В основном полезен для реверс-инжиниринга, анализа во время выполнения и низкоуровневых исследований бинарного кода.
