Ce projet est un framework expérimental d'instrumentation binaire dynamique pour les binaires x86-64 sous Windows. Il est écrit en C++ et organisé autour d'un chargeur, d'un runtime d'instrumentation et d'une cible de test dédiée pour la validation. La base de code comprend des composants pour la traduction, la réécriture, la gestion de la mémoire et le traitement des exceptions, avec des dépendances à Zydis et AsmJIT. Il est surtout utile pour la rétro-ingénierie, l'analyse à l'exécution et la recherche binaire de bas niveau.
