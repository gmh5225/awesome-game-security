Este proyecto es un framework experimental de instrumentación binaria dinámica para binarios x86-64 en Windows. Está escrito en C++ y organizado con un cargador, un runtime de instrumentación y un objetivo de prueba dedicado para la validación. El código incluye componentes de traducción, reescritura, manejo de memoria y procesamiento de excepciones, con dependencias de Zydis y AsmJIT. Es útil principalmente para ingeniería inversa, análisis en tiempo de ejecución e investigación binaria de bajo nivel.
