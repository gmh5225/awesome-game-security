Questo progetto è un framework sperimentale di instrumentazione binaria dinamica per binari x86-64 su Windows. È scritto in C++ e organizzato con un loader, un runtime di instrumentazione e un target di test dedicato per la validazione. Il codice include componenti per traduzione, riscrittura, gestione della memoria ed elaborazione delle eccezioni, con dipendenze da Zydis e AsmJIT. È utile soprattutto per reverse engineering, analisi a runtime e ricerca binaria a basso livello.
