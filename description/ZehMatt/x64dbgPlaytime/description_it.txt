Questo progetto è un plugin per x64dbg che aggiunge il supporto allo scripting Lua al debugger. Combina codice plugin in C/C++ con un runtime Lua incorporato e librerie Lua in bundle per automatizzare i flussi di lavoro del debugger. Il plugin espone funzionalità relative a memoria, registri, breakpoint, label, moduli e operazioni dell'assemblatore, e supporta anche script di autorun ed esempi. È utile per reverse engineer e ricercatori di sicurezza dei giochi che desiderano debugging e analisi programmabili.
