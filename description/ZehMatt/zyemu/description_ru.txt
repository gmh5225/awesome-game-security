Этот проект — находящийся в разработке эмулятор x86-64 в пользовательском режиме, который использует JIT-генерируемые обработчики вместо чисто интерпретируемой модели выполнения. Он написан на современном C++ и опирается на Zydis для декодирования и кодирования инструкций. Репозиторий включает основные компоненты эмулятора: генерацию кода, кэширование кода, логику CPU и памяти, а также playground и тестовые цели. Предназначен для низкоуровневой эмуляции, анализа бинарного кода и продвинутых исследований в области реверс-инжиниринга.
