Questo progetto è un emulatore x86-64 in user-mode ancora in fase di sviluppo, che usa handler generati tramite JIT invece di un modello di esecuzione puramente interpretato. È scritto in C++ moderno e si basa su Zydis per la decodifica e la codifica delle istruzioni. Il repository include componenti core dell'emulatore come generazione del codice, caching del codice, logica di CPU e memoria, oltre a target playground e di test. È pensato per l'emulazione a basso livello, l'analisi binaria e la ricerca avanzata di reverse engineering.
