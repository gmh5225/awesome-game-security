Este proyecto es un emulador x86-64 en modo usuario en desarrollo que usa handlers generados por JIT en lugar de un modelo de ejecución puramente interpretado. Está escrito en C++ moderno y se apoya en Zydis para la decodificación y codificación de instrucciones. El repositorio incluye componentes centrales del emulador como generación de código, caché de código, lógica de CPU y memoria, además de objetivos playground y de pruebas. Está pensado para emulación de bajo nivel, análisis binario e investigación avanzada de ingeniería inversa.
