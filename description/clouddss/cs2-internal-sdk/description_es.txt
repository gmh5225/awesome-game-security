Este proyecto es un SDK interno obtenido por ingeniería inversa para Counter-Strike 2, con definiciones de clases del motor Source 2, punteros de interfaces, offsets de netvars y estructuras del sistema de esquemas. Los encabezados C++ ofrecen representaciones tipadas con precisión del sistema de entidades, clases de jugadores, datos de armas y estructuras de renderizado de CS2, orientadas al desarrollo de cheats internos. Está dirigido a desarrolladores de cheats de CS2 e investigadores de seguridad en videojuegos que necesitan un SDK de Source 2 preciso para hooks y acceso a memoria.
