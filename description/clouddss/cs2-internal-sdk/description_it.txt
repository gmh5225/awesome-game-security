Questo progetto è un SDK interno reverse-engineered per Counter-Strike 2, con definizioni di classi del motore Source 2, puntatori alle interfacce, offset dei netvar e strutture del sistema di schema. Gli header C++ forniscono rappresentazioni tipizzate con precisione del sistema di entità, delle classi giocatore, dei dati delle armi e delle strutture di rendering di CS2, per lo sviluppo di cheat interni. È rivolto a sviluppatori di cheat per CS2 e ricercatori di sicurezza nei videogiochi che necessitano di un SDK Source 2 accurato per hooking e accesso alla memoria.
