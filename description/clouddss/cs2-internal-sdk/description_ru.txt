Этот проект — реверс-инжиниренный внутренний SDK для Counter-Strike 2, содержащий определения классов движка Source 2, указатели интерфейсов, смещения netvar и структуры системы схем. Заголовки C++ дают типоточночные представления системы сущностей CS2, классов игроков, данных оружия и структур рендеринга для разработки внутренних читов. Он рассчитан на разработчиков читов для CS2 и исследователей безопасности игр, которым нужен точный SDK Source 2 для хуков и доступа к памяти.
