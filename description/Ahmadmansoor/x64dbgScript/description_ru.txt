Этот проект предоставляет ресурсы и примеры скриптов для расширения рабочих процессов отладчика в x64dbg. В него входят образцы скриптов, а также вспомогательный код на Python для экспорта меток функций и комментариев из IDA в сессии отладчика. Примеры демонстрируют практические шаблоны автоматизации: многошаговое управление выполнением, использование глобальных переменных и поиск последовательностей инструкций. Проект в первую очередь полезен специалистам по реверс-инжинирингу, которые хотят ускорить повторяющиеся задачи анализа в исследованиях безопасности игр и бинарных файлов.
