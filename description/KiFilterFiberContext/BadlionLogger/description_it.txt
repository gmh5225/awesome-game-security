Questo progetto è un logger kernel proof-of-concept per osservare a runtime un driver anti-cheat di un gioco. Applica hook IAT durante i callback di caricamento immagine per monitorare il comportamento in modalità black-box, in particolare contro un modulo target virtualizzato con VMProtect. L’implementazione è principalmente in C++ e si concentra sulla gestione dei callback kernel e sull’instrumentazione piuttosto che sulla robustezza di produzione. È pensato per la ricerca sull’anti-cheat e per lo studio didattico delle tecniche di monitoraggio a livello di driver.
