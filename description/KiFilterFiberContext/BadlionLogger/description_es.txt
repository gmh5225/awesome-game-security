Este proyecto es un registrador de kernel de prueba de concepto para observar en tiempo de ejecución un controlador de anti-trampas de un juego. Aplica hooks IAT durante las devoluciones de llamada de carga de imagen para supervisar el comportamiento de forma de caja negra, en concreto frente a un módulo objetivo virtualizado con VMProtect. La implementación es principalmente en C++ y se centra en el manejo de callbacks del kernel y la instrumentación, más que en la robustez de producción. Está pensado para la investigación de anti-trampas y el estudio educativo de técnicas de monitorización a nivel de controlador.
