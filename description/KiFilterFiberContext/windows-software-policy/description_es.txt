Este proyecto documenta y analiza la ruta de licencias del kernel de Windows expuesta mediante la clase de consulta SystemPolicyInformation.
Explica cómo los componentes de licencia en modo usuario se comunican con el controlador de directivas del kernel y dónde se gestionan los datos de directiva.
El repositorio incluye código fuente y encabezados en C para experimentación de bajo nivel, además de un script auxiliar en Python para el procesamiento binario relacionado.
Los lenguajes principales son C y Python.
Resulta especialmente útil para el estudio de los internos de Windows, la ingeniería inversa y la protección de software.
