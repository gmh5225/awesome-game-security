Этот проект документирует и анализирует путь лицензирования ядра Windows, доступный через класс запроса SystemPolicyInformation.
В нём объясняется, как компоненты лицензирования в пользовательском режиме взаимодействуют с драйвером политик ядра и где обрабатываются данные политик.
Репозиторий включает исходники и заголовки на C для низкоуровневых экспериментов, а также вспомогательный скрипт на Python для связанной обработки бинарников.
Основные языки — C и Python.
Он в первую очередь полезен для исследований внутренних механизмов Windows, реверс-инжиниринга и программной защиты.
