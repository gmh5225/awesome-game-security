Questo progetto documenta e analizza il percorso di licensing del kernel Windows esposto tramite la classe di query SystemPolicyInformation.
Spiega come i componenti di licensing in user-mode comunicano con il driver delle policy del kernel e dove vengono gestiti i dati di policy.
Il repository include sorgenti e header C per sperimentazione a basso livello, oltre a uno script helper Python per l’elaborazione binaria correlata.
I linguaggi principali sono C e Python.
È soprattutto utile per la ricerca sugli interni di Windows, il reverse engineering e la protezione del software.
