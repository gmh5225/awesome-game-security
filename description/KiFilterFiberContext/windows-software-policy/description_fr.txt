Ce projet documente et analyse le chemin de licence du noyau Windows exposé via la classe de requête SystemPolicyInformation.
Il explique comment les composants de licence en mode utilisateur communiquent avec le pilote de stratégie du noyau et où les données de stratégie sont traitées.
Le dépôt inclut du code source et des en-têtes C pour l’expérimentation bas niveau, ainsi qu’un script d’aide Python pour le traitement binaire associé.
Les langages principaux sont C et Python.
Il est surtout utile pour la recherche sur les internes Windows, la rétro-ingénierie et la protection logicielle.
