Этот проект сосредоточен на графовом анализе.
Он реализует распространённые задачи статического анализа, включая анализ графа вызовов, и может интегрировать информацию о покрытии блоков.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области плагинов для читов / Binary Ninja.
