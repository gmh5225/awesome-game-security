Este proyecto es un plugin de IDA para binarios Apple AARCH64/ARM64 que utiliza LLM para traducir funciones en ensamblador a pseudocódigo legible en C, Objective-C o Swift.
El plugin está diseñado específicamente para analizar binarios Apple Mach-O, incluidas aplicaciones iOS, kernelcaches y bibliotecas dyld_shared_cache.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
