Questo progetto è un plugin IDA per binari Apple AARCH64/ARM64 che utilizza LLM per tradurre le funzioni in assembly in pseudo-codice leggibile in C, Objective-C o Swift.
Il plugin è progettato appositamente per analizzare binari Apple Mach-O, incluse app iOS, kernelcache e librerie dyld_shared_cache.
È soprattutto utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell’ambito di cheat / plugin IDA.
