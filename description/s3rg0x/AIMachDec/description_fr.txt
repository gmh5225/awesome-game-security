Ce projet est un plugin IDA pour les binaires Apple AARCH64/ARM64 qui utilise des LLM pour traduire les fonctions assembleur en pseudo-code lisible en C, Objective-C ou Swift.
Ce plugin est conçu spécifiquement pour analyser les binaires Apple Mach-O, notamment les applications iOS, les kernelcaches et les bibliothèques dyld_shared_cache.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
