Этот проект — плагин IDA для бинарников Apple AARCH64/ARM64, который с помощью LLM переводит функции на ассемблере в читаемый псевдокод на C, Objective-C или Swift.
Плагин специально предназначен для анализа бинарников Apple Mach-O, включая iOS-приложения, kernelcache и библиотеки dyld_shared_cache.
Он в первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / плагинов IDA.
