D3DOverlay-Nvidia-Hijack è un framework di overlay Direct3D9 che esegue il rendering di contenuti ImGui tramite la finestra NVIDIA GeForce Overlay. L'implementazione C++ individua la finestra di overlay esistente, configura un comportamento trasparente click-through e gestisce un callback di rendering per frame. Fornisce inoltre primitive di disegno di supporto per testo, rettangoli e cerchi sulla superficie di visualizzazione di destinazione. Il progetto è usato principalmente in scenari di tooling esterno per giochi che richiedono un livello di overlay riutilizzabile con una gestione personalizzata delle finestre minima.
