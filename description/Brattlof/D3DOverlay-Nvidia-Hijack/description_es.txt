D3DOverlay-Nvidia-Hijack es un framework de overlay Direct3D9 que renderiza contenido ImGui a través de la ventana NVIDIA GeForce Overlay. La implementación en C++ localiza la ventana de overlay existente, configura un comportamiento transparente de click-through y gestiona un callback de renderizado por frame. También ofrece primitivas de dibujo auxiliares para texto, rectángulos y círculos sobre la superficie de visualización de destino. El proyecto se usa principalmente en herramientas externas para juegos que necesitan una capa de overlay reutilizable con una gestión de ventanas personalizada mínima.
