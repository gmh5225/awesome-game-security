D3DOverlay-Nvidia-Hijack — это фреймворк оверлея на Direct3D9, который отображает содержимое ImGui через окно NVIDIA GeForce Overlay. Реализация на C++ находит существующее окно оверлея, настраивает прозрачное поведение с пропуском кликов и управляет покадровым callback отрисовки. Также предоставляются вспомогательные примитивы рисования для текста, прямоугольников и окружностей поверх целевой поверхности отображения. Проект в основном применяется во внешних игровых инструментах, где нужен переиспользуемый слой оверлея с минимальным собственным управлением окнами.
