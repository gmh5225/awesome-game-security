D3DOverlay-Nvidia-Hijack est un framework d'overlay Direct3D9 qui affiche du contenu ImGui via la fenêtre NVIDIA GeForce Overlay. L'implémentation C++ localise la fenêtre d'overlay existante, configure un comportement transparent et clique-traversant, et pilote un callback de rendu par frame. Elle fournit également des primitives de dessin utilitaires pour le texte, les rectangles et les cercles sur la surface d'affichage cible. Le projet est principalement utilisé dans des outils de jeu externes nécessitant une couche d'overlay réutilisable avec une gestion de fenêtre personnalisée minimale.
