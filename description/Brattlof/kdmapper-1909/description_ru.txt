kdmapper-1909 — это маппер драйверов ядра Windows, адаптированный под сборки 1809, 1903 и 1909. Код на C++ загружает уязвимый интерфейс драйвера Intel и использует его для привилегированных операций с памятью ядра. В проекте есть процедуры ручного маппинга, вспомогательные средства разбора PE и логика управления службами для развёртывания неподписанных драйверов из пользовательского режима. Проект в основном применяется для исследований безопасности ядра Windows, низкоуровневых экспериментов с драйверами и изучения обхода античитов.
