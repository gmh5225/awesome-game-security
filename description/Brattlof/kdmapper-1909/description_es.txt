kdmapper-1909 es un mapeador de controladores del kernel de Windows adaptado a las versiones de compilación 1809, 1903 y 1909. El código C++ carga una interfaz de controlador Intel vulnerable y la usa para realizar operaciones privilegiadas de memoria en el kernel. Incluye rutinas de mapeo manual, auxiliares de análisis PE y lógica de gestión de servicios para desplegar controladores sin firma desde modo usuario. El proyecto se usa principalmente en investigación de seguridad del kernel de Windows, experimentación de bajo nivel con controladores y estudios de bypass de anti-cheat.
