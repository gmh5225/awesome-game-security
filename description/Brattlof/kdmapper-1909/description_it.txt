kdmapper-1909 è un mapper di driver del kernel Windows pensato per le build 1809, 1903 e 1909. Il codice C++ carica un’interfaccia di driver Intel vulnerabile e la usa per eseguire operazioni privilegiate sulla memoria del kernel. Include routine di mapping manuale, helper per il parsing PE e logica di gestione dei servizi per distribuire driver non firmati da user mode. Il progetto è usato principalmente per la ricerca sulla sicurezza del kernel Windows, sperimentazione di driver a basso livello e studi sul bypass degli anti-cheat.
