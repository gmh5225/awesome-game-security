Questo progetto è un template di mod menu per giochi Android basato su un approccio semi-JNI che combina un’interfaccia overlay in Java con l’hooking di gioco in C++ nativo. Fornisce un menu flottante renderizzato tramite il WindowManager di Android con toggle e slider, mentre la libreria nativa gestisce patching della memoria, hooking di funzioni e modifica dei valori di gioco tramite callback JNI. È pensato per i modder di giochi Android che creano menu cheat che uniscono UI Java e manipolazione del codice nativo.
