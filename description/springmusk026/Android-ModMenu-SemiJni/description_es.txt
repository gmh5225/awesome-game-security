Este proyecto es una plantilla de menú de mods para juegos Android con un enfoque semi-JNI que combina una interfaz de superposición en Java con el hooking de juegos en C++ nativo. Ofrece un menú flotante renderizado mediante el WindowManager de Android con interruptores y deslizadores, mientras que la biblioteca nativa gestiona el parcheo de memoria, el hooking de funciones y la modificación de valores del juego mediante callbacks JNI. Está orientado a modders de juegos Android que crean menús de trampas que combinan UI en Java con manipulación de código nativo.
