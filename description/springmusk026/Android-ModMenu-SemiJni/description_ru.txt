Этот проект — шаблон мод-меню для Android-игр с полу-JNI подходом, сочетающим Java-оверлейный интерфейс и нативный C++-хукинг игры. Он предоставляет плавающее меню через WindowManager Android с переключателями и ползунками, а нативная библиотека через JNI-колбэки выполняет патчинг памяти, хукинг функций и изменение игровых значений. Предназначен для моддеров Android-игр, создающих чит-меню с Java UI и нативной манипуляцией кодом.
