Ce projet est un modèle de menu de mod pour jeux Android utilisant une approche semi-JNI qui combine une interface d’overlay Java avec le hooking de jeu en C++ natif. Il fournit un menu flottant rendu via le WindowManager d’Android avec des interrupteurs et des curseurs, tandis que la bibliothèque native gère le patch mémoire, le hooking de fonctions et la modification des valeurs de jeu via des callbacks JNI. Il s’adresse aux moddeurs de jeux Android qui construisent des menus de triche alliant UI Java et manipulation de code natif.
