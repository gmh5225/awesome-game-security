Questo progetto è un modello di menu mod per Android che renderizza un'interfaccia Dear ImGui nei giochi Unity. Si aggancia alla pipeline di rendering di Unity su Android per disegnare un overlay ImGui con controlli cheat, usando l'hooking di metodi IL2CPP o Mono per modificare la logica di gioco. La libreria nativa C++ gestisce l'integrazione del rendering OpenGL ES. È rivolto ai modder di giochi Unity su Android che costruiscono interfacce cheat basate su ImGui.
