Este proyecto es una plantilla de menú de mods para Android que renderiza una interfaz Dear ImGui dentro de juegos Unity. Se engancha al pipeline de renderizado de Unity en Android para dibujar una superposición ImGui con controles de trucos, usando enganche de métodos IL2CPP o Mono para modificar la lógica del juego. La biblioteca nativa en C++ gestiona la integración del renderizado OpenGL ES. Está pensado para modders de juegos Unity en Android que crean interfaces de trucos basadas en ImGui.
