Этот проект — шаблон мод-меню для Android, который отображает интерфейс Dear ImGui в играх на Unity. Он перехватывает конвейер рендеринга Unity на Android, рисуя оверлей ImGui с элементами управления читами, и использует хуки методов IL2CPP или Mono для изменения игровой логики. Нативная библиотека на C++ отвечает за интеграцию рендеринга OpenGL ES. Предназначен для моддеров Android-игр на Unity, создающих чит-интерфейсы на базе ImGui.
