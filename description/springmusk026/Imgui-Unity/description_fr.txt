Ce projet est un modèle de menu de mod Android qui affiche une interface Dear ImGui dans les jeux Unity. Il s'accroche au pipeline de rendu Unity sur Android pour dessiner une superposition ImGui avec des contrôles de triche, en utilisant le hooking de méthodes IL2CPP ou Mono pour modifier la logique du jeu. La bibliothèque native C++ gère l'intégration du rendu OpenGL ES. Il s'adresse aux moddeurs de jeux Unity Android qui construisent des interfaces de triche basées sur ImGui.
