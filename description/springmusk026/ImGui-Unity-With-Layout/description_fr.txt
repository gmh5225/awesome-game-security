Ce projet est un modèle amélioré de menu de mod Android pour Unity, utilisant Dear ImGui avec des configurations de disposition préconstruites. Il étend l'intégration ImGui-Unity de base avec des onglets de menu organisés, des options de triche catégorisées, des widgets stylisés et une fonctionnalité de sauvegarde/chargement des paramètres. La bibliothèque C++ offre une expérience de menu de mod soignée dans les jeux Unity sur Android. Elle s'adresse aux moddeurs de jeux Android souhaitant un modèle de menu ImGui complet pour les titres Unity.
