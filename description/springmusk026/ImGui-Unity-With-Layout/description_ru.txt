Этот проект — улучшенный шаблон мод-меню для Android Unity на базе Dear ImGui с готовыми конфигурациями макета. Он расширяет базовую интеграцию ImGui-Unity организованными вкладками меню, категоризированными чит-опциями, стилизованными виджетами и функцией сохранения/загрузки настроек. Библиотека на C++ обеспечивает отточенный опыт мод-меню в играх Unity на Android. Предназначена для моддеров Android-игр, которым нужен полноценный шаблон ImGui мод-меню для Unity-проектов.
