Questo progetto è un modello avanzato di menu mod Android per Unity che usa Dear ImGui con configurazioni di layout predefinite. Estende l'integrazione di base ImGui-Unity con schede di menu organizzate, opzioni cheat categorizzate, widget stilizzati e funzionalità di salvataggio/caricamento delle impostazioni. La libreria C++ offre un'esperienza di menu mod curata nei giochi Unity su Android. È pensata per i modder di giochi Android che cercano un modello completo di menu ImGui per titoli Unity.
