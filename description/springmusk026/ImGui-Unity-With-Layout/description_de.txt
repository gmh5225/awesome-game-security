Dieses Projekt ist eine erweiterte Android-Unity-Mod-Menü-Vorlage mit Dear ImGui und vorkonfigurierten Layout-Einstellungen. Es baut auf der grundlegenden ImGui-Unity-Integration auf und ergänzt organisierte Menü-Tabs, kategorisierte Cheat-Optionen, gestylte Widgets sowie Speichern-/Laden-Funktionen für Einstellungen. Die C++-Bibliothek bietet ein ausgefeiltes Mod-Menü-Erlebnis in Unity-Spielen auf Android. Sie richtet sich an Android-Spielmodder, die eine funktionsreiche ImGui-Mod-Menü-Vorlage für Unity-Titel suchen.
