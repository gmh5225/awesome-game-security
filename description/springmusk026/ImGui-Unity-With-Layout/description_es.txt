Este proyecto es una plantilla mejorada de menú de mods para Unity en Android que usa Dear ImGui con configuraciones de diseño prediseñadas. Amplía la integración básica ImGui-Unity con pestañas de menú organizadas, opciones de trucos categorizadas, widgets con estilo y funcionalidad de guardar/cargar ajustes. La biblioteca C++ ofrece una experiencia de menú de mods pulida dentro de juegos Unity en Android. Está dirigida a modders de juegos Android que buscan una plantilla completa de menú ImGui para títulos Unity.
