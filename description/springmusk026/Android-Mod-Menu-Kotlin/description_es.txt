Este proyecto es una plantilla de menú de mods para juegos Android escrita en Kotlin, con soporte de hooking nativo en C++. Ofrece un menú flotante superpuesto construido con vistas de Android, inyección de biblioteca nativa para parcheo de memoria y hooking de funciones, y código puente JNI para la comunicación entre la UI en Kotlin y las modificaciones nativas del juego. Está orientado a modders de juegos Android que prefieren Kotlin para la UI y C++ para la manipulación de bajo nivel del juego.
