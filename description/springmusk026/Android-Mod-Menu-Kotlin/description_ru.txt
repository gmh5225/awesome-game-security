Этот проект — шаблон меню модов для Android-игр на Kotlin с поддержкой нативного хукинга на C++. Он предоставляет плавающее overlay-меню на Android Views, внедрение нативной библиотеки для патчинга памяти и хукинга функций, а также JNI-мост для связи между UI на Kotlin и нативными модификациями игры. Предназначен для моддеров Android-игр, которые предпочитают Kotlin для UI и C++ для низкоуровневой работы с игрой.
