Ce projet est un modèle de menu de mod pour jeux Android écrit en Kotlin, avec prise en charge du hooking natif en C++. Il fournit un menu flottant en superposition basé sur les vues Android, l’injection d’une bibliothèque native pour le patch mémoire et le hook de fonctions, ainsi qu’un pont JNI pour la communication entre l’interface Kotlin et les modifications natives du jeu. Il s’adresse aux moddeurs de jeux Android qui préfèrent Kotlin pour l’UI tout en utilisant le C++ pour la manipulation bas niveau du jeu.
