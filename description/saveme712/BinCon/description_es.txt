Este proyecto se centra en vEH + PAGE_NOACCESS.
El proyecto de ejemplo proporciona un ejemplo de una aplicación de consola simple endurecida contra análisis de memoria, modificaciones de memoria, depuradores y otras amenazas.
Es principalmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / protección de páginas.
