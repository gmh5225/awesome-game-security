IDA NO MCP esporta i risultati di decompilazione e analisi di IDA Pro come normali file sorgente, così che gli strumenti di coding IA possano fare reverse engineering di binari senza un ponte MCP. È un progetto a doppio percorso: un plugin Python per IDA per esportazioni interattive via GUI e un binario CLI in Rust (inp) basato su idalib per binari grandi, lavori batch e per saltare l’auto-analisi bloccata. Le esportazioni includono C decompilato (o disassembly di fallback), metadati caller/callee, stringhe, import/export, hexdump di memoria opzionali, grafi delle chiamate e una guida AGENTS.md per IDE IA. Le modalità passano da file per funzione a un layout consolidato per database con decine di migliaia di funzioni. Si rivolge ai reverse engineer che vogliono un’analisi binaria assistita dall’IA rapida e a basso attrito in strumenti come Cursor o Claude Code.
