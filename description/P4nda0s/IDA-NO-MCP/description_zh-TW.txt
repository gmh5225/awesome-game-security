IDA NO MCP 將 IDA Pro 的反編譯與分析結果匯出為普通原始碼檔案，讓 AI 程式設計工具無需 MCP 橋接即可逆向分析二進位檔。專案採用雙軌架構：用於互動式 GUI 匯出的 Python IDA 外掛，以及基於 idalib、面向大型二進位檔、批次作業與跳過卡住自動分析的 Rust CLI 工具（inp）。匯出內容包含反編譯的 C（或反組譯後備）、呼叫端/被呼叫端中繼資料、字串、匯入/匯出、可選記憶體十六進位轉儲、呼叫圖，以及面向 AI IDE 的 AGENTS.md 指南。支援在依函式分檔與適用於數萬函式資料庫的合併版面之間切換。面向希望在 Cursor 或 Claude Code 等工具中快速、低摩擦地進行 AI 輔助二進位分析的逆向工程師。
