IDA NO MCP 将 IDA Pro 的反编译与分析结果导出为普通源文件，使 AI 编程工具无需 MCP 桥接即可逆向分析二进制。项目采用双轨架构：用于交互式 GUI 导出的 Python IDA 插件，以及基于 idalib、面向大型二进制、批处理任务和跳过卡住自动分析的 Rust CLI 工具（inp）。导出内容包括反编译的 C（或反汇编回退）、调用方/被调用方元数据、字符串、导入/导出、可选内存十六进制转储、调用图，以及面向 AI IDE 的 AGENTS.md 指南。支持在按函数分文件与适用于数万函数数据库的合并布局之间切换。面向希望在 Cursor 或 Claude Code 等工具中快速、低摩擦地开展 AI 辅助二进制分析的逆向工程师。
