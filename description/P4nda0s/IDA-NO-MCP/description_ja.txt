IDA NO MCP は、IDA Pro のデコンパイルと解析結果を通常のソースファイルとして書き出し、AI コーディングツールが MCP ブリッジなしでバイナリをリバースエンジニアリングできるようにします。対話的な GUI エクスポート用の Python IDA プラグインと、idalib を用いた大規模バイナリ・バッチ処理・固まった自動解析のスキップ向け Rust CLI バイナリ（inp）という二系統で提供されます。出力にはデコンパイルされた C（またはディスアセンブリへのフォールバック）、caller/callee メタデータ、文字列、インポート/エクスポート、任意のメモリ hexdump、コールグラフ、AI IDE 向け AGENTS.md ガイドが含まれます。関数ごとのファイルと、数万関数のデータベース向け統合レイアウトを切り替えられます。Cursor や Claude Code などで高速かつ低摩擦な AI 支援バイナリ解析を求めるリバースエンジニア向けです。
