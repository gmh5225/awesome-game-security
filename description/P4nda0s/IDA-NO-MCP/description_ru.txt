IDA NO MCP экспортирует результаты декомпиляции и анализа IDA Pro в обычные исходные файлы, чтобы ИИ-инструменты для программирования могли проводить реверс-инжиниринг бинарников без MCP-моста. Проект двухканальный: Python-плагин IDA для интерактивного GUI-экспорта и Rust CLI-бинарник (inp) на idalib для крупных бинарников, пакетных заданий и пропуска зависшего автоанализа. Экспорт включает декомпилированный C (или запасной дизассемблер), метаданные caller/callee, строки, импорты/экспорты, опциональные hexdump памяти, графы вызовов и руководство AGENTS.md для ИИ-IDE. Режимы переключаются между файлами по функциям и сводной раскладкой для баз с десятками тысяч функций. Целевая аудитория — reverse-инженеры, которым нужен быстрый и удобный ИИ-анализ бинарников в инструментах вроде Cursor или Claude Code.
