IDA NO MCP exportiert IDA-Pro-Dekompilierungs- und Analyseergebnisse als gewöhnliche Quelldateien, damit KI-Coding-Tools Binärdateien ohne MCP-Brücke reverse-engineern können. Es ist ein zweigleisiges Projekt: ein Python-IDA-Plugin für interaktive GUI-Exporte und ein Rust-CLI-Binary (inp) auf Basis von idalib für große Binärdateien, Batch-Jobs und das Überspringen hängengebliebener Auto-Analyse. Exporte umfassen dekompiliertes C (oder Disassembly als Fallback), Caller-/Callee-Metadaten, Strings, Imports/Exports, optionale Speicher-Hexdumps, Call-Graphen und einen AGENTS.md-Leitfaden für KI-IDEs. Modi wechseln zwischen datei-pro-Funktion und einem konsolidierten Layout für Datenbanken mit Zehntausenden Funktionen. Zielgruppe sind Reverse Engineers, die schnelle, reibungsarme KI-gestützte Binäranalyse in Tools wie Cursor oder Claude Code wollen.
