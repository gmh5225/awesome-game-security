IDA NO MCP exporta los resultados de descompilación y análisis de IDA Pro como archivos fuente normales, para que las herramientas de codificación con IA puedan aplicar ingeniería inversa a binarios sin un puente MCP. Se ofrece como un proyecto de doble vía: un complemento Python de IDA para exportaciones interactivas por GUI y un binario CLI en Rust (inp) impulsado por idalib para binarios grandes, trabajos por lotes y omitir el autoanálisis colgado. Las exportaciones incluyen C descompilado (o desensamblado como respaldo), metadatos de llamador/llamado, cadenas, imports/exports, volcados hexadecimales de memoria opcionales, grafos de llamadas y una guía AGENTS.md para IDE con IA. Los modos alternan entre archivos por función y un diseño consolidado para bases de datos con decenas de miles de funciones. Está pensado para ingenieros inversos que quieren un análisis binario asistido por IA rápido y con poca fricción en herramientas como Cursor o Claude Code.
