IDA NO MCP exporte les résultats de décompilation et d’analyse d’IDA Pro sous forme de fichiers source ordinaires, afin que les outils de codage IA puissent rétroconcevoir des binaires sans pont MCP. Le projet est dual : un plugin Python IDA pour les exports interactifs via l’interface graphique, et un binaire CLI Rust (inp) basé sur idalib pour les gros binaires, les traitements par lots et le contournement d’une auto-analyse bloquée. Les exports incluent du C décompilé (ou un repli en désassemblage), les métadonnées appelant/appelé, les chaînes, les imports/exports, des hexdumps mémoire optionnels, des graphes d’appels, et un guide AGENTS.md pour les IDE IA. Les modes basculent entre des fichiers par fonction et une disposition consolidée pour les bases de données à des dizaines de milliers de fonctions. Il s’adresse aux reverse engineers qui veulent une analyse binaire assistée par IA rapide et sans friction dans des outils comme Cursor ou Claude Code.
