IDA NO MCP는 IDA Pro 디컴파일 및 분석 결과를 일반 소스 파일로 내보내어, AI 코딩 도구가 MCP 브리지 없이 바이너리를 리버스 엔지니어링할 수 있게 합니다. 대화형 GUI 내보내기용 Python IDA 플러그인과, idalib 기반의 대규모 바이너리·배치 작업·멈춘 자동 분석 건너뛰기용 Rust CLI 바이너리(inp)라는 이중 구조로 제공됩니다. 내보내기에는 디컴파일된 C(또는 디스어셈블리 폴백), caller/callee 메타데이터, 문자열, import/export, 선택적 메모리 헥스덤프, 호출 그래프, AI IDE용 AGENTS.md 가이드가 포함됩니다. 함수별 파일 모드와 수만 개 함수 데이터베이스용 통합 레이아웃을 전환할 수 있습니다. Cursor나 Claude Code 같은 도구에서 빠르고 부담 없는 AI 보조 바이너리 분석을 원하는 리버스 엔지니어를 대상으로 합니다.
