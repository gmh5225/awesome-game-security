IDA NO MCP exports IDA Pro decompilation and analysis results as plain source files so AI coding tools can reverse-engineer binaries without an MCP bridge. It ships as a dual-track project: a Python IDA plugin for interactive GUI exports and a Rust CLI binary (inp) powered by idalib for large binaries, batch jobs, and skipping hung auto-analysis. Exports include decompiled C (or disassembly fallback), caller/callee metadata, strings, imports/exports, optional memory hexdumps, call graphs, and an AGENTS.md guide for AI IDEs. Modes switch between per-function files and a consolidated layout for databases with tens of thousands of functions. It targets reverse engineers who want fast, low-friction AI-assisted binary analysis in tools like Cursor or Claude Code.
