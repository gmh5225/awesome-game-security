Este proyecto es una lista curada de extensiones de WinDbg y recursos de depuración relacionados.
Cataloga herramientas para análisis de kernel, respuesta a incidentes, caza de rootkits, inspección de memoria y automatización del flujo de trabajo del depurador.
El repositorio se centra en Markdown y sirve como índice de referencia, no como una implementación de software independiente.
Es útil para ingenieros de ingeniería inversa y analistas de seguridad de juegos que dependen de plugins de WinDbg para la investigación de Windows a bajo nivel.
