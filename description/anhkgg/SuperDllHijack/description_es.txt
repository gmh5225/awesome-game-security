Este proyecto es un ayudante genérico de secuestro de DLL para Windows que simplifica la interceptación al estilo proxy-DLL.
Proporciona código C/C++ y ejemplos para reenviar exportaciones sin recrear manualmente cada firma de función original.
El enfoque usa un módulo original renombrado y una DLL de reemplazo que invoca una rutina auxiliar en DllMain para dejar pasar el comportamiento.
Es útil para experimentar con cargadores, investigación de estilo red team y el estudio del abuso de carga de módulos en entornos de seguridad de juegos.
