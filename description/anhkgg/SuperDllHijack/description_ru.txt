Этот проект — универсальный помощник для DLL hijacking в Windows, упрощающий перехват в стиле proxy-DLL.
Он предоставляет код на C/C++ и примеры для пересылки экспортов без ручного воссоздания каждой исходной сигнатуры функции.
Подход использует переименованный исходный модуль и подменную DLL, которая в DllMain вызывает вспомогательную процедуру для пропуска поведения.
Полезен для экспериментов с загрузчиком, исследований в стиле red team и изучения злоупотреблений загрузкой модулей в среде игровой безопасности.
