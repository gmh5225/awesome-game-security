Ce projet est un outil générique d'aide au détournement de DLL sous Windows, qui simplifie l'interception de type proxy-DLL.
Il fournit du code C/C++ et des exemples pour transmettre les exportations sans recréer manuellement chaque signature de fonction d'origine.
L'approche utilise un module d'origine renommé et une DLL de remplacement qui appelle une routine d'aide dans DllMain pour faire passer le comportement.
Il est utile pour l'expérimentation de chargeurs, la recherche de type red team et l'étude des abus de chargement de modules dans les environnements de sécurité des jeux.
