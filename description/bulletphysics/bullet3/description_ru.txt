Этот проект — Bullet Physics SDK, открытая библиотека на C++ для обнаружения столкновений в реальном времени и мультифизического моделирования. Она обеспечивает динамику твёрдых тел, динамику мягких тел, симуляцию транспорта, контроллеры персонажей и решение ограничений для игр, робототехники, визуальных эффектов и научных вычислений. Библиотека включает GPU-ускоренную broadphase через OpenCL и интегрируется с основными игровыми движками и средами моделирования. Она предназначена для разработчиков игр, исследователей робототехники и инженеров VFX, реализующих физическое моделирование.
