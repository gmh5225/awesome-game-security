Este proyecto es una biblioteca Rust para interactuar con un controlador virtual de Logitech y emular eventos de entrada. Ofrece APIs tipadas para movimiento del ratón, clics, acciones de la rueda, pulsaciones de teclado, entrada de varias teclas y rutinas de escritura de texto. La implementación usa interfaces nativas de Windows y llamadas de control de E/S de dispositivo para comunicarse con el handle del controlador subyacente. El caso de uso principal es la investigación de emulación de entrada, experimentos de automatización y el análisis de la superficie de ataque de controladores vulnerables en contextos de seguridad de juegos.
