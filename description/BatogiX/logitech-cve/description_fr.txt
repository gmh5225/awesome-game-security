Ce projet est une bibliothèque Rust permettant d'interagir avec un pilote virtuel Logitech pour émuler des événements d'entrée. Elle fournit des API typées pour les mouvements de souris, les clics, les actions de molette, les frappes clavier, la saisie multi-touches et les routines de frappe de texte. L'implémentation utilise les interfaces natives Windows et des appels de contrôle d'E/S de périphérique pour communiquer avec le handle du pilote sous-jacent. Le cas d'usage principal est la recherche sur l'émulation d'entrée, les expériences d'automatisation et l'analyse de la surface d'attaque des pilotes vulnérables dans le contexte de la sécurité des jeux.
