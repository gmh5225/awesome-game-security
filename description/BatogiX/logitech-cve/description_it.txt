Questo progetto è una libreria Rust per interagire con un driver virtuale Logitech ed emulare eventi di input. Fornisce API tipizzate per movimento del mouse, clic, azioni della rotella, pressioni di tasti, input multi-tasto e routine di digitazione del testo. L'implementazione usa interfacce native di Windows e chiamate di controllo I/O del dispositivo per comunicare con l'handle del driver sottostante. Il caso d'uso principale è la ricerca sull'emulazione dell'input, esperimenti di automazione e l'analisi della superficie di attacco di driver vulnerabili nei contesti di sicurezza dei giochi.
