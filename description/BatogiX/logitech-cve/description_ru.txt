Этот проект — библиотека на Rust для взаимодействия с виртуальным драйвером Logitech и эмуляции событий ввода. Она предоставляет типизированные API для перемещения мыши, кликов, действий колеса, нажатий клавиш, многоклавишного ввода и процедур набора текста. Реализация использует нативные интерфейсы Windows и вызовы Device I/O Control для связи с дескриптором драйвера. Основное применение — исследование эмуляции ввода, эксперименты по автоматизации и анализ поверхности атаки уязвимых драйверов в контексте игровой безопасности.
