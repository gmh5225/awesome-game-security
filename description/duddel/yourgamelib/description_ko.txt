CMake 기반의 게임형 애플리케이션용 크로스 플랫폼 C++11 프레임워크로, OpenGL/OpenGL ES 렌더링을 제공하며 GLFW, GLM, ImGui, Lua, Box2D, Bullet, miniaudio 같은 공통 라이브러리를 자동으로 통합합니다.
데스크톱(Windows, Linux, macOS), Android, 웹(Emscripten) 타깃을 지원하며 에셋 로딩, 오디오, 수학, 입력 유틸리티 도구 상자를 제공합니다.
관리되는 의존성 통합으로 가벼운 크로스 플랫폼 OpenGL 애플리케이션을 만드는 게임 개발자와 엔진 연구자에게 주로 유용합니다.
