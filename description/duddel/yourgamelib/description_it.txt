Un framework C++11 multipiattaforma per applicazioni di tipo gioco, basato su CMake con rendering OpenGL/OpenGL ES, che fornisce l'integrazione automatica di librerie comuni come GLFW, GLM, ImGui, Lua, Box2D, Bullet e miniaudio.
Supporta target desktop (Windows, Linux, macOS), Android e web (Emscripten), con una cassetta degli attrezzi per caricamento asset, audio, matematica e input.
È soprattutto utile per sviluppatori di giochi e ricercatori di engine che realizzano applicazioni OpenGL multipiattaforma leggere con integrazione gestita delle dipendenze.
