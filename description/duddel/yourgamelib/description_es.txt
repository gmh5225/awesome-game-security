Un framework C++11 multiplataforma para aplicaciones de tipo juego, basado en CMake con renderizado OpenGL/OpenGL ES, que ofrece integración automatizada de bibliotecas comunes como GLFW, GLM, ImGui, Lua, Box2D, Bullet y miniaudio.
Admite destinos de escritorio (Windows, Linux, macOS), Android y web (Emscripten), con una caja de herramientas de carga de assets, audio, matemáticas y entradas.
Resulta especialmente útil para desarrolladores de juegos e investigadores de motores que construyen aplicaciones OpenGL multiplataforma ligeras con integración gestionada de dependencias.
