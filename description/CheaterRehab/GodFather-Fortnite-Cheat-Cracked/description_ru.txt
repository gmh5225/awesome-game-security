Этот проект — утёкшая и модифицированная кодовая база внешнего чита для Fortnite, включающая компоненты пользовательского режима и режима ядра. Пользовательская часть написана на C++ с оверлеем Dear ImGui на DirectX9, а сторона ядра содержит логику драйвера и код связи на основе callback. Сопровождающие заметки обсуждают статус детекта, поведение загрузчика и техники mapping, используемые для развёртывания драйвера. Материал в основном полезен для реверс-инжиниринга читов, анализа обхода античита и изучения реальной «склеенной» (pasted) инфраструктуры читов.
