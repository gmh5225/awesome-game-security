Este proyecto es una base de código de cheat externo de Fortnite filtrada y modificada que incluye componentes de modo usuario y modo kernel. El lado de modo usuario está escrito en C++ con una superposición Dear ImGui de DirectX9, mientras que el lado del kernel contiene lógica de controlador y código de comunicación basado en callbacks. Las notas adjuntas analizan el estado de detección, el comportamiento del loader y las técnicas de mapping usadas para desplegar el controlador. Es principalmente relevante para la ingeniería inversa de cheats, el análisis de evasión de anti-cheat y el estudio de infraestructuras de cheats pegadas del mundo real.
