Ce projet est une base de code de cheat Fortnite externe, divulguée et modifiée, qui comprend des composants en mode utilisateur et en mode noyau. Le côté utilisateur est écrit en C++ avec une overlay Dear ImGui sous DirectX9, tandis que le côté noyau contient la logique du pilote et du code de communication basé sur des callbacks. Les notes associées abordent l'état de détection, le comportement du loader et les techniques de mapping utilisées pour déployer le pilote. Il est principalement pertinent pour la rétro-ingénierie de cheats, l'analyse de contournement d'anti-cheat et l'étude d'infrastructures de cheats collées du monde réel.
