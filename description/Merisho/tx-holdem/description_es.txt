tx-holdem es una biblioteca JavaScript para construir la lógica del póker Texas Hold'em en Node.js y en navegadores.
Incluye modelado de cartas y manos, comparación de combinaciones, selección de la mejor mano y cálculos relacionados con el draw mediante una API reutilizable.
El proyecto incluye ejemplos y pruebas que muestran cómo componer programáticamente boards, cartas de bolsillo y flujos de ranking.
Está dirigido a desarrolladores de juegos y autores de herramientas de simulación que necesitan un motor ligero para las reglas de gameplay del póker.
