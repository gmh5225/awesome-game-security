Este proyecto es una filtración de una base de código de cheat interno de Fortnite implementada en C++.
Las fuentes incluyen manipulación del gameplay impulsada por hooks, módulos de renderizado y utilidades, y referencias a funciones como no spread, exploits de movimiento y vehículos, y comportamientos relacionados con teletransporte.
Utiliza componentes habituales del game-hacking en Windows como MinHook y Detours para la interceptación de funciones.
El repositorio es relevante principalmente como muestra de ingeniería inversa para estudiar la arquitectura de cheats y las superficies de detección.
