Этот проект — утёкший исходный код внешнего чита для Fortnite, который читает память игры и рисует оверлей для прицеливания и ESP.
Кодовая база на C++ с компонентами DirectX9 и ImGui; включает математику камеры и костей, проекцию world-to-screen и настраиваемые параметры аима.
Структура, по всей видимости, построена вокруг внешнего доступа к памяти процесса с чтением через драйвер и внутриигровым меню для переключения функций.
Основное применение — исследование игровых читов и анализ устройства внешних инструментов.
