Этот проект — plthook, кроссплатформенная библиотека на C для перехвата функций путём изменения PLT (Procedure Linkage Table) и GOT (Global Offset Table) в Linux, macOS и Windows. Она перехватывает вызовы динамически связанных функций, заменяя записи GOT/IAT и перенаправляя вызовы на пользовательские функции-замены. Библиотека поддерживает двоичные форматы ELF, Mach-O и PE. Она предназначена для исследователей безопасности, разработчиков инструментов профилирования и всех, кому нужен переносимый динамический перехват функций.
