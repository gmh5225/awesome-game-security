Este proyecto es plthook, una biblioteca C multiplataforma para enganchar funciones mediante la modificación de PLT (Procedure Linkage Table) y GOT (Global Offset Table) en Linux, macOS y Windows. Intercepta llamadas a funciones enlazadas dinámicamente reemplazando entradas GOT/IAT y redirigiendo las llamadas a funciones de reemplazo definidas por el usuario. La biblioteca admite los formatos binarios ELF, Mach-O y PE. Está orientada a investigadores de seguridad, desarrolladores de herramientas de profiling y a quien necesite una interceptación dinámica de funciones portable.
