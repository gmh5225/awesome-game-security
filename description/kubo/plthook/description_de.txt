Dieses Projekt ist plthook, eine plattformübergreifende C-Bibliothek zum Hooken von Funktionen durch Änderung von PLT (Procedure Linkage Table) und GOT (Global Offset Table) unter Linux, macOS und Windows. Sie fängt dynamisch gelinkte Funktionsaufrufe ab, indem sie GOT/IAT-Einträge ersetzt und Aufrufe an benutzerdefinierte Ersatzfunktionen umleitet. Die Bibliothek unterstützt die Binärformate ELF, Mach-O und PE. Sie richtet sich an Sicherheitsforscher, Entwickler von Profiling-Werkzeugen und alle, die eine portable dynamische Funktionsabfangung benötigen.
