Questo progetto è una raccolta di strumenti e demo di ricerca sul kernel Windows, tipicamente associati alle utility di ispezione di sistema in stile ARK. Include esempi di hook tramite registri di debug, metodi di hook IDT e GDT, iniezione APC a livello kernel, inline hooking di Shadow SSDT, operazioni sul registro tramite driver e iniezione cross-architecture WOW64. Il codice è principalmente in C e C++, con più progetti Visual Studio che coprono esperimenti autonomi e uno strumento integrato più ampio. È pensato per ricercatori di sicurezza a basso livello che studiano gli internals di Windows, le tecniche rootkit e il rilevamento difensivo.
