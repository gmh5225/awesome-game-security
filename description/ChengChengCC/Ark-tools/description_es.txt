Este proyecto es una colección de herramientas y demos de investigación del kernel de Windows, asociadas a utilidades de inspección del sistema de estilo ARK. Incluye ejemplos de hooks con registros de depuración, métodos de hook de IDT y GDT, inyección APC en el kernel, inline hooking de Shadow SSDT, operaciones de registro mediante controladores e inyección cross-architecture WOW64. El código está principalmente en C y C++, con varios proyectos de Visual Studio que cubren experimentos independientes y una herramienta integrada más amplia. Está pensado para investigadores de seguridad de bajo nivel que estudian los internals de Windows, técnicas de rootkit y detección defensiva.
