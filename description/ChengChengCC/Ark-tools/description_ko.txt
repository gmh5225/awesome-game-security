이 프로젝트는 ARK 계열 시스템 검사 유틸리티와 관련된 Windows 커널 연구 도구 및 데모 모음입니다. 디버그 레지스터 후크, IDT·GDT 후크 기법, 커널 APC 인젝션, Shadow SSDT 인라인 후킹, 드라이버를 통한 레지스트리 작업, WOW64 교차 아키텍처 인젝션 등의 예제를 포함합니다. 코드베이스는 주로 C와 C++이며, 독립 실험과 통합 도구를 다루는 여러 Visual Studio 프로젝트로 구성되어 있습니다. Windows 내부 구조, 루트킷 기법, 방어적 탐지를 연구하는 저수준 보안 연구자를 대상으로 합니다.
