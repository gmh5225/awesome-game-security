Ce projet est une collection d'outils et de démos de recherche sur le noyau Windows, associés aux utilitaires d'inspection système de type ARK. Il comprend des exemples de hooks via registres de débogage, de méthodes de hook IDT et GDT, d'injection APC noyau, de hooking inline Shadow SSDT, d'opérations de registre via des pilotes, et d'injection cross-architecture WOW64. Le code est principalement en C et C++, avec plusieurs projets Visual Studio couvrant des expériences autonomes et un outil intégré plus large. Il s'adresse aux chercheurs en sécurité bas niveau étudiant les internals Windows, les techniques de rootkit et la détection défensive.
