Этот проект — набор инструментов и демонстраций для исследования ядра Windows, связанных с утилитами системной инспекции в стиле ARK. В него входят примеры хуков через отладочные регистры, методы хуков IDT и GDT, внедрения kernel APC, inline-хукинга Shadow SSDT, операций с реестром через драйверы и кросс-архитектурного внедрения WOW64. Кодовая база в основном на C и C++, с несколькими проектами Visual Studio — от отдельных экспериментов до более крупного интегрированного инструмента. Предназначен для исследователей низкоуровневой безопасности, изучающих внутренности Windows, техники руткитов и защитное обнаружение.
