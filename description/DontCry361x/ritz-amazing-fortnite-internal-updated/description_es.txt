Este proyecto es una base de cheat interno de Fortnite construida en torno a código C++ inyectado y acceso a objetos de Unreal Engine. Incluye un menú configurable en el juego con aim por memoria, silent aim, controles de FOV y varios módulos ESP como caja, esqueleto, línea, nombre, arma y munición. La implementación también contiene ayudas para hallar offsets, rutinas de call-spoofing y hooks de comportamiento de armas como disparo rápido y recarga instantánea. Está pensado principalmente para prototipado ofensivo de cheats y para investigadores de anti-cheat que estudian la arquitectura de cheats internos.
