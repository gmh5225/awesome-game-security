Ce projet fournit un guide sur le débogage distant du noyau Windows.
Il est organisé sous forme de documentation et de matériel de référence pour le domaine des cheats / guides, plutôt que comme une base de code runtime autonome.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / guides.
