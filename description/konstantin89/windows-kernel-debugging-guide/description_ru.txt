Этот проект предоставляет руководство по удалённой отладке ядра Windows.
Он организован как документация и справочные материалы для области читов / гайдов, а не как самостоятельная runtime-кодовая база.
Он в основном полезен исследователям безопасности игр и инженерам по реверс-инжинирингу, изучающим атакующие техники в области читов / гайдов.
