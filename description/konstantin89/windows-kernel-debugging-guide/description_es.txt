Este proyecto ofrece una guía sobre la depuración remota del kernel de Windows.
Está organizado como documentación y material de referencia para el área de cheats / guías, más que como una base de código de ejecución independiente.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheats / guías.
