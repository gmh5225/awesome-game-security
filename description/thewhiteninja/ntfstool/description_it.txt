Questo progetto è uno strumento forense incentrato sui volumi NTFS.
Supporta la lettura delle informazioni sulle partizioni (MBR, tabella delle partizioni, VBR) e anche dati sulla Master File Table, volumi crittografati BitLocker, file crittografati EFS, journal USN e altro ancora.
È soprattutto utile per gli ingegneri anti-cheat e i ricercatori di sicurezza difensiva che operano nell'ambito anti-cheat / sistemi informativi e forensics.
