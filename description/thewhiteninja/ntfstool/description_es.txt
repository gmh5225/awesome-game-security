Este proyecto es una herramienta forense centrada en volúmenes NTFS.
Admite la lectura de información de particiones (MBR, tabla de particiones, VBR), así como datos sobre la Master File Table, volúmenes cifrados con BitLocker, archivos cifrados con EFS, el diario USN y más.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el ámbito del anti-cheat / sistemas de información y forense.
