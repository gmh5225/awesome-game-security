Dieses Projekt ist ein forensisches Tool mit Fokus auf NTFS-Volumes.
Es unterstützt das Auslesen von Partitionsinformationen (MBR, Partitionstabelle, VBR) sowie Informationen zur Master File Table, zu BitLocker-verschlüsselten Volumes, EFS-verschlüsselten Dateien, dem USN-Journal und mehr.
Es eignet sich vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher im Bereich Anti-Cheat / Informationssysteme und Forensik.
