Ce projet est un outil forensique axé sur les volumes NTFS.
Il permet de lire les informations de partition (MBR, table des partitions, VBR), ainsi que des données sur la Master File Table, les volumes chiffrés BitLocker, les fichiers chiffrés EFS, le journal USN et plus encore.
Il est surtout utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine de l'anti-cheat / des systèmes d'information et de la forensique.
