Этот проект — инструмент обнаружения антиотладки для Linux, реализованный на C. Он проверяет несколько индикаторов, включая поведение ptrace, подмену LD_PRELOAD и значения TracerPid в /proc/self/status. Логика обнаружения рассчитана на ранний запуск и сообщает о вероятных условиях отладки с помощью простых проверок во время выполнения. Основной сценарий использования — исследования по антианализу и эксперименты по защитному укреплению Linux-бинарников.
