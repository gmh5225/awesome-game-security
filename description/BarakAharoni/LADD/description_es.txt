Este proyecto es una herramienta de detección anti-depuración para Linux implementada en C. Comprueba múltiples indicadores, entre ellos el comportamiento de ptrace, la manipulación de LD_PRELOAD y los valores de TracerPid en /proc/self/status. La lógica de detección está diseñada para ejecutarse temprano e informar condiciones de depuración probables mediante comprobaciones sencillas en tiempo de ejecución. El caso de uso principal es la investigación anti-análisis y los experimentos de endurecimiento defensivo para binarios de Linux.
