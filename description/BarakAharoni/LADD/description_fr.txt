Ce projet est un outil de détection anti-débogage pour Linux, implémenté en C. Il vérifie plusieurs indicateurs, notamment le comportement de ptrace, la falsification de LD_PRELOAD et les valeurs TracerPid dans /proc/self/status. La logique de détection est conçue pour s'exécuter tôt et signaler les conditions de débogage probables via des contrôles d'exécution simples. Le cas d'usage principal est la recherche anti-analyse et les expériences de durcissement défensif pour les binaires Linux.
