Este proyecto es un complemento de Unreal Engine que integra un entorno de ejecución de scripting Lua para la programación de gameplay. Ofrece lógica de juego impulsada por Lua mediante assets de código accesibles desde Blueprint, soporte de corrutinas, metatablas basadas en reflexión e interfaces de llamada proxy para conectar scripts Lua con actores y componentes de UE4. Resulta especialmente útil para desarrolladores de juegos y modders que buscan gameplay programable mediante scripts en Unreal Engine a través de la integración con Lua.
