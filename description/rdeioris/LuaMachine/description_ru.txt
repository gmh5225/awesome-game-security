Этот проект — плагин для Unreal Engine, встраивающий среду выполнения скриптов Lua для программирования геймплея. Он обеспечивает игровую логику на Lua через кодовые ассеты, доступные из Blueprint, поддержку корутин, метатаблицы на основе рефлексии и прокси-интерфейсы вызовов для связи скриптов Lua с акторами и компонентами UE4. В основном полезен разработчикам игр и моддерам, которым нужен скриптуемый геймплей в Unreal Engine через интеграцию с Lua.
