Ce projet est un plugin Unreal Engine qui intègre un runtime de scripting Lua pour la programmation de gameplay. Il fournit une logique de jeu pilotée par Lua via des assets de code accessibles depuis Blueprint, la prise en charge des coroutines, des métatables basées sur la réflexion, et des interfaces d'appels proxy pour relier les scripts Lua aux acteurs et composants UE4. Il est surtout utile aux développeurs de jeux et aux moddeurs souhaitant un gameplay scriptable dans Unreal Engine grâce à l'intégration de Lua.
