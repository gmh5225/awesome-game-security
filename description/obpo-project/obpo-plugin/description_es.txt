En este proyecto, el núcleo obpo es de código cerrado, pero el obpo-plugin proporcionado es GRATUITO y de CÓDIGO ABIERTO.
Se trata de un servidor backend sencillo escrito en Golang, que recibe solicitudes, las envía a IDA, usa obpo-core para la desofuscación y finalmente devuelve los datos al cliente obpo-plugin.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / corrección de OLLVM.
