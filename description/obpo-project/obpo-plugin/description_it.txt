In questo progetto il core obpo è closed source, ma l'obpo-plugin fornito è GRATUITO e OPEN SOURCE.
Si tratta di un semplice server backend scritto in Golang, che riceve le richieste, le invia a IDA, usa obpo-core per la deoffuscazione e infine restituisce i dati al client obpo-plugin.
È principalmente utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / fix OLLVM.
