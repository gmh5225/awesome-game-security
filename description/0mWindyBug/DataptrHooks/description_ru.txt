Этот проект — proof of concept для ядра Windows, демонстрирующий обмен данными через хуки указателей в .data вместо стандартных путей IOCTL. Показаны цели хуков, достижимые через syscall, включая сценарии, связанные с NtConvertBetweenAuxiliaryCounterAndPerformanceCounter и запросом целостности кода. В репозитории есть пользовательские клиенты и проекты драйверов ядра на C и C++, а также исследовательские заметки о поиске сайтов косвенных вызовов через артефакты Control Flow Guard. Материал предназначен для исследователей безопасности ядра, изучающих более скрытную коммуникацию отображённых драйверов и компромиссы детектирования.
