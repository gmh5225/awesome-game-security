本プロジェクトは、標準の IOCTL 経路ではなく .data ポインタフックを通じた通信を示す Windows カーネルの概念実証です。NtConvertBetweenAuxiliaryCounterAndPerformanceCounter やコード整合性照会に関連する流れを含む、システムコールから到達可能なフック対象を示します。リポジトリには C/C++ のユーザーモードクライアントとカーネルドライバプロジェクト、および Control Flow Guard 関連の成果物から間接呼び出し箇所を特定する研究ノートが含まれます。より隠密なマップドドライバ通信と検知上のトレードオフを研究するカーネルセキュリティ研究者向けです。
