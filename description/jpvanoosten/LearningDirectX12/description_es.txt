Este proyecto es una serie de tutoriales y código de ejemplo para aprender programación gráfica con DirectX 12. Cubre los fundamentos de D3D12, incluida la creación de dispositivos, colas de comandos, objetos de estado de canalización, root signatures, descriptor heaps, gestión de recursos, carga de texturas y técnicas de renderizado de complejidad progresiva. Los ejemplos en C++ van desde el renderizado básico de un triángulo hasta escenas completas. Está dirigido a programadores gráficos que aprenden programación GPU moderna de bajo nivel con DirectX 12.
