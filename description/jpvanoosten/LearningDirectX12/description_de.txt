Dieses Projekt ist eine Tutorial-Reihe mit Beispielcode zum Erlernen der DirectX-12-Grafikprogrammierung. Es behandelt D3D12-Grundlagen wie Geräteerstellung, Command Queues, Pipeline State Objects, Root Signatures, Descriptor Heaps, Ressourcenverwaltung, Texturladen und Rendering-Techniken mit steigender Komplexität. Die C++-Beispiele führen vom einfachen Dreieck-Rendering bis zu vollständigen Szenen. Es richtet sich an Grafikprogrammierer, die moderne Low-Level-GPU-Programmierung mit DirectX 12 lernen.
