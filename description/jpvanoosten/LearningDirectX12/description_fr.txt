Ce projet est une série de tutoriels et du code d'exemples pour apprendre la programmation graphique DirectX 12. Il couvre les fondamentaux de D3D12, notamment la création de périphériques, les files de commandes, les objets d'état de pipeline, les root signatures, les descriptor heaps, la gestion des ressources, le chargement de textures et des techniques de rendu de complexité progressive. Les exemples C++ vont du rendu d'un simple triangle à des scènes complètes. Il s'adresse aux programmeurs graphiques qui apprennent la programmation GPU moderne de bas niveau avec DirectX 12.
