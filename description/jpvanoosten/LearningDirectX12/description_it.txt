Questo progetto è una serie di tutorial e codice di esempio per imparare la programmazione grafica con DirectX 12. Copre i fondamenti di D3D12, tra cui creazione del dispositivo, code di comandi, pipeline state object, root signature, descriptor heap, gestione delle risorse, caricamento delle texture e tecniche di rendering a complessità progressiva. Gli esempi in C++ partono dal rendering di un triangolo di base fino a scene complete. È rivolto ai programmatori grafici che apprendono la programmazione GPU moderna a basso livello con DirectX 12.
