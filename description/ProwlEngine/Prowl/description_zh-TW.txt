Prowl 是一款開源的 C#/.NET 遊戲引擎，採用類似 Unity 的 GameObject 與 MonoBehaviour 元件模型，並提供完整的視覺化編輯器與執行階段。它包含 Prowl.Editor 與 Prowl.Runtime，支援場景編輯、資源匯入、具備熱重載的 C# 腳本、預製體以及桌面建置流程。執行階段涵蓋 PBR 渲染與後處理（bloom、TAA、SMAA、SSR、GTAO、體積霧）、具備剛體、碰撞體與關節的 3D 物理、基於 MiniAudio 的音訊、粒子系統、地形、蒙皮網格以及 canvas UI 系統。跨平台原生函式庫與範例專案支援物理示範、音訊、飛行相機與體素等常見遊戲情境。它面向希望使用以 C# 為先的開放引擎來建構與迭代 3D 遊戲的開發者與引擎貢獻者。
