Prowl は、Unity 風の GameObject と MonoBehaviour コンポーネントモデルに加え、本格的なビジュアルエディタとランタイムを備えたオープンソースの C#/.NET ゲームエンジンです。Prowl.Editor と Prowl.Runtime を同梱し、シーン編集、アセットインポート、ホットリロード対応の C# スクリプティング、プレハブ、デスクトップ向けビルドパイプラインを提供します。ランタイムは PBR レンダリングとポストプロセス（bloom、TAA、SMAA、SSR、GTAO、ボリューメトリックフォグ）、リジッドボディ・コライダー・ジョイントによる 3D 物理、MiniAudio ベースのオーディオ、パーティクルシステム、地形、スキンメッシュ、canvas UI システムをカバーします。クロスプラットフォームのネイティブライブラリとサンプルプロジェクトは、物理デモ、オーディオ、フライカメラ、ボクセルなど一般的なゲームプレイシナリオを支えます。C# を第一に据えたオープンエンジンで 3D ゲームを構築・反復したいゲーム開発者とエンジン貢献者を対象としています。
