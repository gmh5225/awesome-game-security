Prowl 是一款开源的 C#/.NET 游戏引擎，采用类似 Unity 的 GameObject 与 MonoBehaviour 组件模型，并提供完整的可视化编辑器与运行时。它包含 Prowl.Editor 与 Prowl.Runtime，支持场景编辑、资源导入、带热重载的 C# 脚本、预制体以及桌面构建流水线。运行时涵盖 PBR 渲染与后期处理（bloom、TAA、SMAA、SSR、GTAO、体积雾）、带刚体、碰撞体与关节的 3D 物理、基于 MiniAudio 的音频、粒子系统、地形、蒙皮网格以及 canvas UI 系统。跨平台原生库与示例项目覆盖物理演示、音频、飞行相机与体素等常见玩法场景。它面向希望使用以 C# 为先的开放引擎来构建与迭代 3D 游戏的开发者与引擎贡献者。
