Prowl es un motor de juegos de código abierto en C#/.NET con un modelo de componentes GameObject y MonoBehaviour al estilo de Unity, además de un editor visual completo y un runtime. Incluye Prowl.Editor y Prowl.Runtime para edición de escenas, importación de assets, scripting en C# con hot reload, prefabs y pipelines de build para escritorio. El runtime cubre renderizado PBR y postprocesado (bloom, TAA, SMAA, SSR, GTAO, niebla volumétrica), física 3D con rigidbodies, colliders y joints, audio basado en MiniAudio, sistemas de partículas, terreno, skinned meshes y un sistema de UI canvas. Bibliotecas nativas multiplataforma y proyectos de ejemplo respaldan escenarios de gameplay habituales como demos de física, audio, cámaras en vuelo y voxels. Está pensado para desarrolladores de juegos y colaboradores de motores que quieren un motor abierto centrado en C# para crear e iterar juegos 3D.
