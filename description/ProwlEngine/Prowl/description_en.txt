Prowl is an open-source C#/.NET game engine with a Unity-style GameObject and MonoBehaviour component model, plus a full visual editor and runtime. It ships Prowl.Editor and Prowl.Runtime for scene editing, asset import, C# scripting with hot reload, prefabs, and desktop build pipelines. The runtime covers PBR rendering and post-processing (bloom, TAA, SMAA, SSR, GTAO, volumetric fog), 3D physics with rigidbodies, colliders, and joints, MiniAudio-based audio, particle systems, terrain, skinned meshes, and a canvas UI system. Cross-platform native libraries and sample projects support common gameplay scenarios such as physics demos, audio, fly cameras, and voxels. It is aimed at game developers and engine contributors who want a C#-first open engine for building and iterating on 3D games.
