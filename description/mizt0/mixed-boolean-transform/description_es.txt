Herramienta de ofuscación C++ de código fuente a código fuente que sustituye constantes enteras y expresiones aritméticas por expresiones de aritmética booleana mixta (MBA) semánticamente equivalentes, verificadas con el solucionador SMT Z3. Genera grandes identidades MBA polinómicas sobre operaciones bit a bit (AND, OR, XOR, NOT) con álgebra lineal Eigen3 y enteros de precisión arbitraria GMP.
