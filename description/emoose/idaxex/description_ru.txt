Этот проект — idaxex, плагин-загрузчик IDA Pro для файлов XEX (Xenon Executable) Xbox 360. Он выполняет разбор формата XEX, извлечение PE из контейнера XEX, разрешение импортов/экспортов и именование функций ядра Xbox 360. Плагин на C++ позволяет анализировать исполняемые файлы игр Xbox 360 в IDA Pro с корректной аннотацией символов и структур. Он предназначен для реверс-инженеров, исследующих бинарники игр Xbox 360, и специалистов по безопасности консолей.
