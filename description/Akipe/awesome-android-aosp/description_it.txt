Questo progetto è un indice curato di risorse per lo sviluppo di Android AOSP e ROM personalizzate. È principalmente una knowledge base in Markdown che organizza link su sistemi di build, kernel, device tree, Treble, SELinux, debugging e workflow specifici dei vendor. La raccolta privilegia l’ingegneria di piattaforma a basso livello rispetto allo sviluppo di app e funge da mappa di apprendimento strutturata. È utile per ricercatori di sicurezza mobile e di game security che necessitano di un contesto più approfondito sugli interni del sistema Android.
