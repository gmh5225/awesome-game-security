Este proyecto es una biblioteca de matemáticas 3D ligera para programación de juegos y gráficos. Proporciona primitivas y operaciones matemáticas esenciales usadas en motores en tiempo real, incluidos vectores, matrices y ayudas de transformación, con ejemplos y pruebas para facilitar la integración. La biblioteca está escrita principalmente en C, con soporte de compilación mediante CMake y enlaces o código complementario en otros lenguajes. Está dirigida a desarrolladores de juegos y autores de motores/herramientas que necesitan una base matemática portable para renderizado, simulación o herramientas de investigación en seguridad.
