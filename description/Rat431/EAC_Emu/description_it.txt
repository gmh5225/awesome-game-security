Questo progetto è un semplice stub di emulatore EasyAntiCheat x64 implementato come DLL Windows.
Esporta un ampio insieme di funzioni API anti-cheat attese e fornisce implementazioni di interfaccia segnaposto per soddisfare le chiamate del client.
Il codice è principalmente in C++, con un piccolo helper in assembly per routine di patch a basso livello.
È pensato come proof of concept per reverse engineering e test di compatibilità di software che si collega ai componenti EAC.
