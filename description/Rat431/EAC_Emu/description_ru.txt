Этот проект — простой stub-эмулятор EasyAntiCheat для x64, реализованный как Windows DLL.
Он экспортирует большой набор ожидаемых API-функций античита и предоставляет заглушечные реализации интерфейсов для удовлетворения клиентских вызовов.
Кодовая база в основном на C++, с небольшим вспомогательным модулем на ассемблере для низкоуровневых процедур патчинга.
Предназначен как proof of concept для реверс-инжиниринга и тестирования совместимости ПО, связываемого с компонентами EAC.
