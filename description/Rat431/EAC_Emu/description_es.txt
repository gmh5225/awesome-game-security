Este proyecto es un stub sencillo de emulador EasyAntiCheat x64 implementado como una DLL de Windows.
Exporta un amplio conjunto de funciones de API anticheat esperadas y proporciona implementaciones de interfaz de marcador de posición para satisfacer las llamadas del cliente.
La base de código es principalmente C++, con un pequeño helper en ensamblador para rutinas de parcheo de bajo nivel.
Está pensado como una prueba de concepto para ingeniería inversa y pruebas de compatibilidad de software que enlaza contra componentes EAC.
