Ce projet est une simple stub d'émulateur EasyAntiCheat x64 implémentée sous forme de DLL Windows.
Il exporte un large ensemble de fonctions API anti-triche attendues et fournit des implémentations d'interface factices pour satisfaire les appels clients.
La base de code est principalement en C++, avec un petit helper en assembleur pour les routines de patch de bas niveau.
Il est conçu comme une preuve de concept pour l'ingénierie inverse et les tests de compatibilité de logiciels liés aux composants EAC.
