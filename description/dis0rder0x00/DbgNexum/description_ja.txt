Windows Debugging API と共有メモリ（ファイルマッピング）を用い、WriteProcessMemory、VirtualAllocEx、ReadProcessMemory を呼び出さずにペイロードを注入する概念実証の shellcode インジェクタです。
デバッガとしてアタッチし、ハードウェアブレークポイントを設定してスレッドコンテキストのレジスタを操作し、ターゲットプロセス内で API 呼び出しの連鎖を組み立てて shellcode をマップ・実行します。
従来のメモリ操作 API を避けて EDR 検知を回避する高度なインジェクション手法を研究するセキュリティ研究者向けです。
