Windows Debugging API와 공유 메모리(파일 매핑)를 사용해 WriteProcessMemory, VirtualAllocEx, ReadProcessMemory를 호출하지 않고 페이로드를 주입하는 개념 증명용 shellcode 인젝터입니다.
디버거로 연결한 뒤 하드웨어 중단점을 설정하고 스레드 컨텍스트 레지스터를 조작하여, 대상 프로세스 안에서 API 호출 체인을 구성해 shellcode를 매핑하고 실행합니다.
전통적인 메모리 조작 API를 피함으로써 EDR 탐지를 우회하는 고급 주입 기법을 연구하는 보안 연구자에게 주로 유용합니다.
