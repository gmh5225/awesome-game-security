Este proyecto es Recast & Detour, un conjunto de herramientas de mallas de navegación de referencia en la industria para juegos. Recast genera mallas de navegación a partir de geometría 3D arbitraria mediante voxelización, seguimiento de contornos y simplificación de mallas poligonales. Detour ofrece búsqueda de caminos, suavizado de rutas y dirección local sobre las navmeshes generadas, con soporte para conexiones fuera de malla, costes de área y streaming basado en mosaicos. La biblioteca C++ se usa en numerosos motores y títulos de juegos comerciales. Está dirigida a desarrolladores de juegos que implementan sistemas de navegación y pathfinding para IA.
