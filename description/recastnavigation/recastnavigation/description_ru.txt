Этот проект — Recast & Detour, отраслевой стандартный набор инструментов навигационных мешей для игр. Recast строит навигационные меши из произвольной 3D-геометрии с помощью вокселизации, трассировки контуров и упрощения полигональных мешей. Detour обеспечивает поиск пути, сглаживание траектории и локальное управление на сгенерированных навмешах с поддержкой off-mesh соединений, стоимостей областей и потоковой загрузки по тайлам. Библиотека на C++ используется во многих коммерческих игровых движках и проектах. Она предназначена для разработчиков игр, реализующих системы ИИ-навигации и поиска пути.
