Ce projet est une bibliothèque réseau pour le noyau Windows qui encapsule l'interface Winsock Kernel avec une API de type socket. Elle est principalement implémentée en C et C++, livrée avec des fichiers NuGet et MSBuild, et conçue pour les workflows de drivers WDK et Visual Studio. Le code fait correspondre les opérations socket familières en mode utilisateur aux fonctions WSK en mode noyau telles que connect, send, recv et les helpers de conversion d'adresses. Elle s'adresse surtout aux développeurs de drivers noyau et aux chercheurs en sécurité qui ont besoin d'E/S réseau en mode noyau avec une intégration plus simple.
