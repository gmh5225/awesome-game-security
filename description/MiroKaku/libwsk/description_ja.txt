本プロジェクトは、Winsock Kernel インターフェイスをソケット風 API でラップする Windows カーネル向けネットワーキングライブラリです。主に C と C++ で実装され、NuGet および MSBuild ファイル付きでパッケージ化されており、WDK と Visual Studio によるドライバ開発ワークフロー向けに設計されています。ユーザーモードで馴染みのあるソケット操作を、connect、send、recv、アドレス変換ヘルパーなどのカーネルモード WSK 関数へ対応づけます。カーネルモードでのネットワーク I/O を、より低い統合コストで必要とするカーネルドライバ開発者およびセキュリティ研究者を主な対象としています。
