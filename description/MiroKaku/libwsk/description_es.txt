Este proyecto es una biblioteca de red para el kernel de Windows que encapsula la interfaz Winsock Kernel con una API de estilo socket. Está implementada principalmente en C y C++, empaquetada con archivos NuGet y MSBuild, y diseñada para flujos de trabajo de drivers con WDK y Visual Studio. El código mapea operaciones de socket familiares en modo usuario a funciones WSK en modo kernel como connect, send, recv y auxiliares de conversión de direcciones. Está orientada principalmente a desarrolladores de drivers de kernel e investigadores de seguridad que necesitan E/S de red en modo kernel con menor fricción de integración.
