이 프로젝트는 Winsock Kernel 인터페이스를 소켓 스타일 API로 감싼 Windows 커널 네트워킹 라이브러리입니다. 주로 C와 C++로 구현되며 NuGet 및 MSBuild 파일과 함께 패키징되어, WDK와 Visual Studio 드라이버 워크플로에 맞게 설계되었습니다. 익숙한 사용자 모드 소켓 작업을 connect, send, recv 및 주소 변환 도우미와 같은 커널 모드 WSK 함수에 매핑합니다. 커널 모드에서 네트워크 I/O가 필요하면서 통합 부담을 줄이고자 하는 커널 드라이버 개발자와 보안 연구자를 주요 대상으로 합니다.
