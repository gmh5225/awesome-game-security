Этот проект — библиотека сетевого взаимодействия для ядра Windows, которая оборачивает интерфейс Winsock Kernel в API в стиле сокетов. Она в основном реализована на C и C++, поставляется с файлами NuGet и MSBuild и рассчитана на рабочие процессы драйверов с WDK и Visual Studio. Код сопоставляет привычные пользовательские операции сокетов с функциями WSK в режиме ядра, такими как connect, send, recv и вспомогательными средствами преобразования адресов. Библиотека в первую очередь предназначена для разработчиков драйверов ядра и исследователей безопасности, которым нужен сетевой ввод-вывод в режиме ядра с меньшей сложностью интеграции.
