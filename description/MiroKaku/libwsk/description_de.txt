Dieses Projekt ist eine Windows-Kernel-Netzwerkbibliothek, die die Winsock-Kernel-Schnittstelle mit einer socket-ähnlichen API kapselt. Sie ist hauptsächlich in C und C++ implementiert, wird mit NuGet- und MSBuild-Dateien ausgeliefert und ist für WDK- und Visual-Studio-Treiber-Workflows ausgelegt. Der Code mappt vertraute Usermode-Socket-Operationen auf Kernelmode-WSK-Funktionen wie connect, send, recv sowie Hilfsfunktionen zur Adresskonvertierung. Primär richtet sie sich an Kernel-Treiberentwickler und Sicherheitsforscher, die Netzwerkein-/ausgabe im Kernelmodus mit geringerem Integrationsaufwand benötigen.
