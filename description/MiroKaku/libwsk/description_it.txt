Questo progetto è una libreria di networking per il kernel Windows che avvolge l'interfaccia Winsock Kernel con un'API in stile socket. È implementata principalmente in C e C++, fornita con file NuGet e MSBuild, e progettata per i flussi di lavoro driver con WDK e Visual Studio. Il codice mappa le operazioni socket familiari in user mode alle funzioni WSK in kernel mode come connect, send, recv e helper di conversione degli indirizzi. È destinata soprattutto a sviluppatori di driver kernel e ricercatori di sicurezza che necessitano di I/O di rete in kernel mode con minore attrito di integrazione.
