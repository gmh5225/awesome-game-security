Este proyecto es un framework de cheat externo para Counter-Strike 2 implementado en C++. Incluye módulos para variantes de ESP, lógica de aimbot con control de retroceso, triggerbot, radar, bunnyhop, comprobaciones de visibilidad y gestión de offsets. El proyecto usa una pila de UI externa basada en ImGui y utilidades de interacción con la memoria de procesos para impulsar las funciones en tiempo de ejecución. Está pensado principalmente para ingeniería inversa educativa e investigación en seguridad de juegos sobre el diseño de cheats externos y la respuesta anti-cheat.
