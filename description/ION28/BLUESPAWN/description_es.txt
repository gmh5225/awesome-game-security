BLUESPAWN es una plataforma de código abierto de defensa activa de Windows y de detección y respuesta en endpoints (EDR). Ofrece flujos de trabajo Hunt, Mitigate, Monitor y Scan con detecciones orientadas a ATT&CK, escaneo YARA, monitoreo basado en ETW y acciones de respuesta automatizadas como cuarentena o suspensión de procesos. La implementación está mayormente en C++ con una profunda integración de las API de Windows y contenido de detección impulsado por reglas. Está diseñada para operadores de blue team e investigadores de seguridad que defienden endpoints empresariales.
