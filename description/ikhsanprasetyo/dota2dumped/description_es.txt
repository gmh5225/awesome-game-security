Este proyecto es una colección de datos volcados del juego Dota 2, incluidos offsets de netvars, punteros de interfaces y definiciones de estructuras de clases extraídos del motor Source 2. Proporciona encabezados C++ con offsets de campos de entidades, estructuras de datos de habilidades y layouts del estado del juego, actualizados tras los parches. Está dirigido a modders de Dota 2 e investigadores de seguridad de juegos que necesitan información actualizada sobre el layout de memoria del motor Source 2 tal como se usa en Dota 2.
