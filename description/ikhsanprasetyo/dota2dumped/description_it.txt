Questo progetto è una raccolta di dump di dati di gioco di Dota 2 che include offset delle netvar, puntatori alle interfacce e definizioni delle strutture delle classi estratti dal motore Source 2. Fornisce header C++ con offset dei campi delle entità, strutture dati delle abilità e layout dello stato di gioco, aggiornati dopo le patch. È rivolto a modder di Dota 2 e ricercatori di sicurezza dei videogiochi che necessitano di informazioni aggiornate sul layout di memoria del motore Source 2 usato in Dota 2.
