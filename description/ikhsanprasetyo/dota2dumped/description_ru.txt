Этот проект — коллекция дампов игровых данных Dota 2, включая смещения netvar, указатели интерфейсов и определения структур классов, извлечённые из движка Source 2. Он предоставляет заголовки C++ со смещениями полей сущностей, структурами данных способностей и раскладками игрового состояния, обновляемые после патчей. Предназначен для моддеров Dota 2 и исследователей игровой безопасности, которым нужна актуальная информация о раскладке памяти движка Source 2 в Dota 2.
