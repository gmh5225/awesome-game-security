Этот проект посвящён безопасному управлению выгрузкой DLL, перехваченных (hooked) в процесс.
Он особенно полезен в сценариях, когда динамически загруженные модули нужно безопасно выгрузить без сбоев.
В первую очередь он предназначен для инженеров античита и исследователей защитной безопасности, работающих в области античита / программирования античита.
