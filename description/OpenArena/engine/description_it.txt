Questo progetto è il codice sorgente del motore di gioco OpenArena, mantenuto come fork di ioquake3 per il comportamento client e server specifico di OpenArena. È un ampio albero del motore in C/C++ con componenti di rendering, audio, networking e runtime di gioco derivati dallo stack tecnologico di Quake III. Il repository usa build tradizionali basate su make e include file di supporto per le piattaforme e strumenti per più target. È usato principalmente da sviluppatori di motori, modder e ricercatori che studiano gli interni dei classici motori FPS e l'architettura dei giochi multiplayer.
