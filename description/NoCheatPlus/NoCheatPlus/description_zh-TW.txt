NoCheatPlus 是面向 Minecraft Bukkit 與 Spigot 伺服器的開源反作弊外掛，用於偵測並處罰常見的遊戲作弊行為。它使用 Java 撰寫，作為多模組 Maven 專案建置，包含共用工具庫、Bukkit 外掛核心，以及針對特定版本的 CraftBukkit 與 Spigot 相容層。此外掛以模組化方式實作移動、戰鬥、方塊破壞與放置、物品欄濫用、聊天洗版以及可疑網路封包速率等檢查，並支援取消、記錄與回退等可設定的違規處理動作。基於反射存取 Minecraft 內部實作，並搭配客戶端模組 MOTD 處理，以涵蓋廣泛的伺服器版本。其目標使用者為 Minecraft 伺服器營運者，以及研究伺服器端反作弊設計的遊戲安全研究者。
