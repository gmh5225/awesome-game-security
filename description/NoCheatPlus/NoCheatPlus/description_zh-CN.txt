NoCheatPlus 是面向 Minecraft Bukkit 与 Spigot 服务器的开源反作弊插件，用于检测并处罚常见的游戏作弊行为。它使用 Java 编写，作为多模块 Maven 项目构建，包含共享工具库、Bukkit 插件核心，以及针对特定版本的 CraftBukkit 与 Spigot 兼容层。该插件以模块化方式实现移动、战斗、方块破坏与放置、物品栏滥用、聊天刷屏以及可疑网络数据包速率等检查，并支持取消、记录与回退等可配置的违规处理动作。基于反射访问 Minecraft 内部实现，并配合客户端模组 MOTD 处理，以覆盖广泛的服务器版本。其目标用户为 Minecraft 服务器运营者，以及研究服务端反作弊设计的游戏安全研究者。
