NoCheatPlus è un plugin anti-cheat open source per server Minecraft Bukkit e Spigot che rileva e sanziona i cheat di gameplay più comuni. È scritto in Java e realizzato come progetto Maven multi-modulo con utilità condivise, un core del plugin Bukkit e livelli di compatibilità CraftBukkit e Spigot specifici per versione. Il plugin implementa controlli modulari su movimento, combattimento, rottura e piazzamento di blocchi, abuso dell'inventario, spam in chat e tassi sospetti di pacchetti di rete, insieme ad azioni di violazione configurabili come annullamento, registrazione e setback. L'accesso agli interni di Minecraft tramite reflection e la gestione MOTD dei mod client supportano un'ampia copertura delle versioni del server. È pensato per gli operatori di server Minecraft e per i ricercatori di game security che studiano la progettazione di anti-cheat lato server.
