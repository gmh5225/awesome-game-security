Cat Annihilation is a 3D wave-survival game in which the player controls a customizable cat fighting dog enemies, using melee and elemental magic across story, quest, and clan systems. The main product is a from-scratch native engine written in C++20 with Vulkan rendering, CUDA-accelerated physics and particles, an ECS architecture, behavior-tree AI, deferred PBR shading, and related gameplay systems. A separate browser demo of the same concept is built with React Three Fiber, Three.js, and Zustand, while CMake, Catch2 tests, and headless validation support development. It is aimed at game-engine learning and portfolio work rather than anti-cheat or reverse engineering.
