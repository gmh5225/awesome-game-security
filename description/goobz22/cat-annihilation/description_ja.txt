Cat Annihilation は、カスタマイズ可能な猫を操作し、近接攻撃と属性魔法で犬の敵と戦う 3D ウェーブサバイバルゲームで、ストーリー、クエスト、クランの各システムを備えています。主力製品はゼロから構築したネイティブエンジンで、C++20、Vulkan レンダリング、CUDA 加速の物理とパーティクル、ECS アーキテクチャ、ビヘイビアツリー AI、ディファード PBR シェーディング、および関連ゲームプレイシステムを備えます。同じコンセプトのブラウザデモは React Three Fiber、Three.js、Zustand で構築され、開発は CMake、Catch2 テスト、ヘッドレス検証で支えられています。反チートやリバースエンジニアリングではなく、ゲームエンジン学習とポートフォリオ向けです。
