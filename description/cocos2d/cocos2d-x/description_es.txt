Este proyecto ofrece cocos2d-x, un framework multiplataforma para crear juegos 2D, libros interactivos, demos y otras aplicaciones gráficas. Está basado en cocos2d-iphone, pero en lugar de Objective-C utiliza C++. Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el ámbito de motores de juego y código fuente.
