AutoRE — плагин для IDA Pro, который помогает reverse-инженерам ускорять анализ бинарных файлов.
Написан на Python для IDA и автоматически переименовывает фиктивные функции на основе импортируемых вызовов API и целей переходов.
Также помечает функции по поведенческим признакам — сеть, внедрение процессов, криптография и файловая активность — и показывает их в отдельном представлении тегов.
Проект в основном используют в reverse engineering и исследованиях безопасности игр, чтобы быстрее разбирать незнакомый код и подозрительную логику.
