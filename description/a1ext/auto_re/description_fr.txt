AutoRE est un plugin IDA Pro qui aide les ingénieurs en reverse engineering à accélérer l’analyse de binaires.
Écrit en Python pour IDA, il renomme automatiquement les fonctions factices en s’appuyant sur les appels d’API importées et les cibles de saut.
Il étiquette aussi les fonctions selon des indicateurs comportementaux tels que le réseau, l’injection de processus, la crypto et l’activité fichier, puis les présente dans une vue de tags dédiée.
Le projet sert surtout en reverse engineering et en recherche sur la sécurité des jeux pour trier plus vite le code inconnu et la logique suspecte.
