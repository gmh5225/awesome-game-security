AutoRE è un plugin di IDA Pro che aiuta i reverse engineer ad accelerare i flussi di analisi binaria.
Scritto in Python per IDA, rinomina automaticamente le funzioni dummy in base alle chiamate API importate e agli obiettivi di salto.
Assegna inoltre tag alle funzioni secondo indicatori comportamentali come networking, process injection, crypto e attività sui file, presentandole in una vista tag dedicata.
Il progetto è usato soprattutto in reverse engineering e nella ricerca sulla sicurezza dei giochi per triagare più rapidamente codice sconosciuto e logica sospetta.
