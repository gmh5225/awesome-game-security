AutoRE es un plugin de IDA Pro que ayuda a los ingenieros de reverse engineering a acelerar el análisis de binarios.
Escrito en Python para IDA, renombra automáticamente funciones ficticias basándose en llamadas a API importadas y destinos de salto.
También etiqueta funciones según indicadores de comportamiento como red, inyección de procesos, cifrado y actividad de archivos, y las muestra en una vista de etiquetas dedicada.
El proyecto se usa principalmente en reverse engineering e investigación de seguridad en juegos para clasificar más rápido código desconocido y lógica sospechosa.
