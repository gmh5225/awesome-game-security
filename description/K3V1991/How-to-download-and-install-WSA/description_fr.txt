Ce projet est un guide d’installation pas à pas pour Windows Subsystem for Android sur Windows 11.
Il documente les prérequis nécessaires, tels que le mode Développeur et la fonctionnalité Plateforme de machine virtuelle.
Les instructions expliquent comment obtenir les packages de dépendances et installer les bundles via l’Explorateur ou PowerShell.
Il s’adresse principalement aux utilisateurs qui ont besoin d’un flux d’installation manuelle de WSA.
