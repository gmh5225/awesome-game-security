Этот проект — пошаговое руководство по установке Windows Subsystem for Android в Windows 11.
В нём описаны необходимые предварительные условия, такие как режим разработчика и компонент Virtual Machine Platform.
Инструкции объясняют, как получить пакеты зависимостей и установить бандлы через Проводник или PowerShell.
Материал в первую очередь рассчитан на пользователей, которым нужен ручной процесс установки WSA.
