Este proyecto es una guía de configuración paso a paso para instalar Windows Subsystem for Android en Windows 11.
Documenta los requisitos previos necesarios, como el modo de desarrollador y la característica Plataforma de máquina virtual.
Las instrucciones explican cómo obtener paquetes de dependencias e instalar bundles mediante el Explorador o PowerShell.
Está dirigido principalmente a usuarios que necesitan un flujo de instalación manual de WSA.
