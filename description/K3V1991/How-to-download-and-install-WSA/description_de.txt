Dieses Projekt ist eine Schritt-für-Schritt-Anleitung zur Installation von Windows Subsystem for Android unter Windows 11.
Es dokumentiert erforderliche Voraussetzungen wie den Entwicklermodus und die Funktion Virtual Machine Platform.
Die Anleitung erklärt, wie Abhängigkeitspakete bezogen und Bundles über den Explorer oder PowerShell installiert werden.
Es richtet sich vor allem an Nutzer, die einen manuellen Installationsablauf für WSA benötigen.
