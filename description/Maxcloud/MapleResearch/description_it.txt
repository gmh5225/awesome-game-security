MapleResearch è un documento di analisi della community incentrato sugli interni del client MapleStory v95.1.
Il repository è principalmente in Markdown e illustra le strutture del client, inclusi componenti legati all'anti-cheat come CSecurityClient e i suoi punti di integrazione con HackShield.
Privilegia note pratiche di reverse engineering per creare ambienti localhost piuttosto che documentazione formale rifinita.
Il pubblico principale sono reverse engineer di giochi e ricercatori di server privati che necessitano di una mappa iniziale della logica di sicurezza del client.
