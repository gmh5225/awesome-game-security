Dieses Projekt stellt eine Windows-GUI-Hülle um Android-Reverse-Engineering-Werkzeuge wie apktool, signapk, zipalign und baksmali bereit. In C# auf dem .NET Framework entwickelt, vereinfacht es Workflows zum Dekompilieren, Neuaufbauen, Signieren, Ausrichten und zur Framework-Verwaltung für APK-ähnliche Pakete. Es unterstützt außerdem Drag-and-Drop, ADB-Hilfen, eine mehrsprachige Oberfläche und die Automatisierung wiederkehrender Packaging-Aufgaben. Das Tool ist für die Analyse von Android-Apps und legitime Modifikationsabläufe gedacht, nicht zum Umgehen von Schutzmaßnahmen.
