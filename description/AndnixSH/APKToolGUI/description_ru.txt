Этот проект предоставляет графическую оболочку Windows для утилит реверс-инжиниринга Android, таких как apktool, signapk, zipalign и baksmali. Написан на C# под .NET Framework и упрощает рабочие процессы декомпиляции, пересборки, подписи, выравнивания и управления framework для пакетов вроде APK. Также поддерживаются drag-and-drop, вспомогательные средства ADB, многоязычный интерфейс и автоматизация повторяющихся задач упаковки. Инструмент предназначен для анализа Android-приложений и легитимных сценариев модификации, а не для обхода средств защиты.
