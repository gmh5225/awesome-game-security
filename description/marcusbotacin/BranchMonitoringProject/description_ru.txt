Этот проект — исследовательский фреймворк для мониторинга выполнения программ с использованием аппаратных функций Intel Last Branch Record (LBR) и Branch Trace Store (BTS). Он собирает трассировки выполнения на уровне ветвлений у работающих программ через доступ в режиме ядра к регистрам записи ветвлений CPU, обеспечивая детальный мониторинг потока управления без программной инструментации. Драйвер ядра на C и пользовательские инструменты обеспечивают сбор и анализ трассировок ветвлений. Проект предназначен для исследователей безопасности, изучающих аппаратно-поддерживаемый мониторинг выполнения для анализа вредоносного ПО и проверки целостности.
