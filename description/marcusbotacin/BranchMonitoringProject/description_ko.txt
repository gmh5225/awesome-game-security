이 프로젝트는 Intel의 Last Branch Record (LBR) 및 Branch Trace Store (BTS) 하드웨어 기능을 사용해 프로그램 실행을 모니터링하는 연구 프레임워크입니다. 커널 모드에서 CPU 분기 기록 레지스터에 접근하여 실행 중인 프로그램의 분기 수준 실행 트레이스를 수집하며, 소프트웨어 계측 없이도 세밀한 제어 흐름 모니터링이 가능합니다. C 커널 드라이버와 사용자 모드 도구가 분기 트레이스 수집 및 분석을 제공합니다. 악성코드 분석과 무결성 검사를 위한 하드웨어 지원 실행 모니터링을 연구하는 보안 연구자를 대상으로 합니다.
