Dieses Projekt ist ein Forschungsrahmenwerk zur Überwachung der Programmausführung mithilfe der Hardware-Funktionen Last Branch Record (LBR) und Branch Trace Store (BTS) von Intel. Es erfasst Ausführungs-Traces auf Branch-Ebene von laufenden Programmen über kernelmodus-Zugriff auf die CPU-Branch-Aufzeichnungsregister und ermöglicht so eine fein granulierte Kontrolle des Kontrollflusses ohne Software-Instrumentierung. Der C-Kernel-Treiber und die User-Mode-Werkzeuge stellen die Erfassung und Analyse von Branch-Traces bereit. Es richtet sich an Sicherheitsforscher, die hardwaregestützte Ausführungsüberwachung für Malware-Analyse und Integritätsprüfung untersuchen.
