Questo progetto è un framework di ricerca per monitorare l'esecuzione dei programmi usando le funzionalità hardware Last Branch Record (LBR) e Branch Trace Store (BTS) di Intel. Cattura tracce di esecuzione a livello di branch da programmi in esecuzione tramite accesso in modalità kernel ai registri di registrazione dei branch della CPU, consentendo un monitoraggio fine del flusso di controllo senza strumentazione software. Il driver kernel in C e gli strumenti in modalità utente forniscono la raccolta e l'analisi delle tracce di branch. È rivolto ai ricercatori di sicurezza che studiano il monitoraggio dell'esecuzione assistito dall'hardware per l'analisi di malware e la verifica dell'integrità.
