Este proyecto es un marco de investigación para monitorizar la ejecución de programas mediante las funciones de hardware Last Branch Record (LBR) y Branch Trace Store (BTS) de Intel. Captura trazas de ejecución a nivel de ramas de programas en ejecución mediante acceso en modo kernel a los registros de grabación de ramas de la CPU, lo que permite una monitorización detallada del flujo de control sin instrumentación de software. El controlador de kernel en C y las herramientas en modo usuario proporcionan la recolección y el análisis de trazas de ramas. Está dirigido a investigadores de seguridad que estudian la monitorización de ejecución asistida por hardware para el análisis de malware y la verificación de integridad.
