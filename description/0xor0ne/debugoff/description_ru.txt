Этот проект — библиотека на Rust для экспериментов по антианализу и антиотладке в Linux.
Она реализует прямые системные вызовы без зависимостей от libc и добавляет обфускацию на уровне syscall, чтобы усложнить статический анализ.
Также выполняются цепочки рандомизированных проверок на основе ptrace, которые проверяют ожидаемое поведение и завершают выполнение при обнаружении вмешательства.
Основной сценарий использования — исследования в области безопасности по защите бинарных файлов от реверс-инжиниринга в средах Linux.
