Этот проект — инструмент дампа памяти на Rust, ориентированный на получение дампов LSASS с менее очевидными признаками обнаружения.
Вместо прямого открытия LSASS он перечисляет и дублирует существующие дескрипторы через нативные Windows API объектов и системной информации.
Поддерживаются XOR-защищённый вывод дампа, опциональная HTTP-загрузка и режим расшифровки для восстановления данных дампа.
Реализация построена на Rust с паттернами низкоуровневых вызовов Windows API.
Предназначен для контролируемых исследований в offensive security и тестов обхода обнаружения.
