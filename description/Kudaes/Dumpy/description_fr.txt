Ce projet est un outil de dump mémoire écrit en Rust, axé sur l'obtention de dumps LSASS avec des signaux de détection évidents réduits.
Au lieu d'ouvrir LSASS directement, il énumère et duplique des handles existants via les API natives Windows sur les objets et les informations système.
Il prend en charge une sortie de dump protégée par XOR, un téléversement HTTP optionnel et un mode de déchiffrement pour restaurer les données dumpées.
L'implémentation s'appuie sur Rust avec des motifs d'appel d'API Windows de bas niveau.
Il est destiné à la recherche en sécurité offensive contrôlée et aux tests d'évasion de détection.
