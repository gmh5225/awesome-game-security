Este proyecto es una herramienta de volcado de memoria escrita en Rust, centrada en obtener dumps de LSASS con señales de detección obvias reducidas.
En lugar de abrir LSASS directamente, enumera y duplica handles existentes mediante las API nativas de objetos e información del sistema de Windows.
Admite salida de dump protegida con XOR, carga HTTP opcional y un modo de descifrado para restaurar los datos volcados.
La implementación se centra en Rust con patrones de invocación de API de Windows de bajo nivel.
Está pensada para investigación ofensiva de seguridad controlada y pruebas de evasión de detección.
