Questo progetto è uno strumento di memory dump basato su Rust, incentrato sull'ottenimento di dump LSASS con segnali di rilevamento evidenti ridotti.
Invece di aprire LSASS direttamente, enumera e duplica handle esistenti usando le API native di Windows su oggetti e informazioni di sistema.
Supporta output di dump protetto con XOR, upload HTTP opzionale e una modalità di decrittazione per ripristinare i dati dump.
L'implementazione è incentrata su Rust con pattern di invocazione delle API Windows a basso livello.
È destinato alla ricerca di sicurezza offensiva controllata e ai test di evasione del rilevamento.
