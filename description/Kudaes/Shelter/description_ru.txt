Этот проект — Rust-crate для обфускации сна: он шифрует полезную нагрузку в памяти и возобновляет выполнение через ROP.
Включает поддержку AES-128, шифрование всего PE и временное снятие прав на выполнение на время сна.
Архитектура избегает схем с сильной опорой на таймеры и APC и сочетает подмену стека с непрямыми системными вызовами для скрытного управления выполнением.
Основная реализация написана на Rust с ассемблерными заглушками для низкоуровневого поведения.
В основном применяется в исследованиях продвинутых red-team инструментов и экспериментах по уклонению в памяти.
