Этот проект — реализация на Rust техник call stack spoofing, вдохновлённых SilentMoonWalk.
Он предоставляет макросы для вызова обычных функций и indirect syscall при сохранении стабильных поддельных стековых трасс.
Крейт поддерживает передачу аргументов, захват возвращаемого значения и повторяемый цепочечный spoofing без линейного роста стека.
Он сочетает код на Rust со вспомогательными процедурами на ассемблере для управления низкоуровневым потоком выполнения в Windows.
Основное применение — низкоуровневые исследования в области offensive security, сосредоточенные на техниках уклонения от анализа call stack.
