BWSR — кроссплатформенная библиотека inline-хуков для устройств Arm64 и Arm64e. Реализована на C и ориентирована на iOS, Android, Linux и macOS для перехвата кода во время выполнения. Проект сосредоточен на низкоуровневых примитивах патчинга и включает пути сборки для ряда мобильных и настольных платформ. Предназначен для исследователей безопасности и системных разработчиков, которым нужна переносимая инструментация на основе хуков в нативных бинарниках.
