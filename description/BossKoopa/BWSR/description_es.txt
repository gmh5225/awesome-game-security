BWSR es una biblioteca de inline hooking multiplataforma para dispositivos Arm64 y Arm64e. Está implementada en C y se orienta a iOS, Android, Linux y macOS para la interceptación de código en tiempo de ejecución. El proyecto se centra en primitivas de parcheo de bajo nivel e incluye rutas de compilación para varias plataformas móviles y de escritorio. Está pensada para investigadores de seguridad y desarrolladores de sistemas que necesitan instrumentación portable basada en hooks en binarios nativos.
