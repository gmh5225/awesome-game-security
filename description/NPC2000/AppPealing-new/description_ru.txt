Этот проект — модуль Android Xposed, отключающий защиту, предоставляемую Inka AppSealing.
Он нацелен на обход обнаружения root и читов и может дампить расшифрованное содержимое Dex для проверки и отладки.
Модуль использует Java для интеграции с Xposed и нативные компоненты хукинга на C++ с Dobby в рабочем процессе LSPosed и Magisk.
В первую очередь предназначен для тестирования безопасности мобильных приложений и игр, реверс-инжиниринга и анализа средств защиты от вмешательства.
