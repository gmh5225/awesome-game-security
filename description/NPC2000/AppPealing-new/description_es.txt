Este proyecto es un módulo Android Xposed que desactiva las protecciones proporcionadas por Inka AppSealing.
Se centra en eludir la detección de root y de trampas, y puede volcar el contenido Dex descifrado para su inspección y depuración.
El módulo usa Java para la integración con Xposed y componentes nativos de hooking en C++ con Dobby, en un flujo de trabajo LSPosed y Magisk.
Está orientado principalmente a pruebas de seguridad de aplicaciones y juegos móviles, ingeniería inversa y análisis antimanipulación.
