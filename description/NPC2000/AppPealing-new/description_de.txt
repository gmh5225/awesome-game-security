Dieses Projekt ist ein Android-Xposed-Modul, das die von Inka AppSealing bereitgestellten Schutzmechanismen deaktiviert.
Es zielt auf das Umgehen von Root- und Cheat-Erkennung ab und kann entschlüsselte Dex-Inhalte zum Prüfen und Debuggen ausgeben.
Das Modul nutzt Java für die Xposed-Integration sowie native Hooking-Komponenten in C++ mit Dobby in einem LSPosed- und Magisk-Workflow.
Es richtet sich vor allem an Sicherheitstests für mobile Apps und Spiele, Reverse Engineering und Anti-Tamper-Analysen.
