NovaHypervisor es un hipervisor Intel x64 defensivo para proteger regiones sensibles de la memoria del kernel. Está diseñado para mitigar ataques al kernel como BYOVD aplicando políticas de acceso a memoria y protegiendo direcciones seleccionadas con controles de lectura, escritura o ejecución. El proyecto está implementado en C++ y ensamblador como un controlador de kernel de Windows, con un cliente acompañante para añadir y quitar protecciones y herramientas de apoyo para el registro. Se orienta principalmente a la investigación de anti-trampas y defensa de endpoints en seguridad del kernel virtualizado.
