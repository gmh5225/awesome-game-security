EtwLeakKernel es una prueba de concepto que filtra direcciones de memoria del kernel mediante rastros de pila de ETW. Inicia el consumo de ETW, solicita datos de pila a los proveedores y analiza la salida de eventos para recuperar punteros del kernel. La implementación está en C++ para Windows y requiere privilegios de administrador para empezar a consumir eventos de los proveedores. Está pensada para investigación de explotación y para estudiar rutas de exposición de direcciones del kernel.
