EtwLeakKernel — это proof-of-concept, который утекает адреса памяти ядра через трассировки стека ETW. Он запускает потребление ETW, запрашивает данные стека у провайдеров и разбирает вывод событий, чтобы восстановить указатели ядра. Реализация написана на C++ для Windows и требует прав администратора для начала потребления событий провайдеров. Предназначен для исследования эксплуатации и изучения путей раскрытия адресов ядра.
