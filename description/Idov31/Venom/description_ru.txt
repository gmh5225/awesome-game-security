Venom — это однозаголовочная C++-библиотека для скрытой сетевой коммуникации в Windows. Вместо открытия очевидного сокета в процессе вызывающей стороны она создаёт скрытый отсоединённый процесс браузера и повторно использует один из его сокетов для операций отправки и приёма. Реализация опирается на внутренности Win32 и Winsock, включая техники обнаружения и дублирования дескрипторов. Она предназначена для исследований сетевого взаимодействия с упором на уклонение в контексте наступательной безопасности и скрытных инструментов.
