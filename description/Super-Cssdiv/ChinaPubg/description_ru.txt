Этот проект — нативный Android-инструментарий для реверс-инжиниринга, связанного с PUBG, и экспериментов по внешней манипуляции игрой. Он сочетает код C/C++ NDK с несколькими компонентами хукинга и инъекции, включая фреймворки inline hook, утилиты инъекции на основе ptrace и инструменты для работы с памятью. В кодовой базе также интегрированы рендеринг на базе ImGui и вспомогательные модули для оффсетов, данных карт и патчинга во время выполнения. Проект в первую очередь предназначен для исследователей игровой безопасности, изучающих поведение мобильных античитов, архитектуру читов и поверхности обнаружения.
