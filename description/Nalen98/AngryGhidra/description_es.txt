Este proyecto es una extensión de Ghidra que integra la ejecución simbólica de angr en flujos de trabajo interactivos de ingeniería inversa.
Permite a los analistas definir direcciones start, find y avoid, lanzar exploración simbólica y aplicar de nuevo los bytes parcheados al estado de análisis.
El plugin usa Java para la integración con Ghidra y Python con angr y claripy para automatizar el análisis.
Está pensado para resolver CTF, investigar malware y binarios, y para tareas generales de ingeniería inversa de software o juegos.
