Dieses Projekt ist eine Ghidra-Erweiterung, die angr Symbolic Execution in interaktive Reverse-Engineering-Workflows einbettet.
Analysten können Start-, Find- und Avoid-Adressen setzen, die symbolische Exploration starten und gepatchte Bytes wieder auf den Analysezustand anwenden.
Das Plugin nutzt Java für die Ghidra-Integration sowie Python mit angr und claripy zur Analyseautomatisierung.
Es ist für CTF-Lösungen, Malware- und Binary-Forschung sowie allgemeines Software- oder Game-Reverse-Engineering gedacht.
