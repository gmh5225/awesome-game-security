Этот проект — расширение Ghidra, которое встраивает символьное выполнение angr в интерактивные рабочие процессы реверс-инжиниринга.
Оно позволяет аналитикам задавать адреса start, find и avoid, запускать символьное исследование и применять пропатченные байты обратно к состоянию анализа.
Плагин использует Java для интеграции с Ghidra и Python с angr и claripy для автоматизации анализа.
Предназначено для решения CTF, исследования вредоносного ПО и бинарников, а также общего реверс-инжиниринга ПО или игр.
