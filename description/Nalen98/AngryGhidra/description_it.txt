Questo progetto è un'estensione di Ghidra che incorpora l'esecuzione simbolica di angr nei flussi di lavoro interattivi di reverse engineering.
Consente agli analisti di impostare indirizzi start, find e avoid, avviare l'esplorazione simbolica e applicare di nuovo i byte patchati allo stato di analisi.
Il plugin usa Java per l'integrazione con Ghidra e Python con angr e claripy per l'automazione dell'analisi.
È pensato per risolvere CTF, ricercare malware e binari, e per attività generali di reverse engineering di software o giochi.
