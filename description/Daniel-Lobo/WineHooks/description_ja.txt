WineHooks は、レガシーな PC タイトルにフックを注入するゲーム互換・拡張フレームワークです。DirectX と OpenGL のインターセプト、解像度スケーリング、テクスチャ差し替え、オーディオと入力の修正、そして現代システム向けの多数のゲーム固有パッチプロファイルを組み合わせています。リポジトリは C または C++ コンポーネントと自動化・設定資産を混在させており、ネイティブ Windows と Wine 経由の Linux の両方で動作するよう構築されています。主に、古いゲームの挙動を復元または拡張するための実用的なフックを必要とするモッダーやリバースエンジニアリング実務者に利用されています。
