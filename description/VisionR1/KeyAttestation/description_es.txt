Este proyecto es una aplicación de prueba para Android destinada a generar, analizar y verificar evidencias de atestación de claves (key attestation). La app está construida con Java y Kotlin y, en este fork, admite guardar y cargar cadenas de certificados, además del manejo de listas de revocación locales y remotas. Las funciones adicionales del fork incluyen soporte de atestación RSA, opciones de idioma y tema, y controles de visualización de certificados orientados a la privacidad. Su principal caso de uso es la validación de seguridad móvil, las pruebas de integridad de dispositivos y la investigación sobre atestación.
