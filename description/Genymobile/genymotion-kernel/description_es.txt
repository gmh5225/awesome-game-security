Este proyecto es el código fuente del kernel de Linux utilizado por el emulador Android Genymotion. Contiene un árbol completo del kernel con configuraciones y parches específicos de Android, adaptados al entorno de dispositivo virtual de Genymotion. Resulta especialmente útil para desarrolladores de emuladores Android y para investigadores de seguridad móvil que estudian el comportamiento a nivel de kernel en entornos Android virtualizados.
