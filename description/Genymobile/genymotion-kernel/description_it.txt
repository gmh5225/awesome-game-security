Questo progetto è il codice sorgente del kernel Linux usato dall'emulatore Android Genymotion. Contiene un albero completo del kernel con configurazioni e patch specifiche per Android, adattate all'ambiente del dispositivo virtuale Genymotion. È soprattutto utile per gli sviluppatori di emulatori Android e per i ricercatori di sicurezza mobile che studiano il comportamento a livello di kernel in ambienti Android virtualizzati.
