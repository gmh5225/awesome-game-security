scrcpy è uno strumento multipiattaforma che rispecchia e controlla dispositivi Android da desktop tramite USB o TCP/IP. È implementato principalmente in C e Java e punta a prestazioni elevate, bassa latenza e un'impronta minima sul dispositivo, senza accesso root. Le funzioni principali includono inoltro audio, registrazione, supporto per display virtuali, simulazione di input HID e ampia compatibilità desktop. L'uso tipico riguarda test dei dispositivi, automazione e flussi di debug di app o giochi mobile.
