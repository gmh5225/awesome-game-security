OpenGraphic est une base de connaissances sélectionnée de ressources open source sur les graphismes et les moteurs de jeu. Le dépôt est maintenu sous forme d'index Markdown qui regroupent les projets par moteurs, techniques de rendu, sujets d'optimisation, outils et tutoriels. Il agrège des références issues des écosystèmes vendeurs et des frameworks communautaires pour offrir un large panorama technique. Son principal cas d'usage est la veille technologique et l'étude pour les programmeurs graphiques, les développeurs de moteurs et les chercheurs.
