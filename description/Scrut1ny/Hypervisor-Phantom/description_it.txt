Questo progetto è un toolkit di automazione per configurare ambienti di virtualizzazione Linux con patch orientate all’anti-rilevamento. È guidato principalmente da script Bash e organizza moduli di setup per QEMU, EDK2, patching del kernel, passthrough VFIO e helper di deployment. Il repository include anche set di patch mantenuti e script ausiliari per la gestione degli identificatori di host e guest. È usato soprattutto da ricercatori che costruiscono laboratori VM riproducibili per test sul comportamento degli anti-cheat e sperimentazioni di sicurezza sulla virtualizzazione.
