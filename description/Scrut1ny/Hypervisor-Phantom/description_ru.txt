Этот проект — набор инструментов автоматизации для настройки сред виртуализации Linux с патчами, ориентированными на противодействие обнаружению. Он в основном управляется Bash-скриптами и организует модули установки для QEMU, EDK2, патчинга ядра, VFIO passthrough и вспомогательных средств развёртывания. Репозиторий также включает поддерживаемые наборы патчей и вспомогательные скрипты для обработки идентификаторов хоста и гостевой системы. Им в первую очередь пользуются исследователи, создающие воспроизводимые VM-лаборатории для тестирования поведения античитов и экспериментов по безопасности виртуализации.
