Este proyecto es un kit de herramientas de automatización para configurar entornos de virtualización Linux con parches orientados a la anti-detección. Se basa principalmente en scripts Bash y organiza módulos de instalación para QEMU, EDK2, parcheo del kernel, passthrough VFIO y ayudas de despliegue. El repositorio también incluye conjuntos de parches mantenidos y scripts auxiliares para el manejo de identificadores de host y guest. Lo usan sobre todo investigadores que construyen laboratorios de VM reproducibles para probar el comportamiento de anti-cheats y realizar experimentos de seguridad en virtualización.
