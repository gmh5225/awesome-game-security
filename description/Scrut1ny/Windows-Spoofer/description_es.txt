Este proyecto es un kit de herramientas de código abierto para spoofing y limpieza en Windows 10 y 11. Combina módulos Batch y PowerShell para cambiar identificadores del sistema seleccionados, ajustar valores relacionados con la red y borrar artefactos de rastro habituales. El flujo de trabajo integra utilidades externas para tareas como la modificación del volume ID y cambios relacionados con SMBIOS, con notas detalladas sobre el soporte de plataformas y los límites operativos. Está orientado a escenarios de investigación en seguridad de juegos y anti-cheat en los que los analistas necesitan estudiar el comportamiento de fingerprinting de hardware y software en entornos de prueba controlados.
