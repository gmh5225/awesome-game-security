本プロジェクトは、Windows 10 / 11 向けのオープンソースなスプーフィングおよびクリーンアップ用ツールキットです。Batch と PowerShell モジュールを組み合わせ、選択したシステム識別子の変更、ネットワーク関連値の調整、一般的な痕跡アーティファクトの消去を行います。ワークフローには volume ID や SMBIOS 関連の変更などの外部ユーティリティが組み込まれており、プラットフォーム対応と運用上の制限に関する詳細な注記が付いています。ゲームセキュリティおよびアンチチート研究において、分析者が制御されたテスト環境でハードウェアおよびソフトウェアのフィンガープリンティング挙動を調べる場面を想定しています。
