Этот проект — открытый набор инструментов для спуфинга и очистки в Windows 10 и 11. Он объединяет модули Batch и PowerShell для изменения выбранных системных идентификаторов, корректировки сетевых значений и очистки типичных артефактов следов. Рабочий процесс интегрирует внешние утилиты для таких задач, как изменение volume ID и модификации, связанные с SMBIOS, с подробными примечаниями о поддержке платформ и эксплуатационных ограничениях. Он ориентирован на сценарии исследований в области игровой безопасности и античита, где аналитикам нужно изучать поведение аппаратного и программного фингерпринтинга в контролируемых тестовых средах.
