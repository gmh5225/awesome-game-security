Questo progetto è un framework di hooking runtime dei metodi per C# in ambienti Unity e Mono o IL2CPP. Modifica il codice nativo JIT o AOT in memoria invece di patchare i file su disco, ed è progettato per preservare il comportamento di debug e gli stack trace. L’implementazione è incentrata sul C# con un piccolo componente nativo e supporta più generazioni di Unity e varianti del runtime .NET, sia nell’editor sia sui dispositivi. Il caso d’uso principale è l’instrumentazione Unity, il reverse engineering e la ricerca di sicurezza nei giochi che richiede la sostituzione controllata di funzioni.
