Ce projet est un framework d’accrochage (hooking) de méthodes à l’exécution pour le C# dans les environnements Unity et Mono ou IL2CPP. Il modifie le code natif JIT ou AOT en mémoire plutôt que de patcher des fichiers sur le disque, et est conçu pour préserver le comportement de débogage et les traces de pile. L’implémentation s’appuie principalement sur le C# avec un petit composant natif, et prend en charge plusieurs générations d’Unity ainsi que diverses variantes du runtime .NET, aussi bien dans l’éditeur que sur les appareils cibles. Son principal cas d’usage est l’instrumentation Unity, la rétro-ingénierie et la recherche en sécurité des jeux nécessitant un remplacement contrôlé de fonctions.
