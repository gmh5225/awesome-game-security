本プロジェクトは、Unity および Mono または IL2CPP 環境向けの C# ランタイムメソッドフッキングフレームワークです。ディスク上のファイルをパッチするのではなく、メモリ上の JIT または AOT ネイティブコードを改変してフックを行い、デバッグ挙動とスタックトレースの維持を重視して設計されています。実装は C# を中心に少量のネイティブコンポーネントを併用し、複数世代の Unity と各種 .NET ランタイムバリアントを、エディタおよびデバイス双方でサポートします。主な用途は Unity の計装、リバースエンジニアリング、および制御された関数置換を必要とするゲームセキュリティ研究です。
