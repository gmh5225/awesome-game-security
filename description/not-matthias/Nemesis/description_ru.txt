Этот проект посвящён дампу процессов с использованием возможностей пространства ядра.
Некоторые из этих программ доступны бесплатно, но, разумеется, они уступают платным.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования ядра Windows.
