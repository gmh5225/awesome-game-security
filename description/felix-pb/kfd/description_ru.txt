Этот проект — kfd, фреймворк эксплуатации файловых дескрипторов на уровне ядра для iOS/macOS, предоставляющий примитивы произвольного чтения/записи ядра через уязвимости ядра XNU. Он цепочкой объединяет несколько ошибок, чтобы получить стабильный примитив доступа к памяти ядра, пригодный для джейлбрейка, выхода из песочницы или исследований безопасности. Реализация на C нацелена на конкретные версии iOS/macOS с известными уязвимостями ядра. Предназначен для исследователей безопасности iOS и разработчиков джейлбрейков, изучающих эксплуатацию ядра XNU.
