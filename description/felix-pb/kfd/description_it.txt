Questo progetto è kfd, un framework di exploit di file descriptor a livello kernel per iOS/macOS che fornisce primitive arbitrarie di lettura/scrittura del kernel tramite vulnerabilità del kernel XNU. Concatena più bug per ottenere una primitiva stabile di accesso alla memoria del kernel utilizzabile per jailbreak, escape dalla sandbox o ricerca sulla sicurezza. L'implementazione in C mira a versioni specifiche di iOS/macOS con vulnerabilità del kernel note. È rivolto a ricercatori di sicurezza iOS e sviluppatori di jailbreak che studiano lo sfruttamento del kernel XNU.
