Este proyecto es kfd, un framework de exploits de descriptores de archivos a nivel de kernel para iOS/macOS que proporciona primitivas arbitrarias de lectura/escritura del kernel mediante vulnerabilidades del kernel XNU. Encadena múltiples bugs para lograr una primitiva estable de acceso a memoria del kernel utilizable para jailbreak, escape de sandbox o investigación de seguridad. La implementación en C apunta a versiones específicas de iOS/macOS con vulnerabilidades de kernel conocidas. Está dirigido a investigadores de seguridad iOS y desarrolladores de jailbreak que estudian la explotación del kernel XNU.
