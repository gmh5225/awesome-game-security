Hidden es un controlador del kernel de Windows con una interfaz en modo usuario para ocultar artefactos del sistema seleccionados.
El código en C y C++ permite ocultar archivos, directorios, claves del registro y procesos, y también puede proteger procesos elegidos frente a interferencias.
Incluye componentes CLI y de biblioteca para configurar reglas y excepciones, además de scripts de compilación basados en WDK.
El proyecto está orientado a laboratorios de ingeniería inversa y escenarios de investigación de seguridad que requieren un enmascaramiento controlado del entorno.
