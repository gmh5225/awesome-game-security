Hidden è un driver del kernel Windows con un’interfaccia in modalità utente per occultare artefatti di sistema selezionati.
Il codice C e C++ supporta l’occultamento di file, directory, chiavi di registro e processi, e può anche proteggere processi scelti da interferenze.
Include componenti CLI e libreria per configurare regole ed eccezioni, oltre a script di build basati su WDK.
Il progetto è pensato per laboratori di reverse engineering e scenari di ricerca sulla sicurezza che richiedono un mascheramento controllato dell’ambiente.
