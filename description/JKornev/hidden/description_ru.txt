Hidden — это драйвер ядра Windows с пользовательским интерфейсом для сокрытия выбранных системных артефактов.
Код на C и C++ поддерживает скрытие файлов, каталогов, ключей реестра и процессов, а также защиту выбранных процессов от вмешательства.
В проект входят CLI и библиотечные компоненты для настройки правил и исключений, а также скрипты сборки на базе WDK.
Проект предназначен для лабораторий реверс-инжиниринга и исследований безопасности, где требуется контролируемое маскирование окружения.
