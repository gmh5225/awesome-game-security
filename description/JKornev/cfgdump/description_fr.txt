cfgdump est une extension WinDbg pour inspecter la couverture Control Flow Guard dans l'espace d'adressage d'un processus.
Implémentée en C++, elle fournit des commandes pour afficher les cartes CFG, interroger des plages spécifiques et lister les régions protégées.
L'extension aide les analystes à comprendre quelles zones mémoire sont protégées par les bits CFG et comment ces protections sont organisées.
Elle est utile pour la recherche en développement d'exploits, la validation du durcissement binaire et les workflows de débogage Windows bas niveau.
