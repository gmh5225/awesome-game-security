Dieses Projekt konzentriert sich auf ein Plugin zum Laden von MachO-Kernelcache- und dSYM-Dateien.
Das Plugin kann Symbole und Typinformationen aus mehreren im KDK enthaltenen dSYM-Dateien in den entsprechenden, in Binary Ninja geöffneten Kernelcache laden.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Binary-Ninja-Plugins untersuchen.
