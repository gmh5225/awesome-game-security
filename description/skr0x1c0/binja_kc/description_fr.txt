Ce projet se concentre sur un plugin permettant de charger des fichiers MachO kernelcache et dSYM.
Ce plugin peut charger les symboles et les informations de type issus de plusieurs fichiers dSYM contenus dans le KDK vers le kernelcache correspondant ouvert dans Binary Ninja.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / plugins Binary Ninja.
