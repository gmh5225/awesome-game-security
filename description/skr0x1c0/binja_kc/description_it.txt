Questo progetto si concentra su un plugin per il caricamento di file MachO kernelcache e dSYM.
Il plugin può caricare simboli e informazioni sui tipi da più file dSYM contenuti nel KDK nel kernelcache corrispondente aperto in Binary Ninja.
È principalmente utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin di Binary Ninja.
