Этот проект — бинарный пакер на чистом C для исполняемых файлов ELF и PE. Он поддерживает несколько методов упаковки: вставку секций, техники code cave и заражение text-секции, а также варианты шифрования на основе XOR и AES. В кодовой базе есть низкоуровневые компоненты загрузчика и ассемблера для распаковки во время выполнения. В основном используется для практики реверс-инжиниринга, экспериментов с обфускацией и оборонительных исследований упакованных бинарных файлов.
