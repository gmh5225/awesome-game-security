Este proyecto es un empaquetador binario escrito en C puro para ejecutables ELF y PE. Admite múltiples métodos de empaquetado, como inserción de secciones, técnicas de code cave e infección de la sección text, con opciones de cifrado basadas en XOR y AES. El código incluye componentes de cargador de bajo nivel y ensamblador para el comportamiento de desempaquetado en tiempo de ejecución. Se usa principalmente para práctica de ingeniería inversa, experimentos de ofuscación e investigación defensiva sobre binarios empaquetados.
