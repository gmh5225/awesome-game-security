SparkEngine is a cross-platform C++ game engine with a full runtime, editor, build tools, and sample game modules for genres such as ARPG, MMO, and MMOFPS. It includes modern rendering backends (DirectX 12, Vulkan, and Metal), HLSL/GLSL shader pipelines with PBR and ray-tracing paths, Jolt Physics, ECS systems, AI navigation, and AngelScript plus visual scripting. Networking covers dedicated servers, replication, client prediction, lag compensation, encryption, and packet validation, with additional security-oriented pieces such as memory integrity checks. The project targets game developers building networked titles and researchers interested in engine-level multiplayer and client integrity design.
