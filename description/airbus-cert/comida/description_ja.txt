comida は、COM コンポーネントを利用する Windows バイナリの解析を強化する IDA Pro プラグインです。既知の COM GUID 参照をスキャンし、レジストリのメタデータと照合することで、関連するクラスやインターフェースの特定を支援します。Hex-Rays 利用者向けには、CoCreateInstance、CoGetCallContext、QueryInterface などの API 周辺で型推論を行い、デコンパイル結果を整理します。本プラグインは、COM 中心の迅速なトリアージと、より深い Windows 内部の可視性を必要とするマルウェア解析者およびリバースエンジニアを対象としています。
