comida — плагин для IDA Pro, улучшающий анализ Windows-бинарников, использующих компоненты COM. Он сканирует известные ссылки на COM GUID и сопоставляет их с метаданными реестра, помогая аналитикам выявлять связанные классы и интерфейсы. Для пользователей Hex-Rays плагин также выполняет вывод типов вокруг API вроде CoCreateInstance, CoGetCallContext и QueryInterface, чтобы очистить декомпилированный вывод. Плагин предназначен для аналитиков вредоносного ПО и специалистов по реверс-инжинирингу, которым нужна более быстрая COM-ориентированная triage-оценка и более глубокая видимость внутренностей Windows.
