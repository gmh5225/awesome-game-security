comida는 COM 구성 요소를 사용하는 Windows 바이너리 분석을 개선하는 IDA Pro 플러그인입니다. 알려진 COM GUID 참조를 스캔하고 레지스트리 메타데이터와 연관 지어, 분석가가 관련 클래스와 인터페이스를 식별하도록 돕습니다. Hex-Rays 사용자를 위해서는 CoCreateInstance, CoGetCallContext, QueryInterface 같은 API 주변에서 타입 추론을 수행하여 디컴파일 결과를 정리합니다. 이 플러그인은 COM 중심의 빠른 분류·검토와 더 깊은 Windows 내부 가시성이 필요한 멀웨어 분석가와 리버스 엔지니어를 대상으로 합니다.
