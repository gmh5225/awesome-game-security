Этот проект — драйвер ядра Windows, ориентированный на чтение/запись памяти процессов для низкоуровневых инструментов. Он написан на C++ и предоставляет процедуры поиска базы/экспорта модулей, доступа к памяти ядра и записи в защищённые области памяти. В заметках указана загрузка через mapper и упоминается контекст хуков, связанный с dxgkrnl, для работы. В основном используется для исследований игровой безопасности, связанных с доступом к памяти через драйвер и техниками обхода античита.
