Этот проект генерирует заголовки и обёртки C++ SDK из сборок Unity IL2CPP DummyDll для игр, ориентированных на Android. Он сочетает C#-генератор на базе dnlib с вспомогательными C++-заголовками времени выполнения для разрешения классов, методов, полей и экспортов il2cpp в нативном коде. Полученный результат позволяет разработчикам напрямую подключать сгенерированные пространства имён и классы и взаимодействовать с игровой логикой из внешних модулей. В основном используется в рабочих процессах реверс-инжиниринга Unity и исследований в области безопасности игр.
