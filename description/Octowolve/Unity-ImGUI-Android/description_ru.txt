Этот проект — нативный шаблон Android Unity для отрисовки меню ImGui через перехваченные пути графики и ввода. Он использует C++ с хуками Dobby для перехвата eglSwapBuffers и внедрения ввода Unity, чтобы оверлеи и обработка касаний или клавиш работали внутри процесса игры. В репозитории также есть туториал по поиску сигнатур nativeInjectEvent в libunity с помощью IDA и SigMaker. Нацелен на прототипирование мод-меню и исследование реверс-инжиниринга мобильных игр.
