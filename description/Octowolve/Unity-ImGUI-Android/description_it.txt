Questo progetto è un template nativo Android Unity per renderizzare un menu ImGui tramite percorsi grafici e di input agganciati. Usa C++ con hook Dobby per intercettare eglSwapBuffers e l’iniezione di input di Unity, così overlay e gestione di touch o tasti possono essere eseguiti nel processo di gioco. Il repository include anche un tutorial per trovare le signature di nativeInjectEvent in libunity con IDA e SigMaker. È pensato per il prototipaggio di menu mod e la ricerca di reverse engineering su giochi mobile.
