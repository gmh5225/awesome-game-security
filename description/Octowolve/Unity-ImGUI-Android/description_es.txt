Este proyecto es una plantilla nativa de Android Unity para renderizar un menú ImGui mediante rutas de gráficos y entrada enganchadas. Usa C++ con hooks de Dobby para interceptar eglSwapBuffers y la inyección de entrada de Unity, de modo que los overlays y el manejo táctil o de teclas puedan ejecutarse dentro del proceso del juego. El repositorio también incluye un tutorial para encontrar firmas de nativeInjectEvent en libunity con IDA y SigMaker. Está orientado al prototipado de menús de mods y a la investigación de ingeniería inversa de juegos móviles.
