Инжектор DLL для Windows, выполняющий manual mapping в целевые процессы с разрешением импортов и релокаций без использования LoadLibrary. Включает UI-компонент для выбора процесса по имени и библиотечный компонент (MMap), отвечающий за низкоуровневое копирование секций PE, исправление релокаций и разрешение импортов.
