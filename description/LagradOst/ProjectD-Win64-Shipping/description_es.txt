Este proyecto es un dump del SDK de Unreal Engine para Veiled Experts (Project D), con cabeceras C++ de clases y estructuras generadas para agentes, armas, vehículos, sistemas de UI y subsistemas del motor del juego. El dump incluye jerarquías de clases completas, offsets de miembros y tablas de funciones virtuales. Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian los internos de Unreal Engine y las metodologías de dumping de SDK.
