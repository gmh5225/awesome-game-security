Этот проект — Windows proof-of-concept набор тестов для антианализа, моделирующий множество приёмов уклонения вредоносного ПО. Кодовая база в основном на C++ и реализует антиотладку, анти-VM, антидампинг, антидизассемблирование и проверки на основе тайминга через интерфейс командной строки. Включено широкое покрытие обнаружения окружения для платформ и инструментов вроде VirtualBox, VMware, QEMU, Wine, а также типичных сценариев анализа. Основной сценарий использования — проверка видимости песочниц, EDR и антивирусных решений в отношении реальных техник уклонения.
