Dieses Projekt ist eine Windows-Proof-of-Concept-Anti-Analyse-Testsuite, die zahlreiche Malware-Evasionsverhalten simuliert. Die Codebasis ist hauptsächlich C++ und implementiert Anti-Debugging-, Anti-VM-, Anti-Dumping-, Anti-Disassembly- sowie timingbasierte Prüfungen über eine Kommandozeilenschnittstelle. Sie bietet eine breite Umgebungserkennung für Plattformen und Tools wie VirtualBox, VMware, QEMU, Wine sowie gängige Analyse-Workflows. Der Hauptanwendungsfall ist die Validierung der Sichtbarkeit von Sandbox-, EDR- und Anti-Malware-Lösungen gegenüber realen Evasionstechniken.
