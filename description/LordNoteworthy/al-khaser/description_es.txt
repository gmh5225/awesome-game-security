Este proyecto es una suite de pruebas antianálisis proof-of-concept para Windows que simula numerosos comportamientos de evasión de malware. El código está principalmente en C++ e implementa comprobaciones antidepuración, anti-VM, antidumping, antidesensamblado y basadas en temporización a través de una interfaz de línea de comandos. Incluye una amplia cobertura de detección de entorno para plataformas y herramientas como VirtualBox, VMware, QEMU, Wine y flujos de análisis habituales. El caso de uso principal es validar la visibilidad de sandboxes, EDR y soluciones antimalware frente a técnicas de evasión del mundo real.
