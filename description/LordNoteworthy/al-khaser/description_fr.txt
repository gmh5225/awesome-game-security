Ce projet est une suite de tests anti-analyse proof-of-concept pour Windows qui simule de nombreux comportements d'évasion de logiciels malveillants. Le code est principalement en C++ et implémente des contrôles anti-débogage, anti-VM, anti-dumping, anti-désassemblage et basés sur le timing via une interface en ligne de commande. Il couvre largement la détection d'environnement pour des plateformes et outils tels que VirtualBox, VMware, QEMU, Wine, ainsi que les workflows d'analyse courants. L'usage principal est de valider la visibilité des sandboxes, EDR et solutions anti-malware face aux techniques d'évasion du monde réel.
