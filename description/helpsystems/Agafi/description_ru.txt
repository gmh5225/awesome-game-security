Этот проект — инструмент поиска гаджетов и построения ROP-цепочек для платформ x86.
Agafi (Advanced Gadget Finder) — инструмент поиска x86-гаджетов, полезный для нахождения гаджетов в программах, модулях и работающих процессах.
Он в первую очередь полезен исследователям игровой безопасности и инженерам-реверсерам, изучающим атакующие техники в области читов / поиска ROP.
