Этот проект — плагин, который предоставляет функциональность отладчика через HTTP-интерфейс, совместимый с MCP, для x64dbg и связанных вариантов. Он написан на C# под .NET Framework и сопоставляет действия отладчика удалённо вызываемым командам для чтения памяти, дизассемблирования, запросов регистров, разметки (labeling) и автоматизации. Архитектура включает лёгкий самохостируемый слушатель и модульную маршрутизацию команд, рассчитанную на интеграцию инструментов и быстрое расширение. Ориентирован на рабочие процессы обратной разработки с поддержкой ИИ и скриптового анализа безопасности игр.
