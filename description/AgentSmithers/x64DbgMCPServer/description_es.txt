Este proyecto es un plugin que expone la funcionalidad del depurador mediante una interfaz HTTP compatible con MCP para x64dbg y variantes relacionadas. Está construido en C# sobre .NET Framework y asigna acciones del depurador a comandos invocables de forma remota para lectura de memoria, desensamblado, consultas de registros, etiquetado y automatización. La arquitectura incluye un listener autoalojado ligero y un enrutamiento modular de comandos pensado para la integración de herramientas y una extensión rápida. Está orientado a flujos de trabajo de ingeniería inversa asistida por IA y análisis scriptado de seguridad en juegos.
