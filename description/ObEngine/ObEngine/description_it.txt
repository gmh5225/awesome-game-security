ObEngine è un motore di gioco 2D open source incentrato sullo sviluppo accessibile e sullo scripting basato su Lua. È scritto principalmente in C++ e include sistemi come la gestione di scene e oggetti di gioco, strumenti per le mappe, animazione, binding degli input, networking e supporto ai plugin. Il motore integra SFML e altre dipendenze in bundle, esponendo le funzionalità a runtime tramite API scriptabili e strumenti di progetto. Si rivolge a sviluppatori indie e a chi impara a creare giochi 2D con interni del motore personalizzabili.
