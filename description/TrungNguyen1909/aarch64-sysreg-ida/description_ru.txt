Этот проект — плагин для IDA Pro, который повышает читаемость системных регистров AArch64 при дизассемблировании. Он написан на Python и перехватывает пути отображения инструкций, чтобы загадочные кодировки MSR и SYS показывались как осмысленные имена регистров. Плагин включает встроенную базу регистров ARMv8 и может загружать дополнительные определения регистров Apple из внешнего JSON-файла. Основное применение — реверс-инжиниринг ОС и ядер ARM, где быстрая интерпретация регистров особенно важна.
