Questo progetto è un plugin per IDA Pro che migliora la leggibilità dei registri di sistema AArch64 durante il disassemblaggio. È scritto in Python e si aggancia ai percorsi di visualizzazione delle istruzioni in modo che le codifiche MSR e SYS criptiche vengano mostrate come nomi di registri significativi. Il plugin include un database di registri ARMv8 incorporato e può caricare definizioni aggiuntive di registri Apple da un file JSON esterno. Il caso d’uso principale è il reverse engineering di sistemi operativi e kernel ARM, dove un’interpretazione rapida dei registri è essenziale.
