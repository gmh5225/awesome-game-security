Este proyecto es un complemento de IDA Pro que mejora la legibilidad de los registros de sistema AArch64 durante el desensamblado. Está escrito en Python y se engancha a las rutas de visualización de instrucciones para que las codificaciones MSR y SYS crípticas se muestren como nombres de registros significativos. El complemento incluye una base de datos de registros ARMv8 integrada y puede cargar definiciones adicionales de registros Apple desde un archivo JSON externo. Su caso de uso principal es la ingeniería inversa de sistemas operativos y núcleos ARM, donde una interpretación rápida de registros es esencial.
