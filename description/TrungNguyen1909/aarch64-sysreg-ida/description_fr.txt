Ce projet est un plugin IDA Pro qui améliore la lisibilité des registres système AArch64 lors du désassemblage. Il est écrit en Python et s’accroche aux chemins d’affichage des instructions afin que les encodages MSR et SYS cryptiques apparaissent sous forme de noms de registres significatifs. Le plugin inclut une base de données de registres ARMv8 intégrée et peut charger des définitions de registres Apple supplémentaires depuis un fichier JSON externe. Son cas d’usage principal est la rétro-ingénierie des OS et noyaux ARM, où une interprétation rapide des registres est essentielle.
