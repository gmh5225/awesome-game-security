Questo progetto è un prototipo di gameplay Unreal Engine 5.2 che implementa una meccanica di fusione (fuse) tramite un actor component.
Usa sistemi C++ pronti per Blueprint per raccogliere oggetti, allineare i socket, proiettare anteprime di posizionamento e creare vincoli fisici con Chaos.
L'implementazione include matching di prossimità dei socket, controlli di clipping, comportamento di attachment vincolato e hook di visualizzazione di debug per testare le interazioni.
Il caso d'uso principale è il prototipaggio di meccaniche di gioco e l'esplorazione tecnica di sistemi di combinazione di oggetti basati sulla fisica.
