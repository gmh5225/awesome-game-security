Este proyecto es un prototipo de gameplay de Unreal Engine 5.2 que implementa una mecánica de fusión (fuse) mediante un actor component.
Utiliza sistemas en C++ listos para Blueprint para recoger objetos, alinear sockets, proyectar vistas previas de colocación y crear restricciones físicas con Chaos.
La implementación incluye coincidencia por proximidad de sockets, comprobaciones de clipping, comportamiento de adjunto restringido y hooks de visualización de depuración para probar interacciones.
Su caso de uso principal es el prototipado de mecánicas de juego y la exploración técnica de sistemas de combinación de objetos impulsados por física.
