Этот проект — прототип геймплея на Unreal Engine 5.2, реализующий механику fuse (слияния) через actor component.
Он использует системы на C++, готовые к Blueprint, для подбора объектов, выравнивания сокетов, проекции превью размещения и создания физических ограничений с Chaos.
Реализация включает сопоставление сокетов по близости, проверки клиппинга, поведение ограниченного прикрепления и хуки отладочной визуализации для тестирования взаимодействий.
Основной сценарий использования — прототипирование игровых механик и техническое исследование систем объединения объектов на основе физики.
