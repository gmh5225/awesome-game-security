Questo progetto è una libreria Rust e una CLI per trovare gli indirizzi base delle DLL caricate senza attraversare l'elenco dei moduli del PEB. Usa un approccio di stack walking a partire dai dati del TEB e dei limiti dello stack, con varianti che usano VirtualQuery oppure evitano le chiamate alle API Windows per maggiore furtività. Il repository è strutturato sia come codice di libreria riutilizzabile sia come binario di dimostrazione da riga di comando. È destinato principalmente alla ricerca di sicurezza offensiva e a strumenti a basso livello dove i percorsi tradizionali di enumerazione dei moduli possono essere monitorati.
