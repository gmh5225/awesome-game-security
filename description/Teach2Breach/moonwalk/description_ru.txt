Этот проект — библиотека и CLI на Rust для поиска базовых адресов загруженных DLL без обхода списка модулей PEB. Он использует подход обхода стека на основе данных TEB и границ стека, с вариантами, которые либо вызывают VirtualQuery, либо избегают вызовов Windows API ради скрытности. Репозиторий организован и как переиспользуемый библиотечный код, и как демонстрационный командный бинарник. Он в первую очередь предназначен для исследований в области offensive security и низкоуровневых инструментов, где традиционные пути перечисления модулей могут отслеживаться.
