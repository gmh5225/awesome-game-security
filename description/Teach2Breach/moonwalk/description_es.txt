Este proyecto es una biblioteca Rust y una CLI para encontrar las direcciones base de DLL cargadas sin recorrer la lista de módulos del PEB. Utiliza un enfoque de recorrido de pila a partir de datos del TEB y de los límites de la pila, con variantes que usan VirtualQuery o evitan llamadas a la API de Windows para mayor sigilo. El repositorio está estructurado tanto como código de biblioteca reutilizable como binario de demostración por línea de comandos. Está destinado principalmente a la investigación de seguridad ofensiva y a herramientas de bajo nivel donde las rutas tradicionales de enumeración de módulos pueden estar monitorizadas.
