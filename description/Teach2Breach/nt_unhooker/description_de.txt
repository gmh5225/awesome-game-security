Dieses Projekt ist ein Rust-basiertes Windows-Tool und eine Bibliothek zum Erkennen und Entfernen von Hooks in NTDLL. Es analysiert Inline- und IAT-Hook-Zustände, vergleicht den Code im Speicher mit einem sauberen Referenzbild und stellt geänderte Bereiche wieder her, wobei kritische Sicherheitsprüfungen durchgeführt werden. Die Implementierung umfasst PE-Parsing, symbolbasierte Abrufung einer sauberen DLL sowie programmatische und CLI-Nutzungspfade. Es wird hauptsächlich von Malware-Analysten, Red-Team-Forschern und Verteidigern genutzt, die User-Mode-Hook-Manipulationen untersuchen.
