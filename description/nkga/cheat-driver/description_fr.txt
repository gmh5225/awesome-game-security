Ce projet se concentre sur les tests de mmCopyVirtualMemory.
Le moyen le plus simple de contourner les protections anti-cheat depuis le noyau est d'utiliser son propre pilote en mode noyau.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine de l'anti-cheat / des tests de stress.
