Этот проект — proof of concept, демонстрирующий эксплуатацию механизма обработки исключений Valorant/Vanguard. Он анализирует, как Vanguard обрабатывает аппаратные исключения и vectored exception handlers, документируя потенциальные слабые места в потоке диспетчеризации исключений античита, которые могут использоваться для выполнения кода или обхода обнаружения. Предназначен для исследователей античитов, изучающих защиту Vanguard на уровне ядра и архитектуру обработки исключений.
