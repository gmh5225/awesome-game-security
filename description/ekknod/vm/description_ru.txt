Кроссплатформенная библиотека доступа к памяти на C/C++ с едиными API чтения/записи/перечисления модулей для нескольких бэкендов: режим ядра Windows (обход ActiveProcessLinks через EPROCESS), пользовательский режим (ReadProcessMemory/WriteProcessMemory), Linux (/proc/pid/mem), PCILeech (через VMMDLL/LeechCore DMA), KVM (интроспекция гостя QEMU), Proton (игры Windows под Linux) и обмен с ядром через переменные EFI.
Каждый бэкенд реализует один и тот же интерфейс vm.h для поиска процессов, трансляции таблиц страниц на основе CR3, обхода модулей PEB/LDR и сканирования сигнатур, позволяя коду читов или анализа переключаться между драйвером ядра, DMA-оборудованием и транспортами гипервизора без изменения игровой логики.
Библиотека в первую очередь полезна исследователям безопасности игр и разработчикам читов, изучающим кроссплатформенные схемы удалённого доступа к памяти процессов и примитивы чтения-записи на базе DMA/KVM.
