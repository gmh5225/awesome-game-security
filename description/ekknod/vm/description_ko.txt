여러 백엔드에 대해 통합된 읽기/쓰기/모듈 열거 API를 제공하는 크로스 플랫폼 C/C++ 메모리 접근 라이브러리입니다: Windows 커널 모드(EPROCESS를 통한 ActiveProcessLinks 순회), 사용자 모드(ReadProcessMemory/WriteProcessMemory), Linux(/proc/pid/mem), PCILeech(VMMDLL/LeechCore DMA 경유), KVM(QEMU 게스트 인트로스펙션), Proton(Linux에서 호스팅되는 Windows 게임), 그리고 EFI 변수 기반 커널 통신.
각 백엔드는 동일한 vm.h 인터페이스를 구현하여 프로세스 조회, CR3 기반 페이지 테이블 변환, PEB/LDR 모듈 순회, 패턴 스캐닝을 지원하므로, 치트나 분석 코드가 게임 대면 로직을 바꾸지 않고 커널 드라이버, DMA 하드웨어, 하이퍼바이저 전송 간에 전환할 수 있습니다.
주로 크로스 플랫폼 원격 프로세스 메모리 접근 패턴과 DMA/KVM 기반 읽기-쓰기 프리미티브를 연구하는 게임 보안 연구자 및 치트 개발자에게 유용합니다.
