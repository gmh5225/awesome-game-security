クロスプラットフォームの C/C++ メモリアクセスライブラリで、複数のバックエンドに対して統一された読み取り/書き込み/モジュール列挙 API を提供します：Windows カーネルモード（EPROCESS 経由で ActiveProcessLinks を走査）、ユーザーモード（ReadProcessMemory/WriteProcessMemory）、Linux（/proc/pid/mem）、PCILeech（VMMDLL/LeechCore DMA 経由）、KVM（QEMU ゲストイントロスペクション）、Proton（Linux 上でホストされる Windows ゲーム）、および EFI 変数ベースのカーネル通信。
各バックエンドは同一の vm.h インターフェースを実装し、プロセス検索、CR3 ベースのページテーブル変換、PEB/LDR モジュール走査、パターンスキャンを備え、チートや解析コードがゲーム向けロジックを変えずにカーネルドライバ、DMA ハードウェア、ハイパーバイザ転送を切り替えられます。
主に、クロスプラットフォームなリモートプロセスメモリアクセスパターンや DMA/KVM ベースの読み書きプリミティブを研究するゲームセキュリティ研究者およびチート開発者向けです。
