一個跨平台的 C/C++ 記憶體存取函式庫，透過統一的讀/寫/模組列舉 API 對接多種後端：Windows 核心模式（經由 EPROCESS 遍歷 ActiveProcessLinks）、使用者模式（ReadProcessMemory/WriteProcessMemory）、Linux（/proc/pid/mem）、PCILeech（透過 VMMDLL/LeechCore DMA）、KVM（QEMU 客體內省）、Proton（Linux 託管的 Windows 遊戲），以及基於 EFI 變數的核心通訊。
各後端實作同一套 vm.h 介面，涵蓋行程查找、基於 CR3 的頁表翻譯、PEB/LDR 模組遍歷與特徵碼掃描，使作弊或分析程式可在核心驅動、DMA 硬體與虛擬機監控程式傳輸之間切換，而無需改動面向遊戲的邏輯。
它主要面向遊戲安全研究者與作弊開發者，用於研究跨平台遠端行程記憶體存取模式，以及基於 DMA/KVM 的讀寫原語。
