Este proyecto se centra en los siguientes requisitos: Windows 10 Enterprise 1607 LTSB con todas las actualizaciones, CPU AMD Ryzen, placa base B350-B450 (el cliente B550 tiene fallos), Logitech GHUB instalado (para entrada de ratón). Está escrito principalmente en C++ y C/C++ y se centra en la investigación anti-cheat, el desarrollo de controladores y OpenGL. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito cheat / exploración de sistemas anticheat: faceit.
