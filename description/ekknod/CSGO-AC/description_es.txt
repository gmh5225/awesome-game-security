Este proyecto se centra en un plugin anti-cheat de tipo prueba de concepto para CS:GO.
Está escrito principalmente en C++ y C/C++ y se enfoca en la investigación anti-cheat y el desarrollo de plugins.
Resulta especialmente útil para ingenieros anti-cheat e investigadores de seguridad defensiva que trabajan en el área de sistemas anti-cheat / anti-cheat de código abierto.
