Questo progetto è un plugin per il debugger di IDA che migliora la ricostruzione dello stack delle chiamate nel debugging Windows x64. Implementato in C++, usa DbgHelp e StackWalk64 per raccogliere frame di stack più affidabili rispetto alla vista predefinita del debugger. Il plugin si carica in IDA come DLL e viene eseguito automaticamente durante le sessioni di debug. È utile per i reverse engineer che necessitano di tracce di esecuzione più chiare nell'analisi di binari protetti o complessi.
