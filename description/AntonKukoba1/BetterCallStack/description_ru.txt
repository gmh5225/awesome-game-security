Этот проект — плагин отладчика IDA, улучшающий восстановление стека вызовов при отладке Windows x64. Реализован на C++, использует DbgHelp и StackWalk64 для сбора более надёжных кадров стека, чем в стандартном представлении отладчика. Плагин загружается в IDA как DLL и автоматически работает во время сеансов отладки. Полезен для реверс-инженеров, которым нужны более понятные трассы выполнения при анализе защищённых или сложных бинарных файлов.
