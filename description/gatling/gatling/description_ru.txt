Фреймворк нагрузочного тестирования для Scala/JVM с асинхронной неблокирующей архитектурой на базе Akka и Netty, позволяющий моделировать высокую конкурентность при минимальных накладных расходах на потоки. Предоставляет Scala DSL для описания пользовательских сценариев по протоколам HTTP, WebSocket, SSE, JMS, gRPC и MQTT с критериями успеха/неудачи на основе утверждений (assertions). Результаты тестов сводятся в подробные HTML-отчёты с распределениями задержек по перцентилям, графиками запросов в секунду и разбивкой ошибок.
