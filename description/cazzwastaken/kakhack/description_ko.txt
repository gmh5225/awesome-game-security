이 프로젝트는 Counter-Strike: Global Offensive용 내부 치트로, 리버싱된 SDK와 여러 훅 구현을 포함합니다.
ImGui 메뉴와 FreeType 폰트 렌더링, JSON 기반 설정 시스템, 광범위한 비주얼 기능, 게임 인터페이스를 다루는 포괄적인 리버스 SDK가 특징입니다.
C++ 코드베이스는 Visual Studio 2022로 x86 인젝션용 DLL로 빌드되며, 일반적인 요구를 넘는 여러 그래픽 및 게임 훅을 포함합니다.
주로 CSGO 내부 치트 아키텍처, SDK 리버싱, ImGui 기반 오버레이 구현을 연구하는 게임 보안 연구자에게 유용합니다.
