Este proyecto es un cheat interno para Counter-Strike: Global Offensive con un SDK reverseado y múltiples implementaciones de hooks.
Incluye un menú ImGui con renderizado de fuentes FreeType, un sistema de configuración basado en JSON, amplias funciones visuales y un SDK reverseado completo que cubre las interfaces del juego.
El código en C++ se compila como DLL para inyección x86 con Visual Studio 2022 e incluye varios hooks gráficos y del juego más allá de lo habitual.
Es útil principalmente para investigadores de seguridad en videojuegos que estudian arquitecturas de cheats internos de CSGO, reverse del SDK e implementación de overlays con ImGui.
