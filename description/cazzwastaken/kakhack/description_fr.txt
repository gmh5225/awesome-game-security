Ce projet est un cheat interne pour Counter-Strike: Global Offensive, avec un SDK reverse-engineered et plusieurs implémentations de hooks.
Il propose un menu ImGui avec rendu de polices FreeType, un système de configuration basé sur JSON, de nombreuses fonctionnalités visuelles, et un SDK reverse complet couvrant les interfaces du jeu.
La base de code C++ est compilée en DLL pour injection x86 avec Visual Studio 2022, et intègre plusieurs hooks graphiques et de jeu au-delà des besoins habituels.
Il s’adresse surtout aux chercheurs en sécurité des jeux qui étudient les architectures de cheats internes CSGO, le reverse du SDK et les overlays basés sur ImGui.
