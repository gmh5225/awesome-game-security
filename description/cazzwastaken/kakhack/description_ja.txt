本プロジェクトは、Counter-Strike: Global Offensive 向けの内部チートで、リバースされた SDK と複数のフック実装を備えています。
ImGui メニューと FreeType フォント描画、JSON ベースの設定システム、豊富なビジュアル機能、およびゲームインターフェースをカバーする包括的なリバース SDK が特徴です。
C++ コードベースは Visual Studio 2022 で x86 注入用 DLL として構築され、一般的な要件を超える複数のグラフィックス／ゲームフックを含みます。
主に CSGO 内部チートアーキテクチャ、SDK リバース、ImGui ベースのオーバーレイ実装を研究するゲームセキュリティ研究者向けです。
