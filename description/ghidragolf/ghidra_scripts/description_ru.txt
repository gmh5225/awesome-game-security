Этот проект посвящён скриптам. Участники разработали Ghidra Scripts для автоматического решения CTF-задач, основанных на типичных проблемах реверс-инжиниринга. Он в первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / плагинов Ghidra.
