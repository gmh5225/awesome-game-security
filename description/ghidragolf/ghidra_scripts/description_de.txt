Dieses Projekt konzentriert sich auf Skripte. Teilnehmer entwickelten Ghidra Scripts, um CTF-Challenges basierend auf gängigen Reverse-Engineering-Problemen automatisiert zu lösen. Es ist vor allem nützlich für Game-Security-Forscher und Reverse Engineers, die offensive Techniken im Bereich Cheats / Ghidra-Plugins untersuchen.
