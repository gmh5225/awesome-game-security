Framelink MCP for Figma는 AI 코딩 에이전트에 Figma 디자인 데이터에 대한 구조화된 접근을 제공하는 Model Context Protocol 서버입니다. TypeScript로 구축되었으며, 단순화된 디자인 메타데이터를 가져오고 참조 이미지를 다운로드하는 전용 도구를 제공합니다. stdio와 HTTP 등 여러 전송 방식을 지원하며, 원시 Figma API 응답을 모델에 친화적인 레이아웃 및 스타일 컨텍스트로 변환합니다. 주요 사용 사례는 design-to-code 자동화로, 개발자가 어시스턴트가 Figma 파일에서 UI를 정확히 구현하도록 돕는 데 목적이 있습니다.
