Framelink MCP for Figma es un servidor Model Context Protocol que ofrece a los agentes de codificación con IA acceso estructurado a los datos de diseño de Figma. Está escrito en TypeScript y proporciona herramientas dedicadas para obtener metadatos de diseño simplificados y descargar imágenes referenciadas. El servidor admite varios transportes como stdio y HTTP, a la vez que traduce las respuestas en bruto de la API de Figma en un contexto de diseño y estilo apto para modelos. Su caso de uso principal es la automatización design-to-code para desarrolladores que quieren que los asistentes implementen la UI con precisión a partir de archivos de Figma.
