Framelink MCP for Figma — это сервер Model Context Protocol, который даёт ИИ-агентам для написания кода структурированный доступ к данным дизайна Figma. Он написан на TypeScript и предоставляет специализированные инструменты для получения упрощённых метаданных дизайна и загрузки связанных изображений. Сервер поддерживает несколько транспортов, таких как stdio и HTTP, и преобразует необработанные ответы Figma API в удобный для модели контекст макета и стилей. Основной сценарий использования — автоматизация design-to-code для разработчиков, которые хотят, чтобы ассистенты точно реализовывали UI по файлам Figma.
