Framelink MCP for Figma ist ein Model-Context-Protocol-Server, der KI-Coding-Agenten strukturierten Zugriff auf Figma-Designdaten gibt. Er ist in TypeScript gebaut und bietet eigene Tools zum Abrufen vereinfachter Design-Metadaten sowie zum Herunterladen referenzierter Bilder. Der Server unterstützt mehrere Transports wie stdio und HTTP und übersetzt rohe Figma-API-Antworten in modellfreundlichen Layout- und Stilkontext. Der Hauptanwendungsfall ist Design-to-Code-Automatisierung für Entwickler, die Assistenten UI präzise aus Figma-Dateien umsetzen lassen wollen.
