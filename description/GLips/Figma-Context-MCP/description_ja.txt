Framelink MCP for Figma は、AI コーディングエージェントに Figma デザインデータへの構造化されたアクセスを提供する Model Context Protocol サーバーです。TypeScript で構築され、簡略化されたデザインメタデータの取得や参照画像のダウンロード用の専用ツールを備えています。stdio や HTTP など複数のトランスポートに対応し、生の Figma API レスポンスをモデル向けのレイアウトおよびスタイル文脈へ変換します。主な用途は design-to-code の自動化で、開発者がアシスタントに Figma ファイルから UI を正確に実装させることを支援します。
