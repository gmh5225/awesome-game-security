Framelink MCP for Figma è un server Model Context Protocol che offre agli agenti di coding basati su IA un accesso strutturato ai dati di design di Figma. È scritto in TypeScript e fornisce strumenti dedicati per recuperare metadati di design semplificati e scaricare le immagini referenziate. Il server supporta più transport come stdio e HTTP, traducendo al contempo le risposte grezze dell'API Figma in un contesto di layout e stile adatto ai modelli. Il caso d'uso principale è l'automazione design-to-code per gli sviluppatori che vogliono che gli assistenti implementino l'UI in modo accurato a partire dai file Figma.
