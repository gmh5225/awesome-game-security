Questo progetto è una libreria di offuscamento di stringhe C++20 header-only che trasforma i letterali in dati ricostruiti a runtime. Genera uno schema di bytecode di offuscamento in fase di compilazione e usa una piccola macchina virtuale basata su stack per ricostruire le stringhe durante l'esecuzione. L'API espone macro per le forme di stringa narrow e wide, con l'obiettivo di tenere le stringhe in chiaro fuori dai dati statici del programma. È usata principalmente per l'hardening del software e la resistenza al reverse engineering in codebase sensibili alla sicurezza.
