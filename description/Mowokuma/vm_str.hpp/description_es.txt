Este proyecto es una biblioteca de ofuscación de cadenas C++20 de solo cabeceras que transforma literales en datos reconstruidos en tiempo de ejecución. Genera un esquema de bytecode de ofuscación en tiempo de compilación y usa una pequeña máquina virtual basada en pila para reconstruir las cadenas durante la ejecución. La API expone macros para formas de cadena estrecha y ancha, con el objetivo de mantener las cadenas en texto claro fuera de los datos estáticos del programa. Se usa principalmente para el endurecimiento de software y la resistencia a la ingeniería inversa en bases de código sensibles a la seguridad.
