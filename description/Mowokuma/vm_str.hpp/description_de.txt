Dieses Projekt ist eine header-only C++20-Bibliothek zur String-Obfuskation, die Literale in zur Laufzeit rekonstruierte Daten umwandelt. Sie erzeugt zur Compile-Zeit ein Obfuskations-Bytecode-Schema und verwendet eine kleine stackbasierte virtuelle Maschine, um Strings während der Ausführung wiederherzustellen. Die API stellt Makros für Narrow- und Wide-String-Formen bereit und zielt darauf ab, Klartext-Strings aus den statischen Programmdaten fernzuhalten. Sie wird vor allem zur Software-Härtung und zum Schutz vor Reverse Engineering in sicherheitskritischen Codebasen eingesetzt.
