Этот проект — header-only библиотека обфускации строк на C++20, преобразующая литералы в данные, восстанавливаемые во время выполнения. Она генерирует схему обфусцирующего байткода на этапе компиляции и использует небольшую стековую виртуальную машину для восстановления строк во время исполнения. API предоставляет макросы для узких и широких строк, стремясь не оставлять открытый текст в статических данных программы. В основном применяется для усиления защиты ПО и сопротивления реверс-инжинирингу в чувствительных к безопасности кодовых базах.
