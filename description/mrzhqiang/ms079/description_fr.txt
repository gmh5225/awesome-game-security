Ce projet est une base de code de serveur privé MapleStory v079 (classique pré-Big Bang), maintenue à des fins d’étude, avec la logique serveur Java, le schéma SQL, des scripts et un support d’empaquetage.
Il documente la configuration MySQL et JDK 8, les builds Maven/IDEA, ainsi qu’un zip de distribution regroupant les données WZ et les scripts pour faire tourner un serveur privé.
Il s’adresse surtout aux développeurs de jeux et reverse engineers qui étudient le fonctionnement interne des serveurs MapleStory classiques.
