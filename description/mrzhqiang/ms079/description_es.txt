Este proyecto es una base de código de servidor privado de MapleStory v079 (clásico anterior a Big Bang), mantenida para estudio, con lógica de servidor en Java, esquema SQL, scripts y soporte de empaquetado.
Documenta la configuración de MySQL y JDK 8, compilaciones con Maven/IDEA y un zip de distribución que agrupa datos WZ y scripts para ejecutar un servidor privado.
Resulta especialmente útil para desarrolladores de juegos e ingenieros inversos que investigan el interior de los servidores clásicos de MapleStory.
