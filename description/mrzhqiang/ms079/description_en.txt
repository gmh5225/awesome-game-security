This project is a MapleStory v079 (pre-Big Bang classic) private-server codebase maintained for study, with Java server logic, SQL schema, scripts, and packaging support.
It documents MySQL and JDK 8 setup, Maven/IDEA builds, and a distribution zip that bundles WZ data and scripts for running a private server.
It is mainly useful for game developers and reverse engineers researching classic MapleStory server internals.
