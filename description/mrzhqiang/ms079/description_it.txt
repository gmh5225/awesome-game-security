Questo progetto è una codebase di server privato MapleStory v079 (classico pre-Big Bang), mantenuta a scopo di studio, con logica server in Java, schema SQL, script e supporto al packaging.
Documenta la configurazione di MySQL e JDK 8, le build Maven/IDEA e uno zip di distribuzione che include dati WZ e script per eseguire un server privato.
È utile soprattutto a sviluppatori di giochi e reverse engineer che studiano gli interni dei server MapleStory classici.
