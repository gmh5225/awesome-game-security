DroidShield 是一款開源 Android SDK，用於偵測常見執行時期威脅，例如 root、除錯器、Hook 框架、模擬器，以及 APK 竄改或重新打包。它主要以 Kotlin 撰寫，並透過 JNI 上的 C++ 原生層實作更難繞過的檢查，例如 ptrace 反除錯，以及在行程映射中掃描 Frida 或 Xposed 特徵。此函式庫採用清晰的模組化架構：純 Kotlin 領域契約、Android 資料檢查、威脅偵測引擎，以及透過 Dagger 2 組裝的公開 SDK 外觀。Gradle 外掛可依建置多型注入並排序檢查變體，使針對某一建置的繞過不易輕易轉移到另一建置。它鎖定需要用戶端反作弊與防竄改訊號、且不依賴託管後端的 Android 應用與遊戲資安使用情境。
