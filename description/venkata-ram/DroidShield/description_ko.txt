DroidShield는 root, 디버거, 후킹 프레임워크, 에뮬레이터, APK 변조 또는 리패키징과 같은 일반적인 런타임 위협을 탐지하는 오픈소스 Android SDK입니다. 주로 Kotlin으로 작성되며, JNI 위의 C++ 네이티브 계층을 통해 ptrace 안티디버그와 프로세스 맵에서 Frida 또는 Xposed 시그니처를 스캔하는 등 우회하기 어려운 검사를 제공합니다. 라이브러리는 순수 Kotlin 도메인 계약, Android 데이터 검사, 위협 탐지 엔진, Dagger 2로 연결된 공개 SDK 파사드로 구성된 깔끔한 모듈형 아키텍처를 사용합니다. Gradle 플러그인은 빌드마다 검사 변형을 다형적으로 주입하고 순서를 정할 수 있어, 한 빌드에 대한 우회가 다른 빌드로 쉽게 이전되지 않도록 합니다. 호스팅된 백엔드 없이 클라이언트 측 안티치트 및 변조 방지 신호가 필요한 Android 앱·게임 보안 사용 사례를 대상으로 합니다.
