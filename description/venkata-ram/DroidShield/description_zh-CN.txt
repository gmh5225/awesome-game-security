DroidShield 是一款开源 Android SDK，用于检测常见运行时威胁，例如 root、调试器、Hook 框架、模拟器，以及 APK 篡改或重打包。它主要以 Kotlin 编写，并通过 JNI 上的 C++ 原生层实现更难绕过的检测，例如 ptrace 反调试，以及在进程映射中扫描 Frida 或 Xposed 特征。该库采用清晰的模块化架构：纯 Kotlin 领域契约、Android 数据检查、威胁检测引擎，以及通过 Dagger 2 装配的公共 SDK 门面。Gradle 插件可按构建多态注入并编排检测变体，使针对某一构建的绕过难以轻易迁移到另一构建。它面向需要客户端反作弊与防篡改信号、且不依赖托管后端的 Android 应用与游戏安全场景。
