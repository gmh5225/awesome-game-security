DroidShield は、root、デバッガ、フッキングフレームワーク、エミュレータ、APK の改ざんや再パッケージ化など、一般的なランタイム脅威を検出するオープンソースの Android SDK です。主に Kotlin で構築され、JNI 経由の C++ ネイティブ層により、ptrace によるアンチデバッグや、プロセスマップ上での Frida / Xposed シグネチャ走査など、迂回しにくい検査を実現します。ライブラリは、純粋な Kotlin ドメイン契約、Android データチェック、脅威検出エンジン、Dagger 2 で配線された公開 SDK ファサードからなる、きれいなモジュール構成を採用しています。Gradle プラグインはビルドごとに検査バリアントを多相的に注入・順序付けできるため、あるビルド向けの回避手法が別ビルドへ容易に転用されにくくなります。ホステッドバックエンドなしでクライアント側のアンチチート／改ざん検知シグナルを求める Android アプリおよびゲームセキュリティ用途を想定しています。
