DroidShield is an open-source Android SDK that detects common runtime threats such as root, debuggers, hooking frameworks, emulators, and APK tampering or repackaging. It is built mainly in Kotlin with a C++ native layer over JNI for harder-to-bypass checks like ptrace anti-debug and Frida or Xposed signature scanning in process maps. The library uses a clean modular architecture with a pure Kotlin domain contract, Android data checks, a threat detection engine, and a public SDK facade wired with Dagger 2. A Gradle plugin can polymorphically inject and order check variants per build so bypasses against one build do not transfer easily to another. It targets Android app and game security use cases where integrators want client-side anti-cheat and anti-tamper signals without a hosted backend.
