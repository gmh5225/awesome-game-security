Ce projet est un script Frida universel destiné à contourner les contrôles de sécurité courants des applications Android sans personnalisation spécifique à l’application. Il se concentre sur le contournement de la détection du root et du SSL pinning en accrochant les chemins de vérification Java et natifs utilisés par des bibliothèques populaires. L’implémentation est écrite en JavaScript pour Frida et conçue pour fonctionner sur un large éventail de versions d’Android. Il est principalement utilisé dans des flux de tests de sécurité mobile autorisés et de rétro-ingénierie.
