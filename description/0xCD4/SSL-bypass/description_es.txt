Este proyecto es un script Frida universal para eludir comprobaciones de seguridad habituales en aplicaciones Android sin personalización específica de la app. Se centra en el bypass de la detección de root y del SSL pinning enganchando rutas de verificación Java y nativas usadas por bibliotecas populares. La implementación está escrita en JavaScript para Frida y diseñada para funcionar en un amplio rango de versiones de Android. Se usa principalmente en pruebas de seguridad móvil autorizadas y flujos de ingeniería inversa.
