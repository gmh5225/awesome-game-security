Этот проект — универсальный скрипт Frida для обхода распространённых проверок безопасности Android-приложений без кастомизации под конкретное приложение. Он сосредоточен на обходе детекции root и SSL pinning путём перехвата Java- и нативных путей проверки, используемых популярными библиотеками. Реализация написана на JavaScript для Frida и рассчитана на широкий диапазон версий Android. В основном применяется в авторизованном тестировании мобильной безопасности и в рабочих процессах реверс-инжиниринга.
