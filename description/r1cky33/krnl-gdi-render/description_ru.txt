Этот проект — фреймворк GDI-рендеринга в режиме ядра Windows, который перехватывает функции рисования GDI для отрисовки оверлейной графики из пространства ядра. Драйвер C++ WDK включает сканирование сигнатур, утилиты ядра NT и предварительно собранные подписанные бинарники драйверов для развёртывания визуальных оверлеев на уровне ядра. Он в основном полезен исследователям безопасности игр, изучающим техники рендеринга в режиме ядра и реализации оверлеев на основе хуков GDI, обходящие обнаружение в пользовательском режиме.
