Este proyecto es un framework de renderizado GDI en modo kernel de Windows que engancha las funciones de dibujo de GDI para renderizar gráficos de superposición desde el espacio de kernel. El controlador C++ WDK incluye escaneo de firmas, utilidades del kernel NT y binarios de controladores firmados precompilados para desplegar superposiciones visuales a nivel de kernel. Es especialmente útil para investigadores de seguridad en juegos que estudian técnicas de renderizado en modo kernel e implementaciones de superposición basadas en hooks de GDI que evaden la detección en modo usuario.
