Questo progetto è un framework di rendering GDI in modalità kernel Windows che intercetta le funzioni di disegno GDI per renderizzare grafica overlay dallo spazio kernel. Il driver C++ WDK include scansione delle signature, utility del kernel NT e binari di driver firmati precompilati per distribuire overlay visivi a livello kernel. È principalmente utile per i ricercatori di sicurezza nei videogiochi che studiano tecniche di rendering in modalità kernel e implementazioni di overlay basate su hook GDI che aggirano il rilevamento in modalità utente.
