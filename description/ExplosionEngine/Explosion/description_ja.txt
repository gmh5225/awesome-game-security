本プロジェクトは、ECS 指向アーキテクチャを採用したクロスプラットフォームの C++ ゲームエンジンです。Vulkan、DirectX 12、Metal などの最新グラフィックスバックエンドを対象とし、CMake と Conan に基づくビルドワークフローとともにエディタおよびランタイムモジュールを提供します。リポジトリはエンジンコードとツール、フロントエンドコンポーネントを組み合わせており、Qt や Web スタックの依存関係を含み、本番向けの開発ワークフローを想定しています。チート向けツールキットではなく、完全なエンジンフレームワークを必要とするゲーム開発チームやレンダリング研究者を対象としています。
