Questo progetto è un motore di gioco C++ multipiattaforma con architettura orientata a ECS. Mira a backend grafici moderni come Vulkan, DirectX 12 e Metal, e fornisce moduli di editor e runtime con un workflow di build basato su CMake e Conan. Il repository combina il codice del motore con tooling e componenti frontend, incluse dipendenze Qt e dello stack web, per workflow di sviluppo di tipo produttivo. È rivolto a team di sviluppo di giochi e ricercatori di rendering che necessitano di un framework motore completo piuttosto che di un toolkit incentrato sui cheat.
