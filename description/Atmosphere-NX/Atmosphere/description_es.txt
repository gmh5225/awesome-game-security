Atmosphere es una plataforma de firmware personalizado para Nintendo Switch que reemplaza o parchea múltiples componentes del sistema de arranque y de tiempo de ejecución. El código está escrito principalmente en C y C++, con código de plataforma de bajo nivel para cargadores, el comportamiento de TrustZone, sysmodules y soporte de NAND emulada. Está organizado en partes modulares como Fusee, Exosphere, Stratosphere y bibliotecas de desarrollo relacionadas. El proyecto lo utilizan principalmente investigadores de seguridad de consolas y desarrolladores de homebrew que necesitan un control profundo a nivel de sistema.
