Atmosphere は Nintendo Switch 向けのカスタムファームウェアプラットフォームであり、起動および実行時の複数のシステムコンポーネントを置き換えたりパッチしたりします。コードベースは主に C と C++ で、ローダー、TrustZone の挙動、sysmodule、エミュレートされた NAND サポート向けの低レベルプラットフォームコードを含みます。Fusee、Exosphere、Stratosphere、および関連する開発ライブラリといったモジュール構成になっています。本プロジェクトは、深いシステムレベルの制御を必要とするコンソールセキュリティ研究者や homebrew 開発者に主に利用されています。
