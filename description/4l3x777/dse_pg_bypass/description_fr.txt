Ce projet est une preuve de concept de recherche sur le noyau Windows visant à contourner Driver Signature Enforcement et PatchGuard via un modèle d'attaque BYOVD.
Il combine du code C++ avec des notes de reverse engineering détaillées qui retracent les callbacks de validation de signature et les chemins d'exécution des contrôles d'intégrité dans les noyaux Windows modernes.
Le matériel montre comment des pilotes signés vulnérables peuvent être exploités pour interférer avec les décisions d'intégrité du code et le comportement de la protection des correctifs.
Il est destiné à l'analyse éducative de la sécurité du noyau et aux défenseurs qui étudient les surfaces d'attaque BYOVD réalistes et leurs mitigations.
