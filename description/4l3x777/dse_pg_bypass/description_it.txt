Questo progetto è una proof of concept di ricerca sul kernel Windows per aggirare Driver Signature Enforcement e PatchGuard tramite un modello di attacco BYOVD.
Combina codice C++ con note dettagliate di reverse engineering che tracciano le callback di convalida delle firme e i percorsi di esecuzione dei controlli di integrità nei kernel Windows moderni.
Il materiale evidenzia come i driver firmati vulnerabili possano essere sfruttati per interferire con le decisioni di integrità del codice e con il comportamento della protezione delle patch.
È pensato per l'analisi educativa della sicurezza del kernel e per i difensori che studiano superfici di attacco BYOVD realistiche e le relative mitigazioni.
