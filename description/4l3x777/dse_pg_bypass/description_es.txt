Este proyecto es una prueba de concepto de investigación del kernel de Windows para eludir Driver Signature Enforcement y PatchGuard mediante un modelo de ataque BYOVD.
Combina código C++ con notas detalladas de ingeniería inversa que rastrean las callbacks de validación de firmas y las rutas de ejecución de comprobaciones de integridad en kernels modernos de Windows.
El material destaca cómo los controladores firmados vulnerables pueden aprovecharse para interferir en las decisiones de integridad del código y en el comportamiento de la protección de parches.
Está pensado para el análisis educativo de seguridad del kernel y para defensores que estudian superficies de ataque BYOVD realistas y sus mitigaciones.
