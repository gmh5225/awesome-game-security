Этот проект — исследовательский proof of concept для ядра Windows по обходу Driver Signature Enforcement и PatchGuard через модель атаки BYOVD.
Он сочетает код на C++ с подробными заметками по реверс-инжинирингу, прослеживающими callback'и проверки подписи и пути выполнения проверок целостности в современных ядрах Windows.
Материал показывает, как уязвимые подписанные драйверы можно использовать для вмешательства в решения о целостности кода и в поведение защиты от патчей.
Предназначен для образовательного анализа безопасности ядра и для защитников, изучающих реалистичные поверхности атак BYOVD и меры противодействия.
