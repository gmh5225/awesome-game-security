Автономный парсер бинарного формата трассировки .trace64 от x64dbg, который десериализует записи инструкций, дизассемблирует их через движок Capstone и поддерживает фильтрацию трассировок выполнения по регулярным выражениям. Восстанавливает состояние регистров и памяти для каждой инструкции из компактного формата буфера трассировки для офлайн-анализа.
