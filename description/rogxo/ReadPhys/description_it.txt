Questo progetto è un metodo per leggere la memoria fisica mappando manualmente le PTE senza usare API di lettura/scrittura (come MmCopyMemory/MmMapIoSpace...). È stato reverse-engineered da AXE-BASE.sys.
È scritto principalmente in C++ e C/C++ e si concentra sull'analisi della memoria.
È soprattutto utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'area cheat / esplorazione dei sistemi anticheat: ACE.
