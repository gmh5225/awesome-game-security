Этот проект — метод чтения физической памяти путём ручного отображения PTE без использования каких-либо API чтения/записи (таких как MmCopyMemory/MmMapIoSpace...). Получен путём реверса AXE-BASE.sys.
В основном написан на C++ и C/C++ и сосредоточен на анализе памяти.
Прежде всего полезен исследователям игровой безопасности и инженерам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования античит-систем: ACE.
