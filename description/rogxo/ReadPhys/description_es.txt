Este proyecto es un método para leer memoria física mapeando manualmente las PTE sin usar ninguna API de lectura/escritura (como MmCopyMemory/MmMapIoSpace...). Fue revertido a partir de AXE-BASE.sys.
Está escrito principalmente en C++ y C/C++ y se centra en el análisis de memoria.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / exploración de sistemas anticheat: ACE.
