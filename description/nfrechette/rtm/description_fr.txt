Ce projet porte sur cette bibliothèque conçue pour les applications temps réel qui exigent des calculs mathématiques aussi rapides que possible.
Elle contient l'arithmétique 3D et 4D couramment utilisée dans les jeux vidéo et les applications temps réel.
Elle est principalement utile aux programmeurs moteur et aux développeurs de gameplay ou de simulation travaillant dans le domaine des mathématiques.
