Este proyecto es un concepto centrado en la documentación para conectar flujos de automatización con IA entre Blender y Unreal Engine 5.
El repositorio contiene sobre todo documentos de diseño y flujo de trabajo en Markdown, más que una implementación de código lista para producción.
Describe funciones como generación de escenas impulsada por texto, transferencia de assets, automatización de niveles y asistencia con Blueprint.
Los autores señalan que el desarrollo se pausó, por lo que su principal valor es como referencia para la investigación de pipelines de juegos y herramientas de IA.
