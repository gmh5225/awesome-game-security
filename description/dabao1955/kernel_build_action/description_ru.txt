Этот проект — action для автоматической сборки ядра.
Он в основном написан на TypeScript и Python и сосредоточен на работе на уровне ядра.
В первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов и разработки драйверов ядра Android.
