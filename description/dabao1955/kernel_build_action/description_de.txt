Dieses Projekt ist eine Action zum automatischen Bauen eines Kernels.
Es ist hauptsächlich in TypeScript und Python geschrieben und konzentriert sich auf kernelnahe Arbeit.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat-/Android-Kernel-Treiberentwicklung untersuchen.
