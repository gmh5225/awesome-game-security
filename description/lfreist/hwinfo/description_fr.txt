Ce projet est hwinfo, une bibliothèque C++ multiplateforme permettant de récupérer des informations matérielles, notamment le modèle de CPU, le nombre de cœurs, la taille des caches, la capacité de RAM, les disques, les détails GPU et les interfaces réseau. Elle fournit une API unifiée sous Windows, Linux et macOS pour interroger de manière programmatique les spécifications matérielles du système. Elle s'adresse aux développeurs qui créent des outils d'information système, des applications d'inventaire matériel ou des systèmes d'identification basés sur le HWID.
