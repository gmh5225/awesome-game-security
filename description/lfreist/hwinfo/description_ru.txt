Этот проект — hwinfo, кроссплатформенная библиотека на C++ для получения сведений об оборудовании, включая модель CPU, число ядер, размеры кэша, объём RAM, диски, сведения о GPU и сетевые интерфейсы. Она предоставляет единый API для Windows, Linux и macOS, позволяя программно запрашивать спецификации аппаратного обеспечения системы. Предназначена для разработчиков, создающих средства системной информации, приложения инвентаризации оборудования или системы идентификации на основе HWID.
