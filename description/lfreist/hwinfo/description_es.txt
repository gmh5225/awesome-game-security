Este proyecto es hwinfo, una biblioteca C++ multiplataforma para obtener información de hardware, incluyendo modelo de CPU, número de núcleos, tamaños de caché, capacidad de RAM, discos, detalles de GPU e interfaces de red. Proporciona una API unificada en Windows, Linux y macOS para consultar de forma programática las especificaciones de hardware del sistema. Está orientada a desarrolladores que crean herramientas de información del sistema, aplicaciones de inventario de hardware o sistemas de identificación basados en HWID.
