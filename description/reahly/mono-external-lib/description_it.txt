Questo progetto è una libreria esterna per leggere gli internal di giochi Unity/Mono dall'esterno del processo di gioco su Windows. Analizza le strutture dei metadati del runtime Mono in modo esterno tramite lettura della memoria di processo per enumerare classi, metodi, campi e i relativi offset senza injection. La libreria C++ ricostruisce il sistema di tipi Mono dalla memoria grezza per consentire lo sviluppo di cheat esterni per giochi Unity Mono. È rivolta a game hacker e ricercatori di sicurezza che studiano l'accesso esterno agli internal di giochi Mono/.NET.
