Dieses Projekt ist eine externe Bibliothek zum Auslesen der Internals von Unity/Mono-Spielen von außerhalb des Spielprozesses unter Windows. Sie parst Mono-Runtime-Metadatenstrukturen extern über Prozessspeicherzugriff, um Klassen, Methoden, Felder und deren Offsets ohne Injection aufzulisten. Die C++-Bibliothek rekonstruiert das Mono-Typsystem aus dem Rohspeicher und ermöglicht so die Entwicklung externer Cheats für Unity-Mono-Spiele. Sie richtet sich an Game-Hacker und Sicherheitsforscher, die den externen Zugriff auf Mono-/.NET-Spielinternals untersuchen.
