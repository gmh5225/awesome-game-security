Ce projet est une bibliothèque externe permettant de lire les internals de jeux Unity/Mono depuis l'extérieur du processus du jeu sous Windows. Elle analyse les structures de métadonnées du runtime Mono de façon externe via la lecture de la mémoire du processus afin d'énumérer les classes, méthodes, champs et leurs offsets sans injection. La bibliothèque C++ reconstruit le système de types Mono à partir de la mémoire brute pour permettre le développement de cheats externes pour les jeux Unity Mono. Elle s'adresse aux game hackers et aux chercheurs en sécurité qui étudient l'accès externe aux internals des jeux Mono/.NET.
