Este proyecto es una biblioteca externa para leer los internals de juegos Unity/Mono desde fuera del proceso del juego en Windows. Analiza estructuras de metadatos del runtime de Mono de forma externa mediante lectura de memoria del proceso para enumerar clases, métodos, campos y sus offsets sin inyección. La biblioteca C++ reconstruye el sistema de tipos de Mono a partir de la memoria en bruto para permitir el desarrollo de cheats externos para juegos Unity Mono. Está dirigida a game hackers e investigadores de seguridad que estudian el acceso externo a los internals de juegos Mono/.NET.
