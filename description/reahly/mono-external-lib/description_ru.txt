Этот проект — внешняя библиотека для чтения внутренних структур игр Unity/Mono извне процесса игры в Windows. Она внешне разбирает структуры метаданных runtime Mono через чтение памяти процесса, перечисляя классы, методы, поля и их смещения без инъекции. Библиотека на C++ восстанавливает систему типов Mono из сырой памяти, что позволяет разрабатывать внешние читы для игр Unity Mono. Предназначена для гейм-хакеров и исследователей безопасности, изучающих внешний доступ к внутренностям игр Mono/.NET.
