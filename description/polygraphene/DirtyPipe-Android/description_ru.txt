Многоэтапный эксплойт, использующий CVE-2022-0847 (Dirty Pipe) для постоянного получения root на устройствах Pixel 6: повреждает загрузчик модулей ядра путём перезаписи page cache каналов (pipe), внедряет ARM64 shellcode для патча SELinux и учётных данных и автоматически устанавливает Magisk v24.3 для сохранения root-доступа после перезагрузок.
