本プロジェクトは、対象ゲームプロセス内で Unreal Engine のランタイム構造を自動的に発見・マップする Cheat Engine の Lua スクリプト群です。UEInfoScanner は、インジェクションなしでメモリスキャン、vtable 検査、シグネチャ風の探査を用い、Unreal Engine 4 および 5 のビルド横断で名前プール（GNames/FName データ）、オブジェクト配列、UObject と UClass のレイアウト、FProperty チェーン、関連オフセットを特定します。UEInfoStructureDissect は発見したクラスとオブジェクトから Cheat Engine の構造体定義を構築し、Structure Dissect で読みやすいフィールド名を表示するための FName から文字列へのカスタム型も含みます。スクリプトは Cheat Engine に Unreal Engine メニュー項目を追加し、解析者がスキャンの初期化、状態の監視、復元したレイアウトの対話的な適用を行えるようにします。主な用途は Unreal Engine ゲームのリバースエンジニアリング、メモリ上のオブジェクト階層とプロパティの検査、および Cheat Engine に依存するゲームセキュリティやチート研究のワークフロー支援です。
