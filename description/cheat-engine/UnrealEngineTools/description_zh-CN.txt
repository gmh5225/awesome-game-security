本项目是一组 Cheat Engine Lua 脚本，可自动在目标游戏进程中发现并映射 Unreal Engine 运行时结构。UEInfoScanner 通过内存扫描、vtable 检查以及签名式探测，在无需注入的情况下，跨 Unreal Engine 4 与 5 构建定位名称池（GNames/FName 数据）、对象数组、UObject 与 UClass 布局、FProperty 链及相关偏移。UEInfoStructureDissect 根据已发现的类与对象生成 Cheat Engine 结构定义，并包含 FName 到字符串的自定义类型，以便在 Structure Dissect 中显示可读字段名。脚本会在 Cheat Engine 中添加 Unreal Engine 菜单项，便于分析人员初始化扫描、监视状态并交互式应用恢复的布局。主要用途包括逆向工程 Unreal Engine 游戏、在内存中检查对象层次与属性，以及支持依赖 Cheat Engine 的游戏安全或作弊研究工作流。
