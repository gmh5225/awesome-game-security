This project is a set of Cheat Engine Lua scripts that automatically discover and map Unreal Engine runtime structures in a target game process. UEInfoScanner locates name pools (GNames/FName data), object arrays, UObject and UClass layouts, FProperty chains, and related offsets across Unreal Engine 4 and 5 builds, using memory scans, vtable checks, and signature-style probing without requiring injection. UEInfoStructureDissect builds Cheat Engine structure definitions from discovered classes and objects, including an FName-to-string custom type for readable field names in Structure Dissect. The scripts add Unreal Engine menu entries in Cheat Engine so analysts can initialize scanning, monitor status, and apply the recovered layouts interactively. Primary use cases are reverse engineering Unreal Engine games, inspecting object hierarchies and properties in memory, and supporting game-security or cheat-research workflows that rely on Cheat Engine.
