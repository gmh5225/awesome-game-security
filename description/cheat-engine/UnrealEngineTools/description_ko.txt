이 프로젝트는 대상 게임 프로세스에서 Unreal Engine 런타임 구조를 자동으로 발견하고 매핑하는 Cheat Engine Lua 스크립트 모음입니다. UEInfoScanner는 인젝션 없이 메모리 스캔, vtable 검사, 시그니처 방식 탐색을 사용해 Unreal Engine 4와 5 빌드 전반에서 이름 풀(GNames/FName 데이터), 객체 배열, UObject 및 UClass 레이아웃, FProperty 체인과 관련 오프셋을 찾습니다. UEInfoStructureDissect는 발견한 클래스와 객체로부터 Cheat Engine 구조체 정의를 만들며, Structure Dissect에서 읽기 쉬운 필드 이름을 위한 FName-to-string 사용자 지정 타입을 포함합니다. 스크립트는 Cheat Engine에 Unreal Engine 메뉴 항목을 추가해 분석가가 스캔을 초기화하고 상태를 모니터링하며 복구한 레이아웃을 대화형으로 적용할 수 있게 합니다. 주요 용도는 Unreal Engine 게임 리버스 엔지니어링, 메모리에서 객체 계층과 속성 검사, 그리고 Cheat Engine에 의존하는 게임 보안 또는 치트 연구 워크플로 지원입니다.
