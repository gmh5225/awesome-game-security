本專案是一組 Cheat Engine Lua 指令碼，可自動在目標遊戲程序中探索並對應 Unreal Engine 執行時期結構。UEInfoScanner 透過記憶體掃描、vtable 檢查以及簽章式探測，在無需注入的情況下，跨 Unreal Engine 4 與 5 建置定位名稱池（GNames/FName 資料）、物件陣列、UObject 與 UClass 配置、FProperty 鏈及相關偏移。UEInfoStructureDissect 依已發現的類別與物件建立 Cheat Engine 結構定義，並包含 FName 到字串的自訂類型，以便在 Structure Dissect 中顯示可讀欄位名稱。指令碼會在 Cheat Engine 中新增 Unreal Engine 選單項目，方便分析人員初始化掃描、監控狀態並以互動方式套用還原的配置。主要用途包括逆向工程 Unreal Engine 遊戲、在記憶體中檢查物件階層與屬性，以及支援仰賴 Cheat Engine 的遊戲安全或作弊研究工作流程。
