Questo progetto è un SDK ottenuto tramite reverse engineering per un gioco basato sul motore Source (probabilmente CS:GO), contenente definizioni di classi, dichiarazioni di interfacce, strutture netvar e prototipi di funzioni estratti dal binario del gioco. Le intestazioni C++ forniscono rappresentazioni tipizzate con precisione delle strutture dati di gioco, incluse classi di entità, dati delle armi, informazioni sui giocatori e interfacce del motore. È rivolto a sviluppatori di cheat e ricercatori di sicurezza che necessitano di header SDK per interagire con gli interni dei giochi Source.
