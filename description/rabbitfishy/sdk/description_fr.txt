Ce projet est un SDK rétro-ingénieré pour un jeu basé sur le moteur Source (probablement CS:GO), contenant des définitions de classes, des déclarations d'interfaces, des structures netvar et des prototypes de fonctions extraits du binaire du jeu. Les en-têtes C++ fournissent des représentations typées fidèles des structures de données du jeu, notamment les classes d'entités, les données d'armes, les informations joueur et les interfaces du moteur. Il s'adresse aux développeurs de cheats et aux chercheurs en sécurité qui ont besoin d'en-têtes SDK pour interagir avec les internals des jeux Source.
