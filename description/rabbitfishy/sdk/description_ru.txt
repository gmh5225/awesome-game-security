Этот проект — реверс-инжиниринговый SDK для игры на движке Source (скорее всего CS:GO), содержащий определения классов, объявления интерфейсов, структуры netvar и прототипы функций, извлечённые из бинарника игры. Заголовочные файлы C++ дают типоточно точные представления игровых структур данных, включая классы сущностей, данные оружия, информацию об игроке и интерфейсы движка. Он предназначен для разработчиков читов и исследователей безопасности, которым нужны заголовки SDK для взаимодействия с внутренностями игр на Source.
