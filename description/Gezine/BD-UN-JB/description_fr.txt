Ce projet est un exploit de jailbreak basé sur Blu-ray Disc Java (BD-J) pour les consoles PlayStation. Il utilise des payloads xlet BD-J et la classe jdk.internal.misc.Unsafe pour obtenir l'exécution de code, ainsi qu'un outil C bdj_unpatch pour la manipulation BDMV et un client de journalisation Python pour le débogage. Il s'adresse principalement aux chercheurs en sécurité console étudiant les chaînes d'exploitation PlayStation et les techniques d'échappement du sandbox BD-J.
