Этот проект — эксплойт jailbreak на основе Blu-ray Disc Java (BD-J) для консолей PlayStation. Он использует BD-J xlet-пейлоады и класс jdk.internal.misc.Unsafe для выполнения кода, а также C-утилиту bdj_unpatch для манипуляции BDMV и Python-клиент логов для отладки. В первую очередь полезен исследователям безопасности консолей, изучающим цепочки эксплойтов PlayStation и техники выхода из песочницы BD-J.
