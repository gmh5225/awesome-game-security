Questo progetto è un exploit di jailbreak basato su Blu-ray Disc Java (BD-J) per console PlayStation. Usa payload xlet BD-J e la classe jdk.internal.misc.Unsafe per ottenere l'esecuzione di codice, insieme a uno strumento C bdj_unpatch per la manipolazione BDMV e a un client di log Python per il debug. È utile soprattutto ai ricercatori di sicurezza delle console che studiano le catene di exploit PlayStation e le tecniche di escape dalla sandbox BD-J.
