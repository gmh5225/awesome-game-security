Este proyecto es un exploit de jailbreak basado en Blu-ray Disc Java (BD-J) para consolas PlayStation. Utiliza payloads xlet de BD-J y la clase jdk.internal.misc.Unsafe para lograr la ejecución de código, junto con una herramienta en C llamada bdj_unpatch para manipulación BDMV y un cliente de registro en Python para depuración. Es útil principalmente para investigadores de seguridad de consolas que estudian cadenas de exploits de PlayStation y técnicas de escape del sandbox BD-J.
