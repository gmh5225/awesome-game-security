Dieses Projekt ist ein auf Blu-ray Disc Java (BD-J) basierender Jailbreak-Exploit für PlayStation-Konsolen. Es nutzt BD-J-xlet-Payloads und die Klasse jdk.internal.misc.Unsafe zur Codeausführung sowie das C-basierte Tool bdj_unpatch zur BDMV-Manipulation und einen Python-Log-Client zum Debuggen. Es richtet sich vor allem an Konsolen-Sicherheitsforscher, die PlayStation-Exploit-Ketten und BD-J-Sandbox-Escape-Techniken untersuchen.
