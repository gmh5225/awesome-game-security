Этот проект — внутренний игровой фреймворк для Windows, предназначенный для инъекции, runtime-хукинга и модульного переключения функций. Он написан на C++ и включает компоненты для хуков рендеринга Direct3D, перехвата DirectInput, сканирования сигнатур, патчинга памяти и удалённого выполнения с помощью shellcode. Кодовая база построена вокруг интерфейса модулей и менеджера, чтобы функции можно было инициализировать, отрисовывать, обновлять и горячо переключать по клавишам. В основном полезен для практики реверс-инжиниринга и исследований читов или античитов на уровне user-mode внутренних механизмов игр.
