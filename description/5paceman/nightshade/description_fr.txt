Ce projet est un framework interne Windows pour jeux, destiné à l'injection, au hooking à l'exécution et au basculement modulaire de fonctionnalités. Écrit en C++, il comprend des composants pour les hooks de rendu Direct3D, l'interception DirectInput, le pattern scanning, le patch mémoire et l'exécution distante assistée par shellcode. La base de code s'organise autour d'une interface et d'un gestionnaire de modules afin d'initialiser, dessiner, mettre à jour et basculer à chaud les fonctionnalités via des raccourcis. Il est surtout utile pour s'entraîner à la rétro-ingénierie et pour la recherche sur les cheats ou anti-cheats au niveau user-mode des jeux.
