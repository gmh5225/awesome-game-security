Este proyecto es un framework interno de Windows para juegos, orientado a la inyección, el hooking en tiempo de ejecución y el activado modular de funciones. Está escrito en C++ e incluye componentes para hooks de renderizado Direct3D, interceptación de DirectInput, pattern scanning, parcheo de memoria y ejecución remota asistida por shellcode. El código se organiza en torno a una interfaz y un gestor de módulos para inicializar, dibujar, actualizar y activar o desactivar en caliente las funciones mediante atajos. Es útil sobre todo para practicar ingeniería inversa e investigar cheats o anti-cheats en el user-mode de los juegos.
