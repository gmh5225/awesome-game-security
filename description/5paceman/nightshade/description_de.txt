Dieses Projekt ist ein Windows-internes Game-Framework für Injection, Runtime-Hooking und modulares Feature-Toggling. Es ist in C++ geschrieben und enthält Komponenten für Direct3D-Rendering-Hooks, DirectInput-Abfangen, Pattern Scanning, Memory Patching sowie shellcode-unterstützte Remote-Ausführung. Die Codebasis ist um eine Modul-Schnittstelle und einen Manager organisiert, sodass Features initialisiert, gezeichnet, aktualisiert und per Tastenkürzel hot-togglen können. Es eignet sich vor allem für Reverse-Engineering-Übungen sowie Cheat- oder Anti-Cheat-Forschung an User-Mode-Game-Internals.
