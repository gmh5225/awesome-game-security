Questo progetto è un framework interno Windows per giochi, orientato a injection, hooking a runtime e attivazione modulare delle funzionalità. È scritto in C++ e include componenti per hook di rendering Direct3D, intercettazione DirectInput, pattern scanning, memory patching ed esecuzione remota assistita da shellcode. Il codice è organizzato attorno a un'interfaccia e a un manager di moduli, così le funzionalità possono essere inizializzate, disegnate, aggiornate e attivate a caldo tramite tasti. È utile soprattutto per esercitarsi nel reverse engineering e per la ricerca su cheat o anti-cheat a livello user-mode degli interni di gioco.
