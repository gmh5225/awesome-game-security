Ce projet fournit guide-zh.
Il est organisé comme documentation et matériel de référence pour le domaine des cheats / explorateurs de fichiers Android, plutôt que comme une base de code d'exécution autonome.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / explorateurs de fichiers Android.
