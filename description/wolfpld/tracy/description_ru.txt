Этот проект — Tracy, профайлер кадров в реальном времени для C++-приложений, включая игры. Он предоставляет профилирование CPU с разметкой зон, профилирование GPU для OpenGL/Vulkan/Direct3D, отслеживание выделения памяти, анализ конкуренции за блокировки, визуализацию переключений контекста и аннотирование исходного кода. Профайлер состоит из лёгкой клиентской библиотеки, встраиваемой в приложение, и автономного просмотрщика с отображением временной шкалы, flame graph и статистических представлений. Он предназначен для разработчиков игр и инженеров по производительности, оптимизирующих время кадра и выявляющих узкие места в приложениях реального времени.
