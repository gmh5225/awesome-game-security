이 프로젝트는 Tracy로, 게임을 포함한 C++ 애플리케이션을 위한 실시간 프레임 프로파일러입니다. 존 마킹을 통한 CPU 프로파일링, OpenGL/Vulkan/Direct3D용 GPU 프로파일링, 메모리 할당 추적, 락 경합 분석, 컨텍스트 스위치 시각화, 소스 코드 주석을 제공합니다. 프로파일러는 애플리케이션에 내장되는 경량 클라이언트 라이브러리와 타임라인, 플레임 그래프, 통계 뷰를 표시하는 독립형 뷰어로 구성됩니다. 프레임 시간을 최적화하고 실시간 애플리케이션의 병목을 찾는 게임 개발자와 성능 엔지니어를 대상으로 합니다.
