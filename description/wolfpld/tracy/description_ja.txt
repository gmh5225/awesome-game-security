本プロジェクトは Tracy で、ゲームを含む C++ アプリケーション向けのリアルタイムフレームプロファイラです。ゾーンマークアップによる CPU プロファイリング、OpenGL/Vulkan/Direct3D 向け GPU プロファイリング、メモリ割り当て追跡、ロック競合分析、コンテキストスイッチ可視化、ソースコード注釈を提供します。プロファイラは、アプリケーションに組み込む軽量クライアントライブラリと、タイムライン・フレイムグラフ・統計ビューを表示するスタンドアロンビューアで構成されます。フレーム時間の最適化とリアルタイムアプリケーションのボトルネック特定を行うゲーム開発者およびパフォーマンスエンジニアを対象としています。
