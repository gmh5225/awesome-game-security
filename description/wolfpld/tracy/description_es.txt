Este proyecto es Tracy, un perfilador de frames en tiempo real para aplicaciones C++, incluidos los juegos. Ofrece perfilado de CPU con marcado de zonas, perfilado de GPU para OpenGL/Vulkan/Direct3D, seguimiento de asignaciones de memoria, análisis de contención de bloqueos, visualización de cambios de contexto y anotación de código fuente. El perfilador consta de una biblioteca cliente ligera integrada en la aplicación y un visor independiente que muestra la línea de tiempo, el flame graph y vistas estadísticas. Está dirigido a desarrolladores de juegos e ingenieros de rendimiento que optimizan los tiempos de frame e identifican cuellos de botella en aplicaciones en tiempo real.
