Ce projet est Tracy, un profileur de frames en temps réel pour les applications C++, y compris les jeux. Il offre le profilage CPU avec marquage de zones, le profilage GPU pour OpenGL/Vulkan/Direct3D, le suivi des allocations mémoire, l'analyse de contention de verrous, la visualisation des changements de contexte et l'annotation du code source. Le profileur se compose d'une bibliothèque client légère intégrée à l'application et d'un visualiseur autonome affichant la chronologie, le flame graph et les vues statistiques. Il s'adresse aux développeurs de jeux et aux ingénieurs performance qui optimisent les temps de frame et identifient les goulots d'étranglement dans les applications temps réel.
