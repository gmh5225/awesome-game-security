Classe C++ header-only che sostituisce la divisione intera a runtime con operazioni di moltiplicazione e shift precompute, basate sulla teoria dei magic number di «Hacker's Delight». Funziona in modo trasparente come sostituto drop-in di int grazie agli operatori / e % sovraccaricati, rivolgendosi sia a CPU sia a kernel CUDA GPU tramite le annotazioni __host__ __device__, con uno speedup di circa 2x rispetto alla divisione hardware.
