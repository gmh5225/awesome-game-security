Header-only класс C++, который заменяет целочисленное деление во время выполнения на заранее вычисленные операции умножения и сдвига на основе теории «магических чисел» из «Hacker's Delight». Прозрачно работает как замена int с перегруженными операторами / и %, ориентирован как на CPU, так и на CUDA GPU-ядра благодаря аннотациям __host__ __device__, обеспечивая примерно двукратное ускорение по сравнению с аппаратным делением.
