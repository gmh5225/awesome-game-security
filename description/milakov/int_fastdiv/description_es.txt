Clase C++ de solo cabecera que sustituye la división entera en tiempo de ejecución por operaciones de multiplicación y desplazamiento precalculadas, basadas en la teoría de números mágicos de «Hacker's Delight». Funciona de forma transparente como reemplazo directo de int con los operadores / y % sobrecargados, orientada tanto a CPU como a kernels CUDA GPU mediante las anotaciones __host__ __device__, con una aceleración de aproximadamente 2x frente a la división por hardware.
