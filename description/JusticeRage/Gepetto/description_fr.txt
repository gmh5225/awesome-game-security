Ce projet est un plugin Python pour IDA Pro qui utilise des grands modèles de langage pour expliquer les fonctions décompilées.
Il peut également suggérer le renommage de variables et des commentaires de code directement depuis IDA via des actions de menu et des raccourcis clavier.
Le plugin prend en charge plusieurs fournisseurs de modèles via la configuration, y compris des backends d'API cloud et locaux.
Il s'adresse aux reverse engineers qui souhaitent une compréhension plus rapide du code dans l'analyse de malwares, de logiciels et de la sécurité des jeux.
