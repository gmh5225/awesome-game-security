Dieses Projekt ist ein Python-Plugin für IDA Pro, das große Sprachmodelle nutzt, um dekompilierte Funktionen zu erklären.
Es kann außerdem über Menüaktionen und Tastenkürzel direkt in IDA Umbenennungen von Variablen und Codekommentare vorschlagen.
Das Plugin unterstützt über die Konfiguration mehrere Modellanbieter, einschließlich Cloud- und lokaler API-Backends.
Es richtet sich an Reverse Engineers, die in der Malware-, Software- und Gamesicherheitsanalyse Code schneller verstehen möchten.
