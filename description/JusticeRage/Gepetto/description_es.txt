Este proyecto es un plugin de Python para IDA Pro que usa grandes modelos de lenguaje para explicar funciones descompiladas.
También puede sugerir el renombrado de variables y comentarios de código directamente desde IDA mediante acciones de menú y atajos de teclado.
El plugin admite varios proveedores de modelos mediante configuración, incluidos backends de API en la nube y locales.
Está pensado para ingenieros inversos que buscan comprender el código más rápido en el análisis de malware, software y seguridad de juegos.
