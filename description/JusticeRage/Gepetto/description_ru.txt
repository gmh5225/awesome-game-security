Этот проект — плагин на Python для IDA Pro, который использует большие языковые модели для объяснения декомпилированных функций.
Он также может предлагать переименование переменных и комментарии к коду прямо из IDA через пункты меню и горячие клавиши.
Плагин через конфигурацию поддерживает несколько провайдеров моделей, включая облачные и локальные API-бэкенды.
Он предназначен для реверс-инженеров, которым нужно быстрее понимать код при анализе вредоносного ПО, ПО и безопасности игр.
