Este proyecto se centra en un plugin de código para Unreal Engine 4 que ofusca cualquier variable estática (no dinámica) para que sea muy difícil localizarla en memoria.
Así, usar herramientas como Cheat Engine para manipular datos del juego deja de ser una tarea trivial.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva en el ámbito de anti-cheat / protección de motores de juego: Unreal.
