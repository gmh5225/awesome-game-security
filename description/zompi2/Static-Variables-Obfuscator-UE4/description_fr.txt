Ce projet porte sur un plugin de code pour Unreal Engine 4 qui obfusque toute variable statique (non dynamique) afin qu’elle soit très difficile à localiser en mémoire.
Cela rend l’utilisation d’outils comme Cheat Engine pour manipuler les données en jeu nettement moins triviale.
Il est surtout utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant sur l’anti-cheat / la protection de moteurs de jeu dans l’écosystème Unreal.
