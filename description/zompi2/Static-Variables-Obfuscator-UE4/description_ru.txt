Этот проект посвящён плагину кода для Unreal Engine 4, который обфусцирует любые статические (нединамические) переменные, делая их крайне трудно найти в памяти.
Из-за этого использование таких инструментов, как Cheat Engine, для изменения игровых данных перестаёт быть тривиальной задачей.
Плагин в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / защиты игровых движков (Unreal).
