Dieses Projekt konzentriert sich auf ein Code-Plugin für Unreal Engine 4, das beliebige statische (nicht dynamische) Variablen verschleiert, sodass sie im Speicher sehr schwer zu finden sind.
Dadurch wird das Manipulieren von Spieldaten mit Tools wie Cheat Engine zu einer nicht trivialen Aufgabe.
Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher im Bereich Anti-Cheat / Spiel-Engine-Schutz (Unreal) nützlich.
