Ce projet est un module processeur IDA Pro pour désassembler le bytecode de l'Ethereum Virtual Machine (EVM). Il ajoute le décodage des instructions EVM à IDA, affichant des opcodes comme PUSH, POP, SLOAD, SSTORE, CALL et JUMPI avec un formatage correct des opérandes et la génération de références croisées. Le plugin Python permet d'analyser les bytecodes de smart contracts dans l'environnement de désassemblage familier d'IDA. Il s'adresse aux auditeurs de sécurité blockchain et aux chercheurs en smart contracts qui effectuent de la rétro-ingénierie de bytecode EVM.
