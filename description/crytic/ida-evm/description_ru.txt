Этот проект — процессорный модуль IDA Pro для дизассемблирования байткода Ethereum Virtual Machine (EVM). Он добавляет в IDA декодирование инструкций EVM, отображая опкоды вроде PUSH, POP, SLOAD, SSTORE, CALL и JUMPI с корректным форматированием операндов и генерацией перекрёстных ссылок. Плагин на Python позволяет анализировать байткод смарт-контрактов в привычной среде дизассемблирования IDA. Он предназначен для аудиторов безопасности блокчейна и исследователей смарт-контрактов, занимающихся реверс-инжинирингом байткода EVM.
