이 프로젝트는 Ethereum Virtual Machine(EVM) 바이트코드를 디스어셈블하기 위한 IDA Pro 프로세서 모듈입니다. IDA에 EVM 명령 디코딩을 추가하여 PUSH, POP, SLOAD, SSTORE, CALL, JUMPI 같은 Opcode를 올바른 피연산자 형식으로 표시하고 교차 참조를 생성합니다. 이 Python 플러그인을 사용하면 익숙한 IDA 디스어셈블리 환경에서 스마트 컨트랙트 바이트코드를 분석할 수 있습니다. EVM 바이트코드 리버스 엔지니어링을 수행하는 블록체인 보안 감사자와 스마트 컨트랙트 연구자를 대상으로 합니다.
