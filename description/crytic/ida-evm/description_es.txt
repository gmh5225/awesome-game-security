Este proyecto es un módulo de procesador de IDA Pro para desensamblar bytecode de la Ethereum Virtual Machine (EVM). Añade decodificación de instrucciones EVM a IDA, mostrando opcodes como PUSH, POP, SLOAD, SSTORE, CALL y JUMPI con un formato correcto de operandos y generación de referencias cruzadas. El plugin de Python permite analizar bytecodes de contratos inteligentes en el entorno de desensamblado familiar de IDA. Está dirigido a auditores de seguridad blockchain e investigadores de smart contracts que realizan ingeniería inversa de bytecode EVM.
