Questo progetto è un modulo processore di IDA Pro per disassemblare il bytecode dell'Ethereum Virtual Machine (EVM). Aggiunge a IDA la decodifica delle istruzioni EVM, visualizzando opcode come PUSH, POP, SLOAD, SSTORE, CALL e JUMPI con formattazione corretta degli operandi e generazione di riferimenti incrociati. Il plugin Python consente di analizzare i bytecode dei smart contract nell'ambiente di disassemblaggio familiare di IDA. È rivolto ad auditor di sicurezza blockchain e ricercatori di smart contract che eseguono reverse engineering del bytecode EVM.
