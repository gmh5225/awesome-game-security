本プロジェクトは、Ethereum Virtual Machine（EVM）バイトコードを逆アセンブルするための IDA Pro プロセッサモジュールです。IDA に EVM 命令のデコードを追加し、PUSH、POP、SLOAD、SSTORE、CALL、JUMPI などのオペコードを適切なオペランド形式で表示し、クロスリファレンスも生成します。この Python プラグインにより、使い慣れた IDA の逆アセンブル環境でスマートコントラクトのバイトコードを解析できます。EVM バイトコードのリバースエンジニアリングを行うブロックチェーンセキュリティ監査者およびスマートコントラクト研究者を対象としています。
