Этот проект — утилита патчинга на C++, которая изменяет исполняемый файл сервера DayZ, чтобы отключить определённые проверки античита. Она автоматизирует сканирование бинарных сигнатур и применение патчей, поэтому обновления можно повторно обрабатывать без ручного hex-редактирования. Кодовая база лёгкая и сосредоточена на логике манипуляции исполняемыми файлами, а не на полноценном серверном фреймворке. Сценарий использования — реверс-инжиниринг и эксперименты по обходу античита в контролируемых тестовых средах.
