Этот проект — архивированное Android-приложение для обнаружения установок root-фреймворка Magisk на устройствах Android. Оно использует изолированный процесс через AppZygote и удалённое взаимодействие по AIDL, чтобы выполнять проверки вне песочницы основного приложения, выявляя артефакты Magisk и аномалии пространства имён монтирования. В первую очередь полезно инженерам античита и исследователям мобильной безопасности, изучающим методы обнаружения Magisk и задачи обхода root-детекции.
