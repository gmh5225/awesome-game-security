Este proyecto es una aplicación Android archivada diseñada para detectar instalaciones del framework root Magisk en dispositivos Android. Utiliza un proceso aislado mediante AppZygote y comunicación por servicio remoto AIDL para realizar comprobaciones de detección fuera del sandbox de la aplicación principal, buscando artefactos de Magisk y anomalías en el espacio de nombres de montaje. Resulta especialmente útil para ingenieros anticheat e investigadores de seguridad móvil que estudian técnicas de detección de Magisk y los desafíos de eludir la detección de root.
