Este proyecto es una aplicación Android que realiza attestation de claves respaldada por hardware para verificar la integridad del dispositivo y el estado del bootloader. Interactúa con las interfaces HAL de Android Keymaster y KeyMint mediante enlaces AIDL para obtener y validar certificados de attestation, comprobando bootloaders bloqueados, estados de verified boot y propiedades de claves aprovisionadas. Resulta especialmente útil para investigadores de seguridad Android e ingenieros anti-trampas que estudian los mecanismos de attestation por hardware y la verificación de integridad en Android.
