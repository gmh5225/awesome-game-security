Этот проект — утилита сбора памяти Linux, которая сохраняет системную память в удобные для анализа дамп-файлы.
Она написана на Rust и читает /proc/kcore для получения ELF core-вывода, с опциональной упаковкой в сжатый tar.zst для упрощения хранения и передачи.
Полученные дампы рассчитаны на прямую работу с распространёнными инструментами отладки и форензики, такими как gdb, crash и drgn.
Основной сценарий использования — реагирование на инциденты и форензика памяти на Linux-системах без необходимости в пользовательском модуле ядра.
