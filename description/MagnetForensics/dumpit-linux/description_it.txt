Questo progetto è un'utilità di acquisizione della memoria Linux che cattura la memoria di sistema in file di dump adatti all'analisi.
È scritta in Rust e legge /proc/kcore per produrre un output ELF core, con packaging tar.zst compresso opzionale per facilitare archiviazione e trasferimento.
I dump risultanti sono progettati per funzionare direttamente con strumenti comuni di debugging e forensics come gdb, crash e drgn.
Il caso d'uso principale è la risposta agli incidenti e la forensics della memoria sui sistemi Linux, senza richiedere un modulo kernel personalizzato.
