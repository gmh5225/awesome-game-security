Este proyecto es una utilidad de adquisición de memoria en Linux que captura la memoria del sistema en archivos de volcado aptos para el análisis.
Está escrita en Rust y lee /proc/kcore para producir una salida ELF core, con empaquetado opcional en tar.zst comprimido para facilitar el almacenamiento y la transferencia.
Los volcados resultantes están diseñados para funcionar directamente con herramientas habituales de depuración y forense como gdb, crash y drgn.
Su caso de uso principal es la respuesta a incidentes y la forense de memoria en sistemas Linux, sin requerir un módulo de kernel personalizado.
