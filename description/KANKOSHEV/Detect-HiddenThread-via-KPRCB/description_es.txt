Este proyecto es una prueba de concepto en el kernel de Windows para detectar hilos ocultos eliminados de la tabla de identificadores de procesos/hilos.
Recorre la información de hilos a través de estructuras relacionadas con KPRCB y verifica su presencia con comprobaciones de búsqueda de hilos.
La implementación se ofrece como un proyecto de controlador de kernel de Visual Studio escrito en C y C++.
Es útil para la monitorización de integridad anti-trampas y la investigación forense de bajo nivel.
