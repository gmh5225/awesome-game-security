Questo progetto è una proof of concept in kernel Windows per rilevare thread nascosti rimossi dalla tabella degli ID di processo/thread.
Attraversa le informazioni sui thread tramite strutture correlate a KPRCB e ne verifica la presenza con controlli di ricerca dei thread.
L’implementazione è fornita come progetto di driver kernel Visual Studio scritto in C e C++.
È utile per il monitoraggio dell’integrità anti-cheat e la ricerca forense a basso livello.
