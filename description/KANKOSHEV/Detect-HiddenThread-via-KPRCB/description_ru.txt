Этот проект — демонстрация концепции на уровне ядра Windows для обнаружения скрытых потоков, удалённых из таблицы идентификаторов процессов/потоков.
Он обходит сведения о потоках через структуры, связанные с KPRCB, и проверяет наличие потоков с помощью поиска потоков.
Реализация представлена как проект драйвера ядра Visual Studio на C и C++.
Полезен для мониторинга целостности античитов и низкоуровневых криминалистических исследований.
