Este proyecto es una prueba de concepto en modo kernel centrada en la detección de hooks de MouseClassServiceCallback.
Incluye código de manejo de hooks de bajo nivel y lógica de controlador para observar o validar las rutas de ejecución de callbacks del ratón.
El repositorio está estructurado como un proyecto de kernel de Windows construido con Visual Studio y componentes al estilo WDK.
Es principalmente relevante para la investigación de anti-trampas y detección de rootkits en torno a la interceptación de entrada.
