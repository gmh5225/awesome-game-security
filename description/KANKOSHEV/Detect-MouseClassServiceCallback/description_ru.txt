Этот проект — доказательство концепции в режиме ядра, посвящённое обнаружению хуков MouseClassServiceCallback.
Он включает низкоуровневый код обработки хуков и логику драйвера для наблюдения или проверки путей выполнения колбэков мыши.
Репозиторий организован как проект ядра Windows, собранный с помощью Visual Studio и компонентов в стиле WDK.
Он в основном релевантен для исследований античита и обнаружения руткитов в контексте перехвата ввода.
