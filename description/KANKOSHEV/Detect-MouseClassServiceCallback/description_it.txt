Questo progetto è una proof of concept in modalità kernel incentrata sul rilevamento degli hook di MouseClassServiceCallback.
Include codice di gestione degli hook a basso livello e logica del driver per osservare o validare i percorsi di esecuzione dei callback del mouse.
Il repository è strutturato come un progetto kernel Windows costruito con Visual Studio e componenti in stile WDK.
È principalmente rilevante per la ricerca anti-cheat e sul rilevamento di rootkit legati all'intercettazione dell'input.
