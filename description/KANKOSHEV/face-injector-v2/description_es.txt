Este proyecto es un framework de inyección en C++ que usa un controlador de kernel mapeado para inyectar procesos de juego.
Incluye lógica de dropper de payload, helpers de elevación de privilegios, rutas de staging aleatorizadas y el flujo de ejecución del mapper.
El repositorio menciona explícitamente varios objetivos de juego y advierte que usar este enfoque puede provocar bans.
Sirve como muestra educativa para estudiar técnicas de inyección respaldadas por drivers en la investigación de seguridad en juegos.
