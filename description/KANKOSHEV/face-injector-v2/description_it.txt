Questo progetto è un framework di injection in C++ che usa un driver kernel mappato per supportare l'iniezione nei processi di gioco.
Include la logica di dropper del payload, helper per l'elevazione dei privilegi, percorsi di staging randomizzati e il flusso di esecuzione del mapper.
Il repository menziona esplicitamente diversi target di gioco e avverte che l'uso di questo approccio può comportare ban.
Serve come campione didattico per studiare le tecniche di injection basate su driver nella ricerca di sicurezza nei giochi.
