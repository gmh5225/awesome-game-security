Ce projet est un échantillon de recherche noyau destiné à détecter l'utilisation de KeAttachProcess sous Windows.
Il énumère les processus et leurs threads, puis examine les données de contexte des threads pour identifier des processus cibles attachés de façon inattendue.
Le code est fourni sous forme de pilote noyau s'exécutant en continu, compilé avec les outils C et C++ de Visual Studio.
Il s'adresse aux scénarios d'anti-cheat et de surveillance de sécurité où l'attachement furtif de processus est pertinent.
