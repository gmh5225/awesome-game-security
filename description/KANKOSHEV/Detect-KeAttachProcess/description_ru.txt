Этот проект — исследовательский пример на уровне ядра для обнаружения использования KeAttachProcess в Windows.
Он перечисляет процессы и их потоки, затем анализирует данные контекста потоков, чтобы выявить неожиданно присоединённые целевые процессы.
Код оформлен как постоянно работающий драйвер ядра, собираемый средствами C и C++ в Visual Studio.
Он ориентирован на сценарии античита и мониторинга безопасности, где скрытое присоединение к процессам имеет значение.
