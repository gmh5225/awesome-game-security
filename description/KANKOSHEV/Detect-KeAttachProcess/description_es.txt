Este proyecto es una muestra de investigación en modo kernel para detectar el uso de KeAttachProcess en Windows.
Enumera procesos y sus hilos, y luego inspecciona los datos de contexto de los hilos para identificar procesos destino adjuntos de forma inesperada.
El código se empaqueta como un controlador de kernel en ejecución continua, compilado con las herramientas C y C++ de Visual Studio.
Está orientado a escenarios de anti-cheat y monitorización de seguridad donde el adjunto encubierto de procesos es relevante.
