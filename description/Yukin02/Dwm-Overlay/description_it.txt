Dwm-Overlay è un framework di overlay Windows in C++ che combina il rendering DWM/DirectX con un'interfaccia basata su ImGui.
Il progetto include componenti di hook a basso livello, tra cui stub in assembly per l'intercettazione del dispatch e la gestione del percorso Present.
La sua struttura di codice fornisce una base minimale per il disegno di overlay esterni, widget UI e l'integrazione di hook a runtime.
Il caso d'uso principale è la ricerca sul graphics hooking e la prototipazione di strumenti di overlay in contesti di sicurezza dei videogiochi.
