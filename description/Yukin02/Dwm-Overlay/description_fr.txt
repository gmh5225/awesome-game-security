Dwm-Overlay est un framework d'overlay Windows en C++ qui combine le rendu DWM/DirectX avec une interface basée sur ImGui.
Le projet inclut des composants de hook de bas niveau, notamment des stubs assembleur pour l'interception du dispatch et la gestion du chemin Present.
Sa structure de code fournit une base minimale pour le dessin d'overlays externes, les widgets d'UI et l'intégration de hooks à l'exécution.
Le cas d'usage principal est la recherche sur le hooking graphique et le prototypage d'outils d'overlay dans des contextes de sécurité des jeux.
