Dwm-Overlay es un framework de overlay para Windows en C++ que combina el renderizado DWM/DirectX con una interfaz basada en ImGui.
El proyecto incluye componentes de hook de bajo nivel, entre ellos stubs en ensamblador para la interceptación del dispatch y el manejo de la ruta Present.
Su estructura de código ofrece una base mínima para el dibujo de overlays externos, widgets de UI y la integración de hooks en tiempo de ejecución.
El caso de uso principal es la investigación de hooks gráficos y el prototipado de herramientas de overlay en contextos de seguridad de juegos.
