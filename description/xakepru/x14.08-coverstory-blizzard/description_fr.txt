Ce projet est un framework de triche pour World of Warcraft proposant un contournement de l'anti-triche Warden et des capacités d'injection de scripts. La base de code C++ inclut une architecture de hacks modulaire avec analyse de mémoire, patching de code, hooks du chargeur Warden et injection RunScript via un système HacksController basé sur des plugins. Il est principalement utile aux chercheurs en sécurité des jeux étudiant les internals de l'anti-triche Warden et les techniques de manipulation du client de jeu Blizzard.
