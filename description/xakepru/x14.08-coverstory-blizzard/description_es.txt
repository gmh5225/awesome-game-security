Este proyecto es un framework de trucos para World of Warcraft con bypass del anti-trampas Warden y capacidades de inyección de scripts. La base de código en C++ incluye una arquitectura modular de hacks con escaneo de memoria, parcheo de código, hooks del cargador Warden e inyección RunScript a través de un sistema HacksController basado en plugins. Resulta especialmente útil para investigadores de seguridad en juegos que estudian los internals del anti-trampas Warden y las técnicas de manipulación del cliente de juegos de Blizzard.
