Ce projet est un mappeur de pilotes noyau Windows qui utilise le pilote vulnérable S4 de Samsung pour charger des pilotes non signés en mémoire noyau. Il exploite les IOCTL d’accès mémoire de S4 pour mapper manuellement un pilote personnalisé : allocation de mémoire de pool, copie des sections, correction des relocations, résolution des imports et appel du point d’entrée. Il s’adresse aux chercheurs noyau étudiant le mapping de pilotes via les primitives BYOVD de Samsung.
