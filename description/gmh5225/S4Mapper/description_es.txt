Este proyecto es un mapper de controladores del kernel de Windows que usa el controlador vulnerable S4 de Samsung para cargar controladores sin firmar en la memoria del kernel. Aprovecha los IOCTL de acceso a memoria de S4 para mapear manualmente un controlador personalizado: asignar memoria de pool, copiar secciones, corregir reubicaciones, resolver importaciones e invocar el punto de entrada. Está orientado a investigadores del kernel que estudian el mapeo de controladores mediante primitivas BYOVD de Samsung.
