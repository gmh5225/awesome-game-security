Этот проект — маппер драйверов ядра Windows, использующий уязвимый драйвер Samsung S4 для загрузки неподписанных драйверов в память ядра. Он эксплуатирует IOCTL доступа к памяти S4 для ручного маппинга пользовательского драйвера: выделение pool-памяти, копирование секций, исправление релокаций, разрешение импортов и вызов точки входа. Предназначен для исследователей ядра, изучающих маппинг драйверов через примитивы Samsung BYOVD.
