Este proyecto es un kit de herramientas orientado al kernel para extraer certificados: analiza datos Authenticode de PE y obtiene el sujeto y la ventana de validez del certificado de firma principal.
Su archivo se divide en un proyecto de controlador y una biblioteca reutilizable `Lib-SoulExtraction`, que incorpora código adaptado de parsing PKCS#7, ASN.1 y X.509 de Linux junto con envoltorios específicos de Windows para el acceso a archivos en el kernel y la conversión de cadenas.
La rutina central recorre el directorio de firmas PE, analiza el mensaje PKCS#7, elige el certificado primario probable y devuelve campos como el nombre del firmante y las marcas temporales de validez, en lugar de limitarse a comprobar si un archivo está firmado.
Resulta especialmente útil para desarrolladores del kernel de Windows e ingenieros de ingeniería inversa que necesitan extraer en el kernel metadatos de certificados desde archivos PE para telemetría, análisis de confianza o herramientas de triaje.
