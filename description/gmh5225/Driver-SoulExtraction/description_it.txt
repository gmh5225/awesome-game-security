Questo progetto è un toolkit orientato al kernel per l’estrazione di certificati che analizza i dati Authenticode PE e ne ricava soggetto e finestra di validità del certificato di firma principale.
L’archivio è suddiviso in un progetto driver e in una libreria riusabile `Lib-SoulExtraction`, che incorpora codice di parsing PKCS#7, ASN.1 e X.509 Linux adattato insieme a wrapper specifici per Windows per l’accesso ai file in kernel e la conversione delle stringhe.
La routine centrale attraversa la directory delle firme PE, analizza il messaggio PKCS#7, sceglie il certificato primario probabile e restituisce campi come il nome del firmatario e i timestamp di validità, invece di limitarsi a verificare se un file è firmato.
È soprattutto utile per sviluppatori del kernel Windows e reverse engineer che devono estrarre nel kernel metadati dei certificati da file PE per telemetria, analisi della fiducia o strumenti di triage.
