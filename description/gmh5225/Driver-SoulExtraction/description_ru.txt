Этот проект — ориентированный на ядро набор инструментов для извлечения сертификатов: он разбирает данные PE Authenticode и извлекает субъект и окно действия основного сертификата подписи.
Архив разделён на проект драйвера и переиспользуемую библиотеку `Lib-SoulExtraction`; в библиотеке встроен адаптированный код разбора PKCS#7, ASN.1 и X.509 из Linux, а также специфичные для Windows обёртки для доступа к файлам в ядре и преобразования строк.
Основная процедура обходит каталог подписей PE, разбирает сообщение PKCS#7, выбирает вероятный первичный сертификат и возвращает поля вроде имени подписанта и временных меток срока действия, а не просто проверяет, подписан ли файл.
Она в первую очередь полезна разработчикам ядра Windows и специалистам по реверс-инжинирингу, которым нужно извлекать в ядре метаданные сертификатов из PE-файлов для телеметрии, анализа доверия или инструментов triage.
