本项目是一套面向内核的证书提取工具包，可解析 PE Authenticode 数据并提取主签名证书的主体与有效期窗口。
其归档分为驱动项目与可复用的 `Lib-SoulExtraction` 库：库中嵌入了经适配的 Linux PKCS#7、ASN.1 与 X.509 解析代码，并配有面向内核文件访问与字符串转换的 Windows 专用封装。
核心流程会遍历 PE 签名目录、解析 PKCS#7 消息、选出可能的主证书，并返回签名者名称与有效期时间戳等字段，而不仅仅是检查文件是否已签名。
它主要面向需要在内核中从 PE 文件提取证书元数据以用于遥测、信任分析或研判工具的 Windows 内核开发者与逆向工程师。
