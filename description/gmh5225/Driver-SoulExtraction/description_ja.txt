本プロジェクトは、PE Authenticode データを解析し、主署名証明書のサブジェクトと有効期間を取り出す、カーネル向け証明書抽出ツールキットです。
アーカイブはドライバプロジェクトと再利用可能な `Lib-SoulExtraction` ライブラリに分かれており、ライブラリには適応済みの Linux PKCS#7、ASN.1、X.509 解析コードと、カーネルでのファイルアクセスおよび文字列変換用の Windows 固有ラッパーが組み込まれています。
中核ルーチンは PE 署名ディレクトリを辿り、PKCS#7 メッセージを解析し、想定される主証明書を選び、署名済みかどうかの判定にとどまらず、署名者名や有効期限タイムスタンプなどのフィールドを返します。
主に、テレメトリ、信頼性分析、トリアージツール向けに PE ファイルからカーネル内で証明書メタデータを抽出する必要がある Windows カーネル開発者およびリバースエンジニアに役立ちます。
