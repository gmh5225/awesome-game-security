本專案是一套面向核心的憑證擷取工具組，可解析 PE Authenticode 資料並取出主簽章憑證的主體與有效期限區間。
其封存內容分為驅動程式專案與可重用的 `Lib-SoulExtraction` 函式庫：函式庫內嵌經調適的 Linux PKCS#7、ASN.1 與 X.509 解析程式碼，並附有用於核心檔案存取與字串轉換的 Windows 專用包裝層。
核心常式會走訪 PE 簽章目錄、解析 PKCS#7 訊息、選出可能的主要憑證，並回傳簽署者名稱與有效期時間戳等欄位，而不只是檢查檔案是否已簽署。
它主要適用於需要在核心中從 PE 檔擷取憑證中繼資料以用於遙測、信任分析或分流研判工具的 Windows 核心開發者與逆向工程師。
