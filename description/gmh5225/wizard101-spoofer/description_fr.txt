Ce projet est un spoofer d'identifiant matériel (HWID) pour Wizard101 qui modifie les identifiants système afin de contourner les bans matériels. Il change les numéros de série de disque, les adresses MAC, les UUID de carte mère et d'autres empreintes matérielles que le système de ban de Wizard101 utilise pour identifier les joueurs bannis. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les systèmes de ban basés sur le HWID et les techniques de spoofing.
