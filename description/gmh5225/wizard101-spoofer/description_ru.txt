Этот проект — спуфер аппаратного идентификатора (HWID) для Wizard101, который изменяет системные идентификаторы, чтобы обходить аппаратные баны. Он меняет серийные номера дисков, MAC-адреса, UUID материнской платы и другие аппаратные отпечатки, которые система банов Wizard101 использует для идентификации заблокированных игроков. Проект предназначен для исследователей безопасности игр, изучающих системы банов на основе HWID и техники спуфинга.
