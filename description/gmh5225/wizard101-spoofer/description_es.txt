Este proyecto es un spoofer de identificador de hardware (HWID) para Wizard101 que modifica los identificadores del sistema para evadir bans de hardware. Cambia números de serie de disco, direcciones MAC, UUID de la placa base y otras huellas de hardware que el sistema de bans de Wizard101 usa para identificar a jugadores baneados. Está dirigido a investigadores de seguridad en videojuegos que estudian sistemas de ban basados en HWID y técnicas de spoofing.
