Este proyecto es un software de entrenamiento gratuito y de código abierto para Team Fortress 2, basado en SE-Owned.
Está escrito principalmente en C/C++ y C++ y se centra en el renderizado, la red y la animación.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área cheat / game:team fortress 2.
