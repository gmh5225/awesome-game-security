Ce projet est un exploit pour le pilote noyau Windows vulnérable ampa.sys. Il montre comment l'interface IOCTL non sécurisée du pilote peut être abusée pour obtenir une lecture/écriture arbitraire en noyau ou une exécution de code, permettant le chargement de pilotes non signés, l'élévation de privilèges de processus ou le contournement d'anti-cheat. L'exploit en C/C++ automatise le déclenchement de la vulnérabilité. Il s'adresse aux chercheurs BYOVD qui étudient l'exploitation de pilotes vulnérables pour un accès au niveau noyau.
