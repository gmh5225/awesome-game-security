Этот проект — эксплойт для уязвимого драйвера ядра Windows ampa.sys. Он показывает, как небезопасный интерфейс IOCTL драйвера можно использовать для произвольного чтения/записи в ядре или выполнения кода, что позволяет загружать неподписанные драйверы, повышать привилегии процессов или обходить античит. Эксплойт на C/C++ автоматизирует срабатывание уязвимости. Предназначен для исследователей BYOVD, изучающих эксплуатацию уязвимых драйверов для доступа на уровне ядра.
