이 프로젝트는 일반적인 커널 모듈 로딩 경로 대신 드라이버 이미지를 세션 메모리에 수동으로 로드하는 세션 공간 드라이버 매퍼입니다.
보관된 README는 이를 세션 공간으로의 드라이버 매핑으로 설명하며, 솔루션은 커널 구성 요소와 원시 이미지 데이터를 전달해 임포트 수정, 섹션 복사, 재배치, 엔트리포인트 실행을 수행하는 클라이언트를 함께 사용합니다.
주요 훅은 내부 `ntoskrnl` 콜백 경로를 대체하고, 대상 이미지용 세션 공간 메모리를 할당하며, 임포트와 재배치를 해석한 뒤 언로드 시 로더 메타데이터의 일부를 지워 매핑된 드라이버의 가시성을 낮춥니다.
주로 세션 공간으로의 수동 매핑, 대체 드라이버 로딩 흐름, 일반 로더 구조에서 매핑된 모듈을 숨기는 기법을 연구하는 Windows 커널 연구자에게 유용합니다.
