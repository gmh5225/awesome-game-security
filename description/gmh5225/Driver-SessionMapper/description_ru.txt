Этот проект — загрузчик (mapper) драйверов в пространство сессии, который вручную загружает образ драйвера в память сессии вместо обычного пути загрузки модулей ядра.
Архивный README описывает его как отображение драйверов в пространство сессии; решение сочетает компонент ядра с клиентом, который передаёт сырые данные образа для исправления импортов, копирования секций, релокации и выполнения точки входа.
Основной хук заменяет внутренний путь callback в `ntoskrnl`, выделяет память пространства сессии для целевого образа, разрешает импорты и релокации, а при выгрузке очищает часть метаданных загрузчика, чтобы снизить видимость отображённого драйвера.
Он в первую очередь полезен исследователям ядра Windows, изучающим ручное отображение в пространство сессии, альтернативные потоки загрузки драйверов и техники сокрытия отображённых модулей от обычных структур загрузчика.
