Этот проект посвящён aVhook. AVhook — обучающее ПО для Counter-Strike: Global Offensive под Windows.
AVhook также включает несколько шуточных функций.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / CS:GO.
