Этот проект — инструмент для дампа runtime-модулей сканирования shellcode BattlEye из защищённых игровых процессов. Он перехватывает shellcode-полезные нагрузки, которые BattlEye передаёт и исполняет в игровых процессах для обнаружения читов, и сохраняет их для офлайн-анализа методом обратной разработки. Дампнутые модули раскрывают сигнатуры обнаружения и логику сканирования BattlEye. Предназначен для исследователей античита, изучающих методологию детектирования BattlEye.
