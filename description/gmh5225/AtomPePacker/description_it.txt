Questo progetto è AtomPePacker, un packer di eseguibili PE Windows che comprime e cifra i file PE con uno stub di unpacking a runtime. Impacchetta le sezioni PE originali, le cifra e antepone un loader di decompressione che ripristina il binario originale in memoria a runtime. Il packer gestisce import, relocation e callback TLS durante l'unpacking. È pensato per i ricercatori di protezione software che studiano le tecniche di packing PE.
