Этот проект — proof-of-concept на Rust, демонстрирующий уязвимость отображения памяти ядра в драйвере ASUS bsitf.sys / AsusBSItf.sys (CVE-2026-13585). Он открывает устройство \\.\bsitf и с помощью IOCTL 0x222808 выделяет физически непрерывную память пула ядра размера, заданного атакующим, отображает её в вызывающий процесс с полным доступом usermode на чтение/запись и возвращает виртуальный и физический адреса. PoC может проверить отображение записью тестового шаблона, затем освободить его через IOCTL 0x22280C и опирается на Windows API через crate windows. Проблема представляет собой повышение привилегий от администратора к ядру и полезна в сценариях Bring Your Own Vulnerable Driver (BYOVD); последствия включают исчерпание NonPagedPool, раскрытие физических адресов и на старых сборках подготовку исполняемой памяти ядра. Предназначен для уполномоченных исследователей безопасности, изучающих уязвимые подписанные драйверы, злоупотребление kernel IOCTL и техники повышения привилегий в Windows.
