本项目是一个 Rust 概念验证（PoC），用于演示 ASUS bsitf.sys / AsusBSItf.sys 驱动中的内核内存映射漏洞（CVE-2026-13585）。它打开 \\.\bsitf 设备，并通过 IOCTL 0x222808 分配攻击者指定大小的物理连续内核池内存，将其以完整用户态读写权限映射到调用进程，并返回虚拟地址与物理地址。该 PoC 可通过写入测试图案来验证映射，再经 IOCTL 0x22280C 释放，并借助 windows crate 调用 Windows API。该问题属于管理员到内核的权限提升，适用于自带漏洞驱动（BYOVD）场景，影响包括 NonPagedPool 耗尽、物理地址泄露，以及在较旧系统上可用于可执行内核内存暂存。项目面向经授权的安全研究人员，用于研究有漏洞的签名驱动、内核 IOCTL 滥用以及 Windows 权限提升技术。
