Этот проект — репозиторий из раздела cheat / game:rust, посвящённый immortal rust.
Он в основном написан на C/C++ и C++ и сосредоточен на OpenGL и генерации SDK.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:rust.
