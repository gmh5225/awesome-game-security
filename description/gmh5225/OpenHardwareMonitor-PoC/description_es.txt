Este proyecto es una prueba de concepto mínima de una vulnerabilidad en OpenHardwareMonitorLib.sys que expone el acceso directo a registros específicos del modelo (MSR) en modo de usuario.
El código abre el dispositivo y encapsula dos IOCTL, 0x9C402084 y 0x9C402088, para leer y escribir MSR arbitrarios mediante sencillas funciones auxiliares read_msr y write_msr.
Como el repositorio es deliberadamente pequeño y se centra en la interacción con el controlador, funciona más como una demostración enfocada del primitivo expuesto que como un marco de explotación más amplio.
Resulta especialmente útil para investigadores de seguridad de Windows que estudian controladores de monitorización de hardware vulnerables, fallos de exposición de MSR y el riesgo de acceso en modo de usuario a controles privilegiados de la CPU.
