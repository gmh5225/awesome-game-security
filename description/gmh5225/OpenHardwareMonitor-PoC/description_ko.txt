이 프로젝트는 OpenHardwareMonitorLib.sys의 취약점에 대한 최소 개념 증명으로, 해당 취약점은 모델별 레지스터(MSR)에 대한 직접 접근을 사용자 모드에 노출합니다.
코드는 디바이스를 열고 두 개의 IOCTL(0x9C402084 및 0x9C402088)을 래핑하여, 간단한 read_msr 및 write_msr 헬퍼 함수로 임의의 MSR을 읽고 씁니다.
저장소가 의도적으로 작고 드라이버 상호작용 자체에 초점을 맞추고 있으므로, 대규모 익스플로잇 프레임워크라기보다 노출된 프리미티브에 대한 집중 데모에 가깝습니다.
취약한 하드웨어 모니터링 드라이버, MSR 노출 버그, 특권 CPU 제어에 대한 사용자 모드 접근 위험을 연구하는 Windows 보안 연구원에게 주로 유용합니다.
