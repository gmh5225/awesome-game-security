本プロジェクトは、OpenHardwareMonitorLib.sys の脆弱性に対する最小限の概念実証であり、当該脆弱性はモデル固有レジスタ（MSR）への直接アクセスをユーザーモードに公開します。
コードはデバイスを開き、二つの IOCTL（0x9C402084 と 0x9C402088）をラップして、シンプルな read_msr および write_msr ヘルパー関数経由で任意の MSR を読み書きします。
リポジトリは意図的に小さく、ドライバとのやり取り自体に焦点を当てているため、大規模なエクスプロイトフレームワークというより、公開されたプリミティブの集中的なデモとして機能します。
脆弱なハードウェア監視ドライバ、MSR 露出バグ、特権 CPU 制御へのユーザーモードアクセスのリスクを研究する Windows セキュリティ研究者にとって特に有用です。
