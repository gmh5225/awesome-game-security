Этот проект — утилита захвата экрана для Windows, демонстрирующая различные методы создания скриншотов, включая GDI BitBlt, DXGI Desktop Duplication, PrintWindow и захват эскизов DWM. Она сравнивает разные подходы к захвату по способности захватывать содержимое оверлеев, работать с аппаратно ускоренными окнами и взаимодействовать с системами обнаружения скриншотов античитов. Предназначена для разработчиков античитов и исследователей безопасности, изучающих обнаружение на основе скриншотов и техники захвата экрана.
