Este proyecto es una herramienta para encontrar hilos del sistema ocultos que el anti-cheat de BattlEye no puede detectar. Demuestra técnicas avanzadas de enumeración de hilos más allá de las llamadas API estándar para localizar hilos furtivos creados por controladores mapeados manualmente, mediante el recorrido de PspCidTable, el escaneo de colas del planificador o el cruce de listas de hilos. Está dirigido a investigadores de anti-cheat que estudian técnicas de ocultación y detección de hilos relacionadas con BattlEye.
