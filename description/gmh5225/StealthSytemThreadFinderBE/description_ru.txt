Этот проект — инструмент для поиска скрытых системных потоков, которые античит BattlEye не может обнаружить. Он демонстрирует продвинутые методы перечисления потоков, выходящие за рамки стандартных вызовов API, для обнаружения скрытых потоков, созданных вручную отображёнными драйверами, с помощью обхода PspCidTable, сканирования очередей планировщика или перекрёстной проверки списков потоков. Предназначен для исследователей античита, изучающих техники сокрытия и обнаружения потоков в контексте BattlEye.
