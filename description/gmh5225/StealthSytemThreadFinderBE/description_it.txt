Questo progetto è uno strumento per trovare thread di sistema nascosti che l'anti-cheat di BattlEye non riesce a rilevare. Mostra tecniche avanzate di enumerazione dei thread oltre le chiamate API standard per individuare thread stealth creati da driver mappati manualmente, usando approcci come la scansione di PspCidTable, l'analisi delle code dello scheduler o il confronto incrociato delle liste di thread. È rivolto ai ricercatori di anti-cheat che studiano tecniche di occultamento e rilevamento dei thread legate a BattlEye.
