Dieses Projekt ist SCUE4, ein Unreal-Engine-4-Plugin, das UE4-Spielen Anti-Cheat- und Sicherheitsfunktionen hinzufügt. Es bietet Laufzeit-Integritätsprüfungen, Speicherschutz für sensible Spielvariablen, Verschlüsselung von Netzwerkpaketen und Manipulationserkennung für Spielbinärdateien. Das C++-Plugin integriert sich als Spielmodul in das Build-System von UE4. Es richtet sich an UE4-Spieleentwickler, die Anti-Cheat-Schutz hinzufügen, sowie an Gamesicherheitsforscher, die sicherheitsbezogene Plugins auf Engine-Ebene untersuchen.
