Questo progetto è SCUE4, un plugin per Unreal Engine 4 che aggiunge funzionalità anti-cheat e di sicurezza ai giochi UE4. Fornisce controlli di integrità a runtime, protezione della memoria per le variabili di gioco sensibili, cifratura dei pacchetti di rete e rilevamento di manomissioni dei binari di gioco. Il plugin C++ si integra nel sistema di build di UE4 come modulo di gioco. È rivolto agli sviluppatori di giochi UE4 che aggiungono protezione anti-cheat e ai ricercatori di sicurezza nei giochi che studiano plugin di sicurezza a livello di motore.
