Este proyecto es SCUE4, un plugin de Unreal Engine 4 que añade funciones anti-trampas y de seguridad a juegos UE4. Proporciona comprobaciones de integridad en tiempo de ejecución, protección de memoria para variables sensibles del juego, cifrado de paquetes de red y detección de manipulación de binarios del juego. El plugin en C++ se integra en el sistema de compilación de UE4 como un módulo de juego. Está dirigido a desarrolladores de juegos UE4 que añaden protección anti-trampas y a investigadores de seguridad en juegos que estudian plugins de seguridad a nivel de motor.
