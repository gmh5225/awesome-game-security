Dieses Projekt ist QSynthesis, ein IDA-Pro-Plugin zur Synthese vereinfachter Ausdrücke aus komplexem obfuscated Code mittels Programmsynthese. Es nutzt oracle-geleitete Synthese und SMT-Solving, um einfachere äquivalente Ausdrücke für MBA-(Mixed-Boolean-Arithmetic-)obfuscated Berechnungen in geschützten Binaries zu finden. Das Python-Plugin integriert sich in die Decompiler-Ausgabe von IDA. Es richtet sich an Reverse Engineers, die VM-geschützten oder MBA-obfuscated Code in IDA Pro deobfuscieren.
