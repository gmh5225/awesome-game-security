Этот проект — QSynthesis, плагин для IDA Pro, который синтезирует упрощённые выражения из сложного обфусцированного кода с помощью методов программного синтеза. Он применяет oracle-guided synthesis и SMT-решение, чтобы находить более простые эквивалентные выражения для MBA (Mixed Boolean-Arithmetic) обфусцированных вычислений в защищённых бинарниках. Python-плагин интегрируется с выводом декомпилятора IDA. Предназначен для реверс-инженеров, деобфусцирующих VM-защищённый или MBA-обфусцированный код в IDA Pro.
