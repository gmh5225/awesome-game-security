Ce projet est QSynthesis, un plugin IDA Pro qui synthétise des expressions simplifiées à partir de code fortement obfuscqué grâce aux techniques de synthèse de programmes. Il combine une synthèse guidée par oracle et la résolution SMT pour trouver des expressions équivalentes plus simples aux calculs obfuscqués MBA (Mixed Boolean-Arithmetic) présents dans les binaires protégés. Ce plugin Python s'intègre à la sortie du décompilateur d'IDA. Il s'adresse aux reverse engineers qui déobfusquent du code protégé par VM ou obfuscqué MBA dans IDA Pro.
