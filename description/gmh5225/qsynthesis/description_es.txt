Este proyecto es QSynthesis, un plugin de IDA Pro que sintetiza expresiones simplificadas a partir de código ofuscado complejo mediante técnicas de síntesis de programas. Aplica síntesis guiada por oráculo y resolución SMT para hallar expresiones equivalentes más simples a cálculos ofuscados con MBA (Mixed Boolean-Arithmetic) en binarios protegidos. El plugin en Python se integra con la salida del descompilador de IDA. Está pensado para ingenieros inversos que desofuscan código protegido por VM o ofuscado con MBA en IDA Pro.
