Questo progetto è Cheap Engine, un motore di cheat / scanner di memoria leggero per giochi, simile a Cheat Engine ma con un set di funzionalità semplificato. Offre scansione della memoria dei processi, ricerca di valori (esatto, intervallo, modificato/invariato), modifica della memoria e scansione dei puntatori. Lo strumento funge da alternativa più semplice a Cheat Engine per la modifica di base della memoria di gioco. È pensato per principianti del game hacking e per ricercatori che studiano le implementazioni degli scanner di memoria.
