Este proyecto es Cheap Engine, un motor de trucos / escáner de memoria ligero para juegos, similar a Cheat Engine pero con un conjunto de funciones simplificado. Ofrece escaneo de memoria de procesos, búsqueda de valores (exacto, rango, cambiado/sin cambios), edición de memoria y escaneo de punteros. La herramienta sirve como alternativa más sencilla a Cheat Engine para la modificación básica de memoria de juegos. Está dirigida a principiantes en game hacking y a investigadores que estudian implementaciones de escáneres de memoria.
