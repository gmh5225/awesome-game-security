Этот проект — Cheap Engine, лёгкий игровой чит-движок / сканер памяти, похожий на Cheat Engine, но с упрощённым набором функций. Он обеспечивает сканирование памяти процессов, поиск значений (точное, диапазон, изменённое/неизменённое), редактирование памяти и сканирование указателей. Инструмент служит более простой альтернативой Cheat Engine для базовой модификации игровой памяти. Он ориентирован на новичков в game hacking и исследователей, изучающих реализацию сканеров памяти.
