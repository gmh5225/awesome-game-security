이 프로젝트는 \Driver\PEAUTH 의 IRP_MJ_FLUSH_BUFFERS 를 트리거로 재사용하고, 실제 요청 데이터는 공유 사용자 버퍼에 두는 커널·유저 통신 샘플입니다.
드라이버 엔트리는 대상 드라이버 객체를 해석한 뒤 InterlockedExchangePointer 로 MajorFunction[IRP_MJ_FLUSH_BUFFERS] 를 교체하고, \Registry\Machine\SOFTWARE\ucflash 아래 레지스트리 값에서 공유 버퍼 포인터와 클라이언트 PID 를 다시 로드합니다.
훅은 MmCopyVirtualMemory 로 REQUEST_DATA 블록을 복사하고, 세션 동기화를 위해 매직 값을 순환시키며, 물리 메모리 접근 코드를 뒷받침하는 헬퍼 콜백을 통해 read, write, protect, alloc, free, module, main-base 작업을 디스패치합니다.
하이재킹된 IRP 경로, 레지스트리로 시드되는 공유 버퍼, 경량 커널 메모리 서비스를 기반으로 한 대안 드라이버 통신 채널을 비교하는 Windows 커널 연구자에게 주로 유용합니다.
