Этот проект демонстрирует внедрение кода через ключи реестра Windows NLS (National Language Support). Он изменяет связанные с NLS записи реестра, перенаправляя загрузку DLL перевода кодовых страниц NLS, из‑за чего пользовательские DLL загружаются в процессы, инициализирующие подсистему NLS. Эта техника закрепления загружает код на раннем этапе инициализации процесса. Она предназначена для исследователей red team, изучающих внедрение кода и закрепление на основе NLS.
