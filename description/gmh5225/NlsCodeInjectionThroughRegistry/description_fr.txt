Ce projet démontre une injection de code via les clés de registre Windows NLS (National Language Support). Il modifie les entrées de registre liées à NLS afin de rediriger le chargement des DLL de traduction de pages de codes NLS, ce qui provoque le chargement de DLL personnalisées dans les processus qui initialisent le sous-système NLS. Cette technique de persistance charge du code tôt dans l'initialisation du processus. Elle s'adresse aux chercheurs red team étudiant l'injection de code et la persistance basées sur NLS.
