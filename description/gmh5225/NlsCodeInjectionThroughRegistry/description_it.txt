Questo progetto dimostra l'iniezione di codice tramite le chiavi di registro Windows NLS (National Language Support). Modifica le voci di registro relative a NLS per reindirizzare il caricamento delle DLL di traduzione delle code page NLS, facendo sì che DLL personalizzate vengano caricate nei processi che inizializzano il sottosistema NLS. Questa tecnica di persistenza carica codice nelle fasi iniziali dell'inizializzazione del processo. È rivolta ai ricercatori red team che studiano tecniche di iniezione di codice e persistenza basate su NLS.
