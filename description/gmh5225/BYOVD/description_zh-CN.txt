本项目是一个精选的概念验证合集，展示如何滥用多个已签名的漏洞驱动来终止或禁用 AV 与 EDR 组件。
顶层 README 将其定位为教学向的 BYOVD 实验环境，所含子项目分别针对 viragt64.sys、TfSysMon、ksapi64、BdApiUtil 与 wsftprm 等驱动，并提供专用的终止例程。
Viragt64 分支还说明，该 PoC 是在类似驱动滥用开始出现于真实攻击活动之后发布的，从而为仓库提供了超出孤立驱动样本之外的实用作战上下文。
它主要适合研究 BYOVD 技战术、利用已签名驱动进行进程终止滥用，以及公开 PoC 如何对应后续野外所见驱动的 Windows 安全研究人员。
