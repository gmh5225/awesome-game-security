Este proyecto es una colección curada de pruebas de concepto que muestran cómo se pueden abusar de varios controladores firmados vulnerables para terminar o desactivar componentes de AV y EDR.
El README de nivel superior lo presenta como un laboratorio educativo de BYOVD, y los subproyectos incluidos apuntan a controladores como viragt64.sys, TfSysMon, ksapi64, BdApiUtil y wsftprm con rutinas de terminación dedicadas.
La rama Viragt64 también explica que el PoC se publicó después de que abusos similares de controladores empezaran a aparecer en campañas reales, lo que aporta al repositorio un contexto operativo útil más allá de muestras aisladas.
Resulta especialmente útil para investigadores de seguridad de Windows que estudian las tácticas BYOVD, el abuso de terminación de procesos con controladores firmados y cómo los PoC públicos se corresponden con controladores observados después en entornos reales.
