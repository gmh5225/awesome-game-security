本專案是一套精選的概念驗證合集，展示如何濫用多個已簽章的漏洞驅動程式來終止或停用 AV 與 EDR 元件。
頂層 README 將其定位為教學向的 BYOVD 實驗環境，所含子專案分別針對 viragt64.sys、TfSysMon、ksapi64、BdApiUtil 與 wsftprm 等驅動，並提供專用的終止常式。
Viragt64 分支亦說明，該 PoC 是在類似驅動濫用開始出現於真實攻擊活動之後才發布，因此為倉庫提供了超越孤立驅動樣本之外的實用作戰脈絡。
它主要適合研究 BYOVD 技戰術、利用已簽章驅動進行行程終止濫用，以及公開 PoC 如何對應後續野外所見驅動的 Windows 安全性研究人員。
