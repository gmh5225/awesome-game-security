本プロジェクトは、複数の署名済み脆弱なドライバを悪用して AV および EDR コンポーネントを終了または無効化する方法を示す概念実証（PoC）のキュレーション集です。
トップレベルの README では教育用の BYOVD ラボとして位置づけられており、含まれるサブプロジェクトは viragt64.sys、TfSysMon、ksapi64、BdApiUtil、wsftprm などのドライバを対象に、専用のキル処理を備えています。
Viragt64 ブランチでは、同様のドライバ悪用が実世界のキャンペーンで現れ始めた後に PoC が公開されたことも説明されており、孤立したドライバサンプルを超えた運用上の文脈を与えています。
主に、BYOVD の手口、署名済みドライバによるプロセス終了の悪用、および公開 PoC が後に実環境で見られるドライバとどう対応するかを研究する Windows セキュリティ研究者向けです。
