이 프로젝트는 여러 서명된 취약 드라이버를 악용해 AV 및 EDR 구성 요소를 종료하거나 비활성화하는 방법을 보여주는 개념 증명(PoC) 모음입니다.
최상위 README는 이를 교육용 BYOVD 랩으로 소개하며, 포함된 하위 프로젝트는 viragt64.sys, TfSysMon, ksapi64, BdApiUtil, wsftprm 등의 드라이버를 대상으로 전용 종료 루틴을 제공합니다.
Viragt64 브랜치는 유사한 드라이버 악용이 실제 캠페인에 나타나기 시작한 뒤에 PoC가 공개되었다고 설명하여, 고립된 드라이버 샘플을 넘어선 유용한 운영 맥락을 제공합니다.
주로 BYOVD 전술, 서명된 드라이버를 이용한 프로세스 종료 악용, 그리고 공개 PoC가 이후 현장에서 발견된 드라이버와 어떻게 대응하는지 연구하는 Windows 보안 연구원에게 유용합니다.
