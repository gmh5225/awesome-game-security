Этот проект — скрипт автоматического повышения уровня для Maple Story Artale.
Он в основном написан на Python и сосредоточен на анимации.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:maplestory.
