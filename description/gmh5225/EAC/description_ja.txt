このアーカイブは、リバースされた easyanticheat.sys のコードと、付随する EAC または EOS SDK のドロップを含む、Easy Anti-Cheat の混合リファレンスバンドルです。
リバースされたソースは、ドライバーディスパッチ検証、Process Hacker ドライバー検出、ブラックリストドライバー検査、ntoskrnl パッチスキャン、ドライバーハッシュ処理といったカーネルコールバックロジックに焦点を当てています。
あわせて EAC_SDK ツリーには SDK バイナリ、ヘッダー、EOS_FileDecryptionTool などのツールが含まれており、このリポジトリは単一の実行可能プロジェクトというより、EAC 内部と統合成果物の学習パックとして機能します。
主に、Easy Anti-Cheat のカーネルヒューリスティクス、ドライバー整合性監視、および EAC / EOS デプロイで使われる周辺 SDK サーフェスを研究するリバースエンジニア向けです。
