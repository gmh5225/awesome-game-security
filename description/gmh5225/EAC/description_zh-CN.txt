该归档是一份混合的 Easy Anti-Cheat 参考合集，既包含已逆向的 easyanticheat.sys 代码，也附带 EAC 或 EOS SDK 投放内容。
逆向源码侧重点在内核回调逻辑，例如驱动分发验证、Process Hacker 驱动检测、黑名单驱动检查、ntoskrnl 补丁扫描以及驱动哈希例程。
与此同时，EAC_SDK 目录包含 SDK 二进制、头文件以及 EOS_FileDecryptionTool 等工具，因此该仓库更像是研究 EAC 内部机制与集成产物的学习包，而非单一可执行项目。
它主要面向逆向工程师，用于研究 Easy Anti-Cheat 内核启发式、驱动完整性监控，以及 EAC 与 EOS 部署所使用的相关 SDK 接口面。
