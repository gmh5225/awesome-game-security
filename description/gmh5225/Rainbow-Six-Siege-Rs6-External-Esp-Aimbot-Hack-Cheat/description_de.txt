Dieses Projekt enthält den geleakten Quellcode eines Cheats. Der Cheat ist derzeit undetected, aber ich übernehme keine Verantwortung, falls er später erkannt wird und Sie vom Spiel gebannt werden. Scavs (Weiß), PMCs (Weiß, Pink)), ScavBoss (Grün). Durch ausschließliches Hooken von WndProc — aufgerufen bei Mausbewegungen / Tasteneingaben — werden erkannte Vektoren vermieden. Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / game:r6 untersuchen.
