Este proyecto contiene el código fuente filtrado de un cheat. El cheat está actualmente indetectado, pero no me hago responsable si más adelante es detectado y te banean del juego. Scavs (blanco), PMCs (blanco, rosa)), ScavBoss (verde). Evita el uso de vectores detectados enganchando únicamente WndProc, que se llama cuando hay movimiento del ratón / pulsación de teclas. Es principalmente útil para investigadores de seguridad de juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / game:r6.
