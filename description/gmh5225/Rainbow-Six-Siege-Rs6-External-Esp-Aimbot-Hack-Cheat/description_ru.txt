Этот проект содержит исходный код утёкшего чита. В настоящее время чит не обнаруживается, но я не несу ответственности, если позже он будет обнаружен и вас забанят в игре. Scavs (белый), PMCs (белый, розовый)), ScavBoss (зелёный). Избегает использования обнаруживаемых векторов, хукая только WndProc, который вызывается при движении мыши / нажатии клавиш. В основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области cheat / game:r6.
