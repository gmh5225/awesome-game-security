Este proyecto es un complemento de IDA Pro para generar automáticamente firmas de patrones de bytes a partir del código seleccionado en el desensamblado. Crea patrones de firma al estilo IDA y al estilo código, con comodines para bytes variables, garantizando su unicidad dentro del binario. Las firmas generadas se pueden usar para el escaneo de patrones en el desarrollo de cheats o mods. Está orientado a ingenieros de ingeniería inversa que crean patrones de bytes fiables para identificar funciones entre distintas versiones del binario.
