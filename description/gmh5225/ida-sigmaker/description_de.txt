Dieses Projekt ist ein IDA-Pro-Plugin zur automatischen Erzeugung von Byte-Muster-Signaturen aus im Disassembly ausgewähltem Code. Es erstellt Signaturmuster im IDA- und Code-Stil mit Wildcards für variable Bytes und stellt die Eindeutigkeit innerhalb der Binärdatei sicher. Die erzeugten Signaturen können für Pattern-Scanning in der Cheat- oder Mod-Entwicklung verwendet werden. Es richtet sich an Reverse Engineers, die zuverlässige Byte-Muster zur Funktionsidentifikation über verschiedene Binärversionen hinweg erstellen.
