Questo progetto è un plugin per IDA Pro che genera automaticamente firme di pattern di byte dal codice selezionato nel disassemblaggio. Crea pattern di firma in stile IDA e in stile codice, con wildcard per i byte variabili, assicurandone l’unicità all’interno del binario. Le firme generate possono essere usate per lo scanning di pattern nello sviluppo di cheat o mod. È rivolto ai reverse engineer che creano pattern di byte affidabili per identificare funzioni tra diverse versioni del binario.
