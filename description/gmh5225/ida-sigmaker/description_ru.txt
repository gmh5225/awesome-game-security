Этот проект — плагин для IDA Pro, автоматически создающий сигнатуры байтовых шаблонов по выделенному коду в дизассемблере. Он формирует шаблоны сигнатур в стиле IDA и в стиле кода с подстановочными символами для изменяемых байтов и обеспечивает уникальность внутри бинарника. Полученные сигнатуры можно использовать для поиска по шаблону при разработке читов или модов. Предназначен для реверс-инженеров, которым нужны надёжные байтовые шаблоны для идентификации функций в разных версиях бинарников.
