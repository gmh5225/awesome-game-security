Ce projet est un plugin IDA Pro qui génère automatiquement des signatures de motifs d’octets à partir du code sélectionné dans le désassemblage. Il crée des motifs de signature au style IDA et au style code, avec des jokers pour les octets variables, en garantissant leur unicité dans le binaire. Les signatures générées peuvent servir au balayage de motifs dans le développement de cheats ou de mods. Il s’adresse aux ingénieurs en reverse engineering qui créent des motifs d’octets fiables pour identifier des fonctions d’une version de binaire à l’autre.
