Dieses Projekt stellt eine Liste von IDA-Plugins bereit.
Es enthält außerdem zusätzliche Metadaten wie die Sprache, in der das Plugin geschrieben ist, den Zeitpunkt der letzten Aktualisierung sowie einen Versuch, Plugins in Kategorien einzuordnen.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
