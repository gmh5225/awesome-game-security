Этот проект предоставляет список плагинов для IDA.
В нём также есть дополнительные метаданные: язык, на котором написан плагин, дата последнего обновления и попытка разнести плагины по категориям.
В первую очередь он полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / плагинов IDA.
