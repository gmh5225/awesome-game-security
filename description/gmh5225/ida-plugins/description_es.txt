Este proyecto ofrece una lista de plugins de IDA.
También incluye metadatos adicionales, como el lenguaje en el que está escrito el plugin, cuándo se actualizó por última vez y un intento de clasificar los plugins en categorías.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / plugins de IDA.
