本プロジェクトは Offline Crash Dump を実装するための Microsoft EDK2 パッケージであり、ファームウェアがオペレーティングシステムの前、または代わりにメモリダンプを書き込みます。
リポジトリには、ダンプ関連の GUID と構造体の共有ヘッダー、ダンプパーティションの検索と Windows 定義の UEFI 変数の読み取りを行う `OfflineDumpLib` ヘルパー、およびバッファリング・暗号化・リダクション対応でダンプを生成する `OfflineDumpWriterLib` のコードが含まれます。
また、`OfflineDumpWrite.efi` などの再配布可能なサンプル UEFI アプリケーション、ベンチマークおよびテストコンポーネント、UEFI DXE 環境で動作することを想定した完全なパッケージ構成も同梱されており、他のファームウェアスタックの参考にもなります。
アーカイブされた README では、このコードは製品展開ではなく bring-up、デバッグ、デバイス安定化向けであると明記されており、Windows 対応システムにおけるプリ OS クラッシュ収集の構造を学ぶファームウェア／プラットフォーム技術者に最も有用です。
