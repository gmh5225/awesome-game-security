Этот проект — пакет Microsoft EDK2 для поддержки Offline Crash Dump, при котором прошивка записывает дамп памяти до операционной системы или вместо неё.
Репозиторий включает общие заголовки для GUID и структур, связанных с дампами, вспомогательные средства `OfflineDumpLib` для поиска разделов дампа и чтения UEFI-переменных, определённых Windows, а также код `OfflineDumpWriterLib` для создания дампов с буферизацией, шифрованием и редакцией.
Также поставляются распространяемые и примерные UEFI-приложения, такие как `OfflineDumpWrite.efi`, компоненты бенчмарков и тестов, а также полная структура пакета, рассчитанная на среду UEFI DXE и при этом служащая справочником для других стеков прошивки.
Архивный README ясно указывает, что код предназначен для bring-up, отладки и стабилизации устройств, а не для розничных развёртываний, поэтому он наиболее полезен инженерам прошивки и платформы, изучающим устройство сбора сбоев до загрузки ОС в системах с поддержкой Windows.
