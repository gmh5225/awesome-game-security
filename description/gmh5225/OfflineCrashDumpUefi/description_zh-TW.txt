本專案是一個 Microsoft EDK2 套件，用於實作 Offline Crash Dump（離線當機傾印）支援，由韌體在作業系統之前或取代作業系統寫入記憶體傾印。
存放庫包含與傾印相關的 GUID 與結構的共用標頭、用於定位傾印分割區並讀取 Windows 定義的 UEFI 變數的 `OfflineDumpLib` 輔助函式庫，以及用於產生具緩衝、加密與遮罩支援之傾印的 `OfflineDumpWriterLib` 程式碼。
它還提供可再發行與範例 UEFI 應用程式（例如 `OfflineDumpWrite.efi`）、基準測試與測試元件，以及完整的套件配置，旨在於 UEFI DXE 環境中執行，同時也可作為其他韌體堆疊的參考。
歸檔的 README 清楚說明，此程式碼適用於 bring-up、除錯與裝置穩定化場景，而非零售部署，因此最有助於韌體與平台工程師研究 Windows 能力系統上作業系統前當機收集的結構。
