本项目是一个 Microsoft EDK2 软件包，用于实现 Offline Crash Dump（离线崩溃转储）支持，由固件在操作系统之前或代替操作系统写入内存转储。
仓库包含与转储相关的 GUID 和结构体的共享头文件、用于定位转储分区并读取 Windows 定义的 UEFI 变量的 `OfflineDumpLib` 辅助库，以及用于生成带缓冲、加密和脱敏支持的转储的 `OfflineDumpWriterLib` 代码。
它还提供可再分发与示例 UEFI 应用程序（如 `OfflineDumpWrite.efi`）、基准测试与测试组件，以及完整的软件包布局，旨在 UEFI DXE 环境中运行，同时也可作为其他固件栈的参考。
归档的 README 明确指出，该代码面向 bring-up、调试和设备稳定化场景，而非零售部署，因此最适合固件与平台工程师研究 Windows 能力系统上预操作系统崩溃收集的结构。
