该项目是一种概念验证级的内核通信方法，通过在 win32k 路径中交换内部数据指针实现，而非暴露常规设备接口。归档驱动会在 win32kbase.sys 中查找特征码，解析目标 qword 指针，附着到 explorer.exe，并使用 InterlockedExchangePointer 将该指针替换为自己的处理程序。用户态侧从 win32u.dll 解析 NtDCompositionSetChildRootVisual 以触发该路径，因此本仓库本质上是通过 GUI 系统调用进行指针交换通信的紧凑示例，而非完整框架。它主要面向研究隐蔽驱动通信、GUI 子系统内指针重定向，以及一次性钩子部署利弊的 Windows 内核研究者。
