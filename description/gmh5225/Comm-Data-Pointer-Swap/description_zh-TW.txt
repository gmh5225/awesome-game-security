此專案是一種概念驗證級的核心通訊方法，透過在 win32k 路徑中交換內部資料指標來實作，而非暴露一般裝置介面。封存的驅動會在 win32kbase.sys 中尋找特徵碼，解析目標 qword 指標，附著至 explorer.exe，並使用 InterlockedExchangePointer 將該指標替換為自己的處理常式。使用者模式端從 win32u.dll 解析 NtDCompositionSetChildRootVisual 以觸發該路徑，因此本儲存庫本質上是透過 GUI 系統呼叫進行指標交換通訊的精簡範例，而非完整框架。它主要適合研究隱蔽驅動通訊、GUI 子系統內指標重新導向，以及一次性掛鉤部署取捨的 Windows 核心研究者。
