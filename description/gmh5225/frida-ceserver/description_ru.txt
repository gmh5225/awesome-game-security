Этот проект — сервер Cheat Engine, реализованный через инструментацию Frida. Он использует Frida для чтения и записи памяти процессов на мобильных и настольных платформах и предоставляет сетевой протокол ceserver, чтобы клиент Cheat Engine на ПК мог подключаться и сканировать память удалённо. Такой подход работает на нерутованных Android-устройствах, если Frida может выполнить attach. Предназначен для исследователей безопасности мобильных игр, использующих Cheat Engine через кроссплатформенную инструментацию Frida.
