Este proyecto es un servidor de Cheat Engine implementado mediante instrumentación con Frida. Usa Frida para leer y escribir la memoria de procesos en plataformas móviles y de escritorio, exponiendo el protocolo de red ceserver para que un cliente de Cheat Engine de escritorio pueda conectarse y escanear memoria de forma remota. Este enfoque funciona en dispositivos Android no rooteados cuando Frida puede adjuntarse. Está orientado a investigadores de seguridad de juegos móviles que usan Cheat Engine a través de la instrumentación multiplataforma de Frida.
