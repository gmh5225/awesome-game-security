Этот проект — Leela Chess Zero (Lc0), шахматный движок на основе нейронной сети с глубоким обучением с подкреплением. Он использует поиск по дереву Монте-Карло, направляемый нейронной сетью, обученной через самоигру, по подходу AlphaZero от DeepMind. Движок на C++ поддерживает бэкенды CUDA, cuDNN и OpenCL для ускоренной на GPU оценки нейронной сети. Предназначен для разработчиков шахматных движков, исследователей ИИ и соревновательных шахматистов, интересующихся игровым ИИ на основе нейронных сетей.
