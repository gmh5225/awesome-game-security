WatchDogKiller は、WatchDog Anti-Malware の amsdk.sys または wamsdk.sys の BYOVD 欠陥を武器化し、保護されたセキュリティ製品を終了させるための PoC です。
このプログラムは \\.\amsdk またはガードデバイスの GUID を開き、必要に応じて先に IOCTL_REGISTER_PROCESS（0x80002010）で自身の PID を登録して認可をバイパスし、その後 PID と待機フラグを含む IOCTL_TERMINATE_PROCESS（0x80002048）要求を送信します。
README はこの手法を Silver Fox の戦術研究に関連付け、公開時点で検証した WatchDog ビルドが一般的な脆弱ドライバおよび HVCI ブロックリストにまだ載っていなかったと述べています。
実質的にこのリポジトリは、署名付きドライバ問題の曖昧な説明ではなく、EDR および AV を終了させることに特化した PoC です。
