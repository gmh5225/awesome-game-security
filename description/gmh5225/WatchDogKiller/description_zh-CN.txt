WatchDogKiller 是一个 PoC，用于将 WatchDog Anti-Malware 的 amsdk.sys 或 wamsdk.sys BYOVD 缺陷武器化，以终止受保护的安全产品。
该程序打开 \\.\amsdk 或防护设备 GUID，可选择先通过 IOCTL_REGISTER_PROCESS（0x80002010）注册自身 PID 以绕过授权，随后提交包含 PID 与等待标志的 IOCTL_TERMINATE_PROCESS（0x80002048）请求。
README 将该技术与 Silver Fox 战术研究相关联，并指出在发布时，所测试的 WatchDog 版本仍未出现在常见的易受攻击驱动与 HVCI 黑名单中。
实际上，该仓库是一个聚焦的 EDR 与 AV 终止 PoC，而非对又一个已签名驱动问题的泛泛描述。
