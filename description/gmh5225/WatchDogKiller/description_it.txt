WatchDogKiller è una PoC per weaponizzare il difetto BYOVD di amsdk.sys o wamsdk.sys di WatchDog Anti-Malware al fine di terminare prodotti di sicurezza protetti.
Il programma apre \\.\amsdk oppure il GUID del dispositivo di guardia, opzionalmente aggira l'autorizzazione registrando prima il proprio PID con IOCTL_REGISTER_PROCESS (0x80002010) e poi invia richieste IOCTL_TERMINATE_PROCESS (0x80002048) contenenti un PID e un flag di attesa.
Il README collega la tecnica alla ricerca sul tradecraft di Silver Fox e afferma che la build di WatchDog testata era ancora assente dalle comuni blocklist di driver vulnerabili e HVCI al momento della pubblicazione.
In pratica il repository è una PoC mirata per terminare EDR e AV, non una generica descrizione di un altro problema di driver firmato.
