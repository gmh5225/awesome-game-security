WatchDogKiller — это PoC для использования BYOVD-уязвимости amsdk.sys или wamsdk.sys из WatchDog Anti-Malware с целью завершения защищённых продуктов безопасности.
Программа открывает \\.\amsdk или GUID устройства-сторожа, при необходимости обходит авторизацию, сначала регистрируя собственный PID через IOCTL_REGISTER_PROCESS (0x80002010), а затем отправляет запросы IOCTL_TERMINATE_PROCESS (0x80002048) с PID и флагом ожидания.
README связывает эту технику с исследованиями tradecraft Silver Fox и указывает, что на момент публикации протестированная сборка WatchDog ещё отсутствовала в распространённых списках блокировки уязвимых драйверов и HVCI.
На практике репозиторий — это сфокусированный PoC для завершения EDR и AV, а не расплывчатое описание очередной проблемы подписанного драйвера.
