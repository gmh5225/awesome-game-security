Ein PCIe-DMA-Board-Design auf Basis des Xilinx Artix-7 75T FPGA für Speicherzugriffe auf Hardwareebene.
Stellt Firmware und HDL-Quellen für ein benutzerdefiniertes PCIe-Gerät bereit, das DMA-Lese- und -Schreibzugriffe auf den Systemspeicher des Hosts durchführen kann und auf Spielsicherheitsforschung sowie Tests ausgerichtet ist.
