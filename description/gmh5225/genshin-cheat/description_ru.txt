Этот проект — чит для Genshin Impact, предоставляющий модификации геймплея: режим бога, бесконечная выносливость, отсутствие перезарядок, телепортация и спидхаки. Он изменяет память игрового клиента на базе Unity или перехватывает методы IL2CPP, чтобы изменить поведение игры. Читу приходится противостоять античит-защите miHoYo. Он предназначен для исследователей игровой безопасности, изучающих клиентскую защиту Genshin Impact и механизмы обнаружения читов.
