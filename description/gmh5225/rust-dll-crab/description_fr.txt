Ce projet est un injecteur de DLL pour Windows écrit en Rust, qui fournit des capacités d'injection de DLL. Il implémente diverses méthodes d'injection et montre comment utiliser Rust pour des tâches de programmation système Windows habituellement réalisées en C/C++. Il s'adresse aux développeurs Rust et aux chercheurs en sécurité qui explorent Rust pour le développement d'outils de game hacking.
