Dieses Projekt ist ein auf Rust basierender DLL-Injector für Windows, der DLL-Injection-Funktionen in Rust bereitstellt. Es implementiert verschiedene Injectionsmethoden und zeigt, wie sich Windows-Systemprogrammierung, die üblicherweise in C/C++ erfolgt, mit Rust umsetzen lässt. Zielgruppe sind Rust-Entwickler und Sicherheitsforscher, die Rust für die Entwicklung von Game-Hacking-Tools erkunden.
