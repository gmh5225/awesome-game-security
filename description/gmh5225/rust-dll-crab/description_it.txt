Questo progetto è un injector di DLL per Windows basato su Rust che fornisce capacità di iniezione DLL scritte in Rust. Implementa vari metodi di iniezione e mostra come usare Rust per attività di programmazione di sistema Windows tipicamente svolte in C/C++. È rivolto a sviluppatori Rust e ricercatori di sicurezza che esplorano Rust per lo sviluppo di strumenti di game hacking.
