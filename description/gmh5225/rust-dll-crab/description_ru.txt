Этот проект — DLL-инжектор для Windows на Rust, предоставляющий возможности внедрения DLL, написанные на Rust. Он реализует различные методы инъекции и демонстрирует использование Rust для задач системного программирования Windows, которые обычно решают на C/C++. Ориентирован на разработчиков на Rust и исследователей безопасности, изучающих применение Rust в разработке инструментов для game hacking.
