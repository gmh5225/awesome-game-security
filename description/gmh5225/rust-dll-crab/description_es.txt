Este proyecto es un inyector de DLL para Windows basado en Rust que proporciona capacidades de inyección de DLL escritas en Rust. Implementa varios métodos de inyección y muestra cómo usar Rust para tareas de programación de sistemas Windows que normalmente se realizan en C/C++. Está dirigido a desarrolladores de Rust e investigadores de seguridad que exploran Rust para el desarrollo de herramientas de game hacking.
