Este proyecto es un SDK del motor Source de CS:GO que contiene definiciones de clases reverse-engineered, declaraciones de interfaces, estructuras netvar y firmas de funciones. Los encabezados C++ proporcionan el sistema de tipos fundamental para crear cheats de CS:GO, incluyendo clases de entidades, datos de armas, estructuras de información del jugador y punteros a interfaces del motor. Está dirigido a desarrolladores de cheats de CS:GO e investigadores del motor Source.
