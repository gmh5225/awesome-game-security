Этот проект — SDK движка Source для CS:GO, содержащий реконструированные определения классов, объявления интерфейсов, структуры netvar и сигнатуры функций. Заголовочные файлы C++ дают базовую систему типов для разработки читов CS:GO, включая классы сущностей, данные оружия, структуры информации об игроке и указатели на интерфейсы движка. Предназначен для разработчиков читов CS:GO и исследователей движка Source.
