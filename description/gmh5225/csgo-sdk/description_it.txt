Questo progetto è un SDK del motore Source di CS:GO che contiene definizioni di classi reverse-engineered, dichiarazioni di interfacce, strutture netvar e firme di funzioni. Gli header C++ forniscono il sistema di tipi di base per sviluppare cheat di CS:GO, incluse classi di entità, dati delle armi, strutture con informazioni sul giocatore e puntatori alle interfacce del motore. È rivolto agli sviluppatori di cheat CS:GO e ai ricercatori del motore Source.
