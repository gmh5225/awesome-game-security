Ce projet est un SDK du moteur Source pour CS:GO contenant des définitions de classes reverse-engineerées, des déclarations d'interfaces, des structures netvar et des signatures de fonctions. Les en-têtes C++ fournissent le système de types de base pour développer des cheats CS:GO, incluant les classes d'entités, les données d'armes, les structures d'informations joueur et les pointeurs d'interfaces du moteur. Il s'adresse aux développeurs de cheats CS:GO et aux chercheurs sur le moteur Source.
