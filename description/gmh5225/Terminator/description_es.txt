Este proyecto es Terminator, una herramienta de Windows que termina procesos protegidos (antivirus, EDR, anti-cheat) mediante un controlador firmado vulnerable (BYOVD). Aprovecha el IOCTL de terminación arbitraria de procesos del controlador antimalware de Zemana para matar procesos que no pueden terminarse con las API estándar debido a protecciones a nivel de kernel. Está dirigido a operadores de red team e investigadores de seguridad que estudian la terminación de productos de seguridad basada en BYOVD.
