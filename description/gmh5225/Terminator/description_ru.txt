Этот проект — Terminator, инструмент для Windows, который завершает защищённые процессы (антивирус, EDR, античит) с помощью уязвимого подписанного драйвера (BYOVD). Он использует IOCTL произвольного завершения процессов драйвера Zemana anti-malware, чтобы убивать процессы, которые нельзя завершить стандартными API из‑за защиты на уровне ядра. Предназначен для red team‑операторов и исследователей безопасности, изучающих завершение продуктов безопасности на базе BYOVD.
