Questo progetto è Terminator, uno strumento Windows che termina processi protetti (antivirus, EDR, anti-cheat) usando un driver firmato vulnerabile (BYOVD). Sfrutta l'IOCTL di terminazione arbitraria dei processi del driver antimalware Zemana per chiudere processi che non possono essere terminati tramite le API standard a causa di protezioni a livello kernel. È rivolto a operatori red team e ricercatori di sicurezza che studiano la terminazione di prodotti di sicurezza basata su BYOVD.
