Этот проект посвящён инструменту.
ARK — это фреймворк глубокого обучения, специально разработанный для высокооптимизированной производительности на распределённых GPU.
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования ядра Windows.
