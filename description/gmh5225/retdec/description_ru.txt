Этот проект — RetDec; из‑за нехватки ресурсов он сейчас находится в режиме ограниченной поддержки: Pull Request приветствуются.
Они рассматриваются в приоритетном порядке и по возможности без задержек.
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс‑инжинирингу, изучающим атакующие техники в области читов / декомпиляторов.
