Этот проект — руководство по реализации античит-защиты в alt:V Multiplayer, фреймворке модификации мультиплеера для GTA V. В нём описаны методы обнаружения читов, проверки целостности клиента, мониторинга поведения игроков и защиты серверной игровой логики на серверах alt:V. Материал предназначен для разработчиков серверов alt:V, внедряющих античит-меры для своих мультиплеерных сообществ GTA V.
