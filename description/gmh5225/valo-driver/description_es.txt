Este proyecto es un controlador de kernel de Windows diseñado para leer la memoria del proceso del juego Valorant eludiendo las protecciones a nivel de kernel del anti-cheat Vanguard. Implementa la lectura de memoria mediante traducción de direcciones físicas, manipulación de CR3 o mapeo MDL para acceder al proceso del juego sin usar las API estándar que Vanguard monitoriza. El controlador en C demuestra técnicas de bypass anti-cheat a nivel de kernel específicas de Vanguard. Está dirigido a investigadores de anti-cheat que estudian la protección de kernel de Vanguard y sus posibles vectores de bypass.
