Этот проект — драйвер ядра Windows, предназначенный для чтения памяти процесса игры Valorant с обходом ядерных защит античита Vanguard. Чтение памяти реализовано через трансляцию физических адресов, манипуляцию CR3 или отображение MDL, что позволяет обращаться к процессу игры без стандартных API, которые мониторит Vanguard. Драйвер на C демонстрирует техники обхода античита на уровне ядра, специфичные для Vanguard. Он ориентирован на исследователей античита, изучающих ядерную защиту Vanguard и возможные векторы её обхода.
