Questo progetto è un driver kernel Windows progettato per leggere la memoria del processo di gioco di Valorant aggirando le protezioni a livello kernel dell'anti-cheat Vanguard. Implementa la lettura della memoria tramite traduzione di indirizzi fisici, manipolazione di CR3 o mapping MDL per accedere al processo di gioco senza usare le API standard monitorate da Vanguard. Il driver in C dimostra tecniche di bypass anti-cheat a livello kernel specifiche per Vanguard. È rivolto ai ricercatori di anti-cheat che studiano la protezione kernel di Vanguard e i suoi possibili vettori di bypass.
