Questo progetto è una raccolta o uno mirror di file di simboli PDB (Program Database) di Unreal Engine 5. Questi file di simboli di debug contengono nomi di funzioni, definizioni di tipi, layout delle strutture e informazioni sulle variabili dei binari UE5, consentendo reverse engineering e debug dettagliati dei giochi UE5 con informazioni complete sui simboli in strumenti come IDA Pro o x64dbg. È rivolto a reverse engineer di giochi UE5 e sviluppatori di cheat che necessitano di dati sui simboli per analizzare i binari UE5.
