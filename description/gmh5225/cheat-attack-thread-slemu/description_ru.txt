Этот проект демонстрирует потоковую технику атаки для выполнения читов, использующую эмуляцию sleep или манипуляцию состоянием потока для обхода сканирования потоков античитами. Он изменяет контекст потока или состояние планировщика, чтобы скрыть чит-потоки от перечисления системами античита, которые сканируют подозрительные системные потоки. Предназначен для исследователей ядра, изучающих скрытое исполнение потоков и обход обнаружения потоков античитами.
