Ein Proof-of-Concept, der die Kernel Security Support Provider Interface (KSecDD.sys) für die Ausführung beliebigen Kernelcodes ausnutzt.
Injiziert eine DLL in LSASS, um IOCTL_KSEC_IPC_SET_FUNCTION_RETURN zu nutzen, wodurch beliebige Kerneladressen ausgeführt werden können, und deaktiviert anschließend DSE durch Überschreiben von ci.dll!g_CiOptions.
