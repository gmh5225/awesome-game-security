Una proof-of-concept che sfrutta la Kernel Security Support Provider Interface (KSecDD.sys) per l'esecuzione arbitraria di codice nel kernel.
Inietta una DLL in LSASS per usare IOCTL_KSEC_IPC_SET_FUNCTION_RETURN, che consente di eseguire indirizzi arbitrari del kernel, quindi disabilita DSE sovrascrivendo ci.dll!g_CiOptions.
