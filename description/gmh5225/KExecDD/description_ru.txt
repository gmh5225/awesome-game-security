Концептуальное доказательство (proof-of-concept), эксплуатирующее Kernel Security Support Provider Interface (KSecDD.sys) для произвольного выполнения кода в ядре.
Внедряет DLL в LSASS для использования IOCTL_KSEC_IPC_SET_FUNCTION_RETURN, что позволяет выполнять произвольные адреса ядра, затем отключает DSE путём перезаписи ci.dll!g_CiOptions.
