Une preuve de concept exploitant le Kernel Security Support Provider Interface (KSecDD.sys) pour l'exécution arbitraire de code noyau.
Injecte une DLL dans LSASS afin d'utiliser IOCTL_KSEC_IPC_SET_FUNCTION_RETURN, qui permet d'exécuter des adresses noyau arbitraires, puis désactive DSE en écrasant ci.dll!g_CiOptions.
