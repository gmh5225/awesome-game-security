Una prueba de concepto que explota la Kernel Security Support Provider Interface (KSecDD.sys) para la ejecución arbitraria de código en el kernel.
Inyecta una DLL en LSASS para usar IOCTL_KSEC_IPC_SET_FUNCTION_RETURN, lo que permite ejecutar direcciones arbitrarias del kernel, y luego desactiva DSE sobrescribiendo ci.dll!g_CiOptions.
