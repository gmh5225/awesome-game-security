이 프로젝트는 일반적인 장치 인터페이스를 노출하는 대신 win32kbase에서 데이터 포인터 스왑으로 통신하는 비교적 오래된 치트 드라이버 예제입니다.
드라이버는 win32kbase에서 대상 포인터를 스캔하고, explorer.exe에 연결한 뒤 InterlockedExchangePointer로 사용자 정의 NtSetCompositionSurfaceAnalogExclusive 핸들러와 교체하며, 사용자 모드에서 보낸 구조를 통해 요청을 라우팅합니다.
통신 프리미티브 외에도 MmUnloadedDrivers 및 관련 로더 잔여물 같은 흔적을 정리하는 유틸리티가 포함되어 있어, 이 저장소는 통신 데모이자 안티포렌식 보조 도구의 성격을 함께 갖습니다.
데이터 포인터 스왑 통신, win32k 기반 훅 배치, 수동 매핑 치트 드라이버와 함께 쓰이는 정리 로직을 연구하는 Windows 커널 연구자에게 주로 유용합니다.
