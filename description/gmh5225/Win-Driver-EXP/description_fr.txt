Ce projet est une collection d'exploits de pilotes Windows ciblant divers pilotes signés vulnérables pour obtenir un accès au niveau noyau. Il fournit du code d'exploit pour plusieurs pilotes vulnérables, démontrant la lecture/écriture arbitraire du noyau, l'exécution de code et la manipulation de processus via leurs interfaces IOCTL non sécurisées. Il s'adresse aux chercheurs BYOVD et aux opérateurs red team qui étudient le paysage des pilotes noyau Windows exploitables.
