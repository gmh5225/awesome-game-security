Questo progetto è una raccolta di exploit per driver Windows che prendono di mira vari driver firmati vulnerabili per ottenere accesso a livello kernel. Fornisce codice di exploit per più driver vulnerabili, dimostrando lettura/scrittura arbitraria del kernel, esecuzione di codice e manipolazione dei processi tramite le loro interfacce IOCTL non sicure. È rivolto a ricercatori BYOVD e operatori red team che studiano il panorama dei driver kernel Windows sfruttabili.
