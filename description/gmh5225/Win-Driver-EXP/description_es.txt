Este proyecto es una colección de exploits de controladores de Windows que apuntan a varios controladores firmados vulnerables para obtener acceso a nivel de kernel. Proporciona código de exploit para múltiples controladores vulnerables, demostrando lectura/escritura arbitraria del kernel, ejecución de código y manipulación de procesos a través de sus interfaces IOCTL inseguras. Está dirigido a investigadores BYOVD y operadores de red team que estudian el panorama de controladores de kernel de Windows explotables.
