Этот проект — коллекция эксплойтов для драйверов Windows, нацеленных на различные уязвимые подписанные драйверы для получения доступа на уровне ядра. Он предоставляет код эксплойтов для нескольких уязвимых драйверов, демонстрируя произвольное чтение/запись ядра, выполнение кода и манипуляцию процессами через их небезопасные интерфейсы IOCTL. Предназначен для исследователей BYOVD и операторов red team, изучающих ландшафт эксплуатируемых драйверов ядра Windows.
