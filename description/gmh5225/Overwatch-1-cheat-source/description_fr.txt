Une DLL de cheat interne pour Overwatch 1 qui accroche Present de DirectX 11 via Microsoft Detours afin d’afficher une superposition ImGui avec ESP, aimbot, changeur de skins (sélection cosmétique par héros) et un système de menu complet, protégée par le SDK VMProtect.
Le cheat utilise le spoofing d’adresse de retour (Spoofcall.masm) pour contourner les contrôles de pile de Warden, implémente un accrochage basé sur les points d’arrêt physiques (BreakPoint.h), lit les entités du jeu via des offsets SDK (SKD.hpp), simule les entrées souris avec SendInput, et inclut une authentification basée sur curl pour la validation du chargeur/licence.
Il est surtout utile aux chercheurs en sécurité des jeux qui étudient les cheats internes basés sur des hooks DX11 avec intégration VMProtect, spoofing d’adresse de retour et contournement de l’anti-cheat Warden dans Overwatch.
