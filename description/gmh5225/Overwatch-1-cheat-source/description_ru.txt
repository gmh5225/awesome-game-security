Внутренняя DLL-чит для Overwatch 1, которая перехватывает DirectX 11 Present через Microsoft Detours для отрисовки оверлея ImGui с ESP, аимботом, сменой скинов (с выбором косметики для каждого героя) и полноценной системой меню, защищённая SDK VMProtect.
Чит использует подмену адреса возврата (Spoofcall.masm) для обхода проверок стека Warden, реализует хукинг на аппаратных точках останова (BreakPoint.h), читает игровые сущности через смещения SDK (SKD.hpp), имитирует ввод мыши через SendInput и включает аутентификацию на базе curl для проверки загрузчика/лицензии.
В основном полезен исследователям безопасности игр, изучающим внутренние читы на хуках DX11 с интеграцией VMProtect, подменой адреса возврата и обходом античита Warden в Overwatch.
