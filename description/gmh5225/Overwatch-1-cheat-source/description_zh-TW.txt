一個內部的《鬥陣特攻 1》作弊 DLL，透過 Microsoft Detours 掛鉤 DirectX 11 的 Present，以渲染具備 ESP、自動瞄準、皮膚修改器（支援依英雄選擇外觀）以及完整選單系統的 ImGui 疊加層，並由 VMProtect SDK 保護。
該作弊程式使用返回位址偽造（Spoofcall.masm）規避 Warden 堆疊檢查，實作基於硬體中斷點的掛鉤（BreakPoint.h），透過 SDK 偏移（SKD.hpp）讀取遊戲實體，藉由 SendInput 模擬滑鼠輸入，並包含基於 curl 的身分驗證以進行載入器／授權驗證。
它主要適用於研究具備 VMProtect 整合、返回位址偽造以及《鬥陣特攻》中 Warden 反作弊規避的內部 DX11 掛鉤類作弊的遊戲安全研究人員。
