Microsoft Detours で DirectX 11 の Present をフックし、ESP、エイムボット、スキンチェンジャー（ヒーローごとのコスメティック選択付き）、および完全なメニューシステムを備えた ImGui オーバーレイを描画する内部向け Overwatch 1 チート DLL で、VMProtect SDK により保護されています。
このチートは戻りアドレス偽装（Spoofcall.masm）で Warden のスタック検査を回避し、ハードウェアブレークポイントベースのフッキング（BreakPoint.h）を実装し、SDK オフセット（SKD.hpp）経由でゲームエンティティを読み取り、SendInput でマウス入力をシミュレートし、ローダー／ライセンス検証用の curl ベース認証を含みます。
主に、VMProtect 連携、戻りアドレス偽装、Overwatch における Warden アンチチート回避を伴う内部 DX11 フック型チートを研究するゲームセキュリティ研究者向けです。
