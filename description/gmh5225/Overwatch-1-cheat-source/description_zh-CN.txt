一个内部的《守望先锋 1》作弊 DLL，通过 Microsoft Detours 挂钩 DirectX 11 的 Present，以渲染带有 ESP、自瞄、皮肤修改器（支持按英雄选择外观）以及完整菜单系统的 ImGui 叠加层，并由 VMProtect SDK 保护。
该作弊程序使用返回地址伪造（Spoofcall.masm）规避 Warden 栈检查，实现基于硬件断点的挂钩（BreakPoint.h），通过 SDK 偏移（SKD.hpp）读取游戏实体，借助 SendInput 模拟鼠标输入，并包含基于 curl 的身份验证以进行加载器/许可证校验。
它主要适用于研究带有 VMProtect 集成、返回地址伪造以及《守望先锋》中 Warden 反作弊规避的内部 DX11 挂钩类作弊的游戏安全研究人员。
