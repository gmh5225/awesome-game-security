Microsoft Detours로 DirectX 11 Present를 후킹하여 ESP, 에임봇, 스킨 체인저(영웅별 코스메틱 선택), 전체 메뉴 시스템이 포함된 ImGui 오버레이를 렌더링하는 내부용 Overwatch 1 치트 DLL이며, VMProtect SDK로 보호됩니다.
이 치트는 반환 주소 스푸핑(Spoofcall.masm)으로 Warden 스택 검사를 회피하고, 하드웨어 브레이크포인트 기반 후킹(BreakPoint.h)을 구현하며, SDK 오프셋(SKD.hpp)으로 게임 엔티티를 읽고, SendInput으로 마우스 입력을 시뮬레이션하며, 로더/라이선스 검증용 curl 기반 인증을 포함합니다.
주로 VMProtect 통합, 반환 주소 스푸핑, Overwatch의 Warden 안티치트 우회를 갖춘 내부 DX11 후킹 기반 치트를 연구하는 게임 보안 연구자에게 유용합니다.
