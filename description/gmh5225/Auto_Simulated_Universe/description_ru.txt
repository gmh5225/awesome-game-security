Этот проект — бот автоматизации для режима «Симулированная вселенная» в Honkai: Star Rail. Он автоматизирует повторяющийся игровой цикл: прохождение подземелья Симулированной вселенной, выбор благословений, бои и сбор наград с помощью распознавания экрана, автоматизированного ввода и логики принятия решений. Инструмент обеспечивает поиск пути, боевые ротации и стратегии выбора благословений. Предназначен для игроков Honkai: Star Rail, автоматизирующих фарм, и исследователей игровой автоматизации.
