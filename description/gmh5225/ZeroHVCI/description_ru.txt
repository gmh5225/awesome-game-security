Этот проект — proof of concept для обхода Windows HVCI (Hypervisor-Protected Code Integrity) с целью выполнения неподписанного кода ядра. HVCI обычно препятствует загрузке неподписанных драйверов, используя Hyper-V для принудительной проверки целостности кода на уровне гипервизора. Этот инструмент демонстрирует методы обхода защиты HVCI, например эксплуатацию крайних случаев в политике HVCI или использование уязвимых подписанных драйверов. Он предназначен для исследователей безопасности ядра, изучающих методы обхода HVCI и ограничения виртуализационной безопасности Windows.
