Este proyecto es una prueba de concepto para eludir Windows HVCI (Hypervisor-Protected Code Integrity) y ejecutar código de kernel no firmado. HVCI normalmente impide cargar controladores no firmados usando Hyper-V para aplicar la integridad de código a nivel de hipervisor. Esta herramienta demuestra técnicas para eludir la protección HVCI, como explotar casos límite en la política de HVCI o aprovechar controladores firmados vulnerables. Está dirigida a investigadores de seguridad del kernel que estudian métodos de bypass de HVCI y las limitaciones de la seguridad basada en virtualización de Windows.
