이 프로젝트는 Windows HVCI(Hypervisor-Protected Code Integrity)를 우회하여 서명되지 않은 커널 코드를 실행하기 위한 개념 증명(PoC)입니다. HVCI는 일반적으로 Hyper-V를 사용해 하이퍼바이저 수준에서 코드 무결성을 강제함으로써 서명되지 않은 드라이버 로드를 막습니다. 이 도구는 HVCI 정책의 엣지 케이스를 악용하거나 취약한 서명된 드라이버를 활용하는 등 HVCI 보호를 우회하는 기법을 보여줍니다. HVCI 우회 방법과 Windows 가상화 기반 보안의 한계를 연구하는 커널 보안 연구자를 대상으로 합니다.
