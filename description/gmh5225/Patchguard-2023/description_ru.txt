Этот проект представляет собой исследование Windows PatchGuard (Kernel Patch Protection) по состоянию на 2023 год: в нём описываются внутренняя работа, механизмы срабатывания и потенциальные методы обхода. Анализируются основанная на таймерах проверка PatchGuard, шифрование контекста, список защищаемых структур и процедуры восстановления. Материал ориентирован на исследователей ядра Windows, изучающих эволюцию механизмов защиты PatchGuard и способы их обхода.
