Dieses Projekt stellt Code für einen einfachen Cheat für das Spiel Honkai: Star Rail bereit. Es ist hauptsächlich in C/C++ geschrieben und konzentriert sich auf Hooking und Overlays. Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Honkai: Star Rail untersuchen.
