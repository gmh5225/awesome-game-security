Questo progetto è un framework di cheat interno in C++ mirato al gioco Rust, costruito intorno a un core Horizon con rendering DirectX 11 e overlay ImGui.
Include helper per memoria e mappe, supporto al disassemblaggio HDE64, utilità per stringhe sicure e backend ImGui Win32 o DX11 per UI in-process e hook grafici.
È principalmente utile a ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'area cheat / game:rust.
