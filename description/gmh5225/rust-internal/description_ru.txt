Этот проект — внутренний чит-фреймворк на C++ для игры Rust, построенный вокруг ядра Horizon с рендерингом DirectX 11 и оверлеями ImGui.
Он включает помощники для работы с памятью и картой, поддержку дизассемблирования HDE64, утилиты для безопасных строк, а также бэкенды ImGui Win32 или DX11 для внутрипроцессного UI и графических хуков.
В основном он полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области cheat / game:rust.
