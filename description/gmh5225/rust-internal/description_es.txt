Este proyecto es un framework de cheat interno en C++ orientado al juego Rust, construido en torno a un núcleo Horizon con renderizado DirectX 11 y overlays ImGui.
Incluye ayudantes de memoria y mapas, soporte de desensamblado HDE64, utilidades de cadenas seguras y backends ImGui Win32 o DX11 para UI in-process y hooks gráficos.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área cheat / game:rust.
