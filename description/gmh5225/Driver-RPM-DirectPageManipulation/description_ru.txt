Этот проект — минимальный пример для ядра Windows, показывающий, как читать и записывать память другого процесса путём прямой манипуляции структурами пейджинга.
README представляет код как демонстрацию «по минимуму», и реализация именно такова: выделяется непрерывная страница, находится собственный PTE, перезаписывается номер кадра страницы для переназначения произвольных физических страниц, а затем используется ручной перевод виртуальных адресов в физические для реализации процедур копирования памяти процесса.
Точка входа примера драйвера применяет эту технику к целевому процессу, чтобы читать данные модуля без стандартных документированных вспомогательных функций копирования, явно опуская обработку выгруженной памяти и вопросы многопоточности, чтобы сохранить логику простой.
Проект в первую очередь полезен исследователям ядра Windows, которым нужна компактная справка по перезаписи PTE, ручному преобразованию адресов и доступу к памяти процесса через таблицы страниц.
