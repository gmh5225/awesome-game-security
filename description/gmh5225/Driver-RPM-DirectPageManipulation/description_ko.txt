이 프로젝트는 페이징 구조를 직접 조작하여 다른 프로세스의 메모리를 읽고 쓰는 방법을 보여주는 최소한의 Windows 커널 예제입니다.
README는 코드를 최소한의 데모로 설명하며, 구현 역시 연속 페이지를 할당하고 자신의 PTE를 찾은 뒤 페이지 프레임 번호를 덮어써 임의의 물리 페이지를 다시 매핑하고, 수동 가상-물리 주소 변환을 사용해 프로세스 메모리 복사 루틴을 구현합니다.
샘플 드라이버 엔트리는 표준 문서화된 복사 헬퍼에 의존하지 않고 이 기법으로 대상 프로세스의 모듈 데이터를 읽으며, 로직을 단순하게 유지하기 위해 페이지 아웃된 메모리 처리와 멀티스레딩 문제는 의도적으로 제외합니다.
PTE 재작성, 수동 주소 변환, 페이지 테이블 기반 프로세스 메모리 접근에 대한 간결한 참고 자료를 원하는 Windows 커널 연구자에게 주로 유용합니다.
