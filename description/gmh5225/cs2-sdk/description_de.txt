Dieses Projekt konzentriert sich auf ein SDK.
Dieses SDK wurde so vereinfacht wie möglich geschrieben und so nah wie möglich am Source-2-Code gehalten; es unterstützt sowohl die DirectX11- als auch die Vulkan-Grafik-APIs.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / game:cs2 untersuchen.
