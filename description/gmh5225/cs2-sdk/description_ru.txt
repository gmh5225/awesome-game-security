Этот проект посвящён SDK.
Этот SDK написан максимально упрощённо и максимально близок к коду Source 2, с поддержкой графических API DirectX11 и Vulkan.
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / game:cs2.
