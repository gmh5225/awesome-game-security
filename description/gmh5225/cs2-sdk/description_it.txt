Questo progetto si concentra su un SDK.
Questo SDK è stato scritto nel modo più semplificato possibile e mantenuto il più vicino possibile al codice di Source 2, supportando sia le API grafiche DirectX11 sia Vulkan.
È soprattutto utile per i ricercatori di sicurezza nei videogiochi e per i reverse engineer che studiano tecniche offensive nell’ambito dei cheat / game:cs2.
