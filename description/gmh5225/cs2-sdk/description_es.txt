Este proyecto se centra en un SDK.
Este SDK se ha escrito de la forma más simplificada posible y se mantiene lo más cerca posible del código de Source 2, con soporte para las API gráficas DirectX11 y Vulkan.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de los cheats / game:cs2.
