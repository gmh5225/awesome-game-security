Ce projet se concentre sur un SDK.
Ce SDK a été écrit de manière aussi simplifiée que possible et reste aussi proche que possible du code Source 2, tout en prenant en charge les API graphiques DirectX11 et Vulkan.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats / game:cs2.
