Этот проект — загрузчик (mapper) драйверов, который использует уязвимый драйвер ядра Lenovo для загрузки неподписанных драйверов в память ядра. Он эксплуатирует IOCTL доступа к памяти драйвера Lenovo в конвейере ручного отображения (manual mapping). Предназначен для исследователей ядра, изучающих BYOVD-загрузку драйверов через эксплуатацию драйвера Lenovo.
