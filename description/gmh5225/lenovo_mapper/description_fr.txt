Ce projet est un mappeur de pilotes qui utilise un pilote noyau vulnérable de Lenovo pour charger des pilotes non signés en mémoire noyau. Il exploite les IOCTL d'accès mémoire du pilote Lenovo pour le pipeline de mapping manuel. Il s'adresse aux chercheurs noyau qui étudient le mapping de pilotes basé sur BYOVD via l'exploitation d'un pilote Lenovo.
