Este proyecto es un mapeador de controladores que usa un controlador de kernel vulnerable de Lenovo para cargar controladores sin firmar en la memoria del kernel. Aprovecha los IOCTL de acceso a memoria del controlador Lenovo para el pipeline de mapeo manual. Está dirigido a investigadores de kernel que estudian el mapeo de controladores basado en BYOVD mediante la explotación de controladores Lenovo.
