Este proyecto se centra en la memoria compartida (shared Memory).
Los errores se corregirán en cuanto tenga tiempo disponible si se encuentra alguno; todo el código del repositorio se escribió en aproximadamente una mañana y una tarde, por lo que puede haber algunos descuidos.
Es principalmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de comunicación entre cheats y drivers.
