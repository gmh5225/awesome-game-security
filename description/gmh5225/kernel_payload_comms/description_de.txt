Fehler werden behoben, sobald ich Zeit habe, falls welche gefunden werden; der gesamte Code im Repository wurde etwa an einem Vormittag und Nachmittag geschrieben, daher können einige Ungenauigkeiten vorhanden sein.
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich der Cheat-/Treiber-Kommunikation untersuchen.
