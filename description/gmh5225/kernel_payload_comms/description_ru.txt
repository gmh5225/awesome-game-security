Этот проект посвящён shared Memory (разделяемой памяти).
Ошибки будут исправлены, как только появится время, если они будут найдены; весь код в репозитории был написан примерно за утро и день, поэтому возможны некоторые недочёты.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области коммуникации между читами и драйверами.
