Ce projet est Bytecode Viewer, une suite de reverse engineering pour Java/Android APK qui intègre plusieurs décompileurs (CFR, Procyon, FernFlower, JD-GUI, Krakatau) dans une interface à onglets unique. Il permet d'afficher simultanément le bytecode Java, le code source décompilé et le Smali pour les fichiers JAR, class, DEX et APK. L'outil inclut la recherche, l'extraction de chaînes et la prise en charge des plugins. Il s'adresse aux reverse engineers Java et Android qui souhaitent comparer les sorties des décompileurs et analyser le bytecode dans un environnement unifié.
