Этот проект — Bytecode Viewer, набор средств для реверс-инжиниринга Java/Android APK, объединяющий несколько декомпиляторов (CFR, Procyon, FernFlower, JD-GUI, Krakatau) в едином интерфейсе с вкладками. Он позволяет одновременно просматривать Java-байткод, декомпилированный исходный код и Smali для файлов JAR, class, DEX и APK. Инструмент включает поиск, извлечение строк и поддержку плагинов. Он предназначен для специалистов по реверс-инжинирингу Java и Android, которым нужно сравнивать результаты декомпиляторов и анализировать байткод в единой среде.
