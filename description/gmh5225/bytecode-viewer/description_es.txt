Este proyecto es Bytecode Viewer, un conjunto de ingeniería inversa para Java/Android APK que integra varios descompiladores (CFR, Procyon, FernFlower, JD-GUI, Krakatau) en una única interfaz con pestañas. Permite ver simultáneamente el bytecode de Java, el código fuente descompilado y Smali para archivos JAR, class, DEX y APK. La herramienta incluye búsqueda, extracción de cadenas y soporte de plugins. Está dirigida a ingenieros inversos de Java y Android que desean comparar las salidas de los descompiladores y analizar el bytecode en un entorno unificado.
