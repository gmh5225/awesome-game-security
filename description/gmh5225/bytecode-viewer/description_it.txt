Questo progetto è Bytecode Viewer, una suite di reverse engineering per Java/Android APK che integra più decompilatori (CFR, Procyon, FernFlower, JD-GUI, Krakatau) in un'unica interfaccia a schede. Supporta la visualizzazione simultanea di bytecode Java, codice sorgente decompilato e Smali per file JAR, class, DEX e APK. Lo strumento include ricerca, estrazione di stringhe e supporto per plugin. È rivolto ai reverse engineer Java e Android che vogliono confrontare gli output dei decompilatori e analizzare il bytecode in un ambiente unificato.
