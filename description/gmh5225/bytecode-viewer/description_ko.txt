이 프로젝트는 Bytecode Viewer로, 여러 디컴파일러(CFR, Procyon, FernFlower, JD-GUI, Krakatau)를 하나의 탭 인터페이스에 통합한 Java/Android APK 리버스 엔지니어링 스위트입니다. JAR, class, DEX, APK 파일에 대해 Java 바이트코드, 디컴파일된 소스, Smali를 동시에 볼 수 있습니다. 검색, 문자열 추출, 플러그인 지원 기능을 포함합니다. 디컴파일러 출력을 비교하고 통합 환경에서 바이트코드를 분석하려는 Java 및 Android 리버스 엔지니어를 대상으로 합니다.
