Этот проект — cheat engine или фреймворк, специально предназначенный для игр на Unreal Engine 4. Он предоставляет UE4-ориентированное сканирование памяти, перечисление GObject, обход UProperty и генерацию SDK, оптимизированные для поиска и манипуляции игровыми объектами в UE4-проектах. Рассчитан на игровых хакеров и исследователей безопасности, которым нужны инструменты анализа памяти, специфичные для UE4.
