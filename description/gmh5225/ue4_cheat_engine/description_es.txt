Este proyecto es un cheat engine o framework diseñado específicamente para juegos de Unreal Engine 4. Ofrece escaneo de memoria consciente de UE4, enumeración de GObject, recorrido de UProperty y generación de SDK, optimizados para encontrar y manipular objetos de juego en títulos UE4. Está dirigido a game hackers e investigadores de seguridad que necesitan herramientas de análisis de memoria específicas de UE4.
