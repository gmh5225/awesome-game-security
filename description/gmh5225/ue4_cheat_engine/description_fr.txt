Ce projet est un cheat engine ou framework conçu spécifiquement pour les jeux Unreal Engine 4. Il offre un scan mémoire conscient d’UE4, l’énumération des GObject, le parcours des UProperty et la génération de SDK, optimisés pour trouver et manipuler les objets de jeu dans les titres UE4. Il s’adresse aux game hackers et chercheurs en sécurité qui ont besoin d’outils d’analyse mémoire spécifiques à UE4.
