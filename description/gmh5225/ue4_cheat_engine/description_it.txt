Questo progetto è un cheat engine o framework progettato specificamente per i giochi Unreal Engine 4. Offre scansione della memoria consapevole di UE4, enumerazione dei GObject, attraversamento delle UProperty e generazione di SDK, ottimizzati per trovare e manipolare gli oggetti di gioco nei titoli UE4. È rivolto a game hacker e ricercatori di sicurezza che necessitano di strumenti di analisi della memoria specifici per UE4.
