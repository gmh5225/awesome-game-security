Dieses Projekt erreicht sein Ziel, indem es verschiedene Funktionen von steamserver.dll und der WinAPI hookt, sodass das Anticheat einen Fehler im Rückgabewert annimmt. Die wichtige Funktion vac_monitor_manager ist besonders kritisch und kann unerwünschte Nebenwirkungen haben, indem sie den Trust Factor senkt, weil sie VAC vollständig daran hindert, das Spiel zu überwachen. Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Erkundung von Anticheat-Systemen (VAC) untersuchen.
