本项目是一个基于 Intel 硬件性能特性（尤其是 Last Branch Records 与 PEBS 采样）的 Windows 驱动框架，面向以循环为中心的性能剖析。其归档 README 将 LoopHPCs 描述为用于在运行中的二进制中发现热点循环（尤其是面向脱壳的恶意软件）的剖析框架；源码树在类过滤驱动结构中整合了 PEBS 辅助、分支记录处理与循环跟踪逻辑。核心代码消费 PEBS 记录，将 eventing IP、next IP 与数据地址与基于 LBR 的控制流上下文关联，从而构建面向循环的遥测，而非仅采集原始计数器值。它主要适用于研究硬件辅助运行时剖析紧循环与脱壳行为的底层 Windows 与逆向工程研究者。
