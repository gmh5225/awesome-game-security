Este proyecto es un framework de controlador de Windows para perfilado centrado en bucles basado en funciones de rendimiento de hardware Intel, en concreto Last Branch Records y muestreo PEBS. Su README archivado describe LoopHPCs como un framework de perfilado para encontrar bucles calientes en binarios en ejecución, especialmente malware orientado al desempaquetado, y el árbol de código combina helpers de PEBS, procesamiento de registros de ramas y lógica de seguimiento de bucles dentro de un proyecto al estilo de filtro-controlador. El código central consume registros PEBS, correlaciona eventing IPs, next IPs y direcciones de datos con el contexto de flujo de control derivado de LBR, y luego construye telemetría orientada a bucles en lugar de limitarse a valores brutos de contadores. Resulta especialmente útil para investigadores de Windows de bajo nivel y de ingeniería inversa que estudian el perfilado en tiempo de ejecución asistido por hardware de bucles ajustados y del comportamiento de desempaquetado.
