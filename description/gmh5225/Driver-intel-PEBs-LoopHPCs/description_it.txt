Questo progetto è un framework di driver Windows per il profiling incentrato sui loop basato sulle funzionalità di performance hardware Intel, in particolare Last Branch Records e campionamento PEBS. Il README archiviato descrive LoopHPCs come un framework di profiling per individuare hot loop nei binari in esecuzione, soprattutto malware orientato all'unpacking, e l'albero dei sorgenti combina helper PEBS, elaborazione dei branch record e logica di tracciamento dei loop in un progetto in stile filter-driver. Il codice centrale consuma i record PEBS, correla eventing IP, next IP e indirizzi dati con il contesto di control-flow derivato dagli LBR, quindi costruisce telemetria orientata ai loop invece di limitarsi ai valori grezzi dei contatori. È soprattutto utile per ricercatori Windows a basso livello e di reverse engineering che studiano il profiling runtime assistito dall'hardware di tight loop e del comportamento di unpacking.
