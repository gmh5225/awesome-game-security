Этот проект — фреймворк драйвера Windows для профилирования с акцентом на циклы на основе аппаратных средств производительности Intel, в частности Last Branch Records и сэмплирования PEBS. В архивном README LoopHPCs описан как фреймворк профилирования для поиска горячих циклов в работающих бинарниках, особенно в malware, ориентированном на распаковку; дерево исходников объединяет вспомогательные средства PEBS, обработку записей ветвлений и логику отслеживания циклов в проекте в стиле filter-driver. Ядро кода потребляет записи PEBS, сопоставляет eventing IP, next IP и адреса данных с контекстом потока управления, полученным из LBR, и затем строит телеметрию, ориентированную на циклы, а не только собирает сырые значения счётчиков. Он в основном полезен исследователям низкоуровневого Windows и reverse engineering, изучающим аппаратно поддерживаемое runtime-профилирование tight loops и поведения распаковки.
