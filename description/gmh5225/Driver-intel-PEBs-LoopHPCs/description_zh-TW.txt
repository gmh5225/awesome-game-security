本專案是一個基於 Intel 硬體效能特性（尤其是 Last Branch Records 與 PEBS 取樣）的 Windows 驅動程式框架，面向以迴圈為中心的效能剖析。其封存 README 將 LoopHPCs 描述為用於在執行中的二進位檔中找出熱點迴圈（尤其是面向脫殼的惡意軟體）的剖析框架；原始碼樹在類過濾驅動結構中整合了 PEBS 輔助、分支記錄處理與迴圈追蹤邏輯。核心程式碼消費 PEBS 記錄，將 eventing IP、next IP 與資料位址與基於 LBR 的控制流上下文關聯，從而建構面向迴圈的遙測，而非僅收集原始計數器值。它主要適用於研究硬體輔助執行期剖析緊迴圈與脫殼行為的底層 Windows 與逆向工程研究者。
