이 프로젝트는 Intel 하드웨어 성능 기능(특히 Last Branch Records와 PEBS 샘플링)을 기반으로 한 루프 중심 프로파일링용 Windows 드라이버 프레임워크입니다. 보관된 README는 LoopHPCs를 실행 중인 바이너리, 특히 언패킹 지향 멀웨어에서 핫 루프를 찾기 위한 프로파일링 프레임워크로 설명하며, 소스 트리는 필터 드라이버 스타일 프로젝트 안에 PEBS 헬퍼, 분기 기록 처리, 루프 추적 로직을 결합합니다. 핵심 코드는 PEBS 레코드를 소비하고 eventing IP, next IP, 데이터 주소를 LBR 기반 제어 흐름 컨텍스트와 상관시킨 뒤, 원시 카운터 값만 수집하는 대신 루프 지향 텔레메트리를 구축합니다. 타이트 루프와 언패킹 동작의 하드웨어 지원 런타임 프로파일링을 연구하는 저수준 Windows 및 리버스 엔지니어링 연구자에게 주로 유용합니다.
