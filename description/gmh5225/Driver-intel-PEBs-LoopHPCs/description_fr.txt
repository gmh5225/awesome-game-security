Ce projet est un framework de pilote Windows pour le profilage centré sur les boucles, fondé sur les fonctionnalités de performance matérielle Intel, en particulier les Last Branch Records et l'échantillonnage PEBS. Son README archivé décrit LoopHPCs comme un framework de profilage destiné à trouver les boucles chaudes dans des binaires en cours d'exécution, notamment des malwares orientés dépaquetage, et l'arborescence source combine des aides PEBS, le traitement des enregistrements de branches et une logique de suivi de boucles dans un projet de type filtre-pilote. Le code central consomme les enregistrements PEBS, corrèle les eventing IP, next IP et adresses de données avec un contexte de flot de contrôle dérivé des LBR, puis construit une télémétrie orientée boucles plutôt que de se limiter aux valeurs brutes de compteurs. Il est surtout utile aux chercheurs Windows bas niveau et en reverse engineering qui étudient le profilage runtime assisté par le matériel des boucles serrées et du comportement de dépaquetage.
