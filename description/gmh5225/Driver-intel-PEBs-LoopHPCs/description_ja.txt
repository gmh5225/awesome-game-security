本プロジェクトは、Intel のハードウェア性能機能（特に Last Branch Records と PEBS サンプリング）に基づく、ループ中心のプロファイリング向け Windows ドライバーフレームワークです。アーカイブされた README では、LoopHPCs を実行中バイナリ内のホットループ（特にアンパッキング指向のマルウェア）を見つけるプロファイリングフレームワークと説明しており、ソースツリーはフィルタードライバー風の構成の中に PEBS ヘルパー、ブランチレコード処理、ループ追跡ロジックを組み合わせています。コアコードは PEBS レコードを消費し、eventing IP、next IP、データアドレスを LBR 由来の制御フロー文脈と相関付けたうえで、生のカウンター値の収集だけでなくループ指向のテレメトリを構築します。タイトなループやアンパッキング挙動のハードウェア支援ランタイムプロファイリングを研究する、低レベル Windows およびリバースエンジニアリング研究者に主に役立ちます。
