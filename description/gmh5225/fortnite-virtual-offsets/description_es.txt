Este proyecto es un pequeño volcado de offsets relacionados con tablas virtuales / vista de Fortnite, que enumera valores como GetPlayerViewPoint, ProcessEvent, LineOfSightTo y ayudas de cámara.
El archivo contiene un único fichero de texto con offsets, sin herramientas adicionales.
Resulta útil principalmente para investigadores de seguridad en juegos que estudian los layouts de offsets de Unreal Engine en ingeniería inversa relacionada con Fortnite.
