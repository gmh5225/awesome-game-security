Questo progetto è un piccolo dump di offset relativi a virtual table / view di Fortnite, che elenca valori come GetPlayerViewPoint, ProcessEvent, LineOfSightTo e helper della camera.
L’archivio contiene un solo file di testo con gli offset, senza strumenti aggiuntivi.
È utile soprattutto ai ricercatori di sicurezza nei giochi che studiano i layout degli offset di Unreal Engine nel reverse engineering legato a Fortnite.
