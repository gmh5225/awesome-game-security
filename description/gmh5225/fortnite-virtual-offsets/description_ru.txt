Этот проект — небольшой дамп смещений Fortnite, связанных с виртуальными таблицами / view, со значениями вроде GetPlayerViewPoint, ProcessEvent, LineOfSightTo и вспомогательных функций камеры.
Архив содержит один текстовый файл со смещениями без дополнительных инструментов.
Он в основном полезен исследователям игровой безопасности, изучающим раскладку смещений Unreal Engine в реверс-инжиниринге, связанном с Fortnite.
