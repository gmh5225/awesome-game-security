Этот проект — SafeCall, библиотека для Windows, которая подменяет адреса возврата и манипулирует стеком вызовов перед вызовами API. Она заменяет настоящие адреса возврата в стеке на поддельные легитимные, не давая анализу стека вызовов раскрыть истинного вызывающего. Этот приём обходит стековую детекцию продуктов EDR и античитов. Предназначена для red team-операторов и исследователей игровой безопасности, изучающих подмену стека вызовов и её обнаружение.
