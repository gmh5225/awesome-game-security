Этот проект — внешний чит для Valorant, который читает память игры через драйвер ядра без внедрения в игровой процесс. Он предоставляет ESP и аимбот, внешне считывая данные сущностей UE4 и обходя обнаружение инъекций Vanguard. Предназначен для исследователей античита, изучающих архитектуру внешних читов для Valorant.
