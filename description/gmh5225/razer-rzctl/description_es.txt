Este proyecto explota el controlador de kernel rzctl.sys de Razer para simulación de entrada o acceso al kernel. El controlador de Razer ofrece operaciones de E/S privilegiadas que permiten simular entrada de ratón/teclado a nivel de kernel, eludir la detección de entrada en modo usuario por sistemas anti-cheat, o obtener acceso a memoria del kernel mediante IOCTL vulnerables. Está dirigido a investigadores de seguridad en juegos que estudian la explotación del controlador Razer para simulación de entrada y BYOVD.
