Этот проект использует уязвимости в драйвере ядра Razer rzctl.sys для симуляции ввода или доступа к ядру. Драйвер Razer предоставляет привилегированные операции ввода-вывода, с помощью которых можно симулировать ввод мыши/клавиатуры на уровне ядра, обходя обнаружение пользовательского ввода античитами, либо получать доступ к памяти ядра через уязвимые IOCTL. Предназначен для исследователей игровой безопасности, изучающих эксплуатацию драйвера Razer для симуляции ввода и BYOVD.
