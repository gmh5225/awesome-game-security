Этот проект — Tenet, плагин IDA Pro для исследования трасс выполнения программ. Он предоставляет визуализацию временной шкалы трассы, позволяя аналитикам шагать вперёд и назад по записанным трассам выполнения, просматривать состояние регистров и памяти в любой момент и понимать сложный поток управления с помощью визуальной навигации. Плагин поддерживает различные форматы трасс и интегрируется с видом дизассемблирования IDA. Он предназначен для реверс-инженеров, использующих трассы выполнения для анализа вредоносного ПО и исследования уязвимостей.
