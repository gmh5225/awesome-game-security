Ce projet est un cheat interne pour CS:GO écrit en C++, offrant ESP, aimbot, backtrack, skin changer et diverses modifications de gameplay. Il s'intègre au moteur Source via la capture d'interfaces et le VMT hooking, avec un menu de configuration basé sur ImGui. Le code suit les schémas classiques des cheats internes CS:GO. Il s'adresse aux chercheurs en sécurité des jeux qui étudient l'implémentation des fonctionnalités de cheat sur le moteur Source.
