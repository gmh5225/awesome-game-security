Этот проект — внутренний чит для CS:GO на C++, предоставляющий ESP, аимбот, бэктрек, скин-чейнджер и прочие изменения геймплея. Он внедряется в движок Source через захват интерфейсов и VMT-хукинг и имеет меню настроек на базе ImGui. Кодовая база следует типичным паттернам внутренних читов CS:GO. Предназначен для исследователей игровой безопасности, изучающих реализацию чит-функций в движке Source.
