이 프로젝트는 nvaudio.sys를 취약한 전송 경로로 악용하는 커널 드라이버 매뉴얼 매퍼이며, 사용자 모드 로더와 인메모리 매핑 로직을 모두 포함합니다.
로더는 SeLoadDriverPrivilege를 활성화하고, 취약한 드라이버를 설치·시작한 뒤 해당 장치를 열고, 내부 EncodePayLoad 루틴을 재사용하기 위해 nvaudio.sys를 사용자 공간에 로드한 다음, 사용 후 헬퍼가 덜 드러나도록 드라이버의 런타임 목록 항목을 제거합니다.
매퍼 코드에는 명시적인 재배치, 임포트 해석, 이미지 로딩 루틴도 포함되어 있어, 기존 매퍼에 대한 얇은 래퍼가 아니라 nvaudio 경로에 맞게 조정된 상당히 완전한 매뉴얼 맵 구현입니다.
주로 BYOVD 기반 드라이버 매핑, 벤더 헬퍼 루틴 재사용, 런타임 목록 주변의 로드 후 정리를 연구하는 Windows 커널 연구자에게 유용합니다.
