Este proyecto exporta la lista de funciones de IDA Pro a una ruta especificada en formato Markdown.
Está organizado como documentación y material de referencia para el área de cheats / plugins de IDA, más que como una base de código runtime independiente.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / plugins de IDA.
