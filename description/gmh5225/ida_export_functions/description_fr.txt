Ce projet permet d'exporterer la liste des fonctions IDA Pro vers un chemin spécifié au format Markdown.
Il est organisé comme documentation et matériel de référence pour le domaine des cheats / plugins IDA, plutôt que comme une base de code runtime autonome.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans ce domaine.
