Este proyecto es una configuración y guía de IDA Pro para depurar el kernel de invitados Windows en VMware mediante el protocolo GDB. Configura el depurador GDB de IDA para conectarse al stub GDB integrado de VMware y permitir la depuración en vivo del kernel de Windows, con puntos de interrupción, inspección de memoria y ejecución paso a paso del código del kernel. Está dirigido a investigadores de kernel que usan IDA Pro para depurar el kernel de Windows a través de VMware.
