Этот проект — конфигурация и руководство для IDA Pro по отладке ядра Windows-гостей в VMware через протокол GDB. Он настраивает GDB-отладчик IDA на подключение к встроенному GDB stub VMware для живой отладки ядра Windows с точками останова, просмотром памяти и пошаговым выполнением кода ядра. Предназначен для исследователей ядра, использующих IDA Pro для отладки ядра Windows через VMware.
