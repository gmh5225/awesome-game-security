Dieses Projekt ist eine IDA-Pro-Konfiguration und Anleitung für das Kernel-Debugging von Windows-Gästen in VMware über das GDB-Protokoll. Es richtet den GDB-Debugger von IDA so ein, dass er sich mit dem eingebauten GDB-Stub von VMware verbindet und so Live-Windows-Kernel-Debugging mit Breakpoints, Speicherinspektion und schrittweisem Durchlaufen von Kernelcode ermöglicht. Es richtet sich an Kernel-Forscher, die IDA Pro für Windows-Kernel-Debugging über VMware nutzen.
