Ce projet est une configuration et un guide IDA Pro pour le débogage noyau de machines invitées Windows sous VMware via le protocole GDB. Il configure le débogueur GDB d'IDA pour se connecter au stub GDB intégré de VMware afin d'assurer un débogage noyau Windows en direct, avec points d'arrêt, inspection mémoire et pas à pas dans le code noyau. Il s'adresse aux chercheurs noyau qui utilisent IDA Pro pour déboguer le noyau Windows via VMware.
