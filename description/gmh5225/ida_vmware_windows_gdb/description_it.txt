Questo progetto è una configurazione e una guida per IDA Pro per il debug del kernel di guest Windows in VMware tramite protocollo GDB. Configura il debugger GDB di IDA per connettersi allo stub GDB integrato di VMware e abilitare il debug live del kernel Windows, con breakpoint, ispezione della memoria ed esecuzione passo-passo del codice kernel. È pensato per i ricercatori di kernel che usano IDA Pro per il debug del kernel Windows tramite VMware.
