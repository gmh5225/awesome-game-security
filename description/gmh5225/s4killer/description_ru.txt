Этот проект эксплуатирует уязвимый драйвер ядра S4 (Samsung) для получения произвольного чтения/записи в ядре Windows. Он отправляет специально сформированные IOCTL подписанному драйверу Samsung для чтения и записи физической или виртуальной памяти, предоставляя примитив BYOVD для загрузки неподписанных драйверов, патчинга структур ядра или обхода античитов. Предназначен для исследователей BYOVD, изучающих уязвимости драйверов Samsung для доступа к ядру.
