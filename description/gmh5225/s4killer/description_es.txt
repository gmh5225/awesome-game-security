Este proyecto explota el controlador de kernel vulnerable S4 (Samsung) para obtener lectura/escritura arbitraria del kernel en Windows. Envía IOCTL manipulados al controlador firmado de Samsung para leer y escribir memoria física o virtual, proporcionando una primitiva BYOVD para cargar controladores sin firmar, parchear estructuras del kernel o eludir protecciones anti-cheat. Está dirigido a investigadores BYOVD que estudian vulnerabilidades de controladores Samsung para acceso al kernel.
