Este proyecto es PPLKiller, una herramienta para desactivar la protección Windows Protected Process Light (PPL) en procesos. Explota un controlador firmado vulnerable o una primitiva de acceso al kernel para modificar el campo de nivel de protección de EPROCESS, degradando procesos protegidos por PPL (como servicios antimalware) a un estado sin protección para depurarlos o terminarlos. Está orientado a investigadores de kernel que estudian la protección PPL y sus métodos de evasión.
