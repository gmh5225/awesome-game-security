Этот проект — PPLKiller, инструмент для отключения защиты Windows Protected Process Light (PPL) у процессов. Он использует уязвимый подписанный драйвер или примитив доступа к ядру, чтобы изменить поле уровня защиты EPROCESS и понизить PPL-защищённые процессы (например, антивирусные службы) до незащищённого состояния для отладки или завершения. Предназначен для исследователей ядра, изучающих защиту PPL и методы её обхода.
