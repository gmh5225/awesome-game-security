本專案是一個使用者模式 LSASS 傾印概念驗證，結合控制代碼竊取、未掛鉤的 DbgHelp 副本以及輕度混淆的傾印路徑。
歸檔的 README 將其方法描述為：從其他程序複製既有的 LSASS 程序控制代碼，然後透過從磁碟載入的全新 DbgHelp.dll 副本呼叫 MiniDumpWriteDump，而非依賴可能已被掛鉤的匯出函式。
其程式碼列舉 SystemHandleInformation，複製候選程序控制代碼，依完整映像路徑是否包含 lsass.exe 進行篩選，將傾印寫入暫存檔，再對磁碟上的結果檔案加密。
它主要適用於研究基於控制代碼的傾印取得、圍繞 MiniDumpWriteDump 的使用者模式掛鉤規避，以及重用特權控制代碼相關取捨的 Windows 安全性研究人員。
