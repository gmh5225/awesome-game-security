本项目是一个用户态 LSASS 转储概念验证，结合句柄窃取、未挂钩的 DbgHelp 副本以及轻度混淆的转储路径。
归档的 README 将其方法描述为：从其他进程复制已有的 LSASS 进程句柄，然后通过从磁盘加载的全新 DbgHelp.dll 副本调用 MiniDumpWriteDump，而非依赖可能已被挂钩的导出函数。
其代码枚举 SystemHandleInformation，复制候选进程句柄，按完整映像路径是否包含 lsass.exe 进行过滤，将转储写入临时文件，再对磁盘上的结果文件加密。
它主要适用于研究基于句柄的转储获取、围绕 MiniDumpWriteDump 的用户态挂钩规避，以及复用特权句柄相关权衡的 Windows 安全研究人员。
