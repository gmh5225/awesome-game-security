Este proyecto es una prueba de concepto de volcado de LSASS en modo usuario que combina el robo de handles con una copia sin hooks de DbgHelp y una ruta de volcado ligeramente ofuscada.
El README archivado describe el enfoque como duplicar un handle de proceso LSASS existente desde otro proceso y luego llamar a MiniDumpWriteDump a través de una copia nueva de DbgHelp.dll cargada desde el disco, en lugar de depender de exports posiblemente enganchados.
Su código enumera SystemHandleInformation, duplica handles de proceso candidatos, los filtra por la ruta completa de imagen que contiene lsass.exe, escribe el volcado en un archivo temporal y luego cifra el archivo resultante en disco.
Resulta especialmente útil para investigadores de seguridad de Windows que estudian la adquisición de volcados basada en handles, la evasión de hooks en modo usuario en torno a MiniDumpWriteDump y las compensaciones de handles privilegiados reciclados.
