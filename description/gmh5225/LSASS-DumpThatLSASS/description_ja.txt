本プロジェクトは、ハンドル窃取とアンフック済みの DbgHelp コピー、および軽く難読化されたダンプ経路を組み合わせたユーザーモード LSASS ダンプの概念実証です。
アーカイブされた README では、他プロセスから既存の LSASS プロセスハンドルを複製し、フックされている可能性のあるエクスポートに頼らず、ディスクから読み込んだ新しい DbgHelp.dll コピー経由で MiniDumpWriteDump を呼び出す手法として説明されています。
コードは SystemHandleInformation を列挙し、候補のプロセスハンドルを複製し、フルイメージパスに lsass.exe を含むもので絞り込んだうえ、ダンプを一時ファイルに書き出し、その結果ファイルをディスク上で暗号化します。
主に、ハンドルベースのダンプ取得、MiniDumpWriteDump 周りのユーザーモードフック回避、および再利用された特権ハンドルのトレードオフを研究する Windows セキュリティ研究者向けです。
