이 프로젝트는 핸들 탈취와 언훅된 DbgHelp 복사본, 그리고 가볍게 난독화된 덤프 경로를 결합한 사용자 모드 LSASS 덤프 개념 증명입니다.
보관된 README는 다른 프로세스에서 기존 LSASS 프로세스 핸들을 복제한 뒤, 훅되었을 수 있는 export에 의존하지 않고 디스크에서 로드한 새로운 DbgHelp.dll 복사본을 통해 MiniDumpWriteDump를 호출하는 접근으로 설명합니다.
코드는 SystemHandleInformation을 열거하고, 후보 프로세스 핸들을 복제하며, 전체 이미지 경로에 lsass.exe가 포함된 항목으로 필터링한 다음 덤프를 임시 파일에 쓰고 디스크상의 결과 파일을 암호화합니다.
주로 핸들 기반 덤프 획득, MiniDumpWriteDump 주변의 사용자 모드 훅 회피, 그리고 재사용된 특권 핸들의 트레이드오프를 연구하는 Windows 보안 연구자에게 유용합니다.
