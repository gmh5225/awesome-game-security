Инструмент, создающий поддельные файлы PDB (Program Database) для исполняемых файлов без отладочных символов.
Формирует синтетическую информацию о символах на основе результатов анализа в IDA, что позволяет выполнять отладку на уровне исходного кода и разрешать символы в инструментах, работающих с файлами PDB.
