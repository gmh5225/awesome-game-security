Этот проект — исходный код внешнего чита для Valorant с функциями ESP, аимбота и информации об игроках через внешнее чтение памяти. Он использует драйвер ядра для чтения игрового состояния UE4 Valorant без инъекции в процесс, защищённый Vanguard, и отрисовывает оверлеи через отдельное окно или перехваченный оверлей. Предназначен для исследователей античита, изучающих архитектуры внешних читов Valorant и эффективность ядерной защиты Vanguard.
