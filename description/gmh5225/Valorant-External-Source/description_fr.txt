Ce projet est un code source de cheat externe pour Valorant offrant ESP, aimbot et informations joueur via lecture mémoire externe. Il utilise un pilote noyau pour lire l'état de jeu UE4 de Valorant sans injection dans le processus protégé par Vanguard, et affiche les overlays via une fenêtre séparée ou un overlay détourné. Il s'adresse aux chercheurs anti-cheat qui étudient les architectures de cheats externes Valorant et l'efficacité de la protection noyau Vanguard.
