Este proyecto es un código fuente de cheat externo para Valorant que ofrece ESP, aimbot e información de jugadores mediante lectura de memoria externa. Usa un controlador de kernel para leer el estado del juego UE4 de Valorant sin inyectarse en el proceso protegido por Vanguard, y renderiza overlays a través de una ventana separada o un overlay secuestrado. Está dirigido a investigadores de anti-cheat que estudian arquitecturas de cheats externos de Valorant y la efectividad de la protección de kernel de Vanguard.
