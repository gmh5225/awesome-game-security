Este proyecto es un exploit de kernel que aprovecha NtUserHardErrorControl para elevar un hilo a KernelMode y lograr lectura/escritura arbitraria del kernel y más.
Está escrito principalmente en C/C++ y C, y se centra en el trabajo y el modding a nivel de kernel.
Resulta especialmente útil para investigadores de bajo nivel en Windows, Linux y móvil que trabajan en el área some tricks / windows ring3.
