Questo progetto è un exploit kernel che sfrutta NtUserHardErrorControl per elevare un thread a KernelMode e ottenere R/W arbitrario del kernel e altro ancora.
È scritto principalmente in C/C++ e C e si concentra sul lavoro a livello kernel e sul modding.
È soprattutto utile per ricercatori low-level su Windows, Linux e mobile che operano nell'area some tricks / windows ring3.
