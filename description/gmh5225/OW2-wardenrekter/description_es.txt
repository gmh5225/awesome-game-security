Una DLL que desactiva las protecciones antitrampas Warden de Overwatch 2 parcheando múltiples vectores de detección en DLL_PROCESS_ATTACH: sobrescribe KiUserExceptionDispatcher con un RET (0xC3) para anular la supervisión de hooks INT3 basada en VEH de Warden, hace NOP a las comprobaciones de integridad de DbgBreakPoint/DbgUserBreakPoint, pone a cero PEB.BeingDebugged y NtGlobalFlag para ocultar la presencia de un depurador, sustituye GetTickCount64 para eludir las comprobaciones de temporización y parchea NtQuerySystemInformation para bloquear consultas de información del sistema.
También configura el registro de depuración por hardware DR0 mediante SetThreadContext para demostrar la manipulación de registros de depuración frente a la detección de puntos de interrupción por hardware de Warden.
Resulta especialmente útil para investigadores de seguridad en videojuegos que estudian técnicas de bypass del anti-cheat Warden, incluidas la eliminación de hooks VEH, el spoofing del PEB y el parcheo de API en Overwatch 2.
