DLL, которая отключает защиту античита Warden в Overwatch 2, патча несколько векторов обнаружения при DLL_PROCESS_ATTACH: перезаписывает KiUserExceptionDispatcher инструкцией RET (0xC3), чтобы нейтрализовать мониторинг INT3-хуков Warden на основе VEH, NOP-ит проверки целостности DbgBreakPoint/DbgUserBreakPoint, обнуляет PEB.BeingDebugged и NtGlobalFlag для скрытия отладчика, заглушает GetTickCount64 для обхода проверок по времени и патчит NtQuerySystemInformation, чтобы блокировать запросы системной информации.
Также устанавливает аппаратный отладочный регистр DR0 через SetThreadContext, демонстрируя манипуляцию отладочными регистрами против обнаружения аппаратных точек останова Warden.
В основном полезна исследователям безопасности игр, изучающим техники обхода античита Warden, включая удаление VEH-хуков, спуфинг PEB и патчинг API в Overwatch 2.
