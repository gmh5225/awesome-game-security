Una DLL che disabilita le protezioni anti-cheat Warden di Overwatch 2 patchando più vettori di rilevamento in DLL_PROCESS_ATTACH: sovrascrive KiUserExceptionDispatcher con un RET (0xC3) per annullare il monitoraggio degli hook INT3 basato su VEH di Warden, esegue NOP sui controlli di integrità di DbgBreakPoint/DbgUserBreakPoint, azzera PEB.BeingDebugged e NtGlobalFlag per nascondere la presenza di un debugger, stubba GetTickCount64 per eludere i controlli di timing e patcha NtQuerySystemInformation per bloccare le query sulle informazioni di sistema.
Imposta inoltre il registro di debug hardware DR0 tramite SetThreadContext per dimostrare la manipolazione dei registri di debug contro il rilevamento dei breakpoint hardware di Warden.
È soprattutto utile ai ricercatori di sicurezza nei videogiochi che studiano tecniche di bypass dell'anti-cheat Warden, tra cui rimozione di hook VEH, spoofing del PEB e patching di API in Overwatch 2.
