DLL_PROCESS_ATTACH 時に複数の検知ベクタをパッチして Overwatch 2 の Warden アンチチート保護を無効化する DLL です。KiUserExceptionDispatcher を RET（0xC3）で上書きして Warden の VEH ベース INT3 フック監視を無効化し、DbgBreakPoint/DbgUserBreakPoint の整合性チェックを NOP し、PEB.BeingDebugged と NtGlobalFlag をゼロクリアしてデバッガの存在を隠し、GetTickCount64 をスタブ化してタイミング検査を無効化し、NtQuerySystemInformation をパッチしてシステム情報クエリを遮断します。
また SetThreadContext 経由でハードウェアデバッグレジスタ DR0 を設定し、Warden のハードウェアブレークポイント検知に対するデバッグレジスタ操作を示します。
主に、VEH フック除去、PEB スプーフィング、Overwatch 2 における API パッチなど、Warden アンチチート迂回手法を研究するゲームセキュリティ研究者向けです。
