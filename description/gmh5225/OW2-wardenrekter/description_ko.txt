DLL_PROCESS_ATTACH 시점에 여러 탐지 벡터를 패치하여 Overwatch 2의 Warden 안티치트 보호를 비활성화하는 DLL입니다. KiUserExceptionDispatcher를 RET(0xC3)로 덮어써서 Warden의 VEH 기반 INT3 훅 모니터링을 무력화하고, DbgBreakPoint/DbgUserBreakPoint 무결성 검사를 NOP 처리하며, PEB.BeingDebugged와 NtGlobalFlag를 0으로 만들어 디버거 존재를 숨기고, GetTickCount64를 스텁화하여 타이밍 검사를 우회하며, NtQuerySystemInformation을 패치하여 시스템 정보 조회를 차단합니다.
또한 SetThreadContext로 하드웨어 디버그 레지스터 DR0을 설정하여 Warden의 하드웨어 브레이크포인트 탐지에 대한 디버그 레지스터 조작을 시연합니다.
주로 VEH 훅 제거, PEB 스푸핑, Overwatch 2에서의 API 패칭 등 Warden 안티치트 우회 기법을 연구하는 게임 보안 연구자에게 유용합니다.
