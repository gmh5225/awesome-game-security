一個在 DLL_PROCESS_ATTACH 時透過修補多個偵測向量來停用《鬥陣特攻 2》Warden 反作弊保護的 DLL：它以 RET（0xC3）覆寫 KiUserExceptionDispatcher 以破壞 Warden 基於 VEH 的 INT3 hook 監控，對 DbgBreakPoint/DbgUserBreakPoint 完整性檢查進行 NOP，清零 PEB.BeingDebugged 與 NtGlobalFlag 以隱藏偵錯器存在，樁化 GetTickCount64 以繞過計時檢測，並修補 NtQuerySystemInformation 以阻擋系統資訊查詢。
它也會透過 SetThreadContext 設定硬體偵錯暫存器 DR0，以示範針對 Warden 硬體中斷點偵測的偵錯暫存器操控。
此專案主要供研究 Warden 反作弊繞過技術的遊戲安全研究者使用，涵蓋 VEH hook 移除、PEB 偽造以及《鬥陣特攻 2》中的 API 修補等。
