Этот проект — плагин для IDA Pro, улучшающий анализ бинарных файлов, скомпилированных на Go (Golang). Он разбирает метаданные среды выполнения Go, чтобы восстановить имена функций, ссылки на исходные файлы и информацию о типах из стрипнутых исполняемых файлов Go, применяя символы и определения типов, которые стандартный анализ IDA пропускает. Плагин предназначен для специалистов по реверс-инжинирингу, анализирующих бинарники на Go, в особенности вредоносное ПО на Go.
