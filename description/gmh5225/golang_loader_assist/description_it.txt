Questo progetto è un plugin di IDA Pro per migliorare l'analisi dei binari compilati in Go (Golang). Analizza i metadati del runtime di Go per recuperare nomi di funzioni, riferimenti ai file sorgente e informazioni sui tipi da eseguibili Go senza simboli, applicando simboli e definizioni di tipi che l'analisi standard di IDA non rileva. È rivolto ai reverse engineer che analizzano binari Go, in particolare malware basato su Go.
