Este proyecto es un plugin de IDA Pro para mejorar el análisis de binarios compilados en Go (Golang). Analiza los metadatos del runtime de Go para recuperar nombres de funciones, referencias a archivos fuente e información de tipos a partir de ejecutables Go sin símbolos, aplicando símbolos y definiciones de tipos que el análisis estándar de IDA no detecta. Está dirigido a ingenieros de ingeniería inversa que analizan binarios Go, en particular malware basado en Go.
