Этот проект — dp701, тема или цветовая схема для IDA Pro, обеспечивающая настраиваемый внешний вид среды дизассемблирования IDA. Она применяет характерную палитру к подсветке синтаксиса, цветам фона и элементам интерфейса для повышения читаемости во время сессий реверс-инжиниринга. Предназначена для пользователей IDA Pro, ищущих альтернативные визуальные темы.
