Dieses Projekt zeigt, wie man in das Grafik-Capture-System von OBS Studio einklinkt, um benutzerdefinierte Overlays über die vertrauenswürdige Rendering-Pipeline von OBS darzustellen. Durch Hijacking der Game-Capture-Hook-DLL von OBS können benutzerdefinierte Draw Calls in den Frame des erfassten Spiels injiziert werden, ohne ein separates Overlay-Fenster zu erzeugen. Diese Technik nutzt den Whitelist-Status von OBS in Anti-Cheat-Systemen aus. Sie richtet sich an Spielsicherheitsforscher, die Overlay-Rendering durch Hijacking vertrauenswürdiger Anwendungen untersuchen.
