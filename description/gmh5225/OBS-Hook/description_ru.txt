Этот проект демонстрирует перехват системы графического захвата OBS Studio для отрисовки пользовательских оверлеев через доверенный конвейер рендеринга OBS. Перехватив hook DLL игрового захвата OBS, можно внедрять пользовательские вызовы отрисовки в кадр захваченной игры без создания отдельного окна оверлея. Техника использует статус OBS в белых списках античитов. Предназначена для исследователей игровой безопасности, изучающих рендеринг оверлеев через перехват доверенных приложений.
