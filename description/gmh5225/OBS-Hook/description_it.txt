Questo progetto mostra come agganciarsi al sistema di cattura grafica di OBS Studio per disegnare overlay personalizzati tramite la pipeline di rendering affidabile di OBS. Dirottando la DLL di hook della game capture di OBS, è possibile iniettare chiamate di disegno personalizzate nel frame del gioco acquisito senza creare una finestra overlay separata. Questa tecnica sfrutta lo stato di whitelist di OBS nei sistemi anti-cheat. È rivolta ai ricercatori di sicurezza dei giochi che studiano il rendering di overlay tramite dirottamento di applicazioni affidabili.
