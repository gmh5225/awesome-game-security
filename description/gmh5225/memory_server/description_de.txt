Rust-basierter HTTP-Server für gejailbreakte iOS-Geräte, der Prozessspeicher-Operationen—Auflistung, Regionenliste, Lesen, Pattern-Scanning und Ergebnisfilterung—über eine REST-API auf Port 3030 bereitstellt und mit dem mitgelieferten Python-Beispielclient die ferngesteuerte Speicheranalyse von einem verbundenen PC aus ermöglicht.
