HTTP-сервер на Rust для взломанных (jailbroken) устройств iOS, который через REST API на порту 3030 предоставляет операции с памятью процессов—перечисление, список регионов, чтение, сканирование по шаблону и фильтрацию результатов—и позволяет удалённо анализировать память с подключённого ПК с помощью прилагаемого примера клиента на Python.
