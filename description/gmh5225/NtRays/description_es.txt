Este proyecto es NtRays, un complemento de IDA Pro que mejora la salida del descompilador Hex-Rays para código del kernel de Windows al resolver definiciones de tipos y referencias a estructuras del kernel NT. Anota códigos NTSTATUS, definiciones IOCTL, tipos de objetos del kernel y accesos a campos de EPROCESS/ETHREAD con los nombres y tipos adecuados. Está dirigido a ingenieros de ingeniería inversa que analizan controladores del kernel de Windows y código de ntoskrnl en IDA Pro.
