Ce projet est NtRays, un plugin IDA Pro qui améliore la sortie du décompilateur Hex-Rays pour le code du noyau Windows en résolvant les définitions de types et les références de structures du noyau NT. Il annote les codes NTSTATUS, les définitions IOCTL, les types d'objets du noyau et les accès aux champs EPROCESS/ETHREAD avec les noms et types appropriés. Il s'adresse aux ingénieurs en reverse engineering qui analysent les pilotes du noyau Windows et le code de ntoskrnl dans IDA Pro.
