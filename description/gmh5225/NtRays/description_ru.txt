Этот проект — NtRays, плагин для IDA Pro, который улучшает вывод декомпилятора Hex-Rays для кода ядра Windows, разрешая определения типов ядра NT и ссылки на структуры. Он аннотирует коды NTSTATUS, определения IOCTL, типы объектов ядра и обращения к полям EPROCESS/ETHREAD правильными именами и типами. Предназначен для специалистов по реверс-инжинирингу, анализирующих драйверы ядра Windows и код ntoskrnl в IDA Pro.
