Dieses Projekt konzentriert sich auf die Analyse von WebAssembly.
Es ist hauptsächlich in Python und JavaScript geschrieben und stellt die Speicheranalyse in den Mittelpunkt.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats und Cheat-Engine-Plugins untersuchen.
