Questo progetto riguarda tecniche di costruzione o spoofing di stack trace per eludere l'analisi dello stack di chiamate. Manipola gli indirizzi di ritorno o costruisce call stack falsi per ingannare prodotti di sicurezza e sistemi anti-cheat che ispezionano gli stack di chiamata dei thread alla ricerca di pattern di chiamata sospetti. È rivolto a ricercatori di sicurezza che studiano lo spoofing dello stack di chiamate e l'evasione delle rilevazioni basate sullo stack.
