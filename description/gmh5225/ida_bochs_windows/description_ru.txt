Этот проект — руководство по настройке или плагин для использования отладочного бэкенда Bochs в IDA Pro под Windows. Он настраивает IDA для отладки двоичных файлов через эмуляцию Bochs, обеспечивая полноценную системную отладку, включая анализ кода в режиме ядра, за счёт программной эмуляции ЦП в Bochs. Предназначен для пользователей IDA Pro, которым нужна отладка на основе эмуляции Bochs в Windows.
