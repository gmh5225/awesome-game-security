Ce projet est un guide de configuration ou un plugin pour utiliser le backend débogueur Bochs d'IDA Pro sous Windows. Il configure IDA pour déboguer des binaires via l'émulation Bochs, permettant le débogage système complet, y compris l'analyse de code en mode noyau, grâce à l'émulation logicielle du CPU de Bochs. Il s'adresse aux utilisateurs d'IDA Pro ayant besoin d'un débogage par émulation Bochs sous Windows.
