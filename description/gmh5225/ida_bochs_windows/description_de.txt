Dieses Projekt ist eine Konfigurationsanleitung oder ein Plugin zur Nutzung des Bochs-Debugger-Backends von IDA Pro unter Windows. Es richtet IDA so ein, dass Binaries über die Bochs-Emulation debuggt werden können, und ermöglicht damit Full-System-Debugging einschließlich Kernel-Mode-Codeanalyse über die Software-CPU-Emulation von Bochs. Es richtet sich an IDA-Pro-Nutzer, die Bochs-basiertes Emulationsdebugging unter Windows benötigen.
