Dieses Projekt kombiniert einen Benutzerclient und einen manuell gemappten Kernel-Treiber, die NtQueryIntervalProfile als Steuerkanal zum Umschalten des Easy-Anti-Cheat-Zustands zweckentfremden.
Der Treiber ersetzt Einträge in HalDispatchTable, sodass über NtQueryIntervalProfile weitergeleitete Aufrufe einen benutzerdefinierten Handler erreichen; anschließend lokalisiert er Threads von EasyAntiCheat.sys, setzt sie aus oder setzt sie fort und deaktiviert vorübergehend Object-Callbacks auf der verfolgten Altitude, bevor er sie später wiederherstellt.
Der Code auf Benutzerseite baut einen kleinen Shellcode-Stub, löst NtQueryIntervalProfile aus ntdll auf und sendet CODE_DISABLE- oder CODE_RESTORE-Anfragen; zugleich steuert er den DLL-Ladeablauf rund um den Zielprozess.
Er ist vor allem für Reverse Engineers nützlich, die syscall-gestützte Treiberkommunikation, den Missbrauch von HalDispatchTable sowie Callback- oder Thread-Manipulation gegen Easy Anti-Cheat untersuchen.
