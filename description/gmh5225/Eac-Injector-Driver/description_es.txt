Este proyecto combina un cliente de usuario y un controlador de kernel mapeado manualmente que reutilizan NtQueryIntervalProfile como canal de control para alternar el estado de Easy Anti-Cheat.
El controlador reemplaza entradas de HalDispatchTable para que las llamadas enrutadas por NtQueryIntervalProfile lleguen a un manejador personalizado; luego localiza hilos de EasyAntiCheat.sys, los suspende o reanuda y deshabilita temporalmente los object callbacks en la altitude rastreada antes de restaurarlos después.
El código del lado del usuario construye un pequeño stub de shellcode, resuelve NtQueryIntervalProfile desde ntdll y envía solicitudes CODE_DISABLE o CODE_RESTORE, además de gestionar el flujo de carga de DLL en torno al proceso objetivo.
Resulta especialmente útil para ingenieros inversos que estudian la comunicación con controladores basada en syscalls, el abuso de HalDispatchTable y la manipulación de callbacks o hilos frente a Easy Anti-Cheat.
