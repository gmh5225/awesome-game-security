Драйвер ядра, реализующий обмен данными через указатели для скрытого взаимодействия пользовательского режима и ядра.
Использует общие указатели данных вместо традиционных IOCTL для передачи запросов чтения/записи памяти между читом в пользовательском режиме и драйвером режима ядра, обходя обнаружение на основе IOCTL.
