Pilote noyau implémentant une communication basée sur des pointeurs de données pour une interaction discrète entre l'espace utilisateur et le noyau.
Utilise des pointeurs de données partagés plutôt que les IOCTL traditionnels pour échanger des demandes de lecture/écriture mémoire entre un cheat en mode utilisateur et un pilote en mode noyau, contournant ainsi la détection basée sur les IOCTL.
