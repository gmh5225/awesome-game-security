Un controlador de núcleo que implementa comunicación basada en punteros de datos para una interacción sigilosa entre el modo de usuario y el núcleo.
Usa punteros de datos compartidos en lugar de IOCTL tradicionales para intercambiar solicitudes de lectura/escritura de memoria entre un cheat en modo de usuario y un controlador en modo núcleo, evadiendo la detección basada en IOCTL.
