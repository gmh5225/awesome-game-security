Questo progetto è uno strumento per convertire i temi di Visual Studio Code (VsCode) in temi compatibili con IDA Pro.
L’utilità si basa su una versione modificata del progetto Long Night con Celestial come tema di base e poi rimappa i colori definiti nel file JSON del tema VsCode per creare un tema esteticamente gradevole e funzionale per IDA Pro.
È soprattutto utile per ricercatori di sicurezza dei giochi e reverse engineer che studiano tecniche offensive e lavorano nell’ambito dei temi cheat / ida.
