Este proyecto es una herramienta para convertir temas de Visual Studio Code (VsCode) en temas compatibles con IDA Pro.
Esta utilidad aprovecha una versión modificada del proyecto Long Night con Celestial como tema base y luego remapea los colores definidos en el archivo JSON del tema de VsCode para crear un tema atractivo y funcional para IDA Pro.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas y trabajan en el ámbito de los temas cheat / ida.
