Questo progetto è un fixer dell'IAT (Import Address Table) per i binari del gioco Overwatch. Ricostruisce la tabella di import dell'eseguibile protetto di Overwatch risolvendo le voci di import offuscate o crittografate nei nomi di funzione e negli indirizzi originali, consentendo un'analisi corretta nei disassemblatori. È rivolto agli ingegneri di reverse engineering di giochi che analizzano la protezione binaria e l'offuscamento degli import di Overwatch.
