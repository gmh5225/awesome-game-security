Ce projet est un cheat interne Fortnite mis à jour, basé sur le code source du cheat Ritz. Il fournit des fonctionnalités d'ESP, d'aimbot, d'exploits et de manipulation de jeu via un hooking interne du moteur UE4 dans le processus du jeu Fortnite. Ce cheat C++ a été actualisé pour les versions plus récentes de Fortnite, avec des offsets rafraîchis et des techniques de contournement d'EAC. Il s'adresse aux chercheurs en sécurité des jeux étudiant l'évolution des cheats internes Fortnite et les méthodes de contournement d'EAC.
