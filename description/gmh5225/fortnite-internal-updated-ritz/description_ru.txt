Этот проект — обновлённый внутренний чит для Fortnite на основе исходников чита Ritz. Он предоставляет ESP, аимбот, эксплойты и функции манипуляции игрой через внутренний хукинг движка UE4 в процессе Fortnite. Чит на C++ обновлён для более новых версий Fortnite с актуальными оффсетами и техниками обхода EAC. Предназначен для исследователей игровой безопасности, изучающих эволюцию внутренних читов Fortnite и методы обхода EAC.
