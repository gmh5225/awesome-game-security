Этот проект — дампер SDK и смещений для PUBG (PlayerUnknown's Battlegrounds), который извлекает определения классов, смещения свойств и адреса функций из работающего процесса игры PUBG. Он сканирует структуры Unreal Engine в PUBG, чтобы дампить массивы GObject/GNames, иерархии классов сущностей и игровые раскладки данных для разработки читов. Предназначен для исследователей безопасности игр, изучающих раскладку памяти UE4 в PUBG, и исследователей античита, анализирующих инструменты читерства.
