Questo progetto è un dumper di SDK e offset per PUBG (PlayerUnknown's Battlegrounds) che estrae definizioni di classi, offset delle proprietà e indirizzi di funzioni dal processo di gioco PUBG in esecuzione. Analizza le strutture Unreal Engine di PUBG per eseguire il dump degli array GObject/GNames, delle gerarchie di classi delle entità e dei layout di dati specifici del gioco per lo sviluppo di cheat. È rivolto a ricercatori di sicurezza nei giochi che studiano il layout di memoria UE4 di PUBG e a ricercatori anti-cheat che analizzano gli strumenti di cheating.
