Этот проект — исходный код чита для Valorant с функциями ESP, аимбота и смены скинов. Он читает состояние игры UE4 через доступ к памяти драйвером ядра, чтобы отображать информацию об игроках, автоматизировать прицеливание и изменять ID скинов оружия на стороне клиента. Кодовая база на C++ демонстрирует типичную архитектуру чита Valorant, включая техники обхода Vanguard. Материал предназначен для исследователей античита, изучающих реализации читов Valorant и обход ядерной защиты Vanguard.
