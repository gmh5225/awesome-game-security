Questo progetto è un codice sorgente di cheat per Valorant che offre funzioni di ESP, aimbot e skin changer. Legge lo stato di gioco UE4 tramite accesso in memoria con un driver kernel per visualizzare le informazioni sui giocatori, automatizzare la mira e modificare lato client gli ID delle skin delle armi. La codebase C++ illustra un’architettura tipica di cheat Valorant, incluse tecniche di bypass di Vanguard. È rivolto ai ricercatori anti-cheat che studiano le implementazioni di cheat Valorant e il bypass della protezione kernel di Vanguard.
