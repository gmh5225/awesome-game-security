Este proyecto es un código fuente de cheat de Valorant que ofrece funciones de ESP, aimbot y skin changer. Lee el estado del juego UE4 mediante acceso a memoria con un controlador de kernel para mostrar información de jugadores, automatizar la puntería y modificar en el cliente los ID de skins de armas. La base de código en C++ muestra una arquitectura típica de cheat de Valorant, incluidas técnicas de bypass de Vanguard. Está orientado a investigadores de anti-cheat que estudian implementaciones de cheats de Valorant y el bypass de la protección de kernel de Vanguard.
