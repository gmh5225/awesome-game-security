Ce projet est un code source de cheat Valorant proposant des fonctionnalités d’ESP, d’aimbot et de skin changer. Il lit l’état du jeu UE4 via un accès mémoire par pilote noyau afin d’afficher les informations des joueurs, d’automatiser la visée et de modifier côté client les ID de skins d’armes. La base de code C++ illustre une architecture typique de cheat Valorant, y compris des techniques de contournement de Vanguard. Il s’adresse aux chercheurs en anti-cheat qui étudient les implémentations de cheats Valorant et le contournement de la protection noyau Vanguard.
