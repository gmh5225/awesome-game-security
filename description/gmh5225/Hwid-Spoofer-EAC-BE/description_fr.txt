Un spoofeur d’identifiant matériel (HWID) ciblant spécifiquement les systèmes anti-triche EasyAntiCheat et BattlEye.
Il spoofe les numéros de série de disque, les adresses MAC des NIC et les identifiants SMBIOS via des hooks de pilote en mode noyau, afin d’échapper aux bans basés sur le HWID dans les jeux protégés.
