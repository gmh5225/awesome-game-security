Este repositorio es una lista de referencia compacta de rutas de llamada desde modo usuario a modo kernel, útil para estudiar cómo los puntos de entrada de gráficos e interfaz de Windows fluyen hacia componentes del kernel de más bajo nivel.
Su README archivado se centra en ejemplos concretos, como llamadas a `win32u.dll` que se resuelven a través de `win32k.sys` y `win32base.sys` antes de alcanzar objetivos en `dxgkrnl.sys`, por lo que se parece más a un cuaderno de mapeo que a un proyecto de código tradicional.
El valor del repositorio está en los ejemplos de cadenas de llamadas en sí, que ofrecen pistas rápidas para rastrear rutas de comunicación y luego analizarlas en el depurador o el desensamblador.
Resulta especialmente útil para investigadores de kernel de Windows y de ingeniería inversa que buscan un punto de partida para investigar rutas de syscalls y superficies de comunicación de controladores.
