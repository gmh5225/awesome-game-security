Этот репозиторий — компактный справочный список путей вызовов из пользовательского режима в режим ядра, полезный для изучения того, как графические и UI-точки входа Windows переходят в низкоуровневые компоненты ядра.
Архивный README сосредоточен на конкретных примерах: например, вызовы `win32u.dll`, которые разрешаются через `win32k.sys` и `win32base.sys`, прежде чем достичь целей в `dxgkrnl.sys`; это скорее блокнот с картой путей, чем обычный кодовый проект.
Ценность репозитория — в самих примерах цепочек вызовов: они дают быстрые ориентиры для трассировки путей связи, которые затем можно анализировать в отладчике или дизассемблере.
В основном он полезен исследователям ядра Windows и специалистам по реверс-инжинирингу, которым нужна отправная точка для изучения путей syscall и поверхностей взаимодействия с драйверами.
