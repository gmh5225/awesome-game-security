이 저장소는 사용자 모드에서 커널 모드로 이어지는 호출 경로를 정리한 간결한 참고 목록으로, Windows 그래픽 및 UI 진입점이 하위 커널 구성 요소로 어떻게 흐르는지 연구하는 데 활용할 수 있습니다.
보관된 README는 `win32u.dll` 호출이 `win32k.sys`와 `win32base.sys`를 거쳐 `dxgkrnl.sys`의 대상에 도달하는 것과 같은 구체적 예시에 초점을 두며, 전통적인 코드 프로젝트라기보다 매핑 노트에 가깝습니다.
이 저장소의 가치는 호출 체인 예시 자체에 있으며, 통신 경로를 추적하기 위한 빠른 단서를 제공해 이후 디버거나 디스어셈블러에서 분석할 수 있게 합니다.
주로 syscall 경로와 드라이버 통신 표면을 조사할 출발점이 필요한 Windows 커널 및 리버스 엔지니어링 연구자에게 유용합니다.
