Este proyecto es una colección actualizada con regularidad de offsets de memoria de CS:GO, que incluye dumps de netvars, punteros de interfaces y patrones de firmas. Proporciona las direcciones de memoria necesarias para el desarrollo de cheats, actualizadas tras cada parche de CS:GO. Los datos incluyen offsets de campos de entidades, direcciones de datos de armas y punteros globales del motor. Está dirigido a desarrolladores de cheats de CS:GO e investigadores de seguridad en juegos que siguen los cambios del binario del cliente.
