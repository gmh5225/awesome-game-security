Этот проект — регулярно обновляемая коллекция смещений памяти CS:GO, включающая дампы netvar, указатели интерфейсов и сигнатурные паттерны. Он предоставляет адреса памяти, необходимые для разработки читов, и обновляется после каждого патча CS:GO. Данные включают смещения полей сущностей, адреса данных оружия и глобальные указатели движка. Предназначен для разработчиков читов CS:GO и исследователей игровой безопасности, отслеживающих изменения клиентского бинарника.
