게임 위에 투명한 DirectX 9 오버레이 창을 만들고, 커널 드라이버(Driver.h/Imports.h 경유)로 게임 엔티티 데이터를 읽으며, D3DXFont와 D3DXLine 그리기 프리미티브로 플레이어 위치·이름·체력 바·거리를 렌더링하는 외부 DayZ ESP 치트입니다.
오버레이는 FindWindow/GetWindowRect로 대상 창을 지속적으로 추적하고, Direct3DCreate9Ex로 투명 렌더링 서피스를 만들며, SDK에 정의된 오프셋을 따라 게임 엔티티 목록을 순회해 각 플레이어와 아이템의 월드-투-스크린 투영 좌표를 추출합니다.
주로 DayZ/Enfusion 엔진 타이틀에서 외부 오버레이 기반 ESP 구현과 드라이버 지원 메모리 읽기 패턴을 연구하는 게임 보안 연구자에게 유용합니다.
