一款外部 DayZ ESP 作弊工具，會在遊戲上方建立透明的 DirectX 9 覆蓋視窗，透過核心驅動程式（經由 Driver.h/Imports.h）讀取遊戲實體資料，並使用 D3DXFont 與 D3DXLine 繪製基元渲染玩家位置、名稱、血條與距離。
該覆蓋層透過 FindWindow/GetWindowRect 持續追蹤目標視窗，使用 Direct3DCreate9Ex 建立透明渲染表面，並沿 SDK 定義的偏移走訪遊戲實體清單，擷取每位玩家與物品的世界座標到螢幕座標投影。
它主要適用於遊戲安全研究者，用於研究 DayZ/Enfusion 引擎遊戲中基於外部覆蓋層的 ESP 實作，以及由驅動程式支援的記憶體讀取模式。
