一款外部 DayZ ESP 作弊工具，会在游戏上方创建透明的 DirectX 9 覆盖窗口，通过内核驱动（经由 Driver.h/Imports.h）读取游戏实体数据，并使用 D3DXFont 与 D3DXLine 绘制基元渲染玩家位置、名称、血条和距离。
该覆盖层通过 FindWindow/GetWindowRect 持续跟踪目标窗口，使用 Direct3DCreate9Ex 创建透明渲染表面，并沿 SDK 定义的偏移遍历游戏实体列表，提取每位玩家与物品的世界坐标到屏幕坐标投影。
它主要适用于游戏安全研究者，用于研究 DayZ/Enfusion 引擎游戏中基于外部覆盖层的 ESP 实现，以及由驱动支持的内存读取模式。
