Версионированная коллекция дампов смещений памяти PUBG, охватывающая десятки обновлений игры с версии 19.1 по 24.2; каждый заголовочный файл содержит жёстко заданные адреса для GObjects, GWorld, ключей расшифровки Xenuine, FNameEntry, полей состояния игрока (Health, GroggyHealth, TeamNum), параметров камеры, данных оружия, индексов костей, структур транспорта и смещений предметов/лута.
Каждый файл смещений для конкретной версии отслеживает изменения между патчами PUBG, включая зашифрованные базы указателей (XenuineDecrypt), массивы костей скелетных мешей, данные траектории/баллистики оружия и поля обнаружения реплея/наблюдателя, предоставляя историческую запись того, как эволюционировала раскладка памяти UE4 игры по мере обновлений.
В основном полезен исследователям игровой безопасности, изучающим эволюцию смещений UE4, изменения схемы шифрования Xenuine и усиление античита PUBG на протяжении циклов патчей.
