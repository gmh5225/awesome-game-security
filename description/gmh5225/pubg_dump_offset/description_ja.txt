バージョン管理された PUBG メモリオフセットのダンプ集で、19.1 から 24.2 までの数十回のゲームアップデートをカバーし、各ヘッダーファイルには GObjects、GWorld、Xenuine 復号キー、FNameEntry、プレイヤー状態フィールド（Health、GroggyHealth、TeamNum）、カメラパラメータ、武器データ、ボーンインデックス、車両構造、アイテム／ルートオフセットなどのハードコードされたアドレスが含まれます。
バージョン別オフセットファイルは、暗号化ポインタベース（XenuineDecrypt）、スケルトンメッシュのボーン配列、武器の軌道／弾道データ、リプレイ／観戦検出フィールドなど、PUBG パッチ間の変更を追跡し、ゲームの UE4 メモリレイアウトが更新を通じてどう進化したかの履歴を提供します。
主に、UE4 オフセットの進化、Xenuine 暗号化方式の変化、およびパッチサイクルを通じた PUBG のアンチチート強化を研究するゲームセキュリティ研究者向けです。
