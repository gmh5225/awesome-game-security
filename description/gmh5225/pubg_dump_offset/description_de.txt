Eine versionierte Sammlung gedumpter PUBG-Speicher-Offsets, die Dutzende von Spielupdates von Version 19.1 bis 24.2 abdeckt; jede Header-Datei enthält fest kodierte Adressen für GObjects, GWorld, Xenuine-Entschlüsselungsschlüssel, FNameEntry, Spielerzustandsfelder (Health, GroggyHealth, TeamNum), Kameraparameter, Waffendaten, Knochenindizes, Fahrzeugstrukturen sowie Item-/Loot-Offsets.
Jede versionsbezogene Offset-Datei verfolgt Änderungen zwischen PUBG-Patches, darunter verschlüsselte Pointer-Basen (XenuineDecrypt), Bone-Arrays von Skeletal Meshes, Waffen-Trajektorie-/Ballistikdaten und Replay-/Zuschauer-Erkennungsfelder, und liefert so eine historische Aufzeichnung, wie sich das UE4-Speicherlayout des Spiels über Updates hinweg entwickelt hat.
Sie ist vor allem für Game-Security-Forscher nützlich, die die Evolution von UE4-Offsets, Änderungen am Xenuine-Verschlüsselungsschema und die Härtung des PUBG-Anti-Cheats über Patch-Zyklen hinweg untersuchen.
