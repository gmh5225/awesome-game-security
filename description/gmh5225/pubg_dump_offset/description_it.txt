Una raccolta versionata di offset di memoria di PUBG dumpatti che copre decine di aggiornamenti del gioco dalla versione 19.1 alla 24.2; ogni file header contiene indirizzi hardcoded per GObjects, GWorld, chiavi di decrittazione Xenuine, FNameEntry, campi di stato del giocatore (Health, GroggyHealth, TeamNum), parametri della camera, dati delle armi, indici delle bone, strutture dei veicoli e offset di oggetti/loot.
Ogni file di offset specifico per versione tiene traccia delle modifiche tra le patch di PUBG, incluse basi di puntatori crittografati (XenuineDecrypt), array di bone dei skeletal mesh, dati di traiettoria/balistica delle armi e campi di rilevamento replay/spettatore, fornendo una cronologia di come il layout di memoria UE4 del gioco si è evoluto tra gli aggiornamenti.
È principalmente utile per i ricercatori di sicurezza nei videogiochi che studiano l'evoluzione degli offset UE4, i cambiamenti dello schema di crittografia Xenuine e il rafforzamento dell'anti-cheat di PUBG nei cicli di patch.
