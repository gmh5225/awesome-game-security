Этот проект — PTFakeTouch, библиотека для iOS, позволяющая программно симулировать сенсорные события без взаимодействия пользователя. Она использует закрытые API IOKit или внутренности UIKit для внедрения синтетических touch-событий в конвейер событий iOS, обеспечивая автоматизированное взаимодействие с UI и симуляцию игрового ввода. Предназначена для разработчиков автоматизации iOS и создателей игровых ботов, симулирующих сенсорный ввод на устройствах iOS.
