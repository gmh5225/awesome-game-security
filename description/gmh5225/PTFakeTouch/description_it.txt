Questo progetto è PTFakeTouch, una libreria iOS per simulare eventi touch in modo programmatico senza interazione dell'utente. Usa API IOKit private o gli interni di UIKit per iniettare eventi touch sintetici nella pipeline degli eventi iOS, consentendo l'interazione UI automatizzata e la simulazione di input di gioco. È rivolta agli sviluppatori di automazione iOS e ai creatori di bot di gioco che simulano input touch su dispositivi iOS.
