Этот проект посвящён выводу информации о компиляторе, хранящейся в Rich Header PE-исполняемых файлов. Данные расположены между старым stub заголовка MZ (также называемым DOS stub) и заголовком PE. Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области инструментов читов / RE.
