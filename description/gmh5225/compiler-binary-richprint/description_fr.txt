Ce projet se concentre sur l'affichage des informations de compilateur stockées dans le Rich Header des exécutables PE. Ces données se situent entre l'ancien stub d'en-tête MZ (également appelé DOS stub) et l'en-tête PE. Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des outils de cheat / RE.
