Este proyecto se centra en imprimir la información del compilador almacenada en el Rich Header de ejecutables PE. Los datos se encuentran entre el stub antiguo del encabezado MZ (también llamado DOS stub) y el encabezado PE. Es especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de herramientas de cheat / RE.
