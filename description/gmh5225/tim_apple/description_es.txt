Este proyecto es un cheat o herramienta de CS2 (Counter-Strike 2) llamada Tim Apple que interactúa con el motor Source 2. Ofrece funciones de modificación del juego mediante interacción con el SDK de Source 2, pattern scanning o técnicas de acceso a memoria específicas de CS2. Está dirigido a investigadores de seguridad en videojuegos que estudian implementaciones de cheats de CS2.
