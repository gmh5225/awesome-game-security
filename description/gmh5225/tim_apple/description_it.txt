Questo progetto è un cheat o strumento per CS2 (Counter-Strike 2) chiamato Tim Apple che interagisce con il motore Source 2. Fornisce funzionalità di modifica del gioco tramite interazione con l'SDK di Source 2, pattern scanning o tecniche di accesso alla memoria specifiche per CS2. È rivolto ai ricercatori di sicurezza nei videogiochi che studiano le implementazioni di cheat per CS2.
