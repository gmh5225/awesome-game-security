Ce projet est un cheat ou outil CS2 (Counter-Strike 2) nommé Tim Apple qui interagit avec le moteur Source 2. Il fournit des fonctionnalités de modification de jeu via l'interaction avec le SDK Source 2, le pattern scanning ou des techniques d'accès mémoire spécifiques à CS2. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les implémentations de cheats CS2.
