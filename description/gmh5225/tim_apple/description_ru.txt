Этот проект — чит или инструмент для CS2 (Counter-Strike 2) под названием Tim Apple, взаимодействующий с движком Source 2. Он предоставляет возможности модификации игры через взаимодействие с Source 2 SDK, сканирование паттернов или методы доступа к памяти, специфичные для CS2. Предназначен для исследователей игровой безопасности, изучающих реализации читов для CS2.
