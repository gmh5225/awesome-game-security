Este proyecto es una integración entre IDA y VS Code que conecta ambos para ejecutar y depurar fácilmente scripts de IDAPython. IDACode aún se encuentra en una fase muy temprana y es de esperar que haya errores. Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats y plugins de IDA.
