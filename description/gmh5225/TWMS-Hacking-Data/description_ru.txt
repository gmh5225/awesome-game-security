Этот проект — собрание данных по взлому Taiwan MapleStory (TWMS), включая смещения памяти, структуры пакетов, ключи шифрования и техники обхода античит-защиты игры. В нём описаны детали клиент-серверного протокола, адреса памяти для манипуляции состоянием игры и методы обхода GameGuard или аналогичных защит, используемых TWMS. Материал предназначен для исследователей безопасности MapleStory, изучающих региональные различия в защите клиентов и манипуляции игрой на уровне пакетов.
