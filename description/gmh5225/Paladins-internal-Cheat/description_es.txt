Este proyecto se centra en un hack privado para Paladins que luego convertí para usar Mhyprot; no es necesario usar tu propio driver, aunque se recomienda. El overlay está pensado para ventanas sin bordes y no funciona en pantalla completa. Es útil principalmente para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / juego: Paladins.
