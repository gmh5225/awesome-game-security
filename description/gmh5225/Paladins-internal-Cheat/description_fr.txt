Ce projet porte sur un hack privé pour Paladins que j’ai ensuite adapté pour utiliser Mhyprot ; vous n’avez pas besoin d’utiliser votre propre pilote, mais cela est recommandé. L’overlay est conçu pour le mode fenêtré sans bordure et ne fonctionne pas en plein écran. Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / jeu : Paladins.
