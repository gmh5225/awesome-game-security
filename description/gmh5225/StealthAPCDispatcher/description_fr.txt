Ce projet est un outil noyau Windows qui démontre des techniques discrètes de répartition d'APC (Asynchronous Procedure Call). Il met en file d'attente des APC vers des threads cibles à l'aide de méthodes qui évitent la détection au niveau des API standard, permettant l'exécution de code dans des contextes de thread arbitraires sans déclencher la surveillance APC des anti-cheats. Il s'adresse aux chercheurs noyau qui étudient l'exécution de code via APC et le contournement de la détection APC des anti-cheats.
