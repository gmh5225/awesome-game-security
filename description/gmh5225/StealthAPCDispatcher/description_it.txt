Questo progetto è uno strumento kernel Windows che dimostra tecniche furtive di dispatch APC (Asynchronous Procedure Call). Accoda APC ai thread di destinazione con metodi che evitano il rilevamento a livello di API standard, consentendo l'esecuzione di codice in contesti di thread arbitrari senza attivare il monitoraggio APC degli anti-cheat. È rivolto ai ricercatori kernel che studiano l'esecuzione di codice basata su APC e l'elusione del rilevamento APC degli anti-cheat.
