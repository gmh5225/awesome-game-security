Dieses Projekt ist ein Windows-Kernel-Tool, das verdeckte APC-Dispatch-Techniken (Asynchronous Procedure Call) demonstriert. Es reiht APCs in Zielthreads ein und nutzt dabei Methoden, die die Erkennung auf Standard-API-Ebene umgehen, sodass Code in beliebigen Thread-Kontexten ausgeführt werden kann, ohne die APC-Überwachung von Anti-Cheat-Systemen auszulösen. Es richtet sich an Kernel-Forscher, die APC-basierte Codeausführung und die Umgehung der Anti-Cheat-APC-Erkennung untersuchen.
