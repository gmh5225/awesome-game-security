Этот проект — инструмент уровня ядра Windows, демонстрирующий скрытные методы диспетчеризации APC (Asynchronous Procedure Call). Он ставит APC в очередь целевых потоков способами, обходящими обнаружение на уровне стандартных API, что позволяет выполнять код в произвольных контекстах потоков, не вызывая мониторинга APC античитов. Предназначен для исследователей ядра, изучающих исполнение кода на основе APC и обход детектирования APC античитами.
