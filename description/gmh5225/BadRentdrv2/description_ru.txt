Этот проект использует уязвимый подписанный драйвер Rentdrv2.sys для доступа к памяти ядра в Windows. Через небезопасный интерфейс IOCTL драйвера Rentdrv2 обеспечивается произвольное чтение/запись физической памяти — для отображения драйверов, патчинга ядра или обхода античита. Предназначен для исследователей BYOVD, изучающих эксплуатацию драйвера Rentdrv2.
