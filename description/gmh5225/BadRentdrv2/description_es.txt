Este proyecto explota el controlador firmado vulnerable Rentdrv2.sys para acceder a la memoria del kernel en Windows. Usa la interfaz IOCTL insegura de Rentdrv2 para lograr lectura/escritura arbitraria de memoria física, útil para mapeo de controladores, parcheo del kernel o evasión de anti-cheat. Está orientado a investigadores BYOVD que estudian la explotación del controlador Rentdrv2.
