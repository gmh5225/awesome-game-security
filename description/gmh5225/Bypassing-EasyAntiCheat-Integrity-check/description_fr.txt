Une analyse technique et un contournement des contrôles d'intégrité du pilote en mode noyau d'EasyAntiCheat.
Montre comment EAC valide les sections de son pilote via les routines de notification CreateProcess et LoadImage, et fournit un outil de désobfuscation basé sur Capstone qui retire les instructions parasites pour révéler la logique sous-jacente du contrôle d'intégrité.
Inclut le code C++ reconstruit de la fonction de contrôle d'intégrité, illustrant la comparaison section par section avec une copie stockée du pilote.
