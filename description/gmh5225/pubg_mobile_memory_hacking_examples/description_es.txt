Este proyecto ofrece ejemplos de técnicas de hacking de memoria de PUBG Mobile en Android. Demuestra cómo leer datos de entidades del juego, coordenadas de jugadores, información de armas y el estado del juego desde la memoria del proceso de PUBG Mobile mediante métodos de acceso a memoria de Android. Los ejemplos cubren el escaneo básico de memoria y la modificación de valores para el hacking de juegos móviles. Está dirigido a investigadores de seguridad de juegos móviles que estudian el diseño de memoria y los mecanismos de protección de PUBG Mobile.
