Este proyecto demuestra técnicas para asignar páginas de memoria individuales en el kernel de Windows para una ejecución de código sigilosa. Asigna páginas de kernel aisladas mediante métodos no estándar para evitar el seguimiento de pool tags y la detección por escáneres de memoria. La técnica la usan drivers mapeados manualmente para reducir la visibilidad de su huella de memoria. Está dirigido a investigadores de kernel que estudian la asignación sigilosa de memoria y técnicas antiforenses.
