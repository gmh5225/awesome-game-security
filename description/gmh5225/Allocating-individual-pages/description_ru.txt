Этот проект демонстрирует техники выделения отдельных страниц памяти в ядре Windows для скрытного выполнения кода. Он выделяет изолированные страницы ядра нестандартными методами, чтобы обойти отслеживание pool tag и обнаружение сканерами памяти. Технику используют вручную отображённые драйверы, чтобы снизить видимость своего следа в памяти. Материал предназначен для исследователей ядра, изучающих скрытое выделение памяти и антикриминалистические техники.
