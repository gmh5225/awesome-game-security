Ce projet présente des techniques d'allocation de pages mémoire individuelles dans le noyau Windows pour une exécution de code furtive. Il alloue des pages noyau isolées par des méthodes non standard afin d'éviter le suivi des pool tags et la détection par les scanners mémoire. La technique est utilisée par les pilotes mappés manuellement pour réduire la visibilité de leur empreinte mémoire. Elle s'adresse aux chercheurs noyau qui étudient l'allocation mémoire furtive et les techniques anti-forensiques.
