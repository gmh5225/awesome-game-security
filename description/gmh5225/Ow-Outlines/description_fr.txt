Une DLL interne pour Overwatch qui active un ESP de lueur/contour des ennemis en patchant un décalage mémoire spécifique (GlowESP à 0xE22510) relatif à l'adresse de base de Overwatch.exe, injectée dans le processus du jeu sous forme de DLL.
Le module résout l'adresse de base via GetModuleHandleA("Overwatch.exe"), utilise un mécanisme VEH (Vectored Exception Handler) défini dans USEVEH.h, et écrit directement les valeurs de configuration de lueur dans les structures de données de rendu des contours du jeu.
Il est principalement utile aux chercheurs en sécurité des jeux qui étudient la manipulation mémoire interne pour les fonctionnalités ESP visuelles et l'exploitation du rendu des contours dans Overwatch.
