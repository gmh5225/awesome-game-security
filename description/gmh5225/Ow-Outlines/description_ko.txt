Overwatch.exe 베이스 주소 기준의 특정 메모리 오프셋(GlowESP, 0xE22510)을 패치하여 적 glow/outline ESP를 활성화하는 Overwatch 내부 DLL로, 게임 프로세스에 DLL로 주입됩니다.
이 모듈은 GetModuleHandleA("Overwatch.exe")로 베이스 주소를 확인하고, USEVEH.h에 정의된 VEH(Vectored Exception Handler) 메커니즘을 사용하며, glow 설정 값을 게임의 outline 렌더링 데이터 구조에 직접 기록합니다.
주로 Overwatch에서 시각적 ESP 기능과 outline 렌더링 악용을 위한 내부 게임 메모리 조작을 연구하는 게임 보안 연구자에게 유용합니다.
