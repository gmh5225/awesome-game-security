PUBG Mobile(버전 1.5 및 1.9)용으로 덤프된 SDK 및 오프셋 모음으로, 게임의 리플렉션 시스템에서 추출한 전체 UE4 클래스 계층 정의, 멤버 오프셋, 함수 주소, 타입 정보를 포함합니다.
SDK 파일은 World, Level, PlayerController, Character, PrimitiveComponent, SkeletalMeshComponent 및 무기/차량 유형을 포함한 수백 개의 UE4 클래스를 열거하며, ARM32 모바일 빌드에 대한 정확한 바이트 오프셋, 비트마스크 필드, 가상 함수 포인터를 제공하고 렌더링, 물리, 네트워킹, 게임플레이 서브시스템을 다룹니다.
주로 PUBG Mobile의 UE4 메모리 레이아웃을 연구하거나, 외부/내부 치트를 구축하거나, Android 안티치트 커버리지를 분석하는 모바일 게임 보안 연구자에게 유용합니다.
