PUBG Mobile（バージョン 1.5 および 1.9）向けにダンプされた SDK とオフセット集であり、ゲームのリフレクションシステムから抽出した完全な UE4 クラス階層定義、メンバーオフセット、関数アドレス、型情報を含みます。
SDK ファイルには World、Level、PlayerController、Character、PrimitiveComponent、SkeletalMeshComponent、および武器/車両系など数百の UE4 クラスが列挙され、ARM32 モバイルビルド向けの正確なバイトオフセット、ビットマスクフィールド、仮想関数ポインタが記載され、描画、物理、ネットワーク、ゲームプレイ各サブシステムをカバーします。
PUBG Mobile の UE4 メモリレイアウトの研究、外部/内部チートの構築、または Android 上のアンチチートカバレッジ分析を行うモバイルゲームセキュリティ研究者に主に役立ちます。
