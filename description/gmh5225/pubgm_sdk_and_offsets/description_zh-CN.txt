针对 PUBG Mobile（1.5 与 1.9 版本）的 SDK 转储与偏移量合集，包含从游戏反射系统提取的完整 UE4 类层次定义、成员偏移、函数地址与类型信息。
SDK 文件枚举了数百个 UE4 类，包括 World、Level、PlayerController、Character、PrimitiveComponent、SkeletalMeshComponent 以及武器/载具类型，并给出 ARM32 移动端构建中精确的字节偏移、位掩码字段与虚函数指针，覆盖渲染、物理、网络与玩法子系统。
主要面向研究 PUBG Mobile 的 UE4 内存布局、编写外部/内部作弊程序，或分析 Android 平台反作弊覆盖范围的移动游戏安全研究人员。
