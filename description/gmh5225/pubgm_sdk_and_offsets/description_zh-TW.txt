針對 PUBG Mobile（1.5 與 1.9 版本）的 SDK 傾印與偏移量合集，包含從遊戲反射系統提取的完整 UE4 類別階層定義、成員偏移、函式位址與型別資訊。
SDK 檔案列舉了數百個 UE4 類別，包括 World、Level、PlayerController、Character、PrimitiveComponent、SkeletalMeshComponent 以及武器/載具類型，並給出 ARM32 行動端建置中精確的位元組偏移、位元遮罩欄位與虛擬函式指標，涵蓋算繪、物理、網路與玩法子系統。
主要面向研究 PUBG Mobile 的 UE4 記憶體配置、撰寫外部/內部作弊程式，或分析 Android 平台反作弊涵蓋範圍的行動遊戲安全研究人員。
