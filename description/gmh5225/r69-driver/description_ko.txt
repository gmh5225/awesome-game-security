이 프로젝트는 기존의 디바이스 객체 대신 HalPrivateDispatchTable 훅을 통해 읽기, 쓰기, 프로세스 조회 요청을 라우팅하는 소형 커널 통신 라이브러리입니다.
드라이버는 HalTimerQueryAuxiliaryCounterFrequency를 호출자 trap frame에서 c_packet을 꺼내고, read_process_memory, write_process_memory, query_process_data 요청을 디코딩한 뒤, 대상 프로세스 CR3를 기준으로 가상 주소를 물리 페이지로 변환해 처리하는 핸들러로 교체합니다.
HalClearLastBranchRecordStack에 걸린 두 번째 훅은 현재 CR3로 DirectoryTableBase를 갱신하며, 사용자 모드 래퍼는 NtQueryAuxiliaryCounterFrequency를 감싼 c_r69 클래스로 이 경로를 노출하고 프로세스 이름으로 대상에 연결할 수 있습니다.
표준 IOCTL 인터페이스 없이 시스템 호출에 인접한 통신, HalPrivateDispatchTable 남용, 물리 메모리 기반 프로세스 접근을 연구하는 Windows 커널 연구자에게 주로 유용합니다.
