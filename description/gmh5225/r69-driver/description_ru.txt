Этот проект — компактная библиотека ядра для обмена данными, которая маршрутизирует запросы чтения, записи и опроса процессов через хуки HalPrivateDispatchTable вместо обычного объекта устройства.
Драйвер заменяет HalTimerQueryAuxiliaryCounterFrequency обработчиком, который извлекает c_packet из trap frame вызывающей стороны, декодирует запросы read_process_memory, write_process_memory и query_process_data и обслуживает их, переводя целевые виртуальные адреса в физические страницы по CR3 процесса.
Второй хук на HalClearLastBranchRecordStack обновляет DirectoryTableBase из текущего CR3, а пользовательская обёртка открывает этот путь через класс c_r69, который оборачивает NtQueryAuxiliaryCounterFrequency и может подключаться к целевому процессу по имени.
В основном полезен исследователям ядра Windows, изучающим коммуникацию рядом с syscall, злоупотребление HalPrivateDispatchTable и доступ к процессам через физическую память без стандартного интерфейса IOCTL.
