Ce projet rassemble de nombreux binaires win32k.sys versionnés issus des builds Windows 10 et 11 pour une comparaison hors ligne.
Il s’agit d’un corpus binaire pour la recherche sur le sous-système GUI du noyau, et non d’un analyseur prêt à l’emploi.
Il est surtout utile aux reverse engineers du noyau Windows qui étudient les évolutions de win32k, les syscalls et les mécanismes internes du chemin d’entrée.
