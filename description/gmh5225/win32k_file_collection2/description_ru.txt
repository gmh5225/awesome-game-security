Этот проект собирает множество версионированных бинарников win32k.sys из сборок Windows 10 и 11 для офлайн-сравнения.
Это бинарный корпус для исследования GUI-подсистемы ядра, а не готовый анализатор.
Он в первую очередь полезен reverse-инженерам ядра Windows, изучающим изменения win32k, системные вызовы и внутреннее устройство пути ввода.
