Este proyecto recopila numerosos binarios win32k.sys versionados de builds de Windows 10 y 11 para su comparación sin conexión.
Es un corpus binario para la investigación del subsistema GUI del kernel, no un analizador listo para usar.
Resulta especialmente útil para ingenieros inversos del kernel de Windows que estudian los cambios de win32k, las syscalls y los internos de la ruta de entrada.
