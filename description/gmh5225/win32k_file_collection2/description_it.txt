Questo progetto raccoglie numerosi binari win32k.sys versionati relativi a build di Windows 10 e 11 per il confronto offline.
È un corpus binario per la ricerca sul sottosistema GUI del kernel, non un analizzatore pronto all’uso.
È soprattutto utile agli reverse engineer del kernel Windows che studiano le modifiche a win32k, le syscall e gli interni del percorso di input.
