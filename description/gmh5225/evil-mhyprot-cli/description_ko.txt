이 프로젝트는 Genshin Impact의 mhyprot2.sys 드라이버를 악용하여 사용자 모드에서 커널 권한 메모리 작업을 수행하는 명령줄 PoC입니다.
보관된 README는 해당 버그를 MmCopyVirtualMemory 및 memcpy와 유사한 동작을 노출하는 취약한 IOCTL 경로로 설명하며, 서비스가 실행되면 커널과 사용자 메모리를 임의로 읽고 쓸 수 있게 됩니다.
이 CLI는 대상 프로세스를 찾고, mhyprot::init에서 취약한 서비스와 장치를 초기화한 뒤, 작은 드라이버 추상화를 통해 테스트나 유틸리티 작업을 디스패치하므로, 단순한 글이 아니라 문제를 실험할 수 있는 실용적인 프론트엔드가 됩니다.
주로 취약한 안티치트 드라이버, ring3에서 ring0으로의 메모리 프리미티브, 서비스 기반 BYOVD 워크플로를 연구하는 Windows 보안 연구자에게 유용합니다.
