本プロジェクトは、Genshin Impact の mhyprot2.sys ドライバを悪用し、ユーザモードからカーネル特権のメモリ操作を行うためのコマンドライン PoC です。
アーカイブされた README では、この不具合を、MmCopyVirtualMemory や memcpy に似た挙動を露出する脆弱な IOCTL パスとして説明しており、サービス稼働後はカーネルおよびユーザメモリへの任意読み書きが可能になります。
この CLI は対象プロセスを見つけ、mhyprot::init で脆弱なサービスとデバイスを初期化し、小さなドライバ抽象を通じてテストやユーティリティ操作をディスパッチするため、単なる解説文ではなく、問題を実験するための実用的なフロントエンドになります。
主に、脆弱なアンチチートドライバ、ring3 から ring0 へのメモリプリミティブ、サービスベースの BYOVD ワークフローを研究する Windows セキュリティ研究者向けです。
