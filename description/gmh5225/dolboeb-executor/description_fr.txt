Ce projet est un exécuteur de code en mode noyau qui utilise un pilote signé vulnérable pour exécuter du code arbitraire dans l'espace noyau. Il charge et exploite l'interface IOCTL du pilote vulnérable afin d'exécuter un shellcode noyau personnalisé ou d'appeler des fonctions noyau arbitraires depuis le mode utilisateur. Il s'adresse aux chercheurs en exploitation noyau qui étudient les techniques d'exécution de code noyau basées sur BYOVD.
