Dieses Projekt ist ein Kernel-Mode-Code-Executor, der einen anfälligen signierten Treiber nutzt, um beliebigen Code im Kernel-Space auszuführen. Es lädt die IOCTL-Schnittstelle des anfälligen Treibers und nutzt sie aus, um benutzerdefinierten Kernel-Shellcode auszuführen oder beliebige Kernelfunktionen aus dem User-Mode aufzurufen. Es richtet sich an Kernel-Exploitation-Forscher, die BYOVD-basierte Techniken zur Kernel-Codeausführung untersuchen.
