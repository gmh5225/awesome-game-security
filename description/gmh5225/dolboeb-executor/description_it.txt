Questo progetto è un esecutore di codice in modalità kernel che usa un driver firmato vulnerabile per eseguire codice arbitrario nello spazio kernel. Carica e sfrutta l'interfaccia IOCTL del driver vulnerabile per eseguire shellcode kernel personalizzato o chiamare funzioni kernel arbitrarie dalla modalità utente. È rivolto ai ricercatori di exploitation del kernel che studiano le tecniche di esecuzione di codice kernel basate su BYOVD.
