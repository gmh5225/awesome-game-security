Этот проект — исполнитель кода в режиме ядра, который использует уязвимый подписанный драйвер для выполнения произвольного кода в пространстве ядра. Он загружает и эксплуатирует интерфейс IOCTL уязвимого драйвера, чтобы запускать пользовательский kernel shellcode или вызывать произвольные функции ядра из пользовательского режима. Предназначен для исследователей эксплуатации ядра, изучающих техники выполнения кода в ядре на основе BYOVD.
