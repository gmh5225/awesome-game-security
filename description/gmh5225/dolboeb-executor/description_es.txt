Este proyecto es un ejecutor de código en modo kernel que usa un controlador firmado vulnerable para ejecutar código arbitrario en el espacio del kernel. Carga y explota la interfaz IOCTL del controlador vulnerable para ejecutar shellcode de kernel personalizado o llamar a funciones arbitrarias del kernel desde el modo de usuario. Está dirigido a investigadores de explotación de kernel que estudian técnicas de ejecución de código en kernel basadas en BYOVD.
