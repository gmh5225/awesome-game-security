Este proyecto es un framework de trampas para juegos basado en un controlador de núcleo, que usa un controlador de Windows personalizado para el acceso privilegiado a la memoria de los procesos del juego. Implementa lectura/escritura de memoria mediante traducción de direcciones físicas o mapeo MDL, eludiendo las protecciones de handles de los anti-cheat. El controlador ofrece una interfaz de comunicación en modo usuario para que las aplicaciones de trampas accedan a la memoria del juego. Está orientado a investigadores de núcleo que estudian arquitecturas de trampas basadas en controladores y la detección anti-cheat a nivel de núcleo.
