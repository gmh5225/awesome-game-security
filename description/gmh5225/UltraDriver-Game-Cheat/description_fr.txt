Ce projet est un framework de triche de jeu basé sur un pilote noyau, qui utilise un pilote Windows personnalisé pour un accès mémoire privilégié aux processus de jeu. Il implémente la lecture/écriture mémoire via la traduction d'adresses physiques ou le mappage MDL, contournant les protections de handles des anti-cheats. Le pilote fournit une interface de communication en mode utilisateur permettant aux applications de triche d'accéder à la mémoire du jeu. Il s'adresse aux chercheurs noyau qui étudient les architectures de triche basées sur des pilotes et la détection anti-cheat au niveau noyau.
