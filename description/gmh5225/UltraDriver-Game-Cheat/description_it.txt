Questo progetto è un framework di cheat per giochi basato su driver kernel, che usa un driver Windows personalizzato per l'accesso privilegiato alla memoria dei processi di gioco. Implementa lettura/scrittura della memoria tramite traduzione di indirizzi fisici o mapping MDL, aggirando le protezioni sui handle degli anti-cheat. Il driver fornisce un'interfaccia di comunicazione in user mode affinché le applicazioni di cheat possano accedere alla memoria del gioco. È rivolto ai ricercatori kernel che studiano architetture di cheat basate su driver e il rilevamento anti-cheat a livello kernel.
