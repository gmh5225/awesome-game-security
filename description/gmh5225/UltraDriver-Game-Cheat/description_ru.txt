Этот проект — фреймворк игровых читов на основе ядерного драйвера, использующий пользовательский драйвер Windows для привилегированного доступа к памяти игровых процессов. Он реализует чтение/запись памяти через трансляцию физических адресов или отображение MDL, обходя защиту дескрипторов античитов. Драйвер предоставляет интерфейс связи в пользовательском режиме для чит-приложений, обращающихся к памяти игры. Предназначен для исследователей ядра, изучающих архитектуры читов на драйверах и обнаружение античитов на уровне ядра.
