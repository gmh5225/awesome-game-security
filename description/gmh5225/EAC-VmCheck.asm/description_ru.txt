Этот проект — небольшая вырезка ассемблерного кода обнаружения виртуальных машин из easyanticheat.sys.
Архивный README сообщает, что код взят из каталога vm драйвера, а входящий vmCheck.asm показывает процедуру CheckVM, вызывающую небольшой вспомогательный ExecVMREAD, который выполняет VMREAD и ветвится по результату.
Поскольку репозиторий содержит лишь восстановленный файл ассемблера, а не полноценный фреймворк, его ценность — в сохранении точного низкоуровневого потока управления логики детекта ВМ в EAC для повторного анализа.
Он в основном полезен реверс-инженерам, изучающим проверки виртуализации Easy Anti-Cheat, зондирование на основе VMREAD и то, как EAC различает пути «ВМ найдена» и «ВМ не найдена».
