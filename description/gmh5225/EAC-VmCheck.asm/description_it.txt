Questo progetto è una minuscola estrazione dell'assembly di rilevamento delle macchine virtuali da easyanticheat.sys.
Il README archiviato indica che è stato preso dalla directory vm del driver, e il vmCheck.asm incluso mostra una routine CheckVM che chiama un piccolo helper ExecVMREAD che emette VMREAD e ramifica in base al risultato.
Poiché il repository è solo il file assembly recuperato e non un framework completo, il suo valore sta nel preservare l'esatto flusso di controllo a basso livello della logica di rilevamento VM di EAC per la rianalisi.
È soprattutto utile per reverse engineer che studiano i controlli di virtualizzazione di Easy Anti-Cheat, il probing basato su VMREAD e come EAC distingue i percorsi VM trovata e VM non trovata.
