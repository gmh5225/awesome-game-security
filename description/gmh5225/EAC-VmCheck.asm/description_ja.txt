本プロジェクトは、easyanticheat.sys から抜き出した仮想マシン検出用アセンブリのごく小さな抽出物です。
アーカイブされた README によればドライバの vm ディレクトリから取得されたもので、同梱の vmCheck.asm では CheckVM ルーチンが小さな ExecVMREAD ヘルパーを呼び出し、VMREAD を発行して結果に応じて分岐する様子が示されています。
このリポジトリは完全なフレームワークではなく復元されたアセンブリファイルのみであるため、その価値は EAC の VM 検出ロジックの正確な低レベル制御フローを再解析用に保存している点にあります。
主に、Easy Anti-Cheat の仮想化チェック、VMREAD ベースのプロービング、および EAC が VM 検出あり／なしのパスをどう区別するかを研究するリバースエンジニア向けです。
