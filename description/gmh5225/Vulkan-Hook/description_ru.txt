Этот проект — фреймворк для хукинга Vulkan API, перехватывающий вызовы рендеринга Vulkan для отрисовки оверлеев в играх на Vulkan. Он хукает vkQueuePresentKHR и другие функции Vulkan, внедряя пользовательские команды рендеринга и тем самым позволяя показывать меню ImGui и рисовать ESP в Vulkan-играх. Реализация на C++ демонстрирует специфичные для Vulkan паттерны хукинга, отличные от подходов DirectX. Проект предназначен для исследователей игровой безопасности, изучающих хукинг конвейера рендеринга Vulkan и техники оверлеев.
