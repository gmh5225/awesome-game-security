本プロジェクトは、Vulkan ベースのゲーム内でオーバーレイ描画を行うために Vulkan レンダリング呼び出しをインターセプトする Vulkan API フッキングフレームワークです。vkQueuePresentKHR およびその他の Vulkan 関数をフックしてカスタム描画コマンドを注入し、Vulkan ゲーム内で ImGui オーバーレイメニューや ESP 描画を可能にします。この C++ 実装は、DirectX とは異なる Vulkan 固有のフッキングパターンを示しています。Vulkan レンダリングパイプラインのフックとオーバーレイ技術を研究するゲームセキュリティ研究者を対象としています。
