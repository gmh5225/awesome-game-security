Questo progetto è un framework di hooking dell'API Vulkan che intercetta le chiamate di rendering Vulkan per il rendering di overlay nei giochi basati su Vulkan. Aggancia vkQueuePresentKHR e altre funzioni Vulkan per iniettare comandi di rendering personalizzati, consentendo menu overlay ImGui e disegno ESP nei giochi Vulkan. L'implementazione in C++ illustra pattern di hooking specifici di Vulkan, distinti dagli approcci DirectX. È rivolto ai ricercatori di sicurezza nei videogiochi che studiano l'hooking della pipeline di rendering Vulkan e le tecniche di overlay.
