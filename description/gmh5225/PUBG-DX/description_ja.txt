DirectX 11 をフックして ImGui ベースのオーバーレイメニューを描画する内部 PUBG チート DLL で、ESP（プレイヤーボックス、ゲーム内全武器のアイコンを埋め込み PNG テクスチャとして表示）、エイムボット、アイテム ESP を備え、VMProtect SDK 連携で保護されています。
この DLL はカーネルドライバ通信（drive.h）でメモリを読み取り、PUBG の Xenuine 保護ポインタを復号し（decrypt.h）、UE4 の GObjects/GWorld/FNameEntry 配列を解決し、戻りアドレス偽装（SpoofCall.asm）によりアンチチートのスタックウォークから呼び出し元を隠します。
主に、内部 DX11 フックベースのチート描画、Xenuine ポインタ復号、UE4 ゲームチートにおける VMProtect 統合を研究するゲームセキュリティ研究者向けです。
