Внутренняя чит-DLL для PUBG, которая перехватывает DirectX 11 для отрисовки оверлей-меню на ImGui с ESP (рамки игроков, иконки оружия для каждого внутриигрового оружия как встроенные PNG-текстуры), аимботом и ESP предметов, защищённая интеграцией SDK VMProtect.
DLL использует обмен с драйвером ядра (drive.h) для чтения памяти, расшифровывает указатели PUBG, защищённые Xenuine (decrypt.h), разрешает массивы UE4 GObjects/GWorld/FNameEntry и реализует подмену адреса возврата (SpoofCall.asm), чтобы скрыть происхождение вызовов от обхода стека античита.
Она в основном полезна исследователям игровой безопасности, изучающим отрисовку внутренних читов на хуках DX11, расшифровку указателей Xenuine и интеграцию VMProtect в читах для игр на UE4.
