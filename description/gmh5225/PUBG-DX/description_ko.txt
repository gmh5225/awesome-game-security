DirectX 11을 후킹하여 ImGui 기반 오버레이 메뉴를 렌더링하는 내부 PUBG 치트 DLL로, ESP(플레이어 박스, 게임 내 모든 무기 아이콘을 임베디드 PNG 텍스처로 표시), 에임봇, 아이템 ESP를 제공하며 VMProtect SDK 연동으로 보호됩니다.
이 DLL은 커널 드라이버 통신(drive.h)으로 메모리를 읽고, PUBG의 Xenuine 보호 포인터를 복호화하며(decrypt.h), UE4의 GObjects/GWorld/FNameEntry 배열을 해석하고, 반환 주소 스푸핑(SpoofCall.asm)으로 안티치트의 스택 워킹으로부터 호출 출처를 숨깁니다.
주로 내부 DX11 후킹 기반 치트 렌더링, Xenuine 포인터 복호화, UE4 게임 치트에서의 VMProtect 통합을 연구하는 게임 보안 연구자에게 유용합니다.
