Este proyecto es Frinet, un complemento de IDA Pro que conecta la instrumentación dinámica de Frida con el análisis estático de IDA. Permite ejecutar scripts de Frida en un proceso objetivo mientras se navega por el código correspondiente en IDA, sincronizando información de tiempo de ejecución como valores de memoria, argumentos de funciones y valores de retorno de vuelta a la vista de análisis de IDA. Está dirigido a ingenieros inversos que combinan el análisis dinámico de Frida con las capacidades de análisis estático de IDA.
