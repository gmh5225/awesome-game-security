Этот проект — Frinet, плагин для IDA Pro, который связывает динамическую инструментацию Frida со статическим анализом IDA. Он позволяет запускать скрипты Frida в целевом процессе, одновременно просматривая соответствующий код в IDA, и синхронизировать информацию времени выполнения — значения памяти, аргументы функций и возвращаемые значения — обратно в представление анализа IDA. Предназначен для инженеров по реверс-инжинирингу, сочетающих динамический анализ Frida со статическими возможностями IDA.
