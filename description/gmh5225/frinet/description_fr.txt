Ce projet est Frinet, un plugin IDA Pro qui relie l'instrumentation dynamique Frida à l'analyse statique d'IDA. Il permet d'exécuter des scripts Frida sur un processus cible tout en naviguant dans le code correspondant dans IDA, en synchronisant les informations d'exécution telles que les valeurs mémoire, les arguments de fonctions et les valeurs de retour vers la vue d'analyse d'IDA. Il s'adresse aux reverse engineers qui combinent l'analyse dynamique de Frida avec les capacités d'analyse statique d'IDA.
