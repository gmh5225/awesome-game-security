Dieses Projekt ist Frinet, ein IDA-Pro-Plugin, das Fridas dynamische Instrumentierung mit IDAs statischer Analyse verbindet. Es ermöglicht das Ausführen von Frida-Skripten auf einem Zielprozess, während man den entsprechenden Code in IDA navigiert, und synchronisiert Laufzeitinformationen wie Speicherwerte, Funktionsargumente und Rückgabewerte zurück in IDAs Analyseansicht. Es richtet sich an Reverse Engineers, die Fridas dynamische Analyse mit IDAs statischen Analysefähigkeiten kombinieren.
