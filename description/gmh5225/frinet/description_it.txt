Questo progetto è Frinet, un plugin di IDA Pro che collega l'instrumentazione dinamica di Frida con l'analisi statica di IDA. Consente di eseguire script Frida su un processo target mentre si naviga nel codice corrispondente in IDA, sincronizzando informazioni di runtime come valori di memoria, argomenti delle funzioni e valori di ritorno nella vista di analisi di IDA. È pensato per reverse engineer che combinano l'analisi dinamica di Frida con le capacità di analisi statica di IDA.
