Этот проект — эмулятор античита BattlEye, который имитирует клиентское поведение BattlEye без запуска настоящего античита. Он эмулирует протокол связи BattlEye, ответы heartbeat и интерфейс загрузки модулей, позволяя играм работать без активной защиты BattlEye. Инструмент используется для анализа взаимодействия игр с BattlEye. Он предназначен для исследователей античита, изучающих протокол и механизмы интеграции BattlEye.
