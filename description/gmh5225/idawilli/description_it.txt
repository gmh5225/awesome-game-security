Questo progetto è idawilli, una raccolta di script, plugin e utilità per IDA Pro di Willi Ballenthin. Include script Python per attività comuni di reverse engineering come analisi delle funzioni, decifratura di stringhe, annotazione di strutture, navigazione dei riferimenti incrociati e automazione dei flussi di lavoro in IDA Pro. Gli script coprono vari scenari di analisi, dal malware al firmware. È rivolto agli utenti di IDA Pro che cercano script di analisi pronti all'uso e miglioramenti del workflow.
