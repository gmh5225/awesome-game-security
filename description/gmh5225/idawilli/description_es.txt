Este proyecto es idawilli, una colección de scripts, plugins y utilidades de IDA Pro de Willi Ballenthin. Incluye scripts en Python para tareas habituales de ingeniería inversa como análisis de funciones, descifrado de cadenas, anotación de estructuras, navegación de referencias cruzadas y automatización de flujos de trabajo en IDA Pro. Los scripts cubren diversos escenarios de análisis, desde malware hasta firmware. Está dirigido a usuarios de IDA Pro que buscan scripts de análisis listos para usar y mejoras de flujo de trabajo.
