Этот проект — idawilli, набор скриптов, плагинов и утилит для IDA Pro от Willi Ballenthin. Он включает Python-скрипты для типичных задач реверс-инжиниринга: анализ функций, расшифровка строк, аннотация структур, навигация по перекрёстным ссылкам и автоматизация рабочих процессов в IDA Pro. Скрипты охватывают разные сценарии анализа — от вредоносного ПО до прошивок. Проект предназначен для пользователей IDA Pro, которым нужны готовые скрипты анализа и улучшения workflow.
