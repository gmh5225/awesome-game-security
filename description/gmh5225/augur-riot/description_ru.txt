Этот проект — исследовательский инструмент для анализа античит-системы Vanguard от Riot Games. Он документирует поведение драйвера ядра Vanguard, механизмы обнаружения и защиты на уровне системы посредством анализа с помощью реверс-инжиниринга. Инструмент может включать утилиты для мониторинга активности драйвера Vanguard, перечисления его callback'ов или журналирования операций ядра. Он предназначен для исследователей античита, изучающих архитектуру защиты Vanguard на уровне ядра и методологию обнаружения.
