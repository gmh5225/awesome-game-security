Este proyecto es una herramienta de investigación para analizar el sistema anticheat Vanguard de Riot Games. Documenta el comportamiento del controlador de kernel de Vanguard, sus mecanismos de detección y las protecciones a nivel de sistema mediante análisis de ingeniería inversa. La herramienta puede incluir utilidades para monitorizar la actividad del controlador de Vanguard, enumerar sus callbacks o registrar sus operaciones de kernel. Está dirigida a investigadores de anticheat que estudian la arquitectura de protección a nivel de kernel de Vanguard y su metodología de detección.
