Questo progetto è AzureV316, un emulatore di server privato MapleStory v316 che implementa la logica del game server per la versione v316. Gestisce l’autenticazione di login, la persistenza dei dati del personaggio, le istanze delle mappe, lo spawning dei mostri, l’elaborazione delle skill, la progressione delle quest e i sistemi di party/gilda specifici di questa versione di MapleStory. È rivolto agli sviluppatori di server privati MapleStory e ai ricercatori che studiano le implementazioni di server MMO.
