Questo progetto mostra come usare il Vectored Exception Handling (VEH) di Windows per agganciare printf e funzioni di output simili. Imposta page guard sulla memoria delle funzioni target, cattura le relative eccezioni in un gestore VEH e reindirizza l'esecuzione a codice personalizzato di logging o filtraggio. Questa tecnica di hooking non invasiva evita di patchare i byte delle funzioni. È rivolta ai ricercatori di sicurezza che studiano tecniche di intercettazione di funzioni basate su VEH.
