Ce projet montre comment utiliser le Vectored Exception Handling (VEH) de Windows pour accrocher printf et des fonctions de sortie similaires. Il place des page guards sur la mémoire des fonctions cibles, capture les exceptions résultantes dans un gestionnaire VEH, puis redirige l'exécution vers un code personnalisé de journalisation ou de filtrage. Cette technique d'accrochage non invasive évite de modifier les octets des fonctions. Elle s'adresse aux chercheurs en sécurité qui étudient l'interception de fonctions basée sur VEH.
