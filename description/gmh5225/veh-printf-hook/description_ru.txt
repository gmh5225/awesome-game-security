Этот проект демонстрирует использование Vectored Exception Handling (VEH) в Windows для перехвата printf и подобных функций вывода. На память целевых функций устанавливаются page guard, возникающие исключения обрабатываются в VEH-обработчике, а выполнение перенаправляется в пользовательский код журналирования или фильтрации. Эта неинвазивная техника перехвата не требует патчинга байтов функций. Предназначена для исследователей безопасности, изучающих перехват функций на основе VEH.
