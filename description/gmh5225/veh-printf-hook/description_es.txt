Este proyecto demuestra cómo usar el Vectored Exception Handling (VEH) de Windows para enganchar printf y funciones de salida similares. Coloca page guards en la memoria de las funciones objetivo, captura las excepciones resultantes en un manejador VEH y redirige la ejecución a código personalizado de registro o filtrado. Esta técnica de hooking no invasiva evita parchear los bytes de las funciones. Está dirigida a investigadores de seguridad que estudian técnicas de interceptación de funciones basadas en VEH.
