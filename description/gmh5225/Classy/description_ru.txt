Плагин для IDA Pro для управления классами C++, таблицами виртуальных функций (vtable) и сигнатурами функций при анализе обратной инженерии.
Поддерживает генерацию vtable из выбранных диапазонов, назначение функций классам, Itanium name mangling, сопоставление со структурами IDA и генерацию заголовочных файлов C через графический интерфейс на PyQt5.
