Este proyecto es un kernel de Android personalizado para el Samsung Galaxy A14 (A146B) con soporte KernelSU integrado. Proporciona un árbol de fuentes del kernel modificado con parches de KernelSU para la gestión del acceso root en este dispositivo Samsung concreto. Está orientado a desarrolladores de Android y creadores de ROM personalizadas que apuntan a dispositivos Samsung A14 con root KernelSU.
