Этот проект — кастомное ядро Android для Samsung Galaxy A14 (A146B) со встроенной поддержкой KernelSU. Он предоставляет модифицированное дерево исходников ядра с патчами KernelSU для управления root-доступом на этом конкретном устройстве Samsung. Предназначен для Android-разработчиков и сборщиков кастомных ROM, ориентированных на устройства Samsung A14 с root через KernelSU.
