Technique de communication en mode noyau utilisant le callback de registre CmRegisterCallback avec un gadget JMP RCX.
Localise une instruction JMP RCX dans nvraid.sys et l'enregistre comme callback de registre, détournant le mécanisme de callback pour rediriger l'exécution vers un gestionnaire personnalisé afin d'assurer une communication noyau discrète.
