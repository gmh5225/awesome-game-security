Хук режима ядра для KdTrap — глобального обработчика исключений Windows, вызываемого первым при любом исключении.
Изменяет HalpStallCounter и связанные переменные для перехвата потока управления, что позволяет использовать пользовательский обработчик исключений, безопасно перехватывающий разыменования нулевых указателей и сбои по зарезервированным битам CR3.
