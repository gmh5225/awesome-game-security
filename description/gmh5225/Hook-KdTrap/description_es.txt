Un hook en modo kernel para KdTrap, el manejador global de excepciones de Windows que se invoca primero cuando ocurre cualquier excepción.
Modifica HalpStallCounter y variables relacionadas para secuestrar el flujo de control, habilitando un manejador de excepciones personalizado que intercepta de forma segura las desreferencias de puntero nulo y las faltas por bits reservados de CR3.
