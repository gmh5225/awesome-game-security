Рефлексивный загрузчик драйверов ядра, обходящий Windows Driver Signature Enforcement (DSE) с помощью пользовательского PE-загрузчика.
По умолчанию использует руткит Capcom.sys для загрузки неподписанных драйверов; модульная архитектура поддерживает и другие известные эксплуатируемые подписанные драйверы.
