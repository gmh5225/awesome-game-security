File manager GUI per Windows scritto in C#/.NET che esplora, copia e trasferisce file su dispositivi Android tramite il protocollo ADB, offrendo trasferimenti più veloci rispetto all’MTP standard, un’interfaccia a doppio riquadro in stile Esplora risorse e supporto multilingue (inglese, tedesco, spagnolo, giapponese, polacco, ceco).
