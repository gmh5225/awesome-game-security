Этот проект — репозиторий из раздела cheat / game:pubg, посвящённый внешнему читу для PUBG.
Он в основном написан на C/C++ и C++ и сосредоточен на оверлеях и анализе памяти.
Он прежде всего полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:pubg.
