Este proyecto demuestra cómo enganchar (hookear) la aplicación de firmas de controladores de Windows (DSE) para permitir la carga de controladores de núcleo sin firmar. Parchea la función de verificación de firmas de CI.dll o la variable global g_CiEnabled para eludir las comprobaciones de integridad de código que normalmente impiden cargar controladores sin firmar. Está dirigido a investigadores de núcleo que estudian técnicas de evasión de DSE.
