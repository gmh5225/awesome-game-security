Ce projet démontre comment accrocher (hooker) l'application de signature des pilotes Windows (DSE) afin de permettre le chargement de pilotes noyau non signés. Il patche la fonction de vérification de signature de CI.dll ou la variable globale g_CiEnabled pour contourner les contrôles d'intégrité du code qui empêchent normalement le chargement de pilotes non signés. Il s'adresse aux chercheurs noyau qui étudient les techniques de contournement de la DSE.
