Этот проект демонстрирует перехват (hook) механизма принудительной проверки подписи драйверов Windows (DSE), позволяющий загружать неподписанные драйверы ядра. Он патчит функцию проверки подписи в CI.dll или глобальную переменную g_CiEnabled, обходя проверки целостности кода, которые обычно препятствуют загрузке неподписанных драйверов. Предназначен для исследователей ядра, изучающих техники обхода DSE.
