Dieses Projekt zeigt die Nutzung von eBPF (extended Berkeley Packet Filter) auf Android für Tracing und Monitoring auf Kernel-Ebene. Es liefert Beispiele zum Schreiben und Laden von eBPF-Programmen auf Android-Geräten zum Nachverfolgen von Systemaufrufen, Überwachen des Netzwerkverkehrs, Verfolgen von Prozessaktivität und Erfassen von Leistungsdaten auf Kernel-Ebene. Es richtet sich an Android-Kernel-Entwickler und Sicherheitsforscher, die eBPF für die Laufzeit-Systemanalyse auf Android einsetzen.
