Este proyecto demuestra el uso de eBPF (extended Berkeley Packet Filter) en Android para el seguimiento y la monitorización a nivel de kernel. Ofrece ejemplos de escritura y carga de programas eBPF en dispositivos Android para trazar llamadas al sistema, monitorizar el tráfico de red, seguir la actividad de procesos y recopilar datos de rendimiento a nivel de kernel. Está dirigido a desarrolladores del kernel de Android e investigadores de seguridad que usan eBPF para el análisis del sistema en tiempo de ejecución en Android.
