Этот проект демонстрирует использование eBPF (extended Berkeley Packet Filter) на Android для трассировки и мониторинга на уровне ядра. В нём приведены примеры написания и загрузки eBPF-программ на устройствах Android для трассировки системных вызовов, мониторинга сетевого трафика, отслеживания активности процессов и сбора данных о производительности на уровне ядра. Материал предназначен для разработчиков ядра Android и исследователей безопасности, применяющих eBPF для анализа системы во время выполнения на Android.
