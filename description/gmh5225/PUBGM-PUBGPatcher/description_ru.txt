Этот проект — репозиторий из раздела cheat / game:pubgm, посвящённый PUBGM PUBGPatcher.
Он в основном написан на C/C++ и Java и сосредоточен на сетевом взаимодействии, конвейерах ассетов и моддинге.
Он прежде всего полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:pubgm.
