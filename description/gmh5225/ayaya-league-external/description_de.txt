Dieses Projekt konzentriert sich auf eine auf Node.js basierende Scripting-Plattform.
AyayaLeague ist eine externe Skriptplattform, die in Node.js geschrieben ist und benutzerdefinierte Skripte unterstützt.
Sie ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / game:lol untersuchen.
