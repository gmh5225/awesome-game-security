Ce projet est une arborescence source complète du framework de rétro-ingénierie Ghidra (miroir/fork de NSA Ghidra), incluant le décompilateur basé sur Java, les intégrations de débogueur et les modules associés.
Il fournit le code de base standard de Ghidra utilisé pour l'analyse binaire, le désassemblage et les workflows de décompilation.
Il est principalement utile aux ingénieurs en rétro-ingénierie et aux chercheurs en sécurité des jeux qui analysent des binaires de jeux protégés et d'anti-cheat.
