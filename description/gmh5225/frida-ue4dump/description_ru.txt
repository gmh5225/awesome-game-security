Этот проект — скрипт Frida для дампа информации SDK Unreal Engine 4 из запущенных Android-игр. Он перехватывает систему рефлексии UE4 через инструментацию Frida, чтобы во время выполнения перечислять классы UObject, извлекать смещения свойств, дампить сигнатуры функций и генерировать заголовки SDK из мобильных игр на UE4. Предназначен для реверс-инженеров мобильных игр, которые дампят SDK UE4 из Android-игр с помощью инструментации на базе Frida.
