Este proyecto es un script de Frida para volcar (dump) información del SDK de Unreal Engine 4 desde juegos de Android en ejecución. Se engancha al sistema de reflexión de UE4 mediante instrumentación de Frida para enumerar clases UObject, extraer offsets de propiedades, volcar firmas de funciones y generar cabeceras del SDK a partir de juegos UE4 móviles en tiempo de ejecución. Está dirigido a ingenieros inversos de juegos móviles que vuelcan SDKs de UE4 desde juegos Android usando instrumentación basada en Frida.
