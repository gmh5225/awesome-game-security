Ce projet est un script Frida destiné à extraire (dumper) les informations du SDK Unreal Engine 4 depuis des jeux Android en cours d'exécution. Il s'accroche au système de réflexion de UE4 via l'instrumentation Frida pour énumérer les classes UObject, extraire les offsets de propriétés, dumper les signatures de fonctions et générer des en-têtes SDK à partir de jeux UE4 mobiles à l'exécution. Il s'adresse aux reverse engineers de jeux mobiles qui dumpent les SDK UE4 depuis des jeux Android à l'aide d'une instrumentation basée sur Frida.
