Инструмент для отключения Windows Defender со встроенным обходом UAC и повышением привилегий до SYSTEM.
Манипулирует токенами привилегий и использует техники обхода UAC на основе COM для получения повышенного доступа, после чего отключает службы защиты в реальном времени и защиты от несанкционированных изменений.
