本專案是一個小型核心 detour 函式庫，用於在不引入外部相依套件的情況下修補已匯出的 Windows 核心常式。其封存 README 明確將其描述為面向如 `MmCopyMemory`、`MmCopyVirtualMemory` 等函式的舊式但簡單的 hook 函式庫，實作核心是 `c_detour` 類別：保存被覆蓋的原始位元組、建構自訂絕對跳躍 stub，並透過以 MDL 為基礎的可寫對應寫入修補。範例進入點展示了用此函式庫 hook `KeAttachProcess`，同時 detour 輔助邏輯刻意避開更顯眼的 page-guard 類手法，使安裝與移除保持自成一體。它主要面向希望獲得 inline detour、自訂 trampoline stub 以及低摩擦核心 hook 實驗精簡參考的 Windows 核心研究者。
