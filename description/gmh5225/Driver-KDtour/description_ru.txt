Этот проект — небольшая библиотека kernel-detour для патчинга экспортируемых процедур ядра Windows без внешних зависимостей. В архивном README она явно описана как старая, но простая библиотека хуков для функций вроде `MmCopyMemory` и `MmCopyVirtualMemory`, а реализация строится вокруг класса `c_detour`, который сохраняет «украденные» байты, собирает пользовательский stub абсолютного перехода и записывает патчи через доступные на запись отображения на базе MDL. Примерная точка входа показывает хук `KeAttachProcess`, а вспомогательный код detour избегает более заметных приёмов в стиле page-guard и делает установку и снятие самодостаточными. В первую очередь полезна исследователям ядра Windows как компактный ориентир по inline-detour, пользовательским trampoline-stub и низкофрикционным экспериментам с хуками ядра.
