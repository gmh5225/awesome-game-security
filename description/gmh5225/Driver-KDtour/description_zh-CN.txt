本项目是一个小型内核 detour 库，用于在不引入外部依赖的情况下修补已导出的 Windows 内核例程。其归档 README 明确将其描述为面向如 `MmCopyMemory`、`MmCopyVirtualMemory` 等函数的旧式但简单的 hook 库，实现核心是 `c_detour` 类：保存被覆盖的原始字节、构建自定义绝对跳转 stub，并通过基于 MDL 的可写映射写入补丁。示例入口展示了用该库 hook `KeAttachProcess`，同时 detour 辅助逻辑刻意避开更显眼的 page-guard 类手法，使安装与卸载保持自包含。它主要面向希望获得 inline detour、自定义 trampoline stub 以及低摩擦内核 hook 实验紧凑参考的 Windows 内核研究者。
