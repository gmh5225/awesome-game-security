Este proyecto es una pequeña biblioteca de detours de kernel para parchear rutinas exportadas del kernel de Windows sin dependencias externas. Su README archivado la presenta explícitamente como una biblioteca de hooking antigua pero sencilla para funciones como `MmCopyMemory` y `MmCopyVirtualMemory`, y la implementación gira en torno a una clase `c_detour` que guarda los bytes robados, construye un stub de salto absoluto personalizado y escribe los parches mediante mapeos escribibles respaldados por MDL. El punto de entrada de ejemplo muestra el hook de `KeAttachProcess`, mientras que el ayudante de detour evita trucos más obvios estilo page-guard y mantiene la instalación y la eliminación autocontenidas. Resulta útil sobre todo para investigadores del kernel de Windows que buscan una referencia compacta de detours inline, stubs trampolín personalizados y experimentos de hooks de bajo rozamiento.
