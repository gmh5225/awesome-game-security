Dieses Projekt ist eine kleine Kernel-Detour-Bibliothek zum Patchen exportierter Windows-Kernelroutinen ohne externe Abhängigkeiten. Die archivierte README stellt sie ausdrücklich als alte, aber einfache Hooking-Bibliothek für Funktionen wie `MmCopyMemory` und `MmCopyVirtualMemory` vor; die Implementierung dreht sich um eine Klasse `c_detour`, die gestohlene Bytes speichert, einen eigenen Absolute-Jump-Stub baut und Patches über MDL-gestützte schreibbare Mappings schreibt. Der Beispiel-Einstiegspunkt zeigt das Hooken von `KeAttachProcess`, während der Detour-Helfer auffälligere Page-Guard-Tricks vermeidet und Installation sowie Entfernung selbstständig hält. Sie ist vor allem für Windows-Kernel-Forscher nützlich, die eine kompakte Referenz für Inline-Detours, eigene Trampoline-Stubs und reibungsarme Kernel-Hook-Experimente suchen.
