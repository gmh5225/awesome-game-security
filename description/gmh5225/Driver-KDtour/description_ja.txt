本プロジェクトは、外部依存を持ち込まずにエクスポート済みの Windows カーネルルーチンをパッチするための小型カーネル detour ライブラリです。アーカイブされた README では、`MmCopyMemory` や `MmCopyVirtualMemory` などの関数向けの古くて単純なフックライブラリとして明示されており、実装の中心は盗まれたバイト列を保存し、カスタム絶対ジャンプ stub を組み立て、MDL ベースの書き込み可能マッピング経由でパッチを書き込む `c_detour` クラスです。サンプルのエントリポイントでは `KeAttachProcess` をフックする用法が示され、detour ヘルパーはより露骨な page-guard 系の手法を避け、インストールと削除を自己完結に保つよう書かれています。主に、インライン detour、カスタム trampoline stub、低摩擦なカーネルフック実験のコンパクトな参考を求める Windows カーネル研究者向けです。
