Набор C#-сборок (плагинов) для платформы скриптинга LeagueSharp под League of Legends: автоматические скрипты чемпионов Annie, Cassiopeia, Cho'Gath, Darius, Evelynn и Katarina, а также трекер кулдаунов вардов и summoner spells с оверлеем на миникарте.
Каждый модуль чемпиона реализует логику комбо, последовательности заклинаний и выбор цели через API Orbwalker, TargetSelector и предсказания заклинаний SDK LeagueSharp, а трекер рисует индикаторы кулдаунов и позиции вардов на HUD с помощью колбэков Drawing.OnDraw.
В первую очередь полезен исследователям игровой безопасности, изучающим архитектуры плагинов скриптовых платформ и логику автоматизированного геймплея в League of Legends.
