League of Legends 向けの LeagueSharp スクリプティングプラットフォーム用 C# アセンブリ（プラグイン）集で、Annie、Cassiopeia、Cho'Gath、Darius、Evelynn、Katarina の自動チャンピオンスクリプトに加え、ミニマップオーバーレイ付きのワード／サモナースペルクールダウン追跡機能を提供します。
各チャンピオンモジュールは、LeagueSharp SDK の Orbwalker、TargetSelector、スペル予測 API を用いてコンボロジック、スペル詠唱シーケンス、ターゲット選択を実装し、トラッカーは Drawing.OnDraw コールバックで HUD 上にクールダウン表示とワード位置を描画します。
主に、League of Legends におけるスクリプティングプラットフォームのプラグイン構成や自動プレイロジックを研究するゲームセキュリティ研究者向けです。
