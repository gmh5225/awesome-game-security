Este proyecto es un framework de GUI de cheats para Valorant que usa Dear ImGui como sistema de renderizado del menú. Proporciona una interfaz ImGui con estilo, con pestañas, interruptores y deslizadores para controlar funciones de cheat como ESP, aimbot y modificaciones visuales en Valorant. El código en C++ demuestra el diseño de menús de cheat y patrones de interacción con juegos UE4. Está orientado a investigadores de seguridad en juegos que estudian implementaciones de UI de cheats en Valorant y arquitecturas de menús basadas en ImGui.
