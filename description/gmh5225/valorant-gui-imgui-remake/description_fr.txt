Ce projet est un framework d'interface graphique de cheat pour Valorant utilisant Dear ImGui comme système de rendu du menu. Il fournit une interface ImGui stylisée avec onglets, interrupteurs et curseurs pour contrôler des fonctionnalités de cheat telles que l'ESP, l'aimbot et les modifications visuelles dans Valorant. La base de code C++ illustre la conception de menus de cheat et les schémas d'interaction avec les jeux UE4. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les implémentations d'UI de cheat Valorant et les architectures de menus basées sur ImGui.
