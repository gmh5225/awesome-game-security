Этот проект — фреймворк GUI для читов Valorant, использующий Dear ImGui в качестве системы отрисовки меню. Он предоставляет стилизованный интерфейс ImGui со вкладками, переключателями и ползунками для управления функциями чита, такими как ESP, аимбот и визуальные модификации в Valorant. Кодовая база на C++ демонстрирует проектирование чит-меню и паттерны взаимодействия с играми на UE4. Проект предназначен для исследователей игровой безопасности, изучающих реализации UI читов Valorant и архитектуры чит-меню на базе ImGui.
