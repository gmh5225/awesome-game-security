Questo progetto dimostra l'uso dei Performance Monitoring Interrupts (PMI) e dei contatori di prestazione hardware (HPC) per il monitoraggio della sicurezza su Windows. Configura i contatori di prestazione della CPU per generare interrupt su eventi specifici come branch misprediction o cache miss, consentendo di rilevare pattern di esecuzione anomali che possono indicare code injection o attacchi ROP. È rivolto ai ricercatori di sicurezza che esplorano tecniche di rilevamento assistite dall'hardware.
