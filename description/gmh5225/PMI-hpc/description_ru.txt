Этот проект демонстрирует использование прерываний мониторинга производительности (PMI) и аппаратных счётчиков производительности (HPC) для мониторинга безопасности в Windows. Он настраивает счётчики производительности ЦП так, чтобы генерировать прерывания при определённых событиях — например, при ошибках предсказания ветвлений или промахах кэша — что позволяет обнаруживать аномальные паттерны выполнения, указывающие на внедрение кода или ROP-атаки. Проект предназначен для исследователей безопасности, изучающих аппаратно-поддерживаемые методы обнаружения.
