Ce projet démontre l'utilisation des Performance Monitoring Interrupts (PMI) et des compteurs de performance matériels (HPC) pour la surveillance de sécurité sous Windows. Il configure les compteurs de performance du CPU afin de générer des interruptions sur des événements spécifiques tels que les erreurs de prédiction de branchement ou les défauts de cache, permettant ainsi de détecter des modèles d'exécution anormaux susceptibles d'indiquer une injection de code ou des attaques ROP. Il s'adresse aux chercheurs en sécurité qui explorent les techniques de détection assistées par le matériel.
