Este proyecto demuestra el uso de Performance Monitoring Interrupts (PMI) y contadores de rendimiento de hardware (HPC) para la monitorización de seguridad en Windows. Configura los contadores de rendimiento de la CPU para generar interrupciones ante eventos específicos como fallos de predicción de saltos o fallos de caché, lo que permite detectar patrones de ejecución anómalos que pueden indicar inyección de código o ataques ROP. Está dirigido a investigadores de seguridad que exploran técnicas de detección asistidas por hardware.
