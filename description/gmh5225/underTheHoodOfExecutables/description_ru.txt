Этот проект — углублённое исследование того, как программы на C превращаются из исходного кода в исполняемые бинарные файлы.
Репозиторий содержит подробное руководство по пониманию линковки, загрузки и форматов исполняемых файлов.
Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / руководств.
