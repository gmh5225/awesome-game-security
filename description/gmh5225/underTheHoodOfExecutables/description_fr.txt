Ce projet explore en profondeur comment les programmes C se transforment du code source en binaires exécutables.
Ce dépôt contient un guide complet pour comprendre l'édition de liens, le chargement et les formats d'exécutables.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine de l'anti-cheat / des guides.
