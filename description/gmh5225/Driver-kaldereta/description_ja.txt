本プロジェクトは、未署名のカーネルドライバとユーザーモードのサンプルを組み合わせ、カスタム通信フックを通じて幅広いメモリ操作および入力シミュレーション機能を提供します。
アーカイブされた README には、プロセス検索、モジュールベース検出、仮想メモリの割り当てと保護変更、読み書きプリミティブ、入力シミュレーション、パターンスキャン、さらには手動 DLL マッピングが一覧されており、サンプルラッパーは従来の公開デバイスインターフェイスではなく、フックされた Win32 呼び出し経路経由で `KALDERETA_MEMORY` リクエストを送信します。
この構成により、本リポジトリは機能豊富なチートドライバの骨格としても、カーネルメモリ操作と上位のユーザーモードヘルパーライブラリを組み合わせる参考としても有用です。
主に、ドライバ通信フック、メモリツーリングのプリミティブ、ユーザー／カーネル統合の手動マッピングワークフローを研究する Windows カーネルおよびリバースエンジニアリング研究者向けです。
