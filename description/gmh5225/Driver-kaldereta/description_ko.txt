이 프로젝트는 서명되지 않은 커널 드라이버와 사용자 모드 샘플을 결합하여, 사용자 정의 통신 훅을 통해 광범위한 메모리 조작 및 입력 시뮬레이션 기능을 제공합니다.
보관된 README에는 프로세스 조회, 모듈 베이스 탐색, 가상 메모리 할당 및 보호 변경, 읽기·쓰기 프리미티브, 입력 시뮬레이션, 패턴 스캐닝, 수동 DLL 매핑까지 나열되어 있으며, 샘플 래퍼는 일반적인 공개 장치 인터페이스 대신 훅된 Win32 호출 경로로 `KALDERETA_MEMORY` 요청을 전송합니다.
이러한 구조 덕분에 이 저장소는 기능이 풍부한 치트 드라이버 골격으로도, 커널 메모리 작업을 상위 사용자 모드 헬퍼 라이브러리와 짝짓는 참고 자료로도 유용합니다.
주로 드라이버 통신 훅, 메모리 도구 프리미티브, 사용자/커널 통합 수동 매핑 워크플로를 연구하는 Windows 커널 및 리버스 엔지니어링 연구자에게 유용합니다.
