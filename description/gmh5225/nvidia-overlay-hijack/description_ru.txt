Этот проект демонстрирует технику перехвата оверлея NVIDIA GeForce Experience для отрисовки пользовательского контента без создания новых окон. Он захватывает контекст рендеринга существующей поверхности оверлея NVIDIA и рисует через неё пользовательский ESP или меню, обходя античит-системы, которые отслеживают создание новых окон оверлея. Проект предназначен для исследователей безопасности игр, изучающих техники перехвата оверлеев.
