Dieses Projekt, mcpup, ist ein Werkzeug zur Verwaltung und Bereitstellung von MCP-Servern (Model Context Protocol). Es bietet Hilfsmittel zum Entdecken, Installieren, Konfigurieren und Ausführen von MCP-Servern, die KI-Coding-Assistenten um zusätzliche Funktionen erweitern. Es richtet sich an Entwickler, die eine MCP-Server-Infrastruktur für KI-gestützte Entwicklungsabläufe aufbauen.
