Этот проект — Triadz, внутренний чит для Fortnite с ESP, аимботом и возможностями эксплуатации через хукинг движка UE4. Он работает внутри процесса Fortnite с полным доступом к игровым объектам и рендерингу, демонстрируя архитектуру внутренних читов для игр на UE4, защищённых EAC. Предназначен для исследователей игровой безопасности, изучающих паттерны внутренних читов Fortnite и поверхности обнаружения EAC.
