Этот проект предоставляет список callback-функций.
Он организован как документация и справочные материалы для области античита / Windows ring0 callback, а не как отдельная автономная runtime-кодовая база.
Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / Windows ring0 callback.
