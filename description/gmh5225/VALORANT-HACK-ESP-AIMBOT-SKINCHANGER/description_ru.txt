Этот проект — чит для Valorant с функциями ESP (контуры игроков, здоровье, дистанция), аимбота (автонаведение со сглаживанием) и скин-ченджера (клиентское изменение скинов оружия). Он обращается к памяти игры через драйвер ядра, чтобы обойти пользовательские защиты Vanguard, и читает данные сущностей UE4 для отрисовки оверлеев. Предназначен для исследователей античитов, изучающих реализацию функций читов Valorant и паттерны обхода Vanguard.
