Этот проект — репозиторий в разделе cheat / game:rust, посвящённый rico cheat rust external.
Он в основном написан на C++ и C/C++ и сосредоточен на разработке драйверов, рендеринге и Unity.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:rust.
