Ce projet est un miroir de l'arborescence source de Source Engine 2007 (ère Orange Box), couvrant le moteur, le client, le serveur et les modules associés sous se2007.
Il comprend des éléments internes du moteur en C++ tels que le réseau, les demos, les hooks de rendu dans la pile Source, ainsi que des sous-systèmes côté jeu utiles pour étudier cette génération de moteur.
Il s'adresse surtout aux développeurs de jeux, programmeurs de moteurs et reverse engineers travaillant sur des titres et outils Source Engine.
