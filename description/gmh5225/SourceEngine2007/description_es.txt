Este proyecto es un espejo del árbol de código fuente de Source Engine 2007 (era Orange Box) y cubre el motor, el cliente, el servidor y módulos relacionados bajo se2007.
Incluye internos del motor en C++ como redes, demos, hooks de renderizado hacia la pila Source y subsistemas del lado del juego útiles para estudiar esa generación del motor.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e ingenieros inversos que trabajan con títulos y herramientas de Source Engine.
