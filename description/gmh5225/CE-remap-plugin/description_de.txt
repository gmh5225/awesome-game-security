Ein in Delphi geschriebenes Cheat-Engine-Plugin, das Disassembler-Speicherseiten für die Analyse neu zuordnet.
Hookt in das Cheat-Engine-Plugin-SDK, um die Seite einer Disassembler-Adresse neu zuzuordnen und so Code sichtbar zu machen, der sonst verborgen oder verschleiert wäre.
Getestet mit Cheat Engine 7.4.
