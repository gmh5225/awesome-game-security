Un plugin per Cheat Engine scritto in Delphi che rimappa le pagine di memoria del disassemblatore per l'analisi.
Si aggancia all'SDK dei plugin di Cheat Engine per rimappare la pagina di un indirizzo del disassemblatore, consentendo di visualizzare codice altrimenti nascosto o offuscato.
Testato su Cheat Engine 7.4.
