Ce projet est un SDK amélioré du moteur Source pour CS:GO, avec des définitions de classes corrigées, des déclarations d'interfaces supplémentaires et une couverture netvar plus complète que les SDK filtrés classiques. Il fournit des en-têtes C++ affinés pour les interfaces client/engine/server de CS:GO, le système d'entités et les structures de rendu. Il s'adresse aux développeurs de cheats CS:GO et aux chercheurs sur le moteur Source qui ont besoin d'un SDK précis et complet.
