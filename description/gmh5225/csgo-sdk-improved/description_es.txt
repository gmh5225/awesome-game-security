Este proyecto es un SDK mejorado del motor Source de CS:GO, con definiciones de clases corregidas, declaraciones de interfaces adicionales y una cobertura de netvars más completa que los SDK filtrados habituales. Proporciona encabezados C++ refinados para las interfaces client/engine/server de CS:GO, el sistema de entidades y las estructuras de renderizado. Está orientado a desarrolladores de cheats de CS:GO e investigadores del motor Source que necesitan un SDK preciso y completo.
