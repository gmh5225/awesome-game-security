Этот проект — улучшенный SDK движка Source для CS:GO с исправленными определениями классов, дополнительными объявлениями интерфейсов и более полным покрытием netvar по сравнению со стандартными утёкшими SDK. Он предоставляет уточнённые C++-заголовки для интерфейсов client/engine/server CS:GO, системы сущностей и структур рендеринга. Предназначен для разработчиков читов CS:GO и исследователей движка Source, которым нужен точный и полный SDK.
