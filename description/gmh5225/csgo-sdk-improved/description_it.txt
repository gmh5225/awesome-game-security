Questo progetto è un SDK migliorato del motore Source di CS:GO, con definizioni di classi corrette, dichiarazioni di interfacce aggiuntive e una copertura netvar più completa rispetto agli SDK leakati standard. Fornisce header C++ raffinati per le interfacce client/engine/server di CS:GO, il sistema di entità e le strutture di rendering. È rivolto agli sviluppatori di cheat per CS:GO e ai ricercatori del motore Source che necessitano di un SDK accurato e completo.
