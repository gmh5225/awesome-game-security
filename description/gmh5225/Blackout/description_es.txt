Herramienta en modo kernel que aprovecha el controlador gmer (obtenido de loldrivers.io) para deshabilitar o terminar procesos de EDR y antivirus.
Funciona cargando un controlador vulnerable firmado y emitiendo llamadas IOCTL para matar por PID los procesos de los productos de seguridad objetivo.
Admite Windows Defender con supresión continua para evitar que el servicio se reinicie.
