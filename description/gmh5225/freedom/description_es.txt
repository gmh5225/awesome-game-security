Este proyecto es Freedom, una herramienta de Android para eludir la verificación de compras dentro de la aplicación en juegos móviles. Intercepta las llamadas a la API de facturación de Google Play y devuelve respuestas falsas de confirmación de compra, lo que permite a los usuarios obtener artículos premium y moneda sin pagar. Funciona mediante el hooking de la interfaz del servicio de facturación. Está dirigido a investigadores de seguridad móvil que estudian la seguridad de las compras dentro de la aplicación y el abuso de la API de facturación en Android.
