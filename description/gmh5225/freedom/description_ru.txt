Этот проект — Freedom, инструмент для Android, предназначенный для обхода проверки внутриигровых покупок в мобильных играх. Он перехватывает вызовы Google Play Billing API и возвращает поддельные ответы с подтверждением покупки, позволяя пользователям получать премиум-предметы и валюту без оплаты. Работает через перехват (hooking) интерфейса сервиса биллинга. Предназначен для исследователей мобильной безопасности, изучающих защиту внутриигровых покупок и злоупотребление Billing API на Android.
