本项目是一个小型用户态演示，通过枚举系统句柄并从其他进程中复制目标句柄来展示句柄劫持。示例调用 NtQuerySystemInformation（SystemHandleInformation），在返回的表中扫描选定的对象指针，以 PROCESS_DUP_HANDLE 打开拥有该句柄的进程，再用 DuplicateHandle 将该句柄复制到当前进程。其 README 大部分篇幅用于说明攻击模型与 DuplicateHandle 参数，因此代码更适合作为被盗句柄复用的最小概念验证，而非通用框架。它主要供 Windows 安全研究人员参考，用于了解句柄表枚举、跨进程句柄复制以及句柄劫持背后的实际机制。
