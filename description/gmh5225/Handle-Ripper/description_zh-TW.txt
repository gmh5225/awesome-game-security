本專案是一個小型使用者模式示範，透過列舉系統控制代碼並從其他程序複製目標控制代碼來展示控制代碼劫持。範例呼叫 NtQuerySystemInformation（SystemHandleInformation），在回傳的表格中掃描選定的物件指標，以 PROCESS_DUP_HANDLE 開啟擁有該控制代碼的程序，再用 DuplicateHandle 將該控制代碼複製到目前程序。其 README 大部分篇幅用於說明攻擊模型與 DuplicateHandle 參數，因此程式碼更適合作為被竊控制代碼重用的最小概念驗證，而非通用框架。它主要供 Windows 安全性研究人員參考，用於了解控制代碼表列舉、跨程序控制代碼複製以及控制代碼劫持背後的實際機制。
