本プロジェクトは、システムハンドルを列挙し、別プロセスから対象ハンドルを複製することでハンドルハイジャックを示す、小さなユーザーモードのデモです。サンプルは NtQuerySystemInformation を SystemHandleInformation で呼び出し、返されたテーブルから指定のオブジェクトポインタを探し、PROCESS_DUP_HANDLE で所有プロセスを開き、DuplicateHandle でそのハンドルを現在のプロセスへコピーします。README の多くは攻撃モデルと DuplicateHandle のパラメータ解説に割かれており、コードは汎用フレームワークというより、奪取したハンドル再利用の最小限の概念実証として読むのが適切です。ハンドルテーブル列挙、プロセス間ハンドル複製、ハンドルハイジャックの実際の仕組みをコンパクトに学びたい Windows セキュリティ研究者向けです。
