Этот проект — небольшая демонстрация в user mode перехвата handles: перечисляются системные handles и целевой handle дублируется из другого процесса. Пример вызывает NtQuerySystemInformation с SystemHandleInformation, ищет в возвращённой таблице выбранный указатель на объект, открывает владеющий процесс с PROCESS_DUP_HANDLE и с помощью DuplicateHandle копирует этот handle в текущий процесс. README в основном объясняет модель атаки и параметры DuplicateHandle, поэтому код лучше воспринимать как минимальный proof of concept повторного использования украденного handle, а не как универсальный фреймворк. Он полезен в первую очередь исследователям безопасности Windows, которым нужен компактный пример перечисления таблицы handles, межпроцессного дублирования и практической механики handle hijacking.
