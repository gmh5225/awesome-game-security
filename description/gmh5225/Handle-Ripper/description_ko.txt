이 프로젝트는 시스템 핸들을 열거하고 다른 프로세스에서 대상 핸들을 복제하여 핸들 하이재킹을 보여주는 작은 사용자 모드 데모입니다. 샘플은 NtQuerySystemInformation을 SystemHandleInformation으로 호출하고, 반환된 테이블에서 선택한 객체 포인터를 검색한 뒤, PROCESS_DUP_HANDLE로 소유 프로세스를 열고 DuplicateHandle로 해당 핸들을 현재 프로세스에 복사합니다. README의 대부분은 공격 모델과 DuplicateHandle 매개변수 설명에 할애되어 있으므로, 코드는 범용 프레임워크라기보다 탈취한 핸들 재사용에 대한 최소 개념 증명으로 읽는 것이 좋습니다. 핸들 테이블 열거, 프로세스 간 핸들 복제, 핸들 하이재킹의 실제 동작을 간결한 예제로 보고 싶은 Windows 보안 연구자에게 주로 유용합니다.
