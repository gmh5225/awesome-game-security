Этот проект — репозиторий из раздела cheat / game:rust, посвящённый rust external. Он в основном написан на C++ и C/C++ и сосредоточен на рендеринге и сетевом взаимодействии. Он особенно полезен исследователям безопасности игр и инженерам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:rust.
