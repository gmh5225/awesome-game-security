Este proyecto es un framework de overlay emparejado en modo usuario y modo kernel para Escape From Tarkov que se comunica mediante un hook en NtDxgkGetTrackedWorkloadStatistics de dxgkrnl. El componente de kernel parchea la exportación con un pequeño stub de salto, atiende solicitudes de búsqueda de base de módulo y de lectura o escritura de memoria de proceso, y también resuelve las exportaciones de dibujo de win32k para poder renderizar cajas y texto desde el espacio de kernel. El lado de modo usuario envuelve el mismo syscall desde win32u.dll, envía estructuras de solicitud por la ruta enganchada y superpone un overlay basado en ImGui y visuales específicos del juego sobre ese canal de comunicación. Resulta especialmente útil para ingenieros inversos que estudian la comunicación de drivers basada en dxgkrnl, las rutas de dibujo asistidas por el kernel y los diseños de overlay mixtos usuario/kernel para herramientas al estilo Tarkov.
