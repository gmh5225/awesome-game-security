Этот проект — намеренно уязвимый драйвер ядра Windows, созданный в образовательных целях. Он предоставляет намеренно небезопасные обработчики IOCTL, демонстрирующие типичные уязвимости драйверов: произвольное чтение/запись памяти, переполнение буфера, use-after-free и состояния гонки. Драйвер служит учебной целью для практики эксплуатации ядра. Он предназначен для студентов, изучающих эксплуатацию ядра, и специалистов по безопасности, преподающих курсы по уязвимостям драйверов.
