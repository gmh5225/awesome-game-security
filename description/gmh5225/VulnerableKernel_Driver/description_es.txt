Este proyecto es un controlador de kernel de Windows deliberadamente vulnerable, creado con fines educativos. Expone manejadores IOCTL intencionadamente inseguros que demuestran vulnerabilidades comunes de controladores, como lectura/escritura arbitraria de memoria, desbordamiento de búfer, use-after-free y condiciones de carrera. El controlador sirve como objetivo de aprendizaje para practicar la explotación del kernel. Está dirigido a estudiantes de explotación de kernel y a formadores de seguridad que imparten clases sobre vulnerabilidades de controladores.
