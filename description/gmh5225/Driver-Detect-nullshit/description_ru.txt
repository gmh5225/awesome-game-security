Этот проект — компактный детектор определённого стиля вручную отображаемых драйверных хуков, которые подменяют экспортируемые ядерные процедуры простыми заглушками абсолютного перехода.
В архивном README он назван простым детектором Null-драйвера; код обходит таблицы экспорта модулей вроде `win32kfull.sys`, `win32kbase.sys`, `dxgkrnl.sys` и `ntoskrnl.exe`, ища патчи вида `mov rax, imm64 ; jmp rax`, чьи цели лежат вне владеющего образа.
Вместо полноценного каркаса проверки целостности он сосредоточен на одной распространённой сигнатуре inline-хука, часто используемой для перенаправления функций, связанных с syscall, при маппинге драйвера или настройке связи.
Он в основном полезен исследователям античита и защитного ядра, изучающим лёгкое обнаружение грубых трамплинов export-хуков в ключевых модулях Windows.
