Este proyecto es un detector compacto para un estilo concreto de hook de controlador mapeado manualmente que sustituye rutinas del kernel exportadas por simples stubs de salto absoluto.
Su README archivado lo describe como un detector simple de controlador Null, y el código recorre las tablas de exportación de módulos como `win32kfull.sys`, `win32kbase.sys`, `dxgkrnl.sys` y `ntoskrnl.exe` en busca de parches del estilo `mov rax, imm64 ; jmp rax` cuyas dianas quedan fuera de la imagen propietaria.
En lugar de ofrecer un marco completo de integridad, se centra en una firma habitual de inline hook que a menudo se usa para redirigir funciones relacionadas con syscall durante el mapeo del controlador o la configuración de la comunicación.
Resulta especialmente útil para investigadores de anti-cheat y de kernel defensivo que estudian la detección ligera de trampolines rudimentarios de export-hook en módulos centrales de Windows.
