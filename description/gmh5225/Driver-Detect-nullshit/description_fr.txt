Ce projet est un détecteur compact destiné à un style précis de hook de pilote mappé manuellement qui remplace des routines noyau exportées par de simples stubs de saut absolu.
Son README archivé le décrit comme un simple détecteur de pilote Null, et le code parcourt les tables d’export de modules tels que `win32kfull.sys`, `win32kbase.sys`, `dxgkrnl.sys` et `ntoskrnl.exe` à la recherche de correctifs du type `mov rax, imm64 ; jmp rax` dont les cibles se situent hors de l’image propriétaire.
Plutôt que de fournir un cadre d’intégrité complet, il se concentre sur une signature d’inline hook courante, souvent utilisée pour rediriger des fonctions liées aux appels système lors du mapping de pilote ou de la mise en place de la communication.
Il est surtout utile aux chercheurs en anti-cheat et en noyau défensif qui étudient la détection légère de trampolines d’export-hook rudimentaires dans les modules Windows essentiels.
