Questo progetto è un rilevatore compatto per uno stile specifico di hook di driver mappato manualmente che sostituisce le routine kernel esportate con semplici stub di salto assoluto.
Il README archiviato lo definisce un semplice rilevatore di driver Null, e il codice scorre le tabelle di export di moduli come `win32kfull.sys`, `win32kbase.sys`, `dxgkrnl.sys` e `ntoskrnl.exe` cercando patch nello stile `mov rax, imm64 ; jmp rax` le cui destinazioni cadono fuori dall’immagine proprietaria.
Invece di fornire un framework completo di integrità, si concentra su una firma comune di inline hook spesso usata per reindirizzare funzioni legate alle syscall durante il mapping del driver o l’impostazione della comunicazione.
È soprattutto utile per ricercatori di anti-cheat e di kernel difensivo che studiano il rilevamento leggero di trampoline grezze di export-hook nei moduli Windows principali.
