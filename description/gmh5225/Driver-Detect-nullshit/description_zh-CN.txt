本项目是一个精简检测器，用于识别一类手动映射驱动钩子：用简单的绝对跳转桩替换已导出的内核例程。
其归档 README 称之为简单的 Null 驱动检测器；代码会遍历 `win32kfull.sys`、`win32kbase.sys`、`dxgkrnl.sys` 和 `ntoskrnl.exe` 等模块的导出表，查找形如 `mov rax, imm64 ; jmp rax` 的补丁，且跳转目标落在所属映像之外。
它并不提供完整的完整性框架，而是专注于一种常见的内联钩子特征，这类特征常在驱动映射或通信建立阶段用于重定向与系统调用相关的函数。
该工具主要面向反作弊与防御性内核研究者，用于研究如何轻量检测核心 Windows 模块中粗糙的导出钩子跳板。
