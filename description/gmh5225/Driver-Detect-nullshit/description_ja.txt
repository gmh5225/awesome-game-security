このプロジェクトは、エクスポートされたカーネルルーチンを単純な絶対ジャンプスタブで置き換える、特定スタイルの手動マップドドライバーフック向けのコンパクトな検出器です。
アーカイブされた README では単純な Null ドライバー検出器と呼ばれており、コードは `win32kfull.sys`、`win32kbase.sys`、`dxgkrnl.sys`、`ntoskrnl.exe` などのモジュールのエクスポートテーブルを走査し、所有イメージ外を指す `mov rax, imm64 ; jmp rax` 形式のパッチを探します。
完全な整合性フレームワークを提供するのではなく、ドライバーマッピングや通信セットアップ時に syscall 関連関数をリダイレクトするためによく使われる、一般的なインラインフック署名に焦点を当てています。
粗雑なエクスポートフックトランポリンをコア Windows モジュール上で軽量に検出する手法を研究する、アンチチートおよび防御的なカーネル研究者に主に役立ちます。
