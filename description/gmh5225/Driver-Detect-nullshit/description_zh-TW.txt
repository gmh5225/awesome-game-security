本專案是一個精簡偵測器，用於識別一類手動對應驅動程式掛鉤：以簡單的絕對跳躍 Stub 取代已匯出的核心常式。
其封存 README 稱之為簡單的 Null 驅動程式偵測器；程式碼會走訪 `win32kfull.sys`、`win32kbase.sys`、`dxgkrnl.sys` 與 `ntoskrnl.exe` 等模組的匯出表，尋找形如 `mov rax, imm64 ; jmp rax` 的修補，且跳躍目標落在所屬映像之外。
它並不提供完整的完整性框架，而是專注於一種常見的內嵌掛鉤特徵，這類特徵常在驅動程式對應或通訊建立階段用來重新導向與系統呼叫相關的函式。
該工具主要面向反作弊與防禦性核心研究者，用於研究如何輕量偵測核心 Windows 模組中粗糙的匯出掛鉤 Trampoline。
