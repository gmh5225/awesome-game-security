Dieses Projekt ist ein externer ESP-Hack für AssaultCube, ein schlankes FPS, das häufig zum Erlernen von Game Hacking genutzt wird. Es liest Spieler-Entity-Daten extern aus dem Speicher des Spiels und stellt Spielerboxen, Namen und Lebenspunkte als Overlay dar. Das einfache Speicherlayout von AssaultCube macht es zu einem idealen Lernziel für die Entwicklung externer ESP. Es richtet sich an Einsteiger, die externe Game-Hacking-Techniken lernen.
