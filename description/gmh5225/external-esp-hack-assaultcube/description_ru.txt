Этот проект — внешний ESP-хак для AssaultCube, лёгкого FPS, который часто используют для изучения game hacking. Он внешне читает данные сущностей игроков из памяти игры и отображает рамки, имена и здоровье поверх экрана. Простая структура памяти AssaultCube делает его удобной целью для изучения разработки внешнего ESP. Проект рассчитан на новичков, изучающих внешние техники game hacking.
