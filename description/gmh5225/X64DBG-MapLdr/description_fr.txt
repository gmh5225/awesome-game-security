Ce projet est un plugin x64dbg qui charge des fichiers MAP (sortie de carte de l'éditeur de liens) pour appliquer les noms de symboles et les adresses au binaire débogué. Il analyse les fichiers MAP générés par MSVC, Borland ou d'autres éditeurs de liens, et importe les noms de fonctions, les noms de variables globales et les informations de segments dans la base de symboles de x64dbg. Le plugin C++ relie les informations de symboles à la compilation au débogage à l'exécution. Il s'adresse aux reverse engineers qui utilisent x64dbg et disposent des fichiers MAP du binaire cible.
