Questo progetto è un plugin di x64dbg che carica file MAP (output della mappa del linker) per applicare nomi di simboli e indirizzi al binario sottoposto a debug. Analizza i file MAP generati da MSVC, Borland o altri linker e importa nomi di funzioni, nomi di variabili globali e informazioni sui segmenti nel database dei simboli di x64dbg. Il plugin C++ collega le informazioni sui simboli in fase di compilazione al debug in runtime. È pensato per i reverse engineer che usano x64dbg e hanno accesso ai file MAP del binario di destinazione.
