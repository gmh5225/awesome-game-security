Este proyecto se centra en registrar VEH enganchando RtlpCallVectoredHandlers.
Es una biblioteca que permite crear manejadores de excepciones vectorizadas personalizados más rápidos que los normales y que se ejecutan antes que cualquier manejador nativo.
Resulta especialmente útil para ingenieros de anti-trampas e investigadores de seguridad defensiva que trabajan en el área de anti-trampas / callbacks de Windows ring3.
