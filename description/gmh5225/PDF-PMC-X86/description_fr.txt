Ce projet héberge une étude PDF sur les compteurs de surveillance des performances (PMC / PMI) sur l'architecture x86.
L'archive est axée sur la documentation et sert de référence pour la surveillance basée sur les compteurs de performance matériels, la télémétrie et les recherches Windows associées.
Elle est principalement utile aux chercheurs en systèmes bas niveau et en sécurité des jeux qui étudient les compteurs de performance du CPU et la télémétrie de détection.
