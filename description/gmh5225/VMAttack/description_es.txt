Este proyecto es VMAttack, un complemento de IDA Pro para analizar y atacar la ofuscación de código basada en máquinas virtuales. Identifica bucles de despacho de la máquina virtual, extrae tablas de manejadores de opcodes virtuales, traza la ejecución de instrucciones virtuales y ayuda a desvirtualizar código protegido por VM. El complemento en Python ofrece análisis automatizado de arquitecturas VM personalizadas presentes en binarios protegidos. Está dirigido a ingenieros de ingeniería inversa que desofuscan malware y software protegido comercialmente mediante VM.
