Questo progetto è VMAttack, un plugin di IDA Pro per analizzare e attaccare l'offuscamento del codice basato su macchine virtuali. Identifica i loop del dispatcher della macchina virtuale, estrae le tabelle dei gestori di opcode virtuali, traccia l'esecuzione delle istruzioni virtuali e supporta la devirtualizzazione del codice protetto da VM. Il plugin Python fornisce un'analisi automatizzata delle architetture VM personalizzate presenti nei binari protetti. È rivolto agli ingegneri di reverse engineering che deoffuscano malware e software protetti commercialmente con VM.
