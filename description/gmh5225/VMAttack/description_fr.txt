Ce projet est VMAttack, un plugin IDA Pro destiné à analyser et attaquer l'obfuscation de code basée sur une machine virtuelle. Il identifie les boucles de dispersion (dispatcher) de la machine virtuelle, extrait les tables de gestionnaires d'opcodes virtuels, trace l'exécution des instructions virtuelles et aide à la dévirtualisation du code protégé par VM. Le plugin Python fournit une analyse automatisée des architectures VM personnalisées présentes dans les binaires protégés. Il s'adresse aux ingénieurs en rétroconception qui déobfusquent des malwares et des logiciels commercialement protégés par VM.
