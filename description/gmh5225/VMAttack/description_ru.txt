Этот проект — VMAttack, плагин для IDA Pro для анализа и атаки обфускации кода на основе виртуальной машины. Он выявляет циклы диспетчера виртуальной машины, извлекает таблицы обработчиков виртуальных опкодов, трассирует выполнение виртуальных инструкций и помогает девиртуализировать код, защищённый VM. Python-плагин обеспечивает автоматизированный анализ пользовательских архитектур VM в защищённых бинарных файлах. Он предназначен для инженеров по реверс-инжинирингу, деобфусцирующих вредоносное ПО и коммерчески защищённое ПО с VM-защитой.
