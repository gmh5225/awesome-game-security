Este proyecto es una herramienta para resolver las direcciones de las funciones de importación que Vanguard (el anti-cheat de Riot) usa internamente. Documenta el mecanismo de resolución de importaciones del controlador de núcleo de Vanguard y ofrece utilidades para analizar qué funciones de núcleo llama dicho controlador. Está orientado a investigadores de anti-cheat que realizan ingeniería inversa de los internals del controlador de núcleo de Vanguard y sus mecanismos de protección de importaciones.
