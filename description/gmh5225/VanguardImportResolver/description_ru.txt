Этот проект — инструмент для разрешения адресов импортируемых функций, которые Vanguard (античит Riot) использует внутри себя. Он документирует механизм разрешения импортов драйвера ядра Vanguard и предоставляет утилиты для анализа того, какие функции ядра вызывает этот драйвер. Предназначен для исследователей античита, занимающихся реверс-инжинирингом внутренностей драйвера ядра Vanguard и механизмов защиты импортов.
