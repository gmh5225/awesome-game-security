Этот проект эксплуатирует драйвер ядра ASUS для произвольного доступа к памяти ядра. Драйверы утилит материнских плат ASUS предоставляют IOCTL для аппаратного мониторинга, которые можно злоупотребить для чтения/записи физической памяти, образуя примитив BYOVD. Этот инструмент оборачивает уязвимый интерфейс драйвера ASUS для эксплуатации ядра. Он предназначен для исследователей BYOVD, изучающих уязвимости драйверов ASUS.
