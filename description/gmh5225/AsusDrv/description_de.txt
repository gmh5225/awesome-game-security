Dieses Projekt missbraucht den ASUS-Kernel-Treiber für beliebigen Kernel-Speicherzugriff. ASUS-Mainboard-Utility-Treiber stellen IOCTLs zur Hardwareüberwachung bereit, die zum Lesen/Schreiben physischen Speichers missbraucht werden können und so ein BYOVD-Primitive bilden. Dieses Tool kapselt die verwundbare ASUS-Treiberschnittstelle für Kernel-Exploitation. Es richtet sich an BYOVD-Forscher, die ASUS-Treiberschwachstellen untersuchen.
