Este proyecto explota el controlador de kernel de ASUS para obtener acceso arbitrario a la memoria del kernel. Los controladores de utilidades de placa base ASUS ofrecen IOCTLs de monitorización de hardware que pueden abusarse para leer/escribir memoria física, proporcionando un primitivo BYOVD. Esta herramienta encapsula la interfaz del controlador ASUS vulnerable para explotación en kernel. Está dirigida a investigadores BYOVD que estudian vulnerabilidades de controladores ASUS.
