Questo progetto è un cheat per Dota 2 che offre vantaggi di gioco attraverso l'interazione con il motore Source 2. Legge i dati delle entità di gioco per mostrare come overlay le posizioni nemiche, le barre di mana/salute, i tempi di ricarica delle abilità e le informazioni sugli oggetti, con funzioni aggiuntive come il lancio automatico delle abilità. Il cheat in C++ interagisce con le strutture dell'SDK Source 2 di Dota 2. È rivolto ai ricercatori di sicurezza nei videogiochi che studiano le implementazioni di cheat per Dota 2 e lo sfruttamento del motore Source 2.
