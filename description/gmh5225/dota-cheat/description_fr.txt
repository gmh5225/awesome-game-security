Ce projet est un cheat pour Dota 2 qui procure des avantages de jeu via l'interaction avec le moteur Source 2. Il lit les données des entités du jeu pour afficher en superposition les positions ennemies, les barres de mana/santé, les temps de recharge des sorts et les informations d'objets, avec des fonctions supplémentaires comme le lancement automatique de compétences. Le cheat en C++ interagit avec les structures du SDK Source 2 de Dota 2. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les implémentations de cheats Dota 2 et l'exploitation du moteur Source 2.
