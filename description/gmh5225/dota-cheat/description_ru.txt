Этот проект — чит для Dota 2, дающий игровые преимущества за счёт взаимодействия с движком Source 2. Он читает данные сущностей игры и отображает позиции противников, полоски маны/здоровья, кулдауны способностей и информацию о предметах в виде оверлея, а также включает дополнительные функции вроде автокаста способностей. Чит на C++ взаимодействует со структурами Source 2 SDK Dota 2. Предназначен для исследователей безопасности игр, изучающих реализации читов для Dota 2 и эксплуатацию движка Source 2.
