Este proyecto es un cheat de Dota 2 que ofrece ventajas de juego mediante la interacción con el motor Source 2. Lee datos de entidades del juego para mostrar como superposición las posiciones enemigas, barras de maná/salud, tiempos de reutilización de hechizos e información de objetos, con funciones adicionales como el lanzamiento automático de habilidades. El cheat en C++ interactúa con las estructuras del SDK Source 2 de Dota 2. Está orientado a investigadores de seguridad en videojuegos que estudian implementaciones de cheats de Dota 2 y la explotación del motor Source 2.
