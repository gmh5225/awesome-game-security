Microsoft Detours(detours-x64.lib)로 핵심 Windows API를 후킹하여 mhyprot2.sys 커널 드라이버 검사를 무력화하고 게임 프로세스에 디버거를 연결할 수 있게 하는, 원신(Genshin Impact) 안티디버깅 보호용 DLL 기반 우회 도구입니다.
CloseMhyprot2 모듈로 mhyprot2 안티치트 커널 드라이버를 종료하거나 언로드하고, DebuggerBypass.cpp의 API 후크로 NtQueryInformationProcess, IsDebuggerPresent 및 관련 안티디버그 호출을 가로채 게임의 무결성 검사로부터 디버거 존재를 숨깁니다.
miHoYo/HoYoverse의 mhyprot2 커널 수준 보호 시스템에 대한 안티디버그 우회 기법을 연구하는 게임 보안 연구자에게 주로 유용합니다.
