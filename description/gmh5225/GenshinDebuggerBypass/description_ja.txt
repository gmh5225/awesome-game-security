Microsoft Detours（detours-x64.lib）で重要な Windows API をフックし、mhyprot2.sys カーネルドライバのチェックを無力化してゲームプロセスへのデバッガ接続を可能にする、原神（Genshin Impact）のアンチデバッグ保護向け DLL ベースのバイパスです。
CloseMhyprot2 モジュールで mhyprot2 アンチチートカーネルドライバを終了またはアンロードし、DebuggerBypass.cpp 内の API フックで NtQueryInformationProcess、IsDebuggerPresent、および関連するアンチデバッグ呼び出しを横取りして、ゲームの整合性チェックからデバッガの存在を隠します。
miHoYo/HoYoverse の mhyprot2 カーネルレベル保護に対するアンチデバッグバイパス手法を研究するゲームセキュリティ研究者向けです。
