基於 DLL 的《原神》反偵錯保護繞過方案，使用 Microsoft Detours（detours-x64.lib）掛鉤關鍵 Windows API，以削弱 mhyprot2.sys 核心驅動檢查，從而允許將偵錯器附加到遊戲行程。
專案包含 CloseMhyprot2 模組，用於終止或卸載 mhyprot2 反作弊核心驅動，並結合 DebuggerBypass.cpp 中的 API 掛鉤，攔截 NtQueryInformationProcess、IsDebuggerPresent 及相關反偵錯呼叫，向遊戲的完整性檢查隱藏偵錯器存在。
主要面向研究如何針對 miHoYo/HoYoverse 的 mhyprot2 核心層防護系統繞過反偵錯技術的遊戲安全研究者。
