DLL-обход антиотладочной защиты Genshin Impact, который перехватывает критические Windows API с помощью Microsoft Detours (detours-x64.lib), чтобы нейтрализовать проверки ядерного драйвера mhyprot2.sys и позволить подключить отладчик к процессу игры.
Проект включает модуль CloseMhyprot2, который завершает или выгружает античит-драйвер ядра mhyprot2, в сочетании с API-хуками в DebuggerBypass.cpp, перехватывающими NtQueryInformationProcess, IsDebuggerPresent и связанные антиотладочные вызовы, чтобы скрыть присутствие отладчика от проверок целостности игры.
В основном полезен исследователям игровой безопасности, изучающим техники обхода антиотладки против системы защиты уровня ядра mhyprot2 от miHoYo/HoYoverse.
