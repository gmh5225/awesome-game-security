基于 DLL 的《原神》反调试保护绕过方案，使用 Microsoft Detours（detours-x64.lib）挂钩关键 Windows API，以削弱 mhyprot2.sys 内核驱动检查，从而允许将调试器附加到游戏进程。
项目包含 CloseMhyprot2 模块，用于终止或卸载 mhyprot2 反作弊内核驱动，并结合 DebuggerBypass.cpp 中的 API 挂钩，拦截 NtQueryInformationProcess、IsDebuggerPresent 及相关反调试调用，向游戏的完整性检查隐藏调试器存在。
主要面向研究如何针对 miHoYo/HoYoverse 的 mhyprot2 内核级防护系统绕过反调试技术的游戏安全研究者。
