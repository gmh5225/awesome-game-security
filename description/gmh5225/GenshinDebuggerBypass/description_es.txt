Un bypass basado en DLL de las protecciones antidepuración de Genshin Impact que engancha API críticas de Windows con Microsoft Detours (detours-x64.lib) para neutralizar las comprobaciones del controlador de kernel mhyprot2.sys y permitir adjuntar un depurador al proceso del juego.
El proyecto incluye un módulo CloseMhyprot2 que termina o descarga el controlador de kernel anticheat mhyprot2, combinado con hooks de API en DebuggerBypass.cpp que interceptan NtQueryInformationProcess, IsDebuggerPresent y llamadas antidepuración relacionadas para ocultar la presencia del depurador a las comprobaciones de integridad del juego.
Es especialmente útil para investigadores de seguridad en juegos que estudian técnicas de bypass antidepuración frente al sistema de protección a nivel de kernel mhyprot2 de miHoYo/HoYoverse.
