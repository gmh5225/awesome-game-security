Un bypass basato su DLL delle protezioni anti-debugging di Genshin Impact che aggancia API Windows critiche con Microsoft Detours (detours-x64.lib) per neutralizzare i controlli del driver kernel mhyprot2.sys e consentire di collegare un debugger al processo di gioco.
Il progetto include un modulo CloseMhyprot2 che termina o scarica il driver kernel anti-cheat mhyprot2, combinato con hook API in DebuggerBypass.cpp che intercettano NtQueryInformationProcess, IsDebuggerPresent e chiamate anti-debug correlate per nascondere la presenza del debugger ai controlli di integrità del gioco.
È soprattutto utile per i ricercatori di sicurezza nei giochi che studiano tecniche di bypass anti-debug contro il sistema di protezione a livello kernel mhyprot2 di miHoYo/HoYoverse.
