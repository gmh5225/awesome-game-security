Этот проект демонстрирует технику защиты памяти с помощью page guard PAGE_NOACCESS в Windows x86. Он помечает страницы кода как недоступные и использует обработчик VEH для временного восстановления доступа по требованию, создавая форму расшифровки при обращении, которая затрудняет статический анализ и дамп памяти. Предназначен для исследователей программной защиты, изучающих контроль доступа на уровне страниц для anti-tamper и anti-dump защиты.
