Eine Kernelmodus-Technik zum direkten Abbilden von Dateien in den Systemadressraum mithilfe interner NT-Funktionen.
Demonstriert die Nutzung von MmCreateSection und anderen undokumentierten APIs, um Dateiinhalte ohne Standard-Datei-E/A in den Kernel-Speicher zu mappen – nützlich für das verdeckte Laden von Treibern.
