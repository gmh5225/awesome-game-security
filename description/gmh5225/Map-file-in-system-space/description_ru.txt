Техника режима ядра для прямого отображения файлов в системное адресное пространство с помощью внутренних функций NT.
Демонстрирует использование MmCreateSection и других недокументированных API для отображения содержимого файла в память ядра без стандартного файлового ввода-вывода, что полезно для скрытой загрузки драйверов.
