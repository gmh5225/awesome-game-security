MinHook로 DirectX 11 Present 함수를 후킹하고, DirectWrite 기반 D3D11 텍스트 라이브러리인 FW1FontWrapper로 인게임 오버레이와 텍스트를 렌더링하는 내부 PUBG 치트 DLL입니다.
DLL은 PUBG 프로세스에 주입된 뒤 IDXGISwapChain::Present를 가로채 사용자 지정 드로우 호출을 삽입하며, FW1FontWrapper로 게임 렌더링 파이프라인에서 ESP 정보를 고품질 GPU 가속 텍스트로 그립니다.
MinHook 기반 함수 가로채기와 DirectWrite 오버레이 렌더링을 사용하는 내부 D3D11 Present 후킹 치트 아키텍처를 연구하는 게임 보안 연구자에게 주로 유용합니다.
