Внутренняя DLL-чит для PUBG, которая через MinHook перехватывает функцию Present в DirectX 11 и рисует внутриигровой оверлей с текстом через FW1FontWrapper (библиотеку текста для D3D11 на базе DirectWrite).
DLL внедряется в процесс PUBG, перехватывает IDXGISwapChain::Present для вставки собственных вызовов отрисовки и использует FW1FontWrapper для высококачественного GPU-ускоренного вывода ESP-текста прямо в конвейере рендеринга игры.
Она в первую очередь полезна исследователям игровой безопасности, изучающим архитектуры внутренних читов с хуком Present в D3D11, перехватом функций через MinHook и оверлеями на DirectWrite.
