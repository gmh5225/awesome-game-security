Une DLL de cheat interne pour PUBG qui accroche la fonction Present de DirectX 11 via MinHook afin d'afficher une overlay en jeu avec rendu de texte via FW1FontWrapper (bibliothèque de texte D3D11 basée sur DirectWrite).
La DLL est injectée dans le processus PUBG, intercepte IDXGISwapChain::Present pour insérer des appels de dessin personnalisés, et utilise FW1FontWrapper pour un rendu de texte GPU de haute qualité des informations ESP directement dans le pipeline de rendu du jeu.
Elle est surtout utile aux chercheurs en sécurité des jeux qui étudient les architectures de cheats internes basées sur le hook Present D3D11 avec interception de fonctions via MinHook et rendu d'overlay DirectWrite.
