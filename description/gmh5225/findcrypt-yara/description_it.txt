Questo progetto è FindCrypt, un plugin di IDA Pro che identifica costanti crittografiche e implementazioni di algoritmi nei binari disassemblati usando regole YARA. Cerca pattern di byte noti di S-box AES, tabelle di permutazione DES, costanti hash SHA, tabelle di lookup CRC e altre firme crittografiche, annotando le posizioni individuate con i nomi degli algoritmi. Il plugin Python aiuta gli analisti a localizzare rapidamente il codice crittografico nei binari. È rivolto ad analisti di malware e reverse engineer che identificano implementazioni crittografiche in binari sconosciuti.
