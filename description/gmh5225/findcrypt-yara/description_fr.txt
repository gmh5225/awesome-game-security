Ce projet est FindCrypt, un plugin IDA Pro qui identifie les constantes cryptographiques et les implémentations d'algorithmes dans les binaires désassemblés à l'aide de règles YARA. Il recherche des motifs d'octets connus tels que les S-boxes AES, les tables de permutation DES, les constantes de hachage SHA, les tables de recherche CRC et d'autres signatures cryptographiques, en annotant les emplacements trouvés avec les noms d'algorithmes. Ce plugin Python aide les analystes à localiser rapidement le code cryptographique dans les binaires. Il s'adresse aux analystes de malwares et aux reverse engineers qui identifient des implémentations cryptographiques dans des binaires inconnus.
