Этот проект — FindCrypt, плагин для IDA Pro, который с помощью правил YARA находит криптографические константы и реализации алгоритмов в дизассемблированных бинарных файлах. Он сканирует известные байтовые шаблоны S-боксов AES, таблиц перестановок DES, констант хешей SHA, таблиц CRC и другие криптографические сигнатуры, помечая найденные места названиями алгоритмов. Этот Python-плагин помогает аналитикам быстро находить криптографический код в бинарных файлах. Он предназначен для аналитиков вредоносного ПО и реверс-инженеров, выявляющих криптографические реализации в неизвестных бинарниках.
