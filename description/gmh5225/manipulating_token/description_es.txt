Este proyecto demuestra técnicas de manipulación de tokens de acceso de Windows para elevación de privilegios e impersonación. Muestra cómo robar, duplicar y modificar tokens de proceso para obtener privilegios elevados, suplantar a otros usuarios o eludir comprobaciones de control de acceso. Los ejemplos en C/C++ cubren el robo de tokens, la explotación de SeDebugPrivilege y métodos de impersonación de tokens. Está dirigido a investigadores de seguridad que estudian la elevación de privilegios en Windows mediante manipulación de tokens.
