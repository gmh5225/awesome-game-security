Этот проект демонстрирует техники манипуляции токенами доступа Windows для повышения привилегий и имперсонации. Он показывает, как похищать, дублировать и изменять токены процессов, чтобы получить повышенные привилегии, выдавать себя за других пользователей или обходить проверки контроля доступа. Примеры на C/C++ охватывают кражу токенов, эксплуатацию SeDebugPrivilege и методы имперсонации токенов. Материал предназначен для исследователей безопасности, изучающих повышение привилегий в Windows через манипуляцию токенами.
