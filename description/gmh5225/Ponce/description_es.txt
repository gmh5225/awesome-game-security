Este proyecto es Ponce, un complemento de IDA Pro para ejecución simbólica y ejecución con taint. Integra el framework de análisis binario dinámico Triton en IDA, permitiendo la ejecución simbólica desde el desensamblador para resolver restricciones de ruta, rastrear el flujo de datos contaminados (tainted) y generar entradas que alcancen rutas de código específicas. Está dirigido a investigadores de vulnerabilidades e ingenieros de ingeniería inversa que usan ejecución simbólica para el análisis automatizado en IDA Pro.
