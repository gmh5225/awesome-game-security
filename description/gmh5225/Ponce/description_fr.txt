Ce projet est Ponce, un plugin IDA Pro pour l'exécution symbolique et l'exécution par teinte (taint). Il intègre le framework d'analyse binaire dynamique Triton dans IDA, permettant l'exécution symbolique depuis le désassembleur afin de résoudre les contraintes de chemin, suivre le flux de données teintées et générer des entrées atteignant des chemins de code spécifiques. Il s'adresse aux chercheurs en vulnérabilités et aux ingénieurs en rétro-ingénierie qui utilisent l'exécution symbolique pour l'analyse automatisée dans IDA Pro.
