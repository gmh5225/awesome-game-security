Eine Kernel-Treiber-Verbergungsimplementierung, die Spuren aus MmUnloadedDrivers, PsLoadedModuleList, PiDDBCacheTable und Treiberobjektlisten entfernt.
Entwickelt für Stresstests unter Windows 11 21H2; löscht Treiberlade-Artefakte aus mehreren Kernel-Datenstrukturen, die Anti-Rootkit-Tools zur Erkennung nutzen.
