Une implémentation de masquage de pilote noyau qui supprime les traces dans MmUnloadedDrivers, PsLoadedModuleList, PiDDBCacheTable et les listes d'objets pilotes.
Conçue pour les tests de charge sous Windows 11 21H2, elle efface les artefacts de chargement de pilotes dans plusieurs structures de données noyau utilisées par les outils anti-rootkit pour la détection.
