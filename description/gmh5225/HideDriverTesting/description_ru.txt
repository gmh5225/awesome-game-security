Реализация сокрытия драйвера ядра, удаляющая следы из MmUnloadedDrivers, PsLoadedModuleList, PiDDBCacheTable и списков объектов драйверов.
Предназначена для стресс-тестирования на Windows 11 21H2; стирает артефакты загрузки драйвера из нескольких структур данных ядра, используемых антируткит-инструментами для обнаружения.
