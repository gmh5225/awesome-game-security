Una implementación de ocultación de controladores del kernel que elimina rastros de MmUnloadedDrivers, PsLoadedModuleList, PiDDBCacheTable y las listas de objetos de controlador.
Diseñada para pruebas de estrés en Windows 11 21H2, borra artefactos de carga de controladores de múltiples estructuras de datos del kernel usadas por herramientas anti-rootkit para la detección.
