Этот проект — отладчик на базе гипервизора, использующий виртуализацию Intel VT-x для отладки программ без срабатывания стандартного обнаружения отладчика. Он помещает цель под тонкий гипервизор, который перехватывает события выполнения через VM exit, обеспечивая точки останова, пошаговое выполнение и наблюдение за памятью, невидимые для техник антиотладки. Предназначен для реверс-инженеров, отлаживающих защищённое от отладки ПО с помощью аппаратно-ускоренной виртуализации.
