Este proyecto es un depurador basado en hipervisor que usa la virtualización Intel VT-x para depurar programas sin activar la detección habitual de depuradores. Coloca el objetivo bajo un hipervisor ligero que intercepta eventos de ejecución mediante VM exits, ofreciendo puntos de interrupción, ejecución paso a paso y vigilancia de memoria invisibles a las técnicas antidepuración. Está orientado a ingenieros de ingeniería inversa que depuran software protegido contra depuración mediante virtualización asistida por hardware.
