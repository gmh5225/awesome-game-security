Этот проект демонстрирует перехват окна оверлея NVIDIA GeForce Experience для отрисовки пользовательских игровых оверлеев. Он находит поверхность рендеринга оверлея NVIDIA и внедряет через неё пользовательские вызовы отрисовки DirectX, благодаря чему оверлеи отображаются через доверенный системный процесс и обходят обнаружение оверлеев античитами. Предназначен для исследователей игровой безопасности, изучающих hijacking оверлея NVIDIA как технику обхода античита.
