Este proyecto es una herramienta de kernel de Windows que elimina las devoluciones de llamada de notificación de proceso, hilo y carga de imagen registradas por controladores anticheat y de seguridad. Enumera la matriz de devoluciones de llamada PsSetCreateProcessNotifyRoutine y elimina selectivamente las entradas pertenecientes a controladores concretos, cegando de forma efectiva a esos controladores ante los eventos de creación de procesos. Está orientado a investigadores de kernel que estudian la manipulación de devoluciones de llamada para la evasión anticheat y la protección defensiva de callbacks.
