Этот проект — инструмент ядра Windows, который удаляет колбэки уведомлений о процессах, потоках и загрузке образов, зарегистрированные античитом и драйверами безопасности. Он перечисляет массив колбэков PsSetCreateProcessNotifyRoutine и выборочно удаляет записи, принадлежащие конкретным драйверам, фактически «ослепляя» эти драйверы для событий создания процессов. Предназначен для исследователей ядра, изучающих манипуляцию колбэками для обхода античита и оборонительной защиты колбэков.
