Загрузчик неподписанных драйверов уровня ядра, совместимый с Kernel Data Protection (KDP) в Windows 10.
Использует примитивы записи gdrv.sys для обхода Driver Signature Enforcement путём патчинга SeCiCallbacks, который инициализируется CiInitialize и используется SeValidateImageHeader для вызова CiValidateImageHeader.
