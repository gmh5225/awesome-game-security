Chargeur de pilote non signé en mode noyau, compatible avec Kernel Data Protection (KDP) sous Windows 10.
S'appuie sur les primitives d'écriture de gdrv.sys pour contourner Driver Signature Enforcement en patchant SeCiCallbacks, initialisé par CiInitialize et utilisé par SeValidateImageHeader pour appeler CiValidateImageHeader.
