Этот проект — коллекция смещений памяти Fortnite и адресов структур UE4 SDK, обновляемых для конкретных версий игры. Он предоставляет таблицы смещений для сущностей игроков, данных камеры, массивов костей, информации об оружии и глобальных переменных движка, используемых при разработке читов для Fortnite. Эти данные позволяют внешним и внутренним читам находить игровые объекты в памяти. Предназначен для исследователей безопасности игр, отслеживающих изменения раскладки памяти Fortnite и изучающих паттерны читов на основе смещений UE4.
