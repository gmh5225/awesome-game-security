Questo progetto è uno strumento di packing per i file di modifica di MapleStory. Impacchetta le risorse di gioco modificate (sprite, mappe, oggetti, skill) nel formato file WZ usato dai client MapleStory, gestendo compressione, cifratura e la corretta struttura delle directory per la distribuzione di mod lato client. È rivolto ai modder di MapleStory che preparano contenuti personalizzati per server privati o modifiche del client.
