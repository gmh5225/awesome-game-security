Этот проект — инструмент упаковки файлов модификаций для MapleStory. Он упаковывает изменённые игровые ресурсы (спрайты, карты, предметы, навыки) в формат WZ, используемый клиентами MapleStory, обрабатывая сжатие, шифрование и корректную структуру каталогов для распространения модов на стороне клиента. Предназначен для моддеров MapleStory, которые готовят пользовательский контент для приватных серверов или модификаций клиента.
