Este proyecto es una herramienta de empaquetado para archivos de modificación de MapleStory. Empaqueta recursos de juego modificados (sprites, mapas, objetos, habilidades) en el formato de archivos WZ que usan los clientes de MapleStory, gestionando la compresión, el cifrado y la estructura de directorios adecuada para la distribución de mods en el cliente. Está pensado para modders de MapleStory que empaquetan contenido personalizado para servidores privados o modificaciones del cliente.
