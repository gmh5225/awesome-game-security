Этот проект регистрирует события недопустимых page fault, генерируемые Vanguard (vgk.sys) — античит-драйвером ядра от Riot Games. Он отслеживает и записывает page fault, которые Vanguard намеренно вызывает в рамках своего механизма обнаружения, помогая исследователям понять, как Vanguard использует нарушения доступа к памяти для проверки целостности. Предназначен для исследователей античита, изучающих методы обнаружения Vanguard на основе page fault.
