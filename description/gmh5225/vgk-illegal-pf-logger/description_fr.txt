Ce projet journalise les événements de défauts de page illégaux générés par Vanguard (vgk.sys), le pilote noyau anti-triche de Riot Games. Il surveille et enregistre les défauts de page que Vanguard déclenche intentionnellement dans le cadre de son mécanisme de détection, aidant les chercheurs à comprendre comment Vanguard utilise les violations d'accès mémoire pour le contrôle d'intégrité. Il s'adresse aux chercheurs anti-triche qui étudient les techniques de détection de Vanguard basées sur les défauts de page.
