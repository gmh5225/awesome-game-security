Этот проект — Aurora, чит для CS2 (Counter-Strike 2) с функциями ESP, аимбота и визуальных модификаций. Он взаимодействует с движком Source 2 через доступ к системе схем и перечисление сущностей, извлекая данные игроков и отрисовывая оверлеи. Предназначен для исследователей игровой безопасности, изучающих реализации читов для CS2.
