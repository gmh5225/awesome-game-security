Este proyecto es una biblioteca de control para el controlador anticheat mhyprot2.sys de miHoYo, que proporciona una API conveniente para explotar su interfaz IOCTL vulnerable. Envuelve las operaciones de kernel de mhyprot2 —lectura/escritura de memoria de procesos, enumeración de módulos y terminación de procesos— en una interfaz C++ amigable para la explotación BYOVD. Está dirigido a investigadores BYOVD que usan mhyprot2 como primitiva de acceso al kernel.
