Этот проект — библиотека управления для античит-драйвера miHoYo mhyprot2.sys, предоставляющая удобный API для эксплуатации его уязвимого интерфейса IOCTL. Она оборачивает операции ядра mhyprot2 — чтение/запись памяти процессов, перечисление модулей и завершение процессов — в удобный C++-интерфейс для BYOVD-эксплуатации. Предназначена для исследователей BYOVD, использующих mhyprot2 как примитив доступа к ядру.
