Dieses Projekt ist eine Steuerungsbibliothek für den Anti-Cheat-Treiber mhyprot2.sys von miHoYo und stellt eine praktische API bereit, um dessen anfällige IOCTL-Schnittstelle auszunutzen. Es kapselt die Kernel-Operationen von mhyprot2 — darunter Lesen/Schreiben von Prozessspeicher, Modulaufzählung und Prozessbeendigung — in eine benutzerfreundliche C++-Schnittstelle für BYOVD-Exploitation. Es richtet sich an BYOVD-Forscher, die mhyprot2 als Kernel-Zugriffsprimitive nutzen.
