Ce projet est une bibliothèque de contrôle pour le pilote anti-triche mhyprot2.sys de miHoYo, fournissant une API pratique pour exploiter son interface IOCTL vulnérable. Il encapsule les opérations noyau de mhyprot2 — lecture/écriture de mémoire de processus, énumération de modules et terminaison de processus — dans une interface C++ conviviale destinée à l'exploitation BYOVD. Il s'adresse aux chercheurs BYOVD qui utilisent mhyprot2 comme primitive d'accès au noyau.
