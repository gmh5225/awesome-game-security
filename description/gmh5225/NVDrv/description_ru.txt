Этот проект — библиотека для эксплуатации драйвера ядра NVIDIA (nvoclock/nvlddmkm) с целью произвольного чтения/записи физической памяти из пользовательского режима. Она отправляет специально сформированные IOCTL драйверу NVIDIA для чтения и записи физической памяти, предоставляя примитив BYOVD, который можно использовать для отображения драйверов, патчинга ядра или обхода античита без загрузки собственного драйвера. Библиотека на C++ оборачивает уязвимый интерфейс IOCTL. Предназначена для исследователей безопасности ядра, изучающих уязвимости драйверов NVIDIA и эксплуатацию BYOVD.
