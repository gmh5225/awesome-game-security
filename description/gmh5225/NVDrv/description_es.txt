Este proyecto es una biblioteca para explotar el controlador del kernel de NVIDIA (nvoclock/nvlddmkm) y obtener lectura/escritura arbitraria de memoria física desde modo usuario. Envía IOCTL manipulados al controlador de NVIDIA para leer y escribir memoria física, proporcionando una primitiva BYOVD que puede usarse para mapeo de controladores, parcheo del kernel o bypass de anti-cheat sin cargar un controlador personalizado. La biblioteca C++ encapsula la interfaz IOCTL vulnerable. Está dirigida a investigadores de seguridad del kernel que estudian vulnerabilidades de controladores NVIDIA y la explotación BYOVD.
