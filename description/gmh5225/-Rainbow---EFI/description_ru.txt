Этот репозиторий объединяет удобную для Visual Studio и MSBuild среду EDK II с отдельным проектом UEFI-драйвера `rainbow` и отладочными ресурсами — образами OVMF, ISO оболочки UEFI и ROM-файлами для локального тестирования.
Согласно архивному README, полезная нагрузка `rainbow.efi` перехватывает `ExitBootServices`, обходит ранние структуры загрузки Windows вроде `OslLoaderBlock`, затем перехватывает `IopLoadDriver`, чтобы применить логику спуфинга, после чего снимает собственный хук.
Включённая структура проекта показывает как код драйвера на этапе загрузки, так и вспомогательный слой библиотек UEFI, необходимый для сборки и отладки EFI-компонента, а не только отдельный бинарный дроп.
Материал в первую очередь полезен исследователям прошивок и загрузки Windows, которые хотят изучить, как собираются UEFI-драйверы, как они загружаются из EFI shell и как используются для изменения поведения при переходе в ядро Windows.
