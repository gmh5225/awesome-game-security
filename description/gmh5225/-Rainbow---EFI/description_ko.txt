이 저장소는 Visual Studio 및 MSBuild에 친화적인 EDK II 환경과 별도의 `rainbow` UEFI 드라이버 프로젝트, 그리고 OVMF 이미지, UEFI 셸 ISO, ROM 파일과 같은 로컬 테스트용 디버거 자산을 결합합니다.
보관된 README에 따르면 `rainbow.efi` 페이로드는 `ExitBootServices`를 후킹하고 `OslLoaderBlock`과 같은 초기 Windows 부트 구조를 탐색한 뒤, `IopLoadDriver`를 후킹해 스푸핑 로직을 적용하고 자신의 후크를 제거합니다.
포함된 프로젝트 구조는 부트 단계 드라이버 코드와, 단독 바이너리 배포가 아니라 EFI 구성 요소를 빌드하고 디버깅하는 데 필요한 지원 UEFI 라이브러리 계층을 모두 보여 줍니다.
UEFI 드라이버가 어떻게 빌드되고 EFI 셸에서 로드되며 Windows 커널로 넘어가는 과정에서 동작을 어떻게 바꾸는지 연구하려는 펌웨어 및 Windows 부트 연구자에게 주로 유용합니다.
