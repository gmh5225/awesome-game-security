本仓库将便于 Visual Studio 与 MSBuild 使用的 EDK II 环境，与独立的 `rainbow` UEFI 驱动程序项目以及 OVMF 镜像、UEFI shell ISO、ROM 文件等本地调试资源组合在一起。
根据归档 README，`rainbow.efi` 载荷会挂钩 `ExitBootServices`，遍历 `OslLoaderBlock` 等早期 Windows 启动结构，再挂钩 `IopLoadDriver` 以应用欺骗逻辑，随后移除自身挂钩。
所含工程结构同时展示了启动阶段驱动代码，以及构建与调试 EFI 组件所需的配套 UEFI 库层，而不仅是单独的二进制投放。
它主要面向固件与 Windows 启动研究人员，便于研究 UEFI 驱动如何构建、如何从 EFI shell 加载，以及如何在进入 Windows 内核的过渡阶段改变行为。
