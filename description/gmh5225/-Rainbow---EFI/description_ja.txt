このリポジトリは、Visual Studio および MSBuild 向けの EDK II 環境と、独立した `rainbow` UEFI ドライバープロジェクト、さらに OVMF イメージ、UEFI シェル ISO、ROM ファイルなどのローカルテスト用デバッガー資産を組み合わせています。
アーカイブされた README によれば、`rainbow.efi` ペイロードは `ExitBootServices` をフックし、`OslLoaderBlock` などの初期 Windows ブート構造を走査したうえで `IopLoadDriver` をフックしてスプーフィング処理を適用し、その後自身のフックを解除します。
含まれるプロジェクト構成には、ブート段階のドライバーコードと、単体バイナリの投下だけでなく EFI コンポーネントのビルドとデバッグに必要な補助 UEFI ライブラリ層の両方が示されています。
主に、UEFI ドライバーのビルド方法、EFI シェルからの読み込み、Windows カーネルへの移行時に挙動を変更する手法を調べたいファームウェアおよび Windows ブート研究者向けです。
