Этот проект — Windows kernel driver mapper, который загружает неподписанные драйверы, эксплуатируя конкретный уязвимый подписанный драйвер. Он выполняет ручное отображение PE, включая копирование секций, разрешение импортов, обработку релокаций и вызов точки входа через примитив доступа к ядру уязвимого драйвера. Предназначен для исследователей ядра, изучающих реализации отображения драйверов.
