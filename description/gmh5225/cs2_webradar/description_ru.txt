Этот проект посвящён читу с радаром в браузере.
Он в основном написан на C++ и JavaScript и сосредоточен на конвейерах ассетов, моддинге и анализе памяти.
В первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:cs2.
