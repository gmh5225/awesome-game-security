Этот проект архивирует декомпилированный dump PUBG Mobile 1.6 с акцентом на libtersafe.so и libUE4.so, полученные из крупной распаковки APK.
Он сохраняет исторические нативные символы/функции мобильного античита и Unreal из этой «мёртвой» версии игры для исследовательских целей.
В первую очередь полезен исследователям безопасности мобильных игр, изучающим нативные библиотеки античита PUBGM / Tencent.
