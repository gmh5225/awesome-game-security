이 프로젝트는 dxgkrnl의 NtDxgkGetTrackedWorkloadStatistics를 가로챈 뒤, 같은 채널로 간단한 메모리 및 그리기 작업을 수행하는 커널·유저 모드 통신 샘플입니다.
드라이버는 해당 dxgkrnl 내보내기를 작은 절대 점프 스텁으로 덮어쓰고, MmCopyVirtualMemory와 KeStackAttachProcess 기반의 읽기/쓰기 헬퍼를 제공하며, NtUserGetDC, NtGdiPatBlt, NtGdiCreateSolidBrush 같은 win32k GDI 루틴을 해석해 커널 컨텍스트에서 상자를 그립니다.
유저 모드 클라이언트는 win32u.dll을 통해 NtDxgkGetTrackedWorkloadStatistics를 호출하고, 모듈 베이스 요청이나 사각형 그리기가 가능한 NULL_MEMORY 구조를 전달하며, 후킹된 그래픽 시스템 호출을 명령 디스패처로 사용합니다.
그래픽 인접 통신 후킹, 임시 읽기/쓰기 서비스, win32k 내보내기를 통한 커널 보조 오버레이 렌더링을 연구하는 Windows 커널 연구자에게 주로 유용합니다.
