Этот проект — nullmap, загрузчик (mapper) драйверов Windows, который стирает следы отображённого драйвера после выполнения. Он вручную отображает неподписанный драйвер, запускает его точку входа, а затем очищает окружение, обнуляя заголовки, удаляя выделения из пула и отвязывая ссылки, оставляя минимум криминалистических следов отображённого драйвера. Предназначен для исследователей ядра, изучающих антикриминалистические техники отображения драйверов.
