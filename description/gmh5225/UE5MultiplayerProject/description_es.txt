Este proyecto es una plantilla de juego multijugador creada con Unreal Engine 5 que demuestra la implementación de jugabilidad en red. Cubre el sistema de replicación de UE5, la gestión de sesiones de jugador, la configuración de servidor dedicado, la integración del Gameplay Ability System y los widgets de interfaz multijugador. Está dirigido a desarrolladores de UE5 que aprenden patrones de implementación de juegos multijugador.
