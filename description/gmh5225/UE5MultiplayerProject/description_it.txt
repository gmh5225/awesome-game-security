Questo progetto è un template di gioco multiplayer realizzato con Unreal Engine 5 che dimostra l'implementazione del gameplay in rete. Copre il sistema di replica di UE5, la gestione delle sessioni giocatore, la configurazione del server dedicato, l'integrazione del Gameplay Ability System e i widget UI multiplayer. È rivolto agli sviluppatori UE5 che studiano i pattern di implementazione dei giochi multiplayer.
