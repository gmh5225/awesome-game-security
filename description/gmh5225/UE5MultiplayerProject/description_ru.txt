Этот проект — шаблон многопользовательской игры на Unreal Engine 5, демонстрирующий реализацию сетевого геймплея. Он охватывает систему репликации UE5, управление сессиями игроков, настройку выделенного сервера, интеграцию Gameplay Ability System и виджеты многопользовательского UI. Предназначен для разработчиков UE5, изучающих паттерны реализации многопользовательских игр.
