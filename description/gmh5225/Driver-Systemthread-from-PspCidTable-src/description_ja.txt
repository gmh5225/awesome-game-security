このリポジトリには、`PspCidTable` からシステムスレッドまたは関連するプロセスハンドルを削除するチュートリアルに付随するソースコードが含まれています。
実装は、`ExRemoveHandleTable` による対象プロセスハンドルテーブルの削除と、`PspCidTable` からのプロセス／スレッドハンドル破棄に加え、即時のバグチェックを避けるための選択的な CID フィールドのゼロクリアという、二つの連動した手法を扱います。
`EPROCESS`、`ETHREAD`、`PspCidTable`、および関連ルーチン向けのビルド固有オフセットを大量に含んでおり、汎用の再利用可能ライブラリというより、オフセット駆動の研究用リファレンスとしての性格が強いです。
主に、特定のカーネルビルドにおけるプロセスやシステムスレッドの隠蔽を目的としたハンドルテーブルおよび CID テーブル操作手法を学ぶ Windows カーネル研究者向けです。
