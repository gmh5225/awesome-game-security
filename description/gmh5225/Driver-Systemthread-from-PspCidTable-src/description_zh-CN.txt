本仓库包含一篇教程的配套源代码，主题是从 `PspCidTable` 中移除系统线程或相关进程句柄。
实现涵盖两种相互关联的方法：通过 `ExRemoveHandleTable` 移除目标进程句柄表，以及从 `PspCidTable` 销毁进程或线程句柄，同时清零选定的 CID 字段以避免立即触发 bug check。
项目包含大量针对特定构建的 `EPROCESS`、`ETHREAD`、`PspCidTable` 及相关例程偏移，因此更适合作为偏移驱动的研究参考，而非通用可复用库。
它主要面向研究句柄表与 CID 表操纵技术的 Windows 内核研究者，用于在特定内核构建上隐藏进程或系统线程。
