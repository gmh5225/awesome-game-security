本倉庫包含一篇教學的配套原始碼，主題是從 `PspCidTable` 中移除系統執行緒或相關行程控制代碼。
實作涵蓋兩種相互關聯的做法：透過 `ExRemoveHandleTable` 移除目標行程控制代碼表，以及從 `PspCidTable` 銷毀行程或執行緒控制代碼，同時將選定的 CID 欄位清零以避免立即觸發 bug check。
專案包含大量針對特定建置的 `EPROCESS`、`ETHREAD`、`PspCidTable` 及相關常式偏移，因此更適合作為偏移驅動的研究參考，而非通用可重用函式庫。
它主要面向研究控制代碼表與 CID 表操縱技術的 Windows 核心研究者，用於在特定核心建置上隱藏行程或系統執行緒。
