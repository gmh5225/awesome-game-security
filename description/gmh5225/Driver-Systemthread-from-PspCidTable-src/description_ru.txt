Этот репозиторий содержит исходный код к учебному материалу о удалении системного потока или связанных handle процесса из `PspCidTable`.
Реализация охватывает два связанных подхода: удаление таблицы handle целевого процесса через `ExRemoveHandleTable` и уничтожение handle процесса или потока в `PspCidTable` с обнулением выбранных полей CID, чтобы избежать немедленного bug check.
Включён большой набор смещений, зависящих от конкретной сборки, для `EPROCESS`, `ETHREAD`, `PspCidTable` и связанных функций, поэтому проект скорее служит offset-ориентированным исследовательским справочником, чем универсальной переиспользуемой библиотекой.
Он в первую очередь полезен исследователям ядра Windows, изучающим техники манипуляции таблицами handle и CID для сокрытия процессов или системных потоков на конкретных сборках ядра.
