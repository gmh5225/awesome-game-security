Questo repository contiene il codice sorgente che accompagna un tutorial sulla rimozione di un thread di sistema o di handle di processo correlati da `PspCidTable`.
L'implementazione copre due approcci collegati: rimuovere la tabella degli handle di un processo target tramite `ExRemoveHandleTable` e distruggere gli handle di processo o thread da `PspCidTable`, azzerando anche campi CID selezionati per evitare un bug check immediato.
Include un ampio insieme di offset specifici per build relativi a `EPROCESS`, `ETHREAD`, `PspCidTable` e routine correlate, il che rende il progetto più un riferimento di ricerca guidato dagli offset che una libreria generica riutilizzabile.
È soprattutto utile ai ricercatori del kernel Windows che studiano tecniche di manipolazione delle tabelle degli handle e della tabella CID per nascondere processi o thread di sistema su build del kernel specifiche.
