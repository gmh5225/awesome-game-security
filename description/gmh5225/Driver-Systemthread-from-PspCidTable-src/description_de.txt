Dieses Repository enthält den Quellcode zu einem Tutorial über das Entfernen eines Systemthreads oder zugehöriger Prozess-Handles aus `PspCidTable`.
Die Implementierung behandelt zwei verknüpfte Ansätze: das Entfernen der Handle-Tabelle eines Zielprozesses über `ExRemoveHandleTable` sowie das Zerstören von Prozess- oder Thread-Handles aus `PspCidTable` bei gleichzeitigem Nullsetzen ausgewählter CID-Felder, um einen sofortigen Bugcheck zu vermeiden.
Enthalten ist eine große Menge build-spezifischer Offsets für `EPROCESS`, `ETHREAD`, `PspCidTable` und zugehörige Routinen; das Projekt ist daher eher eine offset-getriebene Forschungsreferenz als eine generische, wiederverwendbare Bibliothek.
Es ist vor allem für Windows-Kernel-Forscher nützlich, die Techniken zur Manipulation von Handle- und CID-Tabellen zum Verstecken von Prozessen oder Systemthreads auf bestimmten Kernel-Builds untersuchen.
