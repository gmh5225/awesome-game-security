Este repositorio contiene el código fuente que acompaña un tutorial sobre cómo eliminar un hilo del sistema o handles de proceso relacionados de `PspCidTable`.
La implementación cubre dos enfoques vinculados: quitar la tabla de handles de un proceso objetivo mediante `ExRemoveHandleTable` y destruir handles de proceso o de hilo desde `PspCidTable`, además de poner a cero campos CID seleccionados para evitar un bug check inmediato.
Incluye un amplio conjunto de offsets específicos por build para `EPROCESS`, `ETHREAD`, `PspCidTable` y rutinas relacionadas, por lo que el proyecto es más una referencia de investigación guiada por offsets que una biblioteca genérica reutilizable.
Resulta especialmente útil para investigadores del kernel de Windows que estudian técnicas de manipulación de tablas de handles y de la tabla CID para ocultar procesos o hilos del sistema en builds de kernel concretas.
