Ce dépôt contient le code source accompagnant un tutoriel sur la suppression d'un thread système ou de handles de processus associés depuis `PspCidTable`.
L'implémentation couvre deux approches liées : retirer la table de handles d'un processus cible via `ExRemoveHandleTable`, et détruire les handles de processus ou de thread dans `PspCidTable` tout en mettant à zéro certains champs CID pour éviter un bug check immédiat.
Il inclut un large ensemble d'offsets spécifiques à des builds pour `EPROCESS`, `ETHREAD`, `PspCidTable` et les routines associées, ce qui en fait davantage une référence de recherche guidée par les offsets qu'une bibliothèque générique réutilisable.
Il s'adresse surtout aux chercheurs du noyau Windows qui étudient les techniques de manipulation des tables de handles et de la table CID pour masquer des processus ou des threads système sur des builds de noyau précis.
