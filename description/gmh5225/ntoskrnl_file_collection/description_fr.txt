Ce projet rassemble plusieurs versions des binaires Windows ntoskrnl pour la comparaison et la recherche.
Il s’agit d’un corpus de référence plutôt que d’un outil d’analyse, destiné à la rétro-ingénierie du noyau et aux workflows de diff entre versions.
Il est surtout utile aux chercheurs du noyau Windows qui étudient les évolutions de ntoskrnl d’un build à l’autre.
