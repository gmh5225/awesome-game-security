Этот проект собирает несколько версий бинарников Windows ntoskrnl для сравнения и исследований.
Это скорее эталонный корпус, чем инструмент анализа; он предназначен для реверс-инжиниринга ядра и рабочих процессов сравнения версий.
Он в первую очередь полезен исследователям ядра Windows, изучающим изменения ntoskrnl между сборками.
