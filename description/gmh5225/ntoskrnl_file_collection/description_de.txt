Dieses Projekt sammelt mehrere Versionen von Windows-ntoskrnl-Binärdateien zum Vergleich und zur Forschung.
Es ist eher ein Referenzkorpus als ein Analysetool und richtet sich an Kernel-Reverse-Engineering sowie Versions-Diff-Workflows.
Es ist vor allem für Windows-Kernel-Forscher nützlich, die ntoskrnl-Änderungen über Builds hinweg untersuchen.
