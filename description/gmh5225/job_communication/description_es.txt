Este proyecto es una pequeña prueba de concepto de comunicación no documentada de ring0 a ring3 mediante NtQueryInformationJobObject.
Las notas del lado del kernel muestran la idea de comprobar el campo Job del proceso actual, resolver el server silo del job con PsGetJobServerSilo y copiar datos desde ServerSiloGlobals o UserSharedData al búfer de salida devuelto al modo usuario.
El ejemplo en modo usuario llama después a NtQueryInformationJobObject con JobObjectReserved17Information e interpreta los campos SILO_USER_SHARED_DATA devueltos como el resultado de la comunicación oculta.
Resulta especialmente útil para investigadores de internals de Windows que estudian rutas de comunicación poco conocidas basadas en job objects y silos que evitan las interfaces IOCTL de dispositivo estándar.
