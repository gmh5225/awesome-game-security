Этот проект — небольшой proof of concept недокументированной связи ring0 с ring3 через NtQueryInformationJobObject.
Заметки на стороне ядра показывают идею проверки поля Job текущего процесса, разрешения server silo задания через PsGetJobServerSilo и копирования данных из ServerSiloGlobals или UserSharedData в выходной буфер, возвращаемый в пользовательский режим.
Пример в пользовательском режиме затем вызывает NtQueryInformationJobObject с JobObjectReserved17Information и интерпретирует возвращённые поля SILO_USER_SHARED_DATA как результат скрытой связи.
Он в основном полезен исследователям внутренних механизмов Windows, изучающим малоизвестные пути связи на основе job object и silo, обходящие стандартные интерфейсы IOCTL устройств.
