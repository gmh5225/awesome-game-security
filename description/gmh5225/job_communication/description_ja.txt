本プロジェクトは、非公開の NtQueryInformationJobObject を用いた ring0 から ring3 への通信を示す小さな概念実証です。
カーネル側のメモでは、現在のプロセスの Job フィールドを確認し、PsGetJobServerSilo でジョブの server silo を解決し、ServerSiloGlobals または UserSharedData からユーザーモードへ返される出力バッファへデータをコピーするという着想が示されています。
ユーザーモードのサンプルは、JobObjectReserved17Information を指定して NtQueryInformationJobObject を呼び出し、返された SILO_USER_SHARED_DATA のフィールドを隠蔽通信の結果として解釈します。
標準のデバイス IOCTL インタフェースを回避する、あまり知られていないジョブオブジェクトおよび silo ベースの通信経路を研究する Windows 内部機構の研究者にとって特に有用です。
