本專案是一個小型概念驗證，示範如何利用未公開的 NtQueryInformationJobObject 實作 ring0 到 ring3 通訊。
核心態筆記說明了如下思路：檢查目前處理程序的 Job 欄位，透過 PsGetJobServerSilo 解析工作物件的 server silo，再從 ServerSiloGlobals 或 UserSharedData 將資料複製到回傳給使用者模式的輸出緩衝區。
使用者模式範例接著以 JobObjectReserved17Information 呼叫 NtQueryInformationJobObject，並將回傳的 SILO_USER_SHARED_DATA 欄位解讀為隱藏通訊結果。
此專案主要供 Windows 內部機制研究人員使用，用以研究繞過標準裝置 IOCTL 介面的冷門工作物件與 silo 通訊路徑。
