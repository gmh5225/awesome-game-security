本项目是一个小型概念验证，演示如何利用未公开的 NtQueryInformationJobObject 实现 ring0 到 ring3 通信。
内核侧笔记说明了如下思路：检查当前进程的 Job 字段，通过 PsGetJobServerSilo 解析作业的 server silo，再从 ServerSiloGlobals 或 UserSharedData 将数据复制到返回给用户态的输出缓冲区。
用户态示例随后以 JobObjectReserved17Information 调用 NtQueryInformationJobObject，并将返回的 SILO_USER_SHARED_DATA 字段解释为隐藏通信结果。
该项目主要面向 Windows 内部机制研究人员，用于研究绕过标准设备 IOCTL 接口的冷门作业对象与 silo 通信路径。
