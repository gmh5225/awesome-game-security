이 프로젝트는 문서화되지 않은 NtQueryInformationJobObject를 이용해 ring0에서 ring3으로 통신하는 작은 개념 증명입니다.
커널 측 노트는 현재 프로세스의 Job 필드를 확인하고, PsGetJobServerSilo로 잡의 server silo를 해석한 뒤, ServerSiloGlobals 또는 UserSharedData에서 사용자 모드로 반환되는 출력 버퍼로 데이터를 복사한다는 아이디어를 보여 줍니다.
사용자 모드 샘플은 이어서 JobObjectReserved17Information으로 NtQueryInformationJobObject를 호출하고, 반환된 SILO_USER_SHARED_DATA 필드를 숨겨진 통신 결과로 해석합니다.
표준 장치 IOCTL 인터페이스를 우회하는 잘 알려지지 않은 잡 오브젝트 및 silo 기반 통신 경로를 연구하는 Windows 내부 구조 연구자에게 주로 유용합니다.
