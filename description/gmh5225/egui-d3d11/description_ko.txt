egui(즉시 모드 GUI 프레임워크)를 Direct3D 11 서피스에 직접 렌더링하는 Rust 라이브러리로, D3D11 Present 호출을 후킹하여 인게임 오버레이 메뉴를 구축하도록 설계되었습니다.
이 라이브러리는 egui 메시 출력을 D3D11 정점/인덱스 버퍼로 변환하고, 텍스처 삼각형 렌더링용 HLSL 셰이더를 컴파일하며, egui 폰트 아틀라스의 텍스처 업로드를 관리하고, Win32 입력(키보드, 마우스)을 egui 입력 형식으로 변환합니다. 또한 상태 백업/복원을 통해 게임의 렌더링 파이프라인을 손상시키지 않도록 합니다.
주로 Present 후킹을 통해 DirectX 11 게임에 주입되는 ImGui 스타일 오버레이 메뉴를 구축하거나 연구하는 게임 치트 개발자와 보안 연구자에게 유용합니다.
