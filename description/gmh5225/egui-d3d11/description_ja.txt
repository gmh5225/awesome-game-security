egui（即時モード GUI フレームワーク）を Direct3D 11 サーフェス上に直接描画する Rust ライブラリで、D3D11 の Present 呼び出しをフックしてゲーム内オーバーレイメニューを構築するために設計されています。
本ライブラリは egui のメッシュ出力を D3D11 の頂点/インデックスバッファに変換し、テクスチャ付き三角形描画用の HLSL シェーダをコンパイルし、egui のフォントアトラスからのテクスチャアップロードを管理し、Win32 入力（キーボード、マウス）を egui の入力形式に変換します。また、状態のバックアップ/復元によりゲームのレンダリングパイプラインを壊さないようにしています。
主に、Present フック経由で DirectX 11 ゲームに注入される ImGui スタイルのオーバーレイメニューを構築・研究するゲームチート開発者やセキュリティ研究者向けです。
