Libreria Rust che renderizza egui (un framework GUI in immediate mode) direttamente su superfici Direct3D 11, progettata per creare menu overlay in-game agganciando la chiamata Present di D3D11.
La libreria converte l'output mesh di egui in buffer vertex/index D3D11, compila shader HLSL per il rendering di triangoli texturizzati, gestisce il caricamento delle texture dall'atlas dei font di egui e traduce l'input Win32 (tastiera, mouse) nel formato di input di egui, con backup/ripristino dello stato per evitare di corrompere la pipeline di rendering del gioco.
È soprattutto utile per sviluppatori di cheat e ricercatori di sicurezza che costruiscono o studiano menu overlay in stile ImGui iniettati in giochi DirectX 11 tramite hook Present.
