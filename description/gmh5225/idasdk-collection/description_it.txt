Questo progetto è una raccolta di SDK di IDA Pro di più versioni, che archivia i rilasci dell'SDK per riferimento storico e per lo sviluppo di plugin retrocompatibili. Fornisce header, librerie ed esempi provenienti da diverse versioni dell'SDK di IDA. È rivolto agli sviluppatori di plugin per IDA Pro che devono mantenere la compatibilità tra le versioni di IDA.
