Этот проект — коллекция SDK IDA Pro разных версий, архивирующая выпуски SDK для исторической справки и разработки обратно совместимых плагинов. Он предоставляет заголовочные файлы, библиотеки и примеры из разных версий IDA SDK. Предназначен для разработчиков плагинов IDA Pro, которым нужно поддерживать совместимость между версиями IDA.
