Ce projet est une collection de SDK IDA Pro couvrant plusieurs versions, archivant les releases du SDK pour référence historique et pour le développement de plugins rétrocompatibles. Il fournit des en-têtes, des bibliothèques et des exemples issus de différentes versions du SDK IDA. Il s'adresse aux développeurs de plugins IDA Pro qui doivent maintenir la compatibilité entre les versions d'IDA.
