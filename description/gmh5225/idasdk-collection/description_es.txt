Este proyecto es una colección de SDK de IDA Pro de múltiples versiones, que archiva las versiones del SDK como referencia histórica y para el desarrollo de plugins compatibles con versiones anteriores. Proporciona encabezados, bibliotecas y ejemplos de diferentes versiones del SDK de IDA. Está dirigido a desarrolladores de plugins de IDA Pro que mantienen la compatibilidad entre versiones de IDA.
