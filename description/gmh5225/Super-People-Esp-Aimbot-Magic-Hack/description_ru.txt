Этот проект — чит для Super People (королевская битва), предоставляющий ESP, аимбот и модификацию геймплея. Он читает данные сущностей из процесса на базе Unreal Engine, чтобы отображать позиции игроков, здоровье и дистанцию, а также обеспечивает наведение аимбота и другие боевые преимущества. Чит на C++ взаимодействует со структурами UE4 SDK игры. Он предназначен для исследователей игровой безопасности, изучающих реализации читов для battle royale на базе UE4.
