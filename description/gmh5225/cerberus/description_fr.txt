Ce projet est un moteur de protection de code basé sur une machine virtuelle Win32, construit autour d'une machine virtuelle x86 personnalisée appelée ChaosVm. Il traduit les instructions x86 natives en bytecode propriétaire et les exécute via un émulateur de CPU logiciel, offrant une virtualisation de code pour les binaires PE. La base de code C comprend un désassembleur et un assembleur x86 complets, un analyseur binaire pour la récupération des procédures et du flot d'instructions, l'analyse et le patching de fichiers PE, la vérification d'intégrité CRC32, ainsi qu'une interface graphique Qt pour appliquer la protection. Il s'adresse aux chercheurs en protection logicielle qui étudient les moteurs d'obfuscation basés sur VM et les techniques de virtualisation de code.
