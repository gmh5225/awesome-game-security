Этот проект — движок защиты кода на основе Win32 VM, построенный вокруг собственной x86-виртуальной машины ChaosVm. Он переводит нативные инструкции x86 в проприетарный байткод и выполняет их через программный эмулятор CPU, обеспечивая виртуализацию кода для PE-бинарников. Кодовая база на C включает полноценный дизассемблер и ассемблер x86, бинарный анализатор для восстановления процедур и потока инструкций, разбор и патчинг PE-файлов, проверку целостности CRC32 и GUI-фронтенд на Qt для применения защиты. Предназначен для исследователей защиты ПО, изучающих движки обфускации на базе VM и техники виртуализации кода.
