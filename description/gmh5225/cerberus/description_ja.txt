本プロジェクトは、ChaosVm と呼ばれる独自の x86 仮想マシンを中核とする Win32 VM ベースのコード保護エンジンです。ネイティブ x86 命令を独自バイトコードへ変換し、ソフトウェア CPU エミュレータ上で実行することで、PE バイナリ向けのコード仮想化を実現します。C コードベースには、完全な x86 ディスアセンブラ／アセンブラ、手続きと命令フロー復元用のバイナリ解析器、PE ファイルの解析とパッチ、CRC32 完全性チェック、保護適用用の Qt ベース GUI フロントエンドが含まれます。VM ベースの難読化エンジンやコード仮想化手法を研究するソフトウェア保護研究者を対象としています。
