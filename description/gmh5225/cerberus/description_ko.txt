이 프로젝트는 ChaosVm이라는 커스텀 x86 가상 머신을 중심으로 한 Win32 VM 기반 코드 보호 엔진입니다. 네이티브 x86 명령을 전용 바이트코드로 변환하고 소프트웨어 CPU 에뮬레이터를 통해 실행하여 PE 바이너리에 코드 가상화를 제공합니다. C 코드베이스에는 완전한 x86 디스어셈블러와 어셈블러, 프로시저 및 명령 흐름 복구용 바이너리 분석기, PE 파일 파싱 및 패치, CRC32 무결성 검사, 보호 적용을 위한 Qt 기반 GUI 프론트엔드가 포함됩니다. VM 기반 난독화 엔진과 코드 가상화 기법을 연구하는 소프트웨어 보호 연구자를 대상으로 합니다.
