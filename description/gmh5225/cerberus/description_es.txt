Este proyecto es un motor de protección de código basado en una máquina virtual Win32, construido en torno a una máquina virtual x86 personalizada llamada ChaosVm. Traduce instrucciones x86 nativas a bytecode propietario y las ejecuta mediante un emulador de CPU por software, proporcionando virtualización de código para binarios PE. El código en C incluye un desensamblador y ensamblador x86 completos, un analizador binario para la recuperación de procedimientos y del flujo de instrucciones, análisis y parcheo de archivos PE, comprobación de integridad CRC32 y una interfaz gráfica basada en Qt para aplicar la protección. Está dirigido a investigadores de protección de software que estudian motores de ofuscación basados en VM y técnicas de virtualización de código.
