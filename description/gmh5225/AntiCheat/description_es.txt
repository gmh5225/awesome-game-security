Este proyecto es una implementación de referencia de un sistema anticheat que demuestra técnicas habituales de protección de juegos. Incluye verificación de integridad de procesos, escaneo de módulos, detección de patrones en memoria, detección de depuradores, supervisión de overlays y comunicación con un controlador para comprobaciones a nivel de kernel. El código en C/C++ sirve como referencia de estudio para comprender cómo se estructuran los sistemas anticheat. Está dirigido a desarrolladores de anticheat e investigadores de seguridad de juegos que estudian la arquitectura de sistemas de protección.
