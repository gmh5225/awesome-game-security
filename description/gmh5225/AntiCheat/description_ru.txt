Этот проект — эталонная реализация античит-системы, демонстрирующая распространённые техники защиты игр. В неё входят проверка целостности процессов, сканирование модулей, обнаружение шаблонов в памяти, обнаружение отладчиков, мониторинг оверлеев и взаимодействие с драйвером для проверок на уровне ядра. Кодовая база на C/C++ служит учебным справочником для понимания структуры античит-систем. Проект ориентирован на разработчиков античита и исследователей безопасности игр, изучающих архитектуру систем защиты.
