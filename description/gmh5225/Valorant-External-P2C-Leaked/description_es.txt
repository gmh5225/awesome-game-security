Este proyecto es el código fuente filtrado de un cheat externo de pago (P2C) para Valorant que opera fuera del proceso del juego mediante un controlador de kernel para el acceso a memoria. Implementa ESP, aimbot y triggerbot leyendo el estado del juego de Unreal Engine por acceso externo a memoria mientras el anti-cheat Vanguard está en ejecución. La base de código en C++ muestra la arquitectura típica de los cheats externos comerciales de Valorant. Está orientado a investigadores de anti-cheat que estudian patrones de diseño de cheats P2C y técnicas de evasión de Vanguard.
