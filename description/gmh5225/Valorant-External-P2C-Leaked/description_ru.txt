Этот проект — утёкший исходный код внешнего платного чита (P2C) для Valorant, который работает вне процесса игры и использует драйвер ядра для доступа к памяти. Он реализует ESP, аимбот и триггербот, читая состояние игры Unreal Engine через внешний доступ к памяти при работающем античите Vanguard. Кодовая база на C++ демонстрирует типичную архитектуру коммерческих внешних читов для Valorant. Материал предназначен для исследователей античита, изучающих паттерны проектирования P2C-читов и техники обхода Vanguard.
