Dieses Projekt konzentriert sich auf echte Firmware-Tarnung über die VMD-Funktion des Motherboards (Pcileech-DMA).
Für eine ordnungsgemäße Treiberinitialisierung und Geräteerkennung kann eine Neuinstallation von Windows erforderlich sein.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Cheat-/DMA-Bereich untersuchen.
