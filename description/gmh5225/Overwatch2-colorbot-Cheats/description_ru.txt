Основанный на Python colorbot для Overwatch 2, который с помощью определения цвета пикселей на экране распознаёт контуры врагов (в настройках игры заданы фиолетовым), затем вычисляет дельты прицеливания и отправляет команды перемещения мыши на Arduino Leonardo по последовательному порту на скорости 115200 бод, чтобы обойти обнаружение ввода мыши.
Прошивка Arduino реализует вызовы HID Mouse.move() с разбиением больших перемещений на шаги по 127 единиц и поддерживает команды shoot/silent-aim/reset, получаемые по последовательной линии, делая ввод неотличимым от легитимных аппаратных событий мыши.
В первую очередь полезен исследователям игровой безопасности, изучающим аппаратную подмену ввода через Arduino HID и техники color aimbot на основе пикселей.
