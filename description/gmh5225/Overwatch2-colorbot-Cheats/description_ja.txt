Python ベースの『オーバーウォッチ2』カラーボットで、画面ピクセルの色検出により敵のアウトライン（ゲーム内設定で紫に設定）を識別し、照準デルタを計算したうえで、115200 baud のシリアル通信経由で Arduino Leonardo にマウス移動コマンドを送信し、マウス入力検出を回避します。
Arduino ファームウェアは HID Mouse.move() を実装し、大きな移動は 127 単位ずつに分割して実行します。また、シリアルリンク経由で受信する shoot/silent-aim/reset コマンドに対応し、入力を正規のハードウェアマウスイベントと区別しにくくします。
Arduino HID によるハードウェアベースの入力スプーフィング、およびピクセルベースのカラーエイムボット手法を研究するゲームセキュリティ研究者向けです。
