Python 기반 오버워치 2 컬러봇으로, 화면 픽셀 색상 감지를 사용해 적 윤곽선(게임 내 설정에서 보라색으로 지정)을 식별한 뒤 조준 델타를 계산하고, 115200 baud 시리얼 통신으로 Arduino Leonardo에 마우스 이동 명령을 보내 마우스 입력 탐지를 우회합니다.
Arduino 펌웨어는 HID Mouse.move() 호출을 구현하며, 큰 이동은 127 단위씩 분할해 처리하고, 시리얼 링크로 수신되는 shoot/silent-aim/reset 명령을 지원해 입력을 정상적인 하드웨어 마우스 이벤트와 구별하기 어렵게 만듭니다.
Arduino HID를 통한 하드웨어 기반 입력 스푸핑과 픽셀 기반 컬러 에임봇 기법을 연구하는 게임 보안 연구자에게 주로 유용합니다.
