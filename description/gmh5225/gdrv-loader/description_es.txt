Este proyecto es gdrv-loader, una herramienta de Windows que carga controladores de kernel sin firmar explotando el controlador firmado vulnerable gdrv64.sys de GIGABYTE. Usa el IOCTL de lectura/escritura arbitraria de memoria de gdrv para mapear manualmente un controlador personalizado en memoria del kernel, eludiendo así Driver Signature Enforcement. Está orientado a investigadores de kernel que estudian la explotación del controlador GIGABYTE y el bypass de DSE mediante BYOVD.
