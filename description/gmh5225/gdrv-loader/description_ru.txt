Этот проект — gdrv-loader, инструмент для Windows, который загружает неподписанные драйверы ядра, эксплуатируя уязвимый подписанный драйвер GIGABYTE gdrv64.sys. Он использует IOCTL произвольного чтения/записи памяти gdrv для ручного отображения пользовательского драйвера в память ядра, обходя Driver Signature Enforcement. Предназначен для исследователей ядра, изучающих эксплуатацию драйвера GIGABYTE и обход DSE через BYOVD.
