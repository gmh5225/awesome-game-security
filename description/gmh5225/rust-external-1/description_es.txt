Este proyecto es un cheat externo para el juego Rust escrito en C++ que lee la memoria del juego sin inyección. Accede de forma externa al proceso Unity del juego Rust mediante un controlador de kernel o ReadProcessMemory para extraer posiciones de jugadores, ubicaciones de objetos y el estado del juego para el renderizado ESP en una superposición independiente. Está orientado a investigadores de seguridad en juegos que estudian patrones de cheats externos en juegos Unity protegidos por EAC.
