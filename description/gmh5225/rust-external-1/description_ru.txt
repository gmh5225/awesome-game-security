Этот проект — внешний чит для игры Rust на C++, который читает память игры без инъекции. Он внешне обращается к Unity-процессу игры Rust через драйвер ядра или ReadProcessMemory, извлекая позиции игроков, расположение предметов и состояние игры для отрисовки ESP на отдельном оверлее. Предназначен для исследователей игровой безопасности, изучающих паттерны внешних читов для Unity-игр, защищённых EAC.
