Dieses Projekt ist ein in C++ geschriebenes External Cheat für das Spiel Rust, das den Spielspeicher ohne Injection ausliest. Es greift über einen Kernel-Treiber oder ReadProcessMemory extern auf den Unity-Prozess des Spiels Rust zu, um Spielerpositionen, Item-Standorte und den Spielzustand für die ESP-Darstellung auf einem separaten Overlay zu extrahieren. Es richtet sich an Gamesicherheitsforscher, die External-Cheat-Muster bei EAC-geschützten Unity-Spielen untersuchen.
