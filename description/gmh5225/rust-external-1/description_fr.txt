Ce projet est un cheat externe pour le jeu Rust, écrit en C++, qui lit la mémoire du jeu sans injection. Il accède de l'extérieur au processus Unity du jeu Rust via un pilote noyau ou ReadProcessMemory afin d'extraire les positions des joueurs, l'emplacement des objets et l'état du jeu pour le rendu ESP sur une superposition séparée. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les schémas de cheats externes pour les jeux Unity protégés par EAC.
