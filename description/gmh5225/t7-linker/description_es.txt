Este proyecto es un linker o herramienta para trabajar con scripts y modding de Call of Duty: Black Ops III (motor T7). Procesa archivos de scripts de juego compilados, enlaza módulos de scripts entre sí y maneja el formato de bytecode de scripts T7 usado por el sistema de scripting GSC de Black Ops III. Está dirigido a modders e investigadores de Call of Duty que trabajan con la infraestructura de scripting del motor T7.
