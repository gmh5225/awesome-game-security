Questo progetto è un linker o uno strumento per lavorare con gli script e il modding di Call of Duty: Black Ops III (motore T7). Elabora i file di script di gioco compilati, collega tra loro i moduli di script e gestisce il formato bytecode degli script T7 usato dal sistema di scripting GSC di Black Ops III. È rivolto a modder e ricercatori di Call of Duty che lavorano con l'infrastruttura di scripting del motore T7.
