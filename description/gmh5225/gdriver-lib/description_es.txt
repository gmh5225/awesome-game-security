Este proyecto es una biblioteca que encapsula la interfaz del controlador vulnerable GIGABYTE gdrv64.sys para un acceso conveniente a la memoria del núcleo. Proporciona funciones C++ para leer y escribir memoria física, mapear direcciones físicas y realizar operaciones de núcleo a través de los IOCTL vulnerables de gdrv. Está dirigida a investigadores de núcleo que usan la explotación del controlador GIGABYTE como primitiva de acceso al núcleo.
