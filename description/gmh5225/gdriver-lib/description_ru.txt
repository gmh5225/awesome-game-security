Этот проект — библиотека, оборачивающая интерфейс уязвимого драйвера GIGABYTE gdrv64.sys для удобного доступа к памяти ядра. Она предоставляет функции C++ для чтения и записи физической памяти, отображения физических адресов и выполнения операций ядра через уязвимые IOCTL драйвера gdrv. Предназначена для исследователей ядра, использующих эксплуатацию драйвера GIGABYTE как примитив доступа к ядру.
