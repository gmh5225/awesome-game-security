Dieses Projekt ist eine Bibliothek, die die Schnittstelle des anfälligen GIGABYTE-Treibers gdrv64.sys für bequemen Kernel-Speicherzugriff kapselt. Sie stellt C++-Funktionen zum Lesen und Schreiben von physischem Speicher, zum Mappen physischer Adressen und zur Ausführung von Kernel-Operationen über die anfälligen IOCTLs von gdrv bereit. Sie richtet sich an Kernel-Forscher, die die Ausnutzung des GIGABYTE-Treibers als Kernel-Zugriffsprimitiv nutzen.
