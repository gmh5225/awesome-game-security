Этот проект — репозиторий решений к упражнениям из книги «Practical Reverse Engineering» Брюса Данга и соавторов. В нём приведены развёрнутые решения упражнений по дизассемблированию x86/x64, задач по ARM-ассемблеру, вопросов по анализу ядра Windows и заданий по анализу руткитов. Решения включают аннотированный ассемблерный код и пояснения к анализу. Материал предназначен для студентов, изучающих реверс-инжиниринг по этому учебнику.
