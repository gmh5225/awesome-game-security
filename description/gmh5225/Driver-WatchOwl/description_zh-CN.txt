本项目是一款防御性 Windows 驱动程序，通过监控用户态映像加载与线程创建，并结合栈回溯检查来识别可疑的映射行为。
其实现注册了 `PsSetLoadImageNotifyRoutine` 与 `PsSetCreateThreadNotifyRoutine` 回调，从 `csrss.exe` 解析诸如 `NtMapViewOfSection` 与 `RtlUserThreadStart` 等预期的用户态栈帧，并验证映像加载回调是否来自合法模块的代码段范围。
当观察到的调用栈与预期映射路径不符时，驱动会记录该事件并标出签名级别较低的映像，使本项目成为基于回调的注入检测的精简示例，而非进攻性 hook 集合。
它主要适用于反作弊工程师以及研究基于栈验证的映像映射与可疑用户态代码注入路径的防御性内核研究人员。
