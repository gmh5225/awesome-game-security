Этот проект предоставляет инструменты для взаимодействия с контроллерами PlayStation DualShock 4 (DS4) на ПК. Он обеспечивает чтение ввода контроллера DS4, управление цветом светодиодов, данные тачпада, доступ к датчикам движения и управление моторами вибрации через обмен по протоколу HID. Предназначен для разработчиков игр и исследователей устройств ввода, работающих с функциями контроллера DS4 на платформах ПК.
