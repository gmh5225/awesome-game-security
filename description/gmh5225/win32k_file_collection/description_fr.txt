Ce projet est une collection de win32k.sys et de binaires associés du sous-système GUI du noyau Windows, couvrant plusieurs versions de Windows. Il archive les fichiers du pilote win32k issus de différents builds Windows pour l'analyse comparative, la recherche de vulnérabilités et le patch diffing. La collection permet d'étudier l'évolution de win32k au fil des mises à jour. Elle s'adresse aux chercheurs en vulnérabilités du noyau Windows et aux développeurs d'exploits qui analysent la surface d'attaque de win32k à travers les versions du système.
