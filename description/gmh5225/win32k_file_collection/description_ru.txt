Этот проект — коллекция win32k.sys и связанных бинарных файлов подсистемы GUI ядра Windows для нескольких версий Windows. В нём архивируются файлы драйвера win32k из разных сборок Windows для сравнительного анализа, исследования уязвимостей и patch diffing. Коллекция позволяет изучать изменения win32k между обновлениями. Она предназначена для исследователей уязвимостей ядра Windows и разработчиков эксплойтов, анализирующих поверхность атаки win32k в разных версиях ОС.
