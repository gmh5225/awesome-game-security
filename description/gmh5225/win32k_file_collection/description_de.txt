Dieses Projekt ist eine Sammlung von win32k.sys und zugehörigen Binärdateien des Windows-Kernel-GUI-Subsystems über mehrere Windows-Versionen hinweg. Es archiviert win32k-Treiberdateien aus verschiedenen Windows-Builds für vergleichende Analysen, Schwachstellenforschung und Patch-Diffing. Die Sammlung ermöglicht die Untersuchung von win32k-Änderungen über Updates hinweg. Sie richtet sich an Windows-Kernel-Schwachstellenforscher und Exploit-Entwickler, die die win32k-Angriffsfläche über OS-Versionen hinweg analysieren.
