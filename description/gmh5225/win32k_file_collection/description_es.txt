Este proyecto es una colección de win32k.sys y binarios relacionados del subsistema GUI del kernel de Windows en múltiples versiones de Windows. Archiva archivos del controlador win32k de distintas compilaciones de Windows para análisis comparativo, investigación de vulnerabilidades y patch diffing. La colección permite estudiar los cambios de win32k a lo largo de las actualizaciones. Está dirigida a investigadores de vulnerabilidades del kernel de Windows y a desarrolladores de exploits que analizan la superficie de ataque de win32k entre versiones del sistema.
