Questo progetto è una raccolta di win32k.sys e binari correlati del sottosistema GUI del kernel Windows su più versioni di Windows. Archivia i file del driver win32k da diverse build di Windows per analisi comparative, ricerca di vulnerabilità e patch diffing. La raccolta consente di studiare le modifiche di win32k tra gli aggiornamenti. È rivolta a ricercatori di vulnerabilità del kernel Windows e a sviluppatori di exploit che analizzano la superficie di attacco di win32k tra le versioni del sistema operativo.
