Этот проект — собрание исследовательских материалов по Counter-Strike 2, включающее реконструированные структуры, дампы смещений (offsets), фрагменты SDK и аналитические заметки по движку Source 2, используемому в CS2. В нём документируются раскладки системы сущностей (entity), структуры netvar и указатели интерфейсов движка, обнаруженные методом реверс-инжиниринга. Материал предназначен для разработчиков читов CS2 и исследователей движка Source 2.
