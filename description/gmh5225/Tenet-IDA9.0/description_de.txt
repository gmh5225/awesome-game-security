Dieses Projekt ist ein Port des Tenet-Plugins zur Erkundung von Ausführungstraces, aktualisiert für die Kompatibilität mit IDA Pro 9.0. Es bietet dieselbe Visualisierung der Trace-Zeitachse sowie Vorwärts-/Rückwärts-Schritte in der Ausführung wie das Original-Tenet, jedoch mit API-Anpassungen für das IDA-9.0-SDK. Es richtet sich an Reverse Engineers, die IDA Pro 9.0 nutzen und Funktionen zur Analyse von Ausführungstraces benötigen.
