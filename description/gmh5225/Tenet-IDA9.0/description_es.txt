Este proyecto es un port del plugin explorador de trazas de ejecución Tenet, actualizado para compatibilidad con IDA Pro 9.0. Ofrece la misma visualización de la línea de tiempo de la traza y el avance/retroceso de la ejecución que el Tenet original, pero con actualizaciones de API para el SDK de IDA 9.0. Está dirigido a ingenieros de ingeniería inversa que usan IDA Pro 9.0 y necesitan capacidades de análisis de trazas de ejecución.
