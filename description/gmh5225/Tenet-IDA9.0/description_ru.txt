Этот проект — порт плагина Tenet для исследования трасс выполнения, обновлённый для совместимости с IDA Pro 9.0. Он предоставляет ту же визуализацию временной шкалы трассы и пошаговое выполнение вперёд/назад, что и оригинальный Tenet, но с обновлениями API под SDK IDA 9.0. Предназначен для реверс-инженеров, использующих IDA Pro 9.0 и нуждающихся в возможностях анализа трасс выполнения.
