Questo progetto è un port del plugin Tenet per l'esplorazione delle tracce di esecuzione, aggiornato per la compatibilità con IDA Pro 9.0. Offre la stessa visualizzazione della timeline delle tracce e lo stepping di esecuzione in avanti/indietro dell'originale Tenet, con aggiornamenti API per l'SDK di IDA 9.0. È rivolto agli ingegneri di reverse engineering che usano IDA Pro 9.0 e necessitano di capacità di analisi delle tracce di esecuzione.
