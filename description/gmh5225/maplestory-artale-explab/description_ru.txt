Этот проект — калькулятор опыта и гайд по прокачке для MapleStory Artale, конкретного сервера или версии MapleStory. Он рассчитывает оптимальные места для фарма, скорость набора опыта и пути прокачки с учётом класса персонажа, уровня и доступного контента. Предназначен для игроков MapleStory Artale, которые хотят оптимизировать прогресс прокачки.
