Dieses Projekt ist eine Android-Bibliothek, die mehrere Emulator-Erkennungstechniken implementiert. Sie prüft Emulator-Indikatoren, darunter QEMU-spezifische Dateien, Genymotion-Marker, BlueStacks-Artefakte, Anomalien bei Build-Eigenschaften, fehlende Hardware-Sensoren, Inkonsistenzen im Telefondienst sowie bekannte Emulator-MAC-/IMEI-Muster. Die Java/Kotlin-Bibliothek bietet ein Bewertungssystem, das mehrere Heuristiken kombiniert. Sie richtet sich an Mobile-Game-Entwickler und Sicherheitsingenieure, die Emulator-Erkennung für Anti-Cheat oder Betrugsprävention umsetzen.
