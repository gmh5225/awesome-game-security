Questo progetto è una libreria Android che implementa più tecniche di rilevamento degli emulatori. Controlla indicatori di emulatore, tra cui file specifici di QEMU, marcatori Genymotion, artefatti BlueStacks, anomalie nelle proprietà di build, assenza di sensori hardware, inconsistenze del servizio di telefonia e pattern MAC/IMEI noti di emulatori. La libreria Java/Kotlin fornisce un sistema di scoring che combina più euristiche. È rivolta a sviluppatori di giochi mobili e ingegneri della sicurezza che implementano il rilevamento degli emulatori per anti-cheat o antifrode.
