Этот проект — Android-библиотека, реализующая несколько методов обнаружения эмуляторов. Она проверяет признаки эмулятора, включая файлы, специфичные для QEMU, маркеры Genymotion, артефакты BlueStacks, аномалии свойств сборки, отсутствие аппаратных датчиков, несогласованности телефонного сервиса, а также известные MAC/IMEI-шаблоны эмуляторов. Библиотека на Java/Kotlin предоставляет систему оценки, объединяющую несколько эвристик. Она предназначена для разработчиков мобильных игр и инженеров по безопасности, внедряющих обнаружение эмуляторов для античита или противодействия мошенничеству.
