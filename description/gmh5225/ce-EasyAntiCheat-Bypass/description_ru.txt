Этот проект — обход Cheat Engine для EasyAntiCheat, позволяющий использовать Cheat Engine в играх, защищённых EAC. Он патчит или обходит обнаружение EAC процесса, класса окна, драйвера и шаблонов доступа к памяти Cheat Engine, чтобы разрешить сканирование и изменение памяти. Предназначен для исследователей безопасности игр, изучающих механизмы обнаружения инструментов EAC и техники обхода.
