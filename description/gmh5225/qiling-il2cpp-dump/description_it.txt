Questo progetto usa il framework di emulazione Qiling per estrarre i metadati IL2CPP dai binari di giochi Unity senza eseguire il gioco reale. Emula l'inizializzazione del runtime IL2CPP nell'ambiente sandbox di Qiling per attivare la registrazione dei metadati, quindi estrae definizioni di classi, indirizzi dei metodi e informazioni sui tipi. Questo approccio è adatto all'analisi di giochi IL2CPP offuscati o protetti anti-tamper. È rivolto ai reverse engineer di giochi Unity che analizzano binari IL2CPP fortemente protetti tramite dumping basato sull'emulazione.
