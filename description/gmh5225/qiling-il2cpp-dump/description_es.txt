Este proyecto usa el framework de emulación Qiling para volcar metadatos IL2CPP de binarios de juegos Unity sin ejecutar el juego real. Emula la inicialización del runtime IL2CPP en el entorno aislado de Qiling para activar el registro de metadatos y luego extrae definiciones de clases, direcciones de métodos e información de tipos. Este enfoque sirve para analizar juegos IL2CPP ofuscados o protegidos contra manipulación. Está dirigido a ingenieros inversos de juegos Unity que analizan binarios IL2CPP fuertemente protegidos mediante dumping basado en emulación.
