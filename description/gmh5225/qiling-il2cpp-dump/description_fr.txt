Ce projet utilise le framework d'émulation Qiling pour extraire les métadonnées IL2CPP des binaires de jeux Unity sans exécuter le jeu réel. Il émule l'initialisation du runtime IL2CPP dans l'environnement sandboxé de Qiling afin de déclencher l'enregistrement des métadonnées, puis extrait les définitions de classes, les adresses de méthodes et les informations de types. Cette approche convient à l'analyse de jeux IL2CPP obfusqués ou protégés contre la falsification. Elle s'adresse aux reverse engineers de jeux Unity qui analysent des binaires IL2CPP fortement protégés via un dump basé sur l'émulation.
