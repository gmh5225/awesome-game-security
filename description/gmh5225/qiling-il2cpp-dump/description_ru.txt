Этот проект использует фреймворк эмуляции Qiling для дампа метаданных IL2CPP из бинарников игр на Unity без запуска самой игры. Он эмулирует инициализацию среды выполнения IL2CPP в изолированной среде Qiling, чтобы вызвать регистрацию метаданных, а затем извлекает определения классов, адреса методов и сведения о типах. Такой подход подходит для анализа обфусцированных или защищённых от вмешательства IL2CPP-игр. Он предназначен для reverse-инженеров Unity-игр, анализирующих сильно защищённые IL2CPP-бинарники через дамп на основе эмуляции.
