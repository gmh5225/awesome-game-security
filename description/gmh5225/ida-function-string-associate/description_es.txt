Este proyecto es un complemento de IDA Pro que asocia referencias a cadenas con funciones y muestra las cadenas usadas en cada función en una vista resumen. Examina los cuerpos de las funciones en busca de operandos de referencia a cadenas y crea una lista navegable que indica qué cadenas accede cada función. Ayuda a comprender rápidamente el propósito de una función a partir de su uso de cadenas. Está dirigido a ingenieros de ingeniería inversa que usan IDA Pro y quieren identificar funciones con rapidez mediante el análisis de cadenas.
