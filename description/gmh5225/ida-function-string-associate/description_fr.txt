Ce projet est un plugin IDA Pro qui associe les références de chaînes aux fonctions et affiche les chaînes utilisées dans chaque fonction dans une vue récapitulative. Il parcourt les corps de fonctions à la recherche d'opérandes de références de chaînes et crée une liste navigable indiquant quelles chaînes chaque fonction accède. Cela aide à comprendre rapidement le rôle d'une fonction à partir de son usage des chaînes. Il s'adresse aux ingénieurs en rétro-ingénierie utilisant IDA Pro qui souhaitent identifier rapidement les fonctions via l'analyse des chaînes.
