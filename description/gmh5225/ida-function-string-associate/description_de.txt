Dieses Projekt ist ein IDA-Pro-Plugin, das String-Referenzen Funktionen zuordnet und die in jeder Funktion verwendeten Strings in einer Übersichtsansicht anzeigt. Es durchsucht Funktionskörper nach String-Referenz-Operanden und erstellt eine navigierbare Liste, die zeigt, auf welche Strings jede Funktion zugreift. So lässt sich der Zweck einer Funktion anhand ihrer String-Nutzung schnell erschließen. Es richtet sich an Reverse Engineers mit IDA Pro, die Funktionen durch String-Analyse rasch identifizieren möchten.
