Этот проект — плагин для IDA Pro, который связывает ссылки на строки с функциями и показывает используемые в каждой функции строки в сводном представлении. Он сканирует тела функций на предмет операндов со ссылками на строки и создаёт навигируемый список, указывающий, к каким строкам обращается каждая функция. Это помогает быстро понять назначение функции по использованию строк. Плагин предназначен для специалистов по реверс-инжинирингу, работающих с IDA Pro и желающих быстро идентифицировать функции через анализ строк.
