Questo progetto è un plugin di IDA Pro che associa i riferimenti alle stringhe alle funzioni, mostrando in una vista riepilogativa le stringhe usate in ciascuna funzione. Analizza i corpi delle funzioni alla ricerca di operandi di riferimento a stringhe e crea un elenco navigabile che indica quali stringhe ciascuna funzione accede. Aiuta a comprendere rapidamente lo scopo di una funzione dal suo uso delle stringhe. È rivolto ai reverse engineer che usano IDA Pro e vogliono identificare rapidamente le funzioni tramite l'analisi delle stringhe.
