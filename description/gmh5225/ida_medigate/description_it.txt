Un plugin Python per IDA Pro che ricostruisce gerarchie di classi C++ e tabelle di funzioni virtuali da binari stripped, analizzando le strutture RTTI di GCC (typeinfo, vtables) e mappando le chiamate virtuali alle loro implementazioni concrete tramite la disambiguazione dei tipi basata su union nel decompilatore Hex-Rays.
Il plugin crea struct IDA per ogni classe con le corrette catene di ereditarietà, assegna i tipi ai membri della vtable e consente il cross-reference delle chiamate a funzioni virtuali nel codice decompilato tramite il tracker xref ida-referee incluso, il tutto all'interno del framework nativo structure/union di IDA, quindi supporta qualsiasi architettura gestita da IDA.
È soprattutto utile per i reverse engineer che analizzano binari C++ compilati con polimorfismo, in particolare firmware IoT/embedded e componenti di game engine, dove la ricostruzione manuale delle vtable è proibitivamente tediosa.
