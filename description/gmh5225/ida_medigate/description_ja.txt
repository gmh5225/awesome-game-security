IDA Pro 向けの Python プラグインで、GCC の RTTI 構造体（typeinfo、vtable）を解析し、シンボルが削除されたバイナリから C++ のクラス階層と仮想関数テーブルを再構築します。さらに Hex-Rays デコンパイラ上の union ベースの型曖昧性解消により、仮想呼び出しを具体的な実装へ対応付けます。
各クラスについて正しい継承チェーンを持つ IDA の構造体を構築し、vtable メンバ型を割り当て、同梱の ida-referee xref トラッカーを用いてデコンパイル結果全体で仮想関数呼び出しのクロスリファレンスを可能にします。これらはすべて IDA ネイティブの構造体／共用体フレームワーク内で行われるため、IDA が扱える任意のアーキテクチャに対応します。
ポリモーフィズムを含むコンパイル済み C++ バイナリを解析するリバースエンジニア、とりわけ IoT／組み込みファームウェアやゲームエンジン部品の解析に有用で、そうした場面では手動での vtable 再構築は極めて手間がかかります。
