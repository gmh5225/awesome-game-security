Плагин Python для IDA Pro, который восстанавливает иерархии классов C++ и таблицы виртуальных функций из бинарников без символов, разбирая структуры GCC RTTI (typeinfo, vtables) и сопоставляя виртуальные вызовы с конкретными реализациями через устранение неоднозначности типов на основе union в декомпиляторе Hex-Rays.
Плагин создаёт структуры IDA для каждого класса с корректными цепочками наследования, назначает типы членам vtable и обеспечивает перекрёстные ссылки на вызовы виртуальных функций в декомпилированном коде с помощью входящего в комплект трекера xref ida-referee — всё в рамках нативной модели structure/union IDA, поэтому поддерживаются любые архитектуры, с которыми работает IDA.
В первую очередь полезен реверс-инженерам, анализирующим скомпилированные C++-бинарники с полиморфизмом, особенно прошивки IoT/embedded и компоненты игровых движков, где ручное восстановление vtable крайне трудоёмко.
