Un complemento de Python para IDA Pro que reconstruye jerarquías de clases C++ y tablas de funciones virtuales a partir de binarios sin símbolos, analizando estructuras RTTI de GCC (typeinfo, vtables) y asignando las llamadas virtuales a sus implementaciones concretas mediante desambiguación de tipos basada en uniones en el descompilador Hex-Rays.
El complemento crea structs de IDA para cada clase con las cadenas de herencia correctas, asigna tipos a los miembros de la vtable y permite referencias cruzadas de las llamadas a funciones virtuales en el código descompilado mediante el rastreador xref ida-referee incluido, todo dentro del marco nativo de estructuras/uniones de IDA, por lo que es compatible con cualquier arquitectura que IDA admita.
Resulta especialmente útil para ingenieros inversos que analizan binarios C++ compilados con polimorfismo, sobre todo firmware IoT/embebido y componentes de motores de juegos, donde reconstruir las vtables a mano es prohibitivamente tedioso.
