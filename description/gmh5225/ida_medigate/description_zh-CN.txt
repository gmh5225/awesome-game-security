一款 IDA Pro Python 插件，通过解析 GCC RTTI 结构（typeinfo、vtable），从已剥离符号的二进制中重建 C++ 类层次与虚函数表，并借助 Hex-Rays 反编译器中基于 union 的类型消歧，将虚调用映射到具体实现。
该插件为每个类构建带正确继承链的 IDA 结构体，分配 vtable 成员类型，并借助捆绑的 ida-referee 交叉引用跟踪器，在反编译代码中对虚函数调用进行交叉引用；这一切都在 IDA 原生的结构体/联合体框架内完成，因此支持 IDA 所能处理的任意架构。
它主要面向分析含多态的已编译 C++ 二进制的逆向工程师，尤其适用于 IoT/嵌入式固件与游戏引擎组件——在这些场景中，手工重建 vtable 往往极为繁琐。
