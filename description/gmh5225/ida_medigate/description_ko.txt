IDA Pro용 Python 플러그인으로, GCC RTTI 구조체(typeinfo, vtable)를 파싱하여 심볼이 제거된 바이너리에서 C++ 클래스 계층과 가상 함수 테이블을 재구성하고, Hex-Rays 디컴파일러의 유니온 기반 타입 모호성 해소를 통해 가상 호출을 구체적인 구현에 매핑합니다.
각 클래스에 대해 올바른 상속 체인을 갖춘 IDA 구조체를 만들고 vtable 멤버 타입을 할당하며, 함께 제공되는 ida-referee xref 추적기를 사용해 디컴파일된 코드 전반에서 가상 함수 호출을 교차 참조할 수 있게 합니다. 이 모든 작업은 IDA 네이티브 구조체/유니온 프레임워크 안에서 이루어지므로 IDA가 지원하는 모든 아키텍처에서 동작합니다.
다형성이 있는 컴파일된 C++ 바이너리를 분석하는 리버스 엔지니어에게 특히 유용하며, IoT/임베디드 펌웨어와 게임 엔진 구성 요소처럼 수동 vtable 재구성이 매우 번거로운 환경에서 큰 도움이 됩니다.
