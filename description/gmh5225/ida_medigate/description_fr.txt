Un plugin Python pour IDA Pro qui reconstruit les hiérarchies de classes C++ et les tables de fonctions virtuelles à partir de binaires dépouillés en analysant les structures RTTI de GCC (typeinfo, vtables), et mappe les appels virtuels vers leurs implémentations concrètes grâce à une désambiguïsation de types basée sur des unions dans le décompilateur Hex-Rays.
Le plugin crée des structures IDA pour chaque classe avec les chaînes d'héritage correctes, assigne les types des membres de vtable, et permet le référencement croisé des appels de fonctions virtuelles dans le code décompilé via le tracker xref ida-referee fourni, le tout dans le cadre natif structures/unions d'IDA, ce qui le rend compatible avec toute architecture prise en charge par IDA.
Il est surtout utile aux reverse engineers qui analysent des binaires C++ compilés avec polymorphisme, notamment les firmwares IoT/embarqués et les composants de moteurs de jeux, où la reconstruction manuelle des vtables est prohibitivement fastidieuse.
