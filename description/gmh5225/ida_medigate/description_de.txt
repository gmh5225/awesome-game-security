Ein IDA-Pro-Python-Plugin, das C++-Klassenhierarchien und virtuelle Funktionstabellen aus gestrippten Binaries rekonstruiert, indem es GCC-RTTI-Strukturen (typeinfo, vtables) parst und virtuelle Aufrufe über union-basierte Typdisambiguierung im Hex-Rays-Decompiler ihren konkreten Implementierungen zuordnet.
Das Plugin erzeugt IDA-Structs für jede Klasse mit korrekten Vererbungsketten, weist den vtable-Membern Typen zu und ermöglicht über den mitgelieferten ida-referee-xref-Tracker Querverweise auf virtuelle Funktionsaufrufe im dekompilierten Code — alles innerhalb von IDAs nativem Struct-/Union-Framework, sodass jede von IDA unterstützte Architektur abgedeckt ist.
Es ist vor allem für Reverse Engineers nützlich, die kompilierte C++-Binaries mit Polymorphismus analysieren, insbesondere IoT-/Embedded-Firmware und Game-Engine-Komponenten, bei denen die manuelle vtable-Rekonstruktion unverhältnismäßig aufwendig wäre.
