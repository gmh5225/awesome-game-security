一款 IDA Pro Python 外掛，透過解析 GCC RTTI 結構（typeinfo、vtable），從已剝離符號的二進位檔中重建 C++ 類別階層與虛擬函式表，並借助 Hex-Rays 反編譯器中基於 union 的型別消歧，將虛擬呼叫對應到具體實作。
此外掛為每個類別建立帶有正確繼承鏈的 IDA 結構體，指派 vtable 成員型別，並借助隨附的 ida-referee 交叉參考追蹤器，在反編譯程式碼中對虛擬函式呼叫進行交叉參考；這一切都在 IDA 原生的結構體/聯合體框架內完成，因此支援 IDA 所能處理的任意架構。
它主要適用於分析含多型之已編譯 C++ 二進位檔的逆向工程師，尤其是 IoT/嵌入式韌體與遊戲引擎元件——在這些場景中，手動重建 vtable 往往極為繁瑣。
