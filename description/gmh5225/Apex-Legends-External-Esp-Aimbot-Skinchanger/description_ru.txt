Этот проект — репозиторий из раздела cheat / game:apex legends, посвящённый внешнему ESP, аимботу и скиненджеру для Apex Legends.
Он в основном написан на C++ и C/C++ и сосредоточен на рендеринге, генерации SDK и оверлеях.
Он прежде всего полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:apex legends.
