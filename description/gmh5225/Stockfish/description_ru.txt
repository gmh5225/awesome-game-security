Этот проект — Stockfish, сильнейший открытый шахматный движок, использующий продвинутые алгоритмы поиска (альфа-бета с оценкой NNUE). Он сочетает классические методы шахматного поиска с эффективно обновляемой нейросетевой оценкой для игры сверхчеловеческого уровня. Движок на C++ поддерживает протокол UCI и работает на всех основных платформах. Он предназначен для разработчиков шахматных движков, исследователей ИИ и игроков в соревновательные шахматы.
