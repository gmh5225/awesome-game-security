Este proyecto es bootlicker, una prueba de concepto de bootkit UEFI que carga código malicioso durante el arranque de Windows, antes de que se inicialice el kernel del SO. Parchea el Windows Boot Manager o el cargador del SO para inyectar código en modo kernel que se ejecuta con privilegios completos del sistema, eludiendo DSE, PatchGuard y todas las medidas de seguridad a nivel de SO. Está dirigido a investigadores de seguridad de arranque que estudian técnicas de bootkit UEFI y la elusión de Secure Boot.
