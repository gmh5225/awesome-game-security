Ce projet est bootlicker, une preuve de concept de bootkit UEFI qui charge du code malveillant pendant le démarrage de Windows, avant l'initialisation du noyau OS. Il modifie le Windows Boot Manager ou le chargeur OS pour injecter du code en mode noyau s'exécutant avec tous les privilèges système, contournant DSE, PatchGuard et toutes les mesures de sécurité au niveau de l'OS. Il s'adresse aux chercheurs en sécurité du démarrage étudiant les techniques de bootkit UEFI et le contournement de Secure Boot.
