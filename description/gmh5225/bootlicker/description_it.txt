Questo progetto è bootlicker, una proof of concept di bootkit UEFI che carica codice malevolo durante l'avvio di Windows, prima che il kernel del SO venga inizializzato. Patcha il Windows Boot Manager o il loader del SO per iniettare codice in modalità kernel che viene eseguito con pieni privilegi di sistema, aggirando DSE, PatchGuard e tutte le misure di sicurezza a livello di SO. È rivolto ai ricercatori di sicurezza dell'avvio che studiano le tecniche di bootkit UEFI e il bypass di Secure Boot.
