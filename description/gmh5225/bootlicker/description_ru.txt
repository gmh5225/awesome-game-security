Этот проект — bootlicker, доказательство концепции UEFI-буткита, которое загружает вредоносный код в процессе загрузки Windows до инициализации ядра ОС. Он патчит Windows Boot Manager или загрузчик ОС, внедряя код режима ядра с полными системными привилегиями и обходя DSE, PatchGuard и все меры безопасности на уровне ОС. Предназначен для исследователей безопасности загрузки, изучающих техники UEFI-буткитов и обход Secure Boot.
