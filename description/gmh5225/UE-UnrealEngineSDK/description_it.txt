Questo progetto è un SDK Unreal Engine ottenuto tramite reverse engineering che fornisce definizioni di classi C++, layout di strutture e firme di funzioni per gli interni dei giochi UE4/UE5. Include definizioni della gerarchia UObject, strutture UWorld/ULevel, classi del player controller, layout dei componenti actor e i global del motore necessari per il game hacking e il modding. L'SDK consente di interagire direttamente con gli oggetti di gioco UE da codice iniettato. È pensato per hacker e modder di giochi UE che necessitano di header SDK per lo sviluppo di cheat o mod interni.
