Ce projet est un SDK Unreal Engine issu de la rétro-ingénierie, fournissant des définitions de classes C++, des layouts de structures et des signatures de fonctions pour les internals des jeux UE4/UE5. Il inclut les définitions de la hiérarchie UObject, les structures UWorld/ULevel, les classes de contrôleur de joueur, les layouts des composants d'acteur, ainsi que les globals moteur nécessaires au hacking et au modding de jeux. Le SDK permet d'interagir directement avec les objets de jeu UE depuis du code injecté. Il s'adresse aux hackers et moddeurs de jeux UE qui ont besoin d'en-têtes SDK pour le développement de cheats ou de mods internes.
