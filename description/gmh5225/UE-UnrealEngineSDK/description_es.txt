Este proyecto es un SDK de Unreal Engine obtenido por ingeniería inversa que proporciona definiciones de clases C++, layouts de estructuras y firmas de funciones para los internals de juegos UE4/UE5. Incluye definiciones de la jerarquía UObject, estructuras UWorld/ULevel, clases de controlador de jugador, layouts de componentes de actor y globals del motor necesarios para el hacking y el modding de juegos. El SDK permite interactuar directamente con objetos de juego de UE desde código inyectado. Está dirigido a hackers y modders de juegos UE que necesitan cabeceras SDK para el desarrollo de cheats o mods internos.
