Dieses Projekt ist ein reverse-engineertes Unreal Engine SDK mit C++-Klassendefinitionen, Strukturlayouts und Funktionssignaturen für UE4/UE5-Spielinternals. Es umfasst Definitionen der UObject-Hierarchie, UWorld/ULevel-Strukturen, Player-Controller-Klassen, Layouts von Actor-Komponenten sowie Engine-Globals, die für Game Hacking und Modding benötigt werden. Das SDK ermöglicht die direkte Interaktion mit UE-Spielobjekten aus injiziertem Code. Es richtet sich an UE-Game-Hacker und Modder, die SDK-Header für die Entwicklung interner Cheats oder Mods benötigen.
