Этот проект — реконструированный Unreal Engine SDK с определениями классов C++, раскладками структур и сигнатурами функций для внутренних механизмов игр на UE4/UE5. Он включает определения иерархии UObject, структуры UWorld/ULevel, классы контроллера игрока, раскладки компонентов акторов и глобальные объекты движка, необходимые для взлома и моддинга игр. SDK позволяет напрямую взаимодействовать с игровыми объектами UE из внедрённого кода. Он предназначен для хакеров и моддеров игр на UE, которым нужны заголовочные файлы SDK для разработки внутренних читов или модов.
