Questo progetto fornisce script JavaScript per eludere le tecniche di rilevamento del debugger basate sul browser. Neutralizza misure anti-debugging come le istruzioni debugger, il rilevamento dei DevTools, i controlli di timing tramite console.log e il monitoraggio delle dimensioni della finestra che le applicazioni web usano per rilevare quando gli strumenti per sviluppatori del browser sono aperti. È rivolto a ricercatori di sicurezza web e reverse engineer che analizzano applicazioni web intensive in JavaScript con protezioni anti-debugging.
