Этот проект предоставляет JavaScript-скрипты для обхода техник обнаружения отладчика в браузере. Он нейтрализует антиотладочные меры, такие как операторы debugger, обнаружение DevTools, проверки тайминга через console.log и мониторинг размера окна, которые веб-приложения используют для определения открытия инструментов разработчика браузера. Предназначен для исследователей веб-безопасности и инженеров по реверс-инжинирингу, анализирующих насыщенные JavaScript веб-приложения с антиотладочной защитой.
