Este proyecto proporciona scripts de JavaScript para eludir técnicas de detección de depuradores basadas en el navegador. Neutraliza medidas antidepuración como las instrucciones debugger, la detección de DevTools, las comprobaciones de temporización con console.log y la supervisión del tamaño de la ventana que las aplicaciones web usan para detectar cuándo están abiertas las herramientas de desarrollo del navegador. Está dirigido a investigadores de seguridad web e ingenieros de ingeniería inversa que analizan aplicaciones web intensivas en JavaScript con protecciones antidepuración.
