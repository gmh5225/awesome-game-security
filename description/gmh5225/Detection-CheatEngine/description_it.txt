Questo progetto è un piccolo rilevatore in user mode di artefatti di Cheat Engine, basato sul monitoraggio delle directory con ReadDirectoryChangesW.
L’esempio osserva il profilo utente e C:\ per eventi di creazione, rinomina e scrittura di file, poi segnala i nomi che contengono marcatori come ADDRESSES.FIRST e MEMORY.FIRST definiti in CEInfo.h.
Invece di analizzare memoria o processi, si concentra su un’euristica leggera basata sul filesystem, estendibile con ulteriori stringhe di rilevamento e directory monitorate.
È soprattutto utile per gli ingegneri anti-cheat che studiano strategie semplici di rilevamento di Cheat Engine in user mode basate sulla creazione di tabelle e artefatti.
