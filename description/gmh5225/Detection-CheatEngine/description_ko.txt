이 프로젝트는 ReadDirectoryChangesW 디렉터리 모니터링 위에 구축된 Cheat Engine 아티팩트용 소형 사용자 모드 탐지기입니다.
샘플은 사용자 프로필과 C:\ 에서 파일 생성, 이름 변경, 쓰기 이벤트를 감시한 뒤, CEInfo.h 에 정의된 ADDRESSES.FIRST, MEMORY.FIRST 같은 마커가 포함된 파일명을 표시합니다.
메모리나 프로세스를 스캔하는 대신, 추가 탐지 문자열과 디렉터리 감시로 확장할 수 있는 가벼운 파일시스템 기반 휴리스틱에 초점을 둡니다.
테이블 및 아티팩트 생성 행위에 기반한 단순한 사용자 모드 Cheat Engine 탐지 전략을 연구하는 안티치트 엔지니어에게 주로 유용합니다.
