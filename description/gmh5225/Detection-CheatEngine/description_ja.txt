本プロジェクトは、ReadDirectoryChangesW によるディレクトリ監視の上に構築された、Cheat Engine 関連アーティファクト向けの小さなユーザーモード検出器です。
サンプルはユーザープロファイルと C:\ を監視し、ファイルの作成・名前変更・書き込みイベントを捉え、CEInfo.h で定義された ADDRESSES.FIRST や MEMORY.FIRST などのマーカーを含むファイル名をフラグ付けします。
メモリやプロセスを走査するのではなく、追加の検出文字列や監視ディレクトリで拡張できる、軽量なファイルシステムベースのヒューリスティックに重点を置いています。
テーブルやアーティファクト生成の挙動に基づく単純なユーザーモード Cheat Engine 検出手法を学ぶアンチチート技術者にとって有用です。
