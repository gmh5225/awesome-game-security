Этот проект — небольшой детектор артефактов Cheat Engine в пользовательском режиме, построенный на мониторинге каталогов через ReadDirectoryChangesW.
Пример отслеживает профиль пользователя и C:\ на события создания, переименования и записи файлов, а затем помечает имена, содержащие маркеры вроде ADDRESSES.FIRST и MEMORY.FIRST из CEInfo.h.
Вместо сканирования памяти или процессов он опирается на лёгкую эвристику на основе файловой системы, которую можно расширять дополнительными строками обнаружения и каталогами для наблюдения.
Он в первую очередь полезен инженерам античита, изучающим простые стратегии обнаружения Cheat Engine в user mode по поведению создания таблиц и артефактов.
