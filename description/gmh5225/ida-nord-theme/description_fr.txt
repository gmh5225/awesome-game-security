Ce projet est un thème de couleurs pour IDA Pro inspiré de la palette Nord. Il applique le schéma de couleurs arctique Nord aux vues désassemblage, hexadécimale et graphe d'IDA, ainsi qu'à toutes les fenêtres d'édition, offrant un thème sombre visuellement cohérent aux tons bleu-gris. Il s'adresse aux utilisateurs d'IDA Pro qui préfèrent l'esthétique Nord pour leur environnement de rétro-ingénierie.
