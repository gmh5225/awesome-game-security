Questo progetto è un tema colori per IDA Pro ispirato alla palette Nord. Applica lo schema cromatico artico Nord alle viste disassembly, hex e graph di IDA, nonché a tutte le finestre dell'editor, offrendo un tema scuro visivamente coerente con toni blu-grigio. È pensato per gli utenti di IDA Pro che preferiscono l'estetica Nord nel proprio ambiente di reverse engineering.
