Этот проект — цветовая тема для IDA Pro, вдохновлённая палитрой Nord. Она применяет арктическую цветовую схему Nord к представлениям дизассемблера, hex, графа и ко всем окнам редактора IDA, обеспечивая визуально цельную тёмную тему с сине-серыми оттенками. Предназначена для пользователей IDA Pro, которым ближе эстетика Nord в среде реверс-инжиниринга.
