Ce projet est un Valorant Internal Cheat basé sur le code source de StuzziKLL, avec quelques petits ajustements de ma part.
Il est principalement écrit en C/C++ et C++ et se concentre sur la recherche anti-cheat et le développement de drivers.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / game:valorant.
