Questo progetto è un repository elencato sotto cheat / x64dbg plugins e si concentra sulla visualizzazione delle notifiche DLL in x64dbg.
È scritto principalmente in C++ e C/C++ e punta allo sviluppo di plugin.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell’ambito cheat / x64dbg plugins.
