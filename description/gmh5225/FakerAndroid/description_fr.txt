Ce projet est un outil qui convertit un fichier APK en projet Android classique et prend en charge le so hook, y compris l’échafaudage C++ il2cpp. Parcours de développement secondaire : 1、Ouvrir le projet 2、Déboguer ou exécuter le projet 3、Avancé : à l’aide de javaScaffolding, écrivez votre code Java dans le module app (app/src/main/java) pour appeler les classes d’origine ; conservez les mêmes noms de classe et de package que les classes originales ; lorsqu’il y a des modifications de fichiers smali, la construction se fait avec un minimum de fichiers ; à l’aide de l’API fakeCpp, utilisez JNI pour hooker les méthodes so ; à l’aide de l’échafaudage il2cpp et de l’API fakeCpp, utilisez JNI pour modifier les scripts de jeu il2cpp. Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine cheat / explorateur de moteurs de jeu : Unity.
