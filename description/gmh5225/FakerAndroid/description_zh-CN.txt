该项目是一款将 APK 文件转换为常规 Android 工程的工具，并支持 so hook，包含 il2cpp C++ 脚手架。二次开发流程：1、打开工程 2、调试或运行工程 3、进阶：借助 javaScaffolding 在 app 模块（app/src/main/java）中编写 Java 代码以调用原始类；在 app 模块中编写时请保持类名与包名与原始类一致；当存在 smali 文件修改时，可实现最少文件构建；借助 fakeCpp API，通过 JNI hook so 方法；借助 il2cpp 脚手架与 fakeCpp API，通过 JNI 修改 il2cpp 游戏脚本。该工具主要面向游戏安全研究人员与逆向工程师，适用于在作弊 / 游戏引擎探索（Unity）方向研究攻击技术。
