本プロジェクトは、APK ファイルを一般的な Android プロジェクトへ変換するツールであり、il2cpp の C++ スキャフォールディングを含む so hook をサポートします。二次開発の手順：1、プロジェクトを開く 2、デバッグまたは実行する 3、応用：javaScaffolding を使い、app モジュール（app/src/main/java）で Java コードを書いて元のクラスを呼び出します。app モジュールに書く際は、クラス名とパッケージ名を元のクラスと対応させてください。smali ファイルを改変する場合、最小限のファイル構築が可能です。fakeCpp API を使い JNI で so メソッドを hook できます。il2cpp スキャフォールディングと fakeCpp API を使い、JNI で il2cpp ゲームスクリプトを改変できます。主に、チート / ゲームエンジン探索（Unity）分野で攻撃手法を研究するゲームセキュリティ研究者やリバースエンジニア向けです。
