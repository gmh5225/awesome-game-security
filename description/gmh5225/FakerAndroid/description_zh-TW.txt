此專案是一款將 APK 檔案轉換為一般 Android 專案的工具，並支援 so hook，包含 il2cpp C++ 鷹架。二次開發流程：1、開啟專案 2、除錯或執行專案 3、進階：藉由 javaScaffolding 在 app 模組（app/src/main/java）中撰寫 Java 程式碼以呼叫原始類別；在 app 模組中撰寫時請保持類別名稱與套件名稱與原始類別一致；當存在 smali 檔案修改時，可實現最少檔案建置；藉由 fakeCpp API，透過 JNI hook so 方法；藉由 il2cpp 鷹架與 fakeCpp API，透過 JNI 修改 il2cpp 遊戲腳本。此工具主要面向遊戲安全研究人員與逆向工程師，適用於在作弊 / 遊戲引擎探索（Unity）方向研究攻擊技術。
