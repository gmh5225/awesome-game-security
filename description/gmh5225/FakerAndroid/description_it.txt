Questo progetto è uno strumento che traduce un file APK in un comune progetto Android e supporta so hook, incluso lo scaffolding C++ di il2cpp. Percorso di sviluppo secondario: 1、Aprire il progetto 2、Eseguire il debug o avviare il progetto 3、Avanzato: con l’aiuto di javaScaffolding, scrivere codice Java nel modulo app (app/src/main/java) per chiamare le classi originali; mantenere nome della classe e del package corrispondenti a quelli della classe originale; quando ci sono modifiche ai file smali, la build usa il minimo di file; con l’API fakeCpp, usare JNI per fare hook dei metodi so; con lo scaffolding il2cpp e l’API fakeCpp, usare JNI per modificare gli script di gioco il2cpp. È soprattutto utile per ricercatori di sicurezza dei giochi e reverse engineer che studiano tecniche offensive nell’area cheat / esploratore di motori di gioco: Unity.
