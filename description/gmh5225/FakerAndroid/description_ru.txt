Этот проект — инструмент, преобразующий APK-файл в обычный Android-проект и поддерживающий so hook, включая C++-scaffolding для il2cpp. Курс вторичной разработки: 1、Открыть проект 2、Отладить или запустить проект 3、Продвинутый уровень: с помощью javaScaffolding пишите Java-код в модуле app (app/src/main/java) для вызова исходных классов; сохраняйте имена классов и пакетов такими же, как у исходных классов; при модификации smali-файлов сборка выполняется с минимальным числом файлов; с помощью API fakeCpp используйте JNI для hook методов so; с помощью scaffolding il2cpp и API fakeCpp через JNI модифицируйте игровые скрипты il2cpp. В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / исследования игровых движков: Unity.
