Este proyecto es una herramienta que convierte un archivo APK en un proyecto Android común y admite so hook, incluido el andamiaje C++ de il2cpp. Curso de desarrollo secundario: 1、Abrir el proyecto 2、Depurar o ejecutar el proyecto 3、Avanzado: con la ayuda de javaScaffolding, escriba código Java en el módulo app (app/src/main/java) para llamar a las clases originales; mantenga el nombre de clase y el nombre de paquete iguales a los de la clase original; cuando hay modificaciones de archivos smali, la compilación usa el mínimo de archivos; con la API fakeCpp, use JNI para hacer hook de métodos so; con el andamiaje il2cpp y la API fakeCpp, use JNI para modificar scripts de juego il2cpp. Es especialmente útil para investigadores de seguridad de juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheat / explorador de motores de juego: Unity.
