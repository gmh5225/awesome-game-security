Этот проект — адаптация под Windows 11 схемы driver-cheat от Null, которая перехватывает NtOpenCompositionSurfaceSectionInfo внутри dxgkrnl, чтобы создать скрытый канал связи с ядром.
Установщик хука записывает небольшой jump stub в экспорт, а обработчик принимает структурированные запросы на поиск базы модуля, чтение и запись памяти процесса, а также необязательные вспомогательные средства отрисовки GDI, взятые из экспортов win32k.
В архивном README это явно подаётся как изменённая повторная реализация с другим pool tag и выбором syscall, призванным снизить немедленное обнаружение по сравнению с более простыми подходами на основе устройств.
Проект в основном полезен исследователям, изучающим хуки экспортов dxgkrnl, структурированный обмен сообщениями между user mode и kernel, а также GDI-вспомогательные оверлеи или средства работы с памятью в cheat-драйверах.
