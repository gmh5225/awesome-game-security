本プロジェクトは、Null のドライバーチート手法を Windows 11 向けに適応したもので、dxgkrnl 内の NtOpenCompositionSurfaceSectionInfo をフックし、秘匿されたカーネル通信経路を構築します。
フックインストーラーはそのエクスポートへ小さなジャンプスタブを書き込み、ハンドラーはモジュールベースの検索、プロセスメモリの読み書き、および win32k エクスポート由来の任意の GDI 描画ヘルパー向けの構造化リクエストを受け付けます。
アーカイブされた README では、異なるプールタグを用い、より単純なデバイスベースの手法よりも即時検知されにくくすることを意図して当該システムコールを選んだ、改変された再実装として明確に位置づけられています。
主に、dxgkrnl エクスポートフック、構造化されたユーザー／カーネル間メッセージング、およびチートドライバーにおける GDI 支援のオーバーレイやメモリヘルパーを研究するリバースエンジニア向けです。
