Un framework di hooking delle system call che sfrutta l'hypervisor di Kaspersky (klhk.sys) per intercettare le syscall.
Sfrutta il meccanismo di modifica di IA32_LSTAR di Kaspersky, che reindirizza le system call attraverso la propria tabella di dispatch, caricando klhk.sys e un driver personalizzato per sovvertire il gestore delle syscall.
