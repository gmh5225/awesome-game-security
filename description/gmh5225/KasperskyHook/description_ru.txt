Фреймворк для перехвата системных вызовов, использующий гипервизор Kaspersky (klhk.sys) для перехвата syscall.
Эксплуатирует механизм модификации IA32_LSTAR в Kaspersky, который перенаправляет системные вызовы через собственную таблицу диспетчеризации, загружая klhk.sys и пользовательский драйвер для подмены обработчика syscall.
