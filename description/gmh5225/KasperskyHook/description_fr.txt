Un framework de hooking des appels système qui s'appuie sur l'hyperviseur de Kaspersky (klhk.sys) pour intercepter les syscalls.
Il exploite le mécanisme de modification d'IA32_LSTAR de Kaspersky, qui redirige les appels système via sa propre table de répartition, en chargeant klhk.sys et un pilote personnalisé pour subvertir le gestionnaire de syscalls.
