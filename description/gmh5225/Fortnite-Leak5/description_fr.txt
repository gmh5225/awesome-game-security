Ce projet est le code source de base utilisé par INFARCTED.CC.
Il est principalement écrit en C++ et C/C++ et s'articule autour d'Unity, de la génération de SDK et du hooking.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / game:fortnite.
