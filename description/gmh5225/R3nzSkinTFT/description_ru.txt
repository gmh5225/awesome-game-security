Этот проект — скин-чейнджер для Teamfight Tactics (TFT) и League of Legends, который изменяет ID скинов чемпионов в памяти, чтобы отображать альтернативные скины на стороне клиента. Он читает и патчит структуры данных скинов чемпионов в игре, подменяя внешний вид без обнаружения на стороне сервера. Инструмент на C++ демонстрирует техники клиентской косметической модификации. Он предназначен для исследователей игровой безопасности, изучающих манипуляции с косметикой в играх Riot Games.
