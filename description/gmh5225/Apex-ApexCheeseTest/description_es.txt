Este proyecto es una comunicación núcleo–modo usuario copiada de un usuario de cheat desconocido. Está escrito principalmente en C/C++ y C++ y se centra en la investigación anti-cheat, el trabajo a nivel de kernel y el desarrollo de controladores. Resulta especialmente útil para investigadores de seguridad de juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito cheat / juego: Apex Legends.
