Ce projet est une analyse par rétro-ingénierie d'ACE (Anti-Cheat Expert) tel qu'utilisé dans Honkai: Star Rail. Il documente les mécanismes de protection, les techniques de détection et les méthodes de contournement d'ACE spécifiques à l'implémentation de Star Rail. L'analyse couvre le comportement du pilote noyau d'ACE, les contrôles d'intégrité et les capacités de surveillance. Il s'adresse aux chercheurs en anti-triche qui étudient le système anti-triche ACE de Tencent dans les titres miHoYo/HoYoverse.
