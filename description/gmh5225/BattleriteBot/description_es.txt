Este proyecto es un bot de automatización de juego para Battlerite, un brawler de arena en equipo. Lee el estado del juego mediante acceso a memoria o reconocimiento de pantalla y automatiza acciones de juego como el lanzamiento de habilidades, la selección de objetivos y el movimiento. Está dirigido a investigadores de automatización de juegos que estudian el desarrollo de bots para juegos de combate en arena.
