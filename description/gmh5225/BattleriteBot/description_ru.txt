Этот проект — бот для автоматизации игры Battlerite, командного арена-браулера. Он считывает состояние игры через доступ к памяти или распознавание экрана и автоматизирует игровые действия, включая применение способностей, выбор целей и перемещение. Предназначен для исследователей автоматизации игр, изучающих разработку ботов для арена-боевых игр.
