Este proyecto es una prueba de concepto del kernel de Windows para registrar y usar callbacks de interrupción no enmascarable (NMI). Los callbacks NMI se ejecutan en todos los núcleos a la vez y los sistemas anti-cheat los usan para inspeccionar contextos de hilos y detectar hilos o código ocultos. Esta herramienta demuestra el registro de callbacks NMI y la inspección de contexto. Está dirigida a investigadores de kernel que estudian la detección basada en NMI empleada por anti-cheats como BattlEye.
