Этот проект — демонстрация концепции для ядра Windows по регистрации и использованию колбэков немаскируемого прерывания (NMI). Колбэки NMI выполняются на всех ядрах одновременно и используются античитами для проверки контекстов потоков и обнаружения скрытых потоков или кода. Инструмент демонстрирует регистрацию NMI-колбэков и инспекцию контекста. Он предназначен для исследователей ядра, изучающих NMI-детекцию в античитах вроде BattlEye.
