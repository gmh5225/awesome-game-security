Dieses Projekt ist ein Windows-Kernel-Proof-of-Concept zum Registrieren und Nutzen von Non-Maskable-Interrupt-(NMI-)Callbacks. NMI-Callbacks laufen auf allen Kernen gleichzeitig und werden von Anti-Cheat-Systemen genutzt, um Thread-Kontexte zu prüfen und versteckte Threads oder Code zu erkennen. Das Tool demonstriert die Registrierung von NMI-Callbacks und die Kontextinspektion. Es richtet sich an Kernel-Forscher, die NMI-basierte Erkennung bei Anti-Cheats wie BattlEye untersuchen.
