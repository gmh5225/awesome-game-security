Ce projet est une preuve de concept pour le noyau Windows visant à enregistrer et utiliser des callbacks d'interruption non masquable (NMI). Les callbacks NMI s'exécutent simultanément sur tous les cœurs et sont utilisés par les systèmes anti-cheat pour inspecter les contextes de threads et détecter des threads ou du code masqués. Cet outil démontre l'enregistrement des callbacks NMI et l'inspection de contexte. Il s'adresse aux chercheurs noyau qui étudient la détection basée sur les NMI employée par des anti-cheats tels que BattlEye.
