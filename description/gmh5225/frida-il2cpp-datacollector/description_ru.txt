Этот проект — скрипт Frida для сбора runtime-данных IL2CPP из игр на Unity. Он внедряется в runtime IL2CPP, чтобы перечислять классы, дампить сигнатуры методов, извлекать смещения полей и собирать метаданные типов во время выполнения через инструментацию Frida. Собранные данные помогают создавать SDK для взлома игр на IL2CPP. Проект предназначен для реверс-инженеров мобильных игр, использующих Frida для анализа Unity-игр на IL2CPP.
