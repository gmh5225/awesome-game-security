Este proyecto es un script de Frida para recopilar datos de tiempo de ejecución de IL2CPP desde juegos de Unity. Se engancha al runtime de IL2CPP para enumerar clases, volcar firmas de métodos, extraer offsets de campos y recopilar metadatos de tipos en tiempo de ejecución mediante instrumentación de Frida. Los datos recopilados ayudan a construir SDK para el hacking de juegos IL2CPP. Está dirigido a ingenieros inversos de juegos móviles que usan Frida para analizar juegos Unity con IL2CPP.
