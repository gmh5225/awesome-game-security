Questo progetto è uno script Frida per raccogliere dati runtime IL2CPP dai giochi Unity. Si aggancia al runtime IL2CPP per enumerare le classi, dumpare le firme dei metodi, estrarre gli offset dei campi e raccogliere i metadati dei tipi a runtime tramite strumentazione Frida. I dati raccolti aiutano a costruire SDK per il game hacking su IL2CPP. È rivolto agli reverse engineer di giochi mobile che usano Frida per l'analisi di giochi Unity IL2CPP.
