Ce projet est un script Frida destiné à collecter les données d'exécution IL2CPP des jeux Unity. Il s'accroche au runtime IL2CPP pour énumérer les classes, dumper les signatures de méthodes, extraire les offsets de champs et collecter les métadonnées de types à l'exécution via l'instrumentation Frida. Les données collectées aident à construire des SDK pour le hacking de jeux IL2CPP. Il s'adresse aux reverse engineers de jeux mobiles qui utilisent Frida pour analyser les jeux Unity IL2CPP.
