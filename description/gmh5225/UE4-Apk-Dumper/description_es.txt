Este proyecto es una herramienta para volcar datos del SDK de Unreal Engine 4 desde juegos Android en formato APK. Extrae jerarquías de clases UObject, offsets de propiedades y punteros a funciones de procesos de juegos móviles UE4 en Android, para el desarrollo de cheats en juegos móviles. Está orientado a ingenieros inversos de juegos móviles que generan SDKs de UE4 a partir de paquetes de juegos Android.
