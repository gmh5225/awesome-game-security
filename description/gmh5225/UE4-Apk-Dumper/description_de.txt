Dieses Projekt ist ein Tool zum Dumping von Unreal-Engine-4-SDK-Daten aus Android-APK-Spielen. Es extrahiert UObject-Klassenhierarchien, Property-Offsets und Funktionszeiger aus UE4-Mobile-Game-Prozessen auf Android für die Entwicklung von Mobile-Game-Cheats. Es richtet sich an Reverse Engineers von Mobile Games, die UE4-SDKs aus Android-Spielpaketen erzeugen.
