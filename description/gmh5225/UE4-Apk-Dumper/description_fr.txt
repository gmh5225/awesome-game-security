Ce projet est un outil permettant d’extraire les données SDK d’Unreal Engine 4 depuis des jeux Android au format APK. Il récupère les hiérarchies de classes UObject, les offsets de propriétés et les pointeurs de fonctions depuis les processus de jeux mobiles UE4 sur Android, pour le développement de cheats mobiles. Il s’adresse aux reverse engineers de jeux mobiles qui génèrent des SDK UE4 à partir de packages de jeux Android.
