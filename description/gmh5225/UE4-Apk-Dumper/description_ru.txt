Этот проект — инструмент для дампа данных SDK Unreal Engine 4 из Android-игр в формате APK. Он извлекает иерархии классов UObject, смещения свойств и указатели функций из процессов мобильных UE4-игр на Android для разработки читов для мобильных игр. Предназначен для реверс-инженеров мобильных игр, которые генерируют UE4 SDK из Android-пакетов игр.
