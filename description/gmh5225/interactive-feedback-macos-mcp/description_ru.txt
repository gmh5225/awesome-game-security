Этот проект — нативный MCP-сервер для macOS, предназначенный для сбора интерактивной обратной связи от пользователей с помощью диалогов AppleScript и поддержкой изображений.
Это нативная для macOS переработка оригинального interactive-feedback-mcp от Fábio Ferreira (@fabiomlferreira) со следующими улучшениями.
Он в первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / MCP-серверов.
