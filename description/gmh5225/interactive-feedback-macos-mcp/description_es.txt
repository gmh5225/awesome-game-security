Este proyecto es un servidor MCP nativo para macOS destinado a recopilar comentarios interactivos de usuarios mediante diálogos de AppleScript y con soporte de imágenes.
Es una reimplementación nativa para macOS del interactive-feedback-mcp original de Fábio Ferreira (@fabiomlferreira), con las siguientes mejoras.
Resulta especialmente útil para desarrolladores de juegos, ingenieros inversos y creadores de herramientas que trabajan en el ámbito del desarrollo de juegos / servidores MCP.
