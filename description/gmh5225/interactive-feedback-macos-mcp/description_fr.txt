Ce projet est un serveur MCP natif pour macOS destiné à collecter des retours utilisateurs interactifs via des boîtes de dialogue AppleScript, avec prise en charge des images.
Il s'agit d'une réimplémentation native macOS de l'interactive-feedback-mcp original de Fábio Ferreira (@fabiomlferreira), avec les améliorations suivantes.
Il est principalement utile aux développeurs de jeux, aux reverse engineers et aux créateurs d'outils travaillant dans le domaine du développement de jeux / des serveurs MCP.
