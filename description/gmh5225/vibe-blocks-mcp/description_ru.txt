Этот проект посвящён MCP для Roblox Studio.
Он предоставляет действия Roblox Studio в виде инструментов через MCP (с использованием Server-Sent Events — SSE).
В первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / MCP-серверов.
