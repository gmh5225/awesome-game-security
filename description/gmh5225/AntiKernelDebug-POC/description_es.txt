Este proyecto es una prueba de concepto para detectar y prevenir la depuración en modo kernel en Windows. Implementa comprobaciones de detección de depuradores de kernel como WinDbg y KD mediante el estado del puerto de depuración, el indicador KdDebuggerEnabled, los campos de KUSER_SHARED_DATA y técnicas de detección basadas en interrupciones. El controlador en C demuestra técnicas anti-depuración de kernel usadas por sistemas anti-trampas y software protegido. Está dirigido a investigadores de seguridad del kernel que estudian la detección de depuradores de kernel y métodos de evasión anti-depuración.
