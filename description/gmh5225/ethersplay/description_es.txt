Este proyecto es Ethersplay, un plugin de Binary Ninja para desensamblar bytecode de la Ethereum Virtual Machine (EVM). Añade decodificación de instrucciones EVM, generación de grafos de flujo de control y análisis de referencias cruzadas para binarios de smart contracts dentro de Binary Ninja. El plugin ayuda a analizar smart contracts de Solidity compilados. Está dirigido a auditores de seguridad blockchain que analizan bytecode de smart contracts de Ethereum.
