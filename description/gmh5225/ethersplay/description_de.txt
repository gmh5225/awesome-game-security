Dieses Projekt ist Ethersplay, ein Binary-Ninja-Plugin zur Disassemblierung von Ethereum-Virtual-Machine-(EVM-)Bytecode. Es ergänzt Binary Ninja um EVM-Befehlsdekodierung, Kontrollflussgraphen und Querverweisanalyse für Smart-Contract-Binärdateien. Das Plugin unterstützt die Analyse kompilierter Solidity-Smart-Contracts. Es richtet sich an Blockchain-Sicherheitsauditoren, die Ethereum-Smart-Contract-Bytecode untersuchen.
