Ce projet est Ethersplay, un plugin Binary Ninja pour désassembler le bytecode de l'Ethereum Virtual Machine (EVM). Il ajoute le décodage des instructions EVM, la génération de graphes de flux de contrôle et l'analyse des références croisées pour les binaires de smart contracts dans Binary Ninja. Le plugin facilite l'analyse des smart contracts Solidity compilés. Il s'adresse aux auditeurs en sécurité blockchain qui analysent le bytecode des smart contracts Ethereum.
