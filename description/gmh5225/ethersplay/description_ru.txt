Этот проект — Ethersplay, плагин Binary Ninja для дизассемблирования байткода Ethereum Virtual Machine (EVM). Он добавляет декодирование инструкций EVM, построение графа потока управления и анализ перекрёстных ссылок для бинарников смарт-контрактов в Binary Ninja. Плагин помогает анализировать скомпилированные смарт-контракты на Solidity. Он предназначен для аудиторов безопасности блокчейна, исследующих байткод смарт-контрактов Ethereum.
