Ce projet est une nouvelle application de terminal pour les appareils Android.
NeoTTY a débuté comme un fork expérimental du projet d'émulateur de terminal Termux et se concentre sur une expérience d'utilisation complète d'un système Linux.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / émulateurs de terminal Android.
