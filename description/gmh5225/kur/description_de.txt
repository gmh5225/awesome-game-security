Dieses Projekt ist ein Windows-Kernel-Dienstprogramm bzw. ein Tool zur Ausnutzung von Treiberschwachstellen. Es stellt Kernel-Operationen wie Speicherlesen/-schreiben, Prozessmanipulation oder Treiberladen über einen anfälligen signierten Treiber oder eine eigene Kernel-Schnittstelle bereit. Das C/C++-Tool demonstriert Kernel-Zugriffsprimitive, die in der Spiele-Sicherheitsforschung üblich sind. Es richtet sich an Kernel-Sicherheitsforscher, die Windows-Kernel-Zugriffstechniken untersuchen.
