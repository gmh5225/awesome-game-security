Este proyecto es una utilidad del kernel de Windows o una herramienta de explotación de controladores. Proporciona operaciones a nivel de kernel como lectura/escritura de memoria, manipulación de procesos o carga de controladores mediante un controlador firmado vulnerable o una interfaz de kernel personalizada. La herramienta en C/C++ demuestra primitivas de acceso al kernel habituales en la investigación de seguridad de juegos. Está dirigida a investigadores de seguridad del kernel que estudian técnicas de acceso al kernel de Windows.
