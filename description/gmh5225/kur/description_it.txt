Questo progetto è una utility del kernel Windows o uno strumento di exploit per driver. Fornisce operazioni a livello kernel come lettura/scrittura della memoria, manipolazione dei processi o caricamento di driver tramite un driver firmato vulnerabile o un'interfaccia kernel personalizzata. Lo strumento C/C++ dimostra primitive di accesso al kernel comunemente usate nella ricerca sulla sicurezza dei giochi. È rivolto ai ricercatori di sicurezza del kernel che studiano le tecniche di accesso al kernel Windows.
