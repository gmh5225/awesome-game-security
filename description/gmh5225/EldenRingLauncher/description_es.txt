Lanzador personalizado de Elden Ring creado con ImGui (renderizado mediante SDL2) que ofrece una interfaz gráfica para configurar y lanzar el juego con ajustes modificados, incluidas opciones de evasión del anti-cheat y capacidades de carga de mods.
La aplicación incluye la biblioteca ImGui completa con el backend de renderizado SDL2, usa una biblioteca Resources para los recursos incrustados y presenta una ventana sencilla para activar o desactivar parámetros de lanzamiento antes de iniciar el ejecutable del juego.
Resulta especialmente útil para modders de Elden Ring e investigadores de seguridad en videojuegos que estudian implementaciones de lanzadores personalizados que interactúan con Easy Anti-Cheat o lo evaden.
