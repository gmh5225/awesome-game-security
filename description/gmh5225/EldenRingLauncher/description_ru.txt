Пользовательский лаунчер Elden Ring на ImGui (отрисовка через SDL2), предоставляющий графический интерфейс для настройки и запуска игры с изменёнными параметрами, включая опции обхода античита и загрузку модов.
Приложение включает полную библиотеку ImGui с бэкендом рендеринга SDL2, использует библиотеку Resources для встроенных ресурсов и показывает удобное окно для переключения параметров запуска перед стартом исполняемого файла игры.
В основном полезен моддерам Elden Ring и исследователям игровой безопасности, изучающим реализации пользовательских лаунчеров, которые взаимодействуют с Easy Anti-Cheat или обходят его.
