Этот проект — плагин IDA Pro для управления и изменения имён функций и символов в базах данных IDA. Он обеспечивает пакетное переименование, сопоставление по шаблонам имён, операции с префиксами/суффиксами, а также импорт/экспорт имён для организации больших баз дизассемблирования. Плагин на Python упрощает рабочий процесс именования при работе с крупными бинарными файлами. Он предназначен для специалистов по реверс-инжинирингу, управляющих именами символов в сложных проектах анализа в IDA Pro.
