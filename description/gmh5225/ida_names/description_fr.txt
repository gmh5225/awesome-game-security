Ce projet est un plugin IDA Pro destiné à gérer et manipuler les noms de fonctions et de symboles dans les bases de données IDA. Il offre le renommage par lots, la correspondance de motifs de noms, des opérations de préfixe/suffixe, ainsi que l'import/export de noms pour organiser de grandes bases de désassemblage. Ce plugin Python simplifie le flux de travail de nommage lors de l'analyse de gros binaires. Il s'adresse aux ingénieurs en reverse engineering qui gèrent les noms de symboles dans des projets d'analyse IDA Pro complexes.
