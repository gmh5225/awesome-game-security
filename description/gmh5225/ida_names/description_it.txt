Questo progetto è un plugin di IDA Pro per gestire e manipolare i nomi di funzioni e simboli nei database IDA. Offre ridenominazione in batch, corrispondenza di pattern dei nomi, operazioni di prefisso/suffisso e funzionalità di importazione/esportazione dei nomi per organizzare grandi database di disassemblaggio. Il plugin Python semplifica il flusso di lavoro di naming quando si lavora con binari di grandi dimensioni. È rivolto ai reverse engineer che gestiscono i nomi dei simboli in progetti di analisi IDA Pro complessi.
