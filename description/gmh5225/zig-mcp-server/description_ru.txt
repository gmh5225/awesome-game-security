Этот проект — высокопроизводительная реализация протокола MCP на Zig.
Zig MCP Server — это эффективная по памяти реализация спецификации Model Context Protocol (MCP).
Он в первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / MCP-серверов.
