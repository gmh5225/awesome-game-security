Ce projet est une implémentation haute performance du protocole MCP en Zig.
Le Zig MCP Server est une implémentation économe en mémoire de la spécification Model Context Protocol (MCP).
Il s'adresse principalement aux développeurs de jeux, aux ingénieurs en rétro-ingénierie et aux créateurs d'outils travaillant dans le domaine du développement de jeux / serveurs MCP.
