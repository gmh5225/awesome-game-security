Este proyecto es un emulador de servidor privado de MapleStory escrito en Rust. Implementa la lógica de juego del lado del servidor para una versión de MapleStory, incluyendo servidores de inicio de sesión, de personajes y de canales, con gestión de jugadores, interacción con NPC, sistemas de misiones y mecánicas de combate. La implementación en Rust enfatiza la seguridad de memoria y el rendimiento. Está dirigido a desarrolladores de servidores privados de MapleStory que exploran Rust para el desarrollo de servidores de juegos.
