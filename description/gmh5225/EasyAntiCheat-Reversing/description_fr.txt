Ce projet est un dump de reverse engineering direct d'EasyAntiCheat.sys préparé dans IDA Pro 7.7, et non une base de code nettoyée ou reconstruite.
L'archive se compose surtout d'une sortie décompilée de type C pour le pilote, préservant les noms, constantes, chaînes de registre et routines liées aux politiques sous une forme consultable sans rouvrir l'IDB d'origine.
Comme il n'y a presque aucune logique d'encapsulation au-delà de la décompilation elle-même, le dépôt convient surtout comme instantané de référence consultable pour le reverse engineering d'EAC.
Il est surtout utile aux reverse engineers qui veulent une copie consultable par texte de la logique du pilote Easy Anti-Cheat, des chemins de politiques et du flot de contrôle décompilé hors d'IDA.
