Этот проект — прямой дамп реверса EasyAntiCheat.sys, подготовленный в IDA Pro 7.7, а не очищенная или пересобранная кодовая база.
Архив в основном состоит из декомпилированного C-подобного вывода драйвера с сохранением имён, констант, строк реестра и связанных с политиками процедур в виде, удобном для поиска без повторного открытия исходного IDB.
Поскольку помимо самой декомпиляции почти нет обёрточной логики, репозиторий лучше рассматривать как поисковый эталонный снимок для реверс-инжиниринга EAC.
Он в первую очередь полезен реверс-инженерам, которым нужна текстово-поисковая копия логики драйвера Easy Anti-Cheat, путей политик и декомпилированного потока управления вне IDA.
