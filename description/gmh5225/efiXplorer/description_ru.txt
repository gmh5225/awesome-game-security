Этот проект — efiXplorer, плагин IDA Pro для автоматического анализа бинарных файлов прошивки UEFI. Он идентифицирует протоколы EFI по совпадению GUID, аннотирует вызовы таблиц Boot Services и Runtime Services, разрешает использование интерфейсов протоколов и восстанавливает зависимости драйверов PEI/DXE. Плагин существенно снижает объём ручной работы при реверс-инжиниринге прошивок UEFI. Он предназначен для исследователей безопасности прошивок, анализирующих реализации UEFI BIOS, буткиты и вредоносное ПО уровня EFI.
