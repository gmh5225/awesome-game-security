Dieses Projekt ist efiXplorer, ein IDA-Pro-Plugin zur automatisierten Analyse von UEFI-Firmware-Binärdateien. Es erkennt EFI-Protokolle per GUID-Abgleich, annotiert Aufrufe der Boot-Services- und Runtime-Services-Tabellen, löst die Nutzung von Protokollschnittstellen auf und rekonstruiert PEI/DXE-Treiberabhängigkeiten. Das Plugin reduziert den manuellen Aufwand beim Reverse Engineering von UEFI-Firmware deutlich. Es richtet sich an Firmware-Sicherheitsforscher, die UEFI-BIOS-Implementierungen, Bootkits und Malware auf EFI-Ebene analysieren.
