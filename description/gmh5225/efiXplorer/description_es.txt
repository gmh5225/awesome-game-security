Este proyecto es efiXplorer, un complemento de IDA Pro para el análisis automatizado de binarios de firmware UEFI. Identifica protocolos EFI mediante coincidencia de GUID, anota las llamadas a las tablas Boot Services y Runtime Services, resuelve el uso de interfaces de protocolo y reconstruye las dependencias de controladores PEI/DXE. El complemento reduce significativamente el esfuerzo manual en la ingeniería inversa de firmware UEFI. Está dirigido a investigadores de seguridad de firmware que analizan implementaciones de BIOS UEFI, bootkits y malware a nivel EFI.
