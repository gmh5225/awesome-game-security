Ce projet est un serveur Cheat Engine (ceserver) qui utilise PCILeech pour un accès mémoire basé sur le DMA. Il implémente le protocole ceserver sur un backend PCILeech/LeechCore, permettant à Cheat Engine de scanner et de modifier la mémoire des processus via du matériel DMA sur une machine séparée. Cela rend l'accès mémoire complètement invisible pour l'anti-cheat du système cible. Il s'adresse aux chercheurs en sécurité DMA qui utilisent Cheat Engine via du matériel PCILeech.
