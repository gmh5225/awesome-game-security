Ce projet est un désassembleur VMProtect 3 x64 qui analyse et désassemble le bytecode de la machine virtuelle de VMProtect 3. Il décode le jeu d'instructions personnalisé de VMP3, identifie les gestionnaires d'opcodes virtuels et reconstruit le flux d'instructions d'origine à partir du code virtualisé. L'outil cible la variante 64 bits de la machine virtuelle de VMProtect. Il s'adresse aux ingénieurs en reverse engineering travaillant à la désobfuscation de binaires 64 bits protégés par VMProtect 3.
