Этот проект — дизассемблер VMProtect 3 x64, который анализирует и дизассемблирует байткод виртуальной машины VMProtect 3. Он декодирует пользовательский набор инструкций VMP3, определяет обработчики виртуальных опкодов и восстанавливает исходный поток инструкций из виртуализированного кода. Инструмент ориентирован на 64-битную разновидность виртуальной машины VMProtect. Он предназначен для специалистов по реверс-инжинирингу, занимающихся деобфускацией 64-битных бинарных файлов, защищённых VMProtect 3.
