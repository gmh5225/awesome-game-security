Questo progetto è un disassemblatore VMProtect 3 x64 che analizza e disassembla il bytecode della macchina virtuale di VMProtect 3. Decodifica il set di istruzioni personalizzato di VMP3, identifica i gestori degli opcode virtuali e ricostruisce il flusso di istruzioni originale dal codice virtualizzato. Lo strumento si rivolge alla variante a 64 bit della macchina virtuale di VMProtect. È pensato per i reverse engineer che lavorano alla deoffuscazione di binari a 64 bit protetti da VMProtect 3.
