Этот проект — улучшенный лаунчер для Honkai Impact 3rd, предоставляющий возможности сверх официального лаунчера. Он может включать управление загрузками, переключение версий, выбор сервера, проверку игровых файлов и параметры конфигурации, недоступные в стандартном лаунчере. Предназначен для игроков Honkai Impact 3rd, которым нужны расширенные функции лаунчера и управление игрой.
