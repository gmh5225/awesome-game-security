Ce projet se concentre sur l'approche internal.
Seules les fonctionnalités visuelles visibles dans le fichier image.png du dépôt sont actuellement activées ; l'aimbot, les exploits, etc. sont présents mais désactivés.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / game:Fortnite.
