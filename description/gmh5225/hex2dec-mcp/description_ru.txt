Этот проект посвящён MCP-серверу, обеспечивающему преобразование между шестнадцатеричными и десятичными числами.
Он в основном написан на JavaScript и TypeScript.
Он особенно полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / MCP-серверов.
