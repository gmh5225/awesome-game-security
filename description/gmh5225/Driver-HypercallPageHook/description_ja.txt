本プロジェクトは、Hyper-V 関連のカーネル hypercall 遷移で使われる hypercall ページポインタ `nt!HvcallCodeVa` をフックする概念実証です。
ドライバは `HvlInvokeHypercall` 付近で hypercall ページ参照を特定し、カスタムディスパッチャに差し替え、`HvlEnlightenments` フラグを切り替えることで、`HvlSwitchVirtualAddressSpace` などのコンテキストスイッチ関連 hypercall をフック経由でルーティングします。
実装は C++ のドライバエントリ層と、hypercall 入力コードを検査し対象操作をカスタムコールバックへ転送し、それ以外は元の hypercall ページへフォールバックするアセンブリディスパッチャに分かれています。
主に、トレースや実験のためにカーネル内で Hyper-V hypercall ディスパッチをどのように傍受できるかを研究したい、低レベル Windows / 仮想化研究者向けです。
