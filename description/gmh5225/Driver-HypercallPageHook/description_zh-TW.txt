本專案是一個概念驗證，用於掛鉤 `nt!HvcallCodeVa`，亦即 Hyper-V 相關核心 hypercall 轉換所使用的 hypercall 頁面指標。
驅動程式在 `HvlInvokeHypercall` 附近定位該 hypercall 頁面參照，將其替換為自訂分派器，並翻轉 `HvlEnlightenments` 旗標，從而使諸如 `HvlSwitchVirtualAddressSpace` 等與內容切換相關的 hypercall 經由此掛鉤路由。
其實作分為 C++ 驅動進入層與組譯分派器：後者檢查 hypercall 輸入代碼，將目標操作轉發至自訂回呼，其餘則回退到原始 hypercall 頁面。
它主要適用於希望研究如何在核心內攔截 Hyper-V hypercall 分派以進行追蹤或實驗的底層 Windows 與虛擬化研究人員。
