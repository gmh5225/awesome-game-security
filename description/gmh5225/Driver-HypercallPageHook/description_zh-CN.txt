本项目是一个概念验证，用于挂钩 `nt!HvcallCodeVa`，即 Hyper-V 相关内核 hypercall 转换所使用的 hypercall 页面指针。
驱动程序在 `HvlInvokeHypercall` 附近定位该 hypercall 页面引用，将其替换为自定义分发器，并翻转 `HvlEnlightenments` 标志，从而使诸如 `HvlSwitchVirtualAddressSpace` 等与上下文切换相关的 hypercall 经由此钩子路由。
其实现分为 C++ 驱动入口层与汇编分发器：后者检查 hypercall 输入代码，将目标操作转发到自定义回调，其余则回退到原始 hypercall 页面。
它主要适用于希望研究如何在内核内拦截 Hyper-V hypercall 分发以进行跟踪或实验的底层 Windows 与虚拟化研究人员。
