이 프로젝트는 Hyper-V 관련 커널 hypercall 전환에 사용되는 hypercall 페이지 포인터인 `nt!HvcallCodeVa`를 후킹하는 개념 증명입니다.
드라이버는 `HvlInvokeHypercall` 근처에서 hypercall 페이지 참조를 찾아 사용자 정의 디스패처로 교체하고, `HvlEnlightenments` 플래그를 뒤집어 `HvlSwitchVirtualAddressSpace`와 같은 컨텍스트 스위치 관련 hypercall이 훅을 통해 라우팅되도록 합니다.
구현은 C++ 드라이버 엔트리 계층과, hypercall 입력 코드를 검사해 대상 작업을 사용자 정의 콜백으로 전달하고 그 외에는 원래 hypercall 페이지로 폴백하는 어셈블리 디스패처로 나뉩니다.
추적이나 실험을 위해 커널 내부에서 Hyper-V hypercall 디스패치를 어떻게 가로챌 수 있는지 연구하려는 저수준 Windows 및 가상화 연구원에게 주로 유용합니다.
