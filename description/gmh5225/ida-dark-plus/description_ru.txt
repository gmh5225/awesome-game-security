Этот проект — тёмная тема для IDA Pro, которая задаёт современную тёмную цветовую схему для вида дизассемблирования, hex-вида, структур и всех окон IDA. Она заменяет стандартную палитру IDA тёмным фоном и цветами переднего плана с подсветкой синтаксиса, улучшая читаемость при длительных сессиях анализа. Предназначена для reverse engineers, которые предпочитают тёмный интерфейс в IDA Pro.
