Dieses Projekt ist eine modifizierte Version von MinHook, die ausschließlich die Windows Native API verwendet.
Es ist hauptsächlich in C/C++ und C geschrieben und konzentriert sich auf Asset-Pipelines, Modding und Hooking.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats und Hooks untersuchen.
