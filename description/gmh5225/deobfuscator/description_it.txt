Questo progetto è un deoffuscatore di bytecode Java per annullare l'offuscamento applicato da strumenti come ProGuard, Allatori, ZKM e altri offuscatori Java. Applica passaggi di trasformazione per rimuovere la cifratura delle stringhe, ripristinare il flusso di controllo, semplificare i predicati opachi, rinominare gli identificatori offuscati e ripulire il codice morto nei file delle classi Java. È rivolto a reverse engineer Java e analisti di malware che deoffuscano applicazioni Java/Android protette.
