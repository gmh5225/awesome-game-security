Este proyecto es un desofuscador de bytecode Java para revertir la ofuscación aplicada por herramientas como ProGuard, Allatori, ZKM y otros ofuscadores Java. Aplica pases de transformación para eliminar el cifrado de cadenas, restaurar el flujo de control, simplificar predicados opacos, renombrar identificadores ofuscados y limpiar código muerto en archivos de clases Java. Está orientado a ingenieros de ingeniería inversa de Java y analistas de malware que desofuscan aplicaciones Java/Android protegidas.
