Ce projet est un désobfuscateur de bytecode Java destiné à inverser l'obfuscation appliquée par des outils comme ProGuard, Allatori, ZKM et d'autres obfuscateurs Java. Il applique des passes de transformation pour supprimer le chiffrement de chaînes, restaurer le flot de contrôle, simplifier les prédicats opaques, renommer les identifiants obfusqués et nettoyer le code mort dans les fichiers de classes Java. Il s'adresse aux ingénieurs en rétroconception Java et aux analystes de malwares qui désobfusquent des applications Java/Android protégées.
