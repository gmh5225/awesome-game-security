Этот проект — деобфускатор Java-байткода для снятия обфускации, применённой такими средствами, как ProGuard, Allatori, ZKM и другими Java-обфускаторами. Он выполняет проходы трансформации для удаления шифрования строк, восстановления потока управления, упрощения непрозрачных предикатов, переименования обфусцированных идентификаторов и очистки мёртвого кода в Java class-файлах. Предназначен для Java reverse-инженеров и аналитиков вредоносного ПО, деобфусцирующих защищённые Java/Android-приложения.
