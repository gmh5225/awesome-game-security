Ce projet est sk3wldbg, un plugin IDA Pro qui intègre l’émulateur CPU Unicorn pour l’émulation de code directement dans IDA. Il permet d’exécuter des régions de code sélectionnées via le moteur Unicorn, avec prise en charge de x86, ARM, MIPS et d’autres architectures. Les analystes peuvent configurer l’état des registres et de la mémoire, avancer pas à pas dans le code émulé et observer les résultats sans cible active. Il s’adresse aux reverse engineers qui ont besoin d’une émulation de code rapide lors de l’analyse statique dans IDA Pro.
