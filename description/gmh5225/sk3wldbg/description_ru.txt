Этот проект — sk3wldbg, плагин для IDA Pro, интегрирующий эмулятор CPU Unicorn для эмуляции кода внутри IDA. Он позволяет выполнять выбранные фрагменты кода с помощью движка Unicorn и поддерживает x86, ARM, MIPS и другие архитектуры. Аналитики могут задавать состояние регистров и памяти, пошагово выполнять эмулируемый код и наблюдать результаты без живой цели. Плагин предназначен для исследователей, которым нужна быстрая эмуляция кода при статическом анализе в IDA Pro.
