Este proyecto es sk3wldbg, un complemento de IDA Pro que integra el emulador de CPU Unicorn para emular código dentro de IDA. Permite ejecutar regiones de código seleccionadas con el motor Unicorn y admite x86, ARM, MIPS y otras arquitecturas. Los analistas pueden configurar el estado de registros y memoria, avanzar paso a paso por el código emulado y observar los resultados sin un objetivo en ejecución. Está orientado a ingenieros inversos que necesitan emulación rápida de código durante el análisis estático en IDA Pro.
