Questo progetto è una raccolta di firme a pattern di byte per individuare funzioni e strutture dati chiave nel binario di Fortnite. Fornisce pattern di firma in stile IDA per trovare GObjects, GNames, ProcessEvent, funzioni del controller del giocatore e altri punti di ingresso critici del motore necessari per lo sviluppo di cheat. Le firme vengono aggiornate per versioni specifiche di Fortnite. È pensato per i ricercatori di sicurezza dei giochi che tracciano i cambiamenti del binario di Fortnite e per gli sviluppatori di cheat che mantengono l’accuratezza degli offset.
