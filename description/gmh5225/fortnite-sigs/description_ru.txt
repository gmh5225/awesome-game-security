Этот проект — коллекция байтовых сигнатур-паттернов для поиска ключевых функций и структур данных в игровом бинарнике Fortnite. Он предоставляет сигнатуры в стиле IDA для нахождения GObjects, GNames, ProcessEvent, функций контроллера игрока и других критических точек входа движка, необходимых для разработки читов. Сигнатуры обновляются под конкретные версии Fortnite. Предназначен для исследователей игровой безопасности, отслеживающих изменения бинарника Fortnite, и разработчиков читов, поддерживающих точность смещений (offsets).
