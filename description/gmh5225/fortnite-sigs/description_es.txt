Este proyecto es una colección de firmas de patrones de bytes para localizar funciones y estructuras de datos clave en el binario de Fortnite. Proporciona patrones de firma al estilo de IDA para encontrar GObjects, GNames, ProcessEvent, funciones del controlador de jugador y otros puntos de entrada críticos del motor necesarios para el desarrollo de cheats. Las firmas se actualizan para versiones específicas de Fortnite. Está dirigido a investigadores de seguridad de juegos que siguen los cambios del binario de Fortnite y a desarrolladores de cheats que mantienen la precisión de los offsets.
