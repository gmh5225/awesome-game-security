Этот проект использует драйвер антивирусного ПО Zemana (zam64.sys) для операций в ядре. Интерфейс IOCTL драйвера Zemana позволяет завершать процессы и получать доступ к памяти, что делает его известной целью BYOVD для уничтожения процессов безопасности. Проект предназначен для исследователей BYOVD, изучающих эксплуатацию драйвера Zemana.
