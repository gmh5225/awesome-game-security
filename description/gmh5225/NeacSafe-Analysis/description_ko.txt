이 프로젝트는 NetEase의 NeacSafe 안티치트 통신 인터페이스를 분석하기 위한 소규모 사용자 모드 프로브와 저장된 노트입니다.
C++ 샘플은 0x28바이트 NeacSafeConnectContext를 정의하고, FilterConnectCommunicationPort로 \NeacSafePort에 연결한 뒤, 인코딩된 요청 버퍼를 전송하여 minifilter 통신 채널을 통해 데이터를 조회합니다.
아카이브에는 NeacSafe 인터페이스 분석에 관한 저장된 Pediy 포럼 글도 포함되어 있어, 이 저장소는 포트 프로토콜의 실용적인 재현과 원본 분석 자료를 함께 제공합니다.
NeacSafe 클라이언트-드라이버 IPC, Filter Manager 통신 포트, 그리고 명령 메시지 주변의 버퍼 인코딩을 연구하는 리버스 엔지니어에게 주로 유용합니다.
