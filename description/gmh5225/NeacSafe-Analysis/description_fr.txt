Ce projet est une petite sonde en mode utilisateur accompagnée de notes enregistrées pour analyser l'interface de communication de l'anti-cheat NeacSafe de NetEase.
L'exemple C++ définit un NeacSafeConnectContext de 0x28 octets, se connecte à \NeacSafePort via FilterConnectCommunicationPort, puis envoie des tampons de requête encodés pour interroger des données via le canal de communication du minifilter.
L'archive inclut également un article sauvegardé du forum Pediy sur l'analyse de l'interface NeacSafe, de sorte que le dépôt sert de reproduction pratique du protocole de port aux côtés du matériel d'écriture original.
Il est surtout utile aux reverse engineers qui étudient l'IPC client-pilote de NeacSafe, les ports de communication du Filter Manager, et l'encodage des tampons autour de ses messages de commande.
