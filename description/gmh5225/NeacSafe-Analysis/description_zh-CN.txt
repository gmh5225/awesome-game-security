本项目是一个小型用户态探测程序，并附带用于分析网易 NeacSafe 反作弊通信接口的保存笔记。
该 C++ 示例定义了 0x28 字节的 NeacSafeConnectContext，通过 FilterConnectCommunicationPort 连接到 \NeacSafePort，随后发送编码后的请求缓冲区，经 minifilter 通信通道查询数据。
归档中还包含一篇关于 NeacSafe 接口分析的保存 Pediy 论坛文章，因此本仓库既是对端口协议的实用复现，也保留了原始分析材料。
它主要适用于研究 NeacSafe 客户端与驱动 IPC、Filter Manager 通信端口及其命令消息周围缓冲区编码的逆向工程师。
