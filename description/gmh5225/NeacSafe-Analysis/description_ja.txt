本プロジェクトは、NetEase の NeacSafe アンチチート通信インターフェイスを解析するための、小さなユーザーモードプローブと保存済みノートです。
この C++ サンプルは 0x28 バイトの NeacSafeConnectContext を定義し、FilterConnectCommunicationPort で \NeacSafePort に接続したうえで、エンコード済みリクエストバッファを送信して minifilter 通信チャネル経由でデータを照会します。
アーカイブには NeacSafe インターフェイス解析に関する Pediy フォーラム記事の保存版も含まれており、本リポジトリはポートプロトコルの実用的な再現と、元の書き起こし資料の両方として機能します。
NeacSafe のクライアント・ドライバ間 IPC、Filter Manager 通信ポート、およびコマンドメッセージ周辺のバッファエンコーディングを研究するリバースエンジニア向けです。
