Dieses Projekt ist eine kleine User-Mode-Sonde plus gespeicherte Notizen zur Analyse der Kommunikationsschnittstelle von NetEases NeacSafe-Anti-Cheat.
Das C++-Beispiel definiert einen 0x28-Byte großen NeacSafeConnectContext, verbindet sich mit FilterConnectCommunicationPort zu \NeacSafePort und sendet anschließend kodierte Anfragepuffer, um Daten über den Minifilter-Kommunikationskanal abzufragen.
Das Archiv enthält außerdem einen gespeicherten Pediy-Forumartikel zur NeacSafe-Schnittstellenanalyse, sodass das Repository als praktische Reproduktion des Port-Protokolls neben dem ursprünglichen Writeup-Material dient.
Es ist vor allem für Reverse Engineers nützlich, die NeacSafe-Client-Treiber-IPC, Filter-Manager-Kommunikationsports und die Pufferkodierung rund um seine Befehlsnachrichten untersuchen.
