Questo progetto è una piccola sonda in user-mode con note salvate per analizzare l'interfaccia di comunicazione dell'anti-cheat NeacSafe di NetEase.
L'esempio C++ definisce un NeacSafeConnectContext di 0x28 byte, si connette a \NeacSafePort con FilterConnectCommunicationPort e poi invia buffer di richiesta codificati per interrogare dati tramite il canale di comunicazione del minifilter.
L'archivio include anche un articolo salvato del forum Pediy sull'analisi dell'interfaccia NeacSafe, così il repository funge da riproduzione pratica del protocollo della porta insieme al materiale originale del writeup.
È soprattutto utile per i reverse engineer che studiano l'IPC client-driver di NeacSafe, le porte di comunicazione del Filter Manager e la codifica dei buffer intorno ai suoi messaggi di comando.
