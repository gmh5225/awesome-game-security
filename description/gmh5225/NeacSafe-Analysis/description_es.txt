Este proyecto es una pequeña sonda en modo usuario más notas guardadas para analizar la interfaz de comunicación del anti-cheat NeacSafe de NetEase.
El ejemplo en C++ define un NeacSafeConnectContext de 0x28 bytes, se conecta a \NeacSafePort con FilterConnectCommunicationPort y luego envía búferes de solicitud codificados para consultar datos a través del canal de comunicación del minifilter.
El archivo también incluye un artículo guardado del foro Pediy sobre el análisis de la interfaz NeacSafe, de modo que el repositorio actúa como una reproducción práctica del protocolo del puerto junto al material original del writeup.
Resulta especialmente útil para ingenieros inversos que estudian la IPC cliente-controlador de NeacSafe, los puertos de comunicación del Filter Manager y la codificación de búferes en torno a sus mensajes de comando.
