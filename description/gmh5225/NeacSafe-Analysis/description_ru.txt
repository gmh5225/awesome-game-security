Этот проект — небольшой user-mode зонд и сохранённые заметки для анализа коммуникационного интерфейса античита NeacSafe от NetEase.
Пример на C++ определяет NeacSafeConnectContext размером 0x28 байт, подключается к \NeacSafePort через FilterConnectCommunicationPort и затем отправляет закодированные буферы запросов для получения данных по каналу связи minifilter.
В архиве также есть сохранённая статья с форума Pediy об анализе интерфейса NeacSafe, поэтому репозиторий служит практической реконструкцией протокола порта наряду с исходным материалом разбора.
Он в первую очередь полезен реверс-инженерам, изучающим IPC между клиентом и драйвером NeacSafe, порты связи Filter Manager и кодирование буферов вокруг его командных сообщений.
