Este proyecto es un copiloto de automatización para Honkai: Star Rail que ayuda con tareas diarias, farmeo de recursos y jugabilidad repetitiva. Usa reconocimiento de pantalla y entrada automatizada para navegar menús, completar misiones diarias, gastar resistencia y gestionar asignaciones. La herramienta en Python ofrece estrategias de automatización configurables para distintos objetivos de farmeo. Está dirigida a jugadores de Honkai: Star Rail que automatizan rutinas diarias y a desarrolladores de automatización de juegos.
