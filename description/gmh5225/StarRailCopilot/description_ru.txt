Этот проект — автоматизационный копилот для Honkai: Star Rail, который помогает с ежедневными заданиями, фармом ресурсов и повторяющимся геймплеем. Он использует распознавание экрана и автоматизированный ввод для навигации по меню, выполнения ежедневных миссий, расходования выносливости и управления поручениями. Инструмент на Python предоставляет настраиваемые стратегии автоматизации для разных целей фарма. Он предназначен для игроков Honkai: Star Rail, автоматизирующих ежедневные рутины, и для разработчиков игровой автоматизации.
