Этот проект представляет собой зеркало или коллекцию файлов IDA Pro SDK для разработки плагинов. Он предоставляет заголовочные файлы C++, библиотеки и документацию, необходимые для разработки плагинов IDA Pro и процессорных модулей для нескольких версий IDA. Предназначен для разработчиков плагинов IDA Pro, которым нужны файлы SDK для создания пользовательских инструментов анализа и расширений.
