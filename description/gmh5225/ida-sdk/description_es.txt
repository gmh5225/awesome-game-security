Este proyecto es un espejo o una colección de archivos del SDK de IDA Pro para el desarrollo de plugins. Proporciona los archivos de encabezado C++, las bibliotecas y la documentación necesarios para desarrollar plugins de IDA Pro y módulos de procesador en varias versiones de IDA. Está dirigido a desarrolladores de plugins de IDA Pro que necesitan archivos del SDK para crear herramientas de análisis y extensiones personalizadas.
