Ce projet est un miroir ou une collection de fichiers SDK d'IDA Pro destinés au développement de plugins. Il fournit les fichiers d'en-tête C++, les bibliothèques et la documentation nécessaires pour développer des plugins IDA Pro et des modules processeur sur plusieurs versions d'IDA. Il s'adresse aux développeurs de plugins IDA Pro qui ont besoin des fichiers SDK pour créer des outils d'analyse et des extensions personnalisés.
