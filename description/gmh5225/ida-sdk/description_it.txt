Questo progetto è un mirror o una raccolta di file SDK di IDA Pro per lo sviluppo di plugin. Fornisce i file header C++, le librerie e la documentazione necessari per sviluppare plugin IDA Pro e moduli processore su più versioni di IDA. È rivolto agli sviluppatori di plugin IDA Pro che necessitano dei file SDK per creare strumenti di analisi ed estensioni personalizzati.
