Este proyecto es Rebirth, un emulador de servidor privado de MapleStory para una versión específica del juego. Implementa la lógica del juego del lado del servidor, incluyendo autenticación de inicio de sesión, progresión de personajes, instancias de mapas, IA de monstruos, procesamiento de habilidades, misiones y gestión de grupos/gremios. El código emula el protocolo cliente-servidor de MapleStory. Está dirigido a desarrolladores de servidores privados de MapleStory e investigadores de arquitectura de servidores de juegos.
