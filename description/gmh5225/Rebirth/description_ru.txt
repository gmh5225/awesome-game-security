Этот проект — Rebirth, эмулятор приватного сервера MapleStory для определённой версии игры. Он реализует серверную игровую логику, включая аутентификацию при входе, прогрессию персонажей, экземпляры карт, ИИ монстров, обработку навыков, квесты и управление партиями/гильдиями. Кодовая база эмулирует клиент-серверный протокол MapleStory. Предназначен для разработчиков приватных серверов MapleStory и исследователей архитектуры игровых серверов.
