Questo progetto è Rebirth, un emulatore di server privato MapleStory per una versione specifica del gioco. Implementa la logica di gioco lato server, inclusa l'autenticazione di login, la progressione dei personaggi, le istanze delle mappe, l'IA dei mostri, l'elaborazione delle abilità, le missioni e la gestione di party/gilde. Il codice emula il protocollo client-server di MapleStory. È rivolto agli sviluppatori di server privati MapleStory e ai ricercatori di architetture di server di gioco.
