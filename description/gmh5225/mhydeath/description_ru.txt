Этот проект использует уязвимости античитового драйвера ядра mhyprot2.sys от miHoYo для произвольных операций в ядре. Драйвер mhyprot2, изначально подписанный для античита Genshin Impact, содержит уязвимости, позволяющие непривилегированным пользователям читать/писать память ядра и завершать процессы. Этот инструмент BYOVD опирается на mhyprot2 для доступа к ядру. Он предназначен для исследователей BYOVD, изучающих, как уязвимости античитовых драйверов игр превращаются в средство системного доступа.
