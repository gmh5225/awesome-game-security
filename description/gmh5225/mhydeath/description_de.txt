Dieses Projekt nutzt den Anti-Cheat-Kerneltreiber mhyprot2.sys von miHoYo für beliebige Kerneloperationen aus. Der ursprünglich für das Anti-Cheat von Genshin Impact signierte mhyprot2-Treiber enthält Schwachstellen, die unprivilegierten Benutzern das Lesen/Schreiben von Kernel-Speicher und das Beenden von Prozessen ermöglichen. Dieses BYOVD-Tool nutzt mhyprot2 für Kernelzugriff. Es richtet sich an BYOVD-Forscher, die untersuchen, wie Schwachstellen in Spiel-Anti-Cheat-Treibern für systemweiten Zugriff missbraucht werden.
