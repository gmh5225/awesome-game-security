Este proyecto explota el controlador de kernel antitrampas mhyprot2.sys de miHoYo para operaciones arbitrarias en el kernel. El controlador mhyprot2, firmado originalmente para el antitrampas de Genshin Impact, contiene vulnerabilidades que permiten a usuarios sin privilegios leer/escribir memoria del kernel y terminar procesos. Esta herramienta BYOVD aprovecha mhyprot2 para obtener acceso al kernel. Está dirigida a investigadores BYOVD que estudian cómo las vulnerabilidades de controladores antitrampas de juegos se aprovechan como arma para acceso a nivel de sistema.
