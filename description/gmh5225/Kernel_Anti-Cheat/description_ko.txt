이 프로젝트는 단일 탐지 경로에만 집중하지 않고 여러 텔레메트리 아이디어를 결합한 실험적 커널 안티치트 드라이버입니다.
주요 구성 요소로는 HalSendNMI와 RtlCaptureStackBackTrace를 이용한 NMI 기반 스택 워킹, 시스템 스레드 시작 주소 스캔, big-pool 검사, 부트 UUID 수집, 간단한 하이퍼바이저 검사, 그리고 kdmapper 또는 drvmap 타임스탬프를 위한 PiDDBCacheTable 열거가 포함됩니다.
코드는 가능한 오탐을 명시적으로 다루며, 연구용 샌드박스처럼 읽히도록 NMI 캡처, 스레드 스캔, 풀 분석, 하이퍼바이저 탐지, 트레이스 산출물이 각각 별도 소스 파일로 나뉘어 있습니다.
스택 포렌식, 모듈 범위 검증, 매퍼 잔여물 검사를 하나의 드라이버에서 결합하는 커널 안티치트 프로토타입을 연구하는 방어 보안 연구자에게 주로 유용합니다.
