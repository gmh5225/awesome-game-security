Este proyecto es una versión modificada del spoofer de identificador de hardware BTBD, adaptada para manual mapping en sistemas de la era Windows 1909.
Su código engancha las rutas de control de discos y particiones, reescribe los datos relacionados con el número de serie que devuelven los IOCTL de almacenamiento, borra identificadores GPT, actualiza propiedades del disco, desactiva la predicción de fallos SMART y también recorre unidades RAID de storahci para sobrescribir cadenas de serie más abajo en la pila de almacenamiento.
El resultado no es solo un parche de un único número de serie, sino una canalización más amplia de spoofing de identidad de almacenamiento que intenta mantener coherentes varias superficies de consulta relacionadas con el disco tras cargar el controlador.
Es útil principalmente para investigadores del kernel de Windows que analizan técnicas de spoofing de HWID en la pila de almacenamiento, especialmente las derivadas de controladores mapeados manualmente al estilo BTBD.
