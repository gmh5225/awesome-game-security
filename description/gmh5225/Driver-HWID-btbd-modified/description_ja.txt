本プロジェクトは、Windows 1909 世代向けに manual mapping 用へ調整した BTBD ハードウェア識別子スプーファーの改変版です。
コードはディスクおよびパーティション制御パスをフックし、ストレージ IOCTL が返すシリアル関連データを書き換え、GPT 識別子を消去し、ディスクプロパティを更新し、SMART 障害予測を無効化し、さらに storahci の RAID ユニットを走査してストレージスタックのより深い層でシリアル文字列を上書きします。
結果は単一のシリアルパッチではなく、ドライバ読み込み後に複数のディスク関連問い合わせ面を一貫させようとする、より広いストレージ識別スプーフィングパイプラインになります。
主に、BTBD 系の手動マップドドライバに由来するものを含め、ストレージスタックの HWID スプーフィング手法を分析する Windows カーネル研究者向けです。
