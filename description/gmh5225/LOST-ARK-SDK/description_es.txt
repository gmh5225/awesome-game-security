Este proyecto es un SDK generado al estilo Unreal Engine para el juego completo Lost Ark, e incluye los offsets de GObjects, GNames, ProcessEvent, UEngine y UWorld para EFEngine.dll.
Proporciona LA_Basic, LA_Core y los encabezados relacionados de clases, estructuras y funciones producidos con KN4CK3R sdkgen, para ingeniería inversa educativa e investigación.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / game:lostark.
