Этот проект — сгенерированный полноигровой SDK в стиле Unreal Engine для Lost Ark, включающий смещения GObjects, GNames, ProcessEvent, UEngine и UWorld для EFEngine.dll.
Он предоставляет LA_Basic, LA_Core и связанные заголовки классов, структур и функций, созданные с помощью KN4CK3R sdkgen, для образовательного реверс-инжиниринга и исследований.
В основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области cheat / game:lostark.
