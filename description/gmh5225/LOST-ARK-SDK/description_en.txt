This project is a generated full-game Unreal Engine-style SDK for Lost Ark, including GObjects, GNames, ProcessEvent, UEngine, and UWorld offsets for EFEngine.dll.
It provides LA_Basic, LA_Core, and related class, struct, and function headers produced with KN4CK3R sdkgen for educational reverse-engineering and research use.
It is mainly useful for game security researchers and reverse engineers studying offensive techniques working in the cheat / game:lostark area.
