Этот проект — Baka, библиотека антиотладки для Windows, реализующая несколько техник обнаружения отладчика. Она проверяет наличие отладчика через флаги PEB, вызовы NtQueryInformationProcess, обнаружение аппаратных точек останова, проверки по времени, обнаружение на основе исключений и проверку родительского процесса. Библиотека на C/C++ предоставляет набор примитивов антиотладки, которые можно встроить в приложения. Она предназначена для разработчиков защиты ПО и исследователей безопасности, изучающих техники антиотладки и методы их обхода.
