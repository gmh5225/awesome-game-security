Este proyecto es Baka, una biblioteca anti-debugging de Windows que implementa múltiples técnicas de detección de depuradores. Comprueba la presencia de un depurador mediante flags del PEB, llamadas a NtQueryInformationProcess, detección de puntos de interrupción hardware, comprobaciones de temporización, detección basada en excepciones y validación del proceso padre. La biblioteca C/C++ ofrece una colección de primitivas anti-debug que pueden integrarse en aplicaciones. Está dirigida a desarrolladores de protección de software y a investigadores de seguridad que estudian técnicas anti-debugging y sus métodos de evasión.
