Этот проект — репозиторий из раздела cheat / game:rust, посвящённый внешнему читу (external) для Rust.
Он в основном написан на C/C++ и C++ и сосредоточен на разработке драйверов, моддинге и генерации SDK.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:rust.
