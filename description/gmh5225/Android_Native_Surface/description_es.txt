Este proyecto demuestra cómo crear y renderizar en una superficie nativa de Android desde código C/C++ para construir overlays de juegos en Android. Usa la API ANativeWindow del NDK de Android, o SurfaceFlinger directamente, para crear una superficie de overlay transparente sobre otras aplicaciones y renderizar menús de cheats o información ESP. Está dirigido a modders de juegos Android e investigadores de seguridad que construyen overlays de renderizado nativo.
