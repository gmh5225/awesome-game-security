Этот проект показывает, как создавать и выполнять рендеринг на нативную Android surface из кода на C/C++ для построения игровых оверлеев на Android. Он использует API ANativeWindow из Android NDK или напрямую SurfaceFlinger, чтобы создать прозрачную overlay-surface поверх других приложений и отображать чит-меню или информацию ESP. Предназначен для моддеров Android-игр и исследователей безопасности, создающих нативные оверлеи рендеринга.
