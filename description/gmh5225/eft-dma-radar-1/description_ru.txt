Этот проект — чит-радар на основе DMA для Escape from Tarkov, который читает память игры через DMA-оборудование, совместимое с PCILeech. Он извлекает позиции игроков, расположение лута и данные карты из памяти игрового движка Unity EFT и отображает их на отдельном экране в виде радара в реальном времени. Подход DMA делает чит невидимым для обнаружения BattlEye на уровне процесса. Проект предназначен для исследователей игровой безопасности, изучающих архитектуры читов на основе DMA для Unity-игр, защищённых BattlEye.
