Ce dépôt est une archive contenant du shellcode récupéré vers mars 2023 depuis un jeu protégé par Easy Anti-Cheat.
L’archive ne contient qu’une image mémoire brute nommée shellcode_size0x82C000.mem et un court README l’identifiant comme du shellcode anti-cheat issu d’un dumper de jeu protégé.
Ce README consigne deux points d’entrée d’exécution exposés via le hook inline du jeu protégé aux adresses base+0x79204 et base+0x79304.
Il convient surtout comme matériel d’exemple pour rétro-ingénierer la disposition du shellcode EAC et le flux d’exécution piloté par hooks, plutôt que comme une implémentation réutilisable.
