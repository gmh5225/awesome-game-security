Este repositorio es un volcado archivado de shellcode recuperado hacia marzo de 2023 de un juego protegido por Easy Anti-Cheat.
El archivo solo contiene una imagen de memoria en bruto llamada shellcode_size0x82C000.mem y un breve README que lo identifica como shellcode anticheat de un dumper de un juego protegido.
Ese README registra dos puntos de entrada de ejecución expuestos mediante el hook en línea del juego protegido en base+0x79204 y base+0x79304.
Se entiende mejor como material de muestra para ingeniería inversa del diseño del shellcode de EAC y del flujo de ejecución impulsado por hooks, no como una implementación reutilizable.
