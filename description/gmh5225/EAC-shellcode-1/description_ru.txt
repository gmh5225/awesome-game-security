Этот репозиторий — архивный дамп шеллкода, извлечённого около марта 2023 года из игры, защищённой Easy Anti-Cheat.
Архив содержит только сырой образ памяти shellcode_size0x82C000.mem и короткий README, где указано, что это античит-шеллкод из дампера защищённой игры.
В README зафиксированы две точки входа выполнения, доступные через inline-хук защищённой игры по адресам base+0x79204 и base+0x79304.
Материал лучше рассматривать как образец для реверс-инжиниринга компоновки шеллкода EAC и потока выполнения через хуки, а не как готовое переиспользуемое решение.
