このリポジトリは、2023年3月頃に Easy Anti-Cheat 保護下のゲームから回収されたシェルコードのアーカイブダンプです。
アーカイブには shellcode_size0x82C000.mem という生のメモリイメージと、保護されたゲームのダンパー由来のアンチチートシェルコードである旨を記した短い README のみが含まれます。
その README には、保護ゲームのインラインフック（base+0x79204 および base+0x79304）経由で公開される二つの実行エントリポイントが記録されています。
再利用可能な実装というより、EAC シェルコードの配置とフック駆動の実行フローをリバースエンジニアリングするためのサンプル資料として位置づけられます。
