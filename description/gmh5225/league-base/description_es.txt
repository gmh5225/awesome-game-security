Este proyecto es un framework base para construir cheats de League of Legends. Proporciona infraestructura de lectura de memoria, enumeración de objetos del juego, extracción de datos de campeones y fundamentos de overlays de renderizado para el cliente de LoL. La base en C++ aborda las consideraciones del anti-cheat de Riot (Packman/Vanguard) y las estructuras del motor de juego personalizado de LoL. Está dirigido a investigadores de seguridad en videojuegos que estudian la arquitectura de cheats de LoL y las protecciones anti-cheat de Riot.
