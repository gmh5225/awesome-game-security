Этот проект — базовый фреймворк для создания читов League of Legends. Он предоставляет инфраструктуру чтения памяти, перечисление игровых объектов, извлечение данных чемпионов и основу для оверлеев отрисовки для клиента LoL. База на C++ учитывает античит Riot (Packman/Vanguard) и структуры собственного игрового движка LoL. Предназначен для исследователей игровой безопасности, изучающих архитектуру читов LoL и защиту античита Riot.
