Ce projet est un framework de base pour construire des cheats League of Legends. Il fournit une infrastructure de lecture mémoire, l'énumération des objets de jeu, l'extraction des données de champions et les fondations d'overlay de rendu pour le client LoL. La base C++ gère les considérations liées à l'anti-cheat de Riot (Packman/Vanguard) et les structures du moteur de jeu personnalisé de LoL. Il s'adresse aux chercheurs en sécurité des jeux étudiant l'architecture des cheats LoL et les protections anti-cheat de Riot.
