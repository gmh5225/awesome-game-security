Questo progetto è un framework di base per costruire cheat di League of Legends. Fornisce infrastruttura di lettura della memoria, enumerazione degli oggetti di gioco, estrazione dei dati dei campioni e fondamenti di overlay di rendering per il client LoL. La base in C++ gestisce le considerazioni sull'anti-cheat di Riot (Packman/Vanguard) e le strutture del motore di gioco personalizzato di LoL. È rivolto ai ricercatori di sicurezza nei videogiochi che studiano l'architettura dei cheat di LoL e le protezioni anti-cheat di Riot.
