Un plugin IDA Pro qui importe les symboles du noyau Linux depuis /proc/kallsyms dans une base de données IDA. Il renomme automatiquement les fonctions et les étiquettes de données lors de l'analyse de binaires du noyau à l'aide de la table de symboles kallsyms, pour accélérer la rétro-ingénierie des modules du noyau Linux.
