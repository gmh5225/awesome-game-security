本プロジェクトは、echo.ac の echo_driver.sys における脆弱性を対象とした、writeup 付きの概念実証であり、同梱の例では NT AUTHORITY\SYSTEM への権限昇格に焦点を当てています。
PoC はドライバのメモリ読み取り IOCTL を使い、PsInitialSystemProcess をリークし、ActiveProcessLinks を走査して EPROCESS から SYSTEM トークンを取得し、新たに起動した cmd.exe プロセスのトークンを上書きします。
アーカイブには、echo.ac が競技コミュニティで使われる商用スクリーン共有ツールであるという周辺文脈も残されており、本リポジトリはアンチチート隣接ドライバが攻撃面となり得る事例研究としても機能します。
脆弱なドライバによるトークン窃取、カーネル読み取りプリミティブ、サードパーティのアンチチートソフト上に構築された実世界の権限昇格チェーンを学ぶ Windows カーネル研究者に主に有用です。
