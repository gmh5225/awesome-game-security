이 프로젝트는 echo.ac의 echo_driver.sys 취약점에 대한 writeup 기반 개념 증명으로, 포함된 예제는 NT AUTHORITY\SYSTEM으로의 권한 상승에 초점을 둡니다.
PoC는 드라이버의 메모리 읽기 IOCTL을 사용해 PsInitialSystemProcess를 유출하고, ActiveProcessLinks를 순회한 뒤 EPROCESS에서 SYSTEM 토큰을 회수하여 새로 실행한 cmd.exe 프로세스의 토큰을 덮어씁니다.
아카이브에는 echo.ac가 경쟁 커뮤니티에서 쓰이는 상용 화면 공유 도구라는 배경 맥락도 함께 남아 있어, 이 저장소는 안티치트 인접 드라이버가 어떻게 공격면이 될 수 있는지를 보여주는 사례 연구이기도 합니다.
취약한 드라이버 토큰 탈취, 커널 읽기 프리미티브, 그리고 서드파티 안티치트 소프트웨어 위에 구축된 실제 권한 상승 체인을 연구하는 Windows 커널 연구자에게 주로 유용합니다.
