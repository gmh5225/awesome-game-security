Этот проект — proof of concept по перехвату рендеринга оверлея Steam для отрисовки пользовательского контента в играх. Он перехватывает конвейер рендеринга DLL GameOverlayRenderer Steam, чтобы внедрять пользовательские вызовы отрисовки, используя доверенный статус оверлея Steam для отображения чит-меню или ESP без срабатывания детекта оверлея. Предназначен для исследователей игровой безопасности, изучающих захват оверлея Steam и злоупотребление доверенными оверлеями для обхода античита.
