Этот проект связан с QQ Tang (QQ堂) — казуальной игрой Tencent. Он может включать анализ игрового клиента, эмулятор приватного сервера или инструменты для читерства в QQ Tang с описанием сетевого протокола игры, структур памяти или механизмов античита. Предназначен для исследователей игровой безопасности, изучающих защиту казуальных игр Tencent.
