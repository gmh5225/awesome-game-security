Этот проект содержит руководство по сборке исходного кода GTA V.
Для этого процесса сборки рекомендуется использовать виртуальную машину, хотя сборку можно выполнить и на обычном ПК.
Он в первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр и гайдов.
