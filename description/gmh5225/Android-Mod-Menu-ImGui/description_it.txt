Questo progetto è un template di menu mod per giochi Android che usa Dear ImGui per il rendering in overlay. Fornisce un framework nativo in C++ per iniettare un menu cheat basato su ImGui nei giochi Android, con integrazione OpenGL ES, gestione dell’input touch e hook JNI per modificare le funzioni di gioco. Il template include layout di menu configurabili e controlli a interruttore. È rivolto ai modder di giochi Android che realizzano menu mod visivi con ImGui.
