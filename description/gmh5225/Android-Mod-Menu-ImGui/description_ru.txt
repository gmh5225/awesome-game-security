Этот проект — шаблон меню модов для Android-игр с отрисовкой оверлея на Dear ImGui. Он предоставляет нативный C++-фреймворк для внедрения чит-меню на базе ImGui в Android-игры с интеграцией OpenGL ES, обработкой сенсорного ввода и JNI-хуками для изменения игровых функций. Шаблон включает настраиваемые макеты меню и переключатели. Предназначен для моддеров Android-игр, создающих визуальные меню модов на ImGui.
