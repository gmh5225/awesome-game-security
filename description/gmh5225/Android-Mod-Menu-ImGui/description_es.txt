Este proyecto es una plantilla de menú de mods para juegos Android que usa Dear ImGui para el renderizado en overlay. Proporciona un framework nativo en C++ para inyectar un menú de trucos basado en ImGui en juegos Android, con integración de OpenGL ES, manejo de entrada táctil y hooks JNI para modificar funciones del juego. La plantilla incluye diseños de menú configurables y controles de conmutación. Está orientada a modders de juegos Android que construyen menús de mods visuales con ImGui.
