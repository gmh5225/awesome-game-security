Ce projet est un modèle de menu de mods pour jeux Android utilisant Dear ImGui pour le rendu en overlay. Il fournit un framework natif C++ pour injecter un menu de cheat basé sur ImGui dans des jeux Android, avec intégration OpenGL ES, gestion des entrées tactiles et hooks JNI pour modifier les fonctions du jeu. Le modèle inclut des dispositions de menu configurables et des commandes à bascule. Il s’adresse aux moddeurs de jeux Android qui construisent des menus de mods visuels avec ImGui.
