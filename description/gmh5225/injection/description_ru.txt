Этот проект посвящён тестированию инъекций.
Охватывает Conhost ExtraBytes, PROPagate, Service Control Handler, Print Spooler / ALPC, KernelCallbackTable, WordWarping, Hyphentension, AutoCourgette, Streamception, Oleum, ListPlanting, Treepoline, Windows Notification Facility, Windows Sockets Helper Functions, Multiple Provider Router (MPR) DLL and Shell Notifications, DNS Client API, Breaking BaDDEr, Tooltip or Common Controls, KnownDlls Cache Poisoning.
В первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / стресс-тестирования.
