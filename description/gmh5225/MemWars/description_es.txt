Este proyecto es MemWars, un framework de Windows para análisis de memoria y manipulación de procesos orientado al game hacking. Proporciona en formato de biblioteca escaneo de memoria, búsqueda de patrones, enumeración de módulos, inyección de procesos y edición de memoria, para que los desarrolladores de cheats puedan construir sobre él. El framework en C++ abstrae operaciones habituales de game hacking. Está dirigido a desarrolladores de herramientas de game hacking y a investigadores de seguridad que estudian frameworks de manipulación de memoria.
