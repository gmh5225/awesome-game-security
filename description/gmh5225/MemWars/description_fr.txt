Ce projet est MemWars, un framework Windows d'analyse mémoire et de manipulation de processus destiné au game hacking. Il fournit en bibliothèque le scan mémoire, la recherche de motifs, l'énumération de modules, l'injection de processus et l'édition mémoire, afin que les développeurs de cheats puissent s'en servir de base. Le framework C++ abstrait les opérations courantes de game hacking. Il s'adresse aux développeurs d'outils de game hacking et aux chercheurs en sécurité qui étudient les frameworks de manipulation mémoire.
