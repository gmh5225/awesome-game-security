Questo progetto è MemWars, un framework Windows per l'analisi della memoria e la manipolazione dei processi orientato al game hacking. Offre in forma di libreria scansione della memoria, ricerca di pattern, enumerazione dei moduli, iniezione di processi e modifica della memoria, su cui gli sviluppatori di cheat possono basarsi. Il framework C++ astrae le operazioni comuni di game hacking. Si rivolge agli sviluppatori di strumenti di game hacking e ai ricercatori di sicurezza che studiano i framework di manipolazione della memoria.
