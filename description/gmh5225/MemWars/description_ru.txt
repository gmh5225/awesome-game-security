Этот проект — MemWars, Windows-фреймворк для анализа памяти и манипуляции процессами в контексте game hacking. Он предоставляет в виде библиотеки сканирование памяти, поиск паттернов, перечисление модулей, внедрение в процессы и редактирование памяти, на базе которых разработчики читов могут строить свои решения. C++-фреймворк абстрагирует типичные операции game hacking. Он рассчитан на разработчиков инструментов game hacking и исследователей безопасности, изучающих фреймворки манипуляции памятью.
