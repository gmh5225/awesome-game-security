Kernel Cactus は、Dell の脆弱な dbutil_2_3.sys ドライバをカーネル読み書きのバックエンドとして利用するユーザーモード攻撃ツールキットです。
KernelOps.cpp は \\.\DBUtil_2_3 を開き、IOCTL 0x9B0C1EC4 と 0x9B0C1EC8 上にプリミティブを構築したうえで、ETW の無効化、PPL の切り替え、保護プロセスの終了、トークンの複製、ファイル削除といった上位操作を重ねます。
コマンドセットには、通常のユーザーモードツールでは遮断されがちなプロセスを狙った、shellcode ベースのリモートスレッド注入やスレッドハイジャック経路も含まれます。
したがって、このリポジトリは dbutil_2_3.sys に関する単なる前提メモというより、多機能な BYOVD ポストエクスプロイトコンソールと表現する方が適切です。
