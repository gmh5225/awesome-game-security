Этот проект — реализация сервера Cheat Engine (ceserver) для iOS, позволяющая удалённо сканировать память с настольного клиента Cheat Engine. Он работает на устройствах iOS с jailbreak и предоставляет память процессов через сетевой протокол ceserver, что даёт возможность с ПК выполнять поиск в памяти, изменять значения и сканировать указатели в процессах iOS-игр. Предназначен для исследователей безопасности iOS-игр, использующих Cheat Engine для анализа мобильных игр.
