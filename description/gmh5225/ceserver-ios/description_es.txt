Este proyecto es una implementación de servidor de Cheat Engine (ceserver) para iOS que permite el escaneo remoto de memoria desde un cliente de Cheat Engine de escritorio. Se ejecuta en dispositivos iOS con jailbreak y expone la memoria de procesos mediante el protocolo de red ceserver, lo que permite buscar en memoria, editar valores y escanear punteros en procesos de juegos iOS desde un PC. Está dirigido a investigadores de seguridad de juegos iOS que usan Cheat Engine para el análisis de juegos móviles.
