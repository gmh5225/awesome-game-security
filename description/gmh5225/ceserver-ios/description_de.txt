Dieses Projekt ist eine Cheat-Engine-Server-Implementierung (ceserver) für iOS, die Remote-Speicher-Scans von einem Desktop-Cheat-Engine-Client aus ermöglicht. Es läuft auf gejailbreakten iOS-Geräten und stellt Prozessspeicher über das ceserver-Netzwerkprotokoll bereit, sodass Speichersuche, Wertbearbeitung und Pointer-Scans an iOS-Spielprozessen von einem PC aus möglich sind. Es richtet sich an iOS-Game-Security-Forscher, die Cheat Engine zur Analyse mobiler Spiele nutzen.
