Этот проект документирует или эксплуатирует компонент гипервизора антивируса Avast для операций на уровне ядра. Он демонстрирует использование подписанного драйвера гипервизора Avast для повышения привилегий или выполнения произвольного кода в контексте гипервизора, опираясь на доверенный статус драйвера Avast для обхода DSE и античит-защит. Предназначен для исследователей безопасности ядра, изучающих возможности BYOVD в гипервизорных компонентах антивирусов.
