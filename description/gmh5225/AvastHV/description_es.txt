Este proyecto documenta o explota el componente hipervisor del antivirus Avast para operaciones a nivel de kernel. Demuestra cómo usar el controlador de hipervisor firmado de Avast para obtener privilegios elevados o ejecutar código arbitrario en el contexto del hipervisor, aprovechando el estado de confianza de ese controlador para evadir DSE y las protecciones anti-trampas. Está dirigido a investigadores de seguridad de kernel que estudian oportunidades BYOVD en componentes hipervisor de antivirus.
