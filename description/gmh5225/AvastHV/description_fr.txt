Ce projet documente ou exploite le composant hyperviseur de l'antivirus Avast pour des opérations au niveau du noyau. Il montre comment utiliser le pilote hyperviseur signé d'Avast pour obtenir des privilèges élevés ou exécuter du code arbitraire dans le contexte de l'hyperviseur, en s'appuyant sur le statut de confiance de ce pilote pour contourner la DSE et les protections anti-triche. Il s'adresse aux chercheurs en sécurité noyau qui étudient les opportunités BYOVD dans les composants hyperviseurs d'antivirus.
