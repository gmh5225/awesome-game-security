Этот проект — внешний чит для Rainbow Six Siege (R6S) версии 2, работающий вне процесса игры. Он читает память игры внешними методами (через драйвер ядра или повышение привилегий handle) и предоставляет ESP с позициями игроков, здоровьем и информацией об оперативниках, а также функцию аимбота. C++-чит избегает инъекции в процесс игры, защищённый BattlEye. Предназначен для исследователей игровой безопасности, изучающих паттерны внешних читов для UE4-игр с защитой BattlEye.
