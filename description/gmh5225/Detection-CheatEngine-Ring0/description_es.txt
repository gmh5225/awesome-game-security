Este proyecto es una pequeña prueba de concepto en el kernel que intenta detectar actividad relacionada con Cheat Engine o DBVM a través de la ruta de callback de impresión de depuración del kernel.
La única lógica real registra un controlador DbgSetDebugPrintCallback, copia las cadenas de depuración entrantes, busca el literal dbvm-mode y deja una ruta de informe TODO.
Como el README lo presenta solo como un meme y el código es mínimo, debe tratarse como un experimento en torno a una señal estrecha del lado del kernel, no como un marco completo de detección.
Es útil sobre todo para investigadores de anti-cheat que exploran si la salida de depuración del kernel puede exponer el estado de Cheat Engine o DBVM en entornos de prueba.
