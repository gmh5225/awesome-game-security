Этот проект — небольшой kernel proof of concept, который пытается заметить активность, связанную с Cheat Engine или DBVM, через путь callback печати отладочных сообщений ядра.
Единственная реальная логика регистрирует обработчик DbgSetDebugPrintCallback, копирует входящие отладочные строки, проверяет литерал dbvm-mode и оставляет путь отчёта TODO.
Поскольку README называет это просто мемом, а код минимален, проект стоит рассматривать как эксперимент вокруг одного узкого сигнала на стороне ядра, а не как полноценный фреймворк обнаружения.
Он в основном полезен исследователям античита, которые изучают, может ли отладочный вывод ядра раскрывать состояние Cheat Engine или DBVM в тестовых средах.
