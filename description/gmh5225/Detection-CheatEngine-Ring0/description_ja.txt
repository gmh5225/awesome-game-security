本プロジェクトは、カーネルのデバッグ出力コールバック経路を通じて Cheat Engine や DBVM 関連の活動を検知しようとする小さなカーネル PoC です。
実質的な処理は DbgSetDebugPrintCallback ハンドラの登録、受け取ったデバッグ文字列のコピー、リテラル dbvm-mode の検査、そして TODO 報告パスへの到達だけです。
README では meme と位置づけられ、コードも最小限であるため、完成した検出フレームワークではなく、狭いカーネル側シグナルを巡る実験として扱うべきです。
主に、テスト環境でカーネルデバッグ出力が Cheat Engine や DBVM の状態を露呈し得るかを探るアンチチート研究者向けです。
