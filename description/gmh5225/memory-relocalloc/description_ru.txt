Этот проект демонстрирует технику перемещения/выделения памяти в Windows или Android для сокрытия выделений памяти от средств сканирования. Он выделяет память нестандартными способами или перемещает существующие выделения, чтобы избежать обнаружения античит-сканерами памяти, которые перечисляют стандартные области кучи и виртуальной памяти. Предназначен для исследователей игровой безопасности, изучающих методы сокрытия памяти и обход античит-сканирования памяти.
