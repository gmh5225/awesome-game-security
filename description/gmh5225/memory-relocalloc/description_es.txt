Este proyecto demuestra una técnica de reubicación/asignación de memoria en Windows o Android para ocultar asignaciones de memoria a las herramientas de escaneo. Asigna memoria de formas no convencionales o reubica asignaciones existentes para evitar la detección por escáneres de memoria anti-cheat que enumeran regiones estándar del montón y de la memoria virtual. Está dirigido a investigadores de seguridad en juegos que estudian técnicas de ocultación de memoria y la evasión del escaneo de memoria anti-cheat.
