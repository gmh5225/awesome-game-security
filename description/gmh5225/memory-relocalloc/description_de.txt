Dieses Projekt demonstriert eine Technik zur Speicherumplatzierung/-allokation unter Windows oder Android, um Speicherallokationen vor Scan-Tools zu verbergen. Es allokiert Speicher auf unkonventionelle Weise oder verlagert bestehende Allokationen, um der Erkennung durch Anti-Cheat-Speicherscanner zu entgehen, die standardmäßige Heap- und Virtuelle-Speicher-Bereiche auflisten. Es richtet sich an Gamesicherheitsforscher, die Techniken zur Speicherverbergung und zur Umgehung von Anti-Cheat-Speicherscans untersuchen.
