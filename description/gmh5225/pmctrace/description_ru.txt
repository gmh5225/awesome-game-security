Этот проект — pmctrace, инструмент для сбора данных аппаратных счётчиков производительности с помощью Intel PMC (Performance Monitoring Counters). Он настраивает и считывает счётчики производительности CPU для измерения попаданий/промахов кэша, предсказаний ветвлений, числа инструкций и других микроархитектурных событий при профилировании программ. Инструмент на C обеспечивает низкоуровневый доступ к регистрам PMC. Предназначен для инженеров по производительности и исследователей безопасности, использующих аппаратные счётчики производительности для анализа программ и исследований побочных каналов.
