Este proyecto es pmctrace, una herramienta para recopilar datos de contadores de rendimiento de hardware mediante Intel PMC (Performance Monitoring Counters). Configura y lee los contadores de rendimiento de la CPU para medir aciertos/fallos de caché, predicciones de saltos, recuentos de instrucciones y otros eventos microarquitectónicos con fines de perfilado de programas. La herramienta en C proporciona acceso de bajo nivel a los registros PMC. Está dirigida a ingenieros de rendimiento e investigadores de seguridad que usan contadores de rendimiento de hardware para el análisis de programas y la investigación de canales laterales.
