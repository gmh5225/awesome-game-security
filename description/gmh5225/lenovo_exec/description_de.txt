Dieses Projekt nutzt eine Schwachstelle im Lenovo-Kernel-Treiber für beliebige Codeausführung im Kernel. Es missbraucht die unsichere IOCTL-Schnittstelle des Lenovo-Treibers, um eigenen Code im Kernelmodus auszuführen, und demonstriert eine BYOVD-Exploit-Kette vom Usermode zur Kernel-Codeausführung. Es richtet sich an BYOVD-Forscher, die Lenovo-Treiber-Schwachstellen untersuchen.
