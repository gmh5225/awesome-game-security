Questo progetto sfrutta una vulnerabilità di un driver kernel Lenovo per ottenere esecuzione arbitraria di codice nel kernel. Abusa dell’interfaccia IOCTL non sicura del driver Lenovo per eseguire codice personalizzato in modalità kernel, dimostrando una catena di exploit BYOVD dal modo utente all’esecuzione di codice kernel. È rivolto ai ricercatori BYOVD che studiano le vulnerabilità dei driver Lenovo.
