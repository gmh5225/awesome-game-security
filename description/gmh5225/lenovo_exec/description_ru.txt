Этот проект использует уязвимость драйвера ядра Lenovo для произвольного выполнения кода в ядре. Он злоупотребляет небезопасным IOCTL-интерфейсом драйвера Lenovo для выполнения пользовательского кода в режиме ядра, демонстрируя цепочку эксплуатации BYOVD от пользовательского режима до выполнения кода в ядре. Предназначен для исследователей BYOVD, изучающих уязвимости драйверов Lenovo.
