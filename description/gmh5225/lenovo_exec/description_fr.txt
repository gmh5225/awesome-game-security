Ce projet exploite une vulnérabilité d’un pilote noyau Lenovo pour obtenir une exécution arbitraire de code en mode noyau. Il abuse de l’interface IOCTL non sécurisée du pilote Lenovo pour exécuter du code personnalisé en mode noyau, illustrant une chaîne d’exploitation BYOVD du mode utilisateur à l’exécution de code noyau. Il s’adresse aux chercheurs BYOVD qui étudient les vulnérabilités des pilotes Lenovo.
