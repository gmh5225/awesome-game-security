Este proyecto aprovecha una vulnerabilidad de un controlador de kernel de Lenovo para lograr ejecución arbitraria de código en el kernel. Abusa de la interfaz IOCTL insegura del controlador Lenovo para ejecutar código personalizado en modo kernel, demostrando una cadena de explotación BYOVD desde el modo de usuario hasta la ejecución de código en el kernel. Está dirigido a investigadores BYOVD que estudian vulnerabilidades de controladores Lenovo.
