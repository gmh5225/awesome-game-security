Dieses Projekt ist ein einfach zu nutzendes C++-3D-Game-Framework.
Das CX3D Framework ist ein benutzerfreundliches Game Framework für die Entwicklung von 3D-Spielen in C++.
Es eignet sich vor allem für Spieleentwickler, Engine-Programmierer und Grafikforscher, die im Bereich Game Engine / Source arbeiten.
