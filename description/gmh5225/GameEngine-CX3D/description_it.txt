Questo progetto è un framework per giochi 3D in C++ facile da usare.
Il CX3D Framework è un framework di gioco semplice da utilizzare per lo sviluppo di giochi 3D in C++.
È particolarmente utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell'ambito dei game engine / del codice sorgente.
