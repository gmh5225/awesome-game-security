Ce projet est un framework de jeux 3D en C++ facile à utiliser.
Le CX3D Framework est un framework de jeux simple d'emploi pour le développement de jeux 3D en C++.
Il est surtout utile aux développeurs de jeux, aux programmeurs de moteurs et aux chercheurs en infographie travaillant dans le domaine des moteurs de jeux / du code source.
