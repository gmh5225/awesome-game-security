Questo progetto è MemoryGuard, una libreria di protezione della memoria per Windows che rileva e impedisce modifiche non autorizzate alla memoria. Usa page guard, gestori VEH o controlli periodici di integrità per monitorare manomissioni delle regioni di memoria critiche, segnalando o ripristinando le modifiche apportate da cheat o debugger. È rivolta agli sviluppatori di anti-cheat e agli ingegneri di protezione software che implementano il monitoraggio dell'integrità della memoria a runtime.
