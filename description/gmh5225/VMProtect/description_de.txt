Dieses Projekt ist eine Code-Obfuskationsmethode, die virtuelle Maschinen zum Schutz von Programmen einsetzt.
Eine virtuelle Maschine, die eine CPU sowie einige weitere Hardwarekomponenten simuliert, ermöglicht arithmetische Operationen, Lese- und Schreibzugriffe auf den Speicher sowie die Interaktion mit I/O-Geräten.
Sie ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / VMP-Fix untersuchen.
