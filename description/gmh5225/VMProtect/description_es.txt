Este proyecto es un método de ofuscación de código que usa máquinas virtuales para proteger programas.
Una máquina virtual que simula una CPU y algunos otros componentes de hardware permite realizar operaciones aritméticas, leer y escribir en memoria e interactuar con dispositivos de E/S.
Es especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito del cheat / fix VMP.
