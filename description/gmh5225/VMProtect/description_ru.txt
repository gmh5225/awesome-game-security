Этот проект — метод обфускации кода с помощью виртуальных машин для защиты программ.
Виртуальная машина, имитирующая CPU и ряд других аппаратных компонентов, позволяет выполнять арифметические операции, читать и писать в память и взаимодействовать с устройствами ввода-вывода.
Она в первую очередь полезна исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / fix VMP.
