Этот проект — утилита дампа VAC3 на основе DLL, которая внедряется в steam.exe, чтобы перехватить процедуру, используемую Steam для загрузки модулей VAC.
В архивном README описан рабочий процесс: запуск Steam от имени администратора, внедрение до запуска игры и запись восстановленных модулей в C:\Modules для офлайн-анализа.
Код находит процедуру steamservice.dll, отвечающую за загрузку модулей VAC, перехватывает её с помощью MinHook и захватывает содержимое модуля в момент загрузки, вместо того чтобы пытаться восстановить его позже.
Он в первую очередь полезен исследователям реверса, изучающим загрузку модулей Valve Anti-Cheat, точки перехвата на стороне Steam и простые сценарии live-дампа для анализа VAC.
