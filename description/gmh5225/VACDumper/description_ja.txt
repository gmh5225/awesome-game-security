本プロジェクトは DLL ベースの VAC3 ダンプユーティリティで、steam.exe に注入して Steam が VAC モジュールを読み込むルーチンをインターセプトします。
アーカイブされた README では、Steam を管理者として起動し、ゲーム起動前に注入し、復元したモジュールをオフライン解析用に C:\Modules へ書き出す一連の作業手順が説明されています。
コードは VAC モジュール読み込みを担当する steamservice.dll 内のルーチンを特定し、MinHook でフックし、後から再構築するのではなく読み込み時にモジュール内容を取得します。
主に Valve Anti-Cheat のモジュール読み込み、Steam 側のインターセプトポイント、および VAC 解析向けのシンプルなライブダンプ手順を研究するリバースエンジニア向けです。
