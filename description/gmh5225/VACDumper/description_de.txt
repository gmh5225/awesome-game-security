Dieses Projekt ist ein DLL-basiertes VAC3-Dump-Hilfsprogramm, das in steam.exe injiziert wird, um die Routine abzufangen, mit der Steam VAC-Module lädt.
Die archivierte README erklärt den Ablauf: Steam als Administrator starten, vor dem Spielstart injizieren und die wiederhergestellten Module zur Offline-Analyse nach C:\Modules schreiben.
Der Code findet die für das Laden von VAC-Modulen zuständige Routine in steamservice.dll, hookt sie mit MinHook und erfasst den Modulinhalt beim Laden, statt ihn nachträglich rekonstruieren zu müssen.
Er ist vor allem für Reverse Engineers nützlich, die das Laden von Valve-Anti-Cheat-Modulen, Abfangpunkte auf Steam-Seite und einfache Live-Dump-Workflows für die VAC-Analyse untersuchen.
