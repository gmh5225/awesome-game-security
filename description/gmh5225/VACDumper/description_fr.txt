Ce projet est un utilitaire de dump VAC3 basé sur une DLL, injecté dans steam.exe afin d'intercepter la routine utilisée par Steam pour charger les modules VAC.
Le README archivé décrit le flux de travail : démarrer Steam en tant qu'administrateur, injecter avant le lancement du jeu, puis écrire les modules récupérés dans C:\Modules pour une analyse hors ligne.
Le code localise la routine de steamservice.dll responsable du chargement des modules VAC, l'accroche avec MinHook et capture le contenu du module au moment du chargement, plutôt que d'essayer de le reconstruire ensuite.
Il est surtout utile aux reverse engineers qui étudient le chargement des modules Valve Anti-Cheat, les points d'interception côté Steam et des workflows simples de dump en direct pour l'analyse VAC.
