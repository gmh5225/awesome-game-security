Этот проект — proof of concept перехвата Windows Desktop Window Manager (DWM), который перехватывает конвейер отрисовки DWM для рисования пользовательских оверлеев. Он хукает функции композиции DWM, внедряя вызовы отрисовки в композитор рабочего стола и отображая оверлеи поверх любого окна без создания отдельного окна оверлея. Предназначен для исследователей безопасности игр, изучающих техники отрисовки оверлеев на базе DWM.
