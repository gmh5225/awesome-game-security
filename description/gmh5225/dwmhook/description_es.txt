Este proyecto es una prueba de concepto de hooking del Windows Desktop Window Manager (DWM) que intercepta el pipeline de renderizado de DWM para dibujar overlays personalizados. Engancha las funciones de composición de DWM para inyectar llamadas de dibujo en el compositor del escritorio, renderizando overlays encima de cualquier ventana sin crear una ventana de overlay separada. Está dirigido a investigadores de seguridad en juegos que estudian técnicas de renderizado de overlays basadas en DWM.
