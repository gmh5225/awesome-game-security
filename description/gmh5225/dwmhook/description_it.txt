Questo progetto è una proof of concept di hooking del Windows Desktop Window Manager (DWM) che intercetta la pipeline di rendering di DWM per disegnare overlay personalizzati. Aggancia le funzioni di composizione di DWM per iniettare chiamate di disegno nel compositor del desktop, rendendo overlay sopra qualsiasi finestra senza creare una finestra overlay separata. È rivolto ai ricercatori di sicurezza nei videogiochi che studiano le tecniche di rendering di overlay basate su DWM.
