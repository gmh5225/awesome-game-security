Ce projet est une preuve de concept de hooking du Windows Desktop Window Manager (DWM) qui intercepte le pipeline de rendu de DWM pour dessiner des overlays personnalisés. Il accroche les fonctions de composition DWM pour injecter des appels de dessin dans le compositeur du bureau, affichant des overlays au-dessus de n'importe quelle fenêtre sans créer de fenêtre d'overlay séparée. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les techniques de rendu d'overlays basées sur DWM.
