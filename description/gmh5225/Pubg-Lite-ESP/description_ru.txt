Внешний ESP-чит для PUBG Lite, который использует оверлей Direct2D (через библиотеку D2DOverlay от Coltonon) для отрисовки рамок игроков, имён, полос здоровья и индикаторов дистанции на прозрачном окне поверх игры.
Инструмент внешне читает память игры через RPM, разрешает структуры движка UE4 (GWorld, GameInstance, PlayerController, AcknowledgedPawn) по захардкоженным смещениям, выполняет проекцию из мировых координат на экран через матрицу вида камеры и включает настраиваемые параметры дальности отрисовки, bone ESP и меню, управляемое горячими клавишами.
В первую очередь полезен исследователям игровой безопасности, изучающим внешние ESP-реализации на оверлеях с рендерингом Direct2D и перечисление сущностей по смещениям UE4.
