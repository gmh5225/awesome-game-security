Ce projet est un plugin IDA Pro qui intègre le scan de règles YARA dans le flux de désassemblage. Il permet d'exécuter des règles YARA sur le binaire chargé directement depuis IDA, de surligner les correspondances dans la vue désassemblée et de naviguer vers les emplacements trouvés. Le plugin prend en charge des jeux de règles personnalisés pour identifier des signatures de malwares, des constantes cryptographiques ou des motifs de packers. Il s'adresse aux analystes malware et aux reverse engineers qui utilisent la détection basée sur YARA dans IDA Pro.
