Этот проект — плагин для IDA Pro, интегрирующий сканирование по правилам YARA в рабочий процесс дизассемблирования. Он позволяет запускать правила YARA против загруженного двоичного файла прямо из IDA, подсвечивать совпадения в представлении дизассемблирования и переходить к найденным адресам. Плагин поддерживает пользовательские наборы правил для выявления сигнатур вредоносного ПО, криптографических констант или шаблонов пакеров. Он ориентирован на аналитиков вредоносного ПО и специалистов по реверс-инжинирингу, использующих обнаружение на основе YARA в IDA Pro.
