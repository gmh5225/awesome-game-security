Questo progetto è un plugin di IDA Pro che integra la scansione con regole YARA nel flusso di disassemblaggio. Consente di eseguire regole YARA sul binario caricato direttamente da IDA, evidenziare le corrispondenze nella vista di disassemblaggio e navigare alle posizioni trovate. Il plugin supporta set di regole personalizzati per identificare firme malware, costanti crittografiche o pattern di packer. È rivolto ad analisti malware e reverse engineer che usano il rilevamento basato su YARA in IDA Pro.
