Ce projet contient le code source exposé ou divulgué d’un cheat pour le jeu 8 Ball Pool. Il fournit une assistance à la visée, une prédiction de trajectoire et un calcul de la puissance du coup en lisant les données physiques du jeu et en affichant des overlays de lignes de guidage. Le code illustre des schémas d’implémentation de cheats pour les jeux de billard. Il s’adresse aux chercheurs en sécurité des jeux mobiles qui étudient les cheats fondés sur la physique.
