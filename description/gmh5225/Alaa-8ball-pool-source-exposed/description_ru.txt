Этот проект содержит раскрытый или утёкший исходный код чита для игры 8 Ball Pool. Он обеспечивает помощь при прицеливании, предсказание траектории и расчёт силы удара, считывая физические данные игры и отрисовывая оверлеи направляющих линий. Исходники демонстрируют типичные приёмы реализации читов для бильярдных игр. Материал предназначен для исследователей безопасности мобильных игр, изучающих читы на основе физики.
