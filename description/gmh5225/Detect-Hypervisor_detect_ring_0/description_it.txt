Questo progetto è un driver di test per il rilevamento di hypervisor in ring-0, scritto per il deployment nel kernel tramite manual-map.
I suoi controlli combinano il bit hypervisor di CPUID, confronti tra leaf CPUID non valide e leaf hypervisor, attacchi di timing sul comportamento di VM-exit usando gli MSR TSC, APERF e MPERF, e controlli di coerenza di Intel LBR o DEBUGCTL.
Il README accredita la ricerca di Secret Club sulla rilevazione di hypervisor; il driver stampa semplicemente ciascun risultato euristico da DriverEntry, senza costruire una pipeline di enforcement più ampia.
È soprattutto utile a ricercatori di anti-cheat e sicurezza a basso livello che confrontano più euristiche in modalità kernel per individuare emulazione o hypervisor personalizzati.
