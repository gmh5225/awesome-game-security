Dieses Projekt ist ein Ring-0-Hypervisor-Erkennungstesttreiber für die manuell gemappte (manual-map) Kernelbereitstellung.
Seine Prüfungen kombinieren das CPUID-Hypervisor-Bit, Vergleiche zwischen ungültigen und Hypervisor-CPUID-Leaves, Timing-Angriffe rund um das VM-Exit-Verhalten über die MSR TSC, APERF und MPERF sowie Konsistenzprüfungen von Intel LBR oder DEBUGCTL.
Das README würdigt die Hypervisor-Erkennungsforschung von Secret Club; der Treiber gibt lediglich jedes Heuristik-Ergebnis aus DriverEntry aus, statt eine größere Durchsetzungspipeline aufzubauen.
Er ist vor allem für Anti-Cheat- und Low-Level-Sicherheitsforscher nützlich, die mehrere Kernelmode-Heuristiken zum Erkennen von Emulation oder benutzerdefinierten Hypervisoren vergleichen.
