이 프로젝트는 수동 매핑(manual-map) 커널 배포용으로 작성된 ring-0 하이퍼바이저 탐지 테스트 드라이버입니다.
검사는 CPUID 하이퍼바이저 비트, 유효하지 않은 CPUID leaf와 하이퍼바이저 leaf 비교, TSC·APERF·MPERF MSR을 이용한 VM-exit 주변 타이밍 공격, 그리고 Intel LBR 또는 DEBUGCTL 일관성 검사를 결합합니다.
README는 Secret Club의 하이퍼바이저 탐지 연구를 크레딧하며, 드라이버는 더 큰 집행 파이프라인을 구축하지 않고 DriverEntry에서 각 휴리스틱 결과를 단순히 출력합니다.
에뮬레이션이나 커스텀 하이퍼바이저를 포착하기 위한 여러 커널 모드 휴리스틱을 비교하는 안티치트 및 저수준 보안 연구자에게 주로 유용합니다.
