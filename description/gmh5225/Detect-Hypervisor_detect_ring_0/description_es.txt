Este proyecto es un controlador de prueba de detección de hipervisor en ring-0, escrito para despliegue en el kernel mediante manual-map.
Sus comprobaciones combinan el bit de hipervisor de CPUID, comparaciones entre hojas CPUID inválidas y de hipervisor, ataques de temporización en torno al comportamiento de VM-exit usando los MSR TSC, APERF y MPERF, y comprobaciones de coherencia de Intel LBR o DEBUGCTL.
El README acredita la investigación de Secret Club sobre detección de hipervisores; el controlador simplemente imprime cada resultado heurístico desde DriverEntry, en lugar de construir un pipeline de aplicación más amplio.
Resulta especialmente útil para investigadores de anti-cheat y seguridad de bajo nivel que comparan múltiples heurísticas en modo kernel para detectar emulación o hipervisores personalizados.
