本專案是一個用於手動對應（manual-map）核心部署的 ring-0 Hypervisor 偵測測試驅動程式。
其檢查結合了 CPUID Hypervisor 位元、無效與 Hypervisor CPUID leaf 之間的比較、圍繞 VM-exit 行為並使用 TSC、APERF 與 MPERF MSR 的計時攻擊，以及 Intel LBR 或 DEBUGCTL 一致性檢查。
README 致謝 Secret Club 的 Hypervisor 偵測研究；該驅動程式僅在 DriverEntry 中列印各項啟發式結果，而非建置更大的執行／處置管線。
它主要對反作弊與底層安全研究人員有用，便於比較多種核心模式啟發式方法，以找出模擬或自訂 Hypervisor。
