本プロジェクトは、manual-map によるカーネル展開向けに書かれた ring-0 ハイパーバイザ検出テストドライバです。
検出処理は、CPUID のハイパーバイザビット、無効な CPUID leaf とハイパーバイザ leaf の比較、TSC・APERF・MPERF の MSR を用いた VM-exit 周りのタイミング攻撃、および Intel LBR または DEBUGCTL の整合性チェックを組み合わせています。
README では Secret Club のハイパーバイザ検出研究へのクレジットがあり、ドライバは大規模な執行パイプラインを構築せず、DriverEntry から各ヒューリスティックの結果を単に出力します。
エミュレーションやカスタムハイパーバイザを見抜くための複数のカーネルモードヒューリスティックを比較する、アンチチートおよび低レベルセキュリティ研究者向けに主に役立ちます。
