Этот проект — тестовый драйвер обнаружения гипервизора на ring-0, написанный для развёртывания в ядре методом manual-map.
Его проверки сочетают бит гипервизора CPUID, сравнения недействительных и гипервизорных листьев CPUID, тайминг-атаки вокруг поведения VM-exit с использованием MSR TSC, APERF и MPERF, а также проверки согласованности Intel LBR или DEBUGCTL.
В README указаны исследования Secret Club по обнаружению гипервизоров; драйвер просто выводит результат каждой эвристики из DriverEntry, не строя более крупный конвейер принуждения.
Он в основном полезен исследователям античита и низкоуровневой безопасности, сравнивающим несколько эвристик в режиме ядра для выявления эмуляции или пользовательских гипервизоров.
