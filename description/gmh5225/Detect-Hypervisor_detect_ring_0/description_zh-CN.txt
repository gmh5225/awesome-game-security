本项目是一个用于手动映射（manual-map）内核部署的 ring-0 虚拟机监控程序检测测试驱动。
其检测手段结合了 CPUID 虚拟机监控程序位、无效与虚拟机监控程序 CPUID leaf 之间的比较、围绕 VM-exit 行为并利用 TSC、APERF 与 MPERF MSR 的计时攻击，以及 Intel LBR 或 DEBUGCTL 一致性检查。
README 致谢了 Secret Club 的虚拟机监控程序检测研究；该驱动仅在 DriverEntry 中打印各项启发式结果，而非构建更大的执行/处置流水线。
它主要面向反作弊与底层安全研究人员，用于比较多种内核态启发式方法，以发现仿真或自定义虚拟机监控程序。
