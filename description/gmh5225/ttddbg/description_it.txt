Questo progetto è ttddbg, un plugin di IDA Pro per il debug delle tracce Time Travel Debugging (TTD) registrate da WinDbg. Carica i file di traccia Microsoft TTD (.run) in IDA Pro, consentendo il stepping in avanti e all'indietro dell'esecuzione registrata senza una sessione di debug live. Il plugin ripete syscall, operazioni di memoria e lo stato dei registri dalla traccia. È rivolto a reverse engineer e analisti di malware che desiderano combinare le capacità di analisi di IDA Pro con il debug time-travel di TTD.
