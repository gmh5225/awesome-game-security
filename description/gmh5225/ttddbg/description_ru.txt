Этот проект — ttddbg, плагин IDA Pro для отладки трасс Time Travel Debugging (TTD), записанных WinDbg. Он загружает файлы трасс Microsoft TTD (.run) в IDA Pro, позволяя выполнять шаги вперёд и назад по записанному выполнению без живой сессии отладки. Плагин воспроизводит системные вызовы, операции с памятью и состояние регистров из трассы. Он предназначен для реверс-инженеров и аналитиков вредоносного ПО, которым нужны возможности анализа IDA Pro в сочетании с отладкой с путешествием во времени TTD.
