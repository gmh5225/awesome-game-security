Ce projet est ttddbg, un plugin IDA Pro pour déboguer les traces Time Travel Debugging (TTD) enregistrées par WinDbg. Il charge les fichiers de trace Microsoft TTD (.run) dans IDA Pro, permettant un pas à pas avant et arrière dans l'exécution enregistrée sans session de débogage en direct. Le plugin rejoue les appels système, les opérations mémoire et l'état des registres à partir de la trace. Il s'adresse aux reverse engineers et aux analystes de malwares qui souhaitent combiner les capacités d'analyse d'IDA Pro avec le débogage temporel de TTD.
