Escape From Tarkov 向けの Mono ベース内部チートで、C# で記述され、Unity/Mono ランタイムにフックして ESP（ボックス、名前、距離、ヘルスバー）、サイレントエイム、無反動、速度ハック、無限スタミナ、ルート ESP をメソッドフックとゲームオブジェクトの直接操作により提供します。
このチートは EFT の GameWorld.RegisteredPlayers リストを走査してプレイヤーを列挙し、PlayerBones を読み取って Camera.WorldToScreenPoint によるスケルトンベースのワールド座標からスクリーンへの投影を行い、CreateShot/ApplyShot/BulletMovement などのメソッドをフックして弾を操作し、Unity の OnGUI/GUI.DrawTexture と独自の ImGui 風メニューシステムでオーバーレイを描画します。
主に Escape From Tarkov における Mono/.NET インジェクション手法、Unity ゲームオブジェクトのイントロスペクション、メソッドフックの攻撃面を研究するゲームセキュリティ研究者向けです。
