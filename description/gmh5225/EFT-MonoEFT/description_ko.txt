Escape From Tarkov용 Mono 기반 내부 치트로, C#로 작성되었으며 Unity/Mono 런타임에 후킹하여 ESP(박스, 이름, 거리, 체력바), 사일런트 에임, 무반동, 속도 핵, 무한 스태미나, 루트 ESP를 메서드 후킹과 직접적인 게임 오브젝트 조작으로 제공합니다.
이 치트는 EFT의 GameWorld.RegisteredPlayers 목록을 순회하여 플레이어를 열거하고, PlayerBones를 읽어 Camera.WorldToScreenPoint로 스켈레톤 기반 월드-스크린 투영을 수행하며, CreateShot/ApplyShot/BulletMovement 같은 메서드를 후킹해 탄환을 조작하고, Unity의 OnGUI/GUI.DrawTexture와 커스텀 ImGui 스타일 메뉴 시스템으로 오버레이를 렌더링합니다.
주로 Escape From Tarkov의 Mono/.NET 인젝션 기법, Unity 게임 오브젝트 인트로스펙션, 메서드 후킹 공격면을 연구하는 게임 보안 연구자에게 유용합니다.
