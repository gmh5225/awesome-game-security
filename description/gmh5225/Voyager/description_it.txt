Questo progetto è un framework di hacking Hyper-V per Windows 10 x64 (AMD e Intel).
Funziona su tutte le versioni di Windows 10-x64 (2004–1507).
È soprattutto utile a ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / esplorazione del kernel Windows.
