Este proyecto es un framework de hacking de Hyper-V para Windows 10 x64 (AMD e Intel).
Funciona en todas las versiones de Windows 10-x64 (2004–1507).
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / exploración del kernel de Windows.
