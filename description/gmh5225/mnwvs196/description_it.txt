Questo progetto si concentra sul server privato Server-TMS-196.
Non è un server giocabile, ma ha integrato molte funzioni di base come networking con boost::asio, MySQL basato su Poco, una libreria Wz ad alte prestazioni, e il server di base è già completo (inclusi WvsGame, WvsLogin, WvsCenter e le comunicazioni tra server locali).
È soprattutto utile a ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell’ambito cheat / gioco: MapleStory.
