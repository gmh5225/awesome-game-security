Este proyecto se centra en el servidor privado Server-TMS-196.
No es un servidor jugable, pero ha integrado muchas funciones básicas como red con boost::asio, MySQL basado en Poco, una biblioteca Wz de alto rendimiento, y el servidor básico ya está hecho (incluyendo WvsGame, WvsLogin, WvsCenter y las comunicaciones entre servidores locales).
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / juego: MapleStory.
