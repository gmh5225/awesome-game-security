Dieses Projekt konzentriert sich auf den privaten Server-TMS-196.
Es handelt sich nicht um einen spielbaren Server, aber es vereint viele Basisfunktionen wie boost::asio-Netzwerk, MySQL auf Basis von Poco, eine schnelle Wz-Bibliothek; der Basisserver ist bereits fertiggestellt (einschließlich WvsGame, WvsLogin, WvsCenter und lokaler Serverkommunikation).
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Spiel: MapleStory untersuchen.
