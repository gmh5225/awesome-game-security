Ce projet se concentre sur le serveur privé Server-TMS-196.
Ce n’est pas un serveur jouable, mais il intègre de nombreuses fonctions de base telles que le réseau boost::asio, MySQL basé sur Poco, une bibliothèque Wz haute performance, et le serveur de base est déjà réalisé (notamment WvsGame, WvsLogin, WvsCenter et les communications entre serveurs locaux).
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / jeu : MapleStory.
