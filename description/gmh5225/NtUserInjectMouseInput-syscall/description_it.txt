Questo progetto documenta il percorso della syscall NtUserInjectMouseInput usata per iniettare input del mouse dalla modalità utente tramite win32k.
Il README precisa che il materiale è conservato come riferimento, non presentato come un articolo di ricerca originale.
È soprattutto utile ai ricercatori di sicurezza nei giochi che studiano l’iniezione di input, le primitive di tipo triggerbot/aimbot e le relative superfici di rilevamento.
