Ce projet documente le chemin d’appel système NtUserInjectMouseInput utilisé pour injecter des entrées souris depuis le mode utilisateur via win32k.
Le README précise qu’il est conservé à titre de référence, et non présenté comme un article de recherche original.
Il est surtout utile aux chercheurs en sécurité des jeux qui étudient l’injection d’entrées, les primitives de type triggerbot/aimbot et les surfaces de détection associées.
