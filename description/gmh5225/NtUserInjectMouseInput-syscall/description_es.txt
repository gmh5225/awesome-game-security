Este proyecto documenta la ruta de syscall NtUserInjectMouseInput usada para inyectar entrada de ratón desde modo usuario a través de win32k.
El README indica que se conserva como referencia, no como un artículo de investigación original.
Resulta especialmente útil para investigadores de seguridad en juegos que estudian la inyección de entrada, primitivas de triggerbot/aimbot y las superficies de detección relacionadas.
