В этом проекте описан путь системного вызова NtUserInjectMouseInput, используемый для внедрения мышиного ввода из пользовательского режима через win32k.
В README указано, что материал сохранён для справки, а не представлен как оригинальная исследовательская статья.
Он в первую очередь полезен исследователям игровой безопасности, изучающим внедрение ввода, примитивы triggerbot/aimbot и связанные поверхности обнаружения.
