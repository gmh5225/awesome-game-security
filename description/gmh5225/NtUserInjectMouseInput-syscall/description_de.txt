Dieses Projekt dokumentiert den NtUserInjectMouseInput-Syscall-Pfad, über den Mauseingaben aus dem User Mode über win32k injiziert werden.
In der README heißt es, dass die Inhalte zur Referenz erhalten bleiben und nicht als originale Forschungsarbeit präsentiert werden.
Es ist vor allem für Game-Security-Forscher nützlich, die Input Injection, Triggerbot-/Aimbot-Primitive und verwandte Erkennungsflächen untersuchen.
