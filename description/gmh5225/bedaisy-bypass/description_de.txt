Dieses Projekt konzentriert sich auf den Report-Bypass von bEDaisy.sys. Es stellt sicher, dass keine Reports an den Dienst zurückgesendet werden, während eine Antwort weiterhin übertragen werden kann. Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Erkundung von Anti-Cheat-Systemen (BattlEye, BE) untersuchen.
