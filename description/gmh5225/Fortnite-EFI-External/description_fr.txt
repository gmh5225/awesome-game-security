Un cheat externe pour Fortnite qui utilise les services d'exécution EFI (UEFI) pour un accès mémoire discret.
Il communique avec un pilote UEFI via NtSetSystemEnvironmentValueEx pour lire la mémoire du jeu sans charger de pilote noyau Windows, contournant ainsi la détection des pilotes par l'anti-cheat.
