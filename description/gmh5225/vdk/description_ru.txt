Этот проект — VDK (Vulnerable Driver Kit), набор инструментов и библиотек для эксплуатации уязвимых подписанных драйверов ядра Windows. Он предоставляет единый интерфейс для нескольких бэкендов уязвимых драйверов, обеспечивая произвольное чтение/запись в ядре, манипуляцию процессами и загрузку драйверов с помощью техник BYOVD. Предназначен для исследователей безопасности ядра и операторов red team, изучающих эксплуатацию уязвимых драйверов.
