Este proyecto es VDK (Vulnerable Driver Kit), una colección de herramientas y bibliotecas para explotar controladores del kernel de Windows firmados pero vulnerables. Proporciona una interfaz unificada para múltiples backends de controladores vulnerables, permitiendo lectura/escritura arbitraria en el kernel, manipulación de procesos y carga de controladores mediante técnicas BYOVD. Está dirigido a investigadores de seguridad del kernel y operadores de red team que estudian la explotación de controladores vulnerables.
