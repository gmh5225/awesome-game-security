Этот проект посвящён SDK Dump для Android.
Он в основном написан на C/C++ и C++ и сосредоточен на моддинге, генерации SDK и анализе памяти.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исследования игровых движков: Unreal.
