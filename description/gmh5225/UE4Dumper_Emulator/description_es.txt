Este proyecto se centra en el SDK Dump para Android.
Está escrito principalmente en C/C++ y C++ y se enfoca en el modding, la generación de SDK y el análisis de memoria.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheat / explorador de motores de juego:unreal.
