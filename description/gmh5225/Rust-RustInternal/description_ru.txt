Этот проект — внутренний чит для игры Rust на C++, который перехватывает runtime движка Unity. Он предоставляет ESP, аимбот, информацию об игроках и предметах, отсутствие отдачи и другие изменения геймплея через хукинг методов Mono/IL2CPP и прямой доступ к памяти. Чит работает внутри процесса игры с прямым доступом к игровым объектам Unity. Он предназначен для исследователей безопасности игр, изучающих реализации читов для FPS на базе Unity и эксплуатацию игр, защищённых EAC.
