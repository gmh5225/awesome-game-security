Ce projet est un cheat interne pour le jeu Rust, écrit en C++, qui s'accroche au runtime du moteur Unity. Il fournit ESP, aimbot, informations sur les joueurs et les objets, absence de recul et d'autres modifications de gameplay via le hooking de méthodes Mono/IL2CPP et l'accès direct à la mémoire. Le cheat s'exécute dans le processus du jeu avec un accès direct aux objets de jeu Unity. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les implémentations de cheats pour les FPS basés sur Unity et l'exploitation de jeux protégés par EAC.
