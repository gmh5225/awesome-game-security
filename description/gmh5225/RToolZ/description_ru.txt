Этот проект — RToolZ, системная утилита Windows с возможностями, подобными руткиту, для управления процессами, модулями и callback'ами. Она может скрывать/показывать процессы, перечислять kernel callback'и, удалять notification routines и изменять уровни защиты процессов через драйверный backend. Предназначена для исследователей ядра, изучающих инструменты манипуляции системой и техники руткитов.
