Este proyecto es RToolZ, una utilidad de sistema Windows con capacidades similares a un rootkit para la gestión de procesos, módulos y callbacks. Puede ocultar/mostrar procesos, enumerar callbacks del kernel, eliminar rutinas de notificación y manipular niveles de protección de procesos mediante un controlador. Está dirigido a investigadores de kernel que estudian herramientas de manipulación del sistema y técnicas de rootkit.
