Este proyecto es una herramienta para crear volcados de memoria del sistema Windows (similares a los crash dumps) desde un sistema en ejecución. Captura el contenido de la memoria física en un formato de archivo de volcado compatible con WinDbg y Volatility para el análisis forense sin conexión. Está dirigida a respondedores de incidentes y analistas forenses que necesitan crear volcados de memoria desde sistemas Windows en vivo para sus investigaciones.
