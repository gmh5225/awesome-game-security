Questo progetto è uno strumento per creare dump della memoria di sistema Windows (simili ai crash dump) da un sistema in esecuzione. Cattura il contenuto della memoria fisica in un formato di file dump compatibile con WinDbg e Volatility per l'analisi forense offline. È rivolto a incident responder e analisti forensi che devono creare dump di memoria da sistemi Windows attivi per le indagini.
