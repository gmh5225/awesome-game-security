Этот проект — инструмент для создания дампов системной памяти Windows (аналогичных crash dump) с работающей системы. Он сохраняет содержимое физической памяти в формате дампа, совместимом с WinDbg и Volatility, для офлайн-криминалистического анализа. Предназначен для специалистов по реагированию на инциденты и судебных аналитиков, которым нужно создавать дампы памяти с работающих систем Windows для расследований.
