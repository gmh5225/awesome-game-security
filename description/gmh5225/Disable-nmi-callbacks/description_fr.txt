Un pilote noyau qui désactive les callbacks NMI (Non-Maskable Interrupt) dans le noyau Windows.
Il utilise le pattern scanning pour localiser les variables liées à KiNmiInterruptStart dans ntoskrnl.exe et modifie l'affinité processeur ainsi que l'état NMI afin d'empêcher les détections anti-cheat basées sur le parcours de pile via NMI.
