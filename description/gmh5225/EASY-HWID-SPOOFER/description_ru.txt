Спуфер аппаратных идентификаторов в режиме ядра, изменяющий серийные идентификаторы диска, NIC, GPU и SMBIOS. Работает путём перехвата функций диспетчеризации драйверов и прямого патчинга физической памяти для изменения аппаратных отпечатков, передаваемых античит-системам; протестирован на Windows 10.
