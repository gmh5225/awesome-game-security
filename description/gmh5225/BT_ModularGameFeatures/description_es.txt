Este proyecto es una implementación rápida de funciones de juego modulares para la prueba «BTS».
Está escrito principalmente en C/C++ y C++ y se centra en sistemas de audio, física y animación.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el área de game engine / game engine plugins: Unreal.
