Questo progetto è una rapida implementazione di funzionalità di gioco modulari per il test «BTS».
È scritto principalmente in C/C++ e C++ e si concentra su sistemi audio, fisica e animazione.
È particolarmente utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell'area game engine / game engine plugins: Unreal.
