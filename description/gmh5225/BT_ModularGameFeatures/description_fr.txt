Ce projet est une implémentation rapide de fonctionnalités de jeu modulaires pour le test « BTS ».
Il est principalement écrit en C/C++ et C++ et se concentre sur les systèmes audio, la physique et l'animation.
Il est surtout utile aux développeurs de jeux, programmeurs moteur et chercheurs en graphisme travaillant dans le domaine game engine / game engine plugins : Unreal.
