本プロジェクトは、通常のユーザーモード割り当てだけに頼らず、VAD と PTE を操作してステージング領域を隠そうとする、未署名ドライバ支援の DLL インジェクタです。
アーカイブされた README によれば、ドライバは PatchGuard 安全な xKdEnumerateDebuggingDevices フック経由で通信し、その後ユーザースレッドをカーネルでトラップして、読み取り・書き込み・メモリ割り当て・PTE の偽装・VAD の割り当て・VAD の削除といった要求を処理するコマンドループを実行します。
内部では ntoskrnl 内で MiAllocateVad、MiInsertVad、MiInsertVadCharges をパターン検索し、メモリ操作の前後で KeStackAttachProcess を使い、マッピング後に再び VAD ノードを削除することで、注入領域が NtQueryVirtualMemory などの検査から見えにくくなるようにしています。
主に、VAD ツリーの悪用を伴うマニュアルマッピング、実行可能ページの隠蔽、ドライバによる注入後クリーンアップを研究する Windows カーネル研究者向けです。
