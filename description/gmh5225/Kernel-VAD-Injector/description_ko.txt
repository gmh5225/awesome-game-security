이 프로젝트는 일반적인 사용자 모드 할당에만 의존하지 않고 VAD와 PTE를 조작하여 스테이징 영역을 숨기려는, 서명되지 않은 드라이버 지원 DLL 인젝터입니다.
보관된 README에 따르면 드라이버는 PatchGuard에 안전한 xKdEnumerateDebuggingDevices 훅을 통해 통신한 뒤, 커널에서 사용자 스레드를 트랩하고 읽기, 쓰기, 메모리 할당, PTE 위장, VAD 할당, VAD 제거 요청을 처리하는 명령 루프를 실행합니다.
내부적으로 ntoskrnl에서 MiAllocateVad, MiInsertVad, MiInsertVadCharges를 패턴으로 찾고, 메모리 작업 전후에 KeStackAttachProcess를 사용하며, 매핑 후 VAD 노드를 다시 제거하여 주입 영역이 NtQueryVirtualMemory 등의 검사에 덜 드러나도록 합니다.
주로 VAD 트리 남용을 이용한 수동 매핑, 실행 가능 페이지 은닉, 드라이버 기반 주입 후 정리 기법을 연구하는 Windows 커널 연구자에게 유용합니다.
