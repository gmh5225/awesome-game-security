Этот проект — эмулятор модулей Valve Anti-Cheat (VAC), который загружает и выполняет модули сканирования VAC в изолированной среде (sandbox) для анализа. Он эмулирует интерфейс модулей VAC, предоставляет контролируемые среды памяти для сканирования и журналирует проверки обнаружения и сигнатурные шаблоны, которые ищут модули VAC. Предназначен для исследователей античита, изучающих методологию сканирования VAC и сигнатуры обнаружения.
