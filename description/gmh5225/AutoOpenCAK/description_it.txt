Questo progetto è uno strumento di automazione per aprire o estrarre file CAK (archivi compressi) usati in specifiche distribuzioni di giochi o software. Automatizza il processo di estrazione dei file in formato CAK, gestendo decompressione e organizzazione dei file. È rivolto a modder e ricercatori che lavorano con i formati di archivio CAK nei file di dati di gioco.
