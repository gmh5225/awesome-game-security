Este proyecto es un volcado de datos del juego en vivo de Genshin Impact (GenshinData) que recopila datos de configuración y contenido extraídos del cliente.
Está pensado como un espejo de investigación/datos para tablas de personajes, armas y tablas de juego relacionadas, no como un cliente de juego completo.
Resulta especialmente útil para investigadores de servidores privados y de datos de juego que estudian conjuntos de datos de Genshin Impact.
