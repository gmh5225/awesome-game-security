Questo progetto è un dump di dati del gioco live di Genshin Impact (GenshinData) che raccoglie dati di configurazione e contenuti estratti dal client.
È pensato come mirror di ricerca/dati per tabelle di personaggi, armi e tabelle di gioco correlate, non come un client di gioco completo.
È soprattutto utile a ricercatori di server privati e di dati di gioco che studiano i dataset di Genshin Impact.
