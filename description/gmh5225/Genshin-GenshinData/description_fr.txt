Ce projet est une dump de données du jeu en ligne Genshin Impact (GenshinData) regroupant des données de configuration et de contenu extraites du client.
Il est conçu comme un miroir de recherche/données pour les tables de personnages, d’armes et autres tables de jeu associées, et non comme un client de jeu complet.
Il est surtout utile aux chercheurs de serveurs privés et de données de jeu qui étudient les jeux de données de Genshin Impact.
