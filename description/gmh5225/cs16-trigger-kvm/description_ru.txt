Этот проект — триггербот для CS 1.6, работающий с хоста KVM/QEMU и читающий память игры через API доступа к памяти гостя KVM. Он определяет, когда прицел наведён на врага, и автоматически стреляет, внедряя события ввода. Подход на базе KVM делает чит невидимым для античита, работающего внутри гостевой ВМ. Он предназначен для исследователей безопасности игр, изучающих архитектуры читов на базе KVM/гипервизора.
