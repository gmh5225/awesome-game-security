Este proyecto es un triggerbot de CS 1.6 que opera desde un host KVM/QEMU, leyendo la memoria del juego a través de las API de acceso a la memoria del invitado de KVM. Detecta cuando la mira está sobre un enemigo y dispara automáticamente inyectando eventos de entrada. El enfoque basado en KVM hace que el cheat sea invisible para el anti-cheat que se ejecuta dentro de la VM invitada. Está dirigido a investigadores de seguridad de juegos que estudian arquitecturas de cheats basadas en KVM/hipervisor.
