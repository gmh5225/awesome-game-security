Ce projet est un triggerbot pour CS 1.6 qui s'exécute depuis un hôte KVM/QEMU, en lisant la mémoire du jeu via les API d'accès à la mémoire invité de KVM. Il détecte lorsque le viseur est sur un ennemi et tire automatiquement en injectant des événements d'entrée. L'approche basée sur KVM rend la triche invisible aux anti-cheats exécutés dans la VM invitée. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les architectures de triche basées sur KVM/hyperviseur.
