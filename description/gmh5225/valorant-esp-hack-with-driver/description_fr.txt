Ce projet porte sur un ESP qui exploite une vulnérabilité du pilote Vanguard pour se charger avant lui. Instructions : compiler et injecter avec n’importe quel injecteur noyau ; le pilote est destiné aux non-pasters. Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / jeu : Valorant.
