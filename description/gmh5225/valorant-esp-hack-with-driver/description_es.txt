Este proyecto se centra en un ESP que aprovecha una vulnerabilidad del controlador Vanguard para cargarse antes que él. Instrucciones: compilar e inyectar con cualquier inyector de kernel; el controlador es para no pasters. Es útil principalmente para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / juego: Valorant.
