Questo progetto è AcDrv, un driver del kernel Windows per anti-cheat o ricerca sulla sicurezza. Implementa capacità di monitoraggio a livello kernel come la registrazione di callback di processo, il tracciamento del caricamento dei moduli, l'intercettazione dell'accesso alla memoria o il monitoraggio delle system call tramite un'interfaccia driver personalizzata. È rivolto ai ricercatori di sicurezza del kernel che costruiscono o studiano componenti di driver anti-cheat.
