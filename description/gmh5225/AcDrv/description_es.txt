Este proyecto es AcDrv, un controlador del kernel de Windows para anticheat o investigación de seguridad. Implementa capacidades de monitorización a nivel de kernel como el registro de callbacks de procesos, el seguimiento de carga de módulos, la interceptación de acceso a memoria o la monitorización de llamadas al sistema mediante una interfaz de controlador personalizada. Está dirigido a investigadores de seguridad del kernel que construyen o estudian componentes de controladores anticheat.
