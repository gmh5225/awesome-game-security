NFD for x64dbg は、Nauz File Detector の静的スキャン機能を x64dbg デバッガに取り込む Qt ベースのプラグインです。NFD タブを追加し、現在読み込まれているモジュールやファイルをスキャンして、コンパイラ、パッカ、プロテクタ、および関連するバイナリシグネチャの検出結果を表示できます。プロジェクトは主に C++ で書かれ、Qt の UI コンポーネントを備え、x64dbg プラグイン SDK 経由で統合され、32 ビットおよび 64 ビット Windows 向けのビルドスクリプトも含みます。デバッガ内でバイナリの構築方法や保護方式を静的に特定したいリバースエンジニア、マルウェアおよびゲームセキュリティ分析者向けです。
