NFD for x64dbg — это плагин на Qt, который переносит статическое сканирование Nauz File Detector в отладчик x64dbg. Он добавляет вкладку NFD, где можно сканировать загруженный модуль или файл и просматривать результаты обнаружения компиляторов, пакеров, протекторов и связанных бинарных сигнатур. Проект в основном написан на C++ с UI-компонентами Qt и интегрируется через SDK плагинов x64dbg, включая скрипты сборки для 32- и 64-битных целей Windows. Он предназначен для реверс-инженеров и аналитиков вредоносного ПО или игровой безопасности, которым нужна статическая идентификация в отладчике того, как бинарник был собран или защищён.
