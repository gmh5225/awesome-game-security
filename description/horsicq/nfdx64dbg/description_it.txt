NFD for x64dbg è un plugin basato su Qt che porta la scansione statica di Nauz File Detector nel debugger x64dbg. Aggiunge una scheda NFD con cui è possibile analizzare il modulo o il file attualmente caricato e visualizzare i risultati di rilevamento per compilatori, packer, protector e relative firme binarie. Il progetto è scritto principalmente in C++ con componenti UI Qt e si integra tramite l’SDK dei plugin di x64dbg, includendo script di build per target Windows a 32 e 64 bit. È pensato per reverse engineer e analisti di malware o game security che necessitano di un’identificazione statica nel debugger di come un binario è stato costruito o protetto.
