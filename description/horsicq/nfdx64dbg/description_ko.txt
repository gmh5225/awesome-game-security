NFD for x64dbg는 Nauz File Detector의 정적 스캔 기능을 x64dbg 디버거에 가져오는 Qt 기반 플러그인입니다. NFD 탭을 추가하여 현재 로드된 모듈이나 파일을 스캔하고, 컴파일러, 패커, 프로텍터 및 관련 바이너리 시그니처 탐지 결과를 확인할 수 있습니다. 이 프로젝트는 주로 C++로 작성되며 Qt UI 컴포넌트를 사용하고, x64dbg 플러그인 SDK를 통해 통합되며 32비트 및 64비트 Windows 대상용 빌드 스크립트를 포함합니다. 디버거 안에서 바이너리의 빌드 또는 보호 방식을 정적으로 식별해야 하는 리버스 엔지니어와 멀웨어·게임 보안 분석가를 대상으로 합니다.
