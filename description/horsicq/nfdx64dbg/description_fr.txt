NFD for x64dbg est un plugin basé sur Qt qui intègre le scan statique de Nauz File Detector dans le débogueur x64dbg. Il ajoute un onglet NFD permettant d’analyser le module ou le fichier actuellement chargé et d’afficher les résultats de détection pour les compilateurs, packers, protecteurs et signatures binaires associées. Le projet est principalement écrit en C++ avec des composants d’interface Qt et s’intègre via le SDK de plugins x64dbg, avec des scripts de build pour les cibles Windows 32 bits et 64 bits. Il s’adresse aux ingénieurs en rétro-ingénierie et aux analystes malware ou sécurité des jeux qui ont besoin d’une identification statique dans le débogueur de la façon dont un binaire a été construit ou protégé.
