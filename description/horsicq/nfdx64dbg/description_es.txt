NFD for x64dbg es un complemento basado en Qt que incorpora el análisis estático de Nauz File Detector en el depurador x64dbg. Añade una pestaña NFD donde puedes escanear el módulo o archivo cargado actualmente y ver resultados de detección de compiladores, packers, protectores y firmas binarias relacionadas. El proyecto está escrito principalmente en C++ con componentes de interfaz Qt y se integra mediante el SDK de plugins de x64dbg, e incluye scripts de compilación para objetivos Windows de 32 y 64 bits. Está dirigido a ingenieros de ingeniería inversa y analistas de malware o seguridad en juegos que necesitan identificar de forma estática, dentro del depurador, cómo se construyó o protegió un binario.
