Этот проект — Nauz File Detector (NFD), инструмент для определения пакеров, компиляторов, протекторов и линкеров, использованных при сборке исполняемых файлов. Он сканирует PE, ELF, Mach-O и другие бинарные форматы с помощью сигнатурного обнаружения и распознаёт сотни известных компиляторов (MSVC, GCC, Clang), пакеров (UPX, ASPack), протекторов (Themida, VMProtect) и установщиков. Приложение на C++/Qt предоставляет как графический, так и интерфейс командной строки. Оно предназначено для аналитиков вредоносного ПО и реверс-инженеров, выполняющих первичный анализ бинарников.
