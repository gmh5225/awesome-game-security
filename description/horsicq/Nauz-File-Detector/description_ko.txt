이 프로젝트는 Nauz File Detector(NFD)로, 실행 파일을 만드는 데 사용된 패커, 컴파일러, 프로텍터, 링커를 식별하는 도구입니다. 시그니처 기반 탐지로 PE, ELF, Mach-O 및 기타 바이너리 형식을 검사하여 수백 가지의 알려진 컴파일러(MSVC, GCC, Clang), 패커(UPX, ASPack), 프로텍터(Themida, VMProtect), 설치 프로그램을 식별합니다. C++/Qt 애플리케이션으로 GUI와 명령줄 인터페이스를 모두 제공하며, 초기 바이너리 트리아지를 수행하는 악성코드 분석가와 리버스 엔지니어를 대상으로 합니다.
