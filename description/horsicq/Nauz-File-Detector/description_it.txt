Questo progetto è Nauz File Detector (NFD), uno strumento per identificare packer, compilatori, protettori e linker usati per costruire file eseguibili. Analizza formati PE, ELF, Mach-O e altri binari con rilevamento basato su firme per identificare centinaia di compilatori noti (MSVC, GCC, Clang), packer (UPX, ASPack), protettori (Themida, VMProtect) e installer. L’applicazione C++/Qt offre sia un’interfaccia grafica sia un’interfaccia a riga di comando. È pensata per analisti di malware e reverse engineer che effettuano il triage iniziale dei binari.
