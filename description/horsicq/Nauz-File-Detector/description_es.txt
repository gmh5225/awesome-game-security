Este proyecto es Nauz File Detector (NFD), una herramienta para identificar packers, compiladores, protectores y linkers usados para construir archivos ejecutables. Analiza formatos PE, ELF, Mach-O y otros binarios mediante detección basada en firmas para identificar cientos de compiladores conocidos (MSVC, GCC, Clang), packers (UPX, ASPack), protectores (Themida, VMProtect) e instaladores. La aplicación C++/Qt ofrece interfaces gráfica y de línea de comandos. Está orientada a analistas de malware e ingenieros inversos que realizan el triaje inicial de binarios.
