Dieses Projekt ist Nauz File Detector (NFD), ein Werkzeug zur Identifizierung von Packern, Compilern, Protectors und Linkern, die zum Erstellen von ausführbaren Dateien verwendet wurden. Es scannt PE-, ELF-, Mach-O- und andere Binärformate mit signaturbasierter Erkennung und erkennt Hunderte bekannter Compiler (MSVC, GCC, Clang), Packer (UPX, ASPack), Protectors (Themida, VMProtect) und Installer. Die C++/Qt-Anwendung bietet sowohl eine GUI als auch eine Kommandozeilenschnittstelle. Sie richtet sich an Malware-Analysten und Reverse Engineers bei der ersten Binär-Triage.
