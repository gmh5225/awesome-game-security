Este proyecto es una reimplementación al estilo Unity del código de jugabilidad de Arknights sin los assets originales del juego.
Está escrito mayormente en C# y organiza sistemas para personajes, monstruos, mazmorras, flujos de UI, eventos y la gestión del estado central del juego.
La base de código también incluye capas de gestores para recursos, audio, lógica de tienda y puntos de integración de scripts Lua.
Resulta útil para desarrolladores e investigadores que estudian la arquitectura de proyectos Unity y prototipan mecánicas de RPG de torre de defensa.
