Ce projet est une réimplémentation de style Unity du code de gameplay d'Arknights, sans les assets originaux du jeu.
Il est principalement écrit en C# et organise des systèmes pour les personnages, monstres, donjons, flux d'interface, événements et la gestion de l'état de jeu central.
La base de code inclut aussi des couches de gestion pour les ressources, l'audio, la logique de boutique, ainsi que des points d'intégration de scripts Lua.
Il est utile aux développeurs et chercheurs qui étudient l'architecture de projets Unity et prototypent des mécaniques de RPG tower defense.
