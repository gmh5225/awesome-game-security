Questo progetto è un codice sorgente di kernel Android GKI (Generic Kernel Image) personalizzato basato sul ramo comune 5.10, con supporto ABI multi-vendor. Include definizioni di simboli ABI per i principali OEM Android, tra cui Samsung, Qualcomm, MediaTek, ASUS, Motorola, OnePlus e altri, insieme a uno script di build ZenX. È soprattutto utile per sviluppatori di kernel Android e ricercatori di sicurezza mobile che lavorano con kernel personalizzati conformi a GKI su più piattaforme di dispositivi.
