Этот проект — исходный код кастомного ядра Android GKI (Generic Kernel Image) на базе общей ветки 5.10 с поддержкой ABI нескольких вендоров. В нём есть определения ABI-символов для крупных Android OEM, включая Samsung, Qualcomm, MediaTek, ASUS, Motorola, OnePlus и других, а также сборочный скрипт ZenX. В основном полезен разработчикам ядер Android и исследователям мобильной безопасности, работающим с GKI-совместимыми кастомными ядрами на разных платформах устройств.
