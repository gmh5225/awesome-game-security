Este proyecto es un código fuente de kernel Android GKI (Generic Kernel Image) personalizado basado en la rama común 5.10, con soporte ABI multivendor. Incluye definiciones de símbolos ABI para los principales OEM de Android, entre ellos Samsung, Qualcomm, MediaTek, ASUS, Motorola, OnePlus y otros, junto con un script de compilación ZenX. Es especialmente útil para desarrolladores de kernels Android e investigadores de seguridad móvil que trabajan con kernels personalizados conformes con GKI en múltiples plataformas de dispositivos.
