Este proyecto es una pequeña máquina virtual y runtime optimizadora con compilación JIT. Está implementado en C11, integra Cuik TB para la generación de código e incluye programas de prueba y benchmark basados en Lua. El sistema de compilación usa GNU Make y apunta a múltiples plataformas, incluidos entornos de escritorio y WebAssembly. Está orientado principalmente a la experimentación de rendimiento de VM, compiladores y runtimes.
