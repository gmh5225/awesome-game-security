Este proyecto es un cheat ESP de glow externo para Counter-Strike 2 que aplica efectos visuales de glow a las entidades de jugadores. La base de código en C++ incluye análisis de entidades, lectura del estado del juego, gestión de offsets, rutinas de búsqueda en memoria y gestión de procesos para leer la memoria del juego CS2 de forma externa. Es principalmente útil para investigadores de seguridad en juegos que estudian técnicas de cheats externos de CS2 y el renderizado de glow de entidades mediante manipulación de memoria.
