本プロジェクトは、ELF バイナリの JMPREL 再配置テーブルエントリを改変し、関数が誤った GOT エントリに解決されるようにすることで、逆アセンブラやデコンパイラを欺きます。
jmprel エントリ内の r_offset 値を操作し、静的解析ツール上ではインポート関数名が誤って表示される一方、実行時にはプログラムが正しく動作し続けます。
Python ベースのツールチェーンには、インポート抽出、難読化ヘッダ生成、および partial RELRO の遅延バインディングによる ELF バイナリのパッチ適用用スクリプトが含まれます。
主に、ELF 再配置に基づく難読化やアンチ逆アセンブル手法を研究するバイナリ保護研究者向けです。
