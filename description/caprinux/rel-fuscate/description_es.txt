Este proyecto ofusca binarios ELF modificando las entradas de la tabla de reubicación JMPREL para que las funciones se resuelvan en entradas GOT incorrectas, engañando a desensambladores y descompiladores.
Manipula los valores r_offset de las entradas jmprel para que los nombres de las funciones importadas se muestren incorrectamente en las herramientas de análisis estático, mientras el programa sigue funcionando correctamente en tiempo de ejecución.
La cadena de herramientas basada en Python incluye scripts para extraer importaciones, generar cabeceras de ofuscación y parchear el binario ELF con lazy binding de partial RELRO.
Resulta especialmente útil para investigadores de protección binaria que estudian la ofuscación basada en reubicaciones ELF y técnicas anti-desensamblado.
