이 프로젝트는 ELF 바이너리의 JMPREL 재배치 테이블 항목을 수정하여 함수가 잘못된 GOT 항목으로 해석되게 함으로써 디스어셈블러와 디컴파일러를 오도합니다.
jmprel 항목의 r_offset 값을 조작하여 정적 분석 도구에서는 임포트 함수 이름이 잘못 표시되지만, 런타임에는 프로그램이 정상적으로 동작합니다.
Python 기반 툴체인에는 임포트 추출, 난독화 헤더 생성, partial RELRO 지연 바인딩으로 ELF 바이너리를 패치하는 스크립트가 포함됩니다.
주로 ELF 재배치 기반 난독화 및 안티 디스어셈블리 기법을 연구하는 바이너리 보호 연구자에게 유용합니다.
