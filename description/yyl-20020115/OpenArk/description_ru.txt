Этот проект — OpenArk, открытый набор инструментов для анализа ядра Windows и борьбы с руткитами с графическим интерфейсом на Qt. Он обеспечивает перечисление процессов, потоков, модулей, драйверов и обратных вызовов, инспекцию объектов ядра, просмотр SSDT/теневого SSDT, мониторинг реестра, список сетевых соединений и разблокировку файлов. Инструмент включает компонент драйвера ядра для глубокой проверки системы и обнаружения руткитов. Он предназначен для аналитиков безопасности, исследователей вредоносного ПО и системных администраторов, выполняющих анализ систем Windows и поиск руткитов.
