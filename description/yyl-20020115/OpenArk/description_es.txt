Este proyecto es OpenArk, un kit de herramientas de código abierto para el análisis del kernel de Windows y anti-rootkit con una interfaz gráfica basada en Qt. Proporciona enumeración de procesos, hilos, módulos, controladores y callbacks, inspección de objetos del kernel, visualización de SSDT/shadow SSDT, monitorización del registro, listado de conexiones de red y desbloqueo de archivos. La herramienta incluye un componente de controlador de kernel para la inspección profunda del sistema y la detección de rootkits. Está dirigida a analistas de seguridad, investigadores de malware y administradores de sistemas que realizan análisis de sistemas Windows y caza de rootkits.
