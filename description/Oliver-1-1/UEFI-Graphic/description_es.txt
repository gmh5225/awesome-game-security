Este proyecto es un framework gráfico UEFI en C++ que simplifica el dibujo en entornos de prearranque. Envuelve los protocolos UEFI de bajo nivel y el manejo del framebuffer tras clases reutilizables para pantallas, colores, texto y formas básicas. El código incluye objetivos de soporte e implementaciones para entrada de ratón y operaciones de archivo sencillas, construido con un flujo de trabajo VisualUefi. Está dirigido a desarrolladores de firmware e investigadores de seguridad de bajo nivel que necesitan herramientas gráficas personalizadas antes de que se cargue el sistema operativo.
