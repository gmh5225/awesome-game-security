Этот проект — минимальный Windows proof of concept для обнаружения движения мыши, созданного программным обеспечением, а не оборудованием. Он предоставляет нативный код на C++, который отслеживает поведение движения и демонстрирует API имитации движения для тестирования. Репозиторий намеренно невелик и сосредоточен на одной концепции обнаружения, а не на полном стеке античита. Он полезен как отправная точка для исследований по проверке ввода.
