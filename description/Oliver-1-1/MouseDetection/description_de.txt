Dieses Projekt ist ein minimaler Windows-Proof-of-Concept zur Erkennung von Mausbewegungen, die von Software statt von Hardware erzeugt werden. Es stellt nativen C++-Code bereit, der Bewegungsverhalten überwacht und simulierte Bewegungs-APIs zum Testen demonstriert. Das Repository ist bewusst klein gehalten und konzentriert sich auf ein einziges Erkennungskonzept statt auf einen vollständigen Anti-Cheat-Stack. Es eignet sich als Ausgangspunkt für die Forschung zur Eingabevalidierung.
