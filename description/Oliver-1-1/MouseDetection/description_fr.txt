Ce projet est une preuve de concept Windows minimale pour détecter les mouvements de souris générés par un logiciel plutôt que par le matériel. Il fournit du code C++ natif qui surveille le comportement des mouvements et démontre des API de mouvement simulé pour les tests. Le dépôt est volontairement petit et centré sur un seul concept de détection, plutôt que sur une pile anti-triche complète. Il constitue un point de départ utile pour la recherche sur la validation des entrées.
