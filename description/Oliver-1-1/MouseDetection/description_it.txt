Questo progetto è una proof of concept Windows minimale per rilevare il movimento del mouse generato da software anziché da hardware. Fornisce codice C++ nativo che monitora il comportamento del movimento e dimostra API di movimento simulato per i test. Il repository è intenzionalmente piccolo e incentrato su un solo concetto di rilevamento, anziché su uno stack anti-cheat completo. È utile come punto di partenza per la ricerca sulla validazione degli input.
