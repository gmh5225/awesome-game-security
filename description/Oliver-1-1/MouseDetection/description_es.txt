Este proyecto es una prueba de concepto mínima de Windows para detectar el movimiento del ratón generado por software en lugar de por hardware. Proporciona código C++ nativo que monitoriza el comportamiento del movimiento y demuestra APIs de movimiento simulado para pruebas. El repositorio es intencionadamente pequeño y se centra en un único concepto de detección, en lugar de en una pila completa de anti-trampas. Es útil como punto de partida para la investigación sobre validación de entrada.
