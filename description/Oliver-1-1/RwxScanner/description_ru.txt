Этот проект — сканер режима ядра Windows, который перечисляет таблицы страниц процессов для поиска доступных для записи и исполнения областей памяти. Драйвер обходит структуры PML4, PDPT, PD и PT через чтение физической памяти и журналирует подозрительные отображения в контексте каждого процесса. Он также сообщает метаданные процесса, такие как имя образа и состояние токена администратора, для поддержки расследования. Предназначен для низкоуровневых исследований античита и обнаружения вредоносного ПО, связанных с внедрённым или самоизменяющимся кодом.
