Este proyecto es un escáner en modo kernel de Windows que enumera las tablas de páginas de procesos para encontrar regiones de memoria escribibles y ejecutables. El controlador recorre las estructuras PML4, PDPT, PD y PT mediante lecturas de memoria física y registra asignaciones sospechosas por contexto de proceso. También informa metadatos del proceso, como el nombre de imagen y el estado del token de administrador, para facilitar la investigación. Está pensado para investigación de bajo nivel en anti-cheat y detección de malware en torno a código inyectado o auto-modificable.
