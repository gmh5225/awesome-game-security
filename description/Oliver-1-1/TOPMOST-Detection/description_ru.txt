Этот проект — небольшая утилита на Windows C++ для обнаружения окон поверх всех остальных на рабочем столе. Она перечисляет видимые окна через Win32 API и проверяет расширенные стили, такие как WS_EX_TOPMOST, чтобы помечать всегда поверх остальных оверлеи. В решение также входит сопутствующее консольное приложение, которое с помощью SetWindowPos делает себя topmost для тестирования. В основном полезно для прототипирования античита и экспериментов по безопасности игр, которым нужна базовая логика обнаружения оверлеев.
