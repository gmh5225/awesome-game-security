Questo progetto è una piccola utility Windows C++ che rileva le finestre topmost sul desktop. Enumera le finestre visibili con le API Win32 e controlla stili estesi come WS_EX_TOPMOST per segnalare overlay sempre in primo piano. La soluzione include anche un'app console di accompagnamento che si imposta come topmost con SetWindowPos per i test. È principalmente utile per prototipi anti-cheat ed esperimenti di game security che richiedono una logica di base per il rilevamento degli overlay.
