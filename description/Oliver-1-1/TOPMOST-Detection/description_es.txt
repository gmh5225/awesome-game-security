Este proyecto es una utilidad pequeña en Windows C++ que detecta ventanas topmost en el escritorio. Enumera ventanas visibles con las API de Win32 y comprueba estilos extendidos como WS_EX_TOPMOST para señalar superposiciones siempre encima. La solución también incluye una aplicación de consola asociada que se marca a sí misma como topmost con SetWindowPos para pruebas. Es útil principalmente para prototipos de anti-cheat y experimentos de seguridad en juegos que necesitan lógica básica de detección de overlays.
