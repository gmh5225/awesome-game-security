Ce projet est un petit utilitaire Windows C++ qui détecte les fenêtres au premier plan sur le bureau. Il énumère les fenêtres visibles via les API Win32 et vérifie les styles étendus tels que WS_EX_TOPMOST pour signaler les overlays toujours au premier plan. La solution inclut également une application console associée qui se place elle-même au premier plan avec SetWindowPos pour les tests. Il est surtout utile pour le prototypage d'anti-cheat et les expériences de sécurité des jeux nécessitant une logique de détection d'overlay de base.
