Questo progetto è un'implementazione di hooking ETW a livello kernel Windows che adatta l'intercettazione di syscall in stile InfinityHook alle versioni di sistema più recenti. Mostra come i percorsi di tracing ETW e le callback HalPrivateDispatchTable possano essere usati per reindirizzare la gestione delle syscall senza patchare direttamente le routine Nt. Il codice è scritto in C/C++ come progetto di driver kernel Visual Studio e include note dettagliate di reverse engineering sulla configurazione dei contatori PMC e del tracing. È rivolto alla ricerca avanzata di bypass anti-cheat e alla sperimentazione di sicurezza Windows a basso livello.
