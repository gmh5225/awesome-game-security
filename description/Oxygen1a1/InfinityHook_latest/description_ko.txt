이 프로젝트는 InfinityHook 스타일의 시스템 콜 가로채기를 최신 시스템 버전에 맞게 적응시킨 Windows 커널 ETW 후킹 구현입니다. ETW 추적 경로와 HalPrivateDispatchTable 콜백을 활용하여 Nt 루틴을 직접 패치하지 않고도 시스템 콜 처리를 리다이렉트하는 방법을 보여줍니다. 코드는 C/C++로 작성된 Visual Studio 커널 드라이버 프로젝트이며, PMC 카운터 설정과 추적 구성에 대한 상세한 리버스 엔지니어링 노트를 포함합니다. 고급 안티치트 우회 연구와 저수준 Windows 보안 실험을 대상으로 합니다.
