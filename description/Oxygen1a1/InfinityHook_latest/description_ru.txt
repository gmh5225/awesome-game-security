Этот проект — реализация ETW-хукинга в ядре Windows, адаптирующая перехват системных вызовов в стиле InfinityHook к более новым версиям системы. Он показывает, как пути трассировки ETW и колбэки HalPrivateDispatchTable можно использовать для перенаправления обработки syscall без прямого патчинга Nt-процедур. Код написан на C/C++ как проект драйвера ядра Visual Studio и включает подробные заметки по реверс-инжинирингу настройки счётчиков PMC и конфигурации трассировки. Предназначен для продвинутых исследований обхода античита и низкоуровневых экспериментов по безопасности Windows.
