Este proyecto es una implementación de hooking ETW en el kernel de Windows que adapta la interceptación de syscalls al estilo InfinityHook a versiones más recientes del sistema. Demuestra cómo se pueden aprovechar las rutas de tracing ETW y las callbacks de HalPrivateDispatchTable para redirigir el manejo de syscalls sin parchear directamente las rutinas Nt. El código está escrito en C/C++ como un proyecto de controlador de kernel de Visual Studio e incluye notas detalladas de ingeniería inversa sobre la configuración de contadores PMC y del tracing. Está orientado a la investigación avanzada de bypass de anti-cheat y a la experimentación de seguridad Windows a bajo nivel.
