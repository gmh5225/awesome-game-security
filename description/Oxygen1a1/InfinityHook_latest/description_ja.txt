本プロジェクトは、InfinityHook 風のシステムコール横取りを新しい Windows バージョン向けに適応した、カーネル ETW フッキング実装です。ETW トレース経路と HalPrivateDispatchTable コールバックを活用し、Nt ルーチンを直接パッチせずにシステムコール処理をリダイレクトする方法を示します。コードは C/C++ による Visual Studio のカーネルドライバプロジェクトとして書かれ、PMC カウンタ設定とトレース構成に関する詳細なリバースエンジニアリング解説を含みます。高度なアンチチート回避研究および低レベルな Windows セキュリティ実験を対象としています。
