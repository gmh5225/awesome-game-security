Ce projet est une implémentation de hooking ETW au niveau du noyau Windows qui adapte l'interception de syscalls de type InfinityHook aux versions plus récentes du système. Il montre comment les chemins de tracing ETW et les callbacks HalPrivateDispatchTable peuvent être utilisés pour rediriger le traitement des syscalls sans patcher directement les routines Nt. Le code est écrit en C/C++ sous forme de projet de pilote noyau Visual Studio et inclut des notes détaillées de reverse engineering sur la configuration des compteurs PMC et du tracing. Il s'adresse à la recherche avancée de contournement d'anti-cheat et à l'expérimentation de sécurité Windows bas niveau.
