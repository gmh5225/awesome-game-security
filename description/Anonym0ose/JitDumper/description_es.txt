Este proyecto es una cadena de herramientas centrada en Windows para volcar cuerpos de métodos CIL de .NET interceptando los internos de la compilación JIT. Combina una aplicación de análisis en C# con un componente de hook en C++ que aprovecha Detours y datos de símbolos o PDB. La implementación admite varias generaciones de .NET y reconstruye los metadatos necesarios para inspeccionar el comportamiento de los métodos compilados. Está dirigido a ingenieros inversos y analistas de protección de software que trabajan con los internos del código administrado.
