Ce projet est une chaîne d'outils axée sur Windows pour dumper les corps de méthodes CIL .NET en interceptant les internals de la compilation JIT. Il combine une application d'analyse en C# avec un composant de hook en C++ qui s'appuie sur Detours et des données de symboles ou PDB. L'implémentation prend en charge plusieurs générations de .NET et reconstruit les métadonnées nécessaires pour inspecter le comportement des méthodes compilées. Il s'adresse aux reverse engineers et aux analystes de protection logicielle travaillant sur les internals du code managé.
