Этот проект — ориентированный на Windows набор инструментов для дампа тел методов .NET CIL путём перехвата внутренних механизмов JIT-компиляции. Он сочетает аналитическое приложение на C# с компонентом хуков на C++, использующим Detours и данные символов или PDB. Реализация поддерживает несколько поколений .NET и восстанавливает метаданные, необходимые для анализа поведения скомпилированных методов. Предназначен для реверс-инженеров и аналитиков программной защиты, работающих с внутренним устройством управляемого кода.
