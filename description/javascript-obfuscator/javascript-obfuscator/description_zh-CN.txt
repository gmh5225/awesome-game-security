JavaScript Obfuscator 是一款免费、功能强大的 JavaScript 与 Node.js 混淆器，通过变换源代码来阻碍阅读、逆向工程与自动化反混淆。它主要用 TypeScript 实现，以 CLI 工具和可编程的 Node.js API 形式提供，并支持从轻度到重度保护的可配置预设。核心技术包括标识符重命名、带 base64 或 RC4 编码的字符串数组提取、控制流平坦化、死代码注入、对象键变换，以及可选的自我防护与调试保护辅助功能，用于抵御美化与基于 DevTools 的分析。域名锁定及相关运行时守卫可进一步限制混淆代码的运行环境。它被广泛用于保护客户端与 Node.js 应用逻辑，包括浏览器游戏及其他需要源代码保密性的 JavaScript 资源。
