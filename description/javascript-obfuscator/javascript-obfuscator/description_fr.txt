JavaScript Obfuscator est un obfuscateur gratuit et puissant pour JavaScript et Node.js qui transforme le code source afin de gêner la lecture, la rétro-ingénierie et la désobfuscation automatisée. Implémenté principalement en TypeScript, il est fourni en outil CLI et en API Node.js programmable, avec des préréglages configurables allant d’une protection légère à une protection renforcée. Les techniques principales incluent le renommage des identifiants, l’extraction de tableaux de chaînes avec encodage base64 ou RC4, l’aplatissement du flot de contrôle, l’injection de code mort, la transformation des clés d’objets, ainsi que des aides optionnelles d’auto-défense et de protection contre le débogage qui résistent à l’embellissement et à l’analyse via DevTools. Le verrouillage de domaine et des gardes d’exécution associées limitent encore davantage les environnements où le code obfusqué peut s’exécuter. Il est largement utilisé pour protéger la logique des applications côté client et Node.js, notamment les jeux navigateur et d’autres ressources JavaScript où la confidentialité du source compte.
