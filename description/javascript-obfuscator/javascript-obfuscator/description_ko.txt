JavaScript Obfuscator는 JavaScript 및 Node.js용 무료·강력한 난독화 도구로, 소스 코드를 변환하여 읽기, 리버스 엔지니어링, 자동 난독화 해제를 어렵게 만듭니다. 주로 TypeScript로 구현되며 CLI 도구와 프로그래밍 가능한 Node.js API로 제공되고, 가벼운 보호부터 강한 보호까지 설정 가능한 프리셋을 지원합니다. 핵심 기법에는 식별자 이름 변경, base64 또는 RC4 인코딩을 사용한 문자열 배열 추출, 제어 흐름 평탄화, 죽은 코드 삽입, 객체 키 변환, 그리고 코드 미화와 DevTools 기반 분석을 방해하는 선택적 자기 방어 및 디버그 보호 헬퍼가 포함됩니다. 도메인 잠금과 관련 런타임 가드는 난독화된 코드가 실행될 수 있는 환경을 추가로 제한합니다. 브라우저 게임 등 소스 기밀성이 중요한 JavaScript 자산을 포함해 클라이언트 측 및 Node.js 애플리케이션 로직 보호에 널리 사용됩니다.
