JavaScript Obfuscator è un ofuscatore gratuito e potente per JavaScript e Node.js che trasforma il codice sorgente per ostacolare la lettura, il reverse engineering e la deofuscazione automatizzata. Implementato principalmente in TypeScript, viene fornito come strumento CLI e come API programmatica Node.js, con preset configurabili dalla protezione leggera a quella intensa. Le tecniche principali includono la ridenominazione degli identificatori, l’estrazione di array di stringhe con codifica base64 o RC4, il control-flow flattening, l’iniezione di dead code, la trasformazione delle chiavi degli oggetti e helper opzionali di self-defending e protezione dal debug che resistono all’abbellimento e all’analisi basata su DevTools. Il domain locking e le relative guardie a runtime limitano ulteriormente gli ambienti in cui il codice ofuscato può essere eseguito. È ampiamente usato per proteggere la logica di applicazioni client-side e Node.js, inclusi giochi browser e altri asset JavaScript in cui conta la riservatezza del sorgente.
