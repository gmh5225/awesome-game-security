JavaScript Obfuscator — бесплатный мощный обфускатор для JavaScript и Node.js, который преобразует исходный код, затрудняя чтение, обратную инженерию и автоматическую деобфускацию. Реализован преимущественно на TypeScript и поставляется как CLI-инструмент и программный API для Node.js с настраиваемыми пресетами от лёгкой до сильной защиты. Основные техники включают переименование идентификаторов, извлечение строковых массивов с кодированием base64 или RC4, выравнивание потока управления, внедрение мёртвого кода, преобразование ключей объектов, а также опциональные средства самозащиты и защиты от отладки, устойчивые к форматированию и анализу через DevTools. Блокировка по домену и связанные runtime-ограничения дополнительно сужают среды, в которых может выполняться обфусцированный код. Его широко применяют для защиты логики клиентских и Node.js-приложений, включая браузерные игры и другие JavaScript-ресурсы, где важна конфиденциальность исходников.
