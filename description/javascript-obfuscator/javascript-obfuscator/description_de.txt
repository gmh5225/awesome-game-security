JavaScript Obfuscator ist ein kostenloser, leistungsfähiger Obfuscator für JavaScript und Node.js, der Quellcode so transformiert, dass Lesen, Reverse Engineering und automatisierte Deobfuscation erschwert werden. Er ist vorwiegend in TypeScript umgesetzt und wird als CLI-Tool sowie als programmierbare Node.js-API mit konfigurierbaren Presets von leichter bis starker Absicherung ausgeliefert. Zu den Kerntechniken zählen Umbenennung von Bezeichnern, Extraktion von String-Arrays mit base64- oder RC4-Kodierung, Control-Flow-Flattening, Dead-Code-Injection, Transformation von Objekt-Schlüsseln sowie optionale Self-Defending- und Debug-Protection-Hilfen, die Beautification und DevTools-basierter Analyse widerstehen. Domain Locking und verwandte Runtime-Guards schränken zusätzlich ein, wo obfuskierter Code ausgeführt werden darf. Er wird häufig zum Schutz von Client- und Node.js-Anwendungslogik eingesetzt, einschließlich Browser-Spielen und anderer JavaScript-Assets, bei denen Quellgeheimhaltung wichtig ist.
