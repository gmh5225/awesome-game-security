JavaScript Obfuscator es un ofuscador gratuito y potente para JavaScript y Node.js que transforma el código fuente para dificultar la lectura, la ingeniería inversa y la desofuscación automatizada. Implementado principalmente en TypeScript, se distribuye como herramienta CLI y como API programática de Node.js, con presets configurables desde protección ligera hasta intensa. Las técnicas principales incluyen el renombrado de identificadores, la extracción de arrays de cadenas con codificación base64 o RC4, el aplanamiento del flujo de control, la inyección de código muerto, la transformación de claves de objetos y ayudas opcionales de autodefensa y protección frente a depuración que resisten el embellecimiento y el análisis basado en DevTools. El bloqueo por dominio y las salvaguardas de tiempo de ejecución relacionadas restringen aún más dónde puede ejecutarse el código ofuscado. Se usa ampliamente para proteger la lógica de aplicaciones del lado del cliente y de Node.js, incluidos juegos de navegador y otros activos JavaScript donde importa la confidencialidad del código fuente.
