JavaScript Obfuscator 是一款免費、功能強大的 JavaScript 與 Node.js 混淆器，透過轉換原始碼來阻礙閱讀、逆向工程與自動化反混淆。它主要以 TypeScript 實作，提供 CLI 工具與可程式化的 Node.js API，並支援從輕度到重度保護的可設定預設組。核心技術包括識別碼重新命名、搭配 base64 或 RC4 編碼的字串陣列提取、控制流程平坦化、死碼注入、物件鍵轉換，以及可選的自我防護與偵錯保護輔助功能，用以抵抗美化與基於 DevTools 的分析。網域鎖定及相關執行時期守衛可進一步限制混淆程式碼可執行的環境。它被廣泛用於保護用戶端與 Node.js 應用邏輯，包括瀏覽器遊戲及其他需要原始碼保密性的 JavaScript 資源。
