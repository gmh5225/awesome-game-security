Questo progetto è un’ampia raccolta di esempi di codice orientati ai rootkit Linux, che coprono kernel space, user space ed eBPF.
La maggior parte delle implementazioni è in C e illustra tecniche come l’occultamento di file e connessioni, l’hooking di syscall e ftrace, l’escalation dei privilegi, la persistenza e il comportamento anti-forense.
Il repository è organizzato in molti sottoprogetti mirati, con script di build e brevi spiegazioni di ciascun meccanismo.
Il caso d’uso principale è la ricerca offensiva e difensiva a scopo educativo, inclusi test di rilevamento anti-rootkit e sperimentazione sulla sicurezza del kernel.
