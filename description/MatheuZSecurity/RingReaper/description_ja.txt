本プロジェクトは、io_uring 操作に大きく依存することで検知を最小限に抑えるよう設計された Linux のポストエクスプロイトエージェントです。
主に C で書かれ、Python のコントロールサーバーを備え、従来の read、write、send、receive の多くを非同期カーネル I/O プリミティブに置き換えています。
エージェントはファイル転送、プロセスおよびユーザー列挙、ネットワーク検査、セッション制御などのコマンド機能を提供し、データフローの大部分を io_uring バックエンド上に保ちます。
攻撃的セキュリティ研究および、管理・許可された環境での EDR 回避テストを目的としています。
