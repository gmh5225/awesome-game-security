Этот проект — Linux-агент для постэксплуатации, рассчитанный на минимизацию обнаружения за счёт активного использования операций io_uring.
Он в основном написан на C и включает управляющий сервер на Python, заменяя многие традиционные пути read, write, send и receive асинхронными примитивами ввода-вывода ядра.
Агент предоставляет команды для передачи файлов, перечисления процессов и пользователей, проверки сети и управления сеансом, удерживая большую часть потока данных на бэкендах io_uring.
Он предназначен для исследований в области offensive security и тестирования обхода EDR в контролируемых и авторизованных средах.
