Este proyecto es un agente de post-explotación para Linux diseñado para minimizar la detección apoyándose en gran medida en operaciones io_uring.
Está escrito principalmente en C con un servidor de control en Python, y sustituye muchas rutas tradicionales de read, write, send y receive por primitivas de E/S asíncronas del kernel.
El agente ofrece funciones de comando para transferencia de archivos, enumeración de procesos y usuarios, inspección de red y control de sesión, manteniendo la mayor parte del flujo de datos en backends io_uring.
Está pensado para investigación de seguridad ofensiva y pruebas de evasión de EDR en entornos controlados y autorizados.
