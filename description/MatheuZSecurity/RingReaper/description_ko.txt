이 프로젝트는 io_uring 연산에 크게 의존하여 탐지를 최소화하도록 설계된 Linux 포스트 익스플로잇 에이전트입니다.
주로 C로 작성되었으며 Python 제어 서버와 함께 동작하고, 기존의 많은 read, write, send, receive 경로를 비동기 커널 I/O 프리미티브로 대체합니다.
에이전트는 파일 전송, 프로세스 및 사용자 열거, 네트워크 검사, 세션 제어 등의 명령 기능을 제공하며, 대부분의 데이터 흐름을 io_uring 백엔드에 유지합니다.
공격적 보안 연구와 통제되고 승인된 환경에서의 EDR 우회 테스트를 위해 고안되었습니다.
