Dieses Projekt ist eine schlanke Offset-Datenbank für Fortnite. Sie wird als JSON-Datei bereitgestellt und speichert wichtige Speicher-Offsets für Engine-Zeiger, Entity-Strukturen, Kameradaten und waffenbezogene Felder. Das Repository enthält Daten statt Anwendungslogik und lässt sich daher leicht von externen Tools nutzen. Der Hauptanwendungsfall ist, Cheat- oder Research-Tools mit den aktuellen Speichermustern des Spiels synchron zu halten.
