Ce projet est une base de données d'offsets légère pour Fortnite. Elle est fournie sous forme de fichier JSON qui stocke les offsets mémoire clés pour les pointeurs moteur, les structures d'entités, les données de caméra et les champs liés aux armes. Le dépôt contient des données plutôt que de la logique applicative, ce qui facilite son utilisation par des outils externes. Son cas d'usage principal est de maintenir les outils de cheat ou de recherche synchronisés avec les layouts mémoire actuels du jeu.
