Questo progetto è un database leggero di offset per Fortnite. Viene fornito come file JSON che memorizza offset di memoria chiave per puntatori del motore, strutture delle entità, dati della fotocamera e campi relativi alle armi. Il repository contiene dati anziché logica applicativa, facilitandone l'uso da strumenti esterni. Il caso d'uso principale è mantenere gli strumenti di cheat o di ricerca sincronizzati con i layout di memoria attuali del gioco.
