Este proyecto es una base de datos ligera de offsets para Fortnite. Se proporciona como un archivo JSON que almacena offsets de memoria clave para punteros del motor, estructuras de entidades, datos de cámara y campos relacionados con armas. El repositorio contiene datos en lugar de lógica de aplicación, lo que facilita su uso desde herramientas externas. Su caso de uso principal es mantener herramientas de cheat o de investigación sincronizadas con los layouts de memoria actuales del juego.
