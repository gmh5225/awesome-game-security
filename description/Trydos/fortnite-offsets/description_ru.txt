Этот проект — лёгкая база данных оффсетов для Fortnite. Она предоставляется в виде JSON-файла, в котором хранятся ключевые смещения памяти для указателей движка, структур сущностей, данных камеры и полей, связанных с оружием. Репозиторий содержит данные, а не прикладную логику, поэтому его удобно использовать во внешних инструментах. Основное применение — синхронизация читов или исследовательских инструментов с актуальными раскладками памяти игры.
