WindEffects is a modular C++ game engine with a full editor, runtime, and project-template system for building games across multiple genres. It includes rendering backends for Vulkan and DirectX 12, HLSL-based shaders for deferred/PBR lighting, post-processing (bloom, TAA, SSAO, SSR), volumetric clouds and fog, and sky atmosphere. The engine ships editor tooling such as a content browser and KindUI widget framework, plus subsystems for scene entities, terrain streaming and foliage, text shaping, and cross-platform support via SDL3 and platform backends for Windows, Linux, macOS, Android, and iOS. It is aimed at game developers and engineers studying or extending a modern engine architecture, including graphics and tooling internals relevant to game security research.
