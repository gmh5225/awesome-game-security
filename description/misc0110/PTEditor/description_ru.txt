Модуль ядра Linux и пользовательская библиотека для прямого чтения и изменения всех уровней таблиц страниц (PGD, PUD, PMD, PTE) любого процесса из пользовательского пространства на x86_64 и ARMv8. Поддерживает преобразование виртуальных адресов в физические, программирование типов памяти PAT/MAIR, манипуляцию битом NX и сброс TLB, а также включает сопутствующий драйвер ядра Windows для кроссплатформенного использования.
