Módulo del kernel de Linux y biblioteca de espacio de usuario para leer y modificar directamente todos los niveles de tablas de páginas (PGD, PUD, PMD, PTE) de cualquier proceso desde el espacio de usuario en x86_64 y ARMv8. Admite la traducción de direcciones virtuales a físicas, la programación de tipos de memoria PAT/MAIR, la manipulación del bit NX y el vaciado del TLB, con un controlador del kernel de Windows acompañante para uso multiplataforma.
