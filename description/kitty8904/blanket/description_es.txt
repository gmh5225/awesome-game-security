Este proyecto es Blanket, una herramienta de Windows que oculta la presencia de un proceso desvinculándolo de las listas de procesos del kernel y manipulando las API de enumeración de procesos. Elimina la entrada EPROCESS del proceso objetivo de ActiveProcessLinks, parchea las entradas de PspCidTable y engancha NtQuerySystemInformation para ocultar el proceso del Administrador de tareas y otras herramientas de enumeración. Está dirigido a investigadores de kernel que estudian técnicas de rootkit de ocultación de procesos y sus métodos de detección.
