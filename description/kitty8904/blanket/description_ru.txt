Этот проект — Blanket, инструмент для Windows, который скрывает наличие процесса, отвязывая его от списков процессов ядра и манипулируя API перечисления процессов. Он удаляет запись EPROCESS целевого процесса из ActiveProcessLinks, патчит записи PspCidTable и перехватывает NtQuerySystemInformation, чтобы скрыть процесс от Диспетчера задач и других средств перечисления. Предназначен для исследователей ядра, изучающих rootkit-техники сокрытия процессов и методы их обнаружения.
