DetectNtoskrnlIntegrity — исследовательский проект по целостности ядра Windows, посвящённый проверке ntoskrnl.exe в памяти относительно его образа на диске. Он представляет код и методологию обнаружения подмены ядра с учётом практических сложностей: преобразований, связанных с SSDT, рандомизации таблиц страниц и изменений поведения эпохи retpoline. Реализация написана на C++ для низкоуровневого системного анализа на современных сборках Windows. Проект в первую очередь полезен для исследований anti-rootkit, anti-cheat и защитной безопасности ядра.
