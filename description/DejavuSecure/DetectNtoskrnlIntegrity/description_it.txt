DetectNtoskrnlIntegrity è un progetto di ricerca sull'integrità del kernel Windows incentrato sulla convalida di ntoskrnl.exe in memoria rispetto alla sua immagine su disco. Presenta codice e metodologia per rilevare manomissioni del kernel tenendo conto di complicazioni pratiche come trasformazioni legate alla SSDT, randomizzazione delle page table e cambiamenti di comportamento dell'era retpoline. L'implementazione è scritta in C++ per l'analisi di sistema a basso livello su build Windows moderne. È principalmente utile per flussi di lavoro di ricerca anti-rootkit, anti-cheat e di sicurezza difensiva del kernel.
