DetectNtoskrnlIntegrity ist ein Forschungsprojekt zur Integrität des Windows-Kernels, das sich auf die Validierung von ntoskrnl.exe im Speicher gegenüber seinem Abbild auf der Festplatte konzentriert. Es stellt Code und Methoden zur Erkennung von Kernel-Manipulation bereit und berücksichtigt praktische Komplikationen wie SSDT-bezogene Transformationen, Page-Table-Randomisierung und Verhaltensänderungen der Retpoline-Ära. Die Implementierung ist in C++ für die Low-Level-Systemanalyse auf modernen Windows-Builds geschrieben. Es eignet sich vor allem für Anti-Rootkit-, Anti-Cheat- und defensive Kernel-Sicherheitsforschung.
