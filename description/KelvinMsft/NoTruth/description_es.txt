Este proyecto es una implementación de investigación de código abierto para Windows x64 que oculta memoria en modo usuario mediante técnicas VT-x y EPT. Demuestra cómo las operaciones de lectura pueden redirigirse a valores falsos mientras se mantiene el control de la semántica de ejecución, lo que permite experimentos sobre evasiones de checksum e integridad. La base de código es principalmente C y C++ a nivel de kernel, con componentes de controlador y de prueba diseñados para entornos con capacidad de virtualización. Está dirigido a investigadores de seguridad de bajo nivel que estudian el engaño de memoria, la evasión de anti-cheat y la instrumentación asistida por hipervisor.
