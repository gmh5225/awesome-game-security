Этот проект — открытая исследовательская реализация для Windows x64, скрывающая пользовательскую память с помощью техник VT-x и EPT. Он демонстрирует, как операции чтения можно перенаправлять на поддельные значения, сохраняя контроль над семантикой выполнения, что позволяет экспериментировать с обходом контрольных сумм и проверок целостности. Кодовая база в основном представляет собой код уровня ядра на C и C++ с драйвером и тестовыми компонентами для сред с поддержкой виртуализации. Проект предназначен для исследователей низкоуровневой безопасности, изучающих обман памяти, обход античита и инструментирование с помощью гипервизора.
