Questo progetto è un'implementazione di ricerca open source per Windows x64 volta a nascondere la memoria in user mode con tecniche VT-x ed EPT. Mostra come le operazioni di lettura possano essere reindirizzate a valori falsi mantenendo il controllo della semantica di esecuzione, consentendo esperimenti su bypass di checksum e integrità. Il codice è in gran parte C e C++ a livello kernel, con componenti driver e di test progettati per ambienti con capacità di virtualizzazione. È rivolto a ricercatori di sicurezza low-level che indagano l'inganno della memoria, l'evasione anti-cheat e l'instrumentazione assistita da hypervisor.
