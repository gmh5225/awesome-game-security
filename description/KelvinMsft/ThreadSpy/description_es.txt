Este proyecto es un framework de secuestro de hilos asistido por hardware para Windows. Se centra en tomar el control de hilos en ejecución sobre la marcha sin parchear bytes de instrucciones, lo que permite la inyección de código o operaciones de memoria dentro del contexto de procesos objetivo. El código es principalmente C++ con componentes orientados al kernel y configuración de compilación para las toolchains de Visual Studio y WDK. Está diseñado para investigación avanzada en seguridad sobre redirección furtiva de ejecución y técnicas de bypass de anti-cheat.
