Questo progetto è un framework di thread hijacking assistito dall'hardware per Windows. Si concentra sull'acquisizione al volo di thread in esecuzione senza patchare i byte delle istruzioni, consentendo code injection o operazioni di memoria nel contesto dei processi target. Il codice è principalmente in C++ con componenti orientati al kernel e setup di build per i toolchain Visual Studio e WDK. È progettato per la ricerca avanzata in sicurezza sulla redirezione stealth dell'esecuzione e sulle tecniche di bypass degli anti-cheat.
