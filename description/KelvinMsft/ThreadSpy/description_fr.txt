Ce projet est un framework de détournement de threads assisté par le matériel pour Windows. Il se concentre sur la prise de contrôle à la volée de threads en cours d'exécution sans patcher les octets d'instructions, permettant l'injection de code ou des opérations mémoire dans le contexte des processus cibles. Le code est principalement en C++ avec des composants orientés noyau et une configuration de build pour les toolchains Visual Studio et WDK. Il est conçu pour la recherche avancée en sécurité sur la redirection d'exécution furtive et les techniques de contournement d'anti-cheat.
