Этот проект — аппаратно-поддерживаемый фреймворк перехвата потоков для Windows. Он сосредоточен на захвате работающих потоков на лету без патчинга байтов инструкций, что позволяет выполнять внедрение кода или операции с памятью в контексте целевых процессов. Код написан в основном на C++ и включает компоненты, ориентированные на ядро, а также настройки сборки для toolchain Visual Studio и WDK. Предназначен для углублённых исследований в области безопасности, связанных со скрытым перенаправлением исполнения и техниками обхода античита.
