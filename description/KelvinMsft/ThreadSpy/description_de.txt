Dieses Projekt ist ein hardwaregestütztes Thread-Hijacking-Framework für Windows. Es konzentriert sich darauf, laufende Threads on-the-fly zu übernehmen, ohne Instruction Bytes zu patchen, und ermöglicht so Code Injection oder Speicheroperationen im Kontext von Zielprozessen. Der Code ist hauptsächlich in C++ geschrieben und umfasst kernelnahe Komponenten sowie Build-Setup für Visual Studio- und WDK-Toolchains. Es ist für fortgeschrittene Sicherheitsforschung zu stealthiger Execution Redirection und Anti-Cheat-Bypass-Techniken konzipiert.
