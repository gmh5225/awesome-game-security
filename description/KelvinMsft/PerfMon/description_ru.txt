Этот проект — исследовательский драйвер ядра Windows, использующий механизмы аппаратного мониторинга производительности для низкоуровневого контроля и наблюдения. Он работает с потоками PMU и PMI, обработкой APIC и связанными путями прерываний, чтобы исследовать техники, связанные с мониторингом SSDT и перехватом в стиле hook на современных системах Windows. Реализация в основном на C и C++, а в репозитории есть справочные статьи и небольшая тестовая программа для экспериментов. Предназначен для исследователей безопасности ядра и античита, изучающих аппаратно поддерживаемый мониторинг на платформах эпохи Windows 10.
