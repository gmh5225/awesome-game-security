Questo progetto è un driver di ricerca per il kernel Windows che usa meccanismi di monitoraggio delle prestazioni hardware per controllo e osservazione a basso livello. Lavora con i flussi PMU e PMI, la gestione APIC e i relativi percorsi di interrupt per esplorare tecniche legate al monitoraggio SSDT e all'intercettazione di tipo hook sui sistemi Windows moderni. L'implementazione è principalmente in C e C++, e il repository include articoli di riferimento e un piccolo programma di test per la sperimentazione. È destinato a ricercatori di sicurezza del kernel e anti-cheat che studiano il monitoraggio assistito dall'hardware sulle piattaforme dell'era Windows 10.
