本プロジェクトは、デバイスのデータフローがコンシューマプロセスへ流入する様子を追跡するための、カーネルモード USB / HID モニタリングフレームワークです。IRP および内部 IOCTL または URB パスをフックし、HID レポートを解析し、カスタムデバイス制御コードを通じてキャプチャやマッピング制御を調整するドライバコンポーネントを含みます。実装は主に C で、一部 C++ のプロジェクト足場があり、Windows ドライバ開発と低レベルの入力スタック解析に焦点を当てています。ゲームセキュリティの文脈で USB 入力挙動をリバースエンジニアリングする用途に適しており、HID ベースの攻撃面や検知面の研究にも役立ちます。
