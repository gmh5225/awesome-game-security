Этот проект — фреймворк мониторинга USB и HID в режиме ядра для отслеживания потоков данных устройств в процессы-потребители. Он содержит компоненты драйверов, которые перехватывают пути IRP и внутренние IOCTL или URB, разбирают HID-отчёты и координируют захват или управление отображением через пользовательские коды управления устройством. Реализация в основном на C с некоторой проектной обвязкой на C++, с акцентом на разработку драйверов Windows и низкоуровневый анализ стека ввода. Полезен для реверс-инжиниринга поведения USB-ввода в контексте игровой безопасности, включая исследование поверхностей атаки или обнаружения на основе HID.
