이 프로젝트는 장치 데이터 흐름이 컨슈머 프로세스로 유입되는 경로를 추적하기 위한 커널 모드 USB 및 HID 모니터링 프레임워크입니다. IRP와 내부 IOCTL 또는 URB 경로를 후킹하고, HID 리포트를 파싱하며, 사용자 지정 장치 제어 코드를 통해 캡처 또는 매핑 제어를 조율하는 드라이버 구성 요소를 포함합니다. 구현은 대부분 C이며 일부 C++ 프로젝트 스캐폴딩이 있고, Windows 드라이버 개발과 저수준 입력 스택 분석에 초점을 둡니다. 게임 보안 맥락에서 USB 입력 동작을 리버스 엔지니어링하는 데 유용하며, HID 기반 공격면 또는 탐지면 연구에도 활용할 수 있습니다.
