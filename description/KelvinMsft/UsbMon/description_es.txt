Este proyecto es un framework de monitorización USB y HID en modo kernel para rastrear los flujos de datos de dispositivos hacia procesos consumidores. Contiene componentes de controlador que enganchan rutas IRP e IOCTL internas o URB, analizan informes HID y coordinan la captura o el control de mapeo mediante códigos de control de dispositivo personalizados. La implementación es principalmente en C con algo de andamiaje de proyecto en C++, y se centra en el desarrollo de controladores de Windows y el análisis de bajo nivel de la pila de entrada. Es útil para la ingeniería inversa del comportamiento de entrada USB en contextos de seguridad de juegos, incluidos estudios de superficies de ataque o detección basadas en HID.
