이 프로젝트는 Winlator로, ARM 기반 Android 기기에서 Windows x86/x64 애플리케이션과 게임을 실행하는 Android 앱입니다. x86-to-ARM 바이너리 변환을 위한 Box86/Box64와 Windows API 호환을 위한 Wine을 결합하며, PRoot 기반 Linux 컨테이너, GPU 가속을 위한 Mesa Turnip 또는 VirGL, 터치 컨트롤이 있는 가상 데스크톱 환경을 포함합니다. 모바일 기기에서 Windows 소프트웨어를 실행하려는 Android 사용자와 개발자, 그리고 교차 아키텍처 바이너리 변환을 연구하는 연구자를 대상으로 합니다.
