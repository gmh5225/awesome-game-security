Этот проект — Winlator, приложение для Android, которое запускает приложения и игры Windows x86/x64 на Android-устройствах на базе ARM. Оно сочетает Box86/Box64 для бинарной трансляции x86 в ARM с Wine для совместимости с Windows API и включает Linux-контейнер на основе PRoot, Mesa Turnip или VirGL для GPU-ускорения, а также виртуальную среду рабочего стола с сенсорным управлением. Проект ориентирован на пользователей и разработчиков Android, интересующихся запуском ПО Windows на мобильных устройствах, а также на исследователей кросс-архитектурной бинарной трансляции.
