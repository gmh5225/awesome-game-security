本プロジェクトは Winlator であり、ARM ベースの Android 端末上で Windows の x86/x64 アプリケーションやゲームを実行する Android アプリです。x86 から ARM へのバイナリ変換を行う Box86/Box64 と、Windows API 互換のための Wine を組み合わせ、PRoot ベースの Linux コンテナ、GPU アクセラレーション用の Mesa Turnip または VirGL、タッチ操作対応の仮想デスクトップ環境を備えています。モバイル端末で Windows ソフトウェアを動かしたい Android ユーザーや開発者、およびクロスアーキテクチャのバイナリ変換を研究する研究者を対象としています。
