Ce projet est Winlator, une application Android qui exécute des applications et jeux Windows x86/x64 sur des appareils Android basés sur ARM. Elle combine Box86/Box64 pour la traduction binaire x86-vers-ARM avec Wine pour la compatibilité de l'API Windows, et inclut un conteneur Linux basé sur PRoot, Mesa Turnip ou VirGL pour l'accélération GPU, ainsi qu'un environnement de bureau virtuel avec des commandes tactiles. Elle s'adresse aux utilisateurs et développeurs Android intéressés par l'exécution de logiciels Windows sur appareils mobiles, ainsi qu'aux chercheurs étudiant la traduction binaire multi-architectures.
