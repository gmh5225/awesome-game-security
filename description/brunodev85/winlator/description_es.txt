Este proyecto es Winlator, una aplicación Android que ejecuta aplicaciones y juegos Windows x86/x64 en dispositivos Android basados en ARM. Combina Box86/Box64 para la traducción binaria de x86 a ARM con Wine para la compatibilidad con la API de Windows, e incluye un contenedor Linux basado en PRoot, Mesa Turnip o VirGL para la aceleración por GPU, y un entorno de escritorio virtual con controles táctiles. Está dirigido a usuarios y desarrolladores de Android interesados en ejecutar software de Windows en dispositivos móviles y a investigadores que estudian la traducción binaria entre arquitecturas.
