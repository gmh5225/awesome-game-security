Vanguard Service Monitor è un pacchetto di utilità Windows composto da due strumenti a riga di comando per gestire il servizio anti-cheat a livello kernel di Riot Vanguard (vgk). Scritto in C++ per Visual Studio, include vgkChecker per segnalare se Vanguard è in esecuzione e impostato per l’avvio automatico, e noVanguard per attivare o disattivare la configurazione di avvio automatico del servizio. Gli strumenti usano le API del Windows Service Control Manager, logging a console colorato e controlli dei privilegi di amministratore, con prompt opzionali di riavvio affinché le modifiche abbiano effetto. Sono rivolti a giocatori e amministratori di sistema che vogliono controllare quando Vanguard viene caricato, riducendo il carico inutile a livello kernel quando i giochi Riot non sono in uso.
