Vanguard Service Monitor 是一个 Windows 实用工具包，包含两个用于管理 Riot Vanguard 内核级反作弊服务（vgk）的命令行工具。使用 C++ 为 Visual Studio 编写，其中 vgkChecker 用于报告 Vanguard 是否正在运行以及是否设置为自动启动，noVanguard 用于切换该服务的自动启动配置。这些工具使用 Windows Service Control Manager API、彩色控制台日志和管理员权限检查，并可选择提示重启以使更改生效。面向希望控制 Vanguard 加载时机的玩家和系统管理员，以便在未使用 Riot 游戏时减少不必要的内核级开销。
