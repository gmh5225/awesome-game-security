Vanguard Service Monitor は、Riot Vanguard のカーネル級アンチチートサービス（vgk）を管理するための 2 つのコマンドラインツールからなる Windows ユーティリティパッケージです。Visual Studio 向けに C++ で書かれており、vgkChecker は Vanguard が実行中か自動起動に設定されているかを報告し、noVanguard はサービスの自動起動設定を切り替えます。これらのツールは Windows Service Control Manager API、色付きコンソールログ、管理者権限チェックを利用し、変更を反映するための再起動を任意で促します。Riot ゲームを使っていないときに不要なカーネルレベルの負荷を抑えたいプレイヤーやシステム管理者向けです。
