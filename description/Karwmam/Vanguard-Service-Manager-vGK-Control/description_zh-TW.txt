Vanguard Service Monitor 是一套 Windows 实用工具套件，包含兩個用於管理 Riot Vanguard 核心層反作弊服務（vgk）的命令列工具。以 C++ 為 Visual Studio 撰寫，其中 vgkChecker 可回報 Vanguard 是否正在執行以及是否設定為自動啟動，noVanguard 則可切換該服務的自動啟動設定。這些工具使用 Windows Service Control Manager API、彩色主控台日誌與系統管理員權限檢查，並可選擇提示重新啟動以使變更生效。面向希望控制 Vanguard 載入時機的玩家與系統管理員，以便在未使用 Riot 遊戲時減少不必要的核心層負擔。
