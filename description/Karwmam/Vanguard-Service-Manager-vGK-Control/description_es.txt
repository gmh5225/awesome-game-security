Vanguard Service Monitor es un paquete de utilidades para Windows con dos herramientas de línea de comandos para gestionar el servicio anti-trampas de núcleo de Riot Vanguard (vgk). Escrito en C++ para Visual Studio, incluye vgkChecker para informar si Vanguard está en ejecución y configurado para el inicio automático, y noVanguard para alternar la configuración de inicio automático del servicio. Las herramientas usan las API del Windows Service Control Manager, registro en consola con colores y comprobaciones de privilegios de administrador, con avisos opcionales de reinicio para que los cambios surtan efecto. Están orientadas a jugadores y administradores de sistemas que quieren controlar cuándo se carga Vanguard, reduciendo la sobrecarga innecesaria a nivel de núcleo cuando no se usan juegos de Riot.
