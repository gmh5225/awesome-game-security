Vanguard Service Monitor — это пакет утилит для Windows из двух инструментов командной строки для управления сервисом античита ядра Riot Vanguard (vgk). Написан на C++ для Visual Studio: vgkChecker сообщает, запущен ли Vanguard и настроен ли он на автозапуск, а noVanguard переключает конфигурацию автоматического запуска сервиса. Инструменты используют API Windows Service Control Manager, цветной вывод в консоль и проверки прав администратора, с необязательными запросами на перезагрузку для применения изменений. Они предназначены для игроков и системных администраторов, которым нужен контроль над загрузкой Vanguard, чтобы снизить лишнюю нагрузку на уровне ядра, когда игры Riot не используются.
