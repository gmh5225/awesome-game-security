Vanguard Service Monitor ist ein Windows-Utility-Paket mit zwei Kommandozeilen-Tools zur Verwaltung des Kernel-Anti-Cheat-Dienstes von Riot Vanguard (vgk). In C++ für Visual Studio geschrieben, umfasst es vgkChecker zur Abfrage, ob Vanguard läuft und für den Autostart konfiguriert ist, sowie noVanguard zum Umschalten der automatischen Startkonfiguration des Dienstes. Die Tools nutzen die Windows Service Control Manager APIs, farbige Konsolenprotokollierung und Administratorrechteprüfungen, mit optionalen Neustart-Hinweisen, damit Änderungen wirksam werden. Sie richten sich an Spieler und Systemadministratoren, die steuern möchten, wann Vanguard geladen wird, um unnötigen Kernel-Overhead zu reduzieren, wenn keine Riot-Spiele genutzt werden.
