Vanguard Service Monitor est un ensemble d’utilitaires Windows composé de deux outils en ligne de commande pour gérer le service anti-triche noyau de Riot Vanguard (vgk). Écrit en C++ pour Visual Studio, il comprend vgkChecker, qui indique si Vanguard est en cours d’exécution et configuré pour démarrer automatiquement, et noVanguard, qui permet d’activer ou de désactiver le démarrage automatique du service. Les outils s’appuient sur les API du Windows Service Control Manager, une journalisation console colorée et des vérifications des privilèges administrateur, avec des invites optionnelles de redémarrage pour appliquer les changements. Ils s’adressent aux joueurs et administrateurs système qui souhaitent contrôler le chargement de Vanguard afin de réduire la charge noyau inutile lorsque les jeux Riot ne sont pas utilisés.
