Vanguard Service Monitor는 Riot Vanguard의 커널 안티치트 서비스(vgk)를 관리하기 위한 두 개의 명령줄 도구로 구성된 Windows 유틸리티 패키지입니다. Visual Studio용 C++로 작성되었으며, vgkChecker는 Vanguard가 실행 중인지와 자동 시작으로 설정되어 있는지 보고하고, noVanguard는 서비스의 자동 시작 구성을 전환합니다. 이 도구들은 Windows Service Control Manager API, 색상 콘솔 로깅, 관리자 권한 검사를 사용하며, 변경 사항을 적용하기 위한 재시작 프롬프트를 선택적으로 제공합니다. Riot 게임을 사용하지 않을 때 불필요한 커널 수준 부하를 줄이고자 Vanguard 로드 시점을 제어하려는 플레이어와 시스템 관리자를 대상으로 합니다.
