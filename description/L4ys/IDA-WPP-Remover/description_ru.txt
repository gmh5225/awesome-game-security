WPP Remover — плагин для IDA Pro, который очищает псевдокод Hex-Rays, убирая шум от вызовов Windows Performance Profiling.
Он реализован на Python и применяет проход оптимизации микрокода, заменяя вызовы WPP_SF* до генерации вывода декомпилятора.
Плагин автоматически ориентирован на анализ Windows PE и может включаться прямо из представления декомпилированного кода.
Он предназначен для исследователей обратной разработки, которым нужны более понятные результаты декомпиляции при анализе вредоносного ПО и игровых бинарников.
