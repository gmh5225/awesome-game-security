WPP Remover est un plugin IDA Pro qui nettoie le pseudocode Hex-Rays en supprimant le bruit des appels Windows Performance Profiling.
Il est implémenté en Python et applique une passe d'optimisation microcode pour remplacer les appels WPP_SF* avant la génération de la sortie du décompilateur.
Le plugin cible automatiquement l'analyse des PE Windows et peut être activé ou désactivé directement depuis la vue décompilée.
Il est conçu pour les reverse engineers qui souhaitent des résultats de décompilation plus clairs lors de l'analyse de malwares et de binaires de jeux.
