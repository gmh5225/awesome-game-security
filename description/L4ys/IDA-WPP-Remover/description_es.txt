WPP Remover es un plugin de IDA Pro que limpia el pseudocódigo de Hex-Rays eliminando el ruido de las llamadas de Windows Performance Profiling.
Está implementado en Python y aplica un paso de optimización de microcódigo para reemplazar las llamadas WPP_SF* antes de generar la salida del descompilador.
El plugin se orienta automáticamente al análisis de PE de Windows y se puede activar o desactivar directamente desde la vista descompilada.
Está pensado para ingenieros inversos que buscan resultados de descompilación más claros al analizar malware y binarios de juegos.
