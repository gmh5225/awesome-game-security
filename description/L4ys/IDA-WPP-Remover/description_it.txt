WPP Remover è un plugin di IDA Pro che pulisce lo pseudocodice Hex-Rays rimuovendo il rumore delle chiamate Windows Performance Profiling.
È implementato in Python e applica un passaggio di ottimizzazione del microcodice per sostituire le chiamate WPP_SF* prima che venga generata l'uscita del decompilatore.
Il plugin si rivolge automaticamente all'analisi dei PE Windows e può essere attivato o disattivato direttamente dalla vista decompilata.
È pensato per i reverse engineer che desiderano risultati di decompilazione più chiari durante l'analisi di malware e binari di giochi.
