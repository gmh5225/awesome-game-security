Controlador de Windows en modo kernel y aplicación GUI en C# que vuelca la memoria de procesos comunicándose con un controlador personalizado (KsDumperDriver.sys) mediante IOCTL, usando KDU para cargar un controlador vulnerable y eludir Driver Signature Enforcement. Admite el análisis de encabezados PE32/PE64, la enumeración de procesos mediante API NT no documentadas e incluye parches de registro anti-blocklist para Microsoft Vulnerable Driver Blocklist.
