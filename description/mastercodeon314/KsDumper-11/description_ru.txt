Драйвер Windows в режиме ядра и GUI-приложение на C#, которое дампит память процессов, обмениваясь данными с пользовательским драйвером (KsDumperDriver.sys) через IOCTL и используя KDU для загрузки уязвимого драйвера в обход Driver Signature Enforcement. Поддерживает разбор заголовков PE32/PE64, перечисление процессов через недокументированные NT API и включает анти-блоклист патчи реестра для Microsoft Vulnerable Driver Blocklist.
