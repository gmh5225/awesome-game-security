Pilote Windows en mode noyau et application GUI C# qui dump la mémoire des processus en communiquant avec un pilote personnalisé (KsDumperDriver.sys) via IOCTL, en utilisant KDU pour charger un pilote vulnérable afin de contourner le Driver Signature Enforcement. Prend en charge l'analyse des en-têtes PE32/PE64, l'énumération des processus via des API NT non documentées, et inclut des correctifs de registre anti-blocklist pour le Microsoft Vulnerable Driver Blocklist.
