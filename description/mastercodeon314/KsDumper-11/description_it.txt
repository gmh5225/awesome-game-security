Driver Windows in modalità kernel e applicazione GUI C# che esegue il dump della memoria dei processi comunicando con un driver personalizzato (KsDumperDriver.sys) tramite IOCTL, usando KDU per caricare un driver vulnerabile e aggirare Driver Signature Enforcement. Supporta il parsing degli header PE32/PE64, l'enumerazione dei processi tramite API NT non documentate e include patch del registro anti-blocklist per Microsoft Vulnerable Driver Blocklist.
