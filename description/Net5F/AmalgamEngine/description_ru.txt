Этот проект — движок на C++, предназначенный для создания изометрических онлайн-виртуальных миров на основе спрайтов.
Он предоставляет основы клиента и сервера, сетевое взаимодействие, чат, системы аккаунтов и инструменты живого редактирования в мире.
Кодовая база использует CMake и ориентирована на масштабируемое многопользовательское поведение при возможности развёртывания на относительно скромном оборудовании.
Он рассчитан на разработчиков игр, которым нужна готовая техническая база для устойчивых социальных миров, а не создание всего с нуля.
