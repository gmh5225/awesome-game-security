Dieses Projekt ist eine C++-Engine zum Aufbau isometrischer, sprite-basierter Online-Virtualwelten.
Sie stellt Client- und Server-Grundlagen, Networking, Chat, Kontensysteme sowie Live-Werkzeuge zur Bearbeitung in der Welt bereit.
Die Codebasis nutzt CMake und legt den Fokus auf skalierbares Mehrspielerverhalten, bleibt aber auf relativ bescheidener Hardware einsetzbar.
Sie richtet sich an Spieleentwickler, die eine fertige technische Basis für persistente soziale Welten wollen, statt bei null anzufangen.
