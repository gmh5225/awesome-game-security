Ce projet est un moteur C++ pour créer des mondes virtuels en ligne isométriques basés sur des sprites.
Il fournit les fondations client et serveur, le réseau, le chat, les systèmes de comptes et des outils d’édition en direct dans le monde.
Le code utilise CMake et vise un comportement multijoueur évolutif tout en restant déployable sur du matériel relativement modeste.
Il s’adresse aux développeurs de jeux qui veulent une base technique prête pour des mondes sociaux persistants, plutôt que de partir de zéro.
