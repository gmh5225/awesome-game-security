Этот проект — DLLirant, инструмент для автоматизированного обнаружения уязвимостей DLL hijacking в Windows. Он проверяет исполняемые файлы на hijacking порядка поиска DLL, генерируя прокси-DLL, размещая их в каталогах-кандидатах и отслеживая, загружает ли их целевое приложение. Инструмент автоматизирует выявление отсутствующих зависимостей DLL и небезопасных путей поиска. Предназначен для пентестеров и аудиторов безопасности, ищущих возможности DLL hijacking для повышения привилегий или закрепления в системе.
