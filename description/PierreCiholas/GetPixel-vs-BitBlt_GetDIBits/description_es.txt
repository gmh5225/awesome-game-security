Este proyecto es una utilidad C++ de benchmark y captura que compara GetPixel con BitBlt más GetDIBits para leer píxeles de pantalla o ventana. Implementa una clase de captura Win32 con modos de captura conmutables, manejo de búfer de fotogramas y exportación de bitmaps para pruebas. El código demuestra que BitBlt con GetDIBits es mucho más rápido que llamadas GetPixel por píxel en tamaños de captura prácticos. Resulta especialmente útil para herramientas de juegos y flujos de investigación de seguridad que necesitan captura externa rápida de fotogramas.
