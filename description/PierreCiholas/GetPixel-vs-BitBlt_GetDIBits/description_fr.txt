Ce projet est un utilitaire C++ de benchmark et de capture qui compare GetPixel à BitBlt combiné avec GetDIBits pour lire les pixels d'écran ou de fenêtre. Il implémente une classe de capture Win32 avec des modes de capture commutables, la gestion du tampon d'images et l'export de bitmaps pour les tests. Le code montre que BitBlt avec GetDIBits est nettement plus rapide que des appels GetPixel pixel par pixel pour des tailles de capture réalistes. Il est surtout utile pour les outils de jeu et les workflows de recherche en sécurité qui nécessitent une capture de trames externe rapide.
