Este proyecto es un gestor de biblioteca de juegos y lanzador de código abierto para Windows que unifica títulos de muchas plataformas. Está construido principalmente con C# y XAML sobre .NET, con una arquitectura de plugins para integraciones de biblioteca, temas y extensiones de automatización. La aplicación admite flujos de emulación y gestión local de metadatos, manteniendo los datos del usuario en la máquina local. Su caso de uso principal es ofrecer a jugadores y desarrolladores una interfaz única para organizar y lanzar colecciones de juegos diversas.
