Этот проект — открытый менеджер игровой библиотеки и лаунчер для Windows, объединяющий игры с множества платформ. Он в основном написан на C# и XAML поверх .NET и имеет архитектуру плагинов для интеграции библиотек, тем и расширений автоматизации. Приложение поддерживает эмуляционные сценарии и локальное управление метаданными, сохраняя пользовательские данные на локальной машине. Основной сценарий использования — единый интерфейс для игроков и разработчиков, позволяющий организовывать и запускать разнообразные игровые коллекции.
