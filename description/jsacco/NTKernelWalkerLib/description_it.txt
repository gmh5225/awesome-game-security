Questo progetto si concentra sulla risoluzione in user-mode di simboli e offset di strutture di ntoskrnl tramite dbghelp, insieme alla scansione di gadget nelle sezioni eseguibili (ad es. primitive ROP brevi).
Usa dbghelp per ottenere le RVA dei simboli esportati da ntoskrnl.exe e un image mapper che può scansionare le sezioni eseguibili per trovare gadget ROP come “pop rcx ; ret” o “jmp rax”.
È soprattutto utile a ricercatori di game security e reverse engineer che studiano tecniche offensive nell’ambito di cheat / esplorazione del kernel Windows.
