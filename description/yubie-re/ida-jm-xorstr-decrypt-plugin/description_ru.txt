Этот проект посвящён попыткам расшифровки JM Xorstr в некоторых x64-бинарниках.
Он в основном написан на Python и сосредоточен на разработке плагинов.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / плагинов IDA.
