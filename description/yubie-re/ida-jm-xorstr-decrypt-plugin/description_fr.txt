Ce projet se concentre sur les tentatives de déchiffrement de JM Xorstr dans certains binaires x64.
Il est principalement écrit en Python et centré sur le développement de plugins.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
