Ce projet se concentre sur un simple contournement EQU8 en mode utilisateur qui utilise le registre pour trouver le chemin du pilote et fermer le handle IOCTL.
Les pull requests sont les bienvenues.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine cheat / explore anticheat system:equ8.
