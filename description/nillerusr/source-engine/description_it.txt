Questo progetto è un albero open del Source Engine basato sul codice Source di Valve dell'era Team Fortress 2, mantenuto con build CI e lavoro multipiattaforma.
Fornisce i moduli classici client/server/engine Source per studiare o portare giochi Source al di fuori dell'albero chiuso originale.
È soprattutto utile per sviluppatori di giochi, programmatori di engine e reverse engineer che lavorano con titoli Source Engine.
