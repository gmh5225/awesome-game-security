Этот проект — открытое дерево Source Engine на основе кода Valve Source эпохи Team Fortress 2, поддерживаемое с помощью CI-сборок и кроссплатформенной работы.
Он предоставляет классические модули клиента/сервера/движка Source для изучения или портирования игр Source вне исходного закрытого дерева.
В основном полезен разработчикам игр, программистам движков и специалистам по реверс-инжинирингу, работающим с играми на Source Engine.
