Это компактный драйвер ядра Windows, который выводит растровое изображение на экран после BSOD. Он написан на C и интегрируется с загрузочными видеоподпрограммами через модифицированный интерфейс BOOTVID. Проект демонстрирует низкоуровневый графический вывод и обработку отображения при сбое в пространстве ядра на минимальной кодовой базе. В основном полезен разработчикам драйверов и исследователям, изучающим внутренности Windows и поведение отрисовки на этапе загрузки.
