Questo è un driver kernel Windows compatto che visualizza una bitmap sullo schermo dopo un BSOD. È scritto in C e si integra con le routine video di boot tramite un'interfaccia BOOTVID modificata. Il progetto dimostra l'output grafico a basso livello e la gestione del display in caso di crash nello spazio kernel, con una codebase minimale. È soprattutto utile per sviluppatori di driver e reverse engineer che esplorano gli internals di Windows e il rendering in fase di avvio.
