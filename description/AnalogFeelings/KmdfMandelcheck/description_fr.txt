Il s'agit d'un pilote noyau Windows compact qui affiche un bitmap à l'écran après un BSOD. Écrit en C, il s'intègre aux routines vidéo de démarrage via une interface BOOTVID modifiée. Le projet illustre la sortie graphique bas niveau et la gestion d'affichage en cas de plantage dans l'espace noyau, avec une base de code minimale. Il s'adresse surtout aux développeurs de pilotes et aux reverse engineers qui explorent les internals Windows et le rendu au démarrage.
