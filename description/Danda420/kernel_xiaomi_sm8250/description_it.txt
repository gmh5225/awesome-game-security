Questo progetto è un codice sorgente di kernel Android personalizzato per dispositivi Xiaomi basati su SM8250, inclusi POCO F3 e POCO F4. Include packaging flashabile AnyKernel3, defconfig specifici per dispositivo, integrazione GitLab CI e definizioni GKI ABI per piattaforme Qualcomm. È principalmente utile per sviluppatori di kernel Android e ricercatori di sicurezza mobile che lavorano con build di kernel personalizzati su dispositivi Qualcomm Snapdragon 870/888.
