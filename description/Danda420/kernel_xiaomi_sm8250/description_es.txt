Este proyecto es un código fuente de kernel Android personalizado para dispositivos Xiaomi basados en SM8250, incluidos el POCO F3 y el POCO F4. Incluye empaquetado flashable con AnyKernel3, defconfigs específicos por dispositivo, integración con GitLab CI y definiciones GKI ABI para plataformas Qualcomm. Resulta especialmente útil para desarrolladores de kernels Android e investigadores de seguridad móvil que trabajan con compilaciones de kernels personalizados en dispositivos Qualcomm Snapdragon 870/888.
