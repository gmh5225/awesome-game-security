Ce projet est un code source de noyau Android personnalisé pour les appareils Xiaomi basés sur SM8250, notamment le POCO F3 et le POCO F4. Il inclut un packaging flashable AnyKernel3, des defconfigs spécifiques aux appareils, une intégration GitLab CI et des définitions GKI ABI pour les plateformes Qualcomm. Il s’adresse principalement aux développeurs de noyaux Android et aux chercheurs en sécurité mobile travaillant sur des builds de noyaux personnalisés pour les appareils Qualcomm Snapdragon 870/888.
