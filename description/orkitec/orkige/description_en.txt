Orkige is a custom C++ game engine originally developed around 2009–2012 and now being revived and modernized to ship mobile games with desktop as the development targets. It is built with CMake and vcpkg, uses OGRE (classic and Ogre-Next) with SDL3 for rendering and windowing, and supports Lua-first live scripting with hot-reload during play. The stack includes an ImGui-based editor, cross-platform players for macOS, Windows, iOS, Android, and WebAssembly, plus systems for scenes, materials, particles, character and vector animation, GUI, localisation, and save data. Primary use case is building and iterating on 2D/3D games with a C++ core and Lua gameplay code rather than reverse engineering or anti-cheat tooling.
