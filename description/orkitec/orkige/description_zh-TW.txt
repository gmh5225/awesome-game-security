Orkige 是一款自訂的 C++ 遊戲引擎，約於 2009–2012 年間開發，現正被重新啟用並現代化，以便以桌面端為開發目標發行手遊。它以 CMake 與 vcpkg 建置，使用 OGRE（經典版與 Ogre-Next）搭配 SDL3 進行渲染與視窗管理，並支援以 Lua 優先的即時腳本及遊玩過程中的熱重載。技術棧包含基於 ImGui 的編輯器，面向 macOS、Windows、iOS、Android 與 WebAssembly 的跨平台播放器，以及場景、材質、粒子、角色與向量動畫、GUI、在地化與存檔等系統。主要用途是以 C++ 核心與 Lua 玩法程式碼建構並迭代 2D/3D 遊戲，而非逆向工程或反作弊工具。
