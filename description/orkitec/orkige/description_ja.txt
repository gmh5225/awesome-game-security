Orkige は、およそ 2009–2012 年頃に開発されたカスタム C++ ゲームエンジンで、現在はデスクトップを開発ターゲットとしてモバイルゲームを出荷できるよう再興・近代化されています。CMake と vcpkg で構築され、レンダリングとウィンドウ管理に OGRE（クラシックおよび Ogre-Next）と SDL3 を用い、Lua 優先のライブスクリプティングとプレイ中のホットリロードをサポートします。スタックには ImGui ベースのエディタ、macOS・Windows・iOS・Android・WebAssembly 向けのクロスプラットフォームプレイヤー、さらにシーン、マテリアル、パーティクル、キャラクター／ベクターアニメーション、GUI、ローカライズ、セーブデータなどのシステムが含まれます。主な用途は、リバースエンジニアリングやアンチチートツールではなく、C++ コアと Lua のゲームプレイコードで 2D/3D ゲームを構築・反復することです。
