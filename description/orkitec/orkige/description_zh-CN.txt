Orkige 是一款定制的 C++ 游戏引擎，约于 2009–2012 年间开发，现正被重新启用并现代化，以便以桌面端为开发目标发布手游。它基于 CMake 与 vcpkg 构建，使用 OGRE（经典版与 Ogre-Next）配合 SDL3 进行渲染与窗口管理，并支持以 Lua 优先的实况脚本及游玩过程中的热重载。技术栈包含基于 ImGui 的编辑器，面向 macOS、Windows、iOS、Android 与 WebAssembly 的跨平台播放器，以及场景、材质、粒子、角色与矢量动画、GUI、本地化和存档等系统。主要用途是以 C++ 核心与 Lua 玩法代码构建并迭代 2D/3D 游戏，而非逆向工程或反作弊工具。
