이 프로젝트는 취약한 서명된 드라이버(BYOVD)를 악용해 Windows에서 EDR 및 ETW 기반 탐지를 우회하는 C 도구입니다. 커널 콜백 루틴을 패치하고, 프로세스·스레드 알림 콜백을 제거하며, ETW TI 프로바이더 피드를 비활성화하고, 사용자 모드에서 ntdll 시스템 호출 스텁을 언후킹하여 보안 제품을 무력화합니다. 여러 Windows 빌드에 대한 오프셋 해석을 자동화하고, 드라이버 로드와 정리를 처리하며, 데모용 자격 증명 덤프 페이로드를 포함합니다. EDR 우회 기법과 커널 콜백 조작을 연구하는 레드팀 운영자와 보안 연구자를 대상으로 합니다.
