本プロジェクトは、脆弱な署名済みドライバ（BYOVD）を悪用し、Windows 上で EDR および ETW ベースの検知を回避する C 言語ツールです。カーネルコールバックをパッチし、プロセス／スレッド通知コールバックを除去し、ETW TI プロバイダのフィードを無効化し、ユーザーモードで ntdll のシステムコールスタブをアンフックすることで、セキュリティ製品を無力化します。複数の Windows ビルド向けのオフセット解決を自動化し、ドライバの読み込みとクリーンアップを処理し、デモ用の資格情報ダンプペイロードも同梱しています。EDR 回避手法やカーネルコールバック操作を研究するレッドチームやセキュリティ研究者向けです。
