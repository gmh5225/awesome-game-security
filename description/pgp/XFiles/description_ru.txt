Этот проект посвящён файловому менеджеру для (рутованного) Android. Готовые APK доступны в разделе Releases этого репозитория. Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / android-файловых менеджеров.
