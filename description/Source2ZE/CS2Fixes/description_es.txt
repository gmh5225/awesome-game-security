Este proyecto es un plugin Metamod en C++ que añade correcciones de jugabilidad y funciones del lado del servidor para Counter-Strike 2, con un fuerte enfoque en las comunidades de zombie escape. Incluye lógica de servidor basada en hooks, sistemas de configuración, herramientas de administración y assets de partículas o sonido incluidos con el plugin. La base de código también sirve como referencia práctica para desarrolladores de mods de Source 2 que necesitan patrones comunes de integración y código boilerplate. Su caso de uso principal es ejecutar y ampliar servidores CS2 comunitarios con funcionalidad estable y configurable.
