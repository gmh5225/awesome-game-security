Questo progetto è un plugin Metamod in C++ che aggiunge correzioni di gameplay e funzionalità lato server per Counter-Strike 2, con un forte focus sulle community di zombie escape. Include logica server basata su hook, sistemi di configurazione, strumenti di amministrazione e asset di particelle o suoni inclusi nel plugin. La codebase funge anche da riferimento pratico per gli sviluppatori di mod Source 2 che necessitano di pattern comuni di integrazione e boilerplate. Il caso d’uso principale è gestire ed estendere server CS2 della community con funzionalità stabili e configurabili.
