Ce projet est une preuve de concept pour le noyau Windows qui dissimule du code exécutable dans des code caves au sein de pilotes noyau légitimes. Il analyse les pilotes chargés pour trouver des régions de padding inutilisées (code caves) dans leurs sections .text, y copie du shellcode, puis l'exécute dans le contexte d'un module de pilote de confiance. Cela évite d'allouer de nouvelle mémoire noyau exécutable, facilement détectable par les scans anti-cheat. Il s'adresse aux chercheurs en sécurité noyau qui étudient l'abus des code caves et les techniques d'exécution furtive de code en mode noyau.
