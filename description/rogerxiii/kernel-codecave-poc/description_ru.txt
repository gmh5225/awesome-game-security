Этот проект — proof of concept для ядра Windows, который скрывает исполняемый код в code cave внутри легитимных драйверов ядра. Он сканирует загруженные драйверы в поисках неиспользуемых областей заполнения (code cave) в секциях .text, копирует туда shellcode и выполняет его в контексте доверенного модуля драйвера. Так удаётся избежать выделения новой исполняемой памяти ядра, которую могли бы обнаружить сканы античита. Проект предназначен для исследователей безопасности ядра, изучающих злоупотребление code cave и скрытные техники выполнения кода в режиме ядра.
