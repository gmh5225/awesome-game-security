Гибрид отладчика и эмулятора на базе Unicorn Engine и Capstone, который загружает PE-бинарники, эмулирует выполнение x86/x64 с перехватом API и ведёт подробный журнал на уровне инструкций. Разрешает импорты через dbghelp, отображает секции PE в адресное пространство эмулятора и перехватывает вызовы Windows API для имитации поведения ОС при анализе DRM-защищённых или обфусцированных исполняемых файлов.
