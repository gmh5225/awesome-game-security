GPU_ShellCode è una proof-of-concept che conserva i dati attivi del payload nella memoria GPU NVIDIA anziché nella normale memoria di processo. Usa C/C++ su Windows con le API CUDA e MinHook per intercettare funzioni come Sleep e VirtualAlloc, quindi ripristina l’esecuzione tramite un vectored exception handler quando serve. Il flusso copia un payload preparato nella memoria GPU nei periodi di idle e ripopola le pagine eseguibili al risveglio, dimostrando tecniche di nascondimento della memoria. Il progetto è utile soprattutto per la ricerca di sicurezza offensiva a basso livello e per studiare l’evasione delle ispezioni di memoria di anti-cheat o anti-malware.
