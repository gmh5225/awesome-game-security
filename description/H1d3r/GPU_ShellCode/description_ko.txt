GPU_ShellCode는 활성 페이로드 데이터를 일반 프로세스 메모리가 아닌 NVIDIA GPU 메모리에 저장하는 개념 증명입니다. Windows에서 C/C++로 CUDA API와 MinHook을 사용해 Sleep, VirtualAlloc 같은 함수를 가로채고, 필요할 때 벡터 예외 처리기를 통해 실행을 복원합니다. 유휴 구간에 스테이징된 페이로드를 GPU 메모리로 복사하고 깨울 때 실행 가능 페이지를 다시 채우는 흐름으로 메모리 은닉 기법을 보여줍니다. 이 프로젝트는 주로 저수준 공격 보안 연구와 안티치트·안티맬웨어 메모리 검사 우회 학습에 유용합니다.
