GPU_ShellCode est une preuve de concept qui stocke les données de payload actives dans la mémoire GPU NVIDIA plutôt que dans la mémoire de processus classique. Il utilise C/C++ sous Windows avec les API CUDA et MinHook pour intercepter des fonctions comme Sleep et VirtualAlloc, puis restaure l’exécution via un gestionnaire d’exceptions vectorisées lorsque nécessaire. Le flux copie un payload préparé vers la mémoire GPU pendant les périodes d’inactivité et réalimente les pages exécutables au réveil, illustrant des techniques de dissimulation mémoire. Ce projet sert surtout à la recherche offensive bas niveau et à l’étude de l’évasion des inspections mémoire anti-cheat ou anti-malware.
