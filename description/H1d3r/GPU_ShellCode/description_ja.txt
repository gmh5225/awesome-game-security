GPU_ShellCode は、アクティブなペイロードデータを通常のプロセスメモリではなく NVIDIA GPU メモリに保持する概念実証です。Windows 上で C/C++ を用い、CUDA API と MinHook で Sleep や VirtualAlloc などの関数をフックし、必要に応じてベクタ例外ハンドラで実行を復帰させます。アイドル時にステージングしたペイロードを GPU メモリへコピーし、復帰時に実行可能ページを再配置することで、メモリ隠蔽手法を示します。本プロジェクトは主に低レイヤの攻撃的セキュリティ研究や、アンチチート／アンチマルウェアによるメモリ検査回避の学習に役立ちます。
