GPU_ShellCode — это proof-of-concept, который хранит активные данные payload в памяти NVIDIA GPU, а не в обычной памяти процесса. На Windows он использует C/C++ с CUDA API и MinHook для перехвата функций вроде Sleep и VirtualAlloc, а при необходимости восстанавливает выполнение через vectored exception handler. В простое подготовленный payload копируется в память GPU, а при «пробуждении» исполняемые страницы снова заполняются, демонстрируя техники сокрытия памяти. Проект в основном полезен для низкоуровневых offensive security исследований и изучения обхода проверок памяти античитов и антималвари.
