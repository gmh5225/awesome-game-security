GPU_ShellCode es una prueba de concepto que almacena los datos activos del payload en la memoria de la GPU NVIDIA en lugar de en la memoria de proceso habitual. Usa C/C++ en Windows con las API de CUDA y MinHook para interceptar funciones como Sleep y VirtualAlloc, y restaura la ejecución mediante un manejador de excepciones vectorizadas cuando hace falta. El flujo copia un payload preparado a la memoria de la GPU en periodos de inactividad y vuelve a rellenar las páginas ejecutables al despertar, demostrando técnicas de ocultación de memoria. Este proyecto resulta útil sobre todo para investigación ofensiva de bajo nivel y para estudiar la evasión de inspecciones de memoria de anti-cheat o anti-malware.
