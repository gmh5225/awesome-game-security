Этот проект — Pyda, фреймворк динамического анализа бинарных файлов на Python, построенный на пользовательской эмуляции QEMU. Он позволяет запускать и инструментировать Linux-бинарники с помощью Python-колбэков для перехвата функций, мониторинга обращений к памяти, трассировки системных вызовов и анализа на уровне инструкций без необходимости нативного выполнения. Фреймворк предоставляет скриптуемый интерфейс для исследования бинарников и поиска уязвимостей. Он предназначен для исследователей безопасности и специалистов по реверс-инжинирингу, предпочитающих рабочие процессы анализа бинарников на Python.
