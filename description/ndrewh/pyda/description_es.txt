Este proyecto es Pyda, un framework de análisis dinámico de binarios basado en Python y construido sobre la emulación en modo usuario de QEMU. Permite ejecutar e instrumentar binarios de Linux con callbacks de Python para el enganche de funciones, la monitorización de accesos a memoria, el rastreo de llamadas al sistema y el análisis a nivel de instrucción, sin necesidad de ejecución nativa. El framework ofrece una interfaz programable para la exploración de binarios y la investigación de vulnerabilidades. Está dirigido a investigadores de seguridad e ingenieros de ingeniería inversa que prefieren flujos de trabajo de análisis de binarios basados en Python.
