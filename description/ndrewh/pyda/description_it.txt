Questo progetto è Pyda, un framework di analisi dinamica dei binari basato su Python e costruito sull'emulazione in modalità utente di QEMU. Consente di eseguire e strumentare binari Linux con callback Python per il function hooking, il monitoraggio degli accessi in memoria, il tracing delle system call e l'analisi a livello di istruzione, senza richiedere l'esecuzione nativa. Il framework fornisce un'interfaccia scriptabile per l'esplorazione dei binari e la ricerca di vulnerabilità. È rivolto a ricercatori di sicurezza e reverse engineer che preferiscono flussi di lavoro di analisi binaria basati su Python.
