Questo progetto è uno strumento di raccolta credenziali post-exploitation per Windows, scritto in Go. Estrae password salvate, cookie, segnalibri e cronologia dai principali browser (Chrome, Firefox, Edge), raccoglie le password Wi-Fi, legge i dati delle applicazioni di chat e recupera altre credenziali memorizzate in locale. Lo strumento si compila come un unico binario statico per un facile deployment. È rivolto a penetration tester e operatori red team che effettuano la raccolta di credenziali durante assessment di sicurezza autorizzati.
