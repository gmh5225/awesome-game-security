Этот проект — инструмент сбора учётных данных для постэксплуатации в Windows на Go. Он извлекает сохранённые пароли, cookies, закладки и историю из основных браузеров (Chrome, Firefox, Edge), собирает пароли Wi-Fi, читает данные чат-приложений и другие локально хранимые учётные данные. Инструмент компилируется в один статический бинарный файл, что упрощает развёртывание. Предназначен для пентестеров и операторов red team, выполняющих сбор учётных данных в рамках авторизованных проверок безопасности.
