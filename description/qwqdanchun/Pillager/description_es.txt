Este proyecto es una herramienta de recolección de credenciales post-explotación para Windows, escrita en Go. Extrae contraseñas guardadas, cookies, marcadores e historial de los navegadores principales (Chrome, Firefox, Edge), recopila contraseñas Wi-Fi, lee datos de aplicaciones de mensajería y reúne otras credenciales almacenadas localmente. La herramienta se compila como un único binario estático para facilitar el despliegue. Está orientada a pentesters y operadores de red team que realizan recolección de credenciales en evaluaciones de seguridad autorizadas.
