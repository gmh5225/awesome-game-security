Dieses Projekt konzentriert sich auf die Erkennung von durch Themida/WinLicense und Code Virtualizer obfuskierten Code-Stellen. Das Plugin wurde auf x86- und x86_64-Executables getestet, die mit Oreans-Produkten bis Version 3.1.9 geschützt sind. Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat-/Binary-Ninja-Plugins untersuchen.
