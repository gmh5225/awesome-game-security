Este proyecto se centra en detectar las ubicaciones de código ofuscado de Themida/WinLicense y Code Virtualizer. El plugin se ha probado en ejecutables x86 y x86_64 protegidos con productos de Oreans hasta la versión 3.1.9. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de Binary Ninja.
