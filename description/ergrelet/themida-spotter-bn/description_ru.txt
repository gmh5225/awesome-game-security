Этот проект ориентирован на обнаружение расположений обфусцированного кода Themida/WinLicense и Code Virtualizer. Плагин протестирован на исполняемых файлах x86 и x86_64, защищённых продуктами Oreans вплоть до версии 3.1.9. Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / плагинов Binary Ninja.
