Questo progetto si concentra sul rilevamento delle posizioni del codice offuscato di Themida/WinLicense e Code Virtualizer. Il plugin è stato testato su eseguibili x86 e x86_64 protetti con i prodotti di Oreans fino alla versione 3.1.9. È principalmente utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / plugin di Binary Ninja.
