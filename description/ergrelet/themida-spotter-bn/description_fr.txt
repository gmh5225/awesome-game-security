Ce projet se concentre sur la détection des emplacements de code obfusqué de Themida/WinLicense et Code Virtualizer. Le plugin a été testé sur des exécutables x86 et x86_64 protégés par les produits d'Oreans jusqu'à la version 3.1.9. Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / plugins Binary Ninja.
