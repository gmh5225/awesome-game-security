Unlicense는 Themida 및 WinLicense 2.x·3.x로 보호된 Windows 실행 파일을 동적으로 언팩하는 Python 3 도구입니다. 대상 프로세스를 계측(특히 Frida) 아래에서 실행한 뒤, 원래 진입점을 복구하고 자주 난독화된 가져오기 테이블을 재구성한 다음 언팩된 PE를 덤프합니다. 32비트·64비트 네이티브 PE 실행 파일과 DLL, 그리고 .NET 어셈블리 EXE를 지원하며, LIEF와 Capstone 같은 PE 파싱·분석 라이브러리에 의존합니다. 통제된 환경에서 Themida/WinLicense로 보호된 바이너리를 분석하거나 추가로 리버스해야 하는 리버스 엔지니어와 보안 연구자를 대상으로 합니다.
