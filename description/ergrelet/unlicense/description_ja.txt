Unlicense は、Themida および WinLicense 2.x / 3.x で保護された Windows 実行ファイルを動的にアンパックする Python 3 ツールです。対象プロセスを計装（主に Frida）下で実行し、元のエントリポイントを復元してしばしば難読化されたインポートテーブルを再構築したうえで、アンパック済み PE をダンプします。32 ビットおよび 64 ビットのネイティブ PE 実行ファイルと DLL、ならびに .NET アセンブリ EXE をサポートし、LIEF や Capstone などの PE 解析・分析ライブラリを利用します。制御された環境で Themida/WinLicense 保護バイナリを解析・さらにリバースしたいリバースエンジニアやセキュリティ研究者向けです。
