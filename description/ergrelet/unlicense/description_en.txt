Unlicense is a Python 3 tool that dynamically unpacks Windows executables protected with Themida and WinLicense 2.x and 3.x. It runs the target process under instrumentation (notably Frida), then dumps the unpacked PE after recovering the original entry point and reconstructing the often obfuscated import table. The tool supports 32-bit and 64-bit native PE executables and DLLs as well as .NET assembly EXEs, and relies on PE parsing and analysis libraries such as LIEF and Capstone. It is aimed at reverse engineers and security researchers who need to analyze or further reverse Themida/WinLicense-protected binaries in a controlled environment.
