Este proyecto es una herramienta en Python 3 para desofuscar estáticamente funciones protegidas por la ofuscación basada en mutación de Themida, WinLicense y Code Virtualizer 3.x. La herramienta se ha probado con Themida hasta la versión 3.1.9. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito del cheat / fix de Themida.
