Dieses Projekt ist ein Python-3-Tool zur statischen Entschleierung von Funktionen, die durch die mutationsbasierte Obfuskation von Themida, WinLicense und Code Virtualizer 3.x geschützt sind. Das Tool wurde mit Themida bis Version 3.1.9 getestet. Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Themida-Fix untersuchen.
