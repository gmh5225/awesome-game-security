Этот проект — инструмент на Python 3 для статической деобфускации функций, защищённых мутационной обфускацией Themida, WinLicense и Code Virtualizer 3.x. Инструмент протестирован на Themida вплоть до версии 3.1.9. Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / исправления Themida.
