Questo progetto è uno strumento in Python 3 per deoffuscare staticamente funzioni protette dall'offuscamento basato su mutazione di Themida, WinLicense e Code Virtualizer 3.x. Lo strumento è stato testato su Themida fino alla versione 3.1.9. È soprattutto utile a ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito cheat / fix Themida.
