WinDiff — веб-инструмент для просмотра и сравнения символов, типов и системных вызовов между разными версиями бинарных файлов Microsoft Windows. CLI на Rust загружает PE-файлы через Winbindex, получает соответствующие PDB с сервера символов Microsoft и извлекает экспортируемые символы, восстановленные типы, модули и таблицы syscall в JSON-базы, сжатые gzip. Frontend на Next.js/React загружает эти статические базы и предоставляет представления Browse и Diff для попарного сравнения между сборками ОС. Инструмент предназначен для реверс-инженеров и исследователей безопасности, анализирующих внутренности Windows, поверхности EDR/античита и изменения ядра или user-mode бинарников от версии к версии.
