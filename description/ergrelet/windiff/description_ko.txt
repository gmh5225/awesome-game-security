WinDiff는 서로 다른 버전의 Microsoft Windows 바이너리에서 심볼, 타입, 시스템 콜을 탐색하고 비교하기 위한 웹 기반 도구입니다. Rust CLI는 Winbindex를 통해 PE 파일을 다운로드하고, Microsoft 심볼 서버에서 일치하는 PDB를 가져온 뒤, 내보낸 심볼, 재구성된 타입, 모듈, 시스템 콜 테이블을 gzip으로 압축된 JSON 데이터베이스로 추출합니다. Next.js/React 프론트엔드는 이러한 정적 데이터베이스를 로드하고, OS 빌드 간 나란히 비교할 수 있는 Browse 및 Diff 뷰를 제공합니다. Windows 내부 구조, EDR/안티치트 표면, 버전 간 커널 또는 사용자 모드 바이너리 변화를 분석하는 리버스 엔지니어와 보안 연구자를 대상으로 합니다.
