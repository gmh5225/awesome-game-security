WinDiff 是一款基於 Web 的工具，用於瀏覽與比較不同版本 Microsoft Windows 二進位檔案中的符號、類型與系統呼叫。其 Rust CLI 透過 Winbindex 下載 PE 檔案，從 Microsoft 符號伺服器取得相符的 PDB，並將匯出的符號、重建的類型、模組以及系統呼叫表擷取為 gzip 壓縮的 JSON 資料庫。Next.js/React 前端載入這些靜態資料庫，並提供 Browse 與 Diff 檢視，以便跨作業系統版本進行並排比較。該工具面向分析 Windows 內部機制、EDR/反作弊攻擊面以及核心或使用者模式二進位版本間變化的逆向工程師與安全研究人員。
