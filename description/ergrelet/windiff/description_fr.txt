WinDiff est un outil web permettant de parcourir et de comparer les symboles, types et syscalls entre différentes versions de binaires Microsoft Windows. Une CLI Rust télécharge les fichiers PE via Winbindex, récupère les PDB correspondants depuis le serveur de symboles Microsoft, et extrait les symboles exportés, types reconstruits, modules et tables de syscalls dans des bases JSON compressées en gzip. Une interface Next.js/React charge ces bases statiques et propose des vues Browse et Diff pour une comparaison côte à côte entre builds d'OS. Il s'adresse aux reverse engineers et chercheurs en sécurité qui analysent les internals Windows, les surfaces EDR/anti-cheat, et les changements binaires noyau ou user-mode d'une version à l'autre.
