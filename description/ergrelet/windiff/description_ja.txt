WinDiff は、異なるバージョンの Microsoft Windows バイナリ間でシンボル、型、システムコールを閲覧・比較するための Web ベースのツールです。Rust CLI が Winbindex 経由で PE ファイルをダウンロードし、Microsoft シンボルサーバーから対応する PDB を取得し、エクスポートされたシンボル、再構築された型、モジュール、システムコールテーブルを gzip 圧縮の JSON データベースに抽出します。Next.js/React フロントエンドがこれらの静的データベースを読み込み、OS ビルド間の並列比較向けに Browse および Diff ビューを提供します。Windows 内部、EDR/アンチチートの表面、バージョン間のカーネルまたはユーザーモードバイナリの変化を分析するリバースエンジニアやセキュリティ研究者を対象としています。
