WinDiff es una herramienta web para explorar y comparar símbolos, tipos y syscalls entre distintas versiones de binarios de Microsoft Windows. Una CLI en Rust descarga archivos PE mediante Winbindex, obtiene los PDB correspondientes del servidor de símbolos de Microsoft y extrae símbolos exportados, tipos reconstruidos, módulos y tablas de syscalls en bases de datos JSON comprimidas con gzip. Un frontend Next.js/React carga esas bases estáticas y ofrece vistas Browse y Diff para comparar lado a lado entre builds del SO. Está orientado a ingenieros inversos e investigadores de seguridad que analizan internos de Windows, superficies de EDR/anti-cheat y cambios binarios de kernel o user-mode entre versiones.
