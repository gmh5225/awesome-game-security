WinDiff 是一款基于 Web 的工具，用于浏览和比较不同版本 Microsoft Windows 二进制文件中的符号、类型与系统调用。其 Rust CLI 通过 Winbindex 下载 PE 文件，从 Microsoft 符号服务器获取匹配的 PDB，并将导出的符号、重建的类型、模块以及系统调用表提取为 gzip 压缩的 JSON 数据库。Next.js/React 前端加载这些静态数据库，并提供 Browse 与 Diff 视图，以便跨操作系统版本进行并排比较。该工具面向分析 Windows 内部机制、EDR/反作弊攻击面以及内核或用户模式二进制版本间变化的逆向工程师与安全研究人员。
