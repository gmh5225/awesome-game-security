WinDiff è uno strumento web per esplorare e confrontare simboli, tipi e syscall tra diverse versioni di binari Microsoft Windows. Una CLI Rust scarica i file PE tramite Winbindex, recupera i PDB corrispondenti dal server di simboli Microsoft ed estrae simboli esportati, tipi ricostruiti, moduli e tabelle di syscall in database JSON compressi con gzip. Un frontend Next.js/React carica questi database statici e offre le viste Browse e Diff per il confronto affiancato tra build del sistema operativo. È pensato per reverse engineer e ricercatori di sicurezza che analizzano gli interni di Windows, le superfici EDR/anti-cheat e le variazioni binarie di kernel o user-mode tra versioni.
