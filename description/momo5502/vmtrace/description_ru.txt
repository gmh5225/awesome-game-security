Библиотека на C++, построенная на API Windows Hypervisor Platform (WHP), обеспечивающая trap-управляемое выполнение гостя с памятью на стороне хоста, trap'ами доступа на уровне страниц и перехватом CPUID/syscall. Использует asmjit для генерации кода во время выполнения и предоставляет чистый интерфейс для отображения физической памяти гостя, настройки прав доступа к памяти и обработки VM exits для пошаговой трассировки.
