Bibliothèque C++ basée sur l'API Windows Hypervisor Platform (WHP) qui offre une exécution invitée pilotée par des traps, une mémoire soutenue par l'hôte, des traps d'accès au niveau des pages et l'interception des CPUID/syscalls. Elle utilise asmjit pour la génération de code à l'exécution et expose une interface claire pour mapper la mémoire physique invitée, définir les permissions mémoire et gérer les VM exits pour le traçage pas à pas.
