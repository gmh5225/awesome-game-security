C++-Bibliothek auf Basis der Windows Hypervisor Platform (WHP) API, die trap-gesteuerte Gastausführung mit hostgestütztem Speicher, seitenweisen Zugriffstraps sowie CPUID-/Syscall-Abfangen bereitstellt. Nutzt asmjit zur Laufzeit-Codegenerierung und bietet eine klare Schnittstelle zum Abbilden physischen Gastspeichers, Setzen von Speicherberechtigungen und Verarbeiten von VM exits für Single-Step-Tracing.
