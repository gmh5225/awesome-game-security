Biblioteca C++ basada en la API de Windows Hypervisor Platform (WHP) que ofrece ejecución de invitado impulsada por traps, memoria respaldada por el host, traps de acceso a nivel de página e interceptación de CPUID/syscalls. Usa asmjit para la generación de código en tiempo de ejecución y expone una interfaz limpia para mapear memoria física del invitado, establecer permisos de memoria y gestionar VM exits para el trazado paso a paso.
