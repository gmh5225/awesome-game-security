Emulador de espacio de usuario de Windows basado en Unicorn Engine y Capstone que ejecuta binarios PE x86/x64 con emulación completa de syscalls, carga de minidump y serialización de estado comprimida con Zstd. Incluye una interfaz web (React/TypeScript) para sesiones de emulación interactivas, compatibilidad con compilación cruzada mediante Emscripten/Android NDK e IPC basada en FlatBuffers para trazas de ejecución estructuradas.
