Plugin di IDA Pro che rileva patch e hook in memoria scansionando le regioni di memoria eseguibili di un processo in esecuzione e confrontandole byte per byte con il corrispondente file PE su disco. Usa l'SDK di IDA per l'enumerazione dei segmenti e un parser PE personalizzato per allineare gli indirizzi virtuali agli offset del file, evidenziando le discrepanze nella vista di disassembly.
