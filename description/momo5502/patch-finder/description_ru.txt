Плагин для IDA Pro, который обнаруживает патчи и хуки в памяти, сканируя исполняемые области памяти запущенного процесса и побайтно сравнивая их с соответствующим PE-файлом на диске. Использует IDA SDK для перечисления сегментов и собственный парсер PE для сопоставления виртуальных адресов со смещениями в файле, выделяя расхождения в представлении дизассемблирования.
