Complemento de IDA Pro que detecta parches y hooks en memoria escaneando las regiones de memoria ejecutables de un proceso en ejecución y comparándolas byte a byte con el archivo PE correspondiente en disco. Usa el SDK de IDA para la enumeración de segmentos y un analizador PE personalizado para alinear direcciones virtuales con desplazamientos de archivo, resaltando las discrepancias en la vista de desensamblado.
