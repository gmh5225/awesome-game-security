Experimenteller Ahead-of-Time-(AOT-)Binärübersetzer, der Kontrollflussgraphen aus x86/x64-PE-Executables anhand eines Ghidra-Exports rekonstruiert, Maschinencode über Intel-XED-Disassemblierung zu LLVM IR anhebt und mit dem LLVM-Backend in nativen Code neu kompiliert. Enthält einen PE-Mapper sowie eine Laufzeitumgebung, die kernel32-API-Aufrufe für die Ausführung auf dem Host-Betriebssystem abfängt.
