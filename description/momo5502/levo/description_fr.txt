Traducteur binaire expérimental ahead-of-time (AOT) qui récupère les graphes de flot de contrôle à partir d'exécutables PE x86/x64 via un export Ghidra, élève le code machine en LLVM IR grâce au désassemblage Intel XED, puis recompile en code natif avec le backend LLVM. Inclut un mappeur PE et un environnement d'exécution qui intercepte les appels d'API kernel32 pour l'exécution sur le système hôte.
