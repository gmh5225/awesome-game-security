Экспериментальный ahead-of-time (AOT) бинарный транслятор, который восстанавливает графы потока управления из x86/x64 PE-исполняемых файлов с помощью экспорта Ghidra, поднимает машинный код в LLVM IR через дизассемблирование Intel XED и перекомпилирует в нативный код бэкендом LLVM. Включает PE-маппер и среду выполнения, перехватывающую вызовы API kernel32 для исполнения на хостовой ОС.
