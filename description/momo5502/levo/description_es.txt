Traductor binario experimental ahead-of-time (AOT) que recupera grafos de flujo de control a partir de ejecutables PE x86/x64 mediante exportación de Ghidra, eleva el código máquina a LLVM IR mediante desensamblado con Intel XED y recompila a código nativo con el backend de LLVM. Incluye un mapeador PE y un entorno de ejecución que intercepta llamadas a la API de kernel32 para ejecutarse en el sistema operativo anfitrión.
