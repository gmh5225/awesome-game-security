Traduttore binario sperimentale ahead-of-time (AOT) che recupera i grafi del flusso di controllo da eseguibili PE x86/x64 tramite export di Ghidra, solleva il codice macchina in LLVM IR mediante disassemblaggio Intel XED e ricompila in codice nativo con il backend LLVM. Include un mapper PE e un ambiente runtime che intercetta le chiamate API di kernel32 per l'esecuzione sul sistema operativo host.
