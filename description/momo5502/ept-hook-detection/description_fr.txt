Outil en mode utilisateur qui détecte les hooks EPT (Extended Page Table) placés par les hyperviseurs à l’aide de trois méthodes indépendantes : une détection basée sur le timing qui mesure les écarts de latence d’exécution, des contrôles d’écriture-comparaison qui écrivent dans des pages de code et vérifient si l’hyperviseur redirige silencieusement les lectures, et des contrôles de cohérence inter-threads qui comparent les vues du code entre les cœurs CPU.
