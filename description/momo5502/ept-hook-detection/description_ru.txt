Инструмент в пользовательском режиме, который обнаруживает EPT-хуки (Extended Page Table), установленные гипервизорами, тремя независимыми методами: обнаружение по времени с измерением расхождений задержки выполнения, проверки записи и сравнения, которые записывают в страницы кода и проверяют, не перенаправляет ли гипервизор чтения незаметно, и проверки согласованности между потоками, сравнивающие представления кода на разных ядрах CPU.
