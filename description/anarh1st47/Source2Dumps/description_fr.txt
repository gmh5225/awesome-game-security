Ce dépôt fournit des données dumpées du moteur Source 2, telles que les netvars, les interfaces et les class ID, pour plusieurs titres Valve. Il regroupe des en-têtes générés et des listes textuelles pour des jeux incluant Dota 2, les variantes d'Artifact, Half-Life: Alyx et les environnements Sandbox. La majeure partie du contenu est des données de référence structurées sous forme d'en-têtes C++ et de texte, plutôt que du code applicatif. Il sert principalement aux workflows de reverse engineering Source 2, de suivi des offsets et d'outils de sécurité des jeux.
