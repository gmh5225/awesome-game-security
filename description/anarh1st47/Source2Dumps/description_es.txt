Este repositorio proporciona datos volcados del motor Source 2, como netvars, interfaces y class IDs, para varios títulos de Valve. Agrega encabezados generados y listados de texto para juegos que incluyen Dota 2, variantes de Artifact, Half-Life: Alyx y entornos Sandbox. La mayor parte del contenido son datos de referencia estructurados en forma de encabezados C++ y texto, no código de aplicación. Se usa principalmente en flujos de trabajo de ingeniería inversa de Source 2, seguimiento de offsets y herramientas de seguridad de juegos.
