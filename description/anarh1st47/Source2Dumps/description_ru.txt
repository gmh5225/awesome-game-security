Этот репозиторий предоставляет дампы данных движка Source 2 — netvars, интерфейсы и class ID — для нескольких игр Valve. Он собирает сгенерированные заголовки и текстовые списки для игр, включая Dota 2, варианты Artifact, Half-Life: Alyx и окружения Sandbox. Большая часть содержимого — структурированные справочные данные в виде C++-заголовков и текста, а не прикладной код. Используется в основном для реверс-инжиниринга Source 2, отслеживания смещений и рабочих процессов инструментов игровой безопасности.
