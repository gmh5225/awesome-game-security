Questo repository è una vasta raccolta storica di exploit proof-of-concept per l’escalation di privilegi locali, organizzata per anno e CVE. Contiene soprattutto codice exploit in C, insieme a script shell e Python, oltre a note e riferimenti per ciascuna vulnerabilità. La struttura facilita la consultazione di campioni di exploit dell’era del kernel e la ricostruzione di scenari di ricerca più datati in ambienti di laboratorio controllati. L’uso principale è la formazione sulla sicurezza, lo studio dello sviluppo di exploit e il riferimento cronologico alle vulnerabilità.
