Este proyecto es un monitor interactivo de recursos en terminal para CPU, memoria, discos, red, procesos y métricas GPU opcionales. Está escrito en C++ moderno y ofrece una TUI rápida con soporte de ratón, filtrado de procesos, vistas configurables y personalización de temas. El código es multiplataforma en Linux, macOS y variantes BSD, con opciones flexibles de compilación y empaquetado. Está pensado para desarrolladores, operadores e ingenieros de seguridad que necesitan telemetría del host en tiempo real en entornos de terminal.
