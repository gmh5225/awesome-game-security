Этот проект — интерактивный терминальный монитор ресурсов для CPU, памяти, дисков, сети, процессов и опциональных метрик GPU. Он написан на современном C++ и предоставляет быстрый TUI с поддержкой мыши, фильтрацией процессов, настраиваемыми представлениями и кастомизацией тем. Кодовая база кроссплатформенна для Linux, macOS и вариантов BSD с гибкими возможностями сборки и упаковки. Предназначен для разработчиков, операторов и инженеров по безопасности, которым нужна телеметрия хоста в реальном времени в терминальной среде.
