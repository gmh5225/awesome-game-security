PwnedBoot est une charge utile de démarrage proof-of-concept qui remplace la DLL de mise à jour du microcode Windows afin d’exécuter du code personnalisé très tôt dans la chaîne de démarrage.
Le projet est implémenté en C et C++ avec du code orienté EFI, des fichiers de projet Visual Studio et des composants gnu-efi inclus.
Sa technique centrale montre comment le chargeur d’amorçage peut exécuter le module de remplacement sous certaines options de démarrage, puis remapper l’exécution pour poursuivre le flux de démarrage.
Il est surtout utile pour la recherche sur la sécurité du démarrage, l’analyse de la surface d’attaque en début de boot, et la modélisation de menaces anti-cheat autour de l’exécution pré-OS.
