PwnedBoot es una carga útil de arranque de prueba de concepto que reemplaza la DLL de actualización de microcódigo de Windows para ejecutar código personalizado muy temprano en la cadena de arranque.
El proyecto está implementado en C y C++ con código orientado a EFI, archivos de proyecto de Visual Studio y componentes gnu-efi incluidos.
Su técnica central demuestra cómo el cargador de arranque puede ejecutar el módulo de reemplazo bajo opciones de arranque específicas y luego reasignar la ejecución para continuar el flujo de arranque.
Resulta especialmente útil para investigación de seguridad de arranque, análisis de la superficie de ataque en el arranque temprano y modelado de amenazas anti-cheat en torno a la ejecución pre-OS.
