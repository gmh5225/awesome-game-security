PwnedBoot è un payload di avvio proof-of-concept che sostituisce la DLL di aggiornamento del microcodice di Windows per eseguire codice personalizzato molto presto nella catena di boot.
Il progetto è implementato in C e C++ con codice orientato a EFI, file di progetto Visual Studio e componenti gnu-efi inclusi.
La tecnica centrale mostra come il bootloader possa eseguire il modulo sostitutivo con specifiche opzioni di avvio e poi rimappare l’esecuzione per proseguire il flusso di boot.
È utile soprattutto per la ricerca sulla sicurezza del boot, l’analisi della superficie di attacco nel boot precoce e la modellazione delle minacce anti-cheat legate all’esecuzione pre-OS.
