PwnedBoot — это proof-of-concept загрузочный payload, который подменяет DLL обновления микрокода Windows, чтобы выполнять пользовательский код на самом раннем этапе цепочки загрузки.
Проект реализован на C и C++ и включает код с акцентом на EFI, файлы проектов Visual Studio и входящие в состав компоненты gnu-efi.
Ключевая техника показывает, как загрузчик при определённых параметрах загрузки может выполнить подменённый модуль, а затем переназначить выполнение, чтобы продолжить процесс загрузки.
В первую очередь полезен для исследований безопасности загрузки, анализа поверхности атаки на раннем этапе boot и моделирования угроз античита вокруг выполнения до ОС.
