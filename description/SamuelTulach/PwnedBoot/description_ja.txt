PwnedBoot は、Windows のマイクロコード更新 DLL を置き換えることで、ブートチェーンのごく初期にカスタムコードを実行する概念実証用ブートペイロードです。
プロジェクトは C および C++ で実装され、EFI 向けコード、Visual Studio プロジェクトファイル、同梱の gnu-efi コンポーネントを含みます。
その核心となる手法は、特定のブートオプション下でブートローダーが置換モジュールを実行し、その後実行を再マップしてブートフローを継続する方法を示します。
主にブートセキュリティ研究、早期ブートの攻撃面分析、およびプリ OS 実行に関するアンチチートの脅威モデリングに役立ちます。
