PwnedBoot은 Windows 마이크로코드 업데이트 DLL을 교체하여 부트 체인의 매우 초기에 사용자 지정 코드를 실행하는 개념 증명 부트 페이로드입니다.
이 프로젝트는 C와 C++로 구현되었으며, EFI 중심 코드, Visual Studio 프로젝트 파일, 그리고 포함된 gnu-efi 구성 요소를 갖추고 있습니다.
핵심 기법은 특정 부트 옵션에서 부트로더가 교체 모듈을 실행한 뒤, 실행을 다시 매핑하여 부트 흐름을 이어가는 방식을 보여 줍니다.
주로 부트 보안 연구, 초기 부트 공격 표면 분석, 그리고 OS 이전 실행을 중심으로 한 안티치트 위협 모델링에 유용합니다.
