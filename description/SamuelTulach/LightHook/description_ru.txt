Этот проект — минималистичная однозаголовочная библиотека хуков для систем x86-64 с поддержкой сред Windows, Linux и EFI.
Она написана на чистом C без тяжёлой зависимости от дизассемблера и включает примеры для пользовательского режима, режима ядра и прошивки.
Библиотека делает акцент на переносимости и малом накладном расходе интеграции через платформенные адаптеры выделения, защиты и освобождения памяти.
Она предназначена для низкоуровневой инструментации, реверс-инжиниринга и исследований безопасности игр, где предпочтителен лёгкий хукинг.
