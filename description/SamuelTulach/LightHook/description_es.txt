Este proyecto es una biblioteca de hooks minimalista de un solo encabezado para sistemas x86-64, con soporte para entornos Windows, Linux y EFI.
Está escrita en C puro, sin dependencia de un desensamblador pesado, e incluye ejemplos para modo de usuario, modo kernel y contextos de firmware.
La biblioteca enfatiza la portabilidad y un bajo coste de integración mediante adaptadores específicos de plataforma para asignar, proteger y liberar memoria.
Está pensada para instrumentación de bajo nivel, ingeniería inversa e investigación de seguridad en juegos, donde se prefiere un hooking ligero.
