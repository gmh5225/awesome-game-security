Questo progetto è una libreria di hook minimalista a header singolo per sistemi x86-64, con supporto per ambienti Windows, Linux ed EFI.
È scritta in C puro, senza dipendenze pesanti da disassemblatori, e include esempi per user mode, kernel mode e contesti firmware.
La libreria punta su portabilità e basso overhead di integrazione tramite shim specifici della piattaforma per allocare, proteggere e liberare memoria.
È pensata per strumentazione a basso livello, reverse engineering e ricerca di sicurezza nei giochi, dove si preferisce un hooking leggero.
