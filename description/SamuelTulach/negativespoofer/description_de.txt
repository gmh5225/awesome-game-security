negativespoofer ist ein Boot-Zeit-SMBIOS-Spoofing-Projekt, das Firmware-Tabellen ändert, bevor das Betriebssystem startet.
Die Implementierung folgt einem Clover-ähnlichen Patch-Ansatz und umfasst C- sowie EFI-orientierten Code sowie separate Build- und Nutzungsanleitungen.
Durch die Änderung von SMBIOS-Daten vor dem Booten zielt es auf Hardware-Identitätssignale ab, die später häufig von System- und Sicherheitssoftware ausgewertet werden.
Das Repository dient als historische Ressource für die Forschung zur Manipulation von Hardware-Fingerabdrücken vor dem OS-Start im Kontext von Anti-Cheat und Plattformsicherheit.
