efi-memory は、ファームウェアコンテキストから仮想メモリの読み書きを行うための概念実証（proof-of-concept）EFI ランタイムドライバです。
EfiGuard に着想を得た SetVariable hook による通信方式を用い、ファームウェア側とユーザーモード側の双方のコンポーネントを含みます。
リポジトリには、Windows 上の manual mapping シナリオ向けに kdmapper 風のワークフローから派生した mapper クライアントも含まれています。
主にファームウェアセキュリティ研究、低レベルメモリアクセス実験、および pre-OS やランタイムドライバ連携を扱うゲームセキュリティ研究を対象としています。
