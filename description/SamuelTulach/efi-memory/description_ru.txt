efi-memory — это proof-of-concept EFI runtime-драйвер для чтения и записи виртуальной памяти из контекста прошивки.
Он использует метод связи через SetVariable hook по мотивам EfiGuard и включает сопутствующие компоненты как на стороне прошивки, так и в user-mode.
Репозиторий также содержит mapper-клиент, производный от рабочего процесса в стиле kdmapper, для сценариев manual mapping в Windows.
Проект в основном предназначен для исследований безопасности прошивок, экспериментов с низкоуровневым доступом к памяти и исследований игровой безопасности, связанных с взаимодействием pre-OS или runtime-драйверов.
