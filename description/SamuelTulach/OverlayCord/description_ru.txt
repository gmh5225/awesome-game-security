Этот проект — proof of concept, который перехватывает внутренний конвейер игрового overlay Discord из внешнего процесса.
Он написан на C++ и демонстрирует злоупотребление совместным использованием framebuffer без изменения файлов Discord, внедрения модулей Discord или установки API-хуков.
Пример включает повторно используемый код overlay и образец интеграции для отрисовки через доверенный путь overlay.
Он в основном применяется в исследованиях игровой безопасности и античита для изучения границ доверия overlay и слепых зон обнаружения.
