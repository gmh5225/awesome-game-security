Este proyecto es una prueba de concepto que secuestra el pipeline interno de overlay de juegos de Discord desde un proceso externo.
Está escrito en C++ y demuestra el abuso del uso compartido del framebuffer sin modificar archivos de Discord, inyectar módulos de Discord ni instalar hooks de API.
El ejemplo incluye código de overlay reutilizable y una integración de ejemplo para renderizar a través de la ruta de overlay de confianza.
Se usa principalmente en investigación de seguridad de juegos y anti-trampas para estudiar los límites de confianza del overlay y los puntos ciegos de detección.
