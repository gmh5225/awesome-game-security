Ce projet est un bootkit basé sur UEFI qui s'exécute avant le chargement du noyau Windows, développé avec le framework de firmware EDK-II. Il inclut le support de débogage OVMF pour le développement et les tests sous QEMU, ainsi qu'une solution Visual Studio pour compiler des applications UEFI. Il s'adresse principalement aux chercheurs en sécurité UEFI et aux analystes anti-triche qui étudient les vecteurs d'attaque pré-démarrage, les techniques de bootkit et les mécanismes de persistance au niveau du firmware.
