Este proyecto es un bootkit basado en UEFI que opera antes de que se cargue el kernel de Windows, construido con el framework de desarrollo de firmware EDK-II. Incluye soporte de depuración OVMF para desarrollo y pruebas con QEMU, junto con una solución de Visual Studio para compilar aplicaciones UEFI. Resulta especialmente útil para investigadores de seguridad UEFI y analistas de anti-trampas que estudian vectores de ataque previos al arranque, técnicas de bootkit y mecanismos de persistencia a nivel de firmware.
