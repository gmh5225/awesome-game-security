Этот проект — bootkit на базе UEFI, который работает до загрузки ядра Windows и создан с использованием фреймворка разработки прошивок EDK-II. Он включает поддержку отладки OVMF для разработки и тестирования в QEMU, а также решение Visual Studio для сборки UEFI-приложений. В первую очередь полезен исследователям безопасности UEFI и аналитикам античита, изучающим векторы атак до загрузки ОС, техники bootkit и механизмы персистентности на уровне прошивки.
