unxorer è un plugin IDA per recuperare le stringhe sullo stack da binari offuscati tramite emulazione.
Il plugin è implementato in C++ e usa Unicorn per esplorare i percorsi di ramificazione, preservare lo stato di emulazione e scansionare i dati dello stack alla ricerca di testo decodificato.
Si integra direttamente nei flussi di lavoro di IDA con punti di partenza configurabili e navigazione dell'output verso le stringhe scoperte.
È utile per il reverse engineering nell'analisi di malware e nella sicurezza dei giochi, dove l'offuscamento delle stringhe sullo stack è comune.
