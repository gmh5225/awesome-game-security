unxorer — плагин для IDA, который восстанавливает стековые строки из обфусцированных бинарников с помощью эмуляции.
Плагин написан на C++ и использует Unicorn для обхода ветвящихся путей, сохранения состояния эмуляции и сканирования данных стека в поисках декодированного текста.
Он напрямую встраивается в рабочие процессы IDA с настраиваемыми точками старта и навигацией по найденным строкам в выводе.
Это делает его полезным для реверс-инжиниринга в анализе вредоносного ПО и безопасности игр, где обфускация стековых строк встречается часто.
