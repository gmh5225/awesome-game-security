TPM Spoofer ist ein Proof-of-Concept-Projekt, das die Manipulation TPM-abgeleiteter Kennungen untersucht, die im Hardware-Tracking von Anti-Cheat-Systemen verwendet werden.
Es enthält eine Kernelmode-Komponente, die TPM-Anfrageverarbeitungspfade hookt, sowie ein Usermode-Dienstprogramm zum Lesen von Informationen zur Endorsement Key (EK).
Die Codebasis ist in C und C++ mit Visual Studio und WDK erstellt und richtet sich an das moderne Windows-TPM-Stack-Verhalten.
Der primäre Anwendungsfall ist die Spiele-Sicherheitsforschung mit Fokus darauf, wie TPM- und EK-basierte Signale geprüft, abgefangen und validiert werden können.
