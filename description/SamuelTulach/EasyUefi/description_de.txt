Dieses Projekt ist eine auf Visual Studio ausgerichtete Vorlage zum schnellen Erstellen von GNU-EFI-basierten UEFI-Anwendungen für Forschungszwecke.
Es kombiniert C-Quellcode, Assembly, Linker-Skripte und mitgelieferte GNU-EFI-Komponenten zu einer sofort baubaren Umgebung unter Windows.
Die Vorlage konzentriert sich auf die Vereinfachung von Einrichtung und Iteration statt auf Produktionshärtung und beschleunigt so frühe Firmware-Experimente.
Sie ist vor allem nützlich für Firmware-Sicherheitsforschung, Experimente in der Boot-Phase und Prototypen niedrigstufiger Game-Security-Werkzeuge.
