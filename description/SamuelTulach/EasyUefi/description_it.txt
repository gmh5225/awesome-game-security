Questo progetto è un template orientato a Visual Studio per creare rapidamente applicazioni UEFI basate su GNU-EFI a scopo di ricerca.
Combina codice sorgente C, assembly, script del linker e componenti GNU-EFI in bundle per fornire un ambiente pronto alla compilazione su Windows.
Il template punta a semplificare configurazione e iterazione piuttosto che l'hardening di produzione, rendendo più rapidi i primi esperimenti sul firmware.
È soprattutto utile per la ricerca sulla sicurezza del firmware, esperimenti in fase di boot e prototipi di tool di game security a basso livello.
