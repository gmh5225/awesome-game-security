Ce projet est un modèle orienté Visual Studio pour construire rapidement des applications UEFI basées sur GNU-EFI à des fins de recherche.
Il combine du code source C, de l'assembleur, des scripts de linkage et des composants GNU-EFI inclus pour offrir un environnement prêt à compiler sous Windows.
Le modèle vise à simplifier la configuration et l'itération plutôt qu'à renforcer pour la production, accélérant ainsi les premières expériences firmware.
Il est surtout utile pour la recherche en sécurité firmware, l'expérimentation au stade du démarrage et les prototypes d'outils de sécurité des jeux bas niveau.
