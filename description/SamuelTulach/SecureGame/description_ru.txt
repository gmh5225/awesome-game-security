SecureGame — это демонстрационная игра в стиле Pong, в которой основная игровая логика размещена в Windows VBS enclave.
Решение разделено на хост-приложение для отрисовки и обработки ввода и на enclave DLL, которая хранит состояние выполнения и правила геймплея.
Используются C и C++, SDL2, Visual Studio, Windows SDK и зависимости на основе vcpkg на современных системах Windows с включёнными функциями VBS.
Проект предназначен для исследований античита и доверенного выполнения и показывает, как изолировать чувствительную игровую логику от обычных вмешательств в пользовательском режиме.
