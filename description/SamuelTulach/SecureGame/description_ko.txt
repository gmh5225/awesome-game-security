SecureGame은 핵심 게임 로직을 Windows VBS enclave 안에 두는 Pong 스타일의 샘플 게임입니다.
솔루션은 렌더링과 입력 처리를 담당하는 호스트 애플리케이션과, 런타임 상태 및 게임플레이 규칙을 보관하는 enclave DLL로 나뉩니다.
C와 C++를 사용하며 SDL2, Visual Studio, Windows SDK, vcpkg 기반 종속성을 활용하고, VBS 기능이 활성화된 최신 Windows 시스템을 대상으로 합니다.
이 프로젝트는 민감한 게임 로직을 일반 사용자 모드 변조로부터 격리하는 방법을 보여 주며, 안티치트 및 신뢰 실행 연구용으로 고안되었습니다.
