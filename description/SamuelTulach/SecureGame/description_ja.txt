SecureGame は、コアのゲームロジックを Windows VBS enclave 内に置く Pong 風のサンプルゲームです。
ソリューションは、描画と入力処理を担うホストアプリケーションと、実行時状態およびゲームプレイルールを保持する enclave DLL に分かれています。
C と C++ を用い、SDL2、Visual Studio、Windows SDK、および vcpkg ベースの依存関係を、VBS 機能が有効な現行 Windows 環境で利用します。
本プロジェクトは、機密性の高いゲームロジックを通常のユーザーモード改ざんから隔離する手法を示し、アンチチートおよび信頼実行の研究向けです。
