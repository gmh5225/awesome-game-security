SecureGame es un juego de ejemplo tipo Pong que sitúa la lógica principal del juego dentro de un enclave VBS de Windows.
La solución se divide en una aplicación host para el renderizado y el manejo de entrada, y una DLL de enclave que almacena el estado en tiempo de ejecución y las reglas de gameplay.
Usa C y C++ con SDL2, Visual Studio, Windows SDK y dependencias basadas en vcpkg, en sistemas Windows modernos con las funciones VBS habilitadas.
El proyecto está pensado para investigación de anti-cheat y ejecución confiable, mostrando cómo aislar la lógica sensible del juego de la manipulación en modo usuario normal.
