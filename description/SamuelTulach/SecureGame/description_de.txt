SecureGame ist ein Pong-ähnliches Beispielspiel, das die Kern-Spiellogik in einer Windows-VBS-Enclave ausführt.
Die Lösung ist in eine Host-Anwendung für Rendering und Eingabeverarbeitung sowie eine Enclave-DLL unterteilt, die Laufzeitzustand und Gameplay-Regeln speichert.
Sie nutzt C und C++ mit SDL2, Visual Studio, Windows SDK und vcpkg-basierten Abhängigkeiten auf modernen Windows-Systemen mit aktivierten VBS-Funktionen.
Das Projekt dient der Anti-Cheat- und Trusted-Execution-Forschung und zeigt, wie sensible Spiellogik vor Manipulation im normalen User Mode isoliert werden kann.
