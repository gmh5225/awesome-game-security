SecureGame est un jeu d’exemple de type Pong qui place la logique de jeu principale dans une enclave VBS Windows.
La solution se compose d’une application hôte pour le rendu et la gestion des entrées, et d’une DLL enclave qui stocke l’état d’exécution et les règles de gameplay.
Elle utilise C et C++ avec SDL2, Visual Studio, le Windows SDK et des dépendances basées sur vcpkg, sur des systèmes Windows modernes avec les fonctionnalités VBS activées.
Le projet vise la recherche anti-cheat et d’exécution de confiance, en montrant comment isoler la logique sensible du jeu des altérations en mode utilisateur classique.
