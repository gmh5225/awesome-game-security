mutante est un spoofeur de HWID en mode noyau Windows conçu pour modifier plusieurs sources d’identité matérielle. Le code C et C++ cible les informations de numéro de série des disques, le comportement S.M.A.R.T. et les champs des tables SMBIOS, tout en évitant les dépendances aux hooks permanents. Il est livré sous forme de projet de pilote Visual Studio et reflète une génération plus ancienne d’outils de contournement d’anti-cheat. Aujourd’hui, il est surtout utile comme référence d’archive pour les techniques de manipulation bas niveau des identifiants SMBIOS et de stockage en recherche sur la sécurité des jeux.
