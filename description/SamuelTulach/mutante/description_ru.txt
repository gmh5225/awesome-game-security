mutante — это HWID-спуфер в режиме ядра Windows, предназначенный для изменения нескольких источников аппаратной идентичности. Код на C и C++ затрагивает сведения о серийных номерах дисков, поведение S.M.A.R.T. и поля таблиц SMBIOS, избегая зависимости от постоянных хуков. Проект поставляется как драйверный проект Visual Studio и отражает более раннее поколение инструментов обхода античита. Сегодня он наиболее полезен как архивный справочник по низкоуровневым методам манипуляции SMBIOS и идентификаторами накопителей в исследованиях игровой безопасности.
