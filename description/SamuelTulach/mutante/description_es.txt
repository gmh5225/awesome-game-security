mutante es un spoofeador de HWID en modo kernel de Windows diseñado para alterar varias fuentes de identidad de hardware. El código en C y C++ se centra en la información de número de serie del disco, el comportamiento S.M.A.R.T. y los campos de las tablas SMBIOS, evitando dependencias de hooks permanentes. Se distribuye como un proyecto de controlador de Visual Studio y refleja una generación anterior de herramientas de evasión de anti-cheat. Hoy es más útil como referencia de archivo sobre técnicas de manipulación de bajo nivel de identificadores SMBIOS y de almacenamiento en la investigación de seguridad de juegos.
