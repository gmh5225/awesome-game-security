本プロジェクトは、アプリにアタッチして実行時の値を変更するためのオープンソースの Android メモリ編集プラットフォームです。
ネイティブのスキャンエンジンとモバイルクライアント層、およびプロセス検査・値編集・値フリーズのワークフロー向けユーティリティを組み合わせています。
リポジトリは C++ のコアコンポーネントに加え、Kotlin と Java の Android モジュール、サポート用スクリプトとチュートリアルで構成されています。
また、root 済み・非 root の両方の利用パスを支える、改造およびインジェクション向けツールも含みます。
主にモバイルゲームハッキングの実務者と、Android のランタイム操作を研究するセキュリティ研究者を対象としています。
