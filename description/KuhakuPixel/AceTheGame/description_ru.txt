Этот проект — открытая платформа редактирования памяти Android для подключения к приложениям и изменения значений во время выполнения.
Она сочетает нативный движок сканирования с мобильными клиентскими слоями и утилитами для инспекции процессов, редактирования значений и их заморозки.
Репозиторий охватывает основные компоненты на C++, а также Android-модули на Kotlin и Java, плюс вспомогательные скрипты и руководства.
Также включает инструменты для моддинга и инъекций, поддерживающие сценарии использования с root и без root.
В основном предназначен для практиков мобильного game hacking и исследователей безопасности, изучающих манипуляцию runtime Android.
