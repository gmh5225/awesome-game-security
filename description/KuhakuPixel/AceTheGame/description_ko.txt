이 프로젝트는 앱에 연결하여 런타임 값을 수정하기 위한 오픈소스 Android 메모리 편집 플랫폼입니다.
네이티브 스캔 엔진과 모바일 클라이언트 계층, 그리고 프로세스 검사·값 편집·값 동결 워크플로를 위한 유틸리티 도구를 결합합니다.
저장소는 C++ 핵심 구성 요소와 Kotlin 및 Java Android 모듈, 보조 스크립트와 튜토리얼로 구성됩니다.
루팅 및 비루팅 사용 경로를 모두 지원하는 모딩·인젝션 지향 도구도 포함합니다.
주로 모바일 게임 해킹 실무자와 Android 런타임 조작을 연구하는 보안 연구자를 대상으로 합니다.
