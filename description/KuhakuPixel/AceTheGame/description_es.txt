Este proyecto es una plataforma de código abierto para edición de memoria en Android, pensada para adjuntarse a aplicaciones y modificar valores en tiempo de ejecución.
Combina un motor de escaneo nativo con capas de cliente móvil y herramientas utilitarias para inspección de procesos, edición de valores y congelación de valores.
El repositorio abarca componentes centrales en C++ junto con módulos Android en Kotlin y Java, además de scripts y tutoriales de apoyo.
También incluye herramientas orientadas a modding e inyección para admitir rutas de uso con root y sin root.
Está destinado principalmente a profesionales del game hacking móvil y a investigadores de seguridad que estudian la manipulación del runtime de Android.
