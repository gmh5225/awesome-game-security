Faultline은 워킹 셋 페이지 폴트를 감시하고 의심스러운 스레드를 검사하여, 수동 매핑되었거나 등록되지 않은 메모리에서 실행되는 코드를 탐지하는 Windows 안티치트 개념 증명입니다. 최신 C++23로 작성되고 CMake로 빌드되며, 워킹 셋 감시 API(InitializeProcessForWsWatch, GetWsChangesEx, QueryWorkingSet)를 사용해 PEB에 나열된 모듈 밖에 있는 FaultingPc 또는 상주 실행 가능 페이지를 찾습니다. 탐지되면 해당 스레드를 일시 중단하고 호출 스택을 추적해 실행 체인을 캡처합니다. 호스트 프로세스, 수동 맵 인젝터(원격 스레드 또는 SetThreadContext 하이재킹), 그리고 shellcode 스타일 및 수동 맵 주입 탐지를 시연하는 테스트 페이로드를 포함합니다. 메모리 기반 치트와 주입 기법의 사용자 모드 탐지를 연구하는 게임 보안 및 안티치트 연구자를 대상으로 합니다.
