Faultline is a Windows anticheat proof of concept that detects code executing from manually mapped or otherwise unregistered memory by watching working set page faults and inspecting suspicious threads. Written in modern C++23 and built with CMake, it uses the working set watch APIs (InitializeProcessForWsWatch, GetWsChangesEx, and QueryWorkingSet) to find FaultingPc or resident executable pages that fall outside modules listed in the PEB. When a hit occurs, it suspends the offending thread and walks its call stack to capture the execution chain. The project includes a host process, a manual-map injector (remote thread or SetThreadContext hijack), and a test payload for demonstrating detection of shellcode-style and manual-map injection. It is aimed at game security and anti-cheat researchers studying usermode detection of memory-based cheats and injection techniques.
