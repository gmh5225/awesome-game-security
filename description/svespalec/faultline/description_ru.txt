Faultline — это proof of concept античита для Windows, который обнаруживает код, выполняемый из вручную отображённой или иным образом незарегистрированной памяти, отслеживая page fault рабочего набора и проверяя подозрительные потоки. Написан на современном C++23 и собирается с CMake; использует API наблюдения за рабочим набором (InitializeProcessForWsWatch, GetWsChangesEx и QueryWorkingSet), чтобы находить FaultingPc или резидентные исполняемые страницы вне модулей, перечисленных в PEB. При срабатывании приостанавливает нарушающий поток и обходит его стек вызовов, чтобы зафиксировать цепочку выполнения. В проект входят хост-процесс, инжектор manual-map (удалённый поток или перехват через SetThreadContext) и тестовая полезная нагрузка для демонстрации обнаружения инъекций в стиле shellcode и manual-map. Предназначен для исследователей игровой безопасности и античита, изучающих usermode-обнаружение читов на основе памяти и техник инъекции.
