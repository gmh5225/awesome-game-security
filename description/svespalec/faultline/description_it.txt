Faultline è una proof of concept di anti-cheat per Windows che rileva codice eseguito da memoria mappata manualmente o altrimenti non registrata, monitorando i page fault del working set e ispezionando i thread sospetti. Scritto in C++23 moderno e costruito con CMake, usa le API di monitoraggio del working set (InitializeProcessForWsWatch, GetWsChangesEx e QueryWorkingSet) per trovare FaultingPc o pagine eseguibili residenti fuori dai moduli elencati nel PEB. In caso di rilevamento sospende il thread responsabile e percorre il suo call stack per catturare la catena di esecuzione. Il progetto include un processo host, un injector di manual mapping (remote thread o hijack con SetThreadContext) e un payload di test per dimostrare il rilevamento di iniezioni in stile shellcode e di manual mapping. È rivolto a ricercatori di game security e anti-cheat che studiano il rilevamento in usermode di cheat basati sulla memoria e di tecniche di injection.
