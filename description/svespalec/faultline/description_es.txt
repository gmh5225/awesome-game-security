Faultline es una prueba de concepto de anti-cheat para Windows que detecta código ejecutado desde memoria mapeada manualmente o no registrada de otro modo, vigilando fallos de página del working set e inspeccionando hilos sospechosos. Escrito en C++23 moderno y construido con CMake, usa las API de vigilancia del working set (InitializeProcessForWsWatch, GetWsChangesEx y QueryWorkingSet) para encontrar FaultingPc o páginas ejecutables residentes fuera de los módulos listados en el PEB. Cuando hay una coincidencia, suspende el hilo infractor y recorre su pila de llamadas para capturar la cadena de ejecución. El proyecto incluye un proceso anfitrión, un inyector de mapeo manual (hilo remoto o secuestro con SetThreadContext) y una carga de prueba para demostrar la detección de inyección tipo shellcode y de mapeo manual. Está dirigido a investigadores de seguridad en juegos y anti-cheat que estudian la detección en modo usuario de trampas basadas en memoria y técnicas de inyección.
