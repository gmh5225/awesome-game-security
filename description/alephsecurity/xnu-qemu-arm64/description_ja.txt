本プロジェクトは Aleph Security による QEMU のフォークで、ARM64 エミュレーションを用いて QEMU 上で完全に動作する iOS システムを起動することを目指しています。
launchd サービス、対話型 bash、読み書き可能なセカンダリディスクデバイス、未署名バイナリの実行、TCP トンネリング経由の SSH をサポートし、オプションで KVM ハードウェアアクセラレーションも利用できます。
コードベースは QEMU を拡張し、ARM64 上で XNU カーネルをエミュレートするための iOS 向けマシン／デバイスモデルを追加しています。
主に iOS セキュリティ研究者やカーネルエクスプロイト開発者が、エミュレート環境で XNU 内部や iOS ランタイムの挙動を調べる用途に適しています。
