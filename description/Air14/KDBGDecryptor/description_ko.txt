이 프로젝트는 커널 디버거 데이터 블록(KDBG)을 복호화하는 방법을 보여주는 Windows 커널 모드 샘플입니다. C++로 작성된 Visual Studio 드라이버 프로젝트로 구현되며, KdDecodeBlockData와 같은 네이티브 커널 내부 API를 사용합니다. 코드는 API를 활용한 직접 디코딩과, 메모리에서 복사한 KiWaitNever 및 KiWaitAlways 값을 이용한 수동 복호화라는 두 가지 기법을 보여 주며, 후자는 더 은밀한 워크플로를 위한 것입니다. 주로 커널 리버스 엔지니어링, 저수준 디버깅 연구, 안티치트 관련 메모리 분석 경로를 공부하는 데 유용합니다.
