Этот проект — пример драйвера Windows в режиме ядра, демонстрирующий расшифровку блока данных отладчика ядра (KDBG). Он реализован на C++ как проект драйвера Visual Studio и использует нативные внутренности ядра, такие как KdDecodeBlockData. Код показывает две техники: прямое декодирование с помощью API и ручную расшифровку с использованием значений KiWaitNever и KiWaitAlways, скопированных из памяти, для более скрытного сценария. В основном полезен для реверс-инжиниринга ядра, исследований низкоуровневой отладки и изучения путей анализа памяти, связанных с античитом.
