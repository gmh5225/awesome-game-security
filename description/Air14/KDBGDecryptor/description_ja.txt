本プロジェクトは、カーネルデバッガーデータブロック（KDBG）の復号方法を示す Windows カーネルモードのサンプルです。C++ による Visual Studio ドライバープロジェクトとして実装され、KdDecodeBlockData などのネイティブなカーネル内部機能を利用します。コードでは、API 支援による直接デコードと、メモリからコピーした KiWaitNever および KiWaitAlways の値を用いた手動復号という、よりステルスなワークフロー向けの二通りの手法を示します。主にカーネルのリバースエンジニアリング、低レベルデバッグ研究、およびアンチチート関連のメモリ解析経路の学習に役立ちます。
