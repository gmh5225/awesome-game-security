Este proyecto es un complemento anti-anti-depuración de código abierto basado en hipervisor para x64dbg y x32dbg. Usa un controlador de kernel y técnicas Intel VT-x/EPT para enganchar y sanear muchas superficies de detección de depuradores, incluidos los valores PEB, los indicadores de hilos y procesos, y numerosas consultas a API Nt*. La implementación está orientada a sistemas Windows de 64 bits y se centra en ocultar la presencia del depurador sin perder flujos de depuración prácticos. Es ampliamente aplicable a la ingeniería inversa y a la investigación de seguridad en juegos frente a binarios muy protegidos.
