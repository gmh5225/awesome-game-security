Este proyecto es un cliente RCON (Remote Console) para servidores dedicados de Palworld que permite enviar comandos de administración de forma remota. Implementa el protocolo Source RCON para conectarse a servidores Palworld y ejecutar comandos de gestión como expulsiones de jugadores, baneos, mensajes broadcast y cambios de configuración del servidor. Está dirigido a administradores de servidores Palworld y a desarrolladores de servidores de juegos que gestionan instancias dedicadas de Palworld.
