Questo progetto è un client RCON (Remote Console) per server dedicati Palworld, pensato per inviare comandi di amministrazione in remoto. Implementa il protocollo Source RCON per connettersi ai server Palworld ed eseguire comandi di gestione come kick e ban dei giocatori, broadcast e modifiche alla configurazione del server. È rivolto agli amministratori di server Palworld e agli sviluppatori di game server che gestiscono istanze dedicate di Palworld.
