Questo progetto è un repository elencato sotto anti-cheat / rilevamento:iniezione e si concentra sul rilevatore di iniezione di thread DLL.
È scritto principalmente in C++ e C/C++ e si concentra sul lavoro a livello kernel.
È soprattutto utile per ingegneri anti-cheat e ricercatori di sicurezza difensiva che operano nell'area anti-cheat / rilevamento:iniezione.
