Ce projet est un miroir ou une archive du code source divulgué de Counter-Strike: Global Offensive. Il contient la base de code du moteur Source ainsi que des implémentations propres au jeu couvrant le réseau, le rendu, la logique de jeu et l'architecture client-serveur. Il est surtout utile aux chercheurs sur les moteurs de jeu et aux ingénieurs en rétro-ingénierie qui étudient les internals du moteur Source et l'architecture client-serveur de CSGO.
