Этот проект — unflutter, инструмент для реверса скомпилированных приложений Flutter/Dart. Он извлекает метаданные Dart snapshot из Flutter APK и iOS-приложений, восстанавливая имена классов, функций и сведения о типах, которые иначе теряются при AOT-компиляции. Инструмент на Python разбирает формат snapshot Dart VM и выводит реконструированную информацию о символах. Предназначен для исследователей мобильной безопасности и инженеров по реверсу, анализирующих приложения на Flutter.
