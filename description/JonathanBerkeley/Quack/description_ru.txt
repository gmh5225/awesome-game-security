Этот проект — универсальный исследовательский набор для античита, предназначенный для интеграции с играми и обнаружения читерского поведения. Он в основном написан на C++ и включает несколько модулей, примеры интеграции, документацию, а также сопутствующие red-team инструменты для adversarial-тестирования. Фреймворк содержит клиентские компоненты мониторинга и вспомогательную инфраструктуру, предназначенные для экспериментов, а не для промышленного развёртывания. Основная аудитория — исследователи безопасности игр, разрабатывающие и проверяющие концепции античита.
