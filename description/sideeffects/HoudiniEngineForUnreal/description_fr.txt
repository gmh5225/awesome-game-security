Ce projet se concentre sur le plug-in Houdini Engine pour Unreal Engine.
Ce plug-in intègre le workflow procédural puissant et flexible de Houdini dans Unreal Engine via les Houdini Digital Assets.
Il est principalement utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine game engine / game engine plugins:unreal.
