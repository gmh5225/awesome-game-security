Este proyecto se centra en el complemento Houdini Engine para Unreal Engine.
Este complemento lleva el flujo de trabajo procedimental potente y flexible de Houdini a Unreal Engine a través de Houdini Digital Assets.
Es principalmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el área game engine / game engine plugins:unreal.
