Этот проект посвящён плагину Houdini Engine для Unreal Engine.
Этот плагин переносит мощный и гибкий процедурный рабочий процесс Houdini в Unreal Engine через Houdini Digital Assets.
Он в основном полезен разработчикам игр, программистам движков и исследователям графики, работающим в области game engine / game engine plugins:unreal.
