Questo progetto si concentra sul plugin Houdini Engine per Unreal Engine.
Questo plug-in porta il potente e flessibile workflow procedurale di Houdini in Unreal Engine tramite Houdini Digital Assets.
È principalmente utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell'area game engine / game engine plugins:unreal.
