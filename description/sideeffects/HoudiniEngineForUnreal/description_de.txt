Dieses Projekt konzentriert sich auf das Houdini Engine Plugin für Unreal Engine.
Dieses Plug-in bringt Houdinis leistungsstarken und flexiblen prozeduralen Workflow über Houdini Digital Assets in Unreal Engine.
Es ist vor allem nützlich für Spieleentwickler, Engine-Programmierer und Grafikforscher, die im Bereich game engine / game engine plugins:unreal arbeiten.
