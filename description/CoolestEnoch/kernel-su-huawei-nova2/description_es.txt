Este proyecto es un port de KernelSU para el dispositivo Huawei Nova 2, que proporciona un árbol de código fuente completo del kernel Linux ARM64 modificado para admitir la gestión de acceso root a nivel de kernel. Incluye configuraciones de kernel específicas de Android, definiciones de device tree y los parches del framework KernelSU integrados en el kernel de Huawei. Resulta especialmente útil para investigadores de kernels Android y desarrolladores de acceso root que estudian la integración de KernelSU en árboles de kernel específicos del fabricante.
