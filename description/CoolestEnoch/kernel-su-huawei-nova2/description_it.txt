Questo progetto è un port di KernelSU per il dispositivo Huawei Nova 2 e fornisce un albero sorgente completo del kernel Linux ARM64 modificato per supportare la gestione dell'accesso root a livello di kernel. Include configurazioni del kernel specifiche per Android, definizioni del device tree e le patch del framework KernelSU integrate nel kernel Huawei. È destinato principalmente a ricercatori di kernel Android e sviluppatori di accesso root che studiano l'integrazione di KernelSU su alberi kernel specifici del produttore.
