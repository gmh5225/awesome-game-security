Этот проект — порт KernelSU для устройства Huawei Nova 2, предоставляющий полное дерево исходного кода ядра Linux ARM64, модифицированное для поддержки управления root-доступом на уровне ядра. В него входят Android-специфичные конфигурации ядра, определения device tree и патчи фреймворка KernelSU, интегрированные в ядро Huawei. Он в первую очередь полезен исследователям Android-ядра и разработчикам root-доступа, изучающим интеграцию KernelSU в вендорские деревья ядра.
