Android Inline Hook ARM64 es un framework de hooking nativo para construir bibliotecas compartidas Android ARM64 que parchean rutas de código objetivo en tiempo de ejecución. La implementación se basa en C, C++ y ensamblador ARM64, con scripts de build de Android NDK y lógica stub para el manejo de trampolines inline. Enfatiza el hooking inline puro en lugar del hooking PLT e incluye ejemplos de control a nivel de registros dentro de los handlers de hook. Su principal caso de uso es la ingeniería inversa móvil y la experimentación en seguridad de juegos cuando se requiere la interceptación de funciones nativas.
