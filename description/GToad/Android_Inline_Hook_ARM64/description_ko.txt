Android Inline Hook ARM64는 런타임에 대상 코드 경로를 패치하는 ARM64 Android 공유 라이브러리를 구축하기 위한 네이티브 후킹 프레임워크입니다. 구현은 C, C++, ARM64 어셈블리를 기반으로 하며 Android NDK 빌드 스크립트와 인라인 트램폴린 처리용 스텁 로직을 포함합니다. PLT 후킹이 아닌 순수 인라인 후킹에 중점을 두며, 훅 핸들러 내부에서 레지스터 수준 제어를 하는 예제도 제공합니다. 주요 사용 사례는 네이티브 함수 가로채기가 필요한 모바일 리버스 엔지니어링 및 게임 보안 실험입니다.
