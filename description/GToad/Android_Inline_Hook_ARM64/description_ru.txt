Android Inline Hook ARM64 — нативный фреймворк хукинга для сборки ARM64 Android shared libraries, которые патчат целевые пути кода во время выполнения. Реализация основана на C, C++ и ARM64-ассемблере, включает скрипты сборки Android NDK и stub-логику для обработки inline-трамплинов. Акцент сделан на чистом inline-хукинге, а не на PLT-хукинге; есть примеры управления на уровне регистров в обработчиках хуков. Основной сценарий использования — мобильный reverse engineering и эксперименты в области игровой безопасности, где требуется перехват нативных функций.
