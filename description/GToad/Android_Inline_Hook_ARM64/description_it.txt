Android Inline Hook ARM64 è un framework di hooking nativo per costruire librerie condivise Android ARM64 che patchano i percorsi di codice target a runtime. L'implementazione si basa su C, C++ e assembly ARM64, con script di build Android NDK e logica stub per la gestione dei trampoline inline. Privilegia l'hooking inline puro piuttosto che l'hooking PLT e include esempi di controllo a livello di registri negli handler di hook. Il caso d'uso principale è il reverse engineering mobile e la sperimentazione nella sicurezza dei giochi, quando è richiesta l'intercettazione di funzioni native.
