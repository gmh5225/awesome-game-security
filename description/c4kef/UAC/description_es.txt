Este proyecto es una plataforma de desarrollo anti-cheat diseñada para ayudar a los desarrolladores de cheats a detectar debilidades en sus propias implementaciones.
Incluye un componente de controlador anti-cheat en modo kernel construido con Visual Studio que implementa rutinas de detección para técnicas de cheat comunes.
La base de código en C++ proporciona un framework para probar y validar métodos de detección de cheats desde el nivel del kernel.
Es especialmente útil para ingenieros anti-cheat e investigadores de seguridad en juegos que estudian mecanismos de detección de cheats a nivel de kernel y arquitectura anti-cheat.
