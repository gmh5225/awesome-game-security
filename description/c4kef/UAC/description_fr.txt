Ce projet est une plateforme de développement anti-cheat conçue pour aider les développeurs de cheats à détecter les faiblesses de leurs propres implémentations.
Il comprend un composant pilote anti-cheat en mode noyau construit avec Visual Studio, qui implémente des routines de détection pour les techniques de cheat courantes.
La base de code C++ fournit un framework pour tester et valider les méthodes de détection de cheats depuis le niveau noyau.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité des jeux qui étudient les mécanismes de détection de cheats au niveau noyau et l'architecture anti-cheat.
