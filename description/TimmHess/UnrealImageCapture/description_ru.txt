Этот проект показывает, как сохранять изображения Unreal Engine на диск для рабочих процессов машинного обучения. Он использует код Unreal C++ для захвата кадров с высокой частотой без блокировки потоков рендеринга или основного игрового потока. Реализация охватывает синхронизированный вывод RGB, маски сегментации и захват глубины, а также рекомендации по настройке форматов и зависимостей сборки движка. Основной сценарий использования — создание размеченных синтетических наборов данных из сцен UE для исследований в области компьютерного зрения и симуляции.
