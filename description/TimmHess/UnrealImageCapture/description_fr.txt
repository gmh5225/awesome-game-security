Ce projet montre comment capturer des images Unreal Engine sur disque pour des flux de travail en apprentissage automatique. Il utilise du code Unreal C++ pour réaliser une capture de trames à haut FPS sans bloquer les threads de rendu ou le thread principal du jeu. L'implémentation couvre la sortie RGB synchronisée, les masques de segmentation et la capture de profondeur, avec des conseils de configuration pour les formats et les dépendances de build du moteur. Son principal cas d'usage est la génération de jeux de données synthétiques annotés à partir de scènes UE pour la vision par ordinateur et la recherche en simulation.
