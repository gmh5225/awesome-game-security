Questo progetto mostra come catturare immagini di Unreal Engine su disco per flussi di lavoro di machine learning. Usa codice Unreal C++ per eseguire una cattura di frame ad alto FPS senza bloccare i thread di rendering o il thread principale del gioco. L'implementazione copre l'output RGB sincronizzato, le maschere di segmentazione e la cattura della profondità, con indicazioni di configurazione per formati e dipendenze di build del motore. Il caso d'uso principale è generare dataset sintetici annotati da scene UE per la ricerca in computer vision e simulazione.
