Este proyecto demuestra cómo capturar imágenes de Unreal Engine en disco para flujos de trabajo de aprendizaje automático. Utiliza código Unreal C++ para realizar una captura de fotogramas a alto FPS sin bloquear los hilos de renderizado ni el hilo principal del juego. La implementación cubre la salida RGB sincronizada, máscaras de segmentación y captura de profundidad, con orientación de configuración para formatos y dependencias de compilación del motor. Su principal caso de uso es generar conjuntos de datos sintéticos anotados a partir de escenas de UE para investigación en visión por computadora y simulación.
