Система античита на C++ для мультиплеера GTA V (GVMP), реализующая обнаружение читов, отслеживание игроков и мониторинг безопасности с помощью сетевого стека ENet и хуков рендеринга DirectX.
Включает сканирование сигнатур (pattern scanning), перехват функций на базе MinHook, проверки целостности процессов и систему сборки на CMake для интеграции с мультиплеерным фреймворком alt:V.
В первую очередь полезна операторам игровых серверов и инженерам по античиту, изучающим клиентские реализации обнаружения читов в мультиплеерных средах GTA V.
