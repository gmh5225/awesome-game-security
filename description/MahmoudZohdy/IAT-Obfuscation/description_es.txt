Este proyecto es una herramienta de ofuscación de importaciones PE de Windows que dificulta el análisis estático de secuencias de API.
Está escrita en C++ y reescribe las entradas de la Import Address Table intercambiando funciones importadas dentro de la misma DLL.
Un encabezado complementario basado en TLS restaura el comportamiento correcto de las importaciones en tiempo de ejecución antes de que se ejecute la lógica principal del programa.
El caso de uso principal es la investigación educativa en seguridad sobre técnicas de ocultación de importaciones y su impacto en los flujos de análisis de malware.
