Этот проект — инструмент обфускации импортов Windows PE, усложняющий статический анализ последовательностей вызовов API.
Он написан на C++ и перезаписывает записи Import Address Table, меняя местами импортируемые функции внутри одной и той же DLL.
Сопровождающий заголовок на основе TLS восстанавливает корректное поведение импортов во время выполнения до запуска основной логики программы.
Основной сценарий использования — образовательные исследования в области безопасности, посвящённые методам сокрытия импортов и их влиянию на рабочие процессы анализа вредоносного ПО.
