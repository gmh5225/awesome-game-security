Dieses Projekt ist ein Windows-PE-Import-Obfuskationstool, das die statische Analyse von API-Aufrufsequenzen erschwert.
Es ist in C++ geschrieben und schreibt Einträge der Import Address Table um, indem importierte Funktionen innerhalb derselben DLL vertauscht werden.
Ein begleitender TLS-basierter Header stellt zur Laufzeit das korrekte Importverhalten wieder her, bevor die Hauptlogik des Programms ausgeführt wird.
Der Hauptanwendungsfall ist die bildungsorientierte Sicherheitsforschung zu Import-Verstecktechniken und deren Auswirkungen auf Malware-Analyse-Workflows.
