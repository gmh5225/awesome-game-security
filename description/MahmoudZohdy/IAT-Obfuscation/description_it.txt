Questo progetto è uno strumento di offuscamento delle importazioni PE di Windows che rende più difficile l'analisi statica delle sequenze di API.
È scritto in C++ e riscrive le voci della Import Address Table scambiando le funzioni importate all'interno della stessa DLL.
Un header companion basato su TLS ripristina a runtime il corretto comportamento delle importazioni prima dell'esecuzione della logica principale del programma.
Il caso d'uso principale è la ricerca educativa sulla sicurezza relativa alle tecniche di nascondimento delle importazioni e al loro impatto sui flussi di analisi del malware.
