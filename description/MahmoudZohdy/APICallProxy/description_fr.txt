Ce projet est un framework d’obfuscation d’appels à l’API Windows qui achemine les opérations en mode utilisateur via un pilote noyau.
Il est implémenté en C/C++ et expose de nombreuses actions sur les fichiers, processus, mémoire, registre et réseau via des gestionnaires IOCTL DeviceIoControl plutôt que par des appels API directs.
Le dépôt inclut également plusieurs clients d’exemple illustrant des flux pratiques tels que l’injection basée sur APC, le chargement de pilotes et la communication par sockets.
Il est destiné à la recherche en sécurité bas niveau et à des expériences contrôlées sur l’évasion de la surveillance des API et le renforcement de l’analyse comportementale.
