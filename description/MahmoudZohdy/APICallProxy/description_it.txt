Questo progetto è un framework di offuscamento delle chiamate all’API di Windows che instrada le operazioni in user mode tramite un driver del kernel.
È implementato in C/C++ ed espone molte azioni su file, processi, memoria, registro e rete tramite handler IOCTL di DeviceIoControl invece di chiamate API dirette.
Il repository include anche diversi client di esempio che dimostrano flussi pratici come injection basata su APC, caricamento di driver e comunicazione tramite socket.
È pensato per la ricerca di sicurezza a basso livello e per esperimenti controllati sull’evasione del monitoraggio delle API e sul rafforzamento dell’analisi comportamentale.
