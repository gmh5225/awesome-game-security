Este proyecto es un framework de ofuscación de llamadas a la API de Windows que enruta operaciones en modo usuario a través de un controlador del núcleo.
Está implementado en C/C++ y expone muchas acciones de archivo, proceso, memoria, registro y red mediante manejadores IOCTL de DeviceIoControl en lugar de llamadas directas a la API.
El repositorio también incluye varios clientes de ejemplo que demuestran flujos prácticos como inyección basada en APC, carga de controladores y comunicación por sockets.
Está pensado para investigación de seguridad de bajo nivel y experimentos controlados sobre evasión de la supervisión de API y endurecimiento del análisis conductual.
