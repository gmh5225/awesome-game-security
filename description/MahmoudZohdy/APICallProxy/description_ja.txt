本プロジェクトは、ユーザーモードの操作をカーネルドライバ経由でルーティングする Windows API 呼び出し難読化フレームワークです。
C/C++ で実装され、ファイル、プロセス、メモリ、レジストリ、ネットワーク関連の多くの操作を直接 API 呼び出しではなく DeviceIoControl の IOCTL ハンドラ経由で公開します。
リポジトリには、APC ベースのインジェクション、ドライバ読み込み、ソケット通信などの実用的なワークフローを示す複数のサンプルクライアントも含まれています。
API 監視回避や行動分析の強化に関する低レベルセキュリティ研究および制御された実験を目的としています。
