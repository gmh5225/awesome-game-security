Этот проект — фреймворк обфускации вызовов Windows API, который маршрутизирует операции пользовательского режима через драйвер ядра.
Он реализован на C/C++ и предоставляет множество операций с файлами, процессами, памятью, реестром и сетью через обработчики IOCTL DeviceIoControl вместо прямых вызовов API.
Репозиторий также включает несколько примеров клиентов, демонстрирующих практические сценарии: инъекцию на базе APC, загрузку драйвера и сетевое взаимодействие через сокеты.
Проект предназначен для низкоуровневых исследований безопасности и контролируемых экспериментов по обходу мониторинга API и усилению поведенческого анализа.
