Этот проект — драйвер ядра Windows для перехвата и дампа IOCTL-трафика других драйверов устройств. Он записывает метаданные запросов: код IOCTL, тип пути передачи и размеры буферов, а также может сохранять содержимое входных буферов с дедупликацией повторяющихся комбинаций. Репозиторий включает компоненты драйвера и клиентской части для выбора целевых устройств и сбора журналов в заданном формате вывода. В основном используется для реверс-инжиниринга проприетарных драйверов, включая интерфейсы античита и другие чувствительные к безопасности пути обмена с ядром.
