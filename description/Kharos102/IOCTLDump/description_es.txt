Este proyecto es un controlador del núcleo de Windows para interceptar y volcar el tráfico IOCTL de otros controladores de dispositivos. Registra metadatos de las solicitudes, como el código IOCTL, el tipo de ruta de transporte y los tamaños de búfer, y puede almacenar el contenido de los búferes de entrada deduplicando combinaciones repetidas. El repositorio incluye componentes del controlador y del cliente para seleccionar dispositivos de destino y recopilar registros en un formato de salida definido. Se usa principalmente para la ingeniería inversa de controladores propietarios, incluidas interfaces de anti-cheat y otras rutas de comunicación del núcleo sensibles a la seguridad.
