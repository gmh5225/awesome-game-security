Questo progetto è un plugin per IDA Pro che automatizza la scoperta di vulnerabilità sul pseudocodice decompilato. È scritto in Python ed esegue scansioni SAST basate su regole tramite pattern YAML in stile opengrep/semgrep, presentando poi i risultati in un'interfaccia IDA interattiva. Il plugin supporta anche spiegazioni assistite da IA opzionali per aiutare a valutare e comprendere i problemi individuati. È pensato per reverse engineer, giocatori di CTF e auditor di sicurezza che desiderano flussi di lavoro più rapidi per l'analisi delle vulnerabilità nei binari.
