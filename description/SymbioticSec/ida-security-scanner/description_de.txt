Dieses Projekt ist ein IDA-Pro-Plugin, das die Entdeckung von Schwachstellen im dekompilierten Pseudocode automatisiert. Es ist in Python geschrieben und führt regelbasierte SAST-Scans über opengrep/semgrep-ähnliche YAML-Muster aus und zeigt die Ergebnisse in einer interaktiven IDA-Oberfläche an. Das Plugin unterstützt außerdem optionale KI-gestützte Erklärungen, die bei der Priorisierung und dem Verständnis gefundener Probleme helfen. Es richtet sich an Reverse Engineers, CTF-Spieler und Sicherheitsauditoren, die schnellere Workflows zur Analyse von Binärschwachstellen suchen.
