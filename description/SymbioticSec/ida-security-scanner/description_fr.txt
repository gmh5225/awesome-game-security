Ce projet est un plugin IDA Pro qui automatise la découverte de vulnérabilités sur le pseudocode décompilé. Il est écrit en Python et exécute des analyses SAST basées sur des règles via des motifs YAML de type opengrep/semgrep, puis présente les résultats dans une interface IDA interactive. Le plugin propose également des explications assistées par IA en option pour faciliter le triage et la compréhension des problèmes découverts. Il s'adresse aux ingénieurs en reverse engineering, aux joueurs de CTF et aux auditeurs de sécurité qui souhaitent accélérer l'analyse de vulnérabilités dans les binaires.
