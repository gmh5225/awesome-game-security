Этот проект — плагин для IDA Pro, который автоматизирует поиск уязвимостей в декомпилированном псевдокоде. Он написан на Python и выполняет rule-based SAST-сканирование по YAML-шаблонам в стиле opengrep/semgrep, после чего показывает находки в интерактивном интерфейсе IDA. Плагин также поддерживает опциональные пояснения с помощью ИИ, помогающие приоритизировать и понимать обнаруженные проблемы. Он предназначен для реверс-инженеров, участников CTF и аудиторов безопасности, которым нужны более быстрые рабочие процессы анализа уязвимостей в бинарных файлах.
