Este proyecto es un framework de inyección de código nativo para Android que inyecta bibliotecas compartidas en procesos de aplicaciones Android en ejecución. Utiliza inyección basada en ptrace o enganche de Zygote para cargar archivos .so personalizados en los procesos de las aplicaciones objetivo, lo que permite modificar aplicaciones Android en tiempo de ejecución sin requerir root en algunas configuraciones. Está dirigido a investigadores de seguridad de Android y modders de juegos que realizan instrumentación de aplicaciones en tiempo de ejecución.
