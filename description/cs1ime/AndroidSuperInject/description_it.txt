Questo progetto è un framework di iniezione di codice nativo Android che inietta librerie condivise nei processi di applicazioni Android in esecuzione. Utilizza l'iniezione basata su ptrace o l'hooking di Zygote per caricare file .so personalizzati nei processi delle app di destinazione, consentendo la modifica a runtime delle applicazioni Android senza richiedere root in alcune configurazioni. È rivolto a ricercatori di sicurezza Android e modder di giochi che eseguono strumentazione delle applicazioni a runtime.
