Ce projet est un framework d'injection de code natif Android qui injecte des bibliothèques partagées dans des processus d'applications Android en cours d'exécution. Il utilise l'injection basée sur ptrace ou le hooking de Zygote pour charger des fichiers .so personnalisés dans les processus d'applications cibles, permettant la modification à l'exécution des applications Android sans root dans certaines configurations. Il s'adresse aux chercheurs en sécurité Android et aux moddeurs de jeux qui réalisent de l'instrumentation d'applications à l'exécution.
