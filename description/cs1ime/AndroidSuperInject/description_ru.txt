Этот проект — фреймворк внедрения нативного кода для Android, который внедряет разделяемые библиотеки в работающие процессы приложений Android. Он использует внедрение на основе ptrace или перехват Zygote для загрузки пользовательских .so-файлов в процессы целевых приложений, что позволяет изменять приложения Android во время выполнения без root в некоторых конфигурациях. Предназначен для исследователей безопасности Android и моддеров игр, выполняющих инструментацию приложений во время выполнения.
