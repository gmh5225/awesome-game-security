Dieses Projekt ist ein Framework zur nativen Code-Injektion unter Android, das Shared Libraries in laufende Android-Anwendungsprozesse einschleust. Es nutzt ptrace-basierte Injektion oder Zygote-Hooking, um benutzerdefinierte .so-Dateien in Ziel-App-Prozesse zu laden, und ermöglicht so in einigen Konfigurationen die Laufzeitmodifikation von Android-Anwendungen ohne Root. Es richtet sich an Android-Sicherheitsforscher und Game-Modder, die Laufzeit-Instrumentierung von Anwendungen durchführen.
