Questo progetto è BlackSun Framework, un framework di game hacking che fornisce infrastruttura per creare cheat di gioco con supporto multipiattaforma. Include astrazioni di lettura/scrittura della memoria, pattern scanning, utility di hooking, rendering overlay e livelli di comunicazione che funzionano con diversi metodi di accesso (user-mode, kernel, DMA). Il framework C++ modulare separa i backend di accesso dalla logica del cheat. È rivolto ai ricercatori di sicurezza nei videogiochi che studiano l'architettura dei framework per cheat e i pattern di accesso alla memoria multi-backend.
