Este proyecto es una técnica de renderizado de overlays en modo kernel de Windows que se engancha desde el espacio kernel a la canalización de composición del Desktop Window Manager (DWM). Intercepta las llamadas de renderizado DirectX de DWM a nivel de kernel para inyectar comandos de dibujo personalizados en el compositor de escritorio, creando overlays invisibles para la detección anti-cheat en modo usuario. El controlador en C demuestra el hooking de DWM a nivel de kernel para un renderizado de overlay encubierto. Está dirigido a investigadores de kernel que estudian los internos de la composición DWM y las técnicas de overlay sigilosas.
