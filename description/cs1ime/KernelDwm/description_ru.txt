Этот проект представляет собой технику рендеринга оверлеев в режиме ядра Windows, которая перехватывает конвейер композиции Desktop Window Manager (DWM) из пространства ядра. Он перехватывает вызовы рендеринга DirectX DWM на уровне ядра, чтобы внедрять пользовательские команды отрисовки в композитор рабочего стола, создавая оверлеи, невидимые для античита в пользовательском режиме. Драйвер на C демонстрирует перехват DWM на уровне ядра для скрытого рендеринга оверлеев. Он предназначен для исследователей ядра, изучающих внутренности композиции DWM и скрытные техники оверлеев.
