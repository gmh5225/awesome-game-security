Questo progetto è una tecnica di rendering overlay in modalità kernel Windows che si aggancia dallo spazio kernel alla pipeline di composizione del Desktop Window Manager (DWM). Intercetta le chiamate di rendering DirectX di DWM a livello kernel per iniettare comandi di disegno personalizzati nel compositor desktop, creando overlay invisibili al rilevamento anti-cheat in modalità utente. Il driver in C dimostra l'hooking DWM a livello kernel per il rendering overlay nascosto. È rivolto ai ricercatori kernel che studiano gli interni della composizione DWM e le tecniche di overlay stealth.
