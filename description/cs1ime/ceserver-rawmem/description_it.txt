Questo progetto è un'implementazione di server Cheat Engine (ceserver) che usa l'accesso grezzo alla memoria fisica invece delle API standard di memoria di processo. Implementa il protocollo di rete ceserver per connessioni remote a Cheat Engine, ma legge la memoria del processo target tramite accesso diretto alla memoria fisica (ad es. /dev/mem o DMA), aggirando le protezioni di accesso alla memoria a livello del sistema operativo e il monitoraggio anti-cheat. È rivolto a ricercatori di sicurezza che studiano configurazioni di Cheat Engine basate sulla memoria fisica e scenari di attacco DMA.
