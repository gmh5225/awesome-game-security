Este proyecto es una implementación de servidor de Cheat Engine (ceserver) que usa acceso en bruto a la memoria física en lugar de las API estándar de memoria de procesos. Implementa el protocolo de red ceserver para conexiones remotas de Cheat Engine, pero lee la memoria del proceso objetivo mediante acceso directo a la memoria física (p. ej., /dev/mem o DMA), eludiendo las protecciones de acceso a memoria a nivel del sistema operativo y la supervisión anti-cheat. Está orientado a investigadores de seguridad que estudian configuraciones de Cheat Engine basadas en memoria física y escenarios de ataque DMA.
