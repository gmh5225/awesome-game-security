Dieses Projekt ist eine Python-Bibliothek zum Parsen von Windows-Minidump-Dateien (.dmp). Sie extrahiert Prozessspeicherinhalte, Thread-Kontexte, Modullisten, Ausnahmeaufzeichnungen und Systeminformationen aus Minidump-Dateien, ohne Windows oder WinDbg zu benötigen. Die Bibliothek unterstützt sowohl vollständige als auch Mini-Dump-Formate und kann programmatisch für automatisierte Analysen oder die Extraktion von Anmeldeinformationen aus LSASS-Dumps verwendet werden. Sie richtet sich an Incident Responder, forensische Analysten und Penetrationstester, die mit Windows-Speicherabbildern in plattformübergreifenden Umgebungen arbeiten.
