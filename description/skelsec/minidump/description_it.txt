Questo progetto è una libreria Python per l’analisi dei file minidump di Windows (.dmp). Estrae il contenuto della memoria dei processi, i contesti dei thread, le liste dei moduli, i record delle eccezioni e le informazioni di sistema dai file minidump, senza richiedere Windows o WinDbg. La libreria supporta sia i formati di dump completi sia quelli mini e può essere usata in modo programmatico per analisi automatizzate o per l’estrazione di credenziali da dump LSASS. È rivolta a incident responder, analisti forensi e penetration tester che lavorano con dump di memoria Windows in ambienti multipiattaforma.
