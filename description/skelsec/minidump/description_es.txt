Este proyecto es una biblioteca de Python para analizar archivos minidump de Windows (.dmp). Extrae el contenido de la memoria de procesos, contextos de hilos, listas de módulos, registros de excepciones e información del sistema a partir de archivos minidump, sin requerir Windows ni WinDbg. La biblioteca admite formatos de volcado completo y mini, y puede usarse de forma programática para análisis automatizado o para la extracción de credenciales a partir de volcados de LSASS. Está dirigida a respondedores de incidentes, analistas forenses y testers de penetración que trabajan con volcados de memoria de Windows en entornos multiplataforma.
