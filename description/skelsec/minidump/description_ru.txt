Этот проект — библиотека на Python для разбора файлов Windows minidump (.dmp). Она извлекает содержимое памяти процессов, контексты потоков, списки модулей, записи исключений и системную информацию из файлов minidump без необходимости в Windows или WinDbg. Библиотека поддерживает как полные, так и мини-дампы и может использоваться программно для автоматизированного анализа или извлечения учётных данных из дампов LSASS. Она предназначена для специалистов по реагированию на инциденты, форензик-аналитиков и пентестеров, работающих с дампами памяти Windows в кроссплатформенных средах.
