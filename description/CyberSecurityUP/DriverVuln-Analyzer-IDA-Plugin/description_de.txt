Dieses Projekt ist ein Plugin zur statischen Analyse, mit dem potenziell verwundbare Windows-Kernel-Treiber in IDA triagiert werden können. Es extrahiert automatisch IOCTL-Werte, dekodiert CTL_CODE-Felder, hebt riskante METHOD_NEITHER-Nutzung hervor und kennzeichnet sensible Kernel-API-Muster. Das Werkzeug ist in Python geschrieben und kann konsolidierte JSON-Ergebnisse für nachgelagerte Analyse-Pipelines exportieren. Es wird vor allem von Reverse Engineers und Vulnerability Researchers zur Bewertung der Angriffsfläche von Treibern genutzt.
