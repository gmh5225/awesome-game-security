Questo progetto è un plugin di analisi statica per fare il triage in IDA dei driver del kernel Windows potenzialmente vulnerabili. Estrae automaticamente i valori IOCTL, decodifica i campi CTL_CODE, evidenzia l’uso rischioso di METHOD_NEITHER e segnala pattern sensibili di API del kernel. Lo strumento è scritto in Python e può esportare risultati JSON consolidati per pipeline di analisi a valle. È usato soprattutto da reverse engineer e ricercatori di vulnerabilità che valutano la superficie di attacco dei driver.
