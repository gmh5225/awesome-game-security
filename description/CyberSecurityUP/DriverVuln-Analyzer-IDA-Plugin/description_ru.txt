Этот проект — плагин статического анализа для triage потенциально уязвимых драйверов ядра Windows в IDA. Он автоматически извлекает значения IOCTL, декодирует поля CTL_CODE, подсвечивает рискованное использование METHOD_NEITHER и помечает чувствительные паттерны API ядра. Инструмент написан на Python и может экспортировать сводные JSON-результаты для последующих конвейеров анализа. В основном используется инженерами по реверс-инжинирингу и исследователями уязвимостей при оценке поверхности атаки драйверов.
