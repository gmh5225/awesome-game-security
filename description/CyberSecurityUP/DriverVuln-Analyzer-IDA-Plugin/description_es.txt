Este proyecto es un plugin de análisis estático para triar en IDA controladores del kernel de Windows potencialmente vulnerables. Extrae automáticamente valores IOCTL, decodifica campos CTL_CODE, resalta el uso arriesgado de METHOD_NEITHER y señala patrones sensibles de API del kernel. La herramienta está escrita en Python y puede exportar hallazgos JSON consolidados para pipelines de análisis posteriores. Se usa principalmente por ingenieros de ingeniería inversa e investigadores de vulnerabilidades que evalúan la superficie de ataque de controladores.
