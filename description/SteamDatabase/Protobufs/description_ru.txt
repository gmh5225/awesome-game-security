Этот проект — постоянно отслеживаемая коллекция определений protobuf, используемых Steam и играми Valve.
Репозиторий в основном состоит из файлов .proto, полученных из конвейеров обновлений и автоматизированных protobuf-дамперов.
Он помогает исследователям поддерживать актуальные схемы сообщений для анализа протоколов, сопровождения инструментов и реверс-инжиниринга игровых сетей.
В первую очередь это ресурс данных для разработчиков, работающих с внутренностями экосистемы Steam.
