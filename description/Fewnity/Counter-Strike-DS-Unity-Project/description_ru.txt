Этот проект — рабочее пространство инструментов Unity для поддержки разработки demake Counter-Strike для Nintendo DS. Он предоставляет рабочие процессы экспорта данных карт и геймплея для коллизий, лестниц, зон отсечения (culling), теневых зон, бомбовых зон и данных триггеров. Также включены инструменты генерации путевых точек и матриц pathfinding, а также сценовый пайплайн для экспорта спрайтов оружия. Основное назначение — помочь разработчикам поддерживать и итерировать DS-совместимый контент и логику вне основной кодовой базы игры.
