Ce projet est un espace de travail d’outils Unity destiné à soutenir le développement d’un demake Counter-Strike pour Nintendo DS. Il fournit des workflows d’export de données de cartes et de gameplay pour les collisions, les escaliers, les zones de culling, les zones d’ombre, les zones de bombe et les données de triggers. Il inclut aussi des outils pour générer des waypoints et des matrices de pathfinding, ainsi qu’un pipeline basé sur des scènes pour exporter les sprites d’armes. Le cas d’usage principal est d’aider les développeurs à maintenir et itérer du contenu et une logique compatibles DS en dehors du code de base du jeu.
