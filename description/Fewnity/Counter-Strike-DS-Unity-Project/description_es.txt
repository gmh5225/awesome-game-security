Este proyecto es un espacio de trabajo de herramientas Unity para apoyar el desarrollo de un demake de Counter-Strike para Nintendo DS. Proporciona flujos de exportación de datos de mapas y jugabilidad para colisiones, escaleras, zonas de culling, zonas de sombra, zonas de bomba y datos de triggers. También incluye herramientas para generar waypoints y matrices de pathfinding, además de un pipeline basado en escenas para exportar sprites de armas. El caso de uso principal es ayudar a los desarrolladores a mantener e iterar contenido y lógica compatibles con DS fuera del código base principal del juego.
