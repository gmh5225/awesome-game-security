Этот проект — demake Counter-Strike для Nintendo DS с поддержкой онлайн-мультиплеера на эмуляторе и реальном железе. Он реализован на C для инструментария devkitPro и Nitro Engine и включает игровые системы оружия, ботов, карт и сети. К кодовой базе прилагаются практические инструкции по настройке для устройств семейства DS, сетевой игры в эмуляторе и сборки из исходников. Основное применение — разработка homebrew FPS и технические эксперименты на ограниченном портативном железе.
