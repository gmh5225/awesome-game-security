Ce projet est un demake de Counter-Strike pour Nintendo DS, avec prise en charge du multijoueur en ligne sur émulateur et matériel réel. Il est implémenté en C pour la chaîne d'outils devkitPro et Nitro Engine, et inclut des systèmes de jeu pour les armes, les bots, les cartes et le réseau. Le code source est accompagné d'instructions pratiques de configuration pour les appareils de la famille DS, le réseau sous émulateur et les builds depuis les sources. Son usage principal est le développement de FPS homebrew et l'expérimentation technique sur du matériel portable contraint.
