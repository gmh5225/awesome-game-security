Este proyecto es un demake de Counter-Strike para Nintendo DS con soporte de multijugador en línea en emulador y hardware real. Está implementado en C para la cadena de herramientas de devkitPro y Nitro Engine, e incluye sistemas de juego para armas, bots, mapas y redes. El código base se acompaña de instrucciones prácticas de configuración para dispositivos de la familia DS, redes en emulador y compilaciones desde el código fuente. Su uso principal es el desarrollo de FPS homebrew y la experimentación técnica en hardware portátil con recursos limitados.
