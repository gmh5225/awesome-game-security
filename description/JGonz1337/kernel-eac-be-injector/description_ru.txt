Этот проект — kernel-assisted manual mapper, предназначенный для внедрения DLL в целевой игровой процесс.
Он сочетает обработчик команд на основе kernel-хуков с user-mode маппером, который выполняет relocation, разрешение импортов, запись секций и удалённый вызов DllMain.
Реализация включает процедуры выделения и предоставления kernel-памяти, хуки подмены указателей и этапы очистки для отображённых payload.
Проект в первую очередь предназначен для углублённого исследования устойчивых к античитам сценариев инъекции в Windows.
