Этот проект реализует фреймворк связи между ядром Windows и пользовательским режимом, маршрутизирующий команды через перехваченный указатель функции win32k. Драйвер и пользовательский клиент написаны на C++ и предоставляют операции вроде поиска базы процесса, поиска модулей, чтения и записи памяти, выделения, освобождения и изменения защиты. Используются пользовательские структуры запросов, строки с XOR-обфускацией и компактный управляющий протокол для обмена данными между ring 3 и ring 0. Код ориентирован на исследование обхода античита и инструментов работы с игровой памятью в защищённых средах.
