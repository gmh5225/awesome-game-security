Este proyecto implementa un framework de comunicación de Windows entre el kernel y el espacio de usuario que enruta comandos a través de un puntero a función de win32k enganchado (hooked). El controlador y el cliente de usuario están escritos en C++ y exponen operaciones como búsqueda de base de proceso, búsqueda de módulos, lectura y escritura de memoria, asignación, liberación y cambios de protección. Utiliza estructuras de solicitud personalizadas, cadenas ofuscadas con XOR y un protocolo de control compacto para intercambiar datos entre ring 3 y ring 0. El código está orientado a la investigación de bypass de anti-cheat y herramientas de memoria de juegos en entornos protegidos.
