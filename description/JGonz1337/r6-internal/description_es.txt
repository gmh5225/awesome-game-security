Este proyecto es una base de cheat interno en C++ para Rainbow Six Siege, centrada en el renderizado y en prototipos de funciones de gameplay.
Incluye un SDK pequeño, utilidades de hooking y un pipeline de overlay con ImGui y DirectX 11 para visuales en proceso.
Los ejemplos cubren ESP con snaplines, lógica básica de targeting, manejo de teclado y patrones de acceso a entidades.
El caso de uso principal es la experimentación educativa con la arquitectura del game hacking interno, no un bypass listo para producción.
