Ce projet est un module Magisk/KernelSU/APatch permettant de déplacer les certificats utilisateur vers les certificats système.
Ce scénario doit être préservé.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / Magisk.
