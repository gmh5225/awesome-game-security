本プロジェクトは、ユーザーモードから Windows 組み込みの Intel Processor Trace（IPT）ドライバと連携するための Rust ライブラリです。
現行の Windows IPT ドライバインタフェースのリバースエンジニアリングに基づき、ipt.sys ドライバへの DeviceIoControl 呼び出しを通じて、IPT トレースの開始・停止・取得用 API を提供します。
本ライブラリはドライバ連携のみに焦点を当てており、IPT トレースの解析やカバレッジ分析機能は含みません。
主に、Windows 上で Intel PT によるハードウェア支援コードトレーシングを研究するセキュリティ研究者やアンチチート分析者向けです。
