Этот проект — библиотека на Rust для взаимодействия из пользовательского режима со встроенным в Windows драйвером Intel Processor Trace (IPT).
Она предоставляет API для запуска, остановки и получения IPT-трассировок через вызовы DeviceIoControl к драйверу ipt.sys и основана на реверс-инжиниринге текущего интерфейса драйвера IPT в Windows.
Библиотека сосредоточена только на взаимодействии с драйвером и не включает разбор IPT-трассировок или анализ покрытия.
Она в первую очередь полезна исследователям безопасности и аналитикам античита, изучающим аппаратно поддерживаемое трассирование кода в Windows с помощью Intel PT.
