Ce projet est une bibliothèque Rust permettant d'interagir depuis le mode utilisateur avec le pilote Intel Processor Trace (IPT) intégré à Windows.
Elle fournit des API pour démarrer, arrêter et récupérer des traces IPT via des appels DeviceIoControl au pilote ipt.sys, basées sur la rétro-ingénierie de l'interface actuelle du pilote IPT Windows.
La bibliothèque se concentre uniquement sur l'interaction avec le pilote et n'inclut ni l'analyse des traces IPT ni les fonctionnalités de couverture.
Elle est principalement utile aux chercheurs en sécurité et aux analystes anti-triche qui étudient le traçage de code assisté par le matériel sur Windows via Intel PT.
