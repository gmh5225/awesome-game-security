Dieses Projekt ist eine Rust-Bibliothek zur Interaktion mit dem in Windows integrierten Intel-Processor-Trace-(IPT)-Treiber aus dem Benutzermodus.
Sie stellt APIs zum Starten, Stoppen und Abrufen von IPT-Traces über DeviceIoControl-Aufrufe an den ipt.sys-Treiber bereit und basiert auf der Reverse Engineering der aktuellen Windows-IPT-Treiberschnittstelle.
Die Bibliothek konzentriert sich ausschließlich auf die Treiberinteraktion und enthält keine Funktionen zur IPT-Trace-Analyse oder Coverage-Auswertung.
Sie ist vor allem für Sicherheitsforscher und Anti-Cheat-Analysten nützlich, die hardwaregestütztes Code-Tracing unter Windows über Intel PT untersuchen.
