Questo progetto è una libreria Rust per interagire in modalità utente con il driver Intel Processor Trace (IPT) integrato in Windows.
Fornisce API per avviare, arrestare e recuperare tracce IPT tramite chiamate DeviceIoControl al driver ipt.sys, basate sul reverse engineering dell'interfaccia attuale del driver IPT di Windows.
La libreria si concentra esclusivamente sull'interazione con il driver e non include il parsing delle tracce IPT né funzionalità di analisi della copertura.
È principalmente utile per ricercatori di sicurezza e analisti anti-cheat che studiano il tracciamento del codice assistito dall'hardware su Windows tramite Intel PT.
