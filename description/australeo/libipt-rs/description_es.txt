Este proyecto es una biblioteca Rust para interactuar desde el modo usuario con el controlador Intel Processor Trace (IPT) integrado en Windows.
Proporciona APIs para iniciar, detener y recuperar trazas IPT mediante llamadas DeviceIoControl al controlador ipt.sys, basándose en la ingeniería inversa de la interfaz actual del controlador IPT de Windows.
La biblioteca se centra únicamente en la interacción con el controlador y no incluye análisis de trazas IPT ni funcionalidad de cobertura.
Resulta especialmente útil para investigadores de seguridad y analistas de anti-trampas que estudian el trazado de código asistido por hardware en Windows mediante Intel PT.
