이 프로젝트는 사용자 모드에서 Windows 내장 Intel Processor Trace(IPT) 드라이버와 상호작용하기 위한 Rust 라이브러리입니다.
현재 Windows IPT 드라이버 인터페이스에 대한 리버스 엔지니어링을 바탕으로, ipt.sys 드라이버에 대한 DeviceIoControl 호출을 통해 IPT 트레이스를 시작·중지·가져오는 API를 제공합니다.
이 라이브러리는 드라이버 상호작용에만 초점을 맞추며, IPT 트레이스 파싱이나 커버리지 분석 기능은 포함하지 않습니다.
주로 Windows에서 Intel PT를 통한 하드웨어 지원 코드 트레이싱을 연구하는 보안 연구원과 안티치트 분석가에게 유용합니다.
