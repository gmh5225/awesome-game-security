Questo progetto è un loader di binari ELF che viene eseguito nella tua applicazione anziché come parte del runtime C.
La funzionalità più utile è che, a differenza di dlopen() standard, può caricare un file ELF da un percorso diverso dal filesystem.
È soprattutto utile per ricercatori di sicurezza nei giochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / caricamento in memoria su Android.
