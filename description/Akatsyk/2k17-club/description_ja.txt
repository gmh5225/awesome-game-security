このプロジェクトは、Counter-Strike: Global Offensive 向けの大規模な C++ チートフレームワークです。コードベースには、エイムボットロジック、アンチエイム、オートウォール計算、チャムス、グレネード補助、フックベースのゲームイベント処理といった一般的な攻撃モジュールが含まれています。また DirectX9 の ImGui レンダリングを組み込み、LuaBridge 経由で Lua または LuaJIT のスクリプト対応を提供し、設定可能なゲーム内ツールを実現しています。主にチート開発の研究と Source エンジンのリバースエンジニアリングを目的としています。
