Ce projet est un grand framework de cheat en C++ ciblant Counter-Strike: Global Offensive. La base de code contient des modules offensifs courants tels que la logique d'aimbot, l'anti-aim, les calculs autowall, les chams, les aides aux grenades et la gestion d'événements de jeu basée sur des hooks. Il intègre également le rendu ImGui sous DirectX9 et le support de scripts Lua ou LuaJIT via LuaBridge pour des outils en jeu configurables. Il vise principalement la recherche sur le développement de cheats et la rétro-ingénierie du moteur Source.
