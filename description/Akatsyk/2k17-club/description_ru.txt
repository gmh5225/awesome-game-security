Этот проект — крупный C++-фреймворк читов, нацеленный на Counter-Strike: Global Offensive. Кодовая база содержит типичные атакующие модули: логику аимбота, anti-aim, расчёты autowall, chams, помощники по гранатам и обработку игровых событий на основе хуков. Также встроен рендеринг ImGui через DirectX9 и поддержка скриптов Lua или LuaJIT через LuaBridge для настраиваемых внутриигровых инструментов. В основном предназначен для исследований в области разработки читов и реверс-инжиниринга движка Source.
