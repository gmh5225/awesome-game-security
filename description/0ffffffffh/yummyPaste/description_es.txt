Este proyecto es un complemento de depurador que pega datos binarios en bytes desde texto directamente en las vistas de memoria. Admite varios formatos de entrada, como matrices de bytes al estilo C, cadenas de shellcode con escape, decimales separados por comas y secuencias decimales separadas por espacios. La implementación está en C++ y se integra con el SDK de complementos de x64dbg para flujos de trabajo de ventanas de desensamblador y dump. Es útil para ingenieros de ingeniería inversa y desarrolladores de exploits que necesitan aplicar parches e inyectar bytes rápidamente durante el análisis.
