Questo progetto è un plugin per debugger che incolla dati binari in byte dal testo direttamente nelle viste di memoria. Supporta più formati di input come array di byte in stile C, stringhe di shellcode con escape, decimali separati da virgole e sequenze decimali separate da spazi. L'implementazione è in C++ e si integra con l'SDK dei plugin di x64dbg per i flussi di lavoro delle finestre disassembler e dump. È utile per reverse engineer e sviluppatori di exploit che necessitano di patching rapido e iniezione di byte durante l'analisi.
