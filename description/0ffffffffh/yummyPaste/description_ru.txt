Этот проект — плагин отладчика, который вставляет двоичные байтовые данные из текста напрямую в представления памяти. Он поддерживает несколько форматов ввода: байтовые массивы в стиле C, экранированные строки shellcode, десятичные числа через запятую и десятичные последовательности через пробел. Реализация написана на C++ и интегрируется с SDK плагинов x64dbg для рабочих процессов окон дизассемблера и dump. Плагин полезен реверс-инженерам и разработчикам эксплойтов, которым нужна быстрая правка и внедрение байтов во время анализа.
