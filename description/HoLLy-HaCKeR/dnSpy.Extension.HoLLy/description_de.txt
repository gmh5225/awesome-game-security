Dieses Projekt ist eine Erweiterung für dnSpyEx, die beim Reverse Engineering verschleierter .NET-Binärdateien hilft.
Sie ist in C# entwickelt und ergänzt Funktionen wie Source-Map-ähnliches Umbenennen, Injection verwalteter DLLs während des Debuggings sowie native Disassembly-Unterstützung.
Die Erweiterung bietet außerdem eine Visualisierung von Kontrollflussgraphen und Hilfsbefehle für die Erweiterungsentwicklung.
Sie ist nützlich für Reverse Engineers, Malware-Analysten und Game-Modder, die mit geschütztem Managed Code arbeiten.
