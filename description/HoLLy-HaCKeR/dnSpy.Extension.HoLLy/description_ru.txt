Этот проект — расширение для dnSpyEx, которое помогает анализировать обфусцированные .NET-бинарники.
Оно написано на C# и добавляет возможности вроде переименования в стиле source map, внедрения управляемых DLL во время отладки и поддержки нативной дизассемблеи.
Расширение также предоставляет визуализацию графа потока управления и служебные команды для разработки расширений.
Оно полезно реверс-инженерам, аналитикам вредоносного ПО и моддерам игр, работающим с защищённым управляемым кодом.
