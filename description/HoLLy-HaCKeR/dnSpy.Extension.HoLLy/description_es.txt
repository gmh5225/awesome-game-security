Este proyecto es una extensión para dnSpyEx que ayuda a realizar ingeniería inversa de binarios .NET ofuscados.
Está desarrollada en C# y añade capacidades como renombrado al estilo source map, inyección de DLL administradas durante la depuración y soporte de desensamblado nativo.
La extensión también proporciona visualización de grafos de flujo de control y comandos de utilidad para el desarrollo de extensiones.
Es útil para ingenieros inversos, analistas de malware y modders de juegos que trabajan con código administrado protegido.
