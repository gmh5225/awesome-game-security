Проект посвящён bYOVD: DsArk64.sys (Qihoo 360) — с подписью WHQL, завершение процессов из ring 0 и ядровое R/W. Скачайте любой установщик 360 с 360.cn (без аккаунта и CAPTCHA), запустите его приостановленным (CREATE_SUSPENDED) и внедрите shellcode через CreateRemoteThread; shellcode вызывает CreateFileW("\\.\DsArk"). Драйвер проверяет имя образа через IoGetCurrentProcess(), видит сертификат Qihoo и разрешает открытие. DuplicateHandle возвращает handle атакующему процессу, TerminateProcess завершает процесс-донор. В основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / уязвимых драйверов.
