Este proyecto es una prueba de concepto que engancha MiniDumpWriteDump para interceptar los datos del dump antes de que se escriban en disco. Está implementado principalmente en C++, con scripts auxiliares en Python para recibir y descifrar el contenido del dump transferido. El flujo de trabajo puede cifrar opcionalmente el búfer capturado y exfiltrarlo por un socket a un host remoto, demostrando cómo se pueden modificar en memoria las canalizaciones de dump. Está pensado para investigación de seguridad sobre técnicas de dumping de credenciales, ingeniería de detección y validación defensiva.
