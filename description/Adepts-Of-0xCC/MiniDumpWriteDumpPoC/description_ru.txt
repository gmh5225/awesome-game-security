Этот проект — доказательство концепции, которое хукает MiniDumpWriteDump, чтобы перехватывать данные дампа до их записи на диск. Реализован в основном на C++ с вспомогательными скриптами на Python для приёма и расшифровки переданного содержимого дампа. Рабочий процесс может при необходимости шифровать захваченный буфер и эксфильтрировать его по сокету на удалённый хост, демонстрируя, как конвейеры дампа можно изменять в памяти. Предназначен для исследований в области безопасности, связанных с техниками дампа учётных данных, инженерией обнаружения и проверкой защитных мер.
