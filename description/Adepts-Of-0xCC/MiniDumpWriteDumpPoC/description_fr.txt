Ce projet est une preuve de concept qui accroche MiniDumpWriteDump pour intercepter les données de dump avant leur écriture sur le disque. Il est principalement implémenté en C++, avec des scripts Python auxiliaires pour recevoir et déchiffrer le contenu de dump transféré. Le flux de travail peut éventuellement chiffrer le tampon capturé et l'exfiltrer via un socket vers un hôte distant, démontrant comment les pipelines de dump peuvent être modifiés en mémoire. Il est destiné à la recherche en sécurité sur les techniques de dumping d'identifiants, l'ingénierie de détection et la validation défensive.
