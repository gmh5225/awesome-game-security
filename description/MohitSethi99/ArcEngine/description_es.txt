Este proyecto es un motor de juego C++ personalizado con un editor y una capa de scripting asociados. Su árbol de código fuente muestra subsistemas de renderizado, gestión de escenas, física, audio, entrada y herramientas, junto con backends específicos de plataforma y assets de shaders. El repositorio también incluye un núcleo de scripts en C#, una configuración de build basada en premake, dependencias vendor y un proyecto sandbox para ejecutar contenido de ejemplo. Está pensado principalmente para experimentar con la arquitectura del motor y prototipar el desarrollo práctico de juegos.
