Questo progetto è un motore di gioco C++ personalizzato, con un editor e uno strato di scripting associati. L'albero dei sorgenti mostra sottosistemi per rendering, gestione delle scene, fisica, audio, input e tooling, oltre a backend specifici per piattaforma e asset di shader. Il repository include anche un core di script C#, una configurazione di build basata su premake, dipendenze vendor e un progetto sandbox per eseguire contenuti di esempio. È pensato principalmente per sperimentare l'architettura del motore e per il prototipaggio pratico nello sviluppo di giochi.
