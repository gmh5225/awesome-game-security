UE4Injector ist ein Proof-of-Concept-C++-Injector, der eine Unreal-Engine-4-Schwachstelle zum Laden von Shellcode- oder DLL-Payloads in Zielspielprozesse demonstriert.
Er wird als Visual-Studio-Projekt mit Befehlszeilen-Nutzungsmustern, Build-Hinweisen sowie praktischen Anmerkungen zu Berechtigungsanforderungen und Bereitstellung ausgeliefert.
Das Repository dokumentiert den Ablauf der Injection und stellt den Code als Legacy-Forschung dar, die ungepatchte UE4-Titel weiterhin betreffen kann.
Der Hauptanwendungsfall ist die Sicherheitsforschung zu UE4-Prozessinjektionsvektoren und ihren Auswirkungen auf Anti-Cheat-Systeme.
