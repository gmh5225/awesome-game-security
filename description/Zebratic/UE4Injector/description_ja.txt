UE4Injector は、Unreal Engine 4 の脆弱性を利用して shellcode や DLL ペイロードを対象ゲームプロセスに読み込ませることを示す、概念実証の C++ インジェクターです。
Visual Studio プロジェクトとして提供され、コマンドラインの使用例、ビルド手順、権限要件とデプロイに関する実践的な注意事項が含まれます。
リポジトリではインジェクションのワークフローを解説し、コードを未修正の UE4 タイトルに今なお影響し得るレガシー研究として位置づけています。
主な用途は、UE4 のプロセスインジェクション手法とそのアンチチートへの影響に関するセキュリティ研究です。
