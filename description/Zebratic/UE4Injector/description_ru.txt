UE4Injector — это концептуальный C++-инжектор, демонстрирующий уязвимость Unreal Engine 4 для загрузки shellcode или DLL-нагрузок в целевые игровые процессы.
Он поставляется как проект Visual Studio с примерами использования из командной строки, рекомендациями по сборке и практическими заметками о требованиях к привилегиям и развёртывании.
Репозиторий описывает работу процесса внедрения и представляет код как устаревшее исследование, которое всё ещё может затрагивать незапатченные проекты на UE4.
Основной сценарий применения — исследование безопасности векторов внедрения в процессы UE4 и их последствий для античита.
