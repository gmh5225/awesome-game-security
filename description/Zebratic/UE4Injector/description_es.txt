UE4Injector es un inyector C++ de prueba de concepto que demuestra una vulnerabilidad de Unreal Engine 4 para cargar shellcode o cargas útiles DLL en procesos de juegos objetivo.
Se distribuye como un proyecto de Visual Studio con patrones de uso por línea de comandos, orientación para compilar y notas prácticas sobre requisitos de privilegios y despliegue.
El repositorio documenta el flujo de trabajo de inyección y presenta el código como investigación heredada que aún puede afectar a títulos UE4 sin parchear.
Su caso de uso principal es la investigación de seguridad sobre vectores de inyección de procesos en UE4 y sus implicaciones para el anti-cheat.
