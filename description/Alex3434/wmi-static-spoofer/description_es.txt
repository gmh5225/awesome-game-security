Este proyecto es una prueba de concepto en modo kernel para spoofear de forma estática la información de serie del hardware expuesta a través de WMI y rutas relacionadas. Usa manipulación directa de memoria y actualizaciones del registro en lugar de hooks persistentes, lo que permite descargar el controlador una vez aplicados los cambios. La implementación incluye offsets configurables y generación aleatoria de números de serie para probar distintos entornos. Se usa principalmente en investigación de evasión de HWID frente a anti-cheat y telemetría de licencias.
