Este proyecto es una implementación en C++ de una lógica de detección de máquinas virtuales derivada de un modelo de aprendizaje automático. Scripts de Python usan scikit-learn para entrenar y ajustar un clasificador de árbol de decisión, y las reglas resultantes se traducen a comprobaciones nativas en C++. Según el resultado de la detección, la muestra puede ejecutar comportamientos alternativos como ejecución de shellcode o autoeliminación. Está pensado para investigar técnicas de conciencia de sandbox y conceptos de análisis de malware asistido por automatización.
