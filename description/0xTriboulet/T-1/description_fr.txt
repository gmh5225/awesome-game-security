Ce projet est une implémentation C++ d'une logique de détection de machines virtuelles dérivée d'un modèle d'apprentissage automatique. Des scripts Python utilisent scikit-learn pour entraîner et ajuster un classifieur en arbre de décision, puis les règles obtenues sont traduites en contrôles C++ natifs. Selon le résultat de la détection, l'échantillon peut exécuter des comportements alternatifs tels que l'exécution de shellcode ou l'auto-suppression. Il est destiné à la recherche sur les techniques de conscience de sandbox et les concepts d'analyse de malware assistée par l'automatisation.
