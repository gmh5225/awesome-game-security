Этот проект представляет собой реализацию на C++ логики обнаружения виртуальных машин, полученной из модели машинного обучения. Скрипты на Python используют scikit-learn для обучения и настройки классификатора на основе дерева решений, после чего полученные правила переводятся в нативные проверки на C++. В зависимости от результата обнаружения образец может выполнять альтернативное поведение, например запуск shellcode или самоудаление. Проект предназначен для исследования техник осведомлённости о песочнице и концепций автоматизированного анализа вредоносного ПО.
