Questo progetto è un'implementazione in C++ di una logica di rilevamento di macchine virtuali derivata da un modello di machine learning. Script Python usano scikit-learn per addestrare e ottimizzare un classificatore ad albero decisionale, quindi le regole risultanti vengono tradotte in controlli nativi in C++. In base all'esito del rilevamento, il campione può eseguire comportamenti alternativi come l'esecuzione di shellcode o l'auto-rimozione. È pensato per la ricerca su tecniche di consapevolezza delle sandbox e concetti di analisi malware assistita dall'automazione.
