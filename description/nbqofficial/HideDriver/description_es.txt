Este proyecto se centra en ocultar controladores modificando Flink/Blink.
Puede resultar extremadamente útil para quienes programan software tipo rootkit e intentan eludir ciertos anticheats para circunvenir la seguridad.
Es principalmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheat / ocultación.
