Este proyecto se centra en un cheat de kernel sencillo con un hook para la comunicación.
Está escrito principalmente en C++ y se enfoca en el trabajo a nivel de kernel, el desarrollo de drivers y el modding.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área cheat / game:csgo.
