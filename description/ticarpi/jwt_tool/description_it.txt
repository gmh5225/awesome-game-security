Questo progetto è jwt_tool, un toolkit Python per testare, manipolare e crackare i JSON Web Token (JWT). Supporta la decodifica dei token, la modifica di claim e header, e il test di vulnerabilità comuni, tra cui confusione di algoritmi (none/HS256/RS256), attacchi di confusione delle chiavi, injection JKU/X5U e brute-force di chiavi di firma deboli. Lo strumento include modalità di scansione automatizzata per i test di sicurezza delle API. È rivolto a penetration tester e ricercatori di sicurezza web che auditano implementazioni di autenticazione basate su JWT.
