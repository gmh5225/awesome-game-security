Ce projet est jwt_tool, une boîte à outils Python pour tester, altérer et cracker les JSON Web Tokens (JWT). Elle permet de décoder les jetons, de modifier les claims et les en-têtes, et de tester les vulnérabilités courantes, notamment la confusion d'algorithme (none/HS256/RS256), les attaques par confusion de clés, l'injection JKU/X5U et le force brute des clés de signature faibles. L'outil inclut des modes d'analyse automatisée pour les tests de sécurité des API. Il s'adresse aux testeurs d'intrusion et aux chercheurs en sécurité web qui auditent les implémentations d'authentification basées sur JWT.
