Dieses Projekt ist jwt_tool, ein Python-Toolkit zum Testen, Manipulieren und Knacken von JSON Web Tokens (JWT). Es unterstützt das Dekodieren von Tokens, das Ändern von Claims und Headern sowie das Prüfen gängiger Schwachstellen, darunter Algorithmus-Verwirrung (none/HS256/RS256), Key-Confusion-Angriffe, JKU/X5U-Injection und Brute-Force schwacher Signaturschlüssel. Das Tool bietet automatisierte Scan-Modi für API-Sicherheitstests. Es richtet sich an Penetrationstester und Web-Security-Forscher, die JWT-basierte Authentifizierungsimplementierungen prüfen.
