Este proyecto es jwt_tool, un kit de herramientas en Python para probar, manipular y crackear JSON Web Tokens (JWT). Permite decodificar tokens, modificar claims y cabeceras, y probar vulnerabilidades comunes, incluidas la confusión de algoritmos (none/HS256/RS256), ataques de confusión de claves, inyección JKU/X5U y fuerza bruta de claves de firma débiles. La herramienta incluye modos de escaneo automatizado para pruebas de seguridad de API. Está orientada a pentesters e investigadores de seguridad web que auditan implementaciones de autenticación basadas en JWT.
