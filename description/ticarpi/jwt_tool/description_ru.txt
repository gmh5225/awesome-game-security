Этот проект — jwt_tool, Python-инструментарий для тестирования, подмены и взлома JSON Web Token (JWT). Он поддерживает декодирование токенов, изменение claims и заголовков, проверку типичных уязвимостей, включая путаницу алгоритмов (none/HS256/RS256), атаки с путаницей ключей, инъекции JKU/X5U и перебор слабых ключей подписи. Инструмент включает режимы автоматического сканирования для тестирования безопасности API. Предназначен для пентестеров и исследователей веб-безопасности, аудирующих реализации аутентификации на основе JWT.
