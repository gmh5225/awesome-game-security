Этот проект — uWebSockets, высокопроизводительная библиотека WebSocket- и HTTP-сервера на C++ с привязками для Node.js. Она реализует протокол WebSocket с минимальными накладными расходами, обеспечивая миллионы соединений на сервер и задержку на уровне микросекунд. Библиотека поддерживает TLS через OpenSSL/BoringSSL, HTTP-маршрутизацию, обмен сообщениями pub/sub и обработку обратного давления на уровне сокета. Она предназначена для разработчиков игровых серверов и backend-инженеров, создающих высоконагруженные сервисы сетевого взаимодействия в реальном времени.
