Ce projet est TrollStore, un installateur d'applications iOS jailed à signature permanente qui exploite des bugs CoreTrust et AMFI pour installer des fichiers IPA avec des entitlements arbitraires sans jailbreak. Les applications installées persistent après redémarrage et ne nécessitent pas de re-signature. Cet outil Objective-C s'appuie sur une chaîne de vulnérabilités noyau propre à certaines versions d'iOS pour contourner le contrôle de signature de code tout en restant dans le bac à sable jailed. Il s'adresse aux chercheurs en sécurité iOS étudiant le contournement de signature de code et aux utilisateurs souhaitant sideloader des apps sans certificats développeur.
