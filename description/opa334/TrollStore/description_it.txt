Questo progetto è TrollStore, un installer di app jailed con firma permanente per iOS che sfrutta bug di CoreTrust e AMFI per installare file IPA con entitlement arbitrari senza jailbreak. Le app installate persistono dopo i riavvii e non richiedono una nuova firma. Lo strumento in Objective-C sfrutta una catena di vulnerabilità del kernel specifica di alcune versioni di iOS per aggirare l'enforcement della code signing restando nel sandbox jailed. È rivolto ai ricercatori di sicurezza iOS che studiano il bypass della code signing e agli utenti che vogliono fare sideload di app senza certificati sviluppatore.
