Этот проект — TrollStore, постоянно подписанный установщик jailed-приложений для iOS, который использует уязвимости CoreTrust и AMFI для установки IPA с произвольными entitlements без джейлбрейка. Установленные приложения сохраняются после перезагрузки и не требуют повторной подписи. Инструмент на Objective-C опирается на цепочку уязвимостей ядра, характерную для определённых версий iOS, чтобы обойти принудительную проверку подписи кода, оставаясь в jailed-песочнице. Он предназначен для исследователей безопасности iOS, изучающих обход code signing, и пользователей, желающих устанавливать приложения без сертификатов разработчика.
