Ce projet est Dopamine, un jailbreak rootless semi-untethered pour iOS 15.0–15.4.1 sur tous les appareils. Il enchaîne plusieurs vulnérabilités du noyau pour obtenir une lecture/écriture arbitraire du noyau, contourne les protections PAC et PPL, patch AMFI pour l’exécution de code non signé, et installe Sileo ainsi qu’un bootstrap procursus dans une disposition rootless sous /var/jb. Le code Swift et Objective-C inclut une application GUI soignée pour déclencher l’exploit. Il s’adresse aux chercheurs en sécurité iOS et aux membres de la communauté jailbreak qui étudient l’exploitation moderne du noyau iOS et l’architecture des jailbreaks rootless.
