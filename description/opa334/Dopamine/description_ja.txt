本プロジェクトは Dopamine で、iOS 15.0–15.4.1 の全デバイス向け semi-untethered な rootless ジェイルブレイクです。複数のカーネル脆弱性を連鎖させて任意のカーネル読み書きを実現し、PAC と PPL 保護を迂回し、AMFI をパッチして未署名コード実行を可能にし、/var/jb 配下の rootless レイアウトに Sileo と procursus bootstrap をインストールします。Swift と Objective-C のコードベースには、エクスプロイト起動用の洗練された GUI アプリが含まれます。現代の iOS カーネルエクスプロイトと rootless ジェイルブレイク構成を研究するセキュリティ研究者およびジェイルブレイクコミュニティ向けです。
