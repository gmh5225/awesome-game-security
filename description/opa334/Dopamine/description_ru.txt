Этот проект — Dopamine, semi-untethered rootless джейлбрейк для iOS 15.0–15.4.1 на всех устройствах. Он объединяет несколько уязвимостей ядра для произвольного чтения/записи ядра, обходит защиты PAC и PPL, патчит AMFI для выполнения неподписанного кода и устанавливает Sileo и bootstrap procursus в rootless-раскладке под /var/jb. Кодовая база на Swift и Objective-C включает отполированное GUI-приложение для запуска эксплойта. Предназначен для исследователей безопасности iOS и участников сообщества джейлбрейка, изучающих современную эксплуатацию ядра iOS и архитектуру rootless-джейлбрейков.
