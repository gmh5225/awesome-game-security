Este proyecto es Dopamine, un jailbreak rootless semi-untethered para iOS 15.0–15.4.1 en todos los dispositivos. Encadena múltiples vulnerabilidades del kernel para lograr lectura/escritura arbitraria del kernel, elude las protecciones PAC y PPL, parchea AMFI para la ejecución de código sin firmar e instala Sileo y un bootstrap procursus en un diseño rootless bajo /var/jb. El código en Swift y Objective-C incluye una aplicación GUI pulida para activar el exploit. Está dirigido a investigadores de seguridad de iOS y miembros de la comunidad jailbreak que estudian la explotación moderna del kernel de iOS y la arquitectura de jailbreaks rootless.
