Ce projet est opainject, un outil en ligne de commande pour injecter des bibliothèques dynamiques (dylibs) dans des processus en cours d'exécution sur iOS/macOS. Il utilise task_for_pid et les API de threads Mach pour créer un thread distant dans le processus cible qui charge le dylib spécifié via dlopen. Cet outil Objective-C fonctionne sur les appareils jailbreakés disposant d'un accès tfp0. Il s'adresse aux développeurs de jailbreak iOS et aux chercheurs en sécurité qui ont besoin d'injection de dylib à l'exécution pour le développement de tweaks et l'analyse de processus.
