Этот проект — opainject, инструмент командной строки для внедрения динамических библиотек (dylib) в работающие процессы на iOS/macOS. Он использует task_for_pid и API потоков Mach для создания удалённого потока в целевом процессе, который загружает указанный dylib через dlopen. Инструмент на Objective-C работает на устройствах с джейлбрейком при наличии доступа к tfp0. Он предназначен для разработчиков джейлбрейка iOS и исследователей безопасности, которым нужна runtime-инъекция dylib для разработки твиков и анализа процессов.
