Este proyecto es opainject, una herramienta de línea de comandos para inyectar bibliotecas dinámicas (dylibs) en procesos en ejecución en iOS/macOS. Usa task_for_pid y las API de hilos Mach para crear un hilo remoto en el proceso objetivo que carga el dylib especificado mediante dlopen. La herramienta en Objective-C funciona en dispositivos con jailbreak y acceso a tfp0. Está dirigida a desarrolladores de jailbreak de iOS e investigadores de seguridad que necesitan inyección de dylib en tiempo de ejecución para el desarrollo de tweaks y el análisis de procesos.
