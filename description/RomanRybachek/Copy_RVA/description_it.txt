Questo progetto è un plugin leggero per IDA Pro che copia l’RVA sotto il cursore negli appunti. È implementato come script Python e si integra nel flusso di lavoro dei plugin IDA e nel menu contestuale. Lo strumento aiuta a impostare breakpoint in WinDbg per driver senza simboli pubblici, convertendo rapidamente le posizioni in RVA utilizzabili. È pensato per reverse engineer e ricercatori di sicurezza del kernel o dei giochi che necessitano di una gestione più rapida degli offset.
