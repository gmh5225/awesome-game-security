Este proyecto es un plugin ligero de IDA Pro que copia el RVA bajo el cursor al portapapeles. Está implementado como un script de Python que se integra en el flujo de trabajo de plugins de IDA y en el menú contextual. La herramienta ayuda a establecer puntos de interrupción en WinDbg para controladores sin símbolos públicos, convirtiendo rápidamente ubicaciones en RVAs utilizables. Está orientada a ingenieros inversos e investigadores de seguridad de kernel o de juegos que necesitan manejar offsets con más rapidez.
