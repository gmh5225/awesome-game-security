Dieses Projekt ist ein Desktop-Dienstprogramm zum Senden benutzerdefinierter IOCTL-Anfragen an Windows-Treiber. Es ist in C++ mit Qt Widgets geschrieben und enthält eine integrierte Hex-Editor-Oberfläche zum Erstellen von Eingabepuffern und zur Prüfung von Ausgabedaten. Die Anwendung unterstützt das Öffnen mehrerer Geräteobjekte, die Verwaltung von Handles sowie DeviceIoControl-Aufrufe mit konfigurierbaren Steuerungscodes und Puffergrößen. Sie eignet sich für Treiber-Reverse-Engineering, Tests der Kernel-Kommunikation und Workflows in der Spiele-Sicherheitsforschung.
