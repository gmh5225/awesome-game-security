Этот проект — настольная утилита для отправки пользовательских IOCTL-запросов драйверам Windows. Она написана на C++ с Qt Widgets и включает встроенный интерфейс hex-редактора для формирования входных буферов и просмотра выходных данных. Приложение поддерживает открытие нескольких объектов устройств, управление хэндлами и вызовы DeviceIoControl с настраиваемыми кодами управления и размерами буферов. Полезно для реверса драйверов, тестирования взаимодействия с ядром и рабочих процессов исследований игровой безопасности.
