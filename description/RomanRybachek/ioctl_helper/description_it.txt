Questo progetto è un'utilità desktop per inviare richieste IOCTL personalizzate ai driver di Windows. È sviluppata in C++ con Qt Widgets e include un'interfaccia di editor esadecimale integrata per creare buffer di input e ispezionare i dati di output. L'applicazione supporta l'apertura di più oggetti dispositivo, la gestione degli handle e l'emissione di chiamate DeviceIoControl con codici di controllo e dimensioni dei buffer configurabili. È utile per il reverse engineering dei driver, i test di comunicazione con il kernel e i flussi di lavoro di ricerca sulla sicurezza dei giochi.
