Este proyecto es una utilidad de escritorio para enviar solicitudes IOCTL personalizadas a controladores de Windows. Está construida en C++ con Qt Widgets e incluye una interfaz de editor hexadecimal integrada para preparar búferes de entrada e inspeccionar datos de salida. La aplicación admite abrir varios objetos de dispositivo, gestionar handles y emitir llamadas DeviceIoControl con códigos de control y tamaños de búfer configurables. Es útil para el reverse engineering de controladores, pruebas de comunicación con el kernel y flujos de trabajo de investigación en seguridad de juegos.
