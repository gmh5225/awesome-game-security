Ce projet est un obfuscateur de code léger basé sur LLVM qui applique des passes de transformation simples mais efficaces aux binaires compilés via le pipeline LLVM IR. Il implémente la substitution d'instructions, l'aplatissement du flot de contrôle et le chiffrement de chaînes sous forme de passes d'optimisation LLVM pouvant s'intégrer aux workflows de build standards. Les passes C++ opèrent sur le LLVM IR, ce qui les rend indépendantes du langage et de l'architecture. Il s'adresse aux chercheurs en protection logicielle qui étudient les transformations d'obfuscation de base et leur implémentation en tant que passes de compilateur.
