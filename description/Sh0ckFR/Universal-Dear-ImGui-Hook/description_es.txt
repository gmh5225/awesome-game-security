Este proyecto es un hook gráfico de Windows que inyecta una superposición Dear ImGui en aplicaciones mediante varios backends de renderizado. Está escrito en C++ y apunta a Direct3D 9, 10, 11 y 12, con soporte parcial de Vulkan mediante implementaciones de hooks específicas de cada backend. El código integra componentes comunes de hooking y manejo de entrada para que los usuarios puedan activar o desactivar un menú de interfaz in-process tras la inyección de DLL. Se usa principalmente para prototipar superposiciones de juegos, interfaces de depuración e investigación de pipelines de renderizado.
