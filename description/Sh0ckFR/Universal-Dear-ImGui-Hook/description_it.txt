Questo progetto è un hook grafico per Windows che inietta un overlay Dear ImGui nelle applicazioni usando più backend di rendering. È scritto in C++ e mira a Direct3D 9, 10, 11 e 12, con supporto parziale a Vulkan tramite implementazioni di hook specifiche per backend. Il codice integra componenti di hooking comuni e la gestione dell'input, così gli utenti possono attivare o disattivare un menu UI in-process dopo l'iniezione della DLL. È usato principalmente per la prototipazione di overlay di gioco, interfacce di debug e la ricerca sui pipeline di rendering.
