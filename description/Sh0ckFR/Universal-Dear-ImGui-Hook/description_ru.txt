Этот проект — графический хук для Windows, который внедряет оверлей Dear ImGui в приложения с использованием нескольких бэкендов рендеринга. Он написан на C++ и рассчитан на Direct3D 9, 10, 11 и 12, с частичной поддержкой Vulkan через реализации хуков для конкретных бэкендов. Код объединяет типичные компоненты хукинга и обработку ввода, позволяя пользователям включать и выключать внутрипроцессное UI-меню после инъекции DLL. В основном применяется для прототипирования игровых оверлеев, отладочных интерфейсов и исследования конвейеров рендеринга.
