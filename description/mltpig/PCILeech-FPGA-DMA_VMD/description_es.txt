Firmware FPGA PCILeech para Xilinx Artix-7 XC7A75T que emula un controlador Intel RST VMD (Volume Management Device) con Device ID 9A0B para acceso a memoria basado en DMA. Implementa en SystemVerilog la simulación de dispositivos PCI multifunción, el manejo de interrupciones MSI-X, el procesamiento de comandos NVMe, el direccionamiento BAR configurable y el shadowing avanzado del espacio de configuración PCIe a nivel TLP.
