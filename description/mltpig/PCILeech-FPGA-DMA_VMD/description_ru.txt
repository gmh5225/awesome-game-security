Прошивка FPGA PCILeech для Xilinx Artix-7 XC7A75T, эмулирующая контроллер Intel RST VMD (Volume Management Device) с Device ID 9A0B для доступа к памяти на основе DMA. Реализует на SystemVerilog симуляцию многофункционального устройства PCI, обработку прерываний MSI-X, обработку команд NVMe, настраиваемую адресацию BAR и продвинутое теневое копирование (shadowing) пространства конфигурации PCIe на уровне TLP.
