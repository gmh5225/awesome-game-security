UnrealLive2D è un plugin runtime di Unreal Engine per integrare modelli Live2D Cubism nei progetti UE. Il codice è principalmente in C++ con codice shader Unreal e include moduli framework per il caricamento dei modelli, effetti, riproduzione delle motion, fisica e rendering. Include asset di esempio e contenuti Blueprint che dimostrano espressioni, lip sync, battito delle palpebre e flussi di animazione interattivi. L'uso principale è il rendering e l'animazione di personaggi 2D nei workflow di sviluppo di giochi.
