UnrealLive2D est un plugin runtime Unreal Engine permettant d'intégrer des modèles Live2D Cubism dans des projets UE. Le code source est principalement en C++ avec du code de shaders Unreal, et comprend des modules de framework pour le chargement de modèles, les effets, la lecture de motions, la physique et le rendu. Il est livré avec des assets d'exemple et du contenu Blueprint qui illustrent les expressions, le lip sync, le clignement des yeux et des flux d'animation interactifs. Le cas d'usage principal est le rendu et l'animation de personnages 2D dans les workflows de développement de jeux.
