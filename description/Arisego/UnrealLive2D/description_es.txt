UnrealLive2D es un plugin de runtime de Unreal Engine para integrar modelos Live2D Cubism en proyectos de UE. El código base es principalmente C++ con código de shaders de Unreal e incluye módulos de framework para carga de modelos, efectos, reproducción de motion, física y renderizado. Incluye assets de ejemplo y contenido Blueprint que demuestran expresiones, sincronización labial, parpadeo y flujos de animación interactivos. El caso de uso principal es el renderizado y la animación de personajes 2D en flujos de trabajo de desarrollo de juegos.
