Ce projet est une preuve de concept d'anti-cheat pour Windows qui illustre les techniques de détection courantes utilisées par les logiciels de protection de jeux. Il implémente des contrôles de présence de débogueur, une vérification d'intégrité du code, un scan mémoire des signatures de triches connues, et une énumération des processus pour détecter les outils suspects. La base de code C/C++ sert d'implémentation de référence montrant comment les moteurs anti-cheat détectent et réagissent à la falsification. Il s'adresse aux chercheurs en sécurité des jeux qui étudient l'architecture anti-cheat et les méthodologies de détection.
