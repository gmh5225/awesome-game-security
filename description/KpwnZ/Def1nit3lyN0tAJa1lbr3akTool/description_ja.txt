本プロジェクトは、iOS 16.0 から 16.6.1 までのデバイスを対象とした iOS ジェイルブレイクツールです。
最新のコミュニティによるジェイルブレイクおよびカーネル脆弱性利用の研究成果を基盤とし、異なる trustcache バージョン向けのビルド手順を含みます。
コードベースには低レベルの iOS コンポーネント、豊富なネイティブヘッダー、ならびに Xcode と make ベースのビルドワークフローが含まれます。
主な実装言語は Objective-C および C 系統のネイティブコードです。
ポストエクスプロイトのワークフローを研究する上級の iOS セキュリティ研究者およびジェイルブレイク開発者を対象としています。
