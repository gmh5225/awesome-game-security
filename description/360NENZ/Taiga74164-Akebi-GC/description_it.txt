Questo progetto è un ampio framework C++ di modifica per Genshin Impact, con un injector e un sistema di menu in-game.
Offre un’ampia gamma di funzionalità, tra cui routine di bypass delle protezioni, modificatori di giocatore e mondo, ausili al teletrasporto, visuali in stile ESP e utility di debugging.
Il repository include una quantità sostanziale di codice nativo, asset e documentazione multilingue, con release pubbliche distribuite anche quando le revisioni del codice sorgente più recenti sono diventate più restrittive.
È usato soprattutto da ricercatori di game hacking e sicurezza dei giochi che studiano tecniche di manipolazione lato client nei titoli basati su Unity.
