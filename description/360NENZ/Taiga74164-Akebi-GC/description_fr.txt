Ce projet est un vaste framework C++ de modification de jeu pour Genshin Impact, avec un injecteur et un système de menu en jeu.
Il offre un large éventail de fonctionnalités : routines de contournement des protections, modificateurs de joueur et de monde, aides au téléport, visuels de type ESP et utilitaires de débogage.
Le dépôt contient un volume important de code natif, d’assets et de documentation multilingue, avec des versions publiques distribuées même lorsque les révisions sources plus récentes sont devenues plus restreintes.
Il est principalement utilisé par les chercheurs en game hacking et en sécurité des jeux qui étudient les techniques de manipulation côté client dans les titres basés sur Unity.
