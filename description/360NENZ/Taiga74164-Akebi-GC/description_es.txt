Este proyecto es un amplio framework C++ de modificación de juegos para Genshin Impact, con un inyector y un sistema de menú dentro del juego.
Ofrece un amplio conjunto de funciones, incluidas rutinas de bypass de protección, modificadores de jugador y del mundo, ayudas de teletransporte, visuales al estilo ESP y utilidades de depuración.
El repositorio incluye una cantidad sustancial de código nativo, recursos y documentación multilingüe, con versiones públicas distribuidas incluso cuando las revisiones de código fuente más recientes se volvieron más restringidas.
Lo usan principalmente investigadores de game hacking y seguridad de juegos que estudian técnicas de manipulación del cliente en títulos basados en Unity.
