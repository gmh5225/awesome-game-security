Este proyecto es un perfilador CPU y GPU en tiempo real ligero, entregado como una sola biblioteca C más un visor basado en navegador. Admite instrumentación multihilo, canales de comandos y registros en vivo, reproducción de trazas guardadas y visualización de alta tasa de refresco con UI renderizada por GPU. El perfilador funciona en las principales plataformas de escritorio y móviles y puede muestrear APIs como D3D11/12, OpenGL, Vulkan, CUDA y Metal. Está pensado para programadores de motores e ingenieros de rendimiento que necesitan perfilado de bajo overhead en juegos o aplicaciones intensivas en gráficos.
