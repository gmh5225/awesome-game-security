Questo progetto è un profiler CPU e GPU in tempo reale leggero, fornito come una singola libreria C più un viewer basato su browser. Supporta strumentazione multithread, canali di comandi e log in tempo reale, riproduzione di trace salvate e visualizzazione ad alto refresh con UI renderizzata dalla GPU. Il profiler funziona sulle principali piattaforme desktop e mobile e può campionare API tra cui D3D11/12, OpenGL, Vulkan, CUDA e Metal. È pensato per programmatori di engine e performance engineer che necessitano di profiling a basso overhead in giochi o applicazioni intensive dal punto di vista grafico.
