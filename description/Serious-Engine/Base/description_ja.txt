本プロジェクトは、クラシックな Serious Sam 世代で使われたオープンソースの Croteam ゲームエンジンのコアソースコードを提供します。主に C と C++ で書かれており、レンダリング、ワールド／ブラシ編集、エンティティ、ネットワーキング、数学、オーディオ統合、スクリプト支援などの主要サブシステムを含みます。ツリーには Visual Studio プロジェクトファイルのほか、アセットとサウンド処理向けの Ogg や Vorbis といったサードパーティ製メディアライブラリも含まれます。レガシー FPS エンジンの内部を研究・拡張したいエンジンプログラマ、リバースエンジニア、モッダーに主に役立ちます。
