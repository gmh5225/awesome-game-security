Este proyecto proporciona el código fuente principal de un motor de juego Croteam de código abierto usado por la generación clásica de Serious Sam. Está escrito sobre todo en C y C++ e incluye subsistemas importantes de renderizado, edición de mundos y brushes, entidades, red, matemáticas, integración de audio y soporte de scripting. El árbol también incluye archivos de proyecto de Visual Studio y bibliotecas multimedia de terceros como Ogg y Vorbis para el manejo de assets y sonido. Es especialmente útil para programadores de motores, ingenieros inversos y modders que quieran estudiar o ampliar los internos de un motor FPS legado.
