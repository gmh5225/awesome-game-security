Полиморфный движок мутаций для mimikatz, который использует дизассемблирование, анализ CFG, морфинг на уровне инструкций (переименование регистров, вставка мусорных инструкций, эквивалентная подстановка) и обфускацию потока данных для генерации уникальных вариантов, уклоняющихся от сигнатурного обнаружения.
