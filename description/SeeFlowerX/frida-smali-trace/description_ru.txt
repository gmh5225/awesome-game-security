Этот проект — набор инструментов трассировки на базе Frida для наблюдения за выполнением инструкций Android smali во время выполнения. Он использует агенты на JavaScript и TypeScript для перехвата путей интерпретатора ART и вывода подробных журналов выполнения. Документация объясняет, как с помощью средств статического анализа, таких как IDA, находить архитектурно-зависимые смещения и регистры перед запуском трассировки. Предназначен для реверс-инжиниринга Android и исследований мобильной безопасности с акцентом на анализ поведения во время выполнения.
