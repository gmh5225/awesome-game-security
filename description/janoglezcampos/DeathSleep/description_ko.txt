이 프로젝트는 현재 스레드를 종료한 뒤 실행을 재개하기 전에 복원하고, 실행이 없는 동안 페이지 보호를 변경하는 회피 기법의 PoC 구현입니다. 슬립과 난독화 기법은 maldev 커뮤니티에서 잘 알려져 있으며 구현은 다양하지만, 목표는 대개 슬립 중 메모리 스캐너로부터 숨기는 것으로, 보통 페이지 보호를 바꾸고 shellcode를 암호화하는 등의 기능을 추가합니다. 그러나 shellcode를 숨기는 또 다른 중요한 점은 현재 실행 스레드를 숨기는 것입니다. 주로 안티치트 / 페이지 보호 영역에서 일하는 안티치트 엔지니어와 방어 보안 연구자에게 유용합니다.
