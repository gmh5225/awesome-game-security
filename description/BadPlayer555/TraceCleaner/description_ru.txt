Этот проект — минимальный пример драйвера ядра для удаления типичных артефактов следов драйверов. Он сосредоточен на очистке записей в MmUnloadedDrivers и PiDDBCacheTable, которые часто проверяются в контексте криминалистики и античита. Код написан на C++ для окружений ядра Windows и рассчитан на выполнение через рабочие процессы manual mapping. Основной сценарий использования — образовательное исследование гигиены следов в ядре и поверхностей обнаружения античита.
