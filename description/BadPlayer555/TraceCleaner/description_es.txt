Este proyecto es un ejemplo mínimo de controlador de kernel para eliminar artefactos comunes de rastros de controladores. Se centra en limpiar entradas de MmUnloadedDrivers y PiDDBCacheTable, estructuras que se examinan con frecuencia en contextos forenses y de anti-cheat. El código está escrito en C++ para entornos de kernel de Windows y está pensado para ejecutarse mediante flujos de trabajo de manual mapping. El caso de uso principal es la investigación educativa sobre higiene de rastros en el kernel y superficies de detección anti-cheat.
