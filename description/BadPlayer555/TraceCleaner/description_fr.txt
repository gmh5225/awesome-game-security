Ce projet est un exemple minimal de pilote noyau destiné à supprimer les artefacts de traces courants laissés par les pilotes. Il se concentre sur le nettoyage des entrées dans MmUnloadedDrivers et PiDDBCacheTable, structures fréquemment examinées en forensique et dans les contextes anti-cheat. Le code est écrit en C++ pour les environnements noyau Windows et est conçu pour être exécuté via des workflows de manual mapping. Le cas d’usage principal est la recherche éducative sur l’hygiène des traces noyau et les surfaces de détection anti-cheat.
