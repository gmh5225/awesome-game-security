Questo progetto è un esempio minimale di driver kernel per rimuovere gli artefatti di traccia comuni dei driver. Si concentra sulla cancellazione delle voci da MmUnloadedDrivers e PiDDBCacheTable, strutture spesso esaminate in contesti forensi e anti-cheat. Il codice è scritto in C++ per ambienti kernel Windows ed è pensato per essere eseguito tramite workflow di manual mapping. Il caso d’uso principale è la ricerca educativa sull’igiene delle tracce kernel e sulle superfici di rilevamento anti-cheat.
