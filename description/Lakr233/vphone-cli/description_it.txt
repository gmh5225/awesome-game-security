Questo progetto è uno strumento CLI e un framework di patch del firmware per costruire ed eseguire iPhone virtualizzati (vphone) su Mac Apple Silicon. Include codice sorgente Swift, patcher di firmware in Python per i componenti iBoot, kernel e TXM, set di patch jailbreak, builder di ramdisk e documentazione di ricerca su patch binarie del kernel e pipeline di eventi da tastiera. È destinato soprattutto a ricercatori di sicurezza iOS e sviluppatori di emulatori che costruiscono firmware personalizzati per ambienti iOS virtualizzati.
