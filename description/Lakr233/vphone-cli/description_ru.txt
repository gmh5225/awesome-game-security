Этот проект — CLI-инструмент и фреймворк для патчинга прошивки, предназначенный для сборки и запуска виртуализированных iPhone (vphone) на Mac с Apple Silicon. Включает исходники на Swift, Python-патчеры прошивки для компонентов iBoot, ядра и TXM, наборы jailbreak-патчей, сборщики ramdisk, а также исследовательскую документацию по бинарным патчам ядра и конвейерам событий клавиатуры. В первую очередь полезен исследователям безопасности iOS и разработчикам эмуляторов, создающим кастомные прошивки для виртуализированных сред iOS.
