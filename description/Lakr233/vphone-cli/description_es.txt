Este proyecto es una herramienta CLI y un framework de parcheo de firmware para construir y ejecutar iPhones virtualizados (vphone) en Macs con Apple Silicon. Incluye código fuente Swift, parcheadores de firmware en Python para componentes iBoot, kernel y TXM, conjuntos de parches de jailbreak, constructores de ramdisk y documentación de investigación sobre parches binarios del kernel y pipelines de eventos de teclado. Resulta especialmente útil para investigadores de seguridad iOS y desarrolladores de emuladores que construyen firmware personalizado para entornos iOS virtualizados.
