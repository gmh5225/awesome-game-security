Этот проект — плагин для Binary Ninja, предназначенный для реверс-инжиниринга PTX — виртуальной ISA GPU CUDA.
Плагин можно установить через менеджер плагинов Binary Ninja.
Он в первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов и плагинов Binary Ninja.
