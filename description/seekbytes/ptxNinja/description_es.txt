Este proyecto es un complemento de Binary Ninja para la ingeniería inversa de PTX — la ISA virtual de GPU de CUDA.
El complemento se puede instalar mediante el administrador de complementos de Binary Ninja.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats y complementos de Binary Ninja.
