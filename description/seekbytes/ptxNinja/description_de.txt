Dieses Projekt ist ein Binary-Ninja-Plugin zur Reverse Engineering von PTX — der virtuellen ISA von CUDA-GPUs.
Das Plugin lässt sich über den Plugin-Manager von Binary Ninja installieren.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats und Binary-Ninja-Plugins untersuchen.
