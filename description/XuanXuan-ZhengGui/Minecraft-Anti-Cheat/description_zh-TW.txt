UltraAntiCheat 是一款面向 Spigot 與 Paper（1.13+）的開源 Minecraft 伺服器反作弊外掛，使用 Java 17+ 與 Maven 撰寫。它提供十三項核心偵測模組，涵蓋移動、戰鬥、方塊與封包作弊，例如飛行、加速、Timer、KillAura、Reach、連點、搭路、穿牆、擊退、NoSlow、異常封包、地面偽造與透視。亮點技術包括基於 GCD 的旋轉分析以偵測 KillAura 與輔助瞄準、帶置信度評分的物理移動模擬以降低誤報，以及可選的 ProtocolLib 封包檢查。它可與 GrimAC、NoCheatPlus、Vulcan、Matrix、Spartan 等其他反作弊方案橋接，並提供基於瀏覽器的網頁控制台用於即時監控。主要面向需要可設定警報、懲罰與多重偵測的 Minecraft 伺服器營運者。
