UltraAntiCheat 是一款面向 Spigot 与 Paper（1.13+）的开源 Minecraft 服务器反作弊插件，使用 Java 17+ 与 Maven 编写。它提供十三项核心检测模块，覆盖移动、战斗、方块与数据包作弊，例如飞行、加速、Timer、KillAura、Reach、连点、搭路、穿墙、击退、NoSlow、异常数据包、地面伪造与透视。亮点技术包括基于 GCD 的旋转分析以检测 KillAura 与辅助瞄准、带置信度评分的物理移动模拟以降低误报，以及可选的 ProtocolLib 数据包检查。它可与 GrimAC、NoCheatPlus、Vulcan、Matrix、Spartan 等其他反作弊方案桥接，并提供基于浏览器的网页控制台用于实时监控。主要面向需要可配置告警、惩罚与多重检测的 Minecraft 服务器运营者。
