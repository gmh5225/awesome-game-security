UltraAntiCheat è un plugin anti-cheat open source per server Minecraft Spigot e Paper (1.13+), scritto in Java 17+ con Maven. Fornisce tredici moduli di rilevamento principali che coprono cheating di movimento, combattimento, blocchi e pacchetti, come fly, speed, timer, KillAura, reach, auto-clicker, scaffold, phase, velocity, no-slow, pacchetti non validi, ground spoof e xray. Tecniche rilevanti includono l’analisi della rotazione basata su GCD per KillAura e aim assist, la simulazione fisica del movimento con punteggio di confidenza per ridurre i falsi positivi e l’ispezione opzionale dei pacchetti con ProtocolLib. Può collegarsi ad altri anti-cheat tra cui GrimAC, NoCheatPlus, Vulcan, Matrix e Spartan, e offre una dashboard web basata su browser per il monitoraggio in tempo reale. Il caso d’uso principale è la sicurezza di Minecraft per operatori di server che necessitano di avvisi, punizioni e rilevamento multi-controllo configurabili.
