UltraAntiCheat — это открытый античит-плагин для серверов Minecraft на Spigot и Paper (1.13+), написанный на Java 17+ с Maven. Он предоставляет тринадцать основных модулей обнаружения, охватывающих читы движения, боя, блоков и пакетов: полёт, скорость, timer, KillAura, reach, автокликер, scaffold, phase, velocity, no-slow, некорректные пакеты, подмена земли и xray. Среди заметных техник — анализ поворотов на основе GCD для KillAura и aim assist, физическая симуляция движения с оценкой уверенности для снижения ложных срабатываний и опциональная проверка пакетов через ProtocolLib. Плагин может стыковаться с другими античитами, включая GrimAC, NoCheatPlus, Vulcan, Matrix и Spartan, и предлагает браузерную веб-панель для мониторинга в реальном времени. Основной сценарий использования — безопасность Minecraft для операторов серверов, которым нужны настраиваемые оповещения, наказания и многоуровневое обнаружение читов.
