Ce projet est MAMBO, un framework d'instrumentation binaire dynamique à faible surcoût pour ARM et AArch64 sous Linux. Il traduit et instrumente le code machine à l'exécution via un cache de code logiciel, avec prise en charge des callbacks au niveau instruction, du traçage de blocs de base, de l'interception de fonctions et de plugins d'analyse personnalisés. Le framework en C s'exécute de façon transparente sur des binaires non modifiés. Il s'adresse aux chercheurs en sécurité, analystes de performance et créateurs d'outils ayant besoin d'une instrumentation binaire légère sur les plateformes ARM.
