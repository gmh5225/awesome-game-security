Este proyecto es MAMBO, un framework de instrumentación binaria dinámica de bajo overhead para ARM y AArch64 en Linux. Traduce e instrumenta código máquina en tiempo de ejecución mediante una caché de código software, con soporte para callbacks a nivel de instrucción, trazado de bloques básicos, interceptación de funciones y plugins de análisis personalizados. El framework en C opera de forma transparente sobre binarios sin modificar. Está orientado a investigadores de seguridad, analistas de rendimiento y creadores de herramientas que necesiten instrumentación binaria ligera en plataformas ARM.
