Этот проект — MAMBO, низконакладной фреймворк динамической бинарной инструментации для ARM и AArch64 в Linux. Он транслирует и инструментирует машинный код во время выполнения с помощью программного кэша кода, поддерживая колбэки на уровне инструкций, трассировку базовых блоков, перехват функций и пользовательские плагины анализа. C-фреймворк прозрачно работает с неизменёнными бинарниками. Предназначен для исследователей безопасности, аналитиков производительности и разработчиков инструментов, которым нужна лёгкая бинарная инструментация на платформах ARM.
