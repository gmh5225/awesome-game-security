Questo progetto è MAMBO, un framework di instrumentazione binaria dinamica a basso overhead per ARM e AArch64 su Linux. Traduce e instrumenta il codice macchina a runtime tramite una software code cache, supportando callback a livello di istruzione, tracing dei basic block, intercettazione di funzioni e plugin di analisi personalizzati. Il framework in C opera in modo trasparente su binari non modificati. È pensato per ricercatori di sicurezza, analisti delle prestazioni e sviluppatori di tool che necessitano di instrumentazione binaria leggera sulle piattaforme ARM.
