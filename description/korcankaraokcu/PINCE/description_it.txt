Questo progetto è PINCE (PINCE Is Not Cheat Engine), un frontend per GDB pensato per il game hacking e il reverse engineering su Linux. Offre una GUI basata su Qt con scansione della memoria, modifica dei valori, risoluzione delle catene di puntatori, code injection, gestione dei breakpoint e supporto per tabelle compatibili con Cheat Engine. L'applicazione Python comunica con GDB per l'attacco ai processi e le operazioni in memoria. È rivolta a game hacker e reverse engineer Linux che necessitano di funzionalità simili a Cheat Engine sui sistemi Linux.
