Este proyecto es PINCE (PINCE Is Not Cheat Engine), un frontend de GDB pensado para game hacking e ingeniería inversa en Linux. Ofrece una interfaz gráfica basada en Qt con escaneo de memoria, edición de valores, resolución de cadenas de punteros, inyección de código, gestión de puntos de interrupción y compatibilidad con tablas de Cheat Engine. La aplicación Python se comunica con GDB para adjuntarse a procesos y realizar operaciones de memoria. Está dirigida a game hackers e ingenieros inversos de Linux que necesitan funcionalidad similar a Cheat Engine en sistemas Linux.
