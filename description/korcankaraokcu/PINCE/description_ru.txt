Этот проект — PINCE (PINCE Is Not Cheat Engine), фронтенд для GDB, предназначенный для взлома игр и реверс-инжиниринга в Linux. Он предоставляет GUI на Qt с сканированием памяти, редактированием значений, разрешением цепочек указателей, внедрением кода, управлением точками останова и поддержкой таблиц, совместимых с Cheat Engine. Python-приложение взаимодействует с GDB для подключения к процессам и операций с памятью. Оно рассчитано на Linux-геймхакеров и реверс-инженеров, которым нужна функциональность, близкая к Cheat Engine, в системах Linux.
