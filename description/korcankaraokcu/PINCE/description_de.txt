Dieses Projekt ist PINCE (PINCE Is Not Cheat Engine), ein Frontend für GDB, das für Game Hacking und Reverse Engineering unter Linux entwickelt wurde. Es bietet eine Qt-basierte GUI mit Speicherscan, Wertbearbeitung, Auflösung von Pointer-Ketten, Code-Injection, Breakpoint-Verwaltung und Unterstützung für Cheat-Engine-kompatible Tabellen. Die Python-Anwendung kommuniziert mit GDB für Prozessanbindung und Speicheroperationen. Sie richtet sich an Linux-Game-Hacker und Reverse Engineers, die Cheat-Engine-ähnliche Funktionen auf Linux-Systemen benötigen.
