Ce projet est PINCE (PINCE Is Not Cheat Engine), une interface pour GDB conçue pour le game hacking et la rétro-ingénierie sous Linux. Elle propose une interface graphique basée sur Qt avec scan mémoire, édition de valeurs, résolution de chaînes de pointeurs, injection de code, gestion des points d'arrêt et prise en charge des tables compatibles Cheat Engine. L'application Python communique avec GDB pour l'attachement aux processus et les opérations mémoire. Elle s'adresse aux game hackers et reverse engineers Linux qui ont besoin de fonctionnalités proches de Cheat Engine sur les systèmes Linux.
