cs2-dumper — внешний дампер смещений и интерфейсов для Counter-Strike 2.
Ядро написано на Rust и использует доступ к памяти на базе memflow в Windows и Linux.
Инструмент может выдавать структурированный вывод на C#, C++, Rust и JSON для последующей автоматизации.
Его в основном применяют исследователи игровой безопасности и аналитики читов или античитов, которым нужны актуальные схемы и смещения.
