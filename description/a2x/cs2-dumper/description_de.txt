cs2-dumper ist ein externer Offset- und Interface-Dumper für Counter-Strike 2.
Der Kern ist in Rust geschrieben und nutzt memflow-basierten Speicherzugriff unter Windows und Linux.
Das Tool kann strukturierte Ausgaben in C#, C++, Rust und JSON für nachgelagerte Automatisierung erzeugen.
Es wird vor allem von Game-Security-Forschern sowie Cheat- und Anti-Cheat-Analysten verwendet, die aktuelle Schema- und Offset-Daten benötigen.
