Android Mod Menu는 IL2CPP 및 기타 네이티브 Android 게임을 위한 플로팅 메뉴 프레임워크입니다.
Java UI 코드와 네이티브 C/C++ 훅을 결합하며, KittyMemory, MSHook, And64InlineHook, 문자열 난독화 유틸리티 등의 구성 요소를 통합합니다.
이 템플릿은 넓은 Android 버전 범위의 ARMv7 및 ARM64 기기를 지원하며, 에셋을 Base64 기반 네이티브 데이터로 임베드합니다.
주로 게임 모딩 및 보안 연구자가 런타임 기능 토글과 훅 기반 실험을 프로토타이핑하는 데 사용됩니다.
