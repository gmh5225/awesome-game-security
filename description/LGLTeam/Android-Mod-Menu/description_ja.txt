Android Mod Menu は、IL2CPP をはじめとするネイティブ Android ゲーム向けのフローティングメニューフレームワークです。
Java の UI コードとネイティブ C/C++ のフックを組み合わせ、KittyMemory、MSHook、And64InlineHook、文字列難読化ユーティリティなどのコンポーネントを統合しています。
このテンプレートは幅広い Android バージョンの ARMv7 および ARM64 デバイスをサポートし、アセットを Base64 ベースのネイティブデータとして埋め込みます。
主にゲーム改造やセキュリティ研究者が、実行時の機能トグルやフックベースの実験をプロトタイプする用途で使われます。
