Android Mod Menu es un framework de menú flotante para juegos Android nativos IL2CPP y otros.
Combina código de UI en Java con hooks nativos en C/C++ e integra componentes como KittyMemory, MSHook, And64InlineHook y utilidades de ofuscación de cadenas.
La plantilla admite dispositivos ARMv7 y ARM64 en un amplio rango de versiones de Android e incrusta recursos como datos nativos respaldados por Base64.
Se usa principalmente por investigadores de modding y seguridad de juegos para prototipar interruptores de funciones en tiempo de ejecución y experimentos basados en hooks.
