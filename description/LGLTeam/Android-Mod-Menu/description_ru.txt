Android Mod Menu — это фреймворк плавающего меню для IL2CPP и других нативных Android-игр.
Он сочетает Java UI-код с нативными C/C++ хуками и интегрирует такие компоненты, как KittyMemory, MSHook, And64InlineHook и утилиты обфускации строк.
Шаблон поддерживает устройства ARMv7 и ARM64 в широком диапазоне версий Android и встраивает ресурсы в виде нативных данных на основе Base64.
В основном используется исследователями в области моддинга игр и безопасности для прототипирования переключателей функций во время выполнения и экспериментов на основе хуков.
