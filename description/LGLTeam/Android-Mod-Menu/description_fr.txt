Android Mod Menu est un framework de menu flottant pour les jeux Android natifs IL2CPP et autres.
Il combine du code UI Java avec des hooks natifs C/C++ et intègre des composants tels que KittyMemory, MSHook, And64InlineHook et des utilitaires d'obfuscation de chaînes.
Le modèle prend en charge les appareils ARMv7 et ARM64 sur une large gamme de versions Android et intègre les ressources sous forme de données natives encodées en Base64.
Il est principalement utilisé par les moddeurs de jeux et les chercheurs en sécurité pour prototyper des interrupteurs de fonctionnalités à l'exécution et des expériences basées sur des hooks.
