Este proyecto es una herramienta de defensa para Windows que supervisa y bloquea la carga de ensamblados .NET CLR (Common Language Runtime) en procesos. Usa enganche de DLL mediante ClrHook para interceptar la inicialización del CLR, registrar metadatos PE y hashes de los ensamblados cargados, y opcionalmente ejecutarse como servicio de Windows para un monitoreo persistente. Es especialmente útil para investigadores de seguridad defensiva y operadores de blue team que estudian la detección de ataques basados en .NET y el control de carga del CLR.
