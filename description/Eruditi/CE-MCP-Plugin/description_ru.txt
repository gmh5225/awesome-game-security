Этот проект — плагин Cheat Engine, который предоставляет функции редактирования памяти и управления процессами через ориентированный на ИИ командный канал. Он реализован на C с интеграцией Lua и использует асинхронную TCP-связь для приёма и выполнения удалённых инструкций без блокировки интерфейса хоста. Плагин поддерживает широкий набор команд для чтения и записи памяти, заморозки значений, дизассемблирования и ассемблирования, управления процессами и внедрения DLL. Основной сценарий использования — автоматизация рабочих процессов исследования игровой памяти из внешних инструментов и эксперименты с ИИ-управляемым контролем Cheat Engine.
