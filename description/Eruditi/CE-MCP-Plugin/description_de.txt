Dieses Projekt ist ein Cheat-Engine-Plugin, das Speicherbearbeitung und Prozesssteuerung über einen KI-orientierten Befehlskanal bereitstellt. Es ist in C mit Lua-Integration implementiert und nutzt asynchrone TCP-Kommunikation, um Remote-Anweisungen zu empfangen und auszuführen, ohne die Host-Oberfläche zu blockieren. Das Plugin unterstützt einen breiten Befehlssatz für Speicherlese- und -schreibvorgänge, das Einfrieren von Werten, Disassemblierung und Assemblierung, Prozessverwaltung sowie DLL-Injection. Der Hauptanwendungsfall ist die Automatisierung von Workflows zur Untersuchung von Spielespeicher aus externen Tools sowie das Experimentieren mit KI-gestützter Cheat-Engine-Steuerung.
