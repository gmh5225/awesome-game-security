Este proyecto es un complemento de Cheat Engine que expone funciones de edición de memoria y control de procesos a través de un canal de comandos orientado a la IA. Está implementado en C con integración de Lua y usa comunicación TCP asíncrona para recibir y ejecutar instrucciones remotas sin bloquear la interfaz del host. El complemento admite un amplio conjunto de comandos para operaciones de lectura y escritura de memoria, congelación de valores, desensamblado y ensamblado, gestión de procesos e inyección de DLL. Su caso de uso principal es automatizar flujos de trabajo de investigación de memoria de juegos desde herramientas externas y experimentar con el control de Cheat Engine asistido por IA.
