Questo progetto è un plugin di Cheat Engine che espone funzionalità di modifica della memoria e controllo dei processi tramite un canale di comandi orientato all'IA. È implementato in C con integrazione Lua e usa la comunicazione TCP asincrona per ricevere ed eseguire istruzioni remote senza bloccare l'interfaccia host. Il plugin supporta un ampio set di comandi per operazioni di lettura e scrittura della memoria, congelamento dei valori, disassemblaggio e assemblaggio, gestione dei processi e iniezione di DLL. Il caso d'uso principale è automatizzare i flussi di lavoro di ricerca sulla memoria di gioco da strumenti esterni e sperimentare il controllo di Cheat Engine assistito dall'IA.
