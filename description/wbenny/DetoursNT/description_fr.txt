Ce projet a un objectif simple : rendre Detours dépendant uniquement de NTDLL.DLL, sans modifier le code d'origine.
Consultez le projet injdrv de l'auteur pour comprendre comment cela est réalisé.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine des cheats et des hooks.
