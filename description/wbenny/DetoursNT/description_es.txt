Este proyecto tiene un objetivo sencillo: hacer que Detours dependa solo de NTDLL.DLL sin modificar el código original.
Puedes consultar el proyecto injdrv del autor para ver cómo se logra.
Es útil principalmente para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats y hooks.
