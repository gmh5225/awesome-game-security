Este proyecto es un framework C++ multiplataforma para crear shellcode de Windows.
Está escrito principalmente en C/C++ y C++, y se centra en el trabajo a nivel de kernel, el desarrollo de controladores y OpenGL.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / motores y trucos de shellcode.
