Questo progetto è una libreria di socket in modalità kernel di Windows scritta in C che avvolge l’API Windows Sockets Kernel (WSK) per fornire ai driver kernel un’interfaccia socket semplificata in stile BSD. Supporta connessioni TCP e UDP dal ring 0 senza componenti in modalità utente, consentendo la comunicazione di rete direttamente dal codice kernel. È pensata per i ricercatori di sicurezza del kernel che studiano canali di rete occulti in modalità kernel e per gli sviluppatori che realizzano driver con capacità di rete.
