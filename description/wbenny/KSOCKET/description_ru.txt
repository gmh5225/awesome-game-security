Этот проект — библиотека сокетов режима ядра Windows на языке C, которая оборачивает API Windows Sockets Kernel (WSK) и предоставляет драйверам ядра упрощённый интерфейс сокетов в стиле BSD. Она поддерживает TCP- и UDP-соединения из ring 0 без компонентов пользовательского режима, позволяя выполнять сетевое взаимодействие напрямую из кода ядра. Библиотека предназначена для исследователей безопасности ядра, изучающих скрытые сетевые каналы в режиме ядра, и для разработчиков, создающих драйверы с сетевыми возможностями.
