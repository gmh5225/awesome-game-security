Dieses Projekt ist eine Windows-Kernelmode-Socket-Bibliothek in C, die die Windows Sockets Kernel (WSK) API kapselt und Kernel-Treibern eine vereinfachte Socket-Schnittstelle im BSD-Stil bereitstellt. Sie unterstützt TCP- und UDP-Verbindungen aus Ring 0 ohne User-Mode-Komponenten und ermöglicht so Netzwerkkommunikation direkt aus Kernel-Code. Zielgruppe sind Kernel-Sicherheitsforscher, die verdeckte Netzwerkkanäle im Kernelmodus untersuchen, sowie Entwickler, die netzwerkfähige Treiber erstellen.
