Ce projet est une bibliothèque de sockets en mode noyau Windows écrite en C, qui encapsule l’API Windows Sockets Kernel (WSK) afin d’offrir aux pilotes noyau une interface de sockets simplifiée de style BSD. Elle prend en charge les connexions TCP et UDP depuis le ring 0 sans composants en mode utilisateur, permettant ainsi une communication réseau directement depuis le code noyau. Elle s’adresse aux chercheurs en sécurité noyau qui étudient les canaux réseau clandestins en mode noyau, ainsi qu’aux développeurs qui créent des pilotes capables de communiquer sur le réseau.
