Este proyecto es una biblioteca de sockets en modo kernel de Windows escrita en C que encapsula la API Windows Sockets Kernel (WSK) para ofrecer a los controladores de kernel una interfaz de sockets simplificada al estilo BSD. Admite conexiones TCP y UDP desde el ring 0 sin componentes en modo usuario, lo que permite la comunicación en red directamente desde el código del kernel. Está dirigida a investigadores de seguridad de kernel que estudian canales de red encubiertos en modo kernel y a desarrolladores que crean controladores con capacidad de red.
