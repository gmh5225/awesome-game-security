Ce projet est un pilote injecteur de DLL en mode noyau Windows écrit en C, qui injecte des DLL dans des processus via la livraison d'APC (Asynchronous Procedure Call) depuis l'espace noyau. Le pilote enregistre un callback de création de processus pour intercepter le démarrage du processus cible, puis met en file un APC en mode utilisateur qui charge la DLL spécifiée via LdrLoadDll. Cette approche basée sur le noyau contourne la détection d'injection en mode utilisateur et les hooks anti-cheat. Il s'adresse aux chercheurs noyau étudiant les primitives d'injection en mode noyau et leur détection du point de vue anti-cheat.
