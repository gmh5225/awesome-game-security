Этот проект — драйвер-инжектор DLL в режиме ядра Windows на C, который внедряет DLL в процессы с помощью доставки APC (Asynchronous Procedure Call) из пространства ядра. Драйвер регистрирует callback создания процесса, чтобы перехватить запуск целевого процесса, затем ставит в очередь APC пользовательского режима, загружающий указанную DLL через LdrLoadDll. Такой подход на уровне ядра обходит обнаружение инъекций в пользовательском режиме и хуки античита. Он предназначен для исследователей ядра, изучающих примитивы инъекции в режиме ядра и их обнаружение с точки зрения античита.
