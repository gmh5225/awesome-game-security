本プロジェクトは、Half-Life、Opposing Force、Blue Shift 向けの MOD 開発を目的とした、更新・統合済みの Half-Life SDK です。
主に大規模な C++ コードベースで構成され、CMake によるモダンな Visual Studio および GCC ツールチェーン向けのクロスプラットフォームビルドをサポートします。
多数のバグ修正を統合し、共有システムをリファクタリングして重複を削減するとともに、MOD 開発者向けの充実したドキュメント、ツール、変更履歴を提供します。
チート用途ではなくゲーム開発およびエンジンレベルの研究を想定しており、クラシック FPS のコード構成と安全な MOD 開発フローを理解するうえで有用です。
