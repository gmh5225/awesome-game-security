Этот проект — обновлённый и объединённый Half-Life SDK для разработки модов для Half-Life, Opposing Force и Blue Shift.
В основном это крупная кодовая база на C++ с кроссплатформенной сборкой на CMake для современных инструментальных цепочек Visual Studio и GCC.
Репозиторий объединяет множество исправлений ошибок, рефакторит общие системы для снижения дублирования и предоставляет подробную документацию, инструменты и журналы изменений для разработчиков модов.
Он предназначен для разработки игр и исследований на уровне движка, а не для читерства, и полезен для понимания архитектуры кода классических FPS и безопасных процессов разработки модов.
