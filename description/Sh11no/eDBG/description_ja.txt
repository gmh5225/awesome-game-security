本プロジェクトは、従来の ptrace アタッチではなく eBPF を基盤とした、Android ARM64 向けの軽量コマンドラインデバッガです。主に Go で実装され、C および eBPF コンポーネントを併用し、ブレークポイント、メモリ、レジスタ、スレッド検査コマンドを備えた GDB 風の対話型ワークフローを提供します。ファイル＋オフセット方式のブレークポイントモデルは、高速起動と、保護されたアプリにおけるアンチデバッグ干渉へのより強い耐性を目的として設計されています。モダンなカーネルを備えた root 化デバイス上でのモバイルリバースエンジニアリングおよびゲームセキュリティ分析を対象としています。
