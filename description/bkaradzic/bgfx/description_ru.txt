Этот проект — bgfx, кроссплатформенная библиотека рендеринга, предоставляющая тонкую абстракцию над несколькими графическими API, включая Direct3D 9/11/12, Metal, OpenGL, OpenGL ES, Vulkan и WebGPU. Она предлагает единый API для отправки draw call, управления шейдерами, работы с текстурами, compute-диспетчеризации и управления состоянием рендеринга, позволяя приложениям охватывать все основные платформы одной кодовой базой. Библиотека на C/C++ включает кросс-компилятор шейдеров (shaderc) на основе GLSL. Она предназначена для разработчиков игровых движков и графических программистов, которым нужен переносимый низкоуровневый рендеринг на разных платформах.
