Dieses Projekt ist bgfx, eine plattformübergreifende Rendering-Bibliothek, die eine schlanke Abstraktion über mehrere Grafik-APIs bietet, darunter Direct3D 9/11/12, Metal, OpenGL, OpenGL ES, Vulkan und WebGPU. Sie stellt eine einheitliche API für Draw-Call-Submission, Shader-Verwaltung, Texturverarbeitung, Compute-Dispatches und Render-State-Management bereit, sodass Anwendungen mit einer einzigen Codebasis alle wichtigen Plattformen bedienen können. Die C/C++-Bibliothek enthält einen auf GLSL basierenden Shader-Cross-Compiler (shaderc). Sie richtet sich an Game-Engine-Entwickler und Grafikprogrammierer, die portables Low-Level-Rendering über Plattformen hinweg benötigen.
