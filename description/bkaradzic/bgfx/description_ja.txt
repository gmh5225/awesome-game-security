本プロジェクトは bgfx です。Direct3D 9/11/12、Metal、OpenGL、OpenGL ES、Vulkan、WebGPU など複数のグラフィックス API を薄く抽象化したクロスプラットフォーム描画ライブラリです。描画コールの送信、シェーダー管理、テクスチャ処理、コンピュートディスパッチ、レンダーステート管理向けの統一 API を提供し、単一のコードベースで主要プラットフォームを対象にできます。C/C++ ライブラリには GLSL ベースのシェーダークロスコンパイラ（shaderc）が含まれます。ポータブルな低レベル描画を必要とするゲームエンジン開発者やグラフィックスプログラマ向けです。
