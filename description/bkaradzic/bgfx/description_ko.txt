이 프로젝트는 bgfx로, Direct3D 9/11/12, Metal, OpenGL, OpenGL ES, Vulkan, WebGPU를 포함한 여러 그래픽 API 위에 얇은 추상화를 제공하는 크로스 플랫폼 렌더링 라이브러리입니다. 드로우 콜 제출, 셰이더 관리, 텍스처 처리, 컴퓨트 디스패치, 렌더 상태 관리를 위한 통합 API를 제공하여 단일 코드베이스로 주요 플랫폼을 모두 지원할 수 있습니다. C/C++ 라이브러리에는 GLSL 기반 셰이더 크로스 컴파일러(shaderc)가 포함되어 있습니다. 플랫폼 전반에 이식 가능한 저수준 렌더링이 필요한 게임 엔진 개발자와 그래픽 프로그래머를 대상으로 합니다.
