Questo progetto è bgfx, una libreria di rendering multipiattaforma che fornisce una sottile astrazione su più API grafiche, tra cui Direct3D 9/11/12, Metal, OpenGL, OpenGL ES, Vulkan e WebGPU. Offre un'API unificata per l'invio delle draw call, la gestione degli shader, la gestione delle texture, i dispatch di compute e la gestione dello stato di rendering, consentendo alle applicazioni di supportare tutte le principali piattaforme con un'unica codebase. La libreria C/C++ include un cross-compiler di shader (shaderc) basato su GLSL. È destinata agli sviluppatori di game engine e ai programmatori grafici che necessitano di rendering a basso livello portabile tra piattaforme.
