Ce projet est bgfx, une bibliothèque de rendu multiplateforme qui fournit une fine abstraction au-dessus de plusieurs API graphiques, notamment Direct3D 9/11/12, Metal, OpenGL, OpenGL ES, Vulkan et WebGPU. Elle offre une API unifiée pour la soumission des draw calls, la gestion des shaders, le traitement des textures, les dispatches de calcul et la gestion des états de rendu, permettant aux applications de cibler toutes les principales plateformes avec une seule base de code. La bibliothèque C/C++ inclut un compilateur croisé de shaders (shaderc) basé sur GLSL. Elle s'adresse aux développeurs de moteurs de jeu et aux programmeurs graphiques qui ont besoin d'un rendu bas niveau portable entre plateformes.
