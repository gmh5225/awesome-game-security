Este proyecto es una biblioteca simple y fácil de usar para disfrutar de la programación de videojuegos.
Está escrito principalmente en C y C/C++ y se centra en el trabajo a nivel de kernel, OpenGL y Vulkan.
Es especialmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo de juegos / código fuente.
