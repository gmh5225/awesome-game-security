Dieses Projekt ist eine umfassende Liste benutzerdefinierter Spiele-Engines.
Die Liste basiert auf einem Artikel aus dem Jahr 2020: custom_game_engines_small_study.
Sie ist vor allem für Spieleentwickler, Engine-Programmierer und Grafikforscher nützlich, die an Rendering und Grafik im Bereich Spiele-Engines / Guides arbeiten.
