Este proyecto es una biblioteca de Android para detectar y proteger frente a ataques de superposición (tapjacking). Comprueba ventanas de overlay visibles dibujadas encima de la aplicación que podrían interceptar la entrada del usuario u oscurecer elementos de la interfaz, implementando la detección TYPE_APPLICATION_OVERLAY de Android y el manejo del flag filterTouchesWhenObscured. La biblioteca Java notifica mediante callbacks cuando se detectan overlays. Está orientada a desarrolladores de aplicaciones Android e ingenieros de seguridad que protegen flujos de interfaz sensibles frente a ataques basados en overlays.
