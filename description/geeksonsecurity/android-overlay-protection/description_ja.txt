本プロジェクトは、オーバーレイ攻撃（タップジャッキング）を検出し防御するための Android ライブラリです。アプリの上に描画され、ユーザー入力を傍受したり UI 要素を覆い隠したりし得る可視オーバーレイウィンドウを検査し、Android の TYPE_APPLICATION_OVERLAY 検出と filterTouchesWhenObscured フラグの処理を実装しています。この Java ライブラリは、オーバーレイが検出されたときにコールバックで通知します。センシティブな UI フローをオーバーレイ攻撃から守る Android アプリ開発者およびセキュリティエンジニアを対象としています。
