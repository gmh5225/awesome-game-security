memflow フレームワーク経由でメモリの読み書き性能をベンチマークする、Windows 向け DMA 速度テストツールです。Rust で書かれており、egui ベースの GUI と CLI の両方を提供し、設定可能なチャンクサイズにわたってスループット（MiB/s）、操作レート、レイテンシを測定します。PCILeech（FPGA）およびネイティブコネクタに対応し、explorer.exe や ntdll.dll などのターゲットをプローブでき、結果を CSV または JSON でエクスポートします。主な用途は、リバースエンジニアリングおよびゲームセキュリティのワークフロー向けに、DMA ハードウェアとメモリアクセス性能を評価することです。
