Este proyecto es un framework de emulación de anti-cheat en modo kernel para estudiar cómo se comportan los controladores protegidos en condiciones de kernel simuladas.
Está escrito en C++ e incluye componentes para mapeo PE, resolución de imports, seguimiento de memoria, manejo de excepciones y emulación de instrucciones privilegiadas.
El código modela estructuras clave del kernel y monitoriza lecturas o escrituras para investigar comprobaciones anti-debug y anti-emulación.
Está orientado a investigadores de anti-cheat y seguridad de bajo nivel como plataforma de aprendizaje y experimentación.
