Этот проект — фреймворк эмуляции kernel anti-cheat для изучения поведения защищённых драйверов в условиях симулированного ядра.
Он написан на C++ и включает компоненты для PE-маппинга, разрешения импортов, отслеживания памяти, обработки исключений и эмуляции привилегированных инструкций.
Кодовая база моделирует ключевые структуры ядра и отслеживает операции чтения и записи для исследования anti-debug и anti-emulation проверок.
Он предназначен для исследователей anti-cheat и низкоуровневой безопасности как платформа для обучения и экспериментов.
