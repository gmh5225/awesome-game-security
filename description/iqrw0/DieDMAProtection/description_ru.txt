Этот проект представляет собой proof of concept для Windows, который отключает механизмы защиты DMA (IOMMU/VT-d), чтобы снова разрешить прямой доступ к физической памяти со стороны внешних устройств PCIe. Он манипулирует конфигурацией DMA remapping, чтобы обойти защиты, которые иначе блокировали бы аппаратное обеспечение атак DMA на базе FPGA при чтении памяти игровых процессов. Драйвер ядра на C демонстрирует поверхность атаки реализаций защиты DMA. Он предназначен для исследователей безопасности, изучающих техники обхода защиты DMA и границы безопасности IOMMU.
