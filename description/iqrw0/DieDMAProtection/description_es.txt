Este proyecto es una prueba de concepto para Windows que desactiva los mecanismos de protección DMA (IOMMU/VT-d) para volver a habilitar el acceso directo a la memoria física desde dispositivos PCIe externos. Manipula la configuración de remapeo DMA para eludir protecciones que de otro modo impedirían que hardware de ataque DMA basado en FPGA lea la memoria de procesos de juego. El controlador de kernel en C demuestra la superficie de ataque de las implementaciones de protección DMA. Está dirigido a investigadores de seguridad que estudian técnicas de omisión de la protección DMA y los límites de seguridad de IOMMU.
