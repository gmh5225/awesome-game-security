Suite de módulos de PowerShell para administrar directivas de integridad de código de Windows Defender Application Control (WDAC), que ofrece cmdlets para crear directivas CI base y complementarias, configurar opciones de reglas (UMCI, WHQL, modo de auditoría, instalador administrado), analizar binarios de directivas serializadas mediante CIPolicyParser y implementar directivas con CiTool.exe y compatibilidad de auditoría.
