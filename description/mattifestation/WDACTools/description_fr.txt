Suite de modules PowerShell pour gérer les stratégies d'intégrité du code Windows Defender Application Control (WDAC), offrant des cmdlets pour créer des stratégies CI de base et supplémentaires, configurer les options de règles (UMCI, WHQL, mode d'audit, programme d'installation géré), analyser les binaires de stratégies sérialisées via CIPolicyParser, et déployer les stratégies avec CiTool.exe ainsi qu'un support d'audit.
