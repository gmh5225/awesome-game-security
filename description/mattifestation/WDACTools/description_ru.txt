Набор модулей PowerShell для управления политиками целостности кода Windows Defender Application Control (WDAC), предоставляющий командлеты для создания базовых и дополнительных политик CI, настройки параметров правил (UMCI, WHQL, режим аудита, управляемый установщик), разбора сериализованных двоичных файлов политик через CIPolicyParser, а также развёртывания политик с помощью CiTool.exe и поддержки аудита.
