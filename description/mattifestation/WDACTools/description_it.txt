Suite di moduli PowerShell per gestire i criteri di integrità del codice di Windows Defender Application Control (WDAC), che fornisce cmdlet per creare criteri CI di base e supplementari, configurare le opzioni delle regole (UMCI, WHQL, modalità di controllo, installer gestito), analizzare i binari dei criteri serializzati tramite CIPolicyParser e distribuire i criteri con CiTool.exe e supporto per il controllo.
