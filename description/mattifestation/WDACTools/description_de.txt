PowerShell-Modulsuite zur Verwaltung von Windows Defender Application Control (WDAC) Codeintegritätsrichtlinien. Bietet Cmdlets zum Erstellen von Basis- und ergänzenden CI-Richtlinien, zum Konfigurieren von Regeloptionen (UMCI, WHQL, Überwachungsmodus, verwalteter Installer), zum Parsen serialisierter Richtlinienbinärdateien über CIPolicyParser sowie zum Bereitstellen von Richtlinien mit CiTool.exe und Überwachungsunterstützung.
