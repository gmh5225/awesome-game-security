Этот проект — внешний чит-фреймворк для Valorant с упором на возможности аимбота и ESP. Он в основном реализован на C++ и включает вспомогательные компоненты для взаимодействия с памятью, отрисовки оверлеев и рабочих процессов, связанных со спуфингом аппаратных идентификаторов. Репозиторий заявляет о работе с поддержкой драйвера и о встроенном инструментарии для обходных конфигураций. Основная аудитория — исследование наступательного game-hacking, а не оборонительная разработка античита.
