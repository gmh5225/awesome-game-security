Este proyecto es un framework de cheat externo para Valorant centrado en capacidades de aimbot y ESP. Está implementado principalmente en C++ e incluye componentes de apoyo para interacción con memoria, renderizado de overlays y flujos de trabajo relacionados con el spoofing de ID de hardware. El repositorio anuncia operación asistida por driver y herramientas incluidas para configuraciones orientadas a bypass. Su audiencia principal es la investigación ofensiva de game-hacking, no la ingeniería defensiva anti-cheat.
