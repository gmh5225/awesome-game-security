Questo progetto è un framework di cheat esterno per Valorant incentrato su capacità di aimbot e ESP. È implementato principalmente in C++ e include componenti di supporto per l'interazione con la memoria, il rendering di overlay e i workflow legati allo spoofing dell'ID hardware. Il repository promuove un funzionamento assistito da driver e tooling incluso per setup orientati al bypass. Il pubblico principale è la ricerca offensiva di game-hacking piuttosto che l'ingegneria difensiva anti-cheat.
