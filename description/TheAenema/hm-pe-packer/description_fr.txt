Ce projet est un packer et protecteur PE x64 à vocation pédagogique, implémenté en C++ avec Visual Studio.
Il montre comment les flux de packing d’exécutables peuvent envelopper et protéger des binaires Windows au chargement.
Le dépôt inclut le code source et les ressources de compilation pour étudier les structures PE, les stubs de packing et les mécanismes de protection.
Son public principal est constitué d’apprenants en sécurité et de praticiens du reverse engineering qui explorent la conception de packers.
