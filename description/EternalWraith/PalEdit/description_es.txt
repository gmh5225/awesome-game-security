Este proyecto es un editor de partidas guardadas para Palworld basado en Python que permite editar y generar entradas de datos de compañeros.
Ofrece un flujo de trabajo de escritorio para cargar archivos de guardado, modificar atributos, clonar entidades y escribir los datos actualizados de nuevo en disco.
El repositorio incluye scripts fuente, utilidades de compilación y recursos empaquetados para ejecutarlo directamente o compilar un ejecutable de Windows.
Su caso de uso principal es el modding de partidas guardadas y la experimentación controlada con formatos de datos de guardado estructurados.
