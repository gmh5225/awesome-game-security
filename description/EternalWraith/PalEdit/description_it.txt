Questo progetto è un editor di salvataggi per Palworld basato su Python che consente di modificare e generare voci di dati dei compagni.
Fornisce un flusso di lavoro desktop per caricare i file di salvataggio, modificare gli attributi, clonare entità e riscrivere i dati aggiornati su disco.
Il repository include script sorgente, helper di build e asset confezionati per l'esecuzione diretta o la compilazione di un eseguibile Windows.
Il caso d'uso principale è il modding dei salvataggi di gioco e la sperimentazione controllata con formati di dati di salvataggio strutturati.
