Dieses Projekt ist ein Python-basierter Save-Editor für Palworld, mit dem Benutzer Begleiterdaten bearbeiten und erzeugen können.
Es bietet einen Desktop-Workflow zum Laden von Speicherständen, Ändern von Attributen, Klonen von Entitäten und Zurückschreiben der aktualisierten Daten auf die Festplatte.
Das Repository enthält Quellskripte, Build-Hilfen und paketierte Assets zum direkten Ausführen oder zum Kompilieren einer Windows-Executable.
Der Hauptanwendungsfall ist das Modden von Spielständen sowie kontrollierte Experimente mit strukturierten Save-Datenformaten.
