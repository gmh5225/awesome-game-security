Этот проект — редактор сохранений для Palworld на Python, позволяющий редактировать и создавать записи данных компаньонов.
Он предоставляет настольный рабочий процесс для загрузки файлов сохранений, изменения атрибутов, клонирования сущностей и записи обновлённых данных обратно на диск.
Репозиторий включает исходные скрипты, вспомогательные средства сборки и упакованные ресурсы для прямого запуска или компиляции исполняемого файла Windows.
Основной сценарий использования — моддинг игровых сохранений и контролируемые эксперименты со структурированными форматами данных сохранений.
