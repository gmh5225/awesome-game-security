Questo progetto è una proof-of-concept per spoofare call stack arbitrari durante l'emissione di system call di Windows.
È implementato in C++ e include profili di stack di esempio selezionabili che imitano comportamenti comuni dei processi.
Il repository documenta anche passaggi pratici di debugging e validazione tramite telemetria e flussi di lavoro con debugger.
È pensato per i ricercatori di sicurezza che studiano tecniche di evasione EDR e logica di rilevamento basata sullo stack.
