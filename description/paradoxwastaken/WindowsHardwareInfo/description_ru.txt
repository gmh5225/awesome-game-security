Этот проект посвящён windows-Hardware-Info: проекту на C++, предназначенному для получения информации об оборудовании на ПК под Windows. Он взаимодействует со службой Windows Management Instrumentation (WMI) для запроса интересующей аппаратной информации и предоставляет простой интерфейс командной строки. В основном полезен инженерам по античиту и исследователям защитной безопасности, работающим в области античита / detection:HWID.
