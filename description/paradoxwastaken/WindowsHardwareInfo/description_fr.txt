Ce projet porte sur windows-Hardware-Info : un projet écrit en C++ pour obtenir des informations matérielles sur un PC Windows. Il s'interface avec le service Windows Management Instrumentation (WMI) pour interroger les informations matérielles d'intérêt et fournit une interface en ligne de commande basique. Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine anti-cheat / détection:HWID.
