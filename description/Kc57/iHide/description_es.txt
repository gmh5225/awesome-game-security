Este proyecto es un tweak de iOS para ocultar el jailbreak que ayuda a que las aplicaciones seleccionadas se ejecuten como si el dispositivo no estuviera jailbroken. Se integra con el panel de ajustes de iOS para que los usuarios puedan activar o desactivar el bypass por aplicación e intenta derrotar las comprobaciones habituales de detección de jailbreak. La implementación usa enganches al estilo MobileSubstrate con Objective-C y componentes de tweak de iOS relacionados. Se usa principalmente en pruebas de seguridad de aplicaciones y juegos móviles, donde los analistas necesitan evaluar el comportamiento detrás de las barreras de detección de jailbreak.
