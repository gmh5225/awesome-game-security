Ce projet est un tweak iOS de dissimulation du jailbreak qui permet aux applications sélectionnées de s'exécuter comme si l'appareil n'était pas jailbreaké. Il s'intègre au panneau de réglages iOS afin que les utilisateurs puissent activer ou désactiver le contournement par application, et tente de vaincre les contrôles courants de détection de jailbreak. L'implémentation utilise un accrochage de type MobileSubstrate avec Objective-C et des composants de tweak iOS associés. Il est principalement utilisé dans les tests de sécurité d'applications et de jeux mobiles, lorsque les analystes doivent évaluer le comportement derrière les portes de détection de jailbreak.
