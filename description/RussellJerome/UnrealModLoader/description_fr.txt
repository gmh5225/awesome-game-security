Ce projet est un framework de chargement de mods pour les jeux Unreal Engine 4, prenant en charge les mods Blueprint et les mods C++ basés sur un SDK de base. Il combine des systèmes de hooking natif et de rappels d'événements avec des composants tiers tels que MinHook et ImGui, afin de fournir des hooks de cycle de vie et des interfaces d'outils en jeu. La base de code inclut des chemins de chargement par injecteur et par DLL proxy pour démarrer les modules de mods dans les processus cibles. Il est principalement utilisé par les développeurs de mods UE4 et les chercheurs qui étudient l'extension à l'exécution et les techniques d'instrumentation de jeux.
