Ce projet est un framework de cheat interne Linux pour Counter-Strike: Global Offensive. Il est principalement écrit en C++ et comprend des scripts de build, des aides à l'injection et la gestion de configuration pour déployer un module en jeu riche en fonctionnalités. Le flux de travail s'appuie sur des outils Linux courants tels que CMake, l'injection basée sur GDB et des répertoires de configuration gérés par l'utilisateur pour la personnalisation à l'exécution. Il est surtout utilisé en recherche sur le game hacking pour étudier l'architecture des cheats internes et les techniques de manipulation côté client.
