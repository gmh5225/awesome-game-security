Este proyecto es un framework de cheat interno para Linux orientado a Counter-Strike: Global Offensive. Está escrito principalmente en C++ e incluye scripts de compilación, ayudas de inyección y gestión de configuración para desplegar un módulo en el juego con numerosas funciones. El flujo de trabajo se basa en herramientas habituales de Linux como CMake, inyección basada en GDB y directorios de configuración gestionados por el usuario para la personalización en tiempo de ejecución. Se usa sobre todo en investigación de game hacking para estudiar la arquitectura de cheats internos y las técnicas de manipulación del lado del cliente.
