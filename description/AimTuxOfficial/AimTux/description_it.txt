Questo progetto è un framework di cheat interno per Linux pensato per Counter-Strike: Global Offensive. È scritto principalmente in C++ e include script di build, helper di iniezione e gestione della configurazione per distribuire un modulo in-game ricco di funzionalità. Il flusso di lavoro si basa su strumenti Linux comuni come CMake, iniezione basata su GDB e directory di configurazione gestite dall'utente per la personalizzazione a runtime. È usato soprattutto nella ricerca sul game hacking per studiare l'architettura dei cheat interni e le tecniche di manipolazione lato client.
