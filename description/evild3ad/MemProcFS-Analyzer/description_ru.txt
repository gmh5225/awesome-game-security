Этот проект — MemProcFS-Analyzer, основанный на PowerShell фреймворк криминалистического анализа поверх MemProcFS для исследования дампов памяти Windows. Он автоматизирует извлечение и анализ процессов, сетевых соединений, кустов реестра, журналов событий, артефактов браузеров и индикаторов вредоносного ПО из дампов физической памяти. Инструмент формирует HTML-отчёты со структурированными находками и интегрируется с VirusTotal и другими источниками угрозовой разведки. Он предназначен для специалистов по цифровой криминалистике и специалистов по реагированию на инциденты, выполняющих комплексную форензику памяти.
