이 프로젝트는 MemProcFS-Analyzer로, MemProcFS 위에 구축된 PowerShell 기반 포렌식 분석 프레임워크이며 Windows 메모리 덤프 조사에 사용됩니다. 물리 메모리 덤프에서 프로세스, 네트워크 연결, 레지스트리 하이브, 이벤트 로그, 브라우저 아티팩트, 악성코드 지표의 추출과 분석을 자동화합니다. 구조화된 조사 결과가 담긴 HTML 보고서를 생성하며 VirusTotal 및 기타 위협 인텔리전스 소스와 연동됩니다. 포괄적인 메모리 포렌식을 수행하는 디지털 포렌식 실무자와 사고 대응 담당자를 대상으로 합니다.
