This project focuses on decrypt VMware vTPM-encrypted .vmem/.vmsn/.vmss/.nvram from VM password — pure-Python encobj AES-256-CBC; vmem_flatten.py to Volatility 3-ready image; Win11 partial VM encryption.
For .vmsn / .vmss / .nvram the decrypted output is directly usable.
It is mainly useful for anti-cheat engineers and defensive security researchers working in the anti cheat / information system & forensics area.
