Ce projet se concentre sur le déchiffrement des fichiers .vmem/.vmsn/.vmss/.nvram chiffrés par vTPM VMware à partir du mot de passe de la VM — encobj AES-256-CBC en pure Python ; vmem_flatten.py pour une image prête pour Volatility 3 ; chiffrement partiel des VM sous Win11.
Pour .vmsn / .vmss / .nvram, la sortie déchiffrée est directement utilisable.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine anti-cheat / systèmes d'information et forensics.
