Этот проект посвящён расшифровке файлов .vmem/.vmsn/.vmss/.nvram, зашифрованных VMware vTPM, по паролю ВМ — чистый Python, encobj AES-256-CBC; vmem_flatten.py для образа, готового к Volatility 3; частичное шифрование ВМ в Win11.
Для .vmsn / .vmss / .nvram расшифрованный результат можно использовать напрямую.
В первую очередь полезен инженерам античита и исследователям оборонительной безопасности, работающим в области античита / информационных систем и форензики.
