Este proyecto se centra en descifrar archivos .vmem/.vmsn/.vmss/.nvram cifrados con vTPM de VMware a partir de la contraseña de la VM — encobj AES-256-CBC en Python puro; vmem_flatten.py para una imagen lista para Volatility 3; cifrado parcial de VM en Win11.
Para .vmsn / .vmss / .nvram, la salida descifrada es directamente utilizable.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / sistemas de información y forense.
