Questo progetto si concentra sulla decrittazione di file .vmem/.vmsn/.vmss/.nvram crittografati con vTPM VMware a partire dalla password della VM — encobj AES-256-CBC in puro Python; vmem_flatten.py per un'immagine pronta per Volatility 3; crittografia parziale delle VM su Win11.
Per .vmsn / .vmss / .nvram, l'output decrittato è direttamente utilizzabile.
È principalmente utile per ingegneri anti-cheat e ricercatori di sicurezza difensiva che operano nell'area anti-cheat / sistemi informativi e forensics.
