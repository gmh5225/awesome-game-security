Este proyecto es CFB (Canadian Furious Beaver), un framework de monitoreo de IRP (I/O Request Packet) en modo kernel de Windows. Instala un controlador de filtro que engancha la tabla de despacho IRP de los controladores objetivo, registrando todas las solicitudes IRP con parámetros, búferes y valores de retorno. El controlador de kernel en C y el cliente Python permiten el monitoreo en tiempo real de la comunicación entre controladores para ingeniería inversa de interfaces IOCTL. Está dirigido a investigadores del kernel de Windows y a reverse engineers que estudian protocolos de comunicación de controladores y fuzzing de IOCTL.
