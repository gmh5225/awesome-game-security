本プロジェクトは CFB（Canadian Furious Beaver）で、Windows カーネルモードの IRP（I/O Request Packet）監視フレームワークです。フィルタドライバを導入し、対象ドライバの IRP ディスパッチテーブルをフックして、パラメータ、バッファ、戻り値を含むすべての IRP リクエストを記録します。C 言語のカーネルドライバと Python クライアントにより、ドライバ通信をリアルタイムで監視し、IOCTL インターフェースのリバースエンジニアリングに利用できます。ドライバ通信プロトコルや IOCTL ファジングを研究する Windows カーネル研究者およびリバースエンジニア向けです。
