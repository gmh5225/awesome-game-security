이 프로젝트는 CFB(Canadian Furious Beaver)로, Windows 커널 모드 IRP(I/O Request Packet) 모니터링 프레임워크입니다. 필터 드라이버를 설치하여 대상 드라이버의 IRP 디스패치 테이블을 후킹하고, 매개변수·버퍼·반환값을 포함한 모든 IRP 요청을 기록합니다. C 커널 드라이버와 Python 클라이언트로 드라이버 통신을 실시간 모니터링하여 IOCTL 인터페이스를 리버스 엔지니어링할 수 있습니다. 드라이버 통신 프로토콜과 IOCTL 퍼징을 연구하는 Windows 커널 연구원 및 리버스 엔지니어를 대상으로 합니다.
