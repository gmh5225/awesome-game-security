Этот проект — CFB (Canadian Furious Beaver), фреймворк мониторинга IRP (I/O Request Packet) в режиме ядра Windows. Он устанавливает фильтр-драйвер, который перехватывает таблицу диспетчеризации IRP целевых драйверов и журналирует все запросы IRP с параметрами, буферами и возвращаемыми значениями. Драйвер ядра на C и клиент на Python позволяют в реальном времени отслеживать взаимодействие драйверов для реверс-инжиниринга интерфейсов IOCTL. Предназначен для исследователей ядра Windows и реверс-инженеров, изучающих протоколы обмена драйверов и фаззинг IOCTL.
