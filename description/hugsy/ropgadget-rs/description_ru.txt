Этот проект — поисковик ROP-гаджетов на Rust для обнаружения гаджетов Return-Oriented Programming в исполняемых бинарных файлах. Он сканирует бинарники PE, ELF и Mach-O в поисках последовательностей инструкций, заканчивающихся инструкциями возврата, которые можно объединять в цепочки для эксплуатации. Реализация на Rust обеспечивает быстрое параллельное сканирование крупных бинарников. Инструмент предназначен для разработчиков эксплойтов и исследователей уязвимостей, строящих ROP-цепочки для бинарной эксплуатации.
