Questo progetto è un finder di gadget ROP scritto in Rust per scoprire gadget di Return-Oriented Programming in eseguibili binari. Analizza binari PE, ELF e Mach-O alla ricerca di sequenze di istruzioni che terminano con istruzioni di return e che possono essere concatenate per lo sfruttamento. L'implementazione in Rust offre una scansione parallela veloce su binari di grandi dimensioni. È rivolto a sviluppatori di exploit e ricercatori di vulnerabilità che costruiscono catene ROP per lo sfruttamento binario.
