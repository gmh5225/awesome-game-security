MethaneKit — кроссплатформенный графический и прикладной фреймворк, который абстрагирует современные API 3D-рендеринга за единым интерфейсом.
Кодовая база в основном на C++ с CMake и поддерживает DirectX 12, Vulkan и Metal, сохраняя общие рабочие процессы шейдеров на разных платформах.
Включает руководства, примеры приложений, тестовую инфраструктуру и модульные runtime-компоненты для создания высокопроизводительных приложений рендеринга.
Проект полезен разработчикам движков и инструментов, которым нужна переносимая графическая основа для визуализации, игр или экспериментов с рендерингом, связанных с безопасностью.
