Questo progetto è BinExport, un plugin di esportazione di dati di analisi binaria per IDA Pro, Ghidra e Binary Ninja. Serializza i dati di disassemblaggio — tra cui funzioni, basic block, istruzioni, riferimenti incrociati e grafi delle chiamate — in formato Protocol Buffer per l'uso con BinDiff e altri strumenti di analisi esterni. Il plugin C++ fornisce una rappresentazione intermedia standardizzata per il confronto di binari e i flussi di lavoro di analisi automatizzata. È rivolto a reverse engineer e ricercatori di vulnerabilità che eseguono analisi e diffing binari multi-strumento.
