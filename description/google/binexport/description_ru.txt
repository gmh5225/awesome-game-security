Этот проект — BinExport, плагин экспорта данных бинарного анализа для IDA Pro, Ghidra и Binary Ninja. Он сериализует данные дизассемблирования, включая функции, базовые блоки, инструкции, перекрёстные ссылки и графы вызовов, в формат Protocol Buffer для использования с BinDiff и другими внешними средствами анализа. Плагин на C++ предоставляет стандартизированное промежуточное представление для сравнения бинарников и автоматизированных рабочих процессов анализа. Он ориентирован на инженеров по реверс-инжинирингу и исследователей уязвимостей, выполняющих кросс-инструментальный бинарный анализ и diffing.
