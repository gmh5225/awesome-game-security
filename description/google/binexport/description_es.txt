Este proyecto es BinExport, un complemento de exportación de datos de análisis binario para IDA Pro, Ghidra y Binary Ninja. Serializa datos de desensamblado —incluidas funciones, bloques básicos, instrucciones, referencias cruzadas y grafos de llamadas— en formato Protocol Buffer para su uso con BinDiff y otras herramientas de análisis externas. El complemento en C++ proporciona una representación intermedia estandarizada para la comparación de binarios y flujos de trabajo de análisis automatizado. Está dirigido a ingenieros de ingeniería inversa e investigadores de vulnerabilidades que realizan análisis y diffing binario entre herramientas.
