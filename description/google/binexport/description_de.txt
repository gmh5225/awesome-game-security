Dieses Projekt ist BinExport, ein Plugin zum Export von Binäranalysedaten für IDA Pro, Ghidra und Binary Ninja. Es serialisiert Disassembly-Daten einschließlich Funktionen, Basic Blocks, Instruktionen, Querverweisen und Call Graphs im Protocol-Buffer-Format zur Nutzung mit BinDiff und anderen externen Analysewerkzeugen. Das C++-Plugin stellt eine standardisierte Zwischenrepräsentation für Binärvergleiche und automatisierte Analyse-Workflows bereit. Es richtet sich an Reverse Engineers und Vulnerability Researcher, die werkzeugübergreifende Binäranalyse und Diffing durchführen.
