Este proyecto es ClassyShark, una herramienta independiente de inspección de binarios Android de Google. Analiza archivos APK, DEX, AAR y class, y muestra en una GUI interactiva jerarquías de clases, conteos de métodos, diseños de campos, dependencias y la estructura multidex. La herramienta Java permite un triage rápido de la complejidad de aplicaciones Android, el uso de SDK de terceros y las dependencias de bibliotecas nativas. Está dirigida a desarrolladores Android y analistas de seguridad que realizan análisis estructurales rápidos de APK y auditorías de dependencias.
