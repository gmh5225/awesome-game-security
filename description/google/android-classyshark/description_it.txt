Questo progetto è ClassyShark, uno strumento autonomo di ispezione di binari Android di Google. Analizza file APK, DEX, AAR e class, mostrando gerarchie di classi, conteggi dei metodi, layout dei campi, dipendenze e struttura multidex in una GUI interattiva. Lo strumento Java consente un triage rapido della complessità delle applicazioni Android, dell'uso di SDK di terze parti e delle dipendenze dalle librerie native. È rivolto a sviluppatori Android e analisti di sicurezza che eseguono analisi strutturali rapide degli APK e audit delle dipendenze.
