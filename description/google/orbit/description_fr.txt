Ce projet est Orbit, un profileur C++ autonome pour applications natives développé par Google. Il instrumente dynamiquement les fonctions à l'exécution sans modification du code source ni recompilation, et fournit le chronométrage des fonctions, les graphes d'appels, l'ordonnancement des threads, l'analyse des commutations de contexte et le suivi mémoire. L'outil repose sur une architecture client-serveur permettant le profilage distant sous Linux avec visualisation sous Windows. Il s'adresse aux développeurs de jeux et programmeurs système qui analysent des applications natives complexes et identifient les régressions de performances.
