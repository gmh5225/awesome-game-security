Este proyecto es Orbit, un perfilador C++ independiente para aplicaciones nativas desarrollado por Google. Instrumenta funciones de forma dinámica en tiempo de ejecución sin modificar el código fuente ni recompilar, y ofrece temporización de funciones, grafos de llamadas, programación de hilos, análisis de cambios de contexto y seguimiento de memoria. La herramienta usa una arquitectura cliente-servidor que permite el perfilado remoto en Linux con visualización en Windows. Está orientada a desarrolladores de juegos y programadores de sistemas que perfilan aplicaciones nativas complejas e identifican regresiones de rendimiento.
