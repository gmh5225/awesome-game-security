Этот проект — Orbit, автономный C++-профилировщик для нативных приложений, разработанный Google. Он динамически инструментирует функции во время выполнения без изменения исходного кода и перекомпиляции, предоставляя замеры времени функций, графы вызовов, планирование потоков, анализ переключений контекста и отслеживание памяти. Инструмент использует клиент-серверную архитектуру и поддерживает удалённое профилирование на Linux с визуализацией на Windows. Он предназначен для разработчиков игр и системных программистов, профилирующих сложные нативные приложения и выявляющих регрессии производительности.
