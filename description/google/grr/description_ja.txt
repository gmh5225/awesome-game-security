本プロジェクトは GRR Rapid Response であり、Python で構築されたリモートライブフォレンジック向けインシデント対応フレームワークです。Web ベースの管理インターフェースを備えたサーバーコンポーネントと、エンドポイントに配備され HTTP で通信する軽量クライアントエージェントで構成されます。フォレンジック成果物の収集、ファイル検索、メモリ抽出、レジストリデータのダンプ、および大規模なマシン群に対する Python ベースの任意分析フローの実行をサポートします。大規模なエンドポイントのトリアージと証拠収集を行うインシデント対応担当者、フォレンジック調査員、セキュリティ運用チーム向けです。
