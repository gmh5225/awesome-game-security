Этот проект — GRR Rapid Response, фреймворк реагирования на инциденты для удалённой live-форензики, написанный на Python. Он состоит из серверного компонента с веб-интерфейсом администрирования и лёгких клиентских агентов, развёрнутых на конечных точках и обменивающихся данными по HTTP. Фреймворк поддерживает сбор форензических артефактов, поиск файлов, извлечение памяти, дамп данных реестра и масштабируемое выполнение произвольных аналитических потоков на Python по большим паркам машин. Он предназначен для специалистов по реагированию на инциденты, форензик-исследователей и команд безопасности, выполняющих масштабный triage конечных точек и сбор доказательств.
