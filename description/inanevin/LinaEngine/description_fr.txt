Ce projet porte sur Lina Engine, un moteur de jeu 3D open source multiplateforme, léger et direct.
La motivation derrière Lina est d’abord l’apprentissage personnel, ainsi que la création d’une bibliothèque d’outils pour soi, comme LinaVG, une bibliothèque de graphiques vectoriels utilisée dans l’interface et les systèmes d’éditeur de Lina, ou LinaGX, un backend graphique complet Vulkan, DX12 et Metal.
Il est surtout utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
