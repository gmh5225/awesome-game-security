Этот проект — комплексный набор учебных материалов и примеров кода по DirectX 11 на базе Windows SDK. Он включает поэтапные примеры рендеринга с интеграцией ImGui, поддержку сборки через CMake и Visual Studio и охватывает темы от базового рендеринга треугольников до продвинутых техник вроде shadow mapping, deferred shading и compute shaders. В первую очередь полезен графическим программистам и разработчикам игр, изучающим программирование конвейера рендеринга DirectX 11.
