Este proyecto es una colección completa de tutoriales y código de ejemplo de DirectX 11 construida con el Windows SDK. Incluye ejemplos de renderizado progresivo con integración de ImGui, compatibilidad de compilación con CMake y Visual Studio, y abarca temas desde el renderizado básico de triángulos hasta técnicas avanzadas como shadow mapping, deferred shading y compute shaders. Resulta especialmente útil para programadores gráficos y desarrolladores de juegos que aprenden a programar el pipeline de renderizado de DirectX 11.
