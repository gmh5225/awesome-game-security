Questo progetto è una raccolta completa di tutorial ed esempi di codice DirectX 11 realizzata con il Windows SDK. Include esempi di rendering progressivi con integrazione ImGui, supporto di build per CMake e Visual Studio, e copre argomenti dal rendering di base dei triangoli fino a tecniche avanzate come shadow mapping, deferred shading e compute shader. È principalmente utile per programmatori grafici e sviluppatori di giochi che apprendono la programmazione della pipeline di rendering DirectX 11.
