Ce projet est une collection complète de tutoriels et d'exemples de code DirectX 11 construite avec le Windows SDK. Il propose des exemples de rendu progressifs avec intégration ImGui, une prise en charge de build CMake et Visual Studio, et couvre des sujets allant du rendu de triangle de base à des techniques avancées comme le shadow mapping, le deferred shading et les compute shaders. Il s'adresse principalement aux programmeurs graphiques et aux développeurs de jeux qui apprennent la programmation du pipeline de rendu DirectX 11.
