Este proyecto es una implementación de un juego de Cubo de Rubik 3D creada con DirectX en Windows.
Ofrece tanto una versión Direct3D 11 como una versión anterior de Direct3D 9, escritas en C++ con shaders HLSL y sistemas personalizados de renderizado, cámara y entrada.
El repositorio incluye el código fuente completo, los recursos y compilaciones de release ejecutables, con controles de teclado y ratón para manipular el cubo y medir el tiempo.
Está dirigido principalmente a desarrolladores que aprenden programación gráfica en tiempo real y arquitectura de juegos con pipelines nativos de DirectX.
