Questo progetto è un remake di un gioco shoot-’em-up per Windows sviluppato con tecnologie DirectX.
È scritto principalmente in C++ e combina rendering Direct3D, effetti shader HLSL e riproduzione audio DirectSound per un ciclo di gioco completo in stile arcade.
La codebase include file di dati configurabili per stage e nemici, supporto alle riprese (replay) e input opzionale con controller Xbox oltre ai controlli da tastiera.
Il pubblico principale è costituito da sviluppatori e studenti che studiano pattern pratici di programmazione di giochi nei framework grafici Windows nativi.
