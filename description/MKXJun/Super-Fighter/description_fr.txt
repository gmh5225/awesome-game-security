Ce projet est un remake de jeu de tir Windows développé avec les technologies DirectX.
Il est principalement écrit en C++ et combine le rendu Direct3D, les effets de shaders HLSL et la lecture audio DirectSound pour une boucle de jeu complète de style arcade.
La base de code inclut des fichiers de données configurables pour les stages et les ennemis, la prise en charge des replays, ainsi qu’une entrée optionnelle par manette Xbox en complément du clavier.
Son public principal est constitué de développeurs et d’étudiants qui étudient les motifs pratiques de programmation de jeux dans les frameworks graphiques Windows natifs.
