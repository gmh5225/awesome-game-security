Project Island is an experimental Vulkan renderer and realtime graphics framework for Linux (including Raspberry Pi 5) and Windows, written in C and C++. It is built for rapid prototyping and live tweaking, with hot-reloading for application modules, GLSL and Slang shaders, and imported images, plus a modular architecture aimed at very fast compile times. The Vulkan backend handles resource synchronization and validation, and the framework also supports GPU ray tracing, hardware-accelerated Vulkan Video decode, multi-window swapchains, Tracy profiling, and static release builds. Its primary use case is developers and researchers experimenting with modern realtime rendering and building visual prototypes without fighting a heavyweight engine.
