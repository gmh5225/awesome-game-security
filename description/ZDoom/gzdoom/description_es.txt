gzdoom es un source port del motor clásico de Doom orientado a modders, con soporte moderno de renderizado OpenGL y Vulkan.
La base de código es principalmente C y C++, abarca muchos subsistemas del motor y se distribuye bajo GPLv3 con mantenimiento activo de la comunidad.
Proporciona herramientas e infraestructura para extensiones de jugabilidad, mods de contenido, scripting y desarrollo multiplataforma del motor.
Su caso de uso principal es el trabajo con motores de juego de código abierto y el modding, y resulta relevante para investigadores que analizan la arquitectura de ejecución real de los juegos.
