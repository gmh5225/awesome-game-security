gzdoom est un port source du moteur Doom classique, conçu pour les moddeurs, avec un rendu moderne OpenGL et Vulkan.
Le code est principalement en C et C++, couvre de nombreux sous-systèmes du moteur, et est distribué sous GPLv3 avec une maintenance communautaire active.
Il fournit des outils et une infrastructure pour les extensions de gameplay, les mods de contenu, le scripting et le développement multiplateforme du moteur.
Son usage principal concerne le moteur de jeu open source et le modding ; il intéresse aussi les chercheurs qui analysent l’architecture d’exécution réelle des jeux.
