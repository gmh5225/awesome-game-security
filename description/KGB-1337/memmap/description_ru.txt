Этот проект — фреймворк на C++ для операций с памятью процессов при поддержке драйвера и для маппинга модулей.
Он определяет слой обмена запросами для чтения, записи, выделения, изменения защиты и запросов модулей.
Примерный сценарий включает расширение отображённых модулей и запуск выполнения через перехваченные пути вызовов API.
Предназначен для продвинутых экспериментов по манипуляции памятью, исследованию инъекций и анализу безопасности игр.
