Questo progetto è un framework C++ per operazioni sulla memoria di processo assistite da driver e per il mapping dei moduli.
Definisce un livello di comunicazione basato su richieste per lettura, scrittura, allocazione, modifiche della protezione e query sui moduli.
Il flusso di esempio include l'estensione di moduli mappati e l'avvio dell'esecuzione tramite percorsi di chiamate API hijackate.
È pensato per sperimentazione avanzata nella manipolazione della memoria, nella ricerca sull'injection e nell'analisi della sicurezza dei giochi.
