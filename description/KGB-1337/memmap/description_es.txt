Este proyecto es un framework en C++ para operaciones de memoria de procesos asistidas por controlador y mapeo de módulos.
Define una capa de comunicación basada en solicitudes para lectura, escritura, asignación, cambios de protección y consultas de módulos.
El flujo de ejemplo incluye extender módulos mapeados y desencadenar la ejecución mediante rutas de llamadas API secuestradas.
Está pensado para experimentación avanzada en manipulación de memoria, investigación de inyección y análisis de seguridad en juegos.
