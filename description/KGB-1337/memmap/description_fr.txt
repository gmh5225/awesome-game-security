Ce projet est un framework C++ pour les opérations mémoire de processus assistées par pilote et le mapping de modules.
Il définit une couche de communication basée sur des requêtes pour la lecture, l'écriture, l'allocation, les changements de protection et les requêtes de modules.
Le flux d'exemple comprend l'extension de modules mappés et le déclenchement de l'exécution via des chemins d'appels API détournés.
Il est destiné à l'expérimentation avancée en manipulation mémoire, recherche d'injection et analyse de sécurité des jeux.
