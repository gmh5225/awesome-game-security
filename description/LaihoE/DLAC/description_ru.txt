DLAC — пакет античита на основе глубокого обучения для анализа демо CS:GO, который предсказывает подозрительное поведение прицела для каждого выстрела.
Он сочетает код инференса на Python с парсером демо на Go и поддерживает модели ONNX Runtime с разными профилями размера и точности.
Пользователи могут экспортировать предсказания в вывод терминала, CSV или списки в памяти и настраивать пороги уверенности для практических рабочих процессов.
Проект ориентирован на эксперименты и исследования в области античита по поведенческому обнаружению на основе реплеев.
