DLAC est un package anti-triche par apprentissage profond pour l'analyse de démos CS:GO, qui prédit un comportement de visée suspect à chaque tir.
Il combine du code d'inférence Python avec un parseur de démos en Go et prend en charge des modèles ONNX Runtime selon différents profils de taille et de précision.
Les utilisateurs peuvent exporter les prédictions vers la sortie terminal, un CSV ou des listes en mémoire, et ajuster les seuils de confiance pour des flux de travail pratiques.
Le projet vise l'expérimentation et la recherche anti-triche sur la détection comportementale basée sur les replays.
