DLAC è un pacchetto anti-cheat basato su deep learning per l'analisi delle demo di CS:GO che predice un comportamento di mira sospetto per ogni colpo.
Combina codice di inferenza Python con un parser di demo in Go e supporta modelli ONNX Runtime con diversi profili di dimensione e accuratezza.
Gli utenti possono esportare le predizioni sull'output del terminale, in CSV o in liste in memoria e regolare le soglie di confidenza per flussi di lavoro pratici.
Il progetto è rivolto a sperimentazione e ricerca anti-cheat sul rilevamento comportamentale basato sui replay.
