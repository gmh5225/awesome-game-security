Этот проект — фреймворк деобфускации Android, который упрощает байткод Dalvik путём виртуального выполнения логики приложения. Он в основном написан на Java и включает компонент VM, проходы оптимизации и демонстрационные материалы для рабочих процессов анализа. Оптимизатор выполняет преобразования вроде распространения констант, удаления мёртвого кода и очистки reflection, чтобы восстановленный код было легче понять. Предназначен для реверс-инженеров и исследователей мобильной безопасности, анализирующих обфусцированные приложения.
