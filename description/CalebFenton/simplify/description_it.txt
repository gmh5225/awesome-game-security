Questo progetto è un framework di deoffuscazione Android che semplifica il bytecode Dalvik eseguendo virtualmente la logica dell'app. È scritto principalmente in Java e include un componente VM, passaggi di ottimizzazione e materiali dimostrativi per i flussi di analisi. L'ottimizzatore esegue trasformazioni come propagazione delle costanti, rimozione del codice morto e pulizia della reflection per rendere il codice recuperato più facile da comprendere. È rivolto a reverse engineer e ricercatori di sicurezza mobile che analizzano applicazioni offuscate.
