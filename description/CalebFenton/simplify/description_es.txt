Este proyecto es un framework de desofuscación para Android que simplifica el bytecode Dalvik ejecutando virtualmente la lógica de la aplicación. Está escrito principalmente en Java e incluye un componente de VM, pases de optimización y materiales de demostración para flujos de análisis. El optimizador realiza transformaciones como propagación de constantes, eliminación de código muerto y limpieza de reflexión para que el código recuperado sea más fácil de entender. Está dirigido a ingenieros de ingeniería inversa e investigadores de seguridad móvil que analizan aplicaciones ofuscadas.
