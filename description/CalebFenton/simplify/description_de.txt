Dieses Projekt ist ein Android-Deobfuscation-Framework, das Dalvik-Bytecode vereinfacht, indem es die App-Logik virtuell ausführt. Es ist hauptsächlich in Java geschrieben und umfasst eine VM-Komponente, Optimierungspässe sowie Demomaterialien für Analyse-Workflows. Der Optimizer führt Transformationen wie Konstantenpropagation, Entfernung toten Codes und Reflection-Bereinigung durch, um den wiederhergestellten Code leichter verständlich zu machen. Es richtet sich an Reverse Engineers und Mobile-Security-Forscher, die obfuskierte Anwendungen analysieren.
