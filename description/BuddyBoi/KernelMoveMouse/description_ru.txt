Этот проект — пример на C++ в режиме ядра, который перемещает мышь, применяя относительное движение из драйвера Windows. Он включает простой каркас драйвера в стиле WDK с проверками версии и обработкой смещений ядра для каждой сборки. Реализация намеренно ограничена и демонстрирует низкоуровневую манипуляцию вводом, а не устойчивое сокрытие. В основном полезен для изучения разработки драйверов и экспериментов с управлением игровым вводом на уровне ядра.
