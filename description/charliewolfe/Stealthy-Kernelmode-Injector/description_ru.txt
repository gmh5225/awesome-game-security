Этот проект — инжектор DLL в режиме ядра Windows, использующий скрытные техники для внедрения DLL в целевые процессы из пространства ядра. Он применяет методы вроде APC-инъекции, перехвата потоков или обратных вызовов загрузки образа и реализует меры противодействия обнаружению: удаление следов инъекции из списков модулей PEB и очистку метаданных выделенной памяти. Драйвер на C демонстрирует техники инъекции, рассчитанные на обход античита. Предназначен для исследователей ядра, изучающих скрытные методы инъекции и векторы их обнаружения.
