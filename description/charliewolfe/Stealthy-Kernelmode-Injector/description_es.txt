Este proyecto es un inyector de DLL en modo kernel de Windows que usa técnicas furtivas para inyectar DLL en procesos objetivo desde el espacio de kernel. Emplea métodos como inyección APC, secuestro de hilos o callbacks de carga de imagen, e implementa medidas antidetección como eliminar rastros de inyección de las listas de módulos del PEB y limpiar metadatos de memoria asignada. El controlador en C demuestra técnicas de inyección diseñadas para evadir la detección de anti-cheat. Está dirigido a investigadores de kernel que estudian métodos de inyección furtiva y sus vectores de detección.
