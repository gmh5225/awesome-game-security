Este proyecto es una prueba de concepto para Windows que protege punteros a funciones y entradas de vtable frente a manipulación en tiempo de ejecución. Usa hardware breakpoints o mecanismos de page guard para supervisar ubicaciones críticas de punteros y detectar cuando cheats o exploits intentan redirigir el flujo de ejecución sobrescribiendo punteros. La implementación en C/C++ demuestra comprobaciones defensivas de integridad para la seguridad de juegos y aplicaciones. Está dirigido a desarrolladores de anti-cheat e investigadores de seguridad que estudian técnicas de protección de la integridad de punteros.
