Questo progetto è un'implementazione open source autonoma dello stack API Zygisk per ambienti Android con root. È riscritto in larga parte in C e punta a un'architettura più pulita, binari più leggeri e compatibilità con setup basati su Magisk, KernelSU e APatch. Il repository include packaging dei moduli, integrazione delle dipendenze e strumenti operativi pensati per un'iniezione di processi stabile e hook a runtime su sistemi Android moderni. Il pubblico principale sono ricercatori di sicurezza Android e sviluppatori di moduli che necessitano di un layer runtime trasparente e compatibile con Zygisk.
