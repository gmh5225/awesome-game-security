Este proyecto es una implementación open source independiente de la pila de API Zygisk para entornos Android con root. Está reescrito en gran medida en C y se centra en una arquitectura más limpia, binarios más ligeros y compatibilidad con configuraciones basadas en Magisk, KernelSU y APatch. El repositorio incluye empaquetado de módulos, integración de dependencias y herramientas operativas pensadas para una inyección de procesos estable y hooks en tiempo de ejecución en sistemas Android modernos. Su público principal son investigadores de seguridad Android y desarrolladores de módulos que necesitan una capa de runtime transparente y compatible con Zygisk.
