本プロジェクトは、root 化された Android 環境向けの Zygisk API スタックを独自に実装したオープンソースです。主に C で書き直されており、よりクリーンなアーキテクチャ、軽量なバイナリ、および Magisk、KernelSU、APatch ベース環境との互換性を重視しています。リポジトリにはモジュールのパッケージング、依存関係の統合、運用ツールが含まれ、現代の Android システム上での安定したプロセス注入とランタイムフックを想定しています。主な対象は、透過的で Zygisk 互換のランタイム層を必要とする Android セキュリティ研究者とモジュール開発者です。
