Этот проект — основа для игровых читов на C++, которая продвигает чистую модульную архитектуру вместо монолитного кода функций.
Он даёт структурированные примеры с разделёнными провайдерами данных, модулями функций, раннерами и общим состоянием, чтобы логика массовой обработки объектов оставалась удобной в сопровождении.
В репозитории есть примеры проектов, включая внутренний пример для Unreal, демонстрирующий паттерны расширения на практике.
Основная аудитория — разработчики и исследователи, создающие поддерживаемые прототипы читов для экспериментов.
