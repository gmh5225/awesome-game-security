Этот проект — утилита для извлечения данных времени выполнения из игр на Unreal Engine и генерации пригодной для использования информации SDK.
Она написана на C++ и включает такие возможности, как обнаружение GNames и GObjects, дамп экземпляров и генерация SDK с интерфейсом на базе ImGui.
В кодовой базе также есть компоненты чтения и записи памяти с поддержкой ядра и профили конфигурации для различных целевых движков.
В основном используется в рабочих процессах реверс-инжиниринга Unreal для анализа игр и разработки инструментов.
