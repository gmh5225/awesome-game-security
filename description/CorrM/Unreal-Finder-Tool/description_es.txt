Este proyecto es una utilidad para extraer datos en tiempo de ejecución de juegos de Unreal Engine y generar información de SDK utilizable.
Está escrito en C++ e incluye funciones como el descubrimiento de GNames y GObjects, el volcado de instancias y la generación de SDK con una interfaz basada en ImGui.
El código también incluye componentes de lectura y escritura de memoria asistidos por el kernel y perfiles de configuración para distintos destinos de motor.
Se usa principalmente en flujos de ingeniería inversa de Unreal para el análisis de juegos y el desarrollo de herramientas.
