Ce projet est un utilitaire permettant d'extraire des données d'exécution des jeux Unreal Engine et de générer des informations SDK exploitables.
Il est écrit en C++ et propose notamment la découverte de GNames et GObjects, le dump d'instances, ainsi que la génération de SDK via une interface basée sur ImGui.
Le code inclut également des composants de lecture et d'écriture mémoire assistés par le noyau, ainsi que des profils de configuration pour différentes cibles de moteur.
Il est principalement utilisé dans les workflows de rétro-ingénierie Unreal pour l'analyse de jeux et le développement d'outils.
