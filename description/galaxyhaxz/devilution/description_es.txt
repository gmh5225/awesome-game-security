Ingeniería inversa completa del binario Windows retail original de Diablo 1 a código fuente C/C++ compilable, reconstruyendo todos los subsistemas del juego: generación de mazmorras (drlg_l1–l4), pipeline de renderizado, tablas de datos de hechizos/objetos/monstruos, red multijugador, serialización de guardado/carga y la biblioteca de archivos Storm MPQ. El proyecto conserva la estructura original del proyecto MSVC 4.20 e incluye las fuentes de DiabloUI y Storm DLL, con soporte de compilación para compiladores modernos junto a los archivos DSP/DSW originales.
