Ce projet se concentre sur mCP pour l'analyse à l'exécution et le piratage de processus ; ProcessHacker pour agents IA, extensible via des plugins DLL.
Il est principalement écrit en C++ et C/C++ et porte sur les pipelines d'assets, les outils d'éditeur et le développement de plugins.
Il est surtout utile aux développeurs de jeux, reverse engineers et créateurs d'outils travaillant dans le domaine du développement de jeux / mcp server.
