Этот проект сосредоточен на mCP для анализа во время выполнения и взлома процессов; ProcessHacker для ИИ-агентов, расширяемый DLL-плагинами.
Он в основном написан на C++ и C/C++ и посвящён пайплайнам ассетов, инструментам редактора и разработке плагинов.
Он особенно полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов, работающим в области разработки игр / mcp server.
