Questo progetto si concentra su mCP per l'analisi runtime e il process hacking; ProcessHacker per agenti IA, estensibile con plugin DLL.
È scritto principalmente in C++ e C/C++ e ruota attorno a pipeline di asset, tooling per editor e sviluppo di plugin.
È soprattutto utile per sviluppatori di giochi, reverse engineer e costruttori di tooling che operano nell'area game develop / mcp server.
