Нативный бинарный модуль Android (ARM64), внедряющий виртуальные события касания в подсистему ввода Linux путём записи в узлы устройств /dev/input/event* через фреймворк uinput. Собран с помощью Android NDK; программно имитирует нажатия, свайпы и мультитач-жесты для автоматизации игр и тестирования ввода на устройствах с root-доступом.
