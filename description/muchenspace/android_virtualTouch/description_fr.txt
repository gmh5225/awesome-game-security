Binaire natif Android (ARM64) qui injecte des événements tactiles virtuels dans le sous-système d'entrée Linux en écrivant sur les nœuds de périphérique /dev/input/event* via le framework uinput. Construit avec le Android NDK, il simule de manière programmatique les gestes de tap, de balayage et multi-touch pour l'automatisation de jeux et les tests d'entrée sur des appareils rootés.
