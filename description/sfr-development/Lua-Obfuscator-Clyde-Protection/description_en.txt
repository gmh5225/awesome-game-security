Clyde is a from-scratch Luau obfuscator and protection toolkit that turns Luau source into hard-to-reverse scripts through a full lexer, parser, AST, obfuscator, and virtual-machine pipeline written in TypeScript. It supports the full Luau grammar, including type annotations, and applies multi-pass AST transforms such as identifier renaming, string encoding, and control-flow scrambling with opaque predicates. For stronger protection it can compile code into stack-based or register-based VMs with configurable levels that may shuffle opcodes, encode constants, use polymorphic dispatch, and LZMA-compress bytecode into a self-contained Luau script. The project exposes both a CLI for batch or CI use and a browser UI served by an Express server, targeting developers who need to protect Luau game or client scripts against reverse engineering.
