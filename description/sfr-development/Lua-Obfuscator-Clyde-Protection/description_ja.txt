Clyde はゼロから構築された Luau 用オブファスケータ／保護ツールキットで、TypeScript で書かれた字句解析器、パーサ、AST、オブファスケータ、仮想マシンの一連のパイプラインにより、Luau ソースを解析・難読化しにくいスクリプトに変換します。型注釈を含む完全な Luau 文法に対応し、識別子のリネーム、文字列エンコード、不透明述語を用いた制御フロー攪乱など、多パスの AST 変換を適用します。より強い保護のため、スタックベースまたはレジスタベースの VM へコンパイルでき、オペコードのシャッフル、定数のエンコード、ポリモーフィック・ディスパッチ、LZMA によるバイトコード圧縮を自己完結型 Luau スクリプトにまとめるなど、設定可能なレベルを備えます。バッチ処理や CI 向けの CLI と、Express サーバが提供するブラウザ UI の両方を公開しており、Luau のゲーム／クライアントスクリプトをリバースエンジニアリングから守りたい開発者を対象としています。
