Clyde ist ein von Grund auf entwickeltes Luau-Obfuscator- und Schutz-Toolkit, das Luau-Quellcode über eine vollständige Pipeline aus Lexer, Parser, AST, Obfuscator und virtueller Maschine in TypeScript in schwer zu reverse-engineerende Skripte verwandelt. Es unterstützt die vollständige Luau-Grammatik einschließlich Typannotationen und wendet mehrstufige AST-Transformationen an, etwa Umbenennung von Bezeichnern, String-Kodierung und Control-Flow-Scrambling mit opaken Prädikaten. Für stärkeren Schutz kann Code in stack- oder registerbasierte VMs mit konfigurierbaren Stufen kompiliert werden, die Opcodes mischen, Konstanten kodieren, polymorphes Dispatch nutzen und Bytecode per LZMA in ein eigenständiges Luau-Skript komprimieren. Das Projekt bietet sowohl eine CLI für Batch- oder CI-Einsatz als auch eine Browser-UI, die von einem Express-Server bereitgestellt wird, und richtet sich an Entwickler, die Luau-Spiel- oder Client-Skripte vor Reverse Engineering schützen müssen.
