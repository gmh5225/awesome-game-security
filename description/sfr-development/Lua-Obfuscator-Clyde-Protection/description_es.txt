Clyde es un kit de ofuscación y protección de Luau creado desde cero que convierte el código fuente Luau en scripts difíciles de ingeniar a la inversa mediante un pipeline completo de lexer, parser, AST, ofuscador y máquina virtual escrito en TypeScript. Admite toda la gramática de Luau, incluidas las anotaciones de tipos, y aplica transformaciones AST multipaso como el renombrado de identificadores, la codificación de cadenas y el barajado del flujo de control con predicados opacos. Para una protección más fuerte puede compilar el código a VMs basadas en pila o en registros con niveles configurables que pueden barajar opcodes, codificar constantes, usar despacho polimórfico y comprimir el bytecode con LZMA en un script Luau autocontenido. El proyecto expone tanto una CLI para uso por lotes o en CI como una interfaz de navegador servida por un servidor Express, dirigida a desarrolladores que necesitan proteger scripts Luau de juego o de cliente frente a la ingeniería inversa.
