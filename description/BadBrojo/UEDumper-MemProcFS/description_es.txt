Este proyecto es un dumper de Unreal Engine todo en uno y un editor en vivo para la extracción de SDK en tiempo de ejecución y la inspección de memoria. Está escrito en C++ con una interfaz basada en ImGui y admite una amplia gama de versiones de Unreal Engine mediante offsets y definiciones de motor configurables. El código incluye módulos para generación de SDK, exploración de clases, operaciones de lectura/escritura en vivo e integración con backends de memoria al estilo MemProcFS/LeechCore. El caso de uso principal es la ingeniería inversa de juegos Unreal y el análisis de seguridad de juegos.
