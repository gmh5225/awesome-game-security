Ce projet se concentre sur le 3D en C++. Pour développer une application C++ qui utilise Wicked Engine, vous pouvez compiler le projet de bibliothèque statique WickedEngine_Windows et l’y lier. Il est surtout utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
