Este proyecto se centra en el 3D con C++. Si quieres desarrollar una aplicación C++ que use Wicked Engine, puedes compilar el proyecto de biblioteca estática WickedEngine_Windows y enlazarlo. Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores de gráficos que trabajan en el ámbito de motores de juego / código fuente.
