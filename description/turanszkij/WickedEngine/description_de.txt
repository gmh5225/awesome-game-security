Dieses Projekt konzentriert sich auf C++-3D. Wenn Sie eine C++-Anwendung entwickeln möchten, die Wicked Engine nutzt, können Sie das statische Bibliotheksprojekt WickedEngine_Windows bauen und dagegen linken. Es ist vor allem für Spieleentwickler, Engine-Programmierer und Grafikforscher im Bereich Game Engine / Source nützlich.
