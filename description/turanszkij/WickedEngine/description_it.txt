Questo progetto si concentra sul 3D in C++. Se vuoi sviluppare un’applicazione C++ che usa Wicked Engine, puoi compilare il progetto di libreria statica WickedEngine_Windows e collegarlo. È soprattutto utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell’ambito dei game engine / del codice sorgente.
