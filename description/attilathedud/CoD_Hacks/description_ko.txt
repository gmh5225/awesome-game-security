이 프로젝트는 Call of Duty v1.5용 게임 핵 모음으로, 다양한 치트 구현 기법을 보여줍니다.
Desk.dll(월핵, chams, 나이트 모드, 안개 제거가 포함된 멀티 핵), 월드-투-스크린 후킹을 이용한 내부 ESP, glDrawElements 후킹을 통한 OpenGL 월핵, 시스템 콜 기반 내부 월핵, 그리고 패턴 스캐닝이 포함된 트레이너가 들어 있습니다.
이 C++ 코드베이스는 내부 DLL 인젝션과 외부 트레이너 접근 방식을 모두 다루며 상세한 스크린샷도 제공합니다.
OpenGL 후킹, 게임 루프 가로채기, 패턴 기반 메모리 스캐닝을 포함한 고전 FPS 치트 아키텍처를 연구하는 게임 보안 연구자에게 주로 유용합니다.
