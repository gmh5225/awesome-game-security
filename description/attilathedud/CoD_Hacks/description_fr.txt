Ce projet est une collection de hacks pour Call of Duty v1.5 illustrant diverses techniques d’implémentation de cheats.
Il comprend Desk.dll (un multi-hack avec wallhack, chams, mode nuit et no-fog), un ESP interne basé sur un hook world-to-screen, un wallhack OpenGL via le hook de glDrawElements, un wallhack interne basé sur les syscalls, ainsi qu’un trainer avec pattern scanning.
La base de code C++ couvre à la fois l’injection de DLL interne et les approches de trainer externe, avec des captures d’écran détaillées.
Elle est surtout utile aux chercheurs en sécurité des jeux qui étudient les architectures de cheats FPS classiques, notamment le hooking OpenGL, l’interception de la boucle de jeu et le scan mémoire par motifs.
