Ghidra-Obfuscation-Detection è uno script Ghidra per individuare funzioni potenzialmente offuscate o insolitamente complesse. Applica un’estrazione euristica di caratteristiche sui corpi delle funzioni così che gli analisti possano dare rapidamente priorità alle regioni di codice sospette durante sessioni di reverse engineering. Il progetto è implementato in Java per un uso diretto nell’ambiente di scripting di Ghidra ed è leggero da integrare nei flussi di analisi esistenti. È rivolto principalmente a ricercatori di malware e di binari di giochi che necessitano di un triage più rapido del codice protetto.
