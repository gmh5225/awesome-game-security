Ghidra-Obfuscation-Detection est un script Ghidra destiné à repérer des fonctions potentiellement obfusquées ou anormalement complexes. Il applique une extraction heuristique de caractéristiques sur les corps de fonctions afin que les analystes puissent rapidement prioriser les zones de code suspectes lors de sessions de reverse engineering. Le projet est implémenté en Java pour une utilisation directe dans l’environnement de scripting de Ghidra et s’intègre facilement aux flux d’analyse existants. Il s’adresse principalement aux chercheurs en malwares et en binaires de jeux qui ont besoin d’un triage plus rapide du code protégé.
