Ghidra-Obfuscation-Detection es un script de Ghidra para detectar funciones potencialmente ofuscadas o inusualmente complejas. Aplica extracción heurística de características sobre los cuerpos de las funciones para que los analistas puedan priorizar rápidamente regiones de código sospechosas durante sesiones de ingeniería inversa. El proyecto está implementado en Java para usarse directamente en el entorno de scripting de Ghidra y es ligero de integrar en flujos de análisis existentes. Está pensado principalmente para investigadores de malware y binarios de juegos que necesitan un triage más rápido del código protegido.
