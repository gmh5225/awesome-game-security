Ghidra-Obfuscation-Detection ist ein Ghidra-Skript zum Erkennen potenziell obfuskierter oder ungewöhnlich komplexer Funktionen. Es wendet heuristische Merkmalsextraktion auf Funktionskörper an, damit Analysten verdächtige Codebereiche bei der Reverse Engineering-Arbeit schnell priorisieren können. Das Projekt ist in Java für den direkten Einsatz in der Ghidra-Skriptumgebung umgesetzt und lässt sich leicht in bestehende Analyseabläufe integrieren. Es richtet sich vor allem an Malware- und Game-Binary-Forscher, die geschützten Code schneller triageieren müssen.
