Ghidra-Obfuscation-Detection — это скрипт для Ghidra, предназначенный для выявления потенциально обфусцированных или необычно сложных функций. Он применяет эвристическое извлечение признаков к телам функций, чтобы аналитики могли быстро расставлять приоритеты по подозрительным участкам кода при реверс-инжиниринге. Проект реализован на Java для прямого использования в среде скриптов Ghidra и легко встраивается в существующие рабочие процессы анализа. Он в первую очередь ориентирован на исследователей вредоносного ПО и игровых бинарников, которым нужна более быстрая сортировка защищённого кода.
