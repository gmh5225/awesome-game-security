Questo progetto è Stride (precedentemente Xenko), un motore di gioco C# open source con un editor completo. Offre rendering PBR, illuminazione globale, un sistema di scripting visuale, fisica (Bullet), audio, UI, animazione, sistemi particellari e supporto VR. Il motore usa una pipeline di rendering moderna con backend Vulkan, Direct3D 11/12 e OpenGL. È rivolto agli sviluppatori di giochi che creano giochi 3D e applicazioni interattive con C# e .NET.
