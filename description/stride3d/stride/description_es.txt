Este proyecto es Stride (antes Xenko), un motor de juegos C# de código abierto con un editor completo. Ofrece renderizado PBR, iluminación global, un sistema de scripting visual, física (Bullet), audio, UI, animación, sistemas de partículas y soporte VR. El motor usa un pipeline de renderizado moderno con backends Vulkan, Direct3D 11/12 y OpenGL. Está dirigido a desarrolladores de juegos que crean juegos 3D y aplicaciones interactivas con C# y .NET.
