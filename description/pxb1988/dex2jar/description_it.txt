Questo progetto è dex2jar, un insieme di strumenti Java per convertire il bytecode Android DEX in file di classe Java (formato JAR) e viceversa. I file JAR convertiti possono poi essere analizzati con decompilatori Java standard come JD-GUI o CFR. Il set di strumenti include anche d2j-baksmali per il disassemblaggio e utilità per la firma di APK e la manipolazione di DEX. È rivolto a reverse engineer Android e analisti di sicurezza che devono esaminare la logica delle applicazioni Android con strumenti di analisi Java familiari.
