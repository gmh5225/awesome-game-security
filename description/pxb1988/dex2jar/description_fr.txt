Ce projet est dex2jar, un ensemble d'outils Java permettant de convertir le bytecode Android DEX en fichiers de classes Java (format JAR) et inversement. Les fichiers JAR obtenus peuvent ensuite être analysés avec des décompilateurs Java standards tels que JD-GUI ou CFR. La boîte à outils comprend également d2j-baksmali pour le désassemblage, ainsi que des utilitaires pour la signature d'APK et la manipulation de DEX. Elle s'adresse aux ingénieurs en rétro-ingénierie Android et aux analystes en sécurité qui doivent examiner la logique des applications Android avec des outils d'analyse Java familiers.
