Этот проект — dex2jar, набор Java-инструментов для преобразования байткода Android DEX в файлы классов Java (формат JAR) и обратно. Полученные JAR-файлы можно затем анализировать стандартными Java-декомпиляторами, такими как JD-GUI или CFR. В набор также входят d2j-baksmali для дизассемблирования и утилиты для подписания APK и манипуляций с DEX. Он предназначен для Android-реверс-инженеров и специалистов по безопасности, которым нужно изучать логику приложений Android привычными средствами анализа Java.
