Este proyecto es dex2jar, un conjunto de herramientas Java para convertir el bytecode Android DEX a archivos de clase Java (formato JAR) y viceversa. Los archivos JAR resultantes pueden analizarse después con descompiladores Java estándar como JD-GUI o CFR. El conjunto también incluye d2j-baksmali para el desensamblado, así como utilidades para firmar APK y manipular DEX. Está orientado a ingenieros de ingeniería inversa de Android y analistas de seguridad que necesitan examinar la lógica de las aplicaciones Android con herramientas de análisis Java familiares.
