Obfuscate — это header-only библиотека на C++14 для обфускации строковых литералов на этапе компиляции.
Она шифрует литералы с помощью constexpr-логики и случайных ключей, а расшифровывает их во время выполнения по необходимости.
API намеренно прост: разработчики оборачивают строки макросом и встраивают библиотеку в существующий код с минимальными изменениями.
Основной сценарий использования — усложнить тривиальное извлечение строк в бинарниках, связанных с реверс-инжинирингом и игровой безопасностью.
