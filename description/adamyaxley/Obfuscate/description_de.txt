Obfuscate ist eine header-only C++14-Bibliothek zur Obfuskation von String-Literalen zur Compile-Zeit.
Sie verschlüsselt Literale mit constexpr-Logik und zufälligen Schlüsseln und entschlüsselt sie zur Laufzeit bei Bedarf.
Die API ist bewusst einfach gehalten: Entwickler können Strings mit einem Makro wrappen und sie mit minimalen Änderungen in bestehenden Code integrieren.
Der Hauptanwendungsfall ist, triviale String-Extraktion in Binärdateien aus Reverse Engineering und Gamesicherheit zu erschweren.
