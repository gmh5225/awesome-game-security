Проект посвящён рефакторингу duckDetector: локальному инспектору целостности Android-устройств для выявления root-вмешательства, runtime-хукинга, аномалий монтирования, доверия attestation и признаков виртуализации.
Он объединяет UI на Jetpack Compose, модульные Kotlin-пакеты и нативные зонды на C++ / ассемблере, чтобы показывать карточки детекторов со структурированными находками, покрытием методов и сводками состояния сканирования.
В первую очередь полезен исследователям игровой безопасности и reverse-инженерам, изучающим атакующие техники в области читов / Android root.
