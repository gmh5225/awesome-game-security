本プロジェクトは duckDetector のリファクタリングに焦点を当て、root 改ざん、ランタイムフッキング、マウント異常、アテステーションの信頼性、仮想化の痕跡を対象とする Android ローカルデバイス整合性検査ツールです。
Jetpack Compose UI、モジュール化された Kotlin 機能パッケージ、およびネイティブ C++ / アセンブリプローブを組み合わせ、構造化された検出結果、手法カバレッジ、スキャン状態の要約を検出カードとして提示します。
主に、チート / Android root 領域で攻撃手法を研究するゲームセキュリティ研究者やリバースエンジニア向けです。
