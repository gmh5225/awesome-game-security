Emulatore di server privato in Java/Kotlin per MapleStory v143 (CMS), che implementa l'intero stack del server di gioco, tra cui crittografia di login (LoginCrypto), gestione dei pacchetti via Netty/MINA, gestione delle statistiche del personaggio, sistemi di skill/buff, spawn dei mostri, scripting delle quest (Nashorn) e un tracker anti-cheat lato server (CheatTracker) con categorizzazione delle infrazioni.
