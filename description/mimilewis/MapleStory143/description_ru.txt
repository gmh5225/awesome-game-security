Эмулятор приватного сервера на Java/Kotlin для MapleStory v143 (CMS), реализующий полный игровой серверный стек, включая шифрование входа (LoginCrypto), обработку пакетов через Netty/MINA, управление характеристиками персонажа, системы навыков/баффов, спавн монстров, скриптинг квестов (Nashorn) и серверный античит-трекер (CheatTracker) с категоризацией нарушений.
