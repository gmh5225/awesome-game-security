Questo progetto è un server Model Context Protocol che collega gli strumenti per modelli di linguaggio di grandi dimensioni ai dati di analisi di IDA Pro.
È implementato in Python e include sia un pacchetto server autonomo sia un plugin IDA per la comunicazione bidirezionale.
Il sistema offre capacità orientate all'automazione per interrogare il contesto di disassemblaggio, eseguire flussi di analisi e integrarsi con client compatibili MCP.
Il pubblico principale sono i reverse engineer che desiderano un'interazione assistita dall'IA con i database IDA.
