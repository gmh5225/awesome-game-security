Este proyecto es un servidor Model Context Protocol que conecta las herramientas de modelos de lenguaje grandes con los datos de análisis de IDA Pro.
Está implementado en Python e incluye tanto un paquete de servidor independiente como un complemento de IDA para comunicación bidireccional.
El sistema ofrece capacidades orientadas a la automatización para consultar el contexto de desensamblado, ejecutar flujos de análisis e integrarse con clientes compatibles con MCP.
Su público principal son los ingenieros de ingeniería inversa que desean una interacción asistida por IA con bases de datos de IDA.
