Этот проект — сервер Model Context Protocol, который связывает инструменты больших языковых моделей с данными анализа IDA Pro.
Он реализован на Python и включает как автономный серверный пакет, так и плагин IDA для двусторонней связи.
Система предоставляет возможности, ориентированные на автоматизацию: запросы контекста дизассемблирования, запуск рабочих процессов анализа и интеграцию с MCP-совместимыми клиентами.
Основная аудитория — инженеры по реверс-инжинирингу, которым нужно взаимодействие с базами данных IDA при поддержке ИИ.
