Ce projet est un serveur Model Context Protocol qui relie les outils de grands modèles de langage aux données d'analyse d'IDA Pro.
Il est implémenté en Python et comprend à la fois un package serveur autonome et un plugin IDA pour une communication bidirectionnelle.
Le système offre des capacités orientées automation pour interroger le contexte de désassemblage, exécuter des workflows d'analyse et s'intégrer à des clients compatibles MCP.
Son public principal est les ingénieurs en reverse engineering qui souhaitent une interaction assistée par IA avec les bases de données IDA.
