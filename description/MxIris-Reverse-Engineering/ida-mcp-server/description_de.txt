Dieses Projekt ist ein Model Context Protocol-Server, der Werkzeuge für große Sprachmodelle mit IDA Pro-Analysedaten verbindet.
Es ist in Python implementiert und umfasst sowohl ein eigenständiges Serverpaket als auch ein IDA-Plugin für bidirektionale Kommunikation.
Das System bietet automationsorientierte Funktionen zum Abfragen von Disassembly-Kontext, Ausführen von Analyse-Workflows und zur Integration mit MCP-kompatiblen Clients.
Die Hauptzielgruppe sind Reverse Engineers, die eine KI-gestützte Interaktion mit IDA-Datenbanken wünschen.
