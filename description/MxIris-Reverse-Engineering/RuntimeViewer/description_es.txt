Este proyecto es una aplicación de inspección en tiempo de ejecución para plataformas Apple que explora metadatos de Objective-C y Swift a partir de binarios y frameworks cargados.
Está construido principalmente en Swift con componentes de Objective-C e incluye capas de comunicación para acceso al runtime local o en red.
Las funciones principales incluyen exploración de interfaces, vistas con resaltado de sintaxis, exportación de interfaces descubiertas, carga de frameworks y soporte de inyección de código en desarrollo.
Su caso de uso principal es la ingeniería inversa, el análisis dinámico y la exploración en tiempo de ejecución para macOS y objetivos relacionados de plataformas Apple.
