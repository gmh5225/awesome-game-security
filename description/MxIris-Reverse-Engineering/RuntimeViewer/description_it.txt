Questo progetto è un'applicazione di ispezione runtime per piattaforme Apple che esplora i metadati Objective-C e Swift da binari e framework caricati.
È realizzato principalmente in Swift con componenti Objective-C e include livelli di comunicazione per l'accesso runtime locale o in rete.
Le funzionalità principali comprendono la navigazione delle interfacce, viste con evidenziazione della sintassi, esportazione delle interfacce scoperte, caricamento di framework e supporto per l'iniezione di codice ancora in fase di sviluppo.
Il caso d'uso principale è reverse engineering, analisi dinamica ed esplorazione runtime per macOS e target correlati delle piattaforme Apple.
