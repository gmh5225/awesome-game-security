Этот проект — приложение для инспекции во время выполнения на платформах Apple, которое исследует метаданные Objective-C и Swift из загруженных бинарников и фреймворков.
Оно в основном написано на Swift с компонентами на Objective-C и включает коммуникационные слои для локального или сетевого доступа к runtime.
Основные возможности: просмотр интерфейсов, представления с подсветкой синтаксиса, экспорт обнаруженных интерфейсов, загрузка фреймворков и поддержка внедрения кода, пока находящаяся в разработке.
Основной сценарий использования — реверс-инжиниринг, динамический анализ и исследование runtime для macOS и связанных целевых платформ Apple.
