Dieses Projekt ist ein minimales Beispiel für eine Silent-Aim-Technik in Unreal Engine 4, basierend auf Hooks von Viewpoint-Funktionen.
Es ist in C++ geschrieben und zeigt, wie Spielerkamera- und View-APIs gehookt werden, um die Zielrichtung zu ändern und dabei das sichtbare View-Verhalten beizubehalten.
Der Code konzentriert sich darauf, Rotation und die Logik zur Auswahl von Ziel-Bones während einer eingabegetriggerten Ausführung zu manipulieren.
Der primäre Anwendungsfall ist die Spielsicherheitsforschung und das Verständnis von Aim-Manipulationsvektoren in UE4-Titeln.
