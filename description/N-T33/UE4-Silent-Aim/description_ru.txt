Этот проект — минимальный пример техники silent-aim для Unreal Engine 4 на основе хуков функций точки обзора.
Он написан на C++ и демонстрирует перехват API камеры и вида игрока для изменения направления прицеливания с сохранением видимого поведения обзора.
Код сосредоточен на манипуляции вращением и логикой выбора целевых костей (bone) при выполнении, инициированном вводом.
Основной сценарий использования — исследование безопасности игр и понимание векторов манипуляции прицеливанием в проектах на UE4.
