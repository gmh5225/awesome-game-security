Ce projet se concentre sur la VM. Exemple d'utilisation : protéger un programme. L'implémentation de la VM est soit incluse dans le programme cible par liaison statique, soit copiée manuellement dans l'exécutable par le virtualiseur. Il est surtout utile aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant sur les anti-cheats / moteurs d'obfuscation.
