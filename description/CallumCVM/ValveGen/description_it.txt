Questo progetto è uno strumento di generazione SDK per titoli Valve Source Engine. È implementato in C++ e sembra analizzare strutture di classi di rete e data-table per generare definizioni di classi e offset utilizzabili. Il repository include file di progetto e codice di supporto per costruire strutture generate dai metadati del client. È soprattutto utile per reverse engineering di giochi e flussi di tooling che si basano su rappresentazioni SDK aggiornate del motore Source.
