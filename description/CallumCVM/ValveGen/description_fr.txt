Ce projet est un outil de génération de SDK pour les jeux Valve Source Engine. Il est implémenté en C++ et semble analyser les structures de classes réseau et de tables de données afin d’émettre des définitions de classes et des offsets utilisables. Le dépôt inclut des fichiers de projet et du code associé pour construire des structures générées à partir des métadonnées client. Il est surtout utile pour la rétro-ingénierie de jeux et les flux d’outils qui s’appuient sur des représentations SDK Source à jour.
