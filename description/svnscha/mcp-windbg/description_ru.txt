Этот проект — MCP-сервер (Model Context Protocol), интегрирующийся с CDB/WinDbg, чтобы модели ИИ могли анализировать дампы аварий Windows и подключаться к удалённым сеансам отладки. Сервер на Python оборачивает сеансы CDB, предоставляет промпты для triage дампов и включает примеры программ, демонстрирующие типичные сценарии сбоев: разыменование нулевого указателя, переполнения кучи и ошибки деления на ноль. Он в первую очередь полезен реверс-инженерам и разработчикам ядра, которым нужен ИИ-усиленный анализ дампов аварий и автоматизированные рабочие процессы отладки.
