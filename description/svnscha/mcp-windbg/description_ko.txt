이 프로젝트는 CDB/WinDbg와 통합되어 AI 모델이 Windows 크래시 덤프를 분석하고 원격 디버깅 세션에 연결할 수 있게 하는 MCP(Model Context Protocol) 서버입니다. Python 기반 서버는 CDB 세션을 래핑하고, 덤프 분류(트리아지) 프롬프트를 제공하며, null 포인터 역참조, 힙 오버플로, 0으로 나누기 오류 등 일반적인 크래시 시나리오를 보여주는 예제 프로그램을 포함합니다. AI로 강화된 크래시 덤프 분석과 자동화된 디버깅 워크플로를 원하는 리버스 엔지니어와 커널 개발자에게 주로 유용합니다.
