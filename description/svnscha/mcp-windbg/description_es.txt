Este proyecto es un servidor MCP (Model Context Protocol) que se integra con CDB/WinDbg para permitir que los modelos de IA analicen volcados de memoria de fallos de Windows y se conecten a sesiones de depuración remota. El servidor basado en Python encapsula sesiones CDB, proporciona prompts de triage de dumps e incluye programas de ejemplo que demuestran escenarios de fallo comunes como desreferencias de puntero nulo, desbordamientos de montículo y errores de división por cero. Resulta especialmente útil para ingenieros de ingeniería inversa y desarrolladores de kernel que buscan análisis de dumps de fallos aumentados con IA y flujos de depuración automatizados.
