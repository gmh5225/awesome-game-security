Questo progetto è un server MCP (Model Context Protocol) che si integra con CDB/WinDbg per consentire ai modelli di IA di analizzare i crash dump di Windows e di connettersi a sessioni di debugging remoto. Il server basato su Python wrappa le sessioni CDB, fornisce prompt di triage dei dump e include programmi di esempio che dimostrano scenari di crash comuni come dereferenziazioni di puntatori null, heap overflow ed errori di divisione per zero. È principalmente utile per reverse engineer e sviluppatori kernel che cercano analisi di crash dump potenziata dall'IA e flussi di debugging automatizzati.
