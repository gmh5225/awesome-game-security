Этот репозиторий — proof of concept для ядра Windows по перехвату уведомлений о процессах через механизм таблицы расширений BAM. Вместо стандартного массива process notify callback он нацелен на путь extension host и подменяет указатель BAM callback на собственную процедуру. Код драйвера демонстрирует поиск по смещениям в ntoskrnl, замену указателя callback и временную работу с маской уведомлений. Предназначен для продвинутых исследований античита и EDR по обходу callback, защитной видимости и недокументированным внутренностям ядра.
