Ce projet est un framework Lua multi-utilisateurs prenant en charge le modèle d'acteurs, souvent utilisé dans les jeux.
Il est largement utilisé dans l'industrie du jeu chinoise, mais se répand désormais dans d'autres secteurs et auprès des développeurs anglophones.
Il est principalement utile aux développeurs backend, multijoueur et de jeux en ligne travaillant dans le domaine des réseaux de jeu / du code source.
