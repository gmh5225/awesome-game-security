Este proyecto es un framework Lua multiusuario que admite el modelo de actores y se usa a menudo en juegos.
Se utiliza ampliamente en la industria del juego china, pero ahora también se está extendiendo a otros sectores y a desarrolladores de habla inglesa.
Resulta especialmente útil para desarrolladores de backend, multijugador y juegos en línea que trabajan en el área de redes de juego / código fuente.
