Этот проект — универсальный дампер Unreal Engine и редактор памяти в реальном времени.
Он поддерживает версии Unreal Engine с 4.19 по 5.3.0 и включает генерацию SDK, дамп структур и утилиты просмотра во время выполнения.
Реализация написана на C++ с интерфейсом на ImGui и компонентами для управления объектами, генерации типов и специфичной для движка обработки.
Он широко полезен для реверс-инжиниринга игр на Unreal и рабочих процессов исследований игровой безопасности.
