Этот проект — лёгкая вспомогательная DMA-библиотека для рабочих процессов доступа к памяти игр.
Написана на C++ и оборачивает типовые операции: поиск процесса, получение базового адреса, чтение и запись памяти, сканирование по сигнатурам и scatter-чтения.
Рассчитана на работу с DMA-связанными компонентами, такими как LeechCore и MemProcFS, через внешне поставляемые библиотеки.
Основной сценарий использования — создание DMA-инструментов для исследований игровой безопасности с более чистым и переиспользуемым кодом.
