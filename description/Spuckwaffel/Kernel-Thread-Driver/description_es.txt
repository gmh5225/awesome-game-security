Este proyecto es un controlador de hilos del kernel de Windows junto con un controlador en modo usuario para operaciones de memoria.
Utiliza un modelo de comunicación por códigos de estado entre el kernel y el modo usuario para inicializar, rastrear el estado de la conexión y procesar comandos.
La implementación se centra en tareas prácticas como la configuración del proceso objetivo, la lectura de memoria y la obtención de la base de módulos.
Está pensado para la investigación de bypass de anti-cheat y experimentos de arquitectura kernel-usuario en juegos protegidos.
