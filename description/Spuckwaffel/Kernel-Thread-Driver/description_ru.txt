Этот проект — драйвер потоков ядра Windows в паре с пользовательским контроллером для операций с памятью.
Он использует модель обмена статусными кодами между ядром и пользовательским режимом для инициализации, отслеживания состояния соединения и обработки команд.
Реализация ориентирована на практические задачи: настройка целевого процесса, чтение памяти и получение базовых адресов модулей.
Предназначен для исследований обхода античита и экспериментов с архитектурой ядро–пользовательский режим в защищённых играх.
