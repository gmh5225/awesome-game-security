Dieses Projekt ist ein edukativer Windows-Kerneltreiber, der MmCopyMemory hookt.
Es demonstriert eine einfache Hook-Implementierung, um das Speicher-Kopierverhalten von Anti-Cheat-Scannern zu beobachten.
Der Code ist bewusst einfach und dokumentiert gehalten und wird ausdrücklich als in realen Umgebungen instabil und nachweisbar beschrieben.
Der Hauptzweck ist das Verständnis von Kernel-Hook-Mechanismen und der Telemetrie-Inspektion von Anti-Cheat-Systemen.
