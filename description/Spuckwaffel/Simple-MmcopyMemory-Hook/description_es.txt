Este proyecto es un controlador de kernel de Windows educativo que hace hook de MmCopyMemory.
Demuestra una implementación básica de hook para observar el comportamiento de copia de memoria usado por los escáneres anti-cheat.
El código es intencionadamente simple y documentado para el aprendizaje, y se describe explícitamente como inestable y detectable en entornos reales.
Su caso de uso principal es comprender los mecanismos de hook en el kernel y la inspección de telemetría anti-cheat.
