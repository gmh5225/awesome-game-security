Этот проект — TrinityCore, полноценный открытый MMORPG-фреймворк, реализующий полный эмулятор сервера World of Warcraft. Кодовая база на C++ включает модульные серверы аутентификации и мира, асинхронный пул подключений к БД с подготовленными запросами, обширные скриптовые API для пользовательского контента и поддержку нескольких дополнений WoW, включая системы боя, заданий, достижений и полей боя. Он в первую очередь полезен исследователям игровых серверов и специалистам по реверс-инжинирингу MMORPG, изучающим архитектуру крупных игровых серверов и эмуляцию протоколов.
