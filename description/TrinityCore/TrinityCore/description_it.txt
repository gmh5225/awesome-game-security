Questo progetto è TrinityCore, un framework MMORPG open source completo che implementa un emulatore di server World of Warcraft. La codebase C++ include server di autenticazione e di mondo modulari, pooling asincrono del database con prepared statements, ampie API di scripting per contenuti personalizzati e supporto per più espansioni di WoW, tra cui sistemi di combattimento, missioni, achievement e campi di battaglia. È soprattutto utile per ricercatori di game server e reverse engineer di MMORPG che studiano l’architettura di server di gioco su larga scala e l’emulazione dei protocolli.
