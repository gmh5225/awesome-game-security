Este proyecto es una reimplementación de código abierto de RollerCoaster Tycoon 2 centrada en sistemas modernos. Recrea la jugabilidad de simulación de parques y añade mejoras como multijugador cooperativo, una interfaz y un comportamiento de IA mejorados, herramientas de edición ampliadas y límites del motor más altos. El código está principalmente en C++, con rutas de compilación basadas en CMake y amplio soporte de plataformas, además de exponer puntos de integración para scripting y plugins. Está pensado tanto para jugadores como para desarrolladores que buscan un motor de simulación mantenible y extensible para la tecnología de juegos de parques de atracciones.
