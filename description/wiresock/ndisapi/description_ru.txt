Этот проект — комплексная пользовательская (user-mode) библиотека интерфейса, предназначенная для бесшовного взаимодействия с драйвером Windows Packet Filter.
Её отличает простой, безопасный и эффективный интерфейс для фильтрации (просмотра и изменения) сырых сетевых пакетов на уровне NDIS сетевого стека при минимальном влиянии на производительность сети.
Она в основном полезна исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / перехвата и фильтрации пакетов.
