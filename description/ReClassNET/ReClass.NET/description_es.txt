Este proyecto es una herramienta basada en .NET para la reconstrucción de clases en memoria y la inspección remota de procesos.
Admite objetivos x86 y x64, tipos de nodos de memoria ricos, escáneres, depuradores y compatibilidad de importación/exportación con formatos antiguos de ReClass.
El código combina la lógica de interfaz en C# con componentes nativos en C++ para operaciones de bajo nivel sobre procesos y memoria.
Se usa ampliamente en ingeniería inversa e investigación de seguridad en juegos para mapear estructuras en tiempo de ejecución y generar diseños de clases en C++ o C#.
