Este proyecto es una biblioteca avanzada para proteger y ofuscar controladores del kernel usando el estándar C++ 17.
Está escrita principalmente en C/C++ y se centra en el trabajo a nivel de kernel, el desarrollo de controladores y el modding.
Es especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / cifrado de variables.
