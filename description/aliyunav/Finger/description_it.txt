Finger è un sistema di riconoscimento dei simboli di funzione per l'analisi binaria che aiuta a identificare nomi sconosciuti di librerie o routine. Fornisce sia un SDK Python sia un plugin IDA che estraggono le caratteristiche delle funzioni, le inviano a un backend di riconoscimento e rinomina o evidenzia le funzioni corrispondenti in IDA. Il progetto mira a IDA 7+ e supporta ambienti Python 2.7 e Python 3. È pensato per i reverse engineer che vogliono accelerare il triage binario e il recupero delle funzioni in campioni legati a malware o a giochi.
