Finger — система распознавания символов функций для бинарного анализа, которая помогает определять неизвестные имена библиотек или процедур. Она предоставляет как Python SDK, так и плагин для IDA, которые извлекают признаки функций, отправляют их на сервер распознавания и переименовывают или подсвечивают совпавшие функции в IDA. Проект ориентирован на IDA 7+ и поддерживает среды Python 2.7 и Python 3. Он предназначен для специалистов по реверс-инжинирингу, которым нужно ускорить бинарный триаж и восстановление функций в образцах вредоносного ПО или игровых программ.
