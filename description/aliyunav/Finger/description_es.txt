Finger es un sistema de reconocimiento de símbolos de funciones para el análisis binario que ayuda a identificar nombres desconocidos de bibliotecas o rutinas. Ofrece tanto un SDK de Python como un complemento de IDA que extraen características de las funciones, las envían a un backend de reconocimiento y renombran o resaltan las funciones coincidentes en IDA. El proyecto está orientado a IDA 7+ y admite entornos Python 2.7 y Python 3. Está pensado para ingenieros de ingeniería inversa que desean acelerar el triaje binario y la recuperación de funciones en muestras relacionadas con malware o juegos.
