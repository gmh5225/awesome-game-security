~/.steam/steam/steamapps/compatdata/1245620/pfx/drive_c/) Setzen Sie in Steam die Startoptionen des Spiels auf protonhax init %command%. Starten Sie das Spiel über Steam (EasyAntiCheat zu deaktivieren ist optional). Führen Sie Cheat Engine mit protonhax run 1245620 /path/to/Cheat\ Engine.exe in Ihrem Terminal aus oder legen Sie es in ein Shell-Skript (ersetzen Sie /path/to/ durch den tatsächlichen Installationspfad von CE; Standard ist typischerweise ~/.wine/drive_c/Program\ Files/Cheat\ Engine\ 7.5/Cheat\ Engine.exe). Laden Sie die .CT-Datei über File->Load oder über das Ordnersymbol. Aktivieren Sie das Skript „Open“, indem Sie das Kästchen anhaken.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Cheat-Engine-Plugins untersuchen.
