~/.steam/steam/steamapps/compatdata/1245620/pfx/drive_c/) In Steam, imposta le opzioni di avvio del gioco su protonhax init %command%. Avvia il gioco tramite Steam (disabilitare EasyAntiCheat è facoltativo). Esegui Cheat Engine con protonhax run 1245620 /path/to/Cheat\ Engine.exe nel terminale preferito, oppure mettilo in uno script shell (sostituisci /path/to/ con il percorso reale di installazione di CE; quello predefinito di solito è ~/.wine/drive_c/Program\ Files/Cheat\ Engine\ 7.5/Cheat\ Engine.exe). Carica il file .CT tramite File->Load o cliccando sull’icona della cartella. Attiva lo script «Open» selezionando la relativa casella.
È principalmente utile a ricercatori di sicurezza dei giochi e reverse engineer che studiano tecniche offensive nell’ambito di cheat / plugin di Cheat Engine.
