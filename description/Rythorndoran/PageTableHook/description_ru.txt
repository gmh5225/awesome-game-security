Этот проект — демонстрация хукинга ядра Windows, которая изменяет отображения таблиц страниц вместо прямой патч-модификации исполняемого кода. Драйвер на C++ строит и редактирует структуры пейджинга, перенаправляя выполнение к обработчикам хуков и сохраняя доступ к исходным процедурам. Пример включает перехват системных путей вроде NtCreateFile и строится вокруг избежания типичных подходов, вызывающих PatchGuard. Предназначен для продвинутых исследований безопасности ядра и изучения техник обхода античита.
