Este proyecto es una demostración de hooking en el kernel de Windows que modifica los mapeos de las tablas de páginas en lugar de parchear directamente el código ejecutable. Su controlador en C++ construye y edita estructuras de paginación para redirigir la ejecución a manejadores de hook, preservando el acceso a las rutinas originales. El ejemplo incluye la interceptación en rutas del sistema como NtCreateFile y se plantea en torno a evitar enfoques típicos que activan PatchGuard. Está pensado para investigación avanzada de seguridad del kernel y el estudio de técnicas de bypass de anti-cheat.
