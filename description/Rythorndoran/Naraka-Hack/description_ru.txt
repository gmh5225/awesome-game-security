Этот проект — внешняя cheat-подобная реализация для игры на Unity с доступом к памяти и отрисовкой через оверлей. Код на C++ поддерживает таблицы смещений (offsets), читает данные состояния сущностей и игрока и рисует ESP-элементы вроде рамок и имён через окно NVIDIA Overlay. Также есть управление меню горячими клавишами и логика боевой помощи с уклоном в автоматизацию. Репозиторий в основном используют для практики разработки читов и реверс-инжиниринга структур рантайма игры.
