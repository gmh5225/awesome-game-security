Este proyecto es una implementación de tipo cheat externo para un juego basado en Unity, mediante acceso a memoria y renderizado por overlay. El código C++ mantiene tablas de offsets, lee datos de estado de entidades y del jugador, y dibuja visualizaciones estilo ESP como cajas y nombres a través de una ventana de overlay de NVIDIA. También incluye controles de menú activados por teclas rápidas y lógica de asistencia de combate orientada a la automatización. El repositorio se usa principalmente para practicar el desarrollo de cheats y la ingeniería inversa de estructuras en tiempo de ejecución del juego.
