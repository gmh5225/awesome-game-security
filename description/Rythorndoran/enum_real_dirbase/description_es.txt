Este proyecto es una prueba de concepto de un controlador en modo kernel de Windows para enumerar los valores reales de base de directorio de proceso (CR3) a partir de estructuras de memoria física.
Está escrito en C++ para el Windows Driver Kit e implementa helpers de paginación de bajo nivel, escaneo de patrones en el kernel y recorrido de la base de datos PFN.
El código inicializa bases de tablas de páginas autorreferenciadas, resuelve MmPfnDatabase en tiempo de ejecución y recorre rangos físicos para recuperar datos de contexto de proceso.
Resulta especialmente útil para investigación de anti-trampas en el kernel, experimentos de forense de memoria y el estudio de cómo se rastrean los espacios de direcciones ocultos o protegidos.
