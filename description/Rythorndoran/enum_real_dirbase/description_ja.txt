本プロジェクトは、物理メモリ構造から実際のプロセスディレクトリベース値（CR3）を列挙するための、Windows カーネルモードドライバの概念実証です。
Windows Driver Kit 向けに C++ で記述されており、低レベルのページングヘルパー、カーネルパターンスキャン、および PFN データベースの走査を実装しています。
コードは自己参照ページテーブルベースを初期化し、実行時に MmPfnDatabase を解決し、物理範囲を走査してプロセスコンテキストデータを復元します。
主にカーネルアンチチート研究、メモリフォレンジック実験、および隠蔽または保護されたアドレス空間の追跡方法の学習に役立ちます。
