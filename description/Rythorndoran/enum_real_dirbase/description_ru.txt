Этот проект — proof of concept драйвера режима ядра Windows для перечисления реальных значений directory base процессов (CR3) из структур физической памяти.
Он написан на C++ для Windows Driver Kit и реализует низкоуровневые вспомогательные функции пейджинга, сканирование шаблонов в ядре и обход базы данных PFN.
Код инициализирует самоссылающиеся базы таблиц страниц, разрешает MmPfnDatabase во время выполнения и обходит физические диапазоны для восстановления данных контекста процесса.
Он в основном полезен для исследований античита на уровне ядра, экспериментов по форензике памяти и изучения того, как отслеживаются скрытые или защищённые адресные пространства.
