Dieses Projekt ist ein Proof of Concept eines Windows-Kernelmodus-Treibers zur Enumeration echter Prozess-Directory-Base-Werte (CR3) aus physischen Speicherstrukturen.
Es ist in C++ für das Windows Driver Kit geschrieben und implementiert Low-Level-Paging-Hilfen, Kernel-Pattern-Scanning sowie die Traversierung der PFN-Datenbank.
Der Code initialisiert selbstreferenzierende Seitentabellenbasen, löst MmPfnDatabase zur Laufzeit auf und durchläuft physische Bereiche, um Prozesskontextdaten wiederherzustellen.
Es eignet sich vor allem für Kernel-Anti-Cheat-Forschung, Speicherforensik-Experimente und das Studium der Verfolgung versteckter oder geschützter Adressräume.
