Questo progetto è una proof of concept di un driver in modalità kernel Windows per enumerare i valori reali di directory base dei processi (CR3) dalle strutture della memoria fisica.
È scritto in C++ per il Windows Driver Kit e implementa helper di paging a basso livello, scansione di pattern nel kernel e traversamento del database PFN.
Il codice inizializza basi delle page table auto-referenzianti, risolve MmPfnDatabase a runtime e attraversa intervalli fisici per recuperare i dati di contesto dei processi.
È soprattutto utile per la ricerca anti-cheat a livello kernel, esperimenti di forensics della memoria e lo studio di come vengono tracciati gli spazi di indirizzi nascosti o protetti.
