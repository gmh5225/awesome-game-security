이 프로젝트는 물리 메모리 구조에서 실제 프로세스 디렉터리 베이스 값(CR3)을 열거하기 위한 Windows 커널 모드 드라이버 개념 증명입니다.
Windows Driver Kit용 C++로 작성되었으며, 저수준 페이징 헬퍼, 커널 패턴 스캐닝, PFN 데이터베이스 순회를 구현합니다.
코드는 자기 참조 페이지 테이블 베이스를 초기화하고, 런타임에 MmPfnDatabase를 해석하며, 물리 범위를 탐색하여 프로세스 컨텍스트 데이터를 복구합니다.
주로 커널 안티치트 연구, 메모리 포렌식 실험, 그리고 숨겨지거나 보호된 주소 공간이 어떻게 추적되는지 연구하는 데 유용합니다.
