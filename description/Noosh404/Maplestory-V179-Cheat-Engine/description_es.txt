Este proyecto es una colección de tablas de Cheat Engine y scripts de autoensamblador para MapleStory v179.
Los scripts implementan modificaciones de jugabilidad como ataque de mapa completo, inyección de habilidades, variantes sin retardo, comportamiento de teletransporte de loot de mascotas y ajustes relacionados con el knockback.
El contenido se ofrece como scripts CEA y CT que parchean instrucciones en memoria mediante escaneos de patrones y ediciones de bytes.
Está pensado para practicar el desarrollo de cheats y experimentos de ingeniería inversa en versiones legacy de MapleStory.
