Этот проект — набор таблиц Cheat Engine и скриптов автоассемблера для MapleStory v179.
Скрипты реализуют игровые модификации: атака по всей карте, инъекция навыков, варианты без задержки, телепорт лута питомцем и правки, связанные с отбрасыванием.
Содержимое представлено в виде CEA- и CT-скриптов, которые патчат инструкции в памяти через сканирование сигнатур и правки байтов.
Материал предназначен для практики разработки читов и экспериментов по реверс-инжинирингу устаревших версий MapleStory.
