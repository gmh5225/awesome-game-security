Dieses Projekt konzentriert sich auf ein Lua-Skripting-Plugin für IDA Pro.
LUDA ermöglicht direkten Zugriff auf das IDA SDK aus Lua und unterstützt so schnelles Prototyping sowie die Automatisierung von Reverse-Engineering-Workflows.
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
