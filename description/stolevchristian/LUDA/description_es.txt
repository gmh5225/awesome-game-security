Este proyecto se centra en un plugin de scripting Lua para IDA Pro.
LUDA proporciona acceso directo al SDK de IDA desde Lua, lo que permite prototipar con rapidez y automatizar flujos de trabajo de ingeniería inversa.
Resulta especialmente útil para investigadores de seguridad en juegos y para ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
