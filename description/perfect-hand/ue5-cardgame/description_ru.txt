Этот проект — реализация карточной игры на Unreal Engine 5. Он демонстрирует паттерны разработки игр на UE5, включая управление колодой карт, логику пошагового геймплея, проектирование UI-виджетов с UMG, сетевую репликацию для мультиплеера и визуальные эффекты анимации карт. Проект на Blueprint и C++ служит учебным справочником по разработке игр на UE5. Он предназначен для разработчиков Unreal Engine, изучающих паттерны реализации карточных и настольных игр в UE5.
