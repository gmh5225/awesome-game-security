Ce projet est une implémentation de jeu de cartes réalisée avec Unreal Engine 5. Il illustre des modèles de développement de jeux UE5, notamment la gestion du deck de cartes, la logique de jeu au tour par tour, la conception de widgets UI avec UMG, la réplication réseau pour le multijoueur et les effets visuels pour les animations de cartes. Le projet Blueprint et C++ sert de référence d'apprentissage pour le développement de jeux UE5. Il s'adresse aux développeurs Unreal Engine qui étudient les modèles d'implémentation de jeux de cartes et de plateau dans UE5.
