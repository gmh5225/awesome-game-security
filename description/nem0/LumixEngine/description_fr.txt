Ce projet est une petite collection d'exemples construits avec le Lumix Engine.
Il est principalement écrit en C/C++ et C++ et se concentre sur le développement de pilotes, le travail sur les shaders et le rendu.
Il est surtout utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
