Questo progetto è una piccola raccolta di progetti di esempio realizzati con Lumix Engine.
È scritto principalmente in C/C++ e C++ e si concentra sullo sviluppo di driver, sul lavoro con gli shader e sul rendering.
È soprattutto utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che operano nell'ambito dei game engine / del codice sorgente.
