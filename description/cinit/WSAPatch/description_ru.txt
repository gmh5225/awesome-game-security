Этот проект — патч для Windows Subsystem for Android (WSA), позволяющий запускать его на неподдерживаемых сборках Windows 10 и более старых версиях Windows 11. Он изменяет бинарные файлы WSA, чтобы обойти проверки версии, отключить принудительное требование Hyper-V и применить исправления совместимости. C++-патчер работает с MSIX-пакетами WSA, чтобы включить сайдлоадинг Android-приложений на более широком диапазоне конфигураций Windows. Предназначен для пользователей и разработчиков, которые хотят запускать Android-приложения на неподдерживаемых версиях Windows.
