이 프로젝트는 Windows Subsystem for Android(WSA)용 패치로, 지원되지 않는 Windows 10 빌드와 이전 Windows 11 버전에서도 실행할 수 있게 합니다. WSA 바이너리를 수정하여 버전 검사를 우회하고, Hyper-V 필수 요구 강제 적용을 비활성화하며, 호환성 수정을 적용합니다. C++ 패처는 WSA MSIX 패키지를 처리하여 더 넓은 범위의 Windows 구성에서 Android 앱 사이드로딩을 가능하게 합니다. 지원되지 않는 Windows 버전에서 Android 앱을 실행하려는 사용자와 개발자를 대상으로 합니다.
