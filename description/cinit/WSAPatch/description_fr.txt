Ce projet est un correctif pour Windows Subsystem for Android (WSA) qui permet de l’exécuter sur des builds Windows 10 non prises en charge et sur d’anciennes versions de Windows 11. Il modifie les binaires WSA pour contourner les contrôles de version, désactiver l’obligation Hyper-V et appliquer des correctifs de compatibilité. Le patcher C++ agit sur les packages MSIX de WSA afin d’activer le sideloading d’applications Android sur une plus large gamme de configurations Windows. Il s’adresse aux utilisateurs et développeurs souhaitant faire tourner des applications Android sur des versions de Windows non prises en charge.
