Este proyecto es un parche para Windows Subsystem for Android (WSA) que permite ejecutarlo en builds de Windows 10 no admitidas y en versiones anteriores de Windows 11. Modifica los binarios de WSA para omitir las comprobaciones de versión, desactivar la exigencia de Hyper-V y aplicar correcciones de compatibilidad. El parcheador en C++ opera sobre paquetes MSIX de WSA para habilitar la carga lateral de aplicaciones Android en una gama más amplia de configuraciones de Windows. Está dirigido a usuarios y desarrolladores que desean ejecutar aplicaciones Android en versiones de Windows no admitidas.
