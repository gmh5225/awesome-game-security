이 프로젝트는 IL2CPP Assembly Unhollower로, IL2CPP 메타데이터에서 프록시 .NET 어셈블리를 생성하여 C# 모드가 IL2CPP로 컴파일된 Unity 게임을 표준 Mono 어셈블리처럼 다룰 수 있게 하는 도구입니다. IL2CPP global-metadata.dat와 바이너리 덤프를 처리해 타입 정의, 메서드 시그니처, 필드 레이아웃을 재구성하고, 네이티브 IL2CPP 메서드로 호출을 위임하는 래퍼 어셈블리를 생성합니다. 직접적인 .NET 리플렉션을 사용할 수 없는 IL2CPP 빌드를 다루는 Unity 모더와 게임 보안 연구자를 대상으로 합니다.
