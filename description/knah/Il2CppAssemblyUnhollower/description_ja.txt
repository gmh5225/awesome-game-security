本プロジェクトは IL2CPP Assembly Unhollower です。IL2CPP メタデータからプロキシ .NET アセンブリを生成し、C# MOD が IL2CPP でコンパイルされた Unity ゲームを、通常の Mono アセンブリと同様に扱えるようにするツールです。IL2CPP の global-metadata.dat とバイナリダンプを処理して型定義、メソッドシグネチャ、フィールドレイアウトを再構築し、ネイティブ IL2CPP メソッドへの呼び出しを委譲するラッパーアセンブリを生成します。直接の .NET リフレクションが使えない IL2CPP ビルドを扱う Unity モッダーやゲームセキュリティ研究者向けです。
