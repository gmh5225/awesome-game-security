Questo progetto è IL2CPP Assembly Unhollower, uno strumento che genera assembly .NET proxy dai metadati IL2CPP, consentendo ai mod C# di interagire con i giochi Unity compilati con IL2CPP come se fossero assembly Mono standard. Elabora il file global-metadata.dat di IL2CPP e i dump binari per ricostruire definizioni di tipi, firme dei metodi e layout dei campi, generando assembly wrapper che delegano le chiamate ai metodi IL2CPP nativi. È pensato per modder Unity e ricercatori di sicurezza dei giochi che lavorano con build IL2CPP dove la reflection .NET diretta non è disponibile.
