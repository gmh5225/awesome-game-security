Este proyecto es un motor de aplicaciones interactivas y de renderizado en etapa temprana, centrado en la arquitectura de motores de juego en C++.
Incluye componentes de motor, editor y sandbox, junto con scripts de configuración que preparan dependencias como las herramientas del Vulkan SDK.
El código es principalmente en C++ y se desarrolla tanto como motor funcional como referencia educativa para el diseño de motores.
Su caso de uso principal es el aprendizaje y el prototipado de sistemas de renderizado en tiempo real y frameworks de juego.
