Questo progetto è un plugin IDA che ricostruisce le informazioni originali dello schema Protocol Buffer a partire da binari compilati. È scritto in Python e usa il runtime Google protobuf insieme alle API di IDA per decodificare i descrittori incorporati e mostrare le definizioni proto recuperate. Il plugin si integra nel flusso di lavoro del disassemblatore tramite un'azione di ricerca dedicata e viste dei risultati personalizzate per facilitare l'ispezione. È utile per i reverse engineer che devono recuperare formati di rete o di serializzazione nel lavoro di sicurezza dei giochi e di analisi binaria.
