Ce projet est un plugin IDA qui reconstruit les informations de schéma Protocol Buffer d'origine à partir de binaires compilés. Écrit en Python, il utilise le runtime Google protobuf ainsi que les API IDA pour décoder les descripteurs intégrés et afficher les définitions proto récupérées. Le plugin s'intègre au flux de travail du désassembleur via une action de recherche dédiée et des vues de résultats personnalisées pour faciliter l'inspection. Il est utile aux ingénieurs en rétro-ingénierie qui doivent retrouver des formats réseau ou de sérialisation dans le cadre de la sécurité des jeux et de l'analyse binaire.
