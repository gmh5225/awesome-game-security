Onlooker — это лёгкий профилировщик памяти для Windows, который собирает статистику памяти для дерева процессов, подобно команде time в Linux. В комплект входит сопутствующий GUI-инструмент на Qt для более наглядного просмотра трасс и журналов. Проект в основном реализован на C++ с использованием CMake и Qt Widgets, а также предоставляет утилиты преобразования JSON-трасс. Основное применение — диагностика роста памяти, событий нехватки памяти (OOM) и регрессий производительности в сложных нативных инструментальных цепочках.
