Este proyecto es una herramienta de investigación para simplificar expresiones Mixed Boolean-Arithmetic (MBA), publicada por Denuvo Software Solutions. Incluye un simplificador basado en Python con soporte para expresiones MBA lineales y no lineales, una fábrica de expresiones a nivel de bits y conjuntos de datos experimentales de otras herramientas como NeuReduce, QSynth y Syntia. El repositorio también contiene el artículo de investigación asociado y las diapositivas de la presentación. Resulta especialmente útil para investigadores de ofuscación y desofuscación que estudian la protección de código basada en MBA y las técnicas de simplificación automática de expresiones.
