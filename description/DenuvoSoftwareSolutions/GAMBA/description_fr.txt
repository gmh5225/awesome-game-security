Ce projet est un outil de recherche pour simplifier les expressions Mixed Boolean-Arithmetic (MBA), publié par Denuvo Software Solutions. Il comprend un simplificateur en Python prenant en charge les expressions MBA linéaires et non linéaires, une fabrique d’expressions bit à bit, ainsi que des jeux de données expérimentaux issus d’autres outils tels que NeuReduce, QSynth et Syntia. Le dépôt contient également l’article de recherche associé et les diapositives de présentation. Il s’adresse surtout aux chercheurs en obfuscation et désobfuscation qui étudient la protection de code basée sur les MBA et les techniques de simplification automatique d’expressions.
