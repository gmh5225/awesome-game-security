Ce projet se concentre sur l’approche external. Pour les offsets, téléchargez-les via le script en exécutant offset_download.ps1 dans le même dossier que votre exécutable, ou manuellement en téléchargeant client.dll.json, offsets.json et buttons.json, puis placez ces fichiers dans le même dossier que tim_apple.exe. Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / jeu:CS2.
