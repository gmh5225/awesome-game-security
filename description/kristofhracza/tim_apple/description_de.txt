Dieses Projekt konzentriert sich auf den External-Ansatz. Offsets können per Skript geladen werden: Führen Sie offset_download.ps1 im selben Ordner wie Ihre ausführbare Datei aus. Beim manuellen Download laden Sie client.dll.json, offsets.json und buttons.json herunter und legen diese Dateien in denselben Ordner wie die ausführbare Datei tim_apple.exe. Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Spiel:CS2 untersuchen.
