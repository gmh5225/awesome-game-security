Этот проект — плагин UE4, который копирует переданные строковые аргументы в системный буфер обмена.
Он реализован как модуль Unreal Engine на C++ с упакованными бинарниками и метаданными плагина для окружений UE4.27.
Инструмент лёгкий и сфокусирован на быстрой выдаче текста в буфер обмена из рабочих процессов внутри движка.
Он полезен для инструментов разработки игр и отладочных конвейеров, которым нужна быстрая передача текста при тестировании или скриптовых задачах.
