Este proyecto es un complemento de UE4 que copia los argumentos de cadena suministrados al portapapeles del sistema.
Está implementado como un módulo C++ de Unreal Engine con binarios empaquetados y metadatos del complemento para entornos UE4.27.
La herramienta es ligera y se centra en exponer una salida rápida al portapapeles desde flujos de trabajo dentro del motor.
Es útil para herramientas de desarrollo de juegos y pipelines de depuración que necesitan una transferencia rápida de texto durante pruebas o tareas de scripting.
