Ce projet est un plugin UE4 qui copie les arguments de chaîne fournis vers le presse-papiers système.
Il est implémenté comme un module C++ Unreal Engine, avec des binaires empaquetés et des métadonnées de plugin pour les environnements UE4.27.
L’outil est léger et se concentre sur une sortie rapide vers le presse-papiers depuis les flux de travail dans le moteur.
Il est utile pour les outils de développement de jeux et les pipelines de débogage qui nécessitent un transfert rapide de texte pendant les tests ou les tâches de scripting.
