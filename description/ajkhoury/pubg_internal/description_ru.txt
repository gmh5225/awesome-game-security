pubg_internal — архивный исследовательский проект внутреннего чита для PUBG, сочетающий компоненты пользовательского режима и режима ядра. Он включает драйвер ядра для отображения защищённых процессов, генератор игрового SDK и базовую реализацию внутреннего ESP. Код демонстрирует практическую интеграцию доступа к памяти, извлечения игровых структур и логики функций внутри процесса на C и C++. Проект в первую очередь полезен для изучения исторической архитектуры игровых читов и моделей угроз античита.
