ReClassEx ist ein erweitertes Windows-Tool zur Rekonstruktion und Untersuchung von Speicherklassen, das in der Reverse Engineering eingesetzt wird. Es ist in C++ implementiert und bietet eine interaktive Oberfläche zum Modellieren von Speicherstrukturen, während es an einen Prozess angehängt ist. Zu den zentralen Funktionen gehören RTTI-Unterstützung, PDB-Symbol laden, modulbasiertes Adressieren, Plugin-Erweiterbarkeit und eine verbesserte Navigation für große Klassenlayouts. Das Tool eignet sich weitreichend für Game-Hacking- und Malware-Analyse-Workflows, in denen das Verständnis dynamischer Speicherlayouts entscheidend ist.
