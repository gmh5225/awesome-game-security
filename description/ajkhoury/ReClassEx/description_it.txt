ReClassEx è uno strumento esteso di ricostruzione e ispezione di classi in memoria Windows, usato nel reverse engineering. È implementato in C++ e fornisce un’interfaccia interattiva per modellare le strutture in memoria mentre è collegato a un processo. Le capacità principali includono supporto RTTI, caricamento di simboli PDB, targeting degli indirizzi basato sui moduli, estensibilità tramite plugin e una navigazione migliorata per layout di classi di grandi dimensioni. Lo strumento è ampiamente applicabile ai flussi di lavoro di game hacking e analisi malware, dove comprendere i layout di memoria dinamici è fondamentale.
