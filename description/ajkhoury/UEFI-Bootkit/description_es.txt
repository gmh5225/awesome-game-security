UEFI-Bootkit es un proyecto compacto de bootkit de prueba de concepto que apunta a la cadena de arranque UEFI, escrito principalmente en C con una dependencia mínima de ensamblador. El código incluye una aplicación UEFI y componentes de controlador en tiempo de ejecución pensados para persistir más allá de ExitBootServices y continuar la ejecución durante el arranque del sistema operativo. Demuestra una configuración práctica de compilación EFI, lógica de carga de imágenes y manejo de protocolos en tiempo de ejecución en un contexto de firmware. El proyecto se usa principalmente para investigación de seguridad de arranque de bajo nivel, incluidos estudios sobre persistencia pre-OS y desafíos de detección.
