Este proyecto se centra en una biblioteca y aplicación de acceso remoto para Dear ImGui. Permite que cualquier programa que use Dear ImGui reciba entrada desde un PC remoto y luego reenvíe su renderizado de UI (texturas, vértices, índices, comandos de dibujo). Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores de gráficos que trabajan en el área de motores de juego / plugins de motores de juego: Unreal.
