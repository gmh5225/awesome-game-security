Questo progetto si concentra su una libreria e un'applicazione di accesso remoto per Dear ImGui. Consente a qualsiasi programma che usa Dear ImGui di ricevere input da un PC remoto e di inoltrare il rendering della UI (texture, vertici, indici, comandi di disegno). È soprattutto utile per sviluppatori di giochi, programmatori di engine e ricercatori grafici che lavorano nell'area dei game engine / plugin per game engine: Unreal.
