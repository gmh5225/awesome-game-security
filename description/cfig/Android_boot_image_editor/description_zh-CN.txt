本项目是一款用于逆向工程 Android ROM 镜像的工具，支持 boot.img、recovery、vendor_boot 及其他分区镜像。
它提供基于 Gradle 的解包与重打包工作流，可提取内核、ramdisk、DTB 和 VBMeta 镜像，并完整支持 AVB 签名、LZ4/XZ/GZIP 压缩以及 EROFS/稀疏镜像处理。
该 Kotlin/Java 代码库可在 Linux、macOS 和 Windows 上运行（需 JDK 11+），支持 Android boot 镜像格式第 0 至第 4 版以及 vendor boot 镜像。
主要面向进行 boot 镜像分析、修改与重打包的 Android 安全研究人员和 ROM 开发者。
