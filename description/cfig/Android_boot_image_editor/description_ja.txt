本プロジェクトは、boot.img、recovery、vendor_boot、その他のパーティションイメージを含む Android ROM イメージをリバースエンジニアリングするためのツールです。
Gradle ベースのアンパック／リパックワークフローを提供し、カーネル、ramdisk、DTB、VBMeta イメージを抽出し、AVB 署名、LZ4/XZ/GZIP 圧縮、EROFS／スパースイメージ処理を完全にサポートします。
Kotlin/Java のコードベースは JDK 11+ で Linux、macOS、Windows 上で動作し、Android boot イメージ形式のバージョン 0 から 4、および vendor boot イメージに対応しています。
主に boot イメージの解析、改変、再パックを行う Android セキュリティ研究者や ROM 開発者向けです。
