Este proyecto es una herramienta para la ingeniería inversa de imágenes ROM de Android, incluyendo boot.img, recovery, vendor_boot y otras imágenes de partición.
Proporciona flujos de trabajo Gradle de desempaquetado y reempaquetado que extraen kernels, ramdisks, DTB e imágenes VBMeta, con soporte completo para firma AVB, compresión LZ4/XZ/GZIP y manejo de imágenes EROFS/sparse.
El código Kotlin/Java se ejecuta en Linux, macOS y Windows con JDK 11+ y admite formatos de imagen boot de Android desde la versión 0 hasta la 4, además de imágenes vendor boot.
Es especialmente útil para investigadores de seguridad de Android y desarrolladores de ROM que analizan, modifican y reempaquetan imágenes boot.
