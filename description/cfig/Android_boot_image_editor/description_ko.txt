이 프로젝트는 boot.img, recovery, vendor_boot 및 기타 파티션 이미지를 포함한 Android ROM 이미지를 리버스 엔지니어링하기 위한 도구입니다.
Gradle 기반의 언팩 및 리팩 워크플로를 제공하며, 커널, ramdisk, DTB, VBMeta 이미지를 추출하고 AVB 서명, LZ4/XZ/GZIP 압축, EROFS/스파스 이미지 처리를 완전 지원합니다.
Kotlin/Java 코드베이스는 JDK 11+로 Linux, macOS, Windows에서 실행되며, Android boot 이미지 형식 버전 0부터 4까지와 vendor boot 이미지를 지원합니다.
주로 boot 이미지 분석, 수정 및 리팩을 수행하는 Android 보안 연구원과 ROM 개발자에게 유용합니다.
