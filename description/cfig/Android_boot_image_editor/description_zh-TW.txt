本專案是一款用於逆向工程 Android ROM 映像的工具，支援 boot.img、recovery、vendor_boot 及其他分割區映像。
它提供基於 Gradle 的解包與重打包工作流程，可擷取核心、ramdisk、DTB 與 VBMeta 映像，並完整支援 AVB 簽章、LZ4/XZ/GZIP 壓縮以及 EROFS/稀疏映像處理。
此 Kotlin/Java 程式碼庫可在 Linux、macOS 與 Windows 上執行（需 JDK 11+），支援 Android boot 映像格式第 0 至第 4 版以及 vendor boot 映像。
主要適用於進行 boot 映像分析、修改與重打包的 Android 安全研究人員與 ROM 開發者。
