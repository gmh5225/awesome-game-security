Этот проект — инструмент для реверс-инжиниринга образов Android ROM, включая boot.img, recovery, vendor_boot и другие образы разделов.
Он предоставляет Gradle-процессы распаковки и переупаковки, извлекающие ядра, ramdisk, DTB и образы VBMeta, с полной поддержкой подписи AVB, сжатия LZ4/XZ/GZIP и работы с образами EROFS/sparse.
Кодовая база на Kotlin/Java работает в Linux, macOS и Windows с JDK 11+ и поддерживает форматы образов Android boot версий с 0 по 4, а также образы vendor boot.
В первую очередь полезен исследователям безопасности Android и разработчикам ROM, выполняющим анализ, модификацию и переупаковку boot-образов.
