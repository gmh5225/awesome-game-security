cs2-sdk è un SDK di header C++ generato automaticamente per i binari Source 2 di Counter-Strike 2. Contiene ampi set di dichiarazioni di classi ed enumerazioni per moduli come client, server, rendering, networking e sistemi schema, prodotti con lo strumento source2gen. Il repository include anche file di metadati di patch che collegano gli header generati a una specifica versione del gioco. Il caso d'uso principale è il reverse engineering e la ricerca di sicurezza nei giochi che richiedono informazioni sui tipi aggiornate.
