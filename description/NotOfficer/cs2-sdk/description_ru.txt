cs2-sdk — это автоматически сгенерированный SDK заголовочных файлов C++ для бинарников Counter-Strike 2 на Source 2. Он содержит большие наборы объявлений классов и перечислений для модулей client, server, рендеринга, сетевого взаимодействия и schema-систем, созданных с помощью инструмента source2gen. Репозиторий также включает файлы метаданных патчей, связывающие сгенерированные заголовки с конкретной версией игры. Основное применение — реверс-инжиниринг и исследования игровой безопасности, которым нужна актуальная информация о типах.
