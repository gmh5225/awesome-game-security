cs2-sdk ist ein automatisch generiertes C++-Header-SDK für Counter-Strike-2-Source-2-Binärdateien. Es enthält umfangreiche Klassen- und Enum-Deklarationen für Module wie Client, Server, Rendering, Networking und Schema-Systeme, erzeugt mit dem source2gen-Tooling. Das Repository enthält außerdem Patch-Metadatendateien, die die generierten Header an eine bestimmte Spielversion binden. Der Hauptanwendungsfall ist Reverse Engineering und Spielsicherheitsforschung, die aktuelle Typinformationen benötigt.
