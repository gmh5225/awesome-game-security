cs2-sdk es un SDK de encabezados C++ generado automáticamente para binarios Source 2 de Counter-Strike 2. Contiene amplios conjuntos de declaraciones de clases y enumeraciones para módulos como client, server, renderizado, redes y sistemas de schema, producidos con la herramienta source2gen. El repositorio también incluye archivos de metadatos de parches que vinculan los encabezados generados a una versión concreta del juego. Su caso de uso principal es la ingeniería inversa y la investigación de seguridad en juegos que necesitan información de tipos actualizada.
