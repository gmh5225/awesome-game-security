Este proyecto es un framework MCP sin interfaz, extensible y de múltiples sesiones para IDA Pro, construido con ida-domain y FastMCP.
Admite el análisis simultáneo de varios binarios e incluye plugins integrados para funciones, referencias cruzadas, nombrado de símbolos, comentarios, cadenas, segmentos, parcheo de bytes, tipos y puntos de entrada.
El paquete de Python es extensible mediante plugins personalizados a través de entry points y ofrece documentación generada automáticamente.
Resulta especialmente útil para ingenieros inversos que buscan un análisis headless de IDA Pro integrado con IA mediante el Model Context Protocol.
