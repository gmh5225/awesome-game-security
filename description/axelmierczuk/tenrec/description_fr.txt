Ce projet est un framework MCP sans interface, extensible et multi-sessions pour IDA Pro, construit avec ida-domain et FastMCP.
Il permet l’analyse simultanée de plusieurs binaires et inclut des plugins intégrés pour les fonctions, les références croisées, le nommage de symboles, les commentaires, les chaînes, les segments, le patching d’octets, les types et les points d’entrée.
Le package Python est extensible via des plugins personnalisés par entry points et fournit une documentation générée automatiquement.
Il est surtout utile aux reverse engineers qui cherchent une analyse IDA Pro sans interface, intégrée à l’IA, via le Model Context Protocol.
