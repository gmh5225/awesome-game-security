Questo progetto è un framework MCP headless, estendibile e multi-sessione per IDA Pro, realizzato con ida-domain e FastMCP.
Supporta l’analisi simultanea di più binari e include plugin integrati per funzioni, cross-reference, naming dei simboli, commenti, stringhe, segmenti, byte patching, tipi e entry point.
Il pacchetto Python è estendibile tramite plugin personalizzati con entry points e fornisce documentazione generata automaticamente.
È soprattutto utile per i reverse engineer che cercano un’analisi IDA Pro headless integrata con l’IA tramite il Model Context Protocol.
