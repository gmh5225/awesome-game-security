Этот проект — безголовый (headless), расширяемый многосессионный MCP-фреймворк для IDA Pro, построенный на ida-domain и FastMCP.
Он поддерживает одновременный анализ нескольких бинарников и включает встроенные плагины для функций, перекрёстных ссылок, именования символов, комментариев, строк, сегментов, патчинга байтов, типов и точек входа.
Пакет Python расширяется пользовательскими плагинами через entry points и предоставляет автоматически сгенерированную документацию.
Он в первую очередь полезен reverse-инженерам, которым нужен AI-интегрированный headless-анализ в IDA Pro через Model Context Protocol.
