本プロジェクトは、ida-domain と FastMCP で構築された、IDA Pro 向けのヘッドレスで拡張可能なマルチセッション MCP フレームワークです。
複数のバイナリを同時に解析でき、関数、クロスリファレンス、シンボル命名、コメント、文字列、セグメント、バイトパッチ、型、エントリポイント向けの組み込みプラグインを備えています。
この Python パッケージはエントリポイント経由のカスタムプラグインで拡張でき、自動生成ドキュメントも提供します。
主に、Model Context Protocol を通じて AI 連携のヘッドレス IDA Pro 解析を求めるリバースエンジニア向けです。
