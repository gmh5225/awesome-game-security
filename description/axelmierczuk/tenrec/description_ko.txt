이 프로젝트는 ida-domain과 FastMCP로 구축된 IDA Pro용 헤드리스·확장 가능·다중 세션 MCP 프레임워크입니다.
여러 바이너리를 동시에 분석할 수 있으며, 함수, 크로스 레퍼런스, 심볼 명명, 주석, 문자열, 세그먼트, 바이트 패치, 타입, 엔트리 포인트를 위한 내장 플러그인을 포함합니다.
이 Python 패키지는 엔트리 포인트를 통한 사용자 정의 플러그인으로 확장할 수 있으며, 자동 생성된 문서도 제공합니다.
주로 Model Context Protocol을 통해 AI와 통합된 헤드리스 IDA Pro 분석을 원하는 리버스 엔지니어에게 유용합니다.
