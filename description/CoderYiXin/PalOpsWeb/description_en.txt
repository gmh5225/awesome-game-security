PalOps Web is a modern all-in-one operations console for Palworld Dedicated Server, centered on deep integration with the PalDefender anti-cheat plugin. It combines PalDefender REST, extended RCON, version checks, whitelist and ban data, and managed JSON configuration with native Palworld REST, server lifecycle control, read-only save indexing, player and guild intelligence, offline MapLibre world maps, resource grants, backups, automation jobs, webhook notifications, and audit-ready access control. The stack is ASP.NET Core on .NET 10 with a Vue 3 frontend, and it is designed to run on the same Windows host as the game server rather than as a generic cloud panel. PalDefender remains optional for core admin features, while its protection and extended runtime capabilities target server operators who need anti-cheat-aware Palworld administration.
