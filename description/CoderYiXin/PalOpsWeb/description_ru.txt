PalOps Web — современная универсальная операционная консоль для Palworld Dedicated Server, построенная вокруг глубокой интеграции с античит-плагином PalDefender. Она объединяет PalDefender REST, расширенный RCON, проверки версий, данные белого списка и банов, а также управляемую JSON-конфигурацию с нативным Palworld REST, управлением жизненным циклом сервера, индексированием сохранений только для чтения, аналитикой игроков и гильдий, офлайн-картами мира MapLibre, выдачей ресурсов, резервным копированием, задачами автоматизации, webhook-уведомлениями и контролем доступа, готовым к аудиту. Стек — ASP.NET Core на .NET 10 с фронтендом Vue 3; решение рассчитано на запуск на том же Windows-хосте, что и игровой сервер, а не как универсальная облачная панель. Для базовых функций администрирования PalDefender остаётся необязательным, а его защита и расширенные возможности среды выполнения ориентированы на операторов, которым нужно античит-осознанное управление Palworld.
