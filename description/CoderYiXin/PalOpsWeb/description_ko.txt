PalOps Web은 Palworld Dedicated Server를 위한 현대적인 올인원 운영 콘솔로, PalDefender 안티치트 플러그인과의 깊은 통합을 중심에 둡니다. PalDefender REST, 확장 RCON, 버전 검사, 화이트리스트 및 차단 데이터, 관리형 JSON 구성을 결합하고, 네이티브 Palworld REST, 서버 수명주기 제어, 읽기 전용 세이브 인덱싱, 플레이어 및 길드 인텔리전스, 오프라인 MapLibre 월드 맵, 리소스 지급, 백업, 자동화 작업, webhook 알림, 감사 가능한 접근 제어를 제공합니다. 스택은 .NET 10 기반 ASP.NET Core와 Vue 3 프론트엔드이며, 일반 클라우드 패널이 아니라 게임 서버와 동일한 Windows 호스트에서 실행되도록 설계되었습니다. 핵심 관리 기능에는 PalDefender가 선택 사항이며, 보호 및 확장 런타임 기능은 안티치트를 인식한 Palworld 관리가 필요한 서버 운영자를 대상으로 합니다.
