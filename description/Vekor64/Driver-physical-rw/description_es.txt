Este proyecto es un controlador de kernel de Windows que expone manejadores IOCTL para operaciones de memoria física y virtual.
Está escrito en C++ e incluye estructuras de solicitud y patrones de comunicación usuario-kernel para clientes basados en DeviceIoControl.
Las rutinas principales cubren primitivas de lectura y escritura, asignación de memoria y cambios de protección, búsqueda de la base de módulos y ayudas orientadas a procesos.
Se usa principalmente para experimentación de seguridad a bajo nivel y estudios de comunicación de controladores orientados a trampas.
