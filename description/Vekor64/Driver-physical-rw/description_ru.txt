Этот проект — драйвер ядра Windows, предоставляющий обработчики IOCTL для операций с физической и виртуальной памятью.
Он написан на C++ и включает структуры запросов, а также схемы взаимодействия пользовательского режима с ядром для клиентов на основе DeviceIoControl.
Основные процедуры охватывают примитивы чтения и записи, выделение памяти и изменение защиты, поиск базового адреса модуля, а также вспомогательные функции для работы с процессами.
В основном используется для низкоуровневых экспериментов по безопасности и исследований коммуникации драйверов в контексте читов.
