Este proyecto es una implementación de estudio de cheat externo para Counter-Strike 2 basada en Python.
Usa PyMeow para el acceso a memoria y el renderizado, junto con controles basados en DearPyGui para la configuración en tiempo de ejecución.
Las funciones incluidas cubren visuales ESP comunes como cajas, barras de salud, etiquetas de arma y distancia, renderizado de líneas y lógica de control de retroceso.
El repositorio se presenta como una referencia educativa para comprender flujos de trabajo básicos de game-hacking externo en Python.
