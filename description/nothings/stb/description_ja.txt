本プロジェクトは stb で、ゲーム開発者とグラフィックスプログラマ向けの単一ファイル・パブリックドメイン C ライブラリ集です。代表的なライブラリには、stb_image（JPEG/PNG/BMP/GIF の画像読み込み）、stb_truetype（TrueType フォントのラスタライズ）、stb_vorbis（Ogg Vorbis 音声デコード）、stb_image_write（画像書き出し）があります。各ライブラリは単一のヘッダファイルで、依存関係なしに任意の C/C++ プロジェクトへ組み込めます。軽量で依存のないユーティリティライブラリを求めるゲーム開発者や組み込みシステムプログラマを対象としています。
