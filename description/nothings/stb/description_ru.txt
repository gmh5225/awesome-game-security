Этот проект — stb, набор однофайловых C-библиотек общественного достояния для разработчиков игр и графических программистов. Среди наиболее известных библиотек — stb_image (загрузка изображений JPEG/PNG/BMP/GIF), stb_truetype (растеризация шрифтов TrueType), stb_vorbis (декодирование аудио Ogg Vorbis) и stb_image_write (запись изображений). Каждая библиотека — это один заголовочный файл, который можно подключить к любому проекту на C/C++ без зависимостей. Она предназначена для разработчиков игр и программистов встраиваемых систем, которым нужны лёгкие утилитарные библиотеки без зависимостей.
