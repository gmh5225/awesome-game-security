Questo progetto è stb, una raccolta di librerie C di pubblico dominio in un singolo file per sviluppatori di giochi e programmatori grafici. Le librerie più note includono stb_image (caricamento immagini JPEG/PNG/BMP/GIF), stb_truetype (rasterizzazione di font TrueType), stb_vorbis (decodifica audio Ogg Vorbis) e stb_image_write (scrittura di immagini). Ogni libreria è un singolo file header che può essere inserito in qualsiasi progetto C/C++ senza dipendenze. È rivolta a sviluppatori di giochi e programmatori di sistemi embedded che cercano librerie di utilità leggere e senza dipendenze.
