Этот проект — Cozinha Loader, загрузчик читов для CS:GO на C++, который выполняет DLL-инъекцию с мерами антидетекта. Он загружает модули читов с удалённого сервера, реализует process hollowing или manual mapping injection и включает антиотладочные проверки, шифрование строк и обфускацию импортов для обхода сканирования античита. Предназначен для исследователей игровой безопасности, изучающих техники антидетекта загрузчиков читов и процессы инъекции.
