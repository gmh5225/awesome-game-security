Oxide Dumper es un proyecto de automatización que mantiene actualizados los offsets del juego Rust y los exporta en un formato de cabecera reutilizable.
Usa scripts de Python con SteamCMD, Il2CppDumper y GitHub Actions para obtener datos del juego, procesar dumps y regenerar las definiciones de offsets.
El repositorio prioriza la extracción repetible de offsets frente a pasos manuales puntuales de ingeniería inversa.
Está pensado para investigadores de seguridad en juegos y mantenedores de herramientas que siguen las frecuentes actualizaciones de Rust.
