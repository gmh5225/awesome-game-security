Oxide Dumper è un progetto di automazione che mantiene aggiornati gli offset del gioco Rust e li esporta in un formato header riutilizzabile.
Usa script Python con SteamCMD, Il2CppDumper e GitHub Actions per recuperare i dati di gioco, elaborare i dump e rigenerare le definizioni degli offset.
Il repository privilegia un'estrazione ripetibile degli offset rispetto a passaggi manuali una tantum di reverse engineering.
È pensato per ricercatori di sicurezza nei giochi e maintainer di strumenti che seguono i frequenti aggiornamenti di Rust.
