Oxide Dumper — проект автоматизации, который поддерживает в актуальном состоянии оффсеты игры Rust и экспортирует их в переиспользуемом формате заголовочных файлов.
Он использует Python-скрипты вместе со SteamCMD, Il2CppDumper и GitHub Actions для загрузки игровых данных, обработки дампов и повторной генерации определений оффсетов.
Репозиторий делает акцент на повторяемом извлечении оффсетов вместо разовых ручных шагов реверс-инжиниринга.
Он предназначен для исследователей игровой безопасности и сопровождающих инструменты специалистов, которые отслеживают частые обновления Rust.
