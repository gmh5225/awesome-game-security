Oxide Dumper ist ein Automatisierungsprojekt, das Offsets des Spiels Rust aktuell hält und sie in einem wiederverwendbaren Header-Format exportiert.
Es nutzt Python-Skripte mit SteamCMD, Il2CppDumper und GitHub Actions, um Spieldaten abzurufen, Dumps zu verarbeiten und Offset-Definitionen neu zu erzeugen.
Das Repository legt Wert auf wiederholbare Offset-Extraktion statt manueller Einmal-Schritte beim Reverse Engineering.
Es richtet sich an Spielsicherheitsforscher und Tool-Maintainer, die häufige Rust-Updates verfolgen.
