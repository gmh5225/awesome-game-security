Oxide Dumper est un projet d'automatisation qui maintient à jour les offsets du jeu Rust et les exporte dans un format d'en-tête réutilisable.
Il s'appuie sur des scripts Python avec SteamCMD, Il2CppDumper et GitHub Actions pour récupérer les données du jeu, traiter les dumps et régénérer les définitions d'offsets.
Le dépôt privilégie une extraction d'offsets reproductible plutôt que des étapes manuelles ponctuelles de reverse engineering.
Il s'adresse aux chercheurs en sécurité des jeux et aux mainteneurs d'outils qui suivent les mises à jour fréquentes de Rust.
