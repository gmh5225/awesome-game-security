FastLogs 是一款开源的调试与缺陷上报引擎，面向难以或无法打开引擎控制台的游戏构建环境，例如 WebGL、门户、移动端和主机平台。在运行中的构建里，一次手势或 agent 调用即可将近期日志行、设备信息、上下文面包屑、截图和场景快照发送到自托管的 Node.js 服务器；服务器用 SQLite 存储报告，并返回可分享的简短查看链接。客户端覆盖 Unity（C# UPM 包）和 GameMaker（GML），共用同一套 JSON 接入约定，功能包括崩溃自动采集与离线发件箱、带全文搜索与分诊的可浏览目录、可选接入 Slack、Discord、webhook 与问题跟踪系统，以及默认启用的 PII 脱敏。它还提供无界面发送路径和远程命令通道，便于 QA、开发者或 AI agent 在无调试器的情况下检查并驱动实机构建。主要用途是缩短从分布式构建中出现缺陷到开发者在可筛选的浏览器控制台中看到原因之间的闭环。
