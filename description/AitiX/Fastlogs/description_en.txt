FastLogs is an open-source debug and bug-reporting engine for game builds where the engine console is hard or impossible to reach, such as WebGL, portals, mobile, and consoles. From a running build, a single gesture or agent call can ship recent log lines, device info, context breadcrumbs, screenshots, and scene snapshots to a self-hosted Node.js server that stores reports in SQLite and returns a short shareable viewer link. Clients exist for Unity (C# UPM package) and GameMaker (GML) under one shared JSON ingest contract, with features including crash auto-capture and offline outbox, a browsable catalog with full-text search and triage, optional sinks to Slack, Discord, webhooks, and issue trackers, plus PII scrubbing by default. It also exposes a headless send path and a remote command channel so QA, developers, or AI agents can inspect and drive a live build without a debugger. The primary use case is shortening the loop from a bug in a distributed build to a developer seeing the cause in a filterable browser console.
