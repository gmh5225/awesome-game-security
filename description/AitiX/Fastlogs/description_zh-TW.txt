FastLogs 是一款開源的除錯與缺陷回報引擎，面向難以或無法開啟引擎主控台的遊戲建置環境，例如 WebGL、入口站、行動裝置與主機平台。在執行中的建置裡，一次手勢或 agent 呼叫即可將近期日誌列、裝置資訊、上下文麵包屑、截圖與場景快照送至自架的 Node.js 伺服器；伺服器用 SQLite 儲存報告，並回傳可分享的簡短檢視連結。用戶端涵蓋 Unity（C# UPM 套件）與 GameMaker（GML），共用同一套 JSON 接收約定，功能包括當機自動擷取與離線寄件匣、具備全文搜尋與分診的可瀏覽目錄、可選接至 Slack、Discord、webhook 與問題追蹤系統，以及預設啟用的 PII 脫敏。它也提供無介面傳送路徑與遠端命令通道，方便 QA、開發者或 AI agent 在沒有除錯器的情況下檢查並驅動實機建置。主要用途是縮短從分散式建置出現缺陷到開發者在可篩選的瀏覽器主控台中看到原因之間的迴圈。
