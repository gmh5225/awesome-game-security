FastLogs는 WebGL, 포털, 모바일, 콘솔처럼 엔진 콘솔에 접근하기 어렵거나 불가능한 게임 빌드를 위한 오픈소스 디버그 및 버그 보고 엔진입니다. 실행 중인 빌드에서 한 번의 제스처나 에이전트 호출로 최근 로그 줄, 기기 정보, 컨텍스트 브레드크럼, 스크린샷, 씬 스냅샷을 자체 호스팅 Node.js 서버로 보낼 수 있으며, 서버는 보고서를 SQLite에 저장하고 짧게 공유 가능한 뷰어 링크를 반환합니다. 클라이언트는 Unity(C# UPM 패키지)와 GameMaker(GML)용으로 제공되며, 공통 JSON 수집 계약을 따르고 크래시 자동 캡처와 오프라인 아웃박스, 전문 검색·분류가 가능한 브라우징 카탈로그, Slack·Discord·웹훅·이슈 트래커로의 선택적 싱크, 기본 PII 스크러빙 등의 기능을 갖춥니다. 헤드리스 전송 경로와 원격 명령 채널도 제공해 QA, 개발자 또는 AI 에이전트가 디버거 없이 라이브 빌드를 검사하고 제어할 수 있습니다. 주요 사용 사례는 분산 빌드의 버그에서부터 개발자가 필터링 가능한 브라우저 콘솔에서 원인을 확인하기까지의 루프를 줄이는 것입니다.
