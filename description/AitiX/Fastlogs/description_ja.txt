FastLogs は、WebGL、ポータル、モバイル、コンソールなど、エンジンコンソールに到達しにくい、あるいは到達できないゲームビルド向けのオープンソースのデバッグ／バグレポートエンジンです。実行中のビルドから、単一のジェスチャーまたはエージェント呼び出しで、直近のログ行、デバイス情報、コンテキストのブレッドクラム、スクリーンショット、シーンスナップショットを、自己ホストの Node.js サーバーへ送れます。サーバーはレポートを SQLite に保存し、短い共有可能なビューアリンクを返します。クライアントは Unity（C# UPM パッケージ）と GameMaker（GML）向けがあり、共通の JSON 取り込み契約のもとで、クラッシュの自動キャプチャとオフラインアウトボックス、全文検索とトリアージ付きの閲覧可能なカタログ、Slack／Discord／webhook／課題トラッカーへの任意シンク、デフォルトの PII スクラブなどを提供します。ヘッドレス送信パスとリモートコマンドチャネルも備え、QA、開発者、AI エージェントがデバッガなしでライブビルドを検査・操作できます。主な用途は、分散ビルド上のバグから、開発者がフィルタ可能なブラウザコンソールで原因を確認するまでのループを短縮することです。
