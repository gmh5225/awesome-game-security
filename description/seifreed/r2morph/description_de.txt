Dieses Projekt ist eine metamorphic Binary-Transformations-Engine auf Basis von r2pipe und radare2.
Es ist hauptsächlich in Python und C geschrieben und konzentriert sich auf Modding, Speicheranalyse und Debugging.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / radare-Plugins untersuchen.
