Questo progetto è una libreria di hooking del kernel Windows scritta in C che intercetta le system call manipolando i puntatori a funzioni interni di ETW (Event Tracing for Windows). Patcha il puntatore al callback di tracciamento delle syscall ETW all'interno del kernel per reindirizzare il flusso di controllo, consentendo un'intercettazione trasparente delle syscall senza modificare l'SSDT né applicare patch inline a ntoskrnl. La tecnica supera i controlli di PatchGuard poiché opera attraverso un percorso di codice ETW legittimo. È rivolta ai ricercatori del kernel che studiano l'hooking stealth delle syscall, i bypass degli anti-cheat e l'instrumentazione del kernel compatibile con PatchGuard.
