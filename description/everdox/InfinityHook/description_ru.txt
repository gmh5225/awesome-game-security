Этот проект — библиотека хукинга ядра Windows на языке C, которая перехватывает системные вызовы путём манипуляции внутренними указателями функций ETW (Event Tracing for Windows). Она патчит указатель callback трассировки syscall в ETW внутри ядра для перенаправления потока управления, обеспечивая прозрачный перехват syscall без изменения SSDT и без inline-патчинга ntoskrnl. Техника проходит проверки PatchGuard, поскольку работает через легитимный путь кода ETW. Она предназначена для исследователей ядра, изучающих скрытый хукинг syscall, обходы античита и инструментирование ядра, совместимое с PatchGuard.
