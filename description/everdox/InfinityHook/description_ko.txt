이 프로젝트는 ETW(Event Tracing for Windows) 내부 함수 포인터를 조작하여 시스템 호출을 가로채는 C 언어 Windows 커널 후킹 라이브러리입니다. 커널 내부의 ETW 시스템 호출 추적 콜백 포인터를 패치하여 제어 흐름을 리다이렉트하며, SSDT를 수정하거나 ntoskrnl에 인라인 패치를 하지 않고도 투명한 시스템 호출 가로채기를 가능하게 합니다. 정상적인 ETW 코드 경로를 통해 동작하므로 PatchGuard 검사를 통과합니다. 은밀한 시스템 호출 후킹, 안티치트 우회, PatchGuard 호환 커널 계측을 연구하는 커널 연구자를 대상으로 합니다.
