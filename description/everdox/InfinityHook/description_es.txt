Este proyecto es una biblioteca de hooking del kernel de Windows escrita en C que intercepta llamadas al sistema manipulando punteros a funciones internas de ETW (Event Tracing for Windows). Parchea el puntero de callback de seguimiento de syscalls de ETW dentro del kernel para redirigir el flujo de control, permitiendo una interceptación transparente de syscalls sin modificar la SSDT ni aplicar parches en línea a ntoskrnl. La técnica supera las comprobaciones de PatchGuard porque opera a través de una ruta de código ETW legítima. Está dirigida a investigadores del kernel que estudian el hooking sigiloso de syscalls, las evasiones de anti-cheat y la instrumentación del kernel compatible con PatchGuard.
