Ce projet est une bibliothèque de hooking du noyau Windows écrite en C qui intercepte les appels système en manipulant les pointeurs de fonctions internes d'ETW (Event Tracing for Windows). Elle patche le pointeur de rappel de traçage des syscalls ETW dans le noyau pour rediriger le flux de contrôle, permettant une interception transparente des syscalls sans modifier la SSDT ni appliquer de patchs inline à ntoskrnl. La technique passe les contrôles de PatchGuard car elle s'appuie sur un chemin de code ETW légitime. Elle s'adresse aux chercheurs noyau qui étudient le hooking furtif des syscalls, les contournements d'anti-cheat et l'instrumentation noyau compatible avec PatchGuard.
