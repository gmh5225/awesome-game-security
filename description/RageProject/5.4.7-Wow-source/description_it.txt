Questo progetto è un albero sorgente personalizzato di server privato World of Warcraft 5.4.7, derivato da un'architettura in stile TrinityCore.
È scritto in gran parte in C e C++ con script di build CMake, componenti di server di autenticazione e world, nonché risorse di database e strumenti di supporto.
Le note incluse sottolineano correzioni di gameplay, supporto a classi e talenti e miglioramenti generali di stabilità per una configurazione server dell'era livello 90.
È pensato per sviluppatori di server privati e comunità di reverse engineering che lavorano sull'emulazione di server MMO.
