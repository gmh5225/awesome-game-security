Questo progetto è un launcher di gioco pubblico per client di World of Warcraft che può connettersi a server personalizzati usando certificati TLS validi. È sviluppato principalmente in C# su .NET e supporta più branch di client, con versioni del client ed endpoint CDN configurabili. La build pubblica rimuove intenzionalmente la modalità sviluppatore, il caricamento dei mod e altri strumenti avanzati presenti nell'edizione completa. È pensato soprattutto per la gestione di server privati e per i test di compatibilità dei client.
