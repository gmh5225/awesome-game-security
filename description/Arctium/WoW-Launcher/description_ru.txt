Этот проект — публичный игровой лаунчер для клиентов World of Warcraft, способный подключаться к пользовательским серверам с помощью действительных TLS-сертификатов. Он в основном написан на C# под .NET и поддерживает несколько веток клиентов с настраиваемыми версиями клиента и CDN-эндпоинтами. В публичной сборке намеренно убраны режим разработчика, загрузка модов и другие расширенные инструменты полной редакции. Он в первую очередь предназначен для работы частных серверов и проверки совместимости клиентов.
