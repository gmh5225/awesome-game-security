Dieses Projekt ist ein öffentlicher Spiel-Launcher für World-of-Warcraft-Clients, der sich mit gültigen TLS-Zertifikaten mit benutzerdefinierten Servern verbinden kann. Es ist hauptsächlich in C# auf .NET geschrieben und unterstützt mehrere Client-Zweige mit konfigurierbaren Client-Versionen und CDN-Endpunkten. Der öffentliche Build entfernt absichtlich den Entwicklermodus, das Laden von Mods und andere erweiterte Werkzeuge der Vollversion. Es richtet sich vor allem an den Betrieb privater Server und an Kompatibilitätstests von Clients.
