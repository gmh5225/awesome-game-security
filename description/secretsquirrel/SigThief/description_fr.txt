Ce projet se concentre sur le vol de signatures depuis des fichiers PE.
Certains éditeurs d’antivirus accordent la priorité à certaines autorités de certification sans vérifier que la signature est réellement valide, et d’autres se contentent de vérifier que la certTable contient une valeur quelconque.
Il est surtout utile aux chercheurs bas niveau Windows, Linux et mobile qui travaillent dans le domaine de certaines astuces / Windows ring3.
