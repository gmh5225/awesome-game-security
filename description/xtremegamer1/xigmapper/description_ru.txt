Этот проект посвящён EFI Manual Map.
Драйвер, который вы хотите загрузить, не может находиться на USB-устройстве, поскольку USB-устройства обнаруживаются и загружаются Windows уже после Vanguard.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / EFI-драйверов.
