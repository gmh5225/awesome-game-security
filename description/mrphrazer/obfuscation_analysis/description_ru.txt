Плагин Binary Ninja для анализа и упрощения обфусцированного кода: упрощение MBA-выражений (Mixed Boolean-Arithmetic) через обратный слайсинг и поиск по оракулу (msynth), обнаружение непрозрачных предикатов с помощью анализа потока данных и автоматизированные сценарии деобфускации. Преобразует выражения BNIL в формы, совместимые с Z3, для семантической проверки корректности упрощений.
