Plugin Binary Ninja pour analyser et simplifier le code obfusqué, avec simplification d’expressions MBA (Mixed Boolean-Arithmetic) via backward slicing et recherche basée sur un oracle (msynth), détection de prédicats opaques par analyse de flux de données, et workflows de désobfuscation automatisés. Traduit les expressions BNIL en formes compatibles Z3 pour vérifier sémantiquement l’exactitude des simplifications.
