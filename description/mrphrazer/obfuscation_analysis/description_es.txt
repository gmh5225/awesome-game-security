Complemento de Binary Ninja para analizar y simplificar código ofuscado, con simplificación de expresiones MBA (Mixed Boolean-Arithmetic) mediante backward slicing y búsqueda basada en oráculo (msynth), detección de predicados opacos por análisis de flujo de datos y flujos de trabajo automatizados de desofuscación. Traduce expresiones BNIL a formas compatibles con Z3 para verificar semánticamente la corrección de las simplificaciones.
