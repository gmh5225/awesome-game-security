MCP-сервер, управляющий headless-экземплярами Ghidra для реверс-инжиниринга с помощью ИИ и предоставляющий более 40 инструментов в областях дизассемблирования, декомпиляции, перекрёстных ссылок, управления символами/типами и скриптинга. Включает фейковый бэкенд для офлайн-тестирования, CLI для работы без MCP, поддержку фаззинг-тестов и интеграцию с агентами Claude/Cursor/Copilot через JSON-RPC по stdio.
