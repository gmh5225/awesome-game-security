Server MCP che guida istanze headless di Ghidra per il reverse engineering assistito dall'IA, esponendo oltre 40 strumenti tra disassemblaggio, decompilazione, riferimenti incrociati, gestione di simboli/tipi e scripting. Offre un backend fittizio per test offline, una CLI per l'uso senza MCP, supporto al fuzz testing e integrazione con agenti Claude/Cursor/Copilot via JSON-RPC su stdio.
