Servidor MCP que controla instancias headless de Ghidra para ingeniería inversa asistida por IA, exponiendo más de 40 herramientas de desensamblado, descompilación, referencias cruzadas, gestión de símbolos/tipos y scripting. Incluye un backend falso para pruebas sin conexión, una CLI para uso sin MCP, soporte de fuzz testing e integración con agentes Claude/Cursor/Copilot mediante JSON-RPC sobre stdio.
