Plugin Binary Ninja qui identifie les régions de code obfuscées à l'aide de plusieurs heuristiques : détection de l'aplatissement du flot de contrôle via l'analyse de boucles et les arbres de domination, métriques de complexité au niveau des instructions, analyse de fréquence des n-grammes comparant les blocs de base à une base de référence, et détection statistique des valeurs aberrantes. Inclut des scripts de traitement par lots pour des campagnes d'analyse binaire à grande échelle.
