Plugin di Binary Ninja che identifica regioni di codice offuscato mediante più euristiche: rilevamento dell'appiattimento del flusso di controllo tramite analisi dei loop e alberi dei dominatori, metriche di complessità a livello di istruzione, analisi di frequenza degli n-gram che confronta i basic block con un database di riferimento, e rilevamento statistico degli outlier. Include script di elaborazione batch per campagne di analisi binaria su larga scala.
