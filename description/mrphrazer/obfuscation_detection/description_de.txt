Binary-Ninja-Plugin, das obfuskierte Codebereiche anhand mehrerer Heuristiken erkennt: Erkennung von Control-Flow-Flattening über Schleifenanalyse und Dominatorbäume, Komplexitätsmetriken auf Instruktionsebene, N-Gramm-Häufigkeitsanalyse zum Vergleich von Basic Blocks mit einer Referenzdatenbank sowie statistische Ausreißererkennung. Enthält Batch-Verarbeitungsskripte für großangelegte Binary-Analyse-Kampagnen.
