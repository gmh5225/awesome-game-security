Complemento de Binary Ninja que identifica regiones de código ofuscado mediante múltiples heurísticas: detección de aplanamiento del flujo de control mediante análisis de bucles y árboles de dominadores, métricas de complejidad a nivel de instrucción, análisis de frecuencia de n-gramas que compara bloques básicos con una base de datos de referencia, y detección estadística de valores atípicos. Incluye scripts de procesamiento por lotes para campañas de análisis binario a gran escala.
