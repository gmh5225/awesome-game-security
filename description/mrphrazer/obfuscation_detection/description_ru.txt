Плагин Binary Ninja, выявляющий обфусцированные участки кода с помощью нескольких эвристик: обнаружение уплощения потока управления через анализ циклов и деревья доминаторов, метрики сложности на уровне инструкций, анализ частоты n-грамм с сравнением базовых блоков с эталонной базой данных, а также статистическое обнаружение выбросов. Включает скрипты пакетной обработки для масштабных кампаний анализа бинарных файлов.
