VioletWing 是一款外部 Counter-Strike 2 輔助工具，透過讀取遊戲行程記憶體，在桌面 GUI 中提供 TriggerBot、ESP 覆蓋層、連跳（bunnyhop）與 NoFlash 等功能。主要以 Python 撰寫，使用 PyMeow 實作覆蓋層、customtkinter 建構介面，並在啟動時對正在執行的 CS2 行程執行 cs2-dumper，以便在遊戲更新後刷新記憶體偏移。主要能力包括基於準星的自動開火（可依武器設定延遲），帶方框、骨骼、射線、血條、炸彈計時器與觀戰列表的 ESP，以及與地面旗標同步的連跳與閃光持續時間鉗制。面向離線或私人伺服器上的 CS2 逆向工程與遊戲安全實驗，而非線上配對對戰。
