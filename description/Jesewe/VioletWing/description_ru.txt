VioletWing — внешний помощник для Counter-Strike 2, который читает память игрового процесса и через настольный GUI предоставляет TriggerBot, ESP-оверлей, bunnyhop и NoFlash. Написан в основном на Python: для оверлея используется PyMeow, для интерфейса — customtkinter, а при запуске cs2-dumper выполняется против живого процесса CS2, чтобы обновить смещения памяти после обновлений игры. Ключевые возможности: автоогонь по прицелу с задержками для каждого оружия; ESP с боксами, скелетами, snaplines, полосками здоровья, таймером бомбы и списком зрителей; bunnyhop, синхронизированный с флагом земли, и ограничение длительности флеша. Предназначен для реверс-инжиниринга CS2 и экспериментов по игровой безопасности офлайн или на частных серверах, а не для онлайн-матчмейкинга.
