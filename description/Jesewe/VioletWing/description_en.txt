VioletWing is an external Counter-Strike 2 helper that reads game process memory and provides TriggerBot, ESP overlay, bunnyhop, and NoFlash features through a desktop GUI. Written primarily in Python, it uses PyMeow for the overlay, customtkinter for the interface, and runs cs2-dumper against the live CS2 process at startup to refresh memory offsets after game updates. Key capabilities include crosshair-based auto-fire with per-weapon delays, ESP with boxes, skeletons, snaplines, health bars, bomb timer, and spectator list, plus ground-flag-synchronized bunnyhop and flash-duration clamping. It is aimed at CS2 reverse-engineering and game-security experimentation offline or on private servers, not online matchmaking.
