Este proyecto se centra en un complemento no oficial de YARA para IDA Pro.
La tecla rápida predeterminada de IDA es «Alt-Y», pero se puede configurar otra mediante el archivo de configuración «plugins.cfg» de IDA.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
