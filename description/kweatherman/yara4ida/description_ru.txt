Этот проект посвящён неофициальному плагину YARA для IDA Pro.
Горячая клавиша IDA по умолчанию — «Alt-Y», но её можно изменить через конфигурационный файл IDA «plugins.cfg».
Он в первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / плагинов IDA.
