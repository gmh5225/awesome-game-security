Ce projet porte sur un plugin YARA non officiel pour IDA Pro.
Le raccourci IDA par défaut est « Alt-Y », mais il peut être modifié via le fichier de configuration « plugins.cfg » d’IDA.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / plugins IDA.
