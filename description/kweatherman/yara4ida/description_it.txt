Questo progetto riguarda un plugin YARA non ufficiale per IDA Pro.
Il tasto rapido predefinito di IDA è «Alt-Y», ma può essere configurato su un altro tramite il file di configurazione «plugins.cfg» di IDA.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell’ambito di cheat / plugin IDA.
