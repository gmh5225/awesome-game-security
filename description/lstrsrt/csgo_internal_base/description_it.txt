Questo progetto è una base per cheat interni di CS:GO che offre un framework pulito per costruire cheat interni per il motore Source. Include risoluzione delle interfacce del motore, infrastruttura di VMT hooking, dump dei netvar, pattern scanning e un menu overlay ImGui di base. La base in C++ fornisce la struttura di codice fondamentale su cui si costruiscono le funzionalità di cheat (ESP, aimbot, ecc.). È rivolta ai ricercatori di sicurezza nei giochi che studiano l'architettura tipica dei cheat interni e i pattern di hook del motore Source.
