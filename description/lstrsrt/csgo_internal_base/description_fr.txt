Ce projet est une base de cheat interne pour CS:GO offrant un framework propre pour construire des cheats internes pour le moteur Source. Il inclut la résolution des interfaces du moteur, une infrastructure de VMT hooking, le dump des netvars, le pattern scanning et un menu overlay ImGui basique. Cette base C++ fournit la structure de code fondamentale sur laquelle s'appuient les fonctionnalités de cheat (ESP, aimbot, etc.). Elle s'adresse aux chercheurs en sécurité des jeux qui étudient l'architecture typique des cheats internes et les schémas de hooks du moteur Source.
