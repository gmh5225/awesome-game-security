Este proyecto es una implementación en terminal del juego de puzzle 2048 escrita en C++. Renderiza el tablero en la consola con soporte de color, gestiona la entrada de teclado para deslizar las fichas, implementa la lógica de fusión y puntuación, y registra las puntuaciones máximas. La base de código limpia y minimalista sirve como ejemplo de desarrollo de juegos de consola en C++. Está dirigida a estudiantes de C++ y desarrolladores de juegos aficionados que estudian la implementación de lógica de juego sencilla.
