Этот проект — реализация головоломки 2048 для терминала на C++. Он отрисовывает игровое поле в консоли с поддержкой цвета, обрабатывает ввод с клавиатуры для сдвига плиток, реализует логику слияния и подсчёта очков, а также отслеживает рекорды. Чистая и минималистичная кодовая база служит примером разработки консольных игр на C++. Проект рассчитан на изучающих C++ и на любителей игровой разработки, которые разбирают реализацию простой игровой логики.
