Este proyecto es un puente que sincroniza el contexto de depuración GDB en vivo con la interfaz de GHIDRA.
Combina un script cliente Python del lado de GDB con una extensión Java para GHIDRA para intercambiar el estado de ejecución en tiempo real.
Las capacidades clave incluyen sincronización del cursor y de la pila, propagación de registros para un mejor contexto de descompilación, control de puntos de interrupción y manejo de reubicación.
Lo usan principalmente ingenieros de ingeniería inversa que quieren que la depuración interactiva y el análisis estático trabajen juntos con fluidez.
