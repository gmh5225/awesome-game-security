Этот проект — мост, синхронизирующий живой контекст отладки GDB с интерфейсом GHIDRA.
Он сочетает клиентский Python-скрипт на стороне GDB с Java-расширением для GHIDRA для обмена состоянием выполнения в реальном времени.
Ключевые возможности включают синхронизацию курсора и стека, распространение регистров для лучшего контекста декомпиляции, управление точками останова и обработку релокаций.
Его в основном используют специалисты по реверс-инжинирингу, которым нужно, чтобы интерактивная отладка и статический анализ работали вместе плавно.
