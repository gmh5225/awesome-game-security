Questo progetto è un ponte che sincronizza il contesto di debug GDB in tempo reale con l'interfaccia GHIDRA.
Combina uno script client Python lato GDB con un'estensione Java per GHIDRA per scambiare lo stato di esecuzione in tempo reale.
Le capacità principali includono la sincronizzazione di cursore e stack, la propagazione dei registri per un migliore contesto di decompilazione, il controllo dei breakpoint e la gestione della relocation.
È usato principalmente da reverse engineer che vogliono far collaborare in modo fluido il debugging interattivo e l'analisi statica.
