Ce projet est une preuve de concept pour le noyau Windows qui accroche le callback de code hypercall utilisé lors du basculement d’espace d’adressage. Il localise les routines internes par scan de motifs, installe un callback personnalisé et ajuste à l’exécution les drapeaux liés à l’enlightenment. L’implémentation C++ se concentre sur la gestion bas niveau de CR3 et sur la logique de configuration des pages hypercall par processeur. Son cas d’usage principal est la recherche avancée sur les internals du noyau et le comportement des hyperviseurs dans le contexte de la sécurité des jeux.
