Этот проект — proof of concept для ядра Windows, перехватывающий callback кода hypercall, используемый при переключении адресного пространства. Он находит внутренние процедуры через сканирование по сигнатурам, устанавливает пользовательский callback и во время выполнения корректирует флаги, связанные с enlightenment. Реализация на C++ сосредоточена на низкоуровневой работе с CR3 и логике настройки страниц hypercall для каждого процессора. Основной сценарий использования — продвинутое исследование внутренних механизмов ядра и поведения гипервизора в контексте игровой безопасности.
