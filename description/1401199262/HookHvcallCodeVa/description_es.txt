Este proyecto es una prueba de concepto para el kernel de Windows que engancha (hook) el callback de código hypercall usado durante el cambio de espacio de direcciones. Localiza rutinas internas mediante escaneo de patrones, instala un callback personalizado y ajusta en tiempo de ejecución los flags relacionados con enlightenment. La implementación en C++ se centra en el manejo de bajo nivel de CR3 y en la lógica de configuración de páginas hypercall por procesador. Su caso de uso principal es la investigación avanzada de internos del kernel y del comportamiento del hipervisor en contextos de seguridad de juegos.
