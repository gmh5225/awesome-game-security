Questo progetto è una proof of concept per il kernel Windows che effettua l’hook del callback del codice hypercall usato durante il cambio di spazio di indirizzi. Individua le routine interne tramite pattern scanning, installa un callback personalizzato e regola a runtime i flag relativi all’enlightenment. L’implementazione in C++ si concentra sulla gestione a basso livello di CR3 e sulla logica di configurazione delle pagine hypercall per processore. Il caso d’uso principale è la ricerca avanzata sugli interni del kernel e sul comportamento dell’hypervisor nei contesti di sicurezza dei giochi.
