Ce projet est une preuve de concept noyau Windows pour détecter les pilotes sans module dissimulés via une marche de pile basée sur les NMI.
Il envoie des interruptions non masquables aux CPU sélectionnés et effectue des backtraces de pile dans un callback NMI afin d'inspecter les chemins d'exécution suspects.
L'implémentation est en C, avec une structure de projet pilote noyau Visual Studio standard.
Son cas d'usage principal est la recherche de détection de type anti-rootkit et anti-cheat au niveau noyau.
