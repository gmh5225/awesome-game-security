Questo progetto è una proof of concept a livello di kernel Windows per rilevare driver nascosti senza modulo tramite stack walking basato su NMI.
Invia interrupt non mascherabili alle CPU selezionate ed esegue stack backtrace all'interno di un callback NMI per ispezionare percorsi di esecuzione sospetti.
L'implementazione è in C con il layout standard di un progetto driver kernel di Visual Studio.
Il caso d'uso principale è la ricerca sulla detection in stile anti-rootkit e anti-cheat a livello kernel.
