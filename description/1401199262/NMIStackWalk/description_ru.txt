Этот проект — proof of concept для ядра Windows, предназначенный для обнаружения скрытых драйверов без модуля с помощью обхода стека на основе NMI.
Он отправляет немаскируемые прерывания выбранным CPU и выполняет трассировку стека внутри NMI-callback для проверки подозрительных путей выполнения.
Реализация написана на C в стандартной структуре проекта драйвера ядра Visual Studio.
Основной сценарий использования — исследования по обнаружению в стиле anti-rootkit и anti-cheat на уровне ядра.
