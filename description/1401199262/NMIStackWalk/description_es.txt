Este proyecto es una prueba de concepto de kernel de Windows para detectar controladores ocultos sin módulo mediante un recorrido de pila basado en NMI.
Envía interrupciones no enmascarables a CPU seleccionadas y realiza backtraces de pila dentro de un callback NMI para inspeccionar rutas de ejecución sospechosas.
La implementación está en C con la estructura estándar de un proyecto de controlador de kernel de Visual Studio.
Su caso de uso principal es la investigación de detección al estilo anti-rootkit y anti-cheat a nivel de kernel.
