Este proyecto es una técnica del kernel de Windows que ejecuta código en modo usuario en un proceso objetivo arbitrario.
Encadena una APC del kernel con KeUserModeCallback, desvía la ejecución a través de una rutina de E/S del controlador y regresa con una restauración controlada del contexto.
La implementación en C++ evita asignar memoria de shellcode RWX en el proceso objetivo y, aun así, permite una ejecución invocable en modo usuario.
Su principal caso de uso es la investigación avanzada de inyección de procesos y la evaluación de compromisos de detección en entornos de seguridad de juegos.
