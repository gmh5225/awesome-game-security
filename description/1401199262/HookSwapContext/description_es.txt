Este proyecto es una prueba de concepto del kernel de Windows para enganchar rutas de ejecución relacionadas con el cambio de contexto.
Utiliza una ruta de hook basada en ETW/CKCL y comprobaciones personalizadas de marcos de pila para invocar un controlador durante un flujo de planificación seleccionado.
El código está escrito en C++ e incluye módulos de bajo nivel para el control de trazas, hooks orientados a syscalls y rutinas de utilidad del kernel.
Su caso de uso principal es experimentar con técnicas de interceptación de la planificación de hilos para la investigación de seguridad del kernel.
