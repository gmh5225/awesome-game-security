Этот проект — proof of concept для ядра Windows, предназначенный для перехвата путей выполнения, связанных с переключением контекста.
Он использует путь перехвата на основе ETW/CKCL и пользовательские проверки стековых кадров, чтобы вызывать обработчик в выбранном потоке планирования.
Код написан на C++ и включает низкоуровневые модули для управления трассировкой, хуков на стороне syscall и вспомогательных ядерных процедур.
Основной сценарий использования — эксперименты с техниками перехвата планирования потоков для исследований безопасности ядра.
