QueryWorkingSetExample es una demostración en C de una técnica anti-manipulación basada en los metadatos del working set de Windows.
Muestra cómo comprobar el estado de las páginas compartidas en regiones no escribibles como .text puede revelar puntos de interrupción o cambios de protección introducidos durante la depuración.
El ejemplo usa las API QueryWorkingSet e incluye archivos de compilación mínimos y capturas de pantalla para ilustrar la ejecución normal frente a la manipulada.
Está pensado para ingenieros inversos y defensores que estudian comprobaciones ligeras de integridad de memoria.
