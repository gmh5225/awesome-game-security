QueryWorkingSetExample è una dimostrazione in C di una tecnica anti-manomissione basata sui metadati del working set di Windows.
Mostra come controllare lo stato delle pagine condivise in regioni non scrivibili come .text possa rivelare breakpoint o modifiche alle protezioni introdotte durante il debugging.
L'esempio usa le API QueryWorkingSet e include file di build minimi e screenshot per illustrare l'esecuzione normale rispetto a quella manomessa.
È pensato per reverse engineer e difensori che studiano controlli leggeri di integrità della memoria.
