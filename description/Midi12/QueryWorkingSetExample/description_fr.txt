QueryWorkingSetExample est une démonstration en C d'une technique anti-altération fondée sur les métadonnées du working set Windows.
Elle montre comment vérifier l'état des pages partagées dans des régions non modifiables telles que .text peut révéler des points d'arrêt ou des changements de protection introduits pendant le débogage.
L'exemple utilise les API QueryWorkingSet et fournit des fichiers de build minimaux ainsi que des captures d'écran pour illustrer une exécution normale versus altérée.
Il s'adresse aux reverse engineers et aux défenseurs qui étudient les contrôles légers d'intégrité mémoire.
