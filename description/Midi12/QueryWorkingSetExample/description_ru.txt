QueryWorkingSetExample — демонстрация на C техники защиты от подмены, основанной на метаданных рабочего набора Windows.
Она показывает, как проверка состояния разделяемых страниц в недоступных для записи областях вроде .text может выявить точки останова или изменения защиты, внесённые при отладке.
Пример использует API QueryWorkingSet и включает минимальные файлы сборки и скриншоты, иллюстрирующие нормальное и подменённое выполнение.
Предназначен для реверс-инженеров и защитников, изучающих лёгкие проверки целостности памяти.
