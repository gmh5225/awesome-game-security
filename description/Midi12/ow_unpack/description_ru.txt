ow_unpack — повторно загруженная кодовая база на C++ для распаковки и расшифровки защищённых игровых бинарников, связанных с исследованиями Overwatch.
Проект включает несколько модулей расшифровки и вспомогательных модулей, а также ассемблерные вспомогательные процедуры для низкоуровневых шагов распаковки.
Его структура отражает практический инструмент реверс-инжиниринга, а не отполированный продукт для конечного пользователя.
Основной сценарий использования — анализ бинарников и изучение распаковки в контексте игровой безопасности.
