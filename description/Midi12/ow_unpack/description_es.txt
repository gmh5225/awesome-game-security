ow_unpack es una base de código C++ republicada para desempaquetar y descifrar binarios de juegos protegidos asociados a la investigación de Overwatch.
El proyecto incluye varios módulos de descifrado y auxiliares, además de rutinas asistidas en ensamblador para pasos de desempaquetado de bajo nivel.
Su estructura refleja una herramienta práctica de ingeniería inversa más que un producto acabado para el usuario final.
El caso de uso principal es el análisis binario y el estudio del desempaquetado en contextos de seguridad de juegos.
