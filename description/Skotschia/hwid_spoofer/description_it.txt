Questo progetto è un HWID spoofer in C++ per Windows rivolto a numeri di serie del disco, identificatori SMART e valori legati all’SMBIOS. Fornisce moduli di supporto a basso livello e file di progetto Visual Studio per compilare e testare il concetto. L’autore lo descrive come una proof of concept datata, con ampio margine di miglioramento e di rafforzamento contro il rilevamento. Il caso d’uso principale è la ricerca educativa sulle tecniche di spoofing delle impronte hardware discusse nei contesti di evasione anti-cheat.
