Этот проект — HWID-спуфер на C++ для Windows, нацеленный на серийные номера дисков, идентификаторы SMART и значения, связанные с SMBIOS. Он предоставляет низкоуровневые вспомогательные модули и файлы проектов Visual Studio для сборки и проверки концепции. Автор описывает его как устаревший proof of concept с существенным запасом для доработки и усиления защиты от обнаружения. Основное применение — образовательное исследование техник подмены аппаратных отпечатков, обсуждаемых в контексте обхода античита.
