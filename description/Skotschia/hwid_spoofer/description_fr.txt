Ce projet est un HWID spoofer Windows en C++ ciblant les numéros de série de disque, les identifiants SMART et les valeurs liées au SMBIOS. Il fournit des modules d’aide bas niveau et des fichiers de projet Visual Studio pour compiler et tester le concept. L’auteur le décrit comme une ancienne preuve de concept, avec une marge importante d’amélioration et de durcissement contre la détection. Son principal cas d’usage est la recherche éducative sur les techniques de spoofing d’empreintes matérielles discutées dans les contextes d’évasion anti-cheat.
