Ce projet est un effort de recherche en cours axé sur le contournement de Windows PatchGuard sur les systèmes modernes. Il documente une approche dynamique planifiée avec trois objectifs principaux : désactiver l'exécution de PatchGuard, contourner les contrôles d'intégrité à l'exécution, et vérifier l'état de PatchGuard. L'implémentation est centrée sur les internals Windows de bas niveau et le comportement en mode noyau, avec du code C/C++ utilisé pour l'expérimentation. Il s'adresse principalement aux chercheurs en sécurité du noyau et aux analystes anti-cheat qui étudient les mécanismes défensifs et leur surface d'attaque.
