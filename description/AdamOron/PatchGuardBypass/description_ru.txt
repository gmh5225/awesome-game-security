Этот проект — ведущееся исследование, посвящённое обходу Windows PatchGuard на современных системах. В нём документируется запланированный динамический подход с тремя основными целями: отключение выполнения PatchGuard, обход проверок целостности во время выполнения и проверка состояния PatchGuard. Реализация сосредоточена на низкоуровневых внутренних механизмах Windows и поведении в режиме ядра; для экспериментов используется код на C/C++. Проект в первую очередь предназначен для исследователей безопасности ядра и аналитиков античита, изучающих защитные механизмы и их поверхность атаки.
