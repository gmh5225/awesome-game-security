Eine in C++ geschriebene interne CS:GO-Cheat-Basis, die das Spiel über MinHook hookt, ein ImGui-basiertes Menü-Framework bereitstellt und Engine-Prediction, Glow-ESP sowie Interface-Auflösung über Source-Engine-SDK-Muster enthält.
Sie implementiert DirectX-Surface-Rendering, Game-Event-Management sowie Client-/Engine-Interface-Wrapper für gängige Cheat-Funktionen.
Sie ist vor allem für Gamesicherheit-Forscher nützlich, die Cheat-Architekturen der Source Engine, Hook-Techniken und Anti-Cheat-Erkennungsflächen untersuchen.
