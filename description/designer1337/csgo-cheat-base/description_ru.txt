Внутренняя база чита для CS:GO на C++, которая перехватывает игру через MinHook, предоставляет фреймворк меню на ImGui и включает предсказание движка, glow ESP, а также разрешение интерфейсов по паттернам SDK движка Source.
Реализует рендеринг поверхностей DirectX, управление игровыми событиями и обёртки клиентских/движковых интерфейсов для типичного функционала чита.
В основном полезна исследователям игровой безопасности, изучающим архитектуры читов Source, техники хуков и поверхности детектирования античита.
