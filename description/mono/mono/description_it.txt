Implementazione open source del runtime .NET Framework (CLR), che include un compilatore JIT/AOT, un garbage collector (SGen), librerie di classi e un'API C incorporabile. Ampiamente usata come backend di scripting del motore di gioco Unity, fornisce esecuzione IL, interoperabilità P/Invoke e bridging managed-to-native, ed è un obiettivo primario per il modding di giochi e la ricerca sulla sicurezza.
