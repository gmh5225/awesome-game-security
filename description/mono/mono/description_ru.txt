Открытая реализация среды выполнения .NET Framework (CLR), включающая JIT/AOT-компилятор, сборщик мусора (SGen), библиотеки классов и встраиваемый C API. Широко используется как скриптовый бэкенд игрового движка Unity, обеспечивая выполнение IL, интероперабельность P/Invoke и мост между управляемым и нативным кодом — ключевую цель для моддинга игр и исследований в области безопасности.
