Implementación de código abierto del runtime de .NET Framework (CLR), que incluye un compilador JIT/AOT, un recolector de basura (SGen), bibliotecas de clases y una API C integrable. Ampliamente usada como backend de scripting del motor de juegos Unity, proporciona ejecución de IL, interoperabilidad P/Invoke y puente administrado-nativo, objetivo principal del modding de juegos y de la investigación en seguridad.
