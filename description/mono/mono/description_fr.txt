Implémentation open source du runtime .NET Framework (CLR), comprenant un compilateur JIT/AOT, un garbage collector (SGen), des bibliothèques de classes et une API C intégrable. Largement utilisée comme backend de scripting du moteur de jeu Unity, elle fournit l'exécution IL, l'interopérabilité P/Invoke et le pont managé-natif, cibles majeures pour le modding de jeux et la recherche en sécurité.
