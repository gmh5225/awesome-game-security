Dieses Projekt ist eine datenschutzfreundliche Open-Source-App, mit der Sie die von anderen Apps auf Ihrem Gerät hergestellten Verbindungen verfolgen, analysieren und blockieren können.
Sie können außerdem einen PCAP-Dump des Traffics exportieren, HTTP prüfen, TLS-Traffic entschlüsseln und vieles mehr!
Sie ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / Android Network Explorer untersuchen.
