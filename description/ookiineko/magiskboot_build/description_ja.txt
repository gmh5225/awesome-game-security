本プロジェクトは、Linux およびその他の POSIX システム上で Magisk の magiskboot ユーティリティをスタンドアロンツールとしてコンパイルするためのビルドスクリプトとパッチを提供します。Magiskboot は、Android ブートイメージの展開・再パック、ramdisk のパッチ、カーネル抽出、DTB 操作を扱います。ビルドシステムは Magisk のソースツリーから magiskboot を取り出し、独立コンパイルのための依存関係を解決します。フルの Magisk スイートをビルドせずに magiskboot を必要とする Android 開発者やセキュリティ研究者向けです。
