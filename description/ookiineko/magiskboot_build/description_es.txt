Este proyecto proporciona scripts de compilación y parches para compilar la utilidad magiskboot de Magisk como herramienta independiente en Linux y otros sistemas POSIX. Magiskboot gestiona el desempaquetado y reempaquetado de imágenes de arranque de Android, el parcheo del ramdisk, la extracción del kernel y la manipulación de DTB. El sistema de compilación extrae magiskboot del árbol de código fuente de Magisk y resuelve sus dependencias para una compilación independiente. Está dirigido a desarrolladores de Android e investigadores de seguridad que necesitan magiskboot sin compilar toda la suite Magisk.
