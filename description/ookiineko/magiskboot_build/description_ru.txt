Этот проект предоставляет скрипты сборки и патчи для компиляции утилиты magiskboot из Magisk как автономного инструмента в Linux и других POSIX-системах. Magiskboot выполняет распаковку и перепаковку образов загрузки Android, патчинг ramdisk, извлечение ядра и манипуляцию DTB. Система сборки извлекает magiskboot из исходного дерева Magisk и разрешает его зависимости для независимой компиляции. Предназначен для Android-разработчиков и исследователей безопасности, которым нужен magiskboot без сборки всего набора Magisk.
