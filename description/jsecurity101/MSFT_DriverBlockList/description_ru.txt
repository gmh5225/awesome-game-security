Этот проект предоставляет рекомендуемый список блокировки драйверов, который можно найти здесь: microsoft recommended driver block rules.
Он сосредоточен на разработке драйверов.
В основном полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / чёрных сигнатур.
