Python ベースの Ghidra バイナリ差分フレームワークで、2 つのバイナリに対するヘッドレス解析を自動化し、複数の戦略（構造グラフマッチング、バージョン追跡、BSim 類似度）で関数を対応付け、デコンパイル差分・コールグラフ変更・メタデータ差分を含む詳細な Markdown レポートを生成します。
Windows、macOS、Linux 上の PE、Mach-O、ELF 形式に対応し、CI パイプライン向けの Docker パッケージを備え、ntoskrnl、afd.sys の CVE、iOS dylib の差分に関するガイドを含む Docusaurus 製ドキュメントサイトを提供します。
主に脆弱性研究者、パッチ分析者、リバースエンジニアがバイナリ更新を比較し、セキュリティ修正や挙動の変化を特定するのに役立ちます。
