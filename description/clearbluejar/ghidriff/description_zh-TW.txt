基於 Python 的 Ghidra 二進位差異分析框架，可自動對兩個二進位檔案進行無介面（headless）分析，透過多種策略（結構圖匹配、版本追蹤、BSim 相似度）關聯函式，並產生包含反編譯差異、呼叫圖變更與中繼資料增量的詳細 Markdown 報告。
支援 Windows、macOS 與 Linux 上的 PE、Mach-O 與 ELF 格式，提供適用於 CI 管線的 Docker 封裝，以及以 Docusaurus 建置的文件網站，內含 ntoskrnl、afd.sys CVE 與 iOS dylib 的差異分析指南。
主要適用於弱點研究員、修補程式分析師與逆向工程師，用於比對二進位更新以識別安全性修復與行為變化。
