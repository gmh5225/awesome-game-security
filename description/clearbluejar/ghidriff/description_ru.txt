Фреймворк на Python для бинарного diffing в Ghidra: автоматизирует headless-анализ двух бинарников, сопоставляет функции несколькими стратегиями (структурное сопоставление графов, отслеживание версий, сходство BSim) и формирует подробные Markdown-отчёты с diff декомпиляции, изменениями графа вызовов и дельтами метаданных.
Поддерживает форматы PE, Mach-O и ELF в Windows, macOS и Linux, включает Docker-упаковку для CI-пайплайнов и сайт документации на Docusaurus с руководствами по diff ntoskrnl, CVE afd.sys и iOS dylib.
В первую очередь полезен исследователям уязвимостей, аналитикам патчей и реверс-инженерам, сравнивающим обновления бинарников для выявления исправлений безопасности и изменений поведения.
