Framework de diff binario para Ghidra basado en Python que automatiza el análisis headless de dos binarios, correlaciona funciones con varias estrategias (emparejamiento de grafos estructurales, seguimiento de versiones, similitud BSim) y genera informes Markdown detallados con diffs decompilados, cambios en el grafo de llamadas y deltas de metadatos.
Admite los formatos PE, Mach-O y ELF en Windows, macOS y Linux, incluye empaquetado Docker para pipelines de CI y ofrece un sitio de documentación con Docusaurus con guías para hacer diff de ntoskrnl, CVE de afd.sys y dylibs de iOS.
Resulta especialmente útil para investigadores de vulnerabilidades, analistas de parches e ingenieros inversos que comparan actualizaciones binarias para identificar correcciones de seguridad y cambios de comportamiento.
