Framework Python di binary diffing per Ghidra che automatizza l’analisi headless di due binari, correla le funzioni con più strategie (matching di grafi strutturali, version tracking, similarità BSim) e produce report Markdown dettagliati con diff decompilati, modifiche al call graph e delta dei metadati.
Supporta i formati PE, Mach-O ed ELF su Windows, macOS e Linux, include packaging Docker per pipeline CI e fornisce un sito di documentazione basato su Docusaurus con guide per il diff di ntoskrnl, CVE di afd.sys e dylib iOS.
È particolarmente utile per ricercatori di vulnerabilità, analisti di patch e reverse engineer che confrontano aggiornamenti binari per individuare fix di sicurezza e cambiamenti comportamentali.
