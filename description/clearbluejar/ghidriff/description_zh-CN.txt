基于 Python 的 Ghidra 二进制差异分析框架，可自动对两个二进制文件进行无界面（headless）分析，通过多种策略（结构图匹配、版本跟踪、BSim 相似度）关联函数，并生成包含反编译差异、调用图变更与元数据增量的详细 Markdown 报告。
支持 Windows、macOS 和 Linux 上的 PE、Mach-O 与 ELF 格式，提供面向 CI 流水线的 Docker 打包，以及基于 Docusaurus 的文档站点，包含 ntoskrnl、afd.sys CVE 与 iOS dylib 的差异分析指南。
主要面向漏洞研究员、补丁分析师与逆向工程师，用于对比二进制更新以识别安全修复与行为变化。
