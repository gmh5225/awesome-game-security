Python 기반 Ghidra 바이너리 디핑 프레임워크로, 두 바이너리에 대한 헤드리스 분석을 자동화하고 여러 전략(구조 그래프 매칭, 버전 추적, BSim 유사도)으로 함수를 대응시키며, 디컴파일 디프·콜 그래프 변경·메타데이터 델타가 담긴 상세 Markdown 보고서를 생성합니다.
Windows, macOS, Linux의 PE, Mach-O, ELF 형식을 지원하고 CI 파이프라인용 Docker 패키징을 포함하며, ntoskrnl·afd.sys CVE·iOS dylib 디핑 가이드가 있는 Docusaurus 기반 문서 사이트를 제공합니다.
주로 취약점 연구원, 패치 분석가, 리버스 엔지니어가 바이너리 업데이트를 비교해 보안 수정과 동작 변화를 파악하는 데 유용합니다.
