Windfall AntiCheat F is a server-side Fabric mod that detects Minecraft cheating on Fabric servers, ported one-to-one from the Spigot/Paper Windfall anti-cheat. Written in Java for Minecraft 1.21.5+ with Fabric Loader, Fabric API, Mixins, and Brigadier commands, it intercepts packets and runs dozens of combat, movement, packet, and inventory checks. It includes a vanilla-accurate physics prediction engine, latency compensation and simulation, adaptive thresholds, a severity and punishment system, Discord webhook alerts, and optional Prometheus metrics. Server operators use it to catch common hacks such as KillAura, reach, flight, speed, scaffold, and bad packets while aiming for low false positives on modern Fabric multiplayer servers.
