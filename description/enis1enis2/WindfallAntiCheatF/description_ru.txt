Windfall AntiCheat F — серверный мод Fabric, который обнаруживает читы Minecraft на серверах Fabric; портирован один к одному из античита Windfall для Spigot/Paper. Написан на Java для Minecraft 1.21.5+ с Fabric Loader, Fabric API, Mixins и командами Brigadier; перехватывает пакеты и выполняет десятки проверок боя, движения, пакетов и инвентаря. Включает движок предсказания физики, близкий к ванильному, компенсацию и симуляцию задержки, адаптивные пороги, систему серьёзности и наказаний, оповещения Discord через webhook и опциональные метрики Prometheus. Операторы серверов используют его, чтобы ловить распространённые хаки вроде KillAura, reach, полёта, speed, scaffold и некорректных пакетов, стремясь к низкому числу ложных срабатываний на современных многопользовательских серверах Fabric.
