Windfall Anti-Cheat 是面向企業、以封包為基礎的 Minecraft 伺服器反作弊外掛，可檢查玩家流量並標記作弊行為。它以 Java 編寫，作為 Maven Bukkit/Spigot 外掛，使用 PacketEvents 2 攔截封包，並執行數十項可設定檢查，涵蓋戰鬥、移動、封包濫用與物品欄。它包含延遲補償、移動預測與模擬、依玩家自適應門檻、Bedrock/Geyser 感知，以及其他外掛可用的公開 API，另有 Discord 警報與 Prometheus 指標。單一 JAR 可面向 Spigot、Paper、Folia 與 Purpur，涵蓋廣泛的 Minecraft 版本，適合需要伺服器端遊戲安全與作弊偵測的伺服器營運者。
