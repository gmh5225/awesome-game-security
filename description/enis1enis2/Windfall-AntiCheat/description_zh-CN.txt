Windfall Anti-Cheat 是面向企业的、基于数据包的 Minecraft 服务器反作弊插件，可检查玩家流量并标记作弊行为。它以 Java 编写，作为 Maven Bukkit/Spigot 插件，使用 PacketEvents 2 拦截数据包，并运行数十项可配置检测，覆盖战斗、移动、数据包滥用与背包。它包含延迟补偿、移动预测与模拟、按玩家自适应阈值、Bedrock/Geyser 感知，以及其他插件可用的公共 API，另有 Discord 警报与 Prometheus 指标。单个 JAR 可面向 Spigot、Paper、Folia 与 Purpur，覆盖广泛的 Minecraft 版本，适合需要服务端游戏安全与作弊检测的服务器运营者。
