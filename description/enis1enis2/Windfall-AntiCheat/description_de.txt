Windfall Anti-Cheat ist ein unternehmensorientiertes, paketbasiertes Anti-Cheat-Plugin für Minecraft-Server, das den Spielerverkehr prüft und Betrugsverhalten markiert. Als Maven-Bukkit/Spigot-Plugin in Java geschrieben, nutzt es PacketEvents 2 zum Abfangen von Paketen und führt Dutzende konfigurierbarer Checks für Kampf, Bewegung, Paketmissbrauch und Inventar aus. Es umfasst Lag-Kompensation, Bewegungsvorhersage und -simulation, adaptive Schwellenwerte pro Spieler, Bedrock/Geyser-Unterstützung sowie eine öffentliche API für andere Plugins, plus Discord-Alerts und Prometheus-Metriken. Eine einzelne JAR zielt auf Spigot, Paper, Folia und Purpur über einen weiten Minecraft-Versionsbereich und eignet sich für Serverbetreiber, die serverseitige Spielsicherheit und Cheat-Erkennung benötigen.
