Windfall Anti-Cheat — ориентированный на предприятия пакетный античит-плагин для серверов Minecraft, который анализирует трафик игроков и отмечает читерское поведение. Написан на Java как Maven-плагин Bukkit/Spigot, использует PacketEvents 2 для перехвата пакетов и выполняет десятки настраиваемых проверок по бою, движению, злоупотреблению пакетами и инвентарю. Включает компенсацию лагов, предсказание и симуляцию движения, адаптивные пороги для каждого игрока, поддержку Bedrock/Geyser и публичный API для других плагинов, а также оповещения Discord и метрики Prometheus. Один JAR рассчитан на Spigot, Paper, Folia и Purpur в широком диапазоне версий Minecraft и подходит операторам серверов, которым нужна серверная защита игры и обнаружение читов.
