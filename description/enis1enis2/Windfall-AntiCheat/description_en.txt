Windfall Anti-Cheat is an enterprise-oriented, packet-based anti-cheat plugin for Minecraft servers that inspects player traffic and flags cheating behavior. Written in Java as a Maven Bukkit/Spigot plugin, it uses PacketEvents 2 to intercept packets and runs dozens of configurable checks spanning combat, movement, packet abuse, and inventory. It includes lag compensation, movement prediction and simulation, adaptive per-player thresholds, Bedrock/Geyser awareness, and a public API for other plugins, plus Discord alerts and Prometheus metrics. A single JAR targets Spigot, Paper, Folia, and Purpur across a wide Minecraft version range, making it suited for server operators who need server-side game security and cheat detection.
