Ce projet est une base de code orientée tutoriel qui guide pas à pas la construction d'un hyperviseur x86 personnalisé. Il contient des exemples en C, C++ et assembleur répartis en parties progressives couvrant la configuration VMX, la gestion du VMCS, la traduction EPT et la virtualisation de systèmes en cours d'exécution. Le dépôt associe du code source pratique à du matériel pédagogique progressif pour expliquer les internals de la virtualisation matérielle moderne. Il s'adresse principalement aux programmeurs système bas niveau et aux chercheurs en sécurité qui apprennent les techniques d'analyse basées sur un hyperviseur.
