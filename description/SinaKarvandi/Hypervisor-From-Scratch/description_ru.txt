Этот проект — учебная кодовая база, которая шаг за шагом проводит через создание собственного x86-гипервизора. В ней есть примеры на C, C++ и ассемблере, разделённые на последовательные части: настройка VMX, управление VMCS, трансляция EPT и виртуализация работающих систем. Репозиторий сочетает практический исходный код с поэтапными учебными материалами, объясняющими внутреннее устройство современной аппаратной виртуализации. Он в первую очередь предназначен для системных программистов низкого уровня и исследователей безопасности, изучающих методы анализа на базе гипервизора.
