Este proyecto es una base de código orientada a tutoriales que guía paso a paso la construcción de un hipervisor x86 personalizado. Contiene ejemplos en C, C++ y ensamblador repartidos en partes progresivas que cubren la configuración de VMX, la gestión de VMCS, la traducción EPT y la virtualización de sistemas en ejecución. El repositorio combina código fuente práctico con material de aprendizaje por etapas para explicar los internos de la virtualización hardware moderna. Está pensado principalmente para programadores de sistemas de bajo nivel e investigadores de seguridad que aprenden técnicas de análisis basadas en hipervisores.
