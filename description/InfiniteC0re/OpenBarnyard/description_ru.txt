OpenBarnyard — это открытая реализация на C++ видеоигры Barnyard, построенная на реконструированном движке Toshi. Проект воссоздаёт ключевые игровые системы: рендеринг, анимацию, коллизии, GUI, аудио и симуляцию мира — с целевыми сборками под Windows для DirectX 8 и OpenGL через Premake. В комплекте есть инструменты отслеживания прогресса реверс-инжиниринга, сопоставляющие повторно реализованные методы с адресами исходного бинарника, а также SDK с хуками на базе Detours, загрузкой модов, отладкой через ImGui и примерами модов для улучшенной графики и спидрана. Основное применение — реверс-инжиниринг игр, реконструкция движка и исследование моддинга вокруг устаревших проектов на Toshi.
