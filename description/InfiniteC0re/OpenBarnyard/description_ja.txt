OpenBarnyard は、再構築された Toshi エンジン上に構築された、Barnyard ビデオゲームのオープンソース C++ 再実装です。レンダリング、アニメーション、コリジョン、GUI、オーディオ、ワールドシミュレーションなどのコアシステムを再現し、Premake 経由で DirectX 8 および OpenGL 向けの Windows ビルドターゲットを提供します。元のバイナリアドレスに対して再実装済みメソッドを追跡するリバースエンジニアリング進捗ツールに加え、Detours ベースのフック、MOD 読み込み、ImGui デバッグ、画質向上やスピードラン向けサンプル MOD を含む SDK も備えています。主な用途は、レガシーな Toshi ベースタイトルを対象としたゲーム逆向、エンジン再構築、MOD 研究です。
