Этот проект посвящён мод-меню для iOS.
Рекомендуются изображения 50x50; образец моих изображений можно получить, скопировав стандартную строку base64 (в tweak.xm) и просмотрев картинку на этом сайте.
Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / рендера / отрисовки.
