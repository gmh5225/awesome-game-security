Este proyecto se centra en menús de mods para iOS.
Se recomiendan imágenes de 50x50; puedes obtener una muestra de mis imágenes copiando la cadena base64 estándar (en tweak.xm) y usando este sitio web para mostrar la imagen.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheat / render / draw.
