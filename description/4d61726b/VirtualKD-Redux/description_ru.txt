VirtualKD-Redux — модернизированный набор инструментов для ускорения отладки ядра в виртуальных машинах для Windows и связанных платформ.
Он поддерживает актуальные версии VMware и VirtualBox, работает с ОС от устаревших выпусков Windows до Windows 11 и совместим с современными средствами WinDbg.
Кодовая база в основном на C и C++ и включает драйверные и хостовые компоненты, а также обновлённый процесс сборки на основе современных toolchain Visual Studio.
Основной сценарий использования — низкоуровневая отладка ядра и исследования системной безопасности в виртуализированных средах, где важна более быстрая итерация отладки.
