VirtualKD-Redux は、Windows および関連プラットフォーム向けの、仮想マシン上でのカーネルデバッグを高速化するモダンなツールキットです。
現行の VMware および VirtualBox に対応し、レガシーな Windows から Windows 11 まで幅広く動作し、最近の WinDbg ツールとの互換性も備えています。
コードベースは主に C と C++ で、ドライバおよびホスト側コンポーネントを含み、最新の Visual Studio ツールチェーンを中心とした更新されたビルドワークフローを採用しています。
主な用途は、仮想化環境における低レベルなカーネルデバッグとシステムセキュリティ研究であり、デバッグ反復を高速化したい場面で特に有用です。
