VirtualKD-Redux est une boîte à outils modernisée d’accélération du débogage noyau en machine virtuelle pour Windows et les plateformes associées.
Elle prend en charge les versions actuelles de VMware et VirtualBox, fonctionne des anciennes versions de Windows jusqu’à Windows 11, et reste compatible avec les outils WinDbg récents.
Le code est principalement en C et C++, avec des composants pilote et côté hôte, ainsi qu’un flux de build actualisé centré sur les toolchains Visual Studio modernes.
Son usage principal est le débogage noyau bas niveau et la recherche en sécurité des systèmes dans des environnements virtualisés, là où des itérations de debug plus rapides sont précieuses.
