Este proyecto es Apktool, una herramienta para la ingeniería inversa de archivos APK de Android. Decodifica los recursos a una forma casi original, incluidos layouts, cadenas y assets, y los reconstruye tras la modificación. Admite el desensamblado/reensamblado smali del bytecode DEX, la decodificación de tablas de recursos, el análisis de AndroidManifest.xml y el reempaquetado de APK con la compilación correcta de recursos. La herramienta en Java permite modificar APK sin acceso al código fuente original. Está dirigida a ingenieros inversos, modders e investigadores de seguridad que necesitan inspeccionar, modificar y reempaquetar aplicaciones Android.
