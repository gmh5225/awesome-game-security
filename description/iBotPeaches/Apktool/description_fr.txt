Ce projet est Apktool, un outil de rétro-ingénierie des fichiers APK Android. Il décode les ressources sous une forme quasi originale, y compris les layouts, les chaînes et les assets, puis les reconstruit après modification. Il prend en charge le désassemblage/réassemblage smali du bytecode DEX, le décodage des tables de ressources, l'analyse d'AndroidManifest.xml et le reconditionnement d'APK avec une compilation correcte des ressources. Cet outil Java permet de modifier un APK sans disposer du code source d'origine. Il s'adresse aux ingénieurs inverses, moddeurs et chercheurs en sécurité qui doivent inspecter, modifier et reconditionner des applications Android.
