이 프로젝트는 Android APK 파일을 리버스 엔지니어링하기 위한 도구 Apktool입니다. 레이아웃, 문자열, 에셋을 포함한 리소스를 거의 원본에 가깝게 디코딩하고, 수정 후 다시 빌드할 수 있습니다. DEX 바이트코드의 smali 디스어셈블리/리어셈블리, 리소스 테이블 디코딩, AndroidManifest.xml 파싱, 올바른 리소스 컴파일을 통한 APK 재패키징을 지원합니다. 이 Java 도구는 원본 소스 코드 없이도 APK를 수정할 수 있게 합니다. Android 앱을 검사·수정·재패키징해야 하는 리버스 엔지니어, 모더, 보안 연구자를 대상으로 합니다.
