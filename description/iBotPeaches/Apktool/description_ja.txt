本プロジェクトは Apktool で、Android APK ファイルをリバースエンジニアリングするためのツールです。レイアウト、文字列、アセットを含むリソースをほぼ元の形にデコードし、改変後に再構築できます。DEX バイトコードの smali 逆アセンブル／再アセンブル、リソーステーブルのデコード、AndroidManifest.xml の解析、適切なリソースコンパイルによる APK 再パッケージをサポートします。この Java ツールにより、元のソースコードがなくても APK を改変できます。Android アプリの調査・改変・再パッケージが必要なリバースエンジニア、モッダー、セキュリティ研究者向けです。
