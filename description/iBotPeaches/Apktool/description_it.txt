Questo progetto è Apktool, uno strumento per il reverse engineering dei file APK Android. Decodifica le risorse in una forma quasi originale, inclusi layout, stringhe e asset, e le ricostruisce dopo la modifica. Supporta il disassemblaggio/riassemblaggio smali del bytecode DEX, la decodifica delle tabelle delle risorse, il parsing di AndroidManifest.xml e il ripacchettamento degli APK con la corretta compilazione delle risorse. Lo strumento Java consente di modificare gli APK senza accedere al codice sorgente originale. È rivolto a reverse engineer, modder e ricercatori di sicurezza che devono ispezionare, modificare e ripacchettare applicazioni Android.
