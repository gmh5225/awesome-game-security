Questo progetto è RisohEditor, un editor di risorse Windows gratuito per file eseguibili PE e DLL. Offre un'interfaccia grafica per visualizzare, modificare, aggiungere e rimuovere risorse tra cui finestre di dialogo, menu, icone, bitmap, tabelle di stringhe, informazioni di versione e file manifest. L'applicazione C++ supporta l'importazione/esportazione di script di risorse (RC) e gestisce file PE a 32 e 64 bit. È rivolta a sviluppatori Windows, reverse engineer e traduttori che devono modificare le risorse incorporate negli eseguibili compilati.
