Этот проект — RisohEditor, бесплатный редактор ресурсов Windows для PE-исполняемых файлов и DLL. Он предоставляет графический интерфейс для просмотра, редактирования, добавления и удаления ресурсов, включая диалоги, меню, значки, растровые изображения, таблицы строк, сведения о версии и файлы манифеста. Приложение на C++ поддерживает импорт/экспорт скриптов ресурсов (RC) и работает с 32-битными и 64-битными PE-файлами. Оно предназначено для разработчиков Windows, специалистов по реверс-инжинирингу и переводчиков, которым нужно изменять встроенные ресурсы в скомпилированных исполняемых файлах.
