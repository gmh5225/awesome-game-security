Este proyecto es RisohEditor, un editor de recursos de Windows gratuito para archivos ejecutables PE y DLL. Ofrece una interfaz gráfica para ver, editar, añadir y eliminar recursos como diálogos, menús, iconos, bitmaps, tablas de cadenas, información de versión y archivos de manifiesto. La aplicación en C++ admite importación/exportación de scripts de recursos (RC) y gestiona archivos PE de 32 y 64 bits. Está dirigida a desarrolladores de Windows, ingenieros inversos y traductores que necesitan modificar recursos incrustados en ejecutables compilados.
