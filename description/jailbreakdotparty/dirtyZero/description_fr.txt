Ce projet est dirtyZero, un exploit iOS ciblant une vulnérabilité spécifique du noyau afin d'obtenir une lecture/écriture arbitraire du noyau sur les versions modernes d'iOS. Il s'appuie sur une vulnérabilité zero-day ou récemment corrigée du noyau XNU pour construire une primitive d'exploitation fiable, pouvant être enchaînée avec d'autres composants pour le jailbreak ou la recherche en sécurité. Il s'adresse aux chercheurs en sécurité iOS et aux développeurs de jailbreak qui étudient l'exploitation du noyau XNU sur les versions récentes d'iOS.
