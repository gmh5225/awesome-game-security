Этот проект — dirtyZero, эксплойт для iOS, нацеленный на конкретную уязвимость ядра для достижения произвольного чтения/записи в ядре на современных версиях iOS. Он использует уязвимость нулевого дня или недавно закрытую уязвимость в ядре XNU для построения надёжного примитива эксплуатации, который можно объединять с другими компонентами для джейлбрейка или исследований безопасности. Предназначен для исследователей безопасности iOS и разработчиков джейлбрейков, изучающих эксплуатацию ядра XNU на недавних выпусках iOS.
