Este proyecto es dirtyZero, un exploit de iOS que apunta a una vulnerabilidad específica del kernel para lograr lectura/escritura arbitraria del kernel en versiones modernas de iOS. Aprovecha una vulnerabilidad de día cero o recientemente parcheada en el kernel XNU para construir una primitiva de explotación fiable, que puede encadenarse con otros componentes para jailbreak o investigación de seguridad. Está dirigido a investigadores de seguridad de iOS y desarrolladores de jailbreak que estudian la explotación del kernel XNU en versiones recientes de iOS.
