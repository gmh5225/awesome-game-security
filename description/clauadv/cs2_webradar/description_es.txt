Este proyecto se centra en un cheat de radar basado en navegador para Counter-Strike 2, orientado a pasar desapercibido.
Está escrito principalmente en C++ y JavaScript y se centra en pipelines de assets, modding y Unity.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el área cheat / game:cs2.
