Этот проект посвящён необнаруживаемому браузерному радар-читу для Counter-Strike 2.
Он в основном написан на C++ и JavaScript и сосредоточен на пайплайнах ассетов, моддинге и Unity.
В первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:cs2.
