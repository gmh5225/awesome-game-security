Questo progetto si concentra su un cheat radar basato su browser per Counter-Strike 2, pensato per restare non rilevato.
È scritto principalmente in C++ e JavaScript e ruota attorno a pipeline di asset, modding e Unity.
È soprattutto utile a ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'area cheat / game:cs2.
