BEClient es un pequeño cliente de prueba de concepto en C++ que demuestra cómo inicializar y llamar a una interfaz DLL del cliente BattlEye.
Define estructuras de datos del juego y del anti-trampas, registra funciones de devolución de llamada e invoca rutinas exportadas como los manejadores run, command y exit.
El ejemplo está configurado como un proyecto de Visual Studio para Windows e incluye encabezados de estructuras para los campos de comunicación del cliente.
Su caso de uso principal es la ingeniería inversa y la investigación del comportamiento de integración del cliente anti-trampas en procesos de juego.
