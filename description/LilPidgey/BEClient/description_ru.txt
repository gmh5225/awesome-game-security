BEClient — небольшой C++ proof-of-concept клиент, демонстрирующий, как инициализировать и вызывать интерфейс DLL клиента BattlEye.
Он определяет структуры данных игры и античита, регистрирует функции обратного вызова и вызывает экспортируемые процедуры, такие как обработчики run, command и exit.
Пример оформлен как проект Visual Studio для Windows и включает заголовки структур для полей клиентской коммуникации.
Основной сценарий использования — реверс-инжиниринг и исследование поведения интеграции античит-клиента в игровых процессах.
