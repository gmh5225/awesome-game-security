Anbox는 무거운 가상화에 의존하지 않고 Linux에서 완전한 Android 시스템을 실행하는 컨테이너 기반 런타임입니다. Linux namespaces와 호스트 측 데몬을 사용해 하드웨어 접근을 중개하며, Android 에뮬레이터 구성 요소에서 적응한 OpenGL ES 렌더링 경로를 포함합니다. 코드베이스는 주로 C++와 CMake로 구성되며 LXC, D-Bus, protobuf 같은 종속성을 통합합니다. 데스크톱 및 클라우드 형태의 Android 애플리케이션 사용 사례를 목표로 하며, 유지보수가 아카이브된 상태여도 참고 프로젝트로서 가치가 있습니다.
