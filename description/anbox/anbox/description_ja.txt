Anbox は、重量級の仮想化に頼らず Linux 上で完全な Android システムを動作させるコンテナベースのランタイムです。Linux namespaces とホスト側デーモンを用い、Android エミュレータ構成要素から適応した OpenGL ES レンダリング経路を含むハードウェアアクセスを仲介します。コードベースは主に C++ で CMake を使い、LXC、D-Bus、protobuf などの依存関係を統合しています。デスクトップおよびクラウド型の Android アプリケーション用途を想定しており、メンテナンスがアーカイブ済みであっても参考プロジェクトとして価値があります。
