Anbox — контейнерная среда выполнения, которая запускает полноценную систему Android на Linux без опоры на тяжёлую виртуализацию. Она использует Linux namespaces и демон на стороне хоста для посредничества в доступе к оборудованию, включая пути рендеринга OpenGL ES, адаптированные из компонентов эмулятора Android. Кодовая база в основном на C++ с CMake и интегрирует зависимости вроде LXC, D-Bus и protobuf. Проект ориентирован на настольные и облачные сценарии Android-приложений и остаётся ценным как справочный пример, несмотря на архивный статус сопровождения.
