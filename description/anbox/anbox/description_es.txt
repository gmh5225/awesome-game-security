Anbox es un runtime basado en contenedores que ejecuta un sistema Android completo en Linux sin depender de virtualización pesada. Usa namespaces de Linux y un demonio en el host para mediar el acceso al hardware, incluidos caminos de renderizado OpenGL ES adaptados a partir de componentes del emulador de Android. La base de código es principalmente C++ con CMake e integra dependencias como LXC, D-Bus y protobuf. Está orientado a casos de uso de aplicaciones Android de escritorio y en la nube, y sigue siendo valioso como proyecto de referencia pese a su estado de mantenimiento archivado.
