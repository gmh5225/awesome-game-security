Este proyecto es un extractor de payload OTA de Android de alto rendimiento escrito en Go que extrae imágenes de partición de archivos payload.bin.
Incluye descompresión paralelizada para una extracción rápida, verificación de checksums del payload y soporte para procesar directamente paquetes zip originales que contienen payload.bin.
El código en Go se encarga del análisis de payloads OTA, la descompresión de datos de partición y E/S multihilo, con xz como dependencia externa para un rendimiento óptimo.
Es especialmente útil para investigadores de seguridad de Android y desarrolladores de ROM que necesitan extraer y analizar imágenes de partición de paquetes de actualización OTA de Android.
