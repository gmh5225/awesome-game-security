이 프로젝트는 Go로 작성된 고성능 Android OTA payload 덤퍼로, payload.bin 파일에서 파티션 이미지를 추출합니다.
빠른 추출을 위한 병렬 압축 해제, payload 체크섬 검증, 그리고 payload.bin이 포함된 원본 zip 패키지를 직접 처리하는 기능을 제공합니다.
Go 코드베이스는 OTA payload 파싱, 파티션 데이터 압축 해제, 멀티스레드 I/O를 처리하며, 최적의 성능을 위해 xz를 외부 의존성으로 사용합니다.
Android OTA 업데이트 패키지에서 파티션 이미지를 추출하고 분석해야 하는 Android 보안 연구원과 ROM 개발자에게 주로 유용합니다.
