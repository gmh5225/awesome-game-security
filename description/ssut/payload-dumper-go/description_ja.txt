本プロジェクトは、Go で書かれた高性能な Android OTA payload ダンパーで、payload.bin ファイルからパーティションイメージを抽出します。
並列化された展開による高速抽出、payload のチェックサム検証、および payload.bin を含む元の zip パッケージを直接処理する機能を備えています。
Go のコードベースは OTA payload の解析、パーティションデータの展開、マルチスレッド I/O を担当し、最適な性能のために xz を外部依存として使用します。
Android OTA アップデートパッケージからパーティションイメージを抽出・分析する必要がある Android セキュリティ研究者や ROM 開発者に主に役立ちます。
