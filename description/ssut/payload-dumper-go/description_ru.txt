Этот проект — высокопроизводительный дампер Android OTA payload на Go, который извлекает образы разделов из файлов payload.bin.
Он обеспечивает параллелизованную распаковку для быстрого извлечения, проверку контрольных сумм payload и поддержку прямой обработки исходных zip-пакетов, содержащих payload.bin.
Кодовая база на Go отвечает за разбор OTA payload, распаковку данных разделов и многопоточный ввод-вывод, используя xz в качестве внешней зависимости для оптимальной производительности.
Он в первую очередь полезен исследователям безопасности Android и разработчикам ROM, которым нужно извлекать и анализировать образы разделов из пакетов обновлений Android OTA.
