Dieses Projekt konzentriert sich auf einen Rust Demangler & Normalizer.
Der in diesem Skript verwendete Demangling-Algorithmus basiert auf dem rs-dml-Binary; stellen Sie sicher, dass es bereits installiert ist.
Es ist vor allem für Gamesicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat- / IDA-Plugins untersuchen.
