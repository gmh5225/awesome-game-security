Este proyecto se centra en un Demangler y Normalizer para Rust.
El algoritmo de demangling usado en este script se basa en el binario rs-dml; asegúrese de tenerlo ya instalado.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
