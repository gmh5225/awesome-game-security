OrangeFox Recovery Project (OFRP) Device Tree für das Xiaomi Redmi K60 Pro (Codename mondrian), das die Recovery-Partitionskonfiguration, TWRP-Flags, einen vorgefertigten Kernel mit Goodix-Fingerprint-Kernelmodul sowie die Vendor-HAL-Binärdateien (gatekeeper, keymaster, QSEECOM) bereitstellt, die zum Starten einer benutzerdefinierten Recovery-Umgebung auf Qualcomm-Snapdragon-8-Gen-2-Hardware erforderlich sind.
