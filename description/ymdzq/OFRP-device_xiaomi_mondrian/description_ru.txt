Дерево устройства OrangeFox Recovery Project (OFRP) для Xiaomi Redmi K60 Pro (кодовое имя mondrian), предоставляющее конфигурацию разделов recovery, флаги TWRP, предварительно собранное ядро с модулем ядра отпечатков пальцев Goodix, а также бинарные файлы vendor HAL (gatekeeper, keymaster, QSEECOM), необходимые для загрузки пользовательской среды recovery на оборудовании Qualcomm Snapdragon 8 Gen 2.
