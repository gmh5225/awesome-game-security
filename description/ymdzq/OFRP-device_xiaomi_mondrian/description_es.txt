Árbol de dispositivo de OrangeFox Recovery Project (OFRP) para el Xiaomi Redmi K60 Pro (nombre en código mondrian), que proporciona la configuración de particiones de recovery, banderas TWRP, un kernel precompilado con el módulo de kernel de huellas dactilares Goodix, y binarios HAL del fabricante (gatekeeper, keymaster, QSEECOM) necesarios para arrancar un entorno de recovery personalizado en hardware Qualcomm Snapdragon 8 Gen 2.
