Device tree di OrangeFox Recovery Project (OFRP) per lo Xiaomi Redmi K60 Pro (codename mondrian), che fornisce la configurazione delle partizioni di recovery, i flag TWRP, un kernel precompilato con il modulo kernel per impronte digitali Goodix e i binari HAL del vendor (gatekeeper, keymaster, QSEECOM) necessari per avviare un ambiente di recovery personalizzato su hardware Qualcomm Snapdragon 8 Gen 2.
