Arbre de périphérique OrangeFox Recovery Project (OFRP) pour le Xiaomi Redmi K60 Pro (nom de code mondrian), fournissant la configuration des partitions de recovery, les indicateurs TWRP, un noyau précompilé avec le module noyau d'empreinte digitale Goodix, ainsi que les binaires HAL du fabricant (gatekeeper, keymaster, QSEECOM) nécessaires pour démarrer un environnement de recovery personnalisé sur le matériel Qualcomm Snapdragon 8 Gen 2.
