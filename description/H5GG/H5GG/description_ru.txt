H5GG — это движок моддинга для iOS, сочетающий API скриптинга на JavaScript с пользовательским интерфейсом на базе HTML5. Он предоставляет функции редактирования памяти, загрузку скриптов из локальных или удалённых источников, поддержку динамических плагинов и утилиты автоматического поиска статических цепочек указателей. В проект также входят несколько режимов выполнения для jailbreak- и не-jailbreak-окружений, а также плагин h5frida для хукинга и патчинга функций Objective-C и C/C++. В основном используется для моддинга мобильных игр, инструментации во время выполнения и исследований безопасности iOS-игр.
