H5GG è un motore di modding per iOS che combina API di scripting JavaScript con un'interfaccia utente basata su HTML5. Offre funzioni di editing della memoria, caricamento di script da fonti locali o remote, supporto per plugin dinamici e utilità per la ricerca automatica di catene di puntatori statici. Il progetto include anche più modalità di runtime per ambienti jailbroken e non jailbroken, nonché un plugin h5frida per l'hooking e il patching di funzioni Objective-C e C/C++. È usato principalmente per il modding di giochi mobili, l'instrumentazione a runtime e la ricerca sulla sicurezza dei giochi iOS.
