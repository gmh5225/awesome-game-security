H5GG est un moteur de modding iOS qui combine des API de scripting JavaScript avec une interface utilisateur basée sur HTML5. Il propose des fonctions d'édition mémoire, le chargement de scripts locaux ou distants, le support de plugins dynamiques, ainsi que des utilitaires de recherche automatique de chaînes de pointeurs statiques. Le projet inclut également plusieurs modes d'exécution pour les environnements jailbreakés et non jailbreakés, ainsi qu'un plugin h5frida pour le hooking et le patching de fonctions Objective-C et C/C++. Il est principalement utilisé pour le modding de jeux mobiles, l'instrumentation à l'exécution et la recherche en sécurité des jeux iOS.
