H5GG es un motor de modding para iOS que combina APIs de scripting en JavaScript con una interfaz de usuario basada en HTML5. Ofrece funciones de edición de memoria, carga de scripts desde fuentes locales o remotas, soporte de plugins dinámicos y utilidades de búsqueda automática de cadenas de punteros estáticos. El proyecto también incluye múltiples modos de ejecución para entornos con y sin jailbreak, así como un plugin h5frida para el hooking y el parcheo de funciones Objective-C y C/C++. Se usa principalmente para el modding de juegos móviles, la instrumentación en tiempo de ejecución y la investigación de seguridad en juegos iOS.
