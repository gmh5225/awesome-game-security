Dieses Projekt konzentriert sich auf obfusk8: eine auf C++17 basierende Obfuskationsbibliothek.
Sie erreicht dies durch eine vielfältige Reihe von Compile-Time- und Runtime-Techniken zum Schutz der Logik und Daten Ihres Codes.
Sie ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat / Compile-Time arbeiten.
