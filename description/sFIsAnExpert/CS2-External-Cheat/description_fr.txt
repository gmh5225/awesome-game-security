Ce projet est un cheat externe pour Counter-Strike 2 avec une interface overlay ImGui basée sur D3D11. La base de code C++ inclut un aimbot, l'interaction avec le moteur, des utilitaires mathématiques, des modules de lecture mémoire, la gestion des offsets, ainsi qu'un rendu overlay DirectX 11 pour afficher l'interface du cheat et les visuels ESP. Il est surtout utile aux chercheurs en sécurité des jeux qui étudient l'architecture des cheats externes, le rendu overlay D3D11 et les techniques de lecture mémoire de CS2.
