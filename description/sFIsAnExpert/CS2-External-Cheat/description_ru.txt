Этот проект — внешний чит для Counter-Strike 2 с интерфейсом оверлея ImGui на базе D3D11. Кодовая база на C++ включает аимбот, взаимодействие с движком, математические утилиты, модули чтения памяти, управление оффсетами и оверлей-рендерер на DirectX 11 для отображения UI чита и ESP-визуализаций. В первую очередь полезен исследователям игровой безопасности, изучающим архитектуру внешних читов, рендеринг оверлеев D3D11 и техники чтения памяти CS2.
