Этот проект — плагин для IDA Pro, предназначенный для обнаружения распространённых артефактов античита.
⚠️ Плагин предназначен только для реверс-инжиниринга и академических исследований.
Он в первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / плагинов IDA.
