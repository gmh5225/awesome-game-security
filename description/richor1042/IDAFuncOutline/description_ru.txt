Этот проект — плагин для IDA Pro, который создаёт контуры функций и визуализации иерархии вызовов. Он строит древовидные представления связей вызовов функций, показывая вызывающих и вызываемых с управлением глубиной, помогая аналитикам быстро понять структуру кода и выявить важные функции. Предназначен для специалистов по реверс-инжинирингу, использующих IDA Pro и нуждающихся в визуальном изучении иерархии функций.
