Questo progetto è una raccolta di strumenti di threat research, script e plugin Ghidra/IDA di Security Joes. Include script di analisi per famiglie di malware specifiche, estensioni Ghidra per la deoffuscazione automatizzata, regole YARA e utility di reverse engineering usate nelle attività di incident response. Gli strumenti Python e Java accompagnano report di threat research già pubblicati. È rivolto ad analisti malware e threat researcher che vogliono estendere i propri toolkit di RE con script di analisi specializzati.
