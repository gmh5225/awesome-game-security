Este proyecto es una colección de herramientas de investigación de amenazas, scripts y plugins de Ghidra/IDA de Security Joes. Incluye scripts de análisis para familias de malware específicas, extensiones de Ghidra para desofuscación automatizada, reglas YARA y utilidades de ingeniería inversa usadas en intervenciones de respuesta a incidentes. Las herramientas en Python y Java acompañan informes de investigación de amenazas publicados. Está dirigido a analistas de malware e investigadores de amenazas que quieran ampliar sus kits de RE con scripts de análisis especializados.
