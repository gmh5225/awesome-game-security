Этот проект — коллекция инструментов для исследования угроз, скриптов и плагинов Ghidra/IDA от Security Joes. В него входят скрипты анализа для отдельных семейств вредоносного ПО, расширения Ghidra для автоматизированной деобфускации, правила YARA и утилиты для реверс-инжиниринга, применяемые при реагировании на инциденты. Инструменты на Python и Java сопровождают опубликованные отчёты по исследованию угроз. Предназначен для аналитиков вредоносного ПО и исследователей угроз, расширяющих свои RE-наборы специализированными скриптами анализа.
