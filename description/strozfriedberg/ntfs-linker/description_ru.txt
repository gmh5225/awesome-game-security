Этот проект — инструмент цифровой криминалистики на C++ для разбора и корреляции метаданных файловой системы NTFS, включая записи $MFT, $UsnJrnl и $LogFile. Он связывает записи журнала изменений с записями MFT, чтобы восстанавливать временные шкалы активности файловой системы, разрешать полные пути к файлам и отслеживать события создания, изменения, переименования и удаления. Предназначен для специалистов по цифровой криминалистике, исследующих активность файловой системы на томах Windows NTFS.
