Questo progetto è uno strumento forense in C++ per analizzare e correlare i metadati del file system NTFS, inclusi i record $MFT, $UsnJrnl e $LogFile. Collega le voci del change journal ai record MFT per ricostruire le timeline dell'attività del file system, risolvere i percorsi completi dei file e tracciare eventi di creazione, modifica, ridenominazione ed eliminazione. È rivolto agli analisti di digital forensics che indagano l'attività del file system su volumi NTFS di Windows.
