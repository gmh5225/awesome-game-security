Внешний чит для Unreal Engine 4, нацеленный на Valorant: состоит из драйвера в режиме ядра (ручное отображение через EFI), общающегося через хуки IOCTL dispatch, и пользовательского клиента, читающего память игры через интерфейс драйвера. Клиент рисует оверлей ImGui в окне DirectX 9, использует снимки ToolHelp32 для перечисления процессов и драйвер для межпроцессного чтения структур актёров и игроков UE4.
