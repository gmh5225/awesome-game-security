Questo progetto è un'estensione assistita dall'IA che collega le sessioni di debug di WinDbg a un copilota in stile ChatGPT. È implementato come pacchetto Python che legge l'output dei comandi del debugger e restituisce indicazioni, spiegazioni o suggerimenti sul comando successivo. Lo strumento supporta i backend OpenAI e Azure OpenAI tramite configurazione basata sull'ambiente. È pensato soprattutto per l'analisi dei crash, il debug del kernel e i flussi di reverse engineering in cui un triage più rapido è utile.
