Этот проект посвящён локальному кокпиту для рабочих процессов Codex imagegen — пиксель-арт, редактирование по областям, кадры анимации и спрайт-листы через inbox/outbox codex-handoff; без прямых вызовов OpenAI API.
Это локальное рабочее пространство для генерации пиксель-арта с последующим превращением выбранных ассетов в кадры анимации и спрайт-листы.
В основном полезно разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в сфере game develop / AI agents.
