Ce projet se concentre sur un cockpit local pour les workflows Codex imagegen — pixel art, édition par régions, frames d'animation et sprite sheets via la boîte de réception/d'envoi codex-handoff ; sans appels directs à l'API OpenAI.
C'est un espace de travail local pour générer du pixel art, puis transformer les assets sélectionnés en frames d'animation et en sprite sheets.
Il est surtout utile aux développeurs de jeux, reverse engineers et créateurs d'outils travaillant dans le domaine du développement de jeux / des agents IA.
