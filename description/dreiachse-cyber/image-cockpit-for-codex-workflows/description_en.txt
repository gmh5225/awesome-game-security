This project focuses on local cockpit for Codex imagegen workflows — pixel art, region-based editing, animation frames and sprite sheets via codex-handoff inbox/outbox; no direct OpenAI API calls.
It is a local workspace for generating pixel art, then turning selected pixel-art assets into animation frames and sprite sheets.
It is mainly useful for game developers, reverse engineers, and tooling builders working in the game develop / ai agents area.
