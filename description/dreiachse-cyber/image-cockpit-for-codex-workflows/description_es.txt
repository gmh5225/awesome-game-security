Este proyecto se centra en un cockpit local para flujos de trabajo de Codex imagegen: pixel art, edición por regiones, fotogramas de animación y sprite sheets mediante la bandeja de entrada/salida de codex-handoff; sin llamadas directas a la API de OpenAI.
Es un espacio de trabajo local para generar pixel art y luego convertir los assets seleccionados en fotogramas de animación y sprite sheets.
Resulta especialmente útil para desarrolladores de juegos, ingenieros inversos y creadores de herramientas en el área de desarrollo de juegos / agentes de IA.
