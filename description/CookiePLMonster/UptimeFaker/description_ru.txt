Этот проект — библиотека хуков тайминга для Windows, которая подменяет значения системного uptime для тестирования и задач совместимости.
Она написана на C++ как плагин на базе Detours и может внедряться как DLL или модуль в стиле ASI.
Инструмент может перенаправлять несколько API, связанных с таймерами, и применяет настраиваемое поведение через INI-файл, включая симуляцию времени относительно процесса.
В основном используется для диагностики или смягчения ошибок игр и приложений, проявляющихся на системах с большим временем непрерывной работы.
