Este proyecto es una biblioteca de hooks de temporización para Windows que falsifica los valores de uptime del sistema para pruebas y trabajo de compatibilidad.
Está escrita en C++ como un complemento basado en Detours y puede inyectarse como DLL o módulo de estilo ASI.
La herramienta puede redirigir varias API relacionadas con temporizadores y aplica un comportamiento configurable mediante un archivo INI, incluida la simulación de tiempo relativa al proceso.
Se usa principalmente para diagnosticar o mitigar errores de juegos y aplicaciones que aparecen en sistemas con alto uptime.
