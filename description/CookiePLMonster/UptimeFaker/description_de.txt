Dieses Projekt ist eine Windows-Timing-Hook-Bibliothek, die System-Uptime-Werte für Tests und Kompatibilitätsarbeit vortäuscht.
Es ist in C++ als Detours-basiertes Plugin geschrieben und kann als DLL oder ASI-ähnliches Modul injiziert werden.
Das Tool kann mehrere timerbezogene APIs umleiten und wendet über eine INI-Datei konfigurierbares Verhalten an, einschließlich prozessrelativer Zeitsimulation.
Es wird vor allem genutzt, um Bugs in Spielen und Anwendungen zu diagnostizieren oder abzuschwächen, die auf Systemen mit hoher Uptime auftreten.
