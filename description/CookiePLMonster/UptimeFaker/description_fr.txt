Ce projet est une bibliothèque de hooks de timing Windows qui simule les valeurs d’uptime système pour les tests et le travail de compatibilité.
Écrit en C++ sous forme de plugin basé sur Detours, il peut être injecté en tant que DLL ou module de type ASI.
L’outil peut rediriger plusieurs API liées aux temporisateurs et appliquer un comportement configurable via un fichier INI, y compris une simulation de temps relative au processus.
Il sert principalement à diagnostiquer ou atténuer les bugs de jeux et d’applications qui apparaissent sur des systèmes à fort uptime.
