Questo progetto è una libreria di hook di timing per Windows che simula i valori di uptime del sistema per test e lavoro di compatibilità.
È scritta in C++ come plugin basato su Detours e può essere iniettata come DLL o modulo in stile ASI.
Lo strumento può reindirizzare più API legate ai timer e applica un comportamento configurabile tramite file INI, inclusa la simulazione del tempo relativa al processo.
È usato principalmente per diagnosticare o mitigare bug di giochi e applicazioni che compaiono su sistemi con uptime elevato.
