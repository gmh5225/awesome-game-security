Questo progetto è un resolver runtime IL2CPP esterno per giochi Unity che legge i metadati IL2CPP dall'esterno del processo di gioco. Analizza il file global-metadata.dat e le strutture binarie IL2CPP tramite lettura della memoria esterna per risolvere nomi delle classi, indirizzi dei metodi, offset dei campi e informazioni sui tipi senza iniezione. La libreria C++ consente lo sviluppo di cheat esterni per giochi Unity IL2CPP. È rivolta ai game hacker che costruiscono cheat esterni per titoli Unity basati su IL2CPP.
