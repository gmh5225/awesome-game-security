Este proyecto es un resolver de tiempo de ejecución IL2CPP externo para juegos Unity que lee metadatos IL2CPP desde fuera del proceso del juego. Analiza el archivo global-metadata.dat y las estructuras binarias de IL2CPP mediante lectura de memoria externa para resolver nombres de clases, direcciones de métodos, offsets de campos e información de tipos sin inyección. La biblioteca C++ permite el desarrollo de cheats externos para juegos Unity IL2CPP. Está dirigida a game hackers que construyen cheats externos para títulos Unity basados en IL2CPP.
