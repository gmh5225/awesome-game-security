Ce projet est un résolveur d'exécution IL2CPP externe pour les jeux Unity qui lit les métadonnées IL2CPP depuis l'extérieur du processus du jeu. Il analyse le fichier global-metadata.dat et les structures binaires IL2CPP via une lecture mémoire externe afin de résoudre les noms de classes, adresses de méthodes, offsets de champs et informations de types sans injection. La bibliothèque C++ permet le développement de cheats externes pour les jeux Unity IL2CPP. Elle s'adresse aux game hackers qui construisent des cheats externes pour des titres Unity basés sur IL2CPP.
