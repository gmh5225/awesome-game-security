Этот проект — внешний IL2CPP runtime-резолвер для игр на Unity, который читает метаданные IL2CPP извне процесса игры. Он разбирает файл global-metadata.dat и бинарные структуры IL2CPP через внешнее чтение памяти, чтобы без инъекции получать имена классов, адреса методов, смещения полей и информацию о типах. Библиотека на C++ позволяет разрабатывать внешние читы для Unity-игр на IL2CPP. Она предназначена для гейм-хакеров, создающих внешние читы для Unity-проектов на базе IL2CPP.
