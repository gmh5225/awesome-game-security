ADB およびプラットフォームネイティブのアクセシビリティ API を通じて、自然言語コマンドでデバイス操作を自動化する LLM 駆動エージェントにより、Android および iOS デバイスを制御する Python フレームワークです。
複数の LLM プロバイダー（OpenAI、Anthropic、Gemini、Ollama、DeepSeek）に対応し、scripter エージェントアーキテクチャ、カスタムツール登録、構造化出力により、信頼性の高いデバイス自動化を実現します。
AI 駆動のデバイス制御で Android/iOS アプリのテストおよび操作ワークフローを自動化する、モバイルセキュリティテスターや QA エンジニアに主に役立ちます。
