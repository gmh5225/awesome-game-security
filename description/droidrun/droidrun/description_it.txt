Framework Python per controllare dispositivi Android e iOS tramite agent basati su LLM che automatizzano le interazioni con comandi in linguaggio naturale tramite ADB e le API di accessibilità native della piattaforma.
Supporta più provider LLM (OpenAI, Anthropic, Gemini, Ollama, DeepSeek) con un'architettura di agent scripter, registrazione di tool personalizzati e output strutturato per un'automazione affidabile dei dispositivi.
È principalmente utile per tester di sicurezza mobile e ingegneri QA che automatizzano test di app Android/iOS e flussi di interazione con controllo dei dispositivi guidato dall'IA.
