Python-фреймворк для управления устройствами Android и iOS через агентов на базе LLM, которые автоматизируют взаимодействие с устройством с помощью команд на естественном языке через ADB и нативные API доступности платформы.
Поддерживает несколько провайдеров LLM (OpenAI, Anthropic, Gemini, Ollama, DeepSeek), архитектуру scripter-агента, регистрацию пользовательских инструментов и структурированный вывод для надёжной автоматизации устройств.
В основном полезен специалистам по мобильной безопасности и QA-инженерам, автоматизирующим тестирование приложений Android/iOS и рабочие процессы взаимодействия с помощью ИИ-управления устройствами.
