Framework de Python para controlar dispositivos Android e iOS mediante agentes impulsados por LLM que automatizan las interacciones con comandos en lenguaje natural a través de ADB y las API de accesibilidad nativas de la plataforma.
Admite múltiples proveedores de LLM (OpenAI, Anthropic, Gemini, Ollama, DeepSeek) con una arquitectura de agente scripter, registro de herramientas personalizadas y salida estructurada para una automatización fiable de dispositivos.
Resulta especialmente útil para testers de seguridad móvil e ingenieros de QA que automatizan pruebas de aplicaciones Android/iOS y flujos de interacción con control de dispositivos impulsado por IA.
