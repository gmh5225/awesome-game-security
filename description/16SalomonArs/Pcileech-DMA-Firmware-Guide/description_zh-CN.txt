本项目是一份实操指南，教你构建能正确仿真真实 PCIe 捐赠设备的自定义 PCILeech DMA FPGA 固件，而不仅是改动 VID 与 DID。指南按 Windows 优先流程讲解捐赠设备捕获、PCIe 核心与 BAR 配置、影子配置空间、写掩码保护、能力结构以及基于 Vivado、SystemVerilog、Python COE 生成，并配合 Arbor、TeleScan PE 等工具的 TLP 响应处理。内容覆盖常见 Artix-7 DMA 板卡（Squirrel、CaptainDMA、LeetDMA、Enigma、ZDMA 及相关平台）、板卡专属烧录、冷启动校验，以及固件无法枚举时的恢复方法。还包括 Zero4K BAR 处理、Vivado ILA 调试，以及可选的 Linux 捕获路径等进阶选项。主要面向在受控实验室中从事与游戏安全和逆向工程工具相关的 DMA 与 PCIe 固件工作的研究人员与硬件学习者。
