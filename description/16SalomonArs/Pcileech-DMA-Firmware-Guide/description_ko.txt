이 프로젝트는 VID와 DID만 바꾸는 수준이 아니라, 실제 PCIe 도너 장치를 올바르게 에뮬레이트하는 커스텀 PCILeech DMA FPGA 펌웨어를 구축하는 실습 가이드입니다. Windows 우선 도너 캡처, PCIe 코어 및 BAR 설정, 섀도 구성 공간, 쓰기 마스크 보호, 기능 구조, 그리고 Vivado, SystemVerilog, Python COE 생성과 Arbor, TeleScan PE 같은 도구를 활용한 TLP 응답 작업을 단계적으로 안내합니다. 일반적인 Artix-7 DMA 보드(Squirrel, CaptainDMA, LeetDMA, Enigma, ZDMA 및 관련 플랫폼), 보드별 플래싱, 콜드 부트 검증, 펌웨어가 열거되지 않을 때의 복구 방법도 다룹니다. Zero4K BAR 처리, Vivado ILA 디버깅, 선택적 Linux 캡처 경로 같은 고급 옵션도 포함합니다. 주요 대상은 게임 보안 및 리버스 엔지니어링 도구와 관련된 통제된 랩 환경에서 DMA 및 PCIe 펌웨어 작업을 수행하는 연구자와 하드웨어 학습자입니다.
