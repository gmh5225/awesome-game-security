本專案是一份實作指南，教你建構能正確模擬真實 PCIe 捐贈裝置的自訂 PCILeech DMA FPGA 韌體，而不只是修改 VID 與 DID。指南以 Windows 優先流程說明捐贈裝置擷取、PCIe 核心與 BAR 設定、影子組態空間、寫入遮罩保護、能力結構，以及使用 Vivado、SystemVerilog、Python COE 產生，並搭配 Arbor、TeleScan PE 等工具的 TLP 回應處理。內容涵蓋常見 Artix-7 DMA 板卡（Squirrel、CaptainDMA、LeetDMA、Enigma、ZDMA 及相關平台）、板卡專屬燒錄、冷開機驗證，以及韌體無法列舉時的復原方法。亦包含 Zero4K BAR 處理、Vivado ILA 除錯，以及可選的 Linux 擷取路徑等進階選項。主要對象為在受控實驗室中從事與遊戲安全及逆向工程工具相關之 DMA 與 PCIe 韌體工作的研究人員與硬體學習者。
