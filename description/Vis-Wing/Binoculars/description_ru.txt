Этот проект — плагин для IDA Pro, который встраивает ИИ-ассистента прямо в интерфейс реверс-инжиниринга. Он в основном написан на Python и включает настраиваемые бэкенды моделей, командные промпты и действия UI для интерактивного анализа. Встроенные возможности включают объяснение функций, переименование символов и вспомогательные средства разбора бинарников Go, например рабочие процессы восстановления pclntab. Основной сценарий использования — ускорение повторяющихся задач реверс-инжиниринга для исследователей безопасности.
