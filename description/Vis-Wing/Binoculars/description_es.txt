Este proyecto es un complemento de IDA Pro que integra un asistente de IA directamente en la interfaz de ingeniería inversa. Está basado principalmente en Python e incluye backends de modelos configurables, indicaciones de comando y acciones de IU para el análisis interactivo. Sus capacidades integradas incluyen explicación de funciones, renombrado de símbolos y ayudas para el análisis de binarios Go, como flujos de trabajo de reconstrucción de pclntab. Su caso de uso principal es acelerar tareas repetitivas de ingeniería inversa para investigadores de seguridad.
