Этот проект — C API для разработки мобильных приложений с OpenGL ES и Metal, вдохновлённый GLFW.
Он поддерживает iOS 9+, tvOS 9+, Android 4.1+ и WebGL через Emscripten; среди возможностей — настройка отображения OpenGL ES 2/3 и Metal, поддержка Retina/высокого DPI, события касания и клавиатуры, а также ввод с датчиков (акселерометр, гироскоп, магнитометр).
Библиотека из одного заголовочного файла обеспечивает кроссплатформенное управление окнами, обработку ввода и жизненный цикл приложения для мобильных и веб-платформ.
Она в первую очередь полезна разработчикам мобильных игр и кроссплатформенным графическим программистам, создающим приложения на OpenGL ES или Metal.
