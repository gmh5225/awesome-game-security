Este proyecto es una API en C para el desarrollo de aplicaciones móviles con OpenGL ES y Metal, inspirada en GLFW.
Admite iOS 9+, tvOS 9+, Android 4.1+ y WebGL mediante Emscripten, con funciones que incluyen la configuración de pantalla OpenGL ES 2/3 y Metal, soporte Retina/alta DPI, eventos táctiles y de teclado, y entrada de sensores (acelerómetro, giroscopio, magnetómetro).
La biblioteca de cabecera única proporciona gestión multiplataforma de ventanas, manejo de entrada y ciclo de vida de la aplicación para plataformas móviles y web.
Resulta especialmente útil para desarrolladores de juegos móviles y programadores gráficos multiplataforma que crean aplicaciones OpenGL ES o Metal.
