このプロジェクトは、GLFW に着想を得た、OpenGL ES および Metal 向けモバイルアプリ開発用の C API です。
iOS 9+、tvOS 9+、Android 4.1+、および Emscripten 経由の WebGL をサポートし、OpenGL ES 2/3 と Metal のディスプレイ設定、Retina/高 DPI 対応、タッチとキーボードイベント、センサー入力（加速度計、ジャイロスコープ、磁力計）などの機能を備えています。
このシングルヘッダーライブラリは、モバイルおよび Web プラットフォーム向けに、クロスプラットフォームなウィンドウ管理、入力処理、アプリケーションライフサイクル管理を提供します。
主にモバイルゲーム開発者や、OpenGL ES または Metal アプリケーションを構築するクロスプラットフォームのグラフィックスプログラマに役立ちます。
