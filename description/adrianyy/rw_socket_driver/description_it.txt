rw_socket_driver è un driver del kernel Windows che espone operazioni di lettura e scrittura della memoria di processi protetti tramite socket di rete.
È implementato in C e C++ e usa la comunicazione tramite socket a livello kernel per scambiare comandi con un client remoto.
Il codice è adattato a scenari di manual mapping e si concentra sul controllo esterno della memoria senza hook in-process.
È usato principalmente nella ricerca di sicurezza dei giochi a basso livello per esperimenti di sviluppo di cheat e per valutare la robustezza degli anti-cheat.
