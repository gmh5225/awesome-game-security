rw_socket_driver es un controlador del kernel de Windows que expone operaciones de lectura y escritura de memoria de procesos protegidos a través de sockets de red.
Está implementado en C y C++ y usa comunicación por sockets en el kernel para intercambiar comandos con un cliente remoto.
El código está adaptado a escenarios de manual mapping y se centra en el control externo de memoria sin hooks dentro del proceso.
Se usa principalmente en investigación de seguridad de juegos a bajo nivel para experimentos de desarrollo de cheats y evaluación de la robustez de anti-cheats.
