rw_socket_driver — это драйвер ядра Windows, который предоставляет операции чтения и записи памяти защищённых процессов через сетевые сокеты.
Он реализован на C и C++ и использует обмен командами с удалённым клиентом через сокеты на уровне ядра.
Кодовая база адаптирована под сценарии manual mapping и ориентирована на внешнее управление памятью без внутрипроцессных хуков.
Проект в основном применяется в низкоуровневых исследованиях игровой безопасности для экспериментов по разработке читов и оценки устойчивости античитов.
