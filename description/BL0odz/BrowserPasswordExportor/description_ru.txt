Этот проект — утилита на Python для расшифровки и экспорта сохранённых паролей браузеров в Windows. Поддерживаются браузеры на базе Chromium и Firefox: обрабатываются базы профилей, расшифровка DPAPI/AES и разбор ключевого материала Firefox. Код включает разбор ASN.1 и браузер-специфичные процедуры для восстановления учётных данных из распространённых форматов хранения логинов. Основные сценарии использования — форензика учётных данных, исследования в red-team лаборатории и понимание механизмов хранения секретов браузера.
