Dieses Projekt ist ein Python-Dienstprogramm zum Entschlüsseln und Exportieren gespeicherter Browser-Passwörter unter Windows. Es unterstützt Chromium-basierte Browser sowie Firefox und verarbeitet Profildatenbanken, DPAPI/AES-Entschlüsselung und das Parsen von Firefox-Schlüsselmaterial. Der Code enthält ASN.1-Parsing und browserspezifische Routinen, um Anmeldedaten aus gängigen Login-Speicherformaten wiederherzustellen. Hauptanwendungsfälle sind Credential-Forensik, Red-Team-Laborforschung und das Verständnis des Speicherverhaltens von Browser-Geheimnissen.
