RustDMACheat es un framework de cheats orientado a DMA para el juego Rust. Ofrece ESP y modificaciones de jugabilidad como ajustes de retroceso, cambios de FOV, retoques de iluminación y vistas relacionadas con administración mediante un código base configurable. El proyecto está escrito principalmente en C++ y hace referencia a herramientas habituales del ecosistema DMA como PCILeech, MemProcFS, Il2CppDumper y DMALibrary. Está pensado para la investigación de acceso a memoria asistido por hardware y flujos de trabajo de trampas externas.
