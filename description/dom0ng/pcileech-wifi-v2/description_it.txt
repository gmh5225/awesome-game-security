Una variante di firmware FPGA PCILeech che emula lo spazio di configurazione PCIe di un adattatore di rete wireless, consentendo a un dispositivo di attacco DMA di spacciarsi per una scheda WiFi sul bus PCIe del sistema target.
Si basa sul progetto pcileech-wifi di ekknod, con integrazione di un core IP PCIe 7x in Verilog e script personalizzabili per la generazione degli ID dispositivo su varie schede FPGA.
È soprattutto utile ai ricercatori di sicurezza DMA che studiano le tecniche di impersonificazione di dispositivi PCIe e l'elusione del rilevamento DMA degli anti-cheat.
