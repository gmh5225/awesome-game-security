Una variante de firmware FPGA de PCILeech que emula el espacio de configuración PCIe de un adaptador de red inalámbrica, permitiendo que un dispositivo de ataque DMA se haga pasar por una tarjeta WiFi en el bus PCIe del sistema objetivo.
Se basa en el proyecto pcileech-wifi de ekknod, con integración de un núcleo IP PCIe 7x en Verilog y scripts personalizables de generación de ID de dispositivo para diversas placas FPGA.
Resulta especialmente útil para investigadores de seguridad DMA que estudian técnicas de suplantación de dispositivos PCIe y la evasión de la detección DMA de los anti-cheats.
