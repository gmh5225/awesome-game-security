Этот проект — веб-генератор прицела для Counter-Strike: Global Offensive. Он использует JavaScript вместе с jQuery, jQuery UI и KineticJS, а также ресурсы HTML и CSS, чтобы пользователи могли визуально просматривать и настраивать параметры прицела. Инструмент умеет формировать готовые к использованию значения конфигурации, включает пресеты и поддерживает настройку, связанную с пользовательскими привязками клавиш. Основной сценарий применения — помочь игрокам и любителям моддинга быстро создавать и делиться конфигурациями прицела.
