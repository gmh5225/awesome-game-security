Este proyecto es un generador web de puntos de mira para Counter-Strike: Global Offensive. Usa JavaScript con jQuery, jQuery UI y KineticJS, además de recursos HTML y CSS, para que los usuarios puedan previsualizar y ajustar visualmente los parámetros del punto de mira. La herramienta puede generar valores de configuración listos para usar, incluye ajustes preestablecidos y admite una configuración relacionada con atajos personalizados. Su caso de uso principal es ayudar a jugadores y entusiastas del modding a diseñar y compartir rápidamente configuraciones de punto de mira.
