Shellcrypt ist ein plattformübergreifendes Shellcode-Obfuskations-Hilfsprogramm, das rohe Payload-Bytes in einfügebereite Quellcode-Formate umwandelt.
Es ist in Python implementiert und unterstützt mehrere Verschlüsselungsverfahren, darunter AES, ChaCha20, RC4, Salsa20 und XOR-Varianten.
Das Tool kann Encoding- und Komprimierungsstufen verketten und anschließend Ausgaben für Sprachen wie C, C#, Go, Rust, Nim, Python und PowerShell erzeugen.
Es wird vor allem in Workflows der Offensive Security und der Game-Security-Forschung für Payload-Packing, Formatkonvertierung und Loader-Prototyping eingesetzt.
