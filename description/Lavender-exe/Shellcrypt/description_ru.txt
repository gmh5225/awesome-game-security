Shellcrypt — кроссплатформенная утилита обфускации шеллкода, которая преобразует сырые байты payload в готовые к вставке форматы исходного кода.
Она реализована на Python и поддерживает несколько шифров, включая AES, ChaCha20, RC4, Salsa20 и варианты XOR.
Инструмент может объединять этапы кодирования и сжатия, а затем выдавать результат для языков вроде C, C#, Go, Rust, Nim, Python и PowerShell.
В основном применяется в рабочих процессах offensive security и исследований игровой безопасности для упаковки payload, преобразования форматов и прототипирования загрузчиков.
