Shellcrypt は、生のペイロードバイトをそのまま貼り付け可能なソースコード形式へ変換する、クロスプラットフォームのシェルコード難読化ユーティリティです。
Python で実装されており、AES、ChaCha20、RC4、Salsa20、および XOR 系を含む複数の暗号方式をサポートします。
エンコードと圧縮の段階を連鎖させたうえで、C、C#、Go、Rust、Nim、Python、PowerShell などの言語向け出力を生成できます。
主にオフェンシブセキュリティおよびゲームセキュリティ研究のワークフローで、ペイロードのパッキング、形式変換、ローダーのプロトタイピングに用いられます。
