Shellcrypt es una utilidad multiplataforma de ofuscación de shellcode que transforma bytes de payload en bruto en formatos de código fuente listos para pegar.
Está implementada en Python y admite múltiples cifrados, incluidos AES, ChaCha20, RC4, Salsa20 y variantes de XOR.
La herramienta puede encadenar etapas de codificación y compresión, y luego emitir salida para lenguajes como C, C#, Go, Rust, Nim, Python y PowerShell.
Se usa principalmente en flujos de trabajo de seguridad ofensiva e investigación de seguridad en juegos para empaquetado de payloads, conversión de formatos y prototipado de loaders.
