Symbridge는 IDA Pro와 x64dbg 사이의 실시간 주석 동기화 브리지로, 리버스 엔지니어가 정적 분석과 동적 분석을 오가며 이름, 주석, 타입을 일치시킬 수 있게 합니다. 심볼 이름, 일반 및 repeatable 주석, C 구조체/타입 정의를 모듈과 RVA를 키로 양방향 동기화하므로 ASLR과 서로 다른 이미지 베이스에서도 매칭이 깨지지 않습니다. Python 브로커가 localhost TCP에서 줄바꿈으로 구분된 JSON으로 정식 상태를 유지하고, 얇은 어댑터(IDB_Hooks 기반 IDAPython과 네이티브 C++ x64dbg 플러그인)가 로컬 편집을 푸시하고 에코 루프 없이 원격 변경을 적용합니다. 선택적 영속화로 재시작 후에도 주석이 남으며, 프로토콜은 나중에 다른 도구를 어댑터로 추가할 수 있도록 설계되었습니다. 동일 바이너리에서 IDA와 x64dbg를 함께 쓰는 리버스 엔지니어링 워크플로를 대상으로 합니다.
