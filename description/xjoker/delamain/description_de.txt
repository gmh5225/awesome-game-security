Delamain ist ein headless MCP-Server (Model Context Protocol), der die volle Leistungsfähigkeit des JADX-Android-APK/DEX-Decompilers für Reverse Engineering an KI-Agenten freigibt. Er kombiniert ein Java-Backend (Javalin um jadx’ Headless-API) mit einem Python-FastMCP-Gateway für Authentifizierung, Routing und Out-of-Band-Dateiübertragung in einem einzigen Docker-Image. Agenten können Klassen dekompilieren, Code und Strings durchsuchen, Querverweise und Datenflüsse verfolgen, Manifeste und Ressourcen prüfen, Frida-Hooks erzeugen, Sicherheitsscans ausführen sowie Code umbenennen oder annotieren — über KI-orientierte Tools mit begrenzter, paginierter Ausgabe. Ziel sind Android-APKs und verwandte Formate; ein speichersparender, mmap-basierter Index ist für Server, CI und eingeschränkte Umgebungen ausgelegt, in denen ein KI-Agent statt einer GUI die Analyse steuert.
