Delamain 是一个无界面的 MCP（Model Context Protocol）服务器，向 AI 代理开放 JADX Android APK/DEX 反编译器的全部能力，用于逆向工程。它将 Java 后端（用 Javalin 封装 jadx 的无头 API）与 Python FastMCP 网关（负责认证、路由与带外文件传输）打包在同一 Docker 镜像中。代理可通过面向 AI、带边界与分页输出的工具反编译类、搜索代码与字符串、追踪交叉引用与数据流、检查清单与资源、生成 Frida hook、运行安全扫描，以及对代码进行重命名或标注。它面向 Android APK 及相关格式，采用低内存、基于 mmap 的索引，适用于服务器、CI 与资源受限环境，由 AI 代理而非 GUI 驱动分析。
