Delamain est un serveur MCP (Model Context Protocol) sans interface qui expose toutes les capacités du décompilateur Android APK/DEX JADX aux agents IA pour la rétro-ingénierie. Il associe un backend Java (Javalin encapsulant l’API headless de jadx) à une passerelle Python FastMCP pour l’authentification, le routage et le transfert de fichiers hors bande, le tout dans une seule image Docker. Les agents peuvent décompiler des classes, rechercher du code et des chaînes, suivre les références croisées et les flux de données, inspecter les manifests et ressources, générer des hooks Frida, lancer des analyses de sécurité, et renommer ou annoter le code via des outils orientés IA avec une sortie bornée et paginée. Il cible les APK Android et formats associés, avec un index à faible mémoire basé sur mmap, conçu pour les serveurs, la CI et les environnements contraints où un agent IA mène l’analyse à la place d’une interface graphique.
