Delamain es un servidor MCP (Model Context Protocol) sin interfaz que expone toda la capacidad del descompilador Android APK/DEX JADX a agentes de IA para ingeniería inversa. Combina un backend Java (Javalin envolviendo la API headless de jadx) con una pasarela Python FastMCP para autenticación, enrutado y transferencia de archivos fuera de banda en una sola imagen Docker. Los agentes pueden descompilar clases, buscar código y cadenas, rastrear referencias cruzadas y flujos de datos, inspeccionar manifiestos y recursos, generar hooks de Frida, ejecutar análisis de seguridad, y renombrar o anotar código mediante herramientas orientadas a IA con salida acotada y paginada. Está pensado para APK de Android y formatos relacionados, con un índice de bajo consumo de memoria basado en mmap, diseñado para servidores, CI y entornos limitados donde un agente de IA dirige el análisis en lugar de una GUI.
