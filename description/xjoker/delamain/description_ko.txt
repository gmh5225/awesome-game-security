Delamain은 JADX Android APK/DEX 디컴파일러의 전체 기능을 AI 에이전트에 노출하여 리버스 엔지니어링을 가능하게 하는 헤드리스 MCP(Model Context Protocol) 서버입니다. Java 백엔드(Javalin으로 jadx 헤드리스 API를 래핑)와 인증, 라우팅, 아웃오브밴드 파일 전송을 담당하는 Python FastMCP 게이트웨이를 단일 Docker 이미지로 제공합니다. 에이전트는 경계와 페이지네이션이 적용된 AI 지향 도구로 클래스 디컴파일, 코드·문자열 검색, 교차 참조와 데이터 흐름 추적, 매니페스트·리소스 검사, Frida 훅 생성, 보안 스캔 실행, 코드 이름 변경·주석 달기를 수행할 수 있습니다. Android APK 및 관련 형식을 대상으로 하며, GUI 대신 AI 에이전트가 분석을 주도하는 서버, CI, 제약된 환경을 위해 저메모리 mmap 기반 인덱스를 사용합니다.
