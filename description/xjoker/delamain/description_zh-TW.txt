Delamain 是一個無介面的 MCP（Model Context Protocol）伺服器，向 AI 代理開放 JADX Android APK/DEX 反編譯器的全部能力，用於逆向工程。它將 Java 後端（以 Javalin 封裝 jadx 的無頭 API）與 Python FastMCP 閘道（負責認證、路由與帶外檔案傳輸）打包在同一 Docker 映像中。代理可透過面向 AI、具邊界與分頁輸出的工具反編譯類別、搜尋程式碼與字串、追蹤交叉參考與資料流、檢查清單與資源、產生 Frida hook、執行安全掃描，以及對程式碼重新命名或註解。它面向 Android APK 及相關格式，採用低記憶體、基於 mmap 的索引，適用於伺服器、CI 與資源受限環境，由 AI 代理而非 GUI 驅動分析。
