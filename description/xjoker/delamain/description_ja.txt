Delamain は、JADX Android APK/DEX デコンパイラの全機能を AI エージェントに公開し、リバースエンジニアリングを可能にするヘッドレス MCP（Model Context Protocol）サーバーです。Java バックエンド（Javalin で jadx のヘッドレス API をラップ）と、認証・ルーティング・アウトオブバンドのファイル転送を担う Python FastMCP ゲートウェイを、単一の Docker イメージにまとめています。エージェントは、境界付き・ページング出力の AI 向けツールを通じて、クラスのデコンパイル、コードや文字列の検索、クロスリファレンスとデータフローの追跡、マニフェストとリソースの検査、Frida フックの生成、セキュリティスキャンの実行、コードのリネームや注釈付けが可能です。Android APK および関連形式を対象とし、低メモリの mmap ベース索引により、GUI ではなく AI エージェントが分析を主導するサーバー、CI、制約のある環境向けに設計されています。
