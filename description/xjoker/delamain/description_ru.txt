Delamain — это headless-сервер MCP (Model Context Protocol), который открывает ИИ-агентам полный функционал декомпилятора Android APK/DEX JADX для реверс-инжиниринга. Он объединяет Java-бэкенд (Javalin, оборачивающий headless API jadx) и Python-шлюз FastMCP для аутентификации, маршрутизации и внеполосной передачи файлов в одном образе Docker. Агенты могут декомпилировать классы, искать код и строки, отслеживать перекрёстные ссылки и потоки данных, просматривать манифесты и ресурсы, генерировать хуки Frida, запускать проверки безопасности, а также переименовывать или аннотировать код через ИИ-ориентированные инструменты с ограниченным, постраничным выводом. Проект нацелен на Android APK и связанные форматы и использует низкопамятный индекс на основе mmap для серверов, CI и ограниченных сред, где анализ ведёт ИИ-агент, а не GUI.
