Dieses Projekt konzentriert sich auf Skylicht Engine. Skylicht Engine ist eine Weiterentwicklung des Irrlicht-Projekts.
Skylicht Engine ist eine ultraleichte Game Engine für mobile Plattformen (Android, iOS).
Sie ist vor allem nützlich für Spieleentwickler, Engine-Programmierer und Grafikforscher, die im Bereich Game Engine / Quellcode arbeiten.
