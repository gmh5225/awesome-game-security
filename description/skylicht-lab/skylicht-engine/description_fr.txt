Ce projet se concentre sur Skylicht Engine. Skylicht Engine est une évolution du projet Irrlicht.
Skylicht Engine est un moteur de jeu ultra léger destiné aux plateformes mobiles (Android, iOS).
Il est principalement utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
