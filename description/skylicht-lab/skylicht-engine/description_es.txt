Este proyecto se centra en Skylicht Engine. Skylicht Engine es una evolución del proyecto Irrlicht.
Skylicht Engine es un motor de juegos ultraligero orientado a plataformas móviles (Android, iOS).
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores de gráficos que trabajan en el área de motores de juego / código fuente.
