Questo progetto è un framework di cheat esterno per Windows basato su lettura della memoria e rendering overlay per uno shooter Unreal Engine. L’implementazione in C++ usa DirectX9 e ImGui per fornire visuali ESP, un comportamento aimbot configurabile e modifiche legate al rinculo tramite accesso alla memoria a runtime. Include offset di gioco hardcoded, logica di caching degli actor e menu a schermo per attivare e regolare le funzioni. Il repository è usato soprattutto per prototipare cheat e per la ricerca sull’evasione anti-cheat.
