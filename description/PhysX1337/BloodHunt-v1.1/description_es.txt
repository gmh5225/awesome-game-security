Este proyecto es un framework de cheat externo para Windows centrado en la lectura de memoria y el renderizado de overlays para un shooter de Unreal Engine. La implementación en C++ usa DirectX9 e ImGui para ofrecer visuales ESP, un aimbot configurable y modificaciones relacionadas con el retroceso mediante acceso a memoria en tiempo de ejecución. Incluye offsets del juego fijados en código, lógica de caché de actors y menús en pantalla para activar y ajustar funciones. El repositorio se usa principalmente para prototipar cheats e investigar la evasión de anti-cheat.
