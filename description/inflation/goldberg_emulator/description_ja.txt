本プロジェクトは Goldberg Steam Emulator であり、Steam サーバーに接続せずに Steamworks API 呼び出しをエミュレートする Steam API 互換レイヤーです。steam_api.dll/steam_api64.dll を置き換え、Steam 依存ゲームにオフライン機能を提供し、実績、リーダーボード、マッチメイキング、オーバーレイ、DLC 所有権チェックをローカルでエミュレートします。この C++ ライブラリは、直接ネットワークブロードキャストにより LAN マルチプレイヤーをサポートします。ゲーム保存愛好家、モッダー、Steam DRM および API 依存関係を研究する研究者を対象としています。
