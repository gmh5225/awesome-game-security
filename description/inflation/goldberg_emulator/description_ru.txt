Этот проект — Goldberg Steam Emulator, слой совместимости Steam API, который эмулирует вызовы Steamworks API без подключения к серверам Steam. Он заменяет steam_api.dll/steam_api64.dll, обеспечивая офлайн-функциональность для игр, зависящих от Steam, и локально эмулирует достижения, таблицы лидеров, матчмейкинг, оверлей и проверки владения DLC. Библиотека на C++ поддерживает сетевую игру по LAN через прямую сетевую широковещательную рассылку. Она предназначена для энтузиастов сохранения игр, моддеров и исследователей, изучающих Steam DRM и зависимости API.
