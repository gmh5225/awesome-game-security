Questo progetto è il Goldberg Steam Emulator, un livello di compatibilità Steam API che emula le chiamate Steamworks API senza connettersi ai server Steam. Sostituisce steam_api.dll/steam_api64.dll per offrire funzionalità offline ai giochi dipendenti da Steam, emulando localmente achievement, classifiche, matchmaking, overlay e controlli di possesso DLC. La libreria C++ supporta il multiplayer LAN tramite broadcast di rete diretto. È rivolta agli appassionati di conservazione dei giochi, ai modder e ai ricercatori che studiano il DRM Steam e le dipendenze API.
