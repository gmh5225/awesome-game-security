Este proyecto es el Goldberg Steam Emulator, una capa de compatibilidad de la Steam API que emula las llamadas a la Steamworks API sin conectarse a los servidores de Steam. Reemplaza steam_api.dll/steam_api64.dll para ofrecer funcionalidad sin conexión a juegos que dependen de Steam, emulando localmente logros, tablas de clasificación, matchmaking, overlay y comprobaciones de posesión de DLC. La biblioteca C++ admite multijugador en LAN mediante difusión de red directa. Está orientada a entusiastas de la preservación de juegos, modders e investigadores que estudian el DRM de Steam y las dependencias de la API.
