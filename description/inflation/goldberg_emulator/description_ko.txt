이 프로젝트는 Goldberg Steam Emulator로, Steam 서버에 연결하지 않고 Steamworks API 호출을 에뮬레이트하는 Steam API 호환 계층입니다. steam_api.dll/steam_api64.dll을 교체하여 Steam 의존 게임에 오프라인 기능을 제공하고, 업적, 리더보드, 매치메이킹, 오버레이, DLC 소유권 검사를 로컬에서 에뮬레이트합니다. 이 C++ 라이브러리는 직접 네트워크 브로드캐스팅을 통해 LAN 멀티플레이어를 지원합니다. 게임 보존 애호가, 모더, Steam DRM 및 API 의존성을 연구하는 연구자를 대상으로 합니다.
