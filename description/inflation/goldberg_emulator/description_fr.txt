Ce projet est le Goldberg Steam Emulator, une couche de compatibilité Steam API qui émule les appels Steamworks API sans se connecter aux serveurs Steam. Il remplace steam_api.dll/steam_api64.dll pour offrir un fonctionnement hors ligne aux jeux dépendant de Steam, en émulant localement les succès, classements, matchmaking, overlay et vérifications de possession de DLC. La bibliothèque C++ prend en charge le multijoueur en LAN via une diffusion réseau directe. Elle s'adresse aux passionnés de préservation de jeux, aux moddeurs et aux chercheurs étudiant le DRM Steam et les dépendances API.
