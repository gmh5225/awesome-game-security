Este proyecto se centra en crear plugins para X64Dbg con facilidad. DotX64Dbg pretende ofrecer una forma fluida de escribir y probar plugins para X64Dbg usando .Net 6.0 y C#.
Este gif muestra cómo depurar y editar tu plugin al mismo tiempo, y también cómo registrar comandos personalizados para x64Dbg sobre la marcha; lo mismo funciona para expresiones.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito del cheat / depuración.
