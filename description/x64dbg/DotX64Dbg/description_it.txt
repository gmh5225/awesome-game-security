Questo progetto si concentra sulla creazione semplificata di plugin per X64Dbg. DotX64Dbg mira a offrire un modo fluido per scrivere e testare plugin per X64Dbg usando .Net 6.0 e C#.
Questa gif mostra come eseguire il debug e modificare il plugin allo stesso tempo, e anche come registrare al volo comandi personalizzati per x64Dbg; lo stesso vale per le espressioni.
