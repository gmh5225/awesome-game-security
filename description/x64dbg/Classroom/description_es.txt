Este proyecto es un complemento de x64dbg que ayuda a analizar una aplicación orientada a objetos. Puedes definir funciones miembro y variables miembro mientras depuras la aplicación, y la documentación de la clase se guarda y se muestra cuando la necesites. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheats / complementos de x64dbg.
