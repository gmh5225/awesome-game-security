이 프로젝트는 표준 LoadLibrary 주입 대신 매뉴얼 매핑(manual mapping)에 중점을 둔 Windows DLL 인젝터입니다.
C/C++로 작성된 인젝터와 로더 구성 요소를 모두 포함하며, 스레드 하이재킹과 NtCreateThreadEx 기반 실행 경로를 지원합니다.
구현은 임포트 해석, 재배치 처리, TLS 콜백 실행, 그리고 매핑된 모듈에 대한 선택적 예외 지원을 다룹니다.
또한 헤더 지우기 또는 위조, PEB 언링킹, DLL 이름 스크램블, 핸들 하이재킹 기반 프로세스 접근 등 스텔스 지향 플래그를 제공합니다.
주입 기법과 탐지 회피 행동을 연구하는 고급 게임 보안 및 악성코드 분석 연구자를 대상으로 합니다.
