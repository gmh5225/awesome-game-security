Ce projet est un packer d'exécutables Windows qui reconstruit des fichiers PE avec un stub de décompression personnalisé.
Il compresse le binaire d'origine par codage de Huffman, stocke les données dans une section dédiée et produit une nouvelle image orientée chargeur.
Le stub résout les API de bas niveau NTDLL et KERNEL32, déchiffre les noms d'import, mappe les sections, applique les relocalisations, corrige les imports et saute vers le point d'entrée d'origine.
Le code est écrit en C/C++ avec des projets Visual Studio et cible les internals PE x86 et x64.
Il est surtout utile pour la recherche en reverse engineering, l'analyse de packers et la compréhension des chargeurs anti-analyse dans le contexte de la sécurité des jeux.
