Questo progetto è un packer di eseguibili Windows che ricostruisce file PE con uno stub di unpacking personalizzato.
Comprime il binario originale con codifica di Huffman, memorizza i dati in una sezione dedicata ed emette una nuova immagine orientata al loader.
Lo stub risolve API di basso livello di NTDLL e KERNEL32, decifra i dati dei nomi di import, mappa le sezioni, applica le rilocazioni, corregge gli import e salta al punto di ingresso originale.
Il codice è scritto in C/C++ con progetti Visual Studio e punta agli interni PE x86 e x64.
È soprattutto utile per la ricerca di reverse engineering, l'analisi dei packer e la comprensione dei loader anti-analisi nei contesti di sicurezza dei giochi.
