이 프로젝트는 커스텀 언패킹 스텁으로 PE 파일을 재구성하는 Windows 실행 파일 패커입니다.
Huffman 코딩으로 원본 바이너리를 압축하고 전용 섹션에 데이터를 저장한 뒤, 로더 지향의 새 이미지를 생성합니다.
스텁은 저수준 NTDLL 및 KERNEL32 API를 해석하고, 임포트 이름 데이터를 복호화하며, 섹션을 매핑하고 재배치를 적용한 다음 임포트를 수정하고 원래 진입점으로 점프합니다.
코드베이스는 C/C++로 작성되어 있으며 Visual Studio 프로젝트를 포함하고, x86 및 x64 PE 내부를 대상으로 합니다.
주로 리버스 엔지니어링 연구, 패커 분석, 게임 보안 맥락의 안티 분석 로더 이해에 유용합니다.
