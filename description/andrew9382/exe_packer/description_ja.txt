本プロジェクトは、カスタム展開スタブで PE ファイルを再構築する Windows 実行ファイルパッカーです。
Huffman 符号化で元のバイナリを圧縮し、専用セクションにデータを格納して、ローダー向けの新しいイメージを出力します。
スタブは低レベルな NTDLL および KERNEL32 API を解決し、インポート名データを復号し、セクションをマップして再配置を適用し、インポートを修正したうえで元のエントリポイントへジャンプします。
コードベースは C/C++ で書かれ、Visual Studio プロジェクトを備え、x86 と x64 の PE 内部構造を対象とします。
主にリバースエンジニアリング研究、パッカー解析、およびゲームセキュリティ文脈におけるアンチ解析ローダーの理解に役立ちます。
