Этот проект — упаковщик исполняемых файлов Windows, который пересобирает PE-файлы с пользовательским stub распаковки.
Он сжимает исходный бинарник кодированием Хаффмана, сохраняет данные в отдельной секции и формирует новый образ, ориентированный на загрузчик.
Stub разрешает низкоуровневые API NTDLL и KERNEL32, расшифровывает данные имён импорта, отображает секции, применяет релокации, исправляет импорты и переходит к исходной точке входа.
Кодовая база написана на C/C++ с проектами Visual Studio и ориентирована на внутренности PE x86 и x64.
В основном полезна для исследований по реверс-инжинирингу, анализа упаковщиков и понимания антианалитических загрузчиков в контексте безопасности игр.
