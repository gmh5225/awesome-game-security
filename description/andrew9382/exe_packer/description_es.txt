Este proyecto es un empaquetador de ejecutables Windows que reconstruye archivos PE con un stub de desempaquetado personalizado.
Comprime el binario original con codificación Huffman, almacena los datos en una sección dedicada y genera una nueva imagen orientada al cargador.
El stub resuelve API de bajo nivel de NTDLL y KERNEL32, descifra los nombres de importación, mapea secciones, aplica reubicaciones, corrige importaciones y salta al punto de entrada original.
El código está escrito en C/C++ con proyectos de Visual Studio y se centra en los internos PE de x86 y x64.
Resulta útil principalmente para investigación de ingeniería inversa, análisis de packers y comprensión de cargadores antianálisis en contextos de seguridad de juegos.
