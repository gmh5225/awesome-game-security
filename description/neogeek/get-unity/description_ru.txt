Этот проект посвящён инструменту командной строки для получения URL загрузки последней или конкретной версии Unity.
Он в основном написан на JavaScript и сосредоточен на инструментах редактора и Unity.
В первую очередь полезен разработчикам инструментов и специалистам по реверс-инжинирингу, работающим в области игровых инструментов.
