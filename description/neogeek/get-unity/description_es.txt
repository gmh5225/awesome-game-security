Este proyecto se centra en una herramienta de línea de comandos para obtener la URL de descarga de la última versión o de una versión concreta de Unity.
Está escrito principalmente en JavaScript y se centra en herramientas de editor y Unity.
Resulta especialmente útil para desarrolladores de herramientas e ingenieros de ingeniería inversa que trabajan en el ámbito de las herramientas de juegos.
