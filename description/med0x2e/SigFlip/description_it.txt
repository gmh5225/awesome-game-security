Questo progetto è uno strumento per patchare file PE firmati Authenticode (exe, dll, sys, ecc.) senza invalidare o rompere la firma esistente.
SigInject cifra e inietta shellcode nella tabella certificati [WIN_CERTIFICATE] di un file PE; la chiave di cifratura viene stampata per l'uso con un loader BOF/C/C# di base (SigLoader). SigInject salva le modifiche in un file PE modificato mantenendo intatte la firma e la validità del certificato.
È principalmente utile per ricercatori low-level su Windows, Linux e mobile che lavorano nell'area some tricks / Windows Ring3.
