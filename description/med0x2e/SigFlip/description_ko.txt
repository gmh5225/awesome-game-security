이 프로젝트는 기존 서명을 무효화하거나 손상시키지 않고 Authenticode 서명된 PE 파일(exe, dll, sys 등)을 패치하는 도구입니다.
SigInject는 셸코드를 암호화하여 PE 파일의 [WIN_CERTIFICATE] 인증서 테이블에 주입하고, 기본 BOF/C/C# 로더(SigLoader)에서 사용할 암호화 키를 출력합니다. SigInject는 변경 사항을 수정된 PE 파일에 저장하면서 서명과 인증서 유효성을 그대로 유지합니다.
주로 일부 트릭 / Windows Ring3 영역에서 연구하는 저수준 Windows, Linux 및 모바일 연구자에게 유용합니다.
