Ce projet est un outil permettant de patcher des fichiers PE signés Authenticode (exe, dll, sys, etc.) sans invalider ni casser la signature existante.
SigInject chiffre et injecte du shellcode dans la table de certificats [WIN_CERTIFICATE] d'un fichier PE ; la clé de chiffrement est affichée pour être utilisée avec un chargeur BOF/C/C# basique (SigLoader). SigInject enregistre les modifications dans un fichier PE modifié tout en préservant l'intégrité de sa signature et de son certificat.
Il est principalement utile aux chercheurs bas niveau Windows, Linux et mobile travaillant dans le domaine des quelques astuces / Windows Ring3.
