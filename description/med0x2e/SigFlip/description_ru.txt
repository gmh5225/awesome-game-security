Этот проект — инструмент для патчинга PE-файлов с подписью Authenticode (exe, dll, sys и т.д.) без аннулирования или повреждения существующей подписи.
SigInject шифрует и внедряет shellcode в таблицу сертификатов [WIN_CERTIFICATE] PE-файла; ключ шифрования выводится для использования с базовым загрузчиком BOF/C/C# (SigLoader). SigInject сохраняет изменения в модифицированном PE-файле, сохраняя действительность подписи и сертификата.
Он в основном полезен исследователям низкого уровня Windows, Linux и мобильных платформ, работающим в области some tricks / Windows Ring3.
