Questo progetto si concentra sul fuzzer.
Fuzzable è un framework che si integra sia con codice sorgente C/C++ sia con binari per aiutare i ricercatori di vulnerabilità a identificare le funzioni obiettivo adatte al fuzzing.
È principalmente utile per i ricercatori di sicurezza dei giochi e per i reverse engineer che studiano tecniche offensive nell'ambito dei plugin cheat / Binary Ninja.
