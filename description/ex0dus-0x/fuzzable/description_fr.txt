Ce projet se concentre sur le fuzzer.
Fuzzable est un framework qui s'intègre à la fois au code source C/C++ et aux binaires pour aider les chercheurs en vulnérabilités à identifier les fonctions cibles adaptées au fuzzing.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des plugins cheat / Binary Ninja.
