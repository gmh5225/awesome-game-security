InfinityHookPro-main es un proyecto de hooking del kernel de Windows derivado de InfinityHook, con soporte añadido para máquinas físicas. Está orientado a Windows 7 hasta Windows 11 e incluye código de controlador de bajo nivel para la interceptación de syscalls mediante rutas relacionadas con ETW o CKCL, el manejo de contexto SSDT y ayudas de escaneo de patrones en el kernel. La implementación es principalmente en C o C++ y expone lógica de interceptación basada en callbacks para monitorear o modificar el flujo de despacho de llamadas al sistema. Lo usan sobre todo investigadores de seguridad del kernel que exploran telemetría anti-cheat, monitoreo de syscalls y comportamientos de detección o bypass de hooks.
