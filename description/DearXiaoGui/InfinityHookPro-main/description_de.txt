InfinityHookPro-main ist ein von InfinityHook abgeleitetes Windows-Kernel-Hooking-Projekt mit zusätzlicher Unterstützung für physische Maschinen. Es zielt auf Windows 7 bis Windows 11 und enthält Low-Level-Treibercode zur Syscall-Interception über ETW- oder CKCL-bezogene Pfade, SSDT-Kontextverarbeitung sowie Hilfen zum Kernel-Pattern-Scanning. Die Implementierung ist überwiegend in C oder C++ und stellt callback-basierte Interception-Logik bereit, um den Syscall-Dispatch-Ablauf zu überwachen oder zu verändern. Es wird vor allem von Kernel-Sicherheitsforschern genutzt, die Anti-Cheat-Telemetrie, Syscall-Monitoring sowie Hook-Erkennung oder Bypass-Verhalten untersuchen.
