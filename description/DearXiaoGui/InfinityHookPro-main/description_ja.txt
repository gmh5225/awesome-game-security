InfinityHookPro-main は InfinityHook を基に派生した Windows カーネルフッキングプロジェクトで、物理マシン向けのサポートを追加しています。Windows 7 から Windows 11 を対象とし、ETW または CKCL 関連パスによるシステムコール傍受、SSDT コンテキスト処理、カーネルパターン走査用ヘルパーなど、低レベルなドライバコードを含みます。実装は主に C/C++ で、コールバックベースの傍受ロジックによりシステムコールディスパッチの監視や変更が可能です。アンチチートのテレメトリ、システムコール監視、フック検出や回避の挙動を調べるカーネルセキュリティ研究者向けに主に利用されます。
