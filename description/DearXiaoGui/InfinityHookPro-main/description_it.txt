InfinityHookPro-main è un progetto di hooking del kernel Windows derivato da InfinityHook, con supporto aggiunto per macchine fisiche. Punta a Windows 7 fino a Windows 11 e include codice driver a basso livello per l'intercettazione delle syscall tramite percorsi legati a ETW o CKCL, la gestione del contesto SSDT e helper per lo scanning di pattern nel kernel. L'implementazione è principalmente in C o C++ ed espone una logica di intercettazione basata su callback per monitorare o modificare il flusso di dispatch delle chiamate di sistema. È usato soprattutto da ricercatori di sicurezza del kernel che esplorano la telemetria anti-cheat, il monitoraggio delle syscall e i comportamenti di rilevamento o bypass degli hook.
