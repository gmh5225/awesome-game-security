InfinityHookPro-main은 InfinityHook에서 파생된 Windows 커널 후킹 프로젝트로, 물리 머신에 대한 지원이 추가되어 있습니다. Windows 7부터 Windows 11까지를 대상으로 하며, ETW 또는 CKCL 관련 경로를 통한 시스템 콜 가로채기, SSDT 컨텍스트 처리, 커널 패턴 스캔 헬퍼 등 저수준 드라이버 코드를 포함합니다. 구현은 주로 C/C++이며, 콜백 기반 가로채기 로직으로 시스템 콜 디스패치 흐름을 모니터링하거나 수정합니다. 안티치트 텔레메트리, 시스템 콜 모니터링, 훅 탐지 또는 우회 동작을 연구하는 커널 보안 연구자들이 주로 사용합니다.
