Questo progetto è un framework di automazione di gioco esterno orientato ai servizi, incentrato sull'interazione con la memoria di Apex Legends.
È realizzato in C# su .NET con componenti modulari per l'accesso alla memoria, l'astrazione dei driver, la modellazione dello stato di gioco e livelli separati di funzionalità Aim e Sense.
Il repository include script di build e installazione del servizio Linux, nonché indicazioni di hardening come isolamento dei processi e restrizioni ptrace.
Si rivolge a flussi di lavoro avanzati di ricerca sul game hacking che richiedono tooling di lunga durata, attento allo stealth e guidato dalla memoria.
