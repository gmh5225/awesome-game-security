Этот проект — сервисно-ориентированный внешний фреймворк автоматизации игр с акцентом на взаимодействие с памятью Apex Legends.
Он написан на C# под .NET с модульными компонентами для доступа к памяти, абстракции драйверов, моделирования состояния игры, а также отдельных функциональных слоёв Aim и Sense.
Репозиторий включает скрипты сборки и установки Linux-сервиса, а также рекомендации по усилению защиты, такие как изоляция процессов и ограничения ptrace.
Он рассчитан на продвинутые исследовательские сценарии game hacking, требующие долгоживущих, скрытных и управляемых через память инструментов.
