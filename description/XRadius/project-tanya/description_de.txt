Dieses Projekt ist ein serviceorientiertes externes Spielautomatisierungs-Framework mit Fokus auf Speicherinteraktion in Apex Legends.
Es ist in C# auf .NET mit modularen Komponenten für Kern-Speicherzugriff, Treiberabstraktion, Spielzustandsmodellierung sowie getrennte Aim- und Sense-Funktionsschichten aufgebaut.
Das Repository enthält Build- und Installationsskripte für einen Linux-Dienst sowie Härtungshinweise wie Prozessisolation und ptrace-Einschränkungen.
Es richtet sich an fortgeschrittene Game-Hacking-Forschungsabläufe, die langlebige, stealth-bewusste, speichergetriebene Werkzeuge erfordern.
