Этот проект — доказательство концепции для Windows, демонстрирующее самопереотображение кода (self-remapping) как технику защиты от вмешательства. Он создаёт несколько виртуальных отображений одних и тех же физических страниц, выполняет код из одного отображения, одновременно проверяя целостность другого, что затрудняет отладчикам и инструментам патчинга незаметное изменение выполняемого кода. Реализация на C показывает, как файловые отображения на основе section могут создавать aliased-представления. Проект предназначен для исследователей программной защиты, изучающих антипатчинг и антиотладку через memory aliasing.
