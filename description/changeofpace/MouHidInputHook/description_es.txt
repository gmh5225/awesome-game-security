Este proyecto es un controlador del kernel de Windows que engancha el objeto CONNECT_DATA dentro de los objetos de dispositivo MouHid para filtrar, modificar e inyectar paquetes de entrada del ratón sin modificar las pilas de dispositivos de ratón HID USB.
Emula la estrategia Moufiltr interceptando el callback ClassService usado para transferir datos del ratón a las colas de datos de clase, admitiendo un desenganche seguro sin descargar pilas de dispositivos y notificaciones PnP ante cambios de dispositivo.
La técnica es segura frente a PatchGuard y relativamente sigilosa, ya que evita adjuntar objetos de dispositivo filtro y usa una heurística para resolver offsets no documentados de campos CONNECT_DATA.
Es especialmente útil para investigadores de anti-trampas y analistas de seguridad de entrada que estudian la interceptación de entrada del ratón a nivel de kernel y la detección de simulación de entrada.
