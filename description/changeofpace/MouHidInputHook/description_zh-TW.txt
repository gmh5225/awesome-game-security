本專案是一個 Windows 核心驅動程式，透過掛鉤 MouHid 裝置物件內的 CONNECT_DATA 物件，在不修改 HID USB 滑鼠裝置堆疊的情況下過濾、修改與注入滑鼠輸入封包。
它仿效 Moufiltr 策略，攔截用於將滑鼠資料傳輸到類別資料佇列的 ClassService 回呼，支援在不卸載裝置堆疊的情況下安全解除掛鉤，並透過 PnP 通知處理裝置變更。
此技術對 PatchGuard 安全且相對隱蔽，因為避免掛接過濾裝置物件，並使用啟發式方法解析未公開的 CONNECT_DATA 欄位偏移。
它主要適用於反作弊研究人員與輸入安全分析人員，用於研究核心層級滑鼠輸入攔截與輸入模擬偵測。
