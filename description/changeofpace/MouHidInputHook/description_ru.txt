Этот проект — драйвер ядра Windows, который перехватывает объект CONNECT_DATA внутри объектов устройства MouHid, чтобы фильтровать, изменять и внедрять пакеты ввода мыши без изменения стеков устройств HID USB-мыши.
Он эмулирует стратегию Moufiltr, перехватывая callback ClassService, используемый для передачи данных мыши в очереди данных класса, поддерживает безопасное снятие перехвата без выгрузки стеков устройств и PnP-уведомления об изменении устройств.
Техника безопасна для PatchGuard и относительно скрытна, поскольку не присоединяет объекты устройств-фильтров и использует эвристику для разрешения недокументированных смещений полей CONNECT_DATA.
Она в основном полезна исследователям античита и аналитикам безопасности ввода, изучающим перехват ввода мыши на уровне ядра и обнаружение симуляции ввода.
