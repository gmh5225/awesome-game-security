이 프로젝트는 MouHid 디바이스 객체 내부의 CONNECT_DATA 객체를 후킹하는 Windows 커널 드라이버로, HID USB 마우스 디바이스 스택을 수정하지 않고 마우스 입력 패킷을 필터링·수정·주입합니다.
Moufiltr 전략을 모방하여 마우스 데이터를 클래스 데이터 큐로 전달하는 ClassService 콜백을 가로채며, 디바이스 스택을 언로드하지 않고도 안전하게 언후킹할 수 있고 PnP 알림으로 디바이스 변경에 대응합니다.
필터 디바이스 객체를 어태치하지 않고, 문서화되지 않은 CONNECT_DATA 필드 오프셋을 휴리스틱으로 해석하므로 PatchGuard에 안전하며 비교적 은밀합니다.
주로 커널 수준 마우스 입력 가로채기와 입력 시뮬레이션 탐지를 연구하는 안티치트 연구자 및 입력 보안 분석가에게 유용합니다.
