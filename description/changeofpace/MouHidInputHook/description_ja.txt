本プロジェクトは、MouHid デバイスオブジェクト内の CONNECT_DATA オブジェクトをフックする Windows カーネルドライバであり、HID USB マウスのデバイススタックを変更せずにマウス入力パケットのフィルタリング、改変、注入を行います。
Moufiltr の手法を模倣し、マウスデータをクラスデータキューへ転送する ClassService コールバックを傍受することで、デバイススタックをアンロードせずに安全にアンフックでき、PnP 通知によるデバイス変更にも対応します。
フィルタデバイスオブジェクトのアタッチを避け、未ドキュメントの CONNECT_DATA フィールドオフセットをヒューリスティックに解決するため、PatchGuard セーフで比較的ステルス性があります。
主に、カーネルレベルのマウス入力傍受や入力シミュレーション検出を研究するアンチチート研究者および入力セキュリティ分析者向けです。
