Ce projet est un plugin x64dbg qui force la protection de pages pour les vues mappées en mémoire lorsque NtProtectVirtualMemory échoue.
Il gère les vues mappées avec le type d'allocation SEC_NO_CHANGE, les vues aux paramètres de protection initiale incompatibles, et contourne les mécanismes anti-patching qui exploitent ces limitations de NtProtectVirtualMemory en remappant les vues avec la protection souhaitée.
Le plugin ajoute les commandes ForcePageProtection (fpp) à x64dbg pour outrepasser les restrictions de protection mémoire pendant l'analyse dynamique.
Il est principalement utile aux reverse engineers et aux analystes anti-cheat qui doivent patcher des régions mémoire protégées par des techniques anti-tamper abusant des restrictions sur les vues mappées.
