Este proyecto es un plugin de x64dbg que fuerza la protección de páginas en vistas mapeadas en memoria cuando NtProtectVirtualMemory falla.
Gestiona vistas mapeadas con el tipo de asignación SEC_NO_CHANGE, vistas con ajustes de protección inicial incompatibles, y derrota mecanismos anti-patching que explotan estas limitaciones de NtProtectVirtualMemory reasignando las vistas con la protección deseada.
El plugin añade los comandos ForcePageProtection (fpp) a x64dbg para anular las restricciones de protección de memoria durante el análisis dinámico.
Es especialmente útil para ingenieros inversos y analistas de anti-cheat que necesitan parchear regiones de memoria protegidas por técnicas anti-tamper que abusan de las restricciones de las vistas mapeadas en memoria.
