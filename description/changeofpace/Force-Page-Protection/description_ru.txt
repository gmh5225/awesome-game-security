Этот проект — плагин для x64dbg, который принудительно задаёт защиту страниц для отображённых в память представлений (views), когда NtProtectVirtualMemory завершается ошибкой.
Он обрабатывает представления, отображённые с типом выделения SEC_NO_CHANGE, представления с несовместимыми начальными настройками защиты, и обходит механизмы антипатчинга, эксплуатирующие эти ограничения NtProtectVirtualMemory, путём повторного отображения представлений с нужной защитой.
Плагин добавляет в x64dbg команды ForcePageProtection (fpp) для обхода ограничений защиты памяти при динамическом анализе.
Он в первую очередь полезен реверс-инженерам и аналитикам античита, которым нужно патчить области памяти, защищённые антитампер-техниками, злоупотребляющими ограничениями memory mapped views.
