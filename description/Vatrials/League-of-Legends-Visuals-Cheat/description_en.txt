This project is an internal League of Legends cheat that injects a DLL into the game client to provide gameplay and visual modifications. It is written primarily in C++ as a Visual Studio solution, with a Windows injector (including C++/CLI UI) that loads a DLL such as R3nzSkin into League of Legends.exe. Core features include a skin changer and skin database, an orbwalker for automated attack targeting, spell prediction and champion-specific scripting (for example Morgana), and a zoom hack that adjusts camera limits. Rendering and overlays use ImGui with DirectX 9, while memory interaction relies on hardcoded game offsets, Microsoft Detours, and VMT-style hooks. The primary use case is reverse engineering and studying League of Legends client internals, injection, and cheat techniques for game-security research.
