本專案是一款英雄聯盟內部作弊工具，透過向遊戲用戶端注入 DLL 來提供玩法與視覺修改。主要以 C++ 撰寫，作為 Visual Studio 解決方案，並附帶 Windows 注入器（含 C++/CLI 介面），可將例如 R3nzSkin 的 DLL 載入 League of Legends.exe。核心功能包括皮膚更換器與皮膚資料庫、用於自動攻擊鎖定的 orbwalker、法術預測與英雄專屬腳本（例如魔甘娜），以及可調整鏡頭限制的縮放作弊。渲染與疊加層使用 ImGui 與 DirectX 9，記憶體互動則依賴硬編碼遊戲偏移、Microsoft Detours 以及 VMT 風格鉤子。主要用途是逆向工程與研究英雄聯盟用戶端內部機制、注入及作弊技術，服務於遊戲安全研究。
