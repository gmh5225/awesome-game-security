本项目是一款英雄联盟内部作弊工具，通过向游戏客户端注入 DLL 实现玩法与视觉修改。主体以 C++ 编写，作为 Visual Studio 解决方案，并附带 Windows 注入器（含 C++/CLI 界面），可将诸如 R3nzSkin 的 DLL 加载到 League of Legends.exe。核心功能包括皮肤更换器与皮肤数据库、用于自动攻击锁定的 orbwalker、法术预测与英雄专属脚本（例如莫甘娜），以及可调整镜头限制的缩放作弊。渲染与叠加层使用 ImGui 与 DirectX 9，内存交互则依赖硬编码游戏偏移、Microsoft Detours 以及 VMT 风格钩子。主要用途是逆向工程与研究英雄联盟客户端内部机制、注入及作弊技术，服务于游戏安全研究。
