Этот проект — специализированная утилита дампа для Xbox 360, собирающая данные консоли, необходимые для рабочих процессов низкоуровневой эмуляции. Она реализована преимущественно на C/C++ и включает код для извлечения fuse-ключей, содержимого загрузчика и артефактов, связанных с NAND. Инструмент рассчитан на модифицированные розничные или разработческие среды с привилегированным доступом. Основное применение — реверс-инжиниринг консолей, сохранение данных и исследования по подготовке эмуляторов.
