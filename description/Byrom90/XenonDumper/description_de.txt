Dieses Projekt ist ein maßgeschneidertes Xbox-360-Dumping-Dienstprogramm, das Konsolendaten für Low-Level-Emulations-Workflows sammelt. Es ist hauptsächlich in C/C++ implementiert und enthält Code zum Extrahieren von Fuses, Bootloader-Inhalten und NAND-bezogenen Artefakten. Das Tool ist für modifizierte Retail- oder Entwicklerumgebungen mit privilegiertem Zugriff ausgelegt. Der primäre Anwendungsfall ist Reverse Engineering von Konsolen, Konservierung sowie die Vorbereitung von Emulatoren.
