Этот проект создаёт прозрачные системные потоки в Windows, практически невидимые для системной интроспекции, удаляя загруженный модуль из системных таблиц страниц.
Он организует полностью контролируемое собственное адресное пространство с собственной таблицей дескрипторов прерываний (IDT) и предотвращает инспекцию отдельных потоков через немаскируемые прерывания (NMI).
Драйвер ядра Windows поддерживает создание потоков как извне, так и изнутри этого адресного пространства с прямым выполнением функций без макросов и обёрточных вызовов.
Он в основном полезен исследователям безопасности ядра, изучающим скрытое выполнение потоков, манипуляции с таблицами страниц и антифорензические техники в драйверах Windows.
