このプロジェクトは、読み込まれたモジュールをシステムページテーブルから削除することで、システム内省に対してほぼ不可視な透明なシステムスレッドを Windows 上に作成します。
独自の割り込み記述子テーブル（IDT）を持つ完全に制御されたカスタムアドレス空間を構築し、マスク不可能割り込み（NMI）を通じた個々のスレッド検査を防ぎます。
この Windows カーネルドライバは、カスタムアドレス空間の外部および内部からのスレッド作成をサポートし、マクロやラッパー呼び出しなしで関数を直接実行できます。
主に、Windows ドライバにおける隠蔽スレッド実行、ページテーブル操作、アンチフォレンジック手法を研究するカーネルセキュリティ研究者向けです。
