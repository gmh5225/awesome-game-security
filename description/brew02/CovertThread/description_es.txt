Este proyecto crea hilos del sistema transparentes en Windows que son casi invisibles a la introspección del sistema al eliminar un módulo cargado de las tablas de páginas del sistema.
Configura un espacio de direcciones personalizado totalmente controlado con su propia tabla de descriptores de interrupción (IDT) y evita la inspección individual de hilos mediante interrupciones no enmascarables (NMI).
El controlador del kernel de Windows admite la creación de hilos tanto desde fuera como desde dentro del espacio de direcciones personalizado, con ejecución directa de funciones sin necesidad de macros ni llamadas envoltorio.
Resulta especialmente útil para investigadores de seguridad del kernel que estudian la ejecución encubierta de hilos, la manipulación de tablas de páginas y técnicas antiforenses en controladores de Windows.
