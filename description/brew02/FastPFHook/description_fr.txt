Ce projet se concentre sur le pF Hook.
FastPFHook analyse et traduit les instructions assembleur d'une page contenant les instructions de la fonction à accrocher, puis les traduit pour qu'elles s'exécutent sur une page séparée après qu'une exception #PF a été déclenchée et interceptée par un gestionnaire d'exceptions.
Il est principalement utile aux chercheurs bas niveau Windows, Linux et mobile travaillant dans le domaine some tricks / Windows Ring0.
