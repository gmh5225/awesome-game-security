Este proyecto se centra en pF Hook.
FastPFHook analiza y traduce instrucciones ensamblador de una página que contiene las instrucciones de la función que queremos enganchar, y las traduce para ejecutarse en una página separada tras activarse una excepción #PF y ser capturada por un manejador de excepciones.
Resulta especialmente útil para investigadores de bajo nivel en Windows, Linux y móviles que trabajan en el área some tricks / Windows Ring0.
