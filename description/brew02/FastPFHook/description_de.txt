Dieses Projekt konzentriert sich auf pF Hook.
FastPFHook parst und übersetzt Assembler-Anweisungen von einer Seite, die die Anweisungen der zu hookenden Funktion enthält, und übersetzt sie so, dass sie nach Auslösen und Abfangen einer #PF-Exception durch einen Exception-Handler auf einer separaten Seite ausgeführt werden.
Es ist vor allem für Low-Level-Forscher im Windows-, Linux- und Mobile-Bereich nützlich, die im Umfeld von some tricks / Windows Ring0 arbeiten.
