이 프로젝트는 Supervisor Mode Access Prevention(SMAP)과 Supervisor Mode Execution Prevention(SMEP)을 사용해 확장 페이지 테이블(EPT) 훅과 기능적으로 유사한 인라인 훅을 만드는 개념 증명입니다. 하이퍼바이저나 실제 EPT 조작 없이도 CPU 접근 제어 기능을 전용해 스플릿 페이지와 유사한 훅 동작을 구현할 수 있음을 보여 줍니다. 커널 드라이버에는 이러한 훅의 존재를 더 잘 숨기기 위해 소프트웨어 가상화가 어떻게 보완할 수 있는지를 보여주는 제한적인 예시도 포함되어 있습니다. 주로 대체 훅 기법, SMAP/SMEP 남용, 가상화 없는 EPT 훅 에뮬레이션을 연구하는 커널 보안 연구자에게 유용합니다.
