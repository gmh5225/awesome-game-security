Этот проект — proof-of-concept, использующий Supervisor Mode Access Prevention (SMAP) и Supervisor Mode Execution Prevention (SMEP) для создания inline-хуков, функционально похожих на хуки расширенных таблиц страниц (EPT). Он показывает, как средства контроля доступа ЦП можно переиспользовать для поведения, сходного со split-page хуками, без гипервизора и реальной манипуляции EPT. Драйвер ядра также содержит ограниченный пример того, как программная виртуализация может дополнять эти хуки, чтобы лучше скрывать их присутствие. Проект в первую очередь полезен исследователям безопасности ядра, изучающим альтернативные техники хукинга, злоупотребление SMAP/SMEP и эмуляцию EPT-хуков без виртуализации.
