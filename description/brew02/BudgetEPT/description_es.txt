Este proyecto es una prueba de concepto que usa la prevención de acceso en modo supervisor (SMAP) y la prevención de ejecución en modo supervisor (SMEP) para crear hooks en línea funcionalmente similares a los hooks de tablas de páginas extendidas (EPT). Demuestra cómo las funciones de control de acceso de la CPU pueden reutilizarse para lograr un comportamiento de hook tipo split-page sin necesidad de un hipervisor ni de manipulación real de EPT. El controlador del kernel también incluye un ejemplo limitado de cómo la virtualización por software podría complementar estos hooks para ocultar mejor su presencia. Es útil principalmente para investigadores de seguridad del kernel que estudian técnicas de hook alternativas, el abuso de SMAP/SMEP y la emulación de hooks EPT sin virtualización.
