本プロジェクトは、Supervisor Mode Access Prevention（SMAP）と Supervisor Mode Execution Prevention（SMEP）を利用し、拡張ページテーブル（EPT）フックと機能的に類似したインラインフックを実現する概念実証です。ハイパーバイザーや実際の EPT 操作を必要とせず、CPU のアクセス制御機能を転用してスプリットページ風のフック動作を実現できることを示します。カーネルドライバには、これらのフックの存在をより隠蔽するためにソフトウェア仮想化がどう補完し得るかを示す限定的な例も含まれています。主に、代替フック手法、SMAP/SMEP の悪用、仮想化なしでの EPT フックエミュレーションを研究するカーネルセキュリティ研究者向けです。
