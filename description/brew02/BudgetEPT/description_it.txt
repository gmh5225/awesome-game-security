Questo progetto è una proof-of-concept che usa la prevenzione dell'accesso in modalità supervisor (SMAP) e la prevenzione dell'esecuzione in modalità supervisor (SMEP) per creare hook inline funzionalmente simili agli hook delle extended page table (EPT). Mostra come le funzioni di controllo degli accessi della CPU possano essere riproposte per ottenere un comportamento di hook di tipo split-page senza richiedere un hypervisor né una manipolazione reale delle EPT. Il driver kernel include anche un esempio limitato di come la virtualizzazione software possa integrare questi hook per nasconderne meglio la presenza. È utile soprattutto ai ricercatori di sicurezza del kernel che studiano tecniche di hooking alternative, l'abuso di SMAP/SMEP e l'emulazione di hook EPT senza virtualizzazione.
