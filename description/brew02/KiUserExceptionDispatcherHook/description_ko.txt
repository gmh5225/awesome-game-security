이 프로젝트는 ntdll의 .mrdata 섹션에 저장된 Wow64PrepareForException 함수 포인터를 수정하여, 커널에서 호출되는 초기 사용자 모드 예외 디스패처 KiUserExceptionDispatcher를 후킹합니다.
보호된 MRDATA 섹션 잠금 해제에는 LdrProtectMrdata를 사용하고, ntdll 내 대상 함수 포인터를 동적으로 찾기 위해 Zydis 디스어셈블러를 사용합니다.
이 C++ 구현은 전통적인 VEH 체인 조작을 피하는 대안적 예외 핸들러 후킹 기법을 보여 줍니다.
사용자 모드 예외 디스패치 내부 구조와 은밀한 후크 설치 방법을 연구하는 Windows 커널 및 안티치트 연구자에게 주로 유용합니다.
