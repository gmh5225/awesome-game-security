Este proyecto engancha KiUserExceptionDispatcher, el despachador inicial de excepciones en modo usuario llamado desde el kernel, modificando el puntero de función Wow64PrepareForException almacenado en la sección .mrdata de ntdll.
Usa LdrProtectMrdata para desbloquear la sección MRDATA protegida y el desensamblador Zydis para localizar dinámicamente los punteros de función objetivo dentro de ntdll.
La implementación en C++ demuestra una técnica alternativa de enganche de manejadores de excepciones que evita la manipulación tradicional de la cadena VEH.
Resulta especialmente útil para investigadores de kernel de Windows y anti-cheat que estudian los internos del despacho de excepciones en modo usuario y métodos discretos de instalación de hooks.
