Этот проект перехватывает KiUserExceptionDispatcher — первичный диспетчер исключений в пользовательском режиме, вызываемый из ядра, — изменяя указатель на функцию Wow64PrepareForException, хранящийся в секции .mrdata модуляки ntdll.
Он использует LdrProtectMrdata для снятия защиты с секции MRDATA и дизассемблер Zydis для динамического поиска целевых указателей на функции внутри ntdll.
Реализация на C++ демонстрирует альтернативную технику перехвата обработчиков исключений, избегающую традиционного вмешательства в цепочку VEH.
Она в основном полезна исследователям ядра Windows и античита, изучающим внутреннее устройство диспетчеризации исключений в пользовательском режиме и скрытные методы установки хуков.
