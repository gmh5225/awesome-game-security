本プロジェクトは、ntdll の .mrdata セクションに格納された Wow64PrepareForException 関数ポインタを改変することで、カーネルから呼び出される初期ユーザーモード例外ディスパッチャ KiUserExceptionDispatcher をフックします。
保護された MRDATA セクションのロック解除には LdrProtectMrdata を用い、ntdll 内の対象関数ポインタの動的特定には Zydis ディスアセンブラを使用します。
この C++ 実装は、従来の VEH チェーン操作を避ける代替的な例外ハンドラ・フッキング手法を示します。
ユーザーモード例外ディスパッチの内部構造やステルスなフック導入手法を研究する Windows カーネルおよびアンチチート研究者に主に役立ちます。
