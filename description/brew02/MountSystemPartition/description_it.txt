Questo progetto mostra come montare in modo programmatico la partizione di sistema EFI nascosta su Windows usando solo le API di Windows.
Fornisce un esempio chiaro in C++ di enumerazione e montaggio delle partizioni senza strumenti esterni o utilità da riga di comando con privilegi elevati.
L’implementazione serve come riferimento per accedere alla partizione di sistema dal codice in modalità utente sui sistemi Windows.
È soprattutto utile per i ricercatori di sicurezza UEFI e gli sviluppatori a livello di sistema che necessitano di accesso programmatico alla partizione di sistema EFI.
