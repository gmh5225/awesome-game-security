Este proyecto demuestra cómo montar de forma programática la partición de sistema EFI oculta en Windows usando solo la API de Windows.
Ofrece un ejemplo limpio en C++ de enumeración y montaje de particiones sin herramientas externas ni utilidades de línea de comandos con privilegios elevados.
La implementación sirve como referencia para acceder a la partición del sistema desde código en modo usuario en sistemas Windows.
Resulta especialmente útil para investigadores de seguridad UEFI y desarrolladores a nivel de sistema que necesitan acceso programático a la partición de sistema EFI.
