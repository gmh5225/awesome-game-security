Этот проект показывает, как программно смонтировать скрытый системный раздел EFI в Windows, используя только Windows API.
Он предоставляет аккуратный пример на C++ по перечислению и монтированию разделов без внешних инструментов и утилит командной строки с повышенными привилегиями.
Реализация служит справочным материалом для доступа к системному разделу из кода пользовательского режима в Windows.
В первую очередь проект полезен исследователям безопасности UEFI и системным разработчикам, которым нужен программный доступ к системному разделу EFI.
