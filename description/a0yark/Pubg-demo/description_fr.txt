Ce projet est une démo de cheat externe pour PUBG, écrite en C++ avec le SDK Unreal Engine.
Il comprend un CppSDK dumpé avec des en-têtes de classes générés, des utilitaires bone helper pour le rendu du squelette, un framework ESP et aimbot, ainsi qu’une overlay DirectX 11 basée sur ImGui pour le rendu.
Le module de cheat injecté en DLL lit la mémoire du jeu et gère les overlays visuels via un pipeline de rendu séparé.
Il est surtout utile aux chercheurs en sécurité des jeux qui étudient les architectures de cheats basées sur le SDK Unreal Engine et les techniques d’overlay externe.
