本プロジェクトは、C++ と Unreal Engine SDK で構築された PUBG 向けの外部チートデモです。
ダンプ済みの CppSDK と生成されたクラスヘッダー、スケルトン描画用のボーンヘルパー、ESP とエイムボットのフレームワーク、および描画用の ImGui ベース DirectX 11 オーバーレイが含まれます。
DLL インジェクションされたチートモジュールがゲームメモリの読み取りを行い、別系統のレンダリングパイプラインを通じて視覚オーバーレイを描画します。
主に Unreal Engine SDK ベースのチート構成や外部オーバーレイ技術を研究するゲームセキュリティ研究者向けです。
