Este proyecto es una demo de cheat externo para PUBG, construida con C++ y el SDK de Unreal Engine.
Incluye un CppSDK volcado con encabezados de clases generados, utilidades bone helper para el renderizado del esqueleto, un framework de ESP y aimbot, y un overlay de DirectX 11 basado en ImGui para el renderizado.
El módulo de cheat inyectado por DLL lee la memoria del juego y gestiona los overlays visuales mediante un pipeline de renderizado separado.
Resulta especialmente útil para investigadores de seguridad en juegos que estudian arquitecturas de cheats basadas en el SDK de Unreal Engine y técnicas de overlay externo.
