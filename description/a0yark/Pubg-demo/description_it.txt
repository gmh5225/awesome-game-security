Questo progetto è una demo di cheat esterno per PUBG, realizzata in C++ con l’SDK di Unreal Engine.
Include un CppSDK dumpato con header di classi generati, utility bone helper per il rendering dello scheletro, un framework ESP e aimbot, e un overlay DirectX 11 basato su ImGui per il rendering.
Il modulo cheat iniettato tramite DLL gestisce la lettura della memoria di gioco e gli overlay visivi attraverso una pipeline di rendering separata.
È soprattutto utile ai ricercatori di sicurezza nei videogiochi che studiano architetture di cheat basate sull’SDK di Unreal Engine e tecniche di overlay esterno.
