Этот проект — демо внешнего чита для PUBG на C++ с использованием Unreal Engine SDK.
В него входят дампнутый CppSDK со сгенерированными заголовками классов, утилиты bone helper для отрисовки скелета, фреймворк ESP и аимбота, а также оверлей на ImGui и DirectX 11 для рендеринга.
Модуль чита, внедряемый через DLL, читает память игры и выводит визуальные оверлеи через отдельный конвейер рендеринга.
Он в первую очередь полезен исследователям безопасности игр, изучающим архитектуры читов на базе Unreal Engine SDK и техники внешних оверлеев.
