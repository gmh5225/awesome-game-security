本プロジェクトは、DirectX 11 の共有バッファとコンピュートシェーダーを用いた GPU 支援プロセスホローイングの概念実証です。
C++ コードベースは、shellcode をエンコードして GPU 共有バッファへアップロードする Injector と、HLSL コンピュートシェーダーで GPU 上のペイロードをデコードしたうえで CPU 上で実行する Target から構成されます。
プロセス間同期は、名前付きイベント、共有メモリ、および DXGI 共有ハンドルによって行われます。
主に、GPU ベースのペイロード転送、コンピュートシェーダーの悪用、および新しいコードインジェクション手法を研究するセキュリティ研究者向けです。
