이 프로젝트는 DirectX 11 공유 버퍼와 컴퓨트 셰이더를 이용한 GPU 지원 프로세스 할로잉(process hollowing)의 개념 증명입니다.
C++ 코드베이스는 shellcode를 인코딩해 GPU 공유 버퍼에 업로드하는 Injector와, HLSL 컴퓨트 셰이더로 GPU에서 페이로드를 디코딩한 뒤 CPU에서 실행하는 Target으로 구성됩니다.
프로세스 간 동기화는 명명된 이벤트, 공유 메모리, DXGI 공유 핸들을 통해 처리됩니다.
주로 GPU 기반 페이로드 전송, 컴퓨트 셰이더 악용, 새로운 코드 인젝션 기법을 연구하는 보안 연구자에게 유용합니다.
