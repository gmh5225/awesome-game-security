Este proyecto es una prueba de concepto de process hollowing asistido por GPU mediante buffers compartidos de DirectX 11 y compute shaders.
El código en C++ incluye un Injector que codifica shellcode y lo sube a un buffer compartido en la GPU, y un Target que usa un compute shader HLSL para decodificar la carga en la GPU antes de ejecutarla en la CPU.
La sincronización entre procesos se gestiona con eventos con nombre y memoria compartida mediante handles compartidos de DXGI.
Resulta especialmente útil para investigadores de seguridad que estudian el transporte de cargas vía GPU, el abuso de compute shaders y técnicas novedosas de inyección de código.
