Questo progetto è una proof-of-concept di process hollowing assistito dalla GPU mediante buffer condivisi DirectX 11 e compute shader.
Il codice C++ comprende un Injector che codifica lo shellcode e lo carica su un buffer GPU condiviso, e un Target che usa un compute shader HLSL per decodificare il payload sulla GPU prima di eseguirlo sulla CPU.
La sincronizzazione tra processi avviene tramite eventi denominati e memoria condivisa con handle condivisi DXGI.
È pensato soprattutto per i ricercatori di sicurezza che studiano il trasporto di payload via GPU, l’abuso dei compute shader e tecniche innovative di code injection.
