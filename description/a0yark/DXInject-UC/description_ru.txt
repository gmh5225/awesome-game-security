Этот проект — proof-of-concept GPU-assisted process hollowing с использованием общих буферов DirectX 11 и compute shaders.
Кодовая база на C++ включает Injector, который кодирует shellcode и загружает его в общий GPU-буфер, и Target, который с помощью HLSL compute shader декодирует полезную нагрузку на GPU, а затем выполняет её на CPU.
Межпроцессная синхронизация реализована через именованные события и разделяемую память с DXGI shared handles.
Проект в первую очередь полезен исследователям безопасности, изучающим передачу payload через GPU, злоупотребление compute shaders и новые техники внедрения кода.
