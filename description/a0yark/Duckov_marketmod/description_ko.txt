이 프로젝트는 Duckov용 게임 마켓 모드로, 등록·구매·이행 워크플로를 갖춘 플리마켓 시스템을 구현합니다.
C# 코드베이스에는 UI 관리가 포함된 코어 모듈, Steam 인증 네트워크 API 클라이언트, WebSocket 통신, Harmony 기반 게임 패치, 버전 확인 및 자동 업데이트를 지원하는 모드 로더가 포함됩니다.
게임 내부 구조를 리버스 엔지니어링하여 개발되었으며, 가독성과 2차 개발을 위해 변수를 정리하고 의미 있게 이름을 바꿨습니다.
온라인 마켓플레이스 연동과 리버스 엔지니어링된 모드 아키텍처를 연구하는 게임 모딩 연구자에게 주로 유용합니다.
