Este proyecto es un mod de mercado para Duckov que implementa un sistema de mercadillo con flujos de listado, compra y cumplimiento.
La base de código en C# incluye un módulo central con gestión de UI, un cliente de API de red con autenticación de Steam, comunicación WebSocket, parches del juego basados en Harmony y un cargador de mods con comprobación de versión y actualización automática.
Se desarrolló mediante ingeniería inversa de los internos del juego, con variables saneadas y renombradas semánticamente para facilitar la lectura y el desarrollo secundario.
Resulta especialmente útil para investigadores de modding de juegos que estudian la integración de mercados en línea y arquitecturas de mods obtenidas por ingeniería inversa.
