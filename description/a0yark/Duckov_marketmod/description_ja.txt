本プロジェクトは Duckov 向けのゲームマーケット MOD で、出品・購入・履行のワークフローを備えたフリーマーケットシステムを実装しています。
C# コードベースには、UI 管理を含むコアモジュール、Steam 認証付きネットワーク API クライアント、WebSocket 通信、Harmony ベースのゲームパッチ、およびバージョンチェックと自動更新に対応した MOD ローダーが含まれます。
ゲーム内部のリバースエンジニアリングを通じて開発され、可読性と二次開発のため変数は整理され意味のある名前に変更されています。
オンラインマーケットプレイス連携やリバースエンジニアリングされた MOD アーキテクチャを研究するゲーム MOD 研究者にとって主に有用です。
