Ce projet est un outil de radar et d'ESP basé sur le DMA pour Arc Raiders, qui lit la mémoire du jeu de manière externe via du matériel FPGA en utilisant MemProcFS.
Il utilise Unicorn Engine pour émuler les fonctions de déchiffrement du jeu afin de résoudre des pointeurs obfuscqués tels que GWorld, GameInstance, CameraManager et BoneBase, sans reverse engineering manuel de la logique de déchiffrement.
La base de code C++ inclut l'initialisation DMA, la découverte de processus et de modules, le déchiffrement de pointeurs par émulation, ainsi qu'un framework pour l'itération sur les joueurs et les acteurs.
Il est surtout utile aux chercheurs en sécurité DMA qui étudient la résolution de pointeurs assistée par émulation et les techniques de lecture mémoire externe basées sur le matériel.
