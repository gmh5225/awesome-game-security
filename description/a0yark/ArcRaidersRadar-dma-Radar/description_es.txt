Este proyecto es una herramienta de radar y ESP basada en DMA para Arc Raiders que lee la memoria del juego de forma externa mediante hardware FPGA usando MemProcFS.
Utiliza Unicorn Engine para emular las funciones de descifrado del propio juego y resolver punteros ofuscados como GWorld, GameInstance, CameraManager y BoneBase, sin revertir manualmente la lógica de descifrado.
El código en C++ incluye inicialización DMA, descubrimiento de procesos y módulos, descifrado de punteros basado en emulación y un framework para iterar sobre jugadores y actores.
Resulta especialmente útil para investigadores de seguridad DMA que estudian la resolución de punteros asistida por emulación y las técnicas de lectura de memoria externa basadas en hardware.
