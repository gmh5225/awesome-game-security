Этот проект — внутренняя реализация ESP и аимбота на C++ для игры Smite.
В него входят набор заголовков SDK, полученных обратной разработкой, вспомогательные средства для хуков и утилитарный код для работы со структурами игр эпохи Unreal Engine 3.
Примеры демонстрируют захват целей и отрисовку ESP на экране, попутно документируя особенности указателей, специфичные для этой игры.
Он в первую очередь предназначен для изучающих game hacking, которым нужна опорная база для более старых проектов на UE3.
