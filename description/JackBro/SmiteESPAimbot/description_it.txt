Questo progetto è un'implementazione interna in C++ di ESP e aimbot per il gioco Smite.
Include un set di header SDK ottenuti tramite reverse engineering, helper per gli hook e codice di utilità per interagire con le strutture di gioco dell'era Unreal Engine 3.
Le funzionalità di esempio mostrano l'acquisizione dei target e il disegno ESP a schermo, documentando al contempo peculiarità dei puntatori specifiche del gioco.
È rivolto principalmente a chi studia il game hacking e cerca una base di riferimento per titoli UE3 più datati.
