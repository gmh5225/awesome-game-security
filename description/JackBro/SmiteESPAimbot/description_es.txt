Este proyecto es una implementación interna en C++ de ESP y aimbot para el juego Smite.
Incluye un conjunto de encabezados SDK obtenidos por ingeniería inversa, ayudantes de hooks y código utilitario para interactuar con estructuras de juegos de la era de Unreal Engine 3.
Las funciones de ejemplo demuestran la adquisición de objetivos y el dibujo de ESP en pantalla, documentando al mismo tiempo peculiaridades de punteros específicas del juego.
Está dirigido principalmente a quienes aprenden game hacking y buscan una base de referencia para títulos UE3 más antiguos.
