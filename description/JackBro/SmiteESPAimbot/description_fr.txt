Ce projet est une implémentation interne en C++ d'ESP et d'aimbot pour le jeu Smite.
Il comprend un ensemble d'en-têtes SDK rétro-ingénierés, des helpers de hooks et du code utilitaire pour interagir avec les structures de jeux de l'ère Unreal Engine 3.
Les fonctionnalités d'exemple illustrent l'acquisition de cibles et le dessin d'ESP à l'écran, tout en documentant les particularités des pointeurs propres au jeu.
Il s'adresse principalement aux apprenants du game hacking qui cherchent une base de référence pour les titres UE3 plus anciens.
