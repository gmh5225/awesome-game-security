Este proyecto es una compilación de QEMU/KVM con parches de sigilo orientada a ocultar ante el software invitado los artefactos habituales de VirtualBox, VMware, Bochs y QEMU/KVM.
Se basa en una instantánea git de QEMU upstream con mitigaciones de huella anti-VM para analizar software hostil a las VM o con anti-trampas, no como hipervisor de uso diario.
Resulta especialmente útil para investigadores de seguridad en juegos que estudian comprobaciones anti-VM y ejecutan VM de análisis frente a títulos protegidos.
