Questo progetto è una build QEMU/KVM con patch per la furtività, pensata per nascondere al software guest gli artefatti comuni di VirtualBox, VMware, Bochs e QEMU/KVM.
Si basa su uno snapshot git di QEMU upstream con mitigazioni anti-impronta VM per analizzare software ostile alle VM o protetto da anti-cheat, non come hypervisor per l’uso quotidiano.
È soprattutto utile ai ricercatori di sicurezza dei giochi che studiano i controlli anti-VM e eseguono VM di analisi su titoli protetti.
