Ce projet est un pack d'en-têtes SDK CS2 généré qui expose les types du moteur, les énumérations et les interfaces des sous-systèmes. Il est composé d'en-têtes C++ couvrant les principaux domaines du moteur tels que le client, le rendu, le réseau, schema, panorama, les particules et la physique. Le contenu est organisé sous forme de définitions de référence utilisables par des outils externes et du code de recherche. Son cas d'usage principal est le support de la rétro-ingénierie et les fondations techniques pour les outils d'analyse de Counter-Strike 2.
