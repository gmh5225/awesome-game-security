Этот проект — сгенерированный набор заголовочных файлов SDK для CS2, предоставляющий типы движка, перечисления и интерфейсы подсистем. Он состоит из заголовков C++, охватывающих основные области движка: клиент, рендеринг, сеть, schema, panorama, частицы и физику. Содержимое организовано как справочные определения, которые можно использовать во внешних инструментах и исследовательском коде. Основное применение — поддержка реверс-инжиниринга и техническая база для инструментов анализа Counter-Strike 2.
