Este proyecto es un paquete de encabezados SDK de CS2 generado que expone tipos del motor, enumeraciones e interfaces de subsistemas. Está compuesto por encabezados C++ que cubren dominios principales del motor como cliente, renderizado, red, schema, panorama, partículas y física. El contenido se organiza como definiciones de referencia que pueden consumir herramientas externas y código de investigación. Su caso de uso principal es el soporte de ingeniería inversa y la base técnica para herramientas de análisis de Counter-Strike 2.
