Этот проект — утилита для Windows, которая преобразует игры в оконном режиме в сеансы безрамочного полноэкранного режима.
Она написана на C# и .NET с интерфейсом WinForms и отдельным логическим слоем для отслеживания процессов, управления окнами и поведения при запуске.
В кодовой базе есть языковые пакеты, компоненты, связанные со Steam, и скрипты установщика для автономного развёртывания.
Основной сценарий использования — повысить удобство игр на ПК за счёт более быстрого переключения задач и меньшего числа проблем по сравнению с эксклюзивным полноэкранным режимом.
