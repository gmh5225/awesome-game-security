Ce projet est un outil orienté Ghidra qui récupère les informations de symboles Delphi à partir de binaires compilés.
Il reconstruit les signatures de fonctions qualifiées, les métadonnées de paramètres et le contexte des tables virtuelles à partir des métadonnées du compilateur embarquées, puis mappe les résultats vers les types de données Ghidra.
L’implémentation est un script Python pour pyghidra et couvre un large éventail de versions modernes de Delphi.
Son cas d’usage principal est la rétro-ingénierie de malwares et de logiciels hérités, où la récupération de noms sémantiques accélère l’analyse.
