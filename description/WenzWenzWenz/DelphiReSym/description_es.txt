Este proyecto es una herramienta orientada a Ghidra que recupera información de símbolos Delphi a partir de binarios compilados.
Reconstruye firmas de funciones cualificadas, metadatos de parámetros y el contexto de tablas virtuales a partir de metadatos del compilador embebidos, y luego mapea los resultados a tipos de datos de Ghidra.
La implementación es un script de Python para pyghidra y cubre una amplia gama de versiones modernas de Delphi.
Su caso de uso principal es la ingeniería inversa de malware y software legado, donde recuperar nombres semánticos acelera el análisis.
