Questo progetto è uno strumento orientato a Ghidra che recupera le informazioni sui simboli Delphi da binari compilati.
Ricostruisce firme di funzioni qualificate, metadati dei parametri e il contesto delle tabelle virtuali a partire dai metadati del compilatore incorporati, quindi mappa i risultati sui tipi di dati di Ghidra.
L’implementazione è uno script Python per pyghidra e mira a un’ampia gamma di versioni moderne di Delphi.
Il caso d’uso principale è il reverse engineering di malware e software legacy, dove il recupero di nomi semantici accelera l’analisi.
