Этот проект — инструмент для Ghidra, который восстанавливает символьную информацию Delphi из скомпилированных бинарников.
Он реконструирует квалифицированные сигнатуры функций, метаданные параметров и контекст виртуальных таблиц из встроенных метаданных компилятора, затем сопоставляет результаты с типами данных Ghidra.
Реализация — скрипт на Python для pyghidra; поддерживается широкий диапазон современных версий Delphi.
Основной сценарий использования — реверс-инжиниринг вредоносного и устаревшего ПО, где восстановление смысловых имён ускоряет анализ.
