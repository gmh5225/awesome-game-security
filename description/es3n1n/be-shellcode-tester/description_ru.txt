Этот проект — тестовая среда для выполнения и анализа shellcode-модулей античита BattlEye в контролируемой песочнице. Он загружает дампы сканирующих модулей BattlEye, эмулирует ожидаемое окружение выполнения и журналирует их поведение, включая сканирование памяти, проверки хешей и процедуры обнаружения. C++-тестер помогает исследователям понять покрытие детекции BattlEye. Предназначен для исследователей античита, занимающихся реверс-инжинирингом shellcode-модулей сканирования BattlEye.
