Этот проект — плагин для IDA Pro, интегрирующий учёт времени WakaTime в рабочий процесс IDA. Он автоматически отправляет в WakaTime данные о времени, потраченном на задачи реверс-инжиниринга, отслеживая, какие бинарные файлы анализируются и сколько времени. Python-плагин работает в фоне и передаёт heartbeat-события в API WakaTime. Он предназначен для реверс-инженеров, которые используют WakaTime для отслеживания продуктивности в инструментах разработки и анализа.
