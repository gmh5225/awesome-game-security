Este proyecto es un plugin de IDA Pro que integra el seguimiento de tiempo de WakaTime en el flujo de trabajo de IDA. Informa automáticamente a WakaTime del tiempo dedicado a tareas de ingeniería inversa, registrando qué binarios se analizan y durante cuánto tiempo. El plugin en Python se ejecuta en segundo plano y envía eventos heartbeat a la API de WakaTime. Está dirigido a ingenieros inversos que usan WakaTime para medir su productividad en herramientas de desarrollo y análisis.
