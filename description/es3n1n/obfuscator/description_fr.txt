Ce projet est un obfuscateur binaire PE en C++ qui applique plusieurs passes de transformation aux exécutables x86-64 compilés. Il met en œuvre l’aplatissement du flot de contrôle, l’insertion de code fantôme, la mutation d’instructions, l’obfuscation des imports et des techniques anti-désassemblage directement sur les fichiers PE, sans nécessiter l’accès au code source. L’obfuscateur opère au niveau binaire via désassemblage et réassemblage. Il s’adresse aux chercheurs en protection logicielle qui étudient les techniques d’obfuscation post-compilation et la transformation de code au niveau binaire.
