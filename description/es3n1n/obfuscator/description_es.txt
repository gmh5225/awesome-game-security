Este proyecto es un ofuscador binario PE en C++ que aplica múltiples pases de transformación a ejecutables x86-64 compilados. Implementa aplanamiento del flujo de control, inserción de código basura, mutación de instrucciones, ofuscación de importaciones y trucos anti-desensamblado directamente sobre archivos PE, sin requerir acceso al código fuente. El ofuscador opera a nivel binario mediante desensamblado y reensamblado. Está orientado a investigadores de protección de software que estudian técnicas de ofuscación postcompilación y transformación de código a nivel binario.
