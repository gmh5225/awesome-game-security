Questo progetto è un offuscatore binario PE in C++ che applica più passate di trasformazione a eseguibili x86-64 compilati. Implementa l’appiattimento del flusso di controllo, l’inserimento di junk code, la mutazione delle istruzioni, l’offuscamento delle import e trucchi anti-disassemblaggio direttamente sui file PE, senza richiedere l’accesso al codice sorgente. L’offuscatore opera a livello binario tramite disassemblaggio e riassemblaggio. È rivolto ai ricercatori di protezione del software che studiano tecniche di offuscamento post-compilazione e trasformazione del codice a livello binario.
