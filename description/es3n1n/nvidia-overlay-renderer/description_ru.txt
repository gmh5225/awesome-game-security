Этот проект — Windows-рендерер оверлея, который перехватывает окно оверлея NVIDIA GeForce Experience для отрисовки пользовательского контента. Он находит окно оверлея NVIDIA, получает его контекст отрисовки и рисует меню ImGui или информацию ESP через уже доверенную поверхность оверлея. Такой подход позволяет не создавать новые окна оверлея, которые обнаружили бы античиты. Предназначен для исследователей игровой безопасности, изучающих техники перехвата оверлея и обход обнаружения оверлеев античитами.
