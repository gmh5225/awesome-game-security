Este proyecto es un renderizador de overlay para Windows que secuestra la ventana de overlay de NVIDIA GeForce Experience para dibujar contenido personalizado. Localiza la ventana de overlay de NVIDIA, obtiene su contexto de renderizado y dibuja menús ImGui o información ESP a través de la superficie de overlay confiable existente. Este enfoque evita crear nuevas ventanas de overlay que los sistemas anti-trampas detectarían. Está orientado a investigadores de seguridad en videojuegos que estudian técnicas de secuestro de overlay y la evasión de la detección de overlays por anti-trampas.
