Этот проект — PE-пакер для Windows x64, который сжимает исполняемые файлы с помощью LZMA и запускает их через пользовательский stub загрузчика. Написан в основном на C++, поддерживает упаковку на основе ресурсов и секций, обработку иконок и манифестов, а также надёжную обработку релокаций, импортов и TLS. Решение включает вспомогательные инструменты для встраивания stub загрузчика и автоматизации постсборочной интеграции. Основное применение — исследования защиты исполняемых файлов и эксперименты с загрузчиками в стиле manual-mapping.
