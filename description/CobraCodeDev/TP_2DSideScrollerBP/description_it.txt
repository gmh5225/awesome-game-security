Questo progetto è un template Blueprint di Unreal Engine 5 per creare giochi 2D a scorrimento laterale. Fornisce una struttura di progetto iniziale, asset di esempio e indicazioni di configurazione per flussi di lavoro in stile Paper2D. Il template è versionato per le release più recenti di UE5 e fa riferimento a varianti avanzate opzionali che usano il plugin PaperZD per gli strumenti di animazione. È rivolto agli sviluppatori di giochi che prototipano gameplay a scorrimento laterale, non alla ricerca sulla sicurezza.
