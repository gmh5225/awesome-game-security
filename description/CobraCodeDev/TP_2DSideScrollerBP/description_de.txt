Dieses Projekt ist eine Unreal Engine 5-Blueprint-Vorlage zum Erstellen von 2D-Side-Scroller-Spielen. Es bietet eine Starter-Projektstruktur, Beispiel-Assets und Einrichtungsanleitungen für Paper2D-ähnliche Workflows. Die Vorlage ist für neuere UE5-Releases versioniert und verweist auf optionale erweiterte Varianten, die das PaperZD-Plugin für Animationswerkzeuge nutzen. Sie richtet sich an Spieleentwickler, die Side-Scrolling-Gameplay prototypisieren, nicht an Sicherheitsforschung.
