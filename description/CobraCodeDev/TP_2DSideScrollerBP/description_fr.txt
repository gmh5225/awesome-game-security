Ce projet est un modèle Blueprint Unreal Engine 5 pour créer des jeux de plateforme 2D à défilement latéral. Il fournit une structure de projet de démarrage, des ressources d’exemple et des conseils de configuration pour des flux de travail de type Paper2D. Le modèle est versionné pour les versions récentes d’UE5 et fait référence à des variantes avancées optionnelles qui utilisent le plugin PaperZD pour l’animation. Il s’adresse aux développeurs de jeux qui prototypent un gameplay à défilement latéral, et non à la recherche en sécurité.
