Este proyecto es una plantilla Blueprint de Unreal Engine 5 para crear juegos de plataformas 2D de desplazamiento lateral. Proporciona una estructura de proyecto inicial, recursos de ejemplo y orientación de configuración para flujos de trabajo al estilo Paper2D. La plantilla está versionada para versiones más recientes de UE5 y menciona variantes avanzadas opcionales que usan el plugin PaperZD para herramientas de animación. Está dirigida a desarrolladores de juegos que prototipan jugabilidad de side-scroller, no a la investigación de seguridad.
