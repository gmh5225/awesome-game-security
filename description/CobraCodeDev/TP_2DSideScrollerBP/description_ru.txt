Этот проект — шаблон Blueprint для Unreal Engine 5, предназначенный для создания 2D side-scroller игр. Он предоставляет стартовую структуру проекта, примеры ассетов и рекомендации по настройке рабочих процессов в стиле Paper2D. Шаблон версионирован под более новые релизы UE5 и ссылается на опциональные продвинутые варианты, использующие плагин PaperZD для анимации. Он рассчитан на разработчиков игр, прототипирующих side-scrolling геймплей, а не на исследования в области безопасности.
