Ce projet est un environnement de reverse engineering des structures mémoire du noyau, combinant une extension WinDbg et une interface graphique de bureau. Il permet de se connecter à une session de débogage, d'inspecter les layouts mémoire du noyau, de résoudre les pointeurs de fonctions et de générer des vues de types reconstruits. Le code est principalement en C++ et comprend des composants réseau/configuration ainsi que des intégrations d'éditeurs pour des workflows d'analyse interactive. Le cas d'usage principal est le reverse engineering du noyau Windows et la recherche sur les anti-cheats de jeux.
