Questo progetto è un ambiente di reverse engineering delle strutture di memoria del kernel che combina un'estensione WinDbg con una GUI desktop. Consente di collegarsi a una sessione di debug, ispezionare i layout di memoria del kernel, risolvere i puntatori a funzione e generare viste di tipi ricostruiti. Il codice è principalmente in C++ e include componenti di rete/configurazione e integrazioni con editor per flussi di analisi interattiva. Il caso d'uso principale è il reverse engineering del kernel Windows e la ricerca sugli anti-cheat nei giochi.
