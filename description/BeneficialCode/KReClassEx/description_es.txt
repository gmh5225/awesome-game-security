Este proyecto es un entorno de ingeniería inversa de estructuras de memoria del kernel que combina una extensión de WinDbg con una GUI de escritorio. Permite conectarse a una sesión de depuración, inspeccionar layouts de memoria del kernel, resolver punteros a funciones y generar vistas de tipos reconstruidos. El código está principalmente en C++ e incluye componentes de red/configuración e integraciones de editor para flujos de análisis interactivo. El caso de uso principal es la ingeniería inversa del kernel de Windows y la investigación de anti-cheat en juegos.
