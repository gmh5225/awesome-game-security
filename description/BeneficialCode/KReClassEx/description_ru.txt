Этот проект — среда реверса структур памяти ядра, сочетающая расширение WinDbg с настольным GUI. Она позволяет подключаться к сессии отладки, исследовать раскладки памяти ядра, разрешать указатели на функции и формировать реконструированные представления типов. Кодовая база в основном на C++ и включает сетевые/конфигурационные компоненты, а также интеграции с редакторами для интерактивного анализа. Основной сценарий использования — реверс-инжиниринг ядра Windows и исследования игровых античитов.
