Este proyecto es una plataforma de código abierto de anti-rootkit para Windows y análisis de sistemas a bajo nivel. Contiene un amplio ecosistema en C++ de controladores, bibliotecas de kernel, analizadores de símbolos y PE, módulos de monitorización y componentes de UI orientados a la investigación. La cadena de herramientas apunta a versiones modernas de Windows y está diseñada para seguir siendo efectiva trabajando con la infraestructura de símbolos y rutas profundas de inspección en modo kernel y usuario. El caso de uso principal es la investigación de seguridad, la caza de rootkits y el análisis de los internals del anti-cheat.
