Questo progetto è una piattaforma open source di anti-rootkit per Windows e di analisi di sistema a basso livello. Contiene un ampio ecosistema C++ di driver, librerie kernel, parser di simboli e PE, moduli di monitoraggio e componenti UI orientati all'indagine. La toolchain mira alle versioni moderne di Windows ed è progettata per restare efficace lavorando con l'infrastruttura dei simboli e percorsi approfonditi di ispezione in modalità kernel e user-mode. L'uso principale è la ricerca sulla sicurezza, la caccia ai rootkit e l'analisi degli internals degli anti-cheat.
