Этот проект посвящён symlinkCallback.
Он изменяет объект symlink и заменяет строку LinkTarget на функцию обратного вызова, которая вызывается при каждом обращении к symlink.
В основном полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / Windows ring0 callback.
