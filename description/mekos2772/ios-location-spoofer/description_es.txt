Este proyecto se centra en un spoofing de ubicación de red en iOS sin jailbreak: MITM de las respuestas de consulta de Apple Maps; parcheo de BSSID WiFi y coordenadas de CellTower; módulos Surge/Shadowrocket/Loon/Stash/QX; spoof del estado de movimiento; interfaz web para seleccionar ubicación.
Está escrito principalmente en JavaScript y se centra en redes, desarrollo de plugins y modding.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el área de cheats / red iOS / ubicación.
