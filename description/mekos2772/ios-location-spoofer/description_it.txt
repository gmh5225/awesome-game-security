Questo progetto si concentra su uno spoofer di posizione di rete iOS senza jailbreak — MITM delle risposte di lookup di Apple Maps; patch di BSSID WiFi e coordinate CellTower; moduli Surge/Shadowrocket/Loon/Stash/QX; spoof dello stato di movimento; interfaccia web per la selezione della posizione.
È scritto principalmente in JavaScript e si concentra su networking, sviluppo di plugin e modding.
È soprattutto utile a ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'area cheat / rete iOS / posizione.
