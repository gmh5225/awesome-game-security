Ce projet porte sur un spoofeur de localisation réseau iOS sans jailbreak — MITM des réponses de recherche Apple Maps ; patch des BSSID WiFi et des coordonnées CellTower ; modules Surge/Shadowrocket/Loon/Stash/QX ; spoof de l'état de mouvement ; interface web de sélection de position.
Il est principalement écrit en JavaScript et se concentre sur le réseau, le développement de plugins et le modding.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine cheat / réseau iOS / localisation.
