Cette entrée README pointait à l’origine vers radareorg/r2a pour une intégration de modèle de langage local avec radare2, mais cette URL de dépôt n’existe plus sur GitHub.
Le successeur activement maintenu dans l’organisation radareorg est r2ai (rétro-ingénierie basée sur les LLM pour radare2) ; voir archive/radareorg/r2ai.txt pour un instantané de code local.
Elle sert surtout de pointeur aux reverse engineers à la recherche d’outils LLM pour radare2.
