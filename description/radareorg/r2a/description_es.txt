Esta entrada del README apuntaba originalmente a radareorg/r2a para una integración de modelo de lenguaje local con radare2, pero esa URL del repositorio ya no existe en GitHub.
El sucesor mantenido activamente en la organización radareorg es r2ai (reversing basado en LLM para radare2); consulta archive/radareorg/r2ai.txt para una instantánea local del código.
Sirve principalmente como referencia para ingenieros inversos que buscan herramientas LLM de radare2.
