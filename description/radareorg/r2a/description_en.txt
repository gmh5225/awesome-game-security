This README entry originally pointed at radareorg/r2a for a local language-model integration with radare2, but that repository URL no longer exists on GitHub.
The actively maintained successor in the radareorg org is r2ai (LLM-based reversing for radare2); see archive/radareorg/r2ai.txt for a local code snapshot.
It is mainly useful as a pointer for reverse engineers looking for radare2 LLM tooling.
