Эта запись README изначально указывала на radareorg/r2a — интеграцию локальной языковой модели с radare2, но URL этого репозитория больше не существует на GitHub.
Активно поддерживаемый преемник в организации radareorg — r2ai (реверс на базе LLM для radare2); локальный снимок кода см. в archive/radareorg/r2ai.txt.
В основном полезна как указатель для reverse-инженеров, ищущих LLM-инструменты для radare2.
