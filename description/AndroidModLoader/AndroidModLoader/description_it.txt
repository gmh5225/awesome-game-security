Questo progetto è un framework nativo di caricamento mod per Android per iniettare e gestire mod in giochi e applicazioni. È scritto principalmente in C++ con script di build Android NDK e include patching, scrittura in memoria, function hooking e API per mod basate su interfacce. Il codice include componenti di supporto per l'hooking ARM, la gestione delle dipendenze, utilità orientate a IL2CPP e scaffolding di template per mod. È rivolto a sviluppatori di mod per giochi e reverse engineer che realizzano modifiche runtime su Android.
