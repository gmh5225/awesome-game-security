Этот проект — нативный фреймворк загрузчика модов для Android, предназначенный для внедрения и управления модами в играх и приложениях. Он в основном написан на C++ со скриптами сборки Android NDK и включает патчинг, запись в память, перехват функций и API модов на основе интерфейсов. В кодовой базе собраны вспомогательные компоненты для ARM-хукинга, обработки зависимостей, утилит, ориентированных на IL2CPP, и шаблонов-скаффолдинга для модов. Проект рассчитан на разработчиков игровых модов и специалистов по реверс-инжинирингу, создающих runtime-модификации на Android.
