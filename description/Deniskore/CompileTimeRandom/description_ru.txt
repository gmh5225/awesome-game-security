Этот проект предоставляет утилиту генерации случайных чисел на этапе компиляции для кода на C++11. Он использует хеширование и логику генератора на основе constexpr, включая операции в стиле FNV и Murmur3, для получения детерминированных значений во время компиляции. Заголовок предоставляет макросы для 32- и 64-битных констант случайных чисел времени компиляции без вызовов RNG во время выполнения. В основном полезен для низкоуровневых инструментов, лёгких схем обфускации и кода исследований безопасности игр, где выгодна генерация значений на этапе компиляции.
