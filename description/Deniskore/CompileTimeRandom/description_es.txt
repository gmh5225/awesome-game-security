Este proyecto proporciona una utilidad de números aleatorios en tiempo de compilación para código C++11. Usa hashing y lógica de generador basados en constexpr, incluidas operaciones al estilo FNV y Murmur3, para producir valores deterministas durante la compilación. El encabezado expone macros para constantes aleatorias de compilación de 32 y 64 bits sin llamadas RNG en tiempo de ejecución. Resulta útil principalmente para herramientas de bajo nivel, patrones ligeros de ofuscación y código de investigación en seguridad de juegos que se beneficia de la generación de valores en tiempo de compilación.
