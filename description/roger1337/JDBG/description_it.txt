Questo progetto è JDBG, un debugger Java leggero che consente di collegarsi a processi Java/JVM in esecuzione e ispezionarli. Offre hooking dei metodi, impostazione di breakpoint, ispezione delle variabili ed enumerazione delle classi tramite la JVM Debug Interface (JDWP). Lo strumento può essere usato per l'analisi a runtime di client di giochi basati su Java e di applicazioni Java offuscate. È rivolto a reverse engineer Java e ricercatori di sicurezza che analizzano applicazioni basate su JVM.
