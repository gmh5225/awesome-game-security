Ce projet est JDBG, un débogueur Java léger qui permet de s'attacher à des processus Java/JVM en cours d'exécution et de les inspecter. Il offre le hooking de méthodes, la pose de points d'arrêt, l'inspection de variables et l'énumération de classes via l'interface JVM Debug Interface (JDWP). L'outil peut servir à l'analyse à l'exécution de clients de jeux basés sur Java et d'applications Java obscurcies. Il s'adresse aux ingénieurs en reverse engineering Java et aux chercheurs en sécurité qui analysent des applications basées sur la JVM.
