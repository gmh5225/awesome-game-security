Dieses Projekt ist JDBG, ein schlanker Java-Debugger, mit dem man laufende Java/JVM-Prozesse anbinden und untersuchen kann. Über die JVM Debug Interface (JDWP) bietet er Methoden-Hooking, Breakpoint-Setzung, Variableninspektion und Klassenauflistung. Das Tool eignet sich zur Laufzeitanalyse von Java-basierten Spieleclients und obfuskierten Java-Anwendungen. Es richtet sich an Java-Reverse-Engineers und Sicherheitsforscher, die JVM-basierte Anwendungen analysieren.
