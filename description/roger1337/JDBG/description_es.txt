Este proyecto es JDBG, un depurador Java ligero que permite adjuntarse a procesos Java/JVM en ejecución e inspeccionarlos. Ofrece enganche de métodos, establecimiento de puntos de interrupción, inspección de variables y enumeración de clases a través de la JVM Debug Interface (JDWP). La herramienta puede usarse para el análisis en tiempo de ejecución de clientes de juegos basados en Java y aplicaciones Java ofuscadas. Está dirigida a ingenieros de ingeniería inversa de Java e investigadores de seguridad que analizan aplicaciones basadas en la JVM.
