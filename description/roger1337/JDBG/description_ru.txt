Этот проект — JDBG, легковесный отладчик Java, позволяющий подключаться к работающим процессам Java/JVM и изучать их. Через JVM Debug Interface (JDWP) он обеспечивает перехват методов, установку точек останова, просмотр переменных и перечисление классов. Инструмент можно использовать для анализа во время выполнения Java-клиентов игр и обфусцированных Java-приложений. Он предназначен для Java-реверс-инженеров и специалистов по безопасности, анализирующих приложения на базе JVM.
