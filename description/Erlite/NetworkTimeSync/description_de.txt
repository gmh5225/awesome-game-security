Dieses Projekt ist ein Unreal-Engine-Plugin, das Clients eine genauere Synchronisation der Server-Weltzeit bereitstellt.
Es ist in C++ implementiert und Blueprint-kompatibel integriert, sodass Mehrspielerprojekte synchronisierte Zeitwerte sowohl im Code als auch im visuellen Scripting nutzen können.
Das Plugin ist für die Drop-in-Installation im plugins-Ordner eines Projekts ausgelegt und konzentriert sich darauf, Zeitdrift in vernetzter Gameplay-Logik zu reduzieren.
Der primäre Anwendungsfall ist die Mehrspieler-Spieleentwicklung, bei der zuverlässige gemeinsame Zeit für Simulation, Ereignisse und latenzsensible Mechaniken wichtig ist.
