이 프로젝트는 클라이언트에 더 정확한 서버 월드 시간 동기화를 제공하는 Unreal Engine 플러그인입니다.
C++로 구현되어 있으며 Blueprint 호환 통합을 지원하므로, 멀티플레이어 프로젝트에서 코드와 비주얼 스크립팅 모두로 동기화된 시간 값을 사용할 수 있습니다.
프로젝트 plugins 폴더에 바로 설치할 수 있도록 구성되었으며, 네트워크 게임플레이 로직의 시간 드리프트를 줄이는 데 중점을 둡니다.
주요 사용 사례는 시뮬레이션, 이벤트, 지연에 민감한 메커니즘에서 신뢰할 수 있는 공유 시간이 중요한 멀티플레이어 게임 개발입니다.
