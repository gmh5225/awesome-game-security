Этот проект — плагин для Unreal Engine, обеспечивающий клиентам более точную синхронизацию мирового времени сервера.
Он реализован на C++ с интеграцией, совместимой с Blueprint, чтобы многопользовательские проекты могли использовать синхронизированные значения времени как в коде, так и в визуальном скриптинге.
Плагин рассчитан на установку drop-in в папку plugins проекта и ориентирован на снижение временного дрейфа в сетевой игровой логике.
Основной сценарий использования — разработка многопользовательских игр, где надёжное общее время важно для симуляции, событий и механик, чувствительных к задержке.
