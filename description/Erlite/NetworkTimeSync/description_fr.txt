Ce projet est un plugin Unreal Engine qui offre aux clients une synchronisation plus précise du temps monde serveur.
Il est implémenté en C++ avec une intégration compatible Blueprint, afin que les projets multijoueurs puissent consommer les valeurs de temps synchronisées à la fois dans le code et le scripting visuel.
Le plugin est conçu pour une installation drop-in dans le dossier plugins d’un projet et vise à réduire la dérive temporelle dans la logique de jeu en réseau.
Son cas d’usage principal est le développement de jeux multijoueurs où un temps partagé fiable est important pour la simulation, les événements et les mécaniques sensibles à la latence.
