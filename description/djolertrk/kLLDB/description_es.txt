Depurador de kernel para Linux basado en LLDB que ofrece depuración de kernel en vivo y sin conexión mediante plugins personalizados de LLDB y scripts de Python, construido sobre la infraestructura LLVM-19.
Incluye un plugin de depuración en vivo (kLLDBLive) para sesiones de kernel en ejecución y un plugin sin conexión para análisis post-mortem con scripts automatizados de informe de errores.
Resulta especialmente útil para desarrolladores de kernel e investigadores de seguridad que depuran los internos del kernel de Linux con LLDB en lugar de flujos de trabajo basados en GDB.
