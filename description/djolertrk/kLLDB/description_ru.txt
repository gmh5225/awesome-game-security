Отладчик ядра Linux на базе LLDB, обеспечивающий живую и офлайн-отладку ядра через пользовательские плагины LLDB и скрипты на Python, построенный на инфраструктуре LLVM-19.
Включает плагин живой отладки (kLLDBLive) для сеансов работающего ядра и офлайн-плагин для постмортем-анализа с автоматизированными скриптами отчётов об ошибках.
В основном полезен разработчикам ядра и исследователям безопасности, отлаживающим внутренности ядра Linux с помощью LLDB вместо рабочих процессов на базе GDB.
