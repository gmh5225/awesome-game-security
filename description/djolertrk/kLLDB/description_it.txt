Debugger del kernel Linux basato su LLDB che offre debugging del kernel in tempo reale e offline tramite plugin LLDB personalizzati e scripting Python, costruito sull'infrastruttura LLVM-19.
Include sia un plugin di debugging live (kLLDBLive) per sessioni del kernel in esecuzione, sia un plugin offline per l'analisi post-mortem con script automatici di segnalazione dei bug.
È soprattutto utile per sviluppatori del kernel e ricercatori di sicurezza che depurano gli interni del kernel Linux con LLDB invece di flussi di lavoro basati su GDB.
