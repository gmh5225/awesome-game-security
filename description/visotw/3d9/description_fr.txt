Ce projet se concentre sur la correction des effets stéréoscopiques défectueux dans les jeux DX11.
Il ne s'agit pas de la version grand public de l'outil, mais d'une version destinée aux personnes qui développent le code en corrigeant des bugs, en ajoutant des fonctionnalités ou en documentant son utilisation.
Il est principalement utile aux programmeurs graphiques et aux développeurs d'outils pour jeux Windows travaillant dans le domaine DirectX / outils.
