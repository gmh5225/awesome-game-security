Este proyecto se centra en corregir efectos estereoscópicos defectuosos en juegos DX11.
Esta no es la versión para el usuario final de la herramienta, sino para quienes desarrollan el código corrigiendo errores, añadiendo nuevas funciones o documentando su uso.
Resulta especialmente útil para programadores gráficos y desarrolladores de herramientas para juegos Windows que trabajan en el área de DirectX / tools.
