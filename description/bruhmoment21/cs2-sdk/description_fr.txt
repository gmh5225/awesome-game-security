Ce projet est un SDK Counter-Strike 2 écrit en C++ qui prend en charge Windows et Linux, avec des backends de rendu DirectX 11 et Vulkan.
Il fournit une implémentation simplifiée du SDK Source 2, fidèle au code original du moteur, avec des fonctionnalités de manipulation de jeu et un système de menu basé sur ImGui.
La base de code multiplateforme prend en charge l'injection de DLL via LoadLibrary sur Windows et dlopen sur Linux ; le mapping manuel nécessite des drapeaux de compilateur spécifiques.
Il est principalement utile aux chercheurs en sécurité des jeux qui étudient les internals du moteur Source 2 et les structures du SDK CS2 sur plusieurs plateformes.
