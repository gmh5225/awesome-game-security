Этот проект — SDK для Counter-Strike 2 на C++, поддерживающий Windows и Linux с бэкендами рендеринга DirectX 11 и Vulkan.
Он предоставляет упрощённую реализацию Source 2 SDK, близкую к исходному коду движка, с возможностями манипуляции игрой и системой меню на базе ImGui.
Кроссплатформенная кодовая база поддерживает инъекцию DLL через LoadLibrary в Windows и dlopen в Linux; для manual mapping требуются специальные флаги компилятора.
Проект в первую очередь полезен исследователям игровой безопасности, изучающим внутренности движка Source 2 и структуры CS2 SDK на разных платформах.
