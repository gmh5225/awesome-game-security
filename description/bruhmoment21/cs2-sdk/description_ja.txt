本プロジェクトは C++ で書かれた Counter-Strike 2 SDK で、Windows と Linux の両方をサポートし、DirectX 11 および Vulkan レンダリングバックエンドを備えています。
オリジナルのエンジンコードに近い簡略化された Source 2 SDK 実装を提供し、ゲーム操作機能と ImGui ベースのメニューシステムを含みます。
クロスプラットフォームのコードベースは、Windows では LoadLibrary、Linux では dlopen による DLL インジェクションに対応し、マニュアルマッピングには特定のコンパイラフラグが必要です。
主に、複数プラットフォームにわたる Source 2 エンジンの内部構造や CS2 SDK の構造を研究するゲームセキュリティ研究者向けです。
