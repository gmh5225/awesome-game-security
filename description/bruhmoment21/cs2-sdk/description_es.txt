Este proyecto es un SDK de Counter-Strike 2 escrito en C++ que admite Windows y Linux, con backends de renderizado DirectX 11 y Vulkan.
Proporciona una implementación simplificada del SDK de Source 2, cercana al código original del motor, con funciones de manipulación del juego y un sistema de menús basado en ImGui.
La base de código multiplataforma admite la inyección de DLL mediante LoadLibrary en Windows y dlopen en Linux; el mapeo manual requiere flags específicos del compilador.
Es especialmente útil para investigadores de seguridad en juegos que estudian los internos del motor Source 2 y las estructuras del SDK de CS2 en múltiples plataformas.
