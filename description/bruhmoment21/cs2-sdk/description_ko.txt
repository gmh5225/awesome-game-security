이 프로젝트는 C++로 작성된 Counter-Strike 2 SDK로, Windows와 Linux를 모두 지원하며 DirectX 11 및 Vulkan 렌더링 백엔드를 제공합니다.
원본 엔진 코드에 가깝게 유지된 간소화된 Source 2 SDK 구현을 제공하며, 게임 조작 기능과 ImGui 기반 메뉴 시스템을 포함합니다.
크로스 플랫폼 코드베이스는 Windows에서 LoadLibrary, Linux에서 dlopen을 통한 DLL 인젝션을 지원하며, 매뉴얼 매핑에는 특정 컴파일러 플래그가 필요합니다.
주로 여러 플랫폼에서 Source 2 엔진 내부 구조와 CS2 SDK 구조를 연구하는 게임 보안 연구자에게 유용합니다.
