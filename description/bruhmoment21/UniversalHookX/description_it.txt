Questo progetto è una libreria universale di hooking delle API grafiche per Windows che consente di renderizzare overlay ImGui su più backend di rendering.
Aggancia DirectX 9/9Ex, DirectX 10, DirectX 11, DirectX 12, OpenGL (tramite wglSwapBuffers) e Vulkan (tramite vkQueuePresentKHR) creando dispositivi fittizi per ottenere i puntatori vtable di ciascun backend.
La DLL C++ fornisce un'architettura unificata in cui ogni backend segue lo stesso schema di hooking, con selezione del backend configurabile in fase di compilazione.
È soprattutto utile a ricercatori di sicurezza nei videogiochi e sviluppatori di overlay che studiano l'intercettazione delle API grafiche e le tecniche di rendering ImGui cross-backend.
