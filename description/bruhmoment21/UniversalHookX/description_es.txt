Este proyecto es una biblioteca universal de hooking de APIs gráficas para Windows que permite renderizar overlays de ImGui en múltiples backends de renderizado.
Engancha DirectX 9/9Ex, DirectX 10, DirectX 11, DirectX 12, OpenGL (mediante wglSwapBuffers) y Vulkan (mediante vkQueuePresentKHR) creando dispositivos ficticios para obtener los punteros de vtable de cada backend.
La DLL en C++ ofrece una arquitectura unificada en la que cada backend sigue el mismo patrón de hooking, con selección de backend configurable en tiempo de compilación.
Resulta especialmente útil para investigadores de seguridad en juegos y desarrolladores de overlays que estudian la intercepción de APIs gráficas y técnicas de renderizado ImGui multi-backend.
