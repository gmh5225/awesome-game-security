Этот проект — универсальная библиотека перехвата графических API для Windows, позволяющая отрисовывать оверлеи ImGui на нескольких бэкендах рендеринга.
Она перехватывает DirectX 9/9Ex, DirectX 10, DirectX 11, DirectX 12, OpenGL (через wglSwapBuffers) и Vulkan (через vkQueuePresentKHR), создавая фиктивные устройства для получения указателей vtable каждого бэкенда.
C++ DLL предоставляет единую архитектуру, в которой каждый бэкенд следует одному и тому же шаблону перехвата, а выбор бэкенда настраивается на этапе компиляции.
Она в первую очередь полезна исследователям игровой безопасности и разработчикам оверлеев, изучающим перехват графических API и кросс-бэкендные техники рендеринга ImGui.
