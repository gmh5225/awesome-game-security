이 프로젝트는 여러 렌더링 백엔드에서 ImGui 오버레이를 그릴 수 있는 Windows용 범용 그래픽스 API 후킹 라이브러리입니다.
더미 디바이스를 생성해 각 백엔드의 vtable 포인터를 얻은 뒤, DirectX 9/9Ex, DirectX 10, DirectX 11, DirectX 12, OpenGL(wglSwapBuffers 경유), Vulkan(vkQueuePresentKHR 경유)을 후킹합니다.
이 C++ DLL은 통합 아키텍처를 제공하며, 각 백엔드가 동일한 후킹 패턴을 따르고 컴파일 타임에 백엔드를 선택하도록 구성할 수 있습니다.
주로 그래픽스 API 인터셉션과 크로스 백엔드 ImGui 렌더링 기법을 연구하는 게임 보안 연구자 및 오버레이 개발자에게 유용합니다.
