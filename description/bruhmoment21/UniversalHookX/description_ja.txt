本プロジェクトは、複数のレンダリングバックエンド上で ImGui オーバーレイを描画できる、Windows 向けの汎用グラフィックス API フッキングライブラリです。
ダミーデバイスを作成して各バックエンドの vtable ポインタを取得することで、DirectX 9/9Ex、DirectX 10、DirectX 11、DirectX 12、OpenGL（wglSwapBuffers 経由）、Vulkan（vkQueuePresentKHR 経由）をフックします。
この C++ DLL は統一されたアーキテクチャを備え、各バックエンドが同じフッキングパターンに従い、コンパイル時にバックエンドを選択できます。
主に、グラフィックス API のインターセプトやバックエンド横断の ImGui レンダリング手法を研究するゲームセキュリティ研究者およびオーバーレイ開発者に役立ちます。
