Этот проект — интегрированный пакет Unity для создания, импорта, симуляции и рендеринга волос на основе прядей. Он поддерживает импорт Alembic groom, процедурную генерацию, GPU-управляемые ограничения симуляции, иерархический LOD и рендеринг в пайплайнах Built-in, URP и HDRP. Реализация сочетает системы C# для редактора/рантайма с кодом шейдеров для производительности в реальном времени. Пакет предназначен для продвинутых рабочих процессов рендеринга персонажей, производства цифровых людей и исследований в области графики реального времени.
