Este proyecto es el código fuente de referencia C# oficial del motor y el editor de Unity. Expone una gran parte de la implementación administrada (managed) de Unity para que los desarrolladores puedan inspeccionar el comportamiento de las API y los flujos de trabajo internos del editor. El repositorio es principalmente C# y refleja la estructura del árbol de código fuente interno de Unity, pero se distribuye bajo una licencia solo de referencia. Se usa principalmente para comprender el motor, depurar, investigar herramientas y estudiar en profundidad los internos de Unity.
