Ce projet est le code source de référence C# officiel du moteur et de l'éditeur Unity. Il expose une grande partie de l'implémentation managée de Unity, permettant aux développeurs d'examiner le comportement des API et les flux de travail internes de l'éditeur. Le dépôt est principalement en C# et reflète l'arborescence source interne de Unity, mais il est distribué sous une licence de référence uniquement. Il sert surtout à comprendre le moteur, au débogage, à la recherche d'outils et à l'étude technique approfondie des mécanismes internes d'Unity.
