Этот проект — официальный эталонный исходный код на C# для движка и редактора Unity. Он открывает большую часть управляемой (managed) реализации Unity, позволяя разработчикам изучать поведение API и внутренние рабочие процессы редактора. Репозиторий в основном на C# и отражает структуру внутреннего дерева исходников Unity, но распространяется по лицензии только для ознакомления. В основном используется для понимания движка, отладки, исследования инструментов и глубокого технического изучения внутренностей Unity.
