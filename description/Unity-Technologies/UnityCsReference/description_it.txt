Questo progetto è il codice sorgente di riferimento C# ufficiale del motore e dell'editor Unity. Espone gran parte dell'implementazione managed di Unity, consentendo agli sviluppatori di esaminare il comportamento delle API e i flussi di lavoro interni dell'editor. Il repository è principalmente in C# e rispecchia la struttura dell'albero del codice sorgente interno di Unity, ma è distribuito con una licenza di solo riferimento. Viene usato soprattutto per comprendere il motore, il debugging, la ricerca su strumenti e lo studio tecnico approfondito degli interni di Unity.
