Este proyecto es el juego de ejemplo oficial de disparos en primera persona (FPS) de Unity, que demuestra patrones de desarrollo de juegos a nivel de producción. Incluye sistemas de animación de personajes, manejo de armas, arquitectura multijugador en red y pipelines de assets completos para un FPS jugable construido con el motor Unity. Resulta especialmente útil para desarrolladores de juegos e investigadores de Unity que estudian arquitectura FPS de producción, networking y el diseño de máquinas de estados de animación.
