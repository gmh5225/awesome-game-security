Este proyecto es una extensión de WinDbg que enumera los registros de callbacks del kernel en un sistema Windows.
Consulta PspCreateProcessNotifyRoutine, PspCreateThreadNotifyRoutine y PspLoadImageNotifyRoutine para listar todos los punteros de callback registrados de proceso, hilo y carga de imagen, junto con sus módulos de controlador asociados.
La extensión acepta parámetros para filtrar por tipo de callback (process, image, thread o all), lo que facilita la inspección durante sesiones de depuración del kernel.
Resulta especialmente útil para investigadores de anti-cheat y depuradores de kernel que examinan las rutinas de callback registradas para detectar rootkits y analizar controladores.
