Ce projet est une extension WinDbg qui énumère les enregistrements de callbacks noyau sur un système Windows.
Elle interroge PspCreateProcessNotifyRoutine, PspCreateThreadNotifyRoutine et PspLoadImageNotifyRoutine pour lister tous les pointeurs de callbacks enregistrés pour les processus, les threads et le chargement d’images, avec leurs modules pilotes associés.
L’extension accepte des paramètres de filtrage par type de callback (process, image, thread ou all) pour une inspection pratique lors des sessions de débogage noyau.
Elle est surtout utile aux chercheurs en anti-triche et aux débogueurs noyau qui examinent les routines de callback enregistrées pour la détection de rootkits et l’analyse de pilotes.
