Этот проект — расширение WinDbg, которое перечисляет регистрации ядерных callback’ов в системе Windows.
Оно запрашивает PspCreateProcessNotifyRoutine, PspCreateThreadNotifyRoutine и PspLoadImageNotifyRoutine, чтобы вывести все зарегистрированные указатели callback’ов процессов, потоков и загрузки образов вместе со связанными модулями драйверов.
Расширение принимает параметры фильтрации по типу callback (process, image, thread или all) для удобной проверки во время сеансов отладки ядра.
Оно в основном полезно исследователям античитов и отладчикам ядра, изучающим зарегистрированные callback-процедуры для обнаружения руткитов и анализа драйверов.
