本プロジェクトは、削除された NTFS ファイルを復元し、ファイルシステムからファイルレコードとその内容を永続的に消去するアンチフォレンジックツールです。
ユーザーモードアプリケーションは NTFS の MFT エントリを解析して削除済みファイルを復元し指定ディレクトリに出力する一方、カーネルドライバはファイルレコードとファイル内容データを上書きし、復元を不可能にします。
この C++ コードベースには、WDK ドライバ付きの Visual Studio 2019 ソリューションと、NTFS ファイルシステム操作用のユーザーモードコンソールアプリケーションの両方が含まれます。
主に、NTFS のアンチフォレンジック手法やファイルシステムドライバレベルでの安全なファイル削除を研究するデジタルフォレンジック研究者およびセキュリティアナリストに役立ちます。
