Questo progetto è uno strumento anti-forense che recupera i file NTFS eliminati e cancella definitivamente i record dei file e il relativo contenuto dal filesystem.
L'applicazione in user mode recupera i file eliminati analizzando le voci MFT di NTFS e li esporta in una directory specificata, mentre il driver kernel sovrascrive i record dei file e i dati di contenuto per rendere impossibile il recupero.
Il codebase C++ include sia una soluzione Visual Studio 2019 con driver WDK sia un'applicazione console in user mode per la manipolazione del filesystem NTFS.
È soprattutto utile per ricercatori di digital forensics e analisti di sicurezza che studiano le tecniche anti-forensi NTFS e la cancellazione sicura dei file a livello di driver del filesystem.
