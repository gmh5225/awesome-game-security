이 프로젝트는 삭제된 NTFS 파일을 복구하고 파일 시스템에서 파일 레코드와 그 내용을 영구적으로 지우는 안티포렌식 도구입니다.
사용자 모드 애플리케이션은 NTFS MFT 항목을 파싱하여 삭제된 파일을 복구한 뒤 지정된 디렉터리로 출력하며, 커널 드라이버는 파일 레코드와 파일 내용 데이터를 덮어써서 복구를 불가능하게 만듭니다.
이 C++ 코드베이스에는 WDK 드라이버가 포함된 Visual Studio 2019 솔루션과 NTFS 파일 시스템 조작을 위한 사용자 모드 콘솔 애플리케이션이 모두 포함되어 있습니다.
주로 NTFS 안티포렌식 기법과 파일 시스템 드라이버 수준의 안전한 파일 삭제를 연구하는 디지털 포렌식 연구원 및 보안 분석가에게 유용합니다.
