Este proyecto es una herramienta antiforense que recupera archivos NTFS eliminados y borra permanentemente los registros de archivos y su contenido del sistema de archivos.
La aplicación en modo usuario recupera archivos eliminados analizando las entradas MFT de NTFS y las exporta a un directorio especificado, mientras que el controlador de núcleo sobrescribe los registros de archivos y los datos de contenido para hacer imposible la recuperación.
La base de código en C++ incluye una solución de Visual Studio 2019 con un controlador WDK y una aplicación de consola en modo usuario para manipular el sistema de archivos NTFS.
Resulta especialmente útil para investigadores de criminalística digital y analistas de seguridad que estudian técnicas antiforenses de NTFS y el borrado seguro de archivos a nivel de controlador del sistema de archivos.
