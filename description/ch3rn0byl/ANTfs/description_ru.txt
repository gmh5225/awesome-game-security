Этот проект — инструмент антикриминалистики, который восстанавливает удалённые файлы NTFS и безвозвратно стирает записи файлов вместе с их содержимым из файловой системы.
Пользовательское приложение восстанавливает удалённые файлы, разбирая записи MFT NTFS, и сохраняет их в указанный каталог, а драйвер ядра перезаписывает записи файлов и данные содержимого, делая восстановление невозможным.
Кодовая база на C++ включает решение Visual Studio 2019 с драйвером WDK и пользовательское консольное приложение для работы с файловой системой NTFS.
Оно в первую очередь полезно исследователям цифровой криминалистики и аналитикам безопасности, изучающим антикриминалистические техники NTFS и безопасное удаление файлов на уровне драйвера файловой системы.
