Ce projet est un outil d'anti-forensique qui récupère les fichiers NTFS supprimés et efface définitivement les enregistrements de fichiers ainsi que leur contenu du système de fichiers.
L'application en mode utilisateur récupère les fichiers supprimés en analysant les entrées MFT NTFS et les exporte vers un répertoire spécifié, tandis que le pilote noyau écrase les enregistrements de fichiers et les données de contenu pour rendre toute récupération impossible.
La base de code C++ comprend une solution Visual Studio 2019 avec un pilote WDK ainsi qu'une application console en mode utilisateur pour la manipulation du système de fichiers NTFS.
Il est principalement utile aux chercheurs en criminalistique numérique et aux analystes de sécurité qui étudient les techniques d'anti-forensique NTFS et la suppression sécurisée de fichiers au niveau du pilote de système de fichiers.
