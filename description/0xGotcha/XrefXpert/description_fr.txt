Ce projet est un plugin IDA Pro qui améliore la navigation des références croisées via un panneau de visualisation ancrable. Il liste les xrefs de fonctions en temps réel, permet d’y accéder d’un clic et propose des raccourcis pour se déplacer rapidement entre les références. Écrit en Python avec les API IDA et PyQt, il ajoute des filtres sur les immédiats, le nombre de paramètres et les signatures binaires. Il est destiné aux tâches de reverse engineering telles que l’analyse de binaires de jeux et la recherche de vulnérabilités.
