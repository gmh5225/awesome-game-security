Questo progetto è un plugin per IDA Pro che migliora la navigazione dei riferimenti incrociati tramite un pannello viewer agganciabile. Elenca gli xref delle funzioni in tempo reale, consente il salto con un clic e offre scorciatoie per spostarsi rapidamente tra i riferimenti. È scritto in Python con le API di IDA e PyQt e aggiunge filtri per immediates, conteggio dei parametri e firme binarie. È pensato per attività di reverse engineering come l’analisi di binari di giochi e la ricerca di vulnerabilità.
