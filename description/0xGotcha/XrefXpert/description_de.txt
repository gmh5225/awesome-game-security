Dieses Projekt ist ein IDA-Pro-Plugin, das die Navigation zu Querverweisen über ein andockbares Viewer-Panel verbessert. Es listet Funktions-Xrefs in Echtzeit auf, unterstützt Sprünge per Klick und bietet Hotkeys für schnelles Wechseln zwischen Referenzen. Das Plugin ist in Python mit IDA-APIs und PyQt geschrieben und ergänzt Filter für Immediate-Werte, Parameteranzahlen und Binärsignaturen. Es richtet sich an Reverse-Engineering-Aufgaben wie die Analyse von Spielebinaries und Vulnerability Research.
