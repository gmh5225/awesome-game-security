Этот проект — плагин для IDA Pro, улучшающий навигацию по перекрёстным ссылкам с помощью закрепляемой панели просмотра. Он в реальном времени перечисляет xref функций, поддерживает переход по клику и предоставляет горячие клавиши для быстрого перемещения между ссылками. Плагин написан на Python с использованием API IDA и PyQt и добавляет фильтры по непосредственным значениям, числу параметров и бинарным сигнатурам. Предназначен для задач реверс-инжиниринга, таких как анализ игровых бинарников и исследование уязвимостей.
