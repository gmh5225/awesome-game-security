Este proyecto es un complemento de IDA Pro que mejora la navegación por referencias cruzadas mediante un panel de visualización acoplable. Enumera los xrefs de funciones en tiempo real, permite saltar con un clic y ofrece atajos para moverse rápidamente entre referencias. Está escrito en Python con las API de IDA y PyQt, y añade filtros para inmediatos, número de parámetros y firmas binarias. Está pensado para tareas de ingeniería inversa como el análisis de binarios de juegos y la investigación de vulnerabilidades.
