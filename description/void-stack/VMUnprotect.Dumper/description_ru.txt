VMUnprotect.Dumper — инструмент на C#, который динамически распаковывает .NET-сборки, защищённые VMProtect, чтобы аналитики могли восстановить расшифрованный исполняемый файл. Он загружает цель, принуждает статический конструктор VMProtect восстановить методы в памяти, затем дампит runtime PE-образ с помощью AsmResolver, исправляя точку входа и поля PE magic. Собран для .NET Framework 4.7.2 с AsmResolver и Sharprompt, ориентирован на VMProtect 3.7.0 и близкие версии. Предназначен для реверс-инженеров и исследователей игровой безопасности, изучающих или анализирующих бинарники, защищённые VMProtect.
