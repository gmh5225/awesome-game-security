VMUnprotect.Dumper는 VMProtect로 보호된 .NET 어셈블리를 동적으로 언패킹하여 분석가가 복호화된 실행 파일을 복구할 수 있게 하는 C# 도구입니다. 대상을 로드한 뒤 VMProtect의 정적 생성자를 강제 실행해 메모리에서 메서드를 복원하고, AsmResolver로 런타임 PE 이미지를 덤프하면서 엔트리 포인트와 PE magic 필드를 수정합니다. AsmResolver와 Sharprompt를 사용해 .NET Framework 4.7.2용으로 구축되었으며, VMProtect 3.7.0 및 인접 버전을 대상으로 합니다. VMProtect로 보호된 바이너리를 연구·분석하는 리버스 엔지니어와 게임 보안 연구자를 위한 도구입니다.
