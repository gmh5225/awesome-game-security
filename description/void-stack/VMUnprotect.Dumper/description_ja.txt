VMUnprotect.Dumper は、VMProtect で保護された .NET アセンブリを動的にアンパックし、解析者が復号済み実行ファイルを復元できるようにする C# ツールです。対象を読み込み、VMProtect の静的コンストラクタを強制実行してメモリ上でメソッドを復元したうえで、AsmResolver によりランタイム PE イメージをダンプし、エントリポイントと PE magic フィールドを修正します。.NET Framework 4.7.2 向けに AsmResolver と Sharprompt を用いて構築され、VMProtect 3.7.0 および近傍バージョンを対象とします。VMProtect 保護バイナリを調査・解析するリバースエンジニアやゲームセキュリティ研究者向けです。
