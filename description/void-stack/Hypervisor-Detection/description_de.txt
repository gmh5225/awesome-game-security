Dieses Projekt implementiert derzeit vier Techniken.
Es ist hauptsächlich in C++ geschrieben und konzentriert sich auf Anti-Cheat-Forschung.
Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat / Erkennung: kompromittierter Hypervisor arbeiten.
