VMUnprotect は、VMProtect によって仮想化されたメソッドを動的に解析・計装する .NET ツールで、完全なコード復元ではなくメソッド管理に焦点を当てています。C# で書かれ、Harmony を用いて VMProtect のランタイム挙動をフックし観測でき、仮想化メソッド内の呼び出しのトレースや、引数・戻り値の操作にも対応します。NtQueryInformationProcess、Debugger.IsLogging、Debugger.IsAttached といった一般的な VMProtect のアンチデバッグ検査を回避でき、メモリ保護・インポート保護・リソース保護など典型的な保護が施された VMProtect 3.6.0 およびそれ以前の複数バージョンで動作します。本プロジェクトは、仮想化されたメソッド呼び出しのランタイム可視性を必要とする、VMProtect 保護された .NET アセンブリを研究するリバースエンジニアやセキュリティ研究者を対象としています。
