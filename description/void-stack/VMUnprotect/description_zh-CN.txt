VMUnprotect 是一个 .NET 工具，可动态分析并插桩由 VMProtect 虚拟化的方法，重点在于方法管控而非完整代码还原。它用 C# 编写，通过 Harmony 挂钩并观察 VMProtect 运行时行为，支持跟踪虚拟化方法内的调用，以及操纵参数与返回值。它可绕过常见的 VMProtect 反调试检查，例如 NtQueryInformationProcess、Debugger.IsLogging 和 Debugger.IsAttached，并适用于 VMProtect 3.6.0 及若干更早版本上的内存、导入与资源等典型保护。该项目面向需要在运行时观察虚拟化方法调用的逆向工程师与安全研究人员，用于研究受 VMProtect 保护的 .NET 程序集。
