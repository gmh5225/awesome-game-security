VMUnprotect는 VMProtect로 가상화된 메서드를 동적으로 분석하고 계측하는 .NET 도구로, 완전한 코드 복원보다는 메서드 관리에 중점을 둡니다. C#로 작성되었으며 Harmony를 사용해 VMProtect 런타임 동작을 후킹하고 관찰하며, 가상화된 메서드 내부의 호출을 추적하고 매개변수와 반환값을 조작할 수 있습니다. NtQueryInformationProcess, Debugger.IsLogging, Debugger.IsAttached와 같은 일반적인 VMProtect 안티디버그 검사를 우회할 수 있으며, 메모리·임포트·리소스 보호 등 전형적인 보호가 적용된 VMProtect 3.6.0 및 여러 이전 버전에서 동작합니다. 이 프로젝트는 가상화된 메서드 호출에 대한 런타임 가시성이 필요한, VMProtect로 보호된 .NET 어셈블리를 연구하는 리버스 엔지니어와 보안 연구자를 대상으로 합니다.
