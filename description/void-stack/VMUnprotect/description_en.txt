VMUnprotect is a .NET tool that dynamically analyzes and instruments methods virtualized by VMProtect, focusing on method administration rather than full code recovery. Written in C#, it uses Harmony to hook and observe VMProtect runtime behavior, with support for tracing invokes inside virtualized methods and manipulating parameters and return values. It can bypass common VMProtect anti-debug checks such as NtQueryInformationProcess, Debugger.IsLogging, and Debugger.IsAttached, and works against VMProtect 3.6.0 and several earlier versions across typical protections like memory, import, and resource protection. The project is aimed at reverse engineers and security researchers studying VMProtect-protected .NET assemblies who need runtime visibility into virtualized method calls.
