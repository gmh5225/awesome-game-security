Dieses Projekt ist eine schlanke Android-Dateimanager-Anwendung im modernen Material-3-Stil.
Sie ist hauptsächlich in Kotlin mit etwas Java entwickelt und bietet Kern-Dateoperationen, Multi-Tab-Navigation, Aufgabenverwaltung sowie integrierte Codebearbeitung.
Die App unterstützt außerdem die tiefgehende Inhaltssuche und ist als standardmäßiges Gradle-basiertes Android-Projekt aufgebaut.
Sie richtet sich an Android-Nutzer und Entwickler, die eine Open-Source-Dateiverwaltungsbasis zum Erweitern oder Studieren benötigen.
