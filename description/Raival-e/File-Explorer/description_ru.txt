Этот проект — лёгкое Android-приложение для управления файлами в современном стиле Material 3.
Оно разработано в основном на Kotlin с использованием некоторого кода на Java и предоставляет основные файловые операции, навигацию с несколькими вкладками, обработку задач и встроенное редактирование кода.
Приложение также поддерживает глубокий поиск по содержимому и организовано как стандартный Android-проект на Gradle.
Оно предназначено для пользователей и разработчиков Android, которым нужна открытая база для управления файлами, которую можно расширять или изучать.
