Este proyecto es una aplicación ligera de gestión de archivos para Android con un estilo Material 3 moderno.
Está desarrollada principalmente en Kotlin con algo de Java, y ofrece operaciones básicas de archivos, navegación por pestañas múltiples, gestión de tareas y edición de código integrada.
La aplicación también admite búsqueda profunda de contenido y está estructurada como un proyecto Android estándar basado en Gradle.
Está dirigida a usuarios y desarrolladores de Android que necesitan una base de gestión de archivos de código abierto para ampliar o estudiar.
