Ce projet est une application légère de gestion de fichiers Android au style Material 3 moderne.
Elle est développée principalement en Kotlin avec un peu de Java, et propose les opérations de fichiers essentielles, la navigation multi-onglets, la gestion des tâches et l'édition de code intégrée.
L'application prend également en charge la recherche approfondie dans le contenu et est structurée comme un projet Android Gradle standard.
Elle s'adresse aux utilisateurs et développeurs Android qui ont besoin d'une base open source de gestion de fichiers à étendre ou à étudier.
