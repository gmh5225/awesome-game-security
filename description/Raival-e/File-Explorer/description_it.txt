Questo progetto è un'applicazione leggera di gestione file per Android con uno stile Material 3 moderno.
È sviluppata principalmente in Kotlin con un po' di Java e offre operazioni file di base, navigazione multi-scheda, gestione delle attività e editing del codice integrato.
L'app supporta anche la ricerca approfondita dei contenuti ed è strutturata come un progetto Android standard basato su Gradle.
È rivolta a utenti e sviluppatori Android che necessitano di una base open source di gestione file da estendere o studiare.
