Личная коллекция заметок по разработке игр, объединяющая многолетние внутренние материалы онбординга студии в удобный для просмотра сайт документации. Подробно охватывает Unreal Engine 4 — настройка окружения, программирование геймплея, сеть и репликация, рендеринг, шейдеры, оптимизация производительности и инструменты редактора — а также Houdini, теорию графики и шпаргалки по программированию для C++, Python и связанных инструментов. Заметки также включают внутренности Windows, рабочие процессы отладки и рекомендации по инструментам разработчика. Контент на Markdown и рассчитан на локальный просмотр в стиле MkDocs или Obsidian. Предназначен для игровых программистов и technical artists, которым нужны практические справочные материалы по UE4 и графической разработке.
