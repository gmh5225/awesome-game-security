Este proyecto es una utilidad de escritorio para automatizar la configuración de un mod inicial basado en UE4SS para Palworld.
Ofrece una interfaz gráfica PyQt para descargar archivos de parche, aplicar o revertir modificaciones y gestionar el estado de activación o desactivación de los mods.
El repositorio combina código de aplicación en Python con scripts de mods en Lua que ajustan ajustes de jugabilidad como el coste de resistencia, la visibilidad del mapa y los valores de progresión.
Está pensado para facilitar el modding y la experimentación con el comportamiento de juegos de Unreal Engine.
