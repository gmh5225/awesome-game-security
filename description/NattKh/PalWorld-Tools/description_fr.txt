Ce projet est un utilitaire de bureau qui automatise la configuration d’un mod de démarrage basé sur UE4SS pour Palworld.
Il propose une interface PyQt pour télécharger des fichiers de correctifs, appliquer ou annuler des modifications, et gérer l’activation ou la désactivation des mods.
Le dépôt mélange du code d’application Python avec des scripts de mods Lua qui ajustent des paramètres de gameplay tels que le coût d’endurance, la visibilité de la carte et les valeurs de progression.
Il est destiné à faciliter le modding et l’expérimentation du comportement des jeux Unreal Engine.
