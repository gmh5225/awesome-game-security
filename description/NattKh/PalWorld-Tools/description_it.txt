Questo progetto è un’utilità desktop per automatizzare la configurazione di un mod starter basato su UE4SS per Palworld.
Fornisce una GUI PyQt per scaricare file di patch, applicare o annullare modifiche e gestire lo stato di attivazione o disattivazione dei mod.
Il repository combina codice applicativo Python con script di mod Lua che regolano impostazioni di gameplay come i costi di stamina, la visibilità della mappa e i valori di progressione.
È pensato per semplificare il modding e la sperimentazione del comportamento dei giochi Unreal Engine.
