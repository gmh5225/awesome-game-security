Dieses Projekt ist ein Desktop-Dienstprogramm zur Automatisierung einer UE4SS-basierten Starter-Mod-Einrichtung für Palworld.
Es bietet eine PyQt-GUI zum Herunterladen von Patch-Dateien, zum Anwenden oder Rückgängigmachen von Änderungen sowie zum Verwalten des Aktivierungs- oder Deaktivierungsstatus von Mods.
Das Repository kombiniert Python-Anwendungscode mit Lua-Mod-Skripten, die Gameplay-Einstellungen wie Ausdauerverbrauch, Kartensichtbarkeit und Fortschrittswerte anpassen.
Es dient der bequemen Modding-Arbeit und dem Experimentieren mit dem Verhalten von Unreal-Engine-Spielen.
