Этот проект — настольная утилита для автоматизации настройки стартового мода на базе UE4SS для Palworld.
Она предоставляет графический интерфейс на PyQt для загрузки файлов патчей, применения или отката изменений, а также управления включением и отключением модов.
Репозиторий сочетает код приложения на Python со скриптами модов на Lua, которые изменяют игровые параметры: затраты выносливости, видимость карты и значения прогрессии.
Предназначен для удобного моддинга и экспериментов с поведением игр на Unreal Engine.
