このプロジェクトは、脆弱な CPU-Z ドライバを悪用して Driver Signature Enforcement を無効化し、未署名のカーネルドライバを読み込むための Windows x64 ユーティリティです。
C++ で記述されており、カーネルメモリのパッチ処理、システム変数のパターンスキャン、ドライバ読み込みワークフロー用のヘルパー処理を含みます。
実装は古いシステムでは g_CiEnable、新しいシステムでは g_CiOptions を対象とし、PatchGuard に関連するクラッシュリスクについても明示しています。
主にカーネルセキュリティ研究や、未署名ドライバを用いたアンチチート回避の実験に利用されます。
