이 프로젝트는 취약한 CPU-Z 드라이버를 악용해 Driver Signature Enforcement를 비활성화하고 서명되지 않은 커널 드라이버를 로드하는 Windows x64 유틸리티입니다.
C++로 작성되었으며 커널 메모리 패치 로직, 시스템 변수 패턴 스캐닝, 드라이버 로드 워크플로를 위한 헬퍼 루틴을 포함합니다.
구현은 구형 시스템의 g_CiEnable과 신형 시스템의 g_CiOptions를 대상으로 하며, PatchGuard 관련 크래시 위험에 대해서도 명시적으로 안내합니다.
주로 커널 보안 연구와 서명되지 않은 드라이버를 이용한 안티치트 우회 실험에 사용됩니다.
