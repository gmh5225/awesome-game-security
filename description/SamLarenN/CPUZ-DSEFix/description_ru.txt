Этот проект — утилита для Windows x64, которая использует уязвимый драйвер CPU-Z для отключения Driver Signature Enforcement и загрузки неподписанных драйверов ядра.
Она написана на C++ и содержит логику патчинга памяти ядра, сканирование шаблонов системных переменных и вспомогательные процедуры для сценариев загрузки драйверов.
Реализация нацелена на g_CiEnable в старых системах и на g_CiOptions в новых, с явными замечаниями о риске сбоя, связанном с PatchGuard.
В основном применяется в исследованиях безопасности ядра и экспериментах по обходу античита с использованием неподписанных драйверов.
