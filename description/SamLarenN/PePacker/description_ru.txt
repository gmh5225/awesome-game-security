Этот проект — простой PE-пакер, который шифрует секцию .text исполняемых файлов Windows и добавляет stub для расшифровки.
Он реализован на C++ с собственными компонентами разбора PE для исследования и перезаписи секций.
Метод защиты по умолчанию использует XOR-шифрование как лёгкую демонстрацию, которую можно заменить более стойкими алгоритмами.
В основном полезен для практики реверс-инжиниринга, изучения разработки пакеров и базовых исследований обфускации в контексте безопасности игр.
