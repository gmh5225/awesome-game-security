Открытый воксельный игровой движок (Luanti, ранее Minetest) с API моддинга на Lua, поддерживающий рендеринг Irrlicht/SDL2, процедурную генерацию мира, сетевую игру и кроссплатформенные сборки (Linux, Windows, macOS, Android). Предоставляет полноценный фреймворк для создания контента с определениями узлов (nodes), системами сущностей, управлением инвентарём и UI на основе formspec.
