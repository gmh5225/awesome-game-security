Motore di gioco voxel open source (Luanti, precedentemente Minetest) con API di modding in Lua, che supporta rendering Irrlicht/SDL2, generazione procedurale del mondo, networking multiplayer e build multipiattaforma (Linux, Windows, macOS, Android). Fornisce un framework completo per la creazione di contenuti con definizioni di nodi, sistemi di entità, gestione dell'inventario e UI basate su formspec.
