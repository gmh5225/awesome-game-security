Moteur de jeu voxel open source (Luanti, anciennement Minetest) avec une API de modding Lua, prenant en charge le rendu Irrlicht/SDL2, la génération procédurale de mondes, le réseau multijoueur et les builds multiplateformes (Linux, Windows, macOS, Android). Fournit un framework complet de création de contenu avec définitions de nœuds, systèmes d'entités, gestion d'inventaire et interfaces utilisateur basées sur formspec.
