Motor de juego voxel de código abierto (Luanti, anteriormente Minetest) con una API de modding en Lua, que admite renderizado Irrlicht/SDL2, generación procedural de mundos, redes multijugador y compilaciones multiplataforma (Linux, Windows, macOS, Android). Proporciona un framework completo de creación de contenido con definiciones de nodos, sistemas de entidades, gestión de inventario e interfaces de usuario basadas en formspec.
