Этот проект сосредоточен на wsftprm.sys.
Уязвимость вызывается через IOCTL-код 0x22201C с буфером размером 1036 байт, в первых 4 байтах которого содержится целевой Process ID в виде DWORD.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / уязвимых драйверов.
