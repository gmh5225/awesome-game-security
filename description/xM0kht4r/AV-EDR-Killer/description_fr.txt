Ce projet se concentre sur wsftprm.sys.
La vulnérabilité est déclenchée via le code IOCTL 0x22201C avec un tampon de 1036 octets dont les 4 premiers octets contiennent l'identifiant de processus cible sous forme de DWORD.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / pilotes vulnérables.
