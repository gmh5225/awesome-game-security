Этот проект — PE-дампер на основе DMA для извлечения образов процессов через PCIe FPGA. Он реализован на C++ и интегрируется с компонентами в стиле leechcore и VMMDLL для низкоуровневого доступа к памяти. Инструмент рассчитан на сложные сценарии вроде CR3 shuffling при поиске и дампе памяти целевых исполняемых файлов или DLL. Основной сценарий использования — продвинутая memory forensics и рабочие процессы исследований игровых античитов.
