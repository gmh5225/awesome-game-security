Este proyecto es apksigcopier, una herramienta en Python para copiar firmas APK de un APK a otro. Extrae los bloques de firma v1 (JAR), v2 y v3 de un APK firmado y los trasplanta a un APK modificado, lo que permite volver a firmar sin la clave de firma original en escenarios donde se puede eludir la verificación de firmas. La herramienta manipula el APK Signing Block a nivel binario. Está orientada a investigadores de seguridad de Android e ingenieros de ingeniería inversa que trabajan con el análisis y la modificación de firmas APK.
