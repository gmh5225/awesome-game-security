Этот проект — apksigcopier, инструмент на Python для копирования подписей APK из одного APK в другой. Он извлекает блоки подписи v1 (JAR), v2 и v3 из подписанного APK и переносит их в изменённый APK, что позволяет переподписывать без исходного ключа подписи в сценариях, где проверку подписи можно обойти. Инструмент выполняет манипуляции с APK Signing Block на двоичном уровне. Предназначен для исследователей безопасности Android и специалистов по реверс-инжинирингу, работающих с анализом и модификацией подписей APK.
