FindWindow, CreateToolhelp32Snapshot 같은 Windows API 호출과 서비스 열거를 사용해 알려진 창 클래스, 프로세스 이름, 드라이버 존재 여부, 디버그 흔적을 스캔하여 실행 중인 Cheat Engine 인스턴스를 식별하는 탐지 도구입니다.
ce_detection 모듈은 여러 탐지 벡터를 구현하여 Cheat Engine이 이름이 바뀌었거나 안티탐지 기능이 활성화된 상태에서도 안정적으로 식별합니다.
주로 Cheat Engine 탐지 기법을 연구하는 안티치트 개발자와 CE 스텔스 설정을 테스트하는 게임 보안 연구자에게 유용합니다.
