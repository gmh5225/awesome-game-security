Инструмент обнаружения, который выявляет запущенные экземпляры Cheat Engine, сканируя известные классы окон, имена процессов, наличие драйверов и отладочные артефакты с помощью вызовов Windows API, таких как FindWindow, CreateToolhelp32Snapshot, а также перечисления служб.
Модуль ce_detection реализует несколько векторов обнаружения для надёжной идентификации Cheat Engine даже при переименовании или работе с включёнными средствами антиобнаружения.
В основном полезен разработчикам античитов, изучающим методы обнаружения Cheat Engine, и исследователям игровой безопасности, тестирующим скрытные конфигурации CE.
