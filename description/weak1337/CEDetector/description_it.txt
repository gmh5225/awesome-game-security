Uno strumento di rilevamento che identifica le istanze in esecuzione di Cheat Engine scansionando le sue classi di finestra note, i nomi dei processi, la presenza di driver e gli artefatti di debug mediante chiamate alle API Windows come FindWindow, CreateToolhelp32Snapshot e l'enumerazione dei servizi.
Il modulo ce_detection implementa più vettori di rilevamento per identificare in modo affidabile Cheat Engine anche quando è stato rinominato o è in esecuzione con funzioni anti-rilevamento abilitate.
È soprattutto utile per gli sviluppatori di anti-cheat che studiano le tecniche di rilevamento di Cheat Engine e per i ricercatori di sicurezza dei giochi che testano le configurazioni stealth di CE.
