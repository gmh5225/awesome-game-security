FindWindow、CreateToolhelp32Snapshot などの Windows API 呼び出しとサービス列挙を用いて、既知のウィンドウクラス、プロセス名、ドライバの有無、デバッグ痕跡を走査し、実行中の Cheat Engine インスタンスを特定する検出ツールです。
ce_detection モジュールは複数の検出ベクトルを実装しており、Cheat Engine がリネームされている場合やアンチ検出機能が有効な場合でも、確実に識別できます。
主に、Cheat Engine の検出手法を研究するアンチチート開発者や、CE のステルス設定を検証するゲームセキュリティ研究者向けです。
