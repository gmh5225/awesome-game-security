Un injector di DLL in user-mode che mappa manualmente una DLL in un processo di destinazione analizzando le intestazioni PE, allocando memoria con VirtualAllocEx, scrivendo le sezioni tramite WriteProcessMemory, risolvendo gli import, applicando le relocation ed eseguendo l’entry point tramite uno stub di shellcode iniettato con CreateRemoteThread.
Il mapper gestisce target x86 e x64, supporta i callback TLS e include il parsing delle strutture PE (pestruct.h) per navigare tabelle delle sezioni, directory degli import e blocchi di relocation durante il manual mapping.
È soprattutto utile ai ricercatori di game security che studiano le tecniche di injection con manual mapping in user-mode e gli interni del caricamento PE legati all’evasione degli anti-cheat.
