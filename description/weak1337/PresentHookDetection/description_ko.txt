BattlEye가 IDXGISwapChain::Present 후크를 탐지하는 방식을 재현하는 도구입니다. 더미 D3D11 디바이스와 스왑 체인을 생성하고, vtable에서 Present 함수 포인터를 읽은 뒤, 해당 주소의 코드 바이트를 원본 dxgi.dll 모듈과 비교하여 인라인 후크(JMP 패치)나 vtable 덮어쓰기를 식별합니다. 탐지는 Present 함수의 앞부분 바이트가 예상 프롤로그와 일치하는지, 또는 ESP/에임봇 화면을 그리기 위해 Present를 후킹하는 치트 오버레이에 의해 수정되었는지를 확인합니다. 주로 안티치트 개발자가 Present 후크 탐지를 구현하거나, 치트 개발자가 BattlEye 스타일 검사에 대한 렌더링 후크의 은닉성을 테스트할 때 유용합니다.
