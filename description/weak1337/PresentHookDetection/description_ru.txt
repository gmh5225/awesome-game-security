Инструмент, воспроизводящий метод BattlEye для обнаружения хуков IDXGISwapChain::Present: создаёт фиктивное устройство D3D11 и swap chain, читает указатель функции Present из vtable и сравнивает байты кода по этому адресу с исходным модулем dxgi.dll, чтобы выявить inline-хуки (JMP-патчи) или перезапись vtable. Проверка определяет, совпадают ли первые байты Present с ожидаемым прологом или они были изменены чит-оверлеями, которые хукают Present для отрисовки ESP/aimbot. В основном полезен разработчикам античита для реализации обнаружения Present-хуков и разработчикам читов для проверки скрытности своих хуков рендеринга против проверок в стиле BattlEye.
