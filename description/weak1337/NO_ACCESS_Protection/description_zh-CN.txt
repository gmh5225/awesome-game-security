一个 C++ 库，通过 VirtualProtect 将代码段标记为 PAGE_NOACCESS 加以保护，再借助 Vectored Exception Handler 捕获由此产生的访问违例，并临时将出错页恢复为 PAGE_EXECUTE_READ 权限，随后通过单步陷阱（STATUS_SINGLE_STEP）在执行后重新加保护。
这样可形成自我防护的代码区域：任何试图读取受保护页的外部内存扫描器或调试器都会触发访问违例，而合法执行则通过 VEH trampoline 透明继续。
它主要面向研究基于 PAGE_NOACCESS 防篡改技术的游戏安全研究员，以及评估对抗外部扫描器的内存保护策略的反作弊开发者。
