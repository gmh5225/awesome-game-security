Eine C++-Bibliothek, die Codeabschnitte schützt, indem sie sie mit VirtualProtect als PAGE_NOACCESS markiert und anschließend einen Vectored Exception Handler nutzt, um die resultierenden Zugriffsverletzungen abzufangen und die Rechte der fehlerhaften Seite vorübergehend auf PAGE_EXECUTE_READ wiederherzustellen, bevor sie nach der Ausführung über einen Single-Step-Trap (STATUS_SINGLE_STEP) erneut geschützt wird.
So entsteht ein selbstverteidigender Codebereich: Jeder externe Speicherscanner oder Debugger, der die geschützten Seiten lesen will, löst eine Zugriffsverletzung aus, während legitime Ausführung über den VEH-Trampoline transparent weiterläuft.
Sie ist vor allem für Gamesicherheitsforscher nützlich, die PAGE_NOACCESS-basierte Anti-Tamper-Techniken untersuchen, sowie für Anti-Cheat-Entwickler, die Speicherschutzstrategien gegen externe Scanner bewerten.
