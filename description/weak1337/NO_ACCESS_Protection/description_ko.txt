VirtualProtect로 코드 섹션을 PAGE_NOACCESS로 표시해 보호한 뒤, Vectored Exception Handler로 그로 인한 액세스 위반을 잡고 장애 페이지의 권한을 일시적으로 PAGE_EXECUTE_READ로 복원한 다음, 싱글스텝 트랩(STATUS_SINGLE_STEP)으로 실행 후 다시 보호하는 C++ 라이브러리입니다.
이렇게 하면 보호된 페이지를 읽으려는 외부 메모리 스캐너나 디버거는 액세스 위반을 일으키고, 정상적인 실행은 VEH trampoline을 통해 투명하게 이어지는 자기 방어 코드 영역이 만들어집니다.
주로 PAGE_NOACCESS 기반 안티탬퍼 기법을 연구하는 게임 보안 연구자와, 외부 스캐너에 대한 메모리 보호 전략을 평가하는 안티치트 개발자에게 유용합니다.
