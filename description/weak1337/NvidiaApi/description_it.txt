Libreria C++ che interagisce con il NvAPI non documentato di NVIDIA per interrogare informazioni hardware della GPU, tra cui numeri di serie GPU, handle di GPU fisiche, nomi della scheda e versioni del driver, caricando dinamicamente nvapi64.dll e risolvendo i puntatori alle funzioni API interne tramite NvAPI_QueryInterface.
La libreria wrappa sia le funzioni NvAPI pubbliche (EnumPhysicalGPUs, GetFullName) sia interfacce interne/private per accedere a identificatori hardware non esposti tramite l'SDK NvAPI standard.
È principalmente utile per sviluppatori di spoofer HWID e ricercatori di sicurezza nei videogiochi che studiano il recupero dei numeri di serie GPU e i metodi di fingerprinting hardware NVIDIA usati dai sistemi anti-cheat.
