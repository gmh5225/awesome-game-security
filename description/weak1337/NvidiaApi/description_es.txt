Biblioteca en C++ que interactúa con el NvAPI no documentado de NVIDIA para consultar información de hardware de la GPU, incluidos números de serie de GPU, handles de GPU física, nombres de placa y versiones de controlador, cargando dinámicamente nvapi64.dll y resolviendo punteros a funciones internas de la API mediante NvAPI_QueryInterface.
La biblioteca encapsula tanto funciones públicas de NvAPI (EnumPhysicalGPUs, GetFullName) como interfaces internas/privadas para acceder a identificadores de hardware que no se exponen a través del SDK estándar de NvAPI.
Resulta especialmente útil para desarrolladores de spoofers de HWID e investigadores de seguridad en juegos que estudian la obtención de números de serie de GPU y los métodos de huella digital de hardware NVIDIA usados por los sistemas anticheat.
