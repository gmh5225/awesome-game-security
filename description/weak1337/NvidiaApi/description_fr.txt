Bibliothèque C++ qui interagit avec le NvAPI non documenté de NVIDIA pour interroger les informations matérielles du GPU, notamment les numéros de série GPU, les handles de GPU physique, les noms de carte et les versions de pilote, en chargeant dynamiquement nvapi64.dll et en résolvant les pointeurs de fonctions API internes via NvAPI_QueryInterface.
La bibliothèque encapsule à la fois les fonctions NvAPI publiques (EnumPhysicalGPUs, GetFullName) et des interfaces internes/privées permettant d'accéder à des identifiants matériels non exposés par le SDK NvAPI standard.
Elle est principalement utile aux développeurs de spoofers HWID et aux chercheurs en sécurité des jeux qui étudient la récupération des numéros de série GPU et les méthodes de prise d'empreinte matérielle NVIDIA utilisées par les systèmes anti-triche.
