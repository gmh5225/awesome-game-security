nvapi64.dll을 동적으로 로드하고 NvAPI_QueryInterface를 통해 내부 API 함수 포인터를 해석하여, NVIDIA의 비공개 NvAPI와 연동해 GPU 시리얼 번호, 물리 GPU 핸들, 보드 이름, 드라이버 버전 등 GPU 하드웨어 정보를 조회하는 C++ 라이브러리입니다.
이 라이브러리는 공개 NvAPI 함수(EnumPhysicalGPUs, GetFullName)와 표준 NvAPI SDK에서 노출되지 않는 하드웨어 식별자에 접근하기 위한 내부/비공개 인터페이스를 모두 래핑합니다.
주로 HWID 스푸퍼 개발자와, GPU 시리얼 번호 조회 및 안티치트 시스템에서 사용되는 NVIDIA 하드웨어 핑거프린팅 기법을 연구하는 게임 보안 연구원에게 유용합니다.
