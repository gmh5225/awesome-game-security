nvapi64.dll を動的に読み込み、NvAPI_QueryInterface を通じて内部 API の関数ポインタを解決することで、NVIDIA の非公開 NvAPI と連携し、GPU シリアル番号、物理 GPU ハンドル、ボード名、ドライバーバージョンなどの GPU ハードウェア情報を照会する C++ ライブラリです。
本ライブラリは、公開 NvAPI 関数（EnumPhysicalGPUs、GetFullName）と、標準の NvAPI SDK では公開されていないハードウェア識別子にアクセスするための内部／非公開インターフェースの両方をラップしています。
主に HWID スプーファー開発者や、GPU シリアル番号の取得およびアンチチートシステムで用いられる NVIDIA ハードウェアフィンガープリンティング手法を研究するゲームセキュリティ研究者にとって有用です。
