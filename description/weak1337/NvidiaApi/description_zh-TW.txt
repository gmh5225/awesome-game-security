一個透過動態載入 nvapi64.dll，並利用 NvAPI_QueryInterface 解析內部 API 函式指標，與 NVIDIA 未公開的 NvAPI 互動的 C++ 函式庫，用於查詢 GPU 硬體資訊，包括 GPU 序號、實體 GPU 控制代碼、主機板名稱與驅動程式版本。
該函式庫封裝了公開的 NvAPI 函式（EnumPhysicalGPUs、GetFullName），以及用於存取標準 NvAPI SDK 未暴露之硬體識別碼的內部／私有介面。
它主要適用於 HWID 偽裝工具開發者，以及研究 GPU 序號取得方式與反作弊系統所用 NVIDIA 硬體指紋辨識方法的遊戲安全研究人員。
