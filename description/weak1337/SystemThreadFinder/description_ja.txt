NtQuerySystemInformation ですべてのスレッドを列挙し、各カーネルスレッドの開始アドレスをロード済みドライバーモジュールの範囲と照合し、正当なドライバーのイメージ領域外を指す開始アドレスを持つスレッドを検出することで、隠蔽された・手動マップされたシステムスレッドを見つけるツールです。
BattlEye のスレッド検出ロジックをリバースエンジニアリングして作成されており、非公開の NT API でスレッド情報を取得し、開始アドレスをシステムモジュール一覧と突き合わせて、手動マップまたは注入されたカーネルコードから生成されたスレッドを特定します。
主にアンチチート研究者向けで、カーネルレベルのスレッド検出手法の研究や、マップされていないメモリ領域からシステムスレッドを生成するステルスなドライバーベースのチートの特定に役立ちます。
