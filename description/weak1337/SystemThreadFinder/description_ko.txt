NtQuerySystemInformation으로 모든 스레드를 열거하고, 각 커널 스레드의 시작 주소를 로드된 드라이버 모듈 범위와 대조한 뒤, 정상적인 드라이버 이미지 영역 밖에 있는 시작 주소를 가진 스레드를 표시하여 숨겨진·수동 매핑된 시스템 스레드를 탐지하는 도구입니다.
BattlEye의 스레드 탐지 로직을 리버스 엔지니어링하여 만들어졌으며, 문서화되지 않은 NT API로 스레드 정보를 조회하고 시작 주소를 시스템 모듈 목록과 교차 참조하여, 수동 매핑되거나 주입된 커널 코드에서 생성된 스레드를 식별합니다.
주로 안티치트 연구자가 커널 수준 스레드 탐지 기법을 연구하고, 매핑되지 않은 메모리 영역에서 시스템 스레드를 생성하는 은닉형 드라이버 기반 치트를 식별하는 데 유용합니다.
