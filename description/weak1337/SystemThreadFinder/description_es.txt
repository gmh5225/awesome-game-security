Una herramienta que detecta hilos del sistema ocultos y mapeados manualmente enumerando todos los hilos mediante NtQuerySystemInformation, comprobando la dirección de inicio de cada hilo del kernel frente a los rangos de módulos de controladores cargados, y marcando los hilos cuyas direcciones de inicio quedan fuera de cualquier región de imagen de controlador legítima.
Creada mediante ingeniería inversa de la lógica de detección de hilos de BattlEye, usa API de NT no documentadas para consultar información de hilos y cruza las direcciones de inicio con la lista de módulos del sistema para identificar hilos generados desde código del kernel mapeado manualmente o inyectado.
Resulta especialmente útil para investigadores de anti-cheat que estudian técnicas de detección de hilos a nivel de kernel e identifican cheats sigilosos basados en controladores que crean hilos del sistema desde regiones de memoria no mapeadas.
