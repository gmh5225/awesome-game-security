Инструмент, обнаруживающий скрытые и вручную отображённые системные потоки: перечисляет все потоки через NtQuerySystemInformation, сверяет стартовый адрес каждого потока ядра с диапазонами загруженных модулей драйверов и помечает потоки, чьи стартовые адреса лежат вне области образа любого легитимного драйвера.
Создан путём реверс-инжиниринга логики обнаружения потоков BattlEye; использует недокументированные NT API для запроса сведений о потоках и сопоставляет стартовые адреса со списком системных модулей, чтобы выявлять потоки, порождённые вручную отображённым или внедрённым кодом ядра.
В первую очередь полезен исследователям античита, изучающим методы обнаружения потоков на уровне ядра и выявляющим скрытные читы на базе драйверов, которые создают системные потоки из неотображённых областей памяти.
