一個僅標頭檔的 C++ 函式庫，用於建立函式呼叫 trampoline，跳過目標函式的第一條指令，從而繞過 BattlEye 等反作弊系統在 WinAPI 與遊戲函式上放置的內聯鉤子（0xE9 JMP）與中斷點陷阱（0xCC INT3）。
它使用 HDE（Hacker Disassembly Engine）進行 x86/x64 指令長度解碼以判定第一條指令的大小，然後建立本地 trampoline：先執行原始的第一條指令，再跳轉到 instruction+1，使被掛鉤的序言從不被執行，且返回位址檢查仍看到合法的呼叫來源。
主要用於作弊開發者繞過反作弊 API 鉤子，以及遊戲安全研究者研究基於 trampoline 的鉤子規避技術。
