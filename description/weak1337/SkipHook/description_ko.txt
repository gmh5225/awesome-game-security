대상 함수의 첫 번째 명령을 건너뛰는 함수 호출 트램폴린을 만드는 헤더 전용 C++ 라이브러리로, BattlEye 같은 안티치트가 WinAPI 및 게임 함수에 설치한 인라인 훅(0xE9 JMP)과 브레이크포인트 트랩(0xCC INT3)을 우회합니다.
HDE(Hacker Disassembly Engine)로 x86/x64 명령 길이를 디코딩해 첫 명령 크기를 구한 뒤, 원래 첫 명령을 실행하고 instruction+1로 점프하는 로컬 트램폴린을 만들어 훅된 프롤로그가 실행되지 않게 하며, 반환 주소 검사에도 정상적인 호출 원점으로 보입니다.
안티치트 API 훅을 우회하려는 치트 개발자와 트램폴린 기반 훅 회피 기법을 연구하는 게임 보안 연구자에게 주로 유용합니다.
