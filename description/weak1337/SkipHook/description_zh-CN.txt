一个仅头文件的 C++ 库，用于创建函数调用 trampoline，跳过目标函数的第一条指令，从而绕过 BattlEye 等反作弊系统在 WinAPI 与游戏函数上放置的内联钩子（0xE9 JMP）和断点陷阱（0xCC INT3）。
它使用 HDE（Hacker Disassembly Engine）进行 x86/x64 指令长度解码以确定第一条指令的大小，然后构建本地 trampoline：先执行原始的第一条指令，再跳转到 instruction+1，使被挂钩的序言从不被执行，且返回地址检查仍看到合法的调用来源。
主要用于作弊开发者绕过反作弊 API 钩子，以及游戏安全研究者研究基于 trampoline 的钩子规避技术。
