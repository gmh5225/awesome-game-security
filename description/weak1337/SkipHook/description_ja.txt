ターゲット関数の最初の命令をスキップする関数呼び出しトランポリンを作成するヘッダーオンリーの C++ ライブラリで、BattlEye などのアンチチートが WinAPI やゲーム関数に仕掛けたインラインフック（0xE9 JMP）やブレークポイントトラップ（0xCC INT3）を迂回します。
HDE（Hacker Disassembly Engine）で x86/x64 の命令長をデコードして最初の命令サイズを求め、元の最初の命令を実行してから instruction+1 へジャンプするローカルトランポリンを構築するため、フックされたプロローグは実行されず、戻りアドレス検査も正当な呼び出し元として認識します。
アンチチートの API フックを回避するチート開発者や、トランポリンベースのフック回避手法を研究するゲームセキュリティ研究者に主に役立ちます。
