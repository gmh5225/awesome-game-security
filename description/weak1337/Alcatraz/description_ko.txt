ImGui 기반 GUI를 갖춘 x64 PE 바이너리 난독화 도구로, 컴파일된 실행 파일에 여러 변환 패스를 적용합니다. 명령어 변형(MOV, ADD, LEA를 동등한 명령어 시퀀스로 재작성), 제어 흐름 평탄화(분기 구조를 디스패처 기반 상태 머신으로 변환), 디스어셈블리 방해용 정크 삽입, IAT 난독화, 그리고 사용자 지정 엔트리 포인트 리다이렉션입니다.
디스어셈블리에 Zydis, 코드 생성에 AsmJit을 사용하며, 커스텀 PDB 파서로 PE 섹션과 PDB 심볼을 파싱한 뒤 명령어 수준에서 변환을 적용하고, 재배치와 import를 유지한 채 수정된 바이너리를 재조립합니다.
주로 PE 수준 코드 난독화 기법을 연구하는 게임 보안 연구자와, 바이너리 보호 강도를 평가하는 안티치트 개발자에게 유용합니다.
