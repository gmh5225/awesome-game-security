DetectTpmSpoofing 是一款輕量級 Windows 核心驅動程式，用於偵測 TPM 2.0 回應是否被 TPM 裝置堆疊上的 hook 偽造。它以 C/C++ 撰寫為 KMDF 驅動程式，並以 CMake 與 Windows Driver Kit 建置，會將經一般 IOCTL 路徑取得的 TPM2_ReadPublic 結果，與從 TPM.sys 內部快取回應緩衝區讀取的相同資料進行比對。若 FNV-1a 雜湊不一致，即表示存在偽造硬體身分所用公開金鑰或認可金鑰的篩選器或 DeviceIoControl hook。此專案面向反作弊與遊戲安全研究人員，用於找出在不觸碰真實 TPM 的情況下偽造 TPM 證明的 HWID 欺騙工具。
