DetectTpmSpoofing — небольшой драйвер ядра Windows, который определяет, были ли ответы TPM 2.0 подделаны с помощью хуков на стеке устройств TPM. Написан на C/C++ как драйвер KMDF, собирается с помощью CMake и Windows Driver Kit; сравнивает результат TPM2_ReadPublic, полученный обычным путём IOCTL, с теми же данными из внутренних кэшированных буферов ответов TPM.sys. Несовпадение хешей FNV-1a указывает на фильтр или хук DeviceIoControl, подделывающий открытые ключи или endorsement keys, используемые для аппаратной идентификации. Предназначен для исследователей античита и игровой безопасности, которым нужно выявлять HWID-спуферы, подделывающие TPM-аттестацию без обращения к настоящему TPM.
