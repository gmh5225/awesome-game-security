DetectTpmSpoofing은 TPM 장치 스택의 훅으로 TPM 2.0 응답이 스푸핑되었는지 탐지하는 소형 Windows 커널 드라이버입니다. C/C++로 작성된 KMDF 드라이버로 CMake와 Windows Driver Kit으로 빌드되며, 일반 IOCTL 경로로 얻은 TPM2_ReadPublic 결과를 TPM.sys 내부 캐시 응답 버퍼에서 읽은 동일 데이터와 비교합니다. FNV-1a 해시가 불일치하면 하드웨어 신원에 쓰이는 공개 키나 보증 키를 위조하는 필터 또는 DeviceIoControl 훅이 있음을 나타냅니다. 실제 TPM을 건드리지 않고 TPM 증명(attestation)을 위조하는 HWID 스푸퍼를 찾아내야 하는 안티치트 및 게임 보안 연구자를 대상으로 합니다.
