DetectTpmSpoofing は、TPM デバイススタック上のフックによって TPM 2.0 応答が偽装されているかを検出する、小型の Windows カーネルドライバです。C/C++ で書かれた KMDF ドライバで、CMake と Windows Driver Kit によりビルドされ、通常の IOCTL 経路で得た TPM2_ReadPublic の結果を、TPM.sys 内部のキャッシュ応答バッファから読み取った同一データと比較します。FNV-1a ハッシュが一致しない場合、ハードウェア識別に使われる公開鍵や承認鍵を偽造するフィルタまたは DeviceIoControl フックが存在することを示します。本物の TPM に触れずに TPM アテステーションを偽る HWID スプーファーを見つけたいアンチチート・ゲームセキュリティ研究者向けです。
