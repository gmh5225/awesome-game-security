DetectTpmSpoofing 是一款轻量级 Windows 内核驱动，用于检测 TPM 2.0 响应是否被 TPM 设备栈上的 hook 伪造。它以 C/C++ 编写为 KMDF 驱动，使用 CMake 与 Windows Driver Kit 构建，会将经普通 IOCTL 路径得到的 TPM2_ReadPublic 结果，与从 TPM.sys 内部缓存响应缓冲区读取的同一数据加以比对。若 FNV-1a 哈希不一致，则表明存在伪造硬件身份所用公钥或认可密钥的过滤器或 DeviceIoControl hook。该项目面向反作弊与游戏安全研究人员，用于发现在不触碰真实 TPM 的情况下伪造 TPM 证明的 HWID 欺骗工具。
