Этот проект — дампер процедур расшифровки на C++ для Windows, который во время выполнения отслеживает и восстанавливает логику зашифрованных указателей. Он запускает цель под отладчиком, выполняет инструкции по шагам и с помощью дизассемблирования на базе Zydis отслеживает зависимости регистров и стека, пока не будет получен результат расшифровки. Инструмент включает поиск по шаблонам, восстановление контекста и фильтрацию инструкций для более чистых восстановленных потоков инструкций. Он в первую очередь полезен для реверс-инжиниринга защищённых игровых бинарников и изучения поведения расшифровки под защитой античита.
