Este proyecto es un dumper de rutinas de descifrado en C++ para Windows que rastrea y reconstruye la lógica de punteros cifrados en tiempo de ejecución. Lanza el objetivo bajo un depurador, ejecuta instrucciones paso a paso y usa desensamblado basado en Zydis para seguir dependencias de registros y pila hasta resolver la salida del descifrado. La herramienta incluye escaneo de patrones, restauración de contexto y filtrado de instrucciones para producir flujos de instrucciones recuperados más limpios. Es útil principalmente para la ingeniería inversa de binarios de juegos protegidos y el estudio del comportamiento de descifrado bajo protección anti-cheat.
