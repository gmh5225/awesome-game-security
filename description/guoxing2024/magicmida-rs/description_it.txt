Magicmida-RS è un unpacker automatico di Themida per Windows scritto in Rust, come reimplementazione modulare dello strumento originale Magicmida in Pascal. Rileva e spacchetta gli eseguibili protetti con Themida v1/v2/v3 (x86 e x64) avviando il target sotto un debugger Win32 Debug API, individuando l’original entry point (OEP), eseguendo il dump della memoria di processo e ricostruendo le tabelle di import e delle sezioni del PE. Le funzionalità principali includono la ricostruzione runtime dell’IAT tramite live memory tracing, la scoperta dell’OEP con euristiche MSVC CRT e Go/Delphi, l’integrazione opzionale di ScyllaHide per aggirare l’anti-debugging sulle build più recenti e una modalità verify che confronta l’output unpackato con un riferimento noto. È pensato per reverse engineer e ricercatori di game security che devono analizzare o recuperare binari protetti da Themida.
