Magicmida-RS は、オリジナルの Pascal 製 Magicmida をモジュール構成で再実装した、Rust 製の Windows Themida 自動アンパッカーです。Win32 Debug API デバッガで対象を起動し、オリジナルエントリポイント（OEP）を特定してプロセスメモリをダンプし、PE のインポート表とセクション表を再構築することで、Themida v1/v2/v3 で保護された実行ファイル（x86 および x64）を検出・アンパックします。主な機能には、ライブメモリトレーシングによる実行時 IAT 再構築、MSVC CRT および Go/Delphi ヒューリスティックによる OEP 検出、新しいビルドでのアンチデバッグ回避向けの任意の ScyllaHide 連携、既知の正しい参照との差分による検証モードがあります。Themida で保護されたバイナリの解析や復元を必要とするリバースエンジニアおよびゲームセキュリティ研究者向けです。
