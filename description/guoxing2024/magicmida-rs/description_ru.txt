Magicmida-RS — автоматический распаковщик Themida для Windows на языке Rust, модульная переработка оригинального инструмента Magicmida на Pascal. Он обнаруживает и распаковывает исполняемые файлы, защищённые Themida v1/v2/v3 (x86 и x64), запуская цель под отладчиком Win32 Debug API, находя оригинальную точку входа (OEP), дампя память процесса и восстанавливая таблицы импорта и секций PE. Ключевые возможности включают восстановление IAT во время выполнения через трассировку живой памяти, поиск OEP с эвристиками MSVC CRT и Go/Delphi, опциональную интеграцию ScyllaHide для обхода антиотладки на новых сборках и режим verify, сравнивающий распакованный результат с эталонным образцом. Инструмент предназначен для реверс-инженеров и исследователей игровой безопасности, которым нужно анализировать или восстанавливать бинарники, защищённые Themida.
