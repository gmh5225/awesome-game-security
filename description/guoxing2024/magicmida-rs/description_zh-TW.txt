Magicmida-RS 是以 Rust 撰寫的 Windows Themida 自動脫殼工具，作為原版 Pascal Magicmida 工具的模組化重實作。它透過 Win32 Debug API 除錯器啟動目標程序，找出原始進入點（OEP），傾印行程記憶體並重建 PE 匯入表與區段表，以偵測並脫殼 Themida v1/v2/v3 保護的可執行檔（x86 與 x64）。主要功能包括：透過即時記憶體追蹤進行執行期 IAT 重建、結合 MSVC CRT 與 Go/Delphi 啟發式的 OEP 發現、可選的 ScyllaHide 整合以繞過較新版本上的反除錯，以及將脫殼結果與已知正確參考進行比對的驗證模式。面向需要分析或還原 Themida 保護二進位檔的逆向工程師與遊戲安全研究人員。
