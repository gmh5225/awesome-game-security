Magicmida-RS 是用 Rust 编写的 Windows Themida 自动脱壳工具，作为原版 Pascal Magicmida 工具的模块化重实现。它通过 Win32 Debug API 调试器启动目标进程，发现原始入口点（OEP），转储进程内存并重建 PE 导入表与节表，从而检测并脱壳 Themida v1/v2/v3 保护的可执行文件（x86 与 x64）。主要能力包括：基于实时内存追踪的运行时 IAT 重建、结合 MSVC CRT 与 Go/Delphi 启发式的 OEP 发现、可选的 ScyllaHide 集成以绕过较新版本上的反调试，以及将脱壳结果与已知正确参考进行对比的验证模式。面向需要分析或恢复 Themida 保护二进制的逆向工程师与游戏安全研究人员。
