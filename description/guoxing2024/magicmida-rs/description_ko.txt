Magicmida-RS는 원본 Pascal Magicmida 도구를 모듈형으로 재구현한 Rust 기반 Windows Themida 자동 언패커입니다. Win32 Debug API 디버거로 대상을 실행한 뒤 원본 진입점(OEP)을 찾고, 프로세스 메모리를 덤프하며 PE 임포트 및 섹션 테이블을 재구성하여 Themida v1/v2/v3로 보호된 실행 파일(x86 및 x64)을 탐지하고 언팩합니다. 주요 기능으로는 실시간 메모리 추적을 통한 런타임 IAT 재구성, MSVC CRT 및 Go/Delphi 휴리스틱을 활용한 OEP 탐지, 최신 빌드의 안티디버깅을 우회하기 위한 선택적 ScyllaHide 연동, 그리고 언팩 결과를 검증용 참조와 비교하는 verify 모드가 있습니다. Themida로 보호된 바이너리를 분석하거나 복구해야 하는 리버스 엔지니어와 게임 보안 연구자를 대상으로 합니다.
