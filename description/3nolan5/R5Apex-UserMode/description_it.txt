Questo progetto è un framework esterno in user-mode per la ricerca sul cheating in Apex Legends.
È scritto in C++ e dimostra la comunicazione tramite memoria condivisa con un driver kernel mappato per operazioni di lettura e scrittura sui processi.
Il set di funzionalità di esempio include la manipolazione di glow e highlight, insieme a un'architettura di base minimale pensata per l'estensione.
Il caso d'uso principale è la sperimentazione didattica sul design di cheat esterni e sui pattern di accesso alla memoria assistiti da driver.
