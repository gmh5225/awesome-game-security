Este proyecto es un framework externo en modo usuario para la investigación de trampas en Apex Legends.
Está escrito en C++ y demuestra la comunicación por memoria compartida con un controlador de kernel mapeado para operaciones de lectura y escritura de procesos.
El conjunto de funciones de ejemplo incluye manipulación de glow y highlight, junto con una arquitectura base mínima pensada para ampliarse.
Su caso de uso principal es la experimentación educativa con el diseño de cheats externos y patrones de acceso a memoria asistidos por controlador.
