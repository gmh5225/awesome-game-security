Этот проект — внешний user-mode фреймворк для исследования читерства в Apex Legends.
Он написан на C++ и демонстрирует обмен данными через разделяемую память с отображённым драйвером ядра для операций чтения и записи процесса.
Набор примеров включает манипуляцию glow и highlight, а также минимальную базовую архитектуру для расширения.
Основное применение — образовательные эксперименты с проектированием внешних читов и паттернами доступа к памяти с помощью драйвера.
