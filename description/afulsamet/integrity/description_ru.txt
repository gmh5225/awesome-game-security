integrity — это header-only библиотека на C для проверки целостности памяти образов Windows PE во время выполнения. Она вычисляет базовые контрольные суммы для неперезаписываемых секций и позже повторно проверяет их, чтобы обнаружить несанкционированные изменения кода. Реализация поддерживает аппаратно ускоренный CRC32 с SSE4.2 и позволяет задавать пользовательские алгоритмы контрольных сумм через конфигурацию на этапе компиляции. Библиотека ориентирована на защитные сценарии: обнаружение вмешательства, усиление античита и эксперименты с самозащитой во время выполнения.
