integrity es una biblioteca C de solo cabeceras para la verificación de integridad de memoria en tiempo de ejecución de imágenes PE de Windows. Calcula sumas de comprobación de referencia para las secciones no escribibles y las vuelve a comprobar más tarde para detectar modificaciones de código no autorizadas. La implementación admite CRC32 acelerado por hardware con SSE4.2 y permite algoritmos de suma de comprobación personalizados mediante configuración en tiempo de compilación. Esta biblioteca está orientada a casos de uso defensivos como la detección de manipulación, el endurecimiento anti-cheat y experimentos de autoprotección en tiempo de ejecución.
