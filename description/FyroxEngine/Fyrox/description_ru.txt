Этот проект — функционально богатый, готовый к продакшену 2D- и 3D-игровой движок на Rust со встроенным редактором сцен. Он предоставляет широкую экосистему движка с инструментами редактора, crate'ами движка, примерами и учебными материалами для полноценных рабочих процессов разработки игр. Архитектура делает акцент на нативной разработке на Rust, модульных crate'ах и практичном авторстве через итерации, управляемые редактором. Основной сценарий использования — создание и исследование открытых игровых технологий на современном Rust.
