Questo progetto è un motore di gioco 2D e 3D ricco di funzionalità e pronto per la produzione, scritto in Rust, con un editor di scene integrato. Fornisce un ampio ecosistema del motore con strumenti di editing, crate del motore, esempi e risorse formative per flussi di lavoro completi di sviluppo giochi. L'architettura privilegia lo sviluppo nativo in Rust, crate modulari e un authoring pratico tramite iterazione guidata dall'editor. Il caso d'uso principale è costruire e ricercare tecnologie di gioco open source in Rust moderno.
