Ce projet se concentre sur une version refactorisée de tokenext.
Consultez les démos page_2_svg : version initiale ; ajout de pointeurs vers le tampon local (en vert), pointeurs vers les symboles (en rouge) ; ajout de pointeurs vers les allocations du tas (en bleu) ; ajout de l'historique des changements d'allocation du tas, les rectangles bleus sont cliquables.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / plugins WinDbg.
