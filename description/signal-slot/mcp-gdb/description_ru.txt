Этот проект — MCP-сервер (Model Context Protocol), который предоставляет функциональность отладчика GDB ИИ-ассистентам для разработки. Он позволяет LLM устанавливать точки останова, просматривать память, читать регистры, пошагово выполнять код и анализировать состояние программы через машинный интерфейс GDB посредством вызовов протокола MCP. Инструмент обеспечивает рабочие процессы отладки с помощью ИИ. Предназначен для разработчиков и специалистов по реверс-инжинирингу, использующих ИИ-инструменты для автоматизированной отладки и анализа бинарных файлов через GDB.
