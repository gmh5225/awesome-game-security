Ce projet est un serveur MCP (Model Context Protocol) qui expose les fonctionnalités du débogueur GDB aux assistants de codage IA. Il permet aux LLM de poser des points d'arrêt, d'inspecter la mémoire, de lire les registres, d'exécuter le code pas à pas et d'analyser l'état du programme via l'interface machine de GDB par des appels au protocole MCP. L'outil permet des flux de travail de débogage assistés par l'IA. Il s'adresse aux développeurs et ingénieurs en rétro-ingénierie qui utilisent des outils d'IA pour le débogage automatisé et l'analyse binaire via GDB.
