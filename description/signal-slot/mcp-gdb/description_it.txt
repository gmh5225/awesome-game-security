Questo progetto è un server MCP (Model Context Protocol) che espone le funzionalità del debugger GDB agli assistenti di coding basati su IA. Consente agli LLM di impostare breakpoint, ispezionare la memoria, leggere i registri, eseguire il codice passo passo e analizzare lo stato del programma tramite l'interfaccia macchina di GDB mediante chiamate al protocollo MCP. Lo strumento abilita flussi di lavoro di debugging assistito dall'IA. È rivolto a sviluppatori e reverse engineer che usano strumenti di IA per il debugging automatizzato e l'analisi binaria tramite GDB.
