Dieses Projekt ist ein MCP-Server (Model Context Protocol), der GDB-Debugger-Funktionen für KI-Coding-Assistenten bereitstellt. Es ermöglicht LLMs, über MCP-Protokollaufrufe Breakpoints zu setzen, Speicher zu untersuchen, Register zu lesen, Code schrittweise auszuführen und den Programmzustand über die Maschinen-Schnittstelle von GDB zu analysieren. Das Werkzeug unterstützt KI-gestützte Debugging-Workflows. Es richtet sich an Entwickler und Reverse Engineers, die KI-Tools für automatisiertes Debugging und Binäranalyse mit GDB nutzen.
