Ce projet est un outil de hook basé sur une DLL pour les utilitaires de signature de code Windows. Il utilise C++ et Microsoft Detours pour intercepter les contrôles de validité des certificats et les API de signature liées aux horodatages. La conception permet de rediriger vers des points de terminaison d'horodatage personnalisés et de modifier le comportement temporel via la configuration ou des paramètres en ligne de commande. Il vise la recherche et les tests autour des flux de signature numérique, de la logique de validation des certificats et de la gestion des horodatages.
