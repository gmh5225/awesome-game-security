Este proyecto es una herramienta de hook basada en DLL para utilidades de firma de código de Windows. Usa C++ y Microsoft Detours para interceptar comprobaciones de validez de certificados y API de firma relacionadas con marcas de tiempo. El diseño permite redirigir a endpoints de marca de tiempo personalizados y modificar el comportamiento temporal mediante configuración o parámetros de línea de comandos. Está orientado a la investigación y las pruebas de flujos de firma digital, lógica de validación de certificados y gestión de marcas de tiempo.
