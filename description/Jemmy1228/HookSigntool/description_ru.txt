Этот проект — инструмент перехвата на основе DLL для утилит подписи кода Windows. Он использует C++ и Microsoft Detours для перехвата проверок действительности сертификатов и API подписи, связанных с метками времени. Через конфигурацию или параметры командной строки можно перенаправлять запросы на пользовательские конечные точки меток времени и изменять поведение, связанное со временем. Предназначен для исследований и тестирования процессов цифровой подписи, логики проверки сертификатов и обработки меток времени.
