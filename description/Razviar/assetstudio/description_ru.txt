Этот проект — многопоточный инструмент извлечения Unity-ассетов для версий от Unity 2.x до Unity 6.
Он предоставляет бинарные файлы с GUI и CLI, параллельную загрузку бандлов и параллельный экспорт для ускорения обработки больших объёмов ассетов.
Реализация в основном на C#/.NET и включает парсеры форматов, процедуры распаковки и вспомогательную логику для конкретных игр.
Используется для майнинга игровых данных, моддинга и реверс-инжиниринга Unity asset bundles.
