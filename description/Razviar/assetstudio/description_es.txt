Este proyecto es una herramienta multihilo de extracción de assets de Unity para Unity 2.x hasta Unity 6.
Proporciona binarios GUI y CLI, carga paralela de bundles y exportación paralela para acelerar el procesamiento de assets a gran escala.
La implementación es principalmente en C#/.NET e incluye analizadores de formato, rutinas de descompresión y lógica auxiliar específica de cada juego.
Se usa para minería de datos de juegos, flujos de modding e ingeniería inversa de asset bundles de Unity.
