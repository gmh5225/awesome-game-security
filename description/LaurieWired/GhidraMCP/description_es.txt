GhidraMCP es un servidor MCP y un plugin de Ghidra que permite a los clientes LLM realizar tareas de ingeniería inversa mediante llamadas estructuradas a herramientas.
Expone capacidades como descompilación, enumeración de símbolos y métodos, y renombrado automatizado de binarios cargados en Ghidra.
El proyecto combina una extensión de Ghidra en Java con componentes puente en Python para la integración con clientes de IA de escritorio.
Está pensado para ingenieros inversos y analistas de seguridad que desean flujos de análisis binario semiautomatizados.
