GhidraMCP est un serveur MCP et un plugin Ghidra qui permettent aux clients LLM d’effectuer des tâches de reverse engineering via des appels d’outils structurés.
Il expose des capacités telles que la décompilation, l’énumération des symboles et des méthodes, ainsi que le renommage automatisé des binaires chargés dans Ghidra.
Le projet combine une extension Ghidra côté Java avec des composants pont en Python pour l’intégration avec les clients d’IA de bureau.
Il est conçu pour les reverse engineers et les analystes en sécurité qui souhaitent des workflows d’analyse binaire semi-automatisés.
