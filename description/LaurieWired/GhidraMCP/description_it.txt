GhidraMCP è un server MCP e un plugin Ghidra che consente ai client LLM di eseguire attività di reverse engineering tramite chiamate strutturate agli strumenti.
Espone funzionalità come decompilazione, enumerazione di simboli e metodi e ridenominazione automatica per i binari caricati in Ghidra.
Il progetto combina un’estensione Ghidra lato Java con componenti bridge in Python per l’integrazione con client AI desktop.
È pensato per reverse engineer e analisti di sicurezza che desiderano flussi di lavoro di analisi binaria semi-automatizzati.
