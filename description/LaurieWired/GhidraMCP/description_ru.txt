GhidraMCP — это MCP-сервер и плагин Ghidra, позволяющий LLM-клиентам выполнять задачи реверс-инжиниринга через структурированные вызовы инструментов.
Он предоставляет такие возможности, как декомпиляция, перечисление символов и методов, а также автоматическое переименование бинарных файлов, загруженных в Ghidra.
Проект сочетает расширение Ghidra на стороне Java с Python-компонентами-мостами для интеграции с настольными ИИ-клиентами.
Он предназначен для реверс-инженеров и специалистов по безопасности, которым нужны полуавтоматические рабочие процессы анализа бинарных файлов.
