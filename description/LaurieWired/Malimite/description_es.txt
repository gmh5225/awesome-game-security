Malimite es una herramienta de descompilación de escritorio para analizar archivos IPA de iOS y paquetes de aplicaciones de macOS.
Está escrita principalmente en Java y se basa en flujos de trabajo de descompilación con Ghidra.
El proyecto incluye funciones como la decodificación automática de recursos de Apple, la reconstrucción de clases Swift y el soporte para binarios Swift y Objective-C.
Su caso de uso principal es la ingeniería inversa y la investigación de seguridad en aplicaciones de Apple, incluyendo el análisis de malware y la auditoría de código.
