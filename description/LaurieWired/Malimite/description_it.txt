Malimite è uno strumento di decompilazione desktop per analizzare file IPA iOS e bundle di applicazioni macOS.
È scritto principalmente in Java e costruito attorno a flussi di lavoro di decompilazione basati su Ghidra.
Il progetto include funzionalità come la decodifica automatica delle risorse Apple, la ricostruzione delle classi Swift e il supporto per binari Swift e Objective-C.
Il caso d'uso principale è il reverse engineering e la ricerca sulla sicurezza delle applicazioni Apple, inclusa l'analisi di malware e l'audit del codice.
