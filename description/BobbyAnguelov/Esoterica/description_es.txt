Esoterica es un framework de motor de juegos prototipo con licencia MIT, construido como demostrador tecnológico y plataforma educativa. El código base en C++ proporciona sistemas centrales del motor como serialización, gestión de recursos, generación de código de reflexión y herramientas de editor. También incluye componentes de tiempo de ejecución avanzados como un modelo híbrido de actores y ECS, sistemas de grafos de animación y soporte de ragdoll basado en PhysX. El proyecto está dirigido principalmente a desarrolladores de motores e investigadores que experimentan con renderizado, arquitectura de gameplay y diseño de herramientas.
