Dieses Projekt ist die Storm Engine, die quelloffene Spiel-Engine, die ursprünglich von Akella für die Serie Sea Dogs / Fluch der Karibik entwickelt wurde. Die C++-Engine umfasst einen DirectX-9-Renderer, ein Physiksystem, eine Scripting-Engine, Ressourcenverwaltung, einen Szenengraphen, Wassersimulation sowie Schiffskampfmechaniken. Die Codebasis repräsentiert eine vollständige kommerzielle Spiel-Engine aus den frühen 2000er-Jahren. Sie richtet sich an Historiker von Spiel-Engines, Hobbyentwickler und Forscher, die sich mit der Architektur früher kommerzieller Spiel-Engines beschäftigen.
