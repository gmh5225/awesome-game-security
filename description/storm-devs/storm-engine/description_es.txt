Este proyecto es el Storm Engine, el motor de juego de código abierto desarrollado originalmente por Akella para la serie Sea Dogs / Piratas del Caribe. El motor en C++ incluye un renderizador DirectX 9, sistema de física, motor de scripting, gestión de recursos, grafo de escena, simulación de agua y mecánicas de combate naval. El código representa un motor de juego comercial completo de principios de los años 2000. Está dirigido a historiadores de motores de juego, desarrolladores aficionados e investigadores que estudian la arquitectura de motores comerciales de aquella época.
