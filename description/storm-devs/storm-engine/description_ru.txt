Этот проект — Storm Engine, открытый игровой движок, изначально разработанный компанией Akella для серии Sea Dogs / Pirates of the Caribbean. Движок на C++ включает рендерер DirectX 9, систему физики, скриптовый движок, управление ресурсами, граф сцены, симуляцию воды и механики корабельных боёв. Кодовая база представляет собой полноценный коммерческий игровой движок начала 2000-х годов. Он предназначен для историков игровых движков, разработчиков-любителей и исследователей, изучающих архитектуру коммерческих игровых движков того времени.
