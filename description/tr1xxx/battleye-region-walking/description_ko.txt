이 프로젝트는 프로세스의 가상 주소 공간에 주입된 코드를 탐지하기 위해 BattlEye가 사용하는 메모리 영역 순회(memory region walking) 기법을 보여줍니다.
VirtualQuery 기반 영역 열거를 구현하며, 보호 플래그, 영역 크기, 메모리 유형, 주소 패턴을 바탕으로 의심스러운 실행 가능 메모리 영역을 식별하는 BattlEye의 필터링 로직을 적용합니다.
이 C++ 코드베이스는 MEM_PRIVATE/MEM_MAPPED 유형 검사와 BattlEye가 셸코드 및 수동 매핑 모듈을 표시하는 데 사용하는 특정 크기·주소 휴리스틱에 따라 영역을 유효 또는 무효로 분류합니다.
주로 BattlEye의 메모리 스캔 탐지 방법론을 연구하는 안티치트 연구원과 리버스 엔지니어에게 유용합니다.
