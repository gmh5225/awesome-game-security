Ce projet est un détecteur prototype d'apprentissage automatique qui classe si un échange de tirs a été assisté par un aimbot « legit ». Il utilise des outils Go pour parser les données de démo SourceTV en caractéristiques d'entraînement, et des notebooks Python avec un modèle LSTM pour l'entraînement et l'évaluation. Les signaux principaux incluent les deltas d'angle de vue et les relations angulaires crosshair-cible échantillonnées autour des événements d'attaque. Il est conçu comme une expérience de recherche anti-cheat plutôt que comme un détecteur prêt pour la production.
