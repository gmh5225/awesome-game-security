Этот проект — прототип детектора на машинном обучении, который классифицирует, был ли огневой контакт поддержан «legit» аимботом. Он использует инструменты на Go для разбора данных демо SourceTV в признаки для обучения, а также Python-ноутбуки с моделью LSTM для обучения и оценки. Ключевые сигналы включают дельты углов обзора и угловые соотношения прицел–цель, дискретизированные вокруг событий атаки. Проект задуман как исследовательский эксперимент по античиту, а не как готовый к продакшену детектор.
