Этот проект — серверная система occlusion culling, предназначенная для снижения видимости wallhack в многопользовательских шутерах. Он в основном написан на C++ с файлами проекта Unreal Engine и использует аналитические ray cast вместо грубых приближений видимости. Реализация сочетает кэширование недавних occluder, ускорение BVH и упреждающие проверки с учётом задержки, чтобы сохранять быстрый culling и уменьшать артефакты popping. Предназначен для исследований античита и игровой безопасности в области масштабируемого контроля линии видимости.
