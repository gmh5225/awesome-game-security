Questo progetto è un sistema di occlusion culling lato server pensato per ridurre la visibilità dei wallhack negli shooter multiplayer. È scritto principalmente in C++ con file di progetto Unreal Engine e usa ray cast analitici invece di approssimazioni grossolane della visibilità. L'implementazione combina caching degli occluder recenti, accelerazione BVH e controlli di lookahead sensibili alla latenza per mantenere il culling veloce riducendo al contempo gli artefatti di popping. È destinato alla ricerca su anti-cheat e sicurezza dei giochi sull'applicazione scalabile della line of sight.
