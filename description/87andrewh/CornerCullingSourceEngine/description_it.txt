Questo progetto è un'estensione anti-wallhack per il motore Source che esegue un culling della visibilità rigoroso lato server. Combina codice di estensione C++ con script SourceMod e definizioni di occluder lato mappa per controllare ciò che i giocatori possono vedere nelle partite competitive. Il sistema privilegia la correttezza del ray-cast, un basso overhead sul tempo di frame e un comportamento sicuro rispetto alla latenza per evitare il popping di visibilità negli intervalli di ping normali. È rivolto principalmente a operatori di server e ricercatori che migliorano le protezioni anti-cheat nei giochi basati su Source.
