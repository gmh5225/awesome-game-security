Этот проект — GH Offset Dumper, инструмент для Windows, автоматически дампящий смещения игровых структур и указатели интерфейсов из запущенных игровых процессов. Он использует pattern scanning (сканирование сигнатур), чтобы находить в памяти игровые структуры данных, списки сущностей, интерфейсы клиента/движка и netvars, и выводит результаты в виде заголовочных файлов или JSON. Этот инструмент на C++ часто применяют с играми на Source engine. Он предназначен для гейм-хакеров и моддеров, которым нужно автоматически обновлять смещения в памяти после патчей игры.
