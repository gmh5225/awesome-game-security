Ce projet est GH Offset Dumper, un outil Windows qui dump automatiquement les offsets de structures de jeu et les pointeurs d'interfaces depuis des processus de jeu en cours d'exécution. Il utilise le pattern scanning (signature scanning) pour localiser en mémoire les structures de données spécifiques au jeu, les listes d'entités, les interfaces client/moteur et les netvars, puis exporte les résultats en fichiers d'en-tête ou en JSON. Cet outil C++ est couramment utilisé avec les jeux Source engine. Il s'adresse aux game hackers et modders qui doivent mettre à jour automatiquement les offsets mémoire après les patches de jeu.
