Questo progetto è GH Offset Dumper, uno strumento Windows per dumpare automaticamente gli offset delle strutture di gioco e i puntatori alle interfacce da processi di gioco in esecuzione. Usa il pattern scanning (signature scanning) per individuare in memoria strutture dati specifiche del gioco, liste di entità, interfacce client/engine e netvars, esportando i risultati come file header o JSON. Lo strumento C++ è comunemente usato con i giochi Source engine. È rivolto a game hacker e modder che devono aggiornare automaticamente gli offset di memoria dopo le patch del gioco.
