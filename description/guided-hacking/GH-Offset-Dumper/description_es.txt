Este proyecto es GH Offset Dumper, una herramienta de Windows para volcar automáticamente offsets de estructuras de juego y punteros de interfaces desde procesos de juego en ejecución. Usa pattern scanning (escaneo de firmas) para localizar en memoria estructuras de datos específicas del juego, listas de entidades, interfaces de cliente/motor y netvars, y exporta los resultados como archivos de cabecera o JSON. Esta herramienta en C++ se usa habitualmente con juegos del motor Source. Está pensada para game hackers y modders que necesitan actualizar automáticamente los offsets de memoria tras los parches del juego.
