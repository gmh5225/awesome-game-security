Dieses Projekt ist ein funktionsreicher Windows-DLL-Injektor mit einer in C++ und Qt erstellten GUI. Es unterstützt mehrere Injektionsmethoden, darunter LoadLibrary, Manual Mapping, Thread Hijacking, NtCreateThreadEx, QueueUserAPC sowie Kernel-Mode-Injection über einen anfälligen Treiber. Das Tool bietet Optionen zum DLL-Cloaking (PEB-Unlinking, Header-Löschung), zur Importauflösung, zur Ausführung von TLS-Callbacks und zur Registrierung von Exception Handlern. Es richtet sich an Lernende im Bereich Game Hacking sowie an Sicherheitsforscher, die DLL-Injektionstechniken und deren Erkennungsfläche untersuchen.
