Этот проект — функциональный инжектор DLL для Windows с графическим интерфейсом на C++ и Qt. Он поддерживает несколько методов инъекции, включая LoadLibrary, manual mapping, перехват потоков (thread hijacking), NtCreateThreadEx, QueueUserAPC, а также инъекцию в режиме ядра через уязвимый драйвер. В инструменте есть опции маскировки DLL (отвязка от PEB, стирание заголовка), разрешения импортов, выполнения TLS-колбэков и регистрации обработчиков исключений. Он предназначен для изучающих game hacking и исследователей безопасности, которые анализируют техники DLL-инъекции и поверхность их обнаружения.
