Ghidra FindCrypt is a Ghidra extension that scans binaries for known cryptographic constants and algorithm signatures. It is implemented in Java as a Ghidra analyzer module, built with Gradle against a local Ghidra installation. A JSON signature database covers many common ciphers and hashes, including AES S-boxes, DES tables, MD5, SHA-1, TEA, Salsa, Keccak, and elliptic-curve related constants. Analysts use it during reverse engineering to quickly locate crypto implementations inside protected games, malware, or other closed-source software.
