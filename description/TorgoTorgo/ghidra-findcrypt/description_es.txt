FindCrypt es una extensión de Ghidra que analiza binarios en busca de constantes criptográficas conocidas y etiqueta las implementaciones coincidentes durante el análisis. Está escrita en Java y se construye con Gradle como un complemento de analizador estándar de Ghidra. Incluye una base de datos de firmas JSON con patrones de bytes para algoritmos comunes como S-boxes de AES, tablas DES, MD5, SHA-1, TEA, Salsa, CRC32 y otras constantes de cifrado o hash. El analizador compara esas firmas en la memoria del programa para que los ingenieros inversos puedan localizar rápidamente las rutinas criptográficas. Está orientada a la ingeniería inversa y al análisis de malware o de seguridad en juegos, donde identificar código de cifrado y hash en binarios desensamblados suele ser un primer paso habitual.
