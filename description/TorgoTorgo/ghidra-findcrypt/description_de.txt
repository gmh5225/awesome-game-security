FindCrypt ist eine Ghidra-Erweiterung, die Binärdateien nach bekannten kryptografischen Konstanten durchsucht und passende Implementierungen während der Analyse kennzeichnet. Sie ist in Java geschrieben und wird mit Gradle als standardmäßiges Ghidra-Analyzer-Plugin gebaut. Eine JSON-Signaturdatenbank enthält Byte-Muster für gängige Algorithmen wie AES-S-Boxen, DES-Tabellen, MD5, SHA-1, TEA, Salsa, CRC32 sowie weitere Cipher- oder Hash-Konstanten. Der Analyzer gleicht diese Signaturen im Programmspeicher ab, damit Reverse Engineers Krypto-Routinen schnell finden können. Sie richtet sich an Reverse Engineering sowie Malware- oder Game-Security-Analyse, wo das Identifizieren von Verschlüsselungs- und Hash-Code in disassemblierten Binärdateien oft ein üblicher erster Schritt ist.
