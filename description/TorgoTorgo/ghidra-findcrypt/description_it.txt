FindCrypt è un'estensione di Ghidra che analizza i binari alla ricerca di costanti crittografiche note e etichetta le implementazioni corrispondenti durante l'analisi. È scritta in Java e compilata con Gradle come plugin analizzatore standard di Ghidra. Una database di firme JSON è inclusa con pattern di byte per algoritmi comuni come S-box AES, tabelle DES, MD5, SHA-1, TEA, Salsa, CRC32 e altre costanti di cifratura o hash. L'analizzatore confronta tali firme nella memoria del programma così che i reverse engineer possano individuare rapidamente le routine crittografiche. È pensata per il reverse engineering e l'analisi di malware o di sicurezza nei giochi, dove identificare il codice di cifratura e hashing nei binari disassemblati è spesso un primo passo comune.
