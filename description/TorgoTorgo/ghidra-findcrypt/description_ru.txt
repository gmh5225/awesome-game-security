FindCrypt — расширение Ghidra, которое сканирует бинарники на известные криптографические константы и помечает совпадающие реализации во время анализа. Оно написано на Java и собирается с помощью Gradle как стандартный плагин-анализатор Ghidra. Вместе с ним поставляется база сигнатур в JSON с байтовыми шаблонами для распространённых алгоритмов: S-боксы AES, таблицы DES, MD5, SHA-1, TEA, Salsa, CRC32 и другие константы шифров или хеш-функций. Анализатор сопоставляет эти сигнатуры в памяти программы, чтобы инженеры по реверсу могли быстро находить криптографические процедуры. Оно предназначено для реверс-инжиниринга и анализа вредоносного ПО или игровой безопасности, где выявление кода шифрования и хеширования в дизассемблированных бинарниках — обычный первый шаг.
