FindCrypt est une extension Ghidra qui analyse les binaires à la recherche de constantes cryptographiques connues et étiquette les implémentations correspondantes pendant l'analyse. Elle est écrite en Java et construite avec Gradle comme plugin d'analyseur Ghidra standard. Une base de signatures JSON est fournie avec des motifs d'octets pour des algorithmes courants tels que les S-boxes AES, les tables DES, MD5, SHA-1, TEA, Salsa, CRC32, ainsi que d'autres constantes de chiffrements ou de hachage. L'analyseur fait correspondre ces signatures dans la mémoire du programme afin que les reverse engineers puissent localiser rapidement les routines cryptographiques. Elle s'adresse à l'ingénierie inverse et à l'analyse de malwares ou de sécurité des jeux, où identifier le code de chiffrement et de hachage dans des binaires désassemblés est souvent une première étape courante.
