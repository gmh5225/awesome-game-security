FindCrypt は、バイナリ内の既知の暗号定数をスキャンし、解析中に一致する実装へラベルを付ける Ghidra 拡張機能です。Java で書かれ、Gradle で標準の Ghidra アナライザプラグインとしてビルドされます。同梱の JSON シグネチャデータベースには、AES S-box、DES テーブル、MD5、SHA-1、TEA、Salsa、CRC32 など、一般的なアルゴリズムのバイトパターンやその他の暗号・ハッシュ定数が含まれます。アナライザはプログラムメモリ上でこれらのシグネチャを照合し、リバースエンジニアが暗号ルーチンをすばやく見つけられるようにします。逆アセンブルされたバイナリ内の暗号化・ハッシュ処理コードの特定が最初の一歩となる、リバースエンジニアリングやマルウェア／ゲームセキュリティ分析を想定しています。
