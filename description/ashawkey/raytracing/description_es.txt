Este proyecto es una biblioteca de trazado de rayos sobre mallas acelerada por CUDA, con aceleración BVH y enlaces de Python. Los kernels principales están escritos en CUDA y C++, mientras que el paquete de Python expone una interfaz para consultar intersecciones rayo-malla a partir de tensores de PyTorch. El repositorio incluye un ejemplo de renderizado que visualiza normales y demuestra flujos de generación de cámara y rayos. Es útil para investigación en gráficos, simulación y visión donde se necesitan consultas de rayos GPU rápidas.
