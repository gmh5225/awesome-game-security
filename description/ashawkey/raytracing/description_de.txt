Dieses Projekt ist eine CUDA-beschleunigte Mesh-Raytracing-Bibliothek mit BVH-Beschleunigung und Python-Bindings. Die Kernkernels sind in CUDA und C++ geschrieben, während das Python-Paket eine Schnittstelle zum Abfragen von Ray-Mesh-Schnittpunkten aus PyTorch-Tensoren bereitstellt. Das Repository enthält ein Renderer-Beispiel, das Normalen visualisiert und Arbeitsabläufe zur Kamera- und Ray-Erzeugung demonstriert. Es eignet sich für Grafik-, Simulations- und Vision-Forschung, wenn schnelle GPU-Ray-Abfragen benötigt werden.
