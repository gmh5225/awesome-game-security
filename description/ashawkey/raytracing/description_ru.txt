Этот проект — библиотека трассировки лучей по мешам с ускорением на CUDA, структурой BVH и привязками Python. Основные ядра написаны на CUDA и C++, а пакет Python предоставляет интерфейс для запросов пересечений луч–меш из тензоров PyTorch. В репозитории есть пример визуализатора, который показывает нормали и демонстрирует рабочие процессы генерации камеры и лучей. Полезен для исследований в графике, симуляции и компьютерном зрении, где нужны быстрые GPU-запросы лучей.
