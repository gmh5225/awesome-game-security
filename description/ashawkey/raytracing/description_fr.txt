Ce projet est une bibliothèque de lancer de rayons sur maillages accélérée par CUDA, avec accélération BVH et liaisons Python. Les noyaux principaux sont écrits en CUDA et C++, tandis que le paquet Python expose une interface pour interroger les intersections rayon-maillage à partir de tenseurs PyTorch. Le dépôt inclut un exemple de rendu qui visualise les normales et illustre les flux de génération de caméra et de rayons. Il est utile pour la recherche en graphisme, simulation et vision lorsque des requêtes de rayons GPU rapides sont nécessaires.
