Этот проект — прототип драйвера античита уровня ядра Windows, который сканирует систему на подозрительные признаки читерства.
Он написан на C/C++ с использованием драйверных проектов Visual Studio и сосредоточен на низкоуровневых проверках целостности хоста.
Проверки охватывают неподписанные или аномальные драйверы, злоупотребление handle физической памяти, следы гипервизора, артефакты big pool, следы mapper и подозрительные системные потоки.
Код предназначен для исследований античита и экспериментов с техниками обнаружения в режиме ядра.
