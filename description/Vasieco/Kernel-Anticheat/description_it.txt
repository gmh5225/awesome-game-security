Questo progetto è un driver prototipo di anti-cheat a livello di kernel Windows che analizza un sistema alla ricerca di indicatori sospetti di cheating.
È scritto in C/C++ con progetti driver di Visual Studio e si concentra su controlli di integrità dell'host a basso livello.
I suoi controlli riguardano driver non firmati o anomali, abuso di handle della memoria fisica, tracce di hypervisor, artefatti big pool, tracce di mapper e thread di sistema sospetti.
Il codice è rivolto alla ricerca anti-cheat e alla sperimentazione di tecniche di rilevamento in modalità kernel.
