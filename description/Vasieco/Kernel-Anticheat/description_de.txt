Dieses Projekt ist ein Windows-Kernel-Anti-Cheat-Prototyp-Treiber, der ein System auf verdächtige Cheat-Indikatoren scannt.
Es ist in C/C++ mit Visual-Studio-Treiberprojekten geschrieben und konzentriert sich auf Low-Level-Integritätsprüfungen des Hosts.
Die Prüfungen umfassen unsignierte oder anomale Treiber, Missbrauch physischer Speicher-Handles, Hypervisor-Spuren, Big-Pool-Artefakte, Mapper-Spuren und verdächtige Systemthreads.
Der Code richtet sich an Anti-Cheat-Forschung und Experimente mit Erkennungstechniken im Kernelmodus.
