Ce projet est un pilote prototype d'anti-cheat au niveau du noyau Windows qui analyse un système à la recherche d'indicateurs de triche suspects.
Il est écrit en C/C++ avec des projets de pilotes Visual Studio et se concentre sur des contrôles d'intégrité de l'hôte à bas niveau.
Ses contrôles couvrent les pilotes non signés ou anormaux, l'abus de handles de mémoire physique, les traces d'hyperviseur, les artefacts de big pool, les traces de mapper et les threads système suspects.
Le code vise la recherche anti-cheat et l'expérimentation de techniques de détection en mode noyau.
