IDARustHelper — плагин для IDA Pro, улучшающий анализ бинарных файлов на Rust.
Написан на Python: выполняет demangle символов Rust, нормализует имена для совместимости с IDA и добавляет распространённые определения типов Rust.
Также включает средства восстановления строк с учётом архитектуры для целей x86, ARM и RISC-V.
Инструмент предназначен для реверс-инженеров, которым нужна более быстрая и удобная навигация по вредоносному ПО на Rust, игровым клиентам или системным бинарным файлам.
