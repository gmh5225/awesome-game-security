IDARustHelper ist ein IDA-Pro-Plugin, das die Analyse von Rust-Binärdateien verbessert.
In Python geschrieben, demangelt es Rust-Symbole, normalisiert Namen für die IDA-Kompatibilität und fügt gängige Rust-Typdefinitionen hinzu.
Es enthält außerdem architekturbezogene Hilfen zur String-Wiederherstellung für x86-, ARM- und RISC-V-Ziele.
Das Tool richtet sich an Reverse Engineers, die eine schnellere und übersichtlichere Navigation in Rust-Malware, Spieleclients oder Systembinärdateien benötigen.
