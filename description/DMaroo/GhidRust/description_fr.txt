Ce projet est une extension Ghidra pour analyser les binaires Rust et améliorer les flux de reverse engineering. Elle ajoute des heuristiques de détection des binaires Rust, intègre le matching Function ID pour les fonctions courantes de la bibliothèque standard Rust, et expérimente la traduction de la sortie décompilée de type C vers un code de style Rust. Le plugin est principalement implémenté en Java et empaqueté comme une extension Ghidra standard avec des assets de données associés. Il est utile aux reverse engineers confrontés à des exécutables Rust stripés, bien que le dépôt soit actuellement en maintenance suspendue.
