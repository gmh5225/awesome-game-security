Этот проект — расширение Ghidra для анализа бинарных файлов Rust и улучшения рабочих процессов реверс-инжиниринга. Оно добавляет эвристики обнаружения Rust-бинарников, интегрирует сопоставление Function ID для распространённых функций стандартной библиотеки Rust и экспериментирует с переводом декомпилированного C-подобного вывода в код в стиле Rust. Плагин в основном реализован на Java и упакован как стандартное расширение Ghidra с сопутствующими данными. Оно полезно реверс-инженерам, работающим со stripped-исполняемыми файлами Rust, хотя репозиторий сейчас находится в состоянии приостановленного сопровождения.
