本プロジェクトは、Rust バイナリの解析とリバースエンジニアリング作業の改善を目的とした Ghidra 拡張です。Rust バイナリ検出のヒューリスティックを追加し、よく使われる Rust 標準ライブラリ関数向けの Function ID マッチングを統合し、デコンパイルされた C 風の出力を Rust 風のコードへ近づける実験も行っています。プラグインは主に Java で実装され、サポート用データ資産とともに標準の Ghidra 拡張としてパッケージ化されています。シンボルが除去された Rust 実行ファイルを扱うリバースエンジニアにとって有用ですが、リポジトリは現在メンテナンスが一時停止されています。
