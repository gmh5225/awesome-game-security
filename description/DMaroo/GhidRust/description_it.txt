Questo progetto è un'estensione Ghidra per analizzare i binari Rust e migliorare i flussi di reverse engineering. Aggiunge euristiche di rilevamento dei binari Rust, integra il matching Function ID per le funzioni comuni della libreria standard Rust e sperimenta la traduzione dell'output decompilato in stile C verso codice in stile Rust. Il plugin è implementato principalmente in Java e confezionato come estensione Ghidra standard con asset di dati di supporto. È utile per i reverse engineer che lavorano con eseguibili Rust stripped, sebbene il repository sia attualmente in uno stato di manutenzione in pausa.
