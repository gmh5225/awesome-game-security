이 프로젝트는 Rust 바이너리를 분석하고 리버스 엔지니어링 워크플로를 개선하기 위한 Ghidra 확장입니다. Rust 바이너리 탐지 휴리스틱을 추가하고, 일반적인 Rust 표준 라이브러리 함수에 대한 Function ID 매칭을 통합하며, 디컴파일된 C 스타일 출력을 Rust 스타일 코드로 변환하는 실험도 수행합니다. 플러그인은 주로 Java로 구현되며, 지원 데이터 자산과 함께 표준 Ghidra 확장으로 패키징됩니다. 심볼이 제거된 Rust 실행 파일을 다루는 리버스 엔지니어에게 유용하지만, 저장소는 현재 유지보수가 일시 중지된 상태입니다.
