Este proyecto es una extensión de Ghidra para analizar binarios de Rust y mejorar los flujos de ingeniería inversa. Añade heurísticas de detección de binarios Rust, integra el emparejamiento Function ID para funciones habituales de la biblioteca estándar de Rust y experimenta con traducir la salida descompilada de estilo C hacia código de estilo Rust. El plugin está implementado principalmente en Java y se empaqueta como una extensión estándar de Ghidra con activos de datos de apoyo. Es útil para ingenieros inversos que trabajan con ejecutables Rust sin símbolos (stripped), aunque el repositorio se encuentra en un estado de mantenimiento en pausa.
