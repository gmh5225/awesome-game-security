Ce projet fournit une liste.
Il est organisé sous forme de documentation et de matériel de référence pour le domaine cheat / CVE du noyau Android, plutôt que comme une base de code d'exécution autonome.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine cheat / CVE du noyau Android.
