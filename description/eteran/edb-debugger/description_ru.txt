Этот проект — edb (Evan's Debugger), кроссплатформенный двоичный отладчик для Linux, вдохновлённый OllyDbg. Он предоставляет GUI на Qt с окном дизассемблера, отображением регистров, просмотром карты памяти, стеком, управлением точками останова и расширяемостью через плагины. Отладчик на C++ использует ptrace для управления процессами и поддерживает двоичные файлы x86 и x86-64, а также пошаговое выполнение на уровне инструкций, поиск в памяти и плагины анализа бинарников. Он ориентирован на reverse engineers и разработчиков эксплойтов под Linux, которым нужен графический отладчик с рабочим процессом в стиле OllyDbg.
