Este proyecto es edb (Evan's Debugger), un depurador binario multiplataforma para Linux inspirado en OllyDbg. Ofrece una GUI basada en Qt con vista de desensamblado, visualización de registros, exploración del mapa de memoria, vista de pila, gestión de puntos de interrupción y extensibilidad mediante plugins. El depurador en C++ usa ptrace para el control de procesos y admite binarios x86 y x86-64, con funciones como ejecución paso a paso a nivel de instrucción, búsqueda en memoria y plugins de análisis binario. Está pensado para ingenieros inversos y desarrolladores de exploits en Linux que necesitan un depurador gráfico con un flujo de trabajo al estilo de OllyDbg.
