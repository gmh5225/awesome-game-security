本プロジェクトは edb（Evan's Debugger）であり、OllyDbg に着想を得た Linux 向けクロスプラットフォームのバイナリデバッガです。Qt ベースの GUI を備え、逆アセンブリ表示、レジスタ表示、メモリマップ閲覧、スタックビュー、ブレークポイント管理、プラグインによる拡張性を提供します。C++ で実装されたこのデバッガはプロセス制御に ptrace を用い、x86 および x86-64 バイナリをサポートし、命令単位のステップ実行、メモリ検索、バイナリ解析プラグインなどの機能を持ちます。OllyDbg 風の作業フローを持つグラフィカルデバッガを必要とする Linux のリバースエンジニアやエクスプロイト開発者を対象としています。
