Este repositorio es un framework de cheat externo dirigido al juego Rust, con componentes de kernel y modo usuario separados. El controlador de Windows gestiona las solicitudes de lectura y escritura de memoria de proceso mediante IOCTLs, mientras que el lado cliente incluye renderizado de overlay y ayudantes SDK de gameplay. El código hace referencia al manejo de los módulos UnityPlayer y GameAssembly, lo que indica interacción con la memoria de un juego basado en Unity. Se usa principalmente para experimentos de desarrollo de cheats y para investigación anti-cheat sobre patrones de ataque asistidos por un controlador externo.
