Ce dépôt est un framework de cheat externe ciblant le jeu Rust, avec des composants noyau et mode utilisateur séparés. Le pilote Windows traite les requêtes de lecture et d'écriture de la mémoire processus via des IOCTL, tandis que le côté client inclut le rendu d'overlay et des helpers SDK de gameplay. Le code fait référence à la gestion des modules UnityPlayer et GameAssembly, indiquant une interaction mémoire avec un jeu basé sur Unity. Il sert principalement aux expériences de développement de cheats et à la recherche anti-cheat sur les schémas d'attaque assistés par un pilote externe.
