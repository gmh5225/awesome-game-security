Dieses Repository ist ein externes Cheat-Framework für das Spiel Rust mit getrennten Kernel- und User-Mode-Komponenten. Der Windows-Treiber verarbeitet Lese- und Schreibanfragen auf den Prozessspeicher über IOCTLs, während die Client-Seite Overlay-Rendering und Gameplay-SDK-Hilfen umfasst. Der Code bezieht sich auf die Handhabung der Module UnityPlayer und GameAssembly und deutet damit auf speicherbasierte Interaktion mit einem Unity-Spiel hin. Es dient vor allem Cheat-Entwicklungsexperimenten und Anti-Cheat-Forschung zu externen, treiberunterstützten Angriffsmustern.
