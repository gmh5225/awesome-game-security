Questo repository è un framework di cheat esterno mirato al gioco Rust, con componenti kernel e user-mode separati. Il driver Windows gestisce le richieste di lettura e scrittura della memoria di processo tramite IOCTL, mentre il lato client include il rendering overlay e helper SDK di gameplay. Il codice fa riferimento alla gestione dei moduli UnityPlayer e GameAssembly, indicando un'interazione con la memoria di un gioco basato su Unity. È usato principalmente per esperimenti di sviluppo di cheat e per la ricerca anti-cheat sui pattern di attacco assistiti da driver esterni.
