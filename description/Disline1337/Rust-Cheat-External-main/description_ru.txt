Этот репозиторий — внешний фреймворк чита, нацеленный на игру Rust, с отдельными компонентами ядра и пользовательского режима. Драйвер Windows обрабатывает запросы чтения и записи памяти процесса через IOCTL, а клиентская часть включает отрисовку оверлея и вспомогательные средства gameplay SDK. Код ссылается на работу с модулями UnityPlayer и GameAssembly, что указывает на взаимодействие с памятью игры на базе Unity. Он в основном используется для экспериментов по разработке читов и исследований античита, связанных с внешними атаками с помощью драйвера.
