Dieses Projekt konzentriert sich auf eine plattformneutrale dynamische Analyseplattform.
QEMU kann eine vollständige Maschine rein in Software emulieren, ohne Hardware-Virtualisierungsunterstützung zu benötigen.
Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheat / QEMU / KVM / PVE / VBox untersuchen.
