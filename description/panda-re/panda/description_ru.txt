Этот проект посвящён платформе для архитектурно-нейтрального динамического анализа.
QEMU способен полностью эмулировать машину программно, без поддержки аппаратной виртуализации.
Он особенно полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / QEMU / KVM / PVE / VBox.
