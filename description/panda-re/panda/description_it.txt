Questo progetto si concentra su una piattaforma di analisi dinamica indipendente dall'architettura.
QEMU è in grado di emulare una macchina completa interamente in software, senza bisogno di supporto alla virtualizzazione hardware.
È particolarmente utile per i ricercatori di sicurezza nei videogiochi e per i reverse engineer che studiano tecniche offensive nell'ambito di cheat / QEMU / KVM / PVE / VBox.
