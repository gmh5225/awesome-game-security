本プロジェクトは、モバイル向けアプリ内保護とセキュリティ監視のための freeRASP ランタイムアプリケーション自己保護 SDK を統合した Unity プラグインです。ネイティブの Android（Java）および iOS（Swift/Objective-C++）ブリッジを C# API の背後にラップし、Unity ゲームやアプリが Talsec を初期化し、脅威コールバックを受け取り、ランタイム攻撃に応答できるようにします。主な検知対象には、root および jailbreak、Frida などのフッキングフレームワーク、エミュレータやシミュレータ、アプリ整合性と非公式インストール、デバッグ、スクリーンショットや画面録画、ならびに時刻や位置の偽装といった関連するデバイス状態リスクが含まれます。統合と検証用のサンプル Unity テストアプリも含まれています。主な用途は、Android および iOS 上でリバースエンジニアリング、改ざん、侵害された OS 環境に対する freemium RASP 防御を必要とする Unity モバイル開発者向けです。
