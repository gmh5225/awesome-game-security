This project is a Unity plugin that integrates the freeRASP runtime application self-protection SDK for mobile in-app protection and security monitoring. It wraps native Android (Java) and iOS (Swift/Objective-C++) bridges behind C# APIs so Unity games and apps can initialize Talsec, receive threat callbacks, and respond to runtime attacks. Key detections include root and jailbreak, hooking frameworks such as Frida, emulators or simulators, app integrity and unofficial installs, debugging, screenshots or screen recording, and related device-state risks like time or location spoofing. A sample Unity test app is included for integration and validation. The primary use case is Unity mobile developers who need freemium RASP defenses against reverse engineering, tampering, and compromised OS environments on Android and iOS.
