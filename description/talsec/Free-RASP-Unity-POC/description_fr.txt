Ce projet est un plugin Unity qui intègre le SDK freeRASP de protection applicative à l’exécution (RASP) pour la protection et la surveillance de sécurité in-app sur mobile. Il encapsule les ponts natifs Android (Java) et iOS (Swift/Objective-C++) derrière des API C# afin que les jeux et applications Unity puissent initialiser Talsec, recevoir des callbacks de menaces et réagir aux attaques à l’exécution. Les détections clés incluent le root et le jailbreak, les frameworks de hooking tels que Frida, les émulateurs ou simulateurs, l’intégrité de l’application et les installations non officielles, le débogage, les captures d’écran ou l’enregistrement d’écran, ainsi que des risques liés à l’état de l’appareil comme le spoofing d’heure ou de localisation. Une application de test Unity d’exemple est fournie pour l’intégration et la validation. Le cas d’usage principal concerne les développeurs mobiles Unity qui ont besoin de défenses RASP freemium contre le reverse engineering, la falsification et les environnements OS compromis sur Android et iOS.
