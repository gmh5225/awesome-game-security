本專案是一個 Unity 外掛，整合了 freeRASP 執行階段應用自我保護（RASP）SDK，用於行動端應用內防護與安全監控。它透過 C# API 封裝原生 Android（Java）與 iOS（Swift/Objective-C++）橋接層，使 Unity 遊戲與應用能夠初始化 Talsec、接收威脅回呼，並對執行階段攻擊作出回應。主要偵測能力包括 root 與越獄、Frida 等掛鉤框架、模擬器或模擬器環境、應用完整性與非官方安裝、除錯、螢幕截圖或螢幕錄影，以及時間或位置偽造等相關裝置狀態風險。專案包含範例 Unity 測試應用，便於整合與驗證。主要面向需要在 Android 與 iOS 上使用 freemium RASP 防護、抵禦逆向工程、竄改與受損作業系統環境的 Unity 行動開發者。
