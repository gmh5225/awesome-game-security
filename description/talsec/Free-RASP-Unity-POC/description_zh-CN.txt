本项目是一个 Unity 插件，集成了 freeRASP 运行时应用自我保护（RASP）SDK，用于移动端应用内防护与安全监控。它通过 C# API 封装原生 Android（Java）与 iOS（Swift/Objective-C++）桥接层，使 Unity 游戏和应用能够初始化 Talsec、接收威胁回调，并对运行时攻击做出响应。主要检测能力包括 root 与越狱、Frida 等挂钩框架、模拟器或模拟器环境、应用完整性与非官方安装、调试、截屏或录屏，以及时间或位置伪造等相关设备状态风险。项目包含示例 Unity 测试应用，便于集成与验证。主要面向需要在 Android 与 iOS 上使用 freemium RASP 防护、抵御逆向工程、篡改与受损操作系统环境的 Unity 移动开发者。
