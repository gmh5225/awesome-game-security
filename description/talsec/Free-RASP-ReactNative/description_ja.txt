freeRASP for React Native は、Android および iOS 上の React Native アプリ向けに Runtime Application Self-Protection（RASP）とアプリケーションシールドを提供する、モバイル向けアプリ内保護・セキュリティ監視プラグインです。root や jailbreak（Magisk、KernelSU、Dopamine など）、Frida などのフッキングフレームワーク、アプリ改ざんや信頼できないインストール、デバイスアンバインディング、マルウェアや不審なアプリ、時刻改ざん、コード難読化の欠如といった脅威を検知し、防止を支援します。React Native 側は TypeScript/JavaScript で実装され、ネイティブの Kotlin および Swift ブリッジ経由で Talsec ランタイムに接続し、useFreeRasp などのフックと任意の画面キャプチャ保護を提供します。リバースエンジニアリング、侵害された端末、不正行為および関連攻撃に対する軽量で低レイテンシなランタイムセキュリティ監視を必要とするモバイル開発者向けで、任意の週次セキュリティレポートにも対応します。
