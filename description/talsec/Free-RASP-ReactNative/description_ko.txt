freeRASP for React Native는 Android 및 iOS의 React Native 앱에 Runtime Application Self-Protection(RASP)과 애플리케이션 실딩을 제공하는 모바일 인앱 보호 및 보안 모니터링 플러그인입니다. root 및 jailbreak(Magisk, KernelSU, Dopamine 등), Frida와 같은 후킹 프레임워크, 앱 변조 또는 신뢰할 수 없는 설치, 기기 언바인딩, 멀웨어 및 의심스러운 앱, 시간 스푸핑, 코드 난독화 누락 등의 위협을 탐지하고 방지를 돕습니다. React Native 측은 TypeScript/JavaScript로 구현되며, 네이티브 Kotlin 및 Swift 브리지를 통해 Talsec 런타임에 연결되고, useFreeRasp 등의 훅과 선택적 화면 캡처 보호를 제공합니다. 리버스 엔지니어링, 손상된 기기, 사기 및 관련 공격에 대한 경량·저지연 런타임 보안 모니터링이 필요한 모바일 개발자를 대상으로 하며, 선택적 주간 보안 보고도 지원합니다.
