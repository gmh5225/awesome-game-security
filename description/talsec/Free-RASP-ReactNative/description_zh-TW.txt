freeRASP for React Native 是一款行動端應用內防護與安全監控外掛，為 Android 與 iOS 上的 React Native 應用提供執行階段應用自我保護（RASP）及應用屏蔽能力。它可偵測並協助防範 root 與越獄（包括 Magisk、KernelSU、Dopamine 等工具）、Frida 等掛鉤框架、應用竄改或不可信安裝、裝置解綁、惡意軟體與可疑應用、時間偽造以及缺少程式碼混淆等威脅。此外掛在 React Native 端以 TypeScript/JavaScript 實作，並透過原生 Kotlin 與 Swift 橋接接入 Talsec 執行階段，提供 useFreeRasp 等鉤子以及可選的螢幕擷取防護。它面向需要輕量、低延遲執行階段安全監控的行動開發者，用於對抗逆向工程、遭入侵裝置、詐欺及相關攻擊，並可選用每週安全報告。
