freeRASP for React Native is a mobile in-app protection and security monitoring plugin that provides Runtime Application Self-Protection (RASP) and application shielding for React Native apps on Android and iOS. It detects and helps prevent threats such as root and jailbreak (including Magisk, KernelSU, Dopamine, and similar tools), hooking frameworks like Frida, app tampering or untrusted installs, device unbinding, malware and suspicious apps, time spoofing, and missing code obfuscation. The plugin is implemented with TypeScript/JavaScript on the React Native side and native Kotlin and Swift bridges to the Talsec runtime, and offers hooks such as useFreeRasp plus optional screen-capture protection. It targets mobile developers who need lightweight, low-latency runtime security monitoring against reverse engineering, compromised devices, fraud, and related attacks, with optional weekly security reporting.
