freeRASP for React Native — это плагин мобильной in-app защиты и мониторинга безопасности, обеспечивающий Runtime Application Self-Protection (RASP) и application shielding для React Native-приложений на Android и iOS. Он обнаруживает и помогает предотвращать угрозы вроде root и jailbreak (включая Magisk, KernelSU, Dopamine и аналогичные инструменты), фреймворки хукинга вроде Frida, подмену приложения или недоверенные установки, отвязку устройства, вредоносное ПО и подозрительные приложения, подделку времени и отсутствие обфускации кода. Плагин реализован на TypeScript/JavaScript со стороны React Native и через нативные мосты Kotlin и Swift к runtime Talsec, а также предлагает хуки вроде useFreeRasp и опциональную защиту от захвата экрана. Он предназначен для мобильных разработчиков, которым нужен лёгкий низколатентный runtime-мониторинг безопасности против реверс-инжиниринга, скомпрометированных устройств, мошенничества и связанных атак, с опциональными еженедельными отчётами по безопасности.
