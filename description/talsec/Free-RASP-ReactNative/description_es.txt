freeRASP for React Native es un plugin de protección in-app y monitorización de seguridad móvil que ofrece Runtime Application Self-Protection (RASP) y apantallamiento de aplicaciones para apps React Native en Android e iOS. Detecta y ayuda a prevenir amenazas como root y jailbreak (incluidos Magisk, KernelSU, Dopamine y herramientas similares), frameworks de hooking como Frida, manipulación de la app o instalaciones no confiables, desvinculación del dispositivo, malware y apps sospechosas, spoofing de tiempo y falta de ofuscación de código. El plugin está implementado con TypeScript/JavaScript en el lado de React Native y puentes nativos Kotlin y Swift hacia el runtime de Talsec, y ofrece hooks como useFreeRasp más protección opcional contra captura de pantalla. Está orientado a desarrolladores móviles que necesitan monitorización de seguridad en runtime ligera y de baja latencia frente a ingeniería inversa, dispositivos comprometidos, fraude y ataques relacionados, con informes de seguridad semanales opcionales.
