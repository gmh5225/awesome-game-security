freeRASP for React Native 是一款移动端应用内防护与安全监控插件，为 Android 和 iOS 上的 React Native 应用提供运行时应用自我保护（RASP）及应用屏蔽能力。它可检测并协助防范 root 与越狱（包括 Magisk、KernelSU、Dopamine 等工具）、Frida 等挂钩框架、应用篡改或不可信安装、设备解绑、恶意软件与可疑应用、时间伪造以及缺少代码混淆等威胁。该插件在 React Native 侧以 TypeScript/JavaScript 实现，并通过原生 Kotlin 与 Swift 桥接接入 Talsec 运行时，提供 useFreeRasp 等钩子以及可选的屏幕截取防护。它面向需要轻量、低延迟运行时安全监控的移动开发者，用于对抗逆向工程、被入侵设备、欺诈及相关攻击，并可选用每周安全报告。
