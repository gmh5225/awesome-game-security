freeRASP for React Native è un plugin di protezione in-app e monitoraggio della sicurezza mobile che fornisce Runtime Application Self-Protection (RASP) e application shielding per app React Native su Android e iOS. Rileva e aiuta a prevenire minacce come root e jailbreak (inclusi Magisk, KernelSU, Dopamine e strumenti simili), framework di hooking come Frida, manomissione dell'app o installazioni non attendibili, svincolo del dispositivo, malware e app sospette, spoofing temporale e mancanza di offuscamento del codice. Il plugin è implementato con TypeScript/JavaScript lato React Native e bridge nativi Kotlin e Swift verso il runtime Talsec, e offre hook come useFreeRasp più una protezione opzionale dalla cattura dello schermo. È rivolto agli sviluppatori mobile che necessitano di un monitoraggio di sicurezza runtime leggero e a bassa latenza contro reverse engineering, dispositivi compromessi, frodi e attacchi correlati, con report di sicurezza settimanali opzionali.
