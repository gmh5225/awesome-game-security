Questo progetto è un framework di cheat per Fortnite basato su Rust, con backend di server web, database SQLite e interfaccia bot Discord. Offre un'architettura modulare con crate separate per il server del cheat, i comandi del bot e la comunicazione con il client. È utile soprattutto ai ricercatori di sicurezza nei videogiochi che studiano framework di cheat in Rust e architetture di modifica del gioco controllate via web.
