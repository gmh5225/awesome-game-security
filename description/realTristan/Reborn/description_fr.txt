Ce projet est un framework de cheat Fortnite basé sur Rust, avec un serveur web en backend, une base de données SQLite et une interface de bot Discord. Il propose une architecture modulaire avec des crates distinctes pour le serveur de cheat, les commandes du bot et la communication client. Il est surtout utile aux chercheurs en sécurité des jeux qui étudient les frameworks de cheat en Rust et les architectures de modification de jeu contrôlées via le web.
