Этот проект — фреймворк читов для Fortnite на Rust с веб-сервером в качестве бэкенда, базой данных SQLite и интерфейсом Discord-бота. Он предоставляет модульную архитектуру с отдельными crate для сервера читов, команд бота и клиентской связи. В первую очередь полезен исследователям игровой безопасности, изучающим фреймворки читов на Rust и архитектуры модификации игр с веб-управлением.
