Este proyecto es un framework de cheat de Fortnite basado en Rust, con un backend de servidor web, una base de datos SQLite y una interfaz de bot de Discord. Ofrece una arquitectura modular con crates independientes para el servidor de cheat, los comandos del bot y la comunicación con el cliente. Resulta especialmente útil para investigadores de seguridad en juegos que estudian frameworks de cheat en Rust y arquitecturas de modificación de juegos controladas por web.
