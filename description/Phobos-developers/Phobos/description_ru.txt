Этот проект — сообщественное расширение движка, добавляющее функции и исправления в Command and Conquer: Yuri's Revenge. Это крупная кодовая база на C++, построенная вокруг модифицированных компонентов YRpp и внедрения через Syringe, чтобы расширять поведение игры без замены архитектуры оригинального исполняемого файла. Репозиторий предоставляет подробную документацию и каналы релизов со стабильными, разработческими и nightly-сборками для мод-проектов. Основной сценарий использования — продвинутая разработка RTS-модов и настройка поведения на уровне движка.
