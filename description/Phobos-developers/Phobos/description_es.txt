Este proyecto es una extensión de motor comunitaria que añade funciones y correcciones a Command and Conquer: Yuri's Revenge. Es una amplia base de código en C++ construida en torno a componentes YRpp modificados y a la inyección basada en Syringe, para ampliar el comportamiento del juego sin reemplazar la arquitectura del ejecutable original. El repositorio ofrece documentación extensa y canales de publicación para builds estables, de desarrollo y nightly usadas por proyectos de mods. Su caso de uso principal es el desarrollo avanzado de mods de RTS y la personalización del comportamiento a nivel de motor.
