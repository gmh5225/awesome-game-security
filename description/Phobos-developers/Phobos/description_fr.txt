Ce projet est une extension moteur communautaire qui ajoute des fonctionnalités et des correctifs à Command and Conquer: Yuri's Revenge. Il s'agit d'une vaste base de code C++ bâtie autour de composants YRpp modifiés et d'une injection basée sur Syringe, afin d'étendre le comportement du jeu sans remplacer l'architecture de l'exécutable d'origine. Le dépôt fournit une documentation détaillée et des canaux de publication pour les builds stables, de développement et nightly utilisées par les projets de mods. Son principal cas d'usage est le développement avancé de mods RTS et la personnalisation du comportement au niveau du moteur.
