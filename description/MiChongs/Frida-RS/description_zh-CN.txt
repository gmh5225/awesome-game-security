Frida Server for KernelSU 将官方 Frida server 打包为可安装的 Android KernelSU 模块，并由一个轻量的 Rust 监督进程管理其生命周期。监督进程（frida-ksud）负责启动、停止、重启、PID 跟踪、崩溃恢复与日志记录，同时提供 Material 3 WebUI 用于配置与状态控制。构建产物包含多 ABI 包（arm64-v8a、armeabi-v7a、x86_64、x86），会拉取并校验官方 Frida 二进制，默认监听回环地址，非本地暴露需令牌。面向安全研究员、逆向工程师，以及通过 KernelSU、USB/ADB 或受控 TCP 在已 root 的 Android 设备上进行授权应用调试的场景。
