Frida Server for KernelSU упаковывает официальный Frida server в устанавливаемый модуль KernelSU для Android, а небольшой супервизор на Rust управляет его жизненным циклом. Супервизор (frida-ksud) отвечает за запуск, остановку, перезапуск, отслеживание PID, восстановление после сбоев и логирование, а WebUI на Material 3 обеспечивает настройку и контроль состояния. Сборки поставляют пакеты для нескольких ABI (arm64-v8a, armeabi-v7a, x86_64, x86), загружают и проверяют официальные бинарники Frida и по умолчанию слушают loopback, требуя токен для нелокального доступа. Проект рассчитан на исследователей безопасности, реверс-инженеров и авторизованную отладку приложений на рутованных Android-устройствах через KernelSU, USB/ADB или контролируемый TCP-доступ.
