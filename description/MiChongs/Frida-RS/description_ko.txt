Frida Server for KernelSU는 공식 Frida server를 Android용 설치형 KernelSU 모듈로 패키징하고, 작은 Rust 슈퍼바이저가 수명 주기를 관리합니다. 슈퍼바이저(frida-ksud)는 시작, 중지, 재시작, PID 추적, 충돌 복구, 로깅을 처리하며, Material 3 WebUI로 구성과 상태를 제어합니다. 빌드는 다중 ABI 패키지(arm64-v8a, armeabi-v7a, x86_64, x86)를 제공하고, 공식 Frida 바이너리를 가져와 검증하며, 기본적으로 루프백으로 수신하고 비로컬 노출에는 토큰을 요구합니다. KernelSU, USB/ADB 또는 제어된 TCP를 통해 루팅된 Android 기기에서 보안 연구원, 리버스 엔지니어, 승인된 앱 디버깅을 대상으로 합니다.
