Frida Server for KernelSU は、公式の Frida server を Android 向けのインストール可能な KernelSU モジュールとしてパッケージし、小さな Rust 製スーパーバイザでそのライフサイクルを管理します。スーパーバイザ（frida-ksud）は開始、停止、再起動、PID 追跡、クラッシュ復旧、ログ記録を担当し、Material 3 WebUI で設定と状態の制御を提供します。ビルドは複数 ABI パッケージ（arm64-v8a、armeabi-v7a、x86_64、x86）を同梱し、公式 Frida バイナリを取得・検証し、既定ではループバックで待ち受け、非ローカル公開にはトークンを要求します。KernelSU、USB/ADB、または制御された TCP 経由で、ルート化された Android 端末上のセキュリティ研究者、リバースエンジニア、および認可されたアプリデバッグを対象とします。
