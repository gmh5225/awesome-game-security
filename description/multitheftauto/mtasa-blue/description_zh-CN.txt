Multi Theft Auto: San Andreas（MTA:SA）是面向《Grand Theft Auto: San Andreas》的开源多人游戏模组，提供完整的客户端与专用服务器栈，支持自定义在线玩法。项目以 C++ 为主，并大量使用 Lua 编写资源与模组脚本，通过 Premake 在 Windows 与 Linux 上构建。客户端内嵌 Chromium（CEF）实现游戏内浏览器，挂钩 Direct3D 与 DirectInput 以控制渲染与输入，并包含反作弊模块以及钩子/DLL 分析等工具。该项目面向多人游戏开发，以及围绕 GTA San Andreas 模组、客户端完整性与反作弊技术的游戏安全研究。
