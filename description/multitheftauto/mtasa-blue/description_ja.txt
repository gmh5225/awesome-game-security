Multi Theft Auto: San Andreas（MTA:SA）は、Grand Theft Auto: San Andreas 向けのオープンソース・マルチプレイヤー改造で、カスタムオンラインプレイのための完全なクライアントおよび専用サーバースタックを提供します。主に C++ で書かれ、リソースや改造には Lua スクリプトを広く用い、Windows と Linux 向けに Premake ベースのビルドを使用します。クライアントはゲーム内ブラウザ用に Chromium（CEF）を組み込み、描画と入力制御のために Direct3D と DirectInput をフックし、アンチチートモジュールやフック／DLL 解析ツールなどのユーティリティも含みます。本プロジェクトは、マルチプレイヤーゲーム開発、および GTA San Andreas の改造、クライアント整合性、アンチチート技術をめぐるゲームセキュリティ研究を対象としています。
