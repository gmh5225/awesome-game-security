Multi Theft Auto: San Andreas（MTA:SA）是面向《Grand Theft Auto: San Andreas》的開源多人遊戲模組，提供完整的用戶端與專用伺服器堆疊，支援自訂線上玩法。專案以 C++ 為主，並大量使用 Lua 撰寫資源與模組腳本，透過 Premake 在 Windows 與 Linux 上建置。用戶端內嵌 Chromium（CEF）實作遊戲內瀏覽器，掛鉤 Direct3D 與 DirectInput 以控制渲染與輸入，並包含反作弊模組以及掛鉤/DLL 分析等工具。該專案面向多人遊戲開發，以及圍繞 GTA San Andreas 模組、用戶端完整性與反作弊技術的遊戲安全研究。
