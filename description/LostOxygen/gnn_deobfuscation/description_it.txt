Questo progetto adotta un approccio basato su Graph Neural Network (GNN) per deoffuscare espressioni Mixed Boolean-Arithmetic (MBA). Il codice Python include pipeline di training e testing con dataset provenienti da più offuscatori MBA, tra cui Loki, MBABlast e MBAObfuscator, organizzati per numero di variabili e profondità delle operazioni. È soprattutto utile a ricercatori di deoffuscazione e analisti di sicurezza che studiano approcci di machine learning per semplificare espressioni aritmetiche offuscate.
