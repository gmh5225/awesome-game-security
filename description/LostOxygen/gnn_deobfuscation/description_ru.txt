Этот проект использует подход на основе графовых нейронных сетей (GNN) для деобфускации выражений Mixed Boolean-Arithmetic (MBA). Кодовая база на Python включает конвейеры обучения и тестирования с наборами данных от нескольких MBA-обфускаторов, в том числе Loki, MBABlast и MBAObfuscator, организованными по числу переменных и глубине операций. Он в первую очередь полезен исследователям деобфускации и специалистам по безопасности, изучающим методы машинного обучения для упрощения обфусцированных арифметических выражений.
