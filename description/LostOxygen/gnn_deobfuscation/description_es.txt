Este proyecto aplica un enfoque basado en redes neuronales de grafos (GNN) para desofuscar expresiones Mixed Boolean-Arithmetic (MBA). El código en Python incluye pipelines de entrenamiento y prueba con conjuntos de datos de varios ofuscadores MBA, entre ellos Loki, MBABlast y MBAObfuscator, organizados por número de variables y profundidad de operaciones. Resulta especialmente útil para investigadores de desofuscación y analistas de seguridad que estudian enfoques de aprendizaje automático para simplificar expresiones aritméticas ofuscadas.
