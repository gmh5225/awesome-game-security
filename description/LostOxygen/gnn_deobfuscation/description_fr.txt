Ce projet propose une approche fondée sur les réseaux de neurones en graphes (GNN) pour désobfusquer les expressions Mixed Boolean-Arithmetic (MBA). La base de code Python inclut des pipelines d'entraînement et de test avec des jeux de données issus de plusieurs obfuscateurs MBA, notamment Loki, MBABlast et MBAObfuscator, organisés selon le nombre de variables et la profondeur des opérations. Il s'adresse surtout aux chercheurs en désobfuscation et aux analystes sécurité qui étudient les approches d'apprentissage automatique pour simplifier des expressions arithmétiques obfusquées.
