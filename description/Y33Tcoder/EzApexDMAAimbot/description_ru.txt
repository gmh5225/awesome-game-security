Этот проект — DMA-assisted реализация aimbot и glow-теста для Apex Legends на основе KVM-ридера памяти.
Он сочетает компоненты на C и C++, чтобы внешне читать состояние игры с Linux-хоста и управлять поведением в гостевой среде.
Набор функций включает рандомизацию отдачи, нелинейное сглаживание, рандомизацию целевой кости и командный glow-рендеринг.
Предназначен для экспериментальных исследований игрового читерства в контексте DMA-сценариев и стратегий снижения поверхности обнаружения.
