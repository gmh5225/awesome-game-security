Этот проект — плагин для IDA Pro, который расширяет и заменяет стандартное окно экспорта данных.
Он поддерживает экспорт выбранных данных в нескольких целочисленных и вещественных форматах, а также экспорт текста ассемблера.
Плагин реализован на Python и ориентирован на современные версии IDA Pro через интерфейс с горячими клавишами.
Его рабочий процесс рассчитан на ускорение повторяющихся задач извлечения при реверс-инжиниринге.
Он в первую очередь полезен аналитикам бинарного кода и исследователям безопасности, которым нужен структурированный экспорт из представлений дизассемблирования.
