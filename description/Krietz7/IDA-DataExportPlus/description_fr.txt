Ce projet est un plugin IDA Pro qui étend et remplace la fenêtre d'export de données par défaut.
Il permet d'exporter les données sélectionnées dans plusieurs formats entiers et flottants, ainsi que du texte assembleur.
Le plugin est écrit en Python et cible les versions modernes d'IDA Pro via une interface pilotée par raccourcis clavier.
Son flux de travail vise à accélérer les tâches d'extraction répétitives lors de la rétro-ingénierie.
Il est surtout utile aux analystes binaires et aux chercheurs en sécurité qui ont besoin d'un export structuré depuis les vues de désassemblage.
