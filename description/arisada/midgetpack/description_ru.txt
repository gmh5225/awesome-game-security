Этот проект — упаковщик ELF-бинарников, предназначенный для защиты исполняемых файлов на недоверенных системах. Он предлагает режим с паролем и режим «запрос–ответ» на основе обмена ключами Curve25519 с AES-128 и HMAC-SHA256. Реализация поддерживает кросс-архитектурную упаковку для целевых платформ Linux и FreeBSD, включая x86, x86-64 и ARM. Проект ориентирован на специалистов по безопасности, которым нужно усиливать защиту чувствительных инструментов при контролируемых оценках и развёртываниях.
