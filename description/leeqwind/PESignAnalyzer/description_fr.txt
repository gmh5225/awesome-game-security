Ce projet est un outil simple d'extraction d'informations de signature de fichiers PE.
Ce programme sert à obtenir les informations de signature des fichiers PE signés par un ou plusieurs certificats de signature de code intégrés sous Windows.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des outils de cheat / reverse engineering.
