Dieses Projekt ist ein einfaches Tool zum Extrahieren von Signaturinformationen aus PE-Dateien.
Das Programm dient dazu, Signaturinformationen aus PE-Dateien zu lesen, die unter Windows mit einem oder mehreren eingebetteten Code-Signing-Zertifikaten signiert wurden.
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich von Cheat-/RE-Tools untersuchen.
