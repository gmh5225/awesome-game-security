Questo progetto è un semplice strumento per estrarre informazioni sulle firme dei file PE.
Il programma serve a ottenere le informazioni di firma dai file PE firmati con uno o più certificati di code signing incorporati su Windows.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di tool per cheat / reverse engineering.
