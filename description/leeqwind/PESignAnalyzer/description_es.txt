Este proyecto es una herramienta sencilla para extraer información de firmas de archivos PE.
Este programa se usa para obtener información de firma de archivos PE firmados con uno o varios certificados de firma de código incrustados en Windows.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de herramientas de cheat / ingeniería inversa.
