Questo progetto è un framework C++ per l'offuscamento di costanti, variabili e stringhe in fase di compilazione e in stile runtime. Si concentra su trasformazioni dei dati basate su XOR che rendono più difficile l'ispezione statica dei valori incorporati nei binari compilati. Il repository include output di esempio e strutture di supporto per integrare pattern di offuscamento nel codice applicativo normale. Il caso d'uso principale sono esperimenti di hardening del software e tecniche anti-analisi in codice sensibile alla sicurezza.
