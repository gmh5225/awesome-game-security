Este proyecto es un framework en C++ para ofuscar constantes, variables y cadenas en tiempo de compilación y con un estilo en tiempo de ejecución. Se centra en transformaciones de datos basadas en XOR que dificultan la inspección estática de valores embebidos en binarios compilados. El repositorio incluye salidas de ejemplo y estructuras auxiliares para integrar patrones de ofuscación en código de aplicación habitual. Su caso de uso principal son experimentos de endurecimiento de software y técnicas anti-análisis en código sensible a la seguridad.
