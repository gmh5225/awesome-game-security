Ce projet est un framework C++ pour l'obfuscation de constantes, de variables et de chaînes à la compilation et dans un style runtime. Il s'appuie sur des transformations de données basées sur XOR afin de rendre plus difficile l'inspection statique des valeurs intégrées dans les binaires compilés. Le dépôt fournit des sorties d'exemple et des structures d'aide pour intégrer ces motifs d'obfuscation dans du code applicatif classique. Son usage principal concerne les expériences de durcissement logiciel et les techniques anti-analyse dans du code sensible à la sécurité.
