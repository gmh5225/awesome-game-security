Этот проект — C++-фреймворк для обфускации констант, переменных и строк на этапе компиляции и в runtime-стиле. Он опирается на XOR-преобразования данных, усложняющие статическую инспекцию встроенных значений в скомпилированных бинарниках. Репозиторий содержит примеры вывода и вспомогательные структуры для внедрения шаблонов обфускации в обычный прикладной код. Основной сценарий использования — эксперименты по упрочнению ПО и антианализные техники в коде, чувствительном к безопасности.
