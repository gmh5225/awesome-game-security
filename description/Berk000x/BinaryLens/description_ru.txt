BinaryLens — это плагин для IDA Pro, использующий большие языковые модели для ускорения рабочих процессов реверс-инжиниринга. Плагин может массово переименовывать функции, объяснять логику бинарного кода и переименовывать локальные переменные непосредственно из контекста декомпилятора. Он реализован на C++ с интеграцией IDA SDK и OpenSSL и поддерживает несколько бэкендов моделей для задач анализа. Инструмент предназначен для аналитиков, работающих с крупными бинарными файлами, включая реверс-инжиниринг игровых клиентов и античитов.
