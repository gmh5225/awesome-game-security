Questo progetto è APKiD, un identificatore di applicazioni Android che rileva compilatori, packer, offuscatori e tecniche anti-analisi usate nei file APK e DEX. Utilizza regole YARA per creare fingerprint di strumenti di protezione noti, tra cui ProGuard, DexGuard, Bangcle, Ijiami e decine di altri protettori commerciali e personalizzati. Lo strumento Python restituisce le protezioni rilevate e le relative versioni. È rivolto ad analisti di sicurezza mobile, ricercatori di malware e reverse engineer che eseguono un triage iniziale sui campioni Android per identificare le protezioni applicate prima di un'analisi più approfondita.
