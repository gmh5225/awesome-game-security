本プロジェクトは APKiD であり、APK および DEX ファイルで使用されているコンパイラ、パッカ、難読化ツール、およびアンチ解析技術を検出する Android アプリケーション識別ツールです。YARA ルールを用いて、ProGuard、DexGuard、Bangcle、Ijiami をはじめ、数十の商用・独自プロテクターを含む既知の保護ツールをフィンガープリントします。この Python ツールは、検出された保護とそのバージョンを出力します。モバイルセキュリティアナリスト、マルウェア研究者、リバースエンジニアが、Android サンプルの詳細解析の前に初期トリアージを行い、適用されている保護を特定することを目的としています。
