Este proyecto es APKiD, un identificador de aplicaciones Android que detecta compiladores, packers, ofuscadores y técnicas anti-análisis usadas en archivos APK y DEX. Emplea reglas YARA para crear huellas de herramientas de protección conocidas, como ProGuard, DexGuard, Bangcle, Ijiami y docenas de otros protectores comerciales y personalizados. La herramienta en Python muestra las protecciones detectadas y sus versiones. Está orientada a analistas de seguridad móvil, investigadores de malware e ingenieros de ingeniería inversa que realizan un triaje inicial de muestras Android para identificar las protecciones aplicadas antes de un análisis más profundo.
