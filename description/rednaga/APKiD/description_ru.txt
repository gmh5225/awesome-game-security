Этот проект — APKiD, идентификатор Android-приложений, который обнаруживает компиляторы, пакеры, обфускаторы и антианалитические техники в файлах APK и DEX. Он использует правила YARA для создания отпечатков известных средств защиты, включая ProGuard, DexGuard, Bangcle, Ijiami и десятки других коммерческих и пользовательских протекторов. Инструмент на Python выводит обнаруженные защиты и их версии. Он предназначен для аналитиков мобильной безопасности, исследователей вредоносного ПО и специалистов по реверс-инжинирингу, выполняющих первичную triage-оценку Android-образцов, чтобы определить применённые защиты перед более глубоким анализом.
