AssetRipper — инструмент на базе .NET для извлечения ассетов из сериализованных файлов Unity и asset bundle с последующим преобразованием в вывод в нативном стиле Unity. Проект в основном написан на C# и содержит обширные парсеры, конвертеры и инструментальные модули для многих форматов ресурсов Unity. Он поддерживает широкий диапазон версий Unity и ориентирован на практические рабочие процессы восстановления и реконструкции ассетов. Инструмент широко используется для реверс-инжиниринга Unity, моддинга и анализа игрового контента.
