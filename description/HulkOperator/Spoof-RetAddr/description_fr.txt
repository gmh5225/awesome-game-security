Ce projet est une implémentation proof-of-concept pour le spoofing des adresses de retour WinAPI sur les systèmes x64.
Il combine C et l'assembleur NASM pour démontrer des techniques de manipulation de pile dans un exemple compact et réutilisable.
Le code illustre les concepts d'obfuscation de la pile d'appels fréquemment abordés dans les contextes red-team et anti-détection.
Il est destiné à un usage éducatif en reverse engineering et en recherche sur les conventions d'appel de bas niveau.
