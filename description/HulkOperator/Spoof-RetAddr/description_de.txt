Dieses Projekt ist eine Proof-of-Concept-Implementierung zum Spoofen von WinAPI-Rücksprungadressen auf x64-Systemen.
Es kombiniert C und NASM-Assembly, um Stack-Manipulationstechniken in einem kompakten, wiederverwendbaren Beispiel zu demonstrieren.
Der Code veranschaulicht Call-Stack-Obfuskationskonzepte, die häufig in Red-Team- und Anti-Detection-Kontexten diskutiert werden.
Er ist für den Bildungsgebrauch in der Reverse Engineering und der Forschung zu Low-Level-Aufrufkonventionen gedacht.
