GhidraBoy è un’estensione sperimentale che aggiunge a Ghidra il supporto per il reverse engineering di Nintendo Game Boy. Usa codice Kotlin e Java insieme a definizioni personalizzate del linguaggio Sleigh per la CPU Sharp SM83. L’estensione include il caricamento ROM per cartucce banked e unbanked, varianti di boot ROM, blocchi di memory map e simboli dei registri hardware. Il caso d’uso principale è l’analisi binaria di giochi retrò e la ricerca sul firmware a basso livello.
