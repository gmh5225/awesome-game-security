GhidraBoy — экспериментальное расширение, добавляющее в Ghidra поддержку реверс-инжиниринга Nintendo Game Boy. Оно сочетает код на Kotlin и Java с пользовательскими определениями языка Sleigh для процессора Sharp SM83. Расширение включает загрузку ROM для картриджей с банками и без, варианты boot ROM, блоки карты памяти и символы аппаратных регистров. Основной сценарий использования — анализ бинарников ретро-игр и исследование низкоуровневой прошивки.
