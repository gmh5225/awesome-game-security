GhidraBoy es una extensión experimental que añade soporte de ingeniería inversa de Nintendo Game Boy a Ghidra. Combina código Kotlin y Java con definiciones de lenguaje Sleigh personalizadas para la CPU Sharp SM83. La extensión incluye carga de ROM para cartuchos con y sin bancos, variantes de boot ROM, bloques de mapa de memoria y símbolos de registros de hardware. Su caso de uso principal es el análisis binario de juegos retro y la investigación de firmware de bajo nivel.
