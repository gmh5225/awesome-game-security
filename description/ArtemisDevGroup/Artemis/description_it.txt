Artemis è un framework C++ per la modifica di giochi mirato a Rainbow Six: Siege Shadow Legacy. Include un'infrastruttura di injection e hooking con componenti come ImGui e MinHook, oltre a manager per eventi, tasti di scelta rapida, disegno e interazioni di gioco. L'architettura è progettata per supportare estensioni create dall'utente tramite un approccio modulare. Il progetto è destinato principalmente allo sviluppo di cheat e alla sperimentazione di reverse engineering.
