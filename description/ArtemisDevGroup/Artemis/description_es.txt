Artemis es un framework C++ de modificación de juegos orientado a Rainbow Six: Siege Shadow Legacy. Incluye infraestructura de inyección y hooking con componentes como ImGui y MinHook, además de gestores para eventos, teclas, dibujo e interacciones con el juego. La arquitectura está diseñada para admitir extensiones creadas por el usuario mediante un enfoque modular. Este proyecto está orientado principalmente al desarrollo de cheats y a la experimentación en ingeniería inversa.
