이 프로젝트는 포괄적인 오픈소스 x86_64 가상화 플랫폼인 Oracle VirtualBox입니다. CPU 및 메모리 가상화, 장치 에뮬레이션, Guest Additions, 네트워킹, 스토리지, 원격 데스크톱 서비스를 갖춘 완전한 Virtual Machine Monitor를 구현하며 Windows, Linux, macOS, Solaris 호스트를 지원합니다. 모듈형 아키텍처는 kBuild를 사용하고 여러 CPU 아키텍처를 지원하며 COM API와 IOCTL 기반 드라이버 인터페이스를 제공합니다. 하이퍼바이저 내부 구조, 하드웨어 가상화, VM 기반 보안 연구 환경을 연구하는 가상화 연구원과 보안 분석가에게 주로 유용합니다.
