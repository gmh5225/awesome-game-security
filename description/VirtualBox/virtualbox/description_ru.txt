Этот проект — Oracle VirtualBox, полноценная открытая платформа виртуализации x86_64. Она реализует полноценный Virtual Machine Monitor с виртуализацией CPU и памяти, эмуляцией устройств, guest additions, сетью, хранилищем и службами удалённого рабочего стола на хостах Windows, Linux, macOS и Solaris. Модульная архитектура использует kBuild, поддерживает несколько архитектур CPU и предоставляет COM API, а также интерфейсы драйверов на основе IOCTL. В основном полезна исследователям виртуализации и аналитикам безопасности, изучающим внутреннее устройство гипервизоров, аппаратную виртуализацию и среды исследований безопасности на базе ВМ.
