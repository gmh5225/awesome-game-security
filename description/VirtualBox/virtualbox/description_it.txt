Questo progetto è Oracle VirtualBox, una piattaforma completa di virtualizzazione x86_64 open source. Implementa un Virtual Machine Monitor completo con virtualizzazione di CPU e memoria, emulazione dei dispositivi, guest additions, networking, storage e servizi di desktop remoto su host Windows, Linux, macOS e Solaris. L’architettura modulare usa kBuild, supporta più architetture CPU e fornisce API COM e interfacce driver basate su IOCTL. È soprattutto utile a ricercatori di virtualizzazione e analisti di sicurezza che studiano gli interni degli hypervisor, la virtualizzazione hardware e ambienti di ricerca sulla sicurezza basati su VM.
