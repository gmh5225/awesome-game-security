本プロジェクトは Oracle VirtualBox であり、包括的なオープンソースの x86_64 仮想化プラットフォームです。CPU およびメモリ仮想化、デバイスエミュレーション、Guest Additions、ネットワーク、ストレージ、リモートデスクトップサービスを備えた完全な Virtual Machine Monitor を実装し、Windows、Linux、macOS、Solaris ホストに対応します。モジュール構成は kBuild を用い、複数の CPU アーキテクチャをサポートし、COM API および IOCTL ベースのドライバインタフェースを提供します。主に、ハイパーバイザ内部、ハードウェア仮想化、VM ベースのセキュリティ研究環境を学ぶ仮想化研究者やセキュリティ分析者に有用です。
