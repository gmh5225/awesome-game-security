Este proyecto es Oracle VirtualBox, una plataforma completa de virtualización x86_64 de código abierto. Implementa un Virtual Machine Monitor completo con virtualización de CPU y memoria, emulación de dispositivos, guest additions, red, almacenamiento y servicios de escritorio remoto en hosts Windows, Linux, macOS y Solaris. La arquitectura modular usa kBuild, admite múltiples arquitecturas de CPU y proporciona APIs COM e interfaces de controladores basadas en IOCTL. Resulta especialmente útil para investigadores de virtualización y analistas de seguridad que estudian los internos de hipervisores, la virtualización por hardware y entornos de investigación de seguridad basados en VM.
