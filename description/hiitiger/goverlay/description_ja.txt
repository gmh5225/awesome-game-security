GOverlay は、デスクトップ GUI アプリがゲーム内にインタラクティブなウィンドウを描画できるようにする、オープンソースの DirectX フックおよびインゲームオーバーレイフレームワークです。DirectX 9 から 12 までのタイトルにネイティブオーバーレイ DLL を注入し、Electron、Qt、CEF、WPF などのフレームワークからのオフスクリーンコンテンツを合成します。スタックは C++ ネイティブアドオンとインジェクタに TypeScript/Electron のデモクライアントを組み合わせ、ネイティブビルドには CMake、ホストとゲーム間の通信には IPC と共有メモリを用います。主な機能は、z-order とフォーカス付きのマルチウィンドウ管理、ゲーム内でのドラッグとリサイズ、アルファ対応のマウス処理を備えたカスタム形状ウィンドウ、ホットキーを含む設定可能な入力インターセプトです。ライブの DirectX ゲーム上に Web スタイルの UI を描画する必要があるゲームコンパニオン、オーバーレイ、録画や統計ツール向けです。
