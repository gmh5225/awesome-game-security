GOverlay è un framework open source di hook DirectX e overlay in-game che consente alle app GUI desktop di renderizzare finestre interattive all'interno dei giochi. Inietta DLL di overlay native nei titoli DirectX da 9 a 12 e composita contenuti offscreen da framework come Electron, Qt, CEF e WPF. Lo stack combina addon e injector nativi in C++ con un client demo TypeScript/Electron, usando CMake per le build native e IPC più memoria condivisa per la comunicazione host-gioco. Le capacità principali includono gestione multi-finestra con z-order e focus, trascinamento e ridimensionamento in-game, finestre a forma personalizzata con gestione del mouse consapevole dell'alpha e intercettazione input configurabile (incluse le hotkey). È pensato per companion di gioco, overlay e strumenti di registrazione o statistiche che necessitano di una UI in stile web disegnata sopra un gioco DirectX in esecuzione.
