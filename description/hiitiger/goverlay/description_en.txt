GOverlay is an open-source DirectX hook and in-game overlay framework that lets desktop GUI apps render interactive windows inside games. It injects native overlay DLLs into DirectX 9 through 12 titles and composites offscreen content from frameworks such as Electron, Qt, CEF, and WPF. The stack mixes C++ native addons and injectors with a TypeScript/Electron demo client, using CMake for native builds and IPC plus shared memory for host-to-game communication. Key capabilities include multi-window management with z-order and focus, in-game drag and resize, custom-shaped windows with alpha-aware mouse handling, and configurable input interception (including hotkeys). It is aimed at game companions, overlays, and recording or stats tools that need a web-style UI drawn over a live DirectX game.
