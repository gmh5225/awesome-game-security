Interfaccia grafica Windows Forms intorno a signtool.exe di Microsoft, con supporto per tre modalità di firma: archivio certificati Windows (impronta digitale), file PFX e Azure Trusted Signing (tramite Azure.CodeSigning SDK). Include monitoraggio dei certificati, gestione dei server di timestamp, convalida della firma, firma batch di più file ed esportazione di script PowerShell per l'integrazione CI/CD.
