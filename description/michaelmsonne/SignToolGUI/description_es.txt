Interfaz gráfica Windows Forms sobre signtool.exe de Microsoft que admite tres modos de firma: almacén de certificados de Windows (huella digital), archivo PFX y Azure Trusted Signing (mediante el SDK Azure.CodeSigning). Incluye supervisión de certificados, gestión de servidores de marca de tiempo, validación de firmas, firma por lotes de varios archivos y exportación de scripts de PowerShell para integración CI/CD.
