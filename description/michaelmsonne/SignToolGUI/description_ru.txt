Графическая оболочка Windows Forms для Microsoft signtool.exe с поддержкой трёх режимов подписи: хранилище сертификатов Windows (отпечаток), файл PFX и Azure Trusted Signing (через SDK Azure.CodeSigning). Включает мониторинг сертификатов, управление серверами меток времени, проверку подписи, пакетную подпись нескольких файлов и экспорт скриптов PowerShell для интеграции с CI/CD.
