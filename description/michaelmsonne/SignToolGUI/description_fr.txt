Interface graphique Windows Forms autour de signtool.exe de Microsoft, prenant en charge trois modes de signature : magasin de certificats Windows (empreinte), fichier PFX et Azure Trusted Signing (via le SDK Azure.CodeSigning). Inclut la surveillance des certificats, la gestion des serveurs d'horodatage, la validation de signature, la signature par lots de plusieurs fichiers et l'export de scripts PowerShell pour l'intégration CI/CD.
