Этот проект посвящён VirtualBox с бэкендом KVM.
Аппаратное графическое ускорение пока не интегрировано в GUI и требует ручной настройки.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / QEMU / KVM / PVE / VBox.
