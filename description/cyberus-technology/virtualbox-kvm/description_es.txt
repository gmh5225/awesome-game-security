Este proyecto se centra en VirtualBox con backend KVM.
La aceleración gráfica por hardware aún no está integrada en la GUI y requiere una configuración manual.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito cheat / QEMU / KVM / PVE / VBox.
