Ce projet se concentre sur VirtualBox avec un backend KVM.
L'accélération graphique matérielle n'est pas encore intégrée à l'interface graphique et nécessite une configuration manuelle.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine cheat / QEMU / KVM / PVE / VBox.
