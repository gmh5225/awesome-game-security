LibreHardwareMonitor es un proyecto de telemetría de hardware de código abierto que ofrece tanto un monitor de escritorio para Windows como una biblioteca .NET reutilizable.
Está escrito principalmente en C# y lee temperaturas, velocidades de ventilador, voltajes, cargas y datos de reloj de CPU, GPU, dispositivos de almacenamiento y otros componentes.
El proyecto incluye código de interfaz WinForms y APIs de biblioteca que pueden integrarse en otras aplicaciones.
Su caso de uso principal es el diagnóstico del sistema y la recopilación de sensores, y también suele ser útil en la monitorización del rendimiento de juegos o en herramientas de seguridad conscientes del entorno.
