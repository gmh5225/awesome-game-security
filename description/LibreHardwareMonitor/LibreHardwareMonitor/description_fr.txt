LibreHardwareMonitor est un projet open source de télémétrie matérielle qui fournit à la fois un moniteur de bureau Windows et une bibliothèque .NET réutilisable.
Il est principalement écrit en C# et lit les températures, vitesses de ventilateur, tensions, charges et fréquences d’horloge des CPU, GPU, périphériques de stockage et autres composants.
Le projet livre du code d’interface WinForms et des API de bibliothèque pouvant être intégrées dans d’autres applications.
Son cas d’usage principal est le diagnostic système et la collecte de capteurs ; il est aussi souvent utile pour le suivi des performances de jeu ou des outils de sécurité sensibles à l’environnement.
