LibreHardwareMonitor — это проект с открытым исходным кодом для аппаратной телеметрии, предоставляющий как настольный монитор для Windows, так и переиспользуемую библиотеку .NET.
Он в основном написан на C# и считывает температуры, скорости вентиляторов, напряжения, нагрузки и данные о частотах с CPU, GPU, накопителей и других компонентов.
В проекте есть код интерфейса WinForms и API библиотеки, которые можно интегрировать в другие приложения.
Основное применение — диагностика системы и сбор данных с датчиков; также часто полезен для мониторинга производительности игр или средств безопасности, учитывающих окружение.
