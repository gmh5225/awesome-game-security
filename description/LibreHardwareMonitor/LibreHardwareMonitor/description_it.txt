LibreHardwareMonitor è un progetto open source di telemetria hardware che fornisce sia un monitor desktop per Windows sia una libreria .NET riutilizzabile.
È scritto principalmente in C# e legge temperature, velocità delle ventole, tensioni, carichi e dati di clock da CPU, GPU, dispositivi di archiviazione e altri componenti.
Il progetto include codice UI WinForms e API della libreria che possono essere integrate in altre applicazioni.
Il caso d’uso principale è la diagnostica di sistema e la raccolta dei sensori; è spesso utile anche nel monitoraggio delle prestazioni di gioco o in strumenti di sicurezza sensibili all’ambiente.
