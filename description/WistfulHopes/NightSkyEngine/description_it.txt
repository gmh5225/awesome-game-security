Questo progetto è un framework open source gratuito per fighting game basato su Unreal Engine 5, pensato per titoli 2D e 2.5D.
L’implementazione è principalmente in C++ e include moduli del motore, plugin e sistemi lato gioco pensati per fighter competitivi.
Integra il networking rollback tramite un plugin basato su GGPO e offre strumenti come un editor delle collisioni per supportare l’iterazione del gameplay.
Il framework si rivolge a sviluppatori e technical designer che vogliono prototipare o pubblicare fighting game moderni su PC di fascia alta e console.
