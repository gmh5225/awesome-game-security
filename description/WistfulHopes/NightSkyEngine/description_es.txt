Este proyecto es un framework de juegos de lucha gratuito y de código abierto construido con Unreal Engine 5 para títulos en 2D y 2.5D.
Su implementación es principalmente en C++ e incluye módulos del motor, plugins y sistemas del lado del juego pensados para fighters competitivos.
Integra networking con rollback mediante un plugin basado en GGPO y ofrece herramientas como un editor de colisiones para apoyar la iteración del gameplay.
El framework está dirigido a desarrolladores y diseñadores técnicos que quieren prototipar o publicar juegos de lucha modernos en PC de gama alta y consolas.
