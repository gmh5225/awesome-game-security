Dieses Projekt ist ein kostenloses Open-Source-Framework für Fighting Games auf Basis von Unreal Engine 5 für 2D- und 2.5D-Titel.
Die Umsetzung erfolgt vorwiegend in C++ und umfasst Engine-Module, Plugins sowie spielseitige Systeme, die auf kompetitive Fighter zugeschnitten sind.
Es integriert Rollback-Networking über ein GGPO-basiertes Plugin und bietet Werkzeuge wie einen Collision Editor zur Unterstützung der Gameplay-Iteration.
Das Framework richtet sich an Entwickler und Technical Designer, die moderne Fighting Games auf High-End-PCs und Konsolen prototypisieren oder veröffentlichen möchten.
