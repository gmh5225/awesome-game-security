Этот проект представляет собой дерево исходного кода ядра Linux, связанное с исследованиями и сборками Windows Subsystem for Linux (WSL).
Архив содержит полную структуру ядра (arch, drivers, fs и связанные подсистемы), а не небольшой userspace-обёртку WSL.
Он в основном полезен исследователям подсистем Windows и ядра, изучающим сторону ядра Linux в WSL.
