Este proyecto es un árbol de código fuente del kernel de Linux asociado a la investigación y las compilaciones de Windows Subsystem for Linux (WSL).
El archivo contiene el diseño completo del kernel (arch, drivers, fs y subsistemas relacionados), en lugar de un pequeño wrapper de espacio de usuario de WSL.
Resulta especialmente útil para investigadores de subsistemas de Windows y del kernel que estudian el lado del kernel Linux de WSL.
