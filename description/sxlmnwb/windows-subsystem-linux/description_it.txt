Questo progetto è un albero dei sorgenti del kernel Linux associato alla ricerca e alle build di Windows Subsystem for Linux (WSL).
L'archivio contiene l'intera struttura del kernel (arch, drivers, fs e sottosistemi correlati), piuttosto che un piccolo wrapper in userspace di WSL.
È principalmente utile per i ricercatori di sottosistemi Windows e del kernel che studiano il lato kernel Linux di WSL.
