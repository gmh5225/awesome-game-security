Ce projet est une arborescence source du noyau Linux associée à la recherche et aux builds de Windows Subsystem for Linux (WSL).
L'archive contient la disposition complète du noyau (arch, drivers, fs et sous-systèmes associés) plutôt qu'un petit wrapper en espace utilisateur WSL.
Il est principalement utile aux chercheurs sur les sous-systèmes Windows et le noyau qui étudient le côté noyau Linux de WSL.
