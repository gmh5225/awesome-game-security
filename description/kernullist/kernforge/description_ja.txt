Kernforge は、Windows セキュリティ、アンチチート、ドライバ、テレメトリ、Unreal Engine の整合性ワークフローに特化した AI 支援コーディング CLI およびエンジニアリングエージェントです。主に Go で書かれ、C++ ドライバテンプレート、PowerShell ビルドツール、VS Code 拡張機能を備え、アーキテクチャ分析、構造インデックス、セキュリティオーバーレイを通じて再利用可能なプロジェクト知識パックを構築します。調査とシミュレーションのループ、根本原因分析、ソースレベルでのファズ推論、検証計画、証跡とセッションメモリ、MCP skills、ならびに WDM、minifilter、レジストリフィルタ、WFP callout 形式向けの自動ドライバ POC スキャフォールディングをサポートします。主な用途は汎用コーディングアシスタントではなく、セキュリティおよびアンチチートエンジニアがカーネルとゲームセキュリティのコードベースを分析・強化・検証する支援です。
