Kernforge 是一款 AI 輔助的編碼 CLI 與工程智慧代理，專注於 Windows 安全性、反作弊、驅動程式、遙測以及 Unreal Engine 完整性相關工作流程。主體以 Go 撰寫，並配備 C++ 驅動程式範本、PowerShell 建置工具以及 VS Code 擴充功能；它透過架構分析、結構索引與安全性疊加層，建立可重複使用的專案知識包。支援調查與模擬迴圈、根因分析、原始碼層級模糊推理、驗證計畫、證據與工作階段記憶、MCP skills，以及面向 WDM、minifilter、登錄檔篩選器與 WFP callout 等風格的自動化驅動程式 POC 鷹架。主要用途是協助安全性與反作弊工程師分析、強化並驗證核心與遊戲安全程式碼庫，而非作為通用程式設計助理。
