Kernforge is an AI-assisted coding CLI and engineering agent focused on Windows security, anti-cheat, driver, telemetry, and Unreal Engine integrity workflows. Written primarily in Go with supporting C++ driver templates, PowerShell build tooling, and a VS Code extension, it builds reusable project knowledge packs through architecture analysis, structural indexing, and security overlays. It supports investigate and simulate loops, root-cause analysis, source-level fuzz reasoning, verification plans, evidence and session memory, MCP skills, and automated driver POC scaffolding for WDM, minifilter, registry filter, and WFP callout styles. The primary use case is helping security and anti-cheat engineers analyze, harden, and validate kernel and game-security codebases rather than treating the tool as a generic coding assistant.
