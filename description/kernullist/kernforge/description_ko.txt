Kernforge는 Windows 보안, 안티치트, 드라이버, 텔레메트리, Unreal Engine 무결성 워크플로에 초점을 둔 AI 지원 코딩 CLI 및 엔지니어링 에이전트입니다. 주로 Go로 작성되며 C++ 드라이버 템플릿, PowerShell 빌드 도구, VS Code 확장을 포함하며, 아키텍처 분석, 구조적 인덱싱, 보안 오버레이를 통해 재사용 가능한 프로젝트 지식 팩을 구축합니다. 조사 및 시뮬레이션 루프, 근본 원인 분석, 소스 수준 퍼즈 추론, 검증 계획, 증거 및 세션 메모리, MCP skills, 그리고 WDM, minifilter, 레지스트리 필터, WFP callout 스타일을 위한 자동 드라이버 POC 스캐폴딩을 지원합니다. 주된 사용 사례는 범용 코딩 어시스턴트가 아니라, 보안 및 안티치트 엔지니어가 커널 및 게임 보안 코드베이스를 분석·강화·검증하도록 돕는 것입니다.
