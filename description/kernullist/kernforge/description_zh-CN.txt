Kernforge 是一款 AI 辅助的编码 CLI 与工程智能体，专注于 Windows 安全、反作弊、驱动、遥测以及 Unreal Engine 完整性相关工作流。主体以 Go 编写，并配有 C++ 驱动模板、PowerShell 构建工具以及 VS Code 扩展；它通过架构分析、结构索引与安全叠加层，构建可复用的项目知识包。支持调查与模拟循环、根因分析、源码级模糊推理、验证计划、证据与会话记忆、MCP skills，以及面向 WDM、minifilter、注册表过滤器和 WFP callout 等风格的自动化驱动 POC 脚手架。主要面向安全与反作弊工程师，用于分析、加固并验证内核与游戏安全代码库，而非作为通用编程助手使用。
