Este proyecto es una biblioteca de captura e inyección de paquetes para Windows del Nmap Project.
Es una actualización completa del proyecto WinPcap, que ya no se mantiene, con mayor velocidad, fiabilidad y seguridad.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / captura y análisis de paquetes.
