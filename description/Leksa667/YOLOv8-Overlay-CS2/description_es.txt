Este proyecto es un overlay de CS2 en tiempo real basado en Python que detecta jugadores enemigos con un modelo YOLOv8 ONNX.
Usa ONNX Runtime para la inferencia, mss para la captura de pantalla, y Pygame junto con las API de Win32 para renderizar una ventana de overlay transparente siempre encima.
La implementación incluye aceleración CUDA opcional, filtrado por confianza, teclas rápidas y una rutina suave de aim-assist.
Su caso de uso principal es el prototipado de cheats basados en visión por computadora y la investigación de anti-cheat relacionada con el comportamiento de detección asistida por IA.
