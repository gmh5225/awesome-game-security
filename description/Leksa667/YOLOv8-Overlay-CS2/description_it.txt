Questo progetto è un overlay CS2 in tempo reale basato su Python che rileva i giocatori nemici con un modello YOLOv8 ONNX.
Usa ONNX Runtime per l’inferenza, mss per la cattura dello schermo e Pygame insieme alle API Win32 per disegnare una finestra overlay trasparente sempre in primo piano.
L’implementazione include accelerazione CUDA opzionale, filtro per confidenza, hotkey e una routine di aim-assist fluida.
Il caso d’uso principale è il prototipaggio di cheat basati sulla computer vision e la ricerca anti-cheat sul comportamento di rilevamento assistito dall’IA.
