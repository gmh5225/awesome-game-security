Ce projet est une overlay CS2 temps réel en Python qui détecte les joueurs ennemis avec un modèle YOLOv8 ONNX.
Il utilise ONNX Runtime pour l’inférence, mss pour la capture d’écran, et Pygame ainsi que les API Win32 pour afficher une fenêtre d’overlay transparente au premier plan.
L’implémentation inclut une accélération CUDA optionnelle, un filtrage par confiance, des raccourcis clavier et une routine d’aim-assist fluide.
Son cas d’usage principal est le prototypage de cheats basés sur la vision par ordinateur et la recherche anti-cheat liée aux comportements de détection assistée par IA.
