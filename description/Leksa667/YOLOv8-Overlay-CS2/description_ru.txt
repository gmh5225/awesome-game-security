Этот проект — написанный на Python оверлей CS2 в реальном времени, который обнаруживает вражеских игроков с помощью модели YOLOv8 ONNX.
Для инференса используется ONNX Runtime, для захвата экрана — mss, а для отрисовки прозрачного окна оверлея поверх всех окон — Pygame и Win32 API.
Реализация включает опциональное ускорение CUDA, фильтрацию по уверенности, горячие клавиши и плавную процедуру aim-assist.
Основное применение — прототипирование читов на базе компьютерного зрения и связанные исследования античита, посвящённые поведению ИИ-ассистированного обнаружения.
