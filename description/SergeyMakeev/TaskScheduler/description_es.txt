Este proyecto es un planificador de tareas multiplataforma basado en fibers, diseñado para cargas de trabajo de juegos de alto rendimiento. Es una biblioteca C++ que proporciona ejecución de jobs multihilo, agrupación de tareas, comportamiento de planificación tipo work-stealing y abstracciones de plataforma para objetivos Windows y POSIX. El repositorio incluye pruebas extensas, ejemplos y componentes de terceros de bajo nivel para el cambio de contexto necesarios en sistemas de fibers. Está dirigido principalmente a programadores de motores que necesitan un sistema de jobs escalable para lógica de juego y pipelines de renderizado en paralelo.
