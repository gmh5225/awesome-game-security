Questo progetto è uno scheduler di task multipiattaforma basato su fiber, progettato per carichi di lavoro di gioco ad alte prestazioni. È una libreria C++ che offre esecuzione di job multithread, raggruppamento di task, comportamento di scheduling in stile work-stealing e astrazioni di piattaforma per target Windows e POSIX. Il repository include test estesi, esempi e componenti di terze parti a basso livello per il context switching necessari ai sistemi a fiber. È rivolto principalmente ai programmatori di engine che necessitano di un job system scalabile per logica di gioco e pipeline di rendering parallele.
