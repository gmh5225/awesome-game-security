Этот проект — кроссплатформенный планировщик задач на основе fibers, рассчитанный на высокопроизводительные игровые нагрузки. Это библиотека на C++, которая обеспечивает многопоточное выполнение jobs, группировку задач, поведение планирования в стиле work-stealing и платформенные абстракции для Windows и POSIX. Репозиторий включает обширные тесты, примеры и сторонние низкоуровневые компоненты переключения контекста, необходимые для fiber-систем. Он в первую очередь ориентирован на программистов движков, которым нужна масштабируемая job-система для параллельной игровой логики и конвейеров рендеринга.
