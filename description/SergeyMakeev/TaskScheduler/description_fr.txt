Ce projet est un ordonnanceur de tâches multiplateforme basé sur les fibers, conçu pour les charges de travail de jeux haute performance. Il s'agit d'une bibliothèque C++ qui offre l'exécution de jobs multithreads, le regroupement de tâches, un comportement d'ordonnancement de type work-stealing, ainsi que des abstractions de plateforme pour Windows et POSIX. Le dépôt inclut de nombreux tests, des exemples et des composants tiers de bas niveau pour le changement de contexte nécessaires aux systèmes à fibers. Il s'adresse principalement aux programmeurs moteur qui ont besoin d'un système de jobs évolutif pour la logique de jeu et les pipelines de rendu parallèles.
