本プロジェクトは、高性能なゲームワークロード向けに設計されたクロスプラットフォームのファイバーベースタスクスケジューラです。C++ライブラリとして、マルチスレッドでのジョブ実行、タスクグループ化、ワークスティーリング型のスケジューリング、および Windows / POSIX 向けのプラットフォーム抽象化を提供します。リポジトリには、豊富なテストとサンプル、ならびにファイバーシステムに必要なサードパーティの低レベルコンテキストスイッチングコンポーネントが含まれます。並列なゲームロジックやレンダリングパイプライン向けにスケーラブルなジョブシステムを必要とするエンジンプログラマを主な対象としています。
