Этот проект — инструмент мониторинга системных вызовов Windows, который использует события ETW Threat Intelligence для журналирования активности syscall в реальном времени. Через провайдер EtwTi он фиксирует номера syscall, параметры, сведения о вызывающем процессе/потоке и стек вызовов без перехвата в ядре и без установки драйвера. Реализация на C демонстрирует мониторинг syscall в пользовательском режиме для обнаружения подозрительных шаблонов использования API. Предназначен для исследователей безопасности и разработчиков античита, строящих поведенческое обнаружение на основе syscall.
