Ce projet est un outil de surveillance des appels système Windows qui utilise les événements ETW Threat Intelligence pour journaliser l'activité des syscalls en temps réel. Il capture les numéros de syscall, les paramètres, les informations sur le processus/thread appelant et les traces de pile via le fournisseur EtwTi, sans accrochage noyau ni installation de pilote. L'implémentation en C illustre la surveillance des syscalls en mode utilisateur pour détecter des modèles d'utilisation d'API suspects. Il s'adresse aux chercheurs en sécurité et aux développeurs d'anti-cheat qui construisent une détection comportementale basée sur les syscalls.
