Este proyecto es una herramienta de monitoreo de syscalls de Windows que usa eventos ETW Threat Intelligence para registrar la actividad de llamadas al sistema en tiempo real. Captura números de syscall, parámetros, información del proceso/hilo llamante y rastros de pila mediante el proveedor EtwTi, sin enganches en el kernel ni instalación de controladores. La implementación en C demuestra el monitoreo de syscalls en modo usuario para detectar patrones sospechosos de uso de API. Está orientada a investigadores de seguridad y desarrolladores de anti-cheat que construyen detección conductual basada en syscalls.
