Questo progetto è uno strumento di monitoraggio delle syscall di Windows che usa gli eventi ETW Threat Intelligence per registrare l'attività delle chiamate di sistema in tempo reale. Cattura numeri di syscall, parametri, informazioni su processo/thread chiamante e stack trace tramite il provider EtwTi, senza hook del kernel né installazione di driver. L'implementazione in C dimostra il monitoraggio delle syscall in user-mode per rilevare pattern sospetti di uso delle API. È rivolto a ricercatori di sicurezza e sviluppatori anti-cheat che costruiscono rilevamento comportamentale basato sulle syscall.
