Dieses Projekt ist ein Windows-Syscall-Überwachungstool, das ETW-Threat-Intelligence-Ereignisse nutzt, um Systemaufrufaktivitäten in Echtzeit zu protokollieren. Über den EtwTi-Provider erfasst es Syscall-Nummern, Parameter, Informationen zu aufrufendem Prozess/Thread sowie Stacktraces — ohne Kernel-Hooking oder Treiberinstallation. Die C-Implementierung demonstriert die Überwachung von Syscalls im User Mode zur Erkennung verdächtiger API-Nutzungsmuster. Sie richtet sich an Sicherheitsforscher und Anti-Cheat-Entwickler, die verhaltensbasierte Erkennung auf Syscall-Basis aufbauen.
