Este proyecto es una herramienta de Windows que detecta la ejecución de shellcode oculto analizando inconsistencias en el bitmap de Control Flow Guard (CFG). Escanea la memoria de procesos en busca de regiones ejecutables que son objetivos de llamada CFG válidos pero no forman parte del código legítimo de ningún módulo conocido, lo que indica shellcode inyectado marcado como válido para CFG. La implementación en C demuestra el uso de metadatos CFG como señal de detección. Está dirigida a ingenieros anticheat y desarrolladores de EDR que estudian técnicas de detección de inyección de código basadas en CFG.
