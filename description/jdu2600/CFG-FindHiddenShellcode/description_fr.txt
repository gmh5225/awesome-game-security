Ce projet est un outil Windows qui détecte l'exécution de shellcode dissimulé en analysant les incohérences du bitmap Control Flow Guard (CFG). Il parcourt la mémoire des processus à la recherche de régions exécutables qui sont des cibles d'appel CFG valides sans faire partie du code légitime d'aucun module connu, ce qui indique un shellcode injecté marqué comme valide pour CFG. L'implémentation en C montre comment utiliser les métadonnées CFG comme signal de détection. Elle s'adresse aux ingénieurs anti-triche et aux développeurs EDR qui étudient les techniques de détection d'injection de code basées sur CFG.
