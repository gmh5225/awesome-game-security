Dieses Projekt ist ein Windows-Werkzeug, das versteckte Shellcode-Ausführung erkennt, indem es Inkonsistenzen in der Control Flow Guard (CFG)-Bitmap analysiert. Es durchsucht den Prozessspeicher nach ausführbaren Bereichen, die gültige CFG-Aufrufziele sind, aber nicht zum legitimen Code bekannter Module gehören — ein Hinweis auf injizierten Shellcode, der als CFG-gültig markiert wurde. Die C-Implementierung zeigt, wie CFG-Metadaten als Erkennungssignal genutzt werden können. Sie richtet sich an Anti-Cheat-Ingenieure und EDR-Entwickler, die CFG-basierte Erkennungstechniken für Code-Injection untersuchen.
