Этот проект — инструмент для Windows, который обнаруживает скрытое выполнение shellcode, анализируя несоответствия в битовой карте Control Flow Guard (CFG). Он сканирует память процессов в поисках исполняемых областей, являющихся допустимыми целями вызовов CFG, но не входящих в легитимный код известных модулей, что указывает на внедрённый shellcode, помеченный как допустимый для CFG. Реализация на C демонстрирует использование метаданных CFG в качестве сигнала обнаружения. Она предназначена для инженеров античита и разработчиков EDR, изучающих методы обнаружения внедрения кода на основе CFG.
