IntroVirt è un framework di introspezione della virtualizzazione che consente agli analisti di ispezionare e controllare memoria ed esecuzione della VM guest a runtime.
Combina un componente hypervisor KVM patchato con una libreria C++ in userland e il supporto per il parsing dei simboli per un'analisi dettagliata di guest Windows e Linux.
Il codice include API per l'introspezione di processi e thread, breakpoint, accesso alla memoria e visibilità a livello di syscall.
È usato principalmente da reverse engineer e ricercatori di sicurezza che necessitano di monitoraggio out-of-guest, analisi malware o strumenti di sicurezza VM rafforzati.
