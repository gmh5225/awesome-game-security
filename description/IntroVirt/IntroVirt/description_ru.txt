IntroVirt — фреймворк виртуализационной интроспекции, позволяющий аналитикам во время выполнения исследовать и управлять памятью и исполнением гостевых ВМ.
Он сочетает пропатченный компонент гипервизора KVM с пользовательской библиотекой на C++ и поддержкой разбора символов для детального анализа гостевых систем Windows и Linux.
В кодовой базе есть API для интроспекции процессов и потоков, точек останова, доступа к памяти и видимости на уровне системных вызовов.
Его в основном используют инженеры по реверс-инжинирингу и исследователи безопасности, которым нужен мониторинг вне гостя, анализ вредоносного ПО или защищённые средства безопасности ВМ.
