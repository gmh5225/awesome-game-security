IntroVirt es un framework de introspección de virtualización que permite a los analistas inspeccionar y controlar la memoria y la ejecución de la VM invitada en tiempo de ejecución.
Combina un componente de hipervisor KVM parcheado con una biblioteca C++ en espacio de usuario y soporte de análisis de símbolos para un análisis detallado de invitados Windows y Linux.
El código incluye APIs para introspección de procesos e hilos, puntos de interrupción, acceso a memoria y visibilidad a nivel de syscalls.
Lo usan principalmente ingenieros inversos e investigadores de seguridad que necesitan monitorización fuera del invitado, análisis de malware o herramientas de seguridad de VM reforzadas.
