IntroVirt는 분석가가 런타임에 게스트 VM 메모리와 실행을 검사하고 제어할 수 있게 해주는 가상화 인트로스펙션 프레임워크입니다.
패치된 KVM 하이퍼바이저 구성 요소와 C++ 사용자 영역 라이브러리, 심볼 파싱 지원을 결합하여 Windows 및 Linux 게스트를 상세히 분석할 수 있습니다.
코드베이스에는 프로세스 및 스레드 인트로스펙션, 중단점, 메모리 접근, 시스템 호출 수준 가시성을 위한 API가 포함되어 있습니다.
주로 게스트 외부 모니터링, 악성코드 분석 또는 강화된 VM 보안 도구가 필요한 리버스 엔지니어와 보안 연구원이 사용합니다.
