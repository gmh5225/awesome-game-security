IntroVirt は、分析者が実行時にゲスト VM のメモリと実行を検査・制御できる仮想化イントロスペクションフレームワークです。
パッチ済みの KVM ハイパーバイザーコンポーネントと C++ ユーザーランドライブラリ、シンボル解析サポートを組み合わせ、Windows および Linux ゲストの詳細な分析を可能にします。
コードベースには、プロセスおよびスレッドのイントロスペクション、ブレークポイント、メモリアクセス、システムコールレベルの可視性向けの API が含まれています。
主に、ゲスト外監視、マルウェア解析、または強化された VM セキュリティツールを必要とするリバースエンジニアやセキュリティ研究者向けに利用されます。
