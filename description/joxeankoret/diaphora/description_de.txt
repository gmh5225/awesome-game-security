Dieses Projekt ist Diaphora, das fortschrittlichste Open-Source-Tool zum Binärvergleich, umgesetzt als IDA-Pro-Plugin in Python. Es vergleicht zwei Binärdateien auf Funktionsebene mithilfe mehrerer Heuristiken, darunter Matching von Kontrollflussgraphen, Hash-Vergleich von Basic Blocks, Instruktionsmnemonics, String-Referenzen und Call-Graph-Topologie. Es unterstützt partielles Matching, das Übertragen von Symbolen und Kommentaren zwischen Datenbankversionen und erzeugt detaillierte Diff-Berichte. Es richtet sich an Reverse Engineers, Vulnerability Researcher und Patch-Analysten, die Binär-Updates vergleichen und Codeänderungen über Softwareversionen hinweg identifizieren.
