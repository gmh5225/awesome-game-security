Este proyecto es Diaphora, la herramienta de diff binario de código abierto más avanzada, implementada como un plugin de IDA Pro en Python. Realiza una comparación a nivel de función entre dos binarios mediante múltiples heurísticas, como el emparejamiento de grafos de flujo de control, la comparación de hashes de bloques básicos, mnemónicos de instrucciones, referencias a cadenas y la topología del grafo de llamadas. Admite coincidencias parciales, la migración de símbolos y comentarios entre versiones de bases de datos, y genera informes de diff detallados. Está orientada a ingenieros de ingeniería inversa, investigadores de vulnerabilidades y analistas de parches que comparan actualizaciones binarias e identifican cambios de código entre versiones de software.
