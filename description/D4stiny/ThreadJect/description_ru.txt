Этот проект — proof-of-concept ручного DLL-инжектора, который перехватывает существующий поток в целевом процессе. Он проверяет и отображает (map) полезную нагрузку DLL, готовит метаданные загрузчика, патчит shellcode адресами времени выполнения и перенаправляет выполнение потока для завершения инъекции. Реализация на C++ под Windows с поддержкой проекта Visual Studio; акцент на низкоуровневом контроле цепочки инъекции. Предназначен для исследований инъекции процессов и построения тестовых сценариев обнаружения в исследованиях безопасности конечных точек.
