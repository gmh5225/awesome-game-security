Questo progetto è un injector DLL manuale proof-of-concept che dirotta un thread esistente in un processo target. Valida e mappa il payload DLL, prepara i metadati del loader, applica patch allo shellcode con indirizzi runtime e reindirizza l’esecuzione del thread per completare l’iniezione. L’implementazione è in C++ su Windows con supporto per progetto Visual Studio e enfatizza il controllo a basso livello sulla catena di iniezione. È pensato per la ricerca sull’iniezione di processi e per costruire casi di test di rilevamento negli studi di sicurezza degli endpoint.
