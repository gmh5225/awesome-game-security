Este proyecto es un inyector DLL manual de prueba de concepto que secuestra un hilo existente en un proceso objetivo. Valida y mapea la carga útil DLL, prepara metadatos del cargador, parchea el shellcode con direcciones en tiempo de ejecución y redirige la ejecución del hilo para completar la inyección. La implementación es en C++ en Windows con soporte de proyecto de Visual Studio y enfatiza el control de bajo nivel sobre la cadena de inyección. Está pensado para investigación de inyección de procesos y para construir casos de prueba de detección en estudios de seguridad de endpoints.
