Ce projet est une plateforme de détection de menaces en mode noyau pour Windows, centrée sur la détection de malwares basée sur le comportement. Elle surveille des événements tels que la création de processus, le chargement d'images, l'activité de threads distants, l'exécution de code masqué, ainsi que les opérations suspectes sur le système de fichiers ou le Registre, avec le contexte de la pile d'appels. Le code source comprend un pilote, des composants en ligne de commande et une interface Qt pour la télémétrie et les workflows de réponse. Il vise les expériences de sécurité défensive et la recherche anti-malware, en particulier dans des environnements de test à haute visibilité.
