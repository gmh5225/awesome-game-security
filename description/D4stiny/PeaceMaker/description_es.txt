Este proyecto es una plataforma de detección de amenazas en modo kernel para Windows centrada en la detección de malware basada en el comportamiento. Supervisa eventos como la creación de procesos, la carga de imágenes, la actividad de hilos remotos, la ejecución de código oculto y operaciones sospechosas del sistema de archivos o del registro, con contexto de seguimiento de pila. El código incluye un controlador, componentes de línea de comandos y una interfaz gráfica basada en Qt para telemetría y flujos de respuesta. Está orientado a experimentos de seguridad defensiva e investigación antimalware, especialmente en entornos de prueba de alta visibilidad.
