Этот проект — платформа обнаружения угроз в режиме ядра для Windows с акцентом на поведенческое детектирование вредоносного ПО. Она отслеживает события вроде создания процессов, загрузки образов, активности удалённых потоков, скрытого выполнения кода, а также подозрительных операций с файловой системой или реестром с контекстом стека вызовов. Кодовая база включает драйвер, компоненты командной строки и GUI на Qt для телеметрии и сценариев реагирования. Проект предназначен для оборонительных экспериментов в области безопасности и исследований антивирусной защиты, особенно в тестовых средах с высокой наблюдаемостью.
