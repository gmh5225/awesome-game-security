Эксплойт DLL side-loading для signtool.exe от Microsoft, который перехватывает XmlLite.dll для патча в памяти CertVerifyTimeValidity (crypt32.dll) и GetSystemTimeAsFileTime (KernelBase.dll) с помощью WriteProcessMemory, фактически обнуляя возвращаемые значения, чтобы обойти проверки срока действия сертификата при Authenticode-подписи.
