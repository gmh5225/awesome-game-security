Dieses Projekt stellt ein auf macOS ausgerichtetes Paket und eine Nutzungsanleitung bereit, um ein forensisches Festplatten-Imaging-Tool über die Kommandozeile auszuführen.
Im Mittelpunkt stehen das Erfassen von Laufwerksabbildern, das Aufteilen der Ausgaben sowie die Integritätsprüfung mit Hash-Prüfungen wie MD5 und optional SHA1.
Das Repository enthält ausführbare Nutzungsbeispiele sowie Optionen für E01/SMART-Formate, Fragmentierung, Kompression und Metadatenfelder für Beweismittel.
Die Hauptzielgruppe sind digitale Forensiker und Incident Responder, die Imaging-Workflows unter macOS benötigen.
