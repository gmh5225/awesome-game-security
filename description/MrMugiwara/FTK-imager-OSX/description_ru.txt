Этот проект предоставляет пакет и руководство по использованию с упором на macOS для запуска утилиты создания форензик-образов дисков из командной строки.
В центре внимания — получение образов накопителей, разбиение выходных файлов и проверка целостности с помощью хешей, таких как MD5 и опциональный SHA1.
Репозиторий включает исполняемые примеры использования и параметры для форматов E01/SMART, фрагментации, сжатия и полей метаданных доказательств.
Основная аудитория — специалисты по цифровой криминалистике и сотрудники реагирования на инциденты, которым нужны рабочие процессы создания образов на macOS.
