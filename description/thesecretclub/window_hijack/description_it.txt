Questo progetto è un driver kernel di Windows che esegue il hijacking degli handle delle finestre per una comunicazione nascosta da kernel a user mode. Include handler di controllo dispatch, moduli data getter per leggere informazioni sui processi e header di definizioni NT/driver per interagire con il sottosistema delle finestre di Windows dalla modalità kernel. È soprattutto utile per i ricercatori di sicurezza del kernel che studiano la manipolazione degli oggetti finestra e i canali di comunicazione nascosti tra driver kernel e applicazioni in user mode.
