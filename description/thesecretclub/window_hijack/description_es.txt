Este proyecto es un controlador de kernel de Windows que realiza el secuestro de handles de ventanas para una comunicación encubierta de kernel a usuario. Incluye manejadores de control de dispatch, módulos de obtención de datos para leer información de procesos y encabezados de definiciones NT/controlador para interactuar con el subsistema de ventanas de Windows desde el modo kernel. Resulta especialmente útil para investigadores de seguridad de kernel que estudian la manipulación de objetos ventana y los canales de comunicación encubiertos entre controladores de kernel y aplicaciones en modo usuario.
