Этот проект — драйвер ядра Windows, выполняющий перехват (hijacking) дескрипторов окон для скрытой связи между ядром и пользовательским режимом. Он включает обработчики управления диспетчеризацией, модули получения данных для чтения сведений о процессах, а также заголовки определений NT/драйвера для взаимодействия с оконной подсистемой Windows из режима ядра. В первую очередь полезен исследователям безопасности ядра, изучающим манипуляции с оконными объектами и скрытые каналы связи между драйверами ядра и приложениями пользовательского режима.
