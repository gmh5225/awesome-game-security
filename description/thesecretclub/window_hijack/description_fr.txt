Ce projet est un pilote noyau Windows qui effectue le détournement de handles de fenêtres pour une communication furtive entre le noyau et l'espace utilisateur. Il comprend des gestionnaires de contrôle de dispatch, des modules d'obtention de données pour lire les informations de processus, ainsi que des en-têtes de définitions NT/pilote pour interagir avec le sous-système de fenêtres Windows depuis le mode noyau. Il s'adresse principalement aux chercheurs en sécurité noyau qui étudient la manipulation des objets fenêtre et les canaux de communication furtifs entre pilotes noyau et applications en mode utilisateur.
