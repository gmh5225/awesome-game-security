Dieses Projekt ist ein Windows-Kerneltreiber, der Window-Handle-Hijacking für verdeckte Kommunikation vom Kernel zum User Mode durchführt. Es enthält Dispatch-Control-Handler, Data-Getter-Module zum Lesen von Prozessinformationen sowie NT-/Treiber-Definitionsheader zur Interaktion mit dem Windows-Fenstersubsystem aus dem Kernelmodus. Es ist vor allem für Kernel-Sicherheitsforscher nützlich, die die Manipulation von Fensterobjekten und verdeckte Kommunikationskanäle zwischen Kerneltreibern und User-Mode-Anwendungen untersuchen.
