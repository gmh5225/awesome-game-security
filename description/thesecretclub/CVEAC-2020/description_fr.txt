Ce projet est un pilote noyau Windows de type proof-of-concept qui exploite une vulnérabilité dans le module noyau d'EasyAntiCheat. Le pilote C++ WDK inclut du code d'analyse spécifique à EAC, l'énumération de modules noyau, le parsing PE, l'installation de hooks, ainsi que l'intégration NMD assembly pour la manipulation de code à l'exécution. Il s'adresse principalement aux chercheurs en sécurité anti-triche qui étudient les vulnérabilités du pilote noyau EasyAntiCheat et les techniques de contournement anti-triche.
