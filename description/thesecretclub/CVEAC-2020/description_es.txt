Este proyecto es un controlador de kernel de Windows de tipo proof-of-concept que explota una vulnerabilidad en el módulo de kernel de EasyAntiCheat. El controlador C++ WDK incluye código de análisis específico de EAC, enumeración de módulos de kernel, análisis PE, instalación de hooks e integración de ensamblado NMD para la manipulación de código en tiempo de ejecución. Resulta especialmente útil para investigadores de seguridad antitrampas que estudian vulnerabilidades del controlador de kernel de EasyAntiCheat y técnicas de bypass antitrampas.
