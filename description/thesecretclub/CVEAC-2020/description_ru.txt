Этот проект — proof-of-concept драйвер ядра Windows, эксплуатирующий уязвимость в модуле ядра EasyAntiCheat. Драйвер на C++ для WDK включает код анализа, специфичный для EAC, перечисление модулей ядра, разбор PE, установку хуков и интеграцию NMD assembly для манипуляции кодом во время выполнения. В первую очередь он полезен исследователям безопасности античитов, изучающим уязвимости драйвера ядра EasyAntiCheat и методы обхода античита.
