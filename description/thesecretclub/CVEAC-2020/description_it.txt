Questo progetto è un driver kernel Windows proof-of-concept che sfrutta una vulnerabilità nel modulo kernel di EasyAntiCheat. Il driver C++ WDK include codice di analisi specifico per EAC, enumerazione dei moduli kernel, parsing PE, installazione di hook e integrazione NMD assembly per la manipolazione del codice a runtime. È pensato soprattutto per i ricercatori di sicurezza anti-cheat che studiano le vulnerabilità del driver kernel EasyAntiCheat e le tecniche di bypass anti-cheat.
