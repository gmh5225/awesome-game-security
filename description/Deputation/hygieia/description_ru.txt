Этот проект — драйвер ядра Windows для исследования следов, оставленных уязвимыми драйверами. Он сканирует структуры пейджинга, чтобы находить известные артефакты драйверов, и поддерживает отображения страниц размером 1 ГБ, 2 МБ и 4 КБ. Реализация написана на C/C++ с инструментами драйверов в стиле WDK и ориентирована на низкоуровневую memory forensics. В основном предназначен для исследований античита и безопасности ядра, связанных с обнаружением или анализом предшествующей активности неподписанных драйверов.
