Этот проект — AqHax, внутренний чит для CS:GO на C++, предоставляющий ESP, аимбот, триггербот, хаки передвижения и скин-чейнджер через хукинг SDK движка Source. Он использует захват интерфейсов, VMT-хукинг и дампинг netvar — типичные приёмы внутренних читов CS:GO. Предназначен для исследователей игровой безопасности, изучающих реализации читов CS:GO.
