Этот проект — библиотека на Rust для взаимодействия с протоколом BattlEye RCON (удалённая консоль). Она реализует кодирование пакетов с хешированием контрольной суммы, обмен данными через сокеты и содержит примеры прослушивания, а также чтения и записи команд удалённой консоли BattlEye. В первую очередь полезна администраторам игровых серверов и исследователям безопасности, изучающим протокол удалённой консоли BattlEye и интерфейс управления сервером.
