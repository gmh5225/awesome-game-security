Este proyecto es una biblioteca en Rust para interactuar con el protocolo BattlEye RCON (Remote Console). Implementa la codificación de paquetes con hash de suma de comprobación, la comunicación basada en sockets y ofrece ejemplos para escuchar y leer/escribir comandos de la consola remota de BattlEye. Resulta especialmente útil para administradores de servidores de juegos e investigadores de seguridad que estudian el protocolo de consola remota de BattlEye y la interfaz de gestión de servidores.
