Este proyecto es un plugin de IDA Pro que resuelve APIs y constantes de cadena hasheadas mediante el servicio de consulta HashDB. Escrito en Python, se integra con los menús contextuales de IDA para realizar búsquedas individuales, importaciones masivas de módulos y coincidencia opcional con soporte XOR. También puede localizar algoritmos de hash probables y anotar coincidencias como enumeraciones directamente en el desensamblado. El plugin está orientado a flujos de ingeniería inversa donde la ofuscación basada en hashes es habitual.
