Этот проект — плагин для IDA Pro, который разрешает хешированные API и строковые константы через сервис поиска HashDB. Написан на Python и интегрируется с контекстными меню IDA для одиночных запросов, массового импорта модулей и опционального сопоставления с учётом XOR. Он также может искать вероятные алгоритмы хеширования и помечать совпадения как перечисления прямо в дизассемблере. Плагин рассчитан на рабочие процессы реверс-инжиниринга, где распространено обфусцирование на основе хешей.
