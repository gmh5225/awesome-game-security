Ce projet est un plugin IDA Pro qui résout les API et constantes de chaînes hachées via le service de recherche HashDB. Écrit en Python, il s'intègre aux menus contextuels d'IDA pour effectuer des recherches individuelles, des imports de modules en masse et une correspondance optionnelle prenant en compte le XOR. Il peut aussi identifier des algorithmes de hachage probables et annoter les correspondances sous forme d'énumérations directement dans le désassemblage. Le plugin cible les flux de reverse engineering où l'obfuscation par hachage est courante.
