Questo progetto è un plugin per IDA Pro che risolve API e costanti stringa hashate tramite il servizio di lookup HashDB. Scritto in Python, si integra con i menu contestuali di IDA per eseguire lookup singoli, importazioni massive di moduli e matching opzionale consapevole del XOR. Può anche individuare probabili algoritmi di hash e annotare le corrispondenze come enum direttamente nel disassembly. Il plugin è pensato per flussi di reverse engineering in cui l'offuscamento basato su hash è comune.
