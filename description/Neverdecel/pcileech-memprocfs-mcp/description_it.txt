Questo progetto è un server Model Context Protocol (MCP) nativo Linux che offre agli assistenti IA accesso diretto alle operazioni di memoria basate su DMA tramite PCILeech e MemProcFS. Scritto in Python e basato sulle API memprocfs e leechcorepyc, espone decine di strumenti per lettura/scrittura della memoria in tempo reale, analisi di processi e moduli, scansione di pattern, scoperta di catene di puntatori e ricerca di riferimenti incrociati. Include anche helper specifici per i motori per esportare gli SDK C++ di Unreal Engine 4/5 e le definizioni delle classi Unity IL2CPP, oltre al controllo FPGA per benchmark e operazioni TLP PCIe. Il caso d’uso principale è il reverse engineering assistito da DMA e la ricerca sulla sicurezza dei giochi, consentendo agli analisti di guidare l’ispezione della memoria e l’estrazione degli SDK in linguaggio naturale invece che con lavoro manuale da CLI.
