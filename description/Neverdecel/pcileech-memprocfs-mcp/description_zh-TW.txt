本專案是一個原生 Linux 的 Model Context Protocol（MCP）伺服器，透過 PCILeech 與 MemProcFS 為 AI 助手提供基於 DMA 的記憶體操作直接存取能力。以 Python 撰寫，並建基於 memprocfs 與 leechcorepyc API，對外提供數十種工具，涵蓋即時記憶體讀寫、行程與模組分析、特徵碼掃描、指標鏈發現以及交叉參照尋找。亦包含引擎專用輔助功能，可用於傾印 Unreal Engine 4/5 的 C++ SDK 與 Unity IL2CPP 類別定義，並支援 FPGA 控制以進行基準測試與 PCIe TLP 操作。主要用途為 DMA 輔助的逆向工程與遊戲安全研究，讓分析人員能以自然語言驅動記憶體檢查與 SDK 擷取，而無須手動操作命令列。
