本プロジェクトは、PCILeech と MemProcFS を通じて AI アシスタントに DMA ベースのメモリ操作への直接アクセスを提供する、Linux ネイティブの Model Context Protocol（MCP）サーバーです。Python で書かれ、memprocfs および leechcorepyc API 上に構築されており、ライブメモリの読み書き、プロセス／モジュール解析、パターンスキャン、ポインタチェーン発見、クロスリファレンス検索など、数十のツールを公開します。また、Unreal Engine 4/5 の C++ SDK と Unity IL2CPP のクラス定義をダンプするエンジン固有のヘルパー、ならびにベンチマークや PCIe TLP 操作用の FPGA 制御も含まれます。主な用途は DMA 支援のリバースエンジニアリングとゲームセキュリティ研究で、アナリストは手動の CLI 作業ではなく自然言語でメモリ検査や SDK 抽出を行えます。
