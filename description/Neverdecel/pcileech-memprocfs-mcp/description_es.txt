Este proyecto es un servidor Model Context Protocol (MCP) nativo de Linux que otorga a los asistentes de IA acceso directo a operaciones de memoria basadas en DMA a través de PCILeech y MemProcFS. Escrito en Python y construido sobre las API memprocfs y leechcorepyc, expone docenas de herramientas para lectura/escritura de memoria en vivo, análisis de procesos y módulos, escaneo de patrones, descubrimiento de cadenas de punteros y búsqueda de referencias cruzadas. También incluye helpers específicos del motor para volcar SDK C++ de Unreal Engine 4/5 y definiciones de clases de Unity IL2CPP, además de control FPGA para benchmarks y operaciones TLP de PCIe. El caso de uso principal es la ingeniería inversa asistida por DMA y la investigación en seguridad de juegos, permitiendo a los analistas dirigir la inspección de memoria y la extracción de SDK mediante lenguaje natural en lugar de trabajo manual en la CLI.
