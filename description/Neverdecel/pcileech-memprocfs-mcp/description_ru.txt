Этот проект — нативный для Linux сервер Model Context Protocol (MCP), который даёт ИИ-ассистентам прямой доступ к операциям с памятью на основе DMA через PCILeech и MemProcFS. Написан на Python и построен на API memprocfs и leechcorepyc; предоставляет десятки инструментов для чтения/записи живой памяти, анализа процессов и модулей, сканирования сигнатур, поиска цепочек указателей и перекрёстных ссылок. Также включает специализированные средства для дампа C++ SDK Unreal Engine 4/5 и определений классов Unity IL2CPP, а также управление FPGA для бенчмарков и операций PCIe TLP. Основной сценарий использования — реверс-инжиниринг с помощью DMA и исследования безопасности игр: аналитики могут управлять инспекцией памяти и извлечением SDK на естественном языке вместо ручной работы в CLI.
