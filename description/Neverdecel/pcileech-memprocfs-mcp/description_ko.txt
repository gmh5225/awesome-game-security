이 프로젝트는 PCILeech와 MemProcFS를 통해 AI 어시스턴트에 DMA 기반 메모리 작업에 대한 직접 접근을 제공하는 Linux 네이티브 Model Context Protocol(MCP) 서버입니다. Python으로 작성되었고 memprocfs 및 leechcorepyc API 위에 구축되어, 실시간 메모리 읽기/쓰기, 프로세스 및 모듈 분석, 패턴 스캔, 포인터 체인 탐색, 교차 참조 검색 등 수십 개의 도구를 제공합니다. 또한 Unreal Engine 4/5 C++ SDK와 Unity IL2CPP 클래스 정의를 덤프하는 엔진별 헬퍼, 벤치마크 및 PCIe TLP 작업을 위한 FPGA 제어 기능도 포함합니다. 주요 사용 사례는 DMA 지원 리버스 엔지니어링 및 게임 보안 연구로, 분석가가 수동 CLI 작업 대신 자연어로 메모리 검사와 SDK 추출을 수행할 수 있게 합니다.
