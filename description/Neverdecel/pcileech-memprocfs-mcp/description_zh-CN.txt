本项目是一个原生 Linux 的 Model Context Protocol（MCP）服务器，通过 PCILeech 与 MemProcFS 为 AI 助手提供基于 DMA 的内存操作直接访问能力。使用 Python 编写，基于 memprocfs 与 leechcorepyc API 构建，对外提供数十种工具，涵盖实时内存读写、进程与模块分析、特征码扫描、指针链发现以及交叉引用查找。还包含引擎专用辅助功能，用于导出 Unreal Engine 4/5 的 C++ SDK 与 Unity IL2CPP 类定义，并支持 FPGA 控制以进行基准测试与 PCIe TLP 操作。主要面向 DMA 辅助的逆向工程与游戏安全研究，使分析人员可通过自然语言驱动内存检查与 SDK 提取，而无需手动使用命令行。
