SmKernel-CSGO — это учебный proof of concept, демонстрирующий архитектуру игрового чита с помощью ядерного драйвера для CS:GO. Он использует обмен данными через shared memory между компонентами user mode и kernel mode, чтобы предоставлять операции вроде поиска базового адреса модуля и чтения или записи памяти процесса. В проекте также есть простой пример triggerbot и открыто обсуждаются компромиссы по обнаруживаемости при более сильных ядерных античитах. Материал в основном предназначен для изучения концепций драйверного game hacking и понимания threat model античита.
