SmKernel-CSGO es una prueba de concepto educativa que demuestra una arquitectura de cheat asistida por un controlador de kernel para CS:GO. Usa comunicación por memoria compartida entre componentes en modo usuario y modo kernel para exponer operaciones como la búsqueda de la base de módulos y la lectura o escritura de memoria de procesos. El proyecto también incluye un ejemplo sencillo de triggerbot y analiza abiertamente los compromisos de detectabilidad frente a sistemas anti-cheat de kernel más fuertes. Está pensado principalmente para aprender conceptos de game hacking basados en drivers y comprender los modelos de amenaza del anti-cheat.
