Binary Internals Viewer è un’applicazione Java per l’ispezione visiva e interattiva dei formati di file binari, fino alla struttura a livello di campi e bit. Supporta diversi formati, tra cui BMP, file di classe Java, JPEG, PNG e ZIP, con moduli di formato aggiuntivi in sviluppo. Il progetto è costruito con OpenJDK e Maven e fornisce anche librerie di formato autonome e riutilizzabili. È utile per la formazione sull’ingegneria inversa, lo sviluppo di parser e l’analisi pratica dei formati binari.
