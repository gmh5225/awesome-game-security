Binary Internals Viewer — Java-приложение для визуального интерактивного анализа форматов двоичных файлов вплоть до структуры на уровне полей и битов. Поддерживаются несколько форматов, включая BMP, файлы классов Java, JPEG, PNG и ZIP; дополнительные модули форматов находятся в разработке. Проект собирается с помощью OpenJDK и Maven и также поставляет переиспользуемые автономные библиотеки форматов. Полезен для обучения реверс-инжинирингу, разработки парсеров и практического анализа двоичных форматов.
