Binary Internals Viewer es una aplicación Java para la inspección visual e interactiva de formatos de archivos binarios, hasta la estructura a nivel de campos y bits. Admite varios formatos, incluidos BMP, archivos de clase Java, JPEG, PNG y ZIP, con módulos de formato adicionales en desarrollo. El proyecto se construye con OpenJDK y Maven y también incluye bibliotecas de formato independientes y reutilizables. Es útil para la formación en ingeniería inversa, el desarrollo de analizadores y el análisis práctico de formatos binarios.
