mempatch — инструмент для изменения значений в памяти приложений, работающих на устройствах Android. Он реализован на C++ с поддержкой Android NDK и включает компоненты для работы с адресами, патчинга, снимков состояния, отслеживания диапазонов и опциональных сценариев «заморозки» значений. В репозитории также есть код абстракции платформы и тестовые материалы, помогающие проверять поведение редактирования памяти в разных средах. Инструмент предназначен для оценки уязвимостей и поддержки разработки, а не для применения к уже выпущенным production-приложениям.
