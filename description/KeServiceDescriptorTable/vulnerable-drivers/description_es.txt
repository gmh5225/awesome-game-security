Este proyecto es una colección curada de controladores de kernel de Windows firmados y vulnerables, reunida para investigación de seguridad. Incluye docenas de binarios de controladores .sys de varios proveedores que exponen interfaces IOCTL explotables, primitivas de acceso a memoria u otras debilidades de seguridad. Es principalmente útil para investigadores BYOVD, ingenieros de anti-cheat y analistas de seguridad de kernel que estudian el catalogado de controladores vulnerables y el análisis de la superficie de explotación.
