Este proyecto proporciona el binario del controlador de kernel de Windows vulnerable Cormem.sys para fines de investigación de seguridad. El controlador expone primitivas de lectura y escritura de memoria que pueden explotarse para obtener acceso a nivel de kernel, lo que lo convierte en un candidato para la investigación de ataques BYOVD (Bring Your Own Vulnerable Driver). Es principalmente útil para investigadores de seguridad de kernel y analistas anti-cheat que estudian técnicas de explotación y detección de controladores vulnerables.
