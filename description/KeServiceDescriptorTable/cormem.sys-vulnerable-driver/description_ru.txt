Этот проект предоставляет бинарный файл уязвимого драйвера ядра Windows Cormem.sys для целей исследования безопасности. Драйвер предоставляет примитивы чтения и записи памяти, которые могут быть использованы для доступа на уровне ядра, что делает его кандидатом для исследования атак BYOVD (Bring Your Own Vulnerable Driver). Он в основном полезен исследователям безопасности ядра и аналитикам античита, изучающим техники эксплуатации и обнаружения уязвимых драйверов.
