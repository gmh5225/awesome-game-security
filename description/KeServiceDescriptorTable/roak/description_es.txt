Este proyecto es un controlador en modo kernel de Windows que proporciona acceso de lectura y escritura a memoria mediante un canal de comunicación personalizado. Engancha funciones de consulta del temporizador HAL para una comunicación encubierta usuario-kernel, gestiona operaciones de memoria mediante despacho de solicitudes basado en paquetes e incluye resolución de offsets del kernel y módulos de utilidades del sistema. Resulta principalmente útil para investigadores de seguridad del kernel que estudian técnicas de comunicación encubierta de controladores y patrones de acceso a memoria del kernel.
