Этот проект — репозиторий метаданных и документации для официально поддерживаемых устройств PixelOS. В нём в основном используются Markdown и JSON для публикации списков устройств, данных API, заметок в стиле changelog и шаблонов сообщений о релизах. Содержимое помогает отслеживать поддерживаемые бренды и модели и стандартизирует то, как мейнтейнеры представляют информацию о сборках и релизах. Он в первую очередь полезен мейнтейнерам PixelOS и пользователям, которым нужна авторитетная матрица поддержки.
