Este proyecto es un repositorio de metadatos y documentación para dispositivos PixelOS oficialmente compatibles. Utiliza principalmente Markdown y JSON para publicar listas de dispositivos, datos de API, notas al estilo de registro de cambios y plantillas de comunicación de versiones. El contenido ayuda a rastrear marcas y modelos compatibles y estandariza cómo los mantenedores presentan la información de compilaciones y versiones. Es especialmente útil para mantenedores de PixelOS y usuarios que necesitan una matriz de compatibilidad autorizada.
