Este proyecto es el repositorio de manifests usado para definir y sincronizar los árboles de código fuente de PixelOS. Se compone principalmente de archivos manifest XML y documentación que explica la inicialización de repo, la sincronización y los pasos de compilación de la ROM. El repositorio ofrece manifests predeterminados y bloqueados por revisión, así como archivos de fragmentos para controlar qué proyectos Android upstream se obtienen. Está pensado para mantenedores y constructores de ROM que compilan y siguen las versiones de PixelOS.
