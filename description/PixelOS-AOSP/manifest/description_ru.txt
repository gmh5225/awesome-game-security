Этот проект — репозиторий манифестов для определения и синхронизации деревьев исходного кода PixelOS. Он в основном состоит из XML-манифестов и документации по инициализации repo, синхронизации и шагам сборки ROM. Репозиторий предоставляет манифесты по умолчанию и с зафиксированными ревизиями, а также snippet-файлы для управления тем, какие upstream-проекты Android загружаются. Предназначен для сопровождающих и сборщиков ROM, которые компилируют и отслеживают релизы PixelOS.
