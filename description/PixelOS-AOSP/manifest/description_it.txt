Questo progetto è il repository di manifest usato per definire e sincronizzare gli alberi sorgente di PixelOS. È composto principalmente da file manifest XML e da documentazione che spiega l’inizializzazione di repo, la sincronizzazione e i passaggi di build della ROM. Il repository fornisce manifest predefiniti e bloccati per revisione, oltre a file snippet per controllare quali progetti Android upstream vengono recuperati. È destinato a maintainer e builder di ROM che compilano e seguono le release di PixelOS.
