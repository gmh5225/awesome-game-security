Инструмент для Windows, который дампит смещения памяти, интерфейсы и netvars из Apex Legends, читая память процесса игры, и предоставляет адреса, необходимые для разработки читов, включая указатели SwapChain.
Он выполняет поиск по сигнатурам и перечисление netvars, чтобы извлекать смещения игровых структур, сохраняющиеся после небольших обновлений.
В основном полезен исследователям игровой безопасности, изучающим техники дампа смещений движка Source и требования к обходу античита для доступа к памяти.
