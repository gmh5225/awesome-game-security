Этот проект — демонстрационный драйвер ядра Windows для отрисовки графики без установки традиционных хуков. Он реализован на C++ и сосредоточен на вызове связанных с GDI путей ядра путём подмены необходимых значений контекста потока, чтобы удовлетворить внутренние проверки. Пример включает вспомогательные примитивы и заметки о зависящих от версии NT-смещениях и методах загрузки. В основном используется для исследований низкоуровневой графики и обхода античита в контролируемых средах.
