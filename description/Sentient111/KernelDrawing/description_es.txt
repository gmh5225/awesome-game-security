Este proyecto es un controlador de núcleo de Windows de prueba de concepto para dibujar gráficos sin instalar hooks tradicionales. Está implementado en C++ y se centra en llamar rutas de núcleo relacionadas con GDI falsificando los valores de contexto de hilo necesarios para satisfacer las comprobaciones internas. El ejemplo incluye primitivas de apoyo y notas sobre offsets NT dependientes de la versión y métodos de carga. Se usa principalmente para investigación de gráficos de bajo nivel y evasión de anti-cheat en entornos controlados.
