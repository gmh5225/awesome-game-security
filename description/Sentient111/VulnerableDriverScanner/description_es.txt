Este proyecto es una utilidad de clasificación de controladores de Windows que escanea carpetas en busca de controladores de núcleo potencialmente riesgosos. Está implementada como una aplicación de consola en C++ que analiza las importaciones PE y marca binarios que contienen API seleccionadas relacionadas con controladores. La lógica de detección actual es sencilla y se centra en importaciones indicativas en lugar de un análisis conductual completo. Es útil para la búsqueda preliminar de controladores vulnerables y la evaluación de la superficie de ataque del núcleo en flujos de trabajo de investigación de seguridad.
