Questo progetto è un’utilità di triage per driver Windows che analizza cartelle alla ricerca di driver kernel potenzialmente rischiosi. È implementata come applicazione console C++ che analizza le import PE e segnala i binari contenenti API selezionate correlate ai driver. La logica di rilevamento attuale è semplice e si concentra su import indicative piuttosto che su un’analisi comportamentale completa. È utile per la ricerca preliminare di driver vulnerabili e per la valutazione della superficie di attacco del kernel nei flussi di lavoro di ricerca sulla sicurezza.
