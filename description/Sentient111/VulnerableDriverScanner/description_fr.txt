Ce projet est un utilitaire de triage de pilotes Windows qui analyse des dossiers à la recherche de pilotes noyau potentiellement risqués. Il est implémenté comme une application console C++ qui parse les imports PE et signale les binaires contenant certaines API liées aux pilotes. La logique de détection actuelle est simple et se concentre sur des imports indicatifs plutôt que sur une analyse comportementale complète. Il est utile pour une chasse préliminaire aux pilotes vulnérables et pour l’évaluation de la surface d’attaque noyau dans les workflows de recherche en sécurité.
