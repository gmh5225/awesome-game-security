Questo progetto è una piccola utility per copiare i dati del certificato da un file PE a un altro eseguibile. È scritto in C++ e analizza le intestazioni PE per individuare e duplicare il blob della directory di sicurezza Authenticode. Il file risultante porta metadati del certificato copiati ma non diventa una firma attendibile valida, il che è utile per testare il comportamento dei verificatori. È pensato per la ricerca sulla sicurezza Windows riguardante la gestione delle firme, il rilevamento di manomissioni e i casi limite della pipeline di trust.
