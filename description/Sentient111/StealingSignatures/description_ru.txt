Этот проект — небольшая утилита для копирования данных сертификата из одного PE-файла в другой исполняемый файл. Она написана на C++ и разбирает PE-заголовки, чтобы найти и продублировать blob каталога безопасности Authenticode. Полученный файл содержит скопированные метаданные сертификата, но не становится действительной доверенной подписью, что полезно для проверки поведения верификаторов. Предназначена для исследований безопасности Windows в области обработки подписей, обнаружения подделки и граничных случаев конвейера доверия.
