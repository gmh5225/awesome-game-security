Este proyecto es una pequeña utilidad para copiar datos de certificado de un archivo PE a otro ejecutable. Está escrito en C++ y analiza los encabezados PE para localizar y duplicar el blob del directorio de seguridad Authenticode. El archivo resultante lleva metadatos de certificado copiados, pero no se convierte en una firma de confianza válida, lo cual es útil para probar el comportamiento de los verificadores. Está pensado para investigación de seguridad en Windows sobre el manejo de firmas, la detección de manipulación y los casos límite del pipeline de confianza.
