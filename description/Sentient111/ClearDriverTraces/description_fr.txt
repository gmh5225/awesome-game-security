Ce projet est un pilote noyau Windows destiné à supprimer les traces forensiques laissées par le chargement d'autres pilotes. Il contient des routines C++ en mode noyau qui ciblent des structures telles que MmUnloadedDrivers, PiDDBCacheTable et les caches de hachage d'intégrité du code. L'implémentation s'appuie sur des offsets spécifiques à la version et sur la manipulation bas niveau des structures de données du noyau. Il est principalement utilisé dans la recherche anti-cheat et forensique sur les pilotes pour étudier quels artefacts sont créés et comment la logique de détection peut les suivre.
