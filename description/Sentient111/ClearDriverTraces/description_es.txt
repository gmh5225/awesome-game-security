Este proyecto es un controlador de núcleo de Windows para eliminar rastros forenses dejados al cargar otros controladores. Contiene rutinas C++ en modo núcleo que apuntan a estructuras como MmUnloadedDrivers, PiDDBCacheTable y las cachés de hash de integridad de código. La implementación se basa en offsets específicos de la versión y en la manipulación de bajo nivel de estructuras de datos del núcleo. Se usa principalmente en investigación de anti-cheat y forense de controladores para estudiar qué artefactos se crean y cómo la lógica de detección puede rastrearlos.
