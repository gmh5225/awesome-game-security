Questo progetto è un driver del kernel Windows per rimuovere le tracce forensi lasciate dal caricamento di altri driver. Contiene routine C++ in modalità kernel che mirano a strutture come MmUnloadedDrivers, PiDDBCacheTable e le cache hash di code integrity. L'implementazione si basa su offset specifici della versione e sulla manipolazione a basso livello delle strutture dati del kernel. È usato principalmente nella ricerca anti-cheat e forense sui driver per studiare quali artefatti vengono creati e come la logica di rilevamento può tracciarli.
