Ce projet est un plugin x64dbg qui aide les analystes à déboguer des exécutables protégés par Themida sur Windows 64 bits. Il s’appuie sur le hooking C++ avec MinHook et s’intègre aux outils de débogage courants pour neutraliser les contrôles anti-debug, anti-VM et de surveillance. Le plugin privilégie des workflows pratiques d’attachement et de débogage plutôt qu’une automatisation complète du dépaquetage. Son usage principal concerne l’ingénierie inverse de logiciels protégés dans des contextes de recherche et d’analyse de malwares.
