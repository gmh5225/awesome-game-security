Questo progetto è un plugin di x64dbg che aiuta gli analisti a fare il debug di eseguibili protetti da Themida su sistemi Windows a 64 bit. Usa l’hooking in C++ con MinHook e collabora con gli strumenti di debug comuni per neutralizzare i controlli anti-debug, anti-VM e di monitoraggio. Il plugin si concentra su flussi pratici di attach e debug anziché su una completa automazione dell’unpacking. L’uso principale è il reverse engineering di software protetto in contesti di ricerca e analisi di malware.
