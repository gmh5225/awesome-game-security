Dieses Projekt demonstriert eine Methode zum Hijacken von Notify Routines im Kernelmodus mithilfe eines dedizierten Threads. Es handelt sich um einen kompakten Visual Studio C++-Proof of Concept mit Fokus auf Low-Level-Callback-Manipulation. Das Repository stellt ein minimales Treibersample und eine kurze Demonstration bereit, um den Ablauf der Technik zu verstehen. Es richtet sich vor allem an das Studium von Windows-Kernel-Internals sowie an die Forschung zu Anti-Cheat- oder EDR-Evasion.
