Este proyecto demuestra un método de secuestro de rutinas de notificación en modo kernel mediante un enfoque de hilo dedicado. Es una prueba de concepto compacta en Visual Studio C++ centrada en la manipulación de callbacks de bajo nivel. El repositorio ofrece un ejemplo mínimo de controlador y una breve demostración para comprender el flujo de la técnica. Está pensado principalmente para el estudio de los internals del kernel de Windows y la investigación sobre evasión de anti-cheat o EDR.
