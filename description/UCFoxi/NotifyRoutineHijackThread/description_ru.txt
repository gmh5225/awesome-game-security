Этот проект демонстрирует метод перехвата notify-рутин в режиме ядра с использованием выделенного потока. Это компактный proof of concept на Visual Studio C++, сосредоточенный на низкоуровневой манипуляции callback-ами. Репозиторий содержит минимальный образец драйвера и короткую демонстрацию для понимания хода техники. Он в основном предназначен для изучения внутренностей ядра Windows и исследований обхода античитов или EDR.
