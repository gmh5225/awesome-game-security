Questo progetto dimostra un metodo di hijacking delle notify routine in modalità kernel basato su un thread dedicato. È una proof of concept compatta in Visual Studio C++ incentrata sulla manipolazione di callback a basso livello. Il repository fornisce un esempio minimo di driver e una breve dimostrazione per comprendere il flusso della tecnica. È principalmente destinato allo studio degli interni del kernel Windows e alla ricerca sull'evasione di anti-cheat o EDR.
