Este proyecto presenta un método de comunicación entre el kernel y el modo de usuario basado en búferes compartidos y el manejo de IRP activado por FlushFileBuffers. Demuestra el hooking de IRP_MJ_FLUSH_BUFFERS para procesar comandos sin depender de un hilo de trabajo persistente. El código se divide en componentes C++ de kernel y de modo de usuario compilados con Visual Studio. Es útil para investigar patrones de comunicación de controladores más sigilosos en seguridad de videojuegos y experimentos de bypass de anti-cheat.
