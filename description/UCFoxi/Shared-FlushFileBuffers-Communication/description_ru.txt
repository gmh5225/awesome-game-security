Этот проект представляет метод связи между ядром и пользовательским режимом на основе разделяемых буферов и обработки IRP, инициируемой FlushFileBuffers. Он демонстрирует перехват IRP_MJ_FLUSH_BUFFERS для обработки команд без постоянного рабочего потока. Код разделён на компоненты ядра и пользовательского режима на C++, собираемые в Visual Studio. Проект полезен для исследования более скрытных схем взаимодействия с драйверами в области игровой безопасности и экспериментов по обходу античита.
