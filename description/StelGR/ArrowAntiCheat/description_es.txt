Arrow es un plugin anti-trampas de Minecraft basado en paquetes personalizados, escrito en Java para servidores Bukkit/Spigot, con soporte para clientes Java y Bedrock. Usa PacketEvents para escuchar paquetes e incluye suites de comprobaciones de combate (aim assist, autoclicker, kill aura, reach, velocity, hitbox), movimiento (vuelo, velocidad, motion, suelo, movimiento ilegal) y otros cheats (paquetes incorrectos, scaffold, timer, inventario). La detección se basa en análisis estadístico, utilidades de predicción de movimiento y un manejo configurable de infracciones con alertas, modo verbose y registro. Está pensado para operadores de servidores Minecraft que buscan un anti-trampas de código abierto con licencia AGPLv3, especialmente servidores pequeños que no pueden permitirse soluciones comerciales.
