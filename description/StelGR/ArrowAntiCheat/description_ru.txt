Arrow — это плагин античита для Minecraft на основе пользовательских пакетов, написанный на Java для серверов Bukkit/Spigot и поддерживающий клиенты Java и Bedrock. Он использует PacketEvents для прослушивания пакетов и включает наборы проверок боя (aim assist, autoclicker, kill aura, reach, velocity, hitbox), передвижения (полёт, скорость, motion, земля, нелегальное движение) и прочих читов (некорректные пакеты, scaffold, timer, инвентарь). Обнаружение опирается на статистический анализ, утилиты предсказания движения и настраиваемую обработку нарушений с оповещениями, подробным режимом и журналированием. Плагин предназначен для операторов Minecraft-серверов, которым нужен открытый античит с лицензией AGPLv3, особенно небольшим серверам, которым недоступны коммерческие решения.
