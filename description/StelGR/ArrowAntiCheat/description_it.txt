Arrow è un plugin anti-cheat Minecraft basato su pacchetti personalizzati, scritto in Java per server Bukkit/Spigot, con supporto per client Java e Bedrock. Usa PacketEvents per l’ascolto dei pacchetti e include suite di controlli per il combattimento (aim assist, autoclicker, kill aura, reach, velocity, hitbox), il movimento (volo, velocità, motion, ground, movimenti illegali) e altri cheat (pacchetti non validi, scaffold, timer, inventario). Il rilevamento si basa su analisi statistica, utilità di predizione del movimento e gestione configurabile delle violazioni con alert, modalità verbose e logging. È pensato per gli operatori di server Minecraft che cercano una risorsa anti-cheat open source con licenza AGPLv3, soprattutto server più piccoli che non possono permettersi soluzioni commerciali.
