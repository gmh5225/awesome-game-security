Arrow は、Bukkit/Spigot サーバー向けに Java で書かれた、カスタムパケットベースの Minecraft アンチチートプラグインで、Java 版と Bedrock 版の両方のクライアントに対応します。PacketEvents でパケットを監視し、戦闘（エイムアシスト、オートクリッカー、キルオーラ、リーチ、ベロシティ、ヒットボックス）、移動（フライ、スピード、モーション、地面、不正移動）、その他のチート（不正パケット、スキャフォールド、タイマー、インベントリ）向けのチェックスイートを備えます。検出は統計分析、移動予測ユーティリティ、およびアラート・詳細モード・ログ付きの設定可能な違反処理に依拠します。商用ソリューションを導入できない小規模サーバーを中心に、オープンソースの AGPLv3 ライセンスのアンチチートを求める Minecraft サーバー運営者向けです。
