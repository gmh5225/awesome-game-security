Этот проект — PcapPlusPlus, кроссплатформенная библиотека на C++ для разбора и формирования сетевых пакетов. Она предоставляет высокоуровневые API для захвата, разбора и построения пакетов по более чем 100 сетевым протоколам, включая Ethernet, IPv4/IPv6, TCP, UDP, HTTP, DNS, TLS и многие другие. Библиотека оборачивает libpcap/WinPcap/Npcap для захвата в реальном времени и поддерживает чтение и запись файлов PCAP/PCAPNG. Она предназначена для разработчиков сетевой безопасности, исследователей протоколов и всех, кто создаёт инструменты анализа пакетов или приложения мониторинга сети.
