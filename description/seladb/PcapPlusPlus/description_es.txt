Este proyecto es PcapPlusPlus, una biblioteca C++ multiplataforma para el análisis y la creación de paquetes de red. Proporciona APIs de alto nivel para capturar, analizar y construir paquetes en más de 100 protocolos de red, incluidos Ethernet, IPv4/IPv6, TCP, UDP, HTTP, DNS, TLS y muchos más. La biblioteca encapsula libpcap/WinPcap/Npcap para la captura en vivo y admite la lectura y escritura de archivos PCAP/PCAPNG. Está dirigida a desarrolladores de seguridad de red, investigadores de protocolos y a cualquiera que construya herramientas de análisis de paquetes o aplicaciones de monitorización de red.
