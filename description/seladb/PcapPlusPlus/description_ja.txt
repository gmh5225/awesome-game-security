本プロジェクトは PcapPlusPlus であり、マルチプラットフォーム対応の C++ ネットワークパケット解析・構築ライブラリです。Ethernet、IPv4/IPv6、TCP、UDP、HTTP、DNS、TLS など 100 以上のネットワークプロトコルにわたるパケットのキャプチャ、解析、構築向けの高水準 API を提供します。ライブキャプチャには libpcap/WinPcap/Npcap をラップし、PCAP/PCAPNG ファイルの読み書きもサポートします。ネットワークセキュリティ開発者、プロトコル研究者、パケット分析ツールやネットワーク監視アプリケーションを構築する方を対象としています。
