Ce projet est un générateur de SDK C++ pour les jeux Unreal Engine, couvrant les versions UE4 et UE5.
Il fonctionne en injectant une DLL dans le processus cible, en localisant les structures et offsets du moteur, puis en produisant des en-têtes et du code SDK générés pour l’analyse et les outils.
La base de code inclut des remplacements d’offsets configurables, plusieurs options de compilation, et des conseils pour intégrer la sortie générée dans des projets en aval.
Son cas d’usage principal est la rétro-ingénierie Unreal Engine, notamment la recherche en sécurité des jeux, l’analyse de mods et le mapping des structures internes.
