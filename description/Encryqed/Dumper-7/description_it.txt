Questo progetto è un generatore di SDK C++ per giochi Unreal Engine su versioni UE4 e UE5.
Funziona iniettando una DLL nel processo di destinazione, individuando strutture e offset del motore ed emettendo header e codice SDK generati per analisi e tooling.
Il codice include override di offset configurabili, più opzioni di build e indicazioni per integrare l’output generato in progetti a valle.
Il caso d’uso principale è il reverse engineering di Unreal Engine, inclusa la ricerca di sicurezza nei giochi, l’analisi di mod e la mappatura delle strutture interne.
