Этот проект — генератор C++ SDK для игр на Unreal Engine версий UE4 и UE5.
Он работает путём внедрения DLL в целевой процесс, поиска структур и смещений движка и генерации заголовков и SDK-кода для анализа и инструментов.
Кодовая база включает настраиваемые переопределения смещений, несколько вариантов сборки и рекомендации по интеграции сгенерированного вывода в последующие проекты.
Основной сценарий использования — реверс-инжиниринг Unreal Engine, включая исследования безопасности игр, анализ моддинга и отображение внутренних структур.
