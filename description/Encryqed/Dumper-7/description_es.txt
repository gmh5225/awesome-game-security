Este proyecto es un generador de SDK en C++ para juegos de Unreal Engine en versiones UE4 y UE5.
Funciona inyectando una DLL en el proceso objetivo, localizando estructuras y offsets del motor, y emitiendo encabezados y código SDK generados para análisis y herramientas.
La base de código incluye anulaciones de offsets configurables, múltiples opciones de compilación y orientación para integrar la salida generada en proyectos posteriores.
Su caso de uso principal es la ingeniería inversa de Unreal Engine, incluida la investigación de seguridad en juegos, el análisis de mods y el mapeo de estructuras internas.
