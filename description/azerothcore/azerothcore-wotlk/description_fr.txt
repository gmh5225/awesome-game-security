Ce projet est AzerothCore, un émulateur de serveur World of Warcraft open source pour l'expansion Wrath of the Lich King (3.3.5a). La base de code C++ implémente le serveur de jeu complet, incluant la simulation du monde, la gestion des joueurs, l'IA des PNJ, le moteur de scripts, le contenu piloté par base de données et la gestion du protocole réseau compatible avec les clients WoW 3.3.5a. Il propose une architecture modulaire avec prise en charge des scripts Eluna Lua. Il s'adresse aux développeurs de serveurs de jeux, aux opérateurs de serveurs privés WoW et aux chercheurs étudiant l'architecture des serveurs MMO.
