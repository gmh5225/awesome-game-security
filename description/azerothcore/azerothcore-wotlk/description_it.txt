Questo progetto è AzerothCore, un emulatore di server World of Warcraft open source per l'espansione Wrath of the Lich King (3.3.5a). Il codice C++ implementa il server di gioco completo, inclusa la simulazione del mondo, la gestione dei giocatori, l'IA degli NPC, il motore di scripting, i contenuti basati su database e la gestione del protocollo di rete compatibile con i client WoW 3.3.5a. Presenta un'architettura modulare con supporto allo scripting Eluna Lua. È rivolto agli sviluppatori di server di gioco, agli operatori di server privati WoW e ai ricercatori che studiano l'architettura dei server MMO.
