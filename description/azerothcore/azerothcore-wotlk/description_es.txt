Este proyecto es AzerothCore, un emulador de servidor de World of Warcraft de código abierto para la expansión Wrath of the Lich King (3.3.5a). La base de código en C++ implementa el servidor de juego completo, incluyendo simulación del mundo, gestión de jugadores, IA de PNJ, motor de scripting, contenido impulsado por base de datos y manejo del protocolo de red compatible con clientes WoW 3.3.5a. Cuenta con una arquitectura modular con soporte de scripting Eluna Lua. Está dirigido a desarrolladores de servidores de juegos, operadores de servidores privados de WoW e investigadores que estudian la arquitectura de servidores MMO.
