Morphida は、Frida のソースツリーをベンダーせずに、Android arm64 向けの多形的で検知回避しやすい Frida frida-server バイナリを生成する薄い CI ビルドパイプラインです。各ビルドは上流の Frida リリースタグをクローンし、小さなパッチセットを適用し、プロセス名、memfd 名、agent SO プレフィックス、スレッド名、パス文字列などの静的フィンガープリントをランダム化したうえで、NDK の llvm-strip で gum_* および frida_* シンボルを除去します。CI の strings ゲートは、DEX 外に硬い Frida シグネチャが残っている場合にビルドを失敗させるため、同じ Frida バージョンの 2 回のビルドが同一の静的フィンガープリントを共有しません。ヘルパースクリプトは adb 経由の接続、バージョン確認、ポート転送、任意の認証トークンをサポートします。一般的な静的検知や安価なランタイム Frida 検知に対して見つけにくい frida-server を必要とするリバースエンジニアやモバイルセキュリティ研究者向けです。
