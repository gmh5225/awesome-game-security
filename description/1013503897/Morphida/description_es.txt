Morphida es un pipeline de build CI ligero que produce binarios Frida frida-server polimórficos y anti-detección para Android arm64 sin incluir el árbol de código fuente de Frida. Cada build clona una etiqueta de release de Frida upstream, aplica un pequeño conjunto de parches, aleatoriza huellas estáticas como nombres de proceso, nombres memfd, prefijos SO del agent, nombres de hilos y cadenas de rutas, y luego elimina los símbolos gum_* y frida_* con llvm-strip del NDK. Una puerta CI de strings falla el build si quedan firmas Frida duras fuera del DEX, de modo que dos builds de la misma versión de Frida no comparten la misma huella estática. Los scripts auxiliares permiten conectar por adb con comprobaciones de versión, reenvío de puertos y tokens de autenticación opcionales. Está orientado a ingenieros inversos e investigadores de seguridad móvil que necesitan un frida-server más difícil de detectar frente a detecciones Frida estáticas comunes y runtime baratas.
