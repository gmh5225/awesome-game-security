Morphida — это тонкий CI-конвейер сборки, который создаёт полиморфные, anti-detection бинарники Frida frida-server для Android arm64 без включения исходного дерева Frida. Каждая сборка клонирует тег релиза upstream Frida, применяет небольшой набор патчей, рандомизирует статические отпечатки — имена процессов, имена memfd, префиксы agent SO, имена потоков и строковые пути — затем удаляет символы gum_* и frida_* с помощью NDK llvm-strip. CI-гейт по strings проваливает сборку, если жёсткие сигнатуры Frida остаются вне DEX, поэтому две сборки одной версии Frida не разделяют один и тот же статический отпечаток. Вспомогательные скрипты поддерживают подключение через adb с проверкой версии, пробросом портов и опциональными токенами аутентификации. Предназначен для reverse-инженеров и исследователей мобильной безопасности, которым нужен труднее обнаруживаемый frida-server против распространённых статических и дешёвых runtime-детекций Frida.
