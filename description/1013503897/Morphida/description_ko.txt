Morphida는 Frida 소스 트리를 벤더링하지 않고 Android arm64용 다형적·탐지 회피형 Frida frida-server 바이너리를 생성하는 가벼운 CI 빌드 파이프라인입니다. 각 빌드는 업스트림 Frida 릴리스 태그를 클론하고, 작은 패치 세트를 적용한 뒤 프로세스 이름, memfd 이름, agent SO 접두사, 스레드 이름, 경로 문자열 같은 정적 지문을 무작위화하고, NDK llvm-strip으로 gum_* 및 frida_* 심볼을 제거합니다. CI strings 게이트는 DEX 외부에 하드코딩된 Frida 시그니처가 남아 있으면 빌드를 실패시키므로, 동일 Frida 버전의 두 빌드가 같은 정적 지문을 공유하지 않습니다. 헬퍼 스크립트는 adb 연결, 버전 검사, 포트 포워딩, 선택적 인증 토큰을 지원합니다. 일반적인 정적 탐지와 저렴한 런타임 Frida 탐지에 덜 드러나는 frida-server가 필요한 리버스 엔지니어와 모바일 보안 연구자를 대상으로 합니다.
