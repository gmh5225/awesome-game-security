Morphida è una pipeline di build CI leggera che produce binari Frida frida-server polimorfi e anti-detection per Android arm64 senza includere l’albero sorgente di Frida. Ogni build clona un tag di release Frida upstream, applica un piccolo set di patch, randomizza fingerprint statici come nomi di processo, nomi memfd, prefissi SO dell’agent, nomi di thread e stringhe di percorso, quindi rimuove i simboli gum_* e frida_* con llvm-strip del NDK. Un gate CI sulle stringhe fa fallire la build se rimangono firme Frida dure fuori dal DEX, così due build della stessa versione Frida non condividono lo stesso fingerprint statico. Script di supporto consentono la connessione via adb con controlli di versione, port forwarding e token di autenticazione opzionali. È pensato per reverse engineer e ricercatori di sicurezza mobile che necessitano di un frida-server più difficile da rilevare contro le comuni detection Frida statiche e runtime economiche.
