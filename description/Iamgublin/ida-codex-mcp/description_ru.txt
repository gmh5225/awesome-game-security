Этот проект подключает IDA Pro к MCP-клиентам через локальный мост и сервер. Он предоставляет возможности реверс-инжиниринга: списки функций, графы вызовов, получение псевдокода и дизассемблирования, перекрёстные ссылки, строки, чтение памяти, а также помощники для переименования и типизации. Решение написано на Python и сочетает плагин IDA, обслуживающий TCP JSON-запросы, с MCP stdio-сервером, который повторно экспортирует инструменты и ресурсы. Предназначено для аналитиков, которые хотят автоматизировать рабочие процессы реверс-инжиниринга с помощью ИИ-инструментов.
