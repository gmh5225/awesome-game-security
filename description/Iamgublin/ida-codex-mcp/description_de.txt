Dieses Projekt verbindet IDA Pro über eine lokale Bridge und einen Server mit MCP-Clients. Es stellt Reverse-Engineering-Funktionen bereit, etwa Funktionslisten, Call-Graphen, Abruf von Pseudocode und Disassembly, Xrefs, Strings, Speicherzugriffe sowie Hilfen zum Umbenennen oder Typisieren. Die Lösung ist in Python geschrieben und kombiniert ein IDA-Plugin, das TCP-JSON-Anfragen bedient, mit einem MCP-stdio-Server, der Tools und Ressourcen erneut exportiert. Zielgruppe sind Analysten, die Reverse-Engineering-Workflows aus KI-gestützten Werkzeugen heraus automatisieren möchten.
