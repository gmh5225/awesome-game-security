Este proyecto conecta IDA Pro con clientes MCP mediante un puente y un servidor locales. Expone capacidades de ingeniería inversa como listado de funciones, grafos de llamadas, obtención de pseudocódigo y desensamblado, xrefs, cadenas, lecturas de memoria y ayudas para renombrar o tipar. La solución está escrita en Python y combina un plugin de IDA que atiende peticiones TCP JSON con un servidor MCP stdio que reexporta herramientas y recursos. Está pensada para analistas que quieren automatizar flujos de ingeniería inversa desde herramientas asistidas por IA.
