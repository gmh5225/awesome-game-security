Questo progetto collega IDA Pro ai client MCP tramite un bridge e un server locali. Espone capacità di reverse engineering come elenco delle funzioni, grafi delle chiamate, recupero di pseudocodice e disassemblato, xref, stringhe, letture di memoria e ausili per ridenominazione o tipizzazione. La soluzione è scritta in Python e combina un plugin IDA che serve richieste TCP JSON con un server MCP stdio che riesporta strumenti e risorse. Si rivolge agli analisti che vogliono automatizzare i flussi di reverse engineering da strumenti assistiti dall'IA.
