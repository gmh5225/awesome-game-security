aShell You 是一款功能完善的 Android shell 工具应用，可在本机或通过 OTG、无线调试在其他设备上运行 ADB、root 与 shell 命令，无需电脑。主要使用 Kotlin 编写，采用 Jetpack Compose 与 Material Design 3，内置 ADB 库并可选附带原生组件。核心能力包括通过 Shizuku 或 root 本地执行、远程控制其他 Android 设备、内置 ADB 命令示例、ADB push/pull 文件管理器、对 logcat 等持续命令的支持、输出搜索与保存/分享，以及带本地备份与恢复的命令书签。面向需要便捷机上 ADB 与 shell 调试、设备管理工作流的 Android 开发者、高阶用户与折腾爱好者。
