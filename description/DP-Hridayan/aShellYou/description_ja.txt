aShell You は、PC を使わずに自端末や OTG・ワイヤレスデバッグ経由の他端末で ADB、root、shell コマンドを実行できる、機能充実の Android shell ユーティリティアプリです。主に Kotlin で書かれ、Jetpack Compose と Material Design 3 を採用し、組み込み ADB ライブラリと任意のネイティブコンポーネントを含みます。主な機能は Shizuku または root によるローカル実行、他の Android 端末のリモート操作、組み込み ADB コマンド例、ADB push/pull ファイルエクスプローラー、logcat などの継続コマンド対応、出力の検索と保存/共有、ローカルバックアップ・復元付きコマンドブックマークです。デバッグと端末管理のため、端末上で手軽な ADB・shell ワークフローを求める Android 開発者、パワーユーザー、いじり好き向けです。
