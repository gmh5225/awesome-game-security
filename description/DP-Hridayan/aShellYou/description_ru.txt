aShell You — полнофункциональное Android-приложение-утилита для shell, позволяющее выполнять команды ADB, root и shell на своём устройстве или на других через OTG либо беспроводную отладку без ПК. Написано в основном на Kotlin с Jetpack Compose и Material Design 3, включает встроенную библиотеку ADB и опциональные нативные компоненты. Ключевые возможности: локальный запуск через Shizuku или root, удалённое управление другими Android-устройствами, встроенные примеры ADB-команд, файловый проводник ADB push/pull, поддержка непрерывных команд вроде logcat, поиск и сохранение/отправка вывода, а также закладки команд с локальным резервным копированием и восстановлением. Предназначено для Android-разработчиков, опытных пользователей и энтузиастов, которым нужен удобный рабочий процесс ADB и shell на устройстве для отладки и управления устройствами.
