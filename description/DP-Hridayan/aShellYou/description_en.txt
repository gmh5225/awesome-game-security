aShell You is a fully featured Android shell utility app that lets you run ADB, root, and shell commands on your own device or on other devices over OTG or wireless debugging, without needing a PC. It is written primarily in Kotlin with Jetpack Compose and Material Design 3, and includes an embedded ADB library plus optional native components. Key capabilities include local execution via Shizuku or root, remote control of other Android devices, built-in ADB command examples, an ADB push/pull file explorer, continuous command support such as logcat, output search and save/share, and command bookmarks with local backup and restore. It targets Android developers, power users, and tinkerers who need a convenient on-device ADB and shell workflow for debugging and device management.
