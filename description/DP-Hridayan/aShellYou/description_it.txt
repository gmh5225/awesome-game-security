aShell You è un’app di utilità shell per Android completa che consente di eseguire comandi ADB, root e shell sul proprio dispositivo o su altri dispositivi tramite OTG o wireless debugging, senza bisogno di un PC. È scritta principalmente in Kotlin con Jetpack Compose e Material Design 3, e include una libreria ADB integrata più componenti nativi opzionali. Le funzionalità chiave comprendono l’esecuzione locale tramite Shizuku o root, il controllo remoto di altri dispositivi Android, esempi di comandi ADB incorporati, un esploratore file ADB push/pull, il supporto a comandi continui come logcat, ricerca e salvataggio/condivisione dell’output, e segnalibri dei comandi con backup e ripristino locali. È rivolta a sviluppatori Android, power user e appassionati che necessitano di un flusso ADB e shell comodo sul dispositivo per debug e gestione dei dispositivi.
