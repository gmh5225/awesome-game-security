Questo progetto è DOOM Retro, un source port classico di DOOM per Windows scritto in C con SDL2. Migliora il motore DOOM originale con rendering widescreen, framerate senza limite, illuminazione migliorata, filtraggio delle texture, supporto per gamepad e numerose correzioni di bug, preservando la sensazione di gioco autentica. Il port supporta tutti i formati WAD originali, le patch DeHackEd e le funzionalità delle mappe compatibili con BOOM. È rivolto agli appassionati di FPS classici, ai modder di DOOM e ai ricercatori di game engine che studiano la modernizzazione del motore idTech 1.
