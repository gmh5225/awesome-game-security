Este proyecto es DOOM Retro, un source port clásico de DOOM para Windows escrito en C con SDL2. Mejora el motor DOOM original con renderizado panorámico, tasa de fotogramas sin límite, iluminación mejorada, filtrado de texturas, soporte de mando y numerosas correcciones de errores, preservando la sensación de juego auténtica. El port admite todos los formatos WAD originales, parches DeHackEd y funciones de mapas compatibles con BOOM. Está pensado para aficionados a los FPS clásicos, modders de DOOM e investigadores de motores de juego que estudian la modernización del motor idTech 1.
