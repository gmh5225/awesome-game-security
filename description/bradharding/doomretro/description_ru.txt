Этот проект — DOOM Retro, классический source port DOOM для Windows, написанный на C с использованием SDL2. Он расширяет оригинальный движок DOOM широкоэкранным рендерингом, неограниченным framerate, улучшенным освещением, фильтрацией текстур, поддержкой геймпада и многочисленными исправлениями ошибок, сохраняя аутентичное ощущение геймплея. Порт поддерживает все оригинальные форматы WAD, патчи DeHackEd и функции карт, совместимые с BOOM. Он предназначен для любителей классических FPS, моддеров DOOM и исследователей игровых движков, изучающих модернизацию движка idTech 1.
