Este proyecto es una guía de despliegue en Linux y un conjunto de scripts para ejecutar un servidor dedicado de juego multijugador. Se basa en scripts shell, SteamCMD y archivos de servicio systemd para instalar, actualizar, iniciar, respaldar y restaurar los datos del servidor. El flujo de trabajo está orientado a Debian y Ubuntu e incluye pasos operativos para automatizar el mantenimiento. Es útil principalmente para administradores de servidores, más que para investigación de anti-cheat o ingeniería inversa.
