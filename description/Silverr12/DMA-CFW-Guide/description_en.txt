This project is a step-by-step guide for building custom DMA attack firmware based on pcileech-fpga version 4.15 for FPGA DMA cards such as Squirrel, EnigmaX1, and ZDMA. It walks through collecting donor PCIe device identity and config-space data with tools like Arbor or Telescan PE, then customizing SystemVerilog sources and Xilinx Vivado IP settings including device IDs, DSN, BARs, and related capabilities. The guide also covers TLP emulation, interrupt-related notes, bitstream build and flash workflows, and an alternate shadow configuration-space path using .coe files and writemasks. It is aimed at researchers and practitioners working on PCIe DMA firmware, anti-cheat evasion threat modeling, and low-level game-security hardware research.
