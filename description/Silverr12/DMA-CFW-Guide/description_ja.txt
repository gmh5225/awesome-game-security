本プロジェクトは、Squirrel、EnigmaX1、ZDMA などの FPGA DMA カード向けに、pcileech-fpga バージョン 4.15 を基にしたカスタム DMA 攻撃ファームウェアを構築するための手順ガイドです。Arbor や Telescan PE などのツールでドナー PCIe デバイスの識別情報とコンフィグレーション空間データを収集し、SystemVerilog ソースおよび Xilinx Vivado IP 設定（デバイス ID、DSN、BAR、関連 capability など）をカスタマイズする流れを解説します。TLP エミュレーション、割り込み関連の注意点、ビットストリームのビルドとフラッシュ手順、さらに .coe ファイルと writemask を用いた代替のシャドウ・コンフィグレーション空間パスも扱います。PCIe DMA ファームウェア、アンチチート回避の脅威モデリング、低レベルなゲームセキュリティ向けハードウェア研究に取り組む研究者・実務者を対象としています。
