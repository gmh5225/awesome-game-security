本專案是一份循序漸進的指南，用於基於 pcileech-fpga 4.15 版為 Squirrel、EnigmaX1、ZDMA 等 FPGA DMA 卡建置自訂 DMA 攻擊韌體。內容涵蓋如何以 Arbor 或 Telescan PE 等工具蒐集供體 PCIe 裝置身分與組態空間資料，再自訂 SystemVerilog 原始碼以及 Xilinx Vivado IP 設定（包括裝置 ID、DSN、BAR 及相關能力）。指南亦介紹 TLP 模擬、中斷相關說明、位元串流建置與燒錄流程，以及使用 .coe 檔案與 writemask 的替代影子組態空間路徑。面向從事 PCIe DMA 韌體、反作弊規避威脅建模以及底層遊戲安全硬體研究的研究人員與實務工作者。
