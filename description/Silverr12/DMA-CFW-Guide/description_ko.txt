이 프로젝트는 Squirrel, EnigmaX1, ZDMA와 같은 FPGA DMA 카드용으로 pcileech-fpga 버전 4.15를 기반으로 커스텀 DMA 공격 펌웨어를 구축하는 단계별 가이드입니다. Arbor 또는 Telescan PE 같은 도구로 도너 PCIe 장치 ID와 구성 공간 데이터를 수집한 뒤, SystemVerilog 소스와 Xilinx Vivado IP 설정(장치 ID, DSN, BAR 및 관련 capability 포함)을 맞춤 설정하는 과정을 안내합니다. TLP 에뮬레이션, 인터럽트 관련 참고 사항, 비트스트림 빌드 및 플래시 워크플로, .coe 파일과 writemask를 사용하는 대체 섀도 구성 공간 경로도 다룹니다. PCIe DMA 펌웨어, 안티치트 우회 위협 모델링, 저수준 게임 보안 하드웨어 연구를 수행하는 연구자와 실무자를 대상으로 합니다.
