자동 갱신되는 CS:GO 오프셋 및 netvar 저장소로, engine.dll과 client.dll의 메모리 시그니처와 해석된 오프셋을 여러 출력 형식(JSON, TOML, YAML, C++ 헤더, C#, VB)으로 게시합니다. 함께 제공되는 config.json은 dwClientState, dwEntityList, dwLocalPlayer 같은 구조체와 m_iHealth, m_vecOrigin 등의 netvar 오프셋에 대해 상대/절대 주소 지정 모드를 포함한 바이트 패턴 시그니처를 정의하여, 외부 메모리 읽기 치트가 게임 패치 후에도 최신 상태를 유지할 수 있게 합니다.
