Este proyecto es un framework y plantilla de controlador de núcleo de Windows para programación de sistemas de bajo nivel e investigación.
Está escrito principalmente en C++ moderno e incluye componentes asistidos por hipervisor para entornos Intel VT-x y AMD-V.
El framework expone abstracciones para operaciones de memoria, manejo de IOCTL, hooks, interacciones con CPUID y MSR, y otras primitivas del núcleo.
Se usa habitualmente en depuración avanzada, monitorización e investigación de seguridad del núcleo relacionada con anti-cheat.
