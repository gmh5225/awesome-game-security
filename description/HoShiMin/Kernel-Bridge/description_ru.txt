Этот проект — фреймворк и шаблон драйвера ядра Windows для низкоуровневого системного программирования и исследований.
Он в основном написан на современном C++ и включает компоненты с поддержкой гипервизора для сред Intel VT-x и AMD-V.
Фреймворк предоставляет абстракции для операций с памятью, обработки IOCTL, хуков, взаимодействий с CPUID и MSR, а также других примитивов ядра.
Его часто используют в продвинутой отладке, мониторинге и исследованиях безопасности ядра, связанных с античитом.
