Этот проект — инструмент антидетект-усиления для VMware, который патчит виртуальные машины VMware, чтобы обходить методы обнаружения VM, применяемые вредоносным ПО и античитами. Он изменяет листья CPUID, таблицы SMBIOS, данные ACPI, ключи реестра, MAC-адреса и другие аппаратные отпечатки, чтобы VM выглядела как физическая машина. Загрузчик на C/C++ работает на уровне гипервизора и поддерживает гостевые системы Windows и Linux. Предназначен для аналитиков вредоносного ПО и исследователей игровой безопасности, которым нужно скрывать виртуализацию от ПО, распознающего VM.
