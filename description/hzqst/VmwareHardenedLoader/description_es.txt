Este proyecto es una herramienta de endurecimiento antidetección para VMware que parchea máquinas virtuales VMware para evadir técnicas de detección de VM usadas por malware y sistemas anticheat. Modifica hojas CPUID, tablas SMBIOS, datos ACPI, claves del registro, direcciones MAC y otras huellas de hardware para que la VM parezca una máquina física. El cargador en C/C++ opera a nivel de hipervisor e incluye soporte para invitados Windows y Linux. Está orientado a analistas de malware e investigadores de seguridad en juegos que necesitan ocultar la virtualización ante software consciente de las VM.
