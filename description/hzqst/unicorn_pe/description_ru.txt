Этот проект посвящён Unicorn PE. Unicorn PE — проект/фреймворк инструментации на базе Unicorn, предназначенный для эмуляции выполнения кода Windows PE-файлов, особенно упакованных.
Показывает дизассемблирование всех выполняемых инструкций.
В основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / динамической бинарной инструментации.
