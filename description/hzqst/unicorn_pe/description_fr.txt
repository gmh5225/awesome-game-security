Ce projet porte sur Unicorn PE. Unicorn PE est un projet/framework d'instrumentation basé sur Unicorn, conçu pour émuler l'exécution de code des fichiers PE Windows, en particulier ceux qui sont packés.
Il affiche le désassemblage de toutes les instructions en cours d'exécution.
Il est principalement utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / de l'instrumentation binaire dynamique.
