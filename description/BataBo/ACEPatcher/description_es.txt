Este proyecto es una herramienta de parcheo con interfaz gráfica para modificar ensamblados .NET. Está escrita en C# y usa bibliotecas como dnlib y Harmony para mapear métodos de parche a métodos objetivo en muchos tipos de métodos. La aplicación admite importar/exportar conjuntos de parches, protección opcional con contraseña para archivos de parche y flujos de trabajo para binarios administrados empaquetados u ofuscados. El caso de uso principal es la investigación de ingeniería inversa y la automatización repetible de parches .NET.
