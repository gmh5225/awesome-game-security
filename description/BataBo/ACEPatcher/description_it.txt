Questo progetto è uno strumento di patching basato su GUI per modificare assembly .NET. È scritto in C# e usa librerie come dnlib e Harmony per mappare i metodi di patch ai metodi di destinazione su molti tipi di metodi. L'applicazione supporta l'importazione/esportazione di set di patch, la protezione opzionale con password per i file di patch e flussi di lavoro per binari gestiti packed o offuscati. Il caso d'uso principale è la ricerca di reverse engineering e l'automazione ripetibile di patch .NET.
