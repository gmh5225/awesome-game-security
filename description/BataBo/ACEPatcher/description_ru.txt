Этот проект — инструмент патчинга с графическим интерфейсом для изменения .NET-сборок. Он написан на C# и использует библиотеки вроде dnlib и Harmony для сопоставления методов патча с целевыми методами многих типов. Приложение поддерживает импорт/экспорт наборов патчей, опциональную защиту паролем файлов патчей и рабочие процессы для упакованных или обфусцированных управляемых двоичных файлов. Основной сценарий использования — исследования в области реверс-инжиниринга и воспроизводимая автоматизация патчей .NET.
