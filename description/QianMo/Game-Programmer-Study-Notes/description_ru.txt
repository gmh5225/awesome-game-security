Этот проект — обширная коллекция учебных заметок по игровому программированию с упором на графику и рендеринг в реальном времени.
Большая часть материалов написана в Markdown и сопровождается многочисленными схемами и медиа; рассматриваются такие темы, как GPU Gems, программирование шейдеров, трассировка лучей и постобработка.
Также включены материалы по программной инженерии и проектированию, актуальные для практической разработки игр.
Основная аудитория — учащиеся и разработчики, которым нужны структурированные справочники по концепциям игровой графики и движков.
