Este proyecto es una biblioteca de patrones de diseño en Unity C# que demuestra patrones orientados a objetos clásicos y específicos de juegos en escenas ejecutables.
Contiene implementaciones de los 23 patrones GoF más patrones de programación de juegos seleccionados, cada uno con ejemplos de estructura y casos de uso prácticos.
El repositorio está organizado por categoría de patrón e incluye diagramas, scripts de ejemplo y escenas de Unity para un aprendizaje práctico.
Está dirigido a desarrolladores de Unity que quieran aplicar patrones de arquitectura mantenibles en código de juego real.
