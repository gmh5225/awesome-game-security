Questo progetto è una libreria di design pattern in Unity C# che illustra i pattern orientati agli oggetti classici e quelli specifici per i giochi in scene eseguibili.
Contiene le implementazioni di tutti i 23 pattern GoF più pattern di game programming selezionati, ciascuno con esempi di struttura e casi d'uso pratici.
Il repository è organizzato per categoria di pattern e include diagrammi, script di esempio e scene Unity per un apprendimento pratico.
È rivolto agli sviluppatori Unity che vogliono applicare pattern architetturali manutenibili nel codice di gioco reale.
