Этот проект — пример игры на Unreal Engine 4, демонстрирующий C++-модуль геймплея со стандартным управлением персонажем.
Он сочетает классы Unreal C++ с ассетами Blueprint, привязками ввода, настройкой камеры и картами стартового контента.
Код следует типичной структуре модулей и target UE4 и служит компактным примером организации проекта движка.
Он в основном предназначен для начинающих, практикующих геймплейное программирование на UE4 C++ и основы рабочего процесса редактора.
