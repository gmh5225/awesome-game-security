Este proyecto es un juego de ejemplo de Unreal Engine 4 que demuestra un módulo de gameplay en C++ con control de personaje estándar.
Combina clases de Unreal C++ con assets de Blueprint, enlaces de entrada, configuración de cámara y mapas de contenido inicial.
El código sigue el diseño típico de módulos y targets de UE4 y sirve como ejemplo compacto de la estructura de un proyecto del motor.
Está pensado principalmente para principiantes que practican la programación de gameplay en UE4 con C++ y los fundamentos del flujo de trabajo del editor.
