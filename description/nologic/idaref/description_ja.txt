IdaRef は、カーソル下の命令について完全な CPU 命令ドキュメントを表示する IDA Pro プラグインで、自動コメントに似ていますが、より詳細なリファレンス本文を提供します。主に Python で書かれており、x86-64、ARM、MIPS 32 ビット、Xtensa 向けのニーモニックと説明を収録したアーキテクチャ別 SQLite データベースを読み込みます。ナビゲートに合わせて自動更新したり、現在の命令の表示を更新したり、手動で命令を検索したりでき、共有ドキュメント向けの簡単な説明リダイレクトにも対応します。IDA でバイナリを解析しながら、すばやくオフラインのアセンブリリファレンスを得たいリバースエンジニア向けです。
