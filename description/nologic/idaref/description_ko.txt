IdaRef는 커서 아래 명령에 대한 전체 CPU 명령 문서를 보여주는 IDA Pro 플러그인으로, 자동 주석과 비슷하지만 완전한 레퍼런스 텍스트를 제공합니다. 주로 Python으로 작성되었으며, x86-64, ARM, MIPS 32비트, Xtensa용 니모닉과 설명이 담긴 아키텍처별 SQLite 데이터베이스를 로드합니다. 탐색할 때 자동 새로고침하거나 현재 명령에 맞게 뷰를 갱신하거나 명령을 수동으로 조회할 수 있으며, 공유 문서를 위한 간단한 설명 리다이렉트도 지원합니다. IDA에서 바이너리를 분석하면서 빠른 오프라인 어셈블리 참고 자료를 원하는 리버싱 엔지니어를 위한 도구입니다.
