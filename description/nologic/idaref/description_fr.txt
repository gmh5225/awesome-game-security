IdaRef est un plugin IDA Pro qui affiche la documentation complète des instructions CPU pour l'instruction sous le curseur, à la manière des commentaires automatiques mais avec le texte de référence intégral. Écrit principalement en Python, il charge des bases SQLite spécifiques à chaque architecture, contenant les mnémoniques et descriptions pour x86-64, ARM, MIPS 32 bits et Xtensa. Le plugin peut se rafraîchir automatiquement lors de la navigation, mettre à jour la vue pour l'instruction courante ou rechercher une instruction manuellement, et il prend en charge de simples redirections de description pour la documentation partagée. Il s'adresse aux ingénieurs en rétro-ingénierie qui veulent une référence assembleur hors ligne rapide pendant l'analyse de binaires dans IDA.
