IdaRef es un plugin de IDA Pro que muestra la documentación completa de instrucciones de CPU para la instrucción bajo el cursor, similar a los comentarios automáticos pero con el texto de referencia completo. Escrito principalmente en Python, carga bases de datos SQLite específicas de cada arquitectura con mnemónicos y descripciones para x86-64, ARM, MIPS de 32 bits y Xtensa. El plugin puede actualizarse automáticamente al navegar, actualizar la vista de la instrucción actual o consultar una instrucción de forma manual, y admite redirecciones sencillas de descripción para documentación compartida. Está pensado para ingenieros de ingeniería inversa que quieren material de referencia de ensamblador rápido y sin conexión mientras analizan binarios en IDA.
