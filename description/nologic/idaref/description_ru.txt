IdaRef — плагин для IDA Pro, который показывает полную документацию по CPU-инструкции под курсором: похож на автокомментарии, но с полным справочным текстом. Написан в основном на Python и загружает архитектуро-специфичные базы SQLite с мнемониками и описаниями для x86-64, ARM, MIPS 32-bit и Xtensa. Плагин может автоматически обновляться при навигации, обновлять вид для текущей инструкции или искать инструкцию вручную, а также поддерживает простые перенаправления описаний для общей документации. Он предназначен для реверс-инженеров, которым нужен быстрый офлайн-справочник по ассемблеру при анализе бинарников в IDA.
