Этот проект ориентирован на быстрое создание и запуск оптимизированных виртуальных машин QEMU для Windows/macOS/Linux. Изначальная цель — быстро тестировать дистрибутивы Linux: виртуальные машины и их конфигурацию можно хранить где угодно (например, на внешнем USB-накопителе или в домашнем каталоге), а для запуска не требуются повышенные права. Он особенно полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / qemu/kvm/pve/vbox.
