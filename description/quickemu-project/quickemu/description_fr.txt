Ce projet se concentre sur la création et l'exécution rapides de machines virtuelles QEMU optimisées pour Windows/macOS/Linux. L'objectif initial était de permettre de tester rapidement des distributions Linux : les machines virtuelles et leur configuration peuvent être stockées n'importe où (par exemple sur un stockage USB externe ou dans le répertoire personnel), sans permissions élevées pour les exécuter. Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient des techniques offensives dans le domaine cheat / qemu/kvm/pve/vbox.
