Este proyecto es un complemento de IDA Pro que importa símbolos del kernel de Linux desde un volcado de kallsyms.
Está implementado en Python y asigna nombres de símbolos y direcciones a la base de datos del desensamblador para facilitar la navegación.
Al restaurar el contexto de los símbolos, mejora la velocidad del análisis estático al trabajar con imágenes del kernel sin símbolos o solo parcialmente simbolizadas.
Está pensado principalmente para ingenieros de ingeniería inversa del kernel e investigadores de vulnerabilidades.
