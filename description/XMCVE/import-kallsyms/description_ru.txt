Этот проект — плагин для IDA Pro, который импортирует символы ядра Linux из дампа kallsyms.
Он реализован на Python и сопоставляет имена символов и адреса с базой данных дизассемблера для удобной навигации.
Восстанавливая контекст символов, он ускоряет статический анализ при работе с stripped или частично символизированными образами ядра.
Предназначен в первую очередь для reverse-инженеров ядра и исследователей уязвимостей.
