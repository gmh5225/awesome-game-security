Questo progetto è un plugin di IDA Pro che importa i simboli del kernel Linux da un dump kallsyms.
È implementato in Python e mappa nomi di simboli e indirizzi nel database del disassemblatore per facilitare la navigazione.
Ripristinando il contesto dei simboli, migliora la velocità dell'analisi statica quando si lavora con immagini del kernel stripate o solo parzialmente simbolizzate.
È destinato principalmente a reverse engineer del kernel e ricercatori di vulnerabilità.
