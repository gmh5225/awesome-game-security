Этот проект — командный инжектор для запуска UWP-приложений и внедрения DLL на очень ранней стадии запуска. Он реализован на C с использованием Win32 и AppModel API и применяет похожий на отладчик сценарий приостановленного запуска для раннего получения контроля над процессом. В репозитории есть скрипты сборки и окружения для Windows-инструментария, а также простой workflow подключения DLL. Основное назначение — реверс-инжиниринг UWP, инструментация во время выполнения и поддержка отладки.
