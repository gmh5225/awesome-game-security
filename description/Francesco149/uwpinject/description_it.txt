Questo progetto è un injector a riga di comando per avviare applicazioni UWP e iniettare DLL in una fase molto precoce dell’avvio. È implementato in C con le API Win32 e AppModel e usa un flusso di avvio sospeso simile a un debugger per ottenere il controllo anticipato del processo. Il repository include script di build e di ambiente per toolchain Windows e un flusso di lavoro semplice di drop-in delle DLL. Il caso d’uso principale è il reverse engineering di UWP, l’instrumentazione a runtime e il supporto al debugging.
