Este proyecto es un inyector de línea de comandos para lanzar aplicaciones UWP e inyectar DLLs en una etapa muy temprana del arranque. Está implementado en C con las API Win32 y AppModel, y usa un flujo de lanzamiento suspendido similar a un depurador para obtener control temprano del proceso. El repositorio incluye scripts de compilación y entorno para toolchains de Windows, así como un flujo de trabajo sencillo de inserción de DLL. Su caso de uso principal es la ingeniería inversa de UWP, la instrumentación en tiempo de ejecución y el soporte de depuración.
