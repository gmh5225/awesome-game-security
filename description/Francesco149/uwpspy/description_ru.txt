Этот проект — DLL-инструментарий для слежения и хукинга UWP, предназначенный для отладки и реверс-инжиниринга приложений Windows Runtime. Он написан на C++ и перехватывает выбранные интерфейсы UWP и WinRT, логируя поведение во время выполнения в консоль. Код предоставляет переиспользуемый низкоуровневый каркас хуков и шаблоны перехвата API, подходящие для задач инструментирования. Основная аудитория — исследователи, изучающие внутренности UWP и поведение приложений во время выполнения.
