Este proyecto es un kit de herramientas DLL de espionaje y hooking UWP para depurar y realizar ingeniería inversa de aplicaciones Windows Runtime. Está escrito en C++ y engancha interfaces UWP y WinRT seleccionadas mientras registra el comportamiento en tiempo de ejecución en una consola. El código ofrece un andamiaje de hooks de bajo nivel reutilizable y patrones de interceptación de API adecuados para tareas de instrumentación. Su público principal son investigadores que estudian los internos de UWP y el comportamiento de las aplicaciones durante la ejecución.
