Questo progetto è un toolkit DLL per lo spionaggio e l'hooking di applicazioni UWP, pensato per il debug e il reverse engineering di applicazioni Windows Runtime. È scritto in C++ e intercetta interfacce UWP e WinRT selezionate, registrando il comportamento a runtime su una console. Il codice fornisce uno scaffolding di hook di basso livello riusabile e pattern di intercettazione delle API adatti all'instrumentazione. Il pubblico principale sono ricercatori che studiano gli interni UWP e il comportamento delle applicazioni in esecuzione.
