Dieses Projekt ist ein DLL-basiertes UWP-Spionage- und Hook-Toolkit zum Debuggen und Reverse Engineering von Windows-Runtime-Anwendungen. Es ist in C++ geschrieben und hookt ausgewählte UWP- und WinRT-Schnittstellen, während Laufzeitverhalten in einer Konsole protokolliert wird. Der Code bietet wiederverwendbares Low-Level-Hook-Scaffolding und API-Interzeptionsmuster für Instrumentierungsaufgaben. Die Hauptzielgruppe sind Forscher, die UWP-Internals und das Verhalten von Anwendungen zur Laufzeit untersuchen.
