이 프로젝트는 NMMP(Nativ Method Map Protector)로, Java/Kotlin 바이트코드 메서드를 네이티브 구현으로 변환하는 Android 네이티브 코드 보호 도구입니다. DEX 메서드 바이트코드를 추출한 뒤 인터프리터 또는 JIT와 유사한 변환을 통해 네이티브 ARM/x86 코드로 컴파일하고, 원래 메서드를 해당 네이티브를 호출하는 JNI 브리지로 대체합니다. 이를 통해 일반적인 DEX 디컴파일로 보호된 로직을 복구하기 어렵게 만듭니다. 바이트코드-네이티브 변환을 코드 보호 기법으로 연구하는 Android 개발자와 보안 연구자를 대상으로 합니다.
