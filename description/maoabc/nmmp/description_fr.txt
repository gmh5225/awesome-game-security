Ce projet est NMMP (Nativ Method Map Protector), un outil de protection de code natif Android qui convertit les méthodes en bytecode Java/Kotlin en implémentations natives. Il extrait les bytecodes des méthodes DEX, les compile en code natif ARM/x86 via un interpréteur ou une transformation de type JIT, puis remplace les méthodes d'origine par des ponts JNI qui appellent les équivalents natifs. Cela empêche la décompilation DEX classique de récupérer la logique protégée. Il s'adresse aux développeurs Android et aux chercheurs en sécurité qui étudient la conversion bytecode-vers-natif comme technique de protection du code.
