Une implémentation minimale d'hyperviseur et de noyau invité basés sur KVM, qui montre comment utiliser les API Linux KVM pour créer une machine virtuelle avec des hyperappels personnalisés, la gestion de la mémoire, le traitement des appels système et le chargement ELF.
Le composant hyperviseur agit comme un VMM de type QEMU, tandis que le noyau implémente des sous-systèmes de base incluant mmap, l'exécution de processus et les E/S fichiers via le transfert d'hyperappels.
Il est surtout utile aux chercheurs en sécurité et aux développeurs noyau qui apprennent les internes de KVM et la virtualisation basée sur hyperviseur depuis zéro.
