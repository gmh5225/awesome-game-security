Минимальная реализация гипервизора и гостевого ядра на базе KVM, демонстрирующая использование Linux KVM API для создания виртуальной машины с пользовательскими гипервызовами, управлением памятью, обработкой системных вызовов и загрузкой ELF.
Компонент гипервизора работает как VMM, подобный QEMU, а ядро реализует базовые подсистемы, включая mmap, выполнение процессов и файловый ввод-вывод через пересылку гипервызовов.
В основном полезен исследователям безопасности и разработчикам ядра, изучающим внутреннее устройство KVM и виртуализацию на базе гипервизора с нуля.
