Этот проект — инструмент для Windows, предназначенный для подмены аппаратных идентификаторов NVIDIA GPU, используемых для HWID-банов (Hardware ID) в играх. Он изменяет серийные номера GPU, ID устройств и другие идентификаторы, сообщаемые через интерфейс драйвера NVIDIA, чтобы обойти аппаратные баны. Инструмент патчит запросы на уровне драйвера и записи реестра, которые античиты используют для аппаратного фингерпринтинга. Он рассчитан на исследователей игровой безопасности, изучающих техники HWID-спуфинга и методы аппаратной идентификации античитов.
