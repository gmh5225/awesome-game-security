Ce projet est un outil Windows destiné à usurper les identifiants matériels du GPU NVIDIA utilisés pour les bans HWID (Hardware ID) dans les jeux. Il modifie les numéros de série du GPU, les ID de périphérique et d'autres identifiants rapportés via l'interface du pilote NVIDIA afin de contourner les bans matériels. L'outil corrige les requêtes au niveau du pilote et les entrées de registre que les systèmes anti-triche utilisent pour l'empreinte matérielle. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les techniques de spoofing HWID et les méthodes d'identification matérielle des anti-triche.
