Este proyecto es un buscador automático de gadgets ROP y un constructor de cadenas basado en el framework angr.
Utiliza ejecución simbólica, resolución de restricciones y búsqueda en grafos para modelar los efectos de los gadgets y sintetizar cadenas de explotación de forma programática.
La herramienta está implementada en Python, expone tanto una CLI como una API y es independiente de la arquitectura en múltiples objetivos.
Resulta especialmente útil para el desarrollo de exploits, la investigación de binarios y flujos de trabajo de seguridad ofensiva que se solapan con el análisis de vulnerabilidades de clientes de juegos.
