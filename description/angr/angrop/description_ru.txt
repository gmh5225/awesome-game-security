Этот проект — автоматический поиск ROP-гаджетов и построитель цепочек на базе фреймворка angr.
Он использует символьное выполнение, решение ограничений и поиск по графам, чтобы моделировать эффекты гаджетов и программно синтезировать эксплойт-цепочки.
Инструмент реализован на Python, предоставляет CLI и API и не зависит от архитектуры для множества целевых платформ.
Он особенно полезен для разработки эксплойтов, исследования бинарных файлов и рабочих процессов наступательной безопасности, пересекающихся с анализом уязвимостей игровых клиентов.
