Этот проект — исходный выпуск NVIDIA PhysX, SDK физики реального времени для игр и интерактивной симуляции.
Он включает кроссплатформенные системы твёрдых тел и столкновений, примеры приложений, документацию и материалы для интеграции, используемые крупными игровыми движками.
Кодовая база преимущественно на C++ с генерацией сборки через CMake и Python для платформ Windows, Linux, Android и Apple.
Он рассчитан на разработчиков движков и геймплея, которым нужна физика промышленного уровня для разработки игр и смежных исследований в области симуляции.
