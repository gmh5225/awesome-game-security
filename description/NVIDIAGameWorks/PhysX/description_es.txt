Este proyecto es la publicación del código fuente de NVIDIA PhysX, un SDK de física en tiempo real para juegos y simulación interactiva.
Incluye sistemas multiplataforma de cuerpos rígidos y colisiones, aplicaciones de ejemplo, documentación y material de integración usado por los principales motores de juego.
La base de código es predominantemente C++, con generación de builds impulsada por CMake y Python, en Windows, Linux, Android y plataformas Apple.
Está dirigido a desarrolladores de motores y de gameplay que necesitan tecnología física de calidad de producción para el desarrollo de juegos y la investigación en simulación relacionada.
