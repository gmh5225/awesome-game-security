Este proyecto es el NVIDIA PhysX SDK versión 3.4, un motor de simulación física de producción para juegos y aplicaciones interactivas. Proporciona dinámica de cuerpos rígidos, detección continua de colisiones, consultas de escena, simulación de vehículos, cloth y sistemas de partículas, con una arquitectura basada en componentes y soporte multiplataforma. Resulta especialmente útil para desarrolladores de juegos e investigadores de motores físicos que estudian la implementación e integración de simulaciones físicas de nivel industrial.
