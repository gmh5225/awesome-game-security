Este proyecto es una prueba de concepto en modo kernel de Windows centrada en la manipulación de callbacks anticheat y hooks de integridad. Demuestra técnicas como la redirección de object callbacks mediante code caves y la alteración de rutas seleccionadas de comprobación de integridad. La implementación está escrita en C/C++ como un proyecto de controlador al estilo KMDF orientado a los internals del sistema de bajo nivel. Se usa principalmente para investigación anticheat, pruebas defensivas y para comprender estrategias de detección y endurecimiento del kernel.
