Dieses Projekt ist ein Windows-Kernelmodus-Proof-of-Concept mit Fokus auf die Manipulation von Anti-Cheat-Callbacks und Integrity-Hooks. Es demonstriert Techniken wie die Umleitung von Object Callbacks über Code Caves sowie die Manipulation ausgewählter Integritätsprüfungspfade. Die Implementierung ist in C/C++ als KMDF-artiges Treiberprojekt geschrieben und zielt auf niedrigstufige Systeminternals ab. Es dient vor allem der Anti-Cheat-Forschung, defensiven Tests sowie dem Verständnis von Kernel-Erkennung und Härtungsstrategien.
