Questo progetto è una proof of concept in modalità kernel Windows incentrata sulla manipolazione di callback anti-cheat e hook di integrità. Dimostra tecniche come la redirezione degli object callback tramite code cave e la manomissione di percorsi selezionati di controllo dell'integrità. L'implementazione è scritta in C/C++ come progetto driver in stile KMDF, mirato agli internals di sistema a basso livello. È usato principalmente per la ricerca anti-cheat, i test difensivi e per comprendere le strategie di rilevamento e hardening del kernel.
