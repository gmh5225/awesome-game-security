Dieses Projekt ist ein C++-Proof-of-Concept für das Hijacking externer Overlay-Fenster unter Windows. Es untersucht die Nutzung eines bestehenden Overlay-Fenstermodells unter Beibehaltung nativer Fenster-Flags sowie die Eingabeerfassung über SetWindowsHookEx. Der Code enthält DirectX11- und ImGui-Komponenten für das Rendering sowie eigene Module zur Eingabeverarbeitung. Es richtet sich vor allem an Game-Security- und Anti-Cheat-Forscher, die overlay-basierte Tools sowie Sichtbarkeits- und Erkennungs-Tradeoffs analysieren.
