Этот проект — концептуальное доказательство на C++ для перехвата внешних окон оверлея в Windows. Он исследует использование существующей модели окна оверлея с сохранением нативных флагов окна и перехватом ввода через SetWindowsHookEx. Код включает компоненты DirectX11 и ImGui для отрисовки, а также отдельные модули обработки ввода. Он в первую очередь предназначен для исследователей игровой безопасности и античита, анализирующих инструменты на базе оверлеев и компромиссы между видимостью и обнаружением.
