Este proyecto es una prueba de concepto en C++ para el secuestro de ventanas de overlay externas en Windows. Explora el uso de un modelo de ventana de overlay existente preservando los flags nativos de la ventana y capturando la entrada mediante SetWindowsHookEx. El código incluye componentes de DirectX11 e ImGui para el renderizado, junto con módulos dedicados al manejo de entrada. Está dirigido principalmente a investigadores de seguridad de juegos y anti-cheat que analizan herramientas basadas en overlays y las ventajas y desventajas de visibilidad o detección.
