Questo progetto è una proof of concept in C++ per l'hijacking di finestre overlay esterne su Windows. Esplora l'uso di un modello di finestra overlay esistente preservando i flag nativi della finestra e catturando l'input tramite SetWindowsHookEx. Il codice include componenti DirectX11 e ImGui per il rendering, insieme a moduli dedicati alla gestione dell'input. È pensato principalmente per ricercatori di game security e anti-cheat che analizzano strumenti basati su overlay e i trade-off di visibilità o rilevamento.
