Этот проект — фреймворк фаззинга для сервера Apache HTTPD, который заменяет сокетный слой Apache пользовательскими I/O-фильтрами и подаёт сырые байты напрямую в те же пути кода, что обрабатывают реальный HTTP-трафик. Фреймворк на Python включает воспроизводители известных CVE и подробную документацию по внутренностям Apache: пулам памяти, хукам, фильтрам, bucket и конвейеру запросов. Он в первую очередь полезен исследователям уязвимостей и инженерам по безопасности, изучающим серверный фаззинг и поверхности атаки при разборе HTTP.
