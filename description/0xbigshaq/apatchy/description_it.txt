Questo progetto è un framework di fuzzing per il server Apache HTTPD che sostituisce il layer socket di Apache con filtri I/O personalizzati, inviando byte grezzi direttamente negli stessi percorsi di codice che gestiscono il traffico HTTP reale. Il framework basato su Python include riproduttori per CVE note e documentazione dettagliata sugli interni di Apache, come memory pool, hook, filtri, bucket e la pipeline delle richieste. È soprattutto utile per ricercatori di vulnerabilità e security engineer che studiano tecniche di fuzzing lato server e superfici di attacco del parsing HTTP.
