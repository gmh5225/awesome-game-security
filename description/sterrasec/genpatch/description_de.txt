Dieses Projekt konzentriert sich auf ein Python-Skript zum Patchen von Binärdateien. Wird das Patch-Skript erfolgreich erzeugt, erscheint ein Dialog ähnlich dem folgenden. Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / IDA-Plugins untersuchen.
