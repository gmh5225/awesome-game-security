Этот проект — репозиторий из раздела anti-cheat / open source anti-cheat system, посвящённый RebirthGuard.
Он в основном написан на C++ и C/C++ и сосредоточен на исследованиях античита, моддинге и генерации SDK.
Он особенно полезен инженерам античита и исследователям защитной безопасности, работающим в области anti-cheat / open source anti-cheat system.
