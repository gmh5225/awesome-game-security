Dieses Projekt ist rAthena, ein Open-Source-Ragnarok-Online-Serveremulator in C++. Es implementiert Login-, Charakter- und Map-Server mit Unterstützung für Tausende von Items, Monstern, Skills, Quests, Gilden, WoE (War of Emperium) sowie benutzerdefiniertes Scripting über eine NPC-Skriptsprache. Der Server unterstützt MySQL-Datenbankspeicherung und Multi-Zone-Architektur. Er richtet sich an Spiele-Server-Entwickler, Betreiber privater Server und Forscher, die klassische MMO-Server-Implementierungsmuster untersuchen.
