Questo progetto è rAthena, un emulatore di server Ragnarok Online open source scritto in C++. Implementa i server di login, personaggi e mappe, con supporto per migliaia di oggetti, mostri, skill, quest, gilde, WoE (War of Emperium) e scripting personalizzato tramite un linguaggio di script NPC. Il server supporta l'archiviazione su database MySQL e un'architettura multi-zona. È rivolto a sviluppatori di server di gioco, operatori di server privati e ricercatori che studiano i modelli di implementazione dei server MMO classici.
