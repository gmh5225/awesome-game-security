Este proyecto es una prueba de concepto de explotación del kernel de Android centrada en la escalada de privilegios relacionada con io_uring.
Está implementado principalmente en C, con scripts de compilación de Android NDK y variantes de exploit separadas para distintos dispositivos y objetivos de kernel.
El repositorio incluye herramientas auxiliares para desempaquetar imágenes boot y extraer símbolos del kernel, de modo que el exploit pueda adaptarse a builds de firmware coincidentes.
Su caso de uso principal es la investigación de seguridad del kernel, la reproducción de exploits y el estudio de causas raíz en entornos de prueba móviles autorizados.
