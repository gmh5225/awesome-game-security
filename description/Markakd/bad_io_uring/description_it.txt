Questo progetto è una proof of concept di exploitation del kernel Android incentrata sull'escalation di privilegi legata a io_uring.
È implementato principalmente in C, con script di build Android NDK e varianti di exploit separate per diversi dispositivi e target di kernel.
Il repository include strumenti di supporto per scompattare le immagini boot ed estrarre i simboli del kernel, così da adattare l'exploit a build firmware corrispondenti.
Il caso d'uso principale è la ricerca sulla sicurezza del kernel, la riproduzione di exploit e lo studio delle cause radice in ambienti di test mobili autorizzati.
