Este proyecto es una herramienta de detección anti-depuración para Windows escrita en C que comprueba más de 30 técnicas de detección de depuradores. Prueba comprobaciones basadas en API (IsDebuggerPresent, CheckRemoteDebuggerPresent), flags de NtQueryInformationProcess, campos del PEB, puntos de interrupción hardware, ataques de temporización, callbacks TLS y otros indicadores anti-debug. Los resultados se muestran en una consola con estado de aprobado/fallido para cada comprobación. Está dirigido a investigadores de seguridad y analistas de malware que estudian las técnicas anti-depuración y su eficacia.
