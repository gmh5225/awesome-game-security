Ce projet est un outil de détection anti-débogage Windows écrit en C, qui vérifie plus de 30 techniques de détection de débogueur. Il teste des contrôles basés sur les API (IsDebuggerPresent, CheckRemoteDebuggerPresent), les flags de NtQueryInformationProcess, les champs du PEB, les points d'arrêt matériels, les attaques par timing, les callbacks TLS, ainsi que divers autres indicateurs anti-debug. Les résultats sont affichés dans une console avec un statut réussi/échoué pour chaque contrôle. Il s'adresse aux chercheurs en sécurité et aux analystes de malwares qui étudient les techniques anti-débogage et leur efficacité.
