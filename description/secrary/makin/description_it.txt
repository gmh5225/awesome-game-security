Questo progetto è uno strumento di rilevamento anti-debugging per Windows scritto in C che verifica oltre 30 tecniche di rilevamento del debugger. Testa controlli basati su API (IsDebuggerPresent, CheckRemoteDebuggerPresent), flag di NtQueryInformationProcess, campi del PEB, breakpoint hardware, attacchi di timing, callback TLS e vari altri indicatori anti-debug. I risultati vengono mostrati in una console con stato superato/fallito per ciascun controllo. È rivolto a ricercatori di sicurezza e analisti di malware che studiano le tecniche anti-debugging e la loro efficacia.
