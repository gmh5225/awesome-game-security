Этот проект — инструмент обнаружения антиотладки для Windows, написанный на C, который проверяет более 30 техник обнаружения отладчика. Он тестирует проверки на основе API (IsDebuggerPresent, CheckRemoteDebuggerPresent), флаги NtQueryInformationProcess, поля PEB, аппаратные точки останова, timing-атаки, TLS-колбэки и другие индикаторы антиотладки. Результаты отображаются в консоли со статусом пройдено/не пройдено для каждой проверки. Предназначен для исследователей безопасности и аналитиков вредоносного ПО, изучающих техники антиотладки и их эффективность.
