Este proyecto es idenLibX, una versión en plugin de x64dbg de idenLib para identificar funciones de bibliotecas enlazadas estáticamente durante el análisis dinámico. Escanea los módulos cargados en el proceso depurado en busca de firmas conocidas de funciones de biblioteca y aplica nombres de funciones en tiempo real dentro de x64dbg. El plugin en C++ aporta capacidades de identificación de bibliotecas al flujo de depuración. Está dirigido a ingenieros de ingeniería inversa que prefieren x64dbg para el análisis dinámico y necesitan identificar funciones de biblioteca.
