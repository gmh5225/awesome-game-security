Questo progetto è idenLibX, una versione plugin di x64dbg di idenLib per identificare funzioni di librerie collegate staticamente durante l’analisi dinamica. Analizza i moduli caricati nel processo in debug alla ricerca di firme note di funzioni di libreria e applica i nomi delle funzioni in tempo reale in x64dbg. Il plugin C++ porta le capacità di identificazione delle librerie nel flusso di debugging. È rivolto agli ingegneri di reverse engineering che preferiscono x64dbg per l’analisi dinamica e necessitano di identificare le funzioni di libreria.
