Этот проект — idenLibX, плагин для x64dbg на основе idenLib для идентификации статически связанных библиотечных функций при динамическом анализе. Он сканирует загруженные модули в отлаживаемом процессе на известные сигнатуры библиотечных функций и применяет имена функций в реальном времени в x64dbg. Плагин на C++ переносит возможности идентификации библиотек в рабочий процесс отладки. Он предназначен для специалистов по реверс-инжинирингу, которые предпочитают x64dbg для динамического анализа и нуждаются в идентификации библиотечных функций.
