Ce projet est idenLibX, une version plugin x64dbg d’idenLib destinée à identifier les fonctions de bibliothèques liées statiquement lors de l’analyse dynamique. Il analyse les modules chargés dans le processus débogué à la recherche de signatures de fonctions de bibliothèque connues et applique les noms de fonctions en temps réel dans x64dbg. Le plugin C++ apporte les capacités d’identification de bibliothèques au flux de débogage. Il s’adresse aux ingénieurs en rétro-ingénierie qui préfèrent x64dbg pour l’analyse dynamique et ont besoin d’identifier les fonctions de bibliothèque.
