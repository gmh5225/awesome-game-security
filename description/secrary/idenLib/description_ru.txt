Этот проект — idenLib, плагин для IDA Pro, предназначенный для идентификации статически слинкованных библиотечных функций в бинарниках без символов. Он сопоставляет байтовые шаблоны функций с базой известных сигнатур библиотек (похоже на FLIRT, но с другим подходом к сопоставлению), чтобы восстановить имена функций из runtime Visual C++, STL и других часто подключаемых библиотек. Плагин на C++ помогает переименовывать неизвестные функции в stripped PE-бинарниках. Он рассчитан на специалистов по реверс-инжинирингу, работающих с Windows-исполняемыми файлами без символов.
