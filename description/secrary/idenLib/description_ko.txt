이 프로젝트는 idenLib로, 심볼이 제거된 바이너리에서 정적으로 링크된 라이브러리 함수를 식별하기 위한 IDA Pro 플러그인입니다. 함수 바이트 패턴을 알려진 라이브러리 시그니처 데이터베이스와 대조하여(FLIRT와 유사하지만 다른 매칭 방식 사용) Visual C++ 런타임, STL 및 기타 자주 링크되는 라이브러리의 함수 이름을 복원합니다. 이 C++ 플러그인은 심볼이 제거된 PE 바이너리에서 알 수 없는 함수의 이름을 바꾸는 데 도움이 됩니다. 심볼이 제거된 Windows 실행 파일을 다루는 리버스 엔지니어를 대상으로 합니다.
