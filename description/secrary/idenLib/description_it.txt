Questo progetto è idenLib, un plugin di IDA Pro per identificare le funzioni di librerie collegate staticamente in binari senza simboli. Confronta i pattern di byte delle funzioni con un database di firme di librerie note (simile a FLIRT, ma con un approccio di matching diverso) per recuperare i nomi delle funzioni del runtime Visual C++, della STL e di altre librerie comunemente collegate. Il plugin C++ aiuta a rinominare le funzioni sconosciute nei binari PE senza simboli. È rivolto agli ingegneri di reverse engineering che lavorano con eseguibili Windows privati di simboli.
