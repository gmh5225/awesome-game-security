Este proyecto es idenLib, un plugin de IDA Pro para identificar funciones de bibliotecas enlazadas estáticamente en binarios sin símbolos. Compara patrones de bytes de funciones con una base de datos de firmas de bibliotecas conocidas (similar a FLIRT, pero con un enfoque de coincidencia distinto) para recuperar nombres de funciones del runtime de Visual C++, la STL y otras bibliotecas enlazadas habitualmente. El plugin en C++ ayuda a renombrar funciones desconocidas en binarios PE sin símbolos. Está orientado a ingenieros de ingeniería inversa que trabajan con ejecutables de Windows despojados de símbolos.
