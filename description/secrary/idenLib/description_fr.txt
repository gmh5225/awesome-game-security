Ce projet est idenLib, un plugin IDA Pro destiné à identifier les fonctions de bibliothèques liées statiquement dans des binaires sans symboles. Il compare les motifs d'octets des fonctions à une base de signatures de bibliothèques connues (similaire à FLIRT, mais avec une approche de correspondance différente) afin de retrouver les noms de fonctions du runtime Visual C++, de la STL et d'autres bibliothèques couramment liées. Le plugin C++ aide à renommer les fonctions inconnues dans des binaires PE sans symboles. Il s'adresse aux ingénieurs en reverse engineering travaillant sur des exécutables Windows dépouillés.
