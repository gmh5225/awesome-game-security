Этот проект — Windows proof of concept, перехватывающий возвраты из syscall с помощью механизма instrumentation callback (NtSetInformationProcess с ProcessInstrumentationCallback). После настройки callback срабатывает при каждом переходе из ядра в пользовательский режим, позволяя просматривать или изменять результаты syscall без патча stub'ов ntdll. Реализация на C/C++ журналирует перехваченные вызовы и демонстрирует, как эту технику можно применять для мониторинга или подмены. Предназначена для исследователей безопасности, изучающих альтернативные методы хукинга и техники обхода античита или EDR.
