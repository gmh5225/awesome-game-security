Этот проект — плагин для IDA Pro, расширяющий встроенную функциональность применения типов callee. Python-скрипт улучшает распространение типов в местах вызовов функций и включает тестовые бинарники и манифест ida-plugin.json для простой интеграции в систему плагинов IDA. Он в первую очередь полезен специалистам по реверс-инжинирингу, которым нужны улучшенный вывод типов и разрешение типов callee при анализе сложных бинарных файлов в IDA Pro.
