Questo progetto è un plugin per IDA Pro che estende la funzionalità integrata di applicazione dei tipi callee. Lo script Python migliora la propagazione dei tipi nei punti di chiamata alle funzioni e include binari di test e un manifesto ida-plugin.json per una facile integrazione nel sistema di plugin di IDA. È particolarmente utile per i reverse engineer che necessitano di un'inferenza dei tipi e di una risoluzione dei tipi callee migliori quando analizzano binari complessi in IDA Pro.
