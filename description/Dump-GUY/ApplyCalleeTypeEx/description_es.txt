Este proyecto es un plugin de IDA Pro que amplía la funcionalidad integrada de aplicación de tipos de callee. El script de Python mejora la propagación de tipos en los puntos de llamada a funciones, e incluye binarios de prueba y un manifiesto ida-plugin.json para integrarlo fácilmente en el sistema de plugins de IDA. Resulta especialmente útil para ingenieros de ingeniería inversa que necesitan una mejor inferencia de tipos y resolución de tipos de callee al analizar binarios complejos en IDA Pro.
