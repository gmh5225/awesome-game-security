Este proyecto convierte los encabezados de PHNT Native API en recursos de tipos compatibles con IDA para ingeniería inversa. Proporciona scripts IDC generados y bibliotecas TIL para flujos de trabajo de 32 y 64 bits, basados en el Windows SDK moderno y las definiciones de PHNT. El conjunto de herramientas se centra en el procesamiento de encabezados C/C++ con utilidades de IDA como idaclang y tilib. Lo usan principalmente ingenieros inversos que necesitan tipos internos de Windows y prototipos de funciones más ricos al analizar binarios del sistema, controladores o componentes anti-trampas.
