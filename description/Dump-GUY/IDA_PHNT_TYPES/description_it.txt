Questo progetto converte gli header PHNT Native API in risorse di tipi compatibili con IDA per il reverse engineering. Fornisce script IDC generati e librerie TIL per flussi di lavoro a 32 e 64 bit, basati sul Windows SDK moderno e sulle definizioni PHNT. Gli strumenti sono incentrati sull’elaborazione degli header C/C++ con utility IDA come idaclang e tilib. È usato soprattutto da reverse engineer che necessitano di tipi interni di Windows e prototipi di funzioni più ricchi quando analizzano binari di sistema, driver o componenti anti-cheat.
