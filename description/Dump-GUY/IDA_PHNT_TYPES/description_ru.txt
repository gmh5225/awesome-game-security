Этот проект преобразует заголовки PHNT Native API в совместимые с IDA ресурсы типов для реверс-инжиниринга. Он предоставляет сгенерированные IDC-скрипты и библиотеки TIL для 32- и 64-битных рабочих процессов на основе современных определений Windows SDK и PHNT. Инструментарий сосредоточен на обработке заголовков C/C++ с утилитами IDA, такими как idaclang и tilib. В первую очередь он предназначен для реверс-инженеров, которым нужны более полные внутренние типы Windows и прототипы функций при анализе системных бинарников, драйверов или компонентов античита.
