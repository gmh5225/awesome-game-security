This project is a DMA-based cheat and memory tool for Grand Theft Auto V that reads and writes game memory through external DMA hardware rather than injecting into the game process. It is implemented mainly in C++ as a Visual Studio Win32 application, using MemProcFS/VMMDLL (LeechCore) for DMA access and ImGui with DirectX 11 for the overlay menu UI. Feature modules cover god mode, teleport, no wanted level, health and armor control, vehicle editing, weapon inspection, invisibility, no collision, player speed, ragdoll, time control, and related player or heist helpers, with support for both the Legacy and Enhanced GTA5 executables. The repository also ships Cheat Engine offset tables for those builds, including BattlEye-related patches, making it useful for DMA cheat development, game reverse engineering, and anti-cheat research around out-of-process memory manipulation.
