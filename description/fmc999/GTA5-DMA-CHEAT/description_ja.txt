本プロジェクトは、Grand Theft Auto V 向けの DMA ベースのチートおよびメモリツールで、ゲームプロセスへの注入ではなく外部 DMA ハードウェア経由でゲームメモリを読み書きします。主に C++ で Visual Studio Win32 アプリケーションとして実装され、DMA アクセスには MemProcFS/VMMDLL（LeechCore）を、オーバーレイメニュー UI には ImGui と DirectX 11 を使用します。機能モジュールにはゴッドモード、テレポート、指名手配解除、体力・アーマー制御、車両編集、武器確認、透明化、無衝突、プレイヤー速度、ラグドール、時間操作、および関連するプレイヤー／強盗支援が含まれ、Legacy 版と Enhanced 版の両 GTA5 実行ファイルに対応します。リポジトリにはこれらのビルド向け Cheat Engine オフセット表（BattlEye 関連パッチを含む）も同梱されており、DMA チート開発、ゲームリバースエンジニアリング、プロセス外メモリ操作に関するアンチチート研究に役立ちます。
