本專案是一款基於 DMA 的《Grand Theft Auto V》作弊與記憶體工具，透過外部 DMA 硬體讀寫遊戲記憶體，而無需注入遊戲行程。主體以 C++ 實作為 Visual Studio Win32 應用程式，使用 MemProcFS/VMMDLL（LeechCore）進行 DMA 存取，並以 ImGui 搭配 DirectX 11 建構覆蓋層選單介面。功能模組涵蓋無敵模式、傳送、無通緝、生命值與護甲控制、載具編輯、武器檢視、隱身、無碰撞、玩家速度、布娃娃、時間控制，以及相關玩家或搶劫輔助，並同時支援 Legacy 與 Enhanced 版 GTA5 可執行檔。倉庫亦附帶這些版本的 Cheat Engine 偏移表（含 BattlEye 相關修補），適用於 DMA 作弊開發、遊戲逆向工程，以及圍繞行程外記憶體操作的反作弊研究。
