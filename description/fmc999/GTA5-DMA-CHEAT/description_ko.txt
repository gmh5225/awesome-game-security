이 프로젝트는 Grand Theft Auto V용 DMA 기반 치트 및 메모리 도구로, 게임 프로세스에 주입하지 않고 외부 DMA 하드웨어를 통해 게임 메모리를 읽고 씁니다. 주로 C++로 Visual Studio Win32 애플리케이션으로 구현되며, DMA 접근에는 MemProcFS/VMMDLL(LeechCore)을, 오버레이 메뉴 UI에는 ImGui와 DirectX 11을 사용합니다. 기능 모듈에는 무적 모드, 텔레포트, 지명수배 해제, 체력·아머 제어, 차량 편집, 무기 확인, 투명화, 무충돌, 플레이어 속도, 래그돌, 시간 제어 및 관련 플레이어·하이스트 보조 기능이 포함되며, Legacy 및 Enhanced GTA5 실행 파일을 모두 지원합니다. 저장소에는 해당 빌드용 Cheat Engine 오프셋 테이블(BattlEye 관련 패치 포함)도 제공되어 DMA 치트 개발, 게임 리버스 엔지니어링, 프로세스 외부 메모리 조작에 관한 안티치트 연구에 유용합니다.
