Этот проект — инструмент читов и работы с памятью на основе DMA для Grand Theft Auto V, который читает и записывает память игры через внешнее DMA-оборудование, не внедряясь в процесс игры. Он в основном реализован на C++ как приложение Visual Studio Win32, использует MemProcFS/VMMDLL (LeechCore) для доступа по DMA и ImGui с DirectX 11 для интерфейса оверлей-меню. Модули функций включают режим бога, телепорт, отсутствие розыска, управление здоровьем и бронёй, редактирование транспорта, просмотр оружия, невидимость, отсутствие коллизий, скорость игрока, рэгдолл, управление временем и связанные помощники для игрока или ограблений, с поддержкой исполняемых файлов GTA5 Legacy и Enhanced. В репозитории также есть таблицы смещений Cheat Engine для этих сборок, включая патчи, связанные с BattlEye, что полезно для разработки DMA-читов, реверс-инжиниринга игр и исследований античита в контексте внепроцессной манипуляции памятью.
