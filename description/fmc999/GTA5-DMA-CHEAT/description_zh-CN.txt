本项目是一款基于 DMA 的《Grand Theft Auto V》作弊与内存工具，通过外部 DMA 硬件读写游戏内存，而无需注入游戏进程。主体以 C++ 实现为 Visual Studio Win32 应用，使用 MemProcFS/VMMDLL（LeechCore）进行 DMA 访问，并以 ImGui 配合 DirectX 11 构建覆盖层菜单界面。功能模块涵盖无敌模式、传送、无通缉、生命值与护甲控制、载具编辑、武器查看、隐身、无碰撞、玩家速度、布娃娃、时间控制，以及相关玩家或抢劫辅助，并同时支持 Legacy 与 Enhanced 版 GTA5 可执行文件。仓库还附带这些版本的 Cheat Engine 偏移表（含 BattlEye 相关补丁），适用于 DMA 作弊开发、游戏逆向工程，以及围绕进程外内存操作的反作弊研究。
