Этот проект — инструмент на Python для деобфускации ARM64-кода с обфускацией ветвлений в разделяемых библиотеках.
Он использует эмуляцию Unicorn вместе с Capstone, Keystone и утилитами разбора ELF для анализа и патчинга потока управления.
Рабочий процесс принимает адреса начала и конца функций, обрабатывает обфусцированные области и записывает восстановленные выходные бинарные файлы.
Основной сценарий использования — реверс-инжиниринг и анализ вредоносного ПО или игровых защит в нативных библиотеках Android.
