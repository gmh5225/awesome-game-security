Questo progetto è uno strumento Python per deoffuscare codice ARM64 offuscato mediante branch nelle librerie condivise.
Usa l'emulazione Unicorn insieme a Capstone, Keystone e utilità di parsing ELF per analizzare e patchare il flusso di controllo.
Il flusso di lavoro accetta gli indirizzi di inizio e fine delle funzioni, elabora le regioni offuscate e scrive binari di output ricostruiti.
Il caso d'uso principale è il reverse engineering e l'analisi di malware o protezioni di gioco sulle librerie native Android.
