Este proyecto es una herramienta en Python para desofuscar código ARM64 ofuscado por ramas en bibliotecas compartidas.
Usa emulación con Unicorn junto con Capstone, Keystone y utilidades de análisis ELF para analizar y parchear el flujo de control.
El flujo de trabajo acepta direcciones de inicio y fin de función, procesa regiones ofuscadas y escribe binarios de salida reconstruidos.
Su caso de uso principal es la ingeniería inversa y el análisis de malware o protecciones de juegos en bibliotecas nativas de Android.
