Questo progetto è una demo anti-tampering per Android che rileva ambienti di runtime anomali confrontando i checksum in memoria e su disco delle librerie di sistema critiche.
Combina un'app Android in Java con un componente nativo Rust integrato tramite Gradle e JNI.
I controlli mirano a segni di framework di strumentazione o modifica come Frida, Xposed e ambienti di clonazione.
Il caso d'uso principale è l'hardening della sicurezza mobile e la convalida dell'integrità a runtime per applicazioni Android, inclusi i giochi.
