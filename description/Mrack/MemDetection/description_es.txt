Este proyecto es una demostración anti-manipulación para Android que detecta entornos de ejecución anómalos comparando sumas de verificación en memoria y en disco de bibliotecas del sistema críticas.
Combina una aplicación Android en Java con un componente nativo en Rust integrado mediante Gradle y JNI.
Las comprobaciones buscan indicios de frameworks de instrumentación o modificación como Frida, Xposed y entornos de clonación.
Su caso de uso principal es el refuerzo de seguridad móvil y la validación de integridad en tiempo de ejecución para aplicaciones Android, incluidos los juegos.
