Ce projet est une démonstration anti-falsification Android qui détecte les environnements d'exécution anormaux en comparant les sommes de contrôle en mémoire et sur disque des bibliothèques système critiques.
Il combine une application Android Java avec un composant natif Rust intégré via Gradle et JNI.
Les contrôles ciblent les signes d'instrumentation ou de frameworks de modification tels que Frida, Xposed et les environnements de clonage.
Son cas d'usage principal est le durcissement de la sécurité mobile et la validation d'intégrité à l'exécution pour les applications Android, y compris les jeux.
