OxClient는 로컬 패킷 중계를 통해 게임과 원격 서버 사이에 위치하는 Android용 Minecraft Bedrock Edition 클라이언트입니다. 주로 Kotlin으로 작성되며 Gradle/Android 도구를 사용하고, CloudburstMC Bedrock 프로토콜 코덱과 다중 버전 패킷 파싱을 위한 Java NBT 처리를 내장합니다. Microsoft 디바이스 코드 로그인, KillAura·CrystalAura·비행·속도 같은 모듈형 전투 및 이동 치트, ESP 오버레이와 FOV 변경 같은 시각 보조 기능을 제공합니다. 패킷 리스너와 이벤트 버스가 세션 관리, LAN 브로드캐스트, 인게임 오버레이를 구동합니다. Bedrock 멀티플레이어 치팅과 게임 보안·안티치트 분석에 관련된 프로토콜 리버스 엔지니어링 연구를 위한 것입니다.
