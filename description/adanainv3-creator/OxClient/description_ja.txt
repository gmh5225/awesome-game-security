OxClient は、ローカルなパケット中継を介してゲームとリモートサーバーの間に位置する Android 向け Minecraft Bedrock Edition クライアントです。主に Kotlin で書かれ、Gradle/Android ツール群を用い、CloudburstMC の Bedrock プロトコルコーデックと、多バージョンのパケット解析用の Java による NBT 処理を組み込んでいます。機能には Microsoft のデバイスコードログイン、KillAura や CrystalAura、飛行・速度などのモジュール式コンバット／移動チート、ESP オーバーレイや FOV 変更などの視覚補助があります。パケットリスナーとイベントバスがセッション管理、LAN ブロードキャスト、ゲーム内オーバーレイを駆動します。Bedrock マルチプレイヤーのチートや、ゲームセキュリティおよびアンチチート分析に関わるプロトコル逆向エンジニアリング研究向けです。
