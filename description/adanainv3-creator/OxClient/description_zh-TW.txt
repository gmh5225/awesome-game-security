OxClient 是一款 Android 版 Minecraft Bedrock Edition 用戶端，透過本機封包中繼位於遊戲與遠端伺服器之間。主要以 Kotlin 撰寫，搭配 Gradle/Android 工具鏈，並嵌入 CloudburstMC Bedrock 協定編解碼器以及用於多版本封包解析的 Java NBT 處理。功能包括 Microsoft 裝置碼登入，以及 KillAura、CrystalAura、飛行與加速等模組化戰鬥與移動外掛，還有 ESP 疊加層與 FOV 調整等視覺輔助。封包監聽器與事件匯流排負責工作階段管理、區域網路廣播與遊戲內疊加層。面向 Bedrock 多人遊戲作弊，以及與遊戲安全與反作弊分析相關的協定逆向工程研究。
