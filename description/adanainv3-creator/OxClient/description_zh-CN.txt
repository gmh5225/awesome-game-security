OxClient 是一款 Android 版 Minecraft Bedrock Edition 客户端，通过本地数据包中继位于游戏与远程服务器之间。主要使用 Kotlin 编写，配合 Gradle/Android 工具链，并嵌入 CloudburstMC Bedrock 协议编解码器以及用于多版本数据包解析的 Java NBT 处理。功能包括 Microsoft 设备码登录，以及 KillAura、CrystalAura、飞行与加速等模块化战斗与移动作弊，还有 ESP 叠加层与 FOV 调整等视觉辅助。数据包监听器与事件总线负责会话管理、局域网广播和游戏内叠加层。面向 Bedrock 多人游戏作弊，以及与游戏安全和反作弊分析相关的协议逆向工程研究。
