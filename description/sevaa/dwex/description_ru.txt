Этот проект — DWEX (DWARF Explorer), GUI-инструмент для просмотра отладочной информации DWARF в ELF-бинарях и отдельных отладочных файлах. Он отображает полную древовидную структуру DWARF, включая единицы компиляции, типы, функции, переменные и соответствия номеров строк, в интерактивном древовидном представлении. Python-приложение использует pyelftools для разбора DWARF. Предназначен для разработчиков компиляторов, авторов отладчиков и специалистов по реверс-инжинирингу, которым нужно изучать отладочные данные DWARF в скомпилированных бинарных файлах.
