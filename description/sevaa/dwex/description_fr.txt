Ce projet est DWEX (DWARF Explorer), un outil GUI pour parcourir les informations de débogage DWARF dans les binaires ELF et les fichiers de débogage séparés. Il affiche la structure arborescente DWARF complète, y compris les unités de compilation, les types, les fonctions, les variables et les correspondances de numéros de ligne, dans une vue arborescente interactive. L'application Python utilise pyelftools pour l'analyse DWARF. Elle s'adresse aux développeurs de compilateurs, aux auteurs de débogueurs et aux ingénieurs en rétro-ingénierie qui doivent inspecter les données de débogage DWARF dans les binaires compilés.
