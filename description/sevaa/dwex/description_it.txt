Questo progetto è DWEX (DWARF Explorer), uno strumento GUI per esplorare le informazioni di debug DWARF nei binari ELF e nei file di debug separati. Visualizza l'intera struttura ad albero DWARF, comprese unità di compilazione, tipi, funzioni, variabili e mappature dei numeri di riga, in una vista ad albero interattiva. L'applicazione Python utilizza pyelftools per l'analisi DWARF. È pensata per sviluppatori di compilatori, autori di debugger e reverse engineer che devono ispezionare i dati di debug DWARF nei binari compilati.
