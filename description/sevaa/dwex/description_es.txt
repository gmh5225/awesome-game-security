Este proyecto es DWEX (DWARF Explorer), una herramienta GUI para examinar información de depuración DWARF en binarios ELF y archivos de depuración separados. Muestra la estructura completa del árbol DWARF, incluidos unidades de compilación, tipos, funciones, variables y mapeos de números de línea, en una vista de árbol interactiva. La aplicación Python utiliza pyelftools para el análisis de DWARF. Está dirigida a desarrolladores de compiladores, autores de depuradores e ingenieros de ingeniería inversa que necesitan inspeccionar datos de depuración DWARF en binarios compilados.
