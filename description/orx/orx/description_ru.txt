Этот проект — репозиторий из раздела game engine / source, посвящённый orx.
Он в основном написан на JavaScript и C/C++ и сосредоточен на работе на уровне ядра, шейдерах и рендеринге.
Он в первую очередь полезен разработчикам игр, программистам движков и исследователям графики, работающим в области game engine / source.
