Ce projet est un dépôt listé dans la catégorie game engine / source et s’articule autour d’orx.
Il est principalement écrit en JavaScript et C/C++ et porte sur le travail au niveau noyau, les shaders et le rendu.
Il est surtout utile aux développeurs de jeux, programmeurs moteur et chercheurs en graphisme travaillant dans le domaine game engine / source.
