Dieses Projekt konzentriert sich auf direkte System Calls.
Funktionen mit ENUM werden ebenfalls unterstützt, doch die meisten davon stürzen den Prozess in der Phase des Aufrufs der Parameterkonvertierung ab.
Es ist vor allem für Low-Level-Forscher zu Windows, Linux und Mobile nützlich, die im Bereich some tricks / Windows ring3 arbeiten.
