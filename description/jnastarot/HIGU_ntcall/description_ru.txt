Этот проект посвящён прямым системным вызовам (System Calls).
Функции с ENUM также поддерживаются, но большинство из них приводит к аварийному завершению процесса на этапе вызова преобразования параметров.
Он в основном полезен исследователям низкоуровневых аспектов Windows, Linux и мобильных платформ, работающим в области some tricks / Windows ring3.
