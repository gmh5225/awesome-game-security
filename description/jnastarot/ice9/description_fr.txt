Ce projet est un dépôt listé sous anti cheat / open source anti cheat system et s’articule autour d’ice9.
Il est principalement écrit en C/C++ et C++ et porte sur la recherche anti-triche, le modding et le hooking.
Il est surtout utile aux ingénieurs anti-triche et aux chercheurs en sécurité défensive travaillant dans le domaine anti cheat / open source anti cheat system.
