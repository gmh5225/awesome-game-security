Этот проект — репозиторий из раздела anti cheat / open source anti cheat system, в центре которого находится ice9.
Он в основном написан на C/C++ и C++ и посвящён исследованиям античита, моддингу и хукингу.
Он прежде всего полезен инженерам античита и исследователям защитной безопасности, работающим в области anti cheat / open source anti cheat system.
