Este proyecto es un repositorio listado bajo anti cheat / open source anti cheat system y se centra en ice9.
Está escrito principalmente en C/C++ y C++ y se orienta a la investigación anti-trampas, el modding y el hooking.
Resulta especialmente útil para ingenieros anti-trampas e investigadores de seguridad defensiva que trabajan en el área anti cheat / open source anti cheat system.
