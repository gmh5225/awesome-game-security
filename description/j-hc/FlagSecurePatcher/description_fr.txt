Ce projet permet de désactiver flag secure et les écouteurs de captures d'écran.
Il est organisé comme documentation et matériel de référence pour le domaine cheat / Magisk, plutôt que comme une base de code d'exécution autonome.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine cheat / Magisk.
