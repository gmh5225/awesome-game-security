本项目是用 C++ 编写的原生 open.mp 服务端组件，通过让客户端读取自身内存并将结果与已知特征比对，来检测客户端作弊与修改。模块化检测器涵盖内存特征、客户端版本白名单、毒化检查、移动端客户端、RakNet 数据包检查，以及 Faker5 和 SCC 风格 clientcheck 绕过等欺骗器，针对 s0beit、CLEO、MoonLoader、SilentAim 及类似模组。它使用 CMake 基于 open.mp SDK 构建为 32 位共享库，附带可配置的 anticheat.cfg，支持仅记录模式及按作弊项的忽略/警告/踢出/封禁动作，并可选择暴露 Pawn natives 与 OnPlayerCheatDetected 回调以便与游戏模式集成。主要用途是为 open.mp 与 SA-MP 服主提供防护自身多人服务器的防御性反作弊。
