Этот проект — нативный серверный компонент open.mp на C++, который обнаруживает клиентские читы и модификации, заставляя клиент читать собственную память и сравнивая результаты с известными сигнатурами. Модульные детекторы охватывают сигнатуры памяти, белый список версий клиента, poison-проверки, мобильные клиенты, инспекцию пакетов RakNet и спуферы вроде Faker5 и обходы clientcheck в стиле SCC, нацеливаясь на инструменты вроде s0beit, CLEO, MoonLoader, SilentAim и похожие моды. Он собирается как 32-битная shared library через CMake под open.mp SDK, поставляется с настраиваемым anticheat.cfg для режима только логирования и действий ignore/warn/kick/ban по каждому читу, а также опционально предоставляет Pawn natives и callback OnPlayerCheatDetected для интеграции с игровым режимом. Основной сценарий — защитный античит для операторов серверов open.mp и SA-MP, охраняющих собственные многопользовательские серверы.
