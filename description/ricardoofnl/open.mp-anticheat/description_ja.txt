本プロジェクトは C++ で書かれたネイティブ open.mp サーバーコンポーネントで、クライアントに自身のメモリを読ませ、既知のシグネチャと照合することでクライアント側のチートや改変を検出します。モジュール式検出器はメモリシグネチャ、クライアントバージョンのホワイトリスト、ポイズンチェック、モバイルクライアント、RakNet パケット検査、Faker5 や SCC 系 clientcheck バイパスなどのスプーファーを対象とし、s0beit、CLEO、MoonLoader、SilentAim および類似の MOD を狙います。open.mp SDK に対して CMake で 32 ビット共有ライブラリとしてビルドされ、ログのみモードやチートごとの ignore/warn/kick/ban を設定できる anticheat.cfg を同梱し、必要に応じて Pawn natives と OnPlayerCheatDetected コールバックを公開してゲームモード連携も可能です。主な用途は、open.mp および SA-MP サーバー運営者が自前のマルチプレイヤーサーバーを守るための防御的アンチチートです。
