Plugin di IDA Pro che importa i nomi dei simboli dai file di output del linker .MAP nei formati VC, Borland, Dede, GCC e IDA nel database corrente. Analizza la notazione section:offset di vari formati di file MAP e crea funzioni/etichette con nome agli indirizzi corrispondenti usando le API kernwin e segment dell'SDK di IDA.
