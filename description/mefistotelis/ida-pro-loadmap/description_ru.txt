Плагин IDA Pro, импортирующий имена символов из выходных файлов линкера .MAP в форматах VC, Borland, Dede, GCC и IDA в текущую базу данных. Разбирает нотацию section:offset в различных форматах MAP-файлов и создаёт именованные функции/метки по соответствующим адресам с помощью API kernwin и segment из IDA SDK.
