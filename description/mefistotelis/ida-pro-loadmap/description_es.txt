Complemento de IDA Pro que importa nombres de símbolos desde archivos de salida del enlazador .MAP en formatos VC, Borland, Dede, GCC e IDA a la base de datos actual. Analiza la notación section:offset de diversos formatos de archivos MAP y crea funciones/etiquetas con nombre en las direcciones correspondientes mediante las API kernwin y segment del SDK de IDA.
