Этот проект — утилита командной строки для удаления подписей Authenticode из Windows PE-файлов, таких как EXE, DLL и SYS.
Она реализована на C и поставляется в сборках для 32 и 64 бит вместе с исходным кодом и файлами проекта Visual Studio.
Инструмент удаляет данные, связанные с подписью, и обрабатывает краевые случаи заголовка PE, которые могут мешать повторной подписи.
Он в основном предназначен для реверс-инжиниринга, анализа вредоносного ПО и тестирования безопасности ПО, где требуется работа с неподписанными двоичными файлами.
