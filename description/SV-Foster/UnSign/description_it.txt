Questo progetto è un'utilità da riga di comando per rimuovere le firme Authenticode dai file PE di Windows, come i binari EXE, DLL e SYS.
È implementato in C e fornisce build a 32 e 64 bit, insieme al codice sorgente e ai file di progetto Visual Studio.
Lo strumento rimuove i dati relativi alla firma e gestisce i casi limite dell'header PE che possono interferire con i flussi di ri-firma.
È rivolto principalmente a reverse engineering, analisi malware e test di sicurezza software che richiedono la manipolazione di binari non firmati.
