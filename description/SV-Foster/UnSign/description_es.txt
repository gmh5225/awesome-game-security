Este proyecto es una utilidad de línea de comandos para eliminar firmas Authenticode de archivos PE de Windows, como binarios EXE, DLL y SYS.
Está implementado en C y ofrece compilaciones de 32 y 64 bits, junto con el código fuente y los archivos de proyecto de Visual Studio.
La herramienta elimina los datos relacionados con la firma y aborda casos límite del encabezado PE que pueden interferir en los flujos de re-firma.
Está orientada principalmente a la ingeniería inversa, el análisis de malware y las pruebas de seguridad de software que requieren manipular binarios sin firmar.
