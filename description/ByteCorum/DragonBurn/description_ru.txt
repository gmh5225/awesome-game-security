Этот проект — внешний Windows-фреймворк читов для Counter-Strike 2 с упором на доступ к данным только для чтения с помощью ядра. Он написан преимущественно на C++ и включает меню в стиле ImGui, модули отрисовки, а также обширные настраиваемые функции ESP и радара. Кодовая база делает акцент на автоматизации: поддержке обновления смещений и настройке UI для визуальных оверлеев. Предназначен для экспериментов по разработке читов и исследований обнаружения античитом в среде соревновательных шутеров.
