Ce projet est un framework de cheat externe Windows pour Counter-Strike 2, centré sur l'accès en lecture seule aux données assisté par le noyau. Il est principalement écrit en C++ et propose un menu de style ImGui, des modules de rendu, ainsi que de nombreuses fonctionnalités ESP et radar configurables. La base de code met l'accent sur l'automatisation, notamment la mise à jour des offsets et la personnalisation de l'interface pour les superpositions visuelles. Il est destiné aux expérimentations de développement de cheats et à la recherche sur la détection anti-cheat dans les environnements de shooters compétitifs.
