Dieses Projekt ist ein schlankes Offset-Paket, das Speicher- und Funktions-Offsets für VALORANT nachverfolgt.
Es stellt C/C++-ähnliche Konstanten für Engine-Funktionen und gameplaybezogene Strukturen bereit, die von externen Tools genutzt werden.
Das Repository ist bewusst minimal gehalten und auf schnelle Updates nach Spiel-Patches ausgerichtet.
Der Hauptanwendungsfall ist das Reverse Engineering von Spielen sowie die Wartung von Analyse- oder Automatisierungstools, die auf aktuelle Offsets angewiesen sind.
