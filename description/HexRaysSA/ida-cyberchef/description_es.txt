Este proyecto integra una interfaz de transformación de datos al estilo CyberChef dentro de IDA Pro.
Está implementado principalmente en Python, con una capa de interfaz basada en Qt y metadatos de complemento para la integración con IDA.
La herramienta permite pipelines de decodificación, codificación y transformación sin salir del entorno de ingeniería inversa.
Está orientada a flujos de análisis de malware y triage de binarios que requieren manipulación rápida de datos.
