Этот проект встраивает интерфейс преобразования данных в стиле CyberChef в IDA Pro.
Он реализован в основном на Python с Qt-слоем пользовательского интерфейса и метаданными плагина для интеграции с IDA.
Инструмент позволяет выполнять конвейеры декодирования, кодирования и преобразования, не покидая среду реверс-инжиниринга.
Он предназначен для рабочих процессов анализа вредоносного ПО и triage бинарных файлов, требующих быстрой обработки данных.
