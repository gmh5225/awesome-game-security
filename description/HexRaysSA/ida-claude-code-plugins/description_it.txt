Questo progetto fornisce una raccolta di risorse per workflow di reverse engineering assistito dall'IA in IDA Pro.
È composto soprattutto da strumenti Python e da documentazione Markdown di plugin e skill.
Il pacchetto include indicazioni per lo sviluppo di plugin, scripting automatizzato e configurazioni opzionali di code-evaluation per analisi in sandbox.
È pensato per reverse engineer che desiderano pipeline di analisi binaria ripetibili e fortemente automatizzate.
