Этот проект предоставляет набор ресурсов для рабочих процессов реверс-инжиниринга с поддержкой ИИ в IDA Pro.
В основном он состоит из инструментов на Python и Markdown-документации плагинов и skills.
Пакет включает руководства по разработке плагинов, автоматизированному скриптингу и опциональные настройки code-evaluation для анализа в песочнице.
Он предназначен для реверс-инженеров, которым нужны повторяемые, сильно автоматизированные конвейеры анализа бинарников.
