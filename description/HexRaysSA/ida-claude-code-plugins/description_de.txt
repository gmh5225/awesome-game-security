Dieses Projekt stellt eine Sammlung von Ressourcen für KI-gestützte Reverse-Engineering-Workflows in IDA Pro bereit.
Es besteht hauptsächlich aus Python-Werkzeugen sowie Markdown-basierter Plugin- und Skill-Dokumentation.
Das Paket enthält Leitfäden zur Plugin-Entwicklung, automatisiertes Scripting und optionale Code-Evaluation-Setups für Sandbox-Analysen.
Es richtet sich an Reverse Engineers, die wiederholbare, stark automatisierte Binäranalyse-Pipelines aufbauen möchten.
