Ce projet fournit une collection de ressources pour des workflows d'ingénierie inverse assistés par l'IA dans IDA Pro.
Il se compose surtout d'outils Python et de documentation Markdown pour plugins et skills.
Le package inclut des conseils pour le développement de plugins, l'automatisation de scripts, et des configurations optionnelles d'évaluation de code pour l'analyse en bac à sable.
Il s'adresse aux reverse engineers qui souhaitent des pipelines d'analyse binaire reproductibles et fortement automatisés.
