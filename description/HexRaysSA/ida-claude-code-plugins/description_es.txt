Este proyecto ofrece una colección de recursos para flujos de trabajo de ingeniería inversa asistida por IA en IDA Pro.
Se compone principalmente de herramientas en Python y documentación Markdown de plugins y skills.
El paquete incluye orientación para el desarrollo de plugins, scripting automatizado y configuraciones opcionales de evaluación de código para análisis en entorno aislado.
Está pensado para ingenieros inversos que buscan pipelines de análisis binario repetibles y muy automatizados.
