Questo progetto è un plugin del decompilatore Hex-Rays per semplificare i pattern di offuscamento aritmetico-booleano misto (MBA).
È scritto principalmente in C++ e si integra direttamente nei flussi di lavoro di IDA Pro e Hex-Rays.
Il plugin combina euristiche algebriche, semplificazione MBA lineare e non lineare, e supporto opzionale di oracolo di fingerprint.
Usa inoltre il solver SMT Z3 per verificare la correttezza, rendendolo utile per una deoffuscazione affidabile nel reverse engineering.
