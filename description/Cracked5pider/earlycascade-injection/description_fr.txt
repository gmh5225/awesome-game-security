Ce projet est une preuve de concept de la technique d'injection de processus early cascade sous Windows. Il montre comment créer un processus et déclencher une injection de code plus discrète durant les premières étapes d'initialisation, d'après des recherches documentées publiquement. Le code est écrit en C++ avec des fichiers de projet Visual Studio, et utilise des offsets de structures en dur pour des builds d'OS spécifiques. Il est destiné à l'analyse de malwares, à la recherche sur le contournement d'EDR et aux tests défensifs de détection d'injection.
