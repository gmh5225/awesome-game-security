Questo progetto è una proof of concept della tecnica di process injection early cascade su Windows. Mostra come creare un processo e attivare un'iniezione di codice più furtiva nelle fasi iniziali di inizializzazione, basandosi su ricerche documentate pubblicamente. Il codice è scritto in C++ con file di progetto Visual Studio e usa offset di strutture hardcoded per build di OS specifiche. È pensato per l'analisi di malware, la ricerca sul bypass degli EDR e i test difensivi delle rilevazioni di injection.
