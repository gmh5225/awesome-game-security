Dieses Projekt ist ein Proof of Concept für die Early-Cascade-Prozessinjektionstechnik unter Windows. Es zeigt, wie ein Prozess erstellt und während früher Initialisierungsphasen eine unauffälligere Code-Injektion ausgelöst wird, basierend auf öffentlich dokumentierter Forschung. Der Code ist in C++ mit Visual-Studio-Projektdateien geschrieben und verwendet fest codierte Struktur-Offsets für bestimmte OS-Builds. Es dient der Malware-Analyse, der EDR-Bypass-Forschung und dem defensiven Testen von Injektionserkennungen.
