Este proyecto es una prueba de concepto de la técnica de inyección de procesos early cascade en Windows. Demuestra cómo crear un proceso y desencadenar una inyección de código más sigilosa durante las primeras etapas de inicialización, basándose en investigación documentada públicamente. El código está escrito en C++ con archivos de proyecto de Visual Studio y usa offsets de estructuras codificados de forma fija para compilaciones de SO específicas. Está pensado para el análisis de malware, la investigación de evasión de EDR y las pruebas defensivas de detección de inyecciones.
