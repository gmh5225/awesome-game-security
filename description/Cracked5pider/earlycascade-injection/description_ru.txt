Этот проект — proof of concept техники early cascade process injection в Windows. Он демонстрирует создание процесса и более скрытую инъекцию кода на ранних этапах инициализации, опираясь на публично задокументированные исследования. Код написан на C++ с файлами проекта Visual Studio и использует жёстко заданные смещения структур для конкретных сборок ОС. Предназначен для анализа вредоносного ПО, исследований обхода EDR и защитного тестирования обнаружения инъекций.
