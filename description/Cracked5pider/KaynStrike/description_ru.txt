Этот проект — пользовательский отражающий загрузчик для полезных нагрузок Cobalt Strike Beacon в Windows. Он сосредоточен на подмене адреса старта потока и очистке памяти загрузчика после выполнения точки входа beacon, чтобы уменьшить очевидные артефакты. Реализация сочетает исходный код на C, низкоуровневые ассемблерные процедуры и скрипт Aggressor для сборки и запуска stageless-нагрузок. В основном полезен для исследований в области offensive security и обхода обнаружения, связанных с выполнением полезных нагрузок в памяти.
