Este proyecto es un cargador reflectivo personalizado para payloads de Cobalt Strike Beacon en Windows. Se centra en falsificar la dirección de inicio del hilo y limpiar la memoria del cargador tras ejecutarse el punto de entrada del beacon, para reducir artefactos evidentes. La implementación combina código fuente en C, rutinas de ensamblador de bajo nivel y un script Aggressor para construir y lanzar payloads stageless. Resulta especialmente útil para investigación de seguridad ofensiva y evasión de detección en torno a la ejecución de payloads en memoria.
