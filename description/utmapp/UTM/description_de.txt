Dieses Projekt ist UTM, ein funktionsreicher Virtual-Machine-Host für iOS und macOS auf Basis von QEMU. Es ermöglicht das Ausführen von Windows, Linux und anderen Betriebssystemen auf Apple-Geräten mittels Hardware-Virtualisierung (Apple Hypervisor.framework auf Apple Silicon) oder JIT-gestützter Emulation für x86, ARM, RISC-V und weitere Architekturen. Die Swift/Objective-C-App bietet eine native Oberfläche für die Erstellung und Konfiguration von VMs, USB-Geräte-Passthrough und freigegebene Verzeichnisse. Sie richtet sich an Entwickler und Forscher, die alternative Betriebssysteme auf Apple-Geräten ausführen müssen.
