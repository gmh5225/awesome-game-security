Ce projet est UTM, un hôte de machines virtuelles complet pour iOS et macOS basé sur QEMU. Il permet d'exécuter Windows, Linux et d'autres systèmes d'exploitation sur des appareils Apple via la virtualisation matérielle (Apple Hypervisor.framework sur Apple Silicon) ou une émulation accélérée par JIT pour les architectures x86, ARM, RISC-V et autres. L'application Swift/Objective-C offre une interface native pour la création et la configuration de VM, le passage USB (passthrough) et les répertoires partagés. Elle s'adresse aux développeurs et chercheurs ayant besoin d'exécuter des systèmes d'exploitation alternatifs sur des appareils Apple.
