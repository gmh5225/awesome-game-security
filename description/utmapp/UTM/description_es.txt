Este proyecto es UTM, un host de máquinas virtuales completo para iOS y macOS basado en QEMU. Permite ejecutar Windows, Linux y otros sistemas operativos en dispositivos Apple mediante virtualización por hardware (Apple Hypervisor.framework en Apple Silicon) o emulación impulsada por JIT para arquitecturas x86, ARM, RISC-V y otras. La aplicación en Swift/Objective-C ofrece una interfaz nativa para la creación y configuración de VM, passthrough de dispositivos USB y directorios compartidos. Está dirigida a desarrolladores e investigadores que necesitan ejecutar sistemas operativos alternativos en dispositivos Apple.
