Этот проект — UTM, полнофункциональный хост виртуальных машин для iOS и macOS на базе QEMU. Он позволяет запускать Windows, Linux и другие операционные системы на устройствах Apple с помощью аппаратной виртуализации (Apple Hypervisor.framework на Apple Silicon) или JIT-эмуляции для архитектур x86, ARM, RISC-V и других. Приложение на Swift/Objective-C предоставляет нативный интерфейс для создания и настройки ВМ, проброса USB-устройств и общих каталогов. Предназначено для разработчиков и исследователей, которым нужно запускать альтернативные операционные системы на устройствах Apple.
