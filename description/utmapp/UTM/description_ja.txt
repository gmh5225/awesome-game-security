本プロジェクトは UTM で、QEMU を基盤とした iOS および macOS 向けの高機能仮想マシンホストです。ハードウェア仮想化（Apple Silicon 上の Apple Hypervisor.framework）または JIT によるエミュレーションを用い、Apple デバイス上で Windows、Linux、その他のオペレーティングシステムを実行でき、x86、ARM、RISC-V などのアーキテクチャに対応します。Swift/Objective-C 製アプリは、VM の作成・設定、USB デバイスパススルー、共有ディレクトリ向けのネイティブ UI を提供します。Apple デバイス上で代替 OS を実行する必要がある開発者や研究者を対象としています。
