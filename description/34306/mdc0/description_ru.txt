Этот проект — пользовательское iOS-приложение для jailed-устройств, которое применяет настройки системного UI без традиционного джейлбрейка.
Оно написано на Swift и рассчитано на iOS 15.0–18.3.2, используя exploit-путь для изменения обычно доступных только для чтения системных файлов.
Приложение включает изменения вроде настройки Dock и размытия, правок UI экрана блокировки и других косметических или поведенческих модификаций, которым часто требуется вспомогательное приложение для respring.
Основной сценарий использования — кастомизация iOS и exploit-ориентированные исследования на современных устройствах без джейлбрейка.
