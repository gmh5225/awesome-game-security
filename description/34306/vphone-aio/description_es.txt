Este proyecto es un paquete todo en uno de scripts de shell y archivos comprimidos que configura una instancia del emulador iOS vphone pre-jailbreakeada con bootstrap completo instalado. Automatiza la descarga, fusión y extracción del entorno vphone-cli en macOS, y requiere SIP desactivado y AMFI eludido. Resulta especialmente útil para investigadores de seguridad iOS y usuarios de emuladores que necesitan un entorno iOS jailbreakeado listo para usar en el análisis y las pruebas de aplicaciones.
