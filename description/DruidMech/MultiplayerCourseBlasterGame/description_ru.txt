Этот проект — пример многопользовательского шутера на Unreal Engine 5 на C++ под названием Blaster. В нём есть игровые системы: компоненты боя, компенсация задержек, несколько типов оружия, подбираемые предметы, виджеты HUD, командные режимы и логика захвата флага. Репозиторий также включает плагин мультиплеерных сессий и конфигурацию Steam Online Subsystem для сетевой игры. Он в первую очередь предназначен для разработчиков, изучающих архитектуру многопользовательских игр, и может служить справочником для исследователей игровой безопасности, изучающих сетевое поведение UE.
