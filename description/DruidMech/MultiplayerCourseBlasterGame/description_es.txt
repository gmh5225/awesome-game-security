Este proyecto es un juego de disparos multijugador de ejemplo en C++ para Unreal Engine 5 llamado Blaster. Incluye sistemas de jugabilidad como componentes de combate, compensación de latencia, varios tipos de armas, objetos recogibles, widgets de HUD, modos por equipos y lógica de captura de la bandera. El repositorio también incluye un plugin de sesiones multijugador y la configuración del Steam Online Subsystem para el juego en red. Está orientado principalmente a desarrolladores que aprenden arquitectura de juegos multijugador, y también puede servir de referencia a investigadores de seguridad en juegos que estudian el comportamiento de red de UE.
