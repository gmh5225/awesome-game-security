Этот проект представляет собой скрипт и плагин IDAPython, отображающий промежуточное представление микрокода декомпилятора Hex-Rays. Он обеспечивает визуальный просмотр микрокода, генерируемого в процессе декомпиляции, помогая разработчикам понимать и отлаживать разработку плагинов микрокода Hex-Rays. В первую очередь полезен разработчикам плагинов IDA Pro и специалистам по реверс-инжинирингу, работающим с API микрокода Hex-Rays для создания пользовательских расширений декомпилятора.
