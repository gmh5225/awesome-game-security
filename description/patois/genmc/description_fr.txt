Ce projet est un script et un plugin IDAPython qui affiche la représentation intermédiaire en microcode du décompilateur Hex-Rays. Il permet d'inspecter visuellement le microcode généré lors de la décompilation, aidant ainsi les développeurs à comprendre et à déboguer le développement de plugins microcode Hex-Rays. Il s'adresse principalement aux développeurs de plugins IDA Pro et aux ingénieurs en rétro-ingénierie qui travaillent avec les API microcode Hex-Rays pour des extensions personnalisées du décompilateur.
