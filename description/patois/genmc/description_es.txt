Este proyecto es un script y plugin de IDAPython que muestra la representación intermedia en microcódigo del descompilador Hex-Rays. Proporciona una inspección visual del microcódigo generado durante la descompilación, ayudando a los desarrolladores a comprender y depurar el desarrollo de plugins de microcódigo de Hex-Rays. Resulta especialmente útil para desarrolladores de plugins de IDA Pro e ingenieros de ingeniería inversa que trabajan con las API de microcódigo de Hex-Rays para extensiones personalizadas del descompilador.
