Questo progetto è uno script e un plugin IDAPython che visualizza la rappresentazione intermedia in microcodice del decompilatore Hex-Rays. Consente l'ispezione visuale del microcodice generato durante la decompilazione, aiutando gli sviluppatori a comprendere e a eseguire il debug dello sviluppo di plugin microcodice Hex-Rays. È principalmente utile per gli sviluppatori di plugin IDA Pro e per gli reverse engineer che lavorano con le API microcodice Hex-Rays per estensioni personalizzate del decompilatore.
