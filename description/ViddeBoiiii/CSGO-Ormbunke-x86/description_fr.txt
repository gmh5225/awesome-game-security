Ce projet est une base de cheat interne simple pour CS:GO, avec un menu ImGui et ciblant les builds x86. L’implémentation est principalement en C/C++ et intègre le rendu DirectX9, les hooks Kiero et l’interception basée sur MinHook. Les fonctionnalités incluses couvrent des fonctions de cheat courantes telles que l’ESP, la logique d’aimbot, les assists de mouvement et le comportement de trigger. Il est principalement destiné à servir de point de départ éducatif pour des expériences de game hacking et la personnalisation d’un framework de menu.
