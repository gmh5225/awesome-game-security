Este proyecto es una base sencilla de cheat interno para CS:GO con un menú ImGui y orientada a builds x86. La implementación es principalmente C/C++ e integra renderizado DirectX9, hooks de Kiero e intercepción basada en MinHook. Las funciones incluidas cubren cheats habituales como ESP, lógica de aimbot, asistencias de movimiento y comportamiento de trigger. Está pensado principalmente como punto de partida educativo para experimentos de game hacking y la personalización de un framework de menú.
