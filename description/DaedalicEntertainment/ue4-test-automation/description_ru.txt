Daedalic Test Automation — плагин для Unreal Engine 4 для создания и запуска автоматизированных интеграционных тестов. Он расширяет рабочий процесс тестирования на базе Gauntlet утверждениями, задержками, имитацией ввода, проверками по триггерам, параметризованными тестами и сценариями, ориентированными на производительность. Проект написан в основном на C++ и предназначен для повышения покрытия, ускорения обнаружения регрессий и поддержки автономного запуска тестов в конвейерах разработки. Он ориентирован на игровые команды, которым нужна надёжная QA-автоматизация и повторяемая проверка игрового поведения.
