Daedalic Test Automation ist ein Unreal-Engine-4-Plugin zum Erstellen und Ausführen automatisierter Integrationstests. Es erweitert den Gauntlet-basierten Testablauf um Assertions, Verzögerungen, simulierte Eingaben, triggergesteuerte Prüfungen, parametrisierte Tests und leistungsorientierte Szenarien. Das Projekt ist hauptsächlich in C++ geschrieben und soll die Abdeckung verbessern, die Erkennung von Regressionen beschleunigen und unbeaufsichtigte Testausführung in Entwicklungspipelines unterstützen. Es richtet sich an Spieleteams, die zuverlässige QA-Automatisierung und wiederholbare Validierung von Gameplay-Verhalten wollen.
