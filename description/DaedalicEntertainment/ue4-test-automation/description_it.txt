Daedalic Test Automation è un plugin per Unreal Engine 4 per creare ed eseguire test di integrazione automatizzati. Estende il flusso di test basato su Gauntlet con assertion, ritardi, input simulato, controlli guidati da trigger, test parametrizzati e scenari orientati alle prestazioni. Il progetto è scritto principalmente in C++ ed è progettato per migliorare la copertura, accelerare il rilevamento delle regressioni e supportare l'esecuzione di test non presidiati nelle pipeline di sviluppo. Si rivolge ai team di gioco che desiderano un'automazione QA affidabile e una convalida ripetibile del comportamento di gameplay.
