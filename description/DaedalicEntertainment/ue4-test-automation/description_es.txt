Daedalic Test Automation es un plugin de Unreal Engine 4 para crear y ejecutar pruebas de integración automatizadas. Amplía el flujo de pruebas basado en Gauntlet con aserciones, retardos, entrada simulada, comprobaciones impulsadas por disparadores, pruebas parametrizadas y escenarios orientados al rendimiento. El proyecto está escrito principalmente en C++ y está diseñado para mejorar la cobertura, acelerar la detección de regresiones y permitir la ejecución desatendida de pruebas en pipelines de desarrollo. Está dirigido a equipos de juegos que buscan una automatización de QA fiable y una validación repetible del comportamiento del gameplay.
