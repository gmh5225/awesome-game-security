Source 2 エンジンのゲーム（Deadlock、CS2、Dota 2）向けのランタイム schema および RTTI 抽出ツールで、実行中のゲームプロセスからクラス階層、netvars、インターフェース、protobuf 定義、グローバルシングルトンをダンプします。
パターンスキャン、MSVC x64 の RTTI 階層再構築、リアルタイム内省用の WebSocket ベースライブブリッジ、およびビジュアル schema ブラウザを備えています。
主にゲームセキュリティ研究者やモデダーが、Source 2 エンジン内部のリバースエンジニアリングや Valve 製ゲームからの SDK 定義抽出に役立ちます。
