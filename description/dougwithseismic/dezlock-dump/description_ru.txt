Инструмент извлечения runtime-схемы и RTTI для игр на движке Source 2 (Deadlock, CS2, Dota 2), который дампит иерархии классов, netvars, интерфейсы, определения protobuf и глобальные синглтоны из живых игровых процессов.
Включает поиск по сигнатурам (pattern scanning), реконструкцию иерархий RTTI MSVC x64, live-мост на WebSocket для интроспекции в реальном времени и визуальный браузер схемы.
В основном полезен исследователям игровой безопасности и моддерам, занимающимся реверс-инжинирингом внутренностей движка Source 2 и извлечением SDK-определений из игр Valve.
