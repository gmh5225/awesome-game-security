Source 2 엔진 게임(Deadlock, CS2, Dota 2)을 위한 런타임 schema 및 RTTI 추출 도구로, 실행 중인 게임 프로세스에서 클래스 계층 구조, netvars, 인터페이스, protobuf 정의, 전역 싱글톤을 덤프합니다.
패턴 스캐닝, MSVC x64 RTTI 계층 재구성, 실시간 인트로스펙션을 위한 WebSocket 기반 라이브 브리지, 시각적 schema 브라우저를 포함합니다.
주로 게임 보안 연구자와 모더가 Source 2 엔진 내부를 리버스 엔지니어링하고 Valve 게임에서 SDK 정의를 추출하는 데 유용합니다.
