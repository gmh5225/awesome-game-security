Questo progetto è PE-bear, un visualizzatore ed editor di file Portable Executable (PE) con interfaccia grafica basata su Qt. Mostra informazioni dettagliate sulla struttura PE, inclusi header DOS/NT, tabelle delle sezioni, directory di import/export, risorse, relocation, directory di debug, TLS e metadati CLR. Lo strumento supporta l'editing esadecimale, la manipolazione delle sezioni, l'estrazione dell'overlay e il confronto affiancato di più file PE. È rivolto ad analisti di malware, reverse engineer e ricercatori di sicurezza che effettuano analisi e modifiche manuali di file PE.
