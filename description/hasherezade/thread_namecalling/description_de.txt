Dieses Projekt konzentriert sich auf setThreadDescription.
Anschließend wird GetThreadDescription per APC remote auf dem Ziel aufgerufen, wodurch der Beschreibungs-Puffer in das Working Set des Ziels kopiert wird.
Es ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich cheat / injection:windows untersuchen.
