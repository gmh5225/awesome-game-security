Этот проект посвящён setThreadDescription.
Затем функция GetThreadDescription вызывается удалённо на целевом процессе через APC, из‑за чего буфер описания копируется в рабочий набор (working set) цели.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / injection:windows.
