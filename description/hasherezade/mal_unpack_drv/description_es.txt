Este proyecto se centra en ADVERTENCIA: esta es una versión experimental; úsela únicamente en una máquina virtual.
El controlador está firmado con una firma de prueba; para que la instalación se realice correctamente, Test Signing debe estar habilitado en la máquina de destino.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / desempaquetado de muestras.
