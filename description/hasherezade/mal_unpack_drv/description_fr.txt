Ce projet se concentre sur AVERTISSEMENT : il s'agit d'une version expérimentale, utilisez-la uniquement sur une machine virtuelle !
Le pilote est signé avec une signature de test ; pour que l'installation réussisse, Test Signing doit être activé sur la machine cible.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine de l'anti-cheat / du dépaquetage d'échantillons.
