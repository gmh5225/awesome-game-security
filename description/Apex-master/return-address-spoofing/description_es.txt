Este proyecto es una implementación compacta de spoofing de dirección de retorno para escenarios de evasión en la invocación de funciones nativas. Usa plantillas C++ y stubs de ensamblador x64 para redirigir el flujo de control preservando el comportamiento de llamada esperado. El código es ligero y está pensado para integrarse en una pipeline de invoker existente. Su público principal son investigadores de seguridad de juegos de bajo nivel que exploran técnicas de evasión anti-detección.
