Questo progetto è un'implementazione compatta di spoofing dell'indirizzo di ritorno per scenari di bypass nell'invocazione di funzioni native. Usa template C++ e stub in assembly x64 per redirigere il flusso di controllo preservando il comportamento di chiamata atteso. Il codice è leggero e pensato per essere integrato in una pipeline invoker esistente. Il pubblico principale sono i ricercatori di sicurezza dei giochi a basso livello che esplorano tecniche di evasione anti-rilevamento.
