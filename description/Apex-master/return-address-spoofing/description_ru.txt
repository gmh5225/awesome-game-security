Этот проект — компактная реализация подмены адреса возврата для сценариев обхода вызова нативных функций. Он использует шаблоны C++ и заглушки на x64-ассемблере, чтобы перенаправлять поток управления, сохраняя ожидаемое поведение вызова. Код лёгкий и рассчитан на встраивание в существующий pipeline invoker. Основная аудитория — исследователи безопасности игр низкого уровня, изучающие техники обхода антидетекта.
