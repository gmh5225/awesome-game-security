Dieses Projekt ist eine kompakte Return-Address-Spoofing-Implementierung für Szenarien zum Umgehen nativer Funktionsaufrufe. Es nutzt C++-Templates und x64-Assembler-Stubs, um den Kontrollfluss umzuleiten und dabei das erwartete Aufrufverhalten beizubehalten. Der Code ist leichtgewichtig und zur Integration in eine bestehende Invoker-Pipeline gedacht. Die Hauptzielgruppe sind Low-Level-Gamesicherheitsforscher, die Anti-Detection-Evasion-Techniken untersuchen.
