Ce projet est une implémentation compacte de spoofing d'adresse de retour pour les scénarios de contournement d'invocation de fonctions natives. Il utilise des templates C++ et des stubs assembleur x64 pour rediriger le flux de contrôle tout en préservant le comportement d'appel attendu. Le code est léger et destiné à être intégré dans un pipeline d'invoker existant. Son public principal est les chercheurs en sécurité des jeux de bas niveau qui explorent les techniques d'évasion anti-détection.
