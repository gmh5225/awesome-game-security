Этот проект представляет собой пару компонентов ядра Linux и пользовательского режима для чтения и записи памяти других процессов. Он реализован на C и C++ со скриптами сборки модуля ядра и предоставляет через интерфейс устройства такие операции, как поиск базового адреса, RPM и WPM. Репозиторий также включает пользовательские примеры и тестовые программы, демонстрирующие обмен с драйвером. Предназначен для низкоуровневых инструментов работы с игровой памятью и исследований безопасности Linux.
