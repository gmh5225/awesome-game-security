Este proyecto es un par de componentes de núcleo Linux y modo de usuario para leer y escribir memoria en otros procesos. Está implementado en C y C++ con scripts de compilación del módulo del núcleo, y expone operaciones como búsqueda de dirección base, RPM y WPM a través de una interfaz de dispositivo. El repositorio también incluye ejemplos y programas de prueba en modo de usuario para demostrar la comunicación con el controlador. Está pensado para herramientas de memoria de juegos de bajo nivel e investigación de seguridad en Linux.
