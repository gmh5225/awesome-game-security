Questo progetto è una coppia di componenti kernel Linux e user-mode per leggere e scrivere la memoria di altri processi. È implementato in C e C++ con script di build del modulo kernel, ed espone operazioni come ricerca dell'indirizzo base, RPM e WPM tramite un'interfaccia dispositivo. Il repository include anche esempi e programmi di test in user-mode per dimostrare la comunicazione con il driver. È pensato per tooling di memoria di gioco a basso livello e ricerca sulla sicurezza Linux.
