Ce projet est une reconstruction par rétro-ingénierie d'un célèbre cheat FPS historique. Il est principalement écrit en C++ et documente un flux de décompilation basé sur des bases de données IDA, avec une réimplémentation progressive des modules d'origine. Le code et les notes suivent l'avancement de fonctionnalités telles que glow ESP, radar, no-flash, bunny hop, ainsi que des utilitaires mémoire/signature associés. Il s'adresse aux apprenants qui étudient l'intérieur des cheats, les motifs d'obfuscation de type malware et la rétro-ingénierie pratique.
