Dieses Projekt konzentriert sich auf Compile-Time Regular Expressions in C++. Wenn Sie CMake verwenden, können Sie dieses Verzeichnis als Unterverzeichnis hinzufügen und gegen das Target ctre linken. Es ist vor allem für Anti-Cheat-Ingenieure und defensive Sicherheitsforscher nützlich, die im Bereich Anti-Cheat / Compile-Time arbeiten.
