Este proyecto se centra en la implementación constexpr de las familias de hashes SHA-2 y SHA-3.
La biblioteca también implementa literales hash_value en el espacio de nombres cthash::literals (sufijos entre paréntesis para cada tipo de función hash).
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el ámbito del anti-cheat / tiempo de compilación.
