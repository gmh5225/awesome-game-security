Este proyecto es un framework de depuración basado en el Vectored Exception Handler (VEH) de Windows que implementa un depurador de software usando VEH en lugar de la API Debug. Configura controladores de excepciones para interceptar puntos de interrupción, excepciones de paso a paso y violaciones de acceso, ofreciendo funcionalidad similar a un depurador sin activar los métodos estándar de detección de depuradores. Está dirigido a investigadores de seguridad e ingenieros de ingeniería inversa que necesitan depurar procesos que detectan y bloquean activamente los depuradores convencionales.
