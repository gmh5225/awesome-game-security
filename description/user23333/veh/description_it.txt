Questo progetto è un framework di debugging basato sul Vectored Exception Handler (VEH) di Windows che implementa un debugger software usando VEH al posto della Debug API. Imposta gestori di eccezioni per intercettare breakpoint, eccezioni di single-step e violazioni di accesso, fornendo funzionalità simili a un debugger senza attivare i metodi standard di rilevamento dei debugger. È rivolto a ricercatori di sicurezza e reverse engineer che devono fare il debug di processi che rilevano e bloccano attivamente i debugger convenzionali.
