Этот проект — отладочный фреймворк на основе Windows Vectored Exception Handler (VEH), реализующий программный отладчик с помощью VEH вместо Debug API. Он устанавливает обработчики исключений для перехвата точек останова, исключений пошагового выполнения и нарушений доступа, обеспечивая функциональность, сходную с отладчиком, без срабатывания стандартных методов обнаружения отладчиков. Предназначен для исследователей безопасности и специалистов по реверс-инжинирингу, которым нужно отлаживать процессы, активно обнаруживающие и блокирующие обычные отладчики.
