Dieses Projekt ist ein auf dem Windows Vectored Exception Handler (VEH) basierendes Debugging-Framework, das einen Software-Debugger mittels VEH anstelle der Debug-API implementiert. Es richtet Exception-Handler ein, um Breakpoints, Single-Step-Exceptions und Zugriffsverletzungen abzufangen, und bietet so debuggerähnliche Funktionalität, ohne gängige Debugger-Erkennungsmethoden auszulösen. Es richtet sich an Sicherheitsforscher und Reverse Engineers, die Prozesse debuggen müssen, die herkömmliche Debugger aktiv erkennen und blockieren.
