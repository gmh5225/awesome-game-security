Questo progetto è un toolkit di ricerca sul kernel Windows incentrato sulla disabilitazione del comportamento di PatchGuard a runtime. È implementato in C, C++ e assembly per i target di build x86 e x64, con script per i flussi di lavoro Visual Studio e NMAKE. Il repository include componenti driver e loader e fa riferimento a metodi di caricamento assistiti dalla virtualizzazione. Il pubblico principale è costituito da ricercatori di sicurezza a basso livello che indagano le difese del kernel e i meccanismi di bypass di PatchGuard.
