Este proyecto es un kit de herramientas de investigación del kernel de Windows centrado en deshabilitar el comportamiento de PatchGuard en tiempo de ejecución. Está implementado en C, C++ y ensamblador para destinos de compilación x86 y x64, con scripts para flujos de trabajo de Visual Studio y NMAKE. El repositorio incluye componentes de controlador y cargador, y hace referencia a métodos de carga asistidos por virtualización. Su público principal son investigadores de seguridad de bajo nivel que estudian las defensas del kernel y los mecanismos de bypass de PatchGuard.
