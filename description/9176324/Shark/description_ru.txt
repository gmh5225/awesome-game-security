Этот проект — инструментарий для исследования ядра Windows, ориентированный на отключение поведения PatchGuard во время выполнения. Он реализован на C, C++ и ассемблере для целей сборки x86 и x64 и включает скрипты для рабочих процессов Visual Studio и NMAKE. Репозиторий содержит компоненты драйвера и загрузчика, а также упоминает методы загрузки с поддержкой виртуализации. Основная аудитория — исследователи безопасности низкого уровня, изучающие защиту ядра и механизмы обхода PatchGuard.
