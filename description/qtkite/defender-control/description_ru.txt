Этот проект — GUI-утилита на C# для управления защитой в реальном времени Windows Defender. Она предоставляет простой интерфейс с переключателями для включения или отключения мониторинга в реальном времени Defender, параметров защиты от несанкционированного изменения и автоматической отправки образцов, без необходимости переходить в параметры безопасности Windows. Инструмент изменяет соответствующие ключи реестра и конфигурации служб. Он предназначен для разработчиков и исследователей безопасности, которым нужно временно отключать Defender во время тестирования, анализа вредоносного ПО или разработки программного обеспечения.
