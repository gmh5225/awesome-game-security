Este proyecto es una utilidad GUI en C# para controlar la protección en tiempo real de Windows Defender. Proporciona una interfaz de conmutación sencilla para activar o desactivar la supervisión en tiempo real de Defender, la configuración de protección contra alteraciones y el envío automático de muestras, sin navegar por la configuración de Seguridad de Windows. La herramienta modifica las claves del Registro y las configuraciones de servicio pertinentes. Está dirigida a desarrolladores e investigadores de seguridad que necesitan desactivar temporalmente Defender durante pruebas, análisis de malware o desarrollo de software.
