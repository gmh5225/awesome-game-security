Этот проект — плагин для Ghidra, интегрирующий модели GPT от OpenAI для автоматизированной помощи в реверс-инжиниринге. Он отправляет декомпилированный код функций в OpenAI API и отображает сгенерированный ИИ анализ прямо в интерфейсе Ghidra: пояснение назначения функций, предложения по переименованию переменных и выявление уязвимостей. Плагин на Python/Java упрощает анализ бинарных файлов за счёт подсказок на базе LLM. Он предназначен для реверс-инженеров, использующих Ghidra и желающих получать AI-помощь в понимании и аннотировании кода.
