Ce projet est un plugin Ghidra qui intègre les modèles GPT d'OpenAI pour une assistance automatisée à la rétro-ingénierie. Il envoie le code de fonctions décompilées à l'API OpenAI et affiche directement dans l'interface de Ghidra une analyse générée par l'IA, notamment l'explication du rôle des fonctions, des suggestions de renommage de variables et l'identification de vulnérabilités. Ce plugin Python/Java simplifie l'analyse binaire grâce aux apports d'un LLM. Il s'adresse aux ingénieurs en rétro-ingénierie utilisant Ghidra qui souhaitent une compréhension et une annotation de code assistées par l'IA.
