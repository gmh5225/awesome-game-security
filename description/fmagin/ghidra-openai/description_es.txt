Este proyecto es un complemento de Ghidra que integra los modelos GPT de OpenAI para asistencia automatizada en ingeniería inversa. Envía el código de funciones descompiladas a la API de OpenAI y muestra dentro de la interfaz de Ghidra un análisis generado por IA, que incluye la explicación del propósito de las funciones, sugerencias de renombrado de variables e identificación de vulnerabilidades. El complemento en Python/Java agiliza el análisis binario con perspectivas impulsadas por LLM. Está dirigido a ingenieros inversos que usan Ghidra y desean comprensión y anotación de código asistidas por IA.
