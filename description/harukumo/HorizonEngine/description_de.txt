Dieses Projekt konzentriert sich auf eine 3D-Rendering-Engine.
Horizon Engine wird derzeit nur unter Windows unterstützt und zielt ausschließlich auf moderne Grafik-APIs ab (Direct3D 12, Vulkan, Metal).
Es ist vor allem für Spieleentwickler, Engine-Programmierer und Grafikforscher nützlich, die im Bereich Game Engine / Source arbeiten.
