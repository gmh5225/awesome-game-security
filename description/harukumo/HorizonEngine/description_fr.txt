Ce projet se concentre sur un moteur de rendu 3D.
Horizon Engine n'est actuellement pris en charge que sous Windows et ne cible que les API graphiques modernes (Direct3D 12, Vulkan, Metal).
Il est principalement utile aux développeurs de jeux, aux programmeurs de moteurs et aux chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
