Этот проект посвящён 3D-движку рендеринга.
Horizon Engine в настоящее время поддерживается только в Windows и ориентирован исключительно на современные графические API (Direct3D 12, Vulkan, Metal).
Он в основном полезен разработчикам игр, программистам движков и исследователям графики, работающим в области игровых движков / исходного кода.
