Questo progetto si concentra su un motore di rendering 3D.
Horizon Engine è attualmente supportato solo su Windows e punta esclusivamente alle API grafiche moderne (Direct3D 12, Vulkan, Metal).
È principalmente utile per sviluppatori di giochi, programmatori di engine e ricercatori di grafica che lavorano nell'area dei motori di gioco / del codice sorgente.
