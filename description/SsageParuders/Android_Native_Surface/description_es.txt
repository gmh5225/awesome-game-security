Este proyecto proporciona una biblioteca de renderizado de surface nativa para Android que crea surfaces de superposición sin requerir permisos a nivel de aplicación. Incluye código fuente compatible con AOSP para Android 10 a 14, con soporte para renderizado de surface y grabación de pantalla mediante las API nativas de Android. Resulta especialmente útil para investigadores de seguridad de juegos Android y desarrolladores de overlays que estudian técnicas de creación y dibujo de surfaces nativas en distintas versiones de Android.
