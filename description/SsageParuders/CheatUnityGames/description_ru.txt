Этот проект — шаблон нативной разработки под Android для экспериментов с читами в играх на Unity.
Он построен вокруг рабочих процессов C и C++ NDK в VSCode и включает скрипты для сборки shared objects и нативных бинарников.
В кодовой базе интегрированы типичные компоненты мобильного game hacking, такие как Il2CppResolver и Dobby, для анализа во время выполнения и хукинга.
Основной сценарий использования — изучение техник Android-инъекции и манипуляций на нативном уровне для целей на Unity.
