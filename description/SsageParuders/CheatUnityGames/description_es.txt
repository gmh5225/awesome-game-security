Este proyecto es una plantilla de desarrollo nativo para Android orientada a la experimentación con cheats en juegos Unity.
Está construido en torno a flujos de trabajo NDK de C y C++ en VSCode e incluye scripts para compilar objetos compartidos y binarios nativos.
La base de código integra componentes habituales del hacking de juegos móviles, como Il2CppResolver y Dobby, para análisis en tiempo de ejecución y hooking.
Su caso de uso principal es el estudio de técnicas de inyección en Android y manipulación de la capa nativa en objetivos Unity.
