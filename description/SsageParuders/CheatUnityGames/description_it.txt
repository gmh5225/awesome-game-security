Questo progetto è un template di sviluppo nativo Android pensato per sperimentare cheat su giochi Unity.
È basato su workflow NDK in C e C++ in VSCode e include script per compilare shared object e binari nativi.
Il codice integra componenti comuni del game hacking mobile, come Il2CppResolver e Dobby, per l'analisi runtime e l'hooking.
Il caso d'uso principale è lo studio di tecniche di injection Android e di manipolazione del layer nativo su target Unity.
