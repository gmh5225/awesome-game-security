Shard is a free, open-source, AI-powered anti-cheat plugin for Minecraft servers. Written primarily in Kotlin and built with Gradle, it targets Paper and Folia servers and uses PacketEvents for packet-level analysis. Its core detection path sends player tick data to a remote inference API for AI-based cheating checks, with optional SQLite, MySQL, or MariaDB storage and Redis-backed cross-server alerts. Operators get monitoring, profiling, violation history, punishment rules, WorldGuard region exemptions, and Geyser-related integrations. It is aimed at Minecraft server admins who want modern anti-cheat tooling with AI-assisted detection rather than only heuristic packet checks.
