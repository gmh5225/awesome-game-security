Shard è un plugin anti-cheat gratuito, open source e basato sull’IA per server Minecraft. Scritto principalmente in Kotlin e costruito con Gradle, è pensato per server Paper e Folia e usa PacketEvents per l’analisi a livello di pacchetti. Il percorso di rilevamento centrale invia i dati di tick dei giocatori a un’API remota di inferenza per controlli anti-cheat basati sull’IA, con archiviazione opzionale SQLite, MySQL o MariaDB e avvisi cross-server supportati da Redis. Gli operatori dispongono di monitoraggio, profiling, cronologia delle violazioni, regole di punizione, esenzioni per regioni WorldGuard e integrazioni legate a Geyser. È rivolto agli amministratori di server Minecraft che vogliono strumenti anti-cheat moderni con rilevamento assistito dall’IA, piuttosto che soli controlli euristici sui pacchetti.
