Shard — бесплатный open-source античит-плагин с поддержкой ИИ для серверов Minecraft. Написан в основном на Kotlin и собирается с помощью Gradle, ориентирован на серверы Paper и Folia и использует PacketEvents для анализа на уровне пакетов. Основной путь обнаружения отправляет данные тиков игроков во внешний inference API для ИИ-проверок на читерство, с опциональным хранением в SQLite, MySQL или MariaDB и межсерверными оповещениями на базе Redis. Операторы получают мониторинг, профилирование, историю нарушений, правила наказаний, исключения для регионов WorldGuard и интеграции, связанные с Geyser. Плагин рассчитан на администраторов серверов Minecraft, которым нужны современные античит-инструменты с ИИ-детекцией, а не только эвристические проверки пакетов.
