Shard は、Minecraft サーバー向けの無料・オープンソースの AI 駆動アンチチートプラグインです。主に Kotlin で書かれ、Gradle でビルドされ、Paper および Folia サーバーを対象とし、PacketEvents によるパケットレベルの解析を行います。中核の検出経路はプレイヤーの tick データをリモート推論 API に送信し、AI ベースのチート検査を行います。任意で SQLite、MySQL、または MariaDB による保存と、Redis によるクロスサーバーアラートにも対応します。運用者は監視、プロファイリング、違反履歴、処罰ルール、WorldGuard リージョンの免除、Geyser 関連の連携機能を利用できます。ヒューリスティックなパケット検査だけでなく、AI 支援の検出を備えた現代的なアンチチートを求める Minecraft サーバー管理者向けです。
