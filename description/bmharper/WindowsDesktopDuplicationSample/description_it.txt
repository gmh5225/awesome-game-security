Questo progetto è un’applicazione di esempio che dimostra l’API Windows Desktop Duplication (DXGI Output Duplication) per catturare il contenuto dello schermo del desktop. Mostra come acquisire frame dal compositor desktop, accedere ai dati dei pixel, gestire il rendering del cursore e rilevare le regioni dirty/moved per una cattura dello schermo efficiente. L’esempio in C++ usa DirectX 11 e DXGI 1.2. È rivolto agli sviluppatori che realizzano strumenti di cattura dello schermo, applicazioni di desktop remoto o sistemi anti-cheat basati su screenshot.
