Dieses Projekt ist eine Beispielanwendung, die die Windows Desktop Duplication API (DXGI Output Duplication) zur Erfassung des Desktop-Bildschirminhalts demonstriert. Es zeigt, wie Frames vom Desktop-Compositor abgerufen, Pixeldaten gelesen, die Cursor-Darstellung verarbeitet und dirty/moved-Regionen für eine effiziente Bildschirmerfassung erkannt werden. Das C++-Beispiel nutzt DirectX 11 und DXGI 1.2. Es richtet sich an Entwickler von Bildschirmcapture-Tools, Remotedesktop-Anwendungen oder screenshotbasierten Anti-Cheat-Systemen.
