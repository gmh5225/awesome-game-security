Этот проект — пример приложения, демонстрирующего Windows Desktop Duplication API (DXGI Output Duplication) для захвата содержимого экрана рабочего стола. Он показывает, как получать кадры от композитора рабочего стола, обращаться к пиксельным данным, обрабатывать отрисовку курсора и определять dirty/moved-области для эффективного захвата экрана. Пример на C++ использует DirectX 11 и DXGI 1.2. Предназначен для разработчиков средств захвата экрана, приложений удалённого рабочего стола или античит-систем на основе скриншотов.
