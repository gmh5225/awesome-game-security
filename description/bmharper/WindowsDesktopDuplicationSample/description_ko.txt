이 프로젝트는 Windows Desktop Duplication API(DXGI Output Duplication)로 데스크톱 화면을 캡처하는 방법을 보여주는 샘플 애플리케이션입니다. 데스크톱 컴포지터에서 프레임을 획득하고, 픽셀 데이터에 접근하며, 커서 렌더링을 처리하고, 효율적인 화면 캡처를 위해 dirty/moved 영역을 감지하는 방법을 설명합니다. C++ 샘플은 DirectX 11과 DXGI 1.2를 사용합니다. 화면 캡처 도구, 원격 데스크톱 애플리케이션, 또는 스크린샷 기반 안티치트 시스템을 구축하는 개발자를 대상으로 합니다.
