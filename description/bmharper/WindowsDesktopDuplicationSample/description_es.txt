Este proyecto es una aplicación de ejemplo que demuestra la API Windows Desktop Duplication (DXGI Output Duplication) para capturar el contenido de la pantalla del escritorio. Muestra cómo adquirir fotogramas del compositor de escritorio, acceder a los datos de píxeles, gestionar el renderizado del cursor y detectar regiones dirty/moved para una captura de pantalla eficiente. El ejemplo en C++ usa DirectX 11 y DXGI 1.2. Está dirigido a desarrolladores que crean herramientas de captura de pantalla, aplicaciones de escritorio remoto o sistemas anticheat basados en capturas de pantalla.
