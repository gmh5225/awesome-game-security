本プロジェクトは Eclipse Paho MQTT C++ クライアントライブラリであり、MQTT メッセージングプロトコル通信向けのモダンな C++ API を提供します。MQTT v3.1、v3.1.1、v5.0 をサポートし、QoS レベル 0–2、永続セッション、TLS 暗号化、自動再接続、非同期メッセージ配信などの機能を備えています。このライブラリは Paho C クライアントを RAII スタイルのリソース管理とコールバックベースのイベント処理でラップしています。C++ アプリケーションに MQTT の pub/sub メッセージングを組み込む IoT 開発者およびシステムプログラマ向けです。
