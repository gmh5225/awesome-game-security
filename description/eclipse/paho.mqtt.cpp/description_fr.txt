Ce projet est la bibliothèque cliente Eclipse Paho MQTT C++, offrant une API C++ moderne pour la communication via le protocole de messagerie MQTT. Elle prend en charge MQTT v3.1, v3.1.1 et v5.0, avec des fonctionnalités telles que les niveaux QoS 0–2, les sessions persistantes, le chiffrement TLS, la reconnexion automatique et la livraison asynchrone des messages. La bibliothèque encapsule le client Paho C avec une gestion des ressources de style RAII et une gestion d'événements basée sur des callbacks. Elle s'adresse aux développeurs IoT et aux programmeurs système qui intègrent la messagerie MQTT pub/sub dans des applications C++.
