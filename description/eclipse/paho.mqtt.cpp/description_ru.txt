Этот проект — клиентская библиотека Eclipse Paho MQTT C++, предоставляющая современный C++ API для обмена сообщениями по протоколу MQTT. Она поддерживает MQTT v3.1, v3.1.1 и v5.0 и включает уровни QoS 0–2, постоянные сессии, шифрование TLS, автоматическое переподключение и асинхронную доставку сообщений. Библиотека оборачивает клиент Paho C с управлением ресурсами в стиле RAII и обработкой событий на основе колбэков. Она предназначена для IoT-разработчиков и системных программистов, интегрирующих MQTT pub/sub-обмен сообщениями в приложения на C++.
