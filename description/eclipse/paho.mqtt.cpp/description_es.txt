Este proyecto es la biblioteca cliente Eclipse Paho MQTT C++, que ofrece una API C++ moderna para la comunicación con el protocolo de mensajería MQTT. Admite MQTT v3.1, v3.1.1 y v5.0, con funciones como niveles QoS 0–2, sesiones persistentes, cifrado TLS, reconexión automática y entrega asíncrona de mensajes. La biblioteca envuelve el cliente Paho C con gestión de recursos al estilo RAII y manejo de eventos basado en callbacks. Está orientada a desarrolladores IoT y programadores de sistemas que integran mensajería MQTT pub/sub en aplicaciones C++.
