이 프로젝트는 Eclipse Paho MQTT C++ 클라이언트 라이브러리로, MQTT 메시징 프로토콜 통신을 위한 현대적인 C++ API를 제공합니다. MQTT v3.1, v3.1.1, v5.0을 지원하며 QoS 레벨 0–2, 영속 세션, TLS 암호화, 자동 재연결, 비동기 메시지 전달 등의 기능을 포함합니다. 이 라이브러리는 Paho C 클라이언트를 RAII 스타일 리소스 관리와 콜백 기반 이벤트 처리로 래핑합니다. C++ 애플리케이션에 MQTT pub/sub 메시징을 통합하는 IoT 개발자와 시스템 프로그래머를 대상으로 합니다.
