Обширный курируемый список ресурсов по game hacking, включающий дизассемблеры, отладчики, hex-редакторы, сканеры памяти, .NET-декомпиляторы, графические отладчики и учебные материалы по реверс-инжинирингу, организованные по типам инструментов.
Охватывает Cheat Engine, x64dbg, IDA Pro, Ghidra, dnSpy, RenderDoc и десятки других инструментов, а также руководства по взлому конкретных игр, библиотеки и фреймворки.
В первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, которым нужна централизованная справка по инструментам, техникам и обучающим материалам game hacking.
