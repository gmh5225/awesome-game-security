Eine kuratierte Sammlung von Ressourcen speziell zum Hacking von Online- und Mehrspielerspielen, darunter Reverse Engineering von Netzwerkprotokollen, serverseitige Emulation, Analyse der Paketverschlüsselung und Forschung zu Anti-Cheat-Bypässen.
Sie enthält Papers zur GameGuard-Architektur, zur Echtzeit-Prävention von Map-Hacks, zur Entschlüsselung von MMORPG-Protokollen sowie Vorträge zur Erweiterung proprietärer Spiele um Mehrspielerfunktionen.
Sie ist vor allem für Gamesicherheitsforscher nützlich, die Angriffsflächen von Onlinespielen, die Sicherheit von Netzwerkprotokollen und serverseitige Schutzmechanismen untersuchen.
