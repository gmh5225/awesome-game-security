Ce projet se concentre sur les expressions Mixed Boolean-Arithmetic non linéaires.
Le code de MBA-Obfuscator se trouve dans le dossier "mba_obfuscator", et les échantillons de sortie dans le dossier "samples".
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine des cheats / Mixed Boolean-Arithmetic.
