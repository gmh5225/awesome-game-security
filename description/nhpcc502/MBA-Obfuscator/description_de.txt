Dieses Projekt konzentriert sich auf nichtlineare Mixed Boolean-Arithmetic Expressions.
Der Code von MBA-Obfuscator befindet sich im Ordner "mba_obfuscator", die Ausgabebeispiele im Ordner "samples".
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Mixed Boolean-Arithmetic untersuchen.
