Questo progetto è uno scanner leggero che cerca sul disco driver Windows vulnerabili.
È implementato come una piccola utility da riga di comando in C++ pensata per il triage rapido dei file dei driver.
Lo strumento aiuta a individuare una possibile esposizione bring-your-own-vulnerable-driver nelle posizioni comuni dei driver di sistema.
È utile per la ricerca sul bypass degli anti-cheat, la risposta agli incidenti e gli audit di sicurezza difensiva.
