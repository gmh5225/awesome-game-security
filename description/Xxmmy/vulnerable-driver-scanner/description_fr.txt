Ce projet est un scanner léger qui recherche sur le disque des pilotes Windows vulnérables.
Il est implémenté comme un petit utilitaire en ligne de commande C++ destiné au triage rapide des fichiers de pilotes.
L'outil aide à identifier une exposition potentielle de type bring-your-own-vulnerable-driver dans les emplacements courants des pilotes système.
Il est utile pour la recherche de contournement d'anti-cheat, la réponse aux incidents et les audits de sécurité défensive.
