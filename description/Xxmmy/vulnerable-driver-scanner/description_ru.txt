Этот проект — лёгкий сканер, который ищет на диске уязвимые драйверы Windows.
Он реализован как небольшая утилита командной строки на C++ для быстрого разбора файлов драйверов.
Инструмент помогает выявлять потенциальную экспозицию bring-your-own-vulnerable-driver в типичных каталогах системных драйверов.
Полезен для исследований обхода античита, реагирования на инциденты и оборонительных аудитов безопасности.
