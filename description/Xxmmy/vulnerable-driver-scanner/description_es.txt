Este proyecto es un escáner ligero que busca en el disco controladores de Windows vulnerables.
Está implementado como una pequeña utilidad de línea de comandos en C++ pensada para el triage rápido de archivos de controladores.
La herramienta ayuda a identificar una posible exposición bring-your-own-vulnerable-driver en ubicaciones habituales de controladores del sistema.
Es útil para investigación de bypass de anti-cheat, respuesta a incidentes y auditorías de seguridad defensiva.
