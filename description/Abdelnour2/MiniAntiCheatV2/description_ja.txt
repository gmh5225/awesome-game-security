MiniAntiCheatV2 は、ユーザーモードの「ゲーム」アプリとカーネルドライバを組み合わせ、基本的なプロセス保護を示す小規模な教育用 Windows アンチチートサンプルです。C/C++ で書かれており、ドライバはブラックリスト検査、ゲーム PID の登録、シールド無効化向けの IOCTL インターフェイスを公開し、ゲーム側は DeviceIoControl で通信します。プロセス作成通知ルーチンにより、ゲーム起動時と実行中の両方でブラックリストプロセス（Notepad.exe）をブロックし、さらに ObRegisterCallbacks を使う V2「メモリシールド」を追加して、他のユーザーモードプロセスから VM の読み書きや終了といった機微なハンドル権限を取り除きます。本プロジェクトは本番向けアンチチートではなく、ゲームセキュリティと Windows カーネルアンチチート技術を学ぶ人向けです。
