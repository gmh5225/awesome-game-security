MiniAntiCheatV2 — небольшой учебный пример античита для Windows, в котором пользовательское «игровое» приложение сочетается с драйвером ядра для демонстрации базовой защиты процессов. Написан на C/C++: драйвер предоставляет интерфейсы IOCTL для проверки чёрного списка, регистрации PID игры и отключения щита, а игра взаимодействует с ним через DeviceIoControl. Он блокирует процесс из чёрного списка (Notepad.exe) как при запуске игры, так и во время её работы через процедуры уведомления о создании процессов, а также добавляет V2 «memory shield», который с помощью ObRegisterCallbacks лишает другие пользовательские процессы чувствительных прав на handles — например, чтения/записи виртуальной памяти и завершения процесса. Проект предназначен для изучающих игровую безопасность и техники античита в ядре Windows, а не для промышленного развёртывания античита.
