MiniAntiCheatV2는 사용자 모드 “게임” 앱과 커널 드라이버를 짝지어 기본적인 프로세스 보호를 보여주는 소규모 교육용 Windows 안티치트 샘플입니다. C/C++로 작성되었으며, 드라이버는 블랙리스트 검사, 게임 PID 등록, 실드 비활성화를 위한 IOCTL 인터페이스를 제공하고, 게임은 DeviceIoControl로 통신합니다. 프로세스 생성 알림 루틴을 통해 게임 시작 시와 실행 중 모두에서 블랙리스트 프로세스(Notepad.exe)를 차단하며, ObRegisterCallbacks를 사용하는 V2 “메모리 실드”를 추가해 다른 사용자 모드 프로세스로부터 VM 읽기/쓰기 및 종료와 같은 민감한 핸들 권한을 제거합니다. 이 프로젝트는 프로덕션 안티치트 배포용이 아니라 게임 보안과 Windows 커널 안티치트 기술을 탐구하는 학습자를 위한 것입니다.
