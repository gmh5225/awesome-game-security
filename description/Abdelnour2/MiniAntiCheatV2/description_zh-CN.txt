MiniAntiCheatV2 是一个小型教学用 Windows 反作弊示例，将用户态“游戏”应用与内核驱动配对，用于演示基本的进程保护。项目使用 C/C++ 编写，驱动通过 IOCTL 接口提供黑名单检查、游戏 PID 注册以及护盾关闭等功能，游戏则通过 DeviceIoControl 与驱动通信。它会在游戏启动时以及运行期间，借助进程创建通知例程拦截黑名单进程（Notepad.exe），并新增 V2“内存护盾”，利用 ObRegisterCallbacks 剥夺其他用户态进程对敏感句柄权限（如虚拟内存读写与终止进程）。该项目面向学习游戏安全与 Windows 内核反作弊技术的读者，而非用于生产环境反作弊部署。
