MiniAntiCheatV2 是一個小型教學用 Windows 反作弊範例，將使用者模式「遊戲」應用程式與核心驅動程式搭配，用以示範基本的行程保護。專案以 C/C++ 撰寫，驅動程式透過 IOCTL 介面提供黑名單檢查、遊戲 PID 註冊以及護盾停用等功能，遊戲則透過 DeviceIoControl 與其通訊。它會在遊戲啟動時以及執行期間，藉由行程建立通知常式封鎖黑名單行程（Notepad.exe），並新增 V2「記憶體護盾」，利用 ObRegisterCallbacks 剝奪其他使用者模式行程的敏感控制碼權限（例如虛擬記憶體讀寫與終止行程）。此專案面向探索遊戲安全與 Windows 核心反作弊技術的學習者，而非用於正式環境的反作弊部署。
