MiniAntiCheatV2 es una pequeña muestra educativa de anti-cheat para Windows que combina una aplicación «juego» en modo usuario con un controlador de kernel para demostrar una protección básica de procesos. Escrito en C/C++, el controlador expone interfaces IOCTL para comprobaciones de lista negra, registro del PID del juego y desactivación del escudo, mientras el juego se comunica mediante DeviceIoControl. Bloquea un proceso en lista negra (Notepad.exe) tanto al iniciar el juego como mientras se ejecuta mediante rutinas de notificación de creación de procesos, y añade un «escudo de memoria» V2 que usa ObRegisterCallbacks para quitar derechos sensibles de handles, como lectura/escritura de memoria virtual y terminación, a otros procesos en modo usuario. El proyecto está orientado a quienes aprenden seguridad en juegos y técnicas de anti-cheat en el kernel de Windows, no a un despliegue de anti-cheat en producción.
