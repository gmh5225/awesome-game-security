サーバー検証済みのゲーム時間と軽量なランタイム整合性チェックを提供する Unity UPM パッケージ（C#）で、ゲームプレイが端末の時計だけを信頼しないようにします。GameTime は公開時刻 API のフォールバック連鎖から UTC を取得し、取得の合間はシステム時計を再読みせず Time.deltaTime で進め、一定間隔およびアプリがフォーカスを取り戻したときに再検証します。IntegrityChecker は、デバッガのアタッチ、Android/iOS の root やジェイルブレイク経路、Android エミュレータ、検証済み時刻に対する時計ドリフトをヒューリスティックに検査します。他の SiPVLib モジュールや UniTask と統合でき、報酬、クールダウン、日次リセットなど不正に影響されやすいロジック向けのソフトなアンチチート信号を必要とする Unity 開発者向けです。
