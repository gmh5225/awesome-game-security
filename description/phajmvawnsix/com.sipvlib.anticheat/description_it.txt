Un pacchetto Unity UPM (C#) che fornisce tempo di gioco verificato lato server e controlli di integrità runtime leggeri, così il gameplay non si affida solo all’orologio del dispositivo. GameTime ottiene l’UTC da una catena di fallback di API pubbliche del tempo, avanza tra i fetch con Time.deltaTime invece di rileggere l’orologio di sistema e riverifica a intervalli e quando l’app riprende il focus. IntegrityChecker esegue controlli euristici per un debugger collegato, percorsi di root o jailbreak Android/iOS, emulatori Android e deriva dell’orologio rispetto al tempo verificato. Si integra con altri moduli SiPVLib e UniTask ed è pensato per sviluppatori Unity che necessitano di segnali soft anti-cheat per ricompense, cooldown, reset giornalieri e logica simile sensibile ai cheat.
