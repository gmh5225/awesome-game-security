一個 Unity UPM 套件（C#），提供經伺服器驗證的遊戲時間與輕量級執行時期完整性檢查，使玩法邏輯不再僅依賴裝置時鐘。GameTime 透過一組公開時間 API 的備援鏈取得 UTC，並在兩次擷取之間以 Time.deltaTime 推進時間，而非反覆讀取系統時鐘；也會依間隔以及在應用程式重新取得焦點時再次驗證。IntegrityChecker 以啟發式方式檢查是否附加除錯器、Android/iOS 的 root 或越獄路徑、Android 模擬器，以及相對已驗證時間的時鐘漂移。它可與其他 SiPVLib 模組及 UniTask 整合，面向需要為獎勵、冷卻、每日重置等易受作弊影響邏輯提供軟性反作弊訊號的 Unity 開發者。
