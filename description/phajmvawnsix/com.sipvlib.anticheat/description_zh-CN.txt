一个 Unity UPM 包（C#），提供经服务器校验的游戏时间与轻量级运行时完整性检查，使玩法逻辑不再仅依赖设备时钟。GameTime 通过一组公共时间 API 的回退链获取 UTC，并在两次拉取之间用 Time.deltaTime 推进时间，而非反复读取系统时钟；还会按间隔以及在应用重新获得焦点时重新校验。IntegrityChecker 以启发式方式检测是否附加调试器、Android/iOS 的 root 或越狱路径、Android 模拟器，以及相对已校验时间的时钟漂移。它可与其他 SiPVLib 模块及 UniTask 集成，面向需要为奖励、冷却、每日重置等易受作弊影响逻辑提供软反作弊信号的 Unity 开发者。
