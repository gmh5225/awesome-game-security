Una lista awesome curata di plugin e script per i principali strumenti di reverse engineering, tra cui IDA Pro, Ghidra, x64dbg, OllyDbg e GDB. Le voci coprono patching, binary diffing, deoffuscazione, emulazione, code coverage, scansione YARA, ricerca di costanti crittografiche, analisi di driver e heap, e ausili anti-anti-debug. Molti progetti elencati si basano su Python o IDAPython e integrano framework come Unicorn, Keystone, angr, oltre ad assistenti di decompilazione supportati da ChatGPT. La raccolta è rivolta a reverse engineer, analisti di malware, sviluppatori di exploit e ricercatori di game security che estendono questi disassembler e debugger per l’analisi binaria e la ricerca di vulnerabilità.
