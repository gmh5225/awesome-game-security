主要なリバースエンジニアリングツール向けのプラグインとスクリプトを集めた厳選 awesome リストで、IDA Pro、Ghidra、x64dbg、OllyDbg、GDB を含みます。パッチ適用、バイナリ差分、難読化解除、エミュレーション、コードカバレッジ、YARA スキャン、暗号定数の検出、ドライバおよびヒープ解析、アンチアンチデバッグ支援などを扱います。掲載プロジェクトの多くは Python または IDAPython ベースで、Unicorn、Keystone、angr といったフレームワークや、ChatGPT 支援のデコンパイル補助とも連携します。逆アセンブラやデバッガを拡張してバイナリ解析や脆弱性研究を行うリバースエンジニア、マルウェア解析者、エクスプロイト開発者、ゲームセキュリティ研究者向けのコレクションです。
