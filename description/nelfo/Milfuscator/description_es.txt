Este proyecto es el «Free Milfuscator», ahora de código abierto. Su esencia consiste en mutar el código x32 reconstruyendo por completo el PE mediante las bibliotecas Zydis y AsmJit.
La idea misma de la mutación se toma de un proyecto P2C para CS:GO.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el área de motores anti-cheat / de ofuscación.
