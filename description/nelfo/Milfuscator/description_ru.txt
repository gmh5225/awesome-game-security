Этот проект — «Free Milfuscator», теперь с открытым исходным кодом. Его суть — мутировать x32-код путём полной пересборки PE с помощью библиотек Zydis и AsmJit.
Сама идея мутации взята из одного P2C-проекта для CS:GO.
Он в основном полезен инженерам античита и исследователям защитной безопасности, работающим в области движков античита / обфускации.
