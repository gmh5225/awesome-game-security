Инструментарий инъекции shellcode для Windows, демонстрирующий пять техник выполнения: CreateThread, CreateRemoteThread (в скрытый процесс notepad), QueueUserAPC со сбросом очереди APC через NtTestAlert, злоупотребление callback EnumTimeFormatsEx и планирование через CreateFiber. Все вызовы API динамически разрешаются во время выполнения через GetProcAddress, чтобы избежать записей в IAT, а полезные нагрузки используют XOR-шифрованный shellcode msfvenom с предшествующим NOP sled для обхода сигнатурного обнаружения.
