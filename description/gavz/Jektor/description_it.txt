Toolkit di iniezione di shellcode per Windows che dimostra cinque tecniche di esecuzione: CreateThread, CreateRemoteThread (in un processo notepad nascosto), QueueUserAPC con flush della coda APC tramite NtTestAlert, abuso del callback EnumTimeFormatsEx e scheduling con CreateFiber. Tutte le chiamate API sono risolte dinamicamente a runtime tramite GetProcAddress per evitare voci IAT, e i payload usano shellcode msfvenom cifrato con XOR con NOP sled anteposto per eludere il rilevamento basato su firme.
