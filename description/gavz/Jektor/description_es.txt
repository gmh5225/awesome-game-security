Kit de herramientas de inyección de shellcode para Windows que demuestra cinco técnicas de ejecución: CreateThread, CreateRemoteThread (en un proceso notepad oculto), QueueUserAPC con vaciado de la cola APC mediante NtTestAlert, abuso del callback EnumTimeFormatsEx y programación con CreateFiber. Todas las llamadas API se resuelven dinámicamente mediante GetProcAddress en tiempo de ejecución para evitar entradas IAT, y las cargas útiles usan shellcode msfvenom cifrado con XOR y un NOP sled antepuesto para evadir la detección basada en firmas.
