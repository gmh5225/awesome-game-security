Windows シェルコード注入ツールキットで、CreateThread、CreateRemoteThread（隠し notepad プロセスへの注入）、NtTestAlert による APC キューフラッシュを伴う QueueUserAPC、EnumTimeFormatsEx コールバックの悪用、および CreateFiber スケジューリングという 5 つの実行手法を示します。すべての API 呼び出しは実行時に GetProcAddress で動的に解決され IAT エントリを避け、ペイロードは署名ベース検出を回避するため XOR 暗号化された msfvenom シェルコードに NOP sled を前置して使用します。
