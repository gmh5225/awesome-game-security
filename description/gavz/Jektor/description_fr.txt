Boîte à outils d'injection de shellcode Windows illustrant cinq techniques d'exécution : CreateThread, CreateRemoteThread (dans un processus notepad masqué), QueueUserAPC avec vidage de la file APC via NtTestAlert, abus du callback EnumTimeFormatsEx, et ordonnancement CreateFiber. Tous les appels API sont résolus dynamiquement via GetProcAddress à l'exécution pour éviter les entrées IAT, et les charges utiles utilisent du shellcode msfvenom chiffré en XOR avec un NOP sled en tête pour éluder la détection basée sur les signatures.
