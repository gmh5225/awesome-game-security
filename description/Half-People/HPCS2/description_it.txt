HPCS2 è un progetto di test di cheat esterno per Counter-Strike 2, creato per studiare sia i comportamenti di cheating sia quelli anti-cheat. È realizzato in C++ con Visual Studio e include utilità per la memoria di processo, helper per l’hijacking degli handle e numerosi header di offset del motore per i moduli CS2. Il repository espone funzionalità configurabili tramite un file INI, tra cui opzioni per l’assistenza alla mira, il controllo del rinculo e una logica di rendering in stile ESP. È destinato principalmente alla ricerca educativa sulla sicurezza dei giochi e alla pratica di reverse engineering.
