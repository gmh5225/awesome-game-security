Este proyecto es un framework de código abierto para la recuperación de credenciales que extrae contraseñas almacenadas localmente de numerosas aplicaciones. Está escrito principalmente en Python y ofrece colectores modulares para distintas familias de software y mecanismos de almacenamiento. La herramienta es compatible con varias plataformas y se usa habitualmente en flujos de post-explotación, auditoría y forense. Es especialmente útil para profesionales de seguridad que necesitan evaluar la exposición de credenciales en sistemas comprometidos o de prueba.
