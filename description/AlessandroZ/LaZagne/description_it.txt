Questo progetto è un framework open source per il recupero delle credenziali che estrae le password memorizzate localmente da numerose applicazioni. È scritto principalmente in Python e fornisce collector modulari per diverse famiglie di software e meccanismi di archiviazione. Lo strumento supporta più piattaforme ed è comunemente usato in flussi di post-exploitation, audit e forense. È particolarmente rilevante per i professionisti della sicurezza che devono valutare l'esposizione delle credenziali su sistemi compromessi o di test.
