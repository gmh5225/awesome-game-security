Этот проект — открытый фреймворк для восстановления учётных данных, извлекающий локально сохранённые пароли из множества приложений. Он в основном написан на Python и предоставляет модульные сборщики для разных семейств ПО и механизмов хранения. Инструмент поддерживает несколько платформ и широко применяется в сценариях пост-эксплуатации, аудита и цифровой криминалистики. Он наиболее полезен специалистам по безопасности, которым нужно оценивать раскрытие учётных данных на скомпрометированных или тестовых системах.
