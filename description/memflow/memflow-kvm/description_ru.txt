Коннектор Memflow, реализованный как модуль ядра Linux, который напрямую отображает физические страницы виртуальных машин KVM в пользовательское пространство, обеспечивая быструю межвиртуальную интроспекцию памяти без стандартных API KVM. Модуль ядра использует обход таблиц страниц и vmtools, чтобы предоставлять память гостя через символьные устройства на основе ioctl, с привязками Rust для пользовательского пространства.
