Conector Memflow implementado como módulo del kernel de Linux que mapea directamente las páginas físicas de máquinas virtuales KVM al espacio de usuario, permitiendo una introspección rápida de memoria entre VMs sin las APIs estándar de KVM. El módulo del kernel usa recorrido de tablas de páginas y vmtools para exponer la memoria del huésped mediante interfaces de dispositivo de carácter basadas en ioctl, con enlaces Rust en espacio de usuario.
