Este proyecto proporciona scripts y herramientas para integrar Magisk, KernelSU y Google Apps en imágenes de Windows Subsystem for Android (WSA) de forma local. Incluye scripts de Python y shell para extraer paquetes WSA, aplicar Magisk y GApps a la imagen del sistema, y generar builds WSA instalables con acceso root y soporte para LSPosed. Resulta especialmente útil para investigadores de seguridad Android y desarrolladores que necesitan entornos WSA rootados para pruebas y análisis.
