Questo progetto fornisce script e strumenti per integrare Magisk, KernelSU e Google Apps nelle immagini di Windows Subsystem for Android (WSA) in locale. Include script Python e shell per estrarre i pacchetti WSA, applicare Magisk e GApps all'immagine di sistema e generare build WSA installabili con accesso root e supporto LSPosed. È soprattutto utile per ricercatori di sicurezza Android e sviluppatori che necessitano di ambienti WSA rootati per test e analisi.
