Android Hidden Api Bypass — это Java-библиотека для обхода ограничений Android на не-SDK интерфейсы, позволяющая обращаться к скрытым API фреймворка во время выполнения.
Она предоставляет два варианта, HiddenApiBypass и LSPass, с API для вызова ограниченных методов и конструкторов, чтения скрытых полей и управления префиксами exemption.
Реализация обходится без нативного кода и упакована для современных рабочих процессов зависимостей Android.
Её часто используют в продвинутой инструментации Android, средствах совместимости и исследованиях безопасности.
