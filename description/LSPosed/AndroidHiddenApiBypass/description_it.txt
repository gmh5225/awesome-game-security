Android Hidden Api Bypass è una libreria Java per aggirare le restrizioni sulle interfacce non SDK di Android, così da accedere a runtime alle API nascoste del framework.
Fornisce due varianti, HiddenApiBypass e LSPass, con API per invocare metodi e costruttori soggetti a restrizioni, leggere campi nascosti e gestire i prefissi di esenzione.
L’implementazione evita il codice nativo ed è confezionata per i moderni workflow di dipendenze Android.
È comunemente usata in strumentazione Android avanzata, strumenti di compatibilità e ricerca sulla sicurezza.
