Android Hidden Api Bypass は、Android の非 SDK インターフェース制限を回避し、非公開のフレームワーク API を実行時に利用できるようにする Java ライブラリです。
HiddenApiBypass と LSPass の 2 つのバリアントを提供し、制限付きメソッドやコンストラクタの呼び出し、非公開フィールドの読み取り、exemption プレフィックスの管理用 API を備えています。
ネイティブコードを使わず、現代的な Android の依存関係ワークフロー向けにパッケージされています。
高度な Android インストルメンテーション、互換性ツール、セキュリティ研究でよく利用されます。
