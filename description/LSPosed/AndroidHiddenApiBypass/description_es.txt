Android Hidden Api Bypass es una biblioteca Java para eludir las restricciones de interfaces no SDK de Android y acceder en tiempo de ejecución a las API ocultas del framework.
Ofrece dos variantes, HiddenApiBypass y LSPass, con APIs para invocar métodos y constructores restringidos, leer campos ocultos y gestionar prefijos de exención.
La implementación evita el código nativo y está empaquetada para flujos modernos de dependencias de Android.
Se usa habitualmente en instrumentación avanzada de Android, herramientas de compatibilidad e investigación de seguridad.
