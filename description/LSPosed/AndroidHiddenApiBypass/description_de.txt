Android Hidden Api Bypass ist eine Java-Bibliothek zum Umgehen der Android-Beschränkungen für Nicht-SDK-Schnittstellen, sodass versteckte Framework-APIs zur Laufzeit zugänglich sind.
Sie bietet zwei Varianten, HiddenApiBypass und LSPass, mit APIs zum Aufrufen eingeschränkter Methoden und Konstruktoren, zum Lesen versteckter Felder und zum Verwalten von Exemption-Präfixen.
Die Implementierung verzichtet auf nativen Code und ist für moderne Android-Abhängigkeits-Workflows paketiert.
Sie wird häufig in fortgeschrittener Android-Instrumentierung, Kompatibilitätstools und Sicherheitsforschung eingesetzt.
