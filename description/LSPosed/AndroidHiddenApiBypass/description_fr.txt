Android Hidden Api Bypass est une bibliothèque Java qui contourne les restrictions des interfaces non SDK d’Android afin d’accéder aux API cachées du framework à l’exécution.
Elle propose deux variantes, HiddenApiBypass et LSPass, avec des API pour invoquer des méthodes et constructeurs restreints, lire des champs cachés et gérer les préfixes d’exemption.
L’implémentation évite le code natif et est conditionnée pour les workflows modernes de dépendances Android.
Elle est couramment utilisée dans l’instrumentation Android avancée, les outils de compatibilité et la recherche en sécurité.
