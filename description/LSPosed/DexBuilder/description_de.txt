DexBuilder ist eine C++-Bibliothek zum programmatischen Aufbau von DEX-Bytecode-Strukturen als Alternative zur dexmaker-artigen Generierung.
Sie basiert weitgehend auf AOSP-Komponenten und enthält LSPosed-spezifische Anpassungen für Runtime-Integrationsszenarien.
Die Codebasis konzentriert sich auf die von den Hauptnutzern benötigte Instruktionsabdeckung, anstatt jeden Opcode-Pfad zu implementieren.
Sie ist nützlich für Android-Framework-Forscher und Tool-Entwickler, die eine native DEX-Generierung benötigen.
