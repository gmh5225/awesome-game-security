DexBuilder — это библиотека на C++ для программного построения структур байткода DEX в качестве альтернативы генерации в стиле dexmaker.
Она в значительной степени основана на компонентах AOSP и включает специфичные для LSPosed изменения для сценариев интеграции во время выполнения.
Кодовая база сосредоточена на покрытии инструкций, необходимых основным пользователям, а не на реализации каждого пути opcode.
Она полезна исследователям фреймворка Android и разработчикам инструментов, которым нужна нативная генерация DEX.
