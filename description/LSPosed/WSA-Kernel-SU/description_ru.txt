WSA-Kernel-SU — это модуль ядра, предоставляющий путь /system/xbin/su для ядер Android, особенно в средах Windows Subsystem for Android.
Он работает, перехватывая выбранные системные вызовы и перенаправляя поток выполнения su, одновременно корректируя учётные данные и поведение SELinux для предоставления root-доступа.
Реализация написана на низкоуровневом C для современных ядер и включает параметры для снижения видимости функций суперпользователя.
Проект ориентирован на моддинг платформы Android и исследования безопасности, где требуется root-поведение с поддержкой ядра.
