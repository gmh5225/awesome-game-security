WSA-Kernel-SU es un módulo del kernel que proporciona una ruta /system/xbin/su para kernels de Android, especialmente en configuraciones de Windows Subsystem for Android.
Funciona enganchando syscalls seleccionados y redirigiendo el flujo de ejecución de su, a la vez que ajusta las credenciales y el comportamiento de SELinux para ofrecer acceso root.
La implementación es código C de bajo nivel pensado para kernels modernos e incluye opciones para reducir la visibilidad de la funcionalidad de superusuario.
Está orientado al modding de plataformas Android y a la investigación de seguridad donde se requiere un comportamiento root asistido por el kernel.
