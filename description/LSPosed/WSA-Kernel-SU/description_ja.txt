WSA-Kernel-SU は、Android カーネル向けに /system/xbin/su パスを提供するカーネルモジュールで、特に Windows Subsystem for Android 環境を想定しています。
選択したシステムコールをフックし、su の実行フローをリダイレクトしつつ、資格情報と SELinux の挙動を調整することで root アクセスを実現します。
実装は現代的なカーネル向けの低レベル C コードで、スーパーユーザー機能の可視性を抑えるオプションも含まれます。
カーネル支援による root 挙動が求められる Android プラットフォームの改造およびセキュリティ研究を対象としています。
