Этот проект — репозиторий из раздела cheat / dynamic binary instrumentation, посвящённый cpp veh dbi.
Он в основном написан на C++ и PowerShell.
Он прежде всего полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / dynamic binary instrumentation.
