Este proyecto es ArkDecompiler, un descompilador para el formato de bytecode de ArkCompiler de Huawei usado en aplicaciones HarmonyOS. Analiza el bytecode compilado de ArkTS/ArkJS, reconstruye el flujo de control, recupera construcciones a nivel de código fuente y genera código JavaScript/TypeScript legible a partir de archivos .abc compilados. Está dirigido a investigadores de seguridad móvil e ingenieros de ingeniería inversa que analizan aplicaciones HarmonyOS construidas con ArkCompiler.
