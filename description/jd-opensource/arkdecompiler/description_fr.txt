Ce projet est ArkDecompiler, un décompilateur pour le format de bytecode ArkCompiler de Huawei utilisé dans les applications HarmonyOS. Il analyse le bytecode compilé ArkTS/ArkJS, reconstruit le flot de contrôle, récupère les constructions de niveau source et produit du code JavaScript/TypeScript lisible à partir des fichiers .abc compilés. Il s'adresse aux chercheurs en sécurité mobile et aux ingénieurs en rétro-ingénierie qui analysent les applications HarmonyOS basées sur ArkCompiler.
