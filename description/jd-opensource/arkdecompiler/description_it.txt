Questo progetto è ArkDecompiler, un decompilatore per il formato bytecode di ArkCompiler di Huawei usato nelle applicazioni HarmonyOS. Analizza il bytecode compilato ArkTS/ArkJS, ricostruisce il flusso di controllo, recupera le costruzioni a livello di sorgente e produce codice JavaScript/TypeScript leggibile dai file .abc compilati. È pensato per ricercatori di sicurezza mobile e reverse engineer che analizzano applicazioni HarmonyOS basate su ArkCompiler.
