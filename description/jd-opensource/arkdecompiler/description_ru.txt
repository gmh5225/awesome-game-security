Этот проект — ArkDecompiler, декомпилятор для формата байткода Huawei ArkCompiler, используемого в приложениях HarmonyOS. Он разбирает скомпилированный байткод ArkTS/ArkJS, восстанавливает поток управления и конструкции уровня исходного кода, а также выводит читаемый код JavaScript/TypeScript из скомпилированных файлов .abc. Предназначен для исследователей мобильной безопасности и инженеров по реверс-инжинирингу, анализирующих приложения HarmonyOS на базе ArkCompiler.
