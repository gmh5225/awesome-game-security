Questo progetto è un’utilità Python per raccogliere ed elaborare dati da canali e chat di Telegram. Usa Telethon per l’accesso all’API e supporta l’esportazione dei contenuti analizzati in JSON e CSV, archiviando i dati in MongoDB. Il repository include script di setup multipiattaforma per Windows e sistemi di tipo Unix, nonché dipendenze di base orientate all’NLP come pandas e NLTK. Il caso d’uso documentato riguarda flussi di lavoro di raccolta dati e analisi dei messaggi, e il README avverte che le modifiche all’API possono attualmente rendere rischioso eseguirlo senza aggiornamenti.
