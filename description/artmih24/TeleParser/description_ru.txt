Этот проект — утилита на Python для сбора и обработки данных из каналов и чатов Telegram. Она использует Telethon для доступа к API и поддерживает экспорт разобранного содержимого в JSON и CSV с сохранением данных в MongoDB. Репозиторий включает кроссплатформенные скрипты установки для Windows и Unix-подобных систем, а также базовые NLP-зависимости, такие как pandas и NLTK. Документированный сценарий использования — рабочие процессы сбора данных и анализа сообщений; в README предупреждается, что из‑за изменений API запуск без обновлений сейчас может быть небезопасен.
