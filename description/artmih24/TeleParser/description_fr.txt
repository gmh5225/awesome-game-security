Ce projet est un utilitaire Python destiné à collecter et traiter des données provenant de canaux et de discussions Telegram. Il utilise Telethon pour accéder à l’API et permet d’exporterer le contenu analysé aux formats JSON et CSV tout en stockant les données dans MongoDB. Le dépôt inclut des scripts d’installation multiplateformes pour Windows et les systèmes de type Unix, ainsi que des dépendances de base orientées NLP telles que pandas et NLTK. Le cas d’usage documenté porte sur des workflows de collecte de données et d’analyse de messages, et le README avertit que des changements d’API peuvent actuellement rendre l’exécution risquée sans mises à jour.
