Este proyecto es una biblioteca de captura de pantalla para Windows que ofrece una adquisición eficiente de fotogramas del escritorio mediante la API DXGI Desktop Duplication. Captura la pantalla del escritorio con una carga mínima de CPU gracias a la copia de fotogramas basada en GPU, admite configuraciones multipantalla y entrega fotogramas en formatos de píxeles estándar para su procesamiento. La biblioteca C++/C# es adecuada para crear herramientas de grabación, streaming o capturas de pantalla. Está orientada a desarrolladores que implementan funciones de captura de pantalla en aplicaciones o sistemas de capturas anti-trampas.
