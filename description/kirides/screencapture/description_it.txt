Questo progetto è una libreria di cattura schermo per Windows che fornisce un'acquisizione efficiente dei frame del desktop tramite l'API DXGI Desktop Duplication. Cattura il display del desktop con un overhead minimo della CPU grazie alla copia dei frame basata su GPU, gestisce configurazioni multi-monitor e fornisce i frame in formati pixel standard per l'elaborazione. La libreria C++/C# è adatta per realizzare strumenti di registrazione, streaming o screenshot. È rivolta agli sviluppatori che implementano funzionalità di cattura schermo per applicazioni o sistemi di screenshot anti-cheat.
