Dieses Projekt ist eine Bildschirmcapture-Bibliothek für Windows, die eine effiziente Erfassung von Desktop-Frames über die DXGI Desktop Duplication API bereitstellt. Sie erfasst die Desktop-Anzeige mit minimaler CPU-Last durch GPU-basiertes Frame-Kopieren, unterstützt Multi-Monitor-Setups und gibt Frames in Standard-Pixelformaten zur Weiterverarbeitung aus. Die C++/C#-Bibliothek eignet sich zum Aufbau von Bildschirmaufnahme-, Streaming- oder Screenshot-Tools. Sie richtet sich an Entwickler, die Screen-Capture-Funktionen für Anwendungen oder Anti-Cheat-Screenshot-Systeme implementieren.
