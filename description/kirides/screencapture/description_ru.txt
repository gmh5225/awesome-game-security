Этот проект — библиотека захвата экрана для Windows, обеспечивающая эффективное получение кадров рабочего стола через API DXGI Desktop Duplication. Она захватывает изображение рабочего стола с минимальной нагрузкой на CPU за счёт копирования кадров на GPU, поддерживает конфигурации с несколькими мониторами и выдаёт кадры в стандартных пиксельных форматах для обработки. Библиотека на C++/C# подходит для создания средств записи экрана, стриминга или скриншотов. Она предназначена для разработчиков, реализующих захват экрана в приложениях или античит-системах скриншотов.
