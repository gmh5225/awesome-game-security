Este proyecto es un ejemplo en C++ de cómo hacer hooking del proceso de overlay de Steam para dibujar elementos de UI personalizados. Demuestra la interceptación de llamadas PaintTraverse en el pipeline VGUI y el uso de memoria compartida para comunicar datos de renderizado. La muestra también implementa una lógica de dibujo de formas de respaldo y documenta las limitaciones prácticas del renderizado en overlay. Resulta útil principalmente para la investigación de hooks de overlay en seguridad de juegos y experimentación relacionada con anti-cheat.
