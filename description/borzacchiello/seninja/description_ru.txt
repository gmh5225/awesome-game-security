Этот проект — SENinja, плагин символьного выполнения для Binary Ninja. Он реализует движок символьного выполнения, который символически исследует пути программы, отслеживает ограничения на входные значения, обнаруживает недостижимый код и определяет условия, необходимые для достижения заданных точек программы. Плагин на Python интегрируется с IL Binary Ninja и предоставляет интерактивный интерфейс для управления символьным исследованием. Он ориентирован на исследователей уязвимостей и инженеров по реверс-инжинирингу, использующих символьное выполнение для анализа путей и решения ограничений при анализе бинарных файлов.
