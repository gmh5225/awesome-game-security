Este proyecto es SENinja, un complemento de ejecución simbólica para Binary Ninja. Implementa un motor de ejecución simbólica que explora rutas del programa de forma simbólica, rastrea restricciones sobre los valores de entrada, detecta código inalcanzable e identifica las condiciones necesarias para alcanzar ubicaciones concretas del programa. El complemento en Python se integra con el IL de Binary Ninja y ofrece una interfaz interactiva para controlar la exploración simbólica. Está dirigido a investigadores de vulnerabilidades e ingenieros de ingeniería inversa que usan la ejecución simbólica para el análisis de rutas y la resolución de restricciones en el análisis binario.
