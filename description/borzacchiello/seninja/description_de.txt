Dieses Projekt ist SENinja, ein Plugin für symbolische Ausführung in Binary Ninja. Es implementiert eine Engine für symbolische Ausführung, die Programmpfade symbolisch erkundet, Bedingungen über Eingabewerte verfolgt, unerreichbaren Code erkennt und die Bedingungen ermittelt, die nötig sind, um bestimmte Programmstellen zu erreichen. Das Python-Plugin integriert sich in die IL von Binary Ninja und bietet eine interaktive Benutzeroberfläche zur Steuerung der symbolischen Exploration. Es richtet sich an Vulnerability-Researcher und Reverse Engineers, die symbolische Ausführung für Pfadanalyse und Constraint-Solving in der Binäranalyse nutzen.
