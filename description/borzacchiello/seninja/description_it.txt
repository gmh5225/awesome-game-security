Questo progetto è SENinja, un plugin di esecuzione simbolica per Binary Ninja. Implementa un motore di esecuzione simbolica che esplora i percorsi del programma in modo simbolico, tiene traccia dei vincoli sui valori di input, rileva il codice irraggiungibile e identifica le condizioni necessarie per raggiungere posizioni specifiche del programma. Il plugin Python si integra con l'IL di Binary Ninja e fornisce un'interfaccia interattiva per controllare l'esplorazione simbolica. È pensato per ricercatori di vulnerabilità e reverse engineer che usano l'esecuzione simbolica per l'analisi dei percorsi e la risoluzione dei vincoli nell'analisi binaria.
