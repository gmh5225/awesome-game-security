Ce projet est SENinja, un plugin d'exécution symbolique pour Binary Ninja. Il implémente un moteur d'exécution symbolique qui explore les chemins du programme de manière symbolique, suit les contraintes sur les valeurs d'entrée, détecte le code inatteignable et identifie les conditions nécessaires pour atteindre des emplacements précis du programme. Le plugin Python s'intègre à l'IL de Binary Ninja et fournit une interface interactive pour contrôler l'exploration symbolique. Il s'adresse aux chercheurs en vulnérabilités et aux ingénieurs en rétro-ingénierie qui utilisent l'exécution symbolique pour l'analyse de chemins et la résolution de contraintes en analyse binaire.
