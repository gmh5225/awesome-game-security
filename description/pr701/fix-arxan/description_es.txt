Este proyecto se centra en Arxan, un protector de archivos ejecutables portátiles (PE) que incluye diversas funciones. La herramienta ayuda a obtener información sobre el loader y una imagen descifrada y usable para facilitar la investigación. Resulta especialmente útil para ingenieros de anti-cheat y investigadores de seguridad defensiva que trabajan en anti-cheat / corrección de dumps.
