Этот проект посвящён Arxan — протектору portable executable (PE) файлов с соответствующими возможностями. Инструмент помогает получить сведения о загрузчике и расшифрованный рабочий образ для удобного исследования. Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / исправления дампов.
