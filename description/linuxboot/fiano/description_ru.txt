Этот проект — Fiano, библиотека на Go и набор инструментов для разбора, создания и изменения образов прошивки UEFI. Он обрабатывает UEFI Firmware Volumes, файлы FFS, секции PE32, сжатие (LZMA, Tiano) и GUIDed-секции в образах BIOS/UEFI ROM. Инструменты включают команды для извлечения, замены и удаления модулей прошивки из flash-образов. Предназначен для исследователей безопасности прошивок, разработчиков UEFI и аналитиков безопасности загрузки, работающих с анализом и модификацией прошивки UEFI.
