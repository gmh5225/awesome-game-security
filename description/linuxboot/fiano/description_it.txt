Questo progetto è Fiano, una libreria Go e un set di strumenti per analizzare, creare e manipolare immagini di firmware UEFI. Gestisce UEFI Firmware Volumes, file FFS, sezioni PE32, compressione (LZMA, Tiano) e sezioni GUIDed presenti nelle immagini ROM BIOS/UEFI. Gli strumenti includono comandi per estrarre, sostituire e rimuovere moduli di firmware dalle immagini flash. È rivolto a ricercatori di sicurezza del firmware, sviluppatori UEFI e analisti di sicurezza del boot che lavorano sull’analisi e la modifica del firmware UEFI.
