Este proyecto es Fiano, una biblioteca Go y un conjunto de herramientas para analizar, crear y manipular imágenes de firmware UEFI. Gestiona UEFI Firmware Volumes, archivos FFS, secciones PE32, compresión (LZMA, Tiano) y secciones GUIDed presentes en imágenes ROM BIOS/UEFI. Las herramientas incluyen comandos para extraer, reemplazar y eliminar módulos de firmware de imágenes flash. Está dirigido a investigadores de seguridad de firmware, desarrolladores UEFI y analistas de seguridad de arranque que trabajan con el análisis y la modificación de firmware UEFI.
