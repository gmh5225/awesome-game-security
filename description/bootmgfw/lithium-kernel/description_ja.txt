Lithium は、カスタム IOCTL インターフェース経由で通信するユーザーモードクライアントを備えた Windows x64 カーネルモードドライバーフレームワークです。主に C++ と補助的なアセンブリで書かれており、物理／仮想メモリの読み書き、ディレクトリテーブルベースの解決、ページテーブル走査、IDA 風のパターンスキャンを提供します。また、クロスプロセスのメモリ割り当てと保護、MouClass コールバックによるカーネルレベルマウスエミュレーション、スレッド隠蔽、NMI コールバック抑制、カーネルプールトラッカーのクリーンアップにも対応します。本プロジェクトは、メモリアクセスとアンチ解析回避のためのカーネルプリミティブを必要とする、低レベルの Windows 研究、リバースエンジニアリング、ゲームセキュリティ向けです。
