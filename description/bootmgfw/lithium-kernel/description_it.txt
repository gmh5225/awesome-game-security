Lithium è un framework di driver in modalità kernel Windows x64, affiancato da un client in modalità utente, che comunicano tramite un'interfaccia IOCTL personalizzata. Scritto principalmente in C++ con assembly di supporto, offre lettura/scrittura della memoria fisica e virtuale, risoluzione della directory table base, page table walking e pattern scanning in stile IDA. Supporta anche allocazione e protezione della memoria tra processi, emulazione del mouse a livello kernel tramite callback MouClass, occultamento dei thread, soppressione dei callback NMI e pulizia del kernel pool tracker. Il progetto è rivolto alla ricerca Windows a basso livello, al reverse engineering e al lavoro di game security che richiede primitive kernel per l'accesso alla memoria e l'evasione anti-analisi.
