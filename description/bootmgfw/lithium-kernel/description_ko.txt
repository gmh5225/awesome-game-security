Lithium은 사용자 모드 클라이언트와 함께 동작하는 Windows x64 커널 모드 드라이버 프레임워크로, 커스텀 IOCTL 인터페이스를 통해 통신합니다. 주로 C++와 보조 어셈블리로 작성되었으며, 물리·가상 메모리 읽기/쓰기, 디렉터리 테이블 베이스 해석, 페이지 테이블 순회, IDA 스타일 패턴 스캔을 제공합니다. 또한 크로스 프로세스 메모리 할당 및 보호, MouClass 콜백을 통한 커널 수준 마우스 에뮬레이션, 스레드 숨기기, NMI 콜백 억제, 커널 풀 트래커 정리도 지원합니다. 이 프로젝트는 메모리 접근과 안티분석 우회를 위한 커널 프리미티브가 필요한 저수준 Windows 연구, 리버스 엔지니어링, 게임 보안 작업을 대상으로 합니다.
