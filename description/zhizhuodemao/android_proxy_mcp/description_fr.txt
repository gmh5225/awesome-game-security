Ce projet se concentre sur android Proxy MCP — capture de paquets HTTP/HTTPS pour assistants IA, mitmdump + SQLite + requête en langage naturel.
Il est principalement écrit en Python et centré sur le modding.
Il est surtout utile aux développeurs de jeux, ingénieurs en reverse engineering et créateurs d'outils travaillant dans le domaine du développement de jeux / mcp server.
