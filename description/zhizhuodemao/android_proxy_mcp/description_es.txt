Este proyecto se centra en android Proxy MCP — captura de paquetes HTTP/HTTPS para asistentes de IA, mitmdump + SQLite + consulta en lenguaje natural.
Está escrito principalmente en Python y se centra en el modding.
Es especialmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo de juegos / mcp server.
