Этот проект сосредоточен на android Proxy MCP — перехвате HTTP/HTTPS-пакетов для ИИ-ассистентов, mitmdump + SQLite + запросы на естественном языке.
Он в основном написан на Python и ориентирован на моддинг.
Он прежде всего полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов в области разработки игр / mcp server.
