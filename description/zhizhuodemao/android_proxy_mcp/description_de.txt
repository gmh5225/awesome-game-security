Dieses Projekt konzentriert sich auf android Proxy MCP — HTTP/HTTPS-Paketcapture für KI-Assistenten, mitmdump + SQLite + Abfragen in natürlicher Sprache.
Es ist hauptsächlich in Python geschrieben und legt den Schwerpunkt auf Modding.
Es ist vor allem nützlich für Spieleentwickler, Reverse Engineers und Tool-Entwickler im Bereich Game Develop / mcp server.
