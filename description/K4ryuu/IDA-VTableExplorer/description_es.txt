Este proyecto es un plugin de C++ para IDA Pro que automatiza el análisis de tablas virtuales de C++.
Detecta patrones de vtable de GCC y MSVC, extrae información de clases y anota índices y offsets de funciones virtuales.
El plugin también realiza análisis de herencia basado en RTTI, comparación de overrides y visualización de jerarquías.
Está pensado para flujos de ingeniería inversa en los que comprender el polimorfismo y el despacho virtual es crítico.
