Questo progetto è una reimplementazione open source in C++ del classico platform Captain Claw, con integrazione della fisica Box2D. Include supporto multipiattaforma tramite CMake con configurazioni di build per Android, e fornisce una ricreazione completa dei livelli, degli attori e delle meccaniche di gioco dell'originale. È soprattutto utile per ricercatori di game engine e modder che studiano la reimplementazione di platform 2D e l'architettura di engine personalizzati con integrazione della fisica.
