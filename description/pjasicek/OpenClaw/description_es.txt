Este proyecto es una reimplementación de código abierto en C++ del clásico juego de plataformas Captain Claw, con integración de física Box2D. Incluye soporte multiplataforma mediante CMake con configuraciones de compilación para Android, y ofrece una recreación completa de los niveles, actores y mecánicas de juego del original. Resulta especialmente útil para investigadores de motores de juego y modders que estudian la reimplementación de plataformas 2D y la arquitectura de motores personalizados con integración física.
