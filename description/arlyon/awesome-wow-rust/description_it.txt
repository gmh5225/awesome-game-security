Questo progetto è un indice curato di risorse Rust per costruire e studiare gli ecosistemi di server privati di World of Warcraft. Cataloga implementazioni server, librerie di protocolli e formati di file, nonché strumenti di supporto come renderer e visualizzatori di asset. La raccolta privilegia riferimenti pratici su autenticazione, networking, formati dati e tooling affine agli emulatori. Si rivolge a sviluppatori di game server in Rust e appassionati di reverse engineering che esplorano la tecnologia di emulazione WoW.
