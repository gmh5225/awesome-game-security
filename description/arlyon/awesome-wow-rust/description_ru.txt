Этот проект — курируемый индекс ресурсов на Rust для создания и исследования экосистем приватных серверов World of Warcraft. В нём собраны реализации серверов, библиотеки протоколов и форматов файлов, а также вспомогательные инструменты — рендереры и просмотрщики ассетов. Коллекция делает акцент на практических материалах по аутентификации, сетям, форматам данных и инструментам, связанным с эмуляторами. Она предназначена для разработчиков игровых серверов на Rust и энтузиастов реверс-инжиниринга, изучающих технологии эмуляции WoW.
