Ce projet est un émulateur expérimental écrit en Rust pour la pile serveur de World of Warcraft 3.3.5. Il se concentre actuellement sur le serveur d'authentification et est conçu pour interopérer avec des schémas de base de données de type TrinityCore. Le code utilise des composants Rust asynchrones, des flux de travail centrés sur SQL et des chemins de configuration conteneurisés pour faciliter le développement et les tests. Il s'adresse aux développeurs et aux chercheurs en protocoles qui construisent une infrastructure d'émulation de jeux sûre, modulaire et moderne.
