Этот проект — экспериментальный эмулятор на Rust для серверного стека World of Warcraft 3.3.5. Сейчас основное внимание уделяется серверу аутентификации; решение рассчитано на взаимодействие со схемами баз данных в стиле TrinityCore. Кодовая база использует асинхронные компоненты Rust, SQL-ориентированные рабочие процессы и контейнеризированные пути развёртывания для разработки и тестирования. Целевая аудитория — разработчики и исследователи протоколов, создающие безопасную, модульную и современную инфраструктуру для эмуляции игр.
