Questo progetto è un emulatore sperimentale basato su Rust per lo stack server di World of Warcraft 3.3.5. Attualmente si concentra sul server di autenticazione ed è progettato per interoperare con schemi di database in stile TrinityCore. Il codice usa componenti Rust asincroni, flussi di lavoro incentrati su SQL e percorsi di setup containerizzati per supportare sviluppo e test. Si rivolge a sviluppatori e ricercatori di protocolli che costruiscono un'infrastruttura di emulazione di giochi sicura, modulare e moderna.
