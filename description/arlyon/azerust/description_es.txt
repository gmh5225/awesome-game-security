Este proyecto es un emulador experimental escrito en Rust para la pila de servidor de World of Warcraft 3.3.5. Actualmente se centra en el servidor de autenticación y está diseñado para interoperar con esquemas de base de datos al estilo TrinityCore. El código usa componentes Rust asíncronos, flujos de trabajo centrados en SQL y rutas de configuración contenedorizadas para apoyar el desarrollo y las pruebas. Está dirigido a desarrolladores e investigadores de protocolos que construyen una infraestructura de emulación de juegos segura, modular y moderna.
