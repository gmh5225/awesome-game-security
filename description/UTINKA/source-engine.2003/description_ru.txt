Этот проект — крупный архивный снимок ранней кодовой базы движка Source с игровыми модулями и инструментами эпохи Half-Life 2. В нём обширный код движка, клиента, сервера, рендеринга и утилит на C и C++, а также устаревшие файлы проектов Visual Studio. Дерево включает подсистемы вроде компонентов материалов и графики, логики игровых DLL и сборочных скриптов для исторических Windows-toolchain. В основном полезен как справочный корпус для изучения внутренностей движка, реверс-инжиниринга и исследований безопасности устаревших игр.
