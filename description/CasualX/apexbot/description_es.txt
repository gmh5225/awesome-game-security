Este proyecto es un framework de cheat externo para Apex Legends escrito en Rust y diseñado para integrarse en un backend personalizado de acceso a memoria. Incluye funciones modulares como aim assist con suavizado PID, lógica de triggerbot, control de retroceso, overlays ESP, highlights, radar, seguimiento de observadores y scripts de automatización. El código separa el análisis del estado del juego, la predicción de proyectiles y el manejo de offsets, con herramientas para regenerar gamedata a partir de offsets volcados. Está dirigido principalmente a investigadores avanzados de seguridad en juegos y desarrolladores de cheats que estudian la arquitectura de cheats externos y la ingeniería de funciones.
