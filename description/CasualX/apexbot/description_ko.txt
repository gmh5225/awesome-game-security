이 프로젝트는 사용자 지정 메모리 접근 백엔드에 임베드되도록 설계된 Rust 기반 Apex Legends 외부 치트 프레임워크입니다. PID 스무딩이 적용된 에임 어시스트, 트리거봇 로직, 반동 제어, ESP 오버레이, 하이라이트, 레이더, 옵저버 추적, 자동화 스크립트 등 모듈형 기능을 포함합니다. 코드베이스는 게임 상태 파싱, 발사체 예측, 오프셋 처리를 분리하며, 덤프된 오프셋으로 gamedata를 재생성하는 도구를 제공합니다. 주로 외부 치트 아키텍처와 기능 엔지니어링을 연구하는 고급 게임 보안 연구자 및 치트 개발자를 대상으로 합니다.
