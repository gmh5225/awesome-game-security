Apexdream — библиотека на Rust и пример Win32 для внешнего чита Apex Legends, который подключается к процессу игры и управляет функциями через живую память. Включает aim assist с настройками триггера для каждого оружия, ESP и радар-оверлеи, контроль отдачи, подсветку игроков и предсказание снарядов на основе игрового SDK для сущностей, оружия и связанного состояния. Сопутствующий инструмент apexdumper использует PE-анализ на базе pelite для восстановления смещений (classes, convars, datamaps, recv tables, interfaces) в gamedata.ini. Проект предназначен для исследований в области game-hacking и reverse engineering внутренностей Apex Legends в стиле Respawn/Source и внешних инструментов работы с памятью.
