Apexdream 是一个 Rust 库与 Win32 示例，实现通过附加到游戏进程并从实时内存驱动功能的 Apex Legends 外部作弊工具。它包含按武器配置的瞄准辅助与扳机设置、ESP 与雷达叠加层、后坐力控制、玩家高亮，以及基于游戏 SDK（实体、武器及相关状态）的弹道预测。配套的 apexdumper 工具利用基于 pelite 的 PE 分析，将 classes、convars、datamaps、recv tables 与 interfaces 等偏移恢复到 gamedata.ini。该项目面向 Respawn/Source 风格的 Apex Legends 内部机制与外部内存工具方面的游戏破解与逆向工程研究。
