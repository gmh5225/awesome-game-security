Apexdream は、ゲームプロセスにアタッチし、ライブメモリから機能を駆動する外部 Apex Legends チート向けの Rust ライブラリおよび Win32 サンプルです。武器ごとのトリガー設定付きエイムアシスト、ESP とレーダーオーバーレイ、リコイル制御、プレイヤーハイライト、およびエンティティ・武器・関連状態向けゲーム SDK に基づく弾道予測を含みます。付属の apexdumper ツールは pelite ベースの PE 解析を用い、classes、convars、datamaps、recv tables、interfaces などのオフセットを gamedata.ini に復元します。本プロジェクトは、Respawn/Source 系 Apex Legends 内部構造と外部メモリツーリングに関するゲームハッキングおよびリバースエンジニアリング研究を目的としています。
