Apexdream 是一個 Rust 函式庫與 Win32 範例，實作透過附加到遊戲進程並從即時記憶體驅動功能的 Apex Legends 外部作弊工具。它包含依武器設定的瞄準輔助與扳機設定、ESP 與雷達疊加層、後座力控制、玩家高亮，以及基於遊戲 SDK（實體、武器及相關狀態）的彈道預測。配套的 apexdumper 工具利用基於 pelite 的 PE 分析，將 classes、convars、datamaps、recv tables 與 interfaces 等偏移還原至 gamedata.ini。此專案面向 Respawn/Source 風格的 Apex Legends 內部機制與外部記憶體工具方面的遊戲破解與逆向工程研究。
