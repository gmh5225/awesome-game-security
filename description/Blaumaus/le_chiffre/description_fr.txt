Le Chiffre est une preuve de concept légère de cheat externe pour CS:GO, écrite en C++. Elle implémente des fonctionnalités courantes d'automatisation de gameplay telles que bunnyhop, triggerbot, aimbot, glow ESP et radar hacks. Le projet est conçu comme une base de code éducative pour illustrer la manipulation de jeu basée sur la mémoire et les fondamentaux de la rétro-ingénierie. Il s'adresse principalement aux apprenants qui étudient les techniques de game hacking Windows bas niveau, plutôt qu'à un usage en production.
