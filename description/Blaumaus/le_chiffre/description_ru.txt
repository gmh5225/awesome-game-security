Le Chiffre — лёгкий внешний proof of concept чита для CS:GO на C++. Он реализует распространённые функции автоматизации геймплея: bunnyhop, triggerbot, aimbot, glow ESP и radar hacks. Проект оформлен как учебная кодовая база для демонстрации манипуляций с игрой через память и основ реверс-инжиниринга. Он рассчитан в первую очередь на изучающих низкоуровневые техники Windows game hacking, а не на промышленное использование.
