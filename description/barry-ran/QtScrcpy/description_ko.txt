이 프로젝트는 QtScrcpy로, USB 또는 TCP/IP를 통해 PC에서 Android 기기를 표시하고 제어하는 Qt 기반 GUI 애플리케이션입니다. scrcpy의 서버 구성 요소를 사용해 MediaCodec으로 기기 화면을 캡처하고 낮은 지연으로 데스크톱에 스트리밍하며, 키보드/마우스 입력, 파일 드래그 앤 드롭, 화면 녹화, 다중 기기 관리를 지원합니다. 이 C++ 애플리케이션은 Android 기기의 root 권한 없이 동작합니다. 데스크톱에서 Android 기기를 미러링하고 제어하려는 개발자, 테스터, 사용자를 대상으로 합니다.
