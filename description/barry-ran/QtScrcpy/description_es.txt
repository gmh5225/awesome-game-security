Este proyecto es QtScrcpy, una aplicación GUI basada en Qt para mostrar y controlar dispositivos Android desde un PC por USB o TCP/IP. Utiliza el componente de servidor de scrcpy para capturar la pantalla del dispositivo mediante MediaCodec y transmitirla al escritorio con baja latencia, admitiendo entrada de teclado/ratón, arrastrar y soltar archivos, grabación de pantalla y gestión de varios dispositivos. La aplicación en C++ funciona sin requerir acceso root en el dispositivo Android. Está dirigida a desarrolladores de Android, testers y usuarios que desean reflejar y controlar dispositivos Android desde el escritorio.
