Этот проект — QtScrcpy, GUI-приложение на Qt для отображения и управления устройствами Android с ПК по USB или TCP/IP. Оно использует серверный компонент scrcpy для захвата экрана устройства через MediaCodec и потоковой передачи на рабочий стол с низкой задержкой, поддерживая ввод с клавиатуры/мыши, перетаскивание файлов, запись экрана и управление несколькими устройствами. Приложение на C++ работает без root-доступа на устройстве Android. Оно предназначено для разработчиков Android, тестировщиков и пользователей, которым нужно зеркалировать и управлять устройствами Android с рабочего стола.
