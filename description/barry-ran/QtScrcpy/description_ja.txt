本プロジェクトは QtScrcpy で、USB または TCP/IP 経由で PC から Android 端末を表示・操作するための Qt ベースの GUI アプリケーションです。scrcpy のサーバーコンポーネントを使い、MediaCodec で端末画面をキャプチャして低遅延でデスクトップにストリーミングし、キーボード/マウス入力、ファイルのドラッグ＆ドロップ、画面録画、複数端末の管理に対応します。この C++ アプリケーションは Android 端末の root 権限を必要としません。デスクトップから Android 端末をミラーリングして操作したい開発者、テスター、ユーザー向けです。
