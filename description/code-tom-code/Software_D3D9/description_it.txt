Questo progetto si concentra su dX9.
È pensato per funzionare con giochi reali e altri programmi e, a regime, dovrebbe offrire un alto livello di compatibilità con un'ampia gamma di giochi e programmi D3D9 esistenti.
È particolarmente utile per i programmatori grafici e gli sviluppatori di tooling per giochi Windows che lavorano nell'ambito DirectX / emulazione.
