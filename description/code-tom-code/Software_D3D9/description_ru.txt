Этот проект сосредоточен на dX9.
Он рассчитан на работу с реальными играми и другими программами и в итоге должен обеспечить высокий уровень совместимости с широким кругом существующих D3D9-игр и программ.
В первую очередь он полезен графическим программистам и разработчикам инструментов для Windows-игр, работающим в области DirectX / эмуляции.
