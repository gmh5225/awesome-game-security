Este proyecto se centra en dX9.
Está pensado para funcionar con juegos reales y otros programas, y con el tiempo debería ofrecer un alto nivel de compatibilidad con una amplia gama de juegos y programas D3D9 existentes.
Resulta especialmente útil para programadores gráficos y desarrolladores de herramientas de juegos en Windows que trabajan en el área de DirectX / emulación.
