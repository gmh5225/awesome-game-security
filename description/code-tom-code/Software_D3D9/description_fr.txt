Ce projet se concentre sur dX9.
Il est conçu pour fonctionner avec de vrais jeux et d'autres programmes, et devrait à terme offrir un haut niveau de compatibilité avec un large éventail de jeux et programmes D3D9 existants.
Il est surtout utile aux programmeurs graphiques et aux développeurs d'outils de jeux Windows travaillant dans le domaine DirectX / émulation.
