Dieses Projekt konzentriert sich auf dX9.
Es ist für den Einsatz mit echten Spielen und anderen Programmen gedacht und soll langfristig ein hohes Maß an Kompatibilität mit einer Vielzahl bestehender D3D9-Spiele und -Programme bieten.
Es ist vor allem für Grafikprogrammierer und Entwickler von Windows-Spielwerkzeugen im DirectX-/Emulationsbereich nützlich.
