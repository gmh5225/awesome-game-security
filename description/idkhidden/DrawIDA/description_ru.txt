Этот проект посвящён лёгкому плагину-доске для IDA, который позволяет специалистам по реверс-инжинирингу делать наброски и проводить мозговой штурм прямо внутри IDA.
DrawIDA распространяется по лицензии MIT — подробности см. в файле LICENSE.
Он в первую очередь полезен исследователям игровой безопасности и reverse-инженерам, изучающим атакующие техники в области читов / плагинов IDA.
