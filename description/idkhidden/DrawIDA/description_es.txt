Este proyecto se centra en un plugin de pizarra ligero para IDA que permite a los ingenieros inversos bosquejar y hacer brainstorming directamente dentro de IDA.
DrawIDA está licenciado bajo la licencia MIT; consulte el archivo LICENSE para más detalles.
Es especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / plugins de IDA.
