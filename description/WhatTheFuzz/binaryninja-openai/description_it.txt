Questo progetto è un plugin di Binary Ninja che collega le viste del decompilatore ai modelli OpenAI per un'assistenza interattiva all'analisi.
Può riassumere cosa fanno le funzioni selezionate a partire da HLIL o pseudo-C e proporre rinomine di variabili direttamente nella vista di analisi.
Il plugin è scritto in Python e si integra con le impostazioni dei plugin di Binary Ninja e con le opzioni di gestione delle chiavi API.
È pensato per reverse engineer che desiderano triage e annotazione assistiti dall'IA durante la ricerca sulla sicurezza dei binari.
