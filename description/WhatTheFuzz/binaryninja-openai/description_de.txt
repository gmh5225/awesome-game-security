Dieses Projekt ist ein Binary-Ninja-Plugin, das Decompiler-Ansichten mit OpenAI-Modellen für interaktive Analyseunterstützung verbindet.
Es kann ausgewählte Funktionen anhand von HLIL oder Pseudo-C zusammenfassen und direkt in der Analyseansicht Umbenennungen von Variablen vorschlagen.
Das Plugin ist in Python geschrieben und integriert sich in die Plugin-Einstellungen von Binary Ninja sowie in Optionen zur Verwaltung von API-Schlüsseln.
Es richtet sich an Reverse Engineers, die bei der Binärsicherheitsforschung KI-gestützte Triage und Annotation nutzen möchten.
