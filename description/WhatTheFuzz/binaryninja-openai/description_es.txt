Este proyecto es un complemento de Binary Ninja que conecta las vistas del descompilador con modelos de OpenAI para asistencia interactiva en el análisis.
Puede resumir lo que hacen las funciones seleccionadas a partir de HLIL o pseudo-C y proponer renombrados de variables directamente en la vista de análisis.
El complemento está escrito en Python y se integra con la configuración de complementos de Binary Ninja y las opciones de gestión de claves API.
Está pensado para ingenieros de ingeniería inversa que desean triage y anotación asistidos por IA durante la investigación de seguridad de binarios.
