Этот проект — плагин Binary Ninja, который подключает представления декомпилятора к моделям OpenAI для интерактивной помощи в анализе.
Он может суммировать назначение выбранных функций по HLIL или псевдо-C и предлагать переименование переменных прямо в представлении анализа.
Плагин написан на Python и интегрируется с настройками плагинов Binary Ninja и параметрами управления API-ключами.
Он предназначен для специалистов по реверс-инжинирингу, которым нужна ИИ-поддержка при triage и аннотировании в исследованиях безопасности бинарных файлов.
