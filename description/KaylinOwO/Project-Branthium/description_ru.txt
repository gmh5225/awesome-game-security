Этот проект — Windows-фреймворк для игрового читерства, объединяющий пользовательский клиент и компонент драйвера ядра. В него входят модули логики аимбота, ESP-визуализации, кэширования сущностей и предсказания оружия, а также меню и оверлей на базе ImGui и DirectX9. Основная реализация на C++, с файлами решения Visual Studio и кодом низкоуровневого взаимодействия с памятью в каталогах cheat и driver. Предназначен для экспериментов по game hacking и рабочих процессов реверс-инжиниринга, особенно для целей в стиле Apex Legends.
