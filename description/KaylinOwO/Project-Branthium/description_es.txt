Este proyecto es un framework de trampas para juegos en Windows que combina un cliente en modo usuario con un componente de controlador de núcleo. Incluye módulos para lógica de aimbot, visuales ESP, caché de entidades y predicción de armas, junto con un menú y una superposición basados en ImGui y DirectX9. La implementación principal está en C++, con archivos de solución de Visual Studio y código de interacción de memoria de bajo nivel en los directorios de cheat y driver. Está pensado para experimentación en game hacking y flujos de ingeniería inversa, especialmente para objetivos al estilo de Apex Legends.
