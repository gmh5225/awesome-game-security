Ce projet est un framework de triche pour jeux Windows qui combine un client en mode utilisateur avec un composant pilote noyau. Il comprend des modules pour la logique d’aimbot, les visuels ESP, le cache d’entités et la prédiction d’armes, ainsi qu’un menu et une superposition basés sur ImGui et DirectX9. L’implémentation principale est en C++, avec des fichiers de solution Visual Studio et du code d’interaction mémoire bas niveau dans les répertoires cheat et driver. Il est destiné à l’expérimentation en game hacking et aux flux de reverse engineering, notamment pour des cibles de type Apex Legends.
