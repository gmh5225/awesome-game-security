Questo progetto è un framework di cheating per giochi Windows che combina un client in user-mode con un componente driver kernel. Include moduli per logica aimbot, visuali ESP, caching delle entità e predizione delle armi, insieme a un menu e uno stack di overlay basati su ImGui e DirectX9. L’implementazione principale è in C++, con file di soluzione Visual Studio e codice di interazione a basso livello con la memoria nelle directory cheat e driver. È pensato per sperimentazione di game hacking e flussi di reverse engineering, soprattutto per target in stile Apex Legends.
