本プロジェクトは、ユーザーモードのクライアントとカーネルドライバーを組み合わせた Windows 向けゲームチートフレームワークです。エイムボットロジック、ESP 表示、エンティティキャッシュ、武器予測などの機能モジュールに加え、ImGui と DirectX9 ベースのメニューおよびオーバーレイを備えています。主な実装は C++ で、Visual Studio のソリューションファイルと、チートおよびドライバー各ディレクトリ内の低レベルメモリ操作コードが含まれます。ゲームハッキングの実験やリバースエンジニアリング、とりわけ Apex Legends 系ターゲット向けの用途を想定しています。
