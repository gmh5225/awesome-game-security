Этот проект посвящён GDB.
Собран с помощью mingw-w64 v12.2.0, включает все архитектуры, поддерживает TUI и Python, без иных DLL-зависимостей, кроме собственных библиотек Windows.
В первую очередь полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим наступательные техники в области читов и отладки.
