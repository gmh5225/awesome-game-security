Questo progetto è un repository elencato sotto cheat / game:fortnite e si concentra sul codice sorgente interno di un cheat per Fortnite.
È scritto principalmente in C/C++ e C++ e si concentra sulla generazione di SDK e sul hooking.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell’ambito cheat / game:fortnite.
