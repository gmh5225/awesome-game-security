Este proyecto es un repositorio listado bajo cheat / game:fortnite y se centra en el código fuente interno de un cheat de Fortnite.
Está escrito principalmente en C/C++ y C++ y se centra en la generación de SDK y el hooking.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito cheat / game:fortnite.
