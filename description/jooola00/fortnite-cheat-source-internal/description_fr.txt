Ce projet est un dépôt classé sous cheat / game:fortnite et porte sur le code source interne d’un cheat Fortnite.
Il est principalement écrit en C/C++ et C++ et se concentre sur la génération de SDK et le hooking.
Il est surtout utile aux chercheurs en sécurité des jeux et aux ingénieurs en rétro-ingénierie qui étudient les techniques offensives dans le domaine cheat / game:fortnite.
