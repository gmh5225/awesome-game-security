ida_gpt es un ayudante de IDAPython que conecta los flujos de desensamblado de IDA Pro con un backend compatible con ChatGPT.
Está escrito en Python y ofrece funciones que solicitan descripciones de funciones en lenguaje sencillo y sugerencias automáticas de renombrado para variables, ubicaciones y símbolos de función.
El script está pensado para uso interactivo dentro de IDA, donde las salidas generadas se escriben de nuevo en la base de datos como comentarios e identificadores renombrados.
Es especialmente adecuado para profesionales de la ingeniería inversa que quieran triar y anotar más rápido binarios desconocidos.
