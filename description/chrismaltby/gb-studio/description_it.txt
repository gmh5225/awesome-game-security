Questo progetto è un creatore di giochi avventura retrò per Game Boy, rapido e facile da usare, disponibile per Mac, Linux e Windows.
È scritto principalmente in TypeScript e JavaScript e si concentra sullo sviluppo di driver, sul rendering e sull'animazione.
È soprattutto utile per sviluppatori di giochi, programmatori di engine e ricercatori grafici che lavorano nell'ambito dei game engine / del codice sorgente.
