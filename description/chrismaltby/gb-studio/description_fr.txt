Ce projet est un créateur de jeux d'aventure rétro pour Game Boy, rapide et facile à utiliser, disponible pour Mac, Linux et Windows.
Il est principalement écrit en TypeScript et JavaScript et se concentre sur le développement de pilotes, le rendu et l'animation.
Il est surtout utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
