Este proyecto es un creador de juegos de aventura retro para Game Boy, rápido y fácil de usar, disponible para Mac, Linux y Windows.
Está escrito principalmente en TypeScript y JavaScript y se centra en el desarrollo de controladores, el renderizado y la animación.
Resulta especialmente útil para desarrolladores de juegos, programadores de motores e investigadores gráficos que trabajan en el área de motores de juego / código fuente.
