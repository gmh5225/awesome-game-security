Этот проект — быстрый и простой в использовании конструктор ретро-приключенческих игр для Game Boy, доступный для Mac, Linux и Windows.
Он в основном написан на TypeScript и JavaScript и сосредоточен на разработке драйверов, рендеринге и анимации.
Он особенно полезен разработчикам игр, программистам движков и исследователям графики, работающим в области игровых движков / исходного кода.
