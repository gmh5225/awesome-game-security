Kiteshield — это packer и protector для ELF Linux x86-64, созданный чтобы усложнить реверс-инжиниринг бинарников. Он оборачивает исполняемые файлы многослойным шифрованием RC4 и внедряет собственный загрузчик, который расшифровывает, отображает и запускает код целиком в пользовательском пространстве. Движок времени выполнения использует ptrace, чтобы в расшифрованном виде оставались только функции активного стека вызовов, и добавляет несколько антиотладочных проверок во время работы. Проект написан в основном на C с вспомогательным кодом на ассемблере и служит образовательной платформой для обфускации бинарников и исследований антианализа.
