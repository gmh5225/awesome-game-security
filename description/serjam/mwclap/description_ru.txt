Этот проект посвящён данному хаку, для которого требуется установленный и включённый внутриигровой оверлей NVIDIA GeForce Experience! Игра должна быть в полноэкранном режиме без рамок. Отказ от ответственности: этот исходный код предназначен только для образовательных целей. Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / игры: COD Warzone.
