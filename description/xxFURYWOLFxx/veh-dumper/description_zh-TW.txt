veh-dumper 是一個可注入的 x64 DLL，用於列舉目標行程中所有已註冊的 VectoredExceptionHandler 與 VectoredContinueHandler，並將每一個處理常式傾印為可直接用 IDA 開啟的獨立合成 PE。對每個處理常式，它會精準擷取處理函式位元組（透過 RtlLookupFunctionEntry 展開資訊或啟發式掃描）、RIP 相對資料參考、經深度受限 BFS 得到的直接被呼叫者，以及帶有已解析程式碼的間接呼叫 IAT 槽。位於 Windows 系統模組下的被呼叫者不會被傾印，而是輸出其匯出名稱，以便 IDA 顯示可讀匯入。透過註冊一個立足點處理常式，並用 RtlDecodePointer 動態發現編碼後的處理常式指標偏移來走訪 VEH 清單，面向分析基於 VEH 的保護或反作弊邏輯的逆向工程師。
