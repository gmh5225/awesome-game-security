veh-dumper는 주입 가능한 x64 DLL로, 대상 프로세스에 등록된 모든 VectoredExceptionHandler와 VectoredContinueHandler를 열거하고 각각을 IDA에서 바로 열 수 있는 독립 합성 PE로 덤프합니다. 각 핸들러에 대해 핸들러 함수 바이트(RtlLookupFunctionEntry 언와인드 정보 또는 휴리스틱 스캔), RIP 상대 데이터 참조, 깊이 제한 BFS로 얻은 직접 피호출자, 그리고 해석된 코드가 있는 간접 호출 IAT 슬롯을 정밀하게 추출합니다. Windows 시스템 모듈 아래의 피호출자는 덤프하지 않고 내보내기 이름을 출력해 IDA에서 읽기 쉬운 임포트로 보이게 합니다. 발판 핸들러를 등록하고 RtlDecodePointer로 인코딩된 핸들러 포인터 오프셋을 동적으로 찾아 VEH 목록을 순회하며, VEH 기반 보호나 안티치트 논리를 분석하는 리버스 엔지니어를 대상으로 합니다.
