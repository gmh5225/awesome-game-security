veh-dumper — внедряемая x64 DLL, которая перечисляет все зарегистрированные VectoredExceptionHandler и VectoredContinueHandler в целевом процессе и сохраняет каждый как отдельный синтетический PE, который можно сразу открыть в IDA. Для каждого обработчика она точечно извлекает байты функции (через unwind-информацию RtlLookupFunctionEntry или эвристическое сканирование), RIP-относительные ссылки на данные, прямых callee через BFS с ограничением глубины и слоты IAT косвенных вызовов с разрешённым кодом. Callee из системных модулей Windows не дампятся; вместо этого выводятся имена их экспортов, чтобы IDA показывала читаемые импорты. Список VEH обходится путём регистрации опорного обработчика и динамического обнаружения смещения закодированного указателя через RtlDecodePointer — для реверс-инженеров, анализирующих защиту или античит на базе VEH в Windows.
