Этот проект — DLL, которая повторно включает внутриигровой интерфейс консоли Unreal Engine в контексте конкретной сборки Fortnite.
Она написана на C++ для Windows x64 и использует pattern scanning и логику построения объектов движка, чтобы во время выполнения находить и инициализировать структуры, связанные с консолью.
Репозиторий включает низкоуровневые перечисления объектов Unreal и вспомогательные средства для работы с памятью в поддержку внутренних сценариев инъекции.
Основной сценарий использования — исследование внутренностей Unreal Engine и исследования в области игровой безопасности в контролируемых средах с отключённым античитом.
