Win32kHooker — драйвер ядра Windows, демонстрирующий, как находить и перехватывать пути диспетчеризации функций внутри win32k.sys на современных системах. Код использует C++ с API ядра, присоединение к контексту процесса, отображение syscall и дизассемблирование во время выполнения для разрешения целей перехвата в пространстве сеанса. Он учитывает архитектурные изменения, при которых ключевые указатели хранятся в непрозрачных структурах состояния сеанса, а не в простых глобальных секциях данных. Основной сценарий применения — продвинутое исследование графической подсистемы ядра для античита, защитной инженерии и экспериментов по реверс-инжинирингу.
