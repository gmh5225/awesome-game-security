Win32kHooker は、現行の Windows 上で win32k.sys 内の関数ディスパッチ経路を特定してフックする方法を示すカーネルドライバです。C++ とカーネル API を用い、プロセスコンテキストへのアタッチ、システムコールマッピング、実行時ディスアセンブリにより、セッション空間内のフック対象を解決します。主要なポインタが単純なグローバルデータセクションではなく、不透明なセッション状態構造体に格納されるアーキテクチャ上の変化に対応しています。主な用途は、アンチチート、防御的エンジニアリング、リバースエンジニアリング実験向けの高度なカーネルグラフィックスサブシステムの研究です。
