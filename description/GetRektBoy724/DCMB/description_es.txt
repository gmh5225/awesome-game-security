DCMB es un proyecto de controlador del kernel de Windows para localizar dinámicamente listas de callbacks sin depender de offsets codificados ni de firmas. Está escrito en C para la cadena de herramientas de controladores de Windows y cubre categorías de callbacks como notificaciones de proceso, hilo, imagen, registro, objeto y minifiltro. El proyecto se presenta como una implementación orientada al aprendizaje y expone sus hallazgos mediante salida de depuración en lugar de una API de producto pulida. Su caso de uso principal es la investigación de internos del kernel relacionados con anti-cheat, EDR e inspección de callbacks.
