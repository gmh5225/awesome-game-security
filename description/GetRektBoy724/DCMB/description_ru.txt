DCMB — проект драйвера ядра Windows для динамического поиска списков callback'ов без опоры на жёстко заданные смещения или сигнатуры. Он написан на C для инструментария драйверов Windows и охватывает категории callback'ов, такие как уведомления о процессах, потоках, образах, реестре, объектах и минифильтрах. Проект представлен как учебная реализация и сообщает результаты через отладочный вывод, а не через отполированный продуктовый API. Основной сценарий использования — исследование внутренних механизмов ядра, связанных с античитом, EDR и инспекцией callback'ов.
