DCMB è un progetto di driver del kernel Windows per individuare dinamicamente le liste di callback senza affidarsi a offset hardcodati o a signature. È scritto in C per la toolchain dei driver Windows e riguarda categorie di callback come notifiche di processo, thread, image, registry, object e minifilter. Il progetto è presentato come un'implementazione orientata all'apprendimento ed espone i risultati tramite output di debug piuttosto che tramite un'API di prodotto rifinita. Il caso d'uso principale è la ricerca sugli interni del kernel legati ad anti-cheat, EDR e ispezione dei callback.
