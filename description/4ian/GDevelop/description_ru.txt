GDevelop — это открытый кроссплатформенный инструмент и движок для создания игр, позволяющий делать 2D- и 3D-игры с помощью визуальной событийной логики, а не только традиционного программирования. Архитектура разделяет ядро проекта на C++ (GDCore) с разбором событий и генерацией кода, JavaScript-привязки (GDevelop.js), HTML5/JavaScript-рантайм на базе Pixi.js (GDJS) и десктопную IDE на React (newIDE), которая может экспортировать через Electron и связанные упаковщики. В кодовой базе есть обширная экосистема расширений: спрайты, поведения, тайловые карты, 3D-объекты и эффекты, аудио, ввод, сеть, реклама и средства отладки. Инструмент в первую очередь полезен разработчикам и преподавателям, создающим игры без тяжёлого программирования, а также исследователям открытой архитектуры игровых движков, визуального скриптинга и HTML5-пайплайнов экспорта.
