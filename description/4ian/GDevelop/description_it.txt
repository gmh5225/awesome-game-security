GDevelop è uno strumento e un motore open source multipiattaforma per la creazione di giochi che consente di realizzare titoli 2D e 3D con un sistema di logica basato su eventi visuali, invece di affidarsi solo alla programmazione tradizionale. L’architettura separa un core di progetto in C++ (GDCore) con parsing degli eventi e generazione del codice, binding JavaScript (GDevelop.js), un runtime HTML5/JavaScript basato su Pixi.js (GDJS) e un IDE desktop basato su React (newIDE) che può esportare tramite Electron e packager correlati. Il codice include un ampio ecosistema di estensioni che copre sprite, behavior, tile map, oggetti ed effetti 3D, audio, input, networking, annunci e strumenti di debug. È utile soprattutto a sviluppatori ed educatori che creano giochi senza programmazione pesante, e a ricercatori che studiano la progettazione di game engine aperti, lo scripting visuale e le pipeline di export HTML5.
