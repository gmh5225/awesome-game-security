GDevelop 是一款开源、跨平台的游戏创作工具与引擎，用户可通过基于可视化事件的逻辑系统构建 2D 与 3D 游戏，而无需完全依赖传统编程。其架构将 C++ 项目核心（GDCore，负责事件解析与代码生成）、JavaScript 绑定（GDevelop.js）、基于 Pixi.js 的 HTML5/JavaScript 运行时（GDJS），以及基于 React 的桌面 IDE（newIDE，可通过 Electron 及相关打包工具导出）分离开来。代码库包含庞大的扩展生态，涵盖精灵、行为、瓦片地图、3D 对象与特效、音频、输入、网络、广告与调试工具。它主要面向无需大量编程即可制作游戏的开发者与教育工作者，也适合研究开放游戏引擎设计、可视化脚本与 HTML5 导出流水线的研究人员。
