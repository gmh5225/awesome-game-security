GDevelop はオープンソースのクロスプラットフォーム向けゲーム制作ツール兼エンジンで、従来のコーディングだけに頼らず、ビジュアルなイベントベースのロジックで 2D／3D ゲームを作れます。アーキテクチャは、イベント解析とコード生成を担う C++ プロジェクトコア（GDCore）、JavaScript バインディング（GDevelop.js）、Pixi.js ベースの HTML5／JavaScript ランタイム（GDJS）、Electron など関連パッケージャ経由でエクスポートできる React 製デスクトップ IDE（newIDE）に分かれています。コードベースには、スプライト、ビヘイビア、タイルマップ、3D オブジェクトとエフェクト、オーディオ、入力、ネットワーキング、広告、デバッグツールを含む大規模な拡張エコシステムがあります。主にプログラミング負荷を抑えてゲームを作る開発者や教育者に有用で、オープンなゲームエンジン設計、ビジュアルスクリプティング、HTML5 エクスポートパイプラインを研究する研究者にも適しています。
