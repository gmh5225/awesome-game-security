GDevelop는 오픈소스 크로스플랫폼 게임 제작 도구이자 엔진으로, 전통적인 코딩만으로는 하지 않고 시각적 이벤트 기반 로직 시스템으로 2D 및 3D 게임을 만들 수 있습니다. 아키텍처는 이벤트 파싱과 코드 생성을 담당하는 C++ 프로젝트 코어(GDCore), JavaScript 바인딩(GDevelop.js), Pixi.js 기반 HTML5/JavaScript 런타임(GDJS), Electron 및 관련 패키저로 내보낼 수 있는 React 기반 데스크톱 IDE(newIDE)로 나뉩니다. 코드베이스에는 스프라이트, 동작(behavior), 타일맵, 3D 오브젝트와 이펙트, 오디오, 입력, 네트워킹, 광고, 디버깅 도구를 아우르는 대규모 확장 생태계가 포함되어 있습니다. 주로 무거운 프로그래밍 없이 게임을 만드는 개발자와 교육자에게 유용하며, 오픈 게임 엔진 설계, 비주얼 스크립팅, HTML5 내보내기 파이프라인을 연구하는 연구자에게도 적합합니다.
