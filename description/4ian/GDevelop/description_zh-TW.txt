GDevelop 是一款開源、跨平台的遊戲創作工具與引擎，使用者可透過基於視覺化事件的邏輯系統製作 2D 與 3D 遊戲，而不必完全依賴傳統程式設計。其架構將 C++ 專案核心（GDCore，負責事件解析與程式碼產生）、JavaScript 繫結（GDevelop.js）、以 Pixi.js 為基礎的 HTML5/JavaScript 執行環境（GDJS），以及以 React 為基礎的桌面 IDE（newIDE，可透過 Electron 及相關打包工具匯出）分開。程式碼庫包含龐大的擴充生態系統，涵蓋精靈、行為、圖磚地圖、3D 物件與特效、音訊、輸入、網路、廣告與除錯工具。它主要適用於無需大量程式設計即可製作遊戲的開發者與教育工作者，也適合研究開放遊戲引擎設計、視覺化腳本與 HTML5 匯出流程的研究人員。
