GDevelop is an open-source, cross-platform game creation tool and engine that lets users build 2D and 3D games with a visual events-based logic system rather than traditional coding alone. Its architecture splits a C++ project core (GDCore) with event parsing and code generation, JavaScript bindings (GDevelop.js), a Pixi.js-backed HTML5/JavaScript runtime (GDJS), and a React-based desktop IDE (newIDE) that can export via Electron and related packagers. The codebase includes a large extension ecosystem covering sprites, behaviors, tile maps, 3D objects and effects, audio, input, networking, ads, and debugging tools. It is primarily useful for game developers and educators making games without heavy programming, and for researchers studying open game-engine design, visual scripting, and HTML5 export pipelines.
