GDevelop est un outil et un moteur de création de jeux open source et multiplateforme qui permet de créer des jeux 2D et 3D avec un système de logique basé sur des événements visuels, plutôt que par le seul code traditionnel. Son architecture sépare un cœur de projet C++ (GDCore) assurant l’analyse des événements et la génération de code, des bindings JavaScript (GDevelop.js), un runtime HTML5/JavaScript basé sur Pixi.js (GDJS), et un IDE de bureau React (newIDE) pouvant exporter via Electron et des packagers associés. La base de code inclut un vaste écosystème d’extensions couvrant sprites, comportements, tile maps, objets et effets 3D, audio, entrées, réseau, publicités et outils de débogage. Elle est surtout utile aux développeurs et éducateurs qui créent des jeux sans programmation lourde, ainsi qu’aux chercheurs qui étudient la conception de moteurs de jeu ouverts, le scripting visuel et les pipelines d’export HTML5.
