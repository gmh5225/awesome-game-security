Этот проект посвящён поддельным сертификатам (fake Cert).
Репозиторий позволяет выполнять поддельную подпись без дополнительных зависимостей; все используемые бинарники входят в собственные комплекты разработчика Microsoft.
Он в первую очередь полезен исследователям низкого уровня для Windows, Linux и мобильных платформ, работающим в области some tricks / Windows ring3.
