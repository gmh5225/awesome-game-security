Dieses Projekt konzentriert sich auf gefälschte Zertifikate (fake Cert).
Dieses Repository ermöglicht Fake-Signing ohne zusätzliche Abhängigkeiten; alle verwendeten Binärdateien stammen aus Microsofts eigenen Entwicklungskits.
Es ist vor allem für Low-Level-Forscher zu Windows, Linux und Mobile nützlich, die im Bereich some tricks / Windows ring3 arbeiten.
