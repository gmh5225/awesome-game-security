Ce projet contient le code source du moteur Half-Life 1 et du SDK utilisés pour compiler et modifier les jeux classiques basés sur GoldSrc.
La base de code est principalement en C et C++ et comprend le client, le serveur, les interfaces moteur partagées, ainsi que des ressources de build spécifiques à la plateforme.
Il documente des workflows de compilation modernes pour Windows et Linux tout en préservant l'architecture d'origine orientée modding.
Son public principal est constitué des développeurs de mods et des chercheurs sur les moteurs qui ont besoin d'un accès direct au code de gameplay et aux internals du moteur legacy.
