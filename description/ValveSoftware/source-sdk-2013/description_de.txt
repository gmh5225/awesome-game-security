Dieses Projekt ist die Source SDK 2013-Codebasis zum Erstellen von Source-1-Spielmods.
Es enthält umfangreichen C++-Spiel- und Engine-Code für Titel wie Half-Life 2, Half-Life 2: Deathmatch und Team Fortress 2.
Das Repository stellt Build-Pipelines für Visual Studio unter Windows sowie Steam-Runtime-basierte Builds unter Linux bereit.
Es richtet sich an Entwickler, die nicht-kommerzielle Mods für die Source-Engine erstellen, anpassen und verteilen möchten.
