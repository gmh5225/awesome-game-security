Questo progetto è il codice base Source SDK 2013 per creare mod di giochi Source 1.
Include ampia quantità di codice C++ di gioco e del motore per titoli come Half-Life 2, Half-Life 2: Deathmatch e Team Fortress 2.
Il repository fornisce pipeline di build per Visual Studio su Windows e build basate su Steam Runtime su Linux.
È destinato agli sviluppatori che vogliono compilare, personalizzare e distribuire mod non commerciali del motore Source.
