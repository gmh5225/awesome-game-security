Этот проект — кодовая база Source SDK 2013 для создания модов игр на Source 1.
В неё входит обширный игровой и движковый код на C++ для таких игр, как Half-Life 2, Half-Life 2: Deathmatch и Team Fortress 2.
Репозиторий предоставляет сборочные конвейеры для Visual Studio в Windows и сборки на базе Steam Runtime в Linux.
Он предназначен для разработчиков, которые хотят собирать, настраивать и распространять некоммерческие моды для движка Source.
