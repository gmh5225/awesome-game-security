Este proyecto es el código base de Source SDK 2013 para crear mods de juegos Source 1.
Incluye una gran cantidad de código C++ de juego y del motor para títulos como Half-Life 2, Half-Life 2: Deathmatch y Team Fortress 2.
El repositorio ofrece pipelines de compilación para Visual Studio en Windows y builds basados en Steam Runtime en Linux.
Está pensado para desarrolladores que quieran crear, personalizar y distribuir mods no comerciales del motor Source.
