Ce projet est la base de code Source SDK 2013 pour créer des mods de jeux Source 1.
Il inclut d'importants volumes de code C++ de jeu et de moteur pour des titres tels que Half-Life 2, Half-Life 2: Deathmatch et Team Fortress 2.
Le dépôt fournit des pipelines de compilation pour Visual Studio sous Windows et des builds basés sur Steam Runtime sous Linux.
Il s'adresse aux développeurs qui souhaitent compiler, personnaliser et distribuer des mods non commerciaux du moteur Source.
