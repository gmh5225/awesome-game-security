Este proyecto es una biblioteca de transporte de red para juegos multiplataforma que ofrece mensajería fiable y no fiable mediante una API orientada a conexión.
Está implementada principalmente en C++ con interfaces C, builds de CMake y compatibilidad con entornos de escritorio, móvil y consolas.
Sus principales funciones incluyen fragmentación y reensamblado, fiabilidad avanzada de paquetes con vectores de ack, control de carriles de ancho de banda y herramientas detalladas de simulación de red.
También ofrece transporte cifrado y conectividad peer-to-peer con soporte de NAT traversal, lo que la hace adecuada para backends multijugador y redes de motores de juego.
