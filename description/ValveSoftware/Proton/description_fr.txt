Ce projet est Proton de Valve, une couche de compatibilité basée sur Wine qui permet d’exécuter des jeux Windows sur Linux via Steam. Il comprend des scripts de build, une infrastructure de build basée sur Docker, une configuration de préfixe par défaut, des manifests d’intégration Steam, ainsi que des correctifs de compatibilité pour faire tourner des binaires de jeux Windows sur Linux. Il est surtout utile aux chercheurs en jeux Linux et aux développeurs de couches de compatibilité qui étudient la traduction Windows vers Linux et la technologie Steam Play.
