Этот проект — Proton от Valve, слой совместимости на базе Wine, позволяющий запускать Windows-игры в Linux через Steam. Он включает скрипты сборки, инфраструктуру сборки на Docker, конфигурацию префикса по умолчанию, манифесты интеграции со Steam и исправления совместимости для запуска бинарников Windows-игр в Linux. В первую очередь полезен исследователям Linux-гейминга и разработчикам слоёв совместимости, изучающим перевод игр с Windows на Linux и технологию Steam Play.
