Dieses Projekt ist Valves Proton, eine auf Wine basierende Kompatibilitätsschicht, mit der Windows-Spiele über Steam unter Linux ausgeführt werden können. Es enthält Build-Skripte, eine Docker-basierte Build-Infrastruktur, eine Standard-Prefix-Konfiguration, Steam-Integrationsmanifeste sowie Kompatibilitätsanpassungen für die Ausführung von Windows-Spielbinärdateien unter Linux. Es ist vor allem für Linux-Gaming-Forscher und Entwickler von Kompatibilitätsschichten nützlich, die die Windows-zu-Linux-Spielübersetzung und die Steam-Play-Technologie untersuchen.
