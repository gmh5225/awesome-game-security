Этот проект представляет собой исследовательскую подборку, посвящённую объектам executive callback в Windows и их поведению во время выполнения. Он сочетает технические заметки с примерами kernel proof-of-concept на C, которые регистрируют, исследуют и анализируют активность callback. Материал охватывает несколько семейств callback в областях сети, состояния системы, загрузки и компонентов, связанных с безопасностью. В первую очередь полезен для исследования Windows internals, изучения видимости телеметрии, а также анализа в контексте античита или EDR.
