Este proyecto es el firmware FPGA Fullstealth PCILeech DMA de código abierto de Quantumstealth, con scripts de compilación Vivado para placas como M2, Squirrel, Captain 75T y Enigma X1.
El archivo incluye núcleos IP PCIe, archivos batch de generación y fuentes de firmware usados para construir dispositivos DMA orientados al sigilo.
Resulta especialmente útil para investigadores de seguridad en juegos que modelan amenazas DMA y estudian el comportamiento de firmwares al estilo Fullstealth.
