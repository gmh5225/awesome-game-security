Questo progetto è il firmware FPGA Fullstealth PCILeech DMA open source di Quantumstealth, con script di build Vivado per schede come M2, Squirrel, Captain 75T ed Enigma X1.
L'archivio include core IP PCIe, file batch di generazione e sorgenti firmware usati per costruire dispositivi DMA orientati allo stealth.
È principalmente utile ai ricercatori di sicurezza nei giochi per modellare le minacce DMA e studiare il comportamento dei firmware in stile Fullstealth.
