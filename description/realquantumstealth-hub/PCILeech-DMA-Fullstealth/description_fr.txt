Ce projet est le firmware FPGA Fullstealth PCILeech DMA open source de Quantumstealth, accompagné de scripts de build Vivado pour des cartes telles que M2, Squirrel, Captain 75T et Enigma X1.
L'archive inclut des cœurs IP PCIe, des fichiers batch de génération et les sources du firmware utilisés pour construire des périphériques DMA axés sur la furtivité.
Il est principalement utile aux chercheurs en sécurité des jeux pour modéliser les menaces DMA et étudier le comportement des firmwares de type Fullstealth.
