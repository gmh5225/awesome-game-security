Dieses Projekt konzentriert sich auf die Einführung: Frühere Ansätze zur generischen Erkennung opaker Prädikate prüfen, ob ein bedingter Sprung einen invarianten Ausdruck enthält. Der Schaden bleibt auf der Ebene des Basic Blocks (oder der Funktion) lokalisiert, unabhängig davon, wie der invariante Ausdruck eines opaken Prädikats konstruiert ist. Es ist vor allem für Spiele-Sicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich von Cheats / Binary-Ninja-Plugins untersuchen.
