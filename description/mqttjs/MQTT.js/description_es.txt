Biblioteca cliente MQTT completa para Node.js y navegadores que admite los protocolos MQTT v3.1, v3.1.1 y v5.0 sobre TCP, TLS, WebSocket y WSS. Ofrece mensajería publish/subscribe con QoS 0/1/2, reconexión automática, last-will-and-testament y herramientas CLI (mqtt pub/sub), con ejemplos de integración para Electron y Vite.
