Questo progetto è uno strumento a riga di comando per trovare gadget di return-oriented programming (ROP) nei file binari. È scritto in Python e usa il motore di disassemblaggio Capstone per supportare binari ELF, PE, Mach-O e grezzi su molte architetture CPU. Offre funzioni di filtraggio, ricerca e generazione opzionale di catene ROP per i flussi di sviluppo di exploit. Il caso d'uso principale è lo sfruttamento binario e la ricerca di reverse engineering.
