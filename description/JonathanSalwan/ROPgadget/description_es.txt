Este proyecto es una herramienta de línea de comandos para encontrar gadgets de programación orientada al retorno (ROP) en archivos binarios. Está escrita en Python y utiliza el motor de desensamblado Capstone para admitir binarios ELF, PE, Mach-O y en bruto en muchas arquitecturas de CPU. Ofrece funciones de filtrado, búsqueda y generación opcional de cadenas ROP para flujos de desarrollo de exploits. El caso de uso principal es la explotación binaria y la investigación en ingeniería inversa.
