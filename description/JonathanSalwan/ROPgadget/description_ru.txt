Этот проект — инструмент командной строки для поиска гаджетов return-oriented programming (ROP) в бинарных файлах. Он написан на Python и использует движок дизассемблирования Capstone для поддержки ELF, PE, Mach-O и сырых бинарников на многих архитектурах CPU. Он предлагает фильтрацию, поиск и опциональную генерацию ROP-цепочек для рабочих процессов разработки эксплойтов. Основной сценарий использования — бинарная эксплуатация и исследования в области реверс-инжиниринга.
