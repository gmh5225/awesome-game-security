Questo progetto si concentra sulla demo FPS di UE4.
Questo codice «gonfia» il template rendendolo molto più complesso del necessario se vuoi semplicemente iniziare con C++ in Unreal Engine.
È soprattutto utile per sviluppatori di giochi, reverse engineer e creatori di tool che lavorano nell'ambito dello sviluppo / del codice sorgente di giochi.
