Dieses Projekt konzentriert sich auf die UE4-FPS-Demo.
Dieser Code „bläht“ die Vorlage auf, sodass sie deutlich komplexer wirkt als nötig, wenn Sie lediglich mit C++ in Unreal Engine starten möchten.
Er ist vor allem für Spieleentwickler, Reverse Engineers und Tool-Entwickler im Bereich Spieleentwicklung / Quellcode nützlich.
