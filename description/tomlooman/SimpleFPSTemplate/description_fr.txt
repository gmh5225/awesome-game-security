Ce projet se concentre sur la démo FPS UE4.
Ce code « alourdit » le modèle pour le rendre bien plus complexe que nécessaire si vous souhaitez simplement démarrer avec le C++ dans Unreal Engine.
Il est surtout utile aux développeurs de jeux, aux reverse engineers et aux créateurs d'outils travaillant dans le domaine du développement / du code source de jeux.
