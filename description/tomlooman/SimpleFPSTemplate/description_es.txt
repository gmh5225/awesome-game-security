Este proyecto se centra en la demo FPS de UE4.
Este código «infla» la plantilla para que parezca mucho más compleja de lo necesario si solo quieres empezar con C++ en Unreal Engine.
Resulta especialmente útil para desarrolladores de juegos, ingenieros inversos y creadores de herramientas que trabajan en el ámbito del desarrollo / código fuente de juegos.
