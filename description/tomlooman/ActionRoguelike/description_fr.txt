Ce projet se concentre sur un jeu Roguelike uE (Unreal Engine).
Cela peut affecter la stabilité et le multijoueur n'est pas toujours pris en charge tant que les systèmes ne se sont pas stabilisés au fil du temps.
Il est principalement utile aux développeurs de jeux, ingénieurs en rétro-ingénierie et créateurs d'outils travaillant dans le domaine du développement / du code source de jeux.
