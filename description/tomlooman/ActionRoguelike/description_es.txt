Este proyecto se centra en un juego Roguelike de uE (Unreal Engine).
Esto puede afectar la estabilidad y aún no siempre admite multijugador hasta que los sistemas se estabilicen con el tiempo.
Es principalmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo / código fuente de juegos.
