Questo progetto si concentra su un gioco FPS uE4.
Il corso C++ include la costruzione di un framework di gioco in C++ con un "action system" e codice event-driven per scrivere codice di gioco flessibile e riutilizzabile nello stile Unreal.
È principalmente utile per sviluppatori di giochi, reverse engineer e creatori di tool che lavorano nell'area dello sviluppo / codice sorgente dei giochi.
