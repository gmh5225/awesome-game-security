Este proyecto se centra en un juego FPS de uE4.
El curso de C++ incluye la construcción de un framework de juego en C++ con un «action system» y código impulsado por eventos para escribir código de juego flexible y reutilizable al estilo Unreal.
Resulta especialmente útil para desarrolladores de juegos, ingenieros de ingeniería inversa y creadores de herramientas que trabajan en el área de desarrollo / código fuente de juegos.
