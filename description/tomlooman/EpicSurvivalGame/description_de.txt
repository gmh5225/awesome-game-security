Dieses Projekt konzentriert sich auf ein uE4-FPS-Spiel.
Der C++-Kurs umfasst den Aufbau eines C++-Spiel-Frameworks inklusive eines „Action Systems“ und ereignisgesteuertem Code für flexiblen und wiederverwendbaren Spielcode auf die Unreal-Art.
Es ist vor allem nützlich für Spieleentwickler, Reverse Engineers und Tool-Entwickler, die im Bereich Spielentwicklung / Quellcode arbeiten.
