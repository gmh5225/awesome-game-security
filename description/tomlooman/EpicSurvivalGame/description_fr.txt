Ce projet se concentre sur un jeu FPS uE4.
Le cours C++ porte notamment sur la construction d'un framework de jeu C++ comprenant un « action system » et du code piloté par événements pour écrire du code de jeu flexible et réutilisable à la manière Unreal.
Il est principalement utile aux développeurs de jeux, aux ingénieurs en reverse engineering et aux créateurs d'outils travaillant dans le domaine du développement / du code source de jeux.
