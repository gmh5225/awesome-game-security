Этот проект посвящён FPS-игре на uE4.
Курс по C++ включает построение игрового фреймворка на C++ с «action system» и событийно-ориентированным кодом для гибкого и переиспользуемого игрового кода в стиле Unreal.
Он в первую очередь полезен разработчикам игр, специалистам по реверс-инжинирингу и создателям инструментов, работающим в области разработки игр / исходного кода.
