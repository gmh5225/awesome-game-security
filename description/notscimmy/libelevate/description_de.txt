Dieses Projekt ist eine C++-Bibliothek zum Erhöhen von Handles und zum Umgehen von Handle-Zugriffsbeschränkungen unter Windows. Sie nutzt Treiber- oder Kernel-Primitive, um Prozesse mit vollständigen Zugriffsrechten zu öffnen, selbst wenn sie durch Anti-Cheat- oder Sicherheitssoftware geschützt sind. Die Bibliothek bietet eine saubere API zum Abrufen erhöhter Handles für Speicherlese-/Schreiboperationen. Sie richtet sich an Spielsicherheitsforscher, die Handle-Elevationstechniken und Anti-Cheat-Handle-Schutzmechanismen untersuchen.
