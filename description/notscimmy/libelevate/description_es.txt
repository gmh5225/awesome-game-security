Este proyecto es una biblioteca C++ para elevar handles y eludir las restricciones de acceso a handles en Windows. Aprovecha primitivas a nivel de controlador o kernel para abrir procesos con derechos de acceso completos incluso cuando están protegidos por anti-cheat o software de seguridad. La biblioteca ofrece una API limpia para obtener handles elevados que pueden usarse en operaciones de lectura/escritura de memoria. Está dirigida a investigadores de seguridad en juegos que estudian técnicas de elevación de handles y mecanismos de protección de handles anti-cheat.
