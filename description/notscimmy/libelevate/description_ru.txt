Этот проект — библиотека на C++ для повышения привилегий дескрипторов (handles) и обхода ограничений доступа к ним в Windows. Она использует примитивы уровня драйвера или ядра, чтобы открывать процессы с полными правами доступа даже при защите античитом или средствами безопасности. Библиотека предоставляет чистый API для получения повышенных дескрипторов, которые можно использовать для чтения и записи памяти. Предназначена для исследователей игровой безопасности, изучающих техники повышения дескрипторов и механизмы защиты дескрипторов античита.
