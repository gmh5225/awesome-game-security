React Native ShieldScan — нативный модуль, обеспечивающий обнаружение угроз во время выполнения и защиту экрана для приложений React Native на iOS и Android. Он проверяет jailbreak или root, инструментацию Frida, подключение отладчика, среды эмулятора или симулятора, фреймворки runtime-хукинга, такие как Xposed и Substrate, а также режим разработчика, и может возвращать взвешенную оценку риска устройства с уровнями угроз. Защита экрана включает размытие фона при переходах в переключателе приложений, предотвращение скриншотов и записи экрана, а также обнаружение активной записи экрана, когда это позволяют API платформы. Библиотека реализована на TypeScript с нативным кодом на Kotlin и Swift и предназначена для разработчиков мобильных приложений и команд безопасности, которым нужны сигналы о компрометации на стороне клиента и защита от захвата экрана в продуктах на React Native.
