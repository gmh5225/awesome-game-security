React Native ShieldScan è un modulo nativo che fornisce rilevamento di sicurezza a runtime e funzioni di protezione dello schermo per app React Native su iOS e Android. Verifica jailbreak o root, strumentazione Frida, collegamento di debugger, ambienti emulatore o simulatore, framework di hooking a runtime come Xposed e Substrate e la modalità sviluppatore, e può restituire una valutazione ponderata del rischio del dispositivo con livelli di minaccia. La sicurezza dello schermo copre la sfocatura dello sfondo durante le transizioni dell’app switcher, la prevenzione di screenshot e registrazioni dello schermo e il rilevamento della registrazione attiva dello schermo quando le API della piattaforma lo consentono. La libreria è implementata in TypeScript con codice nativo Kotlin e Swift, ed è rivolta a sviluppatori di app mobili e team di sicurezza che necessitano di segnali di compromissione lato client e protezione dalla cattura nei prodotti React Native.
