React Native ShieldScan は、iOS および Android 上の React Native アプリ向けにランタイムセキュリティ検出と画面保護機能を提供するネイティブモジュールです。ジェイルブレイクやルート化、Frida による計装、デバッガのアタッチ、エミュレータ／シミュレータ環境、Xposed や Substrate などのランタイムフッキングフレームワーク、開発者モードを検査し、脅威レベル付きの重み付けデバイスリスク評価を返すことができます。画面セキュリティでは、アプリスイッチャー遷移時のバックグラウンドぼかし、スクリーンショットおよび画面録画の防止、プラットフォーム API が許可する場合のアクティブな画面録画検出をカバーします。本ライブラリは TypeScript と Kotlin／Swift のネイティブコードで実装されており、React Native 製品でクライアント側の侵害シグナルとキャプチャ保護を必要とするモバイルアプリ開発者およびセキュリティチーム向けです。
