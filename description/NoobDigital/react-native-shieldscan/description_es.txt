React Native ShieldScan es un módulo nativo que ofrece detección de seguridad en tiempo de ejecución y funciones de protección de pantalla para aplicaciones React Native en iOS y Android. Comprueba jailbreak o root, instrumentación con Frida, adjunto de depuradores, entornos de emulador o simulador, frameworks de hooking en tiempo de ejecución como Xposed y Substrate, y el modo desarrollador, y puede devolver una evaluación ponderada del riesgo del dispositivo con niveles de amenaza. La seguridad de pantalla cubre el desenfoque de fondo en las transiciones del conmutador de aplicaciones, la prevención de capturas y grabaciones de pantalla, y la detección de grabación de pantalla activa cuando las API de la plataforma lo permiten. La biblioteca está implementada en TypeScript con código nativo en Kotlin y Swift, y está orientada a desarrolladores de aplicaciones móviles y equipos de seguridad que necesitan señales de compromiso del lado del cliente y protección frente a capturas en productos React Native.
