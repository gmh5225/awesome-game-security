React Native ShieldScan is a native module that provides runtime security detection and screen-protection features for React Native apps on iOS and Android. It checks for jailbreak or root, Frida instrumentation, debugger attachment, emulator or simulator environments, runtime hooking frameworks such as Xposed and Substrate, and developer mode, and can return a weighted device risk assessment with threat levels. Screen security covers background blur on app switcher transitions, screenshot and screen-recording prevention, and active screen-recording detection where the platform APIs allow it. The library is implemented in TypeScript with Kotlin and Swift native code, and is aimed at mobile app developers and security teams who need client-side compromise signals and capture protection in React Native products.
