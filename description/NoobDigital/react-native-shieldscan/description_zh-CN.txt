React Native ShieldScan 是一个原生模块，为运行在 iOS 和 Android 上的 React Native 应用提供运行时安全检测与屏幕保护能力。它可检测越狱或 Root、Frida 插桩、调试器附着、模拟器或模拟环境、Xposed 与 Substrate 等运行时 Hook 框架，以及开发者模式，并返回带威胁等级的加权设备风险评估。屏幕安全涵盖应用切换器过渡时的后台模糊、防止截屏与录屏，以及在平台 API 允许时对正在进行的屏幕录制进行检测。该库以 TypeScript 实现，并配合 Kotlin 与 Swift 原生代码，面向需要在 React Native 产品中获取客户端被入侵信号与录屏/截屏防护的移动应用开发者与安全团队。
