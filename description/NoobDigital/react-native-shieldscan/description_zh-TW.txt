React Native ShieldScan 是一個原生模組，為執行於 iOS 與 Android 的 React Native 應用提供執行階段安全偵測與螢幕保護功能。它可檢查越獄或 Root、Frida 插樁、除錯器附加、模擬器或模擬環境、Xposed 與 Substrate 等執行階段 Hook 框架，以及開發人員模式，並可回傳含威脅等級的加權裝置風險評估。螢幕安全涵蓋切換應用程式時的背景模糊、防止螢幕截圖與螢幕錄製，以及在平台 API 允許時對進行中的螢幕錄製進行偵測。此函式庫以 TypeScript 實作，並搭配 Kotlin 與 Swift 原生程式碼，適合需要在 React Native 產品中取得用戶端遭入侵訊號與擷取防護的行動應用開發者與資安團隊。
