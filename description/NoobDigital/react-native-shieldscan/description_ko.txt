React Native ShieldScan은 iOS 및 Android용 React Native 앱에 런타임 보안 탐지와 화면 보호 기능을 제공하는 네이티브 모듈입니다. 탈옥 또는 루팅, Frida 계측, 디버거 연결, 에뮬레이터 또는 시뮬레이터 환경, Xposed 및 Substrate 같은 런타임 후킹 프레임워크, 개발자 모드를 검사하며, 위협 수준이 포함된 가중 기기 위험 평가를 반환할 수 있습니다. 화면 보안은 앱 스위처 전환 시 백그라운드 블러, 스크린샷 및 화면 녹화 방지, 플랫폼 API가 허용하는 경우 활성 화면 녹화 탐지를 다룹니다. 이 라이브러리는 TypeScript와 Kotlin·Swift 네이티브 코드로 구현되었으며, React Native 제품에서 클라이언트 측 침해 신호와 캡처 보호가 필요한 모바일 앱 개발자 및 보안 팀을 대상으로 합니다.
