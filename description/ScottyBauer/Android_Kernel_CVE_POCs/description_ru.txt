Этот проект — коллекция программ proof-of-concept для воспроизведения известных уязвимостей ядра Android. Он в основном написан на C и содержит множество примеров по конкретным CVE, в том числе связанных с драйверами производителей и компонентами Wi-Fi. Код демонстрирует низкоуровневые техники: специально сформированные вызовы ioctl, состояния гонки и манипуляции с памятью на стороне драйверов. Полезен в первую очередь исследователям безопасности ядра, которым нужно воспроизводить ошибки, проверять меры защиты и изучать примитивы эксплуатации Android.
