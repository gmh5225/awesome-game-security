Этот проект — форк QEMU, доработанный для поддержки виртуализации Apple Silicon и эмуляции устройств iOS. Он включает полный исходный код эмулятора QEMU с платформенными изменениями для запуска ARM-виртуальных машин на Mac с Apple Silicon. В первую очередь полезен исследователям безопасности iOS и разработчикам эмуляторов, которым нужна аппаратно ускоренная ARM-виртуализация на macOS для тестирования и анализа приложений.
