Этот проект — proof-of-concept обхода BattlEye в пользовательском режиме, демонстрирующий ранее уязвимый путь загрузки. Он включает C++-имплантер и примерный рабочий процесс DLL, который перехватывает CreateFileW и манипулирует проверками файлов, чтобы выдать полезную нагрузку за доверенный модуль. Код оформлен как проекты Visual Studio с примерами внедрения в целевой игровой процесс и обработки экспортируемых callback’ов перехватчиков. Основное назначение — исследование уязвимостей античита и исторический анализ слабых мест проверки доверия в пользовательском режиме.
