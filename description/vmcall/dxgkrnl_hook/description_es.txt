Este proyecto se centra en el hook del subsistema gráfico del kernel para permitir la manipulación del búfer de pantalla; para más información, consulte el artículo técnico. Los recuadros de jugadores en este vídeo se dibujan mediante ese hook. Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / renderizado / dibujo.
