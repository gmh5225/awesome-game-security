Этот проект посвящён хуку графической подсистемы ядра, позволяющему манипулировать экранным буфером; подробнее см. в статье. Рамки игроков в этом видео нарисованы с помощью этого хука. Он в основном полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов / рендера / отрисовки.
