Этот проект — минимальная реализация спуфинга адреса возврата для x64 без использования обработчиков исключений. Он сочетает C++ и ассемблерные заглушки для установки и использования поддельных целей возврата с очень низкими накладными расходами в простых сценариях перенаправления вызовов. Репозиторий документирует допущения о сохранении nonvolatile-регистров в соглашении о вызовах Windows x64. В основном применяется для исследований скрытой манипуляции потоком вызовов в контексте реверс-инжиниринга и разработки читов.
