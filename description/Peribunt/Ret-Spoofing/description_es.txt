Este proyecto es una implementación mínima de spoofing de direcciones de retorno x64 que evita el uso de manejadores de excepciones. Combina C++ y stubs en ensamblador para establecer y usar destinos de retorno falsos con muy bajo overhead en escenarios simples de redirección de llamadas. El repositorio documenta las hipótesis sobre la preservación de registros no volátiles en la convención de llamadas Windows x64. Se usa principalmente para investigación sobre manipulación sigilosa del flujo de llamadas en ingeniería inversa y desarrollo de cheats.
