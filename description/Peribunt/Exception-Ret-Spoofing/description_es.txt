Este proyecto es una prueba de concepto mínima de spoofing de dirección de retorno x64 implementada mediante un flujo de manejador de excepciones. El ejemplo en C++ muestra cómo construir una ruta de llamada falsificada y encadenarla con gadgets, documentando a la vez las compensaciones prácticas. Destaca tanto la comodidad como las restricciones de la convención de llamada que afectan la fiabilidad y el rendimiento. El código es principalmente útil para la investigación de flujo de control de bajo nivel en evasión de anti-cheat y experimentos de herramientas ofensivas.
