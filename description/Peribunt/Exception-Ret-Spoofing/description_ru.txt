Этот проект — минимальное proof of concept для spoofing адреса возврата x64, реализованное через поток обработчика исключений. Пример на C++ показывает, как построить поддельный путь вызова и связать его с gadgets, фиксируя практические компромиссы. Он подчёркивает как удобство метода, так и ограничения calling convention, влияющие на надёжность и производительность. Код в основном полезен для исследований низкоуровневого потока управления в обходе античита и экспериментах с наступательным инструментарием.
