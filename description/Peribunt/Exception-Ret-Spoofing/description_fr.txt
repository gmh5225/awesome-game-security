Ce projet est une preuve de concept minimale de spoofing d'adresse de retour x64 implémentée via un flux de gestionnaire d'exceptions. L'exemple C++ montre comment construire un chemin d'appel usurpé et l'enchaîner avec des gadgets, tout en documentant les compromis pratiques. Il met en évidence à la fois la commodité et les contraintes de convention d'appel qui affectent la fiabilité et les performances. Le code est surtout utile pour la recherche sur le flot de contrôle bas niveau dans le contournement d'anti-cheat et les expériences d'outillage offensif.
