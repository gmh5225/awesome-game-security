Questo progetto è una proof of concept in user-mode per verificare se un indirizzo virtuale genererebbe un fault prima di dereferenziarlo direttamente. Si basa sul comportamento dell’istruzione AVX2 VPGATHER e sulla gestione vettorizzata delle eccezioni per inferire l’accessibilità dell’indirizzo con effetti collaterali ridotti sullo stato della memoria target. L’implementazione include controlli di inizializzazione per il supporto CPU e una semplice API per sonde ripetute di validità degli indirizzi. È rilevante soprattutto per il probing furtivo della memoria nel reverse engineering, nel bypass anti-cheat e nella ricerca di sicurezza a basso livello.
