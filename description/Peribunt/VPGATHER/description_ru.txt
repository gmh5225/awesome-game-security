Этот проект — пользовательский proof of concept для проверки того, вызовет ли виртуальный адрес fault до прямого разыменования. Он опирается на поведение инструкции AVX2 VPGATHER и на vectored exception handling, чтобы сделать вывод о доступности адреса с меньшими побочными эффектами для состояния целевой памяти. Реализация включает проверки инициализации поддержки CPU и простой API для повторных проверок валидности адресов. В основном проект относится к скрытому зондированию памяти в реверс-инжиниринге, обходе античита и низкоуровневых исследованиях безопасности.
