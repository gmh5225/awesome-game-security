Este proyecto es una prueba de concepto en modo usuario para comprobar si una dirección virtual provocaría un fallo antes de desreferenciarla directamente. Se basa en el comportamiento de la instrucción AVX2 VPGATHER y en el manejo vectorizado de excepciones para inferir la accesibilidad de la dirección con menores efectos secundarios sobre el estado de memoria objetivo. La implementación incluye comprobaciones de inicialización del soporte de CPU y una API sencilla para sondeos repetidos de validez de direcciones. Es relevante principalmente para el sondeo furtivo de memoria en ingeniería inversa, evasión de anti-cheat e investigación de seguridad de bajo nivel.
