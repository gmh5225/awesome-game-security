Dieses Projekt ist ein Usermode-Proof-of-Concept zum Testen, ob eine virtuelle Adresse einen Fault auslösen würde, bevor sie direkt dereferenziert wird. Es stützt sich auf das Verhalten der AVX2-Instruktion VPGATHER sowie auf vectored Exception Handling, um die Zugreifbarkeit einer Adresse mit geringeren Nebenwirkungen auf den Zielspeicherzustand abzuleiten. Die Implementierung umfasst Initialisierungsprüfungen für CPU-Unterstützung und eine einfache API für wiederholte Gültigkeitstests von Adressen. Sie ist vor allem für stealthy Memory Probing in Reverse Engineering, Anti-Cheat-Bypass und Low-Level-Sicherheitsforschung relevant.
