FRIEND est un plugin IDA qui enrichit le désassemblage en affichant directement une documentation plus complète des registres et des instructions dans la vue d'analyse. Il étend le comportement du processeur avec des composants tiers tels que Capstone et ajoute des indices contextuels, des liens vers de la documentation externe et des résumés de fonctions dans les vues de désassemblage et de décompilateur. Le code source utilise C++ avec CMake et s'intègre au SDK IDA, avec un support optionnel du SDK Hex-Rays, sous Windows, Linux et macOS. Il est principalement conçu pour les ingénieurs en rétro-ingénierie qui ont besoin de comprendre le code plus rapidement, y compris les analystes en sécurité des jeux.
