FRIEND — плагин для IDA, который обогащает дизассемблирование более подробной документацией по регистрам и инструкциям прямо в окне анализа. Он расширяет поведение процессора сторонними компонентами, такими как Capstone, и добавляет контекстные подсказки, ссылки на внешнюю документацию и сводки функций в представлениях дизассемблера и декомпилятора. Кодовая база написана на C++ с CMake и интегрируется с IDA SDK, с опциональной поддержкой Hex-Rays SDK, на Windows, Linux и macOS. Плагин в первую очередь предназначен для инженеров по реверс-инжинирингу, которым нужно быстрее понимать код, в том числе для аналитиков игровой безопасности.
