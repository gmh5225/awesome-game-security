FRIEND は、解析ビュー内でより充実したレジスタおよび命令のドキュメントを直接表示し、逆アセンブリを強化する IDA プラグインです。Capstone などのサードパーティコンポーネントでプロセッサ動作を拡張し、逆アセンブリおよびデコンパイラビューに文脈ヒント、外部ドキュメントへのリンク、関数サマリを追加します。コードベースは C++ と CMake で構成され、IDA SDK と統合し、任意で Hex-Rays SDK にも対応し、Windows、Linux、macOS で動作します。主に、より迅速なコード理解を必要とするリバースエンジニア（ゲームセキュリティ分析者を含む）向けに設計されています。
