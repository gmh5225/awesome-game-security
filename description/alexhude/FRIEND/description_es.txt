FRIEND es un complemento de IDA que enriquece el desensamblado con documentación más completa de registros e instrucciones directamente en la vista de análisis. Extiende el comportamiento del procesador con componentes de terceros como Capstone y añade pistas contextuales, enlaces a documentación externa y resúmenes de funciones en las vistas de desensamblado y descompilador. La base de código usa C++ con CMake e integra el SDK de IDA, con soporte opcional del SDK de Hex-Rays, en Windows, Linux y macOS. Está pensado principalmente para ingenieros de ingeniería inversa que necesitan comprender el código más rápido, incluidos los analistas de seguridad de juegos.
