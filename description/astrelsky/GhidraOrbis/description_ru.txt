Этот проект — расширение Ghidra для бинарников PlayStation 4 Orbis и связанных артефактов прошивки. Оно включает загрузчики, анализаторы, скрипты, данные о системных вызовах и ресурсы сопоставления NID для улучшения восстановления символов и платформенно-специфичного анализа. Код написан на Java с Gradle, интегрируется с моделью расширений Ghidra и при необходимости может использоваться вместе с дополнительными средствами анализа классов C++. Основной сценарий применения — реверс-инжиниринг консолей и исследования безопасности игр на ПО PS4.
