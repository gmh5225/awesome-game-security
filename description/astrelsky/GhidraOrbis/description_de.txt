Dieses Projekt ist eine Ghidra-Erweiterung für PlayStation-4-Orbis-Binärdateien und zugehörige Firmware-Artefakte. Sie umfasst Loader, Analyzer, Skripte, Syscall-Daten und NID-Mapping-Ressourcen, um die Symbolwiederherstellung und plattformspezifische Analyse zu verbessern. Der Code basiert auf Java und Gradle, integriert sich in Ghidras Erweiterungsmodell und kann optional mit zusätzlichen Werkzeugen zur C++-Klassenanalyse kombiniert werden. Der Hauptanwendungsfall ist Console-Reverse-Engineering und Game-Security-Forschung an PS4-Software.
