Este proyecto es una extensión de Ghidra centrada en binarios PlayStation 4 Orbis y artefactos de firmware relacionados. Incluye cargadores, analizadores, scripts, datos de syscalls y recursos de mapeo NID para mejorar la recuperación de símbolos y el análisis específico de la plataforma. El código está basado en Java y Gradle, se integra con el modelo de extensiones de Ghidra y puede combinarse opcionalmente con herramientas adicionales de análisis de clases C++. Su caso de uso principal es la ingeniería inversa de consolas y la investigación en seguridad de juegos sobre software de PS4.
