Ce projet est une extension Ghidra qui analyse les métadonnées de classes C++ et les informations de type à l’exécution dans des binaires compilés. Écrit en Java avec des scripts de build Gradle, il ajoute des analyseurs pour les modèles RTTI de GCC, Clang et MSVC, les vtables, les constructeurs et destructeurs, ainsi que la reconstruction de l’héritage. Il fournit aussi des outils orientés visualisation, tels que des vues de hiérarchie de classes et des scripts qui s’intègrent aux API de l’extension. Le public visé est celui des reverse engineers, y compris les analystes en sécurité des jeux travaillant sur de grandes cibles C++ natives.
