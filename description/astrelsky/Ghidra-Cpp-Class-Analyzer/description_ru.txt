Этот проект — расширение Ghidra, которое анализирует метаданные классов C++ и информацию о типах времени выполнения в скомпилированных бинарниках. Написано на Java со сборочными скриптами Gradle и добавляет анализаторы для моделей RTTI GCC, Clang и MSVC, vtable, конструкторов и деструкторов, а также реконструкции наследования. Также предоставляет средства визуализации, например представления иерархии классов и скрипты, интегрирующиеся с API расширения. Целевая аудитория — специалисты по реверс-инжинирингу, включая аналитиков игровой безопасности, работающих с крупными нативными C++-целями.
