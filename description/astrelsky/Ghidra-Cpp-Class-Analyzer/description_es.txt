Este proyecto es una extensión de Ghidra que analiza metadatos de clases C++ e información de tipos en tiempo de ejecución en binarios compilados. Escrita en Java con scripts de compilación Gradle, añade analizadores para los modelos RTTI de GCC, Clang y MSVC, vtables, constructores y destructores, y la reconstrucción de la herencia. También ofrece herramientas orientadas a la visualización, como vistas de jerarquía de clases y scripts que se integran con las API de la extensión. El público previsto son ingenieros inversos, incluidos analistas de seguridad de juegos que trabajan con grandes objetivos nativos en C++.
