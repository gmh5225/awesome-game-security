Dieses Projekt ist eine Ghidra-Erweiterung, die C++-Klassenmetadaten und Laufzeittypinformationen in kompilierten Binärdateien analysiert. In Java mit Gradle-Build-Skripten geschrieben, ergänzt sie Analyzer für die RTTI-Modelle von GCC, Clang und MSVC, Vtables, Konstruktoren und Destruktoren sowie die Rekonstruktion von Vererbung. Sie bietet außerdem visualisierungsorientierte Werkzeuge wie Klassenhierarchie-Ansichten und Skripte, die sich in die Erweiterungs-APIs integrieren. Die Zielgruppe sind Reverse Engineers, einschließlich Game-Security-Analysten, die an großen nativen C++-Zielen arbeiten.
