Questo progetto è un’estensione di Ghidra che analizza i metadati delle classi C++ e le informazioni sui tipi a runtime nei binari compilati. Scritta in Java con script di build Gradle, aggiunge analyzer per i modelli RTTI di GCC, Clang e MSVC, vtable, costruttori e distruttori, e la ricostruzione dell’ereditarietà. Fornisce anche strumenti orientati alla visualizzazione, come viste della gerarchia delle classi e script che si integrano con le API dell’estensione. Il pubblico previsto sono i reverse engineer, inclusi gli analisti di security dei giochi che lavorano su grandi target nativi in C++.
