Ce projet est un plugin Gradle archivé destiné à simplifier la création d'extensions Ghidra. Il configure les classpaths à partir d'une installation Ghidra locale, intègre les scripts de build d'extension et inclut des tâches d'aide pour la configuration de l'IDE. L'implémentation est principalement en Java et Groovy et se concentre sur l'ergonomie des développeurs autour du packaging d'extensions. Il est surtout pertinent pour les développeurs d'outils de reverse engineering qui maintiennent des plugins Ghidra personnalisés.
