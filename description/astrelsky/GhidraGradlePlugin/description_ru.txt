Этот проект — архивный плагин Gradle, предназначенный для упрощения сборки расширений Ghidra. Он настраивает classpath на основе локальной установки Ghidra, подключает скрипты сборки расширений и включает вспомогательные задачи для настройки IDE. Реализация выполнена в основном на Java и Groovy и ориентирована на удобство разработчика при упаковке расширений. Наиболее полезен разработчикам инструментов реверс-инжиниринга, которые поддерживают собственные плагины Ghidra.
