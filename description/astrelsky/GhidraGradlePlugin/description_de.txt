Dieses Projekt ist ein archiviertes Gradle-Plugin, das den Build von Ghidra-Erweiterungen vereinfachen soll. Es konfiguriert Classpaths aus einer lokalen Ghidra-Installation, bindet Erweiterungs-Buildskripte ein und enthält Hilfsaufgaben für die IDE-Einrichtung. Die Implementierung ist hauptsächlich in Java und Groovy gehalten und konzentriert sich auf Entwicklerergonomie beim Packaging von Erweiterungen. Es ist vor allem für Reverse-Engineering-Tool-Entwickler relevant, die benutzerdefinierte Ghidra-Plugins pflegen.
