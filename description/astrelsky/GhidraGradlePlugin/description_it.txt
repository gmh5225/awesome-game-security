Questo progetto è un plugin Gradle archiviato pensato per semplificare la creazione di estensioni Ghidra. Configura i classpath a partire da un'installazione locale di Ghidra, collega gli script di build delle estensioni e include task di supporto per la configurazione dell'IDE. L'implementazione è principalmente in Java e Groovy e si concentra sull'ergonomia dello sviluppatore nel packaging delle estensioni. È particolarmente rilevante per gli sviluppatori di strumenti di reverse engineering che mantengono plugin Ghidra personalizzati.
