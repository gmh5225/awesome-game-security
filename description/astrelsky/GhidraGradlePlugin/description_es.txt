Este proyecto es un complemento de Gradle archivado pensado para simplificar la creación de extensiones de Ghidra. Configura los classpaths a partir de una instalación local de Ghidra, integra scripts de build de extensiones e incluye tareas auxiliares para la configuración del IDE. La implementación está principalmente en Java y Groovy y se centra en la ergonomía del desarrollador en torno al empaquetado de extensiones. Es especialmente relevante para desarrolladores de herramientas de ingeniería inversa que mantienen plugins personalizados de Ghidra.
