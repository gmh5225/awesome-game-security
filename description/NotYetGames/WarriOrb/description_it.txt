WarriOrb è il rilascio pubblico del codice sorgente di un action-platformer impegnativo realizzato con Unreal Engine. Il progetto include ampio codice di gameplay in C++, configurazione del motore, file di localizzazione e impostazioni di packaging per più piattaforme di destinazione. Il repository si concentra esplicitamente sul codice e omette gli asset commerciali del gioco e alcuni plugin proprietari, quindi non è un pacchetto completo di ricostruzione. Il suo uso principale è lo studio dell'architettura e dei pattern di implementazione di un gioco Unreal di produzione.
