WarriOrb est la publication publique du code source d'un jeu d'action-plateforme exigeant réalisé avec Unreal Engine. Le projet comprend un vaste code de gameplay en C++, la configuration du moteur, des fichiers de localisation et des paramètres d'empaquetage pour plusieurs plateformes cibles. Le dépôt se concentre explicitement sur le code et omet les assets commerciaux ainsi que certains plugins propriétaires, ce n'est donc pas un package complet de reconstruction. Son usage principal est l'étude de l'architecture et des schémas d'implémentation d'un jeu Unreal de production.
