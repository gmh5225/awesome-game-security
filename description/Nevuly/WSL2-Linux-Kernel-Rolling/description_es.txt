Este proyecto ofrece builds rolling del kernel Linux estable y guías de configuración para WSL2.
Se basa en documentación Markdown y automatización CI que reconstruye cuando aparecen versiones estables más nuevas del kernel.
El repositorio incluye instrucciones de compilación para objetivos x86 y ARM64, y hace hincapié en flujos de trabajo de kernel reproducibles.
Está pensado para usuarios de WSL2 y desarrolladores de bajo nivel que necesitan kernels actualizados para pruebas e investigación.
