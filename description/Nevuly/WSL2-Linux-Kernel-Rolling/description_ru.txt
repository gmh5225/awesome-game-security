Этот проект предоставляет rolling-сборки стабильного ядра Linux и руководства по конфигурации для WSL2.
Он опирается на документацию в Markdown и CI-автоматизацию, которая пересобирает ядро при появлении новых стабильных релизов.
Репозиторий включает инструкции по сборке для целей x86 и ARM64 и делает акцент на воспроизводимых рабочих процессах с ядром.
Он предназначен для пользователей WSL2 и низкоуровневых разработчиков, которым нужны актуальные ядра для тестирования и исследований.
