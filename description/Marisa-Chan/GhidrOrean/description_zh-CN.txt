GhidrOrean 是 Deathway 的 Oreans Unvirtualizer 的重新实现，以 Ghidra Python 脚本形式用于分析并还原受 Oreans 虚拟化保护的代码。它将反虚拟化逻辑移植并扩展到 Ghidra 中，使研究人员能在现代逆向工程环境中检查、改进并运行相关工具。项目包含面向 Oreans 虚拟机系列（如 CISC、RISC、FISH 和 TIGER）的汇编器与指令配置，其中 CISC 支持已标记为完成，TIGER 也基本完成。入口是主 Orean Ghidra 脚本，它会从可配置的工作目录加载这些配置。该项目面向处理 Themida、WinLicense 及相关 Oreans Code Virtualizer 保护的逆向工程师与游戏安全研究人员。
