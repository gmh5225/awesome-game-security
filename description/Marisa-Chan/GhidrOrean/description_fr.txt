GhidrOrean est une réimplémentation de l'Oreans Unvirtualizer de Deathway sous forme de scripts Python Ghidra, destinée à analyser et récupérer le code protégé par la virtualisation Oreans. Il porte et étend la logique d'unvirtualizer dans Ghidra afin que les chercheurs puissent inspecter, améliorer et exécuter l'outil dans un environnement moderne de reverse engineering. Le projet inclut un assembleur et une configuration d'instructions pour les familles de VM Oreans telles que CISC, RISC, FISH et TIGER, le support CISC étant marqué comme complet et TIGER largement achevé. L'entrée se fait via le script Ghidra Orean principal, qui charge ces configurations depuis un répertoire de travail configurable. Il s'adresse aux reverse engineers et aux chercheurs en sécurité des jeux confrontés à Themida, WinLicense et aux protections Oreans Code Virtualizer associées.
