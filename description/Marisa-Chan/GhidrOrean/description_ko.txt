GhidrOrean은 Deathway의 Oreans Unvirtualizer를 Ghidra Python 스크립트로 재구현한 것으로, Oreans 가상화로 보호된 코드를 분석하고 복구하는 데 사용됩니다. Unvirtualizer 로직을 Ghidra로 이식·확장하여 연구자가 현대적인 리버스 엔지니어링 환경에서 도구를 검사·개선·실행할 수 있게 합니다. CISC, RISC, FISH, TIGER 등 Oreans VM 계열용 어셈블러와 명령어 설정을 포함하며, CISC 지원은 완료로 표시되어 있고 TIGER도 대체로 완성된 상태입니다. 진입점은 메인 Orean Ghidra 스크립트로, 구성 가능한 작업 디렉터리에서 해당 설정을 로드합니다. Themida, WinLicense 및 관련 Oreans Code Virtualizer 보호를 다루는 리버스 엔지니어와 게임 보안 연구자를 대상으로 합니다.
