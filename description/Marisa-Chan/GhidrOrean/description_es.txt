GhidrOrean es una reimplementación del Oreans Unvirtualizer de Deathway como scripts de Python para Ghidra, orientada a analizar y recuperar código protegido por la virtualización de Oreans. Porta y amplía la lógica del unvirtualizer en Ghidra para que los investigadores puedan inspeccionar, mejorar y ejecutar las herramientas en un entorno moderno de ingeniería inversa. El proyecto incluye ensamblador y configuración de instrucciones para familias de VM Oreans como CISC, RISC, FISH y TIGER; el soporte CISC está marcado como completo y TIGER está en gran medida terminado. La entrada es el script principal Orean de Ghidra, que carga esas configuraciones desde un directorio de trabajo configurable. Está dirigido a ingenieros inversos e investigadores de seguridad en juegos que trabajan con Themida, WinLicense y protecciones relacionadas de Oreans Code Virtualizer.
