GhidrOrean è una reimplementazione dell'Oreans Unvirtualizer di Deathway come script Python per Ghidra, per analizzare e recuperare codice protetto dalla virtualizzazione Oreans. Porta ed estende la logica dell'unvirtualizer in Ghidra così che i ricercatori possano ispezionare, migliorare ed eseguire gli strumenti in un ambiente moderno di reverse engineering. Il progetto include assembler e configurazione delle istruzioni per famiglie di VM Oreans come CISC, RISC, FISH e TIGER; il supporto CISC è indicato come completo e TIGER è in gran parte finito. L'ingresso avviene tramite lo script Ghidra Orean principale, che carica tali configurazioni da una directory di lavoro configurabile. È rivolto a reverse engineer e ricercatori di game security che affrontano Themida, WinLicense e le relative protezioni Oreans Code Virtualizer.
