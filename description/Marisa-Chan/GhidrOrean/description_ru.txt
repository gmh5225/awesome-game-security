GhidrOrean — это реимплементация Oreans Unvirtualizer от Deathway в виде Python-скриптов для Ghidra, предназначенная для анализа и восстановления кода, защищённого виртуализацией Oreans. Проект переносит и расширяет логику unvirtualizer в Ghidra, чтобы исследователи могли изучать, дорабатывать и запускать инструменты в современной среде реверс-инжиниринга. Включает ассемблер и конфигурации инструкций для семейств VM Oreans, таких как CISC, RISC, FISH и TIGER; поддержка CISC отмечена как завершённая, а TIGER в основном готова. Точка входа — основной скрипт Orean для Ghidra, который загружает эти конфигурации из настраиваемого рабочего каталога. Предназначен для реверс-инженеров и исследователей игровой безопасности, работающих с Themida, WinLicense и связанными защитами Oreans Code Virtualizer.
