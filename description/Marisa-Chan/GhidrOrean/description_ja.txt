GhidrOrean は、Deathway の Oreans Unvirtualizer を Ghidra の Python スクリプトとして再実装したもので、Oreans 仮想化で保護されたコードの解析と復元を支援します。Unvirtualizer のロジックを Ghidra に移植・拡張し、研究者が現代的なリバースエンジニアリング環境内でツールを検査・改良・実行できるようにします。Oreans VM 系列（CISC、RISC、FISH、TIGER など）向けのアセンブラと命令設定を含み、CISC 対応は完了、TIGER もほぼ完成とされています。エントリポイントはメインの Orean Ghidra スクリプトで、設定可能な作業ディレクトリからこれらの設定を読み込みます。Themida、WinLicense、および関連する Oreans Code Virtualizer 保護を扱うリバースエンジニアやゲームセキュリティ研究者向けです。
