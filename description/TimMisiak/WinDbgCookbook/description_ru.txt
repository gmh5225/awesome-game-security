Этот проект — коллекция практических скриптов автоматизации WinDbg и запросов для отладчика. Он в основном написан на JavaScript для модели данных WinDbg и включает утилиты для проверки зависимостей, сбора стека, обнаружения повреждения стека и анализа time-travel debugging. В README также приведено множество готовых к запуску примеров команд dx для проверки модулей, потоков и окружения. Основной сценарий использования — повышение продуктивности Windows reverse engineers и исследователей безопасности, ориентированных на отладку.
