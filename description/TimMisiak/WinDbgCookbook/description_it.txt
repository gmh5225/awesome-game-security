Questo progetto è una raccolta di script pratici di automazione WinDbg e di query per il debugger. È scritto principalmente in JavaScript per il data model di WinDbg e include utility per l'ispezione delle dipendenze, la raccolta dello stack, il rilevamento della corruzione dello stack e l'analisi del time-travel debugging. Il README fornisce anche molti esempi di comandi dx pronti all'uso per ispezionare moduli, thread e ambiente. Il caso d'uso principale è migliorare la produttività di reverse engineer Windows e di ricercatori di sicurezza incentrati sul debugging.
