Dieses Projekt ist eine Sammlung praktischer WinDbg-Automatisierungsskripte und Debugger-Abfragen. Es ist hauptsächlich in JavaScript für das WinDbg-Datenmodell geschrieben und enthält Hilfsmittel zur Abhängigkeitsprüfung, Stack-Erfassung, Erkennung von Stack-Korruption sowie zur Time-Travel-Debugging-Analyse. Die README enthält außerdem viele sofort nutzbare dx-Befehlsbeispiele zur Inspektion von Modulen, Threads und der Umgebung. Der Hauptzweck ist die Produktivitätssteigerung für Windows-Reverse-Engineers und debuggingorientierte Sicherheitsforscher.
