Este proyecto es una colección de scripts prácticos de automatización de WinDbg y consultas para el depurador. Está escrito principalmente en JavaScript para el modelo de datos de WinDbg e incluye utilidades para inspección de dependencias, recopilación de pilas, detección de corrupción de pila y análisis de depuración time-travel. El README también ofrece muchos ejemplos de comandos dx listos para ejecutar para inspeccionar módulos, hilos y el entorno. Su caso de uso principal es mejorar la productividad de los ingenieros de ingeniería inversa en Windows y de los investigadores de seguridad centrados en la depuración.
