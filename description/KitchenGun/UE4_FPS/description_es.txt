Este proyecto es un pequeño ejemplo de disparos en primera persona creado con Unreal Engine 4.
Combina clases C++ de UE4 con assets de Blueprint, mapas y archivos estándar de configuración de entrada y del motor.
La mayor parte del repositorio consiste en contenido de assets de Unreal, con una capa ligera de código de gameplay nativo.
Las tecnologías principales son Unreal Engine 4, C++ y las herramientas Blueprint.
Es útil como entorno de aprendizaje para el desarrollo de FPS y como objetivo práctico para la investigación a nivel de motor.
