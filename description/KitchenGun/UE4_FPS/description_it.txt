Questo progetto è un piccolo esempio di sparatutto in prima persona realizzato con Unreal Engine 4.
Combina classi C++ di UE4 con asset Blueprint, mappe e file di configurazione standard di input e del motore.
La maggior parte del repository è costituita da contenuto asset Unreal, con un livello leggero di codice di gameplay nativo.
Le tecnologie principali sono Unreal Engine 4, C++ e gli strumenti Blueprint.
È utile come sandbox di apprendimento per lo sviluppo FPS e come obiettivo pratico per la ricerca a livello di motore.
