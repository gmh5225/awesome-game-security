Ce projet est un petit exemple de jeu de tir à la première personne réalisé avec Unreal Engine 4.
Il combine des classes C++ UE4 avec des assets Blueprint, des maps et des fichiers de configuration d'entrée et de moteur standard.
La majeure partie du dépôt est constituée de contenu d'assets Unreal, avec une couche légère de code de gameplay natif.
Les technologies principales sont Unreal Engine 4, C++ et les outils Blueprint.
Il convient comme bac à sable d'apprentissage pour le développement FPS et comme cible pratique pour la recherche au niveau du moteur.
