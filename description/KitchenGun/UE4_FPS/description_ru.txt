Этот проект — небольшой образец шутера от первого лица, созданный на Unreal Engine 4.
Он сочетает классы C++ UE4 с ресурсами Blueprint, картами и стандартными файлами конфигурации ввода и движка.
Большая часть репозитория состоит из контента Unreal-ассетов с лёгким слоем нативного кода геймплея.
Основные технологии — Unreal Engine 4, C++ и инструменты Blueprint.
Полезен как учебная песочница для разработки FPS и как практическая цель для исследований на уровне движка.
