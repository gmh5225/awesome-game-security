Dieses Projekt ist ein kompakter Manual-Map-DLL-Injector für Windows-Prozesse.
Es unterstützt x86- und x64-Ziele, optionales Entfernen von PE-Headern und Abschnitten, konfigurierbare Schutzeinstellungen sowie x64-Ausnahmebehandlungsszenarien.
Der Code ist in C++ geschrieben und enthält Beispielnutzung für den Loader sowie Test-DLLs zur schnellen Validierung.
Der Hauptanwendungsfall ist das Studium von Prozessinjektion und Manual-Mapping-Techniken.
