Este proyecto es un inyector DLL de manual-map compacto para procesos de Windows.
Admite objetivos x86 y x64, eliminación opcional de encabezados PE y secciones, ajustes de protección configurables y escenarios de manejo de excepciones x64.
El código está en C++ e incluye un ejemplo de uso del cargador además de DLL de prueba para una validación rápida.
Su principal caso de uso es estudiar la inyección de procesos y las técnicas de manual mapping.
