Este proyecto es una utilidad para mapear controladores del kernel de Windows sin firmar explotando una vía vulnerable de un controlador Intel.
Ofrece múltiples opciones de mapeo, pasos de reducción de rastros para estructuras comunes de contabilidad del kernel y compatibilidad con muchas versiones modernas de Windows.
El código está en C++ e incluye proyectos auxiliares para el manejo de símbolos y offsets, incluidos flujos de trabajo basados en PDB.
Se usa ampliamente en investigación de kernel, pruebas de controladores y experimentación de bypass de anti-cheat.
