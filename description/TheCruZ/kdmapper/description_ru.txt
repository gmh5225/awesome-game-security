Этот проект — утилита для загрузки (mapping) неподписанных драйверов ядра Windows путём эксплуатации уязвимого пути через драйвер Intel.
Она предоставляет несколько вариантов mapping, шаги по снижению следов в распространённых учётных структурах ядра и поддержку совместимости со многими современными сборками Windows.
Кодовая база написана на C++ и включает вспомогательные проекты для работы с символами и смещениями, в том числе рабочие процессы на основе PDB.
Её широко используют в исследованиях ядра, тестировании драйверов и экспериментах по обходу античита.
