Questo progetto è un'utilità per mappare driver del kernel Windows non firmati sfruttando un percorso vulnerabile di un driver Intel.
Offre più opzioni di mapping, passaggi di riduzione delle tracce per le strutture di contabilità del kernel comuni e supporto di compatibilità su molte build moderne di Windows.
Il codice è in C++ e include progetti di supporto per la gestione di simboli e offset, inclusi workflow basati su PDB.
È ampiamente usato nella ricerca sul kernel, nei test dei driver e negli esperimenti di bypass degli anti-cheat.
