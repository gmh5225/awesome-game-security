Ce projet est un utilitaire permettant de mapper des pilotes noyau Windows non signés en exploitant une voie de pilote Intel vulnérable.
Il propose plusieurs options de mapping, des étapes de réduction des traces pour les structures de comptabilité noyau courantes, et une compatibilité avec de nombreuses versions modernes de Windows.
Le code source est en C++ et inclut des projets d'aide pour la gestion des symboles et des offsets, y compris des workflows basés sur les PDB.
Il est largement utilisé pour la recherche noyau, les tests de pilotes et les expérimentations de contournement d'anti-cheat.
