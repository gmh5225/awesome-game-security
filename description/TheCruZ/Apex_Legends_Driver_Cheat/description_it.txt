Questo progetto è un framework di cheat esterno assistito dal kernel pensato per Apex Legends.
Combina un driver kernel di Windows per lettura e scrittura della memoria, un controller in user mode in C++ e un overlay trasparente per visuali in stile ESP e logica di mira.
Il flusso di lavoro si basa su tecniche di mapping di driver vulnerabili per caricare il componente kernel.
È usato principalmente per lo sviluppo di cheat e la ricerca sul bypass degli anti-cheat.
