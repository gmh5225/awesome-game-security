Este proyecto es un framework de cheat externo asistido por el kernel orientado a Apex Legends.
Combina un controlador del kernel de Windows para lectura y escritura de memoria, un controlador en modo usuario en C++ y una superposición transparente para visuales tipo ESP y lógica de puntería.
El flujo de trabajo se basa en técnicas de mapeo de controladores vulnerables para cargar el componente del kernel.
Se usa principalmente para el desarrollo de cheats y la investigación de bypass de anti-cheat.
