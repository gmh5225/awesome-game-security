Этот проект — внешний чит-фреймворк с поддержкой ядра, ориентированный на Apex Legends.
Он сочетает драйвер ядра Windows для чтения и записи памяти, пользовательский контроллер на C++ и прозрачный оверлей для визуалов в стиле ESP и логики прицеливания.
Рабочий процесс опирается на техники маппинга уязвимых драйверов для загрузки компонента ядра.
В основном используется для разработки читов и исследования обхода античита.
