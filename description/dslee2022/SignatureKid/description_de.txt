Ein header-only C++-Tool, das Authenticode-Digitalsignaturen von signierten PE-Dateien stiehlt und auf unsignierte Ziel-Executables kopiert, anschließend Windows-Registry-Internals hookt, damit die kopierte Signatur für das Betriebssystem gültig erscheint. Es manipuliert die WIN_CERTIFICATE-Struktur im Security Directory der PE und patcht die Zertifikatsvertrauensprüfung auf Systemebene. Es ist vor allem für Sicherheitsforscher nützlich, die Code-Signing-Bypass-Techniken untersuchen, sowie für Anti-Cheat-Ingenieure, die die Robustheit signaturbasierter Vertrauensprüfung bewerten.
