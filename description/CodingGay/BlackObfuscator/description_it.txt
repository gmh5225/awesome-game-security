Questo progetto è uno strumento di ofuscazione del flusso di controllo DEX per applicazioni Android.
È scritto principalmente in Java e si basa su una pipeline dex2jar modificata per trasformare il bytecode senza alterare il comportamento dell'app.
Lo strumento supporta una profondità di ofuscazione configurabile, il targeting dei package e l'elaborazione basata su regole, e include anche integrazioni GUI e plugin per Android Studio.
È destinato alla protezione delle applicazioni mobili e alla ricerca sulla resistenza al reverse engineering.
