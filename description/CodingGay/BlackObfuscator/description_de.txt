Dieses Projekt ist ein DEX-Kontrollfluss-Obfuskierungswerkzeug für Android-Anwendungen.
Es ist hauptsächlich in Java geschrieben und basiert auf einer modifizierten dex2jar-Pipeline, um Bytecode zu transformieren, ohne das App-Verhalten zu ändern.
Das Tool unterstützt eine konfigurierbare Obfuskierungstiefe, Paket-Targeting und regelbasierte Verarbeitung und bietet begleitende GUI- sowie Android-Studio-Plugin-Integrationen.
Es ist für den Schutz mobiler Anwendungen und die Forschung zur Widerstandsfähigkeit gegen Reverse Engineering gedacht.
