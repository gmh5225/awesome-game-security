Este proyecto es una herramienta de ofuscación de flujo de control DEX para aplicaciones Android.
Está escrita principalmente en Java y se basa en un pipeline dex2jar modificado para transformar el bytecode sin cambiar el comportamiento de la app.
La herramienta admite profundidad de ofuscación configurable, segmentación por paquetes y procesamiento basado en reglas, y también cuenta con integraciones de GUI y plugin de Android Studio.
Está pensada para la protección de aplicaciones móviles y la investigación sobre resistencia a la ingeniería inversa.
