Этот проект — инструмент обфускации потока управления DEX для приложений Android.
Он написан в основном на Java и основан на модифицированном конвейере dex2jar для преобразования байткода без изменения поведения приложения.
Инструмент поддерживает настраиваемую глубину обфускации, таргетинг пакетов и обработку по правилам, а также имеет сопутствующие GUI и плагин для Android Studio.
Предназначен для защиты мобильных приложений и исследований устойчивости к обратной инженерии.
