Questo progetto è OpenProcmon, un monitor di processi open source per Windows ispirato a Sysinternals Process Monitor. Cattura l'attività di sistema in tempo reale, tra cui creazione e terminazione dei processi, operazioni sul file system, accesso al registro, attività di rete e caricamento di DLL, tramite ETW (Event Tracing for Windows) e driver minifilter. L'applicazione C++ offre un'interfaccia grafica con filtri, evidenziazione ed esportazione. È rivolta ad amministratori di sistema, analisti di malware e ricercatori di sicurezza che monitorano l'attività di sistema su Windows.
