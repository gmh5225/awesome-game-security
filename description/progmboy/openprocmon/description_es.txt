Este proyecto es OpenProcmon, un monitor de procesos de código abierto para Windows inspirado en Sysinternals Process Monitor. Captura actividad del sistema en tiempo real, incluida la creación y terminación de procesos, operaciones del sistema de archivos, acceso al registro, actividad de red y carga de DLL, mediante ETW (Event Tracing for Windows) y controladores minifilter. La aplicación en C++ ofrece una interfaz gráfica con filtrado, resaltado y exportación. Está dirigida a administradores de sistemas, analistas de malware e investigadores de seguridad que monitorizan la actividad del sistema Windows.
