Этот проект — OpenProcmon, открытый монитор процессов для Windows, вдохновлённый Sysinternals Process Monitor. Он фиксирует активность системы в реальном времени: создание и завершение процессов, операции с файловой системой, доступ к реестру, сетевую активность и загрузку DLL — с помощью ETW (Event Tracing for Windows) и драйверов minifilter. Приложение на C++ предоставляет графический интерфейс с фильтрацией, подсветкой и экспортом. Оно предназначено для системных администраторов, аналитиков вредоносного ПО и исследователей безопасности, отслеживающих активность системы Windows.
