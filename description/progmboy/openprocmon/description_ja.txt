本プロジェクトは OpenProcmon で、Sysinternals Process Monitor に着想を得た Windows 向けオープンソースのプロセス監視ツールです。ETW（Event Tracing for Windows）とミニフィルタードライバーを用い、プロセスの生成・終了、ファイルシステム操作、レジストリアクセス、ネットワーク活動、DLL の読み込みなど、リアルタイムのシステム活動を捕捉します。C++ 製のアプリケーションはフィルタリング、ハイライト、エクスポート機能付きの GUI を備え、システム管理者、マルウェア解析者、Windows のシステム活動を監視するセキュリティ研究者を対象としています。
