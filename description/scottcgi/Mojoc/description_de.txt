Dieses Projekt ist eine plattformübergreifende, Open-Source-Spiele-Engine in reinem C für mobile Spiele.
Sie basiert auf OpenGLES3 und ist in C99 geschrieben.
Sie ist vor allem nützlich für Spieleentwickler, Reverse Engineers und Tooling-Builder, die im Bereich Spieleentwicklung / Quellcode arbeiten.
