Этот проект — Intel Performance Counter Monitor (PCM), набор инструментов для мониторинга метрик производительности CPU, пропускной способности памяти, пропускной способности PCIe и энергопотребления на процессорах Intel. Кодовая база на C++ поддерживает развёртывание в Docker, мониторинг CXL, а также содержит подробную документацию по пользовательским параметрам сборки и переменным окружения. Он в первую очередь полезен инженерам по производительности и исследователям безопасности, профилирующим системную производительность и поведение аппаратных счётчиков.
