Este proyecto es Intel Performance Counter Monitor (PCM), un conjunto de herramientas para supervisar métricas de rendimiento de la CPU, ancho de banda de memoria, rendimiento PCIe y consumo de energía en procesadores Intel. La base de código C++ admite despliegue con Docker, supervisión CXL y una documentación extensa sobre opciones de compilación personalizadas y variables de entorno. Resulta especialmente útil para ingenieros de rendimiento e investigadores de seguridad que perfilan el rendimiento a nivel de sistema y el comportamiento de los contadores de hardware.
