Ce projet est Intel Performance Counter Monitor (PCM), une suite d'outils permettant de surveiller les métriques de performance du CPU, la bande passante mémoire, le débit PCIe et la consommation électrique sur les processeurs Intel. La base de code C++ prend en charge le déploiement Docker, la surveillance CXL, ainsi qu'une documentation complète sur les options de compilation personnalisées et les variables d'environnement. Elle est surtout utile aux ingénieurs de performance et aux chercheurs en sécurité qui profilent les performances système et le comportement des compteurs matériels.
