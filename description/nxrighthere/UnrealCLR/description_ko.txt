이 프로젝트는 UnrealCLR로, .NET 런타임(CoreCLR)을 Unreal Engine에 임베드하여 C#으로 게임 로직을 개발할 수 있게 하는 플러그인입니다. Actor, 컴포넌트, 입력, 물리, 렌더링, 블루프린트 시스템을 포함한 UE4/UE5 핵심 시스템에 대한 .NET 바인딩을 제공합니다. 개발자는 네이티브 C++ 및 블루프린트 시스템과 상호 운용되는 게임플레이 코드를 C#으로 작성할 수 있습니다. 게임 로직에는 C#을 사용하면서 UE의 네이티브 렌더링과 도구를 활용하려는 Unreal Engine 개발자를 대상으로 합니다.
