Questo progetto è UnrealCLR, un plugin che incorpora il runtime .NET (CoreCLR) in Unreal Engine, consentendo di sviluppare la logica di gioco in C#. Fornisce binding .NET per i sistemi principali di UE4/UE5, inclusi actor, componenti, input, fisica, rendering e il sistema blueprint. Gli sviluppatori possono scrivere codice di gameplay in C# che interoperano con C++ nativo e i blueprint. È pensato per gli sviluppatori Unreal Engine che preferiscono C# per la logica di gioco sfruttando al contempo il rendering nativo e gli strumenti di UE.
