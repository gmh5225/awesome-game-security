Этот проект — UnrealCLR, плагин, который встраивает среду выполнения .NET (CoreCLR) в Unreal Engine и позволяет разрабатывать игровую логику на C#. Он предоставляет .NET-привязки к основным системам UE4/UE5, включая actors, компоненты, ввод, физику, рендеринг и систему blueprint. Разработчики могут писать игровой код на C#, взаимодействующий с нативным C++ и blueprint. Плагин рассчитан на разработчиков Unreal Engine, которые предпочитают C# для игровой логики, используя при этом нативный рендеринг и инструменты UE.
