Ce projet est UnrealCLR, un plugin qui intègre le runtime .NET (CoreCLR) dans Unreal Engine, permettant de développer la logique de jeu en C#. Il fournit des bindings .NET pour les systèmes centraux d'UE4/UE5, notamment les actors, composants, entrées, physique, rendu et le système de blueprints. Les développeurs peuvent écrire du code de gameplay en C# qui interopère avec le C++ natif et les blueprints. Il s'adresse aux développeurs Unreal Engine qui préfèrent le C# pour la logique de jeu tout en tirant parti du rendu natif et des outils d'UE.
