Este proyecto es UnrealCLR, un plugin que integra el runtime de .NET (CoreCLR) en Unreal Engine, permitiendo desarrollar la lógica del juego en C#. Proporciona bindings de .NET para los sistemas principales de UE4/UE5, incluidos actors, componentes, entrada, física, renderizado y el sistema de blueprints. Los desarrolladores pueden escribir código de gameplay en C# que interoperan con C++ nativo y blueprints. Está dirigido a desarrolladores de Unreal Engine que prefieren C# para la lógica del juego mientras aprovechan el renderizado nativo y las herramientas de UE.
