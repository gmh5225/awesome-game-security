Dieses Projekt ist UnrealCLR, ein Plugin, das die .NET-Laufzeit (CoreCLR) in Unreal Engine einbettet und so die Entwicklung von Spiellogik in C# ermöglicht. Es bietet .NET-Bindings für zentrale UE4/UE5-Systeme, darunter Actors, Komponenten, Eingabe, Physik, Rendering und das Blueprint-System. Entwickler können Gameplay-Code in C# schreiben, der mit nativem C++ und Blueprints zusammenarbeitet. Es richtet sich an Unreal-Engine-Entwickler, die C# für die Spiellogik bevorzugen und zugleich das native Rendering und die Tools von UE nutzen möchten.
