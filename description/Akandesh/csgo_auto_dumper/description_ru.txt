Этот проект — утилита автоматизации на C++ для Windows, которая отслеживает обновления Counter-Strike: Global Offensive и запускает дамп оффсетов. Она использует команды steamcmd, разбор build ID и цикл с таймером для обнаружения новых сборок игры. При обнаружении обновления запускается локальный исполняемый dumper и последующие скрипты для обновления сгенерированных данных. Основное назначение — поддерживать актуальные артефакты для реверс-инжиниринга с минимальной ручной работой.
