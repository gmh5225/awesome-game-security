Questo progetto è un'utilità di automazione Windows in C++ che monitora gli aggiornamenti di Counter-Strike: Global Offensive e avvia il dumping degli offset. Usa comandi steamcmd, l'analisi dei build ID e un ciclo temporizzato per rilevare nuove build del gioco. Quando viene trovato un aggiornamento, avvia un eseguibile dumper locale e script successivi per aggiornare i dati generati. È pensato principalmente per mantenere aggiornati gli artefatti di reverse engineering con il minimo intervento manuale.
