Este proyecto es una utilidad de automatización en C++ para Windows que rastrea las actualizaciones de Counter-Strike: Global Offensive y dispara el volcado de offsets. Usa comandos de steamcmd, el análisis de build ID y un bucle temporizado para detectar nuevas builds del juego. Cuando encuentra una actualización, lanza un ejecutable dumper local y scripts posteriores para refrescar los datos generados. Está pensado principalmente para mantener actualizados los artefactos de ingeniería inversa con un mínimo de trabajo manual.
