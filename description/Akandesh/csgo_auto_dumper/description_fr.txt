Ce projet est un utilitaire d'automatisation Windows en C++ qui suit les mises à jour de Counter-Strike: Global Offensive et déclenche le dumping d'offsets. Il utilise des commandes steamcmd, l'analyse des build ID et une boucle temporisée pour détecter les nouvelles builds du jeu. Lorsqu'une mise à jour est trouvée, il lance un exécutable dumper local et des scripts de suivi pour actualiser les données générées. Il est principalement destiné à maintenir à jour les artefacts de reverse engineering avec un minimum de travail manuel.
