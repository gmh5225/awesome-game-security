Этот проект — конвейер на C++ для автоматического дампа и разбора данных игры Rust. Он отслеживает идентификаторы сборок Steam, запускает скрипты дампа и разбирает сгенерированные dump.cs и данные скриптов с помощью извлечения на основе регулярных выражений. Затем инструмент формирует структурированные результаты: JSON, заголовки C++ и константы C#, включая варианты для зашифрованных полей. В основном им пользуются исследователи игровой безопасности и разработчики инструментов, которым нужны синхронизированные offset-данные Rust в нескольких форматах.
