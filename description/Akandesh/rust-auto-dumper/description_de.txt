Dieses Projekt ist eine C++-Pipeline zum automatischen Dumping und Parsen für das Spiel Rust. Es überwacht Steam-Build-IDs, führt Dump-Skripte aus und parst die erzeugten dump.cs- und Skriptdaten per regexbasierter Extraktion. Anschließend erzeugt das Tool strukturierte Ausgaben wie JSON, C++-Header und C#-Konstanten, einschließlich Varianten für verschlüsselte Felder. Es wird vor allem von Game-Security-Forschern und Tool-Entwicklern genutzt, die synchronisierte Rust-Offset-Daten in mehreren Formaten benötigen.
