Questo progetto è una pipeline C++ di dump e parsing automatici per il gioco Rust. Monitora gli ID di build Steam, esegue script di dump e analizza i dump.cs e i dati degli script generati con estrazione basata su espressioni regolari. Lo strumento produce poi output strutturati come JSON, header C++ e costanti C#, comprese varianti per campi crittografati. È usato soprattutto da ricercatori di security nei giochi e da sviluppatori di tooling che necessitano di offset Rust sincronizzati in più formati.
