Este proyecto es una canalización C++ de volcado y análisis automáticos para el juego Rust. Supervisa los identificadores de compilación de Steam, ejecuta scripts de volcado y analiza los dump.cs y datos de scripts generados mediante extracción basada en expresiones regulares. A continuación emite salidas estructuradas como JSON, cabeceras C++ y constantes C#, incluidas variantes para campos cifrados. Lo usan sobre todo investigadores de seguridad en juegos y desarrolladores de herramientas que necesitan offsets de Rust sincronizados en varios formatos.
