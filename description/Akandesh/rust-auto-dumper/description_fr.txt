Ce projet est un pipeline C++ de dump et d'analyse automatiques pour le jeu Rust. Il surveille les identifiants de build Steam, exécute des scripts de dump, puis analyse les dump.cs et données de scripts générés via une extraction basée sur des expressions régulières. L'outil produit ensuite des sorties structurées telles que JSON, en-têtes C++ et constantes C#, y compris des variantes pour les champs chiffrés. Il s'adresse surtout aux chercheurs en sécurité des jeux et aux développeurs d'outils qui ont besoin d'offsets Rust synchronisés dans plusieurs formats.
