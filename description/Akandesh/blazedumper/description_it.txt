Questo progetto è un database di offset mantenuto per le strutture di memoria di Counter-Strike: Global Offensive. Fornisce output leggibili dalle macchine come JSON e definizioni di offset in C++ e C# per l'integrazione diretta in strumenti esterni. Il repository include pattern di signature e dati di configurazione progettati per essere aggiornati automaticamente da una pipeline di aggiornamento associata. È usato principalmente da reverse engineer di giochi e sviluppatori di tooling che necessitano di offset aggiornati dopo le patch del gioco.
