Este proyecto es una base de datos de offsets mantenida para las estructuras de memoria de Counter-Strike: Global Offensive. Ofrece salidas legibles por máquina como JSON, además de definiciones de offsets en C++ y C# para su integración directa en herramientas externas. El repositorio incluye patrones de firmas y datos de configuración pensados para actualizarse automáticamente mediante un pipeline de actualización asociado. Lo usan principalmente ingenieros de ingeniería inversa de juegos y desarrolladores de herramientas que necesitan offsets actuales tras los parches del juego.
