Ce projet est une base de données d'offsets maintenue pour les structures mémoire de Counter-Strike: Global Offensive. Il fournit des sorties lisibles par machine telles que JSON ainsi que des définitions d'offsets en C++ et C# pour une intégration directe dans des outils externes. Le dépôt inclut des motifs de signatures et des données de configuration conçus pour être actualisés automatiquement par un pipeline de mise à jour associé. Il est principalement utilisé par les ingénieurs en rétro-ingénierie de jeux et les développeurs d'outils qui ont besoin d'offsets à jour après les correctifs du jeu.
