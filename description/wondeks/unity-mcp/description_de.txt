Dieses Projekt konzentriert sich auf MCP für Unity.
Dieser Server ist darauf ausgelegt, Ihren Workflow zu vereinfachen und Ihr gesamtes Unity-Entwicklungserlebnis zu verbessern.
Er ist vor allem nützlich für Spieleentwickler, Reverse Engineers und Tooling-Entwickler, die im Bereich Spieleentwicklung / MCP-Server arbeiten.
