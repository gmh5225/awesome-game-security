Ce projet se concentre sur MCP pour Unity.
Ce serveur est conçu pour rationaliser votre flux de travail et améliorer votre expérience globale de développement Unity.
Il est principalement utile aux développeurs de jeux, aux ingénieurs en rétro-ingénierie et aux créateurs d'outils travaillant dans le domaine du développement de jeux / serveurs MCP.
