ClawSearch 是一款 x64dbg 插件，可为当前调试进程提供类似 Cheat Engine 的内存数值扫描功能。该项目以 C++ 编写，作为 Visual Studio 插件工程，借助 x64dbg 插件 SDK 访问与读取内存映射，并使用 IUP 工具包构建搜索对话框界面。它支持对整数与浮点类型（8 位至 64 位、float 与 double）进行首次扫描与再次扫描，并提供精确匹配、大于/小于、已改变/未改变、已增大/已减小等比较模式。可选功能包括十六进制输入、快速扫描对齐步进、浮点截断/四舍五入匹配、扫描时暂停，以及从结果地址跳转到调试器转储视图。该工具面向需要在 x64dbg 中调试 Windows 二进制文件时进行迭代数值查找的逆向工程师与游戏安全研究人员。
