ClawSearch 是一款 x64dbg 外掛，可為目前偵錯中的行程提供類似 Cheat Engine 的記憶體數值掃描功能。此專案以 C++ 撰寫，作為 Visual Studio 外掛專案，透過 x64dbg 外掛 SDK 存取與讀取記憶體對應，並使用 IUP 工具套件建置搜尋對話方塊介面。它支援對整數與浮點類型（8 位元至 64 位元、float 與 double）進行首次掃描與再次掃描，並提供精確符合、大於/小於、已變更/未變更、已增大/已減小等比較模式。選用功能包括十六進位輸入、快速掃描對齊步進、浮點截斷/四捨五入比對、掃描時暫停，以及從結果位址跳至偵錯器傾印檢視。此工具面向需要在 x64dbg 中偵錯 Windows 二進位檔時進行迭代數值搜尋的逆向工程師與遊戲安全研究人員。
