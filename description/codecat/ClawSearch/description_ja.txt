ClawSearch は、現在デバッグ中のプロセスに対して Cheat Engine 風のメモリ値スキャンを追加する x64dbg プラグインです。C++ で書かれた Visual Studio プラグインプロジェクトで、メモリマップへのアクセスと読み取りには x64dbg プラグイン SDK を、検索ダイアログ UI には IUP ツールキットを使用します。整数および浮動小数点型（8 ビットから 64 ビット、float、double）に対する初回スキャンと再スキャンに対応し、完全一致、より大きい/小さい、変更あり/なし、増加/減少などの比較モードを備えます。オプション機能として、16 進入力、高速スキャンのアライメントステップ、float の切り捨て/丸め一致、スキャン中の一時停止、結果アドレスからデバッガのダンプビューへのジャンプがあります。x64dbg で Windows バイナリをデバッグしながら反復的な値探索が必要なリバースエンジニアやゲームセキュリティ研究者向けです。
