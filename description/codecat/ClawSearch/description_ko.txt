ClawSearch는 현재 디버깅 중인 프로세스에 Cheat Engine 스타일의 메모리 값 스캔을 추가하는 x64dbg 플러그인입니다. C++로 작성된 Visual Studio 플러그인 프로젝트로, 메모리 맵 접근과 읽기에는 x64dbg 플러그인 SDK를, 검색 대화상자 UI에는 IUP 툴킷을 사용합니다. 정수 및 부동 소수점 타입(8비트부터 64비트, float, double)에 대한 첫 스캔과 다음 스캔을 지원하며, 정확한 일치, 보다 큼/작음, 변경됨/변경되지 않음, 증가/감소 등의 비교 모드를 제공합니다. 선택 기능으로는 16진수 입력, 빠른 스캔 정렬 스텝, float 잘라내기/반올림 일치, 스캔 중 일시 정지, 결과 주소에서 디버거 덤프 뷰로 이동이 있습니다. x64dbg에서 Windows 바이너리를 디버깅하면서 반복적인 값 검색이 필요한 리버스 엔지니어와 게임 보안 연구자를 대상으로 합니다.
