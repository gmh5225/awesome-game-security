ClawSearch — плагин для x64dbg, добавляющий сканирование значений в памяти в стиле Cheat Engine для текущего отлаживаемого процесса. Написан на C++ как проект плагина Visual Studio; для доступа к карте памяти и чтения использует SDK плагинов x64dbg, а для интерфейса диалога поиска — toolkit IUP. Поддерживает первый и последующие сканы по целочисленным и вещественным типам (от 8 до 64 бит, float и double) с режимами сравнения: точное совпадение, больше/меньше, изменено/не изменено, увеличено/уменьшено. Дополнительно доступны ввод в hex, шаг выравнивания при быстром скане, сопоставление float с усечением/округлением, пауза во время сканирования и переход от адресов результатов к виду дампа отладчика. Предназначен для реверс-инженеров и исследователей игровой безопасности, которым нужен итеративный поиск значений при отладке Windows-бинарников в x64dbg.
