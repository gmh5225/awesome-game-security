ClawSearch is an x64dbg plugin that adds Cheat Engine-style memory value scanning for the currently debugged process. Written in C++ as a Visual Studio plugin project, it uses the x64dbg plugin SDK for memory map access and reads, plus the IUP toolkit for its search dialog UI. It supports first and next scans across integer and floating-point types (8-bit through 64-bit, float, and double), with comparison modes such as exact match, bigger/smaller than, changed/unchanged, and increased/decreased values. Optional features include hex input, fast-scan alignment stepping, float truncate/round matching, pause-while-scanning, and jumping from result addresses into the debugger dump view. It is aimed at reverse engineers and game security researchers who need iterative value finding while debugging Windows binaries in x64dbg.
