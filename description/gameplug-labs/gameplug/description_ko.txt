GamePlug는 통합 플러그인 시스템을 통해 Windows 게임에 사용자 정의 ImGui 오버레이와 게임 로직을 주입할 수 있게 해 주는 C++ 다중 API 가로채기 프레임워크입니다. dinput8.dll, version.dll, winmm.dll 같은 프록시 DLL을 통해 Vulkan과 DirectX 9/10/11/12를 후킹하며, 공유 ImGui 컨텍스트가 있는 깔끔한 C++ 인터페이스로 플러그인을 로드합니다. CMake(C++23)로 빌드되며 MinHook, SafetyHook, Dear ImGui, spdlog, volk 등의 라이브러리를 사용하고, 32비트와 64비트 타깃을 모두 지원합니다. D3D9 텍스처 교체 및 덤프, 구성 가능한 해상도 오버라이드, 오버레이용 키보드 토글 기능도 제공합니다. 여러 렌더링 백엔드에서 재사용 가능한 주입·오버레이 계층이 필요한 게임 모더와 개발자를 대상으로 합니다.
