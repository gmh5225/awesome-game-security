GamePlug is a C++ multi-API interception framework that lets modders inject custom ImGui overlays and game logic into Windows games through a unified plugin system. It hooks Vulkan and DirectX 9/10/11/12 via proxy DLLs such as dinput8.dll, version.dll, and winmm.dll, and loads plugins through a clean C++ interface with shared ImGui context. The project is built with CMake (C++23), uses libraries including MinHook, SafetyHook, Dear ImGui, spdlog, and volk, and supports both 32-bit and 64-bit targets. Additional capabilities include D3D9 texture replacement and dumping, configurable resolution overrides, and keyboard toggles for the overlay. It is aimed at game modders and developers who need a reusable injection and overlay layer across rendering backends.
