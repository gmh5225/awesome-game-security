GamePlug は、統一プラグインシステムを通じてカスタム ImGui オーバーレイとゲームロジックを Windows ゲームに注入できる、C++ 製のマルチ API インターセプトフレームワークです。dinput8.dll、version.dll、winmm.dll などのプロキシ DLL 経由で Vulkan および DirectX 9/10/11/12 をフックし、共有 ImGui コンテキストを持つ明確な C++ インターフェースでプラグインを読み込みます。CMake（C++23）で構築され、MinHook、SafetyHook、Dear ImGui、spdlog、volk などのライブラリを使用し、32 ビットと 64 ビットの両方のターゲットをサポートします。さらに、D3D9 テクスチャの置換とダンプ、設定可能な解像度オーバーライド、オーバーレイ用のキーボードトグルにも対応します。複数のレンダリングバックエンドで再利用できる注入・オーバーレイ層を必要とするゲームモッダーや開発者向けです。
