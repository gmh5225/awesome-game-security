GamePlug 是一个 C++ 多 API 拦截框架，让模组作者通过统一的插件系统，向 Windows 游戏注入自定义 ImGui 叠加层与游戏逻辑。它借助 dinput8.dll、version.dll、winmm.dll 等代理 DLL 挂钩 Vulkan 与 DirectX 9/10/11/12，并通过带有共享 ImGui 上下文的清晰 C++ 接口加载插件。项目使用 CMake（C++23）构建，依赖 MinHook、SafetyHook、Dear ImGui、spdlog、volk 等库，同时支持 32 位与 64 位目标。额外能力包括 D3D9 纹理替换与转储、可配置的分辨率覆盖，以及叠加层的键盘开关。面向需要在多种渲染后端上复用注入与叠加层能力的游戏模组作者和开发者。
