GamePlug es un framework C++ de interceptación multi-API que permite a los modders inyectar overlays ImGui personalizados y lógica de juego en juegos de Windows mediante un sistema unificado de plugins. Engancha Vulkan y DirectX 9/10/11/12 a través de DLL proxy como dinput8.dll, version.dll y winmm.dll, y carga plugins mediante una interfaz C++ clara con un contexto ImGui compartido. El proyecto se construye con CMake (C++23), usa bibliotecas como MinHook, SafetyHook, Dear ImGui, spdlog y volk, y admite destinos de 32 y 64 bits. Las capacidades adicionales incluyen reemplazo y volcado de texturas D3D9, anulaciones de resolución configurables y atajos de teclado para el overlay. Está pensado para modders y desarrolladores de juegos que necesitan una capa reutilizable de inyección y overlay entre distintos backends de renderizado.
