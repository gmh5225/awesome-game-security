GamePlug 是一個 C++ 多 API 攔截框架，讓模組作者透過統一的外掛系統，向 Windows 遊戲注入自訂 ImGui 疊加層與遊戲邏輯。它藉由 dinput8.dll、version.dll、winmm.dll 等代理 DLL 掛鉤 Vulkan 與 DirectX 9/10/11/12，並透過具備共用 ImGui 上下文的清晰 C++ 介面載入外掛。專案以 CMake（C++23）建置，使用 MinHook、SafetyHook、Dear ImGui、spdlog、volk 等函式庫，並同時支援 32 位元與 64 位元目標。額外功能包括 D3D9 紋理替換與傾印、可設定的解析度覆寫，以及疊加層的鍵盤開關。面向需要在多種繪製後端上重複使用注入與疊加層能力的遊戲模組作者與開發者。
