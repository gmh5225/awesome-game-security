Этот проект собирает смещения памяти Fortnite, используемые инструментами читов и реверс-инжиниринга. Оригинальный репозиторий на GitHub больше недоступен; запись в README документирует его как дамп смещений в категории исследований игровых читов Fortnite. Он в основном полезен исследователям безопасности игр и реверс-инженерам, изучающим атакующие техники в области cheat / game:fortnite.
