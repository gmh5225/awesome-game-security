Questo progetto raccoglie gli offset di memoria di Fortnite usati da tool di cheat e di reverse engineering. Il repository GitHub originale non è più disponibile; la voce README lo documenta come un dump di offset nella categoria di ricerca sui cheat di Fortnite. È utile soprattutto a ricercatori di sicurezza nei giochi e a reverse engineer che studiano tecniche offensive nell’ambito cheat / game:fortnite.
