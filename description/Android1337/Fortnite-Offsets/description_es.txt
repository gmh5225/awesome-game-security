Este proyecto recopila offsets de memoria de Fortnite usados por herramientas de trampas y de ingeniería inversa. El repositorio original de GitHub ya no está disponible; la entrada del README lo documenta como un volcado de offsets en la categoría de investigación de cheats de Fortnite. Resulta útil sobre todo para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito cheat / game:fortnite.
