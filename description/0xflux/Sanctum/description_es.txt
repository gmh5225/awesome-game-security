Este proyecto es una plataforma experimental de detección y respuesta en endpoints de Windows, construida principalmente en Rust.
Combina un controlador de kernel, un motor en modo usuario y una interfaz basada en Tauri para monitorizar la actividad de procesos, hilos, sistema de archivos y llamadas al sistema.
El código incluye componentes como consumidores ETW, un minifiltro de sistema de archivos y lógica de hooking y contención en el kernel para telemetría y respuesta.
Su principal caso de uso es la investigación de seguridad de bajo nivel y el prototipado de herramientas defensivas en sistemas Windows modernos.
