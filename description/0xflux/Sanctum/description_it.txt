Questo progetto è una piattaforma sperimentale di endpoint detection and response per Windows, realizzata principalmente in Rust.
Combina un driver kernel, un motore in user-mode e un’interfaccia basata su Tauri per monitorare l’attività di processi, thread, file system e system call.
Il codice include componenti come consumer ETW, un minifilter del file system e logica di hooking e containment lato kernel per telemetria e risposta.
Il caso d’uso principale è la ricerca di sicurezza a basso livello e la prototipazione di strumenti difensivi su sistemi Windows moderni.
