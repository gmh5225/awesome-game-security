Этот проект — лёгкая библиотека inline-хукинга для Android на ARM64. Реализация на C++ патчит целевые инструкции, переносит затронутые ветвления AArch64 и строит трамплины, чтобы исходный код мог безопасно продолжать выполнение после установки хуков. Также обрабатываются низкоуровневые детали среды выполнения, такие как изменение исполняемой памяти и сброс кэша инструкций. Библиотека полезна для мобильного реверс-инжиниринга, инструментирования и экспериментов в области игровой безопасности на платформах ARMv8.
