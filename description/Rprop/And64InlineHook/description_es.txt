Este proyecto es una biblioteca ligera de inline hooking para Android en ARM64. La implementación en C++ parchea las instrucciones objetivo, reubica las ramas AArch64 afectadas y construye trampolines para que el código original pueda continuar de forma segura tras instalar los hooks. También gestiona detalles de tiempo de ejecución de bajo nivel, como cambios en la memoria ejecutable y el vaciado de la caché de instrucciones. La biblioteca es útil para ingeniería inversa móvil, instrumentación y experimentación en seguridad de juegos en plataformas ARMv8.
