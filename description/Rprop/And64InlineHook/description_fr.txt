Ce projet est une bibliothèque légère d'inline hooking pour Android sur ARM64. L'implémentation C++ patche les instructions cibles, relocalise les branches AArch64 concernées et construit des trampolines afin que le code d'origine puisse continuer en toute sécurité après l'installation des hooks. Elle gère également des détails d'exécution bas niveau tels que la modification de la mémoire exécutable et le vidage du cache d'instructions. La bibliothèque est utile pour la rétro-ingénierie mobile, l'instrumentation et l'expérimentation en sécurité des jeux sur les plateformes ARMv8.
