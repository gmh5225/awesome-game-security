Este proyecto es una biblioteca de alto rendimiento de desofuscación dex para Android, diseñada para el descubrimiento de código asistido por nativo. Combina componentes C++ NDK con JNI e integración orientada a Kotlin para buscar clases y métodos por cadenas, relaciones y patrones de opcodes. La biblioteca admite distribución Gradle, empaquetado prefab y enlace CMake para una integración práctica en proyectos Android. Se usa principalmente en flujos de ingeniería inversa de Android, como el análisis de aplicaciones, el descubrimiento de puntos de hook y la navegación de código ofuscado.
