Ce projet est une bibliothèque Android haute performance de désobfuscation dex conçue pour la découverte de code assistée par le natif. Elle combine des composants C++ NDK avec JNI et une intégration orientée Kotlin pour rechercher des classes et des méthodes par chaînes, relations et motifs d'opcodes. La bibliothèque prend en charge la distribution Gradle, le packaging prefab et la liaison CMake pour une intégration concrète dans les projets Android. Elle est principalement utilisée dans les workflows de rétro-ingénierie Android, tels que l'analyse d'applications, la découverte de points d'accrochage et la navigation dans le code obfusqué.
