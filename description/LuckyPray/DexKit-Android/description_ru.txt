Этот проект — высокопроизводительная Android-библиотека деобфускации dex, предназначенная для поиска кода с поддержкой native. Она сочетает компоненты C++ NDK с JNI и интеграцией для Kotlin, позволяя искать классы и методы по строкам, связям и шаблонам opcode. Библиотека поддерживает распространение через Gradle, упаковку prefab и связывание через CMake для практичного встраивания в Android-проекты. В основном применяется в рабочих процессах реверс-инженерии Android: анализ приложений, поиск точек hook и навигация по обфусцированному коду.
