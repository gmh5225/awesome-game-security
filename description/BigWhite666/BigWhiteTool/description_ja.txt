BigWhiteTool は、Unreal スタイルのメタデータ探索と SDK ダンプに焦点を当てた Android ネイティブ向けゲームメモリユーティリティです。C/C++ コードベースには、プロセスメモリの読み書き、GName および GObject に基づくオフセット走査、ダンプ生成ロジックなどのモジュールが含まれます。また、Vulkan または OpenGL の描画パスを用いる ImGui インターフェースと、NDK ベースの CMake ビルドサポートも同梱しています。主にモバイルゲームのリバースエンジニアリングにおいて、オフセット特定、実行時検査、SDK 抽出のワークフローで利用されます。
