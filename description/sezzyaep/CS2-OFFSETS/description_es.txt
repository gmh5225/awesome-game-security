Este proyecto es un repositorio actualizado periódicamente de offsets de memoria y volcados de netvars de Counter-Strike 2. Proporciona offsets de estructuras, direcciones de campos de entidades y punteros de interfaces extraídos de los binarios cliente y servidor de CS2 tras cada actualización del juego. Los datos se ofrecen como encabezados C++ y JSON para el desarrollo de cheats y mods. Está dirigido a desarrolladores de cheats de CS2 e investigadores de seguridad en juegos que necesitan información actualizada sobre el diseño de memoria del motor Source 2.
