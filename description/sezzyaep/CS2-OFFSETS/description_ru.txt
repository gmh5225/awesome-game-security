Этот проект — регулярно обновляемый репозиторий смещений памяти и дампов netvar для Counter-Strike 2. Он предоставляет смещения структур, адреса полей сущностей и указатели интерфейсов, извлечённые из клиентских и серверных бинарников CS2 после каждого обновления игры. Данные оформлены в виде заголовков C++ и JSON для разработки читов и модов. Репозиторий предназначен для разработчиков читов CS2 и исследователей безопасности игр, которым нужна актуальная информация о раскладке памяти движка Source 2.
