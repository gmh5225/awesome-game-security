Questo progetto è un repository aggiornato regolarmente di offset di memoria e dump di netvar per Counter-Strike 2. Fornisce offset di strutture, indirizzi dei campi delle entità e puntatori alle interfacce estratti dai binari client e server di CS2 dopo ogni aggiornamento del gioco. I dati sono formattati come header C++ e JSON per lo sviluppo di cheat e mod. È rivolto agli sviluppatori di cheat per CS2 e ai ricercatori di sicurezza nei videogiochi che necessitano di informazioni aggiornate sul layout di memoria del motore Source 2.
