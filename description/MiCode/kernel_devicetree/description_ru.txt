kernel_devicetree — это индекс веток исходников device-tree ядра устройств Xiaomi для множества моделей и поколений Android.
Его README организован как краткая справочная таблица, сопоставляющая ветки устройствам, меткам платформы и базовым линиям релизов.
Это упрощает поиск данных конфигурации платы, используемых при bring-up ядра, настройке оборудования и работах по совместимости.
Репозиторий в первую очередь полезен инженерам платформы Android и исследователям, изучающим низкоуровневую конфигурацию устройств и поведение безопасности.
