kernel_devicetree est un index de branches pour les sources device-tree du noyau des appareils Xiaomi, couvrant plusieurs modèles et générations Android.
Son README est organisé comme une table de correspondance concise qui associe les branches aux appareils, aux balises de plateforme et aux baselines de release.
Cela facilite la recherche des données de configuration de carte utilisées pour le bring-up du noyau, le réglage matériel et le travail de compatibilité.
Il est surtout utile aux ingénieurs de la plateforme Android et aux chercheurs qui étudient la configuration bas niveau des appareils et le comportement de sécurité.
