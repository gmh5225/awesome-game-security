kernel_devicetree es un índice de ramas para las fuentes del device-tree del kernel de dispositivos Xiaomi, en varios modelos y generaciones de Android.
Su README está organizado como una tabla de consulta concisa que asocia ramas con dispositivos, etiquetas de plataforma y líneas base de release.
Esto facilita encontrar los datos de configuración de placa usados en el bring-up del kernel, el ajuste de hardware y el trabajo de compatibilidad.
Resulta especialmente útil para ingenieros de la plataforma Android e investigadores que estudian la configuración de bajo nivel de dispositivos y el comportamiento de seguridad.
