Xiaomi_Kernel_OpenSource es un repositorio de tipo índice que cataloga muchas ramas de código fuente del kernel de Xiaomi y Redmi por dispositivo y versión de Android.
El contenido son principalmente tablas Markdown que apuntan a etiquetas de ramas y referencias base para distintas plataformas de hardware y versiones.
Sirve como un mapa estructurado para localizar las fuentes del kernel del fabricante necesarias para el desarrollo de controladores, auditorías y depuración específica de la plataforma.
El público principal son desarrolladores del kernel de Android e investigadores de seguridad que trabajan en los internos de dispositivos Xiaomi.
