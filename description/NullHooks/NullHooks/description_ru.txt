NullHooks — учебный проект чита для CS:GO на C++. Он включает модульную кодовую базу с игровыми возможностями, предустановками конфигурации и вспомогательными утилитами, типичными для разработки внутренних читов. В репозитории есть справочные материалы и заметки по внесению вклада; он помечен как неактивный, но по-прежнему полезен для изучения. Основное назначение — образовательное знакомство с архитектурой game hacking и приёмами реализации.
