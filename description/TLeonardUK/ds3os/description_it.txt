Questo progetto è un emulatore di server open source per i servizi online di Dark Souls 2 e Dark Souls 3. Implementa il protocollo multiplayer del gioco tramite definizioni di messaggi Protobuf, supporta il deployment con Docker e gestisce il matchmaking dei giocatori, le evocazioni cooperative, le invasioni e i sistemi di messaggi. È soprattutto utile a ricercatori di server di gioco e reverse engineer che studiano i protocolli di rete della serie Souls e l’architettura dei server multiplayer.
