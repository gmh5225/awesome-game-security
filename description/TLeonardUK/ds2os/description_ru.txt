Этот проект — устаревший заполнитель для ранних работ по онлайн-сервису Dark Souls 2. Архив содержит лишь краткий README, указывающий, что активная разработка перенесена в более новую кодовую базу. В этом снимке репозитория не осталось существенной исходной реализации. Основное назначение — историческая справка для пользователей, ищущих перенесённое продолжение проекта.
