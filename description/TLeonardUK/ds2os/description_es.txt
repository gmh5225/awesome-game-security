Este proyecto es un marcador de posición obsoleto de trabajos anteriores sobre el servicio en línea de Dark Souls 2. El archivo solo incluye un README breve que indica que el desarrollo activo se ha trasladado a una base de código más reciente. En esta instantánea del repositorio no queda ninguna implementación de código fuente sustancial. Su uso principal es como referencia histórica para quienes buscan la continuación migrada del proyecto.
