Este proyecto es un cheat ESP externo para Counter-Strike 2. Lee la memoria del juego de forma externa y renderiza cajas, nombres y superposiciones de salud mediante una ventana basada en GDI, con soporte para el mantenimiento de offsets y scripts de actualización. El código es principalmente C++ en Visual Studio e incluye componentes para el manejo de memoria, el secuestro de handles y la configuración basada en JSON. Está pensado para la experimentación en game hacking y para aprender la arquitectura de cheats externos.
