Этот проект — внешний ESP-чит для Counter-Strike 2. Он читает память игры извне и отрисовывает боксы, имена и оверлеи здоровья через окно на базе GDI, с поддержкой сопровождения оффсетов и скриптов обновления. Кодовая база в основном на C++ в Visual Studio и включает компоненты для работы с памятью, перехвата хендлов и конфигурации на JSON. Предназначен для экспериментов в game hacking и изучения архитектуры внешних читов.
