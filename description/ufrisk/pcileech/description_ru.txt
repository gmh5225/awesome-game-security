Этот проект посвящён PCILeech. PCILeech использует аппаратные устройства PCIe для чтения и записи памяти целевой системы.
Это достигается с помощью DMA по PCIe.
Он в основном полезен исследователям безопасности игр и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / DMA.
