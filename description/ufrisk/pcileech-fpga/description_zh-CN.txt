本项目提供面向 PCILeech DMA 设备的 FPGA 硬件设计与 Vivado 构建流程，可通过 PCIe 访问目标系统内存。实现以 SystemVerilog 与 Verilog 为主，并配合 Xilinx IP 核、约束文件以及用于工程生成、比特流构建与烧录的 Tcl 脚本。仓库覆盖 CaptainDMA、PCIeSquirrel、ScreamerM2、EnigmaX1、ZDMA、NeTV2 以及基于 FT601 或 FT2232H 的多种板卡目标，包含 PCIe TLP 处理、BAR 控制与配置空间影子逻辑。它主要由安全研究人员与逆向工程师用于研究基于 DMA 的内存采集及相关反作弊威胁模型。
