本プロジェクトは、PCIe 経由でターゲットシステムのメモリにアクセスする PCILeech DMA デバイス向けの FPGA ハードウェア設計と Vivado ビルドフローを提供します。実装は主に SystemVerilog と Verilog で、Xilinx IP コア、制約、およびプロジェクト生成・ビットストリーム構築・書き込み用の Tcl スクリプトを含みます。リポジトリは CaptainDMA、PCIeSquirrel、ScreamerM2、EnigmaX1、ZDMA、NeTV2、および FT601 や FT2232H ベースのプラットフォームなど多くのボードを対象とし、PCIe TLP 処理、BAR 制御、コンフィギュレーション空間のシャドウロジックをカバーします。主にセキュリティ研究者やリバースエンジニアが、DMA ベースのメモリ取得および関連するアンチチート脅威モデルの研究に利用します。
