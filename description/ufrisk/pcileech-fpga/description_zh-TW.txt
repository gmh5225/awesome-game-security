本專案提供面向 PCILeech DMA 裝置的 FPGA 硬體設計與 Vivado 建置流程，可透過 PCIe 存取目標系統記憶體。實作以 SystemVerilog 與 Verilog 為主，並搭配 Xilinx IP 核心、約束檔以及用於專案產生、位元流建置與燒錄的 Tcl 腳本。倉庫涵蓋 CaptainDMA、PCIeSquirrel、ScreamerM2、EnigmaX1、ZDMA、NeTV2 以及基於 FT601 或 FT2232H 的多種開發板目標，包含 PCIe TLP 處理、BAR 控制與組態空間影子邏輯。它主要由安全研究人員與逆向工程師用於研究基於 DMA 的記憶體擷取及相關反作弊威脅模型。
