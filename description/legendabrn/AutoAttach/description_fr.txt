Ce projet sert à changer l'état du plugin : vous devez écrire la commande AutoAttachStatus (0/1). Pour spécifier un processus, utilisez la commande AutoAttachProcess et saisissez le nom du processus. Si vous avez besoin d'un délai avant l'attachement, utilisez AutoAttachSleep en indiquant les millisecondes dans les arguments. Exemple délai AutoAttachSleep 1000 ; exemple sélection de processus AutoAttachProcess dota2.exe ; exemple activation AutoAttach AutoAttachStatus 1 ; exemple désactivation AutoAttach AutoAttachStatus 0.
