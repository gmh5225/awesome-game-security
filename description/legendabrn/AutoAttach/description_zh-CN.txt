本项目用于控制插件状态：需使用命令 AutoAttachStatus (0/1)。要指定进程，须使用 AutoAttachProcess 命令并输入进程名。若在附加前需要延迟，可通过 AutoAttachSleep 并在参数中指定毫秒数。示例：延迟 AutoAttachSleep 1000；选择进程 AutoAttachProcess dota2.exe；启用 AutoAttach AutoAttachStatus 1；禁用 AutoAttach AutoAttachStatus 0。相关命令在此输入 :)。
主要使用 C/C++ 和 C++ 编写，围绕插件开发、模组制作与 SDK 生成。
主要面向研究作弊 / x64dbg 插件领域进攻技术的游戏安全研究者与逆向工程师。
