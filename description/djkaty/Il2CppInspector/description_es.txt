Una herramienta y biblioteca C# completa para la ingeniería inversa de aplicaciones Unity IL2CPP, que extrae definiciones de tipos, firmas de métodos y metadatos de global-metadata.dat y de binarios IL2CPP.
Genera scripts para IDA Pro, Ghidra y Binary Ninja con anotaciones de tipos completas, andamiajes de encabezados C++ y frameworks de inyección de DLL, con soporte para todas las versiones y plataformas de IL2CPP.
Resulta especialmente útil para investigadores de seguridad de juegos y modders que analizan juegos Unity compilados con IL2CPP para recuperar las estructuras de clases C# originales y los offsets de métodos.
