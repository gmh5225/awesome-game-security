Комплексный инструмент и библиотека на C# для реверс-инжиниринга приложений Unity IL2CPP, извлекающие определения типов, сигнатуры методов и метаданные из global-metadata.dat и бинарников IL2CPP.
Генерирует скрипты для IDA Pro, Ghidra и Binary Ninja с полными аннотациями типов, каркасы заголовочных файлов C++ и фреймворки DLL-инъекции, поддерживая все версии и платформы IL2CPP.
В основном полезен исследователям игровой безопасности и моддерам, анализирующим игры Unity, скомпилированные с IL2CPP, для восстановления исходных структур классов C# и смещений методов.
