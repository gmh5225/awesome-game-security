Unity の IL2CPP アプリケーションをリバースエンジニアリングするための包括的な C# ツールおよびライブラリで、global-metadata.dat と IL2CPP バイナリから型定義、メソッドシグネチャ、メタデータを抽出します。
IDA Pro、Ghidra、Binary Ninja 向けに完全な型アノテーション付きスクリプト、C++ ヘッダのスキャフォールド、DLL インジェクションフレームワークを生成し、すべての IL2CPP バージョンとプラットフォームに対応します。
主にゲームセキュリティ研究者やモッダーが、IL2CPP でコンパイルされた Unity ゲームを解析し、元の C# クラス構造とメソッドオフセットを復元するために役立ちます。
