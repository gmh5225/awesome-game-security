Unity IL2CPP 애플리케이션을 리버스 엔지니어링하기 위한 종합적인 C# 도구이자 라이브러리로, global-metadata.dat와 IL2CPP 바이너리에서 형식 정의, 메서드 시그니처, 메타데이터를 추출합니다.
IDA Pro, Ghidra, Binary Ninja용으로 완전한 형식 주석이 포함된 스크립트, C++ 헤더 스캐폴드, DLL 인젝션 프레임워크를 생성하며, 모든 IL2CPP 버전과 플랫폼을 지원합니다.
주로 게임 보안 연구자와 모더가 IL2CPP로 컴파일된 Unity 게임을 분석하여 원래 C# 클래스 구조와 메서드 오프셋을 복원하는 데 유용합니다.
