Этот проект — курируемая подборка ресурсов, статей и материалов по разработке игр, охватывающая архитектуру игровых движков, техники рендеринга, физическую симуляцию, сетевое взаимодействие, ИИ, оптимизацию и производственные процессы. Репозиторий систематизирует ссылки на технические статьи, доклады GDC, блог-посты и open-source проекты в разных областях геймдева. Он предназначен для разработчиков игр и technical artists, которым нужны комплексные учебные материалы по игровым технологиям.
