Фреймворк отрисовки оверлеев для Desktop Window Manager, который перехватывает vtable DirectX 11 в DWM для рисования оверлеев на базе ImGui поверх всех окон. Использует рефлексивную DLL-инъекцию, MinHook/PolyHook2 для перехвата vtable, FW1FontWrapper для рендеринга текста через DirectWrite и разрешение PDB-символов (через DIA SDK) для поиска внутренних функций композитора DWM.
