Framework di rendering overlay per Desktop Window Manager che aggancia la vtable DirectX 11 di DWM per disegnare overlay basati su ImGui sopra tutte le finestre. Usa iniezione DLL riflessiva, MinHook/PolyHook2 per l'intercettazione della vtable, FW1FontWrapper per il rendering del testo DirectWrite e risoluzione dei simboli PDB (tramite DIA SDK) per individuare le funzioni interne del compositor DWM.
