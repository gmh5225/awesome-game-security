Framework de renderizado de overlays para Desktop Window Manager que engancha la vtable de DirectX 11 de DWM para dibujar overlays basados en ImGui sobre todas las ventanas. Usa inyección DLL reflexiva, MinHook/PolyHook2 para la interceptación de vtable, FW1FontWrapper para el renderizado de texto con DirectWrite y resolución de símbolos PDB (mediante DIA SDK) para localizar funciones internas del compositor DWM.
