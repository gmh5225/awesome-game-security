Framework de rendu d'overlay pour le Desktop Window Manager qui accroche la vtable DirectX 11 de DWM afin d'afficher des overlays basés sur ImGui au-dessus de toutes les fenêtres. Utilise l'injection DLL réflexive, MinHook/PolyHook2 pour l'interception de vtable, FW1FontWrapper pour le rendu de texte DirectWrite, et la résolution de symboles PDB (via le DIA SDK) pour localiser les fonctions internes du compositeur DWM.
