Dieses Projekt ist ein Windows-HWID-Spoofing-Toolkit, das eine ImGui-basierte Benutzeroberfläche mit Komponenten zum Laden von Kernel-Treibern kombiniert. Die Codebasis ist überwiegend in C++ geschrieben und enthält Module zu Treiber-Mapping, Dienstverwaltung und Änderungen von Hardwarekennungen auf niedriger Ebene. Es richtet sich an praktische Testszenarien, in denen Nutzer Maschinenidentitätsänderungen über einen GUI-Workflow auslösen möchten. Die Hauptzielgruppe sind Forscher im Bereich Game-Bypass und Anti-Cheat, die HWID-basierte Durchsetzung untersuchen.
