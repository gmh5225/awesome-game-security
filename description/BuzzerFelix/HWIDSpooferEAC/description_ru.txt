Этот проект — набор инструментов для подмены HWID в Windows, сочетающий пользовательский интерфейс на ImGui с компонентами загрузки драйверов ядра. Кодовая база в основном на C++ и включает модули, связанные с маппингом драйверов, обработкой служб и низкоуровневым изменением аппаратных идентификаторов. Он рассчитан на практические тестовые сценарии, где пользователи хотят инициировать смену машинной идентичности через GUI-процесс. Основная аудитория — исследователи обхода игр и античитов, изучающие принуждение на основе HWID.
