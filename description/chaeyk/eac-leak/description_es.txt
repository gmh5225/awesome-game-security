Este proyecto es una implementación filtrada del servidor EasyAntiCheat que se integra con Epic Online Services (EOS) para la gestión de sesiones anticheat.
La base de código en C++ incluye un componente de servidor con integración del SDK de EOS para el manejo de sesiones anticheat, y requiere reemplazar el ID de aplicación de EOS para el despliegue.
Proporciona información sobre el protocolo de comunicación del lado del servidor entre los servidores de juego y la infraestructura de validación en la nube de EAC.
Resulta especialmente útil para investigadores de anticheat que estudian la arquitectura del lado del servidor de EasyAntiCheat y la gestión de sesiones anticheat basada en EOS.
