本プロジェクトは、Epic Online Services (EOS) と連携してアンチチートセッションを管理する、漏洩した EasyAntiCheat サーバー実装です。
C++ コードベースには、アンチチートセッション処理のための EOS SDK 統合を備えたサーバーコンポーネントが含まれており、デプロイ時には EOS アプリケーション ID の置き換えが必要です。
ゲームサーバーと EAC のクラウド検証インフラ間のサーバー側通信プロトコルに関する知見を提供します。
EasyAntiCheat のサーバー側アーキテクチャおよび EOS ベースのアンチチートセッション管理を研究するアンチチート研究者にとって主に有用です。
