Questo progetto è un'implementazione server di EasyAntiCheat trapelata che si interfaccia con Epic Online Services (EOS) per la gestione delle sessioni anti-cheat.
Il codice C++ include un componente server con integrazione dell'SDK EOS per la gestione delle sessioni anti-cheat; per il deployment è necessario sostituire l'ID applicazione EOS.
Fornisce indicazioni sul protocollo di comunicazione lato server tra i game server e l'infrastruttura di validazione cloud di EAC.
È utile soprattutto ai ricercatori anti-cheat che studiano l'architettura lato server di EasyAntiCheat e la gestione delle sessioni anti-cheat basata su EOS.
