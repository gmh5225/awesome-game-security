Ce projet est une implémentation serveur EasyAntiCheat divulguée qui s'interface avec Epic Online Services (EOS) pour la gestion des sessions anti-triche.
La base de code C++ comprend un composant serveur avec intégration du SDK EOS pour la gestion des sessions anti-triche, nécessitant le remplacement de l'identifiant d'application EOS pour le déploiement.
Il offre un aperçu du protocole de communication côté serveur entre les serveurs de jeu et l'infrastructure de validation cloud d'EAC.
Il est principalement utile aux chercheurs en anti-triche qui étudient l'architecture serveur d'EasyAntiCheat et la gestion des sessions anti-triche basée sur EOS.
