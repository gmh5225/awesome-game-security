Этот проект — утечка серверной реализации EasyAntiCheat, взаимодействующей с Epic Online Services (EOS) для управления античит-сессиями.
Кодовая база на C++ включает серверный компонент с интеграцией EOS SDK для обработки античит-сессий; для развёртывания требуется заменить идентификатор приложения EOS.
Он даёт представление о серверном протоколе связи между игровыми серверами и облачной инфраструктурой валидации EAC.
В основном полезен исследователям античита, изучающим серверную архитектуру EasyAntiCheat и управление античит-сессиями на базе EOS.
