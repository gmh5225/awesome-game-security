Este proyecto es una colección de Beacon Object Files (BOF) que usan un controlador de kernel vulnerable (PoisonX.sys) para terminar procesos, cargar y descargar controladores, y eliminar archivos desde el modo kernel. Los BOF basados en C se compilan de forma cruzada con MinGW para su uso en Cobalt Strike o frameworks C2 similares, junto con un script auxiliar en Python. Resulta especialmente útil para operadores de red team e investigadores de seguridad que estudian la terminación de procesos basada en BYOVD y las operaciones de archivos a nivel de kernel mediante cargas útiles BOF.
