Questo progetto è una raccolta di Beacon Object File (BOF) che utilizzano un driver del kernel vulnerabile (PoisonX.sys) per terminare processi, caricare e scaricare driver ed eliminare file dalla modalità kernel. I BOF basati su C sono cross-compilati con MinGW per l'uso in Cobalt Strike o framework C2 simili, con uno script helper in Python. È principalmente utile per operatori red team e ricercatori di sicurezza che studiano la terminazione di processi basata su BYOVD e le operazioni sui file a livello kernel tramite payload BOF.
