Этот проект — набор Beacon Object Files (BOF), использующих уязвимый драйвер ядра (PoisonX.sys) для завершения процессов, загрузки и выгрузки драйверов, а также удаления файлов из режима ядра. BOF на языке C кросс-компилируются с помощью MinGW для применения в Cobalt Strike или аналогичных C2-фреймворках и сопровождаются вспомогательным скриптом на Python. В первую очередь полезен операторам red team и исследователям безопасности, изучающим завершение процессов на основе BYOVD и файловые операции на уровне ядра через полезные нагрузки BOF.
