DFoundryFX es un plugin de Unreal Engine 5 que añade herramientas de rendimiento en tiempo real dentro del viewport para proyectos de juego. Proporciona gráficos de métricas personalizables, monitorización de la compilación de shaders y controles rápidos de comandos STAT, sin necesidad de colocar actors adicionales en un nivel. El plugin está implementado con módulos de Unreal C++ y está pensado para permanecer disponible directamente durante el gameplay y las sesiones de profiling. Se usa principalmente por desarrolladores que necesitan diagnósticos runtime ligeros y soporte para el ajuste de rendimiento.
