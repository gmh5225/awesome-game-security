Этот проект — неофициальное зеркало декомпилированных исходников сборок редактора и движка Unity. Оно создано с помощью ILSpy и предоставляет C#-код из релиза Unity 5.x для изучения. Репозиторий содержит большую кодовую базу на C# и файлы решений, упрощающие исследование внутренних API и поведения редактора. В основном полезен для реверс-инжиниринга движка, исследования инструментов и анализа совместимости.
