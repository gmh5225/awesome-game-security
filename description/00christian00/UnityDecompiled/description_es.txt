Este proyecto es un espejo no oficial de código fuente descompilado de los ensamblados del editor y del motor de Unity. Se generó con ILSpy y expone código C# de una versión de Unity 5.x para su inspección. El repositorio contiene una amplia base de código C# y archivos de solución que facilitan el estudio de las API internas y del comportamiento del editor. Resulta útil principalmente para ingeniería inversa del motor, investigación de herramientas y análisis de compatibilidad.
