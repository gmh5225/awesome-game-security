Questo progetto è uno specchio non ufficiale di codice sorgente decompilato degli assembly dell'editor e del motore Unity. È stato generato con ILSpy ed espone codice C# di una release Unity 5.x per l'ispezione. Il repository contiene un'ampia codebase C# e file di soluzione che facilitano lo studio delle API interne e del comportamento dell'editor. È utile soprattutto per reverse engineering del motore, ricerca su strumenti e analisi di compatibilità.
