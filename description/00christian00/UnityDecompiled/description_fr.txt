Ce projet est un miroir non officiel de sources décompilées des assemblages de l'éditeur et du moteur Unity. Il a été généré avec ILSpy et expose le code C# d'une version Unity 5.x pour inspection. Le dépôt contient une large base de code C# et des fichiers de solution qui facilitent l'étude des API internes et du comportement de l'éditeur. Il est surtout utile pour la rétro-ingénierie du moteur, la recherche d'outils et l'analyse de compatibilité.
