Ce projet est un cheat externe pour CS:GO écrit en C++ qui fonctionne en lisant la mémoire du jeu depuis un processus séparé. Il fournit de l'ESP, un aimbot, un radar hack et des fonctionnalités visuelles de base via la lecture mémoire externe avec ReadProcessMemory ou un pilote noyau. Le cheat lit les positions des entités, la santé des joueurs et les données d'armes depuis la mémoire du jeu sans injecter de code dans le processus CS:GO. Il s'adresse aux chercheurs en sécurité des jeux qui étudient les schémas de cheats externes et les méthodes de détection externe des anti-cheats.
