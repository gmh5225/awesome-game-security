PE-пакер бинарников на Rust, который использует crate exe-rs для разбора и перезаписи файлов Portable Executable, встраивая сжатый payload вместе с распаковочным stub на основе trampoline. Stub использует точки входа TLS callback (предоставлены как исходники NASM для x86 и x64), чтобы распаковать и выполнить исходный бинарник во время выполнения. Архитектура намеренно расширяема для добавления проходов обфускации или anti-reversing помимо обычного сжатия.
