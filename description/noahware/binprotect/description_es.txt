Este proyecto es un ofuscador x64 PE bin-to-bin en C++ que transforma binarios compilados sin añadir nuevas secciones ni requerir acceso al código fuente. Incluye un ensamblador y un desensamblador personalizados basados en la biblioteca binwrite, además de análisis PE con soporte para exception directories, estructuras RTTI, escaneo de frame pointers, relocations y recuperación de jump tables. El paso de ofuscación opera a nivel de basic blocks sobre funciones desensambladas. Está orientado a investigadores de protección de software que estudian técnicas de ofuscación por reescritura binaria y las restricciones estructurales de PE.
