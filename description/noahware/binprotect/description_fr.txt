Ce projet est un obfuscateur x64 PE bin-to-bin en C++ qui transforme des binaires compilés sans ajouter de nouvelles sections ni nécessiter le code source. Il comprend un assembleur et un désassembleur personnalisés basés sur la bibliothèque binwrite, ainsi qu’une analyse PE prenant en charge les exception directories, les structures RTTI, le balayage des frame pointers, les relocations et la récupération des jump tables. Le passage d’obfuscation opère au niveau des basic blocks sur les fonctions désassemblées. Il s’adresse aux chercheurs en protection logicielle qui étudient les techniques d’obfuscation par réécriture binaire et les contraintes structurelles du format PE.
