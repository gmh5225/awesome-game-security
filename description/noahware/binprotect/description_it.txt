Questo progetto è un offuscatore x64 PE bin-to-bin in C++ che trasforma binari compilati senza aggiungere nuove sezioni né richiedere il codice sorgente. Include un assembler e un disassembler personalizzati basati sulla libreria binwrite, oltre al parsing PE con supporto per exception directories, strutture RTTI, scansione dei frame pointer, relocation e recupero delle jump table. Il passaggio di offuscamento opera a livello di basic block sulle funzioni disassemblate. È rivolto a ricercatori di software protection che studiano tecniche di offuscamento mediante binary rewriting e i vincoli strutturali del formato PE.
