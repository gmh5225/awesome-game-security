Этот проект — x64 PE bin-to-bin обфускатор на C++, который преобразует скомпилированные бинарники без добавления новых секций и без доступа к исходному коду. Он включает собственный ассемблер и дизассемблер на базе библиотеки binwrite, а также разбор PE с поддержкой exception directories, структур RTTI, сканирования frame pointer, релокаций и восстановления jump table. Проход обфускации работает на уровне basic block по дизассемблированным функциям. Предназначен для исследователей программной защиты, изучающих техники обфускации через binary rewriting и структурные ограничения PE.
