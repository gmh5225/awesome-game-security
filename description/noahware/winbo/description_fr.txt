Ce projet est un outil Windows C++ destiné à détecter le détournement de fenêtres de type overlay via l’analyse ETW et le balayage de la table GDI. Il analyse les événements ETW de dxgkrnl pour identifier quels processus appellent Present sur quels handles de fenêtre, en comparant le PID de l’appelant au PID du propriétaire de la fenêtre afin de signaler un rendu inter-processus non autorisé via DirectX ou OpenGL. Pour le rendu basé sur GDI, il parcourt la table de handles GDI partagée afin de trouver les handles DC dont le propriétaire diffère de la fenêtre cible. Les processus partageant un parent commun sont mis en liste blanche pour autoriser le partage légitime de fenêtres multi-processus. Il s’adresse aux chercheurs en anti-triche et en sécurité défensive qui étudient les techniques de détection d’overlays.
