Этот проект — инструмент на Windows C++ для обнаружения захвата окон в стиле overlay с помощью ETW и сканирования таблицы GDI. Он разбирает события ETW dxgkrnl, чтобы определить, какие процессы вызывают Present для каких дескрипторов окон, сравнивая PID вызывающего с PID владельца окна и помечая несанкционированный межпроцессный рендеринг через DirectX или OpenGL. Для рендеринга на базе GDI он обходит общую таблицу дескрипторов GDI в поисках дескрипторов DC, владелец которых отличается от целевого окна. Процессы с общим родительским процессом заносятся в белый список, чтобы разрешить легитимное совместное использование окон несколькими процессами. Предназначен для исследователей античита и оборонительной безопасности, изучающих техники обнаружения overlay.
