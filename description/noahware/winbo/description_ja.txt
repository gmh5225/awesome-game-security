本プロジェクトは、ETW と GDI テーブル走査によりオーバーレイ型ウィンドウハイジャックを検出する Windows C++ ツールです。dxgkrnl の ETW イベントを解析し、どのプロセスがどのウィンドウハンドルに対して Present を呼び出したかを特定し、呼び出し元 PID とウィンドウ所有者 PID を比較して、DirectX または OpenGL 経由の不正なクロスプロセス描画をフラグします。GDI ベースの描画では、共有 GDI ハンドルテーブルを走査し、所有者が対象ウィンドウと異なる DC ハンドルを見つけます。共通の親プロセスを共有するプロセスはホワイトリストに登録され、正当なマルチプロセスウィンドウ共有を許可します。オーバーレイ検出手法を研究するアンチチートおよび防御的セキュリティ研究者向けです。
