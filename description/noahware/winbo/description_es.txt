Este proyecto es una herramienta Windows en C++ para detectar el secuestro de ventanas de tipo overlay mediante ETW y el escaneo de la tabla GDI. Analiza eventos ETW de dxgkrnl para identificar qué procesos llaman a Present sobre qué handles de ventana, comparando el PID del llamador con el PID del propietario de la ventana para marcar el renderizado interproceso no autorizado vía DirectX u OpenGL. Para el renderizado basado en GDI, recorre la tabla compartida de handles GDI en busca de handles DC cuyo propietario difiera de la ventana objetivo. Los procesos que comparten un padre común se añaden a la lista blanca para permitir el uso compartido legítimo de ventanas multiproceso. Está orientado a investigadores de anti-trampas y seguridad defensiva que estudian técnicas de detección de overlays.
