이 프로젝트는 ETW와 GDI 테이블 스캔을 통해 오버레이 스타일 창 하이재킹을 탐지하는 Windows C++ 도구입니다. dxgkrnl ETW 이벤트를 파싱하여 어떤 프로세스가 어떤 창 핸들에서 Present를 호출하는지 식별하고, 호출자 PID와 창 소유자 PID를 비교하여 DirectX 또는 OpenGL을 통한 무단 크로스 프로세스 렌더링을 표시합니다. GDI 기반 렌더링의 경우 공유 GDI 핸들 테이블을 순회하여 소유자가 대상 창과 다른 DC 핸들을 찾습니다. 공통 부모를 공유하는 프로세스는 화이트리스트에 등록되어 정상적인 멀티 프로세스 창 공유를 허용합니다. 오버레이 탐지 기법을 연구하는 안티치트 및 방어 보안 연구자를 대상으로 합니다.
