Ursus は Rust で書かれたカスタム Vulkan ゲームエンジンで、Cargo workspace として構成され、コアエンジン crate、方針の明確な deferred パイプライン、モデルローダー、デモアプリを備えています。中心となるのは自動バリア追跡付きのレンダーグラフアーキテクチャ、deferred パイプライン（シャドウ、depth prepass、G-buffer、ライティング、ポストプロセス、FSR1 アップスケーリング）、および bindless テクスチャです。ゲームロジックとレンダリングは別スレッドで動作し、ロックフリーのトリプルバッファで同期され、OBJ/glTF アセットの非同期読み込み、hecs ベースの ECS、puffin による GPU タイムスタンププロファイリングを提供します。スタックは Linux と Windows を対象とし、Vulkan 1.3+、ash、ビルド時 SPIR-V シェーダーコンパイルを用い、現代的なリアルタイムレンダリングやゲームエンジン内部を構築・学習する開発者向けです。
