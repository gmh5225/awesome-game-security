Ursus는 Rust로 작성된 커스텀 Vulkan 게임 엔진으로, Cargo workspace로 구성되며 핵심 엔진 crate, 의견이 분명한 deferred 파이프라인, 모델 로더, 데모 앱을 포함합니다. 자동 배리어 추적을 갖춘 렌더 그래프 아키텍처, deferred 파이프라인(그림자, depth prepass, G-buffer, 라이팅, 포스트프로세스, FSR1 업스케일링), bindless 텍스처를 중심으로 설계되었습니다. 게임 로직과 렌더링은 별도의 스레드에서 실행되며 락프리 트리플 버퍼로 동기화되고, OBJ/glTF 에셋 비동기 로딩, hecs 기반 ECS, puffin을 통한 GPU 타임스탬프 프로파일링을 제공합니다. 이 스택은 Linux와 Windows를 대상으로 하며 Vulkan 1.3+, ash, 빌드 타임 SPIR-V 셰이더 컴파일을 사용하고, 현대적인 실시간 렌더링과 게임 엔진 내부를 구축하거나 학습하는 개발자를 위한 것입니다.
