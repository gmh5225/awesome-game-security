Ursus 是一款以 Rust 撰寫的自訂 Vulkan 遊戲引擎，以 Cargo workspace 組織，包含核心引擎 crate、具明確設計取向的延遲渲染管線、模型載入器以及示範應用。其核心是具備自動屏障追蹤的渲染圖架構、延遲管線（陰影、深度預通道、G-buffer、光照、後處理與 FSR1 超解析）以及 bindless 紋理。遊戲邏輯與渲染在獨立執行緒上執行，並透過無鎖三重緩衝同步，同時支援非同步載入 OBJ/glTF 資源、基於 hecs 的 ECS，以及透過 puffin 進行 GPU 時間戳效能分析。該技術堆疊面向 Linux 與 Windows，依賴 Vulkan 1.3+、ash 以及建置時 SPIR-V 著色器編譯，適合正在建構或研究現代即時渲染與遊戲引擎內部機制的開發者。
