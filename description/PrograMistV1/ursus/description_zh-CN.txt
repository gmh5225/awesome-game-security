Ursus 是一款用 Rust 编写的自定义 Vulkan 游戏引擎，以 Cargo workspace 组织，包含核心引擎 crate、有明确设计取向的延迟渲染管线、模型加载器以及演示应用。其核心是带有自动屏障跟踪的渲染图架构、延迟管线（阴影、深度预通道、G-buffer、光照、后处理与 FSR1 超分）以及 bindless 纹理。游戏逻辑与渲染在独立线程上运行，并通过无锁三重缓冲同步，同时支持异步加载 OBJ/glTF 资源、基于 hecs 的 ECS，以及通过 puffin 进行 GPU 时间戳性能分析。该技术栈面向 Linux 与 Windows，依赖 Vulkan 1.3+、ash 以及构建时 SPIR-V 着色器编译，适合正在构建或研究现代实时渲染与游戏引擎内部机制的开发者。
