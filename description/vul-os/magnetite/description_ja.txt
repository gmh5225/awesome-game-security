Magnetite は、権威あるマルチプレイヤーゲームを一度書けば、ノートPCから自前のフリートまで中央クラウドなしでどこでも動かせる、分散型・セルフホスト可能な Rust ゲームプラットフォームです。ゲームは決定的な権威サーバー SDK を対象に、wasm32-wasip1 へコンパイルされ、Wasmtime サンドボックス上で燃料予算、メモリ上限、epoch タイムアウト、シード付き決定的 RNG 付きで実行され、同じ入力は常に同じ状態を生み出します。チート対策は設計上組み込まれており、クライアントは入力のみ送信し、サーバーが検証してシミュレーションを進め、ReplayLog と verify_replay が tick を再シミュレーションして改ざん証拠を残し、magnetite-anticheat はエイムボットスナップ、テレポート、連射洪水などの合成可能な検証器と信頼スコア昇格を追加します。プラットフォームはさらにコンテンツアドレス型のゲーム配信、SingleRoom から Dedicated、Sharded までの自動選択トポロジ、および ID・ディスカバリ・通信・非カストディアル暗号決済向けの差し替え可能な接点も提供します。サンドボックス化されリプレイ検証可能なチート対策と、専有クラウドスタックではなくセルフホスト型マルチプレイヤー基盤を求めるゲーム開発者とホスト向けです。
