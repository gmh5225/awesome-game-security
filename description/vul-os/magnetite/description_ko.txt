Magnetite는 권한 있는 멀티플레이어 게임을 한 번 작성해 중앙 클라우드 없이 노트북부터 자체 보유 플릿까지 어디서나 실행할 수 있는 분산형·자체 호스팅 가능한 Rust 게임 플랫폼입니다. 게임은 결정론적 권한 서버 SDK를 대상으로 하며, wasm32-wasip1로 컴파일되고 Wasmtime 샌드박스에서 연료 예산, 메모리 상한, epoch 타임아웃, 시드 기반 결정론적 RNG와 함께 실행되어 동일한 입력은 항상 동일한 상태를 만듭니다. 안티치트는 구조적으로 내장됩니다. 클라이언트는 입력만 보내고 서버가 검증하며 시뮬레이션을 진행하고, ReplayLog와 verify_replay가 틱을 재시뮬레이션해 변조 증거를 남기며, magnetite-anticheat는 조준봇 스냅·텔레포트·연사 홍수 같은 조합 가능한 검증기와 신뢰 점수 상승을 추가합니다. 플랫폼은 또한 콘텐츠 주소형 게임 배포, SingleRoom에서 Dedicated, Sharded까지의 자동 선택 토폴로지, 신원·디스커버리·통신·비수탁 암호화폐 결제용 플러그형 연결점을 제공합니다. 샌드박스되고 리플레이로 검증되는 안티치트와 독점 클라우드 스택이 아닌 자체 호스팅 멀티플레이어 인프라가 필요한 게임 개발자와 호스트를 위한 것입니다.
