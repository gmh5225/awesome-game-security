Magnetite 是一个去中心化、可自托管的 Rust 游戏平台，让开发者一次编写权威多人游戏，即可从笔记本电脑到自有机群随处运行，无需中心云。游戏面向确定性权威服务器 SDK，编译为 wasm32-wasip1，并在 Wasmtime 沙箱中执行，具备燃料预算、内存上限、epoch 超时和带种子的确定性 RNG，使相同输入始终产生相同状态。反作弊从架构上内建：客户端仅发送输入，服务器校验并推进模拟，ReplayLog 与 verify_replay 重放 tick 以提供篡改证据，magnetite-anticheat 则提供可组合校验器（自瞄甩枪、瞬移、射速洪泛）并支持信任分升级。平台还提供内容寻址的游戏分发、从 SingleRoom 到 Dedicated 再到 Sharded 的自动拓扑选择，以及身份、发现、通信与非托管加密货币支付的可插拔接口。面向需要沙箱化、可回放验证反作弊与自托管多人基础设施，而非专有云栈的游戏开发者与主机方。
