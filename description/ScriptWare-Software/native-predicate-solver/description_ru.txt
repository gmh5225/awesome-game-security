Этот проект — нативный плагин Binary Ninja для удаления непрозрачных предикатов из обфусцированных бинарников. Он написан на современном C++ и анализирует условные ветвления MLIL, чтобы находить всегда истинные или всегда ложные условия. Плагин поддерживает проходы по одной функции и по всему бинарнику, настраиваемые ограничения и многопоточную обработку для ускорения работы с крупными программами. Он предназначен для инженеров по реверс-инжинирингу, которым нужны более быстрые рабочие процессы деобфускации при анализе вредоносного ПО или исследовании защищённых бинарников.
