Este proyecto es un plugin nativo de Binary Ninja para eliminar predicados opacos de binarios ofuscados. Está implementado en C++ moderno y analiza ramas condicionales de MLIL para detectar condiciones siempre verdaderas o siempre falsas. El plugin admite pases de una sola función y de todo el binario, límites configurables y procesamiento multihilo para ganar velocidad en programas grandes. Está pensado para ingenieros inversos que necesitan flujos de desofuscación más rápidos en el análisis de malware o en la investigación de binarios protegidos.
