Questo progetto è un plugin nativo di Binary Ninja per rimuovere i predicati opachi dai binari offuscati. È implementato in C++ moderno e analizza i rami condizionali MLIL per rilevare condizioni sempre vere o sempre false. Il plugin supporta passate su singola funzione e sull’intero binario, limiti configurabili e elaborazione multithread per maggiore velocità su programmi di grandi dimensioni. È pensato per reverse engineer che necessitano di workflow di deoffuscamento più rapidi nell’analisi di malware o nella ricerca su binari protetti.
