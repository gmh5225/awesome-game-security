Este proyecto es un ejemplo de hooking de Direct3D 12 que inyecta una superposición ImGui en juegos DX12. Está implementado en C++ con las API de DirectX 12 y patrones de interceptación de funciones al estilo MinHook. El código muestra un flujo práctico de compilación e inyección de DLL, la configuración del hook de fotograma y el renderizado de menús en tiempo real. Se usa principalmente para investigación de API gráficas, prototipado de overlays de juegos y estudios de instrumentación de bajo nivel en juegos.
