Этот проект — пример хукинга Direct3D 12, который внедряет оверлей ImGui в DX12-игры. Он реализован на C++ с использованием API DirectX 12 и паттернов перехвата функций в стиле MinHook. Кодовая база демонстрирует практический процесс сборки и внедрения DLL, настройку кадрового хука и отрисовку меню в реальном времени. В основном применяется для исследований графических API, прототипирования игровых оверлеев и изучения низкоуровневой инструментации игр.
