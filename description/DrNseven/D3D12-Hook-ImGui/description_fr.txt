Ce projet est un exemple de hook Direct3D 12 qui injecte une overlay ImGui dans des jeux DX12. Il est implémenté en C++ avec les API DirectX 12 et des schémas d’interception de fonctions de type MinHook. Le code illustre un flux concret de compilation et d’injection de DLL, la configuration du hook de frame et le rendu de menus en temps réel. Il sert surtout à la recherche sur les API graphiques, au prototypage d’overlays de jeu et à l’étude de l’instrumentation bas niveau des jeux.
