Questo progetto è un esempio di hooking Direct3D 12 che inietta un overlay ImGui nei giochi DX12. È implementato in C++ con le API DirectX 12 e schemi di intercettazione delle funzioni in stile MinHook. Il codice mostra un flusso pratico di build e iniezione di DLL, la configurazione dell’hook di frame e il rendering di menu in tempo reale. È usato soprattutto per la ricerca sulle API grafiche, la prototipazione di overlay di gioco e lo studio dell’instrumentazione a basso livello dei giochi.
