Этот репозиторий — проект wallhack на основе хуков Direct3D 11 для 32- и 64-битных игр Windows. Он внедряет DLL, перехватывает пути рендеринга и предоставляет меню-ориентированный процесс идентификации целевых моделей через логирование stride и index count. Кодовая база написана на C++ с использованием распространённых библиотек для game hacking, таких как Detours и интеграция ImGui. В первую очередь предназначен для прототипирования читов и анализа графического конвейера в исследованиях игровой безопасности.
