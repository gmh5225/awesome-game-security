Ce dépôt est un projet de wallhack basé sur des hooks Direct3D 11 pour les jeux Windows 32 bits et 64 bits. Il injecte une DLL, accroche les chemins de rendu et propose un flux de travail piloté par menu pour identifier les modèles cibles via la journalisation du stride et du nombre d'index. Le code utilise C++ avec des bibliothèques courantes de game hacking telles que Detours et l'intégration ImGui. Il est principalement destiné au prototypage de cheats et à l'analyse du pipeline graphique dans la recherche en sécurité des jeux.
