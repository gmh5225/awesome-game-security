Este repositorio es un proyecto de wallhack basado en hooks de Direct3D 11 para juegos de Windows de 32 y 64 bits. Inyecta una DLL, engancha las rutas de renderizado y ofrece un flujo de trabajo basado en menús para identificar modelos objetivo mediante el registro de stride y del recuento de índices. El código usa C++ con bibliotecas habituales de game hacking como Detours e integración de ImGui. Está pensado principalmente para el prototipado de cheats y el análisis del pipeline gráfico en la investigación de seguridad de juegos.
