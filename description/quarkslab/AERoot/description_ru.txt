Этот проект — инструмент на Python от Quarkslab для получения root на экземплярах Android Emulator во время выполнения без изменения системного образа. Он использует отладочный канал эмулятора или доступ ADB root, чтобы перемонтировать системный раздел в режим чтения-записи, установить пользовательский бинарный файл su и обеспечить постоянный root. Инструмент автоматизирует процесс для нескольких уровней Android API. Предназначен для исследователей мобильной безопасности и разработчиков Android, которым нужен root на эмуляторах для тестирования и реверс-инжиниринга.
