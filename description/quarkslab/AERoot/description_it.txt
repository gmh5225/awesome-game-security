Questo progetto è uno strumento Python di Quarkslab per rootare istanze di Android Emulator a runtime senza modificare l'immagine di sistema. Sfrutta la debug pipe dell'emulatore o l'accesso ADB root per rimontare la partizione di sistema in lettura-scrittura, installare un binario su personalizzato e stabilire un root persistente. Lo strumento automatizza il processo su più livelli API di Android. È rivolto a ricercatori di sicurezza mobile e sviluppatori Android che necessitano di accesso root sulle istanze dell'emulatore per test e reverse engineering.
