Este proyecto es una herramienta en Python de Quarkslab para rootear instancias de Android Emulator en tiempo de ejecución sin modificar la imagen del sistema. Aprovecha el pipe de depuración del emulador o el acceso ADB root para remontar la partición del sistema en lectura-escritura, instalar un binario su personalizado y establecer un root persistente. La herramienta automatiza el proceso en varios niveles de API de Android. Está dirigida a investigadores de seguridad móvil y desarrolladores de Android que necesitan acceso root en emuladores para pruebas e ingeniería inversa.
