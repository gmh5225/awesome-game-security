本プロジェクトは Quarkslab による Python ツールで、システムイメージを変更せずに Android Emulator インスタンスを実行時に root 化できます。エミュレータのデバッグパイプまたは ADB root アクセスを利用し、システムパーティションを読み書き可能に再マウントし、カスタム su バイナリをインストールして、永続的な root を確立します。複数の Android API レベルにわたってこの処理を自動化します。テストやリバースエンジニアリングのためにエミュレータ上で root が必要なモバイルセキュリティ研究者および Android 開発者を対象としています。
