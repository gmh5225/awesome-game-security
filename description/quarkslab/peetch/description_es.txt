Este proyecto se centra en un toolkit eBPF: dump (sniffear tráfico con PID/proceso), tls (extracción de claves/master-secret de OpenSSL), proxy (interceptar y descifrar TLS); PCAPng + Scapy; OpenSSL, IPv4, TLS 1.2.
El primero, llamado dump, busca sniffer el tráfico de red asociando a cada paquete información sobre el proceso de origen.
Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / exploración del kernel de Android.
