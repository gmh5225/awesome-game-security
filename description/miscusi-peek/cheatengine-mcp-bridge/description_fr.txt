Pont Model Context Protocol (MCP) qui connecte des agents IA à Cheat Engine via une architecture Named Pipe, exposant plus de 40 outils pour la lecture mémoire, le scan AOB, le parcours de chaînes de pointeurs, la dissection de structures, l'identification de classes RTTI, les points d'arrêt matériels (DR0-DR3) et le traçage au niveau hyperviseur DBVM. Utilise un thread worker Lua dans CE synchronisé avec un serveur Python FastMCP pour une latence de commande inférieure à 2 ms.
