Bridge Model Context Protocol (MCP) che collega agenti IA a Cheat Engine tramite un'architettura Named Pipe, esponendo oltre 40 strumenti per la lettura della memoria, scansione AOB, attraversamento di catene di puntatori, dissezione di strutture, identificazione di classi RTTI, breakpoint hardware (DR0-DR3) e tracing a livello hypervisor DBVM. Usa un thread worker Lua in CE sincronizzato con un server Python FastMCP per una latenza dei comandi inferiore a 2 ms.
