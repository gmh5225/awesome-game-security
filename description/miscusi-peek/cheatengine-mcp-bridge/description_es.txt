Puente Model Context Protocol (MCP) que conecta agentes de IA a Cheat Engine mediante una arquitectura Named Pipe, exponiendo más de 40 herramientas para lectura de memoria, escaneo AOB, recorrido de cadenas de punteros, disección de estructuras, identificación de clases RTTI, puntos de interrupción de hardware (DR0-DR3) y trazado a nivel de hipervisor DBVM. Usa un hilo worker Lua en CE sincronizado con un servidor Python FastMCP para una latencia de comandos inferior a 2 ms.
