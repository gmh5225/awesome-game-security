Мост Model Context Protocol (MCP), соединяющий ИИ-агентов с Cheat Engine через архитектуру Named Pipe и предоставляющий более 40 инструментов для чтения памяти, AOB-сканирования, обхода цепочек указателей, разбора структур, идентификации классов RTTI, аппаратных точек останова (DR0-DR3) и трассировки на уровне гипервизора DBVM. Использует рабочий поток Lua в CE, синхронизированный с сервером Python FastMCP, для задержки команд менее 2 мс.
