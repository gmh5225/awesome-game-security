Model Context Protocol (MCP)-Bridge, die KI-Agenten über eine Named-Pipe-Architektur mit Cheat Engine verbindet und über 40 Tools für Speicherlesen, AOB-Scanning, Pointer-Chain-Traversal, Strukturanalyse, RTTI-Klassenidentifikation, Hardware-Breakpoints (DR0-DR3) und DBVM-Hypervisor-Tracing bereitstellt. Nutzt einen Lua-Worker-Thread in CE, synchronisiert mit einem Python-FastMCP-Server, für eine Befehlslatenz unter 2 ms.
