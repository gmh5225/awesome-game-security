MSDoggy — старый проект игрового хака для MapleStory из тайваньского сообщества. Он содержит хуки на C/C++ и встроенном ассемблере для манипуляции геймплеем, включая изменение поведения мобов и логику фильтрации, связанную с предметами. Код отражает устаревшие практики разработки читов с низкоуровневым патчингом и опциональной интеграцией протекторов. В первую очередь полезен как исторический справочный материал для исследований читов в MMORPG.
