Этот проект — система сборки, которая патчит Frida для обхода распространённых механизмов детектирования анти-инструментации. Она использует патч-скрипты на Python и генераторы имён для рандомизации идентифицируемых строк, символов и артефактов Frida в процессе сборки, с поддержкой сборки в WSL и комплексным тестированием на JavaScript. В первую очередь полезна мобильным реверс-инженерам и исследователям безопасности, которым нужны скрытые сборки Frida, обходящие анти-Frida детект в защищённых приложениях.
