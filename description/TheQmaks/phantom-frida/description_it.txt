Questo progetto è un sistema di build che patcha Frida per eludere i comuni meccanismi di rilevamento anti-instrumentation. Usa script di patch basati su Python e generatori di nomi per randomizzare stringhe, simboli e artefatti identificabili di Frida durante il processo di build, con supporto per build su WSL e test completi basati su JavaScript. È soprattutto utile per reverse engineer mobile e ricercatori di sicurezza che necessitano di build Frida stealth in grado di bypassare il rilevamento anti-Frida nelle applicazioni protette.
