Este proyecto es un sistema de compilación que parchea Frida para evadir mecanismos habituales de detección antiinstrumentación. Utiliza scripts de parche en Python y generadores de nombres para aleatorizar las cadenas, símbolos y artefactos identificables de Frida durante el proceso de build, con soporte de compilación en WSL y pruebas exhaustivas basadas en JavaScript. Resulta especialmente útil para ingenieros de ingeniería inversa móvil e investigadores de seguridad que necesitan builds furtivos de Frida que eludan la detección anti-Frida en aplicaciones protegidas.
