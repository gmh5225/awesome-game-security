Este proyecto es una plataforma basada en FPGA para flujos de ataque PCIe DMA prearranque autónomos en hardware compacto.
Combina Verilog HDL, recursos de proyectos Vivado y Vitis, y software embebido que se ejecuta en un soft processor MicroBlaze.
La implementación admite la ejecución autónoma de payloads, así como operación controlada por UART y flasheo SPI para mayor flexibilidad de despliegue.
Su caso de uso principal es la investigación de seguridad de arranque y firmware asistida por hardware, incluido el prototipado de implantes DMA.
