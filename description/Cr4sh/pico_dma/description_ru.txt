Этот проект — FPGA-платформа для автономных pre-boot PCIe DMA атак на компактном оборудовании.
Она объединяет Verilog HDL, ресурсы проектов Vivado и Vitis, а также встроенное ПО, работающее на soft-процессоре MicroBlaze.
Реализация поддерживает автономное выполнение payload, а также управление по UART и прошивку по SPI для гибкого развёртывания.
Основной сценарий использования — аппаратно-поддерживаемые исследования безопасности загрузки и прошивок, включая прототипирование DMA-имплантов.
