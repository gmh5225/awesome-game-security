本プロジェクトは、Xilinx SP605 Spartan-6 FPGA 上にソフトウェア制御可能な PCIe Gen 1.1 エンドポイントを実装し、DMA 攻撃研究に用いる PCI Express DIY ハッキングツールキットです。HDL と MicroBlaze ファームウェアにより、生の Transaction Layer Packet を Ethernet 経由で公開し、ホストメモリの読み書き、物理メモリスキャン、IOMMU 監査用の Python ツールも提供します。概念実証ペイロードでは、UEFI DXE ドライバを注入するプリブート DMA 攻撃、ハイパーバイザ検査とゲストからホストへの脱出研究向け Hyper-V VM-exit バックドア、実行時 DMA シェルコマンドとファイル転送を備えた Boot Backdoor を示します。主要言語は Verilog、MicroBlaze および UEFI ペイロード向け C、ホスト側ツール向け Python、バックドアクライアント向け C++ です。DMA 脅威、Option ROM とプリブート侵害、保護された Windows プラットフォームのリバースエンジニアリングを研究するセキュリティ研究者を対象としています。
