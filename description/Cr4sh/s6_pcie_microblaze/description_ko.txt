이 프로젝트는 Xilinx SP605 Spartan-6 FPGA에서 소프트웨어로 제어 가능한 PCIe Gen 1.1 엔드포인트를 구현하여 DMA 공격 연구에 쓰는 PCI Express DIY 해킹 툴킷입니다. HDL과 MicroBlaze 펌웨어로 원시 Transaction Layer Packet을 Ethernet을 통해 노출하며, 호스트 메모리 읽기/쓰기, 물리 메모리 스캔, IOMMU 감사를 위한 Python 도구를 포함합니다. 개념 증명 페이로드는 UEFI DXE 드라이버를 주입하는 프리부트 DMA 공격, 하이퍼바이저 검사와 게스트-호스트 탈출 연구용 Hyper-V VM-exit 백도어, 런타임 DMA 셸 명령과 파일 전송이 가능한 Boot Backdoor를 보여 줍니다. 주요 언어는 Verilog, MicroBlaze 및 UEFI 페이로드용 C, 호스트 측 도구용 Python, 백도어 클라이언트용 C++입니다. DMA 위협, Option ROM과 프리부트 침해, 보호된 Windows 플랫폼 리버스 엔지니어링을 연구하는 보안 연구자를 대상으로 합니다.
