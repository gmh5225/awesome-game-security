本项目是一套 PCI Express DIY 黑客工具包，在 Xilinx SP605 Spartan-6 FPGA 上实现可由软件控制的 PCIe Gen 1.1 端点，用于 DMA 攻击研究。它包含 HDL 与 MicroBlaze 固件，可通过以太网暴露原始 Transaction Layer Packet，并提供用于主机内存读写、物理内存扫描与 IOMMU 审计的 Python 工具。概念验证载荷演示了在预启动阶段注入 UEFI DXE 驱动的 DMA 攻击、用于虚拟机监视器检查与客户机到宿主逃逸研究的 Hyper-V VM-exit 后门，以及带有运行时 DMA shell 命令与文件传输的 Boot Backdoor。主要语言包括 Verilog、面向 MicroBlaze 与 UEFI 载荷的 C、主机侧工具的 Python，以及后门客户端的 C++。项目面向研究 DMA 威胁、Option ROM 与预启动入侵，以及对受保护 Windows 平台进行逆向工程的安全研究人员。
