Dieses Projekt ist ein PCI-Express-DIY-Hacking-Toolkit, das einen softwaregesteuerten PCIe-Gen-1.1-Endpunkt auf einem Xilinx-SP605-Spartan-6-FPGA für die Forschung zu DMA-Angriffen implementiert. Es enthält HDL und MicroBlaze-Firmware, die rohe Transaction Layer Packets über Ethernet bereitstellen, sowie Python-Tools für Host-Speicherlesen/-schreiben, physisches Speicherscannen und IOMMU-Audits. Proof-of-Concept-Payloads demonstrieren Pre-Boot-DMA-Angriffe mit Injektion von UEFI-DXE-Treibern, eine Hyper-V-VM-exit-Backdoor zur Hypervisor-Inspektion und zur Erforschung von Guest-to-Host-Escapes sowie eine Boot Backdoor mit DMA-Shell-Befehlen zur Laufzeit und Dateitransfer. Zu den Hauptsprachen gehören Verilog, C für MicroBlaze- und UEFI-Payloads, Python für Host-seitige Tools und C++ für Backdoor-Clients. Es richtet sich an Sicherheitsforscher, die DMA-Bedrohungen, Option-ROM- und Pre-Boot-Kompromittierung sowie Reverse Engineering geschützter Windows-Plattformen untersuchen.
