Questo progetto è un toolkit DIY di hacking PCI Express che implementa un endpoint PCIe Gen 1.1 controllabile via software su un FPGA Xilinx SP605 Spartan-6 per la ricerca sugli attacchi DMA. Include HDL e firmware MicroBlaze che espongono Transaction Layer Packet grezzi via Ethernet, oltre a strumenti Python per lettura/scrittura della memoria host, scansione della memoria fisica e audit IOMMU. I payload proof-of-concept dimostrano attacchi DMA pre-boot che iniettano driver UEFI DXE, un backdoor Hyper-V VM-exit per l'ispezione dell'hypervisor e la ricerca di escape guest-to-host, e un Boot Backdoor con comandi shell DMA a runtime e trasferimento file. I linguaggi principali includono Verilog, C per MicroBlaze e payload UEFI, Python per gli strumenti lato host e C++ per i client del backdoor. È rivolto a ricercatori di sicurezza che studiano minacce DMA, compromissione Option ROM e pre-boot, e reverse engineering di piattaforme Windows protette.
