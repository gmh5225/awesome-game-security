本專案是一套 PCI Express DIY 駭客工具包，在 Xilinx SP605 Spartan-6 FPGA 上實作可由軟體控制的 PCIe Gen 1.1 端點，用於 DMA 攻擊研究。它包含 HDL 與 MicroBlaze 韌體，可透過乙太網路暴露原始 Transaction Layer Packet，並提供用於主機記憶體讀寫、實體記憶體掃描與 IOMMU 稽核的 Python 工具。概念驗證載荷展示了在預開機階段注入 UEFI DXE 驅動程式的 DMA 攻擊、用於虛擬機器監視器檢查與客體到宿主逃逸研究的 Hyper-V VM-exit 後門，以及具備執行時期 DMA shell 指令與檔案傳輸的 Boot Backdoor。主要語言包括 Verilog、面向 MicroBlaze 與 UEFI 載荷的 C、主機端工具的 Python，以及後門用戶端的 C++。專案面向研究 DMA 威脅、Option ROM 與預開機入侵，以及對受保護 Windows 平台進行逆向工程的資安研究人員。
