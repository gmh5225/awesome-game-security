This project is a PCI Express DIY hacking toolkit that implements a software-controllable PCIe Gen 1.1 endpoint on a Xilinx SP605 Spartan-6 FPGA for DMA attack research. It includes HDL and MicroBlaze firmware that expose raw Transaction Layer Packets over Ethernet, plus Python tools for host memory read/write, physical memory scanning, and IOMMU audits. Proof-of-concept payloads demonstrate pre-boot DMA attacks that inject UEFI DXE drivers, a Hyper-V VM-exit backdoor for hypervisor inspection and guest-to-host escape research, and a Boot Backdoor with runtime DMA shell command and file transfer. Primary languages include Verilog, C for MicroBlaze and UEFI payloads, Python for host-side tooling, and C++ for backdoor clients. It is aimed at security researchers studying DMA threats, Option ROM and pre-boot compromise, and reverse engineering of protected Windows platforms.
