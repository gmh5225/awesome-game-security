Этот проект — DIY-набор инструментов для исследования PCI Express, реализующий программно управляемую конечную точку PCIe Gen 1.1 на FPGA Xilinx SP605 Spartan-6 для изучения DMA-атак. Он включает HDL и прошивку MicroBlaze, которые передают сырые Transaction Layer Packet по Ethernet, а также инструменты на Python для чтения/записи памяти хоста, сканирования физической памяти и аудита IOMMU. Proof-of-concept полезные нагрузки демонстрируют pre-boot DMA-атаки с внедрением драйверов UEFI DXE, бэкдор Hyper-V VM-exit для инспекции гипервизора и исследования побега guest-to-host, а также Boot Backdoor с DMA shell-командами во время выполнения и передачей файлов. Основные языки — Verilog, C для MicroBlaze и UEFI-нагрузок, Python для инструментов на стороне хоста и C++ для клиентов бэкдора. Проект рассчитан на исследователей безопасности, изучающих угрозы DMA, компрометацию через Option ROM и на этапе до загрузки, а также реверс-инжиниринг защищённых платформ Windows.
