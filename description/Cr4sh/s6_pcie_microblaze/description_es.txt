Este proyecto es un kit de herramientas DIY de hacking PCI Express que implementa un endpoint PCIe Gen 1.1 controlable por software en un FPGA Xilinx SP605 Spartan-6 para la investigación de ataques DMA. Incluye HDL y firmware MicroBlaze que exponen Transaction Layer Packets en bruto por Ethernet, además de herramientas Python para lectura/escritura de memoria del host, escaneo de memoria física y auditorías IOMMU. Los payloads de prueba de concepto demuestran ataques DMA previos al arranque que inyectan controladores UEFI DXE, un backdoor Hyper-V VM-exit para inspección del hipervisor e investigación de escape guest-to-host, y un Boot Backdoor con comandos shell DMA en tiempo de ejecución y transferencia de archivos. Los lenguajes principales incluyen Verilog, C para MicroBlaze y payloads UEFI, Python para herramientas del host y C++ para clientes del backdoor. Está dirigido a investigadores de seguridad que estudian amenazas DMA, compromiso de Option ROM y previo al arranque, y la ingeniería inversa de plataformas Windows protegidas.
