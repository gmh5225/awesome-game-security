Этот проект — фреймворк бэкдора UEFI в режиме System Management Mode с компонентами прошивки и клиентской стороны для низкоуровневых исследований платформы.
Он сочетает код DXE и SMM на C с инструментами на Python для взаимодействия, развёртывания и демонстраций в Windows и Linux.
Архитектура поддерживает несколько путей развёртывания, включая заражение flash-образа и загрузку с помощью DMA до запуска ОС, а также примеры повышения привилегий и взаимодействия с гипервизором.
Предназначен для исследований безопасности прошивок, изучения персистентности и анализа поверхностей атаки до загрузки ОС.
