Este proyecto es un framework de backdoor UEFI en System Management Mode, con componentes de firmware y de cliente para investigación de plataformas a bajo nivel.
Combina código DXE y SMM en C con herramientas Python para interacción, despliegue y demostraciones en Windows y Linux.
El diseño admite varias vías de despliegue, incluida la infección de imágenes flash y la carga asistida por DMA en prearranque, con ejemplos de escalada de privilegios e interacción con hipervisores.
Está pensado para investigación de seguridad de firmware, estudios de persistencia y análisis de superficies de ataque previas al sistema operativo.
