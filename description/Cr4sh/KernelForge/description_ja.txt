本プロジェクトは、VBS および HVCI で強化されたシステム上で、ユーザーモードからカーネルルーチンを呼び出すための Windows ライブラリです。
C++ で実装され、分割設計を採用しています。一方のコンポーネントは署名付きドライバーラッパーを通じてカーネルメモリプリミティブを提供し、もう一方はより高水準の関数呼び出し機能を構築します。
リポジトリにはヘッダー、静的ライブラリ、DLL バインディング、およびカーネルからユーザーモードへの DLL インジェクション技術を示す例が含まれています。
主な用途は、現代的なプラットフォーム防御下での高度なカーネルセキュリティ研究およびエクスプロイトのプロトタイピングです。
