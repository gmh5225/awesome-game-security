Этот проект — библиотека Windows для вызова процедур ядра из пользовательского режима на системах, усиленных с помощью VBS и HVCI.
Он реализован на C++ с разделённой архитектурой: один компонент предоставляет примитивы работы с памятью ядра через обёртку подписанного драйвера, а другой строит возможности вызова функций более высокого уровня.
Репозиторий включает заголовочные файлы, статические библиотеки, привязки DLL и пример, демонстрирующий техники инъекции DLL из ядра в пользовательский режим.
Основной сценарий использования — углублённые исследования безопасности ядра и прототипирование эксплойтов в условиях современных защит платформы.
