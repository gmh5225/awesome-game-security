Este proyecto es una biblioteca de Windows para invocar rutinas del kernel desde el modo de usuario en sistemas reforzados por VBS y HVCI.
Está implementado en C++ con un diseño dividido: un componente proporciona primitivas de memoria del kernel mediante un contenedor de controlador firmado, y otro construye capacidades de llamada a funciones de más alto nivel.
El repositorio incluye encabezados, bibliotecas estáticas, enlaces DLL y un ejemplo que demuestra técnicas de inyección de DLL de kernel a usuario.
Su caso de uso principal es la investigación avanzada de seguridad del kernel y la creación de prototipos de exploits bajo las defensas modernas de las plataformas.
