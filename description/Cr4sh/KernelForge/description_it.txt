Questo progetto è una libreria Windows per invocare routine del kernel dalla modalità utente su sistemi rafforzati da VBS e HVCI.
È implementato in C++ con un design suddiviso: un componente fornisce primitive di memoria del kernel tramite un wrapper di driver firmato, e un altro costruisce capacità di chiamata a funzioni di livello superiore.
Il repository include header, librerie statiche, binding DLL e un esempio che dimostra tecniche di iniezione DLL da kernel a modalità utente.
Il caso d'uso principale è la ricerca avanzata sulla sicurezza del kernel e la prototipazione di exploit sotto le difese moderne delle piattaforme.
