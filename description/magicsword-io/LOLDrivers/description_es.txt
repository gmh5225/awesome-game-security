Este proyecto es un catálogo completo de Living Off The Land Drivers (LOLDrivers): controladores de Windows legítimos y firmados que pueden abusarse con fines maliciosos. Incluye metadatos de controladores en YAML, enriquecimiento con firmas YARA, seguimiento de controladores HVCI, generación de sitios basada en Jinja2 y herramientas de validación para mantener la base de datos de controladores. Es especialmente útil para investigadores de BYOVD, cazadores de amenazas e ingenieros de anti-trampas que siguen controladores de Windows firmados vulnerables y explotables.
