Ce projet est le Turbulenz Engine, un moteur de jeu HTML5 open source écrit en TypeScript/JavaScript. Il fournit un framework complet de développement de jeux avec rendu 3D (WebGL), physique, audio, réseau, gestion des entrées, gestion des assets et un graphe de scène. Le moteur inclut à la fois un runtime de jeu côté client et des services côté serveur pour les classements, le multijoueur et les métriques. Il s’adresse aux développeurs de jeux web qui créent des jeux 2D et 3D dans le navigateur à l’aide des technologies web.
