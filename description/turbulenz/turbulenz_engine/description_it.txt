Questo progetto è il Turbulenz Engine, un motore di gioco HTML5 open source scritto in TypeScript/JavaScript. Fornisce un framework completo per lo sviluppo di giochi con rendering 3D (WebGL), fisica, audio, networking, gestione degli input, gestione degli asset e un scene graph. Il motore include sia un runtime di gioco lato client sia servizi lato server per classifiche, multiplayer e metriche. È rivolto agli sviluppatori di giochi web che creano giochi 2D e 3D basati sul browser con tecnologie web.
