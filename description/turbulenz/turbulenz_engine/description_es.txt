Este proyecto es Turbulenz Engine, un motor de juegos HTML5 de código abierto escrito en TypeScript/JavaScript. Ofrece un framework completo de desarrollo de juegos con renderizado 3D (WebGL), física, audio, redes, manejo de entrada, gestión de assets y un grafo de escena. El motor incluye tanto un runtime de juego del lado del cliente como servicios del lado del servidor para clasificaciones, multijugador y métricas. Está orientado a desarrolladores de juegos web que crean juegos 2D y 3D basados en el navegador con tecnologías web.
