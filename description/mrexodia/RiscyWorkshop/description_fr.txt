Supports d'atelier pour l'obfuscation de payloads basés sur RISC-V, avec un interpréteur de VM personnalisé (riscvm), une chaîne de compilation croisée C vers RISC-V (llvm-mingw), et un obfuscateur appliquant l'analyse de vivacité et la mutation d'instructions aux binaires RISC-V. Inclut des exercices sur les bases de la VM, la génération de shellcode, l'interaction avec les API hôte, la transpilation et les techniques de durcissement anti-analyse.
