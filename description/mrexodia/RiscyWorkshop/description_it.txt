Materiali di workshop per l'offuscamento di payload basati su RISC-V, con un interprete VM personalizzato (riscvm), una toolchain di cross-compilazione da C a RISC-V (llvm-mingw) e un offuscatore che applica analisi di liveness e mutazione delle istruzioni ai binari RISC-V. Include esercizi su basi della VM, generazione di shellcode, interazione con le API host, transpilation e tecniche di hardening anti-analisi.
