Материалы воркшопа по обфускации payload на базе RISC-V: собственный интерпретатор ВМ (riscvm), кросс-компиляторный тулчейн C в RISC-V (llvm-mingw) и обфускатор, применяющий анализ живучести и мутацию инструкций к бинарникам RISC-V. Включает упражнения по основам ВМ, генерации шеллкода, взаимодействию с API хоста, транспиляции и методам антианалитического усиления.
