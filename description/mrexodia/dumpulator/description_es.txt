Framework de Python para emular código x86/x64 directamente desde archivos minidump de Windows mediante Unicorn Engine, reconstruyendo el diseño de memoria del proceso, los contextos de hilos, los módulos cargados y la tabla de handles a partir de los flujos del dump. Proporciona stubs de syscalls NT, emulación de PEB/TEB y enganche de API para ejecutar funciones arbitrarias desde el espacio de direcciones del dump sin un proceso en ejecución.
