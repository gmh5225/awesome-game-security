Python-фреймворк для эмуляции кода x86/x64 напрямую из файлов Windows minidump с помощью Unicorn Engine, восстанавливающий раскладку памяти процесса, контексты потоков, загруженные модули и таблицу дескрипторов из потоков дампа. Предоставляет заглушки NT-системных вызовов, эмуляцию PEB/TEB и перехват API для выполнения произвольных функций из адресного пространства дампа без живого процесса.
