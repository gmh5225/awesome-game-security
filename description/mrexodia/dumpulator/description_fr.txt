Framework Python pour émuler du code x86/x64 directement à partir de fichiers minidump Windows via Unicorn Engine, en reconstruisant la disposition mémoire du processus, les contextes de threads, les modules chargés et la table des handles à partir des flux du dump. Fournit des stubs d'appels système NT, l'émulation PEB/TEB et le hooking d'API pour exécuter des fonctions arbitraires depuis l'espace d'adressage du dump sans processus vivant.
