Framework Python per emulare codice x86/x64 direttamente da file minidump di Windows tramite Unicorn Engine, ricostruendo il layout di memoria del processo, i contesti dei thread, i moduli caricati e la tabella degli handle dai flussi del dump. Fornisce stub di syscall NT, emulazione PEB/TEB e hooking delle API per eseguire funzioni arbitrarie dallo spazio di indirizzi del dump senza un processo attivo.
