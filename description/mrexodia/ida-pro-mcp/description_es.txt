Servidor Model Context Protocol (MCP) para IDA Pro que expone la API completa de IDAPython a agentes de IA, permitiendo tareas automatizadas de análisis binario como renombrado de funciones, anotación de tipos, consultas de referencias cruzadas, descompilación y creación de estructuras. Incluye documentación completa de IDAPython, un complemento instalable de IDA y un marco de pruebas para validar el comportamiento de las herramientas MCP.
