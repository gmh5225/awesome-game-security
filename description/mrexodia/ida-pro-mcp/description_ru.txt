Сервер Model Context Protocol (MCP) для IDA Pro, предоставляющий полный API IDAPython ИИ-агентам и позволяющий автоматизировать анализ бинарных файлов: переименование функций, аннотирование типов, запросы перекрёстных ссылок, декомпиляцию и создание структур. Включает подробную документацию IDAPython, устанавливаемый плагин IDA и тестовый фреймворк для проверки поведения инструментов MCP.
