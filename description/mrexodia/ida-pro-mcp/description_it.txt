Server Model Context Protocol (MCP) per IDA Pro che espone l'intera API IDAPython agli agenti IA, consentendo attività automatizzate di analisi binaria come rinomina di funzioni, annotazione dei tipi, query di riferimenti incrociati, decompilazione e creazione di strutture. Include documentazione IDAPython completa, un plugin IDA installabile e un framework di test per validare il comportamento degli strumenti MCP.
