Serveur Model Context Protocol (MCP) pour IDA Pro qui expose l'intégralité de l'API IDAPython aux agents IA, permettant des tâches d'analyse binaire automatisées telles que le renommage de fonctions, l'annotation de types, les requêtes de références croisées, la décompilation et la création de structures. Inclut une documentation IDAPython complète, un plugin IDA installable et un framework de tests pour valider le comportement des outils MCP.
