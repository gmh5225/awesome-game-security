Плагин x64dbg, отключающий параллельный загрузчик DLL Windows (появился в Windows 10) путём патча LdrpMapAndSnapWork и связанных внутренних структур ntdll через Process Hacker Native API (phnt). Принудительно загружает зависимости последовательно при создании процесса, упрощая трассировку и отладку порядка загрузки DLL в x64dbg.
