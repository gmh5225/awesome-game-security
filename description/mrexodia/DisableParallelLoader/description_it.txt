Plugin di x64dbg che disabilita il caricatore parallelo di DLL di Windows (introdotto in Windows 10) patchando LdrpMapAndSnapWork e le relative strutture interne di ntdll tramite la Process Hacker Native API (phnt). Forza il caricamento sequenziale delle dipendenze durante la creazione del processo, facilitando il tracciamento e il debug dell'ordine di caricamento delle DLL in x64dbg.
