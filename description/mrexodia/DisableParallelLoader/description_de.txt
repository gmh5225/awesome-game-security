x64dbg-Plugin, das den parallelen Windows-DLL-Loader (eingeführt in Windows 10) deaktiviert, indem es LdrpMapAndSnapWork und verwandte ntdll-Internals über die Process Hacker Native API (phnt) patcht. Erzwingt beim Prozesserstellen das sequenzielle Laden von Abhängigkeiten und erleichtert so das Nachverfolgen und Debuggen der DLL-Ladereihenfolge in x64dbg.
