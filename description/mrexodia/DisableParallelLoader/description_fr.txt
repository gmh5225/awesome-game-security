Plugin x64dbg qui désactive le chargeur parallèle de DLL Windows (introduit dans Windows 10) en patchant LdrpMapAndSnapWork et les internals ntdll associés via l'API native Process Hacker (phnt). Force le chargement séquentiel des dépendances lors de la création du processus, facilitant le suivi et le débogage de l'ordre de chargement des DLL dans x64dbg.
