Минимальный шаблон CMake для сборки UEFI-приложений с заголовками EDK2 SDK без полной системы сборки EDK2. Создаёт автономные .efi-бинарники, скомпилированные MSVC (/GS- /EHs-) и слинкованные с /SUBSYSTEM:EFI_APPLICATION, предоставляя bare-metal доступ к точке входа EFI_SYSTEM_TABLE и boot-сервисам.
