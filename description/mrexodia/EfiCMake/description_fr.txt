Modèle CMake minimal pour créer des applications UEFI avec les en-têtes du SDK EDK2, sans le système de build EDK2 complet. Produit des binaires .efi autonomes compilés avec MSVC (/GS- /EHs-) et liés avec /SUBSYSTEM:EFI_APPLICATION, offrant un accès bare-metal au point d’entrée vers EFI_SYSTEM_TABLE et les services de démarrage.
