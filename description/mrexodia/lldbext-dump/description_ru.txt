Расширение LLDB на Python, которое создаёт полные дампы памяти в формате minidump из активной сессии отладки Android, сохраняя все отображённые области памяти, контексты потоков и списки модулей в совместимом с Windows формате minidump. Включает сопутствующий скрипт эмуляции для воспроизведения дампов с выполнением Unicorn Engine в стиле dumpulator на захваченном снимке памяти.
