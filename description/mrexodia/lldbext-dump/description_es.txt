Extensión de Python para LLDB que crea archivos minidump de memoria completa a partir de una sesión de depuración Android en vivo, capturando todas las regiones de memoria mapeadas, contextos de hilos y listas de módulos en un formato minidump compatible con Windows. Incluye un script de emulación complementario para reproducir dumps con ejecución Unicorn Engine al estilo dumpulator sobre la instantánea de memoria capturada.
