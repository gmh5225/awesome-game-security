Extension Python pour LLDB qui crée des fichiers minidump mémoire complets à partir d'une session de débogage Android en direct, en capturant toutes les régions mémoire mappées, les contextes de threads et les listes de modules au format minidump compatible Windows. Inclut un script d'émulation associé pour rejouer les dumps avec une exécution Unicorn Engine de style dumpulator sur l'instantané mémoire capturé.
