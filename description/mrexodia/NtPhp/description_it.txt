Questo progetto è un repository elencato sotto anti cheat / dynamic script e incentrato su nt php.
È scritto principalmente in C/C++ e C e si concentra sul lavoro a livello kernel e sullo sviluppo di driver.
È particolarmente utile per gli ingegneri anti-cheat e i ricercatori di sicurezza difensiva che operano nell'area anti cheat / dynamic script.
