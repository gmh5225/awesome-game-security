Questo progetto è un framework di cheat esterno per Apex Legends pensato per flussi di accesso alla memoria basati su QEMU o KVM. Combina moduli di gameplay in C++ con componenti memflow in Rust e binding C FFI per leggere la memoria di gioco tramite connettori orientati alla virtualizzazione. Il codice include logica ESP e di mira, routine matematiche e di predizione, oltre a script di build per ambienti in stile Linux. È usato principalmente per studiare tecniche di game hacking assistite da DMA o VM e le relative superfici di rilevamento anti-cheat.
