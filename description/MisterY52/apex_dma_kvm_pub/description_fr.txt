Ce projet est un framework de cheat externe pour Apex Legends conçu pour des flux de travail d'accès mémoire basés sur QEMU ou KVM. Il combine des modules de gameplay en C++ avec des composants memflow en Rust et des liaisons C FFI pour lire la mémoire du jeu via des connecteurs orientés virtualisation. Le code inclut la logique ESP et de visée, des routines mathématiques et de prédiction, ainsi que des scripts de build pour des environnements de type Linux. Il sert principalement à étudier les techniques de game hacking assistées par DMA ou VM et les surfaces de détection anti-cheat associées.
