Dieses Projekt konzentriert sich auf einen PoC zum Patchen des Syscall-Dispatchers.
Dazu wird zunächst eine veränderbare/schreibbare Kopie der Systemaufruftabelle erstellt, die Funktionszeiger in dieser Tabelle mit Zeigern auf die Hook-Funktionen überschrieben und anschließend die ersten Bytes des Dispatchers so gepatcht, dass er zu einem eigenen Systemaufruf-Handler springt.
Es ist vor allem nützlich für Spiele-Sicherheitsforscher und Reverse Engineers, die offensive Techniken im Bereich Cheats / Android-Kernel-Exploration untersuchen.
