Ce projet fournit une configuration Clang-Format adaptée au style de codage C++ d’Unreal Engine.
Il inclut des conseils pour appliquer le fichier de formatage dans des éditeurs comme VS Code et pour activer un formatage cohérent au quotidien.
Le dépôt est léger et contient surtout de la configuration et de la documentation, centré sur les règles de formatage C++ plutôt que sur le code du moteur.
Son principal cas d’usage est de garder les bases de code Unreal alignées sur les conventions de style Epic au sein des équipes.
