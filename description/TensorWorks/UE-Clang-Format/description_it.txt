Questo progetto fornisce una configurazione Clang-Format adattata allo stile di codice C++ di Unreal Engine.
Include indicazioni per applicare il file di formato in editor come VS Code e per abilitare una formattazione coerente nello sviluppo quotidiano.
Il repository è leggero e contiene soprattutto configurazione e documentazione, incentrato sulle regole di formattazione C++ piuttosto che sul codice del motore.
Il caso d’uso principale è mantenere le codebase Unreal allineate alle convenzioni in stile Epic tra i team.
