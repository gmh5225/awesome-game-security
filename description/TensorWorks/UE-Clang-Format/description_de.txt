Dieses Projekt bietet eine Clang-Format-Konfiguration, die auf den C++-Coding-Stil von Unreal Engine abgestimmt ist.
Es enthält Hinweise zur Anwendung der Formatdatei in Editoren wie VS Code sowie zur konsistenten Formatierung im täglichen Entwicklungsalltag.
Das Repository ist schlank und besteht hauptsächlich aus Konfiguration und Dokumentation; im Mittelpunkt stehen C++-Formatierungsregeln, nicht Engine-Code.
Der Hauptanwendungsfall ist, Unreal-Codebasen teamübergreifend an Epic-ähnlichen Konventionen auszurichten.
