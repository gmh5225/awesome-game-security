Este proyecto ofrece una configuración de Clang-Format ajustada al estilo de código C++ de Unreal Engine.
Incluye orientación para aplicar el archivo de formato en editores como VS Code y para habilitar un formateo coherente en el desarrollo diario.
El repositorio es ligero y se centra principalmente en configuración y documentación, en reglas de formateo C++ más que en código del motor.
Su caso de uso principal es mantener las bases de código de Unreal alineadas con las convenciones de estilo de Epic entre equipos.
