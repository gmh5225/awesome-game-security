Этот проект предоставляет конфигурацию Clang-Format, настроенную под стиль кодирования C++ в Unreal Engine.
В него входят рекомендации по применению файла форматирования в редакторах вроде VS Code и по обеспечению единообразного форматирования в повседневной разработке.
Репозиторий лёгкий и в основном состоит из конфигурации и документации; акцент на правилах форматирования C++, а не на коде движка.
Основной сценарий использования — поддерживать кодовые базы Unreal в соответствии с конвенциями в стиле Epic в рамках команд.
