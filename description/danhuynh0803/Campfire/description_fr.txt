Résumé du projet : Campfire est un moteur de jeu en cours de développement, construit avec OpenGL et Vulkan. La plupart des fonctionnalités du moteur sont actuellement désactivées pendant la migration vers Vulkan, mais elles restent disponibles sur la branche OpenGL. Il est principalement utile aux développeurs de jeux, programmeurs de moteurs et chercheurs en graphisme travaillant dans le domaine des moteurs de jeu / du code source.
