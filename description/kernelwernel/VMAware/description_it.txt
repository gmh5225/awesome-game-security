Questo progetto è VMAware, una libreria C++ multipiattaforma per il rilevamento di macchine virtuali. Implementa oltre 100 tecniche di rilevamento che controllano le foglie CPUID degli hypervisor, artefatti VM noti in registro, filesystem e indirizzi MAC, side channel basati sul timing, impronte hardware e firme dei driver, per determinare se il codice è in esecuzione in VMware, VirtualBox, Hyper-V, QEMU/KVM o altri hypervisor. La libreria header-only fornisce una API semplice che restituisce punteggi di confidenza. È rivolta a sviluppatori anti-cheat, ricercatori di malware e ingegneri della sicurezza che implementano il rilevamento di VM o studiano tecniche di evasione delle VM.
