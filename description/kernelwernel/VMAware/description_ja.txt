本プロジェクトは VMAware で、仮想マシン検出向けのクロスプラットフォーム C++ ライブラリです。ハイパーバイザの CPUID リーフ、レジストリ／ファイルシステム／MAC アドレス上の既知の VM 痕跡、タイミングベースのサイドチャネル、ハードウェア指紋、ドライバ署名など 100 を超える検出手法を実装し、コードが VMware、VirtualBox、Hyper-V、QEMU/KVM、その他のハイパーバイザ上で動作しているかを判定します。ヘッダオンリーのライブラリで、信頼度スコアを返すシンプルな API を提供します。アンチチート開発者、マルウェア研究者、VM 検出の実装や VM 回避手法の研究を行うセキュリティエンジニアを対象としています。
