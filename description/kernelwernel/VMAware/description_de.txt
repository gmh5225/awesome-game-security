Dieses Projekt ist VMAware, eine plattformübergreifende C++-Bibliothek zur Erkennung von virtuellen Maschinen. Sie implementiert über 100 Erkennungstechniken, die Hypervisor-CPUID-Leaves, bekannte VM-Artefakte in Registry, Dateisystem und MAC-Adressen, timingbasierte Side Channels, Hardware-Fingerabdrücke und Treibersignaturen prüfen, um festzustellen, ob Code in VMware, VirtualBox, Hyper-V, QEMU/KVM oder anderen Hypervisoren läuft. Die Header-only-Bibliothek bietet eine einfache API, die Confidence-Scores zurückgibt. Sie richtet sich an Anti-Cheat-Entwickler, Malware-Forscher und Security-Ingenieure, die VM-Erkennung implementieren oder VM-Evasion-Techniken untersuchen.
