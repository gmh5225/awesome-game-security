Este proyecto es VMAware, una biblioteca C++ multiplataforma para la detección de máquinas virtuales. Implementa más de 100 técnicas de detección que comprueban hojas CPUID de hipervisores, artefactos conocidos de VM en el registro, el sistema de archivos y las direcciones MAC, canales laterales basados en temporización, huellas de hardware y firmas de controladores, para determinar si el código se ejecuta dentro de VMware, VirtualBox, Hyper-V, QEMU/KVM u otros hipervisores. La biblioteca header-only ofrece una API sencilla que devuelve puntuaciones de confianza. Está orientada a desarrolladores de anti-cheat, investigadores de malware e ingenieros de seguridad que implementan detección de VM o estudian técnicas de evasión de VM.
