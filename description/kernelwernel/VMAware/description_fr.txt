Ce projet est VMAware, une bibliothèque C++ multiplateforme dédiée à la détection de machines virtuelles. Elle met en œuvre plus de 100 techniques de détection qui examinent les feuilles CPUID des hyperviseurs, les artefacts VM connus dans le registre, le système de fichiers et les adresses MAC, les canaux auxiliaires basés sur le timing, les empreintes matérielles et les signatures de pilotes, afin de déterminer si le code s’exécute sous VMware, VirtualBox, Hyper-V, QEMU/KVM ou d’autres hyperviseurs. Cette bibliothèque header-only expose une API simple renvoyant des scores de confiance. Elle s’adresse aux développeurs d’anti-cheat, aux chercheurs en malware et aux ingénieurs sécurité qui implémentent la détection de VM ou étudient les techniques d’évasion de VM.
