Этот проект — VMAware, кроссплатформенная библиотека на C++ для обнаружения виртуальных машин. Она реализует более 100 методов детектирования, проверяя CPUID-листы гипервизоров, известные артефакты ВМ в реестре, файловой системе и MAC-адресах, тайминговые побочные каналы, аппаратные отпечатки и подписи драйверов, чтобы определить, выполняется ли код внутри VMware, VirtualBox, Hyper-V, QEMU/KVM или других гипервизоров. Header-only библиотека предоставляет простой API, возвращающий оценки достоверности. Она предназначена для разработчиков античита, исследователей вредоносного ПО и инженеров по безопасности, реализующих обнаружение ВМ или изучающих техники обхода детектирования ВМ.
