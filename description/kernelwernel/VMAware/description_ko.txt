이 프로젝트는 VMAware로, 가상 머신 탐지를 위한 크로스 플랫폼 C++ 라이브러리입니다. 하이퍼바이저 CPUID 리프, 레지스트리/파일시스템/MAC 주소의 알려진 VM 흔적, 타이밍 기반 사이드채널, 하드웨어 지문, 드라이버 서명 등 100가지가 넘는 탐지 기법을 구현하여 코드가 VMware, VirtualBox, Hyper-V, QEMU/KVM 또는 기타 하이퍼바이저 안에서 실행 중인지 판별합니다. 헤더 전용 라이브러리로 신뢰도 점수를 반환하는 간단한 API를 제공합니다. 안티치트 개발자, 악성코드 연구원, VM 탐지를 구현하거나 VM 우회 기법을 연구하는 보안 엔지니어를 대상으로 합니다.
