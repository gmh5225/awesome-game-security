unrealcpp è un'ampia raccolta di esempi tutorial Unreal Engine in C++, organizzati come progetti di lezione autonomi. Illustra pattern comuni di programmazione gameplay come movimento degli actor, comportamento della camera, utility matematiche, gestione dell'input e uso dei component attraverso molti piccoli campioni. Il repository è principalmente codice C++ rivolto ai workflow di Unreal Engine ed è pensato per essere letto e compilato come riferimento pratico. È utile per sviluppatori e ricercatori di sicurezza che necessitano di una conoscenza più approfondita degli interni di Unreal nell'analisi del comportamento di gioco.
