Este proyecto es un mapper en modo usuario que coloca una imagen de kernel personalizada en una sección de un controlador ya firmado. Usa primitivas de lectura y escritura de memoria física y cambios de permisos en las tablas de páginas para hacer ejecutable y escribible la sección seleccionada antes del mapeo. El código también incluye pasos de limpieza de rastros, como borrar artefactos comunes de contabilidad del kernel, y proporciona un formato de ejemplo de entrada del controlador. Su caso de uso principal es la investigación sobre el kernel de Windows y la evasión de anti-cheat centrada en técnicas sigilosas de carga de controladores.
