Этот проект — user-mode mapper, который размещает пользовательский образ ядра в секции уже подписанного драйвера. Он использует примитивы чтения и записи физической памяти, а также изменение прав доступа в таблицах страниц, чтобы сделать выбранную секцию исполняемой и доступной для записи перед маппингом. В коде также есть шаги очистки следов, например удаление типичных артефактов учёта ядра, и приведён пример формата точки входа драйвера. Основной сценарий использования — исследования ядра Windows и обхода античита с акцентом на скрытные техники загрузки драйверов.
