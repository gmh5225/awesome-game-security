Этот проект — инструмент исследования ядра Windows для анализа и перехвата зашифрованных импортов в драйвере Vanguard. Он реализует процедуры поиска зашифрованной таблицы импортов с помощью сканирования сигнатур, расшифровки целевых записей и повторного шифрования указателей при установке хуков. Примерный сценарий хука демонстрирует отслеживание вызовов вроде CiCheckSignedFile и восстановление смещений импортов из vgk.sys. Предназначен для реверс-инжиниринга и исследования внутренней работы античита, а не для обычной разработки приложений.
