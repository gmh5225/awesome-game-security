Questo progetto è uno strumento di ricerca per il kernel Windows per analizzare e intercettare gli import crittografati nel driver di Vanguard. Implementa routine per localizzare la tabella degli import crittografata con signature scanning, decrittare le voci di destinazione e ricrittografare i puntatori durante il patching degli hook. Il flusso di hook di esempio mostra il tracciamento di chiamate come CiCheckSignedFile e il recupero degli offset degli import da vgk.sys. È pensato per reverse engineering e ricerca sugli interni dell'anti-cheat, non per lo sviluppo di applicazioni generali.
