Este proyecto es una herramienta de investigación del kernel de Windows para analizar e interceptar imports cifrados en el controlador de Vanguard. Implementa rutinas para localizar la tabla de imports cifrada mediante signature scanning, descifrar las entradas objetivo y volver a cifrar punteros al aplicar hooks. El flujo de hook de ejemplo muestra el seguimiento de llamadas como CiCheckSignedFile y la recuperación de offsets de imports desde vgk.sys. Está pensado para ingeniería inversa e investigación de internos de anti-cheat, no para el desarrollo de aplicaciones generales.
