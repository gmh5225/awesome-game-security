本プロジェクトは、署名済みのヘルパードライバーを用いて未署名のペイロードドライバーをマップする Windows カーネルドライバーマッパーです。ユーザーモードとカーネルモードの構成要素を組み合わせ、カーネルフック経由でマップ済みイメージを転送し、アンロード済みドライバーやキャッシュ関連アーティファクトなど一般的なフォレンジック痕跡の消去を試みます。リポジトリにはマップ対象ドライバー向けのエントリ契約の例と、マッピング処理全体をビルド可能な Visual Studio プロジェクトが含まれます。ドライバー読み込みの挙動とステルス性のトレードオフを研究する低レベルなゲームセキュリティおよびカーネル研究向けです。
