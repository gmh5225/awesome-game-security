Este proyecto es un mapper de controladores del kernel de Windows que usa un controlador auxiliar firmado para mapear un controlador de carga útil sin firmar. Combina componentes en modo usuario y modo kernel, transfiere las imágenes mapeadas mediante hooks del kernel e intenta limpiar rastros forenses habituales, como artefactos de controladores descargados y de caché. El repositorio incluye un contrato de entrada de ejemplo para controladores mapeados y proyectos de Visual Studio compilables para el flujo completo de mapeo. Está orientado a la investigación de seguridad de juegos y del kernel a bajo nivel, donde se estudian el comportamiento de carga de controladores y las compensaciones de sigilo.
