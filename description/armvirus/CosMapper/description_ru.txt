Этот проект — маппер драйверов ядра Windows, который использует подписанный вспомогательный драйвер для отображения неподписанного драйвера полезной нагрузки. Он сочетает компоненты пользовательского режима и режима ядра, передаёт отображённые образы через хуки ядра и пытается очищать типичные криминалистические следы, такие как артефакты выгруженных драйверов и кэша. Репозиторий включает пример контракта точки входа для отображаемых драйверов и собираемые проекты Visual Studio для полного процесса маппинга. Он предназначен для низкоуровневых исследований игровой безопасности и ядра, где изучают поведение загрузки драйверов и компромиссы скрытности.
