Questo progetto è un'utilità Windows in user-mode che cerca moduli driver o DLL candidati con sezioni sovradimensionate in grado di ospitare un'altra immagine. Enumera i file in System32 o System32\drivers, analizza le intestazioni PE e confronta la dimensione di una sezione scelta con quella dell'immagine di destinazione. Per i target driver salta i driver attualmente caricati interrogando i servizi attivi, aiutando a restringere i risultati ai file sostituibili. Lo strumento è pensato per flussi di lavoro di ricerca Windows a basso livello e anti-cheat in cui i candidati al mapping basato su sezioni devono essere identificati rapidamente.
