Этот проект — пользовательская утилита Windows, которая ищет кандидаты среди драйверов или DLL с чрезмерно большими секциями, способными разместить другой образ. Она перечисляет файлы в System32 или System32\drivers, разбирает PE-заголовки и сравнивает размер выбранной секции с размером целевого образа. Для драйверных целей она пропускает уже загруженные драйверы, опрашивая активные службы, что помогает сузить результаты до заменяемых файлов. Инструмент предназначен для низкоуровневых исследований Windows и античита, где нужно быстро находить кандидатов для mapping на основе секций.
