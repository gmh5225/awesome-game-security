Ce projet est un utilitaire Windows en mode utilisateur qui recherche des modules pilotes ou DLL candidats dont les sections surdimensionnées peuvent héberger une autre image. Il énumère les fichiers dans System32 ou System32\drivers, analyse les en-têtes PE et compare la taille d'une section choisie à celle de l'image cible. Pour les cibles pilotes, il ignore les pilotes actuellement chargés en interrogeant les services actifs, ce qui aide à restreindre les résultats aux fichiers remplaçables. L'outil vise les flux de travail de recherche bas niveau Windows et anti-triche où les candidats au mapping par sections doivent être identifiés rapidement.
