이 프로젝트는 Unreal Engine 5 격투 게임으로, 루트 모션 이동, 전투 대기 전환, 방향별 이동 블렌딩을 포함한 종합적인 애니메이션 시스템을 갖추고 있습니다.
이동 방향과 전투 상태에 따라 정리된 걷기, 달리기, 점프, 공격, 피격 반응, 가드, 회피, 사망 애니메이션 시퀀스가 포함된 캐릭터용 Blueprint와 UAsset을 제공합니다.
UE5의 Animation Blueprint 시스템을 사용하며, 몽타주 기반 전투 기술, 입력 처리, 게임 모드 구성을 포함합니다.
격투 게임 애니메이션 구조와 Unreal Engine 5의 루트 모션 구현을 연구하는 게임 개발자와 UE5 연구자에게 유용합니다.
