VxLang は、静的・動的解析、ファイル改ざん、不正なメモリアクセスによるリバースエンジニアリングを防ぐためのバイナリ保護ツールキットです。ネイティブ Windows x86-64 の実行ファイル、DLL、カーネルドライバに対して難読化、コード仮想化、コード平坦化、デュアルモード保護を提供し、UEFI モジュールおよびベータ版の ELF コード平坦化もサポートします。C/C++ のマーカーとサンプル、ならびに Go と Rust の例を含む SDK により、開発者は SEH 対応の難読化を含め、機密性の高いコード領域をマークして保護できます。拡張モジュールにより、改ざん防止および .NET パッキング機能を追加できます。ゲームセキュリティやアンチチートを含む、リバースエンジニアリングへの耐性を高めたい開発者を対象としています。
