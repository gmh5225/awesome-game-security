KVM-GPU-Passthrough es un proyecto de configuración centrado en Arch Linux para construir un entorno de máquina virtual con GPU passthrough. Proporciona scripts de shell y plantillas de configuración para los pasos de configuración del host con GRUB, libvirt y QEMU. El proyecto documenta los requisitos previos de virtualización de hardware, como IOMMU, VT-d y los ajustes de BIOS relacionados para sistemas AMD e Intel. Está pensado para usuarios avanzados que construyen laboratorios aislados de juegos o investigación de seguridad con gráficos casi nativos en una VM invitada.
