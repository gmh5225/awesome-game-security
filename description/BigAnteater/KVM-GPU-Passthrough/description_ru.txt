KVM-GPU-Passthrough — проект настройки с упором на Arch Linux для создания среды виртуальной машины с GPU passthrough. Он предоставляет shell-скрипты и шаблоны конфигурации для шагов настройки хоста GRUB, libvirt и QEMU. В проекте описаны аппаратные предпосылки виртуализации, такие как IOMMU, VT-d и связанные настройки BIOS для систем AMD и Intel. Он предназначен для опытных пользователей, создающих изолированные игровые или исследовательские лаборатории безопасности с почти нативной графикой в гостевой ВМ.
