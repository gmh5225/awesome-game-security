KVM-GPU-Passthrough est un projet de configuration centré sur Arch Linux pour construire un environnement de machine virtuelle avec GPU passthrough. Il fournit des scripts shell et des modèles de configuration pour les étapes de configuration hôte GRUB, libvirt et QEMU. Le projet documente les prérequis de virtualisation matérielle tels que l'IOMMU, VT-d et les réglages BIOS associés pour les systèmes AMD et Intel. Il s'adresse aux utilisateurs avancés qui construisent des laboratoires de jeu ou de recherche en sécurité isolés avec des graphismes quasi natifs dans une VM invitée.
