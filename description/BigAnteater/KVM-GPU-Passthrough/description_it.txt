KVM-GPU-Passthrough è un progetto di setup incentrato su Arch Linux per costruire un ambiente di macchina virtuale con GPU passthrough. Fornisce script shell e modelli di configurazione per i passaggi di configurazione host di GRUB, libvirt e QEMU. Il progetto documenta i prerequisiti di virtualizzazione hardware come IOMMU, VT-d e le relative impostazioni BIOS per sistemi AMD e Intel. È destinato a utenti avanzati che costruiscono laboratori isolati di gaming o ricerca sulla sicurezza con grafica quasi nativa in una VM guest.
