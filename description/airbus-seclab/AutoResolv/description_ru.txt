AutoResolv — плагин для IDA, который разрешает вызовы внешних библиотек в ELF-бинарниках и сопоставляет обёртки с их реальными реализациями. Он написан на IDAPython и использует PyQt5 и pyelftools для интерактивного рабочего процесса и кэшированных данных анализа. Инструмент может аннотировать места вызовов, перечислять разрешённые пути к библиотекам и импортировать сигнатуры функций из связанных бинарников, повышая точность декомпиляции. Он предназначен для задач реверс-инжиниринга, где аналитикам нужно быстрее понимать динамически слинкованный код на нескольких архитектурах.
