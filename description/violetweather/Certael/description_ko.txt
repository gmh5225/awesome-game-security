Certael은 권위적 온라인 게임을 위한 오픈소스 크로스엔진 안티치트 기반입니다. 안전한 액션 프로토콜, 버전 관리된 C ABI를 갖춘 메모리 안전 Rust 네이티브 클라이언트 런타임, Godot·Unity·Unreal용 어댑터와 함께 .NET 서버 SDK 및 셀프호스팅 ASP.NET Core 컨트롤 플레인을 제공합니다. 세션은 단기·일회용 부트스트랩 티켓과 서명된 액션 인텐트를 사용해 클라이언트가 타입이 지정된 요청을 보내고, 신뢰할 수 있는 게임 서버만 규칙을 검증하고 상태를 커밋합니다. 이식 가능한 서명 규칙 팩, 설명 가능한 증거, 경제·관계 분석 워커, 선택적 에이전트 프로브로 장르별 휴리스틱 없이 인벤토리, 진행, 소셜 시스템, 경쟁 플레이를 지원합니다. 서버 권위형 멀티플레이어 보안과 안티치트 인프라를 셀프호스팅하고 여러 엔진에 통합하려는 게임 스튜디오를 대상으로 합니다.
