Certael è una fondamenta anti-cheat open source e multi-motore per giochi online autoritativi. Fornisce un protocollo di azioni sicuro, un runtime client nativo Rust memory-safe con ABI C versionata e adapter per Godot, Unity e Unreal, oltre a un SDK server .NET e un control plane ASP.NET Core self-hosted. Le sessioni usano ticket di bootstrap a breve durata e monouso e intent di azione firmati, così i client inviano richieste tipizzate mentre solo il server di gioco affidabile valida le regole e conferma lo stato. Pacchetti di regole firmati portabili, evidenza spiegabile, worker di analisi economica e delle relazioni e una sonda agent opzionale supportano inventari, progressione, sistemi sociali e gioco competitivo senza euristiche specifiche di genere. È pensato per gli studi di gioco che costruiscono sicurezza multiplayer server-authoritative e infrastruttura anti-cheat che possono self-hostare e integrare tra motori.
