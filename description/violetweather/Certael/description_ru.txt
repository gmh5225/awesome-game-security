Certael — это открытая кросс-движковая основа античита для авторитетных онлайн-игр. Она предоставляет защищённый протокол действий, memory-safe нативный клиентский runtime на Rust с версионированным C ABI и адаптеры для Godot, Unity и Unreal, а также .NET SDK для сервера и самостоятельно размещаемую control plane на ASP.NET Core. Сессии используют короткоживущие одноразовые bootstrap-билеты и подписанные намерения действий, поэтому клиенты отправляют типизированные запросы, а правила проверяет и состояние фиксирует только доверенный игровой сервер. Переносимые подписанные наборы правил, объяснимые доказательства, воркеры анализа экономики и отношений, а также опциональный agent-зонд поддерживают инвентари, прогрессию, социальные системы и соревновательную игру без жанрово-специфичных эвристик. Решение ориентировано на игровые студии, создающие серверно-авторитетную мультиплеерную безопасность и античит-инфраструктуру, которую можно размещать самостоятельно и интегрировать между движками.
