Certael 是面向权威在线游戏的开源、跨引擎反作弊基础框架。它提供安全的动作协议、带版本化 C ABI 的内存安全 Rust 原生客户端运行时，以及面向 Godot、Unity 和 Unreal 的适配器，另有 .NET 服务端 SDK 与可自托管的 ASP.NET Core 控制平面。会话使用短时、一次性引导票据和签名动作意图，使客户端发送类型化请求，而仅由可信游戏服务器校验规则并提交状态。可移植的签名规则包、可解释证据、经济与关系分析工作进程以及可选的 agent 探针，可在无需特定类型启发式的情况下支持物品栏、成长、社交系统与竞技对战。它面向希望构建可自托管、可跨引擎集成的服务器权威多人安全与反作弊基础设施的游戏工作室。
