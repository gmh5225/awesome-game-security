Certael 是面向權威線上遊戲的開源、跨引擎反作弊基礎框架。它提供安全的動作協定、具備版本化 C ABI 的記憶體安全 Rust 原生用戶端執行階段，以及適用於 Godot、Unity 與 Unreal 的轉接器，另有 .NET 伺服器 SDK 與可自架的 ASP.NET Core 控制平面。工作階段使用短時、一次性引導票證與簽名動作意圖，讓用戶端傳送型別化請求，而僅由可信遊戲伺服器驗證規則並提交狀態。可攜式簽名規則包、可解釋證據、經濟與關係分析工作程序以及可選的 agent 探針，可在無需特定類型啟發式的情況下支援物品欄、進度、社交系統與競技對戰。它面向希望打造可自架、可跨引擎整合的伺服器權威多人安全與反作弊基礎設施的遊戲工作室。
