Certael は、権威あるオンラインゲーム向けのオープンソースでエンジン横断のアンチチート基盤です。セキュアなアクションプロトコル、バージョン付き C ABI を備えたメモリ安全な Rust ネイティブクライアントランタイム、Godot・Unity・Unreal 向けアダプタ、さらに .NET サーバー SDK とセルフホスト可能な ASP.NET Core コントロールプレーンを提供します。セッションは短命の使い捨てブートストラップチケットと署名済みアクションインテントを用い、クライアントは型付きリクエストを送信し、信頼できるゲームサーバーのみがルールを検証して状態を確定します。ポータブルな署名済みルールパック、説明可能な証跡、経済・関係分析ワーカー、オプションのエージェントプローブにより、ジャンル固有のヒューリスティックなしでインベントリ、進行、ソーシャルシステム、競技プレイを支援します。サーバー権威型マルチプレイヤーのセキュリティとアンチチート基盤をセルフホストし、複数エンジンに統合したいゲームスタジオ向けです。
