Este proyecto es un paquete de código fuente de cheat para PUBG Mobile orientado a una versión concreta del juego y a un flujo de herramientas externas. Está escrito principalmente en C++ e incluye componentes habituales de cheat como apuntado, procesamiento de entidades y lógica de renderizado en pantalla. El repositorio también incluye artefactos de controlador y scripts auxiliares para operaciones de memoria privilegiada en tiempo de ejecución. Es relevante sobre todo para ingeniería inversa e investigación de anti-cheat sobre implementaciones de cheats en shooters móviles.
