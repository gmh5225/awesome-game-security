Questo progetto è un pacchetto sorgente di cheat per PUBG Mobile mirato a una versione specifica del gioco e a un flusso di tooling esterno. È scritto principalmente in C++ e include componenti tipici di cheat come aiming, elaborazione delle entità e logica di rendering a schermo. Il repository include anche artefatti di driver e script di supporto per operazioni di memoria privilegiata a runtime. È rilevante soprattutto per reverse engineering e ricerca anti-cheat sulle implementazioni di cheat nei shooter mobile.
