Este proyecto es un framework C++ de reescritura y ofuscación de binarios para código x86. Ofrece reubicación de funciones, parcheo de referencias globales, transformaciones de ofuscación complejas y manejo de ramas, con ejemplos construidos con CMake que demuestran diversas técnicas de manipulación de código. Resulta especialmente útil para investigadores de protección binaria e ingenieros inversos que estudian la ofuscación de código x86 y las metodologías de reescritura binaria.
