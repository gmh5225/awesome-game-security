Rust-Auto-Weapon-Detection-OpenCV-Example ist ein Beispielprojekt, das die automatische Waffenerkennung für das Spiel Rust mit OpenCV demonstriert.
Es ist in C++ geschrieben und enthält Visual-Studio-Projektdateien, Bildassets sowie Code für Farbfilterung und Waffenindex-Zuordnung.
Die Implementierung zeigt eine grundlegende Computer-Vision-Pipeline zur bildschirmbasierten Zustandserkennung statt direkter Manipulation des Spielspeichers.
Der Hauptanwendungsfall ist die bildungsorientierte Experimentierung mit CV-gestützter Spielautomatisierung und Telemetrie-Werkzeugen.
