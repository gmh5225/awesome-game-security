Rust-Auto-Weapon-Detection-OpenCV-Example est un projet d'exemple qui démontre la détection automatique d'armes pour le jeu Rust à l'aide d'OpenCV.
Il est écrit en C++ et comprend des fichiers de projet Visual Studio, des ressources d'images, ainsi que du code pour le filtrage des couleurs et le mappage des index d'armes.
L'implémentation présente un pipeline de vision par ordinateur de base pour la reconnaissance d'état basée sur l'écran, plutôt qu'une manipulation directe de la mémoire du jeu.
Son cas d'usage principal est l'expérimentation éducative autour de l'automatisation de jeux assistée par CV et des outils de télémétrie.
