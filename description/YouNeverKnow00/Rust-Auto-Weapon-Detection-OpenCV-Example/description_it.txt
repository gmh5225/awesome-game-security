Rust-Auto-Weapon-Detection-OpenCV-Example è un progetto di esempio che dimostra il rilevamento automatico delle armi per il gioco Rust tramite OpenCV.
È scritto in C++ e include file di progetto Visual Studio, risorse immagine e codice per il filtraggio dei colori e la mappatura degli indici delle armi.
L'implementazione mostra una pipeline di computer vision di base per il riconoscimento dello stato basato sullo schermo, anziché la manipolazione diretta della memoria di gioco.
Il caso d'uso principale è la sperimentazione educativa con l'automazione di gioco assistita da CV e gli strumenti di telemetria.
