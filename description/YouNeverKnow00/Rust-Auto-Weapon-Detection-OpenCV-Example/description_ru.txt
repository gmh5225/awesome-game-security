Rust-Auto-Weapon-Detection-OpenCV-Example — пример проекта, демонстрирующий автоматическое распознавание оружия для игры Rust с помощью OpenCV.
Он написан на C++ и включает файлы проекта Visual Studio, графические ресурсы, а также код для цветовой фильтрации и сопоставления индексов оружия.
Реализация показывает базовый конвейер компьютерного зрения для распознавания состояния по экрану, а не прямую работу с памятью игры.
Основное назначение — образовательные эксперименты с CV-assisted автоматизацией игр и средствами телеметрии.
