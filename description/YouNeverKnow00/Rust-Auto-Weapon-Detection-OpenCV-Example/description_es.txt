Rust-Auto-Weapon-Detection-OpenCV-Example es un proyecto de ejemplo que demuestra la detección automática de armas para el juego Rust mediante OpenCV.
Está escrito en C++ e incluye archivos de proyecto de Visual Studio, recursos de imagen y código para filtrado de color y mapeo de índices de armas.
La implementación muestra un pipeline básico de visión por computadora para el reconocimiento de estado basado en pantalla, en lugar de manipular directamente la memoria del juego.
Su caso de uso principal es la experimentación educativa con automatización de juegos asistida por CV y herramientas de telemetría.
