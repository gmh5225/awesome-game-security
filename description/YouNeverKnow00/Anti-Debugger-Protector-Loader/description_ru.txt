Anti-Debugger-Protector-Loader — пример проекта на Windows C++ по антиотладке и защите.
Он непрерывно сканирует исполняемые файлы отладчиков, заголовки окон и связанные драйверы, а также включает несколько проверок обнаружения отладчика с опциональным автоматическим завершением.
Реализация предусматривает настраиваемые интервалы сканирования и необязательные защитные действия, а также артефакты интеграции VMProtect SDK.
Основная аудитория — исследователи программной защиты и игровой безопасности, изучающие практические схемы антиотладки.
