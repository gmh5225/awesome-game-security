Ce projet est une preuve de concept en mode noyau et utilisateur Windows qui communique via des callbacks du registre. Il montre comment enregistrer un callback au moyen d'un jump gadget dans un module légitime afin qu'un pilote mappé manuellement puisse recevoir des commandes. Le code C et C++ implémente des opérations telles que la lecture et l'écriture de mémoire virtuelle, le patch de mémoire protégée, la recherche de base de processus et les vérifications de heartbeat. Il sert principalement de référence pour la recherche sur la communication noyau et l'évasion anti-cheat dans des environnements contrôlés.
