Este proyecto es una prueba de concepto de Windows en modo kernel y usuario que se comunica mediante callbacks del registro. Demuestra cómo registrar un callback a través de un jump gadget en un módulo legítimo para que un controlador mapeado manualmente pueda recibir comandos. El código en C y C++ implementa operaciones como lectura y escritura de memoria virtual, parcheo de memoria protegida, búsqueda de la base de procesos y comprobaciones de heartbeat. Es principalmente una referencia para la investigación sobre comunicación en el kernel y evasión de anti-cheat en entornos controlados.
