Этот проект — концептуальное доказательство для Windows в режиме ядра и пользовательском режиме, которое обменивается данными через registry callbacks. Он демонстрирует регистрацию callback через jump gadget в легитимном модуле, чтобы вручную отображённый драйвер мог принимать команды. Код на C и C++ реализует такие операции, как чтение и запись виртуальной памяти, патчинг защищённой памяти, поиск базы процесса и проверки heartbeat. В первую очередь это справочный материал для исследований коммуникации с ядром и обхода античита в контролируемых средах.
