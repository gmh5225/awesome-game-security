Este proyecto es un repositorio listado bajo anti cheat / detection:hwid y se centra en OpenHardwareMonitor.
Está escrito principalmente en C# y JavaScript y se enfoca en trabajo a nivel de kernel, desarrollo de controladores y renderizado.
Es sobre todo útil para ingenieros anti-trampas e investigadores de seguridad defensiva que trabajan en el área anti cheat / detection:hwid.
