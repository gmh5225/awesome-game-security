Questo progetto è una raccolta di binari di driver kernel Windows raccolti per la ricerca su driver vulnerabili e BYOVD. Contiene principalmente file .sys compilati di diversi vendor, insieme a collegamenti di riferimento a CVE, report e analisi di exploitation. Il repository funge da set pratico di campioni per testare regole di detection, pipeline di threat hunting e controlli difensivi contro l'abuso di driver firmati. Il pubblico principale è costituito da ricercatori di sicurezza e difensori che studiano le superfici di attacco a livello kernel, piuttosto che lo sviluppo applicativo.
