Этот проект — комплексный инструмент генерации событий ETW (Event Tracing for Windows), предназначенный для тестирования и исследований.
BamboozlEDR имеет интерфейс TUI и может создавать реалистичные события безопасности через несколько поставщиков Windows ETW для проверки возможностей обнаружения EDR, систем анализа журналов и решений мониторинга безопасности.
Он в первую очередь полезен инженерам античита и исследователям защитной безопасности, работающим в области античита / информационных систем и форензики.
