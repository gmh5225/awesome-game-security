Этот проект — сообщественная база знаний с руководствами по разработке и практическими советами для Unreal Engine 4. Большая часть материалов написана в Markdown и организована по темам: создание сцен, конвейеры оптимизации, материалы, освещение и трассировка лучей. Также рассматриваются продвинутые области, такие как Chaos Physics, и рекомендации по переходу с Blueprints на C++. Основное назначение — помогать разработчикам Unreal улучшать производственные рабочие процессы и производительность на разных этапах проекта.
