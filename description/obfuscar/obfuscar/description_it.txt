Obfuscar è un obfuscator open source e minimalista per assembly .NET che aiuta a proteggere codice e segreti dalla reverse engineering occasionale. Scritto principalmente in C#, rinomina metodi, proprietà, eventi, campi, tipi e namespace mediante overloading massivo, così che gli identificatori si riducano a un insieme ristretto distinguibile soprattutto per firma. Supporta anche funzionalità come string hiding, regole selettive di skip/filter, gestione legata al BAML per le risorse WPF e packaging sia come strumento NuGet sia come .NET global tool. Basato su API di metadati e PE a basso livello, si rivolge agli sviluppatori .NET e Mono che necessitano di un’offuscazione di base degli assembly per distribuire binari protetti.
