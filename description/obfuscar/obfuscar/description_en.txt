Obfuscar is an open-source, minimalistic obfuscator for .NET assemblies that helps protect code and secrets from casual reverse engineering. Written primarily in C#, it renames methods, properties, events, fields, types, and namespaces using massive overloading so identifiers collapse to a small set distinguishable mainly by signature. It also supports features such as string hiding, selective skip/filter rules, BAML-related handling for WPF resources, and packaging as both a NuGet tool and a .NET global tool. Built on low-level metadata and PE APIs, it targets .NET and Mono developers who need basic assembly obfuscation for shipping protected binaries.
