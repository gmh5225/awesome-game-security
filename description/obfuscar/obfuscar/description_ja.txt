Obfuscar は、.NET アセンブリ向けのオープンソースでミニマルな難読化ツールであり、カジュアルなリバースエンジニアリングからコードや秘密情報を守るのに役立ちます。主に C# で書かれ、メソッド・プロパティ・イベント・フィールド・型・名前空間を大量のオーバーロードでリネームし、識別子を主にシグネチャで区別できる小さな集合へ収束させます。文字列隠蔽、選択的なスキップ／フィルタ規則、WPF リソース向けの BAML 関連処理に加え、NuGet ツールと .NET グローバルツールの両方としてのパッケージングもサポートします。低レベルなメタデータおよび PE API 上に構築され、保護されたバイナリを出荷するための基本的なアセンブリ難読化を必要とする .NET および Mono 開発者を対象としています。
