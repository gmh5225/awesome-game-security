Este proyecto es Lumos, un motor de juegos C++ de código abierto con un editor multiplataforma. Ofrece renderizado Vulkan/OpenGL con materiales PBR, un grafo de escena con arquitectura ECS, física (Box2D/Bullet), audio, scripting Lua, soporte 2D/3D y una interfaz de editor basada en Dear ImGui. El motor incluye pipelines de assets, serialización y soporte de builds multiplataforma. Está orientado a desarrolladores indie y entusiastas de motores que crean juegos 2D/3D con un motor de código abierto accesible.
