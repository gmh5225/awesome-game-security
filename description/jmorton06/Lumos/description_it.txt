Questo progetto è Lumos, un motore di gioco C++ open source con un editor multipiattaforma. Offre rendering Vulkan/OpenGL con materiali PBR, un grafo di scena con architettura ECS, fisica (Box2D/Bullet), audio, scripting Lua, supporto 2D/3D e un'interfaccia editor basata su Dear ImGui. Il motore include pipeline di asset, serializzazione e supporto per build multipiattaforma. È rivolto a sviluppatori indie e appassionati di motori che creano giochi 2D/3D con un motore open source accessibile.
