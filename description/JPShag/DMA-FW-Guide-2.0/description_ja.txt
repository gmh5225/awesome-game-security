本プロジェクトは、FPGA プラットフォーム上でカスタム PCIe DMA デバイスエミュレーションを構築するための包括的なファームウェア開発ガイドです。
ドナーデバイスのプロファイリング、PCIe 構成のクローン、BAR および割り込み処理、ファームウェア書き込みなどのワークフロー手順を文書化しています。
リポジトリには、PCILeech スタイルのハードウェアスタックおよび Vivado ベースの開発に関するツール参照と多言語資料も含まれています。
主な対象読者は、DMA 攻撃シミュレーションと低レベル PCIe 挙動を研究するハードウェアセキュリティおよびゲームアンチチート研究者です。
