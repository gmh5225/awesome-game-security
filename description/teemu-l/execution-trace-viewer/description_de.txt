Dieses Projekt ist eine Anwendung zum Anzeigen, Bearbeiten und Analysieren von Execution Traces. Sie wurde ursprünglich für das Reverse Engineering von obfuscated Code entwickelt, kann aber zur Analyse beliebiger Execution Traces verwendet werden. Sie ist vor allem für Game-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Cheat-/Debugging-Bereich untersuchen.
