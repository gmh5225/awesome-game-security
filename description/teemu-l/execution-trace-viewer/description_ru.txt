Этот проект — приложение для просмотра, редактирования и анализа трасс выполнения. Изначально оно создавалось для реверс-инжиниринга обфусцированного кода, но подходит для анализа любых трасс выполнения. В первую очередь оно полезно исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области читов и отладки.
