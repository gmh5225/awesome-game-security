Ce projet est une application destinée à visualiser, éditer et analyser des traces d'exécution. Elle a été conçue à l'origine pour le reverse engineering de code obfusqué, mais peut servir à analyser tout type de trace d'exécution. Elle est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats et du débogage.
