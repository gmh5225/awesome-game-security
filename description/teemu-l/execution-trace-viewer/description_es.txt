Este proyecto es una aplicación para visualizar, editar y analizar trazas de ejecución. Se creó originalmente para el reverse engineering de código ofuscado, pero puede usarse para analizar cualquier tipo de traza de ejecución. Resulta especialmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de los cheats y la depuración.
