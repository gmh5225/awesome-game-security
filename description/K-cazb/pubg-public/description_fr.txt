Ce projet est une base de code C++ d’assistance externe au jeu, centrée sur PUBG.
Elle comprend l’analyse d’entités pilotée par offsets, les calculs d’os et de caméra, le rendu ESP et la logique d’automatisation de la visée.
La base de code contient aussi des aides au déchiffrement, un accès mémoire assisté par pilote et des composants d’overlay DirectX.
Elle sert principalement de référence pédagogique pour les méthodes de game hacking et l’analyse anti-cheat.
