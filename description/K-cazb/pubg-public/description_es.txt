Este proyecto es una base de código C++ de asistencia externa al juego centrada en PUBG.
Incluye análisis de entidades impulsado por offsets, matemáticas de huesos y cámara, renderizado ESP y lógica de automatización de puntería.
La base de código también contiene ayudas de descifrado, acceso a memoria asistido por controlador y componentes de overlay DirectX.
Se usa principalmente como referencia educativa sobre métodos de game hacking y análisis anti-cheat.
