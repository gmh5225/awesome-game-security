Этот проект — кодовая база на C++ для внешней игровой помощи с акцентом на PUBG.
Она включает разбор сущностей по смещениям, математику костей и камеры, отрисовку ESP и логику автоматизации прицеливания.
Также содержит вспомогательные средства расшифровки, доступ к памяти через драйвер и компоненты DirectX-оверлея.
В основном используется как учебный справочник по методам game hacking и анализу античита.
