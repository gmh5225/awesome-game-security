Этот проект — WinDivert, библиотека захвата и перенаправления пакетов для Windows, позволяющая приложениям в пользовательском режиме перехватывать, изменять, отбрасывать или внедрять сетевые пакеты в реальном времени. Она работает через драйвер ядра, который подключается к сетевому стеку Windows с помощью WFP (Windows Filtering Platform) и поддерживает уровневую фильтрацию по IP-адресам, портам, протоколам и направлению пакетов. Библиотека на C предоставляет возможности как сниффинга, так и активной манипуляции пакетами. Она предназначена для разработчиков сетевой безопасности, создающих межсетевые экраны, решения NAT, анализаторы пакетов и средства туннелирования сети в Windows.
