Este proyecto es WinDivert, una biblioteca de captura y desviación de paquetes para Windows que permite a las aplicaciones en modo usuario interceptar, modificar, descartar o inyectar paquetes de red en tiempo real. Opera mediante un controlador en modo kernel que se engancha a la pila de red de Windows usando WFP (Windows Filtering Platform), con filtrado por capas según direcciones IP, puertos, protocolos y dirección de los paquetes. La biblioteca C ofrece capacidades de sniffing y de manipulación activa de paquetes. Está orientada a desarrolladores de seguridad de redes que construyen firewalls, soluciones NAT, analizadores de paquetes y herramientas de tunelización de red en Windows.
