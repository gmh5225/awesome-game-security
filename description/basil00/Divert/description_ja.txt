本プロジェクトは WinDivert であり、ユーザーモードのアプリケーションがネットワークパケットをリアルタイムで傍受、改変、破棄、または注入できる Windows 向けパケットキャプチャ／ダイバージョンライブラリです。カーネルドライバを通じて WFP（Windows Filtering Platform）により Windows ネットワークスタックにフックし、IP アドレス、ポート、プロトコル、パケット方向によるレイヤベースのフィルタリングをサポートします。C ライブラリはスニッフィングと能動的なパケット操作の両方を提供し、Windows 上でファイアウォール、NAT、パケットアナライザ、ネットワークトンネリングツールを構築するネットワークセキュリティ開発者を対象としています。
