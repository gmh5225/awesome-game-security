이 프로젝트는 WinDivert로, 사용자 모드 애플리케이션이 네트워크 패킷을 실시간으로 가로채고, 수정하고, 버리거나 주입할 수 있게 하는 Windows 패킷 캡처 및 전환 라이브러리입니다. 커널 드라이버를 통해 WFP(Windows Filtering Platform)로 Windows 네트워크 스택에 후킹하며, IP 주소, 포트, 프로토콜, 패킷 방향에 따른 계층 기반 필터링을 지원합니다. C 라이브러리는 스니핑과 능동적 패킷 조작 기능을 모두 제공하며, Windows에서 방화벽, NAT, 패킷 분석기, 네트워크 터널링 도구를 구축하는 네트워크 보안 개발자를 대상으로 합니다.
