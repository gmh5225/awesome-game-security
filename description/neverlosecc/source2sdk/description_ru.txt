Этот репозиторий содержит сгенерированные Source 2 SDK для различных игр.
Он в основном написан на C++ и сосредоточен на разработке драйверов, работе с шейдерами и рендеринге.
Он прежде всего полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game engine explorer:source.
