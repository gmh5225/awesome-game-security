Dieses Repository enthält die generierten Source-2-SDKs für verschiedene Spiele.
Es ist überwiegend in C++ geschrieben und konzentriert sich auf Treiberentwicklung, Shader-Arbeit und Rendering.
Es ist vor allem für Spielsicherheitsforscher und Reverse Engineers nützlich, die offensive Techniken im Bereich cheat / game engine explorer:source untersuchen.
