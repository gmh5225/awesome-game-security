Este proyecto se centra en una biblioteca de ofuscación de variables en tiempo de compilación, solo de cabeceras, para C++20 y posteriores.
El descifrado usa instrucciones SIMD (AVX2/SSE2/NEON) en tiempo de ejecución, lo que complica considerablemente el análisis estático.
Es especialmente útil para ingenieros anti-cheat e investigadores de seguridad defensiva que trabajan en el área de anti-cheat / cifrado de variables.
