Ghidra MCP 是一個 Model Context Protocol 伺服器，將 Ghidra 逆向工程與 AI 程式設計代理橋接起來，使其能夠以程式化方式分析並註解二進位檔案。它結合了 Java Ghidra 擴充功能（GUI 外掛與無頭 HTTP 伺服器）以及 Python MCP 橋接層，暴露大量工具介面，涵蓋反編譯、函式與資料型別編輯、註解、交叉參考及相關逆向工作流程。顯著能力包括 BSim 與可選知識庫整合、與 GUI 對等的無頭模式、可串流 HTTP 或 stdio 傳輸，以及面向 Windows 的除錯器路徑以支援即時分析。主要語言為 Java 與 Python，支援 Maven 或 Gradle 建置，並提供 Docker 封裝以便部署。面向希望在 Ghidra 內進行 AI 輔助二進位文件與分析、而非僅依賴手動流程的逆向工程師與遊戲安全研究人員。
