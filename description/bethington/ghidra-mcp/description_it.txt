Ghidra MCP è un server Model Context Protocol che collega il reverse engineering di Ghidra con gli agenti di coding IA, così da analizzare e annotare i binari in modo programmatico. Combina un’estensione Ghidra in Java (plugin GUI e server HTTP headless) con un bridge MCP in Python che espone un’ampia superficie di strumenti per decompilazione, modifica di funzioni e tipi di dati, commenti, riferimenti incrociati e relativi flussi di lavoro RE. Tra le capacità rilevanti figurano BSim e l’integrazione opzionale con una knowledge database, parità headless con la GUI, trasporti HTTP streamable o stdio, e un percorso debugger orientato a Windows per l’analisi live. I linguaggi principali sono Java e Python, con build Maven o Gradle e packaging Docker per il deployment. È rivolto a reverse engineer e ricercatori di game security che vogliono documentazione e analisi dei binari assistite dall’IA dentro Ghidra, invece di flussi di lavoro solo manuali.
