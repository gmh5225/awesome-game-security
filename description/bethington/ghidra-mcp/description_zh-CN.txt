Ghidra MCP 是一个 Model Context Protocol 服务器，将 Ghidra 逆向工程与 AI 编程代理连接起来，使其能够以编程方式分析并标注二进制文件。它结合了 Java Ghidra 扩展（GUI 插件与无头 HTTP 服务器）以及 Python MCP 桥接层，暴露大量工具接口，覆盖反编译、函数与数据类型编辑、注释、交叉引用及相关逆向工作流。显著能力包括 BSim 与可选知识库集成、与 GUI 对等的无头模式、可流式 HTTP 或 stdio 传输，以及面向 Windows 的调试器路径以支持实时分析。主要语言为 Java 和 Python，支持 Maven 或 Gradle 构建，并提供 Docker 打包以便部署。面向希望在 Ghidra 内进行 AI 辅助二进制文档与分析、而非仅依赖手工流程的逆向工程师与游戏安全研究人员。
