Ghidra MCP — это сервер Model Context Protocol, который связывает реверс-инжиниринг в Ghidra с ИИ-агентами для программирования, чтобы они могли программно анализировать и аннотировать бинарные файлы. Он сочетает Java-расширение Ghidra (GUI-плагин и headless HTTP-сервер) с Python-мостом MCP, который предоставляет широкую поверхность инструментов для декомпиляции, редактирования функций и типов данных, комментариев, перекрёстных ссылок и связанных RE-процессов. Среди заметных возможностей — BSim и опциональная интеграция с базой знаний, паритет headless-режима с GUI, потоковые HTTP- или stdio-транспорты, а также ориентированный на Windows путь отладчика для живого анализа. Основные языки — Java и Python; сборка через Maven или Gradle и упаковка в Docker для развёртывания. Предназначен для реверс-инженеров и исследователей безопасности игр, которым нужна ИИ-поддержка документирования и анализа бинарников внутри Ghidra вместо исключительно ручных рабочих процессов.
