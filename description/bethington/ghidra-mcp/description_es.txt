Ghidra MCP es un servidor Model Context Protocol que conecta la ingeniería inversa de Ghidra con agentes de codificación de IA para que puedan analizar y anotar binarios de forma programática. Combina una extensión de Ghidra en Java (complemento GUI y servidor HTTP headless) con un puente MCP en Python que expone una amplia superficie de herramientas para descompilación, edición de funciones y tipos de datos, comentarios, referencias cruzadas y flujos de trabajo de RE relacionados. Entre sus capacidades destacadas están BSim e integración opcional con una base de conocimiento, paridad headless con la GUI, transportes HTTP streamable o stdio, y una ruta de depurador orientada a Windows para análisis en vivo. Los lenguajes principales son Java y Python, con compilaciones Maven o Gradle y empaquetado Docker para el despliegue. Está pensado para ingenieros inversos e investigadores de seguridad en juegos que quieren documentación y análisis de binarios asistidos por IA dentro de Ghidra en lugar de flujos solo manuales.
