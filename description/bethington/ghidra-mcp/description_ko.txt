Ghidra MCP는 Ghidra 리버스 엔지니어링과 AI 코딩 에이전트를 연결하는 Model Context Protocol 서버로, 바이너리를 프로그래밍 방식으로 분석하고 주석을 달 수 있게 합니다. Java Ghidra 확장(GUI 플러그인 및 헤드리스 HTTP 서버)과 Python MCP 브리지를 결합하여 디컴파일, 함수 및 데이터 타입 편집, 주석, 교차 참조, 관련 RE 워크플로를 위한 넓은 도구 표면을 제공합니다. 주요 기능으로는 BSim 및 선택적 지식 데이터베이스 통합, GUI와 대등한 헤드리스 지원, 스트리밍 가능한 HTTP 또는 stdio 전송, 라이브 분석을 위한 Windows 중심 디버거 경로가 있습니다. 주 언어는 Java와 Python이며, Maven 또는 Gradle 빌드와 Docker 패키징으로 배포할 수 있습니다. 수동 작업만 하는 대신 Ghidra 안에서 AI 지원 바이너리 문서화와 분석을 원하는 리버스 엔지니어와 게임 보안 연구자를 대상으로 합니다.
