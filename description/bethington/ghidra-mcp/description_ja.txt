Ghidra MCP は、Ghidra によるリバースエンジニアリングと AI コーディングエージェントをつなぐ Model Context Protocol サーバーであり、バイナリの分析と注釈をプログラムから行えるようにします。Java 製の Ghidra 拡張（GUI プラグインとヘッドレス HTTP サーバー）と、Python 製の MCP ブリッジを組み合わせ、逆コンパイル、関数およびデータ型の編集、コメント、相互参照、関連する RE ワークフロー向けの豊富なツール面を公開します。主な機能には BSim と任意の知識データベース連携、GUI と同等のヘッドレス運用、ストリーミング可能な HTTP または stdio トランスポート、ライブ解析向けの Windows 中心のデバッガ経路が含まれます。主言語は Java と Python で、Maven または Gradle によるビルドと Docker パッケージングでデプロイできます。Ghidra 内での AI 支援によるバイナリ文書化・分析を、手作業だけの流れではなく進めたいリバースエンジニアやゲームセキュリティ研究者向けです。
