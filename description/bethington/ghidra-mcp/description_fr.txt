Ghidra MCP est un serveur Model Context Protocol qui relie l’ingénierie inverse Ghidra aux agents de codage IA afin qu’ils puissent analyser et annoter des binaires de façon programmatique. Il combine une extension Ghidra en Java (plugin GUI et serveur HTTP headless) avec un pont MCP en Python qui expose une large surface d’outils pour la décompilation, l’édition de fonctions et de types de données, les commentaires, les références croisées et les flux de travail RE associés. Parmi ses capacités notables figurent BSim et une intégration optionnelle de base de connaissances, une parité headless avec l’interface graphique, des transports HTTP streamable ou stdio, et un chemin débogueur orienté Windows pour l’analyse en direct. Les langages principaux sont Java et Python, avec des builds Maven ou Gradle et un packaging Docker pour le déploiement. Il s’adresse aux reverse engineers et aux chercheurs en sécurité des jeux qui veulent une documentation et une analyse de binaires assistées par l’IA dans Ghidra plutôt que des workflows purement manuels.
