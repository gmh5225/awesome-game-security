Ghidra MCP is a Model Context Protocol server that bridges Ghidra reverse engineering with AI coding agents so they can analyze and annotate binaries programmatically. It combines a Java Ghidra extension (GUI plugin and headless HTTP server) with a Python MCP bridge that exposes a large tool surface for decompilation, function and data-type editing, comments, cross-references, and related RE workflows. Notable capabilities include BSim and optional knowledge-database integration, headless parity with the GUI, streamable HTTP or stdio transports, and a Windows-oriented debugger path for live analysis. Primary languages are Java and Python, with Maven or Gradle builds and Docker packaging for deployment. It is aimed at reverse engineers and game-security researchers who want AI-assisted binary documentation and analysis inside Ghidra rather than manual-only workflows.
