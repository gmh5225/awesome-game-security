Этот проект — лёгкая Linux-база хуков для исследования интерфейсов CS:GO. Он реализован на C++ и даёт минимальный каркас для загрузки shared object и взаимодействия с игровыми интерфейсами. Репозиторий сосредоточен на структуре и настройке, а не на полноценном многофункциональном чит-фреймворке. Основной сценарий использования — обучающие эксперименты с внутренностями игр под Linux и разработкой хуков.
