Este proyecto es una base ligera de hooks en Linux para la investigación de interfaces de CS:GO. Está implementado en C++ y ofrece un andamiaje mínimo para cargar un objeto compartido e interactuar con las interfaces del juego. El repositorio se centra en la estructura y la configuración, no en un framework de cheats completo y con muchas funciones. Su caso de uso principal es la experimentación educativa con los internos de juegos en Linux y el desarrollo de hooks.
