Ce projet est une base de hook Linux légère pour la recherche sur les interfaces de CS:GO. Il est implémenté en C++ et fournit une structure minimale pour charger un objet partagé et interagir avec les interfaces du jeu. Le dépôt se concentre sur la structure et la mise en place plutôt que sur un framework de cheat complet et riche en fonctionnalités. Son principal cas d'usage est l'expérimentation éducative sur les internals des jeux sous Linux et le développement de hooks.
