Этот проект — плагин Binary Ninja для разработки и анализа шеллкода.
Он обеспечивает многоархитектурную ассемблировку и дизассемблировку, а также преобразование между inline hex, hex с пробелами, байтовыми строками Python и массивами в стиле C.
Инструмент реализован на Python и напрямую интегрируется в рабочий процесс Binary Ninja для быстрой итерации payload.
Основной сценарий использования — реверс-инжиниринг, прототипирование эксплойтов и исследования безопасности с упором на шеллкод.
