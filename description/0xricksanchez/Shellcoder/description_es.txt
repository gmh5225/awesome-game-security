Este proyecto es un plugin de Binary Ninja para el desarrollo y análisis de shellcode.
Ofrece ensamblado y desensamblado multiarquitectura, junto con conversión entre hex en línea, hex con espacios, cadenas de bytes de Python y arrays al estilo C.
La herramienta está implementada en Python y se integra directamente en el flujo de trabajo de Binary Ninja para una iteración rápida de payloads.
Su principal caso de uso es la ingeniería inversa, el prototipado de exploits y la investigación de seguridad centrada en shellcode.
