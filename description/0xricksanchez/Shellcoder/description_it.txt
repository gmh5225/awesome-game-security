Questo progetto è un plugin di Binary Ninja per lo sviluppo e l'analisi di shellcode.
Fornisce assemblaggio e disassemblaggio multi-architettura, insieme alla conversione tra hex inline, hex con spazi, stringhe di byte Python e array in stile C.
Lo strumento è implementato in Python e si integra direttamente nel flusso di lavoro di Binary Ninja per un'iterazione rapida dei payload.
Il caso d'uso principale è il reverse engineering, la prototipazione di exploit e la ricerca di sicurezza incentrata sullo shellcode.
