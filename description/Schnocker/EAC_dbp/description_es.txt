EAC Bypass es un proyecto de prueba de concepto para Windows orientado a depurar aplicaciones protegidas por Easy Anti-Cheat.
Combina un controlador en modo kernel y un módulo en modo usuario para interferir con callbacks seleccionados del anti-cheat, el comportamiento de minifiltros y las rutas de control.
La implementación está escrita en C y C++ con estructura de proyecto de Visual Studio y WDK, e incluye lógica de interceptación de API en capa de usuario.
Su caso de uso principal es la ingeniería inversa y las pruebas de seguridad controladas de procesos de juegos protegidos por anti-cheat.
