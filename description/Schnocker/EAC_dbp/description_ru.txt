EAC Bypass — это Windows proof-of-concept проект для отладки приложений, защищённых Easy Anti-Cheat.
Он сочетает драйвер режима ядра и модуль пользовательского режима, чтобы вмешиваться в выбранные callbacks античита, поведение минифильтров и пути управления.
Реализация написана на C и C++ со структурой проекта Visual Studio и WDK и включает логику перехвата API на пользовательском уровне.
Основной сценарий использования — реверс-инжиниринг и контролируемое тестирование безопасности игровых процессов, защищённых античитом.
