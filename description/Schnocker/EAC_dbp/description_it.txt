EAC Bypass è un progetto proof-of-concept per Windows volto al debug di applicazioni protette da Easy Anti-Cheat.
Combina un driver in modalità kernel e un modulo in modalità utente per interferire con callback anti-cheat selezionati, il comportamento dei minifilter e i percorsi di controllo.
L'implementazione è scritta in C e C++ con struttura di progetto Visual Studio e WDK, inclusa la logica di intercettazione delle API a livello utente.
Il caso d'uso principale è il reverse engineering e i test di sicurezza controllati di processi di gioco protetti da anti-cheat.
