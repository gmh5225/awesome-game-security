NoEye est une boîte à outils de contournement de BattlEye destinée à contrer certains comportements de protection dans les environnements de jeu.
Le dépôt comprend des composants natifs en C++, une couche de service Windows dédiée, ainsi qu'une application d'installation en C# pour les tâches de déploiement.
Sa structure de code couvre l'installation de services, l'interaction avec les processus et des modules de contrôle d'exécution organisés autour de projets Visual Studio.
Il reste surtout pertinent comme référence historique de contournement d'anti-cheat pour les chercheurs qui analysent d'anciennes techniques centrées sur BattlEye.
