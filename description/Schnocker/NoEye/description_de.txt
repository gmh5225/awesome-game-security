NoEye ist ein BattlEye-Bypass-Toolkit, das bestimmte Schutzverhalten in Spielumgebungen gezielt umgehen soll.
Das Repository enthält native C++-Komponenten, eine eigene Windows-Dienstschicht sowie eine C#-Setup-Anwendung für Bereitstellungsaufgaben.
Die Codestruktur umfasst Dienstinstallation, Prozessinteraktion und Laufzeitsteuerungsmodule, aufgebaut um Visual-Studio-Projekte.
Es ist vor allem als historische Anti-Cheat-Bypass-Referenz für Forscher relevant, die ältere BattlEye-bezogene Techniken analysieren.
