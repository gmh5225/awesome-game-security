NoEye — это набор инструментов для обхода BattlEye, предназначенный для противодействия определённым механизмам защиты в игровых средах.
Репозиторий включает нативные компоненты на C++, отдельный слой службы Windows и приложение установки на C# для задач развёртывания.
Структура кода охватывает установку службы, взаимодействие с процессами и модули управления во время выполнения, организованные вокруг проектов Visual Studio.
В основном он полезен как исторический справочный материал по обходу античита для исследователей, анализирующих более ранние техники, ориентированные на BattlEye.
