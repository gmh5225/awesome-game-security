NoEye è un toolkit di bypass di BattlEye pensato per contrastare specifici comportamenti di protezione negli ambienti di gioco.
Il repository include componenti nativi in C++, un layer dedicato di servizio Windows e un'applicazione di setup in C# per le attività di distribuzione.
La struttura del codice copre l'installazione del servizio, l'interazione con i processi e moduli di controllo a runtime organizzati intorno a progetti Visual Studio.
È rilevante soprattutto come riferimento storico di bypass anti-cheat per i ricercatori che analizzano tecniche più datate incentrate su BattlEye.
