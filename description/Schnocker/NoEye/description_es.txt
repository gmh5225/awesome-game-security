NoEye es un kit de herramientas de evasión de BattlEye pensado para contrarrestar comportamientos de protección concretos en entornos de juego.
El repositorio incluye componentes nativos en C++, una capa dedicada de servicio de Windows y una aplicación de instalación en C# para tareas de despliegue.
Su estructura de código abarca la instalación de servicios, la interacción con procesos y módulos de control en tiempo de ejecución organizados en torno a proyectos de Visual Studio.
Resulta relevante principalmente como referencia histórica de evasión de anti-cheat para investigadores que analizan técnicas antiguas centradas en BattlEye.
