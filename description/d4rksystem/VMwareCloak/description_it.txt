Questo progetto è uno script PowerShell che tenta di aiutare gli analisti di malware a nascondere le proprie VM Windows VMware da malware che potrebbero cercare di eludere l'analisi.
Lo script ottiene questo risultato procedendo come segue.
È soprattutto utile per ricercatori di sicurezza nei videogiochi e reverse engineer che studiano tecniche offensive nell'ambito di cheat / ambienti virtuali.
