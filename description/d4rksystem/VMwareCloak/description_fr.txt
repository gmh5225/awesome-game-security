Ce projet est un script PowerShell qui vise à aider les analystes de malwares à masquer leurs VM Windows VMware face à des malwares susceptibles de tenter d'échapper à l'analyse.
Le script y parvient en procédant comme suit.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine des cheats / environnements virtuels.
