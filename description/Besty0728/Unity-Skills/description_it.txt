Questo progetto è un plugin per l'editor di Unity che fornisce skill di automazione basate sull'IA che coprono un'ampia gamma di sottosistemi del motore. La codebase C# include script dell'editor per animator, audio, camera, Cinemachine, materiali, NavMesh, ottimizzazione, fisica, rendering, shader, terrain, UI e molti altri componenti Unity, oltre a un esecutore batch e al supporto per la localizzazione. È utile soprattutto a sviluppatori Unity e technical artist che cercano workflow dell'editor automatizzati e manipolazione delle scene assistita dall'IA.
