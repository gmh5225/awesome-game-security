Этот проект — плагин редактора Unity, предоставляющий навыки автоматизации на основе ИИ для широкого спектра подсистем движка. Кодовая база на C# включает редакторские скрипты для animator, аудио, камеры, Cinemachine, материалов, NavMesh, оптимизации, физики, рендеринга, шейдеров, terrain, UI и многих других компонентов Unity, а также пакетный исполнитель и поддержку локализации. Он в первую очередь полезен разработчикам Unity и техническим художникам, которым нужны автоматизированные рабочие процессы в редакторе и манипуляции со сценой с помощью ИИ.
