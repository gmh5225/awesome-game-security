Avanguard es una biblioteca antiintrusión Win32 para reforzar aplicaciones x86 y x64 frente a la manipulación. La base de código en C y C++ incluye técnicas antidepuración, controles antiinyección, análisis de memoria y de la pila de llamadas, comprobaciones de integridad y utilidades de protección. También admite defensas a nivel de proceso como filtrado de callbacks, comprobaciones de módulos y API defensivas relacionadas, con enlaces opcionales. Su caso de uso principal es proteger clientes de juegos y otro software sensible frente a la depuración, la inyección y la manipulación en tiempo de ejecución.
