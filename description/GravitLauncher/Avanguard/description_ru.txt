Avanguard — библиотека защиты от вторжений для Win32, предназначенная для усиления x86- и x64-приложений против вмешательства. Кодовая база на C и C++ включает методы антиотладки, средства противодействия инъекциям, анализ памяти и стека вызовов, проверки целостности и утилиты защиты. Также поддерживаются защиты на уровне процесса: фильтрация callback'ов, проверки модулей и связанные защитные API с опциональными привязками. Основной сценарий использования — защита игровых клиентов и другого чувствительного ПО от отладки, инъекций и манипуляций во время выполнения.
