Framework de jeu 3D multiplateforme en C++ proposant un pipeline de rendu OpenGL ES 2.0/3.0, un graphe de scène avec système de matériaux/shaders, l'intégration physique Bullet, l'audio OpenAL, l'animation squelettique, le rendu de terrain, les effets de particules et une boîte à outils d'interface utilisateur basée sur des formulaires. Il cible Windows, macOS, Linux, iOS et Android à partir d'une base de code unique basée sur CMake, avec le support du scripting Lua et un outil d'encodage pour convertir les assets FBX/Collada au format binaire .gpb du moteur.
