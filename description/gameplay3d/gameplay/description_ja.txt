クロスプラットフォームの C++ 3D ゲームフレームワークで、OpenGL ES 2.0/3.0 レンダリングパイプライン、マテリアル／シェーダーシステム付きシーングラフ、Bullet 物理統合、OpenAL オーディオ、スケルタルアニメーション、地形レンダリング、パーティクルエフェクト、フォームベースの UI ツールキットを提供します。単一の CMake ベースのコードベースから Windows、macOS、Linux、iOS、Android を対象とし、Lua スクリプティングと、FBX/Collada アセットをエンジンのバイナリ .gpb 形式に変換するエンコーダーツールを備えています。
