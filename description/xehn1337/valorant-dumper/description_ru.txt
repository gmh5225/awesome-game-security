Этот проект — дампер оффсетов и SDK для Valorant, который извлекает определения классов, адреса функций и смещения памяти из работающего процесса игры Valorant. Он сканирует структуры Unreal Engine в Valorant, чтобы дампить массивы GObject, GNames, раскладки сущностей игрока и адреса данных оружия для разработки читов. Предназначен для исследователей игровой безопасности, изучающих раскладку памяти Valorant, и исследователей античита Vanguard, анализирующих инструменты читерства.
