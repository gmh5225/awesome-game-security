Этот проект — header-only библиотека на C++ для манипуляции памятью в приложениях Windows. Она предоставляет типизированные помощники чтения и записи с проверкой указателей, опциональной обработкой исключений и опциональным логированием, чтобы снизить сбои из‑за небезопасного доступа к памяти. Также включены утилиты сканирования по сигнатурам (pattern scanning) и обёртки для вызова функций с распространёнными соглашениями о вызовах. Она полезна для инструментов реверс-инжиниринга, исследования памяти игр и других задач низкоуровневой отладки.
