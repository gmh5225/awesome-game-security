Este proyecto es una biblioteca C++ de solo cabeceras para manipulación de memoria en aplicaciones Windows. Ofrece helpers tipados de lectura y escritura con validación de punteros, manejo opcional de excepciones y registro opcional para reducir fallos por accesos inseguros a memoria. También incluye utilidades de pattern scanning y wrappers para llamar funciones con convenciones de llamada habituales. Es útil para herramientas de ingeniería inversa, investigación de memoria en juegos y otras tareas de depuración de bajo nivel.
