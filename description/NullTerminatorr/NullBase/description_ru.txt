NullBase — простая C++ база для game hacking, рассчитанная на удобное чтение и расширение. Она даёт базовые строительные блоки: хелперы памяти, абстракции сущностей и локального игрока, математические утилиты и код, связанный с world-to-screen. Проект использует понятную структуру Visual Studio, чтобы новички могли быстро ориентироваться и добавлять функции. Основное назначение — показать начинающим, как устроены и итерируются cheat-фреймворки.
