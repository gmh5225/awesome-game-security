NullBase es una base sencilla en C++ para game hacking, diseñada para ser fácil de leer y ampliar. Proporciona bloques fundamentales como helpers de memoria, abstracciones de entidad y jugador local, utilidades matemáticas y código relacionado con world-to-screen. El proyecto usa una estructura clara de Visual Studio para que los aprendices puedan navegar rápidamente y añadir funciones. Su caso de uso principal es enseñar a principiantes cómo se estructuran e iteran los frameworks de cheats.
