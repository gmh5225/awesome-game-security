NullHook è un progetto Windows di game hacking orientato ai tutorial che combina componenti in user mode e in kernel mode. Illustra flussi di lavoro legati a memoria e hook in una solution suddivisa che contiene sia un progetto driver sia un client in userland. L'implementazione è principalmente in C/C++ ed è pensata per essere caricata tramite manual mapping esterno in configurazioni pratiche. Il caso d'uso principale è la ricerca educativa sulle tecniche di sviluppo di cheat assistiti dal kernel.
