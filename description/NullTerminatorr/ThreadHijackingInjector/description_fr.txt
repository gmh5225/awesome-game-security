ThreadHijackingInjector est une preuve de concept C++ minimale pour l'injection de DLL via le détournement de threads sous Windows. Le projet se concentre sur une implémentation compacte qui illustre la manipulation de contexte et la redirection d'exécution dans un processus distant. Son empreinte de code réduite rend la technique plus facile à étudier que les frameworks d'injection complets. L'usage principal est l'ingénierie inverse éducative et la recherche en sécurité sur des méthodes alternatives d'injection de processus.
