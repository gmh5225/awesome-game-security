Este proyecto es un plugin de IDA Pro para gestionar y organizar comentarios de usuario durante el reverse engineering.
Está escrito en Python 3 y engancha acciones relacionadas con comentarios para capturar nuevas anotaciones y mostrarlas en una vista dedicada.
El plugin está pensado para flujos de trabajo de IDA 7.x y 8.x y ofrece acceso por teclado y menú para revisar rápidamente los datos de comentarios.
Es útil para ingenieros inversos que necesitan una mejor gestión de notas al analizar binarios, incluidos clientes de juegos y módulos anti-cheat.
