FreeDirect es una capa de compatibilidad en C++20 que reimplementa un subconjunto estrecho y orientado a juegos de DirectX 3 (2D), para que títulos Win32 DirectX antiguos puedan ejecutarse en plataformas modernas sin el SDK original de DirectX ni Windows. Se centra en las API DirectDraw, DirectSound y DirectPlay que usan realmente juegos objetivo concretos, incluidas superficies, blits, paletas, clippers y reproducción de audio PCM estático. El renderizado y el audio se basan en SDL3, mientras que la red de DirectPlay usa un transporte enchufable con una implementación basada en ENet para crear/unirse a sesiones, gestionar jugadores y mensajería garantizada entre pares enlazados con FreeDirect. El proyecto se construye con CMake y ofrece cabeceras públicas de estilo COM además de pruebas automatizadas. Su caso de uso principal es una compatibilidad profunda, impulsada por los puntos de llamada, para portar y ejecutar juegos DirectX 2D clásicos seleccionados de forma multiplataforma.
