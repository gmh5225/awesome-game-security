FreeDirect は C++20 の互換レイヤーで、ゲーム用途に絞った DirectX 3（2D）の限定的なサブセットを再実装し、レガシーな Win32 DirectX タイトルを、元の DirectX SDK や Windows なしで現代のプラットフォーム上で動作させます。特定の対象ゲームが実際に使う DirectDraw、DirectSound、DirectPlay API に焦点を当て、サーフェス、ブリット、パレット、クリッパー、静的 PCM 音声再生などを扱います。描画と音声は SDL3 で支えられ、DirectPlay のネットワークは差し替え可能なトランスポートを用い、ENet ベースの実装でセッションの作成/参加、プレイヤー管理、FreeDirect 連携ピア間の保証付きメッセージングを提供します。プロジェクトは CMake でビルドされ、COM スタイルの公開ヘッダーと自動テストを同梱します。主な用途は、選定したクラシックな 2D DirectX ゲームをクロスプラットフォームで移植・実行するための、呼び出し箇所に基づく深い互換性です。
