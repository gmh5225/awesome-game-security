FreeDirect 是一個 C++20 相容層，重新實作了以遊戲為導向、範圍較窄的 DirectX 3（2D）子集，讓舊版 Win32 DirectX 遊戲無需原始 DirectX SDK 或 Windows 即可在現代平台上執行。它聚焦於特定目標遊戲實際使用的 DirectDraw、DirectSound 與 DirectPlay API，包括表面、blit、調色盤、裁剪器以及靜態 PCM 音訊播放。繪製與音訊由 SDL3 提供支援，而 DirectPlay 網路則採用可插拔傳輸層，並提供基於 ENet 的實作，用於工作階段建立/加入、玩家管理，以及 FreeDirect 對等端之間的可靠訊息傳遞。專案使用 CMake 建置，並附帶 COM 風格的公開標頭檔與自動化測試。其主要用途是為選定的經典 2D DirectX 遊戲提供深入的、依呼叫點驅動的相容性，以便跨平台移植與執行。
