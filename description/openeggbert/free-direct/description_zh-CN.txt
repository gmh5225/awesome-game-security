FreeDirect 是一个 C++20 兼容层，重新实现了面向游戏的、范围较窄的 DirectX 3（2D）子集，使旧版 Win32 DirectX 游戏无需原始 DirectX SDK 或 Windows 即可在现代平台上运行。它聚焦于特定目标游戏实际使用的 DirectDraw、DirectSound 和 DirectPlay API，包括表面、blit、调色板、裁剪器以及静态 PCM 音频播放。渲染与音频由 SDL3 提供支持，而 DirectPlay 网络则采用可插拔传输层，并提供基于 ENet 的实现，用于会话创建/加入、玩家管理，以及 FreeDirect 对等端之间的可靠消息传递。项目使用 CMake 构建，并附带 COM 风格的公共头文件与自动化测试。其主要用途是为选定的经典 2D DirectX 游戏提供深入的、按调用点驱动的兼容性，以便跨平台移植与运行。
