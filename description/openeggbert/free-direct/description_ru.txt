FreeDirect — слой совместимости на C++20, который заново реализует узкое, ориентированное на игры подмножество DirectX 3 (2D), чтобы устаревшие Win32 DirectX-игры могли работать на современных платформах без оригинального DirectX SDK и Windows. Он сосредоточен на API DirectDraw, DirectSound и DirectPlay, реально используемых конкретными целевыми играми, включая поверхности, blit, палитры, клипперы и воспроизведение статического PCM-аудио. Рендеринг и звук опираются на SDL3, а сетевой DirectPlay использует подключаемый транспорт с реализацией на базе ENet для создания/входа в сессии, управления игроками и гарантированной доставки сообщений между узлами, связанными через FreeDirect. Проект собирается с помощью CMake и поставляется с публичными заголовками в стиле COM и автоматизированными тестами. Основной сценарий — глубокая совместимость на уровне точек вызова для портирования и кроссплатформенного запуска избранных классических 2D DirectX-игр.
