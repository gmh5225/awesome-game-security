KernelMon è un framework di monitoraggio Windows basato sulla virtualizzazione che traccia l'attività del kernel con un flusso di lavoro simile a ProcMon. Aggancia API selezionate in modalità kernel e inoltra i log a un'interfaccia in modalità utente, coprendo operazioni su file system, Registro, processi e thread. L'implementazione usa un driver del kernel con componenti di intercettazione a basso livello in stile VMX/EPT e un'interfaccia desktop associata. È pensato per la ricerca sulla sicurezza del kernel, l'analisi del comportamento e le indagini anti-cheat o malware in ambienti VM controllati.
