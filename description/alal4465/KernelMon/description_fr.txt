KernelMon est un framework de surveillance Windows basé sur la virtualisation qui trace l'activité du noyau selon un flux de travail proche de ProcMon. Il accroche des API en mode noyau sélectionnées et transmet les journaux à une interface en mode utilisateur, couvrant le système de fichiers, le Registre, les processus et les threads. L'implémentation repose sur un pilote noyau avec des composants d'interception bas niveau de type VMX/EPT et une interface de bureau associée. Il est destiné à la recherche en sécurité du noyau, à l'analyse comportementale et aux enquêtes anti-cheat ou malware dans des environnements de VM contrôlés.
