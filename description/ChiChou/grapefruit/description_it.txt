Questo progetto è un toolkit di esplorazione runtime basato su Frida per applicazioni mobili, con un’interfaccia web. Offre method hooking, intercettazione delle API crittografiche, ispezione del filesystem e di SQLite, streaming dei log, segnalazione dei crash e funzionalità specifiche di piattaforma per iOS e Android. Lo stack è principalmente TypeScript e combina servizi di strumentazione backend con un frontend ricco per l’analisi interattiva. Include anche funzioni specializzate di sicurezza mobile come ispezione di keychain e keystore, cattura del traffico, introspezione di Java e Objective-C e manipolazione dell’ambiente. Il caso d’uso principale è il testing di sicurezza delle app mobili, il reverse engineering e i flussi di analisi dinamica.
