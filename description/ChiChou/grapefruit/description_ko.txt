이 프로젝트는 웹 기반 인터페이스를 갖춘 Frida 기반 모바일 애플리케이션 런타임 탐색 툴킷입니다. 메서드 후킹, 암호화 API 가로채기, 파일시스템 및 SQLite 검사, 로그 스트리밍, 크래시 보고, 그리고 iOS와 Android용 플랫폼별 기능을 제공합니다. 스택은 주로 TypeScript로 구성되며, 백엔드 계측 서비스와 대화형 분석을 위한 풍부한 프론트엔드를 결합합니다. 키체인 및 키스토어 검사, 트래픽 캡처, Java와 Objective-C 인트로스펙션, 환경 조작과 같은 전문 모바일 보안 기능도 포함합니다. 주요 사용 사례는 모바일 앱 보안 테스트, 리버스 엔지니어링, 동적 분석 워크플로입니다.
