Este proyecto es un kit de herramientas de exploración en tiempo de ejecución impulsado por Frida para aplicaciones móviles, con una interfaz web. Ofrece enganche de métodos, interceptación de API criptográficas, inspección del sistema de archivos y de SQLite, streaming de registros, informes de fallos y capacidades específicas de plataforma para iOS y Android. El stack es principalmente TypeScript y combina servicios de instrumentación en el backend con un frontend rico para el análisis interactivo. También incluye funciones especializadas de seguridad móvil como inspección de keychain y keystore, captura de tráfico, introspección de Java y Objective-C, y manipulación del entorno. El caso de uso principal son las pruebas de seguridad de apps móviles, la ingeniería inversa y los flujos de análisis dinámico.
