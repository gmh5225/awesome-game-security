本プロジェクトは、Web ベースのインターフェースを備えた、Frida 駆動のモバイルアプリ向けランタイム探索ツールキットです。メソッドのフック、暗号 API の傍受、ファイルシステムと SQLite の検査、ログストリーミング、クラッシュ報告、および iOS と Android 向けのプラットフォーム固有機能を提供します。スタックは主に TypeScript で構成され、バックエンドのインストルメンテーションサービスと、対話的な分析向けの充実したフロントエンドを組み合わせています。キーチェーン／キーストア検査、トラフィックキャプチャ、Java と Objective-C のイントロスペクション、環境操作といったモバイルセキュリティ向けの専門機能も含まれます。主な用途は、モバイルアプリのセキュリティテスト、リバースエンジニアリング、動的解析のワークフローです。
