Этот проект — набор инструментов для исследования приложений во время выполнения на базе Frida с веб-интерфейсом, ориентированный на мобильные приложения. Он предлагает перехват методов, перехват криптографических API, инспекцию файловой системы и SQLite, потоковую передачу логов, отчёты о сбоях, а также платформенные возможности для iOS и Android. Стек в основном построен на TypeScript и сочетает серверные службы инструментирования с богатым фронтендом для интерактивного анализа. Также включены специализированные функции мобильной безопасности: инспекция keychain и keystore, захват трафика, интроспекция Java и Objective-C, манипуляция окружением. Основной сценарий использования — тестирование безопасности мобильных приложений, реверс-инжиниринг и рабочие процессы динамического анализа.
