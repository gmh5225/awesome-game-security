Ce projet est une boîte à outils d’exploration runtime alimentée par Frida pour les applications mobiles, avec une interface web. Il propose le hooking de méthodes, l’interception des API cryptographiques, l’inspection du système de fichiers et de SQLite, le streaming de logs, le reporting de crashs, ainsi que des capacités spécifiques à iOS et Android. La pile repose principalement sur TypeScript et combine des services d’instrumentation côté backend avec un frontend riche pour l’analyse interactive. Il inclut aussi des fonctions spécialisées de sécurité mobile telles que l’inspection du keychain et du keystore, la capture de trafic, l’introspection Java et Objective-C, et la manipulation d’environnement. Le cas d’usage principal est le test de sécurité d’applications mobiles, la rétro-ingénierie et les flux d’analyse dynamique.
