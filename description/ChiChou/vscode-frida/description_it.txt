Questo progetto è un’estensione non ufficiale di VS Code che fornisce un workbench Frida per l’instrumentazione mobile. Integra selezione dei target, modifica e invio degli script, flussi di lavoro REPL, visualizzazione dei log del dispositivo e strumenti di supporto per sessioni iOS e Android. L’estensione è costruita principalmente in TypeScript con punti di integrazione Python ed è progettata per funzionare con frida-tools e le comuni utilità di connettività mobile. È rivolta a ricercatori di sicurezza mobile e reverse engineer che vogliono eseguire flussi di lavoro Frida direttamente in un IDE.
