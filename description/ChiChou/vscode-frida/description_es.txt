Este proyecto es una extensión no oficial de VS Code que ofrece un entorno de trabajo Frida para la instrumentación móvil. Integra selección de objetivos, edición y envío de scripts, flujos REPL, visualización de registros del dispositivo y herramientas auxiliares para sesiones en iOS y Android. La extensión está construida principalmente con TypeScript y puntos de integración en Python, y está diseñada para funcionar con frida-tools y utilidades habituales de conectividad móvil. Está orientada a investigadores de seguridad móvil e ingenieros inversos que desean ejecutar flujos de trabajo Frida directamente en un IDE.
