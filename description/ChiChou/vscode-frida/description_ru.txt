Этот проект — неофициальное расширение VS Code, предоставляющее рабочую среду Frida для мобильной инструментации. Оно объединяет выбор цели, редактирование и отправку скриптов, рабочие процессы REPL, просмотр логов устройства и вспомогательные инструменты для сессий iOS и Android. Расширение в основном написано на TypeScript с точками интеграции Python и рассчитано на работу с frida-tools и распространёнными утилитами мобильного подключения. Оно предназначено для исследователей мобильной безопасности и специалистов по реверс-инжинирингу, которые хотят запускать рабочие процессы Frida прямо в IDE.
