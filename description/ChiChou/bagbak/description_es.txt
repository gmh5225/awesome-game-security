Este proyecto es una herramienta CLI en Node.js y TypeScript que usa Frida para descifrar y volcar aplicaciones iOS a archivos IPA desde dispositivos con jailbreak. Incluye un agente Frida para el descifrado en tiempo de ejecución, gestiona extensiones de app y frameworks embebidos, y admite conexiones USB y remotas. Es especialmente útil para investigadores de seguridad iOS que realizan descifrado de apps, análisis binario e ingeniería inversa en dispositivos con jailbreak.
