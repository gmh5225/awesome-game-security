Questo progetto è una vetrina di un motore di gioco C++ personalizzato più datato, con documentazione, layout Lua/UI, asset di livelli, note sul networking e configurazione runtime correlata.
L'archivio include componenti di terze parti legati al motore, oltre a contenuti di gioco in etc/ e documentazione in doc/.
È soprattutto utile per sviluppatori di giochi e programmatori di motori che studiano layout e tooling di motori legacy.
