Ce projet est une base de code expérimentale précoce destinée à l'apprentissage des concepts de recompilation dynamique sur x86. L'implémentation est en C++ et s'appuie sur la bibliothèque de désassemblage Zydis pour décoder les flux d'instructions, afficher les mnémoniques et inspecter les détails des opérandes. Elle inclut une petite configuration de build avec des Makefiles et un Dockerfile, ainsi que des tables d'aide pour le mappage des noms de registres. Le principal cas d'usage est la pratique éducative de la rétro-ingénierie pour ceux qui étudient le décodage d'instructions et les pipelines de traduction binaire.
