Este proyecto es una base de código experimental temprana para aprender conceptos de recompilación dinámica en x86. La implementación está en C++ y usa la biblioteca de desensamblado Zydis para decodificar flujos de instrucciones, imprimir mnemónicos e inspeccionar detalles de operandos. Incluye una configuración de build sencilla con Makefiles y un Dockerfile, además de tablas auxiliares para el mapeo de nombres de registros. El caso de uso principal es la práctica educativa de ingeniería inversa para quienes estudian la decodificación de instrucciones y las canalizaciones de traducción binaria.
