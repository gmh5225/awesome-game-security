Этот проект — ранняя экспериментальная кодовая база для изучения концепций динамической рекомпиляции на x86. Реализация написана на C++ и использует библиотеку дизассемблирования Zydis для декодирования потоков инструкций, вывода мнемоник и просмотра деталей операндов. В комплект входят небольшая система сборки с Makefile и Dockerfile, а также вспомогательные таблицы для сопоставления имён регистров. Основной сценарий использования — учебная практика по реверс-инжинирингу для тех, кто изучает декодирование инструкций и конвейеры бинарной трансляции.
