Этот проект — компактный Windows proof of concept, демонстрирующий граничные случаи проверок выравнивания при instrumentation callback процессов. Он сочетает C и x64-ассемблер для установки флагов, связанных с AC, вызова невыровненного доступа к памяти и наблюдения возникающих исключений STATUS_DATATYPE_MISALIGNMENT. Пример использует NtSetInformationProcess с ProcessInstrumentationCallback и пользовательскую обработку исключений, чтобы показать, где код callback может завершаться сбоем. Он в первую очередь полезен для исследований низкоуровневой kernel- и runtime-instrumentation.
