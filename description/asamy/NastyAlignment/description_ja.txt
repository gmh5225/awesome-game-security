本プロジェクトは、プロセスの instrumentation コールバックにおけるアライメントチェックのエッジケースを示す、コンパクトな Windows の実証コードです。C と x64 アセンブリを組み合わせて AC 関連フラグを設定し、未整列メモリアクセスを発生させ、その結果としての STATUS_DATATYPE_MISALIGNMENT 例外を観察します。サンプルは NtSetInformationProcess と ProcessInstrumentationCallback、および独自の例外処理を用いて、コールバックコードが失敗し得る箇所を明らかにします。主に低レベルなカーネルおよびランタイム instrumentation の研究に役立ちます。
