Este proyecto es una prueba de concepto compacta para Windows que demuestra casos límite de comprobaciones de alineación con callbacks de instrumentación de procesos. Combina C y ensamblador x64 para establecer flags relacionados con AC, provocar accesos a memoria no alineados y observar las excepciones STATUS_DATATYPE_MISALIGNMENT resultantes. El ejemplo usa NtSetInformationProcess con ProcessInstrumentationCallback y manejo personalizado de excepciones para resaltar dónde puede fallar el código del callback. Es especialmente útil para la investigación de instrumentación de bajo nivel en kernel y runtime.
