Ce projet est un plugin IDAPython destiné à la rétro-ingénierie des kernelcaches iOS ; il associe les appels de méthodes virtuelles C++ à leurs implémentations réelles.
Il s'appuie sur la reconstruction de la hiérarchie de classes d'ida_kernelcache pour synchroniser les fonctions binaires avec les méthodes virtuelles d'origine, permettant la navigation par double-clic sur les appels virtuels C++ et le suivi des références croisées.
Inspiré par ida_medigate, il exploite la structure particulière des kernelcaches iOS pour offrir un environnement de recherche plus puissant afin d'analyser le code C++ fortement basé sur les vtables.
Il est surtout utile aux chercheurs en sécurité du noyau iOS qui étudient les schémas de répartition virtuelle C++ et les flux de code fondés sur les vtables dans les kernelcaches Apple.
