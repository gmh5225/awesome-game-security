Этот проект — плагин IDAPython для реверс-инжиниринга iOS kernelcache, который сопоставляет вызовы виртуальных методов C++ с их реальными реализациями.
Он использует реконструкцию иерархии классов из ida_kernelcache, чтобы синхронизировать бинарные функции с исходными виртуальными методами, обеспечивая переход двойным щелчком по виртуальным вызовам C++ и отслеживание перекрёстных ссылок.
Вдохновлённый ida_medigate, он опирается на уникальную структуру iOS kernelcache и создаёт более мощную исследовательскую среду для анализа C++-кода с активным использованием vtable.
Он в первую очередь полезен исследователям безопасности ядра iOS, изучающим схемы виртуальной диспетчеризации C++ и потоки управления на основе vtable в kernelcache Apple.
