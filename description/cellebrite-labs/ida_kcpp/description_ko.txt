이 프로젝트는 iOS kernelcache를 리버스 엔지니어링하기 위한 IDAPython 플러그인으로, C++ 가상 메서드 호출을 실제 구현에 매핑합니다.
ida_kernelcache의 클래스 계층 재구성을 활용해 바이너리 함수를 원래 가상 메서드와 동기화하며, C++ 가상 호출에 대한 더블 클릭 탐색과 교차 참조 추적을 가능하게 합니다.
ida_medigate에서 영감을 받아 iOS kernelcache의 고유한 구조를 활용하며, vtable을 많이 사용하는 C++ 코드를 분석하기 위한 더 강력한 연구 환경을 제공합니다.
주로 Apple kernelcache에서 C++ 가상 디스패치 패턴과 vtable 기반 코드 흐름을 분석하는 iOS 커널 보안 연구자에게 유용합니다.
