本プロジェクトは、iOS kernelcache のリバースエンジニアリング向け IDAPython プラグインであり、C++ の仮想メソッド呼び出しを実際の実装へ対応付けます。
ida_kernelcache によるクラス階層の再構築を活用し、バイナリ上の関数と元の仮想メソッドを同期することで、C++ 仮想呼び出しへのダブルクリックナビゲーションとクロス参照の追跡を可能にします。
ida_medigate に着想を得ており、iOS kernelcache 特有の構造を利用して、vtable を多用する C++ コードをより強力に解析できる研究環境を提供します。
主に、Apple の kernelcache における C++ 仮想ディスパッチや vtable ベースの制御フローを分析する iOS カーネルセキュリティ研究者向けです。
