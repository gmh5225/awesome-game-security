Questo progetto è un plugin di IDA Pro che risolve le chiamate PPL (Page Protection Layer) nei kernelcache di iOS e macOS alle effettive funzioni PPL sottostanti.
Fornisce navigazione tramite scorciatoia (Ctrl-Shift-X) per saltare tra i siti di chiamata dei gate PPL e le funzioni PPL di destinazione, con analisi automatica dell'IDB per identificare e annotare i riferimenti incrociati PPL.
Il plugin Python usa ida-netnode per lo storage persistente e si integra con il menu contestuale di IDA per la risoluzione bidirezionale di funzioni PPL e siti di chiamata.
È soprattutto utile a ricercatori del kernel iOS e reverse engineer che analizzano percorsi di codice protetti da PPL nei kernelcache Apple.
