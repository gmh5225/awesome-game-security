Ce projet est un plugin IDA Pro qui résout les appels PPL (Page Protection Layer) dans les kernelcaches iOS et macOS vers leurs fonctions PPL sous-jacentes réelles.
Il propose une navigation par raccourci clavier (Ctrl-Shift-X) pour passer des sites d'appel des portes PPL aux fonctions PPL cibles, avec une analyse IDB automatique pour identifier et annoter les références croisées PPL.
Le plugin Python utilise ida-netnode pour le stockage persistant et s'intègre au menu contextuel d'IDA pour une résolution bidirectionnelle des fonctions PPL et des sites d'appel.
Il est principalement utile aux chercheurs du noyau iOS et aux reverse engineers qui analysent les chemins de code protégés par PPL dans les kernelcaches Apple.
