Этот проект — плагин для IDA Pro, который разрешает вызовы PPL (Page Protection Layer) в kernelcache iOS и macOS до реальных нижележащих функций PPL.
Он обеспечивает навигацию по горячей клавише (Ctrl-Shift-X) для перехода между местами вызова PPL gate и целевыми функциями PPL, а также автоматический анализ IDB для выявления и аннотирования перекрёстных ссылок PPL.
Плагин на Python использует ida-netnode для постоянного хранения и интегрируется с контекстным меню IDA для двунаправленного разрешения функций PPL и мест вызова.
Он в основном полезен исследователям ядра iOS и специалистам по реверс-инжинирингу, анализирующим защищённые PPL пути кода в kernelcache Apple.
