本项目是一款 IDA Pro 插件，可将 iOS 与 macOS kernelcache 中的 PPL（Page Protection Layer）调用解析到其实际对应的底层 PPL 函数。
它提供基于快捷键（Ctrl-Shift-X）的导航，可在 PPL gate 调用点与目标 PPL 函数之间跳转，并通过自动 IDB 分析识别并标注 PPL 交叉引用。
该 Python 插件使用 ida-netnode 进行持久化存储，并与 IDA 上下文菜单集成，支持 PPL 函数与调用点的双向解析。
它主要面向分析 Apple kernelcache 中受 PPL 保护代码路径的 iOS 内核研究者与逆向工程师。
