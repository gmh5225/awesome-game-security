本專案是一款 IDA Pro 外掛，可將 iOS 與 macOS kernelcache 中的 PPL（Page Protection Layer）呼叫解析至其實際對應的底層 PPL 函式。
它提供基於快捷鍵（Ctrl-Shift-X）的導覽，可在 PPL gate 呼叫點與目標 PPL 函式之間跳轉，並透過自動 IDB 分析識別並標註 PPL 交叉參考。
此 Python 外掛使用 ida-netnode 進行持久化儲存，並與 IDA 內容功能表整合，支援 PPL 函式與呼叫點的雙向解析。
它主要適用於分析 Apple kernelcache 中受 PPL 保護程式碼路徑的 iOS 核心研究者與逆向工程師。
