이 프로젝트는 iOS 및 macOS kernelcache의 PPL(Page Protection Layer) 호출을 실제 하위 PPL 함수로 해석하는 IDA Pro 플러그인입니다.
단축키(Ctrl-Shift-X) 기반 탐색으로 PPL 게이트 호출 지점과 대상 PPL 함수 사이를 이동할 수 있으며, 자동 IDB 분석을 통해 PPL 교차 참조를 식별하고 주석을 달습니다.
이 Python 플러그인은 영구 저장소로 ida-netnode를 사용하며, IDA 컨텍스트 메뉴와 통합되어 PPL 함수와 호출 지점을 양방향으로 해석합니다.
주로 Apple kernelcache에서 PPL로 보호된 코드 경로를 분석하는 iOS 커널 연구자와 리버싱 엔지니어에게 유용합니다.
