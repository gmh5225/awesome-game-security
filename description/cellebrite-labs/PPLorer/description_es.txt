Este proyecto es un plugin de IDA Pro que resuelve las llamadas PPL (Page Protection Layer) en kernelcaches de iOS y macOS hacia sus funciones PPL subyacentes reales.
Ofrece navegación por atajo de teclado (Ctrl-Shift-X) para saltar entre sitios de llamada de puertas PPL y sus funciones PPL destino, con análisis automático del IDB para identificar y anotar referencias cruzadas PPL.
El plugin en Python usa ida-netnode para almacenamiento persistente y se integra con el menú contextual de IDA para la resolución bidireccional de funciones PPL y sitios de llamada.
Resulta especialmente útil para investigadores del kernel de iOS e ingenieros de ingeniería inversa que analizan rutas de código protegidas por PPL en kernelcaches de Apple.
