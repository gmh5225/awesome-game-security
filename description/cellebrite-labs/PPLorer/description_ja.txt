本プロジェクトは、iOS および macOS の kernelcache 内の PPL（Page Protection Layer）呼び出しを、実際の基盤となる PPL 関数へ解決する IDA Pro プラグインです。
ホットキー（Ctrl-Shift-X）によるナビゲーションで PPL ゲートの呼び出し元と対象 PPL 関数を相互にジャンプでき、自動 IDB 解析により PPL のクロスリファレンスを識別・注釈付けします。
この Python プラグインは永続ストレージに ida-netnode を使用し、IDA のコンテキストメニューと連携して PPL 関数と呼び出しサイトの双方向解決を行います。
主に、Apple kernelcache 内の PPL 保護されたコードパスを解析する iOS カーネル研究者およびリバースエンジニア向けです。
