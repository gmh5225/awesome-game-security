Этот проект — плагин для IDA Pro, который обращает оптимизацию function outlining (например, clang --moutline), встраивая outlined-функции обратно в их вызывающие функции.
Он создаёт клоны outlined-функций для каждого вызывающего, заменяет инструкции BL прямыми переходами к клону, преобразует инструкции RET в клоне в переходы обратно к вызывающему и добавляет каждый клон как function chunk.
Плагин поддерживает как ручной inlining через контекстное меню, так и эвристическое обнаружение всех outlined-функций для пакетной обработки.
Он особенно полезен реверс-инженерам, анализирующим оптимизированные по размеру ARM-бинарники, где function outlining нарушает декомпиляцию Hex-Rays и рабочие процессы статического анализа.
