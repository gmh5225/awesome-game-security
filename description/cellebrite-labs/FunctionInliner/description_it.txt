Questo progetto è un plugin per IDA Pro che inverte l'ottimizzazione di function outlining (ad es. clang --moutline) reinserendo le funzioni outlined nei relativi caller.
Crea cloni delle funzioni outlined per ciascun caller, sostituisce le istruzioni BL con branch diretti verso il clone, converte le istruzioni RET del clone in branch di ritorno al caller e aggiunge ogni clone come function chunk.
Il plugin supporta sia l'inlining manuale tramite menu contestuale sia l'identificazione euristica di tutte le funzioni outlined per l'elaborazione in batch.
È soprattutto utile per i reverse engineer che analizzano binari ARM ottimizzati per lo spazio, dove il function outlining interrompe la decompilazione Hex-Rays e i flussi di analisi statica.
