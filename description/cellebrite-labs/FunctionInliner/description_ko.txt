이 프로젝트는 함수 아웃라이닝 최적화(예: clang --moutline)를, 아웃라인된 함수를 호출자로 다시 인라인하여 되돌리는 IDA Pro 플러그인입니다.
호출자마다 아웃라인된 함수의 클론을 만들고, BL 명령을 클론으로의 직접 분기로 바꾸며, 클론의 RET 명령을 호출자로 돌아가는 분기로 변환하고, 각 클론을 함수 청크로 추가합니다.
플러그인은 컨텍스트 메뉴 기반의 수동 인라인과, 휴리스틱으로 모든 아웃라인 함수를 식별해 일괄 처리하는 방식을 모두 지원합니다.
주로 함수 아웃라이닝이 Hex-Rays 디컴파일과 정적 분석 워크플로를 깨뜨리는, 공간 최적화된 ARM 바이너리를 분석하는 리버스 엔지니어에게 유용합니다.
