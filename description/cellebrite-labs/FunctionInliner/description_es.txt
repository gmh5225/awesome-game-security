Este proyecto es un plugin de IDA Pro que revierte la optimización de function outlining (p. ej., clang --moutline) reintegrando las funciones outlined en sus llamadores.
Crea clones de las funciones outlined por cada llamador, reemplaza las instrucciones BL por saltos directos al clon, convierte las instrucciones RET del clon en saltos de retorno al llamador y añade cada clon como un function chunk.
El plugin admite tanto el inlining manual desde el menú contextual como la identificación heurística de todas las funciones outlined para el procesamiento por lotes.
Resulta especialmente útil para ingenieros inversos que analizan binarios ARM optimizados en tamaño, donde el function outlining rompe la descompilación de Hex-Rays y los flujos de análisis estático.
