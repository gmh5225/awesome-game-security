Dieses Projekt ist ein IDA-Pro-Plugin, das die Function-Outlining-Optimierung (z. B. clang --moutline) rückgängig macht, indem outlined Funktionen wieder in ihre Aufrufer inline eingebettet werden.
Es erzeugt pro Aufrufer Klone der outlined Funktionen, ersetzt BL-Anweisungen durch direkte Sprünge zum Klon, wandelt RET-Anweisungen im Klon in Rücksprünge zum Aufrufer um und fügt jeden Klon als Function Chunk hinzu.
Das Plugin unterstützt sowohl manuelles Inlining über das Kontextmenü als auch die heuristische Erkennung aller outlined Funktionen für die Stapelverarbeitung.
Es ist vor allem für Reverse Engineers nützlich, die speicherplatzoptimierte ARM-Binaries analysieren, bei denen Function Outlining die Hex-Rays-Dekompilierung und statische Analyseabläufe beeinträchtigt.
