Ce projet est un plugin IDA Pro qui inverse l'optimisation de function outlining (par ex. clang --moutline) en réintégrant les fonctions outlinées dans leurs appelants.
Il crée des clones des fonctions outlinées pour chaque appelant, remplace les instructions BL par des branchements directs vers le clone, convertit les instructions RET du clone en branchements de retour vers l'appelant, et ajoute chaque clone comme function chunk.
Le plugin prend en charge à la fois l'inlining manuel via le menu contextuel et l'identification heuristique de toutes les fonctions outlinées pour un traitement par lots.
Il est surtout utile aux reverse engineers qui analysent des binaires ARM optimisés en taille, où le function outlining perturbe la décompilation Hex-Rays et les flux d'analyse statique.
