本プロジェクトは、共有 git リポジトリを通じて、同一バイナリを扱う複数のリバースエンジニア間で IDB データを同期する IDA Pro プラグインです。
保存のたびに、名前、型、インライン関数、その他の IDB データを YAML 形式でエクスポートし、git でコミット・プッシュ・プルを行い、git mergetool によりマージ競合を自動解決します。
入力ファイルの MD5 で IDB を識別してユーザー間同期を実現し、頻繁で邪魔にならない同期を想定して設計されています。
同一バイナリ上で協業しつつ、データベース全体の共有を避けたいリバースエンジニアリングチーム向けの、軽量な IDB 同期手段として主に役立ちます。
