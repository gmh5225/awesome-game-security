이 프로젝트는 공유 git 저장소를 통해 동일한 바이너리를 다루는 여러 리버스 엔지니어 간에 IDB 데이터를 동기화하는 IDA Pro 플러그인입니다.
저장할 때마다 이름, 타입, 인라인 함수 및 기타 IDB 데이터를 YAML 형식으로보낸보내고, git으로 커밋·푸시·풀하며, git mergetool로 병합 충돌을 자동 해결합니다.
입력 파일의 MD5로 IDB를 식별해 사용자 간 동기화를 수행하며, 빈번하고 방해가 적은 동기화에 맞게 설계되었습니다.
전체 데이터베이스를 공유하지 않고도 동일한 바이너리에서 협업해야 하는 리버스 엔지니어링 팀에 가벼운 IDB 동기화 수단으로 주로 유용합니다.
