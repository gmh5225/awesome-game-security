Dieses Projekt ist ein IDA-Pro-Plugin, das IDB-Daten zwischen mehreren Reverse Engineers, die am selben Binary arbeiten, über ein gemeinsames Git-Repository synchronisiert.
Bei jedem Speichern exportiert es Namen, Typen, Inline-Funktionen und andere IDB-Daten im YAML-Format, commitet, pusht und pulled Änderungen über Git und löst Merge-Konflikte automatisch mit git mergetool.
Das Plugin identifiziert IDBs anhand der MD5-Prüfsumme der Eingabedatei für die benutzerübergreifende Synchronisation und ist für häufige, unaufdringliche Syncs ausgelegt.
Es eignet sich vor allem für Reverse-Engineering-Teams, die am selben Binary zusammenarbeiten und eine schlanke IDB-Synchronisation ohne vollständige Datenbankfreigabe benötigen.
