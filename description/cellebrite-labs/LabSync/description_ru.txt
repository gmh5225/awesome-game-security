Этот проект — плагин IDA Pro, который синхронизирует данные IDB между несколькими реверс-инженерами, работающими над одним бинарником, через общий git-репозиторий.
При каждом сохранении он экспортирует имена, типы, встроенные функции и прочие данные IDB в формат YAML, затем выполняет commit, push и pull изменений через git с автоматическим разрешением конфликтов слияния через git mergetool.
Плагин идентифицирует IDB по MD5 входного файла для синхронизации между пользователями и рассчитан на частые, ненавязчивые синхронизации.
Он в первую очередь полезен командам реверс-инженерии, которые совместно работают над одним бинарником и нуждаются в лёгкой синхронизации IDB без полного совместного использования базы данных.
