Este proyecto es un plugin de IDA Pro que sincroniza datos IDB entre varios ingenieros inversos que trabajan sobre el mismo binario a través de un repositorio git compartido.
En cada guardado exporta nombres, tipos, funciones en línea y otros datos IDB a formato YAML, y luego hace commit, push y pull de los cambios mediante git, con resolución automática de conflictos de fusión vía git mergetool.
El plugin identifica los IDB por el MD5 del archivo de entrada para la sincronización entre usuarios y está diseñado para sincronizaciones frecuentes y poco intrusivas.
Es especialmente útil para equipos de ingeniería inversa que colaboran en el mismo binario y necesitan una sincronización IDB ligera sin compartir la base de datos completa.
