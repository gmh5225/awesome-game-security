Questo progetto è un plugin di IDA Pro che sincronizza i dati IDB tra più reverse engineer che lavorano sullo stesso binario tramite un repository git condiviso.
A ogni salvataggio esporta nomi, tipi, funzioni inline e altri dati IDB in formato YAML, poi esegue commit, push e pull delle modifiche tramite git, con risoluzione automatica dei conflitti di merge tramite git mergetool.
Il plugin identifica gli IDB tramite l'MD5 del file di input per la sincronizzazione tra utenti ed è progettato per sync frequenti e poco invasive.
È soprattutto utile per team di reverse engineering che collaborano sullo stesso binario e necessitano di una sincronizzazione IDB leggera senza condividere l'intero database.
