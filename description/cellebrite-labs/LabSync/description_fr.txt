Ce projet est un plugin IDA Pro qui synchronise les données IDB entre plusieurs reverse engineers travaillant sur le même binaire via un dépôt git partagé.
À chaque enregistrement, il exporte les noms, types, fonctions en ligne et autres données IDB au format YAML, puis commit, pousse et tire les changements via git, avec résolution automatique des conflits de fusion grâce à git mergetool.
Le plugin identifie les IDB par le MD5 du fichier d'entrée pour la synchronisation multi-utilisateurs et est conçu pour des syncs fréquentes et peu intrusives.
Il est surtout utile aux équipes de reverse engineering qui collaborent sur le même binaire et ont besoin d'une synchronisation IDB légère sans partage complet de la base de données.
