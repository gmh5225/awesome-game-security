Questo progetto è uno strumento di parsing e manipolazione dei file PDB (Program Database) di Windows, realizzato in C++ con il Microsoft DIA (Debug Interface Access) SDK. Fornisce funzionalità per leggere, unire e analizzare i file di simboli PDB, con supporto per i formati PDB precedenti e configurazione personalizzata tramite file pdb.cfg. È pensato soprattutto per reverse engineer e sviluppatori di debugger che necessitano di accesso programmatico alle informazioni sui simboli di debug di Windows.
