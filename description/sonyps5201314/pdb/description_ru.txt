Этот проект — инструмент для разбора и работы с Windows PDB (Program Database), написанный на C++ с использованием Microsoft DIA (Debug Interface Access) SDK. Он позволяет читать, объединять и анализировать файлы символов PDB, поддерживает старые форматы PDB и настройку через файлы pdb.cfg. В первую очередь полезен reverse-инженерам и разработчикам отладчиков, которым нужен программный доступ к отладочной символьной информации Windows.
