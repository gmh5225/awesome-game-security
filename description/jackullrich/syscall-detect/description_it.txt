Questo progetto è una proof of concept Windows per rilevare invocazioni di syscall dirette e indirette da user mode. Usa l'instrumentation callback o l'ispezione dello stack del thread per stabilire se una syscall proviene da ntdll (normale) o da uno stub personalizzato (sospetto), segnalando una possibile evasione dei hook sulle syscall. L'implementazione in C mostra euristiche di rilevamento utilizzabili da prodotti anti-cheat ed EDR. È pensato per ricercatori di sicurezza difensiva che studiano il rilevamento delle syscall dirette e il monitoraggio dell'integrità in user mode.
