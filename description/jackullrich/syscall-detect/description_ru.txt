Этот проект — Windows proof of concept для обнаружения прямых и косвенных вызовов syscall из пользовательского режима. Он использует instrumentation callback или проверку стека потока, чтобы определить, исходит ли syscall из ntdll (нормально) или из пользовательского stub (подозрительно), отмечая возможный обход перехвата syscall. Реализация на C демонстрирует эвристики обнаружения, которые могут применять anti-cheat и EDR. Предназначен для исследователей защитной безопасности, изучающих обнаружение прямых syscall и мониторинг целостности в пользовательском режиме.
