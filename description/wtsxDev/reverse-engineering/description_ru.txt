Этот проект представляет собой тщательно отобранный список полезных ресурсов по реверс-инжинирингу.
Он посвящён реверс-инжинирингу, сетевым технологиям и инструментам для редакторов.
В первую очередь он полезен исследователям безопасности игр и reverse-инженерам, изучающим атакующие техники в области читов / гайдов.
