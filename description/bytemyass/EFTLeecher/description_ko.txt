이 프로젝트는 Escape From Tarkov용 DMA 기반 치트 도구 모음으로, FPGA 하드웨어를 통해 외부에서 게임 메모리를 읽습니다.
바이저 효과 제거, 야간/열화상 시야 효과, 반동 제거, 스태미나 우회, 무게 제거, INI 파일로 설정 가능한 스텔스 인젝션 등의 기능을 포함합니다.
C++ 코드베이스는 DMA 메모리 접근에 MemProcFS를 사용하며, 맵 파일 로드와 운영 보안을 위한 자동 연결 해제를 지원합니다.
주로 EFT에서 외부 게임 메모리 조작과 안티치트 우회 기법을 연구하는 DMA 보안 연구자에게 유용합니다.
