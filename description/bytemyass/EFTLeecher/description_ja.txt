本プロジェクトは Escape From Tarkov 向けの DMA ベースのチートツールボックスで、FPGA ハードウェアを通じて外部からゲームメモリを読み取ります。
バイザー効果の除去、暗視／熱映像効果、リコイル除去、スタミナバイパス、重量除去、INI ファイルで設定可能なステルスインジェクションなどの機能を含みます。
C++ コードベースは DMA メモリアクセスに MemProcFS を使用し、マップファイルの読み込みと運用セキュリティのための自動切断をサポートします。
EFT における外部ゲームメモリ操作とアンチチート回避手法を研究する DMA セキュリティ研究者にとって主に有用です。
