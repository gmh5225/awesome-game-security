Questo progetto è una toolbox di cheat basata su DMA per Escape From Tarkov che legge la memoria di gioco dall’esterno tramite hardware FPGA.
Include funzioni per la rimozione dell’effetto visore, effetti di visione notturna/termica, eliminazione del rinculo, bypass della stamina, rimozione del peso e iniezione stealth con impostazioni configurabili tramite file INI.
Il codice C++ usa MemProcFS per l’accesso alla memoria via DMA e supporta il caricamento di file mappa e la disconnessione automatica per la sicurezza operativa.
È soprattutto utile ai ricercatori di sicurezza DMA che studiano la manipolazione esterna della memoria di gioco e le tecniche di evasione dell’anti-cheat in EFT.
