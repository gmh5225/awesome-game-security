Этот проект — набор DMA-читов для Escape From Tarkov, который внешне читает память игры через FPGA-оборудование.
Включает снятие эффекта визора, эффекты ночного/тепловизионного зрения, устранение отдачи, обход выносливости, снятие веса и скрытую инъекцию с настройками через INI-файл.
Кодовая база на C++ использует MemProcFS для DMA-доступа к памяти и поддерживает загрузку файлов карт и автоматическое отключение в целях операционной безопасности.
В основном полезна исследователям DMA-безопасности, изучающим внешнюю манипуляцию памятью игры и техники обхода античита в EFT.
