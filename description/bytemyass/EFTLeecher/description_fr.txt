Ce projet est une boîte à outils de triche basée sur le DMA pour Escape From Tarkov, qui lit la mémoire du jeu de l’extérieur via un matériel FPGA.
Il inclut la suppression de l’effet de visière, des effets de vision nocturne/thermique, l’élimination du recul, le contournement de l’endurance, la suppression du poids, ainsi qu’une injection furtive configurable via un fichier INI.
La base de code C++ utilise MemProcFS pour l’accès mémoire DMA et prend en charge le chargement de fichiers de carte ainsi que la déconnexion automatique pour la sécurité opérationnelle.
Il est surtout utile aux chercheurs en sécurité DMA qui étudient la manipulation externe de la mémoire de jeu et les techniques d’évasion anti-triche dans EFT.
