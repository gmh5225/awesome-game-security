Este repositorio es una pequeña base de conocimiento para mejorar el comportamiento de máquinas virtuales KVM al ejecutar juegos y software protegido por anti-cheat. Contiene principalmente notas en Markdown y enlaces de referencia sobre el ajuste del controlador ioapic, problemas de split-lock y errores WRMSR o RDMSR no gestionados en hosts como Unraid y Proxmox. En lugar de incluir código ejecutable, se centra en guías de resolución de problemas e indicaciones de configuración para casos límite de virtualización. El público principal son usuarios e investigadores que depuran fallos de arranque de juegos o problemas de rendimiento en entornos virtualizados.
