Этот репозиторий — небольшая база знаний по улучшению поведения виртуальных машин KVM при запуске игр и ПО с античитом. В основном содержит Markdown-заметки и ссылки по настройке драйвера ioapic, проблемам split-lock и необработанным ошибкам WRMSR или RDMSR на хостах вроде Unraid и Proxmox. Вместо исполняемого кода здесь собраны руководства по устранению неполадок и подсказки по конфигурации для пограничных случаев виртуализации. Основная аудитория — пользователи и исследователи, разбирающие сбои запуска игр или проблемы производительности в виртуализированных средах.
