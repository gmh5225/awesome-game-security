Этот проект — частично утёкший архив исходного кода Counter-Strike: Global Offensive (CS:GO) на базе движка Source. Он содержит клиентскую и серверную игровую логику, механику оружия, код движения игрока, сетевое взаимодействие, системы UI и интерфейсы движка. Кодовая база на C++ даёт представление об архитектуре Source и внутренних игровых системах CS:GO. Предназначен для исследователей игровой безопасности и аналитиков движка Source, изучающих внутреннее устройство CS:GO для разработки античита, обнаружения читов и исследования уязвимостей движка.
