Este proyecto es un editor hexadecimal rico en funciones y una plataforma de análisis binario para ingenieros inversos y desarrolladores de bajo nivel.
Ofrece edición avanzada, análisis y visualización impulsados por un lenguaje de patrones, inspección de datos, comparación (diffing), hashing, desensamblado integrado y extensibilidad mediante plugins.
La base de código es principalmente C++, con soporte de escritorio multiplataforma y numerosos flujos de trabajo de análisis integrados.
Se usa ampliamente para investigar firmware, formatos de archivo, memoria y binarios de juegos en contextos de investigación de seguridad.
