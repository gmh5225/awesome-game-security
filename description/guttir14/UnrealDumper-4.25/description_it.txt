Questo progetto è un dumper SDK esterno per Unreal Engine 4.25+ scritto in C++ che estrae definizioni di classi, offset delle proprietà e firme delle funzioni da un processo di gioco UE4 in esecuzione. Individua gli array GObjects e GNames tramite pattern scanning, attraversa la gerarchia UObject e produce un SDK completo con layout delle struct e catene di ereditarietà. Lo strumento opera esternamente senza injection, leggendo la memoria del gioco tramite handle di processo. È rivolto a game hacker e reverse engineer che generano SDK Unreal Engine per lo sviluppo di cheat o l'analisi di giochi.
