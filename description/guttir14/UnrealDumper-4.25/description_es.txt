Este proyecto es un dumper externo de SDK para Unreal Engine 4.25+ escrito en C++ que extrae definiciones de clases, offsets de propiedades y firmas de funciones de un proceso de juego UE4 en ejecución. Localiza los arrays GObjects y GNames mediante pattern scanning, recorre la jerarquía UObject y genera un SDK completo con layouts de estructuras y cadenas de herencia. La herramienta opera de forma externa sin inyección, leyendo la memoria del juego a través de handles de proceso. Está dirigida a game hackers e ingenieros inversos que generan SDKs de Unreal Engine para el desarrollo de cheats o el análisis de juegos.
