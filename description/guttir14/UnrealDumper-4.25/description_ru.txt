Этот проект — внешний дампер SDK для Unreal Engine 4.25+ на C++, который извлекает определения классов, смещения свойств и сигнатуры функций из работающего процесса игры на UE4. Он находит массивы GObjects и GNames с помощью сканирования сигнатур, обходит иерархию UObject и выводит полный SDK с раскладками структур и цепочками наследования. Инструмент работает извне без инъекции, читая память игры через дескрипторы процессов. Предназначен для гейм-хакеров и реверс-инженеров, генерирующих SDK Unreal Engine для разработки читов или анализа игр.
