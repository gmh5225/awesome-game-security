이 프로젝트는 대규모 언어 모델을 리버스 엔지니어링 워크플로에 통합하여 AI 기반 코파일럿으로 동작하는 IDA Pro 플러그인입니다. AI 지원 디컴파일, 도구 호출을 지원하는 대화형 챗봇 인터페이스, 함수 주석, 취약점 분석, 바이너리 데이터베이스에서의 RAG 기반 컨텍스트 검색을 제공합니다. 이 Python 플러그인은 여러 LLM 제공업체를 지원하며 프롬프트 템플릿, 구문 강조, 커스텀 뷰어 UI를 포함합니다. IDA Pro 내에서 LLM으로 강화된 바이너리 분석을 원하는 악성코드 분석가와 리버스 엔지니어에게 주로 유용합니다.
