Этот проект — плагин для IDA Pro, который интегрирует большие языковые модели в процесс реверс-инжиниринга в роли ИИ-помощника. Он обеспечивает ИИ-ассистированную декомпиляцию, интерактивный чат-бот с поддержкой вызова инструментов, аннотирование функций, анализ уязвимостей и RAG-поиск контекста из бинарных баз данных. Python-плагин поддерживает несколько провайдеров LLM и включает шаблоны промптов, подсветку синтаксиса и пользовательский интерфейс просмотра. Он в первую очередь полезен аналитикам вредоносного ПО и специалистам по реверс-инжинирингу, которым нужен бинарный анализ с поддержкой LLM в IDA Pro.
