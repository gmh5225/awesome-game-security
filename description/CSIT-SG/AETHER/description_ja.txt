本プロジェクトは、大規模言語モデルをリバースエンジニアリングのワークフローに統合し、AI 駆動のコパイロットとして機能する IDA Pro プラグインです。AI 支援の逆コンパイル、ツール呼び出し対応の対話型チャットボット、関数アノテーション、脆弱性分析、バイナリデータベースからの RAG ベースのコンテキスト取得を提供します。この Python プラグインは複数の LLM プロバイダーに対応し、プロンプトテンプレート、シンタックスハイライト、カスタムビューア UI を含みます。IDA Pro 内で LLM を活用したバイナリ解析を求めるマルウェアアナリストやリバースエンジニアに主に役立ちます。
