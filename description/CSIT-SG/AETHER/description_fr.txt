Ce projet est un plugin IDA Pro qui intègre les grands modèles de langage dans le flux de travail de rétro-ingénierie en tant que copilote alimenté par l'IA. Il propose une décompilation assistée par l'IA, une interface de chatbot interactive avec appel d'outils, l'annotation de fonctions, l'analyse de vulnérabilités et la récupération de contexte basée sur le RAG à partir de bases de données binaires. Le plugin Python prend en charge plusieurs fournisseurs de LLM et inclut des modèles de prompts, la coloration syntaxique et une interface de visualiseur personnalisée. Il est surtout utile aux analystes de logiciels malveillants et aux ingénieurs en rétro-ingénierie cherchant une analyse binaire augmentée par LLM dans IDA Pro.
