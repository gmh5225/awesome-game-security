Questo progetto è un plugin per IDA Pro che integra i grandi modelli linguistici nel flusso di lavoro di reverse engineering come copilota basato sull'IA. Offre decompilazione assistita dall'IA, un'interfaccia chatbot interattiva con supporto per le chiamate agli strumenti, annotazione delle funzioni, analisi delle vulnerabilità e recupero del contesto basato su RAG da database binari. Il plugin Python supporta più provider LLM e include modelli di prompt, evidenziazione della sintassi e un'interfaccia viewer personalizzata. È soprattutto utile per analisti di malware e reverse engineer che cercano un'analisi binaria potenziata dagli LLM all'interno di IDA Pro.
