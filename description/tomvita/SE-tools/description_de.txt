Dieses Projekt ist SE-tools, eine Sammlung von Nintendo-Switch-Homebrew-Tools zur Bearbeitung von Spielespeicher und zur Entwicklung von Cheats. Es umfasst einen Speicherscanner, einen Pointer-Searcher, einen Cheatcode-Manager und einen Echtzeit-Speicherbetrachter, die auf der Switch über den dmnt:cht-Dienst des Atmosphere-CFW laufen. Die Tools unterstützen das Erstellen und Anwenden von Cheatcodes im Standard-Atmosphere-Cheatformat. Sie richten sich an Switch-Homebrew-Entwickler und Spielmodder, die speicherbasierte Cheats für Nintendo-Switch-Titel erstellen.
