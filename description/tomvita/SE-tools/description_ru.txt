Этот проект — SE-tools, набор homebrew-инструментов для Nintendo Switch, предназначенных для редактирования памяти игр и разработки читов. В него входят сканер памяти, поисковик указателей, менеджер чит-кодов и просмотрщик памяти в реальном времени, работающие на Switch через сервис dmnt:cht CFW Atmosphere. Инструменты поддерживают создание и применение чит-кодов в стандартном формате Atmosphere. Они ориентированы на homebrew-разработчиков Switch и моддеров игр, создающих читы на основе памяти для игр Nintendo Switch.
