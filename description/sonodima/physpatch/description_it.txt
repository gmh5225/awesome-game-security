Questo progetto è PhysPatch, uno strumento per patchare la memoria del kernel Windows tramite accesso alla memoria fisica. Traduce gli indirizzi virtuali in indirizzi fisici tramite il page table walking, poi modifica direttamente le pagine di memoria fisica per aggirare le protezioni di memoria a livello software. Questa tecnica evita gli hook delle API del kernel e il monitoraggio degli accessi in memoria. È rivolta ai ricercatori del kernel che studiano la manipolazione della memoria fisica e le sue implicazioni per l'integrità di anti-cheat e prodotti di sicurezza.
