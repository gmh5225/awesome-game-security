Этот проект — PhysPatch, инструмент для патчинга памяти ядра Windows через доступ к физической памяти. Он преобразует виртуальные адреса в физические путём обхода таблиц страниц, затем напрямую изменяет страницы физической памяти, обходя программную защиту памяти. Этот метод позволяет избежать хуков API ядра и мониторинга доступа к памяти. Предназначен для исследователей ядра, изучающих манипуляции с физической памятью и их влияние на целостность античитов и продуктов безопасности.
