Este proyecto es PhysPatch, una herramienta para parchear la memoria del kernel de Windows mediante acceso a memoria física. Traduce direcciones virtuales a físicas mediante el recorrido de las tablas de páginas y luego modifica directamente las páginas de memoria física para eludir la protección de memoria a nivel de software. Esta técnica evita los hooks de API del kernel y la monitorización de accesos a memoria. Está orientada a investigadores de kernel que estudian la manipulación de memoria física y sus implicaciones para la integridad de anti-cheats y productos de seguridad.
