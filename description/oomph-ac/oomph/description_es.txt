Oomph es un proxy anti-trampas basado en interceptación para Minecraft: Bedrock Edition que se sitúa entre los clientes y el servidor del juego e inspecciona el tráfico en busca de trampas. Escrito principalmente en Go, procesa paquetes de cliente y servidor y aplica detecciones configurables para problemas como reach, abuso de hitbox, asistencia de puntería, autoclickers, kill aura, scaffolding, nukers y paquetes malformados. Ofrece movimiento y combate autoritativos del servidor con corrección de posición consciente de la latencia, rebobinado de entidades para validar ataques y comprobaciones de impacto basadas en raycast para mitigar trampas comunes de Bedrock. El proyecto puede ejecutarse como proxy Bedrock independiente o integrarse con Dragonfly, y se conecta con backends como PocketMine-MP para operadores de seguridad de juegos que protegen servidores Bedrock.
