Este proyecto proporciona firmware FPGA para hardware DMA al estilo PCILeech, usado para emular dispositivos PCIe en pruebas de seguridad.
Está construido principalmente con Verilog y SystemVerilog, junto con núcleos IP de Xilinx Vivado y scripts de compilación específicos de cada placa.
El código incluye lógica para el comportamiento del espacio de configuración PCIe, patrones de manejo de interrupciones y varios perfiles de persona de hardware.
Se usa sobre todo para investigación de evasión de anti-cheat a bajo nivel y para experimentar con técnicas de bypass de detección basadas en DMA.
