Этот проект предоставляет FPGA-прошивку для DMA-оборудования в стиле PCILeech, используемую для эмуляции PCIe-устройств при тестировании безопасности.
Он в основном написан на Verilog и SystemVerilog и включает IP-ядра Xilinx Vivado, а также скрипты сборки для конкретных плат.
В кодовой базе есть логика поведения пространства конфигурации PCIe, шаблоны обработки прерываний и несколько аппаратных persona-профилей.
В первую очередь применяется для низкоуровневых исследований обхода античита и экспериментов с DMA-техниками обхода обнаружения.
