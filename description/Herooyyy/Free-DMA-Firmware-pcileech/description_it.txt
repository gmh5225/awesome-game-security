Questo progetto fornisce firmware FPGA per hardware DMA in stile PCILeech, usato per emulare dispositivi PCIe nei test di sicurezza.
È realizzato principalmente in Verilog e SystemVerilog, con core IP Xilinx Vivado e script di build specifici per scheda.
Il codice include logica per il comportamento dello spazio di configurazione PCIe, pattern di gestione degli interrupt e diversi profili di persona hardware.
Viene usato soprattutto per la ricerca sull’evasione anti-cheat a basso livello e per sperimentare tecniche di bypass della rilevazione basate su DMA.
