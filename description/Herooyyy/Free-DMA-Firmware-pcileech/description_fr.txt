Ce projet fournit un firmware FPGA pour du matériel DMA de type PCILeech, utilisé pour émuler des périphériques PCIe lors de tests de sécurité.
Il est principalement construit en Verilog et SystemVerilog, avec des cœurs IP Xilinx Vivado et des scripts de build spécifiques aux cartes.
Le code inclut la logique du comportement de l’espace de configuration PCIe, des modèles de gestion des interruptions, ainsi que plusieurs profils de persona matérielle.
Il sert surtout à la recherche sur le contournement bas niveau de l’anti-cheat et à l’expérimentation de techniques de bypass de détection basées sur le DMA.
