Este proyecto documenta técnicas de protección y cracking de activadores de firmware DMA para dispositivos FPGA basados en PCILeech. Incluye código fuente SystemVerilog para configuraciones de firmware PCILeech FT601 básicas y avanzadas, código C++ de análisis de señales de activación y documentación sobre registros de señales y flags de detección usados por paneles comerciales de firmware. Resulta útil principalmente para investigadores de seguridad DMA que estudian mecanismos de activación de firmware y protecciones anti-crack en hardware DMA basado en FPGA.
