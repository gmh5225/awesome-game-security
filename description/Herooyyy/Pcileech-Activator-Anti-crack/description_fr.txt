Ce projet documente les techniques de protection et de crack des activateurs de firmware DMA pour les périphériques FPGA basés sur PCILeech. Il inclut du code source SystemVerilog pour les configurations de firmware PCILeech FT601 de base et avancées, du code C++ d'analyse des signaux d'activation, ainsi que de la documentation sur les registres de signaux et les drapeaux de détection utilisés par les panneaux de firmware commerciaux. Il s'adresse principalement aux chercheurs en sécurité DMA qui étudient les mécanismes d'activation de firmware et les protections anti-crack sur le matériel DMA basé sur FPGA.
