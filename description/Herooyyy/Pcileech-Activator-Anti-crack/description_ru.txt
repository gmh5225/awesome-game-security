Этот проект документирует техники защиты и взлома активаторов DMA-прошивок для FPGA-устройств на базе PCILeech. Включает исходный код SystemVerilog для базовых и расширенных конфигураций прошивки PCILeech FT601, код C++ для анализа сигналов активации, а также документацию по сигнальным регистрам и флагам обнаружения, используемым коммерческими панелями прошивок. В первую очередь полезен исследователям безопасности DMA, изучающим механизмы активации прошивок и анти-crack защиту на FPGA-оборудовании DMA.
