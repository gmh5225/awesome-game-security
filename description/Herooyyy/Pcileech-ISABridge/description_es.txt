Este proyecto implementa un perfil de emulación PCIe estilo puente ISA para experimentos de firmware DMA.
Está construido principalmente con Verilog y SystemVerilog, con archivos de proyecto Vivado y salidas de bitstream generadas.
La versión demuestra la suplantación de PID y VID mediante la simulación de un dispositivo puente para estudiar el filtrado de hardware de los anti-cheat.
Está dirigido a investigadores que analizan cómo los sistemas anti-cheat clasifican y bloquean periféricos PCIe sospechosos.
