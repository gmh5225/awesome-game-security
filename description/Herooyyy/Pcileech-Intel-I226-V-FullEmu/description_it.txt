Questo progetto è una release precompilata di emulazione di scheda di rete correlata al DMA che mira al comportamento degli adattatori Intel I225 e I226.
Si concentra sulla generazione di attività TCP sintetica e sul mantenimento di uno stato di link attivo persistente.
Il repository è minimale e fornisce principalmente un payload binario con un breve README esplicativo.
Il caso d'uso principale è la ricerca sull'evasione anti-cheat contro controlli basati sull'attività in sistemi come Vanguard, Faceit ed EAC.
