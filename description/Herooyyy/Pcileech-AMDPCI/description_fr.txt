Ce projet est un profil de firmware DMA qui émule un modèle de périphérique AMD PCI pour le matériel compatible PCILeech.
Il est principalement implémenté en Verilog et SystemVerilog, avec des composants IP Vivado et des scripts de build pour les cartes 35T, 75T et ZDMA.
L’implémentation se concentre sur un comportement de communication sans interruption et sur les concepts d’usurpation d’identité matérielle.
Son cas d’usage principal est les tests de résilience anti-cheat et la recherche sur la détection basée sur les signatures matérielles en sécurité des jeux.
