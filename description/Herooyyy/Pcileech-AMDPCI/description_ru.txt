Этот проект — профиль DMA-прошивки, эмулирующий модель устройства AMD PCI для оборудования, совместимого с PCILeech.
Он в основном реализован на Verilog и SystemVerilog с компонентами Vivado IP и скриптами сборки для плат 35T, 75T и ZDMA.
Реализация сосредоточена на поведении связи без прерываний и на концепциях подмены аппаратной идентичности.
Основной сценарий использования — тестирование устойчивости античита и исследование обнаружения по аппаратным сигнатурам в игровой безопасности.
