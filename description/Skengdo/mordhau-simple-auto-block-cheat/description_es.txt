Este proyecto es un módulo de trampa C++ interno para Mordhau centrado en la defensa y la respuesta automatizadas en combate cuerpo a cuerpo. Implementa funciones como auto-bloqueo de golpes y patadas más lógica opcional de auto-estocada, y está pensado para inyectarse en el proceso del juego. Se incluye un SDK generado de gran tamaño, que muestra una fuerte dependencia de los encabezados de volcado de clases y funciones de Unreal Engine. El público principal son investigadores de game hacking que exploran técnicas de automatización del combate y la estructura de módulos internos.
