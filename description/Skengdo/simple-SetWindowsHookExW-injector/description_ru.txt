Этот проект — инжектор DLL для Windows на C++, построенный вокруг метода внедрения SetWindowsHookExW. Он включает вспомогательные компоненты для разбора portable executable (PE), операций с реестром и опциональных сценариев подмены сертификатов. Процесс использования сводится к выбору payload-DLL, заданию целевого класса окна и запуску инжектора против выбранного процесса. Основной сценарий применения — изучение и тестирование конвейеров user-mode внедрения в контексте исследований безопасности игр.
