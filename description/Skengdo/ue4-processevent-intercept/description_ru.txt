Этот проект — компактная библиотека на C++ для перехвата вызовов ProcessEvent Unreal Engine 4 на выбранных объектах. Она использует VMT shadowing вместо прямой патчинг, а жизненный цикл хуков рассчитан на повторное применение при пересоздании объектов. Приведённый пример показывает, как во время выполнения перехватывать и изменять вызовы функций, связанных с геймплеем. Основной сценарий использования — реверс-инжиниринг UE4 и внутренняя инструментация для исследований в области безопасности игр.
