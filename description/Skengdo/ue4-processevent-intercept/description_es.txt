Este proyecto es una biblioteca C++ compacta para interceptar llamadas ProcessEvent de Unreal Engine 4 en objetos seleccionados. Usa VMT shadowing en lugar de parcheo directo, y el ciclo de vida de los hooks está diseñado para reaplicarse cuando los objetos se recrean. El ejemplo incluido muestra cómo capturar y modificar en tiempo de ejecución llamadas a funciones relacionadas con el gameplay. El caso de uso principal es la ingeniería inversa de UE4 y la instrumentación interna para investigación de seguridad en juegos.
