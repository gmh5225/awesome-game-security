Questo progetto è uno strumento esterno in C++ per eseguire dump, lettura e modifica delle variabili di missione attive in Arma 3. Include gestione della memoria e codice di supporto specifico del gioco per localizzare le variabili e modificare valori come i campi dell’economia del server. Le note d’uso si rivolgono a server con BattlEye disabilitato e indicano che l’uso online richiede una logica di bypass aggiuntiva. Il caso d’uso principale è la ricerca di game hacking e gli esperimenti di tooling sulla manipolazione dei dati runtime di Arma 3.
