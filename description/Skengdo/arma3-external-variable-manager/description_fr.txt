Ce projet est un outil externe en C++ permettant de dumper, lire et modifier les variables de mission actives dans Arma 3. Il inclut une gestion de la mémoire et du code d’aide spécifique au jeu pour localiser les variables et modifier des valeurs telles que les champs d’économie du serveur. Les notes d’utilisation ciblent les serveurs avec BattlEye désactivé et indiquent qu’une utilisation en ligne nécessite une logique de contournement supplémentaire. Le cas d’usage principal est la recherche en game hacking et les expérimentations d’outils autour de la manipulation des données d’exécution d’Arma 3.
