Этот проект — внешний инструмент на C++ для дампа, чтения и изменения активных переменных миссии в Arma 3. Он включает управление памятью и игровой вспомогательный код для поиска переменных и изменения значений, например полей серверной экономики. Примечания по использованию ориентированы на серверы с отключённым BattlEye и указывают, что для онлайн-работы нужна дополнительная логика обхода. Основной сценарий — исследования game hacking и эксперименты с инструментами по манипуляции runtime-данными Arma 3.
