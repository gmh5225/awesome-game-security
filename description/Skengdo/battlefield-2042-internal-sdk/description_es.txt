Este proyecto es un andamiaje de SDK C++ interno para trabajo de ingeniería inversa de Battlefield 2042. Expone estructuras y utilidades para entidades, jugadores, vehículos, armas, el contexto del juego, acceso al renderizado y cálculos de world-to-screen. El repositorio está marcado explícitamente como incompleto, con notas sobre detalles aún no resueltos de la iteración de la lista de entidades. Su principal caso de uso es apoyar herramientas internas, depuración y prototipos de investigación en seguridad de juegos.
