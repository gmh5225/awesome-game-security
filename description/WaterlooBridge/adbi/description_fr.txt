Ce projet est une boîte à outils d'instrumentation binaire dynamique pour les cibles Android ARM et Thumb.
Il combine un injecteur hijack avec une bibliothèque de hooking de base pour charger du code d'instrumentation dans des processus en cours d'exécution et accrocher en ligne les points d'entrée de fonctions.
Le code et le flux de compilation s'appuient sur le C et le C++ avec les outils Android NDK, ainsi que des instruments d'exemple qui illustrent la journalisation à l'exécution et l'interception de comportements.
Il est destiné à la rétro-ingénierie mobile, à l'analyse à l'exécution et à la recherche en sécurité sur les applications Android.
