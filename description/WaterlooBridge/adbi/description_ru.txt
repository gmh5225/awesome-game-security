Этот проект — набор инструментов динамической бинарной инструментации для целей Android ARM и Thumb.
Он сочетает hijack-инжектор с базовой библиотекой хукинга, чтобы загружать код инструментации в работающие процессы и перехватывать точки входа функций inline.
Код и процесс сборки опираются на C и C++, а также инструменты Android NDK; включены примеры инструментов, демонстрирующие журналирование во время выполнения и перехват поведения.
Предназначен для мобильного реверс-инжиниринга, анализа во время выполнения и исследований безопасности приложений Android.
