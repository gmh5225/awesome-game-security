Este proyecto es un kit de herramientas de instrumentación binaria dinámica para objetivos Android ARM y Thumb.
Combina un inyector hijack con una biblioteca base de hooking para cargar código de instrumentación en procesos en ejecución y enganchar en línea los puntos de entrada de funciones.
El código y el flujo de compilación se basan en C y C++ con las herramientas Android NDK, además de instrumentos de ejemplo que demuestran el registro en tiempo de ejecución y la interceptación de comportamiento.
Está pensado para ingeniería inversa móvil, análisis en tiempo de ejecución e investigación de seguridad en aplicaciones Android.
