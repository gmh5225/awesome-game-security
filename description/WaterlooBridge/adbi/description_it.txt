Questo progetto è un toolkit di strumentazione binaria dinamica per target Android ARM e Thumb.
Combina un iniettore hijack con una libreria di base per l'hooking per caricare codice di strumentazione nei processi in esecuzione e agganciare inline i punti di ingresso delle funzioni.
Il codice e il flusso di build si basano su C e C++ con gli strumenti Android NDK, oltre a strumenti di esempio che dimostrano logging a runtime e intercettazione del comportamento.
È pensato per reverse engineering mobile, analisi a runtime e ricerca sulla sicurezza delle applicazioni Android.
