本プロジェクトは Socket.IO であり、Node.js およびブラウザ向けのリアルタイム双方向イベント通信ライブラリです。主要なトランスポートとして WebSocket を使用し、必要に応じて HTTP ロングポーリングへ自動フォールバックすることで、自動再接続、名前空間による多重化、ルームベースのブロードキャスト、バイナリデータ対応などの機能を備えた信頼性の高いリアルタイムメッセージングを提供します。この JavaScript/TypeScript ライブラリはサーバーとクライアントの両方のコンポーネントを含みます。マルチプレイヤーゲーム、チャットシステム、ライブダッシュボードなどのリアルタイムアプリケーションを構築する Web 開発者を対象としています。
