Этот проект — Socket.IO, библиотека двусторонней событийной связи в реальном времени для Node.js и браузеров. В качестве основного транспорта используется WebSocket с автоматическим переходом на HTTP long-polling, обеспечивая надёжную передачу сообщений в реальном времени с функциями автопереподключения, мультиплексирования через пространства имён, широковещательной рассылки по комнатам и поддержки бинарных данных. Библиотека на JavaScript/TypeScript включает серверные и клиентские компоненты. Она предназначена для веб-разработчиков, создающих приложения реального времени: многопользовательские игры, чаты и живые панели мониторинга.
