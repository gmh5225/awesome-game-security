Este proyecto se centra en una cadena de exploits de userland para iOS 18.4-18.6.2 con inyección de JS en SpringBoard y otros procesos.
Es una modificación de JS en tiempo de ejecución mediante una cadena de exploits.
Es principalmente útil para investigadores de seguridad en juegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el área de cheat / jailbreak de iOS.
