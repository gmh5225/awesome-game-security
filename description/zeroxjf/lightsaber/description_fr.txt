Ce projet se concentre sur une chaîne d'exploits userland pour iOS 18.4-18.6.2 avec injection JS dans SpringBoard et d'autres processus.
Il s'agit d'une modification JS à l'exécution via une chaîne d'exploits.
Il est principalement utile aux chercheurs en sécurité des jeux et aux ingénieurs en reverse engineering qui étudient les techniques offensives dans le domaine du cheat / jailbreak iOS.
