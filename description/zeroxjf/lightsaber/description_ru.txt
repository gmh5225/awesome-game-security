Этот проект посвящён цепочке userland-эксплойтов для iOS 18.4-18.6.2 с JS-инъекцией в SpringBoard и другие процессы.
Это модификация JS во время выполнения через цепочку эксплойтов.
Он в основном полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области читов / джейлбрейка iOS.
