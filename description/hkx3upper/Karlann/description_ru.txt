Этот проект — proof-of-concept драйвера ядра Windows, реализующий инъекцию клавиатурного ввода и сетевую связь Winsock Kernel (WSK) из режима ядра. Драйвер на C включает симуляцию клавиатуры через Kbd.c и операции с сетевыми сокетами через Wsk.c с вспомогательной библиотекой libwsk. Он в первую очередь полезен исследователям безопасности ядра, изучающим техники инъекции ввода в режиме ядра и сетевой связи WSK.
