Questo progetto è una proof-of-concept di un driver del kernel Windows che implementa l'iniezione di input da tastiera e la comunicazione di rete Winsock Kernel (WSK) in modalità kernel. Il driver in C include la simulazione della tastiera tramite Kbd.c e le operazioni sui socket di rete tramite Wsk.c con una libreria di supporto libwsk. È soprattutto utile per i ricercatori di sicurezza del kernel che studiano le tecniche di iniezione di input in modalità kernel e di comunicazione di rete WSK.
