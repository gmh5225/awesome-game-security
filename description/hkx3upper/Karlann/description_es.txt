Este proyecto es una prueba de concepto de un controlador del kernel de Windows que implementa inyección de entrada de teclado y comunicación de red Winsock Kernel (WSK) desde el modo kernel. El controlador en C incluye simulación de teclado mediante Kbd.c y operaciones de sockets de red mediante Wsk.c con una biblioteca auxiliar libwsk. Resulta especialmente útil para investigadores de seguridad del kernel que estudian técnicas de inyección de entrada en modo kernel y comunicación de red WSK.
