Ce projet est une preuve de concept de pilote noyau Windows qui implémente l'injection d'entrées clavier et la communication réseau Winsock Kernel (WSK) depuis le mode noyau. Le pilote en C inclut la simulation clavier via Kbd.c et les opérations de sockets réseau via Wsk.c avec une bibliothèque d'aide libwsk. Il s'adresse principalement aux chercheurs en sécurité noyau qui étudient les techniques d'injection d'entrées en mode noyau et de communication réseau WSK.
