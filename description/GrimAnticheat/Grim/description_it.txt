GrimAC è un anticheat Minecraft open source progettato per versioni server moderne e combinazioni di protocolli. È costruito principalmente in Java e Kotlin e implementa una simulazione del movimento dettagliata, la replica del mondo e una logica di validazione consapevole della latenza. L'architettura è fortemente asincrona e multithread per scalare i controlli riducendo i falsi positivi nei casi limite di gameplay complessi. Il caso d'uso principale è il rilevamento di cheat lato server per le community Minecraft che necessitano di una protezione accurata e attivamente mantenuta.
