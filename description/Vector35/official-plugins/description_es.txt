Este proyecto es el índice oficial de plugins de Binary Ninja mantenido por el equipo principal.
Almacena metadatos estructurados de los plugins compatibles, incluidas descripciones, tipos de plugin, destinos de API y detalles de licencia.
Scripts de indexación en Python generan y actualizan las salidas del catálogo usadas en flujos de gestión de plugins.
Se orienta principalmente a usuarios de Binary Ninja que buscan integraciones de plugins de confianza, de primera parte o respaldadas.
