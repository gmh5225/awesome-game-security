Questo progetto è l’indice ufficiale curato dei plugin Binary Ninja mantenuto dal team centrale.
Conserva metadati strutturati per i plugin supportati, inclusi descrizioni, tipi di plugin, target API e dettagli di licenza.
Script di indicizzazione in Python generano e aggiornano gli output del catalogo usati nei flussi di gestione dei plugin.
È pensato soprattutto per gli utenti di Binary Ninja che cercano integrazioni di plugin affidabili, first-party o approvate.
