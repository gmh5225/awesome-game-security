Этот проект — официальный курируемый индекс плагинов Binary Ninja, поддерживаемый основной командой.
В нём хранятся структурированные метаданные поддерживаемых плагинов: описания, типы, целевые API и сведения о лицензиях.
Скрипты индексации на Python формируют и обновляют каталоги, используемые в рабочих процессах управления плагинами.
Он в основном предназначен для пользователей Binary Ninja, которым нужны доверенные интеграции first-party или одобренных плагинов.
