Este proyecto es un plugin de flujo de trabajo de análisis de Objective-C para Binary Ninja, ahora migrado al repositorio principal de la API.
Está implementado en C++ y amplía el análisis limpiando patrones de llamadas de Objective-C relacionados con el despacho dinámico.
Una función clave es reescribir muchas llamadas al estilo objc_msgSend en representaciones de llamadas directas más claras cuando se pueden inferir los destinos.
Está pensado para ingenieros inversos que analizan binarios de macOS e iOS que dependen en gran medida del comportamiento del runtime de Objective-C.
