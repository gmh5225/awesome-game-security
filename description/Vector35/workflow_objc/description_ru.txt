Этот проект — плагин рабочего процесса анализа Objective-C для Binary Ninja, который теперь перенесён в основной репозиторий API.
Он реализован на C++ и расширяет анализ, упорядочивая шаблоны вызовов Objective-C, связанные с динамической диспетчеризацией.
Ключевая возможность — переписывание многих вызовов в стиле objc_msgSend в более понятные представления прямых вызовов, когда цели можно вывести.
Плагин предназначен для реверс-инженеров, анализирующих бинарные файлы macOS и iOS, которые сильно зависят от поведения среды выполнения Objective-C.
