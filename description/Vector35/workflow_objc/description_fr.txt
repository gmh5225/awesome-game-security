Ce projet est un plugin de workflow d'analyse Objective-C pour Binary Ninja, désormais migré vers le dépôt principal de l'API.
Il est implémenté en C++ et étend l'analyse en nettoyant les motifs d'appels Objective-C liés au dispatch dynamique.
Une fonctionnalité clé consiste à réécrire de nombreux appels de style objc_msgSend en représentations d'appels directs plus claires lorsque les cibles peuvent être déduites.
Il s'adresse aux reverse engineers qui analysent des binaires macOS et iOS fortement dépendants du comportement du runtime Objective-C.
