Questo progetto è un plugin di workflow di analisi Objective-C per Binary Ninja, ora migrato nel repository principale dell'API.
È implementato in C++ ed estende l'analisi ripulendo i pattern di chiamata Objective-C legati al dispatch dinamico.
Una funzionalità chiave è riscrivere molte chiamate in stile objc_msgSend in rappresentazioni di chiamate dirette più chiare quando è possibile inferire i target.
È pensato per reverse engineer che analizzano binari macOS e iOS fortemente dipendenti dal comportamento del runtime Objective-C.
