Questo progetto è un plugin di Binary Ninja per il program slicing, pensato per accelerare la comprensione del codice durante il reverse engineering.
È scritto principalmente in Python e si integra con le viste dell’interfaccia di Binary Ninja e con le rappresentazioni in linguaggio intermedio.
Il plugin aiuta a esaminare le relazioni tra variabili e blocchi tramite slice mirate, invece della complessità di funzioni intere.
Il pubblico di riferimento sono analisti e ricercatori di vulnerabilità che devono navigare più rapidamente binari complessi.
