Этот проект — плагин Binary Ninja для program slicing, ускоряющий понимание кода при реверс-инжиниринге.
Он написан в основном на Python и интегрируется с представлениями UI Binary Ninja и представлениями на промежуточном языке.
Плагин помогает изучать связи переменных и блоков через сфокусированные срезы вместо сложности целых функций.
Целевая аудитория — аналитики и исследователи уязвимостей, которым нужна более быстрая навигация по сложным бинарникам.
