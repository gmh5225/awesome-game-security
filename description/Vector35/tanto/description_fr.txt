Ce projet est un plugin Binary Ninja de program slicing destiné à accélérer la compréhension du code lors de la reverse engineering.
Il est principalement écrit en Python et s’intègre aux vues de l’interface Binary Ninja ainsi qu’aux représentations en langage intermédiaire.
Le plugin aide à examiner les relations entre variables et blocs via des tranches ciblées plutôt que la complexité d’une fonction entière.
Il s’adresse aux analystes et chercheurs en vulnérabilités qui ont besoin de naviguer plus rapidement dans des binaires complexes.
