Ce projet est un compilateur de shellcode open source, d’abord développé comme outil CTF interne puis intégré à Binary Ninja.
Il est principalement écrit en C/C++ et construit avec CMake.
Le compilateur vise une sortie compacte orientée shellcode et fournit une documentation dédiée pour l’utilisation et les builds.
Il est surtout utile pour le développement d’exploits et les flux de reverse engineering qui exigent une génération contrôlée de code bas niveau.
