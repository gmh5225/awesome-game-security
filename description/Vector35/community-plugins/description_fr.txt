Ce projet est l’index des plugins communautaires pour Binary Ninja.
Il agrège les métadonnées des plugins tiers, les informations de licence et de compatibilité, ainsi que l’historique des mises à jour, dans des listes lisibles par machine.
Le dépôt s’appuie sur des outils Python pour valider les manifests de plugins, générer les fichiers d’index et automatiser la maintenance du catalogue.
Son usage principal est d’aider les reverse engineers à découvrir, installer et gérer les extensions Binary Ninja contribuées par la communauté.
