Questo progetto è l’indice dei plugin della community per Binary Ninja.
Aggrega metadati di plugin di terze parti, informazioni su licenze e compatibilità, nonché la cronologia degli aggiornamenti, in elenchi leggibili dalle macchine.
Il repository usa strumenti Python per validare i manifest dei plugin, generare i file di indice e automatizzare la manutenzione del catalogo.
Il caso d’uso principale è aiutare i reverse engineer a scoprire, installare e gestire le estensioni Binary Ninja contribuite dalla community.
