Dieses Projekt ist ein Binary-Ninja-Plugin, das undurchsichtige Prädikate (opaque predicates) aus verschleierten Binärdateien automatisch entfernt.
Es ist in Python implementiert und analysiert MLIL-Verzweigungsbedingungen, um konstant wahre oder falsche Pfade zu erkennen.
Anschließend patcht das Plugin Verzweigungsinstruktionen so, dass sie immer oder nie springen, und führt die Analyse in iterativen Durchläufen erneut aus.
Es richtet sich an Reverse Engineers, die eine schnellere Entschleierung von geschütztem oder absichtlich verwirrendem Code benötigen.
