Ce projet est un plugin Binary Ninja qui supprime automatiquement les prédicats opaques des binaires obscurcis.
Il est implémenté en Python et analyse les conditions de branchement MLIL pour détecter les chemins constamment vrais ou faux.
Le plugin corrige ensuite les instructions de branchement pour toujours ou jamais brancher, puis relance l'analyse en passes itératives.
Il est conçu pour les ingénieurs en reverse engineering qui ont besoin d'un désobscurcissement plus rapide de code protégé ou volontairement confus.
