Este proyecto es un complemento de Binary Ninja que elimina automáticamente los predicados opacos de binarios ofuscados.
Está implementado en Python y analiza las condiciones de ramificación MLIL para detectar rutas constantemente verdaderas o falsas.
El complemento luego parchea las instrucciones de ramificación para que siempre o nunca ramifiquen y vuelve a ejecutar el análisis en pasadas iterativas.
Está diseñado para ingenieros inversos que necesitan una desofuscación más rápida de código protegido o intencionalmente confuso.
