Questo progetto è un plugin di Binary Ninja che rimuove automaticamente i predicati opachi dai binari offuscati.
È implementato in Python e analizza le condizioni di branch MLIL per rilevare percorsi costantemente veri o falsi.
Il plugin poi patcha le istruzioni di branch affinché eseguano sempre o mai il salto e riesegue l'analisi in passate iterative.
È pensato per reverse engineer che necessitano di una deoffuscazione più rapida di codice protetto o intenzionalmente confuso.
