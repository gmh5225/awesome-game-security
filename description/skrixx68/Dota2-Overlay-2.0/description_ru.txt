Этот проект — репозиторий из раздела cheat / game:dota2, посвящённый Dota2 Overlay 2.0.
Он в основном написан на C/C++ и C++ и сосредоточен на оверлеях.
Он прежде всего полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / game:dota2.
