Questo progetto è una proof of concept di clonazione di processi Windows scritta in C che crea una copia di un processo in esecuzione usando NtCreateProcessEx con l'handle del processo padre. Il processo clonato eredita uno snapshot dello spazio di indirizzamento virtuale dell'originale, consentendo tecniche come process hollowing, analisi della memoria o dumping delle credenziali dal clone senza accedere direttamente al target. È rivolto ai ricercatori di sicurezza che studiano le tecniche di manipolazione dei processi e le relative opportunità di rilevamento.
