Этот проект — доказательство концепции клонирования процессов Windows на языке C, создающее копию работающего процесса с помощью NtCreateProcessEx и дескриптора родительского процесса. Клонированный процесс наследует снимок виртуального адресного пространства оригинала, что позволяет применять такие техники, как process hollowing, анализ памяти или дамп учётных данных из клона без прямого доступа к цели. Предназначен для исследователей безопасности, изучающих техники манипуляции процессами и возможности их обнаружения.
