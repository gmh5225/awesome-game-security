Ce projet est une preuve de concept de clonage de processus Windows écrite en C, qui crée une copie d'un processus en cours d'exécution via NtCreateProcessEx avec le handle du processus parent. Le processus cloné hérite d'un instantané de l'espace d'adressage virtuel de l'original, ce qui permet des techniques telles que le process hollowing, l'analyse mémoire ou l'extraction d'identifiants depuis le clone sans accéder directement à la cible. Il s'adresse aux chercheurs en sécurité qui étudient les techniques de manipulation de processus et les opportunités de détection associées.
