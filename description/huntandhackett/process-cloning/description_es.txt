Este proyecto es una prueba de concepto de clonación de procesos de Windows escrita en C que crea una copia de un proceso en ejecución mediante NtCreateProcessEx con el handle del proceso padre. El proceso clonado hereda una instantánea del espacio de direcciones virtual del original, lo que permite técnicas como process hollowing, análisis de memoria o volcado de credenciales desde el clon sin acceder directamente al objetivo. Está dirigido a investigadores de seguridad que estudian técnicas de manipulación de procesos y sus oportunidades de detección.
