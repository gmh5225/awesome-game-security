Fork del motor 3D GamePlay v3.0 que reemplaza el backend de renderizado OpenGL 2.x original por la abstracción gráfica multiplataforma BGFX, habilitando el soporte para múltiples APIs gráficas (DirectX, Vulkan, Metal, OpenGL). Construido sobre SDL2, el motor proporciona un pipeline de renderizado diferido, grafo de escena, integración de física Bullet, sistemas de partículas, interfaz de editor basada en ImGui y compilación de shaders multiplataforma mediante el lenguaje de shaders de BGFX.
