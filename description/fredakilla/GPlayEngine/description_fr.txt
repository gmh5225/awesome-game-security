Fork du moteur 3D GamePlay v3.0 qui remplace le backend de rendu OpenGL 2.x d'origine par l'abstraction graphique multiplateforme BGFX, permettant la prise en charge de plusieurs API graphiques (DirectX, Vulkan, Metal, OpenGL). Construit sur SDL2, le moteur offre un pipeline de rendu différé, un graphe de scène, l'intégration de la physique Bullet, des systèmes de particules, une interface d'éditeur basée sur ImGui, ainsi que la compilation de shaders multiplateforme via le langage de shaders de BGFX.
