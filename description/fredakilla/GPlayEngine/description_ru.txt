Форк 3D-движка GamePlay v3.0, заменяющий исходный бэкенд рендеринга OpenGL 2.x на кроссплатформенную графическую абстракцию BGFX и обеспечивающий поддержку нескольких графических API (DirectX, Vulkan, Metal, OpenGL). Построенный на SDL2, движок предоставляет отложенный рендеринг, граф сцены, интеграцию физики Bullet, системы частиц, редакторский интерфейс на ImGui и кроссплатформенную компиляцию шейдеров через шейдерный язык BGFX.
