Fork der GamePlay-3D-Engine v3.0, der das ursprüngliche OpenGL-2.x-Rendering-Backend durch die plattformübergreifende Grafikabstraktion BGFX ersetzt und so Unterstützung für mehrere Grafik-APIs (DirectX, Vulkan, Metal, OpenGL) ermöglicht. Auf SDL2 aufbauend bietet die Engine eine Deferred-Rendering-Pipeline, einen Szenengraphen, Bullet-Physik-Integration, Partikelsysteme, eine ImGui-basierte Editor-Oberfläche sowie plattformübergreifende Shader-Kompilierung über die Shader-Sprache von BGFX.
