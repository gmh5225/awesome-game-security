Fork del motore 3D GamePlay v3.0 che sostituisce il backend di rendering OpenGL 2.x originale con l'astrazione grafica multipiattaforma BGFX, abilitando il supporto per più API grafiche (DirectX, Vulkan, Metal, OpenGL). Basato su SDL2, il motore offre una pipeline di deferred rendering, scene graph, integrazione della fisica Bullet, sistemi di particelle, interfaccia editor basata su ImGui e compilazione di shader multipiattaforma tramite il linguaggio shader di BGFX.
