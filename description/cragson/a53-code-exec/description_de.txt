Dieses Projekt konzentriert sich auf einen PoC zur Codeausführung auf a53, wie er von der PS5 unter Firmware 02.00 genutzt wird.
Es ist hauptsächlich in C++ und C/C++ geschrieben und dreht sich um Kernel-Arbeit sowie die SDK-Generierung.
Es ist vor allem für Konsolen-Emulator-Entwickler und PlayStation-Forscher im PlayStation-Bereich nützlich.
