Questo progetto si concentra su un PoC di esecuzione di codice su a53 usato dalla PS5 con firmware 02.00.
È scritto principalmente in C++ e C/C++ e ruota attorno al lavoro a livello kernel e alla generazione di SDK.
È soprattutto utile per sviluppatori di emulatori per console e ricercatori PlayStation che lavorano nell'area playstation.
