Una raccolta di script di utilità e strumenti per l'infrastruttura di analisi malware, tra cui configurazione della virtualizzazione KVM/QEMU, script di deoffuscamento delle stringhe per IDA Pro, plugin Volatility3 e helper per la gestione dei database.
Include deoffuscamento basato su flare-emu, container Docker libguestfs e script di configurazione Windows comunemente usati negli ambienti sandbox CAPE/Cuckoo.
È soprattutto utile ad analisti malware e ricercatori di sicurezza che impostano ambienti di analisi automatizzati ed eseguono il deoffuscamento di binari.
