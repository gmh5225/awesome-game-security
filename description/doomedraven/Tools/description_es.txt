Una colección de scripts de utilidad y herramientas para infraestructura de análisis de malware, que incluye configuración de virtualización KVM/QEMU, scripts de desofuscación de cadenas para IDA Pro, plugins de Volatility3 y ayudas para la gestión de bases de datos.
Incluye desofuscación basada en flare-emu, contenedores Docker de libguestfs y scripts de configuración de Windows usados habitualmente en entornos sandbox CAPE/Cuckoo.
Resulta especialmente útil para analistas de malware e investigadores de seguridad que montan entornos de análisis automatizados y realizan desofuscación de binarios.
