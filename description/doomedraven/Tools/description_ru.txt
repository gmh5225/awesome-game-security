Набор служебных скриптов и инструментов для инфраструктуры анализа вредоносного ПО, включая настройку виртуализации KVM/QEMU, скрипты деобфускации строк для IDA Pro, плагины Volatility3 и вспомогательные средства для управления базами данных.
Включает деобфускацию на базе flare-emu, Docker-контейнеры libguestfs и скрипты конфигурации Windows, часто используемые в песочницах CAPE/Cuckoo.
В основном полезен аналитикам вредоносного ПО и исследователям безопасности, которые разворачивают автоматизированные среды анализа и выполняют деобфускацию бинарных файлов.
