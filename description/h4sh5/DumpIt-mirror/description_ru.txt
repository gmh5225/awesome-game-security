Этот проект — зеркало DumpIt, инструмента Comae для снятия дампа памяти в Windows. DumpIt одним щелчком создаёт полный дамп физической памяти работающей системы Windows в формате raw или Microsoft crash dump, пригодный для анализа в Volatility, WinDbg и подобных средствах. Он ориентирован на специалистов по реагированию на инциденты и форензик-аналитиков, которым нужна портативная и простая утилита для live-захвата памяти в ходе расследований.
