Dieses Projekt ist ein Spiegel von Comaes DumpIt, einem Tool zur Speicherakquise unter Windows. DumpIt erstellt mit einem Klick einen vollständigen physischen Speicherdump eines laufenden Windows-Systems und liefert eine Datei im Raw- oder Microsoft-Crash-Dump-Format, die sich mit Tools wie Volatility oder WinDbg analysieren lässt. Es richtet sich an Incident Responder und forensische Analysten, die ein portables, einfach bedienbares Utility zur Live-Erfassung von Speicher während Untersuchungen benötigen.
