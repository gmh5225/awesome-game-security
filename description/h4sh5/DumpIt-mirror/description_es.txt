Este proyecto es un espejo de DumpIt, la herramienta de adquisición de memoria de Comae para Windows. DumpIt crea con un solo clic un volcado completo de la memoria física de un sistema Windows en ejecución, en formato raw o Microsoft crash dump, apto para el análisis con herramientas como Volatility o WinDbg. Está pensado para respondedores de incidentes y analistas forenses que necesitan una utilidad portátil y fácil de usar para capturar memoria en vivo durante investigaciones.
