Este proyecto es un cargador de hipervisor UEFI al estilo Voyager modificado y un framework de payload para sistemas Intel y AMD. Incluye manejadores VM-exit independientes y primitivas de memoria para la traducción física y virtual del huésped, operaciones de lectura y escritura, e inicialización de tablas de páginas. El código también contiene componentes UEFI que enganchan etapas de arranque de Windows como bootmgfw y winload, y parchean rutas relacionadas con Hyper-V antes del inicio del SO. Está orientado a la investigación avanzada de bajo nivel sobre virtualización en el arranque, control del kernel y técnicas de bypass de anti-cheat.
