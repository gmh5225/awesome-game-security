Этот проект — модифицированный загрузчик гипервизора UEFI в стиле Voyager и фреймворк payload для систем Intel и AMD. Он включает отдельные обработчики VM-exit и примитивы памяти для трансляции гостевых физических и виртуальных адресов, операций чтения и записи, а также инициализации таблиц страниц. Кодовая база также содержит компоненты UEFI, которые перехватывают этапы загрузки Windows (например, bootmgfw и winload) и патчат пути, связанные с Hyper-V, до запуска ОС. Проект предназначен для продвинутых низкоуровневых исследований виртуализации на этапе загрузки, контроля ядра и техник обхода античита.
