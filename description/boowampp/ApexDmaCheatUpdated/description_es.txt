Este proyecto es un cheat externo basado en DMA para Apex Legends que usa el framework PCILeech/MemProcFS para el acceso remoto a memoria mediante hardware FPGA. La base de código en C++ incluye aimbot con FOV y suavizado configurables, compensación de retroceso, renderizado ESP, cálculos de cámara y una biblioteca de memoria DMA con gestión de entrada, acceso al registro y soporte para inyección de shellcode. Resulta especialmente útil para investigadores de seguridad DMA que estudian el acceso a memoria de juegos basado en hardware y técnicas de evasión anti-cheat.
