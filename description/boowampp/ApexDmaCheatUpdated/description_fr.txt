Ce projet est un cheat externe basé sur le DMA pour Apex Legends, qui utilise le framework PCILeech/MemProcFS pour l’accès mémoire distant via du matériel FPGA. La base de code C++ inclut un aimbot avec FOV et lissage configurables, une compensation du recul, le rendu ESP, des calculs de caméra, ainsi qu’une bibliothèque mémoire DMA avec gestion des entrées, accès au registre et prise en charge de l’injection de shellcode. Il s’adresse principalement aux chercheurs en sécurité DMA qui étudient l’accès mémoire aux jeux via le matériel et les techniques d’évasion anti-cheat.
