Ce projet est un outil open source de test anti-analyse qui émule les techniques de détection utilisées par les vrais malwares. Écrit en C, il comprend des modules qui recherchent les machines virtuelles, les sandboxes, les débogueurs, les hooks et les artefacts d'environnement sur des plateformes telles que VMware, VirtualBox, QEMU et Wine. Le code est destiné à des tests reproductibles et peut être compilé avec MinGW-w64 et des workflows basés sur make. Il est principalement utilisé par les chercheurs en sécurité pour évaluer les environnements d'analyse et étudier les comportements d'évasion.
