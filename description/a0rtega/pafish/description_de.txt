Dieses Projekt ist ein Open-Source-Anti-Analyse-Testwerkzeug, das Erkennungstechniken nachbildet, wie sie von echter Malware eingesetzt werden. Es ist in C geschrieben und enthält Module zur Prüfung auf virtuelle Maschinen, Sandboxes, Debugger, Hooks und Umgebungsartefakte auf Plattformen wie VMware, VirtualBox, QEMU und Wine. Der Code ist für reproduzierbare Tests gedacht und lässt sich mit MinGW-w64 sowie make-basierten Workflows bauen. Sicherheitsforscher nutzen es vor allem, um Analyseumgebungen zu bewerten und Ausweichverhalten zu untersuchen.
