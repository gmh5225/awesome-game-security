Este proyecto es una herramienta de código abierto para pruebas anti-análisis que emula técnicas de detección usadas por malware real. Está escrita en C e incluye módulos que comprueban máquinas virtuales, sandboxes, depuradores, hooks y artefactos del entorno en plataformas como VMware, VirtualBox, QEMU y Wine. El código está pensado para pruebas reproducibles y se puede compilar con MinGW-w64 y flujos de trabajo basados en make. La usan principalmente investigadores de seguridad para evaluar entornos de análisis y estudiar el comportamiento de evasión.
