Ce projet est un framework de virtualisation d'applications Android permettant d'exécuter et de gérer des applications clonées dans un conteneur isolé. Il est principalement écrit en Java avec des ressources Android XML, ainsi que des composants natifs en C et C++ pour les hooks bas niveau et l'interception à l'exécution. La base de code intègre la compatibilité Xposed et des modules d'instrumentation basés sur SandHook, ainsi que la gestion virtuelle des packages, processus et composants. Il vise le reverse engineering mobile, l'analyse du comportement des applications et les cas d'usage de sandbox multi-instances.
