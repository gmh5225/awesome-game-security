Этот проект — фреймворк виртуализации Android-приложений для запуска и управления клонированными приложениями в изолированном контейнере. Он в основном написан на Java с ресурсами Android XML, а также включает нативные компоненты на C и C++ для низкоуровневых хуков и перехвата во время выполнения. Кодовая база интегрирует совместимость с Xposed и модули инструментирования на базе SandHook, а также управление виртуальными пакетами, процессами и компонентами. Нацелен на мобильный реверс-инжиниринг, анализ поведения приложений и сценарии многоинстансной песочницы.
