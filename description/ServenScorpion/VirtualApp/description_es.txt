Este proyecto es un framework de virtualización de aplicaciones Android para ejecutar y gestionar apps clonadas en un contenedor aislado. Está escrito principalmente en Java con recursos Android XML, junto con componentes nativos en C y C++ para hooks de bajo nivel e interceptación en tiempo de ejecución. El código integra compatibilidad con Xposed y módulos de instrumentación basados en SandHook, además de la gestión virtual de paquetes, procesos y componentes. Está orientado a la ingeniería inversa móvil, el análisis del comportamiento de apps y casos de uso de sandbox multiinstancia.
