Этот проект — KCP, быстрая и надёжная реализация протокола ARQ (Automatic Repeat reQuest) на C, обеспечивающая TCP-подобную надёжность поверх UDP. По сравнению с TCP она обменивает 10–20% дополнительной полосы пропускания на снижение задержки на 30–40% за счёт агрессивной повторной передачи, селективного ACK и быстрого восстановления без накладных расходов контроля перегрузки TCP. Однофайловая библиотека на C независима от протокола и работает поверх любого транспортного уровня. Она предназначена для разработчиков игр и сетевых программистов, создающих низколатентный real-time мультиплеер.
