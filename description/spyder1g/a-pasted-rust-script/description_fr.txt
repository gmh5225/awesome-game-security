Ce projet est un cheat de jeu en Rust avec un chargeur basé sur ImGui, doté de protections anti-débogage, d'utilitaires d'encodage Base64 et d'un système de modules de fonctionnalités. La base de code C++ comprend une solution Visual Studio avec le rendu de menu, des contrôles anti-débogage et des implémentations de fonctionnalités de cheat. Il est principalement utile aux chercheurs en sécurité des jeux qui étudient l'architecture des chargeurs de cheat et les techniques anti-débogage de base dans les outils de modification de jeux.
