Этот проект — игровой чит на Rust с загрузчиком на базе ImGui, включающий защиту от отладки, утилиты кодирования Base64 и систему модулей функций. Кодовая база на C++ содержит решение Visual Studio с отрисовкой меню, проверками anti-debug и реализацией функций чита. Он в первую очередь полезен исследователям безопасности игр, изучающим архитектуру загрузчиков читов и базовые методы защиты от отладки в инструментах модификации игр.
