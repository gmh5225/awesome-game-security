dnSpy プロジェクトがメンテナンスする Mono Soft Debugger クライアントライブラリ（Mono.Debugger.Soft）のフォークで、Unity および Mono ランタイム向けの高度なデバッグ機能をサポートするよう改変されています。
Mono の soft debugger エージェントと通信するためのワイヤプロトコル実装および mirror API を提供し、マネージドコードにおけるブレークポイント、ステップ実行、オブジェクト検査を可能にします。
主に、Unity および Mono ベースのゲームアプリケーションをマネージドコードレベルでデバッグするリバースエンジニアやゲームモデラー向けに有用です。
