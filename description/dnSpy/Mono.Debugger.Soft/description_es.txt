Un fork de la biblioteca cliente de Mono Soft Debugger (Mono.Debugger.Soft) mantenido por el proyecto dnSpy, con modificaciones para mejorar la depuración de runtimes Unity y Mono.
Proporciona la implementación del protocolo de cable y la API mirror para comunicarse con el agente soft debugger de Mono, permitiendo puntos de interrupción, ejecución paso a paso e inspección de objetos en código administrado.
Resulta especialmente útil para ingenieros inversos y modders de juegos que depuran aplicaciones Unity y Mono a nivel de código administrado.
