Un fork della libreria client Mono Soft Debugger (Mono.Debugger.Soft) mantenuto dal progetto dnSpy, con modifiche per supportare capacità avanzate di debugging dei runtime Unity e Mono.
Fornisce l'implementazione del protocollo wire e l'API mirror per comunicare con l'agente soft debugger di Mono, abilitando breakpoint, stepping e ispezione degli oggetti nel codice managed.
È utile soprattutto a reverse engineer e modder di giochi che debuggano applicazioni Unity e Mono a livello di codice managed.
