Форк клиентской библиотеки Mono Soft Debugger (Mono.Debugger.Soft), поддерживаемый проектом dnSpy и доработанный для расширенной отладки runtime Unity и Mono.
Предоставляет реализацию wire-протокола и mirror API для взаимодействия с агентом soft debugger Mono, что позволяет ставить точки останова, выполнять пошаговую отладку и инспектировать объекты в управляемом коде.
В первую очередь полезен реверс-инженерам и моддерам игр, отлаживающим приложения на базе Unity и Mono на уровне управляемого кода.
