Un fork de la bibliothèque cliente Mono Soft Debugger (Mono.Debugger.Soft) maintenu par le projet dnSpy, modifié pour mieux prendre en charge le débogage des runtimes Unity et Mono.
Il fournit l'implémentation du protocole filaire et l'API mirror pour communiquer avec l'agent soft debugger de Mono, permettant les points d'arrêt, le pas à pas et l'inspection d'objets dans le code managé.
Il est surtout utile aux reverse engineers et aux moddeurs de jeux qui déboguent des applications Unity et Mono au niveau du code managé.
