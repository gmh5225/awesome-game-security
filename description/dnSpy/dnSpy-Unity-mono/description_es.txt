Este proyecto proporciona fuentes de Unity Mono parcheadas y herramientas de compilación usadas por dnSpy para producir builds de mono.dll / mono-2.0-bdwgc.dll con soporte de soft-debugger habilitado.
Incluye un ayudante umpatcher y orientación para levantar runtimes Mono con depuración en distintas versiones de Unity.
Resulta especialmente útil para ingenieros inversos e investigadores de trampas que depuran juegos Unity administrados con dnSpy.
