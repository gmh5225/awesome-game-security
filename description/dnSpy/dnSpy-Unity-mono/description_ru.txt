Этот проект предоставляет пропатченные исходники Unity Mono и инструменты сборки, которые dnSpy использует для получения сборок mono.dll / mono-2.0-bdwgc.dll с поддержкой soft-debugger.
В него входят вспомогательная утилита umpatcher и рекомендации по развёртыванию отладочных runtime Mono для разных версий Unity.
Он в первую очередь полезен реверс-инженерам и исследователям читов, отлаживающим управляемые игры Unity с помощью dnSpy.
