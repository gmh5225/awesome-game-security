Questo progetto fornisce sorgenti Unity Mono patchate e strumenti di build usati da dnSpy per produrre build di mono.dll / mono-2.0-bdwgc.dll con supporto soft-debugger abilitato.
Include un helper umpatcher e indicazioni per configurare runtime Mono debuggabili su diverse versioni di Unity.
È soprattutto utile per reverse engineer e ricercatori di cheat che eseguono il debug di giochi Unity managed con dnSpy.
