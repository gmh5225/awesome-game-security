Ce projet est un modèle de module Zygisk pour injecter des overlays ImGui dans des jeux Android basés sur Unity. Il utilise Zygisk (le framework d'injection Zygote de Magisk) pour charger du code natif dans le processus du jeu au démarrage, accroche le pipeline de rendu Unity et affiche un menu de mod basé sur ImGui. Le module C++ gère le partage de contexte OpenGL ES et la traduction des entrées tactiles. Il s'adresse aux moddeurs de jeux Android qui construisent des menus de mod visuels pour des appareils rootés exécutant des jeux Unity.
