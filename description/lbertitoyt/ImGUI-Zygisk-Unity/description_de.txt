Dieses Projekt ist eine Zygisk-Modulvorlage zum Injizieren von ImGui-Overlays in Unity-basierte Android-Spiele. Es nutzt Zygisk (Magisks Zygote-Injektionsframework), um nativen Code beim Start in den Spielprozess zu laden, hookt die Unity-Rendering-Pipeline und rendert ein ImGui-basiertes Mod-Menü-Overlay. Das C++-Modul übernimmt das Teilen des OpenGL-ES-Kontexts und die Übersetzung von Touch-Eingaben. Es richtet sich an Android-Spielmodder, die visuelle Mod-Menüs für gerootete Geräte mit Unity-Spielen erstellen.
