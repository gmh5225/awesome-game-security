Este proyecto es una plantilla de módulo Zygisk para inyectar overlays de ImGui en juegos Android basados en Unity. Usa Zygisk (el framework de inyección Zygote de Magisk) para cargar código nativo en el proceso del juego al inicio, engancha el pipeline de renderizado de Unity y muestra un menú de mods basado en ImGui. El módulo en C++ gestiona el uso compartido del contexto OpenGL ES y la traducción de la entrada táctil. Está orientado a modders de juegos Android que construyen menús visuales de mods para dispositivos rootados con juegos Unity.
