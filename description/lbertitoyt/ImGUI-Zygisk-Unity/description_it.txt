Questo progetto è un template di modulo Zygisk per iniettare overlay ImGui in giochi Android basati su Unity. Usa Zygisk (il framework di iniezione Zygote di Magisk) per caricare codice nativo nel processo del gioco all'avvio, intercetta la pipeline di rendering di Unity e disegna un menu mod basato su ImGui. Il modulo C++ gestisce la condivisione del contesto OpenGL ES e la traduzione dell'input touch. È rivolto ai modder di giochi Android che creano menu mod visuali per dispositivi rootati con giochi Unity.
