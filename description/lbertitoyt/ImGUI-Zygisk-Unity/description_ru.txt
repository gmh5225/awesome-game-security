Этот проект — шаблон модуля Zygisk для внедрения оверлеев ImGui в Android-игры на Unity. Он использует Zygisk (фреймворк инъекции Zygote от Magisk), чтобы загружать нативный код в процесс игры при запуске, перехватывать конвейер рендеринга Unity и отрисовывать меню модов на базе ImGui. Модуль на C++ отвечает за совместное использование контекста OpenGL ES и преобразование сенсорного ввода. Предназначен для моддеров Android-игр, создающих визуальные меню модов для рутованных устройств с играми на Unity.
