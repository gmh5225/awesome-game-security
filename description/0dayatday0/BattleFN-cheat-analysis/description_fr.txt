Ce projet est un package d'analyse technique centré sur une implémentation de cheat Fortnite détectée. Il contient une analyse écrite, des modules de cheat exemples, ainsi que des outils de preuve de concept pour le mappage manuel de pilotes et l'accès mémoire à des processus privilégiés. Les composants pratiques sont implémentés avec des scripts batch et du code C++ afin de démontrer les chemins d'interaction du mode utilisateur vers le mode noyau. Il s'adresse principalement aux chercheurs en anti-cheat et aux ingénieurs en rétro-ingénierie qui étudient les techniques et les erreurs des cheats réels.
