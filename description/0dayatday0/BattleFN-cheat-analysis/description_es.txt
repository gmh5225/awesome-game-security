Este proyecto es un paquete de análisis técnico centrado en una implementación de cheat de Fortnite detectada. Contiene un análisis escrito, módulos de cheat de ejemplo y herramientas de prueba de concepto para el mapeo manual de controladores y el acceso a memoria de procesos privilegiados. Los componentes prácticos se implementan con scripts batch y código C++ para demostrar las rutas de interacción del modo usuario al modo kernel. Está dirigido principalmente a investigadores de anti-cheat e ingenieros de ingeniería inversa que estudian las técnicas y errores de cheats del mundo real.
