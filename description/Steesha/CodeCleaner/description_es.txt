Este proyecto es un complemento de x64dbg que limpia y simplifica el código máquina desensamblado.
Está escrito en C++ y usa Capstone para el análisis de instrucciones junto con AsmJit para la lógica orientada a transformaciones.
El complemento se centra en eliminar patrones basura como NOP redundantes y movimientos de registros sin efecto, para que las trazas sean más fáciles de leer.
Su caso de uso principal es la ingeniería inversa de binarios empaquetados u ofuscados en la investigación de seguridad de juegos.
