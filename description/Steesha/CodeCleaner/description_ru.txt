Этот проект — плагин для x64dbg, который очищает и упрощает дизассемблированный машинный код.
Он написан на C++ и использует Capstone для анализа инструкций вместе с AsmJit для логики, ориентированной на преобразования.
Плагин сосредоточен на удалении мусорных шаблонов вроде лишних NOP и бессмысленных перемещений регистров, чтобы трассы было легче читать.
Основной сценарий использования — реверс-инжиниринг упакованных или обфусцированных бинарников в исследованиях игровой безопасности.
