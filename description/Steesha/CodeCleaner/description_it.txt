Questo progetto è un plugin per x64dbg che pulisce e semplifica il codice macchina disassemblato.
È scritto in C++ e usa Capstone per l'analisi delle istruzioni insieme ad AsmJit per la logica orientata alle trasformazioni.
Il plugin si concentra sulla rimozione di pattern spazzatura come NOP ridondanti e spostamenti di registri senza effetto, per rendere le tracce più leggibili.
Il caso d'uso principale è il reverse engineering di binari packed o offuscati nella ricerca sulla sicurezza dei giochi.
