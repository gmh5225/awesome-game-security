Ce projet est un plugin x64dbg qui nettoie et simplifie le code machine désassemblé.
Il est écrit en C++ et utilise Capstone pour l'analyse des instructions, ainsi qu'AsmJit pour la logique orientée transformation.
Le plugin se concentre sur la suppression des motifs parasites tels que les NOP redondants et les déplacements de registres sans effet, afin de rendre les traces plus lisibles.
Son cas d'usage principal est l'ingénierie inverse de binaires packés ou obscurcis dans la recherche en sécurité des jeux.
