Dieses Projekt bietet einen kurzen Leitfaden zum Extrahieren und Ändern von Assets oder Code aus Spielen, die mit der Unity-Engine erstellt wurden.
Im Mittelpunkt stehen Asset-Pipelines, Modding und Unity.
Es ist vor allem für Spiele-Security-Forscher und Reverse Engineers nützlich, die offensive Techniken im Bereich Cheats / Guides untersuchen.
