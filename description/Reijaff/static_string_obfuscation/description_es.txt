Este proyecto demuestra la ofuscación estática de cadenas en tiempo de compilación con Zig.
Genera claves aleatorias durante el build y aplica una lógica de transformación basada en XOR para que las cadenas en texto plano no se almacenen directamente en los binarios.
El pipeline de build apunta a ejecutables de Windows x86_64 sin símbolos y enfatiza el descifrado ligero en tiempo de ejecución.
Es útil para experimentos de resistencia a la ingeniería inversa y para el endurecimiento anti-análisis en software orientado a la seguridad.
