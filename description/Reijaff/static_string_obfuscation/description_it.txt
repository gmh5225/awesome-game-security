Questo progetto dimostra l'offuscamento statico delle stringhe a tempo di compilazione con Zig.
Genera chiavi casuali durante il build e applica una logica di trasformazione basata su XOR, così che le stringhe in chiaro non siano memorizzate direttamente nei binari.
La pipeline di build punta a eseguibili Windows x86_64 strippati e pone l'accento su una decrittazione leggera a runtime.
È utile per esperimenti di resistenza al reverse engineering e per l'indurimento anti-analisi in software orientato alla sicurezza.
