Этот проект демонстрирует статическую обфускацию строк на этапе компиляции с помощью Zig.
Он генерирует случайные ключи во время сборки и применяет логику преобразования на основе XOR, чтобы строки в открытом виде не хранились напрямую в бинарных файлах.
Сборочный конвейер ориентирован на stripped-исполняемые файлы Windows x86_64 и делает акцент на лёгкой расшифровке во время выполнения.
Проект полезен для экспериментов по устойчивости к реверс-инжинирингу и для антианализационного усиления ПО, ориентированного на безопасность.
