This project is PCILeech FPGA firmware that emulates a C-Media CMI8738/PCI-SX sound card so Artix-7–based DMA boards appear to the host as a legacy PCI audio device. Written mainly in SystemVerilog and Verilog for Xilinx Vivado, it presents authentic config space and a full BAR0 register map captured from real hardware, with a null audio endpoint that advances position counters and fires interrupts without generating audio memory-read TLPs. An optional fake DMA generator can emit periodic MRd traffic that mirrors real audio bus behavior while keeping the PCILeech DMA path isolated. It targets hardware emulation research, OS driver analysis, PCIe protocol study, and related DMA/anti-cheat research on boards such as Captain DMA, Enigma x1, Squirrel, Screamer M.2, and TBX4.
