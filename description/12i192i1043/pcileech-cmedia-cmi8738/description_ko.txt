이 프로젝트는 C-Media CMI8738/PCI-SX 사운드 카드를 에뮬레이트하는 PCILeech FPGA 펌웨어로, Artix-7 기반 DMA 보드가 호스트에 레거시 PCI 오디오 장치로 보이게 합니다. 주로 SystemVerilog와 Verilog로 Xilinx Vivado용으로 작성되었으며, 실제 하드웨어에서 캡처한 구성 공간과 전체 BAR0 레지스터 맵을 제공하고, 위치 카운터를 진행하고 인터럽트를 발생시키되 오디오 메모리 읽기 TLP는 생성하지 않는 null 오디오 엔드포인트를 포함합니다. 선택적 가짜 DMA 생성기는 실제 오디오 버스 동작을 모방하는 주기적 MRd 트래픽을 내보내면서 PCILeech DMA 경로를 분리된 상태로 유지할 수 있습니다. 하드웨어 에뮬레이션 연구, OS 드라이버 분석, PCIe 프로토콜 학습, 그리고 Captain DMA, Enigma x1, Squirrel, Screamer M.2, TBX4 같은 보드에서의 관련 DMA/안티치트 연구를 대상으로 합니다.
