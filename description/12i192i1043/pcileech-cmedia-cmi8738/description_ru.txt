Этот проект — прошивка FPGA для PCILeech, эмулирующая звуковую карту C-Media CMI8738/PCI-SX, чтобы платы DMA на базе Artix-7 выглядели для хоста как устаревшее PCI-аудиоустройство. Написана в основном на SystemVerilog и Verilog для Xilinx Vivado; предоставляет подлинное пространство конфигурации и полную карту регистров BAR0, снятую с реального оборудования, с нулевым аудио-эндпоинтом, который продвигает счётчики позиции и генерирует прерывания, не создавая TLP чтения аудио-памяти. Опциональный генератор ложного DMA может периодически выдавать трафик MRd, имитирующий поведение реальной аудиошины, сохраняя путь DMA PCILeech изолированным. Предназначен для исследований аппаратной эмуляции, анализа драйверов ОС, изучения протокола PCIe и связанного DMA/античит-исследования на платах вроде Captain DMA, Enigma x1, Squirrel, Screamer M.2 и TBX4.
