Ce projet est une collection de plugins d'architecture Binary Ninja pour analyser les fichiers de classes Java, les binaires Renesas H8/300 et les cibles ELF Xtensa.
Il est principalement écrit en Python et inclut le désassemblage, le décodage d'instructions et un support partiel de lifting pour des architectures personnalisées.
Le module orienté Java prend également en charge des workflows de patch, comme le NOP d'instructions et la modification de la logique de branchement.
Il s'adresse aux reverse engineers qui ont besoin d'extensions rapides dans Binary Ninja pour l'analyse de firmware et de bytecode.
