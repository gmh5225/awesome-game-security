Este proyecto es una colección de plugins de arquitectura de Binary Ninja para analizar archivos de clase Java, binarios Renesas H8/300 y objetivos ELF Xtensa.
Está escrito principalmente en Python e incluye desensamblado, decodificación de instrucciones y soporte parcial de lifting para arquitecturas personalizadas.
El módulo centrado en Java también admite flujos de trabajo orientados a parches, como hacer NOP de instrucciones y cambiar la lógica de saltos.
Está pensado para ingenieros inversos que necesitan soporte de extensión rápida en Binary Ninja para el análisis de firmware y bytecode.
