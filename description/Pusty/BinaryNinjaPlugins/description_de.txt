Dieses Projekt ist eine Sammlung von Binary-Ninja-Architektur-Plugins zur Analyse von Java-Class-Dateien, Renesas-H8/300-Binärdateien und Xtensa-ELF-Zielen.
Es ist hauptsächlich in Python geschrieben und umfasst Disassemblierung, Instruktionsdekodierung sowie teilweise Lifting-Unterstützung für benutzerdefinierte Architekturen.
Das Java-orientierte Modul unterstützt außerdem patch-orientierte Workflows wie das NOPen von Instruktionen und das Ändern von Verzweigungslogik.
Es richtet sich an Reverse Engineers, die in Binary Ninja schnelle Erweiterungsunterstützung für Firmware- und Bytecode-Analyse benötigen.
