Questo progetto è una raccolta di plugin di architettura per Binary Ninja per analizzare file di classe Java, binari Renesas H8/300 e target ELF Xtensa.
È scritto principalmente in Python e include disassemblaggio, decodifica delle istruzioni e supporto parziale di lifting per architetture personalizzate.
Il modulo orientato a Java supporta anche flussi di lavoro basati su patch, come il NOP delle istruzioni e la modifica della logica di branch.
È pensato per reverse engineer che necessitano di un supporto di estensione rapido in Binary Ninja per l'analisi di firmware e bytecode.
