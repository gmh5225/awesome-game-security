Этот проект — набор архитектурных плагинов Binary Ninja для анализа Java class-файлов, бинарников Renesas H8/300 и ELF-целей Xtensa.
Он написан в основном на Python и включает дизассемблирование, декодирование инструкций и частичную поддержку lifting для пользовательских архитектур.
Модуль, ориентированный на Java, также поддерживает рабочие процессы с патчами, например NOP инструкций и изменение логики ветвлений.
Он предназначен для реверс-инженеров, которым нужна быстрая поддержка расширений в Binary Ninja для анализа прошивок и байткода.
