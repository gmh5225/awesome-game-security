Ce projet est un package source de cheat en C++ destiné à améliorer le gameplay de Call of Duty via une visualisation externe et une assistance au ciblage.
Il contient du code de rendu d’overlay, la gestion des données joueurs et entités, ainsi que des routines de communication couramment utilisées pour des lectures mémoire via pilote.
La base de code inclut aussi des composants modèles basés sur Direct3D9 et ImGui pour des interfaces de menu et de style ESP.
Il s’adresse principalement aux apprenants en game hacking et aux praticiens de la rétro-ingénierie qui étudient les pipelines d’overlay et d’interaction mémoire sous Windows.
