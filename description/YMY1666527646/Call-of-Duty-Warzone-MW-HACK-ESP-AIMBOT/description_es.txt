Este proyecto es un paquete de código fuente de cheat en C++ para mejorar el gameplay de Call of Duty mediante visualización externa y asistencia de apuntado.
Contiene código de renderizado de overlay, manejo de datos de jugadores y entidades, y rutinas de comunicación habituales para lecturas de memoria respaldadas por controlador.
La base de código también incluye componentes plantilla basados en Direct3D9 e ImGui para interfaces de menú y estilo ESP.
Está pensado principalmente para aprendices de game hacking y profesionales de ingeniería inversa que estudian pipelines de overlay e interacción con memoria en Windows.
