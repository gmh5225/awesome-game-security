Phoenix-Valorant-Cheat — это внешний чит-проект для Valorant с упором на ESP, аимбот и функции в стиле rage mode.
Он в основном реализован на C++ с решением Visual Studio и использует OpenGL/GLFW вместе с ImGui для отрисовки экранного меню и оверлея.
В кодовой базе есть определения оффсетов, утилиты векторной математики и логика внешнего взаимодействия с памятью, типичная для инструментов game hacking под Windows.
Основное назначение — исследования в области безопасности игр и эксперименты с архитектурой внешних читов в шутерной среде Riot.
