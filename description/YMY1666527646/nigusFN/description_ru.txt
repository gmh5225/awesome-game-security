nigusFN — исходный код внешнего чита для Fortnite, воссоздающий интерфейс и набор функций в стиле nigusFN.
Проект написан на C++ для Windows и использует DirectX9 и ImGui для отрисовки меню, а также включает дополнительные драйверные и вспомогательные компоненты.
В структуре присутствуют определения предметов, заголовочные файлы конфигурации и логика меню, типичные для фреймворков внешних читов.
Репозиторий в первую очередь полезен исследователям безопасности игр, изучающим дизайн UI внешних читов и рабочие процессы инструментов, ориентированных на EAC.
