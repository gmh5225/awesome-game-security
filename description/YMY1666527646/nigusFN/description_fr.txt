nigusFN est un code source de cheat externe pour Fortnite qui recrée l'interface et l'ensemble de fonctionnalités de style nigusFN.
Le projet est écrit en C++ pour Windows et utilise DirectX9 et ImGui pour le rendu des menus, avec des composants supplémentaires de pilote et d'utilitaires.
Sa structure comprend des définitions d'objets, des en-têtes de configuration et une logique de menu typiquement utilisées dans les frameworks de cheats externes.
Le dépôt est surtout utile aux chercheurs en sécurité des jeux qui étudient la conception d'interfaces de cheats externes et les workflows d'outils orientés EAC.
