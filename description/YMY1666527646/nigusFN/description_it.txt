nigusFN è un codice sorgente di cheat esterno per Fortnite che ripropone l'interfaccia e il set di funzionalità in stile nigusFN.
Il progetto è scritto in C++ per Windows e usa DirectX9 e ImGui per il rendering del menu, con componenti aggiuntivi di driver e utility.
La struttura include definizioni degli oggetti, header di configurazione e logica del menu tipicamente impiegate nei framework di cheat esterni.
Il repository è soprattutto utile ai ricercatori di sicurezza nei giochi che studiano il design delle UI dei cheat esterni e i flussi di lavoro di tooling orientati a EAC.
