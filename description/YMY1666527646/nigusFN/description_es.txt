nigusFN es un código fuente de cheat externo para Fortnite que recrea la interfaz y el conjunto de funciones al estilo nigusFN.
El proyecto está escrito en C++ para Windows y usa DirectX9 e ImGui para el renderizado del menú, junto con componentes adicionales de controlador y utilidades.
Su estructura incluye definiciones de objetos, cabeceras de configuración y lógica de menú típicas de los frameworks de cheats externos.
El repositorio resulta especialmente útil para investigadores de seguridad en juegos que estudian el diseño de UI de cheats externos y flujos de trabajo de herramientas orientadas a EAC.
