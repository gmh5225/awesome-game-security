Этот проект — фреймворк для скриптинга и моддинга игр на Unreal Engine 4 и 5. Он объединяет Lua API для скриптов, C++ API для модов, загрузку Blueprint-модов, живой просмотр и редактирование свойств, а также несколько дамперов данных. Кодовая база построена на современном C++ и включает инструменты для runtime-хуков и генерации SDK/заголовков. Предназначен для продвинутой разработки модов, runtime-анализа и исследований по реверс-инжинирингу игр Unreal.
