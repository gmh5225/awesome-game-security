Ce projet est un framework de scripting et de modding pour les jeux Unreal Engine 4 et 5. Il combine une API de scripting Lua, une API de modding C++, le chargement de mods Blueprint, l’inspection et l’édition live des propriétés, ainsi que plusieurs dumpers de données. La base de code s’appuie sur le C++ moderne, avec des outils pour les hooks runtime et la génération de SDK/en-têtes. Il est conçu pour le développement avancé de mods, l’analyse runtime et la recherche en rétro-ingénierie de jeux Unreal.
