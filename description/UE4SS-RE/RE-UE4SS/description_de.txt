Dieses Projekt ist ein Scripting- und Modding-Framework für Unreal Engine 4- und 5-Spiele. Es kombiniert eine Lua-Scripting-API, eine C++-Modding-API, das Laden von Blueprint-Mods, Live-Inspektion und -Bearbeitung von Properties sowie mehrere Daten-Dumper. Die Codebasis ist auf modernem C++ aufgebaut und bietet unterstützende Tools für Runtime-Hooks sowie SDK-/Header-Generierung. Es ist für fortgeschrittene Mod-Entwicklung, Runtime-Analyse und Reverse-Engineering-Forschung an Unreal-Spielen ausgelegt.
