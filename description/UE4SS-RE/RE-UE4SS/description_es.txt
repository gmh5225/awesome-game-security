Este proyecto es un framework de scripting y modding para juegos de Unreal Engine 4 y 5. Combina una API de scripting Lua, una API de modding en C++, carga de mods de Blueprint, inspección y edición en vivo de propiedades, y varios dumpers de datos. El código se centra en C++ moderno, con herramientas de apoyo para hooks en tiempo de ejecución y generación de SDK/cabeceras. Está pensado para el desarrollo avanzado de mods, el análisis en tiempo de ejecución y la investigación de ingeniería inversa de juegos Unreal.
