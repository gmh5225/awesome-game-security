Este proyecto es una estación de trabajo moderna de edición y reverse engineering de bytecode Java para objetivos JVM y Android.
Está implementado en Java y ofrece múltiples decompiladores, herramientas de ensamblado de bytecode, soporte de recompilación y búsqueda profunda en clases, constantes y patrones de instrucciones.
También incluye flujos de trabajo orientados a la desofuscación, como el manejo automático de entradas malformadas, asistencia para el renombrado y utilidades de transformación.
El público principal son reverse engineers, investigadores de seguridad y desarrolladores que analizan o modifican software Java compilado.
