Este proyecto es un shooter en primera persona multijugador de código abierto basado en el motor CUBE. La base de código está principalmente en C y C++ e incluye amplios recursos del juego, mapas, scripts y herramientas de lanzamiento multiplataforma para Windows, Linux y macOS. Destaca por su rendimiento ligero, el juego en red de baja latencia, la edición de mapas integrada, el soporte de bots, las demos y varios modos de juego competitivos. El repositorio resulta especialmente útil para desarrolladores de juegos e investigadores de seguridad que necesitan una base de código FPS completa para estudiar el motor, crear mods o experimentar con la jugabilidad y el anti-trampas.
