Этот проект — консольный сетевой инструмент для формирования, инъекции и сниффинга широкого спектра сетевых протоколов.
Он предназначен для пользователей, которым нужен надёжный инструмент для детального анализа и манипуляции протоколами — для диагностики сети, тестирования или обучения.
В первую очередь полезен исследователям игровой безопасности и reverse-инженерам, изучающим атакующие техники в области читов / снифферов и фильтров пакетов.
