Este proyecto es una herramienta de red de línea de comandos para crear, inyectar y capturar (sniffing) una amplia variedad de protocolos de red.
Está pensada para usuarios que necesitan una herramienta sólida para el análisis y la manipulación detallada de protocolos, ya sea para diagnóstico, pruebas o fines educativos.
Resulta especialmente útil para investigadores de seguridad en videojuegos e ingenieros de ingeniería inversa que estudian técnicas ofensivas en el ámbito de cheats / sniffers y filtros de paquetes.
