Android-приложение, использующее уязвимость Dirty Pipe (CVE-2022-0847) для получения временного root одним нажатием на устройствах Pixel 6 с уязвимыми версиями ядра; объединяет проверку уязвимости и нативный эксплойт на C в удобном приложении, которое перезаписывает файлы только для чтения через повреждение page cache каналов Linux.
