Applicazione Android che sfrutta la vulnerabilità Dirty Pipe (CVE-2022-0847) per ottenere root temporaneo con un clic su dispositivi Pixel 6 con versioni del kernel vulnerabili, integrando un controllo delle vulnerabilità e un exploit nativo in C in un'app semplice da usare che sovrascrive file di sola lettura attraverso la corruzione della page cache delle pipe Linux.
