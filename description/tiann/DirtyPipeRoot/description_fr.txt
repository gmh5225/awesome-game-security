Application Android qui exploite la vulnérabilité Dirty Pipe (CVE-2022-0847) pour obtenir un root temporaire en un clic sur les appareils Pixel 6 exécutant des versions de noyau vulnérables, regroupant un vérificateur de vulnérabilité et un exploit natif en C dans une application conviviale qui écrase des fichiers en lecture seule via la corruption du cache de pages des pipes Linux.
