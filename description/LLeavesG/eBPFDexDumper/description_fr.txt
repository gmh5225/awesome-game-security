eBPFDexDumper est un outil de dumping DEX en mémoire pour Android qui utilise des sondes eBPF afin de capturer le code à l'exécution avec un flux de travail peu intrusif.
Le projet est principalement écrit en Go et cible les appareils Android ARM64 rootés, avec filtrage par UID ou nom de package.
Il peut diffuser des traces d'exécution de méthodes, dumper des fichiers DEX à partir de l'activité ART, et réparer automatiquement les fichiers dumpés pour faciliter l'analyse.
Il est destiné à la rétro-ingénierie Android et à la recherche en sécurité des jeux mobiles lorsque le code chargé dynamiquement doit être récupéré.
