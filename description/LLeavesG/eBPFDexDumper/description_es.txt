eBPFDexDumper es una herramienta de dumping DEX en memoria para Android que usa sondas eBPF para capturar código en tiempo de ejecución con un flujo de trabajo de baja intrusión.
El proyecto está escrito principalmente en Go y se orienta a dispositivos Android ARM64 con root, con filtrado por UID o nombre de paquete.
Puede transmitir trazas de ejecución de métodos, volcar archivos DEX a partir de la actividad de ART y reparar automáticamente los archivos volcados para facilitar el análisis.
Está pensado para ingeniería inversa en Android e investigación de seguridad de juegos móviles donde debe recuperarse código cargado dinámicamente.
