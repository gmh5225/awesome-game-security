eBPFDexDumper — инструмент дампа DEX из памяти Android, который с помощью eBPF-зондов с низким уровнем вмешательства захватывает код во время выполнения.
Проект в основном написан на Go и рассчитан на рутованные ARM64-устройства Android с фильтрацией по UID или имени пакета.
Он может потоково передавать трассировки выполнения методов, дампить DEX-файлы из активности ART и автоматически восстанавливать дампы для упрощения анализа.
Предназначен для реверс-инжиниринга Android и исследований безопасности мобильных игр, где необходимо восстановить динамически загружаемый код.
