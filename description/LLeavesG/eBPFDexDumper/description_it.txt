eBPFDexDumper è uno strumento di dumping DEX in memoria per Android che usa sonde eBPF per catturare il codice a runtime con un flusso di lavoro a bassa intrusione.
Il progetto è scritto principalmente in Go e punta a dispositivi Android ARM64 rootati, con filtraggio per UID o nome del package.
Può trasmettere in streaming le tracce di esecuzione dei metodi, dumpare file DEX dall'attività ART e riparare automaticamente i file dumpati per facilitare l'analisi.
È pensato per il reverse engineering su Android e la ricerca sulla sicurezza dei giochi mobile dove è necessario recuperare codice caricato dinamicamente.
