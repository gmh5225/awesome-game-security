Anti-AntiDebuggerDriver è un driver kernel Windows didattico incentrato sulla neutralizzazione dei controlli anti-debugging comuni. È scritto in C++ e intercetta più percorsi di system call native legati a query su processi, thread, handle e informazioni di sistema. Il codice include utility di hook a basso livello e helper di disassemblaggio per reindirizzare le sonde anti-debug a livello kernel. Il caso d’uso principale è la formazione sul reverse engineering e l’analisi di software protetto.
