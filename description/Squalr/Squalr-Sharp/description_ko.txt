Squalr는 데스크톱 게임용 치트를 만들고 공유하기 위한 고성능 Windows 메모리 편집 애플리케이션입니다. 주로 C#/.NET으로 작성되었으며, WPF GUI, CLI, 재사용 가능한 엔진 API를 통해 메모리 스캔, 포인터 해석, x86/x64 어셈블리 인젝션을 제공합니다. 멀티스레딩과 SIMD(SSE, AVX 또는 AVX-512)로 빠른 스캔을 구현하며, NASM 기반 어셈블/디스어셈블과 C# 스크립팅으로 심층 분석과 자동화를 지원합니다. 프로세스 연결, 메모리 보기, 디버깅, .NET 객체 검사 모듈도 포함되어 Windows에서의 게임 해킹, 리버스 엔지니어링, 메모리 연구에 유용합니다.
