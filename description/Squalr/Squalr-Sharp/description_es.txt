Squalr es una aplicación de alto rendimiento para editar memoria en Windows, pensada para crear y compartir cheats en juegos de escritorio. Está escrita principalmente en C#/.NET y ofrece escaneo de memoria, resolución de punteros e inyección de ensamblador x86/x64 mediante una interfaz WPF, una CLI y una API de motor reutilizable. Los escaneos rápidos se logran con multiproceso y SIMD (SSE, AVX o AVX-512), mientras que el ensamblado/desensamblado basado en NASM y el scripting en C# permiten un análisis y una automatización más profundos. El proyecto también incluye módulos para adjuntar procesos, ver memoria, depurar e inspeccionar objetos .NET, lo que lo hace útil para game hacking, ingeniería inversa e investigación de memoria en Windows.
