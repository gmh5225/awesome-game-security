Squalr は、デスクトップゲーム向けのチート作成・共有に使える高性能な Windows メモリ編集アプリケーションです。主に C#/.NET で書かれており、WPF GUI、CLI、再利用可能なエンジン API を通じて、メモリスキャン、ポインタ解決、x86/x64 アセンブリ注入を提供します。マルチスレッドと SIMD（SSE、AVX、または AVX-512）により高速スキャンを実現し、NASM ベースのアセンブル/ディスアセンブルや C# スクリプティングで、より深い解析と自動化にも対応します。プロセスアタッチ、メモリビューア、デバッグ、.NET オブジェクト検査などのモジュールも備え、Windows 上のゲームハッキング、リバースエンジニアリング、メモリ研究に役立ちます。
