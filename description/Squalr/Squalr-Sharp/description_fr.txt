Squalr est une application Windows haute performance d'édition de mémoire pour créer et partager des cheats dans les jeux de bureau. Écrite principalement en C#/.NET, elle offre le scan mémoire, la résolution de pointeurs et l'injection d'assemblage x86/x64 via une interface WPF, une CLI et une API moteur réutilisable. Les scans rapides s'appuient sur le multi-threading et le SIMD (SSE, AVX ou AVX-512), tandis que l'assemblage/désassemblage basé sur NASM et le scripting C# permettent une analyse et une automatisation plus poussées. Le projet inclut aussi des modules pour l'attachement de processus, la visualisation mémoire, le débogage et l'inspection d'objets .NET, ce qui le rend utile pour le game hacking, la reverse engineering et la recherche mémoire sous Windows.
