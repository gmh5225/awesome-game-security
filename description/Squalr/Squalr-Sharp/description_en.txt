Squalr is a high-performance Windows memory editing application for creating and sharing cheats in desktop games. It is written primarily in C#/.NET and provides memory scanning, pointer resolution, and x86/x64 assembly injection through a WPF GUI, a CLI, and a reusable engine API. Fast scans are achieved with multi-threading and SIMD (SSE, AVX, or AVX-512), while NASM-based assemble/disassemble and C# scripting support deeper analysis and automation. The project also includes modules for process attachment, memory viewing, debugging, and .NET object inspection, making it useful for game hacking, reverse engineering, and memory research on Windows.
