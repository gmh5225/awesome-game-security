Ce projet est un jeu de plateforme 2D conçu pour enseigner les concepts du game hacking à travers le gameplay.
Il est développé en C++ sur Cocos2d-x et cible Windows, macOS et Linux.
Le contenu inclut des mécaniques de type entraînement autour de sujets bas niveau tels que les valeurs, les opérateurs et la logique utilisés dans le raisonnement sur la mémoire de jeu.
Son public principal est constitué d'apprenants qui souhaitent une introduction pratique et ludifiée au game hacking et aux idées de reverse engineering.
