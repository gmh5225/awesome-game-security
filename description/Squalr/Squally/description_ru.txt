Этот проект — 2D-платформер, созданный для обучения концепциям game hacking через игровой процесс.
Он разработан на C++ с использованием Cocos2d-x и рассчитан на Windows, macOS и Linux.
Контент включает тренировочные механики вокруг низкоуровневых тем — значений, операторов и логики, применяемой при рассуждениях о памяти игры.
Основная аудитория — учащиеся, которым нужно практическое, геймифицированное введение в game hacking и идеи реверс-инжиниринга.
