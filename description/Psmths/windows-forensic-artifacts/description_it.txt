Questo progetto è una guida strutturata agli artefatti forensi di Windows usati nella risposta agli incidenti e nelle indagini. È un repository incentrato sulla documentazione, scritto in Markdown, che categorizza gli artefatti nei domini esecuzione, account, file, rete, persistenza e attività utente. Il materiale spiega le posizioni degli artefatti, le opzioni di parsing e le strategie di correlazione per costruire timeline investigative. È pensato per professionisti DFIR e ricercatori di sicurezza che necessitano di un riferimento pratico agli artefatti Windows.
