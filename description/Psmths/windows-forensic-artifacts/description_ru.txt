Этот проект — структурированное руководство по форензик-артефактам Windows, используемым при реагировании на инциденты и расследованиях. Это в основном документационный репозиторий на Markdown, в котором артефакты классифицированы по областям выполнения, учётных записей, файлов, сети, персистентности и пользовательской активности. Материалы описывают расположение артефактов, варианты разбора и стратегии корреляции для построения следственных временных шкал. Предназначен для специалистов DFIR и исследователей безопасности, которым нужен практический справочник по артефактам Windows.
