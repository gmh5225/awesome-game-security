Этот проект сосредоточен на сокрытии Callback.
Найдя codecave в легитимном модуле ядра (здесь в качестве примера берётся драйвер APCI, не защищённый PatchGuard), можно записать JMP на свои процедуры так, чтобы начальный адрес зарегистрированной функции обратного вызова находился в допустимом модуле.
В первую очередь полезен исследователям игровой безопасности и реверс-инженерам, изучающим атакующие техники в области cheat / hide.
