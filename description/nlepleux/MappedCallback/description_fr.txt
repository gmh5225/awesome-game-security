Ce projet se concentre sur le masquage de Callback.
En trouvant une codecave dans un module légitime du noyau (ici le pilote APCI, non protégé par PatchGuard), on peut écrire un JMP vers nos routines afin que l'adresse de début de la fonction enregistrée pour le callback se trouve dans un module valide.
Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine du cheat / hide.
