Этот проект — стартовый шаблон для создания драйверов Windows в режиме ядра с помощью CMake и KMDF. Он в основном на C++ и включает каркас точки входа драйвера, интеграцию WDK через FindWDK и опциональные вспомогательные средства обфускации на этапе компиляции. Настройка упрощает конфигурацию инструментария для разных целевых версий Windows, сохраняя код близким к реальным ограничениям ядра. Полезен исследователям безопасности и разработчикам низкого уровня, которым нужна чистая база для прототипирования драйверов.
