Questo progetto è un template iniziale per creare driver in modalità kernel di Windows con CMake e KMDF. È principalmente in C++ e include lo scaffolding per l'entry del driver, l'integrazione WDK tramite FindWDK e helper opzionali di offuscamento a compile-time. La configurazione semplifica l'impostazione della toolchain tra le varie versioni di destinazione di Windows, restando vicina ai vincoli reali del kernel. È utile per ricercatori di sicurezza e sviluppatori low-level che necessitano di una base pulita per prototipare driver.
