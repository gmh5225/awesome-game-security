Este proyecto es una plantilla inicial para crear controladores en modo kernel de Windows con CMake y KMDF. Está escrito principalmente en C++ e incluye andamiaje para el punto de entrada del controlador, integración de WDK mediante FindWDK y ayudas opcionales de ofuscación en tiempo de compilación. La configuración simplifica la cadena de herramientas para distintas versiones de destino de Windows sin alejarse de las restricciones reales del kernel. Resulta útil para investigadores de seguridad y desarrolladores de bajo nivel que necesitan una base limpia para prototipar controladores.
