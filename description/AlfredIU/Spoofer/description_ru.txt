Этот проект — пакет для game hacking на Windows C++, сочетающий user-mode overlay-чит с компонентами на стороне драйвера. Код в user-mode включает модули вроде аимбота, ESP-визуалов, кэширования сущностей и предсказания снарядов, отрисовываемые через интерфейс DirectX9 ImGui. В репозитории также есть артефакты сборки kernel-драйвера и брендинг, ориентированный на spoofing, что указывает на смешанный фокус на читерстве и сокрытии идентичности. Он в основном полезен для изучения архитектуры оффенсивных игровых читов и техник обхода античита.
