Ce projet est un package de game hacking en C++ sous Windows qui combine un cheat overlay en mode utilisateur avec des composants côté pilote. Le code en mode utilisateur inclut des modules tels que l’aimbot, les visuals ESP, la mise en cache des entités et la prédiction de projectiles, rendus via une interface ImGui DirectX9. Le dépôt contient également des artefacts de build de pilote noyau et un branding orienté spoofing, indiquant un double intérêt pour la triche et le contournement d’identité. Il est surtout pertinent pour étudier l’architecture offensive des cheats et les techniques de contournement d’anti-cheat.
