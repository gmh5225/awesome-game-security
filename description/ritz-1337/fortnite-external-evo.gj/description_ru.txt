Этот проект — внешний чит для Fortnite на C++, работающий вне процесса игры и читающий память через драйвер ядра. Он предоставляет ESP (позиции игроков, здоровье, дистанция), аимбот со сглаживанием и управлением FOV, а также ESP предметов/лута за счёт чтения состояния игры UE4 из внешней памяти. Внешний подход позволяет избежать обнаружения античитом, основанным на инъекции. Предназначен для исследователей игровой безопасности, изучающих архитектуру внешних читов для игр на Unreal Engine, защищённых EAC.
