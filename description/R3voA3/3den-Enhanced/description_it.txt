Questo progetto è una mod di espansione delle funzionalità per l'Eden Editor di Arma 3.
È costruita con script SQF, definizioni di configurazione e moduli addon che aggiungono strumenti UI dell'editor, attributi di missione e scorciatoie di workflow.
La mod è progettata affinché gli scenari possano essere giocati senza dipendenze rigide dall'estensione di editing.
Serve principalmente creatori di missioni e modder che desiderano un ambiente di editing Arma 3 più produttivo.
