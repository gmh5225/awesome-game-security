Этот проект — мод с расширением функций для редактора Eden в Arma 3.
Он построен на SQF-скриптах, определениях конфигурации и аддон-модулях, которые добавляют инструменты UI редактора, атрибуты миссий и ярлыки рабочего процесса.
Мод спроектирован так, чтобы сценарии можно было запускать без жёстких зависимостей от расширения редактирования.
Он в первую очередь предназначен для создателей миссий и моддеров, которым нужна более продуктивная среда редактирования Arma 3.
