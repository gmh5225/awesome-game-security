Este proyecto es un mod de ampliación de funciones para el Eden Editor de Arma 3.
Está construido con scripts SQF, definiciones de configuración y módulos de addon que añaden herramientas de interfaz del editor, atributos de misión y atajos de flujo de trabajo.
El mod está diseñado para que los escenarios se puedan jugar sin dependencias estrictas de la extensión de edición.
Sirve principalmente a creadores de misiones y modders que buscan un entorno de edición de Arma 3 más productivo.
