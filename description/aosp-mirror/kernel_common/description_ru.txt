Официальное зеркало общего ядра AOSP Generic Kernel Image (GKI), служащее общей upstream-базой, на которую опираются все производители устройств Android. Содержит исходный код ядра с Android-специфичными патчами, интеграцию сборки Bazel и поддержку инструментария Rust и используется для изучения поверхностей атаки и механизмов защиты ядра в экосистеме Android.
