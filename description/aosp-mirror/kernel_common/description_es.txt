Espejo oficial del kernel común AOSP Generic Kernel Image (GKI), que sirve como base upstream compartida sobre la que construyen todos los fabricantes de dispositivos Android. Contiene el código fuente del kernel con parches específicos de Android, integración de compilación con Bazel y soporte de la cadena de herramientas Rust, y se usa para estudiar superficies de ataque y mecanismos de defensa del kernel en el ecosistema Android.
