Miroir officiel du noyau commun AOSP Generic Kernel Image (GKI), qui sert de base amont partagée sur laquelle s’appuient tous les fabricants d’appareils Android. Il contient le code source du noyau avec les correctifs spécifiques à Android, l’intégration de build Bazel et la prise en charge de la chaîne d’outils Rust, utiles pour étudier les surfaces d’attaque et les mécanismes de défense du noyau dans l’écosystème Android.
