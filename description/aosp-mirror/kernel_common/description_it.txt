Mirror ufficiale del kernel comune AOSP Generic Kernel Image (GKI), che funge da base upstream condivisa su cui si basano tutti i vendor di dispositivi Android. Contiene il codice sorgente del kernel con patch specifiche per Android, integrazione di build Bazel e supporto alla toolchain Rust, ed è utile per studiare superfici di attacco e meccanismi di difesa del kernel nell’ecosistema Android.
