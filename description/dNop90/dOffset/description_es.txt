Este proyecto se centra en usar IDA Pro y Cheat Engine para obtener el offset del módulo actual.
Resulta útil al realizar análisis estático de un archivo mientras se depura simultáneamente con varias herramientas, entre ellas IDA Pro, Cheat Engine, x64dbg y otras.
Está pensado principalmente para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de plugins de cheat / IDA.
