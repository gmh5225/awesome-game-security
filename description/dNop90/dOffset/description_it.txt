Questo progetto si concentra sull'uso di IDA Pro e Cheat Engine per ottenere l'offset del modulo corrente.
È utile quando si esegue l'analisi statica di un file mentre si esegue il debugging contemporaneamente con più strumenti, tra cui IDA Pro, Cheat Engine, x64dbg e altri.
È destinato principalmente a ricercatori di sicurezza nei giochi e a reverse engineer che studiano tecniche offensive nell'ambito dei plugin cheat / IDA.
