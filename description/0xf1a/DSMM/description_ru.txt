Этот проект — proof of concept для ядра Windows, демонстрирующий ручное отображение драйвера в отброшенный раздел образа легитимного драйвера. Он показывает создание системного потока после загрузки с попыткой избежать немедленного обнаружения, вызванного PatchGuard. Реализация сосредоточена на разработке драйверов на C с сопутствующим кодом загрузчика и структур. Предназначен для продвинутых экспериментов по скрытой загрузке в ядре в авторизованных средах исследований античита и безопасности.
