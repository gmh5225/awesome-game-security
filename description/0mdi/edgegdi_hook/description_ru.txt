Этот проект — доказательство концепции перехвата поведения GDI через патч указателя в секции данных, связанной с EdgeGDI. Он использует сканирование сигнатур на C++ и замену указателей во время выполнения для перехвата экспортируемой функциональности, связанной с GDI32, например путей BitBlt. Репозиторий включает код реализации и небольшой тестовый стенд, а также заметки и скриншоты с конкретных сборок Windows 10. Он в первую очередь полезен для исследований графических хуков и низкоуровневого анализа обхода античита, где важно минимизировать патчинг секции кода.
