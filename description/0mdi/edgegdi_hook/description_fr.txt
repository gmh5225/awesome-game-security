Ce projet est une preuve de concept pour accrocher le comportement GDI via un patch de pointeur dans une section de données liée à EdgeGDI. Il utilise le balayage de motifs en C++ et le remplacement de pointeurs à l'exécution pour intercepter des fonctionnalités exportées liées à GDI32, telles que les chemins BitBlt. Le dépôt comprend le code d'implémentation et un petit banc de test, avec des notes et captures d'écran provenant de versions spécifiques de Windows 10. Il intéresse surtout la recherche sur les hooks graphiques et l'analyse d'évasion anti-cheat bas niveau, là où il est important de minimiser les patchs de section de code.
