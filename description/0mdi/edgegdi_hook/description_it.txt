Questo progetto è una proof of concept per l'hooking del comportamento GDI tramite una patch di puntatore in una sezione dati correlata a EdgeGDI. Usa lo scanning di pattern in C++ e la sostituzione di puntatori a runtime per intercettare funzionalità esportate relative a GDI32, come i percorsi BitBlt. Il repository include il codice di implementazione e un piccolo harness di test, con note e screenshot da build specifiche di Windows 10. È rilevante soprattutto per la ricerca sugli hook grafici e per l'analisi di evasione anti-cheat a basso livello, dove è importante ridurre al minimo le patch della sezione di codice.
