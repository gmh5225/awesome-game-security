Este proyecto es una prueba de concepto para enganchar el comportamiento de GDI mediante un parche de puntero en una sección de datos relacionada con EdgeGDI. Usa escaneo de patrones en C++ y reemplazo de punteros en tiempo de ejecución para interceptar funcionalidad exportada relacionada con GDI32, como las rutas de BitBlt. El repositorio incluye código de implementación y un pequeño arnés de pruebas, con notas y capturas de pantalla de compilaciones específicas de Windows 10. Es especialmente relevante para la investigación de hooks gráficos y el análisis de evasión anti-cheat a bajo nivel, donde minimizar el parcheo de la sección de código es importante.
