Questo progetto è un parser C++ multipiattaforma per file minidump in modalità utente di Windows. Fornisce accesso ai dati dei thread, ai contesti dei registri, agli intervalli di memoria virtuale e ai moduli caricati tramite API di libreria e un'utilità di parsing. Il repository include file di build CMake, test e binding Python opzionali per l'automazione. È utile per strumenti di debug, reverse engineering e il triage dei dump nella risposta agli incidenti.
