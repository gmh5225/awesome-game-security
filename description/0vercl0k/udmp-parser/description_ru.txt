Этот проект — кроссплатформенный C++-парсер файлов minidump пользовательского режима Windows. Он предоставляет доступ к данным потоков, контекстам регистров, диапазонам виртуальной памяти и загруженным модулям через библиотечные API и утилиту парсинга. Репозиторий включает файлы сборки CMake, тесты и опциональные Python-привязки для автоматизации. Полезен для отладочных инструментов, реверс-инжиниринга и triage дампов при реагировании на инциденты.
