Ce projet est un parseur C++ multiplateforme pour les fichiers minidump en mode utilisateur Windows. Il donne accès aux données de threads, aux contextes de registres, aux plages de mémoire virtuelle et aux modules chargés via des API de bibliothèque et un utilitaire de parsing. Le dépôt inclut des fichiers de build CMake, des tests et des bindings Python optionnels pour l'automatisation. Il est utile pour les outils de débogage, la rétro-ingénierie et le triage de dumps en réponse à incident.
