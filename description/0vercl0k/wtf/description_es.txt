Este proyecto es un fuzzer distribuido basado en snapshots para Windows y objetivos Linux experimentales. Combina fuzzing guiado por cobertura con múltiples backends de ejecución, incluida la ejecución de snapshots impulsada por emulador y virtualización para escenarios en modo usuario y modo kernel. El código está escrito en gran parte en C++, con scripts y herramientas auxiliares para el manejo de corpus, el procesamiento de cobertura y flujos de trabajo orientados a trazas. Está pensado para investigadores de vulnerabilidades que hacen fuzzing de software complejo como componentes de juegos, controladores y servicios del sistema.
