Questo progetto è una libreria di parsing e un insieme di strumenti in C++ per i dump di crash del kernel Windows. Supporta i formati full dump e active dump ed espone dati come context record, informazioni sulle eccezioni, parametri di bugcheck e viste della memoria fisica. Il codice include una CLI del parser, test automatizzati, script di build CMake e binding Python opzionali per flussi di lavoro basati su script. È pensato per la forensics dei crash a basso livello, l'automazione del debugging del kernel e la ricerca sull'exploitation.
