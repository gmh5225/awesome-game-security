Этот проект — библиотека и набор инструментов на C++ для разбора дампов аварий ядра Windows. Он поддерживает форматы full dump и active dump и предоставляет такие данные, как context records, сведения об исключениях, параметры bugcheck и представления физической памяти. В репозитории есть CLI-парсер, автоматические тесты, скрипты сборки CMake и опциональные привязки Python для сценарных рабочих процессов. Проект предназначен для низкоуровневой форензики аварий, автоматизации отладки ядра и исследований эксплуатации.
