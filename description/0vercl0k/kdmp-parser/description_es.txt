Este proyecto es una biblioteca de análisis y un conjunto de herramientas en C++ para dumps de fallo del kernel de Windows. Admite los formatos full dump y active dump, y expone datos como context records, información de excepciones, parámetros de bugcheck y vistas de memoria física. El código incluye un CLI de análisis, pruebas automatizadas, scripts de compilación CMake y bindings opcionales de Python para flujos de trabajo con scripts. Está diseñado para forense de fallos a bajo nivel, automatización del depurado del kernel e investigación de explotación.
