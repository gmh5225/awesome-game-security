Этот проект — символизатор трасс выполнения Windows, который сопоставляет «сырые» указатели инструкций с символьными адресами. Он использует dbgeng и данные crash dump, чтобы переводить адреса трассы в читаемый вывод на уровне функций для анализа. Реализация написана на C++ и поставляется как утилита командной строки с вспомогательными библиотеками для форматирования и обработки аргументов. В основном полезна для triage аварий, отладки эксплойтов и постмортем-анализа трасс.
