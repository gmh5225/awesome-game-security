Questo progetto è un simbolizzatore di tracce di esecuzione Windows che risolve i puntatori di istruzione grezzi in posizioni simboliche. Usa dbgeng e i dati dei crash dump per tradurre gli indirizzi della traccia in un output leggibile a livello di funzione per l'analisi. L'implementazione è in C++ ed è fornita come strumento a riga di comando con librerie di supporto per la formattazione e la gestione degli argomenti. È soprattutto utile per il triage dei crash, il debug di exploit e i flussi di lavoro di analisi post-mortem delle tracce.
