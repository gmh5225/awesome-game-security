Este proyecto es un simbolizador de trazas de ejecución de Windows que resuelve punteros de instrucción en bruto a ubicaciones simbólicas. Usa dbgeng y datos de crash dump para traducir las direcciones de la traza en una salida legible a nivel de función para el análisis. La implementación está en C++ y se empaqueta como herramienta de línea de comandos con bibliotecas de apoyo para el formateo y el manejo de argumentos. Es especialmente útil para el triage de fallos, la depuración de exploits y los flujos de trabajo de trazas post mortem.
