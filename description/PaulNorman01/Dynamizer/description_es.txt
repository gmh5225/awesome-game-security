Este proyecto es un kit compacto de herramientas C++ anti-análisis pensado para reducir la eficacia de la inspección en tiempo de ejecución. Ofrece módulos como ofuscación de cadenas, lógica anti-step-over, comprobaciones de puntos de interrupción por software y hardware, comprobaciones de integridad de .text, manipulación de direcciones de retorno y unhooking de DLL del sistema. Los componentes están diseñados para integrarse de forma selectiva en bases de código existentes con una fricción mínima. Se usa principalmente en investigación de seguridad ofensiva y experimentación anti-tamper donde se estudia la evasión del análisis dinámico.
