Questo progetto è un’utilità anti-forense per Windows pensata per ridurre le tracce in fase post-exploitation. Include funzionalità come lo shredding dei file, la soppressione di event log e Prefetch, la gestione del journal USN, la pulizia legata ai timestamp e la rimozione di vari artefatti di shell e cache. L’implementazione è principalmente in C++ con interazione diretta alle API Windows e offre anche opzioni per cancellare artefatti di quarantena di Defender e comportamenti di auto-rimozione del file. È destinata soprattutto a simulazioni red team e alla validazione difensiva dei flussi di incident response e forensics.
