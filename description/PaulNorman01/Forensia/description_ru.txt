Этот проект — утилита антифорензики для Windows, предназначенная для снижения следов на этапе пост-эксплуатации. Она включает такие возможности, как шрединг файлов, подавление журналов событий и Prefetch, работу с USN journal, очистку, связанную с временными метками, а также удаление различных артефактов оболочки и кэша. Реализация в основном на C++ с прямым обращением к Windows API и также включает опции очистки артефактов карантина Defender и самоудаления файла. В основном предназначена для red-team симуляций и защитной проверки процессов реагирования на инциденты и форензики.
