Este proyecto es una utilidad anti-forense para Windows orientada a reducir rastros tras la explotación. Incluye capacidades como shredding de archivos, supresión de registros de eventos y Prefetch, manejo del diario USN, limpieza relacionada con marcas de tiempo y eliminación de múltiples artefactos de shell y caché. La implementación está principalmente en C++ con interacción directa con las API de Windows, e incluye opciones para borrar artefactos de cuarentena de Defender y realizar comportamientos de autoeliminación del archivo. Está pensada sobre todo para simulación de red team y validación defensiva de flujos de respuesta a incidentes y forense.
