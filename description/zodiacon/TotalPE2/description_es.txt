Este proyecto es Total PE 2, un visualizador completo de archivos PE (Portable Executable) para Windows con una interfaz WPF moderna. Muestra información detallada de la estructura PE, incluidos encabezados, secciones, imports, exports, recursos, directorios de depuración, reubicaciones, TLS, manejo de excepciones y metadatos .NET. La aplicación C# admite arrastrar y soltar, visualización hexadecimal y extracción de recursos. Está dirigida a ingenieros inversos, analistas de malware y desarrolladores de Windows que necesitan inspeccionar rápidamente la estructura PE de forma visual.
