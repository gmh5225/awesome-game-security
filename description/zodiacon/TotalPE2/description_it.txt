Questo progetto è Total PE 2, un visualizzatore completo di file PE (Portable Executable) per Windows con un'interfaccia WPF moderna. Mostra informazioni dettagliate sulla struttura PE, tra cui header, sezioni, import, export, risorse, directory di debug, relocation, TLS, gestione delle eccezioni e metadati .NET. L'applicazione C# supporta il drag-and-drop, la visualizzazione esadecimale e l'estrazione delle risorse. È rivolta a reverse engineer, analisti di malware e sviluppatori Windows che necessitano di un ispettore visivo rapido della struttura PE.
