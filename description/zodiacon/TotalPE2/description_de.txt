Dieses Projekt ist Total PE 2, ein umfassender PE-(Portable-Executable)-Dateibetrachter für Windows mit einer modernen WPF-Oberfläche. Es zeigt detaillierte PE-Strukturinformationen an, darunter Header, Sections, Imports, Exports, Ressourcen, Debug-Verzeichnisse, Relocations, TLS, Exception Handling und .NET-Metadaten. Die C#-Anwendung unterstützt Drag-and-Drop, Hex-Ansicht und Ressourcenextraktion. Sie richtet sich an Reverse Engineers, Malware-Analysten und Windows-Entwickler, die einen schnellen visuellen PE-Struktur-Inspector benötigen.
