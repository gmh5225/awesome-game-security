Этот проект — Total PE 2, комплексный просмотрщик файлов PE (Portable Executable) для Windows с современным интерфейсом WPF. Он отображает подробную информацию о структуре PE, включая заголовки, секции, импорты, экспорты, ресурсы, каталоги отладки, релокации, TLS, обработку исключений и метаданные .NET. Приложение на C# поддерживает перетаскивание, просмотр в шестнадцатеричном виде и извлечение ресурсов. Оно предназначено для реверс-инженеров, аналитиков вредоносного ПО и Windows-разработчиков, которым нужен быстрый визуальный инспектор структуры PE.
