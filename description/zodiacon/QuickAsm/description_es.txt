Este proyecto es QuickAsm, un editor y ejecutor ligero de ensamblador x86/x64 para Windows. Ofrece una GUI sencilla donde los usuarios pueden escribir instrucciones de ensamblador, ensamblarlas en tiempo real con el motor Keystone, ver los bytes de código máquina resultantes y ejecutar el código ensamblado dentro del proceso de la aplicación. Esta herramienta en C++ es útil para experimentar rápidamente con ensamblador y probar shellcode. Está dirigida a ingenieros de ingeniería inversa, desarrolladores de exploits e investigadores de seguridad que necesitan prototipar y probar fragmentos de código ensamblador con rapidez.
