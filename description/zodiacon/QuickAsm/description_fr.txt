Ce projet est QuickAsm, un éditeur et exécuteur d'assemblage x86/x64 léger pour Windows. Il propose une interface graphique simple où les utilisateurs peuvent saisir des instructions assembleur, les assembler en temps réel avec le moteur Keystone, consulter les octets de code machine résultants et exécuter le code assemblé dans le processus de l'application. Cet outil C++ est utile pour expérimenter rapidement en assembleur et tester du shellcode. Il s'adresse aux ingénieurs en reverse engineering, aux développeurs d'exploits et aux chercheurs en sécurité qui ont besoin de prototyper et tester rapidement des fragments de code assembleur.
