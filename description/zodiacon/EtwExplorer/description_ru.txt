Этот проект — ETW Explorer, графический инструмент Windows для просмотра и изучения поставщиков ETW (Event Tracing for Windows), их определений событий, ключевых слов и метаданных. Он перечисляет зарегистрированные поставщики ETW, отображает схемы событий на основе манифеста или TraceLogging и позволяет фильтровать по имени поставщика или GUID. Приложение на C#/WPF помогает аналитикам находить доступные источники телеметрии. Оно предназначено для исследователей безопасности, системных администраторов и разработчиков EDR, изучающих инфраструктуру Windows ETW для обнаружения и мониторинга.
