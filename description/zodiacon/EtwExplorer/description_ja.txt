本プロジェクトは ETW Explorer で、ETW（Event Tracing for Windows）プロバイダー、そのイベント定義、キーワード、メタデータを閲覧・検査するための Windows GUI ツールです。登録済みの ETW プロバイダーを列挙し、マニフェストベースまたは TraceLogging のイベントスキーマを表示し、プロバイダー名や GUID によるフィルタリングが可能です。この C#/WPF アプリケーションは、分析者が利用可能なテレメトリソースを発見するのに役立ちます。検出と監視のために Windows ETW インフラを調査するセキュリティ研究者、システム管理者、EDR 開発者を対象としています。
