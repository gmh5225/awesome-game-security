Ce projet se concentre sur un script Frida permettant d'intercepter et de dump le trafic USB sur macOS. Les OFFSETS_ ont été dumpés sur macOS Big Sur, autant que je m'en souvienne. Il est surtout utile aux chercheurs en sécurité des jeux et aux reverse engineers qui étudient les techniques offensives dans le domaine du cheat / Frida.
