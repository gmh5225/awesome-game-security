本プロジェクトは、実ハードウェアを必要とせずに合成ユーザ空間 PCIe デバイスエミュレーションを作成するための Linux カーネルフレームワークです。
ホスト OS から正当な PCI デバイスとして見える仮想 PCIe デバイスを作成し、PCIe ドライバの開発とテストをすべてソフトウェア上で可能にします。
このカーネルモジュールは PCI サブシステムに合成カードを登録する独自の手法を用い、libvfio-user とは異なり VM や QEMU を必要とせずホスト上で直接動作します。
物理ハードウェアなしで PCIe デバイスエミュレーションとドライバ連携を研究する PCIe セキュリティ研究者や DMA ツール開発者に主に役立ちます。
