Este proyecto es un framework del kernel de Linux para crear emulación sintética de dispositivos PCIe en espacio de usuario sin necesidad de hardware real.
Crea dispositivos PCIe virtuales que aparecen como dispositivos PCI legítimos ante el sistema operativo anfitrión, permitiendo el desarrollo y las pruebas de controladores PCIe por completo en software.
El módulo del kernel usa técnicas novedosas para poblar el subistema PCI con tarjetas sintéticas, y se diferencia de libvfio-user al operar directamente en el anfitrión sin requerir una VM ni QEMU.
Es útil principalmente para investigadores de seguridad PCIe y desarrolladores de herramientas DMA que estudian la emulación de dispositivos PCIe y la interacción con controladores sin hardware físico.
