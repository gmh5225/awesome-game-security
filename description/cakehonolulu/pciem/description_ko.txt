이 프로젝트는 실제 하드웨어 없이 합성 사용자 공간 PCIe 장치 에뮬레이션을 만들기 위한 Linux 커널 프레임워크입니다.
호스트 OS에 정상적인 PCI 장치로 보이는 가상 PCIe 장치를 생성하여, PCIe 드라이버 개발과 테스트를 전적으로 소프트웨어에서 수행할 수 있게 합니다.
이 커널 모듈은 PCI 서브시스템에 합성 카드를 등록하는 새로운 기법을 사용하며, libvfio-user와 달리 VM이나 QEMU 없이 호스트에서 직접 동작합니다.
물리 하드웨어 없이 PCIe 장치 에뮬레이션과 드라이버 상호작용을 연구하는 PCIe 보안 연구자와 DMA 도구 개발자에게 주로 유용합니다.
