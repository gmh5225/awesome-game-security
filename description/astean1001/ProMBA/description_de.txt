Dieses Projekt ist ProMBA, ein Werkzeug zur Deobfuskation von Mixed-Boolean-Arithmetic-(MBA-)Ausdrücken. Es wendet algebraische Vereinfachung, Musterabgleich und synthesebasierte Ansätze an, um komplexe MBA-obfuskierte Ausdrücke auf ihre einfacheren Originalformen zurückzuführen. Das Tool verarbeitet lineare und polynomielle MBA-Transformationen, wie sie häufig von VM-basierten Protectors genutzt werden. Es richtet sich an Reverse Engineers und Deobfuskationsforscher, die MBA-obfuskierten Code in geschützten Binärdateien analysieren.
