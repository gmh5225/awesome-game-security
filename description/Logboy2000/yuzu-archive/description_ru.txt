Yuzu Archive — это репозиторий-заглушка, который больше не предоставляет исходный код эмулятора.
Его содержимое в основном состоит из документации, включая краткий README и запись уведомления DMCA.
Архив служит историческим снимком контекста снятия (takedown), а не кодовой базой для разработки.
Основное назначение — юридическая и экосистемная справка для исследователей, отслеживающих меры принуждения в отношении эмуляторов.
