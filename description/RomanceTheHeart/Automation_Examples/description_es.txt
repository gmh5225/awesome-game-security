Este proyecto es un ejemplo de Unreal Engine 4 centrado en utilidades de automatización del editor. Usa clases C++ Editor Utility junto con los módulos Blutility y EditorScriptingUtilities para automatizar tareas como renombrar assets, duplicarlos, limpiar contenido no usado y organizar datos del editor. El repositorio también incluye assets de utilidad Blueprint y widgets de panel del editor pensados para flujos de trabajo call-in-editor. Está dirigido principalmente a desarrolladores de Unreal que buscan patrones prácticos de automatización para herramientas de pipeline de contenido.
