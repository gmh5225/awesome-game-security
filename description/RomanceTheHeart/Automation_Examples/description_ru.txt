Этот проект — пример для Unreal Engine 4, посвящённый утилитам автоматизации редактора. Он использует классы C++ Editor Utility вместе с модулями Blutility и EditorScriptingUtilities для автоматизации задач вроде переименования ассетов, их дублирования, очистки неиспользуемого контента и упорядочивания данных редактора. В репозитории также есть утилитарные Blueprint-ассеты и виджеты панели редактора для сценариев call-in-editor. Материал в первую очередь рассчитан на разработчиков Unreal, которым нужны практические паттерны автоматизации для инструментов контент-пайплайна.
