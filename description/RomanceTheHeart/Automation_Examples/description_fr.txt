Ce projet est un exemple Unreal Engine 4 axé sur les utilitaires d'automatisation de l'éditeur. Il s'appuie sur des classes C++ Editor Utility, avec les modules Blutility et EditorScriptingUtilities, pour automatiser des tâches telles que le renommage d'assets, leur duplication, le nettoyage du contenu inutilisé et l'organisation des données de l'éditeur. Le dépôt inclut également des assets utilitaires Blueprint et des widgets de tableau de bord éditeur conçus pour les workflows call-in-editor. Il s'adresse principalement aux développeurs Unreal qui recherchent des modèles d'automatisation concrets pour les outils de pipeline de contenu.
