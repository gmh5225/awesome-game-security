このプロジェクトは、エディタ自動化ユーティリティに焦点を当てた Unreal Engine 4 のサンプルです。C++ の Editor Utility クラスと Blutility、EditorScriptingUtilities モジュールを使い、アセットのリネーム、複製、未使用コンテンツの整理、エディタデータの整理といった作業を自動化します。リポジトリには、call-in-editor ワークフロー向けの Blueprint ユーティリティアセットとエディタダッシュボードウィジェットも含まれます。コンテンツパイプライン向けの実用的な自動化パターンを求める Unreal 開発者を主な対象としています。
