이 프로젝트는 에디터 자동화 유틸리티에 초점을 맞춘 Unreal Engine 4 샘플입니다. C++ Editor Utility 클래스와 Blutility, EditorScriptingUtilities 모듈을 사용해 에셋 이름 변경, 에셋 복제, 미사용 콘텐츠 정리, 에디터 데이터 정리 같은 작업을 자동화합니다. 저장소에는 call-in-editor 워크플로를 위한 Blueprint 유틸리티 에셋과 에디터 대시보드 위젯도 포함되어 있습니다. 콘텐츠 파이프라인 툴링을 위한 실용적인 자동화 패턴을 원하는 Unreal 개발자를 주요 대상으로 합니다.
