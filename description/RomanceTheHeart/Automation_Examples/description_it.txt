Questo progetto è un esempio Unreal Engine 4 incentrato sulle utility di automazione dell'editor. Usa classi C++ Editor Utility con i moduli Blutility ed EditorScriptingUtilities per automatizzare attività come rinominare asset, duplicarli, ripulire contenuti inutilizzati e organizzare i dati dell'editor. Il repository include anche asset utility Blueprint e widget della dashboard dell'editor pensati per i workflow call-in-editor. È pensato soprattutto per gli sviluppatori Unreal che cercano pattern pratici di automazione per gli strumenti della content pipeline.
