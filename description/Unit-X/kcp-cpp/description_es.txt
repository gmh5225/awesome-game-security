Este proyecto es un wrapper ligero en C++ sobre el protocolo UDP fiable KCP. Ofrece abstracciones de cliente y servidor, manejo de datos impulsado por callbacks y parámetros de transporte configurables para redes de baja latencia. La biblioteca está orientada a C++17 y a compilaciones basadas en CMake en plataformas de escritorio habituales. Es adecuada para prototipos de redes de juegos y para experimentar con capas de fiabilidad en aplicaciones en tiempo real.
