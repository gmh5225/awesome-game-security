Этот проект — лёгкая C++-обёртка вокруг надёжного UDP-протокола KCP. Она предоставляет абстракции клиента и сервера, обработку данных через колбэки и настраиваемые параметры транспорта для сетей с низкой задержкой. Библиотека рассчитана на C++17 и сборку через CMake на распространённых настольных платформах. Подходит для прототипов игровых сетей и экспериментов со слоем надёжности в приложениях реального времени.
