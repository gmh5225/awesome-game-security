Dieses Projekt ist ein OLLVM-Unflattening-Werkzeug, das darauf abzielt, klareren Kontrollfluss aus verschleierten Programmen wiederherzustellen. Es ist hauptsächlich in Python implementiert und nutzt Reverse-Engineering-Abhängigkeiten wie miasm sowie Werkzeuge zur Graphvisualisierung. Das Repository enthält Beispielbinärdateien und Testeingaben zum Experimentieren mit dem Deobfuscation-Verhalten. Der Hauptanwendungsfall ist die Reverse-Engineering-Forschung an Software mit abgeflachtem Kontrollfluss.
