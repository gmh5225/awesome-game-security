Este proyecto es una herramienta de desaplanamiento OLLVM que busca recuperar un flujo de control más claro a partir de programas ofuscados. Está implementada principalmente en Python y utiliza dependencias de ingeniería inversa como miasm, junto con herramientas de visualización de grafos. El repositorio incluye binarios de ejemplo y entradas de prueba para experimentar con el comportamiento de desofuscación. Su caso de uso principal es la investigación de ingeniería inversa sobre software con flujo de control aplanado.
