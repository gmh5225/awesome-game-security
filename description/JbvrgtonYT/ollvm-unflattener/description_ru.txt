Этот проект — инструмент развёртывания (unflattening) OLLVM, предназначенный для восстановления более понятного потока управления из обфусцированных программ. Он реализован преимущественно на Python и использует зависимости для реверс-инжиниринга, такие как miasm, а также средства визуализации графов. Репозиторий содержит примеры бинарных файлов и тестовые входные данные для экспериментов с поведением деобфускации. Основное применение — исследования в области реверс-инжиниринга ПО с уплощённым потоком управления.
