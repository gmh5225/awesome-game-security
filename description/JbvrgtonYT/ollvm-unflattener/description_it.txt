Questo progetto è uno strumento di unflattening OLLVM che mira a recuperare un flusso di controllo più chiaro da programmi offuscati. È implementato principalmente in Python e utilizza dipendenze di reverse engineering come miasm e strumenti di visualizzazione dei grafi. Il repository include binari di esempio e input di test per sperimentare il comportamento di deoffuscamento. Il suo caso d'uso principale è la ricerca di reverse engineering su software con flusso di controllo appiattito.
