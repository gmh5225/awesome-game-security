Zodiak es un cheat completo basado en un controlador de kernel para Counter-Strike 1.6, diseñado para minimizar rastros visibles para el usuario en el sistema. Está implementado en C y ensamblador, usando técnicas como renderizado ESP basado en GDI en modo kernel y el abuso de callbacks MouHID para el control de entrada del aimbot. El proyecto también documenta la detección automática de offsets, el spoofing del contexto de hilo y un modelo de ejecución compacto de un solo hilo. Está dirigido a investigadores avanzados de seguridad en juegos que estudian la ingeniería de cheats a nivel de kernel y estrategias de evasión de anti-cheat.
