DirectXHook is a C++ library that hooks DirectX 11 and DirectX 12 so you can draw custom graphics inside a game window as an integrated overlay rather than an external one. It builds as a DLL (commonly loaded via dinput8 proxy injection) and works in both 32-bit and 64-bit modes. An included overlay framework supports rendering boxes, textures, and text with a simple callback-based API for creating custom overlays. Example overlays and sample mods demonstrate use for in-game HUDs and gameplay tweaks. The project targets game modders and reverse engineers who need in-process DirectX rendering hooks for overlays and tooling.
