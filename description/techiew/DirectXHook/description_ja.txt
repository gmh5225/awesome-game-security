DirectXHook は、DirectX 11 および DirectX 12 をフックし、外部ではなくゲームウィンドウ内の統合オーバーレイとしてカスタムグラフィックスを描画できる C++ ライブラリです。DLL としてビルドされ（一般的には dinput8 プロキシ注入で読み込み）、32 ビットと 64 ビットの両方に対応します。同梱のオーバーレイフレームワークは、ボックス、テクスチャ、テキストの描画をサポートし、カスタムオーバーレイ作成向けのシンプルなコールバックベース API を提供します。サンプルのオーバーレイとサンプル mod は、ゲーム内 HUD やゲームプレイ調整への利用例を示します。本プロジェクトは、オーバーレイやツール向けにプロセス内 DirectX レンダリングフックを必要とするゲーム mod 作者やリバースエンジニアを対象としています。
