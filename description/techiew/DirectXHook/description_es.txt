DirectXHook es una biblioteca C++ que engancha DirectX 11 y DirectX 12 para dibujar gráficos personalizados dentro de la ventana del juego como una superposición integrada, en lugar de una externa. Se compila como DLL (normalmente cargada mediante inyección proxy de dinput8) y funciona en modos de 32 y 64 bits. Un framework de overlay incluido admite el renderizado de cajas, texturas y texto con una API sencilla basada en callbacks para crear overlays personalizados. Overlays de ejemplo y mods de muestra demuestran su uso para HUD en el juego y ajustes de jugabilidad. El proyecto está orientado a modders de juegos e ingenieros inversos que necesitan hooks de renderizado DirectX en proceso para overlays y herramientas.
