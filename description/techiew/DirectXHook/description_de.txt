DirectXHook ist eine C++-Bibliothek, die DirectX 11 und DirectX 12 hookt, damit Sie benutzerdefinierte Grafiken als integriertes Overlay im Spielfenster zeichnen können – nicht als externes Overlay. Sie wird als DLL gebaut (häufig über dinput8-Proxy-Injection geladen) und funktioniert sowohl im 32-Bit- als auch im 64-Bit-Modus. Ein mitgeliefertes Overlay-Framework unterstützt das Rendern von Boxen, Texturen und Text mit einer einfachen callback-basierten API zum Erstellen eigener Overlays. Beispiel-Overlays und Sample-Mods zeigen die Nutzung für In-Game-HUDs und Gameplay-Anpassungen. Das Projekt richtet sich an Game-Modder und Reverse Engineers, die In-Process-DirectX-Rendering-Hooks für Overlays und Tooling benötigen.
