Этот проект посвящён d-810ng (Next Generation) — развитию d810 для деобфускации кода на этапе декомпиляции.
Он был разработан с учётом следующих целей.
Он в первую очередь полезен исследователям игровой безопасности и специалистам по реверс-инжинирингу, изучающим атакующие техники в области cheat / fix ollvm.
