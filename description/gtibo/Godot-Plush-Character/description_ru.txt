Этот проект — плагин или демо для Godot Engine, реализующее отрисовку плюшевых/soft body персонажей в стилизованной эстетике мягкой игрушки. Он демонстрирует шейдерное текстурирование ткани, мягкую деформацию и приёмы процедурной анимации для создания визуально привлекательных плюшевых 3D-персонажей в Godot. Предназначен для разработчиков Godot и technical artists, создающих стилизованные эффекты рендеринга персонажей.
