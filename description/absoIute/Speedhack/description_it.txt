Speedhack è un'implementazione Windows leggera per manipolare la velocità di esecuzione percepita di un programma.
Il progetto è scritto in C++ e può essere compilato come DLL da iniettare nei processi target.
Si basa sull'hooking di API legate al timing con Detours per accelerare, rallentare o mettere in pausa il comportamento a runtime.
Lo strumento è comunemente usato in esperimenti di sviluppo cheat e nella ricerca anti-cheat sulla manomissione del tempo.
