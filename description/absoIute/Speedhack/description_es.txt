Speedhack es una implementación ligera para Windows que permite manipular la velocidad de ejecución percibida de un programa.
El proyecto está escrito en C++ y puede compilarse como DLL para inyectarse en procesos objetivo.
Se basa en el hooking de APIs relacionadas con el tiempo mediante Detours para acelerar, ralentizar o pausar el comportamiento en tiempo de ejecución.
La herramienta se usa habitualmente en experimentos de desarrollo de cheats y en investigación anti-cheat sobre manipulación del tiempo.
