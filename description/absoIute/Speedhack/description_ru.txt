Speedhack — лёгкая реализация для Windows, позволяющая изменять воспринимаемую программой скорость выполнения.
Проект написан на C++ и может быть собран в DLL для внедрения в целевые процессы.
Он опирается на перехват связанных со временем API с помощью Detours, чтобы ускорять, замедлять или приостанавливать поведение во время выполнения.
Инструмент часто используют в экспериментах по разработке читов и в исследованиях античита, связанных с подменой времени.
