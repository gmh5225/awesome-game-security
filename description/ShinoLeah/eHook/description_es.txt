Este proyecto es un framework para crear hooks uprobe de Android ARM64 con módulos eBPF. Usa Go para la orquestación y C para la lógica de los hooks eBPF, con wrappers para leer y escribir memoria, registrar logs y enviar datos de eventos personalizados. Los usuarios configuran el paquete objetivo y los offsets de biblioteca, e implementan manejadores on-enter y on-leave para instrumentación o modificación de comportamiento. Está orientado principalmente al análisis dinámico en dispositivos con root, investigación de juegos móviles y trazado ligero en tiempo de ejecución.
