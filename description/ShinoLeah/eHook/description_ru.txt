Этот проект — фреймворк для создания Android ARM64 uprobe-хуков с модулями eBPF. Для оркестрации используется Go, а логика eBPF-хуков пишется на C; предусмотрены обёртки для чтения и записи памяти, логирования и отправки пользовательских данных событий. Пользователи задают целевой пакет и смещения библиотек, затем реализуют обработчики on-enter и on-leave для инструментирования или изменения поведения. В основном предназначен для динамического анализа на рутованных устройствах, исследований мобильных игр и лёгкого runtime-трейсинга.
