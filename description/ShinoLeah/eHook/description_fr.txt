Ce projet est un framework permettant de créer des hooks uprobe Android ARM64 avec des modules eBPF. Il utilise Go pour l'orchestration et C pour la logique des hooks eBPF, avec des wrappers pour la lecture et l'écriture mémoire, la journalisation et l'envoi de données d'événements personnalisées. Les utilisateurs configurent le package cible et les offsets de bibliothèque, puis implémentent des gestionnaires on-enter et on-leave pour l'instrumentation ou la modification du comportement. Il vise principalement l'analyse dynamique sur appareils rootés, la recherche sur les jeux mobiles et le traçage runtime léger.
