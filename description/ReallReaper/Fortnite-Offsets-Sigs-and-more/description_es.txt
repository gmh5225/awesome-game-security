Este proyecto es un repositorio pequeño que publica offsets de memoria de Fortnite y valores de firmas (signatures) relacionados.
Proporciona un archivo de cabecera con cadenas de punteros habituales de Unreal Engine y offsets de estructuras relevantes para el gameplay, usados por herramientas externas.
El contenido consiste principalmente en constantes de estilo C y C++ pensadas para una integración rápida en código de cheat o análisis existente.
Se utiliza como conjunto de datos de referencia para actualizaciones de game hacking y el mantenimiento de offsets.
