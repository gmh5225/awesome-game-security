Этот проект — небольшой репозиторий, публикующий смещения памяти Fortnite и связанные значения сигнатур.
Он предоставляет заголовочный файл с распространёнными цепочками указателей Unreal Engine и смещениями структур, связанных с геймплеем, которые используются внешними инструментами.
Содержимое в основном представляет собой константы в стиле C и C++, предназначенные для быстрой интеграции в существующий код читов или анализа.
Используется как справочный набор данных для обновлений в game hacking и сопровождения смещений.
