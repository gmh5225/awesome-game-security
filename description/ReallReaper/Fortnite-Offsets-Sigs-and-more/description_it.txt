Questo progetto è un piccolo repository che pubblica offset di memoria di Fortnite e relativi valori di signature.
Fornisce un file header con catene di puntatori Unreal Engine comuni e offset di strutture rilevanti per il gameplay, usati da strumenti esterni.
Il contenuto è costituito principalmente da costanti in stile C e C++ pensate per un’integrazione rapida nel codice di cheat o di analisi esistente.
Viene usato come dataset di riferimento per gli aggiornamenti di game hacking e la manutenzione degli offset.
