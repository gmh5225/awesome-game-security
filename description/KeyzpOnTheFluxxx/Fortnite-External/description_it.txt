Questo progetto è una semplice implementazione di cheat esterno per Fortnite. Si basa sull'aggiornamento degli offset e su pattern di interazione con processi esterni, offrendo un esempio leggero di manipolazione del gioco fuori processo. Il repository è scritto principalmente in C++ e organizzato come progetto Visual Studio con la struttura tipica di un cheat esterno. È pensato per l'apprendimento e la sperimentazione nel game hacking e nell'analisi dell'interazione con gli anti-cheat.
