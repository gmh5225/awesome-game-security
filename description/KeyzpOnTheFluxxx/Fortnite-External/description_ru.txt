Этот проект — простая реализация внешнего чита для Fortnite. Он опирается на обновление оффсетов и шаблоны взаимодействия с внешним процессом, представляя собой лёгкий пример внепроцессной манипуляции игрой. Репозиторий написан в основном на C++ и организован как проект Visual Studio с типичной структурой внешнего чита. Предназначен для обучения и экспериментов в области game hacking и анализа взаимодействия с античитами.
