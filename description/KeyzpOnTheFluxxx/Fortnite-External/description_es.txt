Este proyecto es una implementación sencilla de un cheat externo para Fortnite. Se basa en actualizaciones de offsets y patrones de interacción con procesos externos, y constituye un ejemplo ligero de manipulación de juegos fuera de proceso. El repositorio está escrito principalmente en C++ y organizado como un proyecto de Visual Studio con la estructura típica de un cheat externo. Está pensado para el aprendizaje y la experimentación en game hacking y en el análisis de la interacción con anti-cheats.
