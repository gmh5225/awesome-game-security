Этот проект — чит для Apex Legends на Python, который с помощью моделей компьютерного зрения YOLOv5 обнаруживает противников по данным с экрана. Он включает обученные веса модели, настраиваемые параметры выполнения, функции оверлея и прицеливания, а также логику обновления и контроля версий. Архитектура делает акцент на ИИ-обнаружении и управлении вместо прямого хукинга памяти процесса и содержит скрипты для рабочих процессов в стиле Windows и Linux. В первую очередь предназначен для автоматизации игр с помощью машинного обучения и экспериментов с учётом античита.
