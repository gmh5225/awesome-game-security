Questo progetto è un cheat per Apex Legends basato su Python che usa modelli di computer vision YOLOv5 per rilevare i nemici dai dati dello schermo. Include pesi di modello addestrati, parametri di runtime configurabili, funzioni di overlay e mira, nonché logica di aggiornamento/controllo versione. Il design privilegia il rilevamento e il controllo basati sull'IA invece del hooking diretto della memoria di processo, con script per flussi di lavoro in stile Windows e Linux. È rivolto principalmente all'automazione di gioco assistita dal machine learning e alla sperimentazione consapevole dell'anti-cheat.
