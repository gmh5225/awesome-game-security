Dieses Projekt ist ein KI-gestütztes IDA-Pro-Plugin für die semantische Binäranalyse.
Es nutzt einen Gemini-basierten Workflow, um Funktionsnamen, Variablennamen und detaillierte Kommentare aus dekompiliertem Code oder Assembler-Kontext vorzuschlagen.
Die Implementierung ist in Python und bietet eine interaktive Oberfläche mit kontextbezogener Analyse über Aufrufer, Aufgerufene und Referenzen hinweg.
Der Hauptanwendungsfall ist die Beschleunigung von Reverse-Engineering-Triage und Codeverständnis für Sicherheitsanalysten.
