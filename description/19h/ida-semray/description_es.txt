Este proyecto es un plugin de IDA Pro asistido por IA para el análisis semántico de binarios.
Utiliza un flujo de trabajo impulsado por Gemini para sugerir nombres de funciones, nombres de variables y comentarios detallados a partir del código descompilado o del contexto en ensamblador.
La implementación está en Python y ofrece una interfaz interactiva con análisis contextual sobre llamadores, llamados y referencias.
Su principal caso de uso es acelerar el triaje de ingeniería inversa y la comprensión del código para analistas de seguridad.
