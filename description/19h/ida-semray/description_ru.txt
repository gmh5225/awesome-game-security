Этот проект — ИИ-помощник в виде плагина IDA Pro для семантического анализа бинарных файлов.
Он использует рабочий процесс на базе Gemini, чтобы предлагать имена функций, имена переменных и подробные комментарии по декомпилированному коду или контексту ассемблера.
Реализация на Python и предоставляет интерактивный интерфейс с контекстно-зависимым анализом вызывающих, вызываемых и ссылок.
Основной сценарий использования — ускорение первичного разбора при реверс-инжиниринге и понимания кода для специалистов по безопасности.
