Ce projet est un plugin IDA Pro assisté par IA pour l'analyse sémantique de binaires.
Il s'appuie sur un flux de travail alimenté par Gemini pour proposer des noms de fonctions, des noms de variables et des commentaires détaillés à partir du code décompilé ou du contexte assembleur.
L'implémentation est en Python et fournit une interface interactive avec une analyse contextuelle sur les appelants, les appelés et les références.
Son principal cas d'usage est d'accélérer le triage en reverse engineering et la compréhension du code pour les analystes en sécurité.
