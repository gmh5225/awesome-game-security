Этот проект — плагин декомпилятора Hex-Rays для автоматического снятия обфускации Hikari LLVM в IDA Pro.
Он восстанавливает поток управления из flattening и ложных ветвлений, разрешает косвенные передачи управления и восстанавливает зашифрованные конструкции данных.
Плагин реализован в основном на C++ и применяет символьные рассуждения с Z3, а также обширные правила упрощения MBA.
Основной сценарий использования — реверс-инжиниринг сильно обфусцированных бинарных файлов в исследованиях вредоносного ПО и безопасности игр.
