Este proyecto es un complemento del decompilador Hex-Rays para revertir automáticamente la ofuscación Hikari LLVM en IDA Pro.
Restaura el flujo de control a partir del aplanamiento y las ramas falsas, resuelve transferencias de control indirectas y recupera construcciones de datos cifrados.
El complemento está implementado principalmente en C++ y aplica razonamiento simbólico con Z3 junto con amplias reglas de simplificación MBA.
Su caso de uso principal es la ingeniería inversa de binarios muy ofuscados en investigación de malware y seguridad de juegos.
