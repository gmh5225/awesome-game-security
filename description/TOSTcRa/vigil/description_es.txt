Este proyecto es una herramienta de monitorización de seguridad eBPF escrita en Rust que usa programas BPF para el rastreo y análisis de eventos a nivel de kernel. Incluye una arquitectura modular de crates con carga de ELF BPF, manejo de eventos perf, interfaz CLI y comunicación cliente-servidor para monitorización distribuida. Es especialmente útil para investigadores de seguridad en Linux e ingenieros de kernel que estudian sistemas de monitorización en tiempo de ejecución y detección de amenazas basados en eBPF.
