Этот проект — набор Bash-скриптов автоматизации для сборки Android-ядер под цели GKI и non-GKI.
Скрипты координируют repo sync, применение патчей, изменения defconfig, упаковку и публикацию релизов для нескольких семейств устройств и веток ядра.
Они тесно интегрируют потоки патчей KernelSU и SUSFS и включают вендор-специфичные варианты сборки для Pixel, OnePlus, Xiaomi и других.
Основная аудитория — продвинутые моддеры Android-ядер и исследователи мобильной безопасности; некоторые скрипты к настоящему времени могут устареть.
