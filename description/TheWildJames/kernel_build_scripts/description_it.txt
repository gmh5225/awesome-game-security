Questo progetto è una raccolta di script Bash di automazione per compilare kernel Android su target GKI e non-GKI.
Gli script orchestrano repo sync, applicazione di patch, modifiche al defconfig, packaging e pubblicazione dei release per più famiglie di dispositivi e branch del kernel.
Integrano in modo esteso i flussi di patch KernelSU e SUSFS e includono varianti di build specifiche dei vendor per Pixel, OnePlus, Xiaomi e altri.
Il pubblico principale sono modder avanzati di kernel Android e ricercatori di sicurezza mobile; alcuni script potrebbero ormai essere obsoleti.
