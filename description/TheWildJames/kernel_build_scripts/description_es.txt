Este proyecto es una colección de scripts Bash de automatización para compilar kernels de Android en objetivos GKI y no GKI.
Los scripts orquestan el repo sync, la aplicación de parches, los cambios de defconfig, el empaquetado y la publicación de releases para varias familias de dispositivos y ramas de kernel.
Integran en gran medida los flujos de parches de KernelSU y SUSFS, e incluyen variantes de build específicas de fabricantes para Pixel, OnePlus, Xiaomi y otros.
El público principal son modders avanzados de kernels de Android e investigadores de seguridad móvil; algunos scripts pueden estar ya desactualizados.
