Este proyecto es un spoofer de HWID (Hardware Identification) para Windows, diseñado para modificar identificadores de hardware y limpiar rastros de seguimiento.
Este proyecto muestra mis habilidades de programación, mi capacidad para resolver problemas y mi interés en distintas áreas de especialización.
Es principalmente útil para investigadores de seguridad en juegos e ingenieros inversos que estudian técnicas ofensivas en el ámbito de cheats / HWID.
