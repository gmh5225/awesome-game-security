Кроссплатформенный сканер памяти из командной строки для Android и Linux с JIT-компиляцией выражений сканирования через SLJIT, поиском цепочек указателей с настраиваемой глубиной и ограничениями смещений, дифференциальной фильтрацией на основе снимков, интерактивным ncurses TUI и встроенной поддержкой инспекции игровых объектов Unity (U3D).
