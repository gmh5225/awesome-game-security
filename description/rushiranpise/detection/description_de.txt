Dieses Projekt ist eine Sammlung von Android-Erkennungstechniken zur Identifizierung von Root-Zugriff, dem Xposed-Framework, Frida, VPN-Verbindungen, Entwickleroptionen und anderen sicherheitsrelevanten Gerätezuständen. Die Java-Bibliothek implementiert für jede Kategorie mehrere Prüfungen mittels Dateisystemscan, Prozessenumeration, Eigenschaftsabfragen und Inspektion von Native Code. Sie richtet sich an Android-App-Entwickler und Mobile-Security-Ingenieure, die Integritätsprüfungen der Laufzeitumgebung implementieren.
