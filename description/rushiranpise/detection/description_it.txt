Questo progetto è una raccolta di tecniche di rilevamento Android per identificare l'accesso root, il framework Xposed, Frida, le connessioni VPN, le opzioni sviluppatore e altri stati del dispositivo rilevanti per la sicurezza. La libreria Java implementa più controlli per ogni categoria tramite scansione del file system, enumerazione dei processi, query sulle proprietà e ispezione del codice nativo. È pensata per sviluppatori di app Android e ingegneri di sicurezza mobile che implementano controlli di integrità dell'ambiente di runtime.
