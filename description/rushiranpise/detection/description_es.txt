Este proyecto es una colección de técnicas de detección en Android para identificar acceso root, el framework Xposed, Frida, conexiones VPN, opciones de desarrollador y otros estados del dispositivo relevantes para la seguridad. La biblioteca Java implementa múltiples comprobaciones por cada categoría mediante el análisis del sistema de archivos, la enumeración de procesos, consultas de propiedades e inspección de código nativo. Está dirigida a desarrolladores de aplicaciones Android e ingenieros de seguridad móvil que implementan comprobaciones de integridad del entorno de ejecución.
