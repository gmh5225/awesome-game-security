Этот проект — сборник техник обнаружения на Android для выявления root-доступа, фреймворка Xposed, Frida, VPN-соединений, параметров разработчика и других состояний устройства, значимых для безопасности. Java-библиотека реализует несколько проверок для каждой категории с помощью сканирования файловой системы, перечисления процессов, запросов свойств и анализа нативного кода. Она рассчитана на разработчиков Android-приложений и инженеров по мобильной безопасности, внедряющих проверки целостности среды выполнения.
