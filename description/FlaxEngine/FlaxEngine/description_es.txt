Este proyecto es un motor de juegos 3D moderno y un editor de código fuente completo, pensados para el desarrollo de juegos de alto rendimiento. Usa una arquitectura mixta de C++ y C#, con capas de renderizado y de plataforma, además de flujos de trabajo de scripting administrado y herramientas. El repositorio incluye el código del runtime del motor, el código del editor, scripts de compilación y soporte multiplataforma para Windows, Linux y macOS con tecnologías como .NET y Vulkan. Su público principal son desarrolladores de juegos y programadores de motores que necesitan un control profundo a nivel de código fuente y extensibilidad.
