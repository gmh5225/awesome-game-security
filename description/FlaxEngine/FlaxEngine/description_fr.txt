Ce projet est un moteur de jeu 3D moderne et un éditeur en sources complètes, conçus pour le développement de jeux haute performance. Il repose sur une architecture mixte C++ et C#, avec des couches de rendu et de plateforme ainsi que des workflows de scripting managé et d'outils. Le dépôt inclut le code runtime du moteur, le code de l'éditeur, les scripts de build, et un support multiplateforme pour Windows, Linux et macOS avec des technologies telles que .NET et Vulkan. Son public principal est les développeurs de jeux et les programmeurs de moteurs qui ont besoin d'un contrôle profond au niveau du code source et d'extensibilité.
