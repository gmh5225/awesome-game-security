Questo progetto è un motore di gioco 3D moderno e un editor a sorgente completa, progettati per lo sviluppo di giochi ad alte prestazioni. Usa un'architettura mista C++ e C#, con layer di rendering e di piattaforma oltre a workflow di scripting gestito e tooling. Il repository include il codice runtime del motore, il codice dell'editor, gli script di build e il supporto multipiattaforma per Windows, Linux e macOS con tecnologie come .NET e Vulkan. Il pubblico principale è costituito da sviluppatori di giochi e programmatori di engine che necessitano di un controllo approfondito a livello di sorgente e di estensibilità.
