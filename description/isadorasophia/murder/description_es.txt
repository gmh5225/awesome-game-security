Este proyecto es Murder Engine, un motor de juegos 2D de pixel art escrito en C# y basado en MonoGame. Ofrece una arquitectura ECS (Entity Component System), integración de sprites de Aseprite, sistemas de diálogo, gestión de entrada, física, efectos de partículas y un editor visual para crear juegos de pixel art narrativos. El motor prioriza un diseño de API limpio y un renderizado 2D eficiente. Está pensado para desarrolladores independientes que crean juegos 2D impulsados por la historia con estética pixel art.
