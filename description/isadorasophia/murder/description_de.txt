Dieses Projekt ist Murder Engine, eine in C# geschriebene Pixel-Art-2D-Spiel-Engine auf Basis von MonoGame. Sie bietet eine ECS-Architektur (Entity Component System), Aseprite-Sprite-Integration, Dialogsysteme, Eingabeverwaltung, Physik, Partikeleffekte und einen visuellen Editor zum Erstellen narrativer Pixel-Art-Spiele. Die Engine legt Wert auf ein klares API-Design und effizientes 2D-Rendering. Sie richtet sich an Indie-Entwickler, die storygetriebene 2D-Spiele mit Pixel-Art-Ästhetik erstellen.
