Этот проект — Murder Engine, 2D-движок для pixel art игр на C#, построенный на MonoGame. Он предоставляет архитектуру ECS (Entity Component System), интеграцию спрайтов Aseprite, системы диалогов, управление вводом, физику, эффекты частиц и визуальный редактор для создания нарративных pixel art игр. Движок делает акцент на чистом API и эффективном 2D-рендеринге. Он ориентирован на инди-разработчиков, создающих сюжетные 2D-игры в эстетике pixel art.
