Лёгкая библиотека CPU-профилирования на основе scope для игр: собирает иерархические данные о времени выполнения с минимальными накладными расходами через thread-local storage и платформенные высокоточные таймеры. Включает инспектор на ImGui/ImPlot для визуализации flame graph и распределений времени кадра, а также сериализацию файлов захвата с сжатием LZ4.
