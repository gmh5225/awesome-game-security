Doriax は、2D/3D ゲーム制作向けの統合ビジュアルエディタを備えたクロスプラットフォーム C++ ゲームエンジンです。エンティティ・コンポーネントアーキテクチャを中核とし、Lua スクリプティングバインディング、シーン管理、アニメーション、パーティクル、マテリアル、地形ツールを提供します。レンダリングは OpenGL を基盤に、Sokol ベースのグラフィックス層と GLFW または SDL のウィンドウバックエンドを採用し、glslang と SPIRV-Cross によるシェーダーツールも備えます。ImGui ベースのエディタは、シーンビューポート、ギズモ、リソースブラウザ、API サジェスト付きコード編集、エクスポートワークフロー、任意の AI 支援編集機能を提供します。デスクトップ、モバイル、Web にわたるフルなエンジン＋エディタ構成を求めるゲーム開発者を対象としています。
