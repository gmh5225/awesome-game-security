Doriax는 2D 및 3D 게임 제작을 위한 통합 비주얼 에디터를 갖춘 크로스플랫폼 C++ 게임 엔진입니다. 엔티티-컴포넌트 아키텍처를 중심으로 Lua 스크립팅 바인딩, 씬 관리, 애니메이션, 파티클, 머티리얼, 지형 도구를 제공합니다. 렌더링은 OpenGL을 기반으로 하며, Sokol 기반 그래픽 계층과 GLFW 또는 SDL 윈도우 백엔드, 그리고 glslang과 SPIRV-Cross를 통한 셰이더 툴링을 사용합니다. ImGui 기반 에디터는 씬 뷰포트, 기즈모, 리소스 브라우저, API 제안이 있는 코드 편집, 내보내기 워크플로, 선택적 AI 지원 편집 기능을 제공합니다. 데스크톱, 모바일, 웹 플랫폼을 아우르는 완전한 엔진+에디터 스택을 원하는 게임 개발자를 대상으로 합니다.
