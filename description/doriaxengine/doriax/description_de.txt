Doriax ist eine plattformübergreifende C++-Spiel-Engine mit integriertem visuellen Editor zum Erstellen von 2D- und 3D-Spielen. Im Zentrum steht eine Entity-Component-Architektur mit Lua-Skriptbindungen, Szenenverwaltung, Animation, Partikeln, Materialien und Terrain-Werkzeugen. Das Rendering basiert auf OpenGL mit einer Sokol-basierten Grafikschicht und GLFW- oder SDL-Fenster-Backends sowie Shader-Tools über glslang und SPIRV-Cross. Der ImGui-basierte Editor bietet Szenen-Viewports, Gizmos, Ressourcenbrowser, Codebearbeitung mit API-Vorschlägen, Export-Workflows und optionale KI-unterstützte Bearbeitungsfunktionen. Er richtet sich an Spieleentwickler, die einen vollständigen Engine-plus-Editor-Stack für Desktop, Mobile und Web benötigen.
