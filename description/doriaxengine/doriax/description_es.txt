Doriax es un motor de juegos C++ multiplataforma con un editor visual integrado para crear juegos 2D y 3D. Se centra en una arquitectura de entidad-componente con enlaces de scripting en Lua, gestión de escenas, animación, partículas, materiales y herramientas de terreno. El renderizado se basa en OpenGL con una capa gráfica basada en Sokol y backends de ventana GLFW o SDL, además de herramientas de shaders mediante glslang y SPIRV-Cross. El editor basado en ImGui ofrece viewports de escena, gizmos, exploradores de recursos, edición de código con sugerencias de API, flujos de exportación y funciones opcionales de edición asistida por IA. Está dirigido a desarrolladores de juegos que buscan una pila completa de motor más editor para escritorio, móvil y web.
