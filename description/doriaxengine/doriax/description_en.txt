Doriax is a cross-platform C++ game engine with an integrated visual editor for building 2D and 3D games. It centers on an entity-component architecture with Lua scripting bindings, scene management, animation, particles, materials, and terrain tools. Rendering is built around OpenGL with a Sokol-based graphics layer and GLFW or SDL window backends, plus shader tooling via glslang and SPIRV-Cross. The ImGui-based editor provides scene viewports, gizmos, resource browsers, code editing with API suggestions, export workflows, and optional AI-assisted editing features. It targets game developers who want a full engine-plus-editor stack spanning desktop, mobile, and web platforms.
