Este proyecto es una biblioteca C++ solo de cabeceras para realizar syscalls de Windows directos e indirectos sin depender de importaciones de API estándar. Resuelve los números de syscall en tiempo de ejecución recorriendo las estructuras de datos del cargador PEB y analizando las tablas de exportación de ntdll, y admite técnicas como trampolines de syscalls indirectos para la evasión de EDR. Resulta especialmente útil para investigadores de seguridad ofensiva y analistas de anti-cheat que estudian el bypass de ganchos de API a nivel de syscall y las estrategias de detección.
