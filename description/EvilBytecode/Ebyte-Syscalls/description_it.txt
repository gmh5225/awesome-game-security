Questo progetto è una libreria C++ header-only per eseguire syscall Windows diretti e indiretti senza dipendere dagli import delle API standard. Risolve i numeri di syscall a runtime scorrendo le strutture dati del loader PEB e analizzando le tabelle di export di ntdll, supportando tecniche come i trampoline di syscall indiretti per l'evasione EDR. È principalmente utile per ricercatori di sicurezza offensiva e analisti anti-cheat che studiano il bypass degli hook API a livello di syscall e le relative strategie di rilevamento.
