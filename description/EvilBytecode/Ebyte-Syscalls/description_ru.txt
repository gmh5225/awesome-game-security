Этот проект — header-only библиотека на C++ для выполнения прямых и косвенных Windows-системных вызовов без зависимости от стандартных импортов API. Она определяет номера syscall во время выполнения, обходя структуры данных загрузчика PEB и разбирая таблицы экспорта ntdll, и поддерживает такие техники, как трамплины косвенных syscall для обхода EDR. Библиотека в первую очередь полезна исследователям offensive security и аналитикам античита, изучающим обход хуков API на уровне syscall и стратегии их обнаружения.
