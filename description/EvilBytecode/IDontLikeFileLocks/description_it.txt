Questo progetto è una raccolta di ricerca in C++ per estrarre dati da file bloccati da altri processi in esecuzione.
Dimostra diverse tecniche, tra cui il furto di handle di sezioni mappate in memoria, la duplicazione e la chiusura di handle remoti e i relativi flussi di lavoro per aggirare i lock.
Gli esempi riguardano scenari come la lettura di database del browser senza interromperlo, evidenziando un comportamento di acquisizione file a basso rumore.
Il caso d'uso principale è la ricerca di sicurezza autorizzata e la forensics, focalizzate sulla comprensione dell'evasione dei lock sui file e delle tattiche degli info-stealer.
