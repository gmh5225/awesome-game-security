Этот проект — ориентированный на Windows набор средств безопасности на Go для защиты программ от анализа и вмешательства.
Он включает модули anti-debug, anti-virtualization, anti-DLL-injection и обнаружения хуков, организованные как переиспользуемые внутренние компоненты.
Код опирается на низкоуровневое взаимодействие с Windows API для выявления подозрительных условий во время выполнения и подачи защитных сигналов.
Основной сценарий использования — оборонительные исследования и добавление проверок против реверс-инжиниринга в чувствительные к безопасности приложения на Go.
