Este proyecto es un kit de herramientas de seguridad centrado en Windows, escrito en Go, para endurecer programas frente al análisis y la manipulación.
Incluye módulos anti-debug, anti-virtualización, anti-inyección de DLL y detección de hooks, organizados como componentes internos reutilizables.
El código se basa en interacciones de bajo nivel con la API de Windows para detectar condiciones de ejecución sospechosas y emitir señales defensivas.
Su caso de uso principal es la investigación defensiva y añadir comprobaciones anti-ingeniería inversa a aplicaciones Go sensibles a la seguridad.
