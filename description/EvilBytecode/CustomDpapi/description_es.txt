Este proyecto es una prueba de concepto en C++ que llama directamente a la interfaz RPC no documentada de DPAPI, eludiendo la API estándar CryptUnprotectData. Demuestra cómo invocar NdrClientCall3 con los parámetros correctos para descifrar datos a través del endpoint RPC de lsass, basándose en la ingeniería inversa de los internos de dpapi.dll. Resulta especialmente útil para investigadores de seguridad que estudian la protección de credenciales de Windows, los internos de DPAPI y las superficies de ataque basadas en RPC.
