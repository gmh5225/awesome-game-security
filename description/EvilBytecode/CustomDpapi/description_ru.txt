Этот проект — доказательство концепции на C++, которое напрямую вызывает недокументированный RPC-интерфейс DPAPI, обходя стандартный API CryptUnprotectData. Он показывает, как вызывать NdrClientCall3 с корректными параметрами для расшифровки данных через RPC-эндпоинт lsass, опираясь на реверс-инжиниринг внутренностей dpapi.dll. В первую очередь полезен исследователям безопасности, изучающим защиту учётных данных Windows, внутреннее устройство DPAPI и поверхности атаки на базе RPC.
