Este proyecto es un virtualizador de código x86-64 open source de tipo bin-to-bin, diseñado para ofrecer una protección sólida frente a la ingeniería inversa.
Traduce las instrucciones x86-64 de uso habitual a un bytecode personalizado, que es ejecutado por una máquina virtual segura y construida específicamente para ello.
Resulta especialmente útil para ingenieros de anti-cheat e investigadores de seguridad defensiva que trabajan en el ámbito de motores anti-cheat / de ofuscación.
