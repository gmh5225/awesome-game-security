Ce projet est un virtualiseur de code x86-64 open source de type bin-to-bin, conçu pour offrir une protection solide contre le reverse engineering.
Il traduit les instructions x86-64 courantes en un bytecode personnalisé, exécuté par une machine virtuelle sécurisée et conçue à cet effet.
Il est principalement utile aux ingénieurs anti-cheat et aux chercheurs en sécurité défensive travaillant dans le domaine des moteurs anti-cheat / d'obfuscation.
