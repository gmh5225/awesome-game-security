Questo progetto è il codice sorgente del kernel Android per Xiaomi Sweet (Redmi Note 10 Pro) con patch di integrazione KernelSU. Include l’albero completo del kernel Qualcomm, l’integrazione con il sistema di build Android, patch per gli hook KernelSU e configurazioni specifiche del dispositivo. È principalmente utile per gli sviluppatori di kernel Android e per i ricercatori di sicurezza mobile che lavorano con kernel personalizzati abilitati a KernelSU su dispositivi Xiaomi basati su Qualcomm.
