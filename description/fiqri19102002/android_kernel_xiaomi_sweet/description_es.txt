Este proyecto es el código fuente del kernel de Android para Xiaomi Sweet (Redmi Note 10 Pro) con parches de integración de KernelSU. Incluye el árbol completo del kernel de Qualcomm, la integración con el sistema de compilación de Android, parches de hooks de KernelSU y configuraciones específicas del dispositivo. Es especialmente útil para desarrolladores de kernels de Android e investigadores de seguridad móvil que trabajan con kernels personalizados con KernelSU en dispositivos Xiaomi basados en Qualcomm.
