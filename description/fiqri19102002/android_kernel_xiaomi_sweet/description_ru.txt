Этот проект содержит исходный код ядра Android для Xiaomi Sweet (Redmi Note 10 Pro) с патчами интеграции KernelSU. В него входят полное дерево ядра Qualcomm, интеграция со системой сборки Android, патчи хуков KernelSU и конфигурации для конкретного устройства. Он в основном полезен разработчикам ядер Android и исследователям мобильной безопасности, работающим с кастомными ядрами с поддержкой KernelSU на устройствах Xiaomi на платформе Qualcomm.
