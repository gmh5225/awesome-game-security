# Unreal-Binary-Builder

C# tooling focused on building Unreal Engine from source: asset-pipeline / editor / plugin-oriented workflow helpers for compiling UE trees rather than runtime cheat injection. Useful for tooling developers and reverse engineers studying how UE source builds are produced and packaged. (source: wiki/sources/descriptions/ryanjon2040__Unreal-Binary-Builder.md)

Sits under README `Game Tools` beside other UE builders and network profilers such as [[unreal-network-profiler]].
