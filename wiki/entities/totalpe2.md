# TotalPE2

Windows PE (Portable Executable) viewer with a modern WPF UI: headers, sections, imports/exports, resources, debug directories, relocations, TLS, exception handling, and .NET metadata, plus drag-and-drop, hex view, and resource extraction. Aimed at reverse engineers, malware analysts, and Windows developers who need a quick visual PE structure inspector. (source: wiki/sources/descriptions/zodiacon__TotalPE2.md)

Useful for triage of game/client binaries and modules before deeper IDA/Ghidra work—not a debugger or unpacker.
