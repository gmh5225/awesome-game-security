# ida-jm-xorstr-decrypt-plugin

IDA Pro plugin (Python) that attempts to decrypt JM Xorstr in some x64 binaries. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane studying string-obfuscation recovery. (source: wiki/sources/descriptions/yubie-re__ida-jm-xorstr-decrypt-plugin.md)

Not a general unpacker—scoped to JM Xorstr-style string decryption inside IDA on selected x64 targets.
