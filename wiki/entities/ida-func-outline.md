# IDAFuncOutline

IDA Pro plugin that builds function outlines and call-hierarchy trees (callers/callees with depth control) to improve decompiled-code readability—especially for iOS ARM64 binaries. Aimed at reverse engineers in the cheat / IDA Plugins lane who need quick visual triage of call structure. (source: wiki/sources/descriptions/richor1042__IDAFuncOutline.md)

UI/navigation helper for call graphs—not a deobfuscator or unpacker.
