# dma-speedtest-memflow-rs

Windows **DMA speed-test** tool (Rust): benchmarks memory **read/write throughput** (MiB/s), operation rate, and latency through the **memflow** framework, with egui GUI and CLI. Supports **PCILeech (FPGA)** and **native** connectors, probes targets such as `explorer.exe` / `ntdll.dll`, and exports CSV/JSON — useful for evaluating DMA hardware and access performance in RE and game-security workflows. (source: wiki/sources/descriptions/sh1ftd__dma-speedtest-memflow-rs.md)
