# SmallWorld

Multi-emulator binary analysis framework with a unified interface for running code snippets through angr, Ghidra, PANDA, and Unicorn. Supports coverage analysis, crash triage, and cross-platform firmware testing—aimed at streamlined binary harnessing for dynamic analysis rather than full-system game emulation. Useful when RE or AC researchers need the same harness across symbolic, emulation, and whole-system backends. (source: wiki/sources/descriptions/smallworld-re__smallworld.md)
