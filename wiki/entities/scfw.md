# scfw

Cross-platform C++ framework for building **Windows shellcode**. Aimed at anti-cheat engineers and defensive researchers studying shellcode engine & tricks (payload construction / position-independent code patterns) rather than shipping as an AC product. (source: wiki/sources/descriptions/wbenny__scfw.md)

Complements in-process asm/shellcode prototyping such as [[quickasm]] and raw-shellcode packing such as [[2pack]].
