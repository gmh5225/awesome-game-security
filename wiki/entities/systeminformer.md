# System Informer

Open-source Windows system monitoring and analysis toolkit, formerly known as Process Hacker. Used by game-security researchers and reverse engineers studying offensive techniques in the Cheat / Windows kernel explorer lane—process, handle, module, and related host inspection rather than an AC product. (source: wiki/sources/descriptions/winsiderss__systeminformer.md)

Build note from the project: after clone, run `build_init.cmd` under `build/` once (re-run when tools or third-party libraries update). (source: wiki/sources/descriptions/winsiderss__systeminformer.md)
