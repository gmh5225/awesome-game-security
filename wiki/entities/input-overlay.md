# input-overlay

OBS Studio plugin (C++) that renders keyboard, mouse, and gamepad presses as a livestream overlay. Captures real-time input events and draws them with customizable layouts, skins, and positioning via OBS’s rendering pipeline—aimed at streamers showing controller/keyboard input on broadcast, catalogued under Keyboard Mapper. (source: wiki/sources/descriptions/univrsal__input-overlay.md)
