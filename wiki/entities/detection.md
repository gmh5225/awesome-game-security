# detection

Android Java library collecting runtime environment-integrity checks: root access, Xposed, [[frida]], VPN, developer options, and related device states. Implements multiple probes per category via filesystem scanning, process enumeration, property queries, and native-code inspection. Aimed at app developers and mobile security engineers building anti-cheat / anti-tamper signals. (source: wiki/sources/descriptions/rushiranpise__detection.md)

README lane: `[Collection of Various Root Detection Apps for Android]` — sits with Magisk-focused [[magiskdetector]], broader RASP [[droidshield]] / freeRASP, and emulator heuristics [[anti-emulator]].
