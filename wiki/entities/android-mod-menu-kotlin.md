# Android-Mod-Menu-Kotlin

Android game mod-menu template in Kotlin with native C++ hooking: floating overlay via Android views, native library injection for memory patching and function hooks, and a JNI bridge between the Kotlin UI and native game modifications. Aimed at Android modders who prefer Kotlin UI with C++ low-level manipulation; README lane `[IL2CPP Menu]`. (source: wiki/sources/descriptions/springmusk026__Android-Mod-Menu-Kotlin.md)

Complements the Java `WindowManager` + semi-JNI sibling ([[android-modmenu-semijni]]), GLES ImGui Unity templates ([[imgui-unity]], [[imgui-unity-with-layout]]), Android native ImGui apps ([[android-native-app-imgui]]), and iOS ImGui mod menus ([[imgui-ios-mod-menu]]) with a Kotlin-overlay path.
