# PolyHook

x86/x64 hooking library with an abstract C++11 interface over multiple hooking methods. `Tests.cpp` samples cover each method; aimed at game-security researchers and reverse engineers studying offensive user-mode hook/trampoline techniques in the cheat / hook lane. Predecessor to [[polyhook-2-0]]. (source: wiki/sources/descriptions/stevemk14ebr__PolyHook.md)
