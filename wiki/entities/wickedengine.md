# WickedEngine

Open C++ 3D game engine. Applications link against the `WickedEngine_Windows` static library after building that project. Aimed at game developers, engine programmers, and graphics researchers studying a full engine source tree. (source: wiki/sources/descriptions/turanszkij__WickedEngine.md)

Sits in the Game Engine / source lane—an OSS C++ 3D engine codebase for development and RE study, rather than a cheat or anti-cheat artifact.
