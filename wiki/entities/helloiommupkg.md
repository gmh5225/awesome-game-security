# HelloIommuPkg

tandasat sample DXE runtime driver that demonstrates how to program DMA remapping (IOMMU). Simplified learning example for anti-cheat engineers and defensive researchers in the Detection:DMA lane—not intended for production systems. (source: wiki/sources/descriptions/tandasat__HelloIommuPkg.md)

Pairs with the [[iommu]] concept and firmware-layer DMA hardening such as [[x670e-tomahawk-anticheat-update]]. Same author as [[minivisorpkg]] (educational VT-x HV) and [[sushi]] (PatchGuard monitoring).
