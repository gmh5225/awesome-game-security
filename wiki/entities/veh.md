# veh

Windows **Vectored Exception Handler (VEH)** software-debugger framework that implements debugger-like behavior without the Debug API. It registers handlers for breakpoints, single-step exceptions, and access violations so researchers can inspect processes that detect and block conventional debuggers. README tags it as a Cheat Engine plugin for manual-mapped VEH DLLs. (source: wiki/sources/descriptions/user23333__veh.md)

Complements VEH-chain dump tooling such as [[veh-dumper]] and page-protection / VEH anti-tamper samples such as [[no-access-protection]] and [[voidmaw]].
