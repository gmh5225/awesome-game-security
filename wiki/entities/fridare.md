# fridare

Frida repackaging and anti-detection toolkit for iOS and Android. Shell scripts plus a Go hex-replacement utility patch recognizable Frida strings, symbols, and artifact names so modified `frida-server` builds evade common anti-instrumentation checks; includes automated install docs for rooted and rootless iOS. Aimed at mobile reverse engineers who need stealth Frida deployments against anti-Frida detectors. (source: wiki/sources/descriptions/suifei__fridare.md)
