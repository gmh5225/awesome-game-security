# syscalls-cpp

Modern C++20 header-only library for advanced direct system call invocation, built around modularity. Aimed at anti-cheat engineers and defensive researchers in the Anti Cheat → Compile Time lane. (source: wiki/sources/descriptions/sapdragon__syscalls-cpp.md)

Complements SSN extraction tooling such as [[ntsleuth]] and Ring3 syscall-return inspection such as [[instrumentation-callback-syscall-logger]] / [[hidden-syscall-monitoring]]—invocation stubs rather than table dump or telemetry.
