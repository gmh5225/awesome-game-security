# DIE-engine-web

WebAssembly port of Detect It Easy (DIE) that runs the PE/ELF/Mach-O binary identification engine entirely in the browser. Compiled from the original Qt-based C++ codebase via Emscripten with a web frontend for file-format, packer, and compiler detection without a native install. (source: wiki/sources/descriptions/t0asts__DIE-engine-web.md)

Useful for quick packed/protected client triage alongside PE viewers and Binary Packer research samples—not a debugger or unpacker.
