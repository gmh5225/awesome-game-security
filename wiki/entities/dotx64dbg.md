# DotX64Dbg

Framework for writing [[x64dbg]] plugins in .NET 6.0 / C#, aimed at easier plugin authoring and live edit/debug of plugins while registering custom commands and expressions on the fly. Useful for game-security researchers and reverse engineers in the Cheat / Debugging / x64dbg Plugins lane. (source: wiki/sources/descriptions/x64dbg__DotX64Dbg.md)

Not a standalone debugger—hosts managed plugins on top of [[x64dbg]].
