# Sark

IDAPython helper library (“IDAPython Made Easy”) that simplifies writing IDA Pro scripts and plugins. Python 2 / older-IDA support continues on the IDA-6.x branch for bugfixes and community contributions. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane. (source: wiki/sources/descriptions/tmr232__Sark.md)

Not a standalone disassembler—scoped as an IDAPython convenience layer over IDA’s scripting API.
