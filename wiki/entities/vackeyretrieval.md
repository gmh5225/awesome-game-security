# VACKeyRetrieval

C++ research repo that retrieves the VAC module ICE encryption key. Useful for game-security researchers and reverse engineers studying Valve Anti-Cheat under the cheat / explore anticheat system:vac lane. (source: wiki/sources/descriptions/shuruk421__VACKeyRetrieval.md)

Companion research surface to [[vac3-dumper]] (timed module dumps) and [[vac3-inhibitor]] (hooking / memory analysis): this repo focuses on ICE key recovery for VAC module crypto rather than dump or runtime inhibition.
