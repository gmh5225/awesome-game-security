# TDE

Themida Devirtualization Engine — an IDA plugin that recovers native code from Themida virtualization. Aimed at game-security researchers and reverse engineers working the Cheat → Fix Themida lane (offensive technique study on Themida-protected game / AC binaries). (source: wiki/sources/descriptions/sodareverse__TDE.md)

Companion surface to [[themida-research]] (VM_CONTEXT / handler notes) and Fix VMP demos such as [[vmdevirt-vtil]] / [[novmpy]]: plugin-driven Themida recovery rather than research notes or VMProtect lifting.
