# Imgui-Native-ModMenu

Android native mod-menu template that draws Dear ImGui through OpenGL ES in a C++ framework, with touch input handling and JNI hooks for game functions. Includes toggles, sliders, and input fields for cheat features—aimed at Android game modders building visual menus via native code injection. (source: wiki/sources/descriptions/s4m33r89__Imgui-Native-ModMenu.md)

Complements Android native ImGui apps ([[android-native-app-imgui]]), Unity-on-Android ImGui menus ([[imgui-unity]]), Java/Kotlin floating mod menus ([[android-modmenu-semijni]], [[android-mod-menu-kotlin]]), and iOS ImGui samples ([[imgui-ios-mod-menu]]) in the cheat / `[Imgui Menu for Android]` lane.
