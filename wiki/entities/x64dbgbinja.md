# x64dbgbinja

Python Binary Ninja plugin from the x64dbg org, aimed at bridging Binary Ninja workflows with the x64dbg plugin ecosystem. Useful for game-security / RE researchers studying cheat-side static analysis alongside x64dbg-oriented tooling. (source: wiki/sources/descriptions/x64dbg__x64dbgbinja.md)

Not a standalone debugger—scoped to Binary Ninja plugin development in the Cheat Binary Ninja / x64dbg Plugins lane.
