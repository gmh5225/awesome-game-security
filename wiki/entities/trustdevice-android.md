# TrustDevice Android

TrustDecision device-fingerprinting SDK for Android. Kotlin/Java library that generates unique device identifiers and risk signals for fraud detection, with instrumented tests, privacy-policy docs, and ProGuard config for production. Useful for anti-cheat engineers and mobile security researchers studying Android device fingerprinting and device-integrity verification. (source: wiki/sources/descriptions/trustdecision__trustdevice-android.md)

README lane: `[Android]`. Sibling to [[trustdevice-ios]]; sits alongside Android RASP / attestation samples such as [[droidshield]] and [[keyattestation]] in the mobile HWID / integrity lane.
