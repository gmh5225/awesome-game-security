# Static-Variables-Obfuscator-UE4

UE4 code plugin that obfuscates static (non-dynamic) variables so they are hard to locate in process memory—raising the cost of Cheat Engine–style value scanning and in-game data manipulation. Aimed at anti-cheat / Unreal engine-protection research. (source: wiki/sources/descriptions/zompi2__Static-Variables-Obfuscator-UE4.md)

Defensive reference under `Game Engine Protection:Unreal` / encrypt-variable style protections—not a runtime AC product.
