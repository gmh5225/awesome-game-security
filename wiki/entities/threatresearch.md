# ThreatResearch

Security Joes collection of threat-research tools, scripts, and Ghidra/IDA plugins: malware-family analysis scripts, Ghidra deobfuscation extensions, YARA rules, and IR-oriented RE utilities (Python/Java) that accompany published reports. Aimed at malware analysts and researchers extending RE toolkits—useful adjacent tooling for game-security static analysis in the Cheat Ghidra Plugins / RE Tools lane. (source: wiki/sources/descriptions/securityjoes__ThreatResearch.md)

Toolkit/report companion set rather than a single AC product or game-specific dumper.
