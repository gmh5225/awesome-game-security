# XrefsExt

IDA Pro plugin focused on extended cross-reference (xrefs) workflows and plugin development. Aimed at game-security researchers and reverse engineers working in the cheat / IDA Plugins lane. (source: wiki/sources/descriptions/zengfr__XrefsExt.md)

Not a standalone disassembler—extends IDA’s xref navigation / analysis surface for binary RE.
