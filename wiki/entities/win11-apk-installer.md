# win11_apk_installer

Python Win11 APK installer under the README `WSA` lane (sergiovillaverde). Aimed at Android-on-Windows and platform-integration researchers installing / sideloading APKs into Windows Subsystem for Android–style environments. Useful as a small WSA tooling sample beside pacman/builds/kernel-SU siblings in the same category. (source: wiki/sources/descriptions/sergiovillaverde__win11_apk_installer.md)
