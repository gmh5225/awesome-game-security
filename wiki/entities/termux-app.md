# Termux

Android terminal emulator and Linux environment app that provides a full Linux CLI without rooting. Ships a custom sysroot with proot isolation, its own pkg/apt package manager (compilers, interpreters, editors, networking, frameworks), and common research tooling such as SSH, Python, Node.js, and Rust. Canonical Cheat Android Terminal Emulator reference for on-device security research and power-user workflows. (source: wiki/sources/descriptions/termux__termux-app.md)
