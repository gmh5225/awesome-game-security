# big5-decode-ida

Python IDA plugin for decoding selected bytes as Big5 (Traditional Chinese) text. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane when binaries embed Big5 string data that IDA’s default encodings miss. (source: wiki/sources/descriptions/sean2077__big5-decode-ida.md)

Encoding helper for static RE—not a deobfuscator, unpacker, or full string-crypto suite.
