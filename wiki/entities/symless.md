# Symless

IDA Pro plugin for automated structure and type recovery on stripped C/C++ binaries. Uses data-flow analysis and memory-access patterns to reconstruct struct definitions, infer field types and sizes, and propagate types across functions without debug symbols—improving Hex-Rays decompilation output for RE of games and anti-cheat modules that ship stripped. (source: wiki/sources/descriptions/thalium__symless.md)

Not a debugger or unpacker—scoped to IDA-side type/structure reconstruction for stripped binaries.
