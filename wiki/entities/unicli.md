# UniCli

CLI to drive the Unity Editor from a terminal: compile, test, build, and inspect GameObjects, with 80+ commands and AI-agent packaging (Claude Code plugin / Agent Skills). Written in C#. (source: wiki/sources/descriptions/yucchiy__UniCli.md)

Fits the Game Engine Plugins:Unity and Game Develop (AI Agents) lanes—useful when automating editor workflows or wiring agents into Unity project build/inspect loops rather than reverse-engineering shipped IL2CPP/Mono binaries.
