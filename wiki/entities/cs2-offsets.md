# CS2-OFFSETS

Regularly updated Counter-Strike 2 memory-offset and netvar dump repo. Publishes struct offsets, entity field addresses, and interface pointers extracted from CS2 client/server binaries after game updates, as C++ headers and JSON for cheat/mod and Source 2 layout research. (source: wiki/sources/descriptions/sezzyaep__CS2-OFFSETS.md)

Treat as a live offset feed that must track CS2 patches—stale dumps break external/internal memory tooling that depends on them.
