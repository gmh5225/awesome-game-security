# obfCoder

C++ code obfuscation tool that applies transformation techniques to source code and compiled binaries. Ships with CMake build support, experiment examples, OpenSSL crypto integration, and before/after demonstration programs. Aimed at software-protection researchers studying obfuscation and anti-RE transformations under Anti Cheat → Obfuscation Engine. (source: wiki/sources/descriptions/ssyuqixe__obfCoder.md)

Useful as a source+binary obfuscation research reference alongside compile-time libraries and post-compile PE transformers—not a commercial protector or unpacker.
