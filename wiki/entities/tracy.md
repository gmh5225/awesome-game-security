# Tracy

Real-time C++ frame profiler for games and other interactive apps: CPU zone markup, GPU timing (OpenGL / Vulkan / Direct3D), memory-allocation tracking, lock contention, context-switch views, and source annotation. Ships as a lightweight in-process client plus a standalone viewer (timeline, flame graph, stats). Aimed at frame-time and bottleneck work in the Game Testing / gamedev performance lane. (source: wiki/sources/descriptions/wolfpld__tracy.md)
