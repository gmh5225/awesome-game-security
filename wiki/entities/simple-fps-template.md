# SimpleFPSTemplate

UE4 FPS demo template that intentionally “bloats” the starter C++ sample so it looks more complex than a minimal first-person project needs. Aimed at game developers, reverse engineers, and tooling builders studying Unreal C++ structure in the Game Develop / source lane—not a cheat or anti-cheat artifact. (source: wiki/sources/descriptions/tomlooman__SimpleFPSTemplate.md)

Complements other Unreal surfaces in the wiki (tutorials, SDK dumpers, asset graphs) by providing a fuller FPS template for engine-structure study rather than RE or protection tooling.
