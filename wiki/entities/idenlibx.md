# idenLibX

[[x64dbg]] plugin port of [[idenlib]] for identifying statically linked library functions during dynamic analysis. Scans loaded modules in the debugged process for known library function signatures and applies function names in real time. C++ plugin aimed at reverse engineers who prefer x64dbg for live debugging and need library-function identification. (source: wiki/sources/descriptions/secrary__idenLibX.md)

Sits in the Cheat / Library Function Identification / x64dbg Plugins lane—not a standalone signature database or static-only FLIRT-style tool. Static IDA counterpart: [[idenlib]]. (source: wiki/sources/descriptions/secrary__idenLib.md)
