# Stride

Open-source C# / .NET 3D game engine (formerly Xenko) with a full-featured editor. Provides PBR rendering, global illumination, visual scripting, Bullet physics, audio, UI, animation, particles, and VR, with Vulkan / Direct3D 11–12 / OpenGL backends. Aimed at developers building 3D games and interactive apps on C# and .NET. (source: wiki/sources/descriptions/stride3d__stride.md)

Sits in the Game Engine / C# 3D source lane—an OSS managed engine codebase for development and RE study of a complete editor/runtime tree, rather than a cheat or anti-cheat artifact.
