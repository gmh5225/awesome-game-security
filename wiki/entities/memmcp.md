# MemMCP

Python MCP (Model Context Protocol) server that exposes Cheat Engine–like memory tooling to AI assistants and agent workflows. Fits the Game Develop → MCP server lane while staying adjacent to classic memory-scanner / RE tooling; useful for game developers, reverse engineers, and tooling builders wiring agents into process-memory inspection rather than a desktop CE UI alone. (source: wiki/sources/descriptions/un4ckn0wl3z__MemMCP.md)

Complements agent-facing MCP bridges such as [[unity-mcp]] (editor automation) and [[apktool-mcp-server]] (APK RE) by targeting live memory / CE-style workflows instead of engine editors or package decode.
