# MoveCertificate

Magisk / KernelSU / APatch module that moves user-installed certificates into the system certificate store. Supports Android 7–15. Useful for game-security and reverse-engineering workflows that need system-trusted CA trust (e.g. HTTPS MITM on rooted devices) in the Cheat Magisk lane.
