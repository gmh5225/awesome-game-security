# SlothBP

x64dbg plugin for collaborative breakpoint management. Written in C/C++; aimed at plugin development and shared debugging workflows in the Cheat / x64dbg Plugins lane. Useful when multiple researchers need coordinated breakpoints on the same binary under [[x64dbg]]. (source: wiki/sources/descriptions/x64dbg__SlothBP.md)

Not a standalone debugger—extends [[x64dbg]] breakpoint UX for collaborative RE sessions.
