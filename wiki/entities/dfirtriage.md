# DFIRTriage

Python-based digital forensics and incident response (DFIR) triage tool for Windows endpoints. Automates collection of volatile and non-volatile artifacts—running processes, network connections, scheduled tasks, registry hives, event logs, prefetch files, and browser history—into a structured output directory for rapid initial triage and evidence preservation. (source: wiki/sources/descriptions/travisfoley__dfirtriage.md)

Useful lab/IR reference opposite Cheat Anti Forensics and alongside AC Information System & Forensics workflows that need a live host snapshot before deeper disk or memory analysis ([[file-recovery-tool]], [[volatility3]]).
