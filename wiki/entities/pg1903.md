# PG1903

Windows kernel driver research project that disables [[patchguard]] in real time on Windows 10 version 1903 by manipulating NX (no-execute) attributes on PatchGuard context pages. Includes C and x64 assembly memory-scanning routines to locate context structures and neutralize integrity checks. (source: wiki/sources/descriptions/zzhouhe__PG1903.md)

Primarily useful for studying PatchGuard internals, kernel patch-protection bypass techniques, and Windows kernel integrity mechanisms—not as a general-purpose AC bypass.
