# Solemn

Windows CLI that automates adding drivers to the HVCI `HvciDisallowedImages` custom blocklist. Hypervisor-Protected Code Integrity ([[hvci]], Memory Integrity) is a VBS feature; Solemn targets the operator/research workflow for extending that disallow list rather than bypassing CI. Framed for low-level Windows (and adjacent Linux/mobile) researchers in the Some Tricks / Windows Ring3 lane. (source: wiki/sources/descriptions/unkvolism__Solemn.md)
