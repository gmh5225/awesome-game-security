# DWEX

DWARF Explorer (DWEX) — a Python GUI for browsing DWARF debug information in ELF binaries and separate debug files. Uses pyelftools to parse and display the full DWARF tree (compilation units, types, functions, variables, line-number mappings) in an interactive tree view. Aimed at compiler developers, debugger authors, and reverse engineers inspecting DWARF in compiled binaries. (source: wiki/sources/descriptions/sevaa__dwex.md)

Useful as a Linux/ELF debug-info counterpart to Windows PDB tooling—not a debugger or decompiler itself.
