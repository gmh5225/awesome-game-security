# Android-ModMenu-SemiJni

Android game mod-menu template using a semi-JNI split: Java floating overlay UI via `WindowManager` (toggles/sliders) with a native C++ library for memory patching, function hooking, and value edits through JNI callbacks. Aimed at Android modders who want Java UI plus native game manipulation. (source: wiki/sources/descriptions/springmusk026__Android-ModMenu-SemiJni.md)

Complements GLES ImGui Unity templates ([[imgui-unity]], [[imgui-unity-with-layout]]), Android native ImGui apps ([[android-native-app-imgui]]), and iOS ImGui mod menus ([[imgui-ios-mod-menu]]) with a Java-overlay / semi-JNI path.
