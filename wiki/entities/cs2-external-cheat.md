# CS2-External-Cheat

External Counter-Strike 2 cheat (C++) with a D3D11 ImGui overlay. Modules cover aimbot, engine interaction, math utilities, external memory reading, offset management, and a DirectX 11 overlay renderer for cheat UI / ESP; useful for researchers studying external CS2 architecture and D3D11 overlay rendering.
