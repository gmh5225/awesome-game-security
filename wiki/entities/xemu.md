# xemu

Open-source original Xbox emulator (C; QEMU fork): LLE of NV2A GPU, MCPX audio, NForce chipset, and custom x86 Pentium III CPU to run ISO/XISO disc images—OpenGL GPU rendering, USB controllers, and a cross-platform SDL2 UI. (source: wiki/sources/descriptions/xemu-project__xemu.md)

Useful for emulation developers and preservation / hardware-internals research in the README `Xbox` lane—adjacent to the earlier QEMU-based original-Xbox stack [[xqemu]] and Xbox 360 HLE via [[xenia]], but focused on original Xbox LLE rather than 360 binary translation.
