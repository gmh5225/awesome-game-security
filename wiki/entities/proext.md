# ProExt

Rust-based external Counter-Strike 2 cheat. Feature set covers ESP, aimbot, triggerbot, radar, RCS (recoil control), bomb timer, spectator list, and crosshair overlay with a configurable egui UI; useful for researchers studying offensive techniques in the cheat / game:cs2 lane.
