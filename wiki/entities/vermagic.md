# vermagic

Tool to change **vermagic** and CRCs of a Linux Kernel Module (LKM). Also supports (not tested) CRC dump and changes. Useful for game-security researchers and reverse engineers studying offensive techniques in the cheat / RE tools lane—e.g. loading a module built against a different kernel build by rewriting vermagic/CRC metadata the loader checks. (source: wiki/sources/descriptions/yaxinsn__vermagic.md)
