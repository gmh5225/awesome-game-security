# ImGui-Unity-With-Layout

Enhanced Android Unity mod-menu template built on Dear ImGui with pre-built layout configs—organized tabs, categorized cheat options, styled widgets, and save/load settings in a C++ native library. Extends the basic [[imgui-unity]] GLES overlay path into a feature-complete ImGui UI for Unity titles on Android. (source: wiki/sources/descriptions/springmusk026__ImGui-Unity-With-Layout.md)

Complements the base Unity ImGui template ([[imgui-unity]]), Android native ImGui apps ([[android-native-app-imgui]]), and iOS ImGui mod menus ([[imgui-ios-mod-menu]]).
