# UE5-With-Dear-ImGui

Sample Unreal Engine 5.0.1 C++ project that incorporates Dear ImGui, built alongside a short YouTube tutorial. Aimed at game developers, engine programmers, and graphics researchers studying Unreal plugin / in-editor UI integration—not a cheat or anti-cheat artifact. (source: wiki/sources/descriptions/stungeye__UE5-With-Dear-ImGui.md)

Complements other Unreal Game Develop samples ([[action-roguelike]], [[simple-fps-template]], [[ue4-tutorials]]) by focusing on Dear ImGui wiring inside UE5 rather than gameplay systems or SDK dumps. Desktop ImGui overlay research ([[present-hook]], [[directxhook]]) is the adjacent graphics-API lane; this repo is the engine-plugin side of that surface.
