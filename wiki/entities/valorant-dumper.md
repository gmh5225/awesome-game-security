# valorant-dumper

Valorant-oriented Unreal offset/SDK dumper: reads the live game process to extract class definitions, function addresses, and memory offsets—scanning UE structures for GObject arrays, GNames, player entity layouts, and weapon data. Framed for researchers studying Valorant memory layout and for [[vanguard]] tooling analysis rather than as a maintained product feed. (source: wiki/sources/descriptions/xehn1337__valorant-dumper.md)

Sits in the same per-title Unreal dump lane as [[ts-ue4dumper]] / [[fortnite-fltokens-and-offsets]], but scoped to Riot’s Valorant process under Vanguard.
