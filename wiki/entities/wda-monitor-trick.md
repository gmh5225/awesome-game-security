# wda_monitor_trick

Windows Desktop Automation (WDA) monitor-hooking sample in C++ that uses DirectX 9 to intercept and manipulate display output. Includes D3D9 header definitions and Windows helpers for screen capture / display manipulation—useful for studying monitor-level capture interception and rendering tricks on Windows.
