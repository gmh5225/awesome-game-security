# Classroom

[[x64dbg]] plugin for analyzing object-oriented applications while debugging: define member functions and member variables, then persist and display class documentation on demand. Aimed at game-security researchers and reverse engineers in the Cheat / x64dbg Plugins lane. (source: wiki/sources/descriptions/x64dbg__Classroom.md)

Not a standalone debugger—extends [[x64dbg]] for live OOP structure documentation during RE sessions.
