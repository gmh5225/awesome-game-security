# RETools

Curated catalog of reverse-engineering and security tools organized by category—disassemblers, debuggers, decompilers, hex editors, network analyzers, binary-analysis frameworks, sandboxes, and malware-analysis utilities—with descriptions and links. Aimed at researchers selecting a toolkit for binary analysis workflows. (source: wiki/sources/descriptions/stevemk14ebr__RETools.md)

Index hub rather than a single tool binary; complements curated RE learning lists such as [[reverse-engineering]] and concrete debuggers such as [[x64dbg]].
