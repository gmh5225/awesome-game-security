# android-proxy-mcp

MCP (Model Context Protocol) server for Android HTTP/HTTPS packet capture aimed at AI assistants. Stack: mitmdump capture, SQLite storage, and natural-language query over captured traffic. Python; useful for game developers, reverse engineers, and tooling builders in the game-develop / MCP-server space. (source: wiki/sources/descriptions/zhizhuodemao__android_proxy_mcp.md)

Complements agent-facing Android RE tooling such as [[apktool-mcp-server]] by exposing network traffic rather than APK decode/rebuild.
