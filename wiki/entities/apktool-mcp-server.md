# apktool-mcp-server

MCP (Model Context Protocol) server wrapping [apktool](https://apktool.org/) for Android APK reverse engineering. Part of an Android Reverse Engineering MCP suite: exposes apktool workflows as MCP tools so agents and editors can decode/rebuild APKs programmatically. Aimed at game developers, reverse engineers, and tooling builders in the game-develop / MCP-server space. (source: wiki/sources/descriptions/zinja-coder__apktool-mcp-server.md)

Useful as an agent-facing front end to classic apktool decode/rebuild — not a substitute for jadx/IDA native analysis or Frida dynamic work.
