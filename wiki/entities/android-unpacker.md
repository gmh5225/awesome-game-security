# android-unpacker

Android-focused unpack tooling built to assist malware reversing and analysis (handle samples carefully). Aimed at anti-cheat engineers and defensive researchers in the Anti Cheat → Sample Unpacker lane rather than shipping as an AC product. (source: wiki/sources/descriptions/strazzere__android-unpacker.md)

README lane: `[Android]` — complements APK decode / deobf workflows such as [[apktool-mcp-server]], [[delamain]], and [[obfu-de-scate]]; peer to Dump Fix tooling such as [[unmapper]] on the unpack/repair side.
