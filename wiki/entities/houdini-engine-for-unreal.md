# HoudiniEngineForUnreal

Official SideFX Houdini Engine plugin for Unreal Engine. Brings Houdini’s procedural workflow into UE via Houdini Digital Assets (HDAs)—aimed at game developers, engine programmers, and graphics researchers in the Game Engine Plugins:Unreal lane, not a cheat or anti-cheat artifact. (source: wiki/sources/descriptions/sideeffects__HoudiniEngineForUnreal.md)

Sits beside other Unreal plugin / content-pipeline samples ([[ue5-with-dear-imgui]], [[jmap]]): ImGui wiring and `.uasset`/`.umap` graphs study editor/content surfaces; this repo is the DCC↔engine procedural-asset bridge.
