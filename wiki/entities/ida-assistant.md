# IDA-Assistant

IDA Pro plugin that integrates Claude-3 models via an interactive chat UI: ask questions, get guidance, and receive suggestions during reverse-engineering workflows. Aimed at game-security researchers and RE practitioners in the cheat / IDA Plugins lane. (source: wiki/sources/descriptions/stuxnet147__IDA-Assistant.md)

Complements agent-facing IDA bridges such as [[ida-mcp-server-plugin]] (MCP protocol) and [[idac]] (JSON CLI)—this path is an in-IDA Claude-3 chat assistant rather than a remote MCP/CLI client.
