# magiskboot_ndk_on_linux

Build configurations for compiling Magisk's `magiskboot` with the Android NDK on Linux. Patches the magiskboot source for standalone NDK toolchain builds, producing a native binary for Android boot-image unpack/repack and ramdisk manipulation. Aimed at developers and ROM builders who need NDK-built magiskboot for custom recovery and root workflows. (source: wiki/sources/descriptions/xiaoxindada__magiskboot_ndk_on_linux.md)

Relevant as a Boot Image Modification Tool in the Cheat Magisk / Android ROM–root lane: unpacking and repacking boot images underpins Magisk-style root and recovery research that mobile anti-cheat often detects.
