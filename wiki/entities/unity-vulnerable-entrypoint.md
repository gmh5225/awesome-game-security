# UnityVulnerableEntryPoint

Educational C# demos of vulnerable Unity entry points for code injection and game modification. Maps weak spots in Unity runtime initialization, `Assembly-CSharp` loading, and MonoBehaviour lifecycle where custom code can hook or alter behavior. Aimed at Unity developers learning client-side attack surfaces and researchers studying Mono/Unity-specific injection paths (complement to [[il2cpp]] AOT RE). (source: wiki/sources/descriptions/sunnamed434__UnityVulnerableEntryPoint.md)
