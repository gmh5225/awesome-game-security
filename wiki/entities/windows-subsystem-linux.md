# windows-subsystem-linux

Linux kernel source tree for Windows Subsystem for Linux (WSL) research and builds: full tree layout (`arch`, drivers, fs, and related subsystems) rather than a small WSL userspace wrapper. Useful for Windows-subsystem and kernel researchers studying WSL’s Linux-kernel side under the README `WSL` lane. (source: wiki/sources/descriptions/sxlmnwb__windows-subsystem-linux.md)
