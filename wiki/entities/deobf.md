# deobf

Python project aimed at `libtprt.so`, with plugin development and hooking for OLLVM-oriented deobfuscation / fix workflows. Useful for game-security researchers studying cheat-side Fix OLLVM techniques. (source: wiki/sources/descriptions/zhuzhu-Top__deobf.md)

Not a full deobfuscation framework by itself—scoped around the `libtprt.so` surface and plugin/hook plumbing.
