# Island

Experimental Vulkan renderer and realtime graphics framework (C/C++) for Linux (including Raspberry Pi 5) and Windows. Built for rapid prototyping and live tweaking: hot-reload of application modules, GLSL/Slang shaders, and imported images, with a modular layout aimed at very fast compile times. The Vulkan backend owns resource synchronization and validation; features include GPU ray tracing, hardware-accelerated Vulkan Video decode, multi-window swapchains, Tracy profiling, and static release builds. Aimed at developers and researchers prototyping modern realtime rendering without a heavyweight engine. (source: wiki/sources/descriptions/tgfrerer__island.md)

Sits in the Vulkan / Renderer lane—graphics and engine-source study rather than a cheat or anti-cheat artifact.
