# FlatRedBall

Mature C# / .NET 2D game engine with an editor, runtime libraries, and add-ons for content loading, scripting, and debugging. Targets productive 2D development on modern .NET with an editor-driven workflow. (source: wiki/sources/descriptions/vchelaru__FlatRedBall.md)
