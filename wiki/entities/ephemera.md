# Ephemera

Multiplatform `MEMORY.DMP` analysis tool with a WinDbg-style workflow, aimed at faster offline crash-dump inspection than native WinDbg (including on Windows). Useful for anti-cheat engineers and defensive researchers analyzing Windows kernel dumps adjacent to AC / kernel RE. Complements RAM-image frameworks such as [[volatility]] / [[volatility3]] and live WinDbg automation such as [[windbg-scripts]]. (source: wiki/sources/descriptions/vmi-rs__ephemera.md)
