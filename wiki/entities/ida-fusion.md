# IDA-Fusion

Fast signature scanner and creator for IDA. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane who need unique signatures that hold up against anti-RE tricks such as duplicated binary parts—where many existing scanners generate non-unique or buggy patterns. (source: wiki/sources/descriptions/senator715__IDA-Fusion.md)

Complements IDA-side signature/pattern tooling such as [[yarascan-ida]] (Yara file scan) and library-level scanners such as [[patternsleuth]] (SIMD multi-pattern; file or process).
