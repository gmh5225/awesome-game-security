# Skylicht Engine

Lightweight game engine evolved from the Irrlicht project, targeting mobile platforms (Android, iOS). Aimed at game developers, engine programmers, and graphics researchers studying a compact mobile-oriented engine source tree. (source: wiki/sources/descriptions/skylicht-lab__skylicht-engine.md)

Sits in the Game Engine / source lane—an OSS mobile-focused engine codebase for development and RE study, rather than a cheat or anti-cheat artifact.
