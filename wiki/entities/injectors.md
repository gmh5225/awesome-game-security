# Injectors

C/C++ injection-testing repo (README: Injection Testing). Useful as a stress-test / harness reference for anti-cheat and defensive researchers studying how injected code surfaces under detection. Touches driver development and modding-adjacent paths rather than a single AC product. (source: wiki/sources/descriptions/zoand__Injectors.md)

Treat as an injection-testing sample set for defensive evaluation—not a production cheat stack.
