# LUDA

Lua scripting plugin for IDA Pro with direct IDA SDK access from Lua, aimed at rapid prototyping and automation of reverse-engineering workflows. Useful for game-security researchers in the cheat / IDA Plugins lane. (source: wiki/sources/descriptions/stolevchristian__LUDA.md)

Complements Python-oriented IDA tooling such as [[sark]] and agent bridges like [[ida-mcp-server-plugin]]—this path exposes the SDK to Lua rather than IDAPython or MCP.
