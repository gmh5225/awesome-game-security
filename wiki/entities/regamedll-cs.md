# ReGameDLL_CS

Reverse-engineered C++ reimplementation of the Counter-Strike 1.6 server game DLL (`mp.dll`) by s1lentq. Drop-in GoldSource-compatible replacement with bug fixes, performance work, and extended API hooks for server plugins; covers game rules, player/weapon systems, buy menu, and bot AI. Useful for CS 1.6 server operators, plugin developers, and researchers studying GoldSrc/CS1.6 server game logic rather than cheat or anti-cheat artifacts. (source: wiki/sources/descriptions/s1lentq__ReGameDLL_CS.md)
