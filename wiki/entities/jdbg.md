# JDBG

Lightweight Java debugger that attaches to running Java/JVM processes over the JVM Debug Interface (JDWP). Supports method hooking, breakpoints, variable inspection, and class enumeration for runtime analysis of Java-based game clients and obfuscated JVM applications. Aimed at Java reverse engineers and security researchers. (source: wiki/sources/descriptions/roger1337__JDBG.md)

Runtime JDWP attach lane—not a static bytecode assembler/disassembler ([[raung]]) or DEX decompiler ([[jadx]]).
