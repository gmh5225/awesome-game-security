# MapleStory-YOLOv8-Training

Python YOLOv8 training project for MapleStory object detection, accelerated with Apple MPS. Fits the cheat / game:maplestory lane as the model-training half of offensive computer-vision pipelines (label → train → infer), complementary to runtime automation such as [[maplestory-worlds-automation]]. Useful for researchers studying AI visual cheats and title-specific detector training without classic memory RPM. (source: wiki/sources/descriptions/tingwei1111__MapleStory-YOLOv8-Training.md)
