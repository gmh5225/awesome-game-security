# JADX

DEX to Java decompiler for Android APK, DEX, and AAR inputs. Ships a CLI and a Swing GUI with search, navigation, deobfuscation, and export. Reconstructs complex control flow, exception handlers, inner classes, annotations, and enums into readable Java. Aimed at Android reverse engineers, malware analysts, and mobile game-security researchers inspecting application internals. (source: wiki/sources/descriptions/skylot__jadx.md)

Core static DEX/Java recovery lane; agent-facing headless MCP wraps such as [[delamain]] and apktool decode/rebuild via [[apktool-mcp-server]] sit alongside it for AI-driven or resource-focused workflows.
