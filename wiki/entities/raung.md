# Raung

Early-stage assembler/disassembler for Java bytecode. Syntax and feature set are still unstable; many capabilities are unfinished. Aimed at reverse engineers and game-security researchers studying offensive techniques in the cheat / RE-tools lane—especially Java/.NET-adjacent client and launcher analysis alongside tools such as dnSpy. (source: wiki/sources/descriptions/skylot__raung.md)

Useful for bytecode inspect/patch experiments—not a full decompiler, debugger, or production Java toolchain.
