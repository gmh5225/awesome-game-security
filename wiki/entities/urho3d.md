# Urho3D

Free lightweight, cross-platform 2D and 3D game engine. Useful for game developers, engine programmers, and graphics researchers studying a complete open engine source tree. The founder (Lasse Öörni) is currently working on Turso3D. (source: wiki/sources/descriptions/urho3d__Urho3D.md)

Sits in the Game Engine / source lane—an OSS engine codebase for development and RE study, rather than a cheat or anti-cheat artifact.
