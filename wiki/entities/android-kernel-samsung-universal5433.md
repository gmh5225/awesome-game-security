# android_kernel_samsung_universal5433

Samsung Galaxy kernel source for the Exynos 5433 SoC (Galaxy Note 4 / Galaxy Alpha and peers). Linux 3.10-era tree with Samsung board support, ARM big.LITTLE scheduling, and proprietary display/camera/modem drivers — useful for Android kernel-level security research in the cheat / Android Kernel Source lane.
