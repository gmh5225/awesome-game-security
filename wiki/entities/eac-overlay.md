# eac-overlay

C++ proof-of-concept overlay renderer aimed at Easy Anti-Cheat–protected games. Demonstrates ESP/visual overlays via alternative rendering surfaces or window manipulation intended to avoid EAC overlay monitoring and screenshot-style detection—useful for studying overlay-detection bypass and AC visual-monitoring limits, not a maintained product. (source: wiki/sources/descriptions/xBrunoMedeiros__eac-overlay.md)
