# Turbulenz Engine

Open-source HTML5 game engine in TypeScript/JavaScript. Full browser framework: WebGL 3D rendering, physics, audio, networking, input, asset management, and a scene graph, plus server-side services for leaderboards, multiplayer, and metrics. Aimed at web developers building 2D/3D browser games. (source: wiki/sources/descriptions/turbulenz__turbulenz_engine.md)

Sits in the Game Engine / HTML5 source lane—an OSS web-engine tree for development and RE study of browser runtime + services, rather than a cheat or anti-cheat artifact.
