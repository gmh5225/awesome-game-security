# cet-research

C research material on Intel/Windows CET (Control-flow Enforcement Technology), including shadow-stack related behavior. Aimed at Windows kernel and platform-security researchers studying hardware-enforced stack protection under the README `Windows Security Features` / CET lane. (source: wiki/sources/descriptions/yardenshafir__cet-research.md)

Useful context when modeling how CET/shadow stacks raise the cost of ROP-style control-flow abuse alongside VBS/[[hvci]] and other platform-trust baselines used by modern anti-cheat.
