# tinyobjloader

Lightweight single-header C++ library for parsing Wavefront OBJ 3D model files. Loads vertex positions, normals, texture coordinates, materials (MTL), and face indices; supports triangulation, smoothing groups, and per-face materials. Header-only, no external dependencies, aimed at game developers and graphics programmers who need a portable OBJ loader. (source: wiki/sources/descriptions/tinyobjloader__tinyobjloader.md)

Sits in the README Wavefront Obj lane—mesh-format ingest for rendering / engine asset pipelines rather than a GPU Present hook or cheat overlay.
