# PcapPlusPlus

Multiplatform C++ library for network packet capture, parsing, and crafting. High-level APIs cover 100+ protocols (Ethernet, IPv4/IPv6, TCP, UDP, HTTP, DNS, TLS, and more), wrap libpcap/WinPcap/Npcap for live capture, and read/write PCAP/PCAPNG files—aimed at network-security developers, protocol researchers, and packet-analysis tooling. (source: wiki/sources/descriptions/seladb__PcapPlusPlus.md)

Sits under README Cheat → Packet Capture&Parse (`[Pcap]`) beside NDIS packet-filter APIs such as [[ndisapi]].
