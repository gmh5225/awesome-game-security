# binja-kc

Binary Ninja plugin for loading MachO kernelcache and dSYM files. Loads symbols and type information from multiple KDK dSYM files onto a corresponding kernelcache opened in Binary Ninja. Useful for game-security / RE researchers in the Cheat Binary Ninja Plugins lane. (source: wiki/sources/descriptions/skr0x1c0__binja_kc.md)

Not a standalone kernel debugger—scoped to Binary Ninja kernelcache + KDK symbol/type enrichment.
