# WinDbg_Scripts

Collection of WinDbg scripts (primarily JavaScript) oriented toward kernel-level debugging, modding, and analysis. Useful for game-security researchers and reverse engineers working in the README Cheat → WinDbg Plugins lane. (source: wiki/sources/descriptions/yardenshafir__WinDbg_Scripts.md)

Not a standalone debugger—extends WinDbg automation for inspecting kernel state during cheat / AC-adjacent RE workflows.
