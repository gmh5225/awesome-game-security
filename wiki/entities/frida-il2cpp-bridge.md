# frida-il2cpp-bridge

[[frida]] bridge for dumping Unity [[il2cpp]] runtime metadata/types. Aimed at researchers studying offensive dump workflows in the cheat / game engine explorer:Unity lane. (source: wiki/sources/descriptions/vfsfitvnm__frida-il2cpp-bridge.md)

Claimed Unity version coverage: 5.3.0–6000.1.x. Complements static APK tools such as [[il2cpp-spy]] and iOS IL2CPP helpers like [[unityspeedtools]] when the goal is live Frida-driven exploration rather than offline disassembly.
