# kernel_hack

Linux loadable kernel module (LKM) for game memory hacking on Android and Linux. Provides kernel-mode memory read/write, process management, and verification paths, built with Kconfig-based configuration — useful when studying LKM-based game-memory access and process control on mobile/Linux hosts. (source: wiki/sources/descriptions/rogxo__kernel_hack.md)

Sits in the same Android/Linux kernel memory-ops lane as [[android-kernel-hacking-toolkit]], [[root-socket-kit]], and CLI scanners such as [[mypower]].
