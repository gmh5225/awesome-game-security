# Wazuh

Open-source security monitoring / XDR platform: host intrusion detection, log analysis, file integrity monitoring, vulnerability scanning, and compliance auditing across Linux, Windows, and macOS. Agent–manager architecture correlates events, applies detection rules, and integrates with Elasticsearch for visualization; C/Python codebase with real-time alerting, active response, and a REST API. (source: wiki/sources/descriptions/wazuh__wazuh.md)

Useful as a README `[XDR]` reference for enterprise HIDS/SIEM-style host telemetry adjacent to game-security AC/EDR detection research—not a game anti-cheat product. Complements Windows ETW/injection consumers such as [[tietwagent]] and provider browsers such as [[etw-explorer]].
