# simpleperf_demo

Demo / sample project focused on **perf** (Android simpleperf) for profiling Android applications. Useful for game-security researchers and reverse engineers in the cheat / Android kernel explorer lane who need CPU/perf sampling alongside memory and root tooling. (source: wiki/sources/descriptions/yabinc__simpleperf_demo.md)

Complements DIY Android kernel explorers such as [[op7t]] by emphasizing application-level performance profiling rather than custom kernel builds.
