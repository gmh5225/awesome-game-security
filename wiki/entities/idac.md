# idac

Trail of Bits IDA Pro client library and CLI for agents and humans: connects over a Unix socket to a live IDA GUI or headless `idalib`, returning structured JSON for batch analysis, function listing, decompilation, cross-references, and type queries. Supports batch/preview/dry-run workflows and ships a bundled agent skill; explicitly **not** an MCP server (early alpha). (source: wiki/sources/descriptions/trailofbits__idac.md)

Not a disassembler—drives an existing IDA instance for scripted / agent-assisted RE.
