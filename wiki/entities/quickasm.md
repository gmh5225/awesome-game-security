# QuickAsm

Lightweight Windows GUI (C++) for editing, assembling, and executing x86/x64 snippets in-process: type instructions, assemble in real time via Keystone, inspect machine-code bytes, and run the result. Aimed at reverse engineers and security researchers prototyping assembly or testing shellcode. (source: wiki/sources/descriptions/zodiacon__QuickAsm.md)

Useful for quick asm experimentation—not a full emulator, debugger, or game-attached injector.
