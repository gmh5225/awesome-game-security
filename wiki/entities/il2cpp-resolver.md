# IL2CPP_Resolver

C++ [[il2cpp]] resolver for Unity analysis. Aimed at game-security researchers and reverse engineers studying offensive techniques in the cheat / game engine explorer:Unity lane. (source: wiki/sources/descriptions/sneakyevilSK__IL2CPP_Resolver.md)

Complements dump/disasm workflows such as [[il2cpp-spy]] and live Frida bridges like [[frida-il2cpp-bridge]] when the goal is resolving IL2CPP runtime structures rather than offline APK diffing alone.
