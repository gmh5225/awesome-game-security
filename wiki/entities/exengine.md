# exengine

Minimal C99 3D game engine using OpenGL. Provides basic 3D rendering with lighting, IQM model loading, scene management, text rendering, input handling, and a simple entity system. Emphasizes simplicity and readability as a learning resource for hobbyist developers and students studying 3D engine architecture from scratch in C. (source: wiki/sources/descriptions/solenum__exengine.md)

Sits in the Game Engine / OpenGL source lane—an educational engine codebase rather than a cheat or anti-cheat artifact.
