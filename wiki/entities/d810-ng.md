# d810-ng

D-810ng (Next Generation) — evolution of d810 that deobfuscates code at decompilation time. Aimed at game-security researchers and reverse engineers in the cheat / Fix OLLVM lane studying offensive static RE techniques. (source: wiki/sources/descriptions/w00tzenheimer__d810-ng.md)

Not a full unpacker framework—scoped as decompilation-time deobfuscation tooling (successor to classic d810).
