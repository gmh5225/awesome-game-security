# Socket.IO

Real-time bidirectional event-based communication library for Node.js and browsers. Uses WebSocket as the primary transport with automatic HTTP long-polling fallback, plus auto-reconnection, namespace multiplexing, room-based broadcasting, and binary data support. JS/TS server and client components aimed at multiplayer games, chat, and live dashboards. (source: wiki/sources/descriptions/socketio__socket.io.md)

Sits under README `Game Network` / `[Nodejs]` as a study surface for event-based real-time game and backend stacks rather than client-side cheat tooling.
