# PatternSleuth

Rust library for high-performance binary pattern / signature scanning in files and live process memory: multi-pattern match with wildcards, SIMD-optimized scanners, and batch scans over large regions. Aimed at game hackers, reverse engineers, and anti-cheat developers needing fast offset discovery or signature detection; README positions it as an Unreal Engine address scanner and test suite.
