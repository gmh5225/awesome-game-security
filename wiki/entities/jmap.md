# jmap

Tool for mapping and visualizing Unreal Engine asset dependencies: parses UE4/UE5 `.uasset` / `.umap` files into dependency graphs (loading chains, circular refs, content structure). Aimed at modders and analysts studying game asset relationships; README positions it as an Unreal Engine reflection-data format and extractor. (source: wiki/sources/descriptions/trumank__jmap.md)
