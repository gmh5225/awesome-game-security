# Volatility 3

Third major release of the Volatility memory forensics framework: a complete Python 3 rewrite for extracting digital artifacts from volatile memory (RAM) samples. Supports Windows, Linux, and macOS images via a plugin architecture—process lists, network connections, registry hives, kernel objects, and other runtime state—without needing live access to the investigated host. Includes symbol-table generation, layer-based memory translation, and automagic OS-profile detection. Standard tool for incident response, malware investigation, and offline memory analysis adjacent to game-security / AC RE workflows. Predecessor: [[volatility]] (Python 2). (source: wiki/sources/descriptions/volatilityfoundation__volatility3.md)
