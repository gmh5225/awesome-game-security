# Olive.c

Simple dependency-free 2D graphics library in C (single-header). Software-rasterizes lines, filled triangles, circles, text, and basic image compositing into raw pixel buffers via the C standard library only; any windowing system can display the buffer. Aimed at beginners and hobbyists learning graphics fundamentals. (source: wiki/sources/descriptions/tsoding__olive.c.md)

Sits in the README Image Codec / soft-raster lane (alongside stb-style helpers)—a minimal CPU raster study surface rather than a GPU API hook or cheat overlay.
