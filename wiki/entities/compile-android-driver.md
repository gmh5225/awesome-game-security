# compile_android_driver

Guide and build system (**kade**) for compiling out-of-tree Android kernel drivers with ABI symbol management. Covers lyenv-based automation for GKI and non-GKI builds via `kadeflow.yaml`, ABI patching, a sample file-copy module, and Kadeflow CI/CD (GitHub Actions) for automated builds. Useful when researching Android kernel-module / driver workflows adjacent to custom-kernel and LKM tooling. (source: wiki/sources/descriptions/systemnb__compile_android_driver.md)
