# ida_functioncolor

Python IDA plugin that colorizes function definitions in Hex-Rays / pseudocode views. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane for quicker visual triage of decompiled functions. (source: wiki/sources/descriptions/sneakyevil__ida_functioncolor.md)

UI/annotation helper—not a deobfuscator or unpacker.
