# tinygltf

Header-only C++11 library (TinyGLTF) for loading and saving glTF 2.0 3D models. Parses glTF JSON and GLB binary; covers meshes, materials, textures, animations, skins, cameras, and extensions. Depends only on stb_image for textures; supports files, memory buffers, and custom I/O callbacks. Aimed at game developers and 3D apps needing a lightweight portable glTF loader. (source: wiki/sources/descriptions/syoyo__tinygltf.md)

Sits in the README Game Assets / glTF lane—runtime mesh-format ingest for engine and render pipelines, alongside Wavefront OBJ helpers such as [[tinyobjloader]], rather than a GPU Present hook or cheat overlay.
