# SE-tools

Nintendo Switch homebrew toolkit for game memory editing and cheat development. Includes a memory scanner, pointer searcher, cheat-code manager, and real-time memory viewer that talk to Atmosphere CFW via the `dmnt:cht` service; supports creating and applying cheats in the standard Atmosphere format. Aimed at Switch homebrew developers and modders building memory-based cheats. (source: wiki/sources/descriptions/tomvita__SE-tools.md)
