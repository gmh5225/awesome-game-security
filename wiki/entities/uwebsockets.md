# uWebSockets

High-performance WebSocket and HTTP server library (C++ with Node.js bindings). Implements the WebSocket protocol with minimal overhead—millions of connections per server and microsecond-level latency—plus TLS (OpenSSL/BoringSSL), HTTP routing, pub/sub messaging, and per-socket backpressure handling. Aimed at game-server and backend engineers building high-throughput real-time networking services. (source: wiki/sources/descriptions/uNetworking__uWebSockets.md)

Sits in the README `[WebSockets]` / `Game Network` lane as a study surface for authoritative real-time server stacks rather than client-side cheat tooling.
