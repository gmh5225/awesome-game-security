# feather-gb

Rust Game Boy emulator (author’s first Rust project): aimed at retro handheld emulator developers and reverse engineers working in the Game Boy area. (source: wiki/sources/descriptions/vojty__feather-gb.md)

Useful for emulator architecture / hardware-study in the README `Game Boy` category—peer to [[kevboy]], and adjacent to console-emulation research such as [[xqemu]], but focused on Game Boy rather than Xbox full-machine playback.
