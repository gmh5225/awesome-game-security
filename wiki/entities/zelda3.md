# zelda3

Reimplementation of Zelda 3 that is playable from start to end. Aimed at game developers, reverse engineers, and tooling builders working in the Game Develop / source lane—not a cheat or anti-cheat artifact. (source: wiki/sources/descriptions/snesrev__zelda3.md)

Useful as a readable classic-title remake surface for studying game structure and tooling, adjacent to other Game Develop / source samples ([[action-roguelike]], [[cs16-client]]) and distinct from hardware emulators such as [[kevboy]] / [[feather-gb]].
