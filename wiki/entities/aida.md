# AiDA

AI-powered assistant for IDA 9.0+ aimed at accelerating reverse engineering of C++ games (and other Windows C++ binaries). Useful for game-security researchers and RE practitioners in the cheat / IDA Plugins lane. (source: wiki/sources/descriptions/sigwl__AiDA.md)

Complements other LLM/agent IDA paths such as [[ida-assistant]] (in-IDA Claude-3 chat), [[ida-mcp-server-plugin]] (MCP bridge), and [[idac]] (JSON CLI)—this project targets IDA 9.0+ C++ game RE workflows specifically.
