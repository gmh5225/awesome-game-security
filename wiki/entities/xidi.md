# Xidi

Windows C++ proxy DLL that intercepts XInput and DirectInput calls to provide virtual controller mapping and input translation. Acts as a compatibility layer so modern XInput pads work with legacy DirectInput games (and vice versa), with configurable button maps, deadzones, and force-feedback translation—aimed at players and developers fixing controller API mismatches. Catalogued under DirectX Compatibility. (source: wiki/sources/descriptions/samuelgr__Xidi.md)
