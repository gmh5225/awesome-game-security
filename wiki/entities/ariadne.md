# Ariadne

Binary Ninja–oriented graph analysis plugin. Implements common static-analysis tasks including call-graph analysis and can integrate block coverage information. Useful for game-security / RE researchers studying offensive techniques in the Cheat Binary Ninja Plugins lane. (source: wiki/sources/descriptions/seeinglogic__ariadne.md)

Not a full decompiler—scoped to graph / call-graph and coverage-assisted static analysis inside Binary Ninja.
