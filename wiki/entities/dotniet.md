# dotNIET

IDA Pro Python plugin for deobfuscating .NET Native (NativeAOT) binaries. Reconstructs type information, method names, and metadata references lost during Native AOT compilation, mapping native code back to original .NET types and methods. Aimed at reverse engineers analyzing .NET Native apps where standard managed decompilers cannot be used (README lane: Import .NET Symbol). (source: wiki/sources/descriptions/synacktiv__dotNIET.md)
