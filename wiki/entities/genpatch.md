# genpatch

Python script for patching binaries. When a patch script generates successfully, a confirmation dialog appears. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane studying offensive binary-patch workflows. (source: wiki/sources/descriptions/sterrasec__genpatch.md)

Companion to other IDA-side automation such as [[sark]] and [[luda]]—focused on producing binary patches rather than demangling or scripting the SDK.
