# Il2cppSpy

Unity [[il2cpp]] disassembler aimed at Android APKs. Select two APKs and the tool surfaces the difference between them—useful for tracking build-to-build IL2CPP binary/metadata changes in Unity mobile titles. (source: wiki/sources/descriptions/yukiarrr__Il2cppSpy.md)

Fits the cheat / game-engine explorer:Unity lane for researchers comparing protected or updated mobile builds without relying only on IL2CPPDumper-style dumps.
