# ndisapi

User-mode interface library for the Windows Packet Filter driver. Provides a straightforward, low-overhead API to filter (inspect and modify) raw network packets at the NDIS level of the Windows network stack—useful for game-security and RE work in the Cheat → Packet Sniffer&Filter lane.
