# pikabot-deobfuscator

IDA plugin for deobfuscating Pikabot strings with RC4 and AES. After loading a sample, decrypt from the Hex-Rays decompiler context menu for the current function or all functions. Aimed at game-security researchers and reverse engineers in the cheat / IDA Plugins lane studying offensive string-obfuscation recovery. (source: wiki/sources/descriptions/threatlabz__pikabot-deobfuscator.md)

Not a general unpacker—scoped to Pikabot RC4/AES string decrypt inside IDA.
