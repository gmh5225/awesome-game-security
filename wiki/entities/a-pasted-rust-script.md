# a-pasted-rust-script

Rust game-cheat sample with an ImGui-based loader, anti-debugging checks, Base64 helpers, and a feature-module layout. Companion C++ / Visual Studio pieces cover menu rendering, anti-debug paths, and cheat feature wiring. Aimed at researchers studying cheat loader architecture and basic anti-debug techniques in mod tools—not a polished production stack. (source: wiki/sources/descriptions/spyder1g__a-pasted-rust-script.md)

Useful as a compact loader + menu + anti-debug reference alongside [[mini-launcher]] and ImGui overlay samples.
