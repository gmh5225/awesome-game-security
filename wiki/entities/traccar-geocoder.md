# traccar-geocoder

Self-hosted reverse geocoder for the Traccar stack: OpenStreetMap data, Rust server, C++ PBF indexer, Docker deployment. Turns lat/lon into place names without a third-party geocoding API. Aimed at backend and multiplayer/online game developers who need location/geocoding infrastructure in the Game Network lane. (source: wiki/sources/descriptions/traccar__traccar-geocoder.md)

Sits under README `Game Network` → Location / Geocoding as a study surface for self-hosted geo lookup rather than client-side cheat tooling.
