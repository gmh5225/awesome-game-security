# jwt_tool

Python toolkit for testing, tampering, and cracking JSON Web Tokens (JWT). Supports decoding tokens, modifying claims and headers, probing common JWT flaws (algorithm confusion none/HS256/RS256, key confusion, JKU/X5U injection), brute-forcing weak signing keys, and automated API scanning modes. Aimed at penetration testers and web security researchers auditing JWT-based authentication. (source: wiki/sources/descriptions/ticarpi__jwt_tool.md)

Sits in the README `Game Network` → JWT/Auth Token lane as a study surface for game backend / API auth hardening—not client-side cheat tooling.
