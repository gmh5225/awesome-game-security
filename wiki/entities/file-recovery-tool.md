# FileRecoveryTool

Windows disk forensics and file-recovery tool for NTFS, FAT32, and ExFAT: MFT scanning, USN Journal analysis, signature-based file carving, fragmented-file reassembly, and sector-level deleted-file recovery. Pure Win32 C++, zero dependencies, multi-threaded GUI, direct disk access. (source: wiki/sources/descriptions/wesmar__FileRecoveryTool.md)

Useful lab reference for recovering deleted artifacts (cheat payloads, logs, dumps) opposite the Cheat Anti Forensics lane, and for AC Information System & Forensics workflows that inspect disk residue.
