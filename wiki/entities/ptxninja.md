# ptxNinja

Binary Ninja plugin for reverse engineering PTX — NVIDIA CUDA’s GPU virtual ISA. Installable via Binary Ninja’s plugin manager. Useful for game-security / RE researchers studying GPU-side binaries in the Cheat Binary Ninja Plugins lane. (source: wiki/sources/descriptions/seekbytes__ptxNinja.md)

Not a standalone disassembler—scoped to Binary Ninja PTX / CUDA virtual-ISA analysis.
