# Obfusk8

C++17 obfuscation library combining compile-time and runtime techniques to protect code logic and data from static/dynamic analysis. Aimed at Anti Cheat → Compile Time / Obfuscation Engine research for defensive engineers. (source: wiki/sources/descriptions/x86byte__Obfusk8.md)

Useful as a broader compile-time+runtime obfuscation reference; string-focused spin-off [[sbox]] covers constexpr AES-128 / S-box encryption. Not a commercial protector or unpacker.
