# mypower

Cross-platform CLI memory scanner for Android and Linux. Features JIT-compiled scan expressions via SLJIT, pointer-chain discovery with configurable depth/offset limits, snapshot-based differential filtering, an ncurses TUI, and built-in Unity (U3D) game-object inspection.
