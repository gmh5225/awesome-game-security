# Metal-Game-Engine-Tutorial

Open tutorial / sample engine built around Apple's Metal API. The full project is available to view, edit, and optimize—aimed at game developers, engine programmers, and graphics researchers studying Metal-backed engine structure rather than as a cheat or anti-cheat artifact.
