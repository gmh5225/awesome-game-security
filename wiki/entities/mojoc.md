# Mojoc

Cross-platform, open-source pure C game engine for mobile games. Built on OpenGLES3 and written in C99. Aimed at game developers, reverse engineers, and tooling builders studying a compact mobile engine source tree in the Game Develop / Game Engine source lane. (source: wiki/sources/descriptions/scottcgi__Mojoc.md)

Sits in the Game Engine / OpenGL ES source lane—an OSS mobile-focused C99 engine codebase rather than a cheat or anti-cheat artifact.
