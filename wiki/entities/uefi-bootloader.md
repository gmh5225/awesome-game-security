# UEFI-Bootloader

Simple UEFI runtime-driver manual mapper (C/C++) for studying cheat / EFI driver load paths: map an unsigned payload via a UEFI runtime driver rather than a normal Windows driver load. Useful for game-security and RE work on pre-OS / runtime EFI mapping and hooking. (source: wiki/sources/descriptions/sa413x__UEFI-Bootloader.md)
