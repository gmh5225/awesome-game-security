# WDUTF

Windows Driver Unit Test Framework — run unit tests against Windows kernel-driver code using the **Microsoft Unit Testing Framework for C++** in **user space**, avoiding the usual pain of exercising kernel-targeted code only inside a live driver. Aimed at anti-cheat engineers and defensive researchers who need a driver unit-test harness. (source: wiki/sources/descriptions/wpdk__wdutf.md)
