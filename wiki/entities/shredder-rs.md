# shredder-rs

Rust x86_64 polymorphic mutation engine that shreds instructions (fragmentation + context preservation) to defeat static analysis. Aimed at anti-cheat / obfuscation-engine research for defensive engineers. (source: wiki/sources/descriptions/zx0CF1__shredder-rs.md)

Useful as a reference for instruction-level polymorphism alongside packer/OLLVM-style obfuscation in the AC tooling surface—not a game client or AC product itself.
